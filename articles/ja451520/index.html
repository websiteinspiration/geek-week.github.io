VDI：安くて明るい



こんにちは、親愛なるカブロバイト、友人、知人。序文として、1つの興味深いプロジェクトの実装についてお話ししたいと思います。今では、流行になっているため、VDIインフラストラクチャの展開に関する1つの興味深い事例についてお話します。 VDIには多くの記事があり、直接の競合他社との比較があり、競合するソリューションの比較もあり、段階的に比較しています。新しい何かを提供できるように思われましたか？

そして、多くの記事にない新しいものは、実装の経済効果の説明、選択したソリューションの所有コストの計算、そしてさらに興味深いのは、類似のソリューションとの所有コストの比較です。この場合、記事のタイトルに基づいて、キーワードは安いです： どういう意味ですか？今年の初めに、私の友人の1人である友人は、最小数の「ウィンドウ」、つまり無料のハイパーバイザー、Linuxデスクトップ、無料のデータベース、および「最愛の」Microsoftでコストを最小限に抑えるその他の方法でVDIを実装するタスクを抱えていました。

そして、なぜ「最小ウィンドウ」とは？ここでは、さらなるナレーションから離れて、この特定のトピックの開示に興味を持った理由を説明します。私がプロジェクトの展開を手伝った私の友人は、約500人を超えるスタッフがいる平均的な会社で働いています。すべてのソフトウェアが合法であるわけではありませんが、最適化のための作業が進行中です。フロントエンドの情報システムのほとんどはWEBに適合しており、晴れた日まで気分は良好でしたコレクターが来なかったMicrosoftの「パーソナルマネージャー」は会社に割り当てられ、提供せず、要求せず、すべてを緊急に強制的に合法化し、オープンソースとプレスリリースに基づいて使用されるソリューションについて多くの結論を出しました。同社は気にしていないようだったが、脅威に接するこの重要性と強迫観念は、MS製品の使用とオープンソースでの最大限の注意を最小限にするための輸入代替の長期計画に拍車をかけた。外部からの人は、ソフトウェアジャイアントの代表と一緒に説明されている状況を特に信じないかもしれませんが、マイクロソフトの従業員から個人的に指定された圧力で、同じような状況が1対1で繰り返されたことがあります。

一方、これは、有料ソフトウェア製品の使用を多様化するために、IT部門の開発戦略を見直すための追加のトリガーです。繰り返しになりますが、ビジネス向けのオープンソースソリューションの浸透傾向が勢いを増しており、IT-OS 0219カンファレンスでこのトピックについての議論があり、以下のスライドでこれを完全に確認しています。

VDI仮想化の役割を果たしている素晴らしいEMCストレージ、HP c7000のブレードバスケットと7 G8サーバーがありました。 Hyper-Vの役割を持つWindows Server 2012R2がサーバーにインストールされ、SCVMMが使用されました。 XenDesktop 7.18に基づいて取得したVDIプラットフォームが導入され、いくつかのターミナルファームが導入されました。処分と多数のソフトウェアのライセンスの必要性を理解した上で、Linux VDIと完全なターンキーMicrosoftベースのソリューションの導入コストを比較してみましょう。翻訳を段階的に導入することが決定され、最初の段階では会社の支部が影響を受け、2番目の段階では市民社会の残りの仕事の移管が行われました。



ターミナルファームでは、1Cが主に機能し、VDIデスクトップでは標準のオフィススイート、メール、ファイル、インターネットが機能しました（主な機能は読み取りと印刷のみでした）。

Windows Server：

Microsoftのライセンス要件に従って、次の条件を満たす必要があります。

  1. サーバーのすべての物理コアにライセンスを付与する必要があります。
  2. 2コアサーバーライセンスの最小パッケージは8個です。（または1つの16コアライセンス）。
  3. 2コアプロセッサライセンスの最小パッケージは4個です。（このルールは、プロセッサーの数が3つ以上の場合に有効になります）。
  4. Standard License Packは、1台のサーバーでWindows Serverの1つの物理インスタンスと2つの仮想インスタンスを使用する権利を付与します。
  5. Datacenterライセンスパッケージは、単一サーバー上でWindows Serverの1つの物理インスタンスと任意の数の仮想インスタンスを使用する権利を提供します。

13台を超えるWindows ServerおよびWindowsワークステーションの仮想コピーをサーバーにインストールする場合は、Datacenterエディションを購入することが経済的に実現可能であることがわかります。
