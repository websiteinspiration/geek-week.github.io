适用于Android的Firefox预览版Nightly获得对uBlock Origin的支持

按照承诺，Mozilla将第一个扩展程序uBlock Origin从“推荐的扩展程序”移到了夜间版本的Firefox Preview（代号为“ Fenix”）。您已经可以安装它并在下一版Firefox for Android中启用广告拦截。

Mozilla最近发布了具有新功能和改进功能的Firefox Preview v3.0。在用户请求添加扩展支持后，该公司在其博客上宣布扩展支持将在2020年1月出现在Firefox预览版中，并且它正在努力首先将扩展添加到其“推荐扩展程序”列表中。

“将GeckoView和Firefox Preview与以前的Firefox for Android支持的API保持一致不会一overnight而就。 Phillip Quish说：“在2019年剩余时间内和2020年之前，我们将专注于支持“推荐扩展程序”中的自定义扩展程序，该程序可在移动设备上稳定运行并支持许多功能。

基于GeckoView的Firefox Preview Nightly现在支持WebExtensions，而uBlock Origin是更新后添加的第一个扩展。

要在Firefox Preview Nightly中安装uBlock Origin，请执行以下操作：

  1. 从Play商店下载并每晚安装Firefox Preview ;
  2. 每晚启动Firefox Preview，单击三点菜单并转到“设置”；
  3. 在“高级”部分，输入“附加组件”，该附加组件显示在“语言”部分的上方；
  4. 您将看到uBlock Origin作为推荐的附件，单击“ +”图标进行安装。



您可以通过转到设置来控制uBO。安装uBO后，“附加组件管理器”按钮将出现在Firefox的“每晚预览”菜单中。在不久的将来，您可以在Firefox Preview Nightly的“推荐”部分中期待更多可用于安装的扩展。

自2019年夏季以来，Mozilla一直在尝试将下一代Firefox for Android与现有版本的功能进行比较。因为Firefox Preview缺少的最重要的功能之一就是扩展支持。

当Mozilla在使用新的浏览器时，适用于Android的稳定版本Firefox已永久停留在版本68上，并且仅接收现货更新。普通用户可以继续使用稳定版本的Firefox，直到Firefox Preview发行为止。FirefoxPreview将在今年上半年的某个时候成为主要浏览器。

Firefox预览Nightly每天使用主分支代码在Play商店中更新两次。
