CP / M：加里·基尔达尔的兴衰

技术的历史充满了惊奇，急转弯。胜利与失败，背叛与忠诚。有时，为了成功，您只需要在正确的时间到达正确的位置即可。或接受友好的建议。



今天，我们将向您介绍一个人，如果他的生活有些运气的话，他可能会成为亿万富翁，比尔·盖茨（Bill Gates）的名人。这个人的名字叫加里·基尔达尔（Gary Kildall）。



您可能不知道这个名字，但是您很有可能看到甚至使用了它的创意-CP / M操作系统。对于现代PC用户而言，这几乎是未知的，但是一次，此操作系统已在数十种不同的机器上广泛使用，包括ZX Spectrum（及其CIS克隆），Robotron和“专有” Commodore 128（感谢Zilog Z80作为板载协处理器）。那些年中使用过运行CP / M的计算机的许多人甚至都没有想到过它的来源。







早些年



Kildall的职业生涯始于加利福尼亚的蒙特雷。他在海军研究生院教数学。他对精确科学的偏爱促使他开始编程。奇怪的是，活动如此急剧变化的原因是广告。 Kildall宣布出售基于Intel 4004的四位“微型计算机”而感到震惊。他仅用25美元购买了该设备。机器的潜力很小，最初应该将微处理器4004用于计算器，但Kildall足以沉浸在计算机中，甚至可以与处理器的创建者见面。



早在1972年，他就捍卫了计算机科学博士学位，并获得了英特尔顾问的工作。高质量的学术教育，对复杂任务的热爱和进取精神使Kildall尽快取得了显著成绩。



据目击者称，他是一个热情的计算机爱好者，因此决定教儿子如何使用LOGO编程。目前没有合适的口译员，基尔多尔亲自写下了。



在1970年代初期，Kildall开发了PL / M编程语言，即用于微机的编程语言。它基于PL / 1，在当时非常流行。假定程序的“高级”代码最初是在IBM-360上的大型机上编写的，然后转换为所需处理器的机器代码。 PL / M的更高版本已经包含在Intel的第一台计算机Intellec-4 ROM中。







在与英特尔合作多年期间，给他带来的另一个值得注意的想法是，基尔德尔将装有英特尔8008（Intellec-8）的微型计算机的``套件''中不仅包括显示器，键盘和打孔阅读器，还包括了Shugart软盘驱动器。1973年，基尔达尔与一位大学朋友一起开发并成功应用了驱动控制器。顺便说一句，根据一个版本，机智的基尔多尔（Kildall）应该归功于“软盘驱动器”一词的出现。



设计与命运CP / M



在1970年代，计算机软件是单独开发的。不难猜测，这种方法只会惹恼进取和雄心勃勃的程序员。由于进行了一些技术调整，Intellec-8的操作变得更加便捷，但是，为了将其转变为成熟的机器，需要使用操作系统。基础是先前的开发-PL / M。



Kildall将跨平台和曾经编写的软件的广泛使用放在了最前沿。而且他没有尝试与任何人竞争：那些年，微处理器主要用于控制技术，在它们上构建一台成熟的计算机的想法似乎很新且有趣。英特尔的立场似乎很奇怪：他们很高兴从Kildall购买了PL / M，而对CP / M则完全不关心。也许是因为他们没想到他们的Intellec-8会获得足够的知名度以使购买获利。英特尔以自己的方式占领了微处理器生产市场，基尔达尔决定继续开发该操作系统并独立销售。



基尔多尔的同时代人写道，预先确定他未来命运的因素之一是年龄。平均而言，他比现在作为PC时代的奠基人的许多人大10-15岁。他被淘汰出了一代人，但在“青年”中找不到志同道合的人。



1976年，他与妻子多萝西（Dorothy）共同创立了Intergalactic Digital Research公司。尽管名字这么大，但最初的工作人员只有两个人，“总部”位于加利福尼亚州的基尔道斯故居。 1977年，公司更名为Digital Research Inc.。 （DRI）。





加里·基尔德尔（Gary Kildall）和他的妻子多萝西（Dorothy）



那是个巨大的成功。 CP / M的副本发散的速度比热饼快，而Kildall的公司甚至被指控垄断。到1970年代末，最大市场参与者的微型计算机使用具有强大功能的CP / M。其中包括IMSAI 8080，North Star和Osborne。值得注意的是，同时，比尔·盖茨和保罗·艾伦的Traf-O-Data小型公司收集了道路上的交通数据，也积极使用CP / M。根据一些报道，到1980年，CP / M的销量超过25万份。







IMSAI值得一提。在那几年，该公司几乎独家向市场供应Altair的克隆。 1977年，他们从Kildall购买了CP / M许可证，这进一步促进了其在市场上的分销。 Kildall不可避免地蓬勃发展，变得富裕并发展了其操作系统。因此，正是CP / M首次引入了抢先式多任务处理，并且出现了专用的BIOS。迄今为止，Kildall已开发出许多以一种或另一种形式出现的算法和原理。





CP / M在Altair 8800上



同时，微型计算机的时代已经接近尾声。它们逐渐开始被个人计算机取代。这一切始于1976年由史蒂夫·沃兹尼亚克（Steve Wozniak）和公司发行的AppleI。在短短的三年内，个人电脑市场“膨胀”到了十亿美元，IBM终于出现了。







美国最大的技术公司之一的管理层非常了解：市场属于年轻一代，唯一的取胜方法就是为用户提供全面，快速和便捷的服务。与许多有影响力的IBM人员站在一起的盖茨（Gates）建议他们关注Gary Kildall的操作系统，因为微软当时只计划创建自己的操作系统。管理层但是，IBM和Microsoft签署了保密协议，根据该协议，Microsoft员工甚至无权在对话中提及IBM。



在文章的开头，我们写道：有时值得在正确的时间，正确的位置进行。我们将告诉您其中一个传说，它间接地解释了CP / M的“死亡”和MS-DOS的升高。正确与否，不能可靠地说。各方以自己的方式解释故事，因此接受现实更容易，而不是推测如果对方获胜，世界将如何发展。



根据传说，盖茨打电话给基尔德尔，并在阴谋低语中说，在未来的日子里，他应该期待非常重要的客人。进行非常重要的演讲。他不能直接说IBM将使他成为本世纪的要约。 NDA甚至不允许提及最终客户。盖茨真诚地相信，精巧的基尔多尔会理解那是谁。但是，显然，加里对同事的话持怀疑态度。因此，在来自IBM的客人出现在他家门口的那天，他平静地乘坐私人飞机飞行，将所有事情都交给了妻子。



Dorothy和DRI法务部会见了来访者，并尽力介绍了CP /M。他们以一种或另一种方式未能给IBM留下深刻的印象或至少没有引起IBM的兴趣。故事的结尾是基尔多尔的妻子和他的雇员也不得不签署了一份令人不愉快的保密协议，交易本身就失败了。的确，当时Kildall本人还没有16位版本的CP / M，因此他不能完全满足IBM的要求。



这是微软最好的时机。比尔·盖茨耸了耸肩，抓住了这个机会。是的，他的公司当时还没有自己的操作系统。但这并不意味着失去这样的头奖值得。比尔·盖茨（Bill Gates）部分由于其自身的功绩，部分由于其母亲的关系而获得了为第一台IBM个人计算机提供操作系统的合同。



盖茨转而求助于西雅图计算机公司，该公司发布了Kildall OS克隆。该公司的创始人Tim Paterson开发了自己的CP / M版本-86-DOS。正如他后来所说，要复制原始的CP \ M，复制官方文档中的API调用就足够了，因此他没有犯任何非法行为。该系统存在许多错误和缺陷，因此在内部语中称为Q-DOS（快速和肮脏的操作系统）。该系统在PC-DOS中被“重命名”并由IBM提出。将来，基于此，将创建所有读者都知道的MS-DOS。你可以阅读更多关于阴谋的复杂各地CP / M和比尔·盖茨在这篇文章中。



加里很生气。由于自己的粗心，他错过了攀登个人计算机世界的奥林巴斯的绝妙机会。甚至连上帝都用奥林巴斯（Olympus）祝福他-与其等待并诚实地从他那里购买16位版本的操作系统，不如微软转向其直接竞争对手。



IBM预见到版权方面的沸沸扬扬的丑闻，因此采取了明智的行动。两种操作系统均被许可与IBM PC一起使用：PC-DOS和CP / M，最终的赢家是选择市场。







当IBM开始大肆宣传他们的计算机时，基尔多的下一个愤怒就来了。根据广告中指示的价格，PC-DOS的售价为40美元，而来自官方分销商的CP / M Kildall的售价为240美元。据传言，价格相差六倍是故意企图“召回”基尔多尔在这次重要会议上没有露面的样子。



在DRI中，停滞已经开始。一段时间以来，该公司试图留下来，即使不能相提并论，也不会落后于微软两个案例。 Kildall甚至开发了自己的图形界面GEM，但是，在Windows发行前夕，Microsoft没有给他机会完全出售他的开发成果。







与IBM签订不成功的合同后，基尔多尔的生活发生了巨大变化。在接下来的十年中，这些系统并行发展并相互兼容，这就是为什么DR-DOS（自1987年起更名为CP / M）继续销售的部分原因。但是在1991年，Windows 3.1出现了，它不再与Kildall的DR-DOS兼容。在短短一年内，销售额下滑了一个数量级。 1991年，Kildall出售了他在Novell的公司，希望借此拒绝微软的支持。但是，此尝试失败了：Novell DOS不能再与MS-DOS和Windows竞争。



Kildall的生活有了新的伴侣-酒精。他失去了所有的专业联系，并不再担任他主持超过5年的受欢迎的电视节目《计算机纪事》的共同主持人。



对基尔多尔生命的最后几个月知之甚少。他仍然不仅仅是一个富人。他拥有大量的跑车，房地产，但是他住了很多年的技术兴趣和激情消失了。

1994年7月，他52岁，在一个骑自行车的酒吧里打架，死于神秘的环境中。2016年，在加里的孩子们的允许下，他的部分回忆录得以出版。



后记



历史把每个人都放在了自己的位置。有人不认识，被遗忘了。有人是亿万富翁，慈善家，也是世界上最令人垂涎​​的秘密圣诞老人。



加里·基尔德尔（Gary Kildall）在制造计算机方面做了大量工作，以我们今天所了解的方式进行。我们想在此博客上谈论的众多人之一。



即使人才输掉了市场之争，您也不能轻视人才的优点。决不能因为胜利者而失败。
