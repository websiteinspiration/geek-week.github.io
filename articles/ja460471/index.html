レイヤーは、テーブル、行、グラフなどの特定のレイアウトにすることができます。この場合、各レイアウトには異なるレイアウト、つまりネストが含まれる場合があります。たとえば、画面は2つの列に分かれており、左側のフィールドには入力する必要があり、右側にはルックアップテーブルとグラフなどがあります。


画面に表示されるデータを制御するために、処理を処理するコマンドが提供されています。


class ExampleScreen extends Screen
{
    public $name = 'Example Screen';

    public function query(Model $model): array
    {
        return [
            'model' => $model
        ];
    }

    public function commandBar(): array
    {
        return [
            Link::make('-')
                ->link('http://orchid.software/ru')
                ->icon('icon-globe-alt'),
        ];
    }

    public function layout(): array
    {
        return [
            Layout::rows([
                Input::make('model.title')
                    ->type('text')
                    ->max(160)
                    ->required(),
            ])
        ];
    }
}
