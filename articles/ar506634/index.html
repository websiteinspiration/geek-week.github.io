اتحاد الوحدات في Webpack 5 ، وهو مكون إضافي لتبادل الوحدات بين تطبيقات جافا سكريبت والوصف والمثال

في الإصدار الخامس من أداة إنشاء Webpack ، ظهرت مجموعة من المكونات الإضافية لتبادل الوحدات النمطية بين تطبيقات جافا سكريبت.

هذه المقالة وصف موجز ومثال يستند إلى تطبيقين صغيرين مبنيين على إطار عمل ReactJS.

يسمح المكون الإضافي Module Federation للتطبيق بتصدير وحدة نمطية واحدة أو أكثر إلى ملف JS منفصل. طريقة رائعة لبناء تطبيق للواجهة المصغرة. يمكن لتطبيقات الجهات الخارجية استيراد وحدات جاهزة ، على سبيل المثال ، مكونات التفاعل. علاوة على ذلك ، فإن استيراد تبعيات Webpack يتولى. الفرق من NPM هو أنها تستورد إلى وقت التشغيل.



يشير التطبيق المصدر في تكوين حزمة الويب بشكل صريح إلى:

  • وحدات للتصدير
  • اسم ملف التصدير ، مثل export.js
  • التبعيات التي لا تحتاج إلى تصديرها على سبيل المثال export.js

في HTML الخاص بتطبيق المستلم ، يتم استيراد ملف JS الخاص بالتطبيق المصدر كنص JS عادي: 

<script src="http://source-app.com/export.js">

وفي تكوين webpack ، أي الوحدات المأخوذة من الملف.

هكذا تبدو


يجب أن يسجل التطبيق المصدر عملية التصدير في إعدادات webpack.config.js.
في إعدادات المكون الإضافي ، حدد اسم الحاوية التي تم تصديرها ، والوحدات التي سيتم تضمينها فيها وكيف سيتم استدعاء الملف.

مثال التكوين 
const HtmlWebpackPlugin = require('html-webpack-plugin');
const ModuleFederationPlugin = require('webpack/lib/container/ModuleFederationPlugin');
 
module.exports = {
 ...
 plugins: [
   new ModuleFederationPlugin({
     name: 'home',
     library: { type: 'var', name: 'home' },
     filename: 'export.js',
     exposes: {
       ProductCarousel: './src/ProductCarousel'
     },
     shared: ['react', 'react-dom', '@material-ui/core', '@material-ui/icons']
   }),
 ]
};

تطبيق المستلم

  • في javascript.html ، يستورد ملف js للتطبيق المصدر

    مثال 
    <!DOCTYPE html>
<html lang="en">
 <head>
   <script src="http://source-app.com/export.js"></script>

  • في webpack.config.js ، في إعدادات المكون الإضافي ، حدد اسم الحاوية التي تحتاجها ، سيجدها المكون الإضافي في ملف JS

    مثال التكوين 
    const HtmlWebpackPlugin = require('html-webpack-plugin');
const ModuleFederationPlugin = require('webpack/lib/container/ModuleFederationPlugin');
 
module.exports = {
...
 plugins: [
   new ModuleFederationPlugin({
     remotes: {home: 'home' } // 
   }),
 ]
};

  • في JSX ، قم باستيراد الوحدة النمطية من الحاوية المطلوبة ، كما لو كانت وحدة محلية عادية

    مثال التعليمات البرمجية 
    import React from 'react';
import ProductCarousel from 'home/ProductCarousel';
 
function App() {
 return (
     <ProductCarousel />
 );
}
 
export default App;
