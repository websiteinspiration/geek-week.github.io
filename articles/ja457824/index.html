感染性ストレス：犬と飼い主の例におけるコルチゾールレベルの種間同期



人間は社会的な存在です。単一の個人がどのように分離または分離しようとするかに関係なく、他の人々は何らかの方法で彼に影響を与えるでしょう、彼ら自身はそうしたくないかもしれません。この現象は、種内の双方向の精神感情的な反応と呼ばれています。この長い定義における重要な言葉は「種内」です。これは、同様の反応が一群の人々だけでなく、鳥の群れ、ライオンの誇り、さらには羊の群れにも見られることを意味します。長い間密接に接触している生物は、何らかの形で相互に影響を及ぼします。これには、精神感情的な状態の側面も含まれます。今日私たちはあなたとこの研究について話します、著者によると、その結果は種間の双方向の心理的感情的反応の最初の重要な証拠となりました。この現象はどの程度正確に現れますか、誰が研究で実験的になり、科学者は彼らの発見をどのようにサポートしますか？これは研究グループの報告から学ぶ。行け。

研究の基礎


人が他の生き物から際立っている特徴を尋ねると、多くの人は主に知性を呼びますが、これは本当です。他の人たちは社会性と感情性を追加しますが、これらのエピソードは人々だけに適用されません。グループで暮らす生き物は社会的なものです。感情に関しては、人はどんなミーアキャットよりも幅広い感情と表現方法を持つことができ、持っていますが、感情はその独特の特徴ではありません。

個人の精神感情的な状態は、彼の環境、特に彼を取り巻く社会と非常に関連があります。科学者はこの伝染性の感情を呼びます。その中で最も強力な「感染」はストレスです。たとえば、教師が一定のストレスを書いている生徒のレベルが増加していることがわかりましたコルチゾール*。
コルチゾール*は、ストレスの多い状況に対する身体の反応に関与する、生物学的に活性なグルココルチコイドホルモンです。

プレーリーハタネズミ（Microtus ochrogaster）でも同様の画像を観察できます。これは、以前にストレッサー*に曝されたパートナーとのストレスレベルの相関関係を示しています。
ストレッサー* -ストレスを引き起こすあらゆる要因。
しかし、これらはストレスの種内相関の例です。あまり研究されていませんが、種間も存在します。これを修正するために、科学者たちは飼い犬と飼い主のストレスレベルを研究することを提案しました。前の金曜日に議論したように、人と犬は何千年もの間共存しています。どちらも社会的存在であるので、このような関係の中で精神感情的な関係が生じることが予想されます。

科学者たちは、人と彼の4本足の友人の間で、ストレスホルモン（コルチゾール）の種間長期同期が発生する可能性があると主張し、そのレベルは髪の毛で確認できます。これらの材料は最近の期間を説明する結果を与え、髪はあなたが長期的なホルモンの背景を評価することを可能にするので、彼らは血液と唾液を捨てることに決めました。このホルモンの濃度の遡及的カレンダーを取得できるかどうかを検討した結果、髪がどんなに些細な音に聞こえても成長し、コルチゾールはそれらに痕跡を残します。同様の分析は、以前に母親とその子供たちが参加してすでに実施されており、そのコルチゾールレベルは結果に従って同期されました。

犬と同様に、人とペットとのコミュニケーションにおいて重要なのは、訓練と階層の確立です。犬が訓練されていない場合は、いたずらをすることができます（たとえば、好きな花に好きなように水をやらない、または通り過ぎる人に吠えて投げるなど）。規律を達成するために、犬は飼い主がリーダー、つまりパックの支配的な個人であることを理解する必要があります。犬を訓練し、訓練するプロセスは、彼女と人との間の感情的なつながりをもたらすだけでなく、双方向で各ダイアドの行動に影響を及ぼします*。
Dyad * -階層的な依存関係がある共生。ダイアッドは、相互に接続された2つのオブジェクトの組み合わせ、つまりペアです。
犬と人間のコルチゾールレベルに関する研究は、宿主の性格特性や、2系統の犬と宿主の性別の組み合わせにも依存します。


研究参加者：ボーダーコリー（左）とシェルティー（右）。

この研究では、科学者たちは、マンドッグダイオードの例を使用して、種間精神感情同期の問題をより詳細に研究することを決定しました。これを行うために、58人の人々とそのペット（つまり、58組）が被験者として招待されました。科学者は、犬の体の状態に関するデータを送信する特別な首輪を通して、各犬の身体活動を観察しました。生体材料（髪）の摂取間の変化を分析するために、犬と宿主のコルチゾールのレベルを夏と冬の月に測定しました。調査には2種類の犬が参加しました。シェルティー（33個体：女性18匹、男性15匹）とボーダーコリー（25個体：雌13匹、男性12匹）で、通常のペットとさまざまな競技に定期的に参加しています。すべての犬の飼い主は女性で、尾状核の平均年齢は4.7±0歳でした。通常のペットは4年間、競合他社は4.7±0.7。犬の飼い主の平均年齢は46.3±1.7歳でした。

研究成果


犬に対する人間の影響を分析するために、ペットの性別、品種、ライフスタイルを考慮して、一般化線形モデルを使用しました。

としてホストコルチゾールレベルは本当に強く両方の冬のイヌにおける同じインジケータ影響、上のグラフに見ることができる（χN = 55、2 = 13.796を、P <0.001、β= 0.027）、および夏に（N = 57、 χ 2 = 23.697、P <0.001、β= 0.235）。簡単に言えば、人間のコルチゾールの増加に伴い、犬でも同じことが起こりました。

夏には、最も顕著な相関がHCC（毛のコルチゾール濃度）と犬のライフスタイルの間であった（χ 2 = 6.268、P = 0.012、2A χ（）とHCCと犬の性別との間に2を= 5.200、P = 0.023、図2B）。

すべての犬は、性別やライフスタイルに関係なく、人間のHCCに影響を与えましたが、競技に参加している雌と犬が最も強い影響を及ぼしました。

冬には、品種が重要な役割を果たした（χ 2 = 6.451、P = 0.011）：シェルティーHCCでは、ボーダーコリー（12.069±1.203対12.905±1.417）よりも有意に高かったです。

上記の要因に加えて、他にもあります（犬の飼い主が働いているかどうか、犬の年齢、彼女が他のペットと一緒に住んでいたかどうかなど）。しかし、これらの要因は結果にほとんど影響を与えなかったので、研究者はそれらに注意を払わなかった。

しかし、科学者たちは長期的なレベルのコルチゾールと犬の活動の比率をチェックすることを決定したので、身体活動は実際に体内のコルチゾールのレベルに影響を与える可能性があります。 4本足の状態の監視は、特別な首輪を使って1週間行われ、そこから科学者が3人の労働者と1日の休暇を分析に割り当てました。

犬が中程度および高い身体活動の状態で過ごした時間は、HCC犬と相関していました。夏の肝細胞癌（N = 44、r = -0.213、P = 0.165）または冬の肝細胞癌（N = 43、r = -0.239、P = 0.122）との有意な相関関係は見つかりませんでした。

科学者がコルチゾールのレベルに影響を与えるための最も奇妙な要因は、キャラクター、つまり犬とその飼い主の個人的な特性でした。調査の参加者は、DPQ（犬の性格に関するアンケート-犬の性質に関する質問票）とビッグファイブインベントリ（人の個人的な資質に関する質問票）の2つのアンケートにそれぞれ回答しました。

結局のところ、宿主の性質はペットのコルチゾールのレベルに大きく影響します。ヒトでの神経症レベルの増加は、犬のHCCにマイナスの効果を持っていたので（夏：χ 2= 7.951、P = 0.005、β= -0.364; 冬：χ 2 = 4.919、P = 0.027、β= -0.026）。

最後に、サンプルは意図的に夏と冬に採取されたため、科学者は季節自体がコルチゾールのレベルにどのように影響するかを確認しました。冬には、ストレスホルモンのレベルが夏よりもはるかに高いことが判明しました。

季節とライフスタイルの関係を分析した結果、飼い犬は冬にコルチゾール値が高く、夏は競技に参加していることがわかりました（4A）。性差について話すと、冬と夏の両方で女性のHCCが高かった（4B）。

エピローグ


同じ種の2人以上の個人の精神感情状態の関係は、長い間一緒に暮らしており、否定されていません。コルチゾールの種間同期については、特に驚くべき発見はありません。この研究は発見を目的としたものではなく、人と犬は社会種の代表として相互に強い影響力を持つという前述の理論を確認する必要がありました。

この研究を実施したリンシェーピング大学（スウェーデン）の科学者は、犬は飼い主の感情的な状態の予測であると信じています。つまり、人は、コルチゾールレベルの長期的な同期という点で、犬に大きな影響を与えます。さらに、彼らはこれまでそのような研究は行われていないと主張している。

この労働の結果は私たちを驚かせましたか、それは私たちにとって驚きでしたか？はいよりもないでしょう。しかし、「飼い慣らした人には私たちが責任を負う」というフレーズに対する科学的サポートを再度受けました。
