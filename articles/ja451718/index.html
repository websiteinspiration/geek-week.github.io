音声アシスタントを作成

良い一日。最近、音声アシスタントを作ることに興奮しました。10億の問題に直面しました。しかし、それでも私は私の質問を決めました。

はじめに、Visual Studioをダウンロードしてください。

次に、3つのMicrosoftパッケージをダウンロードする必要があります

  1. https://www.microsoft.com/en-us/download/details.aspx?id=3971
  2. https://www.microsoft.com/en-us/download/details.aspx?id=24003
  3. https://www.microsoft.com/en-us/download/details.aspx?id=24974

ダウンロードするときは、x86プラットフォームを選択してください（それを使うとより簡単で、バグがありません）。

今。言語を選択するときは、末尾がTELEのいずれかを選択してください（このMicrosoft Speech Platform-Server Runtime Languagesをダウンロードするとき）、これは必須です。



言語パックのインストールの最後には何も起こりません。「終了」または「終了」ボタンはありません。

「ソリューションエクスプローラー」または「プロジェクトエクスプローラー」を開き、メニューの「プロジェクト」または「プロジェクト」ボタンをクリックして、 2「または」「参照を追加」リンクを追加します。

次のウィンドウがあります。



「ブラウザ」または「参照」をクリックして、「C：\ Program Files（x86）\ Microsoft SDKs \ Speech \ v11.0 \ Assembly」フォルダに移動すると、DLLライブラリが1つあります。それを選択します。

アシスタントの声を出してみましょう。新しい機能を語らせます。この関数はテキストをキャッチしてそれを読み上げます。

private void speak(string text)
        {
            SpeechSynthesizer speaker = new SpeechSynthesizer();
            speaker.Rate = 1;
            speaker.Volume = 100;
            speaker.Speak(text);
        }
