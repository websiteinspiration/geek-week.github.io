机场支持：关于他们在飞机上的工作的重要文章


如何将飞机装在信封中

你好！
曾几何时，您要求继续谈论乘客看不见的航空服务。在机场支持人员的宝贵帮助下飞艇决定总体上描述工作的主要类型。例如，为什么我们需要经常修剪草？

监管文件中详细描述了机场飞行支持规则。他们中的大多数仍然是90年代初。更现代的修订对标准的影响不是很大。尽管技术得到了发展，但多年来，仍然有很多事情得以解决，并以鲜血写下来，请原谅我的悲痛。仍然是机场的主要工具之一-头，腿，眼睛。

建立机场支持规则的一个非常重要的里程碑是灾难涂154飞机于1984年在鄂木斯克机场。然后，在能见度差的情况下，当清洁机在跑道上运行时（没有无线电窃听站且没有闪烁的信标），飞机获得了降落的许可。他们中的两个有每个7.5吨的煤油罐。178人死亡。然后制定了在跑道上工作时的基本规则，对执行工作的汽车的要求，维护无线电通信以及提供航班的服务之间的交互作用。但是人为因素仍然是事故的主要原因，2014年伏努科沃的 Falcon 50 灾难仅证实了这一刻薄的事实。

秋冬季节在机场


在秋季和冬季（与其余时间一样），在机场设有飞行专用道（这些是跑道和末端制动专用道，以使侧面的滚动更少），飞机场是所有车道，滑行道，围裙和不同平台，路缘的地方从人造草皮到地面的过渡（它们经过了加固，因此不会因地面喷流侵蚀而进入发动机），停机坪是乘客上落行李，装卸行李，邮件和货物以及进行其他类型服务的地方。还有一个CTA，它也是一个控制点：飞机场在地图上的坐标。通常，在单车道机场，CTA位于地带的几何中心。

任务是保持飞机场在飞机场的持续准备状态。在冬季，机场服务部门可清理条纹，滑行道，围裙和机场内道路，无线电信标着陆系统区域的冰雪。在12小时班次中，至少要检查四次跑道；夏季和冬季，每次班次中至少要检查一次跑道上的摩擦系数（KSS）。当天气条件变化时，应塔的要求进行额外的检查和测量（那里有ATM控制器）。

跑道的检查由机场服务的值班主管乘汽车进行。他估计在跑道，滑行道，围裙上：

  • 附着系数
  • 降水的存在，类型和厚度；
  • 表面清洁的条件和质量；
  • 白天和便携式标记的状态和可见性。

在飞机场的计划部分：

  • 清除积雪的LP尺寸；
  • 药品的清洁部分与原雪配对的斜率值
  • 土壤密度和表面均匀度。

根据检查结果，轮班负责人向HSE管制员报告跑道上的附着系数，并以以下形式在机场状态日志中进行记录（日志的形式由联邦航空法规确定）：
“检查跑道，滑行道，围裙。跑道nnL / nnR在干燥积雪最多3毫米的地方，Ksts为0.4 / 0.40 / 0.40。跑道nnL / nnR的下雪高度为3毫米，Ksts为0.39 / 0.39 / 0.39。围裙是下雪的，有时很滑。该机场已准备好飞行。”


在日记中，对于每一个这样的条目，都将签署机场服务的轮班主管，机场高级航空主管和ATM的运营总监。

机场准备接收飞机的主要参数是跑道上涂层的附着系数，该系数应至少为0.3。粘附系数是无量纲的量，表示两个物体之间的粘附力与将这两个物体彼此压向的法向压力之比，或者表示在各种条件下飞机气动装置与跑道表面之间的粘附特性。内聚系数是在跑道的第三条上测量的，首先在跑道轴线的一侧上测量，然后在第二条上。将三分之二的测量值减少到算术平均值，然后输入机场状态的日志中。气象条件越差，粘附系数的测量就越频繁。根据附着系数，降水厚度，天气预报的值，机场服务的班长确定是否需要使用防冰剂进行涂层处理和机械清洁。粘附比小于0.3时，机场关闭。

塔式调度员向飞机机组人员报告附着系数的值，机组人员进而可以独立决定着陆，尤其要考虑到侧向风。许多航空公司通过内部文件建立了其规范价值KSC，飞行员必须在其中决定着陆或进入紧急状态。

因此，它降雪了。如何决定先清洁什么？在每个机场，制定清洁程序计划（冬季维护计划），其中建立了清洁机场要素的程序。首先，清洁通常包括跑道综合设施（即，跑道，主滑行道，与HSE一致的所有或部分连接滑行道，与跑道相邻的，跑道宽度未铺设的部分，这取决于RMS的类别），CRM的准备和时区，主要围裙。第二阶段：第二条跑道综合体，如果机场有一条跑道，则其余围裙的体积相同。第三阶段：清洁靠近其末端的飞行条的土壤部分，沿跑道扩大对飞行条的土壤部分的清洁（清洁的宽度再次取决于PMC类别），清洁围裙的围裙，规划雪井，清洁通往无线电设施的入口道路和机场内道路。我们拾起尾巴，等待下一次降雪。或者，如果下雪没有停止，我们从第一阶段开始。

可以通过两种主要方式清洁涂层：机械和化学方法。防冰剂既可用于防止结冰，也可用于清除已形成的沉积物和冰层。在机场，仅允许使用经过认证的试剂。认证检查应确认防冰剂不会对飞机制造的易碎材料造成腐蚀。可以说，生态的优先级要低得多。

从机场清除积雪的技术多种多样，但可以按类型分组：

1.通用吹雪机-这大约是一个15米长的香肠，上面悬挂着一个巨大的刀片，驾驶室后面有一个刷子，尾部有一个强大的鼓风机。她的任务是在跑道边缘铲下大量积雪，用刷子扫走大的残留物，然后用吹风机吹小风。在跑道上，此类汽车与圣诞树或小重叠的猪（约0.8米）一起行驶。如果机场又大又富，那么就买了那么多汽车，这样它们就可以一次通过跑道。但并非所有机场都能负担得起。假设我们有一个A类飞机场，这意味着跑道的宽度为60米。假设我们有一台提供5.5米清洁宽度的机器（考虑到刀片的长度和刀片的工作角度。顺便说一句，清洁速度还取决于工作角度：清洁角度越大，速度越高，但是宽度越小，这意味着，更多的汽车）。总要一次清理跑道，我们至少需要12辆车。选择这种机器时，要考虑许多因素：现有机群，清洁宽度，清洁速度，刀片工作角度和在该角度下的机器速度，刀片和刷子材料，易损件（刷子和犁刀）的可用性，鼓风机功率。更重要的是，驾驶员坐在驾驶室的中央或左侧。由于在机舱中处于中心位置，因此检查更好，更方便，并且在下雪天不是第一天的时候，这样的琐事非常重要。驾驶员坐在驾驶室的中央或左侧。由于在机舱中处于中心位置，因此检查更好，更方便，并且在下雪天不是第一天的时候，这样的琐事非常重要。驾驶员坐在驾驶室的中央或左侧。由于在机舱中处于中心位置，因此检查更好，更方便，并且在下雪天不是第一天的时候，这样的琐事非常重要。

通用吹雪机的示例：Boschung喷射扫帚，OVERAASEN RS400，Vammas psb 5500，Schmidt TJS 560/630，BTsM-470（与Schmidt一起使用）。

2.在跑道两侧的一列犁式吹塑机的后面是一台旋转式吹雪机，可丢弃雪井。在跑道上，使用较大的转子，其降雪范围约为30米；在围裙上，使用较小的转子，以将雪倒入自卸车。跑道上使用的转子可以将天窗或防反冲垫从地面抬起。但是，在此之后，转子就会报废或需要长期维修。

3. 化学品分配机。当您的天气稳定，结霜，干燥，积雪干燥且疏松，风在跑道上扫雪时，机械清洁方法是不错的选择。如果雪是湿的和粘的，如果温度持续超过0°C，如果形成冰和起伏，那么如果没有抗冰剂就无法应对。跑道用液体试剂，围裙-颗粒处理。

4. 扫地机。您会在城市街道上经常看到类似的汽车：前面-犁，后面-刷子。它们用于清洁围裙，机场内道路，场所。以前，使用的是KPM-130浇水机，上面装有刷子和犁。相反，它们仍在使用中。现在，即使在梅赛德斯和博世，任何口袋上都有类似的技术。

5. 拖拉机。可以在拖拉机上悬挂铲斗，犁，刷子或特殊的犁，以清洁跑道上的灯光。一种通用的东西，在下雪的冬天高峰期必不可少。

6. 机场真空机或机场真空吸尘器。在夏天，它用于清洁平台上的灰尘和小物体。在冬季，如果配备了特殊的储罐和泵，则可用于从停机坪表面收集除冰液。

7. 平地机，推土机，装载机和自卸车。

这不是设备的完整列表，更不用说整个公园的附件了。

冰预测


为了防止跑道由于粘附系数的降低而关闭，值班主管应在跑道被冰覆盖之前用防冰剂对其进行处理。冰预报技术是在苏联开发的。进行预测时，要考虑许多参数及其值：

  • 空气温度-从1到负5°;
  • 相对湿度-86％至98％；
  • 露点不足-从0到负7°;
  • 空气与涂层表面之间的温度差为2至4°C；
  • 涂层表面和露点之间的温差为0至负2°C。

现在，这些参数已用于跑道冰雪预警系统的开发中。跑道上安装了气象站和用于人造涂层的表面状态传感器。使用该软件，可以预测跑道上结冰的可能性，这有助于及时更改机场服务，以准备不断变化的天气状况，并积极使用抗冰剂对跑道进行处理。

紧急维修


在冬天，不可能用水泥混凝土铺设沥青或修复碎裂。但是，如果形成了可能导致事故的缺陷，则必须对其进行修复。为了在低温下紧急修复人造涂层，使用了特殊的材料，这些材料在机械密封过程中无需加热即可固化，并且可以在低温下良好地工作。这样的补丁将完美地持续到夏天，然后将其替换为现有涂层的原始材料。

春夏


与往常一样，需要保持飞机场涂层的运行质量：跑道涂层的摩擦性能，保持跑道标记，MRJ，RD，围裙的可见性，并保留设计几何参数，机场要素的形状和尺寸。如果将其翻译成俄文，则意味着必须将机场的所有要素保持在可以接收和释放某种类型飞机的状态下，同时不得损坏飞机，飞行员也不应在机场迷路。这样简短的文字掩盖了大量的工作和巨大的预算。

一切都从设计开始：此时，是飞机场未来元素的物理（涂层强度）和几何特征（长度，宽度，曲率半径）的铺设。AO的任务是保护它们。

涂层修补


在冬季，我们人造表面的地方已经破旧，飞机场的土壤沉没并淹没，标记被抹去了。通常，在三月进行涂层故障排除和排水网络检查。飞机维修服务工程师双手扶着脚，步行时身穿纸制制服，绕过跑道，滑行道和围裙。图中记录了检测到的碎屑，裂纹，空腔，剥落及其尺寸。有关缺陷的信息通常会在图形编辑器中转移到方案中，以便您可以看到涂层破坏的动态及其一般状况。接下来，进行缺陷分析并为即将到来的春夏季制定维修计划。用同样的方法检查机场的土壤部分，水井，沟渠。正在制定回填，淹水，土地开垦，沟渠清理，修理现有的收集器并铺设新的收集器，更换水井。维护土壤元素是一项非常重要且并非总是简单的任务。任何对土壤的浸水都可能导致跑道或停机坪的水浸，这是降低涂层强度或破坏涂层的威胁。

整个夏天，春季和秋季的一部分维修工作是：

  • 如果涂层是水泥混凝土，则清洁和密封跑道和围裙上的接缝；
  • 沥青路面的接缝和密封缝以及裂缝；
  • 修理沥青和水泥混凝土涂料，包括大修；
  • 增加水井和土壤沉降；
  • 恢复冬季丢失的舱口；
  • 铺设新的和修复现有的收集器；
  • 维修机场内道路。

似乎一切都像在普通道路上一样，应该只允许在飞机场上使用材料，并且在该国的所有道路上的工作质量都不能像往常一样。

维修任务每天早上发给轮班。根据工作量的大小，可以由机场的机场服务部门（有必要的专家和工人）来进行工作，或者如果承包商的工作量不允许他们自己进行，则可以由承包商进行。最常见的是，机场有沥青摊铺机，铣刨机，溜冰场，以及诸如缝刀，沥青锅炉，缝缝机，压缩机和振动板之类的小物件。

为了维修，将带关闭。在高强度的机场，这并不容易，因为您需要更改关闭时段的时间表，以免造成大量的延误并使调度员超负荷工作，或者仅在夜间工作。近年来，出现了一些材料，可让您在一夜之间和早上更换带钢中的炉子，以使带钢恢复运行。是的，这非常昂贵，但是由于维修而更改时间表会变得更加昂贵。在停机坪上，维修期间比较容易，停车位或滑行路线的一部分被阻塞，以免在工作期间损坏飞机。

在秋天重复检查。夏季形成的缺陷得到修复。确定了未来的夏季维修，土地开垦和集热器铺设的需求。这些信息是明年预算的基础。

正如工作人员自己所说：
“第一次站在一块超过三公里的水泥画布上，您会感到异常而宽敞。然后，您习惯了它，而没有抬起头，眼睛被埋在厚板中，您会在涂料中寻找缺陷。”

打标


围裙上有红，黄，白线。跑道-白色。涂层标签要求由《联邦航空条例》制定。您还可以参考国际民航组织的文件（《国际民用航空公约》附录I，第I卷），但是国际民航组织的要求与俄罗斯的要求略有不同，但是您可以了解外国机场的标记。在标记器中，一切都受到控制-从线条的粗细到线条的形状和颜色。

黄线表示飞机的滑行路线，红线表示飞机的停车（维护），平台上的白线表示特种车辆的路线。所有跑道标记均为白色。为了使混凝土表面上的黄色滑行线看起来更具对比度，将它们边框化为黑色。

以前，在日常生活中，停车位仅以八边形的形式指定，称为信封，其中仅安装了一架飞机：


1-飞机滑行轴；
2-飞机停车位；
3-飞机服务区的轮廓；
4-特种车辆的行驶方式；
5-特殊车辆的T形停车标志；
6-飞机停下来的地方的T形标志；
7-特种车辆行驶路线的分割轴；
8-特殊车辆进出许可标志。

在外国机场，停车场早已被组合成服务区域，您可以在其中放置一架宽体飞机或两个窄体飞机（两个-宽，四个-窄等）。在俄罗斯联邦，这种对齐方式直到2015年才被合法化，尽管在某些机场早些时候就使用过。
