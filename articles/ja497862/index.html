[]に再割り当てして配列のすべての要素を削除してみませんか？


多くの場合、オプションとして配列のすべての要素を削除したい場合があります。オプションとして、タスクのリストがあり、すべてのタスクを同時に削除したい場合があります。


次の要素のリストを検討してください。


let items = ["tea", "coffee", "milk"];

配列からすべての要素を削除するには、その値を空の配列に設定します


items = [];

これは正常に機能し、この例は常に使用されていることがわかります。しかし、これには問題があります...



次のコードを見てみましょう。


let items = ["tea", "coffee", "milk"];
let copy = items;

ここでは、コピー変数を作成し、要素の配列への参照を割り当てています。そのため、実際には、コピーは項目の配列を指します。


個々の値をチェックしてこれを確認します


console.log(items); // ["tea", "coffee", "milk"]
console.log(copy); // ["tea", "coffee", "milk"]

すべてが期待どおりです。


これで、要素のいずれかを変更すると、元の配列自体が変更されます。これは、単なるリンクであり、正確なコピーではないためです。


copy[2] = "oranges";
console.log(items); // ["tea", "coffee", "oranges"]
console.log(copy); // ["tea", "coffee", "oranges"]

しかし、items配列のすべての要素を削除したい場合は、通常、次のようにします。


items = [];

今、私たちはの値をチェックするとアイテムをしてコピーし、再び


console.log(items); // []
console.log(copy); // ["tea", "coffee", "oranges"]
