チックタックトー：コンテンツサイクル

この記事では、同期サーバーの例を使用して、C ++ Boost.Beastライブラリを使用したバックエンドの実装を検討します。前の記事と同じ機能-バックエンドから0から8までの乱数を取得します。Beastのコンテナを上げるのはFlaskと同じくらい難しいことではないことがわかりました。主なことはすぐに適切な例を見つけることです。ここでは、プロジェクトにDockerfileを使用しました。



写真は、注目を集めるだけでなく、C ++を知らない人の気分やモチベーションを高めるために、このレポートのプレゼンテーションから取られました。

// Return a random number
template<
    class Body, class Allocator,
    class Send>
void
handle_number_request(
    beast::string_view doc_root,
    http::request<Body, http::basic_fields<Allocator>>&& req,
    Send&& send)
{
    http::response<http::string_body> res;
    res.version(11);   // HTTP/1.1
    res.result(http::status::ok);
    res.set(http::field::server, "Beast");
    res.body() = std::to_string(rand() % 9);
    res.prepare_payload();
    return send(std::move(res));
}
