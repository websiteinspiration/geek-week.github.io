蟒蛇和啤酒-摇匀，但不要混合





哈Ha！我决定描述一个小型的自动化系统来制造出精美的饮料。为了使啤酒美味，时间和温度，温度和时间很重要。而且，它们与组成麦芽汁的成分一样必要。要获得最终产品，需要14至20个步骤，并需要不同的温度和冲泡时间。对于任何对这个神秘过程感兴趣的人，我请教。



对于三种类型，我们将需要两个容量不同的大桶：



  • 麦芽酒
  • 麦汁
  • 比尔森


每种类型在时间和温度上都会经历相应的不同步骤。当然，可以使用说明和秒表，但这非常不方便。因此，我们将编译一个csv表，其中将存储以下内容：步骤编号，体积，目标温度，时间。



接下来，将rasbpberry pi设为零，并在其上连接两个DS18B20温度传感器。

每个传感器都可以让您跟踪相应槽中的温度。为了自动计时，请在树莓上安装adafruit 128x64 oled，在显示器上将显示所有必要的信息。

按钮，我们可以控制表格中各个步骤的通过，以及启动计时器。



计时器代码 
def timer_code(seconds, t_counter = 2, restart = False):
    time.sleep(0.023)
    seconds -= time.time() - t_counter
    t_counter = time.time()
    minutes = trunc(seconds / 60.0)
    return seconds, minutes, t_counter
