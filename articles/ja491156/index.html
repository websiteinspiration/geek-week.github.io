測定可能な共感：脳MRIの共感を予測する



ロボットは交響曲を書くことができますか？技術的な観点から、それは可能です。しかし、彼は彼女に彼女に感情的な色を与えることができますか？この問題はまだ議論の余地があります。しかし、発達した脳、直立した姿勢などに加えて、人の主な際立った特徴の1つは、さまざまな感情の存在です。このスキルは本質的にユニークではありませんが、人間では、惑星に住む他の生物と区別するのに十分に発達しています。人を社会的存在であると考えると、多くの感情は何らかの形で個人と他の個人との相互作用、または社会全体との相互作用に関連しています。これらの感情の1つは、共感（または共感）です。

しかし、これは抽象的な概念なので、共感を測定する方法は？彼らの作品がすべての感情の源としての心と魂を指し示している歌詞のポインタがどのように関係なく、それらは脳によって生まれます。そして、脳の働きを分析することができます。カリフォルニア大学（米国）の科学者は、安静時の脳MRIを使用して人の共感の程度を決定できるだけでなく、調査結果を予測することもできるという調査を実施しました。共感の形成に関与している脳の領域はどれですか。それらは個別に作用しますか、それとも活用に作用しますか。この研究は医学にどのように役立ちますか？これについては科学者の報告から学びます。行け。

研究の基礎


共感は、他の人の感情的な状態に対する意識的な共感の形で現れるため、人の最も利他的な感情の1つと呼ぶことができます（理論的には、動物やロボットも、もちろん、感情を示す場合は追加できます）。

進化論の観点から、そのような共感を説明するいくつかの理論があります。共感は、他者の世話を通じて向社会的および協力的な行動を促進し、周囲の個人の内部状態、行動、および意図を予測するために必要であると考えられています。

言い換えると、共感できる人は、反応を形成する脳の部分の活性化につながるさまざまな刺激（視覚、音など）を通じて他の人の状態に関する情報を受け取ります-共感。

共感が機能するためには、私たちの脳が対談者の感情に反応する必要があります。場合によっては、前頭前野に損傷があると、無意識の反射的な行動の模倣が発生することがあります（この研究では、科学者はこのケースを「共鳴」と呼んでいます）。

神経学の観点から見ると、この行動の最もありそうな基礎は、いわゆる「神経共鳴」です。これは、嫌悪感、体性感覚、感情、運動行動の知覚と感情の集合的な神経表現の現象です。

当然ながら、共感と社会化の度合いに影響を与える追加の要素が、個人の愛着、社会的地位、特定の社会的グループのメンバーシップ、信頼度など、神経共鳴に追加されます。文脈的情報と意識的評価が、前頭葉と側頭葉によって制御される行動と意思決定の情動、体性感覚、運動プロセスに関連付けられている場合、感情反応はトップダウンベースで機能すると考えられています。側頭葉-頭頂結節（TPJ）、背内側前頭前皮質（DMPFC）および背外側前頭前皮質（DLPFC）。

ニューロンの活動を監視するこれらのシステムは、意識的評価のプロセスおよび共感または精神化の論理的形態に関連するシステムと大部分が重複しています。おそらく、制御メカニズムは抑制的です。これは、DMPFCとDLPFCの破壊的な神経調節が、向社会的行動に対するこれらの脳領域の抑制効果の低下を引き起こしたという研究によって確認されました。

科学者は共鳴と制御のプロセスが密接な共役で発生することを示唆しているため、認知機能の中​​でそれらを分離することは非常に困難です。たとえば、体性運動と情動情報処理が、他の人々の内部状態、信念、意図の評価に貢献していることが明らかになりました。

言い換えると、それぞれが特定の機能を担う脳の別々の部分がクラスターで働き、対話者の状態（この場合は共感の形成）に対する感情的な反応を実現します。

したがって、各人の共感の程度は、神経レベルでの共振プロセスと制御プロセスの間の安定した特徴的な相互作用に依存します。この仮説は、脳の特定の部分が（特定のタスクを実行する過程で）アクティブな状態で研究された実験によって確認されています。

この同じ研究で、科学者たちは、人の共感機能の個人差は安静時の脳の機能を評価することによって決定できると示唆しています。この理論が確認されれば、共感機能の程度を判断するための調査、アンケート、テストを行う必要はありません（特にこれらの方法では正確な結果が得られないことを考慮して）。

さらに、以前の研究ではかなり相反する結果が得られていたため、科学者は異なる年齢や性別の代表者間の共感に関して脳機能に違いがあるかどうかを再確認することを決定しました。

この研究には、18〜35歳の58人（女性30人、男性28人）が参加しました。被験者の主な選択基準は、右利きの人、以前に記録された神経障害および/または精神障害の欠如、発達障害の欠如、薬物乱用歴またはアルコール乱用歴の欠如でした。

各実験の最後に、すべての参加者が対人反応指数（IRI）のアンケートに回答しました。このアンケート手法は長い間使用されており、回答者の認知および感情状態を評価するためのツールとしての地位を確立しています。アンケートは25のステートメントで構成され、参加者は0（私をあまりよく説明していない）から5（私を非常によく説明している）までの5段階のスケールで評価します。

ステートメントは、共感の4つの理論的側面をテストするように設計されています。

  • ファンタジースケール（FS）：架空のキャラクターを知覚する傾向。
  • 共感的態度（共感的懸念-EU）：他人の苦しみに対する交感神経反応;
  • 遠近法的見方（PT）：他の人の見方をとる傾向。
  • 個人的苦痛（PD）：他の人の苦痛に対する不快な反応。

アンケートの前に、参加者の脳のMRIスキャンが穏やかな状態で行われました。脳の特定の領域（ROI-関心領域、つまり関心領域）に特別な注意が払われ、その数は185


でした。

研究成果


IRIスコアの分析は、分散分析（ANOVA）を使用して実行されました。研究された最初の3つの側面で男性と女性の間に特別な違いはありませんでした：FS（F = 2.68、p = 0.108）、EU（F = 2.59、p = 0.114）およびPT（F = 0.274、p = 0.603）。ただし、女性の個人的苦痛の程度は男性よりも有意に高かった（F = 9.79、p = 0.003）。


表2：男性と女性のIRI指標の比較。

上記のように、MRI中、脳の特定の領域に特別な注意が払われました：視覚、前頭頭頂、帯状-手術、背側、腹側、ならびに共鳴と制御の領域。


画像No. 1：共振および制御ネットワークの対象領域の位置の指定。

特定のネットワーク内の特定の脳領域の活動から共感の強さを予測する能力を研究すると、体性運動ネットワークが被験者の共感反応の程度を有意に予測することが明らかになりました（R = 0.374、p = 0.022）。

個人的苦痛は、共鳴（R = 0.236、p = 0.037）、コントロール（R = 0.22、p = 0.048）、および特異-オペラーショナルネットワーク（R = 0.242、p = 0.033）を使用して、しきい値を超えて予測されました。ただし、FDR（誤った検出率）内の複数の比較のためにデータを調整した後、これらの結果は疑わしいままです。


画像2：イントラネット活動に対する共感の予測の評価。

共感の残りの側面は、さまざまなROIのイントラネットアクティビティでは予測できませんでした。ネットワーク内のアクティビティの評価で個別に被験者の共感の程度をうまく予測できない場合は、おそらく相互接続のオプションを検討する必要があります。

予想どおり、共鳴ネットワークと制御ネットワークの関係を検討すると、共感が予測されました（R = 0.221、p = 0.0475、調整されたFDRデータ）。この事実は、安静時の人間の脳の特定の部分の共役活動が共感を特徴づける特性を示すことができるという事実からなるこの研究の基本理論を完全に確認します。したがって、MRIは追加のアンケートを必要とせずに、人の共感の強さを正常に判断できます。


画像3：相互接続性に対する共感の予測の評価。

MRI研究は、個々のネットワークが相互に連動して継続的に動作し、共感の兆候を示していることを示しています。スキャン中に被験者に特定のタスクが割り当てられなかったため、これは重要です。彼らの脳は安静でしたが、それにもかかわらず、共感の明確な兆候を示しました。

エピローグ


この研究により、科学者は、共感の原因である神経接続の動的な統合モデルをうまく実証することができました。この研究で行われた実験の主な特徴は、被験者の脳が静止していたことです。たとえば、バイオリニストがバイオリンを上手に演奏する能力が、手元にないときでも消えないことは非常に論理的です。しかし、科学界の共感に関しては、その起源と形成のメカニズムについて多くの論争がありました。この作品は、彼の共感と共感を引き起こしたはずの人の前に何もない場合でも、彼の脳の特定のニューラルネットワークが共感を形成し続けることを理解することを可能にしました。

共感に関する新しいデータに加えて、科学者たちは、活動に関係なく起こる多くの関連プロセスがあるため、活動的で穏やかな状態で脳の働きを分割することが不可能であるという追加の証拠を受け取りました。

研究者自身は、理論的だけでなく実用的なアプリケーションも自分たちの研究で目にしています。例えば、特定の患者が特定の介入にどれだけうまく応答するかを予測することができ、それによって最適な治療オプションの作成が可能になります。いずれにせよ、私たちが脳について知るほど、私たちは自分自身をよく理解します。
