魚の生成


したがって、ゲームの主要部分は魚の生成です。それはいくつかの段階で発生し
ます：1.場所に入るとき、次のタイプのラインからです：
ゴールデンカラシウス：25 250-400 [チーズ、ワーム、ウジ、トウモロコシ]魚のサイズは、最大、最小深度、最大深度、リストのパーセンテージとしてどこにありますか魚オブジェクトを取得します（正規表現で行を事前チェックすることを忘れないでください）。コードを美しくするために、文字列をキャストしてフィッシングする演算子を定義しました。

public static explicit operator Fish(FishString fs) {
      return fs.GetFishByStr();
}

2.次に、釣り竿が投げられるのを待つ必要があります。その後、タイマー（各釣り竿ごとに独自のバイトタイム）を開始し、タイマーの目盛りに従って、1000単位のサイズの魚のリストから、生息地の深度に深度が含まれる魚を選択します。釣り竿。

3.このリストから魚をランダムに選択します。設定された餌が食べられるかどうかを確認し、魚の活動の時刻がゲームの現在の時刻に対応していることも確認します。

4.魚が餌を食べることができる場合、餌を攻撃します。取り付けられたフックが降下する可能性に基づいて、降下があるかどうかと、何時以降かを計算します。魚が餌を食べることができない場合は、現在の餌（ある場合）を循環します。餌をつつく魚が近くにいないか確認し、同じことを繰り返します。
