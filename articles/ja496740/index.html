新しいMicrosoft Edgeにアップグレードする10の理由


Microsoft EdgeのコーポレートバイスプレジデントLiat Ben-Zurが執筆した記事

最後の数週間は私の記憶のどこにも似ていません。働く母親として、私は忙しいスケジュール、さまざまな方向への部分の内訳、家族と仕事のニーズのバランスをとることに慣れました。そして、一般的にそれらはホットケーキのように感じます。しかし、ここ数週間で私の日常の現実はまったく新しいレベルになりました。そして、私はこれだけではないのではないかと思います。私はチームに電話をかけ、職場で物事を宣伝しようとしています（そして、私はこの意味で幸運であると私は完全に理解しています）。さらに、私は今、子供たちのために家庭教育を行っています。まるですべてのボリュームがすべて大きくなったような感覚。より多くの電子メール、より多くの情報とニュース、より多くの叫び声（私がこれを書いている間、私の子供たちはドアのすぐ外で口論している）、より多くのビデオ通話、より多くの不安-正直に言うと、これをすべて消化することは簡単ではありません。

本日、Microsoft Edge の新しいコンシューマーサービスを含む一連の新しいコンシューマーサービスを発表  します。これには、組織を維持し、時間を節約し、データとIDを保護し、さらに多くの機能を提供するのに役立ちます。新しいMicrosoft Edgeをまだ起動していない場合は、  今すぐお試しください。

私は、ブラウザが毎日必要でなくても特に重視することのないブラウザが、以前よりも必要なツールになっていることに気づきました。COVID-19の最新ニュースを読んだり、Teamsを使用して会議を整理したり、オンラインでチームでWord文書を操作したり、子供向けのレッスンプランを作成したり、トイレットペーパーを購入したりしています。ブラウザは1日中冷えません。私たちは皆、このツールを信頼して何年も使用しています。

私のブラウザの習慣は変わっていません-しかし、世界は変わっており、それは信じられないほどです。今日、私たちはあなたがオンラインでどのような情報を信頼すべきかについて心配しています。私たちは、個人データ、私たちの身元、オンラインの子供たちの安全について心配しています。マイクロソフトでは、ブラウザの現状に疑問を投げかけています。おそらく、ウェブからの期待を高める時が来たのでしょう。

昨年、私たちはWebを再考し始め、Chromiumエンジンをベースとして、Microsoft Edgeブラウザーの徹底的な再構築を開始しました。これにより、毎日使用するサイトや拡張機能との世界クラスの速度、パフォーマンス、Web互換性を提供できると確信しています。しかし、これで十分ではないと確信しています。ブラウザーからより多くのことを期待する時が来ました-私たちのデータと私たちの家族に関する情報のより多くの制御、私たちの仕事をオンラインで整理してサーフィンの効率を高める新しい方法。

新しいMicrosoft Edgeの私のお気に入りの機能をいくつか紹介します。

1.コレクションで検索を整理する




子供のためのカリキュラムを突然作成する必要がある場合、または家での長期滞在がシェフの休眠能力を呼び覚ました場合は、コレクションが役立ちます。コレクションを使用すると、さまざまなWebサイトからさまざまなファイルに情報を収集、整理、およびエクスポートしたり、調査に戻って中断したところから続行したりすることが簡単になります。特に、さまざまなサイトや複数のデバイスの質量に関しては、これは難しい場合があります。

[コレクション]ボタンをクリックすると、ブラウザの右側のパネルが開き、ウェブページ、テキスト、画像を簡単にドラッグしてグループ化できます。別のページや別のアプリケーションを開く必要はありません。その後、コレクションをWordまたはExcelに送信できます。私はコレクションを使用して子供向けの授業計画を作成します。Webであらゆる種類の興味深いアイデアを検索し、それらを右側の[コレクション]パネルにドラッグします。終了したら、ワンクリックでコレクションを美しいWord文書にエクスポートし、子供に印刷します。コレクションをOutlookまたは別のメールサービスにコピーして共有することもできます。



本日、コレクションがモバイルフォーマットで表示されることを発表しました。これにより、すべてのデバイスでコレクションを簡単に作成、開き、同期することができます。コレクションのデスクトップバージョンは、インサイダーチャネルで既に利用可能で、間もなく安定したチャネルで提供されます。

2.垂直タブで効率的に作業する




私と同じように、多くのサイトにアクセスすると、常に数十のタブが開いています。そして、これが起こると、どれがどれであるかを理解することが難しくなります。これが原因で、紛失したり、誤ってタブを閉じたりすることがあります。それ自体、通常は必要なタブです。

本日、同時に開いている多くのサイトを簡単に見つけて管理するのに役立つ垂直タブの機能を発表しました。Microsoft Edgeは、シングルクリックでサイドのタブを管理できる唯一のブラウザーです。垂直タブは、今後数か月以内にインサイダーチャネルに表示されます。

3.スマートコピーで時間を節約




私のお気に入りの技術革新は通常、最も単純なものであり、無意識のうちに「以前にそれらなしでどのように生活しましたか？」インターネットから何かをカットアンドペーストして、美しいWebコンテンツがすぐに掻き集められる必要があるアウジアン厩舎に変わる頻度はどれくらいありますか？スマートコピーにより、リッチなWeb形式を維持しながら、Webコンテンツを簡単に選択、コピー、貼り付けできます。カーソルで任意の領域またはコンテンツを選択できます。挿入すると、画像やリンクなどの書式が保持されます。とてもシンプルに聞こえますが、私を信じて、それは人生を劇的に変えます。来月インサイダーチャネルでスマートコピーを開始する予定です。

4.追跡防止広告管理




私は率直になります-私は買い物が大好きです。そのため、はい、パーソナライズされた広告が役立つ場合があります。他では見つけられなかったかもしれないさまざまな新製品をオンラインで見つけました。しかし、他の多くの人、特に私の子供たちにとっては、邪魔になったり不気味に思われたりした後にWebをローミングするパーソナライズされた広告があります。エッジトラッキング防止機能により、直接アクセスしないサードパーティのWebサイトによるトラッキングから保護されます。

これにより、表示するものとしないものをより詳細に制御できます。モバイルデバイスまたはデスクトップコンピューターでは、基本、バランス、または厳密の3つの設定のいずれかを選択できます。これに応じて、Microsoft Edgeはブロックするサードパーティのトラッカーを調整します。サーフィン中は、バランスモード（デフォルト）を使用します。私の子供たちのブラウザでは、Strictモードを設定しました。追跡を防止すると、誰がオンラインで追跡しているかを簡単に確認および管理できます。この機能はすでに利用可能です。

5.パスワードモニターを使用して、ダークネットで侵害されたパスワードについて調べます




毎年、何百万ものユーザー名とパスワードがダークネットで販売されており、データ漏洩のために公開されています。

今日、Microsoft Edgeのパスワードモニターを発表します。これは、ハッカーからオンラインアカウントを保護するのに役立ちます。含まれているパスワードモニターは、オートコンプリートの個人データがダークネットで見つかった場合に通知します。 Microsoft Edgeがログインパスワードバンドルとの一致を検出した場合は、ブラウザー内から通知が送信され、追加のアクションが提案されます。設定パネルでは、漏えいしたすべてのデータのリストを表示し、対応するWebサイトにアクセスしてパスワードを変更できます。また、パスワードが変更されたら、新しいデータをオートコンプリートに保存し、Microsoft Edgeとパスワードモニターが監視していることを認識して、静かにサーフィンを続行します。繰り返しになりますが、シンプルで初歩的な解決策ですが、おかげで私はWebサーフィンがはるかに簡単になりました。別の問題は少ないです。パスワードモニターは、今後数か月以内にインサイダービルドに登場します。

6. InPrivateの改善により、真のオンラインブラウジングと検索プライバシーを維持する




オンラインで追加のプライバシーが必要な場合があります。たとえば、デバイスを家族の他のメンバーと共有する場合、誕生日プレゼントを選択していることを彼らに見せたくありません-サプライズを保つことが重要です！このような場合、セッションの終了時に履歴、Cookie、およびサイトデータが自動的に削除されるため、InPrivateモードを使用することを好みます。InPrivateモードでサーフィンしている間、私の検索履歴は私や私のアカウントに関連付けられていません。InPrivateサーフィンが利用可能になりました。BingのInPrivate検索は既にインサイダーチャネルで利用可能で、間もなく安定したチャネルに表示されます。

7.イマーシブリーダーを使用すると、誰もがWebを学習し、その恩恵を受けることができます。




読書はインターネット上で最も一般的な活動の1つですが、失読症やその他の読書障害はそれを困難にする可能性があります。Microsoft Edgeに組み込まれたイマーシブリーダーは、オンラインでの読書を簡素化し、誰でもアクセスできるようにします。イマーシブリーダーのアイコンをクリックすると、すべての注意散漫が画面から消え、ページがシンプルな外観になり、焦点を合わせるのに役立ちます。また、記事を読み上げる、テキストを調整するなど、さまざまな個人用設定ツールを使用することもできます。イマーシブリーダーはすでに利用可能で、シングルラインフォーカシングなどの新機能が間もなく登場します。

8. Microsoft Edgeは、4Kとドルビーオーディオで最高の視聴体験を提供します




最近では、多くの人が通常よりもオンライン映画をよく見ています。私たちの目標は、Webでコンテンツを表示するときに、最高のエンターテイメントオプションを高解像度で提供することです。Microsoft Edgeは、Windows 10でNetflixを4Kで視聴できる唯一のブラウザーです。さらに、ドルビーオーディオとドルビービジョンをサポートするWindows 10の唯一のブラウザーです。Netflixとのパートナーシップのおかげで、あなたとあなたの友達はWindows 10 PCのどこでも4K映画やテレビ番組のストリーミングを楽しむことができます  。  この機能はすでに利用可能です。

9.貢献する：Bingからの寄付




このあたりで起きていることの中で、私は心に留めて、自分の世話であれ他人の世話であれ、小さなことは非常に重要である可能性があることを子供たちに示すようにしています。本日は、Bing with GiveおよびBingのGiveモード機能をご紹介   します。これは、気になるアクティビティをサポートする新しい方法です。これは簡単なことですが、実際に影響を与える可能性があります。 Bingで検索する場合は、Microsoft Rewardsプログラム内のクエリに対してポイントを獲得できます。

報酬設定で新しいギブモード機能を有効にすることで、選択したケースに報酬ポイントを寄付できます。Bingで検索するだけで無料です。100万を超える非営利団体を選択すると、CDC財団やCOVID-19への対応に関与する他の組織を含む、本当に重要な原因をサポートできます。さらに、6月末までに、Microsoft Rewardsはすべてのユーザーからの寄付を複製して、さらに大きな影響を与える予定です。 試してみてください。Microsoft Rewardsに参加して、ギブモードに切り替えて、Bingでギブバックしてください。ギブモードは一部の国ですでに利用可能です。

10. Microsoft Edgeの準備ができています（簡単に切り替えられます）



誰もが新しいMicrosoft EdgeをMicrosoft.com/Edgeからダウンロードできます  。 Windows、Mac、iOS、Androidでご利用いただけます。 Windows 10クライアントで新しいMicrosoft Edgeをまだ受け取っていない場合は、今すぐダウンロードするか、来月の更新を待つことができます。お気に入り、パスワード、自動入力フォームの情報をすべてワンクリックでアクセスできます。自分でダウンロードするか、PCにインストールされるまで待つ必要はありません。 Microsoft Edgeを使用したことがない場合は、  ぜひお試しください。

新しい機能をリリースするときは、引き続き品質に焦点を当てます。これは、すべての機能が予備の内部ビルドでテストされ、最初に実験バージョンに表示されてから、安定した状態になることを意味します-品質レベルに満足した場合のみ。新しいEdgeをダウンロードした場合は、待つ必要があります。機能の可用性に応じてブラウザを更新します。今日それらを使用したい場合は、Microsoft Edge事前テストプログラムに参加できます。

私たちは今日の洗練されたインターネットユーザーの要求を満たすために一生懸命働きました。新しいMicrosoft Edgeを使用して、優れたパフォーマンス、データのより詳細な制御、情報の受信、吸収、および適用のより良い方法を提供できることを願っています。インターネットにもっと期待する時がきたからです。
