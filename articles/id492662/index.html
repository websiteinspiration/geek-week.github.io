Bagaimana saya pertama kali menyampaikan kuliah di universitas

Coronavirus membuat penyesuaian terhadap kehidupan siswa, tetapi sementara universitas memutuskan apa yang harus dilakukan dan bagaimana beralih ke pembelajaran jarak jauh, bagian progresif dari staf pengajar harus membawa masalah ke tangan mereka sendiri. Jadi saya memutuskan bahwa saya tidak akan menunggu almamater saya untuk mencari tahu apa yang harus dilakukan, tetapi memutuskan untuk langsung mentransfer kuliah ke mode jauh.


Hari ini adalah pelajaran pertama dan itu layak. Sekarang saya akan memberi tahu Anda bagaimana saya melakukannya.


TLDR: Banyak laptop + tablet dengan pena + Zoom sudah cukup. Namun ada nuansa.
