惑星スーパーコンピューターを使用して42番の問題を解決する


数学者はようやく3つの数値の立方体を見つけ、その合計は42です。したがって、問題が解決され、65年間も頭を悩ませました。1から100までの自然数のそれぞれを3つの立方体の合計として表すことができますか？

1954年に定式化された問題は、正確にこれに含まれます：x 3 + y 3 + z 3 = k。 Kは1から100までの各数値です。問題は、x、y、zとは何ですか？

過去数十年にわたって、最軽量のソリューションが見つかりました。 2000年に、ハーバード大学の数学者Noam Elkisは、より複雑な数値の解を見つけることを簡素化するアルゴリズムを公開しました。

今年は、最も難しい数字の2つだけが残りました：33と42。

人気の数学YouTubeチャンネルNumberphileで33番の問題に関する動画を 視聴した後、ブリストル大学の数学者Andrew Bookerは、新しいアルゴリズムを作成するように促されました。彼は、University Center for Computational Computing Researchの強力なスーパーコンピュータでそれを立ち上げ、わずか3週間で33という数値の解決策を得ました。

したがって、すべての数値の中で最も困難なものが残っています。42。これははるかに難しい作業であることが証明されたため、ブッカーは、大規模並列計算のスペシャリストであるMITの数学者の同僚であるAndrew Sutherlandの助けを借りました。

記事のタイトルからすでに明らかなように、彼らは問題をなんとか解決しました。彼らの成功を共有するために、彼らは楽しい方法を選びました：The Aperiodical、両方の数学者が密かに個人のWebページを問題の解決策に置き換え、これらのページをLife、Universe、Everything Elseと呼びました。これはDouglas Adamsの本への参照です。

もちろん、解決策を見つけるのは簡単ではありませんでした。 2人の数学者がCharity Engineに助けを求めなければなりませんでした。50 万台を超える家庭用PCの無料のコンピューティングリソースを一種の「惑星スーパーコンピューター」として使用する国際的な取り組みです。

解決策を見つけるには、100万時間以上のマシン時間がかかりましたが、数学者は依然として答えを見つけることができました。 つまり、完全な方程式は（-80538738812075974）3 + 80435758145817515 3 + 12602123297335631の形式になります。

X = -80538738812075974

Y = 80435758145817515

Z = 12602123297335631

3 =42。

「私は非常に安心しました」とブッカーは認めます。

「このゲームでは、何かを見つけることができるかどうかを確認することは不可能です。これは地震予知に少し似ています。おおよその確率のみを処理する必要があります。つまり、検索の数か月で答えが見つかるか、1世紀もかかる可能性があります。」


そして、これで終わりですか？ええと...そうでもない。これまでの答えは、1から100までの数についてのみ見つかりました。1桁、1000まで上げても、未解決の数はまだたくさんあります-114、165、390、579、627、633、732、906、921、975 3つのキューブの合計の形のソリューションをまだ待っています。
