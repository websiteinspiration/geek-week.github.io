ニューラルネットワークと人工知能とは一般に何ですか？


AIの哲学では、人工知能には強いものと弱いものの2種類があります。強力な人工知能の理論は、コンピューターが自分自身を別の人として考える能力を獲得できることを示唆しています（まあ、デトロイトビカムヒューマンで起こったことのようなもの）。弱い人工知能の理論は、そのような知性を構築することはできないか、または非常に困難であることを示唆しており、科学は生物のニューラルネットワークの働きを部分的にのみ繰り返すニューラルネットワークを作成することができます。しかし、哲学は科学には当てはまりません。そこでは何も証明できないため、これには焦点を当てません。現在、弱いニューラルネットワークは、データサイエンスで支配的な地位を占めており、広く使用されています。ビジュアルイメージの認識、商品のスマートな購入、車両のOBSシステムでさえ、AIの学習を開始することは、今では関連性を超えています。

簡略化すると、ニューラルネットワークは何をしますか？


ニューラルネットワークの概念を少し単純化すると、正しい刺激を示す「刺激-反応」の原理についてトレーニングした後、教師と一緒に学習する原理に従って教えられるニューラルネットワークは、見知らぬ人と連携できます。データ。つまり、特定の単語セットを入力ニューラルネットワークに提案した場合（たとえば、フィルム検索のレビューのセット、ポジティブとネガティブ、任意の形式、少なくともtxt、少なくともjsonの場合、問題はこのデータを処理するためのプログラムにのみあります）。ニューラルネットワークを正常に作成するには、2つのデータセットが必要です。作成したニューラルネットワークのパフォーマンスを評価できるテストセットと、データにポジティブ/ネガティブのラベルが付けられたトレーニングセットです（ビッグデータは長いため、ここでビッグデータを分類するという問題が発生します）そして退屈な職業）。ニューラルネットワーク（データの次元、その処理、および最も頻繁に使用されるアルゴリズムに応じて、多くの時間とコンピューターリソースがかかる可能性があります）をトレーニングした後、それを試すことができますある程度正確に予測するために、肯定的または否定的なレビューが彼女の意見になりました。ただし、ニューラルネットワーク（および実際の人間）には、一定の割合のエラーがあります。最適化のタスクはそれを最小限にすることですが、ニューラルネットワークの品質を評価する問題はどこにも行きません。たとえば、彼らはあなたにホットドッグの写真を与え、あなたはそれがホットドッグであると確かに言います。しかし、写真がぼやけている場合はどうなりますか？黒と白？視界不良で撮影されましたか？ここでは、あなたが人生で多くのホットドッグを準備したり食べたりしたとしても、ある程度の確率ですでに肯定することができます。



おもちゃのニューラルネットワークのプログラミングを始める


さあ、行きましょう。おもちゃのニューラルネットワークにはテストデータとトレーニングデータはありません。おもちゃのニューラルネットワークは、データ間の比率の比率を見つけようとします。

やめる。そして何がポイントなのか？同じことが、1つの単純な数式にも見られます。
間違いなく。しかし、私は今、この表現を使ってニューラルネットワークを学習するプロセスを示しています。インチとセンチメートルの間の変換係数が何であるかを見つける作業に直面していると仮定します。 1インチは何センチかかりますか？少なくとも5年生（またはそれ以前）の数学を知っている人にとって、翻訳係数が2.54であることを覚えておくのは難しくありません。

しかし、今のところ、これについてはしばらく忘れて、このパラメーターを普遍的に計算する単純なアルゴリズムを作成する必要があると想像してください。ただし、問題は、ニューラルネットワークが既製の係数値を持つ定数ではないことです。それ以外の場合は、「ライブ」トレーニングがありません。

全体として、私たちは、一連のキューブの前に座って、初めて自分の手にそれらを持ち込み、人生で最初のタレットを構築する子供の立場にいます。彼はオブジェクトの物理学がどのように機能するかをおおよそ知っているだけです;私たちのように、彼は特定の特定の比率が存在することを知っています（彼の場合、それは重力です）。それで、子供は何をしますか？

彼はダイスを取り、ランダムに置きます。同様に、私たちが持つ係数はランダムにしか推測できません（そして、実際の成人ニューラルネットワークもこれを行いますが、通常は正規分布での数値の生成によってのみ導かれます）。
ちょうどランダムに、インチとセンチメートルの結合係数（大人のニューラルネットワークのように、これを重みと呼びましょう）が、たとえば2.4であるとします。次に、最も複雑な数式を取得します。

1 * 2.4 = 2.4

すばらしい、ほとんど推測したところ、結果が出ました。しかし、それは誤りであり、学習プロセスでは正常なことですが、エラーがいくつかあります。フィードバックのある神経系のように、エラーになんとか対応する必要があります。ただし、最初にそのサイズを理解する必要があります。すでに述べたように、教師とのニューラルネットワークのトレーニングでは、データはまずマークアップされたデータで実行され、次にマークアップされたデータではなく類似の分類に送信されます。また、正しい結果を得るには何が必要かを知っているため、エラーを計算できます。

t_target = 2.54
t_now = 2.40
e = t_target - t_now
//  ,   0.14

今、私たちはどのように間違っているかを知っています。しかし、何をすべきか？当然、私たちのおもちゃのニューラルネットワークは、可能な限り低いデータを読み取る必要があります。そして、これは明らかになるニューラルネットワークの別の水中機能です。ほとんどの場合、トレーニング中に損失要因が発生します。最適化と呼ばれるデータサイエンスの一部は、エラーの最小化に取り組んでいます。

しかし今はそれについてではありません。間違いなく、結合係数を少しずつ変更する必要がありますが、何を変更するのでしょうか。明らかに、大きすぎる必要はありません。そうでない場合、結合係数は不正確で小さすぎないことがわかります。そうでない場合は、かなり長い時間ニューラルネットワークをトレーニングする必要があります。このステップを見つけるための100％正しいオプションはありません。ほとんどの場合、実際のニューラルネットワークのこれらのパラメーターは、いくつかの数式またはアルゴリズムに基づくよりも直感的に選択されます（これも可能です）。これで、ステップの値をランダムに選択できます（ニューラルネットワークの言語では、ステップはlearning_rateと呼ばれます）。たとえば、値0.05で十分最適に機能します。さらに、学習率でのインデントが何回発生するかについても合意する必要があります。。これらのインデントの数は、成人のニューラルネットワークと同様に、エポックと呼ばれます。このすべての知識を身に付けた今、おもちゃのニューラルネットワークプログラムを実行する小さなPythonプログラムを作成してみることができます。

import random

#    Python  random,        

inches = 40  #    40    101, 6 
centimetre = 101.6

#  ,        , learning rate  

def kid_neuro(epoch, lr, accur):
    W_coef = random.uniform(0, 2)  #     
    print("    : ", W_coef)
    for i in range(epoch):  #    
        Error = centimetre - (inches * W_coef)
        print("  ", Error)  #      
        if Error > 0:
            W_coef += lr  #    ,    

        if Error < 0:
            W_coef -= lr  #   ,    

        if Error < accur:
            print("  ", W_coef)
            return  #  ,      

epoch = int(input("epoch: "))  #      ""
lr = float(input("enter learning rate: "))  #   
accur = float(input("enter accurancy:  "))  #   ,     ,         

kid_neuro(epoch, lr, accur)  #    -
