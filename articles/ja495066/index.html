Check Point Remote Access VPN-無料のリモートアクセスセットアップコース



みなさん、こんにちは！次の新しいコースへようこそ！私が約束したように、入門コースは最後ではありませんでした。今回は、同様に重要なトピックであるリモートアクセスVPN（リモートアクセス）について説明します。このコースでは、従業員が安全にリモートアクセスできるようにするために、チェックポイントのテクノロジーにすぐに慣れることができます。コースの一部として、私たちはいつものように、理論的な部分と実際の部分を実験室の作業の形で組み合わせます。チェック・ポイントの構成のデモに加えて、リモート・ユーザーを接続するためのさまざまな方法を見ていきます。

いくつかのコメント


  • このコースは、VPNテクノロジーの完全なガイドではありません。これは、最も要求された情報の非常に短い要約です。それら。見た後でVPNの第一人者になることを期待するべきではありません。ただし、原則を理解し、Check Point VPNをすぐに使い始めるには、このコースで十分です。
  • このコースは、前のコース、Check Point Getting Startedを完了してから開始することを強くお勧めします。ここでは、アーキテクチャ、IPアドレスの構成方法、または管理サーバーへのゲートウェイの取得方法など、いくつかの基本的な事項については説明しません。私はあなたがすでにこれを知っているという仮定から進み、VPNのみに焦点を当てます。
