Gabungkan set tekstur, atau cara membuat satu set tekstur di Substance Painter

Gabungkan Set Tekstur dengan cepat dan tanpa banyak usaha ...



Selamat siang, nama saya Julia , dan hari ini saya akan memberi tahu Anda cara menggabungkan Set Tekstur ke dalam Pelukis Substance dan mengekspor satu set tekstur.Saya akan menjelaskan dengan contoh proyek saya, yang saya lakukan di Blender 2.8.

pengantar


Sebagai permulaan, beberapa kata tentang perangkat lunak. Substance Painter adalah program yang cukup kuat dan stabil yang akrab bagi banyak orang, dan mungkin semua seniman 3D dan pengembang game komputer. Apa yang bisa lebih mudah memuat model dengan pemindaian UV dan segera "tekstur" dan "panggang"? Tapi begitu di depan saya ada masalah ...

Masalah


Seperti banyak (yang bekerja dengan Substance Painter ) tahu bahwa ketika mengimpor model 3D dengan satu set warna atau tekstur ke lingkungan, model ini dibagi menjadi area yang terpisah atau "Set Tekstur" . Sebagai contoh, saya mengambil model kapak lama saya.



Texture Set adalah alat yang praktis, karena seorang seniman, ketika bekerja dengannya, dapat dengan mudah menghapus dari tampilan bagian-bagian model yang tidak ia butuhkan, yang akan dicat dengan warna berbeda. Dan semuanya akan baik-baik saja, tetapi apa yang saya dapatkan ketika saya memutuskan untuk mengekspor tekstur dari Substance Painter untuk bekerja dengan model di editor lain (misalnya, di Blender yang sama )?

Yaitu, apa yang seharusnya saya miliki, saya punya satu set tekstur untuk masing-masingSet Tekstur , yaitu 6 * 3 = 18 tekstur yang berbeda ( BaseColor, Roughness, Metallic, Normal, Height, Emissive ).

Mungkin ini akan terasa sangat nyaman bagi seseorang, tetapi jelas tidak bagi saya. Mengkustomisasi render dengan menambahkan 18 tekstur ke satu proyek kecil terlalu banyak ... Saya mencoba menyelesaikan masalah ini. Saya mencari solusi di internet, tetapi ada beberapa jawaban yang tidak jelas bagi saya. Sekarang, setelah beberapa saat, saya ingin berbicara tentang bagaimana saya menyelesaikannya sekarang.

Tahap 1 - Persiapan Model


Saat mengekspor dari editor 3D apa pun (setelah membuat poli-rendah dan UV ), kami harus menetapkan satu warna untuk semua bagian model - ini adalah hal yang paling penting untuk dilakukan pada tahap pertama. Ketika mengimpor model ini ke Substance Painter maka kita akan memiliki satu Set Tekstur dan, oleh karena itu, hanya satu tekstur yang ditetapkan saat mengekspor.



Lebih lanjut, kami bekerja dengan poly-tinggi kami (atas dasar mana mereka biasanya "memanggang" kartu yang berbeda). Tetapkan setiap objek dengan ID warna sendiri (Saya juga bekerja di Blender ), yaitu, kita cukup mewarnai mereka dalam warna berbeda yang berbeda mungkin satu sama lain.

Tahap 2 - Panggang


Setelah semuanya disiapkan, ekspor low-poly ke Substance Painter. Lebih jauh lagi, yang paling penting ... Saya tidak akan berbicara tentang apa "memanggang" dan Peta Normal , ada banyak artikel seperti itu, dan jika Anda tertarik dengan topik ini, Anda mungkin tahu apa yang saya bicarakan.
Saat memanggang di Substance Painter, saya hanya memilih 2 Peta Normal dan kartu ID . Dan sebagai objek, saya pilih poly-tinggi saya , klik Bake dan Anda selesai.

(Untuk membuat kartu “panggang” tanpa artefak, Anda perlu melakukan beberapa operasi tambahan di pengaturan: pada tab Umum di bidang Pertandinganatur nilainya ke “By Mesh Name” , pada tab ID di bidang Color Source atur nilainya menjadi “Material Color” .)



Tahap 3 - Pengaturan Proyek


Sekarang Anda perlu mengkonfigurasi proyek dengan benar, sehingga bekerja dengan nyaman dan dapat dimengerti.

  1. Buat folder ( Folder ) di lapisan dan beri nama sebagai tekstur akan disebut (Dalam kasus saya, Kayu, Cincin atau Baja ). Anda memerlukan banyak folder karena akan ada tekstur pada model Anda.
  2. Kami memilih salah satu folder dan menambahkan mask dengan pemilihan warna di folder mask , pilih Picker dalam pengaturannya dan melihat ID "panggang" pada model low-poly kami . Operasi ini harus dilakukan untuk semua bagian objek.

Sekarang folder ini adalah varian kami dari Set Tekstur , ketika bekerja dengan mana kami tidak akan menyentuh bagian lain dari model. Sekarang mengisi folder secara hierarkis (misalnya, Fill Layer ) Anda dapat melukis bagian-bagian model dalam tekstur Anda.

PS
Mungkin ini bukan satu-satunya solusi untuk jenis masalah ini, tetapi saya menemukannya sendiri dan menggunakannya dengan sukses. Saya akan senang menerima kritik dan harapan, ceritakan tentang metode Anda.
