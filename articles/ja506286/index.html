封筒に飛行機を詰める方法

こんにちは！
むかしむかし、乗客には見えない航空サービスについて話し続けることを求めました。空港サポート担当者の貴重な支援を受けてエアロドロマー一般的に仕事の主なタイプを説明することにしました。たとえば、なぜ草を頻繁に刈る必要があるのでしょうか。

飛行場の飛行支援の規則は、規制文書に詳しく説明されています。それらのほとんどはまだ90年代の初めです。より近代的な改正により、基準はあまり変更されていません。テクノロジーの発展にもかかわらず、長年にわたって多くのことがうまくいき、血で書かれました。まだエアドロームの主要なツールの1つ-頭、脚、目。

飛行場サポートのルールの確立における非常に重要なマイルストーンは災害でした1984年にオムスク空港でTu-154航空機。次に、視界が悪い状況で、航空機はストリップ上でクリーニングマシンが動作しているときに着陸する許可を受け取りました（無線タッピングステーションなし、点滅するビーコンなし）。そのうち2基は、それぞれ7.5トンの灯油タンクを備えていました。178死者。次に、滑走路で作業するときの基本的なルール、作業を行う車の要件、無線通信の維持、フライトを提供するサービス間の相互作用が作成されました。しかし、人的要因は依然として事故の主な原因であり、2014年のヴヌーコボでのFalcon 50の災害 は、このハッキングされた真実を確認するだけです。

空港での秋と冬


秋と冬（他の時間と同様）には、飛行場に飛行レーンがあり（これらは滑走路と端部ブレーキングレーンであるため、側面のロールアウトが少なくなります）、飛行場はすべて、レーン、誘導路、エプロン、さまざまなプラットフォーム、縁石がある場所です。エプロンは、人工芝から地面への移行（地面のジェット侵食のためにエンジンに何も入り込まないように強化されています）、エプロンは乗客が荷物の積み降ろし、荷物の積み降ろし、郵便や貨物、およびその他の種類のサービスを行う場所です。また、コントロールポイントであるCTAもあります。これは、飛行場の地図上の座標です。原則として、シングルレーンのエアロドロームでは、CTAはストリップの幾何学的中心に配置されます。

タスクは、航空機の飛行のために飛行場の一定の準備を維持することです。冬には、飛行場サービスがストライプ、誘導路、エプロン、空港内道路、電波ビーコン着陸システムのエリアの雪と氷をきれいにします。12時間シフトの間に少なくとも4回、滑走路が検査され、シフトごとに少なくとも1回、滑走路の摩擦係数（KSS）が夏と冬の両方で測定されます。気象条件が変化したとき、およびタワーの要求に応じて、追加の検査と測定が行われます（そこにATMコントローラーがいます）。

滑走路の検査は、飛行場サービスのシフト管理者によって車で行われます。彼は滑走路、誘導路、エプロンについて推定している：

  • 接着係数;
  • 降水の存在、種類、厚さ
  • 表面洗浄の状態と品質;
  • 昼間およびポータブルマーキングの状態と可視性。

飛行場の計画された部分について：

  • 雪を取り除いたLPの寸法。
  • 医薬品の洗浄された部分と未使用の雪との共役の勾配の値;
  • 土壌密度と表面の均一性。

検査の結果に基づいて、無線によるシフトの責任者がHSEコントローラーに滑走路の付着係数を報告し、次の形式の飛行場の状態のログブック（ログブックの形式は連邦航空規制によって決定されます）に入力します。
「滑走路、誘導路、エプロンの検査。滑走路nnL / nnRは、最大3 mmの乾燥した雪、Ksts 0.4 / 0.40 / 0.40。滑走路nnL / nnRは3 mmまで雪が降り、Ksts 0.39 / 0.39 / 0.39。エプロンは雪が降り、時々滑りやすくなります。飛行場は飛行の準備ができています。


ジャーナルでは、そのようなエントリごとに、飛行場サービスのシフト責任者、飛行場の上級航空チーフ、およびATMの運用責任者が署名されています。

飛行場が航空機を受け入れる準備の主なパラメータは、滑走路上のコーティングの付着係数であり、少なくとも0.3でなければなりません。付着係数は、2つの物体間の付着力と、これら2つの物体を互いに押し付ける通常の圧力との比、またはさまざまな条件下での航空機の空気圧と滑走路表面間の付着特性を表す無次元量です。凝集係数は、滑走路の3番目、最初は滑走路の軸の一方の側、次に2番目の側で測定されます。 3分の1の測定値は算術平均に換算され、飛行場の状態のログブックに入力されます。気象状況が悪いほど、粘着係数が測定される頻度が高くなります。付着係数、降水の厚さ、天気予報の値に基づいて、飛行場サービスのシフト管理者が決定します防氷剤によるコーティング処理と機械洗浄が必要かどうか。付着率が0.3未満の場合、飛行場は閉じます。

タワーディスパッチャーは、航空機の乗員に対する付着係数の値を報告します。次に、乗員は、特に横風を考慮して、着陸を個別に決定できます。内部文書による多くの航空会社は、パイロットが着陸に行くか緊急事態に行くかを決定しなければならないそれらの規範的価値KSCを確立しています。

それで、それは降雪を充電しました。最初に何を掃除するかはどのように決定されますか？各飛行場では、飛行場要素の掃除のシーケンスが確立されている掃除シーケンス計画（冬のメンテナンス計画）が作成されます。まず第一に、クリーニングには通常、滑走路複合体（つまり、滑走路、主要誘導路、HSEと一致するすべてまたは一部の接続誘導路、滑走路に隣接するフライトストリップの未舗装部分、RMSのカテゴリに依存）、CRMおよびタイミングゾーンの準備が含まれます。メインエプロン。第2ステージ：2番目の滑走路複合体。空港に滑走路がある場合、残りのエプロンは同じ量です。第3段階：フライトストリップの端に隣接する土壌部分のクリーニング、滑走路に沿ったフライトストリップの土壌部分のクリーニングの拡大（クリーニングの幅はPMCカテゴリによって異なります）、エプロンのエプロンのクリーニング、スノーシャフトの計画、無線設備や空港内道路へのアクセス道路の清掃。尾を取って次の降雪を待ちます。または、雪が止まらない場合は、最初の段階から開始します。

コーティングは、機械的および化学的な2つの主な方法でクリーニングされます。氷結防止剤は、氷の形成を防ぐためと、すでに形成された堆積物や氷からコーティングをきれいにするための両方に使用されます。飛行場では、認定された試薬のみが許可されます。認定チェックでは、防氷剤が航空機を構成する繊細な材料の腐食を引き起こさないことを確認する必要があります。エコロジーは、はるかに低い優先順位が付いているとしましょう。

氷の予報


付着係数の低下による滑走路の閉鎖を防ぐために、シフト管理者は、滑走路が氷で覆われる前に、滑走路を着氷防止剤で処理する必要があります。氷の予測技術はソ連で開発されました。予測時には、いくつかのパラメーターとその値が考慮されます：

  • 気温-1からマイナス5°C;
  • 相対湿度-86から98％;
  • 露点不足-0からマイナス7°;
  • 空気とコーティングの表面との間の温度差は2〜4°Cです。
  • コーティング表面と露点の温度差は0〜マイナス2°Cです。

現在、これらのパラメータは滑走路の氷の警告システムの開発に使用されています。滑走路には気象ステーションと人工コーティング用の表面状態センサーが設置されています。ソフトウェアを使用して、滑走路での氷の形成の確率が予測されます。これは、変化する気象条件に備え、滑走路を予防的に抗氷試薬で処理するために、飛行場サービスを時間内に変更するのに役立ちます。

緊急修理


冬は、アスファルトを敷設したり、セメントコンクリートで切りくずを補修することはできません。ただし、事故につながるような欠陥が生じた場合は、修理が必要です。低温での人工コーティングの緊急修理には、加熱せずにメカニカルシール中に硬化し、低温で適切に機能する特殊な材料が使用されます。そのようなパッチは、既存のコーティングの元の材料で置き換えられる夏まで完全に持続します。

春夏


いつものように、飛行場コーティングの運用品質の維持が必要です：滑走路コーティングの摩擦特性、滑走路のマーキングの可視性の維持、MRD、RD、エプロン、設計の幾何学的パラメーター、飛行場要素の形状とサイズの維持。ロシア語に翻訳すると、それは飛行場のすべての要素が特定のタイプの航空機を受け入れて解放できるような状態に保たれなければならないことを意味しますが、航空機は損傷してはならず、パイロットは空港で迷子になってはいけません。このような短い文言は、莫大な量の作業と莫大な予算を隠しています。

すべては設計から始まります。この時点で、飛行場の将来の要素の物理的（コーティング強度）と幾何学的特性（長さ、幅、曲率半径）が設定されます。そして、AOの仕事はそれらを保存することです。

コーティング修理


冬の間、すり減った場所の人工表面、および飛行場の土壌部分が沈んで浸水したため、マーキングは消去されました。通常、3月に、コーティングのトラブルシューティングと排水ネットワークの検査が行われます。飛行場のサービスエンジニアは、足を手に取り、紙のスキームで武装し、滑走路、誘導路、エプロンを迂回して歩きます。検出されたチップ、クラック、キャビティ、シェル、剥離、およびそれらのサイズが図に示されています。欠陥に関する情報は通常、グラフィカルエディターのスキームに転送されるため、コーティングの破壊のダイナミクスとその一般的な状態を確認できます。次に、欠陥分析が実行され、次の春夏期間の修理計画が作成されます。同じように、飛行場の土壌セクション、井戸、溝が検査されます。埋め戻し、水浸し、土地の再生、溝の清掃、井戸を交換して、既存のコレクターを修理し、新しいコレクターを敷設します。土壌要素の維持は非常に重要であり、必ずしも簡単な作業ではありません。土壌の浸水は滑走路またはエプロンベースの浸水につながる可能性があり、これはコーティングの強度またはその破壊を低下させる脅威です。

夏の間、春と秋の修理の一部が行われます：

  • コーティングがセメントコンクリートの場合、滑走路とエプロンの縫い目の清掃とシーリング。
  • ジョイントとアスファルト舗装の亀裂と接合;
  • オーバーホールを含むアスファルトおよびセメントコンクリートコーティングの修理;
  • 井戸と土壌沈下を追加する;
  • 冬の間に失われた孵化の回復;
  • 新しいコレクターの設置と既存のコレクターの修理。
  • 空港内道路の修理。

すべてが普通の道路と同じように思われ、飛行場での使用を許可されているのは材料のみであり、仕事の質は国のすべての道路で通常のようにはなりません。

修理作業は、毎日午前中にシフトに発行されます。彼らの規模に応じて、必要な専門家と労働者がいる空港の飛行場サービス、または自分で実行できない仕事量の場合は請負業者が仕事を行います。ほとんどの場合、空港にはアスファルト舗装機、フライス盤、アイスリンク、そしてシームカッター、ビチューメンボイラー、シームフィラー、コンプレッサー、振動プレートなど、あらゆる小さなものがあります。

修理のため、ストリップは閉じています。高強度のエアロドロームでは、これは簡単ではありません。大規模な遅延を引き起こさずにディスパッチャに過負荷をかけないように、閉鎖期間のスケジュールを変更するか、夜間にのみ作業する必要があるためです。近年では、ストリップのストーブを夜間に変更したり、朝にストリップを稼働状態に戻したりできる材料が登場しています。はい、それは非常に高価ですが、いつか修理のためにスケジュールを変更すると、より高価になります。エプロンでは、修理中の方が簡単です。駐車場またはタクシーのルートの一部は、作業中に航空機に損傷を与えないようにブロックされます。

検査は秋に繰り返されます。夏の間に形成された欠陥は修復されます。将来の夏の修理、土地の再生、およびコレクターの敷設の必要性が確認されています。この情報は、来年の予算の基礎となります。
