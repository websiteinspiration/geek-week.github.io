Pengumuman Microsoft tentang ujian dan sertifikat

Selama 3 bulan terakhir, Microsoft telah membuat 3 pengumuman besar:

  1. MCSA, MCSD, MCSE sejak 2021, bersama dengan semua ujian sebelumnya, kecuali berbasis Peran baru, tidak ada lagi .
  2. Hampir semua ujian berbasis Azure Role sebelumnya sudah usang. Ini dapat dibaca dalam artikel terpisah dengan analisis
  3. Akan ada beberapa ujian baru dan, karenanya, sertifikat untuk mereka:
    • Ujian DP-300: Mengelola Basis Data Relasional di Microsoft Azure (beta) / Bersertifikasi Microsoft: Associate Administrator Database Azure
    • Ujian DA-100: Menganalisis Data dengan Microsoft Power BI (beta) / Microsoft Certified: Data Analyst Associate
    • Ujian AZ-120: Merencanakan dan Mengelola Microsoft Azure untuk Beban Kerja SAP / Bersertifikat Microsoft: Azure untuk Beban Kerja SAP Spesialisasi
    • Ujian AZ-220: Pengembang Azure IoT Microsoft / Microsoft Certified: Spesialis Azure IoT

Hal pertama yang pertama.

MCSA / MCSA / MCSE - semuanya. Itu saja.


Saya yakin bahwa keusangan MCSA / MCSA / MCSE menyebabkan reaksi yang jauh lebih besar daripada semua pengumuman lainnya. Jika tidak ada sembarang ujian baru di SQL 2019 atau Windows Server 2019, itu tidak terlalu buruk. Namun pada kenyataannya, Microsoft semua sertifikat dan ujian yang tidak ada di Azure / Dynamics365 / Office365 dikalikan dengan nol. Tentu saja, kami mempermanis pil, memastikan bahwa dalam transkrip ujian Anda, fakta kelulusan tidak akan hilang, tetapi ini tidak mengubah esensinya.

SAP biru


Semuanya rumit di sini, karena bukan teknologi Microsoft yang dikombinasikan dengan Azure i.e. ujian secara bersamaan pada infrastruktur Azure / komponen dan layanan SAP (untuk pemantauan, untuk optimasi) dan arsitektur.

Azure IoT


Ujian ini menggabungkan keterampilan pengembang IoT dalam C #, Administrator / Integrator dari solusi IoT (Penyediaan Perangkat sangat ditanyakan), pengguna Azure SaaS untuk IoT dan arsitek. Jelas bahwa IoT semakin dalam dan semakin dalam ke kehidupan kita, dan ujian diperlukan bagi mereka yang ingin memverifikasi keterampilan mereka. Secara pribadi, saya tidak yakin bahwa jumlah pengembang IOT akan besar, tetapi ujian jelas diperlukan untuk menutup ceruk yang sesuai.

Kekuatan BI


Sederhana: cara bekerja dengan Power BI. Secara pribadi, saya tidak suka nama sertifikat yang diberikan untuk ujian ini, "Microsoft Certified: Data Analyst Associate." Ada banyak alat hebat lainnya untuk analisis data, PowerBI terlalu sempit untuk nama seperti itu.

Administrator SQL Azure


Di sini semua kata itu penting - tidak ada satu pun yang bisa dibuang. Ini bukan pengembangan, tetapi penyebaran, optimalisasi, kinerja, toleransi kesalahan, dll.
