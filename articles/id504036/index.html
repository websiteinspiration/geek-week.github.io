Tajuk HTTP Tidak Terduga

Beberapa hari yang lalu saya memilih di blog Creditkarma dan memperhatikan tajuk HTTP ini:

X-hacker: If you're reading this, you should visit wpvip.com/careers and apply to join the fun, mention this header.

(X-hacker: jika Anda membaca ini, Anda harus pergi ke wpvip.com/careers dan mendaftar untuk bergabung dalam kegembiraan, sebutkan tajuk ini).

Pikiran pertama adalah: "Wow, dulu programmer memperkenalkan bug milenium untuk menghemat beberapa bit pada tanggal tersebut, dan sekarang perusahaan menerbitkan seluruh penawaran tentang bekerja di header HTTP!"

Saya menjadi sangat tertarik, dan saya melakukan riset.

Judul khusus ini tampaknya menjadi "standar" jika Anda meng-host situs di WordPress VIP , perusahaan hosting WordPress yang diselenggarakan oleh Automattic. Judul yang sama dapat ditemukan di banyak situs terkenal, seperti:

  • nypost.com
  • techcrunch.com
  • www.nbcolympics.com
  • www.thesun.co.uk
  • ribuan lagi

Pengembang situs web dan pemilik situs web dapat menonaktifkannya, tetapi jujur ​​saja, saya ragu mereka bahkan tahu bahwa tajuk ini termasuk dalam setiap paket HTTP. Dan tentu saja, pemikiran kedua segera datang - untuk memeriksa apakah perusahaan lain memiliki berita utama yang kreatif.

Hasilnya luar biasa!

Anda dapat menemukan berbagai tawaran pekerjaan di header HTTP.


Iya! Perusahaan-perusahaan paling keren di dunia tampaknya menerbitkan tawaran pekerjaan di header HTTP "standar" x-recruiting.

Berikut adalah beberapa contoh:

PayPal.me

x-recruiting: If you are reading this, maybe you should be working at PayPal instead! Check out www.paypal.com/us/webapps/mpp/paypal-jobs

Booking.com

x-recruiting: Like HTTP headers? Come write ours: careers.booking.com

Etsy.com

x-recruiting: Is code your craft? www.etsy.com/careers

Otto.de

x-recruiting: Seems you like http headers. To write ours, apply at job.otto.de and mention this header.

Ingin melihat daftar lengkap? Untuk melakukan ini, saya membuka repositori di GitHub .

Selain tawaran pekerjaan, lebih banyak hal kreatif ditemukan dalam proses penelitian . Mereka benar-benar membuat saya bersemangat sebagai penggemar besar omong kosong misterius .

Header HTTP Misterius


Situs web 9kw.eu, yang tampaknya merupakan penyedia captcha, menampilkan pesan rahasia "42":

X-Secret-Message: 42

istreetview.com tidak lagi didukung, tetapi alamat formulir web rahasia dapat ditemukan di header.

X-Secret-URL: https://appio.link/secret

Saya menulis kepada mereka jawaban saya ...

Di thetradersdomain.com ada "saus tersembunyi" di judul, tetapi ini rahasia:

x-secret-sauce: Confidential

Pada gambar-dnxlive.com ada beberapa tautan "rahasia" lagi di salah satu header HTTP:

X-Secret-Message: camscv.dnxnetwork.lu

Jika Anda suka mobil mewah, inilah jaguar. ro dengan header untuk mendeteksi bot:

X-Bot: false

Tetapi metode ini tidak bekerja dengan baik dan gagal jika Anda mengganti agen-pengguna (maaf, kawan dari Jaguar).

Namun ... apakah Anda pernah melihat server dengan nama panggilan ? Ada pasangan di sini:

X-men.com

X-ServerNickName: clint

Howgoodisyourseo.com

X-ServerNickName: The Internet

Dan yang terakhir, tetapi juga contoh yang menarik. Teman-teman kami dari m.bidorbuy.co.ke menyatakan minat mereka dalam tajuk HTTP:

x-powered-by: Passion and tiny cute kittens
x-servernickname: The Beast
x-hacker: If you are reading this, maybe you should be working at bidorbuy instead
