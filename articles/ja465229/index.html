「コンサルタント+」：マテリアルのフローの変化。フォント、スタイル、Pythonでのテキスト配置

「コンサルタント+」-弁護士、会計士などのための参照システム。時計のように安定して動作します。この投稿では、テキスト出力の観点からこのクロックを必要に応じて少し設定することをお勧めします。つまり、システムがPythonで提供するテキスト情報を処理する方法を見てください。途中で、タイトルで宣言されているテキスト要素を操作します。

編み込みフェンスの陰


ヘルププログラム「コンサルタント+」に長い間携わってきた弁護士として、私はいつもこのシステムの通常の機能に欠けていました。この機能は以下の通りでした。規制法に変更が見られると、K +の従業員は2列のテキスト形式で変更の概要を広めます。



左側の列は以前の列で、右側の列は現在有効な標準です。現在（数年前）、機能が更新され、変更が太字で強調表示され、すぐに表示されます。これはすべて非常に便利です。しかし、不快なことがあります。

まず、いくつかの規範は与えられていません、なぜならK +の従業員にはボリュームが大きすぎるため、システムリンクに移動する必要があります。次に、これらの2つの列を通常のExcelまたはWordのテーブルに貼り付けてコピーすることはできません。

おそらくこれは意図的に行われたため、ユーザーはシステムから何も転送しないなど、システムをよりアクティブに操作できます。

まあそれを修正する必要があります。

タスク：可能な場合とそうでない場合の2つの列にテキストを広げる-単に基準を削除し、これをすべてExcelスプレッドシートに入れます。同時に、Pythonを使用してテキストのフォント、配置、その他の些細な点を変更する方法を見てみましょう。

from __future__ import unicode_literals

with open ('.txt',encoding='cp1251') as f:
    lines = (line.strip() for line in f)
    for line in lines:
        if line.startswith(''):
            continue        
        col1=line[:35]
        col2=line[39:]
        col3=line[35:39]
        if line.startswith(a): 
            sheet.cell(row=x, column=1).value=line #   ,  ..

        elif line==None:

        #print (line)

        if len(col2)==0: #   1-     2-
