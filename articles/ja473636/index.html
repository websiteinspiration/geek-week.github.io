4GBの空き容量を持つタブレットを介して16GBの急流をダウンロード


仕事：


インターネットのないパソコンもありますが、USB経由での転送も可能です。このファイルを転送できるインターネット付きのタブレットがあります。タブレットで、目的の急流をダウンロードできますが、十分な空き容量がありません。トレント内のファイルは1つで大きなものです。


解決への道：


私は急流をダウンロードし始めました。空き容量がほとんどなくなったところで、ダウンロードを一時停止しました。タブレットをPCに接続し、ファイルをタブレットからPCに移動しました。彼は一時停止し、驚いたことに、ファイルが再度作成され、急流は何も起こらなかったかのようにさらに揺れ続けました。


トレントクライアントが受信したデータを書き込むファイルにスパースフラグを設定するため、システムはすぐに16GBを予約しようとせず、4GBを超えるファイルに書き込もうとしたときにエラーは発生しません。


手順を4回繰り返し、同じ急流の異なる部分が含まれるPCで4つのファイルを受け取りました。今ではそれらを一緒に組み立てる必要があります。手順は基本的に簡単です。4つのファイルの1つでこの位置にある場合、ゼロバイトを別の値に置き換える必要があります。


そのような単純なプログラムはインターネット上にあるべきだと私には思えました。本当に誰もそのような仕事に直面していませんか？しかし、私は彼女を探すためのキーワードさえ知らないことに気付きました。したがって、私はすぐにこのタスク用のLuaスクリプトを投げ、それを最適化しました。共有したいです。

function merge_part(streams_in, buffer_length)
    local out_part
    for _, stream in ipairs(streams_in) do
        local in_part = stream:read(buffer_length)

        if not out_part then
            out_part = in_part --      
        elseif in_part and #in_part > 0 then

            if #out_part < #in_part then
                out_part, in_part = in_part, out_part
            end

            if out_part ~= in_part  --  
                and in_part:find("[^\0]")   --    in_part
                and out_part:find("\0", 1, true) --     out_part
            then 
                local find_index = 1
--[[

string.gsub .


--]]
                out_part = out_part:gsub("\0+", function(zero_string)

                    if #in_part < find_index then
                        return --    
                    end
--[[

string.gsub . zero_string string.find. .


--]]
                    local start_index = out_part:find("\0", find_index, true)
                    find_index = start_index + #zero_string

--[[

in_part out_part .


--]]
                    if #in_part >= start_index then
                        local end_index = start_index + #zero_string - 1
--[[

in_part .


--]]
                        local part = in_part:sub(start_index, end_index)

                        if (part:byte(1) ~= 0) or part:find("[^\0]") then
--[[

part .


--]]
                            if #part == #zero_string then
                                return part
                            else
--[[

part . .


--]]
                                return part..zero_string:sub(1, end_index - #in_part)
                            end
                        end
                    end
                end)
            end
        end
    end
    return out_part
end
