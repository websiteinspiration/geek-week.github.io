数据准备，用于调整和训练用于目标检测的神经网络

用于调整和训练用于目标检测的神经网络的数据准备。



我们继续进行对象检测的主题。为了使算法能够区分对象或能够检测过程中的偏差，必须在与已分析视频资料的整体图片相对应的数据集上对其进行训练。准备训练集是一个非常耗时的过程，它取决于训练的速度和算法的质量。考虑使用多种工具来帮助您正确准备数据。



准备的第一步是汇编初始数据集。通常，数据量越大，被分析对象的多样性越高，算法的质量就越高。如果您打算分析DVR中的视频，则最好从现有视频资料中截取数据集，然后编译数据集，因为在这种情况下，将需要更少的数据和更少的再培训时间。在其他情况下，数据集的编译要考虑到从中捕获对象的视角，其大小等，训练集中的对象与算法将分析的材料中的对象有多少相似。数据集必须由各种变体组成，即在其上将同时显示一个对象和多个对象，例如，一帧中的一个人和一个电话。



如果使用摄像机或电话对数据集进行汇编，则图像很可能具有高分辨率，这会对学习过程产生负面影响。高分辨率图像的主要问题将是“内存不足”错误，即使batch_size参数很小。为了减少该错误的可能性，有必要减小图像的尺寸。 IrfanView程序将为我们提供帮助。该程序允许您批量处理图像，即您只需要指定源图像的路径，设置转换参数并开始转换过程即可。结果，我们在指定的文件夹中获得了“精简”图像。为此，请在程序的开始窗口中按B键（俄语布局“ I”），或在“文件”选项卡中选择“批处理转换/重命名”，出现如下窗口：







接下来，我们需要：



  1. 选择批量转换模式；
  2. 选择源图像所在的文件夹；
  3. 配置高级高级设置；
  4. 添加所有图像；
  5. 选择转换结果的文件夹；
  6. 开始批量转换。


高级选项中有许多可用的转换，在此示例中，我们仅更改原始图像的大小，将大小减小10倍（指定原始值的10％）：







因此，如果原始图像为4160 x 3120，则生成的图像将变为416 x 312像素。



培训样本的形成是一个相当繁琐的过程，无论是从视频资料中截取屏幕截图还是使用相机和手机拍摄。为了显着增加数据集的数量，使用了各种扩充方法。让我们看看使用Pillow库以Python编程语言实现的一些方法。最受欢迎的增强方法之一是“镜面反射”。



首先，我们导入必要的库： 



from PIL import Image, ImageOps
import os
import numpy


我们用源图像指示目录的位置以及转换结果的路径： 



path_to_origin = "images/" 
output_folder = "Augmentation_result/"
img_names = os.listdir(path_to_origin)

if not os.path.exists(output_folder):
    os.makedirs(output_folder)


我们将编写一个将图像转换为“镜像”的函数： 



def mirror(path_to_origin, img_name, output_folder):
    im = Image.open(path_to_origin + img_name)
    ImageOps.mirror(im).save(output_folder+"mirror"+img_name, "JPEG")


另一种流行的增强方法是图像旋转。对于大型对象，值得考虑的是，使用这种增强方法会丢失部分图像。我们编写一个旋转图像的函数： 



def rotate(path_to_origin, img_name, output_folder):
    im = Image.open(path_to_origin + img_name)
    im.rotate(-30).save(output_folder+"rotate_"+img_name, "JPEG")


值（-30）是图像的旋转角度，该角度是根据对象的类型选择的。例如，旋转180度，如果图像中有电话，则这是增强的好选择，但是如果图像中有人，则此图像的180度是一个不好的例子，因为 您不太可能会看到人们颠倒的视频。



最后，我们将为每种图像花费两种类型的增强： 



for img in img_names:
    rotate(path_to_origin, img, output_folder)
    mirror(path_to_origin, img, output_folder)

有许多不同的增强方法（添加噪声，改变对比度等），所有方法都取决于要处理的数据类型。因此，使用两种扩充方法，我们将训练数据的大小增加了三倍。训练集的形成完成，它仍然标记图像中的对象。



对于标记，我们将使用LabelImage程序，因为我认为它非常易于使用和多功能。该程序能够以两种格式（PascalVOC和YOLO）返回标记结果。第一种格式以XML文件的形式表示，其中指示了图像参数以及描述图像中对象位置的矩形的坐标：







为每个图像生成这样的XML文件，其中描述了所有标记的对象。标记文件名与图像名相同。 YOLO格式以文本文件的形式表示，其中描述了对象的类别和矩形的坐标，并以标准化形式的值x，y，w，h描述：







除了带有用于YOLO格式的标记的文本文件之外，还将生成classes.txt文件，其中指示了对象的名称。对象名称的序列号取决于标记图像时出现的顺序。值得注意的是，重新打开应用程序时，classes.txt文件可能会被覆盖，因此，有必要监视程序中输入名称的顺序，否则会发生坐标不一致。对象的标记按以下顺序发生：







  1. 选择一个包含图像的目录；
  2. 选择将在其中保存标记文件的目录；
  3. 选择一种布局格式（PascalVOC或YOLO）；
  4. 选择“ W键”对象选择工具；
  5. 在图像中选择一个对象；
  6. 设置对象的名称（在重复操作过程中，可以从“确定”按钮下的列表中选择）；
  7. 确认
  8. 保存“ Ctrl + s”（对每个必需的对象重复步骤4-8）；
  9. 转到下一张图像。


完成后，标记文件的数量应与具有相同名称的图像的数量相对应。因此，我们研究了允许您调整图像大小，增加数据量以及在图像上标记要被神经网络检测到的对象的工具和方法。数据的形成和准备是重要且耗时的步骤，这取决于成功创建深度学习算法。
