Entradas al seminario web


Red Hat OpenShift Container Storage
Red Hat OpenShift Container Storage es una solución de almacenamiento desarrollada específicamente para infraestructuras de contenedores y también está estrechamente integrada con la plataforma Red Hat OpenShift Container Platform, que proporciona una interfaz unificada para administrar y acceder a los datos.

Este es Quarkus: el marco Java nativo de Kubernetes
Quarkus es el “ marco Java centrado en Kubernetes de próxima generación” de código abierto. Proporciona tiempos de carga de aplicaciones muy rápidos y bajo consumo de memoria. Esto hace que Quarkus sea ideal para cargas de trabajo Java que se ejecutan como microservicios en Kubernetes y OpenShift, así como cargas de trabajo Java que se ejecutan como funciones sin servidor.
