10 Alasan Teratas untuk Beralih ke Oracle Cloud Bagaimana bisnis dapat mempercepat inovasi?

Halo, Habr!

Hari ini kami ingin berbagi dengan semua orang yang tertarik dengan tema cloud, terjemahan whitepaper dari Oracle tentang topik transisi ke infrastruktur cloud. Terima kasih atas terjemahan rekan kami dan habroavtor permanenobieesupport.

RDTEX adalah salah satu mitra Oracle tertua di Rusia (perjanjian kemitraan pertama antara perusahaan ditandatangani pada tahun 1993). Saat ini, perusahaan kami memiliki status mitra tertinggi dari Mitra Oracle Platinum, dan Pusat Pelatihan RDTEX adalah salah satu pusat utama pelatihan resmi untuk Oracle di Rusia (kami pikir salah satu pembaca kami menghadiri kursus di Pusat Pelatihan kami atau bahkan belajar jarak jauh sekarang).

Rekan-rekan kami juga menciptakan glosarium istilah Oracle yang legendaris , yang digunakan oleh banyak perwakilan "komunitas baseodan" (spoiler: dalam salah satu tulisan berikut kami akan memberi tahu lebih banyak tentang glosarium).

Sekarang, kami sarankan untuk menggali lebih dalam infrastruktur cloud Oracle dan mempelajari bagaimana cloud Oracle dan database offline mempercepat inovasi bisnis. "Buku putih" asli juga dapat diunduh di situs web kami.





1. Oracle menyediakan

cloud generasi kedua Infrastruktur cloud Oracle menawarkan alat dan arsitektur yang membantu perusahaan bermigrasi secara mulus dari lokal ke cloud, menggunakan otomatisasi canggih dan keamanan bawaan untuk mengurangi ancaman, yang pada akhirnya mendukung migrasi superior dan efektivitas biaya.

Oracle Cloud Infrastructure Dirancang untuk Inovasi. Ini termasuk skalabilitas dan ketersediaan terkemuka di industri, manajemen dan kontrol terintegrasi, dan keandalan yang didukung oleh SLA ujung-ke-ujung. Misi cloud Oracle meluas untuk mendukung teknologi baru seperti kecerdasan buatan, pembelajaran mesin, Internet hal-hal, blockchain, dan antarmuka yang berorientasi pada manusia.

Pelanggan korporat Oracle telah beralih dari bereksperimen dengan teknologi ini ke menggunakan mereka untuk aplikasi kritis misi, membangun model bisnis baru, dan menciptakan nilai bisnis baru.

Oracle Cloud Infrastructure menjawab tantangan utama solusi cloud generasi pertama yang tidak dirancang untuk bekerja dengan sistem keuangan besar yang didorong oleh beban kerja atau aplikasi yang membutuhkan data dalam jumlah besar. Solusi cloud generasi pertama dibangun berdasarkan teknologi sejak satu dekade lalu, di mana pengaturan kinerja, keamanan, dan migrasi selalu terlambat. Oracle Cloud Infrastructure generasi baru telah dirancang khusus untuk memenuhi kebutuhan perusahaan saat ini, memberikan kinerja yang lebih cepat dan lebih dapat diprediksi, harga dan keamanan yang lebih baik, dan peningkatan kompatibilitas dengan beban kerja perusahaan.

Oracle adalah satu-satunya penyedia layanan IaaS, PaaS, dan SaaS sebagai bagian dari penawaran cloud generasi kedua. Dan layanan basis data Oracle yang berdiri sendiri menggunakan jaringan berkecepatan tinggi yang sama dengan layanan lainnya, memungkinkan Anda untuk dengan cepat menyebarkan aplikasi penting untuk mendukung inovasi yang berkelanjutan.

2.Oracle Cloud mendukung rencana jangka panjang Anda

Oracle Cloud Infrastructure adalah arsitektur cloud terbuka generasi kedua yang secara fundamental baru yang berfungsi sebagai fondasi bagi Oracle Cloud. Infrastruktur ini dirancang khusus untuk memberikan kinerja yang dapat diprediksi, keamanan dari inti sistem ke keunggulan terdepan, dan kontrol yang diperlukan untuk mendukung beban kerja kritis dan intensif.

Oracle Cloud menyediakan kemampuan dan teknologi untuk komputasi, pergudangan data, basis data, layanan jaringan dan platform yang diperlukan untuk memberikan hasil bisnis yang andal, memungkinkan Anda untuk memikirkan kembali kebutuhan pusat data.

Oracle Cloud Infrastructure mengungguli AWS dalam Kinerja Beban Kerja End-to-End ( Andrew Reichman, "Oracle Menguji Lebih Baik dalam Kinerja daripada Amazon Web Services," Oracle, 18 September 2018 ).

3. Peluang untuk mengoptimalkan kinerja

Oracle IaaS menawarkan berbagai kemampuan yang tak tertandingi dalam industri - mulai dari platform generasi kedua dan serangkaian layanan untuk platform tanpa sistem operasi (bare metal) hingga akses memori langsung jarak jauh (RDMA) untuk kluster komputasi teknis. Diferensiasi ini memastikan bahwa infrastruktur cloud Oracle menunjukkan kinerja yang dapat diprediksi dan isolasi yang baik dari klien cloud.
Infrastruktur cloud Oracle juga membuka pintu bagi inovasi mutakhir seperti Oracle Autonomous Data Warehouse dan Oracle Autonomous Transaction Processing.

Beban kerja mandiri bekerja paling baik pada infrastruktur yang dirancang untuk memberikan latensi rendah, ketersediaan tinggi, toleransi kesalahan, dan kinerja yang konsisten. Ini berarti bahwa data Anda disimpan dalam penyimpanan objek Oracle dan beberapa salinan secara otomatis direplikasi, sehingga memastikan ketersediaan tinggi dan toleransi kesalahan. Kemampuan penyembuhan diri mandiri Oracle memastikan bahwa data Anda tetap "sehat" dan bahwa pengguna selalu dapat mengakses replika terbaru yang diketahui.

4. Perlindungan yang mendalam

Keamanan adalah prinsip desain utama dalam infrastruktur cloud Oracle. Oracle menawarkan perlindungan dari inti sistem ke ujung tombaknya, seperti isolasi klien, keamanan data, deteksi ancaman internal dan eliminasi yang sangat otomatis. Oracle Infrastructure mengisolasi sumber daya komputasi dan jaringan untuk melindungi data dan lalu lintas pribadi Anda dari pengguna lain. Ini juga memisahkan kode, data, dan sumber daya Anda dari mesin kontrol - membantu mencegah penyerang mencuri atau memanipulasi data di cloud.

Oracle Cloud menghadirkan kemampuan serangan yang sangat terbatas melalui isolasi klien granular. Tingkat keamanan dengan firewall bawaan, detektor serangan DDoS, dan enkripsi secara proaktif mendeteksi dan menghentikan ancaman. Pelanggan dapat menetapkan identitas pada batas baru dan menggunakan otentikasi adaptif untuk secara otomatis menambahkan verifikasi tambahan ketika aktivitas pengguna menunjukkan tingkat risiko yang lebih tinggi. Akhirnya, Oracle memiliki salah satu portofolio cloud dan solusi lokal yang paling luas di industri.

5. Optimalisasi untuk beban kerja dari Oracle

Oracle Cloud Infrastructure menawarkan beberapa fitur dan alat unik yang mendukung migrasi yang efisien dan peluncuran portofolio database Oracle dan aplikasi bisnis. Memindahkan aplikasi Oracle membutuhkan perubahan minimal, yang mengurangi biaya dan durasi migrasi ke cloud. Prioritas Oracle adalah untuk menyediakan peralatan dan teknologi terbaru untuk terus meningkatkan produktivitas pelanggan.

Basis data Oracle berkali-kali lipat dalam hal penyimpanan dan kinerja pesaing, mencapai kapasitas hingga 40 terabyte dan jutaan operasi I / O per detik (IOPS) per instance. Ketika jumlah maksimum operasi I / O diperlukan untuk melakukan beban kerja pangkalan data otonom yang paling kompleks, Oracle menawarkan jumlah terbesar dalam memori non-volatile all-flash express (NVMe) industri (hingga 51,2 TB per instance). Meluncurkan cluster Oracle Real Applications Clusters (Oracle RAC) di Oracle Cloud Infrastructure memberikan database ketersediaan tinggi (HA) dengan failover dalam hitungan detik, meningkatkan kinerja hingga ratusan ribu I / O, dan operasi yang lancar berdasarkan tambalan dan pembaruan yang bergulir.Teknologi ini memperkenalkan standar cloud baru untuk aplikasi basis data produksi.

Oracle Exadata Cloud memungkinkan pelanggan untuk menjalankan mesin Exadata form factor di wilayah yang diperluas dan jaringan cloud virtual yang sama dengan bare metal compute dan layanan infrastruktur cloud Oracle lainnya yang dikelola pada sistem yang sama dengan sistem penuh atau di rak penuh. toolkit manajemen dan dapat diakses melalui konsol / API yang sama.

Infrastruktur cloud Oracle menawarkan serangkaian migrasi, penyediaan, dan alat manajemen untuk aplikasi Oracle utama, termasuk Oracle E-Business Suite dan Oracle PeopleSoft, Siebel, dan JD Edwards, serta kustomisasi dan basis data terkait, membantu pelanggan mempercepat transisi mereka ke cloud. Khususnya, untuk database yang berdiri sendiri, infrastruktur cloud Oracle mengoptimalkan analitik transaksi dan beban kerja pemrosesan transaksi. Ini secara khusus dirancang untuk menjalankan beban kerja ML untuk memberikan informasi berdasarkan kecerdasan buatan, yang memungkinkan klien untuk menghubungkan berbagai sumber data dan menggabungkan data kunci untuk mendukung keputusan bisnis yang sehat. Infrastruktur cloud Oracle juga merupakan dasar untuk layanan canggih baru, termasuk integrasi, wadah, layanan mikro dan blockchain,dan memberikan peningkatan skalabilitas untuk mendukung penyebaran Internet berbagai hal.

6. Gunakan dukungan penuh untuk pengembangan aplikasi

Oracle dengan cepat berinovasi dan menawarkan platform pengembangan aplikasi terbuka, berbasis standar, dan terintegrasi penuh yang memungkinkan Anda untuk membuat, menyebarkan, dan mengelola aplikasi berbasis cloud modern berbasis API dan perangkat seluler dengan biaya rendah. Selain itu, Oracle menawarkan dukungan untuk pengembangan kontainer dan cloud dengan sejumlah kecil kode yang dapat dimodifikasi.

Platform Oracle AppDev menyediakan lingkungan DevOps komprehensif untuk integrasi berkelanjutan (CI) / pengiriman kontinu (CD), diagnostik mendalam aplikasi Java dan integrasi yang disederhanakan dengan SaaS dan perangkat lunak lokal.

7. Rasio harga / kinerja terbaik

Oracle Cloud Infrastructure memberikan harga / kinerja terbaik yang tersedia di pasar saat ini. Beban kerja yang digunakan dalam infrastruktur cloud Oracle seringkali membutuhkan lebih sedikit server komputasi dan blok volume penyimpanan, yang mengurangi biaya untuk mengoptimalkan kinerja beban kerja.

Arsitektur infrastruktur cloud non-blocking Oracle tidak hanya membantu meminimalkan latensi, tetapi juga mengurangi risiko pengguna jaringan lain yang memengaruhi lingkungan klien, dan juga didukung oleh SLA industri yang paling komprehensif, aman secara finansial, end-to-end SLA.

8. Oracle Cloud Melampaui AWS

Oracle Cloud memberikan kinerja optimal dengan biaya terendah, menjadikannya pilihan terbaik untuk meluncurkan standar industri Database Oracle untuk aplikasi enterprise yang sangat penting. AWS tidak menawarkan Oracle RAC, Oracle Autonomous Data Warehouse, atau Oracle Autonomous Transaction Processing, dan Amazon Aurora menutup pengguna di cloud AWS, sementara database Oracle menyediakan dan menggabungkan fleksibilitas dari berbagai produsen.

Server komputasi Oracle, seperti Oracle Virtual Machines dan bare metal, membantu perusahaan mencapai penghematan biaya yang signifikan (25-65%) dibandingkan dengan mesin virtual yang bersaing seperti AWS.

Database Oracle lebih cepat dan biaya lebih rendah hingga 66% dibandingkan dengan AWS, yang memungkinkan Anda untuk mendapatkan hasil berdasarkan data bisnis lebih efisien dan terjangkau ( Accenture, “Beban Kerja Perusahaan Bertemu Awan: Memahami Database Oracle di Awan , ", Tinjauan Penyimpanan," Oracle Cloud Infrastructure Compute Bare Metal Instances Review ).



9. Oracle Cloud mendukung pertumbuhan UKM

Oracle Cloud dirancang untuk memenuhi kepentingan semua perusahaan, termasuk usaha kecil dan menengah. Hal ini disebabkan oleh fakta bahwa usaha kecil menjadi lebih besar dan sering mengalami lonjakan yang signifikan dalam kecepatan penerapan aplikasi. Namun, mereka mungkin masih menghadapi masalah yang sama dengan perusahaan besar. Oracle Cloud mendukung setiap skenario yang mungkin: rencanakan dan gunakan alat tradisional dan modern untuk pengembang dan untuk beban kerja yang sangat penting.

10. Oracle dan Microsoft menyediakan sistem cloud yang terhubung

Kemitraan teknologi Oracle dan Microsoft untuk saling berinteraksi cloud saling mendukung aplikasi reguler dan mission-critical. Pelanggan sekarang dapat bermigrasi ke cloud atau membuat aplikasi baru menggunakan yang terbaik dari Oracle Cloud, termasuk Oracle Autonomous Database, dan yang terbaik dari Microsoft Azure dengan kompatibilitas tanpa batas.

Kemitraan strategis ini menyediakan serangkaian kemampuan baru yang memungkinkan pelanggan bersama untuk lebih mudah bertransisi ke cloud, termasuk:

  • Koneksi lintas cloud,
  • Layanan cloud terintegrasi,
  • Model Dukungan Kolaboratif
  • Fleksibilitas penyebaran aplikasi.

Kemampuan utama klien untuk memaksimalkan kinerja masih menjalankan aplikasi Oracle di infrastruktur cloud Oracle.
Oracle Cloud memiliki versi uji coba gratis yang menawarkan Anda kesempatan untuk menjelajahi basis data mandiri pertama di dunia dengan manajemen otonom, perlindungan diri, dan penyembuhan diri. Gunakan versi uji coba ini dan temukan mesin virtual yang berperforma paling tinggi dan hemat biaya.



Untuk bekerja dengan versi uji coba, silakan kunjungi oracle.com/cloud/free atau hubungi mitra Oracle Anda.
