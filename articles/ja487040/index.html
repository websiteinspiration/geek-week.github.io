Android版Firefox Preview NightlyがuBlock Originをサポート

約束し、Mozillaはからの最初の拡張、uBlock起源、移動推奨拡張プログラムを Firefoxのプレビュー（開発コード名「フェニックス」）の毎晩のバージョンに。Android版Firefoxの次のバージョンでは、すでにインストールして広告のブロックを有効にできます。

Mozillaは最近、Firefox Preview v3.0をリリースし、新機能と改善を加えました。拡張機能サポートを追加するユーザーの要求に続いて、同社はブログで拡張機能サポートが2020年1月にFirefoxプレビューに表示され、推奨拡張機能プログラムのリストに拡張機能を最初に追加するよう取り組んでいることを発表しました。

「GeckoViewとFirefox Previewを、Android版Firefoxで以前サポートされていたAPIに合わせるのは一夜にして起こるものではありません。 2019年の残りおよび2020年まで、モバイルデバイスで安定して動作し、多くの機能をサポートする、推奨拡張プログラムからのカスタム拡張のサポートに重点的に取り組んでいます」とPhillip Quish氏は述べています。

GeckoViewに基づくFirefox Preview NightlyがWebExtensionsをサポートするようになりました。uBlockOriginは、更新後に追加された最初の拡張機能です。

uBlock OriginをFirefoxプレビューナイトリーにインストールするには：

  1. PlayストアからFirefoxプレビューナイトリーをダウンロードしてインストールします。
  2. Firefoxプレビューナイトリーを起動し、3ポイントメニューをクリックして[設定]に移動します。
  3. [詳細]セクションで、[言語]セクションの上に表示される[アドオン]を入力します。
  4. uBlock Originが推奨アドオンとして表示されるので、「+」アイコンをクリックしてインストールします。



あなたはその設定に行くことによってuBOを制御することができます。 uBOをインストールすると、アドオンマネージャーボタンがFirefoxプレビューナイトリーメニューに表示されます。近い将来、Firefox Preview Nightlyの「推奨」セクションでインストールできる拡張機能が増えると予想されます。

2019年の夏以降、Mozillaは次世代のAndroid版Firefoxの機能を既存のバージョンと比較しようとしています。 Firefox Previewに欠けていた最も重要な機能の1つが拡張機能のサポートだったからです。

Mozillaが新しいブラウザに取り組んでいる間、Android版Firefoxの安定バージョンはバージョン68に永遠に定着し、スポットアップデートのみを受け取ります。通常のユーザーは、Firefoxの安定版を引き続き使用できます。Firefoxプレビューのリリースは、今年の前半にいつかメインブラウザーになる予定です。

Firefoxプレビューナイトリーは、マスターブランチコードを使用してPlayストアで1日に2回更新されます。
