Microsoft Azure培训日：服务器基础结构迁移（注册完成）



2月13日，我们邀请您参加一个深度技术研讨会，该研讨会着重于为将混合方案转移到云中而实施混合解决方案的本地服务器基础结构的准备。作为活动的一部分，我们将使用一个实际示例来考虑一家大型公司的迁移，在该公司中，Windows Server 2008 R2既部署在物理服务器上，又部署在虚拟环境中，并且迁移了域和网络基础结构，文件服务器和SQL Server。

我们邀请计划迁移到Windows Server 2019的IT专业人员，系统管理员和架构师参加此次研讨会。

在此研讨会的框架中，我们将考虑以下主题： 

  1. 将基本基础结构服务迁移到Windows Server 2019，
  2. 本地基础架构与云服务的集成，
  3. 将虚拟机迁移到Azure平台。
