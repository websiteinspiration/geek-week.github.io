ASE Library for Atomic Simulations: Creating Nanoparticles

ASE library



ASE library is a python library for atomic manipulations and calculations. In this article, we will create nanoparticles using this library.



Installation



ASE is installed standardly through pip: pip install ase.



Atoms Object



The main object that can be manipulated is the Atoms object. It can store various structures. Let's try to create a carbon monoxide molecule: 



from ase import Atoms
from ase.visualize import view
d = 1.1
co = Atoms('CO', positions=[(0, 0, 0), (0, 0, d)])
view(co)

Clusters



Creating a cluster - a particle having a crystal lattice - is possible using the ase.cluster module. Clusters are generated by clipping pieces from an infinite crystal to clipping planes. Create a copper cluster: 



from ase.cluster.cubic import FaceCenteredCubic
surfaces = [(1, 0, 0), (1, 1, 0), (1, 1, 1)]
layers = [6, 9, 5]
lc = 3.61000
atoms = FaceCenteredCubic('Cu', surfaces, layers, latticeconstant=lc)
view(atoms)


where surfaces are clipping planes, layers is the number of layers.



We get the structure:







Nanoparticles



And here we come to the main topic of the article. Nanoparticles are particles of the order of nanometer in diameter (up to hundreds of nanometers). The most common are spherical nanoparticles - we will generate them. How to do it? Let's create a large cluster, and then cut out a sphere from it - by the distance of the atom‚Äôs coordinates from the center. To do this, first you need to center all the coordinates: 



surfaces = [(1, 0, 0), (1, 1, 0), (1, 1, 1)]
layers = [50, 50, 50]
lc = 3.61000
atomsCu = FaceCenteredCubic('Cu', surfaces, layers, latticeconstant=lc)

possCu = atomsCu.get_positions()
cCu = atomsCu.get_center_of_mass()

new_possCu = []
for el in possCu:
    new_possCu.append(el - cCu)
new_possCu = np.array(new_possCu)


The distance of the centered atom from the center is calculated simply: 



def get_radius_centered(pos):
    return (pos[0]**2 + pos[1]**2 + pos[2]**2)**0.5


Now we are writing a function for cutting out our spherical particle (more precisely, their position) from a large cluster: 



def cut_atoms(poss, r):
    poss_cut = []
    for pos in poss:
        if get_radius_centered(pos) < r:
            poss_cut.append(pos)
    poss_cut = np.array(poss_cut)
    return poss_cut


It remains to put this into practice: 



r = 5
possCu = utils.cut_atoms(new_possCu, r)
atoms = Atoms('Cu'+str(len(possCu)), positions=possCu)
view(atoms)


We get the following image of a copper nanoparticle with a radius of 5 angstroms (0.5 nm):







Core-shell structures



In practice, core-shell structures are often interesting - with a nucleus of one type of atom and a shell of another. For example, in real life, core-shell particles of CuAg (with a Cu core) may have catalytic activity comparable to Ag, but be cheaper. We will learn how to generate such structures.



We will adhere to the following strategy: we cut a hole out of a particle and fill this hole with a particle of a different type of atoms. To do this, you need a function that cuts a hole: 



def cut_atoms_inside(poss, r):
    poss_cut = []
    for pos in poss:
        if get_radius_centered(pos) > r:
            poss_cut.append(pos)
    poss_cut = np.array(poss_cut)
    return poss_cut


Create a silver cluster with centered atom coordinates: 



lc_ag = 4.09
atomsAg = FaceCenteredCubic('Ag', surfaces, layers, latticeconstant=lc_ag)

possAg = atomsAg.get_positions()
cAg = atomsAg.get_center_of_mass()

new_possAg = []
for el in possAg:
    new_possAg.append(el - cAg)
new_possAg = np.array(new_possAg)


and create a core-shell particle: 



r1 = 7
r2 = 5
possAg = utils.cut_atoms(new_possAg, r1)
atoms = Atoms('Ag'+str(len(possAg)), positions=possAg)

possAg = utils.cut_atoms_inside(possAg, r2)
possCu = utils.cut_atoms(new_possCu, r2)
poss = np.vstack([possCu, possAg])

atoms = Atoms('Cu'+str(len(possCu))+'Ag'+str(len(possAg)), positions=poss)
view(atoms)


Visualization shows that we have achieved what we wanted:





Now we can write the coordinates to a file: 



atoms.write('Cu'+str(r1)+'Ag'+str(r1-r2)+'.xyz')


Relaxation



And finally, an example of the manipulations for which the library was created. Relax the core-shell particle, i.e. we bring to a state with minimal potential energy. Set the calculator: 



from ase.calculators.emt import EMT
atoms = Atoms(atoms, calculator=EMT())
