Microsoft Azure Training Day: Server Infrastructure Migration (registration completed)



On February 13, we invite you to participate in a deep technical seminar that focuses on preparing the local server infrastructure for the implementation of hybrid solutions with the transfer of specific scenarios to the cloud. As part of the event, we will use a practical example to consider the migration of a large company in which Windows Server 2008 R2 is deployed both on physical servers and in a virtual environment with the transfer of domain and network infrastructure, file servers, as well as SQL Server.

We invite IT professionals, system administrators and architects who plan to migrate to Windows Server 2019 to the seminar.

In the framework of this seminar we will consider the following topics: 

  1. Migration of basic infrastructure services to Windows Server 2019,
  2. Integration of local infrastructure with cloud services,
  3. Migration of virtual machines to the Azure platform.
