道に迷いにくい時代

世界はどのように変化しましたか。アントワーヌドサンテグジュペリの素晴らしい小説「Night Flight」の歌詞を覚えていますか？ストーリーでは、サイクロンが原因で郵便飛行機が失われ、最終的には墜落したのか、緊急着陸に成功したのか、どこで発生したのかは不明です。ナビゲーターが一生を取り囲んでいるので、今、失われる可能性に驚かされる世代が成長しています。そして、「誰かが困っていて誰もそれを知らない」という状況も徐々に消えていきます。飛行機事故は多くのチャネルを通じて非常に迅速に記録されます。賢明な観光客はスマートフォンのサイズのデバイスを持参し、問題が発生した場合に助けを求めることができます。そして、自動車では、事故を自動的に認識し、自分で救助者を呼ぶことができるシステムが導入されています、ドライバーと乗客がこれを行うことができない場合でも。

「ナイトフライト」のイベントから10年も経たないうちに、無線コンパスが飛行機にますます出現し始めました。リングの形をしたアンテナにより、無線信号源、つまり特別な局や通常の放送局への方向を知ることが可能になりました。


ロッキードエレクトラ、リングの形をした無線コンパスアンテナが上からはっきりと見えます。写真提供：Christian Bramkapmp / aitliners.net

機材の故障や機材の操作に必要な資格不足は、有名なパイロットのアメリアイアハートの消失などの「夜間飛行」のような劇的な要因になる可能性があります。世界一周飛行の中間目標であるハウランド島は彼女の飛行機の音を聞き、環礁と燃料が不足していることを発見する試みについての無線メッセージを受信しました。アメリアイアハートの遺体、ナビゲーターのフレデリックヌーナン、飛行機はこれまで発見されていません。

リング状のアンテナにも不快な性質がありました-それは信号源への方向を示しましたが、飛行機がそれに近づいていたのか、逆に離れていったのかはわかりませんでした。このため、第二次世界大戦中に、B-24「レディビーグッド」爆撃機の乗員全員が亡くなりました-彼らはビーコンを飛ばして砂漠に引退しました。航空機が燃料を使い果たしたとき、彼らはそれが700キロであることを知らずにパラシュートでジャンプして基地に到達しようとしました。皮肉なことに、飛行機は砂の上に比較的柔らかい着陸を計画し、行った。 15年後、彼は砂漠で発見されました。助けを求めることができるラジオ局はそのまま残りました。


B-24爆撃機、上部の黒いドロップ型カウルのアンテナリング

しかし、一般的に、無線コンパスは非常に便利なデバイスであることがわかりました-ビーコンの間の廊下に沿って飛行することができ、聴覚ゾーンに複数のビーコンがある場合、良好な位置で位置を特定できます-地図に表示されている送信機からの方位（ビーコンへの方向）の交差が与える平面が配置されている点。衛星ナビゲーションのすべての成果にもかかわらず、ビーコンは依然として航空機ナビゲーションに使用されています。


サンクトペテルブルク周辺の村Ivanovo.rfのラジオナビゲーションマップ

静止ビーコンの上の固定廊下は民間航空には最適ですが、軍は敵がラジオビーコンを支援できない敵の領土に行動しなければならず、爆撃機の目標は毎日変わる可能性があります。すでに第二次世界大戦の初めに、ドイツ人は爆撃機をロンドンに向けるためにますます洗練された無線航法システムを使用しました。


ニッケバインシステム、爆撃機は1つのビームで飛行し、2番目のビームを横切ると爆弾を投下します。イラストはDahnielson / wikimedia.org

イギリス軍は無線干渉で対応し、信号でシステムの通常の運用を妨害しました。別の皮肉は、この戦いを主導したレジナルドビクタージョーンズが集会を崇拝し、ドイツのパイロットを欺くために国全体のリソースを受け取ったという事実をおそらく楽しんだことです。その結果、ドイツ空軍は電子戦に非常に強く駆られ、爆撃機の無線誘導システムへの信頼を失いました。

近距離での民間航空戦争の後、VOR / DMEシステムが標準になり、ビーコンまでの距離と方向の両方を決定で​​きるようになりました。軍事システムは同様の原理で機能します-TACAN西部とソビエト/ロシアのRSBN。飛行機の送信機は、地上局によって中継される要求を送信します。応答遅延時間は、航空機とステーション間の距離を決定します。ビーコンの方向を決定するために、他のアンテナが使用されます。1つは回転し、その信号は円の水平線の周りを走ります。もう一方は、最初のアンテナが北方向に放射するときに無指向性信号を放射します。飛行機での最初の信号と2番目の信号の受信の時間差により、それがステーションに対してどちら側であるかを判断できます。


VOR / DMEとTACANを組み合わせたアンテナ、ソース

長距離の場合、OMEGA、LORAN、Seagull、およびRSDNシステムは異なる原理を使用しました。互いに離れた位置に3つの無線送信機があり、同期して信号を発信しているとします。光速が有限であるため、信号は同時に航空機に到達しません。飛行機のどの駅までの距離もわかりませんが、信号を受信する時間の違い、つまり駅までの距離の違いを知っています。 2つの測点間の距離の違いを知ることは、誇張表現になります。 3つのステーションを使用すると、2つの双曲線を作成できます。その交点は、航空機が存在する可能性のある2つの可能なポイントを提供します。たとえば、サンクトペテルブルクよりモスクワに480 km近いことがわかっている場合は、ドニエプル（旧ドネプロペトロフスク）とウファの両方にいることができます。オムスクよりもモスクワまで50 km離れている場合は、ウファまたはペルミにいることができます。条件の組み合わせは、私がこのテキストを書いているUfaを与えます。


イラストコスミア星雲/ wikimedia.org

この原理は「双曲線航法」と呼ばれ、イギリスの爆撃機をドイツの都市に誘導するためにジーイングリッシュシステムで最初に使用されました。主な困難は、長距離にわたって離れている地上の送信機の同期ですが、原子時計の出現により、問題は一般的に1960年代に解決されました。長距離での動作を保証するために、長波が使用されたため、アンテナシステムは非常に高くなりました。


日本のオメガシステムのアンテナ、かつて国内で最も高いビル、国土交通省の写真/ wikimedia.org

宇宙時代の始まりは、衛星航法への関心を呼び起こしました。ジョンズホプキンス大学の応用物理学研究所の従業員であるウィリアムガイアとジョージウェイフェンバッハは、最初の衛星から信号を受信し、その信号のドップラーシフトを測定することで軌道を計算できることを発見しました。ドップラー効果-移動源からの信号の周波数の変化-衛星が実験室に近づくと増加し、離れると減少しました。



衛星の軌道を知っていれば、逆の問題を解決することができました。衛星信号のドップラーシフトによってその位置を特定することです。トランジットナビゲーションシステムが誕生しました。最初の衛星は1959年に軌道に乗ろうとしましたが（失敗しました）、2番目の衛星は1960年4月に打ち上げられ、同じ年に最初のテストが成功裏に実行されました。システムは1964年に委託されました。


米国国立航空宇宙博物館にあるトランジット5-A衛星

高さ1,100 kmの極軌道の5つの平面にある5つの衛星が、全世界をカバーしました。通常、宇宙には10個の衛星があり、各飛行機に1個ずつありました。自分の位置を決定する作業は簡単ではなく、多数の数学的計算が必要であり、より正確にするには、キャリアの不動性が必要でした。たとえば、アメリカの潜水艦の場合、ハッチにドラッグできるようなフォームファクターで密封および作成された特別なコンピューターAN / UYK-1を開発する必要がありました。


メーカーからのコンピュータ広告

衛星の軌道は時間とともに変化したため、現在の時刻だけでなく、通信局から1日に2回ダウンロードされた軌道の要素も送信しました。極近くの地上局は、その位置を知っており、衛星の軌道を絶えず測定し、それらに軌道のパラメータを送信しました。これらのパラメータを使用して、システムのユーザーの逆問題を解決しました。

しかし、すべての困難は獲得した機会で報われました。潜水艦はアンテナをわずか2分間進め、衛星信号を捕捉し、100メートルの精度でその位置を特定できました。間もなく、トランジットシステムは民間で使用できるようになり、多くの船員を助けるだけでなく、たとえば、多くの測定値を平均することでエベレスト山の高さを調整するなど、かなり珍しい問題を解決することも可能になりました。

ソビエト連邦では、サイクロンナビゲーションおよび通信システムは、シカダの民間版で作成され、同様の原理で動作し、6つの衛星で構成されていました。トランジットは1996年に動作を停止し、サイクロンシステムの最後の衛星は2010年に軌道に乗りました。

もちろん、トランジットとその類似物には欠点がないわけではありません。赤道地帯では、衛星は中緯度で数時間待機しなければなりませんでした。予想は1〜2時間に短縮されました。そして100メートルの精度はすぐに改善したかった。 1973年にすでに、米国は新しいGPSナビゲーションシステムのプロジェクトを開始しました。その最初のプロトタイプは1978年に軌道に乗りました。新しいシステムは、あなたがすでに知っているアプローチの修正を使用しました。

双曲線ナビゲーションを覚えていますか？ GPSとアナログは同じ原理を実装しています。すべての衛星は、軌道の正確な時刻とパラメータをブロードキャストします。光の速度は有限であるため、タイムスタンプはユーザーに同時に届きません。ユーザーは衛星と同期された原子時計を持っていないため、測定値の違いしか知りませんが、これで十分です。 3つの衛星からの信号により、空間に2つの双曲面を構築できます。その交点により、地球の表面に2点で接する双曲線が得られ、1つは正しい位置になり、2つ目は非常に不正確になるため、落下しやすくなります。


ミュンヘン工科大学のイラスト

4番目の衛星を追加すると、3つの双曲面が1つの点で交差し、表面上の高さも決定されます。また、衛星を追加するたびに新しい双曲面が得られ、精度が向上します。 3つの平面に24の衛星があり、24時間体制でシステムを利用できます。

他の最新のナビゲーションシステムも同様の原理で機能します。ロシアのGLONASS、ヨーロッパのガリレオです。中国の北斗も同じ原理ですが、衛星は異なる高さの軌道にあります。そして、最も劇的な物語は、間違いなく、ロシアのGLONASSです。

システムの開発は1976年に開始され、最初のデバイスが軌道に乗ったのは1982年でした。最初に小さなシリーズが打ち上げられ、10、9、12個の衛星が打ち上げられました。


「宇宙飛行士ニュース」、1999、第2

90年代の初めには、すでに12基の衛星がシステムの使用を制限しており、1995年に完全な配備が完了しました。悲しいかな、経済問題の中で、グループは悪化し始めました。デバイスの寿命が比較的短く、打ち上げが珍しい-1995年以降、1998年と2000年に3つの衛星を使った打ち上げが1回あったため、2001年には運用デバイスが6つしかないという事実につながりました。しかし、ゼロの始まりから、グループの復活が始まりました。 2003年、最初の第2世代の車両であるGLONASS-Mが軌道に乗りましたが、その最も重要な違いは寿命の延長でした。


GLONASS-M、写真：Bin im Garten / wikimedia.org

適用された技術的ソリューションは成功し、今日では最も古い現役の衛星が2007年に打ち上げられ、保証期間を半分超えました。しかし今日、星座は新たな課題に直面しています。当初は、第2世代が第3世代「GLONASS-K」に置き換えられ、さらに高い耐久性を約束する非加圧プラットフォームに切り替えられる予定でした。しかし、衛星は輸入されたコンポーネントを使用していましたが、2014年の政治情勢の複雑化によりアクセスできなくなりました。そして最終的に国産部品を使用した改造タイプ「GLONASS-K2」への変更が決定。現在、コンステレーションは劇的な段階を経ており、故障した第2世代の衛星は、すでに製造および保管されている衛星に交換する必要があり、新しい改造の生産が並行して開始されています。


GLONASS-K2、画像USSR BOY / wikimedia.org

これまでのところ、状況は順調です-GLONASS -Mの予備はほぼ終わりました-最後から2番目のデバイスは3月に軌道に乗るでしょう。 「GLONASS-K」は9点在庫あり、そのうち1点は5月の飛行予定。また、最初の「-K2」は2021年に発売されます。

民間ユーザーにとっては、最悪のシナリオでも問題はありません。4つのグローバルナビゲーションシステムは、ナビゲーターが常に衛星を見て位置を特定できることを意味します。ナビゲーションシステムだけでなく、さまざまなケースで役立ちます。 2018年からのヨーロッパと2015年からのロシア連邦では、新車は事故を認識して自動的に緊急サービス（eCallとERA-GLONASS）に通話を転送するシステムをインストールする必要があります。

両方のシステムは互換性があり、同じ原理で機能します。車のセンサーは事故の事実を記録します-エアバッグの展開、身体の変形など、衛星ナビゲーションシステムを使用して事故の程度と事故の座標を決定し、セルラーネットワークを介して救急隊にメッセージを送信します。2019年末の情報によると、460万台を超える車にERA-GLONASSが搭載されており、年間約36千件の通話が記録され、そのうち1万7,000通が自動モードです。専門家によると、このシステムは年間3〜4千人を節約します。
