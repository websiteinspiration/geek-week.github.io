p5.j​​sを使用して3Dオブジェクトに2Dテクスチャをオーバーレイする（パート1-パターンの作成）


3次元オブジェクトにフラットな描画またはパターンを強制することは、常に重要な作業であり、同時にある種のアニメーションでそれを実行したい場合は、複雑さが増すことがあります。

同時に、かなりシンプルなp5.jsライブラリがあり、数十行のコードで複雑な壮大なパターンを作成できます。私は最近そのようなプロジェクトを作成し、今例を挙げたいと思います。

開始する最も簡単な方法は、オンラインeditor.p5js.orgで新しいプロジェクトを作成することです

デフォルトでは、新しいプロジェクトは次のとおりです。

function setup() {
  createCanvas(400, 400);
}

function draw() {
  background(220);
}

function setup（）-一度だけ実行される関数を設定します
function draw（）-無限ループで繰り返す

まず、2D放射状パターンを作成し、それをキューブに適用します。これを作成するときは、最初に極座標を使用します。次に、座標を、x、yの形式で最もよく知られているデカルト座標に変換します。

理解しておくべき重要なこと：ポイントの極座標は2つの数値によって決定されます

。rは、ソースポイント（たとえば、キャンバスの中心）からターゲットポイントまでの距離です。
aは、ターゲットポイントが配置されている開始点を基準にした仮想線の角度です。

したがって、最初のパラメーターは0から無限大まで、2番目のパラメーターは0から360までさまざまです。

このアプローチの優れた点は、1つの式だけで非常に複雑なパターンを描画できることです。



ポーラローズは、方程式r（φ）= 2 \ sinφ

変数を作成するには、プロジェクトの最初に2つの変数を追加してみましょう。最初の変数は中心からの距離を

表し、2番目の変数は角度を表します。let r = 10 let a = 0 次に、無限ループで実行されるdraw（）関数で変数の変更を追加します。 = a + 0.4

その結果、次のコードが得られます。

let r =10
let a =0
function setup() {
  createCanvas(400, 400);

}

function draw() {
  background(220);
    a = a + 0.4
}

ここで、座標を、使い慣れた直交座標x yに変換し始めます。コードで何が起こっているかを正確に理解したい場合は、三角法のチュートリアルを読むことをお勧めしますが、これは結果にとってそれほど重要ではないので、コードを同じdraw（）関数に貼り付けるだけです。

let x = r*cos(a)
let y = r*sin(a)

最後に、数学的計算の結果を確認するには、次の2つのことを行う必要があります。

  • オブジェクトを色でペイントします（内部の色辞書を使用するか、通常のRGB形式コードを指定できます-3つの数値）
  • ポイントの分布の形状を設定します。たとえば、楕円。

function draw() {
  background(220);
  
  
  let x = r*cos(a);
  let y = r*sin(a);
  fill(255,0,0);
  ellipse (x,y,10,10);
  r = r + 0.4
    
}



楕円のパラメータ（Xに沿った位置、Yに沿った位置、Xに沿った直径、Yに沿った直径）は、xおよびyに沿った変数を使用して設定されます。変数xは変数rに依存するため、このサイクルでは絶えず0.4ずつ増加しているため、表示される赤い円は常に左上隅に移動します。

参照点を中心に移動するには、x変数の前に移動関数（200,200）を追加します。これで、円がキャンバスの中央に表示されます。

そして、別の問題を解決しましょう-出現するすべての円は、新しいサイクルごとに消去されます。この悪循環から抜け出すには、キャンバス（）の作成をdraw（）関数を超えて移動し、環境の作成を担当するsetup（）関数に転送する必要があります。

この変更後、ロードサークルに似たものが表示されます。主な問題は、それらが互いに非常に接近して配置されていることです。高次の変数Cに20の値を設定し、それを追加して各サイクルを0.6ずつ（aの横に）変更することにより、スペースを追加してみましょう。それを三角関数の公式に導入します：x = r + c * cos（a）およびy = r + c * sin（a）

let r=10
let a=0
let =20

function setup() {
  background(220);
  createCanvas(400, 400);
  
}

function draw() {
translate(200,200)

  let y = r+*sin(a);

  a = a + 0.4
