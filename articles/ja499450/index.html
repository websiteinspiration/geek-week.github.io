したがって、私たちのコーディングの最終的な目標は、遺伝学/進化/自然淘汰のシミュレーションで最高の衣装を手に入れることを可能にすることです。そして、この経路を視覚的にたどる機会を得てください。

  1. 個人レベルでの遺伝メカニズムを熟考し、実装する（強盗の「遺伝子」は機器の選択に直接影響するはずです）

  3. 遺伝的「競争」、遺伝子の選択を実現するために（それらのキャリアを一緒に押して、衝突の結果が「悪い」遺伝子を打倒し、「良い」遺伝子を高くするため）

3.機器は、強盗の「レーティング」の最終的な値を決定します。これを行うために、calculate_rateメソッドが呼び出され、損傷の数学的期待値を計算します。

calculate_rate 
    #     :
    def calculate_rate(self):
        dbg = DBG_rogue_calculate_rate

        #   :
        p_hit = self.stat_hit / 100

        #      :
        p_glancing = self.stat_glancing_percent / 100
        not_p_glancing = 1 - self.stat_glancing_percent / 100

        #      :
        p_crit = self.stat_crit / 100
        not_p_crit = 1 - self.stat_crit / 100

        #   :
        expectation_modificator = p_hit * (p_glancing * 0.7 + not_p_glancing * (p_crit * 2 + not_p_crit))

        #      :
        expectation_damage = expectation_modificator * self.stat_attackpower
        expectation_damage = round(expectation_damage, 3)

        if dbg:
            print('\t  =', expectation_modificator)
            print('\t  =', expectation_damage)

        return expectation_damage

ステージ3-人口、統計、チャレンジクラス


Population クラスは、次の3つの場合にのみ必要です。

  1. 常に変化しないランダムな遺伝子と生物学的パラメーターを使用して、指定された数の強盗から集団を作成します（wins_to_reproduce-強盗が繁殖のために獲得する必要がある勝利の数、defeats_to_die-個々の死に至る敗北の数）;

        #    :
        rogues_alive = []
