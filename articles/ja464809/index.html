顔認識技術をテストした方法とその結果



バイオメトリクス技術は私たちの生活に急速に浸透しています。顔認識はガジェットに表示されます。世界中の銀行がこの技術をATMで使用し始めています。顔認識システムに接続されたCCTVカメラは、法執行機関が犯罪者を捕らえるのを助けるように設計されています。人の助けを借りて、サービスにログインし、支払いを確認できます。そして、これは、ハイテクの場合と同様に、もちろん始まりにすぎません。人は私たちのパス、名刺、支払い手段になります。忘れられたり失われたりすることはありません（比喩的な意味を除いて）。しかし、このテクノロジーは十分に保護されていますか？

私たちはいくつかの人気のあるデバイスを取り、顔認証技術に基づくチェックを回避することを試みることにしました。簡単なものから始め、顔認証（顔認証）を使用します。

そのため、私たちは4つのデバイスを自由に使用できました。そのうちの1つ-Samsung Galaxy S8-には、顔認識テクノロジーに加えて、網膜認識テクノロジー（Iris Scanner）も使用されていました。

これらのデバイスで使用される2つのハードウェアおよびソフトウェアテクノロジーが強調表示されました。 1つ目は、古い写真認識技術、つまりフラットな画像です。プログラムは、カメラが認識し、保護された情報へのアクセスを提供または提供しないものと画像を照合しようとします。このテクノロジーのバイパスは、OnePlusおよびSamsungスマートフォンでテストされました。

より高度な2番目の技術は、表面上の点のレーザー投影を使用して、一種のグリッドを作成します。デバイスに組み込まれた赤外線カメラがこのグリッドの反射を読み取り、各ポイントからの戻り信号の持続時間によって、カメラの前に立っているオブジェクトの深度と形状を理解します。これにより、三次元画像が得られる。このテクノロジーは、Windows HelloとiPhone X（および一般的なKinectカメラ）で使用されています。

回避策として4つの方法が使用されました。

  1. プリンターで印刷されたカラー写真。
  2. プリンターでモノクロで印刷された赤外線写真。
  3. 自家製マスク;
  4. 未塗装の3D顔モデル、小さなコピー。



顔の3Dモデルの小さなコピーを使用したPavel Novikov

認識テクノロジーをバイパスするために、最も安価で最速の方法を使用したことをすぐに述べます。たとえば、等身大の塗装済み3Dモデルは複雑で高価です。サムスンのアイリススキャナーテクノロジーにより、かなり鮮明な網膜写真を赤外線で撮影できます。このテクノロジを回避することはできませんでしたが、回避策の例を含む非常に面白い記事に出くわしました。この記事から判断すると、良質のカメラで非常に高品質の目の写真を撮り、この写真を印刷して、コンタクトレンズを貼り付けてボリュームを与えるか、別のバージョンでは輝きを与える必要があります。接着剤で。もちろん、現実の世界では、見知らぬ人が自分の目を至近距離で撮影できるような人物を想像することは困難です。遠くから撮影する場合は、非常に強力な望遠レンズが必要になります（被害者自身がまったく動かないことはほとんどありません）。

言い換えれば、そのような方法の実質的な重要性は、かなりの材料投資と多くの状況の組み合わせを必要とするため、いずれにしても非常に小さいものです。攻撃をモデル化する場合、とりわけ、その実装の複雑さを理解する必要があります。

最初の回避策として、顔のカラー写真をA4シートに印刷しました。



カラーと白黒写真のアレクサンダーPoluninは認識の準備ができています

実験は、私たちが自由に使える2つのスマートフォン（OnePlus 5TとSamsung S8）を首尾よく安定して決定する最初の時代遅れの方法であることを示しました。

良い照明では、カラー写真を使用したバイパス方法は結果をもたらさず、スマートフォンはロックを解除しませんでした。暗い場所では、両方の電話が写真を「認識」しました。上記のように、どちらのデバイスも最も単純なテクノロジーを使用しています。カラー写真による認識です。カメラが見るもののフラットカラーイメージが、サンプルとして記録したものと比較されます。

2番目の方法は機能しませんでした-赤外線スペクトルで撮影され、プリンターで印刷された写真。それは白黒であり、センサーはそれを拾いません、それはカラー写真画像を必要とします。

簡単に言うと、ソーシャルネットワークで写真をダウンロードし、任意のフォトサロンでカラー写真を印刷するために数百ルーブルを支払うことで、このテクノロジーを回避できます。

アイリススキャナー網膜認識技術はサムスンS8でテストされました。先に述べたように、このテクノロジーは目のカラー写真にだまされることはありません。カメラとセンサーは所有者の目を「見る」必要があるため、スマートフォンを近づける必要があります。目が画面に表示されます。この画像を見る必要があります。赤外線ライトがオンになります。フロントカメラは目と網膜のパターンを撮影します。高品質で印刷した目の写真でデバイスのロックを解除することをお勧めします。私たちはカメラが彼女の目を見せていることを「理解」しましたが、写真の細部はシステムを訓練するには不十分でした。先に述べたように、この技術を欺く必要のある写真サンプルを含む他の研究者による記事から、この技術を回避する方法があることが明らかになりました。しかし、それは比較的面倒で高価であることがわかりました。

Windows HelloとiPhone Xで使用されている2番目のより近代的なテクノロジーについては、ここで興味深いニュアンスがいくつか見つかりました。

カラー写真の使用（1番目の方法）は、どのデバイスでも機能しませんでした。写真はフラットなイメージであり、Windows Helloでは、iPhone Xのように、3次元モデルが使用されます。このテクノロジーにより、カメラの前にある3次元オブジェクト、つまりその奥行きを「見る」ことができます。

Windows Helloテクノロジにはさまざまな用途があることに注意してください。スキャン、コンピューターゲームでの仮想モデリングに使用されます。あなたは彼女に手を見せます-彼女は仮想スケルトンを構築し、この仮想手を使って自分の手を使ってゲームでさまざまな操作を実行できます。 Windows Helloでの認識は、指紋と顔による2つの基準に基づいています。後者では、コンピューターの前に座っているだけでWindowsにログインできます。カメラは暗い部屋でも所有者を認識します。そのため、このテクノロジーは通常の意味で色を必要としません。カメラには2つのイメージセンサーがあります。1つはカラーで、もう1つはオブジェクトに投影されたマトリックスを読み取って赤外線で確認します。



Intel RealSenseカメラからの3D画像。同様に、顔認識システムは「見えます」。ユーザーにはサングラスがあります。カメラは目に関する情報を受信しませんが、顔に関する利用可能な情報によりユーザーはログインできます。

インテルRealSenseカメラを使用して赤外線写真を撮影し、白黒のプリンターで印刷してログインを試みました（2番目の方法）。 2018年の春まで、このかなり単純な方法でうまくいきました。デバイスソフトウェアに誤りがあることが判明しました。認証中、深度とボリュームは考慮されませんでした。 2018年4月、開発者はエラーを修正しました。

自家製のマスク（3番目の方法）は、古い技術、2次元、3次元のいずれの技術も騙すことができませんでした。

顔の3Dモデル（4番目の方法）の場合、Windows Helloは異常な動作をしました。このモデルのトレーニングのためにカメラを向けたとき、システムはそれが「見た」顔を囲みました。トレーニング中に、これが発生する場合と発生しない場合がありましたが、同時にシステムがトレーニングされました。

Intel RealSenseカメラのテクノロジーにより、ユーザーはシステムを再トレーニングできます。あなたのひげが成長したか、またはあなたの目の下に指を受け取り、システムがあなたを認識しなくなったとしましょう。この場合、あなたは彼女を訓練する機会があります。結局のところ、実際には、作成されたイメージに変更を加えたり、改善したりすることはせず、単に追加のイメージを作成するだけです。このようにして、1つのWindowsアカウントのシステムに5人の完全に異なる人を追加することができ、5人全員がこのアカウントを使用してログインする機会がありました。

iPhone Xの場合、赤外線写真はFace IDテクノロジーとIntel RealSenseテクノロジーの類似性を示しています。システムはそれが顔であることを認識し、それ自体が教育されることを許可しました。しかし同時に、iPhoneに入力された実際のユーザーの画像を印刷したため、ログインすることができませんでした。ボリュームはまだ役割を果たしています。原則は単純です。教えられたこと-したがって承認されました。画像で教え-画像でログインしました。顔を教えた-システムは写真を承認しなかった。

ロボットの過程で、かなり面白いニュアンスに出会いました。 Face IDテクノロジーを習得する場合、iPhone Xでは、おそらく3D画像をキャプチャするために頭を向ける必要があります。実際、自分の頭を向ける必要はありません。いずれの場合も、ゆっくりと正確に頭を左右、上下に回転させる必要はありません。デバイスが顔を見つけたら、カメラの前にあるオブジェクトを回転させることができます。たとえば、接着ペンシルを使用しました。効果はまったく同じに見え、まるで頭を回しているかのように円形のインジケーターが均等に塗りつぶされ、デバイスが顔のすべてのポイントを記憶できるようになりました。これは単なるマーケティング策略ですか？

最終的に


既存のすべての顔認識方法は、完璧とはほど遠いものです。これまでのところ、彼らは彼に不便をかけないように彼らの所有者を識別するのは得意でしたが、彼らは保護が簡単で安価な方法で迂回できないことを完全に保証していません。同じ指紋を偽造するのははるかに困難です。最初に指紋を取得する必要があります（これは簡単ではありません）。次に、物理的なコピーを作成します。最近の電話でPINコードをバイパスすることさえ、かなり複雑な技術的タスクです。すべての顔認識技術に関しては、記事で説明されているいくつかの困難にもかかわらず、それらは安全性が低く、回避がはるかに簡単です。ただし、センサーやアルゴリズムが改善されるため、そのようなテクノロジーが長期にわたって普及することは、おそらくそれらのセキュリティの向上に貢献するでしょう。

それまでの間、非接触型決済や顔認証付きのATMなど、直接お金と接触するような場合には、これらのテクノロジーを慎重に使用することをお勧めします。
