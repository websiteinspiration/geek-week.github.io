Kehidupan setelah studi sarjana: bagaimana saya memutuskan apa yang harus dilakukan selanjutnya ketika pendidikan tinggi dan pekerjaan sudah ada

Dari Juni hingga Oktober 2018, saya lulus dari universitas, menemukan pekerjaan, masuk magistrasi, memutuskan bahwa saya sudah memiliki pekerjaan yang baik dan pendidikan yang lebih tinggi dan bahwa saya tidak perlu berpasangan, saya dikeluarkan dari universitas. Dan kemudian saya masuk hakim lagi.

Dalam artikel ini, saya akan menjelaskan mengapa saya membuat keputusan seperti itu dan kesimpulan apa yang saya ambil dalam proses tersebut.



Artikel itu cukup panjang, jadi di sini saya akan menulis rencananya:

Karena saya tidak pergi ke sekolah pascasarjana HSE, tetapi pergi ke ITMO


Nama saya Olya Kunyavskaya. Pada musim panas 2018, saya lulus dari gelar sarjana SPbAU di bidang Bioinformatika. Kelulusan kami datang pada saat administrasi, guru dan siswa pindah dari SPbAU ke St. Petersburg HSE. Universitas akademik sangat bagus untuk saya dalam hal mendapatkan pendidikan sarjana, tetapi saya tidak mengikuti program master terbuka di kampus HSE St. Petersburg. Dalam program sarjana, saya terlibat dalam bioinformatika dan ingin berkembang di bidang ini, tetapi Pembelajaran Mesin, Rekayasa Perangkat Lunak, dan Bahasa Pemrograman pindah ke program Master St. Petersburg, tetapi bukan Bioinformatika.

Pada saat itu saya mengetahui tentang program master "Bioinformatika dan Sistem Biologi" di ITMO dan memutuskan untuk pergi ke sana. Program ini mencakup beberapa kursus yang telah saya pelajari, tetapi ada banyak yang baru. Saya menyukai gagasan bahwa teman sekelas saya akan menjadi pria dengan latar belakang biologis. Pemikiran para ahli biologi sangat berbeda dengan saya, dan saya berpikir bahwa lingkungan seperti itu dapat sangat memompa saya ke bidang biologi yang misterius dan mengajari saya untuk berpikir dengan cara lain. Saya juga suka bahwa pelatihan berlangsung dalam bahasa Inggris: Saya ingin percaya bahwa saya cukup tahu bahasa Inggris untuk belajar dan bekerja.


Saya mengikuti ujian lisan di bidang biologi di ITMO



Bagaimana saya keluar dari ITMO?


September 2018 Saya memasuki program master "Bioinformatika dan Sistem Biologi" di ITMO dan mendapat peneliti junior penuh waktu di Pusat Bioinformatika dan Bioinformatika Algoritmik di Universitas Negeri St. Petersburg (laboratorium ini sedang mengembangkan pengumpul genom SPAdes di bawah arahan Pavel Pevzner dan sebelumnya di Universitas St. Petersburg).

Oktober 2018 Saya dikeluarkan dari universitas. Saya tidak bisa belajar di ITMO karena detail yang saya ketahui sebelumnya, tetapi tidak pernah terpikir oleh saya bahwa itu akan membuat saya sangat sedih.

Terbiasa dengan suasana kamar SPbAU, tempat kurang dari 500 orang belajar, saya merasa tidak nyaman di universitas, di mana ribuan mahasiswa dapat mengikuti kursus yang sama. Orang-orang berjalan melewatiku, dan aku berpikir untuk melihat mereka untuk pertama dan terakhir kalinya dalam hidupku. Selain itu, ternyata saya tidak siap untuk menghabiskan bahkan upaya minimal untuk sesuatu yang menurut saya tidak berguna. Ketika saya pergi ke program master ini, saya tahu bahwa akan ada kursus yang telah saya ambil. Misalnya, dalam matematika diskrit. Saya akan melewati mereka dengan cepat, dan sisa waktu saya akan melakukan hal-hal penting bagi saya. Namun, ternyata model seperti itu bukan untuk saya. Dia mengajukan pertanyaan "Mengapa saya membuang-buang waktu", dan perasaan ini semakin meningkat karena pada awal tahun, informasi pengantar diceritakan berpasangan.

Dalam tiga minggu pertama, saya memutuskan bahwa karena sudah ada pendidikan tinggi, tidak masuk akal untuk kuliah di universitas. Saya ingin melakukan hal-hal lain - lebih menyenangkan daripada mengunjungi pasangan. Saya menyimpulkan bahwa saya tidak perlu menerima gelar master, dan dikeluarkan.

Tahun jeda. Lalu apa?


Saya berhenti menjadi mahasiswa dan pergi bekerja di laboratorium penuh waktu. Dan kemudian saya mengalami krisis alam yang terkait dengan kelulusan.

Penting bagi saya untuk mendapatkan pendidikan, dan pendidikan sarjana disertai dengan perasaan bahwa hal utama sekarang adalah menginvestasikan banyak energi dan energi ke dalam studi, dan Anda dapat berurusan dengan yang lainnya nanti. Jadi saya menemukan diri saya dalam situasi bahwa saya memiliki pendidikan yang lebih tinggi, saya diatur untuk pekerjaan yang baik, bahkan kehidupan pribadi saya telah ditetapkan. Lalu apa?

Saya menemukan pertanyaan eksistensial, yang mulai saya cari jawabannya. Saya berpikir tentang bagaimana saya ingin hidup di masa depan dan bagaimana mengatur kehidupan sehari-hari saya sehingga saya akan bahagia. Ini adalah pertanyaan yang tidak Anda pikirkan selama ada banyak tugas dan tenggat waktu.

Saya akan memberi tahu Anda pemikiran-pemikiran kunci tesis, satu atau lain cara yang terkait dengan magistrasi:

  • Saya ingin saya hidup dengan baik di sini dan sekarang, dan tidak di masa depan yang jauh. Kesadaran telah datang bahwa saya perlu memahami bagaimana saya ingin hidup, dan mulai hidup bahagia sekarang. Nikmati hidup mulai sekarang. Jika saya terus-menerus berpikir bahwa sekarang saya akan menderita, tetapi dalam "nanti" yang hebat saya akan bahagia, ada bahaya bahwa "nanti" ini tidak akan pernah datang.

Pada Januari, saya menyadari bahwa saya ingin belajar Rekayasa Perangkat Lunak di sekolah pascasarjana St. Petersburg. Saya tahu bahwa saya benar-benar menginginkannya, dan saya mengerti mengapa saya membutuhkannya.

Di musim dingin, saya menulis surat kepada dekan kami dan memberi tahu tentang niat saya untuk masuk, pada musim panas saya lulus ujian dan masuk. Dan pada musim gugur 2019, ia beralih ke paruh waktu dan pergi belajar.

Bagaimana saya pergi belajar lagi. Perbedaan antara studi sarjana dan sarjana.




Bagi saya, belajar di magistrasi adalah tahap baru dalam hidup saya, dan bukan kelanjutan langsung studi sarjana saya. Di sarjana muda itu, mendapatkan pendidikan adalah prioritas utama saya: Saya dengan berani menolak hal-hal yang menghalangi saya untuk belajar, dipersiapkan dengan cermat untuk ujian, pertama lulus pekerjaan rumah dan mengerjakan materi teori 2-3 kali. Pada saat saya memasuki magistrasi, saya menyadari bahwa hidup ini memiliki banyak segi, dan selain belajar, saya ingin melakukan hal-hal lain: membangun ikatan sosial yang kuat, karier, menciptakan keluarga, dan mewujudkan ide-ide saya sendiri. Terkadang saya lebih suka bertemu teman saya dengan pekerjaan rumah saya. Belajar bukan lagi tujuan utama, melainkan tambahan yang menyenangkan yang membuat hidup lebih bahagia.

Ada juga potongan dalam magistrasi (awalnya ada 6 orang di arah kita, 4 bulan kemudian), tetapi sekarang alasan utama untuk pemotongan adalah bahwa orang pergi ke magistrasi bukan dengan inersia, tanpa memahami mengapa mereka membutuhkannya, dan kemudian tidak ingin menghabiskannya waktu. temuan


Jika saya mencoba merangkum pengalaman saya dalam beberapa kesimpulan, saya akan mengatakan ini:
  • Sebelum lulus sekolah, penting untuk memahami mengapa Anda (dan apakah) menginginkannya.
  • Dengarkan dirimu sendiri. Perhatikan kenyamanan psikologis subyektif.
  • Mengubah keputusan Anda adalah hal yang normal! Tidak ada yang salah dengan mengubah universitas, mengeluarkan atau memasuki universitas setelah beberapa tahun bekerja.
  • Anda dapat mengambil jeda tahun untuk memahami di mana Anda ingin pindah.

  • Hanya Anda yang bisa mengerti apa yang Anda butuhkan. Pendapat Anda mungkin berbeda dari pendapat orang tua, mentor, teman, dan guru Anda, dan Anda tidak harus menjelaskan keputusan Anda kepada seseorang.

Ternyata penting bagi saya untuk mengambil istirahat untuk kesadaran, dan dari tindakan sadar saya menerima lebih banyak kepuasan batin dan jauh lebih banyak kembali daripada jika saya bergerak oleh inersia.
