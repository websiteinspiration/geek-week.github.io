Goブロックチェーンでブロックとトランザクションを設計する方法

データベースだけでなく、ブロックチェーンを最終的に作成するには、プロジェクトに3つの重要な要素を追加する必要があります。

  • データ構造とブロックメソッドの説明
  • データ構造とトランザクションメソッドの説明
  • データベースにブロックを保存し、ハッシュまたは高さ（またはその他）でブロックを見つけるブロックチェーン関数。



これは、業界向けのブロックチェーンに関する2番目の記事で、最初の記事はこちらです。

このシリーズの前回の記事で読者が私に尋ねた質問を思い出してください。この場合、LevelDBデータベースはブロックチェーンデータを格納するために使用されますが、他の、つまり同じMySQLの使用を妨げることはありません。これで、このデータの構造を理解できます。

トランザクションから始めましょう：github.com/Rusldv/bcstartup/blob/master/transaction/builder.go

以下はそのデータ構造です。

type TX struct {
	DataType byte		
	TxHash string 
	TxType byte	
	Timestamp int64		
	INs []TxIn
	OUTs []TxOut
}

type TxIn struct {
	ThatTxHash string
	TxOutN int
	ByteCode string
}

type TxOut struct {
	Value int
	ByteCode string
}

TXは、データタイプ（トランザクション2の場合）、このトランザクションのハッシュ、トランザクション自体のタイプ、タイムスタンプ、および入力と出力も格納します。 TxIn入力は、出力が参照されるトランザクションのハッシュ、この出力の数とバイトコードを格納し、TxOut出力は、いくつかの値とバイトコードも格納します。

次に、トランザクションがそのデータに対して実行できるアクションを見てみましょう。メソッドを分析しましょう。

トランザクションを作成するには、transaction.NewTransaction（txtype byte）* TX関数を使用します。

AddTxInメソッド（thattxhash []バイト、txoutn int、コード[]バイト）（* TxIn、エラー）は、入力をトランザクションに追加します。

AddTxOut（値int、データ[]バイト）（* TxOut、エラー）メソッドは、トランザクションに出力を追加します。

ToBytes（）[]バイトメソッドは、トランザクションをバイトスライスに変換します。

内部関数preByteHash（バイト[]バイト）文字列は、作成（）およびチェック（）で使用され、作成されたトランザクションハッシュとJavaScriptアプリケーションから生成されたトランザクションハッシュとの互換性を確保します。

Build（）メソッドは、トランザクションハッシュを次のように設定します。tx.TxHash= preByteHash（tx.ToBytes（））。

文字列のToJSON（）メソッドは、トランザクションをJSON文字列に変換します。

FromJSON（データ[]バイト）エラーメソッドは、バイトスライスとして送信されたJSON形式からトランザクションをロードします。

Check（）boolメソッドは、トランザクションのハッシュフィールドから受信したハッシュと、このトランザクションをハッシュした結果として得られたハッシュ（ハッシュフィールドを除く）を比較します。

トランザクションがブロックに追加されます：github.com/Rusldv/bcstartup/blob/master/block/builder.goブロック

のデータ構造はよりボリュームがあります：

type Block struct {
	DataType byte				
	BlockHeight int					
        Timestamp int64				 
        HeaderSize int					
        PrevBlockHash string				 
        SelfBlockHash string			
	TxsHash string			
	MerkleRoot string
	CreatorPublicKey string			
	CreatorSig string
	Version int
	TxsN int
	Txs []transaction.TX
}

DataTypeは、データ型とそのノードを格納し、トランザクションまたはその他のデータからブロックを切り離します。ブロックの場合、この値は1です

。BlockHeightはブロックの高さを格納します。
タイムスタンプのタイムスタンプ。
HeaderSizeブロックサイズ（バイト単位）。
前のブロックのPrevBlockHashハッシュ、およびSelfBlockHash-現在のブロック。
TxsHashは一般的なトランザクションハッシュです。
MerkleRoot-マークルツリーのルート。

次のフィールドは、ブロックの作成者の公開鍵、作成者の署名、ブロックのバージョン、ブロック内のトランザクション数、およびこれらのトランザクション自体です。

その方法を検討してください：
block.NewBlock（）関数は、ブロックを作成するために使用されます。タイプパッケージの定数からのブロックタイプも設定されます。
b.DataType = types.BLOCK_TYPE.


AddTxメソッド（tx * transaction.TX）は、トランザクションをブロックに追加します。

Build（）メソッドは、ブロックのフィールドに値を読み込み、現在のハッシュを生成して設定します。

ToBytesHeader（）[]バイトメソッドは、ブロックヘッダー（トランザクションなし）をバイトスライスに変換します。

文字列のToJSON（）メソッドは、ブロックをデータの文字列表現のJSON形式に変換します。

FromJSON（データ[]バイト）エラーメソッドは、データをJSONからブロック構造にロードします。

Check（）boolメソッドはブロックハッシュを生成し、それをブロックハッシュフィールドで指定されたものと比較します。

GetTxsHash（）文字列メソッドは、ブロック内のすべてのトランザクションの共通ハッシュを返します。

GetMerkleRoot（）メソッドは、ブロック内のトランザクションのマークルツリーのルートを設定します。

Sign（privk string）メソッドは、ブロック作成者の秘密鍵でブロックに署名します。

SetHeight（height int）メソッドは、ブロックの高さをブロック構造フィールドに書き込みます。

GetHeight（）intメソッドは、ブロック構造の対応するフィールドに示されているように、ブロックの高さを返します。

ToGOBBytes（）[]バイトメソッドは、ブロックをGOB形式でエンコードし、バイトスライスとして返します。

FromGOBBytes（データ[]バイト）エラーメソッドは、GOB形式の送信バイトスライスからブロックデータをブロック構造に書き込みます。

GetHash（）メソッド文字列は、このブロックのハッシュを返します。

GetPrevHash（）メソッドの文字列は、前のブロックのハッシュを返します。

SetPublicKey（pubk string）メソッドは、ブロック作成者の公開鍵をブロックに書き込みます。

したがって、Blockオブジェクトのメソッドを使用して、ネットワーク経由で送信するための形式に簡単に変換し、LevelDBデータベースに保存できます。

ブロックチェーンパッケージ関数は、ブロックチェーンへの保存を担当します：github.com/Rusldv/bcstartup/tree/master/blockchain

このため、ブロックはIBlockインターフェイスを実装する必要があります。

type IGOBBytes interface {
	ToGOBBytes() []byte
	FromGOBBytes(data []byte) error
}

type IBlock interface {
	IGOBBytes
	GetHash() string
	GetPrevHash() string
	GetHeight() int
	Check() bool

}

データベース接続は、パッケージの初期化中にinit（）関数で1回作成されます。 
db, err = leveldb.OpenFile(BLOCKCHAIN_DB_DEBUG, nil).

CloseDB（）はdb.Cloce（）のラッパーです-データベース接続を閉じるためにパッケージ関数を操作した後に呼び出されます。

SetTargetBlockHash（ハッシュ文字列）エラー関数は、データベースのBLOCK_HASH定数で指定されたキーを持つ現在のブロックのハッシュを書き込みます。

GetTargetBlockHash（）（文字列、エラー）関数は、データベースに保存されている現在のブロックのハッシュを返します。

SetTargetBlockHeight（height int）エラー関数は、BLOCK_HEIGHT定数で指定されたキーを持つノードのブロックチェーンの高さの値をデータベースに書き込みます。

GetTargetBlockHeight（）（int、error）関数は、データベースに格納されている特定のノードのブロックチェーンの高さを返します。

CheckBlock（ブロックIBlock）bool関数は、このブロックをブロックチェーンに追加する前に、ブロックが正しいかどうかをチェックします。

AddBlock（ブロックIBlock）エラー関数は、ブロックをブロックチェーンに追加します。

ブロックを受信および表示するための関数は、ブロックチェーンパッケージのexplore.goファイルにあります

。GetBlockByHash（ハッシュ文字列）（* block.Block、error）関数は、空のブロックオブジェクトを作成し、ハッシュが渡されたデータベースからブロックをロードし、それにポインターを返します。
