[说明]创建Google测试（Google表单）

感谢用户 外圈 和 Yuriy_krd 寻求帮助和建设性的批评

有时，根据设计人员创建某些东西似乎非常快速且简单：至少是一个站点，至少是一个调查表，至少是一个表。但是，您常常会忽略一些令人反感的小东西，因为这浪费了时间，并且由于令人反感的琐事而不得不重做工作。同意，这不愉快。我决定为您完成所有流程，并警告可能出现的不愉快琐事，并为新手提供帮助。

在本文中，我们将制作一个Google表单，您可以使用该表单收集数据，采访员工，客户，进行假设检验等。顺便说一下，有关远程工作的信息极为相关。



要创建测试，您必须具有有效的Google帐户。如果您没有，请参考说明“创建Google帐户和使用Google云端硬盘”。

因此，创建一个表单（测试）。

传入任务


我们需要创建一个将收集一组信息的测试。

  1. 测试人员的姓氏和名称。
  2. 电子邮件地址-在讯问知名收件人，链接不是通过电子邮件而是通过Messenger发送给收件人时，这是必需的。如果您通过发送电子邮件发送调查表，则可能无法使用该项目。
  3. 制作一个下拉列表，从中可以选择学校。同样，您可以选择班级，部门或单位。此选项是必需的，以便被测者正确确定他通过了谁的考试。在字段中手动输入数据时，最终表中的过滤器将不同地构造不同的寄存器或编写答案。
  4. 您需要使用一个选项来创建一个简单的问题，以获取正确的答案。

转到Google云端硬盘。为此，请点击链接顶部https://www.google.com/或https://www.google.ru/，然后单击右上角带有点的按钮，然后选择Google云端硬盘（以下称为云端硬盘）。



加载云端硬盘后，最好创建一个文件夹或文件夹结构。我们说“测试”->“求职者测试”，然后转到该文件夹​​，创建一个测试（窗体）。这将允许您构造数据。

步骤2.创建一个测试（窗体）


与Google磁盘上的任何文档一样，工作的开始从屏幕左上方的“创建+”按钮开始。此按钮在云端硬盘中创建文件夹，所有文档均已创建。



通过单击创建按钮，您将看到一个菜单。在其中，您需要选择“更多”。



有几种创建表单的方法：

1. 创建一个空表单：

  • 点击Google表单菜单项（标记1）
  • 单击下拉列表按钮（标记2），在下拉列表中单击“创建空白表格”（标记3）

2. 根据模板创建表单 -重复上述步骤以标记3，然后标记3+。模板库将为您打开，您可以在其中选择最合适的模板。将来，可以根据您的需要进行编辑。

我们创建了一个空表格（测试）。

步骤3.控件


首先要做的是为表单创建一个文件名。此名称是内部名称，该名称将存储在云端硬盘下。



通过单击屏幕左上方的短语“ New form”来创建名称。

屏幕中央是一个带有测试名称的框（窗体）。考生将在运行时看到它。将其更改为所需的。



在同一位置，我们填写测试的说明（表单）。

接下来，将自动创建问题的第一块。



在“无标题的问题”字段中，您需要输入问题。将选项1重命名为以下答案。如果您需要添加其他问题-您需要单击“添加问题”下方。您还可以添加选项“其他”，然后文本字段将出现在测试中，用户可以在其中以任何形式输入任何其他答案。

下拉列表允许您选择问题的类型。在我们的情况下，将自动创建一个带有一个答案选项的问题。如果打开下拉列表，则可以选择其他形式的问题。

在该块的底部，有一些按钮：

  1. “复制” -表示复制整个块。
  2. “删除” -删除整个块
  3. 强制性问题和滑动开关。他提出了强制性的问题：如果不回答，用户将无法继续或完成测试。
  4. 附加设置“三个点按钮” -允许您在答案中添加说明，混合答案，选择答案选项。

表单的右边是一个垂直的按钮块，允许（从按钮的顶部到底部）：

  1. 提出另一个问题
  2. 从先前创建的表单（测试）中导入问题
  3. 添加带有名称和描述的单独块
  4. 插入图片
  5. 插入视频（YouTube）
  6. 添加部分。在这种情况下，将创建另一页的问题。当以一种形式（测试）创建多个主题时，这很方便。

最后的控制块位于页面顶部：



从左到右：

  1. 选择问卷的颜色主题。在创建多个常规调查表时，这非常方便-为了消除员工之间的混淆，在提及报告名称时，还可以提及其颜色形式。这极大地促进了沟通和工作效率。
  2. 视图。允许在新选项卡中当前查看表单的完成版本而不发布它。

常规选项卡和演示文稿不应引起任何问题。一个重要的标签“测试”-在其中可以为表格中的每个问题设置一个标记。为此，激活“测试”滑动开关，您的表格将进入测试类别。激活此按钮后，您可以在问题列表中分配正确的答案以及正确答案的分数。

激活开关后，单击“保存”按钮并转到问题。按钮``答案''将出现在问题块的底部：



单击它，您将进入设置正确答案和为正确答案评分的模式。



因此，让我们从头开始任务。

步骤4.创建一个测试


  1. 我们在云端硬盘中输入文件名
  2. 输入测试标题
  3. 输入测试说明
  4. 输入问题
  5. 打开下拉列表后，选择“文本（行）”项

我们得到：



激活“必需的问题”开关。现在，要求用户输入姓氏和名字以继续测试。

通过单击控件的右垂直面板顶部的按钮添加第二个问题。



我们将使该问题成为可选问题，而无需激活模块右下方的开关。甚至不回答这个问题，用户将继续并能够完成测试。

我们用单个答案



选项创建另一个块，



我们添加了带有多个答案选项的块，这里我们还添加了“其他”答案选项。

现在设置正确答案的分数。转到控件右上角的设置项目（齿轮图标）。转到“测试”选项卡并激活开关。单击保存按钮，然后返回到我们的问题列表。

通过单击问题栏中的任何空白区域，我们将其激活。在左下方，我们激活响应块。我们为我们选择正确的答案并分配分数。



我们点击按钮准备好了。点分配。

现在，我们需要调整问题的频率和电子邮件地址的收集。

再次，转到控件顶部的“设置”。我们激活所需的设置元素。



我们直接转到“演示文稿”选项卡，并确认已激活“显示链接以重新填写表格”选项。单击“保存”按钮

，测试（表单）已准备就绪。

步骤4.提交测试（表单）


通过单击“提交”按钮来提交测试（表单）。在打开的窗口中，您可以选择其他发送选项。

通过电子邮件发送：



在“收件人”字段中，输入收件人的地址。在这种情况下，在“答案”选项卡中，您将看到谁投票和谁尚未回答表格问题。



同样，该表格可以立即包含在信函正文中。

下一个标签是发送链接。在这种情况下，您不仅可以通过邮件发送它，还可以通过任何其他方式（包括通过Messenger）发送它。但是，在这种情况下，您将不得不自己跟踪受访者。

第三个选项是在网站上插入。这是针对更高级的用户的。允许您将带有表单的代码直接嵌入网站模板中。

您已经完成了表单的创建。
