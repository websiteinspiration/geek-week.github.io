「これはすでに起こったようです？」同様のインシデントとクレームを検索する

システムのサポートにある程度の時間を費やしたすべての人は、新しいアプリケーションを受け取ったときのデジャヴの感覚に精通しています。あなたは時間を費やして、以前のアプリケーションを掘り下げ、同様のものを見つけようとすることができます。これは役立ちます。インシデントがより早くクローズされるか、根本原因を検出して問題を一度だけクローズすることさえ可能かもしれません。


チームに参加したばかりの「若い」従業員の頭には、そのような話はありません。ほとんどの場合、彼らは同様の事件が、たとえば6か月から1年前に発生したことを知りません。そして隣の部屋の同僚がその事件を決めた。


最も可能性が高いのは、「若い」従業員がインシデントデータベースで同様のものを探すのではなく、問題を最初から解決することです。より多くの時間を費やして経験を積み、次回はより速く対応します。あるいは多分彼らは新しいアプリケーションの流れの下ですぐにそれを忘れるでしょう。そして次回はすべてが再び起こります。


インシデントの分類にはすでにMLモデルを使用しています。チームがアプリケーションをより効率的に処理できるようにするために、「以前にクローズされた同様のインシデント」のリストを作成するためのさらに別のMLモデルを作成しました。詳細-カットの下。


私たちは何が必要なのか？


受け取ったインシデントごとに、履歴内で「類似した」クローズドインシデントを見つける必要があります。「類似性」の定義は、インシデントの最初、好ましくはサポートスタッフが分析を開始する前に行う必要があります。


インシデントを比較するには、連絡時にユーザーから提供された情報を使用する必要があります：簡単な説明、詳細な説明（ある場合）、ユーザーのレコードの属性。


チームは4つのシステムグループをサポートしています。類似のインシデントの検索に使用するインシデントの総数は約1万です。

def get_first_message(messages):
    res = ""
    if len(messages) > 0:
        # take the first message
        spl = re.split("\d{2}-\d{2}-\d{4} \d{2}:\d{2}:\d{2} - ((\w+((\s|-)\w+)?,(\s\w+)+)|\w{9}|guest)\s\(\w+\s\w+\)\n",
                       messages.lower())
        res = spl[-1]

        # cut off "mail footer" with finalization statements
        res = re.split("(best|kind)(\s)+regard(s)+", res)[0]

        # cut off "mail footer" with embedded pictures
        res = re.split("\[cid:", res)[0]

        # cut off "mail footer" with phone prefix
        res = re.split("\+(\d(\s|-)?){7}", res)[0]
    return res

__stemmer_lock = threading.Lock()

__stemmer = snowballstemmer.stemmer('english')

def stem_string(text: str):
    def stem_words(word_list):
        with __stemmer_lock:
            res = __stemmer.stemWords(word_list)
        return res

    return " ".join(stem_words(text.split()))

def clean_text(text: str):
    res = text
    for p in __replacements:
        res = re.sub(p[0], '#'+p[1]+'#', res)

    return res

def process_record(record):
    txt = ""
    for t in record:
        t = "" if t == np.nan else t
        txt += " " + get_first_message(str(t))

    return stem_string(clean_text(txt.lower()))

class CommentsTextTransformer(BaseEstimator, TransformerMixin):
    _n_jobs = 1

    def __init__(self, n_jobs=1):
        self._n_jobs = n_jobs

    def fit(self, X, y=None):
        return self

    def transform(self, X, y=None):
        features = Parallel(n_jobs=self._n_jobs)(
            delayed(process_record)(rec) for i, rec in enumerate(X.values)
        )
        return np.array(features, dtype=object).reshape(len(X),)
