一見、LyXに新しいエクスポート形式を追加する機能がないように見えますが（たとえば、TeXworksでは、そのような追加の可能性の類似は、新しいレイアウトツールを作成する機能であることを理解しています）、このプログラムにはそのような機会があります。まず、次のエクスポート形式を作成してみましょう。一見したところ、それは一見甘美に思えます。LyXプログラムでは、次のPATH変数が定義されています。

$$s The LyX system directory (e. g. /usr/share/lyx).
$$i The input file
$$o The output file
$$b The base name (without filename extension) in the LyX temporary directory
$$p The full directory path of the LyX temporary directory
$$r The full pathname to the original LyX file being processed
$$f The filename (without any directory path) of the LyX file.
$$l The `LaTeX name'

これは、LyXプログラムマニュアルの一部からの抜粋です。これらの変数の意味、意味の説明を翻訳するつもりはありませんが、代わりに新しいエクスポートフォーマットを作成してみてください（[ファイル]> [エクスポート先 ]に新しいボタンが表示されます.lyx）。このクリックが行われたウィンドウ）、それにエクスポートされたとき、一時ディレクトリ（たとえば、PDFへのエクスポート時に{pdflatex}ファイルが作成され、作成されたウィンドウに加えて.pdf）、puti.txtファイルが生成され、そこに（すべての）これらの値変数。これを行うには、PATHに追加されたフォルダーに、次の内容のputi.batファイルを作成します。
