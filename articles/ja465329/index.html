みなさん、こんにちは。機械学習コースの開始まで数日残っています。授業の開始を見越して、生徒とブログのすべての読者の両方が興味を持つであろう有用な翻訳を用意しました。そして今日、私たちはあなたとこの翻訳の最後の部分を共有しています。




部分依存プロット


部分依存プロット（部分依存プロットまたはPDP、PDプロット）は、機械学習モデル（JH Friedman 2001）の予測結果に対する1つまたは2つの機能の影響がわずかであることを示しています。PDPは、1Dまたは2Dグラフを使用して、ターゲットと選択された機能の関係を表示できます。

使い方？


PDPは、モデルのトレーニング後にも計算されます。上記で説明したフットボールの問題では、パスが転送された、ゴールへの得点の試み、得点されたゴールなど、多くの兆候がありました。まず、1行を見てみましょう。ラインが50％の確率でボールを持っていたチームで、100回のアシスト、10回のシュート試行、1つのゴールを達成したとします。

モデルをトレーニングし、チームにターゲット変数である「ゲームの男」を受け取ったプレーヤーがいる可能性を計算することによって行動します。次に、変数を選択し、その値を継続的に変更します。たとえば、チームが1ゴール、2ゴール、3ゴールなどを獲得した場合、結果を計算します。これらのすべての値はグラフに反映され、最終的に、予測された結果とスコア付けされた目標の依存関係のグラフが得られます。

PDPを構築するためにPythonで使用されるライブラリは、Python部分依存プロットツールボックスまたは単にPDPboxと呼ばれます。


from matplotlib import pyplot as plt
from pdpbox import pdp, get_dataset, info_plots
# Create the data that we will plot
pdp_goals = pdp.pdp_isolate(model=my_model, dataset=val_X, model_features=feature_names, feature='Goal Scored')
# plot it
pdp.pdp_plot(pdp_goals, 'Goal Scored')
plt.show()


解釈


  • Y軸は、元の値または左端の値で予測されたものによる予測の変化を反映しています。
  • 青い領域は信頼区間を示します。
  • ゴールスコアグラフでは、ゴールスコアにより「ゲームの男」賞を受け取る可能性が高くなりますが、しばらくすると飽和状態になります。


2Dグラフィックを使用して、2つの機能の部分的な依存関係を同時に視覚化することもできます。

SHAP値


SHAPは、SHapley Additiveの説明です。この方法は、予測を部分に分割して、各特性の重要性を明らかにするのに役立ちます。これは、ゲーム理論で使用されている原則であるベクターシャプリーに基づいています。これは、各プレーヤーが共同ゲームでの成功に貢献する量を決定するために使用されます（https://medium.com/civis-analytics/demystifying-black-box-models-with-shap- value-analysis-3e20b536fc80）。正確さと解釈可能性の間の妥協点を見つけることは、多くの場合困難なバランスになる可能性がありますが、SHAP値は両方を提供できます。

使い方？


もう一度、フットボールの例に戻りましょう。ここでは、「Man of the Game」賞を受賞した選手がチームにいる可能性を予測したいと思います。SHAP-値は、この特性がいくつかの基本的な値を取る場合に行ったであろう予測と比較して、特性の特定の値の影響を解釈します。

SHAP-値は、PyPIまたはcondaから簡単にインストールできるShapライブラリを使用して計算されます。


SHAP-値は、この特定の機能が予測をどの程度変更したかを示します（この特性のいくつかの基本的な値でこの予測を行う方法と比較して）。チームが固定の基本金額ではなく3つのゴールを決めた場合の予測を知りたいとしましょう。この質問に答えることができれば、次のように他の標識についても同じ手順を実行できます。

sum(SHAP values for all features) = pred_for_team - pred_for_baseline_values

SHAP値の高度な使用


複数のSHAP値を集約すると、モデルのより詳細なビューを形成するのに役立ちます。

  • SHAPサマリーチャート


どの機能がモデルにとって最も重要であるかを知るために、各機能と各サンプルのSHAP-値を作成できます。要約グラフは、どの機能が最も重要であるか、およびデータセットへの影響の範囲を示します。



各ポイントについて：

  • 垂直方向の配置は、それが反映する記号を示しています。
  • この色は、このオブジェクトがこのデータセット文字列に対して強い意味を持つか弱い意味を持つかを示します。
  • 水平配置は、この機能の値の影響がより正確な予測につながったかどうかを示します。


左上隅のドットは、いくつかのゴールを獲得したチームを意味しますが、予測が成功する可能性を0.25減らしました。

  • SHAP貢献度チャート


SHAP要約グラフは各特性の一般的な概要を提供しますが、SHAP依存関係グラフは、モデル出力が特性値にどのように依存するかを示します。SHAPコントリビューション依存関係グラフは、PDPと同様の洞察を提供しますが、さらに詳細を追加します。

結論


機械学習はブラックボックスではなくなります。その作業の結果を他の人に説明できない場合、良いモデルの使用は何ですか？解釈可能性はモデルの品質と同じくらい重要になっています。認識を達成するためには、機械学習システムが彼らの決定の明確な説明を提供できることが不可欠です。アルバートアインシュタインが言ったように、「単純な言語で何かを説明できない場合、それは理解できません。」
