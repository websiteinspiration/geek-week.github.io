コンピュータービジョンタスク-適切な色のオブジェクトを見つける

前書き


今日は、python / opencvを使用して目的の色の輪郭を認識する方法について説明します。そのようなタスクは、ロボット工学やあらゆる種類の自動化でよく見られます。

たとえば、提案されたソリューションを使用して、ロボットが移動してはならない線の輪郭、またはヘリコプターのオブジェクトを区別できます。たとえば、オレンジがバスケットに入れられていることを確認する必要がある場合に、このようなタスクが発生することがあります。



何を学ぶべきか


最初は、Pythonのアイデア（ほとんどの人がそうするだろう）とPythonの基本的な知識があると思います。

次のコマンドを使用してOpenCVライブラリをインストールします。

pip install opencv-python

そしてnumpy：

pip install numpy

コード


必要なライブラリを追加します。

import cv2
import numpy as np

ビデオが移動するカップオブジェクトを作成します。0-デフォルトのカメラ、1-カスタム。

cap = cv2.VideoCapture(0)

_, frame = cap.read()

フレームを読み取ります。代わりに_レートがあるかもしれません-fpsカメラがそこに行きますが、この場合は使用しません。

そして、さらに便利なようにフレームを処理しましょう。削減：


    frameresized = cv2.resize(frame, None, fx=2, fy=2, interpolation=cv2.INTER_CUBIC)
    hsv = cv2.cvtColor(frame, cv2.COLOR_BGR2HSV) #     BGR2HSV
    hsv = cv2.blur(hsv, (5, 5))  #    
    mask = cv2.inRange(hsv, (89, 124, 73), (255, 255, 255))
    #   
    lower_blue = np.array([38, 86, 0])
#    (   )

次に、処理されたフレームの輪郭を認識して並べ替えます。

 contours, _ = cv2.findContours(mask, cv2.RETR_TREE, cv2.CHAIN_APPROX_NONE)
    counturs = sorted(contours, key=cv2.contourArea, reverse=True)
