膝の上のVC単語の雲

最近、私は「コーヒー」という言葉が私の友人の中で頻繁に見つかるようにしたいと思いました。また、投稿に現れる頻度に基づいて、美しい言葉の雲を作りました。
文字通り膝の上でC＃を使用してそれを行う方法に興味がある場合は、猫の下でください。

注意
.. , 2 : , , .
- , - .

VKからデータを取得する


まず、適切なライブラリをインストールします。VKアプリケーションを作成します。

var services = new ServiceCollection();
var vkApi = new VkApi(services);

この段階はスキップできますが、承認しますが、VKで使用可能な「壁」の数は急激に減少します。

 vkApi.Authorize(new ApiAuthParams
                { AccessToken = " ",Settings = Settings.All});

または：

vkApi.Authorize(new ApiAuthParams
	{
		Login = "Login",
		Password = "Password",
		Settings = Settings.All
	});

選択した壁の最新の100件の投稿を取得します。

var posts=vkApi.Wall.Get(new WallGetParams
            {
                OwnerId = (long)Id,//   id    -
                Count = 100
            });

既存のコレクションから、1つの大きな行を作成します。

foreach (var post in posts.WallPosts)
            {
                if (!string.IsNullOrEmpty(post.Text))
                    data += post.Text;
            }

さらに、句読点から選択した行をクリアできます。

 data = Regex.Replace(data, "\\!|\\?|\\(|\\)|\"|\\#|\\,|»|«|-", string.Empty);

さて、単語のコレクションを取得しましょう。

 var words = data.Split(default(Char[]), StringSplitOptions.RemoveEmptyEntries).ToList();

最後から2番目の段階では、単語の頻度で辞書を作成します。

var wordsDictionary = new Dictionary<string, int>();
foreach (var word in words)
        {
            if (wordsDictionary.ContainsKey(word.ToLower()))
                wordsDictionary[word.ToLower()] += 1;
             else
            {
                wordsDictionary.Add(word.ToLower(),1);
            }
        }

最後に、並べ替えて、必要に応じてファイルに保存します。

wordsDictionary = wordsDictionary.OrderByDescending(x => x.Value)
                                 .ToDictionary(x => x.Key, x => x.Value);

単語の雲の作成に移ります


これを行うには、依存関係System.Drawingをプロジェクトとこのパッケージに追加する必要があります。

アプリケーションに依存関係を追加します。

using WordCloudGen = WordCloud.WordCloud;

そして画像を形成します。

            var wc = new WordCloudGen(1024, 1024);
            wc.Draw(wordsDictionary.Keys.ToList(),
                    wordsDictionary.Values.ToList())
              .Save("cloudwords.jpg");
