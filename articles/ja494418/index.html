インテルはプロセッサーのモデル番号をリセットします。しかし、これは確かではありません-投票があります

Core i3 / i5 / i7と数値インデックスが付いた最初のプロセッサーは、2008年にIntelによってリリースされました。さて、ご存じのとおり、第10世代のIntel Coreと最初の10番のモデルがあります。このような番号付けのすべてのロジックには、隠された問題があります。インデックスの増加に伴い、何かを示す他の数値とますます混じり合い、プロセッサの属性として認識されなくなります。たとえば、Core 9400モデルはおなじみで、Core 12800はすでにバッテリーに似ており、21280は一般的にVAZクラシックの分野のものです。指標が消費者にとって心理的に快適な範囲に留まるために、インテルは10世代ごとに最初の桁をリセットするオプションを検討しています。

ただし、このアイデアには明らかな欠点があります。まず、コンピュータテクノロジで12年間-長い期間ですが、禁止されていません。稼働中および流通市場には、旧式のかわいい名前のIntel Core i7-860、Intel Core i5-750などの多くの第1世代Intel Coreプロセッサがあります。インテルの新しいイニシアチブは混乱と混乱を招く可能性がありますか？



肩を切らないよう、思い切った対策を講じる前に、お客様と相談することにしました。最終決定の前に、インテルのWebサイトでグローバルな調査が行われます、インターネットにアクセスできる人なら誰でも参加できます。調査は4月20日に始まると想定されていたが、現状では確実とは言えない。インテルのニュースをフォローしてください！
