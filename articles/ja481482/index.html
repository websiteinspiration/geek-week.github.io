PHP SDKを使用したFacebookページへのクロスポスト

多くのサイトがソーシャルネットワークで新しい素材に関する情報を公開しています。これはFacebookで行うことができます。すでにアプリケーションを作成しており、[設定]メニュー項目の[詳細設定]タブでOAuthリダイレクトの許可ホストを指定していることが前提です。したがって、関数session_start（）を使用してセッションを開始します。セッションが無効になっている場合は、必要に応じてエラー出力をオンにできます。

error_reporting(E_ALL);
ini_set("display_errors", 1);

次に、PHP SDKライブラリー（githubにあります）をダウンロードし、srcフォルダー（名前は変更可能）を抽出します。次に、オートローダーへのパスを指定します。

require_once __DIR__ . '/_src/autoload.php';

次に、Facebookクラスのインスタンスを作成します。

$fb = new Facebook\Facebook([
	  'app_id' => '1234567890123456', //   id 
	  'app_secret' => '12345678901234567890123456789012' //  
	  ]);

承認を実行するためにアシスタントを呼び出し、ページで公開するために必要な権限を指定します。

$helper = $fb->getRedirectLoginHelper();

//  publish_actions,     ,     
$permissions = ['manage_pages','publish_pages'];

$loginUrl = $helper->getLoginUrl('url____', $permissions);

echo '<a href="' . htmlspecialchars($loginUrl) . '"></a>';

ログインする必要があるFacebookサイトにリダイレクトされたら、getLoginUrl（）関数で提供されるリンクをたどります。ここでも、access_tokenユーザーを取得したアシスタントに電話をかけます。

$helper = $fb->getRedirectLoginHelper();
	
try {
	 
    $accessToken = $helper->getAccessToken();
	
}

catch(Facebook\Exceptions\FacebookResponseException $e) {
	 
    echo 'Graph  : ' . $e->getMessage();
    exit;
	
}

catch(Facebook\Exceptions\FacebookSDKException $e) {
	  
    echo 'Facebook SDK  : ' . $e->getMessage();
    exit;
	
}

if (isset($accessToken))
     $_SESSION['facebook_access_token'] = (string) $accessToken;

elseif ($helper->getError())
    exit;

このトークンを使用してページに投稿しようとすると、ページトークンが必要になるため、「権限エラー」エラーが返されます。これを取得するには、Facebookオブジェクトのget（）関数を呼び出し、それに2つのパラメーターを渡す必要があります。1つはページトークンリクエストのURLで、これは投稿したいページのIDとユーザーの以前のaccess_tokenを含みます。

try {
	
	$response = $fb->get('/id_?fields=access_token', $_SESSION['facebook_access_token']);
		
}
		
catch (Facebook\Exceptions\FacebookResponseException $e) {
		
	echo 'Graph  : ' . $e->getMessage();
	exit;
		
}
		
catch (Facebook\Exceptions\FacebookSDKException $e) {
		
	echo 'Facebook SDK  : ' . $e->getMessage();
	exit;
		
}

// 
echo $response->getGraphNode()['access_token'];

受け取ったトークンの期間は無制限です。トークン情報はデバッガーから入手できます。メッセージを投稿してみてください。

$str_page = '/1234567890123456/feed';

$feed = array('message' => '');

try {
	
	$response = $fb->post($str_page, $feed, 'access_token_');
	
}
		
catch (Facebook\Exceptions\FacebookResponseException $e) {
		
	echo 'Graph  : ' . $e->getMessage();
	exit;
		
}
		
catch (Facebook\Exceptions\FacebookSDKException $e) {
		
	echo 'Facebook SDK  : ' . $e->getMessage();
	exit;
		
}
	
$graphNode = $response->getGraphNode();

echo ', id: ' . $graphNode['id'];

必要に応じて、Facebookを終了できます。

$helper = $fb->getRedirectLoginHelper();

$logoutUrl = $helper->getLogoutUrl($_SESSION['facebook_access_token'], 'url____');

echo '<a href="' . htmlspecialchars($logoutUrl) . '"></a>';
