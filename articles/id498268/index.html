API Mosbirge Gratis dalam Rumus Microsoft Excel

Sebelumnya, saya menulis tentang menerima data dari Moscow Exchange melalui formula Google Sheets . Namun, pertanyaannya tetap - apakah mungkin untuk mendapatkan data yang sama saat menggunakan Microsoft Excel atau analog LibreOffice Calc gratis secara lokal? Tanpa menggunakan skrip atau penyalinan manual.


Microsoft Excel dengan rumus untuk mendapatkan data dari Mosbirzha

Dan Anda dapat memberikan jawaban positif untuk pertanyaan ini. Ini bahkan lebih nyaman karena Anda tidak perlu mengharapkan fungsi IMPORTXML untuk memuat hasil di Google Tables.

Sebuah analog dari fungsi ini di Excel dan Calc adalah sekelompok rumus: WEBSERVICE + FILTERXML (FILTER.XML) .

Saat bekerja dengan Microsoft Excel, ada beberapa nuansa:

  • Fitur-fitur ini hanya tersedia di Excel 2013 dan versi yang lebih baru untuk Windows.
  • Fitur-fitur ini tidak akan mengembalikan hasil pada Mac.

LibreOffice Calc memiliki lebih sedikit pembatasan seperti:

  • Membutuhkan LibreOffice 4.2 dan lebih tinggi.
  • Tidak ada batasan pada OS yang digunakan. Bekerja di bawah:
    Windows
    Linux
    Mac OS
  • File Excel .xlsx terbuka dan operasional, tetapi pemformatan visual mungkin rusak.

Saya mengumpulkan contoh " API Microsoft Excel.xlsx" yang bisa diterapkan dengan fungsi yang dijelaskan dalam antarmuka program ke server informasi dan statistik dari Bursa Moskow (ISS / ISS) . Plus, ia menambahkan beberapa fungsi yang ditemukan oleh pembaca bagian pertama artikel tentang spreadsheet Google.

Menguji file ini di Microsoft Excel 2019 di Windows 10 dan di LibreOffice Calc 6.4 di Linux Mint 19.3. Di bawah Mac OS, saya tidak dapat menguji.

Moscow Exchange API dalam formula MS Excel


Secara umum, semua permintaan yang dapat dikirim ke Mosbirzi API ada di direktori . Tetapi bagi saya pribadi, panduan ini masih belum begitu jelas, walaupun saya cukup sering menghubungi API.

Id Mode Tawaran


Di Moscow Exchange API, banyak tergantung pada parameter "Pengidentifikasi mode perdagangan" (primary_boardid), yang dapat dilihat langsung di situs web mereka melalui formulir pencarian .


Pengidentifikasi mode perdagangan untuk saham Tinkoff.

Selain itu, pengidentifikasi ini dapat dilihat melalui permintaan HTTP reguler ke API :

https://iss.moex.com/iss/securities.xml?q=      &iss.meta=off&securities.columns=name,emitent_inn,isin,secid,primary_boardid

penerimaan otomatis nama saham, obligasi dan ETF

Sangat nyaman bahwa Anda bisa mendapatkan nama lengkap atau pendek dari instrumen. Untuk obligasi, nama lengkapnya sangat jelas.


File " API Microsoft Excel.xlsx" dengan contoh mendapatkan nama secara otomatis untuk kelas aset yang berbeda. Berfungsi dengan benar di LibreOffice Calc

Penerimaan otomatis harga saat ini


Tab ini memberikan contoh yang relevan untuk mendapatkan harga saham, obligasi, dan ETF dari Moscow Exchange.

Harga hari sebelumnya diambil melalui PREVADMITTEDQUOTE, dan bukan TERAKHIR dengan penundaan 15 menit, karena untuk beberapa instrumen rendah-cair hingga TERAKHIR mungkin tidak ada harga.
