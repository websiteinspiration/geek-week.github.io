Awal pengujian beta dari plugin untuk Rider dan PVS-Studio C # untuk Linux / macOS


Karena serangkaian berbagai peristiwa, awal pengujian beta plug-in untuk Rider dan C # analyzer untuk Linux / macOS sedikit tertunda. Namun, kami dengan senang hati mengumumkan bahwa hari ini telah tiba - hari ini kami meluncurkan uji beta.

Sebuah catatan kecil untuk mereka yang ketinggalan berita sebelumnya - saat ini kami sedang mengembangkan plug-in untuk IDE Rider, serta memindahkan penganalisis C # kami ke .NET Core untuk menganalisis proyek di Linux dan macOS. Versi Windows akan terus bekerja di bawah .NET Framework. Namun, saya menekankan bahwa dengan bantuannya Anda dapat memeriksa proyek .NET Framework dan .NET Core.

Anda dapat mendaftar untuk pengujian beta dengan mengisi formulir di halaman ini . Kami akan mengirimkan instruksi instalasi kepada Anda serta lisensi untuk penganalisa.

Karena kami sedikit mengubah jadwal rilis dan merilis 7.07lebih awal dari yang direncanakan, plug-in untuk Rider dan C # analyzer untuk Linux / macOS tidak termasuk di dalamnya. Tetapi kami memiliki lebih banyak waktu untuk meningkatkan produk-produk ini sebelum dirilis. :)

Sebagai kesimpulan, saya ingin menawarkan untuk mendaftar untuk tes beta , jika Anda belum melakukannya, minta kode tanpa kode, dan secara umum lebih banyak hal baik di masa yang aneh ini bagi kita semua.
