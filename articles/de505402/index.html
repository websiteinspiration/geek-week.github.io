Überlagern einer 3D-Textur mit 3D-Objekten mit p5.js (Teil 2 - Anwenden eines Musters auf einen Würfel)



Dies ist der zweite Teil des Projekts, in dem wir einem 3D-Objekt ein dynamisches 2D-Muster auferlegen. Sie können mit dem ersten Teil ( Teil 1 ) beginnen oder einfach den resultierenden Code einige Absätze weiter unten kopieren.

Vor einiger Zeit habe ich ein Muster erstellt, das mit mehreren Codezeilen in der Bibliothek p5.js generiert wurde.
p5.js ist eine 2014 erstellte Javascript-Bibliothek. Ihr Hauptziel ist es, Designern, Künstlern, Lehrern und anderen Vertretern kreativer Berufe ein Fenster in die Welt des Programmierens zu bieten. Es unterstützt sowohl Animationen als auch Diagramme und lässt sich problemlos in das Webformat übersetzen.

Jetzt erkläre ich Ihnen, wie Sie den letzten Teil erstellen - ein Muster auf ein 3D-Objekt (in diesem Fall einen Würfel) anwenden.

Um von vorne zu arbeiten, müssen Sie ein Projekt im Online-Editor editor.p5js.org erstellen. Wir ändern weiterhin den folgenden Code:

let r=10
let a=0
let =20

function setup() {
  background(220);
  createCanvas(400, 400);

function draw() {
translate(200,200)
  let x = r+*cos(a);
  let y = r+*sin(a);

Um unsere Leinwand voluminös zu machen, müssen wir der Funktion, die unsere Leinwand erstellt, den Parameter WEBGL hinzufügen: createCanvas (400.400, WEBGL). Da sich der Referenzpunkt für ein solches Objekt bereits in der Mitte befindet, können (und müssen!) Sie die Übersetzung (200.200) entfernen, die die Korrektur für ein flaches Bild durchgeführt hat.

Um den Farbumfang zu diversifizieren und ein spektakuläreres Bild zu erstellen, ersetzen wir die Parameter fill () durch unsere Variablen r, a, c. Mit jeder neuen Iteration der Endlosschleife (draw () ändert sich die Farbe der Ellipsen geringfügig. Fügen wir außerdem einen gewissen Abstand zwischen den Kreisen und den Neigungswinkeln hinzu, indem wir die Änderungsrate der Variablen c und a ändern.

   =  + 0.06
  a = a + 0.3

Als letzte Verbesserung fügen wir push () vor unserem Mustercode und pop () danach hinzu. Durch das Framing auf diese Weise wird der Code isoliert - er wird nicht von anderen Objekten beeinflusst, die wir erstellen.

let r=10
let a=0
let c=20

function setup() {
  background(220);
  createCanvas(400, 400, WEBGL);
  
}

function draw() {
  push()
  let x = r+c*cos(a);
  let y = r+c*sin(a);
  fill(r,a,c);
  ellipse (x,y,10,10);
  c = c + 0.09
  a = a + 0.8
  pop()
}



Am Ende bekommen wir so etwas. Es ist Zeit, einen Würfel zu erstellen.

In derselben draw () -Funktion fügen wir nach pop () den Boxcode hinzu:

  push()
  box(200)
  pop()

Jetzt ist die Box mit einer Größe von 200 nur noch auf einer Seite zu uns gedreht und sieht aus wie ein Quadrat. Um das Volumen abzuschätzen, müssen wir es bereitstellen. Fügen Sie dazu die vierte Variable oben zu den vorhandenen r-, a-, c-Winkeln mit dem Standardwert 0 hinzu. Damit dieser Parameter unseren Würfel nach dem zweiten Druck (), jedoch vor dem Erstellen des Felds, beeinflusst. Fügen Sie drei Funktionen hinzu: rotateX (), rotateY (), rotateZ () mit dem jeweiligen Winkelparameter.

Es ist logisch, dass sich ein Objekt mit einem Drehwinkel von 0 Grad nicht dreht. Daher fügen wir eine Änderung des Werts des variablen Winkels zu 0,003 hinzu (für den Anfang ist dies ausreichend). Als Ergebnis erhalten wir den folgenden Code:

let r=10
let a=0
let c=20
let angle = 0

function setup() {
  background(220);
  createCanvas(400, 400, WEBGL);
  
}

function draw() {
  push()
  let x = r+c*cos(a);
  let y = r+c*sin(a);
  fill(r,a,c);
  ellipse (x,y,10,10);
  c = c + 0.09
  a = a + 0.8
  pop()
  
  push()
  rotateX(angle), 
  rotateY(angle), 
  rotateZ(angle)
  box(200)
  angle = angle + 0.003
  pop()
}

Da unser Hintergrund nur einmal erstellt wird (er befindet sich im Setup), wird der erste Fehler behoben, durch den unser Würfel den Hintergrund beim Drehen löscht.



Um dies zu beheben, können Sie einfach den Hintergrund (200) innerhalb der Funktion draw () übertragen. Jetzt haben wir jedoch ein anderes Problem: Unsere Kreise verschwinden wie zu Beginn, da sie jetzt mit jedem neuen Zyklus überschrieben werden und nur für den Bruchteil einer Sekunde angezeigt werden Nichts.

Um dieses Problem zu lösen, müssen wir ganz am Anfang neben anderen Variablen der obersten Ebene eine neue Variable (z. B. art) ohne Standardwert erstellen.

Innerhalb der Funktion setup () setzen wir sie mit der Funktion createGraphics (400.400) gleich und verwenden sie dann an anderen Stellen.
Stellen Sie sicher, dass Sie dieselben Parameter wie beim Erstellen der Leinwand festlegen, um die Dimensionalität beizubehalten.

Jetzt fügen wir innerhalb des Mustercodes die Kunstvariable durch den Punkt zur Ellipse und ihren Eigenschaften hinzu (z. B. fill () und andere, falls Sie sie erstellt haben). 

art.fill(r,a,c);
  art.ellipse (x,y,10,10);

Danach bleibt das Muster dem Würfel als Textur aufzuerlegen. Fügen Sie dazu die Eigenschaft textur (art) in den Cube-Box-Code ein. Die Logik lautet wie folgt: Wir erstellen die Essenz einer grafisch isolierten Objektkunst, die die Funktionen zum Erstellen eines Musters mit den angegebenen Eigenschaften aufruft, und geben anschließend Kunst als Eigenschaftsparameter eine Textur für den Würfel.

let r=10
let a=0
let c=20
let angle = 0
let art

function setup() {
  createCanvas(400, 400, WEBGL);
  art = createGraphics(400,400);
}

function draw() {
  background(220);
  push()
  let x = r+c*cos(a);
  let y = r+c*sin(a);
  art.fill(r,a,c);
  art.ellipse (x,y,10,10);
  c = c + 0.09
  a = a + 0.8
  pop()
  
  push()
  texture(art)
  rotateX(angle), 
  rotateY(angle), 
  rotateZ(angle)
  box(200)
  angle = angle + 0.003
  pop()
}



Es bleibt noch das letzte Problem zu lösen: Unser Muster beginnt aus den Ecken des Würfels zu erscheinen. Um von der Mitte aus zu beginnen, können wir einfach die Hälfte der Breite und Länge des Würfels zum x- und y-Parameter hinzufügen, um die Mitte des Beginns des Zeichnens des Musters zu verschieben.

art.ellipse (x+200,y+200,10,10);



Wenn Sie das Muster diversifizieren möchten, können Sie versuchen, einfach die Parameter zu ändern oder sogar mehrere sich überschneidende Muster hinzuzufügen, beispielsweise aus einem anderen Projekt - So erstellen Sie den Effekt des Moiré-Musters in der p5.js-Bibliothek für Anfänger (Kurzanleitung)
