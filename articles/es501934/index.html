Publicar la biblioteca de diseño de componentes VKUI en Figma

Las bibliotecas en las herramientas de diseño son una parte importante de un sistema de diseño. Creemos que los diseñadores deben trabajar con elementos que estén lo más cerca posible de la implementación en el código. Por lo tanto, en las bibliotecas hemos recopilado para ellos los componentes más relevantes que reflejan las mismas condiciones que están disponibles durante el desarrollo.


El código fuente de React ya se ha publicado en Github : siguiendo el ejemplo de nuestros desarrolladores, ahora estamos compartiendo bibliotecas de componentes en Figma.



Hay cuatro bibliotecas disponibles en la página de VK en la comunidad de Figma :


  • Biblioteca base VKUI: una biblioteca básica que contiene más de 650 iconos y todos los tokens de color para un tema claro y oscuro (en forma de estilos);
  • VKUI iOS Library y VKUI Android Library: bibliotecas de componentes para iOS y Android, recopiladas en tokens de la Biblioteca base, así como plantillas de pantalla listas para usar;
  • Paleta de colores VKUI: una paleta de colores de las aplicaciones VK que recopilan tokens de colores.
