ソリューション作成者：ミハイルシュパノフ ミッシュッシュ。参加者の50％が問題を解決しました

n個の整数の配列aが与えられました。配列で最も頻繁に見つかる数を見つけるプログラムを作成します。時間制限：2秒、メモリ制限：256 MB。

入力フォーマット入力
の最初の行には、数値n（1≤n≤300,000）が含まれています。
2行目は含まnは整数 I（0≤ I ≤1 000 000 000）。

出力形式
配列aで最も頻繁に見つかる最大の数値xのみを出力します。

決定
通常の辞書とコレクションモジュールのCounter構造の両方を使用して解決できる単純なタスク。

最初に、各要素が発生する回数を計算し、次に、発生回数が最大の要素のうち最大のものを選択できます。このソリューションでは、O（n）時間とO（n）追加メモリが必要です。

from collections import Counter

n = int(input()) 
a = [int(i) for i in input().split()] 

counter = Counter(a)

result = a[0]
max_count = counter[result]
for number, count in counter.items():
    if count > max_count or (count == max_count and number > result):
        result = number
        max_count = count

print(result)

Null値の削除

著者：ミハイル・シュパノフ。参加者の7％が問題を解決

しました。JSON構造から空の辞書（{}）または空のリスト（[]）の値がある限り、そのような値を削除するプログラムを作成してください。辞書の値が削除されると、対応するキーも削除されます。時間制限：0.2秒、メモリ制限：6 MB。

入力フォーマット
単一の入力行には、長さn（1≤n≤1000）のストリングが含まれます。この文字列は有効なJSON文字列であることが保証されています。

出力フォーマット：
スペースを介して、削除された空の辞書の数と削除された空のリストの数を出力します。

決定
. , -, , JSON. json, isinstance, , Python. JSON {}, [], : , ''{}, []'' .

— , .

import json
dict_counter, list_counter = 0, 0
def clean_struct(struct):
    global dict_counter, list_counter
    if isinstance(struct, dict):
        for key, value in struct.copy().items():
            cleaned_struct = clean_struct(value)
            if cleaned_struct is None:
                del struct[key]
            else:
                struct[key] = cleaned_struct
        if len(struct) == 0:
            dict_counter += 1
            return None
    if isinstance(struct, list):
        i = 0
        while i < len(struct):
            cleaned_struct = clean_struct(struct[i])
            if cleaned_struct is None:
                del struct[i]
            else:
                struct[i] = cleaned_struct
                i += 1
        if len(struct) == 0:
            list_counter += 1
            return None
    return struct
struct = json.loads(input())
clean_struct(struct)
print(dict_counter, list_counter)

from collections import Counter, deque
import sys

def main():
    #   N, M  K
    n_of_lines = int(sys.stdin.readline().strip())
    window_size = int(sys.stdin.readline().strip())
    threshold = int(sys.stdin.readline().strip())

    #    «» ,
    #     
    #   
    bad_ips = set()
    last_ips = deque(maxlen=window_size)
    counter = Counter()

    for _ in range(n_of_lines):
        #  IP-
        current_ip = sys.stdin.readline().rstrip()

        # ,   
        if len(last_ips) == window_size:
            #      
            #      
            oldest_ip = last_ips.pop()
            counter[oldest_ip] -= 1

            #      —  
            if not counter[oldest_ip]:
                del counter[oldest_ip]

        #     
        last_ips.appendleft(current_ip)

        #        «»,
        #      
        if current_ip in bad_ips:
            continue

        #     
        counter[current_ip] += 1

        #      K,
        #      «»
        if counter[current_ip] >= threshold:
            bad_ips.add(current_ip)
            #  «»   ,
            #     
            del counter[current_ip]

    #  «»    
    print('\n'.join(current_ip for current_ip in sorted(bad_ips)))
