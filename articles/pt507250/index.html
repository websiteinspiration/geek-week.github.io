Consiste em comparar duas mãos de poker e determinar a vencedora.

As mãos de pôquer são representadas por linhas contendo substrings (indicando cartas individuais) de 2 caracteres, separados por espaços. 2H é um dos dois vermes, TC é uma cruz dez, etc.

O primeiro caractere significa classificação.
Todas as opções possíveis, ordenadas por antiguidade: 2 3 4 5 6 7 8 9 TJQKA .
Com as designações numéricas, tudo fica claro quanto às letras: T é 10 ( T en), J é o valete ( J ack), Q é a rainha ( Q ueen), K é o rei ( K ing),A - Ace ( A ce).

O segundo caractere define o naipe.
Opções possíveis: H - (vermes H earts), C - cruzes ( C Lubs), S - (picos S PAdES), D - tambores ( D iamonds).

Por exemplo, uma linha completa para a mão “2C 5C 3C 4C 6C” - nesse caso, é um flush direto de duas a seis.

A classificação das mãos é feita como no Texas Hold'em .

Você precisa entender que existem dificuldades ocultas - o uso de cartas adicionais para resolver situações de compra, o uso do valor nominal dos pares, etc.

Decisão


Então, como resolver esse problema? Primeiro, você precisa determinar um método de comparação que resolva a questão das mãos no ranking e, em seguida, se o ranking corresponder, o vencedor será determinado, se possível, comparando cartões adicionais.

Como a declaração do problema assume que os naipes são equivalentes, usamos um objeto simples para representar o ranking das mãos:

{ 
   rank: 1,       //     1  9,   ,    
   value: 'ABCDE' //  ,        ,    ASCII-    
}

Agora podemos escrever uma função simples para comparar duas mãos representadas na forma desta estrutura:

function compareHands(h1, h2) {
    let d1 = getHandDetails(h1)
    let d2 = getHandDetails(h2)
    if (d1.rank === d2.rank) {
        if (d1.value < d2.value) {
            return ""
        } else if (d1.value > d2.value) {
            return ""
        } else {
            return ""
        }
    }
    return d1.rank < d2.rank ? "" : ""
}

Verifique o flash


Isso é bastante simples, pois analisamos a linha de entrada e classificamos por naipes. Se o naipe da última carta corresponder ao da primeira - temos um flash.

const flush = suits[0] === suits[4]

Check Street


Rua não é muito mais complicada - se todas as cartas (independentemente do naipe) têm uma denominação consistente, então é isso.

Começamos com a primeira carta na mão e, para cada próxima verificação, se o seu valor nominal é consistente com a carta anterior. Usando o índice transmitido pelo retorno de chamada, da seguinte maneira:

    const first = faces[0].charCodeAt(0)
    const straight = faces.every((f, index) => f.charCodeAt(0) - first === index)

Definir duplicatas


Este passo é um pouco mais complicado. Devemos calcular a quantidade para cada valor nominal em nossa mão. E você também precisa de alguma maneira de determinar pares, triplos etc. para simplificar a classificação de suas mãos da seguinte maneira:

  • Contando a quantidade para cada valor nominal
  • Convertendo um registro em um envio no qual você pode pesquisar.

É necessário garantir que seja possível determinar a presença do “quatro” (quadrado) ou três na mão, o número de pares (não um, um ou dois), etc.

Portanto, primeiro calculamos quantas vezes os cartões de cada valor são encontrados:

function count(c, a) {
    c[a] = (c[a] || 0) + 1
    return c

Agora, temos todas as informações necessárias para classificar as mãos, até o momento, sem levar em consideração possíveis situações de empate.

Assim, um straight flush é a combinação mais forte com uma classificação de 1 (para não complicar, consideramos que um empate é possível se alguém tiver um royal flush), depois quatro do mesmo tipo, full house, etc.

Nesse caso, o operador && é usado de maneira fantástica , que varre o último valor se os valores nos colchetes anteriores estiverem corretos. Assim, (flush && direct && 1) retorna 1 se nivelado e direto forem verdadeiros, caso contrário, retornará falso.

Situação empatada


Se duas mãos tiverem o mesmo valor, o vencedor será determinado adicionalmente pelo valor nominal das cartas, se possível. Algumas regras estão associadas a isso.

  • Casal vs casal. Um par com um valor nominal mais alto vence. Se os valores corresponderem, a próxima carta mais alta vence. (Para dois pares, é resolvido da mesma forma).

    Por exemplo, se compararmos a mão “2H 2D AH KC 3D” com a mão “4H 4C JC TC 3H”, em seguida, quatro da vitória em segunda mão, embora na primeira mão a carta mais alta tenha o maior valor nominal - um ás.
  • Casa cheia versus casa cheia . O trio com o maior valor nominal ganha. Se as denominações de triplos em ambas as mãos são iguais, então analisamos as denominações de dois.
  • Agimos de maneira semelhante ao comparar qualquer mão com a mesma classificação - decidir o valor de face da carta mais alta em combinação.


Portanto, precisamos primeiro classificar por quantidade e depois pelo valor de face em nossa linha de saída. Lembre-se de que toda a sequência de cinco caracteres é usada para resolver casos com classificações de mãos correspondentes.

    let value = faces.sort(byCountFirst).join("")

function byCountFirst(a, b) {
    //    -  ,  
    const countDiff = counts[b] - counts[a]

    if(countDiff) return countDiff //   ,  
    return b > a ? -1 : b === a ? 0 : 1
}

const order = "23456789TJQKA"
function getHandDetails(hand) {
    const cards = hand.split(" ")
    const faces = cards.map(a => String.fromCharCode([77 - order.indexOf(a[0])])).sort()
    const suits = cards.map(a => a[1]).sort()
    const counts = faces.reduce(count, {})
    const duplicates = Object.values(counts).reduce(count, {})
    const flush = suits[0] === suits[4]
    const first = faces[0].charCodeAt(0)
    const straight = faces.every((f, index) => f.charCodeAt(0) - first === index)
    let rank =
        (flush && straight && 1) || /* -,     - -        */
        (duplicates[4] && 2) || /*  -     */
        (duplicates[3] && duplicates[2] && 3) || /* - -  +  */
        (flush && 4) || /*  -      */
        (straight && 5) || /*  -       */
        (duplicates[3] && 6) || /*  -     */
        (duplicates[2] > 1 && 7) || /*   */
        (duplicates[2] && 8) || /* -     */
        9 /*     ,      */

    return { rank, value: faces.sort(byCountFirst).join("") }

    function byCountFirst(a, b) {
        //    -  ,  
        const countDiff = counts[b] - counts[a]

        if(countDiff) return countDiff //   ,  
        return b > a ? -1 : b === a ? 0 : 1
    }

    function count(c, a) {
        c[a] = (c[a] || 0) + 1
        return c
    }
}

function compareHands(h1, h2) {
    let d1 = getHandDetails(h1)
    let d2 = getHandDetails(h2)
    if (d1.rank === d2.rank) {
        if (d1.value < d2.value) {
            return ""
        } else if (d1.value > d2.value) {
            return ""
        } else {
            return ""
        }
    }
    return d1.rank < d2.rank ? "" : ""
}
