パンダで大きなデータセットを処理する方法。Pythonと8GBのメモリを使用してFIASデータベースを操作します

FIASデータベースを具体的に提示する必要はありません。リンクを



クリックしてダウンロードできます。このデータベースは開いており、ロシアのオブジェクトのすべてのアドレスが含まれています（アドレスレジスター）。このデータベースへの関心は、データベースに含まれるファイルが非常に膨大であるという事実によって引き起こされます。したがって、たとえば、最小は2.9 GBです。 8 GBのRAMしか搭載していないマシンで作業している場合は、そこで停止してパンダが対応できるかどうかを確認することをお勧めします。対処できない場合は、このファイルをパンダにフィードするためのオプションは何ですか。

心の手で、私はこのベースに遭遇したことがなく、これは追加の障害です。そこに提示されているデータの形式は完全に不明確です。

ベースとともにfias_xml.rarアーカイブをダウンロードした後、そこからファイルを取得します-AS_ADDROBJ_20190915_9b13b2a6-b3bd-4866-bd1c-7ab966fafcf0.XML。ファイルはxml形式です。

パンダでの作業をより便利にするために、xmlをcsvまたはjsonに変換することをお勧めします。
ただし、サードパーティのプログラムやpython自体で変換しようとすると、「MemoryError」エラーが発生するか、フリーズします。

うーん。ファイルをカットして、パーツに変換するとどうなりますか？それは良い考えですが、すべての「カッター」もファイル全体をメモリに読み取ってハングしようとします。「カッター」のパスをたどるpython自体はそれをカットしません。8 GBでは明らかに十分ではありませんか？さて、見てみましょう。

Veditプログラム


サードパーティのveditプログラムを使用する必要があります。

このプログラムを使用すると、2.9 GBのXMLファイルサイズを読み取って操作できます。
また、分割することもできます。しかし、ちょっとしたトリックがあります。

ファイルを読み取るときにわかるように、ファイルには、特に開始タグAddressObjectsがあります。



そのため、この大きなファイルの一部を作成するときは、ファイル（タグ）を閉じることを忘れないでください。

つまり、各xmlファイルの先頭は次のようになります。

<?xml version="1.0" encoding="utf-8"?><AddressObjects>

そして終了：

</AddressObjects>

次に、ファイルの最初の部分を切り取ります（残りの部分の手順は同じです）。

veditの場合：



次に、[移動]と[行番号]を選択します。開いたウィンドウで、1000000などの行番号を入力し



ます。次に、選択したブロックを調整して、データベース内のオブジェクト全体を終了タグにキャプチャする



必要があります。次のオブジェクトにわずかなオーバーラップがあっても問題ありません。

次に、veditプログラムで、選択したフラグメントを保存します-ファイル、名前を付けて保存。

同様に、ファイルの残りの部分を作成し、選択ブロックの開始と終了を100万行ずつマークします。

その結果、サイズが約610 MBの4番目のxmlファイルが得られます。

XMLパーツを完成させます


次に、新しく作成されたxmlファイルにタグを追加して、それらがxmlとして読み取れるようにする必要があります。

veditでファイルを1つずつ開き、各ファイルの先頭に追加します。

<?xml version="1.0" encoding="utf-8"?><AddressObjects>

そして最後に：

</AddressObjects>

したがって、分割されたソースファイルの4つのxml部分ができました。

ファイルを1つに接着


Windowsでは、これは難しい作業になる可能性があるため、csvkitというPythonのコンソールユーティリティを使用します。Pythonモジュールとしてインストール：

pip install csvkit

*これは実際にはユーティリティのセット全体ですが、そこから1つ必要になります。

コンソールで接着用のファイルを含むフォルダーに入ったら、1つのファイルに接着を実行します。すべてのファイルにはヘッダーがないため、接着時に標準の列名を割り当てます：a、b、cなど：

csvstack -H fias-0-10.csv fias-10-20.csv fias-20-30.csv  fias-30-40.csv > joined2.csv

出力は完成したcsvファイルです。
