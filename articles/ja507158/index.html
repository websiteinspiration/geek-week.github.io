骨董品：Apple iPadの10年

2010年1月26日に、スティーブジョブズは記者会見でiPadタブレットを紹介します。 Appleの共同創設者は、ハードウェア、ソフトウェア、および最初のiPhoneで以前に機能したインターフェースに依存して、デバイスの新しいカテゴリを開きました。成功した試み：新世代のタブレットは、数年後に何百万部も販売され、どこでも使用されています。競合他社はAndroidプラットフォームで成功を繰り返すことを試みていますが、すぐには成功しません。


iPadは現代の話ですが、今年のAppleのタブレットは10年前のものです。つまり、私の条件付きカテゴリの「新しいレトロ」に該当します。記念日を記念して、私は古代のタブレットを購入してテストしましたが、ここでの経験はそれほど重要ではありません。誰もがそれがどのように機能するかをすでに知っています。 iPadやその他の最新のタブレットの歴史において、他に興味深いものがあります。 1つ目は、タッチスクリーンを備え、iPadのキーボードを備えていないウェアラブルデバイスで業界が取り組んでいる期間です。第二に、非常に有望なデバイスカテゴリが再びニッチになった速度。

私は古い鉄の収集家の日記を電報に保管しています。

私の最初のタブレットは、80年代に子供時代に登場しました。コンピューターではなく、紙のホルダーが付いたプラスチック製のA4パネルです。テーブルで「ドキュメントを操作する」方法がない場合に、外出先でメモを取るのに便利です。すべてのコンピューターデバイスの中で、タブレットはおそらく電池のない自然な記憶媒体に最も近い：本、雑誌、白樺の樹皮の手紙、エジプトの粘土のタブレット。一方、コンピュータタブレットの概念は、軍事研究所や民間の研究センターに由来するものではありませんでした。そして、サイエンスフィクション。


スタンリーキューブリックは、2001年の宇宙オデッセイ映画で「iPadの発明者」と呼ばれることがよくあります。この映画は1968年に撮影されましたが、お願いします。宇宙旅行者は、ビデオを再生する機能を備えた、非常に使い慣れた、薄型軽量の画面付きデバイスを使用しています。映画館では、キーボード、ディスクドライブ、パンチカード、ケーブルの負担なしに、視覚的に心地よい技術が求められています。それは非常に正確に推測されました。 iPadの前任者が同様の機能を思いついたのはいつですか？ 1989年までに：


GridPadのウェアラブルをして明らかに必要よりも早くリリースしました。それにもかかわらず、それはタブレットのための基本的な要件を実装します：ポータブルデバイス、タッチスクリーン、そしてキーボードの欠如。 1990年、GridPadには20メガバイトの内蔵ハードドライブ（および画面のバックライト）さえ搭載されました。彼らはなんとか1万個も売れたと言います。


3年早送りし、完全なグラフィカルインターフェイスを備えた商用デバイスを作成する最初の試みを見てください。レイジーゲームレビューチャンネルのビデオは、その時代の宇宙のようなものを示しています。ワイヤレスペンを搭載したタブレットで、Windows 3.xを搭載し、内部に386番目のプロセッサーを搭載しています。理想的には！。現代のタブレットの技術は、ややコンパクトなデバイス（PalmとApple Newtonハンドヘルドコンピューター、さらにはWindows Mobileを実行するPDA）で開発されています。いずれにせよ、典型的なコンピューターが完全に固定されていた90年代には、大衆はこれらすべてにアクセスできませんでした。価格が高いことと、本当に役立つアプリケーションが不足していることが原因です。通常のラップトップを購入する-それはすでに幸せでした。


マイクロソフトは2001年に2度目のタッチデバイスの国への出張を行いました。WindowsXPの特別バージョンのリリースと、タブレットPCのリリースプログラム全体の発表です。大雑把に言うと、「タブレット」は、上の図のThinkPad X61 Tabletハイブリッドなどのデバイスに正確に関連付けられています。手首を振るだけでタブレットに変身する、回転式スクリーンを備えたラップトップ。興味深いデバイスですが、信じられないほど重くて高価です。ペンでの管理のみ、静電容量式画面が最新のデバイスに表示されます。 「キーボードのないノートパソコン」は、外出先でフォームやその他の体の動きが必要とされる企業ではある程度成功しています。 10年の終わりまでに、完全にオリジナルの標本が表示されます。


2007年のソニーVAIO UXシリーズでは、非常にコンパクトなモデルが登場。 600グラム、4.5インチ画面、格納式キーボード、Intel Core Soloプロセッサ、フルWindows XP。ノートブックメーカーは、有望な市場ニッチを見つけようと、タブレットPCを大規模に実験しましたが、他の欠点の中でも、これらのデバイスはすべて、高価格で注目に値しました。それらは低迷して販売されました。SonyUX280などのデバイスは現在、高価な希少品です。価格の問題は、タブレットやネットブックに関連しないデバイスのカテゴリによって解決されました。


2007年に、優れたAsus Eee PC 701モデルがリリースされました。これは、顕微鏡画面を備えた安価でコンパクトなラップトップです。 1つの利点：価格、Linuxのバージョンごとに190ドル。その後、上の写真のEee PC 900モデルなど、通常のWindowsネットブックが表示されます。価格は、構成に応じて400〜600ドルです。それ以前は、そのような値札のラップトップはほとんどなく、重量が2キログラムまでの安価なコンパクトデバイスは確かにありませんでした。ネットブックは入手可能であり、Intel Atomプロセッサーのために機能が制限されていますが、長時間（大容量バッテリーで最大10時間）バッテリー電源で動作し、基本的なネットワークエンターテイメントに非常に適しています。映画を見たり、新しく登場したソーシャルネットワークでチャットしたりできます。インターネットはまだ90年代後半の平均的なスーパーコンピューターのリソースを必要とせず、Webページのサイズはそれぞれ20メガバイトではありません。あなたはネットブックで行うことができます。


最初のApple iPadはネットブックを攻撃します。 Steve Jobsは彼のお気に入りのスタイルで、ネットブックは「実際には何もできない」と発表しました。もちろん、彼はまったく正しくありません。これは、多くのタスクではるかに便利なデバイスを示しており、これは本当です。 1024x768の解像度の10インチ画面。 2010年のiPhone 4で使用されているものと同様のシングルコアApple A4プロセッサー（ただし、周波数が少し高い-1 GHz）。 256メガバイトのRAMと16〜64ギガバイトのデータストレージ。ベースモデルの価格は500ドルで、プレゼンテーションのメイントピックとなる価格です。


しかし、それは手頃な価格だけではありません。 AppleはiPadの所有者に無料で自分のiWorkオフィススイートを提供しています。大画面のタブレットに投獄された他の開発者のソフトウェアを発表します。安価なデバイスで初めて、このような「高価な」テクノロジーは、広視野角のIPSスクリーンとして実装されます。初めて、厚いバッテリーを使わずに10時間のバッテリー寿命を実現しました。 Windows XPとWindows 7のように、AppleはPCソフトウェアとの下位互換性を提供する必要はありません。システムとすべてのプログラムのインターフェースは、最初は指による操作と互換性があり、スタイラスは必要ありません。はい、iPadの成功は多くの点で以前にiPhoneでテストされたソリューションに基づいています。そのような面白い画像が表示されます：


Appleはしばしば、革新の欠如と言われていることを批判しようと試みます。外出先での作業用のタッチスクリーンとコンパクトデバイスの両方。おそらく、しかしiPadは以前のデバイスのいくつかの深刻な問題を間違いなく修正した。Appleは、そのようなデバイスを本当にコンパクト、軽量、使い勝手がよく、バッテリー電源で長持ちさせる方法を示しました。必要なのは、過酷な遺産を取り除くこと、「大画面の電話」を作ることだけでした。わかりました、でもなぜこれまで誰もこれを考えなかったのですか？

古代のタブレットを使った実験


オリジナルのiPadは購入しませんでした。代わりに、2011年モデルのiPad 2を購入しました。概して、このバージョンでは、オリジナルのいくつかの欠陥（凸状の裏表紙など）を修正し、より強力な鉄を追加しましたが、これは一般的に初期のタブレットの特徴です。この特定のモデルを購入する理由はさらに2つありました。まず、Apple A5をベースにしています。これは、ARMアーキテクチャ上のデュアルコアプロセッサであり、2020年に少なくとも何らかのパフォーマンスの可能性を提供します。さらに、この特定のデバイスは私の最初のタブレットであり、9年前の最初のAppleデバイスでした。

現在からの様子


2020年には、デバイスの時代はすでに感じられています。最近は目新しいもののようですが、今では重く、通常の解像度の画面（Retinaではない）は珍しいです：個々のピクセルを見ることができますが、どうですか？スピーカーは大音量ですが1つですが、私たちはまともなステレオサウンドに慣れています。最も重要なのは、iPadがネットワークサービスにしっかりと関連付けられているデバイスであることです。それらは常に進化しており、古代のプロセッサーを搭載した古いデバイスは徐々にそれらに対処することをやめます。その結果、2つの状況が発生します。iOSインターフェースは通常どおり「飛ぶ」ことになります。 App Storeが遅くなります。 Safariブラウザは率直に速度が低下し、一部のサイトが表示されません。


iOSアプリのエコシステムは、2015年以降64ビットアーキテクチャに移行しています。しかし、これがなくても、すべての開発者が古いデバイスをサポートする強さ、時間、欲求を持っているわけではありません。このため、上記のスクリーンショットのように、5年前のOSを使用していても、iPad 2の多くのアプリケーションでエラーが発生します。Androidよりも対処が少し難しく、レトロなアップルファンボーイが苦しんでいる3つの段階を特定できます。

  1. アプリケーションは問題なく古いデバイスでサポートされています。幸運ですね！公開時のiPad 2とiOS 9.3の例は、Telegramメッセンジャーです。
  2. アプリケーションは古いデバイスではサポートされていません。インストールすると、上の図のようにエラーが発生します。しかし、ライフハックがあります。すでにプログラムを以前にダウンロードしている場合は、「購入」で古い（最新の互換性のある）バージョンをダウンロードできます。別の方法：iTunes経由でコンピューターからアプリケーションをダウンロードします。

  3. あなたがそれを購入またはダウンロードしない限り、アプリケーションはサポートされておらず、何もまったく機能しません。この場合、脱獄が必要であり、動作する古いバージョンのアプリを検索します。


以前に使用したアプリケーションのアーカイブが大きく、必要なほぼすべてのものをそこから取得することができたため、私はまだ第3段階に到達していません。このような互換性レイヤーの存在は、古代の愛好家の観点から見たAppleの大きな利点です。確かに、Androidでは、目的のバージョンのAPKを簡単かつ簡単にコピーすることで問題が解決します。iPadはプラットフォームの近さについても批判されました。それでも、カメラやメモリカード用のアダプターが製造されましたが、これはAndroid（特にWindowsのラップトップ）の機能には不十分でした。古いアプリケーション-完全に動作するGoogle Chromeブラウザバージョン63（2017）と動作するYouTubeクライアント。

過去の様子


2011年には、そのような問題はありませんでした。iPad2は当時のWebの機能と一致していました。彼は実際に妻のために購入しましたが、その結果、私たちはそれを共有することができませんでした。後で自分のタブレットを購入しました-Lenovo ThinkPad Tablet。そのレビューはHabréで公開しました。 「購入を後悔しなかった」と書いてありますが、まさにそうです。しかし、iPadと比較して、アプローチには顕著な違いがありました。初期のAndroidタブレットは、互換性のあるソフトウェアの同等のセットを提供できませんでした（そしていくつかの人気のあるアプリケーションは今でも大画面用に最適化されていません！）、そして急いでリリースされたAndroid 3 Honeycombの品質望まれるべき多くを残しました。 Androidタブレットのメーカーは機能を競おうとしましたが、多くの場合、ラップトップのサイズのデバイス（キーボード、USBポート、HDMI出力、メモリカード！）を手に入れました。スマートフォンやタブレット向けのユニバーサルOSであるAndroid 4のリリースにより、状況は改善し始め、プラットフォーム間にそれほど大きな違いはなくなりました。

離陸と暗がり



Apple iPadは個人用デバイスの市場に深刻な影響を与えました。ネットブックは地平線から姿を消しました。iPad2がすぐに販売される直前に購入したLenovo S10ネットブック-もはや必要ありません。 Androidデバイスベンダーは、独自のタブレットの開発に真剣に投資しています。 Windows 8デスクトップオペレーティングシステム全体が、タッチスクリーンがなかった場合でも、「指指向」のインターフェイスを課そうとしました。全員をタッチスクリーンに転送するこの試みは長続きしませんでした。 Appleにとって、iPadの販売に関しては2014年が最も成功した年でした。その後、上記の一般的な統計からわかるように（ここから）、供給がゆっくりと減少し始めました。


どうやってそうなった？ iPadのリリースとともに、大きな対角線を持つスマートフォンの流行が始まりました。スマートフォンは、（当時）5.3インチの巨大な画面を備えたSamsung Galaxy Noteスマートフォンによって設定されました。最新のGalaxy Note 10+の対角は6.8インチです。元のiPadの10インチと比較すると、これは別のデバイスの購入を正当化するほど大きな違いではありません。ゼロの始まりのタブレットは、コンテンツを消費するための、つま​​り先延ばし用のデバイスとして優れていることがわかりました。ソーシャルネットワーク、ビデオ、ゲームなど-これはすべて問題なくスマートフォンで利用できます。なぜ別のデバイスを購入するのですか？


その結果、タブレットは2007年の状態に戻りました。大型のタッチスクリーンを必要とするユーザーのために、特殊なデバイスのニッチ市場に参入しました。安価なユニバーサルiPadは引き続き販売および更新されていますが、2015年以降、13インチディスプレイを搭載したモデルがリリースされ、本格的なラップトップとの競争力を高めています。価格とパフォーマンスの両方で。 iPad Pro 2020はラップトップコンピュータとして位置付けられています。これはアーティストにとって素晴らしいタブレットであり、写真処理に適したポータブルデバイスであり、キーボードを備えています。これはドキュメントを操作するための優れたマシンです。しかし、それに応じてコストがかかります。テラバイトのストレージを備えたトップエンド構成では最大1800ドルです。

当初ミニマリストだったiPadは、キーボードを持ち帰っただけではありませんでした。スタイラスが再び現れましたが、スティーブジョブズは2007年に私たちを排除しようとしましたが失敗しました。機能、価格、パフォーマンスの点で、iPad Proはラップトップに代わるものです。以前は考えられなかった機能がソフトウェアに追加されました-マウスサポート、マルチウィンドウモード。私を含め、タブレットが常に最良の選択であるとは限りません。AndroidとiOSには古いデバイスがありますが、新しいデバイスはありません。キーボードワーカーにとっては、クラシックなラップトップの方が快適です。 iPadは進化しており、かつては誰にとっても普遍的なデバイスであったかつてのニッチを見つけようとしています。しかし、これらの明るい時代は過去にあります。


私のコレクションでは、iPad 2は古いデバイスと最新のデバイスの中間にありました。 Windows MobileおよびSymbian zeroソフトウェアストアが機能しない場合、ネットワークサービスは無効になります。 iPad 2ではすべてが機能しますが、すべてのアプリケーションが利用できるわけではなく、タブレットは関連するほとんどのアプリケーションに対応できません。鉄の要件が増加し、新機能の数に比例して増加しました。古い、多くの場合アップデートされていないソフトウェアが機能し、私は古いiPadを時間通りに購入したことがわかりました。

今後もアプリケーションストアへのアクセスが継続され、ウェブのサポートが完全に失われる可能性があることは事実ではありません。そして、インストールされたアプリケーションは動作します。これは間違いなく私のコレクションのデバイスの新しいカテゴリです。これらはネットワークサービスに関連付けられているため、事前に構成して棚に置くのが最適です。これらのサービスが発展するにつれ、鉄の欠片が存在することが判明する可能性がありますが、脱獄やその他の娯楽を考慮に入れなければ、それで何も実行できません。ヴィンテージソフトウェアのセットを今すぐ作成することをお勧めします。その後、それは遅すぎるかもしれません。
