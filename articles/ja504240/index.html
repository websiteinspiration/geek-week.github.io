パート6。キャリアプログラマ。独自のスタートアップ

ストーリー「キャリアプログラマー」の続編。

「プログラマーのキャリア」という本に1年間触れなかったが、どうやら無駄ではなかったようだ。過去数か月間、私はこの物語をIT企業の作成および開発方法に関する資料で補足する多くの変革を経験しました。当たり前のようです。あなたが野心的なプログラマーまたはフリーランサーであるなら、遅かれ早かれ、あなたは自分自身に尋ねます：「次は何？」。今、私はすでに上級開発者です-それから何ですか？アーキテクトに任命された-次は何？..

しかし、私たちが中断したところに戻りましょう。最後のパーツ番号5。プログラマー結局、フリーランスの開発者としての私のキャリアを終え、スタートアップの作成に私のすべての節約を投資するつもりでした。最後の第5章の発行以来、私は本の継続について尋ねるHabrで、そしてソーシャルネットワークでさえ数十のプライベートメッセージを受け取りました。Habrの多くの読者にとって、この読みは有益で有益であることがわかりました。そして、それぞれの事業が完結する必要があり、その事業に対する需要があるため、私は自分の経験を共有し続けなければなりません。

この章では、次のことを学びます。

  • ITプロジェクトのクレイジーなアイデアはどのようにして生まれ、これの前提条件は何でしたか
  • すべてを落として自分のスタートアップやビジネスを創り出す人の動機について。さらに、彼は快適で収益性の高い仕事を投げかけるだけでなく、居住地や環境を完全に変えます。
  • スタートアップ資本を調達する方法
  • チームをゼロから作成することについて
  • プロジェクトの開発と開発における創設者の役割について
  • 投資の検索とそれが実際にどのように行われるかについて
  • 間違いと結果について
  • そして、この人生の章の比較的ハッピーエンドについても

この章では、2011年末から2013年末までの期間について説明します。

考え


子供の概念は情熱から始まります。山を転がすことができる超力に変換される手に負えない、圧倒的な感情で。この力はとても大きいので、新しい命を与えます。地球に新しい居住者を紹介します。しかし、赤ちゃんが生まれる前に、9ヶ月の準備と有機的成長が経過しなければなりません。小さな細胞から、調和的に相互接続された10億個の他の細胞からなる非常に複雑な生物への成長。ご存じのように、9人の母親が1か月で赤ちゃんを「解放」することはないため、このプロセスと、ITプロジェクトを構築するための鍵は、調和のとれた一貫した成長です。

構想と子供の誕生のこの並行性が、アイデアの誕生からパブリックアクセスの最初のバージョンのリリースまで、初心者のIT起業家のおおよその気分を伝えることを願っています。

私の場合、私のスタートアップのアイデアは、最初のプログラムを書いた瞬間から、いつも頭の中にあるように思えました。では第4章、私は大学で私が知識によってRSDN.ruでひげを生やしたおじさんに到達するために、私自身のC ++コンパイラを開発したと述べました。このストーリーについては、別の記事「C ++コンパイラを書いたとき」で詳しく説明しています。

仲間のプログラマーのために傑出したものを作成するという考えに夢中になって、私はノートブックにお気に入りのIDEで特定のプロセスを自動化する方法をよく書いていました。また、デスクトップ上にファイルがあり、作業中に発生したコードを作成するための日常的なタスクをすべて書き出しました。

1000回の繰り返しサイクルかどうか

for(int i = 0; i < items.count; i++)

または、C ++でGet / Setメソッドを作成します。これらの毎日繰り返されるすべてのアクションは、開発ツールが完全ではないことを示唆しています。

さらに、「そして、将来のプログラミングはどのようなものになるのでしょうか？」だから10-20年で。

2010年の初めにそれを考えると、10年後には、マシンがプログラムを自分で作成し、開発者がフィルム-パスワード「Swordfish」に示すように透明なモニターの近くで手を振ると思いました。


10年が経過し、人工知能と機械学習技術が人気を博しています。Kinectのような非接触タッチコントローラーのように。しかし、私はVisual Studioでコードを書いているので、書いています。バージョン番号のみが変更されました。過去10年間で画期的なことはありません。

スタートアップ資本


しかし、2011年末に戻ります。当時、私はフリーランスの開発者として2つのプロジェクトに同時に熱心に取り組みました。私の目標は、アパートの資金をすばやく調達し、両親から自宅に引っ越すことでした。そして、ソファの下に置いたお金が多ければ多いほど、自分自身と明日への自信が増しました。今では5,000ドルが貯まっています。私はクレジットでさえ、あなたがこのタイプの価格で購入できるアパートの種類を検討し始めました。公平に言うと、アパートを購入するという考えは、両親から押し付けられました。それ自体は、私を明るくしませんでした。その時までに私はすでに車を持っていたので、私は自分のビジネスを作成し、それにスタートアップ資本を蓄積することに目を向け始めました。

私は自由な時間に、ネットワーク上でビジネスを作成することについて、当時のすべてを見て読みました。情報はほとんどありませんでした。私のお気に入りのリソースはZuckerberg Calls（現在はvc.ru）で、後にIT起業家の分野で影響力のある人々に導かれました。

しかし、そのブログのページやソーシャルネットワークに表示されるすべてのものは、私の地方都市から非常に離れているように見えました。遠く離れているので、誰かが何万ドルもの投資をしたかと誰かが言っていたとき、私は映画を見ているように見えました。

それから私は、1960年代初頭のフルシチョフによって四方が整えられた地方の中庭に出かけました。憂鬱な木々と憂鬱な人々と。そして、私に残されたすべては、私の目を閉じて、未来の私のスタートアップが何であるかを想像し始めることでした。百万の画期的なアイデア。

私は歩いて夢を見ました。途中で、ホームレスの人々が移行期に遭遇したとき、タッチインターフェースを備えたデバイスの可能性について考えました。数本の指で一度に画面をひねり、架空のギアを回す方法を想像したところ、幸福感とソフトウェア開発の世界に似たものを作成したいという欲求が生じました。

2年足らずの努力で、20,000ドルも節約できました。私は何も買わず、食べ物は安く、ほとんどの場合両親が買っていました。私は毎月の請求書と私の家族を持っていませんでした。したがって、フリーランスで稼いだものの95％は、ソファの下で直接折りたたまれています。私はこのような累積的なマラソンを人生で2回以上行っています。これには、多くのアカウントと大家族がいる今も含まれます。

このような節約により、最終的にITプロジェクトを開始する必要があるという結論に達しました。

アイデアから実装まで


だからクレイジーなアイデアがありました。購入した新しいiPad 3、映画「アイアンマン」に触発され、数年の開発経験を持っている私は、ビジュアルプログラミング環境を作成することにしました。簡単に言えば、タッチインターフェイスを備えたIDEです。アイデアがあり、機能についてのメモを書き始めました。

UI / UXの基本は、コンポーネントを結合
し、プログラムを構成するブロックをクリックして詳細を掘り下げながら、コンポーネント、クラス、メソッド、インターフェイスをIDE空間全体に移動する機能でした。

それはこのようなものでした：


ビジュアルプログラミングのアイデアは、衝撃的なものでした。彼女は想像力を大いに刺激しました。しかし、そのような手段が生産性の点で本当に役立つほどになることはできませんでした。それにもかかわらず、その時の私のスローガンは、「主なことは、行うこと、そして何が起こるかであること」でした。

チームでアーティストの検索を開始した後、RSDN.ruサイトから何人かの開発者に出会いました。彼らは開発を始める準備ができていました。しかし、創設者は開発に従事する必要がないという致命的な考えが私に課されました。実際、私は本当にしたくはありませんでした。本当のビジネスを作り出し、投資を増やすことをもっと心配していました。すべてを他の成功した新興企業と同じくらい美しく豊かにする。そのとき私が下した決定の多くは、直感と自分のエゴに基づいていました。学ぶ人も学ぶ人もいませんでした。

どういうわけか、私は最初の人をスタートアップチームに雇い入れました。私は彼にテストタスクを与えました-画面の周りでブロックを移動して矢印でそれらを接続することを可能にするiPadのためのシンプルなアプリケーションを作ることです。候補者はすぐにこのタスクを完了し、1時間あたり10ドルという少額のレートで、遅滞なく採用されました。

さらにもっと。私はその「ああ、奇跡！」を見て、開発者が私に彼に与えるタスクを実行します。そして、私はoDeskからより多くの人を雇うことに決めました。したがって、昨日の私のクレイジーなアイデアから、あらゆるプラットフォーム向けの8人の開発者の会社がすぐに形成されました。

私の計画は、デスクトップ、Web、およびモバイルアプリケーションのプロトタイプを作成し、それらを投資家に提供することでした。その瞬間、私は別の方法を知りませんでした。しかし、ロシア語を話すスタートアップコミュニティでは、他の方法を学んだ人はいませんでした。情報スペースの表面には、プロトタイプを作成し、ビジネスプランを作成し、お金を要求する...利益が1つありました。

ネットワーキングを選ぶ理由


チームは私のリーダーシップの下でプロトタイプを開発しました。並行して、私は投資を誘致することを目指し始めました。投資の規模は私には不明であり、私は20〜25,000ドルを超えることはありませんでした。したがって、最初はオープンソースで公開された数値に基づいていました。そしてしばらくして、私はビジネスプランを書いてみましたが、受け取った50万ドルから愚痴に陥り、開発年の8人分の費用を支払うだけでした。

クレイジーなアイデアの下でそのような投資を探すことはそのようなアイデアでした。おそらく、パロアルトのどこかで、それは酔った億万長者でうまくいったでしょうが、それらの年のCISではうまくいかなかったでしょう。別の話がありましたが...

両親と一緒に暮らしている間、私は直感的に、少なくとも何らかの形でITスタートアップへの投資と交差する、または自分で始めるすべての人と知り合うことを求め始めました。当時、ウクライナではそのような出来事はほとんどありませんでした。 2012年の夏の間、私は国の反対側で開催された2つのイベントだけに出席しました。

しかし、これにより、以前は未知だったレベルに触れることができました。人格発達のレベル。 ITパーティーの人々がどのように振る舞ったか、彼らがどれほど自信を持っていたか、彼らからどのようなクレイジーなエネルギーが生まれているかに刺激を受けました。それはすべて屋根を吹き飛ばし、完全に別の世界からのものでした-覗き窓を通して。この世界には、国内の麻薬中毒者、おばあちゃん、そして私が26年間住んでいた暗い都市はありませんでした。

当時、ソーシャルネットワークの友達としてまだリストされている多くのITビジネスマンに会いました。その中には、VKontakteの創設者、ウクライナとロシアの当時の大臣の親戚や友人、ITに多くの億万長者の投資家がいました。

昨日の学生のように見えたにもかかわらず、「私のチームはタッチインターフェイスを備えたプログラミング環境を作る」以外には何も提供できませんでした。それにもかかわらず、この世界の多くの有力な人々が連絡を取り、少なくとも名刺を与えました。そして、サードパーティを介して会議を組織することができた場合は、アイデアを数分間提示できます。

マイクロソフトのサポートを探しています


知人のネットワークと環境は、すべてではないにしても、多くを決定します。

アクティブな投資検索フェーズ


2012年、私は憂鬱な居留地からキエフに引っ越しました。すべてのブリッジと通信を焼きました。フリーランスで完成したプロジェクト。彼は自分の物とコンピュータを市松模様のバッグバッグに詰め、両親から離れて明るい未来を作りました。キエフでは、知り合いがほとんどいなかったので、すべて自分でやらなくてはなりませんでした。難しいと言うのは何も言わないことでした。そのような野生の出来事は私に起こり、それを公の場に出すことは自殺になるでしょう。救急車を頻繁に呼ぶ必要があったとしか言えない。そしてまた、私のアパートの女将は、あらゆる種類のわき柱のために、何度も私をそこから追い出したいと思っていました（結局のところ、私は以前一人暮らしで、外出先で多くのことを学びました）。

しかし私はキエフに住んでいましたが、誰も私と私の親戚を信じていなかったので、常にラスベガスに引き戻されました。今日は逆に、親戚にアパートを買って、引っ越してくれるように呼びました。主なことはあきらめないことです。

ですから、その時に必要なのは私のコンピューター、机、椅子だけでした。最初に借りたアパートにはテーブルがなかったので、小さなodnushkaのベッドとテレビの間にかろうじて収まるサマーコテージの折りたたみオプションを見つけなければなりませんでした。

しかし、移動のアイデアは、徒歩圏内にある適切な連絡先と投資を見つけることでした。それで、2012年9月から2013年2月までの間に、約20人の投資家を回って私のプロジェクトを紹介しました。カンファレンスやイベント、ソーシャルネットワークやニュースで見つけました。 Facebookを通じて直接連絡。失うものは何もありませんでした。蓄積が尽きても後戻りはできないことをはっきり理解しました。

それは感情的な死です。私は戻ることはできませんが、キエフに滞在するためにお金はありません。結局、フリーランスも終わりました。

イギリスでのジュリアス・シーザーのキャンペーンに触発されました。島に陸軍を上陸させたとき、彼は兵士たちが逃げる機会を持たないように船を燃やすように命令しました。

そして兵士たちは島を捕らえました。実行する場所がなかったので、私も同じような状況にありました。自分をやる気にさせることができれば、これは可能な限り行われました。全部かゼロか。

しかし、現実は少し異なります。そして、投資を受け取り、世界規模でスタートアップを立ち上げたいという普遍的な願望にもかかわらず、お金は私たちの目の前で溶けていました。

そして今、私はすべての節約から数百ドル残っています。開発者が支払うものは何もありませんでした。アパートを借りるのに支払うものもありませんでした。お金もありませんでした。ちなみに、キャッチしてすごい効果が出る試作品はありませんでした。 「夜明け前の最も暗い時間」の碑文が貼られた紙をテープで壁に貼り付け、闘争を続けました。

絶望から、私はキエフでビジネストレーニングを実施した部外者に絶対に目を向け、ITとは何の関係もありませんでした。私は彼にiPadのデモをステージのすぐ隣の千人の前で見せた。約10分間話をしましたが、聴衆は期待を失いました。結局、彼は物事がどのようになっていたかについて評決を下しました。私はそれに投資しません。」

しかし、これは私を止めることはありませんでした、私は歩いて、そして何が何であるかを提示し続けました、それは未来が脅かされているという事実にもかかわらず。 ITに優れた人々が実践的なアドバイスをしてくれたので、それを聞いてみると、すべてが異なる可能性があります。誰かが子供向けにアプリケーションを適合させ、この方法でプログラミングを教えるようにアドバイスしました。この考えは、攻撃的ではないにしても、ワイルドであるように思えました。
何の子供？あなたは何者ですか ？！ソフトウェア開発の分野で革命を起こしたいのですが、おもちゃの開発を勧められますか？はい、はい、何年も経って自分を分析するのは良いことです。

完全に開発をやめても、密室で殴り続けた。彼は銀行からマイクロローンを借りて、アパートを借りました。両親が私に食べ物を送ってくれたか、将来の妻が連れてきてくれました。したがって、私は破産していて、すでに拒絶から精神的に落ち込んでいましたが、それにもかかわらず、彼らは新しく開かれたスタートアップアクセラレータから私を呼び出し、インタビューに招待しました。

そして実際には彼らは言う：「すべてがうまくいく-あなたは病気になるだけだ」感情的にも金銭的にも、私はゼロでしたが、自分の会社の潜在的なオフィスに足を踏み入れる強さがわかりました。そして良い知らせが私を待っていました。アクセラレーターおよびパートタイム投資家の創設者は、MySQLおよびGitHubの創設者と十分に知り合いでした。そして、可能であれば彼は私に彼らを紹介することができました。

また、このインキュベーターには約100万ドルの資金が集まり、有望な開発への投資に喜んでいました。結局、私はインキュベーターの最初の居住者の一人とみなされ、あらゆる方法で協力することを約束しました。しかし、私は何かに住み、家賃と食費を払わなければなりませんでした。さらに、チームはプロジェクトに長期間取り組んでいません。しかし、それでも私はこれらのITの世界からの影響力について知り、彼らから十分な知識を借りました。

合計


これらのジェットコースターにもかかわらず、人間の能力の限界で、私はこれらのクレイジーな半年のために多くの前向きなことを手に入れました。IT起業家の世界から多くの有名人に会いました。スタートアップの作成と開発のプロセスを理解し、多くの経験を得ました。

また、妻を与えたのはキエフへの引っ越しでした。そのおかげで、私はその困難な時期にようやく溺れませんでした。彼女のサポートと手助けが私をこの地獄から引き上げました。

そして8年後、冷静に知識と連絡を取り、ITビジネスを再開しました。前章でお話ししましょう。
