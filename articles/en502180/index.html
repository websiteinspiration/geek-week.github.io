Microsoft: Zero Trust. Secure Remote Workplace


May 12, 2020 / 10:00 - 11:30 (GMT +3)

Let's talk about the Zero Trust concept and its application in remote work scenarios:

  • Why is the traditional network perimeter paradigm not effective these days?
  • What is deperimetrization and software-defined perimeter.
  • Principles, architecture and components of the
    Zero Trust model .

Practical implementation of the Zero Trust model based on Microsoft solutions: Azure AD, Azure Identity Protection, Azure MFA, Conditional Access, Azure Application Proxy, Cloud App Security, Azure Sentinel.

Speaker:  Artyom Sinitsyn  (CISSP, MCSE, MCITP Information Security Program Manager in Central and Eastern Europe, Microsoft)

Softline: Microsoft 365. Security is more affordable than it sounds


May 18, 2020 / 10:00 - 11:30 (GMT +3)

At the webinar we will demonstrate the solutions Conditional Access + MFA, Office 365 ATP, Intune, Microsoft Defender ATP. And also we will tell how Microsoft technologies can reduce risks when working outside the corporate network.

  • Getting full access using "conditional".
  • Secure links and attachments in letters.
  • Remote work from mobile devices.
  • Endpoint Protection.
