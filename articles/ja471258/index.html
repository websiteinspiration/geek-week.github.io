データストレージ/ローディングシステム

みなさん、こんにちは！

おそらくUnityでゲームを作り始めたすべての人は、ゲームの進行を保存する方法のジレンマに直面していました。今日は私の方法についてお話します。彼は新しい人ではありませんが、かなり働き者です。

興味があれば、猫へようこそ。

データの保存方法について考えたとき、データをJSONでシリアル化するのが便利だという結論に達しました。別の自転車を作らないようにするために、人気のあるJson.NETライブラリ-Newtonsoftを使用することにしました。さらに、オブジェクトとそのタイプを保存できます。

現時点では、データアクセスクラスは静的です。つまり、どこからでもアクセスできます。短所-個別に保存することはできません。

コードを少し分析しましょう：

public abstract class ISaveItem
{
    public string Key { get; set; }
}

書くことは不可能なので：

Dictionary<string, SaveItem<T>>

SaveItemをISaveItemでラップします。このようにして、さまざまなタイプのデータを含む辞書を作成できます。

public class SaveItem<T> : ISaveItem
{
     public T item;
}

private static Dictionary<string, ISaveItem> items;

UnityエンジンのVector2、Vector3、Color、またはその他のデータ型をシリアル化しようとすると、ループエラーが発生する可能性が高くなります。Json.NET-Newtonsoftでこの問題を回避するために、これに対する解決策があります。コンバーターの作成です。それらの1つを分析しましょう。

クラスを作成し、JsonConverterから継承します。

public class Vector2Converter : JsonConverter

コンストラクターとプライベートフィールドを作成します。

private readonly Type type;
public Vector2Converter(Type type)
{
     this.type = type;
}

メソッドを再定義した後：

public override bool CanConvert(Type objectType)
{
    return type == objectType;
}

public override object ReadJson(JsonReader reader, Type objectType, object existingValue, JsonSerializer serializer)
{
    return null;
}

public override void WriteJson(JsonWriter writer, object value, JsonSerializer serializer)
{
    if (value.GetType() == type)
    {
        var vector = (Vector2)value;
        writer.WriteStartObject();
        writer.WritePropertyName("X");
        writer.WriteValue(vector.x);
        writer.WritePropertyName("Y");
        writer.WriteValue(vector.y);
        writer.WriteEndObject();
    }
}

WriteJson メソッドで最も重要なこと。ここにデータが保存されます。

そして今私の保存システムを使用する方法：私

たちが書いたデータを保存するには：

int myInt = 33;
SaveSystem.Set("MyInt", myInt);

書き込むデータを読み取るには：

var myInt = SaveSystem.Get<int>("MyInt");

理論的には、複雑なデータ構造は簡単に保存でき、これでうまくいくはずです。より複雑なオブジェクトを保存して読み取ってみましょう。

[System.Serializable]
public class Test
{
   public string Name;
   public float FloatParam;
}
    
Test test = new Test() {Name = "TestValue1", FloatParam = 0.1f};
    
//
SaveSystem.Set("MyTestClass", test);

//
var saveTest = SaveSystem.Get<Test>("MyTestClass");
Debug.Log(saveTest.Name);
Debug.Log(saveTest.FloatParam);

ご覧のとおり、すべてがうまく機能しています。

保存、インポート、エクスポート、または削除を編集することもできます。



エディターは通常の型（int、float、stringなど）として表示できます。より複雑です。
