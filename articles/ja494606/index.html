月と惑星を観測する方法

月や惑星を観察することは非常に興味深いです。惑星の観測は露光によって妨げられることはなく、都市から直接観測することができます。惑星を観察するために、広い視野を持つ接眼レンズは必要ありません。安価なPlessla接眼レンズでさえ、生産的な目視観察結果を提供できます。



木星、土星、火星は、おそらく天文観測で最もアクセスしやすい惑星です。 80mmビッグスクールリフレクターで20年以上前に見た土星を初めて見たときのスリルと驚きを今でも覚えています。しかし、初心者のアマチュアから、特に木星と火星の最初の観測についての報告がしばしばあります。 「細部のない光の球だけが見える」または「焦点を合わせることができない小さなディスクが見える」 「私の望遠鏡は故障していますか？」この記事が役立つかもしれないのは、初心者の天文学愛好家の日です。それは太陽系の惑星の視覚的観察の機微と特徴を詳細に説明します。

惑星は空の光の点ですが、月は大きく、非常に明るいです。ただし、月には多くの細部があるため、惑星を観察するのと同じ手法を使用する必要があります。望遠鏡で最高の画像を取得するために考慮すべき重要な要素がいくつかあります。

増加する


最も物議を醸す要因。惑星は小さいので、増加が大きいほど良い！？あんまり。望遠鏡に最適な倍率を使用する必要があります。それを見つける最も簡単な方法は、望遠鏡の最適な射出瞳を計算することです。射出瞳は、望遠鏡の接眼レンズを通して見る焦点の合った画像のサイズです。

射出瞳は次のように計算されます。望遠鏡のレンズの直径（mm）を、接眼レンズの倍率で割ります。望遠鏡の倍率は、レンズの焦点距離（mm）を使用する接眼レンズの焦点距離（mm）で割って計算されることを思い出してください。

望遠鏡のレンズの焦点比（F / D）は、次のように計算されます。レンズの焦点距離をその直径（開口部）で割ります。

人間の目では、1 mmの射出瞳が、明るい物体に最適な解像度を提供することがわかります。焦点距離が900 mm、焦点比がF / D-10の90 mm屈折器があるとします。この場合、月または惑星の最適なビューを取得するには、10 mmの接眼レンズを使用する必要があります。 F / D-5の場合は5 mmの接眼レンズ、F / D-8の場合は8 mmの接眼レンズなどを使用します。この増加を使用して、ほとんどの夜は惑星の美しい景色を楽しむことができます。

解像度は、望遠鏡のレンズの直径（大きいほど良い）と視認性の2つの要因に依存します。可視性（歌う）は大気の安定性の尺度です。安定している場合は、詳細が表示されます。大気の乱気流が多い場合、細部は「洗い流され」ます。視界が悪い場合、10インチの望遠鏡では4インチ以上は表示されません。実際、小さなツールは悪い雰囲気で少し良くなります。また、地表から熱源（たとえば、屋根）からできるだけ離れて観測を行うと、「イメージストリーミング」の悪影響を減らすのに役立ちます。ソビエト文学では、少なくとも300mを登ることをお勧めします。海面から丘の頂上、ピエモンテ台地などに至るまで、表面の熱層の画像への悪影響を排除します。しかし、あなたは知る必要があります自立した丘の頂上は、空気の乱気流のため、適切な選択ではないでしょう。


月とほとんどの惑星は非常に明るいです。多くの場合、接眼レンズの強い照明、焦点面にレンズを構築する明るいスポットでは、細部が失われます。それを制御するには？最も簡単な方法は、光害を作成することです。夜の目の順応は、月や惑星の観測に関して逆効果です。ポーチ、バルコニー、または観察している他の場所のライトをオンにします。さらに良いことに、空がまだ青いときに注意してください。日没直前に見た木星の最高の景色。これで十分でない場合は、レンズの前にある開口部を使用するか（特に、特別なフィルターがない場合は月に推奨）、フィルターを使用できます。絞り設定は、焦点比F / D-4 ... F / D-6の高速望遠鏡に非常に効果的です。F / D-8 ... F / D-15のように、アパーチャが小さい、アパーチャの少ない機器の場合、解像度が低下するため、これはお勧めしません。フィルターはより効果的です（後でフィルターを選択する際の詳細）。

光散乱


光散乱は、月、惑星、または星の明るい光が望遠鏡のガラス表面に当たると発生します。散乱効果は、まぶしさ、コントラストの低下、解像度の低下に似ています。残念ながら、フィルターで光の散乱を制御することはできません。これに対処する唯一の方法は、光散乱のレベルを適切に制御する、対角線、バーロー、接眼レンズ、およびフィルターを選択することです。簡単に言うと、良質のミラー表面に誘電体コーティングを施したダイアゴナルを選択することをお勧めします。

コントラスト


惑星と月を観察する目的は、高いコントラストを提供することです。これは、まぶしさや光の散乱を制御し、コントラストの良い接眼レンズを選択することで実現されます。また、適切なフィルターを使用して、月と惑星の表面の詳細の一部のコントラストを改善することもできます（これについては以下で詳しく説明します）。また、大きな倍率を適用すると、コントラストが低下することがあります。

シャープネス


一部の光学望遠鏡は、他の望遠鏡よりも鮮明な画像を構築できます。おそらくすでに望遠鏡を持っているとしましょう。その場合、バーローの接眼レンズとレンズの意識的な選択に焦点を合わせる方が良いでしょう。多くの接眼レンズモデルは、高倍率で「ぼやけた」画像を生成します。残念ながら、それらのいくつかは惑星の接眼レンズとして販売されています。オルソスコープ接眼レンズ-惑星を観察するのに最適な接眼レンズです。バジェット接眼レンズは、画像の鮮明度も低下させる可能性があります。

望遠鏡とその付属品を選択するための推奨事項：

望遠鏡


惑星観測では、サイズや光学設計に関係なく、どの望遠鏡も使用できます。ただし、特に月/惑星を観察するために購入した場合、F / D-8 ... F / D-15の比率の望遠計器がより良い結果をもたらします。特に大きな倍率を適用する場合、XAは解像度を低下させるため、色収差のない設計が推奨されます。

パフォーマンスに関しては、

80〜120mmの長焦点アクロマティック屈折器と、80〜100mmの小型APO / ED屈折器をお勧めします。

直径5〜11インチの反射屈折望遠鏡（Maksutov、Schmidt-Cassegrain）も推奨できます。しかし、残念ながら、それらのポテンシャルを使用することは、大気の不安定性のために、しばしば成功しません。

より大きなAPO屈折器は、高品質で大幅な増加を生成できますが、高価です。ニュートンの大型望遠鏡と反射屈折は、潜在的に惑星の最高のビューを提供できます。ただし、より大きなアパーチャ（レンズ直径）を利用するには、より高い解像度を得るために、並外れた大気安定性を備えた夜を選択する必要があります。これはそれほど頻繁には発生せず、平均的な夜には、より小さなレンズ直径を使用する方が実用的です。

フィルター


フィルターは望遠鏡の後の次の優先順位である必要があり、それらは良質でなければなりません。多くの製造元から販売されている最新のプラネタリープラスチックフィルターを避けてください。それらは解像度を低下させ、光散乱を増加させます。購入には、Baader、LumiconまたはRefineryガラスフィルターをお勧めします。 ebay、アストロフリマなどで使用されているものを検索できますが、主なことは、フィルターに傷が付いていないことです。

中立密度フィルターと偏光フィルターは、月や惑星によく使用されます。私は最初にそれらを使用しましたが、カラーフィルターがより良い結果をもたらすことに気づきました。

カラーフィルターは、まぶしさを軽減するだけでなく、表面のディテールのコントラストを改善します。オレンジNo. 21は三日月に最適なフィルターで、土星には火星にも適しています。火星に最適なフィルターは、赤23番と大口径-赤25番です。ブルーNo. 80Aは金星と水星に適しており、グリーンNo. 58は満月に適しています。最高のフィルターを選択するという点では、木星が最も困難でした。長年にわたって私は多くのフィルターを試してきました。カラーフィルターは青80Aだけが助かりました。

私が木星、土星、火星に強くお勧めするバーダーの特別なフィルターがいくつかあります（ただし、月、金星、水星には弱すぎます）。ベーダームーンとスカイグローは、ジュピターに最適なフィルターで、ブルーのNo. 80Aよりもはるかに優れています。土星と火星の場合、Baader Contrast Boosterコントラストフィルターを使用すると、最良の結果を得ることができます。惑星が非常に明るい場合（対立の近く）、2つのフィルターを使用できます。BaaderMoonとSky Glow、Baader Contrast Boosterを組み合わせて、3つすべての惑星に使用できます。これらのフィルターで特に気に入っているのは、まぶしさを軽減してコントラストを向上させますが、惑星の表面の自然な色を大幅に変更しないことです。

接眼レンズ


聴診器！あなたの増加が最も効果的であるかどうかに関係なく、私はあなたが惑星のためにそれらの少なくとも一つを購入することを強く勧めます。オルソスコープの接眼レンズは、シャープネス、高コントラスト、および光散乱の優れた低減を兼ね備えています。中古のオルソスコープは、40ドルから60ドルの範囲で簡単に見つけることができます。それらのほとんどは日本の1つまたは2つの工場で生産されているため、品質管理は通常良好です。あなたが新しいものを購入することを好むなら、お金のための最高の価値はバーダークラシックオルソス（BCO）です。また、BCOには50度の視野があり、従来のオルソスコープ接眼レンズやPlessla接眼レンズよりもはるかに大きくなっています。

オルソスコープ方式の2つの制限は、狭い視野（40-50度）と短い焦点距離での短い瞳孔拡張です。たとえば、18 mmのオルソスコピック接眼レンズには、約14 mmの便利な瞳投影があります。 2x Barlowで使用した場合、有効焦点距離は9 mmになります（焦点比F / D-8 ... F / D-10の望遠鏡で使用されます。3xBarlowで使用した場合、有効焦点距離は6 mm（焦点比の望遠鏡で使用されます） F / D-5 ... F / D-7）。

何年にもわたって、初期から中間レベルまでの価格帯で、多くの接眼レンズを試しました。それらのいくつかは、高倍率でぼやけた画像、低コントラスト、光のひどい散乱を持っています。オルソスコープは、惑星に最適なソリューションです。ただし、より広い視野（特にニュートン望遠鏡の所有者に関連し、ドブソンマウントで、マイクロメーターのネジで物体を案内する機能がない場合）または大きな瞳孔拡張を希望する場合は、Vixen SLV、TeleVue Radians and Delites、Explore Scientific 68および高品質の月/遊星接眼レンズとしての82シリーズおよびMeade 5000 UWA。非常に限られた予算で、Plessla接眼レンズを使用して問題を解決できますが、必要なのは高品質の接眼レンズだけです。

誰かが言うだろう：「私の接眼レンズは月の上でうまく働く」ので、それはそうです。月は非常に観察しやすい物体です。接眼レンズが少しぼやけた画像を作成する場合でも、多くの詳細が表示されます。それにもかかわらず、鋭い、トップエンドで非常に予算の大きい接眼レンズを互いに隣り合わせでテストすることは、啓示となるでしょう。優れたアナログテレビからHD放送に切り替えるように、違いは非常に表現力があります

バーロウレンズ


接眼レンズが適切な焦点距離範囲にある場合は、Barlowは必要ありません。さらに、Barlowバジェットレンズはコントラストを低下させ、光の散乱を増加させる可能性があります。ただし、良質のBarlowsが役立つ場合があります。短焦点望遠鏡で射出瞳を1 mm以下にするには、焦点距離の短い接眼レンズを使用する必要があります。この場合、瞳孔の除去は不快なことがある。この場合の最良のオプションは、2倍または3倍のバーローと、より長い望遠接眼レンズを併用することです。さらに、Barlowは望遠鏡の有効焦点距離を増加させます。その結果、Barlowレンズ+接眼レンズの組み合わせでは、短焦点の接眼レンズと比較して、より良い惑星像を得ることができます。 Baader Q barlow 2を強くお勧めします。25倍速バーロー、およびプレミアムセグメントではTeleVue 2倍速および3倍速バーロー。

対角線


光路で見落とされがちな部分は対角線です。これは、望遠鏡の接眼レンズの「星の見え方」が低下する原因になる可能性があります。主な優先事項の1つは、対角線の直径を大きくすることです。望遠鏡に2インチのフォーカサーがある場合は、2インチの誘電対角線に切り替えることをお勧めします。これにより、DSO（ディープスカイオブジェクト）と惑星の両方の画像が改善されます。私は、GSOが提供する中価格の誘電体ダイアゴナルで良い経験をしました。メーカーを推奨することもできます：Celestron、Orion、Explore Scientific。

あなたが月と惑星に最適な対角線を探しているなら、私は良質のプリズムを選ぶでしょう。プリズムは、誘電体ミラーの対角線よりも光の散乱が少なく、月や惑星に適しています。性能と価格の比率という点では、Baader T2プリズムをお勧めします。

観察


月




月では、ほとんどの詳細は、仲間の照らされた表面と照らされていない表面の境界に表示されます。ターミネーター（昼と夜の境界に沿った線）は、月の満ち欠けに合わせて毎日場所が変わるので、毎晩新しい景色を楽しめます。最小の望遠鏡と双眼鏡を使っても、月の表面に多くのクレーターを見ることができます。開口部が大きいほど、細かいディテールが可能になります。私の8インチシュミットカセグレン望遠鏡を使用すると、平均して1泊あたり最大1 kmの詳細を把握し、観察セッション全体を1つのクレーターで過ごし、壁の複雑な形状、中央のスライド、マイクロクレーター、およびその他の詳細を調べることができます。





水星と金星




これらの惑星は数ヶ月間見えません。ほんの短期間で、それらは「モーニングスターまたはイブニングスター」として観測されます。水銀は、太陽からの距離があっても、星にかなり近い位置にあるため、検出がより困難です。肉眼での水星探査はすでに達成されています。珍しい日には、水星の延長（太陽からの最大距離）と一致し、穏やかで澄んだ大気があり、惑星は地平線の近くに見えます。水銀の位相は小さな楽器でも見ることができます。

金星は見やすいです。惑星の伸長は数週間続きます。最も控えめな双眼鏡（10x50など）でも金星の位相を表示できます。大型望遠鏡では、フィルターを使用して、金星の大気の暗い雲を解決できる場合があります。

火星




その年、火星は黄道帯の星座をかなり速く動きます。空にある場合は、ほとんどの場合、詳細のない、惑星の小さなオレンジ色の円盤のみを見ることができます。しかし、2年に1回、火星の見かけの寸法が大幅に増加すると、火星は反対します（太陽との対決）。次の反対は2020年10月13日に行われるので、準備をしてください！ :) 7月から惑星の観測を開始できます。



火星は、表面の細部のコントラストが低いため、観測するのが最も難しい惑星です。フィルターとアイピースは良いはずです。しかし、80 mmの望遠鏡と忍耐があっても、対立の間、その表面の多くの詳細を理解できます。観察の焦点は、急いで惑星を望遠鏡の視野に留め、大気の沈静化の瞬間に表面の細部がよりはっきりと「引き出される」瞬間を待つ必要がないことです。ちなみに、これは木星、火星、土星などの惑星を観測するための一般的な戦略です。

木星




木星は通常、毎年4〜5か月間見えます。衛星のダイナミックなカルテットとその豊かな詳細のおかげで、木星は天文学で最も興味深いオブジェクトの1つです。 10x50光学設計の双眼鏡でさえ、惑星の円盤とその4つの衛星を使用できます。大きな倍率と双眼鏡レンズの直径（たとえば、15x70、20x80）を使用すると、ディスク上のいくつかのメインバンドを簡単に確認できます。高品質なフィルターや接眼レンズで観察すると、80mmの望遠鏡でも、木星のバンドの複雑な系を見ることが可能になります。また、惑星の円盤全体で、大赤斑の遷移と木星の月の影を観察することもできます。望遠鏡の直径を8インチ以上に大きくすると、木星の彩度が上がります。ガスジャイアントのベルトと極域の小さな細部を表示します（小さな嵐や花飾りを含みます）。また、小さなディスク上の衛星を許可します。ジュピターを見るのは素晴らしいスキルです。練習すればもっと見られるようになります。

土星




木星のように、土星は毎年4〜5か月間見えます。しかし、木星とは異なり、見かけのサイズは小さくなります。10x50双眼鏡では卵のように見えますが、練習と鋭い光学系を備えているため、15x70双眼鏡ではディスクの周りに小さなリングを配置できます。適度な望遠鏡でもリングは簡単に検出できます。開口部の比較的小さな増加は、リングに「カッシーニギャップ」を示します（フィルターは必要ありません）。土星の雲システムは、木星に比べてはるかに低いコントラストを持っています。惑星の円盤とそのリングの細部を解決するには、フィルターと望遠鏡の対物レンズの直径を大きくする必要があります。土星の最大の衛星であるタイタンは、低倍率でもはっきりと見えます。大きな望遠鏡を使用すると、さらにいくつかの衛星を解像できます。

天王星と海王星


彼らは何年も同じ星座にとどまる傾向があります。過去数年間、秋はそれらを観察するのに最適な時期です。どちらの惑星も、双眼鏡を使って「青い星」の形で、または小さな望遠鏡で見ることができます。8インチ以上の機器を使用すると、表面の詳細がない、非常に小さく緑がかった惑星の円盤を見ることができます。また、大型の望遠鏡（8インチ以上）を使用すると、海王星の衛星Tritonと天王星の少なくとも3つの衛星を見ることができます。

冥王星


まだ私の認識の惑星！:)彼は過去数年の射手座にいます。非常に安定した雰囲気では、直径8インチ以上の望遠鏡を使用すると、非常にかすかな星としてしか見ることができません。

「惑星のパレード」
