JS プロキシ。知っておくべき落とし穴

この記事は、反応性や反省のためにプロキシを使用する人にとって興味深いものになります。
JSメソッドの振る舞いは、オブジェクトのフレームワーク内で単純に使用すれば、おなじみです。
メソッドがプロパティを介して別のオブジェクトに渡された場合、それは別のオブジェクトのフレームワークで定義されているこれと連動します。

let obj1={prop1:'HEllo',method1(){console.log(this);}}
let obj2={method2:obj1.method1};
obj2.method2();

プロキシを使用する場合、これを明確に理解する必要があります。

class MyProxy{
	constructor(target){
		return new Proxy(target,this);
	}
	get(target,prop){
                console.log(target,prop); //Label1
		return target[prop];
	}
}
class PrimitiveType
{
	constructor(target,prop)
	{
	    this.target=target;
	    this.prop=prop;
	}
	get(){
		console.log(this);// Label2  
		return this.target[this.prop];
	}
}
prim=new PrimitiveType({a:''},'a');
proxy= new MyProxy(prim);
proxy.get();

結果はそのconsole.log（Label2）です。Proxyオブジェクトを提供します。その後、Proxyはターゲットとプロップの両方で機能します（Label1を参照）。しかし、コードは動作するようです。何を入浴するか。

メソッドは、プロキシを介してオブジェクト（this）との通信を開始します。これは、リフレクション（オブジェクトのプロパティのリフレクションとオブジェクトを変更せずに動作を変更する）を書くときに便利で論理的です。しかし、これが不要で、ターゲットオブジェクトを具体的に操作するメソッドが必要な場合は、どうすればよいでしょうか。なぜコードを遅くするのですか？

さらに、プロパティフィルターなどのロジックを追加すると、コードが誤って曲がる可能性があります。また、リアクティブコードを作成する場合、「ロールオーバー」があります。（たとえば、メソッドとその後の実行を要求する場合、メソッドは、すでにイベントがハングしているプロキシを通じてプロパティを要求します）。つまり、イベントは必要のない場所で発生し始め、予期されていませんでした。

直し方


理解されているように、これは、プロキシでHandler.getを呼び出す前に、メソッドに対して既にオーバーライドされています。次のように、再定義する必要があります。

let answer=target[prop];
if(typeof target[prop] ==='function'){
        answer=target[prop].bind(target);
}

次のコードを取得します。

class MyProxy{
	constructor(target){
		return new Proxy(target,this);
	}
	get(target,prop){ //     valueOf.        get 
        let answer=target[prop];
        if(typeof target[prop] ==='function'){
                answer=target[prop].bind(target);
        }
		return answer;
	}
}
class PrimitiveType
{
	constructor(target,prop)
	{
	    this.target=target;
	    this.prop=prop;
	}
	get(){
		console.log(this);
		return this.target[this.prop];
	}
}
prim=new PrimitiveType({a:''},'a');
proxy= new MyProxy(prim);
proxy.get();

最後に、ボーナスとして。


反応性/反射のチェーンを作成します。ネストされた各オブジェクトはプロキシになります：

class MyProxy{
	constructor(target){
		return new Proxy(target,this);
	}
	get(target,prop){
		let answer;
        let tp=target[prop];//    target - Proxy  target[prop] -getter
		
		if(typeof tp==='object' && tp!==null){
			answer =new MyProxy(tp);
		} else 
		if(typeof tp ==='function'){ //   .        
        	answer=tp.bind(target);
        } else {
			answer=tp;
		}
		return answer;
	}
}
class PrimitiveType
{
	constructor(target,prop)
	{
	    this.target=target;
	    this.prop=prop;
	}
	valueOf(){
		console.log(this);
		return this.target[this.prop];
	}
}
prim=new PrimitiveType({a:''},'a');
qwer={q:prim};
proxy= new MyProxy(qwer);
proxy.q
