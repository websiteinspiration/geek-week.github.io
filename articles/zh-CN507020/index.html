要创建软件定义的数据中心，可以使用供应商建议的参考体系结构。下面介绍如何使用Lenovo ThinkAgile VX节点构建VMware软件定义的数据中心。

什么是SDDC？


那么，什么是软件定义的（或软件可配置的）数据中心（软件定义的数据中心，SDDC）？首先，这是一个高度自动化的数据中心，几乎不需要物理干预。此类数据中心的所有基础架构组件均已虚拟化，其管理使用软件实现了完全自动化。对于VMware SDDC，这是vSphere软件。 VMware vSphere在各个级别提供虚拟化，包括计算，网络和存储资源的级别。


SDDC使您可以灵活地为用户提供计算，网络和存储服务。

在软件定义的数据中心中，基于策略的自动化提供资源分配和管理。结果是IT的灵活性和效率，可支持各种设备和应用程序。如今，SDDC是一种基于虚拟化和自动化的行之有效的体系结构方法，可以分阶段实施，而无需立即替换现有的IT基础架构。


在软件定义的数据中心中，整个基础架构都已虚拟化，并且使用软件实现了自动化管理。自2012年以来，VMware一直在推广SDDC概念。

这是为了什么


随着IT复杂性的增加，在数据中心中管理IT基础架构并预测更改的影响变得更加困难。业务需求被叠加在基础结构问题上：降低成本，更灵活地使用可以满足用户对稳定性，性能，可用性和易于更新的要求的应用程序。


VMware SDDC包含构建企业基础架构所需的所有软件。

VMware SDDC产品包括VMware vSphere计算虚拟化软件，VMware vSphere和VMware vSAN存储虚拟化，VMware NSX Network Virtualization以及VMware vRealize云管理平台和集成的VMware EVO系统。

目标是拥有一个灵活，易于管理的IT基础架构，以适应不断变化的需求。得益于计算，存储和SDDC网络的虚拟化，它对物理硬件（底层硬件平台）的依赖性降低了。基于策略的配置和按需提供IT资源可简化数据中心管理，扩展和更新其基础架构。

创建SDDC的结果是高效率和低成本。虚拟化的IT服务和自动化的运营管理为使用资源和提高员工生产力开辟了新的可能性。基于策略的配置使您可以快速部署工作负载，并自动配置资源利用率以满足不断变化的需求。考虑到虚拟化的安全功能自动化可延长正常运行时间。

供应商提供了几种用于部署SDDC的硬件和软件配置，包括统包式超融合解决方案-集成的硬件和软件系统，以及参考，参考体系结构。它们设计用于各种应用程序和工作负载。

VMware vSAN


VMware vSAN是与ESXi虚拟机管理程序集成的软件定义存储（SDS）解决方案。 VMware vSAN将三个或更多连接10GbE的服务器上的闪存缓存合并到一个共享数据存储中。根据Gartner的研究，在未来五年中，软件定义存储在全球市场中的份额将增长到50％。


VMware vSAN最多可扩展到64台服务器，每台服务器最多支持五个驱动器组。该组包括固态驱动器（SSD）和最多七个硬盘驱动器（HDD）。通过添加磁盘，闪存或服务器，可以轻松提高存储性能和容量。缓存用于加快读写速度。

vSAN上托管的主要工作负载。64％是关键业务应用程序（根据VMware调查）。

VMware vSAN使用分布式RAID体系结构：数据分布在整个群集中。使用的方法是RAID-0，RAID-1或RAID-5 / 6。此解决方案应用基于存储策略的管理（SPBM）。存储资源是根据请求的策略动态分配的，而不是像许多传统解决方案一样，是预先分配的。

雅虎测试（YCSB）：联想节点结果


该测试用于指示各种群集计算负载的相对存储性能。 YCSB是NoSQL测试平台。该测试包含用于执行读取，插入，更新，删除和扫描操作的各种负载，这使得在使用NoSQL数据库和I / O系统的性能时可以评估负载。它支持具有不同读/写比率的工作负载定制。

YCSB测试在具有mongodb v3.6.3和WiredTiger存储引擎的Ubuntu 16.04.4 LTS虚拟机上运行。每个虚拟机具有2个虚拟CPU，4 GB内存和50 GB内存。

该测试是用48和96个虚拟机执行的。此外，还使用了另外四个具有YCSB客户端的Ubuntu 16.04.4 LTS虚拟机来模拟mongodb虚拟机上的负载。为了评估I / O系统，使用了125个并发线程来达到每秒5,000个数据库操作的目标速度。

VMware云基金会


VMware Cloud Foundation（VCF）是一个单一的混合平台，用于将VMware SDDC部署为私有云并与公共云集成。该解决方案提供了软件定义的服务，用于管理各种工作负载的计算，存储，网络和云。通过管理组件的生命周期，它简化了安装，升级和SDDC组件。

新的VMware SDDC Manager组件提供了自动化功能。 SDDC Manager软件可自动执行vSphere，vSAN和NSX的安装和生命周期管理-从启动和配置到升级，以及vRealize Log Insight，vRealize Operations和vRealize Automation的安装和配置。

Cloud Foundation将VMware vSphere（计算资源），vSAN（存储）和NSX虚拟化（网络）组合到一个标准集成包中。 Cloud Foundation可以部署在本地私有云中，也可以用作公共云服务。
VCF以集成硬件堆栈或软件包的形式提供，可在客户设备上安装。 VCF支持在各种物理服务器（vSAN就绪节点）和网络交换机之间部署SDDC组件，从而创建灵活的异构基础架构来支持各种工作负载。特别是，VCF可以安装在Lenovo ThinkAgile VX认证的站点上。

该解决方案经过供应商的测试，验证和支持。客户可以“从一个窗口”获得所有必要的组件，从而简化了订购和支持过程。现成的参考体系结构极大地简化了此类复合体的实现和操作。
