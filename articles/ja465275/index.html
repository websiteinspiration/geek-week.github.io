アリスはスキルを得る



この記事では、音声インターフェース（Voice User Interface-VUI）に焦点を当て、Node.js（スキル-Yandex.Dialogsプラットフォームの用語）でアリスの音声アシスタント用のアプリケーションを作成する方法も示します。私たちのアプリケーションはサードパーティのAPIを使用して、有名人からの引用を受け取り、それをAliceに渡します。Aliceは次に、それらをユーザーに読み上げます。また、そのようなアプリケーションを収益化するいくつかの方法の1つである寄付ボタン（寄付）の作成も検討してください。一般的に、誰もが満足する必要があります。

では、コアとなる音声インターフェイスとは何でしょうか。事実は、数年前に、テクノロジーが開発のレベルに達し、音声を介して、人間とコンピュータとの最も自然で自然な対話方法を大量に使用できるようになったことです。多くのIT企業が適切なプラットフォームを作成することにより、サードパーティの開発者向けに「音声アプリケーション」を開発する機会を提供しました：Alexa（Amazon）、Action on Google、Bixby（Samsung）、Alice（Yandex）。現在、これらのプラットフォームは、コンピューターやスマートフォンから家電製品に至るまで、音声インターフェースを広く普及させるための強力な推進力を生み出しています。今後数年間で、人間と機械の相互作用の通常のパラダイムが根本的に変化するようです。ボタンやトグルスイッチの代わりに、これらすべての冷蔵庫、コーヒーメーカー、電子レンジ、アイロンは人間の声によって制御されます。

今日はアリスに焦点を当てます。アリスは、当初はロシア語をサポートすることを目的としており、すでに人間の声の非常に高品質なシンセサイザーを持っています。アリスは次の場所から呼び出すことができます：Yandex.Browser ; YandexのモバイルアプリケーションアリスとのためのAndroidとiOSの。Yandex.NavigatorモバイルアプリケーションのためのAndroidおよび iOS ; Yandex.Stations。スキルについては、アリスはあなたのような独立した開発者が彼らのアプリケーションを公開できる特別なカタログを持っています。そして、ご存知のように、プラットフォームがまだ非常に若い時代にあるという事実にもかかわらず、アリスはすでにかなりたくさんいます。彼女は文字通り私たちの目の前に遍在するようになります！たとえば、 Alice for Androidを使用した Yandexアプリケーションのダウンロードは1億を超え、 StatCounterによると、2019年7月のロシア市場における Yandex.Browserのシェアは13％を超えました。そして、これらは説明のためのほんの数例です。



したがって、イノベーションに遅れをとらないプログラマーにとって、音声インターフェースのために開発する時が来ました！始めましょう。forismatic.comAPIにリクエストを送信し、応答から引用符と作成者の名前を抽出し、適切な形式にすべてフォーマットして、Aliceに送信するスキルを作成します。後者は、（そして、すでに私たちの参加なしで）順番にこれらの声明を発音し、聴衆に示します。さらに、「プロジェクトをサポートする」ボタンを表示する画面を持っているユーザーがクリックされると、ユーザーがYandex.Moneyサービスのページにリダイレクトされるようにアリスに依頼します。。したがって、良い心とお金を持っている人はあなたのプロジェクトの開発を助けることができます。
