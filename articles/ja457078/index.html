テレグラムアバターを時計に変える方法

私は最近、Telegramのプログラマーの1つのコミュニティに座って、非常に興味深いプロファイルに気づきました。次は好奇心が強い-メイン写真で彼は現在の時間を描いた。私は彼がこれをどのように達成したかにひどく興味があり、同じプログラムを書くことをどうしても決心しました。


その前は、PyTelegramBotAPIライブラリを使用してTelegramボットを操作することが多かったので、そのような関数はボットAPIにあることを提案しました。確かに、これは私の人生では馬鹿げた仮定でした。写真はボットではなくユーザーのプロフィール写真にありました-そしてすぐにこれに確信し、プロフィール写真を変更するための前提条件を見つけることができませんでした。

私はインターネットをサーフィンし始め、かなり便利なツールであるテレソンに出くわしました。彼は、ボットとしてではなく、ユーザーとしてTelegramに入ることを許可しただけです。まあ、それは目標に向けた最初の、そして最も大きな一歩だったと思います。次に、プロフィール写真の「時計」を再現する方法を見てみましょう。

第一歩。コードからTelegramログインにアクセスする


私たちは、続くリンクを我々は確認コードを取得した後、電話番号を入力します。詳しくはこちらをご覧ください。したがって、そこから重要な2つのもの、api_idとapi_hashを取得します。config.pyファイルを作成し、そこに次のコードを入力します。

api_id = < API_ID>
api_hash = < API_HASH>

次に、書き込むmain.pyファイルを作成します。

from telethon import TelegramClient, sync
from .config import *

main.pyとconfig.pyが同じファイルレベルにある ことが重要です。それ以外の場合は、.config import *の行でエラーが発生します。TelegramClientクラスは、通常のユーザーとしてTelegramにログインできるようにするものです。さらに、同じファイルに次のように記述します。

client = TelegramClient(< >, api_id, api_hash)
client.start()

したがって、Telegramにログインしたと想定できます。セッション名は自由に選択できます（たとえば、「パイナップル」）。だから、私たちは電報に接続されています。写真について考える時が来ました...

ステップ2 写真のタイムソースを作成する


おそらく、このステップは私にとって最も簡単でした。私は長い間、コンピュータビジョンを使用するさまざまなソフトウェアを開発してきました。そのため、画像の操作は当たり前になっています。ここではすべてが簡単です。opencvライブラリをインストールします。これにより、写真で不思議に作業できます。utils.pyファイルを作成して書き込みます。

def convert_time_to_string(dt):
    return f"{dt.hour}:{dt.minute:02}"

写真にH：MMの形式で時間が表示されるようにするには、この関数が必要です（これを行う方法は他にもありましたが、私はこれを優先しました）。

それで、私たちは自分たちで写真を作成し始めます。generate_time_images.pyファイルを作成して書き込みます。

from .utils import *
import cv2
import numpy as np

これらのライブラリは、タイムスタンプ付きの写真を作成するのに十分です。次に、黒い背景を取得する関数を記述し、その上に時間を記述します。

def get_black_background():
    return np.zeros((500, 500))

ここで、1日ごとにサイクルを実行し、時間を示す写真を生成する必要があります。事前に/写真を保存する場所にtime_imagesフォルダーを作成します。だから、私たちは書きます：

start_time = datetime.strptime("2019-01-01", "%Y-%m-%d")  #    
end_time = start_time + timedelta(days=1)

def generate_image_with_text(text):
    image = get_black_background()
    font = cv2.FONT_HERSHEY_SIMPLEX
    cv2.putText(image, text, (int(image.shape[0]*0.35), int(image.shape[1]*0.5)), font, 1.5, (255, 255, 0), 2, cv2.LINE_AA)
    return image

while start_time < end_time:
    text = convert_time_to_string(start_time)
    image = generate_image_with_text(text)
    cv2.imwrite(f"time_images/{text}.jpg", image)
    start_time += timedelta(minutes=1)

写真のコレクションを作成するには、あと1つ、generate_time_images.pyを実行するだけです。開始後、多くの写真がtime_images /フォルダーに表示されていることがわかります。次のような写真が表示されます。


ステップ3 毎分写真を更新する


でテレソンと呼ばれる-非常に便利なものがあるUploadProfilePhotoRequest。以前に作成したmain.pyにインポートします。

from telethon.tl.functions.photos import UploadProfilePhotoRequest, DeletePhotosRequest
from datetime import datetime
from .utils import *

もちろん、アバターを常に更新する必要はありません。1分に1回だけ更新してください。これを行うには、古い分が終わり、新しい分が始まる瞬間を知る必要があります-この時点で写真を変更します。utils.pyにもう1つの小さな関数を記述します。

def time_has_changed(prev_time):
    return convert_time_to_string(datetime.now()) != prev_time

はい、はい、前回のアバター更新時の文字列を渡します。もちろん他にも実装方法はありましたが、あっという間に書きましたので、最適化についてはあまり考えませんでした。main.pyを埋め続けます：

prev_update_time = ""

while True:
    if time_has_changed(prev_update_time):
        prev_update_time = convert_time_to_string(datetime.now())
        client(DeletePhotosRequest(client.get_profile_photos('me')))
        file = client.upload_file(f"time_images/{prev_update_time}.jpg")
        client(UploadProfilePhotoRequest(file))

以前のアバターを削除するには、DeletePhotosRequestが必要です。これにより、新しい写真を追加しないエフェクトが作成されます-それ自体が変更されます。プロセッサに過負荷をかけないようにするために、ループの最後にtime.sleep（1）を追加できます（もちろん、最初にtime.pyをインポートします）。

できた！main.pyを実行すると、アバターが時計に変わります。より独自のオプションとして、たとえば、写真のテキストを次のように変更してみてください。
