SDL 2レッスン：レッスン2-ゴム以外のメイン

みなさん、こんにちは！これは2回目のSDL 2レッスンです。まだここから情報を入手できます。

だから、レッスンへようこそ

ゴムではなくメイン


前のレッスンでは、Main（）関数にすべてをうまく配置しましたが、大きなプログラムの場合、これは良くありません。それが関数を書く機会が現れた理由です。これを使用します。

SDLを接続し、いくつかのグローバル変数を宣言することから、コードの作成を開始します。

#include <SDL2/SDL.h>

const int SCREEN_WIDTH = 640;
const int SCREEN_HEIGHT = 480;

SDL_Window *win = NULL;
SDL_Surface *scr = NULL;
SDL_Surface *smile = NULL;

ここでは、Main（）関数の外で、ウィンドウオブジェクト（win）、画面の表面（scr）、およびこの絵文字の画像（smile）を宣言しました。

SDLを初期化する関数を記述して、描画用のウィンドウとサーフェスを作成しましょう。次のようになります。

初期化 （）
int init() {
    if (SDL_Init(SDL_INIT_VIDEO) != 0) {
        return 1;
    }

    win = SDL_CreateWindow("Main  ", SDL_WINDOWPOS_UNDEFINED, SDL_WINDOWPOS_UNDEFINED, SCREEN_WIDTH, SCREEN_HEIGHT, SDL_WINDOW_SHOWN);
    if (win == NULL) {
        return 1;
    }

    scr = SDL_GetWindowSurface(win);

    return 0;
}

このコードは、前のレッスンでおなじみのはずです。ここでは、ビデオシステムを初期化し、「Main is not rubber」という名前のウィンドウを作成し、座標が未定義で、以前に設定された寸法を使用し、ウィンドウの表面を取得して、scrに保存しました。エラーの場合、この関数は1を返し

ます。それ以外の場合は0を返します。必要なメディアファイルをロードする関数を記述できるようになりました。彼女が来た：

ロード（）
int load() {
    smile = SDL_LoadBMP("smile.bmp");

    if (smile == NULL) {
        return 1;
    }

    return 0;
}

この機能は非常に簡単です。SDL_LoadBMP関数を使用して.bmpイメージをロードします。この関数は、イメージファイルへのパス（完全または相対）を取得し、SDL_Surfaceクラスのオブジェクトを返します。実行の正確さも確認する必要があります。ダウンロードに問題があった場合（ファイルが見つからない、壊れたファイル）、SDL_LoadBMP関数はNULLを返します。

アプリケーションを完成させるためのプログラムも作成します。

終了する （）
void quit() {
    SDL_FreeSurface(smile);
    smile = NULL;
    
    SDL_DestroyWindow(win);

    SDL_Quit();
}

新しいSDL_FreeSurface関数がここに登場しました。これは単に表面をきれいにするだけです。変数smileをNULLに設定した後。他のすべての機能については、前のレッスンで説明しました。

関数の記述が終わったところで、Main（）に進みます。

メイン（）
int main (int argc, char ** args) {
    if (init() == 1) {
        return 1;
    }

    if (load() == 1) {
        return 1;
    }

    SDL_BlitSurface(smile, NULL, scr, NULL);

    SDL_UpdateWindowSurface(win);

    SDL_Delay(2000);

    quit();

    return 0;
};

ここでは、最初に記述したinit（）関数を呼び出してから、以前に記述したload（）関数を呼び出します。次に、SDL_BlitSurface関数を使用して、笑顔をscrに表示します。この関数は、描画用のサーフェス、このサーフェスからカットする必要がある長方形（これはアニメーションに必要です）を取得します。何もカットする必要がない場合は、NULLが送信されます。長方形が描画されるサーフェスで、座標は描画に使用されます。座標がゼロの場合、NULLを渡します。

座標について話す時が来ました。SDLは、誰もが慣れている座標系ではなく、異なる座標系を使用します。ここで、座標（0、0）のポイントは左上隅にあります。これは、不快な状況を避けるために表示するときに考慮することが重要です。

まあ、それは楽しかった、私たちは続けます。次に、おなじみの関数でウィンドウを更新し、プログラムを2秒間フリーズさせ、作成したquit（）関数を呼び出して、プログラムを終了します。出力には、クールな笑顔を2秒間表示するプログラムがあります。コード全体は次のようになります。

#include <SDL2/SDL.h>

const int SCREEN_WIDTH = 640;
const int SCREEN_HEIGHT = 480;

SDL_Window *win = NULL;
SDL_Surface *scr = NULL;
SDL_Surface *smile = NULL;

int init() {
    if (SDL_Init(SDL_INIT_VIDEO) != 0) {
        return 1;
    }

    win = SDL_CreateWindow("Main  ", SDL_WINDOWPOS_UNDEFINED, SDL_WINDOWPOS_UNDEFINED, SCREEN_WIDTH, SCREEN_HEIGHT, SDL_WINDOW_SHOWN);
    if (win == NULL) {
        return 1;
    }

    scr = SDL_GetWindowSurface(win);

    return 0;
}

int load() {
    smile = SDL_LoadBMP("smile.bmp");

    if (smile == NULL) {
        return 1;
    }

    return 0;
}

void quit() {
    SDL_FreeSurface(smile);
    
    SDL_DestroyWindow(win);

    SDL_Quit();
}

int main (int argc, char ** args) {
    if (init() == 1) {
        return 1;
    }

    if (load() == 1) {
        return 1;
    }

    SDL_BlitSurface(smile, NULL, scr, NULL);

    SDL_UpdateWindowSurface(win);

    SDL_Delay(2000);

    quit();

    return 0;
};

私はすべてが明確だったことを望みます。皆さんお元気で！
