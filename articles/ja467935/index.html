「ブラックボックス」の機能を備えたレコーダーEdic Weeny A110のレビュー

私は戻っシャギーで、世界で最小のボイスレコーダーを生産ゼレノグラード会社テレシステムズ、について書いた2010 ;その後、「Telesystems」は私たちのために、生産への小さなhabraxcourseを組織しました。

ボイスレコーダーEDIC-mini Weeny A110。新ラインEdic-mini Weeny＆Dimeのサイズは29x24 mm、重さは4グラム、厚さは4 mm。同時に、ウィーニーラインには、わずか2ミリメートルの厚さで、正面の寸法が38x24 mmの、より薄いA112モデルもあります。そして、ラインで最小のモデルは、37 x 15 x 4.5 mm、重量4グラムのWeeny A113です。



"Telesystems"は、43x36x3.2 mm、重さ8グラムのEdic Mini A2Mレコーダーがギネスブックに掲載された2004年からブランドを保持しています。最大記録時間は600時間でした。2007年、Edic-mini Tiny B21モデル（8x15x40 mm、6グラム、8 GBのかなりのメモリ）が記録に残りました。



Weenyボイスレコーダーは、ミニチュアサイズでかなり印象的な機能のリストを持っています。A110および他のWeenyシリーズレコーダーのユーザーは、次のことができます。

  • サンプリング周波数を設定します（8〜22 kHz）。
  • 記録品質を選択します（4〜24ビット）。
  • 「ブラックボックス」機能を構成します（たとえば、過去15分間にオフにすると、オーディオ録音を自動的に保存します）。
  • 毎日または1回限りのタイマー録画を設定します。
  • トリガーされるサウンドレベルを示す、サウンドによる録音のアクティブ化をアクティブまたは非アクティブにします。
  • メモリの内容にアクセスするためのパスワードを設定します。

Weeny A110の表面には、2つのオンボタンとオフボタン、単色LED、マイク、設定をリセットするための接点があります。



別々のオンとオフのボタンを使用すると、タッチで録音のオンとオフを切り替えることができます。一見すると、それらの形状と配置の対称性が同一であることは恥ずかしいですが、これはミニチュアの価格です。

特別な自由の最小サイズを見つけたいという願望があるとき、ああ、いや。しかし、どれがUSBコネクタを完全に模索しています。

LEDは、録音の開始/終了、録音中および充電プロセス中のメモリとバッテリーのステータスを示します。これは、1つの色でしか燃えない1つの電球に対するかなり多数のタスクです。その結果、レコーダーの状態に関する情報を送信する方法はモールス信号に似ています。理解するには、LEDの点滅の数と間隔を監視する必要があります。

  • オンボタンを押すと、3回短く点滅します-レコーダーの電源がオンになり、録音が開始されます。
  • オフを押すとフラッシュが1回長くなります-レコーダーの電源がオフになり、録音が完了します。
  • 記録中の2回の短いフラッシュ（各シリーズで1〜4回）の場合、レコーダーはバッテリー（最初の一連のフラッシュ）とメモリ（2番目の一連のフラッシュ）の状態を報告します。発生が多いほど、残りは少なくなります（反対を行う方が論理的です）。

実際には、そのようなメカニズムはあまり便利ではありませんでした。テレシステムの代わりに、
自転車を発明したのではなく、既存の成功したメカニズムをコピーしただけです。 2色のLEDを使用するJBL E25BTヘッドセットでLEDを使用するという原則は、ボイスレコーダーでの使用に最適です。青色はBluetoothの使用を示し、青色はバッテリーの状態を示します（点滅し始め、着座すると、充電中は常に点灯し、充電が完了すると消えます）。

残りの設定はconfig_w.iniファイルで「非表示」になっており、通常のUSBフラッシュドライブのようにレコーダーをコンピューターに接続するときに通常のメモ帳で開くことができます。



理解するのは難しくありません。基本的に、特別なアプリケーションでガジェットを管理することに慣れているユーザーでも、設定を変更するために構成ファイルにアクセスする提案は1つだけ無効にすることができます。

特別なWebページの形式で、より馴染みがあり、ユーザーフレンドリーな「高度なPCユーザー」インターフェイスを備えたオンライン構成コンフィギュレーターは、彼が操作を再開するのに役立ちます。私はこのコンフィギュレーターを見て、それはかなり使いやすいように見えますが、Telesystemsの公式リリースの前に、まだ誰にも見せないように頼まれました。

構成ファイルのフォーマットが落ち着くようにするために、現在はリリースしていないため、後でフォーマットのバージョンの束が乱れることはありません。

ループ録音
ループ録音または「ブラックボックスモード」は、テレシステムボイスレコーダーの最も興味深い機能の1つです。ループ録音モードでは、レコーダーは継続的に録音し、サウンドを最後の期間保存しないようにします。設定では、目的の間隔を設定できます。

;    (), 0 –    
; Size of one part of the cycle recording (minutes), 0 – Cycle recording off

CycleLength=0


これはおそらくレコーダーにとって最も価値のある機能です。通常の状況では、これは不要に思われますが、「オーディオに録音した方がいい」という瞬間を認識できないという予期せぬ状況が発生します。また、そのような場合は、原則として録音ができない場合があります。

デフォルトでは、レコーダーは16x16（16 kHz / 16ビット）に設定されており、約2時間半の録音が可能です。このモードでレコーダーをテストしました。最初に2分間の短い録音を行い、次に電源を入れてそれを使って散歩しました。すべての方法を記録して、メモリがいっぱいになるまでどれだけ続くかを確認しました。



数時間後に帰宅後、レコーダーの内容を確認した。config_w.iniに加えて、2分16秒と2時間17分の長さの2つのwavファイルがデバイスのメモリに表示されました。



したがって、16x16モードでの合計録音時間は2時間19分でした。

ただし、レコーダーをバックパックのストラップにぶら下げて、オーディオレコーダーと同じように歩くことは、Edicの機能を使用するための最も簡単なオプションの1つです。カナダでは、テレシステムボイスレコーダーが首輪に取り付けられていました。科学者は野生のオオヤマネコを装着して、被害者の鳴き声やオオヤマネコの歯の骨の鳴き声などを記録しました。



Wavファイルは、標準のiTunesメディアプレーヤー、Windows Media Player、そしてもちろん、VLCのような他のすべてのプレーヤーで問題なく再生できます。

ただし、これにはまずケーブル経由でコンピューターに接続する必要があります-これは制限です。ちなみに、キットに付属のmicroUSB-USB 2.0アダプターは、すでにUSB-Cに切り替えられている最新世代のMacには対応していません。また、手元にコンピューターがないとレコーダーのメモリの内容を表示および聴くことができません。

ミニチュアミニチュアですが、外出先で記録を聞いて送信するためにBluetooth接続をWeenyレコーダーに追加すると、このコンパクトなデバイスは市場で最も強力な「ハーベスター」の1つになります。
