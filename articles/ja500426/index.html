Python、パンダ、Excelの世界からの3つの問題の解決

Excelは非常に一般的なデータ分析ツールです。それを操作する方法を学ぶのは簡単であり、それはほとんどすべてのコンピューターにあり、それを習得した人はそれで非常に複雑な問題を解決することができます。多くの場合、Pythonは機能がほぼ無制限ですが、Excelよりも習得が難しいツールと見なされています。今日私たちが翻訳を公開している資料の作成者は、通常Excelで解決される3つのタスクのPythonを使用したソリューションについて話したいと思います。この記事は、Excelをよく知っている人のためのPythonの紹介です。



データの読み込み


まず、Pythonライブラリpandasをインポートし、Excel シートsalesとstatesワークブックに保存されているデータフレームにデータをロードします。対応するデータフレームに同じ名前を付けます。

import pandas as pd
sales = pd.read_excel('https://github.com/datagy/mediumdata/raw/master/pythonexcel.xlsx', sheet_name = 'sales')
states = pd.read_excel('https://github.com/datagy/mediumdata/raw/master/pythonexcel.xlsx', sheet_name = 'states')

次に、.head()データフレームメソッドsalesを使用して、データフレームの先頭に要素を表示します。

print(sales.head())

表示される内容とExcelで表示される内容を比較します。


Excelでのデータ出力の外観とpandasデータフレームからのデータ出力の外観の比較

ここでは、データフレームのデータを視覚化した結果が、Excelで表示されるものと非常に似ていることがわかります。ただし、非常に重要な違いがいくつかあります。

  • Excelの行番号は1から始まり、パンダでは最初の行の番号（インデックス）は0です。
  • Excelでは、列には文字で始まる文字Aがあり、パンダでは、列の名前は対応する変数の名前に対応しています。

私たちは、通常Excelで解決される問題を解決できるパンダの可能性を研究し続けます。

PythonでExcel Excel IF機能を実装する


Excelには非常に便利な機能IFがあり、たとえば、別のセルの内容をチェックして、セルに何かを書き込むことができます。Excelで新しい列を作成するとしますB。その列のセルは、列の対応するセルの値が500を超えているかどうかを示します。Excelでは、対応するテキストをセルに書き込むことで、そのような列（この場合は列E）に見出しを割り当てることができます。その後、セルに次のように入力できます。MoreThan500E1E2

=IF([@Sales]>500, "Yes", "No")


ExcelでIF関数を使用する

パンダで同じことを行うには、リスト内包表記を使用できます。

sales['MoreThan500'] = ['Yes' if x > 500 else 'No' for x in sales['Sales']]


Pythonでのリストの包含：現在の値が500を超える場合、はいはリストに含まれます。それ以外の場合、

リストの包含はそのような問題を解決するための優れたツールであり、if / elseのような複雑な構成の必要性を減らすことでコードを簡素化します。同じ問題はif / elseで解決できますが、提案されたアプローチは時間を節約し、コードを少しきれいにします。リストの包含の詳細については、こちらをご覧ください。

PythonでのVLOOKUP Excel機能の実装


私たちのデータセットでは、Excelシートの1つに都市の名前があり、もう1つには州と地方の名前があります。各都市がどこにあるかを正確に知る方法は？これにはExcel関数が適しておりVLOOKUP、2つのテーブルのデータをリンクできます。この関数は、式の左側にあるデータセットの各レコードが保存されるときに、左結合の原理で機能します。関数を使用してVLOOKUP、指定したシートの特定の列で特定の値を検索し、見つかった値の右側にある指定した列数の値を返すことをお勧めします。これは次のようになります。

=VLOOKUP([@City],states,2,false)

関数を使用してこの列のセルに都市が存在する州および地方の名前を入力するとき に、シートのsales列見出しを設定します。FStateVLOOKUP


ExcelでVLOOKUP関数を使用する

Pythonでは、mergeパンダのメソッドを使用して同じことを行うことができます。2つのデータフレームを取り、それらを結合します。この問題を解決するには、次のコードが必要です。

sales = pd.merge(sales, states, how='left', on='City')

それを分析しましょう：

  1. メソッドの最初の引数mergeは、元のデータフレームです。
  2. 2番目の引数は、値を探しているデータフレームです。
  3. 引数howは、データを結合する方法を正確に示します。
  4. 引数onは、接続の実行に使用する変数を指します（ここでは、引数left_onおよびを使用できますright_on。異なるデータフレームで対象のデータの名前が異なる場合に必要です）。

要約表


ピボットテーブルは、Excelの最も強力な機能の1つです。これらのテーブルを使用すると、大規模なデータセットから貴重な情報をすばやく抽出できます。各都市の総売上高に関する情報を表示するExcelピボットテーブルを作成します。


Excelのピボットテーブルを作成する

あなただけのフィールドをドラッグして、このような表を作成するために、見ることができるようにCityセクションにRowsフィールドSalesセクションにValues。その後、Excelは各都市の合計売上高を自動的に表示します。

パンダで同じピボットテーブルを作成するには、次のコードを記述する必要があります。

sales.pivot_table(index = 'City', values = 'Sales', aggfunc = 'sum')

それを分析しましょう：

  1. ここでは、メソッドを使用sales.pivot_tableして、データフレームに基づいてピボットテーブルを作成することをパンダに伝えますsales。
  2. 引数indexは、データを集計する列を指します。
  3. 引数valuesは、集計する値を示します。
  4. 引数aggfuncは、値の処理に使用する関数を指定します（ここでも、関数mean、maxなどminを使用できます）。

概要
