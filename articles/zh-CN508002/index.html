3.现在，您可以在SwiftUI组件上创建100％的应用程序。



是的是的。为此，Apple想出了没有AppDelegate，SceneDelegate和UIHostingViewController的方法。



使用注释主要 和协议应用程序，场景可以实现以下目的： 



@main
struct testmultiplatformApp: App {
  @SceneBuilder var body: some Scene {
        WindowGroup {
            MailViewer()
        }
        Settings {
            SettingsView()
        }
    }
}


主要表示这是您应用程序的新入口点。必须执行App协议。请注意App结构的body属性的类型。一个应用程序可以具有一个或多个所谓的场景，每个场景都实现Scene协议。每个场景都有其自己的根视图及其生命周期。如果要使用多个场景（例如，在多窗口应用程序中），则需要设置主体属性SceneBuilder。这实质上是UISceneDelegate的封装机制及其逻辑。
