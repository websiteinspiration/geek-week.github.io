Beginning of beta testing of the plugin for Rider and PVS-Studio C # for Linux / macOS


Due to a series of various events, the beginning of beta testing of the plug-in for the Rider and C # analyzer for Linux / macOS was a little delayed. However, we are pleased to announce that this day has arrived - today we are launching a beta test.

A small note for those who missed the previous news - we are currently developing a plug-in for the IDE Rider, as well as porting our C # analyzer to .NET Core to analyze projects on Linux and macOS. The Windows version will continue to work under the .NET Framework. However, I emphasize that with its help you can check both the .NET Framework and .NET Core projects.

You can sign up for a beta test by filling out the form on this page . We will send you installation instructions as well as a license for the analyzer.

Due to the fact that we slightly changed the release schedule and released the 7.07 releaseearlier than originally planned, the plug-in for Rider and the C # analyzer for Linux / macOS were not included in it. But we had more time to improve these products before their release. :)

In conclusion, I want to offer to sign up for a beta test , if you have not done so already, wish a codeless code, and in general, more good things in these strange times for all of us.
