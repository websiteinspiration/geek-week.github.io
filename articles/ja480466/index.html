しかし、私たちの残酷な世界でいつもそうであるように、善のあらゆるバレルでは、スプーンなしで行うことはできません。本質的に、ディレクティブ@specは関数宣言コードを複製します。以下では、20行のコードがフォームのデザインで関数の仕様と宣言を組み合わせるのにどのように役立つかを説明します


defs is_forty_two(n: integer) :: boolean do
  n == 42
