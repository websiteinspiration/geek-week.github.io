Pythonでリンクを開く方法。WebBrowserの操作とInternet Explorerの問題の解決

ユニカーソルの作業中に、標準のPythonモジュールであるWebBrowserに出くわしました。このモジュールを通じて、音声アシスタントの作業を実装したいと思いました。デフォルトのブラウザーを使用したLoraですが、すべてが期待どおりにスムーズに行きませんでした。まず、このモジュールとは何か、一般的にどのように機能するかを説明しましょう。

WebBrowserは、Pythonに組み込まれたモジュールで、Webドキュメントを表示するための高レベルのインターフェイスを提供します。

開始するには、次のコマンドでモジュールをインポートします。

import webbrowser

今、リンクを開く方法の選択肢があります。2つの椅子があり

ます。1. 1行で記述します。

webbrowser.open(url, new=0, autoraise=True)

例えば：

webbrowser.open('https://vk.com', new=2)

new = 0の場合、可能であれば、同じブラウザウィンドウでURLが開きます。変数new = 1の場合、可能であれば新しいブラウザウィンドウが開きます。new = 2の場合、可能であれば新しいブラウザページ（「タブ」）が開きます。

すべてのウィンドウの上にブラウザーを開くので、autoraiseの値は安全にスキップできます。最新のブラウザーのほとんどは、Falseであってもこの変数を吐き出します。

2.パラメータを新しく覚えたり、人間が書いたりする必要はありません。

webbrowser.open_new(url)

このデザインは、可能であれば、新しいデフォルトのブラウザーウィンドウでURLを開きます。そうでない場合は、単一のブラウザーウィンドウでURLを開きます。

webbrowser.open_new_tab(url)

この場合、URLはデフォルトのブラウザーの新しいページ（「タブ」）で開きます（可能な場合）。それ以外の場合は、open_new（）と同等

です。デフォルトのブラウザーが必要ない場合を想定します。クールな.get（ ）

webbrowser.get(using=None)

大まかに言えば、使用するブラウザを指定するだけです。

たとえば、Google Chromeで新しいタブを開くには：

webbrowser.get(using='google-chrome').open_new_tab('https://vk.com')

しかし、常に.get（）だけでうまくいくとは限りません。この場合、.register（）関数が役に立ちます。たとえば、次のようになります。

import webbrowser
webbrowser.register('Chrome', None, webbrowser.BackgroundBrowser('C:\Program Files (x86)\Google\Chrome\Application\chrome.exe'))
webbrowser.get('Chrome').open_new_tab('vk.com')

Google Chromeへのパスを示し、名前を付けました。すべてのリンクはその中でのみ開かれます。WebBrowserモジュールを少し整理して、小さな問題に移りましょう。

問題


前述のように、コースプロジェクトでは、音声アシスタントの作成を選択しました。私は彼にリンクをたどり、検索エンジンで情報を検索するように教えたかった。もちろん、これのために多くのライブラリを「詰め込む」ことは可能ですが、基本的には、標準のWebBrowserモジュールを通じてこれを実装したいと考えていました。

最近のほとんどのブラウザーでは、リンクの入力行と検索行は同じものであるため、リンクの送信先と同じ場所にリクエストを転送するだけで済むようです。

例えば：

import webbrowser
webbrowser.open_new_tab('https://vk.com')
webbrowser.open_new_tab('')

このコードのロジックによれば、2つのタブが開くはずです。

  1. ウェブサイトvk.com
  2. 検索エンジンクエリ-りんご

しかし、実際には、2つの異なるブラウザが開きます。リンクはデフォルトのブラウザーによって開かれ、リクエストはInternet Explorerによって行われます（ただし、Windows 10のコストがかかりますが、Edgeの代わりにIEが開きます）。これに対処するには？本当の教祖はWebBrowserモジュール自体に入り込み、そこで修正することができますが、私たちは可能性を厳格に評価し、フローに進みます。

デフォルトのブラウザではリンクのみを開くことが許可されているため、リンクのみを開きます。

解決策の手順


  1. 検索エンジン（Yandex、Googleなど）で検索クエリを実行します
  2. リンクを取得

  3. そして、多くの人がすでに推測しているように、「テキスト= "の後に来るものを除いてリンクを挿入するだけです

    import webbrowser
webbrowser.open_new_tab('https://vk.com')
webbrowser.open_new_tab('https://yandex.ru/search/?lr=10735&text=')
  4. 次に、「+」または「％s」を使用してクエリテキスト自体を追加する必要があります

    • 「+」を介して

      webbrowser.open_new_tab('https://yandex.ru/search/?lr=10735&text='+'')
    • 「％」全体

      webbrowser.open_new_tab('https://yandex.ru/search/?lr=10735&text=%s'%'')
    • しかし、フォーマットを通じて実装する方が良いです

      webbrowser.open_new_tab('https://yandex.ru/search/?lr=10735&text={}'.format(''))


次に、ユーザーにリンクまたは要求を入力させ、プログラム自体がどのメソッドを使用するかを理解します（リンクを呼び出すか、要求リンクに挿入します）。

まず、リンクにはドメイン（.ru、.comなど）が含まれていることを理解しています。リクエストでは、原則として（車や映画をオンラインで購入するなど）ではなく、リンクに終止符を打ちますスペース。

したがって、ユーザーが入力したもののピリオドとスペースを探します。これは、Pythonにネイティブに組み込まれているreモジュールのおかげで可能です。 Pythonは、正規表現に基づいて2つの異なるプリミティブ操作を提供します。一致は文字列の先頭でパターンを検索し、検索は文字列全体を検索します。検索操作を使用します。

import webbrowser
import re
call = input('   : ')
if re.search(r'\.', call):
    webbrowser.open_new_tab('https://' + call)
elif re.search(r'\ ', call):
    webbrowser.open_new_tab('https://yandex.ru/search/?text='+call)
else:
    webbrowser.open_new_tab('https://yandex.ru/search/?text=' + call)

コードについて少し説明します。

ユーザーはリンクまたは要求テキストを呼び出し変数に入力します。

if re.search(r'\.', call):
    webbrowser.open_new_tab('https://' + call)

最初の条件は、呼び出し変数内のポイントをチェックします。文字 '\'は必須です。それ以外の場合、モジュールはその文字がその前のドットであることを理解しません。

elif re.search(r'\ ', call):

この状態では、すべてが最初の状態と同じですが、スペースの検証はすでに進行中です。スペースは、検索クエリがあることを示します。

else:
    webbrowser.open_new_tab('https://yandex.ru/search/?text=' + call)

そして他に、今度は、ユーザーが検索クエリ内のスペースやピリオドなしで書いたことのすべてを割り当てます。

スペースを確認する必要があります;それ以外の場合、WebBrowserはInternet Explorerを開きます。
