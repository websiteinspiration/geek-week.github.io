ジュリアとニューラルネットワーク：Flux


MITが高性能汎用言語Juliaのリリースを発表 してから1年余りが経過しました。それ以来、この言語は人気を博しました。1500を超える大学で使用されており（一部の言語は最初の指導言語として教えられています）、医療診断や宇宙ミッションの計画からスクールバスの交通量の最適化などの差し迫った問題まで、アプリケーションの分野をカバーしています。


ご想像のとおり、多くのプロジェクトの主要な活動分野の1つは機械学習です。これにはJuliaが多くの強力なツールを備えており、かなり興味深いプロジェクトが最近公開されました- 一般的な確率プログラミングシステム「GEN」。


今日は、その名前が示すように、ニューラルネットワークのすべての機能を提供するFluxパッケージに注目します。一連の画像の処理と調査から訓練されたニューラルネットワークに移行して、完全な分類器を取得します。

function remove_background(img)
    mtrx = copy( channelview(img) )
    for i = 1:size(mtrx, 2), j = 1:size(mtrx, 3)
        if reduce(&, mtrx[:,i,j] .> [0.8, 0.8, 0.8])
            #   - 
            mtrx[:,i,j] .= [0.5, 0.5, 0.5]
        end
    end
    colorview(RGB, mtrx)
end
