データテーブルPythonパッケージの概要

「文明の始まりから2003年までに5エクサバイトの情報が人類によって作成されましたが、現在は同じ量が2日ごとに作成されています。」エリック・シュミット


Datatableは、効率的なマルチスレッドデータ処理を実行するためのPythonライブラリです。データテーブルは、メモリに収まらないデータセットをサポートしています。

Rで記述した場合、おそらくすでにパッケージを使用していますdata.table。Data.tableは、data.frame Rパッケージの拡張です。さらに、Rを使用して大規模なデータセットをすばやく集約する人は、このパッケージなしでは実現できません（特に、RAM内の約100 GBのデータについて話しています）。R

のパッケージdata.tableは非常に柔軟で生産的です。使い方は簡単で便利です。それを使用するプログラムは非常に速く書かれています。このパッケージはRプログラマーの間で広く知られています。 1か月に40万回以上ダウンロードされ、約650のCRANおよびBioconductorパッケージで使用されています（ソース）。

Pythonでデータ分析を行う利点は何ですか？datatable重要なのは、data.tableRの世界のアナログであるPythonパッケージがあることです。このパッケージは、datatable明らかに大きなデータセットの処理に重点を置いています。RAMに完全に配置されたデータを操作する場合と、使用可能なRAMの容量を超えるサイズのデータ​​を操作する場合の両方で、パフォーマンスが優れています。マルチスレッドのデータ処理をサポートしています。一般に、datatableパッケージは弟data.tableと呼ぶことができます。

データ表



最新の機械学習システムは、膨大な量のデータを処理し、多くの機能を生成する必要があります。これは、できるだけ正確なモデルを構築するために必要です。datatableこの問題を解決するためにPythonモジュールが作成されました。これは、1台のコンピューター上で大量（最大100 GB）のデータを可能な限り最高の速度で操作するためのツールセットです。開発のスポンサーはdatatableありH2O.ai、およびパッケージの最初のユーザーですDriverless.ai。

このツールキットはパンダに非常に似ていますが、高いデータ処理速度の提供と大きなデータセットのサポートに重点を置いています。パッケージ開発者datatableさらに、ユーザーが使いやすいように努めています。特に、強力なAPIとよく考えられたエラーメッセージについて話します。この記事では、その使用datatable方法とpandas、大規模なデータセットの処理と比較した場合の外観について説明します。

データの読み取り


ここで実験するデータセットは、Kaggle（貸出ローンデータデータセット）から取得されます。このセットは、2007年から2015年に発行されたすべてのローンの完全なデータで構成され、ローンの現在のステータス（現在、延滞、全額支払い済みなど）と最新の支払い情報が含まれます。ファイルは226万行と145列で構成されています。このデータセットのサイズは、ライブラリの機能を示すのに理想的ですdatatable。

#   
import numpy as np
import pandas as pd
import datatable as dt

データをオブジェクトにロードしましょうFrame。の分析の基本単位datatableはFrameです。これは、DataFramefrom pandasまたはSQLテーブルと同じです。つまり、行と列を区別できる2次元配列の形で編成されたデータについて話しているのです。

上記の機能fread()は、強力で非常に高速なメカニズムです。ほとんどのテキストファイルのパラメーターを自動的に検出して処理し、.ZIPアーカイブやExcelファイルからデータをダウンロードし、URLでデータを受信するなど、さまざまなことができます。

さらに、パーサーdatatableには次の機能があります。

  • 区切り文字、見出し、列タイプ、文字エスケープルールなどを自動的に検出できます。
  • 彼はさまざまなソースからデータを読み取ることができます。その中には、ファイルシステム、URL、コマンドシェル、生のテキスト、アーカイブがあります。
  • 最大のパフォーマンスを得るために、マルチスレッドのデータ読み取りを実行できます。
  • 大きなファイルを読み取るときに、進行状況インジケーターを表示します。
  • RFC4180に準拠および非準拠のファイルを読み取ることができます。
