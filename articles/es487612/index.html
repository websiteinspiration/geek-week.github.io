Agregue, elimine y renombre fácilmente archivos y objetivos en proyectos CMake

Trabajar con proyectos CMake en Visual Studio 2019 16.5 Preview 2 ahora es más fácil que nunca. Ahora puede agregar, eliminar y cambiar el nombre de los archivos y destinos de origen en sus proyectos CMake desde el IDE sin editar manualmente los scripts CMake. Al agregar o eliminar archivos con el Explorador de soluciones, Visual Studio edita automáticamente su proyecto CMake. También puede agregar, eliminar y cambiar el nombre de los objetivos del proyecto utilizando la Vista de objetivos de CMake en el Explorador de soluciones.



Más detalles debajo del corte.

Archivos fuente C y C ++


Visual Studio ahora rastrea los archivos fuente C y C ++ a medida que se agregan, renombran o eliminan del Explorador de soluciones, modificando automáticamente el proyecto CMake subyacente. Esta función está habilitada de manera predeterminada en Visual Studio 2019 16.5 Preview 2, pero si desea que Visual Studio no cambie el proyecto automáticamente, puede deshabilitar esta función en Herramientas> Opciones> CMake, "Habilitar modificación automática de script CMake ...":
