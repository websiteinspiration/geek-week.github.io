9 astuces pour les terminaux Windows par Scott Hanselman

Bonjour, Habr! Vous avez peut-être entendu dire que le nouveau terminal Windows arrive bientôt. Nous avons déjà écrit à ce sujet ici . Notre collègue Scott Hanselman a préparé quelques conseils sur la façon de travailler avec le nouveau terminal. Adhérer maintenant!



Donc, vous avez téléchargé Windows Terminal et ... maintenant quoi?


Au début, vous ne serez peut-être pas ravi. C'est toujours le Terminal, et il ne vous mènera pas, en tenant la main.

1) Lisez la documentation utilisateur de Windows Terminal

2) Les paramètres sont exprimés au format JSON . Vous ferez mieux si votre éditeur de fichiers JSON est quelque chose comme Visual Studio Code et prend en charge JSON ainsi qu'intellisense.

  • Vérifiez vos paramètres par défaut! Pour plus de clarté, je vous présente mon profile.json (qui n'est en aucun cas idéal). J'ai installé requestTheme, alwaysShowTabs et defaultProfile.

3) Décidez des raccourcis clavier. Windows Terminal propose de nombreuses options de personnalisation .

  • Toute touche sur laquelle vous appuyez peut être réaffectée.

4) Le design correspond-il à vos envies?

  • Il y a plusieurs terminaux thèmes sur terminalsplash.com.
  • Et voici des dizaines de schémas de couleurs iTerm2 , leurs schémas de couleurs sont clairement visibles ici .

5) Vous voulez passer au niveau suivant? Explorez les images d'arrière-plan.

  • Vous pouvez définir des images d'arrière-plan ou même des GIF. Plus de détails ici .

6) Spécifiez votre répertoire de départ.
  • Si vous utilisez WSL, vous souhaiterez peut-être tôt ou tard que votre répertoire personnel soit situé sur le système de fichiers Linux .

7) Vous pouvez toujours utiliser Far, GitBash, Cygwin ou cmder si vous le souhaitez. Détails dans la documentation .

8) Examinez les arguments de la ligne de commande du terminal Windows.

  • Vous savez peut-être que vous pouvez démarrer Windows Terminal en utilisant «wt.exe», mais vous pouvez maintenant utiliser des arguments de ligne de commande! Voici quelques exemples:

    wt ; split-pane -p "Windows PowerShell" ; split-pane -H wsl.exe
wt -d .
wt -d c:\github

    À ce stade, vous pouvez aller aussi loin que vous le souhaitez. Créez d'autres icônes, épinglez-les dans la barre des tâches, descendez au maximum. Consultez également les sous-commandes telles que new-tab, split-pane et focus-tab.

9) J'ai enregistré une vidéo montrant à une personne habituée à Mac et Linux comment configurer Windows Terminal en conjonction avec WSL (Windows Subsystem for Linux), ce sera peut-être intéressant pour vous.

Veuillez partager ci-dessous vos conseils, profils et sujets préférés pour le terminal!
