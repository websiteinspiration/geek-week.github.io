月面探査機やジョーカーではありません。福島のロボットについて知っていること



人類史上最悪の技術災害の1つが1986年4月26日に発生しました。そして、それは2011年3月12日にほぼ繰り返されました。ご想像のとおり、ソ連のチェルノブイリ原子力発電所と日本の福島第一原子力発電所の事故について話している。 HBOが撮影した「チェルノブイリ」シリーズは、再びチェルノブイリ事故の歴史への関心を高め、破壊された原子炉から大気への放射性核種の放出を停止することがいかに困難であったかを思い出しました。それとは別に、彼らはロボットの不成功な使用と助けのための人々の強制使用について話しました。日本はさまざまなロボットを使って数多くの実験を行っており、人々を根絶する必要を遅らせる可能性があります。

チェルノブイリ原子力発電所と福島での事故は、原因と出来事の記録が異なりますが、共通点が1つあります。それは、駅周辺の広大な地域が感染しているため、長期間安全に滞在することができず、駅の電源ユニットが完全に一掃されることは間違いありません。どちらの場合も、人類はその結果に対処する準備ができていませんでした。ソビエト連邦では数十万人が除外区域の除染と原子炉の上の保護石棺の建設に取り組みました。日本では、彼らは逆に行きました。

リファレンス：暴露基準


チェルノブイリシリーズの放射線背景に関するフレーズは、「3.6 X線-何も良いことではないが、ひどいことではない」という外国のインターネットでミームになった。確かに、これらの同じ時間あたりの3.6 X線を恐れることは価値がありますか、これは本当にひどい背景ではありませんか？以下のポストで与えられた福島の測定値をよりよく理解するために、まず線量のスケールとそれらの身体への影響を理解する必要があります。

X線（P）はロシア人によく知られていますが、SIシステムに含まれていない古い放射バックグラウンド測定単位です。現在はシーベルト（Sv）が代わりに使用されています。簡単に言えば、1 Svは100 Rに相当します。つまり、3.6 R / hは0.036 Sv / hまたは36 mSv / hです。ロシアでは、いくつかの研究所で、X線で測定された現在の背景のパネルを見ることができます。私たちの国の都市の背景は、平均して12-20μR/ hまたは0.12-0.2μSv/ hのレベルで変動します。人は、私たちの惑星の自然放射線バックグラウンドから（国と高度に応じて）年間約2.4 mSvの線量を受けます。この放射線は、ラドンガスのおかげで、宇宙から、土壌、水から、空気中に存在します。

そして今、私たちが一生を通じて受ける線量の小さな表です（以下に示す1回の被ばくは、最大4日間の適切な線量の受け取りです）：





シリーズで言及した3.6 R / hに戻りましょう。事故の場合、0.036 Sv / h（36 mSv / h）は本当にひどいバックグラウンドではありません。軽度の放射線酔いが発生するには、危険ゾーンに1日以上滞在する必要があり、数時間の修理作業は非常に安全で許容できます。そして今、私たちは少し後に、リアクターのバックグラウンドは3.6ではなく15,000のX線（150 Sv / h）であるとシリーズで言ったのを思い出します。このような背景のある4分間は、必死で死に至ります。2分後、救急の機会を与えることができるのは最速の資格のある医療だけであり、25秒は放射線病にかかるのに十分です。

福島の東芝


東芝と東京エネルギー株式会社（TEPCO）のパートナーシップは、福島第一原子力発電所の建設段階から始まった。東芝は、それぞれ1976年と1978年に委託された3番目と5番目の動力装置用のBWR型原子炉の建設を担当しました。 BWR原子炉の利点の1つは、たとえばソビエトVVERよりも一次回路の蒸気圧が低く、動作温度が低いことです。不利な点には、製造の複雑さ、非常に大型のハウジングの必要性、制御および燃料要素の放射線分解の複雑さが含まれ、爆発性水素の発生を伴います。

チェルノブイリ原子力発電所での事故がRBMK原子炉の設計の欠如によって引き起こされ、それが試験中の違反の際に感じられた場合、2つの自然災害が福島の事故を引き起こした。マグニチュード9.0〜9.1の記録的な地震により原子炉が停止し、その後の津波により、福島第一地域と原子炉冷却ポンプに供給されている非常用発電機が浸水しました。非常停止の場合でもすぐには冷却されないステーションの原子炉は、冷水の流入なしに放置されたため、沸騰して蒸気圧と内部温度が上昇し、燃料棒内のジルコニウムと蒸気の接触（蒸気-ジルコニウム反応）とその後の爆発により水素が発生しました。 1、2、3号機では、炉心が溶け、核燃料が漏洩した。福島では放射性核種が冷却水とともに海に流れ込み、チェルノブイリ原子力発電所では火災の際に大気中に放出されてヨーロッパ中に広がった。


3つの電源ユニットで何が起こったかの記録。 4番目の電源装置も損傷しましたが、原子炉には核燃料がなかったため、火災のみが発生しました。出典：Roulex_45 / Wikimedia

福島での事故をなくすには、損傷マップを編集し、溶融燃料とその漏れ箇所を見つけ、使用済み核燃料を原子炉と貯蔵プールから取り除き、瓦礫を分解する、つまり、同じように巨大な放射線の背景の下で途方もない作業を行う必要があります。現在、FAESでのこの作業は、ロボットと遠隔操作車によって行われています。事故ゾーンに人を送ることは危険すぎます。駅の一部の場所では、背景が数分で人を殺す可能性があります。

東芝は、国際原子炉廃止措置研究所（IRID）と協力して、福島の発電所の検査と放射能汚染からの部屋の除染（クリーニング）という特定の作業用のロボットを開発しています。事故の原因に小さくても重要な貢献をした東芝製ロボットのいくつかを紹介します。

ちなみに、福島の清算はチェルノブイリの貴重な経験を考慮に入れていなかったという誤った意見があります。これは根本的に間違っています。事故直後、原子力分野の日露専門家グループが、福島第一原発事故の影響に対処するために設立されました。その中には、1986年の夏からチェルノブイリ原子力発電所に直接参加したモスクワ工学物理研究所のメンバーであり、多大な貢献をしたラリオンアレクサンドロビッチレベデフがいました。放射線条件と石棺の構造の研究で。日本での事故後、ラリオンレベデフは放射性水からトリチウムを分離する技術を改善し、破壊された原子炉を冷却した後に残った大量の重水の洗浄を開始することができました。ラリオンアレクサンドロヴィッチは、日本政府を代表して、朝日勲章を授与されました。軍事的および市民的功績に対して授与されました。

四足スカウト


FAESに初めて登場した東芝ロボットは、無名の4本足の無人機で、事故の直後から開発が始まりました。彼の使命は、初期に発電所で打ち上げられたすべてのロボットと同様に、動力ユニット内部の放射線偵察と損傷評価でした。

駅構内を検査するために、線量計と6台のカメラがシャーシに設置されました。保護された電子機器は、100 mSv / hの放射線で1年間（10時間の稼働日で）動作し、バックグラウンドが短時間ではるかに大きくなるように設計されています。確かに、1回のバッテリー充電は2時間の自律的存在に十分でした。ロボットは約1 km / hの速度で移動しました。これは、破壊されたパワーユニットを調べるときに十分です。管理は、干渉が発生した場合に冗長性を備えて無線で実行されました。


デバイスの下部には、小型偵察用ドローンが取り付けられていました。これは、キャリアから離れて、背の高いロボットが通り抜けられない狭い場所に這い回ることができました。原子炉配管直下の冷却水の漏えい調査に使用される予定でした。

これは、ロボットに高い期待が寄せられたと言っているわけではありません。デモの段階でも、ジャーナリストは機械の速度が遅いことを指摘しました。はしごの各ステップを登るのに最大1分かかり、足を不安定な表面に置くと、ロボットが横転する可能性があります。

しかし、FAESの最初の実験的な東芝メカニズムは、2番目のパワーユニットの建物に侵入し、少し偵察を行うことに成功しました。東京電力は、2012年12月11日にレポートを公開しました。ロボットは、パイプの1つを撮影し、漏れがないことを確認しました。2013年3月、さらに5回訪問。しかしすぐに、建物をさらに詳しく調べようとしたところ、ロボットは階段のバランスを失い、横に倒れました。回転メカニズムがないため、4本足のスカウトは2番目のブロックに横たわっていました。

折りたたみエクスプローラスコーピオン


前モデルの欠点をすべて考慮した同社の次のロボットは、その形にちなんで名付けられた東芝スコーピオンでした（タイトル写真）。それは非常に困難なタスクのために開発されました-原子炉の底を研究して燃料棒を探すことであり、そしてこの使命は巨大な放射線バックグラウンドでの作業を含みます。スコーピオンは脚を省かれ、トラックに置き換えられました。コンパクトさのために、構造は折りたたみ式でした-燃料棒を装填するための通路を通して機械が原子炉に投入されることになっています。動作モードでは、ロボットは尾のマニピュレータを持ち上げ、サソリのように3つの軸に沿って移動します。その端で、スティングの代わりにカメラとバックライトLEDが調整されます。別のカメラが前面に取り付けられており、常に前方を向きます。

スコーピオンはオペレーターがケーブルで制御するので、電力や信号の伝送に問題はありません。さらに、ロボットは返却可能になっています。証言が送信された後、ロボットは原子炉に残ってはなりません。彼を傾けるのは怖くない、カメラの「尾」はロボットを通常の位置に戻す。

数年間、スコーピオンは最終的にFAESに入るために完成しました-時間が経つにつれて、彼は、小道をクリアするための小さな大砲、バリウム、および真皮を扱うためのカッターを受け取りました。2017年2月、ロボットは原子炉に行き、背景を測定してビデオを撮影しました。線量計は、印象的な210 Sv / h（21,000 R）を示しました。ロボットをパイプに積んでいる作業者がいた次の部屋では、背景は6 mSv / hでした。

ROV：小さな潜水艦



期待が高いビデオカメラ搭載の小型潜水艦。出典：東芝

遠隔制御水中ビークル（略してROV）は、下部が6メートルの水柱に隠されている3番目のパワーユニットの原子炉を研究するために構築された最初の東芝フローティングロボットです。寸法が30 x 13 cmで重量が2 kgのこの小さな潜水艦は、2台のカメラとバックライトを搭載し、ケーブルで制御され、水中であらゆる方向に高精度で移動できます。デバイスはオペレーターによって制御され、信号と電源は長いケーブルを介して供給されます。ワイヤーが絡みついたり、くずに付着したりしないように、東芝は摩擦を最小限に抑えた特別なコーティングを開発し、2つの強力なエンジンをROV自体に取り付け、ロボットがケーブルを簡単に引きずることができるようにしました。


ちなみに、ROVの直径が14 cmだったのは、3番目のパワーユニットの原子炉容器への入口が狭いためでした。そのため、東芝のエンジニアは、電子機器、保護装置、エンジンをこのようなコンパクトなケースに収めるのに一生懸命努力しました。オペレーター向けの2か月の集中トレーニングコースの最後に、東芝ROVはFAESに行きました。ロボットは2017年7月19、21、22日に原子炉を訪れ、破壊された内臓の検査に成功しました。

マグネティックSC-ROV


放射性核種が蓄積された水がアクセスできないもう1つの場所は、原子炉の下にあるトロイダル圧力リリーフチャンバーでした。カメラの下で漏れを探すためにロボットを送る必要がありましたが、封じ込めが浸水したという事実により問題は複雑になりました-ロボットは泥水に浸されたパイプを検査する必要がありました。フローティングデバイスはこれには適していませんでした。パイプに固定し、大きな角度でも滑らずにパイプに沿って移動できるプローブが必要でした。

ロボットが転がらないように鋼管に固定する方法は？磁石の助けを借りて。東芝SC-ROVは、圧力リリーフチャンバー専用に設計されました。強力なネオジム磁石で作られた車輪付きのシャーシです。 4つのカメラとマーカーを使用して、オペレーターは水中に沈められたパイプの表面に沿って移動し、見つかった穴にマークを付けることができます。


デモ中、SC-ROVは磁石を傾斜面に完全に固定しました。しかし、実際の状況ははるかに厳しかった。出典：IRID

2014年のSC-ROVの結果によると、リークは確認されなかったが、装置の動作に問題があった。まず、水の濁りのため、視程の範囲は30〜35 cmを超えず、検索が大幅に遅くなりました。第2に、パイプの表面にいくつかの欠陥があるため、ロボットは120°に傾けられたときにパイプからスライドしました。

コリウム検索カメラ


以前の調査任務は憂鬱な状況を示していました-2番目のパワーユニットの原子炉の核燃料は原子炉容器を超えました。格納容器の底にある溶けた火格子の写真は、燃料がすでに底のどこかにあることを明らかにしました。しかし、封じ込めはそれを阻止しましたか、または真皮は地面に沈みましたか？新しいロボットを元の原子炉のまさに「地獄」に送って、信じられないほど燃料を弱める必要がありました。


2番目の原子炉の格納容器内の溶融格子は、制御および保護システムの駆動棒の真上にあります。出典：東京電力

クローラーロボットは平らな床でのみ動作し、潜水艦は水の厚さが必要です。東芝は、5メートルの伸縮管の端に取り付けられたリモートコントロールカメラを思い付きました。パイプが格納容器に入る角度に関係なく、カメラは厳密に垂直な位置をとります-本体は制御ケーブルで保持されています。つまり、カメラは「ワイヤーにぶら下がっている」ようです。レンズは水平方向に360°、垂直方向に120°回転します。実際、デバイスには同時に2台のカメラがあります-直接動作するカメラと向きのカメラです。レンズに加えて、バックライト、線量計、温度計がユニットに取り付けられています。デバイスは、最大1000グレイの吸収線量に耐えます。


カメラの最初のバージョンは2018年1月に福島で機能し、真皮を見つけ、バックグラウンドを530 Sv / h（53,000 X線）で測定しました。しかし、作業はそこで終わりませんでした-分析のためにサンプルを採取する必要がありました。デバイスの更新バージョンでは、バックライトが強化され、スライディングアームが追加されました。2019年2月、東芝のリモートコントロールカメラは、最初に2番目の電源ユニットから真皮サンプルを受け取りました。

洗浄機


電力ユニットの構内は遅かれ早かれ無効にする必要がありますが、最初に汚染マップを作成する必要があります。多数の研究ミッションの結果は非常に暗い状況を示しました：予想に反して、放射性核種のほとんどは床や壁ではなく、パイプラインや換気などの上位レベルの要素に落ち着きました-それらはすべての放射線の最大70％を占めました。床の除染は比較的単純で、壁はもう少し複雑ですが、1階の天井の高さが7〜8メートルであることを考えると、どのようにして天井や複雑なパイプの汚れにたどり着くのでしょうか。


東芝の放射性核種「クリーナー」-高さ8メートルの高さで、熱心に壁をこすり落とします。出典：IRID

東芝は、ドライアイスキャノンで動作するロボットを開発しました。氷の粉が表面を覆い、放射性粒子を結合し、ロボットがそれをこすり取り、それ自体を吸収します。このメカニズムは、構造物を8メートルの高さに上げます。これは別の遠隔制御マシンであるため、そのオペレーターは22台のカメラから同時に画像を受信します。機械は2016年1月にパワーユニットの洗浄を開始しました。その生産性は低いですが、ステーションの除染に少しでも貢献することは価値があります。

解決策ではなくヘルプ


チェルノブイリ原子力発電所の事故から33年が経過し、それ以来、技術は途方もない進歩を遂げました。現在、数十のロボットと種類の遠隔制御建設機械が、福島の影響の排除に参加しています。東京電力のウェブサイトは多くの報告を投稿しており、そのデータは全体として非常に悲しい状況を形成しています。ロボットと技術は存在しますが、チェルノブイリの清算人の有効性にはほど遠いものでした。ロボットの開発とテストには数か月から数年かかり、その成果は数十メートルの進歩、線量測定データの収集、濁ったビデオ、そして予期せぬ問題によるミッションの頻繁な終了です。

福島のすべてのロボットは救われた人々であり、すべての人間の生活はロボットを開発する価値があります。しかし、東京電力の現在の計画では、保守的な見積もりによると、事故の影響に対処するには少なくとも30〜40年かかることが明らかになっています。そして、これはチェルノブイリのソビエト清算人の偉業の英雄と規模を確認するだけです。
