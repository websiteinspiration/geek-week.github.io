C ++は、幼少期が難しいため複雑な性質を持っています。空腹の80年代に育った彼は、シンボルを配置するのに十分なスペースがなかった頻度を覚えています。これは彼を和らげ、構文を厳しくしました。多くの場合、彼は酔っぱらったとき、セグフォルトを投げたり、ガラスを割ったり、足の通行人を撃ったりしました。彼の誘惑する人は誰もいませんでした。彼の祖父はK＆R Cを尊重し、ベトナムで衝撃を受け、孫の娯楽にしばしば参加したからです。

C ++は彼の父親をほとんど覚えていませんでした。彼は「C with Classes」と呼ばれ、過剰摂取で亡くなりました。祖父は彼の一人息子について平然と話しました。彼は孫の質問に不本意ながら答えました：「わかります、いくつかのクラスがあります...そしてあなたもそこにいます...私は今のところ3バイトで615バイトを覚えています...」母親についてはもっと知られていない私たちのヒーローはいくつかのシミュラなしではありませんでした。

彼のクラスへの熱意のために彼の息子と孫からの彼の非難にもかかわらず、おじいちゃんXi自身は長くそしてしっかりとサインに座っていました。彼はそれを「ポインター算術」と呼び、これはすべて彼の哲学の一部であり、私たちが世界中の物事の本質により近づくことができると主張しました。最近、彼はめったに人前に現れませんでした。老人がずっと前に亡くなったと言った人もいましたが、いや、いや、しかしいくつかの場所では、夜に彼の足跡に気づきました。そこでランタンが壊れて、通行人を撃ちました、バッファオーバーフローがあります...

しかし、叔父のクラシカルCといとこのC-99..11では、C ++がよく見られました。家族全員が全体としてお互いを好きではなかった。しかし、コラボレーションと生活により、彼らはAPI契約の枠組み内に留まることを余儀なくされました。 Xi兄弟と彼の父親は、さまざまなクラスの愛好家とは異なり、すべての人の中で最速で通行人を撃ったのは彼らだと確信しました。 C ++は意見の相違でした。しかし、射手は最悪ではないと述べたが、OOPなどの物質の使用は無駄ではないことを心の中で理解しました。劣等感は永遠に彼を追い求めてきました。彼を取り除くために、彼はベンチマークに参加し始めました。時々、ショットガンをリロードする前に、C ++が遅くなり、家族の残りが共感してお互いを見つめ、そして彼の後ろで笑った。

しかし、C ++は都市で最高のシューティングゲームの名声のトップにとどまるまで長くは続かなかった。時間の経過とともに、PLOはあざけりの対象ではなくなり、高齢者のうなり声はもう注意を払わなくなりました。おしゃれにもなってきました。多くの人が時々クラスを投げることを愛していました...そして新世代は完全に異なって成長しました...

最初に自分自身をJavaと宣言しました。彼は足を素早く発射する必要性を頑固に拒否し、主な価値は明快さ...そしてオブジェクト...オブジェクトだけであることを保証しました。実際、彼は実際にクラスを持つオブジェクトに干渉し、このカクテルにプリミティブを追加しました。これにより、「最初のオブジェクトは完全にオブジェクト指向である」の導入が妨げられませんでした。 JavaはアンクルXの未知のOOPからの非合法的な息子であると噂されていました。そして誰かがC ++がここに含まれる可能性が高いと主張しました。確かに定かではありませんが、JavaはSun Corporationの避難所で育ちました。新しい言語が何もひどいことをしないように、そして足の発砲の経路をとらないように、サンのエンジニアはシェルターで彼をロボトミーし、人差し指を切りました。赤ん坊がもう少し年上だったとき、思いやりのある保護者は彼を成人期、兆候の世界に行かせませんでした、仮想マシンの奥深くにそれらをさらに熱心に隠します。さらに、Javaは、誰を狙うのも悪いことであり、武器は完全に禁止されるべきであることを注意深く教えました。幼い頃からの共産主義とのこのような親密な関係はJavaに良い影響を与え、エンタープライズ開発におけるすべての名誉職をすばやく獲得しました。無秩序な無政府状態の時代が過ぎ、世紀がうまく調整されたチーム開発が来たようで、足で隣人を撃つことは悪い形になりました。うまく調整されたチーム開発の時代が到来し、隣人を足で撃つことは悪い形になりました。うまく調整されたチーム開発の時代が到来し、隣人を足で撃つことは悪い形になりました。

陰に隠れて、C ++は、レーザー照準を銃口搭載ショットガンにねじ込み、ポインターにインテリジェンスを与えようとすることで、自分自身をまとめ、時代に遅れをとらないようにしました。それはあまり役に立たなかったので、多くの人には忘れられ、彼はしっかりとした飲酒と長い睡眠に夢中になりました。

そして、窓の外では、Javaはすでに新しい強力な敵と戦っていました。 C＃は、C ++とJavaの遺伝的ミックスの成果です。この実験のためにJava遺伝子があまり正直ではない方法で得られたという噂はまだありますが、当時、少しのスキャンダルが発生しました。しかし、C ++の多くの機能を継承し、完全にオブジェクト指向である別の強力な企業のエンジニアによって作成されたC＃は、Javaに挑戦するために生まれ、企業の過酷な世界に存在する権利を守ることに成功しました。彼がJavaを打ち負かすことができるように、作成者は彼にテンプレート、OOP、およびポインターを渡しました（ただし、彼は慎重にそれらを使用するように後遺しました）。
