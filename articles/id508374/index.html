Lakukan sendiri atau bagaimana menyesuaikan telepon Snom Anda. Bagian 2 ikon dan gambar

Seperti yang kami janjikan di bagian pertama artikel , kelanjutan ini didedikasikan untuk secara independen mengubah ikon di ponsel Snom.



Jadi, mari kita mulai. Langkah satu, Anda perlu mendapatkan firmware dalam format tar.gz. Anda dapat mengunduh dari sumber kami di sini . Semua ikon snom tersedia dan disertakan dalam setiap versi firmware.



Catatan : harap dicatat bahwa setiap versi firmware berisi file konfigurasi khusus untuk versi dan model telepon ini . Menggunakan file pengaturan yang tidak cocok dengan firmware atau telepon akan menyebabkan masalah.


Setelah mengunduh dan membuka file customizing.tar.gz, seharusnya terlihat seperti ini. Isi file yang sebenarnya tergantung pada versi ponsel dan firmware:







Langkah kedua, menyiapkan ikon untuk ponsel. Seperti yang Anda ketahui, ponsel Snom hadir dengan layar warna dan monokrom, dan ikonnya akan berbeda.



I. Ubah Ikon untuk Ponsel Layar Berwarna



Ikon dan gambar pada ponsel dengan tampilan berwarna disimpan dalam format PNG. Ini memungkinkan mereka untuk dengan mudah diedit di hampir semua editor gambar modern. Namun, setelah diedit, Anda disarankan untuk mengoptimalkan file png menggunakan alat seperti optipng, pngquant atau pngcrush untuk menghapus informasi yang berlebihan dan mengoptimalkan ukuran file.



Ukuran Gambar Ikon:



  • Ikon Kunci Konteks-Sensitif 24x24px
  • SmartLabel 24x24px & 18x18px
  • Ikon di bilah judul (Ikon Bilah Batang) 18x18px
  • Ikon Menu 18x18px
  • Selama panggilan (Ikon Layar Panggilan) 18x18px - 48x48px
  • Format File: PNG


Setelah membuat ikon yang diinginkan, unduh ikon itu ke telepon Anda. Ada dua cara untuk mengunduh:



  1. Melalui antarmuka web dalam mode manual
  2. Menggunakan promosi otomatis


Mari kita mulai dengan opsi pertama - unduh melalui antarmuka web. Untuk mengunduh, buka antarmuka web ponsel pada tab Preferensi / Penampilan dan pilih Gambar Khusus :







Selanjutnya kita menemukan ikon yang ingin kita ubah dan muat versi kita sendiri:







Jika Anda tidak menyukai versi Anda sendiri atau "kurva", Anda selalu dapat memutar kembali dengan mengklik tombol "Reset" "







Catatan . "Pembaruan Perangkat Lunak" dan "Kembalikan Pengaturan Pabrik" tidak menghapus gambar yang diunduh.


Seperti yang dapat Anda lihat dalam mode manual, semuanya cukup sederhana, tetapi jika Anda perlu mengubah beberapa ponsel, proses ini akan memakan banyak waktu. Oleh karena itu, kami beralih ke opsi kedua.



Opsi dua - unduh melalui penyediaan otomatis.



Pertama-tama, Anda perlu membuat arsip dalam format tar dari arsip customizing.tar.gz yang sebelumnya diunduh . Saat membuat arsip, hapus semua file dan direktori yang tidak perlu Anda ubah, tetapi pastikan Anda menjaga struktur direktori .

Catatan . Anda tidak perlu mengarsipkan semua file yang awalnya di arsip. Cukup dan disarankan agar Anda hanya mengarsipkan file yang telah Anda modifikasi. Semakin banyak file yang Anda masukkan ke dalam arsip, semakin banyak waktu yang diperlukan telepon untuk mengkonfigurasinya.


Selanjutnya, kami mengambil beberapa langkah:



1) membuat file XML, misalnya, branding.xml dan menyalinnya ke server web Anda (HTTP), mis. http: // webwebserver/branding.xml: 



<?xml version="1.0" encoding="utf-8" ?>
<settings>
 <uploads>
  <file url="http://yourwebserver/branding/branding.tar" type="gui" />
</uploads>
</settings>


2) Kami pergi ke antarmuka web telepon di bagian Advanced -> Update -> Setting URL dan memberikan tautan ke file kamiwebserver / branding.xml















3) Kami me-reboot ponsel dan menikmati hasilnya.

II.



Ikon pada perangkat monokrom tidak disimpan dalam file gambar biasa, seperti .png atau .jpg, tetapi merupakan font bitmap yang menyertakan semua ikon yang digunakan dalam antarmuka pengguna. Di bidang penggunaan pribadi dari tabel unicode dimulai dengan U + EB00, ikon snom didefinisikan dan dapat langsung dimodifikasi menggunakan alat seperti Font Forge .



Membuka file font bitmap dengan Font Forge akan menampilkan daftar ikon yang digunakan. Isi file yang sebenarnya tergantung pada versi ponsel dan firmware:







Spesifikasi ikon untuk ponsel dengan tampilan monokrom.



Untuk model D305, D315, D345, D385, D745, D785, D3, D7:



  • Ikon Kunci Konteks-Sensitif 17x17 - Garis Dasar x → 0 / y → -2
  • Ikon Bilah Judul 17x17 - Garis Dasar x → 0 / y → -2
  • Label Panel Ikon 17x17 - Baseline x → 0 / y → -2
  • Ukuran Maksimum Ikon 32x32


Untuk model D120, D710, D712, D715, D725:



  • Ikon Kunci Konteks Sensitif 7x7 - Garis Dasar x → 0 / y → 0
  • Ikon Bilah Judul 7x7 - Garis Dasar x → 0 / y → 0
  • SmartLabel Icons 7x7 - Baseline x → 0 / y → 0
  • Ukuran Maksimum Ikon 32x32


Setelah membuat "gambar" yang diperlukan dan kemudian mengekspornya dari Font Forge, Anda perlu menggunakan pengaturan berikut:







Setelah mengekspor, buat file tar yang berisi file yang baru saja dibuat dengan nama file yang akan diganti.



tar -cvf fonts.tar fontfile.bdf



Karena kita sebenarnya mengubah bukan gambar, tetapi font, lalu muat melalui promosi otomatis sebagai font, yang menentukan dalam pengaturan file xml: 



<?xml version="1.0" encoding="utf-8" ?>
<settings>
 <uploads>
  <file url="http://192.168.23.54:8080/fonts.tar" type="font" />
 </uploads>
</settings>
