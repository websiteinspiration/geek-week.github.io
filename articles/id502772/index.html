Cara membuat PostgreSQL di GCP SQL


Mari kita mulai dengan membuat server PostgreSQL di GCP SQL.

Sebelum Anda mulai, masuk ke akun GCP Anda. Untuk membuat server database PostgreSQL -> Pilih SQL di panel Layanan Google.



Untuk membuat turunan dari SQL -> Klik CREATE INSTANCE.



Pilih database yang Anda butuhkan â†’ Klik PostgreSQL.



Isi rincian konfigurasi:

ID Instans: - Beri nama instance Anda,

Atur kata sandi - Kata sandi,

Pilih wilayah yang terdekat dengan Anda - Wilayah,

Pilih versi server database - Versi database.

Klik Tampilkan opsi Konfigurasi untuk mengonfigurasi informasi tambahan.



Luaskan detail Konfigurasi



Klik Konektivitas â†’ Secara default, instance tidak dapat diakses dari jaringan eksternal; Untuk memberikan akses ke sebuah instance, izinkan IP atau berbagi tertentu. Klik + Tambahkan jaringan.



Dengan memasukkan 0.0.0.0/0 ke dalam Jaringan, Anda akan mengizinkan semua IPv4 IP dan Anda dapat mengakses instance dari mana saja.



Selanjutnya, kami menunggu pengaturan disimpan.



Untuk berkomunikasi dengan sumber daya tambahan dan API â†’ aktifkan Private IP dan pilih jaringan Terkait (jaringan VPC yang Anda buat, atau default).



Kami menunggu save lagi.



Anda juga dapat memeriksa koneksi layanan Pribadi dengan mengklik VPC â†’ Peering VPC.



Pilih Jenis dan penyimpanan mesin: jumlah core dan memori.



Jika Anda bermaksud menggunakan instance untuk tujuan pengujian, pilih satu zona.



Jika Anda memiliki lingkungan produksi, pilih Ketersediaan tinggi, tetapi ini akan menambah biaya.



Klik Tambahkan item untuk menambahkan bendera basis data.



Pilih bendera yang Anda butuhkan.


.
GCP akan memperbarui database setiap beberapa bulan, jadi kami diminta untuk menetapkan tanggal dan waktu di muka untuk memulai kembali database Anda, yang tidak membutuhkan banyak waktu. Jadi, pilih tanggal dan waktu jika Anda memiliki basis data produksi.



Tentukan kunci dan nilai untuk kejelasan di tab Label.



Klik Buat untuk membuat database.



GCP Membuat Server PostgreSQL:



Sukses! Server PostgreSQL telah berhasil dibuat.



Anda akan melihat informasi dan gambaran umum dari PostgreSQL yang baru dibuat.



Dengan mengklik Koneksi, Anda dapat melihat detail koneksi, dan juga melihat bahwa database telah diizinkan untuk berkomunikasi dengan Mesin Aplikasi GCP.



Untuk mengakses server PostgreSQL yang dibuat, kita perlu membuat pengguna di tab USERS.



Anda juga dapat membuat basis data pada tab DATABASES.



Anda dapat mengelola cadangan di BACKUPS.
