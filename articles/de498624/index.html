[Anleitung] Erstellen von Google-Tests (Google-Formulare)

Vielen Dank an die Benutzer Exosphäre und Yuriy_krd für Hilfe und konstruktive Kritik

Manchmal scheint es sehr schnell und einfach zu sein, etwas gemäß dem Designer zu erstellen: zumindest eine Site, mindestens eine Umfrage, mindestens eine Tabelle. Oft verlieren Sie jedoch einige anstößige Kleinigkeiten aus den Augen, wodurch Zeit verschwendet wird und Sie die Arbeit aufgrund einer anstößigen Kleinigkeit wiederholen müssen. Stimmen Sie zu, es ist unangenehm. Ich habe mich entschlossen, alle Prozesse für Sie durchzugehen, vor möglichen unangenehmen Kleinigkeiten zu warnen und Neulingen zu helfen.

In diesem Artikel erstellen wir ein Google-Formular, mit dem Sie Daten sammeln, Mitarbeiter und Kunden interviewen, Hypothesentests durchführen usw. können. Übrigens äußerst relevante Informationen in Bezug auf Fernarbeit.



Um Tests zu erstellen, müssen Sie über ein aktives Google-Konto verfügen. Wenn Sie noch kein Konto haben, lesen Sie die Anweisungen unter "Erstellen eines Google-Kontos und Arbeiten mit Google Disk" .

Erstellen Sie also ein Formular (Testen).

  3. Erstellen Sie eine Dropdown-Liste, in der die Schule ausgewählt wird. Ebenso können Sie eine Klasse, eine Abteilung oder eine Einheit auswählen. Diese Option ist erforderlich, damit der Testteilnehmer korrekt bestimmt, an wen er den Test besteht. Bei der manuellen Eingabe von Daten in ein Feld werden verschiedene Register oder Schreibantworten durch Filter in der endgültigen Tabelle unterschiedlich strukturiert.
  4. Sie müssen eine einfache Frage mit einer Option für die richtige Antwort erstellen.

Gehen Sie zum Google Drive. Klicken Sie dazu oben rechts im Bereich oben rechts auf die Schaltfläche mit den Punkten und folgen Sie dem Link https://www.google.com/ oder https://www.google.ru/. Wählen Sie anschließend Google Drive (im Folgenden als Drive bezeichnet).



Nach dem Laden von Drive erstellen Sie am besten einen Ordner oder eine Ordnerstruktur. Sagen wir "Testen" -> "Tests für Arbeitssuchende" und erstellen Sie nach dem Aufrufen dieses Ordners einen Test (Formular). Auf diese Weise können Sie die Daten strukturieren.

Schritt 2. Erstellen eines Tests (Formular)


Wie bei jedem Dokument in Google Drive beginnt der Arbeitsbeginn mit der Schaltfläche "Erstellen +" oben links auf dem Bildschirm. Diese Schaltfläche erstellt Ordner in Drive, alle Dokumente werden erstellt.



Wenn Sie auf die Schaltfläche Erstellen klicken, wird ein Menü angezeigt. Darin müssen Sie "Mehr" auswählen.



Es gibt verschiedene Möglichkeiten, ein Formular zu erstellen:

1. Erstellen Sie ein leeres Formular :

  • Klicken Sie auf den Menüpunkt Google Forms (Markierung 1).
  • Klicken Sie auf die Schaltfläche Dropdown-Liste (Markierung 2). Klicken Sie in der Dropdown-Liste auf „Leeres Formular erstellen“ (Markierung 3).

2. Erstellen eines Formulars gemäß der Vorlage - Wiederholen Sie die obigen Schritte, um 3 zu markieren, und markieren Sie stattdessen 3+. Für Sie wird eine Vorlagenbibliothek geöffnet, in der Sie die am besten geeignete Vorlage auswählen können. In Zukunft kann es an Ihre Bedürfnisse angepasst werden.

Wir haben ein leeres Formular erstellt (Testen).

Schritt 3. Kontrollen


Als erstes erstellen Sie einen Dateinamen für das Formular. Dieser Name ist intern, darunter wird das Formular auf dem Laufwerk gespeichert.



Der Name wird durch Klicken auf die Phrase "Neues Formular" im oberen linken Teil des Bildschirms erstellt.

In der Mitte des Bildschirms befindet sich ein Feld mit dem Namen des Tests (Formulars). Es wird von den Testteilnehmern zur Laufzeit gesehen. Ändern Sie es auf das gewünschte.



An derselben Stelle füllen wir die Beschreibung für den Test (Formular) aus.

Als nächstes wird automatisch der erste Block der Frage erstellt.



Im Feld "Frage ohne Titel" müssen Sie Ihre Frage eingeben. Benennen Sie Option 1 in die unten stehende Antwort um. Wenn Sie eine weitere Frage hinzufügen müssen, müssen Sie unten auf "Frage hinzufügen" klicken. Sie können auch die Option "Andere" hinzufügen. Anschließend wird im Test ein Textfeld angezeigt, in das der Benutzer jede andere Antwort in beliebiger Form eingeben kann.

In der Dropdown-Liste können Sie den Fragentyp auswählen. In unserem Fall wird automatisch eine Frage mit einer Antwortoption erstellt. Wenn Sie die Dropdown-Liste öffnen, können Sie eine andere Form der Frage auswählen.

Am unteren Rand des Blocks befinden sich Schaltflächen:

  1. "Kopieren" - bedeutet das Kopieren des gesamten Blocks.
  2. "Löschen" - Löscht den gesamten Block
  3. Obligatorisches Problem und Schiebeschalter. Er macht die Frage obligatorisch: Ohne Beantwortung kann der Benutzer den Test nicht fortsetzen oder beenden.
  4. Zusätzliche Einstellungen „Schaltfläche mit drei Punkten“ - Mit dieser Option können Sie der Antwort eine Beschreibung hinzufügen, die Antworten mischen und die Option für die Antwort auswählen.

Rechts neben dem Formular befindet sich ein vertikaler Block von Schaltflächen, der Folgendes ermöglicht (von oben nach unten auf den Schaltflächen):

  1. Erstellen Sie einen weiteren Fragenblock
  2. Fragen aus zuvor erstellten Formularen importieren (Tests)
  3. Fügen Sie einen separaten Block mit einem Namen und einer Beschreibung hinzu
  4. Bild einfügen
  5. Video einfügen (YouTube)
  6. Abschnitt hinzufügen. In diesem Fall wird eine weitere Seite mit Fragen erstellt. Dies ist praktisch, wenn Sie mehrere Themen in einem Formular erstellen (Test).

Um Verwirrung unter den Mitarbeitern zu vermeiden, können Sie auch die Farbform angeben, wenn Sie den Namen des Berichts angeben. Dies erleichtert die Kommunikation und die Arbeitseffizienz erheblich.

Die Registerkarte Allgemein und die Präsentation sollten keine Fragen verursachen. Eine wichtige Registerkarte „Tests“ - darin können Sie für jede Frage im Formular eine Markierung setzen. Aktivieren Sie dazu den Schiebeschalter „Test“ und Ihr Formular wechselt in die Testkategorie. Nachdem Sie diese Schaltfläche aktiviert haben, können Sie die richtigen Antworten und die Anzahl der Punkte für die richtige Antwort in der Liste der Fragen zuweisen.

Klicken Sie nach dem Aktivieren des Schalters auf die Schaltfläche „Speichern“ und gehen Sie zu den Fragen. Die Schaltfläche „Antworten“ wird am unteren Rand des Fragenblocks angezeigt:



Wenn Sie darauf klicken, gelangen Sie in den Modus zum Einstellen der richtigen Antworten und zum Bewerten von Punkten für die richtige Antwort.



Beginnen wir also mit der Aufgabe am Anfang.

Schritt 4. Erstellen Sie einen Test


  1. Wir geben den Dateinamen in Drive ein
  2. Geben Sie den Titel des Tests ein
  3. Geben Sie die Testbeschreibung ein
  4. Geben Sie die Frage ein
  5. Nachdem Sie die Dropdown-Liste geöffnet haben, wählen Sie den Punkt "Text (Zeile)".

Wir erhalten:



Aktivieren Sie den Schalter "Erforderliche Frage". Der Benutzer muss nun den Nachnamen und den Vornamen eingeben, um den Test fortzusetzen.

Fügen Sie eine zweite Frage hinzu, indem Sie auf die obere Schaltfläche im rechten vertikalen Bedienfeld klicken.



Wir werden diese Frage optional machen, ohne den Schalter unten rechts im Block zu aktivieren. Ohne diese Frage zu beantworten, fährt der Benutzer fort und kann den Test beenden.

Wir erstellen einen weiteren Block mit einer einzigen Antwortoption



.



Wir fügen einen Block mit mehreren Antwortoptionen hinzu. Hier haben wir auch die Antwortoption "Andere" hinzugefügt.

Stellen Sie nun die Punkte für die richtigen Antworten ein. Gehen Sie zum Einstellungselement oben rechts in der Steuerung (Zahnradsymbol). Gehen Sie zur Registerkarte „Test“ und aktivieren Sie den Schalter. Klicken Sie auf die Schaltfläche Speichern und kehren Sie zur Liste unserer Fragen zurück.

Durch Klicken auf eine leere Stelle im Fragenblock aktivieren wir diese. Unten links aktivieren wir den Antwortblock. Wir wählen die richtige Antwort für uns und vergeben die Anzahl der Punkte.



Wir klicken auf die Schaltfläche ist fertig. Zugewiesene Punkte.

Jetzt müssen wir die Häufigkeit der Fragen und die Erfassung der E-Mail-Adressen anpassen.

Gehen Sie erneut zu den "Einstellungen" oben in den Steuerelementen. Wir aktivieren die Einstellungselemente, die wir benötigen.



Wir gehen direkt zur Registerkarte "Präsentationen" und überprüfen, ob die Option "Link zum erneuten Ausfüllen des Formulars anzeigen" aktiviert ist. Klicken Sie auf die Schaltfläche „Speichern“.

Der Test (Formular) ist fertig.

Schritt 4. Senden des Tests (Formular)


Die Übermittlung des Tests (Formulars) erfolgt durch Klicken auf die Schaltfläche „Senden“. Im folgenden Fenster können Sie verschiedene Sendeoptionen auswählen.

Per E-Mail senden: Geben Sie



im Feld An die Adressen der Empfänger ein. In diesem Fall sehen Sie auf der Registerkarte "Antworten", wer abgestimmt hat und wer die Formularfragen noch nicht beantwortet hat.



Das Formular kann auch sofort in den Textkörper des Briefes aufgenommen werden.

Die nächste Registerkarte ist das Senden des Links. In diesem Fall können Sie es nicht nur per Post, sondern auch auf andere Weise senden, auch über den Messenger. In diesem Fall müssen Sie die Befragten jedoch selbst verfolgen.

Die dritte Option ist eine Einfügung auf der Site. Dies ist für fortgeschrittene Benutzer. Ermöglicht das Einbetten von Code mit dem Formular direkt in die Site-Vorlage.

Sie haben die Erstellung des Formulars abgeschlossen.
