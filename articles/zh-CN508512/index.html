我们如何制作有关太空坠毁的VR游戏

阶段，错误，结论



我一年前来过BeaversBrothers。公司推出一项新服务时曾给我打电话-开发企业VR / AR解决方案。为了开发技术并了解我们的能力，我们决定制作一个虚拟游戏。的确，游戏拥有一切：令人着迷的世界，生动有趣的故事情节吸引着脚本，与用户互动，复杂的技术实现。因此，这样的飞行员对我们倍加有趣。



我们已经开发了半年的游戏。这是关于虚拟现实的第一个团队项目，我们踩到了我们可以踩到的所有耙子。结果，我们获得了丰富的经验，解决了所有可能的错误，但仍未在Oculus商店中发布游戏Astro Collapse 。



在本文中，我将讨论创建VR游戏的主要阶段，并向开发人员及其客户提供实用建议。不要犯我们的错误，您将在3个月而不是半年内完成游戏。



关于游戏



Astro Collapse是虚拟现实中的单人太空射击游戏。巨大的小行星雨正在逼近地球。人类文明濒临毁灭。人们紧急撤离航天飞机并离开地球。您是一艘伴随着航天飞机并从小行星清除道路的军舰上的射手。您需要聆听船长的命令，并用穿甲机枪，强大的等离子枪和激光枪消灭小行星。





游戏目的：拯救尽可能多的船只并保护地球



搜寻想法



当我们开始进行该项目时，我们没有一个爆炸性的概念。我们只是想制作一个虚拟游戏来获得经验。前三个想法是偶然出现的。



管道



在我的Instagram提要中，我看到了朋友如何从山上骑“百吉饼”。而且我想，为什么不从虚拟山顶上启动玩家呢？





在游戏的第一次测试中，我们的撰稿人失去了平衡，差点摔倒。



卡丁车



为了纪念2月23日，公司的女孩为我们租借了卡丁车。当我在高速公路上奔波时，我决定在虚拟世界中开车也很棒。所以第二个想法来了。





游戏的原型是简单的机制：绕过盒子而不撞到墙上。



拯救地球



这很简单 我在互联网上阅读了新闻，发现现在他们谈论了很多关于太空的事情：《面具》，《火星上的生命》，《宇宙的起源》。这是Astro崩溃故事的开始。





太空射手的第一个原型：起初它很慢而且很无聊，



因此，我们花了3周的时间开发了所有原型。由于我们没有一个大的想法，因此我们不得不在旅途中快速制定解决方案。我们制作原型时并不知道游戏的受众以及他们是否会很有趣。结果，在他们身上花了两个想法和两个星期。时间就是金钱。



在制作游戏之前，请先考虑游戏的发展和面向谁。为您确定的想法创建游戏的原型。


游戏脚本



在游戏的第一个版本中，机制非常简单。为了逃脱，玩家需要射击飞船上的小行星。游戏分为几级。等级越高，小行星越会攻击船。



我们在公司内部测试了脚本，发现它很无聊。







我们的编辑人员正在测试游戏的第一个版本：



在开发游戏之前，请提出一个有趣的场景，以免浪费时间和金钱进行更改。


我们决定开发剧情并添加其他对象，以使玩家在虚拟世界中不会感到孤单。就这样停了下来。那是个大错误。我们没有想到要对未沉浸在产品中的儿童和成人测试第一版。仅根据团队的意见，我们制作了游戏的第二个原型。



成千上万的小行星飞向地球。几分钟后，他们将达到目标。有必要疏散人员并试图保护地球。需要对小行星进行射击，以免疏散船落在其下方。









我们呼出了游戏第二个原型的场景。我们认为，这款游戏令人兴奋又有趣！但是不是那么简单...



对儿童和游戏玩家的粉碎测试



为了验证我们的游戏是否成功，我们决定在发布之前进行最后的测试。为此，我们邀请了员工的孩子，他们的朋友和熟人来办公室。参与者来自不同的受众-这保证了测试的客观性。我们确信每个人都喜欢这款游戏，如果有任何评论，那么它们将很小。第一位测试人员已经消除了我们的粉红色梦想，并且跨度很小。



玩家不知道发生了什么：向谁开枪，谁是敌人，他们经过的是哪种船。每个人都问：“如何获得另一种武器，为什么没有奖金？”。儿童和成人对游戏的反应不同。



成人想要一种有针对性的游戏：轻松的射击，更少的头部运动，清晰的操作说明和射击指南。





测试过程中的成人用语：为什么要这么快的小行星？为什么这么多？那是什么那是谁？这个是来做什么的？



孩子们没有情节。他们不在乎是什么导致宇宙混乱。他们只是摧毁了一切动静不动的东西。他们说，他们想要更多的小行星，船只和运动。他们没有的最重要的事情是比赛结束时的老板。





儿童短语：为什么小行星这么少？老板小行星在哪里？是爆炸吗？



我们没有收到完全负面的评价，也没有人说：“我一点都不喜欢，重做”。但是我们意识到没有考虑多少细节。最重要的缺陷-人们不了解游戏的情节和目标。



在游戏的第三个版本中，我们添加了广播：来自指挥官的命令，人们的尖叫声，混乱和恐慌的声音。因此，我们能够使玩家沉浸在气氛中。



为了讲述一个故事并向某人介绍该情节，我们制作了一个介绍性视频，讲述了游戏中发生了什么以及需要做什么。





游戏预告片



还添加了一种训练模式，玩家可以在其中学习如何射击，谁是敌人和盟友，如何获得超级武器。玩家的情节和目标变得清晰。强大的最终和弦出现了-一个巨大的上司小行星。



现在，我们不仅对结果感到满意，而且对第一次玩游戏的人也感到满意。



从第一个版本开始针对目标受众测试游戏，这样您就不必在项目结束时从头开始。


最终的主要错误



该项目从一开始就没有经理。它是在我们测试游戏的第一个版本时出现的。在他之前，发展很缓慢，我们经常四处奔走，失去了精神。但是随着经理的到来，情况发生了变化。他组织了测试，建立了团队内部的沟通，监控了项目的时间和质量。我们在游戏到货2个月后发布了该游戏。



一个项目应该始终有一个布道者经理，他将管理开发并使团队保持最新状态。


总结一下



在开发VR游戏时要知道的重要事项：



  1. 首先想出一个您相信的想法。只有这样实施。
  2. 考虑游戏的场景并在目标受众上进行测试。
  3. 从第一个原型开始对游戏进行广泛的测试。
  4. 指派负责的项目经理，以使团队不会放松，也不会失去重点。


这四个简单的技巧可以帮助您将预算和开发时间减少一半甚至三倍。


但是在这方面，我们的耙跳并没有结束。我们仍然必须面对在Oculus商店中发布游戏的困难。稍后，我将发布一份清单，以帮助您首次发布游戏。在下一篇文章中，我将告诉您我们如何进行游戏设计，以及狗到底埋在哪里。



同时，播放Astro Collapse，从小行星拯救地球！对于游戏，您需要Samsung Gear VR眼镜。它们可与智能手机配合使用：Samsung Galaxy Note 4、5、7，S6 / S6 Edge / S6 Edge +，S7 / S7 Edge，S8 / S8 +。
