Menerbitkan Roadmap development Zextras Suite untuk 2020

Sebelumnya, kami berbicara tentang inovasi Zextras Suite Pro dan Team Pro pada awal 2020. Zextras telah menerbitkan Roadmap untuk pengembangan fitur Zextras Suite baru  untuk tahun 2020 .


Juni - Tim: Fungsi baru (DISETUJUI)

  1. Membalas pesan (balas)
  2. Hapus riwayat obrolan
  3. Edit pesan
  4. Hapus pesan

Agustus - Tim: Server Video (BARU)

Munculnya server video bawaan di Tim Zextras akan memungkinkan Anda untuk memperkenalkan sejumlah fungsi baru yang dapat dikontrol langsung dari konsol admin, serta meningkatkan produktivitas layanan. Aliran audio dan video akan dikompresi, dioptimalkan, dan digabungkan untuk mendapatkan aliran terpisah untuk setiap perangkat. Selain itu, server akan secara otomatis memilih codec dan bit rate yang sesuai untuk setiap perangkat untuk memastikan kualitas komunikasi yang maksimal.

Agustus - Cadangan pada titik akhir eksternal (BARU)

Zextras Backup akan memberikan peluang untuk mencadangkan sistem Anda ke penyimpanan lokal, hybrid, atau S3-cloud, atau ke penyimpanan eksternal elemen BLOB, seperti NFS. Pada saat yang sama, cache metadata lokal akan memungkinkan mempertahankan kinerja asli, meskipun faktanya mereka sendiri juga akan disimpan di titik akhir eksternal jika perlu pemulihan bencana.

September - Otentikasi dan Login Zextras (BARU)

Otentikasi dua- faktor, sistem autentikasi dan otorisasi SAML gabungan , serta beberapa kata sandi ponsel akan ditambahkan ke Edisi Open-Source Zimbra Collaboration Suite. Halaman login yang diperbarui juga akan muncul dengan kemungkinan penyesuaian.

September - Pengaturan Privasi Tim (BARU)

Pengguna dan administrator akan dapat memilih apakah informasi tentang keberadaan mereka akan ditampilkan. Secara khusus, adalah mungkin untuk menyembunyikan informasi tentang membaca pesan, status pengguna, dan sebagainya. September - Aplikasi Seluler QR Authentication (NEW) Sistem QR-Login memungkinkan Anda untuk menggunakan aplikasi seluler tanpa harus memasukkan kata sandi melalui penggunaan sistem otentikasi silang, serta mengelola pendaftaran dan otorisasi mereka. September - Buku Alamat Zextras untuk MS Outlook / MAC Mail / Thunderbird (BARU) Hubungkan Microsoft Outlook, Thunderbird dan Mac Mail ke Zimbra GAL yang terpusat dan buku alamat Anda untuk kenyamanan pengguna yang lengkap.










Buku Alamat dapat bekerja bersama dengan IMAP / POP atau EAS. Microsoft Outlook dapat menggunakan EAS dan buku alamat untuk memberikan interaksi penuh dengan Outlook tanpa plug-in di klien!

Oktober - Manajer Infrastruktur Zextrasconfig (BARU)

Semua komponen Zextras dalam infrastruktur dapat dikelola melalui manajer Zextrasconfig khusus. Manajer ini memungkinkan administrator Zimbra untuk mengelola komponen Zextras di berbagai node infrastruktur, termasuk server Zextras Docs yang terintegrasi yang diperbarui dan server video Tim Zextras yang terintegrasi.

Oktober - Antarmuka Admin & Preferensi Pengguna (REFACTORING)

Antarmuka administrasi komponen Zextras baru ditulis dalam React. Dimulai dengan komponen Zextras, yang akan terus disajikan dalam antarmuka administrasi yang akrab, kami akan mentransfer semua opsi ke teknologi baru tanpa menghapusnya dari antarmuka yang sudah dikenal. Transisi penuh akan dilakukan di masa depan.

Oktober - Pemasang baru menggunakan sistem Pengemasan Linux asli (BARU)

Paket Zextras baru akan didistribusikan melalui repositori dengan dukungan Linux asli. Saat menginstal dan memperbarui paket, dimungkinkan untuk menggunakan perintah sistem seperti "apt-get install / update".

November - Server Documents (BARU)

Server Dokumen Zextras akan menjadi bagian dari infrastruktur, dikelola melalui Manajer Manajemen Infrastruktur Terpusat Zextrasconfig. Memasang dan memperbarui server pada masing-masing node tidak lagi diperlukan.

November - Tim: Fungsi baru Rapat (BARU)

Berkat server video baru, sejumlah fitur baru akan muncul di Tim Zextras. Diantara mereka:

  • Mode moderator
  • Kemampuan untuk membisukan suara untuk pengguna tertentu atau langsung untuk semua
  • Kemampuan untuk mengaktifkan mode webinar dan mode konferensi video
  • Log acara
  • Permintaan obrolan
  • Mode Walkie Talkie
  • Rekaman Hangout dapat diunggah ke Zextras Drive

November - Ketersediaan Tinggi (BARU)

Kemampuan untuk menyinkronkan pengguna, grup, domain, dan bahkan seluruh server email antara node infrastruktur yang berbeda. Kemampuan untuk mengkonfigurasi sinkronisasi dengan node cadangan atau node aktif lainnya di cluster. Sistem ini dirancang untuk mencapai ketersediaan 99,99% per bulan.

November - Drive Personal external Account (IMPROVED)

Kemampuan untuk menghubungkan akun Dropbox / Gdrive / nextCloud / OwnCloud

Desember - Tim: Rapat Instan (IMPROVED)

Kemampuan untuk membuat pertemuan instan dan menyediakan akses menggunakan tautan permanen dan sementara.

Desember - Tanda Email & Enkripsi (S / Mime-PGP) (BARU)

Kriptografi ujung-ke-ujung akan diterapkan di klien web. Sistem akan dapat mengenkripsi dan / atau menandatangani email yang dikirim untuk mendekripsi dan memverifikasi tanda tangan dari email yang diterima.

Banyak dari fitur ini sejauh ini eksklusif untuk versi komersial Zimbra. Sudah tahun ini, pengguna Zimbra Collaboration Suite Open-Source Edition akan dapat memanfaatkan ini dan fitur lainnya saat membeli Zextras Suite. Di masa depan kita akan berbicara secara rinci tentang fitur-fitur baru Zextras Suite.

Untuk semua pertanyaan yang terkait dengan Zextras Suite, Anda dapat menghubungi perwakilan perusahaan "Zextras" Ekaterina Triandafilidi melalui email katerina@zextras.com
