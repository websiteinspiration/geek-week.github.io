開発者向けの9つの便利なブラウザー拡張機能（2020のリスト）

ブラウザプラグインを使用して生活を簡素化する方法。

トリッキーなWeb開発者は、ブラウザをゴッドモードレベルで使用できます。拡張機能（プラグイン、アドオン）は、作業環境の改善とコード作成のパフォーマンスの改善に関して非常に役立ちます。

2020年はもうすぐです。多くの新しい拡張機能が登場しました。私はWeb開発者にとって最も便利な拡張機能をまとめ、1つのリストにまとめました。

洗練されたgithub


洗練されたGithubはGitHubインターフェースを簡素化し、有用な機能拡張を追加します。


洗練されたgithub

「GitHubがこれらの改善がどのように必要とされているかに気づき、それらのいくつかを追加してくれることを願っています。したがって、これらの改善点のいずれかが気に入ったら、GitHubサポートにこれについて書いてください。」
-洗練されたGithubの作成者。

インストールガイド：

  • クロム用
  • Firefox用
  • オペラのために。ChromeのプラグインをインストールするOperaのプラグイン

出版をサポートしているのは、モスクワのジュエリー工場でアプリケーションやサイトを開発している会社、エジソンです。

ハッカータブ


ハッカータブは、人気のあるオープンソースプロジェクトについて学びたい場合に便利です。新しいタブを開くたびに、すべての新しいプロジェクトのリストが表示されます。毎週、毎日、毎月のスケジュールを切り替えたり、プログラミング言語を切り替えることができます。


新しいブラウザータブの画面をGitHubトレンドプロジェクトに置き換えます。

インストールガイド：

  • クロム用
  • Firefox用

Daily 2.0-忙しい開発者のためのリソース


優れたソフトウェア開発者になるには、より多くのことを学び、最新のテクノロジーを最新の状態に保つよう努力する必要があります。


Daily 2.0

新しいおよび更新されたソフトウェア開発テクノロジーがインターネット上に毎日表示されます。最新の状態に保つことは大きな問題です。毎日は開発者のために開発者によって作成されました。この拡張機能のおかげで、インターネットで開発ニュースを常に検索するのではなく、コードに集中できます。

インストールガイド：

  • クロム用
  • Firefox用

コインなし


私たちのプロセッサに暗号通貨のマイニングを強制するサイトがあります。No Coinは小さなブラウザ拡張です。それはあなたの同意なしにあなたのコンピュータリソースを使用するCoinhiveのようなマイナーをブロックするために必要です。


何千ものユーザーから信頼されているコインはありません。この拡張機能は、マイナーがプロセッサを使用するのをブロックする安全で信頼できる方法を提供します。場合によっては、マイニングを必要とするキャプチャまたはショートリンクを入力する必要があります。この場合、コインは限られた時間の間鉱夫のブロッキングをキャンセルすることを許可しません。

インストールガイド：

  • クロム用
  • Firefox用

JSONフォーマッター


この拡張により、JSONが読みやすくなります。JSON Formatterはオープンソースプロジェクトです。


JSON Formatter

JSON Formatterは、構造化されたJSONテキストでアクションを実行する機能を追加します。他の同様のアプリケーションとは異なり、JSON Formatterは単にテキストを構造化します。この拡張機能は、ビューア、色、または圧縮機能を追加しません。

インストールガイド：

  • クロム用
  • Firefox用

ポモドーロタイマー



ポモドーロタイマー

の拡張ポモドーロ技術は時間管理技術です。この独自の手法は、6つの段階で構成されています。

  1. 実行するタスクを決定します。
  2. Pomodoroタイマーを開始します（通常は約25分）。
  3. 気を散らすことなくタスクに取り組みます。
  4. タイマーが鳴ったら作業を停止し、紙の上でボックスをチェックします。
  5. チェックマークが4つ未満の場合は、少し休憩を取り（3〜5分）、2番目のステップに進みます。
  6. それぞれの4番目の「トマト」の後、長い休憩を取ります（15〜30分）。チェックマークの数をゼロに減らし、最初のステップに進みます。

この記事を読んでいる間、私はポモドーロを使用しています。ポモドーロは物事を整理するための実用的な方法です。彼は自分の時間を集中して管理することを学ぶのを助けます。集中する方法を学びたいなら、この本をお勧めします。彼女は私を助​​けました。

インストールガイド：

  • クロム用
  • Firefox用

メタSEOインスペクター


通常、インターネットを使用する場合、メタデータは表示されません。この拡張機能は、Webページで見つけたメタデータを見つけて確認するのに役立ちます。


メタタグをチェックして適切なSEO評価を確認してください。この

拡張機能は主に、Googleのウェブマスターガイドラインに従ってサイトのHTMLコードを確認する必要があるウェブデベロッパーが使用します。また、通常は表示されないページのコンテンツを見たい人にも役立ちます。

メタデータが特定の範囲を満たさない場合-説明タグが短すぎるか長すぎる場合にアラートが表示されます。

SEOは別の言語です。「3か月から1位：The 2019」の本を読むことをお勧めします。ベストプラクティスとSEOのヒントがあります。

インストールガイド：

  • クロム用
  • Firefox用
