新しいWindowsターミナルプロファイルを作成します。json

これについて数か月間話しましたが、聞いたことがない場合は、新しい  Windowsターミナルがリリースされました。Windowsストアからダウンロードして、すぐに使用を開始できます  。すべてが無料でオープンソースです。

この記事の執筆時点では、Windowsターミナルはバージョン0.5前後です。まだ1.0として正式にリリースされていないため、常に更新されています。

そして問題は-Windowsターミナルを既にインストールしましたか？すでにprofile.jsonファイルを構成していますか？その場合は、profiles.jsonを削除してください。カットの下の詳細。



あなたのprofiles.jsonはどこかにあります：C：\ Users \ USERNAME \ AppData \ Local \ Packages \ Microsoft.WindowsTerminal_8wekyb3d8bbwe \ LocalState
だが、Windowsターミナルのドロップダウンから次のように



移動できます：「設定」をクリックすると、WindowsターミナルJSONファイルを処理するための登録済みアプリケーションを起動します。私の場合、これはVisual Studio Codeです。

私はprofiles.jsonで多くのカスタマイズを行ったので、profiles.jsonを削除または「ゼロ」にする前に、どこかにコピーを保存します。そして、あなたも忘れないでください！

「ctrl-a」を押すだけで、profiles.jsonをすべて削除できます。WindowsTerminal 0.5以降を使用すると、最初から再作成され、シェルが表示されます。覚えておいてください、  コンソールまたは端末シェルではありません！

新しいprofiles.jsonには、もう1つのヒントが含まれていることに注意してください！ Altキーを押しながら「設定」を押すと、デフォルト設定が表示されます。この新しいprofiles.jsonファイルはデフォルトで継承されるため、読みやすく、理解しやすくなっています。

//     .  : https://aka.ms/terminal-documentation

    "$schema": "https://aka.ms/terminal-profiles-schema",
