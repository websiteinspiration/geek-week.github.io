ほとんどのプログラマーはおそらく依存性注入の現象を知っていますが、誰もがその意味を理解しているわけではありません。おそらくインターフェースとコンテナーを処理する必要があり、時にはそれらを操作することで行き詰まりにつながりました。一方、依存関係注入について何か聞いたことがありますが、その本質をより深く理解したいので、この記事を開きました。この記事では、依存性注入の概念がいかに単純であり、実際にどれほど強力であるかを示します。



依存性注入は、単独で使用できる自己完結型のアプローチです。一方、このアプローチは、依存関係と依存関係の注入/管理の反転（DI / IoC）のコンテナーとコンテナーの両方に適用できます。このコンテキストで依存性注入を適用すると、私が最初に経験したいくつかの混乱が発生する可能性があります。



私のキャリア全体（私はNet / Cの開発を専門としています）で、依存関係注入を最も純粋な形で使用していました。同時に、コンテナーや制御の反転にまったく頼らずにDIを実装しました。最近、コンテナの使用が不可欠な仕事を与えられたとき、すべてが変わった。それから私は以前に知っていたすべてを強く疑いました。



このスタイルで数週間働いた後、コンテナーとインターフェースは依存関係の実装を複雑にするものではないことに気付きましたが、逆に、このパラダイムの機能を拡張しました。


（ここで注意することが重要です：インターフェースとコンテナーは、依存性注入のコンテキストでのみ使用されます。依存性注入は、インターフェース/コンテナーなしで実装できますが、本質的に、インターフェースまたはコンテナーの唯一の目的は、依存関係の実装を容易にすることです）。



この記事では、インターフェースとコンテナーがある場合とない場合の両方で、依存性注入がどのように行われるかを示します。それを読んだ後、依存関係注入の原理を明確に理解し、依存関係を実装するときに、いつどこでインターフェイスとコンテナを使用するかについて情報に基づいた決定を行えるようになることを願っています。

        private static float GetNumber(string message)
        {
            var isValid = false;
            while (!isValid)
            {
                Console.Write(message);
                var input = Console.ReadLine();
                isValid = float.TryParse(input, out var number);
                if (isValid)
                    return number;

                Console.WriteLine("Please enter a valid number. Press ^C to quit.");
            }

            return -1;
        }

        private static char GetOperator()
        {
            var isValid = false;
            while (!isValid)
            {
                Console.Write("Please type the operator (/*+-) > ");
                var input = Console.ReadKey();
                Console.WriteLine();
                var operation = input.KeyChar;
                if ("/*+-".Contains(operation))
                {
                    isValid = true;
                    return operation;
                }

                Console.WriteLine("Please enter a valid operator (/, *, +, or -). " +
                                  "Press ^C to quit.");
            }

            return ' ';
        }

        private static float GetResult(Calculator calc, float number1, float number2, 
            char operation)
        {
            switch (operation)
            {
                case '/': return calc.Divide(number1, number2);
                case '*': return calc.Multiply(number1, number2);
                case '+': return calc.Add(number1, number2);
                case '-': return calc.Subtract(number1, number2);
                default:
                    //    ,        
                    throw new InvalidOperationException("Invalid operation passed: " + 
                                                        operation);
            }
        }
    }
}
