Pythonで無料の駐車スペースを探す



私は良い都市に住んでいます。しかし、他の多くの場合と同様に、駐車スペースの検索は常にテストになります。空きスペースはすぐに占有されます。自分のスペースがあったとしても、駐車する場所がないため、友達があなたに電話するのは困難です。

そこで、私はカメラを窓の外に向けてディープラーニングを使用することを決定しました。スペースが利用可能になったときにコンピューターから通知されます



。必要なすべてのコンポーネントがすでにあります-それらを見つける場所とそれらを組み合わせる方法を知る必要があります。

では、Pythonとディープラーニングを使用して、正確な無料駐車場通知システムを楽しみましょう。

タスクを分解する


機械学習の助けを借りて解決したい難しいタスクがある場合、最初のステップは、それを一連の単純なタスクに分解することです。次に、さまざまなツールを使用してそれぞれを解決します。いくつかの単純なソリューションを組み合わせると、何か複雑なことができるシステムが得られます。

これが私がタスクを



分割した方法です。ウィンドウに向けられたWebカメラからのビデオストリームがパイプライン入力に入ります。



パイプラインを介して、ビデオの各フレームを一度に1つずつ送信します。

最初のステップは、フレーム内のすべての可能な駐車スペースを認識することです。明らかに、空いている場所を探す前に、画像のどの部分に駐車があるかを理解する必要があります。

次に、各フレームですべての車を見つける必要があります。これにより、フレームごとの各マシンの動きを追跡できます。

3番目のステップは、機械が占有している場所とそうでない場所を判別することです。これを行うには、最初の2つのステップの結果を組み合わせます。

最後に、駐車スペースが空くと、プログラムからアラートが送信されます。これは、ビデオのフレーム間のマシンの位置の変化によって決定されます。

これらの各ステップは、さまざまなテクノロジーを使用してさまざまな方法で実行できます。このコンベヤを構成する正しい方法や間違った方法は1つだけではなく、さまざまなアプローチには長所と短所があります。各ステップをさらに詳しく見ていきましょう。

一見すると難しいと思われる問題が発生した場合は、数分かけて問題を解決する別の方法を見つけてください。これは、いくつかの技術的な問題を回避するのに役立ちます。駐車場とは？ここは長い間駐車されている場所です。おそらく、駐車スペースをまったく認識する必要はありません。駐車場に立っていると思い込まず、長い間静止している車だけを認識してみませんか？

言い換えれば、駐車スペースは車が長い間立っている場所にあります。



したがって、車を認識し、フレーム間で移動しない車を見つけることができれば、駐車スペースはどこにあるかを推測できます。そのように単純です-車の認識に移りましょう！

車を認識する


ビデオフレームで車を認識することは、古典的なオブジェクト認識タスクです。認識に使用できる多くの機械学習アプローチがあります。「古い学校」から「新しい学校」への順序でそれらのいくつかはここにあります：

  • HOG（方位勾配のヒストグラム、方向勾配のヒストグラム）に基づいて検出器をトレーニングし、画像全体をウォークスルーしてすべての車を見つけることができます。ディープラーニングを使用しないこの古いアプローチは比較的高速に機能しますが、さまざまな方法で配置されたマシンには十分に対応できません。
  • CNN (Convolutional Neural Network, ) , . , , CNN, . , -, , HOG-.
  • Mask R-CNN, Faster R-CNN YOLO, CNN , . ( GPU), .

一般的なケースでは、最も簡単なソリューションが必要です。これは、正常に機能し、必要なトレーニングデータの量が最小になります。これは最新かつ最速のアルゴリズムである必要はありません。ただし、特に私たちの場合、Mask R-CNNはかなり新しくて高速であるにもかかわらず、合理的な選択です。

マスクR-CNNアーキテクチャは、画像全体のオブジェクトを認識し、リソースを効率的に使用するように設計されており、スライディングウィンドウアプローチを使用しません。つまり、かなり高速に動作します。最新のGPUを使用すると、ビデオのオブジェクトを数フレーム/秒の速度で高解像度で認識できます。私たちのプロジェクトではこれで十分でしょう。

さらに、マスクR-CNNは、認識された各オブジェクトに関する多くの情報を提供します。ほとんどの認識アルゴリズムは、各オブジェクトの境界ボックスのみを返します。ただし、マスクR-CNNは各オブジェクトの場所だけでなく、その輪郭（マスク）も提供します。



マスクR-CNNをトレーニングするには、認識したいオブジェクトの多くの画像が必要です。外に出て車の写真を撮り、写真にマークを付けることができました。これには数日かかる作業が必要になります。幸いなことに、自動車は人々がよく認識したいオブジェクトの1つであるため、自動車の画像を含むいくつかの公開データセットがすでに存在しています。

それらの1つは、人気のあるSOSO データセットです。（コンテキスト内の共通オブジェクトの略）、オブジェクトマスクで注釈が付けられた画像があります。このデータセットには、すでにラベル付けされたマシンを含む12,000以上の画像が含まれています。以下は、データセットのサンプル画像です。

しかし、馬を保持し、ニュースはさらに良いです！ COCOデータセットを使用してモデルをトレーニングしたいと思ったのは、私たちが最初ではありません。多くの人々がすでに私たちの前でこれを行い、その結果を共有しています。したがって、モデルをトレーニングする代わりに、すでに車を認識できる既製のモデルを使用できます。このプロジェクトでは、Matterportのオープンソースモデルを使用します。

カメラからの画像をこのモデルの入力に渡すと、これは「箱から出して」すぐに得られるものです。



モデルは、自動車だけでなく、信号機や人物などのオブジェクトも認識しました。彼女がその木を観葉植物として認識したのはおかしいです。

認識されたオブジェクトごとに、Mask R-CNNモデルは4つのものを返します。

  • 検出されたオブジェクトのタイプ（整数）。事前トレーニング済みのCOCOモデルは、車やトラックなどの80種類の一般的なオブジェクトを認識できます。完全なリストはここにあります。
  • 認識結果の信頼度。数値が大きいほど、モデルはオブジェクトの正しい認識に自信を持っています。
  • 画像内のピクセルのXY座標形式のオブジェクトの境界ボックス。
  • 境界ボックス内のどのピクセルがオブジェクトの一部であるかを示す「マスク」。マスクデータを使用して、オブジェクトの輪郭を見つけることができます。

以下は、事前トレーニング済みのマスクR-CNNおよびOpenCVモデルを使用するマシンの境界ボックスを検出するためのPythonコードです。

#    ,    .

    for i, box in enumerate(boxes):
        #     ,   .
        if class_ids[i] in [3, 8, 6]:
            car_boxes.append(box)

空の駐車スペースを認識


各マシンのピクセル座標を知っています。いくつかの連続するフレームを見ると、どの車が動かなかったかを簡単に判別でき、駐車スペースがあると想定できます。しかし、車が駐車場を離れたことをどうやって理解するのでしょうか？

問題は、車のフレームが部分的に重なっ



ていることです。そのため、各フレームが駐車スペースを表していると想像すると、実際には空であるにもかかわらず、マシンによって部分的に占有されていることがわかる場合があります。 「最も空の」フレームのみを検索するために、2つのオブジェクトの交差の度合いを測定する方法を見つける必要があります。

ユニオン上の交差（総面積に対する交差面積の比率）またはIoUと呼ばれるメジャーを使用します。 IoUは、2つのオブジェクトが交差するピクセルの数を計算し、これらのオブジェクトが占めるピクセルの数で除算することによって見つけることができます。



つまり、車の境界ボックスが駐車スペースのフレームとどれだけ交差するかを理解できます。これにより、駐車場が無料かどうかを簡単に判断できます。 IoUが0.15のように低い場合、車は駐車スペースのごく一部を占めます。そして、それが0.6のように高い場合、これは、車がほとんどのスペースを占有し、そこに駐車できないことを意味します。

IoUはコンピュータービジョンでかなり頻繁に使用されるため、対応するライブラリがこの方法を実装している可能性が非常に高くなります。マスクR-CNNライブラリでは、mrcnn.utils.compute_overlaps（）関数として実装されています。

駐車スペースの境界ボックスのリストがある場合は、コード全体を1行または2行追加することで、このフレームワークに車が存在するかどうかのチェックを追加できます。

    #      .
    car_boxes = get_car_boxes(r['rois'], r['class_ids'])

    # ,        .
    overlaps = mrcnn.utils.compute_overlaps(car_boxes, parking_areas)

    print(overlaps)

この2次元配列では、各行が駐車スペースの1つのフレームを反映しています。そして、各列は、各場所が検出されたマシンの1つとどれだけ強く交差するかを示します。 1.0の結果は、スペース全体が車で完全に占有されていることを意味し、0.02のような低い値は、車が少し所定の位置に上がったが、それでも駐車できることを示します。

空いている場所を見つけるには、この配列の各行をチェックするだけです。すべての数値がゼロに近い場合、その場所はおそらく空いています！

ただし、オブジェクト認識がリアルタイムビデオで常に完全に機能するとは限らないことに注意してください。Mask R-CNNに基づくモデルは非常に正確ですが、ビデオの1フレームで車が1つまたは2つ欠落する場合があります。したがって、場所が空いていることを断言する前に、ビデオの次の5〜10フレームの間、その場所が空いていることを確認する必要があります。このようにして、ビデオの1フレームのグリッチが原因でシステムが誤って場所を空とマークする状況を回避できます。場所が数フレームの間空いていることを確認したらすぐに、メッセージを送信できます！

SMSを送信


コンベアの最後の部分は、無料の駐車スペースが表示されたときにSMS通知を送信することです。

Twilioを使用すれば、Pythonからのメッセージの送信は非常に簡単です。Twilioは、ほんの数行のコードでほとんどすべてのプログラミング言語からSMSを送信できる人気のあるAPIです。もちろん、別のサービスをご希望の場合はご利用いただけます。Twilioとは何の関係もありません。最初に頭に浮かぶのはそれだけです。

Twilioを使用するには、試用アカウントにサインアップし、Twilio電話番号を作成して、アカウント認証情報を取得します。次に、クライアントライブラリをインストールします。

$ pip3 install twilio

その後、次のコードを使用してメッセージを送信します。

import numpy as np
import cv2
import mrcnn.config
import mrcnn.utils
from mrcnn.model import MaskRCNN
from pathlib import Path
from twilio.rest import Client

# ,     Mask-RCNN.
class MaskRCNNConfig(mrcnn.config.Config):
    NAME = "coco_pretrained_model_config"
    IMAGES_PER_GPU = 1
    GPU_COUNT = 1
    NUM_CLASSES = 1 + 80  #   COCO  80  + 1  .
    DETECTION_MIN_CONFIDENCE = 0.6

#    ,    .
def get_car_boxes(boxes, class_ids):
    car_boxes = []

    for i, box in enumerate(boxes):
        #     ,   .
        if class_ids[i] in [3, 8, 6]:
            car_boxes.append(box)

    return np.array(car_boxes)

#  Twilio.
twilio_account_sid = ' Twilio SID'
twilio_auth_token = '   Twilio'
twilio_phone_number = '   Twilio'
destination_phone_number = ',   '
client = Client(twilio_account_sid, twilio_auth_token)

#   .
ROOT_DIR = Path(".")

#       .
MODEL_DIR = ROOT_DIR / "logs"

#       .
COCO_MODEL_PATH = ROOT_DIR / "mask_rcnn_coco.h5"

#   COCO  .
if not COCO_MODEL_PATH.exists():
    mrcnn.utils.download_trained_weights(COCO_MODEL_PATH)

#     .
IMAGE_DIR = ROOT_DIR / "images"

#      —   0,   ,   .
VIDEO_SOURCE = "test_images/parking.mp4"

#   Mask-RCNN   .
model = MaskRCNN(mode="inference", model_dir=MODEL_DIR, config=MaskRCNNConfig())

#   .
model.load_weights(COCO_MODEL_PATH, by_name=True)

#   .
parked_car_boxes = None

#  ,     .
video_capture = cv2.VideoCapture(VIDEO_SOURCE)

#         .
free_space_frames = 0

#    SMS?
sms_sent = False

#      .
while video_capture.isOpened():
    success, frame = video_capture.read()
    if not success:
        break

    #      BGR  RGB.
    rgb_image = frame[:, :, ::-1]

    #    Mask R-CNN   .
    results = model.detect([rgb_image], verbose=0)

    # Mask R-CNN ,       .
    #     ,     .
    r = results[0]

    #  r    :
    # - r['rois'] —      ;
    # - r['class_ids'] —  () ;
    # - r['scores'] —  ;
    # - r['masks'] —   (    ).

    if parked_car_boxes is None:
        #     — ,       .
        #            .
        parked_car_boxes = get_car_boxes(r['rois'], r['class_ids'])
    else:
        #   ,  . ,   .

        #     .
        car_boxes = get_car_boxes(r['rois'], r['class_ids'])

        # ,         .
        overlaps = mrcnn.utils.compute_overlaps(parked_car_boxes, car_boxes)

        # ,    ,      .
        free_space = False

        #        .
        for parking_area, overlap_areas in zip(parked_car_boxes, overlaps):

            #       
            #    (, ).
            max_IoU_overlap = np.max(overlap_areas)

            #         .
            y1, x1, y2, x2 = parking_area

            # ,   ,   IoU.
            if max_IoU_overlap < 0.15:
                #  !     .
                cv2.rectangle(frame, (x1, y1), (x2, y2), (0, 255, 0), 3)
                # ,        .
                free_space = True
            else:
                #     —   .
                cv2.rectangle(frame, (x1, y1), (x2, y2), (0, 0, 255), 1)

            #   IoU  .
            font = cv2.FONT_HERSHEY_DUPLEX
            cv2.putText(frame, f"{max_IoU_overlap:0.2}", (x1 + 6, y2 - 6), font, 0.3, (255, 255, 255))

        #       ,   .
        #   ,  ,    
        #      .
        if free_space:
            free_space_frames += 1
        else:
            #   ,  .
            free_space_frames = 0

        #       ,  ,   .
        if free_space_frames > 10:
            #   SPACE AVAILABLE!!  .
            font = cv2.FONT_HERSHEY_DUPLEX
            cv2.putText(frame, f"SPACE AVAILABLE!", (10, 150), font, 3.0, (0, 255, 0), 2, cv2.FILLED)

            #  ,     .
            if not sms_sent:
                print("SENDING SMS!!!")
                message = client.messages.create(
                    body="Parking space open - go go go!",
                    from_=twilio_phone_number,
                    to=destination_phone_number
                )
                sms_sent = True

        #    .
        cv2.imshow('Video', frame)

    #  'q',  .
    if cv2.waitKey(1) & 0xFF == ord('q'):
        break

#  'q',  .
