5.アルゴリズムのコードへの変換


明白なステップ、必要なものはすべて言語の構文に含まれていることがよくありますが、そうでない場合でも、この問題を解決するためにクラスや関数を作成することに煩わされることはありません。私はこのアルゴリズムを教育大学プログラムの一部として実装したので、実装にはOctaveしか使用できなかったので、あなたに提示するすべてのコードは構文内にあります（Pythonに非常によく似ているため、書式設定にはそのハイライトを使用します）：

A = input("Enter the matrix A: ")
numOfRows = size(A, 1)
numOfCols = size(A, 2)

if numOfRows ~= numOfCols
    disp("Wrong matrix.")
else
    n = numOfCols
    disp("----------------")
    L = eye(3)
    disp("Making U = A")
    U = A
    disp("----------------")

    for i=2:n
        disp("Step "), disp(i-1)
        for j=1:i-1
            L(i,j) = U(i,j)/U(j,j)
            U(i,:) = U(i,:) - U(j,:)*L(i,j)
        end
        disp("----------------")
    end

    disp("Check the answer of L*U: "), disp(L*U)
end
