Android向けGoogle Maps SDKのインターフェースの操作

この記事は、これまでにGoogle Maps SDKを使用したことがない人に役立ちます。

猫の下で、マーカーの追加と管理、地図上でのカメラの移動方法、ズームの制御、ルートのプロット、ジオコーディングなど、地図を操作する基本的な方法について説明します。制限とそれらを回避する方法と同様に。

ソース

記事を書くために、私は自分の経験に励まされました。その経験は、Googleマップを使用して宅配便のアプリケーションを作成するときに得たものです。したがって、すべてのスクリーンショットとビジネスロジックの可能な言及は、宅配便のインターフェースを構築する状況で発生します。

残念ながら、Android向けGoogle Maps SDKでは、いわゆるコントロールボタンの位置を変更することはできません。 UIコントロールには、次のものが含まれます。IndoorLevelPicker-建物のフロアプランを表示、コンパス-コンパス、現在地ボタン-現在地のマップに移動、マップツールバー-ルートを作成してマップを開くためのボタン、およびZoomControls-マップの縮尺を増減します。

マップツールバーとZoomControlsを例として使用して、コントロールの位置を変更できないために発生する可能性のある問題と、その回避方法を見てみましょう。


SDK（オレンジ色で強調表示）とその対応するカスタム（緑色で強調表示）からのUIコントロールの表示に関する問題

最初に行う必要があるのは、元のボタンの表示を非表示にすることです。これを行うには、onMapReadyメソッドをオーバーライドします。カードが使用できるようになると呼び出されます。

ジオコーダーに関するいくつかの単語

ジオコーダーは、座標に基づいて住所を取得するために使用されます。地図上にマーカーを置き、[注文を追加]ボタンをクリックすると、目的のポイントの地理座標が取得されます。緯度と経度。ただし、ユーザーの便宜のために、住所を人が読める形式（国、都市、通り、家など）で表示すると便利です。

Google Maps SDKには、Geocoderクラスが含まれています。そのgetFromLocationメソッドを呼び出すことにより、指定した座標の住所の配列を取得できます。

特にインターネットが遅いかアクセスできない場合に、呼び出しによってUIスレッドを長時間ブロックしないようにするために、RxJavaを使用します。

public class LocationRepostiory {
    private Context context;
    private Location location;


    public LocationRepostiory(Context context, Location location) {
        this.context = context;
        this.location = location;
    }

    public Single<String> getLastLocation() {
        return Single.create(this::subscribeOnLocation);
    }

    private void subscribeOnLocation(SingleEmitter<String> e) {
        Geocoder geocoder = new Geocoder(context, Locale.getDefault());

        String errorMessage = "";

        List<Address> addresses = null;

        try {

//    
            addresses = geocoder.getFromLocation(
                    location.getLatitude(),
                    location.getLongitude(),
                    // In this sample, get just a single address.
                    1);
        } catch (IOException ioException) {
//,      I/O.
            errorMessage = context.getString(R.string.service_not_available);
            Log.e(TAG, errorMessage, ioException);

        } catch (IllegalArgumentException illegalArgumentException) {

// ,      
            errorMessage = context.getString(R.string.invalid_lat_long_used);
            Log.e(TAG, errorMessage + ". " +
                    "Latitude = " + location.getLatitude() +
                    ", Longitude = " +
                    location.getLongitude(), illegalArgumentException);
        }

//  ,    
        if (addresses == null || addresses.size() == 0) {
            if (errorMessage.isEmpty()) {
                errorMessage = context.getString(R.string.no_address_found);
                Log.e(TAG, errorMessage);

            }

        } else {
            Address address = addresses.get(0);
            ArrayList<String> addressFragments = new ArrayList<String>();

//      .
            for (int i = 0; i <= address.getMaxAddressLineIndex(); i++) {
                e.onSuccess(address.getAddressLine(i));
            }

        }
    }
}
