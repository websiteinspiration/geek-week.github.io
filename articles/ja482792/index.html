建物


2019年の最も重要なイベントは、トカマクの建物のトップアウトです。すでに2020年3月には、建設が完了し、クレーンが予備組立棟（2017年にオープン）からトカマク棟に移動し、その結果、原子炉シャフト内の原子炉の組立が始まることが約束されています。


トップアウト！

はい、プロジェクトはこの時期に長い間続きました-ピットの掘削は2010年に始まり、耐震基礎の充填は2011年に始まり、「作業床」の建設は2015年の終わりに始まりました（かなり長い休止は、福島原子力発電所の事故後の建物の再設計に関連していました）。そして今-デザインの高さに達しました！興味深いことに、2014年の計画によると、これは2019年7月に発生するはずでした。一般に、タスクはほぼ遅延なく完了したと言えます。


トカマクの建物内では、中性子放射の吸収と放射性同位元素の不拡散に対する障壁の両方として機能する60トンのドアによって、原子炉シャフトへのアクセスがブロックされます。

最初のプラズマに必要な約40の建物と構造のうち、ほとんどすべてがすでに準備ができているか、または建設の最終段階にあります。準備されていないもののうち、管理棟、マグネットエネルギーダンプ抵抗器（これらの抵抗器はロシアで製造されています）、およびトリチウム棟が約半分製造されていることに注目する価値があります。ただし、残りの6年間は完全に完成し、機器で飽和させることができます。


完成したサイトのレンダリング。灰色はすでに構築され飽和している機器を示し、紫-まだ建設中のトカマクの建物、青-将来の建物。トカマクの建物自体の周りにあるこのすべての青い建物は、最初のプラズマには必要なく、後で建てられます。


さらに、2019年には、磁気システムの電力変換器の完成した建物、無効電力補償機器の建物、2018年の終わりには、放熱システムの建設も手渡しました。

インフラストラクチャ要素


本格的な打ち上げのITER複合施設は、最大の電力消費者になります-支援システムでは約110メガワット、磁石の暖房および電源システムでは最大250メガワット。これらはすべて、400キロボルトの開閉装置に接続された7つの変圧器と2つのASUステーションで構成される複雑なマルチレベルシステムを持つシステム間で分散されます。最初の部分、つまり、110メガワットの負荷を提供する一般的な配電中電圧デバイスは、2019年1月に委託され、ごく少数の消費者（ビルダーおよび設置業者）の供給を引き継ぎました。この入力により、トカマクのすべての主要インフラストラクチャ設備（クライオコンバイン、熱放出システム（これら2人の消費者が負荷の大部分を占める-2人でほぼ100 MW）のテスト）をテストできるようになります。しかし、地元の変電所と配電網の建設に関する作業はまだ進行中です。


将来、プラズマ加熱システムに給電される新しい配送センター（負荷センター）

また、2019年には、可変負荷（磁石、加熱システム）の電力サブシステムからの最初の400 MVARトランス（3つのうち）が稼働しました。磁気システムの電力変換器のテストに使用されますが、2年以内に実施されます。


ITERサイトを屋外の開閉装置側から見た図は、400キロボルトで、一定の負荷のトランス（中央右側）とパルス負荷（中央左側）です。マグネットパワーコンバーターの2つの建物がクライオコンバインの左側にあります（黄色のガスタンク付き）。

2019年に、熱放出システムの設備が積極的に設置されました-これは、異なる水化学と信頼性レベルの5つの自律水循環システム、合計容量が約300メガワットの10個のファン冷却塔、2つの温水と冷水のバッファープール、および4つです。ダースポンプ、熱交換器など システム全体は、起動時にトカマクとその補助システムから最大1150メガワットの熱を受け取る必要があり、この熱を緩衝すると、一時的に停止します。ただし、最初のプラズマの場合、このシステムのパワーが可能性のごく一部で使用されることは明らかです。

クライオコンバイン


クライオコンバインは、2019年に世界で最大の液体ヘリウム製造施設の1つです...積極的に再設計されました。原則として、これは複雑な「その種の最初の」プロジェクトの惨劇です。膨大な数の関係が、いくつかの小さなことの無視が大きな変化につながるという事実につながります。特に、ITERのエンジニアが私に説明したように、負荷の見直しにより、設備のわずかな増加と換気および空調システムの追加が必要になり、その総容量は屋根の能力よりも高くなり、一部の空調機は別館とすべての換気ルートに移動する必要がありました。再設計。したがって、小さな変更により、ここでの機器の設置が1年間停止されました。

磁気コンバーター


動作中のITER超伝導磁石は、最大46ギガジュールを蓄積し、最大68キロアンペアの電流で動作します。さらに、動作中のトカマクでは、磁石の電流をかなり急速に変化させる必要があります。これは、強力な電流源を意味します電流による磁石の「ポンピング」と「ポンピング」。 2つの建物に約40の個別のコンバーターがあり、これらは大型の制御多相整流器です（最大のコンバーターは最大90メガワットの電力であり、すべてのコンバーターの合計電力は2.1 GWです）。特に電流を変更するには電力が必要なので、磁気システムは無効電力補償システムとペアになります。大まかに言えば、一連のコンデンサとインダクタンスが交流ネットワークで切り替えられます。これにより、抽出された磁気エネルギーの一部を保存し、次のサイクルで高電圧電力線を「引っ張る」ことなく戻すことができます。


無効電力補償システムの構築。フレームで興味深いのは、バックグラウンドの高電圧インダクタです。

2019年には、コンバーターの両方の建物に、コンバーターと磁気フィーダーを接続するバスバー（ロシア製）の設置が始まり、インバーターブロック自体を設置するための基盤の準備が始まりました。変圧器の設置もあります（各インバータは入力トランスに依存します）。これは、一般的な計画ではっきりとわかります（建物の外にあります）。


磁気コンバーターの建物内のロシアのバスバー（黄色）。まだコンバータはありません。

2020年には、インバーターが設置され、すべてのコンポーネントが結合されますが、電気テスト自体はまだ遠く離れています。

真空システム


ITERの一部として非常に重要なシステム。400台もの真空ポンプと10 kmの真空パイプが含まれます。 2018-2019年に彼女は再設計ウイルスに襲われたようです、いずれにしても、トリチウムの建物のその部分の建設は、いわば、数十の主排気ポンプが配置された真空ワークショップが、2018年半ばから立っています。ただし、この部屋の上の階では、トリチウムビルディングの床全体が別のアクティブに変化するシステム-2018年に米国からEUに移管されたタスクであるトカマクの水冷のために予約されています。しかし、真空システムのいくつかの新しい要素が点灯しました。

クライオスタットは、磁気システムとともにトカマクが配置される真空容器です。ここの真空は主に、かなり高温の真空チャンバーと周囲の建物からの非常に低温の磁石を断熱するためのものです。 2019年に、「クライオスタットの下側シリンダー」の製造が完了しました。これは、クライオスタットの下から2番目の部分（4つのうち）であり、クライオスタットのベースとの作業はほぼ終了しています-これは最下部です。 2020年3月にベアリングにベースを取り付け、次にベースに下部シリンダーを取り付けることから、ITERアセンブリが開始されます（詳細は以下を参照）。実際、サーマルクライオスクリーン、センサー、およびそれらのケーブルライン用の何百もの支持要素をクライオスタットの両方の部品にまだ溶接する必要はありませんが、この作業は、残りの月にも、鉱山への原子炉の設置後でも行うことができます。


クライオスタットのベース、2019年夏。主要部品の組み立てが完了し、コイルサポートのベースも用意されています。

ちなみに、クライオスクリーンもサイトに向かっています。それらは、ヘリウムが80-100 Kの温度で流れる溶接された冷却管と赤外線放射の反射を改善するために銀メッキされた10-20 mm厚のステンレス鋼シートです。一部のクライオスクリーンは、鉱山に設置する必要がある最初のアセンブリに含まれているため、生産が予定どおりに完了したことを喜んでお知らせします（韓国で使用されています）


サーマルクライオスクリーン。具体的には、これは真空チャンバーとトロイダル磁石の間にあるスクリーンの要素です。

超電導マグネット


以前私の記事を読んだらITERによれば、私は国際熱核融合炉の主要な超伝導磁石の壮大さを賞賛することに飽き飽きしていません。実際、25の大型ITER磁石はすべて、世界で25の最大の超電導磁石になります。最初のプラズマでは、それらすべてを収集する必要があります。ただし、組み立て順序によって、どの磁石が最も優先されるかが決まります。実際、すでに今年は少なくとも最初の2つの磁石を鉱山に設置する必要があります。これらはトカマクチャンバーの下に配置される下部ポロイダルPF6およびPF5です。 1つ目は中国製でKadarashの方向に移動し、2つ目は現在ITERサイトで最終生産作業を行っています。両方の磁石には、（現場での）クライオテストとセンサーを備えた追加の機器がありますが、夏の終わりまでに、それらは設計位置に下げられます。重量（それぞれ〜400トン）と寸法（直径10および18メートル）を考えると、設置作業は非常に壮大です。

トロイダル磁石はヨーロッパと日本で組み立てられています。特に、1年前、ヨーロッパの「クリスマスツリーの下」で、ケースの半分をワインディングバッグにスライドさせる操作を行い（日本では2019年3月にこれを行いました）、2019年を通じて最初の「戦闘」磁石を準備し続けました。これを行うには、ケース内の巻きバッグを正確に露出し、ケースの半分を溶接し、バッグが挿入された蓋を溶接し、内部をエポキシで満たす必要がありました。これらはすべて正常に完了し、最後の操作は残りました-組み立ての不正確さのために残された許容値に従って本体を機械加工しました。この技術の複雑さは、製品の寸法が16x10x3メートルの3つの軸で1 mm以内の実磁気軸と理論磁気軸の一致を取得する必要があるためです。

真空室


最も有望なタイプの制御核融合炉としてのトカマクの形成の黎明期でさえ、エンジニアはチャンバーのトロイダル形状が産業にとって技術的な悪夢であると指摘しました。ただし、現実ははるかに悪いことが判明しました：トロイダルチャンバーだけでなく、高い剛性（したがって、厚くて多数のリブ）を備えた2壁のチャンバー、溶接の極端な要件、および形状精度の法外な要件（これは、二重曲率の表面上にあります-定規とテンプレートでさえ、正確さを測定しないようにとても簡単です）。

ITER真空チャンバーは9つのセクターから組み立てられ、そのうち4つは韓国（現代重工業）で製造され、5つはヨーロッパ（Walter Tosto / Ansaldo / ENSA）で製造されます。製造サイクルには、ワークピースのホットスタンピング、その機械加工、中間機械加工による4段階の拡大溶接が含まれます。これらすべてに、不均一な形状の多くの機器と独自の光学計測が必要です。物事は非常にゆっくりと進んでいます-韓国人は2012年に最初のセクターの金属の切断を開始し、2019年の秋にのみ、完成したセクターの4つのセグメントを溶接しました。ヨーロッパは約2年遅れており、私の考えでは、2021年末までに1セクターを発行することはできません。これは、高い確率で、最初のITERプラズマの計画が1年間ずれることを意味します。


2019年の秋までに、ヨーロッパ人は最初のセクターの1つのポロイダルセグメント（4つのうち）の内部の溶接を終了しました。韓国人は2年前にそのような進歩を遂げました。


2019年9月に韓国人は、単一のセクターで4つの完成したセグメントの溶接を開始しました。引き続き、上部ポートとダイバータポートのパイプの溶接、最終的な測定と機械加工、そして発送を行っています。

真空チャンバーのプロジェクトには、インド（ホウ素鋼から中性子吸収材のブロックを作る）とロシア（ドイツでトップパイプを製造するために支払う、それぞれ18トンの重量があります）の貢献もありますが、ここではすべてが問題なく、ドラマや感情はありません。


ロシア連邦の命令によりMANによって製造された、真空チャンバーの上部ノズルの1つ。

最後に、房内装置がある- ダイバータ、最初の壁、ブランクの「暖かい」磁石でELFの不安定さ、多数のセンサー、送水管を抑制します。ただし、これは最初のプラズマには必要ありません。したがって、ITER作業のこの（非常にハイテクな）領域には進歩と成果がありますが、今日はこのトピックをスキップします。

制御システム


大規模な工業プロジェクトでは、プロセス制御システムの調整により、プロジェクト全体の立ち上げの遅延が決まることがよくあります。第一に、それは論理的に最終段階に進み（非設置機器の制御システムを構成することは不可能です）、第二に、デバッグ中にポップアップするすべての設置、製造などのエラーを収集します。ここで、ITERは完全なビンゴをまとめるリスクを冒しています。プロジェクトが非常に複雑であるだけでなく、多くのコンポーネントも最初のものであり、さまざまな国から来ており、ローカル制御システムを含んでいます。管理スタックとしてのオープンソースソフトウェアの形でとられた予防策にもかかわらず（RHEL + CODAC + EPICS）およびソフトウェアとハ​​ードウェアのキットを関心のあるすべての人に配布し、数万のセンサー、数千のアクチュエータ（多くの場合、独自のソフトウェアとハ​​ードウェアを「内部」に持つ）を備えたシステムを発売します。施設は非常に困難な作業になります。
