Comment activer Google Documents hors ligne dans le nouveau Microsoft Edge

Le nouveau navigateur Edge de Microsoft sur le moteur Chrome a récemment reconstitué le zoo du navigateur basé sur Chromium. Mais il se compare favorablement avec le reste qu'il est fourni avec Windows. Et cela signifie que tous les tests d'ordinateurs portables sur batterie seront effectués avec lui, et Microsoft devra surveiller attentivement cela. D'une transition complète vers Edge, j'ai été arrêté par le mode inactif "hors ligne" dans Google Office: documents, tableaux et présentations. Une chose irremplaçable dans les vols et les trains. Mais maintenant que vous pouvez installer des extensions pour Chrome dans Edge, quelques manipulations simples vous permettent de tromper les services Google et de faire fonctionner hors ligne!

1. Installez l'extension cachée "Google docs offline"


La recherche dans le magasin d'extensions ne l'affichera pas, vous devez donc utiliser le lien direct .



L'installation de cette extension uniquement n'activera pas le mode hors connexion: les services Web Google vérifient que les demandes proviennent du navigateur Google Chrome. Ce qui est généralement raisonnable: la fonctionnalité est encore expérimentale, nécessite une expansion et son déploiement sur tous les navigateurs possibles peut provoquer de nombreuses pannes inattendues. Mais si nous savons ce que nous faisons et sommes prêts à courir un risque raisonnable, nous passons à l'étape suivante.

2. Installez l'extension «User-Agent Switcher and Manager»


Disponible à la fois dans les extensions du magasin de recherche et par lien direct .
