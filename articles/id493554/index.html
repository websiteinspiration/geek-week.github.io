Pratinjau Terminal Windows v0.10

Memperkenalkan Terminal Windows v0.10! Seperti biasa, Anda dapat mengunduhnya dari Microsoft Store , atau dari halaman rilis di GitHub . Di bawah kucing, kita akan melihat lebih dekat pada detail pembaruan!



Input mouse


Terminal sekarang mendukung input mouse dalam aplikasi Windows Subsystem for Linux (WSL), serta dalam aplikasi Windows menggunakan input terminal virtual (VT). Ini berarti bahwa aplikasi seperti tmux dan Midnight Commander akan mengenali klik pada item di jendela Terminal! Jika aplikasi dalam mode mouse, Anda dapat menahan shift untuk membuat pilihan alih-alih mengirim input VT.



Perbarui Pengaturan


Duplikasi panel


Sekarang Anda dapat membuka panel baru dengan menduplikasi profil dari panel yang dipilih, hanya berfokus padanya dan menekan kombinasi tombol. Untuk melakukan ini, di bagian "keybindings" dari profiles.json Anda, Anda perlu menambahkan "splitMode": "duplikat" ke "splitPane" . Anda dapat menggunakan parameter lain, seperti "commandline" , "index" , "startingDirectory" atau "tabTitle" . Jika Anda ingin tahu lebih banyak tentang opsi ini, saya sarankan Anda membaca artikel ini .

{"keys": ["ctrl+shift+d"], "command": {"action": "splitPane", "split": "auto", "splitMode": "duplicate"}}




Koreksi kesalahan


  • Tampilan teks yang ditingkatkan secara signifikan saat mengubah ukuran jendela;
  • Batas-batas tema gelap ditetapkan (mereka tidak lagi putih);
  • Jika bilah tugas disembunyikan dan Terminal Anda diperluas, sekarang bilah tugas itu muncul secara otomatis ketika Anda mengarahkan kursor ke bagian bawah layar;
  • Azure Cloud Shell sekarang dapat meluncurkan PowerShell dan mendukung input mouse, dan juga dapat diinstal sebagai shell pilihan Anda;
  • Kecepatan gulir telah diubah saat menggunakan touchpad atau layar sentuh.

Rencana masa depan


Kami ingin memberi Anda informasi terbaru tentang rencana kami sehingga Anda tahu apa yang diharapkan dalam beberapa bulan mendatang. Saat ini kami sedang mengerjakan perbaikan bug untuk mempersiapkan rilis v1. Windows Terminal v1 sendiri akan dirilis pada bulan Mei. Setelah itu, kami berencana untuk merilis pembaruan berikutnya pada bulan Juni untuk melanjutkan siklus pembaruan bulanan kami. Edisi kami akan terus tersedia di Microsoft Store, dan di GitHub!

Akhirnya


Seperti biasa, jika Anda ingin meninggalkan ulasan atau memiliki pertanyaan, jangan ragu untuk memposting ke Kayla ( @cinnamon_msft ) di Twitter. Selain itu, jika Anda ingin membuat proposal untuk meningkatkan Terminal atau melaporkan kesalahan di dalamnya, maka silakan hubungi kami di GitHub untuk ini . Kami harap Anda menikmati rilis Terminal Windows ini!
