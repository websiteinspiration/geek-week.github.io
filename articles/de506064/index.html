Entwickeln Sie eine interaktive Webanwendung in C # ohne JavaScript-Kenntnisse

Ich bin ein .NET-Entwickler bei Turkmen-Tranzit.

Mein Premierminister brauchte jemanden, der Blazor fühlen wollte. Meiner Meinung nach habe ich in diesem Bereich gute Arbeit geleistet. Während des Studiums stieß ich auf Probleme, deren Antworten im russischsprachigen Internet nicht so leicht zu finden sind.



Warum Blazor und nicht Standard-JavaScript?

Mit einfachen Worten, mit der Blazor-Technologie können Sie Webanwendungen erstellen, deren Clientseite aus HTML-Markup und C # -Code anstelle von JavaScript besteht.

Blazor kann auf das DOM zugreifen, um die Seite neu zu zeichnen, ohne die Seite
zu aktualisieren. Die Webanwendung auf Blazor besteht aus Komponenten. Komponenten dienen zur Anzeige visueller Elemente wie Download-Indikatoren oder Formulare zum Hochladen von Dateien. Das Markup der Elemente wird in einer Datei mit der Erweiterung .razor und der gesamten Logik in normalen C # -Klassen mit dem Namen {ComponentName} .razor.cs gespeichert.

Die Komponenten in Blazor sind in intelligent und dumm unterteilt.

Dumme Komponenten sind unabhängig von der gesamten Anwendung, sie können in andere Komponenten eingesteckt werden und sie können sogar andere dumme Komponenten verwenden.

Intelligente Komponenten sind wegwerfbar, müssen ihren Zustand beibehalten und werden sehr selten in anderen Komponenten verwendet.

Meistens müssen Sie intelligente Komponenten schreiben.

In Blazor können Sie JavaScript-Funktionen mit C # -Code aufrufen und umgekehrt .NET-Methoden mit JavaScript aufrufen.

Ja, wenn Sie über fundierte Kenntnisse in JavaScript verfügen, sich in JavaScript-Frameworks auskennen und gerade erst mit der Welt von .Net vertraut sind, sollten Sie sich nicht mit Blazor befassen. Die Technologie ist relativ neu und wenn Sie sie zum ersten Mal kennenlernen, können Sie Angst bekommen. Der Artikel richtet sich an Personen, die wissen, wie man C # verwendet, aber aus irgendeinem Grund kein JavaScript haben. Sprechen Sie über das vielversprechende Gegenstück von Microsoft, in dem Sie es sich leisten können, alle Funktionen von C # zu nutzen.

Beispielsweise entwickeln wir CRM mit einer Liste von Unternehmen, die möglicherweise unsere Partner sein könnten, mit automatisierter Suche nach Mitarbeitern des Unternehmens, automatisierter Verteilung von Nachrichten an alle potenziellen Kunden, Protokollierung von Benutzeraktionen und vielem mehr.
