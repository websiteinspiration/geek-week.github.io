ライブラリのインストール



TelebotはTelegram APIとやり取りするためのライブラリーであり、そのシンプルさに惹かれ、初心者には100％適していると思います。







ライブラリをインストールするには、ターミナルで次のコマンドを入力します。







pip install pytelegrambotapi
