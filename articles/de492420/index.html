Erstellung und Pflege von öffentlichen Konten


Etwas zwischen der Website des Unternehmens, seinem Newsfeed und einem Tool für die Interaktion mit Abonnenten. Zum Beispiel ein öffentlicher Bericht über Beijingvodokanal.



Nachrichten über ihre Arbeit werden im Feed veröffentlicht, und die Interaktion mit Benutzern wird im folgenden Menü implementiert. Sie können den Saldo Ihrer Zähler überprüfen, für Wasser bezahlen und eine Panne / einen Wassermangel melden. Alle diese Links führen zu dem Gadget, was es ist - etwas später.

In der Tat ist ein öffentliches Konto ein Mittel, um mit Benutzern zu kommunizieren und ihnen Informationen sowie Feedback zu übermitteln. Dies ist auch ein sehr praktisches Landing-Tool - Sie können mit einem Klick ein Konto abonnieren, das für einen Benutzer von Interesse ist. Jedes öffentliche (sowie private) Konto verfügt über einen eindeutigen permanenten Link - wir platzieren überall einen QR-Code - und der Benutzer hat uns bereits mit einem Klick abonniert.

Es gibt zwei Arten von öffentlichen Konten:

  1. Abonnementkonto ist es 订阅 号
  2. Dienstkonto, auch bekannt als 服务 号

Der Hauptunterschied liegt in Anwendungsfällen. In einem Abonnementkonto können Sie einmal täglich Massenversand in einem Dienstkonto durchführen - viermal im Monat. Im Dienstkonto können Sie jedoch Benutzerinteraktionsoberflächen auf hoher Ebene (OAuth, Schnittstelle für technischen Support, Wechat-Zahlungen usw.) aktivieren. Für das Abonnement stehen nur grundlegende Benutzeroberflächen zur Verfügung, und außerdem können keine Zahlungen verbunden werden. Ein Abonnementkonto kann sowohl bei einer juristischen Person als auch bei einem einzelnen Dienstkonto registriert werden - nur bei einer juristischen Person.

Das heißt, Abonnenten konzentrieren sich mehr auf Blogger, Prominente, die den Fans oft etwas erzählen, sind nicht schlecht für eine Gruppe an einer Universität, an der ein Schulleiter Massenmailings durchführen kann und so weiter. Der Service One ist ein Tool zur Landung und weiteren Arbeit mit Abonnenten.

Der dritte Typ, obwohl er bei der Registrierung verfügbar ist, ist Wechat (auch bekannt als 企业 微 信) kein öffentliches Konto mehr, sondern so etwas wie ein Task-Management-System, ein interner Chat eines Unternehmens wie Slack usw.

Natürlich können Sie Benutzer verwalten, ihnen Labels zuweisen, ihren Client-Status aus dem Gadget abrufen, gezielte Mailings nur für bestimmte Gruppen arrangieren und so weiter und so fort.

Das Menü am unteren Rand des öffentlichen Kontos kann ebenfalls problemlos an die Anforderungen des Entwicklers angepasst werden. Sie können Ihre Aktion an jede Schaltfläche anhängen, ein Untermenü erstellen usw.



Den folgenden Schaltflächen können folgende Aktionen zugewiesen werden:

  1. Automatische Antwort spezifische Nachricht
  2. Weiterleiten an eine bestimmte Site
  3. Starten Sie ein bestimmtes Gadget

In der Tat ist dies eine vollständige Alternative zur Website, der Benutzer kann vollständig mit dem Dienst interagieren, ohne Ihren geliebten Wechat zu verlassen.

Benutzerkommunikation


Es gibt zwei Möglichkeiten, die Kommunikation zu organisieren. Das erste ist einfache Kommunikation. Wenn Sie sich die Benutzeroberfläche eines öffentlichen Kontos ansehen, wird unten links eine Schaltfläche



angezeigt, mit der das Menü des öffentlichen Kontos in die Nachrichteneingabezeile geändert wird: Die hier gesendeten Nachrichten werden im Admin-Bereich der Nachrichtenoberfläche angezeigt.
Sie können beantwortet werden. Der Autor der Antwort ist das öffentliche Konto selbst und keine Einzelperson. Aber nur wenige Leute benutzen es, da die Benutzeroberfläche nicht speziell dafür ausgelegt ist und es sich eher um einen Atavismus aus der Vergangenheit handelt. Stattdessen gibt es automatische Antworten für bestimmte Muster. Das heißt, alles ist einfach - Sie legen ein Schlüsselwort im selben Admin-Bereich fest. Wenn es in der Nachricht des Benutzers vorhanden ist, wird eine automatische Antwort an ihn gesendet, die Text, Bild, Ton oder Video enthalten kann. Auf die gleiche Weise werden Antworten auf Nachrichten und Begrüßungsnachrichten konfiguriert.



Für die normale Kommunikation mit Kunden ist es viel besser, den technischen Support von Wechat zu nutzen. Alles ist auch sehr einfach - aktivieren Sie diese Funktion, fügen Sie einen Operator hinzu - und nach dem Anmelden wird ein vollwertiges Support-System mit der Ernennung von Operatoren, Warten, Feinabstimmung, Anzeigen von Kommunikationsprotokollen usw. aufgerufen. usw. Vergessen Sie auch nicht, dass alles, was oben beschrieben wurde, nur die Spitze des Eisbergs ist, der über die GUI im Admin-Bereich konfiguriert wird. Mit der Entwicklerkonsole und der API erweitern sich die Möglichkeiten zur Anpassung und Funktionalität um eine Größenordnung.
