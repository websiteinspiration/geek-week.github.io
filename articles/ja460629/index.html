2人の鉱夫が同じ世界を持っている確率

みなさん、こんにちは！最近、「Minecraftの2人のプレイヤーが同じ1つの世界を持っている可能性はありますか？」という質問に興味を持ちました。

実際のところ、Minecraftの世界は、指定されたシードからランダムに生成されます。手動で設定するか、政府の擬似ランダムで受信できます。同じシードが同じ世界を生成することは注目に値します。

このゲームは非常に人気があるため、すべてのプレーヤーに直接インタビューして、それぞれの世界を比較することはできません。ただし、このイベントの確率はいつでも計算できます。思われるかもしれません。必要なのは、このイベントを満たす基本結果の数を計算し、それをすべての基本結果のセットに分割することだけです。残念ながら、これは非常に重要な作業なので、「誕生日のパラドックス」について思い出しました。

逆説自体は、50％の確率で23人のグループで、2人の誕生日が同じであるという事実にあります。明らかに、タスクは私たちのものに似ています。どのように解決されましたか？非常に単純です。グループ内の各人に一意の誕生日がある可能性を計算する方がはるかに簡単であることがわかりました。これを行うには、1人の人を取り、彼の誕生日を覚えてから2番目の人を選びます。彼の日が最初の人と一致しない可能性は等しくなります。

p2=1−1365

それら。100％から誕生日が同じである可能性を差し引いたもの。3番目のものを使用して、彼の誕生日が前の2つと一致しない確率を考慮します。

p3=1−2365

以下同様にn人目まで

pn=n−1365

次に、グループ内の誰も一致しない確率

p=1∗(1−1365)∗(1−2365)∗...∗(1−n−1365)

そして、少なくとも2つが一致する確率

p=1−p



このソリューションを私たちのケースに適用するだけです。Minecraftには2 ^ 64の可能なシードしかなく、約2億人のプレイヤーがいます。したがって、式は次のようになります

p=1∗(1−1264)∗(1−2264)∗...∗(1−2∗108264)

これを手動で数えることは非常に時間がかかるので、私の代わりにこれを行う小さなプログラムをPython 3で作成しました。



誰かが興味を持っているなら-ここにプログラムコードがありますが、それは非常に簡単です。

a = 2**64
n = 200000000
p = 1

for i in range(n):
    p *= (1 - i/a)

print('Chance that 2 players of minecraft have the same seed: ' + str((1-p)*100) + '%')


ちなみに、可能な種子の数を考えると、0.1％とかなり多くなっています。
