幽灵Windows 3.1


最近，我在Goodwill二手产品上徘徊，遇到了一款专为Windows 10设计的廉价平板电脑。虽然新版售价超过100美元，但我只花了几美元。驱动程序从他身上飞了出来，我试图恢复它们，然后所有任务，甚至加载“开始”菜单，都开始非常缓慢。这使我安装了CloudReady，它是Chrome操作系统的一种变体，虽然不支持内置的WiFi，蓝牙，亮度，声音和其他功能，但在平板电脑上可以很好地运行。 （一个外部WiFi加密狗对我有帮助！）事实上，这对我来说是一个挑战，对Mac Mini来说，这是一个10美元的实验。但这是一个奇怪的，几乎没有用的设备（MacBook Pro不会与这种加密狗相比大惊小怪。）使我想到了使用Windows的奇怪环境，这反过来又引发了Windows 3.1的思考，Windows 3.1是该操作系统的第一个真正的主导版本。在本文中，我将讨论Windows 3.1的异常应用程序。

“您可以用笔直接在表面上画画，留下手写的便笺，就像在纸上一样，计算机会识别出来。因此键盘是可选的。这样的机器可以带您去开会或与客户见面，以非常自然的方式使用它。我们将提供使用Windows进行Pen Computing的便利。”

因此，比尔·盖茨 （Bill Gates）在1991年的营销视频中谈到了Windows for Pen Computing的好处-这是专为第一台平板电脑设计的操作系统。该程序于1992年作为Windows 3.1的一个分支启动，现已成为许多支持手写功能的系统之一，其中包括GoPoint的PenPoint。 （苹果也尝试过然而，她的平板电脑从未进入市场。相反，该公司专注于创建Newton PDA。）当时，对该行业抱有强烈的怀疑态度，事实证明这是合理的。尽管有一些成功的产品用于商业市场，例如GRiDPad，但它还是花了将近20年的时间才出现在竞争性消费类平板电脑市场上，而不是由Microsoft创造的。

但是出于某种奇怪的原因，Radio Shack的母公司Tandy的家伙对此一无所知，并决定尽管如此，还是值得复制CD-i模型。


他们的工作成果就是“坦迪视频信息系统（VIS）”。在美国的Radio Shack商店以Memorex品牌出售的成品设备（并用这部极为刺激的视频吹捧）是一个奇怪的选择。此外，当CD-i和类似的设备Commodore CDTV（实际上是著名的Amiga，但令人惊讶的是与CD32不相似）一起进入市场时，它就进入了市场。

1992年，费城问询者公司（French Inquirer）在多媒体市场上的一篇文章描述了VIS相对CD-i和CDTV的好处：

说“标准计算机语言”吗？这是什么意思？ VIS的秘密如下：实际上，它是一台精简版Windows 3.1的低功耗PC（或者说286），仅与其硬件兼容（被称为“模块化Windows”，“模块化Windows”）。

在随后的几年中，Microsoft将发布许多不同的Windows专用版本（最初称为Windows CE，之后称为Windows Embedded Compact），用于诸如视频游戏机，智能手机，便携式音乐播放器甚至汽车等各种设备。模块化Windows是最早的此类版本之一。

有证据表明Microsoft计划在不同的Tandy设备中使用Modular Windows的可能性。1992年，InfoWorld上的一篇文章谈到了专门针对模块化Windows的软件开发套件的存在。微软不会为一台设备创造它，无论如何它的销量都不是很好。此外，在1992年《卫报》的简要说明中，据报道“模块化Windows的未来版本将适用于手持计算机和电子记事本或PDA。”

模块化Windows意味着很多。除其他外，它已成为Microsoft与Tandy之间不和的严重原因。2003 Usenet邮政Tandy工程师Frank Girda IV已故（不是很经常可以找到Usenet-post链接），他说在创建VIS的工作结束时，Microsoft试图将整个产品转移到Modular Windows品牌。到了关键点，尽管VIS ROM文件中没有空格，但Microsoft秘密地将徽标隐藏在代码中：

还记得1998年左右，每个人都以为准将已经死了吗？事实证明，一切都有些复杂。

今年，荷兰公司Tulip获得了该公司商标的权利，其子公司Web Computers International开始销售一种名为Web.it Internet Computer的设备，这可能是计算机硬件和软件中最奇怪的组合之一。

根据de Volkskrant[荷兰语]，Web International吞并了前Commodore，包括这家传奇计算机公司的管理层。但是在这里，来自多家参与公司的系统的脆弱性介入了-显然，在从解散后的公司获得Commodore名称后，Tulips本身就破产了，未能适当地许可Web International的权利。最有可能的是，所有这些都使产品的营销变得复杂。


但是，尽管前代将军的员工可能产生了影响，但该设备本身仍被程式化为那个时代的Apple PowerBook G3的底部，并配有黑色装饰和触控板。但是，实际上，它是一台台式机，没有带内置键盘的显示器，一次与Commodore差不多。她看起来很吸引人-当时的黑色台式机还不是很常见。

Web Computers International与IBM和AMD的合作伙伴合作，创建了一种机器，该机器通过仿真可以与旧的Commodore机器向后兼容。但是，该机器真正真正令人惊讶的要素是操作系统-Windows 3.1的独特专业版本，称为embOS。

为什么选择Windows 3.1？根据1998年对Web Computers International的Gerard Lindhout的采访，该公司的目标是创建一个用于Web冲浪的嵌入式系统，但是大多数选项不是很高的质量，当时的Windows CE不能满足公司的需求。引用Net4TV的采访：

然后，我们开始研究为Internet控制台设计的许多其他操作系统，例如QNX，但是它们存在很多问题。我们希望消费者能够从网上下载各种各样的软件。因此，我们选择Windows，并停止在3.1，因为它具有Windows 95所不存在的ROM版本。

操作系统不是以前使用Windows 3.1的传统程序管理器附带的，而是文件管理器（特殊的前端外壳）附带的。它还安装了一组与Microsoft产品无关的办公程序，特别是Netscape Navigator 3.0和Lotus Office套件。

也许这是世界上最奇怪的台式计算机-该计算机由一家不拥有它的公司以Commodore的名称出售，在Windows 95发布后使用Windows 3.1年，尽管它是一台性能较弱的PC台式机，但它看起来像当时强大的Mac笔记本电脑。此外，它还被开发为WebTV的竞争对手。这里有很多奇怪的地方，我什至不知道从哪里开始。

最后，我想谈谈一个叫彼得·塔特姆的人。澳大利亚软件开发商Tattem创建了Trumpet Winsock。该程序熟练地利用了Microsoft的监督。

尽管Windows 3.11版本被称为Windows for Workgroups，但Windows创建者在他们自己的方法指导下并着眼于Internet的潜力，并未开发允许带有拨号调制解调器的用户直接连接到Internet的软件，该版本是专门为网络上的工作而设计的。

小号Winsock在这里开始了业务。实际上，该程序提供了自己的Microsoft Windows Sockets API或Winsock版本，该版本允许Internet网络协议直接与Windows图形界面进行通信。这为使用Internet开辟了许多可能性，尤其是当第一个Web浏览器（如NCSA Mosaic和Netscape）出现时。

根据共享软件模型出售的程序获得了巨大成功，并由世界各地的Internet提供商分发-这是可以理解的，因为在Windows 3.1中必须使用图形Internet。再过几年，直到微软意识到错误并开发了自己的方法来轻松建立拨号Internet连接，小号Winsock才是唯一的解决方案。

问题在于，所有这些提供商都免费分发了Trumpet Winsock，并且通常甚至没有支付完整版的费用，也就是说，Tattem的工作没有获得适当的财务成功。

2011年，这个故事浮出水面，并取得了长足发展：另一个澳大利亚计算机用户，雅克·切斯特（Jacques Chester）听说了Tattem的情况，并创建了网站“ Thanks for Trumpet Winsock”，在那里他为Tattem进行了一次募捐活动，以支付他的报酬。比过去大多数人免费免费使用的人要多。

该活动引起了很多关注，并成为一个重要的提醒，即大多数人在了解了自己的旧错误之后，便开始寻求改进。 （他们不时地再次想起她。也许这篇文章会成为另一个原因。）

塔特姆本人对这样的消息有何反应？就他而言，他也很慷慨。Tattem网站说：

“为表示诚意，小号Winsock权利的唯一所有者彼得·塔特姆（Peter Tattem）宣布永久赦免所有小号Winsock用户（无论是自然人还是法律人）侵犯版权的行为，” 。 “但是，他保留了所有其他权利，这些权利受小号温索克版权保护。”

小号温莎的故事结局出乎意料的好。我们没有Windows 3.1时代的许多类似故事。
