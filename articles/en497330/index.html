Features


We create a separate Windows account on the PC (you can use the account as an administrator on a PC, not a user account). Install Google Chrome, if not, do authorization immediately in the browser and go to the Chrome Remote Desktop site . Download the application, set the PC name and password.

Why all of the above: so that the user does not use a Google account (let's call it a remote access account), it will be hidden from him. Previously, there was a separate application and this authorization to the Google account was checked, if you exit it, then it will not be possible to connect to the PC before re-authorization.

The application picks up the connection to the PC from the moment the login page of the Windows account appears. The feature gives freedom to switch between users (very convenient).



You can create several accounts for different PC lists, switching between them is nice and clear: they called your Google account list1, list2 (as you prefer).

It is possible to connect to as many PCs as you like with 1 Google entry. But, you can’t connect to 1 pc together, i.e. parallel connections are not prohibited. Please, use.
