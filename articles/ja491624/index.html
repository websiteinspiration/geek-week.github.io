Unity Universal Render Pipelineでアウトライン効果を作成する

ユニバーサルレンダリングパイプラインでは、独自のRendererFeatureを作成することで、レンダリング機能を簡単に拡張できます。レンダリングパイプラインに新しいパスを追加すると、さまざまな効果を作成できます。この記事では、ScriptableRendererFeatureとScriptableRenderPassを使用して、オブジェクトのアウトライン効果を作成し、その実装のいくつかの機能を検討します。



パイプラインのレンダリングについての紹介またはいくつかの単語


スクリプト可能なレンダリングパイプラインを使用すると、C＃のスクリプトを使用してグラフィックのレンダリングを制御し、オブジェクト、ライト、シャドウなどの処理順序を制御できます。Universal Render Pipelineは、Unityによって開発され、古い組み込みRPを置き換えるために設計された、既製のスクリプト可能なレンダリングパイプラインです。

独自の描画パス（ScriptableRendererFeatureおよびScriptableRenderPass）を作成および追加することにより、ユニバーサルRPの機能を拡張できます。これが現在の記事になります。これは、ユニバーサルRPに切り替える予定で、おそらくユニバーサルRPの既存のScriptableRenderPass'ovの動作をよりよく理解するのに役立つでしょう。

この記事はUnity 2019.3およびUniversal RP 7.1.8で書かれています。

不透明なオブジェクトのストローク効果を作成する例でScriptableRendererFeatureとScriptableRenderPassがどのように機能するかを理解します。

  • 描かれたオブジェクトをぼかす
  • 以前のパスで取得した画像からオブジェクトの輪郭を取得する

OutlineFeatureを作成


ScriptableRendererFeatureは、カスタムレンダリングパッセージ（ScriptableRenderPass）をユニバーサルRPに追加するために使用されます。ScriptableRenderFeatureから継承するOutlineFeatureクラスを作成し、そのメソッドを実装します。

using UnityEngine;
using UnityEngine.Rendering.Universal;

public class OutlineFeature : ScriptableRendererFeature
{
    public override void Create()
    { }

    public override void AddRenderPasses(ScriptableRenderer renderer, ref RenderingData renderingData)
    { }
}

Create（）メソッドは、パスを作成および構成するために使用されます。そして、作成されたパスをレンダリングキューに挿入するためのAddRenderPasses（）メソッド。

引数AddRenderPasses（）

次に、レンダリングパスの作成を開始します。それぞれを実装した後、現在のクラスに戻ります。

レンダリングオブジェクトパス


このパッセージのタスクは、特定のレイヤーからオブジェクトを描画し、シェーダーのグローバルテクスチャプロパティのマテリアルを置き換えることです。これは、Universal RPで使用可能なRenderObjectsPassパッセージの簡略化されたバージョンであり、レンダリングが実行されるターゲット（RenderTarget）のみが異なります。

ScriptableRenderPassから継承したクラスMyRenderObjectsPassを作成します。パッセージのすべてのロジックを含むExecute（）メソッドを実装し、Configure（）メソッドを再定義します。

using UnityEngine.Rendering;
using UnityEngine.Rendering.Universal;

public class MyRenderObjectsPass : ScriptableRenderPass
{
    {
        public override void Configure(CommandBuffer cmd, RenderTextureDescriptor cameraTextureDescriptor)
        { }
        public override void Execute(ScriptableRenderContext context, ref RenderingData renderingData)
        { }
    }
}

Configure（）メソッドは、一時的なテクスチャをレンダリングおよび作成する目的を指定するために使用されます。デフォルトでは、ターゲットは現在のカメラのターゲットであり、パスの完了後、デフォルトで再び表示されます。このメソッドは、基本ロジックが実行される前に呼び出されます。

レンダリングターゲットの置き換え


新しいレンダリングターゲットのRenderTargetHandleを宣言します。それを使用して、一時的なテクスチャを作成し、それをターゲットとして示します。RenderTargetHandleには、使用される一時的なRenderTextureの識別子が含まれます。また、RenderTargetIdentifierを取得することもできます。RenderTargetIdentifierは、たとえば、RenderTexture、Textureオブジェクト、一時的なRenderTexture、または組み込み（フレームのレンダリング時にカメラが使用）として設定できるレンダリングターゲットを識別するために使用されます。

RenderTargetHandleオブジェクトは、OutlineFeatureに作成され、作成時にパスに渡されます。

private RenderTargetHandle _destination;

public MyRenderObjectsPass(RenderTargetHandle destination)
{
    _destination = destination;
}

public override void Configure(CommandBuffer cmd, RenderTextureDescriptor cameraTextureDescriptor)
{  
    cmd.GetTemporaryRT(_destination.id, cameraTextureDescriptor);
}

GetTemporaryRT（）メソッドは、指定されたパラメーターで一時的なRenderTextureを作成し、指定された名前のグローバルシェーダープロパティとして設定します（名前は機能で設定されます）。

レンダリング


詳細なレンダリングは考慮しません。これにより、メイントピックから遠く離れたところにいる可能性があります。レンダリングには、ScriptableRenderContext.DrawRenderers（）メソッドを使用します。指定したレイヤーからのみ不透明なオブジェクトのみをレンダリングするための設定を作成します。レイヤーマスクはコンストラクターに渡されます。

...
private List<ShaderTagId> _shaderTagIdList = new List<ShaderTagId>() { new ShaderTagId("UniversalForward") };
private FilteringSettings _filteringSettings;
private RenderStateBlock _renderStateBlock;
...
public MyRenderObjectsPass(RenderTargetHandle destination, int layerMask)
{
    _destination = destination;

    _filteringSettings = new FilteringSettings(RenderQueueRange.opaque, layerMask);
    _renderStateBlock = new RenderStateBlock(RenderStateMask.Nothing);
}

public override void Execute(ScriptableRenderContext context, ref RenderingData renderingData)
{
    SortingCriteria sortingCriteria = renderingData.cameraData.defaultOpaqueSortFlags;
    DrawingSettings drawingSettings = CreateDrawingSettings(_shaderTagIdList, ref renderingData, sortingCriteria);

    context.DrawRenderers(renderingData.cullResults, ref drawingSettings, ref _filteringSettings, ref _renderStateBlock);
}

ぼかしパス


このパッセージの目的は、前のステップで取得した画像をぼかし、グローバルシェーダープロパティに設定することです。

これを行うには、ブラーシェーダーを使用して、元のテクスチャを一時的なテクスチャに数回コピーします。この場合、元の画像のサイズを縮小（コピーを縮小して作成）できます。これにより、計算が高速化され、結果の品質に影響を与えません。

ScriptableRenderPassから継承したBlurPassクラスを作成しましょう。

using UnityEngine.Rendering;
using UnityEngine.Rendering.Universal;

public class BlurPass : ScriptableRenderPass
{
    public override void Execute(ScriptableRenderContext context, ref RenderingData renderingData)
    { }
}

ソース、ターゲット、および一時的なテクスチャの変数（およびそれらのID）を取得しましょう。

private int _tmpBlurRTId1 = Shader.PropertyToID("_TempBlurTexture1");
private int _tmpBlurRTId2 = Shader.PropertyToID("_TempBlurTexture2");

private RenderTargetIdentifier _tmpBlurRT1;
private RenderTargetIdentifier _tmpBlurRT2;

private RenderTargetIdentifier _source;
private RenderTargetHandle _destination;

RenderTextureのすべてのIDは、Shader.PropertyID（）を介して設定されます。これは、そのようなシェーダープロパティが必ず存在する必要があることを意味しません。

残りのパラメーターのフィールドを追加します。これらはコンストラクターですぐに初期化されます。

private int _passesCount;
private int _downSample;
private Material _blurMaterial;

public BlurPass(Material blurMaterial, int downSample, int passesCount)
{
    _blurMaterial = blurMaterial;
    _downSample = downSample;
    _passesCount = passesCount;
}

_blurMaterial-ぼかしシェーダー付きのマテリアル。
_downSample-テクスチャサイズを縮小するための係数_passesCount-

一時的なテクスチャを作成するには、サイズ、フォーマットなど、それに関するすべての必要な情報を含む記述子を作成します。高さとサイズは、カメラ記述子を基準にしてスケーリングされます。

public override void Configure(CommandBuffer cmd, RenderTextureDescriptor cameraTextureDescriptor)
{
    var width = Mathf.Max(1, cameraTextureDescriptor.width >> _downSample);
    var height = Mathf.Max(1, cameraTextureDescriptor.height >> _downSample);
    var blurTextureDesc = new RenderTextureDescriptor(width, height, RenderTextureFormat.ARGB32, 0, 0);
また、識別子と一時的なRenderTexture自体も作成します。
    _tmpBlurRT1 = new RenderTargetIdentifier(_tmpBlurRTId1);
    _tmpBlurRT2 = new RenderTargetIdentifier(_tmpBlurRTId2);

    cmd.GetTemporaryRT(_tmpBlurRTId1, blurTextureDesc, FilterMode.Bilinear);
    cmd.GetTemporaryRT(_tmpBlurRTId2, blurTextureDesc, FilterMode.Bilinear);

レンダリングターゲットを再度変更するため、別の一時的なテクスチャを作成し、それをターゲットとして指定します。

ぼかし


一部のレンダリングタスクは、コマンドを構成して追加する特別なScriptableRenderContextメソッドを使用して実行できます。他のコマンドを実行するには、プールから取得できるCommandBufferを使用する必要があります。
コマンドを追加してコンテキストに送信したら、バッファをプールに戻す必要があります。

public override void Execute(ScriptableRenderContext context, ref RenderingData renderingData)
{
    var cmd = CommandBufferPool.Get("BlurPass");
    ...
    context.ExecuteCommandBuffer(cmd);
    CommandBufferPool.Release(cmd);
}

Execute（）メソッドの最終的な実装は次のようになります。

public override void Execute(ScriptableRenderContext context, ref RenderingData renderingData)
{
    var cmd = CommandBufferPool.Get("BlurPass");

    if (_passesCount > 0)
    {
        cmd.Blit(_source, _tmpBlurRT1, _blurMaterial, 0);
        for (int i = 0; i < _passesCount - 1; i++)
        {
            cmd.Blit(_tmpBlurRT1, _tmpBlurRT2, _blurMaterial, 0);
            var t = _tmpBlurRT1;
            _tmpBlurRT1 = _tmpBlurRT2;
            _tmpBlurRT2 = t;
        }
        cmd.Blit(_tmpBlurRT1, _destination.Identifier());
    }
    else
        cmd.Blit(_source, _destination.Identifier());
    context.ExecuteCommandBuffer(cmd);
    CommandBufferPool.Release(cmd);
}

アウトラインパス


オブジェクトのストロークを含む最終的な画像は、シェーダーを使用して取得されます。そして、彼の作品の結果は、画面上の現在の画像の上に表示されます。

以下は、一度にパスコード全体を示したものです。すべてのロジックは2行です。

public class OutlinePass : ScriptableRenderPass
{
    private string _profilerTag = "Outline";
    private Material _material;

    public OutlinePass(Material material)
    {
        _material = material;
    }

    public override void Execute(ScriptableRenderContext context, ref RenderingData renderingData)
    {
        var cmd = CommandBufferPool.Get(_profilerTag);

        using (new ProfilingSample(cmd, _profilerTag))
        {
            var mesh = RenderingUtils.fullscreenMesh;
            cmd.DrawMesh(mesh, Matrix4x4.identity, _material, 0, 0);
        }

        context.ExecuteCommandBuffer(cmd);
        CommandBufferPool.Release(cmd);
    }
}

RenderingUtils.fullscreenMeshは1 x 1メッシュを返します。

シェーダー


シェーダーを作成してアウトラインを取得します。2つのグローバルテクスチャプロパティが含まれている必要があります。指定されたオブジェクトとそのぼやけたバージョンの画像の_OutlineRenderTextureおよび_OutlineBluredTexture。

シェーダーコード 
Shader "Custom/Outline"
{
    Properties
    {
        _Color ("Glow Color", Color ) = ( 1, 1, 1, 1)
        _Intensity ("Intensity", Float) = 2
        [Enum(UnityEngine.Rendering.BlendMode)] _SrcBlend ("Src Blend", Float) = 1
        [Enum(UnityEngine.Rendering.BlendMode)] _DstBlend ("Dst Blend", Float) = 0
    }
    
    SubShader
    {       
        HLSLINCLUDE
 
        #include "Packages/com.unity.render-pipelines.universal/ShaderLibrary/Core.hlsl"
        
        struct Attributes
        {
            float4 positionOS   : POSITION;
            float2 uv           : TEXCOORD0;
        };
        
        struct Varyings
        {
            half4 positionCS    : SV_POSITION;
            half2 uv            : TEXCOORD0;
        };
 
        TEXTURE2D_X(_OutlineRenderTexture);
        SAMPLER(sampler_OutlineRenderTexture);
 
        TEXTURE2D_X(_OutlineBluredTexture);
        SAMPLER(sampler_OutlineBluredTexture);
 
        half4 _Color;
        half _Intensity;
 
        Varyings Vertex(Attributes input)
        {
            Varyings output;
            output.positionCS = float4(input.positionOS.xy, 0.0, 1.0); 
            output.uv = input.uv;
            if (_ProjectionParams.x < 0.0) 
                output.uv.y = 1.0 - output.uv.y;    
            return output;      
        }
 
        half4 Fragment(Varyings input) : SV_Target
        {
            float2 uv = UnityStereoTransformScreenSpaceTex(input.uv);
            half4 prepassColor = SAMPLE_TEXTURE2D_X(_OutlineRenderTexture, sampler_OutlineRenderTexture, uv);
            half4 bluredColor = SAMPLE_TEXTURE2D_X(_OutlineBluredTexture, sampler_OutlineBluredTexture,uv);
            half4 difColor = max( 0, bluredColor - prepassColor);
            half4 color = difColor* _Color * _Intensity;
            color.a = 1;    
            return color;
        }
        
        ENDHLSL        
     
        Pass
        {
            Blend [_SrcBlend] [_DstBlend]
            ZTest Always    //  ,      
            ZWrite Off      //      
            Cull Off        //    
 
            HLSLPROGRAM
           
            #pragma vertex Vertex
            #pragma fragment Fragment        
 
            ENDHLSL         
        }
    }
}
