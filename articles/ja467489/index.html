PHPの静的変数は、staticキーワードを使用して宣言される特別な種類の変数です。


static $foo = 3;

これらは通常の変数とは異なります（記事の後半で、これらの点について詳しく説明します）。


  1. 定数と定数式のみを割り当てることができます
  2. 静的変数の存続期間は、それが宣言されたスコープの存続期間に限定されません。
  3. スクリプトで定義できるのは1回だけです
  4. スクリプトの終わりまで破棄されません

順番に。


1.定数と定数式のみを割り当てることができます


これは、関数またはメソッドの操作の結果を静的変数、または一般にコンパイル段階でまだ不明なものに割り当てることができないことを意味します。つまり、そのような広告は機能しません


static $var = foo();

しかし、それはかなり可能です


static $var = 'some str';
static $varInt = 3 + 5;

2.静的変数の有効期間は、それが宣言されているスコープの有効期間に限定されません。
