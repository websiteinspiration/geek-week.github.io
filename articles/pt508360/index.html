Python e cerveja - agite, mas não misture





Oi Habr! Decidi descrever um pequeno sistema de automação para criar uma bebida maravilhosa. Para que uma cerveja seja saborosa, tempo e temperatura, temperatura e tempo são importantes. Além disso, são tão necessários quanto os ingredientes que compõem o mosto. Para chegar ao produto final, são necessárias 14 a 20 etapas com diferentes temperaturas e tempos de fermentação. Para qualquer pessoa interessada nesse processo misterioso, pergunto sob cat.



Vamos precisar de duas cubas de volumes diferentes, para três tipos:



  • cerveja de erva
  • Mosto de Trigo
  • Pilsner


Cada tipo passa por etapas correspondentemente diferentes no tempo e na temperatura. Obviamente, pode-se usar a instrução e o cronômetro, mas é terrivelmente inconveniente. Portanto, compilaremos uma tabela csv na qual armazenaremos: número da etapa, volume, temperatura alvo, tempo.



Em seguida, pegue o rasbpberry pi zero e conecte dois sensores de temperatura DS18B20 a ele .

Cada um dos sensores permitirá que você acompanhe a temperatura no tanque correspondente. Para automatizar o processo de contagem do tempo, instale o adafruit 128x64 oled na framboesa , na tela onde todas as informações necessárias serão exibidas.

Botões, podemos controlar a passagem pelas etapas da tabela e iniciar o cronômetro.



Código do temporizador 
def timer_code(seconds, t_counter = 2, restart = False):
    time.sleep(0.023)
    seconds -= time.time() - t_counter
    t_counter = time.time()
    minutes = trunc(seconds / 60.0)
