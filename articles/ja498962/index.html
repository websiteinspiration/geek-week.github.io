マルチタスクについて：制御下のウィンドウ

ロシアのMicrosoftエキスパートであるMarat Khairulinは、いくつかのタスクを操作するニュアンスを探求し続け、ウィンドウの結合と画面の分割、サイトとドキュメント用の個人用タイムマシン、および仮想テーブルの実際の利点について説明します。

で  前のポスト、  我々はマルチタスクにどのように関連するか、現代の科学について書いたとつのタスクにフォーカスを助ける小さなトリックを共有しました。そして、問題を解決するために一度に複数のアプリケーションで作業する必要がある場合に役立つ便利な機能を思い出させたいと思います（そして誰かが紹介されるかもしれません）。



古い方法と新しい方法で切り替える


アプリケーション間の切り替えは、おそらく私たちのほとんどが「マシン上で」行うことであり、もちろん、Alt + Tabキーの組み合わせに誰も驚かないでしょう。ただし、Ctrl（Ctrl + Alt + Tab）も同時に押すと、この組み合わせにより、すべてのウィンドウが画面上に開いた状態でメニューが修正され、マウスを1回クリックするか指でタッチするだけで目的のアプリケーションを選択できます（キーボードの矢印を使用してウィンドウを選択することもできます。 Enterで開きます）。多くのウィンドウを開いている場合に便利です。

あまり知られていないが、クラシックなWindows + Tabキーボードショートカットも、一見すると思われるよりも多くの機能を提供します。

Windows 10の現在のバージョンでこれらのキーを押すと、「タスクビュー」セクションに移動します。ここでは、アプリケーションを切り替えるだけでなく、「タイムライン」と「仮想デスクトップ」も使用できます。ちなみに、ホットキーの代わりに、[タスクプレゼンテーション]ボタン（通常は[スタート]ボタンの横にあります）をクリックするか、指を左端からタッチスクリーンの中央にスワイプすることができます。ちなみに、最新のラップトップをお持ちの場合は、タッチパッドのジェスチャーを試してください。3本の指で上にスワイプしてください。


タスクプレゼンテーションモード

"タイムライン"


「タイムライン」は2年前にWindows 10に登場しました。これは、以前にコンピューターで作業していたタスクに戻るのに役立ちます。必要に応じて、アカウントを使用して他のデバイスと同期することもできます*。

私にとって「タイムライン」は一種のタイムマシンになりました。多くのプロジェクトでの作業は数日続きます。そして、たとえば、先週の金曜日に特定のサイトやドキュメントで作業し、水曜日にこのプロジェクトに戻った場合、簡単に画像を復元できます。スケールを希望の日付に巻き戻すだけです。その同じ金曜日に、私が没頭していたサイトとドキュメントを表示して開くことができます。


タイムライン検索

「タイムライン」での検索も、何度も役に立ちました。通常のファイル検索とは異なり、デバイス上のすべてのドキュメントの中（つまり、最近使用したドキュメントの中から）を検索することはできません。Explorerや多くのアプリケーションで検索を起動するCtrl + Fの組み合わせに慣れているかもしれません。この組み合わせは「タスクプレゼンテーション」画面で機能します。つまり、最初にWindows + Tabを押し、次にCtrl + Fを押して検索ワードを入力して、「タイムライン」で検索します。

* 「タイムライン」の設定に関する詳細なヘルプ  。

Windows 10仮想デスクトップ


仮想デスクトップの概念は、決して新しいものではありません。私たちがWindowsについて語る場合、その使用のオプションの1つは、かつてMark Russinovichによって開発されたデスクトップユーティリティ  （2012年にリリースされた最新バージョン）でした。 Windows 10では、  仮想デスクトップ  がシステムに組み込まれており、タスクフローを共有してそれらを切り替えることができます。

以前に仮想テーブルを使用したことがない場合は、次の類推を想像して、それらのロジックを理解してください。複数のモニターを使用して、それぞれをワークフローで分割することにより、必要なプログラムを開くことができます。 -複数のWord文書を操作し、3番目に-ブラウザーとOneNoteを操作します。毎回、独自の一連のアプリケーションを備えた1つのモニター（仮想デスクトップ）のみを確認します。また、仮想テーブルを切り替えると、あるモニターから別のモニターに移動しているように見えます。


ウィンドウをドラッグして新しい仮想デスクトップに転送する

「タスクビュー」画面で新しい仮想デスクトップを作成できます。Windows+ Tabキーを押し、開いているアプリケーションの目的のウィンドウを[+デスクトップの作成]というラベルの付いたフィールドにドラッグすると、別の仮想デスクトップに移動します。新しい空の仮想テーブル（Windows + Ctrl + D）を作成して、その上で必要なプログラムを開くこともできます。

「タスクプレゼンテーション」画面で目的のテーブルを選択することで、「目を翻訳する」（つまり、カスタマイズされたデスクトップを切り替える）ことができますが、ホットキーを使用して切り替える方がはるかに便利です。Windows+ Ctrl +右/左矢印、 最新のタッチパッドの  4本の指左か右。

便利なマルチアプリケーションソリューション


ここで、もう1つの日常の必要性について、複数のアプリケーションを同時に操作します。

画面を分割


何年も前にお知らせしたい最初の機会で、元の形式（Aero Snapと呼ばれていました）でWindows 7に登場しました。Windows10では、その機能が拡張され、Snap Assistと呼ばれていました。画面を分割して、2つの（Windows 10では最大4つの）アプリケーションをピン留めすることです。


Snap Assistは、右側のドッキング用に2番目のウィンドウを選択することを提案します

これを行うには、アプリケーションを最上部のストリップまで移動し、画面に「影」が表示されるまで画面の右または左の境界に移動してリリースし（それによって最初のアプリケーションを修正）、次に隣に表示される他のアプリケーションのサムネイルで2番目のアプリケーションを選択する必要があります。近くの留め具。シナリオは単純で、マウスと指の両方で機能します。これは、Windowsキーボードショートカット+左/右矢印キーを使用することで、さらに簡単にすることができます。この組み合わせはすでに10年以上前のものですが、初めて使って今では「デジタルマジック」の感覚を味わう人もいます。

好奇心旺盛なユーザー向けに、Windows 10では、アプリケーションを画面の「クォーター」にコーナーに移動して（または追加のWindowsキー+上/下矢印を使用して）送信できることを思い出します。2つのアプリケーションを修正する場合、それらの間の境界線を移動して、一部のアプリケーションにより多くのスペースを与えることができます。画面で修正するアプリケーションを選択するには、[タスクプレゼンテーション]画面でサムネイルを右クリックします。

上のウィンドウ


あるアプリケーションのウィンドウを別のアプリケーションの上に固定する必要がある状況はかなりありました（テレビでは、これは「ピクチャーインピクチャー」モードと呼ばれていたようです）。

組み込みのビデオプレーヤーのミニモード（「Cinema and TV」アプリケーション。Windows10でデフォルトでビデオを再生します）。ビデオを開始し、右下隅の小さなボタン（ミニモードで再生）をクリックすると、ビデオのあるウィンドウがすべてのウィンドウの上に配置されます。


トップビデオのウィンドウ

同様の機会は、ブラウザーウィンドウをすべてのアプリケーションの上に固定することによってのみ、別のユーティリティを使用して取得できます。ドキュメントを作成する必要が生じた後、常に1つのオンラインサービスのWebサイトで同時に確認し、 Microsoft Storeで入手可能なAlways on Topアプリケーションが役立ちました。 Edgeの[共有]メニューに埋め込まれており、すべてのアプリケーションの上部にあるウィンドウに任意のサイトを送信できます。このオプションは、たとえばExcelのピボットテーブルで作業しているときに「片目」でYouTubeチャンネルを視聴するのに最適だと冗談を言うことができます。ただし、最初のノートで説明したように、このようなマルチタスクはブラウジングと作業の両方に害を与える可能性が高くなります。
