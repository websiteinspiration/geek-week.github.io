すべてのWeb開発者が必要とするブラウザ拡張

最新のブラウザーは、優れたWebブラウジングアプリだけではありません。

ブラウザは、優れたサイトを作成するのに役立つ便利なツールをWeb開発者に提供するプラットフォームでもあります。開発者が自由に使用できるように、組み込みツールに加えて、多数のブラウザ拡張機能があります。この記事の著者は、私たちが今日公開している翻訳で、プログラマーが直面するさまざまなタスクを解決するのに役立つように設計された最高の拡張機能を選択しました。これらは主にGoogle Chromeの拡張機能です。



React / ReduxおよびVue.js用の開発ツール


ReactとVue.jsは、よく知られているフロントエンド開発ツールです。それらをできるだけ効率的に使用するには、適切なブラウザ拡張機能をインストールする価値があります。React Developer Tools

拡張機能は、開発者ツールの機能を強化します。これを使用して、Reactコンポーネント、コンポーネントに渡されるプロパティ、およびコンポーネントの状態を調べることができます。 React開発者に真の利益をもたらすことができます。


React開発者ツール

アプリケーションの状態の操作中に問題が発生した場合は、 Redux DevTools拡張機能が役立ちます。これは、さまざまな方法で配置された状態を処理するために使用できます。特に、Reduxと一緒に使用するとうまく機能します。さて、ここにあなたのためのヒントがあります。この拡張機能をインストールして、airbnb.comなどのサイトにアクセスします。React / Reduxを使用しているため、Redux DevToolsの助けを借りて、このサイトのページの状態管理がどのように機能するかを知ることができます。私にとって-それはとても有益です。

Vue.js devtoolsはReact Developer Toolsと同じですが、 Vue.jsが異なります。この拡張機能は、Vueコンポーネントのデバッグに非常に便利であり、このフレームワークを使用するすべての人に役立ちます。


Vue.js開発ツール

Jsonview


他のプログラマーのブラウザーでは、JSONデータは明確で正確であるように見えても、あなたのブラウザーでは見えないと思ったことはありませんか？おそらく全体の要点は、他の人がJSONView拡張を使用しているということです。この拡張機能は、JSONデータを非常に迅速かつ正確にフォーマットします。JSONViewは間違いなく私のお気に入りのブラウザー拡張機能の1つです。


Jsonview

ライブラリスニファー


Library Snifferは、表示するページの作成に使用されたツール、ライブラリ、またはフレームワークをすばやく見つけたい人に最適なツールです。たとえば、Library Snifferを使用すると、WordPress、Drupal、React、Vue.js、AngularJSなどでサイトが作成されたものを見つけることができます。


ライブラリスニファー

CSSスキャン2.0


CSS Scan 2.0は、開発者がページ要素にカーソルを合わせるだけでCSSを探索およびコピーできる便利な拡張機能です。確かに、彼にはマイナスが1つあります。実際には、それを使用するにはライセンスを購入する必要があります。


CSSスキャン2.0

この拡張機能には、 CSS Peeperという無料の代替機能があります。これは主にデザイナー向けに設計されており、コードを書く人向けではありません。

ウェブ開発者


Web Developer 拡張機能を使用すると、さまざまなツールを含むパネルをブラウザーに装備できます。たとえば、彼らの助けを借りて、ページのJavaScriptをオフにしたり、ページの場所を制御したり、画像を操作したりできます。この拡張機能はChromeとFirefoxで使用できます。 


ウェブ開発者

Whatfont


WhatFontは、Webページで使用されているフォントを識別できるかなり小さな拡張機能です。テキストフラグメントがどのフォント用に設計されているかを確認するには、拡張機能を有効にして、マウスポインターをこのフラグメント上に移動するだけで十分です。ご覧のように、彼との作業は非常に簡単です。これは、前の拡張機能と同様に、Chromeで作業する人とFirefoxで作業する人が使用できます。


Whatfont

Colorzilla


ColorZilla拡張機能は ChromeとFirefoxの両方で使用することができます。これは、開発者にカラープルーフを取るためのツール、カラーを選択するためのツール、グラデーションジェネレーター、およびカラーを扱うための他の便利なツールを提供します。たとえば、これを使用して、Webページのピクセルの色を調べることができます。これには、色の操作履歴が保存され、パレットを表示するためのツールが含まれています。


Colorzilla

スペクトラム


コンテンツのアクセシビリティは、最新のWeb開発の重要な側面です。実際、多くの人々はウェブページと対話する能力が限られています。そのような制限の1つは、色覚異常（CVD）です。それは特定の色を区別する人の能力に影響を与えます。スペクトラム拡張機能を使用すると、さまざまなタイプのCVDの人に対するサイトの適合性を調査できます。


スペクトラム

サイトパレット


サイトパレット 拡張機能を使用すると、サイトで使用されている色をサイトから抽出し、カラーパレットとして配置できます。パレットを生成するいくつかの方法をサポートし、特別なリンクを発行してパレットを発行できるようにし、パレットを印刷する機能をサポートし、パレットをPDFファイルに変換する方法を知っています。また、これはサイトパレットに限定されません。


サイトパレット

トビー


Toby 拡張機能は、作業中に膨大な数のタブを開いたままにする必要がないような方法でブラウザーのタブを整理するように設計されています。これは、マウスで簡単に使用できるタブコントロールパネルの一種です。この拡張機能は、Web開発者の生産性にプラスの影響を与える可能性があります。


トビー

Talend APIテスター


Talend API Testerは、さまざまなAPIを使用してブラウザーから作業できる優れた拡張機能です。これらは、REST、SOAP、およびHTTPテクノロジーに基づくAPIにすることができます。これは、たとえばPostmanに似ています。拡張機能の無料バージョンは、すべてのタイプのHTTPリクエストの実行、安全なデータ交換、認証をサポートしています。


Talend APIテスター
