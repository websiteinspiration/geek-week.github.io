Ludum Dare: daftar periksa seminggu sebelum memulai

Ludum Dare adalah kompetisi akselerasi game komputer internasional di mana Anda membuat game dari awal di akhir pekan berdasarkan sebuah tema. Topiknya terungkap di awal acara.

Kompetisi berlangsung beberapa kali setahun, dan Ludum Dare 46 berikutnya dimulai dalam waktu kurang dari seminggu, atau lebih tepatnya, pada hari Sabtu, 18 April pukul 4:00 pagi waktu Moskow.

Di sini saya ingin berbagi tips yang akan membantu mereka yang mempertimbangkan untuk berpartisipasi dalam acara ini untuk pertama kalinya. Ini adalah daftar periksa untuk pemula.

Situs web resmi berisi aturan untuk partisipasi, serta panduan untuk menyelenggarakan permainan dan mengirimkannya ke anggota komunitas lain untuk dievaluasi. Saya tidak akan memikirkan aturan . Tetapi tentang hosting dan pengiriman ke situs ditulis di bawah dalam artikel ini.

Sebelum Anda mulai, beberapa kata tentang Jam dan Compo.

Jam dan Kompo


Ada dua kategori partisipasi dalam Ludum Dare: Selai - opsi mudah dan Compo - "mode kesulitan tinggi".

Jam (72 jam untuk membuat game) adalah kategori Ludum Dare untuk tim dan lajang. Anda dapat menggunakan alat dan perpustakaan apa saja untuk membuat game. Ini diperbolehkan untuk memulai dengan kode dasar yang tersedia.

Compo (48 jam untuk membuat game) adalah Ludum Dare klasik. Game untuk Compo dibuat sepenuhnya dari nol oleh satu orang. Gim dan semua kontennya harus dibuat dalam waktu 48 jam, dan pada akhirnya Anda harus memberikan semua kode sumber. 

Untuk lebih memahami detail dan batasan mode ini, serta memutuskan disiplin mana yang akan berpartisipasi, lebih baik membaca panduan resmi .

Cara mempersiapkan


Membuat game adalah proses kreatif yang kompleks yang membutuhkan waktu untuk mengerjakan mekanisme permainan, bagian visual, dan suara.

Penting untuk diingat bahwa permainan Anda akan dinilai oleh peserta lain pada poin-poin berikut:

  1. Inovasi (kebaruan); 
  2. Menyenangkan (kesenangan);
  3. Tema
  4. Grafik;
  5. Audio (suara);
  6. Humor (humor);
  7. Suasana hati (mood);
  8. Keseluruhan (peringkat keseluruhan).

Menurut pendapat saya, Ludum Dare adalah acara di mana lebih baik menggunakan teknologi dan alat yang Anda tahu benar untuk mencurahkan lebih banyak waktu untuk mengembangkan mekanika game dan grafik.

Saat macet


Dari pengalaman, selama kemacetan sangat sulit untuk mengevaluasi waktu dengan benar. Sepertinya banyak waktu, tapi ini jebakan. Karena itu, saya menyarankan Anda untuk membangun solusi yang sudah terbukti dan menggabungkannya. Lebih baik membuat proyek jadi kecil dengan 1-2 fitur menarik. Kalau tidak, Anda bisa berkubang dalam detail dan tidak melakukan apa pun.

Sedangkan untuk topiknya, Anda tidak perlu mengikutinya secara harfiah. Lebih baik datang dengan interpretasi Anda sendiri. Deskripsi ide interpretasi Anda dapat ditambahkan setelah nama permainan di Kirim di situs.

Berikut ini adalah informasi dasar dari panduan hosting game yang diposting di situs web resmi.

Anda dapat mengirimkan game Anda kapan saja mulai dari peluncuran Jam / Compo hingga akhir jam aplikasi. Compo berakhir 48 jam setelah topik diumumkan, dan Jam berakhir 72 jam setelah topik diumumkan. Tonton jam hitung mundur di sidebar situs web Ludum Dare.

Baik Compo dan Jam memiliki jam tambahan untuk dikirim setelah selesai. Sangat disarankan agar Anda mengirim dokumen lebih awal, karena ada semua jenis masalah teknis yang mungkin Anda temui. 

Sebelum mengunggah ke situs Anda harus memiliki:

  1. Akun di situs web Ludum Dare;
  2. Beberapa screenshot dari game;
  3. Nama gim Anda;
  4. Tautan ke gim Anda, yang terletak di suatu tempat *;
  5. Tautan ke kode sumber Anda, yang terletak di suatu tempat (hanya untuk Compo) **.

* Jenis hosting yang Anda butuhkan tergantung pada apakah Anda membuat game jaringan atau desktop. 

Panduan dari situs resmi mencantumkan beberapa platform populer. Ini bukan daftar lengkap dan ada banyak cara lain untuk menjadi tuan rumah permainan Anda.

Situs game:

  • Itch.io
     Memungkinkan hosting gratis HTML5, Flash, Java dan Unity, serta game desktop.
  • GameJolt
    Memungkinkan Anda untuk meng-host game web dan desktop secara gratis.
  • Newgrounds
     Memungkinkan hosting gratis game di Flash (swf) dan HTML5 (zip).
  • Kongregate
     Memungkinkan Anda meng-host game gratis di Unity, Flash, HTML5, dan WebGL.

Sistem hosting proyek:

  • Github
  • Gitlab
  • BitBucket

Penyimpanan cloud (lebih cocok untuk hosting game desktop):

  • Dropbox
  • google Drive

Ada banyak situs yang menawarkan hosting web gratis, serta banyak situs berbayar yang menawarkan versi percobaan gratis. Jika Anda memutuskan untuk menggunakan hosting web gratis, Anda disarankan mengunduh game uji sebelum meluncurkan Ludum Dare.

Jika Anda memiliki situs web atau server sendiri, Anda dapat meng-host game di sana.

** Kode sumber permainan Anda (dengan partisipasi dalam kategori Compo) dapat ditempatkan di sistem hosting proyek atau di penyimpanan cloud, yang tercantum di atas. Saat menggunakan layanan cloud, letakkan file .zip dengan semua kode sumber.

Berpartisipasi dalam Ludum Dare adalah cara yang bagus untuk keluar dari zona nyaman Anda, menguji diri sendiri, dan mendapatkan pengalaman baru. 

Menurut pendapat saya, masalah prioritas dan waktu masih menjadi batu sandungan dan alasan kegagalan banyak proyek, dalam kompetisi Anda dapat melalui sebagian besar tahap pengembangan dalam 2-3 hari dan belajar banyak hal, yang pada proyek nyata dapat memakan waktu beberapa bulan atau tahun. . 

Selain itu, esensi dari Ludum Dare tidak hanya minat olahraga dan keinginan untuk membuat proyek game, yang sering kali merupakan prototipe dari sesuatu yang lebih di masa depan, tetapi juga kesempatan untuk bekerja dengan orang-orang baru. Tidak perlu memiliki tim yang terkoordinasi dengan baik untuk berpartisipasi - seringkali orang diungkapkan dalam proses kerja.  

Jika Anda masih memiliki keraguan tentang partisipasi, maka saya dapat mengatakan bahwa Ludum Dare adalah percobaan yang melampaui ruang lingkup yang mungkin. 

Jangan terlalu keras pada diri sendiri. Selamat bersenang-senang!
