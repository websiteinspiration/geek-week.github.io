視野角180°のJabra Panacastパノラマカメラをテストする（ビデオ）

素材は「Video + Conference」サイトの編集者が作成しました。



有名な180度Jabra Panacastカメラをテストしましたが、結果は短いビデオであることがわかりました。過去の人生では、それはAltia Systemsによって開発されました。テクノロジーは、オフィスとコールセンターのGN Audioのオーディオソリューションを提供するデンマークのメーカーであり、同時にJabraブランドの所有者でもありました。2019年に、彼らはハドルルーム-小さな会議室のホットマーケットに参入するプロジェクトを購入しました。現在、カメラはロシアで利用可能です。

そのような小さな会議室で、パートナーを訪ねると、すべてが起こりました。7分間のビデオですが、ビデオがまったく気に入らない場合は、基本的な技術情報と印象を以下に示します。

カメラは本当に小さく、クレジットカードを少し超えています。ゴールデンスニッチを捕まえたハリーポッターのように感じます。ケースは耐久性があります。部屋のどこでも三脚に取り付けたり、壁に取り付けたり、モニターやラップトップに取り付けたりできます。しかし正式には、会議室の固定カメラとして正確に配置されています。シックすぎるノートパソコンに。



上の写真はUnitsolutionsで、下は

ボックスからのJabra USBケーブルで、長さはわずか1メートルで、すべてがルールに従って行われた場合、これは非常に小さいです。また、ルールでは、カメラから0.5〜3.5 mの範囲で完全な画像伝送が実現されることを示しています。この場合、テーブルの周りに座っている人の目の高さにカメラを置くことをお勧めします。したがって、さまざまなケースで、Jabraハブが必要になるか、USB-Cへの追加のUSB-Aケーブルがより信頼できるものになります。

デフォルトの表示角度は180°です。Jabra Directアプリケーションを使用して、90→120→140°に変更し、180°に戻すことができます。部屋には死角がないようです。画面を見て、状況を完全に制御していることを理解します。テーブルは透明を購入します。もちろん、それはズームとフレーミングに依存しますが、それでも非常に楽しい感じです。



解決：

  • パノラマ4K（3840 x 1080 @ 30フレーム/秒）
  • 1080フルHD（1920 x 1080 @ 30フレーム/秒）
  • 720p HD（1280 x 720 @ 30フレーム/秒）

実際、これらは13メガピクセルの3つのカメラで、半円に配置されています。それらの画像は、組み込みのPanaCast Visionプロセッサを使用して、リアルタイムでソフトウェアで合成されます。メーカーは、接着はわずか5 msの遅延で発生すると述べています。肉眼ではそれは本当にごくわずかです。確かに、カメラはかなり熱くなりますが、これはパフォーマンスには影響しません。この点についても説明があります


。JabraのQ＆Aスクリーンショット

端が曲がっている鏡のある部屋のように、人々は自然に見えます。 Jabraは、「フラット」レンズを使用してこれを説明し、詳細を尋ねました。興味深いものがあれば、共有します。次のスクリーンショットでは、特別な処理は行われていません。通常の生息地に住んでいる人は、フレームの端に通常の比率でいます。



「インテリジェントズーム」について。カメラはフレームの変更に応答し、新しく到着した参加者を考慮してビューを最適化します。そして、彼は自分でそれを継続的に行います（ビデオはダイナミクスにあります）。彼女はまた、参加者の数を数え、後で分析のためにAPIを介して会議の出席データを配布する方法も知っています。つまり、そのような会議室が多数ある場合、アナリストは施設のワークロードを評価および規制できます。

さらに、従来の電子パン、チルト、ズーム（ePTZ）がサポートされており、ビデオ通信アプリケーションインターフェイスを介して手動で制御できます。

また、カメラはさまざまな照明条件下でビデオ品質を自動的に最適化し、明るさ、コントラスト、彩度、シャープネス、ホワイトバランスを調整します。

2つの内蔵多指向性マイク。外部マイクを接続しなかったので、音質は標準に達しています。製造元は、スピーカーフォンの使用を推奨しています。

一般的な印象...

Jabra Panacastは、飛躍的にではなくスマートになっている最新の通信ソフトウェアとうまく調和しています。そのクラスとコストのデバイス-約$ 1300-は、高品質の自然な画像を提供すると同時に、独自の視野角を提供します。隠れる場所がないので、参加者の関与は最大に近いです（私たちは、深刻な無関心の個々のケースを考慮していません、そして、個人的な存在はそこで助けにはなりません）。

実際、カメラ自体が周りを見回し、人を見つけ、それらに焦点を合わせ、出席を追跡します。さらに、ミニチュア電話のように非常にコンパクトで、どこにでも簡単に設置できます。会議室を組み立てホールから再構築したり、カメラの真正面に取り付けたりする必要はありません。180度のビューで、テーブルの周りに座っている全員が見えるようになります。したがって、比較的狭い角度であればビデオコミュニケーションの作業に適しています。これは、家賃を節約したり、1つの会議室から2つを作成したりするのに役立つオプションです。
