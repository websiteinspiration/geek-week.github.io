Tinjauan umum kecerdasan buatan (AI) dan pembelajaran mesin. Modul ini ditujukan untuk pengguna dengan sedikit pengetahuan di bidang ilmu komputer dan statistik. Anda akan belajar tentang konsep dasar AI dan mempelajari cara menggunakan solusi pembelajaran mesin kustom dengan alat gratis dan sederhana. Anda akan mempelajari cara-cara sederhana namun efektif di mana para ahli data AI membuat prediksi tentang objek, orang, dan masa depan. Setelah itu, topik menarik dan kompleks yang mungkin pernah Anda dengar, seperti jaringan saraf, visi komputer, pembelajaran mendalam, dan pembelajaran tidak terkontrol, akan diperiksa.

Anda dapat memulai pelatihan di sini

2. Membuat bot pintar di Azure Bot


Interaksi pengguna dengan aplikasi melalui obrolan menggunakan teks, gambar atau ucapan dapat diimplementasikan menggunakan bot. Ini bisa berupa dialog "jawaban pertanyaan" sederhana atau bot kompleks yang memungkinkan orang untuk berinteraksi dengan layanan dengan cara "pintar", menggunakan pencocokan pola, pelacakan negara dan metode kecerdasan buatan yang terintegrasi dengan baik dengan layanan yang ada. Pelajari cara membuat chatbot pintar dengan mengintegrasikan QnA Maker dan LUIS.

Kursus ini akan memakan waktu sekitar 3 jam.

Mulai pelatihan

3. Analisis teks dengan Layanan Bahasa Azure Kognitif


Pelajari cara menggunakan Layanan Bahasa Kognitif untuk menganalisis teks, menentukan niat, menemukan topik dewasa, dan memproses pertanyaan bahasa alami.

Modul Kursus:

  • Klasifikasi dan moderasi teks menggunakan Azure Content Moderator;
  • Menambahkan logika dialog ke aplikasi menggunakan Intelligent Speech Recognition Service (LUIS);
  • Melacak nada suara dialog dalam teks menggunakan API analisis teks.

Mulai pelatihan

4. Kursus insinyur perangkat perbatasan berbasis AI


Teknologi AI, teknologi cloud, dan perangkat perbatasan yang digunakan sebagai wadah dalam mode CICD memungkinkan Anda mengubah seluruh industri dengan menciptakan ekosistem dengan kemampuan belajar mandiri untuk industri tertentu yang mencakup seluruh rantai nilai. Tugas kita adalah membuat seperangkat template dan metodologi untuk menyebarkan perangkat AI di tepi dalam konteks cloud. Dalam skema pelatihan ini, Anda akan menerima informasi berikut:

  • Informasi tentang menciptakan solusi menggunakan Internet dari hal-hal dan cloud;
  • Ikhtisar penggunaan solusi IoT pada perangkat tepi
  • Informasi tentang implementasi model untuk perangkat perbatasan menggunakan wadah;
  • Pelajari cara menggunakan DevOps untuk perangkat tepi.

Kursus ini dikembangkan bersama dengan Universitas Oxford.

Mulai pelatihan

5. Pemrosesan dan Klasifikasi Gambar dengan Layanan Visi Azure Cognitive


Microsoft Cognitive Services menyediakan fitur bawaan untuk menggunakan visi komputer dalam aplikasi. Pelajari cara menggunakan Layanan Visi Kognitif untuk mendeteksi wajah, menandai dan mengklasifikasikan gambar, dan mengidentifikasi objek.

Modul Kursus:

  • Identifikasi wajah dan emosi menggunakan API visi komputer di Azure Cognitive Services;
  • Pemrosesan gambar menggunakan layanan visi komputer;
  • Klasifikasi gambar menggunakan layanan pengenalan visual pengguna;
  • Penilaian persyaratan untuk implementasi API Pengenalan Visual Pengguna.
