Kali Linux NetHunterは、攻撃的なセキュリティプロジェクトです。 Kali Linuxディストリビューションを開発するもの。これは、Androidで実行できるKaliのプログラムとユーティリティのセットです。これにより、「グリーンロボット」上のお気に入りのデバイスを別の視点で見ることができ、完全に武装していなくても、いつもおなじみの一連のマスターキーを使用できます。

NetHunterはデバイスの完全なファームウェアではなく、ソフトウェアアドオンです。それは多くのデバイスでサポートされています：Nexusラインのほとんどすべて、Sony、OnePlus、ZTE、Samsungの他のいくつかのデバイス。全リストはこちら。

サポートされているデバイスの標準ファームウェアと一部のカスタムファームウェア（CyanogenModおよびLineageOS）にインストールされます。この場合、コアは標準を使用することをお勧めします。

しかし、隣人のWi-Fiに溢れ始める前に、インストールについて詳しく説明しましょう。デバイスのフラッシュに関与したことがないユーザーは、問題を経験する可能性があります。

インストール準備フェーズ


NetHunterは、カスタムリカバリを通じて、ルート化されたデバイスにインストールされます。したがって、ルートを取得してリカバリをインストールする必要があります（たとえば、TWRPまたはCM）。ブートローダーのロックを解除するプロセスでは、すべての個人データが削除されるため、インストールを開始する前に、バックアップを作成する必要があります。

デバイスのroot権限を取得することは重要な作業です。プロセスはデバイスによって大きく異なります。ガジェットでこれがどのように行われるかは、テーマフォーラムw3bsit3-dns.comとxda-developers.comで読む価値があります。ブートローダーのロックを解除してルートを取得することが不可能または非常に困難な場合があります。したがって、デバイスを選択するときは、これらのアクションの可能性にすぐに注意してください。

Nexus 10でのインストールプロセスを検討してください。

高品質のケーブル、マザーボードから直接USBポート（デバイスから「ブリック」を取得する可能性を最小限に抑えるため）、開発者のメニューへのアクセス、およびコンピューターを使用したデバイス自体が必要です。通常、開発者のメニューが開き、デバイス情報メニューのビルド番号まで数本のテープが付いています。そこでUSBデバッグをオンにします。ポップアップウィンドウにデバッグの許可が表示されます-同意します。

Kali Linux NetHunterをインストールする


NetHunterは、カーネルコンポーネント（kernel-nethunter）とソフトウェアアドイン（nethunter-generic）自体で構成されています。リリースバージョンでは、すべてのコンポーネントが1つのzipアーカイブに収集されます。これらのコンポーネントが2つのアーカイブに分かれているアセンブリがあります：kernel-nethunter-manta-lollipop-*。Zipおよびnethunter-generic-arm64-kalifs-full-rolling-*。Zip。 Nethunter OSをインストールするには、まずリカバリを通じてカーネルをインストールし、次にソフトウェアアドオンをインストールするか、すべてのコンポーネントを含むアーカイブを使用して一度にすべてをインストールする必要があります。

運が良ければ、デバイスの完成したリリースアーカイブは既にコンパイルされており、Offensive Security Webサイトページにあります。。ただし、運が悪い場合は、自分でインストールアーカイブを収集する必要があります。このため、開発者は新しいコンポーネントからユーティリティを作成しました。「nethunter-installer」ディレクトリをGitLabリポジトリからダウンロードする必要があります。指示に従って、次のコマンドでデバイスストレージを初期化します。

./bootstrap.sh

その後、Pythonスクリプトを使用して、インストールアーカイブを収集します。

このスクリプトを使用するいくつかのビルドオプションを検討してください。

  • Android 5.1.1 Lollipop上のGoogle Nexus 10（manta）デバイスのカーネルとchrootを使用した完全なアセンブリ。 
    python build.py -d manta -l --rootfs full

結果のzipアーカイブ（私の場合、nethunter-manta-lollipop-kalifs-full * .zip）は、デバイスのルートに保存され、リカバリを通じてロードされます。デバイスごとに、リカバリダウンロードは異なります。

Nexus 10：「音量+」+「音量-」+ロックボタン。それらを数秒間クランプし、ブートローダーがロードされ、ボリュームキーを使用して「リカバリモード」を選択し、ロックボタンで確認します。

リカバリTWRPで、[インストール]を選択し、インストーラーのzipファイルを選択して、スライダーを右にドラッグします。インストールが完了したら、完了後にデバイスを再起動します。








TWRPによるインストールプロセス。

Androidが起動し、アプリケーションの最適化が始まります。その結果、壁紙が変更され、いくつかの新しいプログラムがメニューに表示されましたが、残念ながら正しくインストールされていませんでした。


誤ってインストールされたプログラム。

これは簡単に修正できます。zipアーカイブを解凍し、*解凍されたzipアーカイブ* / data / appにあるapkファイルを介して手動でアプリケーションをインストールします。この前に、不明なソースからアプリケーションをインストールする機能を有効にする必要があります。次に、最初にnethunter.apkをインストールしてから、他のすべてのアプリケーションをインストールします。デバイスを再起動すると、アイコンが正常に表示され始めたことがわかります。HackerKeyboardは、端末での作業により適しているため、さらに構成することをお勧めします。

ターミナルを起動し、カーリーターミナルを選択します。そしてアップデートをインストールします：

apt update
apt upgrade

Microsoftリポジトリの問題を修正します。

wget https://packages.microsoft.com/keys/microsoft.asc | sudo apt-key add -
