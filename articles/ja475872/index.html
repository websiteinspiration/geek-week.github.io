ロボットのヒント＃42：笑顔と人々があなたのために手を差し伸べる


私たちを取り巻く世界は、私たちが望むかどうかにかかわらず、絶え間なく継続的に変化しています。そして、テクノロジーはこれに大きな役割を果たしており、その開発と普及は毎年ますます活発になってきています。私たちは車を珍しいものとは考えていませんが、モバイル通信を魔法と見なしています。テクノロジーは私たちの日常生活、仕事、そして私たち自身の一部となっています。ターミネーターについての映画によると、未来派やSF作家の間で最も人気のあるトピックの1つは、常にロボットです。ロボット-人間の生活を楽にしたり、地球上のすべての生命の総虐殺を調整したりできる自律型機械です。いずれにせよ、非常に面白い質問がいくつか残っています。人はどのように機械を操作し、彼はそれを信頼し、彼らの協力はどれほど効果的ですか？米陸軍研究所の科学者たちは一連の実験を行うことに決めました、自律マシンの動作と「外観」のどの側面が人との協力を改善するかを確立する。彼らのレポートから学んだことを正確に見つけたもの。行く

研究の基礎


人と機械の関係をよりよく理解するには、まず人と人の関係を調べる必要があります。惑星の人口は多くのグループ（宗教、国籍、民族など）に分かれているので、人は常に人々を「私たち」と「彼ら」に分けます。研究者によると、この分離のプロセスは潜在意識レベルで進行し、協力のために同じグループに属する人を選択する可能性の増加につながります。科学者たちはまた、これらの好みは他の国籍の人々、宗教、および他のものに対する差別の徴候とは何の関係もないことに注意しています。

自律型機械の場合、人と機械の協調時にそのような社会的分類が行われるかどうかは不明のままです。もしそうなら、それはどのように明示され、どのように規制され、どのように変更できるか？

これをテストするために、科学者は一連の実験を行いました。参加者は、2つの遠い文化（米国と日本）の人々と、同じ文化に対応し、特定の感情を表現する仮想の顔を持つ車でした。

特徴は以下のとおりです。

  • 性別：63.2％-男性、36.8％-女性。
  • 年齢：0.9％-18-21; 58.5％-22-34; 21.6％-35-44; 10.7％-45〜54; 6.2％-55〜64; 2.1％-64歳以上;

  • 年齢：0.6％-18-21; 15.7％-22-34; 36.4％-35-44; 34.7％-45〜54; 10.4％-55-64; 2.1％-64歳以上;
  • 民族性：0.6％-東インド; 99.4％-アジア（南東部）。

個人の分類によって、人々の関係は異なる場合がありますが、自律型機械は人間ではないことを覚えておいてください。したがって、このデバイスは、デフォルトで、すでに別のソーシャルグループに属しています。もちろん、人は機械を社会的俳優（ある種の相互作用の参加者）として知覚できますが、肉や血から他の生きている人を知覚するものとは異なって知覚します。

これの顕著な例は、「じゃんけん」のゲーム中の人間の脳の初期の研究です。ある人が他の人と遊んだとき、内側前頭前野が活性化され、それが精神化*に重要な役割を果たします。
メンタライゼーション* -自分や他の人の精神状態を表す能力。
しかし、機械で遊んだとき、脳のこの部分は活性化されませんでした。

同様の結果は、最後通しのゲームを使った実験によっても示されました。
最終通告ゲーム※は、1番のプレイヤーが一定額のお金を受け取り、初めて目にする2番のプレイヤーと共有しなければならない経済的なゲームです。オファーはどれでもかまいませんが、2番のプレーヤーが拒否した場合、両方のプレーヤーはまったくお金を受け取りません。
プレーヤー＃2がプレーヤー＃1（人間）から不公平なオファーを受けたとき、彼は否定的な感情の形成に関与する脳の中葉のセクションをアクティブにしました。プレーヤー1が車の場合、そのような反応は観察されませんでした。

そのような実験は、人が他の人とは異なる方法で機械を知覚するという事実をもう一度確認します。一部の研究者は、本能の観点から見ると、マシンは私たちにとって競争を表していないと信じており、したがって、脳の特定の部分の同様の減少した（またはまったくない）活動があります。

すべては非常に論理的です。人間は人間同士の狼ですが、狼同士の間でも、ある程度の信頼と、特定の状況で協力したいという欲求があります。しかし、車はキツネのように振る舞い、狼は言葉からまったく協力したくない。しかし、そのような好みは、私たちの世界にますます多くのロボットが存在するという事実を打ち消すものではないため、私たちと彼らの間の関係を改善する方法を見つける必要があります。

なぜ研究者はこのタイプの社会的分類を文化的アイデンティティとして使用することにしたのですか？まず、以前に、人が特定のスピーチ特性（強調など）、社会的規範、人種を持つマシンに積極的に反応することがすでに確立されていました。

第二に、異なる社会集団の代表が実験に参加する場合、協力と競争の明確な織り交ぜが最初の段階で追跡されます。その後の段階では、参加者がお互いをよく知るようになると、経験のタスクに対応する明確な行動のラインが描かれます。簡単に言えば、経験の仕事が協力に関係していたとしても、異なる社会グループの参加者はそもそもお互いを信頼していませんでした。しかし、しばらくしてから、タスクを完了することへの信頼と欲求は、無意識の不信を上回っていました。

第3に、以前の実験では、社会的分類に基づく協力の当初の期待は、より状況的に重要な兆候（感情）に訴えると拒否される可能性があることを示しています。

つまり、研究者はマシンに実験参加者（人）が属していた社会的グループの視覚的特徴といくつかの感情表現を与え、次にさまざまな組み合わせをチェックして関係を特定しました。

これらすべてから、2つの主要な仮説が特定されました。 1つ目は、文化的アイデンティティの前向きな兆候を関連付けることで、人々が機械についてデフォルトで持っている不利な偏見を緩和できることです。 2つ目は、感情を表現すると、文化的アイデンティティに基づく協力/競争の期待を超える可能性があるということです。つまり、特定の文化グループの車の視覚的特徴を与えると、同じグループの人々は無意識のうちにそれをより信頼できるようになります。ただし、特定の感情が現れて、機械が協力に乗りたがらないことを示す場合、すべての文化的偏見はすぐに重要でなくなります。

奇妙なことに、実験は囚人のジレンマに基づいたものでした。協力は勝利の戦術ではありません。ゲームの理論に従って、囚人（プレイヤー）は常に他のプレイヤーの利益を考えずに自分の利益を最大化します。したがって、合理性はプレイヤーに裏切りと協力への意欲を押し下げるはずですが、実際にはそのような実験の結果は合理的とはほど遠いことが多く、この数学的なゲームの名前に「ジレンマ」という言葉が存在することを説明しています。ちょっとした落とし穴があります。一方のプレイヤーが2番目のプレイヤーが協力しないと信じている場合、彼も協力すべきではありません。

実験結果



画像No. 1

合計すると、実験の参加者はそれぞれ、マシンで20ラウンド遊んだ。上の画像はゲイン行列です。ゲーム中に獲得したポイントは、実際に賞金30ドルの宝くじに変換され、実際に金銭的な利益を得ました。

実験の主な目的は、参加者がどのように人や機械と協力/競争するか、文化的アイデンティティと感情がこれらのプロセスにどのように影響するかを識別することでした。


画像No. 2

上記は、異なる感情（中立、悲しみ、後悔、喜び）を表現するマシンの仮想顔の2つの文化的アイデンティティ（米国と日本）です。仮想顔が実験のすべての参加者によって使用されたことは注目に値します。人と車。

感情的なシリーズは偶然に選ばれませんでした。囚人のジレンマに関する以前の実験では、感情によっては協力を促すことができるものもあれば、表現される時間によっては競争できるものもあることが示されています。

このモデルでは、機械の感情とそれを引き起こすアクションが2つの主要なカテゴリに分類されました。

人を競争に追い込む：

  • 相互協力の後で後悔する（マシンはそれ自体の利益のために人間の参加者を操作する機会を逃した）。
  • 操作後の喜び（参加者は協力に行きますが、車からの見返りには何も受け取りません）。
  • 相互競争後の悲しみ/相互協力後の中立的な感情。

人々に協力を促す：

  • 相互協力の喜び
  • 搾取の後悔。
  • 相互競争後の悲しみ。

したがって、実験では同時にいくつかの変数がありました：文化的アイデンティティ（米国または日本）、宣言されたパートナーのタイプ（人または機械）、および表現された感情（競争または協調）。

実験の結果は、参加者のペアのタイプに応じて、1つの文化グループの参加者と異なる文化グループの参加者の2つのカテゴリに分類されました。


画像3

上記は、さまざまな文化グループがゲームに参加した実験の結果（分散分析、つまりANOVA）のグラフです。予想通り、人との協力（M = 41.15、SE = 2.39）は、機械との協力（M = 34.45、SE = 2.25）よりも多く表現されました。感情に関しては、すべてが予測可能でした-プレイヤーは、競争の感情（M = 29.34、SE = 2.70）よりも、協力の感情（M = 45.42、SE = 2.97）を表現した人と協力することが多かったです。中立的な感情（M = 38.63、SE = 2.85）も、競合する感情を上回りました。

人間のプレーヤーへの自信を刺激する感情が、彼らがマシンで遊んでいるという事実を完全に打ち消し、それによって不信の度合いを減らしたのは興味深いことです。


画像4

参加者が同じ文化グループのパートナーに出会ったとき、状況は似ていました。協力的（M = 45.55、SE = 2.61）感情は中立的（M = 37.42、SE = 2.81）および競争的（M = 28.99、SE = 2.78）を上回った。

結果の全体から、ある人が他の人と同じように機械との通信に信頼と不信の心理的パラメータを使用していることが明らかになります。自律型マシンは、最初はその人にとっては異質な社会集団のメンバーとして認識されます。ただし、人間のパートナーに適した特定の文化的アイデンティティの特徴を持つ視覚的な顔を使用すると、マシンへの不信を和らげることができます。

最も重要な観察は、文化的アイデンティティと機械に属することは、正しい、いわば感情によって完全に平準化されるということです。

エピローグ


この研究は非常にあいまいですが、人間の心理学の重要な特徴を示しています。私たちは無意識に私たちの周りの人々を特定の社会的グループに分類し、私たちのグループに属する人々への信頼を高めています。これが私たちの主な反応です。しかし、それだけではありません。人とコミュニケーションするとき、私たちは感情、話し方、声などによって表現される多くの信号（視覚と音）を受け取り始めるからです。これらの信号は、個人の代表の画像を補完するものであり、信頼の有無にかかわらず、意思決定の瞬間に反発されます。言い換えれば、私たちの頭の中で発生する社会的分類は、活発なコミュニケーションに関しては重要ではありません。これは古いことわざにあるとおりです：「彼らは服で迎えられますが、心で付き添われます」と「本をその表紙で判断しないでください」。

車に関連して、人は常に不安と不信に調整されましたが、それはしばしば重大な理由でサポートされませんでした。これは人間の性質です-最初は外国のものを拒否することです。しかし、知識が無知を征服するのと同じように、感情を表現できる自律型機械との実際の相互作用も人間の偏見を克服します。

これは、将来のすべてのロボットが感情を持つべきであることを意味しますか？はいといいえ。私たちが今日調査した研究者たちは、ロボットに対する最初の2、3の感情は、人々の社会による自律型機械の採用プロセスを加速できる優れた機能になると信じています。オオカミと一緒に暮らすために私に言えること-オオカミのような遠吠え。

さらに、私たちはすでにロボットに独自の特性を与えています。私たちはロボットに私たちのような目、私たちのような顔をさせ、声や感情を与えます。人は常に車を人間化しようとします、そしてこの研究が示すように、無駄ではありません。
