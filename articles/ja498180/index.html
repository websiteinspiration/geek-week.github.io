PVS-Studioスタティックアナライザーを使用したNewton Game Dynamicsの再確認


最近、インターネットで、物理エンジンNewton Game Dynamicsを発見しました。そのようなプロジェクトには通常大量の複雑なコードが含まれていることを知っているので、PVS-Studio静的アナライザーで確認すると面白いと思いました。私の熱意は、同僚のAndrei Karpovがすでに2014年にこのプロジェクトをテストしているという事実にさらに拍車をかけられました。つまり、過去6年間のアナライザーの開発を実証する良い機会でもあります。また、執筆時点では、Newton Game Dynamicsの最新リリースは2020年2月27日であり、このプロジェクトも過去6年間活発に開発されていることにも注意してください。したがって、私たちのチームに加えて、この記事が、いくつかのバグを取り除き、コードを修正できるエンジンの開発者にとって興味深いものになることを願っています。
