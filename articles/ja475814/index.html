Dockerの変革：MirantisでのDocker Enterpriseの販売と最新のパス

昨日、Docker Incという会社で一連の変革が行われ、同じ名前の人気のあるコンテナソリューションの背後にありました。彼らは一定の時間待っていたと言えます。結局のところ、Dockerの大規模な普及、コンテナ化のための他のテクノロジーの開発、およびKubernetesの人気の急速な成長により、Docker Incの製品とビジネスに対する質問がますます増えました。

MirantisがDocker Enterpriseビジネスを買収


昨夜の主なイベントは、同社がDocker Incから主要ビジネスであるDocker Enterprise Platformを購入することを Mirantisが発表したことでした。

「Docker Enterpriseは、パブリッククラウドからハイブリッドクラウドまで、開発者があらゆるアプリケーションをシームレスに構築し、共有し、安全に実行できる唯一のプラットフォームです。Fortune 100企業の3分の1は、イノベーションのプラットフォームとしてDocker Enterpriseを使用しています。」

同じプレスリリースによると、 Mirantis が買収したDocker Enterpriseチームは、プラットフォームの開発とサポートを継続し、エンタープライズクライアントが期待する新機能を実装する予定です。ちなみに後者には、Mirantisの保守不要のサービスとしてのアプローチ、Mirantis Kubernetesおよびその他のクラウドテクノロジーとの統合、さらに企業セクター向けの実証済みのビジネスモデルが含まれます。

Docker Inc自体は、 Docker Enterprise の販売を、開発者向けの企業の人生における新しい章と呼んでいました。

「Dockerの将来の焦点は、既存の基盤を使用して、最新のアプリケーションに集中する開発者のワークフローを改善することです。」

「基礎」とは、実際のDocker CLIユーティリティ、Dockerデスクトップ、Docker Hubなど、会社の存続期間中に作成されたソリューションを指します。簡単に言うと、Docker Incは、開発者を直接対象とする製品（DockerデスクトップおよびDocker Hub）の開発に焦点を当てます。ITのベテランであり、オープンソースのジャーナリストであるMatt Asay がこの発表に

Docker Incのアクションは、その管理者のコメントのおかげで、より理解しやすくなっています。そして、変化は彼に影響を与えました。
