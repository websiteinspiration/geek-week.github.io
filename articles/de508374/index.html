Machen Sie es selbst oder wie Sie Ihr Snom-Telefon anpassen. Teil 2 Symbole und Bilder

Wie wir im ersten Teil des Artikels versprochen haben , ist diese Fortsetzung dem unabhängigen Ändern der Symbole in Snom-Telefonen gewidmet.



Also fangen wir an. Schritt eins: Sie müssen die Firmware im tar.gz-Format herunterladen. Sie können von unserer Ressource herunterladen hier . Alle Snom-Symbole sind verfügbar und in jeder Firmware-Version enthalten.



Hinweis : Bitte beachten Sie, dass jede Firmware - Version Konfigurationsdateien speziell für diese enthält Version und Telefonmodell . Die Verwendung von Einstellungsdateien, die nicht mit der Firmware oder dem Telefon übereinstimmen, führt zu Problemen.


Nach dem Herunterladen und Öffnen der Datei customizing.tar.gz sollte sie folgendermaßen aussehen. Der tatsächliche Inhalt der Dateien hängt von der Version des Telefons und der Firmware ab:







Schritt zwei: Vorbereiten der Symbole für die Telefone. Wie Sie wissen, verfügen Snom-Telefone über einen Farb- bzw. einen Schwarzweißbildschirm, und die Symbole unterscheiden sich.



I. Ändern Sie die Symbole für Farbdisplay-Telefone



Symbole und Bilder auf Telefonen mit Farbdisplays werden im PNG-Format gespeichert. Dadurch können sie in fast allen modernen Bildbearbeitungsprogrammen problemlos bearbeitet werden. Nach der Bearbeitung wird jedoch empfohlen, PNG-Dateien mit Tools wie optipng, pngquant oder pngcrush zu optimieren, um redundante Informationen zu entfernen und die Dateigröße zu optimieren.



Symbol Bildgrößen:



  • Kontextsensitive Schlüsselsymbole 24x24px
  • SmartLabel 24x24px & 18x18px
  • Symbole in der Titelleiste (Titelleisten-Symbole) 18x18px
  • Menüsymbole 18x18px
  • Während eines Anrufs (Anrufbildschirmsymbole) 18x18px - 48x48px
  • Dateiformat: PNG


Laden Sie die gewünschten Symbole auf Ihr Telefon herunter, nachdem Sie sie erstellt haben. Es gibt zwei Möglichkeiten zum Herunterladen:



  1. Über das Webinterface im manuellen Modus
  2. Automatische Werbung verwenden


Beginnen wir mit der ersten Option - Download über die Weboberfläche. Rufen Sie zum Herunterladen die Weboberfläche des Telefons auf der Registerkarte " Einstellungen" / "Darstellung" auf und wählen Sie " Benutzerdefinierte Bilder" :







Als Nächstes finden Sie das Symbol, das Sie ändern und unsere eigene Version laden möchten:







Wenn Ihnen Ihre eigene Version oder die "Kurve" nicht gefällt, können Sie jederzeit einen Rollback durchführen, indem Sie auf die Schaltfläche "Zurücksetzen" klicken ""







Hinweis . "Software-Update" und "Werkseinstellungen wiederherstellen" löschen heruntergeladene Bilder nicht.


Wie Sie im manuellen Modus sehen können, ist alles recht einfach. Wenn Sie jedoch mehrere Telefone wechseln müssen, nimmt dieser Vorgang viel Zeit in Anspruch. Daher fahren wir mit der zweiten Option fort.



Option zwei - Download über automatische Bereitstellung.



Zunächst müssen Sie ein Archiv im Tar-Format aus dem zuvor heruntergeladenen Archiv customizing.tar.gz erstellen . Löschen Sie beim Erstellen eines Archivs alle Dateien und Verzeichnisse, die Sie nicht ändern müssen, aber stellen Sie sicher, dass Sie die Verzeichnisstruktur beibehalten .

Hinweis . Sie müssen nicht alle Dateien archivieren, die ursprünglich im Archiv waren. Es ist ausreichend und wird empfohlen, nur Dateien zu archivieren, die Sie geändert haben. Je mehr Dateien Sie in das Archiv einfügen, desto länger dauert es, bis das Telefon sie konfiguriert.


Als Nächstes führen wir mehrere Schritte aus:



1) Erstellen Sie eine XML-Datei, z. B. branding.xml, und kopieren Sie sie auf Ihren Webserver (HTTP), d. H. http: //yourwebserver/branding.xml: 



<?xml version="1.0" encoding="utf-8" ?>
<settings>
 <uploads>
  <file url="http://yourwebserver/branding/branding.tar" type="gui" />
</uploads>
</settings>


2) Wir gehen zur Weboberfläche des Telefons im Abschnitt Erweitert -> Aktualisieren -> URL einstellen und stellen einen Link zu unserer Datei yourwebserver / branding.xml bereit.















3) Wir starten das Telefon neu und genießen das Ergebnis.

II.



Symbole auf monochromen Geräten werden nicht in normalen Bilddateien wie .png oder .jpg gespeichert, sondern sind Bitmap-Schriftarten, die alle in der Benutzeroberfläche verwendeten Symbole enthalten. Im Bereich der privaten Nutzung der Unicode-Tabelle ab U + EB00 werden Snom-Symbole definiert und können mit Tools wie Font Forge direkt geändert werden .



Beim Öffnen einer Bitmap-Schriftartdatei mit Font Forge sollte eine Liste der verwendeten Symbole angezeigt werden. Der tatsächliche Inhalt der Dateien hängt von der Version des Telefons und der Firmware ab:







Symbolspezifikation für Telefone mit monochromem Display.



Für die Modelle D305, D315, D345, D385, D745, D785, D3, D7:



  • Kontextsensitive Schlüsselsymbole 17x17 - Grundlinie x → 0 / y → -2
  • Titelleisten-Symbole 17x17 - Grundlinie x → 0 / y → -2
  • Beschriftungsfeldsymbole 17x17 - Grundlinie x → 0 / y → -2
  • Maximale Größe der Symbole 32x32


Für die Modelle D120, D710, D712, D715, D725:



  • Kontextsensitive Schlüsselsymbole 7x7 - Grundlinie x → 0 / y → 0
  • Titelleisten-Symbole 7x7 - Grundlinie x → 0 / y → 0
  • SmartLabel-Symbole 7x7 - Basislinie x → 0 / y → 0
  • Maximale Größe der Symbole 32x32


Nachdem Sie das gewünschte „Image“ erstellt und anschließend aus Font Forge exportiert haben, müssen Sie die folgenden Einstellungen verwenden:







Erstellen Sie nach dem Export eine TAR-Datei, die die gerade erstellte Datei mit dem Namen der zu ersetzenden Datei enthält.



tar -cvf fonts.tar fontfile.bdf



Da wir tatsächlich nicht die Bilder, sondern die Schriftart ändern, laden Sie sie dann durch automatische Werbung als Schriftart, wobei Sie in den Einstellungen der XML-Datei Folgendes angeben: 



<?xml version="1.0" encoding="utf-8" ?>
<settings>
 <uploads>
  <file url="http://192.168.23.54:8080/fonts.tar" type="font" />
 </uploads>
</settings>
