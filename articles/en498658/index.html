OWASP Consortium Updated Web Security Testing Guide



The Web Security Testing Guide (WSTG) project is the primary security testing methodology for web application developers and information security professionals and is being developed by the OWASP international consortium.

The Open Web Application Security Project (OWASP) is an open source web application security project. The OWASP community includes corporations, educational organizations, and individuals from all over the world. The community is working on creating freely accessible articles, manuals, documentation, tools and technologies.

WSTG is a comprehensive guide to testing the security of web applications and web services. The WSTG, a collaborative effort by cybersecurity professionals and volunteers, provides a set of best practices used by pentesters and organizations around the world.

The current version - [Version 4.1] - 2020-04-2, the previous version - [Version 4.0] - 2014-09-17 - about 6 years have passed between updates, which should cover a large layer of modern technologies, however, fundamental changes will be expected only in 5 versions of WSTG.
