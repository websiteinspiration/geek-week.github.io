React RoboticsのDogBotは建設業界で革命を追求します

今日、メディアはしばしば四本足のロボットについて書いています。彼らが受け取った新機能と機能、およびプロトタイプにどれだけ近いかが報告されています。しかし、1つの質問は常に最後まで答えられませんでした。どうすれば彼らにインテリジェンスを与え、周囲の世界で独立してナビゲートするように教えるのですかさまざまな緊急事態や災害、および軍事利用の結果としての閉塞の分析にこれらのロボットを接続することに加えて、このような4本足のマシンをどのように、どこで使用できますか？同社ボストンダイナミクスは、彼女のロボットの一つがどのように実証してきましたSpotMiniは、建設現場での検査を行っています。そして、これはそのような例だけではありません。



安全でない可能性のある場所を移動したり、現場を清掃したり、建設の進捗状況を評価したりすることは、多くの建設現場で避けられない日常の作業です。そして、それらすべては、建設現場で働く人々の安全を確保するという問題に関連しています。英国の開発者は、これらの仕事を新しい4本足の友人であるDogBotに移すことにより、建設の効率に影響を与えるリスクと遅延を減らすつもりです。彼らは、このロボットが建設業界だけでなく、将来的に建設業界で働くという考えを大きく変えることができると信じています。



四つ足のアシスタント


ロボットの4本足のアシスタントであるDogBotは、機械学習アルゴリズムを使用して、動き、環境の知覚、自分の位置の感覚、空間内の位置、周囲の世界の向きを認識します。React Robotics

社の人工知能に関する英国の研究者たちは、現代の状況でロボット工学を使用する可能性を模索しており、彼らの発案による開発であるDogBotは、そのような最初の開発の1つです。 Autodesk Fusion 360ソフトウェアを使用して作成されたロボット「アシスタント」は、3Dスキャンなどのタスクを実行して、建設プロセスを監視し、建設現場でのロジスティクスを管理し、さまざまなセンサーからリアルタイムでデータを収集します。

車輪の付いたトラックや追跡されたトラックの陸上車両とは異なり、4本足のロボットは、ペイロードや機器を運ぶ、非常に困難な険しい地形を移動できます。



「私たちは現実世界で人工知能を使用する方法を見つけたかったのです。私たちが作成しているロボット機器は、その使用における将来の革命への道です。」と、React Roboticsの共同創設者でCTOであるCharles Galambosは言います。「DogBotのようなロボットは、実際に「世界を理解し、それと相互作用します」。



危険な作業条件の場合


DogBotは、リスクの高い産業環境でタスクを実行するようにプログラムできるため、従業員が負傷する可能性のある建設現場に大きな利点があります。人々の安全を確保するために、DogBotは、作業領域の清掃やツールの収集に使用できます。その後、次のシフトの建設作業員は、より清潔で安全な作業環境に戻ることができます。



働く代わりにロボット


開発者によると、DogBotは、Autodeskと提携してアメリカの関連ゼネコンが実施した最近の調査によると、スタッフが不足している建設会社の約80％のソリューションでもあります。DogBotを作成する際、React Roboticsの目標は、専門家がより効率的に作業できる「ツール」を提供することでしたが、DogBotロボットは特定の領域での労働力不足を埋めることができました。



さらに、DogBotは建設状況をよりよく理解することで、建設現場の全体的な生産性を向上させます。フィードバックは良くなっています。すでに何が構築されているかを正確に確認し、次に何をすべきかを決定して、建設現場の作業に費やす時間とリソースの量を減らすことができます。

インテリジェンスのためのプラットフォーム


DogBotロボットに必要なすべてのことを教えるために必要な高レベルの複雑さと大量のデータ-建設現場の移動やナビゲートからタスクの実行方法の理解まで-には、かなりの計算リソースが必要です。これを行うために、React Roboticsは、ThinkStation P920ワークステーションに適合したAI構成やThinkPad P1モバイルワークステーションなど、さまざまなLenovoハードウェアを活用しました。



Lenovo ThinkStation P920ワークステーションには、2つのIntel Xeonプロセッサー、3つのNVIDIA Quadro RTXグラフィックスカード、AI Tensorコア、およびさまざまな入出力インターフェイスが搭載されています。レンダリング、モデリング、視覚化、ディープラーニングシステム、AIに使用できます。 React Roboticsでは、データコンパイルシステムとして機能し、リソースを必要とする機械学習アルゴリズムを実行し、ディープラーニングワークフローにサービスを提供するために使用されます。



以前は、トレーニングプロセスに数日かかっていましたが、新しい機器により、React Roboticsは結果をほぼ瞬時に確認できるため、設定をすばやく変更して、テスト環境でテストを実施できます。最終的に、市場投入までの時間を短縮しました。ThinkPad P1ワークステーションは、DogBotのプログラミングと展開に使用されます。



ロボットを訓練するには？


「私たちはパートナーとの仕事を非常に重視しています。彼らはテクノロジー採用に関する私たちのビジョンを共有しています」とReact RoboticsのCEOであるGregory Eppsは語っています。



「私たちのロボットは、実際に周囲の世界を認識し、それと相互作用することができます。彼の各脚には12自由度があります。歩くことができ、さまざまなセンサーを装備できます。その部品のほとんどは3Dプリンターで印刷されます。私たちはAutodesk Fusion 360を使用しているため、何か新しいものをすばやく開発し、1日でパーツをテストできます。ロボットの機能が急速に発展していることがわかります。ロボット工学や人工知能の分野では、毎日何か新しいことが起こります。そして、パートナーは私たちが有意義な結果を達成するのを助けます」とチャールズ・ガランボスは言います。



人工知能、機械学習、ディープラーニングがすべての業界に浸透し続けるにつれて、高性能のハードウェアソリューションの必要性も高まっています。Lenovoワークステーションは、複雑なAIプロジェクトの実装を支援します。 Lenovo Pシリーズワークステーションは、今日のAI、機械、ディープラーニングアプリケーションの厳しいパフォーマンス要件を満たすように設計されています。 GPUでのコンピューティングのためのAIプラットフォームと、データ分析および処理のためのソフトウェアシステムの統合により、パフォーマンスが保証されます。

これにより、データ準備、モデルトレーニング、視覚化タスクなどの機械学習アルゴリズムとディープラーニングアルゴリズムの処理を高速化し、テンソルを備えたNVIDIA Quadro RTX GPUに基づくソリューションを使用して、有用な情報の取得を高速化し、データ処理および分析プロジェクトのコストを削減できます。カーネル。



Lenovoワークステーションには、デスクトップPCに基づくディープラーニングモデル用のThinkStation P920から、人工知能およびペリフェラルコンピューティングのモデルを開発するためのThinkStation P520から、人工知能に基づいて推論を生成するユニバーサルThinkStation P330 Tinyまで、さまざまなソリューションが含まれています。
