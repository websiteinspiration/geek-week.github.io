SQL Serverの新しいバージョン-SQL Server 2019がリリースされたことをご存知ですか？多くの新しい便利な機能のおかげで、このリリースは前例のない関心を呼び起こし、組織による情報システムの翻訳の速度は、以前のすべてのバージョンのパフォーマンスを超えています。 SQL Server 2019への移行に伴い、企業はコストのかかるHadoopのインストールを徐々に放棄し、ビッグデータクラスターをマスターしています。データへのアクセス方法も変化しています。最新のデータ視覚化機能により、高価なデータの移動やデータの事前コピーを行うことなく、新しいプロジェクトを設計できます。既存のワークロードは、新しいインテリジェントデータベース機能のホストによって加速されます。
