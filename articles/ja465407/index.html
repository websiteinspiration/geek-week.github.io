会社について


まず、会社とその歴史をより詳しく紹介したいと思います。
エクストリームネットワークス  は、高度なイーサネットテクノロジーソリューションを推進し、イーサネット標準を開発するために1996年に設立された通信会社です。ネットワークのスケーリング、QoS、高速リカバリに関する多くのイーサネット標準は、エクストリームネットワークスのオープンパテントです。本社は米国のサンノゼ（カリフォルニア）にあります。現在、エクストリームネットワークスは、特にイーサネットの開発に焦点を当てた公開企業です。

2015年12月現在、従業員数は1300人です。

エクストリームネットワークスは、ユーザーとデバイスの絶え間ない移動、およびデータセンター内外の仮想マシンのクラウドへの移行により、現代のモバイル世界の要件を満たす有線および無線のネットワークソリューションを提供します。単一のExtremeXOSオペレーティングシステムを使用すると、通信事業者、データセンターネットワーク、ローカル/キャンパスネットワークの両方に対応する高度なソリューションを作成できます。

  • Extreme Networksは、ロシアにRRC、Marvel、OCS、Comptekの4つの公式販売代理店と、100社を超えるパートナーがあり、その数は常に増加しています。
  • エクストリームネットワークスには、ベラルーシにSolideks、IUK、Abrisの3つの公式ディストリビューターがあります。Solidexは認定トレーニングパートナーのステータスを持っています。

  • 中央アジアの国々だけでなく、ジョージア、アルメニア、アゼルバイジャンでも、公式の販売代理店はRRCとAbrisです。

V400シリーズスイッチ


これらは、仮想ポート拡張テクノロジー（IEE 802.1BR仕様に基づく）を使用するスイッチです。スイッチ自体は仮想ポートエクステンダーと呼ばれます。

このテクノロジーの本質は、すべての制御およびデータプレーン機能がスイッチから集約スイッチ（コントローラーブリッジ/ CB）に引き継がれていることです。

スイッチのコントローラーブリッジとして使用できるのは、モデルスイッチのみです。

  • x590
  • x670-g2
  • x620-g2

これらのスイッチの一般的なスイッチング方式を説明する前に、それらの仕様について説明し

802.3af（ポートあたり最大15 W）および802.3at（ポートあたり最大30 W）テクノロジを使用してPoEデバイスを接続および給電するためのPoEポートを備えたスイッチもあります。

仮想ポート拡張テクノロジーの利点：

  • メンテナンスの容易さ-V400スイッチの1つに障害が発生した場合、それを交換するだけで十分で、新しいスイッチが自動的に検出され、CB操作用に構成されます。したがって、各アクセススイッチを設定する必要はありません。
  • 構成全体がCBにのみ配置され、V400スイッチは追加のCBポートとしてのみ表示され、これらのスイッチの管理を簡素化します
  • コントローラブリッジと組み合わせてV400を操作すると、V400スイッチでコントローラブリッジのすべての機能を利用できます

テクノロジーの制限-V400スイッチ（2300アクセスポート）の最大48のポートエクステンダーがサポートされます。

ERS5900シリーズ


ERSシリーズの最新および最も古いモデルは、ERS5900スイッチです。

面白いから：

  • シリーズの一部のスイッチでは、ユニバーサルPoEが実装されています。ポートに60 Wを出力して、特殊なデバイスや小型のスイッチ/ルーターに電力を供給できます。
  • 100のポートスイッチがあり、合計PoEバジェットは2.8 kWです。
  • 2.5GBASE-T（標準802.3bz）をサポートするポートが利用可能
  • MACsec機能のサポート（802.1AE標準）

シリーズスイッチの構成と機能は、次の表で最もよく説明されています。




* 5928GTS-uPWRおよび5928MTS-uPWRスイッチは、いわゆる4ペアPoEイニシアチブ（別名ユニバーサルPoE-uPoE）をサポートします。たとえば、一部のアクセスポートで最大60 Wのデバイスに電力を供給する機能種類のビデオ通信システム、モニター付きの薄型VDIクライアント、PoEを搭載した小さなスイッチまたはルーター、さらには一部のIoTシステム（インテリジェント照明制御システムなど）。

**シリーズ内のすべてのスイッチは、ERSオペレーティングシステムによって制御されます。**

おそらくお気づきのように、シリーズの説明の最後に、どのオペレーティングシステムを実行しているのかを友人に示します。それは、理由があるので行います。多くの人がすでに推測しているように、特定のオペレーティングシステムを管理することは、各オペレーティングシステムの構文コマンドと設定ブロックの個別のセットを意味するという事実があります。

例：
Avayaスイッチのファンがおそらく気づいたように、ERSシリーズスイッチのL2機能の説明には、インターフェースを組み合わせるためのグループの最大可能数（通信リンクの集約と予約）を説明するMLT / LACP Groups行があります。MLT指定リンク集約を構成するときにコマンド構文で直接使用されるAvaya Holdingスイッチでのリンク集約に一般的です。

問題は、ExtremeNetworksが、その開発戦略に従って、2017年から2018年にAvaya Holdingsを買収したことです。Avayaホールディングスには、当時そのスイッチのラインがありました。したがって、ERSシリーズは本質的にAvayaスイッチラインの継続です。

EXOSシリーズスイッチ


EXOSシリーズは、主力のExtremeシリーズと見なされています。このラインのスイッチには、最も強力な機能が実装されています。これは、標準プロトコルのセットと「独自の」Extremeプロトコルのセットの両方です。これについては、今後説明します。

その中にあなたはあらゆる好みのためのスイッチを見つけることができます：

  • あらゆるネットワークレベル-アクセス、集約、コア、データセンタースイッチ
  • 10/100/1000 Base-T、SFP、SFP +、QSFP、QSFP +ポートの任意のセットを使用
  • PoEサポートありまたはなし
  • 重要なネットワークノードのフォールトトレランスを確保するための、いくつかのタイプの「スタッキング」のサポートと「クラスタリング」のサポート

最新シリーズのX440でこのシリーズのレビューを始める前に、EXOSオペレーティングシステムのライセンスポリシーについて説明します。

X460-G2シリーズ


X460-G2シリーズスイッチは、QSFP +ポートを備えた最新のスイッチシリーズです。このシリーズの特徴は次のとおりです。

  • さまざまなポートの柔軟なセットを備えた多数のモデルの存在
  • ポート付きの追加のVIMモジュールを使用するための個別のVIMスロットの存在-SFP +、QSFP +、スタッキングポート
  • 標準2.5GBASE-T（802.3bz）の一部のモデルでのサポート
  • MPLSサポート
  • 同期イーサネットおよびTM-CLKサポート
  • 追加の電源でスイッチのすべてのモデルを完了する機能

X670-G2シリーズ


X670-G2シリーズスイッチは、高性能1RUアグリゲーションレベルスイッチまたは高密度ポートコアであり、V400スイッチのコントローラーブリッジとしても機能します。 48と72の固定10 GE SFP +ポートと4つのQSFP +ポートを備えたスイッチを注文できます。

これらのスイッチには、2種類のライセンス（初期ライセンスとしてのAdvanced Edge License）とコアライセンスが付属し、SummitStack-V、Summit-Stack-80、SummitStack-160、SummitStack-320の4つの異なるスタッキング方法をサポートしています。

大規模なインターネットプロバイダーや非常に大規模な企業の場合、MPLS Feature Packが役立ちます。これにより、機能を拡張し、スイッチをLSRまたはLERカーネルルーターとして使用し、それらを使用してサポート付きのマルチサービスネットワークを構築できます-L2VPN（VPLS / VPWS）、BGPベースのL3VPNS 、LDPプロトコルに基づくLSP、RSVP-TE、静的プロビジョニング、および-VCCV、BFD、CFMなどのさまざまなツール。

スイッチは2つの構成で利用可能です。




*  スタッキングはシリーズと互換性があります-X440、X440-G2、X450、X450-G2、X460、X460-G2、X480、X670、X670V、およびX770
