OpenStackと互換性のあるホスティングで3CXクラウドベースのPBXを作成する

多くの場合、クラウドに3CX PBXをインストールする必要がありますが、選択したクラウドプロバイダーは、3CXがサポートするプロバイダー（Mail.ruクラウドソリューションなど）のリストに含まれていません。何も間違ってない！これを行うのは難しくありません。プロバイダーがOpenstackインフラストラクチャをサポートしているかどうかを確認するだけです。3CXは、他の企業の中でもOpenstack の開発を後援し、Openstack APIと監視および管理のための標準のHorizo​​nインターフェースをサポートしています。 

したがって、3CXをインストールするには、次のものが必要です。

  • アカウントがアクティブ化されたOpenstack API準拠のクラウドプロバイダー
  • Openstack API URLを含む、PBX ExpressサービスのAPIアクセス資格情報

Dreamhostプロバイダーの例を使用して、3CXクラウドのインストールを検討してください。

OpenStackホスティングアカウントの作成


1. https://www.dreamhost.com/に移動し、[ホスティング＆サーバー]> [クラウド]セクションに移動します。開いたページで、[DreamComputeを使ってみる]をクリックします。



2.資格情報を入力します。Compute APIには別のパスワードが必要です。これは、PBX ExpressサービスがDreamst Openstackプラットフォームとデータを交換するために使用します。Openstackプロバイダーは同じ情報を要求します。



3.登録後、アカウントの確認と有効化に必要な情報が記載された電子メールが届きます。通常、オペレーターはパスポートまたはクレジットカード情報のスキャンを要求します。

3CX PBX ExpressのOpenStackアカウント構成


4.アカウントを確認したら、コントロールパネルに移動し、[クラウドサービス]セクションに移動して、[DreamCompute]をクリックします。



5.ここに、テナントIDとユーザー名（テナントIDとユーザー名）が表示されます。それらを修正します-PBX Expressサービスを介して3CXをインストールするときにこのデータが必要になります。次に、[ダッシュボードの表示]をクリックします。



6.コントロールパネルで、[アクセスとセキュリティ]> [アクセスAPI]ページに移動します。ここで、ID行のURLを修正します。このURLで、PBX ExpressはDreamhost OpenStackクラウドにアクセスします。

 
7. [アクセスとセキュリティ]セクションで、[キーペア]タブに移動し、[キーペアの作成]をクリックします。



8.表示されるウィンドウで、キーペアの名前を入力し、[キーペアの作成]をクリックします。キーが作成され、ディスクにダウンロードされます。



PBX Expressを介して3CXクラウドをインストールする


9.これで、3CX PBX Expressサービスを介して3CXクラウドベースのPBXをインストールする準備ができました。PBX Express Wizardの1つの段階で、Openstackホスティングのパラメーターのリクエストが表示されます。これらのパラメーターを使用して、PBX Expressは3CXがインストールされたVPSサーバーを自動的に作成します。Dreamhostには次のパラメーターが指定されています。

  • ホスティングの選択-VPSクラウドプロバイダー（OpenStack API v2準拠）
  • Cloud Provider API URL-ステップ6のURL
  • ユーザー名-手順5のユーザー名
  • パスワード-手順5のテナントID



10.次のページで、何を表示すべきかわからない場合は、[次へ]をクリックします。これらのパラメーターにガイドされている場合は、サーバーの場所、サーバーの容量、および使用するSSHキーのペアを指定できます。インストールするイメージは、Debian-9のみである必要があります。


次に[次へ]をクリックして、個人データを入力します。間もなく、クラウドベースのPBXの準備が整います。メールで通知が届きます。保存してください-重要な情報が含まれています。 

これで、PBX接続ユーザー、外線などの構成を続行できます。
