同期フエテ：ナノテクノロジーにおける生物学的モーター



私たちの銀河の広大な広がりには、世界中の科学者が発見と解明に懸命に努力しているという多くの秘密が隠されています。ただし、神秘的であるために何かが大きい必要はありません。これの鮮明な証拠は、細胞レベルで横たわっている世界です。細胞の形態、構造、機能、目的が最も多様なものの多くは、体の生命の維持という共通のタスクを一緒に果たします。誇張した場合、人々と同様に細胞にも職業があります。郵便配達、細胞と組織の間で情報を伝達すること。感染を特定して戦う国境警備隊;情報収集、保管などのアーキビストこの信じられないほどの専門分野では、少なくとも私たちにとって、細胞の動きに必要な機械的な力を生成する生物学的モーターという非常に珍しい職業があります。

これらの細胞は、ナノテクノロジーのコンテキストで特に興味深いものです。以前は、生物学的モーターに基づく実行可能なナノデバイスの実装に問題がありました-モーターをより大きなシステムに統合して、その機械的動きを他の分子ユニットと効果的に接続できるようにする必要があります。ミュンヘン大学（ドイツ）の科学者たちは、この概念の実装に近づくことができました。モデルの作成に使用された特定の細胞と分子ユニット、それらの作業はどのように制御されたのですか、作業システムのタスクは何でしたか、それはどのような結果を示しましたか？これらの質問に対する回答は、研究グループの報告書に記載されています。行け。

研究の基礎


先に述べたように、生物学的モーターは、細胞の移動と同様に細胞内輸送を実現するために必要な機械力を生成する細胞です。これらのモーターには、モータータンパク質とタンパク質複合体が含まれます。

その非標準的な活動を実行するために、モータータンパク質は他の機械と同様に燃料を必要とします。アデノシン三リン酸（ATP、C 10 H 16 N 5 O 13 P 3）がその役割を果たします。 ATPは、生体システム内のすべての生物学的プロセスのための普遍的なエネルギー源です。

モータータンパク質は加水分解* ATP によって機能し、タンパク質が化学エネルギーを機械的仕事に変換することを可能にします。
加水分解*は、溶解した物質と溶媒の間の交換分解反応であり、物質（この場合はATP）と水の分解が起こり、新しい化合物が形成されます。
特に、モータータンパク質にはミオシン、キネシン、ダイニンが含まれます。ミオシンは、すべての筋肉タンパク質の約40〜60％を占め、筋肉の収縮プロセスに関与しています。

数年前に科学的ミームとなったキネシンは、微小管（タンパク質の細胞内構造）を移動し、有糸分裂、減数分裂、小胞輸送のプロセスに参加しています。


キネシンが微小管を移動し、ミームに変化する様子のビデオデモ。 （オリジナルは「The Inner Life of a Cell」から取られました。ビデオにオーバーレイされている音楽は、Stayin 'Alive、Bee Gees、1977です）。

キネシンと同様に、ダイニンも細胞骨格の微小管に沿って移動し、貨物輸送のプロセスに参加します（小胞、ミトコンドリアなど）。

ナノテクノロジストによるバイオモーターへの関心は、いくつかの重要な要因によるものです。ナノスケール、生体適合性、遺伝子工学を使用して特定の機能を持つバイオモーターを作成する能力。

現在、生物学的モーターの作用原理に基づいた開発がいくつかあります。ただし、単一の分子モーターで実行できる実際の作業量に関する情報は、完全な実装にはまだ十分ではありません。科学者にとってのもう1つのパズルは、分子運動ブロックをより大きな構造に統合して、その方向付けられた動きを構造内の離れた点でのポテンシャルエネルギーの増加に効果的に変換する方法の問題です。

明らかに、多くの障害がありますが、これは科学者を止めたことはありません。今日検討している研究では、科学者たちは分子装置1について説明します。これにより、モーターユニットからリモートレシーバーユニットへのポテンシャルエネルギーの伝達が可能になり、リモートレシーバーユニットの動きが加速します（画像1）。

レシーバーユニットの役割は、アキシャルキラル*ビアリール*によって演じられました。これは、非テザーフォーム（モデルシステム2）では、アトロプ異性化がエナンチオマー1：1（1a）の平衡方向にゆっくりと無方向に回転します。
軸性キラリティー*は、ある軸（キラリティーの軸）に対する置換基の非平面的な配置の結果として発生します。
ビアリール* -単結合で接続されている場合、2つの芳香族化合物またはアリール基の組み合わせである部分構造を含む化合物。
分子モーターブロックの共役により、このアトロプ異性化を平衡からそらすことができます。その結果、ビアリールのアトロプ異性化はもはや受動的ではなく（モーターの作動後）、モーターが能動的に機能しなければならないエネルギー障害となります。

マシン1では、ビアリールの孤立した回転に対する内部障壁を抑制するモーターの作用により、ビアリールのアトロプ異性化率が数桁増加します（モデルシステム2 1bおよび1s）。

Molecular Installation 1には、インディゴ発色団のクラスに属するHTIに基づく分子モーターユニットが含まれています。このタイプのユニットは、光に非常に敏感です。光のおかげで、それは制御することができます。モーターブロックは、モーターの回転に伴って移動しないリモートビアリール軸に共有結合されています。

このインストールと以前のバージョンの違いの1つは、追加の状態（ステップ）の存在です。以前のバージョンには4つの状態があり、モーターの回転ステップによってのみ規制されていました。新しいオプションは6速システムです（画像2）。

6つの異性体状態は次のように命名されています。リラックス状態A、およびBR、CT、CR、DR（システムの中間状態）のAR。

これらのステップの5つを実験的に観察できます。これにより、6番目のステップの存在が確認され、ブロックモーターとブロックレシーバーの関連システムの360度回転の完全な方向性が確認されます。

モーターシステム1は、臭素化されたHTI前駆体に収束法によって合成され、ボロン酸エステルの機能を含む共有結合が銅触媒クリック反応を介して結合されました。

続いて、鈴木反応*は大環状化*を与え、続いて酸化して最終構造1を与えます。
鈴木反応*は、アリールおよびビニルボロン酸とハロゲン化アリールまたはハロゲン化ビニルとの有機反応であり、Pd（0）錯体によって触媒されます。
大環状化* - 大環状化合物の形成につながる環化反応、すなわち 9個以上の結合した原子が環を形成している化合物。
システム1のARとCRの最も安定した2つの状態とシステム2の最も安定した状態について、構造分析に適した結晶が得られました（1dでラセミ* ARの®構成異性体、1eで CR ）。
Racemat *は、互いに鏡像である2つの立体異性体の等モル混合物です。ラセミ化合物は結晶で構成され、それぞれの結晶には両方のエナンチオマーの分子があり、それらの比率は1：1です。
AR / ATとCRはHPLC（高速液体クロマトグラフィー）を使用して分離できるため、異なる温度での加熱と照射の条件下でのそれらの動作を個別に調査できます。 （CDCl 2）2のラセミCR溶液を80°C-140°C に加熱すると、93％のケースでより安定したARが形成されました。これにより、これらの2つの状態の間のこの温度範囲における自由エンタルピーの差∆G = 1.8–1.9 kcal / molが設定されます。対応する速度論的分析により、Z / E二重結合の熱異性化を伴う、80°Cで28.2 kcal / molの高エネルギー障壁が明らかになりました。

CD 2 Cl 2 / CS 2で純粋なCRを-105°Cに冷却する場合（比率4/1）、450 nmの光を照射すると、新しい信号セットが表示されます。これは、既知の信号セットARおよびAT（3a）とは異なります。

DR異性体の信号は、-80°Cの温度および完全な暗闇で28分以内に75％まで減衰しますが、逆にAT異性体の信号は増加します（3）。

DR（残りの12％）とAT（88％）の間の熱平衡は-60°Cで観察され、2つの状態間で∆G = 0.84 kcal / molで表されます。熱分解の速度論的分析により、付随する自由活性化エンタルピー∆‡G = 13.9 kcal / mol（-80°C、このプロセス）が明らかになりました。 -40〜0°Cの温度では、AT信号の減衰と既知のAR信号の付随的な増加が、それらの間で平衡に達するまで観察されます。このプロセスの速度論的分析により、–40 ... 0°Cの温度で∆‡Gが18.4〜19.3 kcal / molに等しいことがわかりました。

このように、CRの照射により最初のDR光異性化生成物が生成され、モーターブロックの単一のらせんを反転させることにより、AT異性体に熱的に変換されることがわかりました。 CD 2でAR / ATの平衡解を冷却する場合Cl 2 / CS 2は-105°Cまでで、450 nmの光を照射すると、その普及によりAR光反応が観察されました。既知のCR（3f）信号とは異なる新しい信号セットも表示されます。これらの新しい信号は、AR光反応の産物であり、ARからBRへの光異性化の予想される方向を確認します。これにより、さらに安定化し、低温でも高速ヘリカルインバージョンによるCTへの反転が起こります。

-80°Cの温度で完全な暗闇の中で、BR / CT信号はほぼ完全に消え、CR信号のみがさらに増幅し続けます（3e）。これにより、CTとCRのエネルギー差の下限が0.98 kcal / molになります。

速度論的分析により、Δ‡G = 13.4–13.5 kcal / molが決定されました。これは、STのCRへの変換に必要です。 CRにおけるCTのアトロプ異性化の活性化エネルギーは、ARにおけるATのアトロプ異性化の活性化エネルギー（18.4 kcal / mol）に比べて減少していることがわかります。これは、ATに比べてCTの応力（ひずみ）が高いためである可能性があります。これは、構造Aではなく、構造Cのリンカーチェーンの接続点として機能する2つのフェノール性酸素原子間の距離が大きいためである可能性があります。結晶状態では、CR 2つの酸素原子の間は10.6ですが、ARでは7.6です。これにより、C構造よりもA構造の方が、リンカーチェーンの構造（原子の空間的位置）の自由度が高くなります。 CTからCRへの熱変換の自由な活性化エンタルピーを考慮に入れると、次に、この変換プロセスを-105°Cで完全に停止できます。

-105°Cの温度で、AR光への強い露出中にCTの蓄積の程度を確認することが可能です。これは、BRとSTのエネルギー差が2.0 kcal / molを超え、ST構造自体が光化学プロセスに関与していない場合に可能です。ただし、-105°CでのAR照射中のCT蓄積は、CTとBRの間に急速な熱平衡があるため、不可能です。

HTIモーターに基づく通常の4速メカニズムとは異なり、360度の全回転には、モーターからビアリールブロックに伝達される位置エネルギーの増加という2つの追加ステップが含まれます。これらの2つのステップ中に、モーターとビアリールブロック（4b）の間にエネルギー結合が形成されます。後者はもはやエネルギー的に退化しません。

代わりに、1回のフル回転中のモーターの「ラチェット」ステップ（5秒および5d）。このエネルギー変化は、次の熱活性化段階でのアトロプ異性体の相互変換の最大90％を提供し、したがって、メカニズムの1つの作動サイクル中にビアリールブロックがほぼ完全に回転します。

この観察は、ビアリールブロックがポテンシャルエネルギーの増加を受けたことを示している可能性があります。これは、モーターブロックの動作の結果です。エネルギーの最初の増加がビアリールブロックに到達するものよりもはるかに大きいことは注目に値します。したがって、モーターからビアリールブロックに伝達されるのは72％だけです。

上記の観察は、モーターとビアリールブロック間の接続の長さが、エネルギー移動の程度を変更するプロセスにおける重要な側面であることを示唆しています。したがって、接続が短いほど、より多くのポテンシャルエネルギーをBRおよびDRからATおよびCTに転送できる可能性があります。

エピローグ


この研究では、科学者たちは生物学的モーターと分子要素の相互作用に基づいて、発明の動作原理を説明しました。開発された設備により、モーターから特定の要素（この場合はビアリールブロック）にポテンシャルエネルギーを伝達できます。運動活動の結果として、ビアリールの熱的アトロプ異性化は、一方向かつより速い速度で進行させられます。エネルギー伝達は、回転メカニズムの熱的に活性化されたラチェット（ステップ）中に発生します。また、モーターの初期エネルギーの約72％が特定のユニットに伝達されることもわかりました。

本発明は、完全な生物学的ナノメーターデバイスの作成に向けた重要なステップである。そのようなメカニズムの設計におけるバイオモーターの使用は、その生体適合性、遺伝子工学による機能の変更の容易さ、およびそれらの自然なナノスケールによるものです。現在の人工モーターは、上記の利点の組み合わせを自慢することはできません。

研究者たちは、彼らの作品の次のステップは、彼らのインスタレーションが抽象的なアクションを実行できるだけでなく、有用であるということを実証することだと言う。言い換えれば、彼らの将来の創造は、分子レベルで有用な機能を実行するためにさらに洗練されるでしょう。
