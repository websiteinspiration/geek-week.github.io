Working with code: three open tools

Let's talk about a couple of text editors and a utility for designing code in the documentation.



More Developer Tools: APIs and UI Elements





Photo - ThisisEngineering RAEng - Unsplash





Micro



This is a terminal text editor, sharpened by modern systems. The source code is on GitHub and is distributed under the MIT license . The developers say that the product is the ideological successor of the Nano editor for Unix-like operating systems.



Among the interesting micro functions, we can single out support for the mouse (drag, double-click, tripple-click) and several cursors, automatic linking of the code (error checking) and syntax highlighting for more than 90 languages . In a thematic thread on Hacker News, residents note that the utility is similar to the proprietary Sublime Text editorand has many of its functions: multi-line editing, plugin management - even hotkeys are similar.



The tool appeared about three years ago, so bugs are still encountered in its work . Although one of the residents of HN noted that recently micro stability has grown. In any case, the tool has a community (more than 14 thousand stars on GitHub), and there is active communication in the Gitter chat.





Atom



It is a cross-platform text editor and development tool provided by GitHub. It is built on HTML, JavaScript, CSS, Node.js and uses the Electron framework .



It offers extensive settings for the interface and text - you can change the styles, key bindings, even the appearance of the editor, if you know JavaScript. Almost all settings are in CSON files (JSON format, but for CoffeeScript objects). Since GitHub is developing Atom, it has an extension that allows you to access the version control system directly from a text editor - manage branches, execute commit , push and pull commands . There is also a Teletype package- Using it, code can be written by several developers at the same time.





Screenshot: atom.io / Working with Git in Atom



Atom is a relatively young tool, so there are performance problems, however, the authors are engaged in optimization .

The tool allows you to create beautiful screenshots of code for presentations, articles or documentation. It is enough to copy a piece of the program in the field on the site and save the image (if you wish, you can post the picture directly to Twitter).
