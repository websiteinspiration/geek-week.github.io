- Warum wird das benötigt?

Der häufigste Fall ist, dass sich die Buchhaltungsdatenbank im Unternehmen befindet und nicht auf dem Heimcomputer des Buchhalters. Wenn ein Buchhalter krank wird oder ihm etwas anderes passiert, wird es schwierig sein, alles zu bekommen. Seltenere Fälle sind, wenn Sie Ihre Arbeitsumgebung von überall auf der Welt auf Ihrem Telefon haben müssen oder wenn Sie häufig die Computer anderer Personen verwenden.

Es ist auch nützlich, um Spieleserver zu erstellen, an der Börse zu handeln, Torrents, verschiedene sich wiederholende Prozesse, Programme zur Automatisierung sozialer Netzwerke (und wie Betrüger, was bereits vorhanden ist), SEO-Dienstprogramme, die Vorbereitung auf die weiße Buchhaltung und Reisen (insbesondere, wenn Sie dies nicht möchten) Bringen Sie Ihren Heimcomputer für einen Monat irgendwohin in eine benachbarte Stadt und so weiter. In den meisten Fällen wird VPS für die Sicherheit der Kundenbank des Unternehmens verwendet.

Die folgende Anleitung enthält häufig gestellte Fragen.

Es gibt eine Reihe von Protokollen wie RDP oder VNC, mit denen Sie das Bild vom Bildschirm des Remotecomputers auf Ihren übertragen und Mausbewegungen und Tastenanschläge auf Ihre Fernbedienung übertragen können. Im einfachsten Fall verwenden Sie die Standard-Windows-Anwendung „Remotedesktopverbindung“. Auf Tablets und Telefonen müssen Sie eine Anwendung installieren, die Remoteverbindungen unterstützt und Klicks auf den Touchscreen mit Mausbewegungen übersetzt oder in Windows auf der anderen Seite nicht auf den „Bildschirm“ klickt. Für das Telefon wird auch ein Bildlauf auf dem Bildschirm hinzugefügt, da dieser nicht immer vollständig passt. Jede Anwendung, jede mit RDP-Unterstützung, z. B. Remotedesktop von Microsoft, ist geeignet. Oder Teamviewer im Allgemeinen, aber es wird auch ein Agent auf Ihrem Win-Desktop benötigt.

Am einfachsten ist es, über einen Browser auf den Remotecomputer zuzugreifen, wenn der Hoster alles dafür konfiguriert hat. Wir sind eingerichtet. Aber durch Anwendungen schneller.

- Wie richte ich das erste Mal ein?

Die Logik hinter dem Aufheben eines Remotedesktopservers besteht darin, eine virtuelle Maschine (VPS) bereitzustellen und anschließend eine Windows-Distribution zu installieren, die Remotekonnektivität unterstützt. Das erste erfolgt über das persönliche VPS-Hosting-Konto (es dauert vier bis sieben Minuten mit SLA für 15 Minuten), das zweite könnte Sie dazu bringen, das Material gut zu lernen, aber in unserem Fall ist Windows Server auf dem Markt und Sie können sofort einen Computer mit einem erweiterten Betriebssystem bestellen System. Der Zugriff auf die Verbindung erfolgt direkt in Ihrem Konto. Es bleibt nur zu verbinden.

- Was sind die Systemanforderungen für Windows Server?

Die Mindestkonfiguration beträgt 2 GB RAM, 20 GB auf der Festplatte. Für Win Server 2003 reicht ein Kern aus, für Win Server 2016 oder 2019 werden bereits zwei benötigt. Es ist besser, etwas mehr Speicherplatz auf der Festplatte hinzuzufügen. Beide Betriebssysteme sind mit 20 GB knapp - S2012 kann beispielsweise bereits nicht aktualisiert werden. Einige Benutzer installieren Win auf VPS mit 500 MB RAM. Wir empfehlen einen Tarif für 579 Rubel pro Monat (dies beinhaltet keine Rabatte für Habr, Sie können 15% des unten stehenden Promo-Codes und weitere 20% bei Zahlung für das Jahr wählen), wenn es einen Kern, 2 GB RAM und 20 GB Festplatte gibt. Zu günstigeren Preisen ist es besser, Windows Server Core oder ein anderes Betriebssystem zu verwenden.

- Ist es möglich, geplante Sicherungen zu konfigurieren?

Ja, Sie können beispielsweise einmal täglich eine Kopie des VPS-Geräts erstellen. Einrichtung über Ihr Konto. Eine Kopie wird in einen unabhängigen Speicher im selben Rechenzentrum erstellt, in dem Ihr VPS bereitgestellt wird. Wenn Sie in ein anderes Rechenzentrum kopieren möchten, müssen Sie es manuell konfigurieren.

- Was ist, wenn Windows auf der anderen Seite nicht mehr reagiert?

Wenn etwas mit dem Betriebssystem oder dem Terminalzugriffsdienst passiert, können Sie keine Verbindung über RDP herstellen, da hierfür ein funktionierendes Betriebssystem, Netzwerktreiber, eine funktionierende Netzwerkverbindung und RDP aktiviert sind. Nach dem Auspacken haben wir alles. Wenn Win jedoch "zusammenbricht", kann ein weiterer spezieller Modus verwendet werden: Wir haben die Möglichkeit, Videos direkt von der Hypervisor-Ebene aus zu senden und Tastenanschläge und Klicks über den Hypervisor zu übertragen. Dieser Modus ist sehr ressourcenintensiv, daher ist seine Verwendung begrenzt. Bei Problemen können Sie es jedoch verwenden, um im abgesicherten Modus neu zu starten. Wenn das Problem schwerwiegender ist, können Sie sich entweder an den Support wenden oder Windows mit wenigen Klicks neu installieren.

Der Notfallmodus kann ziemlich lange auf Aktionen reagieren, sieht aber optisch ziemlich normal aus. Es ist so üblich, dass wir Benutzer haben, die es regelmäßig verwenden. Aber wir müssen absolut nicht im Alltag außerhalb eines Notfalls eingesetzt werden, da es für Windows korrektere und bequemere Möglichkeiten gibt - natürlich ist dies RDP.

- Was ist mit der Betriebssystemlizenz?

Normalerweise bietet Ihnen VPS-Hosting einen virtuellen Server und überprüft nicht, was Sie damit tun (nur zum Laden von Ressourcen und zur Art des Datenverkehrs). Das heißt, theoretisch können Sie dort jede Software ablegen, einschließlich Raubkopien. In der Praxis ist es jedoch bequemer, einen Computer mit einem Betriebssystem mithilfe einer der Hosting-Vorlagen zu erstellen. In unserem Fall ist eine Lizenz für Windows Server (oder Windows Server Core ohne grafische Oberfläche) bereits im Tarifplan enthalten, da wir alle auf Hyper-V hosten und vollständig lizenziert sind. Einige andere Hosting-Unternehmen eröffnen derzeit einen neuen "Stern" mit den monatlichen Kosten für die Anmietung einer Lizenz.

- Kann ich mein eigenes Betriebssystem-Image mitbringen?

Ja, das können Sie, und es gibt zwei Möglichkeiten. Sie können einen Standard-VHDX-Container für Hyper-V erstellen und an den Support übertragen. In diesem Fall kann es beim Einrichten einer Netzwerkverbindung zu Schwierigkeiten kommen. In Ihrem fertigen Image ist möglicherweise kein Treiber für eine geeignete Netzwerkkarte vorhanden. Sie müssen es verbinden und installieren. Trotzdem ist dies eine gute Möglichkeit, alles, zum Beispiel von einem alten Laptop, online zu übertragen. Sie können die Unterstützung auch auf das ISO-Image übertragen, das auf einem virtuellen Server bereitgestellt ist und von dem aus Sie im Notfallmodus über Ihr persönliches Konto die Installation selbst durchführen können.

Diese Dienstleistungen werden nur in Tarifplänen von 1.000 Rubel pro Monat angeboten.

- Was ist mit Lizenzen für andere Software?

Beim Hosting wird normalerweise nicht angezeigt, was auf Ihnen installiert ist (wir sehen es beispielsweise nicht, es sei denn, Sie bitten um Unterstützung für etwas Besonderes). Es ist uns egal, was da ist und wie es funktioniert, da dies Ihr Verantwortungsbereich ist. Denken Sie daran, dass keine Unterstützung für Raubkopien angeboten wird und wenn etwas nicht funktioniert.

- Welcher Speicher wird verwendet?

Alle Laufwerke sind SSD-Regale in Servern, bei Tarifen mit Festplatten ist die Geschwindigkeit jedoch auf die Geschwindigkeit von Spindelfestplatten begrenzt. Duplizierung - RAID 5. Es sind noch einige RAID 6 übrig, aber es gibt bereits einige. Wenn eines der Laufwerke ausfällt, beginnt der Wiederaufbau des Raids. Wenn zu diesem Zeitpunkt ein Stromausfall auftritt, gehen Daten verloren. In diesem Fall verfügen wir über zwei separate Netzteile mit separaten Kabelwegen zu jedem physischen Server und einer technischen Sicherung der gesamten Maschine (ohne selektiven Zugriff auf einen separaten Speicher). Schäden durch Hackerangriffe sind bei AIG versichert.

- Welchen Aufwand verursacht ein Hypervisor für die Leistung?

Mit einer Kombination aus Hyper-V + Windows deutlich weniger als ein Prozent.
