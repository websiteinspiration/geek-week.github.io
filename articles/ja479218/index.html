selfie2animeボット（既製のUGATITモデルを使用） の比較的成功した起動後、私は同じことをやりたかったが、私のものでした。たとえば、写真を漫画に変えるモデルです。

これが私のphoto2comicsbotの例です。同様のことを行います。





UGATIT モデルはビデオカードに対して重すぎるため、古いが、それほど複雑ではないアナロジーCycleGANに注目しました。

この実装では、いくつかのモデルアーキテクチャと、ブラウザーでの学習プロセスの便利な視覚表示があります。

CycleGANは、単一のイメージを介してスタイルを転送するアーキテクチャと同様に、トレーニングのためにペアのイメージを必要としません。そうしないと、トレーニングセットを作成するためにすべての写真を自分でコミックに再描画する必要があるためです。

アルゴリズムに設定するタスクは2つの部分で構成されます。

ポイント「a」は、通常のGANを使用して実装できます。この場合、訓練を受けた批評家が「コミックに似ている」ことを担当します。

GANの詳細

2番目の部分はやや複雑です。ペアの写真があり、セット「A」とセット「B」に写真があったが、コミック（つまり、モデルから取得したいもの）に再描画された場合、 Generatorによって生成された結果を、トレーニングセットのセット「B」からのペアの画像と比較するだけです。

私たちの場合、セット「A」と「B」は決して相互に接続されていません。セット「A」-ランダムな写真、セット「B」-ランダムなコミック。

予測不能な結果は言うまでもなく、これは批評家の機能を少なくとも複製するため、偽のコミックをセット「B」のランダムなコミックと比較しても意味がありません。

ここで、CycleGANアーキテクチャが役に立ちます。

つまり、これはGANペアであり、最初のペアは画像をカテゴリ「A」（たとえば写真）からカテゴリ「B」（たとえば漫画）に変換し、2番目はカテゴリ「B」からカテゴリ「A」に変換します。



モデルは、通常のGANと同様に、元の写真と復元された写真（「A」-「B」-「A」、「フォトコミック写真」のサイクルの結果）と批評家のデータの両方に基づいてトレーニングされます。

これにより、私たちのタスクの両方の部分を完了することが可能になります。他のコミックと区別がつかず、同時に元の写真に似ているコミックを生成することです。

モデルのインストールと検証


私たちの狡猾な計画を実装するには、次のものが必要です。

  • CUDAおよび8GB RAMをサポートするビデオカード
  • Linux OS
  • Miniconda / Anaconda with Python 3.5+

RAMが8GB未満のビデオカードも、設定を思いついた場合は機能します。 Windowsでも動作しますが、ゆっくりと、少なくとも1.5〜2倍の差がありました。

CUDAをサポートするGPUがない場合、またはすべてを設定するのが面倒な場合は、いつでもGoogle Colabを使用できます。希望する人が十分にいる場合は、チュートリアルと、Googleクラウドで以下のすべてを準備する方法を記入します。

Minicondaはここにあります。
インストール手順

Anaconda / Miniconda（以下condaと呼びます）をインストールした後、実験用の新しい環境を作成してアクティブ化します

（Windowsユーザーは最初に[スタート]メニューからAnacondaプロンプトを起動する必要があります）。

conda create --name cyclegan
conda activate cyclegan

これで、他の環境に影響を与えることなく、すべてのパッケージがアクティブな環境にインストールされます。これは、さまざまなパッケージのバージョンの特定の組み合わせが必要な場合、たとえば、他の誰かの古いコードを使用していて、生活や主要な作業環境を損なうことなく古いパッケージをインストールする必要がある場合に便利です。

次に、ディストリビューションのREADME.MDの指示に従い

ます。CycleGANディストリビューションを自分に保存します

（またはGitHubからアーカイブをダウンロードします）。

git clone https://github.com/junyanz/pytorch-CycleGAN-and-pix2pix
cd pytorch-CycleGAN-and-pix2pix

必要なパッケージをインストールします。 

conda install numpy pyyaml mkl mkl-include setuptools cmake cffi typing
conda install pytorch torchvision -c pytorch 
conda install visdom dominate -c conda-forge

完成したデータセットと対応するモデルをダウンロードします。 

bash ./datasets/download_cyclegan_dataset.sh horse2zebra
bash ./scripts/download_cyclegan_model.sh horse2zebra

ダウンロードしたデータセットに含まれている写真に注意してください。

前の段落のスクリプトファイルを開くと、既成のデータセットとモデルが他にもあることがわかります。

最後に、ダウンロードしたデータセットでモデルをテストします。

python test.py --dataroot datasets/horse2zebra/testA --name horse2zebra_pretrained --model test --no_dropout

結果は/ results / horse2zebra_pretrained /フォルダーに保存されます
