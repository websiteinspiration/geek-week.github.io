Pythonリンクリスト：ボックス内の猫

また会いましたね！Python開発者コースの開始を見越して、Pythonのリンクされたリストに関する小さな著作物を用意しました。



Pythonは非常に便利で多面的な言語ですが、デフォルトでは、リンクリストやLinkedListのようなデータ構造はありません。今日は、このトピックに関する私のベストプラクティスを共有し、このデータ構造とは何かについて少し話します。この記事は、リンクされたリストのトピックに最初に遭遇し、アルゴリズムの観点からそれらがどのように機能するかを理解したい人にとって興味深いでしょう。


LinkedListとは何ですか？


LinkedListまたはリンクリストはデータ構造です。リンクされたリストは、任意の要素から双方向キューを作成する機能を提供します。このようなリストの各要素はノードと見なされます。実際、ノードにはその値と、前のノードと後続のノードへの2つのリンクがあります。つまり、リストは、リストを上下に移動するのに役立つノードによって「リンク」されます。これらの構造上の特徴により、リンクリストからスタック、キュー、またはダブルキューを整理できます。

乾燥した定義を視覚化しましょう。今、私たちは箱に座っている猫を持っています。そして、それぞれの箱にはそれが何であるか、そしてそれが何のために立つべきかが書かれています。



リンクされたリストで何をするか：

  1. これまたはその要素が含まれているかどうかを確認します。
  2. 最後にノードを追加します。
  3. インデックスでノードの値を取得します。
  4. ノードを削除します。

実際、それらを操作するためのオプションは他にもたくさんありますが、これらの基本的な手順の実装について詳しく説明します。それらがどのような原理で構築されているかを理解した後、あなた自身はあなた自身のメソッドを簡単に実装することができます。

まず、2つのクラスを作成する必要があります。

class Box:
  def __init__ (self,cat = None):
    self.cat = cat
    self.nextcat = None

class LinkedList:
  def __init__(self):
    self.head = None

一般に、内部に何らかの値を持つノード（猫）と次のノードへのリンクがあります。つまり、Boxクラスにはそれぞれ猫と次のボックスへのリンクがあります。他のリストと同様に、コヒーレントにも始まり、つまりヘッドがあります。最初は何もないので、開始要素はNoneに設定されます。

リスト内のアイテムです




簡単なものから始めましょう。直列に配置されたボックスの1つに特定の猫があるかどうかを確認するには、リスト全体を循環し、リスト内の要素の値で既存の値を確認する必要があります。

def contains (self, cat):
    lastbox = self.head
    while (lastbox):
      if cat == lastbox.cat:
        return True
      else:
        lastbox = lastbox.nextcat
    return False

リストの最後にノードを追加




最初に、新しいボックスを作成する必要があり、すでに新しい猫を入れています。その後、リンクされたリストの先頭から始めて、現在のノードに次のリンクへのリンクがあるかどうか、そしてリンクがある場合はそこに移動する必要があります。そうでない場合、ノードは最後のノードなので、次のノードへのリンクを作成して同じ新しいボックスをそこに配置する必要があります。作成しました。

 def addToEnd(self, newcat):
    newbox = Box(newcat)
    if self.head is None:
      self.head = newbox
      return
    lastbox = self.head
    while (lastbox.nextcat):
        lastbox = lastbox.nextcat
    lastbox.nextcat = newbox

インデックスでノードを取得する




インデックスcatIndexによってボックスノードを取得する必要がありますが、ノードにはそのようなインデックスが提供されていないため、何らかの置換、つまりインデックスとして機能する変数を考え出す必要があります。この変数はboxIndexです。リスト全体を調べて、ノードの「シリアル番号」を確認し、目的のインデックスと一致する場合は、結果を提供します。

 def get(self, catIndex):
    lastbox = self.head
    boxIndex = 0
    while boxIndex <= catIndex:
      if boxIndex == catIndex:
          return lastbox.cat
      boxIndex = boxIndex + 1
      lastbox = lastbox.nextcat

ノードを削除




ここでは、少なくともいくつかの種類を導入するために、インデックスではなく値で要素を削除することを検討します。

検索はリストの先頭、つまり行の最初のボックスから始まり、ボックスが終了するまで、つまりheadcatがNoneになるまで続きます。現在のノードに保存されている値が、削除する値と一致するかどうかを確認します。そうでない場合は、次に進みます。ただし、一致する場合は削除してリンクを「再リンク」します。つまり、N番目のボックスを削除し、N-1ボックスはN + 1ボックスを参照するようになります。

  def removeBox(self,rmcat):
    headcat = self.head

    if headcat is not None:
      if headcat.cat==rmcat:
        self.head = headcat.nextcat
        headcat = None
        return
    while headcat is not None:
      if headcat.cat==rmcat:
        break
      lastcat = headcat
      headcat = headcat.nextcat
    if headcat == None:
      return
    lastcat.nextcat = headcat.nextcat
    headcat = None
