ポーラグラフ


polargraph.co.uk

彼らはロシア人は長い間利用していると言いますが、速く行きます。半段落を収集するという考えは、5年前にフロッピードライブのプレーヤーに関する記事を読んだ後で思い付きました。最初にフロッピーにドローボットを見つけ、次にハーフパラグラフを見つけました。これとの関係は何かを尋ねないでください、そうではありません。主に指示書付きのマニュアルを使用しました。

アクセサリー


  1. Arduino uno
  2. AdaFruitのMotorshieldモータードライバー。Arduino / Wze用のモータードライブシールドL293Dを持っていました。
  3. 2ステッピングモーターNEMA-17。ハイブリッドステッピングモーター42BYGH202 / Wzeを使用しました。

  4. AC / DC電源
  5. ゴンドラ。ディスクスタンドから作ったのですが、コードを取り付けるための穴を2つ、負荷用の穴を3つ作る必要があります。



    instructables.com
  6. ミシンからのボビンのシャーシ（ボビンとエンジンの軸の間に隙間があります、私はゴム片を入れました）。糸ではなく、釣り糸を使う方がいいです。
  7. モーターが取り付けられている面。大きければ大きいほどいい。インストラクターがモーターサポート（ブラケット）を使用している著者は、A2タブレットに接着した段ボールにそれらを取り付けただけです。

Arduino


Arduino IDE の最新バージョンをインストールします（古いバージョンはコードをロードできません）。スケッチを含むフォルダへのパスを設定します（/ sketches /）：ファイル->設定->スケッチフォルダの場所。

euphy githubからコードをダウンロードします。

アーカイブを解凍します。AdafruitおよびAccelStepperライブラリーを/ sketches / libraries /フォルダーに、polargraph_server_a1フォルダーを/ sketches /フォルダーにコピーします。

Arduino IDEを起動します。ファイル->スケッチのあるフォルダー-> polargraph_server_a1。コンパイルしてボードにアップロードします。

ステッピングモーター


ステッピングモーターについては、Habrに関するすばらしい記事があります。試行錯誤により、私はそれらをボードに接続しました：左のエンジン-ターミナルM1、M2、右-M3、M4。


ソフトウェアのインストール


以前にダウンロードしたeuphyアーカイブから、/コントローラー/フォルダーを解凍します。default.properties.txt

ファイルで、必要な寸法（マシンの幅と長さ、シートの幅と長さ、ホームポイント）を指定します。それらはミリメートル単位で非常に正確に測定する必要があります。そうしないと、画像に歪みが生じます。以下に例を示しますが、そのとおりです。instructables.com アプリケーションを起動します。






sudo chmod +x polargraphcontroller
./polargraphcontroller



SETUPタブで、目的のポート（SERIAL PORT）を選択します。緑の碑文がposhrafの準備ができている上部に表示されます。ゴンドラをHOME POINTポイントに設定します。

[入力]タブで、[画像の読み込み]を選択します。画像をアップロードし、必要に応じてサイズを変更します（画像のサイズ変更）。画像を移動（画像を移動）してシート領域に移動し、目的のフラグメント（領域を選択）を選択し、必要に応じてグリッドサイズを縮小して、レンダーピクセルを押します。ポップアップウィンドウで、描画（TOP-RIGHT）とスタイル（VARIABLE SIZE SQUARE WAVE）の開始位置を選択する必要があります。



右側で、コマンドキューを開始するには、赤い碑文（緑色に変わります）をクリックします：ommandQueue：QUEUE RUNNING。
