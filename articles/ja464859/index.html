複数のNema 17ステッピングモーターまたはNemaStepperを同時に制御する



みなさん、こんにちは。

arduino + nema 17で作業した場合、同時に複数のエンジンを始動するのは非常に難しい場合があると思います。

この問題を解決するにはさまざまな方法がありますが、おそらく最も簡単なのは、NemaStepperライブラリを使用することです。ライブラリは、このタスクを何度も簡略化します。主な利点は、プログラムを停止しないことです。他のすべてのライブラリと同様にインストールされます。MITライセンスで配布。

さて、始めましょう。そして、接続することから始めます。

Simple Nema 17を500ルーブルのaliexpress、L298Nドライバー、arduino unoで使用します。どうぞ：







  • この例では、arduinoに接続されている4つのドライバーを示しますが、使用するドライバーは1つだけです。
  • Nema 17をインターネットに接続する方法についてのチュートリアルはたくさんあるので、ここでは詳しく説明しません。

したがって、モーターをドライバーに接続し





ます。ライブラリはオブジェクト指向です。1つのモーターをオンにする例を見てみましょう。

NemaStepper Stepper1(2, 3, 4, 5, 200, 10, false);
void Setup(){
   Stepper1.SetStepCount(100); //   100 
}
void Update(){
   Stepper1.Step(); // 
}

すべてのライブラリメソッドは、ライブラリのソースコードファイルにあります（ライブラリヘッダーに説明があります）。

*例のコードの詳細。

そして今例のために。

ライブラリには組み込みの例があり（現時点では単独で存在）、シリアルを使用して3つのモーターだけを制御できます。

この例では、次のコマンドをポートに受け入れます。

分解してみましょう。

ヘッダーから始めましょう-接続ライブラリ：

//This file - example of NemaStepper library.
#include "NemaStepper.h"
String inString;
bool IsStepperEnabled = false;

次に、3つのモータは、次のパラメータを使用して、宣言されている：

1.まず、ピン
2。第2ピン
3。第3ピン
4。第4ピン
5回転あたりのステップ数-最もネマ17モータのは、これは200です。
6.スタートの速度
値を示す7、必要に応じて停止後にシャフトを保持するかどうか（真の場合、ドライバーはバーベキューに変わります）

NemaStepper Stepper1(2, 3, 4, 5, 200, 10, false);
NemaStepper Stepper2(6, 7, 8, 9, 200, 10, false);
NemaStepper Stepper3(10, 11, 12, 13, 200, 10, false);

次のポート初期化：

void setup() {
  Serial.begin(9600);
}

次に、重要です！メインループでは、Step（）コマンドを使用してエンジンの位置を更新する必要があります

void loop() {
  if (IsStepperEnabled == true){
    Stepper1.Step();
    Stepper2.Step();
    Stepper3.Step();
  }
  GetCommandFromSerial();
}

以下は、ポートからデータを受信し、モーターのオン/オフを切り替え、速度、ブレーキ、回転を設定するサブルーチンです。

void GetCommandFromSerial() {
  if (Serial.available() > 0) {  //   
    int inChar = Serial.read();
    if (inChar == '/') {
      String command = ((String)inString[0] + (String)inString[1] + (String)inString[2]);
      String param;
      int len = inString.length();
      for (int i = 3; i < len; i++) {
        param = (String)param + (String)inString[i];
      }
      if (command == "MV1") {
        Stepper1.SetStepCount(param.toInt());
        Serial.println(param.toInt());
      }
      if (command == "MV2") {
        Stepper2.SetStepCount(param.toInt());
        Serial.println(param.toInt());
      }
      if (command == "MV3") {
        Stepper3.SetStepCount(param.toInt());
        Serial.println(param.toInt());
      }
      if (command == "SS1") {
        Stepper1.SetSpeed(param.toInt());
        Serial.println(param.toInt());
      }
      if (command == "SS2") {
        Stepper2.SetSpeed(param.toInt());
        Serial.println(param.toInt());
      }
      if (command == "SS3") {
        Stepper3.SetSpeed(param.toInt());
        Serial.println(param.toInt());
      }
      if (command == "SB1") {
        Stepper1.SetBrakes(param.toInt());
        Serial.println(param.toInt());
      }
      if (command == "SB2") {
        Stepper2.SetBrakes(param.toInt());
        Serial.println(param.toInt());
      }
      if (command == "SB3") {
        Stepper3.SetBrakes(param.toInt());
        Serial.println(param.toInt());
      }
      if (command == "EMS") {
        IsStepperEnabled = true;
        Serial.println(param.toInt());
      }
      if (command == "DMS") {
        IsStepperEnabled = false;
        Serial.println(param.toInt());
      }
      inString = "";
    } else {
      inString += (char)inChar;
    }
  }
}

それでは、ボードにダウンロードしてみましょう。

アップロードしましたか？

次に、ポートモニターに移動し、コードからコマンドを入力します。
各コマンドは/で終わります。
最初の3文字はチームの名前です。
名前と/-パラメータの間。
コマンド「EMS /」でモーターをオンにしましょう（Enable MotorS）。
次に、コマンド「SS160 /」（速度の設定）でモーター1の速度60を示します。60は速度です。
そして最後に、コマンド「MV1100 /」（MoVe）で最初のモーターをオンにします。ここで、100は回転数です。
すべてが機能しています。やったー

他のモーターについても同様です。
