パート1：Nest.JSを使用してサーバーAPIを開発する


TL; DR：これは、AngularとNest.JSを使用してTypeScript Webアプリケーションを作成する方法に関する一連の記事です。最初の部分では、Nest.JSを使用して簡単なサーバーAPIを記述します。このシリーズの第2部では、Angularを使用したフロントエンドアプリケーションを取り上げます。この記事で開発した最終的なコードは、このGitHubリポジトリにあります。


Nest.Jsとは何ですか？なぜAngularですか？


Nest.jsは、Node.jsウェブサーバーアプリケーションを作成するためのフレームワークです。


特徴的な機能は、他のフレームワークでは解決できない問題を解決することです：node.jsプロジェクトの構造。node.jsで開発したことがある場合は、1つのモジュールで多くのことができることがわかります（たとえば、Expressミドルウェアは認証から検証まですべてを行うことができます）、結局、サポートされない「混乱」につながる可能性があります。 。以下に示すように、nest.jsは、さまざまな問題に特化したクラスを提供することでこれを支援します。


Nest.jsはAngularに強く影響を受けています。例えば、どちらのプラットフォームもガードを使用して、アプリケーションの一部へのアクセスを許可または禁止します。どちらのプラットフォームも、これらのガードを実装するためのCanActivateインターフェイスを提供します。ただし、いくつかの類似した概念にもかかわらず、両方の構造は互いに独立していることに注意することが重要です。つまり、この記事では、他のフレームワーク（React、Vue.JSなど）で使用できるフロントエンド用の独立したAPIを作成します。

(Pipes) Nest.js , . API , , DTO , . , src/common/ validation.pipe.ts:


  import {
    ArgumentMetadata,
    BadRequestException,
    Injectable,
    PipeTransform,
  } from '@nestjs/common';
  import { validate } from 'class-validator';
  import { plainToClass } from 'class-transformer';

  @Injectable()
  export class ValidationPipe implements PipeTransform<any> {
    async transform(value, metadata: ArgumentMetadata) {
      const { metatype } = metadata;
      if (!metatype || !this.toValidate(metatype)) {
        return value;
      }
      const object = plainToClass(metatype, value);
      const errors = await validate(object);
      if (errors.length > 0) {
        throw new BadRequestException('Validation failed');
      }
      return value;
    }

    private toValidate(metatype): boolean {
      const types = [String, Boolean, Number, Array, Object];
      return !types.find(type => metatype === type);
    }
  }

    import { NestMiddleware } from '@nestjs/common';
    import * as jwt from 'express-jwt';
    import { expressJwtSecret } from 'jwks-rsa';

    export class AuthenticationMiddleware implements NestMiddleware {
      use(req, res, next) {
        jwt({
          secret: expressJwtSecret({
            cache: true,
            rateLimit: true,
            jwksRequestsPerMinute: 5,
            jwksUri: 'https://${DOMAIN}/.well-known/jwks.json',
          }),

          audience: 'http://localhost:3000',
          issuer: 'https://${DOMAIN}/',
          algorithm: 'RS256',
        })(req, res, err => {
          if (err) {
            const status = err.status || 500;
            const message =
              err.message || 'Sorry, we were unable to process your request.';
            return res.status(status).send({
              message,
            });
          }
          next();
        });
      };
    }
