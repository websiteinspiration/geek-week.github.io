サンセット-クラスノゴルスク機械工場（ソ連）で製造された小型の隠し監視カメラ。小さな光学機械工場として、1929年に作業を開始しました。ここでは、医療機関向けの顕微鏡と光学ガラスの製造に従事していました。その後、特別なサービスが必要な写真機器や光学システムを製造するための特別なワークショップが設立されました。Zakhodカメラは1989年にリリースされました。F-21AjaxおよびF-27 Neocyteカメラの後継と見なすことができ、主にソビエトKGB諜報機関によって使用され、少なくとも1992年まで製造されていました。



自動カメラの日没の寸法は96 x 65 x 29 mm（レンズを含む）、重量-230グラム以下で、カメラはF-27 Neocyteとほぼ同じサイズでした。このカメラは、幅16 mm、厚さ0.08〜0.1 mmのZP-80タイプの穴の開いていないフィルムを3〜7メートル離れた場所から撮影するためのものです。露出の自動インストールにより、さまざまな撮影条件で高品質のネガを得ることが可能になりました。



夕日は監視、監視のために特別に設計されました。それは、例えばバッグの内部、車室内、または家具の一部、コートのボタンの後ろ、または工作員の体に適応するなど、さまざまな物体に目立たないように隠すことができます。実際、Sunsetには、前作と同じタイプのカモフラージュが使用されていました。古い機械式カメラF-21 Ajaxと電子F-27 Neocyteです。カメラの最も「目を引く」部分を隠すために-レンズ、スライドシャッター付きの特別なノズルが使用され、コーナーブラケットに取り付けられました。撮影中、リモコンの大きな茶色のボタンを押すと、まずカメラのシャッターカーテンが開き、写真が撮られ、穴がふさがれた。これはすべて「ノイズとほこりなしで」非常に迅速に起こりました。そして実際にこのプロセスに気づくことは単に不可能でした。

これはソビエトのカメラではありません-これはソビエト連邦で作られたドイツのカメラです。



クラスノゴルスク機械工場（ソ連）だけがKGBシークレットサービスで使用するカメラの製造に携わっていたわけではありません。第二次世界大戦後、コンタックスカメラの製造に携わっていた工場は、ドイツからキエフに完全に輸出されました（専門家、スペアパーツ、付属品とともに）。アーセナル工場は、1764年に建てられた小さな窓と1.3 mの厚さの壁を備えた部屋であり、その橋頭、またはその最も古い部分になりました。ここではカメラの生産が続けられました。 40代の終わりに、カメラはその新しい名前「キエフ」を取得して成功裏に生産され、1947年に最初の2つのモデルが光を見ました。彼らはコンタックスIIとコンタックスIII（キエフIIとキエフIII）の構築を繰り返しましたが、彼らは盗用されず、合法的に認可されたコンタックスのコピーでした。キエフは高品質の製品であり、ステータスアイテムでした、ソビエト連邦の要人の助けを借りてのみそのようなことを得ることが可能でした。 1956年に同期接点が導入され、シリーズは「A」インデックスを受信し始め、新しいモデルKiev-2AおよびKiev-3Aが登場しました。 50年代の終わりには、アーセナル工場で毎月3.5千台のカメラが生産されました。キエフのいくつかのモデルを考えてみましょう。ミニチュアサイズのため、「スパイ」カメラに起因する可能性があります。

小型16 mmカメラKiev-30


キエフ-30カメラはミニチュア16mmカメラで、1974年にリリースされました。彼女は人気のあるVegaのキエフカメラ（ミニルタ16カメラのコピー）の後継者でした。



カメラは幅16 mmの無孔フィルムで撮影され、フレームサイズは17 x 13 mmでした。フィルムの長さは45または65 cmで、フレーム数はそれぞれ17または25でした。フィルムはプラスチックカセットに入っていました。カメラには、焦点距離23 mm、絞り値f / 3.5、5つの異なる絞り値（3.5-11）のIndustar-Mレンズが搭載されていました。これにより、すべての中間値と3つの速度でf3.5からf11までの絞り値を設定できました。シャッター。シャッターは金属シャッター付きのカーテンであり、3つのシャッタースピード（1 / 30、1 / 60、1 / 200）だけが反発されました。レンズの絞りは2枚の花びらで、四角い穴が開いています。露出計算機は、手動で露出パラメータを設定した天気記号の形で提示されました。露出は、体の右側にある2つのリブ付きホイールによって設定されました。



カメラの寸法は84 x 46 x 27 mm（閉じた状態）で、デバイスの重量は178グラム以下でした。写真を撮る前に、下の図に示すようにカメラをレイアウトする必要がありました。同時に、レンズが開き、フレームが移動し、フォーカスホイールとシャッターボタンが使用可能になりました。実際、身体は制御要素でした。フレームカウンターは、撮影されたフレームの数を示しました。撮影プロセス自体は次のようになりました：シャッターを開く、シャッター速度と絞りを設定する、照準と焦点を合わせる、シャッターを切る。



これで、ユーザーは左側にある小さなフレームビューファインダーを通して被写体を見ることができます。四角いシャッターボタンは、カメラの前端に近い右側にありました。シャッターボタンの左側には、フォーカスコントロールがあります（50 cmから無限遠まで）。

小型で焦点範囲が狭いため、このカメラは秘密の写真に適していました。 KGBなど、旧ソビエト連邦（USSR）の特別サービスで使用されていました。小さくて便利なカメラは、バックパッカーの間で広く認識されています。 1987年にキエフ30がキエフ30Mに置き換えられ、1990年にキエフ303の改良モデルが登場しました。もちろん、工場ではコンポーネントの交換とカメラの修理に1年間の保証が与えられ、「通常の操作、慎重な取り扱い、適切な保管」が条件でした。 1974年のキエフ-30カメラの小売価格は30ルーブルでした、そして1989年のキエフ-30Mは15ルーブルの費用がかかりました。映画「アドベンチャーズオブエレクトロニクス」では、映画の主人公ウリが、キエフ30カメラだけでエレクトロニクスとシロイジキンの「スパイ」写真を撮っています。



動作中のソビエトの16 mmキエフ30カメラ：変装のために、煙草パックを装って金属ケースに取り付けられました。エージェントがタバコをパックから引き出した瞬間に発砲が行われました。信頼性のために、本物のタバコがパッケージに挿入されました。



そのようなKGB用のスパイカメラの人気は高かった。そのため、特にデジタル写真時代のコレクターにとっては、Kiev-30、Kiev-303は、その「スパイ」の過去を直接示唆する珍しいデザインで生産され始めました。 Kiev-JPSは、John Player Special（JPS）ブランドの通常のタバコパックを装ったKiev-30カメラです。



この変装により​​、カメラは秘密の写真には見えなくなりました。すべて同じ寸法は102 x 55 x 28.5 mmで、非鉄金属製で重量は約230グラムです。カメラの端がタバコのパックのように見えるように変更されました。類似性のために、3つのタバコ（フィルター）がパックから少しはみ出しています。これらのタバコの2つは固定位置にあり、3番目（最長）は移動可能（出入り）です。



たばこは、フィルムの輸送メカニズムに関連付けられています。引っ張るとレンズが開き、フィルムが次のフレームを運び、写真を撮ることができます。もともとこのバージョンのカメラはKGB専用に作られていると想定されていましたが、これはおそらく、世間知らずの西欧のコレクターを騙している可能性が高いことが明らかになりました。



ユーザーによると、2つの主な理由により、Kiev-30画像の品質はそれほど高くありませんでした。フィルムはひどく傷がつきました。幅の狭いフィルムは当時は不足していたため、頻繁に延期されていました。強いしみが残る可能性があるため、フィルムは現像中によく洗浄する必要がありました。

キエフ35A


キエフ35Aカメラは、アーセナル工場（ウクライナ）で1985年から1991年にかけて製造されました。このカメラは、冷戦時に監視エージェントの愛情を勝ち取り、カラーミノターレンズを備えた1974年のMinox 35ELコピーでした。固定5レンズMS Corsair 2.8 / 35レンズを備えた小型35 mmカメラ。レンズの解像度：中心に沿って50lin / mm、縁に沿って28lin / mm。シャッター速度が最大1/1000秒の自動露出のカメラ。さらに、シャッター速度を手動で4〜1/500秒に設定できます。バッテリーまたは電圧6 Vのディスクエレメントのセットで動作しました。前壁は折りたたまれていました。壁がリクライニングされ、レンズが作動位置まで伸びるとすぐに、シャッターボタンはロックされなくなり、露出計がオンになりました。シャッターのコッキングとフィルムの巻き戻しは、トリガーの二重平行移動によって実行されました。
