anti-captcha.comに登録する


この特定のサイトを使用する必要はありませんが、それを基礎として使用できます。
名前が示すように、リソースはキャプチャを解決するためのソリューションを提供します。これがトートロジーです。有料。1000キャプチャあたり1ドル。初めてで十分でしょう。彼の作品の本質は単純です。サイト（またはそのサービス）に送信されたキャプチャは、画面の反対側にいる生きている人（またはかなり生きていない人）によって解決されます。このソリューションはほぼ瞬時に実行され、ニューラルネットワークと速度を競うことができます。精度はほぼ100％です。
私たちの場合、アルゴリズムは次のとおりです。画面からキャプチャの写真を撮り、apiを使用してサービスに写真を送信し、答えを受け取りました。したがって、数字、文字などで構成されるほとんどすべてのキャプチャを解決できます。

したがって、サイトに登録して1ドルを支払った後、apiセクションでキーを受け取る必要があります。

オブジェクトのキャプチャはすぐには画面に表示されないため、page_downを押してから、キャプチャの写真とディスクへの保存を押します。画面の解像度は人によって異なります。プログラムは1280x1024の画面用に書かれています。
写真を撮るキャプチャの境界を決定するときに画面上の座標の選択に悩まされないようにするために、画面上でのマウスの位置を決定するコードは残しておきます。
コード
#! python3
# mouseNow.py - Displays the mouse cursor's current position.
import pyautogui
print('Press Ctrl-C to quit.')
try:
    while True:
        # Get and print the mouse coordinates.
        x, y = pyautogui.position()
        positionStr = 'X: ' + str(x).rjust(4) + ' Y: ' + str(y).rjust(4)
        pixelColor = pyautogui.screenshot().getpixel((x, y))
        positionStr += ' RGB: (' + str(pixelColor[0]).rjust(3)
        positionStr += ', ' + str(pixelColor[1]).rjust(3)
        positionStr += ', ' + str(pixelColor[2]).rjust(3) + ')'
        print(positionStr, end='')
        print('\b' * len(positionStr), end='', flush=True)

except KeyboardInterrupt:
    print('\nDone.')
