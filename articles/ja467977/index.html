Vue.jsのStyled-Componentsを使用したアプリケーションの作成

こんにちは、ハブル！先日、ポルトガル語で非常に興味深い記事を見つけました。幸い、私たちはなんとかその英語版を見つけることができました。私はあなたの注目をロシア語への翻訳にもたらします。私の他の翻訳は、Habrの私のページにあります。

オリジナルへのリンク：ポルトガル語、英語

主題に含まれていない人のために。 Styled-Componentsは、React＆Ractネイティブの開発者の間で非常に人気のあるライブラリです。 JSで直接カスタムCSSを記述できます。

Vueでは、必要なすべてのコンポーネントが1か所に集められているため、単一ファイルコンポーネント（SFC）での作業がいかに便利かを理解しています。 SFCパターンはVueの人気を大幅に高めました。

この数か月間、Reactに関する主要なプロジェクトの開発に参加する機会がありました。その中でStyled-Componentsを使用しましたが、これは非常に興味深い体験でした。

ただし、ほとんどのプロジェクトではVueを使用しているので、当然のことながら、新しいエクスペリエンスをStyled-Componentsと組み合わせ、Vue.jsエコシステムの利点を活用したいと考えました。そのとき、そのようなソリューションがすでに存在し、React：vue-styled-componentsの同じライブラリとして同じ作成者によってサポートされていることがわかりました。

開始


空のおしゃべりを脇に置いて、コードに移ります。通常どおり、yarnまたはnpmを使用します。

// for yarn
yarn add vue-styled-components
// for npm
npm install --save vue-styled-components


私のgithubには、このライブラリの使用例を含むリポジトリがあります。

例で使用されている各要素は、Vueスタイルのコンポーネントによって、個別のプロパティを持つ独立したコンポーネントとして、または他のコンポーネントからデータを受信することによって処理できます。

最初の例は、デフォルトのスタイルを持つ単なるボタンです。

import styled from "vue-styled-components";

const CButton = styled.button`
  font-size: 1em;
  text-align: center;
  color: #FFFFFF;
  border-radius: 5px;
  padding: 10px 15px;
  background-color: #0057AA;
`;

export default CButton;


アプリケーションのどこでも、このコンポーネントを使用できます。

    import CButton from '@/components/elements/Button'
    export default {
        name: 'app',
        components: {
            CButton
        },
    }


パラメータを渡す


vue-styled-componentsでは、プロパティを通じてこれらの値を渡すことにより、コンポーネントのスタイルを動的に定義できます。この例は、メイン属性が渡されると、ボタンが新しい背景スタイルとフォント色を受け取ることを示しています。

import styled from "vue-styled-components";

const typeButton = { primary: Boolean };

const CButtonProps = styled('button', typeButton)`
  font-size: 1em;
  text-align: center;
  color: ${props => props.primary ? '#0057AA' : '#FFFFFF'};
  border-radius: 5px;
  padding: 10px 15px;
  background-color: ${props => props.primary ? '#FFFFFF' : '#0057AA'};
`;

export default CButtonProps;


上記の例は改善でき、新しいプロパティを追加することもできます（必要なだけ属性を渡すことができます）。すべてはプロジェクト内のニーズに応じて行われます。

このライブラリーの助けを借りて、ロジックを介してスタイルを制御する機会を得ます。以下の例では、他のユーザーに影響を与えることなく、ボタンに割り当てる配色を選択できるため、アプリケーションを多様化できます。

import styled from "vue-styled-components";

const typeButton = {
    type: String,
    radius: Boolean
};

const styleButton = type => {
    switch (type) {
        case "primary":
            return `
                background-color: #FFFFFF;
                color: #0057AA;
            `;
        case "error":
            return `
                background-color: #B4000B;
                color: #FDFDFD;
            `;
        case "success":
            return `
            background-color: #00C887;
            color: #37435F;
        `;
        default:
            return `
                background-color: #0057AA;
                color: #FFFFFF;
            `;
    }
}

const CButtonPropsCond = styled('button', typeButton)`
  font-size: 1em;
  text-align: center;
  padding: 10px 15px;
  border-radius: ${({ radius }) => radius ? "6px" : null};
  ${(props) => styleButton(props.type)}
`;

export default CButtonPropsCond;


このボタンの作成プロセスをよりよく説明するために、最初に（常に）vue-styled-componentsをインポートし、その直後（3行目から6行目）に、コンポーネントに渡されるプロパティのタイプを決定する必要があります。8行目から31行目では、typeプロパティの値を受け取る関数を作成し、渡されたものに従って、背景の属性を返します。それぞれのレジスタの色と色。プロパティとして何も渡されない場合、ボタンにはデフォルト値が割り当てられます。

この知識のおかげで、カスタムコンポーネントを非常に動的にする新しい属性を実装できます。

これらの小さな例をGitHubに残したので、このライブラリをマスターする最初のステップを実行できます。楽しめ。
