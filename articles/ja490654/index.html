MQシリーズガスセンサーに点を打つ-データシートとチューニングに関する深い理解


ペニーガス漏れ検知器を購入した後、棚にあるものすべてを分解し、内部で何が起こっているのかを知りたいという要望がありました。センサーに関する情報や記事はたくさんありますが、ほとんどは標準的な中国語モジュールのピン配列に制限されており、時には動作原理も制限されています。そこにある情報の絶対値の比較的正確な決定について。今後は、「オウム」を決定する正確な機能、温度と湿度の補正、いくつかの選択の可能性など、データシートからすべてを絞り込もうとするつもりです。

たとえば、可燃性ガスセンサー（主にメタン）MQ-4。つまり、センサー要素は、その化学的性質により、さまざまなガス濃度で抵抗を変化させ、ADCを通じて電圧値を取得する分圧器の抵抗として機能します。これらの特性は、センサーを加熱する必要がある要素の特定の温度で現れます。


上の図は、センサーと分圧回路の略図です。Hは加熱コイルです（33オーム-約150 mA、電流はコントローラー付きのクラフトには大きすぎます。電源回路を設計するときに考慮に入れる必要があります）。AVは、ガス濃度に応じて交換可能な抵抗を持つ敏感な要素の出力です。 、RLは、データシートで推奨されている2番目の分割抵抗-20 kOhmです。

ADCをppmに変換


ADC分圧器の出力では、センサーの抵抗Rs（分圧器RLの2番目の抵抗の値を知っている）を計算できる電圧値（Uadc）を取得します。センサーが送信しているものを特定します。

Rs=（Vref×RL）/Uadc-RL


Rsの値を使用すると、データシートのグラフに従ってガス濃度を決定できます。濃度の決定には、単純なRs / Ro比が使用されます。この場合のRoは、1000 ppmの検出ガス濃度でのセンサーエレメントの抵抗です。

int mq4_get_adc (void)
 {
 int mq4_adc_bits; 
 HAL_ADC_Start(&MQ4_ADC);
 HAL_ADC_PollForConversion(&MQ4_ADC,100);
 mq4_adc_bits = HAL_ADC_GetValue(&MQ4_ADC);
 HAL_ADC_Stop(&MQ4_ADC);
    return mq4_adc_bits;

{
float mq4_adc_volts;
int Rs;
int Ro;
int mq4_adc;
mq4_adc = mq4_get_adc(); 
mq4_adc_volts = (float)mq4_adc/((float)MQ4_ADC_PRECISION/MQ4_REFERENCE_VOLTAGE); 
//     
Rs = (((float)MQ4_REFERENCE_VOLTAGE * (float)MQ4_STATIC_RESISTOR)/mq4_adc_volts)- (float)MQ4_STATIC_RESISTOR;
//    1000 ppm (      RsRo 4.4  MQ-4):
Ro = Rs/MQ4_AIR_RSRO;
   return Ro;
 }
