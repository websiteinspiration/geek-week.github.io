人気のポータブルコンソールのサウンドアンプのリバースエンジニアリング-主な調査結果の説明

5月上旬に、Nintendo Game Boy Colorにインストールされたサウンドアンプをリバースエンジニアリングする作者であるHacker Newsに資料が公開されました。主なことについて話し合います。

以前、Sound Blaster 1.0とIBM PC- Innovation SSI-2001のマップについて話しました。


Photo Elias Castillo / Unsplash

Game Boy Colorは1998年にリリースされ、コンソールはすぐに人気を博しました。彼女のために、さまざまなジャンルの数百のゲームがリリースされました。恐怖がありました-たとえば、アローンインザダーク。

ゲームの雰囲気に浸ることで、ステレオヘッドフォンと内蔵モノスピーカーのサポートが可能になりました。3つのアンプを備えたマイクロチップが彼らの仕事を担当しました。プログラマーファン古い学校と芸術ケンShiriff（ケンShirriff）が開催されたチップのリバースエンジニアリングをし、それ仕組み作りました。このため、エンジニアはクリスタルの写真を使用しました。この写真には、いくつかの異常な技術的解決策が顕著です。

チップ構造


一般に、デバイスの集積回路は5つのコンポーネントに分けることができます。中央にはスピーカー用のアンプがあり、左側には大きな出力トランジスターがあります。それらは、周波数振動のパワーを動的ヘッドの動作値まで増加させます。アンプには、差動ペア、カレントミラー、その他のサービスIC も含まれています。



ヘッドフォンアンプは右上と右下にあります。それらの出力段は、並列に接続された10個のトランジスタで構成されています。システムは電流源と電流ミラーを共有します-これらのコンポーネントはアンプ間に設置されます。チップの下部には制御ロジックがあります。彼女はスピーカーとヘッドフォンのアンプ間でシステムを切り替えます。

トランジスタの機能


ケン・シリフは彼のTwitter で、ゲームボーイのサウンドアンプには約100個のNPNとPNPトランジスタが含まれていると述べました。これらは、エミッター、ベース、コレクターという交互に異なる導電率を持つ3つの半導体層で構成され、電流によって制御されます。これらのトランジスタの使用は、MOSトランジスタ（金属酸化膜半導体）が一般的なデジタル回路では一般的ではありません-バイポーラトランジスタとは異なり、電圧制御されています。

テーマ別のスレッドでハッカーニュースの住民の一つは言ったゲームボーイのサウンドアンプは、私たちの時代に開発されたならば、それはその基礎になるCMOS構造になること。

NPNおよびPNPトランジスタをベースにした任天堂のレトロチップも、デジタル回路では珍しいと思われるいくつかの回路を構築しています。しかし、それらはアナログICにとっては非常に一般的です。それらの1つは現在のミラーであり、現在のスタビライザーの役割を果たします。たとえば、図では、6つのPNPトランジスタで構成されるカレントミラーを見つけることができます。

回路内の最大のトランジスタはNPNトランジスタです。これらはIPの左上隅にあり、スピーカーの出力ステージの役割を果たします。

抵抗とコンデンサ


図の抵抗は、p型半導体（シリコン）で作られた長いジグザグストリップです。このアプローチにより、エンジニアはチップ上のスペースを節約できました。 また、チップ上には3つの大きなコンデンサーがあります（各アンプに1つ）。これらは、金属、シリコン、絶縁酸化膜の「サンドイッチ」のように見え、アンプのスタビライザーの役割を果たします。



その他のゲームボーイ愛好家プロジェクト


昨年、フランスのエンジニアがゲームボーイ用にSoCをリバースエンジニアリングしました。彼はAPU、PPU、I / Oシステムを作成し、GitHubで公開しました。合計で、彼は191時間かけてすべての化合物を検索し、図面を作成しました。


写真こんにちは、私はNikです / Unsplash

愛好家は、昔ながらのゲームコンソールのリバースエンジニアリングを行うだけでなく、エミュレータも開発しています。 GitHubでは、RustでMooneye GBを見つけることができます。エミュレーターは、オリジナルのニンテンドーコンソールとそれ以降のコンソールバージョン（たとえば、2000年代の初めにリリースされたGame Boy Player）の両方をサポートしています。

同じシリーズの別のプロジェクトはCinoopです。それ -C言語で記述されたオープンマルチプラットフォームエミュレーター（Windows、DS、GameCube、3DS、Linux、PSP、PS4で動作）。作品の例はYouTubeで見ることができます-著者はテトリスで再生します。
