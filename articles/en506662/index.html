Already a year since I changed work to a new one. In this article I want to share the experience gained in the past. Introduction


At a previous job, I was an operator of a CNC machine, making furniture facades from MDF. The work is varied: you sit at the computer and draw in a CAD / CAM program, then you stand to control the cutting, then you bring the workpieces and carry away the finished parts.


Before I came to the company, there was a rather meager catalog of drawings, which was clearly not enough for the clients, so they kept asking me to do it like in the photo from the Internet. I painted all the specific drawings by hand, the standard ones were quite simple, and since I was interested in programming, I decided to generate their G-code for the machine directly without the participation of a CAD / CAM program.

from math import sqrt


def b3(x0, y0, x1, y1, x2, y2, x3, y3, d):
    px = (x3 - x0) / 3
    py = (y3 - y0) / 3
    mx1 = x1 - x0 - px
    my1 = y1 - y0 - py
    mx2 = x2 - x3 + px
    my2 = y2 - y3 + py
    d1 = sqrt(mx1 ** 2 + my1 ** 2)
    d2 = sqrt(mx2 ** 2 + my2 ** 2)
    if d1 < d and d2 < d:
        print(x3, y3)
    else:
        x01 = (x0 + x1) / 2
        y01 = (y0 + y1) / 2
        x12 = (x1 + x2) / 2
        y12 = (y1 + y2) / 2
        x23 = (x2 + x3) / 2
        y23 = (y2 + y3) / 2
        x012 = (x01 + x12) / 2
        y012 = (y01 + y12) / 2
        x123 = (x12 + x23) / 2
        y123 = (y12 + y23) / 2
        x0123 = (x012 + x123) / 2
        y0123 = (y012 + y123) / 2
        b3(x0, y0, x01, y01, x012, y012, x0123, y0123, d)
        b3(x0123, y0123, x123, y123, x23, y23, x3, y3, d)
