STM32CubeMonitor vale a pena tentar

Introdução


STM32CubeMonitor é um utilitário para exibição conveniente de dados da família de controladores STM32. Além disso, nenhum equipamento adicional é necessário, apenas uma placa com um controlador e um programador (tentei tanto o apito chinês quanto o st-link \ v2 original). A palavra "Cubo" no nome não significa vinculação ao CubeMx \ HAL , você só precisa de firmware no formato "* .elf", o que é uma boa notícia. Esse utilitário permite que você produza dados de maneira extremamente rápida e fácil do controlador em tempo real na forma de gráfico, além de executar o pós-processamento (existem conjuntos de funções matemáticas e trigonométricas básicas).

Início do trabalho


Vale a pena garantir que o programador esteja conectado, o controlador ligado e o firmware carregado. Também é recomendável atualizar o software do programador para a versão mais recente.
