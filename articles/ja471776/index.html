Laravel：基本的な概念を説明してください。パート2：練習

みなさん、こんにちは！「フレームワークLaravel」コースの開始前夜に著作権に関する一連の出版物を継続します。前回の記事では、Laravelの理論的基礎について説明しました。ただし、どのフレームワークの理論も長期間学習することができ、実際に何かを書くまでは何も理解できません。



したがって、各ユーザーがログインして新しいアルバムを作成し、そこに写真をアップロードできるアプリケーションを作成してみましょう。MVCがまだわからない場合は、これを読んでください。すぐに実際に使用する必要があります。

この種のガイドは、環境のセットアップ方法などの説明から始まります。問題は、プラットフォームごとに異なるということですが、この記事を誰にとっても興味深いものにしたいと思っています。必要なものへのリンクを提供するだけで、自分でインストールして構成してみます。

アルバムテーブルを作成する

では、バックエンドはどこから始まるのでしょうか。データベースにテーブルを作成することから。ターミナルでこのコマンドを実行すると、データベースに移行する機会が作成されます。

php  artisan  make:migration  CreateAlbumsTable

したがって、テーブルに含める必要のある列を決定する必要があります。これは、最後の移行のために作成したファイルで行う必要があります。プロジェクト内のフォルダーにgallery/database/migrationsあり、その名前はその作成時刻に関連付けられています。次のように編集する必要があります。

<?php
use Illuminate\Database\Migrations\Migration;
use Illuminate\Database\Schema\Blueprint;
use Illuminate\Support\Facades\Schema;
class CreateAlbumsTable extends Migration
{
    /**
     * Run the migrations.
     *
     * @return void
     */
    public function up()
    {
        Schema::create('albums', function(Blueprint $table)
        {
          $table->increments('id')->unsigned();
          $table->string('name'); //  
          $table->text('description'); //   
          $table->string('cover_image'); //  
          $table->timestamps(); //  .   ,    
        });
      }
 
    /**
     * Reverse the migrations.
     *
     * @return void
     */

        Schema::dropIfExists('albums');
    }
}

    public function Photos(){

    }
}

画像表

うわー！これでアルバムモデルの準備が整いました。これで画像を作成できます。ここで、プロセスが再び繰り返されます。

php  artisan  make:migration  CreateImagesTable

マイグレーションを編集します。

<?php
use Illuminate\Support\Facades\Schema;
use Illuminate\Database\Schema\Blueprint;
use Illuminate\Database\Migrations\Migration;
 
class CreateImagesTable extends Migration
{
 
    public function up()
    {
        Schema::create('images', function (Blueprint $table) {
            $table->increments('id');
            $table->integer('album_id')->unsigned();
            $table->string('image');
            $table->string('description');
            $table->foreign('album_id')->references('id')->on('albums')->onDelete('CASCADE')->onUpdate('CASCADE');
            $table->timestamps();
        });
    }
 
    public function down()
    {
        Schema::dropIfExists('images');
    }
}

アルバム作成

アルバムで保存、表示、削除するには、コントローラーにいくつかの関数が必要です。ただし、最初にコントローラー自体が必要です。

php  artisan  make:controller  AlbumsController

私たちは入る gallery/app/Http/Controllers/AlbumsController.php

<?php
namespace App\Http\Controllers;
 
use Illuminate\Http\Request;
use Illuminate\Support\MessageBag;
use Validator;
use App\Album;
 
class AlbumsController extends Controller
{
    public function getList() //  
  {
      $albums = Album::with('Photos')->get(); 
      return view('index')->with('albums',$albums);
  }
 
  public function getAlbum($id)
  {
      $album = Album::with('Photos')->find($id);
      $albums = Album::with('Photos')->get();
      return view('album', ['album'=>$album, 'albums'=>$albums]);
  }
 
  public function getForm()
  {
      return view('createalbum'); //  
  }
 
  public function postCreate(Request $request)
  {
      $rules = ['name' => 'required', 'cover_image'=>'required|image'];  //       
      $input = ['name' => null];
    //     
      $validator = Validator::make($request->all(), $rules);
      if($validator->fails()){
        return redirect()->route('create_album_form')->withErrors($validator)->withInput();
      }
      $file = $request->file('cover_image');
      $random_name = str_random(8);
      $destinationPath = 'albums/';
      $extension = $file->getClientOriginalExtension();
      $filename=$random_name.'_cover.'.$extension;
      $uploadSuccess = $request->file('cover_image')->move($destinationPath, $filename);
      $album = Album::create(array(
        'name' => $request->get('name'),
        'description' => $request->get('description'),
        'cover_image' => $filename,
      ));
 
      return redirect()->route('show_album',['id'=>$album->id]);
  }
 
  public function getDelete($id) //   
  {
      $album = Album::find($id);
      $album->delete();
      return Redirect::route('index');
  }
}
