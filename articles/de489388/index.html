1. Implementieren Sie die Ressourcenverwaltungssicherheit in Azure


In nur 3-4 Stunden lernen Sie, wie Sie Ressourcen mit Richtlinien, rollenbasierter Zugriffssteuerung und anderen Azure-Diensten schützen.

Unter den Kursmodulen:

  • Sicherheit, Verantwortung und Vertrauen in Azure
  • Durchsetzen und Überwachen von Infrastrukturstandards mit Azure Policy
  • Überwachen und organisieren Sie Azure-Ressourcen mit Azure Resource Manager
  • Sichern von Azure-Ressourcen mit rollenbasierter Zugriffssteuerung (RBAC)

Hier können Sie mit dem Training beginnen

2. Identitäts- und Zugriffsverwaltung in Azure


Innerhalb von 5 bis 6 Stunden lernen Sie, wie Sie mit Abonnements, Benutzern und Gruppen arbeiten, indem Sie Microsoft Azure Active Directory für Workloads konfigurieren.

Kursmodule:

  • Sicherheit, Verantwortung und Vertrauen in Azure
  • Erstellen Sie ein Azure-Konto
  • Verwalten von Benutzern und Gruppen in Azure Active Directory
  • Erstellen Sie Azure-Benutzer und -Gruppen in Azure Active Directory
  • Sichern Ihrer Anwendung mit OpenID Connect und Azure AD
  • Schützen Sie Azure Active Directory-Benutzer mit der Multi-Faktor-Authentifizierung
  • Ermöglichen Sie Benutzern das Zurücksetzen ihres Kennworts mithilfe eines sich selbst zurücksetzenden Kennworts in Azure Active Directory

3. Implementieren der Netzwerksicherheit in Azure


Nach ungefähr 6 Stunden lernen Sie, wie Sie Netzwerke in Azure einrichten, schützen und isolieren. Weitere Informationen zur Sicherheit und zum Vertrauen in Azure, zum Einrichten eines Netzwerks für virtuelle Maschinen, zum Sichern und Isolieren des Zugriffs auf Azure-Ressourcen mithilfe von Netzwerksicherheitsgruppen und Dienstendpunkten, zur End-to-End-Verschlüsselung des Netzwerkverkehrs mithilfe des Azure Application Gateway sowie zur Überwachung und Fehlerbehebung komplexer Netzwerkinfrastrukturen Azure mit Netzwerküberwachungstools. Jetzt beitreten!
