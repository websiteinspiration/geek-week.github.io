适用于初学者和市场营销人员的39种英文缩写

市场营销及相关领域的专家几乎使用的所有术语都与英语紧密相关。因此，已经在培训专业阶段，您需要学习它们。

根据我们的经验，最难记住的缩写-消耗的能量几乎是它们的两倍。毕竟，您必须先学习解密，然后再学习翻译本身。因此，他们在一篇文章中收集了最常见的缩写词。

我们决定为该专业的初学者简化一些任务。我们希望这本词典对初学者，初级西奥什尼科夫和市场营销人员有用。

重要！我们知道许多缩写都以其原始形式使用英语。但是对于初学者来说，困难在于它们都很相似。因此，存在记错的风险。我们认为，学习它们的最好方法是首先了解成绩单和翻译，然后再深入研究其含义和细微差别。

AMA-询问我-分机。“问我任何事”

网站上的页面，用户可以匿名询问品牌或影响者问题。问答页面的选项之一（问题和答案）。AMA帖子也可以使用特殊工具和应用程序在社交网络上发起匿名民意测验。

B2B-企业对企业-分机 “企业与企业之间”

商业销售。客户和其他客户是其他企业，公司或组织时企业的工作类型。

B2C-企业对消费者-添加。“为消费者服务”

营销活动，其目的是向最终用户销售产品或服务。与B2B相比，B2C销售策略的特点是销售周期短，关于购买的情感决策以及买卖双方之间的亲密关系少。

CMS-内容管理系统-内容管理系统

基于脚本的软件，使您可以管理在线资源的内容，对其进行更改，查看和控制。当今最受欢迎的CMS：WordPress，Joomla，Drupal，MODx，1C Bitrix。

CPC-每次点击费用-每次点击费用

一种标准的广告模型，其中考虑对广告的每次点击以及到广告商网站或页面的过渡进行付款。

每次点击费用=（广告费用）：（点击次数）

示例。您已经在网站上放置了广告横幅，并获得了120次点击。您支付的广告费用：800卢布。应用公式，我们得到CPC = 6.6卢布。为过渡。

CPL-每潜在客户成本-接收潜在客户（潜在客户）的成本

一种标准的广告模型，其中考虑为吸引的每个潜在客户付款。潜在客户是对广告做出反应并留下其联系信息的潜在客户。CPL最常用于需要评估广告活动有效性的B2B公司。

CPL =（广告费用）：（完成的潜在客户数量）

示例。您在网站上放置了一条横幅，并收到了3个应用程序。您在广告上花费了1200卢布。应用该公式，我们得出CPL = 400卢布。为铅。

CPM-每英里成本-CPM

一种标准的广告模型，其中，每千次广告展示计入付款。使用此模型，您可以以相对较小的预算向尽可能多的人展示广告。

CPM =（广告费用）x 1000 ：（广告展示次数）

示例。您在该网站上做过广告，并且已展示2,000次。您在广告上花费了1,500卢布。应用该公式，我们得出CPM = 750卢布。每千次展示。

CR-转化率-转化率

一个系数，显示每1000位访问者带来了多少转化。它以百分比计算。

CR =（转化次数）：（访问者人数）x 100％

示例。1000位访客来到该站点。其中，剩下45项购买商品的申请，使用该公式，我们得出CR = 4.5％。

CRM-客户关系管理-客户关系管理

该系统在流程自动化的帮助下，有助于更有效地与买方进行对话，避免工作中的错误，从而向买方出售更多产品。CRM保留与每个客户的所有通信的完整记录。

在这里您可以收听电话，查看购买历史记录，通过模板创建文档，编写电子邮件或短信，设置任务。通常，这是想要系统地而不是可怕地进行销售的公司必不可少的工具。

CRO-转化率优化-转化优化

旨在提高网站转化率的一系列操作。也就是说，营销人员和SEO的主要功能之一。

通常，CRO包括改善购买界面，创建产品或服务的展示，分析和改善销售渠道，优化目标网页，A / B测试。

CTA-号召性用语-号召性用语

激发用户执行操作的呼叫（转到，填写表格，致电，购买，查找等）。

CTA元素可以是需要采取某些措施的按钮，图像，形状甚至文本。

免费注册或立即购买-经典CTA。

点击率-点击率-点击率

评估内容相关广告效果的最重要指标之一。毕竟，是他影响了Google Adwords和Yandex Direct中的点击价格。它以百分比计算。

点击率越高，每次点击费用越低。

点阅率=（广告点击次数）：（广告展示次数）x 100％

示例。您发布了一个有20,000个用户看到的广告。800人点击了横幅。应用该公式，我们得出的点击率= 4％。

CX-客户体验-客户体验（客户体验）

客户与公司互动时获得的印象集。

通常，“消费者体验”始于对公司的介绍。接下来是互动，订购，付款，使用产品或服务，品牌支持的开始。在每个阶段，客户都会与公司保持紧密联系，并从中汲取自己的独特经验。所有这些都是有效的品牌管理的重要元素。

DA-域名授权

Google的链接指标之一。此值显示如何在搜索引擎中对域（子域）进行排名。

该预测以对数范围内从1到100的点为单位进行度量。也就是说，将其性能提高到10到20的范围比从70到80容易得多。

DMCA-数字千年版权法-数字千年版权法

实际上，这是管辖互联网上版权保护的基本法律。这是全文。

DMCA索赔或DMCA删除通知是指索赔人针对涉嫌侵犯版权提出的投诉。基于其考虑的结果，将决定从索引中删除特定页面或站点。



FOMO-惧怕错过-惧怕失去利润（失利综合症）

一种营销策略，可利用用户担心错过有用或重要内容的恐惧。

实际上，许多广告消息和登录页面都基于此。FOMO工具包括按时间或数量限制要约，社会证明，捆绑的商品和售后，购买礼品，折扣，有限要约或使用意见领袖。

GA-Google Analytics（分析）

Google提供的免费服务可创建有关网站访问者的详细统计信息。任何营销人员和许多专家的阿尔法和欧米茄。

HITS-超链接诱导主题搜索

搜索排名算法之一。以及网页排名。

HITS首先确定最相关的页面以响应搜索查询。此外，这些页面被组合成所谓的根集。此外，此集合增加到与之关联的所有页面以及链接到它的某些页面。事实证明基本设置。

仅在这个狭窄的子图中进行排名，而不是在整个系统中进行排名。



IO-广告订单-广告订单

它包含有关计划的广告活动的信息：客户公司的名称，项目的开始和结束日期，广告材料的个别功能（颜色，大小或持续时间），CPM（每千次展示价格），广告促销的总费用，可能的折扣以及罚款和如果未能履行传播广告信息的义务，将受到制裁。

实际上，承包商在接受IO时签署了一份完整的合同。

LSI-潜在语义索引-潜在语义分析

这就是所谓的隐藏语义索引，即对站点进行索引的方法，其中搜索机器人会考虑文本的一般含义，而不是考虑关键字的饱和度，

即SEO查询“在莫斯科购买便宜的塑料窗”无法使用的算法。

NPS-净促销员评分-消费者忠诚度指数

试图确定一件事的复杂计划：消费者有多爱品牌。

它的基础是售后服务后公司经常问的一个问题。“您将我们的公司推荐给您的朋友或同事的可能性是多少？” 答案通常以1到10的比例来表示。

接下来，将答案的整体分为3组：评论家（从1到6回答），中立（7-8回答），品牌拥护者（9-10回答）。确定NPS的公式如下：

NPS =（（品牌拥护者数量）-（评论者数量））：（受访者数量）x 100％

例如，对100个人的调查显示，其中20个人为评论家，45为中立，35 -品牌拥护者。使用该公式，我们得出NPS = 15％。

PBN-私人博客网络-私人网站网络

个人（或公司）直接控制并积极使用这些站点来增加网站反向链接的配置文件。

指灰色SEO。实际上，Google早在2014年就禁止使用它们。但是跟踪其可用性并不容易，因此一些公司使用此促销工具。

但是，该策略存在风险，因为如果搜索引擎检测到PBN，则可能会受到制裁。

PPC-每次点击付费-每次点击付费

一种广告付款模式，其中，广告客户为广告的点击付费。

有时在俄语中您会找到缩写“ PPC”-这不是一个诅咒词，而是PPC的描图纸。

公关-PageRank

搜索排名算法。适用于通过超链接链接的页面，并为每个页面分配一个数值，以衡量其在其他文档中的“重要性”或“权威性”。

页面的链接越多，它就越重要。页面“ A”的权重取决于页面“ B”传输的链接的权重。这是发生这种情况的一个很好的例子：



QDD-查询值得多样性-分机。“要求值得各种各样”

系统用于模糊或模棱两可的问题的Google排名算法。也就是说，如果用户写“买笔”，则不清楚他在想哪一个：笔，门，窗户，支架或其他东西。

此算法的存在和功能不是100％清楚的。但是实践表明，在请求不确定的情况下，与标准算法不太吻合的项目将包括在内。

QDF-查询值得新鲜-添加。“要求值得新鲜”

Google排名算法，该算法不仅考虑页面的权重，还考虑其发布时间。新鲜内容的排名高于过时的内容。

该算法的特征也没有被研究。SEO专家只能通过分析搜索结果来声称它，其中新鲜材料的排名通常比旧材料高。

投资回报率-投资回报率-投资回报率

它显示特定投资的获利能力或亏损。几乎所有投资都必须考虑投资回报率，包括单个广告活动。
ROI =（响应广告的客户收入）：（广告费用）

示例。6月份的广告活动预算为10,000卢布。广告来自购买了40,000卢布的客户。使用公式，ROI =4。

有时该比率以百分比表示。也就是说，在上面的示例中，它将是400％。

SEM-搜索引擎营销-搜索引擎营销

一系列旨在通过使用付费广告来提高网站在搜索引擎中的位置的措施；

如果简化，则SEM = SEO +上下文广告。

SC-补充内容

这是任何可改善UX（用户体验或用户体验）的内容的名称。

它可能不适合页面或网站的目的，但应与页面的含义直接相关。

辅助内容作为一个整体可以称为不构成访客到潜在顾客转化的任何元素。但通常这些是本机照片和视频，社交网络中插入的帖子，标题，导航菜单，评论栏，推荐材料栏。

SEA-搜索引擎广告-内容相关广告

是的，此缩写隐藏了出现在搜索引擎结果页面上的广告。在Google和Yandex中，通常是页面顶部和底部的几个位置，标有“广告”或“广告”图标。

这里，例如，响应于“购买平板电脑”的请求，一个4（！）个位置的块。



SEO-搜索引擎优化-搜索引擎优化

这是增加针对目标搜索查询的搜索引擎中站点可见性的一组措施。

简而言之，SEO包括：搜索引擎算法分析，需求分析，编译语义核心，外部和内部网站优化，改善行为因素，分析结果，以及一些您无法简要描述的任务。

SERM-搜索引擎信誉管理-搜索引擎信誉管理

这是一组工具，旨在在潜在客户的眼中建立良好的组织形象。通常，这是两个主要任务：直接在品牌或产品（服务）的正面形象上进行工作，以及处理网络上已经存在的负面形象。

SERM主要通过与论坛，社交网络，评论一起发布图像和新闻内容来进行。

SERP-搜索引擎结果页面-搜索引擎页面

实际上，这是在将查询输入搜索栏后搜索引擎给出的页面。在俄语地区，“搜索结果”一词更为常见，然后在英语指南和文章中，最常见的就是SERP。

SMM-社交媒体营销

这是在社交网络上推广被市场营销视为社交媒体的商品和服务的总称。

该定义太大了，无法简要描述，所以让我们集中讨论SMM解决的任务：

  • 公关
  • 品牌推广
  • 贸易营销
  • 社交和互联网对营销活动的支持
  • 批量广告
  • 产品，产品或品牌的社会宣传
  • 增加公司资源的参与
  • 与非竞争性公司和品牌的交叉促销。


以及更多。实际上，这是一个专注于社交网络的成熟营销行业。

SMO-社交媒体优化-社交网络的资源优化

这些事件旨在改变站点的内容，使其在网络社区中最简单的使用。

通常，以下步骤是SMO的一部分：
  • 内容符合常规社交媒体访问者的期望和偏好。
  • 网站的转换方式使其符合社交网络的技术机制（发表评论等功能的能力）。
  • 资源的可用性得到改善。
  • 创建渠道以轻松导出内容。这样任何出版物都可以在社交网络中扩展。


服务条款-服务条款-使用条款

这是要约形式的正式文件，其中详细说明了使用资源的规则。实际上，这是管理任何用户操作的主要文档。

借助它，您可以将商品或服务购买者的关系转移到法律平面。如果有争议，这将有所帮助，因为即使只是简单地访问该网站也将自动意味着用户同意使用条款。

UI / UX-用户界面/用户体验-用户界面/用户体验

UI和UX是不能混合在一起的设计区域，但是不能彼此独立地存在。出色的产品始于UX，其次是UI。

用户体验是识别用户问题并解决这些问题以提高其满意度的过程。为了方便用户使用该网站-任何想法或愿望清单都会立即反映在功能中。

用户界面是设备与人之间的中介。这不仅是外观，而且是产品的交互性。就是说，这样的便利看起来也很漂亮，并取悦眼睛。

用于创建网站的最受欢迎的CMS之一。

ZMOT-真相零时刻-添加。“真相的零时刻”

用户拿起笔记本电脑，移动电话或其他有线设备并开始研究他想尝试或购买的产品或服务（或潜在合作伙伴）的那一刻。

也就是说，这是检测到需求的时刻。卖方的任务是适应ZMOT和特定产品最终选择之后的一段时间。这是获得新客户的策略之一。任何工具都适用于此。
