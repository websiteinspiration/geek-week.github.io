10の便利なVisual Studioコードプラグイン

経験豊富な開発者から初心者のジョーンズまで、誰もが自分たちの生活を簡素化したいと考えています。これには適切なツールが役立ちます。

Visual Studio Codeを開発環境として使用している場合は、運がいいです。たくさんのプラグインをインストールして、人生をできるだけ簡単にすることができます。

プラグインは生産性と仕事へのアプローチに大きな影響を与えます。これについては、以下で説明します。

「最初にツールを変更し、次にツールを変更します」
-Jeff Bezos

すべてのプラグインは無料で、Visual Studio Marketplaceで入手できます。

Visual Studio Intellicode




これは、Visual Studio用に最も頻繁にダウンロードされるプラグインです（3,200,000以上のダウンロード）。そして、私の意見では、最も有用です。

このプラグインは、開発者がコードを記述できるように設計されています。膨大な数のプログラミング言語をサポートしています。Visual Studio Intellicodeは機械学習手法を使用します。彼は、多数のオープンソースGitHubプロジェクトで使用されているパターンを検索および分析し、コードを記述するときにそれらを提供します。



この記事は、アプリケーションやサイトを開発し、スタートアップにも投資している EDISON Softwareのサポートを得て翻訳されました。


Gitのせい




誰がやったんですか？！

時々、特定のコードを書いた人を知る必要があります。Git Blameがお手伝いします。Git Blameは、ファイルの各行がいつ、誰によって最後に編集されたかを通知します。

これは、特に関数ブランチを操作するときに役立つ情報です。Git Blameは、コードの行が変更されたコミット（つまり、ブランチ）で通知するため、この変更の原因となったチケットがわかります。これは、変更の理由をよりよく理解するのに役立ちます。

きれい


Prettierは、開発時に明確に定義された一連のルールに従う必要がある開発者にとって最高のプラグインの1つです。これは、Prettierパッケージを使用できるようにする優れたプラグインです。Prettierは、コードを構造化するための優れたツールです。

PrettierはJavaScript、TypeScript、HTML、CSS、Markdown、GraphQLおよびその他の最新のツールと互換性があり、コードを正しくフォーマットできます。

JavaScript（ES6）コードスニペット


少なくともトピックの少しになろうとするすべての開発者は、おそらくJavaScriptスタックの1つで作業しました。どのフレームワークを好むかは問題ではありません。さまざまなプロジェクトで共通のコードを使用すると、作業が簡単になります。

JavaScript（ES6）コードスニペットは、怠惰な開発者にJavaScriptコードの非常に便利なスニペットを提供する便利なプラグインです。標準の関数呼び出しとホットキーを関連付けます。それを理解すると、生産性が大幅に向上します。

サス


ご想像のとおり、このプラグインは、スタイルシートを使用する開発者に役立ちます。アプリケーションのスタイルシートの作成を開始したら、必ずこのプラグインを使用する必要があります。Sassは構文の強調表示、オートコンプリート、およびフォーマットをサポートしています。

パスインテリセンス


パスインテリセンスは、コードを記述しながら生産性を向上させる最も確実な方法です。同時に多くのプロジェクトで作業し、多すぎるテクノロジを使用している場合は、ファイルパス名を記憶できる便利なツールが必要になるでしょう。このプラグインを使用すると、多くの時間を節約できます。そうしないと、適切なディレクトリを検索するのに無駄になってしまいます。
Path Intellisenceは、オートコンプリートファイル名の単純な拡張子として考えられました。しかし、それ以来、彼はほとんどの開発者にとって貴重なツールキットになりました。

Chrome用デバッガ




JavaScriptをデバッグする必要がある場合は、Visual Studio Codeを離れる必要はありません。MicrosoftのDebugger for Chromeを使用すると、Visual Studio Codeでソースファイルを直接デバッグできます

ESLint


ESLintプラグインは、Visual Studio CodeにESLintを埋め込みます。ESLintは、コードを静的に分析して問題をすばやく見つけるツールです。
ESLintによって検出されたほとんどの問題は自動的に修正できます。ESLintの修正では構文が考慮されるため、従来の検索および置換アルゴリズムによって発生するエラーは発生しません。さらに、ESLintは高度にカスタマイズ可能です。

SVGビューア




SVG Viewer拡張機能は、Visual Studio CodeでSVGを操作するためのユーティリティをいくつか追加します。このプラグインを使用すると、エディターを終了せずにSVGファイルを表示し、その外観を表示できます。さらに、このプラグインを使用すると、ファイルをPNGに変換し、URIデータスキームを作成できます。

テーマ


トピックはリストの最後のトピックですが、重要ではありません。あなたが毎日あなたのエディタを見るとき、それを可能な限り美しくしてみませんか？サイドバーの配色とアイコンを変更する多くのカスタマイズプラグインがあります。いくつかの人気のあるテーマは自由のために用意されています。ワンMonokai、ワンダークプロ、およびマテリアルアイコン。
