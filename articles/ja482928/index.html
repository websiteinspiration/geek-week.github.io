Predator Vision：Thermal Vision Effect


最高のサーマルシェーダー実装を検索しているときに、bgolusがUnityフォーラムに投稿したマットキャップシェーダーに出会いました。これは良い出発点です。

キャップを頂点と法線に結び付けて、モデルをさまざまな角度からより一貫して見えるようにするために、それを大幅に変更しました。つまり、実際、これはもはやマットキャップではありませんが、このシェーダーに敬意を表したいと思います。


順番に移動します。最初に頂点関数です。

1.オブジェクトの法線：

float3 v.normal;

2.頂点の正規化された位置。モデル参照点から始まります

float3 vertexN = normalize(v.vertex);

3.これらの値のベクトル積：

float3 viewCross = cross(vertexN, v.normal);

4. crossproduct.xyとバイアスを半分にして、より均一な結果を作成します

o.cap = viewCross.xy * 0.5 + 0.5;

フラグメント関数で

5.マスクを作成して乗算

fixed4 m = tex2D(_Mask, i.uv);
i.cap *= (1 - m.g);

6.カラーテクスチャを結果に投影します。（私はオレンジバイオレットスタイルを使用していますが、より大きなグラデーションや他のものを使用できます）


fixed4 color = tex2D(_Matcap, i.cap);

これで、サーマルシェーダーの準備は完了です。重要な点は、置換スクリプトで正しく設定されるように、独自のラベルを付ける必要があることです。そのため、ラベルに次のようにしてください。

 Tags{  "ThermalVision" = "ThermalColors"}

これが完全なシェーダーです： Pastebinへのリンク

シェーダーの交換


Thermalシェーダーが別のシェーダーを置き換えることができるようにするには、同じラベルを、置き換える必要のあるシェーダーに適用するだけです。

Tags{  "ThermalVision" = "ThermalColors"}

また、交換可能なシェーダーで必要なプロパティがすべて開いていることを確認してください。

_Mask("G = no thermal", 2D) = "black" {}
_Matcap("Matcap", 2D) = "white" {}
_Strength("Mask Strength", Range(0,2)) = 0.4

シェーダーを切り替えるには、2番目のカメラを追加して呼び出す必要があります

camera.SetReplacementShader(Shader, "NamedShader");
。


したがって、新しいカメラを作成し、メインカメラに対して子にし、位置と回転をリセットして、同じように配置されるようにします。


単色と黒の背景に設定します。


「カメラプレビュー」では、通常のカメラと同じようにすべてが表示されます。スカイボックスがなく、背景は黒です。

シェーダーを変更します。

パブリック変数Shaderを追加し、それに熱効果シェーダーをドラッグします。

次に、カメラを新しい変数として追加SetReplacementShaderし、パブリックシェーダーとラベルで指定された名前を使用してStart呼び出しで

  public Shader ThermalShader;
  Camera cam;
   void Start()
{
 cam = GetComponent<Camera>();
 cam.SetReplacementShader(ThermalShader, "ThermalVision");
  }


開始後、シェーダーが置き換えられたメッシュでのみ熱効果が表示されます。
