私たちは、ニューヨーク大学やカリフォルニア大学の研究者と協力して、基本的なアカウントの衛生状態がハイジャックされるのをどのように効果的に防ぐかを調べました。大規模で標的を絞った攻撃

に関する1年間の研究が水曜日にThe Web Conferenceと呼ばれる専門家、政治家、およびユーザーの会議で発表されました。 Googleの調査によると、Googleアカウントに電話番号を追加するだけで、自動ボットによる攻撃の最大100％、フィッシング攻撃の99％、および調査中に発生した標的型攻撃の66％までをブロックできることがわかりました。


Googleの自動プロアクティブアカウントハイジャック


自動プロアクティブ保護を実装して、すべてのユーザーをアカウントからのハッキングからより適切に保護します。仕組みは次のとおりです。たとえば、新しい場所やデバイスからの不審なログイン試行が見つかった場合は、それが本人であることを示す追加の証拠を求めます。この確認は、信頼できる電話にアクセスできることを制御したり、自分だけが正しい答えを知っている質問への答えであったりします。

スマートフォンにログインした場合、またはアカウント設定で電話番号を入力した場合、2段階認証と同じレベルの保護を提供できます。再設定用の電話番号に送信されたSMSコードが、自動ボットの100％、大量のフィッシング攻撃の96％、および標的型攻撃の76％のブロックに役立つことがわかりました。また、SMSのより安全な代替手段である、操作の確認を要求するデバイスへの要求は、100％自動ボット、99％のフィッシング攻撃、および90％の標的型攻撃の防止に役立ちました。



特定のデバイスの所有と特定の事実の知識に基づく保護は自動ボットへの抵抗に役立ち、特定のデバイスの所有に基づく保護はフィッシングや標的型攻撃さえ防ぐのに役立ちます。

アカウントに電話番号が設定されていない場合、アカウントに最後にログインした場所など、ユーザーに関する知識に基づいて、より弱い保護方法を使用する場合があります。これはボットに対してはうまく機能しますが、フィッシングに対する保護のレベルは10％に低下する可能性があり、標的型攻撃に対する保護は事実上ありません。これは、フィッシングページや悪意のある攻撃者が、Googleが確認を要求する可能性のある追加情報の開示を強制できるためです。

そのような保護の利点を考えると、ログインのたびにそれを使用する必要がない理由を尋ねる人がいるかもしれません。答えは、これはユーザーに追加の困難をもたらすであろうということです（特に訓練されていないものにとって-およそ。。）そしてアカウントロックアウトのリスクを増大させます。実験中、ユーザーの38％がアカウントの入り口で自分の電話にアクセスできないことがわかりました。さらに34％のユーザーは、予備のメールアドレスを思い出せませんでした。

スマートフォンにアクセスできなくなった場合やサインインできない場合は、以前にログインしてアカウントにアクセスした信頼できるデバイスにいつでも戻ることができます。

ハッキング採用攻撃について


ほとんどの自動防御がほとんどのボットとフィッシング攻撃をブロックするところ、標的型攻撃はより有害になります。ハッキングの脅威を監視するための継続的な取り組みの一環として、平均750ドルでハッキングするアカウントを要求する新しい「ハッキング」犯罪グループを常に特定しています。これらの攻撃者は、家族、同僚、政府関係者、さらにはGoogleになりすますフィッシングメールを利用することがよくあります。ターゲットがフィッシングの最初の試みをあきらめない場合、その後の攻撃は1か月以上続きます。


正しいパスワードをリアルタイムでチェックする中間者フィッシング攻撃の例。その後、フィッシングページは被害者にSMS認証コードを入力して被害者のアカウントにアクセスするように誘います。

私たちの推定によると、100万人に1人のユーザーだけがそのような高いリスクにさらされています。攻撃者はランダムな人々を対象とはしていません。調査によると、自動保護は調査した標的型攻撃の最大66％を遅延および防止するのに役立ちますが、リスクの高いユーザーは補完的な保護プログラムに登録することをお勧めします。調査中に指摘したように、セキュリティキーのみを使用するユーザー（つまり、ユーザーに送信されるコードを使用した2段階認証-約 ペレフ。）、標的型フィッシングの被害者となりました。

Googleの調査が示すように、Googleアカウントを保護するためにできる最も簡単なことの1つは、電話番号を設定することです。ジャーナリスト、コミュニティ活動家、ビジネスリーダー、政治キャンペーンチームなどのリスクの高いユーザーに対しては、高度な保護プログラムが最高レベルのセキュリティを保証します。Chrome Password Checkup拡張機能をインストールすることで、サードパーティのサービスアカウント（Googleではなく）をパスワードクラッキングから保護することもできます。
