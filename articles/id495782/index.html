Mozilla COVID-19




Pada 31 Maret, Mozilla mengumumkan di blog-nya tentang penciptaan Dana Solusi COVID-19 sebagai bagian dari Program Dukungan Sumber Terbuka Mozilla (program dukungan sumber terbuka Mozilla, MOSS). Melalui dana ini, perusahaan bermaksud untuk mengalokasikan dana hingga $ 50.000 untuk Open Source untuk proyek-proyek teknologi yang berjuang dengan pandemi COVID-19. Dana tersebut akan menerima aplikasi dari pengembang besi (seperti ventilator) dan perangkat lunak (seperti plugin browser yang melawan berita palsu).

Rambox - instant messenger




Ini FOSS berbicara tentang Rambox - ini adalah salah satu cara terbaik untuk mengelola beberapa layanan untuk komunikasi melalui satu aplikasi yang diinstal. Anda dapat menggunakan beberapa layanan perpesanan seperti Facebook Messenger, Gmail, AOL, Discord, Google Duo, obrolan Viber dan banyak lagi dari satu antarmuka. Dengan demikian, Anda tidak perlu menginstal aplikasi terpisah atau terus-menerus membukanya di browser. Rambox menawarkan versi gratis untuk komunitas sumber terbuka. Versi profesional berbayar memberi Anda akses ke 600+ aplikasi, sementara komunitas memiliki lebih dari 99 aplikasi. Versi Pro memiliki fitur tambahan seperti tema, hibernasi, unit iklan, pemeriksaan ejaan, dan dukungan lanjutan. Namun, menurut publikasi, versi open source sangat bermanfaat dan,dan Anda bahkan mungkin tidak memerlukan fitur profesional ini.

OpenMediaVault 5 Distribusi Penyimpanan Jaringan Tersedia




Setelah hampir dua tahun sejak pembentukan cabang penting terakhir, kit distribusi OpenMediaVault 5 diluncurkan, yang ditujukan untuk penyebaran cepat penyimpanan jaringan (NAS, Network-Attached Storage). OpenMediaVault mempertimbangkan pengembangan dukungan untuk perangkat yang disematkan dan pembuatan sistem yang fleksibel untuk menginstal add-on sebagai prioritas pengembangan. Antarmuka manajemen berbasis web OpenMediaVault ditulis dalam PHP menggunakan kerangka kerja ExtJS JavaScript. Di antara layanan yang terkait dengan organisasi pekerjaan repositori layanan dasar, dapat dicatat: SSH / SFTP, FTP, SMB / CIFS, klien DAAP, RSync, klien BitTorrent, NFS dan TFTP. Anda dapat menggunakan EXT3, EXT4, XFS, dan JFS sebagai sistem file.
