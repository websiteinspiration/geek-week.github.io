Menginstal Debian di Netgear Stora

Suatu hari saya memiliki keajaiban ini di tangan saya: netgear ms 2000 - stora . Saya memutuskan untuk segera meninggalkan penggunaan OS yang tertanam dan menginstal debian pada hard drive.

Informasi pada jaringan agak tersebar, tautan sudah lama mati, jadi saya memutuskan untuk memperbarui proses instalasi debian di stora. Siapa peduli, selamat datang ke kucing.

Artikel ini adalah sumber utama .

Pertama, kita perlu gambar untuk menginstal sistem: Saya membawanya ke sini . Unduh kedua file. File-file ini ditulis ke root pada flash drive yang diformat dalam FAT32.
Konverter USB ke UART PL2303TA juga diperlukan.

Saya punya ini


Anda juga akan memerlukan perangkat lunak untuk menghubungkan ke sepotong besi, misalnya, hiperterminal atau dempul (saya tidak mendapatkannya dengan dempul: sepanjang waktu saya menjadi gila, jadi saya menggunakan hyperterminal.

Untuk menghubungkan sepotong besi dengan kabel, saya perlu membukanya dulu. Prosesnya sederhana, jadi saya tidak akan menjelaskannya. Nah, Anda jangan lupa untuk memasukkan hard drive ke slot pertama dari sisi, di mana instalasi akan berlangsung.

Setelah perangkat keras diurai, kita menghubungkan adaptor. Perhatian, jangan menghubungkan kabel merah, yaitu, Anda hanya perlu menghubungkan 3 kabel (dari baterai: hitam: , hijau, putih).
Jadi, kabel terhubung, driver terhubung. Pada driver port com, kami menetapkan parameter: kecepatan 115200, jumlah bit 8, bit berhenti 1, tanpa paritas. Setelah itu, nyalakan sepotong besi dan hubungkan ke terminal. Saat Anda melihat Tekan tombol apa saja ... tekan tombol apa saja untuk masuk ke pemuat u-boot.

Penyimpangan kecil.

Daftar perintah yang akan kami operasikan dan yang akan berguna:
usb reset, ide reset — usb, ide
fatls, ext2ls — fat ext2 .
setenv —

printenv —

Setelah memasuki bootloader, atur parameter jaringan, inisialisasi perangkat usb, periksa apakah ada file yang diperlukan pada flash drive, simpan parameter ini ke memori perangkat dan reboot:

Tim 
usb reset
fatls usb 0
setenv mainlineLinux yes
setenv arcNumber 2743
setenv ipaddr your_IP
setenv gatewayip your_GW_IP
setenv dnsip your_DNS_IP
saveenv
reset

Setelah reboot, masukkan perintah untuk memulai instalasi debian:

usb reset
fatload usb 0 0x200000 uImage
fatload usb 0 0x800000 uInitrd
setenv bootargs console=ttyS0,115200n8 base-installer/initramfs-tools/driver-policy=most
bootm 0x200000 0x800000

Setelah itu, instalasi standar debian dalam mode teks akan masuk. Kami menginstal sistem, reboot setelah instalasi, masukkan uboot dan masukkan perintah untuk mem-boot perangkat dari hard drive:

setenv bootcmd_ide 'ide reset; ext2load ide 0 0x200000 /uImage; ext2load ide 0 0x800000 /uInitrd'
setenv bootcmd 'setenv bootargs $(console) root=/dev/sda2; run bootcmd_ide; bootm 0x200000 0x800000'
saveenv
reset

Setelah reboot, debian diambil dari hard drive, yang awalnya dicari.

PS Memulihkan bootloader asli: 
setenv mainlineLinux=no
setenv arcNumber
setenv bootcmd_ide
setenv bootcmd 'nand read.e 0x800000 0x100000 0x300000; setenv bootargs $(console) $(bootargs_root); bootm 0x800000'
