予期しないHTTPヘッダー

数日前、Creditkarmaブログを選んでいて、このHTTPヘッダーに気づきました

X-hacker: If you're reading this, you should visit wpvip.com/careers and apply to join the fun, mention this header.

（X-hacker：これを読んでいる場合は、wpvip.com / careersにアクセスして、このヘッダーに言及して参加を申請してください）。

私の最初の考えは次のとおりです。「うわー、プログラマーがミレニアムバグを導入して日付を少し節約すると、会社はHTTPヘッダーでの作業に関するオファー全体を公開します！」

私はとても興味を持ち、いくつかの研究をしました。

AutomatticがホストするWordPressホスティング会社であるWordPress VIPで サイトをホストしている場合、この特定のタイトルは「標準」のようです。同じタイトルが、次のような多くの有名なサイトで見つかります。

  • nypost.com
  • techcrunch.com
  • www.nbcolympics.com
  • www.thesun.co.uk
  • 数千以上

ウェブサイトの開発者とウェブサイトの所有者はそれを無効にすることができますが、正直に言うと、このヘッダーがすべてのHTTPパケットに含まれていることさえ知らないのです。そしてもちろん、2つ目の考えはすぐに思いつきました。他の企業にクリエイティブな見出しがあるかどうかを確認することです。

結果は素晴らしいです！

HTTPヘッダーには、さまざまな求人情報があります。


はい！世界で最もクールな企業は、「標準」HTTPヘッダーで求人を公開しているようですx-recruiting。

次に例を示します

。PayPal.me

x-recruiting: If you are reading this, maybe you should be working at PayPal instead! Check out www.paypal.com/us/webapps/mpp/paypal-jobs

Booking.com

x-recruiting: Like HTTP headers? Come write ours: careers.booking.com

Etsy.com

x-recruiting: Is code your craft? www.etsy.com/careers

Otto.de

x-recruiting: Seems you like http headers. To write ours, apply at job.otto.de and mention this header.

完全なリストを確認したいですか？これを行うために、GitHubでリポジトリを開きました。

求人に加えて、よりクリエイティブなものが研究プロセスで見つかりました。彼らは本当に神秘的なナンセンスの大ファンとして私を興奮させました。

神秘的なHTTPヘッダー


captchaプロバイダーのように見える9kw.eu Webサイトに秘密のメッセージ「42」が表示されます

X-Secret-Message: 42

。istreetview.comはサポートされなくなりましたが、秘密のWebフォームのアドレスはヘッダーにあります。

X-Secret-URL: https://appio.link/secret

私は彼らに私の答えを書きました...

thetradersdomain.comの見出しには「隠されたソース」がありますが、これは機密です：

x-secret-sauce: Confidential

images-dnxlive.comのHTTPヘッダーの1つにさらにいくつかの「秘密の」リンクがあります：

X-Secret-Message: camscv.dnxnetwork.lu

高級車が好きなら、ここにジャガーがあります。ボットを検出するためのヘッダー付きのro：

X-Bot: false

しかし、この方法はうまく機能せず、ユーザーエージェント（申し訳ありませんが、ジャガーの人）を置き換えると失敗します。

それでも... ニックネームの付いたサーバーを見たことがありますか？ここにはカップルがあります：

X-men.com

X-ServerNickName: clint

Howgoodisyourseo.com

X-ServerNickName: The Internet

そして最後ですが、興味深い例でもあります。m.bidorbuy.co.keの友人は、HTTPヘッダーに情熱を示しました。

x-powered-by: Passion and tiny cute kittens
x-servernickname: The Beast
x-hacker: If you are reading this, maybe you should be working at bidorbuy instead
