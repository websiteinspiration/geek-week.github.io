OntologyネットワークでスマートPythonコントラクトを作成する方法。パート3：ランタイムAPI



これは、Ontologyブロックチェーンネットワーク上のPythonでスマートコントラクトを作成する一連のチュートリアルのパート3です。以前の記事で会った

  1. ブロックチェーンとブロックAPI
  2. APIのストレージ。

OntologyネットワークでPythonを使用してスマートコントラクトを開発するときに適切な永続ストレージAPIを呼び出す方法のアイデアがわかったので、Runtime API（Contract Execution API）の使用方法を学びましょう。ランタイムAPIには8つの関連APIがあり、コントラクトを実行するための一般的なインターフェイスを提供し、開発者がデータを受信、変換、および検証するのに役立ちます。

以下は8 APIデータの簡単な説明です：8 APIデータの



使用方法を詳しく見てみましょう。その前に、Ontology SmartXスマートコントラクト開発ツールで新しいコントラクトを作成し、以下の手順に従ってください。

ランタイムAPIの使用方法


ランタイムAPI をインポートするには、ontology.interop.System.Runtimeとontology.interop.Ontology.Runtimeの 2つの方法があります。オントロジーパスには、最近追加されたAPIが含まれています。以下の行はAPIデータをインポートします。

from ontology.interop.System.Runtime import GetTime, CheckWitness, Log, Notify, Serialize, Deserialize
from ontology.interop.Ontology.Runtime import Base58ToAddress, AddressToBase58, GetCurrentBlockHash

通知API


Notify関数は、ネットワーク全体にイベントをブロードキャストします。以下の例では、Notify関数は「hello word」の16進文字列を返し、それをネットワーク経由で渡します。

from ontology.interop.System.Runtime import Notify
def demo():
    Notify("hello world")

これはログで確認できます。



GetTime API


GetTime関数は、関数が呼び出されたUnix時間を返す現在のタイムスタンプを返します。単位は秒です。

from ontology.interop.System.Runtime import GetTime
def demo():
    time=GetTime()
    return time # return a uint num

GetCurrentBlockHash API


GetCurrentBlockHash関数は、現在のブロックのハッシュを返します。

from ontology.interop.Ontology.Runtime import GetCurrentBlockHash
def demo():
    block_hash = GetCurrentBlockHash()
    return block_hash

シリアライズとデシリアライズ


これらは、直列化および非直列化関数のカップルです。Serialize関数はオブジェクトをバイト配列オブジェクトに変換し、Deserialize関数はバイト配列を元のオブジェクトに変換します。次のコードサンプルは、受信パラメーターを変換し、コントラクトの永続的なストレージに格納します。また、コントラクトの永続的なストレージからデータを取得し、元のオブジェクトに変換します。

from ontology.interop.System.Runtime import GetTime, CheckWitness, Log, Notify, Serialize, Deserialize
from ontology.interop.System.Storage import Put, Get, GetContext

def Main(operation, args):
    if operation == 'serialize_to_bytearray':
        data = args[0]
        return serialize_to_bytearray(data)
    if operation == 'deserialize_from_bytearray':
        key = args[0]
        return deserialize_from_bytearray(key)
    return False


def serialize_to_bytearray(data):
    sc = GetContext()
    key = "1"
    byte_data = Serialize(data)
    Put(sc, key, byte_data)


def deserialize_from_bytearray(key):
    sc = GetContext()
    byte_data = Get(sc, key)
    data = Deserialize(byte_data)
    return data

Base58ToAddressおよびAddressToBase58


このアドレス変換関数のペア。Base58ToAddress関数は、エンコードされたbase58アドレスをバイト配列の形式のアドレスに変換し、AddressToBase58は、バイトアレイの形式のアドレスをエンコードされたbase58のアドレスに変換します。

from ontology.interop.Ontology.Runtime import Base58ToAddress, AddressToBase58
def demo():
    base58_addr="AV1GLfVzw28vtK3d1kVGxv5xuWU59P6Sgn"
    addr=Base58ToAddress(base58_addr)
    Log(addr)
    base58_addr=AddressToBase58(addr)
    Log(base58_addr)

チェックウィットネス


CheckWitness（fromAcct）関数には2つの関数があります。

  • オブジェクトfromAcctが現在の関数を呼び出しているかどうかを確認します。はいの場合（つまり、署名の検証に合格した場合）、関数は戻ります。
  • 現在の関数を呼び出すオブジェクトがコントラクトかどうかを確認します。それが契約であり、機能が契約から実行される場合、検証はパスします。つまり、getCallingScriptHash（）の戻り値がfromAcctかどうかを確認します。GetCallingScriptHash（）関数は、現在のスマートコントラクトのコントラクトのハッシュ値を取得できます。
