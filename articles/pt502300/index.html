Projetos JavaScript para aprendizado e desenvolvimento



Bom dia amigos

Apresento a você uma seleção de 22 projetos em JavaScript puro ou, como se costuma dizer, baunilha (pessoalmente, prefiro apenas JavaScript).

Para mim, esse é um tipo de subtotal no aprendizado de JS.

O código é escrito em momentos diferentes, portanto, parece diferente e tem um nível diferente de complexidade.

A maioria dos projetos não pode ser chamada de aplicativos completos, mas cada um deles resolve com êxito a tarefa.

O código do projeto está aqui .

O formato será o seguinte: nome do projeto, sua breve descrição, sandbox.

Existem artigos separados para alguns projetos. Segundo outros, eles podem aparecer um dia.

Então vamos.

1. Progresso e visualização de áudio


Um aplicativo para reproduzir e visualizar dados de áudio usando a API de áudio da Web e a API do Canvas.


2. Fuga


O jogo "Breakout".


3. Calculadora


Calculadora. Biblioteca usada Math.js. V. 2.0.


4. Conversor de Moedas


Conversor de moeda. Uma API especial é usada.


5. Validador de Formulário


Validação do formulário.


6. Jogo da Forca


O jogo "Forca".


7. Galeria de Imagens e Slider Maker


Gerador flexível de galeria de imagens com controle deslizante embutido.


8. Pesquisa de Letras


Procure por letras (letras). Uma API especial é usada.


9. Pesquisa de refeições


Procure pratos com o método de cozimento e os ingredientes. Uma API especial é usada.


10. Meditação App


O pedido de meditação. Trabalhe com áudio e vídeo.


11. Cartões de Memória


Cartões para lembrar. Controle deslizante interativo.


12. Notes App


Pedido de notas (tarefas) com a capacidade de definir a data do lembrete. Para salvar dados no lado do cliente, o IndexedDB é usado.


13. Adivinhação de números


Adivinha o jogo dos números. Usado pela API de fala da Web.


14. Ping Pong


O jogo "Ping Pong".


15. Pedra Papel Tesoura


O jogo "Pedra, tesoura, papel".


16. Digitação rápida


Simulador de digitação rápida.


17. Tetris


O jogo "Tetris".


18. Leitor de texto


Pedido de leitura de texto. Usado pela API de fala da Web. Compare vozes da Microsoft e do Google.


19. Tic Tac Toe


O jogo "Tic Tac Toe". Um algoritmo especial é usado. Tente bater no carro.


20. Carrossel Tinder-Like


Um controle deslizante complexo, mas interessante. A biblioteca Hammer.js é usada.


21. Teclado virtual


Teclado virtual.


22. Weather App


Pedido de exibição do tempo. Uma API e um proxy especiais são usados.



O repositório será atualizado regularmente, portanto, fique atento.

Obrigado pelo seu tempo. Espero que tenha sido bem gasto.
