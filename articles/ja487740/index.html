ポータブル磁力計の組み立て

Instructablesトレーニング資料サイトの記事の翻訳



磁力計（ガウスメーターとも呼ばれます）は、磁場の強さを測定します[ この場合、磁気誘導/約。ペレフ。 ]。これは、永久磁石と電磁石の強度を測定するため、および磁石の重要な組み合わせのフィールド形状を確立するために必要なデバイスです。金属物体の磁化を検出するのに十分な感度があります。プローブが十分に高速で動作する場合、モーターとトランスからの時変場を検出できます。

携帯電話は通常3軸磁力計を備えていますが、1ガウス= 0.1 mTの地球の弱い磁場に最適化されています[ ミリリットル]そして、数mTの誘導があるフィールドでは飽和します。通常、このセンサーが電話のどこに正確に配置されているかは不明であり、マグネットカットなどのボトルネック内に配置することが不可能であることがよくあります。また、スマートフォンを強い磁石に近づけない方が良いでしょう。

この記事では、一般的なコンポーネントから最も単純なポータブル磁力計を作成する方法について説明します。リニアホールセンサー、Arduino、ディスプレイ、およびボタンが必要です。デバイスの総コストは5ユーロを超えず、-100 mTから+100 mTの誘導を0.01 mTの誤差で測定します-予想よりもはるかに優れています。正確な絶対インジケーターを取得するには、キャリブレーションを行う必要があります。長い自家製ソレノイドを使用して、これがどのように行われるかを説明します。

ステップ1：ホールセンサー


ホール効果は、磁場の測定によく使用されます。電子が磁場中に置かれた導体を通過するとき、それらは側面に運ばれ、その結果、導体に横方向の電位差が現れます。半導体の材料と形状を正しく選択することで、測定可能な信号を取得できます。信号は増幅され、磁場の1つの成分の測定値を生成します。

安くて手頃なSS49Eを使っています。そのドキュメントから注目に値するもの：

  • 電源：2.7-6.5 V、これはArduinoの5 Vと完全に互換性があります。
  • ゼロ信号：2.25-2.75 V、0〜5 Vのほぼ中間。
  • 感度：1.0-1.75 mV / G、正確な結果を得るにはキャリブレーションが必要になります。
  • 出力電圧：1.0-4.0 V（5 Vからの操作用）：範囲はArduino ADCでカバーされます。
  • 範囲：最小±650 Gs、通常+ / 1 1000 Gs。
  • 応答時間：3μs、つまり数十kHzの周波数での測定が可能。
  • 動作電流：6-10 mA、バッテリーに十分。
  • 温度エラー：摂氏1度あたり0.1％。少し見えますが、0.1％のずれは3mTの誤差となります。

センサーはコンパクトで4x3x3 mmで、前面に垂直な磁場の成分を測定します。磁石の南極に面している場合など、背面から前面に向かう磁場に正の値を与えます。センサーには+5 V、0 Vの3つの接点があり、出力は顔から見て左から右に向かっています。

ステップ2：必要な材料


  • リニアホール効果センサーSS49E。10個入り€1。
  • 試作ボードを備えたArduino Uno、またはポータブルバージョン用のピンなしのArduino Nano。
  • I2Cインターフェイスを備えたモノクロOLEDディスプレイSSD1306 0.96インチ。
  • ボタン。

プローブの場合：

  • ボールペンまたはその他の耐久性のあるチューブ。
  • チューブより少し長い3本の細いワイヤー。
  • 直径1.5 mmで12 cmの熱収縮。


ポータブルバージョンの場合：
  • 大きなTic-Tacボックス（18x46x83）または類似のもの。
  • 9Vバッテリー接点
  • スイッチ。

ステップ3：最初のバージョン-プロトタイピングボードの使用






まず、常にプロトタイプを作成して、すべてのコンポーネントとソフトウェアの動作をテストします。接続は写真に表​​示されています。ホールセンサーはArduino + 5V、GND、A1の接点に接続されています（左から右へ）。ディスプレイはGND、+ 5V、A5、A4（左から右）に接続します。ボタンを押すと、地面とA0が閉じます。Arduino IDE vで書かれた

コード 1.8.10。Adafruit_SSD1306およびAdafruit_GFXライブラリのインストールが必要です。

すべてが正しく行われていれば、ディスプレイにDCおよびAC値が表示されます。

ステップ4：コードについて


コードに興味がない場合は、この部分をスキップできます。

コードの重要な特徴は、磁場が連続して2,000回測定されることです。 0.2-0.3秒かかります。測定の合計の合計と二乗を追跡することにより、DCとACとして与えられる平均と標準偏差を計算することができます。多数の測定値を平均すると、精度が理論的に√2000≈45増加します。10ビットADCを使用すると、15ビットADCの精度が得られることがわかります。そしてそれは重要です：ADCの1ステップは4 mV、つまり、0.3 mTです。平均化により、誤差が0.3 mTから0.01 mTに減少します。

おまけとして、標準偏差を取得して、変化するフィールドを決定します。 50 Hzの周波数で振動するフィールドは、測定中に約10サイクル通過するため、ACの値を測定できます。

コンパイル後、次の統計が得られました。Sketchは16852バイト（54％）のプログラムストレージ領域を使用します。最大は30720バイトです。グローバル変数は352バイト（17％）の動的メモリを使用し、ローカル変数用に1696バイトを残します。最大は2048バイトです。

スペースの大部分はAdafruitライブラリで占められていますが、機能を追加する余地はまだたくさんあります。

ステップ5：プローブを調理する






プローブは細いチューブの端に固定するのが最適です。狭い場所に配置して保持するだけです。非磁性材料で作られたチューブであれば問題ありません。古いボールペンは私にはぴったりでした。

チューブよりわずかに長い3つの細いフレキシブルワイヤを準備します。私のケーブルには、ワイヤーの色（オレンジ+5 V、赤0 V、灰色-信号）にロジックがありません。覚えるのは簡単です。

プロトタイプでプローブを使用するには、ワイヤの一部をケーブルの端にはんだ付けし、熱収縮で絶縁します。後でそれらを切り離してArduinoに直接はんだ付けできます。

ステップ6：ポータブル機器の組み立て




9Vバッテリー、OLED画面、Arduino Nanoは、大きなTic-Tacボックスの中に快適に収まります。その利点は透明性です-画面は中に入っていても読みやすいです。すべての固定コンポーネント（プローブ、スイッチ、ボタン）はカバーに配置されているため、ボックスからすべてを取り外して、バッテリーを交換したりコードを更新したりできます。

私は9Vバッテリーが好きではありませんでした-それらは高価格で低容量です。しかし、私のスーパーマーケットでは、彼らは突然、充電式のNiMHを1ユーロで販売し始めました。11Vが100オームの抵抗器を介して供給され、一晩放置された場合、簡単に充電できました。安いバッテリーコネクターを自分で注文しましたが、送ってくれなかったので、古い9Vバッテリーを分解してコネクターを作りました。さらに、コンパクトさの9Vバッテリー、そしてArduinoはVinに接続されている場合、その上でうまく機能します。+5 Vでは、5 Vの調整可能な電圧が発生します。これは、OLEDとホールセンサーに必要です。

ホールセンサー、スクリーン、ボタンはプロトタイプと同じように接続されています。バッテリーとArduinoの間には、電源ボタンのみが追加されています。

ステップ7：校正








コード内の校正定数は、ドキュメントで規定されている数値（1.4 mV / G）に対応していますが、この値の範囲（1.0-1.75 mV / G）はドキュメントで許可されています。正確な結果を得るには、プローブを校正する必要があります。

明確な磁場を得るための最も簡単な方法は、ソレノイドを使用することです。ソレノイド磁場の磁束密度であるB =μ 0 * N * I.磁気定数（又は真空の透磁率）天然の定数である：μ 0 = 1.2566×10 -6T / m /A。磁場は均一であり、巻線密度nと電流Iにのみ依存します。これらは約1％の誤差で測定できます。この式は無限長のソレノイドで機能しますが、長さと直径の比が10

を超える場合は、中心のフィールドの非常に優れた近似値として機能します。適切なソレノイドを組み立てるには、直径の10倍長い中空の円筒パイプを取り、絶縁体から巻きますワイヤー。外径23 mmのPVCチューブを使用し、20.2 cmまで伸びる566ターンを作成しました。これにより、n = 28 / cm = 2800 / mになります。ワイヤの長さは42 m、抵抗は10オームです。

コイルに電力を供給し、マルチメーターで電流を測定します。電流を制御するには、調整可能な電流源または可変抵抗器を使用します。さまざまな電流値の磁場を測定し、測定値を比較します。

キャリブレーションの前に、6.04 mT / Aを受け取りましたが、理論的には3.50 mT / Aであったはずです。したがって、コードの18行目のキャリブレーション定数に0.58を掛けました。完了-磁力計が調整されました！
