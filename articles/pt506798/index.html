Ciência de dados para as ciências humanas: o que são dados

Reflexões sobre informação, memória, análise e distribuição


Tudo o que nossos sentidos percebem são dados, embora seu armazenamento em nossas tartarugas deixe muito a desejar. Gravar é um pouco mais confiável, especialmente quando gravamos em um computador. Quando esses registros estão bem organizados, os chamamos de dados ... embora eu tenha visto alguns rabiscos eletrônicos mal organizados terem o mesmo nome. Não sei por que algumas pessoas pronunciam a palavra dados como se tivesse uma letra maiúscula D.

Por que pronunciamos dados com uma letra maiúscula?

Precisamos aprender a ser desrespeitosamente pragmáticos em relação aos dados, para que este artigo ajude os iniciantes a olhar nos bastidores e os profissionais a explicar o básico para iniciantes que mostram sintomas de adoração de dados.

Análise e generalização


O problema com esses 27 números é que, mesmo que sejam classificados, eles significam pouco para nós. Lendo-os, esquecemos o que lemos um segundo atrás. Este é o cérebro humano para você; peça para lermos uma lista ordenada de um milhão de números e, na melhor das hipóteses, lembraremos dos últimos. Precisamos de uma maneira rápida de classificar e somar para que possamos entender o que estamos vendo.

É para isso que serve a análise!

median(weight)

Com o feitiço certo, podemos descobrir instantaneamente qual é o peso médio. (Mediana significa "média".)

Acontece que a resposta é 284g. Quem não gosta de gratificação instantânea? Existem todos os tipos de opções de resumo: min (), max (), média (), mediana (), modo (), variação () ... tente tudo! Ou tente esta palavra mágica para descobrir o que está acontecendo.

summary(weight)

A propósito, essas coisas são chamadas de estatísticas. As estatísticas são uma maneira de coletar seus dados. Não é isso que constitui um campo da estatística - aqui está uma introdução de 8 minutos à disciplina acadêmica.



Construção e visualização


Esta seção não é sobre o tipo de conspiração que inclui a dominação mundial (fique atento às notícias deste artigo). É sobre somar dados usando imagens. Acontece que uma imagem pode ser mais informativa do que mil palavras.



Se quisermos saber como os pesos são distribuídos em nossos dados - por exemplo, ainda existem itens entre 0 e 200 g ou entre 600 e 800 g? - O histograma é nosso melhor amigo.



Os histogramas são uma maneira (entre muitas) de somar e exibir nossos dados de amostra. Blocos mais altos para valores de dados mais populares.

.

Para criar um aplicativo de planilha, um feitiço é uma longa série de toques em vários menus. No R, é mais rápido:

eis o que obtivemos com uma única linha:

hist(weight)



O que estamos vendo?

No eixo horizontal, temos colunas. Por padrão, eles são instalados em incrementos de 200g, mas vamos mudar isso em um momento. As contagens estão no eixo vertical: quantas vezes vimos peso de 0 a 200 g? O gráfico diz 11. Que tal entre 600 g e 800 g? Apenas um (isto é sal, se a memória servir).

Podemos escolher o tamanho de nossas colunas - por padrão, o que obtemos sem nenhum problema com o código - 200 g, mas talvez desejemos usar 100 g. Sem problemas! Magos no processo de aprendizado podem refazer meu feitiço para descobrir como ele funciona.

hist(weight, col = "salmon2", breaks = seq(0, 1000, 100))

Aqui está o resultado:



Agora podemos ver claramente que as duas categorias mais comuns são 100–200 e 400–500. Alguém está interessado? Provavelmente não. Fizemos isso apenas porque podíamos. Um verdadeiro analista, por outro lado, se destaca pela ciência de navegar rapidamente pelos dados e pela arte de observar onde estão as pepitas interessantes. Se eles são bons em seu ofício, valem seu peso em ouro.

O que é distribuição?


Se esses 27 pontos são todos que nos interessam, o histograma de amostra que forneço também reflete a distribuição da população.

Isso é quase o mesmo que a distribuição: este é o histograma que você obteria se aplicasse Hist () a toda a população (a todas as informações que lhe interessam) e não apenas à amostra (os dados que você tem sob mão). Existem várias notas de rodapé, por exemplo, a escala ao longo do eixo Y, mas as deixaremos para outra postagem no blog - por favor, não me bata, matemática!



Se nossa população empacotasse toda a comida, a distribuição seria na forma de um histograma de todos os seus pesos. Essa distribuição existe apenas em nossa imaginação como uma idéia teórica - alguns alimentos embalados são perdidos há séculos. Não podemos criar esse conjunto de dados, mesmo que quiséssemos, então o melhor que podemos fazer é adivinhar usando um bom exemplo.

O que é ciência de dados


Existem muitas opiniões, mas prefiro a seguinte definição: "A ciência de dados é a disciplina que torna os dados úteis". Suas três subseções incluem a análise de uma grande quantidade de informações para procurar insights (análises), tomada de decisão razoável com base em informações limitadas (estatísticas) e o uso de modelos em dados para automação de tarefas (ML / AI).

Toda a ciência dos dados se resume ao seguinte: conhecimento é poder.

O universo está cheio de informações aguardando coleta e uso. Embora nosso cérebro seja versado em nossas realidades, não é tão bom para armazenar e processar alguns tipos de informações muito úteis.

É por isso que a humanidade se voltou primeiro para as pastilhas de argila, depois para o papel e, finalmente, o silício para obter ajuda. Desenvolvemos software para visualizar rapidamente as informações e hoje as pessoas que sabem como usá-las se chamam cientistas ou analistas de dados. Os verdadeiros heróis são aqueles que criam ferramentas que permitem que esses profissionais dominem melhor e mais rapidamente as informações. A propósito, até a Internet é uma ferramenta analítica - raramente pensamos nisso, porque até as crianças podem realizar essa análise de dados.



Atualização de memória para todos


Tudo o que percebemos é armazenado em algum lugar, pelo menos temporariamente. Não há nada mágico sobre os dados, exceto que eles são registrados com mais confiabilidade que o cérebro. Algumas informações são úteis, algumas são enganosas, o resto está no meio. O mesmo vale para dados.

Somos todos analistas de dados e sempre o fizemos.

Tomamos nossas incríveis capacidades biológicas como garantidas e exageramos a diferença entre o processamento inato de informações e a diversidade automática. A diferença é longevidade, velocidade e escala ... mas em ambos os casos as mesmas regras do senso comum se aplicam. Por que essas regras saem pela janela ao primeiro sinal da equação?

Fico feliz por chamarmos informações de combustível para o progresso, mas não faz sentido adorar os dados como algo místico. É melhor falar apenas sobre dados, pois somos todos analistas de dados, e sempre foi. Vamos dar a todos a oportunidade de se verem assim.
