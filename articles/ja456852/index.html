Unityでの列挙型フラグの描画を簡素化



この出版物では、C＃での列挙型、フラグとしての使用、Unityインスペクターでの描画を簡略化する方法について簡単に説明します。

Enumとは何ですか？


列挙型は、一連の定数で構成されるC＃の重要な型です。キーワードenumを使用して宣言します。各列挙子には整数値があります。最初の既定値0、そしてそれに続くものは増加1。

enum Color
{
    Red, // 0
    Green, // 1
    Blue // 2
}

初期化子を使用して値を上書きできます。

enum Color
{
    Red = 1,
    Green = 2,
    Blue = 3
}

列挙の各タイプには基本タイプがあり、charを除く任意の整数タイプにすることができます（デフォルトはintです）。明示的に指定することもできます。

enum Color : byte
{
    Red = 1,
    Green = 2,
    Blue = 3
}

旗


エンティティに多数のプロパティを付与することが必要になる場合があります。複数のフィールドを宣言したり、リストを作成したりできますが、1つの列挙で十分な場合もあります。列挙をフラグとして使用するには、特別な属性System.FlagsAttributeを追加します。これには、明示的に値を初期化する必要があり、それぞれの値が累乗されます。

[System.FlagsAttribute]
enum Color : byte
{
    None = 0,
    Red = 1, // 2 ^ 0
    Green = 2, // 2 ^ 1
    Blue = 4 // 2 ^ 2
}

ビットごとのOR演算を使用すると、列挙要素を組み合わせることができ、HasFlag（Enum）メソッドを使用して、インスタンスにビットフィールドが存在するかどうかを確認できます。

var color = Color.Red | Color.Green | Color.Blue;
var hasFlag = color.HasFlag(Color.Red | Color.Green); // True

AND ビット演算を使用してチェックを実行することもできます。

var aColor = Color.Red | Color.Green;
var bColor = Color.Green | Color.Blue;

//    
var contains = (aColor & bColor) == bColor; // False

//    
var overlaps = (aColor & bColor) != 0; // True

Unityでの転送


たとえば、次のコードを見てください。

using UnityEngine;

public enum Color
{
    Red,
    Green,
    Blue
}

public class Example : MonoBehaviour
{
    public Color Color;
}

組み込みのUnityツールを使用すると、列挙型をドロップダウンリストの形式で表示できます。



残念ながら、エディターはフラグの形式で列挙型を自動的に描画できません。これらの目的のために、検査官の再定義が必要ですが、これは常に便利なことではありません。しかし、あなたはトリックに行き、描画列挙をグローバルに再定義することができます。まず、例を変更します。

using System;
using UnityEngine;

[Flags]
public enum Color
{
    Red = 1,
    Green = 2,
    Blue = 4
}

public class Example : MonoBehaviour
{
    public Color Color;
}

次に、PropertyDrawerを実装する必要があります。シリアル化可能なプロパティにFlags属性がある場合、描画にはEditorGUI.MaskFieldメソッドを使用します。それ以外の場合は、標準のEditorGUI.PropertyFieldメソッドを使用します。また、プロパティは配列の要素になる場合があることに注意してください。以下のコードは、Editorというフォルダーに配置する必要があります。

[CustomPropertyDrawer(typeof(Enum), true)]
public sealed class EnumPropertyDrawer : PropertyDrawer
{
    public override void OnGUI(Rect position, SerializedProperty property, GUIContent label)
    {
        using (new EditorGUI.PropertyScope(position, label, property))
        {
            if (HasEnumFlagsAttribute())
            {
                var intValue = EditorGUI.MaskField(position, label, property.intValue, property.enumDisplayNames);

                if (property.intValue != intValue)
                {
                    property.intValue = intValue;
                }
            }
            else
            {
                EditorGUI.PropertyField(position, property, label);
            }
        }

        bool HasEnumFlagsAttribute()
        {
            var fieldType = fieldInfo.FieldType;

            if (fieldType.IsArray)
            {
                var elementType = fieldType.GetElementType();

                return elementType.IsDefined(typeof(FlagsAttribute), false);
            }

            return fieldType.IsDefined(typeof(FlagsAttribute), false);

        }
    }
}

Enumタイプ のフィールドがインスペクタに正しく表示されるようになりました。



特定の列挙値に加えて、エディターはさらに2つを追加します。

  • なし -整数値は0です。
  • すべて--1の整数値を持ちます。
