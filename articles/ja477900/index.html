素数-私たちの無力さはどれほど素晴らしいですか？

あなたが無限に高い壁に囲まれているが、壁の背後にあるものについてはまったく何も知られていないと想像してください。ここで、この壁がこの方程式によって擬人化されていると想像してください。

ブラックホールとこの方程式は、現実的で抽象的なものの究極の状態です。そして、1つ目について、2つ目について十分な推測とアイデアがある場合、実際には何も知られていません。しかし、それが本当に「数学的な」ブラックホールであるとしたらどうでしょう。私たちが地平線の下に落ちたら何が起こり得るのか知りたくないですか？

壁は何で構成されていますか？


数字-それらは現実の世界にはありません。7つのサイコロ、7つの原子、7つの致命的な罪がありますが、7つ自体は存在しません-それは抽象化です。はい、数は抽象的なオブジェクトの多くに過ぎないと言えますが、これは全世界です。現実世界と同様に、法律が存在する世界。これについての考えは非常に奇妙に思えます。それにもかかわらず、数論のような数学の枝の存在は、この「奇妙さ」が私たちにとって非常に重要であることを示唆しています。

最もエキサイティングなことは、これらの架空のオブジェクトの中に特別なもの-素数があることです。それらは決定論的カオスのようなものです-考慮の規模に応じて、予測可能と予測不可能は同時にです。たとえば、それらの隣にいると、任意の数nの前の数が次の値を超えないことが

壁の見え方


特定のアルゴリズムの操作の結果として特定の数列が得られた場合、この数列は無限に大きくても、列挙可能であると見なされます。列挙セットには1つの注目すべきプロパティがあります-Diophantine。つまり、このようなセットは、ディオファントス方程式（整数の正の係数と累乗を持つ多項式）で表すことができます。

次のステートメントはまったく信じがたいように思えるかもしれませんが、この定義に基づいて、ハッシュ関数（ビットコインを含む）のすべてのセキュリティキーと値はディオファントス方程式で表現できると主張できます。それら。整数で解かれる方程式。そしてはい、理論的には、誰かはどんな秘密でも学ぶことができ、無限に豊かで影響力のある人になることができます。しかし、そのような人になるためには、これらの方程式を最初に推定し、次に解かなければなりません。

コンピュータは、部分的または完全にディオファントス方程式によってセットを表すタスクに対処できます。しかし、ここで別の問題が発生します-ディオファントス方程式は一般的な形式で解かれません。それらを解決するための単一のアルゴリズムはありません。これは大きな問題ではないようです。なぜなら、いくつかの方程式は別々の形式に区別され、その効果的な方法はすでに見つかっているためです。

しかし、これらの方法が存在していても、計算の精度または反復の速度のいずれかに関連する計算上の問題が必ず発生します。

素数が列挙可能だったのはなぜですか？列挙可能なセットを表す方程式を作成するまさにそのプロセスは、数学の基礎-算術と論理に依存します。また、特定のセットのオブジェクトのプロパティに関する十分な知識がある場合、この知識に基づいて、オブジェクトを取得できるようにするアルゴリズムについて仮定を立てることができます。そして、結局のところ、この目的のために素数の特性に関する知識が蓄積されました。方程式が取得され、素数に関連するすべての質問がそれだけに減らされました。

しかし、それを解決することはできません。

壁を蹴る


ペル方程式から始めます。それらを解決するために、小さなスクリプトを書きます。

from decimal import *
getcontext().prec = 50

def peq_dec(N):
    n = Decimal(N).sqrt()

    a = int(n)
    x = n - a
    p0, q0 = 1, 0
    p1, q1 = int(a), 1

    while True:
        a = int(1/x)
        x = 1/x - a
        p_i = a*p1 + p0
        q_i = a*q1 + q0
        if p_i**2 - N*q_i**2 == 1:
            return p_i, q_i
            break
        p0, q0 = p1, q1
        p1, q1 = p_i, q_i
