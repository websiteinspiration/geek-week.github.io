termtosvg


ライブラリとライブラリをgithubに配置して、作成のライブデモがコンソールに表示される美しいアニメーションを表示するのが一般的です。



伝統は間違いなく良いと正しいです。そのようなアニメーションを記録することだけが困難/怠惰/一度になる可能性があります。termtosvgの作者は、すべての鳥を1つの石で撃ち、コンソールのデモを録音するための素晴らしいものをプログラマーに提供しました。


他の多くのソリューションとは異なり、termtosvgはビデオやgifを書き込みません。これは、コンソールで発生するすべてのものをキャッチし、ターミナルに表示される内容に基づいてSVGアニメーションをレンダリングします。出力では、画面アニメーションだけでなく、コンソール操作の美しくスタイリッシュなレンダリングも得られます。SVGは他の形式よりもはるかに軽量で、非常に簡単に編集することもできます。


目の肥えた人々のために、ライブデモをカスタマイズできるように、いくつかの事前定義されたカラーソリューションが含まれています。


ライトトピックで出力をレンダリングできます。



以下のウィンドウコントロールボタンと進行状況バーを使用して、このような暗いテーマでレンダリングできます。



また、他のダースの配色も可能です。


すべてが通常どおり設定され、 


pip install termtosvg

Mac OSドライバーは、brewを介して配信することもできます（必要なバージョンのPythonと一連の依存関係が自動的にプルアップされます）


brew install termtosvg
