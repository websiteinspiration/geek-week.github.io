この例では、前の例を利用して、Haarカスケード（Haarカスケード）に基づく顔認識システムを構築します。

Haarカスケードは、Haarウェーブレット手法に基づいてトレーニングされるカスケード分類子です。彼らは特定の兆候を検出するために画像のピクセルを分析します。 Haar Cascadesの詳細については、以下のリンクをクリックしてください。

Viola-Jonesオブジェクト検出フレームワーク
カスケード分類子
Haarのような機能

既にトレーニング済みのカスケードをここからダウンロードします。現在の例では、カスケードを使用して、顔の向きを特定します。

これを行うには、分類子を作成し、トレーニング済みのファイルをフィードする必要があります（上記のリンクを参照）。 pencv_haarcascade_frontalface_default.xmlファイルを、プログラムが配置されているディレクトリにすでにアップロードしました。

harrcascade := “opencv_haarcascade_frontalface_default.xml”classifier := gocv.NewCascadeClassifier()classifier.Load(harrcascade)
defer classifier.Close()

画像内の顔を検出するには、DetectMultiScaleメソッドを使用する必要があります。この関数は、カメラのビデオストリームから読み取られたばかりのフレーム（タイプMat）を取り、タイプRectangleの配列を返します。配列のサイズは、分類子がフレームで検出できた顔の数を表します。次に、彼が見つけたものを確認するために、四角形のリストを調べて、Rectangleオブジェクトをコンソールに移動し、検出された四角形の周囲に境界線を作成します。これは、Rectangle関数を使用して行うことができます。カメラで読み取ったマット、DetectMultiScaleメソッドによって返されたRectangleオブジェクト、境界線の色と太さを取得します。

for _, r := range rects {
fmt.Println(“detected”, r)
gocv.Rectangle(&img, r, color, 2)
}





package main
 
import (
"fmt"
"image/color"
"log"
 
"gocv.io/x/gocv"
)
 
func main() {
webcam, err := gocv.VideoCaptureDevice(0)
if err != nil {
log.Fatalf("error opening web cam: %v", err)
}
defer webcam.Close()
 
img := gocv.NewMat()
defer img.Close()
 
window := gocv.NewWindow("webcamwindow")
defer window.Close()
 
harrcascade := "opencv_haarcascade_frontalface_default.xml"
classifier := gocv.NewCascadeClassifier()
classifier.Load(harrcascade)
defer classifier.Close()
 
color := color.RGBA{0, 255, 0, 0}
for {
if ok := webcam.Read(&img); !ok || img.Empty() {
log.Println("Unable to read from the device")
continue
}
 
rects := classifier.DetectMultiScale(img)
for _, r := range rects {
fmt.Println("detected", r)
gocv.Rectangle(&img, r, color, 3)
}
 
window.IMShow(img)
window.WaitKey(50)
}
}
