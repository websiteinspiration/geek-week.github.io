検疫中に海外市場でスタートアップを立ち上げる方法：Reddit、Product Hunt、ラテンアメリカのリソース



画像：Unsplash

ここ数週間は、ロシアのビジネスプロジェクトにとって難しいテストになりました。状況を改善するための前提条件はありません。ルーブルは下落し、検疫下での作業はますます困難になっていますが、何かする必要があります。オンラインプロジェクトの場合は、海外市場への参入を試みることをお勧めします。しかし、それを行う方法は？

ここ数年、私はアメリカに住んでおり、ここおよびラテンアメリカで企業を宣伝するためのオプションを積極的に研究してきました。その結果、一定の経験が蓄積されました。この記事で共有する便利なツールとアプローチのリストです。重要なポイント-説明されているすべてのアクションはオンラインでのみ行われます。これは最近重要です。

この資料は、限られた予算の条件で海外の需要をテストし、国内市場の見通しがあまり良くない、理想的には外国為替収益でビジネスをサポートしたいプロジェクトマネージャーを対象としています。投稿は素晴らしかった、それが役に立つことを願っています。行こう！

テスト対象の国の特定、基本的なローカリゼーション


最初の段階では、テスト中のどの国に興味があるかを決定し、国際展開を試みる必要があります。

通常、誰もが「米国に入国」することを望みます。大きな市場だけでなく、大きな競争があることを忘れます。その結果、トラフィックとユーザーを引き付けることはそれほど単純で安くないかもしれません。

一方、ラテンアメリカにはアメリカに近い国があり、それが「追いつく」ことに成功すれば、アメリカではより簡単に始めることができます。ここには普遍的なレシピはありませんが、あなたのプロジェクトに似たプロジェクトがどこから来たのかを調査することをお勧めします（もちろん、すべてのスタートアップはユニークですが、それでも）トラフィックを獲得します。 SimilarWebはこれに最適です。

たとえば、新しい瞑想アプリケーションを作成している場合、SimilarWebこの市場の2つの主要なプレーヤーであるCalmとHeadspaceは、米国、イギリス、ドイツ、カナダで主要なトラフィックがあり、オーストラリアと中国でほぼ同じ数値であることを示します。

ヘッドスペース

この情報からの結論は異なる場合がありますが、一般的に、新しい国のオーディエンスとのコミュニケーションを開始するために、サイト/製品をローカライズする必要がある言語が明らかになります。

ローカリゼーション自体は、独自の複雑さを持つ独立したプロセスです。ただし、基本的な問題はUpworkで解決できます-このフリーランスのプラットフォームには、コンテンツをあらゆる言語に安価に適応させるのに役立つ編集者と翻訳者がいます。その中にはバイリンガルがいるので、ロシア語と英語を同等によく知っている人を見つけ、簡単にするために英語のテキストを条件付きスペイン語に適合させることができます。

もちろん、翻訳者/編集者のチームを見つけるには、ある程度の時間と労力が必要です。私は数か月間、意図的にこれに対処しなければなりませんでした。ただし、最終的には、最小限のコストで適切なコンテンツをすばやく入手できます。

戦略開発


海外市場でのマーケティングテストのアクティブフェーズを開始する前に、目標を決定することが重要です。それらは次のようになります。

  • 最初の英語を話すユーザーとヒスパニックのユーザーを引き付け、彼らの行動パターンを比較します。
  • 対象地域でのトラフィックの増加に対応します。
  • 売り上げを伸ばしましょう。

これらのケースのそれぞれで、異なるツールを異なる順序で使用でき、コストも異なる場合があります。

一般に、海外ではあまり知られていないスタートアップにとって、このスキームはうまく機能します

。1か月または2か月のコンテンツマーケティング作業-ブログの記入、ゲストの投稿、ソーシャルネットワークの維持。
注目を集めるために、特定の地域で人気のあるプラットフォームにアクセスできます。
新しい市場で販売目標到達プロセスを構築し、トラフィックの魅力を最適化するために取り組みます。

米国であなたのことを誰も知らない場合、フォーブスやTechCrunchに入ろうとしても無駄であり、お金を稼いでもうまくいきません。まず自分のリソースについて、そしてあまり知られていないリソースについて自分について何か言って、ソーシャルネットワークで製品を提供し、利益を得ることの方がはるかに論理的です。そして、それから初めて注目度の高い活動を計画し、有料マーケティングチャネルの使用を増やします。

たとえば、IndieHackers.comリソースを使用して、米国の起業家と通信できます。このサイトのユーザーは、まず自分自身がユーザーになることができます。次に、新しいプロジェクトに非常に忠実であり、3番目に、製品の使用経験があるため、貴重なフィードバックを提供します。これは、建設的な批判を求める別のプロジェクトの作成者からの投稿の例です。



英語圏以外の国のユーザーがどのリソースと通信しているかわからない場合は、Redditを開始するのが理にかなっています。このサイトには、主要言語のサブセクションがあります。たとえば、中南米のユーザーと簡単にチャットできます。以下はサブレディットの一部です。

  • / r / es / -一般的な主題のヒスパニック系サブレディット、36,000人のユーザー。
  • / r /アルゼンチン/-149千人の加入者;
  • / r / vzla / -ベネズエラのセクション、55,000人の加入者;
  • / r /コロンビア/ -24千人の加入者;
  • / r /チリ/ -82千ユーザー。

必要な言語のネイティブスピーカーを知らず、RedditのメッセージをUpworkでネットに翻訳するための予算がない場合は、Deepl.comトランスレータを使用することをお勧めします。それは主要な言語で動作し、その翻訳の品質はグーグル翻訳よりもはるかに高いです。ソーシャルネットワークへの投稿とユーザーとの基本的なコミュニケーション-これが必要です。



交通誘致プロジェクトのお知らせ


最初のユーザーを引き付け、話しかけ、プロジェクトの存在のデジタルトレースを目的の言語でいくつか残したら、より積極的なマーケティング活動に進むことができます。

この段階では、大量の（比較的）トラフィックと新規ユーザーを呼び込み、最小限のコストで売上を伸ばすために、プロジェクトの発表について話し合っています。

英語を話す視聴者について話す場合、いくつかの効果的なオプションがあります。 1つ目はBetalist.comサイトです。スタートアップのお知らせはこのサイトで公開されています。これは無料で行えますが、プロジェクトがメインページに表示されるまでに約1か月かかります。同時に、サービスのメーリングリストでの発表により、モデレートが加速される可能性があります-129ドルです。

私たちの経験では、このサイトに投稿すると、1日あたり数百人の訪問者が発生する可能性があります。ソーシャルネットワークや検索エンジンでコンテンツ広告を使用した場合のクリック1回あたりの費用と比較すると、それほど悪くはありません。もちろん、アナウンスと画像の準備には時間をかける必要がありますが、それだけの価値はあります。



Betalistでの「ゼロ」プロジェクトの発表

次のオプション：HackerNews Webサイト。これは、技術記事へのリンクのアグリゲーターです。このプロジェクトは、最も有名なスタートアップインキュベーターY Combinatorによって開始され、1300万人以上が訪れています。

優れた技術記事には、メインページにアクセスして、トップのロシア語のリソースに関する投稿が成功するという事実に匹敵するトラフィックを獲得するあらゆる機会があります。スタートアップのアナウンスのために特別に設計されたShow HNセクションもあります。



私たちの経験では、パフォーマンスの高いプレースメントはまだありませんが、無料であり、成功する可能性が最小限であっても努力する価値があります。

もちろん、スタートアップの発表に最も重要な英語のリソースはProduct Huntです。このサイトへのアクセスを準備する方法について、私はここに書きました。

つまり、アルゴリズムは次のとおりです。

  • プロジェクトへのリンクを公開することに同意する人気のあるユーザー（ハンター）を見つける必要があります。これにより、トラフィックの最初のフローが提供されます。
  • プラットフォームでおなじみの形式でコンテンツを準備します。あなたは、製品、アニメーションロゴをデモンストレーションする良い画像、gif、ビデオが必要になります。
  • プロジェクトをサポートするリリース日のアクティビティを計画し、1日を通して定期的にいいね！を受け取ります。
  • ユーザーと積極的にコミュニケーションし、質問に答えます。
  • Product Huntのオーディエンスに向けたユニークなオファーをいくつか準備します。



私たちの経験では、スペイン語圏の国については、Meneameプラットフォームが適切に機能します。これは人気のあるヒスパニック系リソースです（1か月あたり約1000万人の訪問者）。視覚的にはRedditに似ています-興味深い資料やテキスト投稿へのリンクも公開しています。資料の主な聴衆はスペインからですが、北米のヒスパニック系住民もいます。



「N最高のアプリケーション」形式のパルチザンコレクション、または「トップの方法...」はここでうまく機能します。リンクは企業ブログの投稿に移動できるため、すべてのトラフィックが直接サイトに移動します。

それ以外、タリンガ-ラテンアメリカのサイトで、ユーザー数は約2,500万人です。また、さまざまなセクションがあり、ユーザーは無料で資料を公開できます。



結論：次は


上記のアクティビティの実装には、少なくとも2か月かかります。この期間中は、製品とWebサイトを必要な言語にローカライズし、最初のユーザーを獲得してチャットし、最初の販売を行います。

次に、さらなるアクションの分析と計画の段階が始まります。海外旅行にはどれくらいの費用がかかるか、正確に何がもたらされるか、重要な指標がどれだけ急速に成長するかを理解する必要があります。必要なレベルまで成長するのを待つのに十分なリソースがありますか？

これらの質問に対する回答で続行できる場合は、事前に計画を立てる必要があります。ブログやProduct Huntなどの1回限りのアクティビティで信頼できるマーケティング戦略を構築することは不可能です。したがって、SEOに取り組み、メディアと対話し、アフィリエイトプログラムを構築する必要があります。
