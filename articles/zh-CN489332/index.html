世界上最有效的情报服务情报中的基本助手（第3部分）

历史学家在评估克格勃活动方面的立场分为赞成和反对。可以质疑从匿名来源或特种部队退伍军人回忆录中收集的材料和事实，因为人为因素和主观判断并未取消。即使是关于两个人去年夏天的天气的故事也会有所不同，更不用说对过去事件的秘密判断了。因此，本文的目的不是评估特殊服务，而是考虑克格勃在“照片方向”业务中的技术潜力。摄影作品在情报中占据着特殊的位置。在PSU的第14部门，对军官进行了特殊摄影方向的积极培训，他们获得了绰号“ fotiki”。受训人员执行最简单的任务-从研究肖像摄影的规则到掌握复杂的进口相机模型-梦想着职业发展，他的短语是：“你的雇员（姓）是一位来自上帝的摄影师。”

摄影指导人员是一个由知识分子组成的团队，是一群专家，特别是在PSU方面很受重视。国外OT（操作技术控制）官员的活动在很大程度上与各种各样的照相设备的使用有关，从影印机和用于暗中（不起眼）拍摄人物的照相机到用钢笔，打火机甚至口红制成的微型秘密照相机。所有这些不同的照相系统和相机都需要用于处理胶卷的特殊技术，以及用于打印照片的特殊技能（通常数量很多），以实现每个照片剪辑的可读性。所有这一切都必须由OT的常驻员工胜任地完成，而且不能出错，不要忘记他工作的其他重要领域。

战后时代，所有苏联情报人员都必须精通显微摄影：能够制作微点并将其安全地隐藏在常规主题中。准备状态的测试是检测隐藏的微点，提取，显影，干燥和读取的任务。

关于微点的一点。允许将大量信息压缩成微型微缩胶片的微点或微点。 Microdot技术可减少数百倍的图像。这是标准胶卷的一部分，在该胶卷上，与普通镜框一起放置了图像，如果没有特殊的放大设备，就无法用肉眼看到该图像。例如，在书的正文中切出并粘贴了这样的一个微点，而不是一个点（在句子的末尾），它的位置是事先由代理人商定的。

微型隐藏式日落监控摄像机


日落-由克拉斯诺戈尔斯克机械厂（苏联）生产的微型隐藏式监控摄像机。作为一家小型的光学机械厂，它于1929年开始工作，在这里他们从事医疗机构用显微镜和光学眼镜的生产。后来，成立了一个专门的车间，用于生产照相设备和光学系统，以满足特殊服务的需求。Zakhod相机于1989年发布，可以认为是F-21 Ajax和F-27 Neocyte相机的后继相机，该相机主要由苏联克格勃情报机构使用，并且至少在1992年之前投入生产。



日落自动照相机的尺寸为96 x 65 x 29毫米（带镜头），重量-不超过230克，照相机的尺寸与F-27 Neocyte大致相同。该相机旨在在16毫米宽，0.08-0.1毫米厚的ZP-80型无孔胶片上从3到7米的距离拍摄。自动安装曝光可以在各种拍摄条件下获得高质量的底片。



日落是专门为监视，监视而设计的。它可以谨慎地隐藏在各种物体中，例如，在包内，乘客舱或一件家具中，大衣的纽扣后面或适用于手术人员的身体。实际上，日落使用的迷彩与之前的迷彩相同，包括老式机械相机F-21 Ajax和电子F-27 Neocyte。为了隐藏相机中最“醒目的”部分-镜头，使用了带有滑动百叶窗的特殊喷嘴，将其安装在角架上。在拍摄期间，通过按遥控器上的棕色大按钮，首先在相机上打开百叶窗，拍照，然后再次关闭孔。所有这一切都很快发生，“没有噪音和灰尘”，实际上，要注意这一过程是根本不可能的。



相机存放在带有人造革拉链的手提箱中。整个工具包被放置在一个尺寸为28 x 20 x 5.5厘米的灰色木盒子中，照相机和盒子上都标有一个五位数的数字：前两位数字表示生产年份，后三位数字是序列号。



除照相机外，该盒子还包含电池供电的遥控器，电源控制单元，胶卷暗盒，各种辫子，餐巾纸，松鼠圆形毛刷，一套工具和配件，两个不带锁的喷嘴和一个带锁的喷嘴，两个暗盒。

相机配备了两个镜头：带有自动曝光控制TTL曝光表的标准2.8 / 25毫米镜头和可互换镜头。

TTL曝光表（通过镜头，TTL：“通过镜头”）是一种内置的曝光表，用于测量直接通过照相机或电影摄影机镜头拍摄的场景的亮度。一段时间以来，苏联摄影文献使用“内部测光”的概念以及相应的缩写“ BC”，例如，以“ Zenit-15 BC”相机的名义。但是，此名称后来被国际术语TTL取代。

标准镜头由4个组件组成，每个组件都固定在其框架中：

  • 半月板阳性
  • 粘液弯月面
  • 双凹透镜
  • 双凸透镜

相机正面有一个孔，螺纹M 13x0.5，用于安装镜头。摄像机的左侧是一条长约20厘米的集成电缆，带有一个14针微型插头，用于连接各种控制设备。案件的左侧是一个框架计数器。

摄像机由位于两个遥控器内部的3 V和5 V镍镉电池供电。可以通过将充电电缆连接到外部电源（PSU）对电池充电。



图像上方是一组备用盒带。该暗盒用于存放和运输长1.2米（至少30帧）的胶卷。它由三个可分离的节点组成：外壳，盖子和接收线圈。

该套件包括3个暗盒，因为不能中断照片监视过程，因此需要为代理商提供足够的胶卷。相机内装有一个暗盒，另外两个备用暗盒则存储在一个木箱中。每个带有胶片的备用物都存储在人造革制成的小箱子中，可以轻松放入手术人员的口袋。

带有固定电缆的遥控器已连接到摄像机，在它的左侧是一个ON和OFF模式开关。棕色的大按钮是快门按钮。使用右侧的两个开关（两个键一起移动），可以在自动和固定1/30秒之间进行选择。博览会。



除标准控制面板外，还开发了具有扩展功能的遥控器。其中，ON / OFF开关已移至右侧面板。最左侧的开关用于选择ONE / NPR摄像机模式（单次/连续）。



除了设备上的棕色大按钮外，还可以通过将附加按钮连接到2针连接器（位于电缆入口旁边）来使用附加按钮控制快门。



如果需要将摄像头遮盖在特工的身上，则这样的快门控制按钮是必不可少的：电缆穿过手术人员睡袍的袖子并连接到扩展的遥控器，而扩展的遥控器又连接到了摄像头。如果电缆断裂，则提供了一个额外的套件。

电源控制单元设计用于控制摄像机远程控制面板的电池电源。这种单元由外壳，两个塑料盖和一个电子单元组成。在底部有两个三针连接器，用于通过孔连接到控制台连接器。设备的前面板上分别有两个3 V和5 V以下的LED。它们发出的光表明遥控器电池中的电池正在运行。

延期。摄像机本身和遥控器都使用较短的电缆。尽管实际上这对于大多数类型的伪装就足够了，但是可能会出现电线短路的情况。解决此问题的方法很简单-30厘米延长线形式的延长线，两端带有14针插头。

相机及其配件必须存放在包装箱中，干燥的房间（最好在橱柜或架子上）中，并远离暖气片或窗户，温度不得低于+5°C且不高于+35°C，湿度不应超过80％可以在-40°C至+50°C的温度下，将摄像机运送到包装箱中。为防止箱体起雾（在不同温度条件下传送时），应在12小时后打开包装箱。

1992年，最后一部胶卷相机扎霍德（Zakhod）出现在俄罗斯特种部队兵工厂。对于她来说，伪装涵盖了“附件箱”，“裤子腰带”，“男式皮包”等。 Sunset的更高版本的相机带有一个遥控器，该遥控器仅需要一个6V电源，这样的遥控器较厚，并且具有圆形手柄，手柄中装有圆形镍镉电池。手柄的“展平”部分有一个矩形的塑料快门释放按钮和一个开关。

相机基辅


这不是苏联的照相机，而是苏联制造的德国照相机。



不仅克拉斯诺戈尔斯克机械厂（苏联）从事照相机的生产，以供克格勃特勤局使用。第二次世界大战后，从事Contax相机生产的工厂从德国完全出口到了基辅（以及专家，零件和配件）。阿森纳工厂成为其桥头堡，或者说是其最古老的部分，始建于1764年，是一间小窗户和1.3m厚的墙壁的房间。在这里，相机的生产仍在继续。 40年代末，该相机获得了新的名称“ Kiev”，并成功生产，1947年，前两个型号看到了光。尽管他们重复了Contax-II和Contax-III（基辅II和基辅III）的建造，但并未not窃，但它们是Contax的合法授权副本。基辅是高品质的产品，是一项重要项目，只有在苏联政要的帮助下，才有可能得到这样的东西。 1956年，引入了同步接触，该系列开始获得“ A”指数，出现了新型号的Kiev-2A和Kiev-3A。在50年代末，阿森纳工厂每月生产3.5万台相机。考虑基辅的几种型号，由于尺寸小，它们可以归因于“间谍”相机。

16毫米微型相机Kiev-30


Kiev-30相机是微型16毫米相机，于1974年发布。她是受欢迎的Vega系列基辅相机（微型Minolta 16相机的副本）的继承者。



相机在16毫米宽的无孔胶卷上拍摄，镜框尺寸为17 x 13毫米。胶片长度可以是45或65厘米，分别取决于帧数17或25。胶片在塑料盒中。相机配备了Industar-M镜头，焦距为23毫米，光圈为f / 3.5，具有5个不同的光圈（3.5-11）-您可以将光圈从f3.5设置为f11，所有中间值和3档速度快门。百叶窗是带有金属百叶窗的窗帘，只有3种快门速度被排斥-1 / 30、1 / 60、1 / 200。镜头的光圈为两瓣状，并形成一个方孔。曝光计算器以天气符号的形式显示，并具有手动设置的曝光参数。曝光是通过机身右侧的两个带肋轮设置的。



相机的尺寸为84 x 46 x 27毫米（关闭时），设备重量不超过178克。拍摄照片之前，必须先布置好相机，如下图所示。同时，镜头打开，镜架被运输，调焦轮和快门按钮变为可用。实际上，身体是控制元素。帧计数器显示拍摄的帧数。拍摄过程本身看起来像这样：翘起快门，设置快门速度和光圈，瞄准和对焦，释放快门。



现在，用户可以通过左侧的小框取景器看到物体。方形快门释放按钮在右侧，更靠近相机的前边缘。快门释放按钮的左侧是对焦控件（从50厘米到无限远）。





该套件包含以下内容：

  • 相机
  • 相机包
  • 盒式磁带
  • 放大镜框架“ Light”
  • 罐盘
  • 包装盒
  • 手册

由于体积小，对焦范围小，因此适合秘密摄影。它曾被前苏联（苏联）的特种部队，如克格勃使用。尽管小型，方便的相机已在背包客中得到普遍认可。 1987年，基辅30被基辅30M取代，并在1990年推出了改进型的基辅303。当然，该工厂对部件的更换和相机的维修提供了一年的保修，当然要遵循“正常操作，小心处理和正确存放”的条件。 1974年Kiev-30相机的零售价为30卢布，而1989年Kiev-30M的零售价为15卢布。在电影《电子冒险》中，电影Uri的主人公只是用Kiev-30相机拍摄了电子和Syroezhkin的“间谍”照片。



苏联16毫米Kiev-30相机在行动：为了掩饰，它以香烟盒的形式安装在金属盒中。射击是在特工从包装中抽出香烟时进行的。为了可靠性，将一根真正的香烟插入包装中。



这种用于KGB的间谍相机非常受欢迎。因此，特别是对于收藏家（已经在数码照片时代），Kiev-30，Kiev-303开始采用不寻常的设计生产，直接暗示了其“间谍”的过去。 Kiev-JPS是基辅30相机，伪装成John Player Special（JPS）品牌普通香烟包装。



这种伪装使照相机对秘密摄影不可见。在黑色金属制成的情况下，所有相同的尺寸均为102 x 55 x 28.5毫米，重约230克。一端进行了更改，使相机看起来像一支烟盒，出于相似的原因，三支香烟（其过滤嘴）从烟盒中伸出了一点。这些香烟中有两根处于固定位置，第三根（最长）香烟是可移动的（进出）。



香烟与胶片传送机构有关。如果拉动它，镜头会打开，胶卷会传送到下一帧，然后可以拍照。尽管最初认为该型号的相机是专门为克格勃设计的，但后来变得很清楚，这很可能是对天真的西方收藏家的欺骗。



根据用户的说法，由于两个主要原因，Kiev-30图像的质量不是很高。电影被严重刮伤了。窄膜当时供不应求，因此经常被严重逾期使用。胶卷在冲洗过程中必须冲洗干净，因为可能会留下斑点。

Kiev-303是16毫米微型相机，于1990年在乌克兰基辅的阿森纳工厂生产，是受欢迎的Kiev-30和Kiev-30M的后继产品。



实际上，这是Kiev-30的改进版本，快门速度提高了1/250。相机配备了相同的Industar-M镜头，其焦距为23 mm，光圈为f / 3.5，从50 cm到无限远对焦，光阑分别为1和2 m。









可以购买五种颜色的相机：黑色，绿色，红色，白色和灰色。摄像机的小尺寸使其成为特工和操作员军械库中必不可少的助手：处于关闭状态-84 x 46 x 27 mm，重量不超过178克。像Kiev-30一样，此案是控制要素。不能倒卷（16毫米无孔）。可以随时更换暗盒，而只有一小部分胶卷被照亮。



实际上，Kiev-303是对Kiev-30相机的改进，外观和其他快门速度有所变化。新的快门速度范围是1 / 30、1 / 60、1 / 125、1 / 250。

基辅35A


Kiev-35A相机是1985年至1991年在阿森纳工厂（乌克兰）生产的。该相机是1974年的Minox 35EL副本，带有彩色米诺塔尔镜头，在冷战期间赢得了监视人员的厚爱。微型35毫米相机，带有固定的5镜头MS Corsair 2.8 / 35镜头。镜头分辨率：中心50lin / mm，边缘28lin / mm。具有自动曝光功能的相机，快门速度最高可达1/1000 s，此外，可以手动将快门速度设置为4至1/500秒。它以6 V的电压在电池或一组磁盘元件上工作。前壁已折叠。墙壁重新倾斜后，镜头伸出到工作位置，快门按钮不再锁定，曝光计打开。快门的翘起和胶片的倒卷通过扳机的两次平移来进行。



Kiev-35A是全尺寸半自动摄影机相机，其光圈为24x36 mm。快门是中央镜头间电子控制，快门排是触发的。专注于从1 m到无穷远的比例。

相机的尺寸为101.5 x 64 x 32.5毫米，重量-小于200克，温度范围为-15至+45C。在关闭状态下，它可以放入任何口袋。

尽管乍看之下Kiev-35A和Minox 35EL相机看起来完全一样，但是Kiev 35A的制造质量仍然要求最好。苏联版本因其多次电子崩溃而“闻名”。即使半米之内相机也无法掉落。相机使用了电动指针曝光计，它与相机的电子快门相连，在受到最小的冲击时，最精密的机械装置受到了损坏。没有这种脆弱的“带箭头的弹簧”，相机将无法正常工作。他们试图消除Kiev-35AM模型中的这一弱点，于1991年用LED指示器代替了指针指示器。
