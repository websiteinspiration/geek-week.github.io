複数行のサイトのCSS変数とカラーテーマ

今日CSS変数を使用する1つの方法


ライト、ダーク、カラーのテーマを動的にサポートするサイトを作成します。


インタラクティブデモ



変化するベースカラーを作成します。htmlのデータテーマにバインドします。


コード例では＆from less / scss構文を使用しています


html[data-theme='green'] {
    --theme-color: 110;
}

今、私たちは色そのものが必要です。むしろ、その明るさと彩度。これを行うには、hslスキームを使用します。すべての変数をrootに配置します。


:root {
    --color: ~'hsl(var(--theme-color), 15%, 44%)';
    --background-color: ~'hsl(var(--theme-color), 30%, 10%)';
}

上手。小さなケース。目的の要素を取得し、それに変数を適用します。


.class-name {
    color: var(--color);
    background-color: var(--background-color);
}

次に、コントラストを変更します。置き換え：ルートをhtml [属性付き]に置き換えます。


// :root = html
html {
    &[data-theme-style='normal'] { }
    &[data-theme-style='dark'] { }
}

次に、各トピックについて、s、lの値を使用します。


html {
    &[data-theme-style='normal'] {
        --color: ~'hsl(var(--theme-color), 15%, 44%)';
        --background-color: ~'hsl(var(--theme-color), 30%, 10%)';
    }
    &[data-theme-style='dark'] {
        --color: ~'hsl(var(--theme-color), 70%, 31%)';
        --background-color: ~'hsl(var(--theme-color), 3%, 3%)';
    }
}
