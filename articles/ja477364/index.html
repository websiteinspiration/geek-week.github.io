Java開発者になるには？または多分Pythonを選択しますか？

こんにちは、ハブル！

私は最近、Java開発者になる方法についての良い記事を見つけました。「成功するJava開発者になる方法は？」著者AakashSingh_17。そして今、私はあなたと翻訳を共有したいと思います。ただし、客観的な利点とこの言語に関連する神話のために、多くの若い開発者が最初の言語としてPythonを選択していることがわかります。したがって、私はこの言語の分析を追加して、初心者が比較してナビゲートできるようにしました-そして、2019年には実際には2020年に何を選ぶべきですか？



それともJava？


今日、私たちの未来を形作る多くのプログラミング言語があります。ただし、プログラミングを学び始めるほとんどの人はCから始めて、C ++または他の多くのプログラミング言語の1つに進みます。しかし、今日は最も人気のあるプログラミング言語であるJavaについて考えます。Javaの人気は、その多様性によって説明できます。Javaは、軽量で高速でさまざまな目的に役立つ個々のアプリケーションの開発に使用できるためです。

Java開発者


2019年のTIOBEインデックス（言語の名前を含む検索結果の計算に基づいてプログラミング言語の人気を測定するインデックス）によると、Javaはナンバーワンのプログラミング言語です。実際、Javaは、小規模な企業であれ、GoogleやAmazonなどの大規模なテクノロジージャイアントであれ、ほとんどすべての企業で使用されています。したがって、現代のJava開発者になると、キャリアを上手に構築できます。そして、それが私が成功するJava開発者になる方法について私の考えを共有する理由です：

1.コアJava SEをマスターする


まず、Javaカーネル（Java SE）から始めて、それをできるだけ深くマスターします。ループ、配列、演算子などの基本的な概念を理解しておく必要があります。さらに、Java SEには、java.lang、java.io、java.mathなどの多くの汎用APIがあります。そして、あなたはそれらの少なくとも基本的な知識を持っている必要があります。また、OOPの概念を学ぶ必要があります。

2. Javaの練習を続ける


Core Javaで学んだことは何でも、構文エラーなどの重大なエラーなしで書くように練習してください。通常、理論的には問題の解決策を見つけるのは簡単ですが、アプローチの実際の実装では、言語制限が発生する可能性があるため、問題が発生する可能性があります。したがって、Javaのコーディングと実践を続け、興味のあるトピックに関するミニプロジェクトを作成してみてください。

3. Java EEを学ぶ


Advanced Javaとも呼ばれるJava EE（Enterprise Edition）の学習を開始します。Java EEには、分散コンピューティングやWebサービスなどの追加機能の仕様があるため、Java EEの学習を開始すると、Web開発に取り組むことができます。これには、サーブレット、WebSocket、Java Server Facesなどのテクノロジーが含まれます。

4.サーブレットとJSPを使用してプロジェクトを作成する


Java EEで学んだことはすべて、単にHTML、CSS、JavaScriptなどのインターフェース技術と組み合わせて、特定のトピックに関するミニプロジェクトを作ってみてください。さらに、少なくとも2つのプロジェクトを実行することが重要です。1つはサーブレットを使用し、もう1つはJSPを使用します。

5. HibernateおよびSpring Frameworkを探索する


Java SEとJava EEの探索を終えたら、Hibernate FrameworkとSpring Frameworkを含むJava Frameworkについて理解してください。これらは最も人気のあるJavaフレームワークです。実際、ほとんどのJava開発会社は、Webアプリケーションの開発にSpring Boot、Spring MVC、Spring CloudなどのSpring Frameworkを使用しています。Hibernateフレームワークは、オブジェクト指向ドメインモデルをリレーショナルデータベースにマッピングするためにも非常に人気があります。 Hibernateクエリ言語も同様です。したがって、これらのフレームワークを知っていて、それらのフレームワークでプロジェクトを作成できる場合は、どの企業でもJava開発者の役割に選ばれる大きなチャンスがあります。



Javaプログラマーは、ITの分野で高い給与と人気の高い職業の1つです。競争が激しいため、学生はより多くの情報を吸収し、仕事の初日からプロ意識を発揮する必要があります。今日の5つのヒントは、初心者から興味深く、給与の高い仕事に少し早く到達するのに役立ちます。

多分Python？


Pythonは、開発者の生産性とコードの可読性の向上に重点を置いた高水準の汎用プログラミング言語です。Pythonカーネルの構文は最小限です。同時に、標準ライブラリには大量の便利な関数が含まれています。



Pythonは、構造、オブジェクト指向、関数型、命令型、アスペクト指向のプログラミングをサポートしています。

Pythonは簡単に学習、読み取り、使用できます。また、労働市場で最もホットなテクノロジーの1つでもあります。しかし、これは一言で言えば単なるPythonです。もっとたくさんあります。Pythonが初心者に最適である主な理由を読んでください。
