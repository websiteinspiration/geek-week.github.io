Membuat bot Discord berdasarkan discord.js

Latar Belakang


Saya menjadi tertarik untuk membuat bot untuk Discord, tetapi yang saya temukan hanyalah beberapa video di YouTube, dan bahkan di sana mereka hanya menulis kode, tanpa penjelasan apa pun. Jadi saya ingin memulai serangkaian artikel tentang cara membuat bot berdasarkan discord.js.


Mendapatkan token bot


Pertama, kita perlu pergi ke halaman pembuatan aplikasi dan klik Aplikasi Baru .



Setelah itu, kita akan diminta untuk menyebutkan aplikasi kita. Masukkan nama yang diinginkan dan klik Buat . Kita sampai ke halaman pengaturan aplikasi, di mana kita dapat mengubah nama, avatar, deskripsi, dll. Tapi kami belum membutuhkannya, buka tab Bot . Dan buat bot.



Tetap hanya menyalin token.



Mempersiapkan lingkungan pengembangan


Pertama, Anda perlu menginstal Node.js dan Visual Studio Code (VS code), pada akhirnya, pada kenyataannya, proses membuat bot akan berlangsung. Kami memulai kode VS. Kami membuka folder tempat bot kami akan disimpan. Klik View-> Terminal , setelah itu terminal akan muncul di bagian bawah aplikasi.

const Discord = require('discord.js'); 
const bot = new Discord.Client();
//  
let config = require('./botconfig.json'); 
//""   
let token = config.token; 
let prefix = config.prefix;
// -  
bot.on('ready', () => { 
    console.log(`  ${bot.user.username}`);
    bot.generateInvite(["ADMINISTRATOR"]).then(link => { 
        console.log(link);
    });
});
//,  ,    
bot.on('message', msg => {
    if (msg.content === prefix + 'habr') {
        msg.reply('The Best!');
    }
});
bot.login(token);
