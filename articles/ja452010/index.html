Osmo Action：DJIの最初のアクションカメラ



DJIはブランドを維持し続けます。今回、同社はデバイスをニッチ市場でリリースすることを恐れなかったが、ニッチ市場は長い間、いくつかの主要な業界リーダーによって占められてきた。私たちはアクションカメラについて話している-最もダイナミックなシナリオと極端な性格でビデオを記録するために不可欠なデバイス。ビデオ機器の作成経験と最新のテクノロジーを組み合わせて、DJIはほとんどすべての環境での撮影用に設計された別のデバイスで製品ラインを拡大しました。Osmo Actionは、広角レンズと2つのカラーディスプレイを備えた耐衝撃性と防水性のポケットカメラです。新しいデバイスを詳しく見てみましょう。

外観




この点で、DJIはホイールを再発明しないことを決定しました-Osmoアクションはすべてのアクションカメラの90％のように見えます。いくつかのデザインと技術的な改善がありますが、デバイスの元のデザインは呼び出すことができません。

Osmo Actionは、ライトグレーとダークグレーの耐久性のあるプラスチック製で、特別な滑り止めコーティングがあります。ケースのすべての要素は非常に密に隣接しており、アクションカメラの手の中では完全に一体感を感じます。デバイスの重量は非常に快適です-124グラム。

前面には、自画像用のスクリーン、ねじ込み式保護レンズ付きのレンズ、プロセッサーから熱を取り除くためのグリル、およびLEDインジケーターがあります。

上部には、マイクの穴、2番目のLEDインジケーター、録画を開始またはシャッターを解放するボタン、電源ボタン、画面のオン/オフも行います。

左側には、クイックメニューキー（QuickSet）とドングルがあり、その下には最大256 GBのmicroSDカードスロットとUSB-Cポートがあります。右側はスピーカーと2番目のマイクジャックです。バッテリーコンパートメントは下端まで運ばれ、2つのラッチで開きます。裏面はほぼ完全にタッチスクリーンで占められています。

デザインの特徴




Osmo Actionには、DJIがユーザーに焦点を当てたいくつかの主要な機能があります。まず、耐衝撃性と防水性があり、最大1.5メートルの高さからの落下に耐え、追加の保護カバーなしで水深11メートルで使用できます。購入後すぐに、追加のアクセサリーについて考えることなく、極端な冒険の撮影を開始できます。

第二に、2つのカラーディスプレイ。メインのタッチスクリーンの対角は2.25インチで、輝度レベルは750ニットです。これは晴れた日にカメラを操作するには十分です。ジューシーでクリアな画像になります。また、ディスプレイの反射防止コーティングにより、直射日光の下でも撮影できます。フロント画面はタッチではありませんが、まったく同じ特性を持っていますが、それはサイズのほぼ半分-1.4インチです。そのアクティブ化は、QuickSetボタンを長押しすると発生します。手袋をしている場合やフロントディスプレイを使用している場合は、同じボタンでモードを切り替えます。

競合モデルと同様に、Osmo Actionのメイン画面は背面パネルの中央ではなく、少し左にシフトしています。これにより、撮影時には親指が強調され、ディスプレイに触れてしまう心配がありません。

興味深い革新は、レンズの保護レンズです。反時計回りに回すだけで簡単に交換できます。他の同様のチャンバーでは、これのためにガラスとフレームを壊さないように、いくつかの努力をする必要があり、常に心配する必要がありました。DJIは多くの人の生活を大幅に簡素化しました。

カメラ




Osmo Actionは、視野角145度のレンズ、歪みの少ない撮影用の3層非球面レンズ、2層の疎油性および反射防止コーティングを施した保護ガラスを備えています。1 / 2.3インチのCMOSセンサーとf / 2.8の開口部を備えた12メガピクセルのカメラは、最高60フレーム/秒の速度と100 Mbpsのビットレートで高品質の4Kビデオをキャプチャします。

ちなみに、これは機械式手ブレ補正を使用しない最初のDJIデバイスです。 Osmo ActionはRockSteadyと呼ばれる電子安定化技術を使用しています。動作原理は次のとおりです。カメラは受信した画像を分析し、エッジを切り取り、多くのフレームを1つのシャープなフレームに結合します。そして、これらすべてをリアルタイムで。安定化を有効にできるビデオの最大品質は、60フレーム/秒の速度で4Kです。

ユーザーには、撮影のための2つのオプションがあります。広角レンズモードと、歪み低減のアクティブ機能（Dewarp）です。 2番目のオプションでは、実質的に視野角を犠牲にすることなく、「フィッシュアイ」の影響を排除できます。

設定には、自動カメラ設定と手動カメラ設定の両方があります。シャッタースピードの範囲は、写真の場合は1/8000〜120秒、4Kビデオの場合は最大1/60秒（1080Pの場合は最大1/25秒）です。ISOの範囲も100〜3200とかなり広いです。RAW形式の写真撮影、D-Cinelikeカラーモードでのビデオ録画、2000K〜10000Kのホワイトバランス調整もあります。

ハードウェア機能




Osmo Actionは文字通り1秒でオンになり、すぐに仕事に取り掛かれます。さらに、設定で目的のモードを簡単に選択できます。これは、デバイスの電源を入れた後にアクティブになります。

カメラには取り外し可能な1300 mAh LiPoバッテリーが搭載されています。電子式手ブレ補正をオンにした状態で30フレーム/秒の速度で93分の4Kビデオを撮影するか、30フレーム/秒の速度で1080pで135分間撮影するだけで十分です。フル充電時間は88分です。

スマートフォンに接続して映像を送信するためのいくつかの通信モジュールがあります。接続はWi-Fi（802.11a / b / g / n / ac）とBluetoothバージョン4.2を介して行われます。同期はDJI Mimo経由です。このアプリケーションは、スマートフォンを2番目のビューファインダーに変え、さらに、必要なカメラ設定の選択、ビデオテンプレートの適用、カメラからのファイルのダウンロード、それらの編集などの機能を提供します。ただし、Osmo Actionを完全に使用するために、プログラムをインストールする必要はありません。

ソフトウェア機能




Osmo Pocketに実装されているように、スワイプを使用してカメラメニュー内を移動します。システムはシンプルで直感的です-さまざまな設定が非表示になっている場所をすぐに理解できるので、Osmo Actionを使用して数分で作業できます。たとえば、撮影モードのあるポイントに移動するには、左下のアイコンをタップする必要があります。選択したモードのパラメーターを構成するには、上にスワイプします。追加設定、広角レンズの歪みを取り除くための機能（Dewarp）の有効化、シャッター速度とISOの手動調整モード-左にスワイプします。下にスワイプ-コントロールセンターに移動します（基本的なOsmoアクション設定）。

カメラは音声コマンドで制御することもできます。残念ながら、これまでのところ英語または中国語のみです。 5つのコマンドが利用可能になりました：記録の開始、記録の終了、写真の撮影、画面の切り替え、オフ。

撮影モードはたくさんありますので、Osmo Actionなら飽きません。手始めに、動画を4倍または8倍も遅くする素晴らしいSlowMotionがあります！

ほとんどすべてのDJIデバイスと同様に、Osmo ActionにはTimeLapseシューティングがあります。 4K、2.7K、または1080Pの品質を選択し、間隔（0.5秒の1フレームから30秒の1フレームまで）と最終的なクリップの長さを設定するだけです。



HDRには撮影モードがあり、多くの記録品質設定があります。安定化なしで、アスペクト比3：4、フレームレート30、25、24 fpsの4Kビデオの撮影が可能です。アクティブなEIS-標準のアスペクト比と60 fps〜24 fpsのフレームレートの4Kビデオ。その他の映画の解像度：2.7K、1080P、720P。そして最後の2つは最大240 fpsのフレームレートをサポートします。シリアル撮影もあり、毎秒3、5、7フレームを実行できます。

選択した設定を別のプロファイルに保存して、後で数回クリックするだけで選択できます。合計で、5つのユーザープロファイルがメモリに保存されます。

右にスワイプすると、簡単なファイルマネージャが開きます。ここでは、お気に入りのビデオや写真を選択するか、削除します。タイプ別にファイルを表示するためのフィルターがあります-slowmo-video、タイムラプス、AEB、ビデオまたは写真のみ。

コントロールセンターまたはOsmoアクションのメインメニューでは、さまざまなカメラ設定が非表示になります。タッチスクリーンの明るさの調整やロック、ユーザープロファイルの構成、セカンダリスクリーンとフルスクリーンでの画像のワイドスクリーン表示の選択、露出のスポットメータリングの実行、カメラの位置に応じた自動画面回転の有効化、音声コマンドの有効化は簡単です。さらに、詳細設定のメニューがあります。その中で、クイックメニューからアイテムを追加または削除し、ノイズリダクションをアクティブにし、スピーカーの音量、Wi-Fi設定、メニュー言語などを調整します。

アクセサリー




メーカーはOsmo Actionと一緒に、追加のアクセサリをいくつか箱に入れました。これは、オプションアクセサリー用のユニバーサルマウントと片手で着脱できるアクセサリー用の自己接着マウント、およびOsmo Actionを平面または曲面にマウントするための接着プラットフォームを含むフレームマウント付きのキットです。

さらに、DJIはそのアクションカメラ用のさまざまな追加アクセサリを発表しました。Osmoアクションを水深60メートルまで潜水するための防水ケース、外部マイクを接続するための3.5 mmアダプター、1.4インチネジマウント付きのモノポッド、モノポッドフロート、レンズ用のさまざまなフィルター、最大3つまで充電できる充電ハブが含まれています電池。

Osmo Action-最初のパンケーキはゴツゴツしません




ここに隠すことは何もありません。OsmoActionは、DJIの新しい、まだ探求されていない方向性です。しかし、同社はその仕事に100％対処しました。設計の観点から、画期的なことを行うことができなかった場合（これは課題ではありませんでした）、機能的には、多くの競合他社が依然として成長および成長しなければならない最高レベルのパフォーマンスが示されました。これらは、豪華で快適なディスプレイ、優れたカメラ、豊富な撮影とデバイスの設定、独自のモード、そしてDJIの驚くべき品質です。一般的に、エクストリームスポーツのファンで、冒険を頻繁に始める場合、Osmo Actionは本当の喜びを与えてくれます。
