SDL 2のレッスン：レッスン1-こんにちは、SDL 2

みなさん、こんにちは！ここでSDL 2を見ることにしましたが、ロシア語で意味のあるものは見つかりませんでした。ここからインスピレーションを得て、自分で書くことにしました。

SDL 2のインストールはスキップし、すぐにプログラムを作成することから始めます。

最初のウィンドウ




SDL 2を接続することから始めましょう。

#include <SDL2/SDL.h>

ここでは、いくつかのグローバル変数を宣言します。

const int SCREEN_WIDTH = 640;
const int SCREEN_HEIGHT = 480;

次に、おなじみのメイン関数をすべてのユーザーに公開します。

int main (int argc, char ** args) {
    
    if( SDL_Init( SDL_INIT_EVERYTHING ) != 0 )
    {
        return 1;
    }

    SDL_Surface* screen_surface = NULL;

    SDL_Window* window = NULL;

コードを少し見てみましょう。メインでは、コマンドラインパラメータはコマンドラインから受け入れられますが、受け入れなければなりません。次に、SDL_Init（）関数でSDL 2を初期化します。これにSDL_INIT_EVERYTHINGを渡しました。つまり、すべてのSDLモジュール（ビデオ、オーディオなど）を初期化します。SDL_Initにはさらにいくつかのフラグがあります。

旗
SDL_INIT_TIMER — ;
SDL_INIT_AUDIO — ;
SDL_INIT_VIDEO — , ;
SDL_INIT_JOYSTICK — ;
SDL_INIT_HAPTIC — ( , );
SDL_INIT_GAMECONTROLER — , ;
SDL_INIT_EVENTS — ;
SDL_INIT_EVERYTHING — , ;
SDL_INIT_NOPARACHUTE — .

次に、SDL_CreateWindow（）関数でウィンドウを作成します。意図したとおりに問題が発生した場合は、1を返します。

    window = SDL_CreateWindow("Hello, SDL 2!",SDL_WINDOWPOS_UNDEFINED, 
    SDL_WINDOWPOS_UNDEFINED, SDL_SCREEN_WIDTH, SDL_SCREEN_HEIGHT, 
    SDL_WINDOW_SHOWN);
    
    if (window == NULL) {
        return 1;
    }

SDL_CreateWindow（） 関数は、ウィンドウ名、OXウィンドウの位置、OYウィンドウの位置、ウィンドウの幅、ウィンドウの高さ、およびフラグを受け入れます。次に、フラグSDL_WINDOW_SHOWNを回避します。

ウィンドウを作成するための他のフラグがあります。

旗
SDL_WINDOW_FULLSCREEN — , ;
SDL_WINDOW_FULLSCREEN_DESKTOP — ;
SDL_WINDOW_OPENGL — OpenGL;
SDL_WINDOW_HIDDEN — ;
SDL_WINDOW_BORDERLESS — ;
SDL_WINDOW_RESIZABLE — ;
SDL_WINDOW_MINIMIZED — ;
SDL_WINDOW_MAXIMIZED — ;

ここでプログラムをコンパイルして実行すると、640x480のウィンドウが白く塗りつぶされ、しばらくすると閉じます。素晴らしい！この奇跡を少し楽しむために、SDL_Delay（2000）と書くことができます。プログラムが2秒間フリーズします。

もちろん、窓だけでもいいのですが、それを描くとさらにクールになりますよね。これを行うには、すべてを描画するサーフェスを作成します。ウィンドウ内で直接描画すると、どういうわけか醜いか何かになります。

   screen_surface = SDL_GetWindowSurface(window);

    SDL_FillRect(screen_surface, NULL, SDL_MapRGB(screen_surface->format, 0, 255, 0));

    SDL_UpdateWindowSurface(window);

まず、SDL_GetWindowSurface（）関数を使用してウィンドウサーフェスを取得します。それは、表面の寸法とそのフォーマットを示す必要がなく、すぐに仕上げられた表面を得るために作成されました。また、このサーフェスはウィンドウに描画する必要はありません-それ自体がすべてを行います。

次に、SDL_FillRect（）を使用して、表面を緑色にペイントします。私たちは書いたSDL_MapRGBを私たちは色を設定するRGB（赤、緑、青）を使用することをプログラムに伝えます。

そして最後に、SDL_UpdateWindowSurface（）関数を使用してウィンドウを更新し、指示した内容を表示する必要があります。

この段階で、プログラムを起動すると、明るい緑色で塗りつぶされたウィンドウが表示されますが、2秒以内には閉じません。

プログラムが完了したら、不要なファイルからRAMをクリアし、ウィンドウを閉じ、通常、プログラムの正しい終了を確認する必要があります。特別に用意された関数は、これに役立ちます。

    SDL_DestroyWindow(window);

    SDL_Quit();

SDL_DestroyWindow（） 関数はウィンドウを閉じ、SDL_Quit（）関数は以前にSDL_Init（）関数で接続されたすべてのモジュールを初期化解除します。

ええと、レッスン1は終わりました。すべてが明確になったことを願っています。これが私が最後に得たコードです：

#include <SDL2/SDL.h>
const int SCREEN_WIDTH = 640;
const int SCREEN_HEIGHT = 480;

int main (int argc, char ** args) {
    
    if( SDL_Init( SDL_INIT_EVERYTHING ) != 0 )
    {
        return 1;
    }

    SDL_Surface* screen_surface = NULL;

    SDL_Window* window = NULL;

    window = SDL_CreateWindow("Hello, SDL 2!",SDL_WINDOWPOS_UNDEFINED, 
    SDL_WINDOWPOS_UNDEFINED, SCREEN_WIDTH, SCREEN_HEIGHT, 
    SDL_WINDOW_SHOWN);
    
    if (window == NULL) {
        return 1;
    }

    screen_surface = SDL_GetWindowSurface(window);

    SDL_FillRect(screen_surface, NULL, SDL_MapRGB( screen_surface->format, 0, 255, 0));

    SDL_UpdateWindowSurface(window);

    SDL_Delay(2000);

    SDL_DestroyWindow(window);

    SDL_Quit();

    return 0;
};
