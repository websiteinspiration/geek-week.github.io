DIY视频录制到云端

亲爱的读者，您在一个夏日的夜晚沿着街道走去，不碰任何人，然后……他们碰到了您（如他们所说的pah-pah-pah）。流氓，只是路人，特殊的同志（就像他们唱的一首老歌）-没那么重要。您拿出手机，然后开始拍摄视频中发生的情况。对于开车的人来说，这不是一件很愉快的事，并且电话是从您手中取下的（或者他们被合法扣押了-强调必要之处）。没有目击者，电话上也没有视频录像，也没有警察和法院的证据。


解决这种情况的方法很明显：视频不应录制到手机上的本地文件中，而应直接录制到远程服务器上。的确，没有很多现成的软件解决方案可以实现这一想法（例如，在此处）：在大多数情况下，提供的手机应用程序要么付费，要么工作非常差。我不会考虑诸如“万一流氓攻击开始在YouTube上播报”之类的异乎寻常的建议，因为在实际情况下，您根本没有时间开始广播。此外，视频将被写入他人的云中，这通常是不好的。


当然，您可以学习Java或Kotlin（同时包含Swift），或者在最差的情况下，掌握PhoneGap并编写自己的应用程序。但是，一切都变得更加简单：在这种情况下，通过HTML5视频/音频API可以轻松解决此问题。


联系WebRTC


当然，WebRTC  是一件很酷的事情，允许您直接广播到云中。但是，执行此类翻译仍然是痔疮，因此我选择解决方案要容易得多。视频会每分钟（例如）写入手机的RAM（注意，不是写入SD卡，而是写入RAM），并且还会在录制后发送到服务器。也就是说，即使流氓开始将您的手机从您身边夺走，您也可以按停止按钮，最后一个视频文件进入服务器。


在默认设置下，一分钟的录制文件大小约为20 MB。同时，没有应用程序，至少是现成的，至少是自编写的，只有硬核的，只有HTML和javascript的。


跨浏览器问题


公平地讲，必须说对HTML5视频/音频API的支持尽管发展迅速，但仍然给开发人员带来很多问题。在下面的代码中，我故意没有提供跨浏览器的选项，以免使看法复杂化。老实说，我还没有在各种操作系统和浏览器上测试过此代码：Debian下的Mozilla Firefox 68和Android 7下的Chrome 83都可以正常工作。在Debian下的Chromium 80中，以及在许多用于Android的浏览器中，它都无法以其编写的形式运行。

"use strict";

//      
const recTime = 60;

//    queryString
let pwd = location.search || 'a'; pwd = pwd.trim().replace('?', '');

const video = document.querySelector("video"),
      butt  = document.querySelector("button");

let media, playFlag = false;

//   
const play = async () => {
   try {
      //     ,   
      let c = /Android|iPhone/i.test(navigator.userAgent) ?
         {video:{facingMode:{exact:"environment"}}, audio:true} :
         {video:true, audio:true};

      //        
      let stream = await navigator.mediaDevices.getUserMedia(c);
      video.srcObject = stream;
      video.play();

      //      recTime 
      media = new MediaRecorder(stream);
      media.ondataavailable = d => {
         fetch("api.php", {
            method: "POST",
            headers: {"Content-Type": "video/webm", "X-PWD": pwd},
            body: d.data
         })
      };
      media.start(recTime * 1000);
   }
   catch(err) {alert(err);}
};

//    /
const go = () => {
   if (!playFlag) {
      butt.innerHTML = "&#9209;";
      play();
   }
   else {
      butt.innerHTML = "&#9210;";
      video.pause();
      video.srcObject = null;
      media.stop();      
   }
   playFlag = !playFlag;
}
