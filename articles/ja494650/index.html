言語のすべての単語でアナグラムとサバナグラムを検索する

アナグラムの問題を解決することがアイデアにつながりました。
ロシア語の辞書からすべてのアナグラムとサンガングラムを削除した場合の残りの単語数

辞書には含まれ、様々な形で150万人以上の言葉を。

あなたは、それぞれに各単語を比較することができますが、150万個のエントリのためにそれは長く、次善のです。
無限のメモリがある世界では、各単語のすべての順列の部分文字列を生成し、それらの辞書をチェックできます

が、より良い解決策はありますか？

用語から始めましょう：

アナグラム-文字の再配置によって得られた単語
例：ロケットとキャリッジ

サバナグラム-別の単語の文字で構成できる単語
例：アーチ - ロケットという単語のサバナグラム

 public boolean isAnagramOrSubAnagram(Word word) {
        Character minCharacter = word.getMinCharacter();

        Stack<TrieNode> stack = new Stack<>();
        stack.add(root);

        while (!stack.isEmpty()) {
            TrieNode node = stack.pop();

            for (Entry<TrieKey, TrieNode> entry : node.getChildren().entrySet()) {
                char character = entry.getKey().getCharacter();
                if (character < minCharacter) {
                    stack.add(entry.getValue());
                } else if (character > minCharacter) {
                    break;
                } else if (entry.getKey().getCount() >= word.getCharacterCount(minCharacter)) {
                    if (doesMinWordCharacterNodeContainAnagram(entry, word)) {
                        return true;
                    }
                }
            }
        }
        return false;
    }
