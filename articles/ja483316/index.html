SwiftUIの簡単な紹介

SwiftUIは、Xcodeでユーザーインターフェイスを作成するための新しい便利な方法です。 SwiftUIの簡単な紹介が必要な場合は、このブログ投稿が役立ちます。

はじめに、SwiftUIをサポートするプロジェクトを作成しましょう。これの最小システム要件はmacOS CatalinaとXcode 11です。新しいプロジェクトを作成し、「ファイル>新規>プロジェクト...」、「シングルビューアプリ」を選択して、「ユーザーインターフェイス：SwiftUI」が選択されていることを確認します。



プロジェクトは次のようになります。



お気づきかもしれませんが、プレビューですぐにすべての変更を確認できます。たとえば、テキストを「Hello、Habr！」に変更してみてください。。かっこいい？

変更がない場合は、右上隅にある[ 再開 ]ボタンをクリックします。プレビューが表示されない場合- エディターオプションをクリックし、キャンバスが選択されていることを確認してください（⌥Options+⌘Command+↵Enter）。


ビュー、テキスト、スタック...


ContentView.swift を選択すると、次のようになります。

struct ContentView: View {
    var body: some View {
        Text(", !")
    }
}

ここで、ContentView>の本文に「Hello、Habr！」を表示するテキストがあることがわかります。。別のテキストを追加してみましょう。body viewに直接追加しようとすると、Xcodeは誓います。これを行うには、まずHStackまたはVStackを使用します。コマンドを押し、テキスト（⌘コマンド+テキスト）をクリックして、「HStackに埋め込む」を選択します。



このコマンドは、HStackを自動的に作成し、そこにテキストを追加します。これで、別のテキストを追加できます。

struct ContentView: View {
    var body: some View {
        HStack {
            Text(",")
            Text("!")
        }
    }
}

Swiftuiインスペクター


「⌘Command+ Text」 をクリックし、「Show SwiftUI Inspector ...」を選択すると、テキストプロパティを視覚的に変更できるインスペクターが開きます。たとえば、インスペクタで「フォント」が「大きなタイトル」に変更された場合、コードも.font（.largeTitle）に変更されます。非常にシンプル：



何 状態？


何を理解するために 状態アクションを追加しましょう。以前に行った処理を繰り返します。「+ Command + HStack」、「Embed in VStack」を選択して、Button をこのVStackに追加します。

struct ContentView: View {
    @State var tapCount = 0
    var body: some View {
        VStack {
            HStack {
                Text(",")
                    .font(.largeTitle)
                    .foregroundColor(Color.green)
                Text("!")
                    .font(.largeTitle)
                    .fontWeight(.bold)
            }
            Button(action: {
                self.tapCount += 1
            }) {
                Text("Tap count \(tapCount)")
                    .font(.title)
            }
        }
    }
}

ボタンを押すたびに、tapCountの値が変更されます。しかし、それはどういう意味ですか状態？値が変更されたときにビューを再度レンダリングしたい場合は、状態。この場合、我々はボタンをクリックしたとき、我々は変更tapCountとして示され、状態。そのため、クリックするたびに体が再びレンダリングされます。
