Java 13では、「テキストブロック」を追加したい

テキストのブロックは、Java 13でリリースされる予定です。これは、JEP 355で発表されました。

テキストのブロックは複数行の文字列リテラルであり、ほとんどの特殊文字をエスケープする必要がなく、自動的に改行されます。

これは、JEP 326から始まる研究のさらなる試みです（未処理の文字列リテラルがリコールされました）。



JEP 355ゴール


  • エスケープシーケンスを回避することにより、Javaコードの記述を簡素化します。
  • コードの読みやすさの向上。

JEP 355が確実に達成しようとしていないこと


  • これは、新しい参照タイプをとして定義する試みではありませんjava.lang.String。
  • これは、文字列演算子「+」を上書きする試みではありません。
  • テキストのブロックはライン補間をサポートしていません。この機能は、後続のJEPで追加される可能性があります。


動機


XML、HTML、JSONなどを埋め込むためのJava。通常、シールドと文字列の連結を伴う大幅な編集が必要です。多くの場合、フラグメントは読みにくく、保守が困難です。

したがって、新しいJEPは読みやすさの両方を向上させ、さまざまな「行」からなる行を使用して、視覚的な混乱のないJavaプログラムの幅広いクラスを作成する機能を追加します。文字。実際、これはテキストの2次元ブロックであり、1次元の文字シーケンスではありません。

構文と説明


行のブロックは、右側と左側の"""両方"""にフレームが表示されます。ブロックの内容は、の後の最初の文字で始まり、"""前の最後の文字で終わり"""ます。三重引用符は、これらがテキスト行であることを明確にするために選択されていますが、通常の文字列リテラル（"..."）とは区別されています。

ブロックには"、スラッシュ（\）なしで引用符（）を直接含めることができます。およびを使用できます\"が、これはお勧めできません。

行の折り返しは自動的に行われます。\ nの使用は許可されていますが、お勧めできません。

"""
line 1
line 2
line 3
"""

同様に 

"line 1\nline 2\nline 3\n"

または 

"line 1\n" +
"line 2\n" +
"line 3\n"

空のテキストブロックの例を次に示します。

String empty = """
""";

以下は、テキストのブロックを使用する際の悪い習慣です。

           ";

           abc \ def
           """;

テキストブロック内のエスケープシーケンス


エスケープシーケンスが解釈されます。つまり、開発者は、\nブロック内などにエスケープシーケンスを記述できます。

例


HTML


String html = """
              <html>
                  <body>
                      <p>Hello, world</p>
                  </body>
              </html>
              """;

古い方法
String html = "<html>\n" +
              "    <body>\n" +
              "        <p>Hello, world</p>\n" +
              "    </body>\n" +
              "</html>\n";


SQL


String query = """
               SELECT `EMP_ID`, `LAST_NAME` FROM `EMPLOYEE_TB`
               WHERE `CITY` = 'INDIANAPOLIS'
               ORDER BY `EMP_ID`, `LAST_NAME`;
               """;

古い方法
String query = "SELECT `EMP_ID`, `LAST_NAME` FROM `EMPLOYEE_TB`\n" +
               "WHERE `CITY` = 'INDIANAPOLIS'\n" +
               "ORDER BY `EMP_ID`, `LAST_NAME`;\n";


脚本


ScriptEngine engine = new ScriptEngineManager().getEngineByName("js");
Object obj = engine.eval("""
                         function hello() {
                             print('"Hello, world"');
                         }
                         
                         hello();
                         """);

古い方法
ScriptEngine engine = new ScriptEngineManager().getEngineByName("js");
Object obj = engine.eval("function hello() {\n" +
                         "    print('\"Hello, world\"');\n" +
                         "}\n" +
                         "\n" +
                         "hello();\n");
