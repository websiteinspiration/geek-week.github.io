Perpustakaan ASE untuk Simulasi Atom: Membuat partikel nano

Perpustakaan ASE



Pustaka ASE adalah pustaka python untuk manipulasi dan perhitungan atom. Pada artikel ini, kita akan membuat partikel nano menggunakan perpustakaan ini.



Instalasi



ASE dipasang secara standar melalui pip: pip install ase.



Objek Atom



Objek utama yang dapat dimanipulasi adalah objek Atom. Itu dapat menyimpan berbagai struktur. Mari kita coba membuat molekul karbon monoksida: 



from ase import Atoms
from ase.visualize import view
d = 1.1
co = Atoms('CO', positions=[(0, 0, 0), (0, 0, d)])
view(co)


Sebagai hasilnya, kami memperoleh struktur dari bentuk berikut:







Cluster



Membuat cluster - partikel yang memiliki kisi kristal - dimungkinkan menggunakan modul ase.cluster. Cluster dihasilkan dengan memotong potongan-potongan dari kristal yang tak terbatas ke pesawat kliping. Buat kluster tembaga: 



from ase.cluster.cubic import FaceCenteredCubic
surfaces = [(1, 0, 0), (1, 1, 0), (1, 1, 1)]
layers = [6, 9, 5]
lc = 3.61000
atoms = FaceCenteredCubic('Cu', surfaces, layers, latticeconstant=lc)
view(atoms)


di mana permukaan memotong pesawat, lapisan adalah jumlah lapisan.



Kami mendapatkan struktur:







Partikelnano



Dan di sini kita sampai pada topik utama artikel. Partikelnano adalah partikel dari orde nanometer dengan diameter (hingga ratusan nanometer). Yang paling umum adalah nanopartikel bola - kami akan menghasilkan mereka. Bagaimana cara melakukannya? Mari kita buat sebuah cluster besar, dan kemudian potong sebuah bola darinya - berdasarkan jarak koordinat atom dari pusat. Untuk melakukan ini, pertama-tama Anda harus memusatkan semua koordinat: 



surfaces = [(1, 0, 0), (1, 1, 0), (1, 1, 1)]
layers = [50, 50, 50]
lc = 3.61000
atomsCu = FaceCenteredCubic('Cu', surfaces, layers, latticeconstant=lc)

possCu = atomsCu.get_positions()
cCu = atomsCu.get_center_of_mass()

new_possCu = []
for el in possCu:
    new_possCu.append(el - cCu)
new_possCu = np.array(new_possCu)


Jarak atom terpusat dari pusat dihitung secara sederhana: 



def get_radius_centered(pos):
    return (pos[0]**2 + pos[1]**2 + pos[2]**2)**0.5


Sekarang kami menulis fungsi untuk memotong partikel bola kami (lebih tepatnya, posisinya) dari sebuah kelompok besar: 



def cut_atoms(poss, r):
    poss_cut = []
    for pos in poss:
        if get_radius_centered(pos) < r:
            poss_cut.append(pos)
    poss_cut = np.array(poss_cut)
    return poss_cut


Tetap menerapkan hal ini: 



r = 5
possCu = utils.cut_atoms(new_possCu, r)
atoms = Atoms('Cu'+str(len(possCu)), positions=possCu)
view(atoms)


Kami mendapatkan gambar nanopartikel tembaga berikut dengan radius 5 angstrom (0,5 nm):







Struktur inti shell



Dalam praktiknya, struktur cangkang inti sering menarik - dengan inti dari satu jenis atom dan kulit lainnya. Misalnya, dalam kehidupan nyata, partikel cangkang inti CuAg (dengan inti Cu) mungkin memiliki aktivitas katalitik yang sebanding dengan Ag, tetapi lebih murah. Kita akan belajar bagaimana menghasilkan struktur seperti itu.



Kami akan mematuhi strategi berikut: kami memotong lubang dari partikel dan mengisi lubang ini dengan partikel dengan jenis atom lain. Untuk melakukan ini, Anda memerlukan fungsi yang membuat lubang: 



def cut_atoms_inside(poss, r):
    poss_cut = []
    for pos in poss:
        if get_radius_centered(pos) > r:
            poss_cut.append(pos)
    poss_cut = np.array(poss_cut)
    return poss_cut


Buat kluster perak dengan koordinat atom terpusat: 



lc_ag = 4.09
atomsAg = FaceCenteredCubic('Ag', surfaces, layers, latticeconstant=lc_ag)

possAg = atomsAg.get_positions()
cAg = atomsAg.get_center_of_mass()

new_possAg = []
for el in possAg:
    new_possAg.append(el - cAg)
new_possAg = np.array(new_possAg)


dan membuat partikel core-shell: 



r1 = 7
r2 = 5
possAg = utils.cut_atoms(new_possAg, r1)
atoms = Atoms('Ag'+str(len(possAg)), positions=possAg)

possAg = utils.cut_atoms_inside(possAg, r2)
possCu = utils.cut_atoms(new_possCu, r2)
poss = np.vstack([possCu, possAg])

atoms = Atoms('Cu'+str(len(possCu))+'Ag'+str(len(possAg)), positions=poss)
view(atoms)

atoms.write('Cu'+str(r1)+'Ag'+str(r1-r2)+'.xyz')


Relaksasi



Dan akhirnya, contoh manipulasi yang membuat perpustakaan dibuat. Santai partikel inti sel, mis. kami membawa ke keadaan dengan energi potensial minimal. Atur kalkulator: 



from ase.calculators.emt import EMT
atoms = Atoms(atoms, calculator=EMT())


dan mulai relaksasi: 



dyn = BFGS(atoms, trajectory='atoms.traj')
dyn.run(fmax=0.05)
