Pour de nombreux modèles d'épidémie - SIR, SEIR, etc. (pour plus de détails sur la description mathématique, voir, par exemple, www.idmod.org/docs/hiv/model-compartments.html ), l'énoncé suivant est vrai: au stade initial de l'épidémie, lorsque le nombre de personnes infectées (I ) est beaucoup plus petite que la taille de la population, le taux de croissance du nombre de cas est proportionnel au nombre de cas:

∂I/∂t=βIoù β est le coefficient caractérisant le taux d'infection.

La solution de cette équation est une fonction exponentielle. Pour la fonction exponentiellef(t)=at L'équation fonctionnelle suivante est vraie:

f(t+loga⁡2)=2f(t)


À. nombreloga⁡2 est la période de doublement d'une fonction f(t)=at. Par définition, si la période de doublement pour une fonction lisse non décroissante est constante, alors la fonction est exponentielle.

En relation avec ce qui précède, il est proposé d'utiliser une période de doublement comme indicateur applicable pour l'évaluation visuelle de fonctions similaires à celle indicative. Dans le cas général, pour une fonction augmentant de façon monotonef(t) période de doublement D(t) peut être déterminée à partir de l'équation fonctionnelle suivante:

f(t+D(t))=2f(t)


Différence D(t) de constante indique la différence f(t)de l'exposant. Par rapport à la dynamique des taux d'incidence, la croissanceD(t)(idéalement - à l'infini) indique l'efficacité des mesures prises pour contenir l'épidémie.
