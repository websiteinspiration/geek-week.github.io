あなたはユーザーにあなたの新しいアプリケーションに登録する機会を与えました、彼らはそこで彼らのパスワードを入力することができます、そしてあなたはこのパスワードを処理する方法を非常によく知っています。

それで？

次に、ユーザーがアプリケーションにログインできるようにする必要があります。これを行うために、PHPには、ハッシュされたシーケンスと一致するパスワードをチェックする組み込み関数があります。この関数はと呼ばれpassword_verify()ます。それはこのように動作します：

$hash = '$2y$07$BCryptRequires22Chrcte/VlQH0piJtjXl.0t1XkA8pw9dMXTpOq';
if (password_verify('rasmuslerdorf', $hash)) {

    echo '  !';

その結果、条件付き演算で使用できる論理値が得られます。したがって、ユーザーをアプリケーションに入れるか、問題が発生したことをユーザーに通知できます。
