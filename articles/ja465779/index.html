Yealink T19自動プロビジョニング+動的アドレス帳

私がこの会社で働くようになったとき、IPデバイスにいくつかのベースがあり、アスタリスクとFreeBPXの形のスプラッシュを備えたいくつかのサーバーがありました。さらに、Samsung IDCS500アナログ電話交換機が並行して機能し、一般的には会社の主要な通信システムでした。ipテレフォニーは販売部門でのみ機能しました。そして、すべてがどんどん沸騰していたでしょうが、ある晴れた日に、全員をIPテレフォニーに移行する命令が与えられ、日付が合意され、機器が購入され、企業を21世紀に移行する計画が実施され始めました。
この状況で最初に気になるのは、なんとかして管理する必要がある電話の数が急速に増加していることです。2番目に非常に心配していたのは、電話帳でした。エンドポイントマネージャー（ちなみに、最新バージョンのFreePBXから切り取られたもの）が最初のマネージャーの助けになるとしたら、この本でいくつかの疑問が生じました。

  • 最初に、ユーザーの場所/売上高を常に変更して、その精度をどのように保証するのですか？
  • 次に、電話を完全に個人化解除する方法。そして、毎回連絡先の名前を記入しないのですか？

タスクは面白く、解決策は間もなく実現しました。次に、完全なリストを示し、その後、順に分析します。

from scapy.all import sniff
from scapy.layers.inet import IP
import mysql.connector
import ldap
import getpass

import os
import time
from string import replace

def conn_ldap(login):
    ad = ldap.initialize('ldap://***.local')
    ad.simple_bind_s('voip@***.local', 'password')
    basedn = 'OU=IT,DC=***,DC=LOCAL'
    basedn_user = 'OU=***,OU=***,DC=***,DC=LOCAL'
    scope = ldap.SCOPE_SUBTREE
    filterexp = "(&(sAMAccountName=" + login + ")(ObjectClass=person))"
    filterexp2 = "(&(ObjectClass=organizationUnit))"
    attrlist = ['cn']
    attrlist2 = ['OU']
    search = ad.search_s(basedn, scope, filterexp, attrlist)
    adname = search[0][1]['cn'][0].decode('utf-8')
    if adname == ' ':
        search = ad.search_s(basedn_user, scope, filterexp2, attrlist2)
        for i in range(1, len(search)+1):
            group = search[i][1]['ou'][0]
            basedn_user2 = 'OU='+group+','+basedn_user
            search = ad.search_s(basedn_user2, scope, filterexp, attrlist)
            adname = search[0][1]['cn'][0].decode('utf-8')
            if adname != ' ':
                return adname
        adname = search[0][1]['cn'][0].decode('utf-8')
    ad.unbind_s()
    return adname


def tftp_file_change(config,place,adname,current_account,current_account_password):

    client = tftpy.TftpClient("192.168.0.3", 69)
    client.download('template.cfg', place)
    fileread = open(place, 'r')
    line = fileread.readlines()
    fileread.close()
    line[5] = (('account.1.label = ').encode('utf-8') + adname.encode('utf-8') + '\n')
    line[2] = (('account.1.auth_name = ').encode('utf-8') + current_account.encode('utf-8') + '\n')
    line[3] = (('account.1.display_name = ').encode('utf-8') + current_account.encode('utf-8') + '\n')
    line[6] = (('account.1.password = ').encode('utf-8') + current_account_password[0][0] + '\n')
    filewrite = open(place, 'w')
    for i in line:
      filewrite.write(i)
    filewrite.close()
    print place
    print config
    client.upload(config,place)

def sniff_frame():

これを行うために、テンプレート構成は事前構成されたtftpサーバーからダウンロードされ、そこに変更を加えてmac.cfgとして保存します。つまり、Yealinkの設定には2つのタイプがあります。1つはグローバルで、もう1つは特定の電話に適用され、形式はmac_phone.cfgである必要があり

ます。ファイルにすべての変更を加え、それをtftpサーバーに保存してから、デバイスにプロビジョニングおよび再起動するためのコマンドを電話に与えます。
