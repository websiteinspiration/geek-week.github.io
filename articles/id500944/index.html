WexLang Reader - cara membaca buku dalam bahasa asing dengan senang hati (104 bahasa)

Halo!

Saya sudah menulis di sini tentang proyek saya - Wexlang. Proyek telah diperbarui dan sekarang merupakan aplikasi desktop gratis.

Aplikasi ini memiliki 104 bahasa dan arah terjemahan apa pun di antara bahasa yang tersedia.



Sedikit tentang proyek lama: Aplikasi

web Wexlang ditinggalkan karena beberapa masalah dengan fungsi, kompleksitas dan biaya pemasaran yang tinggi, dll.

Tetapi saya tidak ingin membuang proyek, jadi diputuskan untuk menyelesaikan fungsionalitas, memperbaiki kesalahan dan mengemas proyek dalam aplikasi desktop.

Pembaca WexLang


Saya menggunakan bahasa Inggris terutama untuk membaca dan korespondensi, sehingga proyek ini bertujuan untuk mengembangkan keterampilan khusus ini.

Aplikasi ini ditulis dalam php + laravel + sqlite + jQuery dan dikemas dalam aplikasi desktop berkat proyek PHP Desktop .

Kemampuan


  • 104 bahasa dan semua arah terjemahan
  • statistik terperinci tentang teks dan setiap halaman teks, statistik frekuensi penggunaan setiap kata
  • teks apa pun dapat diunduh, tidak ada batasan hak cipta, dll.
  • Anda dapat menandai semua kata pada halaman sebagai familiar / terpelajar
  • pengolah kata bekerja dengan kata kerja phrasal dan dengan frasa apa pun
  • Anda dapat mengaktifkan / menonaktifkan penyorotan untuk kata-kata baru dan yang dipelajari
  • ini adalah aplikasi desktop, jadi Anda tidak perlu khawatir bahwa sesuatu akan berhenti berfungsi, Anda dapat membuat cadangan dari basis kata Anda
  • impor dan ekspor kata-kata

Cara bekerja dengan aplikasi tersebut


Setiap kata dalam teks diberikan satu dari tiga status:

  • baru - disorot dengan warna biru di teks
  • dipelajari - disorot dalam oranye dalam teks
  • akrab - tidak menonjol dalam teks

1. Pertama Anda mengunduh teks dalam bahasa asing



2. Setiap teks memiliki statistik terperinci. Jumlah kata-kata baru, akrab, dipelajari, serta daftar semua kata-kata unik.

Setiap kata menunjukkan frekuensi penggunaan - berapa kali kata tersebut muncul dalam teks dan berapa persen jumlah total kata.

Dalam statistik teks, Anda dapat menelusuri kata-kata untuk mengatur kata-kata status yang diinginkan - akrab atau belajar. Anda juga dapat mengekspor daftar kata.



3. Setelah mengunduh teks, Anda dapat mulai membaca.



Opsi berikut tersedia di halaman bacaan teks:

  • sorot / jangan sorot kata-kata yang tidak dikenal
  • sorot / bukan sorot kata-kata yang dipelajari
  • atur semua kata pada halaman status untuk dipelajari
  • atur semua kata pada halaman status yang sudah dikenal
  • ketika Anda mengklik sebuah kata, itu diterjemahkan ke google. Jika kata sebelumnya diterjemahkan, terjemahan diambil dari database
  • perubahan status kata

Juga untuk setiap halaman ada statistik kata-kata. Daftar kata-kata unik dan berapa kali setiap kata muncul dalam teks. Di sini Anda juga dapat mengatur status kata-kata.



4. Pada halaman My Words , Anda dapat menambah, menghapus kata, mengedit terjemahan.



5. Dalam pengaturan aplikasi, Anda dapat memilih bahasa antarmuka, pasangan bahasa dan mengatur kunci API untuk google translate.



Transfer


Kata-kata diterjemahkan melalui google translate . Selama terjemahan, kata dan terjemahan disimpan dalam basis data, sehingga kata tersebut diterjemahkan di Google hanya sekali .

Ada 2 opsi:

  1. Gunakan terjemahan melalui terjemahan google gratis - dalam hal ini ada batasan dan Google dapat memblokir IP Anda untuk beberapa waktu. Apalagi jika Anda sering menggunakan fungsi "tandai semua kata sebagai."
  2. Gunakan Google translate API resmi. API hingga $ 10 per bulan gratis. Mengingat kata-kata tersebut disimpan dalam database, Anda tidak perlu membayar untuk API sama sekali atau jumlahnya akan sangat kecil.

Isu saat ini


  • Terjemahan dari google tidak terlalu bagus.
  • Fungsionalitas tidak bekerja dengan baik dengan bahasa Asia, sampai saya tahu bagaimana cara memperbaikinya, karena tanpa petunjuk bagaimana bahasa dengan hieroglif disusun.

Rencana


Ada rencana untuk mengulang fungsi terjemahan dan kamus sedikit, menambah kemampuan untuk menambahkan bagian-bagian berbeda dari pidato dan terjemahan untuk setiap kata ke kata, dan memuat nilai kata dari kamus penjelas (dalam bahasa asing).
