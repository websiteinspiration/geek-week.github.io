  ---: ;
}

Variablen fÃ¼r den Planeten werden wie folgt deklariert:

Wie Sie sehen kÃ¶nnen, werden Variablen fÃ¼r das Bild, den Neigungswinkel, die Dauer des Tages und die Farbe des Planeten deklariert.

    }
