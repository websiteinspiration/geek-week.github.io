CSSとJavascript Antカルーセル

CSS3の登場により、jQueryなどのJSライブラリを使用せずにアニメーションを作成できるようになりました。CSS3遷移プロパティを使用すると、要素の他のプロパティ（幅、高さ、マージン、不透明度など）をスムーズに変更し、変換の時間と法則をパラメーターとして設定できます。純粋なJavascriptで小さいながらも非常に機能的なカルーセルを提供しています。アリのように小さく、フクロウよりずっと小さい。しかし、ほぼ同じ機能を備えています。

Antカルーセルを使用すると、次のことができます。

  • 1つ以上の要素を表示します。
  • 要素は有限または無限のリボン（ループ内）の形で表示できます。
  • 自動スクロール項目。
  • ナビゲーションは、矢印、インジケーターポイント、または裏返し（触覚画面の場合）によって実行されます。
  • 画面の幅に合わせて自動的に調整されます。

カルーセルをindex.htmlファイルに配置します（以下のサンプルファイルを参照）。

HTML
<div class="ant-carousel">
  <div class="ant-carousel-hider">
    <ul class="ant-carousel-list">
      <li class="ant-carousel-element"><img src="images/img1.jpg" alt="1"> <p> 1</p> </li>
      <li class="ant-carousel-element"><img src=" images /img2.jpg" alt="2"> <p>2</p> </li>
	…
      <li class="ant-carousel-element"><img src=" images /imgN.jpg" alt="N"> <p> N</p> </li>
    </ul>
  </div>
  <div class="ant-carousel-arrow-left"></div><div class="ant-carousel-arrow-right"></div>
  <div class="ant-carousel-dots"></div>
</div>


ここでは<ul> <li>要素を使用していますが、必要に応じて<div> <div>を代わりに使用できます。矢印とインジケーターポイントは、それぞれのコンテナーに絶対的に配置されます。矢印の場合は、三角括弧の形式のパターンが使用されます。必要に応じて、パターンまたは画像生成に置き換えることができます。疑似要素：beforeおよび：afterを使用できます。

3つの可視要素と270ピクセルの要素幅を持つカルーセルを作成します。次に、最大カルーセル幅は810ピクセルです。CSSファイルを含めます。

ant-carousel-list コンテナーに要素を配置し、display：flexプロパティを設定して、すべての要素をjustify-content：flex-startの左端に押します。flexプロパティ：0 0 autoはflex-shrinkを0に設定します（デフォルトは1）。カルーセル要素のスクロールは、transitionプロパティを使用して、マージンの左インデントをゼロから要素の幅（一方向）または要素の幅からゼロ（他の方向）にスムーズに変更することによって実行されます。変換（スクロール）関数では、easeの値が使用されます。

プログラムに移ります。プログラムオプションでは、以下を構成できます。

  • 可視要素の数;
  • ( );
  • ;
  • ;
  • ;
  • / : , , ( ).

プログラムの初期化は、カルーセル要素の数が決定され、初期値が内部変数に割り当てられ、イベントハンドラーが矢印とポイント（接続されている場合）に割り当てられることから始まります。自動スクロールオプションが有効な場合、カルーセル要素の上にマウスを置くとスクロールを停止する追加のハンドラーが割り当てられます。指が画面に触れるポイントと指が画面から離れるポイントの間に20ピクセル以上あり、指が画面に触れる合計時間が80ミリ秒未満の場合、タッチスクロールがトリガーされます。著者はまだこのカルーセルを操作した経験があまりないので、おそらく与えられた値を明確にする必要があります。スクロールハンドラーのより信頼性の高い操作のために、ポイント間の距離は10ピクセルまたは15ピクセルに減らす価値があります。タッチ時間は100または120 msに増加します。このカルーセルのユーザーは、使用の経験を積んだ後、これらの値を自分で修正できます。

要素のスクロールアルゴリズムは、ループオプションが有効かどうかによって異なります。有効にした場合、右（にスクロールするとき、elemPrevの機能）、マージン左のプロパティの行全体this.crslList要素は負の値にゼロから減少は、の幅に等しいelemWidthの要素。同時に、右側の最後の要素が複製され、最初の要素の前に挿入されます。その後、最後の要素が削除されます。この行には、プロパティ'transition：margin' + options.speed + 'ms ease'が割り当てられています。ここで、options.speedはアニメーションの速度、easeはアニメーションの機能です。これでスクロールできます。余白のプロパティ負の値からゼロにスムーズに変化し、ルーラー全体がスムーズに右にシフトし、最後にあった要素が最初に来ます。options.speedマイクロ秒の後、行には'transition：none'と同じ値が与えられます。

var elm, buf, this$ = this;
elm = this.crslList.lastElementChild;
buf = elm.cloneNode(true); this.crslList.insertBefore(buf, this.crslList.firstElementChild);
this.crslList.removeChild(elm);
this.crslList.style.marginLeft = '-' + this.elemWidth + 'px';
window.getComputedStyle(this.crslList). marginLeft;
this.crslList.style.cssText = 'transition: margin '+this.options.speed+'ms ease;';
this.crslList.style.marginLeft = '0px';
setTimeout(function() {
	this$.crslList.style.cssText = 'transition: none;'
}, this.options.speed);

同時にn個の要素をスクロールする必要がある場合、要素の置換はループでn回実行され、マージンの左距離はn回増加します。

左へのスクロール（elemNext関数）は、逆の順序で行われます。最初に、'transition：margin' + options.speed + 'ms ease'プロパティがthis.crslListルーラーに割り当てられ、ルーラーが左にスムーズにスクロールします（crslList.style.marginLeft = '-' + elemWidth + 'px'）。次のoptions.speedマイクロ秒。最初の要素が複製され、定規の最後に挿入されます。その後、最初の要素が削除されます。ルーラーは値 'transition：none'を返します。同時にn個の要素をスクロールする必要がある場合、前の場合と同様に、要素の再配置はループ内でn回実行され、マージンの左距離はn回増加します。

var elm, buf, this$ = this;
this.crslList.style.cssText = 'transition: margin '+this.options.speed+'ms ease;';
this.crslList.style.marginLeft = '-' + this.elemWidth + 'px';
setTimeout(function() {
	this$.crslList.style.cssText = 'transition: none;';
	elm = this$.crslList.firstElementChild;
	buf = elm.cloneNode(true); this$.crslList.appendChild(buf);
	this$.crslList.removeChild(elm)
	this$.crslList.style.marginLeft = '0px'
}, this.options.speed);

ループオプションが無効になっている場合、この場合、要素の順列はなく、要素の行全体が全体として左または右に端点にシフトされます。要素の行this.crslListプロパティ'transition：margin' + options.speed + 'ms ease'は、カルーセルが初期化されて削除されなくなった場合でも割り当てられます。

カルーセルは、ant-carouselクラスの名前または識別子によって呼び出されます。2番目のケースでは、1つのページに複数のカルーセルを配置できます。カルーセルのindex.htmlファイルは次のようになります。

</head>
<body>
	…
	<div class="ant-carousel">
		<!--    -->
		…
	</div>
	…
	<footer>
	…
	</footer>

1つのページに複数のカルーセルを配置するには、IDでカルーセルを呼び出す必要があります。カルーセルごとに要素の数が異なる場合があります。

<div class="ant-carousel" id=”first”>
	<!--   -->
	…
<div class="ant-carousel" id=”second”>
	<!--   -->
	…
<script>new Ant("first"); new Ant("second");</script>
