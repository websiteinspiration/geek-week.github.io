1. Menerapkan keamanan manajemen sumber daya di Azure


Hanya dalam 3-4 jam, Anda akan belajar cara melindungi sumber daya dengan kebijakan, kontrol akses berbasis peran, dan layanan Azure lainnya.

Di antara modul kursus:

  • Keselamatan, Tanggung Jawab, dan Kepercayaan pada Azure
  • Menegakkan dan memantau standar infrastruktur dengan Kebijakan Azure
  • Memantau dan mengatur sumber daya Azure dengan Manajer Sumber Daya Azure
  • Mengamankan Sumber Daya Azure dengan Kontrol Akses Berbasis Peran (RBAC)

Anda dapat memulai pelatihan di sini

2. Identitas dan manajemen akses di Azure


Dalam 5-6 jam, Anda akan belajar cara bekerja dengan langganan, pengguna, dan grup dengan mengonfigurasi Microsoft Azure Active Directory untuk beban kerja.

Modul Kursus:

  • Keselamatan, Tanggung Jawab, dan Kepercayaan pada Azure
  • Buat Akun Azure
  • Kelola pengguna dan grup di Azure Active Directory
  • Buat pengguna dan grup Azure di Direktori Aktif Azure
  • Mengamankan aplikasi Anda dengan OpenID Connect dan Azure AD
  • Lindungi Pengguna Direktori Aktif Azure dengan Otentikasi Multi-Faktor
  • Bolehkan pengguna untuk mereset kata sandi mereka dengan menggunakan kata sandi yang diset ulang sendiri di Azure Active Directory

Mulai pelatihan

3. Menerapkan keamanan jaringan di Azure


Sekitar 6 jam belajar dan Anda akan belajar cara mengatur, melindungi, dan mengisolasi jaringan di Azure. Pelajari lebih lanjut tentang keamanan dan kepercayaan pada Azure, pengaturan jaringan untuk mesin virtual, pengamanan dan isolasi akses ke sumber daya Azure menggunakan grup keamanan jaringan dan titik akhir layanan, enkripsi end-to-end lalu lintas jaringan menggunakan Azure Application Gateway, pemantauan dan pemecahan masalah infrastruktur jaringan yang kompleks Azure dengan alat pemantauan jaringan. Bergabung sekarang!

Mulai pelatihan
