正月が近づいています。怠惰な人だけが2020年の傾向について書いていないので、来年の最も重要なイベント-2019を記録することにしました。Javaの実践とVoronezhのReksoft開発センターのフロントエンドから、開発の世界でTOP-7イベントを保管してください。

です。1. Nginxとランブラーのケース

開発者として、私たちはイゴール・シソエフの発案による状況を監視せざるを得ません。このような紛争は、オープンソースソリューションの開発とIT業界全体に大きな影響を与えると確信しています。彼らは刑法に頼ることなく解決されなければならない。そのようなニュースは、開発者に何か新しいものを作成する動機を与えます。疑問や懸念があります。しかし、この種の人気のある広く使用されているライブラリや製品がすべて自宅で作成されたわけではありません。先例が作成されると、他の会社で同様のケースの連鎖を開始する可能性があります。

これは来年の唯一の否定的な出来事ですが、それを回避することはできませんでした。

2. Java 13出力

Javaの新バージョンのリリースが5年間の出来事だった時代を今も記憶に留めています。しかし、すべてが変化しています。オラクルは半期リリースに切り替えただけでなく、この傾向を確実に統合しました。今年の2つのリリースと現在の現在の13番目のバージョンは、その証拠です。一方では、このようなリリースの頻度により、言語はより近代的で、関連性があり、学ぶことが望ましいものになります。一方、レアリリースもJavaの強みでした。安定性と信頼性は、開発するテクノロジーを選択する際の意思決定に影響を与える2つの主要な要素です。そして今、新しいバージョンのサポート期間は大幅に減少しました。長いサポートのある最新リリースは11番目です。しかし、現代の現実は彼らの条件を決定します：技術開発のスピード、新しいソリューションの要求とプログラミング言語間の激しい競争には、確立された伝統の変更が必要です。

4. TypeScript出力3. *

TypeScriptはこの1年で大きな進歩を遂げました。この間、このテクノロジは、コンパイルの高速化、コード分離の簡素化、分析の改善に役立つ多数の便利な機能を獲得し、言語の機能を使用する新しい方法を開きました。言語に追加された新しい演算子、オプションのチェーン、Nullish Coalescingなどに気づくのは難しいことではありません。ドラフト形式でのみ交渉されているものはすべてTypeScriptですでに利用可能です。

タイピングが弱い言語で書かれた企業レベルのプロジェクトを想像するのは難しいです。重要なのは開発速度ではなく、元の製品の信頼性であるため、ますます多くの開発者がコンパイル済み言語を選択しています。

5. React v16.8出力

現在、ほとんどすべての開発者がこのライブラリの使用方法を知っており、重要なことは知っています。そのような人気は、その背後にあるアイデアによって推進されています。コンポーネントアプローチ、同型、速度、および作業の容易さにより、Reactは絶対的な勝者となります。

過去1年間で多くの更新がありましたが、特にフックを選びたいと思います。開発者は、ライブラリにステートフルコンポーネントの再利用を許可することでいくつかの問題があることを理解しました。コミュニティはさまざまなパターンを使用してそれらを解決しましたが、多くの場合、これはコードをより混乱させただけです。
フックは、コードを非常に変換する単純なアイデアを実装することにより、この状況を完全に修正し、コードの読み取りと認識がいかに簡単かを示しました。来年、図書館がどのように変化するかを知ることは非常に興味深いでしょう。

6. Raspberry PI 4の終了

今日のプログラミングは、ますますマイクロコントローラー、ウェアラブルデバイス、IoTに向かっています。この世界のRaspberry PIは真ん中のどこかにあります-本格的なコンピューターですが、手のひらに収まります。最初のバージョンから始まって、このシングルボードプレーヤーは新しい可能性の範囲を開きました。今ではすでにクアッドコアプロセッサになっていますが、ついに4kモニターをサポートする最新のusb-cが登場しました。以前のバージョンのRaspberry PI 4と比較すると、ライン開発の飛躍的な進歩です。また、低価格で十分な価格のため、このコンピュータは専門家向けだけでなく、トレーニングや家庭での実験にも利用できます。
