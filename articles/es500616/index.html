Seminario web Fujitsu y Microsoft Joint Fuinitsu PRIMEFLEX


Hola Habr!

El jueves 14 de mayo de 2020, a las 12:00 hora de Moscú, lo invitamos a unirse al seminario web conjunto de Fujitsu y Microsoft: “Fujitsu PRIMEFLEX Family. Solución de implementación de infraestructura hiperconvergente con Microsoft Azure Stack HCI " .

Regístrese hoy

La creación y el desarrollo de la infraestructura del centro de datos, la introducción de innovaciones, el aumento constante de los requisitos para las tecnologías utilizadas lleva mucho tiempo, supone la presencia de amplias competencias, recursos y fondos significativos.
El enfoque tradicional, que consiste en combinar varios componentes para resolver problemas similares, es un proceso complejo y lento, que a menudo puede crear riesgos adicionales para la empresa.

Los sistemas integrados Fujitsu PRIMEFLEX ofrecen al mercado un enfoque diferente: ¡una solución eficiente y racional lista para usar!

Fujitsu PRIMEFLEX es una familia representativa y en constante crecimiento de sistemas potentes, preconfigurados y probados. Cada sistema se entrega como un complejo único, que incluye servidores y software preinstalado, lo que reduce significativamente la complejidad de la implementación y operación posteriores.

PRIMEFLEX para Microsoft Azure Stack HCI es un sistema integrado que incluye todo el hardware y software necesarios para implementar una infraestructura de TI hiperconvergente basada en tecnologías de Microsoft. Esta solución utiliza servidores Fujitsu PRIMERGY x86 de alto rendimiento energéticamente eficientes, servidor definido por software y tecnologías de almacenamiento (Hyper-V y Storage Spaces Direct), integrados en Windows Server 2019, así como servicios híbridos de Azure integrados en Windows Admin Center, que proporcionan una conexión fácil servicios empresariales locales a la nube.

En el seminario web aprenderá:

  • ¿Qué son los sistemas integrados Fujitsu PRIMEFLEX?
  • ¿Para qué sirve Azure Stack HCI y para qué sirve?
  • Escenarios de aplicación Fujitsu PRIMEFLEX para Azure Stack HCI
