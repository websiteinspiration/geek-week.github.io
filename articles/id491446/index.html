Cara "meretas" RedBull

Bahkan, lebih tepat untuk menyebut artikel "Cara mendapatkan poin dalam kompetisi untuk memenangkan kulkas RedBull secara keseluruhan". Ngomong-ngomong, kami sudah memiliki lemari es dengan minuman di kantor kami.



Ketika sekali lagi jus, soda, dan minuman lain dikirim ke kantor, saya menemukan selebaran yang mengiklankan kompetisi, intinya adalah menyelesaikan permainan dan pemenangnya akan memenangkan kulkas untuk minuman energi.

Kulkas kedua tidak akan menyakiti kita - kita hanya memperluas, dan kita memiliki ruang kantor baru. Jadi saya mengikuti tautan dan memutuskan untuk mencoba keberuntungan saya, usaha saya gagal dan saya berakhir di suatu tempat di akhir peringkat. Permainan itu harus melewati beberapa tingkat labirin secepat mungkin.

Sekarang Anda perlu memakai kacamata hitam, nyalakan vpn, semua jenis proxy, dll.

Mengetahui sepenuhnya bahwa setiap permainan di web adalah urutan permintaan, dan mengetahui pasar pengembangan, di mana ada banyak amatir, saya memutuskan untuk mencoba keberuntungan saya ke arah lain dan tiba-tiba tersandung ke dalam lubang dalam permainan. Bukan berarti saya sangat membutuhkan kulkas ini, hanya saja ada minat olahraga.

Langkah pertama adalah permintaan untuk memulai permainan:

https://rb-prjct-crtns-gamebackend-eu.herokuapp.com/api/high-scores/games/8970c40a-98f0-4395-85c8-66177fb38af0/runs

Pada saat yang sama, kami perhatikan di tautan id = 8970c40a-98f0-4395-85c8-66177fb38af0.
Beberapa kali kami me-refresh halaman di browser, membuka dari ip yang berbeda dan di browser lain, parameter ini tidak berubah, kami menganggap bahwa ini adalah pengidentifikasi konstan untuk permainan.
