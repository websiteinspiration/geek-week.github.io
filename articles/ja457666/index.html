Raspberry Piの代替

2012年2月29日私たちの世界は少し良くなりました。 Raspberry Pi Foundationは、Raspberry Piのブランド名で小型シングルボードコンピューターの販売を開始すると発表しました。販売当初、デバイスの購入は簡単ではありませんでしたが、少しずつ市場は飽和状態になり、2012年7月からようやく店舗に行って最初のMalinkaをインターネット経由で購入または注文できるようになりました。RaspberryPiモデルBは、700 MHzプロセッサー、0.5 GBのシングルボードLinuxコンピューターですRAM、USB、イーサネット、26本のGPIOピンにより、外部カード、入出力デバイス、アクチュエータを接続できます。比較的安価な4層ボードに基づいて構築されたこのデバイスは、エンドユーザー向けの価格が35ドルと低価格で、Raspberry PiモデルBのアルファ版がARM TechCon 2011のハードウェアデザイン候補に選ばれました。



コンピューターが開発され、家族は成長しました。メインブランチには、ますます強力なプロセッサと最新のハードウェアが搭載されています（たとえば、3B +モデルには、クロック周波数が1.4 GHz、1 GBのRAM、4つのUSBポート、Wi-Fi 802.11acおよびBluetooth 4.2のクアッドコア64ビットプロセッサがあります）。さらに、Raspberry Pi Zeroファミリはマザーボードを65×30 mmに縮小して開発され、わずか5ドル（Wi-Fi 802.11nを搭載したRaspberry Pi Zero Wのバージョンでは10ドル）でした。

最近、Raspberry Pi Foundationは、ミニコンピュータの新しいバージョンであるRaspberry Pi 4を発表しました。「4」は、SoC Broadcom BCM2711の一部として、クロック速度1.5 GHzのクアッドコア64ビットCortex-A72プロセッサを発表しました。 2つのモニター。 GPU VideoCore VIは、60 fpsで4Kビデオストリームを処理できます。最後に、RAMの容量（1、2、または4 GB LPDDR4 SDRAM）を選択できます。

何百万ものコピーで販売されているRaspberry Piは、ポータブルシングルボードコンピュータのまったく新しい市場と、数多くの代替設計の例を開拓してきました。Raspberry Pi 3B +を出発点として使用すると（Raspberry Pi 4はまだ購入できません。Quartetがすぐに利用可能になることを願っています）、その進化によって彼らが時々非常に珍しい環境ニッチに陥った競合他社を提供できます。そして、私たちが、バイヤーとして、この研究にいくら支払う必要があるか。競合するコンピューターの説明では、可能であれば、参加者が利用できるすべてのハードウェアの利点について言及します。これにより、説明はやや乾燥しますが、将来のプロジェクトにコンピューティングコアを選択することは間違いありません。

オレンジパイプライム


Raspberry Pi 3との違いは、2 GBのRAMと、SoC AllWinner H5に統合されたMali-450 GPUアクセラレータの存在であり、これにより2Kビデオを再生できます。興味深い機能の中でも、IRレシーバーの存在に注目する価値があります。ボードは、リモコンまたはIRLEDが組み込まれた携帯電話の一部のモデル（Redmi Note 7など）から制御できます。非標準の機器には、30 fpsで最大1080pのビデオストリームをサポートする組み込みのマイクとCSIビデオインターフェイスもあります。



サイズが98×60 mmのボードには、メモリカード用スロット（最大32 GB）、Wi-Fi 802.11 b / g / n、Bluetooth 4.0、ギガビットイーサネット、4つのUSB（3つのUSB 2.0ホストと1つのUSB 2.0 OTG）および40ピンGPIOコーム。 TTLレベルの個別に派生したUARTもあるので、ターミナルでLinuxブートの詳細を見ることができます。オーディオ機器からは、上記のマイクに加えて、HDMIにはリニア出力とオーディオ出力もあります。ビデオアクセラレータは、OpenGL ES 2.0およびOpenVG 1.1をサポートしています。サポートされているOSには、Ubuntu、Debian、Android 5.1があります。

一般的に、オレンジパイはラズベリーの最も成功した競争相手の1つであると言わざるを得ません。 Orangeはまた、Orange Pi 4G-IOTを含む、異種シングルボードコンピュータの全シリーズの先駆者となりました。（GSM / GPS / LTEモジュールを搭載）およびOrange Pi Zero（オプションでPoEから電力を供給できるため、別の電源を供給しなくてもイーサネットネットワークの離れたセグメントにドロップできます）。

バナナpi m3


オレンジと一緒に、中国の会社SinoVoipによって開発されたバナナは、ラズベリーの出現以来よく成長しているもう1つの「フルーツ」ブランチです。Raspberryウェイクに続いて、バナナは一貫してモデルBanana Pi M1、M1 Plus、M2 Plus、M2 Ultra、M2 Zero、M3をリリースしました。



現在の主力製品であるバナナPi M3は、8コアのAllwinner A83T SoC（ARM Cortex-A7プロセッサー、PowerVR SGX544MP1グラフィックスプロセッサー）に基づいて構築されており、最大1.8 GHzで動作し、2 GBのRAMと8 GBのフラッシュメモリの環境で動作します。ギガビットイーサネット、2つのUSB、Wi-Fi 802.11 b / g / n、Bluetooth 4.0およびHDMIに加えて、ボード上にSATAがあります。Orange Pi Primeと同様に、M3にはIRレシーバー、CSIビデオインターフェイス、デバッグUART、マイク、ライン出力、HDMIオーディオ出力があります。オレンジとは異なり、バナナには、タッチスクリーン用のI2Cと組み合わせたMIPI DSIディスプレイインターフェイスがあります。当然、40接点のGPIOコームがあります。

Rock64


Rock64シングルボードコンピュータには、4 GBのRAMがすでに搭載されており、64ビットARM Cortex A53に対応しています。ビデオサブシステムは、60 fpsの周波数で4Kストリームに対応できます。このデバイスはPOEを搭載しています。グラフィックサブシステムARM Mali 450MP2は、OpenGL ES 2.0、OpenVG1.1に対応しています。 Debian、Cent OS、Fedora、およびAndroid 8 OSはRock64に移植されました。一般に、このコンピューターの開発者や愛好家が多数のLinuxベースのOSを移植したことに注意してください。 Rock64には豊富で詳細なドキュメントがあり、活発で活発なコミュニティがあるため、ハードウェアの優れた仕様と節約価格を考慮に入れると、このシングルボードプレーヤーは、ハードウェアの要件が高まるプロジェクトでRaspberry Pi 3を置き換えるのに適しています。



Rock64開発者は64のGPIOピンを惜しみなく注ぎ、イーサネット信号をそれらに出力したので、シングルボードで分岐した外部周辺機器を制御する場合は、このコンピューターをよく見てください。また、USB3.0もあります。

Rock64は兄、有するROCKPro64 4ギガバイトLPDDR4と2つのUSB 3.0と8コア64ビットなRockchip RK3399上に構築され、（4 ARMのCortex A53と2つのARMのCortex A72）を、。Rock64およびROCKPro64のプロデューサーであるPINE64は、一般的にDIYビジネスの価値ある例です。スパークファンやアダフルーツなどの先駆者たちに打ち負かされて、開発に完全な設計ドキュメント一式を提供できると同時に、商業的に収益性の高い企業になることを示した先駆者たち。

ASUSティンカーボードS


Tinkerは、クアッドコアARM Cortex-A17プロセッサを搭載したRockchip RK3288 SoCをベースにしており、DebianベースのTinkerOSで動作し、Androidをインストールできます。Mali-T760 MP4ビデオプロセッサは、OpenGL ES 3.1、OpenCL 1.1、Renderscript、およびDirect3D 11.1をサポートしています。



GPIOコームはカラーコーディングを受け取ったので、外部デバイスを接続するときに、間違いを犯すのが少し難しくなります。競合するボードとは別に購入されるプロセッサーケースに接着された小さなパッシブラジエーターは、標準キットに注意深く含まれています。

Asus TinkerボードSには弟のAsus Tinkerボードがあり、オンボードフラッシュメモリはありませんでしたが、あまり成功せず、現在では販売されていません。

Libre Computer RenegadeおよびRenegade Elite


Renegade、またはLibre Computer ROC-Rk3328-CC Renegadeは、Raspberryと可能な限り類似するように設計されています。たとえば、ROC-Rk3328-CCをMalinka用に購入したケースに直接配置できます。



SoC RK-3328は、最大1.5 GHzの動作周波数を持つクアッド64ビットARM Cortex-A53プロセッサに基づいています。 SoCはRock64と同じなので、ここにも動作周波数500 MHzの同じMali 450MP2 GPUがあります。注文するときは、RAMの量を変更する機会があります。1GB DDR4（その後、コンピューターは35ドルかかります）、50ドルの場合は2 GB、80ドルの場合は4 GBを選択できます。オペレーティングシステムのうち、Ubuntu 18.04、Debian 9、OpenMediaVault 4、Station OS、Android 7.1が現在利用可能です。

Asus Tinkerボード（Sインデックスなし）と同様に、Renegadeにはオンボードフラッシュメモリがありません。microSDカードスロットまたはeMMCインターフェイスを使用する必要があります。

Libre Computerは、Raspberry Piの登場により生まれた新しいビジネスニッチ市場で大きな成長を遂げたメーカーの1つです。Renegadeに加えて、2018年の夏にLibre ComputerはRenegade Eliteの生産を開始しました。これは、6コアプロセッサー（2つのARM Cortex-A72コアと4つのARM Cortex-A53）およびGPU ARM Mali-を搭載したSoC Rockchip RK3399をベースにしたシングルボードコンピューターLibre Computer ROC-Rk3328-CC Renegade Eliteです。 T860MP4。2つのUSB 3.0 Type-Cポート、60ピンPCIeスロット、128 MBのオンボードメモリ、60ピンGPIOコームを備えています。Renegade EliteにはオプションのPoEサポートがあり、Linux 4.19またはAndroid 8を実行できます。

オドロイドh2


その比較的低価格を考えると、64ビット4コアインテルCeleron Gemini Lake J4105に基づいて構築されたOdroid H2は、ARMプロセッサーをベースとするシングルボードコンピューターが占める市場ニッチの一部を十分に主張することができます。 x86互換ソリューションを好む場合は、この110×110 mmの小型のパッシブ冷却マザーボード（Intel UHDグラフィックス600 GPU、PCI-E gen2バス、デュアルSATA 6 GB /秒）をお勧めします。



Intelの仕様によると、最大RAMサイズは8 GB（SO-DIMM DDR4 2400 MHz、含まれていない、「pisyuk」とは別に購入）ですが、製造元のOdroid H2は2つの16 GBストリップ（合計32 GB）そしてすべてが完璧に機能しました。もちろん、このボードはWindows 10 / Linux x64、DirectX 12、OpenGL 4.3、OGL ES 3.0、OpenCL 2.0をサポートしています。Odroid

H2の代替として、UDOO X86を検討できます（Intel Pentium N3710 2.56 GHzおよびIntel Celeron N3160 2.24 GHzにはオプションがあります）。

Arduino Mega 2560


People Xフランチャイズのザビエル博士でなくても、読者の心の叫びを今すぐ聞くことができます。 MalinkaとArduinkaはどこにありますか？チーターと大腸菌を比較するのと同じです！」はい、そうです、そうです。RaspberryPiとArduinoはかなり異なる市場ニッチにありますが、記事の最後にあるファイナルテーブルを見ると、これらの非常に多様な開発の2つを関連させる1つのポイントが見つかります。そして、このアイテムは価格です。はい、彼らはArduino Mega 2560 Rev3に31 ドル、Arduino Uno WiFi Rev2に42ドルを求めています（最も速い読者が推測しているように、ボードにはWi-Fiが装備されています）。



この「シングルボードコンピュータ」のハードウェアの詰め物を見てみましょう（または、引用符を省略する必要がありますか？）。 16 MHzのクロック周波数を備えた8ビットマイクロコントローラーATmega2560、256キロバイトのフラッシュメモリ（うち8キロバイトはブートローダーが占有）、8キロバイトのRAM、および4キロバイトの不揮発性メモリ。通信機能のうち、以前の比較モデルで使用されていた通常のシングルボードコンピューターのイーサネット、Wi-Fi、Bluetoothの代わりに、Arduino Mega 2560の場合、4つのUARTに言及する必要があります。そのうちの1つは組み込みのUSB-UARTコンバーターに接続されています。ちなみに、コンバータ自体はAVRマイクロコントローラATmega8U2-MUにも実装されていますが、これは重要ではありません。専用のFTDIチップを備えたオプションがあります。一般的に、ボードの仕様（出力ごとに20 mAの許容電流、または15個のGPIOをPWM出力として構成する機能など）は、ボードが組み込まれているマイクロコントローラーの特性を繰り返し予測します。メーカー自身は、Arduino Megaを3Dプリンティングとロボット工学に関連する趣味のプロジェクトの良い基盤として位置づけています。

Arduinoの世界にはRaspberry Pi Zeroマイクロコンピューターの独自のアナログがあります。これはArduino Nanoボードです。 ATmega328Pマイクロコントローラーに基づいて構築され、寸法は18×45 mmで、デバイスは2キロバイトのRAMと32キロバイトのフラッシュメモリ、16 MHzクロック、22本のGPIOピンを備え、そのうち6本はPWM出力として機能します。

一般に、Arduinoシリーズはハードウェアデバイスの点で非常に単純なので、私が趣味の憂鬱な状態にあったときでも、私は座って（Altium Designerの助けを借りて）Arduino Nanoのバージョンを描きました。 USBLC6-4SC6保護アセンブリをUSB回路に組み込み、FTDI ICのAN-146 USBハードウェア設計ガイドラインに従ってUSB-UARTコンバータを完成させ、AVR042：AVRハードウェア設計の考慮事項に従って回路に小さな変更を加えることにより、ノイズ耐性を向上させます。 PINダイオードにガンマ線センサーを追加しました（「なぜ？できるから！」）。放射線センサーの比較的高圧の電源を外部から引きずる必要がないように、マイクロコントローラーによって制御される小さな電圧マルチプライヤが追加されました。結果のボードの説明はこちらです。と、ここからダウンロードできる完全なプロジェクトアーカイブ（すべてのAltiumソースとライブラリを含む）。

もちろん、一般的に、ArduinoがRaspberryとそのクローンの全盛期を過ぎても絶滅しなかったのは少し奇妙です。 Arduinoの人気は、Atmelが開発したAVRアーキテクチャの2000年代初頭から中頃に起こった出来事によるものではないと思います。言うまでもなく、8ビットマイクロコントローラーの市場は空っぽで、i51とPICの間にはかなり良い選択肢があり、省エネソリューションには16ビットMSP430が利用可能でした。

しかし、AVRシリーズのマイクロコントローラーは、成功した新しいRISCアーキテクチャを組み合わせ、1クロックサイクルで命令を実行し（間違いなく当時のi51の息吹の下で打撃となり、12クロックサイクルで命令を実行しました）、安価な内部フラッシュメモリにちょうど間に合いました。これは超新星でした。弱くない古き良き時代を背景にさえ。私が今覚えているように、私はフラッシュAVRマイクロコントローラーを繰り返しフラッシュしたことを嬉しく思います。 JTAGは含まれておらず、動作中にレジスタを直接確認することは不可能でしたが、紫外線消去機能を備えたH1830BE51 + EPROMのペアと比較すると、それは新しい未来でした。

多分それはソフトウェア開発を促進するArduino IDE、またはArduino互換の開発者の友好的なコミュニティでしょう。いずれにせよ、ラズベリーのサポーターは何か考えるべきことがある。たぶん、最新のAシリーズARMプロセッサの開発しきい値は高すぎるため、多くの開発者、特に趣味は、最新のプロセッサを操作するためのすべてのトリックを習得する準備ができていないようです。 STM32CubeMXのようなものが表示され（ARM Cortex-M0 ...
