タイプスクリプトと反応

Javascriptの開発は、探偵の仕事に似ている場合があります。他の誰かのコードを理解するには？他の人がその点を理解できるように、開発者が変数の命名に巧妙なスキルを持っているとよいでしょう。しかし、チームメンバーがまだ同僚の意図を理解できない場合はどうでしょうか。関数の引数に何が入るのかを理解するにはどうすればよいですか？


関数の引数をエラーと呼びます。おそらくエラーには配列があります。最も可能性の高いライン？まあ、その配列は理解できます。結局、その長さはさらにチェックされます。ただし、文字列プロパティには長さプロパティもあります。正確に把握するには、ブレークポイントを設定してスクリプトを実行する必要があるようです。次に、UIのスクリプトを完全に実行します（たとえば、フォームの最後のステップが必要です）。 devtoolsでは、エラーは長さフィールドを含む特定のフィールドのセットを持つオブジェクトであることがわかります。


JavaScriptコードの解析におけるこのようなあいまいさは、開発者の時間の浪費につながります。この場合の良い解決策は、typescript（以下、ts）です。次のプロジェクトで使用することも、既存のプロジェクトでtsをサポートすることもできます。その後、他の誰かのコードを理解する時間が大幅に短縮されます。実際、データの構造を理解するには、1回のクリックで十分です。データを操作するロジックに集中でき、いつでもコードを明確に理解していることがわかります。


tsのいくつかの利点に注意する必要があります。さまざまなフレームワークで広く使用されており、JavaScriptと密接に関連しています。tsの開発は、フロントエンド開発者のニーズによって決定されます。


この記事では、todoアプリケーションの開発について説明しますが、興味深い点については簡単に説明します。完全なコードはここにあります。


私は、react、typescript、mobxを使用しました。Mobxは、アプリケーションの状態を管理するための柔軟なツールです。Mobxは簡潔です。これにより、反応コンポーネントの状態を同期スタイルで操作できます。次のような問題はありません：


this.setState({name: 'another string'});
alert(this.state.name);

この場合、古いstate.nameが表示されます。


さらに、mobxは便利で、tsタイプの操作を妨げません。状態は個別のクラスとして、または直接reactコンポーネント内に記述できます。


簡単にするために、すべてのコンポーネントはコンポーネントフォルダーに配置されます。コンポーネントは、コンポーネントの表示と操作に論理的に関連付けられている状態の説明とともに、コンポーネントのフォルダーで定義されます。

// TodoItem.tsx
import React, { ChangeEventHandler } from 'react';
import { observer } from 'mobx-react';
import { TodoItemState } from './TodoItemState';
import { EditModal } from 'components/EditModal';
import classNames from 'classnames';
import classes from './TodoItem.module.scss';

export interface ITodoItemProps {
  todo: TodoItemState;
  onDelete: (id: string) => void;
}

@observer
export class TodoItem extends React.Component<ITodoItemProps> {
  private handleCompletedChange: ChangeEventHandler<HTMLInputElement> = e => {
    const {
      todo: { editCompleted },
    } = this.props;
    editCompleted(e.target.checked);
  };

  private handleDelete = () => {
    const { onDelete, todo } = this.props;
    onDelete(todo.value.id);
  };

  private get editModal() {
    const { todo } = this.props;
    if (!todo.isEditMode) return null;
    return (
      <EditModal
        name={todo.value.name}
        onSubmit={this.handleSubmitEditName}
        onClose={this.closeEditModal}
      />
    );
  }

  private handleSubmitEditName = (name: string) => {
    const { todo } = this.props;
    todo.editName(name);
    this.closeEditModal();
  };

  private closeEditModal = () => {
    const { todo } = this.props;
    todo.setIsEditMode(false);
  };
  private openEditModal = () => {
    const { todo } = this.props;
    todo.setIsEditMode();
  };

  render() {
    const { todo } = this.props;
    const { name, completed } = todo.value;
    return (
      <div className={classes.root}>
        <input
          className={classes.chackbox}
          type="checkbox"
          checked={completed}
          onChange={this.handleCompletedChange}
        />
        <div
          onClick={this.openEditModal}
          className={classNames(
            classes.name,
            completed && classes.completedName
          )}>
          {name}
        </div>
        <button onClick={this.handleDelete}>del</button>
        {this.editModal}
      </div>
    );
  }
}

// TodoListState.ts
import { action as a, observable as o, computed as c } from 'mobx';
import { ITodoItem, TodoItemState } from 'components/TodoItem';

export enum TCurrentView {
  completed,
  active,
  all,
}

export class TodoListState {
  @o public currentView: TCurrentView = TCurrentView.all;
  @o private _todos: TodoItemState[] = [];

  @c
  public get todos(): TodoItemState[] {
    switch (this.currentView) {
      case TCurrentView.active:
        return this.activeTodos;
      case TCurrentView.completed:
        return this.completedTodos;
      default:
        return this._todos;
    }
  }

  @c
  public get completedTodos() {
    return this._todos.filter(t => t.value.completed);
  }
  @c
  public get activeTodos() {
    return this._todos.filter(t => !t.value.completed);
  }

  @a public setTodos(todos: ITodoItem[]) {
    this._todos = todos.map(t => new TodoItemState(t));
  }

  @a
  public addTodo = (todo: ITodoItem) => {
    this._todos.push(new TodoItemState(todo));
  };
  @a
  public removeTodo = (id: string): boolean => {
    const index = this._todos.findIndex(todo => todo.value.id === id);
    if (index === -1) return false;
    this._todos.splice(index, 1);
    return true;
  };
}
