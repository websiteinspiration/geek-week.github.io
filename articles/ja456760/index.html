ML Boot Campでのクラウドソーシング。Odnoklassnikiからの新しいタスクでは、写真なしのmIOUを検討します

こんにちは！暑い夏。"IT"チャンピオンシップの主催者は、太陽の下で多くの時間を費やし、火傷を負って打撃を与えましたが、最も重要なのは、ML Boot Campプラットフォームでの次の（すでに9回目の）コンテストの新しいタスクを収集したことです。チャンピオンシップは1か月以内にオンラインで開催されます。


現在、1万人を超える専門家がプラットフォームに登録されています。その上に表示されるタスクが全員の歯ではないことがよく起こります（Kaggleの神々を含む）。なぜこれを行うのですか？さらに、合成のでたらめではなく、実際のデータを開発して試す必要があります。勝利は時間とともに来るでしょう。

このコンテストで解決するために提供したいタスクは、これまでのすべてのタスクとは異なります。タスクのテーマは、画像内のオブジェクトの検出です。問題の説明は、データセットに画像が含まれることを意味しますが、面白いことに、画像は含まれていません。そして、これらはペタバイトのデータではありません。ギガバイトもありません。

このトピックは人気があり、実際には非常に重要です。 Odnoklassnikiソーシャルネットワークでは、他の製品と同様に、コンテンツをフィルターするタスクがあります。あなたの子供が間違った絵を見ることを想像してください-彼の世界は永遠に変わります。

2年前、Alexey SennikovはOdnoklassnikiがこの問題を解決するためにユーザーをどのように惹きつけるかについての記事を書きました。簡単に言うと、記事の中で、AlexeyはOdnoklassniki Moderatorゲームアプリケーションについて話しました。ソーシャルネットワークのユーザーは画像を良いものと悪いものに分類し、さまざまなパンを受け取り、アルゴリズムを使用して増加するDAU（Daily Active Users）アプリケーションの問題を解決する方法を共有しました機械学習。

ちなみに、現在DAUには4万人のユーザーがいて、1日あたり1,000万を超える決定があります。悪くないでしょ？

新しいコンテストの挑戦


Odnoklassnikiソーシャルネットワークには、データをマークアップするためのプラットフォームがあります。その上で、ユーザーは長方形の写真で特定のオブジェクトを強調表示するタスクを与えられました。人々の答えによると、オブジェクトの真の位置を復元する必要があります。


エントランスでは、クラウドソーシングのマークアップとデータをすべてのお気に入りのラベルの形式で受け取ります。応答として、予測を含むファイルを送信する必要があります。

データセットとベースラインは、MLブートキャンプでコンテストが始まる日に公開されます。


ソリューションを評価するために、mIOU（ユニオン上の平均交差）メトリックが使用されます。このメトリックが表示されていない場合は、それに関する記事を読むことをお勧めします。
