最新のテクノロジーライブラリを作成する

こんにちは、ハブル。この記事では、Hello worldを最新のテクノロジーで作成する方法について説明します。


最後に、次のようなHello Worldライブラリを取得します。


  • typescriptを使用
  • コードスタイルの面倒を見る
  • ドックを生成します
  • テストを実施

プロジェクトの構造


プロジェクトには次のディレクトリがあります。


  • src-ライブラリ自体のファイルになります（typescript）
  • lib-ここでtscはsrcからコンパイルされたファイルを置きます
  • テスト-ここにテストがあります
  • 例-使用例

またに追加する必要があります.gitignore：


lib/
node_modules/

活字


typescript:


npm i typescript -D

tsconifg.json


 {
  "compilerOptions": {
    "target": "es5",
    "module": "commonjs",
    "declaration": true,
    "outDir": "./lib",
    "strict": true,
    "sourceMap": true
  },
  "include": ["src"],
  "exclude": ["node_modules", "**/__tests__/*"]
}
