Sass MediaScreen-CSSデバイスの定義

最近、さまざまなデバイスのグループ間での画面サイズの違いがますますぼやけてきていると思います。それにもかかわらず、私はデバイスのグループ（携帯電話、タブレット、ラップトップ、デスクトップ）と特定のデバイス（iPhone 5、iPhone X、iPad Pro 12など）の両方を特定するツールを作成しようとしました。私の意見では、SASSミックスインのかなり便利なパックであることがわかりました。そして重要なことに、このツールは使いやすく、ソースを編集せずにデバイスリストを拡張することもできます。


ちなみに、こちらのデモをご覧ください。

ほんのいくつかの予約：


  • DevToolsブラウザーでの適合性をテストしないでください。デバイスの横向きの側面のサイズはそこで正しく計算されません。実際のデバイスまたはシミュレーター（xCode Simulatorなど）で確認することをお勧めします。
  • 同じメディアクエリを組み合わせるには、group-css-media-queriesを使用します。それがないと、@media ...便宜上、@include device()各セレクターで個別にミックスインを使用すると、多くの繰り返しコードが生成されます。Gulpラッパー-gulp-group-css-media-queries。

取り付け


ライブラリをダウンロードします。

$ yarn add sass-mediascreen

$ npm install sass-mediascreen

$ curl -O https://raw.githubusercontent.com/gvozdb/sass-mediascreen/master/_mediascreen.scss

アプリケーションでミックスインを接続します。

@import "mediascreen";

例


これは、特定のデバイスをテストする方法です。

@include device(iPhone5, portrait) {
    // portrait orientation
    // iPhone 5, iPhone 5s, iPhone 5c, iPhone SE
}
@include device(iPhone6Plus iPhoneXR, landscape) {
    // landscape orientation
    // iPhone 6+, iPhone 6s+, iPhone 7+, iPhone 8+, iPhone XR
}
@include device(iPadPro10 iPadPro11 iPadPro12) {
    // all orientations
    // iPad Pro 10.5, iPad Pro 11, iPad Pro 12.9
}

デバイスグループは次のとおりです。

@include device(desktop) {
    // all orientations
    // desktop
}
@include device(mobile tablet laptop, landscape) {
    // landscape orientation
    // mobile, tablet, laptop
}
@include device(mobile-landscape tablet laptop) {
    // landscape orientation
    // mobile

    // all orientations
    // tablet, laptop
}
@include device(mobile-landscape tablet laptop, portrait) {
    // landscape orientation
    // mobile

    // portrait orientation
    // tablet, laptop
}

そして、これらは画面サイズ（from、to）とデバイスの現在の向きをチェックするための標準のミックスインです：

@include screen(min-width, max-width, orientation) {/*...*/}
@include min-screen(width) {/*...*/}
@include max-screen(width) {/*...*/}

@include screen-height(min-height, max-height, orientation) {/*...*/}
@include min-screen-height(height) {/*...*/}
@include max-screen-height(height) {/*...*/}

@include landscape() {/*...*/}
@include portrait() {/*...*/}

デバイスのリストの拡張


前に述べたように、ライブラリソースを編集せずにカスタムデバイスまたはデバイスグループのサポートを追加できます。これを行うには、をインポートする前に、デバイスのリスト@import "mediascreen"を使用してSass変数を宣言する必要があり$ms-devicesます。

$ms-devices: (
    desktop-sm: (
        group: true, //   -  
        min: 1200px,
        max: 1919px,
    ),
    desktop-md: (
        group: true,
        min: 1920px,
        max: 2879px,
    ),
    desktop-lg: (
        group: true,
        min: 2880px,
    ),
    pixel2xl: (
        group: false, //   -  
        width: 411px, // or 412px?..
        height: 823px,
        pixel-ratio: 3.5,
    ),
    macbook12: (
        group: false,
        orientation: landscape,
        width: 1440px,
        height: 900px,
        pixel-ratio: 2,
    ),
    imac27: (
        group: false,
        orientation: landscape,
        width: 5120px,
        height: 2880px,
    ),
);
@import "mediascreen";

その後、デバイスと標準ライブラリデバイスを確認できます。

@include device(desktop-sm) {
    // desktop-sm
}
@include device(desktop-md) {
    // desktop-md
}
@include device(desktop-lg) {
    // desktop-lg
}
@include device(Pixel2XL, landscape) {
    // landscape orientation
    // Google Pixel 2XL
}
@include device(MacBook12) {
    // landscape orientation
    // MacBook 12
}
@include device(iMac27) {
    // landscape orientation
    // iMac 27
}
