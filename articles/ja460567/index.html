React Native：ドラッグ＆スワイプ可能なリストを作成する

今日、モバイルアプリケーションでリストアイテムをスワイプする機能を備えたユーザーを驚かせることは困難です。私たちの反応ネイティブアプリケーションの1つにもそのような機能がありましたが、最近、リストアイテムをドラッグアンドドロップする機能でそれを拡張する必要がありました。そして、解決策を見つけるプロセスにはある程度の神経細胞が必要だったので、将来の世代のために貴重な時間を節約するために短い記事を提出することにしました。



このアプリケーションでは、パッケージを使用してスワイプ可能なリストを作成しましたreact-native-swipe-list-view。最初に考えたのは、ドラッグアンドドロップ機能を備えたパッケージを受け取り、ハリネズミをヘビと交差させることでした。

インターネットの広域にわたる検索によりreact-native-draggable-list、react-native-sortable-listとの3つの候補が与えられましたreact-native-draggable-flatlist。

最初のパッケージを使用すると、添付されている例を実行することもできませんでした（ただし、私だけではなく、対応する問題がissueに示されています）。

2番目のパッケージをいじくり回さなければなりませんでしたが、ドラッグ＆スワイプ可能なリストを作成することができました。ただし、結果は影響を受けていません。コンポーネントは恥ずかしくないほど明滅しています。再描画の明滅、リストをはるかに超える要素の落下、またはそれらの消滅さえありました。この形では使えないことが判明しました。

最初は、最後のパッケージも気まぐれな女性のように振る舞いましたが、調理方法がわからないことがわかりました。この「女性」の心の鍵を手に入れ、私はなんとか許容できる結果を達成することができました。

私たちのプロジェクトでは、ドラッグアンドドロップを固定する必要があるスワイプ可能なリストがありましたが、実際には、もう一方の端から開始することをお勧めします。最初にドラッグ可能なリストを作成してから、スワイプする機能を追加します。

読者は反応ネイティブプロジェクトの作成方法を知っているはずなので、必要なリストの作成に集中しましょう。以下で説明する例はTypeScriptコードです。

ドラッグ可能なリストを作る


だから、パッケージをインストールすることから始めましょう：

yarn add react-native-draggable-flatlist

必要なモジュールをインポートします。 

import React, { Component } from 'react'
import { View } from 'react-native'
import styles from './styles'
import DraggableFlatList, { RenderItemInfo, OnMoveEndInfo } from 'react-native-draggable-flatlist'
import ListItem from './components/ListItem'
import fakeData from './fakeData.json'

ここでDraggableFlatList実装ドラッグアンドドロップ機能は、ことインストールパッケージから成分であるListItemリスト項目を表示するための我々の成分（コードは以下提示される）、fakeData偽データを含むJSONファイルである-この場合、フォームのオブジェクトの配列：

{"id": 0, "name": "JavaScript", "favorite": false}

実際のアプリケーションでは、このデータはほとんどの場合、小道具からコンポーネントに送られるか、ネットワークからダウンロードされますが、私たちの場合、ほとんど血で管理しません。

この例ではTypeScriptを使用しているため、いくつかのエンティティについて説明します。

type Language = {
  id: number,
  name: string,
  favorite: boolean,
}

interface AppProps {}

interface AppState {
  data: Array<Language>
}

タイプLanguageは、リスト項目がどのフィールドを持つかを示します。

この例では、小道具から何も取得しないので、インターフェースAppPropsは取るに足らないものであり、ストーリーLanguageでは、インターフェースに示されているオブジェクトの配列を格納しますAppState。

コンポーネントコードはそれほど大きくないので、全体を示します。

アプリコンポーネントコード
class App extends Component<AppProps, AppState> {
  constructor(props: AppProps) {
    super(props)

    this.state = {
      data: fakeData,
    }
  }

  onMoveEnd = ({ data }: OnMoveEndInfo<Language>) => {
    this.setState({ data: data ? [...data] : [] })
  }

  render() {
    return (
      <View style={styles.root}>
        <DraggableFlatList
          data={this.state.data}
          renderItem={this.renderItem}
          keyExtractor={(item) => item.id.toString()}
          scrollPercent={5}
          onMoveEnd={this.onMoveEnd}
        />
      </View>
    )
  }

  renderItem = ({ item,  move, moveEnd, isActive }: RenderItemInfo<Language>) => {
    return (
      <ListItem
        name={item.name}
        move={move}
        moveEnd={moveEnd}
        isActive={isActive}
      />
    )
  }
}


このメソッドonMoveEndは、アイテムが移動されたときに呼び出されます。この場合、状態の要素の新しい順序でリストを配置する必要があるため、メソッドを呼び出しますthis.setState。

このメソッドrenderItemは、リストアイテムを表示するために使用され、RenderItemInfo <Language>タイプのオブジェクトを取ります。このオブジェクトには次のフィールドが含まれます。

  • item -リストにデータとして渡される配列の次の要素
  • moveそしてmoveEnd-リスト項目が移動されるときに呼び出される関数は、コンポーネントは、これらの機能を提供しDraggableFlatList、
  • isActive -アイテムが現時点でドラッグ可能かどうかを決定する論理タイプのフィールド。

実際、リストアイテムを表示するためのコンポーネントTouchableOpacityはmove、長押しするとを呼び出し、解放すると-になりmoveEndます。

ListItemコンポーネントコード
import React from 'react'
import { Text, TouchableOpacity } from 'react-native'
import styles from './styles'

interface ListItemProps {
  name: string,
  move: () => void,
  moveEnd: () => void,
  isActive: boolean,
}

const ListItem = ({ name, move, moveEnd, isActive }: ListItemProps) => {
  return (
    <TouchableOpacity
      style={[styles.root, isActive && styles.active]}
      onLongPress={move}
      onPressOut={moveEnd}
    >
      <Text style={styles.text}>{name}</Text>
    </TouchableOpacity>
  )
}

export default ListItem


すべてのコンポーネントのスタイルは個別のファイルに移動され、ここには記載されていませんが、リポジトリで表示できます。

スワイプする機能を追加する


さて、私たちは最初の部分にうまく対処し、マルレゾンバレエの後半に進みます。

リストアイテムをスワイプする機能を追加するには、パッケージを使用しますreact-native-swipe-list-view。

開始するには、それをインストールしましょう：

yarn add react-native-swipe-list-view

このパッケージにはSwipeRow、ドキュメントによると、次の2つのコンポーネントが含まれているはずのコンポーネントがあります。

<SwipeRow>
    <View style={hiddenRowStyle} />
    <View style={visibleRowStyle} />
</SwipeRow>

最初のビューは2番目のビューの下に描画されることに注意してください。

コンポーネントコードを変更してみましょうListItem。

ListItemコンポーネントコード
import React from 'react'
import { Text, TouchableOpacity, View, Image } from 'react-native'
import { SwipeRow } from 'react-native-swipe-list-view'
import { Language } from '../../App'

import styles from './styles'

const heart = require('./icons8-heart-24.png')
const filledHeart = require('./icons8-heart-24-filled.png')

interface ListItemProps {
  item: Language,
  move: () => void,
  moveEnd: () => void,
  isActive: boolean,
  onHeartPress: () => void,
}

const ListItem = ({ item, move, moveEnd, isActive, onHeartPress }: ListItemProps) => {
  return (
    <SwipeRow
      rightOpenValue={-180}>

      <View style={styles.hidden}>
        <TouchableOpacity onPress={onHeartPress}>
          <Image source={item.favorite ? filledHeart : heart} />
        </TouchableOpacity>
      </View>

      <TouchableOpacity
        activeOpacity={1}
        style={[styles.root, isActive && styles.active]}
        onLongPress={move}
        onPressOut={moveEnd}
      >
        <Text style={styles.text}>{item.name}</Text>
      </TouchableOpacity>
    </SwipeRow>
  )
}

export default ListItem


まず、要素をスワイプできる距離を決定SwipeRowするプロパティrightOpenValueを持つコンポーネントを追加しました。

第二に、我々は移動SwipeRow私たちの内側をTouchableOpacityして、このボタンの下に描画されますビューを追加しました。

このビュー内には、言語がお気に入りかどうかを判断するための絵が描かれています。クリックすると値が反転します。データは親コンポーネントにあるため、このアクションを実行するコールバックをここでスローする必要があります。

親コンポーネントに必要な変更を加えます。

アプリコンポーネントコード
import React, { Component } from 'react'
import { View } from 'react-native'
import styles from './styles'
import DraggableFlatList, { RenderItemInfo, OnMoveEndInfo } from 'react-native-draggable-flatlist'
import ListItem from './components/ListItem'
import fakeData from './fakeData.json'

export type Language = {
  id: number,
  name: string,
  favorite: boolean,
}

interface AppProps {}

interface AppState {
  data: Array<Language>
}

class App extends Component<AppProps, AppState> {
  constructor(props: AppProps) {
    super(props)

    this.state = {
      data: fakeData,
    }
  }

  onMoveEnd = ({ data }: OnMoveEndInfo<Language>) => {
    this.setState({ data: data ? [...data] : [] })
  }

  toggleFavorite = (value: Language) => {
    const data = this.state.data.map(item => (
      item.id !== value.id ? item : { ...item, favorite: !item.favorite }
    ))
    this.setState({ data })
  }

  render() {
    return (
      <View style={styles.root}>
        <DraggableFlatList
          data={this.state.data}
          renderItem={this.renderItem}
          keyExtractor={(item) => item.id.toString()}
          scrollPercent={5}
          onMoveEnd={this.onMoveEnd}
        />
      </View>
    )
  }

  renderItem = ({ item, move, moveEnd, isActive }: RenderItemInfo<Language>) => {
    return (
      <ListItem
        item={item}
        move={move}
        moveEnd={moveEnd}
        isActive={isActive}
        onHeartPress={() => this.toggleFavorite(item)}
      />
    )
  }
}

export default App
