Pythonでの一般的な財務分析（パート1）

前回の記事では、金融商品に関する情報を取得する方法を検討しました。次に、取得したデータを使用して最初に何ができるか、戦略を分析して作成する方法について、いくつかの記事が公開されます。資料は、いずれかのオンラインプラットフォームでの外国のソースおよびコースの出版物に基づいています。

この記事では、収益性、ボラティリティを計算し、主要な指標の1つを構築する方法について説明します。

import pandas as pd
import yfinance as yf
import numpy as np
import matplotlib.pyplot as plt

sber = yf.download('SBER.ME','2016-01-01')

収益性


この値は、1取引日の株価の変化率を表します。配当や手数料は考慮されていません。Pandasパッケージのpct_change（）関数を使用して簡単に計算できます。

原則として、ログの収益性が使用されます。これにより、時間の経過に伴う変化をよりよく理解して調査できるからです。

#   `
daily_close = sber[['Adj Close']]

#  
daily_pct_change = daily_close.pct_change()

#  NA   0
daily_pct_change.fillna(0, inplace=True)

print(daily_pct_change.head())

#   
daily_log_returns = np.log(daily_close.pct_change()+1)

print(daily_log_returns.head())



取得したデータから週次および/または月次の収益性を見つけるには、resample（）関数を使用します。

#   `sber`      
monthly = sber.resample('BM').apply(lambda x: x[-1])

#  
print(monthly.pct_change().tail())

#  `sber`        
quarter = sber.resample("4M").mean()

#  
print(quarter.pct_change().tail())



pct_change（）関数は便利に使用できますが、値の取得方法を非表示にします。メカニズムの理解に役立つ同様の計算は、Pandasパッケージのパッケージからのshift（）を使用して実行できます。 1日の終値は過去の（1シフトした）価格で除算され、受け取った値から単位が差し引かれます。しかし、マイナーマイナスが1つあります。結果として最初の値はNAです。

収益性の計算は次の式に基づいています。



ここで、pは価格、tは時刻、rは収益性です。

次に、収益性分布図が作成され、主要な統計が計算されます。

#  
daily_pct_change = daily_close / daily_close.shift(1) - 1

print(daily_pct_change.head())




#  `daily_pct_c`
daily_pct_change.hist(bins=50)

plt.show()

#  
print(daily_pct_change.describe())



分布は非常に対称的に見え、通常は0.00の値の周りに分布しています。他の統計値を取得するには、description（）関数を使用します。その結果、平均値はゼロよりわずかに大きく、標準偏差はほぼ0.02であることがわかります。

累積リターン


累積日次リターンは、指定された期間の投資の価値を決定するのに役立ちます。以下のコードに示すように計算できます。

#   
cum_daily_return = (1 + daily_pct_change).cumprod()

print(cum_daily_return.tail())




#    
cum_daily_return.plot(figsize=(12,8))

plt.show()



毎月の利回りを再計算できます：

#   
cum_monthly_return = cum_daily_return.resample("M").mean()

print(cum_monthly_return.tail())



株を分析する場合、リターンの計算方法を知ることは重要です。しかし、他の株と比較すると、それはさらに価値があります。

いくつかの株を取り（その選択は完全にランダムです）、チャートを作成します。

ticker = ['AFLT.ME','DSKY.ME','IRAO.ME','PIKK.ME', 'PLZL.ME','SBER.ME','ENRU.ME']

stock = yf.download(ticker,'2018-01-01')

#    `daily_close_px`
daily_pct_change = stock['Adj Close'].pct_change()

# 
daily_pct_change.hist(bins=50, sharex=True, figsize=(20,8))

plt.show()



別の有用なグラフは、散布行列です。これは、pandasライブラリの一部であるscatter_matrix（）関数を使用して簡単に構築できます。Daily_pct_changeが引数として使用され、カーネル密度推定パラメーターが設定されます。さらに、alphaパラメーターを使用して透明度を設定し、figsizeパラメーターを使用してグラフのサイズを設定できます。

from pandas.plotting import scatter_matrix

#   `daily_pct_change`  
scatter_matrix(daily_pct_change, diagonal='kde', alpha=0.1,figsize=(20,20))

plt.show()
