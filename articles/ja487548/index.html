新しい実験：gRPC-Webを使用してブラウザーから.NET gRPCサービスを呼び出す

.NETでのgRPC-Webの実験的サポートを発表できることをうれしく思います。gRPC-Webを使用すると、JavaScript SPAやBlazor WebAssemblyアプリケーションなどのブラウザベースのアプリケーションからgRPCを呼び出すことができます。

gRPC-Web for .NETは、ブラウザーアプリケーションに多くの優れたgRPC機能をもたらすことを約束します。

  • 厳密に型指定されたコード生成クライアント
  • コンパクトなProtobufポスト
  • サーバーストリーミング



gRPC-Webとは


HTTPリクエストを詳細に制御できるブラウザAPIがないため、ブラウザにgRPC HTTP / 2仕様を実装することはできません。 gRPC-Webは  、HTTP / 1.1およびHTTP / 2と互換性があることでこの問題を解決します。

gRPC-Webは新しいテクノロジーではありません。安定した  gRPC-Web JavaScriptクライアントと、 サービス用のgRPCとgRPC-Web間の変換用の  プロキシがあります。新しい実験的パッケージにより、ASP.NET Core gRPCアプリケーションはプロキシサーバーなしでgRPC-Webをサポートし、.NET Core gRPCクライアントはgRPC-Webサービスを呼び出すことができます。（Blazor WebAssemblyアプリケーションに最適！）

gRPC-Webの新機能


  • ブラウザーからASP.NET Core gRPCアプリケーションを呼び出す-ブラウザーAPIはgRPC HTTP / 2を呼び出すことができません。gRPC-Webは互換性のある代替手段を提供します。
    • JavaScript SPA
    • .NET Blazor Webアセンブリアプリケーション
  • IISおよびAzure Application ServiceでASP.NET Core gRPCアプリケーションをホストします。IISやAzure Application Serviceなどの一部のサーバーは、現在gRPCサービスをホストできません。彼らが積極的に取り組んでいる間、gRPC-Webは、今日どの環境でも機能する興味深い代替手段を提供します。
  • .NET Core以外のプラットフォームからgRPCを呼び出します。一部の.NETプラットフォームHttpClientはHTTP / 2をサポートしていません。gRPC-Webを使用して、これらのプラットフォーム（Blazor WebAssembly、Xamarinなど）でgRPCサービスを呼び出すことができます。

gRPC-Webはパフォーマンスが低く、2つのgRPC機能（クライアントストリーミングと双方向ストリーミング）がサポートされなくなったことに注意してください。（サーバーへのストリーミングは引き続きサポートされています！）

GRPC-Webサーバーの手順


.NETでgRPCをまだ学習していない場合は、  簡単なチュートリアルを使用して始めることができます。

gRPC-Webではサービスを変更する必要はありません。変更点は初期構成のみです。ASP.NET Core gRPCサービスでgRPC-Webを有効にするには、リンクをGrpc.AspNetCore.Webパッケージに追加します。gRPC-Webを使用するようにアプリケーションを構成し、を追加AddGrpcWeb(...)しUseGrpcWeb()、起動ファイルを起動します。Startup.cs

public void ConfigureServices(IServiceCollection services)
{
    services.AddGrpc();
}

public void Configure(IApplicationBuilder app)
{
    app.UseRouting();

    //    gRPC-Web     
    app.UseGrpcWeb();

    app.UseEndpoints(endpoints =>
    {
        endpoints.MapGrpcService<GreeterService>().EnableGrpcWeb();
    });
}

ブラウザからgRPC-Webを呼び出すには、CORSをサポートするようにアプリケーションを構成するなど、追加の構成が必要になる場合があります。

GRPC-Webクライアントの手順


gRPC-Web JavaScriptクライアントには 、JavaScript SPAブラウザで使用するためにgRPC-Webクライアントを設定するための手順があります。

.NETクライアントでのgRPC-Webの呼び出しは、通常のgRPCに似ています。唯一の変更点は、チャネルの作成方法です。gRPC-Webを有効にするには、Grpc.Net.Client.Webパッケージへのリンクを追加します。使用するチャネルを構成しますGrpcWebHandler。

//     gRPC-Web
var handler = new GrpcWebHandler(GrpcWebMode.GrpcWebText, new HttpClientHandler());
var channel = GrpcChannel.ForAddress("https://localhost:5001", new GrpcChannelOptions
    {
        HttpClient = new HttpClient(handler)
    });

var client = Greeter.GreeterClient(channel);
var response = await client.SayHelloAsync(new GreeterRequest { Name = ".NET" });

.NETが動作するgRPC-Webを確認するには、Blazer WebAssemblyでgRPC-Webを使用した同僚の最もクールなブログ投稿を読んでください  。
