Uso de HAProxy como una malla de servicio de plano de datos en una arquitectura de microservicio



La API de plano de datos HAProxy es un programa que funciona con HAProxy para que pueda configurar completamente el equilibrador de carga HAProxy en tiempo de ejecución.







Esta característica le permite utilizar una serie de casos de uso adecuados para una configuración creada dinámicamente.







El mejor ejemplo es el uso de HAProxy como la malla de servicio del plano de datos en la arquitectura del microservicio, donde el plano de control central controla la configuración del conjunto de proxy del sidecar.







Esta sección le muestra cómo instalar y configurar la API del plano de datos y comenzar con algunos ejemplos.







Consulte también la documentación de la especificación API para más detalles.







Requisitos



  • Comunidad HAProxy 1.9 o posterior (2.0 cuando se utiliza Process Manager)
  • HAProxy Enterprise 1.9 o posterior
  • Conocimiento práctico de HTTP y uso de la API REST
