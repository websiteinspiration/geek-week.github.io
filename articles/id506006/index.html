Pythonists, berhenti menggunakan tanda kurung siku untuk mendapatkan nilai dari kamus

Halo, Habr! Saya mempersembahkan untuk Anda terjemahan dari artikel "Berhenti Menggunakan Notasi Braket Kotak untuk Mendapatkan Nilai Kamus dengan Python" oleh Jonathan Hsu.

Melampaui metode tradisional dan melindungi kode Anda


Kamus adalah kumpulan kunci dan nilai yang tidak disortir. Artinya:

  • Setiap elemen kamus terdiri dari istilah (kunci) dan definisi (artinya).
  • Kunci harus unik untuk kamus ini - tidak ada pengulangan.
  • Kunci dalam kamus tidak memiliki pengaturan eksplisit yang dimiliki item daftar.

Untuk membuat kamus, gunakan kurung kurawal, di dalamnya tuliskan kuncinya: pasangan nilai dipisahkan oleh koma. Seperti ini:

author = {
   "first_name": "Robinson",
   "last_name": "Crusoe",
   "username": "Friday1719"
}

Cara tradisional (dan tidak aman) untuk mendapatkan makna dari kamus.

Untuk akses normal ke nilai elemen dari kamus, tanda kurung kotak digunakan. Saat menggunakan entri seperti itu, kunci ditunjukkan dalam tanda kurung setelah nama kamus:

author = {
   "first_name": "Robinson",
   "last_name": "Crusoe",
   "username": "Friday1719"
}
print(author['username']) #  Friday1719
print(author['middle_initial']) #   KeyError: 'middle_initial'

Perhatikan bahwa mencoba mengakses nilai dengan kunci yang tidak ada dalam kamus akan mengeluarkan pengecualian KeyError. Ini dapat menciptakan masalah serius, terutama ketika berhadapan dengan data kerja yang tidak dapat diprediksi.

Tentu saja, Anda dapat menggunakan coba / kecuali membangun atau menggunakan pernyataan if. Tetapi peningkatan perhatian seperti operasi sederhana hanya mengacaukan kode.

author = {}
try:
   print(author['username'])
except KeyError as e:
   print(e) # 'username'

if 'username' in author:
   print(author['username'])

Jika Anda memiliki pengalaman pengembangan JavaScript di belakang Anda, maka Anda mungkin ingin mendapatkan nilai dari kamus menggunakan notasi bertitik. Tetapi dengan Python, ini tidak akan berhasil.

author = {
   "first_name": "Robinson",
   "last_name": "Crusoe",
   "username": "Friday1719"
}
print(author.username)
# AttributeError: 'dict' object has no attribute 'username'

Kami menggunakan metode get ()


Saat Anda perlu mendapatkan nilai dari kamus, cara teraman adalah menggunakan metode get (). Metode ini memiliki dua parameter:

  • Pertama (wajib): nama kunci yang ingin kami peroleh nilainya. Nama ini bisa berupa string atau nama variabel jika kunci kami dapat berubah selama program.
  • Kedua (opsional): nilai yang akan digunakan jika kunci kita tiba-tiba tidak muncul dalam kamus.

author = {
   "first_name": "Robinson",
   "last_name": "Crusoe",
   "username": "Friday1719"
}
print(author.get('username')) #   Friday1719

Jika kunci ada di kamus, maka metode get () bekerja dengan cara yang persis sama dengan mengakses kunci dalam tanda kurung siku. Tetapi dalam kasus ketika tidak ada kunci seperti itu dalam kamus, metode get () akan mengembalikan nilai default, menyelamatkan Anda dari keharusan menangani pengecualian.

Nilai default dapat berupa objek apa pun yang valid dalam konteks ini. Ingat bahwa parameter ini opsional. Oleh karena itu, jika Anda tidak menentukannya secara eksplisit, maka ketika Anda mencoba mengakses kunci yang tidak ada dalam kamus, metode get () akan mengembalikan objek None.

Kami menggunakan metode setdefault ()


Terkadang Anda tidak hanya perlu menerima data dari kamus dengan aman, tetapi juga menambahkan data baru ke kamus dengan aman. Untuk ini, kamus memiliki metode setdefault (). Ini memiliki parameter yang sama dengan metode get (), tetapi tidak seperti yang terakhir, ketika mengakses kamus menggunakan kunci tidak ada, itu tidak hanya akan mengembalikan nilai default, tetapi juga membuat elemen baru dalam kamus dengan kunci ini dan nilai yang dikirimkan. Jika saat mengakses kamus menggunakan metode setdefault (), kunci yang dikirimkan sudah ada di kamus, metode ini akan membuat kamus tidak berubah.

author = {
   "first_name": "Robinson",
   "last_name": "Crusoe",
   "username": "Friday1719"
}

print (author.setdefault ('username')) # akan menampilkan
print Friday1719 (author.setdefault ('middle_initial', "Monday")) # akan menampilkan Senin dan membuat

Dalam contoh di atas, kita melihat bahwa perilaku metode setdefault () tidak berbeda dengan perilaku metode get () atau dari penggunaan tanda kurung kotak saat mengakses kamus menggunakan kunci yang ada di dalamnya. Jika tidak ada kunci seperti itu di kamus, maka metode setdefault () tidak hanya akan mengembalikan nilai argumen keduanya ke program (seperti metode get ()), tetapi juga membuat elemen dalam kamus dengan kunci dan nilai yang diteruskan ke sana. Ini adalah perilaku dari metode setdefault () dan membedakannya dari metode get ().

Sekarang, jika Anda mengikuti contoh di atas dan menampilkan elemen kamus,maka kita mendapatkan hasil sebagai berikut:

print(author)
{
  "first_name": "Robinson",
  "last_name": "Crusoe",
  "username": "Friday1719"
  "middle_initial": "Monday"
}

Menggunakan metode get () dan setdefault () adalah teknik kelas satu untuk mengakses nilai kamus. Hanya perlu waktu untuk meninggalkan kebiasaan lama dan mulai menggunakan teknik ini dalam praktik.

Jika Anda hanya perlu mendapatkan nilai dari kamus, maka helper Anda adalah metode get ().

Jika Anda perlu menambahkan nilai baru ke kamus dengan aman, panggil metode setdefault ().
