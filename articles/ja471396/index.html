レイアウトの構造化


Flexboxを使用する場合と同様に、グリッドの性質上、レイアウトは少し異なりますが、レイアウトを構築する必要があります。

これは、グリッドがより複雑であると人々が主張できる状況の1つですが、実際には、各コンテンツの周りにブロックを描画するだけです。



フレックスボックスを使用して、列として目立つ2つのブロックを作成しました。代わりにグリッドを使用する場合は、親要素のグリッド全体を構成し、このグリッドのどこに配置するかを子要素に示します。

グリッドを設定するには、次のようにします。

.card {  
    display: grid;
    grid-template-columns: 1fr 3fr;
}

frの測定単位はグリッドに固有であり、使用可能なスペースの比率を示します。この使用法は、Flexboxに2つの列をインストールし、その幅をそれぞれ25％と75％に決定することと非常に似ています。
