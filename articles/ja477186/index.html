最初の仕事


2006年、私は21歳で、経済学のコンピュータサイエンスの学位を取得して大学を卒業しています。 Pascal、デザイン、リレーショナルデータベース、4種類の会計、GOST 19および34グループでプログラムすることを教えられた場所。私は、コンピュータ会社でソフトウェアの修理、販売をしています。

彼らは私に市政のある州の機関に行き、コンピュータハードウェアの供給に関する行動に署名する仕事を与えました。この機関は、ITソリューションの実装と都市のニーズに対応するソフトウェアの開発に特化しており、偶然にも、Bobruiskaya Streetにありました。

監督との契約に署名することで、「プログラマーとしてあなたと一緒に働きたい」というフレーズを決めてぼかし、私の高等教育と私に何ができるかを簡単に説明しました。当時、私は工芸品をC＃で彫刻することができました。幸いなことに、これより少し前に、マイクロソフトはVisual Studio 2005およびSQL Server 2005のライセンスされたコピーを含むディスクを発行するプログラマ向けのセミナーを開催しました。私はCarly Watsonの厚い本「C＃」を大騒ぎし、購入し、読みました。運命は私に好意的でした、そして、しばらくして、彼らは私に電話して、プロジェクトを作ることを申し出ました。だから私は私の最初の仕事を得ました。

期待に反して、この政府機関はかなりレベルにあり、専門的にその機能を果たしました。一部のプロセスは開発部門で設定され、プロジェクトチームはデザイナー（ディレクター）とプログラマーで構成され、Visual Source Safeバージョン管理システムが使用されました。きれいな滝がありました。ディレクターは技術的な割り当てを書きました、彼らは国家権力の長い廊下にいる大きなボスによって署名されました。その後、タスクはプログラマーに下がった。テストと受け入れは、ディレクターによって行われました。

プロジェクトはシンプルで定期的なCRUDに加えて、WordとExcelのレポートでした。最初にWinForms、次にWeb。時々、外部のドキュメント管理システムとの統合が発生しました。

それはのんきで、計り知れ、幸せな時間でした。そしてこの会社には感情的な企業パーティーがありました。今は懐かしいです。次の10年間、私は定期的に元同僚とお茶に行き、私の成功について話しました。しかし、彼らはまだ同じ教室に座っていましたが、彼ら自身が上司に育ちました。

最初の経済危機と最初のウダレンカ


2008年、危機が襲った。そして、私の友人はオランダ人との面会を提案しました、彼らはプログラマーを募集して古いプロジェクトを復活させたいと言っています。会議で、彼らはさりげなくリモート操作モードについて言及しました。それから私は驚きました：「そして、そのように何が起こったのでしょうか？」それらから何も来なかった。しかし、私の頭にはすでにチャンスの粒が蒔かれていました。

私はウダレンカを探し始めました。どこでも検索：フォーラム、フリーランスの交換、xx.ru、superbobで。

フリーランスはすぐにはうまくいきませんでした。なぜ注文を検索してペニーを稼ぐのに多くの時間を費やさなければならないのか、そのときの私の現在の給与よりも少ない理由がわかりませんでした。 「10,000ルーブルのクラスメートの完全な類似物を私にしてください」のレベルの多くの不十分な顧客。最初にプロアカウントにお金を要求するフリーランス取引所。これにより、深刻な注文を確認できます。彼らのウェブサイトでダイムを獲得していない場合、どうすれば彼らに支払うことができますか？

プロジェクトの副業を見つけたら。最初はすべてが順調に進み、1つの小さなプロジェクトを実行し、次に2番目のプロジェクトで顧客が支払うことを約束しました。 2回目は間違えたので結局直したが、その代金は払わないとお客様から言われた。この時点で、私はいくつかの結論を出しました。それは、顧客/雇用者の目にもっと適切になるようになり、今後は将来の設計作業がなくなるということです。

私はフルタイムの仕事を見つけることに集中しました。当時（2009年）のxxには、私のプロフィールにウダレンカの54名の求人がありました。 「オフィスのみ」の空席を含め、何百もの応答を送信しました。その過程で、IE USNを6％オープンしました。私と雇用主が関係を正式化する方が簡単だと思いました。

2009年の終わりに、運命は私にキャンディーを投げました。モスクワの会社でインタビューを受けました。もともとオフィスは空いていたのでラッキーだったと思います。そして、彼らは私をフルタイムの仕事についての結果について考えることを約束してプロジェクトに連れて行きました。一部の政府機関の委託条件に基づいて、Web、CRUDなどのプロジェクトを実行する方法をすでに知っていました。 「モスクワ」の給与を請求しました。同意した。全体として、これは最も強い動機付けの推進力を与えました。最初の成功と主な仕事の数時間後の夜の時間の過ごし方では、地方の月給の半分を稼ぐことができました。

新しい年のために、彼らは私への贈り物を用意しました。プロジェクトが完了し、従業員の1人が会社を辞め、同じ条件でフルタイムの仕事udalenkaを提供してくれました。



便利なIPを入手してください。主な仕事を辞めたとき、人事部門の責任者は驚いたように尋ねました。危機は同じで、給与は削減され、解雇されています。」私は肩をすくめて、笑顔で「そうなった」と答えました。

2年


レースが始まりました。あなたが首都から数千キロ離れている場合でも、モスクワのペースの速いライフスタイルと仕事は引きずっています。どうしようもないと思った。プロジェクトはより複雑です。サブジェクトエリア（証券取引所、オークション、購入、ターゲット広告）は私には不明です。私は最初は2人ではなく、実際のチームで働いていました。脂肪のタスキは、信じられないほどのスピードで次々と成功しました。以前は2か月で500時間を終了していました。最初の1年半で、州で3つ以上学びました。そのとき、地域のプロジェクトが資本のプロジェクトに追いつくことは決してないだろうと気づきました。最良のケースでは、モスクワの企業が地域にやってきます（後に起こりました）。したがって、そこには人は必要ありません。したがって、誰もがモスクワに行きます。それから私はモスクワへのウダレンカが一種のライフハックであることに気づきました。プロジェクトはよりクールで、より多くの経験とより多くのお金です。

1年後、私は昇給を依頼しました。半年ほどで退屈になりました。私は同僚の1人の仕事に注目しました。彼はソリューションフレームワークを作成し、アプローチとソリューションを定め、ストレステストを書きました。そして、チームはビジネスロジックを完成させました。彼は建築家と呼ばれた。そして私は同じになりたかった。

会社は別の建築家を必要としないと言ったので、私は仕事を探し始めました。たくさんのインタビューがありました。私に新しいことを学ぶように勧めた人々に感謝します。私は多くの有用な本（「ギャングオブフォー」や「マーティンの原則、パターン、アジャイルテクニック」など）に促されました。自己啓発に取り組んでいます。現在のプロジェクトでいくつかのアイデアを試しました。私は以前彼をばかにしていた会社の1つにインタビューに出かけました。本を読むことを勧められました。その結果、「一緒に働きたい！」と聞きました。お世辞になりました。その結果、2012年の仕事で私自身が見つかりました。再びモスクワ、再び削除されましたが、私はすぐに私の目標の概要を説明しました-主要なプログラマーになることです。

私たちは州に家を出ることなくモスクワを征服します


既存のプロジェクトで6か月間働いた後、同社は銀行から、銀行と電子政府サービスの間の仲介システムを開発するように命令を受けました。そして私のプロジェクトマネージャーは、私がこのプロジェクトを自分で行うことを提案し、技術リーダーとしての立場を示しました。そして、私は私の夢を実現する機会を得ました。

命令を受けた。彼女は配られました。モスクワのプロジェクトマネージャー、ビジネスアナリスト、数人のプログラマーとテスター。そして私と他の何人かのプログラマーはロシアとCIS諸国に散らばっていました。この会社では、プロセスは私が過去と未来で見たことがなかったよりも適切に調整されました。カンバン、ビジネスアナリストによるタスクの分析と詳細設定、それから彼らは設計、評価、配布のために私にやって来ました。アーキテクチャ、インフラストラクチャ、一般的なアプローチ、コードの複雑なセクションに携わっていました。プログラマー間でのタスクの分散、作業の監督、コードレビューの実施。私はプロジェクト全体を頭の中に持っていて、いつでもその部分で何が起こっているのかを知ることができました。すべてが調和してしっかりと機能しました。誰もが自分のことをやっていて、彼が最善を尽くして見事にやった。さらに、たとえば、ビジネスアナリストは設計作業をほとんど残さず、説明はとても論理的で完全でした。

時間の経過とともに、プロジェクトはプラットフォームとコンポーネントへと進化しました。各コンポーネントは独自の外部サービスと連携しました。プロジェクトは、ロシア全土の約30の銀行で実施されました。おそらく、今でも、自動モードでの要求に応じて、どこかでFSSPまたはGIS GMPにデータを提供します。プロジェクトをリードすることの意味も理解しました。私のチームは私の手の延長であり、私はそれらを使用してより複雑なプロジェクトを行うことができました。私はそれが好き。

その結果、ある日、2014年に私はモスクワのメトロを旅していて、家を出ずにモスクワでキャリアを積んだことがありました。結局のところ、私のモスクワとその当時の同僚とでさえ、私はモスクワとサンクトペテルブルクで2016年から2018年にのみ会いました。また、今後数年間で、同僚の1人がスウェーデンの会社のリモートサイトのCTOになったと言います。その間、私はモスクワをリードするプログラマーと同じレベルで競争し始めました。はい、多くの興味深いプロジェクトが通り過ぎましたが、何かも私に行きました。同じ頃、私は彼らのリモートワークに関して3つのタイプの会社を識別しました：

私は望んでいません、私はしません-ウダレンカ、オールドスクールのリーダー、および/または指導方法を知らない、および/または新しいことを学びたくないという断定的な拒否。彼らは、遠隔作業が彼らに合わなかった理由を何千もの言い訳にした。 95％でのコミュニケーションはSkypeを介したもので、ビデオカメラを使わずに音声のみでしたので、彼らは次のように感じまし



た。同じグループでは、奇妙なことに、多くのスタートアップがひざまずいています。しかし、彼らにはプロセスをセットアップする時間がありません。要するに、あなたはこのグループの会社にあなたの時間を無駄にすべきではありません。

あなたは試すことができます -両方向の移行グループ。以下は、udalenkaを試す準備ができている企業と、すでに試みて燃やした企業です（多くはプログラマー自身に依存するため）。これらの企業が完全な幸福を得るためには、何かでは不十分でした。あなたは彼らと一緒に仕事をすることを試みることができますが、可能性は50から50です。

私たちはリモートワーカーを探しています -リモートワークの準備ができている進歩的な会社や幹部。長所と短所を参照してください。彼らは他のグループの会社と比較してプロセスをカスタマイズしました（そしてしばしばより良い）。そして、彼らは失敗しませんでした。その時までに、私のプロフィールのxxにあるudalenkaの求人の数は増え、300から400個に達することもあります。これらの企業は、モスクワの給与のより低いまたは平均的なバーの地域から、良質で忠実な（結局のところ、地方の給与は平均3万から5万ルーブル）プログラマーを雇い、彼らの人員配置のニーズを閉じることが可能であることを認識しました。しかし、このグループには激しい貪欲な会社があります。彼らは村から離れた労働者がジャガイモの袋のために働くべきだと信じています。そのような会社もまた迂回されるべきです。

第二（通貨）経済危機


2014年に2回目の危機が勃発し、ルーブルが急落しました。さて、モスクワに連れて行かれるのは人生のハックなので、惑星規模でもう一度引き上げてみませんか？私は外国のウダレンカを探し始め、同時に引っ越しの選択肢を検討しました。前のバージョンの給与によると、私は自分で10万ドルか、この地域でのプログラマーの平均給与を、この額よりも多いと判断しました。

このような興味深いリンクwww.bls.gov/oes/current/oes_nat.htm#15-0000があります。これは、コンピュータ専攻学生に対する米国全体の平均給与です。そこでは、毎年、ほぼ同じ数字が9万ドルから10万ドルです。州と都市についても同じですwww.bls.gov/oes/current/oessrcst.htm。

それは大変で、すべて同じでした。 1000人の顧客にとって、数千人のインディアンとのフリーランスのやり取り、Facebookの不適切な殺し屋。このキャンペーンは国に依存していません。

ボディショッパーに会いました。スープのボウルの夢の国への移転（バージニア州アーリントンで75,000ドル）。また、かなり適切なロシア系アメリカ人の会社が出会いました。初年度に10万ドルを提供した。しかし、7万ルーブルの場合、ノボシビルスクで凍結する必要がありました。私は同意しませんでした。

2015年の初めに、運命は再び私の側にありました、それに対して私は彼女にとても感謝しています。アメリカの会社にインタビューをしました。ハード通貨での給与。ルーブルx2の観点から。それはスリルでした。



私は非常に幸運です。そこで働いた唯一のロシア人プログラマーからインタビューを受けました。アルバイトとして始まり、2015年の夏にフルタイムで出ました。デザインはアップワークによるものでした。

私はほぼ4年間そこで働いていました。多目的な体験でした。プロジェクトは適度に複雑で大規模でしたが、私の考えでは、以前のロシアの会社のプロセスはより成熟していて明確でした。私には明白な真実をアメリカ人に伝えようとしましたが、彼らは自分たちのやり方で行きました。非常に多くの種類のバグトラッカーがあることも知りませんでした。幸いなことに、結局、彼らは脂肪に切り替えました。

しかし、主なものは新しい感覚です。あなたはロシアの怪しげな町に座って仕事をしていて、インターネット上の米国企業のリポジトリにコードをアップロードしています。彼女はこれから、彼女のソフトウェアを使用するアフリカの企業からも、お金を稼いでいます。真にグローバルなプロセスが、当時（そして現在）地球上で起こった。そして、あなたはまた、石油依存の解消に貢献します。結局のところ、年次の外国経済報告のどこかに、私の惨めなドルもあり、その仕事のために海外から電信送金で受け取りました。

文化的価値。結局のところ、彼らは同じ人々ですが、手のひらサイズの北米のゴキブリがいます。制裁については私たちと同じように冗談を言いましたが。おそらくこれは、この特定の会社の特異性です。たとえば、一般の従業員が私の仕事に不満を持っている場合、彼は上司に不満を言いましたが、私には不満はありませんでした。一般的に、すべての関係は健康的な平等に基づいていました（私はこれを理解し、感情を捨てます）。会社にとって、より遠い人はいつでも解雇される可能性があり、彼自身はいつでも離れることができる人物でした。したがって、彼らは彼に頼らず、グローバルなプロセスを信頼せず、特に耳を傾けませんでした。たとえば、システムのユーザー向けのスクリーンキャスト形式のトレーニングシステムなど、ビジネスプロセスをよりよく理解するためにプログラマーも参加できる、いくつかのアイデアが実装されました。

終わりに近づくにつれ、すべてがこの記事で説明したものに似たものになりました。しかし、彼らは私から結果を絞り出すことを学びました。そして、私は削減の対象となりました。

そして現時点ではロシアで


私はアメリカの会社で働いている間ずっと、労働市場を監視し、インタビューに行きました。次々と注目に値するイベントがいくつか起こりました。それはすべて、アメリカの企業でのリモートワークの面接への招待状から始まりました。招待状はそこで働いているロシア人から来ました。しかし、1日か2日と答えましたが、面接の時間はありませんでした。彼らは私に「たくさんの参入者、私たちはすでに発見しました」のようなことを言った。その結果、私はそれまでとほぼ同じ条件（給与は安いが、有給の休暇プラス）で別のオファーを受け、けいれんしませんでした。しかし、プログラマーの成果はすでに始まっています。外国のウダレンカに駆けつけたり、引っ越したりできる人は誰でもいるように思えました。キャンペーンは激しい競争でした。もう一度、私は最初にこの列車に飛び乗ることを許可してくれた運命に感謝した。

そしてしばらくすると、150〜20万ルーブルの空室が一斉に出現し始めました。モスクワとサンクトペテルブルグでの重要な職務（プレゼンター、建築家、技術リーダー、チームリーダー）。結果は終わり、ロシアの企業は空席の代わりを探していたという印象を受けました。彼らの側での検索の状況については知りません。私は彼らに答えたのを覚えています：「通貨を削除して通貨で支払うことは可能ですか？肯定的な反応を待たなかった。

今何？


そして今、私は34歳です。スウェーデン、カナダ、アメリカ、オーストラリアでの大変動とプロジェクトの後に、運命は私に別の贈り物をくれました。彼の地方の町で、半日自宅/オフィスで仕事をします。現代の若者の間で25-30年。IoTの有望な分野のプロジェクトで、海外市場への参入を計画しています。だからレースはまだ終わっていません。たぶん、新年の企業パーティーもあるでしょう。
