CI / CD di AWS, Azure dan Gitlab. Kursus baru dari OTUS

Perhatian! Artikel ini bukan rekayasa dan ditujukan untuk pembaca yang tertarik dengan pendidikan di bidang CI / CD. Kemungkinan besar, jika Anda tidak tertarik untuk belajar, materi ini tidak akan menarik bagi Anda.



Jika Anda seorang pengembang atau administrator yang bertanggung jawab untuk mengatur integrasi berkelanjutan / proses pengiriman berkelanjutan , maka OTUS telah membuka serangkaian kursus khusus untuk Anda: kursus intensif praktis tentang metode populer pengembangan berkelanjutan dan pengiriman perangkat lunak Integrasi Berkelanjutan dan Pengiriman Kontinu di seluruh platform Amazon Web Service, Azure, GitLab, dan Jenkins .

Selama pelatihan, siswa akan belajar cara mengonfigurasikan proses perakitan dan pengujian aplikasi dan proses instalasi dengan tiga penyedia terkemuka, serta membentuk pemahaman tentang arsitektur penyedia cloud dan mempelajari otomatisasi analisis kode dan pencarian kerentanan.

Sebagai hasil dari pelatihan, setiap siswa akan membuat pekerjaan kelulusan, yang akan menjadi implementasi proses CI / CD untuk setiap proyek open source untuk dipilih. Setelah pelatihan, tentu saja, setiap siswa akan menerima materi untuk semua kelas, sertifikat kelulusan, dan yang paling penting, mengonfigurasi perakitan dan pengujian aplikasi dan dapat menemukan kerentanan.



Tentu saja, kursus ini bukan untuk semua orang. Tetapi jika Anda memiliki pengalaman :

  • Bekerja dengan git
  • Administrasi sistem Linux atau Windows
  • Pengembangan atau operasi
  • Bekerja dengan penyedia cloud

maka OTUS sedang menunggumu! Anda dapat mengikuti tes masuk untuk menentukan apakah Anda memiliki pengetahuan yang cukup untuk menyelesaikan kursus CI / CD pada AWS, Azure, dan Gitlab.

Proses pelatihan pada kursus "CI / CD pada AWS, Azure dan Gitlab" berlangsung dalam format webinar online. Sepanjang pelatihan (dan berlangsung 3 bulan), siswa dapat bertanya kepada guru berpengalaman yang selalu berhubungan. Latihan praktik akan diselesaikan menggunakan Google Cloud Platform (GCP), Amazon Web Service, dan Microsoft Azure.

Program kursus terdiri dari empat modul utama:

  1. Pengembangan di cloud (Kode)
  2. Otomatisasi perakitan dan pengujian (Integrasi Berkelanjutan)
  3. Instalasi Otomatis (Pengiriman Kontinu)
  4. Modul ringkasan

Masing-masing dari mereka akan dipahami secara menyeluruh di kelas dalam format webinar online, dan pekerjaan rumah akan membantu untuk mengkonsolidasikan pengetahuan yang diperoleh, yang menurutnya, jika perlu, Anda bisa mendapatkan umpan balik terperinci dari guru.
