JavaScript Meetup SuperJobへようこそ10月10日

10月10日、JavaScript開発者の会議がSuperJobオフィスで開催されます。

SuperJobスペシャリストは、React / Reduxでのアプリケーションの最適化、Flowでの静的型の使用、Node.jsでの作業の経験を共有します。
従来、このプログラムにはハンバーガーと非公式のコミュニケーションが含まれています。

日時：10月10日19 ：00〜21：30
住所：モスクワ、マラヤドミトロフカ、ハウス20
登録：superjob-meetup.timepad.ru/event/1063170/



スピーカー：

1.  アレクサンダーKukhta、のリード開発者  SuperJob  テーマとした  「JSONの経験：クライアントのAPI」：

  • クライアントに1,500のエンティティを保存する方法
  • Reactコンポーネントでそれらを使用する方法
  • そして、それが遅くならないようにする方法

2.   テーマ「タイプシステムとフロントエンドタイピング」を持つ  SuperJob開発グループのリーダーである  Pyotr Solopov： 

  • 存在する型システムとその開発方法
  • フローと TypeScript
  • 大規模プロジェクトでのFlowの使用方法

3.  アレクサンダーサビン、「JSONサーバーの作成：Node.jsとPostgreSQL上のAPI」を  テーマにした  SuperJobフロントエンド開発マネージャー  ：

  • 既存のデータベースのAPIをすばやく取得するのに役立つツール
  • Node.jsとPostgreSQLを使用してそのようなツールを自分で作成する方法
  • 「最初のDB」アプローチを使用するときに発生する可能性のある問題と、それらに対処するのに役立つもの

入場無料ですが、十分なスペースを確保するため、ご登録ください。

当日は、パスポートまたは運転免許証を持参してください。これは、ビジネスセンターへのアクセスに必要です。
