スーパーケトルの仮想環境にOpenCV-Pythonをインストールする

ここでは、PythonにOpenCVライブラリをインストールするための段階的な例を示します。


  • Pythonのインストール
  • 仮想環境のセットアップ
  • OpenCV + jupiterlab、numpy、matplotlibをインストールする
  • テスト中

64ビットオペレーティングシステムを搭載したMicrosoft Surfaceタブレット、Windows 10 Proですべてをテストしました。


デバイスが事前にインストールされていないとします。


  1. 最初にPythonをインストールします。
    目的のバージョンをダウンロードし、.exeファイルを実行します。パスの追加ボックスをチェックすることを忘れないでください。2019年3月25日からPython 3.7.3をインストールしました。これは、2019年7月8日からの新しいバージョンのPython 3.7.4が正しく機能していなかったため、一部のコマンドがターミナルでハングしていたためです。コマンドラインを開きます。
  2. virtualenvをインストールします。
    他のプロジェクトに依存せず、混乱することのない、インストールされたライブラリの独自のバージョンを備えた個別の「部屋」を各プロジェクトが持つように、仮想環境が必要です。pip
    を使用してパッケージをインストールします。最近はPythonですぐに実行されていますが、通常は次のコマンドで更新する必要があります：
    python -m pip install --upgrade pip
    pipを更新して、仮想環境をインストールします：
    pip install virtualenv
    コマンドcdを使用して、環境を作成するフォルダーに移動し、コマンドを入力し
    mkdir opencvtutorial_envます。
    次に、コマンドを入力virtualenv opencvtutorial_envしてアクティブ化するには、環境フォルダーに移動し、Tabキーを使用してアクティブ化します。
    .\opencvtutorial_env\Scripts\activate
  3. OpenCV-Python, Numpy Matplotlib, opencv.
    . :
    pip install opencv-python
    opencv-python numpy. matplotlib: pip install matplotlib.
  4. pip install jupyterlab jupyter notebook.
