10 extensiones para VS Code, sin las cuales no puedo programar

VS Code es mi editor de código favorito. Este es el más popular de los editores existentes, cuyas posibilidades se pueden ampliar casi indefinidamente. Y, sorprendentemente, fue desarrollado por Microsoft. Creo que ninguno de los otros editores y ninguno de los otros IDEs pueden darle al desarrollador al menos la mitad de lo que VS Code puede darle. Una de las fortalezas de VS Code es su sistema de extensión. Le permite crear extensiones literalmente para todas las ocasiones. Quiero contarles sobre mis 10 extensiones principales para VS Code.



1. Embellecer



Para instalar la extensión, use el atajo de teclado en VS Code CTRL+P, ingrese el siguiente comando en el campo que se abre y haga clic Enter.

ext install HookyQR.beautify

La extensión Beautify le permite formatear código html, js, css, JSON y sass. Las opciones de formato se pueden controlar. Esta extensión es un complemento sobre el sistema estándar js-beautify y permite al desarrollador formatear el código exactamente como lo necesita.

2. Mejores comentarios



ext install aaron-bond.better-comments

La extensión Better Comments le permite ampliar las posibilidades de trabajar con comentarios. En particular, le permite dividir los comentarios en categorías. Estas pueden ser notificaciones, solicitudes, listas de tareas, notas.


Usando mejores comentarios

3. Marcadores



ext install alefragnani.Bookmarks

La extensión Bookmarks es una verdadera línea de vida para los programadores. Especialmente para aquellos que se pierden entre miles de líneas de código. Esta extensión le permite adjuntar marcadores a líneas de código. Si la línea de código tiene un marcador, puede ir fácilmente a esta línea utilizando la lista de marcadores.


Marcadores en código

4. Soporte Par Colorizer 2



ext install CoenraadS.bracket-pair-colorizer-2

La extensión Bracket Pair Colorizer 2 le permite colorear los brackets, lo que facilita encontrar pares de brackets de apertura y cierre. Esto es muy útil en casos en los que tiene que trabajar con construcciones de software anidadas.


Los pares de paréntesis están resaltados.

5. Formato en menús contextuales



ext install lacroixdavid1.vscode-format-context-menu

La extensión Formatear en menús contextuales le permite formatear archivos resaltándolos en la barra lateral y llamando al comando del menú contextual. Esto es especialmente conveniente en casos donde hay muchos archivos que necesitan formatearse, pero al mismo tiempo en el entorno existente no hay soporte para los medios para formatear y alinear el código.


Trabajar con formato en menús contextuales

6. Gráfico de Git



ext install mhutchie.git-graph

La extensión Git Graph le permite ver la estructura del repositorio y, utilizando esta estructura, realizar varias operaciones. Esta extensión es altamente personalizable y tiene muchas características. Probablemente, para describirlo adecuadamente, necesitará un artículo completamente separado.


Trabaja con Git Graph

7. GitLens



ext install eamodio.gitlens

La extensión GitLens ayuda a identificar la autoría del código utilizando anotaciones y lentes de culpa de git. Esta extensión, entre otras cosas, le permite ver de manera muy conveniente el contenido de los repositorios y aprender información útil sobre el código.


Trabaja con GitLens

8. Sangría Rainbow



ext install oderwat.indent-rainbow

La extensión Indent Rainbow le permite colorear la sangría, resaltando la sangría de diferentes niveles con diferentes colores. Esto es especialmente útil cuando se trabaja con construcciones profundamente anidadas en lenguajes como Python.

9. Senda Intellisense



ext install christian-kohler.path-intellisense

La extensión Path Intellisense proporciona al editor la capacidad de autocompletar rutas de archivos. Aunque VS Code admite la finalización de rutas sin extensiones, esta función se limita solo a archivos HTML, CSS y JavaScript. Gracias a esta extensión, la finalización de la ruta se puede utilizar al trabajar con cualquier archivo y cualquier idioma.


Trabaja con Path Intellisense

10. Líneas totales



ext install praveencrony.total-lines

La extensión Total Lines es una pequeña herramienta conveniente que muestra información sobre el número de líneas en un archivo abierto en la barra de estado. Esta es una buena adición a mi colección de extensiones.
