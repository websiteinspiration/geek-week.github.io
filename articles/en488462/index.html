Back to the future with ember-data polyfill packages


The ember-data package API allows you to import objects and methods from small packages of the ember-data module, following the recommendations similar to those installed in Ember Modules RFC # 176 (note: RFC - Request for comment) . With real and solvable package paths, many developers can benefit from the improved code reference in their IDE. The ember-data package API is available starting with version 3.11 .
