Pacsafeの新作：Intasafe XおよびIntasafe X Slimバックパック



お気に入りのラップトップ-信頼できるバックパック。柔らかな裏地、充電用の場所、アダプター、ドングルなどがあります。今日は、対角線が15インチの2つのラップトップバックパック、Pacsafe Intasafe XとPacsafe Intasafe X Slimについて説明します。ボーナス：盗難防止。

保護のしくみ


私たちはすでにPacsafeブランドについて話しました。要するに、この会社は熱心な旅行者ロブ・シュリッパーとマグナス・マグラシャンによって1998年に設立されました。それ以来、スリから保護された旅行者向けのバックパックとバッグを（そしてそれだけでなく）作成しています。



保護はPacsafe設計者自身の経験によって決定され、実際の状況で正常にテストされています。Intasafe XおよびIntasafe X Slimの場合、3つの部分から構成されます。

1つ目は、メインコンパートメントのジッパーロックです。 2つのランナーのタングが互いに挿入され、ロックロッドであるロックラッチに掛けられます。泥棒は静かに素早くバックパックを開けません。まず城のデザインを理解する必要があります。スリは何かを盗むための最も簡単な方法を探しています。したがって、彼らはおそらくPacsafe Castleに連絡しません。稲妻は信頼性が高く、二重で、ハンドルを貫通しません。



2つ目は、右ショルダーストラップのバックルです。ストラップは固定されたオブジェクトの周りに巻き付けられます：カフェのテーブル脚、通りや木のフェンスポスト。バックルは目立たないピンを回すことでしっかりと固定され、バックパックは動かせなくなりました。ストラップにはスチールストラップが縫い付けられており、カットできません。



3つ目は、シールド付きの内ポケットです。ポケットは、潜在的なハッキングからプラスチックカードとRFIDタグを保護します。そのような攻撃はまだまれですが、Pacsafeは夏にそりを準備しています。



通常、Pacsafeバックパックはスチールメッシュで補強されています。Intasafe XとIntasafe X Slimはアーバンバックパックであるため、デザイナーは簡単なオプションを選択しました。それは長い間役立ち、拭かれません。

Pacsafe Intasafe X


これは2つのバックパックの中で最も広々としています。 Intasafe Xの容量は27リットルで、寸法は47.5 x 29 x 16 cm、重量は830グラムです。ほとんどの航空会社のフライトの手荷物として適しています（お知らせ：特定の航空会社の制限を事前に確認してください）。



外側には、バックパックに3つのポケットがあります。 1つは、前面に隠しジッパー付き。迅速なアクセスが必要な小さなものを対象としています（ただし、ウォレットは対象外です）。ボトルと傘用の2つのポケットは側面に隠されています。スナップボタン付きの調節可能なポケット。



内部のコンパートメントは1つで、大きいです。背面には、15インチのラップトップ用の柔らかい裏地付きの密集したポケットがあります。上部には、10インチタブレット用の別のものと、外部バッテリー用の別のポケットがあります。その真上にはワイヤー用の穴があります。ワイヤーを通し、内部でパワーバンクに接続すると、バックパックからバッテリーを取り外すことなくスマートフォンを充電できます。



バックパックの後半では、ポケットの「オーガナイザー」が自慢します。これは、内側にキー用のカラビナが付いたジッパー付きの広いポケット、カード、ワイヤーまたはフラットガジェット用のオープンポケット、ペン用のホルダーです。ベルクロポケットはシールドと同じです。



Intasafe Xはスーツケースのハンドルに掛けられています。これには、背中にゴムバンドがあります。背中はやわらかめです。

Pacsafe Intasafe X Slim


コンパクトなIntasafe X Slimは薄型（43 x 29 x 11 cm）で、手荷物にも適し、飛行機の座席の下にもフィットします。バックパックの重さは510グラム、容量は20リットル。



外には、バックパックにポケットがありません。これは、スリのために重要なものを残す誘惑がなく、スリが何かを引き出す可能性がないことを意味します。



内部には、Intasafe Xとほぼ同じ編成があります。背面には、15インチのラップトップと10インチのタブレット用の高密度のソフトポケットがあります。パワーバンク用のポケットと、外出先でガジェットを充電するためのワイヤー用の穴もあります。



反対側の「オーガナイザー」の配置が少し異なります。大きなファスナー付きポケットはなく、キーカラビナはバックパックのメインコンパートメントのすぐ内側にあります。ただし、シールド付きのベルクロポケットと小さなもの用のポケットが付いています。


左側はPacsafe Intasafe X Slim、右側はPacsafe Intasafe X
