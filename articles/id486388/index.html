Tinjauan Proyek Open Source Nirlaba MyVPN



Lampiran MyVPN dengan open source memungkinkan untuk mengotomatisasi manajemen jaringan pribadi virtual. Penggunaannya tidak memerlukan keterampilan administrasi sistem.



Saat memilih penyedia VPN, kepercayaan memainkan peran yang lebih penting daripada biaya layanan. Layanan gratis sering disalahkan karena kurangnya enkripsi yang dapat diandalkan dan untuk melacak pengguna , sementara layanan komersial dapat mengikuti aturan regulator dan mulai memblokir akses ke sumber daya yang terlarang atau mereka sendiri akan masuk daftar hitam oleh Roskomnadzor. Pilihan ideal dari sudut pandang untuk memberikan anonimitas adalah server Anda sendiri, tetapi hanya sedikit orang yang ingin mengkonfigurasinya. MyVPN sangat menyederhanakan pembuatannya.

Mengapa memiliki VPN untuk pengguna biasa?


Biasanya mereka terhubung ke VPN untuk memastikan operasi yang aman dengan titik akses nirkabel publik atau untuk memotong kunci yang telah diperkenalkan oleh otoritas pengawas dari beberapa negara dalam beberapa tahun terakhir. Layanan video streaming dapat membatasi akses ke konten berdasarkan fitur geografis - kebutuhan untuk mengenkripsi lalu lintas atau secara virtual pindah ke lokasi lain cukup umum.

Setiap penyedia komersial dengan reputasi baik cocok untuk menyelesaikan masalah keamanan, namun situasi dengan mem-bypass kunci lebih rumit. Tahun lalu, Roskomnadzor menuntut agar penyedia VPN besar mulai mematuhi undang-undang domestik. Sejauh ini, para pihak hanya bertukar basa-basi, tetapi setiap saat layanan asing yang populer dapat diblokir. Mengganti mereka tidak mudah: pengguna akan menerima harga rendah (2-3 dolar sebulan) hanya ketika membayar berlangganan setidaknya selama satu tahun, atau bahkan tiga. Jika ILV mencapai penyedia layanan, langganan ini di Rusia akan berubah menjadi labu.

Perlu mempertimbangkan pengalaman kawan-kawan Cina dan tidak membayar lebih dari sebulan, tetapi rencana tarif bulanan untuk penyedia besar adalah 7-12 dolar. Dengan harga seperti itu, ide menaikkan VPN Anda terlihat menarik, dan dari sudut pandang anonimitas, opsi ini lebih menarik: siapa yang tahu data apa yang dikumpulkan penyedia VPN tentang kami? Pengguna biasa dihentikan hanya oleh kebutuhan untuk mengkonfigurasi dan mengelola server - proyek MyVPN menyelesaikan masalah ini.

Bagaimana cara kerja MyVPN?


Penting untuk menekankan bahwa MyVPN bukan layanan, tetapi aplikasi yang berjalan melalui API penyedia hosting untuk Windows, macOS, GNU / Linux dan Android. Ini mengotomatiskan pembuatan dan penghapusan server VPN di infrastruktur cloud CryptoServers.Net , DigitalOcean atau Linode . Pengguna perlu menginstal aplikasi pada komputer atau perangkat seluler, masuk ke akun host yang dipilih (jika tidak ada akun, Anda harus mendaftar) dan menentukan wilayah yang diinginkan, serta protokol. Untuk memulai server Anda, cukup klik satu tombol.

Membuat server VPN membutuhkan waktu beberapa menit, setelah itu aplikasi akan menawarkan untuk menyimpan detail untuk akses ke sana. Langkah ini diperlukan karena program tidak menyimpan data untuk meningkatkan keamanan.


Tidak ada kemungkinan untuk mengkonfigurasi koneksi VPN di aplikasi untuk OS desktop (pengecualian dibuat hanya dalam versi untuk Android): untuk terhubung ke server, Anda harus menggunakan alat sistem. Ini juga dilakukan untuk alasan keamanan, tetapi prosesnya tidak terlalu rumit - instruksi detail tersedia di situs web MyVPN. Anda dapat membuat beberapa server, dan mereka dihapus secara harfiah dalam satu klik.


Mengapa MyVPN aman?


Aplikasi MyVPN open-source berjalan pada perangkat pengguna, tidak mentransfer data pribadi ke pengembang, dan tidak menyimpan akses ke server VPN yang dibuat di akun penyedia hosting Anda. Tentu saja, program ini membutuhkan otorisasi di akun pribadi penyedia hosting, tetapi tanpa itu Anda tidak bisa mendapatkan akses ke API dan membuat / menghapus server, dan kode sumber terbuka memungkinkan Anda untuk memastikan bahwa informasi login tidak akan pergi ke samping. Selain itu, dalam versi desktop, Anda dapat memasukkan kunci API dari kantor penyedia yang dipilih.

Integrasi dengan CryptoServers.Net paling baik diimplementasikan.. Hoster ini menjaga privasi dengan baik, chip-nya adalah VPS yang benar-benar anonim dan kemampuan membayar dalam bitcoin. DigitalOcean dan Linode juga tidak terlihat dalam skandal spionase, tetapi, selain kartu kredit yang valid, mereka terkadang meminta pemindaian dokumen identitas. Bagaimanapun, hanya pengguna yang mengetahui server IP dan memiliki kunci akses untuk itu - pada dasarnya, ini adalah VPS biasa, dan layanan apa yang berjalan di sana adalah hal ketiga. Dalam hal privasi, opsi ini melampaui layanan VPN khusus yang berpotensi dapat melakukan apa saja dengan data Anda.

Berapa MyVPN?


Aplikasi MyVPN tidak memerlukan pembelian lisensi dan tidak menyiratkan komisi untuk digunakan: hanya layanan penyedia hosting yang dibayar. Sebagai contoh, CryptoServers.Net VPN mesin virtual dengan saluran 1 Gbit / s berharga $ 0,02 per jam dan saluran ini dirancang untuk satu pelanggan. Pengembangan aplikasi dimonetisasi melalui program afiliasi, sementara penyedia hosting sendiri membayar penulisnya untuk menarik pelanggan. Skema yang sederhana dan logis, harga yang sebanding dengan tarif penyedia VPN besar: ketika membayar segera selama satu tahun, mereka dapat menemukan opsi yang lebih murah, tetapi dengan risiko kehilangan uang karena blok mendadak oleh Roskomnadzor. Saat menggunakan MyVPN, hanya masa pakai server yang dibebankan, itu dapat dihapus kapan saja dan dibuat ulang.
