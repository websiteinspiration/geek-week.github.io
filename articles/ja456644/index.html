より安価なフォトポリマー3DプリンターがKickstarterに登場

キックスターターのサイトで、より長いフォトポリマー3Dプリンターの生産のための資金を調達する興味深いプロジェクトを通過できませんでした。



モデルはOrange 30と呼ばれます。このフォトポリマー3Dプリンター（SLA UV）は、誰でも利用できるシンプルで信頼性の高いデスクトップモデルです。 Orange 30の大きなプラスは、基本的な印刷に真剣なスキルを必要としないことです。製造業者のLongerは、3Dプリンティングビデオの基本を教えている人がいくつかいます。プリンタは非常に騒々しいわけではなく、可動部品や加熱部品がなく、家庭やオフィスで使用するのに実質的に安全です。

→ Longer3d.com公式ストア



Longer Orange 30フォトポリマープリンターは、20 x 20 x 39 cmの「タワー」で、ベースは巨大な金属製で、上部に遮光キャップが付いています。ちなみに、キャップはユーザーを保護するほどではなく、フォトポリマーを外部からの偽の照明から保護する必要があります。キャップの下には、フォトポリマー用のトレイと、モデルの作成時に印刷テーブルを徐々に上に持ち上げるZ軸ドライブがあります。ベースには、タッチスクリーン付きのコントロールボード、フォトポリマーを照らすためのUVマトリックス、2Kディスプレイ（解像度2560 x 1440ピクセルのLCDディスプレイ）があります。バスのサイズは120 x 68 mmで、Z軸に沿った最大作動ストロークは170 mmです。これは、人気のある予算のSparkmaker SLA 3Dプリンターを超えています。

パン屋になることを決めた場合、現時点では229ドルのレートが利用可能であることを覚えておいてください（オレンジ30プリンター+フォトポリマーボトル。）ロシア連邦での配送料は110ドルです。2019年8月にサンプルの発送を開始することを約束しています。



これは予算オプションの1つです。このプリンターを待って販売することができます。しかし、価格が上がる可能性があります（Kickstarterでは、取り消し線の価格は450ドルです）。上昇しない場合があります。以前のLongerモデルのOEMクローンは中国のストアで自由に販売されています（150ドル以上のFDMプリンター、250ドルからのSLA）。

設計はシンプルですが効果的です。MGN12レールは単一のZ軸に沿って取り付けられます。湾曲したシャフトではなく、通常の（中国の）レール。フォトポリマーにはメタルトレイを使用しています。はい、それは簡単で、鋲で刻印されていますが、プラスチックではありません。モデルを作るためのテーブルも金属製です。プリンタ開発者は、テーブル調整メカニズムを提供しています。これを行うには、プリンターメニューに対応する操作があります。

このモデルの大胆なプラスは、プリンター管理、印刷、プレビューモデルを備えたタッチスクリーンの存在です。組み込みのファイルビューアはどこにも見たことがありません。私の意見では、これは予算モデルのキラー機能です。



フォトポリマープリンターが必要な理由-高精細（20〜50ミクロン）の正確なモデル（コピー）を印刷するため。このようなプリンターは、歯科、宝石、モデリングなどで使用されます。3Dモデルを使用してキャストできる、キャスタブルなどの特殊なフォトポリマーがあります。



Longer3Dとそれ以前は、OEMモデルのメーカーとして、その後、Alfawiseなどの他のブランドの下に登場しました。

興味深いモデルのLonger Orange 120がカタログにあります。これは、ディスプレイと金属バスを備えた、改造されたSparkmakerです。 Longerの仲間たちは、予算プリンターを再設計し、主要な側枠を修正しました。

モデルLonger Orange 10に注意してください。これはOrange 30の「最年少」モデルです。同様のデザインですが、印刷領域はわずか98 x 55 x 140 mmで、マトリックスの「子」解像度（854 x 480ピクセル）もあります。そのようなプリンターの価格は小売りで$ 300よりわずかに安いです。製造元は、Orange 10モデルのセットアップと使用に関するビデオを公開しており、サイトではドライバー、スライサー、ファームウェア、および手順を見つけることができます。

あまり知られていないスタートアップがプロジェクトを開発し、頭に浮かび上がらせているのが好きです。さらに、ロンガーには他にも興味深いモデルがあります。他のメーカーがLongerからOEMライセンスを購入し、同じデバイスを自社ブランドで製造しているのは、当然です。納品後でも、このプリンターの価格は、人気のWanhao Duplicator 7とAnycubic Photonの半分の価格です。もちろん、一度に2セット購入することでさらに節約できます（$ 550 +送料$ 195）。実際、これらは1つのAnycubic Photonの価格で2Kマトリクスを備えた2つのフォトポリマープリンターです。



いずれにせよ、興味深いので、私はプロジェクトに従います。
