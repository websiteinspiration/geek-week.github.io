Raspberry Piを有益に使用する5つの方法 二部

こんにちは、ハブル。最初の部分、ラズベリーパイを使用する5つの方法を検討しました。このトピックは非常に興味深いものであることがわかりました。今日は、このマイクロコンピューターを効果的に使用する方法について、さらにいくつかのオプションを検討します。learn.adafruit.comからの写真 前のパートと同様に、プログラミングを必要としないメソッドを検討します。 興味のある方は、カットを続けてください。









1.監視カメラ



出典：www.raspberrypi-spy.co.uk/2017/04/raspberry-pi-zero-w-cctv-camera-with-motioneyeos

Raspberry Piは、ほとんどすべてのCCTVカメラで使用できます。
Raspberry Piを使用すると、次のことができます。

  • USBウェブカメラ（Logitech C910など）
  • PoEインジェクター付きのIPカメラ（Axisなど）（48Vの電力がネットワークケーブル経由でそのようなカメラに供給され、建物の外に運ぶことができます）
  • RPiコネクタに直接接続するカメラ（上の写真のように）。

ソフトウェアの設定には多くのオプションがあります。設定が非常に柔軟なMotionパッケージを使用できます。ffmpegを使用してコンソールから直接書き込むことができます。最後に、PythonとOpenCVを使用してハンドラーを記述できます。ビデオストリームのブロードキャスト、モーション検出器の使用、電子メールによる写真の送信など

を行うことができます。興味のある方は、次のチュートリアルをご覧ください。

  • www.bouvet.no/bouvet-deler/utbrudd/building-a-motion-activated-security-camera-with-the-raspberry-pi-zero
  • hackernoon.com/spy-your-pet-with-a-raspberry-pi-camera-server-e71bb74f79ea
  • medium.com/@gonzalovazquez/raspberry-pi-and-motioneye-setting-up-your-own-video-surveilance-95444e0faad

重要：前の部分で既に説明されていますが、繰り返すことをお勧めします。Raspberry Piでのリソースを大量に消費するタスク（ビデオ処理を含む）の場合、高品質のブランド2.5A電源が必要で、CPUにパッシブヒートシンクが望ましい（検索でraspberry piヒートシンクを入力すると、1〜2ドルで中国で安価に入手できる）。そうしないと、デバイスがフリーズしたり、ファイルコピーエラーなどが発生したりすることがあります。

2.音声録音




USBマイクを使用すると、Raspberry Pi をオーディオ録音用のバグの非常にコンパクトなデバイスとして使用できます。ここでも、ソフトウェアをセットアップするための多くのオプションがあります。ローカルでSDカードにファイルを書き込んだり、別のPCにブロードキャストしたり、ネットワーク経由でブロードキャストしたりできます。

繰り返すいくつかのチュートリアル：

  • iltabiai.github.io/2018/03/17/rpi-stream.html
  • blog.mutsuda.com/raspberry-pi-into-an-audio-spying-device-7a56e7a9090e
  • www.heidislab.com/tutorials/audio-streaming-with-raspberry-pi-zero-w-and-adafruits-i2s-mic

ちなみに、マイクを使用すると、Raspberry PiをAmazon Alexaで使用したり、デバイスを音声コマンドに使用したりできます。

3.教授 写真


n3とn1を混同しないでください。最初の段落では監視カメラについて説明しましたが、Raspberry Piはキヤノン、ニコン、ソニーなどのプロ用カメラも制御できます。USB経由でカメラをRaspberry Piに接続するだけで十分です。www.movi​​ngelectrons.net/blog/2017/08/09/Camera-Time-lapse-Controller-with-Python-and-Raspberry-Pi.html の


写真gphoto2およびlibgphoto2

ライブラリは、コマンドラインから機能する機能と、 PythonおよびC ++のインターフェイス。RaspberryPiを使用して、たとえばタイムラプス撮影用の「SLR」を制御できます。サポートされているカメラのリストは非常に大きく、10年前から現代までのほとんどすべてのモデルをカバーしています。 Libgphoto2は十分ですAPIを開発し、シャッターを制御できるだけでなく、設定の変更、ファイルのアップロードなども行うことができます。

繰り返しのチュートリアル：

  • pimylifeup.com/raspberry-pi-dslr-camera-control
  • medium.com/@cgulabrani/controlling-your-dslr-through-raspberry-pi-ad4896f5e225

ちなみに、カメラのメモリカードに画像を書き込むことも、Raspberry Piに直接画像を書き込むこともできます。たとえば、画像を「クラウド」に自動的にアップロードできます。一眼レフだけでなく天体（ZWO ASIなど）のカメラを制御するためのライブラリもあり、自動ガイドも含まれます。

4.気象ステーション


Raspberry PiはLinuxプログラムを実行する方法を「知っている」だけでなく、かなり開発された周辺機器（シリアル、I2C、SPI、GPIO）も備えています。このため、デバイスは、温度センサーや湿度センサーからガイガーカウンターに基づく線量計まで、さまざまなセンサーからデータを収集して送信するのにほぼ理想的です。


Webサイトの写真www.raspberrypi.org/blog/build-your-own-weather-station

ちなみに、完全に怠惰な場合は、センサーだけでなくweb-aからもデータを取得でき、このオプションには存在する権利もあります。ただし、Raspberry Pi用のセンサーを搭載したボードは、簡単に個別に購入できます。

学習するチュートリアル：

  • www.instructables.com/id/Build-a-Raspberry-Pi-SUPER-Weather-Station
