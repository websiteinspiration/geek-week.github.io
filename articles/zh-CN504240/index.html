第6部分。职业程序员。自己创业

延续故事“职业程序员”。

整整一年我都没有碰过“程序员的职业生涯”这本书，而且显然没有白费。在过去的几个月中，我经历了许多变革，这些变革将用有关如何创建和发展IT公司的材料来补充这个故事。看起来很自然。如果您是一位雄心勃勃的程序员或自由职业者，那么迟早您会问自己：“下一步呢？”。现在，我已经是高级开发人员了，然后呢？被任命为建筑师-下一步是什么？..

但是让我们回到我们停下来的地方。最后零件编号5。程序员职业最终结束了我作为自由开发人员的职业生涯，并打算将我所有的积蓄投资于创建初创公司。自从第5章出版以来，我已经收到了数十封有关Habr的私人信息，甚至在社交网络上也有数条私人信息，询问有关本书的延续性。对于Habr的许多读者而言，该阅读证明是有用和有益的。并且由于每个业务都需要完成并且对此业务有需求，因此我急于继续分享我的经验。

在本章中，您将学习：

  • IT项目的疯狂想法是如何产生的，以及实现此目标的先决条件
  • 关于放弃一切并创建自己的创业公司或企业的人的动机。此外，他不仅提供舒适而有利可图的工作，而且还彻底改变了他的居住地和环境。
  • 如何筹集启动资金
  • 关于从头开始创建团队
  • 关于创始人在项目开发和发展中的作用
  • 关于寻找投资及其在现实中的发生方式
  • 关于错误和后果
  • 还有关于本章生活相对幸福的结局

本章涵盖从2011年底到2013年底的时间段。

理念


怀念孩子始于激情。带着无法抑制的压倒性情绪，这转变成了可以翻滚山脉的超级力量。这种力量是如此之大，它赋予了新的生命。向地球介绍新居民。但是在婴儿出生之前，必须经过9个月的准备和有机成长。从一个小细胞到由十亿个其他细胞组成的极其复杂的生物和谐地互连。如您所知，有9个母亲不会在一个月内“释放”婴儿，因此此过程以及构建您的IT项目的关键是和谐一致的成长。

我希望从概念的诞生到孩子的诞生之间的这种相似之处能够传达出一个IT新手的大概情绪，从一个想法的诞生到第一个版本向公众发布。

就我而言，从编写第一个程序的那一刻起，启动我的想法似乎就一直在我的脑海中。在第4章中，我提到在大学里我开发了自己的C ++编译器，以便通过知识与RSDN.ru接触那些the的叔叔。在另一篇文章“ 我编写C ++编译器 ”中对此故事进行了更详细的描述。

我着迷于为其他程序员创建出色的东西的想法，我经常在笔记本中写到如何在我最喜欢的IDE中自动执行特定过程。我在桌面上也有一个文件，在那里我写出了工作期间编写代码的所有例行任务。

是否重复一千次

for(int i = 0; i < items.count; i++)

或在C ++中创建Get / Set方法。所有这些日常重复动作都表明开发工具远非完美。

此外，我开始更加想知道-“ 将来的编程会是什么样？ ”。因此在10-20年内。

想到2010年初，我想像机器会在10年内自行编写程序，而开发人员将在透明的监视器附近挥动双手，如影片“密码” Swordfish中所示。


十年过去了，人工智能和机器学习技术已经普及。就像Kinect等非接触式触摸控制器一样。但是，当我在Visual Studio中编写代码时，我正在写。仅版本号已更改。在过去的十年中没有革命性的成果！

启动资金


但是回到2011年底。当时，我同时作为两个项目的自由开发人员勤奋工作。我的目标是迅速为公寓筹集资金，然后从父母那里搬到自己的家中。而且我在沙发下放的钱越多，我对自己和明天的信心就越大。现在已经积累了5,000美元。我开始研究您可以以这种金额购买哪些类型的公寓，即使是信用也可以。公平地说，买房的想法是父母提出的。就其本身而言，它并没有点燃我。到那时我已经有车了，所以我开始着眼于创建自己的业务并为此积累启动资金。

在我的业余时间里，我浏览并阅读了当时有关在网络上开展业务的所有内容。信息很少。我最喜欢的资源是Zuckerberg Calls（现在为vc.ru），后来使我成为了IT企业家领域有影响力的人。

但是，该博客页面和社交网络上显示的所有内容似乎都与我所在的省市相距甚远。如此遥远，当有人告诉他他如何筹集数万美元的投资时，我似乎正在看电影。

然后我去了我的省级庭院，庭院由四个侧面的1960年代初期的赫鲁晓夫装饰。与阴沉的树木和阴郁的人。剩下的就是闭上我的眼睛，开始想象我对未来的开始会是什么样的。突破想法百万。

我走着做梦。在此过程中，我为过渡中的无家可归者s绊绊，我想到了带有触摸界面的设备的潜力。我想像过如何用几根手指一次在屏幕上扭曲，转动一个假想的齿轮，这引起了欣喜，并渴望为软件开发领域创建类似的东西。

在不到两年的努力中，我节省了20,000美元。我什么都没买，食物很便宜，大部分是我父母买的。我没有每月帐单，也没有家人。因此，自由职业者赚取的95％直接折叠在沙发下。在我的生活中，我已经不止一次地参加了这样的累积性马拉松比赛，包括现在，当我拥有大量账户和一个大家庭时，我已经不止一次。

有了这些节省，我终于得出结论，您需要启动IT项目。

从构想到实施


因此，有一个疯狂的主意。受购买的新iPad 3，电影《钢铁侠》的启发，并拥有多年的开发经验，我决定创建一个可视化的编程环境。或者简单地说，就是带有触摸界面的IDE。有一个主意，我开始草拟有关功能的注释。

UI / UX的基础是能够在IDE空间中移动组件，类，方法和接口，同时将它们连接在一起并通过单击
组成程序的块来深入研究细节。

可视化编程的想法令人震惊。她极大地激发了想象力。但是我无法说出这样一种工具在生产率方面真正有用的程度。不过，我当时的口号是：“主要是要做事，然后做会发生的事情。”

在我的团队中开始寻找艺术家之后，我遇到了来自RSDN.ru网站的几个开发人员。他们准备开始开发。但是，我想到了一个致命的想法，即创始人无需参与开发。实际上，我并不是真正想要的，我更关心的是创建一家真正的企业并筹集投资。使一切都像其他成功的创业公司一样美丽和丰富。那时我做出的许多决定都是基于直觉以及我自己的自我。没有人可以学习和学习。

一种或另一种方式是，当我想到这个想法时，我几乎立即雇用了第一个人到启动团队。我给他做了一个测试任务-为iPad创建一个简单的应用程序，它使您可以在屏幕上移动块并将其与箭头连接。候选人迅速完成了这项任务，并以他每小时10美元的小费率，立即被录用。

更进一步。我看到了“哦，奇迹！”，开发人员完成了我给他的任务。我决定从oDesk聘请更多人。因此，从我昨天的疯狂想法开始，一个由8个开发人员组成的公司（针对任何平台）迅速成立。

我的计划是创建桌面，Web和移动应用程序的原型，并与之一起提供给投资者。那时我不知道另一种方式。但是，没有人在讲俄语的创业社区中培养出另一种方式。信息空间的表面上只有一件事：您创建一个原型，编写一个业务计划，要钱...获利！

为什么要联网


该团队在我的领导下开发了原型。同时，我开始着眼于吸引投资。这项投资的规模对我来说是未知的，我所持的从未超过20-25,000美元。因此，一开始它是根据公开发布的数字进行指导的。仅仅过了一段时间，我试图写一份商业计划书，并从收到的500,000美元中跌入一个昏昏欲睡的地方，只花了8个人就支付了发展年的费用。

在一个疯狂的想法下寻找这样的投资就是这样的想法。也许在帕洛阿尔托的某个地方，它可以和一个喝醉了的亿万富翁一起工作，但是在那些年的独联体国家却没有。虽然有不同的故事...

当我仍然与父母住在一起时，我凭直觉开始寻求结识至少在某种程度上与投资IT初创企业有交集或自己创办他们的每个人。当时，乌克兰几乎没有此类事件。在2012年夏季，我仅参加了在该国另一端举行的2个活动。

但是，这使我能够达到以前未知的水平。人格发展水平。来自IT部门的人们的举止，他们的自信，他们的疯狂能量使我受到启发。这一切都炸破了屋顶，来自一个完全不同的世界-通过窥镜。在这个世界上，没有国内吸毒者，阿妈和一个我住了26年的阴郁城市。

那时，我遇到了许多IT企业家，他们仍然被社交网络列为我的朋友。其中包括VKontakte的创始人，当时的乌克兰和俄罗斯部长的亲戚和朋友，以及许多IT领域的千万富翁投资者。

尽管我看上去更像昨天的学生，但除了“我的团队使用触摸屏构建编程环境”之外，我什么也没提供。尽管如此，这个世界上许多有权势的人还是保持联系，至少给了名片。而且，如果您设法通过第三方组织会议，则可以提出您的想法几分钟。

积极的投资搜寻阶段


在2012年，我从郁闷的定居点搬到了基辅。烧毁了所有桥梁和通讯。完成自由职业者的项目。他把自己的东西和一台电脑装在格子袋中，离开父母，创造了美好的未来。在基辅，我几乎没有熟人，所以我必须自己做所有事情。说很难，什么也没说。如此疯狂的事情发生在我身上，以至于将它们公开露面都是自杀。我只能说我不得不经常叫救护车。还有我公寓的主人，不止一次地想将我踢出各种浅滩（毕竟，我以前并不孤单，在旅途中学到了很多东西）。

但是我仍然住在基辅，尽管没有人相信我，而我的亲戚不断地把我拉回到他们的拉斯维加斯。相反，今天，我给亲戚买了一套公寓，然后叫他们搬到我这里。最主要的是不要放弃。

所以，那时我只需要电脑，桌子和椅子。第一个租住的公寓里没有桌子，所以我不得不为一个夏季小屋找到一个折叠的选择，它几乎不能放在床和电视之间，放在一个小的odnushka中。

但是此举的想法是在步行距离之内找到正确的联系和投资。因此，从2012年9月到2013年2月，我约有20位投资者向他们介绍了我的项目。我在会议和活动，社交网络和新闻中找到了它们。通过Facebook直接联系。我没有什么可失去的。我清楚地知道，当我的积蓄用光时，就不会回头。

这将是情感上的死亡。我不能回去，但是要留在基辅，我将没有钱。毕竟，自由职业者也结束了。

我受到英国尤利乌斯·凯撒（Julius Caesar）竞选活动的启发。当他将军队降落在岛上后，他命令将船只焚毁，以使士兵们没有机会逃跑。

士兵占领了该岛。我处于类似情况，因为无处可跑。如果有可能自我激励，那就尽可能多地这样做。要么全部要么一无所有。

但是实际情况有所不同。尽管普遍希望获得投资并成立一家全球性初创公司，但钱却在我们眼前融化了，我建议与之投资的人也不急于写出五位数的支票。

现在我所有的积蓄还剩下几百美元。开发商无需支付任何费用，租用公寓无需支付任何费用，没有钱。顺便说一句，没有原型产品能够捕捉并引起令人赞叹的效果。我只是在墙上贴了一张纸胶带，上面刻着“黎明前最黑暗的时刻”的字样，然后继续努力。

出于绝望，我完全转向了一位在基辅进行业务培训且与IT无关的局外人。我在舞台旁一千个观众面前，向他展示了iPad上的演示。我们聊了大约10分钟，而听众却因期待而沮丧。最后，他对事情的真相作出了判决：“我在平板电脑屏幕上看到一些方块在移动，您要我要几万美元。我不会投资。”

但是，这并没有阻止我，尽管未来受到威胁，我仍继续走着走，向人们介绍当时的情况。具有IT技能的人员会提供实用的建议，如果我听了他们的话，一切可能会有所不同。有人建议让儿童适应该应用程序，并以这种方式教他们编程。在我看来，这个想法，即使不是冒犯性的，也是如此。
什么孩子？你是做什么的 ？！我想在软件开发领域进行一场革命，您建议我开发玩具？是的，是的，多年以后分析自己是一件好事。

即使我完全停止了开发工作，我仍然继续朝着封闭的大门前进。他从银行借了一笔小额贷款来支付租房的费用。我父母给我送了食物，或者我未来的妻子给了我。因此，尽管我已经破产，并且在道德上因拒绝而沮丧，但他们还是从新开业的创业加速器给我打电话，并邀请我接受采访。

实际上，他们说：“一切都会解决-您只需要生病即可。”在情感上和财务上，我处于零状态，但我发现有能力踏入公司潜在的办公室。好消息在那儿等着我。加速器和兼职投资者的创始人非常熟悉MySQL和GitHub的创始人。如果可能的话，他可以向我介绍他们。

这个孵化器也吸引了大约一百万美元的资金，他们很高兴投资有前途的发展。最后，我被视为孵化器的首批居民之一，并承诺将以各种方式进行合作。但是，我不得不靠某种东西为生，支付房租和食物。此外，该团队已经很长时间没有从事该项目了。但是，我还是结识了来自IT界的这些影响者，并从他们那里借了足够的知识。

总


尽管有这些过山车，但在人的能力极限下，我对这些疯狂的半年却收获颇丰。我遇到了许多来自IT创业领域的名人。获得了很多经验，了解了创建和开发启动程序的过程。

而且，正是搬到基辅的举动给了我我的妻子，这在很大程度上要归功于我在那个困难的时候最终没有淹死。她的支持和帮助使我摆脱了困境。

经过八年的努力，我不寒而栗，获得了知识和联系，再次开始了IT业务。我将在上一章中告诉您。
