En-têtes HTTP inattendus

Il y a quelques jours, je sélectionnais un blog de Creditkarma et j'ai remarqué cet en-tête HTTP:

X-hacker: If you're reading this, you should visit wpvip.com/careers and apply to join the fun, mention this header.

(X-hacker: si vous lisez ceci, vous devriez aller sur wpvip.com/careers et postuler pour vous amuser, mentionnez cet en-tête).

La première pensée a été: "Wow, une fois que les programmeurs ont introduit un bogue du millénaire pour enregistrer quelques bits sur la date, et maintenant la société publie des offres entières sur le travail dans l'en-tête HTTP!"

Je suis devenu très intéressé et j'ai fait des recherches.

Ce titre particulier semble être «standard» si vous hébergez le site sur WordPress VIP , la société d'hébergement WordPress hébergée par Automattic. Le même titre peut être trouvé sur de nombreux sites bien connus, tels que:

  • nypost.com
  • techcrunch.com
  • www.nbcolympics.com
  • www.thesun.co.uk
  • plusieurs milliers de plus

Les développeurs de sites Web et les propriétaires de sites Web peuvent le désactiver, mais pour être honnête, je doute qu'ils sachent même que cet en-tête est inclus dans chaque paquet HTTP. Et bien sûr, la deuxième pensée est venue tout de suite - pour vérifier si d'autres sociétés ont des titres créatifs.

Les résultats sont incroyables!

Vous pouvez trouver différentes offres d'emploi dans les en-têtes HTTP.


Oui! Les entreprises les plus cool du monde semblent publier des offres d'emploi dans l'en-tête HTTP "standard" x-recruiting.

Voici quelques exemples:

PayPal.me

x-recruiting: If you are reading this, maybe you should be working at PayPal instead! Check out www.paypal.com/us/webapps/mpp/paypal-jobs

Booking.com

x-recruiting: Like HTTP headers? Come write ours: careers.booking.com

Etsy.com

x-recruiting: Is code your craft? www.etsy.com/careers

Otto.de

x-recruiting: Seems you like http headers. To write ours, apply at job.otto.de and mention this header.

Vous voulez voir la liste complète? Pour ce faire, j'ai ouvert le référentiel sur GitHub .

En plus des offres d'emploi, des choses plus créatives ont été trouvées dans le processus de recherche . Ils m'ont vraiment excité en tant que grand fan de non-sens mystérieux .

En-têtes HTTP mystérieux


Le site Web 9kw.eu, qui semble être un fournisseur de captcha, affiche le message secret "42":

X-Secret-Message: 42

istreetview.com n'est plus pris en charge, mais l'adresse du formulaire Web secret peut être trouvée dans l'en-tête.

X-Secret-URL: https://appio.link/secret

Je leur ai écrit ma réponse ...

Sur thetradersdomain.com il y a une "sauce cachée" dans les en-têtes, mais c'est confidentiel:

x-secret-sauce: Confidential

Sur images-dnxlive.com il y a quelques liens plus "secrets" dans l'un des en-têtes HTTP:

X-Secret-Message: camscv.dnxnetwork.lu

Si vous aimez les voitures de luxe, voici un jaguar. ro avec un en-tête pour détecter les bots:

X-Bot: false

Mais cette méthode ne fonctionne pas très bien et échoue si vous remplacez l'agent utilisateur (désolé, les gars de Jaguar).

Et pourtant ... avez-vous déjà vu un serveur avec un surnom ? Il y en a un couple ici:

X-men.com

X-ServerNickName: clint

Howgoodisyourseo.com

X-ServerNickName: The Internet

Et le dernier, mais aussi un exemple intéressant. Nos amis de m.bidorbuy.co.ke ont exprimé leur passion dans les en-têtes HTTP:

x-powered-by: Passion and tiny cute kittens
x-servernickname: The Beast
x-hacker: If you are reading this, maybe you should be working at bidorbuy instead
