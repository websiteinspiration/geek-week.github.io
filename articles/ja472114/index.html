ゲームコンソールと自分でできるゲーム。リトルゲームエンジン+ ESPboy

Little Game Engine（LGE）は架空の構成の仮想ゲームコンソールであり、架空のプロセッサとオンラインのWeb-SDKを備えており、Cのようなコードコンパイラーをアセンブラーに組み込み、さらにマシンコードに再コンパイルして、このコードを同じ場所でオンラインで実行する可能性があります。エミュレータ。



名前が示すように、2Dレトロゲームをすばやく作成できるように、セット全体が研ぎ澄まされています。 20以上のゲームが準備されており、いくつかは開発中です。



オンラインでプレイするには、以下のゲームリンクをたどり、「コンパイル」、「実行」の順に選択します。

TankCity、1916、FourInaRow、BlackJack、ZombieDefence、MicroRace、DwarfClicker、銀河、思い出、NinjaEscape、鉱山、ブレイクアウト、タワーディフェンス、FlappyBird、WormBlast、ESProgue、Snake、FishLife、Columns、MarsAttack、CityRunner、Asteroids、Bashe ...



もう1つの利点は、オンラインバージョンに加えて、LGE仮想マシンエミュレーターが、ESPboyガジェット用に書かれていることです。ESPboyガジェットは、ハードウェアに組み込まれたLGE VMに似た周辺機器を持ち、私がすでに書いています。

  • 128x128カラー画面、
  • 8ボタン
  • シングルビットサウンド
  • RGB LED
  • エミュレーション速度に適したESP8266マイクロコントローラー
  • SPIFFSファイルシステムを備えた内蔵フラッシュメモリ


このようにして、オンラインのLGE SDKにコンパイルされたゲームをポータブルESPboyにダウンロードし、持ち歩いて、興味を持って無料で過ごすことができます。



ゲームは有線（spiffsでのファイルのダウンロードを参照）とWiFiの両方でダウンロードできます。ESPboyを起動するときに「B」ボタンを押したまま、ガジェットをアクセスポイントに変え、ブラウザーを介して「ESPboy」という名前のWiFiネットワークに接続すると、ファイルシステムのWebインターフェイスが表示され、ファイルを削除したり、新しいファイルをアップロードしたりできます。



ガジェットを組み立てるためだけに残っています。ガジェットには、図、指示、組み立て用キットキットがあり、間もなくtindie.comで入手可能になります。
既存のゲームで十分にプレイしたので、すぐに独自のゲームを書き始めることができます。

オペコードで直接記述しないようにするために、SDKには自己記述型コンパイラ「LGE C」が含まれています。これは、「C」の比喩的な高級言語です。現時点では、このコンパイラーはC言語標準を完全にサポートしているわけではなく、コンパイル時に、理解できない場所で理解できないエラーが発生する可能性があります。しかし、ソースコードの行数が2000行未満であり、常に進化しているため、高速です。

しつこくて興味がある人のために、以下はLGE SDKでのゲームの短い例です。コンパイルされた形式では、100行未満で、1 KB以下です。実行するには、オンラインのLGE SDKにアクセスし、コードを「ソース」ボックスにコピーして、「コンパイル」を選択してから「実行」する必要があります。

LGE Cでの簡単なゲームの例
int stickCount;
char key,previouseKey,takenSticks;
void redraw(){
	int i;
	//  
	setcolor(2);
	//  
	for(i = 0; i < stickCount; i++)
		line(22 + i * 6, 74, 22 + i * 6, 84);
	//  
	setcolor(11);
	// 
	for(i = stickCount; i < 15; i++)
		line(22 + i * 6, 74, 22 + i * 6, 84);
	//    
	setcolor(1);
	//  
	delayredraw();
}

void playersMove(){
	//    ,     . 
	while(key == previouseKey){
		key = getkey();
	}
	while(key != KEY_LEFT && key != KEY_DOWN && key != KEY_RIGHT){
		key = getkey();
	}
	if(key & KEY_LEFT){
		takenSticks = 1;
	}else if(key & KEY_DOWN){
		takenSticks = 2;
	}else{
		takenSticks = 3;
	}
	printf("%d, ", takenSticks);
	stickCount -= takenSticks;
	previouseKey = key;
}

void computersMove(){
	if(stickCount % 4){
		//   ,   
		takenSticks = stickCount % 4;
	}else{
		//     
		takenSticks = 1 + random(1);
	}
	stickCount -= takenSticks;
	printf("%d, ", takenSticks);
}

void game(){
	//
	stickCount = 15;
	clearscreen();
	//      
	gotoxy(8,0);
	puts("");
	gotoxy(2,1);
	puts(" 1,2  3 .  ,       . :\n");
	// 27,25  26  
	printf(" %c 1    %c 2    %c 3", 27, 25, 26);
	gotoxy(0,12);
	redraw();
	while(1){
		playersMove();
		if(stickCount <= 0){
			gotoxy(3,8);
			puts(" ");
			return;
		}
		redraw();
		computersMove();
		redraw();
		if(stickCount <= 0){
			gotoxy(3,8);
			puts(" ");
			return;
		}
	}
}

		game();

}
