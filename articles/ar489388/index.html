1. تنفيذ أمان إدارة الموارد في Azure


في غضون 3-4 ساعات فقط ، ستتعلم كيفية حماية الموارد بالسياسات والتحكم في الوصول المستند إلى الأدوار وخدمات Azure الأخرى.

من بين وحدات الدورة:

  • السلامة والمسؤولية والثقة في Azure
  • فرض ومراقبة معايير البنية التحتية باستخدام Azure Policy
  • مراقبة موارد Azure وتنظيمها باستخدام Azure Resource Manager
  • تأمين موارد Azure باستخدام التحكم في الوصول المستند إلى الدور (RBAC)

2. إدارة الهوية والوصول في Azure


في غضون 5-6 ساعات ، ستتعلم كيفية العمل مع الاشتراكات والمستخدمين والمجموعات من خلال تكوين Microsoft Azure Active Directory لأحمال العمل.

وحدات الدورة:

  • السلامة والمسؤولية والثقة في Azure
  • قم بإنشاء حساب Azure
  • إدارة المستخدمين والمجموعات في Azure Active Directory
  • إنشاء مستخدمي ومجموعات Azure في Azure Active Directory
  • تأمين تطبيقك باستخدام OpenID Connect و Azure AD
  • حماية مستخدمي Azure Active Directory بمصادقة متعددة العوامل
  • السماح للمستخدمين بإعادة تعيين كلمة المرور الخاصة بهم باستخدام كلمة مرور ذاتية التعيين في Azure Active Directory

3. تنفيذ أمان الشبكة في Azure


حوالي 6 ساعات من التعلم وستتعلم كيفية إعداد وحماية وعزل الشبكات في Azure. تعرف على المزيد حول الأمان والثقة في Azure ، وإعداد شبكة للأجهزة الافتراضية ، وتأمين الوصول إلى موارد Azure وعزلها باستخدام مجموعات أمان الشبكة ونقاط نهاية الخدمة ، والتشفير الشامل لحركة مرور الشبكة باستخدام بوابة تطبيق Azure ، ومراقبة البنية التحتية المعقدة للشبكة واستكشاف أخطائها وإصلاحها أزور مع أدوات مراقبة الشبكة. نضم الان!
