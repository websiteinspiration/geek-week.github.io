NotionとPythonを使用してホームライブラリを作成する

電子図書館で本をよりよく配布する方法を常に考えていました。その結果、私はページ数やその他の便利な機能を自動計算してこのオプションを選択しました。猫の下に興味のある方全員にお願いします。


パート1. Dropbox


私が持っているすべての本はドロップボックスにあります。私がすべてに分けた4つのカテゴリがあります：教科書、リファレンス、芸術、非芸術。しかし、私は参考書をタブレットに追加しません。


本のほとんどは.epub、残りは.pdfです。つまり、最終的なソリューションは、どちらのオプションも何らかの形でカバーする必要があります。


本への道はこのようなものです：


///// /      .epub

本がフィクションの場合、カテゴリ（つまり、上記の場合は「デザイン」）は削除されます。


私はDropbox APIに煩わされないことにしました。幸いにも、フォルダーを同期するアプリケーションがあります。つまり、計画は次のとおりです。本をフォルダーから取り出し、各本をワードカウンターで実行し、それをNotionに追加します。

パート4.すべてのパーツの接続


ブックフォルダー内のすべての可能なパスを回避する必要があります。Notionにすでに本があるかどうかを確認します。ある場合は、行を作成する必要はありません。
次に、ファイルの種類を決定する必要があります。これに応じて、単語数を数えます。最後に本を追加します。

for root, subdirs, files in os.walk(BOOKS_DIR):

def check_for_existence(filename):
    for row in current_rows:
        if row.title in filename:
            return True

        elif filename in row.title:
            return True

    return False
