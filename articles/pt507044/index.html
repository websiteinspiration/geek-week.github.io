A placa mais acessível para experimentar SoC'om MCU e FPGA

A tecnologia está avançando. E há vários anos, existem SoCs e MCU + FPGAs. E no
KDPV um cachecol apenas com esse SoC'om.



Esta é a placa-mãe Microsemi SmartFusion 2. Quem se importa com o tipo de taxa que eu peço para o gato. Haverá uma breve revisão.

Na verdade, no KDPV SmartFusion 2 Maker-Board da Digi-Key. O preço em seu site é de cerca de
US $ 35.

Programa educacional










  • LED’;
  • SPI flash 2MB;
  • 1G ethernet;
  • Light sensor;
  • FTDI/JTAG/USB;
  • M2S010 SoC Microsemi/Microchip.

SoC M2S010


  • Cortex-M3 166MHz;
  • : Flash 256K/SRAM 64K;
  • FPGA 10k LE;
  • : CAN/USB/UART/I 2C/SPI;
  • DDR(PCIe/SerDes VFGx);

Essa. este pequeno SoC é um animal sério, recheado pelo menor mimo. E sua aplicação pode ser muito ampla: das máquinas CNC à carne picada (controle do motor, etc.). Gostei dessa pedra porque é muito mais barata que a Zynq / CycloneV.

E será mais fácil aprender.

Para trabalhar com essa pedra, você precisa de alguns pacotes de software: Libero SoC (para configurar lógica programável) e SoftConsole (para desenvolver software para Cortex-M3). Conselho para aqueles que querem tentar fazer algo com esta pedra: use o sistema listado nos suportados. No meu caso, este é o Centos 7. Instalar software é muito difícil, para dizer o mínimo.

Impressões


Em geral, o conselho é muito interessante. Especialmente pelo seu preço. Você pode jogar com o FPGA e aprender algo novo, também com o MCU. Aprender ferro é simplesmente fantástico. Mas foi muito, muito difícil para mim piscar os LEDs. lidar com software é bastante difícil: começar na instalação, terminar com o lançamento de uma licença e dançar com os ambientes de desenvolvimento.
