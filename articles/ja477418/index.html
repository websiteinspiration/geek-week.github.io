3ステップでの3面屈折

3Dオブジェクトをレンダリングするときは、常にいくつかのマテリアルをオブジェクトに追加して、オブジェクトが表示され、希望どおりに見えるようにする必要があります。これを特別なプログラムで行うか、WebGLを介してリアルタイムで行うかは問題ではありません。


ほとんどのマテリアルは、Three.jsなどのライブラリの組み込みツールを使用してシミュレートできますが、このチュートリアルでは、3つのステップで、オブジェクトをガラスのようにする方法を説明します。


ステップ1：セットアップと前面反射


この例では、ダイヤモンドの形状を使用しますが、単純な立方体または他の形状を使用できます。


プロジェクトを設定しましょう。ビジュアライザ、シーン、カメラ、ジオメトリが必要です。私たちの表面を視覚化するためには、材料が必要です。この教材の作成がレッスンの主な目的になります。それでは、先に進んで、頂点シェーダーとフラグメントシェーダーを含む新しいSharedMaterialオブジェクトを作成します。


あなたの期待に反して、私たちの素材は透明ではなく、実際にはダイヤモンドの背後にあるものを歪めます。これを行うには、テクスチャ内のシーン（ダイヤモンドなし）を視覚化する必要があります。直交カメラを使用してスコープ全体のサイズの平面をレンダリングするだけですが、他のオブジェクトを使用してシーンをレンダリングすることもできます。Three.jsでダイヤモンドから背景表面を分離する最も簡単な方法は、レイヤーを使用することです。

uniform sampler2D envMap;
uniform vec2 resolution;

varying vec3 worldNormal;
varying vec3 viewDirection;

float Fresnel(vec3 eyeVector, vec3 worldNormal) {
    return pow( 1.0 + dot( eyeVector, worldNormal), 3.0 );
}

void main() {
    // get screen coordinates
    vec2 uv = gl_FragCoord.xy / resolution;

    vec3 normal = worldNormal;
    // calculate refraction and add to the screen coordinates
    vec3 refracted = refract(eyeVector, normal, 1.0/ior);
    uv += refracted.xy;

    // sample the background texture
    vec4 tex = texture2D(envMap, uv);

    vec4 output = tex;

    // calculate the Fresnel ratio
    float f = Fresnel(eyeVector, normal);

    // mix the refraction color and reflection color
    output.rgb = mix(output.rgb, vec3(1.0), f);

    gl_FragColor = vec4(output.rgb, 1.0);
}
