初心者を理解する方法、希望する職業のために選択する言語は？

免責事項：このテキストは、13歳未満の子供（別のプライバシーポリシーを確立する必要があります）、高齢者、中間者、メンタリティが弱い人、VBAとALGOL言語の愛好家、および妊娠中の女性に読むことはお勧めしません。上記のすべては不十分な作者の個人的な意見であり、静的および動的型付けの支持者の間で不一致を蒔くことを目的としてここでは述べられていません。いきましょう：新しいスレッド（）開始（）



正直なところ、この記事は長い間熟成してきました。最近、私が議論、ホリバール、および同様の記事に出くわしているためです。そこで、著者と参加者は、さまざまな仮説をタイトルの質問への回答として最大限に理解して提案します。そして、それらの全体の意味は、上に書かれた華やかな文と同じ混乱する議論に要約されます。

私は、精子から、パスカルとウィンドウ3.11を愛する10代の若者から、ロールと端末以外は好きではない男まで、長い道のりを歩んできました。だから、私はあなたにすべてを説明し、おそらくいくつかの写真を見せます。行け。

それでは、どのプログラミング言語を選ぶべきでしょうか？


このようなすべての質問の愚かさは、真空の中で条件付き球状プログラマーになることはできないということです。プログラマーは現在、ソフトウェアとハ​​ードウェアが二次的である過去の離散数学者ではありません。ほとんどの場合、プログラマーは現在、テクノロジーの現在の「窓」の枠組みの中で働く職人です。さらに、これはあなたが走っている移動中の車の「窓」です。テクノロジーは常に変化しており、アリスキャロルのように、私たちはその場に留まるために逃げることを余儀なくされています。少なくとも私のお気に入りのGoogleを使用する場合、1つのAndroid APIを理解する時間がなく、すでに廃止された機能の半分があります。それらもネジを締めます...
ええ、はい、はい、初心者向けの記事のように、わかります。そもそも、自分が何をしたいのかを正確に判断する必要がありますか？面積、まあ、少なくともおよそ？

何もわからない？


アルゴリズム構成の学習を開始します。分岐、ループ、データ構造、ブロック図。アルゴリズムとブロック図を実際のプログラミング言語に転送（Wikipediaからのコピーアンドペーストではありません！）するのに役立ちます。

基礎を築きたい場合、どの言語から始めますか？


原則として。単純なプログラムの書き方を学び、変数、演算子を理解し、最大検索、並べ替え、二分検索などの古典的なアルゴリズムを探るだけでよいので、次にインタビューで尋ねられるので、OOPとデータ構造に取り掛かります。ベクトルがリストとどのように異なるか！

しかし、適用されたタスクのための言語が必要な場合はどうでしょうか？
はい、あなたはプログラマーになりたくありません。たぶん、あなたはDNAチェーンを使ってテキストファイルを解析する必要がある科学者、またはトレーディングロボットを書くというアイデアを持つ投資家です。ここで私はPythonに助言します、他の言語の支持者が私を許すかもしれません。 Pythonはめまい機能と低いエントリしきい値を備えた言語です。最初は、これはある種の簡単な「教育的」言語であることが初心者にさえ思われます（コルーチンに入れて機能的なものにするまで）。

さて、あなたは言いますが、次にどこへ行くのですか？単一のpythonではない...？もちろん違います。どこに引っ張っているの？ウェブ？デスクトップ？スクリプト？システムレベル携帯電話？あなた、親愛なる、決定、普遍的な解決策はありません！

サイト構築のように？さて、ここではオウムのように書いてみましょう：html、javascript、およびそれらのような他のもの（まだdjangoを備えたpythonとそこに静かに潜んでいる他のもの）

過酷なWebサービスが必要ですか？ PHP、node.js、ruby、Java、および同じPython。

ハードウェアとオペレーティングシステムの近くになりたいですか？ C / C ++を掴みます（誰もがキャストに行ったとしても、自分の足を撃ってカットしないでください）。ちなみに、私はすべてのプログラマーに、少なくともC / C ++の基本とSTLの基本クラスを扱うようにアドバイスします。そうすれば、GCを別の目で見るのに役立ちます。鉄片が頭に落ちた場合は、アセンブラについて学びます。

スクリプト、解析、マイニング、ビッグデータの操作、モデルの構築、ボットの作成、OSの管理などを行いたいですか？すでに発声されたPython3 +を取り上げてください。データでさらに作業する場合-あなたはまだRを取ることができます。

デスクトップアプリケーションを書くことを夢見ていますか？ここに何でも！デルファイはそれを取るが！もちろん冗談です。ここでは、書き込むシステムと、クロスプラットフォームが必要かどうかを確認する必要があります。たとえば、PythonまたはJavaコードは移植可能です。 C / C ++は、プラットフォームごとに個別にビルドする必要があります。

Javaコアを知りたいですか？したがって、Javaが必要な理由はすでにわかっています。わかりません？なぜそれが必要なのか、今どのように説明しますか。空気を見た？それにもかかわらず、それなしでは生きられない。

もちろん、たとえばKotlinなどの派手なものもあります。クールなことに、この言語は、NPEのニックネームである私の古い友人を殺したので、彼を殺しました-彼は負傷しました！ Javaを知らない人のために普通に勉強する方法がわからなかっただけなのですが。いいえ、「Hello world」はJavaなしで簡単に書かれていますが、それ以上はわかりません。

たぶん、あなたはゲームを書くことに決めましたか？多くの既製のデザイナー、同じUnity3D、2次元エンジンがあり、通常は一連のプラットフォームのすぐ下にあります。あなたはすでに自分自身を見ています：C＃、Javaなど。そしてこれらすべては、さまざまなデータ転送プロトコル、オペレーティングシステム、フレームワーク、暗号化と暗号化、承認と認証、XML、JSON、データベーステクノロジー、アプリケーション保護の知識によって上から磨かれています。バージョン管理システムなど。

次に、プログラマーになるための最大の秘密、プログラムを公開します。なんでも！誰もがそれを必要としないときでさえ、プログラムを書いてください。他の人ではなく、必要なときに書いてください。常に無能に対して休み、これらのギャップを埋め、再び新しいギャップに対して休みます。そして、あなたの頭のすべての可能なキャッシュに十分なメモリがある限り。

他にどのようにして「どの言語を学ぶべきか」という質問に答えることができます。明らかにあなたが話をするつもりです。
