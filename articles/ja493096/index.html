iOSでダークテーマを追加する

みなさん、こんにちは！

私の名前はアンドレイです。私のブローカーチームの出身です。iOSでダークテーマのサポートを追加したことをお伝えします。

iOS 13のAppleはシステム全体に暗いテーマを追加しました。ユーザーはiOS設定で明るいまたは暗い外観を選択できます。ダークモードでは、システムはすべての画面、ビュー、メニュー、コントロールに暗いカラーパレットを使用します。



誰も気にしない-猫の下に行きます。

暗いデザインのサポート


Xcode 11でデフォルトで作成されたアプリケーションは、iOS 13のダークデザインをサポートします。ただし、ダークモードを完全に実装するには、追加の変更を行う必要があります。

  • 色は明暗のデザインをサポートする必要があります
  • 画像は明るいデザインと暗いデザインをサポートする必要があります

Appleは、明るいデザインと暗いデザインをサポートするいくつかのシステムカラーを追加しました。


iOS 13では、新しいUIColor初期化子が導入されました：

init (dynamicProvider: @escaping (UITraitCollection) -> UIColor)

静的な関数を追加して、明るいデザインと暗いデザインの切り替えをサポートする色を作成します。

extension UIColor {
    
    static func color(light: UIColor, dark: UIColor) -> UIColor {
        if #available(iOS 13, *) {
            return UIColor.init { traitCollection in
                return traitCollection.userInterfaceStyle == .dark ? dark : light
            }
        } else {
            return light
        }
    }
}

CGColorは、明暗の自動切り替えをサポートしていません。デザインを変更した後、CGColorを手動で変更する必要があります。

override func traitCollectionDidChange(_ previousTraitCollection: UITraitCollection?) {
    super.traitCollectionDidChange(previousTraitCollection)
        
    layer.borderColor = UIColor.Pallete.black.cgColor
}

リソースに暗い装飾の色を追加することもできます。

例として小さなアプリケーションを作ってみましょう。


アプリケーションには2つのウィンドウと3つの画面が含まれます。

最初のウィンドウ：認証画面。

2番目のウィンドウ：リボン画面とユーザープロファイル画面。

明るいデザインと暗いデザインのスクリーンショット



明るいテーマと暗いテーマを切り替える


トピックの列挙を作成します。 

enum Theme: Int, CaseIterable {
    case light = 0
    case dark
}

アプリケーションの再起動後に復元するために、現在のテーマを保存する機能を追加します。

extension Theme {
    
    //   UserDefaults
    @Persist(key: "app_theme", defaultValue: Theme.light.rawValue)
    private static var appTheme: Int
    
    //    UserDefaults
    func save() {
        Theme.appTheme = self.rawValue
    }
    
    //   
    static var current: Theme {
        Theme(rawValue: appTheme) ?? .light
    }
}

持続する
@propertyWrapper
struct Persist<T> {
    let key: String
    let defaultValue: T
    
    var wrappedValue: T {
        get { UserDefaults.standard.object(forKey: key) as? T ?? defaultValue }
        set { UserDefaults.standard.set(newValue, forKey: key) }
    }
    
    init(key: String, defaultValue: T) {
        self.key = key
        self.defaultValue = defaultValue
    }
}


デザインを強制するには、すべてのアプリケーションウィンドウのスタイルを変更する必要があります。 
アプリケーションにテーマ切り替えを実装します。 

extension Theme {
    
    @available(iOS 13.0, *)
    var userInterfaceStyle: UIUserInterfaceStyle {
        switch self {
        case .light: return .light
        case .dark: return .dark
        }
    }
    
    func setActive() {
        //   
        save()
        
        guard #available(iOS 13.0, *) else { return }
        
        //       
        UIApplication.shared.windows
            .forEach { $0.overrideUserInterfaceStyle = userInterfaceStyle }
    }
}

また、ウィンドウを表示する前に、ウィンドウスタイルを現在のテーマに変更する必要があります。 

extension UIWindow {
    
    //     
    //     
    func initTheme() {
        guard #available(iOS 13.0, *) else { return }
        
        overrideUserInterfaceStyle = Theme.current.userInterfaceStyle
    }
}

明るいまたは暗いテーマを選択するスクリーンショット


システムテーマにスイッチを追加する


enumテーマにシステムテーマを追加します。
enum Theme: Int, CaseIterable {
    case system = 0
    case light
    case dark
}

明るいまたは暗いテーマを強制インストールした後、システムに含まれているデザインを特定することはできません。システム設計を認識するために、アプリケーションにウィンドウを追加します。このウィンドウでは、設計の変更は強制されません。アプリケーションにシステムテーマがインストールされていて、ユーザーがiOSでデザインを変更した場合も、デザインの変更を実装する必要があります。

final class ThemeWindow: UIWindow {
    
    override public func traitCollectionDidChange(_ previousTraitCollection: UITraitCollection?) {

        //         iOS,     .
        // :      .
        if Theme.current == .system {
            Theme.system.setActive()
        }
    }
}

let themeWindow = ThemeWindow()

class AppDelegate: UIResponder, UIApplicationDelegate {

    func application(_ application: UIApplication, didFinishLaunchingWithOptions launchOptions: [UIApplication.LaunchOptionsKey: Any]?) -> Bool {
        ...
        //    ,    
        //       
        themeWindow.makeKey()
        ...
        return true
    }
}

extension Theme {
    
    @available(iOS 13.0, *)
    var userInterfaceStyle: UIUserInterfaceStyle {
        switch self {
        case .light: return .light
        case .dark: return .dark
        case .system: return themeWindow.traitCollection.userInterfaceStyle
        }
    }
    
    func setActive() {
        //   
        save()
        
        guard #available(iOS 13.0, *) else { return }
        
        //       
        //        
        UIApplication.shared.windows
            .filter { $0 != themeWindow } 
            .forEach { $0.overrideUserInterfaceStyle = userInterfaceStyle }
    }
}
