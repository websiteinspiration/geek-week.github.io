GraphQLバックエンドを使用して数分でReactアプリケーションを生成する

8base CLI（コマンドラインインターフェイス）に新しい実験的機能を追加しました。これは、データモデルから既製のReactアプリケーションを生成します。これは、GraphQLとReactを使い始めるとき、およびアプリケーションのユーザーインターフェイスをすばやく作成するときに非常に便利です。Ruby-on-Railsの足場のように見えます。

作成されたアプリケーションはCreate React Appに基づいており、GraphQLクエリにApolloクライアントを使用し、Auth0に基づく認証とファイルをダウンロードする機能が含まれています。

使い方


いくつかの簡単な手順を実行する必要があります。
  1. 8baseでデータモデルを定義します。

  2. 8base CLIをインストールし、 npmを使用して8base loginを実行します。
  3. 8base generate app $ {YOU_APP_NAME}コマンドを実行して、アプリケーションテンプレートを生成します。
  4. 8base generate scaffold $ {TABLE_NAME}を実行して、選択したデータテーブルのCRUD UIを生成します（追加するテーブルごとにこの手順を繰り返すことができます）。
  5. 最後に、yarn install && yarn startを実行してアプリケーションを開始します。
