如何为初学者在p5.js库中重新创建摩尔纹图案效果（快速指南）



波纹图案是在应用两个周期性网格图案时发生的光学错觉。两个图案的重复元素要么重叠要么形成间隙。

在平常的生活中，例如，当我们看着薄纱的窗帘时，我们经常会遇到它。毫不奇怪，法语名称Moire本身最初的意思是类似类型的丝绸。



从摄影到放射学和导航，通常可以在各个领域观察到这种效果。

为了达到这种效果，并同时在p5.js中进行实践，我决定编写简单的代码来重新创建它。

p5.j​​s是2014年创建的Javascript库。它的主要目标是成为设计师，艺术家，教师和其他创意专业代表的编程世界的一扇窗户。它同时支持动画和图表，可轻松转换为Web格式。

首先，您需要创建一个新的项目在线编辑器。默认情况下，新项目如下：

function setup() {
  createCanvas(400, 400);
}

function draw() {
  background(220);
}

函数setup（）-设置仅执行一次的
函数function draw（）-无限循环重复

我们从设置“画布”开始。对于createCanvas（）函数，我们设置参数720和500（分别为宽度和高度）。

为了使椭圆形出现在背景上，我们将以下参数插入draw（）函数（我们记得，它负责重复）：

{
  noFill()
  stroke ('purple') 
  strokeWeight(3)
  ellipse(360,250,100,100) }

这里发生了什么？

noFill（）-不会用任何颜色填充椭圆的内部。
描边（“紫色”）-创建一条紫色线。
strokeWeight（3）-设置这条线的粗细。
椭圆（360,250,100,100）-表示应将线扭曲为具有给定参数（沿X的位置，沿Y的位置，沿X的直径，沿Y的直径）的椭圆。第一个和第二个参数等于createCanvas（400，400）函数中参数的一半-正好在中间。相同的3和4参数创建圆形而不是椭圆形。

结果，您得到这样的东西。

现在，我们需要使这些椭圆更多，每一个都比前一个略大。为此，我们需要创建一个循环，将椭圆形移入其中，并使第三个和第四个参数可变。

 background(220);
  for(let i=0;i<1000; i+=10) {
  noFill()
  stroke ('purple') 
  strokeWeight(3)
  ellipse(360,250,i,i)

如果您一生中至少处理过一次周期，您将了解正在发生的事情。我们将初始值设置为i = 0，因此一开始我们会在零半径处绘制一个椭圆。然后将极限值设置为i，我们将重复该循环（最大1000），此后确定更改步长i + = 10-每次将i添加到先前的结果i。

将参数i替换为椭圆绘图函数中的第三和第四参数负责直径的宽度和高度。

这不是波纹，但已经很相似了。该代码做了什么？他只是简单地连续画了100个圆（直到后者的直径变成1000个），它们的直径相差10个，每次都保存结果。
不要忘记输入noFill（），否则您将保存并仅显示最后一次迭代的结果。

现在，要获得第二个椭圆，我们将其输入到同一循环中，但参数略有不同：

function draw() {
  background(220);
  for(let i=0;i<1000; i+=10) {
  noFill()
  stroke ('purple') 
  strokeWeight(3)
  ellipse(360,250,i,i)
    
  stroke("red")
  strokeWeight(3)
  ellipse(360, 250,i-500,i-500)
  }

结果，您将获得一系列的圆，其中最大的圆直径将等于1000减去500。我们的画布高度仅为500，因此将使内部完美主义者满意。好吧，它将是红色的。

现在是采取最后行动的时候了-让这些圈子相互超越。

为此，请在循环的OUT之外，在整个代码的开始处添加变量x，并将其值添加为0，然后在draw（）函数之后添加if循环

let x = 0
function setup() {
  createCanvas(720, 500);
}

function draw() {
  background(220);
  for(let i=0;i<1000; i+=10) {
  noFill()
  stroke ('purple') 
  strokeWeight(3)
  ellipse(360,250,i,i)
    
  stroke("red")
  strokeWeight(3)
  ellipse(x,250,i-500,i-500)
  }
if (x>width) {
  x= 0
} else {x= x+3}
}

有条件。如果x小于画布的宽度，则将x重置为默认的零值。在此之前，我们将x值加3。可以理解，通过X，我们希望设置第二个圆沿X轴的坐标。由于每次都会稍有变化，因此我们的圆圈将向右水平移动，直到其中心到达屏幕边缘为止。


不要忘记将第二个椭圆的第一个参数从360更改为变量x。

做完了 结果，您在计算机屏幕上会出现反复无休止的波纹现象，可以轻松地集成到您的Web项目中。
