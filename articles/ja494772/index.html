米国でロボット呼び出しとの戦いがどのように進展しているか-政治家と電気通信会社の対策について

ロシアでは電話詐欺師の数が増加しています。2019年に、FinCERT中央銀行は侵入者に属する5000の番号をブロックしました。図はかなり控えめですが、1年前の39倍です。そして今日、危機の間に、この種のリスクは増加しているだけです。したがって、私たち全員がもっと注意する必要があります。規制当局が増大する問題に注意を払うまで、状況は次第に緊迫します。

このシナリオは、ロボット呼び出しとの戦争がある米国で現在展開されています。私たちは、政治家、通信会社、IT企業がどのような闘いの方法を提供しているかを確認することにしました。


/アンスプラッシュ/ クリス・フラー

状況はどうなっていますか？


YouMailによると、2018年には米国で約450 億のロボット呼び出しが登録されました。この数字は2019年に22％増加し、値は585億に達しました。現在、米国連邦取引委員会（FTC）に対する消費者からの苦情の最も一般的な原因は、迷惑電話です。

ダイヤラのかなりの部分の後ろには詐欺師があります。昨年、FTCは、彼らからお金を強要した住民からの40万の申請を登録しました。被害総額は1億5300万ドルと推定された。

しかし、すべてがそのような犯罪を報告しているわけではないので、実際の数字ははるかに大きいかもしれません。米連邦通信委員会のネイサンLeamer（FCC）は考慮しますそのロボット呼び出しは経済に深刻な脅威をもたらします。彼によると、詐欺師のために、アメリカの消費者は毎年最大30億ドルを失っています。

政府の取り組み


問題の規模を考慮して、政治家はロボット呼び出しに取り組むことにしました。今年の初めに、追跡法である電話ロボコール虐待刑事執行および抑止法が米国で施行されました。彼は違法な電話の主催者からの罰金の徴収に関連する問題でFCC の権限を拡大します。すでに最初の結果が出ています-過去3か月間で、法務省は高齢者から金銭を強要した400人を起訴しました。これは昨年の2倍です。国の司法長官、ウィリアム・バーによれば、攻撃者の4分の1はすでに有罪判決を受けています。

Habréに関するブログからの新しい投稿：

  • 米国のロボコール戦争-勝者とその理由
  • 米国におけるネットワーク中立性のための戦いの主なものは、イベントの年表と現在の状況です
  • E2E暗号化の禁止につながる可能性のある新しい米国の法案であるEARN IT法について議論します

この問題は、連邦取引委員会でも取り扱われています。3月の終わりに、その代表は主要なVoIPプロバイダーに手紙を送り、違法な自動通話サービスの作業をブロックする必要があることを思い出させました。それ以外の場合、FTCは法的請求を準備します。過去に、規制当局は NetDotSolutionsの所有者であるJames Christiano を訴え、10億件のロボット呼び出しの組織化に関与しました。

企業が行うこと


また、通信会社はインフラストラクチャを更新して、顧客への不要な通話に対抗しています。たとえば、新しいSHAKEN / STIR標準を導入しています。これは、発信者を識別するデジタル証明書の操作に基づいています。先週、T-Mobile は Bandwidth Inc.とパートナーシップを結び、より多くの顧客に新しいサービスを提供しました。 SHAKEN / STIRは、Sprint、Comcast、AT＆T でも実装されています。 USTelecom は、 SHAKEN / STIRは人々にもっとコントロールを与えるだろうと言っています-彼らが電話を取るかどうかを決めるのは簡単です。


/ Unsplash / Brunno Tozzo

電話会社も結論すべての州の検察官との協力協定。彼らは、自動ダイヤラーの事実に関する情報を警察当局に迅速に送信することを約束しました。大規模なIT企業もロボット呼び出しと戦っています。1か月前、Google はすべてのPixelスマートフォンに[画面を呼び出す]オプションを追加したことを発表しました。不要な通話を自動的にフィルタリングします。以前は、この機能は企業の主力デバイスでのみ利用可能でした。同時に、同社はすべてのモバイルアプリケーション開発者がロックを使用できるように、Android 11に適切なAPIを追加することを約束しました。
