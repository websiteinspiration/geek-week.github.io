13の有用なJavaScriptシングルライナー

本日の翻訳を公開している記事の著者は、長年JavaScriptでプログラミングを行っていると述べています。この間、彼は優れた単一行ファイルのコレクションを収集しました。1行に収まるので、コードフラグメントは驚くほど強力です。彼によれば、大きな機会は大きな責任でもあるので、これらの構造を注意深く使用して、プログラムの可読性を損なわないようにする必要があります。 こちらが13人のシングルライン選手です。Node.js v11.xを使用して準備された例。別の環境で使用すると、実行に影響する可能性があります。





1.論理型への値の強制


値をブール型にキャストする方法は次のとおりです。

const myBoolean = !!myVariable;

!!JavaScriptルールに関してtrueである値がに変換されtrue、false-に変換されるように、 二重否定（）が必要falseです。

2.配列内の重複する値を取り除く


配列から重複する値を削除する方法は次のとおりです。

const deDupe = [...new Set(myArray)];

タイプデータ構造Setは、一意の値のみを格納します。その結果、このようなデータ構造とスプレッド構文を使用すると、myArray配列に基づいて、重複する値のない新しい配列を作成できます。

3.条件によるオブジェクトのプロパティの作成と設定


演算子を使用してオブジェクトのプロパティを設定&&するには、spread構文を使用できます。

const myObject = { ...myProperty && { propName: myProperty } };

式の左側を計算した結果、JSが誤った値として認識したものを受け取った場合、それ&&以上の計算は実行されず、新しいプロパティは作成および設定されません。オブジェクトmyObjectは空になります。...myPropertyJSがtrueと認識する結果がコンストラクションから返された場合、コンストラクションにより、受信した値を格納&&するプロパティがオブジェクトに表示されpropNameます。

4.オブジェクトのマージ


他の2つのオブジェクトを組み合わせる新しいオブジェクトを作成する方法は次のとおりです。

const mergedObject = { ...objectOne, ...objectTwo };

このアプローチは、無制限の数のオブジェクトのマージを編成するために使用できます。さらに、オブジェクトに同じ名前のプロパティがある場合、他の右側にある元のオブジェクトのプロパティに属する最終オブジェクトには、そのようなプロパティが1つだけ残ります。ここでは、オブジェクトプロパティの小さなコピーが使用されていることに注意してください。

5.変数値の交換


補助変数を使用せずに2つの変数間で値を交換するには、次のようにします。

[varA, varB] = [varB, varA];

その後、中にあったものはにvarA該当しvarB、その逆も同様です。これは、破壊の内部メカニズムを使用することで可能になります。

6.配列から誤った値を削除する


JavaScriptでfalseと見なされるすべての値を配列から削除する方法は次のとおりです。

const clean = dirty.filter(Boolean);

この操作中に、それは、次のような配列値から削除されnull、undefined、false、0、および空白行として。

7.数値を文字列に変換する


配列に格納されている数値を文字列表現に変換するには、次のようにします。

const stringArray = numberArray.map(String);

そのような変換中の配列の文字列要素は文字列のままです。

タイプString値をタイプ値に変換することにより、逆変換を実行することもできますNumber：

const numberArray = stringArray.map(Number);

8.オブジェクトのプロパティ値を取得する


オブジェクトのプロパティの値を抽出して、このプロパティの名前とは異なる名前の定数に書き込む方法を次に示します。

const { original: newName } = myObject;

この構造の使用のおかげnewNameで、originalオブジェクトのプロパティの値が書き込まれる新しい定数が作成されmyObjectます。

9. JSONコードのフォーマット


JSONコードを読みやすい外観に変換する方法は次のとおりです。

const formatted = JSON.stringify(myObj, null, 2);

このメソッドstringifyは3つのパラメーターを取ります。1つ目はJavaScriptオブジェクトです。2番目のオプションは、オブジェクトの変換から得られるJSONコードを処理するために使用できる関数です。最後のパラメーターは、JSONコードでインデントするときに使用する必要があるスペースの数を示します。最後のパラメーターを省略すると、受信したすべてのJSONコードは1つの長い文字列になります。オブジェクトmyObjに循環参照がある場合、JSON形式に変換できません。

10.数値配列をすばやく作成する


配列を作成して数値で埋める方法は次のとおりです。

const numArray = Array.from(new Array(52), (x, i) => i);

このような配列の最初の要素のインデックスは0です。配列のサイズは、数値リテラルまたは変数を使用して指定できます。ここでは、52要素の配列を作成します。これは、たとえば、カードのデッキに関するデータを格納するために使用できます。

11. 2要素認証のコードの作成


2要素認証メカニズムまたは他の同様のメカニズムで使用される6桁のコードを生成するには、次のようにします。

const code = Math.floor(Math.random() * 1000000).toString().padStart(6, "0");

返される結果が返される数値のゼロの数は、メソッドのMath.random()最初のパラメーター（targetLength）に対応している必要があることに注意してくださいpadStart。

12.アレイの混合


正確に何が含まれているかを知らずに配列をシャッフルするには、次のようにします。

myArray.sort(() => { return Math.random() - 0.5});

配列を混合するためのより良いアルゴリズムがあります。たとえば、Fisher-Yatesシャッフルアルゴリズム。アレイを混合するためのさまざまなアルゴリズムについては、こちらをお読みください。

13.オブジェクトのディープコピーの作成


ここで提案されているオブジェクトのディープコピーの方法は、特に高いパフォーマンスで違いはありません。ただし、ワンライナーを使用してこの問題を解決する必要がある場合は、次のコードを使用できます。

const myClone = JSON.parse(JSON.stringify(originalObject));

originalObject循環リンクが ある場合は、そのコピーを作成できないことに注意してください。この手法は、自分で作成した単純なオブジェクトで使用することをお勧めします。

オブジェクトの浅いコピーは、spread構文を使用して作成できます。

const myClone = { ...orignalObject };

概要：単一行のコードの結合と拡張について


ここに示すコードスニペットを組み合わせる方法はたくさんあります。これにより、多くの問題を解決すると同時に、非常にコンパクトな言語構造を使用できます。この資料の作成者は、JavaScriptの進化の間に、コンパクトで強力なコードの作成に貢献する新しい機能がその中に現れると信じています。
