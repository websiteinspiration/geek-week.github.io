文字列で動作するように設計されたいくつかのJavaScriptライブラリについて話しましょう。

1. String.js


ライブラリーstring.js、または単にS、それは、ブラウザーおよびNode.jsで使用できる小さな（5 KB未満に圧縮された）JavaScriptライブラリーです。これにより、プログラマは文字列を操作するための多数のメソッドセットを利用できます。これらはオブジェクトメソッドstring.jsであり、便宜上、標準の文字列メソッドが含まれます。オブジェクトstring.jsは、通常の文字列のラッパーの一種です。

▍インストール


ライブラリをインストールするにはstring.js、次のコマンドを使用します。

npm i string

▍注目すべき方法


メソッド間


このメソッドbetween(left, right)は、文字列から、行leftとの間に含まれる部分文字列を抽出しますright。たとえば、このメソッドを使用して、2つのHTMLタグの間の要素を抽出できます。

var S = require('string');
S('<a>This is a link</a>').between('<a>', '</a>').s 
// 'This is a link'

ラクダ化法


このメソッドcamelize()は、処理された文字列からスペース、アンダースコア、ダッシュを削除し、この文字列を「ラクダ」スタイルに変換します。この方法は、資料の冒頭で述べた問題を解決するために使用できます。

var S = require('string');
S('---Foo---bAr---').camelize().s; 
//'fooBar'

ヒト化法


このメソッドhumanize()は、処理された文字列を読み取り可能な形式にします。このようなものを最初から実装する必要がある場合は、かなりの数行のコードを記述する必要があります。

var S = require('string');
S('   capitalize dash-CamelCase_underscore trim  ').humanize().s
//'Capitalize dash camel case underscore trim'

StripPunctuationメソッド


このメソッドstripPunctuation()は、句読点と、アスタリスクや角括弧などのさまざまな特殊文字を文字列から削除します。このような関数を自分で作成した場合、処理する必要のある文字を忘れるリスクが高くなります。

var S = require('string');
S('My, st[ring] *full* of %punct)').stripPunctuation().s; 
//My string full of punct

2.ボーカ


JavaScriptライブラリVocaは、文字列の使いやすさを向上させる貴重な機能を提供します。その機能には次のものがあります。

  • 文字の大文字と小文字を変更します。
  • タイムラインの最初と最後にある指定された文字を削除します。
  • 文字列を指定された長さに追加します。
  • 文字列を文字列に変換します。文字列はハイフンで区切られます。
  • 文字列をラテン文字で書く。
  • パターンに応じた線の組み立て。
  • 文字列を指定された長さにトリミングする
  • HTMLの特殊文字のエスケープ。

ライブラリのモジュール設計により、必要に応じて、ライブラリ全体ではなく個々の関数をロードできます。これにより、このライブラリを使用してアプリケーションのバンドルを最適化できます。ライブラリコードは完全にテストされ、十分に文書化され、適切に保守されています。

▍インストール


インストールするにVocaは、次のコマンドを実行します。

npm i voca

▍注目すべき方法


CamelCaseメソッド


このメソッドcamelCase()は、渡された文字列を「ラクダ」スタイルで記述された文字列に変換します。

var v = require('voca');
v.camelCase('foo Bar');
// => 'fooBar'

v.camelCase('FooBar');
// => 'fooBar'

v.camelCase('---Foo---bAr---');
// => 'fooBar'

ラテン語化メソッド


このメソッドlatinize()は、渡された文字列をラテン文字で書かれた文字列に変換した結果を返します。

var v = require('voca');
v.latinise('cafe\u0301'); // or 'café'
// => 'cafe'

v.latinise('août décembre');
// => 'aout decembre'

v.latinise('   ');
// => 'kak prekrasen etot mir'

IsAlphaDigitメソッド


渡された文字列に英数字のみが含まれている場合、 メソッドisAlphaDigit()は戻りますtrue。

var v = require('voca');
v.isAlphaDigit('year2020');
// => true

v.isAlphaDigit('1448');
// => true

v.isAlphaDigit('40-20');
// => false

CountWordsメソッド


このメソッドcountWords()は、渡された文字列の単語数を返します。

var v = require('voca');
v.countWords('gravity can cross dimensions');
// => 4

v.countWords('GravityCanCrossDimensions');
// => 4

v.countWords('Gravity - can cross dimensions!');
// => 4

EscapeRegExpメソッド


このメソッドescapeRegExp()は、渡された文字列に基づいて、特殊文字がエスケープされた文字列を返します。

var v = require('voca');
v.escapeRegExp('(hours)[minutes]{seconds}');
// => '\(hours\)\[minutes\]\{seconds\}'

3. Anchorme.js


これはコンパクトで高速なJavaScriptライブラリで、テキスト内のURLと電子メールアドレスを見つけて、機能するハイパーリンクに変換します。主な特徴は次のとおりです。

  • 高感度、偽陽性率が低い。
  • IANA完全リストからのリンクと電子メールアドレスの検証。
  • ポート番号の検証（存在する場合）。
  • IPアドレスの検証（利用可能な場合）。
  • ラテン文字以外の文字で記述されたURLのサポート。

▍インストール


インストールするにanchorme.jsは、次のコマンドを実行します。

npm i anchorme

▍使用


import anchorme from "anchorme"; 
// 
//var anchorme = require("anchorme").default;
const input = "some text with a link.com"; 
const resultA = anchorme(input);
//some text with a <a href="http://link.com">link.com</a>
