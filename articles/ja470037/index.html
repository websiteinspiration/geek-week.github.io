F＃3：テキストのフォーマット

どの言語を使用する場合でも、テキストをフォーマットする必要がある可能性が高く、F＃も同じです。

F＃は.NET言語であるため、常にConsole.WriteLine（..）とString.Format（..）を使用できます。ここでは、通常の.NETコードで使用した通常のフォーマッタを使用できます。

ただし、F＃は、より構文に似たCメソッドもサポートしています。これは、Core.Printfモジュールで使用できます。その内部の1つの関数は、Console.WriteLine（..）.NETクラスを使用する代わりに使用できます。

標準出力への書き込みに使用する必要がある同等のF＃関数はprintfnです。これは、F＃でテキストの書式設定を行うときに実際に推奨される方法です。

printfnがConsole.WriteLine（..）よりも望ましいのはなぜですか？


printfnが推奨される理由はいくつかありますが、主な理由のいくつかを以下に示します。

  • 静的型チェック。これは、文字列が期待される場所にint値を渡すと、これについて警告されることを意味します
  • printfnはネイティブF＃関数であるため、他のF＃関数と同様に機能します。

次のコードスニペットを検討してください

//  
printfn "should be a decimal value %d" "cat"
//
printfn "should be a decimal value %d" 42
//  ,  
Console.WriteLine("should be a decimal value {0}" , "cat")
 
 
//    
printfn "this only expected 1 arg of type decimal %d" 42 "dog"
//    ,  
Console.WriteLine("this only expected 1 arg of type decimal {0}" , 42, "dog")
// ,  
printfn "this only expected 2 args of type decimal %d and string %s" 42 "dog"

Visual Studio IDEでこのコードのスクリーンショットを見ると、失敗すると予想された行がエラーとして明確に示されていることがわかります。



printfnの使い方


おそらく、値の操作に使用できる標準フォーマッターのいくつかを調べることから始めるのが最善でしょう。次のMSDNの表は、使用可能な標準の演算子を示しています。msdn.microsoft.com / en-us / library / ee370560.aspx

では、標準以外の優れたフォーマットを作成することはどうでしょうか？


それで、フォーマットされた出力を標準出力に書き込む方法を見ました（printfnがこれを行いました）が、フォーマットされた行を他の場所で使用したい場合はどうでしょうか？文字列値を、上記の書式設定ツールを使用して作成した、またはStringBuilderまたはTextWriterに書き込みたい、美しく書式設定された文字列に関連付ける場合、F＃はこれを簡単にしますか？

そうですね、実際にそうです。Core.Printfモジュールには他にも多くの便利な機能があります。この執筆時点での完全なリストは次のとおりです。

リスト
bprintf StringBuilder.
eprintf stderr.
eprintfn stderr, .
failwithf . .

私はこれをすべて行うつもりはありませんが、最も一般的なものを検討します：

bprintf


StringBuilderに出力します。

let builder = new StringBuilder(524288)
Printf.bprintf builder "This will be a string line : %s\r\n" "cat"
Printf.bprintf builder "This will be a bool line : %b\r\n" true
Printf.bprintf builder "This will be a int line : %u\r\n" 42
Printf.bprintf builder "This will be a hex line : %X\r\n" 255
 
printfn "%s" (builder.ToString())

これは、F＃コンソールアプリケーションでこの出力を提供します

fprintf


テキストライターで印刷します。

use sw = new StreamWriter(@"c:\temp\fprintfFile.txt")
 
fprintf sw "This is a string line %s\r\n" "cat"
fprintf sw "This is a int line %i" 10
 
sw.Close()

これにより、ファイルが一時的に作成されます



スプリント


let result = Printf.sprintf "This will return a formatted string : %s\r\n" "cat"
printfn "%s" result
