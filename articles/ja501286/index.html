Easygui pythonの基本。パート1

この記事は、Pythonのeasyguiモジュールの基本についてです。タブとスペースを組み合わせることができるため、Python 2-xブランチの使用をお勧めします。もちろん、このモジュールはプログラムを作成するためのものではなく、追加機能です。記事はほとんどの初心者向けに書かれているため、多くのコメントがあります。

-1。easyguiをインポートする

import easygui
from easygui import * # .

1. inputboxへのデータの入力と変数へのデータの書き込み

注意enterboxは変数に 1文字だけを書き込みます！これで入力を行うことができます。

n = input() # .

easyguiを使用して、入力を行って変数に書き込むこともできます。このためのエンターボックス機能があります。次に、vvod関数を作成し、すぐに変数を作成します。

def vvod():
    global var1 #    .

次に、フィールドを作成するためのリストを作成する必要があります（複数の入力フィールドを作成することもできます）。

以下をvvod関数に追加します。

2.はいまたはいいえ。

Easyguiにはynbox関数（YesNoBox）があります。この機能により、ユーザーはさらにアクションを選択できます。たとえば、「続行しますか？」提案されたものからオプションを選択します。標準の「はい」と「いいえ」ですが、他のボタンの機能は自分で行う必要があります（パラメーターでは、これらの「default_choice」と「cancel_choice」のみ）。それらをキーにバインドして、いくつかの選択を行うことができます。

変数を設定しましょう：

msg = "  ?"
title = "!"
choices = ("[<F1>]", "[<F2>]") #

さて、ここに「はい」の意味と「いいえ」の意味があります、私たちはそれを自分で書きます。

ynbox(msg, title, choices, image=None, default_choice="[<F1>]", cancel_choice="[<F2>]") #  "",   "".

次のウィンドウが表示されます。


変数への書き込みは、inputboxに似ています。

def login_procedure():

fieldValues = multpasswordbox(msg, title, fieldNames) #
