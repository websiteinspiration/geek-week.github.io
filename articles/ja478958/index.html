更新ルールには何が書かれていますか？


更新ルールの意味は、更新プロセスを予測可能にし、ユーザーに警告を出す手順を定義して、ユーザーがそれに応じて作業を計画し、予期しないダウンタイムを回避できるようにすることです。ルールには、更新の失敗からのロールバックなど、予期しない問題を処理するためのプロトコルも含まれています。

合理的な更新ルールでは、毎月更新を処理するために一定の時間がかかります。小規模な組織では、各PCのサービススケジュールの特別なウィンドウがこの目的に役立ちます。大規模な組織では、ユニバーサルソリューションが機能する可能性は低く、PC全体を更新グループ（Microsoftでは「リング」と呼ばれます）に分割する必要があり、それぞれに独自の更新戦略があります。

ルールは、いくつかの異なるタイプの更新を記述する必要があります。最もわかりやすいタイプは、毎月第2火曜日に表示される毎月の累積的なセキュリティと信頼性の更新です（「パッチの火曜日」）。このリリースには通常、Windows悪意のあるソフトウェアの削除ツールが含まれており、次の種類の更新プログラムが存在する可能性があります。

  • .NET Frameworkのセキュリティ更新
  • Adobe Flash Playerのセキュリティ更新
  • サービススタックの更新（最初からインストールされます）。

これらの更新プログラムのインストールは、最大30日間延期できます。

PCの製造元によっては、ハードウェアおよびファームウェアのドライバーをWindows Updateチャネルを通じて配布することもできます。これを拒否するか、他の更新と同じスキームに従って管理できます。

最後に、機能の更新はWindows Updateを通じて配布されます。これらの主要なパッケージは、Windows 10を最新バージョンにアップグレードし、長期サービスチャネル（LTSC）を除くすべてのエディションのWindows 10で6か月ごとにリリースされます。 Windows Update for Businessを使用してコンポーネントの更新のインストールを最大365日間延期できます。 EnterpriseおよびEducationエディションでは、インストールが最大30か月遅延する場合があります。

大規模組織で働くためのツール（エンタープライズ）


Windowsネットワークインフラストラクチャを備えた大規模な組織は、Microsoft更新サーバーをバイパスし、ローカルサーバーから更新を展開できます。これには、企業のIT部門の注意を高める必要がありますが、会社に柔軟性を追加します。最も一般的な2つのオプションは、Windows Server Update Services（WSUS）とSystem Center Configuration Manager（SCCM）です。

WSUSサーバーの方が単純です。これはWindowsサーバーとして機能し、組織内のWindows更新の集中ストレージを提供します。管理者はグループポリシーを使用して、組織全体のファイルの唯一のソースとして機能するWSUSサーバーにWindows 10 PCを誘導します。管理者コンソールから、アップデートを承認し、別のPCまたはPCのグループにインストールするタイミングを選択できます。 PCを手動で別のグループにバインドするか、クライアント側のターゲティングを使用して、既存のActive Directoryセキュリティグループに基づいて更新を展開できます。

Windows 10の累積的な更新は、新しいリリースごとにますます大きくなるので、通信チャネルの帯域幅のかなりの部分を占める可能性があります。 WSUSサーバーは高速インストールファイルを使用してトラフィックを節約します。これにより、北部でより多くの空き領域が必要になりますが、クライアントPCに送信される更新ファイルのサイズが大幅に減少します。

WSUS 4.0以降を実行しているサーバーでは、Windows 10コンポーネントの更新を管理することもできます。

2つ目のオプションであるSystem Center Configuration Managerは、Windowsの豊富なConfiguration Manager機能をWSUSと組み合わせて使用​​して、品質とコンポーネントの更新を展開します。コントロールパネルを使用すると、ネットワーク管理者は、ネットワーク全体でのWindows 10の使用を監視し、サポートサイクルの終了間際のすべてのPCに関する情報を含むグループベースのサービスプランを作成できます。

組織に以前のバージョンのWindowsで動作するように既にConfiguration Managerがインストールされている場合、Windows 10サポートをそれに追加するのは非常に簡単です。
