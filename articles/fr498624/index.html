[Instruction] Création de tests Google (formulaires Google)

Merci aux utilisateurs Exosphère et Yuriy_krd pour de l'aide et des critiques constructives

Parfois, il semble que créer quelque chose selon le concepteur soit très rapide et simple: au moins un site, au moins une enquête, au moins une table. Cependant, vous perdez souvent de vue certaines petites choses offensantes, à cause desquelles du temps est perdu et vous devez refaire le travail à cause d'une bagatelle offensive. D'accord, c'est désagréable. J'ai décidé de passer par tous les processus pour vous et d'avertir d'éventuelles bagatelles désagréables, ainsi que d'aider les nouveaux arrivants.

Dans cet article, nous allons créer un formulaire Google avec lequel vous pouvez collecter des données, interroger des employés, des clients, effectuer des tests d'hypothèse, etc. Soit dit en passant, des informations extrêmement pertinentes en termes de travail à distance.



Pour créer des tests, vous devez avoir un compte Google actif. Si vous n'en avez pas, reportez-vous aux instructions "Création d'un compte Google et utilisation de Google disk" .

Alors, créez un formulaire (test).

Tâche entrante


Nous devons créer un test qui collectera un ensemble d'informations.

  1. Nom et prénom de la personne testée.
  2. Adresse e-mail - elle est nécessaire lors de l'interrogation de destinataires connus, lorsque le lien leur a été envoyé non pas par e-mail, mais dans le messager. Cet article ne peut pas être utilisé si vous envoyez un questionnaire en envoyant un e-mail.
  3. Faites une liste déroulante dans laquelle l'école sera sélectionnée. De même, vous pouvez faire un choix de classe, département, unité. Cette option est nécessaire pour que le candidat détermine correctement à qui il réussit le test. Lors de la saisie manuelle de données dans un champ, différents registres ou écrit des réponses seront structurés différemment par des filtres dans le tableau final.
  4. Vous devez créer une question simple avec une option pour la bonne réponse.

Accédez au lecteur Google. Pour ce faire, cliquez sur le bouton avec des points dans le panneau en haut à droite, en suivant le lien https://www.google.com/ ou https://www.google.ru/ Ensuite, sélectionnez Google Drive (ci-après dénommé Drive).



Après avoir chargé Drive, il est préférable de créer un dossier ou une structure de dossiers. Disons «Test» -> «Tests pour les demandeurs d'emploi» et après avoir accédé à ce dossier, créez un test (formulaire). Cela vous permettra de structurer les données.

Étape 2. Création d'un test (écran)


Comme pour tout document sur Google Drive, le début du travail commence par le bouton "Créer +" dans la partie supérieure gauche de l'écran. Ce bouton crée des dossiers dans Drive, tous les documents sont créés.



En cliquant sur le bouton Créer, vous verrez un menu. Dans ce document, vous devez sélectionner "Plus".



Il existe plusieurs façons de créer un formulaire:

1. Créez un formulaire vide :

  • Cliquez sur l'élément de menu Google Forms (marque 1)
  • Cliquez sur le bouton de la liste déroulante (repère 2), dans la liste déroulante, cliquez sur «Créer un formulaire vierge» (repère 3)

2. Création d'un formulaire selon le modèle - Répétez les étapes ci-dessus pour marquer 3 et au lieu de cela marquer 3+. Une bibliothèque de modèles s'ouvrira pour vous, où vous pourrez choisir le modèle le plus approprié. À l'avenir, il pourra être modifié selon vos besoins.

Nous avons créé un formulaire vide (testing).

Étape 3. Contrôles


La première chose à faire est de créer un nom de fichier pour le formulaire. Ce nom est interne, le formulaire sera stocké sur Drive.



Le nom est créé en cliquant sur la phrase «Nouveau formulaire» dans la partie supérieure gauche de l'écran.

Au centre de l'écran se trouve une boîte avec le nom du test (formulaire). Il sera vu par les candidats au moment de l'exécution. Remplacez-le par celui souhaité.



Au même endroit, nous remplissons la description du test (formulaire).

Ensuite, le premier bloc de la question est automatiquement créé.



Dans le champ «Question sans titre», vous devez saisir votre question. Renommez l'option 1 à la réponse ci-dessous. Si vous devez ajouter une autre question - vous devez cliquer ci-dessous «Ajouter une question». Vous pouvez également ajouter l'option «Autre», puis un champ de texte apparaîtra dans le test où l'utilisateur peut entrer toute autre réponse sous n'importe quelle forme.

La liste déroulante vous permet de sélectionner le type de question. Dans notre cas, une question avec une option de réponse est automatiquement créée. Si vous ouvrez la liste déroulante, vous pouvez choisir une autre forme de question.

Au bas du bloc, il y a des boutons:

  1. «Copier» - signifie copier tout le bloc.
  2. "Supprimer" - supprimer le bloc entier
  3. Problème obligatoire et interrupteur à glissière. Il rend la question obligatoire: sans y répondre, l'utilisateur ne pourra pas continuer ou terminer le test.
  4. Paramètres supplémentaires «bouton à trois points» - vous permet d'ajouter une description à la réponse, de mélanger les réponses, de sélectionner l'option pour la réponse.

À droite du formulaire se trouve un bloc vertical de boutons qui permet (de haut en bas sur les boutons):

  1. créer un autre bloc de questions
  2. importer des questions à partir d'un formulaire créé précédemment (tests)
  3. ajouter un bloc séparé avec un nom et une description
  4. Insérer une image
  5. insérer une vidéo (YouTube)
  6. ajouter une section. Dans ce cas, une autre page de questions sera créée. Ceci est pratique lors de la création de plusieurs sujets dans un seul formulaire (test).

Et le dernier bloc de contrôle en haut de la page:



De gauche à droite:

  1. Sélectionnez un thème de couleur pour le questionnaire. Ceci est très pratique lors de la création de plusieurs questionnaires réguliers - afin d'éliminer la confusion parmi les employés, vous pouvez également mentionner sa forme de couleur lorsque vous mentionnez le nom du rapport. Cela facilite grandement la communication et l'efficacité du travail.
  2. Vue. Permet dans un nouvel onglet de visualiser la version finale du formulaire pour le moment sans la publier.

L'onglet Général et la Présentation ne doivent pas poser de questions. Un onglet important «Tests» - vous pouvez y mettre une note pour chaque question du formulaire. Pour ce faire, activez le curseur "Test" et votre formulaire passera dans la catégorie test. Après avoir activé ce bouton, vous pouvez attribuer les bonnes réponses et le nombre de points pour la bonne réponse dans la liste des questions.

Après avoir activé le commutateur, cliquez sur le bouton «Enregistrer» et passez aux questions. Le bouton «Réponses» apparaîtra en bas du bloc de questions:



En cliquant dessus, vous entrerez dans le mode de définition des bonnes réponses et de notation des points pour la bonne réponse.



Commençons donc la tâche au début.

Étape 4. Créez un test


  1. Nous entrons le nom du fichier dans Drive
  2. Entrez le titre du test
  3. Entrez la description du test
  4. Entrez la question
  5. Après avoir ouvert la liste déroulante, sélectionnez l'élément «Texte (ligne)»

Nous obtenons:



Activez le commutateur «Question requise». L'utilisateur doit maintenant saisir le nom et le prénom pour continuer le test.

Ajoutez une deuxième question en cliquant sur le bouton du haut dans le panneau vertical droit des commandes.



Nous rendrons cette question facultative sans activer l'interrupteur en bas à droite du bloc. Sans même répondre à cette question, l'utilisateur continuera et pourra terminer les tests.

Nous créons un autre bloc avec une seule



option de réponse .



Nous ajoutons un bloc avec plusieurs options de réponse. Ici, nous avons également ajouté l'option de réponse «Autre».

Définissez maintenant les points pour les bonnes réponses. Accédez à l'élément de paramètres dans la partie supérieure droite des commandes (icône d'engrenage). Accédez à l'onglet «Test» et activez le commutateur. Cliquez sur le bouton Enregistrer et revenez à la liste de nos questions.

En cliquant sur un espace vide dans le bloc de questions, nous l'activons. En bas à gauche, nous activons le bloc de réponse. Nous choisissons la bonne réponse pour nous et attribuons le nombre de points.



Nous cliquons sur le bouton est prêt. Points attribués.

Nous devons maintenant ajuster la fréquence des questions et la collecte des adresses e-mail.

Encore une fois, allez dans les "Paramètres" en haut des commandes. Nous activons les éléments de configuration dont nous avons besoin.



Nous allons directement à l'onglet «Présentations» et vérifions que l'option «Afficher le lien pour remplir le formulaire» est activée. Cliquez sur le bouton «Enregistrer»

Le test (formulaire) est prêt.

Étape 4. Soumettre le test (formulaire)


La soumission du test (formulaire) s'effectue en cliquant sur le bouton «Soumettre». Dans la fenêtre qui s'ouvre, vous pouvez sélectionner différentes options d'envoi.

Envoi par e-mail:



dans le champ À, saisissez les adresses des destinataires. Dans ce cas, dans l'onglet réponses, vous verrez qui a voté et qui n'a pas encore répondu aux questions du formulaire.



De plus, le formulaire peut être immédiatement inclus dans le corps de la lettre.

L'onglet suivant consiste à envoyer le lien. Dans ce cas, vous pouvez l'envoyer non seulement par courrier, mais également de toute autre manière, y compris via Messenger. Cependant, dans ce cas, vous serez obligé de suivre vous-même les répondants.

La troisième option est un encart sur le site. C'est pour les utilisateurs plus avancés. Vous permet d'incorporer du code avec le formulaire directement dans le modèle de site.

Vous avez terminé de créer le formulaire.
