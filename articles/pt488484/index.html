Lançamento do SQL - evento Microsoft SQL Server 2019



Os melhores especialistas da Microsoft falarão sobre as principais inovações do SQL Server 2019: tecnologias do SQL Server Big Data Clusters para trabalhar com big data e aprendizado de máquina, tecnologias Polybase para acessar dados em fontes externas sem copiá-las, suporte a contêineres, trabalhar no OS Linux e muitas outras Novo no MS SQL Server 2019!

Um relatório separado será dedicado à plataforma de business intelligence Microsoft Power BI!

Convidamos você a visitar a apresentação da nova versão da plataforma híbrida de gerenciamento de dados SQL Server 2019! Registre-se , mas os detalhes estão sob o corte.

Você sabia que lançamos uma nova versão do SQL Server - SQL Server 2019? Graças a muitos novos recursos úteis, esta versão causou interesse sem precedentes, e a velocidade de tradução dos sistemas de informação pelas organizações excede o desempenho de todas as versões anteriores. Com a transição para o SQL Server 2019, as empresas estão gradualmente abandonando suas instalações caras do Hadoop, dominando os clusters de big data. As formas de acessar dados também estão mudando: os modernos recursos de visualização de dados permitem que você crie novos projetos sem exigir movimentação de dados dispendiosa e cópia preliminar de dados. As cargas de trabalho existentes são aceleradas por uma série de novas funções inteligentes de banco de dados.

Pode não ser fácil para você acompanhar todas as tendências emergentes. É por isso que temos o prazer de convidá-lo a visitar nosso evento em 3 de março às 09:30 na Spartakovsky Lane 21, entrada 7, para entender os motivos do interesse em tantas empresas no SQL Server 2019. Os palestrantes da Microsoft de diferentes países e nossos parceiros compartilharão suas impressões. Temos certeza de que você determinará por si mesmo todas as vantagens potenciais desta nova plataforma. Esperamos encontrar você e discutir as possibilidades de redução de custos, aceleração do trabalho em projetos e interação confortável com os usuários.
