初心者のためにp5.jsライブラリでモアレパターン効果を再現する方法（クイックガイド）



モアレパターンは、2つの周期的なメッシュパターンが適用されたときに発生する目の錯覚です。 2つのパターンの繰り返し要素は、重複するか、ギャップを形成します。

普通の生活では、チュールのカーテンを見るときなどによく遭遇します。当然のことながら、フランスの名前モアレ自体はもともと同様の種類のシルク生地を意味していました。



この影響は、写真から放射線医学、ナビゲーションまで、さまざまな分野でよく見られます。

この効果を実現すると同時に、p5.jsで練習するために、それを再作成する簡単なコードを書くことにしました。

p5.j​​sは2014年に作成されたJavaScriptライブラリです。その主な目標は、デザイナー、アーティスト、教師、その他のクリエイティブな職業の代表者のためのプログラミングの世界への窓になることです。アニメーションとダイアグラムの両方をサポートしており、簡単にWeb形式に変換できます。

開始するには、新しいプロジェクトのオンラインエディターを作成する必要があります。デフォルトでは、新しいプロジェクトは次のとおりです。

function setup() {
  createCanvas(400, 400);
}

function draw() {
  background(220);
}

function setup（）-一度だけ実行される関数を設定します
function draw（）-無限ループで繰り返します

まず、「キャンバス」を設定します。createCanvas（）関数では、パラメーター720と500（それぞれ幅と高さ）を設定します。

楕円を背景に表示するには、次のパラメーターをdraw（）関数に挿入します（思い出してください）。

{
  noFill()
  stroke ('purple') 
  strokeWeight(3)
  ellipse(360,250,100,100) }

ここで何が起こっているのですか？

noFill（）-楕円の内側を色で塗りつぶしません。
ストローク（ '紫'）-紫色の線を作成します。
strokeWeight（3）-この線の太さを設定します。
ellipse（360,250,100,100）-指定されたパラメーター（Xに沿った位置、Yに沿った位置、Xに沿った直径、Yに沿った直径）で線を楕円にねじる必要があることを示します。最初と2番目のパラメーターは、createCanvas（400、400）関数のパラメーターの半分に等しく、正確に中央に配置されます。同じ3と4のパラメーターは、楕円ではなく、円の形状を作成します。

次に、これらの楕円をさらに大きくする必要があります。次の楕円はそれぞれ前の楕円より少し大きくなります。これを行うには、ループを作成し、楕円を削除して、3番目と4番目のパラメーターを変数にする必要があります。

 background(220);
  for(let i=0;i<1000; i+=10) {
  noFill()
  stroke ('purple') 
  strokeWeight(3)
  ellipse(360,250,i,i)

人生で少なくとも一度はサイクルに対処したことがあれば、何が起こっているかを理解できます。初期値i = 0を設定するため、最初は半径0の楕円を描画します。次に、サイクルを繰り返す値i（最大1000）に制限を設定し、その後、変更ステップi + = 10-iが前の結果iに追加されるたびに決定します。

パラメーターiを3番目と4番目のパラメーターとして楕円描画関数に代入します幅と高さの直径を担当します。

これはモアレではありませんが、すでに似ています。コードは何をしましたか？彼は一貫して（後者の直径が1000になるまで）100個の円を描きましたが、直径は10ずつ異なり、毎回結果を保存しています。
noFill（）を入力することを忘れないでください。そうしないと、最後の反復の結果のみが保存および表示されます。

次に、2番目の楕円を取得するために、同じループに入力しますが、パラメーターが少し異なります。

function draw() {
  background(220);
  for(let i=0;i<1000; i+=10) {
  noFill()
  stroke ('purple') 
  strokeWeight(3)
  ellipse(360,250,i,i)
    
  stroke("red")
  strokeWeight(3)
  ellipse(360, 250,i-500,i-500)
  }

その結果、一連の円が得られます。円の最大のものは、直径から1000-500を引いたものになります。キャンバスの高さはわずか500なので、内部完璧主義者は満足します。さて、それは赤になります。

最後のアクションの時間です-これらの円を互いに通過させる。

これを行うには、ループのOUTで、コード全体の最初に変数xと値0を追加し、ifループの後にdraw（）関数を追加します。

let x = 0
function setup() {
  createCanvas(720, 500);
}

function draw() {
  background(220);
  for(let i=0;i<1000; i+=10) {
  noFill()
  stroke ('purple') 
  strokeWeight(3)
  ellipse(360,250,i,i)
    
  stroke("red")
  strokeWeight(3)
  ellipse(x,250,i-500,i-500)
  }
if (x>width) {
  x= 0
} else {x= x+3}
}

状態があります。xがキャンバスの幅より小さい場合、xをデフォルトのゼロ値にリセットします。これが発生するまで、xの値に3を加えます。理解できるように、Xを介して、2番目の円のX軸に沿った座標を設定します。毎回わずかに変化するため、円の中心が画面の端に達するまで、円は右に水平に移動します。


2番目の楕円の最初のパラメーターを360から変数xに変更することを忘れないでください。
