Styled Components v5の新バージョンの発表：Animal Grin


styled-componentsの新しい5番目のバージョンを発表できることを非常に嬉しく思います。新しいバージョンは、React ^ 16.8が使用されている場合、以前のバージョンと完全に下位互換性があります。


5番目のバージョンのベータ版を今すぐ試すことができます。


npm install styled-components@beta

React.hooksをサポートするバージョンのReactが必要です。react@^16.8 react-dom@^16.8 react-is@^16.8


そして今、変更について！


早く。さらに速く。スタイル付きコンポーネント。

5番目のバージョンでは、styled-componentsライブラリがさらに高速になりました。4バージョンと比較すると、次のようになります。


  • 19％より小さい minifitsirovannogoコード（16.2kB対13.63kB分+ GZIP） ️

  • 動的スタイルの更新を17％高速化
  • 45％（!!!）速いサーバーレンダリング
