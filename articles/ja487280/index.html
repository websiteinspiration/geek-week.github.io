開発者にとって重要な8つのWebアプリケーション

medium.comで公開されたJamie Bullockの記事の翻訳に慣れることをお勧めします。著者は、彼が自分の作品で通常使用しているWebアプリケーションについて話します。


写真提供：HéctorMartínez。アンスプラッシュから撮影

1.オンラインGDB




オンラインGDBは、ブラウザーコンパイラー、インタープリター、デバッガーです。C、C ++、Python、Ruby、C＃、Swift、 JavaScript など、20のプログラミング言語をサポートしています。

このWebアプリケーションは、IDEまたはターミナルセッションを実行せずにスニペットをチェックするために非常に貴重です。

また、プログラミングスキルの向上や新しい言語の学習にも最適です。

2. JSONスキーマバリデーター




オンラインのJSON検証ツール はたくさんありますが、JSONスキーマ検証ツールが最適だと思います。JSON、JSONスキームを検証し、ユーザーが設定したスキームに対してJSONを検証することもできます。

また、JSON検証を実行できるものと比較して、事前定義されたスキームが多数含まれています。私は毎日JSONスキーマバリデーターを使用していませんが、JSONファイルの解析に問題がある場合は、問題の修正に不可欠です。

3. Regex 101




Regex 101は、正規表現を学習するための優れたツールです。任意のテキストを入力し、ユーザー定義の正規表現に基づいてその中の一致を見つけることができます。

このWebアプリケーションは、一致を即座に見つけるだけでなく、何かを間違って入力した場合のエラーの検出など、正規表現アクションの結果を動的に表示します。

4.素晴らしいフォント




Font Awesomeは、標準のアプリケーションアイコン、会社のアイコン、その他の便利なシンボルを含む、1,500以上の無料のアイコンを含むオンラインディレクトリです。

アイコンは、OpenTypeフォントのセットとして、またはSVG形式でダウンロードできます。Webアプリケーションでは、JavaScriptを介して、iconタグを使用して直接使用できます。

私の意見では、Font Awesomeはプロトタイプを作成するときに特に便利です。これを使用すると、公式のデザインが承認される前でも、開発したアプリケーションにアイコンをすばやく入力できます。

5.スタック交換




重要なWebアプリケーションを1つだけ 選択する必要がある場合は、Stack Exchangeを選択します。

Stack Exchangeは、さまざまなトピックに関する質問と回答を持つサイトのネットワークです。このリソースは1,100万人以上が使用しており、その堅固さを示しています。 Stack Overflowプログラミングサイトでは、投稿されてから数分以内に質問に回答することがよくあります。Stack Overflowの

ウェブサイトで多くのことを学びました。それは多くの実際的な問題を解決するのに役立ちました。 あなたが初心者プログラマーであるか、この分野でのあなたの知識レベルが平均的である場合、次に問題が発生したときに質問をしてみてください。かなり高度な専門家であれば、

いくつかの質問に自分で答えてください。

6. GitHub




このリストはGitHubなしでは完全ではないようです。GitHubは、Gitリポジトリにアクセスして管理するためのWebサービスです。また、バグ追跡、wiki、プロジェクトリリース、プロジェクトカンバンボードなどの機能も含まれています。

GitHubページは、基本的なWebサイトまたはブログのホスティングを提供します。私にとって、GitHubは、ソフトウェアプロジェクトの管理、ドキュメント化、リリースに必要なすべてのものが含まれているユニバーサルリソースです。

7. Unicodeテーブル




Unicode Tableは、検索可能な巨大なUnicode文字データベースです。これには、メモ、矢印、絵文字、通貨記号、数学記号、漢字など、想像できるすべてのものが含まれています。

各文字について、Unicode、HTML、CSSコードとともに、挿入バージョンと完全なUTFエンコーディングデータが提供されます。アプリケーションでこれらの文字を表示するには、Unicode互換フォントが必要です。

8.プログラミングのための音楽




最後に、私のお気に入りのWebアプリケーションの1つであるmusicForProgrammingに出会いましょう！集中力を高めるために特別に設計された音楽を含む50以上のプレイリストが含まれています。

音楽が集中するのを助けるという事実についての議論があります、そして、私はいくつかの音楽スタイルが他のためにこれのためにより良いと思います。私は毎日このサイトを使用していませんが、私にはすごいようです！時々このリソースは私が集中力を高めるために必要なものです。
