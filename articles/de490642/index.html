Plugin zur Überwachung von K8s-Anwendungen DevOpsProdigy KubeGraf v1.3.0: neue Version und neue Funktionen

Diese Woche fand die Veröffentlichung einer neuen Version unseres Plugins für Grafana statt, mit der Kubernetes-Anwendungen DevOpsProdigy KubeGraf v1.3.0 überwacht werden sollen .

Kleiner Haftungsausschluss: Dieses Plugin ist eine überarbeitete Version des offiziellen Plugins von Grafana (das seit etwa zwei Jahren nicht mehr unterstützt wird). Wir entwickeln es seit mehr als sechs Monaten. Hauptmerkmale des Plugins sind:

  • Integration mit k8s-api zum Erstellen einer Karte Ihrer Anwendungen, gruppiert nach Namespace / Clusterknoten + Bindung an bestimmte Pods / Dienste;
  • Eine Übersichtsseite mit allen Fehlern / Warnungen zum Betrieb von Knoten und Clusteranwendungen.
  • die Möglichkeit, das Plugin bei Cloud-K8s-Anbietern durch Autorisierung mithilfe von Bearer-Tokena zu installieren.
