OSCP-攻撃的なセキュリティ

OSCP（Offensive Security Certified Professional）証明書を取得する方法


おそらく、「攻撃的な」セキュリティは、今日お話しする活動の分野の最良の翻訳です。もちろん、攻撃は攻撃的でもあり、攻撃的でもありますが、今はそれについてではありません。

したがって、情報セキュリティの分野でのキャリアへの信頼をまだ失っていませんが、ベンダーからの証明書のほとんどは意味のない紙であり、時間の価値がないと最終的に確信しています。私の同僚の1人は既に、ベンダー認定ダンプスペシャリストの証明書を試験に支払ったすべての人に渡すことを申し出ています。これでこのサーカス全体の終わりです。しかし現実的に考えてみましょう。トレーニングと認定は深刻なビジネスであり、近い将来どこにも行きません。

さらに悪いことに、確認することに決めた知識とスキルが特定のベンダーに関連付けられていない場合。すでにCISSPについて話しましたが、情報セキュリティの分野での実践的なスキルはどうですか？

一方では、非常に人気のあるCertified Ethincal Hacker試験があります。この試験に合格して初めて正式にハッカーと呼ばれるようになるという理由だけで彼が人気を博していることは間違いありません。ここでの問題は異なります。125の質問に正しく回答するだけで、ハッカーになったことにも同意する必要があります。状況の不条理にもかかわらず、この試験は西側諸国で非常に人気があり、特に公的部門と軍事部門の個々の職位に必要なだけです。



一方、偉大で恐ろしいSANSと彼らのサイバーガーディアンプログラムがあります。とてもクールに聞こえますが、修了証明書は、キアヌリーブス氏が人気のイメージで個人的に提示したものだと思います。そして、マイナスはたった1つです-すべての喜びのための数千ドルのコスト。

最後のまま-OSCP。この認定について多くのことが書かれていますが、今度は私の番です。
だから、否定、怒り、交渉、そしてうつ病を経験した後、降伏の準備を始めます。将来を見据えて、準備の過程でこれらのすべての段階を2回以上、さらには異なる順序で、そして奇妙な組み合わせで実行することを言いたいと思います。

コースの支払い後、重いPDFファイル、コメント付きのいくつかのビデオファイル、および仮想ラボへのアクセスが得られます。まだ気になりませんか？そうです、ここには「トレーニング」はありません。 PDFファイルには、nc、curl、findなど、すでに知っている可能性が高いユーティリティのリファレンス情報が含まれています。また、Kali Linuxディストリビューションツールキットについても簡単に説明しています。厳密に言えば、あなたは自分で学ぶ機会のためにお金を与えました、そして仮想実験室はこれであなたを助けます。


合計すると、仮想ラボには60台弱のマシン（ワークステーション、メンバーサーバー、メーラー、ウェブサイトなど）があります。当初、私は試験の課題の難易度を評価するために半分以上を行う予定でした。しかし、私がだましている人は、結局すべてをしました。同時に、ネットワークは独自の生活を送っており、ユーザーはお互いに手紙を書いてWebページにアクセスし、サーバーはいくつかの情報を相互に要求します。

さらに、ネットワークはファイアウォールで区切られたサブネットに分割されており、ある時点で神聖な場所に到達することもできます。すべての車を手に入れるのに40日弱かかりました。もちろん、かなりの数の合格タスクがあり、実際のモンスターがあります。苦痛、苦しみ、そして謙虚であるグーグル。それらのそれぞれのために、それは私に多くの時間がかかりました。 gh0stもありますが、CTFスタイルの車であり、一般的な文脈からは真剣に外れています。


コースの説明には、練習を始めるために深刻な経験は必要ないことが記載されています。しかし、実際にはそうではありません。初心者の専門家がこの仕事を引き受けることはほとんどなく、経験豊富なエンジニアは非常に異なる手荷物に適しています。経験豊富な暗号化のスペシャリストがSufferanceに簡単に対処し、経験豊富なDBAが10分で謙虚さを理解し、経験豊富なLinux管理者が2つに苦痛を打ち砕きます。

いずれにせよ、あなたはほとんどの時間を勉強と実験に費やします。これはOSCP全体です。あなたは実験装置に従事している間、あなたはフォーラムで同じ学生と通信する機会があります。しかし、手がかりは禁じられており、あなたが聞く最良のものは「もっと頑張ってください！」です。まあまあの動機。

ラボネットワークの時間は必然的に終了し、試験の予約時間になります。そして、これはおそらく、OSCPコースと他のすべてのコースとの最大の違いです。質問も、「正しい順序で」、「最も正しい答えを選択する」こともできません。これは起こりません。 5台のサーバーをハッキングするための1日と、その方法を正確に報告するための1日があります。すべて。

ところで、今年はルールが少し変わったので、試験に合格するにはウェブカメラが必要になります。 Offensive Securityの担当者が接続し、画面で何が起こっているかを監視します。また、部屋の中で直接監視します。これは特に煩わしいことではなく、15分後には忘れてしまいます。悲しいかな、これは試験の人気と不正行為の結果として避けられない結果です。

私の場合、試験は最初から最後のサーバーへのアクセスまで12時間続きました。実行したすべてのアクションの再確認とスクリーンショットの記録にさらに数時間を費やしました。ビンゴ！

まず、高価です。そのようなものです。しかし、代替案は何ですか？ SANSコースはさらに高額で、自分のお金で購入する人は誰もいません。

第二に、彼らは何も教えていません。確かに、OSCPコースだけでは何も教えられません。この本では、基本的なツールキットは非常に表面的に分析されており、あなたはほぼ確実にそのことをすべて知っていました。そして、私はビデオについては一般的に黙っています。しかし、コースの本質は異なります-あなたは自分ですべてを教えなければなりません、そしてこれのためにすべての条件が作成されます。教科書に収まらないほど資料が多すぎます。

第三に、古い車。これもまた真実です。間違いではないが、ラボのネットワークで出会った最新のオペレーティングシステムはWindows Server 2012R2でした。私はそれをこのように見ています。新しい脆弱性が毎日発見されています。毎日更新された場合のコースの費用を想像してみてください。このコースで教えることができる主なことは方法論であり、発見された脆弱性の年齢とは完全に無関係です。ちなみに、試験では、状況は逆です-新鮮なWindows 10マシン、パッチを当てたカーネルlinux-マシンなど。

第四に、試験は知識をテストしません。部分的にではあるが、ここで私は同意せざるを得ない。実際、この試験はあなたのスキルそのものをテストするものではなく、時間管理スキル、ストレスの多い状況で作業する能力、およびマルチタスクをテストします。最も重要なことは、方法論に従って、解決策がすでに明らかであるように気が散らないようにする方法です。画面で10時間後、ロジックが失敗し始め、注意が分散し、円を描くように歩き始めます。これは、あなたの基本的な知識だけでなく、別の角度から問題を混乱させて見る能力も役に立ちます。


次は何ですか？攻撃的なセキュリティ自体は、認証の次のステップを提供しません。それとは別に、ワイヤレスネットワーク、Webアプリケーションのセキュリティに関する試験があり、少し離れて、エクスプロイトの開発者のための試験があります。
