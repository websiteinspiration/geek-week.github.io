Habrastatistics：読者のコメントの分析

こんにちはHabr。で前の部分、サイトのさまざまなセクションの人気を分析し、並行して、疑問が生じた-データは記事のコメントから抽出することができるもの。また、以下で説明する1つの仮説をテストしたいと思いました。


データは非常に興味深いものであることが判明し、コメンテーターの小さな「ミニレーティング」を作成することも可能でした。カットの下に続きました。

データ収集


分析のために、特にcsvの形式で記事のリストを既に受け取っているので、2019年のデータを使用します。幸いなことに、各記事からコメントを抽出することに変わりはありません。コメントはそこに保存されており、追加のリクエストは必要ありません。

記事からコメントを抽出するには、次のコードで十分です。

r = requests.get("https://habr.com/ru/post/467453/")
data_html = r.text
comments = data_html.split('<div class="comment" id=')

comments_list = []
for comment in comments:
    body = Str(comment).find_between('<div class="comment__message', '<div class="comment__footer"').find_between('>', '</div>')# .replace('\n', '-')
    if len(body) < 4: continue

    body = body.translate(str.maketrans(dict.fromkeys("\t\n\r\v\f")))
    body = body.replace('"', "'").replace(',', " ").replace('<br>', ' ').replace('<p>', '').replace('</p>', '').replace('  ', ' ')

    user = Str(comment).find_between('data-user-login', '>').find_between('"', '"')
    date_str = Str(comment).find_between('<time class="comment__date-time comment__date-time_published', 'time>').find_between('>', '<')
    vote = Str(comment).find_between('<div class="voting-wjt', '</div>').find_between('<span', 'span>').find_between('>', '<')
    date = dateparser.parse(date_str)

    csv_data = "{},{},{},{}".format(user, date, vote, body)
    comments_list.append(csv_data)
