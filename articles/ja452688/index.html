Windowsサービス。システムエラーを検索し、WinForm Cで表示します＃

この記事では、Windowsサービスで動作し、WinForm（C＃）でシステムエラーを表示するアプリケーションをゼロから作成する方法について説明します。

この記事の概要：

  • サービスの作成
  • イベントビューア
  • サービスコード
  • サービスの確認（手動によるサービスの開始）
  • WinFormマッピング

サービスの作成


Visual Studioを開きます。次のファイル→新規→プロジェクト→（Windowsデスクトップ）→Windowsサービス（.Net Framework）→OK。

次に、インストーラーを作成する必要があります。開いたウィンドウでRMBをクリックし、「インストーラーの追加」を選択します。「ProjectInstaller.cs [Design]」が作成された後、「F7」コードまたはRMB「View Code」に移動する必要があります。「InitializeComponent（）;」という行を見つけてカーソルを置き、「F12」を押してから、次の行を追加する必要があります。

this.serviceProcessInstaller1.Account = System.ServiceProcess.ServiceAccount.LocalSystem; //      
this.serviceInstaller1.Description = "Show me system error."; //      
this.serviceInstaller1.DisplayName = "GetSystemError"; //

ただし、これらの行を追加する必要があるのは、次の順序と場所だけです。そうしないと、サービスのインストール時にエラーが発生します。



イベントビューア


これは、プログラムの正しい動作を確認するために必要です。

イベントビューア-Windowsを搭載したすべてのコンピュータにあるイベントログを表示するためのプログラム。コンピュータで実行される各プログラムは、停止する前にイベントログに通知を発行します。システムへのアクセス、セキュリティの変更、オペレーティングシステムの調整、ハードウェアの障害、ドライバの障害-これらはすべてイベントログに記録されます。イベントビューアはテキストログファイルをスキャンし、それらを結合してインターフェイスに配置します。

どうやって開くの？ -[スタート]→[イベントビューア（検索中）]→[イベントログの表示]。

次に、「カスタムビュー」→「管理イベント」。ここでは、すべてのエラー、警告、およびそれらに関する情報を確認できます。

ログには、アプリケーション、システム、セキュリティの3種類があります。必要なのはシステム（システム）だけです。

サービスコード


サービスの名前が付いた.csファイルが見つかります。「Service1.cs」を使用して開きます。ファイルには2つのオーバーライドされたメソッドが必要です。

  • OnStart（string [] args）-サービスの開始時に実行され、
  • OnStop（）-サービスが停止したときに実行されます。

他にもいくつか方法がありますが、今は必要ありません。あなたはそれらを自分で見つけることができます。

取得したデータは更新されたテキストファイルに保存されるので、

using System.IO;

OnStart（string [] args）メソッドにコードを追加します。

 
EventLog myLog; //      
string filepath = AppDomain.CurrentDomain.BaseDirectory +  @"\ServiceLog.txt";; //   
List<string> list; //   
protected override void OnStart(string[] args)
{
        myLog = new EventLog();
        myLog.Log = "System"; //     - (Application),    (System).
        myLog.Source = "System Error";

        for (int index = myLog.Entries.Count - 1; index > 0; index--) //        
        {
           var errEntry = myLog.Entries[index]; \\    
           if (errEntry.EntryType == EventLogEntryType.Error) \\   
           {
                //    
                var appName = errEntry.Source;
                list = new List<string>();
                list.Add("Entry Type: " + Convert.ToString(errEntry.EntryType));
                list.Add("Event log: " + (string)myLog.Log);
                list.Add("Machine Name: " + (string)errEntry.MachineName);
                list.Add("App Name: " + (string)errEntry.Source);
                list.Add("Message: " + (string)errEntry.Message);
                list.Add("Time Written: " + errEntry.TimeWritten.ToString());
                list.Add("-*-");

                WriteToFile(list); //   
            }
        }
}
public void WriteToFile(List<string> list) //  
{
        using (StreamWriter sw = File.AppendText(filepath))
        {
            for (int i = 0; i < list.Count; i++)
                 sw.WriteLine(list[i]);
        }
}

次に、ソリューション「ソリューション」->「ソリューションの再構築」を収集する必要があります。組み立てが完了したら、動作を確認できます。

サービスの確認（手動によるサービスの開始）


Windowsサービスを通常のアプリケーションとして開始することはできません。管理者としてのみコマンドラインから実行できます。

管理者としてコマンドラインを実行します。次のコマンドを入力します。

cd C:\Windows\Microsoft.NET\Framework\v4.0.30319
InstallUtil.exe  \  .exe
(InstallUtil.exe C:\Users\\source\repos\WindowsService1\WindowsService1 \bin\Debug\WindowsService1.exe)
   : InstallUtil.exe -u \  .exe

次に、Win + Rキーを押します。「Services.msc」と入力します。リストからサービスを見つけてクリックし、「開始」をクリックします。正常に起動すると、システムエラーのリストが配置されるコードで指定されたパスにファイルが生成されます。
確認後、サービスを削除することを忘れないでください。

WinFormマッピング


コンソールで表示する場合、試してみると記事は見つかりますが、WinFormで表示する場合は見つけられなかったので、こちらにあります。デフォルトでは、タイプがアプリケーションのサービスプロジェクトが作成されます。コンソールを介して表示するには、WinFormがそのままにしておくために、このパラメーターを設定で変更する必要があります。次に、フォームをプロジェクトに追加する必要があります。「WindowsService1」→RMB→追加→Windowsフォーム→追加。そして、次のデザインについて行います。次に、「Program.cs」ファイルを変更します。

次を使用して追加：

using System.Windows.Forms;
using System.Security.Principal;
using System.ComponentModel;
using System.Diagnostics;

Mainメソッドを変更します。

static void Main(string[] args)
        {
            WindowsPrincipal windowsPricipal = new WindowsPrincipal(WindowsIdentity.GetCurrent());
            bool hasAdministrativeRight = windowsPricipal.IsInRole(WindowsBuiltInRole.Administrator);

            if (hasAdministrativeRight == false) //   
            {
                ProcessStartInfo processInfo = new ProcessStartInfo(); //  
                processInfo.Verb = "runas"; //      
                processInfo.FileName = Application.ExecutablePath; 
                try
                {
                    Process.Start(processInfo); //  
                }
                catch (Win32Exception){}
                Application.Exit();
            }
            else
            {
                Application.EnableVisualStyles();
                Application.SetCompatibleTextRenderingDefault(false);
                Application.Run(new Form1(args)); // 
            }
        }

新しいクラス「SystemError」を追加します。（ "WindowsService1"-> RMB-> Add-> Class-> Add）。ここにエラーデータを保存します。それを変更：

public string EntryType{ get; set; }
public string EventLog{ get; set; }
public string MachineName { get; set; }
public string AppName { get; set; }
public string Message { get; set; }
public string TimeWritten { get; set; }

さらに「Service1.cs」で、サービスを開始するメソッド「RunFromForm（string [] args）」を追加します。 

public void RunFromForm(string[] args)
        {
            OnStart(args);
            OnStop();
        }

新しいクラス「GetListErrors」を追加します。（ "WindowsService1"-> RMB-> Add-> Class-> Add）。ここでは、ファイルからデータを取得します。次を使用して追加：

using System.IO;

それを変更：

string filepath = AppDomain.CurrentDomain.BaseDirectory + @"\ServiceLog.txt";

        SystemError systemError;
        public List<SystemError> listSysErrs;
        public void ReadFile() //         
        {
            systemError = new SystemError();
            listSysErrs = new List<SystemError>();

            using (StreamReader sr = new StreamReader(filepath))
            {
                string line;
                while ((line = sr.ReadLine()) != null)
                {
                    if (line.Contains("-*-"))
                    {
                        listSysErrs.Add(systemError);
                        systemError = new SystemError(); //      
                    } 
                    if (line.Contains("Entry Type"))
                    {
                        systemError.EntryType = line.Substring(12);
                    }
                    else if (line.Contains("Event log"))
                    {
                        systemError.EventLog = line.Substring(11);
                    }
                    else if (line.Contains("Machine Name"))
                    {
                        systemError.MachineName = line.Substring(14);
                    }
                    else if (line.Contains("App Name"))
                    {
                        systemError.AppName = line.Substring(10);
                    }
                    else if (line.Contains("Message"))
                    {
                        systemError.Message = line.Substring(9);
                    }
                    else if (line.Contains("Time Written"))
                    {
                        systemError.TimeWritten = line.Substring(14);
                    } 
                }
            }
        }

        private void dataGridView1_CellClick(object sender, DataGridViewCellEventArgs e) //               
        {
            SystemError sys = listSysErrs[dataGridView1.CurrentRow.Index];

            label1.Text = "Entry Type: " + sys.EntryType +
                "\nEvent log: " + sys.EventLog +
                "\nMachine Name: " + sys.MachineName +
                "\nApp Name: " + sys.AppName +
                "\n\nMessage: " + FormatMessage(sys.Message) +
                "\n\nTime Written: " + sys.TimeWritten;
        }
        private string FormatMessage(string msg) //       ,                
        {
            string retMsg = "";
            int count = 75;

            if (msg.Length > count - 9)
            {
                retMsg += msg.Substring(0, count - 9) + "\n";
                msg = msg.Substring(count - 9);
            }

            while (msg.Length > count)
            {
                retMsg += msg.Substring(0, count) + "\n";
                msg = msg.Substring(count);
            }
            retMsg += msg + "\n";
            return retMsg;
        }
