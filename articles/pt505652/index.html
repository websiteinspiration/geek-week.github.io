gogrep: pesquisa estrutural e substituição do código Go

O gogrep é um dos meus utilitários favoritos para trabalhar com o Go. Permite encontrar código por padrões sintáticos, filtrar os resultados por tipo de expressão e executar substituições (também por padrão).


Neste artigo, mostrarei como usá-lo gogrep, bem como a extensão VS Code, para um trabalho mais conveniente gogrepdiretamente do editor.



Por que o gogrep é necessário


Se no resumo, gogreppode ser útil para:


  • Reestruturação
  • Estudo de base de código
  • Procure por código suspeito (exemplo: ruleguard )

Considere um exemplo que demonstra a elegância e a eficácia da pesquisa estrutural.


Funções a()e b()execute as mesmas operações:


func a(xs []int) []int {
  xs = append(xs, 1)
  xs = append(xs, 2)
  return xs
}

func b(xs []int) []int {
  xs = append(xs, 1, 2)
  return xs
}

Suponha que queremos reescrever todos os lugares onde as chamadas appendpodem ser recolhidas.


Vamos tentar gogrep:


  • Encontre todos os pares correspondentes usando o -xmodelo.$x=append($x,$a); $x=append($x,$b)
  • Através do -smodelo $x=append($x,$a,$b), obtemos a substituição desejada
  • Ao passar um argumento, -wtodos os arquivos afetados serão atualizados.

gogrep -w -x '$x=append($x,$a);$x=append($x,$b)' -s '$x=append($x,$a,$b)' ./...

Se você instalar a extensão para o VS Code , isso se tornará ainda mais fácil.
