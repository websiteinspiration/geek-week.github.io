スケジュールの解析方法

こんにちはハブル！

親愛なる読者！HTML解析とAndroid開発に興味がある場合は、この記事が役立ちます。その中にたくさんの興味深く有用なものが見つかるといいのですが。その中で私はこの分野での私の経験を共有したいと思います。

問題の説明


私自身について少し。私はITA SFUの3年生です。他の生徒と同様に、毎日クラスのスケジュールを確認する必要があります。そして、翌日だけでなく、1週間か2週間前のスケジュールを知る必要があります。

どうやら、スケジュールを保存して使用するだけではどうでしょうか。残念ながら、これを防ぐにはいくつかの理由があります。

  • 1週間のスケジュールは別の1週間のスケジュールと大きく異なる場合があります
  • スケジュールは一定ではなく、変更される可能性があります

もちろん、スケジュールのあるサイトもありますが、20週間分のスケジュールの生の表を表示するだけなのであまり便利ではありません。生徒は、大きなページをめくって、希望する日のスケジュールを探す必要があります。また、オフラインモードでは、スケジュールは使用できなくなります。
私は、研究所のスケジュールでサイトを解析できる小さなアプリケーションを作成することに決めました。これには、次のような利点があります。

  • 表示：現在の週番号、日付、曜日、その日のスケジュール
  • ボタン「戻る」と「次へ」でスケジュールをスクロールする機能
  • インターネットがない場合は、ダウンロードした最新のオフラインバージョンのスケジュールを表示します

1週間のスケジュール


使用したメソッドを分析しましょう：

Documentタイプの要素を作成します

Document document = null;

ページを取得

document = Jsoup.connect(url).get();

bodyタグの内容を取得します

answer = document.body().html();

Jsoupでは、他のコアタグのコンテンツを取得することもできます。たとえば、title（）メソッドなどを使用してページタイトルを取得できます。Html（）メソッドはHTMLコードを返し、text（）はHTMLタグのないプレーンテキストです。

HTMLコードを受け取ったら、それをプレーンテキストに変換して、すべてのタグを削除できます。これは、解析（htmlcode）.text（）を使用して実行できます。

return Jsoup.parse(answer).text();

使用されていないいくつかのより有用なJsoupメソッドを共有したいと思います。

Element link = document.select("tag");//   
String url = link.attr("attribute"); //

上のスポイラーの写真は1週間のスケジュールの例です。実際、そのような20週間は私たちに返されます。今、私たちの仕事はこのデータセットで今日見つけてそれを表示することです。

頭に浮かぶ


それで私たちは何を持っていますか？ページのHTMLコードを簡単に解析できる文字列にキャストする方法を学びました。これは、文字列メソッド.split（）および.replace（）を使用して簡単に行うことができます。

一般に、アルゴリズムは次のようになります。

まず、Androidから目的の日付を取得します。次に、2つのサイクルを実行し、一方をもう一方にネストします。最初のサイクルは週を通過し、2番目のサイクルはその中にあり、曜日を通過します。その日の日付がAndroidから受け取った日付と一致する場合、この日のスケジュールがテキストボックスに表示されます。ただし、誰でもこのアルゴリズムを独自の方法で作成できます。その実装のバージョンを添付しました。
