Apresentando o .NET 5.0 Preview 3

Apresentando a nova versão do .NET 5.0 Preview 3. Ele contém um conjunto de novos recursos e melhorias de desempenho. Continuamos a trabalhar em recursos mais fundamentais que estarão contidos na versão final do 5.0. O artigo sobre o .NET 5.0 Preview 1 cobre o que planejamos implementar no .NET 5.0. Confira a publicação e o repositório do dotnet / designs e compartilhe seus comentários. E, é claro, instale a Visualização 3 e teste suas tarefas com a nova versão.

Suporte do .NET SDK para os assemblies do .NET Framework


O .NET SDK agora fará referência automaticamente  ao   pacote NuGet Microsoft.NETFramework.ReferenceAssemblies NuGet, com base no .NET Framework de destino no arquivo do projeto. Essa alteração permite criar projetos do .NET Framework em um computador sem o pacote de direcionamento do .NET Framework instalado. Essa melhoria se aplica apenas aos pacotes de direcionamento e não leva em consideração outras dependências que o projeto possa ter.
