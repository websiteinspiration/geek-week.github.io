IRインターフェイス、RaspberryおよびLIRC

私の仕事は、エアコンや家の中の他の機器にコマンドを送信する方法を学ぶことです。当初、これらのデバイスにはIRリモコンしかありません。この問題を解決するために、Raspberry PiとIRトランシーバーシールドを使用しています。記事では、構成、チーム、ヒント、および小さな理論を見つけることができます。ソフトウェアからはLIRC（Linux赤外線リモートコントロール）とPythonになります。


私はGoogleの助けを借りてLIRCを見つけました。調査の過程で、LIRCはIR信号の送信機と受信機の両方で機能し、受信信号をデコードして、この点に関していくつかのアクションを実行できることがわかりました。今では信号の受信は必要ありませんが、将来的には役立つかもしれません。LIRCを自分でいじった場合は、LIRC構成ガイドを読むことを強くお勧めします。

#!/usr/bin/env python3

import io
import sys
import os

r = """begin remote

  name  DEVICE_NAME_CHANGE_ME
  flags RAW_CODES
  eps            30
  aeps          100

  gap          19037

  begin raw_codes

"""

while True:
        print()
        print("enter the command name, or just press Enter to finish")

        a = sys.stdin.readline().rstrip("\n\r")
        if not a:
                break

        first_space = True
        n = 0

        r += "    name " + a + "\n"

        for line in os.popen('timeout 5s mode2 -d /dev/lirc0 2> /dev/null').read().split('\n'):
                words = line.split()
                if len(words) < 2:
                        continue

                if words[0] != 'space' and words[0] != 'pulse':
                        continue
                if first_space and words[0] == 'space':
                        first_space = False
                        continue

                if n % 4 == 0:
                        r += "\n        "

                r += words[1] + "  "
                n = n+1

        if n > 5:
                print ("got", n, "values, looks good")
        else:
                print ("no signal, something went wrong")

        r += "\n\n"

r += """  end raw_codes
end remote
"""

print()
print(r)
print()
