視覚ではなく聴覚：暗闇に適応するように脳ニューロンを再構成する



人間は、地球上の他の多くの動物と同様に、ほとんどが日中の生き物と考えられています。私たちの体のほとんどすべてのシステムは、日中の活動と夜間の休息のために調整されています。私たちが眠るときと私たちが起きているときの最も重要な調節因子は私たちの脳です。また、感覚を通じて受け取った情報の処理も担当しています。この情報の正確さは、それが受信された状況とそれをどのように解釈するかによって異なります。夕暮れには、コートが掛かっているハンガーがあり、私たちの脳は「これは今私たちを食べるモンスターです」と告げています。ただし、特定の状況下では、一部の信号は他の欠点を補うことができます。

今日は、メリーランド大学（米国）の科学者が実験用マウスを真っ暗闇に1週間置いた研究についてお話しします。実験中にマウスの行動はどのように変化し、脳内のニューラルネットワークにはどのような変化があり、科学者は観察からどのような結論を導きましたか？研究者の報告はこれらの質問に光を当てるでしょう。行け。

研究の基礎


明るい部屋から暗い部屋に移動するとき、私たちのビジョンは少し適応する必要があることは誰もが知っています。このプロセスはかなり平凡と呼ばれています-目の順応。暗い部屋に入ると、目が新しい状態にパニックになり始め、その後このプロセスが遅くなります。数分間暗闇に入る前に目を閉じれば、順応はより良く、ゆっくりと進みます。まさにこの理由で、海賊は片方の目で目隠しをしていたと考えられています。照明が良くなかった船の船倉に入ると、もう一方の目で目隠しを変更したため、以前は閉じていた情報がよりよく知覚されました。それは本当のようですが、それは単純なおとぎ話でありえます。

しかし、視覚に加えて、聴覚を含む他の感情があります。目の不自由な人のほうがよく聞こえるとよく耳にします。また、1つのモダリティ*内の感覚的可塑性は子供でより発達すると考えられていますが、大人でも、クロスモーダル可塑性が発達する可能性があります。
モダリティ* -特定のセンサーシステムへの特定の信号の帰属。モダリティには、視覚、聴覚、痛み、運動感覚などの種類があります。
クロスモーダル可塑性の場合、脳ニューロンは、2つ以上のシステムの機能を組み合わせるような方法で再編成されます。この現象は、外傷または疾患による特定のシステムの感覚遮断（悪化）を背景にして発生することがよくあります。最も顕著な神経再編成は、誕生以来盲目または聴覚障害者であった人々に存在します。つまり、剥奪が長いほど、クロスモーダルな可塑性が強くなります。

言い換えれば、感覚モダリティの喪失は、他の感情の可塑性によって補うことができます。これまでの研究では、出生時の盲人が音の信号、音源の位置（音の定位）、周波数特性をよりよく知覚していることが示されています。明らかに、そのような変化は先天性感覚障害の場合により発達します。ただし、さまざまな種の成虫でさえ、環境の変化により感覚やクロスモーダルの可塑性を発達させる可能性があります。

たとえば、細胞レベルでのげっ歯類の視覚剥奪が短期間になると、聴覚皮質（A1）の周波数選択性が高まり、しきい値を下げてL4ニューロンによるインパルスの伝達率を増加させます（視床受容層4）聴覚皮質。細胞レベルでの同様の変化は、神経回路の変化の結果です。暗闇の影響により、L4の視床入力信号が増加し、L4ニューロンからL2 / 3ニューロン（視床受容層2/3）への皮質内接続が上昇し、L2 / 3内の興奮性および抑制性の層内結合が増加します。 L2 / 3へのフィードバック接続とL2 / 3からL4へのフィードバック接続。

ただし、感覚刺激は、個々のニューロンではなく、そのグループ（母集団）によって情報に変換されます。したがって、そのような集団内のニューロン間の接続の変化は、着信信号からの情報の形成に直接影響を与える可能性があります。

音の周波数の知覚はL4とL2 / 3の間で変化するため、科学者たちは成熟した年齢の個人に対する暗闇の一時的な影響が実験室マウスの聴覚皮質内でニューロンとそれらの相互関係を再構築できるかどうかを確認することにしました。二光子カルシウム（Ca 2+）イメージングを使用して、脳の変化の過程を観察しました。 L2 / 3およびL4ニューロンの音による活動が測定され、周波​​数選択性の増加の存在が示されました。


画像No. 1：マウスのA1におけるニューロンの2光子カルシウム（Ca 2+）イメージング。

実験用マウスは実験用マウスであり、無作為に2つのグループに分けられました：対照（通常の照明下で生活、9個体）と試験（暗​​所で生活、6個体）。暗闇での滞在（DE-暗い露出）は比較的短命でした-1週間（1A）。


表1：カルシウムイメージングにより、聴覚皮質の各層における数百のニューロンの活動の画像を得ることができました。

科学者たちはまた、以前の研究ではDEが内部細胞透過性の特性に変化を引き起こさないことを示していることにも言及しています（1C）。イメージング中にGCaMP6（1B）カルシウム指示薬が使用されたため、これは非常に重要です。

対照群と試験群のマウスの個々のニューロンの活動を特徴付けるために、300〜300μmのプロットが、音にさらされた瞬間（純音）4〜64 kHz、60 dB（1D）のL2 / 3およびL4層で視覚化されました。まず、これらの刺激に反応した細胞が特定されました。少なくとも何らかの形で反応した細胞は「反応性」として分類されます。暗闇にさらされた後、L4の反応している細胞の割合は変化しませんでしたが、L2 / 3少ない細胞が刺激に反応しました（1E）。

個別ユニットの登録* テストグループのマウスのL4細胞は、自発的で暗闇に誘発されたニューロン頻度（ニューロン活動）が高いことを示しました。
個々のユニットの登録* -微小電極を使用して個々のニューロンの電気生理学的応答を測定する方法。
さらに、L2 / 3レイヤーの個々のセルのレベルに同様の変化が存在するかどうかを確認することが決定されました。マウスGCaMP6を発現するニューロン（SD）の自発的活動を評価するために、一時的な蛍光プロセスが、刺激の開始に先立って、刺激なしで長時間イメージング中に測定されました。

暗闇にとどまった後、ニューロンの自発活動はL4とL2 / 3の両方で増加しました（1F）。

調性感度の低下は、聴覚皮質のニューロンの周波数調整の変化に関連している可能性があります。以前の微小電極の記録では、テストグループマウスのL4細胞は周波数選択性が向上しており、L2 / 3はL4から入力を受け取るため、L2 / 3にも同じ変化が見られます。これに続いて、音の再生によって引き起こされる最大応答に基づいて、応答セルごとにチューニング曲線が作成されました（2A）。


画像No. 2：暗闇に留まると、L4とL2 / 3の両方のニューロンの感度と周波数選択性が向上します。

最初に、誘発された応答の振幅が最良の周波数（BF、つまり曲線のピーク周波数）で測定されました。暗闇への曝露後の応答の振幅は、L4ニューロンとL2 / 3ニューロンの両方で増加しましたが、L4の方が大きかった（2B）。この観察は、電気生理学的記録およびL4への視床皮質求心性*の増加と完全に一致しています。
視床皮質共鳴* -視床および大脳皮質の関連領域のさまざまな核のニューロンの同期振動（ニューロンの個々の集団の周期的な同時活性化）の現象。

求心性*は、受容体からのインパルスを脳または脊髄に伝達するニューロンです。
次に、試験群と対照群のマウスの細胞の周波数選択性を評価しました。分析は、暗闇への曝露後のニューロンL4およびL2 / 3の帯域幅が、コントロールグループの動物よりも小さいことを示しました（2C）。

これらの観察の全体は、個々のセルのレベルで、暗闇の後の変化がL4とL2 / 3の両方で同じであることを示していますが、L4の応答振幅は暗闇の後に増加しますが、L2 / 3では増加しません。

したがって、一定の時間が経過すると、暗闇にさらされると、L4とL2 / 3の両方で、聴覚皮質のニューロンの音による反応が変化する可能性があります。しかし、L4では、これらの変更はL2 / 3よりもさらに重要になります。暗闇の影響を受けた後、音に反応する細胞は少なくなりましたが、A1で反応した細胞は、音に対してより敏感で選択的になります。つまり、暗闇では、聴覚皮質のニューロンは「量ではなく質」の原理に基づいて作用します。これは、応答細胞の数は減少しますが、その活動は増加するためです。


画像3：暗闇にさらされると、聴覚皮質の周波数選択性の分布が変化します。

観察データは、L4聴覚皮質の緊張に敏感な細胞が応答のより高い振幅を示し、両方のA1層の細胞が暗闇にさらされた後に増加した音選択性を示したことを示しています。ただし、個々の細胞のレベルでのこれらの変化は、暗闇にさらされた後、L2 / 3の反応ニューロンが少ない理由を説明していません。感覚皮質層のニューロンは、行動要因に基づいて設定を調整できます。さらに、初期の感覚的経験は、特定の周波数の音に反応する聴覚皮質の領域を変える可能性があります。この情報に基づいて、科学者たちは個々の神経集団が外部刺激に対する好みを変えたと示唆しました。

この仮説をテストするために、科学者はテストとコントロールグループからのマウスのニューロンの集団の優先周波数の分布を調査しました。

コントロールグループのマウスは、4〜64 kHzの範囲の「優先」周波数を選択しますが、ほとんどのニューロンは、8〜32 kHzの範囲の音声を選択します。しかし、テストグループのマウスは、32〜64 kHzの範囲の高周波に対してより積極的に反応しました（3B）。

L2 / 3レイヤーでは高周波数（32〜64 kHz）に反応する細胞数の増加が見られ、L4レイヤーではそれより低い細胞（4〜8 kHz）が観察されました。両方の層で、8〜16 kHz（3A）の範囲の中周波数の知覚の低下が観察されました。

ご存知のように、感覚刺激は個々のニューロンだけでなく、ニューロンの集団によってもエンコードされています。ニューロン間の活動の相関関係は、今度は情報をエンコードするプロセスに貢献します。 L4とL2 / 3の両方で、隣接セルは信号（SC）の高い相関を示します。これは、刺激によって駆動される相関活動を反映しています。実験間の刺激に依存しない共分散を表すペア相関（NC）もあります。

科学者たちは、暗闇にさらされた後に機能的な中間層と層内の結合を改善すると、対相関の減少につながる可能性があることを示唆しています。したがって、NCを使用すると、暗闇にさらされた後のL4とL2 / 3のニューロン間の相関活動レベルの変化を研究できます。


画像番号4：暗闇の影響によるL4レイヤーのNCの減少。

暗闇の影響により、L4層の細胞間のNCとSCが減少しました。ただし、L2 / 3レイヤーでは、NCは実質的に変化しませんでした（5A）。


画像No. 5：暗闇の影響によるL2 / 3レイヤーのNC低減。

闇は、L2 / 3（5B）のニューロン間のSCの減少をもたらしました。このことから、暗闇は個々のニューロンの設定だけでなく、L2 / 3層とL4の両方での相互の相互接続も変化させることがわかります。

エピローグ


1週間暗闇にとどまったマウスは、通常の光で生きたマウスとはまったく異なるニューロン活動を示しました。テストグループのマウスの聴覚皮質のニューロンは、高周波数と低周波数のより正確な知覚を実現するために、それら自体と相互の接続を再構成し、中間のニューロンの知覚を犠牲にしました。科学者自身はまだそのような選択性を正確に説明することはできません。彼らは、これはマウスがより良く聞きたかったことに起因している可能性があることを示唆しています：彼らのステップ、他のマウスからの音など。

実験用マウスは健康であることに注意することが重要です。彼らの視力は誕生から全く正常でした。以前は、神経活動のそのような変化は、出生から（または幼い頃から）何らかの感覚機能障害に苦しんでいる人に独占的に発生する可能性が高いと想定されていました。しかし、結局のところ、大人の脳でさえ、環境条件に適応して変化する可能性があります。

将来、科学者はマウスが聞く音の操作を追加することにより、研究を拡大することを計画しています。これにより、どの音がどの程度正確にマウスが暗闇で反応するかをより正確に判断できます。
