フェルマーの大定理を超えた「見事な」数学的橋

数学者は、数学の世界の遠く離れた2つの大陸を結ぶ謎の橋を長くする方法を理解しました




アンドリュージョンウィルズが 1990年代初頭にフェルマーの大定理を証明したとき、それは数学者だけでなくすべての人類にとって記念碑的な一歩でした。定理のステートメントは非常に単純です-方程式x n + y n = z nn> 2には肯定的な解決策はまったくありません。しかし、フランスの数学者ピエール・ド・フェルマーがディオファントスの「算術」の余白に1637年の定理のステートメントをさりげなくスケッチしたため、この単純なステートメントは350年以上もそれを証明したいと望む多数の人々を魅了しました。フェルマートの定式化も有名です。「これについては本当に素晴らしい証拠が見つかりましたが、本の余白が狭すぎるためです。」何世紀にもわたって、プロの数学者やアマチュア愛好家は、フェルマーの証拠、またはその他の証拠を求めてきました。

ウィルズが（リチャードテイラーの助けを借りて） 最終的に取得した証明は、フェルマートには決して発生しなかったでしょう。それは定理に直接影響を与えなかったが、数学者によれば、存在すると想定されていた巨大な橋を建設した-2つの離れた数学的「大陸」の間の橋。ウィルズの証明は、2つの大陸間の2つの小さな土地をつなぐこの橋を定義することに帰着しました。証明は新しく深いアイデアに満ちていて、この橋の両側に新しい結果のカスケードが生まれました。

この観点から、ウィルズの驚くべき証拠は、はるかに大きな謎のごく一部を解決しました。彼の証明は「20世紀の数学における最高の出来事の1つ」でした、とToby Guyは言いましたインペリアルカレッジロンドン出身。そしてそれは、ラングランズの幾何学的対応として知られている橋の「小さなストレッチ」に属していました。

橋全体によって、数学者は数学の広大な広がりに光を当て、概念をある部分から別の部分に伝えることができます。フェルマーの大定理を含む多くのタスクは、橋の片側では難しいようですが、反対側に移動することですぐに簡単なタスクに変わります。

ウィルズが彼の証明を思いついた後、他の数学者はこの橋を2つの大陸のより大きなセクションに熱心に拡大し始めました。そして、彼らは障害にぶつかりました。この橋を拡張するための2つの自然な方向性がありますが、どちらの場合も、テイラーワイルズ法は克服できない障壁に遭遇したようです。インペリアルカレッジロンドンのアンナカラヤニ氏は 、


フェルマーの大定理を証明し、2016年にアーベル賞を受賞した数学者のアンドリューウィルズ氏は

、「これは長い間望んでいた」と語った。しかし、「一般的に言って、これが原則的に可能であるとは考えていませんでした。」

現在、2人の作品（10人以上の数学者の作品の集大成を表す）がこの障壁を克服し、本質的に両方の問題を解決しています。いつの日か、これらの発見は、数学者が正の整数を超える数値システムのフェルマーの大定理を証明するのに役立つ可能性があります。

これらは「上位の結果」であるとシカゴ大学のマシュー・エマートン氏は語った。「それらは数論からいくつかの基本的な現象を明らかにし、私たちはそれらが何であるかを理解し始めたばかりです。」

真空中の針


Langlandsブリッジの片側は、書き留めることができる最も単純な方程式に集中しています。これらはディオファントス方程式、または指数と整数係数の変数の組み合わせです。たとえば、y = x 2 + 6x + 8、またはx 3 + y 3 = z 3。数千年の間、数学者は整数のどの組み合わせが特定のディオファントス方程式を満たすかを解明しようとしました。基本的に、彼らの動機はこの問題の単純さと自然さに基づいていますが、最近、彼らの作品の一部は暗号などの分野で予期せぬ継続を受けました。

古代ギリシャ以来、数学者は変数が2つしかなく、次数が2を超えないディオファントス方程式の整数解を見つける方法を知っていました。ただし、次数が高い場合、整数解の検索は単純な問題とは言えません-楕円曲線から始めます。これらは、y 2が等号の左側にあり、右側が最大次数が3の項の組み合わせである方程式です。たとえば、x 3 + 4x + 7です。ガイは、次数の低い方程式と比較して、„根本的にもっと複雑な問題です。」

橋の反対側には、オートモーフィックフォームと呼ばれるライブオブジェクトがあります。これは、非常に高度な対称性を持つタイルのカラーリングに似ています。ウィルズによって研究されたケースでは、タイルはエッシャーのモザイクに似たものに似ているかもしれません。そこでは、悪魔がディスクに表示されている魚や天使は、境界に近づくにつれて減少します。より一般的なラングランズの世界では、タイルは3次元のボールや他の図形をより高次元で敷くことができます。

これらの2つのタイプの数学オブジェクトは、互いに完全に異なります。それにもかかわらず、20世紀の半ばに、数学者たちはそれらの間の深い関係を明らかにし始め、1970年代の初めまでに、高等研究所のロバートラングランズはディオファントス方程式と保型形式が相互に特定の方法で相関できるという仮説を表明しました。


50年前にコンプライアンスの仮説を提唱したロバートラングランズは、2016年にニュージャージー州プリンストンの高等研究所で講演を行います。

つまり、ディオファントス方程式と保型形式の両方で、無限の数列を生成する自然な方法があります。ディオファントス方程式を使用すると、モジュラー演算で解の数を計算できます（これは、時計の文字盤にある数値として表すことができます。たとえば、12時間ダイヤルの場合、10 + 4 = 2）。そして、ラングランズに従って表示されるそのような自己同形フォームの場合、量子エネルギーのレベルに類似した数の無限のリストを取得できます。

素数のみに基づいてモジュラー演算を使用する場合、ラングランズの仮定により、これら2種類のシーケンスは驚くほど広い範囲の異なる条件で一致します。言い換えると、自己同形の場合、そのエネルギーレベルはディオファントス方程式のモジュラーシーケンスを制御し、その逆も同様です。

この関係は「テレパシーでさえ変だ」とエマートン氏は語った。 「私はこの現象を20年以上研究してきましたが、これらの2つの側面が互いに通信する方法は、私にとって驚くべき、信じられないようです。」

1950年代と1960年代に、数学者はこの橋の存在の最初の兆候を、方向の1つに発見しました。ある種のオートモーフィック形式から有理数（整数からなる小数）の係数を持つ楕円曲線への移動方法。その後1990年代に、ワイルズはテイラーと共に、楕円曲線の特定のファミリーのための橋の別の方向を発見しました。彼らの結果は自動的にフェルマーの大定理の証明を生み出しました。なぜなら、それが正しくない場合、これらの楕円曲線の少なくとも1つは対応する自己同型形を持たないことを数学者がすでに示しているからです。

フェルマーの偉大な定理は、この橋の建設に続く唯一の発見からはほど遠いものでした。たとえば、数学者はそれを証明するためにそれを使用しました佐藤立仮説、楕円曲線のモジュラー解の数の統計的分布に関連する数十年前の問題、ならびに20世紀初頭のスリニバサラマヌジャンアイエンゴアの伝説的な数学者によって表現された、自己同型形式のエネルギーレベルに関する仮説を証明すること。

WilesとTaylorが調査結果を発表した後、彼らの方法はまだ可能性に満ちていることが明らかになりました。すぐに、数学者はそれを有理係数をもつ楕円曲線に拡張する方法を理解しました。その後、数学者考え出しなど3 +√2のような単純な無理数、との係数をカバーする方法を。

しかし、彼らが成功しなかったことは、Taylor-Wiles法を、i（√-1）または3 + i、または√2iなどの複素係数を持つ楕円曲線に拡張することでした。また、それらは楕円曲線よりもはるかに多くのパワーを持つディオファンタス方程式に対処できませんでした。等号の右側が3ではなく4である方程式は、テイラーワイルズ法を使用して簡単に解くことができましたが、次数が5に増えるとすぐに、この方法は機能しなくなりました。

数学者は、ラングランズ橋のこれら2つの自然な拡張の問題は、テイラーワイルズ法にわずかな改善を加えるだけではないことに徐々に気づき始めました。どうやら、障害は根本的なものでした。

これらは「私に起こったちょうど次の例」でした、とガイは言いました。 「しかし、彼らはあなたに言った：いいえ、これらのことは絶望的に手の届かないところにあります。」

問題は、Taylor-Wiles法がディオファントイン方程式に対応する自己同型形式を、他の自己同型形式を使用して連続的に近似することによって見つけることでした。ただし、方程式の係数に4を超える複素数またはべき乗が発生する場合、保形形式は非常に少なく、ほとんどすべての保形形式は、それを近似するために使用できる最も近い保形形式を持たない可能性があります。

ウィルズのもとでは、私たちが必要とする自己同形の形状は「干し草の山の針ですが、この山は存在します」とエマートンは言いました。「そしてこれは、磁石を持ってくる金属ファイリングのスタックと比較することができます。ファイリングは整列され、必要な針を指し示します。」

しかし、彼によると、複雑な係数または高次の次数の場合、それは「真空中の針に似ています」。

月への飛行


今日の数論の専門家の多くは、ウィルズが彼の証明を思いついたときに育っていました。 「これは新聞の表紙で見た数学の唯一の例でした」と当時13歳だったガイは回想します。 「それは多くの人にインスピレーションを与え、彼らはそれを理解したかったので、その結果、彼らがこの分野で働き始めたのはこのためでした。

したがって、2012年に2人の数学者- シカゴ大学のフランクカラガリとデビッドジェラティ（現在はFacebookの研究者）- がテイラーワイルズ法の拡張を許可しない障害を克服する方法を提案したとき、このアイデアは、新世代の数論の専門家から絶賛されました。

彼らの仕事は、「私たちの進歩を妨げるこの根本的な障害はまったく障害ではなかった」とガイは言った。彼は、実際には、テイラー・ワイルズ法の明らかな制限は「あなたはカレガリとジェラティによって提示された実際の、より一般的な方法の影を感じただけだ」と示唆していると説明しました。


2015年にボストン大学でDavid Geraty氏

突然障害物が発生した場合、オートモーフィックフォームは、Wilesが調査した2次元のEsherスタイルのタイルよりも高次元のタイルにあります。これらの高次元の世界では、保型形式が非常にまれであることは不快です。しかし、より高い次元のタイルは、多くの場合、2次元のものよりも豊かな構造を提供します。カリガリとジェラティは、この豊富な構造を使用して、保型形式の欠如を補うというアイデアを思いつきました。

より正確には、特定のオートモーフィックフォームごとに、そのタイルの「カラーリング」を、選択したタイルの任意の部分の平均色を計算できる測定ツールとして使用できます。 2次元の状況では、実際には、保型形式だけが利用可能なそのような測定ツールです。しかし、高次元のタイルには新しいツール、いわゆるねじれクラス、およびそれらの助けを借りて、各タイルセクションには、平均色ではなく、モジュラー演算からの数を割り当てることができます。そして、そのようなクラスのねじれは10ダースです。

KalegariとGeratiは、いくつかのディオファントス方程式では、他の自己同型形式ではなく、クラスをねじることによって、対応する自己同型形式を見つけることが判明する可能性があることを示唆しました。 「それらについてのこの考えは素晴らしいものであることが判明しました」とカラジャニは言った。

カリガリとジェラティは、ディルファンティン方程式からオートモーフィック形式への、WilesとTaylorが構築したものと比較して、はるかに幅広いブリッジを構築するためのスキームを示しました。しかし、彼らの考えは本格的な橋とは考えられませんでした。それを機能させるには、まず3つの大きな定理を証明する必要がありました。カレガリによれば、これは彼らのゲラティとの仕事が月への飛行の計画を説明しているという事実と比較することができます。そして、これら3つの定理は「私たちの手の届かないところに完璧でした」とKalegariは言いました。

特に、CalegariとGeratiの方法は、保型形式からディオファントス方程式まで、別の方向に進む既製の橋の存在を要求しました。そして、この橋は、保型形式だけでなく、ねじれたクラスも組み合わせることになっています。 「カレガリとジェラティが最初に彼らのプログラムを説明したとき、多くの人がこれを絶望的な仕事だと思ったと思います」と現在スタンフォード大学にいるテイラーは言いました。

KalegariとGeratiの作品の出版後、一年未満ピーター・スコールズがあるボン大学からの若い天才受信フィールド賞を、数学の最高賞、数論の驚くべきスペシャリスト、係数が3 + 2iまたは4-√5iのような単純な複素数である楕円曲線の場合に、ねじれクラスからディオファントス方程式の側面に切り替える方法を考え出します。 「彼は多くの驚くべきことをしましたが、これはおそらく彼の最も驚くべき成果です」とテイラーは言いました。


数学者ピーター

ショルツェショルツェは、カレガリとゲラティの3つの定理の最初を証明しました。そして、ショルツェとカラヤニによるその後の共同作業のカップルは 、第2の定理を証明することに非常に近くなり、ショルツェによって発見された橋に正しい特性が存在することを示しました。

このプログラムは簡単に習得できると感じていたため、2016年の秋に、カラジャニとテイラーは、さらなる進歩を達成することを目的とした高等研究所の「秘密のワークショップ」であるKalegariに従って組織しました。 「私たちはそこで1人の聴衆を占領し、誰も入れませんでした」とKalegariは言いました。

2、3日の準備会話の後、ワークショップの参加者は、2番目の定理を同時に処理し、3番目の定理を回避する方法を理解し始めました。 「そして、おそらくすべてのタスクを策定してから1日以内に、全員で解決しました」とプロジェクト参加者の1人であるガイは言いました。

週の残りの期間、参加者は証拠のさまざまな側面の詳細な調査に専念し、次の2年間で作業での発見を正式化しました。10人の著者の場合-この数は、数論の研究では前代未聞です。実際、彼らの研究は、有理数と単純な非有理数および複素数で構成される任意の数体系からの係数を持つ楕円曲線のラングランズ橋の存在を確立します。

「ワークショップは、主にソリューションにどれだけ近づくことができるかを理解するために組織されました」とガイは言いました。「私たちの誰もがすべてを証明することを期待していたとは思いません。」

橋の続き


一方、楕円曲線を超えた橋の継続に関連する別の物語が展開されていました。 CalegariとGuyは、Diophantine方程式の最高次数が5または6であるケース（既知のケースでは3と4ではない）について、George Boxer（現在はリヨンの高等師範学校で働いています）と協力しました。しかし、3人の数学者が証明の要点で行き詰まりました。

数週間で4人の数学者がこの問題を解決しましたが、2、3年の作業と約300ページのアイデアの詳細な説明が必要でした。彼らの作品と10人の著者の作品は、2018年12月に4日間の差でインターネットで公開されました。

は、「これは非常に深刻な成果です」と、これら2つの作品についてエマートンはコメントしました。彼はそれらとそれらに先行する構成要素を「芸術作品」と呼んだ。

実際、これらの2つの作品は、ディオファントス方程式と保型形式の間の神秘的なテレパシー接続が新しい条件に転送されることを証明しますが、1つの問題があります。2つの数学的な海岸の間に理想的な橋を架けないことです。作品は「自己同型の存在の可能性」のみを述べている。これは、各ディオファントス方程式に対応する保形形式があることを意味しますが、科学者によれば、この保形形式が大陸のその場所に存在するかどうかはわかりません。ただし、潜在的な自己同型性は、多くのアプリケーションで十分です。たとえば、ディオファントス方程式のモジュラーソリューションの統計に関する佐藤立仮説では、以前よりもはるかに広い景観での操作性が10人の著者によって証明されました。

そして、新しい作品が橋を以前よりもラングランズ大陸のはるかに広いセクションに接続していますが、それらはまだ広大な領域を未知のままにします。ディオファントス方程式の側から見ると、これらの方程式には、次数が6より大きいすべての方程式と、3つ以上の変数を持つ方程式が含まれます。一方、未知の領域には、今日までに研究されてきたものよりも複雑な対称空間に住んでいる保型形式が含まれています。

「今日、この作品は成功の頂点を象徴しています」とエマートン氏は語った。 「しかし、ある時点で、それらは目標を達成するためのステップの1つと見なされます。」

ラングランズ自身は、ねじれた形を研究したことはありませんでした。そのため、数学者にとって困難な作業の1つは、これら2つの異なるアプローチの統一的な見方を見つけることです。「私達は私達の範囲を拡大しています」とテイラーは言いました。「私たちはラングランズと何らかの形で道を外れましたが、どこに行くのかわかりません。」
