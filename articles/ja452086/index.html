負の電位を加えると、PANIシェル（PANI 0）が減少し、c 0 = 642 nmに散乱ピークが生じます。そして、電位の逆はc 2+ = 578 nm への共鳴シフトにつながりますが、Δλ* = 64 nmは以前に実行されたモデリングと一致しています（1s）。

理論的には、科学者によれば、ギャップが減少すると、共鳴波の長さとそのスペクトル調整の範囲が増加するはずです。実際には、すべてが異なることがわかりました。PANIシェルの薄化により、レドックスサイクル中の色の範囲が狭くなりました。研究者はこれを、（理論的には）モデリングで考慮されなかった追加の構造的要因で説明します。
