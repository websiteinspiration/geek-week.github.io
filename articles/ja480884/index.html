5分でPython GUI



Pythonは使いやすいです。その中には、あらゆるもののための膨大な数のライブラリーがあります。そして、これが彼の主な利点です。数行のコードからは何もしません。個人的な使用や技術に詳しい読者のためのスクリプトが必要な場合は、グラフィカルインターフェイスについて考える必要すらありません。

ただし、対象読者が技術にあまり詳しくない場合もあります。人々は、1行のコードを見る必要があるまで、Pythonスクリプトの使用を気にしません。この場合、コマンドラインスクリプトでは不十分です。理想的には、グラフィカルインターフェイスが必要です。この投稿の目的は、Pythonのみを使用することです。

GUIをプログラムする


グラフィカルインターフェイスを作成するには、次のコードを使用できます。

import PySimpleGUI as sg
layout = [
    [sg.Text('File 1'), sg.InputText(), sg.FileBrowse(),
     sg.Checkbox('MD5'), sg.Checkbox('SHA1')
     ],

     sg.Checkbox('SHA256')
     ],

ロジックをつなぐ


ユーザーインターフェイスがある場合、残りのコードを接続する方法を理解するのは簡単です。ユーザーが入力した内容を追跡し、それに応じて行動するだけです。これは次のコードで非常に簡単に実行できます。

import PySimpleGUI as sg
import re
import hashlib
def hash(fname, algo):
    if algo == 'MD5':
        hash = hashlib.md5()
    elif algo == 'SHA1':
        hash = hashlib.sha1()
    elif algo == 'SHA256':
        hash = hashlib.sha256()
    with open(fname) as handle: #opening the file one line at a time for memory considerations
        for line in handle:
            hash.update(line.encode(encoding = 'utf-8'))
    return(hash.hexdigest())
layout = [
    [sg.Text('File 1'), sg.InputText(), sg.FileBrowse(),
     sg.Checkbox('MD5'), sg.Checkbox('SHA1')
     ],
    [sg.Text('File 2'), sg.InputText(), sg.FileBrowse(),
     sg.Checkbox('SHA256')
     ],
    [sg.Output(size=(88, 20))],
    [sg.Submit(), sg.Cancel()]
]
window = sg.Window('File Compare', layout)
while True:                             # The Event Loop
    event, values = window.read()
    # print(event, values) #debug
    if event in (None, 'Exit', 'Cancel'):
        break
    if event == 'Submit':
        file1 = file2 = isitago = None
        # print(values[0],values[3])
        if values[0] and values[3]:
            file1 = re.findall('.+:\/.+\.+.', values[0])
            file2 = re.findall('.+:\/.+\.+.', values[3])
            isitago = 1
            if not file1 and file1 is not None:
                print('Error: File 1 path not valid.')
                isitago = 0
            elif not file2 and file2 is not None:
                print('Error: File 2 path not valid.')
                isitago = 0
            elif values[1] is not True and values[2] is not True and values[4] is not True:
                print('Error: Choose at least one type of Encryption Algorithm')
            elif isitago == 1:
                print('Info: Filepaths correctly defined.')
                algos = [] #algos to compare
                if values[1] == True: algos.append('MD5')
                if values[2] == True: algos.append('SHA1')
                if values[4] == True: algos.append('SHA256')
                filepaths = [] #files
                filepaths.append(values[0])
                filepaths.append(values[3])
                print('Info: File Comparison using:', algos)
                for algo in algos:
                    print(algo, ':')
                    print(filepaths[0], ':', hash(filepaths[0], algo))
                    print(filepaths[1], ':', hash(filepaths[1], algo))
                    if hash(filepaths[0],algo) == hash(filepaths[1],algo):
                        print('Files match for ', algo)
                    else:
                        print('Files do NOT match for ', algo)
        else:
            print('Please choose 2 files.')
window.close()

最終的な考え


これは最も美しいユーザーインターフェースではないかもしれませんが、PySimpleGUIを使用すると、簡単なPythonユーザーインターフェースをすばやくデプロイして、誰とでも共有できます。これに必要なコードはシンプルで読みやすいものです。ユーザーインターフェースを取得するためのコードの実行には引き続き問題があります。このため、コードの共有が難しい場合があります。PyInstallerのようなものをダウンロードして、Pythonスクリプトを.exeファイルに変換することをお勧めします。人々はそれを2回クリックするだけでそれを起動することができます。
