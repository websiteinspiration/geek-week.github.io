Pythonで金融危機を予測する

2020年3月9日現在、米国市場は急激に減少しており、2020年2月20日の歴史的な高値から始まり、現在は約-16％です。ニュースはコロナウイルスによる差し迫った不況についての見出しでいっぱいでした、ロシアはOpec +取引から撤退しました、そしてそれは原油価格（1日あたり-20％）と明日（2020年3月10日）にヒットしました、MICEX市場も20％下落すると予想され、西側市場における当社の株式の相場。


世界的な不況は私たちを待っていますか？この記事では、Pythonを使用して景気後退の始まりのシグナルを事前に確認する方法を理解しようとします。

この質問に答えるために、債券、株式、およびテクニカル分析の利回りを使用してみます。世界初の最大の株式市場として、米国の金融市場の過去のデータを使用します。実際、米国で景気後退が始まると、それは世界中で始まります（2008年に起こったように）。また、米国市場には数十年分のデータがあり、かなりの歴史的期間にわたって分析できるので便利です。Fedサイトから、yfinance

ライブラリを使用してYahoo Financeから履歴データを取得します。fredapiライブラリを使用し、pandas_datareaderを介したさまざまな財務情報を含む Quandlのウェブサイトからの米国留保。FedおよびQuandlの場合、APIキーを受け取るには登録する必要があります（これは無料です）。

米国のすべての金融危機は、チャートの灰色の領域に直接マークされます。

リスクプレミアム


リスクプレミアムは、リスクの増加を想定することで投資家が受け取る追加の収益性を反映する指標です。

これが金融市場の仕組みです。投資収益率が高いほど、リスクも高くなります。

リスクプレミアムを計算するためには、リスクのある資産の期待リターンからリスクフリーリターン率を差し引く必要があります。

米国市場の場合、リスクのない収益率は米国連邦準備制度率（当社の主要金利の類似物）-FEDです。

株式と債券のリスクプレミアムは異なる方法で考慮されます。
債券の場合：債券利回り（利回り）-FED。
株式の場合：利益/価格（E / P）インジケーターからFEDを引いたもの
。利益はその年の会社の利益であり、価格はインジケーターが計算された時点の株価です。

配当は会社が株主に支払う利益の一部にすぎないため、1株の収益性を計算するために、配当金を収益性としてではなく、会社が受け取った利益として扱います。株式の購入を通じて会社の所有者になることは、実際には、私たちの最終的な収入は、会社の清算時に完全に株主に分配される利益になります。

S＆P500（SP500_EARNINGS_YIELD_MONTH）、無リスクレート（FEDファンドレート）、および幅広い市場の社債利回り（バーボンドイールド）のE / Pを取得します。

# E/P

ご覧のように、危機前の期間は、株式のリスクプレミアムがマイナスで、債券のリスクプレミアムが低下している領域です。

同時に、債券については、リスクプレミアムの低下が債券市場の危機の兆候であるとは限らない（ボフォ指数は2008年にのみ低下した）が、株式市場でのプレミアムの低下はほとんど常に（1980年代初頭の危機後の期間を除いて）負のプレミアムが長期間保持された）株式の価値の減少につながります。

負のリスクプレミアムは株式にとって何を意味しますか？

株式を購入すると、市場でのリスクフリーレートを下回るリターンが得られます。同時に、株価の変動や潜在的な損失のリスクについて全責任を負います。これは標準的な状況ではありません。投資家はリスクプレミアムがマイナスになることはないことを理解しています。その結果、市場は株式の現在の値が高すぎ、販売が始まっていることに気付きました。幅広い投資家（機関投資家ではなく普通の人々）がパニックに陥り、ポートフォリオを売却して秋を悪化させた後、下落は激化します。したがって、市場の反応は速くて強いです。
市場の下落が以前のレベルのリスクプレミアムに及ばないが、常にはるかに強く発生することは注目に値します。これにより、リスクプレミアムが大幅に増加し、投資家にとっての証券の魅力が再び高まります。

さらに、1つのチャートで株式と債券のリスクプレミアムを見ると、債券のリスクプレミアムは伝統的に株式のリスクプレミアムよりも高くなっていますが、過去5年間でそれらは同期しており、両方が徐々にゼロになる傾向にあります。

この問題におけるテクニカル分析の無力さの例として、RSI指標を取り上げます。
RSI-理論上の相対強度指数は、「買われ過ぎ」および「売られ過ぎ」の市場を示します。
買われ過ぎ市場-これは価格が調整されなければならない状態、つまり株式市場の危機です。

インディケータの説明が記載されたWikiへのリンク

Pythonを使用してこのインディケータを計算できます。計算には、期間-244トレーディングセッション（1暦年）を使用します。

SP500_returns = GSPC_h['Close'].pct_change()
delta = GSPC_h['Close'].diff()
window_length = 500

# Make the positive gains (up) and negative gains (down) Series
up, down = delta.copy(), delta.copy()
up[up < 0] = 0
down[down > 0] = 0

# Calculate the EWMA
roll_up1 = up.ewm(span=window_length).mean()
roll_down1 = down.abs().ewm(span=window_length).mean()

# Calculate the RSI based on EWMA
RS1 = roll_up1 / roll_down1
RSI1 = 100.0 - (100.0 / (1.0 + RS1))

RSIチャートには何が表示されますか？

「買われすぎ」の多くのゾーン。これは市場が多すぎることを示している可能性があり、調整する時期が来ています。しかし、それは上昇し続けました（たとえば、1995年以降の長い期間、指標は「買われすぎ」を示していますが、2001年の危機以前は、通常のゾーンに戻り、「買われすぎ」については信号を送っていません。

言い換えれば、危機を予測するために発振器を使用することは非常に物議を醸している演習です。

危機の前は、市場はスムーズに成長します。下落とは異なり、上向きのボラティリティは高くありません。常に鋭くて迅速です。これは、発振器の下の境界のすぐ近くにあります。その交差点は、ほとんどの場合、危機が来て、落下の底が近くなったときに正確に示されました。シグナルを購入しますか？

調査結果


現時点では、状況は憂慮すべきものに見えます。景気後退の可能性を示す直接的な指標があります。レビューされたものに加えて、鉱工業生産指数もあります（経済危機（金融危機ではない！）の危機をリードしています）。マクロ経済統計を詳しく調べて、国際収支やGDPなどを分析に追加することもできますが、機関投資家はすでにこれを行っています。その反応は、長期債と短期債の利回りスプレッドに反映されています。金融投資がない場合は、リラックスしてショーを視聴する必要があります。そして、もしあるならば-あなたのポジションの可能なヘッジについて考えるか、より良い時までキャッシュに行きなさい。
