digisparkでラップトップスタンドを管理する

私は二晩で作られた非常にシンプルなデバイスについての記事を書くことにしました。それは長年の問題を解決しました。それは私がとても幸せで、今日までそれを使用しています。約3か月間正常に動作しました。実際、これが最後に起こったことです。猫の下で興味のある方、お願いします。



結果を見ました。話を遠くから始めましょう。

私はラップトップ用のスタンドを持っています（私にとっては、それがメインコンピューターとして機能するため、常にオンになっています）。スタンドはシンプルではありません。冷却するために、3つのクーラーを取り付け、スタンドをUSBに接続すると、クーラーが回転します。クーラーが正常に取り付けられている場合、プロセッサーから7度下がります。一般的には、事が必要です。すべては大丈夫ですが、スタンドのクーラーは、まあまあまあまあ、常にフルパワーで回転します。これは、沈黙の中で一定の単調な音がすることを意味します。まるで誰かがスタンドに小さなチェーンソーを持っていたかのようです。これが記事の実際の犯人です：



ある良い夜、小包がアリエクスプレッサーで到着している間、私は静かにスタンドの歌声を「楽しんだ」。そして彼はしっかりと決めました-私にとっては十分です。USBでスタンドを管理することになりました。手はすでにある種のstm32f1をハードウェアusbで固定するために手を差し伸べています。そして、それは私に始まりました-トランジスタをクリックするためのdigispark（同じattiny85、ボード上のみ）安く、すでにソフトウェアUSBのバインディングがあり、このタスクに理想的です。彼は、ピンの1つで制御されるmosfetと、RGB LEDの場所を備えた素早く小さなスカーフを彼に広めました。

コンポーネントが封止される前の最終結果：コンポーネントがはんだ付けされました（はんだ付けする必要があります-USBコネクタ、smdのペア、はいdigispark自体）。一般的には、ボードをコンピュータに接続する際に細いコードを使用した方がいいと思いますが、そうです。最初の写真で見た結果。

コードを書くことは残りました。最初はmakefileと純粋なCを使用したかったのですが、arduino ideの方が速いと判断しました。私はそうしました。唯一のことは、プログラマーが公式の16進ブートローダーにパッチを当てるまで、中国のdigisparkはUSB経由を望まなかったことです。その後、pidコントロールとファンシムを使用してコードを書きたかったのですが、気が変わりました。その結果、digisparkでは、仮想シリアルポートをリッスンするだけのコードがあり、そこに現れた場合はファンがオンになり、ゼロの場合はオフになります。

今、コンピュータの一部がありました。私はLinuxにラップトップを持っているので、bashスクリプトを作成してそれをオートロードに詰め込みました。デバイスのリストでdigisparkを探し、N分ごとにセンサーコマンドの出力を解析し、温度> 80が1を送信した場合、<60が0を送信した場合。

デバイスは3か月間安定して動作しました。人生はもっと楽しくなりました。以下はbashスクリプトコードです。

#!/bin/bash
FLAG=1
while [ true ] 
do
for sysdevpath in $(find /sys/bus/usb/devices/usb*/ -name dev); do
    (
        syspath="${sysdevpath%/dev}"
        devname="$(udevadm info -q name -p $syspath)"
        [[ "$devname" == "bus/"* ]] && continue
        eval "$(udevadm info -q property --export -p $syspath)"
        [[ -z "$ID_SERIAL" ]] && continue
	if [ $ID_SERIAL =  "digistump.com_Digispark_Serial" ] 
	then 
        #echo "/dev/$devname - $ID_SERIAL"
	#echo "$FLAG" >> /dev/$devname
	CORE_0=$(sensors | grep 'Core 0' | awk '{print int($3)}')
	CORE_1=$(sensors | grep 'Core 1' | awk '{print int($3)}')
	#Phys_1=$(sensors | grep 'Physical id 0' | awk '{print int($3)}')
	#TEMP1=$(sensors | grep 'temp1' | awk '{print int($3)}')
	#TEMP2=$(sensors | grep 'temp2' | awk '{print int($3)}')
	#echo "$CORE_0"
	#echo "$CORE_1"
	let "TEMP = (CORE_0 + CORE_1) / 2 "	
	#echo "$TEMP"
	if [ $TEMP -gt 80 ]
	then
	#echo HIGH
	#notify-send "HIGH TEMP, fan on, T="$TEMP
	echo "1" >> /dev/$devname
	echo "1" >> /dev/$devname
	fi
	if [ $TEMP -lt 60 ]
	then
	#echo LOW
	echo "0" >> /dev/$devname
	echo "0" >> /dev/$devname
	fi
	sleep 1
	fi
    )
done
sleep 4
done
