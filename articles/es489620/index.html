9 consejos sobre la terminal de Windows por Scott Hanselman

Hola Habr! Es posible que haya escuchado que la nueva Terminal de Windows llegará pronto. Ya escribimos sobre esto aquí . Nuestro colega Scott Hanselman preparó algunos consejos sobre cómo trabajar con la nueva terminal. ¡Únete ahora!



Entonces, descargaste Windows Terminal y ... ¿y ahora qué?


Al principio puede que no te emocione. Esta sigue siendo la Terminal, y no te llevará, sosteniendo la mano.

1) Lea la documentación del usuario para Windows Terminal

2) La configuración se expresa en formato JSON . Le irá mejor si su editor de archivos JSON es algo así como Visual Studio Code y es compatible con JSON e intellisense.

  • ¡Comprueba tu configuración predeterminada! Para mayor claridad, presento mi profile.json (que de ninguna manera es ideal). Instalé requestTheme, alwaysShowTabs y defaultProfile.

3) Decide los atajos de teclado. Terminal de Windows tiene amplias opciones de personalización .

  • Cualquier tecla que presione puede reasignarse.

4) ¿El diseño se ajusta a tus deseos?

  • Hay varios terminales temas en terminalsplash.com.
  • Y aquí hay docenas de esquemas de color iTerm2 , sus esquemas de color se pueden ver claramente aquí .

5) ¿Quieres pasar al siguiente nivel? Explora imágenes de fondo.

  • Puede configurar imágenes de fondo o incluso GIF. Más detalles aquí .

6) Especifique su directorio de inicio.
  • Si usa WSL, tarde o temprano querrá que su directorio de inicio esté ubicado en el sistema de archivos de Linux .

7) Todavía puede usar Far, GitBash, Cygwin o cmder si lo desea. Detalles en la documentación .

8) Examine los argumentos de la línea de comando de la Terminal de Windows.

  • Puede saber que puede iniciar Windows Terminal usando "wt.exe", ¡pero ahora puede usar argumentos de línea de comandos! Aquí hay unos ejemplos:

    wt ; split-pane -p "Windows PowerShell" ; split-pane -H wsl.exe
wt -d .
wt -d c:\github

    En este punto, puede ir tan lejos como lo desee. Haga otros íconos, fíjelos a la barra de tareas, salga al máximo. Además, consulte los subcomandos como nueva pestaña, panel dividido y pestaña de enfoque.

9) Grabé un video que demuestra a una persona que está acostumbrada a Mac y Linux cómo configurar Windows Terminal junto con WSL (Windows Subsystem for Linux), tal vez sea interesante para usted.

¡Comparta a continuación sus consejos, perfiles y temas favoritos para la terminal!
