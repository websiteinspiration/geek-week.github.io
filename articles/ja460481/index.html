ファンレスパフォーマンスコンピューターMIC-7000


MIC-7000 シリーズのコンピューターはコンパクトでありながら、完全にパッシブな冷却機能を備えた高性能の産業用コンピューターであり、第8世代の本格的なIntelプロセッサーと最新のチップセットを搭載しています。MIC-7000コンピュータの際立った特徴は、ケースの上部全体を占有し、換気グリルやファンを使用せずに放熱を提供する大規模なラジエーターです。

リソースを多用するさまざまなタスクを解決するために、コンピューターの構成を拡張できます。独自のi-Module拡張モジュールを使用すると、機械学習システムを構築してニューラルネットワークを起動するために、フルサイズのGPUを含む任意のPCI-e x16ボードをインストールできます。

MIC-7000 無人車両で中央データ処理サーバーとして使用されると同時に、ビデオキャプチャカードを使用して高解像度ビデオを処理し、ニューラルネットワーク計算を実行し、I / Oポートと産業用インターフェイスを使用して周辺機器を制御します。


ファンレス冷却システム



産業用コンピューターは、極端な温度、汚染、ほこりなどの過酷な環境でよく使用されます。このような状況でのアクティブな空冷の従来のシステムは、すぐに目詰まりして故障する可能性があり、換気口が破片や大きな粒子の侵入の原因になります。MIC-7000シリーズのコンピューターでは、完全にパッシブな冷却システムが使用されており、クーラーの故障による機器の故障を排除しています。そのようなシステムは、メンテナンスを必要とする可能性がはるかに低く、完全に静かです。

産業用インターフェイスとI / Oポート


クラシックな産業用プロトコルとの互換性のために、MIC-7000コンピュータには、組み込みの産業用RS-232 / 422/485インターフェイスと汎用デジタルI / Oポートがあります。記事「UNO-1000 / 2000シリーズのモジュラーエンベデッドコンピューター」で説明したiDoorモジュールもサポートされており、PROFIBUS、EtherCAT、PROFINETなどの独自プロトコルのサポートを追加できます。


Flex I / O拡張モジュール


モジュールを使用すると、フロントパネルにHDMI、DVI、RS-232、LANなどのコネクタを追加できます。マザーボードに接続されたパッシブ延長コードを使用する。
汎用I / Oモジュール（GPIO）AIIS-DIO32-00A1Eおよび98R1750090Eが利用可能です。Flex I / O拡張モジュールテーブル（クリック可能）



トラステッドプラットフォームモジュール（TPM）



TPMモジュールが統合されていないマザーボードの場合、リモートTPMモジュールPCA-TPM-00B1Eがあり、この機能をほとんどのMIC-7000コンピュータに統合できます。このモジュールは、ハードウェア保護の要件が高まっているシステム、OSのブートローダーの確認、秘密鍵の保存を目的としています。

CFastカード


MIC-7700担体は、カードをCFAST高速、ホットスワップ可能なSATAインターフェイスと。このタイプのカードは、最近のカメラBlackmagick、Canon、Arriで使用されており、SSDドライブのコンパクトな代替品として使用できます。


独自のiモジュール拡張モジュール


産業用システムでは、PCI-eカードの形式の高度に専門化されたハードウェアが必要になることがよくあります。 i-Module拡張モジュールは、MIC-7000シリーズコンピューターに接続して単一のシステムユニットを形成する特殊なケースの外部PCI-eバス拡張です。

1.2.4 PCI-e x16 / x8 / x4ポートの形式のモジュールと、工作機械や自動化システムにある古いボードを接続するための通常のPCIがあります。生産性の高いグラフィックカードなど、消費電力が増加したボードを接続するために、MIC-75G20などの追加の電力線を備えたモジュールがあります。
モジュールには、追加のHDD / SSDディスクをインストールするためのトレイもある場合があります。



コンピューター付きIモジュールMIC-75G20拡張モジュールMIC-7700は、総電力が最大600WのPCI-eカードをサポートします。2つのドライブトレイとオプションの電源

ビデオ：i-Moduleのインストール


i-Moduleのインストール手順は、ボードを通常のATXケースにインストールするよりも複雑ではありません。これを行うには、MIC-770コンピュータの下部カバーを取り外し、拡張ボードにドッキングします。

事例：ブレーキディスク品質試験システム


MIC-7500 コンピュータとMIC-75M20モジュールに基づいて、自動車用ブレーキディスクの品質をテストするためのシリアルステーションが発売されました。ディスクは回転軸に取り付けられており、ねじれがなく、回転速度に関するデータはPCIE-1802 ADCカードによって制御されます。蓄音機の針のような特別なハンマーが、16のアナログ入力、1MS / sのサンプリング周波数、16ビットの分解能を持つPCIE-1816カードを使用して、ディスク表面の最小の振動を読み取ります。収集されたデータは分析システムに転送され、ディスクの品質に関するレポートが生成されます。


ケース：ドローンコントローラー


タスクは、自動運転システム用のさまざまなセンサーから統合データ処理システムを構築することでした。最小限の遅延でビデオストリームをリアルタイムで処理するために、DVP-7640Eキャプチャボードを介してSDI経由で接続されたカメラが使用されました。カメラからのビデオは、NVIDIA Tesla P4グラフィックプロセッサが使用される加速のために、「マシンビジョン」システムによって処理されます。

MIC-7700Qコンピュータに基づく無人車両システムのコンポーネントの概略図

エンジンコントローラー（ECU）、各種センサー、ドライバーアシスタンスシステム（ADAS）などのカーエレクトロニクスシステムには、CANバスを介してアクセスできます。コンピューターをCANバスに接続するには、iDoorモジュールPCM-26D2CAを使用します。

システムの主なコンポーネント：

  • MIC-7700Q-第7世代Intelプロセッサー、Q170 / H110チップセットを搭載したファンレスコンピューター
  • MIC-75S20-1x PCIe x16、1x PCIe x4、2x HDD / SSDスロットを備えたi-Module拡張モジュール
  • PCM-26D2CA -CANバスインターフェイスを備えた iDoor拡張モジュール
  • DVP-7640E -H.264ハードウェアコーデックを備えた8つのSDIチャネル用のビデオキャプチャカード
  • TESLA-P4 -NVIDIA Tesla P4アクセラレータ


結論


MIC-7000は、アプリケーションの広い範囲のためのコンパクトなソリューションです。パッシブクーリングと幅広い動作温度により、悪条件で使用できます。i-Module、iDoor、Flex I / O拡張モジュールを使用すると、さまざまなタスクのオプションを構成できます。これは、拡張の可能性が制限されることが多いほとんどの組み込みシステムと比較して、はるかに柔軟なソリューションです。
