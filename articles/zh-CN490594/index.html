自由软件基金会宣布了计划创建新的代码托管，以支持协作开发工具并满足以前开发的免费软件托管的道德标准。除了现有的Savannah托管服务外，还将创建新平台，并将继续提供支持。创建新平台的目的是解决用于开发开源软件的基础结构的问题。现在，许多免费项目依赖于不发布其代码并强制使用专有软件的平台。该平台计划在2020年投入使用，它是在已经创建的免费解决方案的基础上进行构建的，该解决方案用于处理独立社区开发的，与单个公司的利益无关的代码。最有可能的竞标者被认为是Pagure平台，由Fedora Linux Developers开发。

Manjaro Linux 19.0发行版




根据OpenNET，已经发布了Manjaro Linux 19.0发行版的GNU / Linux版本，该版本建立在Arch Linux的基础上，但针对初学者。 Manjaro具有更简单的图形安装程序，支持自动硬件检测和驱动程序安装。分发以带有图形环境KDE，GNOME和Xfce的实时程序集的形式出现。 Manjaro使用自己的Boxit工具箱（以Git的形象设计）来管理存储库。除了其自己的存储库之外，还支持使用AUR存储库（Arch用户存储库）。在19.0版中，引入了Linux 5.4内核，Xfce 4.14（具有Matcha的新主题），GNOME 3.34，KDE Plasma 5.17，KDE Apps 19.12.2的更新版本。 GNOME提供了具有不同主题的桌面模式开关。最高版本9。3 Pamac软件包管理器已经更新，默认情况下包括对snap和flatpak格式的自包含软件包的支持，可以通过新的Bauh应用程序管理界面进行安装。

Monado的第一版，虚拟现实设备平台




OpenNET宣布了Monado项目的第一个版本，该项目旨在创建OpenXR标准的开放实现。OpenXR是一种开放，免许可的标准，用于访问用于虚拟现实和增强现实的平台和设备。项目代码是用C编写的，并根据与GPL兼容的免费Boost软件许可1.0进行分发。Monado提供了一个完全符合OpenXR要求的运行时环境，可用于在智能手机，平板电脑，PC和其他设备上使用虚拟现实和增强现实。在Monado的框架内，开发了几个基本子系统：

  1. 空间视觉引擎；
  2. 字符跟踪引擎；
  3. 复合服务器；
  4. 互动引擎；
  5. 工具。

Open Cybersecurity Alliance




ZDNet宣布OpenDXL本体的问世，该框架旨在在网络安全相关程序之间交换数据和命令。旨在弥合网络安全工具之间的分歧的新框架被引入开源社区。 OpenDXL本体是由开放网络安全联盟（OCA）开发的，开放网络安全联盟是由IBM，Crowdstrike和McAfee等网络安全提供商组成的联盟。 OCA表示，OpenDXL本体是“用于通过通用消息传递系统连接网络安全工具的第一种开源语言”。 OpenDXL本体旨在在工具和网络安全系统之间创建一种通用语言，从而消除了产品之间自定义集成的需求，而这些自定义集成在彼此交互时可能会非常有效，最终系统，防火墙等之间的冲突，但存在碎片化和特定于供应商的体系结构的问题。



Brave archive.org




据OpenNET称，自1996年以来一直保存许多站点档案的Archive.org（Internet存档Wayback Machine）项目谈到了与Brave Web浏览器开发人员的一项联合倡议，如果站点可用性出现任何问题，该倡议可以增加Internet访问。如果尝试在Brave中打开不存在或不可访问的页面，浏览器将在archive.org中检查该页面是否存在，如果找到该页面，则显示打开该归档副本的建议。在Brave Browser 1.4.95版本中实现了此功能。 Safari，Chrome和Firefox的插件具有类似的功能。 Brave浏览器的开发由JavaScript语言的创建者，前Mozilla负责人Brendan Ike领导。该浏览器基于Chromium引擎，它专注于确保用户的隐私和安全，并根据免费的MPLv2许可证进行分发。
