Les joueurs de Reddit PC mettent en commun leur puissance de calcul autour du projet Folding @ Home pour lutter contre COVID-19

Les modérateurs du subreddit de course PC Master ont exhorté les propriétaires de puissantes cartes vidéo de jeu à rejoindre le cloud computing scientifique du projet Stanford Folding @ Home , qui est maintenant engagé dans le calcul de modèles dans huit directions à la fois pour lutter contre COVID-19. L'initiative de la communauté Reddit était déjà soutenue par Nvidia, qui a appelé Twitter pour fournir aux scientifiques la puissance de calcul de leurs cartes vidéo:


Il est important de noter que le subreddit de course PC Master n'essaie pas de faire du battage médiatique sur le sujet du coronavirus : le support Folding @ Home et le premier méga-thread systématique dédié au projet ont été publiés il y a 4 mois, alors qu'ils n'avaient même pas entendu parler du coronavirus. De plus, les utilisateurs ont mentionné F @ H dans un sous-programme au cours des cinq dernières années.

Comment participer aux calculs collectifs, même si vous ne vous souciez pas de COVID-19 et que vous souhaitez trouver rapidement un remède contre le cancer - sous le chat.

Afin de participer aux calculs et de soutenir les scientifiques avec leurs propres capacités, vous avez trois options.

Premièrement: rejoignez le groupe informatique reddit de la course subreddit PC Master.

Deuxièmement: entrez le code spécifique de l'étude d'intérêt et comptez-le vous-même, dans la direction choisie.

Une option pour ceux qui ne se soucient pas de COVID-19: choisissez parmi les quatre domaines proposés (cancer, alzheimer, etc.) et comptez là.

Pour faire partie du groupe informatique, vous devrez installer un logiciel spécial F @ H sur votre PC. Une liste complète des installateurs peut être trouvée ici . Désormais prêt à l'emploi, Windows 10, MacOS, Ubuntu, CentOS, Fedora, RedHat, Debian et Mint sont pris en charge. A lire également cet article.et le fil mentionné précédemment sur Reddit lui-même . Pour les intéressés, un référentiel sur GitHub .

Pour ne pas perdre de temps: pour vous connecter aux calculs, vous devez spécifier le code du projet (groupe de décompte) dans les paramètres logiciels F @ H. Les développeurs suggèrent d'utiliser une interface Web, mais elle peut être appelée stable avec beaucoup de difficulté. Dans la configuration du client, dans le champ "Numéro d'équipe", entrez le code du groupe Reddit, qui prend en compte tout dans une rangée, ou un calcul COVID spécifique.

Pour les postes de travail puissants avec la dernière génération de cartes graphiques ou avec des configurations complètes sur le GPU, il est recommandé d'obtenir une clé personnelle en contactant Stanford. Cela optimisera la charge et les calculs.

Important! Au départ, il peut vous sembler que FAHControl ne montre aucun signe de vie et que tout est tombé en panne sur les réglages automatiques. Si oui, alors vous avez probablement rencontré les mêmes problèmes que la rédaction du blog: très longtemps pour obtenir le poste. Après plusieurs sessions et connexions, le logiciel devrait enfin recevoir les données du calcul et votre carte vidéo fera le travail.
