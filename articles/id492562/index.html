Tiga webinar Apache Ignite yang berguna dalam program karantina Anda

Setidaknya sampai akhir April, kami tidak mungkin melihat satu sama lain secara offline: konferensi dan acara lainnya telah dibatalkan atau online. Saatnya menonton video berguna yang Anda tunda nanti (lihat apa yang baru di saluran komunitas Apache Ignite ). Jika Anda ingin lebih banyak interaktivitas, bergabunglah dengan webinar tentang komputasi terdistribusi, Ignite SQL, dan pengaturan keamanan.

18 Maret - Bagaimana memahami Ignite SQL yang tepat untuk Anda dan mulai menggunakannya


Jika Anda dihadapkan dengan batasan pada ukuran, skalabilitas, atau kinerja database relasional Anda, mungkin sudah saatnya untuk meningkatkan ke sistem terdistribusi. Apache Ignite adalah platform terdistribusi yang dapat berfungsi sebagai database, menyediakan API SQL dan JCache untuk bekerja dengan data. Di webinar ini kita akan melihat contoh kehidupan nyata dan mendiskusikan pro dan kontra dari setiap pendekatan:

  • Fitur SQL di Ignite
  • Bagaimana SQL berinteraksi dengan Java API (memuat melalui JCache API via SQL dan sebaliknya)
  • Jaminan konsistensi (Ignite SQL dan JCache API)
  • Kompleksitas SQL terdistribusi dan bagaimana colokasi data dapat menguranginya.

Pendaftaran untuk webinar "Bagaimana memahami apa yang tepat Ignite SQL untuk Anda dan mulai menggunakannya"

25 Maret - Komputasi Terdistribusi dengan Apache Ignite


Untuk mengambil keuntungan dari potensi penuh dari platform In-Memory, seringkali tidak cukup untuk memuat data ke dalam cluster dan mulai bekerja dengannya menggunakan kunci-nilai atau SQL API. Anda juga harus secara efisien menempatkan data dalam sebuah cluster dan menggunakan komputasi terdistribusi yang meminimalkan pergerakan data di seluruh jaringan. Di webinar ini, Anda akan belajar bagaimana merancang dan melakukan perhitungan dengan memperhitungkan semua pro dan kontra dari sistem terdistribusi.

Kami akan mempertimbangkan:

  • Bagaimana mendistribusikan informasi secara efisien dalam sebuah cluster menggunakan pemodelan dan kolokasi data yang tepat;
  • Bagaimana MapReduce API diimplementasikan dan berfungsi di Apache Ignite;
  • Cara meminimalkan penundaan operasi menggunakan colocation data dengan perhitungan;
  • Bagaimana query SQL dapat dieksekusi secara efisien dalam lingkungan terdistribusi.

Webinar akan bermanfaat bagi mereka yang mulai berkenalan dengan Apache Ignite dan ingin belajar bagaimana mendesain secara optimal dan menggunakan kemampuan komputasi terdistribusi pada platform ini.

Daftar untuk Komputer Terdistribusi dengan Apache Ignite Webinar

14 April - Panduan Pengaturan Keamanan untuk Apache Ignite dan GridGain dengan Contoh


Apache Ignite dan GridGain memungkinkan komputasi cepat dan kueri data terdistribusi yang sangat efisien. Baik Ignite dan GridGain menyediakan konfigurasi yang fleksibel yang akan membantu membuat operasi cluster lebih aman. Di webinar ini, kami akan membahas topik keamanan berikut:

  • Koneksi aman antar node (SSL / TLS);
  • Otentikasi Pengguna
  • Otorisasi pengguna.

Dengan menggunakan contoh, kami akan mempertimbangkan konfigurasi untuk:

  • SSL / TLS untuk membuat koneksi yang aman antara semua node, thin client dan koneksi menggunakan REST;
  • Otentikasi melalui Apache Ignite Advanced Security;
  • Plugin Keamanan GridGain.

Webinar ini akan membantu para pengembang dan pengembang khusus memahami dasar-dasar keamanan untuk Apache Ignite dan GridGain, dan memberikan contoh konfigurasi untuk kasus yang paling umum.

Daftarkan untuk Webinar "Panduan Pengaturan Keamanan untuk Apache Ignite dan GridGain dengan Contoh"

Bergabunglah dengan webinar, tawarkan topik Anda sendiri (tidak perlu bekerja di GridGain untuk berbagi pengalaman dengan komunitas) dan berhati-hatilah.
