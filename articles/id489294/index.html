Mempelajari Variabel CSS dengan Contoh Praktis



Selamat siang teman!

Sekali surfing web membawa saya ke kode yang luar biasa ini .

"Kebesaran" itu (yaitu, kode) adalah penggunaan praktis variabel dalam CSS, yang sangat memudahkan penulisan gaya (penggunaan variabel juga menyebabkan struktur CSS yang tidak biasa). Dan, tentu saja, topiknya sangat mengesankan - planet-planet tata surya.

Proyek asli ditulis dalam Pug dan Sass, tetapi kami akan menganalisisnya dalam HTML / CSS.

Jadi ayo pergi.

Kami melihat yang berikut:

  • Pembungkus untuk planet ini dan informasi tentangnya - .card
  • Mengikat pembungkus ke planet tertentu - .mercury
  • Wadah untuk planet - .planet
  • Di dalamnya ada wadah untuk atmosfer - .atmosphere
  • Di dalamnya adalah permukaan planet -. Permukaan
  • Wadah Informasi Planet - .info
  • Nama planet yang sesuai adalah .title
  • Wadah Informasi Tambahan - .form
  • Unit Informasi Kontainer - .item
  • Secara total, kami memiliki tiga unit informasi:
    1. Yang pertama berisi informasi tentang kemiringan sumbu planet dan gambar panah
    2. Dalam yang kedua - informasi tentang panjang hari dalam jam
    3. Di ketiga - sekitar panjang tahun dalam beberapa hari

Mari beralih ke CSS.

Sintaks berikut digunakan untuk mendeklarasikan variabel dalam CSS:

:root {
  ---: ;
}

Baca lebih lanjut tentang: root pseudo-class di sini .

Variabel untuk planet ini dinyatakan sebagai berikut:

:root {
    --mercury-image: url(https://thebestcode.ru/media/solarSystem/mercury.jpg);
    --mercury-tilt: rotate(0.034deg);
    --mercury-day: 1407.6;
    --mercury-color: #999;
}

Seperti yang Anda lihat, variabel dideklarasikan untuk gambar, sudut kemiringan, durasi hari dan warna planet.

Untuk memutar planet-planet, animasi berikut digunakan:

@keyframes planetRotate {
    0% {
        background-position: 0% center;
    }

    100% {
        background-position: -200% center;
    }
}

Variabel yang digunakan dalam CSS adalah:

var(---);

Inilah cara variabel digunakan untuk membuat gaya planet:

.mercury .atmosphere {
    box-shadow: inset 10px 0px 12px -2px rgba(255, 255, 255, 0.2), inset -70px 0px 50px 0px black, -5px 0px 10px -4px var(--mercury-color);
}

.mercury .surface {
    background-image: var(--mercury-image);
    transform: var(--mercury-tilt) scale(1.2);
    animation: planetRotate calc(var(--mercury-day)*.1s) linear infinite;
}

.mercury .planet::before {
    transform: var(--mercury-tilt) scale(1.2);
    border-color: var(--mercury-color);
    color: var(--mercury-color);
}

.mercury .icon {
    transform: var(--mercury-tilt);
    color: var(--mercury-color);
}

Perhatikan bagaimana variabel untuk panjang hari (--mercure-day) digunakan untuk menentukan durasi animasi (kecepatan rotasi planet).
