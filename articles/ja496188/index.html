私たちは皆、この物語がどのように終わるか知っています。彼女のエンディングが悲しいことを理解しているなら、この記事はあなたのためです。

ビデオの目的は、情報を伝え、教え、楽しませ、刺激することです。しかし、ビデオから何も取らなかったり、覚えていなかったり、役立つヒントを使用したり、冗談を理解したりせず、刺激を受けて演技を始めたりしないと、意味が失われます。

そしてこれはすべて、酔っ払ってNetflixでテレビ番組を見たり、YouTubeで動画を見たりした結果です。一見すると、これは先延ばしの単純な問題のように見えますが、悪の根源ははるかに深いです。はい、私たちはビデオを通じて退屈を克服しようとしています。しかし、なぜこれがそんなに欲しいのか見てみましょう。

今日、私たちは子供の頃から私たちと一緒にいた理由を特定します。次に、知識を使用してビデオの消費を制限し、プロセスに意識的な目標が表示されるようにします。

途中です。

私たちはみな無実に生まれました...


私は1994年に最初の映画「ライオンキング」を観ました。私は3歳でした。以前は動画を見たことがありますが、ほとんどの家族と同じように、両親は私をスクリーンだけでなくすべての電気から厳しく制限しました。私たちの脳、特に子供たちはビデオの力に逆らう機会がないので、それは正しかった。

ビデオは、空間の視覚、聴覚、感覚に等しく影響します。また、モーションセンサーを起動し、感情を呼び起こします。ビデオは脳の右半球全体の活動を引き起こします。 TVのクリエイターは、このことを最初から知っていました。 Malcolm Gladwellが沸騰ポイントで示したように、セサミストリートの作成者「子供の注意を引くために「注意散漫者」と呼ばれるツールを使用しました。

「各エピソードについて、7秒半、つまり約400の参照ポイントに関する情報がありました。これらすべてのポイントを1つの赤い線で結んだ場合、ウォールストリートとの為替レートのようになります。突然下がったり、徐々に下がったりすることがあり、何が問題かを理解するのをやめました。関心のあるグラフが表の上枠に近づく場合もありましたが、実際に子供たちの注意を引くセグメントに注目しました。」

それは60年代でした。ビデオプロデューサーが現在注目を集めている機会を想像してみてください。セサミストリートの目標は非常に高貴なものでしたが、読み、書き、数えることを子供たちに教える   ことでした。ショーのプレミア公開の1か月半前、クリエイターは、架空の要素と現実の要素がフレームに隣接していると、子供たちが気に入らないことに気付きました。心理学者は人形と生きている人々を混ぜることは不可能であると信じていました：これは子供たちを動揺させますが、彼らは一部の大人を見ることに興味がないので、ショーの配給業者はこのアドバイスを無視しました。

「私たちは、幻想と現実が混在する危険性を認識している発達心理学の専門家に手紙を書くことにしました。しかし、とにかくそれを行います。そうでなければ、私たちはカーディクです。」


人と人形が混ざらない？今、それは逆です

私たちの頭をハッキングすることは、ビデオコンテンツの作成者にとって大きな課題でした。彼らの生存は彼にかかっていた。

これは内部システムの欠陥でした。同様ツールドフランスでドープ：あなたはそれを受け入れていない、またはすべてのあなたかもしれないと、それを公開しようとしているのどちらか。セサミストリートの作成者は、私たちの脳のだましやすい部分を悪用して善意に導かれましたか？もちろん！しかし、それにもかかわらず、それは搾取でした。

その結果、私たちの無邪気さはすべてドレインになりました。

...しかし、人生は独自の調整を行いました


テレビを見る時間が限られていることは、成長する途中で「最もひどい」ように見えました。しかし、正直なところ、私はこれについてあまり心配していませんでした。それはほとんどの場合、テレビで面白いことが起こっていなかったからです。私たちの時代には、空気を捉えるというコンセプトがあったことは幸運でしたが、今では毎分パンドラの箱がどんどん開かれています。James Bridge はYouTubeの現象を次のように説明しています。

「子供向けのコンテンツは、3Dアニメーターのお金を稼ぐための可能な形態の1つです。早い段階で、私たちの美的基準は低くなります。したがって、独立した生産はそのサイズを犠牲にして利益を上げることができます。サイトのアルゴリズムは子供たちのように品質を気にしないため、既存の簡単にアクセスできるコンテンツを使用し、同じことを無限に無意味に繰り返して処理できます。」

テレビチャンネルや映画スタジオなどのゲートキーパーのおかげで、私たちは貧弱なコンテンツから保護されましたが、テレビを見ることの習慣さえ身につけていました。私が夢中になった最初のショーを覚えています。それは主にアニメでした：ポケモン、ゲームの王、およびデジモン。初めに間に合うようにバス停から走らなければならないことがよくありました。


これは、テレビを見ての私の習慣が始まった場所。そして、どこから始めましたか？The Hook

のNeil Ellのモデルを使用すると、10代の若者がテレビに夢中になっている理由が簡単にわかります。

  1. 私たちは強い好奇心に駆り立てられ、要件は低く、最も単純なストーリーでさえ瞬間的なトリガーとして機能しました。
  2. 必要なアクションはリモコンのボタンを押すことだけでした。これは、両親が私たちを家に置いてきたときに非常に簡単でした。
  3. 報酬は、ドーパミンの海をもたらした満足度の高い好奇心でした。
  4. 「投資」としては、お気に入りのキャラクターと一緒に過ごした時間でした。これにより、次のエピソードの好奇心がさらに高まり、依存症の輪が閉じました。

多くの人が独立してテレビを見る習慣を身につけてきたことが今や明らかであるようです。テレビを絶え間なく消費することには利点があるにもかかわらず、何人かは止めることができました。

良い


教育的価値に 加えて、子供だけでなく、テレビを見ることでストレスを和らげることができます。落ち着いて疲れた目で隠れて帰れる避難所です。

研究によると、「架空の世界と架空の世界」で時間を過ごすことは、社会活動の短期的な代替となり得ることが示されています。

「残りを置き換える可能性を分析したところ、疲れた人々が社会的代理を探していることがわかりました。[結果]は、自己規律の練習を使い尽くした後、人々が（たとえば、お気に入りのシリーズの）おなじみの架空の世界に注意を向けることを喜んでいることを示しています。追加の分析は、彼らの社会的性質により、これらの身近な架空の世界が休息を助けることを示しています。

あなたやあなたの友人が同じシリーズを視聴している場合は、サロゲートの世界を離れてオフラインになるときに、そのシリーズについて話し合うことができます。あなたの好きなキャラクターは、実際の生活の中で重要だが刺激的な決定にあなたを鼓舞することができます。このシリーズの最後の記事を完了した後、金曜日の夜にTor 3：Ragnarok

を見に行くのは良い報酬でした。私は笑い、別の世界に飛び込み、無限に刺激を受けました。 しかし、2008年3月、すべてがそれほどバラ色になりました。



悪い


私の最初で最も印象的な飲酒のエピソードは、イギリスの友人とインターンしたときでした。私たちのホストファミリーはSkyアカウントを持っていたので、毎晩10日間、シンプソンズ、グリフィンズ、フューチュラマの4時間のマラソンを開催しました。私たちはたくさん笑ったが、疲れて混乱した。

シリアルビンジに最初に入ったのはいつですか？この瞬間を思い出してみてください。あなたはまだ10代でしたか？私たち全員が経験する成長段階があります。これは、酔っ払ってテレビ番組を見る絶好の機会です。それは10代の暴動と呼ばれます。

心理学者カール・ピカールによると：
「暴動は若者が自分の利益に抗議するように導く可能性があります-子供の頃の傾向、活動、態度を否定し、時には自尊心をサポートします。[...]暴動は、若者が自己決定と個人的な意見に依存し、他の人々の欲望と矛盾する何かをするという事実につながります。

10代の若者は反抗しており、9歳から23歳の間のどこかです。この人生のフェーズでは、ほとんどすべてを否定し、長い間何もしないことがよくあります。全世界が私たちに反対していて、何も変化することができないと認識して、私たちは何を求めていますか？正解！架空の世界へ。


Netflixが反撃したとき

これは、私たちの脳に影響を与えるNetflixとYouTubeの危険な魔法が働く場所です。そして、老人、彼らはそれを最大限に楽しみます。

  1. Netflixは系統的にA / Bテストを行い、すべての画面、すべての画像、すべての単語を細部にわたってテストします。
  2. 具体的には、どのエピソードがあなたをシリーズに引き込んだかを追跡します。
  3. 同社は最近、連続飲酒試合への関与を強化する機能を試して、ショーのどこまで進んだかを示しています。
  4. 自動再生は、こことYouTubeの標準機能になりました。

その結果、ミレニアル世代の92％がNetflixにアカウントを持っているか、誰か他の人を使って毎日1.5時間 使っています。10代の若者は、無条件の心理的操作のためにリハビリセンターに行く必要があります。 それはまた、彼らが成長するとき、彼らは止めることができないことを意味します。



悪の


思春期に「はい、そして何もする必要がない」結果としてメディアが過度に消費される場合は、まったく正常です。成人にとって、この振る舞いは受け入れられないようです。仕事がほとんどの時間を占めるとき、私たちはもっと多くの必要な活動をします。もちろん理論的には。

しかし、どういうわけか習慣が持続します。サミュエルジョンソンは次のように述べています。「習慣の連鎖は弱すぎて感じられませんが、強すぎて手放せません。」Emil Steinerは

彼の研究で、テレビ番組を酔っ払って見る6つの理由を特定しました。

  1. 歴史へのより深い没入感からのポジティブな体験の改善。
  2. Zeigarnik効果の影響下で発生する、完了感、開始されたすべての作業を終了したいという願望。
  3. 友人があなたを隠蔽しないように、文化と通信する。
  4. 中断したところに簡単に戻ることができるので便利です。
  5. 新しいシーズンのリリース前に、逃したに追いつくという願望。
  6. 休息と郷愁、簡単な休憩と身近なものの認識。

これらの要因はすべて、連続した飲酒の習慣を強化しますが、最も強いのは最後です。前に調べたのと同じ方法であなたを植えるのは彼です。

「テレビは私の充電方法です」は単なるファサードです。大きな問題との不愉快な出会いから私たちを分離する安価な画面。私たちは自然な好奇心を失い、何も変えることができません。

真実は、私たちがソファーに座って見始めたとき、以前に計画していたことは決して起こらないことに同意したということです。「充電」する必要はありません。私たちは、戦いの開始前に降伏する誰かになったという事実から逃れる必要があります。

酔っ払ってテレビ番組を見ることが、2015年の今年の流行語になりました。それは非常に普及している薬物であり、現在では喫煙よりも社会で許容されています。この研究では、酔っ払ったテレビ番組を「新しい基準」と呼んでいます。Netflixは笑って、顔に黒い鏡をもたらします。

「TVストリーマーの76％は、優れたシリーズのいくつかのエピソードを見ることを、忙しい生活の中で居心地の良い避難所と呼んでいます。」

酔っ払ってテレビ番組を見るのは、本質的に悪いことではないという事実によって、問題はさらに複雑になります。これには、すでに述べたような利点があります。テレビの視聴が私たちのアイデンティティを変える役割を果たしてきたため、私たちにとってそれは手に負えない怪物になりました。私たちは人生でもっと価値があると信じています。しかし、私たちの冒険の日々は過ぎ去りました。したがって、私たちは熱意を少し減らし、ソファから他の人の冒険を見る必要があります。

問題は、Netflixが中毒性があることではありません。そして、私たちの生活がそれを引き起こさないという事実。

私たちがこの感覚で生きている限り、私たちは自分たちの好奇心を2番目のレートの製品で次々にエピソードで消すように強制することを選択します。飽和させる他の方法を探し始めなければ、私たちの人生はそれを見る画面の4インチ、9インチ、42インチを超えることはありません。

敵と向き合うことです。

被害者にならない環境を設計する


Facebookのチェックや音楽を強制的に聴く のとほぼ同じように、ビデオの過剰な消費は、私たちの欠陥を受け入れることができないことの症状です。

「他に何もせずに自分らしくいられる能力を生み出す必要があります。ただ座ることができる...このように...人間であることを意味します」ルイスX.ケイ。

自分の退屈を見る必要がありますが、これには退屈のためのスペースを作る必要があります。ここに役立つ2つのツールがあります。1つ目はNetflix、2つ目はYouTubeです。

1.集中力を保つ


「Stay Focus」はChromeの拡張機能であり、動画の使用を制限します。1日を通して特定のサイトの分数を制限します。


最大2つのシリーズのみ。できました。インストール

後、シリーズの2エピソードなど、視聴する最大時間を設定できます。次に、[ブロックされたサイト]に「netflix.com」と入力し、[追加]をクリックします。 45分間サイトにいると、拡張機能は1日の終わりまでアクセスをブロックします。デフォルトの「アクティブ時間」は終日ですが、「アクティブ日」を使用すると、たとえば日曜日にSlackをブロックするなどの例外を設定できます。 「核オプション」は本当に面白いように見えます。







これにより、最大許容時間に達したときに「完全なシャットダウン」を設定できます。コンピュータ制御システムを使用しない限り、元に戻すことはできません。

これはまさにあなたが必要とするものです。退屈し始めます！ここでは、さらに進めるためのヒントをいくつか紹介します。

  1. iPad、電話、その他の場所でNetflixを使用している場合は、Freedomを使用してください。デバイスからアプリケーションを削除することをお勧めします。
  2. 衛星、ケーブル、または「通常の」テレビの他の形式を使用している場合は、リモコンを非表示にしたり、友人に渡して1週間保管したりできます。
  3. どこでもNetflixアカウントからログアウトします。このナンセンスなハードルは、何かを見始めるという決定について考える十分な時間を与えてくれます。

2. YouTubeのおすすめを削除します


StayFocusdにYouTubeを追加することはできますが、私はそれを研究に頻繁に使用するため、代替案を見つける必要がありました。 YouTubeへのこだわりは同じ場所から始まるため、「推奨事項」セクションの 1つだけを削除するだけで十分です。


YouTubeの始まり...

ビデオを見ると、このセクションは画面の右側に転送されます。ほとんどの言語は左から右に読み取られるため、ウェブを閲覧すると目がその方向に移動します。


...そして決して終わらない。

YouTubeで最初にしたいことは、まだ行っていない場合は、自動再生をオフにします。これは、上のスクリーンショットの右上隅にあります。

次のステップは、Chromeの別の拡張機能をインストールすることです。これにより、YouTubeの推奨事項が完全に削除されます。シンプルなアイコンを拡張バーに追加します。そこから、画面をそのような魅力的なブラックホールではないようにする多くの視覚要素をオフにすることができます。

延期する


大声で言います：「退屈です」再び。大声で。

"私は退屈です"。

あなたは自由の精神を感じますよね？これはストレスを和らげます。行動し、成功しているように見え、大きな使命を持っており、楽しいことを求めるこのすべての圧力。見逃していることを覚えていると、義務から解放されます。時々それが必要なだけです。

「退屈」を受け入れることは重要ですが、これは最初のステップにすぎません。私は感覚が怖いことを知っていますが、それらは正しい質問と答えを導きます。さらに質問を。このようなもの：セサミストリートの最初のエピソードを見る前に、すべてが始まった瞬間に出発した場合はどうなりますか？

私たちが子供だったとき、私たちは退屈をゲームに置き換えました。

チャーリー・ヘムの本、ここから再生：

「ゲームとは、自分が本物になり、自我と恐れを手放し、自分が愚かに見えるようにする状態です。これが私たちの友情を強め、最も貴重な思い出と人生の無限の喜びを与えてくれます。」

価値観の台無しにされたシステムを修正することは可能ですか？かくれんぼをするのと同じくらい簡単です。チャーリーはあなたができることを確信しています：

「これを見るにはもっと時間がかかりますが、ようやく理解しました。私は人生を作品と見なしても、実際には機能しません。私は人生をゲームとして考えています。別に考えれば、すべてを真剣に受け止めすぎる。私の健康と幸福は崩壊しつつあり、私の仕事はひどいです。」


単に。最高です。打ち合わせ、会議。

ゲームは、好奇心に対する私たちの生来のニーズを満たす最も純粋な形です。しかし、私たちはそれをそのように認識することをやめました。おそらくそれが、ガウス博士が大人を「時代遅れの子供」と呼んだ理由です。もちろん、大人の世界は私たちが好奇心を持ち続けるように促しません。
