左連想三項演算子


三項演算子が右から左に処理される他のほとんどの言語とは異なり、PHPでは、これは逆に起こります。この動作は開発者を混乱させることが多いため、左結合性の代わりに括弧を使用することが推奨されました。


次の例を見てみましょう：


return $a == 1 ? 'one'
 	: $a == 2 ? 'two'
 	: $a == 3 ? 'three'
 	: $a == 4 ? 'four'
           	: 'other';

他のほとんどの言語では、次のようになります。


return $a == 1 ? 'one'
 	: ($a == 2 ? 'two'
 	: ($a == 3 ? 'three'
 	: ($a == 4 ? 'four'
           	: 'other')))

PHPでは、これは次のようになります。


return ((($a == 1 ? 'one'
 	: $a == 2) ? 'two'
 	: $a == 3) ? 'three'
 	: $a == 4) ? 'four'
           	: 'other';

多くの場合、これは計画されたものではないため、結果としてエラーが発生します。
