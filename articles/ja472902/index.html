その他のハードウェアプラットフォーム


10月14日に、i.MX6、i.MX7、i.MX 8Mおよびi.MX 8M Miniプロセッサーファミリーを含むNXP i.MXファミリーのプロセッサー向けのWindows 10 IoT Coreボードサポートパッケージの一般提供を発表しました。

NXPは、組み込みアプリケーションを接続するための高セキュリティソリューションのリーダーであり、NXP i.MXとWindows 10 IoT Coreの組み合わせは、商業レベルの安全な接続デバイスを作成するための追加オプションを開発者に提供します。

Windows 10 IoT Coreを使用すると、最適化された消費電力で安全でスマートなデバイスを最大限に活用できます。開発者は、使い慣れた強力なWindows開発環境を使用して新しいスマートデバイスをすばやく作成し、Azure IoTプラットフォームを使用してそれらを商品化できます。 NXPプロセッサーのサポートを追加することにより、IoTプロジェクトの作成者は、多くの新しい産業用マイクロ回路を使用する機会が与えられます。

NXPのビジネス開発担当バイスプレジデントであるセキュアIoTデバイスとクラウド製品のジェフマイルズは、次のように述べています。「現在NXPのお客様は、業界をリードするスマートデバイスソリューションにアクセスでき、IoTプロジェクトを含む産業用アプリケーション向けに、Windows 10 IoT Coreでi.MX 6、i.MX 7、およびi.MX 8Mプロセッサーを使用できます。」スマート製品のためのAzureの力を発見してください。」

継続的なエッジボーダーイノベーション


今年の初めに、Windows 10 IoT EnterpriseとWindows Server IoT 2019にAzure IoT Edgeのサポートを含めました。これにより、IoT Edgeランタイムを簡単に統合できるようになり、機械学習アルゴリズムやその他の同様に高度なコンピューティング機能をクラウドから周辺機器に転送できるようになります。Windows for IoTは、Windows Machine LearningとWindows Vision Skillsもサポートしているため、Azureで開発された高度なAIアルゴリズムを任意のWindows IoTデバイスで実行できます。

業界最先端のエッジラーニング機械学習ツール：新しいWindows MLコンテナー


Insider Programのメンバーがすでに利用できる新しいWindows MLコンテナは、Windows ML APIを使用して作成されたインテリジェントなワークロードを必要とするタスクのために特別に設計されています。 Windows MLコンテナーは、エンタープライズクラスのIoTソリューションを作成するための高速で柔軟なプラットフォームを企業に提供します。高度なWindows機能とWindows 10 IoTプラットフォームのセキュリティおよびAzure IoT Edge管理機能を組み合わせています。

Windows MLコンテナーは、ハードウェアアクセラレーションによるインテリジェントなワークロードと、ホスト周辺機器への直接アクセスを提供します。ホストされたワークロードは、USB、I2C、SPI、またはGPIOを介して接続されたセンサーまたはカメラを使用できます。ハードウェアアクセラレーションには、Microsoft Compute Driver Modelと互換性のある任意のDirectX 12 GPUまたはホスト統合ASICを使用できます。

特にAIワークロード向けのコンテナー最適化のおかげで、ディスク上のコンテナーサイズは約350 MBであり、GPU処理をサポートする市場の他のコンテナーよりも大幅に小さくなっています。サイズが小さいほど、Windows MLコンテナの管理が容易になり、コストが削減されます。Windows MLコンテナのプレビューバージョンについて詳しくは、  こちらをご覧ください。

SQL Server IoT 2019の紹介：IoT向けのWindowsデータストレージと分析


SQL Serverは組み込みデバイスで長年使用可能であり、組み込みデバイスの開発者に最新のSQL Server 2019機能を紹介できることを嬉しく思います。 SQL Server 2019のバイナリに相当するSQL Server IoT 2019の紹介。これは、OEMチャネルを通じてライセンスされ、アプリケーションを備えたエッジデバイスで実行されている専用サーバーで使用するために特別に設計されます。 Microsoft SQL Server IoT 2019は、Windows Server IoT 2019と組み合わせることで、比類のないスケーラビリティ、業界をリードするパフォーマンス、セキュリティ、およびあらゆるデータをインテリジェントに処理する機能を備えた、ミッションクリティカルなデータ分析のための包括的なデータベースプラットフォームを提供します。 SQL Server IoTは、インメモリテクノロジーを使用して、運用データのリアルタイム分析を可能にします。読み取り専用メモリにより、より高い並列性とスケーリングを提供します。 Microsoft SQL Server IoT 2019は今年後半に利用可能になります。
