我给你带来了grafonia！神经网络如何将旧游戏的分辨率提高到高清





本文是我在Medium上的文章的免费翻译。



我小的时候就喜欢在电脑上玩。我发现卡带ZS Spectrum上的一个小游戏很小，但是90年代的多彩发现才是真正的发现。然后出现了大多数现有类型。经过一番怀旧之后，我决定回忆起我的青春，并在Dosbox模拟器上运行其中的一款旧游戏，但由于像素大和分辨率低而受到不愉快的打击。尽管大像素的旧图形可能具有自己的魅力，但现在许多人对此质量不满意。



为了提高分辨率并消除游戏中的棱角，目前使用了各种后处理和抗锯齿算法（有关更多详细信息，请参见此处zen.yandex.ru/media/id/5c993c6021b68f00b3fe919c/kak-rabotaet-sglaj​​ivanie-v-kompiuternyh-igrah-5c9b3e76d82a083cc9a0f1a7），但抗锯齿算法会导致所有讨厌的像素（通常更少）。



同时，提高图形质量对于游戏玩家而言通常至关重要。Ubisoft在2014年花费了大约六个月的时间重新绘制了Heroes 3 HD游戏的纹理，并引起了对该游戏的兴趣激增。最近的新闻-重新发布了首款高清图形CNC。详细地，您可以看到人们对在Google趋势上以高清图形重新发布的兴趣日益浓厚。



并非对于每款游戏来说，高质量的打扰和重新发布都是有意义的-重新绘制纹理是一项昂贵的练习。但是您可以尝试使用超分辨率技术来提高图形的质量。超分辨率的想法是通过基于可用数据在神经网络中绘制缺失的像素来提高图像的分辨率。现在已经取得了令人印象深刻的结果，让人联想起电影《剑道高手》中模因探索的图像增强场景





超分辨率技术改善了图片的视觉感知能力，例如github.com/tg-bomze/Face-Depixelizer，但为图像带来了新的信息。并可以用来提高电影质量：





但是，大多数算法都占用大量资源，因此我想创建一个脚本来实时改进游戏。



一点理论



以下所有内容都将适用于卷积神经网络 - 卷积神经网络的一种子类型，用于处理图像。首先，让我们看一下神经网络如何解决超分辨率问题。该任务与自动编码器问题的解决方案非常相似。有必要将图像提交到网络输入，以在输出处获得相同的图像。但是，自动编码器通常用于解决有效数据压缩的问题，因此，其体系结构的一个特征是瓶颈-瓶颈，即具有少量神经元的网络层。这种层的存在使其余部分学习有效的信息编码和解码。为了训练超分辨率网络，首先要有意识地降低高质量图像的分辨率，并将其输入到神经网络的输入中。该输出是高质量的预期源图像。



超分辨率任务定义了所用网络的体系结构：

通常，输入和输出数据之间存在连接（跳过连接），这大大加快了学习速度。输入数据的像素大小增加并添加到卷积网络的输出中。因此，实际上，您无需训练网络即可将图像转换为几乎相同的图像。您只需要训练她就可以通过增加像素大小和实际图像来画出放大后的差异。在各个级别并通过不同数量的层具有跳过连接的想法非常有效，并导致了网络的残差网络类的出现。如今，此类连接已用于几乎所有流行的体系结构中。在这里可以找到解决超分辨率问题的最新体系结构的概述。。我的任务是创建一个神经网络来实时解决超分辨率问题。



首先，我选择了具有4个块的edsr体系结构（通常使用16个以上的块层）ResNet，其分辨率提高了4倍。在这里，我部分使用了github.com/krasserm/super-resolution的开发以及同一项目的数据生成器。



常规网络架构如图所示。每个块都是X * Y * N图像，其宽度对应于通道数。过渡-对应于3x3卷积（在res块的情况下，随后激活ReLU以实现非线性）。升级步骤-由于通道变平而导致尺寸增加。







如果仔细观察如此小的网络，您会发现最后一个块是线性的，并且没有几个升级步骤是不合理的。您可以用更少的过滤器将它们替换为一层，而又不会损失质量而又不会损失网络质量：







事实证明，这样的网络几乎可以以相同的方式处理超分辨率任务，但速度更快，并且可以启动以实时处理小视频分辨率。



从data.vision.ee.ethz.ch/cvl/DIV2K下载了高质量的照片。原则上，您可以使用任何照片进行培训。 



import tensorflow as tf
from model import resolve_single
from model. rtsr import rtsr
import numpy as np
from utils import load_image, plot_sample
from data import DIV2K
from tensorflow.keras.optimizers.schedules import PiecewiseConstantDecay
# Model creation
rtsr_model= rtsr(scale=4, num_res_blocks=4)
#Data loading and generator creation
train_loader = DIV2K(scale=4,             
                     downgrade='bicubic',
                     subset='train')      
                     


train_ds = train_loader.dataset(batch_size=16,
                                random_transform=True,
                                repeat_count=None)  
valid_loader = DIV2K(scale=4,  
                     downgrade='bicubic',
                     subset='valid')  
                     

valid_ds = valid_loader.dataset(batch_size=16,           # use batch size of 1 as DIV2K images have different size
                                random_transform=True, # use DIV2K images in original size
                                repeat_count=40)         # 1 epoch
#Saving checkpoints during training
cbcks=tf.keras.callbacks.ModelCheckpoint('callback_x4{epoch:02d}-{loss:.2f}', monitor='loss', verbose=0, save_best_only=False)

learning_rate=PiecewiseConstantDecay(boundaries=[20000], values=[1e-4, 5e-5])
#model compilation before training
rtsr_model.compile(loss='mae', optimizer=tf.keras.optimizers.Adam(learning_rate=learning_rate))
#Training
rtsr_model.fit(train_ds,validation_data=valid_ds, steps_per_epoch=3000, epochs=100, validation_steps=10,callbacks=[cbcks])
#saving weights
rtsr_model.save_weights('weights/ myedsr-lastdiv2_4block-x4_weights.h5')


输出是具有简单体系结构的相对较小的（2.6 MB）神经网络。同时，该检查与预训练的16块网络有细微的差别：







左边的图像是原始图像，右边的图像使用16层edsr进行了改进，中间的图像使用rtsr。从我的角度来看，2和3之间的差异并不明显。

如何开始？



为了快速工作，您需要具有支持cuda和cudnn（https://developer.nvidia.com/cuda-gpus）的视频卡以及已安装的库cuda / cudnn。您需要安装python 3.7和tensorflow（版本高于2.0并具有gpu支持）。这可能是一项艰巨的任务，并且可能会出现兼容性问题。



最简单的方法是安装Anaconda发行版（https://www.anaconda.com/products/individual），然后使用conda安装tensorflow-gpu： 



conda install tensorflow-gpu


如果由于冲突而无法解决问题，则可以尝试 



conda install cudnn
pip install tensorflow-gpu


应该管用。



可以使用pip安装其他库： 



pip install opencv-python
pip install pywin32
pip install mss


接下来，您需要使用以下命令运行脚本： 



python superres.py


使用wsad和ijkl键（调整大小）控制捕获窗口。q-关闭窗口。0-打开/关闭超分辨率模式。数字1和2-图像模式（默认情况下为2表示游戏使用由于像素增加2倍而导致的增加）。
