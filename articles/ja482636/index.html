UniversitätまたはHochschule


ドイツでは、大学は2つのタイプに分けられます。

  • 大学は古典的な大学です。より理論的な学問分野、より多くの研究があり、博士号を取得することも可能です。
  • Hochschule（lit.「高等学校」）は、実践指向の大学です。

Hochschuleの評価は通常低くなります（Hochschuleであり、非常に高い評価を持つRWTHアーヘン大学を除く）。大学への入学は、将来博士号を取得する予定の方、卒業後に就職する予定の方は、Hochschuleを選択することをお勧めします。個人的には「Universität」に重点を置いていましたが、高い評価とバックアップ計画としてのHochschule Fuldaがあるため、RWTHアーヘン大学の2つの「Hochschule」に私のリストに含まれていました。

1.3。入学要件


入学要件は、大学によって、また同じ大学のプログラムによって異なる場合があるため、要件のリストは、大学のウェブサイトのプログラムの説明で指定する必要があります。ただし、どの大学にも関連する基本的な要件のセットを区別できます。

  1. 高等教育の卒業証書（「学位証明書」）
  2. スコアシート（「記録の筆記録」）
  3. 言語証明書（IELTSまたはTOEFL）
  4. 動機書簡（「目的の表明」）
  5. CV（CV）

一部の大学では、追加の要件を提示しています。

  1. 特定のトピックに関する科学的エッセイ
  2. GREテスト
  3. 推薦状
  4. 専門分野の説明-各学科と学習したトピックの時間数を示す公式文書（卒業証書に示された専門分野について）。
  5. カリキュラム分析-卒業証書の科目と大学で教えられた科目の比較、科目を特定のカテゴリに分割するなど
  6. あなたの卒業プロジェクトの本質の簡単な説明。
  7. 学校の証明書。

さらに、大学は通常、あなたの業績と資格を確認する他の文書（出版物、出席証明書、専門家の証明書など）をダウンロードする機能を提供しています。

1.4。IELTS


私は、IELTSの準備と配信から導入キャンペーンを開始しました。十分なレベルの英語が確認されていなければ、純粋に正式な基準に合格することはなく、他のすべては必要なくなります。

IELTSは特別認定センターの講堂で開催されます。ミンスクでは、毎月試験が行われます。配達の約5週間前に登録する必要があります。さらに、録音は3日間しか行われなかったため、都合のよい日に録音がスキップされるリスクがありました。記録と支払いはIELTSウェブサイトでオンラインで行うことができます。

ほとんどの大学では、9点のうち6.5点を採点するだけで十分です。これは、おおよそ中級レベルに相当します。一部の大学（および、RWTHアーヘン大学など、常に最新のランキングではない）では、5.5ポイントで十分です。 7.0を超える場合、ドイツに単一の大学は必要ありません。また、言語証明書のスコアが高いからといって、入学の可能性が高いとは限らないという話にもよく耳にしました。ほとんどの大学では、あなたが基準を通過したかどうかだけが重要です。

英語のレベルが高い場合でも、試験の準備自体を怠らないでください。テストに合格するためのある程度のスキルと、その構造と要件に関する知識が必要です。準備のために、ミンスクでの対応する2か月のフルタイムコースとeDXの無料オンラインコースにサインアップしました。

フルタイムのコースでは、ライティングの部分（グラフを分析してエッセイを書く方法）を理解するのに大いに役立ちました。審査官は、彼らがポイントを取る偏差から、非常に厳密な構造を見ることを期待しています。また、コースでは、「はい」または「いいえ」と答えた場合にTRUEまたはFALSEを答えるべきではない理由、回答バンクに大文字で記入する方がよい理由、回答に記事を含める場合と含まない場合など、純粋に試験の瞬間なども理解しました。フルタイムのコースと比較して、edXコースは少し退屈であまり効果的ではないように見えましたが、一般的に、試験に必要なすべての情報もそこにまとめられていました。理論的には、edXのオンラインコースを受講し、過去数年間のテストの3〜4のコレクションを解く場合（急流で見つけることができます）、十分なスキルがあるはずです。「IELTSの語彙をチェックする」および「IELTS言語の実践」という本も私を助けてくれました。 「IELTS vocabulary in use」、「Using Collocations for Natural English」、「IELTS for Academic Purposes-Practice Tests」、「IELTS Practice Tests Plus」などの書籍も推奨されましたが、十分な時間がありませんでした。

テストに合格してから2週間後、IELTS Webサイトで結果を確認できます。これは、友達以外への発送には適さない情報です。正式な結果は、試験を受けた試験センターで取得する必要がある証明書です。これは試験センターの署名とシールが貼られたシートA4です。このドキュメントのコピーは、大学に送信できます（大学はIELTS Webサイトで信頼性を確認できるため、公証なしでも可能です）。

1.5。GRE


アメリカの大学とは異なり、ドイツの大学では、GRE結果の要件はそれほど一般的ではありません。どこかが必要な場合、それはあなたの能力の追加の指標としてより可能性が高いです（たとえば、UniversitätBonn、TUカイザースラウテルン）。私が検討したプログラムからの特定のGRE結果に対する厳格な要件は、コンスタンツ大学にのみ存在していました。

12月中旬にIELTSの結果を受け取ったとき、私は残りのドキュメントの準備を始め、GREテストにもサインアップしました。GREを準備するために私の体力を1日費やしたので、私はおそらく彼を失敗させました（私の意見では）。私の結果は次のとおりです。口頭推論では149ポイント、定量分析では154ポイント、分析的記述では3.0ポイントでした。それでも、GREの結果が必要な大学へのアプリケーションにこのような結果を適用しました。練習が示したように、これは悪化しなかった。

1.6。書類の作成


高等教育の卒業証書、成績表、学校の証明書は、アポスティーユ、英語またはドイツ語に翻訳、公証されている必要があります。これらはすべて、どの翻訳会社でも行うことができます。ユニアシストシステム（TUミュンヘン、TUベルリン、TUドレスデンなど）を介してドキュメントを受け入れる大学に入る予定の場合は、各ドキュメントの追加の公証済みコピー1部を翻訳会社にすぐにリクエストしてください）。一部の大学（TUミュンヘン、ハンブルク大学、FAUエアランゲン-ニュルンベルクなど）では、書類のコピーを郵送する必要があります。この場合、そのような大学ごとに、翻訳局に各文書の公証済みのコピーを1枚追加で依頼してください。

翻訳会社に連絡してから1週間後に、文書の翻訳、アポスティーユ、公証の翻訳を受け取りました。

翻訳をピックアップするときは、必ず品質を再確認してください。私の場合、翻訳者は「オペレーションシステム」（「オペレーション」ではなく）、「サテイデオロギー」（「ステート」ではなく）など、いくつかのエラーやタイプミスを犯しました。残念ながら、私はこれにかなり遅れて気づきました。幸い、単一の大学がこれに欠陥を見つけたわけではありません。翻訳されたドキュメントの電子コピーを求めることは理にかなっています-そこから名前をコピーできます。これにより、紹介フォームに記入する際の時間を節約できます。

また、ドイツの大学で専門分野の説明が必要な場合は、その専門分野が英語で存在するかどうかを確認してください。わからない場合や見つからない場合は、質問を書いた手紙を学部長や事務局に遠慮なく送ってください。私の場合、専門分野の説明は「ベラルーシ共和国の教育基準」であり、その正式な翻訳はありませんでした。この場合、2つのオプションがあります。自分で翻訳するか、もう一度翻訳会社に行きます。幸いなことに、彼には公証は必要ありません。個人的に、私は以前に言及された「教育基準」から意味のない事務作業を切り取っていた翻訳会社に目を向けました。

IELTS証明書は、通常の未認証のコピーの形式で提供できます。ほとんどの大学は、IELTS検証システムにアクセスできます。このシステムでは、証明書の信頼性を検証できます。証明書の原本（およびその他の文書）を郵送しないでください。受け取っていない場合、返却されることはほとんどありません。

GREテストの結果は通常ets.orgの主催者のWebサイトから電子的に送信されますが、一部の大学（TUカイザースラウテルンなど）は、ETS Webサイトの個人アカウントからダウンロードした通常の証明書の形式で結果を受け入れる準備ができています。

履歴書は通常、表形式で提供され、学校から始まり、入学の瞬間で終わるすべての活動を示し、活動のすべての開始日と終了日、この期間の成果（たとえば、学校と大学での平均スコア、職場での完了したプロジェクト）を示します。スキルだけでなく（たとえば、プログラミング言語の知識）一部の大学では、ユーロパス形式の履歴書が必要です。

2.1。ユニアシスト



Uni-assistは、外国の書類の検証と多くの大学への入学申請の検証に従事している会社です。彼らの仕事の結果は「VPD」です-卒業証書の真正性の確認、ドイツの採点システムの平均スコア、選択した大学で選択したプログラムに入学する許可を含む特別な文書。 TUミュンヘン、TUベルリン、TUドレスデンに入るには、Uni-assistを経由する必要がありました。さらに、このドキュメント（VPD）はさまざまな方法で使用されています。

たとえば、TUMünchenと入力すると、Uni-assistは個人的にVPDを送信します。このVPDは、TUミュンヘンへの入学のためのオンライン申請システムであるTUMOnlineにアップロードする必要があります。これに加えて、このVPDは他の紙の郵送書類と一緒にTUミュンヘンに送信する必要があります。

他の大学（TUベルリン、TUドレスデンなど）では、Webサイトで個別のアプリケーションを作成する必要はありません。Uni-assistは、VPDを（ドキュメントと連絡先の詳細とともに）直接送信します。その後、大学から研究への招待状を送信できます。メールで。

ユニアシストの最初のアプリケーションの費用は75ユーロです。他の大学へのその後の各申請には30ユーロの費用がかかります。ドキュメントを1回送信するだけで十分です-ユニアシスト会社はすべてのアプリケーションでそれらを使用します。

支払い方法は私を少し驚かせた。最初の方法は、カードの詳細を指定した特別なシートをドキュメントのパッケージに添付することです（CV2コード、つまりすべての秘密情報を含む）。何らかの理由で、彼らはこのメソッドを便利に呼んでいます。支払いについて2要素の承認があり、支払いごとに新しいコードが携帯電話に届くという条件で、彼らがどのようにお金を引き出すかはまだわかりませんでした。たぶん、拒否します。どの支払いシステムでもカードで支払う方法がないのは奇妙です。

2番目の方法はSWIFT変換です。以前はSWIFTの翻訳を利用したことがなく、次のような驚きに遭遇しました。

  1. 最初に来た銀行は送金を拒否しました。ユニアシストからの手紙は、外国の法定口座への送金の根拠にはなりません。契約書または請求書が必要です。
  2. 第二銀行は私を送金することを拒否しました、なぜなら 手紙はロシア語ではなかった（それは英語とドイツ語であった）。私が手紙をロシア語に翻訳したとき、彼らは私を拒否しました、なぜなら 「奉仕の場所」を示すものではありませんでした。
  3. サードバンクは私のドキュメントを「そのまま」受け入れ、SWIFT送金を完了しました。
  4. 異なる銀行での送金の費用は17ドルから30ドルです。

3.1。ブロックされたアカウントを開く


ブロックされたアカウントとは、出金できないアカウントです。代わりに、銀行から毎月別の銀行口座に送金されます。そのようなアカウントの存在は、ドイツへの学生ビザを取得するための前提条件です。このようにして、ドイツ政府は最初の月にすべてのお金を使い、ホームレスになることを保証します。

翌日、PINコード、モバイルバンキングにアクセスするための写真コード、およびプラスチック製カードを郵送で受け取りました。このカードは、インターネットで支払いをしたり、自転車のレンタルサービスなどに添付したりすることができない、最も単純なカードであることがわかりました。また、このカードから現金を引き出す過程にも少し驚いた。モバイルバンキングを通じて、手数料なしで現金を引き出すことができる最寄りのATMについて学びました。そこに着くとガソリンスタンドがありました。四方から回りましたが、ATMはありませんでした。それから、このガソリンスタンドのレジで「ATMはどこですか？」という質問に答えました。その後、彼は私のカードを受け取り、ターミナルに挿入して、「どれだけ引き出しますか？」と尋ねました。つまり、ガソリンスタンドのレジ係は同じ現金自動支払機であることがわかりました。

モバイルバンキングは、私がベラルーシで持っていたバンキングと比較して、その原始性に少しがっかりしました。ベラルーシのモバイルバンキングで支払い（モバイル通信、インターネットなど）、銀行へのアプリケーションの送信（新しいカードの発行など）、すべてのトランザクション（不完全なトランザクションを含む）の監視、通貨の即時変更、開くことができる場合預金とローンを取る、ここでは残高のみを確認でき、完了したトランザクションを確認でき、指定した銀行口座に送金できます。つまり、モバイル通信の料金を支払うためには、対応する会社の支店に行き、彼らのレジで支払うか、スーパーでプリペイドカードを購入する必要があります。私が理解しているように、地元の人がSIMカードを購入するとき、彼らは同じ方法で彼らの口座から直接お金が差し引かれるという合意を結びます、健康保険のお金のように。そして、おそらく、この不便はそれほど明白ではありません。
