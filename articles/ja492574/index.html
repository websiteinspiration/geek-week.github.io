離陸確率に関する考察：Momentum True Wireless 2搭載のApple AirPods

完全ワイヤレスヘッドフォンの市場は、Apple製品の適切な代替品となるものを待ち望んでいます。ハッチングされたセグメントの少なくとも一部からアップル社を移動しようとする定期的な試みがありましたが、これまでのところ誰もできませんでした。ソニーはWF-1000XM3を試してみましたが、「キラー」の最初のバージョンは扱いにくく、「感謝」したところ、人間工学的な操作上の欠陥がいくつかありました。



ゼンハイザーがもう一度試みます。祖父フリッツの発案も最初の試みではありません。ソニーとは異なり、ドイツのトライアルボール-Sennheiser Momentum True Wirelessは中間目標を達成し、プレミアム価格にもかかわらず、それは本当に求められているガジェットとしての地位を確立しました。私たちは、ドイツの「狙撃兵」であるゼンハイザーモメンタムトゥルーワイヤレス2がアメリカのアップルに2度目を向けた精度を理解しています。

最初の試みについて少し


Momentum True Wirelessの最初のバージョンは、Apple AirPodsの競合他社として位置付けられていませんでした。まず第一に、それは音楽に、そしてそれ以外のすべてに必要とされる高価なプレミアムヘッドフォンとして作成されたという事実のために。伝統的に、ゼンハイザーはドライバーの品質、低歪み、レゾナンスの欠如、つまり音楽愛好家や他の哲学者からこの会社の製品が高く評価されているすべてのものに多くの注意を払っていました。

しかし、ゼンハイザーの概念主義者やマーケティング担当者の計算は完全に正確ではありませんでした。それがどうあるべきかについての大衆市場の考えは音楽愛好家が望むものから非常に遠いです。市場は、根本的に異なるヘッドホンではないにしても、「小さなプラグからの息をのむような音」からの強調が機能性と快適性にシフトするデバイスを必要としていました。そして、これは許容範囲内（最悪の場合）で、理想的にはまともな音楽とオーディオファンの基準で響きます。



したがって、Momentum True Wirelessは、金の耳に対する特定の主張、高調波歪み、相互変調の主張、および周波数応答を等しく求めている人々のセグメントでのみ機能しました。これらは伝統的に、低、高、中、高音に重点を置いた独自の「Sencheiserサウンド」によって区別されてきました。

エラー処理


市場が何を必要としているかを分析した後、ヨーロッパの主要なヘッドホンメーカーは、理論的には日本と韓国の企業の新しい開発と競争できる方法を実行することを決定し、理想的には、市場の一部をAppleから絞り出しました。したがって、Momentum True Wireless（第2世代）の2番目のバージョンは、アクティブノイズの低減、再設計されたデザイン、およびバッテリ寿命の延長を受けました。現時点では、1回の充電後、デバイスは最大7時間自律的に動作でき、充電ケース（+3自律充電）を使用すると、自律使用の可能性が28時間に増加します。最大時間を示したことに注意してください。非常に集中的に使用すると、3〜4時間削減されますが、この場合でも、バッテリー寿命は1日です（つまり、最悪のシナリオでは、AirPods Proの上限以上です）。



さらに、ヘッドフォンは本当に大音量で「低音」であり、感度が107 dBの強力なアンプと7 mmのドライバーが本当に深い底を作り出します。最初のモデルでは、ディフューザーの直径はさらに大きくなりましたが、その減少は低周波性能にあまり影響しませんでした。



「スマート」な革新として、ヘッドフォンはデバイスが耳から外されると音楽の再生を一時停止し、挿入されたときに再生を開始します。また、必要に応じて、ノイズ低減をオフにして、必要に応じて外部ノイズや大声をスキップする「透明聴覚」モードを使用できます。ガジェットはSennheiser Smart Controlアプリケーション（Android / iOS用）で動作します。これにより、各ヘッドフォンのタッチパネルを設定して、再生や音量を制御したり、GoogleアシスタントやSiriを呼び出したりできます。

論争の的となる次元のモーメント


アップルの対応物と比較して、それらのボリュームははるかに大きいです。一方で、これはサウンドに適しています。スピーカーシステムの音量が大きいほど、再生の忠実度が高くなる一方で、人間工学的に収益性が低下し、AirPodsの小さいサイズに慣れているユーザーを怖がらせる可能性があります。寸法はSony WF-1000XM3の寸法ほど大きくないことにも注意してください。再生の忠実度は日本人にかなり匹敵し、おそらくそれを超えます。
