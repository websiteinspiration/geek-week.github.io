Python SAXパーサーとPython DOMパーサー。Parsim FIAS-houses

前回の記事では、FIASが公開したデータに基づいてxmlからcsvを作成する方法を検討しました。解析は、処理前にファイル全体をメモリにロードするDOMパーサーに基づいていたため、限られた量のRAMを考慮して大きなファイルを分割する必要がありました。今回は、SAXパーサーがいかに優れているかを確認し、その速度をDOMパーサーと比較することを提案します。FIASデータベースファイルの中で最大の27.5 GBのサイズのハウスがテスト対象として使用されます。

前書き


私たちは、最も尊敬される聴衆を即座に混乱させることを余儀なくされました-FIXのハウスデータベースファイルをSAXパーサーにフィードすることに失敗しました。パーサーは、「整形式ではありません（無効なトークン）」というエラーでクラッシュします。そして、最初はデータベースファイルが壊れているという疑いがありました。ただし、データベースをいくつかの小さな部分に切り分けた後、家や建物の番号のコーディングが変更されたことによって逸脱が引き起こされたことがわかりました。つまり、STRUCNUMタグまたはHOUSENUMタグで、家は奇妙なエンコーディング（UTF-8ではなく、ドキュメント自体が形成されたANSIではない）で書かれた文字に遭遇しました

from datetime import datetime
import re
from unidecode import unidecode

start = datetime.now()

f= open('AS_HOUSE.462.xml', 'r',encoding='ANSI')
def remove_non_ascii(text):
        return unidecode(unidecode(text))

for line in f:    
        b=remove_non_ascii(line) 
        for c in re.finditer(r'\w{5}NUM="\d{1,}\"\w\"',b): 
                print(c[0])                      
                c1=c[0][:-3]+c[0][-2]
                print(c1) 
                b=b.replace(c[0],c1) #           

                #

                f1.write(b)
                f1.close()

f.close()
print(datetime.now()- start)


このコードは、xmlファイルでnon_ascii文字を通常の文字に変換し、建物や家の名前から不要な ""を削除します。

SAXパーサー


最初に、小さなxmlファイル（58.8 MB）を取得します。出力でtxtまたはcsvを取得することを計画しており、pandasまたはexcelでさらに処理するのに便利です。

コード
import xml.sax
import csv
from datetime import datetime

start = datetime.now()

class EventHandler(xml.sax.ContentHandler):
    def __init__(self,target):
        self.target = target
    def startElement(self,name,attrs):
        self.target.send(attrs._attrs.values())          
    def characters(self,text):
        self.target.send('')
    def endElement(self,name):
        self.target.send('')

def coroutine(func):
    def start(*args,**kwargs):
        cr = func(*args,**kwargs)
        cr.__next__()
        return cr
    return start

with open('out.csv', 'a') as f:
    # example use
    if __name__ == '__main__':
        @coroutine
        def printer():
            while True:
                event = (yield)                            
                print(event,file=f)
                

        xml.sax.parse("out.xml", EventHandler(printer()))

print(datetime.now()- start)

DOMパーサー


最初に同じファイル全体をメモリにロードして、同じファイルを処理します。これは、まさにDOMパーサーが使用する方法です。

コード
import codecs,os
import xml.etree.ElementTree as ET
import csv
from datetime import datetime

parser = ET.XMLParser(encoding="ANSI")
tree = ET.parse("out.xml",parser=parser)
root = tree.getroot()

Resident_data = open('AS_HOUSE.0001.csv', 'a',encoding='ANSI')
csvwriter = csv.writer(Resident_data)

attr_names = [
    'HOUSEID', 'HOUSEGUID', 'AOGUID', 'HOUSENUM', 'STRUCNUM', 
    'STRSTATUS', 'ESTSTATUS', 'STATSTATUS', 'IFNSFL', 'IFNSUL', 
    'TERRIFNSFL', 'TERRIFNSUL', 'OKATO', 'OKTMO', 'POSTALCODE', 
    'STARTDATE', 'ENDDATE', 'UPDATEDATE', 'COUNTER', 'NORMDOC', 
    'DIVTYPE', 'REGIONCODE'
]
start = datetime.now()
object = []
for member in root.findall('House'):    
    object = [member.attrib.get(attr_name, None) for attr_name in attr_names]
    csvwriter.writerow(object)    
Resident_data.close()
print(datetime.now()- start)
