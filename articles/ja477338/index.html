OWASPモスクワ（ロシア）ミートアップ12/19 CFP


今年のOWASPのロシア支部の最後の会議は、BI.ZONEのモスクワ事務所で2019年12月6日に開催されます。

興味深いレポートやディスカッション、コミュニティプログラムと2020年の計画、コミュニティ活動への参加、地元のOWASP支部の再編成などが見つかります。

OWASP


Open Web Application Security Project（OWASP）は、世界中の大企業、教育機関、個人を結びつけます。コミュニティのメンバーは、記事やチュートリアルを書き、ドキュメント、ツール、テクノロジーを作成します。すべてのOWASP開発は公開されています。

参加する


OWASPのモスクワ（ロシア）支部の活動に参加するには、さまざまな方法があります。
  • アクティブなプロジェクトへの参加（ユーティリティやドキュメントの作成を含む）
  • 新規プロジェクトのご提案
  • 既存プロジェクトのロシア語への翻訳
  • 会議や会議の開催を支援
  • OWASPプロジェクト全体の推進とサポート
