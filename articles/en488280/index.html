How to look at Oracle WebLogic Server using a large monitor and console WLSDM utility

On the open spaces of the console extension utilities Oracle WebLogic Server, there was one very useful one - WLSDM , as the authors themselves position it - a WebLogic Server monitoring utility with a large set of features. If you go to the developer's site, you will see that there is another powerful tool nearby, but for a fee.

After downloading the archive, in just two clicks we get an already suitable tool for work, with a set of threshold values, the ability to send messages and large chart speedometers. Especially for a large monitor, or to help a paired DevOps engineer who needs to keep abreast of spinning applications.

A few words about server metrics in general. Of course, you can use the built-in metric view. You can even learn how to customize metric tables in the server itself.
But we proceed from the convenience of use and the comparative likeness of this tool.
Plus, the set thresholds are immediately displayed on the charts and warnings are sent by letter. Which, in fact, is necessary for DevOps.
