使用p5.js在3d对象上覆盖2d纹理（第1部分-创建模式）


在三维对象上叠加平面图形或图案始终是一项艰巨的任务，如果同时要用某种动画来完成，复杂性有时会增加。

同时，有一个相当简单的p5.js库，您可以用它在几十行代码中创建复杂的壮观模式。我最近创建了一个这样的项目，现在我想举一个例子。

最简单的开始方法是在在线编辑器中创建一个新项目。p5js.org

默认情况下，新项目如下：

function setup() {
  createCanvas(400, 400);
}

function draw() {
  background(220);
}

函数setup（）-设置仅执行一次的
函数函数draw（）-在无限循环中重复

我们首先创建2D径向图案，然后将其应用于立方体。创建它时，我们将首先使用极坐标，然后才将其转换为x的直角坐标，这是大多数人熟悉的形式。

需要理解的重要一点：一个点的极坐标由两个数字确定：

r是从源点（例如，画布的中心）到目标点的距离。
a是假想线相对于目标点所在起点的夹角，

因此，第一个参数的范围可以从0到无穷大，第二个参数的范围可以从0到360。

这种方法的优点在于，它允许您仅用一个公式绘制相当复杂的图案。



极地玫瑰由等式r（φ）= 2 \ sinφ给出

为了创建变量，让我们在项目的开始处添加两个变量：第一个变量负责距中心的距离，第二个变量负责角度：let r = 10 let a = 0

然后，在无限循环中运行的draw（）函数中，添加变量change： = a + 0.4

结果，您得到以下代码：

let r =10
let a =0
function setup() {
  createCanvas(400, 400);

}

function draw() {
  background(220);
    a = a + 0.4
}

现在我们开始将坐标转换为我们熟悉的笛卡尔坐标x y。如果您想了解代码中到底发生了什么，最好阅读三角学教程，但是由于这对结果并不重要，因此您只需将代码粘贴到相同的draw（）函数中：

let x = r*cos(a)
let y = r*sin(a)

为了最终看到我们的数学计算结果，我们需要做两件事：

  • 用颜色绘制对象（您可以使用内部颜色字典或指定通常的RGB格式代码-三个数字）
  • 设置点的分布形状。例如，椭圆。

function draw() {
  background(220);
  
  
  let x = r*cos(a);
  let y = r*sin(a);
  fill(255,0,0);
  ellipse (x,y,10,10);
  r = r + 0.4
    
}



使用沿x和y的变量设置椭圆的参数（沿X的位置，沿Y的位置，沿X的直径，沿Y的直径）。由于变量x取决于变量r，在我们的循环中变量r不断增加0.4，因此我们看到的红色圆圈将不断在左上角移动。

要将参考点移至中心，请在x变量之前添加平移函数（200,200）。现在，圆圈将出现在画布的中间。

让我们解决另一个问题-现在，每个新循环都会删除所有出现的圆圈。为了摆脱这种恶性循环，我们需要将canvas（）的创建移到draw（）函数之外，并将其转移到负责创建环境的setup（）函数。

更改之后，我们将看到类似于加载圆的内容。主要问题是现在它们彼此之间非常靠近。让我们添加一些空间，方法是将一个较高阶的变量C设置为20，然后将其相加以将每个周期更改0.6（a旁边）。好吧，我们将其引入到三角公式中：x = r + c * cos（a）和y = r + c * sin（a）
