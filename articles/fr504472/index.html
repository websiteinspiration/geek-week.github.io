Mes meilleurs outils de développement gratuits

Ceci est une traduction de l' article de dev.to. Son auteur Fayaz Ahmed partage une liste de ses outils de développement préférés.



Je développe des projets tels que Tvflix , GSON , Appy Dev , ainsi que de nombreux microprojets différents, donc je suis intéressé à trouver des outils gratuits pour mon travail.

Ceci est ma liste personnelle, mais vous pouvez la compléter avec d'autres outils efficaces. Voici quelques-uns de mes favoris.

Services et plateformes pour déployer votre frontend / backend


1. Netlify
Utilisez Netlify pour déployer des sites Web statiques modernes. Vous bénéficiez d'une infrastructure CDN, d'un déploiement continu, d'un HTTPS en un clic et de tous les services dont vous avez besoin. Commencez gratuitement.

2. Vercel (anciennement Zeit)
Vercel est le moyen le plus simple de déployer des sites Web. Hébergez vos projets Web sans configuration avec SSL automatique et une infrastructure CDN globale.

3. Les projets Web de Surge
Forwarding doivent être rapides, faciles et sécurisés. Surge est une fonctionnalité de publication Web statique pour les développeurs frontaux directement à partir de la CLI.

4. Firebase
Une alternative à Netlify / Zeit de Google. Firebase est une plate-forme mobile qui vous aide à développer rapidement des applications de haute qualité et à développer votre entreprise.

5. Azure Static Web Apps
À partir d'aujourd'hui, vous avez la possibilité d'utiliser Static Web Apps pour héberger des applications d'une seule page dans Azure.

6.
Sites Web GitHub Pages pour vous et vos projets qui sont publiés directement à partir de votre référentiel GitHub. Il suffit d'éditer et d'exécuter pour que les modifications passent à la version live.

7. Pages GitLab
Comme GitHub, le service GitLab Pages vous permet de publier des sites Web statiques directement depuis le référentiel vers GitLab.

8. Voltige
Hébergement statique robuste. Fonctionne très bien avec des générateurs de sites statiques comme Hugo, Jekyll, Gatsby, Gridsome.

9. Glitch C'est
peut-être l'un des outils que j'utilise le plus souvent aujourd'hui. Un service gratuit simple et puissant pour créer et héberger des applications basées sur Node.js.

10. Heroku
Heroku, une version plus avancée de Glitch, est une «plateforme en tant que service» (PaaS) qui permet aux développeurs de créer des applications, de les exécuter et de les gérer complètement dans le cloud.

Outils d'automatisation


1. Cron-job.org
Cron-job est un service open source gratuit et vous permet d'exécuter un nombre illimité de tâches au format cron. Tout ce dont vous avez besoin est d'utiliser votre API ou l'adresse e-mail de votre webhook, qui sera utilisée pour se connecter au service avec la fréquence spécifiée.

2. n8n
Cet outil open source est une alternative à Zapier. Le service offre des possibilités illimitées, mettant constamment à jour la liste des outils et applications disponibles pour la synchronisation. Vous pouvez également héberger n8n sur votre propre serveur.

3. Integromat
Integromat est un outil facile à utiliser et pourtant efficace pour automatiser les processus manuels avec des caractéristiques uniques. Synchronisez vos applications, services et appareils préférés entre eux, même si vous n'avez pas de compétences en programmation.

4. IFTTT
Essayez IFTTT - le moyen le plus simple et le plus gratuit de faire plus avec vos applications et appareils préférés. Les tâches ménagères nécessiteront moins d'attention, le travail deviendra encore plus productif, vous préserverez la confidentialité et vos données seront en sécurité. Je suis sûr que tout fonctionne mieux dans des conditions de synchronisation.

5. Pipedream
Pipedream est une plateforme sur laquelle les développeurs peuvent créer et publier des projets liés au traitement des données. Ouvrez des projets et créez une fourchette sur une plate-forme qui contient des milliers d'exemples de code source et de projets terminés.

Outils sans code


1. NoCodeAPI
Le moyen le plus simple pour un développeur frontal de connecter son application à des services tiers sans écrire de code supplémentaire et un serveur dédié. NoCodeAPI peut fonctionner avec Google Sheet, Airtable, Google Analytics, Twitter, Instagram, XML, RSS, JSON, Slack, les systèmes de paiement et avec plus de cinquante services.

2. GlideApps
Glide transforme les tables dynamiques en applications conviviales, sans nécessiter de code. Sélectionnez une table ou commencez par un modèle, configurez les paramètres de votre application, puis partagez-la avec les autres en un instant. Essayez le gratuitement aujourd'hui!

3. Airtable
Airtable fonctionne comme une table dynamique, mais en même temps, il offre des capacités de base de données pour organiser n'importe quel contenu.

4. Bubble
Bubble vous présente une nouvelle façon de créer des applications Web. Cet outil de programmation ne nécessite aucun travail avec le code et fonctionne au format pointer-cliquer. Bubble héberge toutes les applications sur sa plateforme cloud.

5. Générateur d'icônes de matériaux
Générez des icônes SVG dans le style de conception de matériaux pour vos applications. Service open source.

Outils de référencement / marketing Internet / promotion / assistance juridique


1. Balises Meta
Générez et prévisualisez précédemment toutes les balises Meta et les balises des réseaux sociaux sur une page Web.

2. Outils de code Web
Créez des schémas, des balises et de nombreux extraits de code répétitifs.

3. Carbon
Carbon est le moyen le plus simple de créer et de publier des images visuelles de votre code source.

4. Codeimg
Créez et publiez des images visuelles de votre code source.

5. GetTerms Générez le
texte de base de l'accord d'utilisation et de la politique de confidentialité de votre site Web.

6. BEE Free
BEE est un éditeur d'e-mails qui vous permet de créer rapidement et sans problèmes de bons e-mails de réponse. Ses auteurs ont cherché à développer l'éditeur de courrier électronique parfait. Il est possible d'utiliser BEE en ligne ou de l'intégrer dans votre application.
