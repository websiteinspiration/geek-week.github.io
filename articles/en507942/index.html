Webinar "Creating a custom project in the Winnum dynamic application editor"





Winnum invites everyone to attend the free webinar "Creating a Custom Project in the Winnum Dynamic Application Editor". The event will take place on June 30 at 13:00 Moscow time.



As part of the webinar, we will tell you about the Winnum dynamic application editor, show how to work with it and create our own custom project. The event will be hosted by Sergey Solovyov, Winnum's Internet of Things specialist.



A dynamic application in Winnum is an application that is created by a user in the dynamic application editor in Winnum Platform.

The dynamic application editor is used to create and develop your own custom projects; it allows you to design applications and their mobile versions in HTML5 using drag-and-drop tools.

This editor speeds up the process of creating new user applications, and also allows you to quickly make changes to them.
