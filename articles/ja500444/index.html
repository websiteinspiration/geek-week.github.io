問題解決と質問のテクニック


プログラマーとして、問題の解決に十分な経験がないと、優れたコードの作成を開始するのが難しくなります。問題の解決に問題がある場合、問題の最善の解決策を見つけるのは簡単ではないでしょう。プログラミングはコードを書くだけではありません。問題解決スキルは、初心者のカテゴリからクロールする必要があるスキルです。

プログラミングについて質問することも非常に重要です。何もせずに誰かに問題の解決を依頼するだけでは、あなたは間違ったことをしています。これは簡単ではありませんが、自分で問題を解決しようとしないと、問題に対する他の誰かの解決策から何の利益も得られません。

リストを操作する


リストは優れた用途の広いものです。
たとえば、ここでは整数値と文字列を混合します：

my_list = ['a' , 'b' , 'n' , 'x' , 1 , 2 , 3, 'a' , 'n' , 'b']
for item in my_list:
    print (f'current item: {item}, Type: {type(item)}')



それらをどのように混合したか見てください。リストをソートしようとすると、エラーが発生します。

print (my_list.sort())



文字と数字を分離する必要がある場合はどうなりますか？最初の方法は、リストの各要素をチェックするループを使用して問題を解決することです。ループはプログラミングの重要な部分であるため、初心者は早い段階でサイクルを知るようになります。

コードは次のようになります。

my_list = ['a' , 'b' , 'n' , 'x' , 1 , 2 , 3 , 'a' , 33.3 , 'n' , 'b']
number_list = []
string_list = []
for item in my_list:
    print (f'current item: {item}, Type: {type(item)}')
    if not isinstance(item,str):
        number_list.append(item)
    else:
        string_list.append(item)
my_list = string_list

これは正しいアプローチですが、少し面倒です。コードは機能しますが、同じ方法で問題を1行で解決できるようにすることもできます。

幸せになりたい場合は、リスト式がPythonでどのように機能するかを学んでください。これは同じ問題ですが、リストの仕組みを理解することで解決しました。

my_list = [letter for letter in my_list if isinstance(letter,str)]


それで全部です！

そして、これは終わりではありません。を使用filterして同じ結果を得ることができます。

def get_numbers(input_char):
    if not isinstance(input_char,str):
        return True
    return False

my_list = [1,2,3,'a','b','c']
check_list = filter(get_numbers, my_list)
for items in check_list:
    print(items)




おそらくすでに理解しているように、同じ問題を解決するには多くの方法があります。あなたはあなたとあなたのチームのために特にうまくいくことを知っておく必要があります。

ループの使用


同様のコードに会ったことがありますか？

greek_gods = ['Zeus' , 'Hera' , 'Poseidon' , 'Apollo' , 'Bob']

おそらく他の言語からこの構成要素を学ぶでしょうが、これはPythonのアプローチとはかけ離れています。Pythonでは、以下を使用します。

for name in greek_gods:
    print (f'Greek God: {name}')

インデックスがないことにすぐに気づくでしょう。インデックスを操作したい場合はどうなりますか？Pythonでは利用できますenumerate。これは、必要なものすべてにアクセスするための優れた方法です。

for index, name in enumerate(greek_gods):
    print (f'at index {index} , we have : {name}')

オブジェクト指向プログラミング


Pythonはオブジェクト指向言語であり、オブジェクトには多くの能力があります。オブジェクトを描画と考えます—描画を使用する場合は、そのインスタンスを作成します。したがって、必要な数のインスタンスを作成でき、元の図面を使用しても何も台無しになりません。

オブジェクト指向プログラミング（OOP）は大きなトピックであるため、このセクションにはそれについて知る必要があるすべての内容は含まれていませんが、作業に役立ついくつかの簡単な例について説明します。

以前にOOPについて読んだことがあれば、おそらく勉強に飽きていますが、ここでもまたです。クラスの定義から始めましょうStudent。名前（name）と専門分野のリスト（subject_list）があります。

class Student():
    def __init__(self,name):
        self._name = name
        self._subject_list = []

これ__init__がクラスのコンストラクタです。クラスのコアを定義します。後で、メソッドがどのように追加されるかがわかります。メソッドは、プログラム内でクラスを使用するときに呼び出すことができるクラスに属するプロシージャまたは関数です。
生徒を作成する場合は、次の変数を作成します。

student1 = Student('Martin Aaberge')

さらに生徒が必要な場合は、同じクラスを使用して新しいオブジェクトを作成します。

student2 = Student('Ninja Henderson')

そしてstudent1、そしてstudent2クラスのインスタンスになりますStudent。彼らは同じスキームを持っていますが、共通点はありません。現時点では、生徒にはほとんど対応できませんが、専門分野のリストを追加しました。このリストを作成するには、別のメソッドを作成します。メソッドを呼び出して、クラスインスタンスと対話できます。

クラスを更新しましょう：

class Student():
    def __init__(self,name):
        self._name = name
        self._subject_list = []
    def add_subject(self, subject_name):
        self._subject_list.append(subject_name)
    def get_student_data(self):
        print (f'Student: {self._name} is assigned to:')
        for subject in self._subject_list:
            print (f'{subject}')
        print()

このクラスを使用して、生徒の作成、生徒に関する情報の編集および取得を行うことができます。

#create students:
student1 = Student('Martin Aaberge')
student2 = Student('Heidi Hummelvold')
#add subjects to student1
student1.add_subject('psychology_101')
student1.add_subject('it_security_101')
#add subject to student2
student2.add_subject('leadership_101')
#print current data on students
student1.get_student_data()
student2.get_student_data()

通常、クラスは個別のファイルに保存され、コードにインポートされます。この例ではStudent、ファイルにクラスを作成し、student.pyそれをファイルにインポートしましたmain.py。（この場合、両方のファイルが同じフォルダーにあります）。
