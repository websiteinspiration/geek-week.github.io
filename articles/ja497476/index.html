1.スタック（呼び出し）（スタック）




スタックはLIFOの原則に従います（後入れ先出し-後入れ先出し）。本を積み重ねて、一番下の本を取りたい場合は、最初に一番上の本を取り、次に次の本を取ります。ブラウザの[戻る]ボタンを使用すると、前のページに戻る（戻る）ことができます。

スタックには次のメソッドがあります。

  • プッシュ：新しいアイテムを追加する
  • pop：一番上の要素を削除して返します
  • ピーク：トップ要素を返す
  • length：スタック上の要素の数を返します

JSの配列にはスタック属性がありますが、Stack（）関数を使用してゼロから構築します。

function Stack() {
    this.count = 0
    this.storage = {}

    this.push = function(value) {
        this.storage[this.count] = value
        this.count++
    }

    this.pop = function() {
        if (this.count === 0) return undefined
        this.count--
        let result = this.storage[this.count]
        delete this.storage[this.count]
        return result
    }

    this.peek = function() {
        return this.storage[this.count - 1]
    }

    this.size = function() {
        return this.count
    }
}

2.キュー（キュー）




キューはスタックに似ています。違いは、キューがFIFOの原則に従うことです（先入れ先出し-先入れ先出し）。あなたが並んでいるとき、その最初が常に最初になります。

キューには次のメソッドがあります。

  • エンキュー：キューに入り、最後にアイテムを追加します
  • デキュー：キューを離れ、最初の要素を削除して返します
  • フロント：最初の要素を取得
  • isEmpty：キューが空かどうかを確認します
  • サイズ：キュー内のアイテム数を取得します

JSの配列にはいくつかのキュー属性があるため、それをデモンストレーションに使用できます。

function Queue() {
    let collection = []

    this.print = function() {
        console.log(collection)
    }

    this.enqueue = function(element) {
        collection.push(element)
    }

    this.dequeue = function() {
        return collection.shift()
    }

    this.front = function() {
        return collection[0]
    }

    this.isEmpty = function() {
        return collection.length === 0
    }

    this.size = function() {
        return collection.length
    }
}

3.リンクリスト（ノードとリンクまたはポインターのリンクリスト）（リンクリスト）




文字通り、リンクリストは連鎖データ構造であり、各ノードは2つの部分で構成されています。ノードデータと次のノードへのポインターです。リンクリストと条件付き配列は、シリアル化されたストレージを持つ線形データ構造です。違いは次のとおりです。

基準 アレイ リスト
メモリ割り当て 静的、コンパイル時に順次発生 動的、起動時に非同期で発生（実行）
アイテムの入手 インデックス検索、高速 すべてのキューノードを検索します。速度は遅くなります
アイテムの追加/削除 順次および静的メモリ割り当てのため、速度は遅くなります 動的なメモリ割り当てにより、速度が速くなります
構造 1つ以上の方向 単方向、双方向、またはサイクリック

単一リンクリストには次のメソッドがあります。

  • サイズ：ノードの数を返す
  • head：最初の要素を返します（head-head）
  • 追加：要素を末尾に追加します（尾-尾）
  • 削除：複数のノードを削除
  • indexOf：ノードインデックスを返す
  • elementAt：インデックスでノードを返す
  • addAt：特定の場所にノードを挿入します（インデックスにより）
  • removeAt：特定のノードを（インデックスで）削除します

function Node(element) {

function LinkedList() {
    let length = 0
    let head = null
    this.size = function() {
        return length
    }

    this.head = function() {
        return head
    }

    this.add = function(element) {
        let node = new Node(element)
        if (head === null) {
            head = node
        } else {
            let currentNode = head
            while (currentNode.next) {
                currentNode = currentNode.next
            }
            currentNode.next = node
        }
        length++
    }

    this.remove = function(element) {
        let currentNode = head
        let previousNode
        if (currentNode.element !== element) {
            head = currentNode.next
        } else {
            while (currentNode.element !== element) {
                previousNode = currentNode
                currentNode = currentNode.next
            }
            previousNode.next = currentNode.next
        }
        length--
    }

    this.isEmpty = function() {
        return length === 0
    }

    this.indexOf = function(element) {
        let currentNode = head
        let index = -1
        while (currentNode) {
            index++
            if (currentNode.element === element) {
                return index
            }
            currentNode = currentNode.next
        }
        return -1
    }

    this.elementAt = function(index) {
        let currentNode = head
        let count = 0
        while (count < index) {
            count++
            currentNode = currentNode.next
        }
        return currentNode.element
    }

    this.addAt = function(index, element) {
        let node = new Node(element)
        let currentNode = head
        let previousNode
        let currentIndex = 0
        if (index > length) return false
        if (index === 0) {
            node.next = currentNode
            head = node
        } else {
            while (currentIndex < index) {
                currentIndex++
                previousNode = currentNode
                currentNode = currentNode.next
            }
            node.next = currentNode
            previousNode.next = node
        }
        length++
    }

    this.removeAt = function(index) {
        let currentNode = head
        let previousNode
        let currentIndex = 0
        if (index < 0 || index >= length) return null
        if (index === 0) {
            head = currentIndex.next
        } else {
            while (currentIndex < index) {
                currentIndex++
                previousNode = currentNode
                currentNode = currentNode.next
            }
            previousNode.next = currentNode.next
        }
        length--
        return currentNode.element
    }
}

4.コレクション（値）（セット）




コレクション（多く）は、数学の基本的な概念の1つです。明確に定義され、孤立したオブジェクトのセットです。ES6は、配列に似たコレクションを導入しました。ただし、コレクションでは重複する要素を含めることはできず、インデックスは含まれません。

標準コレクションには次のメソッドがあります。

  • 値：コレクション内のすべてのアイテムを返します
  • サイズ：要素の数を返します
  • has：アイテムがコレクション内にあるかどうかを確認します
  • 追加：アイテムを追加
  • 削除：アイテムを削除します
  • union：2つのコレクションの交差領域を返します
  • 違い：2つのコレクションの違いを返します
  • サブセット：あるコレクションが別のコレクションのサブセットであるかどうかを確認します

//   Set  JS
function MySet() {
    let collection = []
    this.has = function(element) {
        return (collection.indexOf(element) !== -1)
    }

    this.values = function() {
        return collection
    }

    this.size = function() {
        return collection.length
    }

    this.add = function(element) {
        if (!this.has(element)) {
            collection.push(element)
            return true
        }
        return false
    }

    this.remove = function(element) {
        if (this.has(element)) {
            index = collection.indexOf(element)
            collection.splice(index, 1)
            return true
        }
        return false
    }

    this.union = function(otherSet) {
        let unionSet = new MySet()
        let firstSet = this.values()
        let secondSet = otherSet.values()
        firstSet.forEach(i => unionSet.add(i))
        secondSet.forEach(i => unionSet.add(i))
    }

    this.intersection = function(otherSet) {
        let intersectionSet = new MySet()
        let firstSet = this.values()
        firstSet.forEach(function(e) {
            if (otherSet.has(e)) {
                intersectionSet.add(e)
            }
        })
        return intersectionSet
    }

    this.difference = function(otherSet) {
        let differenceSet = new MySet()
        let firstSet = this.values()
        firstSet.forEach(function(e) {
            if (!otherSet.has(e)) {
                differenceSet.add(e)
            }
        })
        return differenceSet
    }

    this.subset = function(otherSet) {
        lat firstSet = this.values()
        return firstSet.every(value => otherSet.has(value))
    }
}

5.ハッシュテーブル（ハッシュテーブル）




ハッシュテーブルは、キーと値に基づいて構築されるデータ構造です。キーによる値の検索が高速であるため、マップ、辞書、オブジェクトなどの構造で使用されます。図に示すように、ハッシュテーブルには、キーの名前（値）として使用される数値のリストにキーを変換するハッシュ関数があります。キー値の検索時間はO（1）に達する可能性があります。同一のキーは同じ値を返す必要があります-これがハッシュ関数の本質です。

ハッシュテーブルには次のメソッドがあります。

  • 追加：キー/値のペアを追加します
  • 削除：ペアを削除
  • ルックアップ：キーで値を検索

function hash(string, max) {
    let hash = 0
    for (let i = 0; i < string.length; i++) {
        hash += string.charCodeAt(i)
    }
    return hash % max
}

function HashTable() {
    let storage = []
    const storageLimit = 4

    this.add = function(key, value) {
        let index = hash(key, storageLimit)
        if (storage[index] === undefined) {
            storage[index] = [
                [key, value]
            ]
        } else {
            let inserted = false
            for (let i = 0; i < storage[index].len; i++) {
                if (storage[index][i][0] === key) {
                    storage[index][i][1] = value
                    inserted = true
                }
            }
            if (inserted === false) {
                storage[index].push([key, value])
            }
        }
    }

    this.remove = function(key) {
        let index = hash(key, storageLimit)
        if (storage[index].length === 1 && storage[index][0][0] === key) {
            delete storage[index]
        } else {
            for (let i = 0; i < storage[index]; i++) {
                if (storage[index][i][0] === key) {
                    delete storage[index][i]
                }
            }
        }
    }

    this.lookup = function(key) {
        let index = hash(key, storageLimit)
        if (storage[index] === undefined) {
            return undefined
        } else {
            for (let i = 0; i < storage[index].length; i++) {
                if (storage[index][i][0] === key) {
                    return storage[index][i][1]
                }
            }
        }
    }
}

6.ツリー




ツリー構造は、多層（マルチレベル）構造です。配列、スタック、キューとは異なり、これも非線形構造です。この構造は、要素の追加と検索に関して非常に効果的です。ツリー構造のいくつかの概念を次に示します。

  • root：ルート要素、親なし
  • 親ノード：最上位レイヤー（レベル）の直接ノード。1つしか存在できません
  • 子ノード：下位レベルの直接ノード（複数も可）
  • 兄弟：1つの親ノードの子
  • 葉：「子供」のいない結び目
  • エッジ：ノード間のブランチまたはリンク（リンク）
  • パス：開始ノードからターゲット要素へのパス（リンクのセット）
  • ツリーの高さ：特定の要素から子を持たないノードへの最長パスのリンクの数
  • ノードの深さ：ルートノードから特定の要素へのリンクの数。
  • ノードの次数：子孫の数

二分探索木（BST）の例を次に示します。各ノードには子孫が2つだけあり、左側（子）ノードは現在の（親）ノードよりも小さく、右側は大きくなります。



このツリーのメソッドは次のとおりです。

  • 追加：ノードを追加
  • findMin：最小ノードを取得
  • findMax：最大ノードを取得
  • find：特定のノードを見つけます
  • isPresent：特定のノードを確認します
  • 削除：ノードを削除します

class Node {
    constructor(data, left = null, right = null) {
        this.data = data
        this.left = left
        this.right = right
    }
}

class BST {
    constructor() {
        this.root = null
    }

    add(data) {
        const node = this.root
        if (node === null) {
            this.root = new Node(data)
            return
        } else {
            const searchTree = function(node) {
                if (data < node.data) {
                    if (node.left === null) {
                        node.left = new Node(data)
                        return
                    } else if (node.left !== null) {
                        return searchTree(node.left)
                    }
                } else if (data > node.data) {
                    if (node.right === null) {
                        node.right = new Node(data)
                        return
                    } else if (node.right !== null) {
                        return searchTree(node.right)
                    }
                } else {
                    return null
                }
            }
            return searchTree(node)
        }
    }

    findMin() {
        let current = this.root
        while (current.left !== null) {
            current = current.left
        }
        return current.data
    }

    findMax() {
        let current = this.root
        while (current.right !== null) {
            current = current.right
        }
        return current.data
    }

    find(data) {
        let current = this.root
        while (current.data !== data) {
            if (data < current.data) {
                current = current.left
            } else {
                current = current.right
            }
            if (current === null) {
                return null
            }
        }
        return current
    }

    isPresent(data) {
        let current = this.root
        while (current) {
            if (data === current.data) {
                return true
            }
            data < current.data ? current = current.left : current = current.right
        }
        return false
    }

    remove(data) {
        const removeNode = function(node, data) {
            if (node === null) return null
            if (data === node.data) {
                //  
                if (node.left === null && node.right === null) return null
                //   
                if (node.left === null) return node.right
                //   
                if (node.right === null) return node.left
                //   
                let tempNode = node.right
                while (tempNode.left !== null) {
                    tempNode = tempNode.left
                }
                node.data = tempNode.data
                node.right = removeNode(node.right, tempNode.data)
                return node
            } else if (data < node.data) {
                node.left = removeNode(node.right, data)
                return node
            } else {
                node.right = removeNode(node.right, data)
                return node
            }
        }
        this.root = removeNode(this.root, data)
    }
}

7.ロードされた（プレフィックス）ツリー（Trie、「try」と読みます）




接頭辞ツリーは検索ツリーの一種です。その中のデータは順次（段階的に）保存されます-各ツリーノードは1つのステップを表します。プレフィックスツリーは検索を大幅に高速化するため、辞書で使用されます。

各ツリーノードはアルファベットの文字であり、分岐に続いて単語が形成されます。また、現在のノードが最後の文字であることを決定するための「ブールインジケーター」も含まれています。

プレフィックスツリーには次のメソッドがあります。

  • add：辞書に単語を追加します
  • isWord：単語を確認する
  • 印刷：すべての単語を返す

function Node() {
    this.keys = new Map()
    this.end = false
    this.setEnd = function() {
        this.end = true
    }
    this.isEnd = function() {
        return this.end
    }
}

function Trie() {
    this.root = new Node()
    this.add = function(input, node = this.root) {
        if (input.length === 0) {
            node.setEnd()
            return
        } else if (!node.keys.has(input[0])) {
            node.keys.set(input[0], new Node())
            return this.add(input.substr(1), node.key.get(input[0]))
        } else {
            return this.add(input.substr(1), node.keys.get(input[0]))
        }
    }

    this.isWord = function(word) {
        let node = this.root
        while (word.length > 1) {
            if (node.keys.has(word[0])) {
                return false
            } else {
                node = node.keys.get(word[0])
                word = word.substr(1)
            }
        }
        return (node.keys.has(word) && node.keys.get(word).isEnd()) ? true : false
    }

    this.print = function() {
        let words = new Array()
        let search = function(node = this.root, string) {
            if (node.keys.size !== 0) {
                for (let letter of node.keys.keys()) {
                    search(node.keys.get(letter), string.concat(letter))
                }
                if (node.isEnd()) {
                    words.push(string)
                }
            } else {
                string.length > 0 ? words.push(string) : undefined
                return
            }
        }
        search(this.root, new String())
        return words.length > 0 ? words : null
    }
}

8.グラフ（グラフ）（グラフ）




ネットワークとも呼ばれるグラフは、相互接続されたノードのコレクションです。リンクには方向があるかどうかに応じて、2種類のグラフがあります-有向と無向です。グラフはあらゆる場所で使用されます。たとえば、ナビゲーションアプリケーションで最適なルートを計算したり、ソーシャルネットワークの推奨事項のリストを作成したりします。

グラフは、リストまたはマトリックスの形式で表示できます。

リスト


この場合、すべての親ノードは左側にあり、その子は右側にあります。



マトリックス


この場合、ノードは行と列に分散され、行と列の交差はノード間の関係を示します。0はノードが接続されていないことを意味し、1-ノードは接続されています。



グラフは2つの方法で検索されます-幅優先検索（Breath-First-Search、BFS）と深度深さ検索（Depth-First-Search、DFS）。

BFSを検討してください。

function bfs(graph, root) {
    let nodesLen = {}
    for (let i = 0; i < graph.length; i++) {
        nodesLen[i] = Infinity
    }
    nodesLen[root] = 0
    let queue = [root]
    let current
    while (queue.length !== 0) {
        current = queue.shift()

        let curConnected = graph[current]
        let neighborIdx = []
        let idx = curConnected.indexOf(1)
        while (idx !== -1) {
            neighborIdx.push(idx)
            idx = curConnected.indexOf(1, idx + 1)
        }
        for (let i = 0; i < neighborIdx.length; i++) {
            if (nodesLen[neighborIdx[i]] === Infinity) {
                nodesLen[neighborIdx[i]] = nodesLen[current] + 1
                queue.push(neighborIdx[i])
            }
        }
    }
    return nodesLen
}
