Windows Terminal Preview v0.10

Einführung in Windows Terminal v0.10! Wie immer können Sie es aus dem Microsoft Store oder von der Release-Seite auf GitHub herunterladen . Unter cat werden wir uns die Update-Details genauer ansehen!



Mauseingabe


Das Terminal unterstützt jetzt die Mauseingabe in Windows Subsystem für Linux (WSL) -Anwendungen sowie in Windows-Anwendungen mit VT-Eingabe (Virtual Terminal). Dies bedeutet, dass Anwendungen wie tmux und Midnight Commander Klicks auf Elemente im Terminalfenster erkennen! Wenn sich die Anwendung im Mausmodus befindet, können Sie die Umschalttaste gedrückt halten , um eine Auswahl zu treffen, anstatt eine VT-Eingabe zu senden.



Update Einstellungen


Vervielfältigung von Panels


Jetzt können Sie ein neues Bedienfeld öffnen, indem Sie ein Profil aus einem ausgewählten Bedienfeld duplizieren, sich nur darauf konzentrieren und eine Tastenkombination drücken. Dazu müssen Sie im Abschnitt "Tastenkombinationen" Ihrer profile.json "splitMode": "duplicate" zu "splitPane" hinzufügen . Sie können andere Parameter verwenden, z. B. "Befehlszeile" , "Index" , "Startverzeichnis" oder "TabTitle" . Wenn Sie mehr über diese Optionen erfahren möchten, empfehle ich Ihnen, diesen Artikel zu lesen .

{"keys": ["ctrl+shift+d"], "command": {"action": "splitPane", "split": "auto", "splitMode": "duplicate"}}




Fehler Korrektur


  • Deutlich verbesserte Textanzeige beim Ändern der Fenstergröße;
  • Die Ränder des dunklen Themas sind festgelegt (sie sind nicht mehr weiß).
  • Wenn die Taskleiste ausgeblendet und Ihr Terminal erweitert ist, wird sie automatisch angezeigt, wenn Sie den Mauszeiger über den unteren Bildschirmrand bewegen.
  • Azure Cloud Shell kann jetzt PowerShell starten und unterstützt Mauseingaben. Sie kann auch als bevorzugte Shell installiert werden.
  • Die Bildlaufgeschwindigkeit wurde bei Verwendung des Touchpads oder des Touchscreens geändert.

Zukunftspläne


Wir möchten Sie über unsere Pläne auf dem Laufenden halten, damit Sie wissen, was Sie in den kommenden Monaten erwartet. Wir arbeiten derzeit an Fehlerkorrekturen, um die Veröffentlichung von Version 1 vorzubereiten. Windows Terminal v1 selbst wird im Mai veröffentlicht. Danach planen wir, das nächste Update im Juni zu veröffentlichen, um unseren monatlichen Update-Zyklus fortzusetzen. Unsere Editionen werden weiterhin im Microsoft Store und auf GitHub verfügbar sein!

Abschließend

Wenn Sie einen Vorschlag zur Verbesserung des Terminals machen oder einen Fehler darin melden möchten, kontaktieren Sie uns bitte auf GitHub . Wir wünschen Ihnen viel Spaß mit dieser Version von Windows Terminal!
