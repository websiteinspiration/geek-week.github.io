ロシアでの社会的格付けの導入に備えましょう



現在、銀行は、顧客のスマートフォン上のアプリケーションによって収集されたデータに基づいて提案を作成しています。特に、提供されるローン金利は、デバイスでホストされているファイルとアクセスしたサイトに大きく依存します。そのため、銀行はインストールを永続的に提供しています（サービスを拒否し、必要なアクションを自分で実行する可能性を挙げて）。このため、銀行の顧客はデバイスに対する最大の権限を必要とします。たとえば、Sberbank Onlineは、デバイス上のウイルスをスキャンするためにファイルシステムにアクセスする必要があると主張しています。

社会的に有用な行動のためのローンで低金利を取得することは遠い将来の見通しであるとあなたに思われるかもしれません。しかし、今日、若い家族の住宅ローン率は10％前後ですが、残りの提案は20％近くです。それら。ずいぶん前のことですが、あなたはそれを重要視していませんでした。

ウラジミールプーチン大統領の最後の大統領任期中に、多くの不人気な決定が下されると思います。この人はもはや「顔を救う」必要はありません。したがって、2024年までに、ロシアでの社会的格付けの導入を含む、「ねじを締める」という新しいラウンドが見られます。


この出版物はスマートフォンの情報のプライバシーの問題を扱います。実際のところ、分析に便利な形式でユーザーに関する大量の情報を取得できるのはこれらのデバイスです。私は、デバイスに対する複雑で危険なアクションなしに、ほとんどのセキュリティホールを閉じるシンプルなソリューションを提案します。スマートフォンでのアクションのプライバシーをほぼ完全に保護するために必要な作業の20％を割り当てました。

予備措置


デバイスでプライバシーを取得する場合は、出荷時の設定に戻すことを強くお勧めします。長年の使用において、多数の「痕跡」がすでにその上に蓄積されており、現在の完全なセキュリティがあっても、個人を分析することができます。転送されたデータをアナログの紙に保存することをお勧めします。怠惰であるため、古いゴミの束を新しいスマートフォンにドラッグすることはできません。

残念ながら、最初の2つのポイントはroot権限を必要とするため、使用が非常に複雑になるため、ほとんどの場合、そのようなアクションは実行されません（不明なソースからのソフトウェアでデバイスをフラッシュして、ブリックに変えるリスクがあります）。そして、簡単な解決策を見つける必要があります。したがって、私たちは倫理的なアプリケーションに目を向けます。

町民は、コンピューター（およびスマートフォンは実際のコンピューターです！）に対する主な脅威はウイルスであると信じています。しかし、主な脅威は公式のソフトウェア（ソースコードがクローズされている）であり、ユーザーが自発的に任意のアクションを実行する権利を与えたため、これは長い間行われていませんでした。「アンチウイルス」はシグネチャ（コードの一部）による検索を使用するため、（新しいコードが作成されているため）予期しないアクションについて公式プログラムをキャッチすることはできません。このため、2020年7月以降、ロシアから販売されるすべてのクローズドスマートフォンには、ロシア語のクローズドソフトウェアがプレインストールされます（つまり、ほとんどのオープン企業がお客様の個人データを求めて戦っています）。専有ソフトウェアの反対は倫理的アプリケーションです。

2.不要なプレインストールプログラムを削除する（これにより、デバイスの速度も向上します）


このアイテムを実行するには、ADB（Android Debug Bridge）を使用します。最初に、デバイスでUSBデバッグを有効にする必要があります（原則として、設定でデバイスのバージョン番号に5-7のクイックポークが行われ、「USBデバッグ」フラグが設定されます）。必要なコンポーネントをPCにインストールする必要があります（これは、さまざまなOSで詳しく説明されています）この問題は、記事「Androidユーザー向けのAdbコマンド」で説明されています。次に、データケーブルを使用してデバイスをPCに接続し、接続されているデバイスのリストを取得します。

adb devices -l


結果が成功した場合は、パッケージの削除に進みます。

adb shell pm uninstall --user 0 __


キー--user 0は、IDが0のユーザーに代わって行動することを意味します。 rootユーザーから（デバイスのフラッシュによる複雑な操作なしで）。削除するプログラムは、パッケージ名の形式で指定する必要があります。プログラムとパッケージの名前の対応は、お使いのデバイスの設定で見つかった（関心の適用に関する追加情報を参照）、または使用することができますのAppインスペクタプログラムをからPlayストア。たとえば、次のアプリケーションをアンインストールしました。

概要


すべてのアクションを見てみましょう。多数の権利を持つ（PlayストアとADBの両方を使用して）クローズされたプログラムを削除し、最小限のアクセス許可を持つ倫理的な対応物に置き換えました。彼らは、アプリケーションがインターネット上のデータをマージすることを許可しないファイアウォールを提起しました。彼らは、サンドボックス内の必要だが煩わしいアプリケーションを分離し、識別子からソーシャルネットワークプロファイルの関連付けを解除しました。
