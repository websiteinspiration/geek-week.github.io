データストーリーの改善に役立つ5つの視覚化

この記事は、Python開発者コースの前に翻訳されました。



ストーリーテリングは、データ分析の専門家にとって最も重要なスキルの1つです。アイデアを伝達し、説得力をもって行うには、効果的なコミュニケーションを構築する必要があります。この記事では、古典的な理解を超えて、データストーリーをより美しく美しくすることができる5つの視覚化方法を紹介します。PythonのPlotlyグラフィックライブラリ（Rでも利用可能）を使用します。これにより、最小限の労力でアニメーションおよびインタラクティブな図を作成できます。

Plotlyの良いところ


Plotlyグラフはさまざまな環境に簡単に統合できます。Jupyterノートブックで適切に機能し、Webサイトに埋め込むことができ、ダッシュボードや分析アプリケーションを作成するための優れたツールであるDashと完全に統合されています。

はじめましょう


plotlyをまだインストールしていない場合は、次のコマンドでこれを行うことができます。 

pip install plotly

これで続行できます。

1.アニメーション


たとえば、特定のメトリックの進化を検討する場合など、私たちの仕事は時間データに関連していることがよくあります。plotlyのアニメーションは、たった1行のコードで時間の経過に伴うデータの変化を反映するのに役立つクールなツールです。



import plotly.express as px
from vega_datasets import data
df = data.disasters()
df = df[df.Year > 1990]
fig = px.bar(df,
             y="Entity",
             x="Deaths",
             animation_frame="Year",
             orientation='h',
             range_x=[0, df.Deaths.max()],
             color="Entity")
# improve aesthetics (size, grids etc.)
fig.update_layout(width=1000,
                  height=800,
                  xaxis_showgrid=False,
                  yaxis_showgrid=False,
                  paper_bgcolor='rgba(0,0,0,0)',
                  plot_bgcolor='rgba(0,0,0,0)',
                  title_text='Evolution of Natural Disasters',
                  showlegend=False)
fig.update_xaxes(title_text='Number of Deaths')
fig.update_yaxes(title_text='')
fig.show()

時間によるフィルタリングに役立つ変数があれば、ほとんどすべてのグラフをアニメーション化できます。散布図アニメーションの例：

2.サンバーストチャート


サンバーストチャートは、操作ごとにグループを視覚化する優れた方法です。利用可能なデータ量を1つ以上のカテゴリ変数に分割する場合は、サンバーストチャートを使用します。

性別と時間帯ごとにチップ分布を取得する必要があるとします。そのため、group by演算子を2回使用して、通常のテーブル出力が表示されないように受信データを簡単に視覚化できます。



図はインタラクティブで、カテゴリをクリックして、各カテゴリを個別に表示できます。あなたがしなければならないのは、これらのカテゴリを決定し、それらの間の階層を検討することです（引数parentsコード内で）、適切な値を割り当てます。この場合は、演算子によるグループの出力になります。

import plotly.graph_objects as go
import plotly.express as px
import numpy as np
import pandas as pd
df = px.data.tips()
fig = go.Figure(go.Sunburst(
    labels=["Female", "Male", "Dinner", "Lunch", 'Dinner ', 'Lunch '],
    parents=["", "", "Female", "Female", 'Male', 'Male'],
    values=np.append(
        df.groupby('sex').tip.mean().values,
        df.groupby(['sex', 'time']).tip.mean().values),
    marker=dict(colors=px.colors.sequential.Emrld)),
                layout=go.Layout(paper_bgcolor='rgba(0,0,0,0)',
                                 plot_bgcolor='rgba(0,0,0,0)'))

fig.update_layout(margin=dict(t=0, l=0, r=0, b=0),
                  title_text='Tipping Habbits Per Gender, Time and Day')
fig.show()

次に、別の階層レベルを追加します。



これを行うには、別のグループbyの結果を追加します。そこからさらに3つのカテゴリーが取得されます。

import plotly.graph_objects as go
import plotly.express as px
import pandas as pd
import numpy as np
df = px.data.tips()
fig = go.Figure(go.Sunburst(labels=[
    "Female", "Male", "Dinner", "Lunch", 'Dinner ', 'Lunch ', 'Fri', 'Sat',
    'Sun', 'Thu', 'Fri ', 'Thu ', 'Fri  ', 'Sat  ', 'Sun  ', 'Fri   ', 'Thu   '
],
                            parents=[
                                "", "", "Female", "Female", 'Male', 'Male',
                                'Dinner', 'Dinner', 'Dinner', 'Dinner',
                                'Lunch', 'Lunch', 'Dinner ', 'Dinner ',
                                'Dinner ', 'Lunch ', 'Lunch '
                            ],
                            values=np.append(
                                np.append(
                                    df.groupby('sex').tip.mean().values,
                                    df.groupby(['sex',
                                                'time']).tip.mean().values,
                                ),
                                df.groupby(['sex', 'time',
                                            'day']).tip.mean().values),
                            marker=dict(colors=px.colors.sequential.Emrld)),
                layout=go.Layout(paper_bgcolor='rgba(0,0,0,0)',
                                 plot_bgcolor='rgba(0,0,0,0)'))
fig.update_layout(margin=dict(t=0, l=0, r=0, b=0),
                  title_text='Tipping Habbits Per Gender, Time and Day')

fig.show()

3.パラレルカテゴリ


カテゴリ間の関係を視覚化する別の良い方法は、この並行カテゴリチャートを使用することです。外出先で値をドラッグ、選択、取得できるため、プレゼンテーションに最適です。



import plotly.express as px
from vega_datasets import data
import pandas as pd
df = data.movies()
df = df.dropna()
df['Genre_id'] = df.Major_Genre.factorize()[0]
fig = px.parallel_categories(
    df,
    dimensions=['MPAA_Rating', 'Creative_Type', 'Major_Genre'],
    color="Genre_id",
    color_continuous_scale=px.colors.sequential.Emrld,
)
fig.show()

4.平行座標


平行座標図は、上記のグラフを拡張したものです。ここでは、グラフの各部分が1つの観測を反映しています。これは、外れ値（残りのデータから分離された単一のストリーム）、クラスター、傾向、および冗長データを検出するための優れたツールです（たとえば、2つの変数がすべての観測値で同じ値を持っている場合、それらは水平線上にあり、冗長性の存在を示します）。



import plotly.express as px
from vega_datasets import data
import pandas as pd
df = data.movies()
df = df.dropna()
df['Genre_id'] = df.Major_Genre.factorize()[0]
fig = px.parallel_coordinates(
    df,
    dimensions=[
        'IMDB_Rating', 'IMDB_Votes', 'Production_Budget', 'Running_Time_min',
        'US_Gross', 'Worldwide_Gross', 'US_DVD_Sales'
    ],
    color='IMDB_Rating',
    color_continuous_scale=px.colors.sequential.Emrld)
fig.show()


5.チャート、センサー、インジケーター




美学のためにセンサー図が必要です。これらは、成功またはパフォーマンスの指標を報告し、それらを目標に関連付けるための良い方法です。



指標は、ビジネスおよびコンサルティングのコンテキストで非常に役立ちます。それらは聴衆の注意を捕らえ、聴衆に成長指標を放送するテキストで視覚効果を補足します。

import plotly.graph_objects as go
fig = go.Figure(go.Indicator(
    domain = {'x': [0, 1], 'y': [0, 1]},
    value = 4.3,
    mode = "gauge+number+delta",
    title = {'text': "Success Metric"},
    delta = {'reference': 3.9},
    gauge = {'bar': {'color': "lightgreen"},
        'axis': {'range': [None, 5]},
             'steps' : [
                 {'range': [0, 2.5], 'color': "lightgray"},
                 {'range': [2.5, 4], 'color': "gray"}],
          }))
fig.show()

import plotly.graph_objects as go
fig = go.Figure(go.Indicator(
    title = {'text': "Success Metric"},
     mode = "number+delta",
    value = 300,
    delta = {'reference': 160}))
fig.show()


fig = go.Figure(go.Indicator(
     title = {'text': "Success Metric"},
    mode = "delta",
    value = 40,
 delta = {'reference': 160}))
fig.show()
