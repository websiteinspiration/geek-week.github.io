Google PageSpeedを100以上に加速



Google PageSpeedやLighthouseの指標を詳しく調べなくても、ウェブサイトを最大化できるシンプルで便利なヒント。

Javascript

  • JavaScriptの代わりに、HTML5タグをより頻繁に使用するようにしてください：<input type = date>、<input type = time>、Details / Summaryなど。

    ブラウザーに組み込まれている、さまざまなデバイスでより適切に機能し、追加のJavaScriptコードを必要としない既製のソリューションについて忘れたり、知らなかったりすることがよくあります。
  • JavaScriptコードの縮小を使用します。
  • すべてのJavaScriptコードをフッターに入れるか、読み込みを遅らせてください。<head>に含まれるJavaScriptが少ないほど、Google PageSpeedのパフォーマンスが向上します。
  • <script src = "path-to-file">には、サーバーへの追加のアクセスが必要です。これに注意してください。

    JavaScriptファイルの代わりに、ファイルの内容をフッターに挿入します。

    <script>   </script>

    そうすると、最初のダウンロードは速くなりますが、JavaScriptファイルのキャッシュはありません。
