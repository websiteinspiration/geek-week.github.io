Como mudar de qualquer provedor de host com cPanel para Plesk em Rusonix em apenas cinco etapas

Acontece que as pessoas mudam seus provedores de hospedagem. Isso é feito por vários motivos: alguns simplesmente não estão satisfeitos com a velocidade ou a qualidade dos serviços, outros encontram a mesma qualidade, mas a um preço mais baixo. Outros ainda não gostam de painéis de controle ...

Se você estiver nessa situação, agora é a hora de transferir seus sites e projetos para a Rusonix .

No entanto, mover o site pode parecer uma tarefa assustadora. Neste guia, explicaremos um procedimento simples com instruções passo a passo para que você possa transferir com confiança e facilidade todos os seus dados para os servidores Rusonix .

A transferência do seu site pode ser feita em apenas cinco etapas:

Etapa 1. Carregando arquivos do site de um antigo provedor de host

Usando o gerenciador de FTP ou gerenciador de arquivos do seu provedor de hospedagem atual, copie os arquivos do site para o seu computador. Certifique-se de fazer um backup em uma mídia independente.

Etapa 2. Exportando o banco de dados do seu site

Como você está usando o cPanel, exportar bancos de dados não será uma tarefa difícil. Para fazer isso, use o phpMyAdmin, que é parte integrante de todo cPanel. Não se assuste se vir nomes estranhos de arquivos e bancos de dados: mesmo que você não entenda tudo, você ainda pode fazer isso sozinho.

  • Localize a guia onde os bancos de dados estão ocultos e abra-a.
  • Clique no ícone phpMyAdmin para abrir a interface.
  • Encontre e clique no botão "Exportar".
  • Selecione "método rápido - mostre apenas os parâmetros mínimos".

Etapa 3. Baixar arquivos do site

Abra o painel Plesk no Rusonix e vá para a seção "Sites e domínios". Clique no botão "Download" e selecione um arquivo salvo no seu site. Observe que o painel Plesk suporta apenas arquivos no formato zip.

Selecione o arquivo salvo no computador e clique no botão "Extrair arquivos".

Etapa 4. Importando bancos de dados

Como na exportação, você usará o MySQL para importar os bancos de dados. Na verdade, não é tão difícil. Você também precisa inserir o phpMyAdmin, encontrar o botão "Importar" e clicar nele. Em seguida, basta selecionar o arquivo sql salvo no seu computador e importá-lo.

Etapa 5. Mude o DNS

Este é um pequeno detalhe fácil de perder. Após a transferência do site, você deve mudar o registro DNS do domínio do servidor anterior para o servidor Rusonix. Se isso não for feito, os usuários serão redirecionados para a página com um erro. O novo DNS é fácil de aprender em suporte ao Rusonix.

Depois de enviar uma solicitação para mover registros DNS, pode levar várias horas e, em alguns casos, um dia inteiro para trocá-los.

Se, por algum motivo, você não conseguir, esses. O suporte ao Rusonix transferirá independentemente todos os seus dados para o servidor .

Você não está satisfeito com seu provedor de hospedagem atual? Não perca seu tempo - escolha Rusonix! Aqui você encontrará não apenas uma infraestrutura poderosa e rápida, mas também será agradavelmente surpreendido pelo profissionalismo e pela atmosfera calorosa e amigável do suporte técnico.
