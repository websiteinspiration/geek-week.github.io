使用WSL在Windows 10上安装和配置RTKRCV（RTKLIB）

您好，本文讨论了由Tomaka Takasu（日本）团队开发的RTKLIB开放库安装和运行RTKRCV应用程序的过程。

RTKLIB是用于使用GNSS技术（全球导航卫星系统）进行标准和精确定位的开源软件包。

本文适用于谁？


对于所有希望将精确定位集成到其设备中的人来说，该材料都具有实际重要性，并且还可以作为优化RTKLIB库提供的软件产品工作的参考。本资料中的信息旨在优化Windows 10系统上专家与CUI的交互。

首先，让我们弄清楚什么是RTKRCV。 RTKRCV是RTKNAVI软件产品的CUI类似物，旨在处理导航数据。 AP实时接收来自GPS / GNSS接收器的原始观测数据，并执行实时导航处理。

解决大地测量学领域的工程问题时，可以使用RTKRCV的控制台模拟。对于此CUI，该库的说明中有关于此产品使用的相当完整的信息。

要开发自己的后处理方案或解决任何任务，在Linux和Windows之间切换会带来极大的不便。至少，您必须具有单独的系统或使用虚拟机，这也很不方便。

最终的结果是Windows 10中对Linux内核上的子系统的支持。

用于配置子系统和启动RTKRCV的算法：

  1. 安装WSL（Ubuntu，Debian等）；
  2. 在WSL上安装必要的实用程序；
  3. 配置ssh-key（PC <-> GitHub）;
  4. 使用RTKLIB获取存储库；
  5. RTKRCV安装；
  6. 配置配置文件；
  7. 启动RTKRCV。

1.安装WSL（ubuntu）


开始安装之前，必须使用管理员权限运行PowerShell并运行以下命令：

Enable-WindowsOptionalFeature -Online -FeatureName Microsoft-Windows-Subsystem-Linux

出现提示时，请重新启动计算机。

要下载并安装您的首选发行版，您有三个选择：

  • 从Microsoft Store下载并安装；
  • 使用命令行或脚本下载并安装；
  • 下载，然后手动解压缩并安装。

我们将从Microsoft应用商店中安装发行版。


将使用Ubuntu 18.04 LTS发行版检查此示例。

在PowerShell中安装配置并下载分发包之后，请运行分发包并等待其完全安装。



安装完成后，系统将提示您创建一个用户帐户（及其密码）。

2. WSL


由于RTKLIB库是用C / C ++编写的，因此我们需要安装make来构建应用程序和一组gcc编译器。

sudo apt install make
sudo apt install gcc
