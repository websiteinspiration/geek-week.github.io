Firefox 74リリース




世界で2番目に人気があり、フリーソフトウェアで最初のブラウザであるFirefox 74のメジャーリリースが行われました。OpenNETによる革新の一部：

  1. GNU / Linux用のアセンブリは、サードパーティライブラリの脆弱性をブロックすることを目的としたRLBox分離メカニズムを使用します。
  2. DNS over HTTPSは、米国のユーザーに対してデフォルトで有効になっています。
  3. TLS 1.0およびTLS 1.1プロトコルの無効なサポート。
  4. リリースノートでは、Facebookコンテナーアドオンを推奨し、プライバシーを向上させるためにウィジェットを使用するサイトからメインのFacebookサイトを分離しました。
  5. 回避策によってインストールされ、ユーザープロファイルに関連付けられていないアドオンのサポートは終了しました。
  6. WebRTCは、音声およびビデオ通話中の内部IPアドレス情報の漏洩に対する保護を強化しています。
  7. WindowsおよびmacOSでは、Chromiumエンジンに基づいてMicrosoft Edgeブラウザーからプロファイルをインポートする機能が実装されています。

Firefox 74の革新とバグ修正に加えて、20の脆弱性が修正されました。

Memcached 1.6.0




OpenNETは、キー/値データで動作し、その使いやすさを特徴とするMemcached 1.6.0 RAMデータキャッシュシステムの重要なリリースをリリースしました。新しいバージョンでは、extstoreストレージの実装が安定し、SSD /フラッシュドライブを使用してキャッシュサイズを拡張できるようになりました。新しいバージョンの2番目の重要な変更は、ネットワーク相互作用コードの処理でした。これは、単一のシステムコール内でのバッチコールの自動処理に適合されています。また、ネットワークサブシステムをリサイクルすることで、バッファの静的割り当てではなく、必要に応じて動的バッファ割り当てに切り替えることができました。また、サーバーと対話するための古いバイナリプロトコルのカテゴリへの移行を発表しました。バイナリプロトコルは新しいテキストプロトコルに置き換えられました。パフォーマンスと信頼性の最適な組み合わせを示し、以前はバイナリプロトコルとテキストプロトコルで利用できたすべての操作をカバーします。古いテキストプロトコルは変更されずに引き続き開発されます。

SIFTオブジェクト認識アルゴリズムの特許が失効しました




2020年3月8日、画像内の標識を識別するために設計されたSIFT技術（Scale Invariant Feature Transform）を説明する特許US6711293B1の期限が切れました。SIFTは、画像内のオブジェクトの認識、拡張現実システムでの実際の画像への3Dモデルの重ね合わせ、マップマッチング、3D空間での配置、パノラマステッチなどの領域に適用できます。以前に商用プロジェクトでSIFTを使用するためにライセンスまたは許可を取得する必要があった場合は、今では誰でも利用できるようになりました。SIFT実装はOpenCVで提供されていますが、個別の組み込みを必要とするモジュールの非フリーセットとして分類されています。特許の満了により、OpenCVの主要構成にSIFTを転送でき、無料のプロジェクトでのパターン認識にSIFTを制限なく使用できます。
