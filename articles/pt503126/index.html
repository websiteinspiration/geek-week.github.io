- Por que isso é necessário?

O caso mais comum é que o banco de dados contábil está na empresa e não no computador doméstico do contador. Se um contador ficar doente ou algo mais acontecer com ele, será difícil conseguir tudo. Os casos mais raros são quando você precisa ter seu ambiente de trabalho no telefone de qualquer lugar do mundo ou quando usa frequentemente os computadores de outras pessoas.

Também é útil para criar servidores de jogos, negociar na bolsa de valores, torrents, vários processos repetitivos, programas para automatizar redes sociais (e como trapaceiros, o que já existe), utilitários de SEO, se preparar para a contabilidade branca, viajar (especialmente quando você não deseja leve seu computador doméstico para uma cidade vizinha por um mês) e assim por diante. Mais frequentemente, o VPS é usado para a segurança do banco de clientes da empresa.

- Como configurar pela primeira vez?

A lógica por trás do levantamento de um servidor de área de trabalho remota é implantar uma máquina virtual (VPS) e instalar uma distribuição do Windows que ofereça suporte à conectividade remota. O primeiro é feito a partir da conta pessoal de hospedagem VPS (leva de quatro a sete minutos com SLA por 15 minutos), o segundo pode fazer você aprender bem o material, mas, no nosso caso, o Windows Server está no mercado e você pode solicitar uma máquina imediatamente com um sistema operacional expandido sistema. O acesso à conexão estará diretamente na sua conta. Resta apenas conectar.

- Quais são os requisitos de sistema para o Windows Server?

A configuração mínima é de 2 GB de RAM, 20 GB no disco rígido. Para o Win Server 2003, basta um núcleo; para o Win Server 2016 ou 2019, dois já são necessários. É melhor adicionar um pouco mais de espaço no disco rígido, ambos os sistemas operacionais terão 20 GB - já o S2012, por exemplo, não poderá atualizar. Alguns usuários instalam o Win no VPS com 500 MB de RAM. Recomendamos uma tarifa por 579 rublos por mês (isso não inclui descontos para a Habr, você pode discar 15% do código promocional abaixo e outros 20% ao pagar pelo ano), onde há um núcleo, 2 GB de RAM e 20 GB de disco rígido. A taxas mais baratas, é melhor colocar o Windows Server Core ou outro sistema operacional.

- É possível configurar backups agendados?

Sim, você pode tirar uma cópia da máquina VPS, por exemplo, uma vez por dia. Configure através da sua conta. Uma cópia é feita em um armazenamento independente no mesmo datacenter em que seu VPS é implantado. Se você deseja copiar para outro data center, precisará configurá-lo manualmente.

- E se o Windows do outro lado parar de responder?

Se algo acontecer com o sistema operacional ou o serviço de acesso ao terminal, você não poderá se conectar via RDP, pois isso requer um sistema operacional ativo, drivers de rede, uma conexão de rede ativa e o RDP ativado. Fora da caixa, temos tudo. Mas se o Win "quebrar", será possível usar mais um modo especial: temos a capacidade de transmitir vídeo diretamente do nível do hypervisor e transmitir pressionamentos de tecla e cliques no hypervisor. Este modo é bastante exigente em recursos, portanto, há um limite em seu uso. Mas em caso de problemas, você pode usá-lo para reiniciar no modo de segurança. Se o problema for mais sério - você pode entrar em contato com o suporte ou simplesmente reinstalar o Windows com apenas alguns cliques.

O modo de emergência pode ter uma resposta bastante longa às ações, mas visualmente parece bastante usual. É tão comum que tenhamos usuários que o usem regularmente. Mas absolutamente não precisamos ser usados ​​na vida cotidiana fora de uma emergência, porque existem maneiras mais corretas e confortáveis ​​para o Windows - é claro, esse é o RDP.

- E a licença do sistema operacional?

Normalmente, a hospedagem VPS oferece um servidor virtual e não analisa o que você está fazendo com ele (apenas para fazer o download por recursos e pela natureza do tráfego). Ou seja, em teoria, você pode colocar qualquer software lá, incluindo software pirata. Mas, na prática, é mais conveniente criar uma máquina com um sistema operacional usando um dos modelos de hospedagem. No nosso caso, uma licença para o Windows Server (ou Windows Server Core sem interface gráfica) já está incluída no plano tarifário, pois todos nós temos hospedagem no Hyper-V e é totalmente licenciada. Algumas outras empresas de hospedagem neste momento abrem uma nova "estrela" com o custo mensal do aluguel de uma licença.

- Posso trazer minha própria imagem do sistema operacional?

Sim, você pode, e há duas opções. Você pode criar um contêiner VHDX padrão para Hyper-V e transferi-lo para o suporte. Nesse caso, pode haver dificuldades no estágio de configuração de uma conexão de rede - talvez na sua imagem finalizada não haja driver para uma placa de rede adequada. Você precisará conectar e instalar. No entanto, essa é uma boa maneira de transferir tudo, por exemplo, de um laptop antigo para online. Ou você pode transferir o suporte para a imagem ISO, montada em um servidor virtual, e da qual você, usando o modo de emergência por meio de sua conta pessoal, pode executar a instalação sozinho.

Esses serviços são fornecidos apenas em planos tarifários de 1.000 rublos por mês.

- E as licenças para outro software?

A hospedagem geralmente não vê o que está instalado em você (não vemos, por exemplo, a menos que você solicite suporte para algo especial). Não nos importamos com o que existe e como funciona, pois essa é sua área de responsabilidade. Lembre-se de que o suporte para software pirateado não é fornecido e se algo não funcionar.

- Que armazenamento é usado?

Todas as unidades são prateleiras SSD em servidores, mas nas tarifas com HDDs elas são limitadas em velocidade às velocidades dos discos rígidos de eixo. Duplicação - RAID 5. Existem vários RAID 6 restantes, mas já existem alguns. Se uma das unidades falhar, a reconstrução do ataque começará. Se ocorrer uma falha de energia nesse ponto, os dados serão perdidos. Nesse caso, temos duas fontes de alimentação separadas com rotas de cabos separadas para cada servidor físico e um backup técnico de toda a máquina (sem acesso seletivo a um armazenamento separado). Os danos causados ​​por ataques de hackers estão garantidos na AIG.

- Que sobrecarga um hipervisor impõe ao desempenho?

Com uma combinação do Hyper-V + Windows, significativamente menos de um por cento.
