今週の結果：GoogleはAIで成功していますが、Stadiaではうまくいきません。Durovが訴えられ、農家がVRをテストしています



ハブレに関する先週の結果。このダイジェストでは、11月22日から29日まで話し合った最も重要で興味深く注目度の高いイベントです。Googleは「説明可能な」AIを開発しましたが、Stadiaはまだ離陸せず、翻訳者ツールキットは間もなく終了します。科学者はおそらく強誘電体コンデンサをベースにした非常に信頼性の高いドライブに近く、Visionoxディスプレイは丸めることができ、Web上で、パブリックドメインに12億レコードのElasticsearchサーバーが見つかりました。米国では、Durovが法廷に召喚され、ロシアでは、国外のユーザーデータを保存する場合の罰金の最大額が決定され、Samsungは30のスマートフォンモデルをAndroid 10に更新します。

上


1. Google は「説明可能な」AI（説明可能なAI）を示した


同社はロンドンにAIを導入しました。これにより、データ処理中に機械学習モデルがどのように結論に至ったかを正確に理解することができます。アルゴリズムは、機械学習モデルの出力に対する各データ要素の寄与を定量化する特別な説明モジュールを使用して拡張されました。ちなみに、この情報はAIモデルをさらに適応させるために使用できます。

2.強誘電体コンデンサの電位分布を測定


MIPTの科学者は、アメリカ人およびドイツ人の同僚とともに、いわゆる強誘電性コンデンサー内の電位分布を測定することができました。これらの酸化ハフニウムコンデンサは、将来的にストレージデバイスとして使用される可能性があります。新世代のデバイスは、動作の信頼性が高くなり、従来のフラッシュドライブよりも数十万倍も多くの書き換えサイクルに耐えることができます。それまでの間、HPはHPE SSDのファームウェアパッチを公開しています。約3年と9か月の作業におけるデータ損失とディスク障害の重大な問題を解決するように設計されています。このパッチは、バージョンHPD8までのファームウェアを備えたHPE SAS SSDドライブ用にリリースされました。更新は重要であり、データを保存する場合に必要です。

3. AMOLEDはチューブに折りたたむことを学びました


Xiaomi Mi Mix AlphaとXiaomi Mi Note 10のディスプレイを作るVisionoxは、新しい画面のプロトタイプを示しました。新しいAMOLEDフレキシブルパネルがロールに巻き上げられます。市場の巨人が新しいフォームファクターに関心を持つかどうかは明らかではありません。これが将来の主力製品であるSamsung Galaxy S11です。どうやら、通常の画面は6.63インチから6.7インチで、側面が曲がっています。しかし、Google はアンビエントモードを発表しましたAndroidスマートフォンとタブレットをスマートディスプレイに変える音声アシスタント用。このモードは、ユーザーがデバイスを充電するときにアクティブになります。ロックされた画面の一部の機能（Googleフォトアルバムの写真、カレンダーのリマインダー、音楽コントロールパネル、接続されているスマートホームデバイス、Googleアシスタントなど）にすばやくアクセスできます。

失敗


1.パブリックドメインの10億人のユーザーのデータ


情報セキュリティの専門家であるVinnie TroyとBob Dyachenko が、ネットワークに合計4 TBの12億（！）のレコードを持つオープンElasticsearchサーバーを発見しました。Google Cloud Serviceサイトにあります。サーバーは10月4日にパブリックドメインに登場しました。TroyとDyachenkoは、直接収集された、またはソーシャルネットワークとプラットフォーム（Facebook、Twitter、Github、LinkedIn）のプロファイルから取得された、数億人のパブリックデータを含む4つのデータベースのコレクションを発見しました。また、サーバーには約5,000万の一意の電話番号と6億2,200万の一意の電子メールアドレスが格納されていました。

2. GoogleはStadiaで4K画像を提供していません


Googleは、Stadiaストリーミングサービスが、信頼性の高いインターネット接続とStadia Proへのサブスクリプションにより、4K品質および60フレーム/秒でビデオゲームをプレイできると約束しました。ただし、多くのユーザーは今日、Stadiaの一部のゲームはこれらの指標に達しないと述べています。そのため、StadiaのゲームRed Dead Redemption 2をレビューすると、ゲームは実際には2560x1440の解像度で動作し、Chromecast Ultraを使用して4Kにスケーリングされることがわかりました。Googleは責任をStadia開発者に移しました：「私たちは開発者がStadiaでゲームを改善できることを望んでおり、将来的にそうするでしょう。」

3.デュロフは証言するために米国の裁判所に召喚された


ニューヨーク南部地方裁判所は、10月に停止されたグラムデジタルトークンの問題について証言するために、パベルデュロフ、テレグラム副社長のイリヤペレコプスキー、および会社の従業員シャムパレフを召喚しました。彼らは1月7日または8、2020に表示されます「当事者間で合意された場所」に。ペレコプスキーとパレフの尋問は、12月16日と10日にロンドンで開催されます。1か月後、SECが未登録のセキュリティと見なしているグラムトークンの運命は、テレグラムと米国証券取引委員会の間の裁判所で決定されます。

待っている


1.翻訳キットは間もなく終了します


12月4日以降、Google はサービス翻訳ツールまたはGoogle翻訳ツールキットを閉鎖します。ユーザーはこの時点までデータをアップロードできます。サービスの終了後しばらくしてから、データをGoogleデータエクスポートからダウンロードすることもできます。

2.ロシア連邦外のユーザーデータを保存する場合の罰金のサイズを決定


議会は、このデータをロシア国外に保管するロシア人の個人データを収集する事業者に対する罰金の額を修正する法案を承認しました。法に繰り返し違反する法人の最大額は、1800万ルーブルになります。この文書は署名のために大統領に引き渡され、その後直ちに発効します。

3. Samsungは30台のスマートフォンを最新のAndroid 10にアップグレードします


サムスンのイスラエル部門は、同社が約30モデルのスマートフォンを、改良されたオペレーティングシステムAndroid 10のファームウェアOne UI 2.0 にアップデートすることを計画していると語った。1月にGalaxy S10シリーズ、Galaxy Note 10シリーズ、Galaxy Note 9シリーズを初めてアップデート。4月には、新しいOSにスマートフォンシリーズGalaxy AとGalaxy J、Galaxy S9、および一部のタブレットGalaxy Tabが搭載されます。

楽しい：農家がVRを試す


モスクワ地方の農場の1つが、牛の生産性に対する仮想現実の影響を研究する実験を行いました。開発者は人間のVRメガネを牛の頭に合わせ、動物用の夏の野外シミュレーションプログラムを作成しました。VRメガネのテスト中に、専門家は牛の不安が減少し、群れの全体的な感情的な気分が改善したことに気づきました。VR技術が乳量に与える影響を調査する必要があります。

フィクション：小さな町で開発者を育てる方法


ブリャンスクの開発者は、小さな町に住んでいる間に、いくつかのプログラミングブックをリリースし、外国のポッドキャストのゲストになり、スピーカーとしてさまざまな会議に参加する方法についての経験を共有しました。
