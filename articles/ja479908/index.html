コンテキストから始めましょう。

過去18か月間、AppleがAR / VR（拡張現実/仮想現実）デバイスを開発しているという噂が続いています。コンテキストについては、MicrosoftからHolosLensを使用して、FacebookからOculusを使用して、Googleからラフな段ボールクラフトまたはMagic Leapの野心的なデバイスを使用して冒険を見ることができます。これはすべて、AppleのレースAR / VRへの参加に必然性の光輪を与えました。コメンテーターは、申し立てられたこれらのデバイスを会社の次の製品の波に乗せざるを得ませんでした。彼らの観点から見ると、飽和状態の市場でiPhoneが経験した数年の問題の後、スマートグラスはついにAppleのデフレ資本を元気づけるでしょう。悪名高いミンチコの予測の絶え間ない流れの例に続いて、誰もが同意したAppleのAR / VR製品は2020年に、おそらく後半に発売される予定です。

そして、異常なことが起こりました。

どうやら、Appleは聴衆の中で魔法の眼鏡のプロジェクトに関係しているすべての従業員と会議を開いた。スティーブ・ジョブズ。そこでは、数百人の群衆（何千人と言う人もいる）の前で、マイクロックウェル副大統領が作業スケジュールを発表しました：AR / VRハイブリッドヘッドセット（コードネームN301）は2022年にリリースされ、N421メガネのライトバージョンのみがリリースされます

すべての従業員が集まること自体は、それほど珍しいことではありません。しかし、その本質は、特に2022/2023に関する声明に照らして、AR / VR市場に参入するためにAppleが実行する必要があるすべてのステップを検討することで、よりよく理解できます。

しかし、ソフトウェアとハ​​ードウェアの領域から必要なコンポーネントの問題を掘り下げる前に、コンテキストが必要です。 Appleデバイスは、垂直ではなく水平です。つまり、小さなユーザーグループ向けの他の高度に専門化されたツールとは異なり、幅広いデバイスをカバーします。言い換えれば、Appleは外科医用のVRメガネやロボット用のスマートメガネを製造しません。

これらすべてを前提として、最初の必要なコンポーネントを想像することができます。仮想現実の世界から私たちに説得力のある幻覚を与えることができるCPU / GPUの組み合わせです。今日、最も魅力的なVRキットは、強力なPCに接続された眼鏡またはヘルメットで構成されています。 Appleメガネは、何らかの方法でコンピューターから切断し、独自のコンピューティングエンジンをApple Watchと同じ方法で携帯電話から切断する必要があります。

デバイスが広く認識されるためには、よく知られているが深刻な問題が発生します。CPU/ GPUは小さなバッテリーと連携して動作する必要があります。大まかに言えば、今日のiPhoneよりも優れた計算能力とグラフィックス能力を備えている必要がありますが、1日中動作します。これは言うよりも簡単であり、この要因だけでもこのような長いリリース日を説明できます。

その後、まだ鍵穴の問題があります。Googleのスマートグラスを使った実験に関連する社会的なぎこちないさと虐待を思い出すことができます。カメラを備えたスマートグラスをかけたテーブルに座っている私を見ると、ビデオと会話の音声を記録できることを知っていると、どれほど快適に感じるでしょうか。カメラなしでスマートグラスを使用すること（ただし、可能性は制限されます）、またはフレーム上でのカメラの操作のインジケーター、および個人的な会話の記録の受け入れまたは拒否に関連する社会的習慣の進化を想像できます。簡単な移行ではありません。おそらく、それは市場に参入する以前のプレーヤーによって促進されるでしょう。

これは、アプリケーション開発者の観点から最もよく研​​究された使用例につながります。スマートグラスとそれらの複合現実アプリケーション（コンピューターで生成された画像は半透明のグラスに投影され、距離を見るのに便利になるように焦点が合わせられます）の場合、開発者はより創造的なアプローチが必要になります。退屈な会議でメガネにメッセージやメールがどのように表示されるか、あるいは通りにいる人が外出先で本を読む方法を想像することができます。個人的には、ルーヴル美術館を訪れ、エリザベス・ヴィジー・レブルンの絵画技法についてコメントを得たい。 VR / ARの新しいユースケースセットについては、開発者はiOSが現在持っていないプラットフォームが必要になります。そして、これらのプラットフォーム自体には、スマートフォンや時計が提供するものとは根本的に異なる対話を処理するためのiOSプラットフォームが必要になります。

この一連の困難は難攻不落のようであり、賞がユーザーの頭であるAR / VRレースの既存の参加者の誰も前例のないことです。マイクロソフトには、起業家を対象とした$ 3,500のHoloLens 2スイートがあります。 Mark Zuckerbergは、Facebookが仮想現実に賭けた$ 20億の賭けは、以前のように早くは打ち負かされなかったことを認めています。 GoogleはDayDream VRプラットフォームのサポートを段階的に廃止します。 26億ドルの投資を行ったMagic Leapは、より多くの資金を集めようとしています。JPモルガンチェースに既存のポートフォリオを構築する必要がありました。これは、2295ドルでCreator Editionデバイスで革命を起こすことができなかったためです。 Appleは、その開発に関する情報を誰とも共有せずに、AR / VR製品または消費者品質の製品ラインへの同じ道をたどることが困難であることは論理的です。AppleのCEOであるTim Cookは最近の困難についてコメントしました。
