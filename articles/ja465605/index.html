抜粋。LSTM RNNを使用した歌詞の生成


LSTMを使用してZayn Malik歌詞を生成する方法を見てみましょう。Zaneの歌詞歌詞データセットは、https：//github.com/sudharsan13296/Hands-On-Reinforcement-Learning-With-Python/blob/master/07.%20Deep%20Learning%20Fundamentals/data/ZaynLyrics.txtからダウンロードできます。。

作業は、必要なライブラリをインポートすることから始まります。

import tensorflow as tf
import numpy as np

次に、歌詞を含むファイルが読み込まれます。

with open("Zayn_Lyrics.txt","r") as f: 
    data=f.read() 
    data=data.replace('\n','') 
    data = data.lower()

これで、すべての文字がall_chars変数に格納されます。

all_chars=list(set(data))

一意の文字の数は、unique_charsに格納されます。

unique_chars = len(all_chars)

そして、文字の総数は変数total_charsに格納されます：

total_chars =len(data)

まず、各文字にインデックスを割り当てます。char_to_ixには文字とインデックスのマッピングが含まれ、ix_to_charには文字とインデックスのマッピングが含まれます。

char_to_ix = { ch:i for i,ch in enumerate(all_chars) }
ix_to_char = { i:ch for i,ch in enumerate(all_chars) }

例：

次に、入力値とターゲット値を生成するgenerate_batch関数を定義します。ターゲット値は、入力値のシフトとiの積に等しくなります。

たとえば、shift = 1のinput = [12,13,24]の場合、ターゲット値は[13,24]になります。

def generate_batch(seq_length,i):
    inputs = [char_to_ix[ch] for ch in data[i:i+seq_length]]
    targets = [char_to_ix[ch] for ch in data[i+1:i+seq_length+1]]
    inputs=np.array(inputs).reshape(seq_length,1)
    targets=np.array(targets).reshape(seq_length,1)
    return inputs,targets

シーケンスの長さ、トレーニングの速度、およびニューロンの数と等しいノードの数を決定します。

seq_length = 25
learning_rate = 0.1
num_nodes = 300

LSTM RNNをビルドします。TensorFlowは、LSTMセルを構築するためのBasicLSTMCell（）関数を提供します。LSTMセルのユニット数と、使用するアクティブ化機能のタイプを指定する必要があります。

したがって、LSTMセルを作成し、tf.nn.dynamic_rnn（）関数を使用してこのセルでRNNネットワークを構築します。これにより、出力と状態値が返されます。

def build_rnn(x):
        cell= tf.contrib.rnn.BasicLSTMCell(num_units=num_nodes,
activation=tf.nn.relu)
        outputs, states = tf.nn.dynamic_rnn(cell, x, dtype=tf.float32)
        return outputs,states

次に、入力XとターゲットYの代わりを作成します。

X=tf.placeholder(tf.float32,[None,1])
Y=tf.placeholder(tf.float32,[None,1])

XとYをintに変換します。

X=tf.cast(X,tf.int32)
Y=tf.cast(Y,tf.int32)

XとYのワンホットビューも作成します。

X_onehot=tf.one_hot(X,unique_chars)
Y_onehot=tf.one_hot(Y,unique_chars)

build_rnn関数を呼び出して、RNNから出力と状態を取得します。

outputs,states=build_rnn(X_onehot)

出力を転置します。

outputs=tf.transpose(outputs,perm=[1,0,2])

重みとオフセットを初期化します。

W=tf.Variable(tf.random_normal((num_nodes,unique_chars),stddev=0.001))
B=tf.Variable(tf.zeros((1,unique_chars)))

出力に重みを乗算し、オフセットを追加することにより、出力を計算します。

Ys=tf.matmul(outputs[0],W)+B

次に、softmaxアクティベーションを実行し、確率を取得します。

prediction = tf.nn.softmax(Ys)

cross_entropy=tf.reduce_mean(tf.nn.softmax_cross_entropy_with_logits(labels
