HTML tingkat lanjut

Dalam artikel ini saya ingin memberi tahu sedikit tentang perpustakaan, versi pertama yang saya buat pada akhir tahun lalu. Intinya sangat sederhana - untuk memperluas kemampuan bahasa HTML sehingga Anda dapat menulis hal-hal sederhana dan rutin tanpa JavaScript: mengirimkan formulir dalam format json, memuat file template HTML ke halaman tertentu (pada dasarnya sistem modular untuk HTML melalui permintaan http / s), turbin ( hai untuk pengguna RoR), templat sederhana berdasarkan respons permintaan ajax dan sedikit lagi.



Perpustakaan disebut EHTML atau Extended HTML. Ini didasarkan pada ide terkenal komponen web . Ini tersedia di github , ada dokumentasi yang cukup terstruktur dengan contoh. Pada artikel ini saya hanya akan menjelaskan ide-ide utama, mungkin itu akan pergi ke seseorang.

Untuk memulainya, untuk menggunakannya, Anda tidak memerlukan npm atau cli tools untuk membuat proyek, yang diperlukan hanyalah halaman HTML tempat Anda memasukkan skrip perpustakaan:

  <script src="/../js/ehtml.bundle.min.js" type="text/javascript"></script>
</head>

Sistem modular


Mari kita mulai dengan yang sederhana - kompilasi halaman HTML dari yang lain:


<!DOCTYPE html>
<html xmlns="http://www.w3.org/1999/xhtml" lang="en">
  <head>
    <script src="/../js/ehtml.bundle.min.js" type="text/javascript"></script>
  </head>

  <body class="main">
    <div class="articles">
      <e-html data-src="/../html/first.html"></e-html>
      <e-html data-src="/../html/second.html"></e-html>
      <e-html data-src="/../html/third.html"></e-html>
    </div>
  </body>
</html>

Untuk melakukan ini, kami menggunakan elemen khusus e-htmlyang memiliki atribut tunggal data-src, di mana kami menunjukkan di mana bagian HTML halaman didistribusikan. Artinya, jika misalnya first.htmlterlihat seperti ini:


<div class="article">
  <!-- some content of the first article -->
</div>

maka pada akhirnya halaman kami akan dirender sebagai berikut:


<!DOCTYPE html>
<html xmlns="http://www.w3.org/1999/xhtml" lang="en">
  <head>
    <script src="/../js/ehtml.bundle.min.js" type="text/javascript"></script>
  </head>

  <body class="main">
    <div class="articles">
      <div class="article">
        <!-- content of the first article -->
      </div>
      <div class="article">
        <!-- content of the second article -->
      </div>
      <div class="article">
        <!-- content of the third article -->
      </div>
    </div>
  </body>
</html>
