Panduan Pengujian Keamanan Web Yang Diperbarui Konsorsium OWASP



Proyek Panduan Pengujian Keamanan Web (WSTG) adalah metodologi pengujian keamanan utama untuk pengembang aplikasi web dan profesional keamanan informasi dan sedang dikembangkan oleh konsorsium internasional OWASP.

Proyek Keamanan Aplikasi Web Terbuka (OWASP) adalah proyek keamanan aplikasi web sumber terbuka. Komunitas OWASP mencakup perusahaan, organisasi pendidikan, dan individu dari seluruh dunia. Komunitas ini sedang berupaya membuat artikel, manual, dokumentasi, alat, dan teknologi yang dapat diakses secara bebas.

WSTG adalah panduan komprehensif untuk menguji keamanan aplikasi web dan layanan web. WSTG, upaya kolaborasi oleh para profesional dan sukarelawan cybersecurity, menyediakan serangkaian praktik terbaik yang digunakan oleh pentester dan organisasi di seluruh dunia.

Versi saat ini - [Versi 4.1] - 2020-04-22, versi sebelumnya - [Versi 4.0] - 2014-09-17 - sekitar 6 tahun telah berlalu di antara pembaruan, yang harus mencakup lapisan besar teknologi modern, namun, perubahan mendasar akan diharapkan hanya di 5 versi WSTG.
