Pharmacokinetic Modeling in Julia: Practical Use of DiffEquations.jl and Optim.jl

A pharmacokinetic model is often a simplified mathematical description of how the concentration (amount) of a test substance changes over time in a biological system. Simplified because the body is a complex mechanism in which absorption, distribution, metabolism and excretion of substances ( ADME) can occur in various ways, depending both on the substance itself and on the current state of the body and not infrequently with non-obvious feedbacks. Therefore, to reflect and describe all possible factors acting on the kinetics of a substance is practically impossible and in most cases not advisable. Nevertheless, a representation is possible in which a substance is distributed in limited areas of the body, which will be called cameras, and the transition of a substance from one chamber to another is described by a certain equation.







Pharmacokinetic modeling can be considered as a special case of nonlinear modeling. For this, commercial packages are widely used, such as: Phoenix WinNonlin, NONMEM, Monolix, etc., which allow you to get estimates of model parameters of different (arbitrary) complexity. There are also packages available for the computing environment R Project (nlme, nls, saemix) and Julia (CurveFit.jl, LsqFit.jl, NLreg.jl), which can be used to determine the parameters of pharmacokinetic models. The limitations of the R Project and Julia packages are related to the fact that the researcher must specify a nonlinear function in an explicit form, which is not always possible. In addition, not all of these packages have sufficient functionality to evaluate a model with mixed effects (random and fixed effects).







Pumas.jl is a little apart, as Despite the serious declared functionality, the package has not received wide distribution (perhaps for now), probably this (wide distribution) is prevented by an indistinct use license. The purpose of this work: a description of the main approaches to pharmacokinetic modeling, the construction of a pharmacokinetic model in the form of a system of differential equations, the selection of parameters of the pharmacokinetic model.







Models can be classified by the number of chambers, the method of receipt and the dosage interval of the substance, the mechanism (function) of elimination:







  1. Number of cameras:

    • Single chamber
    • Bicameral
    • Three-chamber (and more)
  2. Way of receipt:

    • Intravenously bolus
    • Intravenously infusion
    • Inside (per os)
  3. Dosing Interval:

    • Once
    • Many times
  4. Elimination mechanism:

    • Linear
    • According to the Michaelis-Menten equation

using DifferentialEquations,  Plots, Optim, LossFunctions, Random, Distributions, DiffEqParamEstim

#Function
function pkf!(du, u, p, t)
  K‚Çê = p[1]
  K‚ÇÅ‚ÇÇ= p[2]
  K‚ÇÇ‚ÇÅ= p[3]
  K‚Çë = p[4]
  A·µç = u[1]
  C‚ÇÅ = u[2]
  C‚ÇÇ = u[3]
  Vf = p[5]
  du[1] = - K‚Çê  * A·µç
  du[2] =   K‚Çê  * A·µç / Vf   - K‚Çë * C‚ÇÅ  - K‚ÇÅ‚ÇÇ * C‚ÇÅ  + K‚ÇÇ‚ÇÅ * C‚ÇÇ
  du[3] =   K‚ÇÅ‚ÇÇ * C‚ÇÅ - K‚ÇÇ‚ÇÅ * C‚ÇÇ

function loss_function(sol, x, y)
   tot_loss = 0.0
   if any((s.retcode != :Success for s in sol))
     tot_loss = Inf
   else
     tot_loss = sum(value(L2DistLoss(), hcat(sol.(x)...)'[:,2], y))
