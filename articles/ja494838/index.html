例としてQuectel EC21 LTEモデムとYocto Projectを使用する組み込みシステムのGSM / 3G / 4Gモデム


組み込みシステムに基づく多くのデバイスはインターネットを必要とします。有線インターネット接続は常に可能であるとは限りません。デバイスが携帯電話の通信可能範囲内にある場合は、GSM / 3G / 4Gモデムを使用してインターネットに接続できます。


このメモでは、Yocto Projectツールキットを使用して組み込みシステム用のLinuxディストリビューションを作成するプロセスについて説明します。これにより、GSM / 3G / 4GモデムとPPPプロトコルを介してデバイスをインターネットに接続できます。
