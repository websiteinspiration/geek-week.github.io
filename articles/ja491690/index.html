データプライバシーの保護-世紀の問題

データはどの程度保護されていますか？すべてが相互に関連している現代の世界では、この問題は大企業と一般ユーザーの両方を悩ませています。データプライバシーデーが捧げられたのは彼にとって、ヨーロッパの理事会の47の加盟国だけでなく、米国、カナダ、イスラエルでもサポートされている国際的な取り組みです。その目標は、個人情報がどのように使用され、漏洩から身を守るかについて人々の意識を高めることです。イベントの参加者は、オンラインプライバシーを強化する方法を提供し、立法レベルでイノベーションの可能性について話し合います。ソフトウェアとハ​​ードウェアの開発者は、今度は新しいソリューションとプロジェクトを発表します。

個人情報保護デーは、2006年から1月28日に祝われます。日付は、個人データの自動処理に関連する個人の保護に関する1981年1月28日の欧州評議会会議の署名に合わせて調整されます。この文書は、個人情報とプライバシー権を保護するための国際的な義務を初めて定義しました。

現在までに50か国以上が条約に署名していますが、問題はまだ解決されていません。これまで、何百万人もの人々が、企業や政府機関がデータを収集、保存、使用する方法や、サイバー犯罪者の抽出の標的になる方法を知りません。



インターネットに時間を費やすたびに、詐欺師が使用できる個人情報を共有します。ユーザーは、適切な予防策を講じるために、知識を身につける必要があります。その中でも-強力な暗号化または安全なクラウドサービスを使用したバックアップ。

統計によると、世界中のすべてのサイバー犯罪の約70％が個人情報の盗難に関連しています。これは、パスワードまたはコンピューター自体の盗難が原因で発生する可能性があり、犯罪者が企業の機密情報を手に入れるという事実につながります。 2秒ごとに、新しいハッキングとID盗難事件が世界で発生しています。

ユーザーが自分自身を保護するのを助けるために、National Cyber​​ Security Security Alliance（NCSA）はプライバシー設定を制御する単一のポータルを作成しました。これには、人気のあるeコマースサイト、郵便およびモバイルサービス、検索エンジン、ソーシャルネットワークへのリンクが含まれています。これらの多くのリソースはそれぞれユーザーに関する情報を収集しているため、適切なレベルのプライバシーを確​​認して確立することをお勧めします。

メーカーはまた、デバイスの保護に積極的に取り組んでいます。たとえば、それらは、一緒に、または別々に機能するマルチコンポーネントおよび多要素認証システムを提供します。最も信頼できるオプションの1つは、ラップトップ用のスマートカードです。

プライバシー+セキュリティ


プライバシー設定は非常に重要ですが、それだけでは十分ではありません。データを保存するためにあらゆる努力を払うことが重要です。プライバシーが彼の個人情報を管理する人権に関係している場合、セキュリティは保護方法の組み合わせです。毎日新しい脅威が出現するため、定期的に更新する必要のあるソフトウェアから始め、ソフトウェアメーカーはそれらを排除するために更新をリリースします。

ただし、ハードウェアセキュリティ機能は、データセキュリティにおいて重要な役割を果たします。それらは、開発の最初から製品アーキテクチャに組み込む必要があります（設計によるセキュリティ）。

デバイスレベルのセキュリティは、ユーザーを保護する堅牢な設計ソリューションから始まります。各コンポーネントとその供給元を確認する必要があります。これだけで、「組み込み」の脅威が存在しないことが保証されます。犯罪者は、製造および輸送中にデバイスに脆弱性を注入するサプライチェーンをますますターゲットにしています。



セキュリティハードウェアの1つの例は、ノートパソコンのカメラを閉じるThinkShutterです。昨年Lenovoによって導入されたこのシンプルなソリューションは、ハッカーがカメラを使用してあなたをスパイするのを防ぎます。


レノボのThinkShutterカメラシャッターは、ラップトップユーザーが誰かに見られるかどうかを自分で決める機会を与えます。

オンラインカメラを使用して人々を監視し、自宅の環境を評価するハッカーがいますが、彼らはしばしば別の目標を持っています-現在コンピューターを使用しているかどうかを確認することです。これは、画面に表示される一連のハッキングスクリプトを実行するために行われます。

セキュリティシールドの後ろ


Lenovoは、いくつかの重要なセキュリティソリューションを作成した最初の企業です。これには、生体認証の主要コンポーネントとなった指紋スキャナーや、最初のTPM（Trusted Platform Module）チップ（組み込みの暗号化キーでハードウェアを保護する専用のマイクロコントローラー）が含まれます。
今日、同社は製品を改良し続け、個人情報への不正アクセスを防止しています。そのため、Lenovoは企業顧客向けに、基本的なプライバシー、認証、データ保護、およびネットワーク脆弱性検出を提供する包括的なThinkShieldセキュリティプラットフォームを作成しました。

問題と解決策


企業や政府機関は、ウェブサイトのCookieからマシンに埋め込まれたセンサー、さらには顧客、市民、従業員に関する前例のない量のデータを収集するデバイスまで、さまざまな最新のテクノロジーを使用しています。彼らのタスクには、気分や好みの評価、ニーズの予測、労働生産性の向上、詐欺の検出、場所の追跡、健康状態の監視、セキュリティの監視が含まれます。

このデータのほとんどは、適切な目的で収集されます。しかし、誰もが最も差し迫った質問の1つに答えることができません：国境はどこですか？人々は、個人情報がどのように処理され、保存されるか、そして彼らの機密性がいかに安全であるかを知りたいと思っています。 「忘却の権利」と他の多くの国際、州、および地域の安全規則が尊重されていますか？

専門家によると、企業は信頼を構築するための対策を講じ、特定の情報を収集する理由、その使用方法、相手が得られるメリット、そしてもちろん、このすべてのデータを保護する方法を顧客に説明する必要があります。

特にクラウド環境の開発において、セキュリティの確保はあらゆる企業にとって課題です。新興企業はデータの保護と管理の方法を変革することを約束しており、テクノロジープロバイダーとコンサルティング会社は独自のプラットフォームと技術を開発しています。今日、プライバシーは業界となっています。

また、個人データ保護デーは、問題を解決する方法をもう一度考える機会です。現在、約80の州が個人情報の保護に関する欧州評議会のイニシアチブを支持しており、それを正しく処理する慣行は、国連を含む主要な世界組織で使用されています。
