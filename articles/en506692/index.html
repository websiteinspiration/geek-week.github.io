The 14 most popular Node.js libraries in 2020



Good day, friends!

I present to you the translation of the article “14 Most Useful NodeJS Libraries in 2020” .

In this article, the author shares his findings among the many Nodejs packages.

Express


Fast, flexible, minimalistic, easy to learn web development framework.

Great for

  • developing applications using the MEAN stack (MongoDB, Express, Angular, Node) or MERN (React instead of Angular)
  • easy handling of various types of requests, such as GET, PUT, POST and DELETE
  • rapid deployment of single-page, multi-page and hybrid applications

11 million

Alternatives

Koa, Hapi, Meteor

Cheerio


Designed for processing ("parsing") markup, for example, HTML, provides an interface for manipulating ("traverse") the final data structure.

	const cheerio = require('cheerio')
	const $ = cheerio.load('<ul id="fruits">...</ul>')

Great for

  • creating web scanners ("scrapers")
  • easy, intuitive syntax and toolkit

Nodemailer


Designed to send emails.

	const nodemailer = require('nodemailer')

	const testAccount = await nodemailer.createTestAccount()

	const transporter = nodemailer.createTransport({
		host: 'smtp.ethereal.email',
		port: 587,
		secure: false,
		auth: {
			user: testAccount.user,
			pass: testAccount.pass
		}
	})

	const info = await transporter.sendMail({
		from: '"John Smith" <smith@example.com>',
		to: 'foo@example.com, bar@example.com',
		subject: 'hello',
		text: 'hello crazy world',
		html: '<strong>hello crazy world</strong>'
	})

Great for

  • sending messages via SMTP

980 thousand

Socket.io


Designed to set up a two-way event-based connection in real time.

	const server = require('http').createServer()
	const io = require('socket.io')(server)
	io.on('connection', client => {
		client.on('event', data => { ... })
		client.on('disconnect', () => { ... })
	})
	server.listen(3000)

Great for

  • implementing live analytics, binary data flow, messaging, collaboration with documents

Weekly downloads

3 million

Alternatives

pusher, websockets

Designed to receive fake information in the browser and “node”.

	const faker = require('faker')

	const randomEmail = faker.internet.email()

Great for

  • testing the frontend in terms of working with data until the development of the backend
  • A large number of methods, including obtaining an arbitrary address, company name, database, image, first name, last name, etc.

Morgan


It is a middleware for registering requests.

	morgan
	(':method:url :status :res[content-length] - :response-time ms')

	---

	const express = require('express')

	const morgan = require('morgan')

	const app = express()

	app.use(morgan('combined'))

	app.get('/', (req, res) => res.send('hello crazy world'))

Great for

  • write requests to the console, file, database
  • debugging and "logging"

Weekly downloads

2 million

Http-errors


Designed to create specific errors in order to test applications on Express, Koa, Connect, etc.

	app.use((req, res, next) => {
		if(!req.user) return next(createError('401', ',    .'))
		next()
	})

Great for

  • emulation errors
  • a large number of properties of the Error object are available

	expose
	headers
	message
	status
	statusCode

Weekly downloads

27 million

Body parser


It is a middleware for parsing data.

Parses incoming requests (req.body) before processing them.

	const express = require('express')
	const bodyParser = require('body-parser')

	const app = express()

	//  application/x-www-form-urlencoded
	app.use(bodyParser.urlencoded({ extended: false }))

	//  application/json
	app.use(bodyParser.json())

Great for

  • query body interpretations
  • contains many useful functions (inflate, type, verify, etc.)

Weekly downloads

13 million

Sequelize


It is a Promise-based technology for object-relational mapping of Postgres, MySQL, MariaDB, SQLite, and Microsoft SQL Server.

It supports secure (reliable) transactions, data exchange, energetic and delayed loading, data replication, etc.

	const sequelize = new Sequelize(
		'database', 'username', 'password', {
			host: 'localhost',
			dialect: /*   'mysql' | 'mariadb' | 'postgres' | 'mssql' */
		}
	)

Weekly downloads

720 thousand

Passport


It is an Express compatible middleware for authentication.

The main task is to authenticate requests using a set of plugins - strategies.

	passport.use(new LocalStrategy(
		(username, password, done) => {
			User.findOne({ username: username }, (err, user) => {
				if(err) { return done(err) }

				if(!user) { return done(null, false) }

				if(!user.verifyPassword(password)) { return done(null, false) }

				return done(null, user)
			})
		}
	))

Great for

  • authentication
  • integration with OAuth and OpenID (log in with your Facebook account, Twitter, etc.)

Weekly downloads

810 thousand

Dotenv


It is a module of zero dependency (point) for loading variables from the .env file into process.env.

Keeps variables separate from code.

	require('dotenv').config()

	const db = require('db')

	db.connect({
		host: process.env.DB_HOST,
		username: process.env.DB_USER,
		password: process.env.DB_PASS
	})

	//  .env
	DB_HOST = localhost
	DB_USER = root
	DB_PASS=12345

Great for

  • loading environment variables such as AWS, sql usernames, passwords needed to deploy applications, connecting to other tools
  • storing environment settings separately from the code

Weekly downloads

10 million

Multer


Middleware for processing multipart / form-data - image loading.

	const express = require('express')

	const multer = require('multer')

	const upload = multer({ desc: 'uploads/' })

	const app = express()

	app.post('/profile', upload.single('avatar'), (req, res, next) => {
		// req.file -   'avatar'
		// req.body     
	})

	app.post('/photos/upload', upload.array('photos', 12), (req, res, next) => {
		// req.files -    'photos'
		// req.body      
	})

Great for

  • quick download multipart / form-data files

Weekly downloads

92 thousand

Axios


Promise-based HTTP client for browser and node.

	const axios = require('axios')

	//     ID
	axios.get('/user?ID=12345')
		.then(response => {
			// 
			console.log(response)
		}).catch(error => {
			//  
			console.log(error)
		}).finally(() => {
			//  
		})

Great for

  • sending requests from Node
  • sending requests from the browser
  • Promise Support

Weekly downloads

9.6 million

CORS


Connect / Express middleware designed to define common-origin policy rules.

	const express = require('express')

	const cors = require('cors')

	const app = express()

	app.use(cors())

	app.get('/products/:id', (req, res, next) => {
		res.json({ msg: '    !' })
	})

	app.listen(80, () => {
		console.log('ok')
	})

Great for

  • CORS-related tasks

Weekly downloads

3.7 million
