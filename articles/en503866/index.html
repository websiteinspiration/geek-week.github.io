Published Roadmap development Zextras Suite for 2020

Earlier, we talked about the innovations of Zextras Suite Pro and Team Pro at the beginning of 2020. Zextras has published the Roadmap for the development of new Zextras Suite features  for 2020 .


June - Team: New functions (IMPROVED)

  1. Reply to a message (reply)
  2. Clear chat history
  3. Edit message
  4. Delete message

August - Team: Video Server (NEW)

The appearance of the built-in video server in Zextras Team will allow you to introduce a number of new functions that can be controlled directly from the admin console, as well as increase the productivity of the service. Audio and video streams will be compressed, optimized and combined in order to get a separate stream for each device. In addition, the server will automatically select the appropriate codec and bit rate for each device in order to ensure maximum quality of communication.

August - Backup on external endpoint (NEW)

Zextras Backup will provide an opportunity to back up your system to local, hybrid or cloud S3-compatible storage, or to external storage of BLOB elements, such as NFS. At the same time, a local metadata cache will allow you to maintain the original performance, despite the fact that they themselves will also be stored in an external endpoint in case of a need for disaster recovery.

September - Zextras Authentication and Login (NEW) Two

-factor authentication, federated SAML authentication and authorization system, as well as multiple mobile passwords will be added to the Zimbra Collaboration Suite Open-Source Edition. An updated login page will also appear with the possibility of customization.

September - Team Privacy settings (NEW)

Users and administrators will be able to choose whether information about their presence will be displayed. In particular, it will be possible to hide information about reading messages, user status, and so on. September - Mobile Apps QR Authentication (NEW) The QR-Login system allows you to use mobile applications without having to enter a password through the use of a cross-authentication system, as well as manage their registration and authorization. September - Zextras Address Book for MS Outlook / MAC Mail / Thunderbird (NEW) Connect Microsoft Outlook, Thunderbird and Mac Mail to a centralized Zimbra GAL and your address books for complete user convenience.










Address Book can work in conjunction with IMAP / POP or EAS. Microsoft Outlook can use EAS and the address book to provide full interaction with Outlook without any plug-in in the client!

October - Zextrasconfig Infrastructure Manager (NEW)

All Zextras components in the infrastructure can be managed through a special Zextrasconfig manager. This manager enables Zimbra administrators to manage Zextras components on various infrastructure nodes, including the updated integrated Zextras Docs server and the integrated Zextras Team video server.

October - Admin Interface & User Preferences (REFACTORING)

The new Zextras component administration interface is written in React. Starting with the components of Zextras, which will continue to be presented in the familiar administration interface, we will transfer all the options to the new technology without removing them from the familiar interface. A full transition will be made in the future.

October - New installer using native Linux Packaging system (NEW) The

new Zextras package will be distributed through repositories with native Linux support. When installing and updating the package, it will be possible to use system commands like ‚Äúapt-get install / update‚Äù.

November - Docs Server (NEW)

The Zextras Docs server will become part of the infrastructure, managed through the Zextrasconfig Centralized Infrastructure Management Manager. Installing and updating the server on individual nodes will no longer be necessary.

November - Team: Meeting's new functionalities (NEW)

Thanks to the new video server, a number of new features will appear in the Zextras Team. Among them:

  • Moderator mode
  • The ability to mute the sound for a specific user or immediately for all
  • Ability to enable webinar mode and video conferencing mode
  • The event log
  • Chat request
  • Walkie Talkie Mode
  • Hangout recordings can be uploaded to Zextras Drive

November - High Availability (NEW) The

ability to synchronize users, groups, domains, and even entire mail servers between different infrastructure nodes. Ability to configure synchronization with a spare node or another active node in the cluster. The system is designed to achieve 99.99% availability per month.

November - Drive Personal external Account (IMPROVED)

Ability to connect Dropbox / Gdrive / nextCloud / OwnCloud accounts

December - Team: Instant Meeting (IMPROVED)

Ability to create an instant meeting and provide access using permanent and temporary links.

December - Email Sign & Encryption (S / Mime-PGP) (NEW)

End-to-end cryptography will be implemented in the web client. The system will be able to encrypt and / or sign the emails sent to decrypt and verify the signature of the received emails.

Many of these features have so far been exclusive to the commercial version of Zimbra. Already this year, users of the Zimbra Collaboration Suite Open-Source Edition will be able to take advantage of these and other features when purchasing Zextras Suite. In the future we will talk in detail about the new features of Zextras Suite.
