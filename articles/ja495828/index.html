それでは始めましょう。繰り返しになりますが、私は自分自身の下にすべての質問を思い付いたのではなく、さまざまなフォーラムやソーシャルネットワークから収集されました。

Q1：これは一般的なインフルエンザです（実際にはありません）


コロナウイルスは通常のインフルエンザではありません。

彼の生殖数はR0の2倍（1.3対2-2.5、平​​均で2倍の感染力）、人が感染を拡大できる潜伏期間（4対14日）の4倍、5〜10倍高い割合入院、そして約10倍高い死亡率。それぞれの要素は、それ自体はすでに不快ですが、これらすべてを組み合わせると、言葉の真の意味で、かなりキラーなものが得られます。

Q2：ああ、私たちはみな死にます、死亡率21％（実際にはそうではありません）


以前のものと同じくらい有害な神話。多くの人がwww.worldometers.info/coronavirusを開いて、次の画像を見てください



。ただし、実際には、この数字は covidによる致死とは関係ありません。

簡単な例を考えてみましょう。 100人がウイルスに感染したとします。これらのうち、50％は無症候性であり、まったく医師のもとに行かず、統計に含まれていませんでした。残りの50人のうち20人は軽度の病気で、SARSと診断されました。残りの30人は入院し、そのうち5人は肺炎と診断され、残りの25人はcovidの実際の検査を受けました。その結果、この25人のうち、3人が亡くなりました。結果：「22が回復、3が死亡、12％が死亡」は、世界計の公式統計に分類されます。問題は、これらの12％がコロナウイルスによる実際の死亡率とどう関係しているのかということです。そうです、いいえ。一般に、この数字に物理的な意味があるかどうかは疑問です。これは、特定の国の人口に対するテストの可用性の指標です。ちなみに、ロシアだけでなくアメリカやイギリスでもどこでもテストが欠けている状況です。

Covidによる推定死亡率は、さまざまな推定値によると、0.5〜3％です。これもたくさんありますが、21％ではありません。さらに、統計によると、症例の約50％は一般的に無症候性であり、40〜50歳の人の死亡率は約0.4％です。

Q3：死亡率と死亡率


それはすでに何度も議論されてきましたが、多くの人々はまだ混乱しています。

死亡率：人口1000人あたりの死亡数。むしろ、「ウガンダでは年間10人が下痢で亡くなっている」、「イスラエルの癌による死亡率はロシアよりも低い」など、医療システムの質の指標として使用されています。

死亡率：病気の症例数に対する死亡数の比率。ほとんどの人が通常関心を持っているのはこのパラメーターです。しかし、ニュアンスもあります。病気の正確な患者数は不明ですが、コロナウイルスの正確な死亡率はまだ不明です。前の段落を参照してください。

Q4：死者が非常に少ない場合、ロシアで検疫を導入する必要があるのはなぜですか？


この質問に答えるために、さまざまな国のケース数の曲線は互換性があります。

その結果、答えは簡単です。たとえ数が少なくても、将来の死亡を防ぐために検疫が導入されます。そしてこれは、概して、ロシア/ウクライナ/ベラルーシでは、自分自身ではなく他人の過ちから学ぶことができる特定の成功です。流行は後で始まった。サンクトペテルブルクで2週間で3人がコロナウイルスによって死んでいることは明らかですが、これは「刺激を与えない」ことになりますが、何もしなければ3は簡単に30になり、その後は300に変わるなどと理解する必要があります。

Q5：隔離が機能しません


本当じゃない。まず、検疫は失敗しません。連絡先の数を減らすと、もちろんウイルスが拡散する可能性も低くなります。もちろん、この検疫を観察し、会社でバーベキューや親戚に行かないようにしてください。

次に、検疫の結果はすぐには表示されず、2〜3週間後に表示されることに注意してください。これは潜伏期間と、通常すぐには検査しないという事実によるものですが、すでに悪化した状態で入院している人たちです。

検疫が導入されたさまざまな国の1日あたりの新しいケースのグラフを見てみましょう。







すべてのグラフで、「骨折」の瞬間がはっきりとわかります。数日ごとにケースが2倍になる指数関数的な増加が線形に置き換えられ、その後減少します。ロシアでは、執筆の時点で、検疫の導入から数日が過ぎているため、データはまだありません。しかし、新しいケースの数が減少するという事実は疑いの余地がありません。

Q6：検疫はいつ終了しますか？


理論的には、1日あたりの新しいケースの数は着実に減少します。残念ながら、私はロシアの公式サイトで便利なインフォグラフィックに出くわしたことがなかったので、www.worldometers.info / coronavirus / country / russiaのパラメーター「Daily New Cases」で確認するのが最も便利です。

しかし、ワクチンは1年以内に約束されたものであり、これまでに病気にかかった患者の実際の数は1％以下であることを覚えておくことは重要です。これは、たとえば、公共のイベントに関連するビジネスを計画する場合に重要です。

Q7：コロナウイルスは米国/イスラエル/ロシア/などで発明されました


本当じゃない。コロナウイルスのゲノムはすでに解読されており、中国からの起源は疑いの余地がない。インタラクティブなデータはこちらでご覧いただけます：www.gisaid.org/epiflu-applications/next-hcov-19-app



Q8：ウイルスは長い間私たちと一緒にいて、誰もがすでに彼らと病気になっています


残念ながらこれは正しくありません。実際、統計によると、covidの症例の約50％は無症候性であり、80％は入院を必要とせず、そのような症例の多くはテストされていませんが、「集団免疫」について話す必要はありません。3月30日付けのブリティッシュインペリアルカレッジの推定によると、感染者のおよその数は、スペインでは15％、イタリアでは9％からフランスでは3％、ノルウェーでは0.5％です。



このような推定はロシアでは見られませんでしたが、劇的に異なることはほとんどありません。

ちなみに、比較的すでに病気になっているため、2つの重要な質問があります。 1つ目は再感染の可能性です。問題はまだ未解決ですが、そのような可能性は非常に小さいと考えられています。。2番目の質問は、人が以前に病気にかかったかどうかを確認する方法です。このため、すでに使用され始めている特別な抗体検査が開発されています。ロシアでの入手については、執筆時点ではまだデータはありません。

自己分析CSVファイルは、ジョンズホプキンス大学のgithubアカウントのCOVID-19リポジトリから取得できます：github.com/CSSEGISandData/COVID-19。

Python 3.7のコード 
import pandas as pd
import matplotlib.pyplot as plt
from matplotlib.ticker import FormatStrFormatter, StrMethodFormatter, ScalarFormatter, FuncFormatter

# Load the dataset.
# Source https://github.com/CSSEGISandData/COVID-19/blob/master/csse_covid_19_data/csse_covid_19_time_series/time_series_covid19_confirmed_global.csv
df = pd.read_csv('covid19_confirmed_global.csv')
# print(df)

def get_data(country: str):
    df_ = df.loc[(df['Country/Region'] == country) & (df['Province/State'].isnull())]
    df_data = df_.loc[:, '1/22/20':'4/4/20']
    values = df_data.values.tolist()[0]
    new_cases = df_data.diff(axis=1).values.tolist()[0][1:]

    # Shift data to axis beginning
    treshold = 20
    start_index = values.index(next(filter(lambda x: x >= treshold, values)))
    return values[start_index:], new_cases[start_index:]


values_rus, values_rus_daily = get_data('Russia')
values_ita, values_ita_daily = get_data('Italy')
values_uk, values_uk_daily = get_data('United Kingdom')
values_ger, values_ger_daily = get_data('Germany')
values_spa, values_spa_daily = get_data('Spain')
values_fin, values_fin_daily = get_data('Finland')

fig, ax = plt.subplots()
fig.set_size_inches(12, 5)

# Draw
total_cases, new_cases = True, False

# 1. Total cases
if total_cases:
    ax.semilogy()
    ax.get_yaxis().set_major_formatter(ScalarFormatter())
    ax.plot(values_rus, label='')
    ax.plot(values_ita, label='')
    ax.plot(values_uk, label='')
    ax.plot(values_ger, label='')
    ax.plot(values_spa, label='')
    ax.plot(values_fin, label='')
    plt.xlabel('')
    plt.ylabel('  ')

# Daily new cases
if new_cases:
    fig.set_size_inches(12, 5)
    plt.grid(False)
    ax.bar(range(len(values_rus_daily)), values_rus_daily, width=0.6, label='')
    # ax.bar(range(len(values_ita_daily)), values_ita_daily, width=0.9, label='')
    # ax.bar(range(len(values_spa_daily)), values_spa_daily, width=0.9, label='')

    # ax.bar(range(len(values_ger_daily)), values_ger_daily, width=0.8, label='')
    plt.xlabel('')
    plt.ylabel('  ')

plt.legend(loc='upper left')
plt.show()
