React Nativeでのモバイルアプリケーションの作成



みなさん、こんにちは。これは、モバイルアプリケーションを作成する最初の部分です。

反応ネイティブとエキスポを使用してモバイルアプリケーションを実行します。素晴らしいニュースアプリケーションは作成しません。このため、rest apiを介してphpを使用します。

このアプリケーションでは、phpmyadminにある記事を表示します。

万博とは？


Expoは、JavaScriptを使用してiOSおよびAndroid用の独自のアプリケーションを作成できるツール、ライブラリ、およびサービスのセットです。それは有望に聞こえます。簡単に言えば、Expoは、プロジェクトの開発中にプロジェクトを表示するのに役立ちます。Expo CLIは、開発URL（インターネット上のローカルホストに似ています）を作成し、Expoクライアントで開いてアプリケーションをプレビューできます。

エキスポのインストール


expoをインストールする前に、npmをインストールしてください。

npm install expo-cli —global


Expoをiphoneに
インストールする


ExpoをAndroidにインストールする Expoにはオプションがあります。モバイルデバイスのExpoクライアントアプリケーションを介して、またはコンピューターのiOS / Androidシミュレーターを使用して、アプリケーションをプレビューできます。アプリケーションを作成するとき、私はAndroidでexpoを使用しました。

エキスポアプリケーションの作成


epxo init

Expoをインストールすると、Expo CLIインターフェースが開き、プロジェクト名とテンプレートを選択できます。空白のページを選択します。

cd newsApp

expo start —tunnel

その後、expoはローカル開発サーバーを起動し、サーバーを表示するための新しいウィンドウを開き、モバイルデバイスでプロジェクトを開くためのQRコードを提供します。または、モバイルアプリケーションとexpo cliに登録すると、モバイルアプリケーションの現在の開発状況が自動的に表示されます。

コンピューターでexpoを入力するには、次のコマンドを使用します。

expo login

反応ナビゲーションをインストールする


ある画面から別の画面に切り替えるには、react-nativationをダウンロードする必要があります。

yarn add react-nativation
