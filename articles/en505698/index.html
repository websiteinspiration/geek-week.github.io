Pixli, Webflow, Tilda:

Pixli


Pixli is a professional platform for visual digital design and frontend development. The service allows you to assemble an adaptive prototype or design layout of any complexity.

Service at the moment is absolutely free.
Moreover, the user has no restrictions on the number of projects and access to functionality.

Multi-user account access

Simultaneous work on projects is implemented on the platform through multi-user account access, several people can work on the project, however, unfortunately, the function of web sockets is not implemented, which does not allow online editing. The rest of the functions for collaboration is enough:

  • The ability to share design
  • Ability to add team members
  • ,
  • online


The service gives a very clean and valid code, so search engines will not lower the rating of a project created on Pixli, and a developer can transfer the site to any CMS without additional investments and work. You can also download the site archive.

Work in the browser

The platform does not require the installation of separate software on the computer, the user only needs a browser. However, working with the phone in the editor is not possible.

Multi-page

In the editor there are no restrictions on the number of pages in the project, even when working with large projects with a large number of blocks, the editor does not “slow down”. It’s easy to add SEO optimization and third-party meta tags to each page.

To work with repeating elements, there is a function for setting up a common block. Make the block shared, and the changes made in one instance will be applied to the entire group in the layout.

Templates The

service currently has a meager set of templates, it is clear that they are gradually being added, however, it lags behind other projects, however, I believe that the authors' idea is to develop from scratch.

Copying between projects

Designers working on several similar projects will greatly simplify the job of end-to-end copying: you can integrate all the necessary elements - blocks, layouts, styles, files and other content - from other projects into a new project.

Dynamic content management

The service has a visual CMS and allows you to quickly and conveniently fill the project with content, and the content can be customized to your taste or the needs of the client, this is a huge plus, since the content comes from your own CMS service, and not from the editor’s page.

Powerful animation with variable settings. The

service has the ability to create a truly flexible and complex animation, which is one level higher than the capabilities of other services (perhaps the only exception is Webflow). In Pixli, you can create not only effects of moving out, attenuation, displacement, but also set up a complex interaction between elements and flexible change of their styles and properties and manage all this on the timeline. This allows you to implement very complex and unusual ideas, which will clearly please the sophisticated designer.

Adapted to the commercial assembly: for designers, custom

typesetting Pixli has a white label function integrated, which makes it possible to transfer the site from the designer to the customer in a separate personal account. Together with the library of ready-made solutions, this feature makes the platform the most suitable for designers who put the assembly of sites on the stream.

Triggers

With the help of triggers, site elements can interact with each other, for example, you can create a survey on the site and when you click on the button you can start a timer, after which the user will see that the time is up and the form data will be sent to the site administrator.

Of course, you can create other interactions, for example, just show the hidden block when you hover over an element or make a button that, in the literal sense of the word, will run away from the user before performing any action. At Pixli, you are limited only to flying your own imagination.

Connect your own domain name and SSL certificate.

Easily connect your own domain and SSL certificate.

Disadvantages:
  • In the current version, the E-commerce functionality is not implemented;
  • The number of templates is limited;
  • Integrations with other services are not registered;
  • The threshold for entering the service is high enough, but the Lite mode still does not allow a beginner to use it, we hope that developers are actively working on it, then the service has a great future (the opinion of the author may not coincide with the opinion of the publisher)


Webflow


A startup, at startup, emphasized the formal absence of the need for coding: the user can create a project without using coding at all. But the insertion of code for those who wish will not cause difficulties.

A large library of templates.

To work with the program, the user can not only do without writing code, but also take advantage of a huge selection of custom templates. The only negative is that almost all of them are paid.

Assembling a project from elements

On the sidebar of the editor, the user can find all possible variants of elements (unstyled elements for embedding HTML, sliders, buttons, tabs, forms, videos and images) to assemble a page from them.

Adaptive design

When developing a layout, you can check its display on different devices and configure the necessary elements.

CSS Grids Function

Creating grids is simple and multifunctional; there is a function for viewing grid overlays. Expanded Functionality of Symbols

Design elements can be turned into symbols, which greatly simplifies the work with repeating structures: speed of repeated placement, automatic change of all duplicate symbols through editing one.

Password protection

Before publishing a site, you can protect it with a password from unwanted visitors entirely or only for individual pages.

Version control

Webflow took care of the safety of the results of the designer. The platform automatically saves the changes made and the ability to create recovery points is registered.

Integration

Webflow easily integrated Ecwid, Zapier, Shopify, PayPal, MailChimp, Google Analytics , and others that you can add to the sites of service options: CRM, mailing Store Page (or other payment formats), advanced analytics, internal search.

In addition to working with native fonts, the program provides interaction with Google and Adobe TypeKit

Webflow Ecommerce

A separate block of functionality is dedicated to e-commerce. For online stores, you can customize the checkout process, integrate the store into a blog or blog into the store. Analytics and promotion tools.

Disadvantages:

  • Despite the formal lack of coding, the design interface may seem difficult for novice users, especially in comparison with other designers (for example, the Wix platform);
  • Projects (no more than two) in the free version of the program will be created and published on the webflow.io subdomain. The volume of widgets is significantly reduced in comparison with the paid version. The number of functions is also reduced, for example, command access is not active;
  • Among the elements of the site there are no (and there is no possibility of creating) custom selectors;
  • With a large number of elements, the speed of the editor decreases noticeably;

Tilda Publishing


Site builder, work with which can be completely built on the sequential placement of blocks from the library.

The ease of layout and the diverse library

Tilda developers have drawn and added to the library more than 480 blocks for all design purposes: from full-screen covers to the application form. It is enough for you to collect the site as a constructor and thanks to a huge number of options the design will be individual. To speed up the process, you can use one of 180 page templates.

Drag-and-drop image integration

To upload your image, the designer does not need to use the internal service library, just drag the file from a folder on your computer or from a browser window.

Adaptability

All blocks from the library are already adapted for the image on different devices, while layout you will not have to think about it and adjust them for different platforms of end users.

Advanced editing features for professional designers.

Zero Block, a graphical editor, is built into the platform, providing professional designers the opportunity to transform all the elements of the site to their taste or create their own blocks.

Design control

Not only a font library is preloaded onto the platform (but designers can add their own font files, use TypeKit or Google Fonts), for each block on Tilda, the proportions of the content are written: drawing, leading, size to increase readability and stylish presentation to the viewer. Trends in typography and design are reflected on the current template base.

Online Stores, Ecommerce

For e-commerce, not only separate template assemblies and blocks are provided, but also interaction with various payment systems: payment by card, mobile phone, Yandex. Money or through PayPal. Data transfer is easily tied to Google Drive or to CRM

Optimization and promotion.

With concern for digital marketing, Tilda adds the ability to create landing pages, 12 services for data collection forms are presented. The project owner always knows exactly how the site will look in search results or on social networks, and you can connect Google Analytics and Yandex.Metrica to control traffic.

You can expand the functionality of the site at your discretion through the services integrated into Tilda.

Disadvantages:
  • The free version of the site has significant limitations: you can develop only 1 project with a dedicated memory capacity of 50Mb on the server. You will not be able to access the complete collection of blocks, many analytics, SEO tools and adding your own fonts.
  • All templates are single page.
  • Custom CSS for Zero Block only There is no way to edit the styles of building blocks.
