人事の大企業があなたを探すために学ぶべきプログラミング言語



10年前、PHPプログラミング言語は評価のリーダーでしたが、現在、Web上のプロジェクトはJavaScriptとPythonで作成されることが多くなっています。言語の学習に1〜2年費やし、その後仕事なしで留まるのは残念です。

Mail.ru Cloud Solutionsの私たちは、今後数年間でどのプログラミング言語が必要になるかについての分析、調査、開発者や大規模雇用主の意見を調査しました。そして、選択するときに何に焦点を当てるべきかを教えてくれます。

給料


言語の需要と見通しは、プログラマーに提供される給与によって見積もることができます。Habr Careersによると、2019年後半の最高給与の中央値は、次の言語を使用するロシアの開発者です。

  • 1位-Elixir、165,000ルーブル。
  • 2位-Objective-C、150,000ルーブル。この言語で求人を見る場合、開発者は通常、Objective-Cの後継であるSwiftを知る必要があります。
  • 3位-ScalaとGolang、150 000ルーブル。そして2018年には、Scalaがランキングのリーダーになりました。

PHP、Python、C ++、Swift、1C、Rubyの開発者の間で、給与の中央値の増加が見られます。給与は、Kotlin（-4％）とDelphi（-14％）のプログラマーによって削減されます。


Delphi開発者は、残りの価格のほぼ半分を獲得しています。

国際市場の状況は異なります。90,000人のStackOverflowユーザーが参加した調査によると、Clojure、Scala、Go、Rust、およびRプログラマーが最も多くの収入を得ています。PHP、Assembly、VBAの開発者は、経験が豊富であるにもかかわらず、給与が少なくて済みます。


図の円が大きいほど、言語を使用する開発者が多くなります。

雇用主への需要


空室を分析すると、有望な言語を特定することはできなくなりますが、雇用主が現在必要としているテクノロジーを確認できます。そして、状況が今後数年で変化することはほとんどありません-結局のところ、言語が古くても、プロジェクトはサポートまたは書き直される必要があります。たとえば、Swiftに置き換えられたにもかかわらず、Objective-Cは依然として必要です。DevSkillerエージェンシーに

よれば、国際的な雇用主はしばしばJavaScriptの専門家を必要としています-この言語は空席の40％で言及されています。HTML / CSSが2位、SQLが4位であるという事実から判断すると、フロントエンド開発者の需要は高くなっています。Javaは、欠員の31％、. NET / Cで言及されています-広告の12％で。

原則として、企業は一度に複数の言語を話す開発者を必要としています。つまり、Java + SQLバンドルは空室の26％、. NET + JavaScript-8％で見つかります。8つの組み合わせのうち5つはJavaScriptを特徴としており、JavaScriptはフロントエンドとバックエンドの両方で使用されます。HHによる小規模な調査に

よると、2019年の秋、雇用主はJava、PHP、1C、SQL、Python、C＃、JavaScript、C ++のプログラマーをより頻繁に探していました。JSは、そのフレームワーク（Vue、React、Ember、JQuery、Angular）が個別に考慮されているため、少し遅れています。一般的な統計を見ると、ロシアではこの言語も人気があります。

コミュニティの関心


開発者が理解できない何かに直面すると、質問をグーグルでググります。TIOBEサービスは、要求の数を分析し、プログラミング言語の人気度をまとめます。

格付けは慎重に扱う必要があります-リクエスト数の増加は、必ずしも雇用主からの言語の需要を意味するわけではありません。これは、コミュニティにおけるテクノロジーへの関心の一般的な指標です。開発者が使用および学習する言語が表示されます。

2020年2月、ほとんどのプログラマーはJava、C、Pythonに関する情報を検索しました。年間で、CとC＃への関心は4.3％と3.08％増加しました。Objective-Cは20の人気のある言語のリストにあるようです。サービスは、月に7つのポジションの減少が「言語の陳腐化とそれに代わるSwiftのポジションの強化」によるものであると想定しています。

TIOBEは年間平均データを考慮して、過去5年間の変化を示しています。Pythonの急速な成長-彼は7位から3位になりました。リーダーは、古典的なJavaとC、C ++、C＃です。

コミュニティの意見


Stack Overflow ITポータルは、 90,000人の開発者に人気のプログラミング言語について尋ねました。調査は国際社会の見解を示しています-あなたが他のプロジェクトより頻繁にプロジェクトで見つける最も人気のある言語のコード。

ほとんどの場合、開発者はJavaScriptを使用します -これは数年間の主要言語です。2018年と2019年の評価を比較すると、Pythonの人気が高まっています。

2019年のランキング（2018年の意見）では、Rust、Python、TypeScriptが最も人気のある開発者です。1年前、Rust、Kotlin、Pythonがリストに含まれていました。

コミュニティへの関心はPython、JavaScript、Goによって引き起こされ、それぞれ25.7％、17.8％、15％の開発者がそれらを研究したいと考えています。開発者が通信したくない言語のランキングでは、最初の3つの場所はVBA、Objective-C、およびAssemblyによって占められました。

コード人気


Github のOctoverseレポートでは、サービスの参加者が最もよく使用する言語を確認できます。言語の人気が高いほど、将来のチームリーダーがその言語でプロジェクトを開発することを決定する可能性が高くなります。

JavaScriptは数年前から最初の場所にあります。2019年の調査では、Pythonが2位になり、Javaが1行下に移動しました。C＃とシェルは成長を示しています。


グラフは、個々の言語の人気を示しています。近年、Rubyの使用頻度は低くなっています。

学習する有望な言語を選択する方法


雇用主は、ITの一般的な理論的基礎を研究し、言語とテクノロジーをツールとして扱うことをお勧めします-特定のプロジェクトに必要な場合は理解し、そうでない場合は時間を無駄にしないでください。

人気のある言語を推測しようとしないでください。さまざまな理由で技術の発展を予測することは不可能です。開発者の好みは、言語の技術的または商業的有効性だけでなく、ITセクターにおける州の存在、および大企業間の違いにも影響されます。

したがって、Javaの使用をめぐるOracleとの摩擦のために、GoogleのマネージャーはKotlinを支持する選択をしました。また、Androidオペレーティングシステムを開発するための公式ツールであるAndroid Studio 3.0にも含まれています。

Dmitry Rogov、技術開発担当次長「ID-Management Technologies」

時代遅れや流行りを避けます。いくつかの言語を学ぶことにした場合は、その関連性を確認してください-多分それは徐々により有望な技術に置き換えられています。

旅の初めにReksoftソフトウェア開発部門のNikolay Sokornovのディレクターは、GoやRustのような流行の言語を追いかけるのではなく、長年の実績のある言語に焦点を当てることをお勧めします：バックエンド用のJava、フロントエンド用のJavaScriptを学びます。

学習する言語や技術を選択するときは、雇用主の需要だけでなく、労働市場の一般的な状況も考慮する必要があります。そのため、Dmitry Rogovは、Java言語で過熱が顕著になったと主張しています。確かに、初心者の応募者のレベルでのみ、次のようになります。彼らは雇用主の要求と彼ら自身の給与期待を満たしていません。」

自分を1つの言語に限定しないでください。市場の需要だけでなく、あなたの興味も考慮する必要があります。言語を知っているだけでなく、テクノロジーの本質を理解していれば、仕事を見つけるのは簡単です。

言語は思考を定義します。最初にJavaScriptを選択すると、バックエンドを理解するのが難しくなります。C ++で開発する場合、何百ものHTTP RESTリクエストの並列処理の容易さを感じません。1つのトピックを選択し、それに慣れた後、他のテクノロジーを見てください。
