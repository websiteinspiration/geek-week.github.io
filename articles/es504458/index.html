Supongamos que su dispositivo no está instalado de antemano.


  1. Primero instala Python .
    Descargue la versión deseada y ejecute el archivo .exe. No olvide marcar la casilla de agregar ruta. Instalé Python 3.7.3 desde el 25 de marzo de 2019, porque la nueva versión de Python 3.7.4 desde el 8 de julio de 2019 no funcionaba correctamente, es decir, algunos comandos estaban colgados en el terminal. Abre la línea de comando.
  2. Instalar virtualenv.
    Necesitamos un entorno virtual para que cada proyecto individual tenga su propia "sala" con sus propias versiones de bibliotecas instaladas, que no dependan de otros proyectos y se confundan entre sí.
    Instalaremos paquetes usando pip. Se ha estado ejecutando inmediatamente con Python recientemente, pero generalmente necesita actualizarse con el comando:
    python -m pip install --upgrade pip
    Pip actualizado, ahora instale el entorno virtual : con el
    pip install virtualenv
    comando, cdvaya a la carpeta en la que desea crear el entorno e ingrese el comando:
    mkdir opencvtutorial_envasí es como creamos un entorno llamado opencvtutorial_env.
    Luego, ingrese el comando virtualenv opencvtutorial_envy para activar, vaya a la carpeta del entorno y luego use Tab para activar.
    .\opencvtutorial_env\Scripts\activate
