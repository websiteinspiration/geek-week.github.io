Sehen sich vierzehn Personen dieses Produkt mit Sicherheit an?



Online-Shops gehen zu jedem Trick, um Sie zum Kauf ihrer Waren zu drängen. Normalerweise sind diese Ausweichmanöver jedoch immer noch nicht so transparent wie direkte Lügen, auf die ich in diesem Artikel eingehen werde.

Wenn Sie die Seiten eines Online-Shops durchgehen, stoßen Sie manchmal auf eine Zahl, die angibt, wie viele andere Kunden derzeit angeblich dasselbe Produkt anzeigen. Diese Taktik ist besonders beliebt bei Dienstleistungen zur Buchung von Flugtickets und Hotelzimmern. Es wird wahrscheinlich mit der Erwartung implementiert, dass Sie versuchen, einen Kauf so schnell wie möglich zu tätigen, bevor jemand anderes abfängt, was Sie wollen.

Manchmal ist dies eine echte Zahl, manchmal zufällig. Das heißt, es ist einfach völlig zufällig.

Schauen wir uns einen Online-Shop namens Dafni an . Wenn Sie nicht wissen, um welche Art von Geschäft es sich handelt, machen Sie sich keine Sorgen, ich weiß es auch nicht. Ich weiß nur, dass sie dort erstens etwas verkaufen und zweitens die Leute darüber täuschen, wie viele Besucher dieses oder jenes Produkt gerade sehen. Bitte beachten Sie, dass dieser Artikel nicht als gezielte Exposition eines bestimmten E-Commerce-Unternehmens gedacht war. Worüber ich sprechen werde, ist ein ziemlich häufiger Fokus, nur in anderen Fällen fällt er normalerweise nicht so auf. Am Beispiel dieser Website können Sie deutlich zeigen, wie Online-Shops im Allgemeinen ihre Kunden täuschen.

Ich ging zur Website und wählte das erste Produkt aus, das auf der Hauptseite angezeigt wurde. In meinem Fall stellte sich heraus, dass das Produkt folgendermaßen aussah:



Der Screenshot wurde am 11. Juni 2020 aufgenommen, der orangefarbene Rahmen wurde vom Autor des Artikels hinzugefügt.

Ja, ich verstehe, dass Sie bereits von der Tatsache abgelenkt sind, dass in Fettdruck in Großbuchstaben steht: „Innovation vom Feinsten“, obwohl es nach allen Regeln der Grammatik „sein“ sein sollte. Aber bitte lenken Sie Ihre Aufmerksamkeit auf die Aufschrift: "14 Personen betrachten derzeit dieses Produkt" ("Dieses Produkt wird derzeit von 14 Personen angesehen").



Sie können dies nicht in einem statischen Screenshot wiedergeben, aber anfangs zog die Zahl meine Aufmerksamkeit auf sich, da sie sich alle paar Sekunden sehr stark änderte. Ich habe mich entschlossen, einen Blick auf den Quellcode zu werfen, um zu verstehen, welche Methode beim Aktualisieren dieser Daten verwendet wird.

Für diejenigen, die sich nicht auskennen: Wenn ein Entwickler einige Elemente einer Website aktualisieren muss, nachdem sie bereits geladen wurde, werden normalerweise Client-Sprachen verwendet. Solche Sprachen führen Operationen in Ihrem Webbrowser aus und dementsprechend können Sie mit eigenen Augen beobachten, was unter der Haube passiert. Klicken Sie dazu einfach an einer beliebigen Stelle auf dem Bildschirm mit der rechten Maustaste und wählen Sie "Seitenquelle" in der Dropdown-Liste oder ähnliches. Anschließend können Sie den Code lesen, der für die Aktualisierung verschiedener Arten von Elementen verantwortlich ist, einschließlich der aktuellen Anzahl der Besucher auf der Seite.

Auf dieser Website habe ich Folgendes gefunden:



Ich habe das gewünschte Fragment ohne Änderungen kopiert. Ja, es wurde im Quellcode "prodiutc" geschrieben. Interessanterweise wird aus diesem Code jedoch deutlich, dass die Anzahl, die theoretisch die Anzahl der Personen widerspiegeln sollte, die das Produkt betrachten, zufällig bestimmt wird.

Ich werde diese Passage in die menschliche Sprache übersetzen:



Alle fünf Sekunden: Wählen Sie einen Container aus, in dem eine Nummer angezeigt wird, generieren Sie eine Zufallszahl im Bereich von 3 bis 14 und geben Sie diese Nummer in den ausgewählten Container ein.

Die Anzahl der Käufer, die sich für das Produkt interessieren, ist also nichts weiter als eine Fiktion. Der Code zeigt einfach alle fünf Sekunden eine beliebige Nummer auf der Seite an. Infolgedessen scheinen die Besucher ständig auf der Website zu zirkulieren, das Leben dort ist in vollem Gange, was bedeutet, dass es beliebt ist. Es ist davon auszugehen, dass eine große Anzahl hypothetischer Käufer das Geschäft auch in den Augen des Nutzers vertrauenswürdig macht. Darüber hinaus ermutigt es die Menschen, schneller einzukaufen - es scheint, dass die Waren, die sie interessieren, sehr gefragt sind.

Für den Fall, dass jemand in ferner Zukunft auf diesen Artikel stößt, wenn der Code bereits geändert wurde, habe ich einige gute Nachrichten: Am 16. Mai 2020 wurde die Site von Suchrobotern von Webarchiven indiziert - insbesondere haben sie das angegebene Fragment gespeichert . Sie können die indizierte Version der Site hier anzeigen .

Wenn Sie auf Wanderungen im World Wide Web auf einer anderen Website auf etwas Ähnliches stoßen, zögern Sie nicht, in den Quellcode einzusteigen - Sie wissen nie, welche Monster dort im Darm der Benutzeroberfläche lauerten. Ladenbesitzer lügen manchmal, aber der Code lügt nicht.
