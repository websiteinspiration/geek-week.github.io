カートリッジ内のシステム：エンジニアがゲームコンソールの機能を拡張した方法


では前回の記事、我々は興味深い機能と3つのカートリッジについて話しました：彼らは他のカートリッジが挿入されたにコネクタを持っていました。他のいくつかのカートリッジは、個々のビデオゲームコンソールをコンソールコネクタに接続できるようにすることでこのアイデアを開発しました。これらは通常のハードウェアアドオンではなく、メインシステムと互換性のない独自のカートリッジが必要でした。

今日は、3つのまったく異なるメーカーの3つのまったく異なる「カートリッジ内のコンソール」を見てみましょう。3つのシステムすべてが登場したことには、技術的および戦略的な理由がすべてあり、興味深い話をしてくれます。最も神秘的なシステムから始めましょう。ライセンスされておらず、コンソールの作成者が作成したものではない3つのうちの1つだけです。

アラジンデッキエンハンサー


1990年代初頭には、Codemastersオフィスに印象的な研究開発ラボがあったはずです。同社は当初、コモドアの家庭用コンピュータ向けのゲームを開発することを目的として作成されましたが、数年後、任天堂エンタテインメントシステム（NES）のライセンスのないゲームとハードウェアアドオンを作成する部門を開設しました。おそらく、少なくともコンソールプレーヤーの中で、この会社の最も有名な製品はGame Genie 1チートデバイスでしたが、NESのライセンスされていない機器の唯一の例ではないことが判明しました。

ビデオゲームカートリッジの最も驚くべき機能の1つは、その優れた汎用性です。技術開発の過程で、カートリッジはより強力になり、コンソール自体の機能を補完しました。 NES機器は1983年半ばに開発され、90年代の初めまでに彼はすでにあらゆる可能な支援を必要としていました。その後、RAM、メモリバンクを切り替えるためのコントローラー、特別なハードウェアマッパーなど、さまざまなコンポーネントがNESカートリッジに登場しました2。

異なるゲームのいくつかのカートリッジの内部には同じ機器があり、唯一の違いはROMチップに保存されたゲームデータにありました。 Codemastersの共同創設者であるRichard Darlingはこれに気づき、ゲームをより簡単で安価にするためのNESのアドオンを作成するという素晴らしいアイデアを思いつきました。このアイデアの結果が、1992年11月にリリースされたアラジンデッキエンハンサーです。エンハンサーは、ゲームデータ以外のすべてを含むNESカートリッジでした。いわゆる「コンパクトカートリッジ」（コンパクトカートリッジ）を接続するためのコネクターが付いていました。これらのコンパクトなカートリッジは、通常のNESカートリッジよりもはるかに小さくなると想定されており、ROMチップに記録されたゲームデータのみで構成されています。


コンパクトカートリッジの冒険家の目まい。

コードマスターは、エンハンサーが任天堂の公式ライセンスプログラムを回避し、コンパクトカートリッジでゲームをリリースすることに関心のあるパブリッシャーにとって、独自の実行可能なターゲットプラットフォームになることを望んでいました。サイズが小さく、従来のカートリッジに必要な回路のほとんどがすでにEnhancerに組み込まれているため、コンパクトカートリッジは通常のライセンスのNESゲームよりもはるかに安価に販売できます。 NES（SuperNES）の後継製品が市場に登場した後にリリースされたという事実を考えると、Enhancerのターゲットユーザーは、新技術だけでなくゲームだけに関心のあるファンドバイヤーに限られていたことは明らかです。

コードマスターにとって残念なことに、このアイデアは成功しませんでした。1993年末までに24のゲームをリリースするという約束にもかかわらず、リリースされたのは7ゲームのみであり、多くの売れ残ったデバイスは数十年間在庫に残っていました。信じられないことに、真新しい未開封のエンハンサーは、リリース後25年経ってもインターネット上で見つかります3。

スーパーゲームボーイ


Super Game BoyはSuper Nintendo Entertainment System（Super NES）を補完するものであり、Game Boyのゲームをプレイする素晴らしい機会を提供しました。 Super NESの通常のカートリッジと同じくらいの大きさのSuper Game Boyの上端には、標準のGame Boyゲームを挿入できるコネクタがありました。ゲームボーイカートリッジを挿入してスーパーゲームボーイをコンソールに取り付けることで、初めてプレイヤーはテレビでお気に入りのゲームをプレイする機会が与えられました。 Game Boyは、その粒子が粗く、ぼやけて、鈍い画面で悪名高いので、これは素晴らしい改善のように見えました。

現代のプレーヤーは古いゲームと新しいシステムの互換性に慣れていますが、90年代の場合、このアイデアはかなり新鮮でした4。 90年代のコンソールは、他のシステムのソフトウェアエミュレーションには十分強力ではなかったため、スーパーゲームボーイカートリッジには、通常のゲームボーイのすべての内部機器が実際に含まれていました。彼は主にカートリッジのゲームボーイコンソールにすぎなかったという事実にもかかわらず、スーパーゲームボーイには、さらにいくつかのトリックがありました。


スーパーゲームボーイのドンキーコングと特別なフレーム。

ユーザーは、スーパーファミコンのゲームを、ハンドヘルドデバイスのディスプレイではなく大きなテレビ画面でプレイする必要があったため、ゲームボーイゲームよりも解像度が高くなりました。つまり、ゲームボーイのゲームをテレビ画面に表示するとき、その画像がフレームに挿入されているはずです。プレーヤーは、多くの組み込み装飾フレームの1つを選択するか、仮想描画ツールを使用して独自のフレームを作成できます。さらに、プレーヤーはゲームに色を含めることができ、ゲームボーイの色のモノクロシェードをカスタムパレットに置き換えます。それ以降のゲームボーイのゲームの一部は、独自の組み込みフレームワークでリリースされ、スーパーゲームボーイのハードウェア機能を使用して、ゲームのより多くの色を表示することができました5。ゲーマーにとって残念なことに、そのような改善は根付きませんでしたが、それでもリリースされたものは驚くべきように見えます6。

Super Game Boyは1994年にリリースされました。GameBoyのライフサイクルの開始から5年後、Super NESのリリース後3年です。多くの点で、スーパーゲームボーイのリリースは非常に賢明な動きでした。まず、スーパーファミコンで利用できるゲームのライブラリを大幅に拡張し、すぐに300の追加ゲームへのアクセスを導入しました7。第二に、彼は敗北したゲームボーイに再び注意を向けました。任天堂はゲームボーイに第二の風を与えるのに長い間待たなかった：最初のポケットモンスターポケモンゲーム1996年に日本でリリースされ、ポップカルチャーの現象全体が生まれました。1990年代後半までに、任天堂はゲームボーイ8コンソールの3つの追加バージョンをリリースしました。最終的に、ゲームボーイのラインナップは、歴史の中で最も売れたビデオゲームシステムの1つになりました9。

スーパーゲームボーイ2は1998年に日本で発売されました。彼はスタイルを更新し、90年代後半の「透明なプラスチック」のファッションに対応し、多くの改良を加えました。 Super Game Boyの最初のバージョンは、元のポータブルコンソールのCPU速度と完全には一致していなかったため、ゲームの実行速度は2.4％高速でした。 Super Game Boy 2では、このバグが修正されました。さらに、ゲームボーイリンクケーブルコネクタが追加されました。これにより、2人が2番目のコンソール10と一緒にプレイできるようになりました。

全体として、スーパーゲームボーイはおそらく最も成功した「カートリッジシステム」でした。任天堂は2003年に同じアイデアを使用してGame Boy Playerをリリースしました。これにより、Game Boy Advance、Game Boy Color、および最初のGame BoyのGameCubeゲームをプレイできました。

セガ32X


私たちが検討している3つのシステムのうち、Sega 32Xは、少なくとも従来のゲーミングカートリッジに似ています。32 XがSega Genesisカートリッジコネクタに差し込まれたことで、この徐々に古くなるシステムに大幅なグラフィックス能力が追加されました。セガのデザイナーが、特にシステムの主な対象読者である10代の少年11を考慮して、ターボチャージャーとのバイヤー協会で強力なオイルカーを呼び起こそうとしたことは想像に難くない。

最初の2つのシステムとは異なり、32Xは単純なプラグアンドプレイデバイスではありませんでした。 32Xプレーヤーを接続する前に、Genesis内に金属製のステープルを取り付ける必要がありました。次に、32Xを（ジェネシスではなく）テレビに接続し、2番目のビデオケーブルで32Xをジェネシスに接続する必要がありました。最後に、32Xには専用の電源があり、大きな変圧器がコンセントから突き出ていました。システム全体を接続した後、Genesis / 32X設計の後部は混乱したワイヤーでした12。

1994年にリリースされた32Xは、創世記のリリースから5年以上経過して登場しました。当時、任天堂や他の強力な次世代コンソールを備えた競合他社からのプレッシャーが高まっていました。次世代のSegaコンソールはリリースの準備がほぼ整っていましたが、現在のSega Genesis 13の 4倍以上の次世代の価格で販売されます。

セガは、次世代ゲームに少し近づくために比較的少量を消費することを望む消費者のための巨大な潜在的な市場が存在することを望んでおり14、この戦略はうまくいくように思われた。しかし、日本では北米におけるスーパー32Xのリリース後の日、実際の土星と呼ばれる次世代のコンソール。 2つの新しいシステムを同時にサポートするように開発者を引き付ける試みは困難な作業であり、現在の次世代がすでに非常に近いため、次世代のコンソールへの「ハーフステップ」に対する消費者の需要は、それぞれの通過とともに低下しました。これはセガに圧力をかけ、計画されていなかった北アメリカで土星を打ち上げました。それは災害でした。

ただし、32Xが成功する世界を想像するのは簡単です。 32Xが少し早くリリースされた場合、または土星が少し遅れてリリースされた場合、32Xは人気を達成するための時間をおそらく持っています15。Genesisの価格を考慮しても、32Xは土星の半分の価格であり、ハードウェアの改善という点でプレイヤーに多くを提供することができます。残念ながら、32Xはセガの将来の成功の基礎となる代わりに、コンソール開発市場からのセガの段階的な撤退の前兆となりました。

アドオン追加


カートリッジコネクタは、システム用ではなくゲーム用ですが、勇敢なエンジニアも同様に、可能性の境界を拡大し、それに基づいて印象的なデバイスを作成しました。カートリッジから光ディスクへ、そしてコンテンツのデジタルダウンロードへの移行は、カートリッジスロットの時代が去り、終わったことを意味しました。

ハードウェアの追加は、ビデオゲームの現代史にも見られます。たとえば、ソニーを思い出してください。ただし、その使用の複雑さは変わっていません。アドオンは常にルーレットです。それらの共通の可能な市場は、メインシステムのユーザーの既存のベースによって制限されており、アドオンが市場にリリースされるときまでに、プレーヤーは多くの場合、次世代のテクノロジーを待っています。ほとんどのアドオンは大きな影響を与えません。しかし、彼らはできる プレイヤーに大きな利点がある場合の「シュート」-スーパーゲームボーイの成功はこれを証明しています。
