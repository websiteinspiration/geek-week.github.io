触摸终端，适用于恶劣环境


研华TPC设备是用于运行人机界面（HMI）应用程序的瘦客户端。它们是功能齐全的x86计算机，具有内置于带触摸屏的安全监视器中的被动冷却功能。

HMI终端专为恶劣的工业环境而设计。操作员可以在不脱下手套的情况下以及在污染加剧的情况下控制终端，而电阻式触摸面板则可以避免在滴下液体时意外按压。

在本文中，我们测试了手套中的设备并分析了几种模型的特性。


演示：戴着手套的触摸屏



在工业环境中，液滴，碎屑，灰尘和其他污染物会进入终端屏幕。因此，重要的是，污染不能引起屏幕上的误点击，就像流行的电容式触摸屏一样。只需记住雨滴如何引起智能手机屏幕上的意外点击即可。

对于这种情况，电阻型触摸屏更适合。它们可以在不戴手套的情况下进行控制，并且在受到污染的情况下可以防止误报。该视频演示了触摸屏的相当高的准确性，而操作员使用的是粗糙的橡胶手套，单击较小的界面元素并不困难。


[视频]您可以使用带手套的电阻屏工作，

该视频显示了Windows 10上Google Chrome Web浏览器中运行的SCADA演示系统。

技术指标


TPC系列终端基于Intel Atom处理器，支持64位OS和完全被动冷却。在TPC-1840WP和TPC-2140WP机型采用了AMD T56E平台具有更强大的图形子系统运行要求苛刻的OpenGL应用程序。所有设备上的RAM量为4GB。


型号TPC-1551T与后盖取下

该系列的所有器件具有UEFI支持。可以通过PXE通过网络下载终端，或者可以在终端中安装2.5英寸HDD / SDD磁盘或CFast存储卡以本地加载操作系统。可以使用本地驱动器备份关键数据。
尽管设备的概念涉及将它们用作瘦客户端，但该系统的性能足以启动Windows 10等台式机操作系统和完整的浏览器。


后面板上有以下接口：

2x-RS232 / RS485
1x-USB 3.0
3x-USB 2.0
2x-具有网络启动支持的独立以太网
iDoor扩展模块插槽也
提供带有音频端口In / Out，HDMI的型号，用于连接第二台显示器。
大多数TPC型号都有一个完整尺寸的mPCIe插槽，用于安装扩展卡。

IDoor扩展模块



通过专有的研华iDoor模块可以扩展TPC接口集。这些模块是带有外部接口面板的mPCIe模块。
在iDoor模块的外形尺寸中，专有工业接口有很多选择，例如PROFIBUS，EtherCAT，PROFINET，3G / 4G调制解调器的网络接口模块，WiFi /蓝牙以及数字量输入和输出模块。

安装




TPC终端可以作为榫眼面板嵌入到与表面齐平的位置。为此，在设备的所有末端都需要安装支架和螺孔。在每台设备的页面上，都可以查看3D模型。



也可以安装在标准VESA支架100x100或品牌支架TPC-1000H-SMKE / WMKE上，以安装在桌子或墙壁上。

电气合规


所有设备均通过了符合国际电工委员会（IEC）标准和IEC60664第2部分的测试。它们具有防止不正确的电源连接（反接正负接反）的保护措施，可以承受短期的电源电压升高，并可以持续升高的电压高达20％从名义上。

在恶劣条件下工作


所有TPC设备均可在-20至60 C°的宽温度范围内运行。符合IEC60068-2-64和IEC60068-2-27，可承受高达10G的恒定振动和冲击。
如果将端子正确安装在插板中，则前面板的防护等级将达到IP66等级。

为什么需要HMI终端？


使用瘦客户端可以使您集中管理和更新嵌入式系统中的接口。完整的操作系统使使用任何熟悉的开发工具（包括Web应用程序）成为可能。结果，与构建硬件控制面板或对专用微控制器进行编程相比，此类接口的开发速度明显更高。

一体式显示面板无间隙和开口，可避免液体和污染物进入设备，并且电阻式触摸屏使您可以戴着手套，指针和笔进行操作。

扩展模块的存在使您可以向终端添加任何工业接口，以及用于使用HMI应用程序本地处理数字信号的离散I / O端口。在研华HMI Designer程序中创建自己的HMI界面的示例，我们在研华基于Havan控制面板的文章中回顾了研华的HMI
