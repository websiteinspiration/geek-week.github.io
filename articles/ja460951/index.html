Androidライブ壁紙を作成する

私は彼の状態、より正確にはテストプール内の彼の状態に関する技術情報を電話画面に表示する必要があります。私は常にこの情報を、つまりホーム画面で、追加の身体の動きなしで見たいと思っています。


他のアプリケーションの実行に影響を与えない方法は、ウィジェットまたはライブ壁紙の2つだけです。私はライブ壁紙を選択しました。ホーム画面のすべてのページ、さらにはロック画面に自動的に移動するため、これらも「ライブ壁紙」です。この記事では、ライブ壁紙の作成方法に関する実用的なガイダンスを提供します。


真実を求めて


「ライブ壁紙」の猫に関するドキュメントは泣いた。9年以上前に行われたブログでの最初の（そして唯一の）発表以来、Googleはこのトピックについて単一のわかりやすい例やコードラボを作成していません。私は理解しなければなりませんでした。


最初の基本。Androidの内部構造は、デバイスにアプリケーションをインストールすることのみが可能であり、すべてのアプリケーションのデバイスは同じです。「ライブ壁紙」もアプリケーションであるため、コントロールコンポーネントの選択はあまりよくありません。サービスになることを期待する必要があります。それを見つけるのは簡単です。それはWallpaperServiceです。


「ライブ壁紙」のインスタンスがいくつか存在する可能性があり、それらのライフサイクルはアクティビティまたはビューのライフサイクルと同じではありません。したがって、もう1つの基本クラスが必要です。これは、WallpaperService.Engine（そしてそれが必ずしもあるインナー用WallpaperService！）。よく見ると、アクティビティ、サービスなどのライフサイクルイベントのプロバイダーと同じです。

fun drawSynchronously(holder: SurfaceHolder) {
    if (!isVisible) return

    var c: Canvas? = null
    try {
        c = holder.lockCanvas()
        c?.let {
            dummyDraw(it)
        }
    } finally {
        c?.let {
            holder.unlockCanvasAndPost(it)
        }
    }
}

override fun onSurfaceCreated(holder: SurfaceHolder?) {
    super.onSurfaceCreated(holder)
    redrawHandler.planRedraw()
}

override fun onSurfaceChanged(holder: SurfaceHolder?, format: Int, width: Int, height: Int) {
    super.onSurfaceChanged(holder, format, width, height)
    redrawHandler.planRedraw()
}

override fun onVisibilityChanged(visible: Boolean) {
    super.onVisibilityChanged(visible)
    redrawHandler.planRedraw()
}    

override fun onSurfaceDestroyed(holder: SurfaceHolder?) {
    super.onSurfaceDestroyed(holder)
    redrawHandler.omitRedraw()
}     

override fun onDestroy() {
    super.onDestroy()
    redrawHandler.omitRedraw()
}

override fun onSurfaceRedrawNeeded(holder: SurfaceHolder) {
    super.onSurfaceRedrawNeeded(holder)
    redrawHandler.omitRedraw()
    drawSynchronously(holder) // do it immediately, don't plan
}

val redrawHandler = RedrawHandler()

inner class RedrawHandler : Handler(Looper.getMainLooper()) {
    private val redraw = 1

    fun omitRedraw() {
        removeMessages(redraw)
    }

    fun planRedraw() {
        omitRedraw()
        sendEmptyMessage(redraw)
    }

    override fun handleMessage(msg: Message) {
        when (msg.what) {
            redraw -> drawSynchronously()

            else -> super.handleMessage(msg)
        }
    }
}
