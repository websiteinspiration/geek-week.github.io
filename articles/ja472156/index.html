ページオブジェクトとは


これは、Web製品テストの自動化における事実上の標準である一般的なパターンです。基本的な考え方は、テストロジックを実装から分離することです。


各プロジェクトのWebページは、クラスオブジェクトとして説明できます。ユーザーインタラクションはクラスメソッドで記述され、ビジネスロジックのみがテストに残ります。このアプローチは、Webアプリケーションのレイアウトを変更するときのテストの問題を回避するのに役立ちます。ページを説明するクラスのみを修正する必要があります。


ページオブジェクトはパーツを定義します。


  • Base Page \ Base Class-Webdriverを操作するために必要なメソッドを実装します。
  • Page Object \ Page Class-Webページの要素を操作するためのメソッドを実装します。
  • テスト-テストケースのビジネスロジックによって記述されたテストを実装します。

ページオブジェクトパターンのスキーマ。

トピックを明確に説明するために、自動テストを実装します。

from selenium.webdriver.support.wait import WebDriverWait
from selenium.webdriver.support import expected_conditions as EC

class BasePage:

    def __init__(self, driver):
        self.driver = driver
        self.base_url = "https://ya.ru/"

    def find_element(self, locator,time=10):
        return WebDriverWait(self.driver,time).until(EC.presence_of_element_located(locator),
                                                      message=f"Can't find element by locator {locator}")

    def find_elements(self, locator,time=10):
        return WebDriverWait(self.driver,time).until(EC.presence_of_all_elements_located(locator),
                                                      message=f"Can't find elements by locator {locator}")

    def go_to_site(self):
        return self.driver.get(self.base_url)

from BaseApp import BasePage
from selenium.webdriver.common.by import By

class YandexSeacrhLocators:
    LOCATOR_YANDEX_SEARCH_FIELD = (By.ID, "text")
    LOCATOR_YANDEX_SEARCH_BUTTON = (By.CLASS_NAME, "search2__button")
    LOCATOR_YANDEX_NAVIGATION_BAR = (By.CSS_SELECTOR, ".service__name")

class SearchHelper(BasePage):

    def enter_word(self, word):
        search_field = self.find_element(YandexSeacrhLocators.LOCATOR_YANDEX_SEARCH_FIELD)
        search_field.click()
        search_field.send_keys(word)
        return search_field

    def click_on_the_search_button(self):
        return self.find_element(YandexSeacrhLocators.LOCATOR_YANDEX_SEARCH_BUTTON,time=2).click()

    def check_navigation_bar(self):
        all_list = self.find_elements(YandexSeacrhLocators.LOCATOR_YANDEX_NAVIGATION_BAR,time=2)
        nav_bar_menu = [x.text for x in all_list if len(x.text) > 0]
        return nav_bar_menu
