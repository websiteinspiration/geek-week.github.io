Raspberry Pi上のSwiftプログラミング言語


Raspberry PI 3モデルB +

このマニュアルでは、Raspberry PiでのSwiftの使用の基本について説明します。 Raspberry Piは小型で安価なシングルボードコンピューターであり、その可能性はそのコンピューティングリソースによってのみ制限されます。彼は技術者やDIY愛好家によく知られています。これは、アイデアで実験したり、実際に特定のコンセプトをテストしたりする必要がある人にとっては素晴らしいデバイスです。さまざまなプロジェクトで使用され、ほぼどこにでも簡単に適合します。たとえば、モニターカバーに取り付けてデスクトップとして使用したり、ブレッドボードに接続して電子回路を制御したりできます。

公式のRaspberryプログラミング言語はPythonです。Pythonはかなり簡単に使用できますが、型の安全性に欠けている上、大量のメモリを消費します。対照的に、SwiftはARCメモリ管理を備えており、Pythonのほぼ8倍高速です。Raspberry PiプロセッサのRAMとコンピューティング機能は限られているため、Swiftのような言語を使用すると、このミニPCのハードウェアの可能性を最大限に引き出すことができます。

Swiftをインストールする


「ラズベリー」にSwiftをインストールするには、それをインターネットに接続する必要があります（ボードのモデルに応じて、イーサネットまたはWiFiを使用）。インターネットに接続したら、Swiftのインストールを開始できます。

それは2つの方法で行うことができます。 1つ目はSwiftビルドを個別に作成すること、2つ目は既にコンパイルされたバイナリを使用することです。前者は数日間の準備が必要になるため、後者の方法を強くお勧めします。 2番目の方法は、Swift-ARMグループのおかげで登場しました。彼女はapt（A Advanced P ackage T ool）を使用してSwiftをインストールできるリポジトリを所有しています。

これはコマンドラインツールであり、アプリケーション用のApp StoreやLinuxデバイス用のパッケージのようなものです。端末にapt-getを導入して、aptの操作を開始します。次に、実行中のタスクを明確にするいくつかのコマンドを指定する必要があります。今回の場合、Swift 5.0.2をインストールする必要があります。関連するパッケージはここにあります。

さて、始めましょう。aptでSwiftをインストールすることがわかったので、リポジトリのリストにリポジトリを追加する必要があります。

add / install repo swift-armコマンドは次のようになります。

curl -s <https://packagecloud.io/install/repositories/swift-arm/release/script.deb.sh> | sudo bash



次に、追加したリポジトリからSwiftをインストールします。

sudo apt-get install swift5=5.0.2-v0.4



それで全部です！これでSwiftが「ラズベリー」にインストールされました。

テストプロジェクトを作成する


現時点ではSwift REPLは機能しませんが、それ以外はすべて可能です-はい。テストでは、Swift Package Managerを使用してSwiftパッケージを作成しましょう。

まず、MyFirstProjectというディレクトリを作成します。

mkdir MyFirstProject



次に、現在の作業ディレクトリを新しく作成したMyFirstProjectに変更します。

cd MyFirstProject



新しいSwift実行可能パッケージを作成します。

swift package init --type=executable



これらの3行により、MyFirstProjectという空のSwiftパッケージが作成されます。実行するには、swift runコマンドを実行します。



コンパイルが完了すると、「Hello、world！」というフレーズが表示されます。コマンドラインで。

Piで最初のプログラムを作成した後、何かを変更しましょう。MyFirstProjectディレクトリーで、main.swiftファイルに変更を加えます。これには、swift runコマンドでパッケージを実行したときに実行されるコードが含まれています。

ディレクトリをSources / MyFirstProjectに変更します。

cd Sources/MyFirstProject



組み込みのnanoエディターを使用してmain.swiftファイルを編集します。

nano main.swift



エディターが開いたら、プログラムのコードを変更できます。main.swiftファイルの内容を次のように置き換えます。



print("Hello, Marc!")

もちろん、あなたの名前を挿入することができます。変更を保存するには、以下のアクションを実行する必要があります。

  • CTRL + Xでファイルを保存します。
  • 「Y」を押して変更を確認します。
  • Enterキーを押して、main.swiftファイルへの変更を確認します。





すべての変更が行われたので、プログラムを再起動します。

swift run



おめでとう！コードがコンパイルされるとすぐに、ターミナルには変更された行が表示されます。

Swiftがインストールされたので、何かする必要があります。したがって、LED、サーボ、リレーなどのハードウェアを制御するために、SwiftyGPIOと呼ばれるLinux / ARMカード用の「鉄」プロジェクトのライブラリを使用できます。
