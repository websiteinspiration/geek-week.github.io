电子邮件营销和其他领域的同类群组分析

您越了解客户，您使用的营销策略就越有效。为了找出联系人数据库中的人，经常使用细分：按性别，年龄，购买的商品。这很好。



重要的是不要忘记，如果性别是唯一可以将某个部门的人团结在一起的事物，那么这些人就是非常不同的人，因为基于一个部门进行的细分并未考虑与公司互动的不同阶段。例如，无论性别，新客户与忠实客户都不相同。



为了评估客户在与公司互动的整个过程中的行为，使用了队列分析，结果将有助于：



  • 个性化沟通；
  • 防止客户流失；
  • 优化您的广告费用。


什么是同类群组分析和同类群组



同类群组是一组由通用功能和时间间隔结合在一起的用户。例如，在同一周内已注册该服务的用户。他们在一个间隔内执行了相同的操作。时间分布将同类群组与细分受众群区分开。



同类群组分析是市场营销和产品分析中最受欢迎的分析之一。它显示了每个队列中关键指标随时间的变化。因此，根据用户何时采取行动，就有可能跟踪特定目标的实现：转化，ROI，CAC，LTV，保留率-并了解前提。例如，您看到第四周的用户比第一周的用户转换得更好。而且您知道在第三周就开始了入职培训。因此，入职有助于更好地转换用户。

根据业务需求和特征选择所有参数。群体的征兆和规模取决于关键指标的选择。

电子邮件营销中的同类群组分析



许多公司在评估电子邮件营销时会使用广义的统计数据：公开程度，可点击性，每月或单独退订的指标。但是，这种评估无法回答谁与新闻通讯进行交互以及如何进行交互的问题。一种选择是细分用户。好，但还不够，因为行为不同且品牌期望不同的“新”和“旧”订户都可以归为一类。为了获得更详细的信息，有必要使用队列分析。



使用它，您可以：



  • 跟踪在不同时间注册的用户之间的互动动态；
  • 找出最活跃的人群；
  • 在队列中检验假设；
  • 建立更多的个性化沟通；
  • 确定订户的寿命；
  • 按时开始重新激活。


如果您在同类群组分析中添加其他细分，信息将更加详细，您将获得更多有用的见解。



我将向您展示一个示例，该示例说明如何使用构建同类群组分析的基本方法而无需进行额外的细分。这足以理解构造原理。



建立基本同类群组分析的示例



问题的提法



在此示例中，同类群组分析的目的是基于点击率（点击数）跟踪订户与新闻通讯的交互。



参数定义



分析将基于以下指标。



  1. 队列形成的标志是订阅日期。
  2. 队列的大小是一个月。
  3. 时间间隔是从2018年1月到2019年11月。
  4. 关键指标是CR。


进一步的工作步骤将取决于参数值。



初始数据



在表1中，我们收集有关订户的信息：



  • 订户ID（电子邮件可以是ID）；
  • 订阅日期。






在表2中-有关交互的信息（一行-一项操作）：



  • 订户编号
  • 生效日期；
  • 提交操作的邮件。






原始表格已经显示了唯一的点击次数。如果您的源数据包含非唯一值，请在开始工作之前从集合中删除重复的行。例如，如果一个人双击一个链接，就会形成它们，并且由于某些平台会提供他们拥有的所有数据，因此表中将出现两行相同的行-区别仅在于这些事件发生的时间。



工作算法



  1. 我们会根据订阅的日期将用户分组，并考虑到同类群组的规模。我们获得每个队列中的订户数量。
  2. 我们将表2和表1结合在一起，使用一个公共字段-订户ID作为关键字。
  3. 我们将从第2节获得的表格分组到“操作完成日期”和“订阅日期”字段中，并计算出此类对的点击总数。
  4. 我们为每对计算关键指标的值。


结果分析



结果以矩阵的形式表示，其中行是同类群组，列是月份，值是关键指标（CR）。







例如，对于2018年2月注册的用户，当月的CR百分比为2.11％，但在2018年3月，他们的点击次数更为活跃-14.20％。

其他应用



同类群组分析是一种通用方法，可让您分析各种信息。同类群组分析通常用于：



  • 评估广告渠道的有效性；
  • 评估A / B测试结果；
  • 形成理想客户的肖像；
  • 优化SaaS业务的销售周期；
  • 分析移动应用的有效性；
  • LTV的定义。


制作工具



最简单的



要建立同类群组分析，您可以使用Excel，Google表格等。这些工具的可用性是一个主要优势。在这些缺点中，可以突出显示源数据的大小有限，更改参数时无法快速重新计数，缺少自动更新等。



具有适当功能的工具



一些广告和分析服务具有用于构建同类群组分析的内置功能。这种方法的优点是用户只需要选择参数，就可以快速建立同类群组分析报告。



这种方法的缺点包括：



  • 平台限制的参数集；
  • 需要使用该系统进行数据收集，这意味着跟踪分析和广告。


这些系统包括：



  • 谷歌分析
  • Yandex.Metrica（AppMetrica）
  • 调整
  • Roistat
  • Kissmetrics
  • 其他




通用工具
