VS Codeの10個の拡張機能なしではプログラムできません

VS Codeは私のお気に入りのコードエディターです。これは既存のエディターの中で最も人気があり、その可能性はほぼ無制限に拡張できます。そして、驚いたことに、それはマイクロソフトによって開発されました。他のエディターやIDEはどれも、開発者にVS Codeが提供できるものの少なくとも半分を与えることはできないと思います。VS Codeの長所の1つは、その拡張システムです。それはあなたが文字通りあらゆる機会のために拡張を作成することを可能にします。VS Codeの拡張機能トップ10についてお話ししたいと思います。



1.美化



拡張機能をインストールするには、VS Codeのキーボードショートカットを使用し、CTRL+P開いたフィールドに以下のコマンドを入力してをクリックしますEnter。

ext install HookyQR.beautify

美しく拡張子ができますあなたのフォーマット、HTML、JS、CSS、JSONとSASSコードに。書式設定オプションを制御できます。この拡張機能は、標準のjs-beautifyシステムのアドオンであり、開発者は必要に応じてコードを正確にフォーマットできます。

2.より良いコメント



ext install aaron-bond.better-comments

より良いコメント拡張子を使用して、コメントを操作するための可能性を拡張することができます。特に、コメントをカテゴリに分割できます。これらは、通知、リクエスト、タスクリスト、メモにすることができます。


より良いコメントの使用

3.ブックマーク



ext install alefragnani.Bookmarks

ブックマーク 拡張機能は、プログラマーにとって真のライフラインです。特に、何千行ものコードの中で失われたもののために。この拡張機能により、コード行にブックマークを添付できます。コードの行にブックマークがある場合、ブックマークリストを使用してこの行に簡単に移動できます。


コード内のブックマーク

4.ブラケットペアカラライザー2



ext install CoenraadS.bracket-pair-colorizer-2

Bracket Pair Colorizer 2 拡張機能を使用すると、ブラケットに色を付けることができ、開閉ブラケットのペアを見つけやすくなります。これは、ネストされたソフトウェア構造で作業する必要がある場合に非常に役立ちます。


ブラケットのペアが強調表示されます。

5.コンテキストメニューでのフォーマット



ext install lacroixdavid1.vscode-format-context-menu

Format in Context Menus 拡張機能を使用すると、サイドバーでファイルを強調表示して、コンテキストメニューコマンドを呼び出すことにより、ファイルをフォーマットできます。これは、フォーマットする必要のあるファイルが多数ある場合に特に便利ですが、同時に、既存の環境では、コードのフォーマットとリンティングの手段がサポートされていません。


コンテキストメニューのフォーマットの操作

6. Gitグラフ



ext install mhutchie.git-graph

Gitグラフ 拡張機能を使用すると、リポジトリの構造を表示し、この構造を使用してさまざまな操作を実行できます。この拡張機能は高度にカスタマイズ可能で、多くの機能があります。おそらく、それを適切に説明するには、完全に別の記事が必要になります。


Gitグラフを操作する

7. GitLens



ext install eamodio.gitlens

GitLens拡張子が役立ちます特定のgit非難注釈やレンズを使用してコード原作者。この拡張機能は、特に、リポジトリのコンテンツを非常に便利に表示し、コードに関する有用な情報を学習することを可能にします。


GitLensを使用する

8.インデントレインボー



ext install oderwat.indent-rainbow

インデントレインボー 拡張機能を使用すると、インデントに色を付けて、異なるレベルのインデントを異なる色で強調表示できます。これは、Pythonなどの言語で深くネストされた構成要素を操作する場合に特に便利です。

9.パスインテリセンス



ext install christian-kohler.path-intellisense

Path Intellisense 拡張機能は、エディターにファイルパスをオートコンプリートする機能を提供します。VS Codeは拡張なしのパス補完をサポートしていますが、この機能はHTML、CSS、JavaScriptファイルにのみ制限されています。この拡張機能のおかげで、ファイルや言語を操作するときにパス補完を使用できます。


Path Intellisenseを使用する

10.合計ライン



ext install praveencrony.total-lines

Total Lines 拡張機能は、開いているファイルの行数に関する情報をステータスバーに表示する小さな便利なツールです。これは私の拡張機能のコレクションへの素晴らしい追加です。
