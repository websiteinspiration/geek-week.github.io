フィルムオーディオ形式の歴史：コンパクトカセットに対するドイツの回答-開発の運命はどうでしたか

フィデリパック、ステレオパック、ステレオ8、RCAサウンドテープについては、すでに説明しました。それらのすべてはアメリカで開発されました。しかし、ヨーロッパの企業は同様のプロジェクトに従事していた。

今日はDC Internationalについて話します。彼はドイツの企業グルンディッヒによって紹介されました。


写真：Ulrich Miemietz / CC BY-SA / DCインターナショナルファーストプレーヤー-グルンディッヒC100

フォーマットの誕生


DCインターナショナルの歴史は、フィリップスが開発した有名なコンパクトカセットと密接に関連しています。オランダの会社は、1963年にドイツのエレクトロニクス展示会でそれを一般大衆に最初に示しました。当初、テープは会話コンテンツ用であり、音楽の録音にはあまり適していませんでした。しかし、すぐに、開発者はノイズをマスクして音質を改善するテクノロジーを導入しました。彼らの決定は人気を得始めました。

現時点では、ドイツの懸念であるグルンディッヒの代表であるフリードリッヒラクナーは、ソニーにオランダ製品と競合する新しいフォーマットを設計することを提案しました。しかし、ラーナーは幸運ではありませんでした-同時に、フィリップス電子部門の1つの従業員は日本に連絡しました。彼らは、ソニーがパートナーシップ契約を締結し、共同でコンパクトなカセットフォーマットを開発することを提案しました。会社の経営陣はすべての選択肢を検討し、最終的にオランダの同僚の申し出を受け入れました。しかしグルンディッヒは彼らの考えを放棄しなかった。

彼らは無線機器メーカーのTelefunkenとBlaupunktのサポートを求め、1965年にDC Internationalフォーマットを導入しました。 DC は頭字語を表しますフィルムの付いた2つのリールがカートリッジのプラスチックケースに隠されているため、ダブルカセット（「2カセット」）のように。

写真：Ulrich Miemietz / CC BY-SA
DCはコンパクトカセット（120×73×12 mm）よりも大きく、両側に最大45分の音声コンテンツが含まれていました（90分のバージョンと120分のバージョンがありました）。テープの速度は5.08cm / sで、40Hz〜10KHzの周波数範囲の音を再現することができました。

グルンディッヒは、再生デバイスであるグルンディッヒC100Lもリリースしました。同様のデバイスがテレフンケンによって導入されました-それはマグネトフォン401と呼ばれていました。ブラウプンクト社に関しては、そのエンジニアは標準的なラジオセットに接続された車のための2つの専用プレーヤーを開発しました。

ショートライフDCインターナショナル


フォーマットの配布を促進するために、グルンディッヒ幹部は、他の会社によるカートリッジ製造のライセンス料を完全に拒否しました。フィリップスはカセットの製造も許可しませんでしたが、製造用の機器の使用料を徴収しました。

1965年のDCインターナショナルの発売以来、主要な音楽グループによるいくつかのリリースがカートリッジに登場しています。たとえば、ローリングストーンズコレクションなどです。しかし、DCインターナショナルはコンパクトカセットを破ることに失敗しました。ドイツの会社の形式は、フィリップス製品より2年遅れて光を見たので、重要な聴衆を魅了することはできませんでした。

1967年、グルンディッヒは敗北を受け入れるしかありませんでした。同社は、ビデオおよびオーディオ機器の生産に焦点を当て、オーディオテープの開発を拡大しました。


LORA / Unsplash による写真

しかし、DCインターナショナルは完全に忘れられていません。今日でも、愛好家は古いカートリッジを集めてプレイヤーをアップグレードします。たとえば、アーカイブと非公開のビデオおよびオーディオ録音をデジタル化する会社であるDigiOmmelの従業員は、Grundig C100Lテープレコーダーを購入し、それに最新の低ノイズプリアンプをインストールしました。古いゲルマニウムトランジスタからの干渉を減らし、DC Internationalカートリッジの再生品質を改善しました。

また、ネットワーク上で、著者がこのフォーマットを熱心に思い出したビデオを見つけることができます。
