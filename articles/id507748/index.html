Juga menggunakan contoh dari sumber daya https://scadavis.io/learn.html dan fungsionalitas node Templat, Anda dapat memproses berbagai peristiwa objek SCADA dan meneruskannya ke node lain, misalnya seperti ini: 
(function(scope) {
substationsynoptic.on("click", function (event, tag) {
    var v = substationsynoptic.getValue(tag);
    if (event.currentTarget.id === "TAPUP")
        substationsynoptic.setValue(tag, v + 1, false, false);
    else
        if (event.currentTarget.id === "TAPDOWN")
            substationsynoptic.setValue(tag, v - 1, false, false);
    if (event.currentTarget.id === "XCBROPEN")
        substationsynoptic.setValue(tag, false, false, false);
    else
        if (event.currentTarget.id === "XCBRCLOSE")
            substationsynoptic.setValue(tag, true, false, false);
    if (v === true)
        substationsynoptic.setValue(tag, false, false, false);
    else
        if (v === false)
            substationsynoptic.setValue(tag, true, false, false);
    scope.send( {'payload':{ 'tag': tag, 'value': v }} );
});
})(scope);




Editor grafis



Beberapa kata tentang editor grafis untuk membangun halaman visual untuk sistem SCADA. Seperti disebutkan di atas, di luar kantor diusulkan untuk membeli Editor Sinoptik SCADAvis.io. Sendiri, lagi - ini bukan satu-satunya pilihan. Seperti yang Anda ketahui dari deskripsi itu sendiri - editor didasarkan pada Inkscape SVG Editor. Dalam praktiknya, ini adalah Inkscape dengan plugin yang memungkinkan Anda untuk menetapkan tag (variabel) dan properti lainnya ke berbagai komponen grafis dari file svg. Itu Untuk dengan mudah menampilkan halaman dalam konteks scadavis, cukup untuk memiliki file svg, yang dapat diedit di editor yang nyaman. Tapi itu belum semuanya. Ada banyak kesamaan antara editor ECAVA INTEGRAXOR www.integraxor.comInkscape 0.91 SAGE 4.16 didistribusikan secara gratis (Anda hanya perlu mendaftar di situs) dan Editor Sinoptik. Ternyata, sangat mungkin untuk membuat dan mengedit file svg untuk Scadavis menggunakan Inkscape 0.91 SAGE 4.16. Perbedaannya diamati ketika merancang animasi skrip dan beberapa hal spesifik lainnya yang hilang dalam properti Object dari editor SAGE.







Menurut deskripsi Editor Sinoptik SCADAvis.io , untuk menetapkan warna objek sesuai dengan kondisi inklusi, perlu untuk menunjukkan nomor 2 di kolom Batas:



  • 0 - tidak diinisialisasi
  • 1 - false (off) state
  • 2 - kondisi benar (aktif)
  • 129 - false (off) state plus nilai gagal
  • 130 - kondisi true (on) ditambah nilai gagal






Plugin PLC untuk Siemens



Arsenal Node-red memiliki plug-in untuk bekerja dengan Siemens PLC - node-red-contrib-s7 , berdasarkan kerangka nodeS7 . Plugin ini cukup sederhana dan intuitif untuk digunakan.
