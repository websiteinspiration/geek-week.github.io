コミュニティまたはVKページからDiscordに投稿を自動的に公開する


バックグラウンド


私は参加者の平均数、Discordサーバー〜2000参加者の作成者です。
ある日、最新のゲームニュースを含むVKontakteグループのエントリを公開する必要がありました。VKontakteはWebHooksとの連携方法がわからないため、Googleにアクセスして適切なソリューションを探す必要がありました。

いくつかの解決策が見つかりましたが、期待どおりに機能しなかったか、まったく機能しませんでした。それからJavaScriptの知識があったので、自分で書くことにしました。

そこで登場したVK2DISCORD


私はスクリプトの最初のバージョンを作成しました。これは、投稿からテキストと1つの画像しか受信できず、WebHookを使用してすべてをメッセージとしてDiscordチャネルに送信できました。スクリプトは、Githubのオープンスペースで公開されました。しばらくして、私は劇的な改善が必要であることに気づき、以下を追加してスクリプトを完全に書き直しました。
  • 投稿のすべての添付ファイルを取得する
  • 再投稿処理
  • VK LongPoll APIサポート
  • キーワードと＃ハッシュタグによる投稿の選択
  • 一般ユーザーの壁から投稿を取得する


スクリプトの例


スクリプトの設定


1. 公式サイトからNode.jsをダウンロードしてインストールします。

2. リポジトリからスクリプトをダウンロードし、便利な場所に解凍します。

3.スクリプトを解凍したフォルダーのconfig.jsonファイルを構成します。これは次のような構造になっています。

注意：スクリプトは頻繁に更新されるため、構成構造が変わる可能性があります。設定する前に、スクリプトリポジトリのメインページを参照することをお勧めします

{
 "clusters": [
  {
      "vk": {
      "token": "   (   https://vk.cc/9bJ69C)   ",
      "group_id": "club1", // ID        .

      "keywords": [],      //  ,  ,   .       -    .        : #news@stevebotmc.   ,      .

      "filter": true,      //      ,     . 

      "longpoll": false,   //  Longpoll API. 

      "interval": 30 //        .
   },
   "discord": {
      "webhook_urls": [  //   Webhook,        Discord.
         "https://discordapp.com/api/webhooks/",
         "https://discordapp.com/api/webhooks/"
      ],
      "bot_name": "VK2DISCORD", //   WebHook, c    .
      "color": "#aabbcc"  //    Discord   HEX.
      }
   },
   {
      "vk"...     //     ,        .    .
   }
   ],
   "version_dont_modify_me": 1 //  ,   .
}

4.コマンドラインを開き、次のコマンドを使用してスクリプトを実行します。

  • スクリプトフォルダーへのcdパス
  • npm i
  • ノードindex.js

セットアップが完了し、最初のポストは指定された間隔の後に受信されます。

リポジトリにリンクしてください。改善に貢献したり提案したりできます。
