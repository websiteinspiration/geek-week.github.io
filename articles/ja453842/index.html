マイクロソフトがノイズをカットして、有用で美しいサウンドの世界を作成する方法

メッセージが届いたとき、バッテリーが切れたとき、または会議のリマインダーが表示されたときにコンピューターが鳴る音について考えたことはありますか？マシュー・ベネットは考えていました。私はたくさん考えました。

Bennettは、完璧なものを選択する前に、900以上のバージョンのWindows 10カレンダーアラートサウンドを作成、実行、処理しました。

「長い時間でしたが、うまくいきました」ベネットは、レドモンドにある彼のレコーディングスタジオを最近訪れたときに肩をすくめながら思い出しました。キャンドルに照らされた防音室には、ミキサー、いくつかの高品質のスタジオモニター、大きなLCD画面があり、中央にはフルオクターブのシンセサイザーキーボードがあります。

多くのマイクロソフトのアプリケーションやデバイスに取り組んでいる作曲家およびサウンドエンジニアとして、ベネットは15年間、会社のサウンドデザインで重要な役割を果たしてきました。彼は強い信念と音に関する一般的な哲学を持っています、そして彼はこれらの問題を議論する専門用語に堪能です。



この記事は当サイトにあります。

彼の役割を要約すると、彼は次のように述べています。「消費者に対する私たちの責任は、まず第一に、害を及ぼさないことです。耳を切らさず、邪魔にならないサウンドを作成します。さらに、サウンドは機能的で美しいものでなければなりません。美しさと機能性は密接に関連しています。音が美しいほど、人々に呼び起こしたい製品の全体的な印象に良い影響を与えます。」


ベネット氏によると、1年間、4人のアシスタントと共に、Windows 10のサウンドファミリを作成して、自分の仕事を完璧なものにしました。 「狂ったように、私はこれらの音を整理していました。それは私がそれがどれくらいの時間がかかったか少し気になります。私はコントロールにこだわっています。」

53歳のベネットは、キーボードを演奏し、西アフリカの音楽に興味を持っているジャズファンで、音楽の社会的および文化的側面の科学である音楽民族誌学の修士号を取得しています。 2000年代初頭、彼は10年間マイクロソフトを去り、フリーランスの作曲家として働き、サウンドデザインを作成し、音楽を書きました。

この間、彼は映画やフォーチュン500企業向けにいくつかのサウンドトラックを録音しましたが、結局この仕事は彼を満足させることをやめました。

新しいインスピレーションを求めて、作曲をあきらめ、西アフリカ、インド、中東、インドネシアで中世の歌と音楽の勉強を始めました。彼は次第に書き直し始め、個人的な音楽言語、つまり「私が共存できる音の世界」を作ろうと努めた。与えられた例は、これらの検索の結果を示しています。

マイクロソフトに戻って、ベネットは仕事に真剣に取り組みました。彼の作品は、Windowsだけでなく、Xboxオペレーティングシステムや、アプリケーション、Office、Cortana、Skypeなどの他の製品でも聞くことができます。ユーザーインターフェイスデザインデザイナーが会社のすべての製品で視覚的および機能的な一貫性を達成しようと努力するのと同じように、それはMicrosoftのサウンドプロファイルを統一するサウンドランドスケープを作成します。



さらに、マイクロソフトは、社外で使用されるサウンドに影響を与えるために、Fluent Sound and Sensory Design開発環境を作成しました。

「サウンドを使用して、ユーザーインターフェイスのリズムと感情の構造を形成します」とBennett氏は言います。 -サウンドは、タッチ、グラフィックテクスチャ、ダイナミクスなど、ユーザーの他の感覚と一致する要素になります。マイクロソフトでは、サウンドデザインに対する態度を変更しています。デバイスと感情の調和を実現したいのです。」

マイクロソフトのメディアアーキテクトであるリックセネシャルは、20年間ベネットに勤務しています。セネシャルは国際的な音楽サービスを管理しており、さまざまな企業や機関のイベント、ビデオ、ポッドキャスト、製品に毎年約4,000曲を選択しています。

セネシャルは、ベネットは急いでいないと非常にうるさいと言います。

「マシューは私がこれまでに働いた中で最も集中している人です」とセネシャルは言います。-彼は彼が取り組んでいるものを無限に改良し、改善することができます。彼は自分の分野について深く幅広い視野を持ち、非常によく考えています。彼は音を出すだけでなく、「そうですね、いいですね」と言っています。それが作り出す音と質感は論理と理性のためです。」

それが作り出す音と質感の背後には論理と理由があります。


ベネットはすぐにマイクロソフトの音はそうではないと言った。

「これらは効果音ではなく、ゲームの音でも、ビープ音のような不協和音でも、犬の鳴き声や轟音のトランペットでも、未来的な音でも、音楽ではすべてが満たされていなくても、音声やアラームではありません」と彼は言います。 「コンガ、リュート、バイオリンなど、これらの楽器の音の断片を録音することはありません。特定の文化的な場所や時間に、時には自分の過去に連れて行ったり、自分のしていることに気を取られたりする可能性があるからです。」

彼はブランドに名前を付けていませんが、主要なスマートフォンメーカーからの音はこの点で完全に無理であると言います。

「これらすべての着信音を聴いてください。それらの少なくとも1つが実生活で使用されたと聞いたことはありますか？」バイザシーのデジタルバージョン、アルペジエイトされたマイナーギターのコード、劇的なタッチで鳥のさえずりを合成。 「彼らは最初は面白かったし、約15年前は面白かったかもしれないが、最近は多くのことが望まれている。」

ベネットは、古いバージョンのWindowsのサウンドはWindows 10のサウンドとは大幅に異なっていたと述べています。最初に、もっとたくさんありました。 「ダウンロードが成功したことを示す意気揚々とした音はもはや必要ありません」と彼は言います。 -コンピュータがオンになったときに祝う必要があるのはなぜですか？今、私たちはすでにそれを当たり前のことと考えるべきであり、それが一般的に私たちがしていることです。

今日の音はずっと短いです。 Windows NT Workstation 4.0（1996）のシャットダウン信号などの以前のサウンドは、最大8秒続く可能性があり、今日の標準では無限であり、ほんの一瞬で測定される煩わしくないサウンドが必要です。システムの起動音とシャットダウンの音は過去のものであり、現在、技術的な騒音公害のもう1つの不要な原因と見なされています。


マシューベネット氏、Microsoft Production StudiosでサウンドエンジニアのDan Sharettと一緒に。

現代のサウンドは、前作ほど壮大ではありません。 Windows 2000の起動音はほぼ12秒続きました。戦闘機の飛行隊が離陸しているように見え、その後虹色のマリンバが登場しました。 「今日、私たちは最も有機的で、穏やかで、静かで目立たない音を使用しています」とベネットは言います。 「私たちは沈黙を念頭に置いて音を出します。」

専門家がスキューモーフィックと呼ぶ音は消えました。たとえば、文書を削除するときのしわくちゃの紙やタイプするときのタイプライターのキーなど、実際のオブジェクトの音をコピーします。

「そのような音は人々が新しいテクノロジーに慣れるのに役立ちましたが、それらはもはや必要ではなく、もはや役に立ちません。今、これらは意味のない装身具だ」とベネットは言う。

今日のシンプルなデザインでは、意味のない音は歓迎されません。

「約5年前、ビジュアルデザインの世界は、影、ワイドフレーム、照明効果、大きなフォントなど、不必要なものすべてを切り離し始めました」とBennett氏は付け加えます。「同じことがサウンドデザインでも起こります。」
沈黙を念頭に置いて音を出す
Windows 7では約40のサウンドがありました。 Windows 10では約8ですが、古いサウンドは「不明な技術上の理由で」残っています。ベネット氏は次のように述べています。「私が始めたとき、エラーメッセージには7種類の音があり、それらの意味が誰にもわかりませんでした。パートナーにも私たちにも明確なルールはありませんでした。これらの音をすべて取り除き、2つに置き換えました。1つは重大ではないエラー用、もう1つはより重要なエラー用です。」

ベネットはイントネーションの輪郭の比較に役立ちます-異なる言語で同じことを発音する人々の声の変化。質問準備はいいですか？英語、フランス語、日本語、中国語、スペイン語、ロシア語でほぼ同じ高さの変化で発音：「上、下、下、急激に上」。

ベネットはこのイントネーションを音楽的な手段で体現しており、Windows 10のカレンダーアラートが表示されたときに聞こえる2.5秒間の音が鳴りましたが、これは彼が誇りに思っている音の1つです。

イントネーションの輪郭は、新しいメッセージのメロディ（「Message for you！」-「Message to you！」）と一般的な通知のメロディ（「Everything's good」-「Everything is good」）の両方で聞こえます。

ベネットは、オペレーティングシステムのサウンドは、それらが関連付けられているアクションの機能を考慮する必要があると考えています。たとえば、メッセンジャーのメッセージは通常、電子メールよりも緊急の応答を必要とするため、Windows 10ではメッセンジャーでメッセージを受信する音は「ストレスが多く、ユーザーの行動を促す」と述べています。新しい手紙を受け取ったとき、「あなたはまだ何かが到着したことを知りたいが、音は穏やかです。」

彼は自分の作品を音楽と呼んでいますか？

「メロディー、リズム、ハーモニーがあるという意味で、彼らは音楽のようです」とベネットは言います。 「彼らは豊かで音楽的に聞こえますが、音楽そのものではありません。」 Windows 8のように、エレクトロニックミュージックを演奏し、それに感情を込めるミュージシャンをユーザーが想像することは望ましくありません。これは、Windows 7と比較して大きな前進でしたが、まだ成熟していません。そして今、サウンドスタイルはエレクトロニックのままですが、小さな男がコンピュータの中に座っているとは思わないでしょう。同時に、魂はコンピュータに感じられ、音の瞬間にのみ生命が生まれます。これは私たちの主なアイデアの1つです。

音楽の概念は確かにベネットの作品において重要な役割を果たしています。

「マイナーな非和音（m9）がエラーメッセージのメロディで使用されています。そのため、少し不調和であり、実際のところ、これらの単語に注意を払う必要があります」と彼は説明します。

ベネットなどの健全な作曲家を採用しているIT企業はごくわずかです。現在の問題について話し合い、議論する専門家協会や志を同じくする専門家の大規模なグループはありません。

「サウンドデザインは、業界、ハードウェア、グラフィックスよりもわずかに遅れています」とBennett氏は言います。 -サウンドエンジニアは、ゲーム以外の製品設計チームと密接な関係はありません。マイクロソフトは、サウンドエンジニアと直接的な製品体験を結び付けた最初の企業の1つでした。」

ベネットは、サウンドデザインはマイクロソフトや他のIT企業だけでなく、全世界に影響を与えるべきだと考えています。 2018年2月9日に、ユビキタス現代の電子デバイスによって作成された不協和音についての記事がニューヨークタイムズに掲載されました。「雷が鳴り、注意が必要なため、無機ノイズが標準になりつつあります。音信号をそらすと、パブロフの犬のように反射が生じます。」

ベネットはこれを聞きます。


Microsoft Production StudiosのMatthew Bennett氏は、

「Windowsだけでは1日に何億回も聞こえる」と述べています。 「これらは多くの音であり、たとえそれらが1秒未満持続しても、多くの人々の生活に影響を与えます。」それらは気分や幸福に影響を及ぼし、有益な場合も有害な場合もあります。」

世界保健機関は、予期しない大きな音が体内に危険なコルチゾールのサージを引き起こす可能性があることを認識しています。また、不要な音や過度の音量は、騒音公害の一形態です。

「熱帯雨林には、鳥の鳴き声や風の音など、情報を伝える多くの音があります。しかし、これらは簡単に知覚される有機的な音です。これは、サウンドデザインとの関係を示す強力な比喩です。」

私の会話の終わりに向かって、私はマシューに告白しました。1990年かそれ以降、自分のWindowsコンピュータでシステムサウンドをミュートしていたことです。私の意見では、それらは必要ではなく、迷惑なだけです。

私はこれから何かを失ったのではないかと彼に尋ねました-多分、オペレーティングシステムでの作業のいくつかの自明ではない側面が見過ごされていました。

「音は相互作用に深みと美しさを加えることができます。もう一度音をオンにできるようになりました。私たちは敬意を払います。私たちはあなたの子供を目覚めさせません。あなたが図書館や会議にいるとき、私たちは沈黙を破ることはしません。これはもはやランダムノイズではありません。これらは、あなたの周囲で有機的に聞こえるいくつかの注意深く選ばれた音です。

あとがきの代わりにソフトなリマインダー


Windows 10カレンダーの警告音を作成するマシューベネット


多くの人は、カレンダーが鳴る音に悩まされます。そのような音はそれぞれ、何かをする必要があることを意味するからです。誰かがこれらの音が火災警報のようなものであるとさえ信じています、それはあなたがどこかで、しかし一度ではなく一日中走らなければならないからです。私たちは、警告音をかなり爽快にしたいが、邪魔にならないようにしたかった。私たちは行動を促したが、それについて心配することなく、適切な量の楽観主義とエネルギーを見つける必要がありました。

このような音は小音量で聞こえると同時に、聞こえるよりも感じられるべきです。注意深く聞くと、これらが等間隔の7つのトーンであることを理解できます。メロディは非常に静かでスムーズに始まり、ユーザーインターフェイスに十分な長さ（2.5秒）続きます。ただし、通常の音量では、最初の3つのトーンはほとんど聞こえないため、実際には半分以上は聞こえません。彼らは音楽のヒントのように呼吸のように聞こえ、何かが今起こることを知らせます。その後、音量が上がり、音が聞こえやすくなります。最後にフェーディングリバーブがあり、音源から離れているときに聞くことができます。



長い音ですが、とても開放的です。それはかなり長いが、不快な感覚を引き起こしません。軽くて気持ちよくて美しいです。

ステレオチャンネルが適切に分離されたヘッドフォンまたはスピーカーを介して、左から右への動きを聞くことができます。それは非常に説得力があり、自然に聞こえます。

このワンピースのサウンドは、いくつかのレイヤーで構成されています。前景、中間、背景があります。フォアグラウンドには、エレクトロニックな撥ねた音色のパーカッションサウンドがあります。それらは実際の楽器に似ていませんが、完全にエイリアンのようには見えません。それらは記録されていませんが、デジタル技術を使用してモデル化されています。

この短いメロディー（他の多くのWindows 10と同様）では、トリオールが感じられます。最も調和的で心地よいと思われるすべてのメロディーは、私たちが長年慣れ親しんでいるトリオールのリズムに基づいています。

私たちは、音がコンピュータープログラムや人に似ていないことを確認するように努めています。彼らは洗練され、自然でなければなりません。私たちのオペレーティングシステムの息吹のように。これらの音がなければ、彼女はそのような調和の感覚を持っていなかったでしょう。
