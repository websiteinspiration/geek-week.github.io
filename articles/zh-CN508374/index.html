自己动手或如何自定义您的Snom手机。第2部分的图标和图像

正如我们在文章的第一部分中所承诺的那样，此延续致力于独立更改Snom手机中的图标。



所以，让我们开始吧。第一步，您需要获取tar.gz格式的固件。您可以在此处从我们的资源下载。所有信号图标均可用，并包含在每个固件版本中。



注意：请注意，固件的每个版本都包含特定于该版本和手机型号的设置文件。使用与固件或手机不匹配的设置文件会导致问题。

第二步，为电话准备图标。如您所知，Snom手机分别带有彩色和单色屏幕，并且图标会有所不同。



I.更改彩色显示电话的图标



具有彩色显示屏的手机上的图标和图像以PNG格式存储。这使它们几乎可以在所有现代图像编辑器中轻松编辑。但是，在编辑后，建议使用诸如optipng，pngquant或pngcrush之类的工具优化png文件，以删除所有冗余信息并优化文件大小。



图标图像大小：



  • 上下文相关键图标24x24px
  • SmartLabel 24x24px和18x18px
  • 标题栏中的图标（标题栏图标）18x18px
  • 菜单图标18x18px
  • 通话期间（呼叫屏幕图标）18x18px-48x48px
  • 档案格式：PNG


创建所需的图标后，将其下载到手机。有两种下载方法：



  1. 通过手动模式下的Web界面
  2. 使用自动促销


让我们从第一个选项开始 -通过Web界面下载。要下载，请转到“ 偏好设置/外观”标签上的手机网络界面，然后选择“ 自定义图片”：







接下来，我们找到我们要更改的图标并加载自己的版本：







如果您不喜欢自己的版本或“曲线”，则可以始终通过单击“重置”按钮来回滚”







注意。“软件更新”和“恢复出厂设置”不会删除下载的图像。


如您在手动模式下所看到的，一切都很简单，但是如果您需要更换多部电话，则此过程将花费很多时间。因此，我们继续第二个选择。



选项二 -通过自动配置下载。



首先，您需要从先前下载的customizing.tar.gz存档中以tar格式创建存档。创建归档文件时，删除不需要更改的所有文件和目录，但是请确保保留目录结构。

注意。您不需要存档所有原来在存档中的文件。这就足够了，建议您仅归档已修改的文件。您放入存档中的文件越多，电话对其进行配置所花费的时间就越长。


接下来，我们执行几个步骤：



1）创建一个XML文件（例如branding.xml）并将其复制到您的Web服务器（HTTP），即 http：//yourwebserver/branding.xml： 



<?xml version="1.0" encoding="utf-8" ?>
<settings>
 <uploads>
  <file url="http://yourwebserver/branding/branding.tar" type="gui" />
</uploads>
</settings>


2）我们进入手机的网络界面，位于高级->更新->设置URL部分，并提供指向文件yourwebserver / branding.xml的链接















。3）重新启动手机并享受搜索结果，

II.



单色设备上的图标未存储在常规图像文件（例如.png或.jpg）中，而是位图字体，其中包括用户界面中使用的所有图标。在以U + EB00开头的unicode表的私有使用区域中，定义了snom图标，可以使用诸如Font Forge之类的工具直接对其进行修改。



用Font Forge打开位图字体文件应显示已使用图标的列表。文件的实际内容取决于电话的版本和固件：







带单色显示屏的电话的图标规格。



对于型号D305，D315，D345，D385，D745，D785，D3，D7：



  • 上下文相关键图标17x17-基线x→0 / y→-2
  • 标题栏图标17x17-基线x→0 / y→-2
  • 标签面板图标17x17-基线x→0 / y→-2
  • 图标的最大尺寸32x32


对于D120，D710，D712，D715，D725型：



  • 上下文相关键图标7x7-基线x→0 / y→0
  • 标题栏图标7x7-基线x→0 / y→0
  • SmartLabel图标7x7-基线x→0 / y→0
  • 图标的最大尺寸32x32
