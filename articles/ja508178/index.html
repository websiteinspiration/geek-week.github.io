Tesla Model 3の内部：3年間でハードウェアがどのように変化したか

建設ジェットパックの休憩中に、テスラモデル3を少しいじっています。チャンネル@teslahackersに登録してください。ここでは、世界中のハッカーがテスラに侵入する方法と、モスクワのテスラに「横向きに積み込む」方法について説明します。その間、私はフランスのリバースエンジニアリングラボSystem Plus Consultingの研究の翻訳を提案します。







Tesla Model 3は3年前にリリースされました。それでも、ソフトウェアの更新とハードウェアの交換（HW 2.5からHW 3.0へ）のおかげで、TeslaはTesla 3を最新の状態に保ち、無人運転で将来に備えることを約束します。



少なくとも彼らはそれを約束している。



テクノロジーに精通したシリコンバレーのバイヤーの一部で、テスラの崇拝への愛情は培われてきました。彼らはこれらの車を愛し、電気で作業するのが好きで、エロンムスクのツイートを宗教として伝えています。



まず第一に、彼らは一から設計されたテスラ車のクリーンでエレガントな建築を賞賛します。テスラは新しい機能を追加し、ほとんど魔法のように車を無線（OTA）で更新することができます。他のOEM（それらの自動車は時代遅れのプラットフォームに関連付けられている）は、自動車用のソフトウェアをアップデートするこのような大規模な方法を開発していません。



テスラのファンは、物議を醸しているオートパイロット機能についてあまり心配する傾向がありません。彼らは彼女がしていることにも、彼女がしていることに集中していません。彼らは、テスラの約束に基づいて、いつの日かオートパイロットがどうなるかを重視することを好みます。一連のソフトウェアアップデートに加えて、テスラは昨年、ハードウェアコンポーネントをテスラHW 2.5からHW 3.0にアップグレードすることで料金を引き上げました。



HW 3.0の導入に伴い、Elon Musk氏はツイートしました。 「完全無人運転」がムスクの意味するところを見てみましょう。



興味をそそられたのは、HW 3.0への切り替えでした。今日モデル3の内部には何があり、このマシンはどのように変化しますか？



モデル3は、2017年半ばに最初にリリースされた、よりコンパクトで手頃な電気自動車です。 Teslaは昨年導入された独自のオンチップシステムのおかげで、無人の制御ソフトウェアのフルスイートを購入すると、サービスセンターにアクセスするだけでHW 2.5からHW 3.0にアップグレードできることをモデル3のお客様に約束しています。



物事を明確にするために：今日の完全無人運転用のソフトウェアパッケージでは、テスラ車が人間の介入なしで運転することはまだ許可されていません。これは、自動操縦システムに対する一連の増分更新です。さらに、先月のElon Maskのツイートによると、FSDパッケージの現在のコストは7,000ドルに相当し、7月1日に約1,000ドル増加する予定です。



今回の「アンダーザフード」とシステムプラスコンサルティング（フランス、ナンテス）では、Teslaモデル3についてさらに詳しく説明し、自動車センサーとモデル3にインストールされているTesla Autopilot ECUに焦点を当てます。







最初の可用性



車の計算能力は、ますます重要な機能になりつつあります。安全機能をアクティブにするだけでなく、最適なドライバー支援と自動運転を提供するには、かなりの計算能力が必要です。



自動運転を最適化するために、多くのOEMおよび大手自動車メーカーは、カメラ、レーダー、ライダー、超音波センサーなどのさまざまなセンサーを使用して、車が周囲を検出できるようにしています。センサーから受信したすべてのデータを一緒に収集する必要があり、ここでコントロールユニットが機能します。



テスラがサポートする高度なテクノロジーのイメージを考えると、モデル3内のすべてのハードウェアが技術的に市場で最も先進的で手頃な価格であると彼女が信じれば、一般市民は多くのことを許します。



内面を見ると、モデル3を設計する際のテスラの主な目標は、高度なドライバーアシスタンスシステムのコストを削減することでした。これにより、モデルが「手頃な価格」になりました。



モデル3は、8つのカメラ、1つのレーダー、およびセンサーとして12の超音波センサーを使用します。モデル3はLIDARを使用していません。これは、LIDARが「愚かなアイデア」であるというマスクのよく知られたステートメントに対応しています。



次に、System Plusはセンサーと計算ユニットのモデル3に関する情報を共有します。



ADAS用センサー







（出典：システムプラス）



テスラモデル3のセンサースイートには、半径250メートル以内で車の周囲360度の視界を提供する8台のカメラと、このビジョンシステムを補完する12個の超音波センサーが含まれています。これらを組み合わせることで、以前のシステムのほぼ2倍の精度で、離れた場所にある硬い物体と柔らかい物体を検出できます。このセンサーセットには、高度な画像処理機能を備えた直視レーダーシステムも含まれています。このシステムは、大雨、霧、ほこり、さらには前の車の外を通過する可能性がある過剰な波長で追加の環境データを提供します。



フロントパネルには、レーダーをサポートし、異なる特性を持つ4つのカメラがあります。メインカメラは250メートルをカバーしますが、視野角は小さく、他の距離は短い距離（150、80、60メートル）をカバーしますが、車の周囲の環境の視野角は広く、交通標識の読み取りに使用されます。残りの4台のカメラは車の側面と後部を見ており、最大100メートルの距離で見ることができます。



次に、ソナーは超音波を使用して、車の周囲8メートル以内の障害物を検出します。それは任意の速度で動作し、死角も制御します。ソナーによって収集されたデータは、追い越し中の自動車線変更システムを制御するためにオートパイロットによっても使用されます。最後に、GPSを使用して、道路に対する車の位置を特定します。

（出典：System Plus）



直視用に、TeslaはOn Semiconductorの3つのイメージセンサーを備えた3チャンバーモジュールを開発しました。モデル3では、直視用に2つのサイドカメラ、2つのサイドカメラと背面カメラも使用しています。



モデル3用に設計された8台のカメラは、オンセミコンダクターが2015年にリリースした同じ1.2メガピクセルのイメージセンサーに基づいています。「彼らは安価です。それらは新しいものではなく、高解像度でも違いはありません。



Fro氏は、1つのサプライヤーから8つすべてのイメージセンサーを購入することは、「テスラはより良い購入価格を約束する必要がある」ことを意味します。



トリプルカメラの比較







（情報源：System Plus）



Tesla Model 3トリプルカメラには、3.75μmピクセルサイズで解像度が1280×960 1.2 MPのON Semi AR0136A CMOSイメージセンサーが3つ搭載されています。このカメラは、自動操縦制御モジュールとドライバー支援によって使用される、最大250メートルの距離での前面画像キャプチャシステムの操作を提供します。



Teslaの3室モジュールにコンテキストを追加するために、System Plusはそれを、第1レベルの自動車の最大のサプライヤーの1つであるZFが開発した3室モジュールと比較しました。 ZF S-Cam4には、モノカメラとトリプルカメラの2つのソリューションが付属しています。



ZFモジュールのトリプルカメラバージョンであるS-Cam4には、Omnivision CMOSイメージセンサーとMobileye'a EyeQ4コンピュータービジョンプロセッサーが搭載されています。



上の画像に示すように、テスラが使用するPCB取り付け技術は、BMW技術とは異なります。BMWは、3つの異なるプリント回路基板上のセンサーの独立した組み合わせを好みます。次に、プリント基板上の3つのCMOSセンサーすべてが、チップシステムを使用せずにTesla 3ポジションカメラモジュールに組み込まれています。



ZFのS-Cam4には、Mobileyeのビジョン処理機能が含まれています。



オンセミコンダクターの成熟したイメージセンサーを選択し、後処理を追加しないことで、Teslaはカメラモジュールを「最新のイメージセンサーではなく、コストについて」作成しました。同社は、3チャンバーのZFモジュールのコストを165ドル、3チャンバーのTeslaモジュールのコストを65ドルと見積もっています。







（出典：System Plus）



レーダー







（出典：System Plus）



テスラは、コンチネンタルの実績あるレーダーモジュールを選択しました。 Continental ARS4-Bには、77 GHzレーダーチップセットとNXPセミコンダクターズが提供する32ビットマイクロコントローラーが含まれています。 System Plus Froは、MediaTekやTexas Instrumentsを含むいくつかのチップ企業が自動車レーダー市場への参入を発表しましたが、NXPとInfineonは紛れもなく1組のリーダーです。コンチネンタルはレーダーモジュールのサプライヤーの間で主要なプレーヤーです。彼らのARS4-Bは、「オーディオQ3、VWティグアン、日産ローグなどを含む、少なくとも15台の他の車両にも見られる」とフロ氏は説明した。



Continental ARS4-Aレーダーシステムは、衝突警報、緊急ブレーキ支援、衝突緩和、またはアダプティブクルーズコントロール（ACC）に使用されます。このシステムの重要な要素は、2つのオブジェクト間の相対的な速度と角度、長距離（最大0.2 mの精度で最大250 m）と短距離（最大70 m）の同時測定です。



このシステムは、NXP半導体マイクロコントローラーとBroadcomイーサネットトランシーバーを含む2つの電子ボードで構成されています。無線周波数（RF）ボードは、フラットアンテナを備えたPTFE / FR4ハイブリッド基板を使用した非対称構造で作られています。



高周波レーダートランシーバーとして、4つのレシーバー、2つのトランスミッター、および関連する電圧安定化発振器（VCO）で構成されるNXPセミコンダクター77 GHzチップセットが使用されます。

それでも、テスラだけではありません。オートパイロットシステム用に独自のプロセッサを開発することでテスラが行ったことに熱心に取り組む自動車メーカーはたくさんいます。
