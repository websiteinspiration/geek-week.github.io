現象の本質


実際、ドップラー効果-動いている物体または動いている物体から記録された波の周波数の変化の現象-は至る所で観察されます。これは、音と電磁波の両方に当てはまります（もちろん、特定の修正が必要です）。

ドップラー効果により、たとえば、近づいてくるサイレンの音が聞こえた後、離れるサイレンの音が聞こえたときなど、私たちは人生で常に遭遇します。最初は音が高く、それから-逆に音色が低くなります。理論的には、それは1842年にオーストリアの物理学者によって正当化されました、そして今日、この効果は科学技術の多くの分野で適用されます-宇宙研究から罰金を書くことまでドライバーに。



ドップラー効果は、次のように簡単に説明できます。あるオブジェクトの別のオブジェクトに対する動きにより、波のピークがソースからレシーバーに遅れて（または先に）到着し、これにより信号周波数が変化します。

したがって、検出された放射線の波長は式によって決定されます（すべての計算はWikipediaにありますが、もしあなたが怠惰な場合は、ここにも移動しました）。



そして、固定受信機が登録する周波数は、式によって決定されます



警察レーダー


警察のレーダーの場合、私たちは電波について話しているので、計算式はパラメーターC（物理的環境における信号伝搬の速度）を考慮していません。代わりに、特殊な相対性理論が使用され、式は次の形式になります



。この式では、Cは光速、Vはレシーバーに対する光源の速度です。シータの余弦は、ソースへの方向とレシーバーの参照フレームの速度ベクトルとの間の角度を反映しています。

警察のレーダーは、信号源であり、反射した無線周波数放射の周波数を記録する受信機でもあります。

最新のレーダーのほとんどはK帯域で動作します。つまり、約24.15 GHzの周波数で放射を発します。動く物体から反射される信号は、異なる周波数を持っています。そして、車の速度を決定するために、差は半分になり、相対半径速度は上記の式に従って計算されます。

ARROWなどの最新のレーダーのメーカーは、1 km / h以下の誤差を宣言しており、KRIS-Pの誤差は1 km / hです。古いレーダーモデルでは、最大2 km / hの誤差がある場合があります。精度の高いレーダーは使用されなくなりました

したがって、ほとんどの場合、車のレーダー速度がレーダー視野に入ると、2 km / hの精度で決定されます。この場合、レーダーは、垂直に向けられた速度の成分を考慮しません。つまり、曲がり角のある地形だけでなく、曲がり角を走行する場合、記録された車の速度は遅くなり、レーダーと車を結ぶ線に垂直な移動方向の成分が大きくなります。

動きが直線で発生し、仰角や曲がり角に変化がない場合は、それを考慮することができます。レーダーはあなたの速度を記録し、最悪の場合、「再保険」係数を考慮して+2または+5 km / hを記録します。

ただし、これらの時速kmの単位は、車の速度計によって決定できるとは考えないでください。今日、製造業者は交通安全を高めるためにスピードメーターを特に過大評価しています。そして、あなたが100 km / hの速度で走行していると思っていても、実際には実際の速度が90 km / hを超えないことがわかる場合があります。

そのため、速度を決定するために、GPS受信機を備えたDVR（Playme Sigmaなど）、レーダー探知機（Playme Silent 2やPlayme Hard 3など）、またはコンボデバイス（Playme P570など）など、GPSレシーバーが組み込まれたデバイスが使用されています。衛星データによると、実際の速度を決定し、可能な限り迅速に、ただし許可モードで移動できます。

法律と罰金


今日、ロシア連邦では、法律により20 km / hの非罰金のしきい値が定義されており（近い将来、10 km / hに減らすことができます）、この制限で速度を超えた場合に罰金を回避することができます。つまり、15〜18 km / h以上（そしてすぐに5〜8 km / h以上）の速度で旅行している場合、罰金を恐れることはありません。そうしないと、カメラのエラーが気に入らない場合があります。

ヨーロッパでは、罰金の形成のためのわずかに異なるスキームがあります。国によって速度違反のしきい値が異なります。これらのしきい値では、機器のエラーはすでに「保護」されています。たとえば、オーストリアでは、このパラメータは固定レーザーレーダーの場合は3 km / h、固定無線レーダーの場合は5 km / h、モバイルレーダーの場合は7 km / hです。ベルギー、フランス、イタリア-すべてのタイプのレーダーで5 km / h。フィンランドでは、7 km / h +レーダーエラーの違反が考慮に入れられ、セルビアでは、レーダーのエラーを考慮せずに、1 km / hを超えた場合でも罰金が科されます。これは、オーバークロックしすぎる必要がない場所です。
