「野原のささやきよりも静か」または忍術OSとは



Ninjutsu-OSは、Windows 10ベースの専用OS（ビルド1909）であり、800以上のサイバーセキュリティテストツールが含まれています。Windowsの世界からだけの一種のKali Linux。リリースは非常に新鮮で、2020年4月18日に行われました。

OS自体のインストールは、Win 10の標準インストールと何ら変わりませんが、例外として、非常に長いです。



OS自体を起動した後、すべてを再インストールして構成するのにさらに約10分かかります。すべての操作の後、わずかにオーバーロードされた（私の意見では）非常にカラフルなデスクトップが表示されます。



















中身を見てみましょう。

アセンブリには、合計800以上のユーティリティが含まれています。著者はWindowsで実行できるようにそれらのいくつかを適応させなければなりませんでした。

上部には別のバーがあり、すばやくアクセスできるように、カテゴリごとにすべてのユーティリティが含まれています。ユーティリティは、Kali Linuxでの分類を目的として、OSCP手法に重点を置いて分類されています。
-Mr. Robot tools kit-アセンブリの作者によると、最も一般的に使用されるユーティリティおよびその他の追加の材料。個人のトップ、いわば;
-Red Teaming-カテゴリ別のテストに最もよく使用されるユーティリティ。実際、これらはすべて1つのタブの下にある以下のすべてのカテゴリのトップユーティリティです。
-情報収集-情報の収集;
-悪用ツール-脆弱性の悪用;
-マルウェア分析-マルウェア分析;
-モバイルセキュリティツール-モバイルデバイスのテスト。
-ネットワーク攻撃-ネットワーク攻撃。
-パスワード攻撃-パスワードのクラッキング;
-プロキシおよびプライバシーツール-プロキシと匿名化。
-リバースエンジニアリング-リバースエンジニアリング。
-ストレステスト-ストレステスト（DDoS）;
-脆弱性分析-脆弱性の検索;
-Webアプリケーション攻撃-Webペンテスト;
-ワイヤレス攻撃-ワイヤレスネットワーク（Wi-FiおよびBluetooth）への攻撃、
-ファジングとペイロード-ファジングとワークロード。
-リモートコントロールツール-リモートコントロール。
-ユーティリティツール-その他の補助ユーティリティ（Winrar、MySQLなど）;

利用可能なユーティリティの完全なリストは、プロジェクトのWebサイトにあります。 OSは、純粋なPentestと、リバースエンジニアリング、OSINTまたはマルウェア分析に従事している人々の両方に最適です。 OSでの作業の数日間、私はソフトウェアの重大なバグや機能不全を捕らえませんでした。 Hack the Box plusを搭載したマシンでテストし、OSINTの追加プロジェクトを立ち上げました。

それとは別に、コンソールコマンド用に作成者が追加したエイリアスについて言及する必要があります。個人的に私から-下にハイライトを表示してくれた大きな人間の感謝3.



追加されたエイリアスの完全なリストは、プロジェクトのウェブサイトでも表示できます。一般的に、それらは作業を非常に容易にします。



また、筆者と直接話をしたことで、OSの将来についての疑問もいくつか明らかになりました。この段階で、実際には、ほとんどすべてがアセンブリに追加されました。さらに、フィードバックに応じて、アセンブリの作成者はユーティリティを追加または削除する予定です。


対話は対話者の同意を得て公開されてい

ますが、この記事では、OSの詳細な分析を行ったり、OSが正常に機能したりすることを目標にしていません。私は、いわば、Kali Linuxだけを使用しないことを喜びと共有することにしました。

一般に、WindowsエコシステムでのみKali Linuxの類似物を武器庫に表示することに興味がある場合は、無視しないでください。必ずこのOSをテストし、何を追加または削除するかについてレビューを作成者に書き込んでください。
