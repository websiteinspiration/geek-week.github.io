提案された一般的な考え方は次のとおりです：

入力フィールド用のクラスと検証マネージャー用のクラスがあります。入力フィールドは、予想どおり、UITextFieldから継承されます。検証マネージャー-UIControlから継承され、検証された要素のコレクションを含み（UITextFieldの子孫である必要はまったくありません）、「オブザーバー」パターンを実装します。さらに、彼は、検証された要素の状態が変化したときにアクセシビリティのステータスを変更する必要がある他のコントロールのマネージャーとして機能します。つまり、フィールドに無効なメールが含まれている場合、登録ボタンは使用できません。



入力フィールドは自己検証型です。入力フィールドに含まれるデータが有効であるかどうかの質問に答え、状態を変更できます。一部の開発者は、入力フィールドを検証するために個別のバリデータークラス（およびその子孫）を使用することを好みますが、この方法では、宣言的でスケーラブルなコードの移植性を完全に享受できなくなります。入力フィールドまたはその他の検証済みコントロールがフォームのシンプルなインターフェースをサポートすることが重要です。

これで、すべてのプログラミングは次の簡単なステップに進みます。

  1. ValidationManagerのVerifiedControlsコレクションを入力フィールドに関連付けます。


  2. ValidationManagerのmanagedControlsコレクションを、管理するボタンに関連付けます。


  3. 入力フィールドを反対方向にValidationManagerに関連付けて、デリゲートにします。

  4. 入力フィールドで、検証用の正規表現とエラーメッセージ、およびその他のカスタムプロパティと標準プロパティを標準のXcodeマネージャーで設定します。原則として、正規表現を除くすべてのものを「そのまま」残すことができます。キーボードを使用して、tyrnetから数式をコピーして貼り付けるだけです。エラーメッセージが表示されない場合は、画面に表示されません。


DSValidationManager コードは醜いプリミティブです。

import UIKit

@objc protocol IValidationManager : class {
    func verificated()
}

@objc protocol IValidatable : class {
    var isValid: Bool { get }
}

class DSValidationManager : UIControl, IValidationManager
{
    @IBOutlet var verifiedControls: [UIControl]?
    @IBOutlet var managedControls: [UIControl]?

    private (set) var valid : Bool = false {
        didSet {
            self.sendActions(for: .valueChanged)
        }
    }
    
    overrideinit(frame: CGRect) {
        super.init(frame: frame)
        self.verificated()
    }
    
    required init?(coder: NSCoder) {
        super.init(coder: coder)
        self.verificated()
    }

    func verificated() {
        self.checkVerifiedControls()
        self.controlsAction()
    }

    private func checkVerifiedControls() {
        guard let list:[IValidatable] = self.verifiedControls?.filter({$0 isIValidatable}) as? [IValidatable]
            else { return }
        self.valid = list.filter({!$0.isValid}).count==0
    }
    
    private func controls Action() {
        guard let list = self.managedControls else { return }
        for item in list {
            item.isEnabled = self.valid
        }
    }
}
