10.ペパーミント




ペパーミントはクラウドベースのLinuxディストリビューションであり、強力なハードウェアを実行する必要はありません。Lubuntuに基づいており、よりスムーズな操作のためにLXDEデスクトップ環境を使用しています。

ペパーミントOSには、あらゆるウェブサイトをデスクトップアプリケーションとして統合するためのアプリケーションであるICE が付属しています。

9.ルブントゥ




リストの次のディストリビューションはLubuntuです。名前が示すように、LubuntuはUbuntuに基づいていますが、デスクトップ環境としてLXDEを使用しています。 Lubuntuは最も軽量なUbuntu派生のLinuxディストリビューションであるため、速度と古いデバイスのサポートに特化しています。

デフォルトでは、このディストリビューションにはUbuntuよりもはるかに少ないアプリケーションが付属しており、それらは主にシステムに要求されないプログラムで構成されています。



画像表示用のGPicView、描画用のMTPaint、PDFファイル用のEvince、音楽用のAudacious、ビデオ用のGnome-Player、Webカメラ用のguvcview、WebブラウザーとしてのChromium、Eメール用のSylpheed、インスタントメッセージ用のPidgin、Transmission as BitTorrentクライアント、テーブル用のGnumeric、Microsoft Wordの代替としてのAbiword、ノート用のXpadなど。

以前にUbuntuを使用したことがある場合、Lubuntuは見慣れているように見えます。彼らのソフトウェアとリポジトリは同じなので、Lubuntu上のUbuntuで使用したすべてのソフトウェアをインストールできます。ただし、必要なアプリケーションをインストールするときは、システムについて忘れないでください。リソースを大量に消費するアプリケーションは使用しないでください。

Lubuntuの最小システム要件：
RAM：512 MB RAM（1 GB推奨）
CPU：Pentium 4またはPentium M、またはAMD K8以上ここ
からディストリビューションをダウンロードできます。

8. Linux Lite




名前が示すように、Linux Lite（「軽量」）は、実行するために高性能ハードウェアを必要としない軽量のLinuxディストリビューションです。 Linux Liteは、Ubuntu LTS（長期サポート）のリリースに基づいています。 LTSは、ディストリビューションを5年間サポートします。つまり、コンピューターにLinux Liteをインストールした後、アップデートは必ず5年間すべてリリースされます。

Linux Lite開発チームは次のように述べています。「Linux Liteは箱から出してすぐに使用できます。つまり、ディストリビューションを初めて実行するときに追加のソフトウェアをインストールする必要はありません。」必要なアプリケーションの検索とインストールに時間を費やす必要がないため、これは初心者にとって非常に朗報です。ほとんどのコアアプリケーションはすでにOSに組み込まれています。

プリインストールされているアプリケーションのほんの一部です。WebブラウザーとしてのFirefox、メール用のThunderbird、クラウドストレージとしてのDropbox、音楽用のVLC Media Player、Microsoft Officeの代替としてのLibreOffice、画像編集用のGimp、デスクトップのカスタマイズ用のLite微調整。

このディストリビューションはUbuntuに基づいているため、問題が発生した場合は、インターネット上で多くのチュートリアル、ヒント、テーマ別フォーラムなどを見つけることができます。

Linux Liteの最小システム要件：
RAM：512 MB RAM（1 GB推奨）
CPU：700 MHzプロセッサー
モニター：VGA画面、解像度1024×768（推奨VGA、DVIまたはHDMI画面、解像度1366×768）
ハードディスク容量：5 GB以上の空きディスク容量ここ
から配布パッケージをダウンロードできます。

6. CrunchBang ++




CrunchBang ++は、CBPPまたは＃！++またはCrunchBang Plus Plusとも呼ばれます。 CrunchBang ++は、そのシンプルさと軽量さで知られているCrunchBang Linuxと呼ばれる死んだディストリビューションのクローンです。 CrunchBang ++は古いハードウェアをサポートし、問題なく動作します。 Debian 9に基づいており、最小限のインターフェース設計を使用しています。また、Openboxウィンドウマネージャーに基づいています。

このプロジェクトは、CrunchBang Linuxと同じ目標を継続します。ユーザーに使いやすい軽量のLinuxディストリビューションに優れた機能を提供することです。そのため、Crunchbang ++には、ミニマルなデザイン、シンプルでエレガントなインターフェースが含まれています。

5. Bodhi Linux




古いコンピュータに新しい命を与えるもう1つの軽量LinuxディストリビューションはBodhi Linuxです。 Bodhi Linuxは、その軽さで主に知られています。 Bodhi Linuxにはソフトウェアがプリインストールされていないため、ISOファイルのサイズは小さくなっています。ただし、その場合は、aptを使用して必要なアプリケーションをインストールできます。古いコンピュータでは問題なく動作し、少量のメモリを消費します。

さらに、Enlightenmentウィンドウマネージャーは、配布をさらに高速化します。 Enlightenmentは、他のウィンドウマネージャーよりも高速です。 Enlightenmentまたは他のBodhi機能の使用に問題がある場合は、開発チームが作成したチュートリアルを読むことができます。

Bodhi Linuxには、グラフィック用のEphoto、WebブラウザーとしてのMidori、ePadテキストエディターなどの基本的なアプリケーションがありますが、マルチメディアアプリケーションはありません。でも心配しないで！言ったように、aptを使用して他のアプリケーションをインストールできます。UbuntuリポジトリはBodhi Linuxでも利用できます。

Bodhi Linuxの最小システム要件：

4. antiX Linux



antiXは、Debianをベースにした軽量のLinuxディストリビューションです。システムが「systemd」を使用していないことで有名です。「systemd」という用語に慣れていない場合は、antiXが「systemd」を使用しないことを気にしないでください。

通常のグラフィカルデスクトップ環境の代わりに、antiXはicewmウィンドウマネージャーを使用するため、システムは最も弱い機器でも問題なく動作します。antiXは、コンピューターを回復するための高速LiveCD / LiveUSBシステムとして使用できます。

antiX Linuxの最小システム要件：
RAM：256 MB RAM
CPU：PIIIプロセッサー
ハードディスク容量：2.7 GBの空きディスク容量ここ
から配布パッケージをダウンロードできます。

2.パピーリナックス




Puppy Linuxの最新リリースはPuppy Linux 6.0.2 Tahrpup CEで、サイズはわずか199〜201 MB（！）です。このディストリビューションは非常に小さなUSBドライブからダウンロードできます。

パピーリナックスはLive CD / DVD / USBから起動することができ、起動後はCD / DVD / USBを取り出すことができ、パピーリナックスは引き続きスムーズに動作します。パピーはRAMからロードして実行するため、非常に高速です。特に、Puppy Linuxを起動するのと同じUSBにデータを保存できます。

デフォルトでは、Puppy LinuxはJWMおよびOpenboxウィンドウマネージャーを使用します。これにより、リソース消費が大幅に削減されます。パピーリナックスは基本的なアプリケーションのセットのみを含みます。たとえば、表のGnumeric、Microsoft Wordの代替としてのAbiword、グラフィックの編集やマルチメディアの再生のためのさまざまなプログラム。

Puppy Linuxの最小システム要件：
RAM：64 MB（256 MB推奨）
CPU：333 MHz ここ
からディストリビューションをダウンロードできます。

1.タイニーコア




Tiny Coreの興奮を体験する準備をしてください。 Windows上の最小の画像編集アプリケーションでさえ、Tiny Core Linuxよりも難しいでしょう。はい、その重量はわずか16 MB（!!）です。小さく低品質のビデオをダウンロードすると、Tiny Core Linuxをダウンロードするよりもさらに時間がかかります。

システムには、最も必要なプログラムと最小限のインターフェースのみが付属しています。それでも非常に古いコンピュータを使用している場合は、Tiny Coreをインストールして、この小さな奇跡に慣れてください。他の既知のOSよりも速くロードされます。



Tiny Core LinuxはDamn Smallディストリビューションに基づいていますが、現時点では完全に独立しています。デフォルトでは、Tiny CoreにはFLTK / FKWMおよびBusyBox GUIが付属しています。多くのことを見逃しますが、心配しないでください。必要に応じて、すべて手動でインストールできます。

Tiny Coreには、Core、Tiny Core、CorePlusの3つのバージョンがあります。最初のバージョンにはCLI（コマンドラインインターフェース）のみが含まれており、経験豊富/上級ユーザーのみが操作できます。 Coreの現在のバージョンの重量はわずか9 MBです。

Tiny Coreは、GUI（グラフィカルユーザーインターフェイス）に精通している平均的なユーザー向けです。 TinyCoreの現在のバージョンのサイズは16 MBです。

Core Plusはインストールイメージであり、ディストリビューションではありません。ワイヤレスネットワークにのみアクセスできる新規ユーザー、または英語以外のキーボードレイアウトを使用するユーザーに推奨されます。ディストリビューションには、システムをカスタマイズするためのユーティリティを備えた基本的なコアシステムが含まれます。7つのウィンドウマネージャーの選択、多くのファームウェアファイルとndlswrapperによるワイヤレスサポート、英語以外のレイアウトのサポート、リマスタリングツール。 CorePlusの現在のバージョンのサイズはわずか72 MBです。

Tiny Core Linuxの最小システム要件：
RAM：46 MB（64 MB推奨）
CPU：Pentium 2 ここ
からディストリビューションをダウンロードできます。
