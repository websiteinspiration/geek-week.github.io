Lanzamiento de InterSystems IRIS 2020.1


A finales de marzo, se lanzó una nueva versión de la plataforma de datos InterSystems IRIS 2020.1. Incluso la pandemia de coronavirus no impidió la liberación.


Una de las cosas importantes de la nueva versión es el aumento del rendimiento del kernel, la generación de una aplicación REST de acuerdo con la especificación OpenAPI 2.0, particionamiento para objetos, una nueva vista del Portal de administración, soporte MQTT, un caché de consulta universal, un nuevo marco para crear elementos de producto en Java o .NET.
