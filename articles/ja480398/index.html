そして今、非常に役立つ9つのJavaScriptのヒント。

  1. すべてを交換


    string.replace（）関数は最初の出現のみを置換することがわかっています。正規表現の最後に/ gを追加すると、すべての出現箇所を置き換えることができます。

    var example = "potato potato";
console.log(example.replace(/pot/, "tom")); 
// "tomato potato"
console.log(example.replace(/pot/g, "tom")); 
// "tomato tomato"

  2. 一意の値の取得


    SetオブジェクトとSpreadオペレーターを使用して、一意の値のみを持つ新しい配列を作成できます。

    var entries = [1, 2, 2, 3, 4, 5, 6, 6, 7, 7, 8, 4, 2, 1]
var unique_entries = [...new Set(entries)];
console.log(unique_entries);
// [1, 2, 3, 4, 5, 6, 7, 8]

  3. 数値を文字列に変換


    空の引用符で連結演算子を使用するだけです。

    var converted_number = 5 + "";
console.log(converted_number);
// 5
console.log(typeof converted_number); 
// string

  4. 文字列を数値に変換する


    +演算子だけが必要です。

    文字列番号でのみ機能するため、これには注意してください。

    the_string = "123";
console.log(+the_string);
// 123

the_string = "hello";
console.log(+the_string);
// NaN

  5. 配列からアイテムをシャッフルする


    私は毎日shufflinです ':-)

    var my_list = [1, 2, 3, 4, 5, 6, 7, 8, 9];
console.log(my_list.sort(function() {
    return Math.random() - 0.5
})); 
// [4, 8, 2, 9, 1, 3, 6, 5, 7]

  6. 多次元配列を平滑化する


    Spreadステートメントを使用するだけです。

    var entries = [1, [2, 5], [6, 7], 9];
var flat_entries = [].concat(...entries); 
// [1, 2, 5, 6, 7, 9]

  7. ショートカット条件


    この例を見てみましょう：

    if (available) {
    addToCart();
}

    そして、関数と一緒に変数を使用するだけでそれを短くすることができます：

    available && addToCart()

  8. 動的プロパティ名


    動的プロパティを割り当てる前に、まずオブジェクトを宣言する必要があるといつも思っていました。

    const dynamic = 'flavour';
var item = {
    name: 'Coke',
    [dynamic]: 'Cherry'
}
console.log(item); 
// { name: "Coke", flavour: "Cherry" }

  9. 長さを使用してサイズを変更/配列をクリア


    基本的に、配列の長さを書き換えます。

    配列のサイズを変更したい場合：

    var entries = [1, 2, 3, 4, 5, 6, 7];  
console.log(entries.length); 
// 7  
entries.length = 4;  
console.log(entries.length); 
// 4  
console.log(entries); 
// [1, 2, 3, 4]

    配列をクリアしたい場合：

    var entries = [1, 2, 3, 4, 5, 6, 7]; 
console.log(entries.length); 
// 7  
entries.length = 0;   
console.log(entries.length); 
// 0 
console.log(entries); 
// []
