Svelteでのブレイクアウトの開発

MDNには、「2D Pure JavaScript Game」というタイトルのチュートリアルがあり、HTML5の<canvas>要素の使用の基本を探ります。



このチュートリアルでは、Svelteでこのゲームの開発を繰り返します。


1. キャンバスを作成して描画する

純粋なJavaScript - Svelte


この例では、正方形、長方形、円の3つの幾何学的形状を表示します。



canvas DOM, onMount. canvas canvas this.


<script>
    import { onMount } from 'svelte';

    let canvas;

    onMount(() => {
        const ctx = canvas.getContext('2d');

        ctx.beginPath();
        ctx.rect(20, 40, 50, 50);
        ctx.fillStyle = "#FF0000";
        ctx.fill();
        ctx.closePath();

        ctx.beginPath();
        ctx.arc(240, 160, 20, 0, Math.PI*2, false);
        ctx.fillStyle = "green";
        ctx.fill();
        ctx.closePath();

        ctx.beginPath();
        ctx.rect(160, 10, 100, 40);
        ctx.strokeStyle = "rgba(0, 0, 255, 0.5)";
        ctx.stroke();
        ctx.closePath();
    });
</script>

<canvas
    bind:this={canvas}
    width={480}
    height={320}
></canvas>

<style>
    canvas { background: #eee; }
</style>
