PIndastrial shield - catu daya dan modul antarmuka RS-485 untuk Raspberry PI


Ada banyak proyek di mana Raspberry PI digunakan sebagai dasar sistem, tetapi ada sejumlah kesulitan dalam menyediakan daya dan berinteraksi dengan perangkat eksternal. Saya ingin berbagi pengalaman membuat perisai kecil untuk komputer mini, menyederhanakan aplikasi dan meningkatkan keandalan seluruh sistem.


Tugas khas diselesaikan dengan bantuan Raspberry PI adalah otomatisasi rumah / industri kecil, di mana ada satu set modul perangkat didukung oleh 12-24 volt dan memiliki bus RS-485. Untuk menyalakan Raspberry dari daya yang sama dengan modul, Anda harus meletakkan konverter 5 volt tertentu dan menghubungkannya melalui GPIO atau konektor USB. Untuk ini ditambahkan adaptor USB-RS485 atau UART-RS485 lain yang menonjol dari USB atau entah bagaimana menggantung pada konektor GPIO. Pilihan desain seperti itu tidak dapat diandalkan.
