CSS3を使用したチェックボックスとラジオボタンのスタイル

HTMLフォームのCSSスタイルを作成する場合、開発者はしばしばチェックボックスとラジオボタンの外観を直接変更できないことに直面します。CSS3ツールを使用して、JavaScriptコードを使用せずにこの制限を回避する方法を見てみましょう。

そのため、まず、いくつかの通常のチェックボックスとスイッチをフォームに追加します。 

/*   */

<div>
<input type="checkbox" id="cb1"> <label for="cb1"> 1</label>
</div>

<div>
<input type="checkbox" id="cb2"> <label for="cb2"> 2</label>
</div>

<div>
<input type="checkbox" id="cb3"> <label for="cb3"> 3</label>
</div>

/*   */

<div>
<input type="radio" name="rb" id="rb1" checked> <label for="rb1"> 1</label>
</div>

<div>
<input type="radio" name="rb" id="rb2"> <label for="rb2"> 2</label>
</div>

<div>
<input type="radio" name="rb" id="rb3"> <label for="rb3"> 3</label>
</div>



要素の標準表示をスコープを超えて移動し、隣接するラベルにインデントを追加します。

input[type="checkbox"]:checked, 
input[type="checkbox"]:not(:checked), 
input[type="radio"]:checked, 
input[type="radio"]:not(:checked) 
{
    position: absolute;
    left: -9999px;
}

input[type="checkbox"]:checked + label, 
input[type="checkbox"]:not(:checked) + label, 
input[type="radio"]:checked + label, 
input[type="radio"]:not(:checked) + label {
    display: inline-block;
    position: relative;
    padding-left: 28px;
    line-height: 20px;
    cursor: pointer;
}



ラベルの前に、カスタム要素の様式化されたコンテナーを追加します。フラグの場合、これらは美しさのためにわずかに丸みを帯びた四角形になり、スイッチの場合は小さな円になります。

input[type="checkbox"]:checked + label:before, 
input[type="checkbox"]:not(:checked) + label:before,
input[type="radio"]:checked + label:before, 
input[type="radio"]:not(:checked) + label:before {
    content: "";
    position: absolute;
    left: 0px;
    top: 0px;
    width: 18px;
    height: 18px;
    border: 1px solid #dddddd;
    background-color: #ffffff;
}

input[type="checkbox"]:checked + label:before, 
input[type="checkbox"]:not(:checked) + label:before {
    border-radius: 2px;
}

input[type="radio"]:checked + label:before, 
input[type="radio"]:not(:checked) + label:before {
    border-radius: 100%;
}



次に、選択インジケーターを追加します。フラグの場合、これらはdawsになります。スイッチの場合、色で塗りつぶされた小さな円はコンテナ自体よりも小さくなります。より大きな効果のために、小さなアニメーションも設定します：

input[type="checkbox"]:checked + label:after, 
input[type="checkbox"]:not(:checked) + label:after, 
input[type="radio"]:checked + label:after, 
input[type="radio"]:not(:checked) + label:after {
    content: "";
    position: absolute;
    -webkit-transition: all 0.2s ease;
    -moz-transition: all 0.2s ease;
    -o-transition: all 0.2s ease;
    transition: all 0.2s ease;
}

input[type="checkbox"]:checked + label:after, 
input[type="checkbox"]:not(:checked) + label:after {
    left: 3px;
    top: 4px;
    width: 10px;
    height: 5px;
    border-radius: 1px;
    border-left: 4px solid #e145a3;
    border-bottom: 4px solid #e145a3;
    -webkit-transform: rotate(-45deg);
    -moz-transform: rotate(-45deg);
    -o-transform: rotate(-45deg);
    -ms-transform: rotate(-45deg);
    transform: rotate(-45deg);
}

input[type="radio"]:checked + label:after, 
input[type="radio"]:not(:checked) + label:after {
    left: 5px;
    top: 5px;
    width: 10px;
    height: 10px;
    border-radius: 100%;
    background-color: #e145a3;
}



カラスを描くために、2つの側面が反時計回りに45度色で塗られた小さな長方形を回転させます。

注意：beforeおよびafterセレクターを使用すると、ラベル自体のコンテンツの直前と直後にコンテンツを追加できます。ラベルには相対位置（position：relative）を設定するので、コンテンツのコンテキスト絶対位置を設定できます。

アイテムが選択されていない場合は選択インジケーターが非表示のままになり、アイテムが選択された状態の場合はインジケーターが表示されます。

input[type="checkbox"]:not(:checked) + label:after, 
input[type="radio"]:not(:checked) + label:after {
    opacity: 0;
}

input[type="checkbox"]:checked + label:after, 
input[type="radio"]:checked + label:after {
    opacity: 1;
}



ここで説明するソリューションは、Chrome、Firefox、Safari、Operaの最新バージョンのすべてのブラウザで機能し、バージョン9以降のInternet Explorerでも機能することを付け加えます。
