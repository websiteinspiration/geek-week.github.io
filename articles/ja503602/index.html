JavaScriptを使用したAmazonのマルチタスクソリューション



良い一日。Trung Anh Dangによる記事「Amazon Coding Interview Questions」の翻訳を紹介します。

この記事では、著者はAmazon（彼の主張どおり）からのいくつか（正確には3つ）のタスクと彼のソリューションを示しています。

タスクの条件を理解したら、自分で解決してみてください。次に、あなたの決定を著作権の決定と比較してください。私の決定は著者の決定に近いことがわかりましたが、少し悪いです。あなたの解決策がより良いと思われる場合は、コメントで共有してください。

じゃ、行こう。

タスク番号1


ランレングスエンコーディングまたはランレングスエンコーディングは、文字列をすばやく簡単にエンコードする方法です。このアルゴリズムの本質は、繰り返される文字（シリーズ）を1つの文字とその繰り返し数で置き換えることです。シリーズは、いくつかの同一の文字のシーケンスです。エンコードする場合、シリーズを構成する同一の文字の文字列は、繰り返し文字自体とその繰り返し数を含む文字列に置き換えられます。

たとえば"AAAABBBCCDAA"、繰り返しをエンコードした後の行は次のようになります"4A3B2C1D2A"。

繰り返しのエンコードとデコードを実装します。エンコード用の文字列は文字のみで構成され、数字は含まれません。デコード用の文字列は有効です。

決定


文字列エンコーディング


解決策は非常に簡単です。

const encoding_string = function(string) {
    if(string.length === 0) return ''
    
    let currChar = string.charAt(0)
    let count = 1
    let encoding = ''
    
    for(let i = 1; i < string.length; i++) {
        const char = string.charAt(i)
        if(char === currChar) count++
        else {
            encoding += count + currChar
            
            // 
            count = 1
            currChar = char
        }
    }
    
    encoding += count + currChar
    
    return encoding
}

文字列のデコード


2つの補助関数を実装する必要があり

ます。1.文字が数値かどうかをチェックする関数：

const isNumber = function(str) {
    return /^\d+$/.test(str)
}

2.行末までの文字数を返す関数：

const addCountAmount = function(string, char, count) {
    for(let i = 1; i <= count; i++) {
        string += char
    }
    
    return string
}

これが解決策です：

const decoding_string = function(string) {
    if(string.length === 0) return ''
    let currCount = 0
    let i = 0
    let decoding = ''
    
    while(i < string.length) {
        const char = string.charAt(i)
        if(isNumber(char)) {
            const num = +char
            currCount = currCount * 10 + num
        } else {
            decoding = addCountAmount(decoding, char, currCount)
            currCount = 0
        }
        
        i++
    }
    
    return decoding
}

解決策を確認し

ます。1.文字列"AAAABBBCCDAA"を次のように変換する必要があります"4A3B2C1D2A"。

console.log(encoding_string("AAAABBBCCDAA")) // 4A3B2C1D2A

2.逆変換：

console.log(decoding_string("4A3B2C1D2A")) // AAAABBBCCDAA

タスク番号2


N段の階段があります。1つのステップで、1つまたは2つのステップを登ることができます。与えられたN。階段を上るユニークな方法の数を返す関数を記述します。ステップを克服する順序は重要ではありません。

たとえば、N = 4の場合、5つのユニークな方法があります。

    1、1、1、1
    2、1、1
    1、2、1
    1、1、2
    2、2

ステップ数の制限を削除するとどうなりますか？たとえば、X = {1、3、5}の場合、一度に1、3、または5ステップを登ることができます。

決定


階段を上るユニークな方法の数を返す関数（制限付き）：

const climb_stairs_1 = function(stairs) {
    if(stairs === 1) return 1
    if(stairs === 2) return 2

    let prev = 1
    let curr = 2
    for(let i = 2; i < stairs; i++) {
        const sum = prev + curr
        prev = curr
        curr = sum
    }
    return curr
}

階段を上るユニークな方法の数を返す関数（制限なし）：

const climb_stairs_2 = function(stairs, nums) {
    const dp = Array(stairs + 1).fill(0)

    for(let i = 1; i <= stairs; i++) {
        //    f(i - x),  x -    nums
        let total = 0
        for(let j = 0; j < nums.length; j++) {
            const num = nums[j]

            //    
            if(i - num > 0) total += dp[i - num]
        }
        dp[i] += total

        //  i   nums,  
        if(nums.indexOf(i) !== -1) dp[i] += 1
    }
    //     dp
    return dp[dp.length - 1]
}

解決策を確認します。

console.log(climb_stairs_1(4)) // 5

console.log(climb_stairs_2(4, [1, 3, 5])) // 3

タスク番号3


整数Kと文字列Sが与えられた場合、K以下の異なる文字を含む最長の部分文字列の長さを見つける必要があります。

たとえば、与えられたs = "abcba"とk = 2。最大でK個の異なる文字を含む最長の部分文字列は"bcb"です。

決定


これが私の解決策です：

const k_longest_substring = function(string, k) {
    let l = 0
    let r = 0
    const charCount = new Map()
    let longestSubstring = ''

    while(r < string.length) {
        const char = string.charAt(r)

        //  
        if(charCount.has(char)) {
            charCount.set(char, charCount.get(char) + 1)
        } else {
            charCount.set(char, 1)
        }

        //  charCount  k
        //      ,        k  
        if(charCount.size > k) {
            while(charCount.size > k && l < r) {
                const leftChar = string.charAt(l)
                const count = charCount.get(leftChar)

                if(count === 1) charCount.delete(leftChar)
                else charCount.set(leftChar, count - 1)

                l++
            }
        }

        const substring = string.substring(l, r + 1)
        longestSubstring = substring.length > longestSubstring.length ? substring : longestSubstring
        r++
    }
    return longestSubstring
}

解決策を確認します。

console.log(k_longest_substring("abcba", 2)) // bcb
