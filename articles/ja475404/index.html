簡単な例のBLoCパターン

そして再び、古典的なFlutterカウンターの例を使用したBLoCについて。


リアクティブプログラミングに関するいくつかの記事を読み、アプリケーションでBLoCパターンを使用していると、何かに追いついていないことに気付きました。いつものように、すべてに十分な時間はありませんが、今は空き時間があり、強さがあります-決定、BLoCパターンを使用した最も単純なFlutterアプリケーションを作成します。


猫animashkaアプリケーションと説明の下で、なぜ私はそれをそのように書いたのか。非常に興味深いのはコミュニティの意見です。



はい、既にこのパターンについて何度も書いていますが、いずれにしても、明確な指示とアプリケーションルールはなく、アプリケーションにロジックを正しく実装する方法について疑問が生じることがよくあります。


この記事の目的は、自分自身と読者に少し明快さをもたらすことです。


したがって、パターンの定義は、Googleのエンジニアが表明したとおりです。BLOCは、次のような単純なクラスです。


  1. すべての出力ストリーム
  2. すべての入力はストリームです
  3. このクラスは、ビジュアルインターフェイスからロジックを削除する必要があります

このパターンを実装するために、必要に応じてrxdartライブラリを使用できますが、使用しない場合もあります。


これは、あなたが思うように反応性を取り除きません。パッケージの作成者自身が書いているように、rxdartは、ストリームを操作するためのDart言語の組み込みの豊富な機能に機能を追加します。

class MyHomePage extends StatelessWidget {
  CounterBloc counterBloc = CounterBloc();

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: StreamBuilder<int>(
            stream: counterBloc.pressedCount,
            builder: (context, snapshot) {
              return Text(
                'Flutter Counter Bloc Example - ${snapshot.data.toString()}',
              );
            }),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: <Widget>[
            Text(
              'You have pushed the button this many times:',
            ),
            StreamBuilder<int>(
                stream: counterBloc.pressedCount,
                builder: (context, snapshot) {
                  return Text(
                    '${snapshot.data.toString()}',
                    style: Theme.of(context).textTheme.display1,
                  );
                }),
          ],
        ),
      ),
      floatingActionButton: Container(
        width: 100.0,
        height: 100.0,
        child: FloatingActionButton(
          onPressed: () {
            counterBloc.incrementCounter.add(null);
          },
          tooltip: 'Increment',
          child: Text(
            "+ \n send \n to BLoC",
            textAlign: TextAlign.center,
          ),
        ),
      ),
    );
  }
}

class CounterBloc {
  int _counter;

  CounterBloc() {
    _counter = 1;
    _actionController.stream.listen(_increaseStream);
  }

  final _counterStream = BehaviorSubject<int>.seeded(1);

  Stream get pressedCount => _counterStream.stream;
  Sink get _addValue => _counterStream.sink;

  StreamController _actionController = StreamController();
  StreamSink get incrementCounter => _actionController.sink;

  void _increaseStream(data) {
    _counter += 1;
    _addValue.add(_counter);
  }

  void dispose() {
    _counterStream.close();
    _actionController.close();
  }
}
