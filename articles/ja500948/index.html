Rust 1.43.1リリース：調整リリース

RustチームはRustのパッチリリース1.43.1を公開しました。Rustは、誰でも信頼性の高い効率的なソフトウェアを作成できるプログラミング言語です。


toolsを使用して以前のバージョンのRustをインストールした場合rustup、バージョン1.43.1にアップグレードするには、次のコマンドを実行するだけです。


rustup update stable

まだインストールしていない場合は、Googleのウェブサイトの対応するページからインストールrustupできます。また、GitHubの詳細なリリースノートもご覧ください。


バージョン1.43.1に含まれるもの


Rust 1.43.1は、1.43.0で発生した2つのリグレッション専用です。また、このリリースでは、Cargoで使用されるOpenSSLが更新されています。


検出できないCPU拡張機能を修正


Rust 1.27.0では、標準ライブラリがマクロを使用してx86 CPU拡張を定義するためのサポートを導入しましたis_x86_feature_detected!。内部リファクタリング中に、以前は可能であった不安定な拡張機能（AVX-512など）の定義が修正されました。Rust 1.43.1ではこれが修正されています。この回帰の詳細については、問題＃71473を参照してください。
