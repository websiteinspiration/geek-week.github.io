スライディングウィンドウ（ムービングウィンドウ）


タイトルには、直訳を引用しました。誰かが私を訂正し、別の用語がより適切である場合-ありがとう。

スライディングウィンドウの意味は、新しい値を使用するたびに、関数が一定期間再計算されることです。これらは非常に多くの機能を果たします。たとえば、rolling.mean（）、rolling.std（）は、在庫変動の分析で最もよく使用されます。rolling.mean（）は、短期的な変動を平滑化し、全体的な傾向を視覚化できる一般的な移動平均です。

#     
adj_close_px = sber['Adj Close']

#   
moving_avg = adj_close_px.rolling(window=40).mean()

#  
print(moving_avg[-10:])



この関数の結果として何が得られるかを理解できるグラフ：

#    
sber['40'] = adj_close_px.rolling(window=40).mean()

#    
sber['252'] = adj_close_px.rolling(window=252).mean()

#   
sber[['Adj Close', '40', '252']].plot(figsize=(20,20))

plt.show()



ご覧のとおり、rolling.mean（）はこのタスクに対応しています。この関数は、短期的な変動を平滑化し、判断を下すことができる長期的な傾向を確認できるようにします。価格は、考慮される移動平均よりも高くなります-株式を取得します-株式を売却します-単純で、この方法に従うことをお勧めしません。原則として、移動平均に加えて、決定の正しさを確認できる他の指標も使用されます。取引スタイルに応じて、誰もが自分で決断しなければなりません。

ボラティリティ


株式のボラティリティとは、一定期間における株式のリターンの変動の変化です。彼らは通常、1つの株のボラティリティを別の株と比較して、リスクが最も高い可能性のあるアイデアを得るか、市場指数と比較して、市場に対する株のボラティリティを理解します。原則として、ボラティリティが高いほど、この株への投資はリスクが高くなります。これは一定ではなく、時間とともに変化することに注意してください。これは、pandasパッケージに含まれているrolling.std（）関数でも確認できます。ボラティリティの変化を計算する例：

#   
min_periods = 60 

#  
vol = daily_pct_change.rolling(min_periods).std() * np.sqrt(min_periods) 

#  
vol.plot(figsize=(10, 10))

plt.show()
