EDAにおけるアメリカの独占を破壊する。イノポリスが最初の一歩を踏み出す



1990年代以来、デジタルマイクロエレクトロニクスの全世界の設計が、シノプシスとケイデンスの2つの互いに離れたカリフォルニアのオフィスによって管理されていることに驚かされました。当時、世界の設計の4分の1は日本で行われており（当時、中国本土は原始的な状態でした）、ソニー、日立、富士通などの巨人はすべて米国に頭を下げ、日本のデザイナーが使用したプログラムに数百万ドルを支払いました。現在は、サムスン、ファーウェイ、さらには宇宙用のマイクロチップを設計するロシアのオフィスまで続いています。

ロシアの土地はGoogleと比較してYandexを成長させることができたので、マイクロチップを設計するためのいくつかのプログラムを作成してみませんか？小規模から始めることができます。設計自動化アルゴリズム（Electronic Design Automation-EDA）の開発のためのコンテストやハッカソンを普及させるために。これらのアルゴリズムは、多くのレベルの複雑さを持っているという点で便利です。学生は週末に最も簡単な配置配線プログラムを書くことができますが、高度なプログラムでは、数百人が何十年も、何十億ドルもの研究開発が必要になります。

現在カザンの近くのイノポリスで、「2週間のトレーニング+ハッカソン」形式の学生向けイベント。トピックの1つは、従来のEDAタスクでした-電子回路のグラフをスタンダードセルの行に配置してトレースしました。C ++ / Java / Python、テキストを解析する方法、グラフアルゴリズム、およびGUIを使用してデータ構造を視覚化するスキルを基本的に理解しているプログラマの小さなチームが短時間で実装できるものを見るのは興味深いでしょう。

タスクは、さまざまな学生が処理できる3つのサブタスクで構成されています。

  1. デジタル回路グラフのテキスト表現を解析します。
  2. マイクロ回路の標準セルの行にグラフを配置し、これらのセルをトラックに接続します（トレース）。
  3. 結果の視覚化-回路トラックと接続の画面に表示します。

プログラムへの入力は、Verilogハードウェア記述言語の非常に限られたサブセットのテキストファイルです。このファイルは、回路の入力と出力（入力、出力）、内部接続（ワイヤー）を記述し、「type instance-name（接続のリスト）」の形式で論理要素のリストを提供します。ファイルは、Lex + Yaccまたは同様のプログラムを使用してC / C ++で解析できます。



テキストを解析した結果は、メモリ内の要素のグラフの表現になるはずです。このビューは、配置およびトレースアルゴリズムによって使用され、別の構造によって作成されます。ハッカソンチームに十分な参加者がいる場合、最初の解析の結果は、ハッカソン中に追加のタスクとして視覚化できます。美しくなくても、ほぼこの精神で：

標準セルとは何ですか？これは、このASICライブラリの標準の高さのセルです。つまり、ある技術を使用して工場でマイクロ回路を製造するためのプリミティブのライブラリです。 ASIC-特定用途向け集積回路。現在、スマートフォンなどのほとんどのチップはASICです。 ASICライブラリのセルには標準の高さがあり、列に配置して電力を供給し、トレースアルゴリズムを容易にすることができます。ライブラリ内の異なるセルは、ANDやORなどのプリミティブだけでなく、マルチプレクサ、ラッチ、2つまたは3つの論理要素の組み合わせ（AND-OR）などのより複雑な構成も実装できます。
