レインボープルーフはグラフの標準コンポーネントの存在を示します

数学者は、小さなグラフのコピーが常に理想的には大きなグラフをカバーできることを証明しています。



1月8日、3人の数学者が、リンゲル仮説として知られる、ほぼ60年前に定式化された組み合わせ論からの定理の証明を発表しました。大まかに言えば、彼女は、グラフ（点と線で構成される構成）は、より小さいサイズの同一のピースから理想的に折りたたむことができると予測しています。

数学者はこの仮説の確認を熱心に受け入れました。

「幸運なことに、この研究は他の方法では検証できなかった非常に古い仮説を解決します」とエルサレムのヘブライ大学の数学者であるギル・カライは、この研究とは無関係であると述べました。

リンゲルの仮説は、何兆もの頂点とエッジを備えた特殊なタイプの複雑なグラフを「タイリング」できる、つまり 特定のタイプの小さなグラフの個別のコピーで完全にカバーします。概念的な観点から見ると、この質問は次のようなものです。店のタイルと同じコピーでキッチンの床を完全に並べることはできますか？実際には、ほとんどのタイプのタイルはキッチンには適していません。床を完全に覆うには、さまざまな形を組み合わせる必要があります。しかし、グラフ理論の世界では、仮説は、グラフを常にタイリングできると予測しています。

そして、キッチンやグラフでは、最初のタイルを正確にどこに配置するかが重要です。新しい作品はこの重要な問題を浮き彫りにし、そして驚くほど直観的であるようにしています。

木と森


組み合わせ論では、数学者は頂点（点）と辺（線）がどのように結合して、グラフと呼ばれるより複雑なオブジェクトを形成するかを研究します。グラフについて多くの質問をすることができます。基本的なものの1つ：大きくて複雑なグラフに理想的に埋め込まれた小さくて単純なグラフはどのような場合ですか？スタンフォード大学のジェイコブフォックス

氏は、次のように述べています。「パズルのピースのセットがあり、これらのピースから組み立てることができるかどうかはわかりません」 1963年、ドイツの数学者ゲルハルトリンゲル

彼はこの種の単純だがより広い質問を投げかけました。まず始めに、3より大きい奇数の頂点を使用してみましょう（仮説の妥当性のために、頂点の数は奇数でなければなりません。後で説明します）。それらをエッジで接続し、各頂点が他のすべての頂点に接続するようにします。次に、完全なグラフと呼ばれるオブジェクトを取得します。



次に、別のタイプのグラフを想像してみてください。これは簡単な方法である場合があります-複数のエッジを1つのラインで接続します。または、分岐リブのあるパス。ブランチは他のブランチに追加できます。閉じたループだけを回避して、任意に複雑なグラフを作成できます。このようなグラフはツリーと呼ばれます。



リンゲルの質問は、完全なグラフとツリーの関係に関係しています。彼は言った：最初に、2n + 1頂点（奇数）の完全なグラフを想像してください。次に、n + 1個の頂点で構成されるツリーを想像してください。そのようなツリーは非常に異なる場合があります。

次に、これらのツリーの1つを取り、ツリーの各エッジが完全なグラフのエッジと一致するように配置します。次に、同じツリーの別のコピーを完全なグラフの別の部分に配置します。

リンゲルは、このように行動することを予測し、適切な場所から始めれば、理想的には完全なグラフ全体をブリッジすることができます。つまり、完全なグラフの各エッジはツリーのエッジで覆われ、ツリーのコピーは重複しません。



「私は木のコピーをとることができます。グラフ全体に1つのコピーを配置しました。彼女はいくつかの肋骨を覆っています。それから私はプロセスを繰り返します。仮説はこれがすべてをカバーできることを示唆している、「言ったベニー・スーダコブとともにスイス連邦工科大学、新たな証拠の共著者のリチャード・モンゴメリーバーミンガム大学のとアレクセイポクロフスキーロンドン大学バークベック・カレッジのは。

最後に、Ringelは、使用する多くのツリーオプションに関係なく、グラフを並べて表示できると予測しました。そのような発言は一般的すぎるように見えるかもしれません。リンゲルの予想は、11個の頂点を持つ完全なグラフと、11兆+ 1個の頂点を持つ完全なグラフの両方に適用されます。完全なグラフが大きいほど、n + 1個の頂点に対して描画できるツリーの数が増えます。それらのそれぞれは、対応する完全なグラフをどのように理想的に埋めることができますか？

しかし、リンゲルの仮説が真実であると信じる理由がありました。最初の確認は、最も単純な組み合わせ算術がそのようなオプションを除外しなかったことでした。2n+ 1の頂点を持つ完全なグラフのエッジの数は、常にn + 1の頂点を持つツリーのエッジの数で完全に除算できます。

「完全なグラフのエッジの数が木のエッジの数で除算されるという事実は非常に重要です」とモンゴメリーは言った。

数学者はすぐに仮説の妥当性のさらなる証拠を見つけ、それが一連​​の発見のきっかけとなり、最終的に証明に至りました。

置き、回す


最も単純なツリーの1つは星であり、そこから発散するエッジを持つ中央ピークです。ただし、エッジが上部の周りに均等に分布している必要がないため、典型的な星の画像とは異なります。それらはすべて1つの場所から来る必要があり、中央のピーク以外の場所で交差してはなりません。リンゲル仮説の正しさを証明したい場合は、星形の木が自然な出発点になります。



そしてもちろん、数学者はすぐに、n + 1個の頂点の星が常に2n + 1個の頂点を持つ完全なグラフを完全に敷き詰めることを発見しました。この事実はそれ自体興味深いものですが、その証拠は数学者をさらに考えさせました。

簡単な例を見てみましょう。 11個のピークから始めましょう。それらを円に配置し、各頂点を他のすべての頂点と接続して、完全なグラフを取得します。



次に、それに対応する星を考えてみましょう。そこから出ている5つのエッジを持つ中心点です。



次に、中心の頂点が完全なグラフの頂点の1つと一致するように星を配置します。すべてではなく、いくつかのエッジをカバーします。次に、コンパスダイヤルを回すように、星を1つの頂点に移動します。完全なグラフの完全に異なるエッジのセットに重ねられた星の新しいコピーを取得します。

星をさらに回転させます。開始位置に戻るまでに、リンゲルが予測したとおりに、星を重ねることなく完全なグラフ全体を舗装します。



「少なくとも星の形をした木をとれば、仮説をすぐに破棄できないことはわかっています」とスダコフ氏は語った。 「さらに、これを非常に美しく表示することもできます。円にグラフを描き、星をシフトし、新しいコピーを取得し、グラフ全体を置き換えます。」

リンゲルが彼の仮説を提唱した直後に、スロバキア-カナダの数学者、アントン・コツィグはこの例を使用して、リンゲルよりもさらに大胆な予測をしました。 Ringelは、2n + 1の頂点を持つすべての完全なグラフは、n + 1の頂点を持つ任意のツリーでタイリングできると述べ、Kotsigは、スターとまったく同じ方法でタイリングできることを提案しました。

アイデアは現実的ではないように見えました。星は対称なので、どのように配置しても構いません。ほとんどの木は曲がっています。回転方法を機能させるには、非常に特殊な方法で配置する必要があります。

「星は手作業で配置できるシンプルな構造でしたが、さまざまな長さの枝の束が広がる木を手に持っている場合、それを慎重に配置する方法を想像するのは困難です」とポクロフスキー氏は語った。

Kotsig回転法を使用してリンゲル仮説を解くために、数学者は、通過できない茂みが発生しないように、木の最初のコピーを配置する方法を理解する必要がありました。幸いにも、彼らはカラフルな解決策を見つけました。

レインボーカラーはしばしば生活を楽にします。カレンダーを整理したり、大家族の食品容器をすばやく区別したりするのに役立ちます。これは、最初のツリーを完全なグラフ内に正しく配置する方法を理解するための効果的な方法にもなり得ます。

11個の頂点が円状に配置された完全なグラフをもう一度想像してください。 2つの頂点間の距離を考慮に入れる単純なルールに従って、カラーコードでエッジをマークします。

この距離は、円内のあるパスから別の頂点に（円内のパスをカットせずに）移動するために必要なステップ数によって決まります。もちろん、円では常に2方向に進むことができるため、2つのピーク間の最短経路として距離を定義します。これらが隣接する頂点である場合、それらの間の距離は10ではなく1になります。それらが1つの頂点で区切られている場合、距離は2になります。



次に、距離に従ってグラフのエッジに色を付けます。距離1にあるピークを接続するすべてのエッジを1つの色（たとえば、青）でペイントします。他の2単位の距離にあるピークを接続するすべてのエッジをペイントします（たとえば、黄色）。

同じ距離にある頂点を接続するすべてのエッジが同じ色になるように、グラフに色を付け続けます。異なる距離は異なる色でエンコードされます。 2n + 1個の頂点を持つ完全なグラフの場合、これにはn色が必要です。結果は美しく、有用な写真です。



RingelとKotzigが仮説を立てた直後に、Kotzigは彼の完全なグラフの色付けがその上にツリーを配置するためのガイドになる可能性があることに気付きました。

アイデアは、各色の1つのエッジをカバーし、どのカラーも2回カバーしないようにツリーを配置することです。数学者はこの配置を木の虹のレプリカと呼びます。カラーリングにはn色が必要であり、n + 1頂点のツリーにはnエッジがあるため、高い確率でレインボーコピーが見つかることがすぐにわかります。



1960年代の終わりまでに、数学者は木の虹のコピーには1つの特別な特性があることを理解しました。それは、木を回転させることによってグラフ全体をブリッジできる位置を示すだけです。

「レインボーコピーを作成すれば、すべてが機能します」とポクロフスキーは言いました。

その後、数学者は、2n + 1の頂点を持つすべての完全なグラフに、n + 1の頂点を持つ木のレインボーコピーが含まれていることを証明することで、リンゲル仮説を証明できることをすでに知っていました。レインボーコピーが常に存在する場合は、グラフをいつでもブリッジできます。

ただし、常に何かが存在することを証明することは困難です。このため、数学者は、完全なグラフが虹の木のコピーを含むことしかできないことを示す必要があります。それは40年以上かかりましたが、これはまさにスダコフと共著者が新作で達成したことです。

完璧なパッケージ


11個の頂点を持つ完全なグラフと6個の頂点を持つツリーが与えられたとします。完全なグラフは5つの異なる色で描かれています。ツリーには5つのエッジがあります。あなたの仕事は、完全なグラフ内のツリーのレインボーコピーを見つけることです。

ツリーのエッジを順番にグラフに配置できます。 1つ目は簡単に配置できます。そのため、任意の色の任意のエッジを選択できます。 2番目は少し難しくなります。カバーしたのと同じ色のリブを除いて、ほとんどどこにでも置くことができます。また、木の端を配置するほど、作業は難しくなります。最後のリブに到達すると、色の選択が完全に失われます-色は1つだけです。事前にすべてを計画したことを期待してください。

グラフのエッジをグラフに配置するにつれて、ツリーのレインボーコピーを見つける作業がより複雑になるというこの考えは、3人の数学者がこの問題を解決するために使用した方法の必要な部分でした。彼らは、プロセスの終わりに向けて、可能な限り多くの柔軟性を維持する方法を探しました。

当初から、彼らは非常に単純な木の虹のコピーが簡単に見つかることを知っていました-枝のない、または少数の長い道のように見える木最も難しいことは、星のように、より不規則で不規則な形状でのみ、多くのエッジが1点に集束する木を正しく配置することでした。それらを配置することは、ベビーカーを半分満たされた車のトランクに押し込むようなものです。

「完全なグラフを星のように見える厄介なものでいっぱいにしようとすると、困難が始まります」とポクロフスキー氏は語った。

トランクを埋める人なら誰でも、最も複雑なオブジェクト（大きなスーツケースや自転車などの柔軟性のないもの）から始める必要があることを知っています。ジャケットはいつでも押し込めます。そして、数学者たちはこの哲学を採用しています。

11のエッジを持つツリーを想像してください。それらの6つは中心点に収束します。他のほとんどすべてが、メインのフォームから離れた単一のフォームを形成します。



配置するのが最も難しい部分は、6つのエッジを持つピークで構成されるツリーの部分です。したがって、数学者はそれをツリーの他の部分から分離し、最初に配置して、後でこの部分を取り付けます。まるでベッドを分解して2階にドラッグしてから、目的の部屋に再度組み立てるようにした場合と同じです。彼らは一度星のこの部分をグラフに配置することすらしませんでした-彼らは完全なグラフの中にそのコピーを置くのに適した様々な場所を見つけました。

そして、ランダムに1つのコピーを選択しました。そうすることで、グラフの残りの空きスペースもランダムであることが保証されました。つまり、異なる色のエッジの分布がほぼ等しくなっています。彼らが木の残りの部分、つまり彼らが最初に置いた道をこの場所に置く必要がありました。

配置のオプションを選択するとき、彼らは制限に直面しました。パスは星に接続する必要があります。これは、すでに配置されている木の部分であり、接続する星によって以前はカバーされていなかった色もカバーする必要がありました。

しかし、数学者たちは自分たちに選択肢を残しています。彼らは、星のさまざまなコピーのほとんどすべてとパスを接続できました。さらに良いのは、星の周りのスペースがランダムであるため、ツリーの残りの部分を覆うために異なる色を選択するオプションがあったことです。

「ツリーを構築する最後の方に向かって、状況が複雑になったとき、私は1つの必須の色ではなく、小さな選択肢があります」とモンゴメリーは言いました。

3人の数学者が確率理論からの議論で証明を完了します。彼らは、木々の最も複雑な部分を埋め込んだ後、列全体に残っているスペースが本質的にランダムであるならば、いつでも残りの木を組み込んで虹のコピーを得る方法を見つけることができることを示しました。

プリンストン大学のノガアロン氏は、「これで、最初から延期したものを強制して、まるで木の残りを吸収して完全な虹色にするかのように強制できます」と語った。

数学者は、2n + 1の頂点を持つ各完全なグラフで、n + 1の頂点を持つ各ツリーのレインボーコピーを見つける正確な方法を説明していません。しかし、彼らはレインボーコピーがその中になければならないことを証明しました。

また、レインボーコピーが常に存在する場合は、リンゲルによって予測された方法でグラフをいつでもブリッジできます。したがって、彼の仮説は真実です。

証明は、組み合わせ論における同様の未解決の問題を解決するための新しいツールも提供します。たとえば、「より適切なマークアップ」の問題は、より厳密な条件下でも完全なグラフをタイリングできるため、ツリーをさらに正確に配置する必要があることを予測します。

「人々が長い間考えてきた方法が本当に大きな可能性を秘めていることを示しています」とフォックスは言いました。「それらを正しく適用すれば、以前は難攻不落と思われていた問題を解決できます。」
