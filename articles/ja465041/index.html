コイン残高のリストを取得する方法


Electrumプログラムの場合、この方法を見つけました（コンソールコマンド）：


' '.join((x["value"]) for x in listunspent())

特定のコインを、たとえば特定の住所で除外したい場合は、次のように実行できます。


' '.join((x["value"]) for x in listunspent() if x["address"] != "bad address")

他の財布については、そのような簡単な方法が見つからず、手で再入力する必要がありました。
