Microsoft ya no emitirá certificados MCSE / MCSD / MCSA

En cinco años, bajo el sabio liderazgo de Satya Nadella, Microsoft ha perdido el liderazgo tecnológico global. A pesar del hecho de que todos los esfuerzos se dirigieron a la dirección de la nube, Azure no pudo ponerse al día y omitir aún más la plataforma de popularidad de Amazon, el pionero y líder permanente en este mercado. No queriendo atraer financiamiento externo para el desarrollo de Azure, la compañía bombeó dinero desde otras direcciones, como resultado Microsoft perdió tabletas Apple y computadoras de escritorio profesionales, tecnologías móviles de Google, juegos y AR / VR para todos en una fila, los sistemas corporativos de Salesforce no pudieron llevar su publicidad la red Bing y el motor de búsqueda, y las personas decentes con mujeres ni siquiera recuerdan las redes sociales decentes de Yammer. Siendo desorganizado, comido internamente por la competencia interna e incapaz de innovar,recientemente, solo compra desarrollos de otras personas: GitHub, LinkedIn, Xamarin y otros proyectos ya completados y maduros.

En esencia, Microsoft se está moviendo para convertirse en un importante fabricante de editores de texto y un servidor de Linux. Por supuesto, dicha empresa no necesita socios y desarrolladores, por lo tanto, en febrero de este año, anunció que cerraría los programas de certificación para desarrolladores y administradores de ingenieros , a saber:

  • MCSA: informes de BI
  • MCSA: Dynamics 365 para operaciones
  • MCSA: Desarrollo de BI de SQL 2016
  • MCSA: Administrador de base de datos SQL 2016
  • MCSA: Dev de base de datos SQL 2016
  • MCSA: SQL Server 2012/2014
  • MCSA: plataforma universal de Windows
  • MCSA: aplicaciones web
  • MCSA: Windows Server 2012
  • MCSA: Windows Server 2016
  • MCSD: Creador de aplicaciones
  • MCSE: aplicaciones comerciales
  • MCSE: Infraestructura central
  • MCSE: gestión de datos y análisis
  • MCSE: productividad

Como puede ver en la lista, Microsoft cree que en el mundo moderno no son necesarios: desarrolladores web, desarrolladores y administradores de bases de datos, desarrolladores de aplicaciones de escritorio y tabletas, administradores de servidores locales. Se suponía que debía detener la certificación de estos programas hasta el 30 de junio.

El nuevo conjunto de exámenes implica principalmente la capacitación de especialistas en la configuración de una tarjeta de crédito corporativa para pagar los servicios de Azure, hacer clic con el mouse para registrarse en Azure Active Directory y promover el mismo editor de texto como parte de Office 365:



no habría felicidad, pero la desgracia ayudó: en relación con la pandemia COVID-19, Microsoft Learning amplía la aceptación de aplicaciones de certificación recientespara programas para especialistas reales hasta el 31 de enero de 2021, y también promete aumentar el número de exámenes tomados de forma remota.

Si aún no ha actualizado su MCSA, ahora sabe qué hacer durante la cuarentena.
