キングストンのサノスレベルの脅威

ほとんどのユーザーがメモリカードを購入するとどのように見えますか？価格タグと保管容量が満たされている場合、店で最初の人目を引くモデルが採用されます。啓発期間は後で始まります。写真が古くなっても、カメラは最大解像度でビデオを記録せず、デバイスは悪条件での作業に耐えることができません。したがって、メモリカードを選択するときは、データの読み取り/書き込みの速度に注意を払う必要があります。同時に、このようなカードは深刻な揺れ、温度変化、湿気との接触に耐えることができますか。キングストンキャンバスセレクトプラスは、写真を撮ったり、旅行に出かけたり、機器を過酷な条件にさらしたりする場合に最適です。なぜこれが下のケースなのか。



メモリーカードを選ぶときに知っておくべきことは？


訓練を受けていない人は、メモリカードのパッケージのマーキングに混乱する可能性があります。また、Habrリーダーである上級ユーザーにとっても、知識をリフレッシュすることは役に立ちます。適切なドライブを選択するには、いくつかのパラメーターに注意を払う必要があります。

現在、SDXCカードのフォーマットが最も適切です。古いSDやSDHCとは異なり、そのようなカードの容量は2 TBに達することがあります。 SDXCの「フルサイズ」バージョンがほとんどのラップトップとカメラでの使用に適している場合、microSDXCはスマートフォン、アクションカメラ、タブレットPCで広く使用されています。



通常のスピードクラスは、文字Cと数字で示されます。たとえば、C10は、読み取りおよび書き込みについて、少なくとも10 MB /秒のカードパフォーマンスを保証します。

ビデオ速度クラス ビデオを録画する予定で、制限に直面したくない人には特に重要です。パッケージとカード本体では、文字V内の数字で示されます。これは、書き込み速度をMB /秒で示します。ビデオを超高解像度で撮影する場合は、V30以降のマークが付いているドライブを選択することをお勧めします。

UHSスピードクラス（Ultra High Speed）は、文字U内の数字（1または3 ）で示されます。これは、快適にビデオを録画するために注意する必要があるもう1つのパラメーターです。フルHD解像度のビデオストリームにU1で十分である場合、U3はすでに4K形式での撮影を保証しています。

多くのバイヤーにとって不可解なローマ数字IまたはIIは、UHSバスの速度を示します。これは、上記の速度クラスとは直接関係がなく、インターフェースの可能な最大速度を示すだけです。技術的な詳細には触れませんが、今日のほとんどの場合、UHS-Iパラメータは十分すぎるため、「誇示」がマップに表示されても動揺しないでください。 UHS-IIフォーマットは、100 MB /秒を超える速度が本当に必要なプロフェッショナル向けです。そして、そのようなカードの値札はより印象的です。

アプリケーションパフォーマンスクラス-SDカードには少し便利なオプションですが、microSDには重要です。小さなブロックで情報を読み書きする速度を示します。したがって、スマートフォンまたはタブレットのメモリを拡張しようとするすべての人に注意を払うことが不可欠です。ここでは、カテゴリA1とA2が区別されます。最小線形書き込み速度はどちらの場合も同じです-10 MB /秒ですが、A2とマークされたカードを使用すると、データのランダムな書き込みと読み取りが速くなります-毎秒4,000の小さなファイルを読み取り、2,000を書き込みます。 A1。ガジェットの特定のニーズと機能を満たすカテゴリを選択する必要があります。

この知識を武器に、カード本体の「SDXC V30 U3 C10」のようなマーキングを解読することは難しくありません。次に、テスト参加者を見てみましょう。

接続するには、専用のカードリーダーKingston FCR-HS4を使用します。このデバイスは、メモリカードのすべての主要なフォーマットで動作し、最大5 Gb /秒のデータ転送速度をサポートします。まず、CrystalDiskマークベンチマークを実行して結果を評価します。

キングストンキャンバスセレクトプラスmicroSDの機能


Canvas Select Plus microSDモデルに移ります。このフォーマットのカードは、多くの場合、内蔵ドライブとペアでスマートフォンにインストールされます。同時に、最新のデバイスは4K解像度でファイルを記録し、高速UFSメモリを誇示します。通常の使用例では、速度に顕著な違いはありますか？



まず、Redmi Note 8 ProスマートフォンにインストールされたA1 SDベンチプログラムを使用してフォーマット済みカードをテストします。



次に、外部ドライブに記録された「重い」ファイルの再生状況を確認します。前述のスマートフォンで、MX Playerを介して2時間の5.77 GBムービーを起動します-再生に問題はありませんでした。

達成したことに止まることなく、H.264コーデックを使用して、毎秒30フレームの頻度で4Kでビデオを撮影します。 5分のビデオはすぐにスマートフォンのギャラリーに表示されず、忍耐が必要でした。ただし、ビデオはアーティファクトやフレームドロップによって損なわれることはありませんでした。



microSDカードを使用する別のシナリオは、すでに「古い」ものの、活発なスマートフォンにも当てはまります。誰もが従業員を忠実に変えることを望んでいるわけではありませんが、彼の記憶は必然的に数年でいっぱいになります。たとえば、かつて人気のあった「予算」のXiaomi Redmi 4Aに16 GBの内部メモリが搭載されています。組み込みのファイルマネージャーを使用して、データの一部をKingston Canvas Select PlusスマートフォンにインストールされているmicroSDに転送します。合計重みが1.24 GBの28個のファイルを転送するには61秒かかり、この操作後のファイルを開く速度は低下しませんでした。



多くのAndroidデバイスでは、外部カードを内部メモリとして使用できます。写真や動画だけでなく、一部のアプリケーションもmicroSDカードに移動することで、スマートフォンの内部メモリのかなりの部分を解放できます。この方法でメモリカードを使用するには、まずデバイス自体でフォーマットする必要があります。これは、設定メニューから実行できます。しかし、ほとんどの場合、新しいドライブが接続されたときにスマートフォン自体がこの手順を実行することを提案し、カーテンで通知を発行します。

アプリケーションが多すぎる場合、手動での移動は長くて不便です。さらに、すべてのプログラムが、インストール後の別のドライブへのデータ転送をサポートしているわけではありません。ローミング要素をすばやく見つけるために、AppMgr IIIなどの特別なアプリケーションをインストールできます。ここでは、簡単な指示に従って、ドライブ間で移動できるものをすぐに確認できます。これにより、スマートフォンの内部メモリを500MB以上解放することが可能になります。多くのアプリケーションがある場合は、数ギガバイトのデータについて話します。



そして最後のテストはYI 4Kアクションカメラです。実行時に毎秒60フレームのフルHDでビデオを撮影します。カードは問題なくテストに合格しました。振ることは障害ではありません。録音は中断されませんでした。



水も霜も怖くない


最後に、最も興味深い。 Kingston Canvas Select Plusカードのセキュリティをチェックします。ドライブは、急激な温度変化や液体との接触に耐えますか？確かに、印象的な動作温度範囲が特性で宣言されています：-25°〜85°C。

はじめに、メモリーカードは冷凍庫に直行し、氷上に20分間留まりました。その後すぐに操作性を確認し、SDカードをカメラに、microSDをアクションカメラに挿入して撮影を開始した。寒さの中で、どちらのドライブも害を与えることなく移動しました。録画は「途切れ」ません。ビデオシーケンスに欠陥はありません。



体験はそこで終わりませんでした-各カードはきれいな真水に30分間放置されました。そしてCanvas Select Plus SD、およびmicroSD-modificationは、水分テストで優れた仕事をしました。空母は完全に機能し続け、強制水泳は仕事の質に影響を与えませんでした。



このようなカードは、スマートフォンやカメラだけでなく、DVR、監視システム、さまざまなサービスの従業員のポケットレコーダーでも安全に使用できます。そして、非常に極端な労働条件のために、キングストンは産業で使用できるインダストリアルシリーズのドライブの別のラインを持っています。
