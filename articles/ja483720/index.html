私たちにパンダ1.0をもたらしたもの


1月9日、Pandas 1.0.0rcがリリースされました。ライブラリの以前のバージョンは0.25です。


最初のメジャーリリースには、データフレームの自動集計の改善、より多くの出力形式、新しいデータタイプ、さらには新しいドキュメントサイトなど、多くの優れた革新が含まれています。


すべての変更はここで確認できます。この記事では、最も重要なことについて、技術的ではない小規模なレビューに限定します。


ライブラリは通常どおりpipを使用してインストールできますが、執筆時点ではまだPandas 1.0がリリース候補であるため、バージョンを明示的に指定する必要があります。


pip install --upgrade pandas==1.0.0rc0

注意：リリースはメジャーなので、更新により古いコードが壊れる可能性があります！


ちなみに、このバージョンではPython 2のサポートが完全に廃止されています（これは、アップグレードするのに十分な理由かもしれません-およそTransl。）Pandas 1.0には少なくともPython 3.6+が必要です。不明な場合は、インストールされているものを確認してください。


$ pip --version
pip 19.3.1 from /usr/local/lib/python3.7/site-packages/pip (python 3.7)

$ python --version
Python 3.7.5

Pandasのバージョンを確認する最も簡単な方法は次のとおりです。


>>> import pandas as pd
>>> pd.__version__
1.0.0rc0
