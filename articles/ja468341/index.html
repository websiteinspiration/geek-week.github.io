Windowsターミナル上のAzure Cloud Shell

これで、WindowsターミナルからAzure Cloud Shellに接続できます。



これを行うために、Windowsターミナルアプリケーションを介してAzureディレクトリ/クライアントにアクセスできる新しいデフォルトプロファイル-Azure Cloud Shellを追加しました。

すでにWindowsターミナルがインストールされている場合


Windowsターミナルの設定は新しいプロファイルに自動的に更新されません（Windowsターミナルを開くたびにファイルが更新されないため）。手動で追加する方法は次のとおりです。

  1. Windowsターミナルを起動
  2. 設定を開く（ドロップダウンリストを使用）
  3. このプロファイルをプロファイルのリスト（アイテムプロファイル）に追加します。 

    {"acrylicOpacity" : 0.6,
    "closeOnExit" : false,
    "colorScheme" : "Vintage",
    "commandline" : "Azure",
    "connectionType" : "{d9fcfdfa-a479-412c-83b7-c5640e61cd62}",
    "cursorColor" : "#FFFFFF",
    "cursorShape" : "bar",
    "fontFace" : "Consolas",
    "fontSize" : 10,
    "guid" : "{b453ae62-4e3d-5e58-b989-0a998ec441b8}",
    "historySize" : 9001,
    "icon" : "ms-appx:///ProfileIcons/{b453ae62-4e3d-5e58-b989-0a998ec441b8}.png",
    "name" : "Azure Cloud Shell",
    "padding" : "0, 0, 0, 0",
    "snapOnInput" : true,
    "startingDirectory" : "%USERPROFILE%",
    "useAcrylic" : true}

これを実行すると、Azure Cloud Shellタブが使用可能になります。

標準設定


  1. [Azure Cloud Shell]タブをクリックします。
  2. 「microsoft.com/devicelogin」に移動して、表示されたコードを入力するように求められます。
  3. ブラウザにコードを入力したら、アカウントでログインする必要があります。また、アクティブなAzureディレクトリ/クライアントを持つアカウントでログインしていることを確認してください。
  4. ターミナルに戻ります。数秒後、「Authenticated」というメッセージが表示されます。
  5. : , . , .
  6. . , 1-5.
  7. Cloud Shell ( , ).
  8. Azure Cloud Shell!

「n」を押して別のアカウント/クライアントでログインするか、「r」を押して保存された接続を削除することもできます。これらのパラメーターはセッション中に保存されるため、数日後に端末を起動しても、ブラウザーを再度開かなくても、保存した接続パラメーターでログインできます。
