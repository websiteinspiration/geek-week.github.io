みなさん、こんにちは。「Fullstack JavaScript Developer」コースの開始を見越して、興味深い資料の翻訳を用意しました。




TL; DR：独自のコンポーネントを作成するか、設計システムを使用するか、さまざまなブラウザのサポートを実装するか、アクセシビリティを提供するか、サードパーティの依存関係を追加するかに関係なく、Web開発は困難です。 Lionは、フル機能を備えた、手頃な価格の、生産性の高い、特定されていないコンポーネントの実装に取り​​組むことで、あなたの人生をより簡単にすることに努めています。ing-bank / lionリポジトリをチェックしてください。

INGには、Polymerライブラリーから最近のLitElementまで、Webコンポーネントの作成に関する長くて豊かな歴史があることをすでにご存知の方もいるかもしれません。

あなたの記憶 をリフレッシュしたいですか？Webコンポーネントは、ネイティブで再利用可能なカプセル化されたモジュール式コンポーネントを作成できるようにする一連のブラウザー標準です。

Webコンポーネントの標準がついに強化され、今日では、Angular Elements、Stencil、Vue、Svelteなど、Webコンポーネントを実装および使用する多くの方法があります。また、Chromium Edgeのリリースにより、すべての主要ブラウザがそのままWebコンポーネントをサポートすることが明らかになりました。

したがって、本日は、INGがWebコンポーネントLionのライブラリで無料で利用できるようになったことを皆さんと共有できることを嬉しく思います。

ライオンとは？


ウェブを宣伝したい-一度に1つのコンポーネント。
Lionは、フレームワークに依存しないオープンソースのホワイトラベルライブラリであり、会社の設計システムの基礎になることができます。外観と機能の調整は困難な作業です。Lionを使用して作業を円滑に行えることを願っています。

白いラベル


LionはWebコンポーネントの基本的なホワイトラベルパッケージです。つまり、コンポーネントのスタイルは最小限ですが、基本的な機能はすべて揃っています。ホワイトラベル製品は、多くの場合、他のユーザーがコンポーネントの視覚的アイデンティティを自分自身に適合させることができるように設計されています。驚くべきことに、誰もがオレンジ色を好むわけではないので、これは、誰もが独自のブランディングまたはデザインシステムを使用してコンポーネントの基本機能を使用できることを意味するので、これは素晴らしいことです。

それがINGで行っていることです。独自のing-webコンポーネントは、Lionコンポーネントを拡張し、Lionの上の薄層である視覚的なING IDを適用します。ライオンの

主な焦点


Lionは、グローバルな使用と再利用性に重点を置いて開発されました。この結果、最初から以下の機能が追加されました。

  • 再利用-私たちのコンポーネントは、世界中で配布および使用されるように設計されています。
  • アクセシビリティ-私たちのコンポーネントはすべての人がアクセスできるように設計されています。
  • 生産性-私たちのコンポーネントは小さく、生産的で高速でなければなりません。

これらのすべての機能により、コンポーネントをグローバルに拡張し、それらに取り組むときに相互理解を見つけることができます。これにより、INGのすべてのユーザーが、アプリケーションを構築して迅速に開始するための確実なビルディングブロックのセットを持つことが保証されます。

学んだ教訓


INGは非常に早い段階でWebコンポーネントの使用を開始しており、Lionは私たちが作成した最初のコンポーネントライブラリではありません。したがって、これらのコンポーネントが前の世代とどのように異なるのか疑問に思うかもしれません。

Lionは、アクセシビリティと拡張性を提供するためにゼロから構築されました。開発の後の段階でこれらを追加することはほとんど不可能であることを知っていたからです。Lionの作成中に学んだいくつかの教訓を強調したいと思います。

  • すべてがWebコンポーネントになる必要はありません。特定の機能を追加するには、通常のJavaScriptを使用することをお勧めします。
  • ネイティブHTML要素にできる限り近づけます。
  • 最初からアクセシビリティに取り組みます。
  • すべてがシャドウDOMにある必要はありません。これは、ariaの関係とアクセシビリティにとって特に重要です。
  • UIコンポーネントは複雑です。

コンポーネントの拡張


独自の設計システムを実装するためのベースとしてLionを使用できます。

彼女がライオンを選んだ場合、リーの物語がどのように起こったか見てみましょう。

開始するには、必要なものをすべてインストールします。

npm i lit-element @lion/tabs

lea-tabsLionの機能を再利用して コンポーネントを作成します。これにより、Leahは独自のタブコンポーネントを実装するために必要なすべての機能的負荷とアクセシビリティを取得できました。

import { css } from 'lit-element';
import { LionTabs } from '@lion/tabs';

export class LeaTabs extends LionTabs {
  static get styles() {
    return [
      super.styles,
      css`
        /* my stylings */
      `
    ];
  }

  connectedCallback() {
    super.connectedCallback();
    this._setupFeature();
  }

  _setupFeature() {
    // being awesome
  }
}

customElements.define('lea-tabs', LeaTabs);

リーはまた、スタイルできるようにしたいと考えてタブやタブパネルを Betatechsのビジュアルアイデンティティによります。これを行うために、彼女はlea-tab-panelコンポーネントとlea-tabコンポーネントを作成します。これらは、適切と思われるスタイルにして、最終的にlea-tabsコンポーネントを中に配置できます。以下の例で、Leaがこれをどのように行ったかを確認できます。

import { LitElement, html, css } from 'lit-element';

export class LeaTab extends LitElement {
  static get styles() {
    return css`/* my stylings */`;
  }

  render() {
    return html`
      <!-- dom as needed -->
      <slot></slot>
    `;
  }
}

customElements.define('lea-tab', LeaTab);

完璧に！これで、Leaはコンポーネントtabsを次のように使用できます。

<lea-tabs>
  <lea-tab slot="tab">Info</lea-tab>
  <lea-tab-panel slot="panel">
    Info page with lots of information about us.
  </lea-tab-panel>
  <lea-tab slot="tab">Work</lea-tab>
  <lea-tab-panel slot="panel">
    Work page that showcases our work.
  </lea-tab-panel>
</lea-tabs>

さて、Leaコンポーネントの準備が整いました。次はドキュメントを作成します。Leaのドキュメントのライブページをご覧ください。完全なコードlea-tabsはGitHubで確認できます。

PS：Leaがlea-tabsドキュメントの更新を担当し、Lionドキュメントへの変更はLeaドキュメントに自動的に反映されることに注意してください。

なぜオープンソースなのか？

コンポーネントライブラリは需要が高いです。拡張可能なコンポーネントのソースコードへのアクセスを提供することで、オープンソースコミュニティのすべての利点を利用しながら、必要なことを実行できるように支援します。さらに、Lionへの貢献は、それを使用する各設計システム（ing-web）世界規模で。これは、私たちが両方の世界を最大限に活用し、コンポーネントで人々を助け、コミュニティから貴重な意見を得るということです！

私たちのリポジトリを見てください！そして、最新の状態にしたい場合は、追跡を開始するか、アスタリスクを付けてください。Twitterはまだ（まだ）ありませんが、私に登録することはできます。こんにちは、トーマスオルマーです。

しかし、それだけではありません！


アプリケーションの作成は困難です。たとえば、検証、フォーム、オーバーレイ、ローカリゼーションなど、特定のコンポーネントが1つ以上必要な場合もあります。恐れる必要はありませんが、Lionが助けになります。
