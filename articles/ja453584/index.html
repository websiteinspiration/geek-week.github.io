TeTesBed.getを操作するときの任意のタイプの使用


メソッドにTesBed.getは2つのシグネチャがありました。1つは型付き、もう1つは受信と戻りの型anyです。タイプの使用を提供するメソッドシグネチャanyは非推奨になりました。このメソッドは、特定のタイプでのみ使用できます。これは、たとえば、文字列トークン（サポートされていない）や他のいくつかのトークンを操作する場合に影響します。

以前に使用されたそのようなデザイン：

TestBed.configureTestingModule({
  providers: [{ provide: "stringToken", useValue: new Service() }],
});
let service = TestBed.get("stringToken"); //  any

ここで、次のアプローチが適用されます。

const SERVICE_TOKEN = new InjectionToken<Service>("SERVICE_TOKEN");
TestBed.configureTestingModule({
  providers: [{provide: SERVICE_TOKEN, useValue: new Service()}],
});
let service = TestBed.get(SERVICE_TOKEN); //  Service
