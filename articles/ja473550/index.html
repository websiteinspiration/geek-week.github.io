ジョーカー2019：今年の勝利

1週間前の2019年10月25〜26日、サンクトペテルブルクでJoker 2019 Javaカンファレンスが開催されました。その上で何が起こっていたのか、そこにどのような報告があったのか、興味深いと思われたこと-これらすべてについてさらに。



今年のホアキンフェニックスというタイトル の映画の世界中での驚異的な配信と同様に、大規模なExpoforumのExpoforumで開催されたJavaカンファレンスも非常に成功しました-これは2,000人の記録を打ち破りました（コメントで正確に何人だったかを指定します）。

キャパシティの異なる4つのホールに加えて、並行トラックのレポートが読み込まれたほか、会社のスタンド、企業やコミュニティからのミニプレゼンテーション用のデモステージ、Lightningトーク（会議参加者からのミニプレゼンテーション）、BOFセッションなどの展示エリアがありました。



展示エリア

Java User Groupsブース


8月末にTechTrain 2019 ITフェスティバルで始まった良い伝統が続き、Javaコミュニティーの共同スタンドが組織されました。ご覧のように、スタンドにはモスクワ、サンクトペテルブルク、ノボシビルスクのコミュニティのリーダーや活発な参加者が再び出席しました。 スタンドは、古いコミュニティメンバー、スピーカー、通りがかりの会議参加者のためのコミュニケーションポイントとして機能しました。コミュニケーションや感想交換のレポートの打ち合わせの場としても便利でした。下の写真は、Andrei Kogun（ついに彼の有名なTシャツ「The Same Kogun」を見た）とAlexei RagozinおよびAndrey Ershovの写真です。TechTrain 2019 とは異なり







、賞品は配りませんでしたが、Guess the Speakerゲームのアップデートを用意しました。ゲームはjugspeakers.onlineで引き続き利用できます。共同開発アプリケーションのコードを含むリポジトリがGitHubにあり ます（「スター」を自由に付けてください。作者は喜んでいます）。プログラムに次の機能的な変更が加えられました。

IvanPonomarev（彼はまた、翌日のOlegのレポートにも行きました。Olegもこのサービスについて言及しています）。Dependabotは、リポジトリにプルリクエスト（PR）を作成することで、アプリケーションの依存関係バージョンの更新を自動化します。これに必要な手順：

  • リポジトリをサービスに追加します。
  • 作成されたPRサービスを順次扱います。
  • 必要に応じて、README.mdサービスステータスバッジをファイルに追加します（ここにあるバッジのリンクを参照）。
