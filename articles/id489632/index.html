Studi RedHat: Open Source mendorong perangkat lunak berpemilik dari segmen perusahaan




Perangkat lunak Open Source secara bertahap mendapatkan segmen perusahaan, kesimpulan ini diterbitkan dalam laporan penelitiannya oleh tim RedHat. Perusahaan mencapai hasil seperti itu setelah survei terhadap 950 eksekutif perusahaan IT di seluruh dunia. Berikut adalah hasil ringkasannya:

  1. 95% menganggap Open Source secara strategis penting;
  2. Pangsa perangkat lunak berpemilik di segmen perusahaan telah menurun dengan kecepatan serius selama beberapa tahun, dari 55% pada 2019 menjadi 42% sekarang dan, menurut perkiraan, menjadi 32% pada 2021;
  3. 77% berpendapat bahwa Open Source perusahaan akan terus tumbuh;
  4. keunggulan utama Open Source disebut "kualitas lebih tinggi", biaya lebih rendah hanya di tempat kedua;
  5. area utama aplikasi: keamanan, alat manajemen infrastruktur cloud, database, data besar dan analitik;
  6. hambatan untuk digunakan: keamanan kode, tingkat dukungan, kompatibilitas, kurangnya kompetensi dukungan sendiri.

MyPaint 2.0




Menurut It's FOSS, tempo hari ada rilis besar MyPaint, salah satu alternatif Open Source terbaik untuk Microsoft Paint. Ini adalah program kecil yang berguna untuk membuat sketsa dan membuat sketsa cepat. Meskipun ada alat yang lebih canggih untuk seniman digital seperti Krita, MyPaint masih bagus untuk tugas-tugas sederhana. Itu juga dapat digunakan pada tablet Wacom tanpa masalah. Rilis 2.0 baru termasuk dukungan untuk Python 3, mode layer baru, opsi kuas baru dan perubahan lainnya.

Sistem GARANT tersedia untuk GNU / Linux




Perusahaan Garant menawarkan untuk mengunduh paket distribusi referensi hukum periodik elektronik miliknya untuk OS GNU / Linux. Peran OpenShift dalam evolusi model organisasi dalam transisi ke PaaS




RedHat pada posting blog HabrÃ© mencatat tentang bagaimana keluarga perangkat lunak OpenShift containerization mengubah struktur organisasi organisasi TI ketika pindah ke Paas. Kutipan - "pengembalian maksimum investasi dalam PaaS seringkali hanya mungkin jika peran organisasi, bidang tanggung jawab (tugas) dan hubungan berubah." Prinsip yang mengarah pada perubahan tersebut:

  1. memecah pekerjaan menjadi tahap kecil untuk menerima umpan balik pada tahap awal, mengurangi risiko dan menghindari "kelumpuhan analitis";
  2. mengotomatiskan operasi secara memadai agar tidak menciptakan hambatan atau hambatan dalam proses penyebaran aplikasi;
  3. berbagi pengetahuan adalah kunci untuk membangun kepercayaan;
  4. secara teratur membayar utang teknis, mengalokasikan sejumlah waktu dalam setiap siklus kerja untuk perbaikan sistematis.

Utilitas KDE GPG Disetujui di Jerman untuk Transfer dan Pemrosesan Informasi Rahasia




Menurut komunitas KDE VKontakte, Administrasi Keamanan Informasi Federal Jerman telah menyetujui penggunaan Gpg4KDE dan Gpg4win untuk mengirim dan memproses informasi nasional yang sensitif. Gpg4KDE adalah mekanisme enkripsi yang digunakan KMail saat mengenkripsi dan menandatangani email. Gpg4win adalah seperangkat alat enkripsi file dan email untuk Windows, yang mencakup Kleopatra, manajer sertifikat dari KDE.

GamePad Mengumumkan Platform Permainan FOSS Baru yang Dirancang Untuk GNU / Linux

  1. antarmuka yang nyaman;

Di Inggris, orang tua disarankan untuk menghubungi polisi jika anak-anak mereka menggunakan VirtualBox, Discord, Kali Linux dan Tor




Poster-poster muncul di sekolah-sekolah di county West Midlands Inggris, menyerukan kepada orang tua dan guru untuk menghubungi polisi jika ada siswa yang menggunakan OS Kali Linux, Discord messenger, tool onion routing Tor, alat virtualisasi VirtualBox dan beberapa perangkat lunak lainnya. Poster-poster itu diduga dikeluarkan atas perintah polisi setempat. Publikasi informasi ini segera mendapat reaksi negatif yang tajam dari banyak orang dan segera bergegas untuk menolak poster dari poster NCA (Badan Anti-Kejahatan Nasional Inggris), yang logonya diletakkan di poster. Dalam kritik, argumen utama terhadap poster semacam itu adalah ketidakmampuan teknis penulis dan fakta bahwa perangkat lunak yang terdaftar sama sekali tidak menunjukkan dengan jelas bahwa anak tersebut melakukan sesuatu yang mencurigakan.Setelah dikritik, polisi setempat juga menyalahkan pihak ketiga.
