Perbandingan platform pengembangan 1C dan Communicator Klien

Dalam artikel ini saya akan mempertimbangkan 1C dan KliK bukan sebagai solusi yang siap untuk diimplementasikan, tetapi sebagai platform untuk menciptakan solusi individual.
1C Communicator Klien
Versi gratis
Tidak ada.
Tutorial gratis tersedia dengan banyak batasan berbeda.
 Konfigurasi apa pun dapat digunakan dalam mode pengguna tunggal secara gratis dan tanpa batasan.
Instalasi
Opsi file (tidak disarankan untuk operasi jaringan):
1. Instal platform;
2. Buat basis informasi.
Versi server :
1. Menginstal DBMS pilihan Anda (MS SQL, PostgreSQL, IBM DB2, PostgresPro, Oracle Database);
2. Menginstal platform di server;
3. Buat infobase di server;
4. Instal di komputer klien dan konfigurasikan akses ke infobase.

Opsi server diinstal, sebagai aturan, oleh spesialis 1C yang berpengalaman.
 1. Menginstal MS SQL;
2. Kembalikan database di server;
3. Instal di komputer klien dan konfigurasikan akses ke database.
Instalasi bahkan tidak ditangani oleh spesialis IT.
