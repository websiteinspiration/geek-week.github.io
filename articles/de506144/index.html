Vergleich der Entwicklungsplattformen 1C und Client Communicator

In diesem Artikel werde ich 1C und KliK nicht als implementierungsbereite Lösung betrachten, sondern als Plattform zum Erstellen einer individuellen Lösung.
1C Client Communicator
Kostenlose Versionen
Sind abwesend.
Ein kostenloses Tutorial ist mit vielen verschiedenen Einschränkungen verfügbar.
 Jede Konfiguration kann im Einzelbenutzermodus kostenlos und ohne Einschränkungen verwendet werden.
Installation
Dateioption (nicht für den Netzwerkbetrieb empfohlen):
1. Installieren Sie die Plattform.
2. Erstellen Sie eine Informationsbasis.
Serverversion :
1. Installieren eines DBMS Ihrer Wahl (MS SQL, PostgreSQL, IBM DB2, PostgresPro, Oracle Database);
2. Installieren der Plattform auf dem Server;
3. Erstellen Sie eine Infobase auf dem Server.
4. Installieren Sie auf Client-Computern und konfigurieren Sie den Zugriff auf die Infobase.

Die Serveroption wird in der Regel von einem erfahrenen 1C-Spezialisten installiert.
 1. Installieren von MS SQL;
2. Stellen Sie die Datenbank auf dem Server wieder her.
3. Installieren Sie auf Clientcomputern und konfigurieren Sie den Zugriff auf die Datenbank.
Die Installation wird nicht einmal von einem IT-Spezialisten durchgeführt.

Schlussfolgerungen:
1. Der anfängliche Kaufpreis in der ursprünglichen Version ist der gleiche.
2. Der Preis für 1C-Service ist viel höher, weil Sehr oft wird die Hilfe eines Programmierers benötigt.
3. Ein Teil der Funktionalität in 1C ist verfügbar, jedoch gegen eine zusätzliche Gebühr.
4. Die Benutzeroberfläche im Client Communicator ist wesentlich komfortabler.
5. Verbesserungen in 1C sind zeitaufwändiger und erfordern eine bestimmte Qualifikation eines Spezialisten.
6. Für 1C verwendet das Unternehmen den besten verfügbaren Server, auch nur für Buchhaltungsaufgaben.
