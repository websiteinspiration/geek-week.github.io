“Ditemukan, lihat, terima”: undangan wawancara yang tidak biasa - dari tajuk HTTP ke pesan di mesin pencari

Beberapa perusahaan menyembunyikan penawaran untuk pengembang dalam kode HTML atau header HTTP, sementara yang lain menawarkan terlebih dahulu untuk memecahkan masalah algoritmik di situs. Mari kita bicara tentang beberapa kasus seperti itu.





— Nikita Kachanovsky — Unsplash



«»



Banyak perusahaan menyembunyikan tawaran pekerjaan dalam kode HTML. Misalnya, Microsoft memasukkan iklan SDM pada halaman Azure. Pendekatan serupa diikuti oleh The New York Times dan Imgur . Mozilla juga mengirim undangan untuk bekerja di konsol browser. Tetapi beberapa organisasi sangat inventif dan menyembunyikan tawaran di header HTTP .



Pada 2015, Troy Hunt, seorang spesialis keamanan informasi, mengadakan lokakarya di sebuah perusahaan Australia. Selama latihan, salah satu siswa memperhatikan respons HTTP menarik yang diterima melalui API seluler dari server Airbnb. Itu berisi undangan untuk bekerja dan alamat email kepala departemen pengembangan platform online untuk menemukan perumahan: 



X-Hi-Human: The Production Infrastructure team added this header.
Come work with us! Email kevin.rice+hiring@airbnb.com


Sebulan yang lalu , materi muncul di Hacker News , penulis yang menemukan situs lain dengan tawaran pekerjaan di header HTTP. Mereka ditemukan di Paypal.me, Booking.com, Etsy.com dan Otto.de dan banyak lainnya. Penulis telah menerbitkan daftar lengkap di repositori di GitHub .



Terkadang, untuk menemukan undangan tersembunyi, Anda perlu menggali lebih dalam. Misalnya, pada tahun 2017, halaman khusus dengan penawaran tersebut diposting di Apple. Untuk membukanya, perlu untuk menganalisis lalu lintas aplikasi di iPhone. Beberapa dari mereka meneruskan paket ke server blobstorebiasa digunakan untuk hosting data iCloud. Tawaran itu hanya terletak di salah satu alamat, tetapi Apple harus menghapusnya dari situs. Faktanya adalah bahwa ia ditemukan oleh editor majalah ZDNet dan diterbitkan di Twitter. Setelah itu, korporasi mulai menerima terlalu banyak aplikasi.
