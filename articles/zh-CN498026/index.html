Git 15-




Git版本控制系统的第一版于15年前的2005年4月7日发布。 Git最初是Linux内核的硬通货，因为之前使用过的BitKeeper中使用的许可证已更改。但是直到今天，Git的硬货币功能已经远远超过了原来的硬货币功能，它已成为开发全球几乎所有免费，开放甚至专有软件的基础。

“ 自2005年推出以来，Git已发展成为易于使用的系统，同时保留了其原有的品质。他出奇的快速，高效地处理大型项目，并且拥有出色的非线性开发分支系统，“ Scott Chacon和Ben Straub在他们的书“ Git for a Professional Programmer”中写道。
