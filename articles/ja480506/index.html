4. Check Pointフォレンジックを使用したマルウェア分析。CloudGuard SaaS



Check Point Forensicsの記事シリーズの最後の製品に行きました。今回はクラウド保護についてお話します。クラウドサービス（いわゆるSaaS）を使用しない会社を想像するのは困難です。 Office 365、GSuite、Slack、Dropboxなどここで最も重要なのは、クラウドベースの電子メールとクラウドベースのファイルストレージです。従業員が毎日使用するもの。ただし、クラウドサービスはネットワークの外部に配置されているため、境界はありません。これにより、ユーザーへの攻撃の可能性が大幅に増加します。クラウドアプリケーションのセキュリティオプションは多くありません。以下では、Check Point CloudGuard SaaSソリューションについて説明します。それが何から保護するか、そして最も重要なことは、それが提供するフォレンジックとレポートの内容です。これは、クラウドサービスのセキュリティ監査を実施する場合に役立ちます。

Check Point CloudGuard SaaS


CloudGuard SaaSの動作原理は非常に単純です。このサービスは、APIを介して他のSaaSサービス（office365、GSuite、box、dropboxなど）と統合するクラウドプラットフォームです。



本質的に、CloudGuard SaaSはクラウドサービスとユーザーの間のレイヤーです。すべての手紙やファイルは、ユーザーに届く前にさまざまなCheckPointエンジンによってチェックされます。プラットフォーム自体は、Check Point ThreatCloudおよびSandBlastクラウドサンドボックスと自然に統合されています。さまざまなユーザー認証サービス（Centryfy、okta、Azure ADなど）との統合を構成して、接続デバイスを完全にチェックすることもできます。すべての制御は、直感的なWebインターフェースを介して行われます。



Check Point CloudGuard SaaSの主な機能：

  • ゼロデイ脅威防御
  • フィッシング対策
  • アイデンティティ保護
  • データ漏洩防止
  • SaaSシャドウITディスカバリー
  • 直感的なクラウド管理

これらの機能の詳細については、アレクセイベログラゾフ（チェックポイント社）の優れたウェビナーをご覧ください。

結論


最近、クラウドサービスがどれほど便利で、手頃な価格で、信頼できるかを説明する価値はないと思います。しかし、「クラウド」は「セーフティネット」にとって真の課題です。多くの場合、妥協を模索するか、完全にその使用を放棄する必要があります。 Check Point CloudGuard SaaSは、クラウドインフラストラクチャを制御下に置くための優れたツールです。
もう1つの重要な詳細は、CloudGuard SaaSの統合の容易さです。これは、従来のゲートウェイやサンドボックスを使用するよりもはるかに簡単です。ブラウザで数回クリックするだけで設定できます。この場合、無料試用版（30日）がご利用いただけますクラウドサービスの現在のセキュリティレベルを監査するために、このサービスを使用します。検出モードでは、インフラストラクチャに影響を与えることなく、すべての脅威に関する完全なレポートを受け取ります。CloudGuard SaaSの使用に関するアドバイスだけでなく、試用版ライセンスもリクエストできます。
