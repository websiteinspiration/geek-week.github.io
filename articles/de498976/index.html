.NET SDK-Unterstützung für die .NET Framework-Assemblys


Das .NET SDK verweist jetzt automatisch auf  das   NuGet-Paket Microsoft.NETFramework.ReferenceAssemblies. NuGet-Paket basiert auf dem .NET Framework-Ziel in der Projektdatei. Mit dieser Änderung können Sie .NET Framework-Projekte auf einem Computer erstellen, ohne dass das .NET Framework-Targeting-Paket installiert ist. Diese Verbesserung gilt nur für Targeting-Pakete und berücksichtigt keine anderen Abhängigkeiten, die das Projekt möglicherweise aufweist.
