它包括比较两个扑克手并确定获胜手。

扑克之手用包含2个字符的子字符串（表示单个纸牌）的行表示，并用空格分隔。2H是两个蠕虫，TC是交叉十，等等。

第一个字符表示排名。
所有可能的选项，按优先级排列：2 3 4 5 6 7 8 9 TJQKA。
使用数字标记，所有字母都清晰可见：T是10（T en），J是千斤顶（J ack），Q是皇后（Q ueen），K是国王（K ing），A -A（A ce）。

第二个字符定义西装。
可能的选项：H-蠕虫（H耳），C-十字架（C lubs），S-峰（S垫），D-铃鼓（D钻石）。

例如，“ 2C 5C 3C 4C 6C”这样一条完整的手线-在这种情况下，这是从2到6的十字平齐。

手牌排名与德州扑克相同。

您需要了解其中存在隐藏的困难-使用其他卡来解决抽奖情况，使用成对的面值等。

决断


那么，如何解决这个问题呢？首先，您需要确定一种比较方法来解决排名手的问题，然后，如果排名匹配，则可以通过比较其他卡来确定获胜者。

由于问题的陈述假设两套衣服是等价的，因此我们使用一个简单的对象来表示手的排名：

{ 
   rank: 1,       //     1  9,   ,    
   value: 'ABCDE' //  ,        ,    ASCII-    
}

现在我们可以编写一个简单的函数来比较以这种结构形式表示的两只手：

function compareHands(h1, h2) {
    let d1 = getHandDetails(h1)
    let d2 = getHandDetails(h2)
    if (d1.rank === d2.rank) {
        if (d1.value < d2.value) {
            return ""
        } else if (d1.value > d2.value) {
            return ""
        } else {
            return ""
        }
    }
    return d1.rank < d2.rank ? "" : ""
}

因此，现在我们要做的就是用我们的双手创建一个对象-从这里开始乐趣！

因此，同花顺是等级为1的最强组合（为了不使它复杂化，如果有人进行皇家同花顺，我们认为可以抽签），然后是同花顺四，满屋等。

在这种情况下，&&运算符会被奇妙地使用，如果前括号中的值正确，它将扫描到最后一个值。因此，(flush && direct && 1) 如果flush和direct为true，则返回1，否则返回false。

抽签情况


如果两只手的等级相同，则如果可能，还由纸牌的面值决定赢家。一些规则与此相关。

  • 一对夫妇。面值较高的对将获胜。如果值匹配，则以最高的下一张牌为准。（对于两对，其解决方法类似）。

    例如，如果我们将“ 2H 2D AH KC 3D”牌与“ 4H 4C JC TC 3H”牌进行比较，则从第二张牌中获胜，尽管第一张牌中最高的牌面值为ace。
  • 满屋还是满屋。面值较高的三人组获胜。如果双手的三元组的面额相等，那么我们看一下双向的面额。
  • 比较具有相同等级的任何手牌时，我们的行为类似-决定组合中最高牌的面值。


因此，我们需要先按数量排序，然后再按输出行中的面值排序。请记住，整个五个字符的字符串用于解决具有匹配手位的案例。

    let value = faces.sort(byCountFirst).join("")

function byCountFirst(a, b) {

    return b > a ? -1 : b === a ? 0 : 1

const order = "23456789TJQKA"
function getHandDetails(hand) {
    const cards = hand.split(" ")
    const faces = cards.map(a => String.fromCharCode([77 - order.indexOf(a[0])])).sort()
    const suits = cards.map(a => a[1]).sort()
    const counts = faces.reduce(count, {})
    const duplicates = Object.values(counts).reduce(count, {})
    const flush = suits[0] === suits[4]
    const first = faces[0].charCodeAt(0)
    const straight = faces.every((f, index) => f.charCodeAt(0) - first === index)
    let rank =
        (flush && straight && 1) || /* -,     - -        */
        (duplicates[4] && 2) || /*  -     */
        (duplicates[3] && duplicates[2] && 3) || /* - -  +  */
        (flush && 4) || /*  -      */
        (straight && 5) || /*  -       */
        (duplicates[3] && 6) || /*  -     */
        (duplicates[2] > 1 && 7) || /*   */
        (duplicates[2] && 8) || /* -     */
        9 /*     ,      */

    return { rank, value: faces.sort(byCountFirst).join("") }

    function byCountFirst(a, b) {
        //    -  ,  
        const countDiff = counts[b] - counts[a]

        if(countDiff) return countDiff //   ,  
        return b > a ? -1 : b === a ? 0 : 1
    }

    function count(c, a) {
        c[a] = (c[a] || 0) + 1
        return c
    }
}

function compareHands(h1, h2) {
    let d1 = getHandDetails(h1)
    let d2 = getHandDetails(h2)
    if (d1.rank === d2.rank) {

            return ""
        } else if (d1.value > d2.value) {
            return ""
        } else {
            return ""
        }
    }
    return d1.rank < d2.rank ? "" : ""
}
