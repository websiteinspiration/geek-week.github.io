Ampere Altra-世界上第一个80核ARM处理器



加州公司Ampere在64位架构Ampere Altra上推出了业界首个基于ARM的80核服务器处理器。

几年来，专家预测ARM平台将在数据中心与x86竞争，但这并没有发生。到2019年底，英特尔以95.5％的份额占据主导地位，AMD为4.5％。

但是，SPECrate 2017整数基准测试中的新型ARM处理器显示出比最快的64核AMD EPYC或Cascade Lake系列的高端28核Xeon更高的性能。这是一个严肃的应用程序（尽管基准测试结果有些“扭曲”，请参见下文）。

ARM的主要优势是能效，根据定义，由于架构的原因，x86处理器无法匹敌。 80核Ampere Altra的TDP为45-210瓦，时钟频率为3 GHz。

Ampere认为，每个内核一个线程而不是两个线程可以提高安全性，因为这种设计可以更好地保护单个内核免受第三方通道（如Meltdown和Spectre）的攻击。





该处理器设计用于服务器应用程序，例如数据分析，人工智能，数据库，存储，电信堆栈，边界计算，虚拟主机和云应用程序。特别是对于硬件级别的机器学习应用程序，实现了对数据格式FP16（半精度）和INT8（整数的单字节表示）的支持。还有AES和SHA-256的硬件加速哈希。



芯片是使用7 nm工艺技术在台积电工厂制造的。第一批CPU样品已经发送给潜在客户，并且计划于2020年中期开始量产。

Ampere首席执行官兼前英特尔总裁RenéeJames于2017年10月在破产的应用微电路公司（1979-2017）的基础上创立了Ampere Computing，该公司还设计了ARM服务器处理器。特别是在2011年，她推出了基于ARMv8-A的64位X-Gene平台。

詹姆斯现在将Ampere Computing的执行董事和董事会主席与国家安全电信咨询委员会副主席的职位合并，该委员会为美国总统提供咨询服务。

我不知道新尝试将ARM处理器推向服务器市场的成功程度。

詹姆斯说： “我们发布了市场上内核数量最多的处理器。” “现在，我们将它（进行测试）发送给了业内一些最大的云服务提供商……我想人们会感到惊讶。” [替换以前的技术]总是有新的东西。如果不是来自现有公司，则来自新公司。我认为这是该行业发展的下一阶段，这很有趣。”

过去几年，当AMD和上述的Applied Micro试图生产类似的处理器时，关于64位ARM服务器芯片的讨论很多。但是这些公司都失败了。 AMD关闭了ARM项目，出售了 Applied Micro资产Macom。 2017年，凯雷投资集团从她手中收购了ARM处理器部门。该交易于2019年底完成，詹姆斯接任新公司首席执行官，离开卡莱尔集团首席运营官一职。

Jeff Wittich表示，在测试中，Ampere处理器的结果比AMD最快的EPYC处理器好4％，功耗降低14％。我们谈论的是64核EPYC
7742 处理器，TDP为225 W，成本为6950美元。这是Zen 2微架构上EPYC 2处理器系列中功能最强大的系列，该系列于2019年8月推出。





Wittich还引用了Cascade Lake系列的28核Xeon处理器。 Ampere Altra处理器的性能超过其“ 2.23倍，能源效率为2.11倍”。在此，将指标与28核Xeon Platinum 8280（205 W）进行了比较，并基于一个核计算了能效。

据报道，在整数基准SPECrate 2017中，Ampere Altra处理器显示的结果大于259。这导致表下方的ASUS RS720A-E9（KNPP-D32）服务器系统（2.20 GHz的AMD EPYC 7601）的峰值性能和华硕RS500A-E10（KRPA-U16）服务器系统2.25 GHz的AMD EPYC 7742.

然而，安培应用与AMD自身使用的GCC 8.2相比，AMD使用AMD64编译器软件包编译基准代码的结果为0.85，这是因为AMD C / C ++编译器比ARM的GCC生成的代码更优化。

尽管基准表现出如此“扭曲”，但Ampere Altra在性能和能效方面仍然令人印象深刻。在标准的42U服务器机架中，可以将大约3500个处理器核“打包”到电源为12.5 kW的标准42U服务器机架中，从而从计算每个核的瓦数中节省了成本。





而这仅仅是个开始。 Jeff Wittich说，一年后，另一种代号为Mystique的产品将出现在市场上，其中Ampere将进一步增加内核数量。

Mystique将支持相同的连接器，因此不需要更换主板。下一代Siryn SoC计划于2022年推出。



近年来，我们观察到了几次尝试从不同公司发布服务器端ARM处理器的尝试：Broadcom / Cavium / Marvell，Calixeda，华为，富士通，Phytium，Annapurna /亚马逊和AppliedMicro / Ampere。这些尝试大多数都不成功。但是有迹象表明情况正在改变。在2019年12月，亚马逊推出了配备64核Graviton2 ARM处理器的生产服务器。是与ARM Neoverse N1内核在同一内核上的片上系统。在某些测试中，ARM实例（M6g和M6gd）的性能更好，有时甚至比x86好。

据报道，2019年11月，美国初创公司Nuvia 筹集了5300万美元的风险投资资金。这家初创公司是由三位领先的工程师创立的，他们参与了Apple和Google处理器的创建。他们还承诺将开发与英特尔和AMD竞争的服务器处理器。根据现有信息，Nuvia从头开始设计了一个处理器内核，该处理器内核可以在ARM体系结构的“顶部”构建，但无需获得ARM许可证。

所有这些表明，RISC处理器不仅可以在移动设备中找到应用程序，而且可以在服务器以及台式计算机和笔记本电脑中找到应用程序。顺便说一句，有传言称，未来的Apple MacBook笔记本电脑也将在ARM处理器上问世。



实际上，最新的配备A12X ARM处理器的iPad Pro型号几乎与配备Core i7和Core i9处理器的15英寸MacBook Pro一样好，因此这种升级将变得非常合乎逻辑。
