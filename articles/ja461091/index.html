LED Camelionランプ

2つのシリーズの16個のCamelion電球を購入してテストしました。シリーズ間の違いは、予想ほど明確ではありませんでした。



かつてはシンプルでした：CRI（Ra）80+演色評価数の緑色のボックスにCamelion LEDウルトラランプがあり、すべてが問題なく（リップルが100％のG4およびG9ランプを除く）、白いボックスにBasicPowerエコノミーシリーズがありましたCRI（Ra）70+を使用。緑色のボックスの色が変更され、大きな銘「Meets GOST」がそれらに表示されました。パッケージには、安全要件のみを規制し、技術的パラメーターは規制しないGOST R IEC 62560-2011が示されています。


白と緑のボックスの技術パラメータは、まったく同じように示されています。ランプの品番も揃っています。



白いボックス内のすべてのランプで、1つを除いて、演色評価数Ra 82+が示されています。 2018年1月にリリースされたLED7-A60 / 830 / E27ランプボックスは、Ra 77+を示しています。今後の見通しとして、このランプの実際の演色評価数は80でした。ただし、他の2つは、ボックスに82以上と書かれていますが、残念ながら、72と74と低くなっています。

光束、色温度、演色評価数は、2メートルの積分球とInstrument Systems CAS 140 CT分光計、Robiton PM-2機器での電力消費と力率、Uprtek MK350D機器での脈動を使用して測定されました。光束が公称値の5％以下しか低下しない最小動作電圧は、Lamptest-1デバイス、Stabil Instab 500スタビライザー、およびLantra Suntek TDGC2-0.5を使用して測定されました。測定前に、ランプのパラメータを安定させるために、ランプを30分加熱しました。


すべてのランプにリップルはありませんが、これは非常に良好です。

パルスドライバーのおかげで、すべてのランプは、明るさを低下させることなく、電圧を大幅に下げても動作します（最低電圧103〜137 V）。

さまざまなランプの測定された光束（ランプが与える光の量）は、指定された光束の82〜111％に達しましたが、一連のランプで宣言された光束の間の接続に気づきませんでした。

ほとんどのランプの実際の等価電力（白熱ランプの電力に対応）は、示されている値よりわずかに低くなっています。

残念ながら、白いパッケージの2つのランプ-LED8-G45 / 830 / E14電球とLED8-C35 / 830 / E14キャンドルは、2018年3月にリリースされましたが、これらのランプのパッケージに記載されているにもかかわらず、測定された演色評価数は低くなりました「82+」。

インジケーター付きのランプでは、すべてのランプが機能します（2018年1月に製造されたLED7-A60 / 830 / E27の1つだけが、スイッチがオフになっていると弱く燃えます）。

すべてのランプのパッケージには、0.7を超える力率（PF）が示されています。どうやらこれはロシア政府の政令1356号の難しい要件を満たすために行われます。測定された力率は0.49から0.58でした。奇跡はありません。ただし、家庭で使用する場合は問題ありません。

すべてのランプには2年間の保証が付いています。

記事が公開された後、ブランドの代表者から連絡があり、白い箱に入ったBasicPowerランプは製造されていませんが、残りはまだ販売されています。ミントグリーンパッケージのCamelion LED Ultraランプのみが現在入手可能であり、購入する価値があります。
