誰もが電気自動車に乗るのはいつですか？

1914年1月11日、ヘンリーフォードの声明がニューヨークタイムズに掲載されました。

「1年以内に電気自動車の生産が始まることを願っています。 １年前には話したくないのですが、私の計画についてお話したいと思います。実際、エジソン氏と私は、安くて実用的な電気自動車を作るために数年間働いてきました。これらは実験として作成されたものであり、成功への道のりが明らかであることに満足しています。電気自動車の問題は、これまで、充電せずに長距離を走行できる軽量のバッテリーを作成することでした。エジソン氏はしばらくの間、そのようなバッテリーを実験してきました。」

しかし、何かがうまくいかなかった...


デトロイト・エレクトリックのトーマス・エジソン

この出版物は、私の以前の記事「業界の発展の法則としてのロジスティック関数の調査」の論理的な続きです。



ここで、パラメーターrは指数の指数に立っているため、市場シェアの成長率に影響します。この係数が高いほど、新しいテクノロジーが市場を征服する速度が速くなります。毎年、技術はその利便性のためのより多くの人々に興味深いものになるはずです。、Kの係数はすなわち、新技術の成長の可能性を説明K値が低い場合、テクノロジーは市場全体を捉えることはできませんが、以前のテクノロジーよりも興味深い市場セグメントを取り戻すことができます。

問題の説明は、電気自動車産業の発展を予測できるロジスティック方程式に必要なパラメーターを見つけることです。

  • 「ゼロ年」-世界で販売された車の半分が電気モーターを搭載する年（P0 = 0.5、t = 0）;
  • 電気自動車の市場シェア（r）の成長率。

この場合は、次のように仮定します。

  • 電気自動車は、市場（K = 1）の自動車を内燃機関（ICE）で完全に置き換えます。これは、自動車市場を分割できるラインが見当たらないためです。

    モデルを作成するときに重機や特殊機器の市場は考慮されなかったため、この業界にはまだ電気自動車市場はありません。
  • ここで、「負の時間」（P（t）<0）に住んでいます。関数では、時間（t-t0）の「ゼロ年」に対するオフセットを使用します。

import matplotlib.pyplot as plt
import numpy as np
import math

x = np.linspace(2012, 2019, 8)
y1 = np.array([60936407, 63429200, 65708230, 66314155, 69464432, 70694834, 68690468,  64341693])
y2 = np.array([52605, 97507, 320713, 550297, 777495, 1227117, 2018247,  1940147])
y = y2/y1

k=1
p0=0.5

x0=2028   
r=0.37 
y1=k*p0*math.e**(r*(x-x0))/(k+p0*(math.e**(r*(x-x0))-1))
#      2012  2019 
fig, ax = plt.subplots(figsize=(30, 20), facecolor="#f5f5f5")

fig, ax = plt.subplots(figsize=(30, 20), facecolor="#f5f5f5")

調査結果


ICEによるエンジン開発の歴史の説明と同じ論理に従って、利用可能な統計を使用して電気自動車産業の発展を予測しようとしました。

得られた結果は、2030年までに世界で販売された車の半分が電気モーターを搭載し、2040年までにICEを搭載した乗用車が過去のものになることを示しています。

もちろん、2030年以降は、2030年以前に購入したガソリン車を運転する人もいますが、次の購入は電気自動車になることがわかります。
電気自動車の成長率は、内燃エンジン搭載車の成長率の4倍です。これは、新しいテクノロジーが私たちの生活にますます浸透し、私たちの日常生活（ここでは携帯電話を思い出します）で当たり前になってきていることを示しています。

今後数年間で、エジソンが解決できなかった問題を解決する必要があります-充電ステーション間の走行距離を増加させる十分に容量のあるバッテリー。

既存のガソリンスタンドのネットワークと同等の充電スタンドのネットワークを作成するには、大都市や高速道路に沿った既存の電気ネットワークを近代化する必要があります。

また、ジェバンズのパラドックスは電気自動車の売上高の伸びを妨げますが、石炭需要の減少の中で石油にも影響を与えました。
