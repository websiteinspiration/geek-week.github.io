关于每个平台的几句话


Skype的



www：skype.com

也许他可以被称为大众消费者视频通信的先驱。该服务通过客户端运行，该客户端已下载并安装在整个平台列表上。网络版本也可用。

优点

其中一个好处是可以拨打世界各地的固定电话和移动号码，但并非免费。此外，即使在俄罗斯，您也可以用手机号代替CALLER ID。在某些国家/地区，可以通过特殊号码从手机打入Skype。 Skype

的缺点

更多地用于小型会议或个人咨询。在Skype上举行大型会议非常罕见。

Skype不允许连续通信超过四个小时。此外，群组视讯通话的数量限制为每月100小时（每天10小时）-这是开发人员设定的安全性限制。

人们经常批评该工具对网络问题的反应比同一个Google Hangouts或Zoom差。另外，移动客户端对设备的资源要求很高。

值得注意的是，微软拥有Skype for Business-并未作为Skype家族中的独立产品销售，该产品使用了最好的功能。您无法使用常规Skype连接到Skype for Business中的会议，但可以在浏览器中使用Web客户端。Skype for Business具有广播会议的功能，该功能允许多达1万人演示正在发生的事情（我们在表中已对此进行了显示）。但是我们并没有开始单独拆卸该产品，因为现在它已被Microsoft Teams取代，Microsoft Teams是团队的工作区。

Google视讯聚会



www：hangouts.google.com

Google Hangouts是公司于2013年推出的多项服务的继承人。

优点该

工具非常方便，因为它与日历和Gmail绑定在一起。也就是说，通过Google日历进行约会，您会自动生成一个指向Google Hangouts房间的链接。

Google Hangouts插件可用于通过流行的Slack进行群组视频通话（我们的通讯录中未包含Messenger，因为它更侧重于团队内部的交流）。

Google环聊不需要其他注册或激活：如果您拥有Google帐户，请考虑在环聊中也进行了注册。在平台之间切换以及同步聊天和视频通话历史记录没有问题。

缺点

该服务在不稳定的环境中不能很好地处理视频会议。免费版本最多支持10位参与者-但是，本地会议可以自动广播到YouTube。

在该工具的基础上，2017年立即为企业用户提供了两项服务：环聊聚会和环聊。在2018年底，宣布将关闭Google Hangouts，并将用户转移到新服务中。没错，没有具体日期。考虑到全球形势，谷歌不太可能在未来几个月内这样做。相反，由于存在冠状病毒，该公司从3月初开始将视频会议的允许参加人数增加到250人。此限制将一直持续到2020年7月1日。

值得注意的是，Google Hangouts能够通过Pexip网关与其他视频会议工具（例如Skype for Business）进行交互。这是视频市场的重要一步，每个开发人员都使用自己的协议和编解码器，无法与竞争服务进行交互。但是建立这种交互是另一回事。

放大



WWW：zoom.us

在几天之内，变焦已经成为小型研讨会和讨论的事实标准。在新闻中，注意到用户数量出现了雪崩般的增长，其中包括由于教育领域而几乎完全消失的情况。

在较大的活动中使用相同的工具来组织听众在演讲后或演讲后与演讲者之间的互动（例如，在最近翻译的Moscow Python Conf ++ 2020在线格式的框架中以这种方式使用Zoom）。

优点

Zoom专注于相对较大的会议。与竞争对手不同，它具有“举手”功能和各种组织器工具，可让您控制视频通信。将事件记录在本地系统和云中也很重要。除了竞争对手之外，Zoom消耗的流量也相对较少。

对于付费资费，Zoom具有用于实现其他功能的API。通常，该工具具有许多选项，并且可以仅为所使用的功能付费。在列表中，您甚至可以找到报告以及与Outlook或Lync的交互。针对特定受众的网络研讨会组织者有单独的软件包。组织者仅需支付此费用-参与者可以免费连接。

缺点

它们是有条件的。例如，会议持续时间不能超过24小时。其他限制取决于所选的关税计划。

免费使用“缩放”功能，最多可以容纳100人的视频通话，通话时间不超过40分钟。一对一时间限制不适用于通信。

思科Webex会议



www：webex.com/video-conferencing

来自思科视频会议解决方案“怪物”的企业级工具。对于公司而言，该解决方案的优势在于，它可以与硬件视频会议解决方案以及可以较早集成的专用设备相连接。对于最近才访问过远程站点的最终用户，此功能并不是特别有趣，但是我们无法提及该解决方案。

正式地，该工具仅在俄罗斯通过合作伙伴提供-如果没有VPN，该工具将无法注册和下载客户端。关税政策还取决于合作伙伴。

上课



www：getcourse.ru

该平台是为在线教育和该市场的收入而设计的。它使您可以根据特定的时间表（在可设置发送预录课程的频率的情况下）自动创建实时课程或在录制中提供给订户的课程和课程。

服务的定价取决于系统中授权的已连接用户（课程侦听器）的数量。文件存储需要单独付费。

优势

除了视频课程本身之外，该平台还提供用于完成任务，监控和多级用户管理和筹款的工具，以及与讲师与学生交流的移动平台-Chatium的工具。

不幸的是，该工具没有免费的关税，但是它提供了一个试用版，最多可容纳1000个用户。以收费的价格，可以为广大观众（以我们的评论术语-广播）和网络研讨会提供讲座。参加人数是有限制的（最多500人），而每个人都可以发言。

YouTube广播




在上述其他网站的背景下，YouTube着重于在不与演讲者互动的情况下广播视频流而脱颖而出。提出问题的唯一方法是将其写在评论中，并希望演讲者能看到。

该广播不适用于任何帐户-仅适用于已确认电话号码的用户。

不受所有通讯限制，YouTube广播的简洁性给人留下了深刻的印象。视频可以通过计算机的网络摄像头或应用​​程序中的移动设备进行广播。YouTube Live支持高清视频，但是由于在大流行期间流量增加，该服务对SD施加了强制限制。

举行过数十场网络研讨会和在线会议的人怎么说？


我们设法从我们的Leader-ID网站收集了十个人的意见，该网站的在线活动计数器已超过100。十分之八的人会定期使用“缩放”功能，请注意很多设置和选项。其余部分，由于具体情况，可以与网络研讨会平台（Webinar.ru，Yandex服务）一起使用。

选择工具时，您需要了解在线格式具有其自身的特征。以下是那些长期举办活动的人所指出的：

  • 对于事件组织者而言，在线交流要比离线交流更为困难，因为在视觉上“读取”的信息有限。
  • 由于缺乏听众的反馈，一些演讲者避免在线发言。
  • 在在线活动中，留住观众更加困难，因为周围的干扰太多了。
