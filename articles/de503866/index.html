Veröffentlichte Roadmap-Entwicklung Zextras Suite für 2020

Früher, wir sprachen über die Innovationen von Zextras Suite Pro und Team Pro zu Beginn des Jahres 2020. Zextras hat die Roadmap für die Entwicklung neuer Funktionen der  Zextras Suite für 2020 veröffentlicht .


Juni - Team: Neue Funktionen (VERBESSERT)

  1. Auf eine Nachricht antworten (Antwort)
  2. Chatverlauf löschen
  3. Nachricht bearbeiten
  4. Nachricht löschen

August - Team:

Videoserver (NEU) Mit dem in Zextras Team integrierten Videoserver können Sie eine Reihe neuer Funktionen einführen, die direkt über die Administratorkonsole gesteuert werden können, und die Produktivität des Dienstes steigern. Audio- und Videostreams werden komprimiert, optimiert und kombiniert, um für jedes Gerät einen eigenen Stream zu erhalten. Darüber hinaus wählt der Server automatisch den geeigneten Codec und die entsprechende Bitrate für jedes Gerät aus, um eine maximale Kommunikationsqualität sicherzustellen.

August - Backup auf externem Endpunkt (NEU)

In Zextras Backup können Sie Ihr System auf einem lokalen, Hybrid- oder Cloud S3-kompatiblen Speicher oder auf einem externen Speicher von BLOB-Elementen wie NFS sichern. Gleichzeitig ermöglicht ein lokaler Metadaten-Cache die Beibehaltung der ursprünglichen Leistung, obwohl sie selbst im Falle einer Notfallwiederherstellung auch auf einem externen Endpunkt gespeichert werden.

September - Zextras Authentifizierung und Anmeldung (NEU) Die

Open-Source-Edition der Zimbra Collaboration Suite wird um eine Zwei- Faktor-Authentifizierung, ein Verbund-SAML-Authentifizierungs- und Autorisierungssystem sowie mehrere mobile Kennwörter erweitert. Eine aktualisierte Anmeldeseite mit der Möglichkeit der Anpassung wird ebenfalls angezeigt.

September - Team-Datenschutzeinstellungen (NEU)

Benutzer und Administratoren können auswählen, ob Informationen zu ihrer Anwesenheit angezeigt werden sollen. Insbesondere können Informationen zum Lesen von Nachrichten, zum Benutzerstatus usw. ausgeblendet werden. September - QR-Authentifizierung für mobile Apps (NEU) Mit dem QR-Login-System können Sie mobile Anwendungen verwenden, ohne ein Kennwort über ein Cross-Authentifizierungssystem eingeben zu müssen, und deren Registrierung und Autorisierung verwalten. September - Zextras-Adressbuch für MS Outlook / MAC Mail / Thunderbird (NEU) Verbinden Sie Microsoft Outlook, Thunderbird und Mac Mail mit einer zentralen Zimbra GAL und Ihren Adressbüchern.










Das Adressbuch kann in Verbindung mit IMAP / POP oder EAS verwendet werden. Microsoft Outlook kann EAS und das Adressbuch verwenden, um eine vollständige Interaktion mit Outlook ohne Plug-In im Client bereitzustellen!

Oktober - Zextrasconfig Infrastructure Manager (NEU)

Alle Zextras-Komponenten in der Infrastruktur können über einen speziellen Zextrasconfig-Manager verwaltet werden. Mit diesem Manager können Zimbra-Administratoren Zextras-Komponenten auf verschiedenen Infrastrukturknoten verwalten, einschließlich des aktualisierten integrierten Zextras Docs-Servers und des integrierten Zextras Team-Videoservers.

Oktober - Admin-Oberfläche und Benutzereinstellungen (REFACTORING)

Die neue Administrationsoberfläche für Zextras-Komponenten ist in React geschrieben. Beginnend mit den Zextras-Komponenten, die weiterhin in der vertrauten Administrationsoberfläche dargestellt werden, werden wir alle Optionen auf die neue Technologie übertragen, ohne sie von der vertrauten Oberfläche zu entfernen. Ein vollständiger Übergang wird in Zukunft erfolgen.

Oktober - Neues Installationsprogramm mit nativem Linux-

Paketsystem (NEU) Das neue Zextras-Paket wird über Repositorys mit nativer Linux-Unterstützung verteilt. Bei der Installation und Aktualisierung des Pakets können Systembefehle wie "apt-get install / update" verwendet werden.

November - Docs Server (NEU)

Der Zextras Docs-Server wird Teil der Infrastruktur, die über den Centralized Infrastructure Management Manager von Zextrasconfig verwaltet wird. Die Installation und Aktualisierung des Servers auf einzelnen Knoten ist nicht mehr erforderlich.

November - Team: Neue Funktionen des Meetings (NEU)

Dank des neuen Videoservers werden im Zextras-Team eine Reihe neuer Funktionen angezeigt. Unter ihnen:

  • Moderator-Modus
  • Die Möglichkeit, den Sound für einen bestimmten Benutzer oder sofort für alle stummzuschalten
  • Möglichkeit, den Webinar-Modus und den Videokonferenzmodus zu aktivieren
  • Das Ereignisprotokoll
  • Chat-Anfrage
  • Walkie-Talkie-Modus
  • Hangout-Aufnahmen können auf Zextras Drive hochgeladen werden

November - Hochverfügbarkeit (NEU) Die

Möglichkeit, Benutzer, Gruppen, Domänen und sogar ganze Mailserver zwischen verschiedenen Infrastrukturknoten zu synchronisieren. Möglichkeit zum Konfigurieren der Synchronisation mit einem Ersatzknoten oder einem anderen aktiven Knoten im Cluster. Das System ist auf eine Verfügbarkeit von 99,99% pro Monat ausgelegt.

November - Persönliches externes Konto einrichten (VERBESSERT)

Möglichkeit zum Verbinden von Dropbox- / Gdrive- / nextCloud- / OwnCloud-Konten

Dezember - Team: Sofortbesprechung (VERBESSERT)

Möglichkeit zum Erstellen einer sofortigen Besprechung und zum Bereitstellen des Zugriffs über permanente und temporäre Links.

Dezember - E-Mail-Signierung und Verschlüsselung (S / Mime-PGP) (NEU)

Die End-to-End-Kryptografie wird im Webclient implementiert. Das System kann die zum Entschlüsseln gesendeten E-Mails verschlüsseln und / oder signieren und die Signatur der empfangenen E-Mails überprüfen.

Viele dieser Funktionen waren bisher nur in der kommerziellen Version von Zimbra verfügbar. Bereits in diesem Jahr können Benutzer der Open-Source-Edition der Zimbra Collaboration Suite diese und andere Funktionen beim Kauf der Zextras Suite nutzen. In Zukunft werden wir ausführlich über die neuen Funktionen der Zextras Suite sprechen.
