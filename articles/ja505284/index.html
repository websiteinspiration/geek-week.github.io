wxpython 可用性はさらに手頃な価格です。

みなさん、こんにちは。

多くのアプリケーションがますますWEBになっていますが、この記事ではアプリケーションソフトウェアの話題を取り上げたいと思います。

より具体的には、WXウィジェットのラッパーである素晴らしいwxpython GUIライブラリについて。

非常に便利で、習得も簡単で、複雑なグラフィックインターフェイスを構築するための多くのコンポーネントがあります。

そして最も重要なこと-それはアクセシビリティの面で最も手頃な価格です。

また、ほとんどのコンポーネントはそのまま使用できますが、可用性の低いコンポーネントもあります。

しかし、少し前までは、開発者が新しいコンポーネントとすでに作成された以前のコンポーネントの両方の可用性をさらに向上させるのに役立つ新しいツールが登場しました。

wx.Accessible


実際、彼は以前のバージョン4.0.4に登場しました。ただし、このバージョンでは、wxpython開発者が基本的なアクセシビリティを破りました。これは、アクセシビリティへの大きな回帰でした。

この問題は、たとえば、次の場所で確認できます。wx.Accessible：コントロールがIAccessibleオブジェクトを実装しないことを選択した場合、またはwx.Controlコンストラクターの名前キーワードが指定されていない場合にスクリーンリーダーに表示される一般的なラベル。

しかし、最後の大きなアップデート4.1.0では、この問題が修正され、新しいツールを完全に確認できるようになりました。

その助けを借りて、初期のアクセス不可能なコンポーネントを利用可能にし、既存の非常にアクセス可能なコンポーネントの機能を拡張することができます。そのため、グラフィック表現を変更せずにスクリーンリーダーの拡張テキスト情報を出力する可能性を探していたときに、それを見つけました。

アクセシビリティのトピックは最も関連性が高くなく、このコンポーネントは比較的新しいため、インターネットの英語圏でも、ロシア語圏でも、わかりやすい例を見つけられませんでした。

私はこの分野に少しでも貢献できるよう努力します。

わかりやすくするために、それを使用する例を検討してください。

wx.Frameウィンドウを作成するための追加の標準コードは提供しません。これらの例は、あらゆる好みに対応できます。したがって、wx.Accessibleの使用に関する革新のみを簡単に引用します。
この例では、adv.CalendarCtrlウィジェットを取ります。

このウィジェットでフォーカスを取得すると、キーボードの矢印を使用して日付をナビゲートできますが、同時に音声はまったく表示されず、ユーザーは現在の日付を理解できません。

さらに、ウィジェットには他の問題があります。たとえば、キーボードを使用してそこからフォーカスを変更して次のインターフェイスコンポーネントに移動できないなどです（ps：ウィジェットからそのようなウィジェットからウィンドウにフォーカスを戻すオプションを知っている場合は、コメントを書き込んでください）。、しかし今の仕事はカレンダーウィジェットを完全にアクセス可能にすることではなく、wx.Accessibleで動作を示すことです。

したがって、プログラムでは、このウィジェットを初期化しています。

self.date = wx.adv.CalendarCtrl(self.panel, wx.ID_ANY,
    wx.DateTime.Today(),
    style=wx.adv.CAL_MONDAY_FIRST |
          wx.adv.CAL_SEQUENTIAL_MONTH_SELECTION)

以下を追加する必要があります：

self.date.SetAccessible(AdvCalendarAccessible(self.date))

AdvCalendarAccessibleクラスは、wx.Accessibleから継承された完全に新しいクラスで、以下に示されています。

スクリーンリーダーに必要なメソッドをオーバーライドします。
そこで、新しいクラスを作成します。

class AdvCalendarAccessible(wx.Accessible):

    def __init__(self, window):
        self.window = window
        super().__init__(window)

    def GetName(self, index):
        date = self.window.GetDate()
        return (wx.ACC_OK, date.Format('%d.%m.%Y'))

    def GetDescription(self, index):
        text = ''
        date = self.window.GetDate()
        attr = self.window.GetAttr(date.day)
        if attr is not None:
            text = attr.GetBackgroundColour().GetAsString()
        return (wx.ACC_OK, text)

このオプションでは、スクリーンリーダーからの要求に応じて、スクリーンリーダーに日付を返します。もちろん、プログラムによってマークされている場合は、選択した日付の背景色も返します。


以上です。このウィジェットでキーボードの矢印を使用してナビゲートすると、ユーザーはすべての日付を聞くことができます。

これらの方法以外にも、他の方法があります。たとえば、コンポーネントまたはその文字列プロンプトの値を音声で伝えるために。

完全なドキュメントはwx.Accessibleにあります。

誰かがこの情報が役に立つと思います。そうすれば、あなたのプログラムは、さらに一歩アクセスしやすくなります。
