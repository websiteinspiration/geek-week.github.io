最近、The Coding Trainチャンネルからのインスピレーションを受けて、10分の課題の1つに参加することにしました。この課題では、宇宙を高速で飛んでいるような錯覚を作成する必要がありました。

このプロジェクトを実装するために、私はよく知られたp5.jsを選択しました。これは、アルゴリズムでアートを作成するために設計されたJavaScriptのライブラリです。アドビの標準グラフィックパッケージを使用できないのはなぜですか？

まず、コードでこれを行うのは美しいです。
第2に、コードにより、いくつかのパラメーターを変更するだけで別の結果を得ることができ、実験と反復の速度が大幅に向上します。
そして3番目に、このコードは、JSのスクリプトの形で任意のWebプロジェクトに簡単に統合できます。

プロジェクトを開始するには、最も簡単な方法は、editor.p5js.orgのオンラインエディターにアクセスすることです。

まず、スターを作成する必要があります。肉眼ではまったく星が見えない宇宙空間を見つけることは技術的には難しくありませんが、多数の星が必要であることは明らかですが、この効果についての私たちのアイデアは、SF映画を使用して作成されました。有効性のために精度は無視できると思います。そう。ランダムスタージェネレーターを作成する必要があります。

最初に、星を格納する最上部に配列を作成します。

Star（）と呼ばれる新しい関数を作成してみましょう。x、y、変数zに沿った位置の4つの主要なパラメーターが必要です。初期点のランダムな発生を作成するには、random（）関数を使用し、パラメーターとして画面の幅と高さの境界を設定できます。

random（）関数は、0、1、または2つの引数で機能します。引数なし-0から1までの乱数を返します。1は-0から引数まで、2は最初のものから2番目までです。


星を作るための「ファクトリー」を構築するので、すぐにthisキーワードを導入しましょう。thisキーワードは、この関数（またはメソッド）を呼び出すオブジェクトの情報にアクセスしてアクセスします。

function Star() {
  this.x = random(-width, width);
  this.y = random(-height, height);
  this.z = width;

次に、さまざまな軌道とサイズを持つように、星の形を作成する必要があります。これは、最初は画面の幅（width）と同じ変数zを使用して行いますが、1未満になるまで各フレームで25ずつ減少し、その後、サイクルが繰り返されます。

中心から端までの星の美しい動きを得るために、位置がどのように変化するか、つまり変数xとyを見てみましょう。各星は架空の光線に現れ、それに沿って画面の端に移動します。

開始点が座標200,200であると想像してください。これら両方の指標を徐々に下方に変化させる必要があります

zは常に画面の幅（600など）に等しく、xおよびyの範囲は-600〜600であるため、開始時の変数xおよびyは常にz以下になります。これは、xまたはyをzで除算するだけで、値を0から1の管理しやすい値に変換できることを示唆しています。

幅= 600が私たちの宇宙であると想像してください。x = -300、y = -300。この場合、開始点は-300/600 = -0.5になります。4フレーム後、zが4 * 25 = 100減少すると、座標はすでに-300 / 500 = -0.6になります。つまり、座標は次のようになります。画面の中心（座標0,0）から左上に、zが1未満になるまでLINEAR。

それらが正の場合、すべてがまったく同じように発生し、下に移動し、右に移動します。

それらが異なる場合、オプションは下と左（正のyと負のxの場合）または上と右（負のyと正のxの場合）です。

関数ellipseは4つのパラメーターを取ります。xに沿った位置、yに沿った位置、xに沿った直径、およびyに沿った直径であり、それに基づいて楕円が作成されます。

それだけです、私たちは飛んだが、2つの問題があります。最初のもの-デフォルトでは座標の基準点は中央ではなく左上隅にあるため、最初は負の座標値を持たない星の部分のみが表示されます。残りは見えません。



この問題は簡単に解決できます。移動コマンド（幅/ 2、高さ/ 2）で中心を移動するだけです。



2つ目の問題は、観測者に近づいたときに星のサイズが変化しないことです。潜在意識のレベルでは、何かが正しくないと感じています。

この問題を解決するために、楕円に戻り、別の変数rを作成します。これもzに依存し、楕円の3番目と4番目のパラメーターを決定します。これらのパラメーターは、水平および垂直の直径に関与します。最終的なコードは次のとおりです。

let stars = [];
let speed = 25;

function setup() {
  createCanvas(600, 600);
  for (let i = 0; i < 800; i++) {
    stars[i] = new Star();
  }
}

function draw() {
  background("black");
  translate(width / 2, height / 2);
  for (let i = 0; i < stars.length; i++) {
    stars[i].update();
    stars[i].show();
  }
}

function Star() {
  this.x = random(-width, width);
  this.y = random(-height, height);
  this.z = random(width);
 
  this.update = function() {
    this.z = this.z - speed;
    if (this.z < 1) {
      this.x = random(-width, width);
      this.y = random(-height, height);
      this.z = width;
    }
  };

  this.show = function() {
    fill("white");
    var sx = map(this.x / this.z, 0, 1, 0, width);
    var sy = map(this.y / this.z, 0, 1, 0, height);
    var r = map(this.z, 0, width, 10, 0);
    ellipse(sx, sy, r, r);
  };
}
