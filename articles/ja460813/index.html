ボロノイ図を使用してAIを制御する方法


最も安全なルートはどこですか。最も多くの敵がいる場所と、最も近い救急箱はどこですか これらの頻繁に遭遇する空間関係の問題はすべて、「ボロノイ図」と呼ばれる数学的パーティションを使用して効果的に解決できます。この投稿では、ゲームカードを分析し、人工知能のリアリズムと成功を保証する情報を受け取る方法を学びます。


空間関係


空間関係とは、空間内のあるオブジェクトが別のオブジェクトとどのように関連しているかを説明する情報です。例：それらの間の距離、それらの各スペースによってカバーされる領域、およびこれらの領域の交点、1つの領域にあるそのようなオブジェクトの数。

このような関係はビデオゲームで常に使用されており、プレイヤー自身だけでなく、非常に役立つAI情報を提供できます。


ボロノイは答えを持っています


ボロノイ線図は、近接した点またはそれらの最近傍点間の空間関係を記述します。これは、ポイントまたはロケーションから取得された接続されたポリゴンのセットです。ボロノイの「エリア」の各線は、2点間の中央にあります。

ご覧のとおり、各線は2点間の真ん中にあり、それらすべてが中央で接続されています。シーンにいくつかのポイントを追加して、何が起こるかを確認します。


写真がもっと面白くなりました！私たちはすでに現実の領域を持っています。

それぞれの領域から何がわかりますか？エリア内にいることは、エリア内にもある1つのポイントに最も近い場所にあることが保証されていることはわかっています。これは、近くにあるものについて多くを教えてくれます。これは、ボロノイ図の基本的な空間関係です。


ボロノイを裏返しにする：Delaunay三角形分割


ボロノイ図の反対側のシステムは、ドローネ三角形分割と呼ばれます。この図は、各点から最も近い隣接点までの線で構成され、各線はそれが交差するボロノイエッジに垂直です。これは次のようになります。


白はドロネー線をマークした。 Delaunayの各線は、1つだけのボロノイエッジに対応します。最初はいくつかのエッジを横切っているように見えますが、よく見るとそうではないことがわかります。


図では、緑のドロネー線がボロノイのピンクのリブに対応しています。ピンクのリブがさらに進んで、それらが交差していることがわかります。

Delaunay三角形分割のおかげで、ポリゴンの代わりに多くの三角形ができていることがわかります。エリアをレンダリング可能な三角形に分割したので、これは非常に便利です。この技法は、図形のテッセレーションまたは三角形分割に使用できます。いいね！

さらに、これは、あるポイントから別のポイントに移動したい場合に備えて、複数のポイントからグラフを作成するための優れた方法です。たとえば、ドットは都市を示す場合があります。


ボロノイデータ構造


ボロノイ図がどのように見えるかはすでに知っています。次に、そのデータ構造がどのように見えるかを見てみましょう。まず、ボロノイ図の基礎となる点を保存する必要があります。

class VoronoiPoint {
    float x
    float y
    VoronoiRegion* region
}

それぞれVoronoiPointに場所(x, y)と、それが置かれているエリアへのリンクがあります。

次に説明する必要がありますVoronoiRegion：

class VoronoiRegion {
    VoronoiPoint* point
    Edge *edges[] // our list of edges
}

領域には、その領域へのリンクと、それVoronoiPointを制限するエッジのリストが格納されますVoronoiEdges。

それがどのように見えるか見てみましょうVoronoiEdges：

class VoronoiEdge {
    VoronoiPoint* pointA
    VoronoiPoint* pointB
    float distance // distance between point A and point B
    float x1, z1, x2, z2 // to visualize start & end of the edge
}

エッジは、エッジを定義する2つのポイントとそれらの間の距離を認識しています。視覚的に表示するため、およびポリゴン領域の形状を構築するために、エッジの始点と終点を保存する必要があります。

これですべてです。この情報を使用して、ボロノイ図を簡単に作成できます。以下では、ボロノイ図がどのように生成されるかを学習します。しかし、今のところ、このデータの使用方法の例をいくつか見てみましょう。


最寄りの薬箱を探す


もう一度、ボロノイ図でポイントを確認します。


各ポイントが応急処置キットを示している場合、最も近い場所をすばやく特定できますが、まず、自分がいるエリアを特定する必要があります。ボロノイ図は、領域を定義する効果的な方法を提供しませんが、検索を高速化するために、各領域へのリンクを四分木またはR ツリーに格納できます。そして、その地域を学んだので、私たちはその隣人とその隣人の隣人を認識することができます。

たとえば、お住まいの地域に救急箱がなくなった場合は、最も近い別の場所への経路を見つける必要があります。上記のデータ構造と疑似コードから、領域を知ることで、そのエッジを認識できることがわかります。そして、これらの肋骨の助けを借りて、私たちは隣人を得ることができます。私たちは最も近い隣人を連れて、その中に救急箱があるかどうかを確認します。

ここでDelaunay三角形分割を適用することもできます。救急キット間の線で構成されています。次に、A *パス検索アルゴリズムを使用して回避し、最も近い救急箱を見つけることができます。


安全なルートを検索


すべての救急箱を敵の監視塔に交換してください。引っ掛からないように、それらの間の最も安全なルートを見つける必要があります。ビデオゲームでのグラフ走査の標準的な方法は、A *アルゴリズムを使用することです。ボロノイ図はグラフであるため、検索の実装は非常に簡単です。必要なのは、一般的なグラフ構造をサポートするA *アルゴリズムだけです。事前に計画し、それが将来あなたを助けるでしょう。

グラフを準備したら、各エッジに重みを割り当てる必要があります。私たちにとって、重みの値はこれらの監視塔までの距離であり、データ構造から直接取得できます。誰もVoronoiEdgeが2点間の距離をすでに知っています。通常、エッジA *の値は小さいほど良いですが、この場合はタワーまでの距離を示すため、値は大きくなります。

厳密に設定されたアイテムセットを検索する


地面に座っているたくさんのアザラシのために、飛行機からキャットニップ小包を落とす必要があるとします。最も多くの猫がそれを使用できるように、それを落とすのに最適な場所はどこですか？これは非常にコストがかかる可能性があります。しかし、幸いにも、Delaunay三角形分割を使用して合理的な仮定を行うことができます。

ヒント： Delaunay三角形分割は、ボロノイ図の単なる逆であることを忘れないでください。これは、各ボロノイ点をエッジのリストから取得した隣接点に接続することによって形成されます。

この三角形のコレクションを使用すると、三角形のそれぞれがカバーする領域を探索できます。面積が最小の三角形が見つかった場合、3つの最も近い点、つまり猫があります。これは、表面上で最も密度の高い平均クラスターではない可能性がありますが、適切な仮定になります。ミントを使用していくつかの区画を破棄できる場合は、すでに選択されている三角形にマークを付け、次第に大きくなるサイズに移動します。

このような地域の指定は、ドローネ三角形分割の外接円とも呼ばれます。各円は、三角形の点に収まる最大の円です。これは、ボロノイ図の外接円の画像です。


円の正確な中心を使用して、キャットニップが出荷されるエリアの中心を決定できます。実際、特に三角形の2つの点が互いに非常に近く、3番目の点が遠い場合、円の半径は、三角形の領域の代わりに折りたたむのに最適な三角形を決定するためのより適切な方法です。次に、小さな領域を持つ非常に鋭い三角形を取得しますが、それを定義する点は実際には非常に離れています。


ボロノイ図の実装


ボロノイ図を生成する方法はいくつかあり、使用する方法の選択は、データを受信する時間によって異なります。

フォーチュンアルゴリズム


最速の方法はFortuneのアルゴリズムと呼ばれます。これはO(n log(n))、グラフの生成に使用されるすべてのポイントが、生成の開始時にわかっているために実行され、必要です。後で新しいポイントを追加する場合は、グラフ全体を再生成する必要があります。ポイントが少ない場合、問題は発生しない可能性がありますが、10万ポイントある場合、これには長い時間がかかる可能性があります。

これを理解するために、各ポイントが別のボールと衝突するまで膨張する膨張可能なバルーンを含んでいると想像してください。このアイデアを2D平面上で拡張する円に転送できます。別のボールを前方にして、各点に45度の傾斜角を持つ逆円錐を配置し、無限大に増やします。次に、同じく45度の線の形の掃引線を想像してください。これは、コーンと衝突するまでスライドします。平面と円錐は同じ角度で配置されているため、交差すると放物線を形成します。


成長すると、コーンは遅かれ早かれ1つ以上の他のコーンと交差します。円錐または円の交点を見ると、ボロノイエッジの直線が得られます。図では、赤い線は円錐の交差を示しています。コーンがさらに（垂直方向に無限大まで）成長すると、赤い線は伸び続けます。

飛行機はステージ上を動き続けます。遭遇する各点について、すでに放物線を持つ掃引線上の隣接する点を調べ、その点で新しい放物線を開始します。この新しい放物線が以前に重ね合わされたものと重ならないようになるまで、彼女は先へ進み、成長し続けます。その後、この以前の放物線は閉じます。これは、ボロノイの3点の線が交わる点です。
