CSI Linux：サイバー調査およびOSINT用のLinuxディストリビューション


今年の初めに、サイバー調査とOSINT用の別のLinuxディストリビューションがCSI Linux Investigatorという名前でリリースされました。

簡単な説明


CSI Linux Investigatorは、Ubuntuオペレーティングシステムに基づくLinuxディストリビューションのアセンブリであり、事前にインストールされた専用のソフトウェアパッケージを備えています。アセンブリはOVAファイルとして配布され、Oracle VM Virtual Boxに簡単にインポートできます。

ハードウェアおよびソフトウェアの最小要件
-8 GB RAM
-70 GBの空きディスク容量
-4コアプロセッサ
-インストールされたOracle VM Virtual Box

このLinuxディストリビューションには、次の問題を解決するために必要なソフトウェアが含まれています。

  • OSINT
  • デジタルフォレンジック
  • インシデント対応
  • マルウェア分析

公式サイトから 配信リンクをダウンロードできます。配布キットの使用に関する概要ガイドとマニュアルもあります。

個人的な意見と役立つリンク


CSI Linux Investigatorに関するレビュー記事を個人的な意見で完成させ、このディストリビューションでの作業に関するいくつかのヒントを提供したいと思います。

CSI Linux Investigatorの構築は、その種の最初のものではなく、長所と短所があります。
このディストリビューションでは、仮想マシンを3つのコンポーネント（ディストリビューション自体、TORゲートウェイ、SIEMアセンブリ）に分離することが好きでした。

アセンブリをソフトウェアで埋めるという観点から見ると、意見は2つあります。一方で、必要なものはすべてありますが、一方で、配布のサイズに悪影響を及ぼす不要なソフトウェアがたくさんあります。
