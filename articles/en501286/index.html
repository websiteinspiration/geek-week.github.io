2. Yes or no?

Easygui has a ynbox function (YesNoBox). This function allows the user to select his further actions. For example, ‚ÄúWould you like to continue?‚Äù and choosing an option from those proposed. The standard "Yes" and "No", but the functionality of the other buttons you have to do yourself (in the parameters, only these "default_choice" and "cancel_choice"). You can bind them to the keys and make several selections.

Let's set the variables:

title = "!"

Well, here is what ‚ÄúYes‚Äù means, and what ‚ÄúNo‚Äù means, we will write it ourselves.
