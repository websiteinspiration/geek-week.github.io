Vite - memimpin pengembangan tanpa bundel di Vue

Hai, Habrovsk. Pada bulan Juni, Otus meluncurkan Developer Vue.js saja . Untuk mengantisipasi dimulainya kursus, kami secara tradisional menyiapkan terjemahan materi yang menarik.


Suatu hari, Ewan Yu (pencipta Vue.js) memutuskan untuk tidak tidur di malam hari dan akhirnya menciptakan Vite .



Vite memungkinkan Anda membuat aplikasi di Vue dengan komponen file tunggal tanpa langkah perakitan bundel. Impor diminta oleh browser sebagai modul ES asli. Server dev memotong permintaan untuk file .vue dan mengkompilasinya dengan cepat. Dan itu terjadi secara instan.

Harap perhatikan bahwa Vite masih merupakan solusi eksperimental, dan saya tidak tahu apa yang akan terjadi di masa depan dan berapa lama artikel ini akan tetap berguna. Anggap saja sebagai minuman beralkohol atau hanya sebagai sesuatu yang menyenangkan.

Mari mencoba


Buat file berikut:

Comp.vue

<template>
  <button @click="count++"> {{ count }}</button> 
</template>

<script>
export default {
  data: () => ({ count: 0 })
}
</script>

<style scoped>
button{
  font-size: 2rem;
}
</style>

index.html

<div id="app"></div>
<script type="module">
  import { createApp } from 'vue'
  import Comp from './Comp.vue'

  createApp(Comp).mount('#app')
</script>

Kemudian lakukan:

npx vite

Pergi ke http://localhost:3000, edit file .vueuntuk melihat perubahan instan.

Majelis Produksi


Sekarang mari kita buat aplikasi siap produksi .

Membuatpackage.json

{
  "scripts": {
    "build": "vite build"
  }
}

Lari:

npm i -D vite
npm run build


Periksa direktori dengan kode tersebut.
Masalah yang saya temui:

  • Kesalahan: Tidak dapat menemukan modul 'tslib' Diselesaikan dengan
    menjalankan npm i -D tslib
  • Kesalahan: ENOENT: tidak ada file atau direktur seperti itu
    Dipecahkan dengan secara manual membuat direktori kosong

Pikiran


Itu sangat mudah untuk memulai, dan pengembangan berjalan lebih cepat, karena saya segera melihat hasil dari perubahan. Sesuatu seperti kompilasi dengan cepat. Jadi itu terlihat sangat menjanjikan, mungkin setelah beberapa saat, kami akan beralih ke metode pengembangan untuk Vue.

Evan Yu, lakukan sihirmu.

Fakta menyenangkan: vue diterjemahkan dari bahasa Perancis sebagai tampilan, dan vite lebih cepat.

Tampaknya pada saat ini, pengerjaan Vite telah selesai, dan mungkin akan dilanjutkan setelah Vue 3. Cara termudah untuk mencoba adalah Vue 3 beta.

Saya mengatakan bahwa itu selesai berdasarkan tweet dari Evan Yu.Namun, melihat komitmen dalam repositori, kita dapat mengatakan bahwa pekerjaan masih berlangsung.
