目標を見つける方法。チャンスの役割

1.はじめに


検索は、複雑な場合と単純な場合があります。目標自体とその達成方法の両方がわからない（または部分的にしかわからない）場合は、ランダム性が重要です。

この記事の目的は、動いている（黄色のオブジェクト）と動いていない両方のターゲットを見つける方法を比較することです。

これらの方法：

  • ランダム検索（赤いオブジェクト）
  • メモリを使ったランダム検索（青いオブジェクト）
  • メモリと階層によるランダム検索（緑色のオブジェクト）
  • 最初のルート（紫のオブジェクト）を検索します
  • 短いルート（茶色のオブジェクト）を検索します

2.主要な部分


2.1。クラスP-ランダム検索

コンストラクターは、クラス属性と変数を初期化します。メインウィンドウ、色、yおよびx座標、カウンター、訪問済み辞書、ターゲット、階層辞書、近隣辞書。init_3_dictメソッドは、3つの辞書を作成します。showメソッドは、セルを目的の色で描画します。updateメソッドはオブジェクトを更新します。top_showメソッドは、トップレベルウィンドウを作成し、ターゲットに対して実行されたステップの数を示します。

class P():
    def __init__(self, root, color, node, target, maps, ban):
        self.root = root
        self.color = color
        self.y = node[0]
        self.x = node[1]
        P.target = target
        self.count = 0
        
        self.visit = {}
        P.hierarchy = {}
        P.neighbor = {}
        
        self.init_3_dict(maps, ban)

is_en関数はinit_3_dictメソッドで宣言され、座標が禁止されておらず、マップを超えていないことを確認します。 

def init_3_dict(self, maps, ban):
        def is_en(yx):
            if yx[0] < 0 or yx[0] > len(maps)-1: return
            if yx[1] < 0 or yx[1] > len(maps[0])-1: return
            if yx in ban: return
            return True

ループでは、座標キーでアクセスしたディクショナリに初期値ゼロを書き込みます。同じキーの階層辞書に、この座標の階層名を記述します。次に、is_en関数を呼び出して、隣人の辞書を入力します。

for y in range(len(maps)):
            for x in range(len(maps[0])):
                self.visit[(y,x)] = 0
                P.hierarchy[(y,x)] = maps[y][x]
                
                n = []
                if is_en((y-1,x)):
                    n.append((y-1,x))
                if is_en((y+1,x)):
                    n.append((y+1,x))
                if is_en((y,x-1)):
                    n.append((y,x-1))
                if is_en((y,x+1)):
                    n.append((y,x+1))
                    
                P.neighbor[(y,x)] = n

showメソッドは、y、x座標でセルを目的の色で描画します。

def show(self, y, x, color):

カウンターを増やします。目標が達成されているかどうかを確認します。目標が達成されると、クラス変数J.disableが正しく書き込まれます。top_show（）メソッドを呼び出します。

def move(self):

def update(self):

2.2。クラスM-メモリを

使用したランダム検索コンストラクターは親クラスのコンストラクターを呼び出し、移動先のフィールドのセルの値を増やします。moveメソッドはオブジェクトを移動します。選択メソッドは、メモリを使用したランダム検索アルゴリズムが選択した座標を返します。

moveメソッドで、choiceメソッドを呼び出します。それ以外は、彼の作品は親の方法と似ています。

def move(self):
        yx = self.choice((self.y, self.x))

選択メソッドは、近傍の座標を反復処理し、タプルをリストvに追加します。タプルの最初の要素は隣人の座標、2番目の要素は訪問された回数です。次に、小さいものから大きいものへと並べ替えます。すべてのタプルを並べ替え、リストvには、最初のタプルに記録された回数だけ発生したタプルのみを残します。それらのいずれかをランダムに選択します。

def choice(self, yx):
        v = []
        for i in P.neighbor[yx]:
            v.append((i, self.visit[i])) 
            
        v.sort(key = lambda x: x[1], reverse = False)
        v = [i for i in v if v[0][1] == i[1]]
        v = random.choice(v)
        self.visit[v[0]] += 1
        return v[0]

2.3。クラスN-メモリと階層を使用したランダム検索。

選択方法では、ターゲットの階層名に最もよく一致するタプルを選択します。この一致が同じ場合、最も訪問されていない座標が選択されます。

コンストラクターは親コンストラクターを呼び出し、一致属性をターゲットとの階層名の文字一致数に初期化します。

def __init__(self, root, color, node, target, maps, ban):
        super().__init__(root, color, node, target, maps, ban)
        self.coincidence = 0

ループの選択メソッドは、近傍の現在の座標の階層名を書き込みます。変数rには、この変数とターゲットの一致数が格納されます。一致数rの方が大きい場合、一致は上書きされます。この場合、リストvのタプル（座標、訪問数）がリストdに書き込まれます。rが偶然に等しい場合、dでvからタプルを追加します。

def choice(self, yx):
        v = []
        for i in P.neighbor[yx]:
            v.append((i, self.visit[i]))
             
        d = []
        for l in v:
            c = P.hierarchy[l[0]]
            
            r = 0
            for i in range(len(P.target)):
                if c[i] == P.target[i]:
                    r +=1
                else: break 
                
            if r > self.coincidence:
                self.coincidence = r
                d = [l]
            if r == self.coincidence:
                d.append(l)
        
        if d: v = d 
           
        v.sort(key = lambda x: x[1], reverse = False)
        v = [i for i in v if v[0][1] == i[1]]
        v = random.choice(v)
        self.visit[v[0]] += 1
        return v[0]

2.4。クラスK-ルートを検索し、最短ルートを検索します。

コンストラクターは終了座標を終了変数に書き込みます。 shortパラメータがFalseの場合、ルート検索を呼び出します。それ以外の場合は、最短ルートを見つけます。

find_pathメソッドは、現在の座標をルートに追加します。目標に到達した場合、ルートを返します。私たちはすべての隣人を分類します。パスに隣人がいない場合は、自分自身を再帰的に呼び出します。そして、メソッドが戻るという事実はnewpathに書き込まれます。最初に見つかったルートはnewpathです。 find_short_pathメソッドの違いは、newpathの長さがshortestの長さ以上の場合、shortestが上書きされることです。

def find_path(self, node, end, path=[]):
        path = path + [node]
        if node == end:
            return path
            
        for v in P.neighbor[node]:
            if v not in path:
                newpath = self.find_path(v, end, path)
                if newpath: 
                    return newpath

2.5。クラスJは移動ターゲットです。

クラスJのオブジェクトはクラスPのように移動します。クラスJ.disableの変数がtrueの場合、オブジェクトは停止します。

2.6。機能メイン。

メインウィンドウを作成します。ターゲット変数には、ターゲットの階層名を記述します。禁止リストには、禁止されたセルの座標が含まれています。マップのリストで、マップ自体。ループでは、マップを描画します。クラスオブジェクトを作成します。次に、オブジェクトを更新する更新関数を呼び出します。
