Arrays


配列の初期化：


$ArrayExample = @(“First”, “Second”)

空の配列の初期化：


$ArrayExample = @()

インデックスで値を取得する：


$ArrayExample[0]

配列全体を取得します。


$ArrayExample

アイテムを追加する：


$ArrayExample += “Third”

$ArrayExample += @(“Fourth”, “Fifth”)

並べ替え：


$ArrayExample | Sort

$ArrayExample | Sort -Descending

しかし、この並べ替えの配列自体は変更されていません。また、配列内のソートされたデータが必要な場合は、ソートされた値を割り当てる必要があります。


$ArrayExample = $ArrayExample | Sort

実際、PowerShellの配列から削除された要素はありませんが、次の方法で行うことができます。


$ArrayExample = $ArrayExample | where { $_ -ne “First” }

$ArrayExample = $ArrayExample | where { $_ -ne $ArrayExample[0] }

配列を削除：


$ArrayExample = $null
