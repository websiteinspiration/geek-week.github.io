ノンブロッキングの描画とボケによるチャートの更新



計算を行うPythonスクリプトが1つあります。約2000回の反復のサイクルがあり、それぞれが数分と見なされました。

そして、そのスクリプトを巧妙にデバッグし、反復回数からいくつかのメトリックのグラフを表示することにしました。そして、次の反復が計算されると、このスケジュールと更新が行われます。

これを行う最も簡単な方法は、ボケです。より正確には、ボケサーバーを使用してグラフを描画します。方法-今からお話ししましょう。

最初に、サーバーを起動します。サーボボカはボケ自体とともに箱から出てくるので、pip install bokehの後に、コンソールでbokeh serveと入力するだけでサーバーが起動します。

なぜそれが必要なのですか？そして、チャートを表示する

  • 残りのコードの実行をブロックしませんでした（ブラウザーで、別のプロセスで発生するため）。
  • グラフがウィンドウのサイズ変更（または最小化-最大化）に応答するように
  • そしていつでも、Pythonプロセスからいつでもこのスケジュールを好きなように変更できます！

これは次のように行われます： 

import time
import sys
from bokeh.plotting import figure
from bokeh.client import pull_session
from bokeh.models import ColumnDataSource

#     --    -,    bokeh serve
# Please run "bokeh serve" in console before start!


if __name__ == "__main__":
    #    (  ,     )
    session = pull_session()

    #  .. ,      (    )
    fig = figure(title=("Total TBS (in bits)"), plot_height=300, plot_width=800)

    #        
    datasource = ColumnDataSource(data={"x": [], "y": []})
    line = fig.line(x="x", y="y", source=datasource, line_width=2, legend=("Super dooper line from hell"))

    #       
    session.show(fig)

    #    
    for i in range(10000):

        #           .   datasource    
        #       = )
        datasource.stream({"x": [i], "y": [i ** 2]})

        #       30-40      ,  
        session.force_roundtrip()


#  !
