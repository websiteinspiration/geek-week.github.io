SAP on Microsoft Azureツアー（モスクワ）

Microsoft Azure上のSAPに特化したワールドツアーのロシアステージにご招待します。このイベントでは、AzureクラウドにおけるSAPソリューションの配置についての新しい見方を提供し、移行のベストプラクティスを共有します。

2019年12月5日9:00から17:00
ノボテルモスクワシティ、Presnenskaya nab。、2、モスクワ

→  登録



イベントのプログラム：

SAPとMicrosoft Azure：将来の概観。

円卓会議：実装履歴、AzureでSAPをホストする利点、バグに取り組む。

SAPシステムをS / 4 HanaからAzureにMicrosoftで移行した実務経験。

SAPシステムをAzureクラウドにデプロイするためのIntelの革新的なインフラストラクチャコンピューティングおよびストレージソリューション。

テクノロジーの概要：さまざまなSAPワークロード用のクラウドインフラストラクチャ。

演習ラボ：Terraform / Ansible、ARM、またはSAP CALを使用して、AzureでS / 4 HANAソリューションをすばやく起動します（ラップトップを持ち込む*）。

特別招待講演者：

Kevin Scott（戦略的追跡チームEMEAのゼネラルマネージャー）とJan Tretina（SAP on Azure Architect、Microsoft、中央ヨーロッパ、東ヨーロッパ）。
