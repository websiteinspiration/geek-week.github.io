簡単な通話追跡サービスの作成、パート1

この記事では、サイトにコールトラッキングを実装するための可能なオプションの1つを検討します。この資料は、純粋に教育目的で作成されたものであり、コールトラッキングサービスを提供する既存の専門サービスを置き換えるという目的はありません。

機能要件


  1. このサービスは、サイトへの移行のソース（検索エンジンからの移行、広告への移行、utmタグとのリンクの通過）によって数字（数字）を置き換える機能を提供する必要があります。
  2. サービスは、表示された番号をサイト訪問者に修正し、次回の訪問時にこの番号を表示する必要があります。
  3. 表示された番号への通話を開始する場合、サービスはGoogleアナリティクスでイベントの作成を開始する必要があります。

一般的な力学


サイトにアクセスすると、遷移のソースが決定され、この訪問者に番号が割り当てられているかどうかが確認されます。訪問者が最初にサイトを訪問した場合は、一意の識別子を割り当て、その後、この参照ソースに関連する番号のプールから番号を割り当てます。将来、訪問者が再びサイトに戻ると、以前に割り当てられた番号が表示されます。

通話履歴に注目したい。番号の置き換え、ユーザーへの割り当ては技術的な問題です。主なことは、特定の通話と現在ユーザーに割り当てられている番号を比較できるようにすることです。使用されているテレフォニーで進行中の通話に関する詳細情報を継続的に受信できる場合、特定の番号への通話の瞬間を知り、その瞬間にどのユーザーのWebサイトにどの番号が表示されたかについての情報がある場合、Googleアナリティクスでイベントの作成を開始し、それをROKに関連付けることができます。

構造


  1. ct-server.ru-実行可能なWebサービススクリプトがここに配置されます。これは、部屋の設定を保存し、訪問者を記録するためのデータベースです。
  2. refer-to-site-with-ct-code.ru-このサイトでは、検索ネットワークからの移行をエミュレートします。
  3. site-with-ct-code.ru-番号置換スクリプトが配置されるサイト。

番号置換スクリプト


次に、ct-server.ru / script.jsスクリプトに移動して、site-with-ct-code.ruサイトに接続します。

最初に、2つの関数を作成します-作成するためのsetcookieとCookieを読み取るためのgetcookieです。これらを使用して、ユーザーのCookie IDと電話番号を記録および取得します。

// 
function setcookie(phone, value, expires, path, domain, secure) {
  document.cookie =
    phone +
    "=" +
    escape(value) +
    (expires ? "; expires=" + new Date(expires) : "") +
    (path ? "; path=" + path : "") +
    (domain ? "; domain=" + domain : "") +
    (secure ? "; secure" : "");
}

// 
function getcookie(phone) {
  var cookie = " " + document.cookie;
  var search = " " + phone + "=";
  var setStr = null;
  var offset = 0;
  var end = 0;

  if (cookie.length > 0) {
    offset = cookie.indexOf(search);

    if (offset != -1) {
      offset += search.length;
      end = cookie.indexOf(";", offset);

      if (end == -1) {
        end = cookie.length;
      }

      setStr = unescape(cookie.substring(offset, end));
    }
  }

  return setStr;
}

その後、makeid関数を追加して、ユーザーのIDを生成します。

//  id 
function makeid(length) {
  var result = "";
  var user_id = "abcdefghijklmnopqrstuvwxyz0123456789";
  var user_idLength = user_id.length;
  for (var i = 0; i < length; i++) {
    result += user_id.charAt(Math.floor(Math.random() * user_idLength));
  }
  return result;
}

メソッドを使用して、数値のリクエストを生成します。ct-server.ruから得られた数値は、Cookieに1時間書き込まれ、要素のIDによるページ上の数値の最初の置換を実行します。

//    ,  ,       id  
function getNumberFirstTime() {
  var method = "get_num_first";
  var ct_user = getcookie("ct_user");
  var gid = getcookie("_ga");
  var ref = document.referrer;
  var host = window.location.origin;
  var page = document.location.href;
  var xhr = new XMLHttpRequest();
  var body =
    "&method=" +
    encodeURIComponent(method) +
    "&page=" +
    encodeURIComponent(page) +
    "&ct_user=" +
    encodeURIComponent(ct_user) +
    "&ref=" +
    encodeURIComponent(ref) +
    "&host=" +
    encodeURIComponent(host) +
    "&gid=" +
    encodeURIComponent(gid);

  xhr.open("POST", "http://ct-server.ru", true);
  xhr.setRequestHeader("Content-Type", "application/x-www-form-urlencoded");
  xhr.withCredentials = true;
  xhr.send(body);
  xhr.onreadystatechange = function() {
    //console.log(xhr.responseText);
    var t = xhr.responseText;
    document.getElementById("calltracking").innerHTML = t;
    setcookie("phone", t, new Date().getTime() + 60 * 60 * 1000); // 
  };
}

function sendToCalltracking() {
  var ggg = getcookie("ct_user");

  if (
    typeof ggg === "undefined" ||
    ggg === null ||
    ccc === "undefined" ||
    ccc === null ||
    ccc.length < 1 ||
    ggg.length < 1 ||
    ccc === "" ||

  }
}
