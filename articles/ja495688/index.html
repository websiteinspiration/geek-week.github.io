Active DirectoryベースのZimbraコラボレーションOSEユーザー同期

1.背景


zimbraサーバーを構成するときに、Active Directory（AD）でユーザーをグループごとに同期する問題が発生しました。ADで新しいユーザーを作成すると、通常は追加されますが、グループごとにメールサーバーにアクセスすると、初めてすべてのユーザーが同期されます。ただし、グループの変更は、zimbraメールサーバーのユーザーの変更には影響しません。

これらの記事で気に入らなかったのは、Powershellスクリプトの使用（ldapsearchがある場合）とzmprovユーティリティの継続的な使用であり、多数のユーザーが同期されている場合、スクリプトは長時間実行されます

2.ソースデータ


サーバーOS：CentOS 7
スクリプト言語：bash
Zimbraドメイン：test.ru
Zimbraサーバー：zimbra.test.local
Active Directoryドメイン：test.local
メールにアクセスするためのADグループ：メール

ユーザーは、ログインなどのメールとは異なるメールを持っている可能性がありますメールフィールドのADに追加し、それを使用してzimbraにエイリアスを作成します（たとえば、Vasya Pupkinはユーザー名vasiaを使用してシステムにログインしますが、メールはユーザーv.pupkin@test.ruとして送受信する必要があります）

3.スクリプトのスキーム


  1. メールグループに含まれるADユーザーファイルに保存する
  2. すべての属性を持つすべてのzimbraユーザーをファイルに保存します
  3. すべてのzimbraユーザーのリストを含むファイルをフォーマットファイルに分割します。ファイル名-ユーザーログイン、コンテンツ-ユーザー属性
  4. ユーザーのリストからジンブラシステムアカウント（管理者、ギャル、ウイルス対策）を削除します
  5. ADとzimbraのユーザーリストを比較する
  6. コマンドを使用してファイルを作成し、追加、削除（私の場合、ユーザーをブロック）、同期、エイリアスを作成します
  7. これらのアクションをzimbraに適用します（zmprovを1回呼び出す）
  8. 管理者にメールでレポートを送信します（送信するものがある場合）
  9. 一時ファイルとディレクトリを削除する

4.同期スクリプト


スクリプトには2つの操作モードがあります-これはパラメーターなしで起動するため、ユーザーのブロックと追加のみが機能します。そして、「all」パラメーターで開始すると、ADのメールグループのすべてのユーザーが同期されます。

また、同期機能でのbase64デコードユーティリティの使用に注意する必要があります。これは、ロシア語の文字が使用されるADフィールドで使用する必要があります。

function search_users_AD()
{
	echo "     AD..." >> $log
	$ldapsearch -x -o ldif-wrap=no -H $ldap_server -D $binddn -w $bindpw -b $basedn $filter $fields | 
	grep sAMAccountName | 
	egrep -v '^#|^$' | 
	awk '{print $2}' |
	sort > $usname.ad
	echo "Found () "`cat $usname.ad | wc -l`" Group in AD (  AD)" >> $log
}

function all_user_attr_zimbra()
{
	#        
	$zmprov -l gaa -v $domain > $usname.gaa
	#         
	cd $zim_us
	#      ,     
	csplit $usname.gaa --prefix='user.' --suffix-format='%03d.zim' --elide-empty-files -s /"# name"/ '{*}'

	for i in $( ls $zim_us )
	do
		nam=`grep "# name" $zim_us/$i | awk '{ print $3}' | sed 's/@.*//g'`
		mv -f $zim_us/$i $zim_us/$nam
	done
	cd $path
}

function search_user_zimbra()
{
	echo "    zimbra..." >> $log
	ls $zim_us | sort > $usname.tem

	diff -u -i $usname.tem $path/system.acc  | sed 1,3d | grep ^- | cut -c 2- | sort > $usname.zim
	#rm -f $usname.tem
	echo "Found () "`cat $usname.zim | wc -l`" Group in Zimbra (  Zimbra)" >> $log
	
}

	fi

function block_user()
{
	deldif==`grep ^- $usname.diff | sed '1!d'`
	if [ -n $deldif ];
	then
		for delus in $( grep ^- $usname.diff | cut -c 2- )
		do
			#zimbraAccountStatus closed
			if [ $delus != "" ];
			then
				ifclos=`grep "zimbraAccountStatus:" $zim_us/$delus | awk '{print $2}'`
				if [ "$ifclos" != "closed" ];
				then
					echo "user closed - $delus"
					echo "ma $delus@$domain zimbraAccountStatus closed" >> $zmcmdfile
					echo " $delus !     !" >> $tmp_dir/send.txt
					echo $delus >> $path/close.1
					cat $path/close.1 | sort > $path/close.diff
					echo "$delus" 
				fi
			fi
		done
	fi
}
