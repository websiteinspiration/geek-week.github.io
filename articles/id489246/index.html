Saya berumur 14 dan saya memutuskan untuk mengembangkan game

Hai, nama saya Georgy, saya di kelas 8 dan saya 14 tahun. Dengan menunjukkan usia saya yang sebenarnya, saya hanya mencoba untuk menarik lebih banyak perhatian pada proyek saya, masih tidak setiap hari Anda akan bertemu dengan siswa ke-14 yang mengembangkan bukan hanya "5 menit clicker "Tapi benar-benar sesuatu yang berharga, saya tidak ingin terlihat seperti orang dengan harga diri yang tinggi, saya hanya suka apa yang saya lakukan saat ini).

Semuanya dimulai dari akhir 2017, ketika saya mencoba untuk menjalankan server saya di minecraft, mengkonfigurasi plugin, kita dapat mengatakan bahwa saat itulah saya berkenalan dengan pemrograman. Beberapa saat kemudian, saya menyadari bahwa saya tidak akan dapat menjaga seluruh server di tangan saya dan saya memutuskan untuk meninggalkan usaha ini. Setelah pertemuan 2018 baru, liburan musim dingin dimulai dan sejak musim dingin di daerah saya adalah sesuatu seperti itu ...



... saya sedang duduk di rumah. Sekali lagi, berselancar di rekomendasi YouTube, saya melihat video, saya tidak ingat persis apa namanya, tetapi sesuatu seperti ini - "Cara membuat game di UNITY hanya dalam 30 menit." Setelah menonton video ini, saya memiliki keinginan yang kuat untuk membuat game saya, dan membaca, menonton, membandingkan berbagai mesin dan platform, secara umum saya memutuskan untuk mengembangkan game ini pada kesatuan untuk smartphone dan tablet, khususnya untuk android.

Saluran ANDROIDHELPER memberi saya dorongan luar biasa dalam memahami mesin dan pemrograman dalam C #, terima kasih Andrey dan Zhenya, Anda adalah yang terbaik! Yah, berbulan-bulan berlalu dan saya terus membuat game, meskipun apa yang saya lakukan kemudian tidak bisa disebut game, itu adalah aplikasi dengan judul dan beberapa tombol. Di musim panas, ketika saya tidak mengganggu studi saya dan memiliki banyak waktu luang, saya memutuskan untuk membuat aplikasi ( Tentu saja, mengembangkan aplikasi sederhana di Unity adalah tidak masuk akal, tetapi kemudian bagi saya itu ide yang bagus ), saya menambahkan sejumlah besar tim untuk Yandex Alice ke aplikasi, berbagi semuanya ke dalam kategori. Menambahkan iklan, diterbitkan di google play dan bergegas ...



Kebenaran tidak bertahan lama, sekitar satu bulan, tetapi saya masih berhasil mendapatkan $ 360. Saya tidak dapat menyebutkan alasan yang tepat untuk pelarangan aplikasi, saya pikir itu disebabkan oleh kenyataan bahwa saya tidak memiliki kebijakan pendingin udara, dan dalam hal ini saya diwajibkan untuk melakukan ini, karena permainan memiliki iklan, terutama dari appodeal. Untuk waktu yang lama saya praktis tidak mempublikasikan apa pun, tetapi hanya terlibat dalam pengembangan di industri ini, berlatih, mendapatkan pengalaman ...

Dan sekarang, akhir tahun 2019, dan saya berpikir, "Mungkin saya bisa membangkitkan permainan baru ...", saya mulai berpikir dan membuat sketsa segala sesuatu di atas kertas. Secara umum, inti dari permainan ini adalah bahwa beberapa pemain dapat bermain di satu smartphone sekaligus, tentu saja ini bukan inovasi di industri game, itu sebabnya saya ingin mengalahkannya entah bagaimana dengan indah, saya datang dengan mode dan memutuskan untuk mengimplementasikannya, secara umum ternyata:



Saya senang bahwa saya menyadari pada waktunya bahwa ada baiknya beralih ke 3D, saya hampir tidak bekerja dengan game 3D sebelumnya, jadi saya memiliki sedikit pengalaman, dan saya ingin terbiasa dengan game tiga dimensi sesegera mungkin, setelah beberapa saat, ternyata:



Adapun saya jauh lebih baik daripada di 2D. Kemudian saya mulai mengembangkan mode baru, mengoptimalkan dan memperbaiki sejumlah besar bug. Setelah beberapa waktu, mode Sprint dan Darts muncul, dan yang terbaru saya memperbarui permainan dengan menambahkan mode - Boxing


Saat ini, permainan sedang dalam tahap pengujian BETA, karena ada beberapa mode, dan tidak ada hal-hal dasar, misalnya: ekonomi, papan peringkat, penonaktifan iklan, dll.

Nah, cerita pendek saya akan segera berakhir, jika saya melihat resonansi besar tentang ini, saya siap untuk terus menulis tentang topik ini, dan mungkin kata demi kata, kesulitan apa yang saya temui ketika membuat game dan bagaimana saya mengatasi kesulitan ini.

Jika seseorang tertarik, sambut

semua orang, terima kasih sudah membaca! Saya ingin mengucapkan terima kasih khusus kepada semua orang yang menginstal game, Anda dapat menulis ulasan dan saran di sini, saya akan mencoba menjawab semua orang.

Sampai jumpa lagi!
