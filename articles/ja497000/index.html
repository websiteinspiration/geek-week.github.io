Power Automate VS Logic Apps。Logicアプリ

全てにおいて良い日！Power AutomateとLogic Appsに関する以前の記事では、Power Automateを使用する可能性のいくつかを検討しました。この記事では、Logic Appsを使用するいくつかのシナリオと、Power Automateとのいくつかの違いについて詳しく説明します。以前に発見したように、Power AutomateとLogic Appsは、それらの場所（Office 365、Azure）だけでなく、ライセンスへのアプローチといくつかの内部機能のみが異なるツインサービスです。Power Automateとは対照的に、Logic Appsの機能を今日見てみましょう。時間を無駄にしません。

1.トリガートリガー周波数


Power Automateでは、トリガー条件をチェックする頻度を構成する方法はありません。デフォルト値に依存する必要があります。Logic Appsには、トリガーチェックの間隔と頻度を構成する機能があり、イベント処理を大幅に高速化します。ただし、Power Automateには、Logic Appsよりもトリガーの設定がはるかに少ないことがよくあります。

Logic Appsには、このトリガーのタイムゾーンと開始時間の設定もあります。

2.ストリーム表示モードを切り替える


ロジックアプリは、Power Automateとは異なり、デザインビューとコードビューを切り替える機能を備えています。この機能は、スレッドのデバッグに役立ちます。また、スレッドのロジックにさらに微妙な変更を加えることができます。



3.デバッグスレッド


多くの場合、スレッドを設定するとき、スレッドに埋め込まれた1つまたは別のロジックの実行の正しさを確認する必要があります。そして、ここではデバッグなしで行うことはできません。Logic Appsには非常に便利なストリームデバッグモードがあり、ストリームの各アクティビティの入力データと出力データを表示できます。このモードを使用すると、いつどのアクティビティでどの情報がアクティビティに流れ込み、何がアクティビティの出力であることが判明したかをいつでも確認できます



。PowerAutomateでは、このモードは存在しますが、非常に切り捨てられたバージョンです。

4.プレミアムコネクタ


すでにご存じのとおり、Power Automateでは、コネクタがタイプ



ごとにレギュラーとプレミアムに分かれています。レギュラーコネクタは常に利用可能です。プレミアムコネクタは、ユーザーまたはフロー用に別のプランを購入する場合にのみ利用できます。Logic Appsでは、すべてのコネクターを一度に使用できますが、コネクターの使用に合わせて充電が行われます。従来のコネクタのストリームでの実行は安価で、プレミアムはより高価です。

5.ボタンでストリーミングを開始する


また、Logic AppsはPower Automateを失います。たとえば、Power AppsアプリケーションのボタンではLogic Appsストリームを開始できません。前の記事で説明したように、Power Automateを使用すると、スレッドを作成してPower Appsアプリケーションに接続し、アプリケーションのボタンをクリックするなどして、後で呼び出すことができます。Logic Appsの場合、そのようなシナリオを実装する必要がある場合は、さまざまな出力を考え出す必要があります。たとえば、「HTTP要求の受信時」トリガーを使用して、POSTアプリケーションから事前構成されたアドレスに要求を送信します。



6. Visual Studioを使用してストリームを作成する


Power Automateとは異なり、Logic AppsはVisual Studioを介して直接スレッドを作成できます。
Azure Logic Apps拡張機能がインストールされている場合は、たとえばVisual Studio CodeからLogic Appsストリームを作成および編集できます。拡張機能をインストールすると、Azureに接続できるようになります。承認が成功すると、この環境の既存のLogic Appsストリームにアクセスできるようになり、必要なストリームの編集に進むことができます。



もちろん、これら2つの製品のすべての違いを挙げたわけではありませんが、Power AutomateとLogic Appsを使用してスレッドを開発するときに目に留まった機能に注目しようとしました。次の記事では、Power Platformラインの他の製品を使用した興味深い機能と実装事例を紹介し、Logic Appsに何度も戻ります。みなさん、良い一日を！
