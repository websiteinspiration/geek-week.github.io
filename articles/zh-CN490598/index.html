Microsoft关于考试和证书的公告

在过去的三个月中，Microsoft发布了3条重要的公告：

  1. 自2021年以来，MCSA，MCSD，MCSE以及所有以前的考试（基于角色的新考试除外）均不复存在。
  2. 几乎所有以前的基于Azure角色的考试都已过时。可以在带有分析的另一篇文章中阅读
  3. 将会有几项新的考试以及相应的证书：
    • 考试DP-300：在Microsoft Azure（测试版） / Microsoft认证的关系数据库上进行管理：Azure数据库管理员
    • 考试DA-100：使用Microsoft Power BI（测试版）分析数据 /获得Microsoft认证：数据分析师助理
    • 考试AZ-120：针对SAP Workloads的规划和管理Microsoft Azure / Microsoft认证：针对SAP Workloads的Azure
    • 考试AZ-220：Microsoft Azure IoT开发人员 / Microsoft认证：Azure IoT开发人员专业

首先是第一件事。

MCSA / MCSA / MCSE-一切。就这样。


我敢肯定，与所有其他公告相比，MCSA / MCSA / MCSE的过时引起了更大的反响。如果在SQL 2019或Windows Server 2019中不仅有任何新考试，也不会那么吓人。但是实际上，Microsoft将不在Azure / Dynamics365 / Office365中的所有证书和考试乘以零。当然，我们为您加了丸，以确保您的考试成绩单中通过的事实不会消失，但这不会改变本质。

Azure SAP


这里的一切都很复杂，因为没有Microsoft技术与Azure相结合。同时检查Azure基础结构/ SAP组件和服务（用于监视，优化）和体系结构。

Azure物联网


考试结合了C＃中的IoT开发人员的技能，IoT解决方案的管理员/集成商（非常需要设备配置），IoT的Azure SaaS用户和架构师的技能。显然，物联网越来越深入我们的生活，想要验证其技能的人需要进行考试。就个人而言，我不确定物联网开发人员的数量会很大，但是显然需要进行考试才能结束相应的领域。

Power BI


很简单：如何使用Power BI。就个人而言，我不喜欢该考试的证书名称“ Microsoft认证：数据分析师助理”。还有许多其他出色的数据分析工具，PowerBI太狭for了。

Azure SQL管理员


这里所有的单词都很重要-不能丢掉一个单词。这不是开发，而是部署，优化，性能，容错等。本地SQL Server仅被视为Hybrid的一部分（在有关Private的技能评估中没有一个关于Private的词）。
