Menambahkan, menghapus, dan mengganti nama file serta tujuan dengan mudah dalam proyek-proyek CMake

Bekerja dengan proyek CMake di Visual Studio 2019 16,5 Pratinjau 2 sekarang lebih mudah dari sebelumnya. Sekarang Anda dapat menambah, menghapus, dan mengganti nama file sumber dan target dalam proyek CMake Anda dari IDE tanpa secara manual mengedit skrip CMake. Saat menambahkan atau menghapus file menggunakan Solution Explorer, Visual Studio secara otomatis mengedit proyek CMake Anda. Anda juga dapat menambah, menghapus, dan mengganti nama sasaran proyek menggunakan CMake Target View di Solution Explorer.



Lebih detail di bawah potongan.

File sumber C dan C ++


Visual Studio sekarang melacak file sumber C dan C ++ ketika mereka ditambahkan, diganti namanya, atau dihapus dari Solution Explorer, secara otomatis memodifikasi proyek CMake yang mendasarinya. Fungsi ini diaktifkan secara default di Visual Studio 2019 16.5 Pratinjau 2, tetapi jika Anda ingin Visual Studio tidak mengubah proyek secara otomatis, Anda dapat menonaktifkan fungsi ini di Alat> Opsi> CMake, â€œAktifkan modifikasi skrip CMake otomatis ...â€:
