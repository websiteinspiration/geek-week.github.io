▍1。文字列の連結


文字列のリストを連結する必要がある場合は、ループforでこれを行うことができ、各行を1つずつ最終結果に追加します。ただし、この方法は、特にリストが十分に長い場合は非常に非効率的です。Pythonでは、文字列は不変のエンティティです。その結果、各文字列連結操作は、数行を新しい行にコピーする必要があることを意味します。

この問題を解決するより効果的な方法は、次の関数を使用することjoin()です。

characters = ['p', 'y', 't', 'h', 'o', 'n']
word = "".join(characters)
print(word) # python

▍2。リストジェネレーターの使用


リストジェネレーターは、他の反復可能なオブジェクトから新しいリストを作成するために使用されます。ジェネレータはリストを返すため、その説明は角括弧で囲まれた式であり、リストの各要素に対して実行されます。これには、for各要素を通過するループの説明も含まれます。Pythonインタープリターは、リストをたどったときに繰り返されるテンプレート用に最適化されているため、リストジェネレーターはリストの処理を高速化できます。

例として、リストジェネレータを使用して最初の5つの整数の二乗を見つけることを検討してください。

m = [x ** 2 for x in range(5)]
print(m) # [0, 1, 4, 9, 16]

次に、2つのリストのそれぞれに表示される番号を見つけます。

list_a = [1, 2, 3, 4]
list_b = [2, 3, 4, 5]
common_num = [a for a in list_a for b in list_b if a == b]
print(common_num) # [2, 3, 4]

▍3。enumerate（）でリストを反復する


このメソッドenumerate()は、反復可能なコレクションに番号付けを追加し、要素のインデックスとこの要素自体で構成される要素のペアを生成するオブジェクトを返します。

インタビューで提供される古典的なFizzBu​​zzの問題を解決します。このタスクの条件は次のとおりです。「数値のリストを表示するプログラムを作成します。この場合、3の倍数である数値の代わりに、fizzが表示されます。5の倍数である数値-バズの代わりに、3と5の倍数である数値の代わりに-fizzbuzzが表示されます。

numbers = [30, 42, 28, 50, 15]
for i, num in enumerate(numbers):
    if num % 3 == 0 and num % 5 == 0:
       numbers[i] = 'fizzbuzz'
    elif num % 3 == 0:
       numbers[i] = 'fizz'
    elif num % 5 == 0:
       numbers[i] = 'buzz'
print(numbers) # ['fizzbuzz', 'fizz', 28, 'buzz', 'fizzbuzz']

▍4。リストを操作するときにzip（）を使用する


同じ長さの複数のリストを組み合わせて、結果のリストを表示する必要があるとします。他の場合と同様に、いわば「額」で同様の問題を解決できます。または、ユニバーサル関数のようなものを使用できますzip()。

countries = ['France', 'Germany', 'Canada']
capitals = ['Paris', 'Berlin', 'Ottawa']
for country, capital in zip(countries,capitals):
    print(country, capital) # France Paris 
                              Germany Berlin
                              Canada Ottawa

▍5。itertoolsモジュールの使用


Pythonモジュールitertoolsは、イテレーターを操作するためのツールのセットです。このモジュールの構造には、さまざまなシーケンスを生成するための多くのツールが含まれています。ここでは、例としてメソッドを考えitertools.combinations()ます。このメソッドは、組み合わせを作成するために使用されます。入力値をグループ化するためのツールもあります。

上記を理解するために実際の例を考えてみましょう。

あるトーナメントで4つのチームが参加していると想像してください。グループラウンドでは、各チームが他の各チームと対戦します。あなたの仕事は、互いに対戦するチームの可能なすべての組み合わせを生成することです。

次のコードを見てください。

import itertools
friends = ['Team 1', 'Team 2', 'Team 3', 'Team 4']
list(itertools.combinations(friends, r=2)) # [('Team 1', 'Team 2'),      ('Team 1', 'Team 3'),  ('Team 1', 'Team 4'),  ('Team 2', 'Team 3'),  ('Team 2', 'Team 4'),  ('Team 3', 'Team 4')]

ここでは、値の順序が重要ではないという事実に注意を払う必要があります。組み合わせ('Team 1', 'Team 2')て('Team 2', 'Team 1')1つの同じチームのペアを表すので、最終的なリストに含まれるのはそのうちの1つだけです。同様にitertools.permutations()、このモジュールのメソッドおよびその他のメソッドを使用できます。詳細なガイドitertoolsはここにあります。

▍6。Pythonコレクションの使用


Pythonコレクションはコンテナーデータ型です。特に、これらはリスト、セット、タプル、辞書です。このモジュールcollectionsは、コードを改善し、コードをより簡潔に、そして扱いやすくするための高性能なデータ型を開発者に提供します。このモジュールには、多くの便利なメソッドが含まれています。ここでメソッドを見てみましょうCounter()。

このメソッドは、リストやタプルなどの反復可能なオブジェクトを取り、検査されているリスト内のさまざまなオブジェクトの数に関する情報を含む辞書（Counter Dictionary）を返します。そのような辞書のキーは反復可能なオブジェクトで表される一意の要素であり、値はそのような要素の数量です。

オブジェクトを作成Counterするには、反復可能なオブジェクト（リストなど）をメソッドに渡す必要がありますCounter()。

import collections 
count = collections.Counter(['a','b','c','d','b','c','d','b'])
print(count) # Counter({'b': 3, 'c': 2, 'd': 2, 'a': 1}) 

モジュールの詳細はcollections見つけることができるここに。

▍7。2つのリストを辞書に変換する


2つのリストがあるとします。1つは学生の名前を含み、2つ目は成績を含みます。これらの2つのリストを辞書に変換する方法は？このの関数を使用するzip()と、この問題は次のように解決できます。

students = ["Peter", "Julia", "Alex"]
marks = [84, 65, 77]
dictionary = dict(zip(students, marks))
print(dictionary) # {'Peter': 84, 'Julia': 65, 'Alex': 77}

▍8。ジェネレーター関数の使用


ジェネレータ関数は、イテレータのように動作する関数です。これにより、プログラマはすばやく簡単にきちんとしたイテレータを作成できます。この考えを明らかにする例を考えてみましょう。

最初の1億個の整数の2乗の合計を1から探す必要があるとします。

このタスクは、難しくないように見えます。リストジェネレーターを使用して問題なく解決できます。ただし、ここでの問題は、大量の入力データです。次のコードを見てください。

import time
t1 = time.clock()
sum([i * i for i in range(1, 100000000)])
t2 = time.clock()
time_diff = t2 - t1
print(f"It took {time_diff} Secs to execute this method") # It took 13.197494000000006 Secs to execute this method

さらに多くの数の平方和を求める必要がある場合、解を見つけるのに長い時間がかかるため、この手法はそれらを見つけるにはうまくいかないことがわかります。この状況では、Pythonジェネレーター関数が役立ちます。今回のケースでは、リストジェネレーターからジェネレーター関数に切り替えるには、ジェネレーター式の角かっこを丸めに変更するだけで十分です。これを行い、問題を解決するために必要な時間を見つけます。

import time
t1 = time.clock()
sum((i * i for i in range(1, 100000000)))
t2 = time.clock()
time_diff = t2 - t1
print(f"It took {time_diff} Secs to execute this method") # It took 9.53867000000001 Secs to execute this method

ご覧のとおり、同じ問題を解決するのにかかる時間が短くなりました。そして、入力データの増加に伴い、この影響はさらに顕著になります。

▍9。関数から複数の値を返す


Pythonには、関数から複数の値を返す機能があります。これは、他の多くの一般的なプログラミング言語では見られません。関数から複数の値を返すには、それらをカンマで区切る必要があります。この値のリストに基づいて、Pythonはタプルを作成し、関数の呼び出し元にそれを返します。次に例を示します。

def multiplication_division(num1, num2):
    return num1*num2, num1/num2
product, division = multiplication_division(15, 3)
print("Product =", product, "Quotient =", division) # Product = 45 Quotient = 5.0

▍10。ソート済み（）関数の使用


Pythonではsorted()、関連するすべてのタスクを解決する組み込み関数を使用して、特定のデータシーケンスを簡単に並べ替えることができます。この関数は、シーケンス（リスト、タプル）を並べ替え、常に並べ替えられた要素を含むリストを返します。数値のリストを昇順でソートする例を考えてみましょう。

sorted([3,5,2,1,4]) # [1, 2, 3, 4, 5]

次に、行のリストを降順で並べ替える例を示します。

sorted(['france', 'germany', 'canada', 'india', 'china'], reverse=True) # ['india', 'germany', 'france', 'china', 'canada']
