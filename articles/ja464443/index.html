コードを解析します。

  1. キーshowpasswordを持つデータ配列の値が「yes」の場合、パスワードを使用できます。

  2. データ配列はloadData関数によって作成され、デフォルトのデータがパラメーターとして渡されます。



  3. loadData関数は、Cookieからデータ値をロードします（base64でデータをエンコードし、不明なキーでxorを暗号化し、json形式でデータをデコードします）;



  4. 受信した値を設定します。

するべきこと：

  1. XOR暗号化キーを回復します。
    • 暗号化されたデータをCookieから取得します。
    • base64をデコードします。
    • デフォルトのデータをjson形式でエンコードします。
    • Proxorはタイムラインを受け取りました。
  2. 逆アルゴリズムを使用して新しいデータをエンコードおよび暗号化します。ここでshowpassword == yesです。
  3. 新しいデータをCookieに挿入し、ページを再読み込みします。



<?php
function xor_encrypt($in, $k) {
    $key = $k;
    $text = $in;
    $outText = '';
    for($i=0;$i<strlen($text);$i++) {
		$outText .= $text[$i] ^ $key[$i % strlen($key)];
    }
    return $outText;
}
 
$old_data_code = "ClVLIh4ASCsCBE8lAxMacFMZV2hdVVotEhhUJQNVAmhSEV4sFxFeaAw";
$old_data_decode = array( "showpassword"=>"no", "bgcolor"=>"#ffffff");
$new_data_decode = array( "showpassword"=>"yes", "bgcolor"=>"#ffffff");
$KEY = xor_encrypt(base64_decode($old_data_code), json_encode($old_data_decode));
echo "key: ". $KEY . "\n";
$KEY="qw8J";
$new_data_code = base64_encode(xor_encrypt(json_encode($new_data_decode), $KEY));

  • -u "URL"

B：ブールベースのブラインドSQLインジェクション
U：UNIONクエリSQLインジェクション
T：時間ベースのブラインドSQLインジェクション
E：エラーベースのSQLインジェクション
S：スタッククエリSQLインジェクション
