テクノロジーとビジネスの岐路に立つ：ITリーダーがスイスに新しい大学を開設

こんにちは、ハブル！今日は、ITと管理の分野の教育についてお話ししたいと思います。より正確には、世界をリードする科学者と起業家によって設立されたシャフハウゼン工科大学（SIT）についてお話します。新しい教育プロジェクトの開発には、30人を超える科学的専門家が参加しました。その中には、ノーベル賞受賞者、プログラミング言語の作成者、優れた科学者や起業家がいます。このすべてについて、およびSITトレーニングプログラムが発表されるオンライン会議について、カットの下をお読みください。



高等教育制度を改革する必要性について長い間話し続けてきました。そして、COVID-19は、この世界の多くがリモートで実行できることが判明したため、スキルとトレーニングの転送という従来の慣習を終えたと言えます。しかし、市場の真のニーズは数年前から教育プログラムを上回っており、卒業生はすでに自分の実践に追いつき、個人的なバンプを詰め込む必要があります。

さらに、コンピューターサイエンスコースの大部分-はい、プログラミングスキルを習得するのに役立ちますが、卒業生にはリーダーシップのポジションとプロジェクト管理の準備をしません。スイスのシャフハウゼン工科大学は、新世代の人材を育成するために特別に設立されました。大学のプログラムは、情報技術、物理学、ビジネスの交差点における教育と業界の真のニーズとの間のギャップを埋めることを目的としています。そして、私たちの会社が活発な成長の段階にあることを考えると、SITへの初期投資はAcronisからのものであり、Sergey BelousovはSITの創設者でした。

なぜスイスなのか？


研究所の場所は偶然ではありませんでした。シャフハウゼンはチューリッヒから45 kmの場所にあります。つまり、ヨーロッパ最大の技術拠点は居住者が利用できます。すぐ近くにスイスの都市、南ドイツ、北イタリアがあります。しかし、シャフハウゼン自体は独特のビジネス環境を持っています。過去20年間、カントン政府は高度なテクノロジー企業を誘致するという慎重な政策を追求してきました。2008年にこの都市を本部に選んだアクロニスは、シャフハウゼンにオフィスを開設した600人を超えるビジネス担当者の1人です。



チームには誰がいますか？


EUの最も革新的な地域の1つに位置するおかげで、SITはヨーロッパの主要な科学者を魅了しました。合計すると、SITのリーダーシップは30人以上で構成され、全員が著名な科学者、起業家、および先見の明のある人です。これには、2010年のノーベル賞受賞者であるコンスタンティンノボセロフ卿、ハーバード大学物理学教授、ミハイルルキン氏、経済学および公共政策教授、カーネギーメロン大学名誉副学長、マークカムレット氏、ジュネーブ大学ニコラスジザン校の量子情報学教授、言語の作成者が含まれます。高等教育への現代的なアプローチのエッフェルプログラミングと他のイデオロギー。

SIT副学長-Bertrand Meyer教授（エッフェルプログラミング言語の作成者）。マイヤーはソフトウェアエンジニアリングの第一人者であり、イノポリス大学の教授であり、ETHチューリッヒのソフトウェアエンジニアリングの名誉教授でもあります。同時に、Bertrand Meyerは引き続きEiffel Software（California）の創設者兼CTOであり、パリ工科大学、スタンフォード大学、ナンシー大学で学位を取得しており、IT業界における実際の成功と学術開発の組み合わせの優れた例を提供しています。


SITは、ピッツバーグのカーネギーメロン大学およびシンガポール国立大学と提携しています。したがって、SITの教育プログラムは、人工知能、サイバー防御、新素材の開発、量子技術、ビジネスのデジタル化、デジタルスポーツとヘルスケアなどの分野の科学者の国際的なグループの参加によって形成されます。

SIT Virtual Nano Conferenceオンライン会議


2020年5月26日（火曜日）11:00（モスクワ時間）だれでも、サイバー防御、量子コンピューティング、ソフトウェア開発、ITシステムの開発の分野における近代教育の発展について、直接（英語で）学ぶことができます。研究所のスタッフとキュレーター。国際オンライン会議でSIT Virtual Nano Conference。彼らは一般市民に新しい教育モデルを提示し、最初のSIT修士プログラムを発表します。これは主にロシアの大学の学士が興味を持つものです。この会議では、情報セキュリティ、量子コンピューティング、ソフトウェアの開発とテストを確実にし、将来のコンピュータシステムに関する問題を提起します。
