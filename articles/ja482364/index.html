assertは2つの引数を取ることができます


これassertは通常、いくつかの条件をチェックするために使用されAssertionError、条件が満たされない場合にスローされます。ほとんどの場合、チェックは次のようになります。


assert list.isEmpty();

ただし、次のようになります。


assert list.isEmpty() : list.toString();

スマートリーダーは、2番目の式（ちなみに遅延）が、Object渡された型の値を返しAssertionError、エラーに関する追加情報をユーザーに伝えることを既に推測しています。より正式な説明については、言語仕様の対応するセクションを参照してください：https : //docs.oracle.com/javase/specs/jls/se13/html/jls-14.html#jls-14.10

strictfp


これは呪いの言葉ではありません-これはあまり知られていないキーワードです。ドキュメントによると、その使用には浮動小数点数の厳密な計算が含まれます：


public interface NonStrict {
  float sum(float a, float b);
}

腕のフリックでに回すことができます 


public strictfp interface Strict {
  float sum(float a, float b);
}

また、このキーワードは個々のメソッドに適用できます：


public interface Mixed {
  float sum(float a, float b);

  strictfp float strictSum(float a, float b);
}
