Kami mengundang Anda ke DINS JS EVENING: kami berbicara tentang pemrograman berorientasi aspek dan API komposisi Vuejs 3

Kami mengundang pengembang JavaScript dan siapa pun yang tertarik dengan topik ini untuk bergabung dengan DINS JS EVENING mitap online! Kami bertemu pada 29 April pukul 19:00.

Pada pertemuan tersebut, Vitaly Perov dari DINS akan berbicara tentang pemrograman berorientasi aspek di JS, dan Mikhail Kuznetsov dari ING akan mengungkapkan keuntungan dari kerangka API komposisi Vuejs v3 yang baru. Para pembicara akan menunjukkan contoh-contoh praktis dan menjawab pertanyaan dari audiens.

Untuk berpartisipasi, silakan mendaftar.
Baca program terperinci dan informasi tentang speaker di bawah kucing.




Program


19: 00-19: 40 - Pemrograman berorientasi aspek di JS (Vitaly Perov, DINS)
Selama presentasinya, Vitaly akan memperkenalkan pengantar paradigma pemrograman berorientasi aspek dan berbicara tentang ruang lingkupnya. Vitaliy akan menunjukkan contoh kode untuk implementasi dasar AOP dan memberi tahu bagaimana mereka dapat digunakan untuk memecahkan masalah dalam memperkenalkan fungsionalitas baru.
Laporan ini akan relevan bagi pengembang yang tertarik pada solusi arsitektur dan pendekatan baru.
Vitaly Perov adalah pengembang frontend di DINS. Dia memulai karirnya dalam pemrograman sebagai pengembang backend Java dan PHP. Sekarang terlibat dalam pengembangan frontend dalam JavaScript dan TypeScript.

19: 40-20: 20 - Vuejs v3 komposisi API: pembaruan dan keuntungan (Mikhail Kuznetsov, ING)
Versi baru kerangka Vuejs v3 yang populer meninggalkan status beta dalam beberapa bulan mendatang. Michael akan mempertimbangkan esensi dari pendekatan API Komposisi yang ditambahkan dalam v3 dan keuntungan yang diberikannya untuk tugas pengembangan komponen yang khas.
Laporan ini relevan bagi mereka yang bekerja dengan Vuejs 2 dan mengikuti tren pengembangan pengembangan frontend.
Mikhail Kuznetsov, dosen di kursus pengembang Vue.js di Otus, pengembang fullstack di ING International Finance Corporation Mendesain dan mendukung proses pengembangan WEB. Dia telah pemrograman di JS selama sekitar 8 tahun. Dia terlibat dalam mempopulerkan kerangka front-end Vue.js. Dia berbicara di konferensi Frontend Conf Moscow, UtahJS, dll.
