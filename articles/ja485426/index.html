オランダの自動車：2019年の統計と情報

こんにちは、ハブル。

この記事は、オランダのサイクルサイクルサイクルインフラストラクチャの続きです。今回は自動車についてお話します。



最近、あるプロジェクトで、車の番号を認識するカメラからデータにアクセスすることができました。数を知っていると、かなり多くの追加情報を得ることができます。しかし、これらのデータの一部に見えたため、公式に公開されていたオランダの車のデータベース全体を処理することが面白くなりました。

結果は読者と共有するのに十分興味深く思われた。それでも、オランダは生活の質の点でトップ10の国の1つであり、自転車以外のオランダの乗り物を見るのは非常に興味深いものでした。

カットの下に続きました。

初期データ


統計は、ローカルテストと一般データの2つの部分に分かれています。最初のケースでは、プロのAXIS監視カメラがデータソースとして使用され、ナンバープレート認識を統合する機能を備えていました。数値自体は統計にとってそれほど興味深いものではありません。それから取得できる追加情報は、はるかに興味深いものです。そして、これはオランダで簡単に行われます。rdw.nlのWebサイトでは、任意の番号について、単純なGETリクエストを送信することで車の情報を無料で取得できます（たとえば、乱数H789GFを確認できます）。製造年、容量、カタログコストなど、非常に多くのデータが返されます。これは既に興味深いものです。

そして最後に、第2部ではベース全体が考慮されます。チャートを大きくしすぎないようにするため、2019年の情報のみが考慮されます-過去1年間に登録された自動車の最新のブランドとモデル。

それでは始めましょう。

ローカルデータ


上記のように、最初のケースでは、車の番号を認識する機能を持つAXIS監視カメラが使用されました。カメラはオフィスセンターの入り口に設置されています。ここでのデータセットはもちろん小さいですが、ITワーカーを含めることができる一般の人々、中産階級の代表者が使用するマシンからの実際のデータであることに関心があります。トラックもリストに含まれる可能性がありますが、トラックの数はそれほど多くなかったので、そのようなモデルには特別なフィルタリングを行いませんでした。

3日間のカメラ作業で370件の記録が収集され、約100台の車であることが判明しました。これらすべてから、次のようなデータセットが作成されました。

PythonとPandasを使用して抽出できるデータを見てみましょう。コードに興味がない人は、テキストをすぐにグラフまでスクロールできます。

必要なコンポーネントを接続し、データセットをパンダにロードします。

車のモデルに関心があるので、必要なサブセットを選択して重複を削除します（車の通過ごとに、カメラは3〜4個のイベントを生成します）。グラフを表示しやすいように、データセットを車のモデルで並べ替えています。

df_cars = df[['car_id', 'manufacturer', 'model', 'year', 'price']]
df_cars = df_cars.drop_duplicates()

df_cars['year'] = df_cars["year"].map(int)
df_cars['price'] = df_cars["price"].map(get_rounded, na_action=None).astype('Int64')

補助データ抽出関数をヒストグラムの形で記述します。

def extract_as_hist(dataframe, param: str):
    g = dataframe.groupby([param])
    msg_count = g.size().reset_index(name='counts')
    grouped = g.mean().reset_index()
    grouped['counts'] = msg_count['counts']
    return grouped[param].values, grouped['counts'].values

これで、最も興味深い-実際のグラフに進むことができます。

もちろん、100台のマシンのより大きなデータセットに使用することはできません-サンプルサイズが小さすぎます。次のステップに進みましょう。国全体のデータを検討します。

オランダの一般的なデータ


まず、データベース全体をダウンロードする必要があります。これにより、より多くのデータを抽出できるようになります。ファイルはRDWページで入手できます。必要なCSVファイルのサイズは約7 GBです。

ここのコードはほぼ同じです。繰り返さないで、結果に直接移動します。すべてのグラフは2019年のデータに関連しています。それ以外の年については、自分で表示することもできます。オランダの全人口が1700万人であることを考えると、年間で合計67万2千7台の自動車が登録されました。

色
