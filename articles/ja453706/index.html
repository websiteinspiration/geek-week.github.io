Google Cloud Professional Data Engineer認定試験に合格した方法

推奨される3年間の実務経験なし


データエンジニア コースのクラスの開始を予測して、おそらく将来のエンジニアにとって役立つ1つの非常に興味深いストーリーの翻訳を共有します。行こう！


Googleフーディ：着用。真剣に働く表情：プレゼント。YouTubeでのこの記事のビデオ版の写真。
注意。この記事は、2019年3月29日までのGoogle Cloud Professional Data Engineer認定試験に関するものです。この日付以降、いくつかの変更がありました。「詳細」セクションに含めました。

それで、あなたは私のカバーのように、新しいパーカーが欲しいですか？または、Google Cloud Professional Data Engineer証明書の取得を計画していて、その方法を考えていますか。

過去数か月間、私はGoogle Cloudを使用してコースを受講し、プロのデータエンジニアのための試験の準備をしました。それから私はそれを通過しようとし、それを通過しました。そして数週間後、私のパーカーが配達されました。証明書が速くなりました。

この記事では、学びたいことがいくつかあり、Google Cloud Professional Data Engineer証明書を取得するために行った手順を示します。

Google Cloud Professional Data Engineerの資格を取得したいのはなぜですか？


データはどこにでもあります。また、データを処理して使用できるシステムを作成する方法に関する知識が必要です。 Google Cloudは、これらのシステムを構築するためのインフラストラクチャを提供します。

あなたはすでにGoogle Cloudを使用するスキルを持っているかもしれませんが、これを将来の雇用主またはクライアントにどのように実証しますか？プロジェクトのポートフォリオまたは認証の2つの方法があります。

この証明書は、見込み客と雇用主に次のように伝えます。「私にはスキルがあり、認定を受けるための努力をしました。」

グーグルからの簡単な説明が要約されます。

データ処理システムを設計および構築する能力と、Google Cloud Platformで機械学習モデルを作成する能力を示します。

スキルがまだない場合は、認定トレーニング資料に慣れることで、Google Cloudで世界クラスのデータ処理システムを作成する方法についてすべて学ぶことができます。

Google Cloud Professional Data Engineerの対象となるのは誰ですか？


あなたは数字を見ました。クラウドは成長しています。それはすでにここにあり、どこにも行きません。数を見ていないのであれば、信じてください、クラウドは成長しています。

すでにデータスペシャリスト、データエンジニア、データアナリスト、機械学習エンジニア、またはデータの世界でのキャリアパスをお探しの場合は、Google Cloud Professional Data Engineer認定資格が最適です。

クラウドコンピューティングを使用する機能は、あらゆるデータ指向の地位の要件となっています。

優れたデータエンジニア/データセット/機械学習エンジニアになるには、証明書が必要ですか？


番号。

証明書がなくても、Google Cloudを使用してデータ転送ソリューションを操作できます。

証明書は、既存のスキルを検証するための1つの方法にすぎません。

それはどれくらいしますか？


試験の費用は200ドルです。あなたが失敗した場合、あなたは再び新しい試みのために支払う必要があります。

準備コースとプラットフォーム自体の使用に関連する考えられるコスト。

プラットフォーム費用は、Google Cloudサービスの使用料です。あなたが洗練されたユーザーであれば、あなたはすでにこれに気づいています。そうでない場合は、この記事で説明されているトレーニング資料に慣れているだけで、新しいGoogle Cloudアカウントを作成して、登録時にGoogleが提供する300ドルの制限を満たすことができます。

すぐにコースの費用に移ります。

認定はどのくらいの期間有効ですか？


2年。その後、もう一度試験を受ける必要があります。

また、Google Cloudは日々進化しているため、証明書に必要なものは変わる可能性があります（気付いたように、この記事を書き始めた時点ですでに変更されています）。

試験の準備には何が必要ですか？


Googleは、専門的な認定を受けるために、3年以上の業界経験と、GCPを使用したソリューションの開発と管理の1年以上を推奨しています。

上記のどれも持っていませんでした。

強さから6か月の関連経験。不足を補うために、オンライントレーニングリソースを組み合わせて使用​​しました。

あなたが私のようで、推奨される要件がない場合は、以下のコースのいくつかを受講して資格を向上させることができます。

試験後


試験に合格すると、合格または不合格の結果のみが得られます。少なくとも70％を目指して努力することをお勧めします。したがって、実際の試験では最低90％を目指しました。

これが完了すると、メールでクーポンコードと公式のGoogle Cloud Professional Data Engineer証明書が届きます。おめでとう！
