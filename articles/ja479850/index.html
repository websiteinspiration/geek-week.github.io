Windows Server Core 2019にExchange 2019をインストールする

Microsoft Exchangeは、レターの受信と処理、メールサーバーのWebインターフェース、企業のカレンダーとタスクへのアクセスを含む大きなプロセッサです。ExchangeはActive Directoryに統合されているため、すでに展開されていると想像してください。

まあ、Windows Server 2019 CoreはグラフィカルインターフェイスのないWindows Serverのバージョンです。

このバージョンのWindowsには、従来のWindowsはなく、クリックするものも、スタートメニューもありません。黒いウィンドウと黒いコマンドラインのみ。しかし同時に、誰も重要なシステムを掘り下げたくないので、攻撃の領域が狭くなり、エントリのレベルが高くなります。 

このガイドは、GUIを備えたサーバーにも適しています。


1.サーバーに接続する


Powershellを開き、次のコマンドを入力します。

Enter-PSSession 172.18.105.6 -Credential Administrator

オプション：RDPをオンにします。これにより、インストールが簡略化されますが、まったくそうではありません。

cscript C:\Windows\System32\Scregedit.wsf /ar 0

Ultravdsの画像には、RDPがすでに含まれています。

2.サーバーをADに接続します


これは、Windows管理センターまたはRDPのSconfigを使用して実行できます。

2.1 DNSサーバーまたはドメインコントローラーの指定 



Windows管理センターでサーバーに接続し、ネットワークセクションに移動して、ドメインコントローラーまたはDNSドメインサーバーのIPアドレスを指定します。


RDPを使用して、コマンドラインに「Sconfig」と入力し、青色のサーバー構成ウィンドウに移動します。そこで、8）ネットワーク設定を選択し、DNSドメインサーバーを指定して同じことを行います。

2.2サーバーをドメインに参加させる



WACで[コンピューターIDの変更]をクリックすると、ワークグループまたはドメインを選択するための使い慣れたウィンドウが表示されます。すべてが通常どおり、ドメインを選択して参加します。


Sconfigを使用して、最初に項目1を選択し、ワークグループまたはドメインのどちらに参加するかを選択し、ドメインに参加する場合はドメインを指定する必要があります。また、手順の完了後にのみサーバー名を変更できますが、その場合でもパスワードを再入力する必要があります。

Powershellを使用すると、さらに簡単になります。

Add-Computer -DomainName test.domain -NewName exchange  -DomainCredential Administrator

3.インストール




RDPを使用する場合は、Exchange自体をインストールする前に、必要なコンポーネントをインストールする必要があります。

Install-WindowsFeature Server-Media-Foundation, RSAT-ADDS

次に、Exchangeインストーラーでディスクイメージをダウンロードする必要があります。

Invoke-WebRequest -UseBasicParsing -Uri 'https://website.com/ ExchangeServer2019-x64.iso -OutFile C:\Users\Administrator\Downloads\ExchangeServer2019-x64.iso

ISOをマウント：

Mount-DiskImage C:\Users\Administrator\Downloads\ExchangeServer2019-x64.iso

これをすべてコマンドラインで行う場合は、ダウンロードしたディスクをマウントして次のコマンドを入力するだけです。

D:\Setup.exe /m:install /roles:m /IAcceptExchangeServerLicenseTerms /InstallWindowsComponents

結論


ご覧のとおり、Windows Server CoreにExchangeをインストールすること、およびドメインに入るのは簡単なプロセスではありません。セキュリティでどのように成功したかを考えると、価値がありました。

Powershellを使用してADにサーバーを入力する方が、GUIまたはWindowsアドミンセンターを使用するよりも簡単であることを特に嬉しく思います。

ExchangeインストールオプションがExchange 2019にのみ追加されたのは残念です。
