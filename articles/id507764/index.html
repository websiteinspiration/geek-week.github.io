Bagaimana saya melakukan sendiri ABP untuk generator





Beberapa tahun yang lalu saya membuat sendiri saklar transfer otomatis (input cadangan otomatis) untuk bekerja di negara ini dari generator. Sekarang banyak pekerja TI pindah ke lokasi terpencil, bekerja dari pondok, di mana kualitas catu daya dapat meninggalkan yang terbaik. Oleh karena itu, saya memutuskan untuk menulis tentang pengalaman saya dengan ABP buatan rumah pada mikrokontroler ATmega8A. Jika topiknya menarik, selamat datang di kat, akan ada banyak huruf dan kode.



Tentang pentanahan



Sebelum Anda melakukan sesuatu dengan listrik, Anda harus berhati-hati agar tidak memiliki landasan yang baik di rumah Anda. Hanya saja tidak mungkin untuk mengambil dan menghubungkan generator gas / diesel / gas rumah tangga biasa ke catu daya di rumah. Kewaspadaan harus diperhatikan. Yang pertama adalah generator Anda harus di-ground dengan baik. Maka Anda memiliki peluang bagus untuk tidak tersengat listrik ketika statis dari sweter favorit Anda menerobos insulasi lilitan generator. Secara umum, generator yang sedang berjalan sebaiknya tidak disentuh.



Perlu diingat bahwa jaringan tidak selalu 220V. Beralih pada saluran, debit petir di kejauhan, pelepasan statis memberikan pickup sedemikian rupa sehingga pulsa pendek beberapa kilovolt sering terjadi dalam jaringan. Mereka berjuang dengan ini dengan memasang arester dan SPD di pintu masuk rumah, tetapi ini adalah praktik yang sangat langka di Federasi Rusia. Jadi biarkan percikan masuk ke bumi, dan bukan melalui Anda - buat landasan yang baik di seluruh rumah. Tanpa ini, Anda tidak dapat melakukan apa pun lebih jauh!



Tentang generator



Ngomong-ngomong, bagi banyak generator bensin rumah tangga, belitan tidak terhubung dengan tanah. Dan ini sangat normal ketika Anda menggunakan satu alat listrik dari generator. Tetapi ketika Anda perlu menghubungkan generator ke rumah, Anda perlu membuat kabel netral (N) dan kabel fase (L). Untuk melakukan ini, salah satu terminal generator di-ground-kan dan dari titik ground ini sudah perlu untuk membawa dua kabel ke dalam rumah secara independen - satu akan menjadi N netral, dan yang kedua adalah bumi pelindung (PE). Ketika memilih generator, Anda perlu memperhatikan apakah mungkin untuk membumikan outputnya, kadang-kadang dilarang dalam instruksi untuk generator, maka generator seperti itu tidak cocok untuk Anda.



Seringkali di Jaringan Anda dapat melihat diagram koneksi generator tanpa membumikan dan memisahkan garis N dan PE. Jangan lakukan ini, Anda akan hidup lebih lama. Skema semacam itu bekerja dengan baik sampai keadaan pertama yang tidak menguntungkan. Catu daya khas untuk perangkat elektronik modern termasuk kapasitor dari L, N line ke ground. Jika N tidak terhubung ke generator, maka karena kapasitor ini di jalur N akan ada, jika Anda beruntung, 110 volt relatif terhadap tanah. Omong-omong, banyak boiler gas dalam mode ini umumnya berhenti bekerja. Tentang pengaruh statika tanpa kehadiran landasan, saya sudah menulis di atas.



Tentang skema ABP



Ada beberapa skema berbeda untuk mengimplementasikan ATS. Selanjutnya saya akan menulis tentang rangkaian teraman ATS satu fase dari sudut pandang saya. Saya tidak merekomendasikan hemat melakukan ABP pada satu kontaktor atau hanya dengan mengganti satu kabel fase. Hanya dengan netral.







Dalam diagram di atas, daya dari jaringan dan dari generator disuplai melalui input 1 dan 2. Mereka dilindungi oleh mesin berpasangan. Melalui mesin tambahan, switching dan sirkuit indikasi diumpankan. Dapat dilihat bahwa kumparan relai saling bertautan secara elektrik. Untuk penyertaan input tertentu, untuk kesederhanaan, mikrokontroler, tidak ditampilkan dalam diagram, yang menutup rangkaian pada titik switching TK1 atau TK2, bertanggung jawab.



Poin mendasarnya adalah kehadiran dalam skema pemblokiran 2x ATS - pemblokiran mekanis mutual input switching kontaktor dan pemblokiran listrik mutual kontaktor. Dibuat sendiri demi ekonomi, itu terjadi, dalam desain mereka mengabaikan kunci ini, tetapi sia-sia. Sirkuit tanpa pemblokiran mungkin berfungsi untuk beberapa waktu, tetapi pada titik tertentu kontak akan terbakar, pegas yang kembali akan melemah dan korsleting akan terjadi di antara input. Pertama, itu mengancam broads besar jika kedua jalur diberi energi, tetapi ini bukan masalah terbesar. Adalah jauh lebih penting bahwa generator Anda, secara tak terduga untuk listrik yang memperbaiki kabel, dapat memasok tegangan ke jaringan umum - dalam keadaan yang tidak menguntungkan, listrik yang memperbaiki saluran dapat mati. Ini sudah menjadi artikel kriminal untuk Anda.



Tentang kontaktor



Dengan demikian, penggunaan relay konvensional menghilang bagi kami, hanya kontaktor khusus yang cocok. Untuk kapasitas besar, masih ada opsi dengan drive bermotor, tetapi mahal dan berlebihan untuk penggunaan rumah biasa.



Untuk membuat interlock mekanis, Anda harus memilih kontaktor yang dapat bekerja berpasangan. Pemblokiran timbal balik biasanya dicapai dengan memasang kontaktor yang sama di samping satu sama lain dan memasang opsi tambahan - interlock mekanis. Itu dijual terpisah dari kontaktor dan biaya satu sen.



Saling saling bertautan listrik dimungkinkan jika kontaktor memiliki kontak pensinyalan tambahan yang beroperasi pada pembukaan. Kadang-kadang mereka langsung dibangun ke kontaktor, kadang-kadang mereka dapat dibeli dan diinstal sebagai opsi.



Produsen kontaktor terkemuka memiliki peralatan seperti itu di jalurnya. Jadi mencari dan membeli kit tidak sulit. Benar, harga kontaktor bermerek jauh lebih tinggi daripada kita / Cina. Karena jumlah siklus switching tidak diharapkan besar, pilihan kontaktor Cina dibenarkan. Kerugian hanya dapat dikaitkan dengan fakta bahwa kumparan kontaktor selama operasi cukup banyak buzz.



Lebih lanjut tentang daya yang diaktifkan. Kontak atau kontak harus menahan daya maksimum yang diperbolehkan untuk dikonsumsi di rumah. Saya memilikinya 10 kW, jadi saya memilih kontaktor untuk arus yang diizinkan melalui satu kontak pada sekitar 50 ampere. Perlu dicatat bahwa untuk beberapa alasan daya sakelar untuk kontaktor tiga fase yang khas ditunjukkan dalam total paspor untuk ketiga fase, jadi Anda perlu hati-hati melihat arus yang diizinkan melalui satu kontak.

Setiap sumber memiliki pembagi tegangan resistif, yang dilindungi terhadap tegangan lebih oleh dioda zener - digunakan untuk menghitung tegangan jaringan menggunakan mikrokontroler ADC menggunakan perhitungan sederhana.



Untuk mengganti koil kontaktor, sirkuit lembar data standar digunakan untuk mengontrol tujuh sel. 2 buah ). Kumparan adalah beban induktif, sehingga sirkuit snubber pada output dari resistor dan kapasitor diperlukan.

Dari fitur-fiturnya, TL431 juga digunakan sebagai pembangkit tegangan referensi. Kalau tidak, semuanya dimasukkan sebagai standar untuk Atmega 8. Ada LED untuk menunjukkan keberadaan tegangan suplai pada input dan satu LED status perangkat. Rangkaian ini clock menggunakan kuarsa 16 MHz eksternal.



Elang memberiku papan sirkuit tercetak seperti itu. Tidak ada SMD, triac, atau stabilizer dengan radiator yang ringan.







Dua transformator toroidal dipasang langsung di papan. Papan sirkuit diproduksi dengan cara radio amatir tradisional menggunakan photoresist. Setelah instalasi, ditutup dengan tiga lapis pernis akrilik. Saya berharap tegangan tinggi tidak akan pecah.

void abp_timer_start( ABP_TMRS tmr ) {
   abp_timers[tmr].passed_secs = 0;
   abp_timers[tmr].state = TMR_ON;    
}

//   
void abp_timer_stop( ABP_TMRS tmr ) {
   abp_timers[tmr].state = TMR_OFF;
   abp_timers[tmr].passed_secs = 0;
}

//   
unsigned char abp_timer_check( ABP_TMRS tmr ) {
   if (abp_timers[tmr].passed_secs >= abp_timers[tmr].set_secs) {
      return 1; 
   } else {
      return 0;
   }
}

// COUNTER2      led_State
void counter2_init( void )
{
   ASSR = 0;  /* AS0 = 0 */  /* disable asynchronous mode */
   while (ASSR); /*EMPTY*/

   OCR2 = 223;             /* 70    */

   TCCR2 |= (1 << CS22) | (1 << CS21) | (1 << CS20);  /* prescale /1024 */
   
   TCCR2 |= (1 << WGM21);                /* mode CTC */
   TCCR2 &= ~(1 << WGM20);
   
   TCCR2 &= ~(1 << COM21);              /*    OC2 */
   TCCR2 &= ~(1 << COM20);
   
   TIMSK |= (1 << OCIE2);          /* enable compare interrupt */
}

void ADC_init() // ADC1,0
{
   //   2,5, 10 bit 
   ADMUX = (0 << REFS0) | (0 << REFS1) | (0 << ADLAR) |
   (0 << MUX3) | (0 << MUX2) | (0 << MUX1) | (0 << MUX0); // ADC0
   // , free running,  
   ADCSRA = (1 << ADEN) | (1 << ADFR) | (1 << ADIE) |
   (1 << ADPS2) | (1 << ADPS1) | (1 << ADPS0); //  128
   
   ADCSRA |= (1 << ADSC);             // Start ADC Conversion
}

ISR(ADC_vect) //     2500    
{
   if ((ADMUX & (1 << MUX0))) { //    ADC1
      if (sampleCount++) //   
         tempVoltage1 += ADC;
      if (sampleCount >= SAMPLES) {
         sampleCount = 0;
         sumVoltage1 = tempVoltage1;
         tempVoltage2 = 0;
         tempVoltage1 = 0;
         ADMUX &= ~(1 << MUX0); //   ADC0
      }        
   } else { //    ADC0
      if (sampleCount++) //   
         tempVoltage2 += ADC;
      if (sampleCount >= SAMPLES) {
         sampleCount = 0;
         sumVoltage2 = tempVoltage2;
         tempVoltage2 = 0;
         tempVoltage1 = 0;
         ADMUX |= (1 << MUX0); //   ADC1
      }  
   }
   ADCSRA |=(1 << ADIF);              // Acknowledge the ADC Interrupt Flag
}

void validate220() {
   //       
   realV1 = DIVIDER1 * ((sumVoltage1 * REFERENCEV) / 1024) / SAMPLES;
   realV2 = DIVIDER2 * ((sumVoltage2 * REFERENCEV) / 1024) / SAMPLES;
   
   if( realV1 > MAX_V || realV1 < MIN_V ) { //    
      u220g = U_INVALID;
      } else {
      u220g = U_VALID;
   }
   if( realV2 > MAX_V || realV2 < MIN_V ) { //    
      u220n = U_INVALID;
      } else {
      u220n = U_VALID;
   }
}

      && abp_timers[TMR_220G_OFF].state == TMR_ON && abp_timer_check(TMR_220G_OFF)) { //  ,   ,  
         abp_timer_stop( TMR_220G_OFF ); //    
         abp_timer_stop( TMR_220G_ON ); //    
         contactors[RLY_220G] = RLY_OFF; //    
      }
      if( u220g == U_INVALID && contactors[RLY_220G] == RLY_OFF
      && abp_timers[TMR_220G_ON].state == TMR_ON) { //  ,  ,    
         abp_timer_stop( TMR_220G_ON ); //    
      }
      
      //      
      if( contactors[RLY_220N] == RLY_OFF && contactors[RLY_GEN] == RLY_OFF) { //  ,  
         abp_timer_stop( TMR_GEN_OFF ); //    
         contactors[RLY_GEN] = RLY_ON; //    
      }
      if( contactors[RLY_220N] == RLY_ON && abp_timers[TMR_GEN_OFF].state == TMR_OFF 
      && u220g == U_VALID) { //  220      
         abp_timer_start( TMR_GEN_OFF ); //    
      }
      if( contactors[RLY_220N] == RLY_ON && abp_timers[TMR_GEN_OFF].state == TMR_OFF
      && u220g == U_INVALID) { //  220      
         abp_timer_stop( TMR_GEN_OFF ); //    
         contactors[RLY_GEN] = RLY_OFF; //    
      }
      if( contactors[RLY_220N] == RLY_ON && abp_timers[TMR_GEN_OFF].state == TMR_ON
      &&  abp_timer_check(TMR_GEN_OFF) ) { //  220      
         abp_timer_stop( TMR_GEN_OFF ); //    
         contactors[RLY_GEN] = RLY_OFF; //    
      }
}

void switch_contactors() {
   //   
   if( contactors[RLY_220N] == RLY_ON ) {
      PORTD |=  (1 << PD4); //   -   1 
       _delay_ms(50); //      
      PORTD &= ~(1 << PD3); //   -   2 220
      led_State = ABP_2RELAY;
   } else {
      if( contactors[RLY_220G] == RLY_ON ) {
         PORTD |=  (1 << PD3); //   -   2 220
         PORTD &= ~(1 << PD4); //   -   1 
         led_State = ABP_1RELAY;
      } else {
         PORTD |=  (1 << PD3); //   -   2 220
         PORTD |=  (1 << PD4); //   -   1 
         led_State = ABP_UNDEF;
      }
   }
   if( contactors[RLY_GEN] != RLY_ON ) { // v2       -     
      PORTD &= ~(1 << PD2); //   -   -     -  
   } else {
      PORTD |= (1 << PD2); //   -   -    
   }
}

Program ini dikembangkan menggunakan AVR Studio gratis dan menggunakan pustaka AVR standar.



Pada loop utama, program memeriksa voltase pada input input, mengevaluasi keadaan switching pada kontaktor, memperhitungkan operasi penghitung waktu program, membuat penyesuaian yang diperlukan dengan menghidupkan atau mematikan relay dan kontaktor, kemudian beralih ke hibernasi. Untuk debugging, kesimpulan dari pencetakan debugging ke port serial mikrokontroler dibuat.



Untuk mengontrol pembekuan, timer anjing penjaga disediakan.



Semua siklus pengukuran dilakukan pada interupsi dan menggunakan pengatur waktu perangkat keras. Penghitung kedua diatur pada penghitung waktu 1. Setelah penghentian penghitung waktu 1, penghitung waktu peranti lunak diperbarui, yang bertanggung jawab atas penundaan dan mematikan kontaktor dan relay generator.

Timer kedua digunakan untuk membuat efek kedip LED status. Ada tiga pola berkedip. Nilai-nilai dari pola berkedip diambil dalam penghentian penghitung waktu 2. Dengan berkedip, Anda dapat menilai status pengontrol.



Dua ADC juga bekerja pada timer dan rata-rata 2500 sampel pengukuran tegangan. Untuk mengkonversi pengukuran ke volt nyata, konstanta kalibrasi disediakan. Nilai-nilai mereka harus dikoreksi selama konfigurasi ATS.



Selain itu, ada sejumlah konstanta yang perlu ditentukan saat commissioning.
