カリプソ作戦：新しいAPTグループが世界中の政府機関を攻撃

Positive Technologies Security Center（PT Expert Security Center）の専門家は、 Calypsoと呼ばれるAPTグループを特定しました。このグループは2016年から運営されており、政府機関を対象としています。現在、6カ国で事業を展開しています。



攻撃中にサイバー犯罪者がWebシェルに送信したチーム

専門家によると、インド（被害者の34％）、ブラジル、カザフスタン（各18％）、ロシア、タイ（各12％）、トルコ（6 ％）。攻撃者はネットワークの境界に侵入し、侵害された組織の内部ネットワークにアクセスするための特別なプログラムを配置しました。

ハッカーはどうやって


調査が示したように、攻撃者はリモートコード実行の脆弱性MS17-010を悪用するか、盗まれた資格情報を使用して、ネットワーク内を移動しています。



悪意のあるソフトウェアの一般的な構造とそのインストール

このグループの攻撃の成功は、ネットワーク内で前進するために使用されるユーティリティのほとんどが、ネットワーク管理のために世界中の専門家によって広く使用されているという事実によって主に促進されます。グループ化では、SysInternals、Mimikatz、EternalBlue、EternalRomance などの公開ユーティリティとエクスプロイトを使用しました。犯罪者は一般的なエクスプロイトを使用して、組織のローカルネットワーク上のコンピューターに感染し、機密データを盗みます。

組織は、ディープトラフィック分析の特殊なシステムを使用してこのような攻撃を防ぐことができます。これにより、ローカルネットワークへの攻撃者の侵入の初期段階で疑わしいアクティビティを計算し、攻撃者が会社のインフラストラクチャに足場を固めるのを防ぐことができます。さらに、情報セキュリティイベントを監視し、境界とWebアプリケーションを保護することで、攻撃を検出してそれらに対抗することができます。

得られたデータによると、特定されたAPTグループはおそらくアジアにルーツがあり、中国の話者の1人です。



攻撃の1つ中に、不注意により、攻撃者が誤ってプロキシサーバーを構成し、それによってChina Telecomプロバイダーに属する実際のIPアドレスが発行されました

攻撃の1つでは、このグループは、中国の多くのAPTグループで伝統的に使用されているPlugXマルウェアと、2017年にSongXYマルウェアキャンペーンで使用されたByeby Trojanを使用しました。さらに、個別の攻撃中に、攻撃者は中国のプロバイダーに属している実際のIPアドレスを誤って公開しました。

結論


グループはすでにいくつかのハックを成功させていますが、ミスを犯してその起源を判断することができます。提示されたすべてのデータによると、グループはアジアから来ており、これまで知られていなかったマルウェアを使用しています。Byebyトロイの木馬は、このグループを以前に発見したSongXYグループに接続します。そのピークは2017年に発生しました。

私たちはカリプソグループの活動を注意深く監視し続け、その参加により新しい攻撃を予測します。
