Analisis Data Koronavirus SARS-CoV-2 (2019-nCov)

Baru-baru ini, saya melihat banyak rilis berita tentang virus baru, tetapi apakah itu sangat buruk, memiliki epidemi mencapai puncaknya atau masih di depan. Sebagian besar, sulit untuk menarik kesimpulan apa pun dari rilis berita. Pada akhirnya, saya memutuskan untuk mencari tahu sendiri, dan inilah yang terjadi.



Untuk memahami situasinya, saya perlu mendapatkan data untuk dianalisis, saya berhasil menemukannya di situs web Organisasi Kesehatan Dunia (WHO) . Data diterbitkan sebagai laporan pdf untuk setiap hari mulai 21 Januari 2020. Tidak mungkin menulis parser universal, karena semua laporan berbeda, saya harus menyalin data dari mereka secara manual.
