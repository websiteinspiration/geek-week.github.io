SCSS IntelliSense


→ IntelliSense SCSSページ

SCSSで変数を作成するのは簡単で楽しいです。それらを覚えるには？ここでは通常、これが困難になります。

私のように見える場合_variables.scss、朝3時に頭に浮かんだ数百ドルのアイコンと変数名が入ったファイルがあります。SCSS IntelliSense拡張機能は、プロジェクトコードで作業するときに変数名のヒントを提供します。また、これらの名前を完全に覚えておく必要がなくなりました。それらを呼び出すために変数名を含むファイルをあちこち探す必要はありません。

タグの自動名前変更


→ タグの自動名前変更ページ

使用するフロントエンドフレームワークに関係なく、HTML（またはJSX）の記述は面倒な作業になる場合があります。特に、開始タグと終了タグの間にコードが50行あるような状況では。タグH2をに変更する必要がある場合がありH3ます。おそらく、交換する必要があったdivとはspan。いずれにせよ、補助ツールを使用せずにこのような問題を解決すると、大規模なコードフラグメントでは、終了タグの検索が難しくなります。

タグを自動的に編集するために車を呼んでみませんか？これを行うには、Auto Rename Tag拡張機能を使用します。開始タグを編集すると、自動的に変更されて終了します。終了タグが変更されると、同じことがまったく逆に起こります。

重複したアクション


→ 複製アクションページ

右クリックメニューを使用して複製ファイルとフォルダーを作成する機能は非常に便利なので、VS Codeに組み込む必要があります。しかし、この編集者は、そのような機会がなくて素晴らしいので、そのような欠陥を許すことができます。

実際、VS Codeに上記の機能を装備するには、Duplicate Action拡張機能をインストールするだけで十分です。オンにすると、開発者はショートカットメニューコマンドを使用できますDuplicate file or directory。

コードストリーム


→ CodeStreamページCodeStream

について簡単に説明すると、この拡張機能は将来のコードコメントシステムに相当すると言えます。

多くのプログラマーはIDEを離れたくない。しかし、同僚に何かを説明したり、自分で質問したりするには、使い慣れた環境から出て、Trello、Slack、Asana、Bitbucket、Microsoft Teamsを使用する必要があります。CodeStreamは多くのツールとIDEをサポートしているため、コードエディターの使い慣れた環境を離れることなく、問題や質問に取り組むことができます。仕事をしている人の多くは、いわゆる「流動状態」に陥る傾向があります。通常、エディターを終了すると、この状態は解除されます。CodeStreamのおかげで、使い慣れた環境を離れることなく、コードに関する質問を解決できます。

夜更かしをする人


→ Night OwlページNight Owl

は、IDEのためにこれまでに作成された最高のテーマの1つです。今から3年間、毎日使っています。以下は、実際の作業画面のスクリーンショットです。このトピックのタイトルは、夜間にコンピュータの前に座る人を対象としていることを示唆しています。確かに、夜間の作業に最適で、目の疲れを防ぎます。また、使用される柔らかい青色のトーンは、複雑な問題を穏やかに解決するのに役立ちます。
