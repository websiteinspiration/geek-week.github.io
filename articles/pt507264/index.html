Novos recursos no Python 3.9

Uma revisão dos melhores recursos incluídos na mais recente iteração do Python.



Chegou a hora, o lançamento de uma nova versão do Python é inevitável. Atualmente, está na versão beta (3.9.0b3), mas em breve veremos a versão completa do Python 3.9.

Alguns dos recursos mais recentes são incrivelmente interessantes e será incrível ver seu uso após o lançamento. Vamos considerar o seguinte:

  • Operadores de combinação de dicionário
  • Dica de tipo
  • Dois novos métodos de string
  • Novo Python Parser é muito legal

Vamos primeiro olhar para os novos recursos e como vamos usá-los.

Combinação de dicionário


Um dos novos e já meus recursos favoritos com sintaxe. Se temos dois dicionários ae bprecisamos combinar, agora usamos os operadores de união.

Temos o operador de mesclagem “ |”:

a = {1: 'a', 2: 'b', 3: 'c'}
b = {4: 'd', 5: 'e'}
c = a | b
print(c)

[Out]: {1: 'a', 2: 'b', 3: 'c', 4: 'd', 5: 'e'}

E a instrução de atualização " |=", que atualiza o dicionário de origem:

a = {1: 'a', 2: 'b', 3: 'c'}
b = {4: 'd', 5: 'e'}
a |= b
print(a)

[Out]: {1: 'a', 2: 'b', 3: 'c', 4: 'd', 5: 'e'}

Se nossos dicionários tiverem uma chave comum , um par de valores-chave do segundo dicionário será usado:

a = {1: 'a', 2: 'b', 3: 'c', 6: 'in both'}
b = {4: 'd', 5: 'e', 6: 'but different'}
print(a | b)

[Out]: {1: 'a', 2: 'b', 3: 'c', 6: 'but different', 4: 'd', 5: 'e'}

Atualização de dicionário usando iterações


Outro comportamento interessante do operador " |=" é a capacidade de atualizar o dicionário com novos pares de valores-chave usando um objeto iterativo - por exemplo, uma lista ou gerador:

a = {'a': 'one', 'b': 'two'}
b = ((i, i**2) for i in range(3))
a |= b
print(a)

[Out]: {'a': 'one', 'b': 'two', 0: 0, 1: 1, 2: 4}

Se tentarmos repetir o mesmo com o operador de união padrão " |", obtemos TypeError, porque ele só permitirá junções entre tipos dict.

Métodos de String


Não é tão espetacular quanto os outros novos recursos, mas vale a pena mencionar, pois pode ser útil. Dois novos métodos de sequência foram adicionados para remover prefixos e sufixos:
