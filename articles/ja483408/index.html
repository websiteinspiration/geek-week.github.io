Jupyter Notebookで5分で生産性を向上

まず、メインのホットキーを繰り返しましょう。それらをまだ使用していない場合は、必ず開始してください。長期的には、勉強する時間は何度も報われるでしょう。


0.キーのホットキー


  • Esc：実行モードと編集モードを切り替えます
  • A：上に空のセルを追加する
  • B：空のセルを下に追加する
  • DD：細胞除去
  • C：セルをコピー
  • X：細胞切断
  • V：セルを挿入

1.ノートブックを再起動します


再起動するには、ESC + 00を押します。



2.スクロールするメモ帳


スペースを押すとメモ帳が下にスクロールし、Shift +スペース -上にスクロールします。



3.セルの削除をキャンセルする


セルの削除をすばやくキャンセルするには、ESC + Zキーを押します 



4.細胞分裂


1つのセルにコードを書き始めると、後でコードを別のセルに分離する必要に常に直面しています。手動でセルを作成してテキストをコピーする代わりに、2番目のセルが開始する行の前にカーソルを置き、Ctrl + Shift +-を押し  ます。 



5.すべての操作の出力


デフォルトでは、セルが実行されると、最後の操作の結果のみが表示されます。



すべてを印刷するには、このコードをコピーしてノートブックで実行します。


from IPython.core.interactiveshell import InteractiveShell
InteractiveShell.ast_node_interactivity = "all"


このアプローチの欠点は、表示される情報が多すぎることです。これを回避するには、行の最後にセミコロンを追加します。その出力は不要です。


6.セルの実行をスキップ

7.現在実行中のセルに移動します。


ノートブックでこのコードを実行すると、Alt + Iを押して、現在実行中のセルにすばやく移動できます

Jupyter.keyboard_manager.command_shortcuts.add_shortcut('Alt-I', {
    help : 'Go to Running cell',
    help_index : 'zz',
    handler : function (event) {
        setTimeout(function() {
            // Find running cell and click the first one
            if ($('.running').length > 0) {
                //alert("found running cell");
                $('.running')[0].scrollIntoView();
            }}, 250);
        return false;
    }
