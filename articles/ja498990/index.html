Omega Redエミュレーターのゲームパッドサポート

読者のみなさん、こんにちは！

PlayStation 1、PlayStation 2、PlayStation Portable-Omega Redのソフトウェアエミュレーターのテーマを続けます。詳細に：

  • Omega Red Emulator用の新しいPS1グラフィックレンダラー
  • Omega Red + PS1 Emulator = Kojima Genius
  • PS2 / PSPエミュレーター+ゲームプレイストリーミング（YouTube、Facebook、Twitch）=新しいバージョンのOmega Red
  • PS2 / PSPエミュレータ+ Googleドライブ+ YouTube =「狂気」が続く
  • PlayStation 2とPlayStation Portable（スポイラー-1つのエミュレーターに統合）を友達にした方法
  • Red Omega PS2エミュレータ

この出版物は、私のOmega RedプロジェクトにXInputゲームパッドのサポートを含めることに関連して作成しました。

キャットへようこそ。


当初、私はMicrosoft Surface 3のプロジェクトを開発していました-はい、プラットフォームは弱く、i5は指を燃やしますが、驚いたことに、私が興味を持っている古いゲームが起動し、うまくいっていなかったことがわかりました。タッチパッドをエミュレートするためのコードを使用していたので、プロジェクトの早い段階でゲームパッドのサポートを追加することにしましたが、つい最近になって、まともな解決策を得ることができました。

タッチパッドプロジェクトにゲームパッドを追加したのはなぜですか？プログラミングの芸術への愛情。

ゲームコントローラのタイプの選択は、[ゲームコントローラ]メニューから行うことができます。

接続のタイプが認識され（有線およびワイヤレス）、フィードバック振動のサポートと、ゲームパッドの現在のアクティビティの表示。

ゲームパッドはボタンの組み合わせをサポートしており、迅速な保存と迅速な読み込みが可能です。






さらに、PS1レンダラーのテクスチャをシャープにするサポートも含めました。





現在、プログラムはOmega Redで入手でき、GitHub：OmegaRedで提供されています。
