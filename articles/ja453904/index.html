「vk.com/away.php」を取り除くか、健康な人のリンクをたどる

Vkontakteに投稿されたリンクをクリックすると、他のソーシャルネットワークと同様に、最初に「安全な」リンクへの移行が発生し、その後ソーシャルネットワークがユーザーをさらに進めるかどうかを決定します。ほとんどの注意深い人々は、ブラウザのアドレスバーに「vk.com/away.php」が0.5秒表示されることに気づきましたが、もちろんこれを重要視していませんでした。



バックグラウンド


かつて、あるプログラマーが別のプロジェクトを完成させたとき、それをみんなに伝えたいという欲望に夢中になっていることに気づきました。プロジェクトは、一意のIPを持つサーバーでホストされていましたが、ドメイン名はありませんでした。したがって、私は急いで、.ddns.netドメインに美しい第3レベルのサブドメインを作成し、最終的にリンクとして使用しました。 

しばらくして投稿に戻ると、プログラマーはサイトの代わりにVKスタブが開いて、安全でないサイトへの移行について通知して



いることに気付きました。松葉杖なしでリンクをたどることはできません。

どうしましたか


この実装には、いくつかの重大な欠点があります。

  • . , . — .
  • . . , , , , .
  • . , , id .


上記のすべての問題に対する最適な解決策は、ブラウザ拡張です。明らかな理由により、選択はChromeに依存します。問題については、Chromeの拡張機能の作成に特化した優れた記事があります。

このような拡張機能を作成するには、json-Manifestとアドレスの現在のURLを監視するためのJavaScriptファイルの2つのファイルを別のフォルダーに作成する必要があります。

マニフェストファイルを作成


私たちが必要とする主なことは、タブを操作して実行可能スクリプトを割り当てるための拡張権限を与えることです：

{
  "manifest_version": 2,
  "name": "Run Away From vk.com/away",
  "version": "1.0",
  "background": {
    "scripts": ["background.js"]
  },
  "permissions": ["tabs"],
  "browser_action": {
    "default_title": "Run Away From vk.com/away"
  }
}

jsファイルを作成する


ここではすべてが単純です。新しいタブを作成するときに呼び出されるイベントで、URLアドレスのチェックを追加します。「vk.com/away.php」で始まる場合は、GETリクエスト内の正しいアドレスに置き換えます。

chrome.tabs.onCreated.addListener( function (tabId, changeInfo, tab) {
	chrome.tabs.query({'active': true, 'lastFocusedWindow': true}, function (tabs) {
		var url = tabs[0].url;
		if (url.substr(0,23) == "https://vk.com/away.php"){
			var last = url.indexOf("&", 0)
			if(last == -1)last = 1000;
			var url = decodeURIComponent(url.substr(27, last-27));
			chrome.tabs.update({url: url});
		}
	});
});

エクステンションを置く


両方のファイルが同じフォルダにあることを確認した後、Chromeを開き、拡張機能のタブを選択して[解凍された拡張機能をダウンロード]をクリックします。開いたウィンドウで、「advanced」と書かれたフォルダーを選択し、「収集」をクリックします。できた！これで、vk.com / awayのようなすべてのリンクが元のリンクに置き換えられます。

結論の代わりに


もちろん、このタイプのスタブは何百万もの詐欺サイトから多くの人々を救いましたが、人々自身が危険なリンクに行くかどうかを決定する権利を持っていると私は信じています。
便宜上、プロジェクトをgithubに投稿しました。
