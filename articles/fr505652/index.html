gogrep: recherche structurelle et remplacement du code Go

gogrep est l'un de mes utilitaires préférés pour travailler avec Go. Il vous permet de trouver du code par modèles syntaxiques, de filtrer les résultats par type d'expression et d'effectuer des remplacements (également par modèle).


Dans cet article, je vais vous expliquer comment l'utiliser gogrep, ainsi que l' extension VS Code pour un travail plus pratique gogrepdirectement depuis l'éditeur.



Pourquoi Gogrep est nécessaire


Si dans l'abstrait, il gogreppeut être utile pour:


  • Refactoring
  • Étude de base de code
  • Recherche de code suspect (exemple: règleguard )

Prenons un exemple qui démontre l'élégance et l'efficacité de la recherche structurelle.


Fonctionne a()et b()effectue les mêmes opérations:


func a(xs []int) []int {
  xs = append(xs, 1)
  xs = append(xs, 2)
  return xs
}

func b(xs []int) []int {
  xs = append(xs, 1, 2)
  return xs
}

Supposons que nous voulons réécrire tous les endroits où les appels appendpeuvent être réduits.


Essayons gogrep:


  • Trouvez toutes les paires correspondantes en utilisant le -xmodèle.$x=append($x,$a); $x=append($x,$b)
  • Grâce au -smodèle, $x=append($x,$a,$b)nous obtenons le remplacement souhaité
  • En passant un argument, -wtous les fichiers affectés seront mis à jour.

gogrep -w -x '$x=append($x,$a);$x=append($x,$b)' -s '$x=append($x,$a,$b)' ./...

Si vous installez l' extension pour VS Code , cela devient encore plus facile.
