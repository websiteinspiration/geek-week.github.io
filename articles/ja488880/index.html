DevOpsセクションの概念とテーマ


今年は、実用的な解決策を最大限に、理論を最小限にしたいと考えています。どこが痛いか教えてください。何が起こったのか、何が起こらなかったのか。私たちは一緒に喜んで/泣きます。

以下は、2020 DevOpsの現実に関連すると思われるトピックのリストです：

CI / CD

  • 素晴らしいCI / CDパイプライン
  • GitHubアクション（理論なし、練習のみ）

雲

  • 雲の中のCI / CD（Spinnakerなど）
  • GKE、Kubernetes、Istio、Helmなどに深く浸る
  • クラウド内のデータ（PVC、DBなど）
  • MLのクラウド
  • サーバーレス（練習のみ）
  • ロシアの雲（立法機能、152-、Yandex、MailRuケース、およびこの点であなたを心配するすべてのもの）

DevOps / SRE

  • システムを監視する方法（可観測性）：サービスメッシュ、監視、監査
  • セキュリティ（DevSecOps）
  • 構成管理（Ansible、Terraformなど）
  • 文化について話す（ベストプラクティス）
  • 企業ストーリーをシフトする
  • 管理：ライフハック、役立つヒント、fakapy。
