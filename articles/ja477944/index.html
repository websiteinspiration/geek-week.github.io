状況：Python 2.7のサポートは2020年に廃止されました

Python 2.7は1月1日にサポートされなくなりました。しかし、多くの企業はまだ更新されたバージョンに切り替えていません。資料では、この状況の理由について説明します。


写真  -JanKopřiva -Unsplash

誰もが新しい「python」の準備ができているわけではありません


2020年に Python 2.7 が  廃止され、セキュリティ更新が  一時停止されます。これらの計画が5年前に知られるようになったという事実にもかかわらず-Pythonの作者であるGuido van Rossumは個人的に   開発者と企業にPython 3への移行を奨励しました-新しいバージョンを適応させるプロセスは遅いです。

今年の初めに、Pythonパッケージインデックス（PyPI）からダウンロードされたパッケージの約60％  が  バージョン2.7でした。9月までにこの数値は40％に減少しましたが、ほとんどの場合、期限までにゼロに到達しません  。

移行を阻害するもの


大企業では、新しいフレームワークの導入や更新されたテクノロジーへの移行には、常にかなりのリソースと時間が必要です。場合によっては、プロセスが何ヶ月も、さらには何年も続くことがあります。これは、大規模なコードベースと多数の依存関係によるものです。 Facebook  は  2014年に同社のサービスをPython 3 に移行し始めました。このプロジェクトの実装には1年かかりました。かなりの数のライブラリを書き直し、数千のリグレッションを修正する必要がありました。その後、同社のエンジニアがInstagramを採用しました。この場合、移行に  は  10か月かかりました。 Dropboxで-Python 3への移行  は  すでに3年です。

一部の企業は、セキュリティサービスとの重要なテクノロジーの更新について交渉するための厳しいルールを持っています。このユニットは、PIPパケットのロードを調整することもあります。 「セキュリティ」は、Python 3に切り替えると、サービスに重大な脆弱性が現れ始めるのではないかと心配しています。実際、銀行や医療などの分野では、エラーのコストが高くなる可能性があります。そのため、昨年、英国のTSBは、新しいITシステムを導入する過程 で、モバイルバンキングシステムに不具合を引き起こすバグに遭遇しました  。その結果、190万人がアカウントにアクセスできなくなりました。組織はまだ   結果に対処しています。

とりわけ、Python 2は依然として主要なLinuxベースのオペレーティングシステムをサポートしています。たとえば、RHELでは、ユーザーは使用するバージョンを選択できます。Red Hatオペレーティングシステムは徐々にPython 3に移行していますが、このプロセスは希望どおりにスムーズに進んでいません。その過程で、主にポインタの操作に関連するバグが定期的に発見されます。DebianはPython 2のサポートも保持していますが、アップグレードはすでに始まっています。それもゆっくりと動いています-エンジニアはライブラリの互換性を一貫してテストしています。

いずれにせよ、Python 3への完全な移行後でも、一部のパッケージは企業やコミュニティの愛好家によってサポートされるため、多くは以前のバージョンで引き続き機能します。

コミュニティが考えること


テーマ別スレッドのハッカーニュースの居住者は、  移行が遅い主な理由は、このプロセスから目に見えるメリットが（最近まで）ないことにあると指摘しています。開発者の興味を引き、Python 3への移行を促す新しい関数はありませんでしたが、言語の作成者がいくつかの決定を下したため、反対に、プログラミングの印象が台無しになりました。特に、 バイト文字列のサポートの除外とUnicodeでの排他的な作業への移行によって、大きな不満が  生じました。 Stack Overflowはまた、Pythonエコシステムには、コードをあるバージョンから別のバージョンに自動的に変換するための組み込みツールが欠けていたとも  述べています。この問題は、2to3などのユーティリティの登場により解決され  ました。 と  6。

近年、Python 3の機能は大幅に拡張されました  。行列の乗算、競合プログラミングを整理するためのasyncioモジュール、関数の  変数型、クラスフィールド、引数、戻り値の注釈を追加しました  。Python 3のサポートは、MLのscikit-learnなど、他の一般的なライブラリでも表示されています。更新された機能セットにより、企業は次のバージョンの言語に切り替えるように説得されます。また、Python 3の支持者の数は来年大幅に増加しますが、完全な移行にはかなりの時間がかかります。
