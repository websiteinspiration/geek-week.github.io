Introdução


Uma interface circular é uma interface gráfica cujos elementos são circulares, organizados ou animados em um caminho circular.


A interface circular complexa consiste principalmente em vários elementos circulares com um único centro.
A Figura 1 mostra exemplos de uma interface circular abstrata.



FIG. 1 - Exemplos de interfaces circulares


Interfaces circulares podem ser usadas para:


  • diversificar o design de um projeto ou produto;
  • forneça compacidade ao visualizar informações;
  • Obtenha um analógico digital de interfaces de controle reais ou instrumentos de medição.

Este artigo é dedicado aos aspectos principais do desenvolvimento de interfaces circulares, como classificação, princípios de design, conceitos básicos de animação e implementação de interfaces circulares em JavaScript usando uma biblioteca de classes especializada.
