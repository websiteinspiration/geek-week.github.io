def tokenize_ru(x):
      tokens = x.lower()
       tokens = re.sub(r'[^\w\s]+|[\d]',' ',tokens)
       tokens = word_tokenize(tokens)
       tokens = [i for i in tokens if (i not in string.punctuation)]
       stop_words = stopwords.words('russian')
       stop_words.extend(['', '', '', '', '', '', '', '—', '–', '', '', '...'])
       tokens = [i for i in tokens if (i not in stop_words)]
       tokens = [i.replace("«", "").replace("»", "") for i in tokens]
       return tokens

f-mera使用公式f-mera = 2 * Precision * Recall /（Precision + Recall）计算得出，其中精确度和查全率（完整性）是从混淆矩阵中计算得出的，这是准确性和完整性之间的谐波平均值，对于二进制分类，这足以评估模型的准确性。 SGDClassifier + TF-IDF的组合可获得最佳结果。

此外，在已经选择的模型中，我们以较高的准确性替换了从2016年到2018年开始的不相关的评估评论，这些评论没有参加训练，以验证训练和模型的正确性，结果得到f-mera：0.9326314212969897。以后可以使用此模型对评论进行分类，以免浪费时间手动搜索负面评论。
