IoT-gateway para protocolos industriales basados ​​en i.MX6 y Linux



WISE-710 es un dispositivo universal que puede ser simultáneamente una puerta de enlace para interfaces industriales, un dispositivo para recopilar y procesar datos, un concentrador para dispositivos IoT y un enrutador. Construido en SoC i.MX6 y se ejecuta en Ubuntu o Yocto Linux.

De hecho, es una computadora compacta en un diseño industrial con interfaces de E / S incorporadas para conectar cualquier automatización, sensores y otros periféricos industriales. Junto con el dispositivo, los desarrolladores tienen acceso a SDK abiertos para construir el kernel y los programas de Linux, así como también opciones con el software EdgeLink preinstalado que admite más de 200 protocolos industriales propietarios, así como protocolos abiertos como MQTT, Modbus, etc.

En el artículo, analizamos las funciones del dispositivo y los casos de uso.

Especificaciones


El dispositivo está equipado con un potente procesador para ejecutar programas productivos: almacenamiento local y procesamiento de datos, conversión de protocolos y computación. Los módulos de comunicación inalámbrica le permiten usar el dispositivo como una puerta de enlace de IoT independiente y un dispositivo de comunicación con el objeto.

  • Procesador: ARM Cortex-A9 i.MX6 dos núcleos de 1 GHz
  • Unidad: 8 GB eMMC
  • RAM: 1GB DDR3
  • Ranura Micro-SD: 1
  • Interfaces: 8 puertos digitales de entrada-salida, RS-485 / RS-232
  • Ethernet: 2x puertos de 1 Gb / s
  • Módulos de expansión: LTE, WiFi, puertos de E / S
  • USB: 1x puerto en modo host



El dispositivo se puede instalar tanto en un riel DIN como en cualquier superficie utilizando orificios de montaje en la parte inferior de la carcasa. Para un montaje confiable de los contactos de la interfaz, se utilizan terminales de abrazadera sin tornillos.



Modern SoC (System on Chip) i.MX6 tiene un bajo consumo de energía y no requiere enfriamiento activo o incluso instalar un disipador de calor en el procesador. La foto a continuación muestra la placa WISE-710 . Durante el funcionamiento, el dispositivo casi no se calienta. Para trabajar con datos confidenciales, el dispositivo tiene un módulo criptográfico incorporado ATECC508A, que le permite realizar operaciones con claves privadas sin la posibilidad de extraer la clave en forma clara. Esto protege contra el compromiso de datos y la fuga de claves en caso de que el dispositivo sea robado.

Módulos de expansión de interfaz



WISE-710 tiene un puerto mPCIe integrado para conectar módulos de expansión. Se puede utilizar para conectar módulos Advantech de marca y placas de terceros, como módems, controladores, etc. La cubierta posterior del dispositivo se abre sin tornillos, lo que le permite conectar rápidamente los módulos. En este momento, estamos desarrollando módulos de expansión que vamos a presentar en un futuro próximo.

Para una fácil integración con el sistema en la nube WISE-PaaS, los dispositivos WISE-710 se pueden entregar con el agente Edgelink preinstalado, compatible con más de 200 controladores de dispositivos, que le permite combinar sistemas industriales de diferentes fabricantes para conectarse a una sola nube la plataforma.

SDK abierto



Junto con el dispositivo, se proporciona una cadena de herramientas gratuita para el autoensamblaje de Yocto Linux para las necesidades del cliente. Si es necesario, el sistema operativo se puede reconfigurar por completo, incluido el kernel del sistema operativo. Por ejemplo, para instalar parches Linux-RT que le permiten realizar operaciones en tiempo real.

Conclusión


El dispositivo WISE-710 reemplaza inmediatamente un conjunto completo de dispositivos: un enrutador de red y un conmutador, un dispositivo de adquisición y transmisión de datos, un controlador lógico programable y otros. Una plataforma completamente abierta ofrece oportunidades ilimitadas para desarrollar sus propias soluciones de automatización, recopilación de datos, informática local y mucho más.
