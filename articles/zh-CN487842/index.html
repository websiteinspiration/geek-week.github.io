2个用于乡村路由器的SIM卡-是很多还是少一点？

去年，我为乡间别墅进行了3G / 4G路由器的测试，这个话题对许多人来说似乎很有趣。经过测试，我什至在车上内置了一个路由器，以便在旅途中获得稳定的Internet访问，但最重要的还是在家中访问网络的稳定性。去年年底，我注意到俄罗斯双SIM 3G / 4G路由器的发展出现在市场上。这些不仅仅是基于USB调制解调器的工艺品，还是基于Linux机器的成熟设备。我知道了，测试了一个月，今天决定分享我的观察结果。按照传统，他为不喜欢阅读的人制作了视频。


为什么选择2SIM？


这个问题类似于苹果树的所有者用一对SIM卡向Android的信徒问的问题。答案显而易见，很简单：两张SIM卡可让您使用两个不同的运营商，选择最佳的资费/速度。另外，这大大提高了可靠性：当一个运营商的BS下降或余额中的钱用完时，路由器将切换到备用运营商，并且不会中断对网络的访问。最后，如果使用桅杆上的外部天线，则两个SIM卡的存在可以简化并加速地面上电信运营商的选择：您可以通过编程方式更换运营商的卡并继续调谐天线，而不必每次都随路由器一起拆除天线或​​断开路由器并更换卡，而无需每次都随路由器一起拆除天线。

怎么运行的？




我从俄罗斯制造商Microdrive那里选择了OEM板-Tandem-4GS-OEM形式的路由器，但是我要求制造商将其发送到密闭盒中，因为我的路由器位于天线附近的未加热的阁楼中。长度为3 m的电缆组件可以将路由器放置在相对舒适的气候条件下，但这在某种程度上降低了信号强度。随后，我想将路由器直接转移到天线上，卸下电缆组件，从而进一步改善通信参数。



路由器本身就是一块板上，上面装有所有元件，外部天线的尾纤和自己的Wi-Fi天线都连接到该板上。在这些接口中，有一个支持PoE的100 Mbit / s以太网连接器和一个用于连接外部电源适配器的连接器。该电路板可以通过9至36 V的电压供电，也就是说，即使在汽车中，至少在提供24V电源的卡车中，甚至在具有12-24V冗余电源的自制系统中，也可以轻松使用此路由器。最方便的是，这种路由器可以与具有足够容量的常规汽车或凝胶电池配合使用，以确保几天之内无需外部网络即可运行。也就是说，即使外部网络已关闭，并且您具有不带逆变器的不间断电源设备，路由器也可以直接连接到电池，并且互联网不会丢失。



现在让我们来谈谈路由器的技术特征。图片清楚地表明Quectel EC25-E芯片负责与操作员进行通信。从制造商网站上的模块的技术特性可以看出，它支持2G / 3G / 4G（LTE）技术，并且Cat.4类别支持LTE，从而可以实现最高150 Mbps的下载速度和50 Mbps的上传速度。同时，将速度为100 Mbps的以太网接口焊接到板上，也就是说，无论如何，我们都将深入研究有线连接的性能，但是稍后我将再次讨论。重要的是，此模块必须具有很高的灵敏度，并且即使在3G网络中即使信号高达-110 dBm的微弱信号下也可以工作。简而言之，即使在距离BS很远的地方，即使信号很弱，该路由器也可以使您进入网络。从今年开始，制造商转而使用具有更高灵敏度指标的Quectel EC25-EU芯片。要改善微弱的信号，请使用我的天线测试。



该设备设计用于在室内和室外的密闭盒中安装。为了使路由器即使在严重霜冻下也能轻松启动，它配备了加热系统：将4个电阻器（红色圆圈）焊接到板上，在启动之前先加热板及其周围的元件。制造商保证路由器即使在-40度下也可以运行。如果电路板温度低于-18度，则在路由器启动时，加热器本身会打开，而在温度达到-15度时，加热器会自行关闭。此外，由于芯片工作中的自热，电路板成功运行。

该路由器配备了2.4 GHz Wi-Fi模块，具有良好的信号水平和广泛的覆盖范围，但是您不应期望它具有很高的速度。即使连接通道速度高达150 Mbps的笔记本电脑，下载速度也不会超过55 Mbps。但是，这种速度足以让您舒适地上网和观看视频，而不会拖延。如果想获得全速，电缆连接肯定会更好。

该路由器的一大优点是可以进行一些修改，例如：更改IMEI和调整TTL值，这对“智能手机”无限资费的所有者而言是值得注意的。但是，由于这违反了使用手机资费的规则，因此您自行承担执行这些操作的风险。有关这些修改的信息很容易在网上搜索，但我没有这么说。

自我测试


我需要带有两个SIM卡的路由器吗？实际上，确实需要。尽管我距大城市仅5公里，但接受调查的9家提供商中，没有一家同意支持光学设备或给我扔无线电桥，因此仅剩下移动互联网。我有一个好的定向天线，增益为27 dB PRISMA 3G / 4G MIMO我可以安全地将其推荐给任何想要与运营商的基站保持足够距离的体面速度的人。正如我一开始所写的那样，我的路由器处于未加热的阁楼中，为了更换操作员，我不得不搬到不太舒适的环境中，并用双手冻结重新布置SIM卡。关于沟通渠道的保留我们能说些什么？在秋天有雷雨后，Yota运营商（又名Megafon）几乎在整个工作日都关闭了铁塔上的数据传输，尽管与BS有连接。直线塔同时运作非常成功。幸运的是，根据路由器的证词判断，不同运营商的塔架位于一个方向，尽管彼此之间并不遥远，但不必重新配置天线。



屏幕截图清楚地表明，Wifire运算符（在Megafon基础上运行）在20 MHz的信道宽度下工作，并提供80-90 Mbit / s的数据传输速率，而对ya.ru服务器的ping处于60-80 ms的水平。 Beeline运算符的信道宽度为10 MHz，下载速度为35-40 Mbit / s，并ping到ya.ru服务器大约30-40 ms。也就是说，除了备份之外，当您需要下载大量信息时，切换到Wifire更方便，而如果要在线玩，则切换到响应时间较短的Beeline更加合乎逻辑。



此外，路由器具有非常酷的内置逻辑，可以切换到第二张SIM卡。除了路由器本身会定期对给定主机执行ping操作以建立连接这一事实之外，您还可以为缺乏通信设置事件条件。例如，路由器可能不会切换到另一个SIM卡，而是重新启动当前连接。或者，切换到第二张SIM卡后，对其进行指定的时间（介于1到20,000分钟之间），然后返回到第一张SIM卡。这样做是为了在恢复第一个操作员的工作时自动返回到它。例如，第二张SIM卡可以具有兆字节付款的储备，而第一张是无限的。但是，当通信的可用性至关重要时，则可以使用兆字节资费一段时间，然后在一定时间后返回anlim。



因此，解决了预订问题，这对于移动点（在线收银台，终端）和固定点（加油站的收银台，ATM，私人住宅，具有Internet连接的安全系统）都是至关重要的。

另外，值得注意的是切换到第二张SIM卡所需的时间很短，这需要3到10秒。而且我注意到，从1到2，SIM卡几乎立即切换，从第二到第一张SIM卡切换需要更长的时间，但是在双向切换时YouTube视频不会中断。
