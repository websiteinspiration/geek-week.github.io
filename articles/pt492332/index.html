Crie aplicativos gratuitamente com o Nível Gratuito do Azure Cosmos DB

Deseja criar um novo aplicativo, desenvolver e testar ou executar pequenas cargas de trabalho de produção com o  Azure Cosmos DB ? Nosso novo nível de uso gratuito permite que você comece a trabalhar sem custos e economize dinheiro ao criar e desenvolver novos aplicativos.



Com o nível gratuito incluído do Azure Cosmos DB, você obtém os primeiros 400 RU / s de largura de banda e 5 GB de armazenamento em sua conta gratuitamente todos os meses, durante todo o período da conta. Isso significa que você pode começar pequeno e crescer com confiança, sabendo que seu aplicativo será executado em um serviço de banco de dados de alto desempenho. Você pagará apenas se sua conta exceder 400 RU / se 5 GB. Além disso, se seu aplicativo tiver muitos contêineres, você poderá criar até 25 contêineres no banco de dados com uma largura de banda total e todos eles usarão 400 RU / s grátis. Você pode ter até uma conta gratuita do Azure Cosmos DB para cada assinatura do Azure.


O que você pode criar de graça?


Pedimos aos membros de nossa comunidade de desenvolvimento que fornecessem exemplos de aplicativos com 400 RU / se 5 GB de espaço em disco e recebemos muitos exemplos excelentes dos aplicativos que eles criaram: o aplicativo Alexa, que ajuda os passageiros a usar o transporte público, e um aplicativo da Web para gerenciar e compartilhar convenientemente  listas on-line recursos . Falaremos sobre alguns desses desenvolvedores e seus aplicativos nas próximas semanas e meses.

Você tem um aplicativo com largura de banda menor que 400 RU / se 5 GB de armazenamento? Conte-nos sobre ele  - teremos o maior prazer em compartilhá-lo com a comunidade.

Início do trabalho


Crie uma nova conta do Azure Cosmos DB e ative o  nível gratuito  durante a criação. Há um limite para uma conta de assinatura gratuita do Azure. Confira nossos  guias e guias de início rápido  ou use  um exemplo para entender o que você pode fazer com o Azure Cosmos DB.

Perguntas frequentes (FAQ)


P: Você não ofereceu uma opção gratuita antes? O que mudou?

R: Sempre oferecemos maneiras de testar o Azure Cosmos DB gratuitamente - como parte da Conta Gratuita do  Azure , com um  emulador local e como uma  avaliação gratuita de 30 dias  - mas essas opções eram limitadas em disponibilidade e volume. O nível gratuito está disponível por tempo ilimitado.

P: Como o desconto de nível gratuito é exibido na minha conta?

R: Nas contas de nível gratuito, você recebe os primeiros 400 RU / se 5 GB de armazenamento em sua conta gratuitamente. Quaisquer RU / se armazenamento além de 400 RU / se 5 GB serão cobrados de acordo com as taxas regulares de preços da página de preços. Na fatura, você não verá uma cobrança ou um item de linha pelos 400 Ru / se 5 GB gratuitos, apenas as RU / se armazenamento além do que é coberto pelo nível gratuito. Nas contas do nível gratuito, você receberá os primeiros 400 RU / se 5 GB de espaço em disco na sua conta gratuitamente. Quaisquer RU / se armazenamento acima de 400 RU / se 5 GB serão pagos a taxas regulares. Na conta, você não verá uma taxa ou posição gratuita de 400 Ru / se 5 GB, apenas RU / se armazenamento além do que é coberto pelo nível gratuito.

P: O nível gratuito pode ser aplicado a todas as APIs do Azure Cosmos DB?

Sim, o nível gratuito funciona para todas as APIs do Azure Cosmos DB, incluindo Core (SQL), APIs para MongoDB, Cassandra e Gremlin.

P: Posso aplicar um desconto gratuito para  bancos de dados e contêineres do piloto automático ?

A: sim Com o serviço de piloto automático, você é cobrado pelas RU / s mais altas, pelas quais o banco de dados ou contêiner é escalado por hora. Quando um desconto de nível gratuito é aplicado, 400 RU / s são deduzidos desse valor. Confira nossa  documentação para obter detalhes.

P: Como o desconto de nível gratuito funciona se eu tiver uma conta em várias regiões?

R: Nas contas RU / s multirregionais, o banco de dados ou o contêiner é replicado em todas as regiões. Por exemplo, se você tiver um contêiner com 400 RU / s, e a conta estiver localizada em 3 regiões, o número total de contas de RU / s será de 1200 RU / s. Quando o desconto é aplicado, você será cobrado a 1200 RU / s - 400 RU / s = 800 RU / s por hora. Confira nossa  documentação para obter detalhes.

P: Eu já tenho uma conta e quero transferir meus dados para a conta do nível de uso gratuito. Como devo fazer isso?

R: Depois que sua nova conta for criada e ativada com o Nível gratuito, será fácil mover os dados existentes para a nova conta.  Ferramenta de Migração de Dados pode ser usado para transferir pequenos conjuntos de dados se você usar a API Core (SQL). Você também pode migrar dados do Azure Data Factory  para a API do Azure Cosmos DB Core (SQL)  e a API do Azure Cosmos DB  para MongoDB .

P: A opção para habilitar o nível gratuito ao criar uma nova conta no portal do Azure não é exibida. Onde ela está?

R: Uma assinatura do Azure pode ter até uma conta de nível gratuito. Se essa opção não for exibida, significa que outra conta na assinatura já está ativada no nível gratuito.
