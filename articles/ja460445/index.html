特に、このセクションでは、さまざまなフーリエ変換（サイン、コサイン、直接、逆、多次元、実数）を計算するためのPythonパッケージscipy.ffpackについて説明します。

フーリエ変換を使用すると、任意の関数を一連の調和信号として表すことができます。フーリエ変換は、畳み込み法とデジタル相関器の設計の基礎であり、スペクトル分析で積極的に使用されており、長い数を扱うときに使用されます。

2.離散シーケンスが実数の場合、そのようなシーケンスのスペクトル密度係数は偶関数であり、引数は奇数周波数関数です。

畳み込みと相関


このセクションでは、離散ランダムおよび決定論的シーケンスの相関と畳み込みの概念を紹介します。自己相関関数と畳み込みを伴う相互相関関数の関係を示します。たたみ込みの特性について説明します。特に、離散シーケンスの例を詳細に分析して、離散信号の線形および循環たたみ込みの方法を検討します。さらに、FFTアルゴリズムを使用して「高速」畳み込みを計算する方法が示されています。

実際の問題では、あるプロセスの別のプロセスへの類似度や、あるプロセスの別のプロセスからの独立性についての質問がよく出されます。つまり、信号間の関係、つまり相関関係を見つける必要があります。。相関法は、ターゲットの特性を決定し、オブジェクトまでの距離を決定するレーダーの問題で、信号検索、コンピュータービジョン、画像処理などの幅広いタスクで使用されます。さらに、相関は、ノイズの弱い信号を検索するために使用されます。

畳み込みは、信号の相互の相互作用を表します。信号の1つがフィルターのインパルス応答である場合、入力シーケンスとインパルス応答のたたみ込みは、入力アクションに対する回路の反応にすぎません。つまり、結果の信号は、フィルターを通過する信号を反映しています。

自己相関関数（ACF）は情報のコーディングに使用されます。長さ、周波数、および形状のパラメーターに応じたコーディングシーケンスの選択は、主にこのシーケンスの相関特性によるものです。最適なコードシーケンスは、誤検出または誤動作（信号の検出、しきい値デバイスの場合）または誤同期（コードシーケンスの送受信の場合）の可能性が最も低くなります。

ランダム信号とノイズ


このセクションでは、ランダム信号、確率密度、ランダム分布則の概念を紹介します。数学的モーメントが考慮されます-平均（数学的期待値）と分散（またはこの量の根は標準偏差です）。また、このセクションでは、正規分布と関連するホワイトノイズの概念を、信号処理中の主要なノイズ源（干渉）と見なしています。

ランダム信号は時間の関数であり、その値は事前に不明であり、ある程度の確率でのみ予測できます。ランダム信号の主な特徴は次のとおりです。

  • 分布則（特定の間隔での信号値の相対滞留時間）、
  • 信号パワーのスペクトル分布。




DSPタスクでは、ランダム信号は2つのクラスに分類されます。

  • ノイズ-一連の異なる周波数と振幅で構成されるランダムな変動、
  • 情報を運ぶ信号、その処理のために確率的方法に訴える必要がある。


確率変数を使用して、ソースからデータレシーバーへの信号の通過に対する実際の媒体の影響をシミュレートできます。信号がノイズの多いリンクを通過すると、いわゆるホワイトノイズが信号に追加されます。原則として、そのようなノイズのスペクトル密度はすべての周波数で均一に（等しく）分布し、時間領域のノイズ値は通常分布します（ガウス分布則）。ホワイトノイズは選択した時間サンプルの信号振幅に物理的に追加されるため、加法性ホワイトガウスノイズ（AWGN）と呼ばれます。

信号、変調、および操作


このセクションでは、高調波信号の1つ以上のパラメーターを変更するための基本的な方法を示します。振幅、周波数、および位相変調の概念が導入されています。特に、レーダーの問題で使用される線形周波数変調が強調表示されています。信号の主な特性、変調パラメータに応じた変調信号のスペクトルが表示されます。



便宜上、上記のタイプの変調を実装する関数のセットがPythonで作成されています。チャープ信号の実装例：

def signal_chirp(amp=1.0, freq=0.0, beta=0.25, period=100, **kwargs):
    """
    Create Chirp signal

    Parameters
    ----------
    amp : float
        Signal magnitude
    beta : float
        Modulation bandwidth: beta < N for complex, beta < 0.5N for real
    freq : float or int
        Linear frequency of signal
    period : integer
        Number of points for signal (same as period)
    kwargs : bool
        Complex signal if is_complex = True
        Modulated by half-sine wave if is_modsine = True
    """
    is_complex = kwargs.get('is_complex', False)
    is_modsine = kwargs.get('is_modsine', False)

    t = np.linspace(0, 1, period)
    tt = np.pi * (freq * t + beta * t ** 2)
    
    if is_complex is True:
        res = amp * (np.cos(tt) + 1j * np.sin(tt))
    else:
        res = amp * np.cos(tt)

    if is_modsine is True:
        return res * np.sin(np.pi * t)
    return res

デジタル信号処理用のPython CicFilterクラス 
class CicFilter:
    """
    Cascaded Integrator-Comb (CIC) filter is an optimized class of
    finite impulse response (FIR) filter.
    CIC filter combines an interpolator or decimator, so it has some
    parameters:

    R - decimation or interpolation ratio,
    N - number of stages in filter (or filter order)
    M - number of samples per stage (1 or 2)*

    * for this realisation of CIC filter just leave M = 1.

    CIC filter is used in multi-rate processing. In hardware
    applications CIC filter doesn't need multipliers, just only
    adders / subtractors and delay lines.

    Equation for 1st order CIC filter:
    y[n] = x[n] - x[n-RM] + y[n-1].


    Parameters
    ----------
    x : np.array
        input signal
    """

    def __init__(self, x):
        self.x = x

    def decimator(self, r, n):
        """
        CIC decimator: Integrator + Decimator + Comb

        Parameters
        ----------
        r : int
            decimation rate
        n : int
            filter order
        """

        # integrator
        y = self.x[:]
        for i in range(n):
            y = np.cumsum(y)

        # decimator

        y = y[::r]
        # comb stage
        return np.diff(y, n=n, prepend=np.zeros(n))

    def interpolator(self, r, n, mode=False):
        """
        CIC inteprolator: Comb + Decimator + Integrator

        Parameters
        ----------
        r : int
            interpolation rate
        n : int
            filter order
        mode : bool
            False - zero padding, True - value padding.
        """

        # comb stage
        y = np.diff(self.x, n=n,
                    prepend=np.zeros(n), append=np.zeros(n))

        # interpolation
        if mode:
            y = np.repeat(y, r)
        else:
            y = np.array([i if j == 0 else 0 for i in y for j in range(r)])

        # integrator
        for i in range(n):
            y = np.cumsum(y)

        if mode:
            return y[1:1 - n * r]
        else:
            return y[r - 1:-n * r + r - 1]
