初心者に最適なプログラミング言語

私はプログラミングの学習に関するさまざまなアイデアに常に興味を持っています。多分これは私がこの分野で永遠の学生であるからです。しかし、今日、私はJavaの学習を始めるという、思いがけないアイデアを1つ知りました。そして、私は沈黙を保つことができませんでした。

私は教育学の優れたスペシャリストではありません。おそらくコンピュータサイエンスでは、生徒を水中に投げ込み、より深い場所を選んで、そこに来る運命にある人がいます。ただし、複雑さが増すにつれて、教師が生徒に1つずつさまざまなプログラミング概念を紹介する場合に、トレーニングが最も効果的であるように思えます。したがって、「トレーニング」PLの主な要件は、最も基本的な機能から始めて、機能を分離して使用できることです。

繰り返しになりますが、私の素人っぽい意見では、プログラミング言語がこの要件を満たしているかどうかを確認するのは簡単です。Rosetta Codeの「Hello World」セクションを開くだけです。

これらの基本的なプログラムを理解するために必要な概念をリストしてみましょう。

それはラジオのリスナーの要求で補足されます。Brainfuck、PHP、C、Julia。


Python 2


print 'Hello world!'


このコードを解析するとき、教師は、少なくとも簡単にはどのような彼の学生に説明する必要があり、キーワード、演算子、および文字列データ型です。もちろん、そのような説明がなくても、一部の生徒（すべてではありません！）は、これらの機能を直感的にすぐに理解できます。ただし、確実性をすぐに達成することをお勧めします。

ルソン3


print('Hello world!')


関数 の概念がキーワードと行に追加されます。はい、関数は産業用PLとしてのPython 3のプラスです。しかし、同時に、関数は演算子よりも高レベルの機能であり、これは最初のPLとしてのPython 3の研究を複雑にします。はい。教師は、次のいずれかのクラスで角かっこを使用する必要性の説明を延期できますが、これは生徒の頭の痛い問題のままです。print

ジュリア


println("Hello world!")


Python 3の場合と同じ基本的な概念ですが、Juliaはネイティブコードにコンパイルされた高性能の動的Javaです。ブラボー！

ベーシック


10 PRINT "Hello world!"


キーワード、文字列AP、演算子、コード行の番号付け。Basicの行番号付けは、コンピュータの物理メモリデバイスを模倣した、かなり複雑な低レベルの概念です。教師が彼女に注意を払わないと、学生にとってつまずきの原因となる可能性があります。

VisualBasicのような後の方言では、コード行は単純な計算不可能なラベルになります。最初のプログラムでは、タグは必要ありません。したがって、言語は初級学習にとってより簡単になります。

パスカル


program HelloWorld(output);
begin
  writeln('Hello, World!');
end.


キーワード、文字列TD、演算子、関数、その他 プログラムは複数の行、ブロック、区切り文字（またはターミネータ？常に混乱します）を必要とし、インデントが最初の概念に追加されます。そして、オペレーターはprogramそれほど単純ではありません...パスカルの親しみやすさはやや誇張されているようです。

C


#include <stdlib.h>
#include <stdio.h>
 
int main(void)
{
  printf("Hello world!\n");
  return EXIT_SUCCESS;
}


キーワード、文字列AP、演算子、関数、ブロック、区切り文字、インデント、およびプリプロセッサディレクティブとマクロ。この例ではマクロなしで実行できましたがreturn EXIT_SUCCESS、この場合は非常に示唆的です。Cは従来のシステムプログラミング言語であるため、オペレーティングシステムの操作を十分に理解し、移植性を目的としてそれを検討することをお勧めします。それ以外の場合、このPLはPascalよりも少し複雑に見えます。

PHP


<?php
echo "Hello world!\n";
?>


理解のほか、キーワード、文字列のAP、オペレーターおよび区切り文字（またはターミネータ？）、この例では、次のような特定のウェブ技術の基本を理解せずに学習することができないマークアップ言語（HTML）とテンプレートエンジン。実際、PHPはテンプレート言語であり、DSLの一種です。その結果、PHPは、バックエンドテクノロジーをより深く掘り下げたいフロントエンドベンダーにとって優れた学習言語です。しかし、PHPをゼロから学ぶことは非常に困難です。

C ++


#include <iostream>
 
int main () {
  std::cout << "Hello world!" << std::endl;
}


キーワード、文字列AP、演算子、関数、ブロック、区切り文字、インデント、およびそのディレクティブ、スコープ、ストリーミング入力/出力を備えたプリプロセッサ ...

ジャワ


public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello world!");
    }
}


キーワード、文字列AP（2つの文字列APでも、静かに保つことができます）、空のAP、配列、ブロック、区切り文字、インデント、およびクラス、オブジェクト（暗黙的に、ただし他に説明することはできませんstatic）、属性、メソッド、アクセス修飾子 ...神々、私はすでにこれすべてを見たいです！結局のところ、私はMinecraftのModだけを書きたかったのです！

namespace HelloWorld
{
    class Program
    {

同じJavaで、アクセス修飾子と名前空間を除いたもの。面白いことは何もありません。

ブレインファック


ここにリストアップしたくありません-ボリュームが多すぎて単調です。この例を理解するために学習する必要がある基本的な概念のみをリストします。キーワード、演算子、変数、ポインター、条件、ループ、文字の数値表現（ASCIIテーブル）。はい、Brainfuckは初心者にとって最もアクセスしやすい言語ではありません。

結論


もちろん、「簡単なエントリー」（初級プログラムの理解が容易）に加えて、トレーニング用のPLの選択に影響を与えるいくつかの要因があります。これは、コーディングに便利な環境とツールの可用性、ドキュメントの品質、そして最後に、実用性です。しかし、学習の最初のステップが痛みと誤解に関連している場合、これは他のすべての議論を上回る可能性があります。一般的に、簡単に学び、気にしないでください！
