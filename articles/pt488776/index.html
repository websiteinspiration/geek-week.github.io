Aprimoramentos de acessibilidade no Visual Studio 2019 para Mac



O lançamento do Visual Studio 2019 para Mac versão 8.4.4 inclui inúmeras alterações na paleta de cores de ícones, avisos e mensagens de erro. O novo visual é impressionante e o novo esquema de cores é ainda mais perceptível. Neste artigo, explicarei por que essas mudanças são realmente importantes e o que exatamente foi alterado.

Atualmente, mais de 1 bilhão de pessoas têm algum tipo de deficiência. As pessoas têm que lidar com vários problemas quase diariamente. Isso inclui limitações de mobilidade, distúrbios cognitivos e nervosos, problemas de fala e audição. Nesse sentido, queremos falar sobre melhorias de acessibilidade visual nas versões mais recentes do Visual Studio para Mac.

Segundo a Organização Mundial da Saúde, atualmente cerca de 200 milhões de pessoas vivem com algum tipo de deficiência visual. Nosso objetivo é tornar o Visual Studio para Mac acessível a todos. Para fazer isso, precisamos fornecer a melhor interface de usuário para cada usuário, independentemente de ele ter problemas de visão ou não. Existem muitos usuários com baixa visão, cor ou cegueira completa, catarata. A interface atualizada os ajudará a trabalhar mais plenamente no programa. Afinal, mesmo uma coisa comum como erupção solar pode ser um problema ao usar a interface do usuário do programa. Um dos métodos que permite que usuários com deficiência visual interajam com mais eficiência com os aplicativos é a disponibilidade de uma paleta de cores.

Uma das principais maneiras de criar uma interface de usuário acessível é aumentar o limite de contraste para todo o conteúdo interativo - principalmente texto e ícones. No Mac, a proporção de contraste entre o plano de fundo e o texto deve ser de pelo menos 3: 1 e, no modo de alto contraste, de 4,5: 1. Mais tarde retornaremos a este tópico.

Obviamente, não se pode confiar apenas nas diferenças de cores para mostrar a seriedade das informações. Por exemplo, nas alterações de estado entre ícones ativos e inativos. Isso significa que elementos como mensagens de erro ou avisos não devem usar apenas a cor de plano de fundo para transmitir seu status. Precisamos de algo mais: por exemplo, um sinal claramente visível de um erro ou aviso. Nas versões mais antigas do Visual Studio para Mac, geralmente a diferença de estado era exibida apenas em cores. Agora, estamos usando uma renderização mais clara dos ícones ativados, desativados e parados, sem depender apenas da cor. Excluímos essas situações no interesse de maior clareza visual.

Modo de alto contraste


Em um Mac, você pode ativar o modo de alto contraste visitando a seção Preferências de acessibilidade em Preferências do sistema e marcando a caixa de seleção no modo Aumentar contraste :



Alto contraste aumenta o contraste de cores de toda a interface. Os controles começam a usar traços e formas e rótulos facilmente visíveis. As cores são ajustadas para parecerem mais brilhantes, e a diferença de brilho entre o primeiro plano e o fundo é muito mais perceptível.

Infelizmente, nem todos os aplicativos em nossos PCs de mesa suportam o modo de alto contraste.

Obviamente, o uso de novas cores e ícones não é a única maneira de melhorar a acessibilidade visual. Também queríamos garantir a conveniência de trabalhar com leitores de tela e fornecer a capacidade de usar teclas de atalho. Ainda temos muitos planos, alguns dos quais falaremos hoje e o restante apresentaremos um pouco mais tarde. Neste artigo, focaremos na nova paleta de cores e no conjunto de ícones aprimorado, além dos novos recursos disponíveis para todos os usuários do Visual Studio para Mac.

Nova paleta de cores


Nossa antiga paleta de cores do Visual Studio para Mac, criada há muitos anos, usava taxas de contraste muito baixas, especialmente no tema claro do IDE. Portanto, é hora de corrigi-lo. Compare nossas paletas antigas e novas e as taxas de contraste de ícones e plano de fundo.



A paleta antiga tinha duas opções: uma para a luz e a outra para o tema IDE escuro. Como você pode ver acima, a paleta antiga sofria de muitos problemas, principalmente pelo contraste da cor dos ícones de aviso em um tópico claro, o que não era o ideal. Amarelo em branco ou cinza claro é muito difícil de ver.

A nova paleta corrige todos esses problemas e também é mais simples, com melhor significado semântico dos grupos de cores. Além disso, está pronto para funcionar no modo de alto contraste.

Emblemas aprimorados


Sempre tivemos muitos ícones no Visual Studio para Mac. Quando lançamos as alterações detalhadas neste artigo, ele tinha 1.142 emblemas. A maioria dos ícones foi apresentada em quatro versões: uma para um tema claro e escuro e duas para estados selecionados.

Agora, temos muito mais emblemas e um enorme trabalho foi feito para isso. Cada um deles teve que ser verificado quanto a problemas de disponibilidade, como descrito acima, convertido, duplicado e repintado usando uma nova paleta de alto contraste. Isso significa que não apenas introduzimos novos ícones com alto contraste, mas também melhoramos os existentes. O Visual Studio para Mac atualmente usa 13.704 arquivos de ícone.

Finalmente


As alterações descritas acima foram projetadas para facilitar a interface do Visual Studio para Mac para todos os desenvolvedores. Agora, temos não apenas ícones legíveis para usuários com problemas de visão, mas também nosso conjunto de ícones padrão já instalado, que demonstra um novo contraste mais perceptível nas cores, o que ajuda os usuários a não recorrerem aos recursos de acessibilidade. De qualquer forma, ainda temos muito trabalho, mas todos os dias o Visual Studio está melhorando.
