タスク：黄色さと対称性を表す2つのカテゴリ機能について、オブジェクトが属するクラス（リンゴまたは梨）を決定します（オブジェクトを分類するようにモデルを教えます）。

はじめに、データセットをアップロードします。

対数損失関数を使用します。関数の



左側のパラメーターは現在の重みw1、w2、w0による予測です。関数の

右側のパラメーターは正しい値です（クラスは0または1）

。σ（x）はxのシグモイド活性化関数です。log

（x）はxの自然対数です。

損失関数の値が小さいほど、重みw1、w2、w0が適切に選択されていることは明らかです。これを行うには、確率的勾配降下法を選択します。

SGDでは選択全体（またはミニバッチ勾配降下の場合のようにサブサンプル）ではなく1つの要素を選択するため、LogLossの式は異なるビューをとります。


ソリューションの進行状況：

初期の重みw1、w2、 w0はランダムな値

を設定しますデータセットのi番目のオブジェクト（たとえば、ランダム）を取り、そのLogLossを計算します（最初にランダムな値を割り当てたw1、w2、およびw0を使用）。次に、各重みw1の偏導関数を計算します。 w2とw0の場合、各重みを更新します。

少し準備：

import pandas as pd
import numpy as np

X = data.iloc[:,:2].values  #  -
y = data['target'].values.reshape((-1, 1))  #  (    )

x1 = X[:, 0]
x2 = X[:, 1]

def sigmoid(x):
    return 1 / (1 + np.exp(-x))


実装：

import random
np.random.seed(62)
w1 = np.random.randn(1)
w2 = np.random.randn(1)
w0 = np.random.randn(1)

print(w1, w2, w0)

# form range 0..999
idx = np.arange(1000)

# random shuffling
np.random.shuffle(idx)
x1, x2, y = x1[idx], x2[idx], y[idx]
# learning rate
lr = 0.001

# number of epochs
n_epochs = 10000

for epoch in range(n_epochs):

  i = random.randint(0, 999)

  yhat = w1 * x1[i] + w2 * x2[i] + w0
  
  w1_grad = -((y[i] - sigmoid(yhat)) * x1[i])
  
  w2_grad = -((y[i] - sigmoid(yhat)) * x2[i])
  
  w0_grad = -(y[i] - sigmoid(yhat))
  
  w1 -= lr * w1_grad
  
  w2 -= lr * w2_grad
  
  w0 -= lr * w0_grad
 
print(w1, w2, w0)

* _gradは、対応する重みに関する導関数です。一般式を書きます。



自由項w0の場合、係数xは省略されます（1に等しいと見なされます）。

導関数の最終式によれば、損失関数を明示的に計算する必要がないことがわかります（偏導関数のみが必要です）。

np.around（x）-xの値を丸めます。私たちの場合：x> 0.5の場合、値は1です。x≤0.5の場合、値は0

です。オブジェクトの属性の数が5の場合はどうしますか？ 10？ 100？そして、適切な量の重み（バイアス用に1つ）を用意します。各重みを手動で操作し、その勾配を計算するのは不便であることは明らかです。

人気のあるPyTorchライブラリを使用します。

PyTorch = NumPy + CUDA + Autograd（勾配の自動計算）

PyTorchを使用した実装：

import torch
import numpy as np 
from torch.nn import Linear, Sigmoid

def make_train_step(model, loss_fn, optimizer):
    def train_step(x, y):
        model.train()
        yhat = model(x)
        loss = loss_fn(yhat, y)
        loss.backward()
        optimizer.step()
        optimizer.zero_grad()
        return loss.item()
    return train_step

X = torch.FloatTensor(data.iloc[:,:2].values)  
 
y = torch.FloatTensor(data['target'].values.reshape((-1, 1)))

from torch import optim, nn

neuron = torch.nn.Sequential(
    Linear(2, out_features=1),
    Sigmoid()
)
print(neuron.state_dict())

lr = 0.1
n_epochs = 10000
loss_fn = nn.MSELoss(reduction="mean")
optimizer = optim.SGD(neuron.parameters(), lr=lr)
train_step = make_train_step(neuron, loss_fn, optimizer)

for epoch in range(n_epochs):
    loss = train_step(X, y)
    
print(neuron.state_dict())
print(loss)
