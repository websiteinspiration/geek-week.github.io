思考の力で明かりをつける


アルゴリズムを書くことは、ホームオートメーションにおいて私にとっておそらく最も興味深い部分です。しかし、センサーとスクリプトの全体でさえ、生命のあふれんばかりの想像力に対処できず、直接制御の方法を追加する必要があります。手動制御として、通常のドアベルを置くことができますが、さらに必要な場合はどうしますか？この記事の主人公、Silicon LabsのSI1143に基づくジェスチャーセンサーに出会います。

SI1143自体：



本当に未来的に見えます。

そして、いくつの利点があります：

  • 高感度フォトダイオード、低ノイズADC
  • 6〜360ミリアンペアの電流のLEDドライバー
  • 1.71から3.6ボルトまで電力を供給
  • およびスマートI2C-1秒あたり最大3.4メガビット

これは、わずか5 x 3ミリメートルのサイズです。あなたがそれを見るのはこれが初めてではありません。さて、ここでもう少しストラップが必要です。



これは削除できますか？私の美意識は尋ねました。そして、このレアゲストのご要望にお応えして、ボードをビル内に隠していきますが、その間、力を加えて何が起こっているのか見ていきましょう。
注意深いコメント。実際、予想よりも詳細がいくつかあります。Enviriotプロジェクトのノードがベースとして使用されました。STM32F051マイクロコントローラーに加えて、このボードにはテキサスインスツルメンツの868メガヘルツCC1101トランシーバーもあり、ファームウェアを入力すると、電源を供給するのに十分であり、デバイスはMQTT-SNサーバーに接続します。

動作原理は、3つのLEDのそれぞれからの反射信号のレベルを測定することに基づいています。オンにしてレスポンスを確認します。



LEDの非対称配置のため、LED 1からの信号は著しく大きく、最初に測定結果を正規化する必要があります。



したがって、依存関係はすでにかなり理解できます。次のアルゴリズムをコーディングしてみます。

紙の上では滑らかでしたが、スピーカーで問題が発生しました。同じ高さに固定された白い紙の場合、期待された結果は3つのうち2つのケースで得られました。私が手を振ろうとすると、信号がジャンプし始め、私の素晴らしいアルゴリズムが読み取り値で混乱し始めました。手を見たのですが、白い紙と違いがたくさんあります。しかし、どういうわけかそれはうまくいくはずです。はい。説得された！ドキュメントを読んでみます。


被災者のために、Silicon LabsはAppNote AN580-「赤外線ジェスチャーセンシング」をリリースしました。ジェスチャーとその可能な組み合わせを決定するための2つの基本的な方法について説明します。最初の方法は、各瞬間の位置の決定と座標に基づくジェスチャーの決定です。 2番目の方法は、信号間の位相シフトを決定します。最初の方法の変形の1つはすでにテストされており、印象的ではありませんでした。 2つ目を実装してみます。

行け。 2点まで実装する必要があります。エントリの瞬間を特定し、このデータをステートマシンに転送します。入力しきい値は、総信号の1/8で実験的に決定されました。ノイズから保護するために、シュミットトリガーを追加し、15％オンにして、10％オフにします。

まあ、状態マシン自体。状態1から3-上向きの動き、状態-1から-3-下向きの動き、両方のLEDが同時に動作するというまれなイベントの状態4。



さあ、これですべてを始めましょう。

var LSt = new Int8Array([-3, -3, -3,  1,  1,  3,  3,  4, 
                         -3, -3, -1, -1,  3,  3,  3,  4, 
                         -2, -2, -2,  4,  2,  2,  2,  4]);
/*   ......   */
this.r1 = false; this.r3 = false;
this.button = new Int8(0);
/*   ......   */
if(n1 > 15) { this.r1 = true; } else if(n1 < 10) { this.r1 = false; } 
if(n3 > 15) { this.r3 = true; } else if(n3 < 10) { this.r3 = false; } 

let st = (this.r1?1:0) | (this.r3?2:0);
if(st == 0){  // .     
  this.button = 0;
} else {
   this.button = LSt[st*8 + this.button - 5]; 
}

変数n1とn3には、対応するLEDの正規化された値が含まれています。結果はボタンフィールドにあります。

さらに使用するには、状態2-upおよび-2-downが役立ちます。

プログラムはJavaScriptのサブセットで記述され、バイトコードにコンパイルされてデバイスにアップロードされます。JavaScriptは解析され、ASTは同志のNiL.JSライブラリによって生成されますIaIojek、彼に感謝します。



状態2が0.3秒未満続く場合、A14およびA15ブロックがこれを担当し、明るさが最大に設定されます。ブロックA01およびA13は、明るさを0に設定して、短い下向きの動きを実現します。

上に移動してブロックA10、A09、A04を保持すると、明るさが滑らかに増加します。押し下げたままにすると、ブロックA12、A11、A08が機能し、レベルを下げることができます。
