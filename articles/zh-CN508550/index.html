脑结构。渠道概念







在本文中，我将讨论关于大脑结构的一种假设。







推理基于非常简单的渠道概念。也许读者甚至会发现它很幼稚。但是，我不得不付出很大的努力来解决这个问题。这不是科学文章。如果有人称它为关于大脑主题的另一种推测，那么从形式上来说是正确的。欢迎批评。







介绍



该假设的历史始于2015年。再次思考大脑的结构，我想到了一个主意。其本质是大脑必须以某种方式建立环境模型。需要一个模型来打破受体和效应子之间的直接联系。没有这种模型的生物几乎完全依赖于外部信号。可以乘坐出租车或随意散步。从某种意义上说，这样的生物就像一个无生命的物体。例如，只有当球从足球运动员的脚上收到“外部信号”时，球才会移动。这与传统的单细胞沿化学梯度向营养物质移动没有太大不同。他的行为同样完全取决于外部信号。您可以通过在受体和效应子之间放置记忆来改变这种模式，它存储了环境的原始模型。

class dotdict(dict):
    __getattr__ = dict.get
    def __setattr__(self, name, value):
        self[name] = value

class Machine:
    """
         tick.
            tock.
     tick  tock        .
    """

    def __init__(self, cells):
        self.cells = cells
        self.tick = dotdict()
        self.tock = dotdict()
        for key in cells:
            self.tick[key] = 0b0000
            self.tock[key] = 0b0000

    def step(self):
        """    """
        for name, func in self.cells.items():
            if func is None:
                #  ,     
                self.tock[name] = self.tick[name]
            else:
                #       
                self.tock[name] = func(self.tick)
        #   
        self.tick, self.tock = self.tock, self.tick
        self.show()

    def show(self, comment = None):
        """    """
        if comment is not None:
            print(f': {comment}')
        for key in cells:
            print(key, f'{self.tick[key]:04b}')
        print('------------------------------------------------')
