プリント基板の調整ビア

ビアの設計について話しましょう-本格的な電子機器では、ビアの品質は非常に重要です。記事の冒頭で、信号の完全性に影響を与える要因を強調し、次に、単一ビアと差動ビアのインピーダンスの計算と調整の例を示しました。



皆さんこんにちは、私の名前はビャチェスラフです。私は5年間プリント基板を開発してきましたが、この間、トレースに関する多くのルールや推奨事項を読むだけでなく、ソースを見つけてそれらを操作しました。YADRO社を開発している

複雑なコンピューターシステムでは、トランスミッターからレシーバーまでのパス上の高速信号が長距離を移動し、複数のボードを通過して数十のビアを形成します。このような状況では、不注意に設計された各ビアが信号の劣化にわずかに寄与し、その結果、インターフェイスが機能しない可能性があります。

シグナルインテグリティ


ビア（以下、p / o、英語のビア）は、伝送ラインの不均一性です。他の異質性のように、彼らは信号を台無しにします。この効果は低周波数ではわずかに顕著ですが、周波数の増加とともに大幅に増加します。多くの場合、開発者はビアの構造にあまり注意を払っていません。ビアは「隣接」プロジェクトからコピーしたり、データシートからコピーしたり、CADでまったく指定しなかったり（デフォルト設定）できます。

計算された構造を使用する前に、なぜそのように作成されたのかを理解する必要がありますか？盲目的な繰り返しは害を及ぼすだけです。

ビアを通過するときのチャネル内の信号の整合性は、主に次の要因の影響を受けます。

  • 波動インピーダンスの変化による信号反射;
  • 浮遊容量とインダクタンスによる信号の劣化;
  • 内層への移行中の半プラスチックの未使用のストレッチからの反射（以下、英語経由のスタブと呼ばれるスタブ）
  • クロストーク（英語のクロストーク）;
  • 電力バスの干渉。

これらの影響の原因とその排除方法について、さらに詳しく検討してみましょう。

プリント回路基板の導体は、広い範囲の波抵抗で作成できますが、ほとんどの場合50オームです。一方では、これは過去の連続性によるものです。50Ωインピーダンスは、ドライバーの負荷レベルと信号エネルギー損失の間の妥協として、同軸ケーブル用に標準化されました。一方、50Ω導体は、標準ボード上で簡単に製造できます。

開発者にとって重要なのは、伝送線路全体の不変性ほど、波抵抗の特定の値ではありません。

波動インピーダンスの値が固定された伝送ラインを作成するために、開発者はトラックの幅と基準層までの距離を選択します。伝送ラインの線形容量とインダクタンスを特定の値に変更します。

p / oでは、誘導成分が非常に重要です。最初の近似では、合理的な枠組みの中で、浮遊インダクタンスを可能な限り最小化し、次に、p / oパラメータを変更して、所定の静電容量、つまりインピーダンスを実現する必要があります。

半波の静電容量が過度に減少すると、インピーダンスが局所的に増加し、結果として信号反射が発生します。

ファクター3.スタブ経由


信号がスタブスルーホールを通過するとどうなりますか？


図3.スタブのある遷移ホール、1/4波長での共振。

この例では、信号はトップレイヤーから上から下に伝播します。内側の信号層に到達すると、信号が分離されます。一部は内側の層のパスに沿って移動し、一部はビアを下に移動し続け、その後、下側の層から反射されます。反射された信号が内層に到達した後、再び分割され、一部はパスに沿って移動し、一部はソースに戻ります。

反射された信号は元の信号と加算されて歪められます。これは、アイダイアグラムのウィンドウを狭め、挿入損失のレベル（Eng。Insertion Loss）を増加させることで表現されます。

最悪の場合、TDセグメントは信号の波長の1/4に等しくなり、反射信号は周期の半分の遅延で内層のパスに到達し、元の信号と逆位相で重なります。

完全性を分析するときは、5ナイキスト周波数の帯域幅を考慮することをお勧めします。良好な近似は許容可能なスタブと見なされ、7倍以上の高調波で共振が発生します[2]。


図4.スタブ0、0.65、1.2 mmでのp / oの挿入損失レベルのグラフ。

図4は、約24 GHzの周波数での巨大な共振を示しています。信号が2〜3 GHzの周波数で動作する場合、7つの高調波内で「すべてが穏やか」であるため、スタブを排除する余裕はないと結論付けることができます。Polar計算機で

スタブの重要度をすばやく評価できます。：


図5. polarinstruments.comからの画像。立ち上がり時間が500 psを超える信号には、2.5 mmのスタブ長が許容されます。

記事[2]で与えられている式は、わずかに正確な結果を与えます。これらは、p / oジオメトリを考慮に入れ、Z軸に沿った誘電体の誘電率の補正を計算できるようにします。

「バックドリル」操作（英語のバックドリル）またはマイクロ遷移（英語のブラインドおよび埋め込みビア）を使用して、スタブを削除できます。選択は、プロジェクトの機能によって異なります。リバースドリルはより簡単で安価です。ボードを製造した後、直径の大きなドリルを使用して、スタブを所定の深さまで穴あけします。開発者は、掘削ゾーンでトポロジの追加のインデントを指定する必要があります。また、製造業者は、設計ドキュメントで掘削要件を示すこともできます。最新のCADシステムはこの機能をサポートしています。

マイクロジャンクションは、主に高密度ボード（HDI）を対象としていますが、場合によっては、ドリルバックを拒否するコストを平準化し、ボード上のレイヤー数を減らすことで使用できます。HDIボードを開発するときは、次の点に注意してください。

  • 新しい種類の給与は、取締役会のコストを増加させます。
  • レーザー穴あけには、特別に最適化されたプリプレグが使用されますが、その特性はさまざまです。
  • 止まり穴のメタライゼーションにより、内層の銅の厚さが増加します。

ボード構造については、事前にメーカーと合意しておくことを強くお勧めします。

ファクター4.クロストーク


クロストーク-あるラインから次のラインへの不要な信号伝送。この転送が発生するのは、2つの間隔の狭い導体が容量性と誘導性の結合を持っているためです。

信号導体とp / oのクロストークの性質は少し異なります。
信号にはn / aに参照層がなく、戻り電流は隣接するn / aに沿って流れ、大きなループを形成します。誘導成分によるn / aの信号のクロストーク。

クロストークの最小化への最大の効果は、p / o間の距離を増やすことで達成できます。ただし、トポロジーには多くのスペースがありません。

差動ペアでのp / oの収束は、占有面積を削減するだけでなく、耐ノイズ性にもプラスの効果があります[3]。

隣接する信号p / o間のクロストークを最小限に抑える一般的な方法は、それらの間にシールドp / oを配置することです。この方法では、約2 mm刻みで信号を伝送する必要があります（図6）。十分なスペースがない場合は、図7に示すように、シフトの小さいステップ（英語の千鳥パターン）を使用できます。シミュレーションを使用して、理想的なシフト角度を選択できます[4]。


図6.シールドハーフワイヤを使用したクロストークの最小化。


図7.斜めの「チェッカーボード」シフトを使用したクロストークの最小化。

クロストークは、エキゾチックな方法、たとえば、長いスタブ（誘導-容量バランスp / oの変位による）によっても低減できます[5]。また、超小型回路の筐体を設計する段階で干渉を低減することができます[6]。

ファクタ5.パワーバス干渉


隣接する信号回路に加えて、内部層からの干渉が信号品質に影響を与える可能性があります。

大電流が食品試験場を流れる可能性があります。ポリゴンのエッジでのインダクタンスの増加により、流れる電流は、カットアウトを含めて、ポリゴンのすべての境界に沿ってエッジフィールド（英語のフリンジフィールド）を形成します。エッジフィールドは、空間への電磁放射（生まれたエッジ発射）の発生源です。電磁放射線の放出を低減するために、陸地面積に対して電源面積を狭めることにあるルール20Hが適用されます（図8）。


図8.エッジフィールドと20Hルール。

干渉からp / oを保護するために、可能であれば、食品の範囲でアンチパッドを増やす必要があります。p / oの20Hルールを提供することは困難であり、不必要に、通常、直径約2 mmの落下防止が推奨されます（図9）。


図9.食品層の落下防止の向上

シングルビアのインピーダンス計算


p / o要素のインピーダンスへの影響に関する知識に基づいて、理想的なp / oを設計できます。計算機でインピーダンスを計算するとよいでしょう。PCB設計

エンジニアは、Saturn PCB Design ToolkitやPolar Instruments Si9000eなどの計算機が好きです。どちらも、1つのp / oのインピーダンスをすばやく計算できます。

これらの計算機で得られる結果は互いに非常に異なります。これは、これらのツールのアプローチが異なるためです。

Polarは、電源ラインが電力範囲と交差する2次元平面でインピーダンスを計算します。計算式はありません。計算は、同軸ケーブルのインピーダンス公式に従って行われることが実験的に確立されました。

z0=138APVPEr1





図10. polarinstruments.comからの画像

この図は、誘電率Er1が標準と比較してかなり低い値を示しています。これは、誘電体構造の不均一性によるものです。樹脂（Er 3.2）とグラスファイバーフィラメント（Er 6.1）で構成されているため、平均誘電率は約4.1です。この値は、ローカルでかなり異なる場合があります。そのため、樹脂は表面近くに存在するため、誘電率は下向きに再計算されます[7]。

土星PCBは、次の式でインピーダンスを計算します。

z0=ViaInductanceViaCapacitance∗0.001

最終的な推奨事項


プリント回路基板のビアは、複雑で異種の構造です。パラメータを正しく計算するには、高価な3Dソルバー、コンピテンシー、および多大な時間の投資が必要です。

重要な信号から他の層への遷移の使用を回避できない場合は、最初に、発生する不均一性が信号の整合性に及ぼす影響の度合いを評価する必要があります。不均一性が電気的に短い場合（遅延時間がフロントの1/6未満）、スタブは通過帯域外の周波数で共振します。最適化に時間とお金を浪費しても意味がありません。

最初の近似では、データシートまたは以前のボードから既製の構造を使用すると便利ですが、現在のプロジェクトの機能を覚えておいてください。

計算機を使用すると、ソフトウェアのパラメーターをすばやく評価できますが、計算に悪影響を与える高度に簡略化されたモデルを使用します。
