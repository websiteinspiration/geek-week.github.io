最大のIT企業の1つがCNCFに参加した理由-クラウドインフラストラクチャを開発するファンド

1か月前に、AppleはCloud Native Computing Foundationに参加しました。これが何を意味するのか理解しています。


写真-Moritz Kindler -Unsplash

CNCFを選ぶ理由


Cloud Native Computing Foundation（CNCF）は、Linux Foundationをサポートしています。その目標は、クラウドテクノロジーの開発とプロモーションです。この基金は2015年に、大規模なIaaSおよびSaaSプロバイダー、IT企業、およびネットワーク機器のメーカー（Google、Red Hat、VMware、Cisco、Intel、Dockerなど）によって設立されました。

今日、基金のメンバーには、アディダス、GitHub、ニューヨークタイムズなどの組織も含まれています。1か月前にAppleが参加しました。プラチナステータスを取得しており、オープンプロジェクトの開発に年間37万ドルを支払う予定です。

アップルとオープンソースプロジェクトにはかなり長い歴史があります。同社は、製品開発でオープンソースソフトウェアを積極的に使用した最初の企業の1つでした。例としては、OS Xがあります。このオペレーティングシステムは、別のOS-Darwinのコンポーネントに基づいています。これは、組み合わせのNeXTSTEPとFreeBSDから受信したことで、アップル自身が書いたコードを。

CNCFとLinux Foundationの代表は、オープンファンドに参加することにより、アップル社はその専門知識を共有したいと語っています。エンジニアは、オープンソースコミュニティに彼らの仕事に返済し、クラウドITインフラストラクチャの開発に貢献したいと考えています。Appleの担当者は通常の方法で、企業の決定についてコメントしません。

それは何に影響しますか


クラウド開発はより速く進みます。 Kubernetesコンテナオーケストレーションシステム、Prometheusインフラストラクチャ監視ツール、CoreDNSサーバー、Envoyプロキシサービスなどのプロジェクトは、CNCFから生まれました。 CNCFに参加する前から、Appleはその開発（特にKubernetes）に積極的に関与していました。

クラウドネイティブコンピューティング財団のメンバーになることで、企業は同僚とより緊密にコミュニケーションできるようになります。プラチナのステータスにより、クラウドツールの開発のベクトルを決定する際には、アップルの代表者の意見が考慮されます。 CNCFは現在、クラウド内の本番環境とファイル、およびメッセージングを保護するために、他の15のプロジェクトに取り組んでいます。アップル社の調査はそれらの開発を加速するかもしれません。


写真-Moritz Kindler -Unsplash

より多くのオープンソースプロジェクトがあるでしょう。アップルは、既存のプロジェクトの開発を支援し、新しいプロジェクトをもたらします。同社はすでに、前述のダーウィンのコンポーネントであるオープンソースカーネルXNUと、現在TIOBEランキングで13位にあるSwiftプログラミング言語に移行しています。

1年前、アップルは、分散型NoSQLデータベースであるFoundationDB のソースコードを明らかにしました。他の同様のシステムとは異なり、FoundationDBの操作はACIDの原則（原子性、一貫性、分離、データの耐久性）に準拠しています。

数週間にわたって、7000人を超える開発者がプロジェクトに関心を示し、 100の新しいスレッドがフォーラムで開かれました。同社は、コミュニティとさらに新しいオープンツールを開発する予定です。

最近CNCFに参加した人


今年3月、CNCFの代表は、59の新しい組織がコミュニティに参加したことを発表しました。5月末、ファンドの参加者数は400社を突破しました。その中には、小規模な新興企業と大規模なIT企業の両方があります。

たとえば、クラウドでの人工知能システムの開発に従事するNvidiaは、ファンドの新しいメンバーになりました。注目すべきは、Elastic-Elasticsearch、Kibana、Beats、Logstashで構成されるスタックの開発者-通信機器メーカーのEricssonです。

これらの組織に加えて、いくつかのクラウドプロバイダー、インターネットサービスプロバイダー、コンサルティングエージェンシー、インテグレーター、情報セキュリティ会社もリストされています。

Cloud Native Computing Foundationは、新規参入者とそのテクノロジーがクラウド市場の発展に貢献し、オープンソースエコシステムに貴重な専門知識をもたらすと確信しています。
