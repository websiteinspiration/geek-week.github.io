Majelis 486 - pemilihan komponen

Untuk memerangi nostalgia - sehingga tidak lagi mengunyah saya ketika kejang terjadi - saya memutuskan untuk mendapatkan tiga komputer retro.

Saya sudah menulis tentang yang pertama - ini adalah laptop Compaq dengan prosesor Pentium. Ini adalah mesin compact universal, yang, pada prinsipnya, dapat menyelesaikan semua tugas dosovskie dan inisial di bawah Windows 9x. Jika Anda meninggalkan satu hal, maka saya akan terbatas pada mereka. Tetapi saya masih ingin mengumpulkan pasangan lagi.



Salah satunya - sesuatu dari zaman Pentium3 - saya mulai mengumpulkan, tetapi menyingkir - ide yang sangat menarik (bagi saya pribadi) muncul dan Pentium3 telah disimpan untuk saat ini. Anda bisa mengotak-atiknya untuk waktu yang lama, pilihan komponennya cukup besar.

Komputer kedua lebih menarik bagi saya - itu bukan pengulangan dari yang lama, tetapi sesuatu yang baru - lebih tepatnya, mengisi kekosongan. Saya ingin mengumpulkan 486. Dan selalu dengan bus VLB. Karena pada waktunya saya melewatkan tahap ini dan segera melompat dari ISA ke 386 pada PCI 486 dan Pentium yang lebih tua. Jadi diputuskan untuk melakukan upaya ke arah ini.

486 dengan bus PCI, saya pikir tidak terlalu berbeda dari Pentium - pengontrol terintegrasi, periferal yang sama, kecuali bahwa prosesor lebih kecil dan mungkin ada masalah dengan memori-EDO. Tidak tertarik. VLB adalah masalah lain. Tentu saja, mereka mengatakan bahwa ada pentium awal dan akhir 386 dengan VLB, tetapi bahkan jika itu benar, maka ini juga tidak menarik bagi saya.

Jadi 486 komputer akan menjadi sekitar 1993-1995. DOS, Windows 3.xx, mungkin Windows 95 dan poros setengah lainnya, tetapi tidak di tempat pertama.

Saya tidak terpaku pada keaslian, karena jika ada pertanyaan tentang pilihan antara "persis seperti dulu" dan "bagaimana itu lebih nyaman bagi saya hari ini" - saya memilih yang kedua.

Meskipun, tentu saja, jika memungkinkan, saya mencoba melakukan apa adanya.

Saya tidak mengejar sepotong besi tertentu, saya punya daftar prioritas dan anggaran terbatas, oleh karena itu saya melihat apa yang berjalan di masa lalu, saya tertarik pada harga dan seberapa tinggi itu dalam daftar prioritas. Jika harga memuaskan, dan posisi dalam daftar tinggi, maka dia mengambilnya. Jika sesuatu tidak sesuai, maka lepaskan.

Papan induk


Ada beberapa upaya untuk mendapatkan papan yang cocok. Ada 2-3 slot VLB dalam persyaratan, setidaknya dua slot untuk sim 72-pin, baterai 2032 atau setidaknya barel yang tidak dioksidasi - masalah utama motherboard lama. Nah, cache yang berfungsi, mereka mengatakan hal-hal buruk tentang papan dengan cache palsu. Saya berpikir tentang papan dengan PCI dan VLB - ada yang seperti itu, tetapi ini bukan monster yang sangat sukses, jadi saya menolak mendukung VLB murni.

Akibatnya, saya memilih papan Data Expert EXP4045 - saya mengaturnya untuk harga, fitur, dan penampilan. Dia memiliki pertanyaan dengan prosesor yang lebih tua pada socket3-5x86, dll., Tetapi mereka tidak benar-benar menarik minat saya. Ada tegangan 3,3 volt, Anda bisa mendapatkan 100-120 megahertz - itu cukup bagi saya. Semua slot memori 72-pin - ok, memori 30-pin hanya menarik bagi saya untuk AWE32.



Alih-alih barel baterai, ada sekitar baterai yang sama - sudah mati - saya mungkin akan menggantinya dengan barel baterai. Misalkan ada konektor untuk baterai eksternal.

RAM


"Kami sudah bersama kami." Saya tidak benar-benar mencari apa pun di sini, saya menggali sekantong sim di deposito saya - ternyata ada dua modul memori FPM masing-masing 32 megabytes (tidak semua 486 didukung EDO). Delapan lainnya kemudian tersesat. Sebagai hasilnya, saya mulai berpikir bahwa saya akan menempatkan minimal 2 * 32, dan jika mungkin 2 * 32 + 8 dan mendapatkan RAM besar untuk 486 72 megabyte.

Semua jenis Cirrus Logic mengikutinya dengan cepat, tetapi di sini saya menginginkan hal yang aneh - sehingga ketika Anda menghidupkan komputer, surat berwarna tentang Trident bersinar. Sisa-sisa S3, yang memberinya kenangan kepada leluhurnya. Ngomong-ngomong, sudah jelas bahwa papan PCI mulai meletakkan mikrosirkuit pada sisi terbalik relatif terhadap ISA - jika Anda memasang mikrosirkuit ke atas, maka konektor di sisi lain diperoleh.



Pengontrol


Integrasi besar-besaran pengontrol port dan perangkat disk dimulai hanya pada zaman PCI, sebelum itu merupakan case yang terisolasi. Oleh karena itu, multicard dengan port, disk, dan pengontrol hard disk wajib di setiap unit sistem.

Pilihan di sini adalah antara ISA dan VLB, serta antara IDE dan SCSI. ISA IDE lambat, VLB SCSI mahal.

Karena saya berhenti di VLB IDE. Selain itu, demi ini, semuanya dikandung - perangkat di VLB. Pada saat yang sama, saya membandingkan kinerja pengendali ISA dan VLB.



Saya juga akan menambahkan kartu jaringan ke titik ini - 3Com Etherlink III yang relatif modern, bahkan tanpa konektor BNC.
