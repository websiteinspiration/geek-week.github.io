Pythonで進行状況を追跡する

なぜ進捗インジケータが必要なのですか？


進行状況インジケーター（進行状況バー）-作業プロセスの視覚的表示。彼らは、スクリプトがハングしているかどうかを心配する必要をなくし、スクリプトの実行速度を直感的に示し、完了までの残り時間を提案します。


以前に進行状況インジケーターを使用したことがない人は、その実装がコードを非常に複雑にする可能性があることを示唆している可能性があります。幸い、これはそうではありません。以下の小さな例は、コンソールまたは急速に普及しているPySimpleGUIグラフィックライブラリのインターフェイスで進行状況の追跡をすばやく簡単に開始する方法を示しています。


進捗状況の使用


最初にProgressモジュールがあります。


必要なのは、予想される反復の数、インジケーターのタイプを示し、各反復で関数を呼び出すことだけです。


import time
from progress.bar import IncrementalBar

mylist = [1,2,3,4,5,6,7,8]

bar = IncrementalBar('Countdown', max = len(mylist))

for item in mylist:
    bar.next()
    time.sleep(1)

bar.finish()

import PySimpleGUI as sg
import time

mylist = [1,2,3,4,5,6,7,8]

progressbar = [
    [sg.ProgressBar(len(mylist), orientation='h', size=(51, 10), key='progressbar')]
]
outputwin = [
    [sg.Output(size=(78,20))]
]

layout = [
    [sg.Frame('Progress',layout= progressbar)],
    [sg.Frame('Output', layout = outputwin)],
    [sg.Submit('Start'),sg.Cancel()]
]

window = sg.Window('Custom Progress Meter', layout)
progress_bar = window['progressbar']

while True:
    event, values = window.read(timeout=10)
    if event == 'Cancel'  or event is None:
        break
    elif event == 'Start':
        for i,item in enumerate(mylist):
            print(item)
            time.sleep(1)
            progress_bar.UpdateBar(i + 1)

window.close()
