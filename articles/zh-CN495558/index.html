明天是乌云密布：云层将具有智能，使聘用IT专家的工作变得复杂，但它们将有助于销售软件



在年度云计算预测中， Forrester副总裁兼首席分析师Dave Bartoletti表示，到2020年，云计算市场（即SaaS，PaaS和IaaS）将增长到2994亿美元。但是，这实际上意味着什么？对于业务和IT专业人员，这将发生什么变化？Mail.ru云解决方案团队收集了专家预测以了解情况。  

主要发展趋势


云开发人员赚更多钱 


“ 云应用程序是开源软件开发人员成功的获利策略。第一代开放源代码公司使用了付费的客户支持模型和培训，但事实证明它不能很好地工作。 

2020年，按使用付费的云服务将成为开源公司的领先商业模式。云提供了灵活性，速度和可伸缩性，它们的使用具有回报。”

InfluxData首席执行官Evan Kaplan。

开发和启动应用程序变得更加容易。


“ 2020年将致力于简化应用程序的开发和部署。在2019年，我们看到开发人员转向了从复杂性中抽象出来的技术，并以它们所处的形式工作。我预计这一趋势将继续下去。我们还将看到Kubernetes的新一轮流行。 ”

Cloud Foundry基金会执行董事Abby Kearns。

Kubernetes将在有大数据的任何地方开始使用


“ Kubernetes已成为受容器化支持最受关注的技术。将来，所有大数据技术都将在其上运行。也许这种情况不会在2020年发生，但是随着越来越多的大型提供商在其软件上部署其平台，Kubernetes的普及程度将会提高。 

有一些缺点需要解决：例如，持久性存储问题，但是目前正在通过诸如BlueK8之类的举措来解决。

Unravel Data首席执行官Kunal Agarwal。

业务主要趋势


供应商将帮助企业竞争数字化


“公司的问题之一是如何保持巅峰状态，不断引入创新，而这些创新正在变得越来越多。现代技术世界的速度惊人，企业消费者很难跟上时代的步伐。

因此，云提供商将扩展支持和培训服务，以帮助公司应对这种疲劳，迁移到云中，并使其更易于实施人工智能和机器学习等高级服务。”

CSS Insights的副总裁兼公司治理和人工智能研究主管Nick McQuire。

PaaS赢得盒子解决方案


“谈到DevOps时，公司不仅需要工具。他们正在寻找可与当前工具结合并为他们增加价值的现成平台。例如，以人工智能（AI），机器学习和分析的形式。 

2019年，公共云的实施水平为94％，私有云的开发水平为72％，因此我预测到2020年，我们将看到向灵活PaaS平台的过渡。这样的平台比盒装解决方案更容易适应公司的需求。”

Tej Redkar，LogicMonitor产品总监。

云使IT招聘更加困难


“根据研究，大多数公司都面临这样一个事实，即现有员工无法应对90％的在云中实现的任务。结果，您必须寻找新的。

流行的多云方法通常需要针对每个使用的云平台访问专家和专用工具。许多公司几个月都无法寻找具有必要经验的工程师。“您必须雇用平庸的人或投入大量资金来培训自己的员工，这也可能需要很长时间。”

AWS HeleCloud首席执行官，咨询合作伙伴首席执行官Dob Todorov。 

数字化具有感染力：甚至银行和药品也将被淘汰


零售业，金融业，医疗保健，医药和其他行业将转向纯云模式。也就是说，他们开始考虑将云基础架构构建为唯一的业务开发方案。 

这是因为云服务使您可以切换到现代架构并升级IT系统。并且还使用现成的组件：PaaS（平台即服务）和市场中的应用程序-与设计器多维数据集一样，它们使得可以组装满足业务需求的基础结构。作为基础架构砖，您可以使用数据库，机器学习环境，监视系统，应用程序容器等等。 

此外，在云中测试新假设并更快地实施创新技术（大数据，人工智能，物联网等）的成本更低。” 

Mail.Ru云解决方案主管Ilya Letunov。

云不再是趋势，而是转变的实用工具


“以前，公司经常谈论什么是云计算，他们可以做什么，为什么需要它们。现在，他们真正想到了如何使用云来提高工作效率，更快地将产品推向市场，分析数据并节省IT系统。 

公司希望创新，开发新的服务和应用程序，而不是浪费精力维护基础架构。云技术有助于减少支持IT系统的预算。”

IBM软件集团销售战略与转型副总裁Lauren States：



建立IT基础架构的主要趋势


许多公司将同时处理多个云。


“关于多云的讨论很多，但是很少有公司完全采用多云策略。大多数企业在多个公共云平台上部署应用程序。但是，它们通常将一个应用程序整体完整地存储在一个云中，而另一个则完整地存储在另一云中，而不是将一个应用程序部署在多个云中。 

2020年，情况将发生变化：越来越多的应用程序将部署在两个或更多的公共云中。这种方法使您可以更好地利用每个云的独特功能。它还提高了应用程序的可用性：如果一个云关闭，该应用程序将在另一个云中运行。另一个优点是能够遵守法律要求，例如在放置应用程序和数据时。”

Volterra首席执行官Ankur Singla。 

但这并不准确：许多人也只会选择一个提供者 


在过去的几年中，大型公司越来越选择多云。但是接下来，我们将看到组织如何放弃这种方法，并继续选择一个战略性的云服务提供商。可以满足企业数字化转型的所有需求的产品。

“我们在2019年看到的一些最大的云交易是与选择了一家云提供商的公司达成的。”

CSS Insights的副总裁兼公司治理和人工智能研究主管Nick McQuire。

云变得更智能并处理PB级数据


“由数据量增加引起的转变导致了技术的发展，例如使用人工智能创建的“智能云”平台和“ 智能边缘计算 ”，即使用AI进行数据处理离其收集地点越近，例如，到物联网传感器的水平。这为分析和数据使用提供了新的机会。”

微软首席执行官Satya Nadella：



混合云可帮助处理业务数据


“混合云是两个或多个云环境的组合。我们看到，大型企业没有在自己的基础架构中开发数据处理解决方案（例如Hadoop），而是在公共云上进行投资。 

但是，他们仍然不希望在那里传输基本公司数据。这些数据将保留在公司内部，而计算将进入云，这对于不可预测的工作负载尤其重要。”

 李浩远，Alluxio的创始人兼首席技术官。

分布式云将成为公共云发展的新水平


分布式云-公共云服务在提供商的物理数据中心之外的位置，而它们仍由提供商控制。云服务提供商仍然负责云服务架构，管理和更新的所有方面。从集中式公共云到分布式公共云的转变引发了云计算的新时代。 

分布式云使数据中心可以位于任何地方。这解决了技术问题以及立法法规方面的问题。”
