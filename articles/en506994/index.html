Regular Cheat Sheet



Good day, friends!

I present to you the translation of the article “Regex Cheat Sheet” by Emma Bostian.

Regular expressions or "regex" are used to find matches in a string.

Looking for a pattern match

We use the .test () method

const testString = 'My test string'
const testRegex = /string/
testRegex.test(testString) // true

Looking for a match across multiple patterns

We use | - alternation

const regex = /yes|no|maybe/

Ignore case

Use flag i

const caseInsensitiveRegex = /ignore case/i
const testString = 'We use the i flag to iGnOrE CasE'
caseInsensitiveRegex.test(testString) // true

Retrieve the first match in a variable

We use the .match () method

const match = 'Hello World!'.match(/hello/i) // 'Hello'


Extract all matches to an array

Use the g flag

const testString = 'Repeat repeat rePeAt'
const regexWithAllMatches = /Repeat/gi
testString.match(regexWithAllMatches) // ['Repeat', 'repeat', 'rePeAt']

We are looking for any character

We use the symbol.

const regexWithWildCard = /.at/gi
const testString = 'cat BAT cupcake fAt mat dog'
const allMatchingWords = testString.match(regexWithWildCard) // ['cat', 'BAT', 'fAt', 'mat']

We are looking for one variable symbol

We use classes that allow us to define a group of characters in []

const regexWithCharClass = /[cfm]at/g
const testString = 'cat fat bat mat'
const allMatchingWords = testString.match(regexWithCharClass) // ['cat', 'fat', 'mat']

Looking for the letters of the alphabet

Use the range [az]

const regexWithCharRange = /[a-e]at/
const catString = 'cat'
const batString = 'bat'
const fatString = 'fat'

regexWithCharRange.test(catString) // true
regexWithCharRange.test(batString) // true
regexWithCharRange.test(fatString) // false

We are looking for certain numbers or letters

We use the range [a-z0-9]

const regexWithLetterAndNumberRange = /[a-z0-9]/ig
const testString = 'Emma19382'
testString.math(regexWithLetterAndNumberRange) // true

Looking for an exception

To eliminate unnecessary characters, use the ^ symbol - negative set

const allCharsNotAllowed = /[^aeiou]/gi
const allCharsOrNumbersNotAllowed = /[^aeiou0-9]/gi

We are looking for characters that occur in the string one or more times

Use the + symbol

const oneOrMoreAsRegex = /a+/gi
const oneOrMoreSsRegex = /s+/gi
const cityInFlorida = 'Tallahassee'

cityInFlorida.match(oneOrMoreAsRegex) // ['a', 'a', 'a']
cityInFlorida.match(oneOrMoreSsRegex) // ['ss']

We are looking for characters that occur in the string zero or more times

Use the symbol *

const zeroOrMoreOsRegex = /hi*/gi
const normalHi = 'hi'
const happyHi = 'hiiiiii'
const twoHis = 'hiihii'
const bye = 'bye'

normalHi.match(zeroOrMoreOsRegex) // ['hi']
happyHi.match(zeroOrMoreOsRegex) // ['hiiiiii']
twoHis.match(zeroOrMoreOsRegex) // ['hii', 'hii']
bye.match(zeroOrMoreOsRegex) // null

Lazy Match Finder

We are looking for the smallest part of the string that satisfies the given condition.
Regex is greedy by default (looking for the longest part of the string that satisfies the condition). Use a character?

const testString = 'catastrophe'
const greedyRegex = /c[a-z]*t/gi
const lazyRegex = /c[a-z]*?t/gi

testString.match(greedyRegex) // ['catast']
testString.match(lazyRegex) // ['cat']

We are looking for using the start template (start line template)

To search for a string by the start pattern, use the ^ character (outside the character set in [], as opposed to the negative set)

const emmaAtFrontOfString = 'Emma likes cats a lot.'
const emmaNotAtFrontOfString = 'the cats Emma likes are fluffy'
const startingStringRegex = /^Emma/

startingStringRegex.test(emmaAtFrontOfString) // true
startingStringRegex.test(emmaNotAtFrontOfString) // false

We are looking for using the final template (end of line template)

To search for a string by the final pattern, use the $

const emmaAtBackOfString = 'The cats do not like Emma'
const emmaNotAtBackOfString = 'Emma loves the cats'
const endingStringRegex = /Emma$/

endingStringRegex.test(emmaAtBackOfString) // true
endingStringRegex.test(emmaNotAtBackOfString) // false

We are looking for all the letters or numbers

Use \ w

const longHand = /[A-za-z0-9_]+/
const shortHand = /\w+/
const numbers = '42'
const myFavouriteColor = 'magenta'

longHand.test(numbers) // true
shortHand.test(numbers) // true
longHand.test(myFavouriteColor) // true
shortHand.test(myFavouriteColor) // true

We search for any characters except letters and numbers

Use \ W

const noAlphaNumericCharRegex = /\W/gi
const weirdCharacters = '!_$!'
const alphaNumericCharacters = 'ab24EF'

noAlphaNumericCharRegex.test(weirdCharacters) // true
noAlphaNumericCharRegex.test(alphaNumericCharacters) // true

Looking for numbers

Use \ d instead of [0-9]

const digitsRegex = /\d/g
const stringWithDigits = 'My cat eats $20.00 worth of food a week'

stringWithDigits.match(digitsRegex) // ['2', '0', '0', '0']

We are not looking for numbers

Use \ D

const nonDigitsRegex = /\D/g
const stringWithLetters = '101 degrees'

stringWithLetters.match(nonDigitsRegex) // [' ', 'd', 'e', 'g', 'r', 'e', 'e', 's']

Looking for spaces (whitespace characters)

Use \ s

const sentenceWithWhitespace = 'I like cats!'
const spaceRegex = /\s/g
spaceRegex.match(sentenceWithWhitespace) // [' ', ' ']

We search for any characters except spaces

Use \ S

const sentenceWithWhitespace = 'C a t'
const nonWhitespaceRegex = /\S/g
sentenceWithWhitespace.match(nonWhitespaceRegex) // ['C', 'a', 't']

We are looking for a certain number of characters

We use {from, to} - quantifier

const regularHi = 'hi'
const mediocreHi = 'hiii'
const superExcitedHey = 'heeeeyyyyy!!!'
const excitedRegex = /hi{1,4}/

excitedRegex.test(regularHi) // true
excitedRegex.test(mediocreHi) // true
excitedRegex.test(superExcitedHey) // false

We are looking for the minimum number of characters

We use {from,}

const regularHi = 'hi'
const mediocreHi = 'hiii'
const superExcitedHey = 'heeeeyyyyy!!!'
const excitedRegex = /hi{2,}/

excitedRegex.test(regularHi) // false
excitedRegex.test(mediocreHi) // true
excitedRegex.test(superExcitedHey) // false

We are looking for the exact number of characters

Use {number of characters}

const regularHi = 'hi'
const mediocreHi = 'hiii'
const superExcitedHey = 'heeeeyyyyy!!!'
const excitedRegex = /hi{2}/

excitedRegex.test(regularHi) // false
excitedRegex.test(mediocreHi) // true
excitedRegex.test(superExcitedHey) // false

Looking for zero or one character

Use ?after the desired character

const britishSpelling = 'colour'
const americanSpelling = 'Color'
const langRegex = /colou?r/i

langRegex.test(britishSpelling) // true
langRegex.test(americanSpelling) // true
