金融アプリケーションの作成方法：開発者を支援する5つのAPI



みなさん、こんにちは！テクノロジーの観点から金融の領域の研究を続けています。では前の記事、私たちは、既存の取引端末とその機能を学び、今日の私は、金融アプリケーションの開発については、実際には、話をすることを提案しています。

新しいトピックでは、さまざまな金融サービスの5つのAPIを取り上げました。これらのAPIを使用すると、多種多様なデータを受信し、書かれたロボットを使用して取引所で直接取引することができます。

Yahooファイナンス


おそらく、世界で最も有名な金融サービスの1つです。開発者は無料のAPIを使用して、株価、価格の変化、市場の動きに関するサービスデータを使用できます。情報は、さまざまな銘柄、および通貨ペアなどの他の商品について入手できます。

Rapid APIサービスを使用して、APIに非常に便利に接続できます。



SMARTcom


ロシアの取引所で取引するためのアプリケーションを作成するためのAPI。開発者は、取引所取引システムと既存の取引システム（独立して作成または購入）の間のレイヤーとしてAPIを使用したり、独自の取引ロボットを作成したり、「カスタム」インターフェースを備えた取引端末全体を作成したりできます。

APIへのアクセスは無料です。APIを開発したブローカー会社のシステムでアカウントを作成するだけです。仮想取引のテストアクセスを使用して、ソフトウェアの作業をデバッグすることが可能です。



スクリーンショット：SMARTcom ドキュメント

支払い帳による同期


海外市場向けの金融アプリケーションの開発者に役立つサービス。銀行、金融および公益事業会社、さまざまな国の政府機関から1か所でデータを収集できます。このサービスは有料で、アクセス費用は50ドルからです。統合に利用できるサービスのリストはここにあります。



Currencylayer API


為替レートと168の世界通貨の価値に関する情報を取得するための便利なサービス。データは、銀行サイト、商業情報サービスなどの多くのソースから収集されます。開発者は、あらゆるタイプおよびあらゆるプログラミング言語のアプリケーションとの互換性を約束します。

APIの無料の限定バージョン（1か月あたり250リクエスト）があり、より適切なバージョンは$ 9.99（1か月あたり10,000リクエスト）から始まります。Currencylayer APIに基づくソリューションは、Lyft、Blizzard、Intelなどの企業によって開発されています。



簡単なファイナンス


RuNetで知られている、個人の財務を会計するサービスは、サードパーティの開発者にAPIを提供します。プロジェクトのWebサイトに示されているように、リソースの最も人気のあるユーザーは、ソーシャルネットワーク用のサードパーティアプリケーション、モバイルアプリケーション、およびブラウザー用のプラグインです。

アプリケーションはサービスチームによってテストされ、承認されたアプリケーションは特別な市場に投入され、サブスクリプションによって配布されます。 EasyFinanceの管理からの統計によると、アプリケーション開発者はユーザーあたり月額15-50ルーブルを獲得できます。この収入は、サービス自体の料金への加入による収入、プラットフォームパートナーの統合支払い、および広告で構成されます。
