DIY ECGバックパック


現代の大学のキャンパスには、熱心な学生という共通点が1つあります。

今日の学生は、社会的側面と科学的側面の両方からの圧力のために、一度にいくつかの領域の間でしばしば引き裂かれます。このため、多くの学生は対応できないと感じ、最終的には科学とスポーツの両方、およびコミュニケーションの両方で貧弱な結果を示しています。

今学期、私たちのクラスでは、筋肉の収縮、脳波、心臓の波など、患者の体から放出されるあらゆる種類の生体信号を測定できるさまざまな生体認証装置を研究しました。私たちは、脈拍に基づいて学生のストレスレベルを追跡するために使用できる医療ツールを開発したいと考えていました。私たちの「ECGバックパック」には、患者の心拍数を監視するECGセンサーが搭載されており、LEDとBluetoothを介して通信する電話アプリを通じて瞬時にフィードバックを提供します。ECGバックパックは、安静時のパルスに基づいてストレスレベルが通常よりもはるかに高い場合、LEDを使用してユーザーに警告します。モバイルアプリケーションのデータをExcelにエクスポートして、さらに分析することができます。

ステップ2：準備



1つの心周期

ECGバックパックのすべての可能性を理解するには、最初にECGが実際に何を記録するかについての基本的な理解が必要です。 ECGは心臓からの電気信号を記録します。各心周期では、心臓はいくつかの異なる収縮期（収縮）と拡張期（弛緩）を通過し、それぞれが特定の波に対応します。 ECGを調査すると、P波、QRSコンプレックス、およびT波という3つの主要なセクションがあります。それらは図に示されています。より高い。 2つのQRS群の間の時間により、心拍数を決定できます。これはすべてコードで考慮されます。

ECGメソッドの基本的な理解に加えて、Arduinoをプログラミングできる必要があります。Floraボードサポートを追加する必要があります。。最後に、GitHubからAdafruit Bluetooth用のライブラリをダウンロードする必要があります。

ステップ3：セキュリティ




電子回路を扱うときは、いくつかの注意事項を守る必要があります。

  • 回路を変更して、電源がオフになっていることを確認してください。
  • 回路の近くに液体や食べ物を置かないでください。
  • エクササイズ中にECGバックパックを携帯しないでください。ワイヤーがボードから外れたり、汗が電子機器に浸透したりしないようにします。
  • 絶縁されたワイヤーとツールで作業します。

注意！これは医療機器ではありません。心拍数を追跡するためのツールにすぎません。より正確な測定値とヒントについては、医師に相談してください。

ステップ7：バックパックの組み立てを開始する




準備段階、安全性、ヒントに慣れたら、ECGバックパックの収集を開始できます。

ステップ8：ボタンをAdafruit Floraにはんだ付けする




ボタンは、生地にボードを取り付けるのに役立ちます。Adafruitには、ボタンに関する役立つチュートリアル資料があります。誤った接続は回路全体を台無しにする可能性があるため、非常に注意深くはんだ付けすることが重要です。

マイクロプロセッサには次の接点が必要です。

  • 両方のピンは3.3 Vです。
  • RX＃0。
  • TX＃1。
  • 2つのアース（GND）（写真には3つが示されていますが、使用できるのは2つだけです-左下の接点と左上の接点）。
  • デジタルピン＃9
  • デジタルピン＃10

写真の一部の接点（3番目のGNDおよびSCL＃3は不要）。

手順9：Bluefruit Bluetoothモジュールのボタンをはんだ付けする




モジュールの以下の接点にボタンが必要です。

  • 電源3.3 V
  • TX
  • 処方箋
  • GND

ステップ10：ボタンを縫い、Bluefruit BluetoothをFloraに接続する




最初は、ボタンがマイクロプロセッサとBluetoothモジュールのボタンの位置と一致するようにボタンを配置することが非常に重要です。これはチョークを使用して行うことができます-各ボタンをそれでこすり、ボードを正しい場所に押します。

その後、バックパックの背面から始めて、4つの穴のそれぞれに糸を通し、導電性の糸でボタンを縫います。

BluetoothモジュールとFloraボードを相互に接続するには、導電性の糸で対応するボタンを接続する必要があります。

プロセスの最後に、バックパックの背面にある結び目で糸を結びます。

2つのボードを接続するには、次の接点間でスレッドを実行する必要があります。

  • フローラGND-Bluetooth GND
  • Flora TX＃1-Bluetooth RX
  • フローラRX＃0-Bluetooth TX
  • Flora 3.3V-Bluetooth 3.3V

上の写真では、Bluetoothモジュールが左側にあり、Floraボードが右側にあります。

ステップ11：回路に電力を追加する




フローラの下に小さな穴を開け、バッテリーからワイヤーを伸ばします。穴にワイヤーを通し、ボードに接続します。Floraボードがオンになると、緑のLEDが点灯し、Bluetoothでは赤のLEDが点滅します。

ステップ12：ポテンショメーターとLEDに電力を供給するボタンを縫う




次に、フローラボードの上にあるボタンを縫う必要があります。その中で：

  • GND、これをLEDとポテンショメータに接続します。
  • 3.3 V、これをポテンショメータに接続します。
  • ポテンショメータに接続するピン＃10。
  • ピン＃9をLEDに接続します。

ボタンを縫ってバックパックの後ろにワイヤーを引っ張った後、後で縫製を続けることができるように糸の供給を残すことをお勧めします。一方、これらのコンポーネントの一部はボタンに接続します。

ステップ13：複数のコンポーネントが接続される内部ボタンを縫う




複数のコンポーネントをアースと3.3 Vに接続する必要があるため、バックパック内にボタンを1つ追加して、いくつかのワニがそれに接続できるようにする必要があります。

前のステップでは、スレッドを残しました。それで別のボタンを縫う、ちょうどワイヤーが互いに交差しないことを確認してください。

ステップ14：ECGケーブルを組み立てる




ケーブルは、ECGチップ、3つの接点を持つ電極ケーブル、およびUC-E6ケーブルで構成されています。ケーブルを回路に接続するために、ワニをさまざまなECGコンポーネントに接続しました。

  • 赤いECGワイヤーは赤いワニにあります。
  • ECGの黒いワイヤーは緑のワニに向かっています。
  • 紫色のECGワイヤーは青いワニに向かっています。

ステップ15：ECGケーブルの組み立てを続行します




ECGケーブルにワニで短いワイヤーを追加することで、ECGケーブルを引き続き補完します。ワイヤーの接続方法：

  • 赤いECGケーブル-赤いワニ-オレンジ色のワイヤー。
  • 黒いケーブルECG-緑のワニ-黄色のワイヤー。
  • 紫のECGケーブル-青いワニ-緑のワイヤー。

信頼性のために、接続を電気テープで包みました。これらの短い配線は、回路をFloraボードに接続するために必要です。

手順16：ワイヤーをECGからバックパックの上部に配線するための穴を開けます




すべてのワイヤーが配置されているフロントポケットの反対側の角に、ポケットをバックパックのメインコンパートメントに接続する穴を開けます。これにより、ECG電極とBitalinoからのワイヤーをメインコンパートメントに配置できます。上の写真は、ワイヤーが穴をどのように通過するかを示しています。

手順17：ワイヤーがストラップを通るように、ECGのワイヤーをバックパックに通します




上の図のように、穴（赤い四角で囲まれている）にワイヤーを通した後、メインコンパートメントを通過させて、上から吊るす必要があります。

手順18：バックパックのストラップにワイヤーを通す




裏地があるストラップのそれぞれの上部に穴を開けます。ワイヤーをストラップの下から指示された場所に配線します。

黒い電極はユーザーの右側にあり、赤い電極の上にある必要があります。赤い電極は左側と下にあるはずです。それらの間の線は、心臓を通る対角線を形成する必要があります。白い電極は骨に配置する必要があります。肩の骨にフィットするようにストラップの上部に配置できます。

ステップ19：ECGをフローラに接続する




上で説明したように、ECGはワニに接続され、ワニにワイヤを接続しました。

  • 赤いECGケーブル-赤いワニ-オレンジ色のワイヤー。
  • 黒いケーブルECG-緑のワニ-黄色のワイヤー。
  • 紫のECGケーブル-青いワニ-緑のワイヤー。

その後、ワイヤーをピン6（アナログ7）、アース（GND）、3.3 V

に持ってくる必要があります。バックパックにいくつかの小さな穴を開け、ワイヤーがポケットの内側から外側に出るようにしました。

ワイヤーをピンに接続する方法は次のとおりです。

  • 緑色の線（紫ECG）-ピン番号6。
  • オレンジ（赤ECG）-3.3V。
  • 黄色（黒ECG）-アース（GND）。

アース線と3.3 Vワイヤーに接続し、対応するボタンに巻き付けました。緑色のワイヤーがピンNo. 6に巻き付けられています。

ステップ20：電位差計をフローラに接続する




ポテンショメータは、ピン10、グラウンド、3.3 Vに接続する必要があります。これらの場所は、写真の円で示されています。ピン10は、このボタンとフローラボードに接続されたスレッドです。参照しやすいようにポテンショメータ接続図をダウンロードしてください。

ステップ21：ユーザーにLEDが見えるように小さな穴を開けます




上の写真のように、LEDライトが貫通できる小さな穴を開けます。左は穴あけ後の写真です。右側-すでにLEDが挿入されています。それを光らせるために、私はそれを接続してコードをダウンロードしました。

ステップ22：LEDをフローラに接続する




LEDを接続するには：

  • LEDを穴に通して、バックパックと平行になるように脚を押します。
  • 短い脚（陰極）をワニに接続し、次に220オームの抵抗器に接続します。2番目のワニを使用して、抵抗をグラウンドに接続します。
  • LEDの陽極（長い脚）をピン9に接続します。

ステップ23：コードをFloraにアップロードします！



Floraを使用するには、コードをそれにロードする必要があります。それは可能なのGitHubからダウンロード。ビデオは、コードをダウンロードして使用するための詳細な手順を提供します。

ダウンロードする前に、必要なソフトウェアをインストールする必要がある場合があります。手順については、AdafruitのWebサイトを参照してください。

バックパックはほぼ準備ができています！

ステップ24：心拍数が正常値を超えたときにLEDがオンになるようにコードを修正します




プレートには、性別と年齢に応じた通常の心拍数の値が表示されます。値を見つけて、コードに貼り付けます。

パルスの上限はコードで設定され、90に等しくなります。この境界を離れる場合は、何も変更しないでください。それ以外の場合は、次のコードセクションの番号を変更します。

if (HR > 90) { 
  digitalWrite(LEDpin, HIGH);
} // If the resting HR is above 90, or another preset HR, turn the LED on. HR number can be changed based off of chart in Instructable

ステップ25：粘着性電極を身体に配置します




図に示されているポイントで人体の電極を配置します。緑のドットが付いた電極は、肩の骨にあります。赤い点は心電図の黒いワイヤーを示し、黄色の点は心電図の赤いワイヤーを示します。

ステップ26：バックパックから身体の電極にケーブルを接続する




電極は、ストラップから見える場所の隣に接続されています。ストラップがユーザーに合うように調整し、身体にある電極を接続できるようにします。

ステップ27：ポテンショメーターを調整する




Floraをコンピューターに接続し、左上隅の矢印をクリックしてコードをダウンロードします。ロード後、「シリアルプロッター」（シリアルプロッター）を選択してください。ポテンショメーターを回すと、グラフに線が表示されます。上図に示すように、QRSコンプレックスのピークがポテンショメーターラインを通過することを確認してください。そうしないと、心拍数の計算が正確に機能しません。

ステップ28：Bluefruit LE Con​​nectをダウンロードする




Bluefruit LE接続アプリをダウンロードします。それを使用すると、携帯電話にデータをダウンロードできます。

手順29：電話をBluetoothに接続する




アプリを起動し、上記の設定でAdafruit Bluefruit LEに接続します。Uartを押すと、データが表示されます。心拍数はラインの最初の数値になります。

ステップ30：バックパックを着用する




最初にフローラをオンにします。次に、バックパックを身につけて持ち運び、ジャンプしたり走ったりしないようにしてください。そのような動きはアーティファクトにつながる可能性があります。

ステップ31：データをエクスポートする




データを収集したら、アプリケーションの右上隅にある歯車をクリックし、[エクスポート]をクリックします。テキストファイルは、自分自身または分析したい人に電子メールで送信できます。

ステップ32：Excelでデータを調べる




データをExcelにコピーします。すべてのデータを選択し、折れ線グラフをプロットします。黄色の線はポテンショメータのしきい値を示し、灰色の線は心拍を示します。脈拍は、1分あたりの収縮数、つまり1分間に灰色の線が黄色の線と交差した回数として定義されます。

ステップ33：改善できること




このスキームは改善することができます：

  • ポテンショメーターを調整のためのより便利な場所に配置する。今度はこれを行うたびに、電子機器でポケットを開ける必要があります。
  • 呼吸数を追跡するために圧力計を追加します。これにより、増加した呼吸数を追跡し、患者のストレスとリラックスについて結論を出すことができます。バックパックには通常、胸部のストラップを接続する留め具があり、そのようなセンサーをそこに配置すると便利です。

ステップ34：最終製品






最初の写真は、アプリケーションがBluetooth経由で受信したデータを示しています。この時の脈拍は毎分96拍でした。私はデータをApple Watchと比較しました、そしてそれらはかなり良いです-同じ期間に、時計は95から98ビート/分を示しました。パルスが90 bpmを超えているため、写真のLEDが点灯しています。私の場合90はしきい値で、その超過は高レベルの電圧と励起を示します。
