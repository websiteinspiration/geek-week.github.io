.NET Core Windows Forms Designerの更新

Visual Studio 16.6の暫定バージョン-Visual Studio 2019バージョン16.6 Preview 1をリリースしました。これには、Windowsフォームの.NET Coreコンストラクターの新しいバージョンが含まれています。



このリリースには


  • 次のコントロールのサポート：

    明らかにする
    • FlowLayoutPanel、
    • GroupBox
    • ImageList
    • MenuStrip ( PropertyBrowser ),
    • Panel,
    • SplitContainer,
    • Splitter,
    • TabControl,
    • TableLayoutPanel,
    • ToolStrip ( PropertyBrowser, designer actions).

  • ローカルリソースとローカライズされたフォームがコンストラクターに含まれていました。
  • ツール->オプションを介したLayoutModeおよびShowGrid / SnapToGrid設定のサポート。
  • 信頼性とパフォーマンスの向上。
  • その他のマイナーな修正と微調整。

さらに


今後のリリースでは、ユーザーコントロールとサードパーティコントロールのサポート、一般的なコントロールプロバイダーとの統合、データコントロールと関連シナリオのサポート、パフォーマンスの向上、その他の機能に取り組んでいきます。

コンストラクターの使用方法


  • Visual Studio Previewフィードを使用する必要がある
  • Visual Studioでデザイナーを有効にする必要があります。[ツール]→[オプション]→[環境]→[プレビュー関数]に移動し、[Windowsフォームデザイナを.NET Coreアプリケーションに使用する]オプションを選択します。



問題を報告する方法


あなたのフィードバックは非常に重要です。Visual Studioフィードバックチャネルから問題を報告し、リクエストを送信してください。次に示すように、Visual Studioの右上隅にある[フィードバックの送信]アイコンを使用して、WinForms .NET Core領域に関連付けられていることを示します。
