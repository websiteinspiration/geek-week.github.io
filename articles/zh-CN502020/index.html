RoadHunters.io-将摩托车手带到世界各地

大家好，我叫Sergey，我骑摩托车已经7年多了。当然，现在并不是世界上最好的情况，尤其是对我们国家而言，不是我们服务的最佳状态，但是，我们在2018年就开始开发它，当时我们甚至都不怀疑未来。



RoadHunters是一个地理定位服务，用于查找路线和创建摩托车事件，我们希望将世界各地的摩托车手召集在一起，让他们彼此分享经验和知识。
（到目前为止，该服务仅可在PC上运行）

首先，有关其工作方式的一些信息将在门户网站的``烟灰之下''，而稍低一些的信息。
创建服务的想法多久会发生一次，解决了创建者面临的个人问题。我们在一起666granik也不例外。

我不知道摩托车阅读者中是否有很多人在阅读，所以我将简要介绍一下这些痛苦：

  1. 找人一起骑。此外，需要类似水平的滑雪，相似的技术和邻近的区域。
  2. . . , .
  3. . , : , , . — .

因此，我们开始开发RoadHunters.io服务，
该门户免费使用，不仅适合摩托车骑士，尽管它首先侧重于摩托车。

与许多儿童时代一样，我们在GTA和NFS Underground 1-2系列游戏中花费了数小时，我们希望将这个世界的一部分转移到“我们的世界”中。我们向交互式地图添加了即将发生的事件，风景如画的有趣路线，并能够以可读格式将其上传给导航员。



主要思想是创建一个全球航线目录，来自当地骑手。没有人比住在这些地方的人更了解最有趣的道路。因此，在世界各地旅行时，您可以轻松地选择沿途最有趣的路线，并从其他摩托车手那里阅读有关它们的故事。



地图上显示的即将发生的各种类型的事件将使您可以在旅行中找到公司或加入现有的摩托车事件。

在您自己的新地方，您可能不认识任何人，但是有了RoadHunters服务，这将不再是问题。您会永远发现许多人对摩托车同样充满热情。



关于技术部分的一些知识：


666granik他对Mail.ru从事开发工作，除了对创建服务即产品感兴趣之外，在现代技术上做到这一点并扩展其知识的愿望也很重要。

简要介绍一下：

  • Google Cloud Platform，用于托管virtuala，数据和CI + Docker
  • Mongo Atlas数据库即服务
  • Nginx作为前端服务器
  • 具有NodeJS / React / Mobx / TypeScript的同构Web应用程序
  • NodeJS / Express / TypeScript上的API应用
  • Github private用于存储源代码
  • Zepplin用于处理设计师的布局

选择之所以落在NodeJS的方向上是因为我想更深入地了解这项技术，作为UI库的React是业界最常见的解决方案之一，TypeScript奇怪地在代码中添加了“可靠性”，但即使是在初创公司开发中，速度比可靠性更重要，TS并没有让我们放慢脚步。

我们几乎不了解数据库，但是我想尝试使用一些现代的NoSQL解决方案，选择是在PostgreSQL和MongoDB之间，我们选择Mongu是因为它具有NodeJS的便捷客户端和免费的Mongo Atlas云解决方案来启动，顺便说一句，他遇到了一个有趣的情况。

我们开始缺乏免费的流量解决方案，即使是最便宜的无限制关税计划对我们来说也太昂贵了。流量对我们来说非常重要，因为在应用程序中，主要要点是路线，而路线本质上是非常长的二维数组。

结果，为了降低可靠性和便利性，但增加了预算，决定退回到免费选项-“一种沙箱”并放弃云解决方案。

然后，Mongo Atlas团队的成员写信给我们，以了解我们出于什么原因拒绝提供服务。我们概述了当前的情况：我们是一家小型初创企业，我们仍然没有钱支付他们的解决方案。答案是不同寻常的-他们将我们带到一个项目中，为新兴企业提供支持，并允许我们在大约99年的时间内完全免费使用其Mongo Atlas服务。

他们不是唯一支持我们项目的人。 Github在2019年中期为少于3人的团队免费提供了私人帐户。每月节省7美元-有点小事，但很好。

我们还为Google的启动支持计划填写了一个应用程序。经过数个故事和完整的表格后，Google为我们提供了一个促销代码，用于支付相当大的押金以使用其服务。因此，我们能够使用Google Cloud Build和Google Compute Engine配置完整的CI / CD周期，免费将Docker映像存储在它们的存储库中，监视服务的负载，等等。我们很激动。

是的，您可能已经注意到，我们在前几段中多次重复使用Google和Mongo的话，这是我们对他们说的另一种幼稚方式：“谢谢！”。

如果您需要在技术部分进行深入研究，请在评论中写-我们将发表您感兴趣的主题的文章。

谁可以从我们的服务中受益？通过创建RoadHunters，我们希望使摩托车社区变得更加集中：

  • 连接来自世界各地的旅行者。

  • 让新生或初学者找到朋友和公司旅行。
  • 经验丰富的骑手为自己找到新的路线。
