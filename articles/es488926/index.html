Una visión general de la inteligencia artificial (IA) y el aprendizaje automático. Los módulos están destinados a usuarios con poco conocimiento en el campo de la informática y la estadística. Aprenderá sobre los conceptos básicos de IA y aprenderá a usar soluciones personalizadas de aprendizaje automático con herramientas gratuitas y fáciles. 2. Crear bots inteligentes en Azure Bot


La interacción del usuario con las aplicaciones a través del chat mediante texto, imágenes o discurso se puede implementar mediante bots. Esto puede ser un simple diálogo de "pregunta-respuesta" o un bot complejo que permite a las personas interactuar con los servicios de una manera "inteligente", utilizando la coincidencia de patrones, el seguimiento del estado y métodos de inteligencia artificial que están bien integrados con los servicios existentes. Aprenda a crear un chatbot inteligente integrando QnA Maker y LUIS.

El curso durará aproximadamente 3 horas.

3. Análisis de texto con Azure Cognitive Language Services


Aprenda a utilizar los servicios de lenguaje cognitivo para analizar texto, determinar intenciones, descubrir temas para adultos y procesar consultas en lenguaje natural.

Módulos del curso:

  • Clasificación y moderación de texto con Azure Content Moderator;
  • Agregar lógica de diálogo a las aplicaciones utilizando el Servicio inteligente de reconocimiento de voz (LUIS);
  • Seguimiento de la tonalidad del diálogo en el texto utilizando la API de análisis de texto.

4. Curso de ingeniero de dispositivos fronterizos basado en IA


Las tecnologías de inteligencia artificial, las tecnologías de nube y los dispositivos fronterizos implementados como contenedores en modo CICD le permiten transformar industrias enteras creando ecosistemas con capacidades de autoaprendizaje para una industria específica que abarca toda la cadena de valor. Nuestra tarea es crear un conjunto de plantillas y metodologías para implementar AI en dispositivos de borde en el contexto de la nube. En este esquema de capacitación, recibirá la siguiente información:

  • Información sobre la creación de soluciones utilizando Internet de las cosas y la nube;
  • Descripción general de la implementación de soluciones de IoT en dispositivos de borde
  • Información sobre la implementación de modelos para dispositivos fronterizos utilizando contenedores;
  • Aprenda a usar DevOps para dispositivos de borde.

El curso fue desarrollado en conjunto con la Universidad de Oxford.

Empezar a entrenar

5. Procesamiento y clasificación de imágenes con Azure Cognitive Vision Services


Microsoft Cognitive Services proporciona características integradas para usar la visión por computadora en las aplicaciones. Aprenda a utilizar los Servicios de visión cognitiva para detectar rostros, etiquetar y clasificar imágenes e identificar objetos.

Módulos del curso:

  • Identifique rostros y emociones con la API de visión por computadora en Azure Cognitive Services;
  • Procesamiento de imágenes utilizando el servicio de visión por computadora;
  • Clasificación de imágenes utilizando el servicio de reconocimiento visual del usuario;
  • Evaluación de requisitos para la implementación de la API de reconocimiento visual de usuarios.
