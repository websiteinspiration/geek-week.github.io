It consists in comparing two poker hands and determining the winning one.

Poker hands are represented by lines containing substrings (denoting individual cards) of 2 characters, separated by spaces. 2H is a two of worms, TC is a cross ten, etc.

The first character means ranking.
All possible options, ordered by seniority: 2 3 4 5 6 7 8 9 TJQKA .
With the numeric designations, everything is clear as for the letters: T is 10 ( T en), J is the jack ( J ack), Q is the queen ( Q ueen), K is the king ( K ing),A - Ace ( A ce).

The second character defines the suit.
Possible options: H - worms ( H earts), C - crosses ( C lubs), S - peaks ( S pades), D - tambourines ( D iamonds).

For example, such a complete line for the hand “2C 5C 3C 4C 6C” - in this case, this is a cross straight flush from two to six.

Hand ranking is taken as in Texas Hold'em .

You need to understand that there are hidden difficulties - the use of additional cards to resolve draw situations, the use of par value of pairs, etc.

Decision


So, how to solve this problem? First you need to determine a comparison method that solves the issue of ranking hands, and then, if the ranking matches, the winner is determined, if possible, by comparing additional cards.

Since the statement of the problem assumes that the suits are equivalent, we use a simple object to represent the ranking of hands:

{ 
   rank: 1,       //     1  9,   ,    
   value: 'ABCDE' //  ,        ,    ASCII-    
}

Now we can write a simple function for comparing two hands represented in the form of this structure:

function compareHands(h1, h2) {
    let d1 = getHandDetails(h1)
    let d2 = getHandDetails(h2)
    if (d1.rank === d2.rank) {
        if (d1.value < d2.value) {
            return ""
        } else if (d1.value > d2.value) {
            return ""
        } else {
            return ""
        }
    }
    return d1.rank < d2.rank ? "" : ""
}

In this case, the && operator is used fantastically , which scans to the last value if the values ​​in the previous brackets are correct. Thus, (flush && direct && 1) returns 1 if flush and direct are true, otherwise returns false.

Drawn situation


If two hands have the same rank, the winner is determined additionally by the face value of the cards, if possible. Some rules are associated with this.

  • Couple vs couple. A pair with a higher face value wins. If the values ​​match, the highest next card wins. (For two pairs it is solved similarly).

    For example, if we compare the “2H 2D AH KC 3D” hand with the “4H 4C JC TC 3H” hand, then fours from the second hand win, although in the first hand the highest card has the highest face value - an ace.
  • Full house versus full house . The trio with the higher face value wins. If the denominations of triples in both hands are equal, then we look at the denominations of twos.
  • We act similarly when comparing any hands with the same rank - decides the face value of the highest card in combination.


Therefore, we need to first sort by quantity, and then by face value in our outgoing line. Remember, the entire five-character string is used to solve cases with matching hand rankings.

const order = "23456789TJQKA"
function getHandDetails(hand) {
    const cards = hand.split(" ")
    const faces = cards.map(a => String.fromCharCode([77 - order.indexOf(a[0])])).sort()
    const suits = cards.map(a => a[1]).sort()
    const counts = faces.reduce(count, {})
    const duplicates = Object.values(counts).reduce(count, {})
    const flush = suits[0] === suits[4]
    const first = faces[0].charCodeAt(0)
    const straight = faces.every((f, index) => f.charCodeAt(0) - first === index)
    let rank =
        (flush && straight && 1) || /* -,     - -        */
        (duplicates[4] && 2) || /*  -     */
        (duplicates[3] && duplicates[2] && 3) || /* - -  +  */
        (flush && 4) || /*  -      */
        (straight && 5) || /*  -       */
        (duplicates[3] && 6) || /*  -     */
        (duplicates[2] > 1 && 7) || /*   */
        (duplicates[2] && 8) || /* -     */
        9 /*     ,      */

    return { rank, value: faces.sort(byCountFirst).join("") }

    function byCountFirst(a, b) {
        //    -  ,  
        const countDiff = counts[b] - counts[a]

        if(countDiff) return countDiff //   ,  
        return b > a ? -1 : b === a ? 0 : 1
    }

    function count(c, a) {
        c[a] = (c[a] || 0) + 1
        return c
    }
}

function compareHands(h1, h2) {
    let d1 = getHandDetails(h1)
    let d2 = getHandDetails(h2)
    if (d1.rank === d2.rank) {
        if (d1.value < d2.value) {
            return ""
        } else if (d1.value > d2.value) {
            return ""
        } else {
            return ""
        }
    }
    return d1.rank < d2.rank ? "" : ""
}
