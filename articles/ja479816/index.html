Pythonの組み合わせ論

バージョン2.2以降のpython標準ライブラリは、組み合わせオブジェクトを生成するための多くのツールを提供しますが、それらの操作について詳しく説明する記事をインターネット上で見つけることができませんでした。したがって、私はこの省略を修正することにしました。


まず、組み合わせ論とその基本的な公式についてお話します。数学のこのセクションにすでに精通している場合は、これらの段落をスキップできます。


n個の異なる文字で構成される文字列があり、新しい行を取得するような方法でこれらの文字を再配置するすべての方法を計算するとします。行の最初の位置まで、n文字の1つを選択し、2番目の位置まで、n- 1 番目の文字の1つを選択することができます。その結果、積n （n-1） ... * 1 = n！が得られます。繰り返しのないn個の要素の順列の数。


文字列の文字数が制限されていると想像してください。n個の文字が利用可能であり、それらから長さkの文字列を作成する方法の数を計算します。ここで、k <nの場合、各文字は1回だけ使用できます。次に、行の最初の位置にn文字の1つを配置し、2番目の位置にn-1文字の1つを配置し、k番目の位置にn-k + 1文字の1つを配置します。行の総数は、n （n-1）（n-2）...（n-k + 2）（n-k + 1）= n！/（Nk）！nからkまでの配置数。文字の一意性が必要でない場合、式n ... n n = n ^ kを繰り返し数nからkまでの配置数で取得します。。
