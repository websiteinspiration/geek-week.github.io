Bagaimana TIDAK untuk Wawancara

Kata pengantar


Selamat sore. Sebagai pengembang android sederhana, saya melalui lebih dari 20 wawancara di posisi tengah dalam 2 minggu. 18 di antaranya ada di Skype. Lebih dari 50% perusahaan menawarkan saya. Sisanya mengabaikan jawaban atau langsung menolak. Beberapa melukis secara terperinci mengapa saya tidak cocok, dan tidak bodoh mengirim templat dengan penolakan. Hormati perusahaan semacam itu.
Akibatnya, saya berhenti di sberbank.ru


Untuk semua wawancara ini, saya memperoleh pengalaman dan ingin berbagi dengan Anda bagaimana TIDAK untuk melakukan wawancara .


Jangan malu dan jangan meremehkan keterampilan profesional Anda


Apakah Anda tahu bahwa pewawancara harus mengajukan pertanyaan yang sama kepada puluhan kandidat seperti Anda sehari? Bagaimana Anda berbeda dari mereka? Meskipun Anda akan pemalu dan menekan belas kasihan, maka tidak ada apa-apa. Anda akan dilupakan setelah 5 menit setelah wawancara, jika Anda tidak berbeda. Agar ini tidak terjadi, maka tinggalkan setidaknya beberapa emosi. Anda akan diingat. Untuk melakukan ini, Anda perlu menambahkan sedikit cat dan hiperbola untuk pengalaman Anda. Dan yang paling penting - JANGAN Bohong

Ketika pewawancara bertanya tentang pengalaman Anda, jangan katakan ini:


TIDAK benar - "Mmm ... yah, Anda tahu, saya hanya melakukan CRUD apk. Saya pikir tidak ada yang menarik dalam karir saya. Pada proyek terakhir, saya melihat dukungan sederhana untuk jejaring sosial. Saya menulis algoritma sederhana untuk menyaring pengguna. Dan saya juga membuat perpustakaan sederhana yang memecahkan masalah X, tapi begitu, hal-hal kecil ... "


Itu benar - "Saya membuat aplikasi yang paling banyak dimuat. Kadang-kadang saya harus membuat algoritma yang paling rumit untuk menyelesaikannya. Saya terus membaca buku tentang algoritma dan sekarang menggunakannya dalam praktik. Saya menulis perpustakaan saya yang memecahkan masalah yang tidak dapat diselesaikan dan memasukkannya ke github. Dia mencetak 10 bintang dalam satu hari! Aktivitas seperti itu di github menginspirasi saya untuk menjadi co-kontributor open source aktif "


Wawancara itu menyenangkan!
