機械学習データの抽出

次のMLプロジェクトの3つのデータマイニング方法について学びたいですか？次に、MediumのTowards Data Scienceブログで公開されているRebecca Vickeryの記事の翻訳を読んでください。彼女は初心者には面白いでしょう。


高品質のデータを取得することは、機械学習プロジェクトの最初の最も重要なステップです。データサイエンスのスペシャリストは、データセットを取得するためにさまざまな方法を使用することがよくあります。公開されているデータや、APIを介して利用できるデータ、またはさまざまなデータベースから取得したデータを使用できますが、ほとんどの場合、これらの方法を組み合わせます。

この記事の目的は、Pythonを使用してデータを取得するための3つの異なる方法の概要を提供することです。Jupyter Notebookでこれを行う方法を説明します。前回の記事では、ターミナルで実行されるいくつかのコマンドのアプリケーションについて書きました。

SQL


リレーショナルデータベースからデータを取得する必要がある場合は、おそらくSQL言語を使用します。SQLAlchemyライブラリを使用すると、ラップトップコードを最も一般的なタイプのデータベースに関連付けることができます。上のリンクは、各タイプにバインドするデータベースがサポートされているかについての情報とどのように見つけるでしょう。

SQLAlchemyライブラリを使用して、テーブルやクエリデータを参照したり、生のクエリを書き込んだりできます。データベースにバインドするには、資格情報を含むURLが必要です。次に、メソッドcreate_engineを初期化して接続を作成する必要があります。

from sqlalchemy import create_engine
engine = create_engine('dialect+driver://username:password@host:port/database')

これで、データベースクエリを記述して結果を取得できます。

connection = engine.connect()
result = connection.execute("select * from my_table")

こする


Webスクレイピングは、Webサイトからデータをダウンロードし、そのページから必要な情報を抽出するために使用されます。これに利用できるPythonライブラリはたくさんありますが、最も簡単なのはBeautiful Soupです。

パッケージはpipからインストールできます。

pip install BeautifulSoup4

使い方の簡単な例を見てみましょう。Beautiful Soupとurllibライブラリを使用して、トリップアドバイザーからホテルの名前と価格を取得します。

まず、使用するすべてのライブラリをインポートします。

from bs4 import BeautifulSoup
import urllib.request

スクラップするページのコンテンツをロードします。ギリシャのクレタ島のホテルの料金に関するデータを収集し、この場所にあるホテルのリストを含むURLアドレスを取得したいと考えています。



以下のコードは、URLを変数として定義し、urllibライブラリを使用してページを開き、Beautiful Soupライブラリを使用してそれを読み取り、結果を単純な形式で返します。出力データの一部をコードの下に示します。

URL = 'https://www.tripadvisor.co.uk/Hotels-g189413-Crete-Hotels.html'
page = urllib.request.urlopen(URL)
soup = BeautifulSoup(page, 'html.parser')
print(soup.prettify())



次に、ページ上のホテルの名前のリストを取得します。find_all興味のあるドキュメントの一部を抽出できる機能を紹介します。find_all単一行、正規表現、またはリストを渡す関数を使用して、異なる方法でフィルタリングできます。また、タグ属性の1つを除外することもできます。これは、適用する方法とまったく同じです。 HTMLタグと属性を初めて使用する場合は、この記事で概要を確認してください。

タグ内のデータへのアクセスを提供する最善の方法を理解するには、ページ上のこの要素のコードを確認する必要があります。下の図に示すように、リスト内の名前を右クリックすると、ホテルの名前のコードが見つかります。



クリックした後inspectアイテムコードが表示され、ホテル名のセクションが強調表示されます。



ホテルの名前は、クラス内での名前を持つ唯一のテキストであることがわかりますlisting_title。クラスの後には、この属性のコードと名前、およびfind_allタグが続きdivます。

content_name = soup.find_all('div', attrs={'class': 'listing_title'})
print(content_name)

ホテル名を含むコードの各セクションがリストとして返されます。



コードからホテル名を抽出するには、getTextBeautiful Soupライブラリの関数を使用します。

content_name_list = []
for div in content_name:
   content_name_list.append(div.getText().split('\n')[0])
print(content_name_list)

ホテル名はリストとして返されます。



同様に、価格データを取得します。価格のコード構造を以下に示します。



ご覧のとおり、ホテルで使用するコードとよく似たコードで作業できます。

content_price = soup.find_all('div', attrs={'class': 'price-wrap'})
print(content_price)

価格の場合は難ありです。次のコードを実行すると確認できます。

content_price_list = []
for div in content_price:
   content_price_list.append(div.getText().split('\n')[0])
print(content_price_list)

結果を以下に示します。ホテルのリストに値下げが示されている場合、テキストに加えて、初期価格と最終価格の両方が返されます。この問題を修正するには、今日の現在の価格を返すだけです。



シンプルなロジックを使用して、テキストに示されている最新の価格を取得できます。

content_price_list = []
for a in content_price:
       a_split = a.getText().split('\n')[0]
       if len(a_split) > 5:
           content_price_list.append(a_split[-4:])
       else:
           content_price_list.append(a_split) 
      
print(content_price_list)

これにより、次の結果が得られます。



API


API-アプリケーションプログラミングインターフェイス（英語のアプリケーションプログラミングインターフェイスから）。データマイニングの観点から見ると、プログラミングを通じて連絡できるデータエンドポイントを提供するWebベースのシステムです。通常、データはJSONまたはXML形式で返されます。

この方法は、おそらく機械学習で役立ちます。公開されているDark Sky APIから気象データを取得する簡単な例を紹介します。接続するには、登録する必要があります。1日あたり1000回の無料通話が可能です。テストにはこれで十分です。

ライブラリを使用してDark Skyのデータにアクセスしますrequests。まず、リクエストの正しいURLを取得する必要があります。予測に加えて、Dark Skyは過去の気象データを提供します。この例では、それらを取得して、ドキュメントから正しいURLを取得します。

このURLの構造は次のとおりです。

https://api.darksky.net/forecast/[key]/[latitude],[longitude],[time]

て、特定の緯度と経度、および日付と時刻の結果を取得します。クレタ島のホテルの価格に関する毎日のデータを抽出した後、価格設定ポリシーが天候に関連しているかどうかを調べることにしたと想像してください。

たとえば、リストにあるホテルの1つであるMitsis Laguna Resort＆Spaの座標を考えてみましょう。



最初に、正しい座標と要求された日時を含むURLを作成します。ライブラリを使用requestsして、JSON形式のデータにアクセスします。

import requests
request_url = 'https://api.darksky.net/forecast/fd82a22de40c6dca7d1ae392ad83eeb3/35.3378,-25.3741,2019-07-01T12:00:00'
result = requests.get(request_url).json()
result
