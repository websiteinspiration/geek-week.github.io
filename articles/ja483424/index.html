DBA：PostgreSQLデータベース間でのSEQUENCE値の転送

テーブルに削除がある可能性があり、「max（pk）を置き換える」だけでは機能しない場合、シリアルタイプの「auto-increment」フィールドに最後に割り当てられた値を他のPostgreSQLデータベースに転送するにはどうすればよいですか？別のセッションからそのようなフィールドのシーケンスの最後の値を見つけるために

PGがバージョン10までの機能を提供していなくても、これはまだ実行できることを知っている人はほとんどいません。



PostgreSQLは、シリアル疑似タイプで定義された自動インクリメントフィールドをサポートしています。
smallserial、serial、bigserialのデータ型は実際の型ではなく、一意の識別子を持つ列を作成するための便利なツールにすぎません（一部のDBMSのAUTO_INCREMENTプロパティと同様）。
技術的には、これはフィールドのDEFAULT定義とSEQUENCEオブジェクトによって生成された値の自動リンクにすぎません。また、これらの値を調べたい場合もあります。たとえば、構造が類似したデータベースにそれらを転送します。

しかし、これは簡単ではありません。
currval 現在のセッション
でこのシーケンスの最後のnextval呼び出しから返された値を返します。（特定のセッションで特定のシーケンスに対してnextvalが呼び出されたことがない場合、エラーが返されます。）この値はセッションのスコープによって制限されるため、この関数は、nextvalがその後他のセッションで呼び出されたかどうかに関係なく、予測可能な結果を​​生成します。

まず、フィールドに対応するシーケンスの名前を見つけます。

SELECT pg_get_serial_sequence('tst', 'id');

pg_get_serial_sequence
----------------------
public.tst_id_seq

次に、結果のシーケンス名をクエリのテーブルとして使用します。

SELECT * FROM public.tst_id_seq;

last_value | log_cnt | is_called
--------------------------------
         4 |      29 | t

実際、last_valueフィールドには、呼び出し時にシーケンスが生成した「最後の」値も格納されます。

次に、dblinkモジュールを使用して、あるデータベースから別のデータベースにシーケンス値を転送するための簡単なスクリプトを作成します。

--   ,   
SELECT
  (
    SELECT
      nlv
    FROM
      dblink(
        'host=... port=5432 dbname=... user=... password=...'
      , $q$
          SELECT setval(pg_get_serial_sequence('$q$ || quote_ident(sequence_schema) || $q$.$q$ || quote_ident(sequence_table) || $q$', '$q$ || sequence_column || $q$'), $q$ || lv || $q$)
        $q$
      ) T(nlv bigint) --  
  ) nlv
, *
FROM
  (
    SELECT
      (
        SELECT
          relname
        FROM
          pg_class
        WHERE
          oid = (dp).refobjid
      ) sequence_table
    , (
        SELECT
          attname
        FROM
          pg_attribute
        WHERE
          (attrelid, attnum) = ((dp).refobjid, (dp).refobjsubid)
      ) sequence_column
    , *
    FROM
      (
        SELECT
          --    -
          (
            SELECT
              lv
            FROM
              dblink(
                'dbname=' || current_database()
              , $q$
                  SELECT last_value FROM $q$ || quote_ident(sequence_schema) || $q$.$q$ || quote_ident(sequence_name) || $q$
                $q$
              ) T(lv bigint) --   ,   ""-
          ) lv
        , (
            SELECT
              dp
            FROM
              pg_depend dp
            WHERE
              (classid, objid, refclassid, deptype) = ('pg_class'::regclass, (quote_ident(sequence_schema) || '.' || quote_ident(sequence_name))::regclass, 'pg_class'::regclass, 'a')
            LIMIT 1
          ) dp --     
        , *
        FROM
          information_schema.sequences
        WHERE
          sequence_schema IN ('public') --   
      ) T
  ) T
WHERE
  sequence_table !~ '^_'; --
