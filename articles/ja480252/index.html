SDL 2チュートリアル：レッスン7-ビューポート

みなさん、こんにちは！私はあなたのための新しいレッスンで帰ってきました！そして、突然私からのレッスンを待つことに飽きたら、ここで英語のレッスンをいつでも見つけることができます。

さて、タイトルから7回目のレッスンを始めましょう

ビューポート


ビューポートは、表示場所として英語から翻訳されたものです。私たちの場合はレンダリング場所です。本質はこれです。描画面をウィンドウ全体ではなく一部に限定します。これは、たとえばゲームでミニマップを描画する場合に役立ちます。

理論によると、私はすべてを持っている、私たちは練習に目を向けます。

#include <SDL2/SDL.h>
#include <SDL2/SDL_image.h>
#include <iostream>

using namespace std;

const int SCREEN_SIZE[2] = {640, 480};

SDL_Window *window = NULL;
SDL_Renderer *ren = NULL;
SDL_Texture *flower = NULL;

まず、いつものように変数を宣言します。

この花を描きます。


初期化
bool init() {
    if (SDL_Init(SDL_INIT_VIDEO | SDL_INIT_EVENTS) != 0) {
        cout << "Can't init SDL: " << SDL_GetError() << endl;
        return false;
    }
    int flags = IMG_INIT_PNG;
    if (!(IMG_Init(flags)&flags)) {
        cout << "Can't init IMG: " << IMG_GetError() << endl;
        return false;
    }
    window = SDL_CreateWindow("VIEWPORT", SDL_WINDOWPOS_UNDEFINED, SDL_WINDOWPOS_UNDEFINED, SCREEN_SIZE[0], SCREEN_SIZE[1], SDL_WINDOW_SHOWN);
    if (window == NULL) {
        cout << "Can't create window: " << SDL_GetError() <<endl;
        return false;
    }
    ren = SDL_CreateRenderer(window, -1, SDL_RENDERER_ACCELERATED | SDL_RENDERER_PRESENTVSYNC);
    if (ren == NULL) {
        cout << "Can't create renderer: " << SDL_GetError() <<endl;
        return false;
    }
    return true;
}


負荷
bool load() {
    SDL_Surface *temp_surf = NULL;
    temp_surf = IMG_Load("flower.png");
    if (temp_surf == NULL) {
        cout << "Can't load image: " << IMG_GetError() << endl;
        return false;
    }
    flower = SDL_CreateTextureFromSurface(ren, temp_surf);
    if (flower == NULL) {
        cout << "Can't create texture from surface: " << SDL_GetError() << endl;
        SDL_FreeSurface(temp_surf);
        temp_surf = NULL;
        return false;
    }
    return true;
}


終了する
void quit() {
    SDL_DestroyWindow(window);
    window = NULL;

    SDL_DestroyRenderer(ren);
    ren = NULL;

    SDL_DestroyTexture(flower);
    flower = NULL;

    SDL_Quit();
    IMG_Quit();
}


初期化、イメージのロード、および初期化解除の機能については、以前のものとほとんど同じであるため、ここでは説明しません。

そして今、楽しい部分のために：

int main() {
    if (!init()) {
        quit();
        return 1;
    }
    if (!load()) {
        quit();
        return 1;
    }

    SDL_Rect top_vp = {0, 0, SCREEN_SIZE[0], SCREEN_SIZE[1] / 2};
    SDL_Rect bottom_vp = {0, SCREEN_SIZE[1] / 2, SCREEN_SIZE[0], SCREEN_SIZE[1] / 2};

ここでは、SDLを接続し、画像をアップロードして、上下に配置され、ウィンドウの幅全体を占める2つの長方形を作成しました。これらはビューポートです。

    SDL_RenderSetViewport(ren, &top_vp);
    SDL_RenderCopy(ren, flower, NULL, NULL);

    SDL_RenderSetViewport(ren, &bottom_vp);
    SDL_RenderCopy(ren, flower, NULL, NULL);

次に、すでに描き始めています。まず、描画の場所を確立する必要があります。これは、SDL_RenderSetViewport関数によって行われます。この関数は、レンダーと、描画したい長方形の2つのパラメーターを取ります。領域を設定した後、よく知られたSDL_RenderCopyメソッドを使用して、その中に花を描きます。その後、2番目の長方形についても同じことを繰り返します。

    SDL_RenderPresent(ren);
    SDL_Delay(2000);

    quit();
    return 0;
}

次に、ウィンドウを更新し、2秒待ってプログラムを終了します。

このプログラムをコンパイルして実行すると、2つの扁平な花が表示されます。今日お見せしたかったのはそれだけです。

さて、これはとても小さなレッスンです。質問があれば、私は説明させていただきますが、私はすべてを持っています。バイバイ！
