Cloud Security Alarm Security Hub

Sistem alarm securityHub Rusia bekerja dengan sensor kabel dan nirkabel, dapat mengontrol berbagai perangkat dari jarak jauh, terintegrasi dengan pengawasan video iVideon, dan menggunakan koneksi Ethernet kabel dan Internet seluler untuk komunikasi.

Saya sudah mempelajari sistem ini selama dua minggu sekarang.



Saya melihat stand perusahaan Kazan Teko di pameran Securika setahun yang lalu. Antara lain, mereka menunjukkan alarm rumah Security Hub. Saya tertarik pada dua hal: mendukung tidak hanya nirkabel tetapi juga sensor kabel dan kemampuan untuk terhubung ke Internet melalui kabel Ethernet biasa.

Sekarang banyak produsen memproduksi alarm rumah yang hanya mendukung sensor nirkabel. Tidak hanya mahal, tetapi juga kurang dapat diandalkan. Saya percaya bahwa setidaknya dua sensor (membuka pintu depan dan gerakan di koridor) harus ditransfer. Koneksi kabel ke router juga tampaknya lebih dapat diandalkan daripada Wi-Fi.

Saya akan mulai dengan fitur Security Hub.

  • dukungan untuk keamanan kabel dan sensor kebakaran untuk hubungan pendek, terbuka atau dua tingkat;
  • hingga 4 input untuk sensor kabel (sejumlah sensor dapat dihubungkan ke setiap input);
  • mendukung hingga 30 sensor nirkabel dengan protokol pertukaran amannya sendiri;
  • dukungan hingga enam keluaran kabel (suar, sirene, aktuator);
  • Input Memory Sentuh (TM) untuk mengelola tombol dan / atau keyboard iButton;
  • manajemen dan transmisi informasi tentang alarm dan status melalui Internet kabel (konektor RJ45) dan Internet seluler (dua kartu SIM);
  • manajemen dan konfigurasi melalui aplikasi seluler, program komputer atau situs web;
  • integrasi dengan layanan iVideon (merekam klip 10 detik selama alarm);
  • didukung oleh tiga sumber independen, baterai built-in.

Untuk membuat sistem alarm rumah yang tidak mahal, Anda dapat melakukan pembelian satu kontroler dan menggunakannya dengan sensor kabel yang murah.


Tentu saja, untuk konfigurasi apa pun Anda dapat membeli sensor nirkabel, ada banyak dari mereka.

Security Hub memiliki banyak mode operasi dan pengaturan yang halus, tetapi jika Anda menganggap alarm sebagai â€œkit out of the boxâ€, semuanya cukup sederhana. Daya disuplai dari catu daya USB konvensional melalui kabel USB AB. Aplikasi Security HUB diinstal pada smartphone, sebuah akun dibuat.


Anda dapat mempersenjatai dan melucuti alarm menggunakan aplikasi seluler. Jika terjadi alarm, pemberitahuan push akan datang, pemberitahuan suara akan berbunyi (ada peluang untuk mengatur suara Anda untuk setiap acara) dan pemberitahuan suara.

Kartu SIM di Security Hub hanya digunakan untuk transfer data. Sistem alarm tidak tahu cara menelepon dan mengirim SMS. Untuk meningkatkan keandalan pengiriman pesan alarm, Anda dapat mengkonfigurasi salah satu output dan menghubungkan dialer eksternal dengan kartu SIM sendiri, misalnya, pelacak GPS mobil murah.

Jika alarm harus sering dipersenjatai dan dihilangkan darinya, tombol-tombol pada aplikasi pada smartphone bukanlah pilihan yang paling nyaman. Anda dapat menghubungkan pembaca TM ke Hub Keamanan (150-200 rubel) dan menggunakan tablet tombol "on-door" "tombol-i" (masing-masing 30 rubel) untuk mempersenjatai dan menghapusnya.


Perangkat tambahan lain untuk pengaturan / pelepasan juga ditawarkan: keyboard kabel untuk memasukkan kode (1411 rubel), keyboard nirkabel (2065 rubel), gantungan kunci (1062 rubel).


Dimungkinkan untuk mengatur penundaan sensor (ini diperlukan ketika sensor pembukaan berada di pintu depan, dan lengan / pelucutan dilakukan menggunakan tombol TM atau keyboard dalam ruangan). Secara default, penundaan adalah 45 detik untuk keluar dan 60 detik untuk masuk, dengan menggunakan program konfigurator LocalConfig opsional, Anda dapat mengubah waktu tunda (20-30-45-60-90-120 dtk).

Papan pengontrol memiliki slot untuk dua kartu SIM, sakelar tamper, blok dua pin untuk menghubungkan pembaca TM, kontak untuk menghubungkan dua catu daya 12 V, kontak untuk empat input / output yang dapat diprogram dan dua output.


Memasang kartu sim adalah opsional, tetapi jika Ethernet tidak berfungsi, data akan dikirim melalui kartu SIM pertama, jika tidak berfungsi dan itu melalui yang kedua. Anda juga dapat bekerja hanya melalui Internet seluler tanpa koneksi kabel.

Pada tingkat bebas, kamera merekam klip sepuluh detik menggunakan detektor gerakannya (Anda dapat menontonnya di aplikasi iVideon). Ketika alarm dipicu, kamera juga merekam klip sepuluh detik (mereka dapat dilihat di aplikasi Security Hub). Rol disimpan hari.


Aplikasi ini menampilkan informasi lengkap tentang sensor nirkabel dan riwayat acara.


Sayangnya, tidak semua pengaturan Security Hub intuitif. Jadi untuk menambahkan kunci TM, Anda perlu mengklik tombol "Manajemen", di sana "+", "pengenal nyata", "aktifkan mode interaktif".


Selanjutnya, sentuh tombol pembaca, beri nama ke tombol dan tunjukkan bagian yang akan diletakkan / dihapus oleh tombol ini.


Tidak ada masalah dengan menambahkan sensor: cukup buka "sensor", tekan "+", pilih metode menghubungkan sensor (kabel atau nirkabel), tentukan nomor input untuk sensor kabel, pilih jenis sensor dan bagian di mana sensor akan ditambahkan.


Dengan keluar, hal-hal sedikit lebih rumit. Empat kontak pertama dapat berupa input atau output. Kelima dan keenam - hanya keluar.

Yang paling sederhana adalah menambahkan relay yang dikendalikan secara manual. Kami pergi ke "relay / output", tekan "+", "add relay", pilih metode koneksi, datang dengan nama, masukkan nomor terminal dari 1 hingga 6 untuk relay kabel, pilih "kontrol manual".



Di bagian â€œRelays / outputsâ€, relai yang dibuat akan muncul dengan tombol â€œnyalakanâ€ dan â€œmatikanâ€. Relai semacam itu dapat digunakan untuk mengontrol cahaya, pemanasan, atau untuk menyalakan sirene yang kuat secara manual.


Ada dua mode keluar khusus - "suar" dan "sirene". "Beacon" adalah indikator cahaya yang menunjukkan kondisi alarm yang menyala dan berkedip (menyala ketika alarm berbunyi, itu berkedip perlahan selama penundaan melucuti / mempersenjatai, dan berkedip cepat ketika alarm terjadi). "Sirene" menyala terus menerus selama alarm normal dan sebentar-sebentar saat kebakaran.

Kontrol skrip memungkinkan Anda untuk mengimplementasikan berbagai logika output. Secara total, Hub Keamanan memiliki 23 skrip dalam lima kategori. Secara singkat sebutkan beberapa fitur:

  • Menghidupkan relai untuk interval tertentu (alih-alih menyalakan / mematikan, relai akan memiliki satu pulsa tombol, yang menyalakannya, misalnya, selama 5 detik).
  • Menghidupkan relai dengan sensor apa pun dan mematikannya setelah waktu yang ditentukan (dapat digunakan untuk mengontrol cahaya dengan sensor gerak);
  • Dimasukkannya penyiraman tanaman pada sensor kelembaban;
  • Kontrol gerbang (menerapkan pulsa ke relai saat tombol ditekan);
  • Manajemen keran dari sensor kebocoran;
  • Manajemen keluar terjadwal (sekali sehari atau tiga kali sehari);
  • Kontrol relay tergantung pada suhu.

Mengonfigurasi dan mengelola Hub Keamanan dimungkinkan tidak hanya melalui aplikasi seluler. Ada cloud.security-hub.ru situs manajemen.


Ada juga program untuk stasiun pemantauan PC Astra.

Security Hub memiliki banyak fitur dan memiliki banyak keunggulan:

  • dukungan untuk sensor kabel dan nirkabel yang beroperasi melalui protokol yang aman;
  • Dukungan TM;
  • ketersediaan output dengan kemampuan kontrol yang hebat;
  • Koneksi internet melalui Internet kabel dan seluler;
  • manajemen dan konfigurasi melalui aplikasi seluler, program komputer atau situs web;
  • integrasi dengan iVideon.

Kelemahan utama dari Hub Keamanan adalah kurangnya peringatan melalui panggilan dan SMS. Semuanya bekerja hanya melalui cloud. Jelas bahwa sistem itu dikembangkan, pertama-tama, untuk kendali jarak jauh, dan di sana tidak diperlukan, tetapi karena Hub Keamanan dijual sebagai sistem untuk "perlindungan diri", panggilan alarm dapat ditambahkan. Tentu saja, Anda dapat menghubungkan dialer eksternal dengan kartu sim yang terpisah ke salah satu output controller, tetapi ini adalah "kruk".

Kelemahan yang kurang signifikan adalah kurangnya kemungkinan mempersenjatai menggunakan tombol (dan akan lebih mudah meninggalkan rumah hanya dengan menekan tombol) dan logika operasi TM: semua kunci dipersenjatai dan dilucuti, sehingga satu kunci dipersenjatai, dan tembakan lain, Anda tidak bisa.

Dan Hub Keamanan tidak memiliki instruksi rinci. Punya WIKIdengan deskripsi fungsi dan pengaturan, tetapi, misalnya, tidak ada daftar skrip yang menjelaskan apa yang dilakukan semua orang (mereka hanya ada di aplikasi seluler dan Anda perlu mengklik semua item dalam satu baris untuk memahami apa yang mereka lakukan).

Namun, Security Hub mengesankan dengan jumlah pengaturan dan fitur. Saya bisa membayangkan berapa banyak waktu dan upaya yang telah dihabiskan untuk mengembangkan sistem seperti itu. Saya sangat berharap bahwa pengembang akan menambahkan peringatan panggilan dan SMS di firmware baru, serta menambahkan kemampuan untuk mempersenjatai sistem dengan mengunci input TM, dan menulis instruksi terperinci.
