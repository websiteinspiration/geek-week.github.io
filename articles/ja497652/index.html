マテリアルデザインライブラリのスライダーコンポーネント

それは小さな探偵小説から始まりました-マテリアルデザインのウェブサイトを見て、私はスライダーページに出くわしました。説明では、このコンポーネントはAndroidで利用可能であり、githubへのリンクも提供されているとのことです。私は彼のことを聞いたことがなかったので、これは私を少し驚かせました。私はリンクをたどります-それはGithubでコンポーネントがまだ活発に開発中であることを示しており、Javaのいくつかの乏しい例が与えられています。 「これらのあなたのインターネットで」スライダーの言及は見つかりませんでした。そこへの参照ではない公式のライブラリのドキュメントでは、どちらか。

幸いなことに、ソースコードがあります。

好奇心が広まり、私は自分でそれを掘り下げ始めました。

外観は、Sliderは標準のSeekbarに似ています。ただし、若干の違いがあります。それらを見るために、私は簡単なプロジェクトをスケッチしました。いくつかのスライダーを画面、ボタンに追加し、動作中のコンポーネントを確認します。

アクティビティクラスのコード。

//    ,    ,
//   ,   ,   .
// http://developer.alexanderklimov.ru/android/

package ru.alexanderklimov.as36

import android.os.Bundle
import androidx.appcompat.app.AppCompatActivity
import com.google.android.material.slider.Slider
import kotlinx.android.synthetic.main.activity_main.*


class MainActivity : AppCompatActivity() {

    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)

        setContentView(R.layout.activity_main)

        button.setOnClickListener {
            slider.value = 5.0F
        }

        slider.values = listOf(1F, 4F, 6F)

        slider.addOnSliderTouchListener(object : Slider.OnSliderTouchListener {
            override fun onStartTrackingTouch(slider: Slider) {
                println("Start Tracking Touch")
            }

            override fun onStopTrackingTouch(slider: Slider) {
                println("Stop Tracking Touch")
            }
        })

    }
}

結果を見てみましょう。



最初に目を引くのは、slider.valuesを通じていくつかのスライダーを設定できることです。これらは独立して動作し、相互に干渉しません。

2番目のスライダーには、android：stepSize属性で設定される離散モードがあります。このモードでは、小さなドットがトラックに表示されていることに気付くでしょう。

スライダーをクリックすると、現在の値を示すプレートが上部に表示されます。



プログラムでボタンをクリックすると、最初のスライダーの適切な場所にスライダーが設定されます。この例では、値のリストをドロップして1つだけ残したため、コンポーネントは1つのスライダーを持つカボチャスライダーに変わります。

スライダーにはいくつかのリスナーがあります。実例として、そのうちの1つ、Slider.OnSliderTouchListenerを持ち込みました。

ダイ上のテキストは変更できます。これは、大きな数を使用する場合に役立ちます。次に、ゼロの数が多い数値（数百万、数兆など）の代わりに、省略されたオプションを使用できます。LabelFormatterインターフェースがこの動作を担当します。ヘルパーインターフェイスBasicLabelFormatterもあります。これには、9100ではなく9.1Kのような大きな数の便利な略語がすでに含まれています。



少し混乱させて、3文字の単語を書いてみましょう。

slider2.setLabelFormatter {
    ""
}



新しい要素は興味深く、プロジェクトでの使用に非常に適しているように見えました。
