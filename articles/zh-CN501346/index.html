机械键盘Vortex Core RGB概述

我对键盘一无所知，一生中只有Genius，Logitech和罂粟键盘。我想爱键盘，但它们让我生气。我不会盲目地打印，也没有接触到按钮，没有接触到一些东西，我需要记住不同的布局，其中相同的按钮可以位于不同的位置。除此之外，令人烦恼的是标准布局不便于编程：右手小指附近的所有括号和标志都使它过载，没有乐趣。箭头是为右手绘制的，附近还有鼠标，您经常需要移动手。

我的不满终于结束了Nikita 关于键盘设计的帖子。我意识到这不是唯一的一种，您可以寻找一些选择。我去了陈列室，并选择了最不寻常的：40％的紧凑型键盘，两个空格，手边的箭头和中间的支架。我无法抗拒，经过几个月的使用，我准备对其进行审查。



多少钱？昂贵


键盘是一种相当简单的电子产品：数十个按钮，一块板，一个用于信号识别的控制器和键帽。

在普通键盘中，除了按钮之外，还有一个薄膜，该按钮由于橡胶的弹性而反弹。这样可以降低键盘的价格，但是无法维修，并且很难提供任何特殊的打印体验。

否则：在每个按钮中放置一个单独的弹簧机构。然后事实证明可以控制按下按钮的性质，响应时间和按键声音。这种密钥的机制已经发明了很长一段时间，您可以在Wikipedia上了解它。

薄膜键盘的价格非常低廉，起价为100卢布。机械键盘更昂贵，因为制作许多机构更加困难。因此，价格迅速上涨至10,000卢布，很容易找到并且价格更高。随着价格的增长，键盘的质量和使用感觉也在不断提高。

在陈列室的所有键盘中，我最喜欢Vortex Core RGB，它是最不寻常的：紧凑的箭头直接位于字母上。它的价格为10,000卢布，但我说服自己买了，因为这是工作的主要工具。顺便说一下，Apple Magic Keyboard的价格是相同的。

五层和负载重新分配


即使是最普通的键盘也至少具有两层：正常一层和按Shift之后。通常也可以按住Alt键并访问稀有字符。
Vortex Core的按键明显更少，但它们的功能并未消失，而是移至其他层（在键盘上，它们的侧面分别用红色，绿色和蓝色签名）。它们通过夹紧功能键激活：

  • Fn将允许访问箭头和多媒体。
  • Fn1-数字和F键。
  • Fn1 + Shift-特殊符号。



您可以在一个月内习惯这种布局。学习比切换到DVORAK或COLEMAK（字母在其中改变位置）容易得多。在这里，新的安排仅适用于附加键和一些俄语字母。

购买此键盘，我期望：

  • 箭头在方便的位置。
  • 可以用不同的手指（不仅仅是小指）按下键盘中央的括号。
  • 您不需要达到的数字和符号。

实际上，有些事情变得更方便了，但是意外的缺点也出现了，所以我将告诉您详细信息。

箭


在常规键盘中，箭头位于右侧，离手臂的基本位置很远。因此，不仅必须将手移动到鼠标上，而且还必须将其移动到箭头上。但是动作越少，工作就越容易。您可以通过移动IJKL键上的箭头来解决此问题。在此键盘中，箭头应保持原来的状态：



在空格的右边是Fn按钮，如果按住该按钮，则键盘上的所有蓝色标记都将变为可用。我只需要箭头和音量键。方便地，Fn位于右手拇指的正下方，并一键按下。我几乎半小时就习惯了。

数字和F1-F12




还有另一个修改器按钮-Fn1。它用红色标题替换按钮：数字出现在基线上，而F键代替qwerty。我也不必长时间适应这种情况。对于我来说，奇怪的是数字被一个按钮向左移动-1代替了制表符。

括号和标志


随着蓝色和红色按钮整理出来，保持绿色。在这里，主要的失望让我感到震惊-我期望可以使用Pn按钮将其打开，但是事实证明，可以使用Fn1 + Shift组合键将它们打开。

这是最重要的键盘错误！要同时按下两个按钮，您必须将手向右移动，但是很难触及中间的键。尽管过程中没有涉及到整个手的张力，但在拇指上还是可以看到的：



结果，整个手都在移动，失去了小键盘的便利性。您可以重新刷新键盘，这样很好。我要做的第一件事是将Fn1移到正确的空间，问题消失了。

所有括号都在食指和中指下方，这非常方便。我不得不习惯了大约一个星期。

Pn按钮


如果Pn不包括绿色按钮，那么它将做什么？事实证明，Pn仅控制背光。您可以选择颜色，闪烁模式，按键后，灯光可能会发散开来，但这似乎是多余的。该按钮太重要，很少使用。但在新年+1花环。

俄语字母


然后痛苦开始了。但是要理解它，您需要键盘世界中拐杖的背景知识。

Klava由物理部分和软件组成。该软件已经解决，我们将任何键盘连接到任何计算机，通常可以立即开始工作。自古以来，键码及其修饰符就一直存在。因此，俄语X将在英语键{上，因此将其输入到系统中。您可以在计算机中更改布局，但是不能只连接键盘并开始工作，而是需要安装布局。由于其复杂性，很少有人打扰。

俄语字母比英语字母多，因此它们被放置在各种特殊符号上。如果将括号移近中心（方便），则俄语按钮将随其离开（不方便）。因此，字母E，E，X和Kommersant位于红色层的中心。新位置必须学习最长的时间，因为这些按钮没有签名，必须通过详尽的搜索才能找到。结果，我已经习惯了，但是实际上很难输入：：您必须将任何一只手的手指拉到Fn1，Shift和字母

额外的钥匙


在小型键盘中，每个按钮都应该有用。但是我立即发现了一些额外的键。有足够的废话：

  1. 第二个空间。默认情况下，它没有任何用处。
  2. Pn按钮控制背光，但如果它包括绿色层，效果会更好。
  3. Fn1按钮不方便。您可以将其移动到正确的空间。
  4. Del按钮，尽管Backspace在它旁边。如果您确实想要，可以通过Fn + Backspace来完成。
  5. 右Alt和Win。有人使用它们吗？

快捷方式变得不同


我在编程时会使用很多快捷方式。其中一些通过支架连接。从现在起，括号被放到新的位置，快捷键的手势也变得不同。改变这种习惯是最困难的，因为每个快捷方式都必须经过3个阶段：

  1. 记住我之前按下过的按钮
  2. 了解现在点击的位置
  3. 记住这个手势。

结果，我什至忘记了一些捷径，因为太多的事情是懒惰的。其中一些变得很长，以至于用两只手都无法按压它们，而我停止使用它们。

图层和宏


我们已经处理了四层：常规层，红色层，蓝色层和绿色层。但这还不是全部！此外，您可以再记录4个层，并在必要时使用Fn切换到它们。但这还不是全部：键盘可以存储宏，您甚至不需要程序来记录它们，一切都在键盘上完成。

也许这对某人非常重要，但是我还没有找到应用程序。更改图层时，LED始终点亮并点亮，这会产生干扰。

人机工程学


键盘很小。它比紧凑的Apple Magic Keyboard键盘小，并且与其余的相比，它似乎很小。为了进行比较，照片显示了120％的擎天柱。



我在笔记本电脑上工作，但附加键盘使它一开始不舒服：桌子的深度不足以同时容纳键盘和笔记本电脑。支架不仅解决了这个问题，而且还纠正了头部的位置，您不再需要往下看。



键盘很高。我以前是在笔记本电脑上打印的，它的钥匙在桌子的平面上，这个高度高了三厘米。刷架使打印更舒适。
键盘是矩形的，像砖头一样：外壳和按键都不倾斜。刚开始时，让我感到不舒服，然后才意识到总体上我握错了手。我的笔刷几乎拉直了，但我不得不挤压手指，以便将它们垂直放置在主行上。

Micro-USB位于顶部。如果使用标准电线，它将伸出键盘并干扰将其靠近显示器。欧尚的A型线完全可以解决该问题。



移动，但易受攻击。由于键盘的大小，便于携带，但是角键距离边缘太近，因此很容易飞走。每次将其从公文包中取出时，我都必须寻找其中一个按钮并将其放置到位。钥匙扣很可能已磨损，随着时间的流逝，问题将变得更加严重。它可以用布袋解决，但不包括在内。

尽管键盘很大，但它很重。金属是金属的基础。我的投资组合只是她的投资组合，但是如果您穿其他衣服，那么400克的额外重量可能会造成干扰。进行比较：Vortex Core RGB的重量是Magic键盘的两倍，但比iPad轻。

闪烁


可以（并且应该）更改布局。键盘支持主层和最多四个附加层，您可以使用快捷方式在它们之间进行切换。您可以直接从键盘记录宏（我没有使用过），也可以从计算机重写布局（我经常使用过）。

您可以从制造商那里下载固件，该固件可以在同一站点上配置键盘。只有Windows才可以安装固件，因此我不得不继续访问。新固件使键盘令人恶心：俄语按钮移到了新位置，而从Caps Lock中，LED开始在眼前发光。您仍然不能更改标准层。

幸运的是，房地美已经遇到了这个问题。他做了一个网站 和固件，使您可以更改主层。

我的布局修改


我将Fn1设置在正确的位置以打开红色按钮。将其与Shift一起按以获取一组字符变得更加容易，但事实证明，我用右手按下空格键的次数很多，我不得不取消学习它。代替制造商，我会走得更远，并将空间分成较小的按钮。

将按钮和//从其他层移到了主要层，对于编程和Xcode快捷方式来说，它们太必要了。他们处于中心位置，将他们推向更加熟悉的位置。俄语字母也随它们一起移动：E原来在右上角，而E则在Fn1层的Shift右边。X和Kommersant保留在红色图层上。




我喜欢百分之四十的键盘格式。重新培训其他按钮的布局并不困难，但是俄语和快捷方式可能存在问题。我大约一个月就习惯了，但是键盘对我的计算机的其他用户失去了可访问性。

Vortex Core RGB键盘给人留下了深刻的印象。一方面，她使我的工作更加舒适：布景本身变得更加愉悦，编程按钮位于正确的位置。另一方面，我不能向所有人推荐它：我必须精通俄语，但是我需要习惯按钮的布局。

las，我没有冷静下来。经过键盘模拟器的培训后，我意识到按键的水平移动使我感到困扰。手指不会那样移动，但是我不习惯于移动按钮。模拟器将我的手的位置校正为更正确的手，但是腕关节对此感到厌倦。
