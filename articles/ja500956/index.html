モバイルML：ポケットで機械学習。パート1-鉄


Andrey Batutin、DataArtのシニアiOS開発者による投稿

今日、誰もが私たちの電話を含む機械学習を行っています。はい、すぐにお使いの携帯電話は本当にスマートになります。まあ、または少なくともあなたと私より賢いです。どうして？それを理解しましょう。モバイルデバイスに既にインストールされているハードウェアから始めましょう。

TL、DR


モバイルシステムオンチップ（SoC）のすべての主要メーカーは、ML固有のハードウェアをソリューションに3〜4年積極的に追加しています。トップエンドのandroidまたはiPhoneを使用している場合、GPUの隣に、MLタスク用に特別に調整されたチップがあることがほぼ保証されています。

ほとんどのMLチップは、コンピュータビジョン、オーディオ、カメラ/写真/ビデオ用に設計されています。主なタスク：写真のノイズの低減、ズームによる最高品質、顔と動物の認識、音声認識とテキスト合成。

さらに、主要ベンダーは、サードパーティの開発者がこのMLチップにアクセスできるようにするSDKを提供しています。SnapchatやSiriを作成できます。とにかく家に座っていると思います（いつ終了するのですか？！KFCでどのようにしたいですか！）

鉄


主要なモバイルチップベンダーは、ニューラルプロセッシングユニット（NPU）、デジタルシグナルプロセッサー（DSP）、および専門のAIコアを活用して、SoCアーキテクチャを積極的に拡張しています。これらのコンポーネントは、MLモデルの操作のために特別にシャープ化されています。

クアルコム


MLハードウェア： DSP + GPU
SDK： QualcommニューラルプロセッシングSDK

によって提供されるQualcommニューラルプロセッシングSDK。これは、Snapdragonチップ用の一連のDSP + GPU + CPUでMLモデルのハードウェアアクセラレーションを提供します。DSPは、オーディオ/ビデオでの作業用に調整されています。スマートカメラ、ノイズから画像をクリーニング、ズームなどの最高の品質で音質を向上させるツールです。

HiSilicon / Huawei


MLハードウェア： NPU
SDK： HiAI SDK

Da Vinci NPUは、MLコンピューティング用の2つの高性能コアと1つのエネルギー効率の高いコアの3つのコアで構成されています。

HiAI SDKは、NPUマトリックスの操作のためのグラウンドへのアクセスを提供します。つまり、NPUはディープニューラルネットワークモデルに最適です。便利なのは、Android Studioのプラグインです。

HiAIは次の目的でシャープになります。

  • コンピュータビジョン
  • 自動音声認識;
  • 自然言語理解。


MediaTek


MLハードウェア： APU + GPU
SDK： NeuroPilot SDK

NeuroPilot SDKを使用すると、AI処理ユニット（APU）+ GPUを使用してMLの問題を解決できます。APUは、Deep Neural Networkモデルで投獄されています。畳み込み、完全に接続されたレイヤー、アクティブ化機能などにハードウェアアクセラレーションを提供します

。NeuroPilot2.0は、リアルタイムで次のことを可能にします。

  • 複数人のポーズ追跡;
  • 3Dポーズトラッキング。
  • 複数のオブジェクトの識別。
  • セマンティックセグメンテーション。
  • 画像の強化。


サムスン


MLハードウェア： NPU
SDK： Samsung Neural SDK / EDEN SDK
Samsungは、Exynos SoCに専用のNPUを追加しました。これは、行列の演算用にシャープにされた2つの乗算-累算ユニットで構成されます。

また、CPU + GPU + NPUの組み合わせを使用してMLモデルにハードウェアアクセラレーションを提供するSamsung Neural SDKも提供します。

林檎


MLハードウェア： NPU
SDK： CoreML SDK

Apple A11以降、BionicはNPUを使用しています。A13 NPUは8コアに成長しました。新しいNPUのキラー機能の1つは、Deep Fusion画像処理です。9枚のショットを取り、それらを1つに結合します。特に夜間撮影に適しています。

CoreML SDKを提供します。他のSDKと同様に、MLモデルにハードウェアアクセラレーションを提供します。パンから-MLの作成-MLモデルをトレーニングするためのUIのみの環境。あなたが訓練することができます：

  • オブジェクト検出/分類器;
  • サウンド分類子;
  • モーション分類器;
  • テキスト分類/単語のタグ付け;
  • 表分類子;
  • 推奨エンジン。
