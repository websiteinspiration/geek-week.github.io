Git 15-




Gitバージョン管理システムの最初のリリースは、2005年4月7日から15年前に行われました。以前に使用されていたBitKeeperで使用されていたライセンスが変更されたため、GitはLinuxカーネルのハードカレンシーとして始まりました。しかし今日に至るまで、Gitはコアのみのハードカレンシーという本来の役割を大きく超えており、世界中のほとんどすべてのフリーでオープンなプロプライエタリなソフトウェアの開発の基礎となっています。

「2005年の導入以来、Gitは元の品質を維持しながら、使いやすいシステムに進化しました。彼は驚くほど速く、大規模なプロジェクトでの作業に効率的であり、非線形開発のための優れたブランチシステムを持っています。「スコットチャコンとベンストラウブの本を書きます。
