Cree aplicaciones de forma gratuita con la capa gratuita de Azure Cosmos DB

¿Desea crear una nueva aplicación, desarrollar y probar o ejecutar pequeñas cargas de trabajo de producción con  Azure Cosmos DB ? Nuestro nuevo nivel de uso gratuito le permite comenzar a trabajar sin costos y ahorrar dinero al crear y desarrollar nuevas aplicaciones.



Con el nivel gratuito de Azure Cosmos DB incluido, obtiene los primeros 400 RU / s de ancho de banda y 5 GB de almacenamiento en su cuenta de forma gratuita todos los meses durante toda la duración de la cuenta. Esto significa que puede comenzar de a poco y crecer con confianza, sabiendo que su aplicación se ejecutará en un servicio de base de datos de alto rendimiento. Pagará solo si su cuenta supera los 400 RU / sy 5 GB. Además, si su aplicación tiene muchos contenedores, puede crear hasta 25 contenedores en la base de datos con un ancho de banda total, y todos utilizarán 400 RU / s libres. Puede tener hasta una cuenta gratuita de Azure Cosmos DB por cada suscripción de Azure.


¿Qué puedes crear gratis?


Pedimos a los miembros de nuestra comunidad de desarrollo que proporcionaran ejemplos de aplicaciones con 400 RU / sy 5 GB de espacio en disco y recibimos muchos ejemplos excelentes de las aplicaciones que crearon: la aplicación Alexa, que ayuda a los pasajeros a usar el transporte público, así como una aplicación web para administrar y compartir listas en línea de manera conveniente  recursos . Hablaremos sobre algunos de estos desarrolladores y sus aplicaciones en las próximas semanas y meses.

¿Tiene una aplicación con un ancho de banda inferior a 400 RU / sy 5 GB de almacenamiento? Cuéntanos sobre él  , estaremos encantados de compartirlo con la comunidad.

Comienzo de trabajo


Cree una nueva cuenta de Azure Cosmos DB y active el  nivel gratuito  durante la creación. Hay un límite para una cuenta de suscripción gratuita de Azure. Consulte nuestras  guías y guías de inicio rápido  o use  un ejemplo para comprender lo que puede hacer con Azure Cosmos DB.

Preguntas frecuentes (FAQ)


P: ¿No ha ofrecido una opción gratuita antes? ¿Qué cambió?

R: Siempre ofrecimos formas de probar Azure Cosmos DB de forma gratuita, como parte de la  cuenta gratuita de Azure , con un  emulador local y como una  prueba gratuita de 30 días  , pero estas opciones tenían una disponibilidad y un volumen limitados. El nivel gratuito está disponible por tiempo ilimitado.

P: ¿Cómo se muestra el descuento de nivel gratuito en mi cuenta?

R: En las cuentas de nivel gratuito, obtendrá los primeros 400 RU / sy 5 GB de almacenamiento en su cuenta de forma gratuita. Cualquier RU / s y almacenamiento más allá de 400 RU / sy 5 GB se facturarán a las tarifas de precios regulares por página de precios. En la factura, no verá un cargo o una línea de pedido por los 400 Ru / sy los 5 GB gratuitos, solo los RU / sy el almacenamiento más allá de lo que cubre el nivel gratuito. En las cuentas del nivel gratuito, recibirá los primeros 400 RU / sy 5 GB de espacio en disco en su cuenta de forma gratuita. Cualquier RU / sy almacenamiento de más de 400 RU / sy 5 GB se pagarán a tarifas regulares. En la cuenta no verá una tarifa o posición por 400 Ru / sy 5 GB gratis, solo RU / sy almacenamiento más allá de lo que cubre el nivel gratuito.

P: ¿Se puede aplicar el nivel gratuito a todas las API de Azure Cosmos DB?

Sí, Free Tier funciona para todas las API de Azure Cosmos DB, incluidas Core (SQL), API para MongoDB, Cassandra y Gremlin.

P: ¿Puedo aplicar un descuento de nivel gratuito para  bases de datos y contenedores de piloto automático ?

A: si. Con el servicio de piloto automático, se le facturan las RU / s más altas, según las cuales la base de datos o el contenedor se escalan por hora. Cuando se aplica un descuento de nivel gratuito, se deducen 400 RU / s de este valor. Consulte nuestra  documentación para más detalles.

P: ¿Cómo funciona el descuento de nivel gratuito si tengo una cuenta en varias regiones?

R: En las cuentas de RU / s multirregionales, la base de datos o el contenedor se replica en todas las regiones. Por ejemplo, si tiene un contenedor con 400 RU / s, y la cuenta se encuentra en 3 regiones, el número total de cuentas RU / s es de 1200 RU / s. Cuando se aplica el descuento, se le facturará a 1200 RU / s - 400 RU / s = 800 RU / s por hora. Consulte nuestra  documentación para más detalles.

P: Ya tengo una cuenta y deseo transferir mis datos a la cuenta del nivel de uso gratuito. ¿Cómo debería hacerlo?

R: Una vez que su nueva cuenta se crea y se activa con Free Tier, mover los datos existentes a la nueva cuenta será fácil.  Herramienta de migración de datos puede usarse para transferir pequeños conjuntos de datos si usa nuestra API Core (SQL). También puede migrar datos de Azure Data Factory  para la API Azure Cosmos DB Core (SQL)  y la API Azure Cosmos DB  para MongoDB .

P: No se muestra la opción para habilitar el nivel gratuito al crear una nueva cuenta en Azure Portal. ¿Donde esta ella?

R: Una suscripción de Azure puede tener hasta una cuenta de nivel gratuita. Si no se muestra esta opción, significa que otra cuenta en la suscripción ya está habilitada en el nivel gratuito.
