ArduinoでWindowsプログラムを作成する方法



かつて、500のレーザーポインターを1か所に持ってくるのはおかしなアイデアだと思いました。私は多くの時間を費やしてそれをやりました。それは壮観で役に立たないことがわかりましたが、私はそれが好きでした。6か月前に、もう1つの奇妙なアイデアが思い付きました。今回はまったく壮観ではありませんが、はるかに便利です。私もそれに多くの時間を費やしました。そして、この記事では、私の2番目のクレイジーなアイデアのベータ版を紹介します。

私はプロジェクトをNanonyam（Nanonyam）と呼び、それにロゴを付けました（私は5分間塗装しました）。



Arduinoに関して考える人にとって、NanonyamはWindowsを管理するための仮想arduinoシールドであると言えます。

言い換えれば、NanonyamはAVRマイクロコントローラーのファームウェアをバイトコードとして使用する仮想マシンです（ATMEGA2560を推奨）。この仮想マシン内にはAVRカーネルシミュレータがありますが、0x0060から0x01FFのSRAMアドレスにある周辺デバイスの代わりに、仮想関数（Windows API関数を含む）への特別なインターフェイスがあります。ここですぐに理解することが非常に重要です。Nanonyamのコードは、たとえばファイルの削除やディスクのフォーマットなどの関数を誤って呼び出さないように、指定されたメモリ範囲にアクセスできないようにする必要があります。 0x0200〜0xFFFFの残りのSRAMメモリ範囲（これは実際のマイクロコントローラーよりも多い）は、ユーザーがあらゆる目的で使用できます。すぐにメモします実際のマイクロコントローラーのファームウェア（または別のアーキテクチャーのファームウェア）を誤って起動しないように特別な保護があること：「危険な」関数をアクティブにする前に、特別なトリッキーな仮想関数を呼び出す必要があります。他のいくつかのセキュリティ機能があります。

Nanonyam用のプログラムを作成するには、現在利用可能なすべての仮想関数を実装する特別なライブラリを使用する必要があります。 Nanonyam仮想マシンとそのライブラリは、こちらからダウンロードできます。そして、こちらが仮想機能の説明ページです。そして、はい、私のサイトは非常に原始的であり、モバイルデバイスに適応していません。

Nanonyamは、家庭用および商業用に無料です。 Nanonyamは「現状のまま」の基準です。ソースコードは提供されていません。

現在、プログラムはテスト段階にあります。 Windows用の簡単なプログラムを作成できる約200の仮想関数が実装されています。
コードのメモリが256 kBしかないため、このような仮想マシンで複雑なものを作成しても機能しません。データは個別のファイルに保存でき、グラフィックパーツのバッファは外部で実装されます。すべての機能が簡素化され、8ビットアーキテクチャに適合しています。

Nanonyamで何ができますか？私はいくつかのタスクを思いつきました。

プログラムブロックの開発


かつて、128x64のドットグラフィックディスプレイ用の複雑なメニューを開発する必要がありました。ピクセルを確認するためにファームウェアを実際のマイクロコントローラーに常にダウンロードしたくありませんでした。それで、ナンノヤムのアイデアが生まれました。下の図では、同じメニューの項目の1つの実際のOLEDディスプレイからの画像。今、私は実際のデバイスなしでそれを解決することができます。



Nanonyam（最終的なアイデア）は、マイクロコントローラー用のソフトウェアブロックを処理するための優れたツールです。グラフィック（表示とインジケーターをシミュレートできます）、ファイル（ログを作成、テストデータを読み取ることができます）、キーボード（最大で同時に10個のボタン）、COMポート付き（ここに別のアイテムがあります）。

高速プログラムを作成する


たとえば、100,500のテキストファイルをすばやく処理する必要があるとします。誰もが開く必要があり、いくつかの単純なアルゴリズムに従って少し変更し、保存して閉じる必要があります。あなたがPythonの達人なら、おめでとうございます、あなたはすべてを持っています。しかし、あなたが熱心なarduinoワーカーである場合（そしてそれらの多くが存在する場合）、Nanonyamはこの問題の解決に役立ちます。これはまさにNanonyamの私の2番目の目標です。ワードプロセッシング、スクリーンショットの作成、システムでのキーストロークのシミュレーションなど、多くの便利な機能を追加することです（ちなみに、これは既にそこにあります）。また、ルーチンタスクを解決するための他の多くの機能も追加します。

COMポートを介した腺のテスト


Nanonyamは、アルゴリズムに従って動作する端末として機能できます。小さなメニューを描画してデバイスを制御し、ポートから受信したデータを表示できます。分析用にファイルからデータを保存して読み取ることができます。腺の簡単なデバッグと校正、および簡単な仮想計器制御パネルの作成に便利なツール。学生や若い科学者にとって、このプロジェクトは非常に役立ちます。

プログラミングトレーニング


ただし、Arduinoプロジェクト全体と同様に、Nanonyamの主なユーティリティは、その機能、インターフェイス、およびブートローダーの簡素化です。したがって、このプロジェクトは、初心者プログラマーやarduinoのレベルに満足しているプログラマーにとって興味深いものになるはずです。ちなみに、私はいつもArduinoを詳しく研究していません。いつもWinAVRまたはAVR Studioを使用していたのですが、アセンブラーから始めました。したがって、以下のサンプルプログラムは少し間違っていますが、かなりうまくいきます。

こんにちは、ハブル！


Nanonyamのいくつかの機能を理解し、簡単なプログラムを作成する時が来ました。Arduinoで記述しますが、通常の方法ではありませんが、今は可能な方法で記述します（この環境にはあまり対応していないと既に述べました）。まず、新しいスケッチを作成し、Mega2560ボードを選択します。



スケッチをファイルに保存し、近くのNanonyamライブラリをコピーします。ライブラリヘッダーを含めるのは正しいことですが、Arduinoで個々のファイルのコンパイルを書き込む方法がわからないため、ここではライブラリを直接（一度に）オンにします。

#include <stdio.h>
#include "Nanonyam\nN_System_lib.c"
#include "Nanonyam\nN_Keyboard_lib.c"
#include "Nanonyam\nN_File_lib.c"
#include "Nanonyam\nN_Math_lib.c"
#include "Nanonyam\nN_Text_lib.c"
#include "Nanonyam\nN_Graphics_lib.c"
#include "Nanonyam\nN_RS232_lib.c"

Arduinoから直接インストールできる特別なモジュール「Nanonyam for Arduino」を作成するのがさらに正しいでしょう。これに対処したらすぐに行いますが、ここでは仮想マシンを操作する本質を示します。次のコードを書きます。

//      
void setup() {
  sys_Nanonyam();//   
  g_SetScreenSize(400,200);//   400200 
  sys_WindowSetText("Example");// 
  g_ConfigExternalFont(0,60,1,0,0,0,"Arial");//  Windows    0
  g_SetExternalFont(0);//   0   
  g_SetBackRGB(0,0,255);//  
  g_SetTextRGB(255,255,0);//  
  g_ClearAll();//  (  )
  g_DrawTextCenterX(0,400,70,"Hello, Habr!");// 
  g_Update();//    
}

//   
void loop() {
  sys_Delay(100);//   
}

このプログラムのスケッチはここからダウンロードできます。サイトの機能の詳細な説明を探してください。このコードのコメントが要点を理解するのに十分であることを願っています。ここでは、sys_Nanonyam（）関数が仮想マシンの「パスワード」の役割を果たし、仮想関数の制限が削除されます。この機能がない場合、プログラムは操作の3秒後に閉じます。

[チェック]ボタンをクリックすると、エラーは発生しません。



次に、バイナリファイル（ファームウェア）を取得する必要があります。メニュー「スケッチ>>バイナリファイルのエクスポート（CTRL + ALT + S）」を選択します。



この場合、2つのHEXファイルがスケッチのあるフォルダーにコピーされます。接頭辞「with_bootloader.mega」のないファイルのみを取得します。

Nanonyam仮想マシンのHEXファイルを指定するには、いくつかの方法があります。そのすべてについて、このページで説明しています。ファイルの横にNanonyam.exeファイルNanonyam.pathを作成することをお勧めします。これは、HEXファイルへの完全パスを定義します。その後、Nanonyam.exeを実行できます。私たちは私たちの碑文が付いたウィンドウを取得します。
