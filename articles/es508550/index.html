Arquitectura del cerebro Concepto de canales







En este artículo hablaré sobre una de mis hipótesis con respecto a la arquitectura del cerebro.







El razonamiento se basa en un concepto muy simple de canales. Quizás el lector incluso lo encuentre ingenuo. Sin embargo, tuve que hacer grandes esfuerzos para llegar a esto. Este no es un artículo científico. Si alguien lo llama otra especulación sobre el tema del cerebro, entonces formalmente será correcto. La crítica es bienvenida.







Introducción



La historia de esta hipótesis comenzó en 2015. Una vez más, pensando en la estructura del cerebro, llegué a una idea. Su esencia era que el cerebro debe de alguna manera construir un modelo del medio ambiente. Se necesita un modelo para romper la conexión directa entre receptores y efectores. Una criatura sin ese modelo depende casi por completo de señales externas. La locomoción es posible ya sea como taxis o como una caminata aleatoria. En cierto sentido, tal criatura es como un objeto inanimado. Por ejemplo, la pelota se mueve solo cuando recibe una "señal externa" del pie del jugador. Esto no es muy diferente del unicelular convencional, que se mueve a lo largo del gradiente químico hacia los nutrientes. Su comportamiento también está completamente determinado por señales externas. Puede cambiar este patrón colocando memoria entre receptores y efectores,que almacena un modelo crudo del medio ambiente.

class dotdict(dict):
    __getattr__ = dict.get
    def __setattr__(self, name, value):
        self[name] = value

class Machine:
    """
         tick.
            tock.
     tick  tock        .
    """

    def __init__(self, cells):
        self.cells = cells
        self.tick = dotdict()
        self.tock = dotdict()
        for key in cells:
            self.tick[key] = 0b0000
            self.tock[key] = 0b0000

    def step(self):
        """    """
        for name, func in self.cells.items():
            if func is None:
                #  ,     
                self.tock[name] = self.tick[name]
            else:
                #       
                self.tock[name] = func(self.tick)
        #   
        self.tick, self.tock = self.tock, self.tick
        self.show()

    def show(self, comment = None):
        """    """
        if comment is not None:
            print(f': {comment}')
        for key in cells:
            print(key, f'{self.tick[key]:04b}')
        print('------------------------------------------------')
