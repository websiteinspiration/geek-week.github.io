最初に実行したいのは、winAnimationロジックを置き換える関数です。メッセージが表示され、別の関数が実行されます。引数として、テキスト、反復回数、速度、コールバックを持つオブジェクトを受け取ります。 



const message = async ({text = [], count = 'infinity', speed = 1000, callback = null}) =>  {
    if(!text[0]) throw new Error('need array for text');
    let i = 0;
    let textIndex = 0;
    while(true) {
        if(count <= i && count !== 'infinity') {
            //    ,   
            if(callback) callback();
            break;
        }
        if(textIndex >= text.length) textIndex = 0;
        //      1    
        document.title = text[textIndex];
        i++;
        textIndex++;
        await sleep(speed)
    }
};

また、それらを実行および制御できるアニメーションマネージャを作成したいと考えています。これは、アニメーションリポジトリといくつかのメソッドを持つクラスになります。 

class Animations {
    animations = [];
    setAnimation(animation, args = {}) {
        //   
        for(const item of this.animations) {
            item.play = false
        }
        //  ,    
        const Animation = this.animations.find(item => item.name === animation);
        if(!Animation) return;
        Animation.play = true;
        Animation.callback(args)
    };
    isPlay(animation) {
        return this.animations.find(item => item.name === animation).play
    }
    createAnimation(name, callback) {
        this.animations.push({
            play: false,
            name,
            callback,
        })
    }
}


メソッドの説明：

setAnimation-アニメーションを呼び出します。

isPlayは、アニメーションが「再生中」かどうかを確認するための条件関数です。

createAnimation-リポジトリにアニメーションを追加します。
