ClangFormatとNotepad ++の統合

ClangFormatは、C、C ++、Java、JavaScript、Objective-C、C＃でソースコードを自動的にフォーマットするための最良のツールの1つです。一般的な開発環境（IDE）用のプラグインがありますが、多くの場合、面倒なIDEを起動せずにファイルまたはソースファイルの一部をすばやくフォーマットする必要があり、フォーマット設定とさまざまなバージョンのClangFormatを試し、変更をすばやく取り消すことができます。この目的でコンソールバージョンのClangFormatを使用するのは不便です。考えられる解決策は、テキストエディターからClangFormatを呼び出すことです。公式サイトでは Vimの、Emacsやいくつかの他の人との統合方法を説明しますが、メモ帳++との統合についての情報はありません。以下は、メモ帳++（Windows用）の簡単な手順です。

初期要件


  • ClangFormatを使用してメモ帳++で開いたファイルをフォーマットします。
  • ファイル内の選択したフラグメントをフォーマットします。
  • 変更を破棄;
  • スタイル（ルールのセット）のフォーマットの切り替え;
  • ClangFormatの別のバージョンへの切り替え
  • ClangFormatを再構築したり、メモ帳++用の新しいプラグインを作成したりせずに、可能な限り標準ツールを使用してください。

インストールとセットアップ


1. Notepad ++がまだインストールされていない場合は、ダウンロードしてインストールします


https://notepad-plus-plus.org

2. Notepad ++でNppExecプラグインをインストールします


NppExecを使用すると、メモ帳++からサードパーティのアプリケーションを呼び出し、どのメモ帳++が作成されたかに基づいて、Scintillaライブラリのコンポーネントとやり取りすることができます。

Plugins –> Plugin Admin... –> NppExec –> Install
Notepad ++が再起動し、その後、ディレクトリ<Notepad++>/plugins/NppExec/とメニュー項目が表示されますPlugins –> NppExec

3. ClangFormat実行可能ファイルをダウンロードします


これを行うには、https：//llvm.org/builds/ページで、たとえばWindows用のインストールファイルを見つけてダウンロードしますLLVM-X.X.X-rYYYYYY-win64.exe。パッケージ全体をインストールすることはできませんclang-format.exe。アーカイバでファイルを抽出するだけです。7-zipを使用できます。ファイルから拡張子を削除し、7-zipでファイル.exeを開いて、サブディレクトリからbin/ファイルを抽出しますclang-format.exe。ファイルclang-format.exeをディレクトリに置きます<Notepad++>/plugins/NppExec/clang-format/。

4. ClangFormatの構成ファイルを追加します


構成ファイルの名前は.clang-formatまたはにする必要があります_clang-format。これらには、一連のフォーマットルール（スタイル）が含まれています。そのフォーマットについては、ClangFormatスタイルオプションガイドで説明しています。

たとえば、LinuxカーネルおよびQtプロジェクトのファイルを使用します。

GitHubからダウンロードされたファイル.clang-formatは、適切なディレクトリに配置されます。
<Notepad++>/plugins/NppExec/clang-format/LINUX_KERNEL/
<Notepad++>/plugins/NppExec/clang-format/QT/

5. NppExecのスクリプトを作成する


NppExecスクリプトを編集および実行するためのウィンドウを開くPlugins –> NppExec –> Execute...か、をクリックしますF6。下のスクリプトテキストをコピーしてウィンドウに貼り付け、clang-formatボタンの名前でスクリプトを保存しますSave...。

NppExecスクリプト
// Hide console
NPP_CONSOLE 0

//------------------------------------------------------------------------------
// Uncomment a line to select a style
//set style = LINUX_KERNEL
set style = QT
//------------------------------------------------------------------------------

//------------------------------------------------------------------------------
set clangformatexe = $(CWD)\plugins\NppExec\clang-format\clang-format.exe
set clangformatcfgdir = $(CWD)\plugins\NppExec\clang-format
set tmpdir = $(SYS.TEMP)
set clangformatcfgfile = $(clangformatcfgdir)\$(style)\.clang-format
set srcfiletmp = $(tmpdir)\~src.tmp
//------------------------------------------------------------------------------

cmd.exe /c if exist "$(clangformatexe)" (exit 0) else (exit 1)
if $(EXITCODE) != 0 then
    NPP_CONSOLE 1
    echo ERROR: "$(clangformatexe)" not found
    exit
endif

cmd.exe /c if exist "$(clangformatcfgfile)" (exit 0) else (exit 1)
if $(EXITCODE) != 0 then
    NPP_CONSOLE 1
    echo ERROR: "$(clangformatcfgfile)" not found
    exit
endif

// Copy $(clangformatcfgfile) to $(tmpdir)\.clang-format if their temestamps are different
cmd.exe /v /c " for %i in ("$(clangformatcfgfile)") do set date1="%~ti" && for %i in ("$(tmpdir)\.clang-format") do set date2="%~ti" && if not "!date1!"=="!date2!" ( echo !date1! != !date2! && echo COPYING $(clangformatcfgfile) to $(tmpdir)\ && copy "$(clangformatcfgfile)" "$(tmpdir)\" /Y )"
if $(EXITCODE) != 0 then
    NPP_CONSOLE 1
    echo ERROR copying "$(clangformatcfgfile)" to "$(tmpdir)"
    exit
endif

// Get selected text length
sci_sendmsg SCI_GETSELTEXT

// If nothing is selected - select the current line
if $(MSG_RESULT) == 1 then
    sci_sendmsg SCI_VCHOMEWRAP
    sci_sendmsg SCI_LINEENDWRAPEXTEND
endif

// Save selected text to $(srcfiletmp)
sel_saveto "$(srcfiletmp)" :a

cmd.exe /c if exist "$(srcfiletmp)" (exit 0) else (exit 1)
if $(EXITCODE) != 0 then
    NPP_CONSOLE 1
    echo ERROR: "$(srcfiletmp)" not found
    exit
endif

// Run ClangFormat
$(clangformatexe) -i -style=file "$(srcfiletmp)"
if $(EXITCODE) != 0 then
    NPP_CONSOLE 1
    echo ERROR running "$(clangformatexe)"
    exit
endif

// Replace selected text with $(srcfiletmp)
sel_loadfrom "$(srcfiletmp)"

// Delete $(srcfiletmp) file
cmd.exe /c del "$(srcfiletmp)"


スクリプトを開始した後、変数styleで指定された選択されたフォーマットスタイルに基づいて、目的のファイルを選択し、.clang-formatその変更の日付を確認し、必要に応じて一時ディレクトリにコピーします。そこで、選択したソースコードのフラグメントが一時ファイルにコピーされ、起動されclang-format.exeます。フォーマットされたフラグメントは、メモ帳++ウィンドウにコピーされます。その後、一時ファイルは削除されます。

ClangFormatでは、構成ファイルへのパスを指定する方法はありません.clang-format。 ClangFormatは、フォーマットされたファイルのディレクトリで検索し、見つからない場合は、親ディレクトリを検索します。スクリプトが終了した後、.clang-formatフォーマットが開始されるたびにコピーされないように、ファイルは一時ディレクトリに残ります。

スクリプトのすべての段階でエラーがチェックされ、エラーが発生すると、NppExecコンソールウィンドウが開き、メッセージが表示されます。

6. Notepad ++で、スクリプトを実行するための新しいメニュー項目を追加します


を開きPlugins –> NppExec –> Advanced Options...、ドロップダウンリストからAssociated scriptスクリプト名を選択clang-formatして、ボタンをクリックしますAdd/Modify。

Notepad ++を再起動すると、メニュー項目が表示されますPlugins –> NppExec –> clang-format。



7.メモ帳++で、スクリプトを実行するためのキーボードショートカットを構成します。


Ctrl + IQtCreator と同様の組み合わせを使用します。オープンしSettings –> Shortcut Mapperます。

デフォルトでは、組み合わせはCtrl + Iビジー状態であるためClear、タブのボタンMain Menu（Split Lines現在のバージョンのNotepad ++の38行目）でボタンを離す必要があります。その後、タブでPlugin commands組み合わせCtrl + Iをスクリプトに割り当てますclang-format。

できました、使用できます！

使用説明書


ソースファイルをメモ帳++で開き、目的のフラグメントまたはすべてのテキストを選択して、をクリックしますCtrl + I。何も選択されていない場合、現在の行がフォーマットされます。



変更を元に戻すには、標準のエディターツール（Ctrl + Z）を使用します。
フォーマットルールを変更するに.clang-formatは、ディレクトリ内の構成ファイルを編集します。<Notepad++>/plugins/NppExec/clang-format/
別のバージョンのClangFormatを使用する必要がある場合は、スクリプトで実行可能ファイルへのパスを変更します。
set clangformat = "\path\to\clang-format.exe"
別のフォーマットスタイルを選択するには、をクリックF6して、スクリプトテキストでclang-format目的のスタイルを選択します
set style = STYLE_NAME。

必要なファイルはすべてアーカイブにあります。
