範囲の前後のC ++コード例

また会いましたね。次の資料の翻訳は、6月27日に開始されるコース「C ++ Developer」の学生向けに特別に準備されたものです。



Rangesライブラリは、昨年11月にサンディエゴで開催された標準委員会の会議でC ++ 20に採用されました。ライブラリは、コードを簡略化するために設計された値の範囲を処理するためのコンポーネントを提供します。残念ながら、Rangesライブラリは十分に文書化されていないため、それを習得したい人にとって理解が難しくなります。この投稿は、範囲を使用して、または使用せずに記述されたコードの例を提供することを目的としています。

Eric NieblerのRangesライブラリの実装は、こちらから入手できます。Clang 3.6.2以降、gcc 5.2以降、VC ++ 15.9以降で動作します。以下のコード例は、最新のコンパイラーで作成およびテストされています。これらの例は典型的な実装であり、必ずしも考えられる唯一のソリューションではないことに注意してください。

Rangesライブラリの標準の名前空間はですがstd::ranges、この現在のライブラリ実装ではranges::v3です。

以下の例では、次の名前空間エイリアスが使用されています。

namespace rs = ranges::v3;
namespace rv = ranges::v3::view;
namespace ra = ranges::v3::action;

また、簡略化のために、以下のオブジェクト、関数、ラムダを参照します。

std::string to_roman(int value)
{
   std::vector<std::pair<int, char const*>> roman
   {
      { 1000, "M" },{ 900, "CM" },
      { 500, "D" },{ 400, "CD" },
      { 100, "C" },{ 90, "XC" },
      { 50, "L" },{ 40, "XL" },
      { 10, "X" },{ 9, "IX" },
      { 5, "V" },{ 4, "IV" },
      { 1, "I" }
   };
 
   std::string result;
   for (auto const & [d, r]: roman)
   {
      while (value >= d)
      {
     	result += r;
     	value -= d;
      }
   }
 
   return result;
}
 
std::vector<int> v{1,1,2,3,5,8,13,21,34};
 
auto print_elem = [](auto const e) {std::cout << e << '\n'; };
 
auto is_even = [](auto const i) {return i % 2 == 0; };

std::for_each(

rs::for_each(

範囲のすべての要素を逆の順序で出力します。

範囲の偶数の要素のみを出力しますが、逆の順序で出力します。

範囲に 鳴った後
C ++ C ++ 
std::for_each(
   std::crbegin(v), std::crend(v),
   [print_elem](auto const i) {
      if(i % 2 == 0)
     	print_elem(i);
   });

 
for (auto const i : v
                  | rv::reverse
                  | rv::filter(is_even))
{
   print_elem(i);
};


範囲の最初の2つの要素をスキップし、次の3つの要素のうちの1つだけを印刷します。

範囲に 鳴った後
C ++ C ++ 
 
auto it = std::cbegin(v);
std::advance(it, 2);
auto ix = 0;
while (it != std::cend(v) && ix++ < 3)
{
   if (is_even(*it))
      print_elem(*it);
   it++;
}

 
for (auto const i : v
                  | rv::drop(2)
                  | rv::take(3)
                  | rv::filter(is_even))
{
   print_elem(i);
};

範囲[101、200]で7で割り切れる最後の3つの数字のローマ数字を逆の順序で出力します。

範囲に 鳴った後
C ++ C ++ 
 
for (int n = 200, count=0; n >= 101 && count < 3; --n)
{
   if (n % 7 == 0)
   {
      print_elem(to_roman(n));
      count++;
   }
}

 
for (auto n : rs::iota_view(101, 201)
            | rv::reverse
            | rv::filter([](auto v) {
                return v % 7 == 0; })
            | rv::transform(to_roman)
            | rv::take(3))
{
   print_elem(n);
}


範囲[101、200]の7の倍数である最後の3つの数字のローマ数字を含む文字列の範囲を逆の順序で作成します。

範囲に 鳴った後
C ++ C ++ 
 
std::vector<std::string> v;
for (int n = 200, count = 0;
 	n >= 101 && count < 3; --n)
{
   if (n % 7 == 0)
   {
      v.push_back(to_roman(n));
      count++;
   }
}

auto v = rs::iota_view(101, 201)
   	| rv::reverse
   	| rv::filter([](auto v) {return v % 7 == 0; })
   	| rv::transform(to_roman)
   	| rv::take(3)
   	| rs::to_vector;


並べ替えられていない範囲を変更して、一意の値のみが保持されるようにしますが、順序は逆になります。

範囲に 鳴った後
C ++ C ++ 
 
std::vector<int> v{ 21, 1, 3, 8, 13, 1, 5, 2 };
 
std::sort(std::begin(v), std::end(v));
v.erase(
   std::unique(std::begin(v), std::end(v)),
   std::end(v));
std::reverse(std::begin(v), std::end(v));

 
std::vector<int> v{ 21, 1, 3, 8, 13, 1, 5, 2 };
 
v = std::move(v) |
    ra::sort |
    ra::unique |
    ra::reverse;


範囲の2つの最小値と2つの最大値を削除し、残りを2番目の範囲に並べたままにします。

範囲に 鳴った後

std::vector<int> v{ 21, 1, 3, 8, 13, 1, 5, 2 };
std::vector<int> v2 = v;
std::sort(std::begin(v2), std::end(v2));
      
auto first = std::begin(v2);
std::advance(first, 2);
auto last = first;
std::advance(last, std::size(v2) - 4);
 
v2.erase(last, std::end(v2));
v2.erase(std::begin(v2), first);

 
std::vector<int> v{ 21, 1, 3, 8, 13, 1, 5, 2 };
auto v2 = v |
          rs::copy |
          ra::sort |
          ra::slice(2, rs::end - 2);
