A.温度計


状態


ナビゲーターを使用して、多くの人が車のルートを構築するときに「温度計」を見ました。ルート上の道路の混雑を表す多色の直線です。このタスクでは、「温度計」のデータをさまざまな画面サイズに適合させる関数を作成することが提案されています。

関数入力には、温度計が表示される長さの色lengthと画面サイズwidth（length ≥ width）の配列が与えられます。色が使用されGREEN、YELLOWそしてREDそれぞれ、低、中、高ワークロードに対応しています。トラフィックの点で色は同等です:: GREEN < YELLOW < RED。

最初の配列は、最初の要素から始まり、長さの連続した互いに素なサブ配列に分割されますlength / width（数値は常に整数です）。各サブアレイでは、道路の混雑度の増加に応じて色を調整し、中央値の色を選択して、セット全体をそれと置き換える必要があります。偶数の長さの配列の場合、「より低い中央値」が選択されます（要素n/2の順序付けられた行に番号を持つn要素）。結果は色の長さの配列になりますwidth。

ソリューションはCommonJSモジュールの形式で提供する必要があります。

module.exports = function (segments, width) {
    // Your code here.
};

RE判定は、送信されたソリューションが正しくないことも意味します。

追加オプション
const segments = ['GREEN', 'GREEN', 'RED', 'GREEN', 'YELLOW', 'RED', 'GREEN', 'YELLOW', 'RED', 'YELLOW'];
const width = 5;
 ['GREEN', 'GREEN', 'YELLOW', 'GREEN', 'YELLOW']

決定


1.セグメントの初期配列をlongのセグメントに分割しlength / widthます。
2.各セグメントで、条件に基づいて中央値の色を選択し、見つかった色を結果の配列に追加します。

solution.js
module.exports = function solution(segments, width) {
    const blockSize = segments.length / width;

    const result = [];

    for (let i = 0; i < width; i++) {
        result.push(getMedian(segments.slice(i * blockSize, (i + 1) * blockSize)));
    }

    return result;
};

function getMedian(array) {
    const map = {
        GREEN: 1,
        YELLOW: 2,
        RED: 3
    };

    return array.sort((a, b) => map[a] - map[b])[Math.floor((array.length - 1) / 2)];
}

B.トレントクライアント


状態


あなたはあなたの急流のクライアントを書くことにしました。その特徴は、その助けを借りて、テキストのみを送信することが可能であることです。

トレントクライアントはほぼ準備ができています。最も重要なことは残ります。送信のために分割された部分からソーステキストを収集することです。

すべてのテキストが読み込まれてそれらからソースを収集するのを待つ関数を記述します。

この関数は、chunkCountおよびの2つのフィールドを持つ入力オブジェクトを受け取りemitter、指定された形式の文字列の形式でソーステキストまたはエラーを含むpromiseを返します。

chunkCount-テキストが分割されたピースの数。

各テキストには一意の識別子と送信時刻があります。送信時間が遅いピースは、テキストの先頭からより遠くに配置されます。

emitter-ダウンロードしたテキストを取得できるオブジェクト。テキストの一部が任意の時間遅延で到着する場合があります。ピースの順序は任意です。

ダウンロードが正常に完了する前に同じテキストが2回受信された場合、関数はエラーをスローする必要があります"Duplicate: <id>"（テキストのIDが所定の位置にある場合<id>）。

すべてのテキストを受け取ったら、それらを1行にまとめ、promiseを使用してこの行を返す必要があります。2つのピースの送信時間が同じ場合、返される文字列でのこれらのピースの順序は任意です。

転送が1秒以内に完了しない場合、関数はエラーをスローします"Timed out"。

入力はそのようなTypeScriptインターフェースに対応します
( TS.)

interface Input {
    chunkCount: number;
    emitter: Emitter;
}

interface Emitter {
    on: (callback: (chunk: Chunk) => void) => void;
}

interface Chunk {
    id: number;
    timestamp: Date;
    data: string;
}


ソリューションはCommonJSモジュールの形式で提供する必要があります。

module.exports = function ({chunkCount, emitter}) {
    //  Promise
};

決定


  • ロードされたピースをオブジェクトに保存しますchunk。
  • 同時に、存在を確認しidます。すでに存在する場合は、約束をキャンセルしてください。
  • すべてのピースをロードした後、それらを並べ替えて結合します。
  • これと並行して、タイムアウトを1秒に設定する必要があります。


solution.js
module.exports = function ({chunkCount, emitter: {on}}) {
    return new Promise((resolve, reject) => {
        const chunks = {};
        let chunksDownloaded = 0;
        on(({id, data, timestamp}) => {
            if (typeof chunks[id] !== 'undefined') {
                reject(`Duplicate: ${id}`);
            } else {
                chunks[id] = {data, timestamp};
                chunksDownloaded += 1;

                if (chunksDownloaded === chunkCount) {
                    const result = Object.values(chunks)
                        .sort((a, b) => a.timestamp - b.timestamp)
                        .map(({data}) => data)
                        .join('');
                    resolve(result);
                }
            }
        });
        setTimeout(() => {
            reject('Timed out');
        }, 1000);
    });
};

class BinaryTreeNode {
    /**
     * @param  {...Data} items
     * @returns {BinaryTreeNode}
     */
    static create(...items) {
        // e - .
        const root = new BinaryTreeNode(items[0]);
        //  return   .
        //  .slice(1),     .
        return items.slice(1).reduce((node, data) => node.insert(data), root);
    }

    /**
     * @param {Data} data
     */
    constructor(data) {
        /**
         * @type {Data}
         */
        this.data = data;
        //    .
        /**
         * @type {BinaryTreeNode | null}
         */
        this.left = null;
        /**
         * @type {BinaryTreeNode | null}
         */
        this.right = null;
    }

    /**
     *    .
     *    ,      .
     *
     * @param {Date} data
     * @returns {BinaryTreeNode}
     */
    insert(data) {
        //    .
        if (data < this.data) {
            if (this.left === null) {
                this.left = new BinaryTreeNode(data);
            } else {
                this.left.insert(data);
            }
        } else {
            if (this.right === null) {
                this.right = new BinaryTreeNode(data);
            } else {
                this.right.insert(data);
            }
        }

        //    ,   .
        return this;
    }

    /**
     *     .
     *   ,   .
     *
     * @param {Data} data
     * @returns {BinaryTreeNode | null}
     */
    remove(data) {
        //     {}.
        //    .
        if (data < this.data) {
            //     `this.left`.
            this.left = this.left && this.left.remove(data);
        } else if (data > this.data) {
            //     `this.right`.
            this.right = this.right && this.right.remove(data);
        } else {
            if (this.left === null && this.right === null) {
                return null;
            }

            if (this.left === null) {
                return this.right;
            } else if (this.right === null) {
                return this.left;
            }

            const aux = findMinNode(this.right);
            this.data = aux.data;

            this.right = this.right.remove(aux.data);
        }

        //    ,   .
        return this;
    }

    /**
     *     .
     *
     * @param {Data} data
     * @returns {BinaryTreeNode | null}
     */
    search(data) {
        //    .
        if (data < this.data) {
            //     `this.left`.
            return this.left && this.left.search(data);
        }
        if (data > this.data) {
            //     `this.right`.
            return this.right && this.right.search(data);
        }
        //  ,     ,    `null`.
        if (data === this.data) {
            return this;
        }
        return null;
    }

    /**
     *    ,           .
     *     .
     *
     * @param {Function} callback
     * @returns {BinaryTreeNode}
     */
    inorder(callback) {
        if (this.left !== null) {
            this.left.inorder(callback);
        }

        callback(this.data);

        if (this.right !== null) {
            this.right.inorder(callback);
        }

        //    ,   .
        return this;
    }

    /**
     *   ,           .
     *
     * @param {Function} callback
     * @returns {BinaryTreeNode}
     */
    preorder(callback) {
        callback(this.data);

        if (this.left !== null) {
            //       .
            this.left.preorder(callback);
        }

        if (this.right !== null) {
            this.right.preorder(callback);
        }

        //    ,   .
        return this;
    }

    /**
     *   ,            .
     *
     * @param {Function} callback
     * @returns {BinaryTreeNode}
     */
    postorder(callback) {
        if (this.left !== null) {
            this.left.postorder(callback);
        }

        if (this.right !== null) {
            //       .
            this.right.postorder(callback);
        }

        //   ,     .
        callback(this.data);

        return this;
    }
}

/**
 *   ,   .
 *
 * @param {BinaryTreeNode} node
 * @returns {BinaryTreeNode}
 */
function findMinNode(node) {
    //       .
    //    true  false.
    if (node.left === null) {
        return node;
    } else {
        return findMinNode(node.left);
    }
}

決定


可能な解決策の1つは、動的計画法です

。つまり、1。プチの義務の初日を担当します。
2.この日の最良の解決策を見つけるために、チケットごとに可能なオプションを計算します。そのような各オプションは、チケットのコストと、このチケットの有効期限後の勤務日に最高の決定のコストで構成されます。2番目の項も同様に計算され、再帰が得られます。
3.さらに、そのようなオプションがいくつかある場合は、チケットの数を考慮します。
4.当面の間は、キャッシングソリューションに特に注意を払う必要があります。

solution.js
module.exports = function (days, tickets) {
    if (days.length === 0 || tickets.length === 0) {
        return [];
    }

    tickets = tickets
        .map((ticket, idx) => ({
            ...ticket,
            idx
        }))
        .sort((a, b) => a.duration - b.duration);

    const daysSolutions = new Map();

    function getDaySolution(idx) {
        if (daysSolutions.has(idx)) {
            return daysSolutions.get(idx);
        }

        const solution = {
            totalCost: Number.POSITIVE_INFINITY,
            totalTickets: Number.POSITIVE_INFINITY,
            ticketToBuy: null,
            next: null
        };

        for (let i = 0, j = idx; i < tickets.length && j < days.length; i++) {
            while (j < days.length && days[j] < days[idx] + tickets[i].duration) {
                j++;
            }

            const nextDaySolution = j < days.length ? getDaySolution(j) : null;
            let totalCost = tickets[i].cost;
            let totalTickets = 1;

            if (nextDaySolution) {
                totalCost += nextDaySolution.totalCost;
                totalTickets += nextDaySolution.totalTickets;
            }

            if (
                totalCost < solution.totalCost ||
                (totalCost === solution.totalCost && totalTickets < solution.totalTickets)
            ) {
                solution.totalCost = totalCost;
                solution.totalTickets = totalTickets;
                solution.ticketToBuy = tickets[i].idx;
                solution.next = nextDaySolution;
            }
        }

        daysSolutions.set(idx, solution);

        return solution;
    }

    let solution = getDaySolution(0);
    const res = [];

    while (solution) {
        res.push(solution.ticketToBuy);
        solution = solution.next;
    }

    return res;
};
