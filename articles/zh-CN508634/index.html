一点简单的Javascript就可以

我从来没有做过专业的前端开发人员，尽管我为小型项目编写HTML / CSS / JS已有15年了，但是所有项目都很小。碰巧的是，我在这些项目之间已经多年没有用Java脚本编写了，而且经常不确定我是否做对了所有事情。



这就是为什么我经常使用库的部分原因！十年前，我使用jQuery，自2017年以来，我在小型项目中经常使用vue.js（您可以看到一个作为我对Vue的介绍而制作的小型sapper游戏）。



但是上周，很长时间以来，我第一次写了没有库的简单Javascript，而且很有趣，所以我想谈一谈！



试用简单的Javascript



我真的很喜欢Vue。但是上周，当我开始开发https://questions.wizardzines.com时，我的限制比平常略有不同-我想使用相同的HTML来创建PDF文件（使用Prince）并创建一个交互式版本问题。



我真的没有看到Vue怎么可能（因为Vue想要创建所有HTML本身），并且由于这是一个小项目，我决定尝试使用不带库的简单Javascript编写它-只需编写一些HTML / CSS并添加一个即可<script src="js/script.js"> </script>



。



我已经有一段时间没有这样做了，一路上我学到了一些东西，这些东西会使过程比我开始时想的要容易。



通过添加和删除CSS类来执行几乎所有操作



我决定通过简单地添加和删除CSS类并使用CSS过渡（如果我想为过渡设置动画）来实现几乎整个用户界面。



这是一个小示例，其中单击next



问题按钮将类添加done



到父div。 



div.querySelector('.next-question').onclick = function () {
    show_next_row();
    this.parentElement.parentElement.classList.add('done');
}


效果很好。和往常一样，我的CSS有点混乱，但是似乎可以管理。



使用.classList添加/删除CSS类



我开始进行类编辑，如下所示：x.className = 'new list of classes'



。尽管这有点脏，但我想知道是否有更好的方法。他是！



CSS类可以这样添加： 



let x = document.querySelector('div');
x.classList.add('hi');
x.classList.remove('hi');


element.classList.remove('hi')



-这比我以前的方法干净得多。



使用document.querySelectorAll查找项目



当我开始学习jQuery时，我记得我曾想过，如果您需要轻松地在DOM中找到某些内容，则需要使用jQuery（例如$('.class')



）。我刚发现这周您可以改写document.querySelectorAll('.some-class')



，然后您就不必依赖任何库了！



被介绍时我很好奇querySelectorAll



。我在Google上搜索了一下，看起来Selectors API是在2008年至2013年之间创建的-我发现jQuery作者在2008年讨论了拟议的实现，并在2011年发表了一篇博客文章，内容如下：当时所有主流浏览器中都有它，所以当我开始使用jQuery时它可能并不存在，但肯定存在了很长一段时间:)

使用.scrollIntoView滚动页面



我发现的最后一件有趣的事是这个.scrollIntoView



。当有人单击“下一个问题”按钮时，我想自动滚动到下一个问题。原来这只是一行代码： 



row.classList.add('revealed');
row.scrollIntoView({behavior: 'smooth', block: 'center'});
