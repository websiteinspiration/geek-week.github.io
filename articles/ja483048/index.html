ターミナルの楽しみ：10の古典的なコンソールジョーク

年末年始の長い休暇の半分が終わったので、今がUnix管理者のあごひげを生やした冗談を思い出す時です。 UNIXライクなオペレーティングシステムのテキストシェルは、内部コマンドの実行、さまざまなユーティリティの実行、およびスクリプトの作成に適しています。一部のプログラムの結果は、光の中をさまよっているゲストを楽しませることができます。RUVDSブログの編集委員会は、クリスマスに読者を祝福し、多くの世代のシステム管理者を楽しませている古典的なコンソールジョークを思い出すように提案しています。

apt-get install make
