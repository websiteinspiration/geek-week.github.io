1. SQL Server en Linux


SQL Server ahora se ejecuta en Linux. Este curso lo ayudará a obtener una comprensión básica de SQL Server en Linux. Luego aprenderá cómo ejecutar contenedores de SQL Server en Linux e implementar SQL Server en Linux. Después de eso, aprenderá cómo configurar automáticamente la implementación de SQL Server en Linux.

Puedes comenzar a entrenar aquí

2. Cree soluciones de análisis de código abierto con Azure HDInsight


En este curso, aprenderá sobre HDInsight y cómo aplicar esta tecnología para resolver una variedad de tareas del mundo real.

Entre los módulos del curso:

  • Introducción a la oferta de análisis de código abierto
  • Elija la configuración correcta de HDInsight para crear soluciones de análisis de código abierto
  • Crear y configurar un clúster de HDInsight
  • Realice análisis de cero ETL con HDInsight Interactive Query

Empezar a entrenar

3. Implementación de una solución de flujo de datos con Azure Streaming Analytics


Conozca los principios del procesamiento y la transmisión de eventos y vea cómo esto se aplica a Azure Stream Analytics. Luego configure su trabajo de Stream Analytics para la transmisión de datos y aprenda a monitorear y rastrear la ejecución del trabajo.

Empezar a entrenar

4. SQL Server Learning Nuevo


La última versión de SQL Server ofrece muchos beneficios para un mayor rendimiento y una funcionalidad mejorada, así como un soporte mejorado para la plataforma de base de datos. Este tutorial le muestra cómo hacer un inventario de las implementaciones de SQL Server con el Kit de herramientas de evaluación y planificación de Microsoft. Además, aprenderá cómo actualizar las bases de datos de SQL Server con el Asistente de migración de datos y optimizar su sistema actualizado con el Asistente de experimentación de bases de datos y el Asistente de ajuste de consultas.

Empezar a entrenar

5. Trabajar con datos NoSQL en Azure Cosmos DB


Los datos NoSQL son una forma eficiente de almacenar datos que no cumplen con los requisitos de una base de datos SQL relacional. Descubra cómo usar el portal de Azure, la extensión Azure Cosmos DB para Visual Studio Code y .NET Core SDK para Azure Cosmos DB para trabajar con datos NoSQL en cualquier ubicación, así como proporcionar a los usuarios un alto nivel de accesibilidad sin importar dónde se encuentren.

Empezar a entrenar

6. Azure para ingenieros de datos


Aprenda cómo está evolucionando el mundo de los datos y cómo la llegada de la tecnología en la nube ofrece nuevas oportunidades de negocio. Aprenderá sobre las diversas tecnologías de plataforma de datos disponibles y cómo un ingeniero de datos puede aprovechar estas tecnologías para el beneficio de la organización.

Entre los módulos:

  • El mundo en evolución de los datos.
  • Descripción general de los servicios de Azure Data Platform
  • Definir las tareas de un especialista en ingeniería de datos en una arquitectura alojada en la nube

Empezar a entrenar

7. Crear y usar informes analíticos en Power BI


Este curso le presentará Power BI y le enseñará cómo usar y crear informes de inteligencia empresarial.

Entre los módulos:

  • Introducción a Power BI
  • Recuperación de datos con Power BI Desktop
  • Modelado de datos en Power BI
  • Usar elementos visuales en Power BI
  • Explorando datos en Power BI
  • Publica y comparte en Power BI

Empezar a entrenar
