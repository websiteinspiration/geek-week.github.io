How to simplify the management of home devices and secure them (share ideas about Kauri Safe Smart Home)

We specialize in working with data - we develop and implement Internet of Things (IoT) solutions that work for all business sectors. But recently, we have switched our attention to a new product intended primarily for a ‚Äúsmart‚Äù home or office.

Now the average resident of a metropolis in the apartment has a Wi-Fi router, a set-top box from an Internet provider or a media player, a hub for IoT devices.

We thought that all these devices can not only be combined into one device, but also completely secure the home network. That is, this is a device that combines a router, smart firewall with an antivirus, Zigbee router (as an option - local data processing and decision making, including script execution). And, of course, it works with a mobile application for management and monitoring. A smart home setup is possible by those specialists of the provider. The device will work with Alice, so no one canceled home discos and games in the cities.



So, depending on the modification, the device can be:

a) Anti-Virus;
b) Wifi access point with antivirus;
c) Wifi / Zigbee access point with antivirus, optionally
managing UD;
d) Wifi / Zigbee / Ethernet router with antivirus, optionally
managing UD.

Unfortunately, completely secure IoT systems do not exist. Somehow, they are all vulnerable. According to Kaspersky, in the first half of 2019, hackers attacked Internet of things devices more than 100 million times - most often with the help of the Mirai and Nyadrop botnets. We understand that security is a headache for the user, which is why our Kauri Hub acts as an antivirus. It scans all traffic on the network for malicious activity. As soon as the device detects an anomaly, it blocks all attempts to access gadgets on the network from the outside. At the same time, anti-virus operation does not affect the speed of the Internet, but absolutely all connected devices will be protected.

Anticipating some objections:

- I myself can assemble it on a router with Zigbee USB and OpenWrt.

Yes, you geek. And if you want to tinker with it - why not? And
you will certainly write applications for the smartphone too. But are there many like you?

- Combines do not do well any function.

Not certainly in that way. Processing network protocols is conveniently combined in one device. Modern home routers already combine many features, we just add a few more.

- Zigbee is unsafe.

Yes, if you use the cheapest sensors with a key that is stitched by default. We suggest using the more secure Zigbee 3.0 standard. But sensors will be more expensive.
