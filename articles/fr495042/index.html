# 01 - Et un octet entier ne suffit pas ... | Qu'est-ce que l'intro?

Mesdames et messieurs, aujourd'hui est un grand jour!

Vous vous souvenez probablement qu'il existe une telle forme d'art informatique comme une «démoscène», mais si vous entendez ce mot pour la première fois, lisez simplement le centre thématique et les étiquettes sur le Habré, ainsi que quelques articles populaires sur ce sujet.


Une forme de concurrence dans la demi - scène est le codage de taille - la programmation d'effets visuels dans une quantité extrêmement limitée de code machine. Avant de passer aux réalisations d'aujourd'hui (et il y en a plus que des ambitieuses), passons rapidement en revue l'héritage des grands-pères. Tous les travaux ci-dessous sont effectués sur différentes années, leur exigence pour la plate-forme - PC, x86, DOS (pas de GPU, carte vidéo ordinaire) les unit. Idéalement, vous devriez mettre DosBox et exécuter chaque travail vous-même, car les auteurs ne sont pas particulièrement perplexes en créant une vidéo et en la téléchargeant sur YouTube. Nous avons essayé de choisir un travail d'exposition dans chaque nomination.

Voyons d'abord les "grandes" versions, celles qui occupent des kilo-octets. Ici, l'étalon-or est de 64 kilo-octets. La catégorie des 40 kilo-octets est populaire sur l' ordinateur Commodore Amiga , et dans le cas des PC, il y a généralement des œuvres que les auteurs ne voulaient pas ajouter à 64k. Parlant de 32k et 16k - il s'agit le plus souvent d'une intro cracktro et bbs, qui peut également être attribuée à la démoscène.

C'est là que s'arrêtent les graduations officielles à Pouet , mais la recherche d'effets dans le minimum de code ne s'arrête pas. Ici, à titre d'exemple, la «pluie de symboles dans la matrice» est de huit (8) octets:

8b intro - m8trix 8b



Question du jour: peut-il y avoir une intro de 4 octets? Et en 2 octets?
Réponse: OUI.

Voici le code machine du processeur x86:

0xEB 0x80

Ou dans une langue plus compréhensible:

; 2B or not 2B - main environment code
; (c) 2020 / ShakeSpirt / RMDA

; fasm 1

org $100

	jmp short $82		; jump to command line to launch the code :)
