NModbus（RTU）について

この記事は初心者を対象としています。

理論


Modbus は、マスタースレーブアーキテクチャに基づく通信プロトコルです。RS-485、RS-422、RS-232インターフェイス、およびイーサネットTCP / IPネットワークをデータ転送に使用します。

NModbusは、すべてのプロトコル操作モードの実装を含む大きなC＃ライブラリです。このライブラリのクラスを実装するためのモデルでは、任意のModbusデバイスを操作できますが、ライブラリクラス自体がポートをカプセル化し、複数のModbusオブジェクト間の同期を防止するため、デバイスは1つだけです。このプロトコルは、スマートホームやモノのインターネット向けにさまざまな周辺機器を開発するときに非常に人気があります。

簡単なプログラム例
using Modbus.Device;
using System;
using System.Collections.Generic;
using System.IO.Ports;
using System.Linq;
using System.Text;
using System.Threading.Tasks;

namespace Modbus
{
    class Program
    {
        static void Main(string[] args)
        {
            SerialPort serialPort = new SerialPort(); //Create a new SerialPort object.
            serialPort.PortName = "COM1";
            serialPort.BaudRate = 9600;
            serialPort.DataBits = 8;
            serialPort.Parity = Parity.None;
            serialPort.StopBits = StopBits.One;
            serialPort.Open();
            ModbusSerialMaster master = ModbusSerialMaster.CreateRtu(serialPort);

            byte slaveID = 1;
            ushort startAddress = 0;
            ushort numOfPoints = 1;
            ushort[] holding_register = master.ReadHoldingRegisters(slaveID, startAddress,
            numOfPoints);
            Console.WriteLine(holding_register);
            Console.ReadKey();
        }
    }
}
