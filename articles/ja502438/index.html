マテリアルPython。OpenGL効果のあるカスタムカード


Pythonでの挨拶、愛する愛好家、専門家！

この記事では、アプリケーションでKivyフレームワークやこのフレームワークのマテリアルデザインライブラリであるKivyMDなどのクロスプラットフォームツールを使用する場合に、カスタムカードにOpenGL効果を適用する方法を示します。行こう！

KivyMDは、標準コンポーネントがあるMDCard様々なカスタムカード（作成するための基本クラス- 材料設計仕様、カードを）。詳細に触れない場合は、MDCardの内部には通常のBoxLayoutがあります。これは、他のウィジェットを垂直方向または水平方向に配置できるコンテナです。つまり、ユーザーに関する情報など、ある種のカードを作成する必要がある場合は、自分で作成します。MDCardは、ripple_behavior、touch_behaviorおよびキャストシャドウのみを実装します。

ブランクカードを表示するプログラムの例は次のとおりです。

from kivy.lang import Builder

from kivymd.app import MDApp

KV = '''
Screen:  #  

    MDCard:  # 
        #    
        size_hint: .6, .5
        pos_hint: {"center_x": .5, "center_y": .5}
'''


class TestCard(MDApp):
    def build(self):
        return Builder.load_string(KV)

TestCard().run()

そしてもちろん、カードでぼかし効果や丸みを帯びたエッジが表示されないため、結果は予想されたものとは大きく異なります。ぼかし効果から始めましょう。Kivyには、その子にさまざまなグラフィック効果を適用できる標準のEffectWidgetウィジェットがあります。カスタムOpenGLシェーダーを使用してFboインスタンスをレンダリングすることで機能します。カードのメイン画像とシャドウ画像にぼかし効果を適用する必要があります。したがって、これらのコンポーネントをEffectWidgetウィジェットに配置する必要があります。

#:import effect kivy.uix.effectwidget.EffectWidget
#:import HorizontalBlurEffect kivy.uix.effectwidget.HorizontalBlurEffect

<RestaurantCard>

    ...

    RelativeLayout:
        
        #   ,      .
        EffectWidget:
            #  .
            effects: (HorizontalBlurEffect(size=root.blur),)

            FitImage:
                source: root.source

            FitImage:
                source: root.shadow
                size_hint_y: None
                height: "120dp"

    ...

ぼかし効果の度合いの値のフィールドを追加します。

class RestaurantCard(MDCard):
    ...
    blur = NumericProperty(8)

マウスオーバー（デスクトップの場合）またはタップ（モバイルの場合）しても何も起こりません。カードがon_focusイベントに応答するには、RestaurantCardルールのプロパティでこのイベントの読み取りを有効にし、このイベントが登録されたときに実行されるメソッドを割り当てる必要があります。

完全なサンプルコード 
from kivy.lang import Builder
from kivy.properties import StringProperty, NumericProperty

from kivymd.app import MDApp
from kivymd.uix.card import MDCard

KV = """
#:import Stencil kivymd.uix.graphics.Stencil
#:import Animation kivy.animation.Animation
#:import effect kivy.uix.effectwidget.EffectWidget
#:import HorizontalBlurEffect kivy.uix.effectwidget.HorizontalBlurEffect


<Effect@EffectWidget+Stencil>
    radius: [20,]


<RestaurantCard>
    md_bg_color: 0, 0, 0, 0
    elevation: 12
    focus_behavior: True
    on_enter: Animation(blur=0, d=0.3).start(self)
    on_leave: Animation(blur=8, d=0.3).start(self)
    radius: [20,]

    RelativeLayout:

        Effect:
            effects: (HorizontalBlurEffect(size=root.blur),)

            FitImage:
                source: root.source

            FitImage:
                source: root.shadow
                size_hint_y: None
                height: "120dp"

        MDLabel:
            text: root.text
            markup: True
            size_hint_y: None
            height: self.texture_size[1]
            x: "10dp"
            y: "10dp"
            theme_text_color: "Custom"
            text_color: 1, 1, 1, 1


FloatLayout:

    RestaurantCard:
        text: "[size=23][b]Restaurant[/b][/size]\\nTuborg Havnepark 15, Hellerup 2900 Denmark"
        shadow: "shadow-black.png"
        source: "restourant.jpg"
        pos_hint: {"center_x": .5, "center_y": .5}
        size_hint: .7, .5
"""


class RestaurantCard(MDCard):
    source = StringProperty()
    text = StringProperty()
    shadow = StringProperty()
    blur = NumericProperty(8)


class BlurCard(MDApp):
    def build(self):
        return Builder.load_string(KV)


BlurCard().run()
