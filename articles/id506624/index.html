Linux Foundation 2020: Linux, Cloud Engineer Bootcamp




Pengetahuan tentang GNU / Linux lebih banyak diminati saat ini daripada sebelumnya ketika bekerja dengan teknologi cloud, bahkan di Microsoft Azure GNU / Linux lebih populer daripada Windows. Yang paling penting adalah bagaimana dan di mana orang belajar bekerja dengan sistem bebas ini. Dan di sini secara alami hadir Linux Foundation. ZDNet menulis, The Linux Foundation adalah pelopor sertifikasi TI, menawarkan program sertifikasi jarak jauh pertamanya pada tahun 2014. Sebelum ini, hampir tidak mungkin untuk mendapatkan sertifikat IT di luar pusat pelatihan. Linux Foundation telah menetapkan prosedur pembelajaran jarak jauh yang andal dan terbukti. Pelatihan ini sangat disederhanakan dan sangat penting sekarang, pada masa pandemi, bagi para profesional yang ingin mendapatkan sertifikasi tanpa pergi ke mana pun.

humanID:





Linux.com berbicara tentang proyek baru yang dirancang untuk meningkatkan keamanan dan kenyamanan bekerja di Internet. Setiap hari, milyaran orang menggunakan akun sosial, seperti Facebook Login dan sejenisnya, untuk mengakses aplikasi melalui Internet. Kerugian utama dari sistem ini adalah ketidakmampuan untuk membedakan pengguna nyata dari bot, tulis publikasi. Organisasi nirlaba humanID, penerima Dana Dampak Sosial Universitas Harvard, datang dengan ide inovatif: untuk mengembangkan login anonim dengan satu klik, yang berfungsi sebagai alternatif untuk input sosial. "Dengan bantuan humanID, semua orang dapat menggunakan layanan tanpa meninggalkan privasi dan tidak menjual data mereka. Botnet secara otomatis dikecualikan, sementara aplikasi dapat dengan mudah memblokir penyusup dan troll, menciptakan komunitas digital yang lebih sipil , â€kata Bastian Purrer, salah satu pendiri humanID.
