*不在のため、フェニックスライブビュー

に関する記事の続き。今日は、テクノロジーを快適に使用するための基本構成を説明し、新しいアプリケーションを作成します。

elixirとphoenixをインストールする


elixirが必要です（インストールプロセスは参考です）。ただし、Windowsでは、DockerまたはVagrantを配置することをお勧めします。これらのテクノロジーについてはすでにご存知だと思います。データベースは無視します。別のプロジェクトでの使用を検討することをお勧めします。

elixirをインストールしたら、phoenixをインストールします（ホームリファレンスへのリンク）。実際、エリクサーの必要なバージョンを確認する必要があります

elixir -v

今日に関連して1.8.2を発行する必要があります。そして、フェニックス自体のインストールを開始します

mix archive.install hex phx_new 1.4.6

1.4.6は今日に関連しています。現在のバージョンはここで確認できます。プロジェクトはwebpackを使用します。フェニックスではそのまま使用できるため、nodejsが必要です。

IDE


IDEを選択することは頭痛の種ですが、私自身はルビーマインで開発することにしました。elixirと呼ばれる優れたプラグインがあります。「目を引き裂く」という配色は徐々に理解できるようになり、vscodeに戻りたくありませんが、対応するプラグインを介してエリクサーのサポートも非常に適切に実装されていることを言いたいので、コード愛好家にお勧めします。


プロジェクトの作成


elixirにプロジェクトを保存するための便利なフォルダーを作成し、コンソールに移動します。

cd ~/ElixirProjects/
mix help #   
mix help phx.new #          
mix phx.new fire --no-ecto #   fire          
Fetch and install dependencies? [Yn] #

コンパイルには時間がかかります

cd fire
mix phx.server

すべての依存関係が正常にインストールされ、[info] Access FireWeb.Endpoint at http://localhost:4000そのようなものが得られた場合は、大胆にブラウザhttp：// localhost：4000に移動します。ページを数回リロードしてターミナルに戻ります。そこで、興味深いログが見つかります。ここで、µsはマイクロ秒であり、FireWeb.PageController.indexこれは要求を処理した関数の名前です。IDEでコードを開いて続行します。
