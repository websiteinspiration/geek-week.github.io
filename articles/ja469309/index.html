サーバーへのオブジェクトの移動を監視するための簡単なサービスをインストールします

今日私は、追加の費用なしでタンバリンと踊って、小さな組織または私的使用のニーズのためにサーバーに車両監視システムをインストールする方法のトピックに触れたいです。サービスとして、無料版のP.O. "Autoradar"とtk102-103ファミリートラッカー用のソフトウェアゲートウェイを使用します。

まず、そのようなタスクを実装するには、仮想サーバーとコンピューターまたはLinuxまたはWindowsサーバーを備えたサーバーの両方を使用できます。

例としてUbuntu 18.4 OSを使用するDockerのインストールオプションを検討してください。サーバーマシンに最初にインストールする必要があるのは、Docker自体です。これを行うには、コンソールに移動します

$ sudo apt-get update
$ sudo apt-get install wget
$ wget -qO- https://get.docker.com/ | sh

次のステップはdocker-composeをインストールすること$ pip install docker-composeです。pipがインストールされていない場合は入力し、インストールする必要があります。システムにpython 3がインストールされている場合は、コマンドを使用してdocker-composeをインストールできます$ pip3 install docker-compose。

docker-composeをインストールしたら、メイン部分に進みます。これは、監視サービスのインストールと構成です。

$ mkdir mydirectoryコマンドで任意の名前のディレクトリを作成し、そこに移動します$ cd mydirectory。

アーカイブをダウンロードします$ wget https://github.com/gadson/Autoradar-docker-installer/releases/download/Autoradar/Autoradar.zip。

コマンド$ unzip Autoradar.zipで解凍し、unzipがインストールされていない場合は設定し $ apt-get install unzipます。

さらに、設定を変更したくない場合は、次の手順をスキップしてインストールに進むことができます。

1.セットアップ：

セットアッププロセスを簡略化するには、mcファイルマネージャーをインストールし、次のコマンドで$ sudo apt-get install mc

実行します$ mc

。mc コマンドを実行します。/Autoradar / GeoServer /ディレクトリに移動し、settings.pyファイルを編集します（mcで、目的のファイルを選択してF4キーを押します）

。データベースへのアクセス設定を変更します。これを行うには、データベース名-'NAME'、データベースにアクセスするためのログイン-'USER'、パスワード-'PASSWORD'、ポート-'PORT 'を変更できる行「DATABASES」を見つけます。

1.2。次に、ファイルの最後に移動して、次を見つけます。

1.4。重要！settings.pyでデータベースへのアクセスの設定を変更した場合は、ルート「mydirectory」にあるファイルを修正する必要があります。docker-compose.yml

その中の次の行を変更する必要があります。

  • POSTGRES_PORT = 5432-settings.pyのようにポートを指定します
  • POSTGRES_NAME = postgres-データベースの名前を指定します
  • POSTGRES_USER = postgres-データベースアクセスログイン
  • POSTGRES_PASSWORD = postgres-データベースアクセスパスワード

1.5。変更をファイルに保存します。

2.インストール：

重要！ポート5432（データベースポートを変更しなかった場合）、8888および8001が空いていて、他のアプリケーションまたはサービスによって占有されていないことを確認します。

コンソールから$ ./instsall.shと入力します。スクリプトが起動しない場合は、$ sudo chmod u + x ./install.shと入力して、前のコマンドを繰り返します。

インストールプロセス中に、インストーラーは管理者の登録を求め、管理者のログイン、電子メール、およびパスワードを2回入力します。パスワードには、数字と大文字と小文字の両方（レイアウト）を含める必要があります。

これでインストールは完了です。Webサービスはポート8001、データベースはポート5432（設定を変更しなかった場合）、およびGPS / GSMトラッカーを接続するためのゲートウェイはポート8888で機能します。複数のゲートウェイを接続できますが、これは別の記事のトピックです。Androidトラッカーは、APIによってゲートウェイなしで機能します。

Windows、Android、iPhone用の監視クライアントは、サービスのアカウントからのリンクを使用してダウンロードできます。

また、このバージョンの「Autoradar」は、Android用のソフトウェアGPSトラッカーをサポートしています。
