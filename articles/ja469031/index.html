Video Indexerはアニメーションと多言語コンテンツのサポートを導入します


昨年のIBCでは、受賞歴のあるAzure Media Services Video Indexerを一般に公開し  、今年はさらに改善されました。Video Indexerは、話し言葉、顔、感情、テーマ、ブランドなどのメディアファイルから情報とメタデータを自動的に抽出します。これを使用するために機械学習の専門家である必要はありません。

私たちの最新のオファーには、2つの非常に人気があり差別化された機能の予備バージョンが含まれています-アニメーションキャラクターの認識と多言語音声の文字起こしのほか、Video Indexerで現在利用可能な既存のモデルへのいくつかの追加。

複数の言語でのコンテンツの識別と文字起こし


ニュース、ニュースリール、インタビューなどの一部のメディアリソースには、さまざまな言語を話す人々の録音が含まれています。音声をテキストに翻訳するための既存のオプションのほとんどは、音声認識言語を事前に示す必要があるため、多言語ビデオの転記が困難になります。

さまざまなタイプのコンテンツの音声言語を自動的に識別するための新機能は、機械学習テクノロジーを使用して、メディアリソースにある言語を識別します。検出後、各言語セグメントは自動的に対応する言語の文字起こしプロセスを経て、すべてのセグメントがいくつかの言語で構成される1つの文字起こしファイルに結合されます。



結果の復号化は、JSONビデオインデクサーの出力の一部として、字幕付きのファイルの形式で利用できます。出力の復号化はAzure Searchにも統合されているため、ビデオ内のさまざまな言語セグメントをすぐに検索できます。さらに、ビデオインデクサーポータルでの作業時に多言語の文字起こしが利用できるので、時間とともに文字起こしと識別された言語を表示したり、各言語のビデオの特定の場所に移動して、ビデオの再生中に署名の形で多言語の文字起こしを確認したりできます。結果のテキストを、ポータルとAPIを介して54の使用可能な言語のいずれかに翻訳することもできます。ドキュメントで

私たちのテーマ検出モデルは、テーマが明示的に述べられていなくても、文字起こし、光学式文字認識（OCR）、発見された有名人に基づいてビデオの主題を識別します。これらの発見されたトピックを、Wikipedia、Bing、IPTC、IABの4つの分類領域と関連付けます。この拡張により、第2レベルのIPTC分類を含めることができます。

新しいライブストリーミング機能


Azure Media Servicesのプレビューバージョンでは、ライブストリーミング用の2つの新機能も提供しています。

AIリアルタイムの文字起こしは、ライブブロードキャストを次のレベルに引き上げます


リアルタイムストリーミングにAzure Media Servicesを使用すると、オーディオおよびビデオコンテンツに加えて、自動生成されたテキストトラックを含む出力ストリームを取得できるようになります。テキストは、人工知能に基づいてリアルタイムの音声を転記することによって作成されます。結果を改善するために、音声をテキストに変換する前後にカスタムメソッドが適用されます。テキストトラックは、DASH、HLS CMAF、HLS TSのいずれであるかに応じて、IMSC1、TTML、またはWebVTTにパッケージ化されています。

24時間年中無休のOTTチャネル用のリアルタイムリニアコーディング


API v3を使用すると、OTT（オーバーザトップ）テクノロジを使用してチャネルを作成し、それらを介してライブブロードキャストを管理および実施し、オンデマンドのライブビデオブロードキャスト（VOD、ビデオオンデマンド）、パッケージング、デジタル著作権管理（DRM）。
