Voting for another share from Yealink


I suggested Yealink to hold another campaign - to reduce the minimum package of YMS licenses for the purchase. But they still doubt the need for such measures.

The fact is that the YMS software server itself does not cost money, payment is only for competitive connections and additional options. At the same time, the server, as I already wrote, is a professional videoconferencing solution that has a large number of convenient and useful functionality. Added to all this is the lack of licensing for point-to-point calls through the YMS server. Such calls do not use a competitive license, but give the user the opportunity to use the entire functionality of the server, including a unified management system, address book, and short numbering of subscribers.

In order not to introduce the concept of the cost of the system core, Yealink introduced a minimum number of competitive YMS licenses available for purchase equal to 10. These licenses begin to be used when creating an event with the number of participants starting from three (Two-party calls, as I wrote above, can be any number - they are not charged).

Since during the crisis, the cost of a minimum package (10 licenses) can become unbearable for a number of enterprises, we initiated a reduction in this number to a minimum of 3 competitive connections.

Thus, for the price of only three YMS licenses, you can get a professional video conferencing system for use, with free unlimited p2p calls through the server, free software and mobile clients, WebRTC support, integration with IP PBX, AD and S4B.
