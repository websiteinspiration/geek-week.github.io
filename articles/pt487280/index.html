8 aplicativos web importantes para desenvolvedores

Convidamos você a se familiarizar com a tradução de um artigo de Jamie Bullock publicado em medium.com. O autor fala sobre quais aplicativos da web ele costuma usar em seu trabalho.


Foto de: Héctor Martínez . Retirado de Unsplash

1. GDB Online




O Online GDB é um compilador, intérprete e depurador de navegadores. Ele suporta 20 linguagens de programação, incluindo C, C ++, Python , Ruby , C #, Swift e JavaScript.

Acho esse aplicativo Web inestimável para verificar trechos sem precisar executar uma IDE ou sessão de terminal.

Além disso, é perfeito para melhorar as habilidades de programação ou aprender um novo idioma.

2. Validador de esquema JSON




Existem muitas ferramentas de validação JSON online , mas acho que o validador de esquema JSON é o melhor. Ele pode validar esquemas JSON, JSON e também validar JSON nos esquemas que o usuário define.

Ele também contém um grande número de esquemas predefinidos, em comparação com os quais a validação JSON pode ser executada. Não uso o validador de esquema JSON todos os dias, mas se tiver dificuldade em analisar o arquivo JSON, é indispensável para corrigir o problema.

3. Regex 101




O Regex 101 é uma ótima ferramenta para aprender expressões regulares . Ele permite inserir qualquer texto e encontrar correspondências nele com base em expressões regulares definidas pelo usuário.

Esse aplicativo Web não apenas localiza correspondências instantaneamente, mas também apresenta dinamicamente o resultado de ações de expressão regular, incluindo a detecção de erros se você digitou algo incorretamente.

4. Fonte impressionante




Font Awesome é um diretório on-line que contém mais de 1.500 ícones gratuitos, incluindo ícones de aplicativos padrão, ícones de empresas e outros símbolos úteis.

Você pode baixar os ícones como um conjunto de fontes OpenType ou no formato SVG. Em aplicativos da Web, eles podem ser usados ​​diretamente via JavaScript usando a tag icon.

Na minha opinião, o Font Awesome é especialmente útil ao criar protótipos: com ele, você pode inserir rapidamente ícones no aplicativo desenvolvido, mesmo antes da aprovação do design oficial.

5. Troca de Pilha




Se eu tivesse que escolher apenas um aplicativo Web crítico, escolheria o Stack Exchange .

Stack Exchange é uma rede de sites com perguntas e respostas sobre vários tópicos. Esse recurso é utilizado por mais de 11 milhões de pessoas, o que indica sua solidez. No site de programação Stack Overflow, as perguntas geralmente são respondidas alguns minutos após serem postadas.

Aprendi muito no site Stack Overflow , que me ajudou a resolver muitos problemas práticos.

Se você é um programador iniciante ou seu nível de conhecimento nesta área é médio, tente fazer uma pergunta na próxima vez que encontrar dificuldades. Se você é um especialista bastante avançado, poderesponda você mesmo a algumas perguntas.

6. GitHub




Parece-me que esta lista não estaria completa sem o GitHub ! GitHub é um serviço da web para acessar e gerenciar repositórios Git . Ele também inclui recursos como rastreamento de bugs, wiki, lançamentos de projetos e quadros Kanban de projetos.

As páginas do GitHub fornecem hospedagem para sites ou blogs básicos. Para mim, o GitHub é um recurso universal que contém tudo o que você precisa para gerenciar um projeto de software, documentá-lo e liberá-lo.

7. Tabela Unicode




A tabela Unicode é um enorme banco de dados de caracteres Unicode pesquisável. Inclui notas, setas, emojis, sinais de moeda, símbolos matemáticos, caracteres chineses e tudo o que você pode imaginar.

Para cada caractere, junto com os códigos Unicode, HTML e CSS, é fornecida uma versão de inserção, além de dados de codificação UTF completos . Observe que é necessária uma fonte compatível com Unicode para exibir esses caracteres no aplicativo.

8. Música para programação




Por fim, conheça um dos meus aplicativos da web favoritos - musicForProgramming ! Ele contém mais de 50 listas de reprodução com músicas projetadas especificamente para melhorar a concentração.

Há argumentos para o fato de que a música ajuda a se concentrar, e acredito que alguns estilos musicais são melhores para isso do que outros. Eu não uso este site todos os dias, mas me parece incrível! Às vezes, esse recurso é exatamente o que eu preciso para aumentar a concentração.

***

Espero que esta lista tenha sido útil para você. Se você conhece outros aplicativos da Web interessantes para desenvolvedores, seria ótimo ler sobre eles nos comentários deste artigo.
