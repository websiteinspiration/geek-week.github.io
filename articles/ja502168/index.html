HAProxyのセットアップ


HAProxyはMaxMindデータベースを直接操作できませんが、ネットワークリストに基づいてACLを構築できます。HAProxy構成での記述方法は次のとおりです。

frontend frontend-https
    bind *:80

# GeoIP ACL
    acl acl_SNG src -f /etc/haproxy/geoip/SNG.txt

ファイル/etc/haproxy/geoip/SNG.txtの内容：

193.8.167.0/24
193.16.111.0/24

...

つまり、SNG.txtファイルには、ACLに該当するネットワークのリストが含まれています。

MaxMindデータベースからのネットワークのリストを含むファイルの作成を自動化するタスクに直面しています。

私たちが解決したい問題を定式化しましょう。ロシア、ベラルーシ、ウクライナからのトラフィックは1つのバックエンドに行き、他の場所からのトラフィックは別のバックエンドに行く必要があります。この問題を解決するために、シェルスクリプトを作成します。スクリプトはいくつかの段階で構成されます。

まず、MaxMindデータベースの新しいコピーをcsv形式で取得します。

# Download geoip2 lite csv database
wget 
"https://download.maxmind.com/app/geoip_download?edition_id=GeoLite2-Country-CSV&license_key=${MAXMIND_LICENSE}&suffix=zip" -qO geoip2lite.zip 
&& unzip -j geoip2lite.zip

ここで$ {MAXMIND_LICENSE}は、MaxMind Webサイトで以前に書いたライセンス番号です。

コマンドを実行すると、アーカイブの内容は次のようになります。

国別のipv4ネットワークの所有権に関する情報を含むGeoLite2-Country-Blocks-IPv4.csvと、国名とデジタルコードの対応を含むGeoLite2-Country-Locations-en.csvの2つのファイルに関心があります。

したがって、ロシアに属するすべてのネットワークを取得する必要がある場合、最初にロシアのコードを取得します。

ここで2017370は検索コードです。

番号2017370が各行に2回出現することに注意してください。ファイル形式の説明を参照してください。

root@ash:/tmp/geo2lite# cat GeoLite2-Country-Blocks-IPv4.csv | head -n 1
network,geoname_id,registered_country_geoname_id,represented_country_geo
name_id,is_anonymous_proxy,is_satellite_provider

ここで、registered_country_geoname_idはネットワークが登録されている国のコードであり、presented_country_geoname_idはネットワークがルーティングされる国のコードです。 

ネットワークがロシアなどに登録されていて、ルーティングがオランダからのものである可能性があります。このような状況で何をすべきかを決定するため、私は要求された国から登録およびルーティングされたネットワークのみを選択しました。

したがって、GeoLite2-Country-Blocks-IPv4.csvファイルから必要なネットワークを選択すると、HAProxy ACLのファイルを作成できます。

HAProxy自体のACLでの作業：

frontend frontend
# GeoIP ACL
    acl acl_SNG src -f /etc/haproxy/geoip/SNG.txt

    use_backend backend1 if acl_SNG
    default_backend backend2

したがって、ソースIPアドレスがSNG.txtファイルに記述されているネットワークからのものである場合、リクエストはbackend1に送信されます。それ以外の場合-backend2に送信されます。

操作を自動化するために、スクリプトが作成されました。

root@ash:~# cat haproxy_geoip_list.sh
#!/bin/bash
#-----------------------------------------------------------------------------------------------------------------------
# Variables
TMPDIR="/tmp/geo2lite"
MAXMIND_LICENSE="xxxxxxxxxxxxx"
LOCATIONS="Russia Ukraine Belarus"
HAPROXY_PATH="/etc/haproxy/geoip"
HAPROXY_FILE="SNG.txt"
#-----------------------------------------------------------------------------------------------------------------------
mkdir -p ${TMPDIR}
pushd ${TMPDIR}
# Download geoip2 lite csv database
wget 
"https://download.maxmind.com/app/geoip_download?edition_id=GeoLite2-Country-CSV&license_key=${MAXMIND_LICENSE}&suffix=zip" -qO geoip2lite.zip 
&& unzip -j geoip2lite.zip
if [ $? -gt 0 ]; then
 logger -t "haproxy_geoip" "Download not success."
 exit 1
fi

exit
for COUNTRY in ${LOCATIONS}; do
 echo "# ${COUNTRY}"
 COUNTRY_CODE=`cat GeoLite2-Country-Locations-en.csv | grep -i 
"${COUNTRY}" | awk -F "," '{print $1}'`
 grep ",${COUNTRY_CODE}," GeoLite2-Country-Blocks-IPv4.csv | awk -F "," 
'{print $1}'
done > ${HAPROXY_FILE}

# Check file
if [ ! -s ${HAPROXY_FILE} ]; then
    logger -t "haproxy_geoip" "File is not exist"
    exit 1
fi

if [ -s ${HAPROXY_PATH}/${HAPROXY_FILE} ]; then
    MD5_FILE=`md5sum ${HAPROXY_PATH}/${HAPROXY_FILE} | awk '{print $1}'`
fi

MD5_TMPFILE=`md5sum ${HAPROXY_FILE} | awk '{print $1}'`

if [ "${MD5_TMPFILE}" != "${MD5_FILE}" ]; then
 logger -t "haproxy_geoip" "File ${HAPROXY_FILE} is changed"
 cp ${HAPROXY_PATH}/${HAPROXY_FILE} 
${HAPROXY_PATH}/${HAPROXY_FILE}.backup
 mv ${HAPROXY_FILE} ${HAPROXY_PATH}/${HAPROXY_FILE}
 # Test new haprpoxy cfg
 /usr/sbin/haproxy -c -f /etc/haproxy/haproxy.cfg
 if [ $? -gt 0 ]; then
  logger -t "haproxy_geoip" "New file is corrupted, replace by older one from backup."
  mv ${HAPROXY_PATH}/${HAPROXY_FILE}.backup 
${HAPROXY_PATH}/${HAPROXY_FILE}
 fi
fi

popd
rm -rf "${TMPDIR}"

始める前に、必要に応じて変数セクションに記入することを忘れないでください。
