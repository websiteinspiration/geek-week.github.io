人気のOSでどのような音楽が「結ばれた」か

状況：2003年の中庭で。あなたの家族の誰かが電話で話している、何がモデムが接続されている回線を占めているのか。あなたは自分の順番とオンラインになる機会を見越して、開発者がWindowsに置いた無料の音楽のライブラリを掘り下げることをもう一度決定します。当時、彼女はSamples of Musicフォルダにいました。

今日、Windowsのさまざまなバージョンで聴くことができるものを思い出すことにしました。


写真この男 Unspalsh

Windows 3.1


音楽が登場し、90年代に導入され、Windows 3.1の中で。この頃、最初のMIDI対応サウンドカードが市場に登場しました。また、ソフトウェアはこのフォーマットで動作するように見え始めています。たとえば、Steinberg CubaseやFinaleなどです。

Windows 3.1には、C：\ WINDOWS \ MEDIAフォルダーにある9つのMIDI曲が含まれていました。それらはすべてArchive.orgで聞くことができます。

2つのトラックは、バッハとヴィヴァルディのクラシック作品からの抜粋です-「ブランデンブルクコンサート」と「季節」。さらに2つの作品で、Passport Designs MIDIシーケンサーの機能（PASSPORT.MIDおよびCANYON.MID）が実証されました。

最初の作者は有名ではありませんが、2番目の作曲家はジャズミュージシャンのジョージストーンでした。しかし、彼は音楽がWindowsとともに配信されるとは想定していませんでした（MACでシーケンサーを宣伝するために書いたものです）。 Passport Designsは財政難を経験し、Stoneの知識なしにMicrosoftの構成に対する権利を売却しました。ジョージはこの取引からお金を受け取っていませんが、彼の仕事は広く認められていました。

YouTubeにはたくさんのカバーがありますCANYON.MID。作曲は、コンピューターゲームとその中の音楽に特化したMAGFestフェスティバルの1つでさえ行われました。


Windows 95


ここでは、CLOUDS.MIDが前の9つのトラックに追加されました。 「イースターエッグ」と考えられていたため、MEDIAディレクトリにはありませんでした。楽曲を聴くには、フォルダを使っていくつかの操作を行う必要がありました。手順は、当時MicrosoftのインターンであったCLOUDS作曲家のBrian Orrのブログにあります。

新しいMIDIコンポジションに加えて、2つのミュージックビデオが Windows 95に登場しました。イベントは、オペレーティングシステムに付属するVideo For Windowsの無料バージョンのリリースに合わせて行われました。最初のビデオは、アメリカの歌手Edie Brickellによる "Good Times"のクリップです。

2番目- クリップウィーザーのアルバムの「バディ・ホリー」に。コメディーHappy Daysのシーンとして様式化されたこのクリップは、Best Alternative Clipのタイトルを含む4つの MTV アワードを受賞しました。今日、バディホリーは90年代の最も有名な作曲の1つと見なされており、カバーはまだ作られています。

Windows 98


Windows 98では、すべてのMIDIトラックが3つの新しいファイルに置き換えられました。マイクロソフトの代表によると、彼らに関与した楽器はほとんどなかった。

これらの3つの曲は、マルチメディア拡張機能の問題を診断するためのツールとして、マイクロソフトのテクニカルサポートスペシャリストによってしばしば使用されました。彼らの協力を得て、サウンドドライバーの操作性をチェックしました。


新しい曲はFLOURISH.MID、TOWN.MID、ONESTOP.MIDと呼ばれていました。最初の2つのトラックの作者はNathan Grieg（Nathan Grigg）の名を冠し、今日はFEARシリーズと中つ国のゲームのサウンドトラックで知られています。3番目のトラックはDavid Yackleyによって書かれたもので、Microsoftの最悪のMIDI音楽ファイルの1つであるという名声がそこに詰まっています。

Windows MEおよびXP


Windows ME は ITコミュニティでの評判は良くありませんが、このOSはアメリカのマルチインストゥルメンタリストのベックヘンセンによって「美しい道」というトラックで知られています。この作曲は、批評家とミュージシャンファンの両方から高く評価されたMidnite Vulturesアルバムの一部です。 新しいトラックは、ストリーミングオーディオとビデオを含むASFマルチメディアフォーマットの機能を示しました。通常、これらのファイルの拡張子は.asfですが、コンテナに音楽のみが含まれている場合は、.wmaとして指定されていました。作曲がWindows XPで提示されたのはこの形式でした。写真Simon Goldin CC BY-SA /写真「Serenity」Windows XP、2006年に撮影





 Windows XPは、Windows MEのわずか1年後に登場し、有名なサンプルミュージックディレクトリを含む最初のオペレーティングシステムになりました。これは、ロックバンドの最新アルバムからの組成物を含んでトーキング・ヘッズ、だけでなく、スケルツォベートーベンの交響曲第九番からと「と呼ばれる光のジャズナンバーハイウェイブルースマークシルスで」。

Windows Vista


Windows Vistaでは、システムに付属する音楽トラックの数は11に達しました。その中には、イタリアのパーカッションデュオ、アイシャデュオの作品が含まれていました。セレクションには、アルバム「Quiet Songs」から2 曲が含まれています。サンプルフォルダーには、ブラジルのジャズシンガー、Luciana Souzaの音楽「Mutia Bobiera」と、ニューヨークのピアニストAaron Goldbergの「MAO's Blues」の曲も含まれていました。

セネガルのギタリスト、ハビブコイテがデビューアルバムから2曲を提供その中で彼は西洋のギター音楽とマリの人々の伝統音楽を組み合わせました。また、Vistaでは、アジアのアンダーグラウンドダンスジャンルのパイオニアと見なされているアメリカのパーカッショニストのKurs Kaleの作品に出くわす可能性があります。Bandcampで彼の楽曲を聴くことができます。


The Posiesの曲で民族音楽とジャズを薄めた。バンドは8年ぶりに彼らの最初のアルバムをリリースしました。おそらく、チームは新しいオペレーティングシステムの出力を思い出させる機会として使用したかったのでしょう。

Windows 7以降


Windows 7はクロード・ドビュッシーの構成「を導入亜麻仁髪の少女」、「仕事のアウェイ睡眠をボブアッコとによって」電子軌道から氏 イギリスのダウンテンポアーティスト、スクラフ。
