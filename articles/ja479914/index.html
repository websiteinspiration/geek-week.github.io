オブジェクト注入


可能であれば、外部オブジェクトにリンクを埋め込むことをお勧めします。外部オブジェクトを変更すると、使用されたすべてのムービープロジェクト（プレゼンテーション）内ですぐに変更されます。それ以外の場合は、すべてのビデオを確認して、それぞれの内部のすべてを変更する必要があります。


たとえば、背景画像が変更され、すべてのプレゼンテーションで自動的に変更されます。または、たとえばラップトップの写真を使用すると、製造元のロゴが消去されていないことに気付きました。合計-元の画像のロゴを削除します。すべてのプレゼンテーションで、変更はすぐに反映されます。

これは、オーディオ、ビデオ、画像、アイコンに適用されます。特に以降のビデオ 少なくとも実装すると、コピーされてPPTファイルが巨大になります。別の PowerPoint ファイルから要素

埋め込まれたビデオ


ビデオは非常に簡単に挿入できます-PowerPointインターフェースからすべてが直感的です。しかし、バグが発見されました。スライド間のアニメーション化された遷移中に、ビデオフレームのピクセル単位のサイズが16の倍数でない場合、ビデオの端に色付きのストライプが表示されます。動画は幅と高さの倍数である必要があります。たとえば、HD（1280x720）およびフルHD（1920x1080）サイズの場合、この条件は満たされます。

サイズが変更されたために新しく挿入されたビデオを編集する必要がないように、すぐに注意することをお勧めします。そして、バグは気づきにくい-ビデオにエクスポートされたときにのみ再生される-スライドショーの通常の表示中には観察されない。

しかし、マイクロソフトが好きなのは、COMを介してオフィスアプリケーションを自動化する機能です。PowerPointには、アイテムの表示を制御したり、ビデオをエクスポートしたりするためのインターフェイスがあります。

ローカリゼーションの切り替え


PHPのコードの例を示します。COMをサポートする他の言語で実行できます。
まず、プレゼンテーションを開きます。

$oPptApp = new \COM( 'PowerPoint.Application' );
$oPptFile = $oPptApp -> Presentations -> Open( 'my.ppt' );

次に、プレゼンテーションのすべての要素を調べ、言語に依存するものを見つけ、現時点で不要なものを隠す必要があります。たとえば、ロシア語（ru）が必要だとします。

foreach( $oPptFile -> Slides as $oSlide )
{
    foreach( $oSlide -> Shapes as $oShape )
    {
        $posLangSep = strrpos( $oShape -> Name, '.' );
        if( $posLangSep !== false )
            $oShape -> Visible = substr( $oShape -> Name, $posLangSep + 1 ) == 'ru' ? msoTrue : msoFalse;
    }
}

プレゼンテーションの要素はグループ化できるため、コードを少し再構築してこれを検討してください。ニュアンス-Visibleメソッドは、インターフェイスでのマウスクリックとは異なり、ネストされたすべての要素を即座に管理します。したがって、プロパティを「親」から「子供」に変更します。

foreach( $oPptFile -> Slides as $oSlide )
    foreach( $oSlide -> Shapes as $oShape )
        EnsureElemVisibleByLang( $oShape, 'ru' );

function EnsureElemVisibleByLang( $oShape, $langCode )
{
    $posLangSep = strrpos( $oShape -> Name, '.' );
    if( $posLangSep !== false )
        $oShape -> Visible = substr( $oShape -> Name, $posLangSep + 1 ) == $langCode ? msoTrue : msoFalse;

    if( $oShape -> Type == msoGroup )
        foreach( $oShape -> GroupItems as $oShapeChild )
            EnsureElemVisibleByLang( $oShapeChild, $langCode );
}
