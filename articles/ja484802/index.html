自己破壊モード：光による内視鏡カプセルの分解



胃腸管（消化管）の最も一般的で効果的な診断方法の1つは、内視鏡検査です。患者は手技に来て、（通常は常にではありませんが）バレルの上に横になり、良い医者は自然な方法で体内に内視鏡プローブを導入します。このプロセスでは、患者にとって非常に正確ではありません。ただし、この方法では、消化管内の特定の組織の損傷や疾患の症状を特定できます。

1997年、Gaby IddanとPaul Swainは新しいタイプの内視鏡検査を開発しました-カプセルは、患者が数時間で数万枚の写真を撮るカメラで「錠剤」を飲み込んだときです。しかしながら、人体に異物を導入する手順は常に一定のリスクを伴います。仕事を終えた使い捨てのカプセルは自然に体から排泄されますが、離れることを決定すると事件も発生します。そのような悲惨な状況では、それを削除するために特別な操作を実行する必要があります。より正確には、MIT（米国マサチューセッツ工科大学）の科学者が光にさらされると壊れる新しいタイプのカプセルを開発したため、以前はそうでした。新しいデバイスのベースとなった素材自己破壊モードはどのように正確にアクティブ化され、次に何が起こりますか？これについては科学者の報告から学びます。行け。

研究の基礎


内視鏡検査用の新しいタイプのカプセルの基礎は、特別な材料-ヒドロゲルになりました。この物質の生体適合性と吸収特性は、生体組織の特性に似ており、医療で使用されるデバイスの理想的な建築材料になります。ヒドロゲルベースのプローブの開発は長い間続けられており、それから生じるデバイスは、それらの意図された目的および自己破壊を活性化する方法の両方において多様です。現在、ヒドロゲルプローブを制御する最も一般的な方法は、温度、電磁波、pH、および化学薬品です。これらすべての方法は、プローブ材料の特定の反応を引き起こし、それらの活性化または自己破壊を引き起こします（ただし、スパイや秘密の実験室に関する映画のような大きな「広がり」はありません）。

しかし、それぞれの制御方法には欠点があると研究者たちは述べています。そのため、熱制御はデバイスの周囲の組織に悪影響を与える可能性があり、デバイス自体が放出する電磁波は、より古典的な診断（MRIなど）に干渉する可能性があります。プローブは、デバイスのかなり狭いpH範囲に適合するように身体の特定の領域に導入されている場合にのみ、pHで制御できます。ただし、化学的制御では、プローブと化学試薬を直接接触させる必要があるため、この複雑な手順を正しく実行しないと、周囲の組織に悪影響を与える可能性があります。

つまり、さまざまな方法がありますが、どれも多少悪いか不完全です。研究者によると、自己分解が可能なプローブは、生体適合性、非接触活性化/非活性化、空間制御、溶解度、および動的送達（侵襲的方法なし）の重要な機能を組み合わせる必要があります。

このフレームワークでは、光プローブは他のものよりもうまく適合します。このようなデバイスは体のどの部分でも使用できますが、他の方法（pH、化学的または熱的）は動的環境により健康に悪影響を及ぼす可能性があるため、科学者は特に胃腸管に関心を持っています。カプセルの分解を活性化する開発された光トリガーは、それが置かれている環境に実際には影響しないため、患者の健康に悪影響を与えることはありません。



ハイドロゲルは異なるため、すべて特定のトリガー（外部刺激）に対して異なる応答をします。科学者たちは、アクリル化オルトニトロベンジル（oNB）を使用して重合したポリエチレングリコール（PEG、C 2n H 4n + 2 O n + 1）でできたヒ​​ドロゲルが、青色光（365-405 nm）に曝されたときに活性化できることを発見しました。そのようなヒドロゲルからのプローブの崩壊生成物は完全に生体適合性があります。消化管の組織に安全です。

研究成果


まず、ヒドロゲルを作成するプロセスを検討する価値があります。モノマー鎖の両端にあるアクリレート官能基は、結合を容易に形成し、ラジカル重合によって結合された3次元ネットワークを形成します。


Image No. 1

したがって、容易に切断可能なoNBフラグメントを含み、アクリレート官能基によって制限されるポリマーは、アクリレート結合に基づく任意のポリマーネットワークの光学的に敏感なリンカー（バインダー）として機能できます（1A）光によって引き起こされるリンカーの分解は、ネットワーク自体の破壊の制御されたプロセスにつながります。さらに、oNBリンカーは、さまざまな比率でアクリルリンカーと混合できます。この機能により、3次元ポリマーネットワークの減衰の程度を部分的から完全に制御できます。

oNBリンカーの合成のベース材料は、PEG 4-（3-（1-アクリロイルオキシエチル）-4-ニトロフェノキシ）ブタノエート（1B）でした。

適用されたヒドロゲル合成技術は、単一および二重ネットワークの両方でゲルの作成を可能にします。二重ゲルは、共有結合したポリマーの相互侵入ネットワークで構成されています。このオプションは、伸びと靭性の向上を特徴としています。以前の研究では、ポリ（アクリルアミド）とポリ（2-アクリルアミド-2-メチルプロパンスルホン酸）の相互侵入ネットワークの機械的特性が向上していることが示されています（2A）。


画像番号2：PAAM-PAAと略されるポリ（アクリルアミド）; PAMPS-ポリ（2-アクリルアミド-2-メチルプロパンスルホン酸）、略してH-AMS。

このタイプのヒドロゲル（H-AMS / PAA）は、高度な生体適合性を持っています。人工軟骨の製造材料としても提供されました。さらに、この材料は300〜800 nmの波長範囲で透明であるため、光トリガーを備えたプローブを作成するための理想的な候補になります。

H-AMS / PAAゲルの剛性と拡張性は、主鎖を形成するモノマー鎖の長さと濃度を変更することによって、およびプレポリマー溶液中のリンカーの濃度を変更することによって制御できます。

単一および二重ネットワークによるヒドロゲルの比較分析は、oNB-H-AMS / PAAゲル（二重ネットワーク）がoNB-PAAゲル（単一ネットワーク）よりも12倍強いことを示しました（2B）圧縮のゲルをテストするとき、同様の傾向が観察されます。 MBAAアクリレートリンカー（N、N'-メチレンビスアクリルアミド）を介して結合されたN-AMS / PAAゲルは149±49 kPaで分解し始め、oNBリンカーによって結合されたN-AMS / PAAゲルは40.8±3.2 kPaで破壊されます。したがって、リンカーで形成されたoNBは、MBAAで形成されたハイドロゲルよりも剛性が低くなります。これは、感光性リンカーの鎖長の増加が原因である可能性があります。

人の最大胃圧の範囲は0.01〜0.013 MPaであるため、圧迫について覚えておくことが重要です。開発したoNBの圧縮モジュールは、このデータよりも優れていることがわかりました。

デュアルネットワークゲルの細胞毒性は正常範囲内でした（2C）これを確認するために、腸細胞HT29とCaco-2を使用し、これらをゲルとインキュベートした後、細胞生存率分析を行いました。

2つ目は、ヒドロゲルに続いて、研究の重要な部分は光であり、その分解を引き起こします。分解の程度と速度は、さまざまなパラメーターを調整することによって制御および変更できます：光強度、波長、組成、およびゲル内の感光性oNBリンカーの分布。

これらの各パラメーターがゲルの機械的特性に及ぼす影響を確認するために、光の強度と光源からの距離を制御するための特別な装置が作成されました。インストールのプラットフォームには、365または405 nmの波長の光を放出する1〜5個のLEDを取り付けることができました。 1、3、または5個の365 nm LED（3A）からの光強度を測定するために必要なデジタル露出計をサンプルプラットフォームに配置しました。

光の強度と距離は、理論的には逆二次依存性を持つはずであり、実際に確認されています。

光への曝露に応じたoNBヒドロゲルの動的分解は、レオロジーテストを使用して監視されました。

PAA-oNBゲルに3つのLEDのマトリックス（365 nm）を45分間照射し、0、15、30、45分後に剪断弾性率（G）を測定しました（3B）。ゲル（距離0 mm）の真上にある光源の出力は、分解開始から15分後に52％、30分後に28％、45分後に25％に減少しました。

光源からの距離が5、10、15、および30 mmに増加すると、45分のゲル分解後、せん断弾性率は初期値の32、37、50、および95％に減少しました。これは、光の強度が0 mmから0.9 mW / cm 2の距離でほぼ11.4 mW / cm 2から低下するため、劣化の程度がゲルが経験する光の強度に大きく依存することを示しています。30mmの距離で。

思い出すと、ヒドロゲル内の感光性リンカーのパーセンテージがゲルの分解の程度を決定します。完全な溶解は、oNBリンカーのみを使用して達成できますが、部分的な溶解は、oNBが他のアクリル化リンカーと混合されたときに発生します。

照射後0および45分のPAA-oNBゲル剪断弾性率の測定は、50、75、および90％のoNBリンカー含有量のゲルで行われました（3C）。これらのゲルバリアントのそれぞれで、残りは、3Dヒドロゲルネットワークの形成に必要なMBAAリンカーで占められていました（それぞれ50、25、および10％）。予想通り、ハイドロゲル内の光切断可能なリンカーの割合が増加すると、11.4 mW / cmで365 nmの波長の光を45分間照射した後、ゲルの機械的強度が大幅に低下しました。２。分解したゲルのせん断弾性率は、それぞれ50、75、90％のoNBリンカーを含むゲルの初期の分解前の値の85、52、25％に低下しました。

MBAAリンカーはoNBリンカー分子よりも短くて硬いため、ハイドロゲル内のoNBリンカーの割合が増加すると、分解前の状態のハイドロゲルの機械的強度が低下します（2B）。したがって、光感度と最大機械的強度の間にはトレードオフがあります。

物質の光に対する応答を変えるもう1つのパラメータは、この光の波長です。 oNBリンカーは、365〜405 nmの波長範囲の青色光に敏感です。実験により、そのようなゲルは高周波放射線に曝された場合によりよく壊れることがわかっています。

11.4 mW / cm 2で405 nmの波長で45分間照射した後、ゲルのせん断弾性率は元の値の73％に低下し、365 nmでは25％に低下します（3D）。照射されるゲル光の波長に対するこの感度と、露光時間を制御する能力を考えると、設計段階で定められた要件に従って分解するゲルを得ることが可能です。

私たちが思い出すように、oNBリンカーの細胞毒性は、分解前の材料の状態ですでに評価されています（完全な生体適合性が確立されました）。光による分解後の細胞毒性の分析は、分解副産物も細胞適合性であることを示した（3E）。

研究者は、現時点では、患者の消化管内に一定時間置かれるべき治療と診断のための多くの装置があることを思い出させます。そのようなデバイスの分解は、それらを構成する材料の特性に基づいています。多くの場合、そのようなデバイスの取り外しには外科的介入が必要です。印象的な例は、肥満（胃内）バルーンです。これは、患者の胃の中に配置され、食物摂取を減らします（肥満の外科的治療）。これらのシリンダーは、内視鏡手術でのみ取り外すことができます。別の例は、ステント-中空の臓器への損傷に配置され、疾患によって狭められた領域を拡大する、円筒形の中空の足場です。これらのデバイスも、外科的介入によってのみ削除されます。

彼らの研究では、科学者たちはこれらの2つのデバイス（シリンダーとステント）を変更することを提案しました。

ヒドロゲルベースの自己分解肥満バルーンプロジェクト


体温、胃の酸性度、湿度、細菌数、酵素要素など、いくつかの要因が同時に存在するため、このデバイスはかなり困難な状況で動作する必要があるため、ハイドロゲルからバルーンを作成することには多くの問題があります。さらに、デバイスは、最初は食道を容易に通過できるほど小さく、胃に到達したら必要な機能サイズに拡張する必要があります。これにより、外科的介入の必要がなくなります。化学的および物理的効果に加えて、誇張して言えば機械的効果もあります。胃内のデバイスは、3 Nの力での蠕動運動にもかかわらず、適切に機能するはずです。

分解の活性化後、バルーンのサイズが小さくなり始めるため、幽門（幽門、実際には胃と十二指腸を隔てる括約筋）を通過するのに十分なほど小さくなります（4A）。

科学者たちは、濡れるとすぐに膨張する物質で満たされた弾性多孔質高分子シェルからバルーンを作成しました。キャップピンの形状に成形されたoNBヒドロゲルは、シリンダーの開口端を通して絡み合い、それを密封して、膨張するフィラーがシリンダーのポリマーシェルから逃げるのを防ぎます（4B）。材料として、4 M PAAと0.1 mol％oNBからなる強力で十分に弾性のあるヒドロゲル組成物を選択しました。


胃の中で作成されたハイドロゲルバルーンの拡張のデモンストレーション。

人工胃液でのバルーンの予備テストにより、ポリマーシェルの多孔度やフィラー材料（ヒドロゲル）の組成など、多くのパラメーターを最適化することができました。したがって、最適な設計パラメーターが達成されました：シリンダーの膨張速度、膨張比、および形状保持時間。ラテックス膜には、ポリアクリル酸ナトリウム（150 mg）とPolySnow（1350 mg）で満たされた、サイズが300μmの一連の細孔が含まれていました。

設計されたバルーンは、元の容量に比べてサイズが22倍になり71 mlになります。このボリュームの平均余命は24時間でした。

次に、科学者たちは、バルーンが胃の圧縮ぜん動力にどのように反応するかを確認することを決定しました。これを行うために、膨張後24時間、デバイスに周期的な圧力（10 N）を加えるテストを行いました。観察では、シェルやハイドロゲルへの機械的な損傷は明らかになりませんでした。これは、フィラーの漏れがないことで確認されました。

次に、ヒドロゲル棒の中心点に、11.4 mW / cm 2で365 nmの波長の光を照射しました。30分間、その後、周期的な圧力を使用して再確認しました。光で弱くなったハイドロゲルは3Nの力に耐えられず、フィラーが漏れてしまいました。したがって、そのようなデバイスは胃の環境で完全に機能し、必要に応じて、その取り外しで光信号をアクティブにするだけで十分です。これにより、ヒドロゲルが弱くなります。

胃腸管の解剖学的パラメーターは人間のものと非常に類似しているため、65〜85 kgのヨークシャー豚が参加して、実際の状態でのテストが行​​われました。

食道を通してシリンダーを挿入した。内視鏡検査とX線（4C）装置は胃の中で完全に機能し、いかなる損傷もブタの体への悪影響もありませんでした。次に、変更された内視鏡を使用して、光がアクティブになり（3つのLED、365 nm、11.4 mW / cm 2）、次に分解プロセスがアクティブになりました（4D）。 LEDをオンにするために内視鏡を導入する必要があるため、この起動方法は侵襲的です。しかし、2番目の非侵襲的方法は、磁石付きのバッテリーのLEDによって作成されました。磁石は食道を通して導入されると、バルーンに接続し、その分解を活性化します（4E）。


胃内のLEDとバルーンの内視鏡的ドッキング。

LEDとバルーンを正常に結合した後、ライトを30分ではなく70分間オンにして、一定のエネルギー密度（25.2 J / cm 2）を維持しながら、11.4から5.19 mW / cm 2への光強度の減少を確認しました。 バルーンとLEDが胃液に浮いたため、LEDのドッキングは、胃の中に入れてから数分以内に起こりました。胃の蠕動により、デバイスが互いに十分に接近し、LEDのバルーンへの磁気引力が作動しました。分解の活性化後（6時間後）、繰り返しX線（4F）が実行され、バルーンサイズの大幅な縮小が示されました。



内視鏡と自律型LEDによって分解が引き起こされたシリンダーは、分解の活性化の1時間後に体積がそれぞれ初期の68％と70％に減少しました（4G）。

研究者は、内視鏡による露出または磁気LEDを介して実際にバルーンをアクティブにする方法は、1つの選択肢に過ぎないと述べています。将来的には、同じ内視鏡を必要としない、より便利なメソッドを作成できます。このインジケーターに影響を与える変数はいくつかあるため、LEDとバルーンの接続に必要な時間を検討することも必要です。

ハイドロゲルベースの自己分解性ステントプロジェクト


ヒドロゲルに基づいて、潜在的な構造的支持および/または薬物の局所送達のために、食道に設置するためのステントが作成されました。単純な円筒リング（4I）は、ポリ（e-カプロラクトン）ビーズ（PCL）を含むoNBヒドロゲルから作成されました。 PCLは、薬物の放出制御（4H）に広く使用されているポリマーで、デバイスの材料には4 M H-AMSと4 mol％のoNBリンカーが含まれていました。

リング上のPCLインサートは、硫酸バリウムの混合物で特別に染色されて放射線不透過性になり、豚の食道内のX線で見えるようになりました（4J）。

実際の観察の前に、作成されたステントは食道の壁の蠕動振動に耐えることができることがわかりました。分解が活性化した後、ステントの蠕動圧縮抵抗がその初期値（4K）の25％に低下したことがわかりました。その後、食道の蠕動により、光によって活性化されたステントが完全に破壊され、食道から完全に除去されました（4L）。そのようなデバイスは固定された場所にあり、光の分解が活性化されるまで、蠕動にうまく抵抗してその機能を実行することになります。
