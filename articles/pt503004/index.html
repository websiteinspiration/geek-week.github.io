9 principais anúncios da conferência Microsoft Build 2020

Nossa conferência de dois dias, Microsoft Build 2020, cuja transmissão principal foi realizada em Habré , chegou ao fim. Neste artigo, falaremos sobre os anúncios mais ambiciosos e interessantes do evento passado. Olhe embaixo do gato!

E de qual novidade você mais gostou? Compartilhe nos comentários.



Versão do Windows Terminal 1.0


Temos o prazer de anunciar o lançamento do Windows Terminal 1.0! O Windows Terminal percorreu um longo caminho desde o seu anúncio no Microsoft Build 2019 . Como sempre, você pode baixar o Windows Terminal na Microsoft Store ou na página de lançamento no GitHub . O Windows Terminal terá atualizações mensais a partir de julho de 2020.

Dissemos com mais detalhes sobre o novo terminal ontem neste artigo sobre Habr.



Criou um dos 5 supercomputadores mais poderosos do mundo


O novo supercomputador da Microsoft foi  desenvolvido  com base na nuvem do Azure em  colaboração  e exclusivamente para a  OpenAI , uma organização de pesquisa fundada por Elon Musk e Sam Altman. Sua capacidade é de mais de 285.000 núcleos de processador (CPU core), 10.000 processadores gráficos (GPUs) e a velocidade de conexão de rede é de 400 gigabits por segundo para cada servidor. Ele foi projetado especificamente para treinar a próxima geração de modelos de IA distribuídos em massa que podem executar uma ampla gama de funções e, consequentemente, exigir significativamente mais recursos de computação.

Leia os detalhes em nosso site de notícias (em russo).



Visualizar o Gerenciador de Pacotes para Windows


O gerenciador de pacotes foi projetado para ajudar você a economizar tempo. De fato, é um conjunto de ferramentas de software que ajudam a automatizar o processo de download de software para o seu computador. Você indica quais aplicativos deseja instalar, e ele procura a versão mais recente (ou a que você especificou) e a instala no computador.

Quase todo desenvolvedor queria ter um gerenciador de pacotes nativo no Windows. Este dia finalmente chegou. Uma das novidades interessantes é que é de código aberto. Agora é muito simples, por exemplo, terminal de instalação de winget, depois instalação de winget, powershell e depois de instalação de winget powertoys.

Leia mais sobre o Windows Package Manager em nosso blog .



Atualizações do Fluid Framework


A Plataforma Interativa de Colaboração na Web do Fluid Framework,  introduzida  há um ano no Build 2019, está  se tornando  um software de código aberto. Além disso, em breve a funcionalidade do Fluid Framework estará parcialmente disponível não apenas para desenvolvedores, mas também para usuários finais: seus componentes e ambientes de trabalho aparecerão no Office.com e na versão web do Outlook.

As primeiras integrações do Fluid Framework no Microsoft 365 permitirão que vários usuários façam alterações nos documentos simultaneamente e criem componentes conectados que podem ser compartilhados instantânea e facilmente nos aplicativos. Por exemplo, tabelas, gráficos e listas de tarefas podem ser coladas na versão web do Outlook para que seus dados de vendas, tarefas de projeto e relatórios de pesquisa sejam atualizados constantemente para refletir as alterações mais recentes. No Office.com, você pode criar e gerenciar os espaços de trabalho do Fluid Framework, inclusive no fluxo de atividades do seu documento, na lista de recomendações, em @ menções ou pesquisá-los no Office.com. E como os componentes do Fluid Framework são de baixo peso, as edições são aplicadas instantaneamente.

Um ambiente de rede Fluid pode transformar instantaneamente seu aplicativo em uma ferramenta de colaboração. Inclui estruturas de dados de sincronização de baixa latência e um serviço de comutação para conectar pontos de extremidade. Se você substituir estruturas de dados estáticas por estruturas de dados Fluid, seu aplicativo começará instantaneamente a dar suporte à colaboração em tempo real.

É possível liberar todo o potencial do Fluid Framework ao criar uma comunidade multicultural, aberta e ativa de desenvolvedores. Por esse motivo, a Microsoft transformará o Fluid Framework em um ambiente de código aberto, permitindo que os desenvolvedores aproveitem o Fluid Framework principal em seus próprios aplicativos. Lançamos documentação e ferramentas adicionais e também convidamos desenvolvedores a trabalhar com a Microsoft na criação e no lançamento do Fluid Framework.

Microsoft Edge: aprimoramentos para desenvolvedores


O Microsoft Edge apresenta uma série de novos recursos para melhorar os fluxos de trabalho para consumidores, desenvolvedores e clientes comerciais.

Primeiro de tudo, esta é uma maneira mais rápida e contextual de pesquisar usando a barra lateral. Basta selecionar uma palavra ou frase, clique com o botão direito do mouse e selecione “pesquisar na barra lateral” no menu de contexto. Os resultados serão exibidos no painel no lado direito da janela do navegador, para que o usuário não perca o contexto do conteúdo original.

Para os desenvolvedores, contribuímos com o projeto Chromium, incluindo melhorias na acessibilidade, escrita manual, rolagem e localização. As versões de visualização do Webview 2.0 também estão disponíveis para desenvolvimento com .NET e UWP. E, graças aos novos aprimoramentos do Progressive Web App (PWA) (na visualização), esses aplicativos parecem mais nativos, com uma interface de usuário mais natural para compartilhamento, notificações e muito mais.



Iniciativas educacionais para desenvolvedores


Nós introduziu  novos módulos de formação gratuito para o Microsoft Saiba plataforma, que permitirá aos desenvolvedores obter formação em trabalhar com a computação quântica usando a linguagem de programação #Q eo Kit de Desenvolvimento de Quantum. Também  lançaremos  o programa diário Learn TV para desenvolvedores, que incluirá programação em tempo real e várias discussões com especialistas.



Lançamento do Project Reunion - Uma nova maneira de desenvolver aplicativos Windows


Os usuários gastam um total de mais de 4 trilhões de minutos por mês no Windows 10 - 75% a mais do que no ano passado. Apresentando o Project Reunion para ajudar os desenvolvedores a criar ainda mais inovação para 1 bilhão de dispositivos Windows 10 ativos por mês. A iniciativa visa fornecer fácil integração entre as interfaces de programação Win32 e Universal Windows Platform, fornecendo uma única plataforma de desenvolvimento com suporte à compatibilidade com versões anteriores.
