DJI Osmo Mobile 3の12の革新



DJI Osmo Mobile 2は、多くのブロガーや初心者のビデオグラファーにとって必要なデバイスになりました。バッグや特別なケースなしで運ぶのはかなり問題があるという事実にもかかわらず、Osmo Mobile 2の機能は、輸送中のわずかな不快さを常に上回っていました。DJIはモデルの欠如を考慮し、バグに取り組み、どこへでも持ち歩ける新しいデバイスを導入しました。Osmoのラインナップには、年に2度目となる優れたガジェット、つまりスマートフォン専用に作成された真にモバイルなOsmo Mobile 3スタビライザーが補充されました。

第三世代のイノベーション




はい、折りたたみスタビライザーのアイデアはまったく新しいものではありませんが、これまでのところDJIだけが最高レベルでそれを実現しています。このモデルでは、すべてが文字通り細部まで考えられています。同社のエンジニアは、Osmo Mobile 2のユーザーの要望や要望をすべて聞き取り、新しいデバイスに組み込んだ。

これは、折りたたみデザインだけでなく、スタンバイモードでマウントからスマートフォンを取り外さないようにする機能、スタビライザーボタンを使用してモバイルアプリケーションのメニューをナビゲートする機能、さまざまな機能を実行する追加のトリガーボタン、スマートフォンの水平位置から垂直位置への切り替え、充電にも関係します記録中のOsmo Mobile 3の電話およびその他の便利な機能については、後で説明します。

そしてもちろん、DJIはモバイルアプリケーションの統合の道を元気に進んでいます。数か月前に会社の最後のヘリコプターの1人、スマートフォンのスタビライザー、たとえばOsmoアクションアクションカメラの所有者になれた運が良ければ、携帯にDJI GO、DJI GO 4、DJI Mimoの3つの異なるアプリケーションをインストールする必要があります。Osmo Mobile 3は、DJI Mimoアプリケーション（偶然、最近Russifiedになりました）での作業をサポートしており、DJI GOは徐々に古いデバイス用のプログラムになりつつあります。

Osmo Mobile 3の主な機能を簡単に見てみましょう。

  • 見開きのサイズ285×125×103 mm;
  • 折りたたんだ寸法157×130×46 mm;
  • バッテリー容量2450 mAh;
  • 作業時間15時間。

回転角度の機械的範囲：

  • 回転：-162.5°から+ 170.3°;
  • -85.1°から+ 252.2°まで回転;
  • -104.5°から235.7°に傾斜;
  • 最大制御速度120°/秒。

Osmo Mobile 3の主な新機能




1番。折りたたみ可能なデザイン

第3世代のOsmo Mobileを際立たせるもの。目新しさでは、以前のモデルの3セクションの代わりに、L字型の3軸スタビライザーが取り付けられました。さらに、DJIは彼らのアイデアをさらに進めました-Osmo Mobile 3は単に合計するだけでなく、安全な輸送のためにすべての軸を修正します。仕組み：スタビライザーを組み立てます。小さな突起がヒンジの対応する凹部に落ちます。それがすべてです。デバイスがバッグの中でねじれて、敏感なモーターが壊れることを心配する必要はありません。

2番。追加のトリガーボタン

ハンドルの裏側にあるこの便利な要素により、快適にビデオを撮影できます。たとえば、3回押すと前面カメラがアクティブになり、スマートフォンを2回中央に配置します。 2回押し続けるとスポーツモードがアクティブになり、押し続けると、スタビライザーを握る角度に関係なく、スマートフォンが1つの位置にロックされます。また、ActiveTrack 3.0トラッキングモードを開始または停止するには、トリガーを1回引きます。

ナンバー3。スタンバイモード

最も便利なイノベーションの1つ。ビデオを編集したり、映像を見たりするために、マウントからスマートフォンを取り出す必要はありません。スマートフォンを固定した状態でOsmo Mobile 3を折りたたんでMボタンを3回押すだけで、スマートフォンを使ってすぐにすぐに撮影に戻ることができます。

ナンバー4。新しいコントロールとナビゲーション



メカニカルボタンに加えて、シューティングを制御するために、ジェスチャーを使用できます。開いた手のひらまたは2本の指を文字Vの形で表示すると、カメラがビデオの録画を開始するか、写真を撮ります。ちなみに、これはメインカメラとフロントカメラの両方で動作します。これで、ジョイスティックとスタビライザーボタンを使用してアプリケーションメニューをナビゲートできます。手袋をしたまま撮影する場合や、秒針が忙しい場合に便利です。

5番ActiveTrack 3.0

オブジェクトトラッキングモードの新しいバージョンがOsmo Mobile 3に追加されました。ActiveTrack3.0は、非常に迅速に、非常にスムーズかつ正確に機能するようになりました。以前は自分で追跡オブジェクトを選択する必要があった場合、新しいバージョンでは自動化がこれを担当します。人間の顔を認識し、フレームの中央に保持します。ソフトウェアアルゴリズムを信頼しませんか？次に、前のように関数を使用して、画面上のオブジェクトを強調表示します。ちなみに、ActiveTrack 3.0は、ビデオ撮影モードだけでなく、写真の撮影だけでなく、HyperlapseクリップやSlowMoクリップの作成中にも機能するようになりました。

6番。 Mボタンを使用してスマートフォンの位置を変更する

Mボタンを2回押すと、スマートフォンが90度回転します。これで、横長モードから縦長モードへの移行に数秒しかかかりません。録音中でも動作します！

ナンバー7。 Bluetooth 5.0

Osmo Mobile 3は、Bluetooth 5.0ワイヤレス規格の最新バージョンをサポートしています。これは、スタビライザーが以前の2倍の速度で動作し、消費電力が少なく、データを2倍の速度で送信することを意味します。

8番。 USB Type-Cポート



DJIが現在の傾向をたどるもう1つの兆候。最近のほとんどすべてのデバイスは徐々にこの標準に移行しており、約2年後には古いmicroUSBが残っているガジェットはなくなります。そのため、ワイヤーコレクションの更新を検討する価値があります。

9番。新しいスタビライザー-新しいムーブメント

Osmo Mobile 3スタビライザーの形状とデザインにより、カメラのさまざまな動きの範囲が広がります。フレームの中心を基準にしてさまざまな方向に回転したり、ねじったりすることができます。新しい取り付け方法は、ケース内の大型の電話やスマートフォンに対応しています。

10番。スポーツモード

クリエイティブな撮影機会のリストOsmo Mobile 3は、更新されたスポーツモードを補充しました。その助けを借りて、1つのダイナミックシーンから別のシーンへの移行でさえ、ビデオ上でスムーズで映画のようになります。

11番。軽量

すべての革新とサイズの増加にもかかわらず、Osmo Mobile 3は小型化され、405グラムとOsmo Mobile 2バージョンの485グラムを比較しています。軽量化-長時間の撮影：非常に長いクリップを録画しても、ステディカムを維持するのに飽きません。

12番。 DJI Mimoのサポート



すでに述べたように、DJIは新しいデバイスを制御するためにDJI GOアプリの使用を徐々に停止します。 DJI Mimoは、さまざまな芸術的効果と創造的な実装の機会に取って代わっています。 Osmo Mobile 3も例外ではなく、このソフトウェアも使用しています。どうして？少なくとも3つの理由があります。

「ストーリー」モード。このセクションには、クリエイティブなクリップを撮影するための多くのテンプレートが含まれています。各モードでは、特別なスタビライザーの動き、ビデオのムードに対応するメロディー、さまざまなフィルター、シーン間のトランジションがすでに選択されています。「ストーリー」を撮ることは楽しみです。各テンプレートはいくつかの部分に分かれています。たとえば、ビデオの真ん中など、撮影方法が気に入らない場合は、もう一度記録するか、完全に削除してください。次に、プログラムはすべての作品を独立して1つの美しい映画に接着します。

創造力を最大限に発揮し、事前定義されたスタイルに限定されませんか？便利なエディタを使用すると、複数のビデオまたは写真から美しい映画を接着し、キャプション、ステッカー、音楽、フィルタを追加できます。一般的に、ソーシャルネットワークから競合他社との差別化を図るためのすべてのものです。

結論




レビューの最後の部分では、Osmo Mobile 3の欠点について実際に話したかったのですが、不満は何もないことがわかりました。もちろん、スタビシストを少し批判することもできます。これは、スタビライザーの新しい設計により、カメラの角度が非常に小さく、「フラッシュライト」操作モードがないためです。しかし、DJIはスタビライザーを右または左に90度回転させる必要がある新しいサイドグリップモードでこの欠点を回避し、水平回転軸を完全に解放しました。デバイスの残りの部分は、幅広い機能と最高レベルの品質を備え、可能な限り便利であることがわかりました。そして、ロシア市場での価格は私たちに非常に快適な価格であると約束されました。だから、最初の配達を待っています！
