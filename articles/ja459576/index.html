プログラマーにとって便利なGoogle Chrome拡張機能

プログラマーの作業は非常に複雑で多様であるため、さまざまなツールの助けがなければ実行できません。便利で時間を節約できる便利なGoogle Chrome拡張機能のリストを34種類用意しました。



1. Octotree

GitHubが好きで多くのコードを読んでいる場合、GitHubファイルをオンラインで閲覧するのがいかに不便かをご存知でしょう。このプロセスは、Octotree拡張機能を使用して簡略化できます。ナビゲーションが簡単なツールバーを提供し、プライベートリポジトリもサポートします。

2. WellDoneCodeによるPerfectPixel

この拡張機能により、開発者はHTMLの上に半透明のレイヤーで画像をオーバーレイし、ピクセルごとの比較を行うことができます。

3. GitHubリポジトリのサイズ

単一のリポジトリのサイズとその中の各ファイルのサイズの両方を表示する便利でシンプルなプラグイン。

4. 洗練されたGithub

拡張機能は、最近公開されたブランチを表示し、新しい方法でリクエストを操作できるようにし、コメントに応答したユーザーのアバターを表示します。

5. Code Cola

CSSスタイルの豊富なビジュアル編集機能を提供するプラグイン。
この拡張機能を使用すると、基本的なCSS3スタイルをサイトで直接編集できます。

6. Chromeデベロッパーツールですばやく作業できるDomFlags

拡張機能。
コードをすばやくナビゲートするためのリンクのメニューを作成します。

7. JSON Formatter

Extensionは、JSONの読みやすさを簡素化します。構文の強調表示、インデント、クリック可能なリンクがあります。

8. EditThisCookie

EditThisCookieはCookieマネージャーです。それを使用して、Cookieを追加、削除、変更、検索、保護、およびブロック できます。9

. Redux DevTools

Reduxを使用してアプリケーションをデバッグするためのツール。

10. Chrome向けiMacros

特定のアクションを繰り返して Webページのテストを構成する必要がある場合、このアプリケーションがまさに必要です。

11. github npm stats

プラグインは、NPM Webサイトでのパッケージダウンロードの統計を表示します。

12. ログファイルを処理するRailsアプリケーションを開発するためのRailsPanel

拡張。 Railsアプリケーションのすべてのリクエスト情報にコンソールから直接アクセスします。

13. Xdebugヘルパー

Xdebug

14 のデバッグ、プロファイリング、およびトレースの単純な拡張機能。解像度テスト

解像度解像度テスト-さまざまな解像度と画面サイズでWebアプリケーションをテストするときに役立ちます。リストから共通の解像度を選択するか、必要なサイズを手動で入力できます。この拡張機能は、ブラウザーのサイズを変更し、目的の画面解像度でアプリケーションをエミュレートします。

15. Webサイトで使用される色とアセットを表示するためのCSS Peeper

Extension：イラスト、テキストドキュメント、レイアウトファイル、ビデオファイル。

16. Android SDK検索この

拡張機能は、開発者の公式WebサイトでAndroid SDKをすばやく見つけるのに役立ちます。

17. WhatRuns

WhatRunsでは、WordPressやその他のGoogleアナリティクスなどの資料を含め、サイトを機能させるすべてのフレームワークとツールを確認できます。

18. Wordpress Site Managerこの

拡張機能は、サイトの承認に必要なすべての情報を1か所で収集したい場合に役立ちます。 WordPress Site Managerは情報を保存し、ブラウザーから直接サイトのメインページへのアクセスを提供します。

19. BuiltWith Technology Profiler

拡張機能により、Webサイトがどのように設計されたかを判断できます。

20. サイトがさまざまなデバイスでどのように表示されるかを確認するためのUser-Agent Switcher

Extension。

21. Stylebot

StylebotはChromeブラウザの拡張機能で、ブラウザのページの表示を好みに応じてカスタマイズできます。そして、これは簡単で便利な方法で行われ、変更が瞬時に表示されるため、HTMLやCSSの知識がなくても誰でも対応できます。

22. PHP Ninja Manual

Extensionを使用すると、ブラウザからPHP 5.5のドキュメントにすぐにアクセスできます。
すべての機能を覚えておくことは困難です。Googleで検索に何時間も費やしていた場合、この拡張機能はあなたの生活を楽にします。

23. Restletクライアント-REST APIテスト

この拡張機能でAPIテストを自動化します。

24. CSSシャック

拡張機能を使用して、デザインを作成し、サイトで使用するためにCSSファイルにエクスポートできます。レイヤーをサポートし、標準のフォトエディターで使用できる多くのツールが含まれています。

25. Webメーカー

この拡張機能は、Web実験のための便利で自己完結型のプラットフォームを提供します。
HTML / CSS / JSで実験または練習したい開発者に最適です。インターネットに接続していなくても動作します。

26. npmhub

package.jsonファイルがあるgithub.com上のリポジトリを表示すると、この拡張機能はパッケージ内の依存関係を分析し、READMEファイルのすぐ下にリンクと説明を表示します。

27. Augury

プラグインは、AngularJS

28のデバッグおよびプロファイリングツールを追加することにより、開発ツールを 拡張します。JetBrainsIDEを使用してHTML / CSS / JavaScriptを編集し、JavaScriptをデバッグするためのJetBrains IDEサポート

拡張機能。

29. PHPコンソールこの

拡張機能は、PHPエラーとダンプをGoogle Chromeコンソールと通知ポップアップに表示します。

30. 高度なRESTクライアント

この拡張機能を使用すると、フォームデータに入力して、POST、PUT、GET、DELETEリクエストを実行し、RestFulサービスにリクエストを送信し、サーバーからの応答を監視できます。

31. DevToolsの自動保存

プラグインを使用すると、Chrome Dev環境のツールを使用してCSSまたはJSの変更をソースファイルに自動的に保存できます。

32. JSONドキュメントをチェックおよび表示するためのJSONView

プラグイン。 JSONの可読性を高めるために、フォーマット、構文の強調表示、その他の設備を自動的に実行できます。

33. Isometric Contributions

Extensionを使用すると、GitHubの通常の貢献度チャートとバージョンの等尺性ピクセルアートを切り替えることができます。

34. OctoLinker

拡張機能は、コード要素（include、require、importなど）をクリック可能なリンクに変換することにより、迅速なコードナビゲーションを提供します。このリンクをクリックすると、ソースを見つけることができます。プログラミング言語に応じて、拡張機能はソースファイルを表示するか、ユーザーを外部サイトに移動します。
