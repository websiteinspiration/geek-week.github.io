9 Windows Terminal-Tipps von Scott Hanselman

Hallo Habr! Möglicherweise haben Sie gehört, dass das neue Windows-Terminal bald verfügbar sein wird. Wir schrieben schon darüber hier . Unser Kollege Scott Hanselman hat einige Tipps für die Arbeit mit dem neuen Terminal vorbereitet. Jetzt beitreten!



Sie haben also Windows Terminal heruntergeladen und ... was nun?


Zuerst können Sie nicht begeistert sein. Dies ist immer noch das Terminal, und es wird Sie nicht führen, die Hand zu halten.

1) Lesen Sie die Benutzerdokumentation für Windows Terminal.

2) Die Einstellungen werden im JSON-Format ausgedrückt . Sie werden es besser machen, wenn Ihr JSON-Datei-Editor so etwas wie Visual Studio Code ist und JSON sowie Intellisense unterstützt.

  • Überprüfen Sie Ihre Standardeinstellungen! Aus Gründen der Übersichtlichkeit präsentiere ich meine profile.json (was in keiner Weise ideal ist). Ich habe requestTheme, alwaysShowTabs und defaultProfile installiert.

3) Entscheiden Sie sich für die Tastaturkürzel. Windows Terminal verfügt über zahlreiche Anpassungsoptionen .

  • Jede Taste, die Sie drücken, kann neu zugewiesen werden.

4) Passt das Design Ihren Wünschen?

  • Auf terminalsplash.com gibt es mehrere Terminal- Themen.
  • Und hier sind Dutzende von iTerm2-Farbschemata , deren Farbschemata hier deutlich zu sehen sind .

5) Willst du zum nächsten Level gehen? Erkunden Sie Hintergrundbilder.

  • Sie können Hintergrundbilder oder sogar GIFs festlegen. Weitere Details hier .

6) Geben Sie Ihr Startverzeichnis an.
  • Wenn Sie WSL verwenden, möchten Sie möglicherweise früher oder später, dass sich Ihr Home-Verzeichnis im Linux-Dateisystem befindet .

7) Sie können weiterhin Far, GitBash, Cygwin oder cmder verwenden, wenn Sie möchten. Details in der Dokumentation .

8) Überprüfen Sie die Windows Terminal-Befehlszeilenargumente.

  • Sie wissen vielleicht, dass Sie Windows Terminal mit "wt.exe" starten können, aber jetzt können Sie Befehlszeilenargumente verwenden! Hier sind einige Beispiele:

    wt ; split-pane -p "Windows PowerShell" ; split-pane -H wsl.exe
wt -d .
wt -d c:\github

    An diesem Punkt können Sie so weit gehen, wie Sie möchten. Erstellen Sie andere Symbole, heften Sie sie an die Taskleiste an und gehen Sie in vollen Zügen davon. Überprüfen Sie auch Unterbefehle wie New-Tab, Split-Pane und Focus-Tab.

9) Ich habe ein Video aufgenommen , das einer Person zeigt, die an Mac und Linux gewöhnt ist, wie man Windows Terminal in Verbindung mit WSL (Windows Subsystem für Linux) konfiguriert. Vielleicht ist es für Sie interessant.

Bitte teilen Sie unten Ihre Tipps, Profile und Lieblingsthemen für das Terminal!
