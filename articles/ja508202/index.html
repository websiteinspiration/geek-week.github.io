原子シミュレーション用のASEライブラリ：ナノ粒子の作成

ASEライブラリ



ASE ライブラリは、原子操作と計算のためのPythonライブラリです。この記事では、このライブラリを使用してナノ粒子を作成します。



取り付け



ASEは通常、pipを介してインストールされます：pip install ase。



Atomsオブジェクト



操作できる主なオブジェクトはAtomsオブジェクトです。さまざまな構造を保存できます。一酸化炭素分子を作成してみましょう： 



from ase import Atoms
from ase.visualize import view
d = 1.1
co = Atoms('CO', positions=[(0, 0, 0), (0, 0, d)])
view(co)


その結果、次の形式の構造が得られます。







クラスター



ase.clusterモジュールを使用すると、クラスター（結晶格子を持つ粒子）を作成できます。クラスターは、無限の結晶からクリップ面に断片をクリップすることによって生成されます。銅クラスターを作成します。 



from ase.cluster.cubic import FaceCenteredCubic
surfaces = [(1, 0, 0), (1, 1, 0), (1, 1, 1)]
layers = [6, 9, 5]
lc = 3.61000
atoms = FaceCenteredCubic('Cu', surfaces, layers, latticeconstant=lc)
view(atoms)


サーフェスがクリッピングプレーンの場合、layersはレイヤーの数です。



構造を取得します。







ナノ粒子



そして、ここで記事のメイントピックに行きます。ナノ粒子は、直径がナノメートルのオーダーの粒子です（数百ナノメートルまで）。最も一般的なのは球形のナノ粒子です-私たちはそれらを生成します。どうやってするの？大きなクラスターを作成してから、中心からの原子の座標の距離で、そこから球を切り取りましょう。これを行うには、最初にすべての座標を中央に配置する必要があります。 



surfaces = [(1, 0, 0), (1, 1, 0), (1, 1, 1)]
layers = [50, 50, 50]
lc = 3.61000
atomsCu = FaceCenteredCubic('Cu', surfaces, layers, latticeconstant=lc)

possCu = atomsCu.get_positions()
cCu = atomsCu.get_center_of_mass()

new_possCu = []
for el in possCu:
    new_possCu.append(el - cCu)
new_possCu = np.array(new_possCu)


中心からの中心にある原子の距離は簡単に計算されます： 



def get_radius_centered(pos):
    return (pos[0]**2 + pos[1]**2 + pos[2]**2)**0.5


ここで、大きなクラスターから球状粒子（より正確には、その位置）を切り出すための関数を記述しています。 



def cut_atoms(poss, r):
    poss_cut = []
    for pos in poss:
        if get_radius_centered(pos) < r:
            poss_cut.append(pos)
    poss_cut = np.array(poss_cut)
    return poss_cut


これを実行に移すことは残っています： 



r = 5
possCu = utils.cut_atoms(new_possCu, r)
atoms = Atoms('Cu'+str(len(possCu)), positions=possCu)
view(atoms)


半径5オングストローム（0.5 nm）の銅ナノ粒子の次の画像を取得します。







コアシェル構造



実際には、コアシェル構造は興味深いものです。あるタイプの原子の核と別のタイプの原子核があります。たとえば、現実には、CuAgのコアシェル粒子（Cuコアを含む）は、Agに匹敵する触媒活性を持っている可能性がありますが、安価です。そのような構造を生成する方法を学びます。



次の戦略に従います。粒子から穴を切り、この穴を別のタイプの原子を持つ粒子で埋めます。これを行うには、穴を開ける関数が必要です。 



def cut_atoms_inside(poss, r):
    poss_cut = []
    for pos in poss:
        if get_radius_centered(pos) > r:
            poss_cut.append(pos)
    poss_cut = np.array(poss_cut)
    return poss_cut


中心に配置された原子座標を持つ銀のクラスターを作成します。 



lc_ag = 4.09
atomsAg = FaceCenteredCubic('Ag', surfaces, layers, latticeconstant=lc_ag)

possAg = atomsAg.get_positions()
cAg = atomsAg.get_center_of_mass()

new_possAg = []
for el in possAg:
    new_possAg.append(el - cAg)
new_possAg = np.array(new_possAg)


コアシェル粒子を作成します： 



r1 = 7
r2 = 5
possAg = utils.cut_atoms(new_possAg, r1)
atoms = Atoms('Ag'+str(len(possAg)), positions=possAg)

possAg = utils.cut_atoms_inside(possAg, r2)
possCu = utils.cut_atoms(new_possCu, r2)
poss = np.vstack([possCu, possAg])

atoms = Atoms('Cu'+str(len(possCu))+'Ag'+str(len(possAg)), positions=poss)
view(atoms)

リラクゼーション



そして最後に、ライブラリが作成された操作の例。コアシェル粒子を緩和します。潜在的なエネルギーが最小限の状態にします。電卓を設定します。 



from ase.calculators.emt import EMT
atoms = Atoms(atoms, calculator=EMT())
