当選者：TopCoder Open 2019



TopCoder Open 2019チャンピオンシップの決勝戦は11月13日から16日にヒューストンで開催され、Gena Korotkevich（ベラルーシ）はアルゴリズムとマラソンの2つの競技トラックで一度に1位になりました。誰もこれまでに1年でこれを行うことができませんでした！ 2019年シーズンのすばらしい終わりで、ジーンはすでに他の2つの主要なコンテスト、Google Code JamとFacebook Hacker Cupの決勝に勝っていました。

Topcoder Openは、Topcoder Inc.が主催する個人向けプロプログラミングトーナメントです。 2001年からTopcoder方式で実施されています。 Topcoderには、アルゴリズム、開発、First2Finish、UIデザイン、UIプロトタイプ、マラソンの6種類のコンテストがあります。

マラソンマッチ（MM）-これは最適化タスクです。主催者でさえ正解を知りません。プログラマーは、短時間で最も効果的なソリューションを提供する必要があります。

アルゴリズム（SRM）は、古典的なスポーツプログラミングです。1時間半で、Java、C＃、C ++またはPythonの問題を解決する必要があります。


この記事は、電子診察システムを開発し、ソフトウェアもサポートする EDISON Softwareの支援を受けて作成されました。
