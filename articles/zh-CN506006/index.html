Pythonists，停止使用方括号从字典中获取值

哈Ha！我向您介绍了乔纳森·许（Jonathan Hsu）撰写的文章“停止使用方括号表示法在Python中获得字典的值”的翻译。

超越传统方法并保护您的代码


字典是键和值的未排序集合。代表着：

  • 字典的每个元素都包含一个术语（键）及其定义（含义）。
  • 该词典的键必须唯一-不能重复。
  • 词典中的键没有列表项的明确排列。

要创建字典，请使用大括号，在大括号内输入键：值对，以逗号分隔。像这样：

author = {
   "first_name": "Robinson",
   "last_name": "Crusoe",
   "username": "Friday1719"
}

一种从字典中获取含义的传统方法（不安全）。

为了从字典中正常访问元素的值，请使用方括号。使用此类条目时，关键字会在字典名称后的方括号中显示：

author = {
   "first_name": "Robinson",
   "last_name": "Crusoe",
   "username": "Friday1719"
}
print(author['username']) #  Friday1719
print(author['middle_initial']) #   KeyError: 'middle_initial'

请注意，尝试通过不在字典中的键访问值将引发KeyError异常。这会造成严重的问题，尤其是在处理不可预测的工作数据时。

当然，您可以使用try / except构造或使用if语句。但是，对这种简单操作的更多关注只会使代码混乱。

author = {}
try:
   print(author['username'])
except KeyError as e:
   print(e) # 'username'

if 'username' in author:
   print(author['username'])

如果您有JavaScript开发经验，那么您可能希望使用点分符号从字典中获取值。但是在Python中，这将不起作用。

author = {
   "first_name": "Robinson",
   "last_name": "Crusoe",
   "username": "Friday1719"
}
print(author.username)
# AttributeError: 'dict' object has no attribute 'username'

我们使用get（）方法


当您需要从字典中获取值时，最安全的方法是使用get（）方法。此方法有两个参数：

  • First（必填）：我们要用来获取值的键的名称。如果我们的密钥可以在程序中更改，则此名称可以是字符串，也可以是变量名。
  • 第二个（可选）：如果我们的键突然没有出现在字典中将使用的值。

author = {
   "first_name": "Robinson",
   "last_name": "Crusoe",
   "username": "Friday1719"
}
print(author.get('username')) #   Friday1719
print(author.get('middle_initial', “Monday1917”)) #  Monday1917

如果密钥存在于字典中，则get（）方法的工作方式与访问方括号中的密钥完全相同。但是在字典中没有这样的键的情况下，get（）方法将返回默认值，从而使您不必处理异常。

默认值可以是在此上下文中有效的任何对象。请记住，此参数是可选的。因此，如果您未明确指定它，则当您尝试访问字典中不存在的键时，get（）方法将返回None对象。

我们使用setdefault（）方法


有时，您不仅需要安全地从字典中接收数据，而且还需要安全地将新数据添加到字典中。为此，字典具有setdefault（）方法。它具有与get（）方法相同的参数，但是与后者不同，当使用不存在的键访问字典时，它不仅会返回默认值，而且还会使用此键和传递的值在字典中创建一个新元素。如果使用setdefault（）方法访问字典时，传输的键已经在字典中，则此方法将使字典保持不变。

author = {
   "first_name": "Robinson",
   "last_name": "Crusoe",
   "username": "Friday1719"
}

print（author.setdefault（'username'））＃将输出Friday1719
print（author.setdefault （'middle_initial'，“星期一”））＃将输出星期一，并
在字典中创建一个带有键“ middle_initial”的元素，该键的值为“星期一” ”

另外，在上述的例子中，我们看到，setdefault（）方法的行为是从get（）方法的行为或从使用方括号访问使用现有的在它的键字典时没有什么不同。如果字典中没有这样的键，则setdefault（）方法不仅将其第二个参数的值返回给程序（如get（）方法），而且还会在字典中创建一个元素并将其键和值传递给它。这是setdefault（）方法的行为，并将其与get（）方法区分开。

现在，如果您遵循上面的示例并显示字典中的元素，然后我们得到以下结果：

print(author)
{
  "first_name": "Robinson",
  "last_name": "Crusoe",
  "username": "Friday1719"
  "middle_initial": "Monday"
}

使用get（）和setdefault（）方法是访问字典值的一流技术。只需花一些时间就可以摆脱旧习惯，并开始在实践中使用此技术。

如果您只需要从字典中获取值，那么您的助手就是get（）方法。

如果需要安全地将新值添加到字典中，请调用setdefault（）方法。
