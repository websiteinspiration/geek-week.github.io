Memperkenalkan .NET 5.0 Pratinjau 3

Memperkenalkan versi baru .NET 5.0 Preview 3. Berisi serangkaian fitur baru dan peningkatan kinerja. Kami terus bekerja pada fitur yang lebih mendasar yang akan dimuat dalam rilis final 5.0. Artikel tentang .NET 5.0 Pratinjau 1 mencakup apa yang kami rencanakan untuk implementasikan di .NET 5.0. Silakan periksa publikasi dan repositori dotnet / designs dan bagikan umpan balik Anda. Dan tentu saja, silakan instal Pratinjau 3 dan uji tugas Anda dengan versi baru.

Anda dapat  mengunduh .NET 5.0 Preview 3 untuk Windows, macOS dan Linux:

  • .NET 5.0 Pratinjau 3 dan Runtime
  • Gambar Docker
  • Pemasang snap

ASP.NET Core  dan  EF Core  juga sekarang tersedia.

Detail tentang versi di bawah cut.



Anda harus menggunakan Visual Studio 2019 16.5 untuk bekerja dengan .NET 5.0. Instal ekstensi C # terbaru  untuk menggunakan .NET 5.0 dengan Visual Studio Code. .NET 5.0 belum didukung di Visual Studio untuk Mac.

Dukungan .NET SDK untuk .NET Framework Assemblies


.NET SDK sekarang akan secara otomatis merujuk  ke   paket Microsoft.NETFramework.ReferenceAssemblies NuGet Paket NuGet berdasarkan target .NET Framework dalam file proyek. Perubahan ini memungkinkan Anda untuk membuat proyek .NET Framework di komputer tanpa paket penargetan .NET Framework diinstal. Peningkatan ini hanya berlaku untuk paket penargetan dan tidak memperhitungkan dependensi lain yang mungkin dimiliki proyek.
