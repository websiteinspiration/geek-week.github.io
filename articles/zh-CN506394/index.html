首次见面



300x300毫米的印刷台令人印象深刻。即使您不打算批量打印，也可以在这样的桌子上同时打印多个小型模型，从而节省大量时间。

冲压钢框架既厚又重，实际上是欲望的极限。

电源，主板和大多数内部组件位于打印机基座的坚固金属外壳中。因此，可以显着减小打印机在桌子上所占的面积。

由于其坚实的基础，如果将打印机移至其他位置，则无需对齐打印机。

令人惊喜的是-Mega X具有灯丝末端传感器，通常仅在更昂贵的3D打印机型号中才能找到。如果线断了或用完了，此传感器将暂停打印。此功能对大型项目非常有用。

和伟大的笔来调平桌子！手动对齐打印平台的任何人都知道旋转小高度调整旋钮有多不便，而您可以用手将平台钩在底部，而所有的工作都被浪费了。 Anycubic解决了此问题-Mega X具有较大的手柄，并且在校平螺钉上的螺距较小。手柄非常容易旋转，几乎可以用一根手指进行操作。

桌子的表面由公司玻璃制成，顶部有一层薄网孔，因此附着力没有问题-印刷品可以很好地固定在印刷品表面上。

配乐有点烦人。当您触摸屏幕时，会听到很大的喀哒声，并且在打印机启动时，会播放标准的声音片段。幸运的是，这些声音可以轻松地一劳永逸地关闭。

部件



组装非常简单：拧紧几颗螺丝并连接电缆。

Mega X组装仅用了20分钟。像当今市场上大多数3D打印机一样，Mega X几乎没有需要连接在一起的单独部件。从零件完全组装3D打印机的日子已经一去不复返了。

通常，组装非常简单：卸下固定水平杆的一根螺栓（上面有电线），请别人抬起打印机的前部并将框架滑向打印机的中心。然后拧紧两侧的四个螺栓。

安装灯丝传感器支架和线轴支架。

最后，连接四根电缆，所有电缆均带有连接器，连接起来非常方便。

工作腔尺寸



如前所述，Anycubic Mega X的打印体积为300 x 300 x 305毫米。即使与其他具有较大打印面积的廉价3D打印机相比，这也是一个令人印象深刻的空间。例如，与同一生产线的3D打印机（Anycubic Mega S）相比，体积增加了三倍，从9升增加到27升！

这种构建量的优点不仅在于能够打印大型对象。您可以同时打印超大型模型的多个部分或多个不同模型。在旅行之前，本文的作者急需印刷冰箱的几个支架，衣架，支架和备件。所有这些都能够在一个打印周期内完成打印，并在一夜之间完成工作。

底座的尺寸和面积



确保打印机所在的桌子足够深，并且打印机距离墙壁足够远。

与其他Anycubic打印机一样，Mega X组件主要位于设备底部。因此，打印机不会在桌子上占用太多空间。但是，线圈架向侧面突出，必须考虑到这一点。

打印机非常紧凑：宽度和深度均为50厘米，但考虑到沿Y轴的整个行程，最大深度约为65厘米。选择打印机支架时，请使用此信息。

打印质量



印刷品的分辨率较低：0.2毫米的层厚。

正如Anycubic Mega 3D打印机所期望的那样，Mega X可以高质量打印。

对于作者而言，第一张印刷不成功。原来，沿X轴的传送带被削弱了，所以我们出现了小的缺陷。断开3个夹子，拉紧皮带，然后再次拧紧夹子，大约花费了30秒。

下一次打印更好，但不是完美的，因为不正确地使用了Cura切片机-挤出机的温度对于选定的长丝设置得太高，并且移动速度不必要地提高了。

选择正确的设置后，打印结果开始无瑕疵，与作者以前的打印机的打印质量相对应-生物终结者3。

对准



巨大的手柄极大地促进了调平过程。

像大多数廉价3D打印机一样，Anycubic Mega X可以手动对齐打印平台。幸运的是，有了这样的调节旋钮，这很容易做到。将打印机设置为零，然后在所有四个角以X的形式插入一张纸，并根据需要调整每支笔。

手柄的设计和校平螺钉的细牙螺距使您无需用力整个手指，就可以轻松用手指进行微调，因此无需触摸平台。

建筑质量




Mega X 3D打印机主要由耐用的冲压钢零件组成，可以很好地契合在一起。这辆车看起来很坚固。

运动系统设计



两个圆柱导轨，沿Z轴的两个阶梯螺钉和沿Y轴的宽轮廓增加了稳定性。

廉价的3D打印机通常在Z轴上只有一个导轨，但是Mega X有两个（和两个电机）。这样可以消除振动并提高打印质量。此外，宽而稳定的双铝制Y轴滑架提高了稳定性。

包含Mega X电源和电子设备的大型机壳为打印机提供了额外的稳定性。

电源



Mega X 3D打印机使用MeanWell电源。优点：

  • MeanWell电源由确保电源稳定性的高质量组件组装而成；
  • 仅在进行打印时才打开电源风扇，因此打印机通常比平时安静。
  • 电源具有保险丝，这是确保任何3D打印机安全所必需的。

印刷表材料



印刷台上的材料非常有趣。Anycubic将其称为Ultrabase平台。它由加热的铝层组成，在其上有一层硼硅酸盐玻璃（具有更高的耐热性和抗机械损伤性），顶部有薄网孔。

该模型可以很好地固定在这样的表面上，因此不需要使用发胶或胶水将其固定。

挤出机



与现在的许多FDM 3D打印机一样，Mega X配备了Bowden型挤出机。简而言之：在此模型中，带有进纸机构的外壳与热端分开。到达热端的灯丝通过长管送入。重型步进电机与外壳一起安装在3D打印机的框架上，带有打印头的托架很轻，因此打印速度提高了。

作者不喜欢挤出机的管子碰到Z轴的滚珠丝杠并可以擦掉润滑剂。也许您应该拿出某种持有人。

接口



至少在配置OctoPrint之前，您必须使用Mega X触摸屏来控制打印机并开始打印。

嗯，作为“必须”，原则上很方便。 3.5英寸彩色显示屏可提供相当宽的视角。触摸传感器很灵敏，并且较大的菜单项易于按下。

每个菜单项都由带有图标的按钮表示。您需要了解菜单：事实是，当您单击某些项目时，会弹出一个附加菜单，而当您单击其他项目时，将立即执行操作。

例如，触摸温度图标将调出温度控制菜单，而触摸冷却图标将关闭所有加热器。前进和后退导航也有些混乱。

但是，菜单正常工作，并且更新固件时界面很有可能会改善。无论如何，不​​是很成功的3D打印机用户界面是常见的，这就是为什么要使用OctoPrint这样的打印机管理软件的原因。

印刷过程



为了切片3D模型，使用了Cura切片机，但是由于Anycubic Mega X是最近发布的打印机，因此您必须执行手动调整。

将.gcode文件写入SD驱动器上的任何目录，然后将其插入计算机-SD-USB适配器随打印机一起提供。然后，从菜单开始打印过程。

加载打印件的一个小问题是，打印机将以递归方式搜索SD卡上的任何.gcode文件，即使该文件已在垃圾箱（.Trashes目录）中删除了也是如此。这意味着，如果删除垃圾箱中的文件，但文件仍保留在SD卡上，则该文件仍会出现在打印屏幕上。因此，建议您每次保存文件时定期删除.Trashes目录或选择一个新名称。

Mega X可以使用以下材料进行打印：

  • 解放军
  • ABS
  • 热塑性聚氨酯
  • HIPS；
  • 木。

当断电或断线时，Mega X会自动暂停-这非常方便。
