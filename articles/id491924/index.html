PVS-Studio analyzer beta test # di Linux dan macOS


Semakin banyak pengguna penganalisa C # dari PVS-Studio tertarik pada kemungkinan menggunakannya untuk menguji kode C # di Linux dan macOS. Dan hari ini kami punya kabar baik.

Sejak akhir tahun lalu, kami telah secara aktif mengerjakan porting C # analyzer kami ke platform .NET Core, serta memastikan kinerjanya pada sistem operasi mirip Unix. Kami berencana untuk merilis PVS-Studio C # untuk .NET Core di Linux dan platform macOS pada akhir April - paruh pertama Mei.

Di Windows, penganalisis akan terus bekerja di bawah .NET Framework, karena kami memutuskan untuk berkonsentrasi pada rilis penganalisis untuk OS baru. Selain itu, Anda dapat memeriksa proyek .NET Core lintas platform pada Windows sekarang.

Kami berencana untuk memulai tes beta pada awal hingga pertengahan April. Jika Anda tertarik untuk berpartisipasi di dalamnya, isi formulir ini . Ketika pengujian beta dimulai, kami akan mengirimkan Anda petunjuk untuk memasang alat analisis dan lisensi uji coba.

PS Kami juga mengembangkan plug-in untuk lingkungan pengembangan JetBrains Rider untuk semua platform yang didukung oleh penganalisa. Sebagai bagian dari pengujian beta, sebagai "bonus", Anda juga akan memiliki kesempatan untuk mencoba plugin ini, termasuk dengan "klasik" PVS-Studio C # analyzer di Windows.
