Windowsターミナルプレビューv0.9

Windowsターミナルのバージョン0.9のリリースが行われました。これはターミナルの最新バージョンで、v1のリリース前に新機能が含まれます。Windowsターミナルは、Microsoft StoreまたはGitHubのリリースページからダウンロードできます。アップデートの詳細を詳しく見てみましょう！



コマンドライン引数


エイリアスwtがコマンドライン引数をサポートするようになりました。これで、新しいタブとパネルを使用して、好きなディレクトリに好きなプロファイルで好きなように分割したターミナルを起動できます！可能性は無限大！次に例をいくつか示します

。wt-d。
現在の作業ディレクトリにデフォルトのプロファイルでターミナルを開きます。

wt -d ;; new-tab -d C：\ pwsh.exe
2つのタブでターミナルを開きます。最初の-現在の作業ディレクトリから始まるデフォルトのプロファイル。 2つ目は、C：\ディレクトリで始まる「デフォルトのプロファイル「コマンドライン」ではなく）「コマンドライン」としてのpwsh.exeを含むデフォルトのプロファイルです。

wt -p "Windows PowerShell" -d ;;分割ペイン-V wsl.exe
垂直に分離された2つのパネルでターミナルを開きます。「Windowsターミナル」という名前のプロファイルが上部のパネルで起動され、下部のパネルでデフォルトのプロファイルが使用されます。デフォルトのプロファイル「コマンドライン」の代わりにwsl.exeを「コマンドライン」として使用します。

wt -d C：\ Users \ cinnamon \ GitHub \ WindowsTerminal; split-pane -p「コマンドプロンプト」; split-pane -p "Ubuntu" -d \\ wsl $ \ Ubuntu \ home \ cinnak -H

PowerShell自動検出


PowerShell Coreの 大ファンなら、素晴らしいニュースがあります。WindowsターミナルでPowerShellのすべてのバージョンが検出され、新しいプロファイルが自動的に作成されます。私たちの意見では、最も優れているPowerShellバージョン（最大のバージョン番号からGAバージョンまで、最高のバンドルを持つバージョンまで）は、「PowerShell」と呼ばれ、ドロップダウンリストの元のPowerShellコアスロットを占有します。



すべてのタブを閉じることを確認


あなたは毎回すべてのタブを閉じるように求められたくない人ですか？はいと答えた場合、この新しい機能は間違いなくあなたのためです！「すべてのタブを閉じる」確認ダイアログボックスを常に非表示にできる新しいグローバルパラメータが作成されました。これを行うには、profiles.jsonファイルの上部で「confirmCloseAllTabs」パラメーターをfalseに設定します。これにより、このポップアップが再び表示されることはなくなります。この新しいパラメーターの貢献をありがとう@ rstat1。
