如何观察月球和行星

观察月球和行星非常有趣。暴露在光线下并不能阻止对行星的观察，可以直接从城市观察到它们。为了观察行星，不需要具有大视野的目镜。甚至价格便宜的Plessla目镜也可以提供高效的视觉观察结果。



木星，土星和火星也许是天文观测中最容易接近的行星。我仍然记得20年前在80毫米Big School Refractor上看到的土星第一眼所带来的激动和惊喜。但是，经常有来自初学者的报告，关于首次观测的报道，特别是木星和火星，其中有些令人失望。 “我只是看到一个没有细节的光球”或“我看到一个我无法完全专注的小磁盘”。 “我的望远镜有故障吗？”这篇文章对新手天文学爱好者来说可能是有用的。它详细描述了对太阳系行星的视觉观察的细微之处和特征。

行星是天空中的光点，但月亮很大且非常明亮。但是，月球有许多最小的细节，因此出于对它们的考虑，有必要使用与观察行星相同的技术。为了使用望远镜获得最佳图像，需要考虑几个重要因素：

  1. 增加
  2. 解析度
  3. 闪耀
  4. 光散射
  5. 对比
  6. 清晰度

增加


最有争议的因素。行星很小，因此增加幅度越大越好！并不是的。您需要为望远镜使用最佳放大倍率。找到它的最简单方法是计算望远镜的最佳出瞳。出瞳是您通过望远镜的目镜看到的聚焦图像的大小。

出瞳计算如下：望远镜中透镜的直径（毫米），除以一个或另一个目镜的放大倍数。让我提醒您，望远镜的放大倍数是通过将镜头的焦距（以毫米为单位）除以所用目镜的焦距（也以毫米为单位）来计算的。

望远镜镜头的焦比（F / D）的计算方法如下：将镜头的焦距除以其直径（光圈）

事实证明，对于人眼来说，1 mm的出瞳可为光线充足的物体提供最佳分辨率。假设您有一个90毫米折射镜，其焦距为900毫米，焦距为F / D-10。在这种情况下，要获得月球或行星的最佳视野，必须使用10毫米目镜。对于F / D-5，使用5毫米目镜，对于F / D-8，使用8毫米目镜，依此类推。使用这种增加，大多数夜晚您都可以欣赏到行星的美丽景色。

分辨率取决于两个因素：望远镜镜头的直径（越大越好）和可见度。能见度（唱歌）是大气稳定性的量度。如果稳定，您将看到更多详细信息。如果大气中有很多湍流，那么小的细节将被“洗掉”。如果能见度很差，那么10英寸望远镜的显示范围不会超过4英寸。实际上，在恶劣的环境下，小型工具的性能要好一些。同样，从地表和远离热源（例如屋顶）的地方进行尽可能高的观测将有助于减少“图像流”的负面影响。在苏联文学中，建议至少爬300m。从海平面到山顶，山麓高原等，以排除对表面热层图像的负面影响。但是你要知道由于空气湍流，独立山丘的山峰将是不佳的选择。


月亮和大多数行星非常明亮。通常，在目镜的强光照射下，最细微的细节会丢失，这是在镜头焦平面上建立镜头的亮点。如何控制呢？最简单的方法是产生光污染。在观察月亮和行星时，夜间的眼睛适应会适得其反。打开门廊，阳台或您正在观察的任何其他地方的灯。更好的是，观察天空仍然是蓝色的。我在日落之前就可以欣赏到木星的最佳风景。如果这还不够，您可以使用镜头前面的光圈（特别是在没有专用滤镜的情况下在月球上建议使用），也可以使用滤镜。光圈设置对于焦距为F / D-4 ... F / D-6的快速望远镜非常有效。对于孔径较小的较小孔径的仪器，例如：F / D-8 ... F / D-15，我不建议这样做，因为这会降低分辨率。过滤器将更有效（以后选择过滤器的更多信息）。

光散射


当月亮，行星或恒星的亮光撞击望远镜的玻璃表面时，就会发生光散射。散射效果类似于眩光，对比度降低和分辨率降低。不幸的是，您无法使用滤光片控制光的散射。解决此问题的唯一方法是选择对角线，巴洛镜，目镜和滤光片，并控制好光散射水平。简而言之，我建议您选择质量良好的对角线，该对角线的镜面应具有介电涂层。

对比


观察行星和月球的目的是提供高对比度。这可以通过控制眩光和光散射以及选择具有良好对比度的目镜来实现。您还可以使用适当的滤镜来改善月球和行星表面某些细节的对比度（有关更多信息，请参见下文）。同样，当应用大放大倍率时，您会注意到对比度降低。

清晰度


某些光学望远镜能够比其他望远镜建立更清晰的图像。假设您可能已经有了望远镜，那么最好集中注意巴洛的目镜和镜片。许多目镜模型会在高放大倍率下产生“模糊”图像。不幸的是，其中一些以行星目镜的形式出售。矫正目镜-是观察行星的最佳目镜。廉价目镜也会降低图像清晰度。

选择望远镜及其配件的建议：

望远镜


就行星观测而言，无论大小和光学设计如何，都可以使用任何望远镜。但是，如果您购买的是专门用于观测月球/行星的产品，则F / D-8 ... F / D-15比率的长焦仪器将提供更好的结果。首选无色差的设计，因为XA会降低分辨率，尤其是在应用大放大倍率时。

在性能方面，我们建议：

80-120mm长焦消色差折射镜和80-100mm小型APO / ED折射镜。

也可以建议使用直径为5到11英寸的折反射望远镜（Maksutov，Schmidt-Cassegrain）。但是不幸的是，由于大气层的不稳定性，发挥他们的潜能往往不会成功。

较大的APO折射镜能够产生高质量，大幅度增长的产品，但价格昂贵。牛顿的大型望远镜和折反射镜可以提供行星的最佳视野。但是，为了利用更大的光圈（镜头直径），以获得更高的分辨率，必须选择具有出色大气稳定性的夜晚。这种情况很少发生，在平均夜晚使用较小的镜头直径会更实用。

筛选器


滤镜应该是望远镜之后的下一个优先事项，而且它们的质量也应该很高。远离许多制造商出售的现代行星式塑料过滤器。它们降低分辨率并增加光散射。购买时，我建议使用Baader，Lumicon或Refinery玻璃过滤器。您可以搜索在ebay，跳蚤市场等上使用过的产品，主要是不要刮擦

滤镜，通常建议对月亮和行星使用中性密度和偏振滤镜。我一开始就使用它们，但是我意识到彩色滤光片可以提供更好的结果。

彩色滤光片不仅可以减少眩光，还可以改善表面细节的对比度。橙色21号是新月和土星的最佳过滤器，它在火星上也表现良好。适用于火星的最佳滤镜是23A红色和大光圈-25号红色。蓝色的80A号适用于金星和水星，绿色的58号适用于满月。在选择最佳滤镜方面，木星是最困难的。多年来，我尝试了许多过滤器。在彩色滤光片中，只有蓝色80A可以帮助我。

我强烈建议使用Baader的一些特殊过滤器，它们适用于木星，土星和火星（尽管它们对于月球，金星和水星来说太弱了）。Baader Moon and Sky Glow是木星最好的滤镜，比蓝色80A更好。对于土星和火星，您可以使用Baader Contrast Booster对比度滤镜获得最佳效果。当行星非常明亮时（对峙附近），可以使用两个滤镜：Baader Moon和Sky Glow和Baader Contrast Booster一起使用，并将它们用于所有三个行星。我对这些滤镜特别喜欢的是，它们可以减少眩光并增强对比度，但不会显着改变行星表面的自然颜色。

目镜


矫正镜！不管您的增加是最有效的，我强烈建议您至少购买其中一个用于行星。正交镜目镜兼具清晰度，高对比度和出色的光散射减少效果。用过的矫正镜很容易在40-60美元之间找到。它们大多数在日本的一两个工厂生产，因此质量控制通常是良好的。如果您想购买新的，那么性价比最高的是Baader Classic Orthos（BCO）。 BCO的视野也为50度，远大于传统的正视镜和Plessla目镜。

矫正镜方案的两个局限性是狭窄的视场（40-50度）和在小焦距下瞳距延长短。例如，一个18毫米的正视目镜具有约14毫米的方便瞳孔投影。当与2倍增倍镜一起使用时，有效焦距变为9毫米（用于焦距为F / D-8 ... F / D-10的望远镜。当与3倍增倍镜一起使用时，有效焦距为6 mm（在具有焦距比的望远镜中使用F / D-5 ... F / D-7）。

多年来，我尝试了许多目镜，其价格从初始到中等水平。其中一些在高放大倍率，低对比度和光的可怕散射下图像模糊。矫形器是行星的最佳解决方案。但是，如果您希望更广阔的视野（特别是对于牛顿望远镜的所有者，在Dobson支架上，无法使用测微螺钉引导物体）或较大的瞳孔扩展，Vixen SLV，TeleVue Radians和Delites，Explore Scientific 68和82系列和Meade 5000 UWA作为高质量的月球/行星目镜。在非常有限的预算范围内，您可以配戴Plessla目镜，但您只需要购买优质的目镜即可。

有人会说：“我的目镜在月球上工作正常，”事实如此。月亮是很容易观察到的物体。如果您的目镜产生稍微模糊的图像，您仍然会看到很多细节。尽管如此，将尖锐的，高端的和非常经济的目镜并排进行测试将是一个启示。就像从优质的模拟电视切换到高清广播一样，区别非常明显。

增倍镜


如果您的目镜在合适的焦距范围内，则不需要Barlow。此外，巴洛廉价镜头可能会降低对比度并增加光散射。但是，优质的Barlows可能会有所帮助。要在短焦望远镜中获得1 mm或更小的出瞳，您需要使用焦距短的目镜。在这种情况下，移开瞳孔可能会不舒服。在这种情况下，最好的选择可能是使用2倍或3倍增倍镜以及更长的望远目镜。另外，巴洛镜增加了望远镜的有效焦距，因此，与短焦距目镜相比，使用巴洛镜+目镜的组合可以获得更好的行星图像。强烈建议使用Baader Q barlow 2。25倍增倍镜，以及高端细分市场TeleVue 2倍和3倍增倍镜。

对角线


对角线是光路中经常被忽略的部分。这可能是导致“望远镜目镜中的恒星视图变少”的原因。主要优先事项之一应该是增加对角线的直径。如果望远镜有一个2英寸的聚焦器，建议切换到2英寸的电介质对角线，这将改善DSO（深空天体）和行星的图像。我在GSO的中等价格，介电对角线方面拥有丰富的经验。您还可以推荐制造商：Celestron，Orion，Explore Scientific。

如果您正在寻找适合月亮和行星的最佳对角线，那么我会选择质量好的棱镜。棱镜散射的光线少于电介质反射镜对角线，并且对月亮和行星更适用。在性能价格比方面，我建议使用Baader T2棱镜。

观察


月亮




在月球上，大多数细节都可以在我们伴侣的照明和未照明表面的边界上看到。由于终结器（沿昼夜边界的线）每天都会随着月相的变化而改变位置，因此您每天晚上都可以欣赏到新的风景。即使使用最小的望远镜和双筒望远镜，您也可以在月球表面看到许多陨石坑。较大的光圈可提供更精细的细节。使用我的8英寸Schmidt-Cassegrain望远镜，平均每晚，我可以找出大约1公里以内的细节，并将整个观测过程集中在一个陨石坑中，研究墙壁的复杂形状，中央滑道，微陨石坑和其他微小的细节。





水星和金星




这些行星几个月都看不见了。在短时间内，它们被视为“早晨或傍晚的星星”。汞很难检测到，因为即使在距离太阳一定距离的地方，汞也仍然非常靠近我们的恒星。用肉眼寻找水星已经是一项成就。在极少的日子里，与水星的延伸（与太阳的最大距离）相吻合，在平静，晴朗的气氛中，可以在地平线附近看到行星。甚至在小型仪器中也可以看到汞的阶段。

金星更容易看到。行星的延伸持续了数周。即使是最普通的双筒望远镜（例如10x50）也可以显示金星的相位。在大型望远镜中，使用滤光片有时可以解决金星大气层中的暗云。

火星




在这一年中，火星在生肖星座中移动得相当快。如果它在天空中，大多数时候您只能看到该行星的一个小橙色圆盘，没有任何细节。但是，每两年一次，当火星的表面尺寸显着增加时，就会与之对峙（与太阳对峙）。下一场反对派将于2020年10月13日举行，所以请做好准备！ :)您可以从七月开始观察地球！



由于表面细节对比度低，火星是最难观察到的行星。滤镜和目镜必须是好的。但是即使使用80毫米的望远镜和耐心，在对抗期间，您仍可以了解其表面的许多细节。观察的重点在于，您不必急于将行星保持在望远镜的视野内，而不必等到表面细节更清晰地“抽出”的时刻（在使大气平静的时刻）。顺便说一下，这是观测诸如木星，火星和土星等行星的一般策略。

木星




木星通常每年可见4-5个月。由于卫星的动态四重奏及其丰富的细节，木星是天文学中最有趣的物体之一。甚至具有10x50光学设计的双筒望远镜也可以使用行星盘及其4个卫星。使用大放大倍率和双目镜的直径（例如15x70、20x80），您可以轻松地在其光盘上看到几个主带。当使用高质量的滤光镜和目镜进行观察时，即使使用80毫米望远镜，也可以看到复杂的木星波段系统。您还可以观察大红点的过境和木星满月的影子，它们掠过整个行星盘。将望远镜的直径增加到8英寸或更多，将增加木星的色彩饱和度，在天然气巨头的带和极地地区（包括小暴风雨和花期）显示更多小细节。它还将允许在小磁盘上安装卫星。观看木星是一项很棒的技能，通过练习，您将学到更多。

土星




像木星一样，土星每年出现4-5个月。但与木星不同，它的表观尺寸较小。在10x50的双筒望远镜中，它看起来像一个鸡蛋，经过一些实践和清晰的光学观察，在15x70的双筒望远镜中，圆盘周围可以分辨出微小的环。即使在适度的望远镜中，环也很容易被检测到。相对较小的孔径增加将在其环中显示“卡西尼间隙”（不需要过滤器）。与木星相比，土星的云系统对比度要低得多。为了解决行星盘及其环上的细节，需要使用滤镜并增加望远镜物镜的直径。土星最大的卫星-土卫六即使在低倍率下也清晰可见。使用大型望远镜，可以分辨更多的卫星。

天王星和海王星


他们往往会在同一星座保持多年。在过去的几年中，秋天是观察它们的最佳时间。可以用双筒望远镜或小型望远镜以“蓝星”的形式看到两个行星。使用8英寸或更大的仪器，可以看到很小的绿色行星盘，没有表面细节。此外，使用大型望远镜（8英寸及以上），您可以看到海王星卫星海卫一和至少三颗天王星卫星。

冥王星


在我看来还是一颗星球！:)最近几年，他在射手座。在非常稳定的气氛中，只能使用直径为8英寸或更大的望远镜将其视为非常微弱的恒星。

“行星游行”


每隔两三年，行星就会排列在一起，并且在一夜之内一次都可以看到它们。我过去曾观察到这种现象-非常令人印象深刻！:)下次我将提前报告此现象。
