Was gibt es in Quarantäne zu sehen? Eine Auswahl von Materialien aus Technostream (Teil 3)



Wir setzen unsere Auswahl interessanter Materialien fort ( erster Teil , zweiter Teil ). Diesmal ein Technosphere-Schulungskurs, ein Vortrag über Unit-Tests und eine Talkshow-Talkshow für Oh, mein Code.

Applied Python, Technopark, Herbst 2018


Der Technopark ist eines unserer gemeinsamen Bildungsprojekte, in diesem Fall mit MSTU. Bauman.

Der Kurs führt in die Python-Sprache ein - eine der beliebtesten und beliebtesten Sprachen des heutigen IT-Marktes. Die Nachfrage nach der Sprache ist nicht von Grund auf neu: Die Einfachheit der Eingabe und Syntax, die reichhaltigste Auswahl an Werkzeugen zur Lösung aller Arten von Aufgaben - dies und vieles mehr hat dazu geführt, dass Python weltweit weit verbreitet ist. Dank dieses Kurses können Sie sich dem Ökosystem der Sprache anschließen.

  • Vorlesung 1. Einführung. Entwicklungstools, Git kennenlernen. Eine kurze Einführung in die Prinzipien, Konzepte und Funktionen von Python. Grundlegende Syntax.
  • Vorlesung 2. Datenstrukturen . Grundlegende Datenstrukturen in Python. Modulsammlung.
  • Vorlesung 3. Module . Wie Code in Python organisiert ist. Was sind Module, wie werden sie erstellt, wie arbeitet Python mit ihnen? OOP.
  • Vorlesung 4. Input-Output . Kontextmanager, Arbeit mit FS, Arbeit mit dem Netzwerk, Grundlagen des Multithreading.
  • Vorlesung 5. Metaprogrammierung . Magische Methoden, Dekorateure, Metaklassen, Deskriptoren.
  • Vorlesung 6. Relationale Datenbanken . Das Gerät, Funktionen, Anwendungsmethoden.
  • Vorlesung 7. Asynchrone Programmierung . Ausbeute von, asyncio, aiohttp, asyncio-redis.
  • Vorlesung 8. Webentwicklung auf Django . HTTP-Grundlagen, Webframeworks, Informationssicherheit, Bereitstellung von Webanwendungen.
