過酷な環境向けのタッチターミナル


Advantech TPCデバイスは、ヒューマンマシンインターフェース（HMI）アプリケーションを実行するためのシンクライアントです。それらは、タッチスクリーン付きの安全なモニターに組み込まれた受動冷却を備えた本格的なx86コンピューターです。

HMI端子は、過酷な産業環境向けに設計されています。オペレーターは、手袋を外すことなく、汚染が増加している状態で端末を操作できます。一方、抵抗膜式タッチパネルは、水滴が付着したときに誤って押すことを防ぎます。

この記事では、手袋でデバイスをテストし、いくつかのモデルの特性を分析します。


デモ：手袋をしたタッチスクリーンの使用



産業環境では、液滴、チップ、ほこり、およびその他の汚染物質が端末画面に付着する可能性があります。したがって、ポピュラーな静電容量式タッチパネルの場合のように、汚染が画面上の誤ったクリックを引き起こさないことが重要です。雨滴がスマートフォンの画面で偶発的なクリックを引き起こす可能性があることを思い出すにはそれで十分です。

このような状況では、抵抗タイプのタッチスクリーンの方が適しています。それらは手袋を外すことなく制御でき、汚染の場合の誤警報から保護されます。ビデオはタッチスクリーンの精度がかなり高いことを示していますが、オペレーターは粗いゴム手袋を使用していますが、小さなインターフェイス要素をクリックすることは難しくありません。


[ビデオ]手袋をした抵抗型スクリーンで作業できます。

このビデオは、Windows 10のGoogle Chrome Webブラウザーで実行されているSCADAデモシステムを示しています。

仕様書


TPCシリーズ端末は、64ビットOSと完全にパッシブな冷却をサポートするIntel Atomプロセッサーをベースにしています。TPC-1840WPおよびTPC-2140WPモデルは、 OpenGLアプリケーションを要求実行するために、サブシステム、より強力なグラフィックスとAMD T56Eプラットフォームを使用しています。すべてのデバイスのRAM容量は4GBです。背面カバーを取り外した


モデルTPC-1551T

シリーズのすべてのデバイスはUEFIをサポートしています。端末は、PXEを介してネットワーク経由でロードできます。または、オペレーティングシステムのローカルロード用に、2.5インチHDD / SDDディスクまたはCFastメモリカードを端末にインストールできます。ローカルドライブを使用して、重要なデータをバックアップできます。
デバイスの概念にはシンクライアントとしての使用が含まれるという事実にもかかわらず、システムのパフォーマンスは、Windows 10のようなデスクトップオペレーティングシステムとフルブラウザーを実行するのに十分です。


次のインターフェイスは背面パネルにあります：

2x-RS232 / RS485
1x-USB 3.0
3x-USB 2.0 2x-
ネットワークブートサポート付き独立型イーサネット
iDoor拡張モジュール用スロット
オーディオポートイン/アウト、セカンドディスプレイを接続するためのHDMIを備えたモデルもあります。
ほとんどのTPCモデルには、拡張カードを取り付けるためのフルサイズのmPCIeスロットがあります。

IDoor拡張モジュール



TPCインターフェイスのセットは、独自のAdvantech iDoorモジュールを使用して拡張できます。これらのモジュールは、外部インターフェイスパネルを備えたmPCIeモジュールです。
iDoorモジュールのフォームファクターには、PROFIBUS、EtherCAT、PROFINET、3G / 4Gモデムのネットワークインターフェイスモジュール、WiFi / Bluetooth、デジタル入力および出力モジュールなど、独自の産業用インターフェイスの多くのオプションがあります。

取り付け




TPC端子は、表面のあるレベルにホゾ型パネルとして埋め込むことができます。これを行うには、デバイスのすべての端に取り付けブラケットとネジ穴があります。各デバイスのページで、3Dモデルの表示が可能です。



標準のVESAブラケット100x100またはブランドブラケットTPC-1000H-SMKE / WMKEに設置して、テーブルや壁に設置することもできます。

電気的コンプライアンス


すべてのデバイスは、International Electrotechnical Commission（IEC）規格に準拠するためのテストに合格し、IEC60664パート2に準拠しています。これらのデバイスは、不適切な電源接続（プラスとマイナスの逆端子）に対する保護を備えており、供給電圧の短期間の増加に加え、最大20％の絶えず増加する電圧に耐えることができます名目から。

過酷な条件での作業


すべてのTPCデバイスは、-20〜60℃の広い温度範囲で動作できます。IEC60068-2-64およびIEC60068-2-27に準拠し、最大10Gの一定の振動や衝撃に耐えることができます。
ほぞ穴パネルに端子を正しく取り付けると、前面パネルの保護はIP66クラスに対応します。

なぜHMI端末が必要なのですか？


シンクライアントを使用すると、組み込みシステムのインターフェイスを集中管理および更新できます。本格的なオペレーティングシステムにより、Webアプリケーションなど、使い慣れた開発ツールを使用できます。その結果、そのようなインターフェースの開発速度は、ハードウェア制御パネルの構築や特殊なマイクロコントローラーのプログラミングに比べて大幅に高速になります。

隙間や開口部のない一体型ディスプレイパネルは、液体や汚染物質のデバイスへの侵入を排除し、抵抗膜式タッチスクリーンを使用すると、手袋、ポインター、スタイラスを操作できます。

拡張モジュールの存在により、HMIアプリケーションを使用してローカルでデジタル信号を処理するためのディスクリートI / Oポートだけでなく、ターミナルに産業用インターフェイスを追加できます。AdvantechのHMIに基づくHabr Control Panelの記事でレビューしたAdvantech HMI Designerプログラムで独自のHMIインターフェイスを作成する例
