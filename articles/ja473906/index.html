1. Windows 10向けのアプリケーション開発


完全な学習には約4〜5時間かかります。コース中、あなたは：

  • まず、Windows 10向けのアプリケーション開発の基本を理解します。
  • 次に、Visual Studioで作業をマスターします。
  • 次に、Windowsの最も一般的な開発環境（UWP、WPF、およびWindowsフォーム）でアプリケーションを作成する方法を学びます。
  • 最後に、インターネットに接続されたアプリを作成する方法を学びます。

このコースを完了するために必要なのは次のとおりです。

  • Windows 10コンピューター
  • C＃または同様の言語の基本的な知識

詳細については、このリンクからトレーニングを開始してください。

2. Xamarin.Formsを使用してモバイルアプリを作成する


このコースは、ツールの全機能をすでに完全にまたはほぼ完全にカバーしており、10時間のトレーニング用に設計されています。彼は、Xamarin.Formsの操作方法と、C＃とVisual Studioを使用して、iOSおよびAndroidデバイスで実行されるアプリケーションを作成する方法を教えます。したがって、学習を開始するには、Visual Studio 2019が必要であり、C＃と.NETを操作するスキルが必要です。

コースモジュール：

  • Xamarin.Formsを使用してモバイルアプリケーションを作成します。
  • Xamarin.Androidの概要;
  • Xamarin.iOSの概要;
  • XAMLを使用してXamarin.Formsアプリケーションでユーザーインターフェイスを作成します。
  • Xamarin.FormsのXAMLページでのレイアウトのカスタマイズ。
  • 共有リソースとスタイルを使用して、一貫したXAML Xamarin.Formsページを設計します。
  • Xamarinアプリケーションを公開するための準備。
  • XamarinアプリケーションでのREST Webサービスの使用;
  • Xamarin.FormsアプリケーションでSQLiteを使用してローカルデータを保存します。
  • スタックとタブナビゲーションを使用して、複数ページのXamarin.Formsアプリケーションを作成します。

詳細を確認して学習を開始する

3. Azureでのデータストレージ


Azureは、非構造化データストレージ、アーカイブストレージ、リレーショナルストレージなどを使用して、データを保存する多くの方法を提供します。3.5〜4時間で、Azureでストレージを管理し、ストレージアカウントを作成し、クラウドに格納する必要のあるデータに適したモデルを選択する方法についての基本的な知識を得ることができます。

コースモジュール：

  • データストレージへのアプローチの選択。
  • ストレージアカウントを作成します。
  • アプリケーションをAzureストレージサービスに接続する
  • Azureストレージアカウントの保護（このモジュールは、「クラウドでのデータの保護」コースにも含まれています）;
  • BLOBストレージの使用

詳細とトレーニングの開始

4. PythonおよびAzure Notebookを使用した機械学習の概要


このコースの所要時間は約2〜3時間ですが、実用的なスキルが数多く提供されます。学習した後、Azure Notebooksで実行されているJupyter NotebooksでPythonおよび関連ライブラリを使用して、パターンを予測し、傾向を識別する方法を学習します。

コースでは、気候データを個別に分析し、フライトの遅延の可能性を予測し、ユーザーレビューの気分を分析します。これらはすべて機械学習とPythonを使用しています。

完了するには、Pythonプログラミングの基本的な知識が必要です。

5.クラウドデータ保護


そして、ここにセキュリティに関するかなり大きなコースがあります-それを研究するために約6-7時間が必要となります。このコースでは、Azure組み込みサービスを使用してアプリケーションデータを安全に格納し、承認されたサービスとクライアントのみがこのデータにアクセスできるようにする方法を学びます。

コースモジュール：

  • Azureの安全なアーキテクチャ
  • 実装前に考慮すべき5つの主要なセキュリティ要素。
  • Azureストレージアカウントの保護（このモジュールは「Azureでのデータのストレージ」コースにも含まれています）;
  • Azure Key Vaultを使用してサーバーアプリケーションのシークレットを管理します。
  • Azure Application Servicesを使用したブラウザーアプリケーションでの認証。
  • 条件付きアクセスによるAzureリソースの保護
  • ロールベースのアクセス制御（RBAC）によるAzureリソースの保護
  • Azure SQLデータベースの保護。

6.サーバーレスアプリケーションの作成


Azureの機能を使用すると、イベント駆動型で、さまざまな外部イベントが発生したときにトリガーされるオンデマンドコンピューティングシステムを作成できます。6〜7時間で、Azure関数を使用してサーバー側ロジックを実行し、サーバーレスアーキテクチャを作成する方法を学びます。

コースモジュール：

  • ビジネスプロセスの自動化に最適なAzureサービスを選択する。
  • Azure Functionsを使用してサーバーレスロジックを作成する
  • トリガーを使用してAzure関数を実行する。
  • 入力バインディングと出力バインディングを使用してAzure機能を組み合わせる
  • Sustainable Functionsを使用してサーバーフリーの長いワークフローを作成します。
  • Visual Studioを使用したAzure機能の開発、テスト、および展開。
  • Azure機能のWebスニファーを使用してGitHubイベントを監視します。

7. DevOpsプラクティスの開発


開発者向けのこのコレクションの最後のコースに行きました。そして、彼が英語でその中で唯一のものです-彼らがそれをロシア語に翻訳することができたまで。このコースの所要時間は1〜1.5時間で、DevOpsの入門知識を提供します。

DevOpsは、エンドユーザーに価値を継続的に提供するための人、プロセス、および製品の関連付けです。Azure DevOpsは、この機能を提供するサービスのスイートです。Azure DevOpsを使用すると、クラウドまたはオンプレミスで任意のアプリケーションを作成、テスト、デプロイできます。透明性、コラボレーション、継続的デリバリー、継続的デプロイメントのためのDevOpsプラクティスは、ソフトウェア開発サイクルに統合されています。

このトレーニングスキームでは、DevOpsへの移行を開始し、次のことを学びます。

  • バリューストリームパターンが現在のプロセスとテクノロジーの評価にどのように役立つか。
  • 無料のAzure DevOpsアカウントを登録する方法
  • Azureボードを使用して作業項目を計画および追跡する方法。
