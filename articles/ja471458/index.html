勾配降下


その本質は非常に単純です。たとえば、次の関数があります。

y = x*x + 4 * x + 3



関数のドメインからxの任意の値を取ります。これが点x1 = -4であると想像してください。

次に、点x1でこの関数のxに関する導関数を使用します（関数が複数の変数（たとえば、aおよびb）に依存する場合、各変数に関して偏導関数を使用する必要があります）。y '（x1）= -4 <0

ここで、xの新しい値を取得します：x2 = x1-lr * y'（x1）。lr（学習率）パラメータを使用すると、ステップサイズを設定できます。したがって、以下が得

られます。特定の点x1 <0での偏微分（関数が減少する）の場合、極小点に移動します。（x2はx1より大きくなります）

与えられた点x1> 0での偏微分（関数が増加する）の場合、極小点に移動しています。（x2はx1より小さい）

このアルゴリズムを繰り返し実行することにより、最小値に近づきます（ただし、到達しません）。

実際には、これはずっと単純に見えます（ただし、係数aとbが上記のストアの場合に最も正確に適合するとは思いません。したがって、y = 1 + 2 * xという形式の依存関係を使用して、データセットを生成し、モデルをトレーニングします。このデータセット）：（
コードはここに書かれています）

import numpy as np

#     
np.random.seed(42) 

#  np-  1000     0..1
sz = 1000
x = np.random.rand(sz, 1) 

#   y = f(x)     
y = 1 + 2 * x  + 0.1 * np.random.randn(sz, 1) 

#    0  999 
idx = np.arange(sz)
#   
np.random.shuffle(idx)

train_idx = idx

#    
x_train, y_train = x[train_idx], y[train_idx]

#        
a = np.random.randn(1)
b = np.random.randn(1)
print(a,b)

#  
lr = 0.01
#  
n_epochs = 10000

#  
for epoch in range(n_epochs):
  
    #       a  b
    #     
    yhat = a + b * x_train
    
    # 1.  
    #      :
    error = (y_train - yhat)
    
    # 2.   (  )
    #   a
    a_grad = -2 * error.mean()
    #   b
    b_grad = -2 * (x_train * error).mean()
    
    # 3.  ,    
    a = a - lr * a_grad
    b = b - lr * b_grad
print(a,b)

コードをコンパイルすると、aとbの初期値がそれぞれ必要な値1と2から離れており、最終値が非常に近いことがわかります。

a_gradとb_gradがそのように考えられている理由を少し明らかにします。
F(a, b) = (y_train - yhat) ^ 2 = (1 + 2 * x_train – a + b * x_train)。aに関するFの偏微分はに等しくなり -2 * (1 + 2 * x_train – a + b * x_train) = -2 * errorます。bに関するFの偏微分はに等しくなり-2 * x_train * (1 + 2 * x_train – a + b * x_train) = -2 * x_train * errorます。我々は平均値をとる(mean())ためerrorとx_train、そして y_train値の配列であり、bはスカラーです。
