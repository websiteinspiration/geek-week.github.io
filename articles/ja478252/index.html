PowerShellを使用して写真アーカイブで注文する

こんにちは、ハブル！

先日、典型的なタスクが発生しました-友人がたくさんの写真を秩序だった階層に変えるのを助けるために。すべてうまくいきますが、写真の山は1つではなく2つあります。MacとWin10のラップトップです。解決策を探して、Linux用のスクリプトをいくつか見つけましたが、そのようなクロスプラットフォームのものが見つかりませんでした。私たちは自分で書きます-猫の下で。

したがって、Windowsを含め、何かを自動化する必要があります。自動化とはPowershellを意味します。私の幸福なことに、Microsoftは最近クロスプラットフォーム化しました。使用します。

写真には必須の共通属性である作成日があるため、次の順序で階層に基づいて階層を構築します。アーカイブルート→年→月→日。行動の過程で：

  1. ルートディレクトリを作成する
  2. ファイルのリストを取得する
  3. ファイルごとに、EXIFから作成日を決定し、
    • 必要に応じて、ディレクトリ構造を補足します。
    • ファイルをドラッグします。

それは簡単に聞こえます、行きましょう。

ユーザーは熟練していないので、ターミナルでの作業からできる限り彼を救うことは論理的です。スクリプトがそのディレクトリとサブフォルダーで動作するようにします。ユーザーは、スクリプトをルートフォルダーに配置して、それをpowershellターミナルウィンドウにドラッグするだけです。

[CmdletBinding()]
$curDir=$MyInvocation.MyCommand.Path | Split-Path -Parent

exiftoolの可用性を確認します。 

if (!(ExifTool)) {
    Write-Host "Install exiftool first!" -ForegroundColor Red -BackgroundColor Black
    break
}

関心のある拡張子を示し、使用するファイルのリストを取得します。必要な拡張子があり、まだソートされていないもののみを選択します。

将来の構造のルートを作成します。

$ExtList = "arw,jpg,jpeg,NEF"
$FileList = Get-ChildItem $curDir -Recurse | `
    Where-Object {(($ExtList.ToUpper() -match $_.Name.Split('.')[-1]) `
        -or ($ExtList.ToLower() -match $_.Name.Split('.')[-1])) `
        -and ($_.FullName -notmatch "Ordered")
    }

# Define & reate archive root folder
$ArchiveRoot = Join-Path -Path $curDir -ChildPath "Ordered"
if (!(Test-Path $ArchiveRoot)) {
    New-Item -ItemType Directory -Path $ArchiveRoot
}

すべてのjpegファイルに必要な情報がEXIF内にあるとは限らないため（たとえば、ソーシャルネットワークから再保存した後）、それらのファイル用に別のディレクトリを作成します。ユーザーは個別に運命を決定します：

$ChaosRoot = Join-Path -Path $curDir -ChildPath "Chaos"
if (!(Test-Path $ChaosRoot)) {
    New-Item -ItemType Directory -Path $ChaosRoot
}

解析するファイルのそれぞれについて、次のことを行います。

exiftoolを使用して作成日を決定し、年、月、日の3つのコンポーネントに分割します。exiftoolはcsvへのエクスポートをサポートしているため、powershellはcsvで動作します。

foreach ($File in $FileList) {
    $FileDateString = $null
    $FileDate = $null
    try {
        $FileDateString = ConvertFrom-Csv (ExifTool.exe $File.FullName -CSV) | `
            Select-Object -ExpandProperty CreateDate -ErrorAction Stop    
    }
    catch {
        Move-Item -Path $File.FullName -Destination $ChaosRoot -Force
        continue
    }
    
    $FileDate = $FileDateString.Split(" ")[0].Split(":")

必要な属性を取得できた場合は、年、日、月の個別の値に分割してさらに進みます。そうでない場合は、識別されていないファイルがすぐに混乱します。

なので、日付を受け取ります。ディレクトリを作成し、ファイルをドラッグします。

$YearPath = Join-Path $ArchiveRoot $FileDate[0]
$MonthPath = Join-Path $YearPath $FileDate[1]
$FileDest = Join-Path $MonthPath $FileDate[2]
if (!(Test-Path $FileDest)) {
   New-Item -ItemType Directory -Path $FileDest
}
Move-Item -Path $File.FullName -Destination $FileDest -Force
}

さて、すでに不要な空のディレクトリを削除してゴミ箱をきれいにすることを忘れないでください：

$AllDirsAtFinal = Get-ChildItem -Path $curDir -Recurse -Force | Where-Object {$_.PSIsContainer -eq $true} 
foreach ($Dir in $AllDirsAtFinal) {
    $DirLenght = Get-ChildItem -Path $Dir.FullName -Recurse -Force -ErrorAction SilentlyContinue | `
        Measure-Object -Property Length -Sum -ErrorAction SilentlyContinue
    if ($null -eq $DirLenght.Sum) {
        Remove-Item $Dir.FullName -Recurse -Force -ErrorAction SilentlyContinue
    }
}

それが一般的にはすべてです。
