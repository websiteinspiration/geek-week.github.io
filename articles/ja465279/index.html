シャムネットワークを使用した顔認識



シャムニューラルネットワークは、最も単純で最も人気のある単一学習アルゴリズムの1つです。クラスごとに1つのケーススタディのみを使用する方法。したがって、シャムネットワークは通常、各クラスにデータの単位があまりないアプリケーションで使用されます。

約500人を雇用する組織の顔認識モデルを作成する必要があるとします。たたみ込みニューラルネットワーク（CNN）に基づいてそのようなモデルを最初から作成する場合、モデルをトレーニングして優れた認識精度を達成するには、これらの500人それぞれの多くの画像が必要になります。ただし、そのようなデータセットをコンパイルできないことは明らかなので、CNNまたはその他のディープラーニングアルゴリズムに基づいてモデルを作成しないでください。十分なデータがない場合。そのような場合は、シャムネットワークのような複雑な1回限りの学習アルゴリズムを使用して、より少ないデータでトレーニングできます。

実際、シャムネットワークは、重みとアーキテクチャが同じ2つの対称ニューラルネットワークで構成され、最終的にはエネルギー関数を組み合わせて使用​​します。E

シャムのネットワークはマーキング付きのペアの値を入力する必要があるので、そのようなセットを作成しましょう。同じフォルダからランダムに2枚の写真を撮り、それらを「本物の」ペアとしてマークします。次に、異なるフォルダから2枚の写真を撮り、それらを「インポジット」（インポジット）としてマークします。



すべての写真をマークされたペアに配布したら、ネットワークを調べます。各ペアから、1つの写真をネットワークAに転送し、2番目の写真をネットワークBに転送します。どちらのネットワークもプロパティベクトルのみを抽出します。これを行うには、2つのたたみ込み層を使用して、整流線形単位（ReLU）をアクティブ化します。プロパティを調査した後、両方のネットワークによって生成されたベクトルを、類似性を推定するエネルギー関数に転送します。ユークリッド距離を関数として使用します。

def read_image(filename, byteorder='>'):
    
    #first we read the image, as a raw file to the buffer
    with open(filename, 'rb') as f:
        buffer = f.read()
    
    #using regex, we extract the header, width, height and maxval of the image
    header, width, height, maxval = re.search(
        b"(^P5\s(?:\s*#.*[\r\n])*"
        b"(\d+)\s(?:\s*#.*[\r\n])*"
        b"(\d+)\s(?:\s*#.*[\r\n])*"
        b"(\d+)\s(?:\s*#.*[\r\n]\s)*)", buffer).groups()
    
    #then we convert the image to numpy array using np.frombuffer which interprets buffer as one dimensional array
    return np.frombuffer(buffer,
                            dtype='u1' if int(maxval)

def get_data(size, total_sample_size):
    #read the image
    image = read_image('data/orl_faces/s' + str(1) + '/' + str(1) + '.pgm', 'rw+')
    #reduce the size
    image = image[::size, ::size]
    #get the new size
    dim1 = image.shape[0]
    dim2 = image.shape[1]

    count = 0
    
    #initialize the numpy array with the shape of [total_sample, no_of_pairs, dim1, dim2]
    x_geuine_pair = np.zeros([total_sample_size, 2, 1, dim1, dim2]) # 2 is for pairs
    y_genuine = np.zeros([total_sample_size, 1])
    
    for i in range(40):
        for j in range(int(total_sample_size/40)):
            ind1 = 0
            ind2 = 0
            
            #read images from same directory (genuine pair)
            while ind1 == ind2:
                ind1 = np.random.randint(10)
                ind2 = np.random.randint(10)
            
            # read the two images
            img1 = read_image('data/orl_faces/s' + str(i+1) + '/' + str(ind1 + 1) + '.pgm', 'rw+')
            img2 = read_image('data/orl_faces/s' + str(i+1) + '/' + str(ind2 + 1) + '.pgm', 'rw+')
            
            #reduce the size
            img1 = img1[::size, ::size]
            img2 = img2[::size, ::size]
            
            #store the images to the initialized numpy array
            x_geuine_pair[count, 0, 0, :, :] = img1
            x_geuine_pair[count, 1, 0, :, :] = img2
            
            #as we are drawing images from the same directory we assign label as 1. (genuine pair)
            y_genuine[count] = 1
            count += 1

    count = 0
    x_imposite_pair = np.zeros([total_sample_size, 2, 1, dim1, dim2])
    y_imposite = np.zeros([total_sample_size, 1])
    
    for i in range(int(total_sample_size/10)):
        for j in range(10):
            
            #read images from different directory (imposite pair)
            while True:
                ind1 = np.random.randint(40)
                ind2 = np.random.randint(40)
                if ind1 != ind2:
                    break
                    
            img1 = read_image('data/orl_faces/s' + str(ind1+1) + '/' + str(j + 1) + '.pgm', 'rw+')
            img2 = read_image('data/orl_faces/s' + str(ind2+1) + '/' + str(j + 1) + '.pgm', 'rw+')

            img1 = img1[::size, ::size]
            img2 = img2[::size, ::size]

            x_imposite_pair[count, 0, 0, :, :] = img1
            x_imposite_pair[count, 1, 0, :, :] = img2
            #as we are drawing images from the different directory we assign label as 0. (imposite pair)
            y_imposite[count] = 0
            count += 1
            
    #now, concatenate, genuine pairs and imposite pair to get the whole data
    X = np.concatenate([x_geuine_pair, x_imposite_pair], axis=0)/255
    Y = np.concatenate([y_genuine, y_imposite], axis=0)

    return X, Y

今度はシャムネットワークを作成します。まず、コアネットワークを定義します。これは、プロパティを抽出するたたみ込みニューラルネットワークです。ReLUアクティベーションを使用して2つの畳み込みレイヤーと、フラットレイヤーの後に最大値（最大プーリング）を決定するレイヤーを作成します。

def build_base_network(input_shape):
    
    seq = Sequential()
    
    nb_filter = [6, 12]
    kernel_size = 3
    
    
    #convolutional layer 1
    seq.add(Convolution2D(nb_filter[0], kernel_size, kernel_size, input_shape=input_shape,
                          border_mode='valid', dim_ordering='th'))
    seq.add(Activation('relu'))
    seq.add(MaxPooling2D(pool_size=(2, 2))) 
    seq.add(Dropout(.25))
    
    #convolutional layer 2
    seq.add(Convolution2D(nb_filter[1], kernel_size, kernel_size, border_mode='valid', dim_ordering='th'))
    seq.add(Activation('relu'))
    seq.add(MaxPooling2D(pool_size=(2, 2), dim_ordering='th')) 
    seq.add(Dropout(.25))

    #flatten 
    seq.add(Flatten())
    seq.add(Dense(128, activation='relu'))
    seq.add(Dropout(0.1))
    seq.add(Dense(50, activation='relu'))
    return seq


次に、コアネットワークの画像のペアを転送します。これにより、ベクトル表現、つまりプロパティベクトルが返されます。

input_dim = x_train.shape[2:]
img_a = Input(shape=input_dim)
img_b = Input(shape=input_dim)

base_network = build_base_network(input_dim)
feat_vecs_a = base_network(img_a)
feat_vecs_b = base_network(img_b)

feat_vecs_aとfeat_vecs_b画像のペアのプロパティベクトルです。それらのエネルギー関数を渡して、それらの間の距離を計算してみましょう。そして、エネルギーの関数として、ユークリッド距離を使用します。

def euclidean_distance(vects):
    x, y = vects
    return K.sqrt(K.sum(K.square(x - y), axis=1, keepdims=True))


def eucl_dist_output_shape(shapes):
    shape1, shape2 = shapes
    return (shape1[0], 1)

distance = Lambda(euclidean_distance, output_shape=eucl_dist_output_shape)([feat_vecs_a, feat_vecs_b])

エポックの数を13に設定し、RMSプロパティを適用して最適化し、モデルを宣言します。

epochs = 13
rms = RMSprop()

model = Model(input=[input_a, input_b], output=distance)

次に、損失contrastive_loss関数関数を定義し、モデルをコンパイルします。

def contrastive_loss(y_true, y_pred):
    margin = 1
    return K.mean(y_true * K.square(y_pred) + (1 - y_true) * K.square(K.maximum(margin - y_pred, 0)))

model.compile(loss=contrastive_loss, optimizer=rms)
