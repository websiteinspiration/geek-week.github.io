GoとMggoフレームワークの紹介

みなさん、こんにちは！Goでの最初のプログラミング経験を共有したいと思います。

私はフロントエンド開発者としてプログラミングに着手しました。次に、Pythonのバックエンドに切り替え、少しだけC＃Asp.Net MVCに切り替えました。しかし、最近、2か月前に、この美しい言語Goに出会いました。

しかし、少し頭が痛くなり、Go言語が明らかになり、私はそれに夢中になりました。もちろん、最初に「バイク」を書くことにした最初のこと、つまりMGGOと呼ばれるWebフレームワークです。

私が欲しかったものと私が志したもの


もちろん、すぐにAsp.NetのようなMVCフレームワークを作りたかったのです。しかし、Go言語は独特であるため、それほど単純ではないことがわかりました。しかし、これが起こったことです：

コントローラー


コントローラは、メソッドが外部（api）と内部のみの両方であり、ビューを担当することができる構造です。

すぐに例として、ニュースコントローラについて考えてみましょう。

import mggo

type News struct{
   ID int
   Title string
   Text string 
}
func NewNews() *NewNews{
   return &NewNews{}
}
func init() {
    //  
   mggo.RegisterController("news", NewNews)
}

func(n *News) Read(ctx *mggo.BaseContext) News{
   return News{1, "First News", "Text first News"}
}

func(n *News) IndexView(ctx *mggo.BaseContext, data *mggo.ViewData, path []string){
   data.View = "news/news.html"
   data.Data["News"]
   data.Data["News"] = n.Read()
}


その後、ニュースはリンク/ news /で入手できます。

ここで、newsはコントローラーの名前で、メインページはIndexViewメソッドが担当するインデックスです。

次にAPIについて。NewsコントローラにはReadメソッドがあります。デフォルトでは、サービスのみです。このメソッドへのapiリクエストを許可するには、コントローラの「init」関数でメソッドに権限を追加する必要があります。

func init(){
   ...
   mggo.AppendRight("News.Read", mggo.RRightGuest)
}

その後、すべてのユーザーがAPIを呼び出すためにReadメソッドを使用できます。 

fetch("/api/", {
        "method": "POST",
        "headers": {
          'Accept': 'application/json',
          'Content-Type': 'application/json'
        },
        "body": JSON.stringify({
            "method": "News.Read",
            "params": {}
        })
}).then(res => res.Json()).then(console.log)

結果 

{result: {ID:1, Title:"First News", Text:"Text first News"}}

パラメータについて少し。paramsオブジェクト内のすべてのパラメーターは、構造体フィールドとして渡されます。

たとえば、Read関数を少し変更してみましょう。

func(n *News) Read(ctx *mggo.BaseContext) News{
   if n.ID ==1 {
      return News{1, "First News", "Text first News"}
   } else {
      return News{}
   }
}

また、API呼び出しはパラメーターID：1-params {ID：1}で行う必要があります）

ほかに何か？


PostgreSQL Go-pgライブラリはフレームワークに接続されています。すぐに例を見てみましょう。ニュースコントローラーには静的データがあります。変更する必要があります。これを行うには、最初に行う必要があるのは、ニューステーブルを作成することです。init関数に数行を追加します。

func init(){
   ...
   mggo.InitCallback(func(){
      mggo.CreateTable([]interface{}{(*News)(nil)})
   })
}

ここで、初期化後、id、title、textというフィールドを持つニューステーブルを取得します。

Readメソッドを変更します。

func(n *News) Read(ctx *mggo.BaseContext) News{
   mggo.SQL().Select(n)
   return *n
}

ID：1パラメーターを使用してNews.Readメソッドを呼び出すと、結果として、キー1を使用してニューステーブルからデータが取得されます。go
