入力信号がバッファsnにあるとすると、2つの環状均質化バッファの正弦波と余弦波の位相-bsとbcサイズNがあります。ヘッドポインタとテールポインタは、bHとbTに共通です。最初の時点では、bH = N-1、bT = 0です。平均化サイクルカウンターC = 0。

入力バッファーから4つのサンプルを取得し、文字のシーケンスに従ってそれらを追加します。

つまり、入力信号を分析して、サンプリング周波数の4分の1の周波数の存在を定量化するパラメーターを評価できます。このパラメーターに特定のしきい値を設定すると、すべてを2値化でき、簡単に言えば、質問に答えることができます。入力信号に特定のトーンがあるかどうか。

これは非常に良いですが、ビットを送信する必要があり、ビットは2つの値を取ることができます。

1つを使用して2つの信号状態でシステムを実装することはまあまあの考え方なので、無音（一時停止）で状態の1つをエンコードしません。これは検出を非常に困難にします。前提の始まりを何らかの形で強調し、その終わりをどのように配置するかを理解する必要があります。

代わりに、「1」と「0」を異なる長さのパルスでエンコードします。ビット間には、いわゆる ガードインターバル-マルチパスの伝播とリバーブを処理する必要があるため。簡単に言うと、ガードインターバルは、前のビットのすべての反射、すべてのアフターサウンド、エコーが消滅する場所（時間）です。

この信号構造により、レシーバーの操作アルゴリズムが大幅に簡略化されることを考慮して、トーンが表示されるのを待ち、最初に気づき、トーンが消えるのを待ち、時間を書き留めます。受信した時間の長さが「1」に近い場合、おそらく「1」の値を少し取り、「0」のように見える場合は、明らかに「0」の値を少し取りました。

一般的に、これはある種のモールス信号であると言えます。

public short[] ModulateData(byte[] data)
{
   double alpha = 0;
   double phase = 0;           

   List<short> samples = new List<short>();            
   BitArray bits = new BitArray(data);         

   for (int i = 0; i < bits.Length; i++)
   {
         int sLim = (bits[i]) ? oneDurationSmp : zeroDurationSmp;
         alpha = 0;
         phase = 0;
   
         for (int sIdx = 0; sIdx <= sLim; sIdx++)
         {
               alpha = Math.Sin(phase);
               phase += delta;
               if (phase >= alimit)
                        phase -= alimit;
               samples.Add(Convert.ToInt16(alpha * short.MaxValue));
         }
         samples.AddRange(new short[defenseIntervalSmp]);                
     }
     return samples.ToArray();
}

public double TransmitData（byte [] data）
public double TransmitData(byte[] data)
{
     var samples = ModulateData(data);
     double txTime = ((double)samples.Length) / SampleRateHz;

     var rawBytes = new byte[samples.Length * 2];
     for (int i = 0; i < samples.Length; i++)
     {
          var bts = BitConverter.GetBytes(samples[i]);
          rawBytes[i * 2] = bts[0];
          rawBytes[i * 2 + 1] = bts[1];
     }

     using (var ms = new MemoryStream(rawBytes))
     {
          using (var rs = new RawSourceWaveStream(ms, new    WaveFormat(Convert.ToInt32(SampleRateHz), 16, 1)))
                {
                    using (var wo = new WaveOutEvent())
                    {
                        wo.Init(rs);
                        wo.Play();
                        while (wo.PlaybackState == PlaybackState.Playing)
                        {
                            Thread.SpinWait(1);
                        }                        
                    }

                    rs.Close();
             }
             ms.Close();
       }
       return txTime;
}
