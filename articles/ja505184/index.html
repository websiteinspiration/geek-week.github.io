Quest Softwareの最長12か月間の無料トライアルライセンス

Quest Softwareは、情報セキュリティシステム（Change Auditor、InTrust、Enterprise Reporter）、Microsoft Office365またはAzureクラウド環境の監査とリカバリ（オンデマンドライセンス管理、オンデマンド監査など）、モニタリング（Foglight、スポットライト）、データベースレプリケーション（Shareplex）は、環境内で上記のツール（だけでなく）を長い間試すことができる特別なプログラムの立ち上げを発表しました。

Hals Softwareは、Questのパートナーとして、QuestのディストリビューターであるMerlionがパイロットプロジェクトを実施し、ベンダー製品を使用する効果的なアプローチについて話し合う準備ができています。これは、長時間の試運転を行い、提案されたツールの信頼性と有効性を評価するのに最適な時期です。

はめ込む




InTrustは、さまざまな情報ソースから非構造化データを収集し、それをファイルシステムに効率的に格納できる、主力のQuestソリューションです。ソリューションの主要な機能は、異常とそれらに対する既製の反応を検出するための事前定義されたルールです。ワークステーションのWindowsシステムログのイベントの種類については、すでに監視して応答することが望ましいと述べています。InTrustを使用した暗号化ウイルス攻撃の検出について読むこともできます。

変更監査人




Change Auditorは、Microsoft環境およびディスクアレイの変更をオンラインで監査するためのツールです。ADなど、Power Shellスクリプトを使用して監査を行う時代の一部のお客様は、Windows管理者という一人の人間に大きく依存していました。この依存関係は、Change Auditorが解決する主な問題です。

ここで言及しなければならないツールのもう1つの機能は、Windowsシステムの内部呼び出しに基づいてChange Auditorが変更を監査することです。ベンダーによると、この方法は深くネストされたグループでも変更を検出し、ログを記録および抽出するときよりも負荷が少なくなります。もう1つの機能は、ドメインのスーパーユーザーのアクションをブロックする機能です。

スポットライト




Spotlightは、OracleおよびSQL Server DBMSを監視するためのツールで、赤で強調表示されている各メトリックを改善するためのヒントを提供できます。Spotlightは、各インスタンスに適切なパフォーマンスレベルを自動的に決定し、しきい値を設定します。同様の設定を手動で行うこともできます。Spotlightは、Oracleトレースログファイルの内容をグラフィカルに表示して、SQLクエリの処理の詳細を表示し、リソースを利用してこれらのクエリを実行することもできます。

Spotlightでは、データベースの状態を表示できるモバイルアプリケーションが開発されています。インストールには約10分かかります。

SharePlexは、独自の完全なレプリケーションツールセットを通じて、追加のアップデート、アドオン、および管理パックの必要性を排除します。Oracleを使用してデータを複製する場合、価格はOracle独自のツールと比較して低くなります。EnterpriseバージョンからStandardへ、およびローカル環境からクラウドへの一般的な移行ケースですが、Shareplexは幅広い移行オプションをサポートしています。
