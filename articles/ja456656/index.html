SDL 2のレッスン：レッスン4-PNGのストレッチ

みなさん、こんにちは！これはSDL 2の4番目のレッスンです。元のレッスンは小さいため、2つのレッスンを1つにまとめることにしました。しかし、それらはこことここにあります。さて、レッスンを始めましょう。

まず、何をするかを決めます。今日は、別の拡張子（BMPではない）、つまりPNGの画像をアップロードする方法を学びます。作業を高速化し、画像のサイズを変更するには、別の形式で画像をやり直します。この画像で作業します。


取り付け


したがって、まずPNGをダウンロードする方法を学びます。標準のSDL 2機能セットでは、非BMP形式の画像のアップロードはありません。別の形式をダウンロードするには、拡張機能をインストールする必要があります。

コードを書く


SDL_imageを 設定した後、コードの記述を開始します。まず、いつものように、ライブラリを接続し、ウィンドウサイズを設定して、3つのグローバル変数を作成します。

#include <SDL2/SDL.h>
#include <SDL2/SDL_image.h>
#include <iostream>

const int SCREEN_WIDTH = 640;
const int SCREEN_HEIGHT = 480;

SDL_Window *win = NULL;
SDL_Surface *scr = NULL;
SDL_Surface *flower = NULL;

SDL_image.hは個別のライブラリなので、個別に接続した ことに注意してください。

次に、既知の3つの関数を記述します。これらの関数では、通常どおり、何かを追加および変更します。

変換する時が来ました。最初に、少し理論。知っている人はほとんどいませんが、画像をダウンロードする場合、画像は24ビット形式でダウンロードされ、最新のディスプレイのほとんどは32ビット形式です。また、画像を画面に表示するたびに、最初に32ビット形式でやり直しました。単純なプログラムの場合は問題ありませんが、ある種の大きなプロジェクトを作成する場合はパフォーマンスに大きな影響を与えるため、それを採用し、画像のダウンロードの段階で表示形式に再作成します。SDL_ConvertSurface関数が役立ちます。これは3つの値を取ります：フォーマットしたい表面、フォーマットしたいフォーマット、そしてフラグ。画像の花をフォーマットしたい、最初のパラメータとして渡します。2番目のパラメーターはウィンドウサーフェスの形式です。フラグは使用しません。この関数は、表面のコピーを返します。これは、すぐにflowerに割り当てます。エラーを見つけるには、SDL_GetError（）関数を使用します。これは、既にサーフェスを操作しており、SDL_imageからではなく、SDLからのものだからです。 次に、Quit関数を記述し、関数を1つだけ追加します。

その後、クラスSDL_Rectのオブジェクトを作成します。画像を引き伸ばすために必要になります。お気づきの場合は、花の画像はウィンドウサイズの4分の1に小さく（2分の1に2分の1に小さく）、ウィンドウサイズに拡大する必要があります。したがって、長方形bg_flowerの幅にウィンドウの幅の値を書き込み、高さ-ウィンドウの高さを書き込みます。座標を0に設定すると、画像が左上隅に表示されます。

画像をリサイズして表示するための特別な機能があります。次のようになります。

   SDL_BlitScaled(flower, NULL, scr, &bg_flower);

4つの意味があります。1つ目は表示したい画像、2つ目はこの画像から切り出したい四角形（画像全体を取得したい場合はNULLを書き込みます）、3つ目は画像を表示したい面、4つ目は同じ長方形、サイズです画像を引き伸ばしたり圧縮して表示するために使用する座標。

次に、画面の表面を更新し、遅延を設定して終了し、0を返します。

    SDL_Delay(2000);

これでプログラムが終了します。この奇跡をコンパイルするには、別のコンパイルオプションとして-lSDL2_imageを追加する必要があります。

そして、これが今日の終わりです。ここに次のコードがあります：

#include <SDL2/SDL.h>
#include <SDL2/SDL_image.h>
#include <iostream>

const int SCREEN_WIDTH = 640;
const int SCREEN_HEIGHT = 480;

SDL_Window *win = NULL;
SDL_Surface *scr = NULL;
SDL_Surface *flower = NULL;

bool init() {
    if (SDL_Init(SDL_INIT_VIDEO) != 0) {
        std::cout << "Can't init video: " << SDL_GetError() << std::endl;
        return false;
    }

    int flags = IMG_INIT_PNG;
    if ( !( IMG_Init( flags ) & flags ) ) {
        std::cout << "Can't init image: " << IMG_GetError() << std::endl;
        return false;
    }

    win = SDL_CreateWindow(" PNG", SDL_WINDOWPOS_UNDEFINED, SDL_WINDOWPOS_UNDEFINED, SCREEN_WIDTH, SCREEN_HEIGHT, SDL_WINDOW_SHOWN);
    if (win == NULL) {
        std::cout << "Can't create window: " << SDL_GetError() << std::endl;
        return false;
    }

    scr = SDL_GetWindowSurface(win);

    return true;
}

bool load() {
    flower = IMG_Load("flower.png");
    if (flower == NULL) {
        std::cout << "Can't load: " << IMG_GetError() << std::endl;
        return false;
    }

    flower = SDL_ConvertSurface(flower, scr->format, NULL);
    if (flower == NULL) {
        std::cout << "Can't convert: " << SDL_GetError() << std::endl;
        return false;
    }

    return true;
}

void quit() {
    SDL_FreeSurface(flower);
    
    SDL_FreeSurface(scr);
    SDL_DestroyWindow(win);

    SDL_Quit();
    IMG_Quit();
}

int main (int argc, char ** args) {
    if (!init()) {
        system("pause");
        quit();
        return 1;
    }

    if (!load()) {
        system("pause");
        quit();
        return 1;
    }

    SDL_Rect bg_flower;
    bg_flower.w = SCREEN_WIDTH;
    bg_flower.h = SCREEN_HEIGHT;
    bg_flower.x = 0;
    bg_flower.y = 0;

    SDL_BlitScaled(flower, NULL, scr, &bg_flower);

    SDL_UpdateWindowSurface(win);

    SDL_Delay(2000);

    quit();

    return 0;
};
