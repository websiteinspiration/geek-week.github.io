新年、新しいブラウザ：Microsoft Edgeの予備評価が終了し、ダウンロードできるようになりました



1年ほど前、Chromiumオープンソースプロジェクトに基づいてMicrosoft Edgeを再構築する意向を発表しました。これは、すべてのユーザーの互換性を改善し、Web開発者の断片化を減らし、Chromiumコミュニティとのパートナーシップを確立してChromium自体のコアを改善するために必要でした。 Igniteでは、Webテクノロジーと検索の新しいビジョン、新しいカラフルなロゴ、Microsoft Edge + Bing（ビジネス向けのブラウザーと検索エンジン）の新しいコンセプトを紹介しました。そして今、私たちはあなたのフィードバックを楽しみにしています！

私たちは素晴らしい仕事をしました。本日、新しいMicrosoft EdgeがサポートされているすべてのバージョンのWindowsおよびmacOSで90を超える言語でダウンロードできることをお知らせいたします。 Microsoft EdgeはiOSおよびAndroidでも利用できます。つまり、優れたクロスプラットフォームサポートです。新しいMicrosoft Edgeは、プライバシー、速度、効率を向上させ、以前のEdgeの2倍の速さで世界クラスのパフォーマンスを提供します。 Googleのプライバシーポリシーは新しいブラウザに適用されます。デフォルトで有効になっており、Webを閲覧するときに3つのレベルの制御を提供する追跡防止などの新機能に感謝します。

Microsoft Edgeのもう1つの革新的な機能では、Webページとのやり取りをカスタマイズして、たとえば、新しいタブのレイアウト、デザイン、コンテンツを選択して、オンライン環境をより有効に活用できます。



過去数か月間、AADサポート、Internet Explorerモード、4Kビデオストリーミング、ドルビーオーディオ、PDF手書き、BingとのMicrosoft Search統合、Chrome拡張機能のサポートなど、Microsoft Edgeの新機能に熱心に取り組んできましたはるかに。

ビジネスまたは教育のIT管理者であり、組織内で広範囲に展開したい場合は、スタンドアロンパッケージとポリシーをダウンロードし、新しい商用サイトですべての詳細を確認することもできます。



人々は何百万回も新しいMicrosoft Edgeのプレリリースバージョンをデバイスにダウンロードしており、多くの組織がこれらのバージョンをユーザー間で展開し始めています。重要な古いアプリケーションとWebサイトを持っているが、最新のWebインターフェイスとセキュリティも必要とする企業や学校は、新しいInternet Explorerモードを利用して、「1つの石で2羽の鳥を殺す」ことができました。また、Microsoft 365のユーザーは、組織のローカルネットワークでMicrosoft Searchを使用して、Microsoft Edgeのアドレスバーに必要なクエリを入力するだけで、ファイル、人、オフィスプランなどを検索できます。これらの機能をすでにテストした人は、それらを高く評価しています。

新しいMicrosoft Edgeへの切り替え-何を期待しますか？


このマイルストーンに到達した今、コンピューターとガジェットのユーザーはおそらく何を期待すべきか疑問に思っています。新しいMicrosoft Edgeを入手するには2つの方法があります。今すぐ手動でダウンロードするか、通常のユーザーの場合は、Windows Updateを使用してデバイスに自動的にインストールされるまで待ちます。切り替えると、パスワード、お気に入り、フォームに入力する情報、基本設定が新しいMicrosoft Edgeに自動的に転送されます。展開計画の詳細については、こちらをご覧ください。

IT管​​理者は、新しいMicrosoft Edgeが商業顧客に自動的に展開されないため、企業環境でのパイロット展開用のスタンドアロン展開パッケージをダウンロードする必要があります。さらに、これらのチャネルはテストと検証に並行して使用できるため、Microsoft Edge Insiderチャネルは新しいMicrosoft Edgeにアップグレードされません。

新しいブラウザの導入は単に「切り替え」だけではないことを理解しているため、移行をできるだけ簡単にしたいと考えています。IntuneやConfiguration Managerなどのツールを使用して展開を簡素化することに加えて、新しいMicrosoft Edgeへの移行を支援するために、さらに多くのことを行います。Igniteで、FastTrackおよびApp Assure for Microsoft Edgeのサポートを発表しました。FastTrackは、Microsoft 365、Azure、またはDynamics 365に対応するサブスクリプションを既に持っている場合、追加料金なしでMicrosoft Edgeを組織にインストールするのに役立ちます。サイトがInternet Explorer 8以降、Google Chrome、または古いMicrosoft Edgeと互換性がある場合は、新しいMicrosoft Edgeで作業します。そうでない場合は、App Assureに連絡してください。修正をお手伝いします。

次は何ですか？


もちろん、イノベーション、テスト、新機能の登場は止まりません。この最初のリリースはほんの始まりにすぎません。次に何が起こるかを確認したい場合は、引き続きベータ版、開発版、カナリー版の事前評価チャネルを使用することをお勧めします。これらのチャネルは、Microsoft Edge InsiderのWebサイトからダウンロードできます。Microsoft Edgeの機能パイプラインを理解できるだけでなく、貴重なレビューでMicrosoft Edgeの改善に役立てることもできます。あなたの参加は、世界中のすべての人々にとって、新しいMicrosoft Edgeとインターネット全体をより良くするのに役立ちます。

感謝！


Microsoft Edge Insidersのコミュニティと、新しいMicrosoft Edgeの作成に協力してくれたChromiumエンジニアに感謝します。私たちは引き続きオープンソースのChromiumプロジェクトに積極的に参加し、その開発に貢献しています。これまでに、アクセシビリティ、最新のデータ入力（タッチ、音声、デジタル、手書きを含む）、およびその他の多くの分野で、1900以上の改善を提案してきました。
