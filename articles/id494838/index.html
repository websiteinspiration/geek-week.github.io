Modem GSM / 3G / 4G dalam sistem tertanam menggunakan modem Quectel EC21 LTE dan Proyek Yocto sebagai contoh


Banyak perangkat yang didasarkan pada sistem tertanam memerlukan Internet. Koneksi Internet kabel tidak selalu memungkinkan bagi mereka, dan jika perangkat berada di area jangkauan seluler, maka Anda dapat terhubung ke Internet menggunakan modem GSM / 3G / 4G.


Catatan ini menjelaskan proses membuat distribusi Linux untuk sistem tertanam menggunakan Yocto Project toolkit, yang memungkinkan Anda untuk menghubungkan perangkat ke Internet melalui modem GSM / 3G / 4G dan protokol PPP.


Kandungan


  • Perangkat keras dan perangkat lunak
  • Mempersiapkan PC untuk membangun distribusi
  • Membuat proyek dan membangun kit distribusi
  • Menambahkan Dukungan PPP ke Kernel OS
  • Menambahkan paket ppp ke paket distribusi
  • Skrip untuk pppd dan obrolan
  • Resep Ekstensi untuk Paket ppp
  • Kesimpulan dan tautan ke proyek
