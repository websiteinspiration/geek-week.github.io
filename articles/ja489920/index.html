サイト、IPv6に移動、2つ

昨年9月18日、ベラルーシ人は予想外の政令350号に満足した。他の官僚の中で、特に興味深い段落がありました：
6.インターネットサービスプロバイダがに必要とされています
...
2020年1月1日から、インターネット上の情報システムおよび（または）情報資源を配置するネットワークデバイスによって、バージョン4と6のインターネット・プロトコルを完全にサポートを提供して技術を使用して対処するためのサービスを提供する際に、
...
つまり、1月1日から、ベラルーシでホストされているすべてのサイトがIPv6経由でアドレス指定できるようになります。

このようにして、ベラルーシは、IPv6が法に定められた最初の国となりました。
ニュースでは、次の技術革新についてのスローガンが集まりましたが、IPv6の管理および管理の実装がどれほど成功したかを見てみましょう。

まず、「共通の」リソースを調べます。残念ながら、パブリックドメイン内にすべてのBYドメインドメインのリストが見つからなかったため、いくつかの代表的なサンプルを作成し、サイトでAAAAレコードを確認します。

最初のリストは、liveinternet.ruによる一般的なリソースに基づいています。

ドメイン総数：2461
AAAAレコードあり：773
合計：31.4％


1つの地域プロバイダーのDNSサーバーから数時間で取得されたログに基づいて、「一般市民」の代替サンプルが作成されました。

ドメイン合計：14280
AAAAレコードあり：3924
合計：27.47％

ここでは、BYゾーンのすべてのサイトがベラルーシでホストする必要があるわけではないことに注意してください。
2010年2月1日付けのベラルーシ共和国大統領令第60号「インターネットの国内セグメントの使用を改善するための措置について」および2010年4月29日付けのベラルーシ共和国閣僚会議の布告令第644号「グローバルコンピュータネットワークインターネットの国別セグメントの使用の改善に関するいくつかの問題について」ベラルーシ共和国の領土にあるインターネットの国別セグメントの情報ネットワーク、システム、およびリソースは、国家登録簿への登録の対象となります。ただし、これは、商品を販売し、仕事を行い、ベラルーシ共和国の領土でサービスを提供するためにWebサイトを使用して活動が行われる場合にのみ必要です。

したがって、結果は非常に驚くべきものでした。これらのリストでは、AAAAレコードを持つリソースは10％以下になると予想していましたが、平均的なサイト所有者は非常に意識していることがわかりました。

さらなる分析はより正確になります。そして、これらの分析のリソースは州の構造であるか、または法令60と決議644に該当します。つまり、それらはベラルーシ共和国に配置する必要があり、350令の機能を利用できます。

最初のものは演算子のリストに基づいていますコミュニケーションの分野での活動のライセンスを発行した人。彼らはインターネットサービスのプロバイダーであり、注文なしでIPv6を使用する必要があります。一部の組織はサイトを持たない、いくつかのライセンスを受け取った、BYゾーン外のドメインを持っているなどの理由で、サンプルの位置はライセンシーよりも少なくなっています。

ドメイン合計：159
AAAAレコードあり：35
合計：22％

予期せず、プロバイダーとホスティング業者は平均的なベラルーシ人より無責任でした。そして、これらの22％でさえ完全に正直ではありません。実行はショーのためのものです-彼らはメインドメインに登録され、残りは「忘れられました」、例えば：

$ dig -t aaaa +short beltelecom.by
2a02:2208:1:1::89
$ dig -t aaaa +short www.beltelecom.by
$ dig -t aaaa +short my.beltelecom.by
$ dig -t aaaa +short sms.beltelecom.by

はい、法令はIPv6が無料で提供されることを明記していないため、一部のプロバイダーはそれらの価格を設定することに失敗しませんでした。たとえば、個人の場合、Beletecom は / 64サブネットに 16 BYN（〜475 RUB）を要求します。

しかし、銀行は購入するのに十分なお金を持っている必要があります。国立銀行のリストに従ってそれらをチェックします：

合計ドメイン：27
AAAAレコードを持っている：1
合計：3.7％

ideabank.byだけが、このような高価なサービスをフォークすることができました。

まあ、銀行は商業組織であり、お金を節約しますが、政府機関は大統領の命令の執行を節約せず、100％の実施が確実にあります。

合計ドメイン：127
AAAAレコードを持つ：6
合計：4.72％
