Three Tricks for Working with SOLIDWORKS for Modeling Parts for 3D Printing



The SOLIDWORKS provides a number of tools to help check how tech will be a model in the production of the additive method.

How to make sure that the designed model is technologically advanced


In SOLIDWORKS, validation tools are grouped in the Print3D group .

The Preview tab contains the most important checks: SOLIDWORKS clearly shows which elements will require support, whether the selected wall thickness matches the width of the nozzle used, whether the part is optimally oriented in the printer, etc. Test results are visualized on the right side of the screen.

How to check the gaps if you want to print the finished assembly


In some cases, you can immediately print a node from several parts on a 3D printer, rather than each part individually. When designing such models, you need to make sure that, firstly, individual parts will not stick to each other, and secondly, they will not be able to move relative to each other. Therefore, it is important to leave the correct gap between the parts of the printed product, and the width of this gap for each case will be individual.

The SOLIDWORKS to check clearances has a special feature. Suppose we have two parts that are connected using pins or bolts.



The gap check function instantly shows that for the selected 3D printer with the specified print settings, the gaps will be observed.



How to save on supports


When a designer models a part for manufacturing in an additive way, he has to think in other categories. For example, to think not only about how to design an overhanging fragment of the model, but also about what this element will be based on during printing. SOLIDWORKS helps you find the best solution for both the structural element and its support.

On the left side of the figure, we see that several elements need supports, which means that the material consumption will increase and it will take time to remove the supports after printing. If on the cylinders special chamfers are provided, you can do without supports, and less material will be spent on the structural elements themselves.
