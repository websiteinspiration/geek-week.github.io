前のパートでは、8ビットのレジスタ変数の操作について詳しく説明しました。以前の投稿を見逃した場合は、読むことをお勧めします。その中には、記事の例を自分で試すためのライブラリへのリンクがあります。以前にライブラリをダウンロードした方は、最新バージョンをダウンロードすることをお勧めします。ライブラリは常に更新されており、一部の例は古いバージョンのライブラリでは動作しない場合があるためです。


残念ながら、以前に考慮されたレジスタ変数のビット深度は、メモリポインタとして使用するには明らかに不十分です。したがって、ポインターの説明に直接進む前に、別のクラスのデータ記述について検討します。 AVR Megaアーキテクチャのほとんどのコマンドは、レジスタオペランドでのみ機能するように設計されています。つまり、両方のオペランドと結果のサイズは8ビットです。ただし、連続して配置された2つのRONレジスタが1つの16ビットレジスタと見なされる操作がいくつかあります。そのような操作はほとんどなく、主にポインタを操作するようになっています。


ライブラリの構文の観点から見ると、レジスタペアの操作は、レジスタ変数の操作とほとんど同じです。レジスタペアを操作する小さな例を考えてみます。ここと下のスペースを節約するために、コード生成の特定の機能を説明する必要がある場合にのみ実行結果を示します。

var m = new Mega328();
var block1 = AVRASM.Label;
var block2 = AVRASM.Label;
var block3 = AVRASM.Label;
var arr = m.ARRAY(6);
var ptr = m.MEMPTR();
//   
m.Temp.Load(block1);
m.Temp.Store(arr[0]);
m.Temp.Load(block2);
m.Temp.Store(arr[2]);
m.Temp.Load(block3);
m.Temp.Store(arr[4]);
ptr.Load(arr[0]);
//    
var loop = AVRASM.NewLabel();
m.SWITCH(ptr);
//   ,    
AVRASM.NewLabel(block1);
AVRASM.Comment("  1");
ptr.Load(arr[2]);  //      
m.GO(loop);
AVRASM.NewLabel(block2);
AVRASM.Comment("  2");
m.Temp.Load(block3);
ptr.MStore(m.Temp);  //      
m.GO(loop);
AVRASM.NewLabel(block3);
AVRASM.Comment("  3");
ptr.Load(arr[0]); //      
m.GO(loop);
