Szenarien für die Übertragung eines Mitarbeiters auf Remote-Arbeit:

1. Erstellen einer L2TP über IP Sec VPN-Verbindung


Das Büro des Unternehmens verfügt über ein Zyxel-Gateway. Die VPN-Verbindung wird regelmäßig mit jedem Betriebssystem hergestellt.

Vorteile : mehr freie VPN - Verbindungen zur Verfügung

Minuses : clientseitige Konfiguration erforderlich

2. Erstellen einer SSL-VPN-Verbindung


Das Büro des Unternehmens verfügt über ein Zyxel-Gateway.

a) Auf einem Windows-Client-Computer ist ein kostenloser Zyxel SecuExtender-Client installiert. Es funktioniert ohne zeitliche Begrenzung. Die Anzahl der Verbindungen ist durch die Standardanzahl der SSL-VPN-Tunnel begrenzt, die mithilfe von Lizenzen erhöht werden können (siehe Tabelle).

b) Auf einem MacOS-Client-Computer ist ein kostenpflichtiger Zyxel SecuExtender-Client mit einer Testdauer von 30 Tagen installiert. Die Anzahl der Verbindungen ist durch die Standardanzahl der SSL-VPN-Tunnel begrenzt, die mithilfe von Lizenzen erhöht werden können (siehe Tabelle).

Vorteile : Keine clientseitige Konfiguration erforderlich

Nachteile : Bezahlter Client für MacOS. Standardmäßig sind weniger VPN-Tunnel verfügbar.

Die Anzahl der möglichen VPN-Tunnel für jedes der Geräte ist in der Tabelle aufgeführt. Bitte beachten Sie, dass für jedes Gerät eine bestimmte Anzahl kostenloser Verbindungen verfügbar ist, für die keine zusätzlichen Lizenzen erworben werden müssen.
