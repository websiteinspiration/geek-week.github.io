1. Juni um 21:00 Uhr Sehen Sie sich die neue Version der Show mit dem Programmkomitee der Hydra-Konferenz "Hydra Heads" an. Die führenden Alexei Fedorov und Vitaly Aksenov werden mit Sergey Kuksenko sprechen. Sergey ist ein Java Performance Engineer, der seit Version 1.0 mit Java arbeitet. Die vorherige Ausgabe von Hydra Heads ist bereits online. Sein Gast war Roman Elizarov, ein Teamleiter bei JetBrains, der bei Kotlin an Coroutinen und Bibliotheken arbeitet.

Bericht von Anton Weiss / „Kenne dich und deinen Feind: CI / CD im Königreich der Cloud Native“
Start: 2. Juni um 18:00 Uhr (Moskauer Zeit) Füge
dieses Ereignis deinem Kalender hinzu - .ics | Google Kalender

Am 2. Juni um 18:00 Uhr wird der IT-Futurist, Experte für technische Ausbildung, Sprecher von DevOops 2020 Moskau, Anton Weiss, einen 30-minütigen Online-Bericht über CI / CD geben. Darin wird er komplexe Fragen zur kontinuierlichen Bereitstellung beantworten, insbesondere für komplexe native Cloud-Systeme mit vielen Benutzern und Umgebungen:

  • Überprüfe ich alle Filialen?
  • Cloud-Dienste oder Host nutzen?
  • Was ist wichtiger - Einfachheit oder Flexibilität?
  • Ist die Lieferantenbindung eine echte Bedrohung?
  • Ist GitOps jetzt massiv genug? Welche Maßnahmen zur Informationssicherheit sind wirklich erforderlich?

Nach dem Gespräch wird Anton mit dem leitenden Mikhail Druzhinin, technischer Manager bei Amazon und Trainer für Architektur und Java-Entwicklung, sprechen. Michael wird Anton die interessantesten Fragen aus dem Telegramm-Chat der DevOops-Konferenz stellen.
