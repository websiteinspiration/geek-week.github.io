宇宙衝突に関するVRゲームの作り方

ステージ、エラー、結論



１年前にビーバーズブラザーズに来ました。会社が新しいサービスを立ち上げたときに私は呼ばれました-ビジネスのためのVR / ARソリューションの開発。テクノロジーを開発し、私たちの能力を理解するために、仮想ゲームを作ることにしました。実際、ゲームにはすべてがあります。魅惑的な世界、スクリプトを魅了する鮮やかなストーリー、ユーザーとの対話、複雑な技術的な実装。したがって、そのようなパイロットは私たちにとって二重に興味深いものでした。



私たちは半年間ゲームを開発してきました。これはバーチャルリアリティに関する最初のチームプロジェクトであり、私たちは踏むことができるすべてのレーキを踏みました。結果として、私たちは途方もない経験を得て、起こり得るすべての間違いを解決しましたが、それでもOculusストアのゲームAstro Collapseをリリースしていませんでした。



この記事では、VRゲームを作成する主な段階について説明し、開発者とその顧客に実際的な推奨事項を示します。間違えないでください。半年ではなく3か月でゲームが終了します。



ゲームについて



Astro Collapseは、バーチャルリアリティのシングルプレイヤースペースシューティングゲームです。小惑星の巨大なシャワーが地球に近づいています。人類の文明は破壊の危機に瀕しています。人々はシャトルに急に避難し、惑星を去りました。あなたは、シャトルを伴い、小惑星から進路を空ける軍艦の射手です。あなたは船の船長の命令に耳を傾け、装甲貫通機関銃、強力なプラズマガン、レーザー銃から小惑星を破壊する必要があります。





ゲームの目的：できるだけ多くの船を救い、地球を守る



アイデアを探す



プロジェクトに取り組み始めたとき、爆発的なコンセプトはありませんでした。私たちは経験を得るために仮想ゲームを作りたかっただけです。最初の3つのアイデアは偶然に思いついたものです。



チューブ



私のInstagramフィードでは、友達が丘から「ベーグル」に乗る様子を目にしました。そして私は、仮想山の頂上から彼らの上にプレーヤーを発射してみませんか？！





ゲームの最初のテストで、私たちのコピーライターは彼のバランスを失い、ほとんど落ちました。



カート



2月23日を記念して、会社の女の子たちはゴーカートを貸してくれました。高速道路を急いで走ったとき、仮想世界でも運転するのは素晴らしいことだと思いました。したがって、2番目のアイデアが出ました。





ゲームのプロトタイプは単純なメカニズムでした。ボックスを回って壁にぶつからないようにしてください。



地球を救う



ここでは非常に簡単です。私はインターネットでニュースを読んで、彼らが今宇宙について多く話していることに気づきました：Ilon Mask、火星での生活、宇宙の起源。これがアストロコラプスの物語の始まりです。





スペースシューティングの最初のプロトタイプ：最初は遅くて退屈だっ



たため、3週間かけてすべてのプロトタイプを開発しました。大きなアイデアがなかったため、外出先ですばやく解決策を作成する必要がありました。ゲームのオーディエンスや、彼らが面白いかどうかを知らずにプロトタイプを作成しました。その結果、2つのアイデアとそれらに費やされた2週間はどこにも行きませんでした。そして時は金なり。



あなたがゲームを作る前に、それが何のために、誰のためにあるのかを考えてください。確信のあるアイデアのゲームのプロトタイプを作成します。


ゲームスクリプト



ゲームの最初のバージョンでは、メカニズムは非常に単純でした。脱出するには、プレイヤーは自分の船で飛んでいる小惑星を撃つ必要があります。ゲームはレベルに分かれています。レベルが高いほど、より多くの小惑星が船を攻撃します。



社内でスクリプトをテストしたところ、非常に退屈であることがわかりました。







私たちのエディターはゲームの最初のバージョンをテストしています：



ゲームを開発する前に、変更に時間とお金を浪費しないように、興味深いシナリオを考えてください。


プレイヤーが仮想世界で孤独を感じないように、プロットを開発してオブジェクトを追加することにしました。これで止まった。それは大きな間違いでした。製品に浸っていない子供と大人で最初のバージョンをテストすることは考えていませんでした。チームの意見のみに基づいて、ゲームの2番目のプロトタイプを作成しました。



何千もの小惑星が地球に飛んでいきます。数分で目標に到達します。人を避難させ、地球を守ろうとすることが必要です。小惑星は、避難船がそれらの下に落ちないように撃たれる必要があります。





ゲームの第2プロトタイプのシーン



を吐き出しました。私たちの意見では、ゲームはエキサイティングで面白いことがわかりました！しかし、それほど単純ではありません...



キッズとゲーマーのスマッシングテスト



ゲームの成功を確認するために、公開前に最後のテストを実施することにしました。そのために、従業員の子供たち、その友人、知人をオフィスに招待しました。参加者はさまざまな聴衆から来ました-これはテストの客観性を保証しました。私たちは誰もがゲームを気に入ってくれると確信しており、コメントがあった場合、それは小さなものになるでしょう。すでに最初のテスターがピンクの夢を払いのけ、小さなスパンを作りました。



プレイヤーは何が起こっているのか理解していませんでした：誰を撃つべきか、誰が敵であるか、彼らが通り過ぎていた船の種類。誰もが尋ねました：「別の武器を手に入れる方法とボーナスがない理由？」子供と大人はゲームに対する反応が異なりました。



大人は、ゆったりとした射撃、頭の動きの少なさ、何をすべきか、誰を撃つべきかについての明確な指示を求めて、測定されたゲームを望んでいました。





テスト中の大人のフレーズ：なぜそんなに速い小惑星なのか？なぜそんなにたくさんあるのですか？そして、それは何ですか？あれは誰？これは何のため？



子供たちは陰謀なしでした。彼らは宇宙の混乱の原因を気にしませんでした。彼らは単に動いて静止しているすべてのものを破壊しました。彼らはさらに小惑星、船、そして動きが欲しいと言った。彼らが持っていなかった最も重要なことはゲームの終わりにボスでした。





子供のフレーズ：小惑星がなぜそれほど少ないのですか？ボス小惑星はどこですか？爆発？



完全に否定的なレビューは得られず、「まったく気に入らない、やり直す」という人はいませんでした。しかし、考慮に入れていない詳細の数がわかった。最も重要な欠陥-人々はゲームの計画と彼らの目標を理解していませんでした。



ゲームの3番目のバージョンでは、放送を追加しました。司令官からの命令、人々の悲鳴、カオスとパニックの音。そのため、プレイヤーを大気に浸すことができました。



ストーリーを伝え、人をプロットに紹介するために、私たちは、ゲームで何が起こったのか、そして何が残っているのかを紹介する紹介ビデオを作りました。





ゲームトレーラー



は、プレイヤーが敵と味方である射撃方法、スーパーウェポンの入手方法を学習するトレーニングモードも追加しました。プレイヤーの陰謀と目標が明らかになった。強力な最後の和音-巨大なボスの小惑星が登場しました。



さて、結果に満足しただけでなく、初めてゲームをプレイした人も満足しました。



プロジェクトの最後に最初からやり直す必要がないように、最初のバージョンから対象ユーザーのゲームをテストします。


最後の主な間違い



プロジェクトには当初からマネージャーがいませんでした。ゲームの最初のバージョンをテストしたときに表示されました。彼の前は、発達が遅く、しばしば荒野に行って精神を失っていました。しかし、マネージャーの登場により、状況は変化しました。彼はテストを組織し、チーム内でコミュニケーションを確立し、プロジェクトのタイミングと品質を監視しました。発売から2ヶ月でゲームをリリースしました。



プロジェクトには常に、開発を管理し、チームを最新の状態に保つ伝道管理者が必要です。


要約する



VRゲームを開発するときに知っておくべき重要なこと：



  1. まず、あなたが信じているアイデアを思いつきます。その後、実装します。
  2. ゲームのシナリオを考え直して、対象ユーザーでテストします。
  3. 最初のプロトタイプからゲームの広範なテストを実施します。
  4. チームがリラックスしたり集中力を失ったりしないように、責任のあるプロジェクトマネージャーを割り当てます。


これらの4つの簡単なヒントは、予算と開発時間を半分または3倍に削減するのに役立ちます。


しかし、これでレーキジャンプは終わりませんでした。それでも、Oculusストアでゲームを公開するのは困難でした。少し後で、ゲームの初回リリースに役立つチェックリストを投稿します。そして次の記事では、ゲームデザインの方法と、犬がどこに埋葬されているかを説明します。



それまでの間、Astro Collapseをプレイして、惑星を小惑星から救ってください！ゲームには、Samsung Gear VRメガネが必要です。彼らはスマートフォンで動作します：Samsung Galaxy Note 4、5、7、S6 / S6 Edge / S6 Edge +、S7 / S7 Edge、S8 / S8 +。
