みなさん、こんにちは！今年で3回目のPhpStormのメジャーリリースを発表できることをうれしく思います。

新しいバージョンは、WebサイトまたはToolboxアプリを使用してダウンロードできます。いつものように、30日間の試用版を利用できます。フルバージョンは、PhpStormまたはAll Productsパックのアクティブなサブスクリプションの所有者が使用でき、学生やオープンソースプロジェクトの開発者も無料で使用できます。


パフォーマンスと品質


各リリースでは、パフォーマンスの向上に多くの注意を払っています。しかし、今回はさらに進んで、3か月にわたる質の高いマラソンを開催しました。その間、IntelliJプラットフォームチーム全体がさまざまな側面からIDEを磨くことに集中しました。そして、これが私たちが何とか達成したものです：

  • IDEの起動が速い
  • UIの応答性が向上しました

栄誉に甘んじるつもりはなく、今後のリリースで起動時間の短縮やその他の最適化に引き続き取り組んでいきます。

PHP 7.4


PHP 7.4はすでにリリースされています！そして、それは多くのクールな機能を備えています。以前のリリースでPHP 7.4のサポートのロールアウトを開始しました。PhpStorm2019.3では、言語のすべての新機能が完全にサポートされています。

型付きプロパティ


これはおそらくPHP 7.4で最も期待されている機能であり、PhpStormはバージョン2019.2以降でサポートしています。しかし、小さな問題を修正し、いくつかの境界線のケースをカバーしたので、もう一度見てみましょう。
もちろん、すべての誤用と型の不一致が強調表示さ

ネストされた三項演算子


ほとんどすべてのプログラミング言語では、3項演算子は右から左に数えます。PHPでは、すべてが逆であり、非常に直感的でなく、エラーが発生する可能性がありました。PHP 7.4では、三項演算子の左結合性はサポートされなくなりました。つまり、ネストされた3項式をまったく使用しないか、または使用しますが、ブラケットを使用して実行順序を明示的に指定します。
PhpStorm 2019.3は廃止された式を強調表示し、クイックフィックスで何を意味していたかを明確にすることができます（Alt + Enter-ええ、あなたはすでに知っていますよね？）。

共変戻り値と反変パラメーター


PHP 7.4は多くの不整合を修正し、特に、相続人の戻り値の型を「絞り込み」、パラメーターの型を「拡張」できるようになりました。



非推奨の機能


PHPでは、新しい機能を追加するだけでなく、古い機能を削除して、言語をよりクリーンで作業しやすくします。PHP 7.4には、いくつかの「非推奨」があります。
PhpStorm 2019.3は、注意が必要なコードを強調しています。可能であれば、廃止された式を修正するためのクイックフィックス（Alt + Enter）が提供されます。

WSL


PhpStormは、WSL（LinuxのWindowsサブシステム）環境での開発をサポートするようになりました。つまり、Windows 10でIDEを実行すると、ターゲットのWSL / Linux開発システムになります。WSLからリモートPHPインタープリターを指定し、それを使用してスクリプト、テスト、Composerコマンド、およびデバッグを実行できます。
開始するには、指示に従ってWSLをインストールおよび構成し、選択したLinuxディストリビューションに同梱されていない場合はPHPをインストールします。
