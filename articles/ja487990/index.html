サイバー脅威。2020年の予測：人工知能、雲のギャップ、量子コンピューティング

2019年、サイバーセキュリティの脅威がかつてないほど急増し、新たな脆弱性が出現しました。私たちは、記録的な数の国家支援のサイバー攻撃、身代金キャンペーン、そして過失、無知、不正確な状況の評価、またはネットワーク環境の構成のエラーによるセキュリティホールの増加を目にしました。



パブリッククラウドへの移行は速いペースで進んでおり、組織は新しい柔軟なアプリケーションアーキテクチャに移行できます。ただし、このような移行の利点に加えて、セキュリティと脆弱性に対する新たな脅威も意味します。データの機密性を侵害することの危険性と誤報キャンペーンの結果の深刻さを認識し、組織は個人情報の保護を強化するために緊急対策を講じようと努めています。

2020年のサイバーセキュリティの展望は？人工知能から量子コンピューティングまでの技術のさらなる開発は、新しいサイバー脅威への道を開きます。

人工知能は偽のニュースや偽情報キャンペーンを開始するのに役立ちます


誤報や虚偽のニュースは、企業や組織に悪影響を及ぼす可能性があります。今日のデジタル世界では、人工知能の重要性が増しており、州レベルのサイバー兵器のツールとして使用されています。

偽の画像や動画を生成するディープラーニングアルゴリズムは、より高度になっています。この人工知能の使用は、各被害者の行動的および心理的プロファイルを考慮して、ターゲットを絞ってパーソナライズされた大規模な偽情報または偽のキャンペーンの触媒となります。

愚かさや過失によるデータ漏えいの発生頻度が低くなる


Wall Strert Journalのレポートは、クラウド内のデータ侵害は、適切なサイバー保護対策とツールの欠如から生じていることを示しています。 Garterの推定によると、クラウドインフラストラクチャのギャップの最大95％は人為的エラーの結果です。クラウドベースのセキュリティ戦略は、クラウドテクノロジーの使用のペースと規模に遅れをとっています。企業は、パブリッククラウドに保存されている情報への不正アクセスの危険にさらされています。



記事の執筆者であるラドウェアのサイバーセキュリティ専門家であるパスカルジーネンスの予測によると、2020年には、パブリッククラウドでの不適切な構成によるデータ漏えいは徐々になくなります。クラウドとサービスプロバイダーは積極的な地位を占めており、組織が「攻撃対象」を減らすのを支援することに真剣に取り組んでいます。組織は、経験を積み、他の企業が犯した過ちから学びます。パブリッククラウドへの移行に関連するリスクを回避するために、企業はより適切に配置されます。

量子通信はセキュリティポリシーの不可欠な部分になります。


量子通信を使用して、情報チャネルをデータの不正な傍受から保護する量子力学を使用することは、機密情報や貴重な情報を扱う組織にとって重要な技術になります。

量子暗号の最も有名で開発された領域の1つである量子鍵の配布は、さらに普及するでしょう。私たちは、古典的なコンピュータの対象ではない問題を解決する可能性を秘めた量子コンピューティングの優位性の夜明けです。

量子コンピューティング技術をさらに研究すると、貴重な機密情報を扱う組織の階級に緊張が生じます。一部の企業は、量子通信技術を使用した暗号攻撃から通信を保護するために、これまでにない手段をとらざるを得ません。著者は、2020年にこの傾向が始まると想定しています。

ラドウェアの調査とレポート「The State of Web Application Security」では、Webアプリケーションに対するサイバー攻撃の構成と特性、アプリケーションのサイバーセキュリティを確保する方法、およびマイクロサービスアーキテクチャへの移行の影響に関する最新のアイデアが調査されました。
