それでは、冒険を続けましょう。リンカとコンパイラを考え出した初期化スクリプトはすでに作成しています。 LEDを点滅させるときがきました。この記事では、RCCおよびGPIOブロックの基本について説明し、将来のプロジェクトで使用するいくつかのヘッダーを追加します。行け。

RCC-リセットとクロック制御。プロセッサと周辺機器のクロックを制御するレジスタのブロック。このユニットは、HSI（高速内部）、HSE（高速外部）、PLL / PLLSAI / PLLI2S（フェーズドループロック）のいくつかのクロックソースを制御します。もちろん、RCCブロックレジスタが正しく構成されていれば、これらのクロックソースの3つすべてをプロセッサで使用できます。これについては次の記事で詳しく説明します。

GPIO-汎用入力および出力。 LEDを点滅させるために作成されたレジスタのブロック、つまり、実世界へのポータルであるミクロンのレッグを制御するためのブロックです。おなじみのAtmega328p（Arduino）とは異なり、stm32は脚の幅広いカスタマイズを提供します。たとえば、メガではオープンドレインフット機能は使用できません。また、英国が脚の電圧を上げる速度の制御を設定できます。

だから、まばたきをしましょう、はい。まず、これらのレジスターの構造を記述する2つのヘッダーが必要です。したがって、リファレンスマニュアルを入手し、必要な情報を探して調査します。私はSTM32F746NG-Discoveryに取り組んでいるので、このボードのすべてのコードを処理します。

最初に必要なのは、基本的なこれらのブロックのアドレスです。RCCブロックから始めて、rcc.hヘッダーを作成します。その中で、マクロRCC_BASEを作成し、レジスタブロックの構造を記述し、構造へのポインタを作成します。

  volatile unsigned int AHB2ENR;      
  volatile unsigned int AHB3ENR;      
  unsigned int          RESERVED2;    
  volatile unsigned int APB1ENR;      
  volatile unsigned int APB2ENR;

同じ操作をGPIOレジスタブロックを使用して、ポインターなしで行い、別のマクロGPIO_OFFSETを追加してみましょう。以下でそれについて話します。

gpio.h

#define GPIO_OFFSET 0x400

typedef struct

ここで何が問題で、なぜ構造が必要なのかを見てみましょう。実際には、構造体へのポインターを介してレジスターにアクセスできます。物理的には既に存在しているため、初期化する必要はありません。例えば：

RCC->AHB1ENR = 0;


これはメモリ空間を節約し、時にはそれをまったく必要としないこともあります。しかし、今日の節約についてではありません。

これで2つのヘッダーの準備が整いました int main();。これらのレジスターの助けを借りてジャンプしてを作成する方法を学ぶ必要があります。それはすべて単純であり、完全ではありません。 STM32では、レジスタブロックにアクセスするには、最初にレジスタブロックにクロックを適用する必要があります。そうしないと、データがレジスタブロックに到達します。タイヤの構造を深く掘り下げるのではなく、そのまま言います。ブロックは別のタイヤに配置されます。私たちのユニットはAHB1バスにあります。つまり、AHB1バスの特定のポートを有効にする必要があります。私の場合はポートIです。もちろん、最初にメイン関数が必要です。

スタートアップファイルを少し更新して、int main（）;を追加してみましょう。そしてmain.cファイル自体を作成します

extern void *_estack;

void Reset_Handler();
void Default_Handler();

//   
int  main();

void NMI_Handler()                    __attribute__ ((weak, alias ("Default_Handler")));
void HardFault_Handler()              __attribute__ ((weak, alias ("Default_Handler")));

void *vectors[] __attribute__((section(".isr_vector"), used)) = {
    &_estack,
	&Reset_Handler,
	&NMI_Handler,
	&HardFault_Handler
};

extern void *_sidata, *_sdata, *_edata, *_sbss, *_ebss;



void __attribute__((naked, noreturn)) Reset_Handler()
{


	void **pSource, **pDest;
	for (pSource = &_sidata, pDest = &_sdata; pDest != &_edata; pSource++, pDest++)
		*pDest = *pSource;

	for (pDest = &_sbss; pDest != &_ebss; pDest++)
		*pDest = 0;

    //   
    main();

	while(1);
}

void __attribute__((naked, noreturn)) Default_Handler(){
    while(1);
}
