序文


私はphpで作業しているので、phpQueryライブラリに目を向けました。もちろん、デフォルトでphpに組み込まれているライブラリを含め、他にも多くのライブラリがあることには同意しますが、週末にフリーランスで働く普通のプログラマーには、ある種の奇跡が必要です。幸いにも、怠惰は私たち全員を動かします。1つのチェコの怠惰がphpQueryの作成につながりました。

このライブラリに関するロシア語のドキュメントは見つかりませんでした（たぶん私はひどく見ていましたか？）。初心者からフォーラムへのたくさんの質問を見つけて、英語のドキュメントを読むことができないので、この記事を書くことを考えました。この記事は主に初心者向けに書かれたものであることに注意してください。

実用的 部


上記のデータをすべて取得してみましょう。

//  
$html = file_get_contents("https://-_.com/");

//   dom
$dom = phpQuery::newDocument($html);

//    dom    .product-essential,    find().        .
foreach($dom->find(".product-essential") as $key => $value){

        //  dom    phpQuery.       pq();    ($)  jQuery.
    $pq = pq($value);

    //        .brand-link     "href"    attr();
    $productHref[$key]["brand-href"] = $pq->find(".brand-link")->attr("href");

    //   .     <span class="brand-name">- </span>.
    //    ,   <span>       text();
    $productHref[$key]["brand-name"] = $pq->find(".brand-name")->text();

    //      .
    //    ,      .
    //         <h1>,     <div class="brand-name">
    $productHref[$key]["product-name"] = $pq->find(".product-name h1")->text();

    // PhpQuery    ,    , .
    //      !
    //     .
    $productHref[$key]["product-price"] = $pq->find(".price-info .price-box .regular-price .price")->text();

    //   
    $productHref[$key]["product-description"] = $pq->find(".description .product-description")->text();

    //       .
    //      next();
    //           .

結論


PhpQueryは非常に便利なライブラリですが、残念ながら重すぎます。したがって、要素を確認した後、ドキュメントをアンロードすることをお勧めします。

phpQuery::unloadDocuments();

ライブラリは便利ですが、慣れないことをお勧めします。小さな問題を解決するには、おそらく何よりも良いでしょう。しかし、これはまだ少し古いライブラリです。
