Get best of the week

Plotlyで奇妙なアトラクタを視覚化することは傑作です

詩は非常に美しく、思いやりのある音節であり、日常生活では使用しませんが、そのように楽しむことが大好きです。数学についても同じことが言えます。映画「Pi」では主人公が数学を「自然の言語」と呼び、映画「マインドゲーム」では主人公がそれを「特別な芸術」として語っています。日常の中で、私たちはそれを完全に忘れることができます。

奇妙なアトラクタの外観は、2次元の次元でも異常で魅力的です。Plotlyを使用すると、3次元でそれらを構築でき、「ねじる」ことができ、「飛行」できる「タッチ」の感覚を持つ3Dモデルを非常に簡単に取得できます。

それはずっと昔、大学の2007年のどこかで私は自己組織化の理論に精通し、初めて本でローレンツアトラクター（白黒のイラスト）を目にしました。それから、何かがそのような異常な軌道に沿って動くことができるのは不思議に思えました。私には、世界のほとんどすべてが単一の理論で説明できるという考えそのものに不思議に思えました。

一般的に、すべてはいつものようです-私の世界観は変わりました、人生は続きました、時間が経ちました。そして、最近、私はリンクに出会い、これを見る：

def ChenLee(XYZ, t, alpha, beta, delta):
    x, y, z = XYZ
    x_dt = alpha*x - y*z
    y_dt = beta*y + x*z

#  :
def ChuaAttractor(XYZ, t, alpha, beta, zeta, delta):
    x, y, z = XYZ
    h = zeta*x + (0.5*(delta - zeta))*(np.abs(x + 1) - np.abs(x - 1))
    x_dt = alpha*(-x + y - h)
    y_dt = x - y + z
    z_dt = -beta*y
    return x_dt, y_dt, z_dt
