Qu'est-ce que le HTML?


HTML signifie HyperText Markup Language. Il s'agit du langage de balisage pour les documents sur le World Wide Web (WWW). HTML est un langage standardisé pour le texte formaté. Le navigateur l'interprète et affiche des éléments de pages Web à l'écran.

En quoi consiste le texte au format HTML?


Tout d'abord, le HTML est le contenu et les balises. Les balises vous permettent de spécifier la façon dont le contenu est affiché sur les pages Web.

Qu'est-ce qu'une balise HTML?


Une étiquette est un mot de service spécial placé entre crochets. Il est également appelé «élément HTML». Il existe de nombreuses balises dans le langage HTML et chacune fait quelque chose avec le contenu, qui se trouve généralement à l'intérieur des crochets ou entre les balises.
Si la balise est appariée, la balise <TEG> correspond à </ TEG>.

Tous les tags sont-ils associés?


Non, pas tous. Il existe des balises uniques, par exemple, <img> - pour les images. Dans ce cas, le contenu avec d'autres mots de service (par exemple, pour <img> il peut s'agir de src = "image url") est placé entre les crochets, mais après le mot img.

Quels sont les principaux types de listes utilisés pour créer une page Web?


Lors de la création d'une page Web, vous pouvez utiliser:

  1. Liste des menus (liste des menus - <menu>).
  2. Liste de répertoires (liste de répertoires - <rép>).
  3. Liste ordonnée (liste numérotée - <ol>).
  4. Liste non ordonnée (liste à puces - <ul>).
  5. Liste de définitions (<dl>).


Pourquoi utiliser des listes en HTML?


Les listes rendent les informations plus visuelles et faciles à lire. De plus, à l'intérieur des listes (plus précisément, pour l'affichage arbitraire des éléments de liste), vous pouvez également utiliser différentes balises.

Pouvons-nous écrire des commentaires en HTML? Comment?


Oui nous pouvons. Pour ce faire, le texte du commentaire doit être placé entre <! - et ->.

Pourquoi certains caractères d'une page Web s'affichent-ils parfois de manière incorrecte?


Le plus souvent, les caractères ne s'affichent pas correctement en raison de problèmes d'encodage. C'est la raison - dans les paramètres du navigateur ou du serveur Web.

Qu'est-ce que l'Image Map?


En HTML, Image Map est une technologie qui vous permet de lier des liens vers des fragments d'images. En cliquant sur des parties individuelles de l'image, l'utilisateur peut suivre tel ou tel lien vers différentes pages Web.

Qu'est-ce que l'espace blanc?


L'espace blanc est une propriété qui vous permet de modifier le nombre d'espaces entre les mots lors de l'affichage d'une page Web. Par défaut, le navigateur interprète n'importe quel nombre d'espaces dans le code HTML comme un seul espace.

Quel est le principal avantage de l'espace blanc?


Lors de l'écriture de code HTML, le développeur n'a pas à se soucier des espaces supplémentaires qu'il pourrait saisir: le navigateur supprimera tous les espaces supplémentaires lorsque la page Web sera affichée.
Le développeur peut ajouter des espaces au code, non seulement par accident, mais aussi exprès: pour rendre le code plus lisible.

Puis-je attribuer des valeurs à tous les attributs de balise?


Oui, dans ce cas, le navigateur utilisera les valeurs par défaut. Si vous n'êtes pas à l'aise avec la façon dont il a affiché la page Web, vous devriez peut-être spécifier explicitement les valeurs de certains attributs.

Comment placer la marque de copyright sur une page Web, car elle n'est pas sur le clavier?


Le symbole du droit d'auteur fait référence aux caractères spéciaux. Pour le placer sur une page Web, vous devez taper la séquence & copier ou & # 169 sur le clavier dans le fragment correspondant du code HTML.

Comment créer des liens vers différents fragments (sections) de la même page Web?


Pour ce faire, recherchez le début du fragment dont nous avons besoin dans le code HTML. À cet endroit, créez la soi-disant «ancre» en utilisant l'attribut name de la balise <a> et donnez le lien vers le nom du fragment en utilisant l'attribut href et le signe #:
href = "#link_name_to_fragment"

Pouvons-nous aligner des éléments de liste dans un fichier HTML?


Oui, nous pouvons stocker des éléments de liste directement dans le fichier HTML à l'aide de retraits. L'indentation peut également être utilisée pour toute liste imbriquée à l'intérieur du parent.

Quelle adresse de page Web est généralement considérée comme l'adresse du site?


Il s'agit généralement d'une des pages de niveau supérieur, la page principale. À partir de là, l'utilisateur accède à des pages situées à des niveaux inférieurs. Et leur adresse est le plus souvent formée sur la base de l'adresse de la page principale.

Pourquoi utilisons-nous un texte alternatif (attribut alt de la balise <img>) pour les images?


Un texte alternatif s'affiche lorsque les graphiques sont désactivés dans le navigateur de l'utilisateur ou en cas de problème de chargement d'une image spécifique. Ce texte montre clairement ce qui aurait dû être représenté sur cette image.

Les fichiers HTML peuvent-ils bien fonctionner dans certains navigateurs rares ou de pointe?


Bien sûr, les fichiers HTML peuvent et doivent bien fonctionner dans n'importe quel navigateur, car le navigateur doit se conformer aux normes HTML. Si les développeurs ont décidé de ne pas prendre en charge certaines des fonctionnalités de la norme, cela reste dans leur conscience.

Pensez-vous que l'hyperlien ne peut être que du texte?


Non, un lien hypertexte peut être non seulement du texte, mais aussi, par exemple, un graphique. Cela signifie que nous pouvons convertir l'image en un lien qui dirige l'utilisateur vers une autre page. Un lien hypertexte ne dépend pas du type de contenu qui se trouve «en dessous».

Quels sont les attributs de la balise <li> - un élément de liste?


Il y a deux d'entre eux:

  1. type définit l'apparence d'une puce ou d'un marqueur de liste de puces;
  2. valeur - le numéro à partir duquel la liste numérotée commencera.


À quoi servent les feuilles de style (CSS)?


Les feuilles de style vous permettent de définir des règles claires qui déterminent l'apparence du contenu, divers éléments HTML et la page entière dans son ensemble. De plus, les mêmes règles peuvent être appliquées à plusieurs pages Web à la fois, ce qui simplifie la conservation et, si nécessaire, modifie le style uniforme du site.

Quels types de numérotation dans les listes connaissez-vous?


Les valeurs suivantes peuvent servir d'éléments de numérotation:

  • Chiffres arabes (1, 2, 3, ...);
  • lettres majuscules latines (A, B, C, ...);
  • lettres latines minuscules (a, b, c, ...);
  • nombres romains majuscules (I, II, III, ...);
  • chiffres romains en minuscules (i, ii, iii, ...).


Comment définir différentes couleurs pour les fragments de texte d'une page Web?


Pour définir différentes couleurs pour les fragments de texte, vous devez placer chacun des fragments dans la balise de police appariée:

<font color=“color”>-    “color”</font>


Où sont stockés les codes de caractères numériques en HTML?


Dans la table ASCII. Lorsqu'une page Web s'affiche, le navigateur prend les codes de caractères numériques du document HTML et les remplace par le caractère correspondant du tableau.

Quels sont les avantages du regroupement de plusieurs cases à cocher (éléments de case à cocher)?


Le regroupement des indicateurs présente plusieurs avantages:

  • il permet de mieux les organiser et les structurer;
  • cela vous permet d'accéder à des indicateurs spécifiques à partir du groupe en utilisant id;
  • cela vous permet de créer un autre groupe de drapeaux sur la même page Web sans craindre de confondre les noms ou identifiants des drapeaux.


Comment le chevauchement des balises affecte-t-il l'affichage du contenu?


Les balises superposées en HTML ne reconnaissent que la première balise. Mais de tels problèmes ne se produisent que si le navigateur essaie de reconnaître les balises sans texte.

S'il n'y a pas de texte entre les balises, quel sera le résultat? Donne un exemple?


S'il n'y a pas de texte entre les balises, le contenu sera impossible à formater. Par exemple, les balises simples, telles que <img>, ne nécessitent pas de saisie de texte et, par conséquent, la mise en forme n'est pas requise dans ce cas.

Comment spécifier les couleurs des bordures de tableau?


Cela peut être fait dans la feuille de style, et si elle est manquante, la couleur du texte sera utilisée.

Pouvons-nous créer un lien qui mène à une autre page Web?


Oui, vous devez utiliser l'attribut href pour cela.

Les feuilles de style peuvent-elles aider à aligner l'image et à déterminer la position du texte par rapport à celle-ci?


Oui, une feuille de style peut contenir des règles de positionnement du texte et des images.

Un seul lien hypertexte peut-il mener à différentes pages?


Non, un seul lien hypertexte ne peut mener qu'à une seule page Web dont vous fournissez l'adresse.

Y a-t-il une différence entre une liste à puces et une liste de répertoires et de menus?


Oui, la différence est qu'une liste non ordonnée a des attributs pour changer le style de marqueur.

Comment changer la couleur du marqueur?


La couleur du marqueur correspond généralement à la couleur du premier caractère de la liste. Si vous changez la couleur du premier caractère à l'aide de la balise <{, la couleur du marqueur changera également.
Si nous voulons définir la couleur du marqueur dans la liste sans changer la couleur du texte, nous devons mettre la balise <span> à l'intérieur du <li>, et mettre le texte à l'intérieur. Et puis la couleur du marqueur sera déterminée par la couleur de la propriété de style du sélecteur LI, et la couleur du texte par la propriété de couleur du sélecteur SPAN.

Est-il possible d'introduire des restrictions sur la taille des champs de texte en HTML?


La taille standard est de 20 à 25 caractères. Mais en utilisant les attributs size ou maxsize, vous pouvez définir d'autres restrictions. Par exemple:

<input type="text" size="5" />


Quelle est la taille maximale d'un champ de texte?


La largeur de la fenêtre du navigateur.

Que se passe-t-il si vous définissez la taille du champ de texte sur zéro?


Le navigateur utilisera la valeur par défaut.

Quelles sont les similitudes entre les attributs de bordure et de règles?


Si l'attribut border est défini sur une valeur non nulle, la bordure par défaut d'une cellule d'une épaisseur d'un pixel sera ajoutée automatiquement. Si l'attribut rules est ajouté à la balise <table>, l'attribut border ne sera pas inclus.
Mais en même temps autour de chaque cellule (la même que la bordure), un cadre d'une épaisseur d'un pixel par défaut apparaîtra, formant ainsi une grille.

Qu'est-ce qu'un chapiteau? Comment pouvons-nous appliquer cela?


Marquee aide à personnaliser le défilement du texte sur une page Web. Pour activer le défilement, vous devez utiliser la balise <marquee>.

Que fait la balise <br>?


La fonction de la balise <br> est de séparer les morceaux de texte les uns des autres.

Existe-t-il un autre moyen de fractionner du texte sans utiliser <br>?


Oui, il existe d'autres façons de diviser le texte. Vous pouvez utiliser la balise <p> ou la balise <blockquote>.

Le texte peut-il être affiché en dehors de la fenêtre du navigateur?


Par défaut, le texte ne peut pas être affiché en dehors du navigateur, mais si le texte fait partie d'une cellule de tableau avec une largeur prédéterminée, il peut dépasser la fenêtre du navigateur.

Quelle est la différence entre les liens actifs et inactifs?


Le lien actif a le focus, c'est-à-dire que la souris est dessus. Dans d'autres cas, l'accent est mis ailleurs.

La feuille de style a-t-elle des restrictions sur le nombre de règles?


Non, les feuilles de style ne limitent pas le nombre de règles pour un sélecteur. Cependant, chaque nouvelle règle doit être séparée par des points-virgules.

Les feuilles de style ont-elles une hiérarchie de règles?


Oui, il existe une hiérarchie qui comprend quatre niveaux (de la priorité la plus élevée à la moins prioritaire):

  1. Style en ligne: s'attache directement à l'élément pour lequel le style est créé. Exemple:
    <h1 style="color: #ffffff;">
  2. Un identifiant est un identifiant unique pour les éléments de page, tels que #navbar.
  3. Classes, attributs et pseudo-classes: ces catégories incluent .class [attribut] et pseudo-classes telles que: hover ,: focus, etc.
  4. Éléments et pseudo-éléments: cette catégorie comprend les éléments et pseudo-éléments, tels que h1, div ,: avant et: après.


Peut-on regrouper différents sélecteurs avec des noms de classe différents?


Oui, pour cela, il vous suffit d'écrire les noms de classe avec une virgule.

Puis-je joindre un fichier CSS externe dans un navigateur?


Non, un fichier CSS externe peut être inclus à l'aide de la balise <link /> dans le document HTML lui-même.

Une propriété de type style liste peut-elle affecter un paragraphe?


Non, la propriété de type liste ne peut pas affecter un paragraphe, elle l'ignorera très probablement.

Qu'est-ce qu'un élément canvas?


L'élément HTML5 canvas peut être utilisé pour insérer des images, des dégradés et des animations complexes. Il peut également aider à créer des images 2D directement à l'intérieur du code source d'un document HTML.
