Python 3.9の新機能

最新のPythonイテレーションに含まれる最高の機能のレビュー。



時が来て、Pythonの新しいバージョンのリリースは避けられません。現在ベータ版（3.9.0b3）ですが、まもなくフルバージョンのPython 3.9が表示されます。

最新の機能のいくつかは非常に興味深いものであり、リリース後にそれらの使用を見るのは素晴らしいでしょう。以下を検討します。

  • 辞書の組み合わせ演算子
  • タイプヒント
  • 2つの新しい文字列メソッド
  • 新しいPythonパーサーはとてもクールです

まず、新機能とそれらの使用方法を見てみましょう。

辞書の組み合わせ


構文を使用した新しいお気に入り機能の1つ。我々は2つの辞書を持っている場合 aとb、我々が結合する必要があることを、我々は今、労働組合の演算子を使用します。

マージ演算子「|」があります。

a = {1: 'a', 2: 'b', 3: 'c'}
b = {4: 'd', 5: 'e'}
c = a | b
print(c)

[Out]: {1: 'a', 2: 'b', 3: 'c', 4: 'd', 5: 'e'}

そして、|=ソース辞書を更新する更新ステートメント「」：

a = {1: 'a', 2: 'b', 3: 'c'}
b = {4: 'd', 5: 'e'}
a |= b
print(a)

[Out]: {1: 'a', 2: 'b', 3: 'c', 4: 'd', 5: 'e'}

辞書に共通のキーがある場合、2番目の辞書のキーと値のペアが使用されます。

a = {1: 'a', 2: 'b', 3: 'c', 6: 'in both'}
b = {4: 'd', 5: 'e', 6: 'but different'}
print(a | b)

[Out]: {1: 'a', 2: 'b', 3: 'c', 6: 'but different', 4: 'd', 5: 'e'}

反復を使用した辞書の更新


「|=」演算子のもう1つの興味深い動作は、リストやジェネレータなどの反復オブジェクトを使用して、新しいキーと値のペアでディクショナリを更新する機能です。

a = {'a': 'one', 'b': 'two'}
b = ((i, i**2) for i in range(3))
a |= b
print(a)

[Out]: {'a': 'one', 'b': 'two', 0: 0, 1: 1, 2: 4}
