Comment déterminer l'opérateur et la région par numéro de téléphone



Il est utile de savoir quel opérateur téléphonique le client a choisi et dans quelle région. Ensuite, vous pouvez répartir géographiquement la clientèle et ne pas appeler la nuit. Ou effectuez des promotions avec les opérateurs. Et certains de nos clients économisent: pour chaque opérateur, choisissez l'agrégateur SMS le moins cher.

Par conséquent, " Dadata " trouve depuis longtemps l'opérateur par numéro de téléphone. L'algorithme est simple - faites-le et utilisez-le. À moins qu'en 2013, lorsque «l'esclavage mobile» a été aboli, j'ai dû ajuster légèrement le processus. Dans cet article, je vais vous expliquer comment tout fonctionne pour nous.

Il y a une grande difficulté dans le processus - accéder à la base de données des numéros transférés. Il n'est pas délivré aux particuliers et les entreprises auront du mal. À ce sujet - dans une section distincte.

Brisez le numéro de téléphone


La première chose à faire est de démonter le téléphone en trois parties:

  • code du pays - pour la Russie, il est +7 ou 8;
  • code opérateur - 3 à 5 chiffres (en fait, ce n'est pas un code opérateur, mais un code de zone de numérotation. Je vous le dirai plus tard);
  • le numéro de téléphone réel est composé de 5 à 7 chiffres.


Le code de zone de numérotation pour un numéro de mobile se compose toujours de trois chiffres. À l'arrêt - de trois, quatre ou cinq. Dans tous les cas, les trois premiers chiffres suffisent pour déterminer la région par numéro de téléphone.

Si vous ne savez pas quelle partie du numéro se compose de quels chiffres, alors ce sera difficile. La solution consiste à mettre un masque pour que les utilisateurs saisissent immédiatement les nombres en plusieurs parties. Ou démontez-le après, la bibliothèque Google pour travailler avec les téléphones fera face à la tâche .

Trouver un opérateur par numéro de téléphone en consultant le plan de numérotation


Le principal document russe, selon lequel les opérateurs reçoivent des numéros de téléphone, est le plan de numérotation Rossvyaz. Il a clairement énoncé toutes les dispositions: quels codes et numéros à quels opérateurs dans quelles régions appartiennent. Ce qui est bien, la référence Rossvyaz est gratuite sur rossvyaz.ru/deyatelnost/resurs-numeracii/vypiska-iz-reestra-sistemy-i-plana-numeracii .


Le guide a été téléchargé en deux formats à la fois, ce qui est très agréable de la part de Rossvyaz. Une fois par mois

, quatre fichiers sont mis à jour dans le plan de numérotation: trois pour les numéros fixes et un pour les numéros mobiles. La différence entre le contenu des fichiers est notable si vous regardez les préfixes dans les noms:

  • «ABC-» signifie que le fichier contient des nombres fixes. En fait, «ABC» est une désignation de zones de numérotation définies sur une base géographique;
  • «DEF-» est un fichier avec des numéros de mobile. Le préfixe signifie des zones de numérotation qui ne sont pas définies géographiquement.

C'est déjà clair ici: l'annuaire vous indiquera non seulement l'opérateur, mais aussi le type de téléphone:

  • pour stationnaire, le code commence par 3, 4 ou 8;
  • sur mobile - à partir de 9.

Il existe également des mobiles directs - extérieurement, ils ne peuvent pas être distingués des stationnaires. Je ne connais qu'une seule façon de découvrir le mobile direct: trouver son opérateur et le comparer avec la liste des opérateurs mobiles en Russie. Mais même sans garantie, car certains opérateurs mobiles en même temps - et les opérateurs fixes.
Le numéro +7 495 999-99-34 peut être directement mobile. Ou peut-être que ce bureau achète une connexion fixe à un opérateur mobile - seul l'opérateur le sait.

Mais je m'éloigne du sujet, revenons au plan de numérotation. À l'intérieur des fichiers, tout est très simple - une table avec six champs.


L'ordre des champs est le suivant: code de zone ABC ou DEF, début de plage, fin de plage, capacité, opérateur, région. La capacité est simplement le nombre de numéros inclus dans la gamme.

Rechercher selon le plan de numérotation d'une région et d'un opérateur est un plaisir.

  1. Nous obtenons le code de zone de numérotation à partir du numéro souhaité.
  2. À l'aide du code, sélectionnez les plages appropriées dans le répertoire.
  3. Nous ne quittons du téléphone que ce qui suit après l'indicatif régional - le numéro réel.
  4. Nous recherchons dans quelle plage le nombre se situe.
  5. Succès! Nous avons trouvé la région, l'opérateur et le type de téléphone.


Le numéro +7 301 220-22-47 appartient à Mobilon Telecommunications LLC. Région - République de Bouriatie

Parfois, le téléphone ne rentre dans aucune plage. Cela signifie que le numéro est incorrect. Il n'y a aucune exception. À ce sujet, je dois dire que la bibliothèque Google susmentionnée est perforée - elle approuve un téléphone apparemment fidèle, ne le compare pas au plan de numérotation.

Auparavant, à cette étape, la recherche était terminée. Mais en 2013, les autorités ont adopté une loi abolissant «l'esclavage mobile». Depuis lors, les numéros de téléphone sont cloués aux opérateurs pas si serrés.

Maintenant la loi le permet: changer d'opérateur, en gardant le numéro. Mais attention, uniquement dans la région. Vous ne pouvez pas retirer le téléphone de la région fournie par Rossvyaz. Il est impossible de «casser» les plages de distribution des nombres. Par conséquent, si seule la région du numéro de téléphone présente un intérêt, vous ne pouvez pas lire plus loin. Et je vais continuer et vous dire comment trouver les numéros d'opérateur modifiés.

Accédez à la base de données des numéros transférés


Il s'agit d'une étape bureaucratique, respectivement, la plus difficile. Et pour la plupart des entreprises, c'est complètement infranchissable, malheureusement.

Les numéros qui ont changé d'opérateur sont stockés dans la base de données des numéros d'abonné transférés (BDPN). Son opérateur n'est plus Rossvyaz, mais le  Central Research Institute of Communications .

Le problème est que l'accès au DDP est très difficile. Bien sûr, si vous n'êtes pas un opérateur mobile, un sujet d'un système de paiement national, une structure de pouvoir ou une autre agence.


Tout ce qu'un utilisateur ordinaire peut faire est de vérifier manuellement un numéro spécifique.

Nous avons accédé au BDPN en 2013 en tant qu '«organisation différente». Mais depuis lors, les règles semblent s'être durcies. Même le formulaire de demande a beaucoup changé, le dernier se trouve sur le site Web de l'Institut des communications. Par conséquent, je ne peux rien conseiller ici, hélas.

Puis il a téléphoné pour la fidélité. "

Vérifiez si le numéro est allé à un autre opérateur


Si vous avez accès à la base de données officielle des numéros portés, alors c'est facile. Nous vérifions si l'abonné a changé d'opérateur, abandonnant les entraves de «l'esclavage mobile».

L'Institute of Communications télécharge le BDPN sur le serveur SFTP dans trois dossiers.


Le premier dossier contient une référence complète. Dans le second - mises à jour incrémentielles. Et le troisième stocke les numéros retournés aux "propriétaires"

  •   PortAllNew . ,  00:00  ;
  • PortIncrementNew — .   ;
  • ReturnIncrementNew — , «» . .

Dans les dossiers, les données de tous les opérateurs dans toutes les régions sont stockées sans division par fichier. Des fichiers sont nécessaires pour distinguer les mises à jour.


La date et l'heure de la mise à jour sont insérées dans le nom des incréments. Si vous souhaitez recevoir les numéros transférés le plus rapidement possible - vous y êtes. S'il n'y a pas d'urgence, il est plus facile de télécharger le répertoire complet une fois par jour.

Format de fichier - CSV, compressé en ZIP. À l'intérieur - il n'y a nulle part plus facile.


Il y a trois champs dans les fichiers de répertoire: nombre, opérateur et nombre de lignes. De plus, le troisième champ n'est rempli que pour la première ligne - une solution élégante L'

historique n'est pas stocké dans le BDPN: tous les jours tous les numéros transférés sont téléchargés sur le serveur. Si le téléphone n'est pas dans le répertoire complet ou par incréments par jour, il obéit au plan Rossvyaz. Par conséquent, pour nos besoins, le dossier avec les numéros retournés estReturnIncrementNew  - pas nécessaire.

Chaque jour, nous téléchargeons un nouveau BDPN depuis le serveur. Tout client SFTP convient à cela: WinSCP ou FileZilla FTP Client conviennent à Windows et Nautilus à Ubuntu. Une erreur classique dans ce cas est de laisser le FTP par défaut dans les paramètres au lieu du SFTP défini. Vous pouvez donc souffrir longtemps, ne répétez pas nos erreurs.

Lorsque le répertoire des numéros transférés est à portée de main, il reste à y rechercher le numéro sous enquête. Il y a deux options:

  • nombre trouvé dans le BDPN. Dans ce cas, nous changeons l'opérateur du plan Rossvyaz à un nouveau;
  • le nombre n'est pas dans le BDPN - nous laissons l'opérateur tel quel.

C’est de la sagesse.
