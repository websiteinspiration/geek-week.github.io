Rおよびテキストマイニング技術を使用したロシアのラップシーンの比較。Noize McとKasta対PharaohとMorgenshtern

R.テキストマイニング。ラップ


多くのコンテンポラリーラップアーティストの人気は、私や他の「オールドスクール」のフォロワーにとって謎のままです。誰がより上手で、その歌詞がより興味深く、より多様な音楽が多くのインターネットユーザーの心を占めているかについて、常に議論が交わされています。これらの論争を言葉だけでなく事実で確認するために、プログラミング言語Rを使用して、4人のロシアのラップアーティストのテキストを分析しました。

それらのいくつかは2000年代初頭にめちゃくちゃ人気がありました。現在、彼らはすべてリスナーを引き付けていますが、残念ながら、彼らはますます少なくなっています。そして、2つは現在人気のピークにあり、大多数の若い聴衆を魅了しています。そして、私のさらなる分析は、彼らが使用する語彙を考えると、この事実は喜びを引き起こさないことを示します。私が使用した曲のアーティストは、Caste、Noize-Mc、Pharaoh、Morgensternであるため、誰が誰であるかを知るのは簡単です。カーストとノイズを「古い学校」、ファラオとモルゲンシュテルンを「新しい」と呼ぶことは誰もが理解していると思います。

分析アルバム


分析のために、アーティストがリリースしたすべてのスタジオ公式アルバムを選択しました（アルバムに関する情報は、サイトwww.wikipedia.comから取得したもので、すべての資料は最後にあります）。

  1. カスタ：「水よりもラウド、草よりも高い」; 「目のフラッシュ」; 「四つの叫び声」; 「欠陥は明らかだ」-74曲。
  2. Noize Mc：「グレイテストヒッツVol。1 "; 「最後のアルバム」; "新しいアルバム"; 「Protivo Gunz」; "錯乱"; "ハードリブート3.0"; "キングオブザヒル"; 「ヒポペラ：オルフェウスとエウリュディケ」-160曲。
  3. ファラオ： "The Wadget"; フロラ "Dolor"; 蛍光体 「ピンク・フロイド」; プネラル 「ルール」-95トラック。
  4. モルゲンシュテルン：「知られる前に」; 「笑顔、あなたはばかです！」; 「Legendary Dust」-30トラック。

私は前述のアーティストを特別に選択しました。作品に少しでも精通している人でも、テキストが非常に異なることに同意し（Casta + Noise vs Pharaoh + Morgenstern）、それらを互いに比較するのは興味深いでしょう。論理的な問題が発生します。カーストの4つのアルバムとファラオの8つのアルバムを客観的かつ正確に比較するにはどうすればよいですか。すべてが非常に単純です。後で説明するいくつかの操作を行うと、単語の量はほぼ同程度になります。結局のところ、誰もが知っているように、量は品質と同じではありません。
言葉そのものを収集するために、私はgenius.com WebサイトとそのAPIを使用しました。幸い、サービスの開発者は、その後の分析のためにデータベースから歌詞（アーティスト、アルバム）を簡単に抽出できるようにするオープンアプリケーションプログラミングインターフェイス（API）を提供します。

すべての分析は、Rプログラミング言語を使用して実行され、加えてステミング（特定のソースワードの単語ベースを見つけるプロセス）のpythonが使用されました。これは、Rおよびmystemプログラムでのエンコードに対応できなかったためです（Windows 10はUTF-8と友達になることを好みません）とR、彼らはアップルOSまたはLinuxを使用してそのような問題が発生しないと言います）。

処理前。テキストを閲覧します。単語数


ライブラリを使用して歌詞をダウンロードするには、「天才」。このパッケージ「genius_album」の関数を使用すると、アルバムのすべてのテキストを一度に簡単にダウンロードできます。すべてのアーティストが常にすべての歌詞を利用できるわけではないため、注意して慎重に確認してください。それらの一部は手動で追加する必要がありました。ダウンロード後、曲全体で（代名詞、前置詞、助詞などとともに）いくつの単語が使用されているかが興味深くなりました。次に、これらの数値を既に処理されたステミングとストップワードの結果と比較します。使用されている単語数に対するアルバムとトラックの数の比率を理解しやすくするために、この情報をもう一度複製します。

  1. Noize Mc-8アルバム、160トラック。
  2. Casta-4アルバム、74トラック。
  3. ファラオ-7アルバム、95トラック。
  4. Morgenstern-3アルバム、30トラック。



興味深いことに、ファラオとノイズはほぼ同じ数のアルバム（それぞれ7枚と8枚）を持っていますが、グラフからわかるように、アルバムの品質は曲の数と語彙の豊かさの両方で非常に異なります（57962対24184）。

この違いを最小限に抑え、比較をより正確にするために、平均的なアーティストが曲の1つで使用する単語数を計算しました。

  1. Noize Mc-362ワード。
  2. カースト-388語。
  3. ファラオ-254語。
  4. モルゲンシュテルン-273ワード

このような比較は条件付きであり、概算であることは明らかですが、数値はそれ自体を物語っています。

そして、これが各アーティストの上位10の単語の外観とこれらの単語への参照の数です。





予想どおり、「上位の単語」は処理されない場合、結果を反映せず、特別な意味的負荷を持たない前置詞、代名詞、および接続詞です。したがって、この段階では、興味深いことや予期しないことは何も起こりませんでした。

次のステップは、分析のためのテキストの処理と準備でした。ステミングのプロセスは、YandexのPythonのmystemプログラムを使用して実行されました。これは誰でも利用できます。このステップは、アーティストが使用するユニークな単語の数と、彼らのテキストでロシア語がどれだけ広く使用されているかを理解するために行われました。結局のところ、同じ単語を異なるケースで数回数えるのは間違いです。これは、歌手の語彙の幅ではなく、歌手の多様性と説得力を示しています。

また、より代表的な結果を得るためには、感情的および意味的な負荷をかけないストップワード（前置詞、代名詞、助詞など）を取り除く必要があります。残念ながら、Rパッケージには、ロシア語のストップワードを含む優れたライブラリはありません。作者本人がこれかストップワードかストップワードか、削除すべきかを判断しなくてはならないという事実に注目したい。この種類の辞書を常に注意深く見直して、適切で有用な単語を除外しないようにしてください。ストップワードパッケージはかなり多くの言語をサポートしていますが、私は自分の洗練された外部リソースからの単語を使用することを好みました。

処理後


グラフからわかるように、ストップワードをスタンプして削除すると、単語数が大幅に減少しています。元の最も人気のある単語のほとんどすべてが停止したことを考えると、これは当然のことです。



一般に、ストップワードをスタンプして削除した後に残ったワードの数は、最初の数に対するパーセンテージとして、すべてのユーザーにとって実質的に同じです。それらがグループで等しいことは注目に値します。 「古い学校」では55〜58％、「新しい学校」では46〜50％です。
非常に重要で興味深い情報は、各アーティストが持つユニークな単語の数です。ノイズの場合、これは8891ワード、カースト5307、ファラオ3899、モーゲンスターン1242です。語彙を少し増やしたいが、本を読みたくない人は、Noize McとCasteを聞くことができます。

もちろん、多くの人は、処理後、現在どの単語がリードしているかに関心があります。私は各アーティストの上位10ワードをグラフィックスで提示





します。確かに、読者の多くはアスタリスクの付いた言葉に悩まされていました。ファラオとモルゲンシュテルンはテキストに非常に冒とく的な言葉が多く含まれており、私の個人的な意見では、テキストの完全な構造とその認識にかなり悪い影響を与えています。これらの2人のパフォーマーは、2番目の位置に同じ単語があります。彼らの音楽の精神と文化を完璧に表す言葉。少し後、出演者の歌詞のなかでどんな感情のトーンがつながっているのかを明確に示します。

普通の言葉。単語比較


情報をより視覚的にするために、「wordcloud」パッケージの「comparison.cloud」関数を使用して、パフォーマーのすべての単語を1つのチャートに配置しました。これにより、比較と認識が容易になります（マットがどのように目立つかを確認できます）。棒グラフで単語を表示することは、問題が多くなる可能性があります。それらの数が増えると、多くのスペースが必要になるからです。また、同じ名前のパッケージの優れた関数は「wordcloud2」です。単語にカーソルを合わせると、その使用頻度を示すウィンドウが表示されます。



アーティストは同じ言語を使用して曲を書いているため、アーティストに分割することなく、アーティストが最も頻繁に使用する言葉を見るのは興味深いでしょう。このグラフには、wordcloudパッケージのcommonality.cloud関数が使用されました。フォントサイズは、テキストで単語を言及する頻度に対応しています。



感傷的なテキスト分析


映画、本、歌にはそれぞれ独自のムードがあり、それが聴衆やリスナーに伝えられ、彼らに影響を与えます。新旧の学校の出演者が生徒にどのような気分で放送しているのかを見るのは興味深いことです。ミュージシャンの歌には、「ネガティブ」、「ポジティブ」、「ニュートラル」のカテゴリの単語を分析するとわかります。予想通り、ロシア語の場合、Rの単語の感傷分析を備えた高品質の辞書はありません（誰かがこれを知っている場合は、共有してください）。そのため、アップグレードでは外部のものを使用する必要がありました（テキストの最後にある辞書へのリンク）。

辞書にすべての単語が対応しているわけではありませんが、もちろん少し悲しいですが、英語ではそのような問題は実際には発生しません。したがって、私は最も頻繁に繰り返される単語の感情的な色付けを示すことにしました。聞き手が最も頻繁に聞くのはこれらの言葉であり、聞き手に最も強力な影響を与え、歌全体の認識を決定するのはそれらの言葉です。一般に、読者がすべての作者の作品に少しでも精通している場合、彼は驚かないでしょう。まあ、誰かのために分析された名前が新しい場合は、どうぞ歓迎してください、彼らの仕事に精通してください。以下にグラフを示します。すべてのアーティストについて、最も頻繁に使用される単語が表示されます。

モルゲンシュテルン。単語の繰り返し率が10回を超えている。赤い柱の数は非常に多く、これらの言葉が何であるかを整理すると、このアーティストが聴衆に伝えているメッセージから二重に悲しくなるでしょう。



ファラオ。語彙集には、まだまだ多くの要望があります。 20回以上の頻度。



次はロシアのラップシーンの古き良き時代のための時間です。本当に恥ずかしくなくて、聴いてもらえる方にオススメです。

カースト。前向きな意味合いを持つ明るい優勢な単語。そして否定的な言葉は彼らの不道徳に衝撃的ではありません。 Frequency> = 25



そして最後に、韻マスターと単語Noize Mc（frequency> = 30）。



モーゲンシュテルンとファラオが彼らの歌で使用する負の色の語彙の豊富さは、彼らの歌の知覚と彼らが放送する気分に影響を与えます。反対のことをするように最善を尽くすとき、音楽から心地よい感情を得るのは難しいです。
感情分析を伴う使用済み辞書にはすべての単語が含まれていないため、多くの場合コンテキストにも依存するため、アーティストによる歌のムードについて100％で確実な結論を描くことは困難です。ただし、アーティストが使用する単語の数と使用する単語（アーティストが何とかアタッチしたものから）を示します。



明らかに、すべてのアーティストの言葉のほとんどには中立的な色合いがあり、実際にはリスナーには影響しません。しかし、興味深いことに、ファラオとモルゲンシュテルンは、肯定的な意味よりも否定的な意味を持つ言葉を多く使用しています。そして、これは、辞書の劣等性と多数のわいせつな単語とそのバリエーションの欠如にもかかわらず（辞書には28,248語が含まれており、そのうちのいくつかを手動で追加する必要がありました）。



カーストとノイズMsも先頭に中立的な言葉がありますが、2番目にポジティブな言葉があり、私たちにネガティブな感情を引き起こしません。

もちろん、このタイプの分析ではコンテキストの影響を評価することはできません。たとえば、「愛」という言葉は「否定」の粒子とともに使用でき、否定的な意味合いを持つ場合があります。しかし、認めなければなりません。「私はあなたを愛していません」というフレーズは、「私はあなたを憎む」というフレーズよりも楽しいです。そして、このフレーズの否定は、「not」という粒子によっても修正されません。それでも、「憎しみ」という言葉しか聞こえません。

音楽の好みは個人の問題であり、誰もが何を聞くかを決定します。しかし、グラフをもう一度見て、日常生活をどのように満たしたいかを考えてください。音楽はどこにでも付いており、気分に非常に強く影響することが多いので、なぜ意識的に毎日悪化させているのでしょうか。

一般的に、この記事はプログラミングが興味深いものであり、さまざまな分野に適用できるという事実についても述べています。見慣れた情報を新しい角度から表示し、明白または重要ではないと思われたことを考えさせることができます。それは、コード行の背後に何が隠されているのか、そして彼らが何を興味深いものにするのかをあなたにのみ依存しています。

プログラミング言語を学び、高品質の音楽を開発して聴き、YouTubeでのオンライン時間が7日以上かかります。知らない人のために、モーゲンスターンのアルバム「Legendary Dust」は、YouTubeでのオンラインブロードキャスト中に6日間録音され、その結果、モルゲンシュテルンのキャリアで最も成功し、リリースの最初の30分でVKontakteの100万回の聴取、11時間で500万回のプレイを獲得しました。リリース後の最初の2日間で、アルバムはVKontakteによって2100万回以上聴かれました。これはソーシャルネットワークの記録です。
