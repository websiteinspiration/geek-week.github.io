限られた数のパルス出力を備えたSimatic S7-1200によるステッピングモーター制御

昨年末、私は穀物の包括的なソリューションを提供する会社から、小さな穀物サンプリングシステムの自動化プロジェクトを検討するよう提案されました。このプロジェクトの特徴は、設計ソリューションとアクチュエータがすでに開発され、ハードウェアに実装されていることです。サンプリングの技術的プロセスの詳細については触れませんが、自動化の目標は、穀物と空気の流れの機械的バルブの制御、サンプルの均一性のためのスクリューミキサーの始動、空気タービンの電気モーターの制御、オペレーターのプロセス制御信号、およびいくつかの操作ステップのセンサーであると言えます。バルブとミキサーは、ステッピングモーターで駆動するように設計されています。



以前は、シングルボードマイクロコンピューターOrange Orange pi plus 2eとArduino Nanoマイクロコントローラーに基づいて自動化システムを構築することが決定されました。これらのボードでは、別の同様のプロジェクトのアプリケーションが見つかりましたが、これは別の話です。しかし、後で、すべての長所と短所について話し合った後、最大4つのパルス制御出力とデジタル出力モジュールSM 1222 DQ16 x 24VDCを備えたカタログ番号6ES7 214-1AG40-0XB0のPLC CPU 1214C DC / DC / DCで解決しました。カタログ番号6ES7 222-1BH32-0XB0。ステッピングモーターは、TB6560 V2ドライバーによって駆動されるKRS56シリーズから選択されました。
