1. SQL Server no Linux


O SQL Server agora é executado no Linux. Este curso o ajudará a obter um entendimento básico do SQL Server no Linux. Em seguida, você aprenderá como executar contêineres do SQL Server no Linux e implantar o SQL Server no Linux. Depois disso, você aprenderá como configurar automaticamente a implantação do SQL Server no Linux.

Você pode começar a treinar aqui

2. Crie soluções de análise de código aberto com o Azure HDInsight


Neste curso, você aprenderá sobre o HDInsight e como aplicar essa tecnologia para resolver uma variedade de tarefas do mundo real.

Entre os módulos do curso:

  • Introdução à oferta do Google Analytics de código aberto
  • Escolha a configuração correta do HDInsight para criar soluções de análise de código aberto
  • Criando e configurando um cluster HDInsight
  • Execute análises de ETL zero com a consulta interativa HDInsight

Comece a treinar

3. Implementando uma solução de fluxo de dados usando o Azure Streaming Analytics


Aprenda os princípios de processamento e streaming de eventos e veja como isso se aplica ao Azure Stream Analytics. Em seguida, configure seu trabalho do Stream Analytics para transmitir dados e saiba como monitorar e acompanhar a execução do trabalho.

Comece a treinar

4. Aprendizado do SQL Server Novo


A versão mais recente do SQL Server oferece muitos benefícios para desempenho aprimorado e funcionalidade aprimorada, além de suporte aprimorado para a plataforma de banco de dados. Este tutorial mostra como inventariar implantações do SQL Server usando o Microsoft Assessment and Planning Toolkit. Além disso, você aprenderá como atualizar os bancos de dados do SQL Server usando o Assistente de Migração de Dados e otimizar o sistema atualizado usando o Assistente de Experimentação de Banco de Dados e o Consultor de Otimização de Consulta.

Comece a treinar

5. Trabalhando com dados NoSQL no Azure Cosmos DB


Os dados NoSQL são uma maneira eficiente de armazenar dados que não atendem aos requisitos de um banco de dados SQL relacional. Descubra como usar o portal do Azure, a extensão do Azure Cosmos DB para Visual Studio Code e o .NET Core SDK para Azure Cosmos DB para trabalhar com dados NoSQL em qualquer local, além de fornecer aos usuários um alto nível de acessibilidade, independentemente de onde estejam.

Comece a treinar

6. Azure para engenheiros de dados


Aprenda como o mundo dos dados está evoluindo e como o advento da tecnologia em nuvem oferece novas oportunidades de negócios. Você aprenderá sobre as várias tecnologias de plataforma de dados disponíveis e como um engenheiro de dados pode tirar proveito dessas tecnologias para o benefício da organização.

Entre os módulos:

  • O mundo em evolução dos dados
  • Visão geral dos Serviços da Plataforma de Dados do Azure
  • Definindo as tarefas de um especialista em engenharia de dados em uma arquitetura hospedada na nuvem

Comece a treinar

7. Crie e use relatórios analíticos no Power BI


Este curso apresentará o Power BI e ensinará como usar e criar relatórios de inteligência de negócios.

Entre os módulos:

  • Introdução ao Power BI
  • Recuperando dados usando o Power BI Desktop
  • Modelagem de Dados no Power BI
  • Usando elementos visuais no Power BI
  • Explorando dados no Power BI
  • Publique e compartilhe no Power BI
