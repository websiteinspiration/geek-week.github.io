すでに述べたように、クリストファリスーパーコンピューターは格付けを一気に上げました。Linpackテストでのパフォーマンスは6.7 PFlop /秒、ピークパフォーマンス-8.8 PFlop /秒という結果を示しました。 T-Platforms社の次のLomonosov-2システムがそれぞれ2.5 PFlop / sおよび4.9 PFlop / sのこれらの数値を持っていることを考慮すると、クリストファリはリーダーシップの地位にしっかりと確立されていると想定できます。プロジェクトの技術基盤は、Tesla V100グラフィックアクセラレータを備えたNVIDIA DGX-2コンピューティングノードによってコンパイルされます。スーパーコンピューターは、さまざまな問題領域（医療診断、顔認識、音声バイオメトリ）の人工知能アルゴリズムと連携するように設計されています。 TOP500の世界ランキングでは、クリストファリが29位です。



最初の「新人」はクリストファリスーパーコンピューターです

「新しい」の2番目-UITiSクラスターNOVATEK STC

Computer EcosystemsとTechnoCityによって製造されたスーパーコンピューター「PetaNode 1.0 Cluster」は、Linpackテストで79.1 Tflop / sのパフォーマンスと141.47 Tflop / sのピークパフォーマンスを持っています。開発の際立った特徴は、ノードあたり10 GPUのレートでAMDのアクセラレータを使用することです。このスーパーコンピューターの主な目的は気候モデリングです。TOP50の参加者が習得したニッチな分野です。

プロジェクトの総量の技術設備に関しては、変化は根本的なものではなく、具体的なものです。 Intelプロセッサーは、最新世代のスーパーコンピューターの不可欠な属性であり続けています。現時点では、ランキングに代替ソリューションはありません。 InfiniBandネットワーク（33ケース対35）およびより頻繁に-10ギガビットイーサネット（6ケースに対して8ケース）は、ノード間の通信にあまり一般的に使用されなくなりました。

格付けの新参者3人のうち2人（KristofariとPetaNode 1.0 Cluster）は、異機種混在のハイブリッドクラスターの構築から、グラフィックアクセラレーターを使用した同種システムへの移行の明確な傾向を示しています。この傾向は、エネルギー効率と密度の基準によって推進されているようです。

スーパーコンピュータのアプリケーションは、ほとんどの場合、変更されていません。科学と教育の分野は依然として最も関連性が高く、23のシステムがサービスを提供していますが、応用研究では10台のスーパーコンピューターが使用されています（前のバージョンの3台より少ない）。業界は、最新の技術進歩にまだ十分に対応できていません。50台のマシンのうち3台だけがそのニーズに応えています。さらに、格付けには、地震予報、ITサービス、クラウドストレージ、商業および金融セクターへのサービスなどのタスク用に設計されたプロジェクトが含まれます。

最後に、Hewlett-Packard Enterprise、RSKグループの企業、およびT-Platformは、最高水準のスーパーコンピュータの最もアクティブなサプライヤであり続けています。
