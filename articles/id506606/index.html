Pencipta clicker PIXI.js

Sebagian besar dari mereka yang mulai belajar pemrograman berawal dari keinginan untuk membuat game mereka. Nuuu ... Saya tidak terkecuali, tetapi nasib membawa saya ke pengembangan web.



Oke, mari kita langsung ke intinya. Buat clicker dengan pixi.js.

Tugas itu sepele, tetapi karena Saya sedikit mengerti tentang pengembangan game, saya harus mempelajari topik ini sedikit lebih baik.

Mengapa js? Kenapa pixi.js? Mengapa clicker sama sekali ?!

- Saya memutuskan untuk hanya menunjukkan implementasi tugas saya. Saya tidak ingin mengajari seseorang teknologi apa yang harus digunakan, cara menulis kode. Hanya solusi saya untuk masalah ini.

Sebelum Anda menulis kode:

Pertama

Ini adalah video dari proses membuat game. Jika Anda terlalu malas untuk membaca, maka Anda dapat menonton dan mendengarkan


Kedua, saya akan menyisipkan semua kode dengan gambar, tetapi pada akhir artikel akan ada tautan ke github dengan sumber proyek sehingga Anda dapat menyodok diri Anda sendiri.

Dalam kode, saya meninggalkan komentar jelas untuk Anda, saya harap ini akan menjadi jelas.

Pertama, kami mendeklarasikan alias dan menginisialisasi aplikasi itu sendiri.

Kami membuat variabel yang diperlukan untuk semua elemen permainan dan memuat gambar.



Untuk gambar, saya memilih format atlas.

Jika Anda tidak terbiasa dengan atlas, maka ini adalah format json untuk mengunduh sejumlah besar gambar sekaligus.

Beginilah tampilannya dalam kode.



Saya membuatnya menggunakan TexturePacker. Untuk melakukan ini secara manual, menurut pendapat saya, tidak disarankan.

Dengan menggunakan fungsi pengaturan, saya mengurai atlas, mengatur lokasi elemen, gaya untuk teks, menambahkan nilai awal poin.



Ini adalah tindakan standar untuk semuanya (tata letak). Yang paling menarik adalah tiga sifat target. interaktif, buttonMode dan on ("pointerdown", handlerClick). Dua yang pertama diperlukan untuk interaksi klik dan elemen target. pada adalah pengendali acara reguler.

Sekarang kita perlu menulis semua fungsi yang diperlukan agar gim bekerja.



Game loop - gameLoop Status permainan
- bermain
Pemrosesan klik - handlerKlik

gameLoop cukup memanggil fungsi status.

Dalam permainan, saya memeriksa nilai timer (ini 10) dan memungkinkan klik pada tombol, juga memperbarui ukuran target.

handlerClick memeriksa kemungkinan klik dan klik dibuat.

Dari cacat yang saya lihat adalah ukuran pada elemen target. Itu berubah dari 0,9 menjadi 1. Segera. Tidak ada animasi. Tetapi ketika Anda bermain itu tidak melihat, dibiarkan apa adanya.

Seperti yang dijanjikan, di bawah ini adalah tautan ke proyek github.

Jika Anda belum menemukan pixi, tetapi ingin menggunakan gim sendiri, maka Anda memerlukan server. Saya menggunakan openserver. Ini untuk permintaan ajax.
