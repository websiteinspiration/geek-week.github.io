また、キーボードを使用して文字を入力できるようにしたいと考えています。これを行うには、「ドキュメント」または「ウィンドウ」オブジェクトに「キーストローク」イベントハンドラーを追加してから、たとえば正規表現を使用して、キーの「キー」プロパティの不要な値を除外する必要があります。

document.addEventListener('keydown', event => {
    if ((event.key).match(/[0-9%\/*\-+\(\)=]|Backspace|Enter/)) calc(event.key)
})

この場合の一致メソッドは、フィルターの役割を果たします。指定された条件に一致しない引数をcalc関数に渡すことはできません。

条件自体は次のとおりです。event.keyの値が角括弧（[]で示された文字の1つである場合）、0〜9の数字、除算、乗算、加算、減算、開き括弧、等号、または等号の記号、バックスラッシュ-エスケープ）または（|-変更）バックスペースまたはEnterキーを押してから、calcをパラメーターとしてevent.keyで呼び出します。それ以外の場合は何もしません（Shiftも正常に破棄されます）。

私たちの主な（そして唯一の）calc関数は次のとおりです（コードは下から上に読む必要があります）：

//       
function calc(value) {
    //      Enter
    if (value.match(/=|Enter/)) {
        //   
        try {
            //   
            //     "evaluate"  "math"
            // Math.trunc      
            output.textContent = Math.trunc(math.evaluate(output.textContent))
            
        //    
        } catch {
            //   
            let oldValue = output.textContent
            //   
            let newValue = ' '
            //      
            output.textContent = newValue
            //       
            setTimeout(() => {
                output.textContent = oldValue
            }, 1500)
        }
        
    //    "C"
    } else if (value === 'C') {
        //  
        output.textContent = ''
    
    //    ""  Backspace
    } else if (value.match(/CE|Backspace/)) {
        //     
        output.textContent = output.textContent.substring(0, output.textContent.length - 1)
        
    //     ()   
    } else {
        //     
        output.textContent += value
    }
}

結論として、宣言されたスケーラビリティと完全な機能についてのいくつかの言葉。

evaluateメソッド（以前のeval）と他のMath.jsメソッドには、非常に優れた機能があります。これらの機能に基づいて、新しい文字や演算子を追加することで計算機の機能を簡単に拡張でき、浮動小数点数を操作できるようになります（スイッチと「toFixed」メソッドを使用して小数点以下の桁数を調整）など。
