docker-composeでnode-red認証をデプロイして構成する



承認を有効にし、Dockerボリュームを使用して、node-redをdocker-composeにデプロイします。



docker-compose.ymlファイルを作成します。 



version: "3.7"

services:
  node-red:
    image: nodered/node-red
    environment:
      - TZ=Europe/Moscow
    ports:
      - "11880:1880" # 11880 -     , 1880 -     node-red  .
    volumes:
      - "node-red:/data" # node-red -    docker   , /data -   .
    restart: always
volumes:
  node-red: #   node-red  .


コンテナーをデバッグモードで起動します（最初の起動では ` -d`キーを使用せず、エラーが表示されたときにエラーを表示する必要があります）。 



$ docker-compose up node-red
Creating node-red_node-red_1_3e3e59f5e044 ... done
Attaching to node-red_node-red_1_bca4cb987984
node-red_1_bca4cb987984 |
node-red_1_bca4cb987984 | > node-red-docker@1.0.3 start /usr/src/node-red
node-red_1_bca4cb987984 | > node $NODE_OPTIONS node_modules/node-red/red.js $FLOWS "--userDir" "/data"
...


コンテナーを停止し、ボリュームを表示するコマンドを実行します。 



$ docker volume ls
DRIVER              VOLUME NAME
local               node-red_node-red


ボリュームの詳細情報を表示します。 



$ docker volume inspect node-red_node-red
[
    {
        "CreatedAt": "2020-05-02T18:37:33Z",
        "Driver": "local",
        "Labels": {
            "com.docker.compose.project": "node-red",
            "com.docker.compose.version": "1.23.0",
            "com.docker.compose.volume": "node-red"
        },
        "Mountpoint": "/var/lib/docker/volumes/node-red_node-red/_data", #   
        "Name": "node-red_node-red",
        "Options": null,
        "Scope": "local"
    }
]


ボリュームディレクトリに移動します。このディレクトリには、コンテナのデプロイ時にdockerが作成したファイルがすでに含まれています。 



$ sudo ls /var/lib/docker/volumes/node-red_node-red/_data
lib  package.json  settings.js


settings.js ファイルに興味があります。それを開いて、adminAuthコードを探します。コメントを外します。 



 // Securing Node-RED
    // -----------------
    // To password protect the Node-RED editor and admin API, the following
    // property can be used. See http://nodered.org/docs/security.html for details.
    adminAuth: {
        type: "credentials",
        users: [{
            username: "admin",
            password: "$2a$08$zZWtXTja0fB1pzD4sHCMyOCMYz2Z6dNbM6tl8sJogENOMcxWV9DN.",
            permissions: "*"
        }]
    },


パスワード キー：ノードレッドのパスワードのハッシュを挿入します。



node-redハッシュパスワードの取得
node.jsがインストールされている任意のマシンにnode-red-adminパッケージをインストールします。 



npm i node-red-admin -g


: 



node-red-admin hash-pw




コンテナーを起動し、ポート11880に接続します。 



http://192.168.0.100:11880/


承認ウィンドウが表示されます。







ログインパスワードを入力します。



すべてが機能する場合は、-dスイッチを使用してコンテナを再起動します。 



$ docker-compose up -d node-red


このようなもの。
