スマートプリンター。Octoprintの改善

4つの押出機Z-Boltプリンターのオープンプロジェクトを開発しているときに、プリンターを制御するための画面を選択するという問題に直面しました。利用可能な画面のうち、Duet Wifiソリューションのみが2台以上の押出機を制御できます。控えめに言っても、その解決策は予算がかかりませんし、インターフェースも万人向けではありません。それで、私たちは好みと遊女を持つ代替案を探すことにしました。この話は彼についてです...


インターネットには、Octoprintの使用に関する多くの情報があります。半年以上前、私たちはこのソリューションのユーザーになり、その利点をすぐに理解しました。Web担当者を介してプリンターを制御する機能は、過大評価することは非常に困難です。Octoprintには多くのプラグインがあり、最も重要なのはOctoprintが2を超える押出機の数で非常によく管理されていることです。

Octoprint愛好家の多くは、「完全に」という言葉の画面が不要になったと自信を持って主張しています。

しかし、私はOctoPrintユーザーの別の部分があると確信しています。誰もが3Dプリンターを日常的に使用するための独自の習慣と要件を持っています。また、画面を持っていると、実行するのにより便利な操作があります。

  • フィラメントの交換
  • テーブルレベルのキャリブレーション
  • 以前に印刷されたファイルの印刷を開始します
  • プリンターの予熱をオンにする
  • 印刷ステータスを確認する機能
  • ...

Z-Boltプリンタのデフォルト画面はMKS TFTです。あなたはTFT24、TFT32またはTFT35を置くことができます。これらは優れた画面ですが、Octoprintと統合できません。Octoprintが印刷を開始した場合、画面はそれについて何も知りません。異なるファイル保管庫などを使用しています。また、MKS TFTスクリーンは、3つ以上のヘッドを持つ3Dプリンターでは機能しません。

私の目標は、2台以上の押出機を制御できるOctoprintの画面を見つける/作成する/設定することでした。

少しググって、TouchUIプラグインに出くわしました。



第一印象は次のとおりでした：ここにあります-解決策!!!

私はすぐにプラグインをインストールし、電話からOctoprintを開きました。 TouchUIインターフェイスは愚かです：/。ボタンが正常に押された場合もあれば、速度が低下した場合もあり、最初はまったく押されなかった場合もあります。接続された画面からは、TouchUIの動作は電話の場合と同じように悪かった。

並行して、別のソリューションが見つかりました。



TouchUIとは異なり、この説明は心強いものでした。これは、ブラウザーなしで実行される独立したアプリケーションであるため、必要なコンピューティングリソースが大幅に少なくなります。

インストールと構成で問題は発生しませんでした。このインターフェースは、実際にははるかに予測可能で応答性の高いものでした。



しかし、ここではすべてが順調に進んだわけではありません。印刷を開始すると、アプリケーションが定期的にクラッシュしました。 github でこのエラーの問題がありましたほぼ1年前に作成され、活発に議論されましたが、クローズされていません。

主な問題は、アプリケーションの作成者がどうやら彼への興味を失い、現在は開発を行っていないことであることが判明しました。

私の意見では、アプリケーションは十分に記述されており、潜在的な可能性があるため、「ハードフォーク」を作成し、プロジェクトを徹底的に行うことが決定されました。

ここに私たちが得たものがあります：

  • プロジェクトに最適な画面が選択されます
  • Octoprint-TFTによって「ハードフォーク」を作成
  • プリンターのハードウェアおよびソフトウェアアーキテクチャに関する見通しの改訂
  • Octoprintインターフェイスのカスタムテーマを作成しました

しかし、まず最初に…

画面の選択


TFT 3.5の画面から始めましたが、控えめに言っても批判に耐えられませんでした。視野角がなく、画像がかすれている。したがって、より良いソリューションを選択することが決定されました。

主な基準は次のとおりです。

  • IPSマトリックス
  • サイズ4-5インチ
  • 静電容量式スクリーン（オプション）
  • 許容価格

その結果、2つのオプションを決定しました

。1.オプションの方が簡単です。 4 '' IPS抵抗膜方式タッチスクリーン



2.オプションはより豊富です。 4.3 '' IPS静電容量式タッチスクリーン


両方の画面の解像度は800x400です。どちらも素晴らしい画像、色再現、視野角を持っています。主な違いは、容量性タッチスクリーンです。あなたはそれにすぐに慣れ（すべてが良いように）、抵抗よりもはるかに快適です。ただし、抵抗膜方式のタッチスクリーンを備えた画面も非常に実用的です。

これはハードフォークです、ベイビー！
