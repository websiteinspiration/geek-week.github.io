DOM APIを使用した進行状況バー


DOM APIを使用して、長い行のかさばりを減らすことができます。要素を作成し、必要なクラスと属性を追加し、値を設定して、DOMに追加します。

const heroPlaceholder = document.querySelector('.hero-list');
const progressEl = document.createElement('progress');
progressEl.classList.add('hero-list', 'progress', 'is-medium', 'is-info');
const maxAttr = document.createAttribute('max');
maxAttr.value = '100';
progressEl.setAttributeNode(maxAttr);
heroPlaceholder.replaceWith(progressEl);

function createListWithInnerHTML(heroes: Hero[]) {

DOM APIを使用した文字マッピング


function createListWithDOMAPI(heroes: Hero[]) {
  const ul = document.createElement('ul');
  ul.classList.add('list', 'hero-list');
  heroes.forEach(hero => {
    const li = document.createElement('li');
    const card = document.createElement('div');
    card.classList.add('card');
    li.appendChild(card);
    const cardContent = document.createElement('div');
    cardContent.classList.add('card-content');
    card.appendChild(cardContent);
    const content = document.createElement('div');
    content.classList.add('content');
    cardContent.appendChild(content);
    const name = document.createElement('div');
    name.classList.add('name');
    name.textContent = hero.name;
    cardContent.appendChild(name);
    const description = document.createElement('div');
    description.classList.add('description');
    description.textContent = hero.description;
    cardContent.appendChild(description);
    ul.appendChild(li);
  });
  heroPlaceholder.replaceWith(ul);
}

テンプレートを使用した文字表示


テンプレートを使用してキャラクターのリストを表示できます。これを行うには、要素の表示と同じ手法を使用します<prgress>。まず、HTMLページにテンプレートを作成します。このHTMLはと比べて少し複雑になり<prgress>ます。しかし、それは問題ではありません。これはHTMLページ内の単なるHTMLなので、美しいVS Codeエディターでエラーやフォーマットを簡単に修正できます。

<template id="hero-template">
  <li>
    <div class="card">
      <div class="card-content">
        <div class="content">
          <div class="name"></div>
          <div class="description"></div>
        </div>
      </div>
    </div>
  </li>
</template>

これで、文字のリストを作成するコードを記述できます。まずul、文字で行を折り返すために作成しますli。これで、配列を調べ、同じ方法を使用して、document.importNode()各文字に同じテンプレートを適用できます。パターンを再利用して各行を作成できることに注意してください。必要な数の線を作成するソース図面に変わります。

文字のリストには、名前と説明を含める必要があります。つまり、テンプレート内の特定の値を置き換える必要があります。これを行うには、これらの値を挿入する場所を何らかの方法で識別して参照することをお勧めします。この例では、各文字へのリンクを取得するために、メソッドが使用されquerySelector('your-selector')、その後に名前と説明が与えられます。

function createListWithTemplate(heroes: Hero[]) {
  const ul = document.createElement('ul');
  ul.classList.add('list', 'hero-list');
  const template = document.getElementById('hero-template') as HTMLTemplateElement;
  heroes.forEach((hero: Hero) => {
    const heroCard = document.importNode(template.content, true);
    heroCard.querySelector('.description').textContent = hero.description;
    heroCard.querySelector('.name').textContent = hero.name;
    ul.appendChild(heroCard);
  });
  heroPlaceholder.replaceWith(ul);
}
