WISE-PaaS-産業用モノのインターネットのためのクラウドプラットフォーム



WISE-PaaS-（Platform as a Service）産業用モノのインターネット向けのAdvantechクラウドプラットフォーム。データの収集、処理、視覚化のためのさまざまなツール、自動化、デバイス管理、人工知能システム、機械学習を組み合わせています。このプラットフォームは、業界、ヘルスケア、製造、ロジスティクスなどで複雑なシステムを構築するための、いくつかの既製のソフトウェアソリューションセットを組み合わせています。

WISE / PaaSプラットフォームは、クラウドプロバイダーのアマゾンウェブサービス（AWS）、Google Cloud Platform（GCP）、Microsoft Azureのキャパシティーで実行でき、OpenStackでローカルに実行できます。

この記事では、インフラストラクチャの構築に気を取られることなくアプリケーションを迅速に開発できるWISE / PaaSソフトウェア製品について説明します。一般的な言語でのアプリケーションの実行をサポートしています：Java、.NET、Ruby on Rails、Node.js、Grails、Scala on Lift、Python PHP、および主要なMySQLデータベースエンジン、MongoDB、PostgreSQL、Redis、RabbitMQ。データを視覚化するために、Grafanaフレームワークを利用できます。組み込みシステム用の個別のソフトウェアツールセットにより、デバイスの初期設定、ファームウェアの更新、IoTデバイスからのデータ収集が簡素化されます。

アプリケーションマーケットプレイス


プラットフォームは、アプリケーションストアで必要に応じて購入できるさまざまなツールのコンストラクタです。提案されたソリューションの中には、アドバンテックの自社製品とパートナー製品の両方が用意されています。一部のアドバンテック製品では、無料の試用期間が利用できます。

市場のソフトウェアソリューションは、目的に応じていくつかの大きなカテゴリに分類されます。

  • クラウドアプリケーション管理システム -オーケストレーター、監視、バックアップ
  • エンドデバイスの管理 -ファームウェアの更新、デバイスからのデータの収集、監視、自動構成
  • セキュリティ -侵入検知および防止システム（IPS / IDS）、アンチウイルス

WISE-PaaS /ダッシュボード


WISE-PaaS /ダッシュボード-Grafanaフレームワークに基づいてデータを視覚化するための一連のツール。通常、グラフ、チャート、および長時間発生するプロセスの視覚的表示を作成するために使用されます。産業目的に加えて、スマートホームやヘルスケアシステムの気候プロセスの監視にも使用できます。


WISE-PaaS /ダッシュボードデータ視覚化プラットフォーム

ウィジェット


Grafanaフレームワークには、テーブル、グラフ、チャート、ヒートマップなど、データを表示するための多くのオプションがあります。プログラミングスキルがほとんどなく、さまざまなデータを表示するための完全なダッシュボードを作成できます。マウスでウィジェットを追加できます。

したがって、WISE-PaaS /ダッシュボードでは、さまざまなソースからのデータを組み合わせて、それらを単一のパネルに表示できます。

データソース


ダッシュボードは、さまざまなソースから表示するデータを受信できます。現在サポートされているデータベース：CloudWatch、Elasticsearch、Graphite、InfluxDB、MySQL、OpenTSDB、PostgreSQL、Prometheus、RMM-SimpleJson、SCADA-SimpleJson、SimpleJson。これらのデータベースに加えて、さまざまなソースからデータを収集するように任意のクエリ形式を構成できます。システムを調査するためのテストデータセットも利用できます。

WISE-PaaS / SaaS Composer


SaaS Composer-技術プロセスの2次元および3次元スキームを構築するためのフレームワーク。従来の古いスキームとは異なり、進行中のプロセスのより有益で視覚的なモデルを作成できます。生産ライン全体と建物を3Dで視覚化し、リアルタイムのイベントを3Dモデルで表示します。



主な機能：

  • HTML 5 Canvas上のWebインターフェイス。図を表示するための追加のソフトウェアをインストールする必要がない、優れたパフォーマンス。
  • 2Dおよび3Dモデルの作成。3Dモデルを.OBJ + .MTL形式でインポート
  • JPEG、PNG、SVG、OBJ、MTL画像のサポート。3Dモデル構築時のベクターグラフィックスSVGのサポート。既存のスキームからグラフィックをインポートできます。
  • 3Dオブジェクトの上にアニメーションを追加してデータを表示する
  • スクリプト言語のサポート
  • 他のWISE-Paas製品、特にWISE-PaaS /ダッシュボードとの統合

3Dモデルの設計者には、パイプ、バルブ、ワイヤー、モーター、工作機械、グリルなどの主要コンポーネントのライブラリがあります。その中で、実際のオブジェクトの現実的なモデルを作成し、データ付きのウィジェットを追加できます。



電力システムのリアルタイムステータス、さまざまな場所でのエネルギー消費量、空気状態：CO2レベル、空気中の微粒子などを示すアドバンテックの建物のデモ


SaaS Composerを使用して作成されたデモは、建物内のさまざまなパラメーターのステータスを示します。

WISE-PaaS / EnSaaS-デバイスの操作（Edge to Cloud）


エンドデバイスをクラウドインフラストラクチャに簡単に統合するために、WISE-PaaSは組み込みシステムとIoTを操作するための一連のツールを提供します。

WISE-Paas / DeviceOnは、センサー、端末、組み込みシステムなど、多数のエンドデバイスを管理および構成するためのプラットフォームです。


主な機能：

  • ゼロタッチプロビジョニング -最終機器の自動構成とシステムへの追加
  • アクセス制限 -デバイスのセキュリティを確保し、不正アクセスを防止する
  • 更新（OTA） —エンドデバイスのソフトウェアとファームウェアを自動的に更新します
  • 監視 -プッシュ通知、SMSまたはEメールによる機器のステータスと問題の通知の監視
  • バックアップとアーカイブ -デバイス構成とそのデータのバックアップを作成する
  • デバイスマップの構築-建築計画およびマップ上でデバイスレイアウトを構築するためのコンストラクター

WISE-Paas / WISE-Agent


WISE-Agent-WISE-PaaS / DeviceOnと対話するためにエンドデバイスにインストールされたソフトウェア。すべての主要なオペレーティングシステムがサポートされています。コンパイルされたパッケージは、Windows、Ubuntu、Android（RISC）、OpenWRT（RISC）で利用できます。
クラウドプラットフォームとのやり取りは、MQTT（s）プロトコルに従って行われます。

[ケース]スマートシップの開発におけるWISE-PaaSプラットフォームの使用


SaierNico company-海運会社向けのインテリジェントシステムの開発と船用機器の開発を行っています。SaierNicoは、Wise-PaaSプラットフォームを使用して、船舶とインシデントレスポンスのリアルタイムリモートモニタリングシステムを開発しました。



センサーは、船舶のさまざまなコンポーネント（エンジン速度、圧力、空調システムの状態、ポンプなど）からデータを収集します。データ転送には、RabbitMQブローカーが使用されます。これにより、船舶との通信が不安定になるため、メッセージ配信の信頼性が向上します。データはWebAccess / SCADAシステムに流れ込みます。

システムアーキテクチャー


WISE-PaaS / APMは 、船のコンポーネントのパフォーマンスを追跡するために使用されます。
コントロールセンターのデータの視覚化は、WISE-PaaS /ダッシュボードとWISE-PaaS / SaaS Composerに基づいています。

船舶システムのエンドデバイスのファームウェアアップグレードは、WISE-PaaS / OTAを使用して実行されます。
