JavaScript-Projekte zum Lernen und Entwickeln



Guten Tag, Freunde!

Ich präsentiere Ihnen eine Auswahl von 22 Projekten in reinem oder wie gesagt Vanille-JavaScript (ich persönlich bevorzuge nur JavaScript).

Für mich ist dies eine Art Zwischensumme beim Erlernen von JS.

Der Code wird zu unterschiedlichen Zeiten geschrieben, sieht also anders aus und ist unterschiedlich komplex.

Die meisten Projekte können nicht als vollwertige Anwendungen bezeichnet werden, aber jedes von ihnen löst die Aufgabe erfolgreich.

Der Projektcode ist hier .

Das Format lautet wie folgt: Name des Projekts, Kurzbeschreibung, Sandbox.

Für einige Projekte gibt es separate Artikel. Anderen zufolge könnten sie eines Tages auftauchen.

So lass uns gehen.

1. Audio-Fortschritt und Visualisierung


Eine Anwendung zum Abspielen und Visualisieren von Audiodaten mithilfe der Web-Audio-API und der Canvas-API.


2. Ausbruch


Das Spiel "Breakout".


3. Rechner


Taschenrechner. Benutzte Bibliothek Math.js. V. 2.0.


4. Währungsrechner


Währungsrechner. Eine spezielle API wird verwendet.


5. Formularvalidator


Validierung des Formulars.


6. Henker


Das Spiel "Hangman".


7. Bildergalerie und Slider Maker


Flexibler Bildergalerie-Generator mit integriertem Schieberegler.


8. Lyrics Search


Suche nach Texten (Texte). Eine spezielle API wird verwendet.


9. Mahlzeitensuche


Suchen Sie nach Gerichten mit Kochmethode und Zutaten. Eine spezielle API wird verwendet.


10. Meditations-App


Der Antrag auf Meditation. Arbeiten Sie mit Audio und Video.


11. Speicherkarten


Karten zum Erinnern. Interaktiver Schieberegler.


12. Notizen App


Antrag auf Notizen (Aufgaben) mit der Möglichkeit, das Datum der Erinnerung festzulegen. Um Daten auf der Clientseite zu speichern, wird IndexedDB verwendet.


13. Zahlen erraten


Errate das Zahlenspiel. Wird von der Web Speech API verwendet.


14. Tischtennis


Das Spiel "Ping Pong".


15. Steinpapierschere


Das Spiel "Stein, Schere, Papier."


16. Geschwindigkeitstyp


Geschwindigkeitstippsimulator.


17. Tetris


Das Spiel "Tetris".


18. Textleser


Antrag zum Lesen von Text. Wird von der Web Speech API verwendet. Vergleichen Sie Stimmen von Microsoft und Google.


19. Tic Tac Toe


Das Spiel "Tic Tac Toe". Ein spezieller Algorithmus wird verwendet. Versuche das Auto zu schlagen.


20. Zunderartiges Karussell


Ein komplexer, aber interessanter Slider. Die Hammer.js-Bibliothek wird verwendet.


21. Virtuelle Tastatur


Virtuelle Tastatur.


22. Wetter App


Anwendung zur Anzeige des Wetters. Eine spezielle API und ein Proxy werden verwendet.



Das Repository wird regelmäßig aktualisiert, bleiben Sie also auf dem Laufenden.

Vielen Dank für Ihre Zeit. Ich hoffe es wurde gut angelegt.
