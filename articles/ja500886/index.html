ローカルCSVファイルの操作

CSVデータセットをDataFrameにロードするには、pandasライブラリのread_csv関数を使用できます。次のスニペットは、この章の前半で作成されたCSVのaccounts.csvファイルをダウンロードして表示します。

In  [1]: import pandas as pd
In  [2]: df = pd.read_csv('accounts.csv',
     ...:                      names=['account', 'name', 'balance'])
     ...:

In  [3]: df
Out[3]:
      account     name    balance
0          100    Jones      24.98
1          200       Doe    345.67
2          300    White        0.00
3          400     Stone    -42.16
4          500       Rich    224.62

names引数は、DataFrameの列名を指定します。この引数がない場合、read_csvは、CSVファイルの最初の行に、列名のコンマ区切りのリストが含まれていると見なします。
DataFrameデータをCSVファイルに保存するには、DataFrameコレクションのto_csvメソッドを呼び出します。

In [4]: df.to_csv('accounts_from_dataframe.csv', index=False)

キー引数index = Falseは、行名（フラグメント[3]のDataFrame出力の左側にある0〜4）をファイルに書き込まないことを意味します。結果のファイルの最初の行には、列名が含まれています。

account,name,balance

200,Doe,345.67
300,White,0.0
400,Stone,-42.16
500,Rich,224.62

9.12.3。タイタニック号の災害データセットを読む


タイタニック号の災害データセットは、最も人気のある機械学習データセットの1つであり、CSVを含む多くの形式で利用できます。

タイタニックディザスタデータセットをURLからダウンロードする


CSV形式でデータセットを表すURLがある場合は、read_csv関数を使用してそれをDataFrameにロードできます-GitHubから例を挙げましょう。

In [1]: import pandas as pd
In [2]: titanic = pd.read_csv('https://vincentarelbundock.github.io/' +
    ...:       'Rdatasets/csv/carData/TitanicSurvival.csv')
    ...:

タイタニック号の災害データセットのいくつかの行の表示データ

セットには1300を超える行が含まれ、各行は1人の乗客を表しています。ウィキペディアによれば、約1317人の乗客が搭乗しており、そのうち815人が死亡しました1。大きなデータセットの場合、DataFrameの出力時に最初の30行のみが表示され、次に省略記号「...」と最後の30行が表示されます。スペースを節約するために、DataFrameコレクションのheadメソッドとtailメソッドを使用して、最初と最後の5行を確認します。どちらのメソッドもデフォルトで5行を返しますが、表示される行数は引数で渡すことができます

。In [3]：pd.set_option（ 'precision'、2）＃浮動小数点値の形式


注意：パンダは、列または列名の最も広い値に基づいて各列の幅を調整します（幅が最も大きいものに依存します）。行1305の経過時間列にはNaN-データセット内の欠損値の記号があります。

列名

9.12.4。タイタニック号の災害データセットを例として使用した簡単なデータ分析


次に、パンダを使用して、記述統計のいくつかの特性を例として使用して簡単なデータ分析を行います。数値列と非数値列の両方を含むDataFrameコレクションに対してdescribeを呼び出すと、describeは数値列のみ（この場合は、age列のみ）の統計特性を計算します。



カウント（1046）とデータセット内のデータの行数の違いに注意してください（1309-tailを呼び出すと、最後の行のインデックスは1308でした）。年齢の値が含まれたデータ（カウント値）の1046行のみ。 1305行のように、残りの結果は欠落しており、NaNでマークされていました。計算を実行するとき、pandasライブラリはデフォルトで欠落データ（NaN）を無視します。有効な年齢の1046人の乗客の場合、平均年齢（期待）は29.88歳でした。最も若い乗客（分）はたった2か月（0.17 * 12は2.04を与える）であり、最も古い（最大）は80歳でした。年齢の中央値は28歳でした（50％の四分位数で示されます）。 25％の四分位数は、乗客の前半の年齢の中央値（年齢でランク付け）を表します。75％の四分位数は、乗客の後半の中央値です。

生き残った乗客に関する統計を計算するとします。生き残った列を値「yes」と比較して、True / False値を持つ新しいSeriesコレクションを取得し、describeを使用して結果を説明できます。

In [9]: (titanic.survived == 'yes').describe()
Out[9]:
count      1309
unique        2
top       False
freq        809
Name: survived, dtype: object

非数値データの場合、describeは記述統計のさまざまな特性を表示します。

  • count-結果の要素の総数。
  • 一意-結果としての一意の値の数（2）-True（乗客が生き残った）またはFalse（乗客が死亡した）;
  • top-結果として最も頻繁に遭遇する値。
  • freq-値topの出現回数。

9.12.5。乗客年齢の棒グラフ


視覚化は、データをよりよく知るための良い方法です。Pandasには、Matplotlibに基づく多くの組み込み視覚化ツールが含まれています。それらを使用するには、最初にIPythonでMatplotlibサポートを有効にします。

In [10]: %matplotlib

ヒストグラムは、数値データの範囲にわたる値の分布を明確に示しています。DataFrameコレクションのhistメソッドは、各数値列のデータを自動的に分析し、対応するヒストグラムを作成します。データの各数値列のヒストグラムを表示するには、DataFrameコレクションに対してhistを呼び出します。

In [11]: histogram = titanic.hist()

タイタニック号の災害データセットにはデータの数値列が1つしかないため、グラフには年齢分布のヒストグラムが表示されます。複数の数値列を持つデータセットの場合、histは数値列ごとに個別のヒストグラムを作成します。
