PHP：複雑なXMLファイルを解析し、ネイティブコードで溺れさせない方法

一日のいい時間！

XML形式の範囲は非常に広範囲です。XMLは、CSV、JSONなどと並んで、さまざまなサービス、プログラム、サイト間で交換するためのデータを提示する最も一般的な方法の1つです。例として、1C“ Trade Management”とオンラインストアの間で商品や注文を交換するためのCommerceMLフォーマットを引用できます。

したがって、Webサービスを時々作成するほとんどすべての人は、XMLドキュメントを解析する必要性に対処する必要があります。私の投稿では、XMLReaderを使用してこれを可能な限り明確かつ透過的に行う方法の1つを提案します。

PHPは、XML形式を操作するいくつかの方法を提供します。詳細は省きますが、原則として次の2つのグループに分けることができます。

  1. XMLドキュメント全体をオブジェクトとしてメモリに読み込み、このオブジェクトを操作する
  2. タグ、属性、テキストコンテンツのレベルでのXML文字列の段階的な読み取り

最初の方法はより直感的で、コードはより透過的に見えます。この方法は、小さなファイルに適しています。

2番目の方法は、低レベルのアプローチです。これにより、多くの利点が得られると同時に、寿命がやや長くなります。もっと詳しく見てみましょう。長所：

  • 解析速度。詳しくはこちらをご覧ください。
  • より少ないRAMの消費。メモリ内で非常に負荷の高いオブジェクトの形式ですべてのデータを保存するわけではありません。

しかし：私たちはコードの読みやすさを犠牲にします。解析の目的が、たとえば、XML内の特定の場所の値の合計を単純な構造で計算することである場合、問題はありません。
ただし、ファイル構造が複雑な場合、データを操作する場合でも、このデータへの完全パスに依存し、結果に多くのパラメーターが含まれる場合は、ここではやや乱雑なコードになります。

それで私はその後私の人生を楽にするクラスを書きました。これを使用すると、ルールの記述が簡略化され、プログラムの可読性が大幅に向上し、サイズが何倍も小さくなり、コードがより美しくなります。

主な考え方は次のとおりです。XMLのスキーマとその操作方法の両方を、必要なタグのみの階層を繰り返す単一の配列に格納します。また、同じ配列内の任意のタグに対して、タグを開く、閉じる、属性を読み取る、テキストを読み取る、またはすべてをまとめて処理するためのハンドラーである必要な関数を登録できます。したがって、XML構造とハンドラーを1か所に格納します。 XMLファイルで何をしているのかを理解するには、処理構造を一目見れば十分です。以下の例のように、単純なタスクでは可読性の利点は小さいが、比較的複雑な構造のファイル（たとえば、1Cの交換フォーマット）で作業する場合は明らかであることを予約します。

次に詳細を説明します。ここに私たちのクラスがあります：

class XMLReaderStruct extends XMLReader {

  public function xmlStruct($xml, $structure, $encoding = null, $options = 0, $debug = false) {
    $this->xml($xml, $encoding, $options);
    $stack = array();
    $node = &$structure;
    $skipToDepth = false;
    while ($this->read()) {
      switch ($this->nodeType) {
        case self::ELEMENT:
          if ($skipToDepth === false) {
            //       ,     ,  :     
            //  ,   ,      ,      . 
            //  ,    ,        .
            if (isset($node[$this->name])) {
              if ($debug) echo "[  ]: ",$this->name," -   .   .\r\n";
              $stack[$this->depth] = &$node;
              $node = &$node[$this->name];
              if (isset($node["__open"])) {
                if ($debug) echo "                 ",$this->name," - .\r\n";
                if (false === $node["__open"]()) return false;
              }
              if (isset($node["__attrs"])) {
                if ($debug) echo "                 ",$this->name," - .\r\n";
                $attrs = array();
                if ($this->hasAttributes)
                  while ($this->moveToNextAttribute())
                    $attrs[$this->name] = $this->value;
                if (false === $node["__attrs"]($attrs)) return false;
              }
              if ($this->isEmptyElement) {
                if ($debug) echo "               ",$this->name," .   .\r\n";
                if (isset($node["__close"])) {
                  if ($debug) echo "                 ",$this->name," - .\r\n";
                  if (false === $node["__close"]()) return false;
                }
                $node = &$stack[$this->depth];
              }
            } else {
              $skipToDepth = $this->depth;
              if ($debug) echo "[  ]: ",$this->name," -    .        ",$skipToDepth,".\r\n";
            }
          } else {
            if ($debug) echo "(  ): ",$this->name," -    .\r\n";
          }
          break;
        case self::TEXT:
          if ($skipToDepth === false) {
            if ($debug) echo "[     ]: ",$this->value," -  .\r\n";
            if (isset($node["__text"])) {
              if ($debug) echo "                 - .\r\n";
              if (false === $node["__text"]($this->value)) return false;
            }
          } else {
            if ($debug) echo "(     ): ",$this->value," -    .\r\n";
          }
          break;
        case self::END_ELEMENT:
          if ($skipToDepth === false) {
            //  $skipToDepth  ,   ,        ,
            //       .
            if ($debug) echo "[  ]: ",$this->name," -   .   .\r\n";
            if (isset($node["__close"])) {
              if ($debug) echo "                 ",$this->name," - .\r\n";
              if (false === $node["__close"]()) return false;
            }
            $node = &$stack[$this->depth];
          } elseif ($this->depth === $skipToDepth) {
            //  $skipToDepth ,   ,    ,         .
            if ($debug) echo "[  ]: ",$this->name," -   ",$skipToDepth,".    .\r\n";
            $skipToDepth = false;
          } else {
            if ($debug) echo "(  ): ",$this->name," -    .\r\n";
          }
          break;
      }
    }
    return true;
  }

}

ご覧のとおり、このクラスは標準のXMLReaderクラスの機能を拡張し、それに1つのメソッドを追加しました。

xmlStruct($xml, $structure, $encoding = null, $options = 0, $debug = false)

パラメーター：

  • $ xml、$ encoding、$ options：XMLReaderの場合と同様：:: xml（）
  • $構造：ファイルの操作方法を完全に説明する連想配列。その外観は事前にわかっていることが理解されており、私たちはどのタグと何をすべきかを正確に知っています。
  • $ debug：（Debugバージョンのみ）デバッグ情報を出力するかどうか（デフォルトではオフ）。

$構造 引数。

これは連想配列であり、その構造はXMLファイル内のタグの階層を繰り返します。さらに、必要に応じて、各構造要素にハンドラー関数（対応するキーを持つフィールドとして定義）を含めることができます。

  • "__open" -タグを開くときの関数-関数（）
  • "__attrs" -タグ属性を処理する関数（ある場合）-関数（$ assocArray）
  • "__text" -タグにテキスト値がある場合の関数-関数（$テキスト）
  • "__close" -タグを閉じるときの関数-関数（）

ハンドラのいずれかがfalseを返す場合、解析は中止され、xmlStruct（）関数はfalseを返します。次の例は、$構造体引数を作成する方法を示しています。

include_once "xmlreaderstruct.class.php";
$x = new XMLReaderStruct();
$productsSum = 0;
$servicesSum = 0;
$structure = array(
  'shop' => array(
    'record' => array(
      'type'    => array( "__text" => function($text) use (&$currentRecord) {
        $currentRecord['isService'] = $text === 'service';
      } ),
      'qty'     => array( "__text" => function($text) use (&$currentRecord) {
        $currentRecord['qty'] = (int)$text;
      } ),
      'price'   => array( "__text" => function($text) use (&$currentRecord) {
        $currentRecord['price'] = (int)$text;
      } ),

      '__open'  => function() use (&$currentRecord) {
        $currentRecord = array();
      },
      '__close' => function() use (&$currentRecord, &$productsSum, &$servicesSum) {
        $money = $currentRecord['qty'] * $currentRecord['price'];
        if ($currentRecord['isService']) $servicesSum += $money;
        else $productsSum += $money;
      }
    )
  )
);
$x->xmlStruct(file_get_contents('example.xml'), $structure);
echo 'Overal products price: ', $productsSum, ', Overal services price: ', $servicesSum;
