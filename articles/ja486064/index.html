アニメーション化されたCSSスライドショーを作成する



こんにちは！

JSにうんざりしたら、CSSに切り替えてください！

仕事


CSSを使用してアニメーションのスライドショーを作成します。スライドショーは、たとえば、特別な場合に、プロジェクターを介して画面に写真を表示するために使用できます。ループを介して、表示された写真は厳粛なスピーチの背景として使用できます。

可能な解決策


スライドショーでは、奇妙な「マクロ写真」PawełCzerwińskiを撮影します（これはかっこいいです）。

じゃ、行こう。

マークアップは次のようになります。

<div class="slideshow">
    <img src="https://thebestcode.ru/media/animatedSlideshow/1.jpg" alt="#" class="i_1">
    <p class="p_1">a-slide</p>
    <img src="https://thebestcode.ru/media/animatedSlideshow/2.jpg" alt="#" class="i_2">
    <p class="p_2">b-slide</p>
    <img src="https://thebestcode.ru/media/animatedSlideshow/3.jpg" alt="#" class="i_3">
    <p class="p_3">c-slide</p>
    <img src="https://thebestcode.ru/media/animatedSlideshow/4.jpg" alt="#" class="i_4">
    <p class="p_4">d-slide</p>
    <img src="https://thebestcode.ru/media/animatedSlideshow/5.jpg" alt="#" class="i_5">
    <p class="p_5">e-slide</p>
    <img src="https://thebestcode.ru/media/animatedSlideshow/6.jpg" alt="#" class="i_6">
    <p class="p_6">f-slide</p>
</div>

ここで何を観察していますか？

写真は、要素を水平方向および垂直方向に配置するために使用されるスライドショークラスにラップされます。

.slideshowには次のスタイルがあります。

.slideshow {
    width: 100vw;
    height: 100vh;
    display: flex;
    justify-content: center;
    align-items: center;
}

6枚の写真があります。各写真には、「i_number of photos」というクラスが割り当てられています。各写真の下に、「p_number of photo」というクラスの署名があります。

Imgには次のスタイルがあります。

img {
    position: absolute;
    max-width: 90vw;
    max-height: 80vh;
    opacity: 0;
    animation: slideshow 36s ease-in-out infinite;
}

アニメーション時間は、写真の枚数と各写真の表示時間に依存します。この場合：6枚の写真をそれぞれ6秒間。

アニメーションに移りましょう：

@keyframes slideshow {
    from {
        opacity: 0;
        filter: grayscale(100%);
    }

    5% {
        opacity: 1;
        filter: grayscale(50%);
    }

    10% {
        opacity: 1;
        filter: grayscale(0);
        transform: scale(1.1);
    }

    15% {
        opacity: 1;
        filter: blur(0);
    }

    20% {
        opacity: 0;
        filter: blur(2px);
        transform: scale(0.9);
    }

    25% {
        opacity: 0;
    }
}

アニメーションの25％のみを使用しますが、6つのステップで行います。 25％は4番目の部分なので、写真はわずかに重なります。 「フェードインフェードアウト」効果を実装しているため、私たちは透明性をもって作業します。これがまさに私たちが必要としていることです。

「フィルター：グレースケール」を使用して、表示された（変色した）写真の色を飽和させます。「変換：スケール」を使用して、出力時に写真を少し前に、置換時に少し後ろに拡大します。「フィルター：ぼかし」を使用して、置換された写真をぼかします。

次に、個々の写真を操作します。

.slideshow .i_1 {
    animation-delay: 6s;
    transform: translateX(-100px) rotate(-3deg);
}

.slideshow .i_2 {
    animation-delay: 12s;
    transform: translateX(100px) rotate(3deg);
}

.slideshow .i_3 {
    animation-delay: 18s;
    transform: translateY(-100px) rotate(3deg);
}

.slideshow .i_4 {
    animation-delay: 24s;
    transform: translateY(100px) rotate(-3deg);
}

.slideshow .i_5 {
    animation-delay: 30s;
    transform: rotate(-6deg);
}

.slideshow .i_6 {
    animation-delay: 36s;
    transform: rotate(6deg);
}

各写真のアニメーションの遅延を設定します（最初の6秒と後続の各6秒）。変更のために、「transform：translate rotate」を使用して各写真をシフトおよび回転します（値は経験的に選択されました）。

写真が完成しました。次に、署名について説明します。

Pには次のスタイルがあります。

p {
    margin: 0;
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
    color: #e4e4e4;
    text-shadow: 0 0 1px #000;
    text-transform: uppercase;
    font-size: 2.8em;
    opacity: 0;
    animation: title 36s linear infinite;
}

キャプションのアニメーション時間は、写真のアニメーション時間に対応しています。

署名アニメーションはずっと簡単に見えます：

@keyframes title {
    from {
        opacity: 0;
    }

    5% {
        opacity: 1;
    }

    20% {
        opacity: 1;
    }

    25% {
        opacity: 0;
    }
}

また、アニメーション遅延を使用して別の署名を表示します。

.slideshow .p_1 {
    animation-delay: 7s;
}

.slideshow .p_2 {
    animation-delay: 13s;
}

.slideshow .p_3 {
    animation-delay: 19s;
}

.slideshow .p_4 {
    animation-delay: 25s;
}

.slideshow .p_5 {
    animation-delay: 31s;
}

.slideshow .p_6 {
    animation-delay: 37s;
}

キャプションのアニメーション遅延は、対応する写真の値よりも1秒長いことに注意してください。これは、特定の写真の出現と変換後にキャプションを表示するために行われます。
