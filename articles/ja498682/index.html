Goでランダムパスワードジェネレーターを作成する

こんにちは、ハブル！Mihalis Tsoukalosによる記事「Goでランダムで安全なパスワードを作成する」の翻訳を紹介します。

標準ライブラリを使用して、Goで簡単なランダムパスワードジェネレータを作成します。



Goの乱数ジェネレーターは、複雑なパスワードを生成するのに適したオプションです。

Goプログラミング言語で提供される乱数ジェネレーターを使用して、ASCII文字で構成される複雑なパスワードを生成できます。この記事で紹介するコードは読みやすいですが、Goの基本を理解して理解している場合に最適です。プログラミング言語に慣れていない場合は、ツアーオブゴーに進んで詳細を確認してから、ここに戻ってください。

ASCII文字の印刷可能な小数点以下の値が33から126までの範囲。他のASCII値は、パスワードに含めるのに適していません。したがって、この記事で紹介するユーティリティは、この範囲のASCII文字を印刷します。

乱数を作成する


最初のユーティリティはrandom.goと呼ばれ、指定された範囲の指定された数の乱数を生成します。random.goの最も重要な部分は次の関数です。

func random(min, max int) int {
        return rand.Intn(max-min) + min
}

この関数は、Goのrand.Intn（）関数を使用して、指定された範囲に属するランダムな整数を生成します。rand.Intn（）は、[0、n]に属する負でない乱数を返すことに注意してください。引数が負の数の場合、関数はパニックになります。パニックメッセージは次のようになります：Intnへの引数が無効です。math / randパッケージのドキュメントはドキュメントにあります。

random.goユーティリティは、生成された整数の最小値、最大値、および生成された整数の数の3つのコマンドラインパラメーターを受け入れます。
random.goをコンパイルして実行すると、次の出力が生成されます。

$ go build random.go
$ ./random
Usage: ./random MIX MAX TOTAL
$ ./random 1 3 10
2 2 1 2 2 1 1 2 2 1

Goでより安全な乱数を生成する場合は、Goライブラリのcrypto / randパッケージを使用します。

ランダムなパスワードを作成する


2番目のユーティリティであるrandomPass.goは、ランダムなパスワードを生成します。 randomPass.goはrandom（）関数を使用して、次のGoコードを使用してASCII文字に変換される乱数を生成します。

for {
        myRand := random(MIN, MAX)
        newChar := string(startChar[0] + byte(myRand))
        fmt.Print(newChar)
        if i == LENGTH {
                break
        }
        i++
}

MINは0、MAXは94、startCharは！、ASCIIテーブル（ASCII 10進コード33）の最初の印刷可能な文字です。したがって、生成されるすべてのASCII文字は後です！そして〜文字の前にあり、これは126のASCII 10進コードを持っています。

したがって、生成される各乱数はMINより大きく、MAXより小さく、ASCII文字に変換されます。このプロセスは、生成されたパスワードが必要な長さになるまで続きます。

randomPass.goユーティリティは、生成されたパスワードの長さを決定する1つの（オプションの）コマンドラインパラメーターを受け入れます。デフォルト値は8で、これはかなり一般的なパスワードの長さです。 randomPass.goユーティリティを実行すると、次の出力が生成されます。

$ go run randomPass.go 1
Z
$ go run randomPass.go 10
#Cw^a#IwkT
$ go run randomPass.go
Using default values!
[PP8@'Ci
