Flask-DJ：フラスコ上のプロジェクトのDjango（mvc）構造


みなさん、こんにちは！


私が最初にFlaskに出会ったとき、私はすぐにプロジェクトアーキテクチャの構築について質問しました。


Habréに関するいくつかの記事（https://habr.com/en/post/275099/およびhttps://habr.com/en/post/421887/）を読んだ後、Djangoでプロジェクトを作成した経験を思い出し、ツールを作成することにしましたおかげで、アーキテクチャについて考える必要はありませんが、Flaskのすべての機能を使用できます。


取り付け


$ pip install Flask-DJ

プロジェクトの作成


( -t -st)


$ flask-dj startproject app

setup.py
(
need_templates=True, need_static=True).


from flask_dj import startproject
from os import getcwd
your_project_name = 'app'
project_dir = getcwd()
startproject(your_project_name, project_dir)

URL


index:


# index/urls.py
from utils.urls import relative_path
from .views import index

urlpatterns = [
    relative_path("", index),
]

:


# app/urls.py
from utils.urls import add_relative_path, include

urlpatterns = [
    add_relative_path("/", include("index.urls")),
]

サーバーを起動します


$  python manage.py runserver
