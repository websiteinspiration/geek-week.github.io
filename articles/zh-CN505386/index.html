在Netgear Stora上安装Debian

前几天，我掌握了这个奇迹：netgear ms 2000-stora。我决定立即放弃使用嵌入式OS，并在硬盘驱动器上安装debian。

网络上的信息有点分散，链接早已消失，所以我决定在stora上更新debian安装过程。谁在乎，欢迎来猫。

首先，我们需要安装系统的映像：我在这里拿到了。下载两个文件。这些文件将写入以fat32格式化的闪存驱动器的根目录。
还需要一个USB转UART PL2303TA转换器。

您还将需要用于连接铁片的软件，例如，超级端子或腻子（我从来没有用腻子

来连接它：我一直疯狂，所以我使用超级端子。要用电缆连接那块铁，我需要先将其拆开。过程很简单，所以我不会描述它好了，您一定不要忘记将硬盘驱动器插入将要进行实际安装的侧面的第一个插槽中。

解析硬件后，我们连接适配器。请注意，不要连接红色的电线，也就是说，您只需要连接3根电线（来自电池：黑色）即可。 ，绿色，白色）。
因此，电线已连接，驱动器已钩住。在com端口驱动程序中，我们设置参数：速度115200，位数8，停止位数1，无奇偶校验。之后，打开铁片并在端子中连接它。当您看到按任意键...按任意键进入u-boot加载程序。

进入引导加载程序后，设置网络参数，初始化USB设备，检查闪存驱动器上是否有必要的文件，将这些参数保存到设备的内存中并重新启动：

队伍 
usb reset
fatls usb 0
setenv mainlineLinux yes
setenv arcNumber 2743
setenv ipaddr your_IP
setenv gatewayip your_GW_IP
setenv dnsip your_DNS_IP
saveenv
reset

重新引导后，输入命令以开始debian安装：

usb reset
fatload usb 0 0x200000 uImage
fatload usb 0 0x800000 uInitrd
setenv bootargs console=ttyS0,115200n8 base-installer/initramfs-tools/driver-policy=most
bootm 0x200000 0x800000

之后，将以文本模式进行标准的debian安装。我们安装系统，安装后重新引导，输入uboot并输入命令以从硬盘驱动器引导设备：

setenv bootcmd_ide 'ide reset; ext2load ide 0 0x200000 /uImage; ext2load ide 0 0x800000 /uInitrd'
setenv bootcmd 'setenv bootargs $(console) root=/dev/sda2; run bootcmd_ide; bootm 0x200000 0x800000'
saveenv
reset

重新引导后，将从最初寻求的硬盘驱动器中加载debian。
