Tipps zur Auswahl eines 2G / 3G / 4G / 5G-Signalverstärkers

Ein Artikel über Optionen zur Verstärkung eines Mobilfunksignals an verschiedenen Standorten: eine Sommerresidenz, ein Büro, ein Lagerhaus ... sowie eine kurze Übersicht über beliebte Angebote auf dem Markt.


Es gibt zwei Hauptoptionen zur Verbesserung der Mobilfunkkommunikation:


  1. Verwendung einer Antenne, eines Modems und eines Routers.
  2. Verwendung eines aktiven Verstärkers (Repeaters).

Die erste Option zur Stärkung des Internet 3G / 4G / 5G verbessert die Sprachkommunikation nicht im für uns üblichen Format (Sie können jedoch Anrufe über Instant Messenger verwenden: WhatsApp, Viber, Skype, Telegramm).


Die zweite Option ist universell - sie stärkt das Internet und die Sprachkommunikation, weist jedoch eine Reihe von Einschränkungen auf (zu den folgenden Einschränkungen).



Verstärkung des 3G / 4G-Internets mit Antenne, Modem und WLAN-Router


Das System besteht aus zwei Knoten: einer Antenne auf der Straße und einem Modem + WLAN-Router in Innenräumen.



Externe Antenne für 3G / 4G-Modem / Router



4G-Modem, das über Adapter an eine Straßenantenne angeschlossen und in einen WLAN-Router eingesetzt wird


Wie es funktioniert? Die Antenne ist auf die vorgeschlagene Basisstation des Mobilfunkbetreibers gerichtet. Dann geht das Signal an ein 4G-Modem mit einem Router, der das Internet durch Verteilen eines Wi-Fi-Signals verteilt. Die Option ist bei privaten Mastern beliebt, um ein Vorort-Internet zu installieren. Billig und funktioniert!

  5. Wenn der RSSI-Signalpegel am Installationspunkt der Außenantenne -95 oder schlechter ist (z. B. -105), ist es praktisch unmöglich, ein solches Signal mit einem Repeater herauszuholen. Wenn dies gelingt, beträgt der Radius des Signals von der internen Antenne weniger als einen Meter, selbst wenn leistungsstarke Verstärker installiert werden. In extremen Fällen müssen Sie die externe Antenne stark anheben!
