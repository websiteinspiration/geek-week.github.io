サイトレイアウトの崇高なテキスト3。外観をカスタマイズしてプラグインをインストールします。初心者向けガイド

今日は、長年サイトのレイアウトに使用してきたコードエディターについて説明します。高速で便利な、完全にカスタマイズ可能なSublime Text 3。



外観を一からカスタマイズし、AutoFileName、BracketHighliter、ColorHighliter、Emmet、GotoCSSDeclaration、Tagプラグインを使用して、サイトのレイアウトを大幅に支援します！言葉から行動まで！

プログラムの様子


まず、Sublime Text 3をインストールした後、プログラムの外観をカスタマイズします。

サイドパネル


パネルを表示するには、[表示]-[サイドバー]-[開いているファイルを表示]に移動します。このパネルには、プロジェクト、つまりレイアウトのフォルダーとファイルが表示されます。わかりやすくするために、そのうちの1つを追加します。そこで必要なのは、プロジェクトフォルダーをパネルにドラッグするだけです。



カラースキーム


次に、プログラムとコードの配色を調整します。わかりやすくするために、たとえばHTMLコードで任意のファイルを開きます。これを行うには、目的のファイルをダブルクリックします。次に[ 設定]-[配色]に移動します。ウィンドウが開き、インストールされているカラーソリューションが表示されます。トピックを選択すると、コードの外観とプログラム自体が変化します。



個人的には、事前定義されたオプションのどれも自分に合っていないため、追加の配色をインストールします。それらの多くはSublime packagecontrol.ioへの追加とともにサイトで見つけて表示することができます。インストールするスキームがわかっているので、これを実行します。

新しい配色、プラグイン、その他のアドオンをインストールするには、まずこれらのアドオンのインストーラーをインストールする必要があります。複雑に聞こえますが、実際にはすべてがはるかに簡単です。

メニューの[ツール ] -[コマンド] [ パレット ]をオンにするか、Ctrl + Shift + Pを押します。このウィンドウが検索で開きます。[ パッケージコントロールのインストール]項目が表示されるまで検索フィールドに「install」と入力し、クリックします。数秒後、パッケージコントロールが正常にインストールされたというメッセージが表示されます。これで、目的の配色の設定を開始できます。 これを行うには、Ctrl + Shift + Pを押してコマンドパレットを再度呼び出します。パッケージコントロールが表示されるまで、検索フィールドに「install」と入力します。パッケージのインストール



（アドオンパッケージのインストール）をクリックします。少し待った後、Sublimeで利用可能なすべてのアドオンのリストを含むウィンドウが表示されます。まあ、ほとんどの人。

検索フィールドに、インストールするアドオンの名前を入力します。現時点では、これはBase16 Color Schemesと呼ばれる配色です。より正確には、多くの配色を持つパッケージ全体です。見つかったアイテムをクリックして、インストールを待ちます。インストールの進行状況インジケーターは、プログラムの下部パネルに表示されます。すべてがインストールされているという短いメッセージを受け取ったので、回路を変更します。これを行うには、おなじみのメニュー項目である[ 設定]-[カラースキーム]を開きます。開いたウィンドウには、さまざまなオプションがすでにたくさんあるのがわかります。私はオーシャンスキームに興味があります。検索に名前を入力します。そのうちの2つがあります。1つは他と対照的です。



私のお気に入りの配色が設定されていますが、もちろん他の色も自由に選択できます。

プログラムの全体的な外観がより暗いのが好きなので、[ 設定]-[テーマ]に切り替え、[ アダプティブ]に切り替え



ます。色の組み合わせは、コードの操作にとって非常に重要です。これは、知覚だけでなく、目の疲労や健康にも影響するためです。だから、あなたの配色を拾って、それについて何も気にしないようにしてください。そして、仕事の休憩について覚えておいてください！

追加の設定


コードの認識を高め、コードの操作を快適にするために、次の追加設定が役立ちます。[設定]-[設定]に移動します。このようなウィンドウは2つの部分に分かれて開かれます。デフォルト設定の左側では変更できませんが、右側ではこれらの設定を再割り当てできます。次の各設定はカンマで区切る必要があります。ここに私が使用する設定があります：

"margin": 0, //  
"font_size": 10, //    
"draw_indent_guides": true, // /  
"draw_white_space": "all", //   
"tab_size": 3, // 
"remeber_open_files": true, //

それぞれの設定について説明します。 

"margin": 0, //

プログラム内のコードとサイドバーの間の奇妙なインデントを削除し、全体像をよりコンパクトにします。



"font_size": 10, //

既定のフォントサイズを設定します。コントロールを押したままマウスホイールを回すことで、コードの操作中にフォントサイズをすばやく変更することもできます。

誤ってサイズを突然変更した場合は、[ 設定]-[フォント]-[リセット]に移動してリセットできます。



"draw_indent_guides": true, // /

これらの行は、開始要素と終了要素を視覚的に接続しているため、初心者には非常に役立ちます。これにより、コードの向きが大きく変わります。

個人的には、2つの理由でそれらを無効にします（false）。1つ目は、私は初心者ではなく、彼らなしでかなりうまくいくことができるということです。2つ目の理由は次の設定です。



"draw_white_space": "all", //

非表示のタブとスペースを表示します。つまり、これはコードの一部ではなく、使いやすくするための単なるマークアップです。そして今、あなたがdraw_indent_guidesとdraw_white_spaceの両方をオンにすると、あなたは個人的に私を悩ませているスラリーを手に入れます。何を使うべきか、何を決めないか。



"tab_size": 3, //

タブが押されたときに線幅のサイズをインデントします。個人的に、私は3番目のサイズが好きです:)



"remember_open_files": true, //

たとえば、私たちは働いて、働いて、それからプログラムを突然閉じました:)それは起こりますか？いつも持っているので、この設定にはSublimeの「メモリ」が含まれており、次にプログラムを開くと、最新のファイルがすべて開かれます。そして、はい、メモリはコンピュータを再起動した後にも機能します。

作業ウィンドウを分割する


作業ウィンドウを分割すると、コードの認識がさらに向上し、レイアウトプロセスがスピードアップします。

ほとんどの場合、たとえばindex.html style.cssやある種のjsファイルなど、いくつかの異なるファイルを使用する必要があるため、コードを含むウィンドウをセクターに分割すると、さらに便利になります。いくつかのオプションがあります。[表示]-[レイアウト]に移動して、ここにあるものを確認します。



したがって、Singleは現在のようにパーティションのないビューであり、Columns：2、3、4は垂直分割です。たとえば、[列：2]をクリックすると、コードを含む画面が半分に分割され、ファイルの一部を空の列に移動できます。たとえば、HTMLの左側にあるCSSと右側にあるJSを使用すると、作業がより視覚的で迅速になります。



画面を水平に分割するオプションもあります：行：2と行：3、たとえば、行：3をクリックすると、3つの水平ウィンドウが表示され、JSファイルを3番目のウィンドウに転送できます。最後のオプションはグリッド：4で、画面を水平および垂直の両方に半分に分割します。結果は4つのウィンドウです。ホットキーAlt + Shift +対応する番号を使用してモードを切り替えることができることに注意してください。



だからあなたの好みに合わせて幅広い選択肢があります。ほとんどの場合、1つの垂直分割を使用します。

構文のハイライト


最終的な外観のカスタマイズは、構文の強調表示になります。言葉はひどいですが、実際にはすべてが簡単です。

構文の強調表示は、コード内の1つまたは別の要素の色を使用した特別な色です。たとえば、HTMLファイルのこの配色では、タグは赤で強調表示され、クラス名は緑、テキストは白などです。これにより、コードの認識がより明確になります。プログラムウィンドウの右下隅に注意してください。現在アクティブなファイル、CSS HTML JSなどによって、この特定のファイルのコードの表示が切り替わります。

すべてがすでに正常に動作している場合、なぜ何かする必要があるのですか？実際のところ、初期のSublimeパッケージでは、構文の強調表示パッケージがすべてのフォーマットにインストールされているわけではありません。

たとえば、今後SCSSファイルを操作する予定で、それを開くと、白い文字のセットの固体シートが表示されます。あまり便利ではないですよね？



そのため、そのような場合は、目的のパッケージを追加でインストールします。

使い慣れたコマンドパレットを開くには、[ ツール]-[コマンドパレット ] を選択するか、Ctrl + Shift + Pを押して「install」と入力し、[ パッケージコントロールのインストール ] をクリックして、SCSSアドオンを探します。それをクリックすると、インストール後にディスプレイをSCSSに切り替えることができます。今、すべてがよりきれいに見えますよね？



コードの作成にはかなりの時間を費やしているため、エディターの外観のカスタマイズは重要です。それは私たちの疲労、生産性、健康、そして私たちの気分だけに影響します。

次に、いくつかのプラグインとアドオンをインストールします。それらの数は非常に多く、私が積極的に自分で使用していて、レイアウトプロセス全体を時々高速化する主なものについて説明します。始めましょう！

プラグイン


AutoFileName


実際のところ、レイアウトのプロセスでは、さまざまな画像を接続する必要があります。このプラグインは、時々接続プロセスを容易にします。インストールしてください。押してCtrlキー+ Shiftキー+ P、その後パッケージ制御、インストールのために見AutoFileNameをインストールします。ここで画像を接続してみましょう。ローカルファイルのリストを含むドロップダウンメニューの外観にすぐに気づきます。画像のある目的のフォルダーに移動して、必要なファイルとそのサイズを確認します。これも便利です。



ブラケット


BracketHighliterは、バックライトの引用として翻訳できます。インストールしてください。Ctrl + Shift + Pを押して、Package Controlをインストールし、BracketHighliterを探します。インストール。インストールが成功したことを示すメッセージを含む別のファイルも開きます。これで、コードでは、開始要素と終了要素、たとえば開始タグと終了タグ、開始引用符と終了引用符、開始括弧と終了角かっこが強調表示されます。これは、コードの方向付けに大きく役立ちます。



カラー


このコードに注意してください。これはカラーコードですが、どれが明確ではありません。



プラグインをインストールしましょう。Ctrl + Shift + P、パッケージコントロールのインストール、ColorHighliterを探します。この場合も、別のファイルが開き、インストールの成功に関するメッセージが表示されます。そして、いくつかのエラーメッセージは無視してください。コードを色で見て、英数字コードの背後に隠されている色を確認します。プラグイン自体は、[ ツール]-[ColorHighliter]メニューで追加設定できますが、ここでは詳しく説明しません。すべてがはるかに良くなった。



偉大で強大なエメット


定められたスキームに従って設置。Ctrl + Shift + P、Package Controlのインストール、Emmetを探します。別のファイルが開き、プログラムを再起動してインストールを完了するように求めるメッセージが表示されます。Sublimeを閉じて開きます。すべてがインストールされたというメッセージが表示されます。

それで、エメットは何をしますか？


HTMLファイルに移動して、.blockクラスを使用してdivタグを書き込もうとします。これを行うには、末尾に「block」と書き込み、Tabキーを押すだけです。取得します：

<div class="block"></div>

より複雑なエントリを作成するには、たとえば、.block classのdivタグにspanタグを挿入し、次のエントリを使用します。

.block>span

Tab キーを押して、以下を取得します。

<div class="block"><span></span></div>

クルチャク、本当に！？

しかし、それだけではありません。独自のスニペット、つまりテンプレート/テンプレートを作成できます。
これを行うには、[設定]-[パッケージ設定]-[Emmet]-[設定ユーザー]に移動します。ここでは、次のように記述します。

{
	"snippets":{
		"html":{
			"snippets":{
				"bl":"<div class=\"block\"></div>"
			}
		}
	}
}

次に、エントリを呼び出します。

<div class="block"></div>

blと 書いてTabキーを押すだけで十分です。

このことから、Emmetの可能性は無制限であり、好きなようにスニペットを構成して非常にすばやく作業できます。プラグインの機能に関する詳細情報は、https：//emmet.ioにあります。

GotoCSSDeclaration


プラグインは、コード内のナビゲーションに役立ちます。以前のプラグインとまったく同じようにインストールします。次に、プラグインを操作するためのホットキーを割り当てます。このためには、[ 設定]-[キーバインディング]に移動し、次のコードを記述します

{
"keys": ["ctrl+1"], "command": "goto_css_declaration",
"args": {"goto": "next"}
},

[" ctrl + 1 "]は、目的のキーの組み合わせです。Cntr + Sを維持します。
そして、これはそれがどのように機能するかです。CSSファイルとHTMLファイルを開いておく必要があります。CSSは通常非常に大きく、すぐに何かを見つけることは困難ですが、それは過去のものです。たとえば、content-mainpageクラスの CSSを変更し、マウスでクリックしてctrl + 1を押す必要があります。ほら、CSSファイルの適切な場所にいます。



鬼ごっこ


さて、ここまで読んだ人には、インストールが簡単ではない別のプラグインを紹介しますが、それだけの価値はあります。

彼の名前はタグです。手動でインストールする必要があります。このリンクによると、プラグインを使用してアーカイブをダウンロードできます。次に、特別なプログラムフォルダーに解凍します。 Windows 10の場合、これはC：\ Users \ [ユーザー名を置き換えます] \ AppData \ Roaming \ Sublime Text 3 \ Packagesです。次に、Sublimeを閉じて再度開き、[ 設定]-[キーバインディング]を開いて以下を追加して、ホットキーを追加します。

{ "keys": ["ctrl+`"], "command": "tag_classes" }

ここで、ctrl + `はキーの組み合わせです。

それで、なぜ私たちはこのようにすべての困難な道を行きましたか原則として、コードを記述するプロセスでは、最初にHTMLを処理し、次にクラス名をCSSに転送して、すでにそこで処理します。したがって、この転送は手動で長時間行う必要があります。そしてここで、Tagプラグインが役に立ちます。とりあえず、必要なHTMLコードを選択し、ctrl + `キーの組み合わせ



を押して、コピー結果をCSS ctrl + vに貼り付けるだけで十分です。YouTubeチャンネル「Freelancer for life」でシリーズ「ウェブサイト組版用のプログラムとツール」から
