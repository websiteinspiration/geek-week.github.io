Während des Workflows wurden zunächst alle Diagramme in MS Excel erstellt. Es ist sehr praktisch, schnell Daten zu skizzieren, ein paar Kurven zu erstellen und fortzufahren. Bei der Arbeit in MS Excel gibt es jedoch keine Kontrolle über die resultierende Zeichnung. Daher verwende ich beim Schreiben von Ergebnissen für Artikel häufig Gnuplot.

Aber das scheint nicht viel zu sein, was ich bekommen wollte.

Die offizielle Dokumentation half dabei, die richtigen Namen der Beschriftungen auf der Achse ( xtic ) und der Beschriftungen im Diagrammbereich ( sprintf ) zu verstehen :

set xtics rotate by -60
Label(String,Size) = sprintf("{/=%d %s}", Size, String)

plot "data.csv" using 2:3:xtic(2) title "1" w lp ls 1 ,\
     "data.csv" using 2:4 title "2" w lp ls 2 ,\
     "data.csv" using 2:($3>$4?$3+20:$4+20):(Label(stringcolumn(2), 12)) with labels notitle

Die Entscheidung wurde im Prozess des Streifens durch Foren und Artikel getroffen - nummerieren Sie die Zeilen .

Am einfachsten ist es, manuell zu nummerieren.

Die zweite Möglichkeit besteht darin, automatisch zu nummerieren:

 nl -v 0 -s ';' -w 1 data.csv >> data_nums.csv

Wenn Sie auskommentierte Zeilen ausschließen müssen, hat der Befehl die folgende Form:

grep -v "^#" data.csv | nl -v 0 -s ';' -w 1  >> data_nums.csv

Wenn Sie mit einem solchen Datensatz arbeiten, verteilt gnuplot die Daten in regelmäßigen Abständen entlang der Achse:



Zusammenfassungsskript:

#!/usr/bin/gnuplot -persist

set encoding utf8
set terminal pngcairo size 640,400 enhanced monochrome
set output "img.png"

set ylabel "Y"
set xlabel "X"
set key left top samplen 5 spacing 1.15 width 2
set xrange [-0.5:24.5]
set mytics 2
set xtics rotate by -60
set grid
set pointsize 0.5
set datafile separator ";"
set style line 1 linetype 1 pointtype 7 linewidth 2 linecolor black
set style line 2 linetype 2 pointtype 7 linewidth 2 linecolor black

Label(String,Size) = sprintf("{/=%d %s}", Size, String)

plot "data_nums.csv" using 1:4:xtic(3) title "1" w lp ls 1 ,\
     "data_nums.csv" using 1:5 title "2" w lp ls 2 ,\
     "data_nums.csv" using 1:($4>$5?$4+20:$5+20):(Label(stringcolumn(2), 10)) with labels notitle


Update:
Dank AchtsamkeitDeosisund ein genauer Verweis auf die Dokumentation (Absatz Pseudospalten ) stellte sich heraus, dass Sie auf künstliche Nummerierung verzichten können.
Für eine gleichmäßige Verteilung der Daten entlang der Achse können Sie die Nullspalte ($ 0) verwenden, die Zeilenindizes bereitstellt.

In diesem Fall ist es möglich, den ursprünglichen Datensatz - data.csv - zu verwenden und die folgenden Änderungen im Skript vorzunehmen:
plot "data.csv" using 0:3:xtic(2) title "1" w lp ls 1 ,\
     "data.csv" using 0:4 title "2" w lp ls 2 ,\
     "data.csv" using 0:($3>$4?$3+20:$4+20):(Label(stringcolumn(2), 12)) with labels notitle
