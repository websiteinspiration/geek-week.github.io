Windows 10でWindows Defenderを完全に無効にする方法



Windows 10には、Windows Defenderウイルス対策（ "Windows Defender"）が組み込まれており、コンピューター、データをウイルス、スパイウェア、ランサムウェア、その他多くの種類のマルウェアやハッカーから保護します。

また、ほとんどのユーザーには組み込みのセキュリティソリューションで十分ですが、このプログラムを使用したくない場合もあります。たとえば、オンラインにならないデバイスを構成した場合。このプログラムによってブロックされたタスクを完了する必要がある場合。組織のセキュリティポリシー要件を満たす必要がある場合。

唯一の問題は、Windows Defenderを完全に削除または無効にできないことです。このシステムはWindows 10に深く統合されています。ただし、ローカルグループポリシー、レジストリ、またはWindows設定を使用して、ウイルス対策を無効にできるいくつかの回避策があります。 [セキュリティ]セクション（一時的）。

Windowsのセキュリティ設定でWindows Defenderを無効にする方法


特定のタスクを実行する必要があり、Defenderを完全に無効にする必要がない場合は、これを一時的に行うことができます。これを行うには、[スタート]ボタンの検索を使用して、[Windows Defenderセキュリティセンター]セクションを見つけ、その中の[ウイルスと脅威からの保護]を選択します。



そこで、「ウイルスやその他の脅威に対する保護」セクションに移動し、「リアルタイム保護」スイッチをクリックします。



その後、アンチウイルスはリアルタイムのコンピュータ保護を無効にします。これにより、アプリケーションをインストールしたり、アンチウイルスが必要なアクションをブロックしたために利用できなかった特定のタスクを実行したりできます。

リアルタイム保護を再度有効にするには、コンピューターを再起動するか、すべての構成手順をもう一度実行しますが、最後の手順でスイッチをオンにします。

このソリューションは永続的なものではありませんが、特定のタスクでWindows 10アンチウイルスを無効にするのに最適です。

グループポリシーを使用してWindows Defenderを無効にする方法


Windows 10 ProおよびEnterpriseのバージョンでは、ローカルグループポリシーエディターにアクセスできます。ここで、次のようにDefenderを永続的に無効にでき

ます。[スタート]ボタンから、gpedit.msc実行可能スクリプトを実行します。ポリシーエディタが開きます。次のパスに移動します：コンピューターの構成>管理用テンプレート> Windowsコンポーネント> Windows Defenderウイルス対策。



ダブルクリックして[Windows Defenderウイルス対策プログラムをオフにする]オプションを開きます。「有効」設定を選択してこのオプションを有効にし、それに応じてディフェンダーを無効にします。



「OK」をクリックして、コンピュータを再起動します。

その後、アンチウイルスはデバイス上で永久に無効になります。ただし、盾のアイコンがタスクバーに残ることに気づくでしょう。このアイコンは、ウイルス対策自体ではなくWindowsセキュリティアプリケーションに属しているため、そうであるはずです。

気が変わった場合は、これらの手順を繰り返すことでいつでもDefenderを再度有効にできます。最後の手順で[未設定]オプションを選択すると、その後コンピューターを再起動する必要があります。

レジストリを介してWindows Defenderを無効にする方法


ポリシーエディターにアクセスできない場合、またはWindows 10 Homeをインストールしている場合は、Windowsレジストリを編集して、Defenderを無効にすることができます。

レジストリを編集することは危険であり、この問題でエラーが発生すると、現在インストールされているWindowsのコピーに修復不可能な損傷を与える可能性があることを思い出します。編集を開始する前に、システムをバックアップすることをお勧めします。

レジストリで完全に無効にディフェンダーに、次のパスに「スタート」ボタンとナビゲート通過にregeditプログラムを起動します。

HKEY_LOCAL_MACHINE \ SOFTWARE \ポリシー\マイクロソフト\ Windows Defenderの

ヒント：このパスをコピーし、レジストリエディタのアドレスバーに貼り付けることができます。



次に、Windows Defenderキー（ディレクトリ）を右クリックし、[新規]と[DWORD（32ビット）値]を選択します。新しいキーにDisableAntiSpywareという名前を付け、Enterキーを押します。次に、ダブルクリックしてキーエディターを開き、1に設定します。



[OK ]をクリックして、コンピューターを再起動します。

その後、Windows Defenderはシステムを保護しなくなります。これらの変更を元に戻す場合は、すべての手順を繰り返しますが、最終的にはこのキーを削除するか、0に設定します。

推奨事項


Windows Defenderを無効にする方法はいくつかありますが、ウイルス対策プログラムのないコンピューターを使用することはお勧めしません。ただし、この機能を無効にする方が適切な場合があります。また、サードパーティのウイルス対策プログラムをインストールする場合は、Defenderを手動で無効にする必要はありません。インストール中に自動的に停止するためです。
