2020年最佳小型企业服务器



尽管云服务器被广泛采用，但许多人选择提供更多控制的本地服务器。TechRadar门户汇总了前十大服务器的列表，从为小型企业设计的服务器到大型公司可能需要的服务器。

1.戴尔PowerEdge T30



拥有众多连接

处理器的至强之梦：英特尔至强E3-1225 v5 |英特尔®开发人员专区显卡：Intel HD Graphics P530 |内存：64GB |存储：最多六个SATA HDD |连接接口：10个USB端口，HDMI，DisplayPort，VGA，串行端口，PS2，千兆以太网|尺寸（宽x长x高）：17.5厘米x 43.5厘米x 36厘米

+快速四核Xeon处理器
+可以容纳六个硬盘驱动器

-一个千兆位网络端口
-无热插拔驱动器

周围有很多不同的服务器，有时您需要研究整个价目表才能找到最适合您的小型服务器。看看Dell PowerEdge T30-您可以购买用于自我组装和配置的版本，也可以以完整的形式购买。T30针对入门级和小型办公市场，具有扩展到小型塔形建筑物的巨大潜力，并且还具有许多默认服务器功能，使其成为办公工作站的理想替代产品。

2.戴尔PowerEdge T20



展示组装解决方案的价格有多便宜。

处理器：Intel Pentium G3220 |图形：Intel HD Graphics |内存：4GB |储存：不包括|连接接口：多达USB端口（4 x USB 3.0），2 x DisplayPort，VGA，串行，2 x PS2，千兆以太网|尺寸（宽x长x高）：17.5厘米x 43.5厘米x 36厘米

+非常便宜
+紧凑型迷你塔，可轻松访问内部部件

-G3220更像是台式机，而不是服务器处理器
-无驱动器和操作系统

服务器不必昂贵且庞大。看看PowerEdge T20：戴尔拥有20年服务器制造经验中的最深厚知识。尽管构建版本没有硬盘，但它肯定是便宜的-不久前它的价格还不到130美元（考虑到现金返还-您应该始终跟踪这些机器）。

该服务器具有基于Haswell微体系结构的处理器，该处理器可加速至3 GHz，并支持高达32 GB的DDR3 ECC RAM（请注意，该型号随附4 GB）。扩展选项包括四个SATA端口（如果安装8 TB硬盘驱动器，则为32 TB），四个I / O插槽和10个USB端口。令人惊讶的是，这种价格的计算机有两个DisplayPort插槽，一个VGA，两个PS2和一个串行端口。除了千兆以太网端口外，还需要290瓦电源和Intel的RAID控制器。

3.联想ThinkServer TS150



高性能塔式服务器

处理器：英特尔至强E3-1200 v6 |内存：高达64GB |存储：HDD高达40TB |连接接口：8个USB 3.0，串行，视频，2个DisplayPort，音频，千兆以太网|尺寸（宽x长x高）：17.5厘米x 37.5厘米x 43厘米

+ ThinkServer系列中最实惠的型号
+绝对安静

-没什么可抱怨的

联想在2014年从IBM手中抢占了x86服务器市场，并创建了最好的产品阵容之一ThinkServer。 TS150是该系列产品中最实惠的型号之一，它是可与Dell T20媲美的企业级4U服务器。它具有RAID 0、1、10和5支持（通过主板上的控制器）。与竞争对手一样，该服务器最多可容纳四个3.5英寸硬盘驱动器，这意味着在安装相应的硬盘驱动器时，它最多可存储40 TB的存储。相对较新的Intel Xeon处理器E3-1200 v6对于中小型企业应该足够有效。

联想还声称，TS150的工作声音比磁带库中的噪音小，约为26 dB。与其他服务器一样，您将获得令人印象深刻的端口和连接接口：8个USB端口，4个PCI / PCI-E插槽，三个视频输出（包括两个DisplayPort连接器），一个串行端口，千兆以太网和3个用于音频。

强大而多功能的

处理器：英特尔至强可扩展4210 |德州仪器TI.com.cn 显卡：Intel HD Graphics P530 | 内存：16GB | 存储：不包括驱动器| 尺寸（宽x长x高）：18厘米x 65厘米x 47厘米

+可扩展
+体面的设计

-大型机器
-不含硬盘

如果您需要可靠且多功能的服务器，那么HP ProLiant ML350 Gen 10是一个不错的选择。由于安装了英特尔至强可扩展处理器，该型号可提供高性能。您自己需要安装自己的存储设备，但是该服务器支持多种快速操作，包括处理图形和计算任务。尽管HPE ProLiant ML350 Gen 10作为塔式服务器提供，但随着业务的扩展，它可以转换为机架式服务器，这使您可以扩展业务而无需额外的成本，也无需购买全新的服务器。

5.富士通Primergy TX1310 M1



富士通可靠性保证的

无与伦比的处理器：Intel Xeon E3-1226 v3 |显卡：Intel HD Graphics P4600 |内存：16GB |存储：2 x 1TB HDD |连接接口：5个USB 3.0、2个千兆以太网端口|尺寸（宽x长x高）：17.5厘米x 42厘米x 39.5厘米

+令人难以置信的保修
+带有光盘驱动器

-没有RAID 5-
富士通-不是最明显的品牌

在服务器方面，富士通并不是第一个想到的品牌。但是，富士通是为数不多的处理从商业服务器到超级计算机的各种产品的制造商之一。 TX1310是一款入门级的面向业务的服务器，已在运行中证明了自己的优势，并具有市场上最好的可靠性保证。如果服务器在购买后一年内发生故障，富士通不仅会维修或更换它，还将退还您为该服务器支付的款项。

与该细分市场中的大多数型号一样，该服务器设计为24/7运行，并支持RAID 0/1/10（不支持RAID 5）。其中安装了第三版的Intel Xeon E3-1216，两个硬盘驱动器具有1 TB和16 GB的RAM。还有一个优点是具有光驱和两个千兆位以太网端口作为备用。该服务器具有四个DIMM插槽和四个存储托架，可存储多达32 TB的数据，并支持多达32 GB的RAM。

紧凑但噪音很小的服务器

处理器：英特尔赛扬G1610T |图形：Matrox G200 |内存：4GB |存储：不包括驱动器|连接接口：4 x USB 2.0、2 x USB 3.0，VGA，2 x千兆以太网|尺寸（宽x长x高）：23厘米x 24.5厘米x 23厘米

+专业的构建质量，易于使用

在增长最快的细分市场之一中，只有一家公司占据主导地位。 HP Proliant Microserver Gen8成功地将功能和许多折扣的吸引人的组合进行了竞争-最终占领了市场。由于专业消费者的欢迎以及NAS NAS上的优势广告，这些小型服务器也获得了市场需求。

尽管结构紧凑（体积小于0.013立方米）和重量轻（小于7千克），但该机器仍具有令人印象深刻的功能。我们正在谈论对Intel Xeon E3处理器家族的支持，最多可安装16 GB RAM的能力，系统管理模式，两个千兆以太网端口，一个PCIe插槽，对RAID 0/1/10的支持，一个DVD刻录机，最多四个硬盘驱动器，一个内部microSD卡插槽，一个集成的Matrox G200图形芯片和七个USB端口。另一方面，该服务器只有一个VGA连接器和两个RAM连接器。

功能强大的服务器，最多可容纳8个磁盘

处理器：至强E3-1200V6 |显卡：Intel HD Graphics P630 |内存：高达64GB |存储：2个内部固定的5.25英寸托架，最大80TB |连接接口：6 x USB 3.0，串行端口，视频连接器，2 x千兆以太网|尺寸（宽x长x高）：44.4厘米x 58.3厘米x 17.3厘米

+令人印象深刻的性能
+三年保修，带现场服务 -

高成本
-大尺寸

如果您需要比上述服务器更强大的功能，请查看TS460。这是一个昂贵得多的解决方案，但是您将获得不同级别的服务器。首先，它比以前的型号要大得多。它的体积为0.05立方米，重25公斤。该5U外形服务器在采用Turbo Boost技术的Intel Xeon E3处理器家族上运行，并保证提供现场服务。

它最多支持64 GB的RAM，并且内置控制器支持四种主要类型的RAID。您还将获得DVD刻录机，300瓦电源和两个千兆以太网端口。最多可以安装八个硬盘驱动器和多达八个USB连接器。该机箱有一个可上锁的门，支持ECC服务器内存，以及串行和VGA连接器。

配备精良的大功率服务器处理器：英特尔至强E5-2603 v3 |图形：Matrox G200 |内存：8GB |存储：不包括驱动器|连接接口：4个千兆以太网|尺寸（宽x长x高）：未知

+六核Xeon处理器
+三年的现场服务保证在治疗后的第二天

-您的钱包肯定会破产
-没有什么可抱怨的了

该型号与TS440和ML350属于同一类别。除了具有集成的视频卡（在Matrox G200上）之外，还有为期三年的第二天工作场所服务保证，四个千兆以太网端口和12 Gb / s SAS支持（请注意，这仅需要2.5英寸驱动器）。
此外，该服务器不能与第三版本的Intel Xeon E5-2603处理器一起使用（并且不能在常规E3处理器上使用），并且支持两个处理器。E5是六核处理器，因此特别适合解决更复杂的任务。RAM可用的插槽数量（24个插槽）也给我们留下了深刻的印象，这些插槽总共可以收集3 TB（在安装128 GB的LRDIMM带时）。此外，除了可上锁的前门和存储控制器外，该服务器还具有两个带有热插拔功能的500瓦冗余电源，从而获得了额外的积分。

10.华硕TS500



具有灵活性的主流塔式服务器

处理器：Intel Xeon E5-2600 v3 |图形：未知| RAM：取决于顺序|存储：不包括驱动器|连接接口：8个USB端口，VGA，PS2、3个千兆以太网|尺寸（宽x长x高）：22厘米x 54.5厘米x 45.5厘米

+服务器和工作站的理想解决方案
+不乏功能和灵活性

-并非每个人都需要这些功能
-华硕徽标可能会让某些人感到惊讶

像Supermicro一样，华硕并不是以其服务器而闻名。这家台湾公司是最大的计算机组件供应商之一，以其广泛的消费产品（包括主板）而闻名。 TS500-E8-PS4是5U主流塔式服务器，非常适合用作服务器或工作站。

它配备了第三版的最新Intel Xeon E5-2600处理器，八个DDR4 DIMM RAM插槽（最多支持一半TB的RAM），六个扩展插槽，三个5.25英寸存储托架和一个带有80 Plus铜牌证书的500瓦电源。还有四个3.5英寸热插拔SATA / SAS硬盘驱动器托架，如果需要灵活的存储，可以将其替换为8个HDD托架。

此外，借助与英特尔智能平台管理接口（IPMI）兼容的ASMB8-iKVM模块，您可以远程监视，控制和管理服务器。其他主要功能包括10个SATA端口，DVD刻录机，8个USB端口，一个PS2端口，一个VGA端口和三个千兆以太网端口。请注意，这是一个组装解决方案，但是很显然，它为您提供了该机器库存中的极大灵活性。
