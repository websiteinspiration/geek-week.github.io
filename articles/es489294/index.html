Aprendizaje de variables CSS con un ejemplo práctico



¡Buen dia amigos!

Una vez que la navegación web me llevó a este maravilloso código .

La "grandeza" del mismo (es decir, el código) es el uso práctico de variables en CSS, lo que facilita enormemente la escritura de estilos (el uso de variables también causa una estructura CSS inusual). Y, por supuesto, el tema es muy impresionante: los planetas del sistema solar.

El proyecto original está escrito en Pug y Sass, pero lo analizaremos en HTML / CSS.

Entonces vamos.

El diseño de un planeta (en este caso, Mercurio) es el siguiente (traigo mi propia versión ligera):

<div class="card mercury">
    <div class="planet">
        <div class="atmosphere">
            <div class="surface"></div>
        </div>
    </div>
    <div class="info">
        <h2 class="title">mercury</h2>
        <div class="form">
            <div class="item">
                <label class="label">Tilt</label><span class="line"></span><i class=" icon fas fa-long-arrow-alt-right"></i><span class="detail">0.034° </span>
            </div>
            <div class="item">
                <label class="label">Day</label><span class="line"></span><span class="detail">1,407 hours</span>
            </div>
            <div class="item">
                <label class="label">Year</label><span class="line"></span><span class="detail">88 days</span>
            </div>
        </div>
    </div>
</div>

Vemos lo siguiente:

  • Un contenedor para el planeta e información al respecto - .card
  • Encuadernar una envoltura a un planeta en particular: mercurio.
  • Contenedor para el planeta - .planet
  • En el interior, que es el contenedor de la atmósfera: atmósfera.
  • Dentro de la cual está la superficie del planeta - .superficie
  • Contenedor de información del planeta - .info
  • El nombre del planeta correspondiente es .title
  • Contenedor de información adicional - .form
  • Contenedor de la Unidad de Información - .item
  • En total, tenemos tres unidades de información:
    1. El primero contiene información sobre la inclinación del eje del planeta y la imagen de la flecha.
    2. En el segundo: información sobre la duración del día en horas
    3. En el tercero, aproximadamente la duración del año en días

Sí, casi lo olvido: la fuente está conectada en el elemento principal:

<link href="https://fonts.googleapis.com/css?family=Major+Mono+Display" rel="stylesheet">

... y Font Awesome (para la flecha):

<script src="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/5.12.1/js/all.min.js"></script>

Pasemos a CSS.

La siguiente sintaxis se utiliza para declarar variables en CSS:

:root {
  ---: ;
}

Lea más sobre la: pseudo-clase raíz aquí .

Las variables para el planeta se declaran de la siguiente manera:

:root {
    --mercury-image: url(https://thebestcode.ru/media/solarSystem/mercury.jpg);
    --mercury-tilt: rotate(0.034deg);
    --mercury-day: 1407.6;
    --mercury-color: #999;
}

Como puede ver, las variables se declaran para la imagen, el ángulo de inclinación, la duración del día y el color del planeta, respectivamente.

Para rotar los planetas se usa la siguiente animación:

@keyframes planetRotate {
    0% {
        background-position: 0% center;
    }

    100% {
        background-position: -200% center;
    }
}

Las variables utilizadas en CSS son:

var(---);

Así es como se usan las variables para crear estilos de planeta:

.mercury .atmosphere {
    box-shadow: inset 10px 0px 12px -2px rgba(255, 255, 255, 0.2), inset -70px 0px 50px 0px black, -5px 0px 10px -4px var(--mercury-color);
}

.mercury .surface {
    background-image: var(--mercury-image);
    transform: var(--mercury-tilt) scale(1.2);
    animation: planetRotate calc(var(--mercury-day)*.1s) linear infinite;
}

.mercury .planet::before {
    transform: var(--mercury-tilt) scale(1.2);
    border-color: var(--mercury-color);
    color: var(--mercury-color);
}

.mercury .icon {
    transform: var(--mercury-tilt);
    color: var(--mercury-color);
}

Observe cómo se usa la variable para la duración del día (--mercure-day) para determinar la duración de la animación (velocidad de rotación del planeta).
