10 ekstensi untuk VS Code, tanpanya saya tidak bisa memprogram

VS Code adalah editor kode favorit saya. Ini adalah yang paling populer dari editor yang ada, kemungkinan yang dapat diperluas hampir tak terbatas. Dan, yang mengejutkan, ini dikembangkan oleh Microsoft. Saya percaya bahwa tidak ada editor lain dan tidak ada IDE lain yang dapat memberikan pengembang setidaknya setengah dari apa yang VS Code dapat berikan kepadanya. Salah satu kekuatan VS Code adalah sistem ekstensi. Ini memungkinkan Anda membuat ekstensi secara harfiah untuk semua kesempatan. Saya ingin memberi tahu Anda tentang 10 ekstensi teratas saya untuk Kode VS.



1. Mempercantik



Untuk menginstal ekstensi, gunakan pintasan keyboard di VS Code CTRL+P, masukkan perintah di bawah ini di bidang yang terbuka dan klik Enter.

ext install HookyQR.beautify

The ekstensi Mempercantik memungkinkan Anda ke format html, js, css, JSON dan kode sass. Opsi pemformatan dapat dikontrol. Ekstensi ini merupakan tambahan atas sistem js-beautify standar dan memungkinkan pengembang untuk memformat kode persis seperti yang dia butuhkan.

2. Komentar yang Lebih Baik



ext install aaron-bond.better-comments

The Lebih baik Komentar ekstensi memungkinkan Anda untuk memperluas kemungkinan untuk bekerja dengan komentar. Secara khusus, ini memungkinkan Anda untuk membagi komentar menjadi beberapa kategori. Ini bisa berupa notifikasi, permintaan, to-do list, catatan.


Menggunakan Komentar yang Lebih Baik

3. Penanda



ext install alefragnani.Bookmarks

Ekstensi Bookmarks adalah garis hidup sejati bagi para programmer. Terutama bagi mereka yang hilang di antara ribuan baris kode. Ekstensi ini memungkinkan Anda untuk melampirkan penanda ke baris kode. Jika baris kode memiliki bookmark, Anda dapat dengan mudah pergi ke baris ini menggunakan daftar bookmark.


Bookmark dalam kode

4. Bracket Pair Colorizer 2



ext install CoenraadS.bracket-pair-colorizer-2

Ekstensi Bracket Pair Colorizer 2 memungkinkan Anda untuk mewarnai braket, sehingga memudahkan menemukan pasangan tanda kurung buka dan tutup. Ini sangat berguna dalam kasus di mana Anda harus bekerja dengan konstruksi perangkat lunak bersarang.


Pasang kurung disorot.

5. Format dalam Menu Konteks



ext install lacroixdavid1.vscode-format-context-menu

Ekstensi Format dalam Konteks Menu memungkinkan Anda untuk memformat file dengan menyorotnya di sidebar dan memanggil perintah menu konteks. Ini sangat nyaman dalam kasus di mana ada banyak file yang perlu diformat, tetapi pada saat yang sama di lingkungan yang ada tidak ada dukungan untuk sarana untuk memformat dan menyisir kode.


Bekerja dengan Format dalam Menu Konteks

6. Git Graph



ext install mhutchie.git-graph

Ekstensi Git Graph memungkinkan Anda untuk melihat struktur repositori dan, menggunakan struktur ini, melakukan berbagai operasi. Ekstensi ini sangat dapat disesuaikan dan memiliki banyak fitur. Mungkin, untuk mendeskripsikannya dengan benar, Anda akan membutuhkan seluruh artikel terpisah.


Bekerja dengan Git Graph

7. GitLens



ext install eamodio.gitlens

The ekstensi GitLens membantu mengidentifikasi kode kepenulisan menggunakan penjelasan git menyalahkan dan lensa. Ekstensi ini, antara lain, memungkinkan Anda untuk dengan mudah melihat konten repositori dan mempelajari informasi yang berguna tentang kode.


Bekerja dengan GitLens

8. Indent Rainbow



ext install oderwat.indent-rainbow

Ekstensi Indent Rainbow memungkinkan Anda untuk mewarnai lekukan, menyoroti lekukan dari berbagai tingkat dengan warna yang berbeda. Ini sangat berguna ketika bekerja dengan konstruksi bersarang dalam bahasa seperti Python.

9. Jalan Intellisense



ext install christian-kohler.path-intellisense

Ekstensi Path Intellisense menyediakan editor dengan kemampuan untuk melengkapi jalur file secara otomatis. Meskipun VS Code mendukung penyelesaian jalur tanpa ekstensi, fitur ini hanya terbatas pada file HTML, CSS, dan JavaScript. Berkat ekstensi ini, penyelesaian jalur dapat digunakan saat bekerja dengan file dan bahasa apa pun.


Bekerja dengan Path Intellisense

10. Total Garis



ext install praveencrony.total-lines

Ekstensi Total Lines adalah alat kecil yang nyaman yang menampilkan informasi tentang jumlah baris dalam file terbuka di bilah status. Ini adalah tambahan yang bagus untuk koleksi ekstensi saya.
