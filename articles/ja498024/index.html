HoudiniとPythonでマウスをクリックするだけで都市を構築



ゲームの開発では、パイプライン、アルゴリズム、小さなプロセスが常に作成および適用されるため、時間と労力を大幅に節約できます。多くの場合、これらのソリューションはシンプルですがエレガントです。

この記事はこれについてです。著者は仮想都市を構築する方法を思いつきました、それは将来的に彼がレベルデザイナーの本格的なツールになりたいと思っています。

この記事では、マウスをクリックするだけで都市を構築できるHoudiniのCity Builderツールの基本設定について説明します。これは、Python Viewerステートを使用してHoudini内でのみ機能することに注意してください。

初期設定


最初にすることは、家が建てられるグリッドまたは平面です。理論的には、三角形、六角形など、何でもかまいません。私が見つけた最も簡単な方法は、クワッドメッシャーを使用することです。

サークルノードから始めて、インスタントメッシュノードをその下に配置します（Labsツールからのものであるため、インストールされていることを確認してください）。



その後、押し出しノードを配置します。後でこれを使用して、マウスで選択したプリミティブを押し出します。



セットアップはHDAで最小化され、Pythonコードを操作するためのパラメーターを作成します。この場合、グループパラメータを制御して、マウスをクリックしたプリミティブを拡張して埋める必要があります。



Python Viewerの状態


これで、Pythonコードに移動できます。あなたが視聴者の状態に不慣れな場合、私は以前に基本的な紹介を書きました。

新しいビューア状態を作成し、空のテンプレートから始めます。これにより、本当に必要なコードのみが生成されます。



追加することがいくつかあります。それらを2つの部分に分割できます。

  • マウスクリックで表示するジオメトリを取得します。
  • マウスクリックでプリミティブを取得します。

開始するには、initパーツで、self.geometry = Noneという名前のジオメトリの新しい変数を追加します。次に、OnEnter関数を追加します。このメソッドは、ユーザーがノードを作成/選択し、ビューポートでEnterキーを押すことによって状態がアクティブになったときに呼び出されます。

これは、現在のジオメトリがself.geometry自体に格納される場所です。 node.geometry（）を使用して行われます。



これで、ジオメトリとの接続が確立されました。

次の部分は、クリック時のマウスの位置に基づいてプリミティブ番号を取得することです。これを行うには、OnMouseEvent関数を作成します。これは、マウスイベント（クリックやマウスの位置など）の処理です。

マウスイベントの開始時に、機能する変数を作成します。次に、GeometryIntersector関数を使用してジオメトリとの交点を取得し、self.geometryを使用します。

この交差点から、必要な情報を取得し、プリミティブ番号を見つけて印刷します。
