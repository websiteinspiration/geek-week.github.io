こんにちは、親愛なる読者。
ここにいくつかの情報を追加しました。発行から時間が経過し、開発中に間違いに気付いたため、これは指示ではないことを付け加えておきます。これは私が発明した自転車で、ペダルは反対方向に回転しています。


私は非常に簡潔で、記事を段落に分けます。

組織の問題


AutoCADソフトウェア製品のユーザー数がローカルネットワークライセンスの数を超えています。

  1. AutoCADソフトウェアで働く専門家の数は、内部文書によって標準化されていません。
  2. 段落1に基づいて、プログラムのインストールを拒否することはほとんど不可能です。
  3. 仕事の不適切な構成はライセンスの欠如につながり、アプリケーションとこの問題のある情報技術サービスの加入者への電話につながります。

技術的な問題


  1. 使用中のライセンスのリストを表示するための資金不足。

ソリューションオプション



  1. ソフトウェアメーカーがサポートするターンキーソリューションで、ユーザーは占有されているライセンスのリストを個別に表示できます。
  2. ライセンスマネージャーの作業のレポートをWebページの形式で表示するための適切なソリューションの開発。

決定と実施


技術的なタスク


  1. OSライセンスを節約する機会
  2. ライセンスを保持しているユーザーのリスト

ライセンスマネージャーの作業の実装


必要な機能を独自に実装することにしました。実行の順序：

  1. CentOS 7を仮想化サーバーにインストールして構成する
  2. Linux用Autodesk Network License Managerをインストールして実行する
  3. OS再起動時の自動起動ユーティリティの設定
  4. パラメータファイルの設定（以下で説明します）
  5. ローカルWebサーバーとPHPのインストール

使用中のライセンスのリストを実装する


  1. 以下の内容の.shファイルを作成します。

    	#! /bin/bash
	/opt/flexnetserver/lmutil lmstat -a -c [   .lic]> "/var/www/html/log.txt"

    これは便利なディレクトリにあり、実行可能ファイルとして構成されています。

    このコマンドを使用して、ライセンスマネージャーのステータスがlog.txtファイルにアップロードされます
  2. 使用したコマンド 

    watch -n 5 [    №1  .sh]

    これにより、以前に作成したbashスクリプトを5秒ごとに呼び出すことができます。
  3. 項目1のディレクトリlog.txtに、index.phpファイルは次の内容で配置されます

    <html>
<head>
<meta http-equiv="Content-Type" content="text/html; charset=utf-8">
<script src="/jq.js"></script>
<title>License server AutoCAD</title>
<style>
</style>
</head>
<body>
<h1>    autoCAD</h1>

<div style="margin: 10px;">
<?php
$log = file_get_contents('./log.txt');
$logrp = nl2br($log);
$arraystr = explode(PHP_EOL,$logrp);
$busy = explode(" ",$arraystr[13]);
echo "   : ".$busy[12]." <br/><br/>";
$i = 18;
while($i<=37){
//var
$a = $i-17;
$data = explode(" ", $arraystr[$i]);
$time = str_replace('<br', '', $data[13]);
//varEND

echo "<span>".$data[4]."</span> ";
echo "<span>".$data[12]."</span> ";
echo "<span>".$data[11]."</span> ";
echo "<span>".$time."</span>";
echo "<br>";
$i++;
}
?>
</div>
</body>
</html>

    PHPをコードのせいにしないようにお願いします。より専門的なスペシャリストがより良い結果を出すでしょう。私はできる限りのことをしました。

    index.phpの本質：

    1. スクリプトによって以前に作成されたlog.txtファイルのテキストを取得し、5秒ごとに更新します。
    2. 転送タグをhtmlタグに置き換えます。
    3. テキストを行ごとに配列に分割します。
    4. 行の順序と内容をフォーマットします。
