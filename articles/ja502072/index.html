JavaScriptとコンパスを使用したISSのドッキング

悪名高いアイロンマスクによって設立されたSpaceX社は、クルードラゴン宇宙船をISSに手動でドッキングするためのシミュレーターをリリースしました。すべてが計画どおりに進んだ場合、ドッキングは2020年5月27日に行われます。全自動モードで行われますが、搭乗員は手動制御に切り替えることができます。実際、シミュレータで再現されるのは手動モードです。

シミュレータ自体はサイトにあり、最初の尾根にはかなり問題のあるおもちゃです...

ドキュメントでのキーボードクリックをエミュレートできるコードを作成するか、ネットワーク上で見つけます。私の場合、コードは次のようになりました。

function simulateKey(keyCode, type, modifiers) {
    var evtName = (typeof (type) === "string") ? "key" + type : "keydown";
    var modifier = (typeof (modifiers) === "object") ? modifier : {};

    var event = document.createEvent("HTMLEvents");
    event.initEvent(evtName, true, false);
    event.keyCode = keyCode;

    for (var i in modifiers) {
        event[i] = modifiers[i];
    }
    document.dispatchEvent(event);
}

function keyPress(keyCode) {
    simulateKey(keyCode)
    setTimeout(() => simulateKey(keyCode, "up"), 15);
}

そして、それはかなり許容できることがわかりました。古典的なフィードバック制御回路。そして、その船がISSから離れていたとき、私たちは（当時はそう思われていたように）非常にスムーズに私たち自身の方へ飛んでいきました。しかし、問題は船自体の近くで始まりました。実際、船は非常にソーセージであり、0.2メートルの精度を得るのは物理的に不可能でした。実際には、私たちの船の混合が行われたということです...（非常に正確に）連続的な空間で言ってみましょう。そしてもちろん、200ミリ秒ごとにそれらに応答しようとすることで、非常に強力な規制措置が取られました。ボタンを少しだけずらしすぎました。そして船に近づくほど、オフセット値が大きくなり始め、実際に船をさらに揺らしました...その動きの振幅を増やしています...

欠けている精度をどこかで取ることが必要でした。この問題を解決するための2番目の反復では、私自身は速度変位のみに基づいてすでに計算を試みました。そして、はい、それもかなりうまくいったように見えましたが、これは運動の問題を解決しませんでした...

運動の問題の本質は何ですか？まあ、見て、私たちは宇宙空間で見つけて、コントロールボタンをクリックすることで、私たちは船を何らかの飛行機で加速させます。しかし、ボタンを離しても動きは止まりません。宇宙空間では、真空のため、抵抗はありません。そして、（ボタンを押すことによって）衝撃を与えるとすぐに、船はこの速度で動き始めました...そして何とかして停止する必要があります。シミュレーターでの停止は非常に簡単です。リバースインパルスを与える必要があります。

しかし、ソリューションの2回目の反復では、エラーの精度が向上したため、速度を調整する方法についての答えは得られません

でした... そして、ここで「コンパス」が必要でした。船/船の船長は、ルートを事前に計算する必要があります。入港後に速度を落とすように指示した場合、宝石を係留することはほとんどありません。そして、私たちはこれが必要です。

現在の偏差値に基づいて、carculatePath 関数は5つのステップを計算します。これは理論的にはこの偏差を0に減らすはずです。必ずしもこの反復ではないが、より詳細なグリッドで大切なゼロに近づくたびに。

function calculatePath(data, value) {
    data.path = [];
    if (data.prev === value) {
        data.speed = 0;
    }
    for (let i = 0; i < 5; i++) {
        if (0 < value + data.speed * (i + 1)) {
            data.speed -= 0.1;
            data.path.push(-1);
        } else if (value + data.speed * (i + 1) < -0) {
            data.speed += 0.1;
            data.path.push(1);
        } else if (i > 0) {
            if (0 < data.speed) {
                data.speed -= 0.1;
                data.path.push(-1);
            } else if (data.speed < 0) {
                data.speed += 0.1;
                data.path.push(1);
            } else {
                data.path.push(0);
            }
        } else {
            data.path.push(0);
        }
    }
    data.prev = value;
}
