前書き


Nuxt.jsは、Vue.jsを使用してアプリケーションを構築するためのフレームワークです。起動可能なアプリケーションを開発するように設計されており、webpackとbabelを使用して優れたプロジェクト構造を提供します。

Vue.jsは最も人気のあるフロントエンドフレームワークの1つであり、GitHubには10万を超えるスターがあります。2014年にEvan Yuによって作成されたVue.jsは、シンプルなAPI、小さなライブラリサイズ、優れたパフォーマンスという3つの主な利点のおかげで、すぐにJavaScriptフレームワークのリーダーの1つになりました。

RESTは既存のすべてのAPIの99％を使用します。SOAPのおかげで、RESTはそのシンプルさからすぐに事実上の標準になりました。

2015年、FacebookはGraphQLを公開しました-APIからデータを受け取るために使用されるクエリ言語は、成長を続け、GitHub、Twitter、Yelpなどの企業に採用されています。

Strapiは、APIの開発にかかる時間を節約できるオープンソースのヘッドレスCMSです。

Strapiは拡張可能なプラグインシステムを使用して、管理パネル、認証とアクセス制御、コンテンツ管理、API生成など、多数の組み込み機能を提供します。

Strapiは100％オープンソースプロジェクトです（GitHubをご覧ください）。

  • Strapiは完全に無料のプロジェクトです。
  • Strapiをサーバーにインストールできるため、データの制御を失うことはありません。
  • プラグインシステムのおかげで完全にカスタマイズ可能で拡張可能です。

Stripeは、支払いを処理するときに開発者の生活を楽にするオンライン支払いシステムです。このチュートリアルでは、注文に使用します。

すでに開発を開始したいですか？はじめましょう！

注：一連の記事全体のソースコードはこちらから入手できます。

Nuxtをインストールする


まず、vue cliを使用してNuxt.jsプロジェクトを作成する必要があります。以前にインストールされていない場合は、@ vue / cliを配置しましょう。

yarn global add @vue/cli  
# 
npm install -g @vue/cli

ディレクトリを作成しますdeliveroo-clone-tutorial。

mkdir deliveroo-clone-tutorial

次にそれに行きます：

cd deliveroo-clone-tutorial

次に、Nuxt.jsプロジェクトを作成します。これを呼び出しますfrontend。

yarn create nuxt-app frontend  
# 
npx create-nuxt-app frontend  
# 
npm init nuxt-app frontend

上記のコマンドを実行した後、質問に答えることができますが、この場合は、パッケージマネージャーとしてyarnを選択することで1つだけ回答できます。

? Choose the package manager: Yarn

残りについては、単にEnterキーを押してください。

また、次のことも実行します。

cd frontend && yarn dev

いいね！localhost：3000を開いて、アプリケーションが正しく起動したことを確認します。

UIkitをインストールする


UIkitは、高速で機能的なWebインターフェースを開発するためのモジュール式の軽量CSSフレームワークです。

このチュートリアルでは、このフレームワークを使用します。

ディレクトリにuikitをインストールしますfrontend。

yarn add uikit

Nuxtアプリケーションにuikitを追加する必要があります。プラグインを使用してこれを行います。

ファイル/frontend/plugins/uikit.jsを作成し、次のコードをその中にコピーします。

import Vue from 'vue'

import UIkit from 'uikit/dist/js/uikit-core'  
import Icons from 'uikit/dist/js/uikit-icons'

UIkit.use(Icons)  
UIkit.container = '#__nuxt'

Vue.prototype.$uikit = UIkit

プラグインとcssファイルを接続する必要があるnuxt.config.jsため、プラグインを開き、以下を追加します。

...
css: [  
  "uikit/dist/css/uikit.min.css",
  "uikit/dist/css/uikit.css",
],

/*
**      
*/
plugins: [  
  { src: '~/plugins/uikit.js', ssr: false }
],
...


いいね！UIkitの準備ができました！

では、最初のコンポーネントを作成しましょう。

ヘッダーコンポーネントを作成する


コンポーネントHeader.vueは、アプリケーションのすべてのページで使用されます。コンポーネントディレクトリにファイルを

作成するHeader.vue/frontend/components/

<template>  
  <client-only>
    <nav class="uk-navbar-container" uk-navbar>
        <div class="uk-navbar-left">
            <ul class="uk-navbar-nav">
              <li class="uk-active">
                <router-link tag="a" class="navbar-brand" to="/" exact>Deliveroo clone</router-link>
              </li>
              <li>
                <router-link tag="a" class="navbar-brand" to="/restaurants" exact>Restaurants</router-link>
              </li>
            </ul>
        </div>
    </nav>
  </client-only>
</template>

コンポーネントlayouts/default.vueは、すべてのページのルートです。その中に<nuxt />アプリケーションがあります。（このチュートリアルで作成するページ）

次に、コンポーネントを使用するには、コンポーネントHeader.vueをインポートして不要なcssスタイルを削除し、アプリケーションを配置するコンテナーを決定する必要があります。

にあるコードをコピーして次のコードに置き換えます/frontend/layouts/default.vue。

<template>  
  <div>
    <Header />
    <div class="uk-section uk-section-default">
      <div class="uk-container uk-container-large">
        <nuxt />
      </div>
    </div>
  </div>
</template>

<script>  
import Header from '~/components/Header.vue'  
export default {  
  components: {
    Header
  }
}
</script>

次に、ファイルのメインページを変更し/frontend/pages/index.vue、コンテンツとスタイルを削除して、次のコードに置き換えます。

<template>  
  <div>
    <img src="https://media.giphy.com/media/zBL9j9oiR3VM4/giphy.gif" class="uk-position-center" alt="">
  </div>
</template>

これで、メインページ（index.vue）にアクセスできます。このページlayouts/default.vueは、コンポーネントがインポートされ、接続されています。Header.vue

  • このページをリロードして、localhostの変更を確認してください：3000
  • 以前にコマンドを停止した場合は、yarn devまたはnpm run devを実行します。




ストラピ


フロントエンドがあればこれで十分ですが、アプリケーションにはユーザー、レストラン、料理、注文を管理するためのバックエンドが明らかに必要です。

この魔法を実現するために、コンテンツを管理するためにStrapiでプロジェクトを作成しましょう。

Strapiをインストールする


Node.js 10以降を使用していることと、MongoDB、Postgres、またはMySQLのいずれかのデータベースがマシンにインストールされて実行されていることを確認してください。

Strapiはさまざまなデータベースで使用できます。オプションを使用--quickstart して、SQLiteデータベースでStrapiを簡単にインストールできます。このオプションは、Strapiでのプロトタイピングと開発に推奨されます。（MongoDBを使用しない場合）

Strapiをインストールbackendし、ディレクトリに名前を付けるプロジェクトを生成しますdeliveroo-clone-tutorial

yarn create strapi-app backend --quickstart  
# 
npx create-strapi-app backend --quickstart

プロジェクトが始まるまで数秒待ちます。ブラウザーが自動的に開くはずです。そうでない場合は、次のステップのためにlocalhost：1337 / admin /ご自身を開いてください。

注：これは、Strapiサーバーを停止した場合に再度開始する方法です。

strapi develop

管理者を作成する


登録ページで 管理者を追加します。



すばらしい仕事です。Nuxt.jsとStrapiを使用してプロジェクトを正常にインストールしました！
