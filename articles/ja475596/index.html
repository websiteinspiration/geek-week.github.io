少数のシステムで証明された大きな素数の仮説

双子の素数仮説は、最も重要で複雑な数学の問題の1つです。2人の数学者が、小さな数値システムに対してこの問題の並列バージョンを解決しました。




9月7日、2人の数学者が、数学における最も有名な未解決問題の1つのバージョンの証明を公開しました。この結果は、100年以上にわたって数学者を悩ませてきた数学の最も深遠な特性のいくつかに関連付けられている双子素数の仮説の研究に新しい前線を開きます。オックスフォード出身の数学者であるジェームズメイナード氏は、

「私たちは長い間横滑りしていて、このタスクのアイデアが不足しているため、だれでも新しいアイデアを思いつくことは当然です」と述べています。

双子素数仮説は、2だけ異なる素数のペアに適用されます。双子は、5と7、17と19です。仮説では、自然数の間にそのようなペアの数が無限であるとしています。過去10年間で、数学者は大きな進歩を遂げましたが、問題の完全な解決にはまだほど遠いです。

新しい証拠の著者、コロンビア大学のウィル・サビンとマディソンにあるウィスコンシン大学のマーク・シュースターマンは、より小さいがまだ注目に値する数学の世界の仮説を証明しました。彼らは数が少ない有限数値システムの場合にその有効性を証明した。

このような数体系は「有限体」と呼ばれます。サイズは小さいものの、無数の整数に固有の数学的特性の多くを保持しています。数学者は、有限体での算術の質問に対する答えを見つけようとしており、これらの結果をすべての整数に変換したいと考えています。

「究極の夢は、少しナイーブではありますが、有限体の特性をよく理解することで、整数の世界に光を当てることができることです」とメイナード氏は語った。

双子素数の仮説を証明することに加えて、SavinとShustermanは、小さな数値システムにおける素数の動作に関して、さらに驚くべき結果を発見しました。彼らは、ツインプライムが小さな間隔で現れる頻度を証明しました-この結果により、ツインプライムの現象を非常に正確に制御することが可能になります。数学者は、通常の数値で同様の結果を達成することを夢見ています。彼らは数直線上の素数に適用できるアイデアを求めて新しい証拠を探ります。

新しい種類の素数


ツインプライム仮説の最も有名な予測は、2だけ異なる数のペアが無限に多数存在することです。ただし、このステートメントはより一般的です。これは、4（たとえば、3と7）、または14（293と307）、または指定した任意の数だけ異なる無限の素数があることを示しています。

アルフォンス・デ・ポリニャックは、1849年に近代的な形でこの仮定を行いました。次の160年の間、数学者は彼とあまり進歩しませんでした。しかし、2013年には、氷が割れたか、少なくともひび割れました。その年、チャンイーサンは、7千万人を超えない、非常に多くの素数のペアの存在を証明しました。来年、メイナードやTerry Taoは、このギャップを真剣に減らしました。現在の記録は、互いにわずか246だけ異なる、無限に多数の素数のペアが存在することの証明です。

しかし、この進展が収まった後、数学者は、この問題を完全に解決するためのまったく新しいアイデアの必要性を理解しています。そして、有限数システムは、このアイデアを探すのに適した場所です。

有限体を作成するには、最初に自然数から有限数のサブセットを抽出する必要があります。たとえば、最初の5つの数値（または任意の素数）を取ることができます。そして、いつものように、ナンバーラインで数字を表す代わりに、文字盤の形の新しいナンバーシステムを想像してみてください。

算数は、ご想像のとおり、限られたスペースで働き始めます。 5つの要素で構成される有限数値システムの4 + 3は何でしょうか？ 4から始めて、時計回りに3つの除算を数え、2に進みます。減算、乗算、除算は同様に機能します。



しかし、唯一の難点があります。有限体では、素数の通常の定義は意味がありません。最後のフィールドでは、任意の数値が他の数値で除算されます。たとえば、7は通常3で除算されませんが、5つの要素の最後のフィールドでは-が除算されます。これは、この最後のフィールドでは、数字の7が数字の12と同じであるためです。両方とも文字盤の2マークで終わります。したがって、7を3で除算すると、12を3で除算した場合と同じ結果が得られ、12を3で除算すると4が得られます。

したがって、有限体に対する双子双生児の仮説は、単純な多項式（たとえば、x 2 + 1 など）を指します。

たとえば、最終的な体に数値1、2、3が含まれているとします。これらの数値は多項式の係数になります。この有限体であり、「単純な」多項式は多項式因子に分解できないものになります。したがって、x 2 + x + 2は因数分解できないので単純ですが、x 2-1はそうではありません。これは（x + 1）と（x-1）の積です。

単純な多項式を定義したので、単純なツイン多項式（単純で固定値が異なる多項式のペア）について質問するのは自然です。たとえば、多項式x 2+ x + 2は、x 2 + 2x + 2のように単純です。これらはxが異なります。

有限体に対する単純双子多項式の仮説は、xだけでなく任意の値が異なる単純双対多項式のペアが無限に多数存在することを示します。

きちんとしたカット


有限体と単純な多項式の概念は、長すぎると思われ、数値の性質を研究するのには役に立たないようです。しかし、それらはハリケーンシミュレータのように見えます。大きな世界で起こっている出来事についてのアイデアを与える宇宙自体です。

「整数と多項式の間には古代の類似点があり、整数に関連する潜在的に非常に複雑な問題を多項式に関連するタスクに変換できます。多項式も複雑になる可能性がありますが、より簡単に解決できます」とShustermanは述べています。

有限体は、1940年にアンドレワイルが少数システムの演算を整数の演算に変換する正確な方法を開発したときに有名になりました。 Weilはこの接続を使用して驚くべき結果を得ました。彼は証明したおそらく数学で最も重要な問題-リーマン仮説-有限体上の曲線のセットの場合（この問題はリーマン幾何仮説の名前で知られています）。この証明と一連の追加のワイル仮説により、有限体は数学的な発見にとって豊かな景観になりました。

Weilの重要なアイデアは、有限体では、幾何学的手法を使用して数値に関する質問への回答を見つけることができるということでした。 「これは有限体の特徴です。解決したいタスクの多くは、幾何学的に言い換えることができます」とShustermanは言いました。

このような環境でジオメトリがどのように表示されるかを理解するには、各多項式を空間内の点として想像してください。多項式の係数は、その位置を決定する座標として機能します。 1、2、3の最後のフィールドに戻ると、多項式2x + 3は2次元空間の点（2、3）にあります。

しかし、最も単純な有限空間であっても、無限の数の多項式があります。最大の指数または表現の次数を増やすことにより、より複雑な多項式を作成できます。我々の場合には、多項式X 2 - 3X - 1は、三次元空間内の点によって表されます。多項式3× 7 + 2× 6 + 2× 5 - 2× 4 - 3× 3 + X 2-2x + 3は、8次元空間ではドットで表されます。

新しい作品では、この幾何学的空間は、特定の有限体に対する特定の次数のすべての多項式を表します。質問は次のようになります。単純な多項式を表すすべての点を分離する方法はありますか？

SavinとShustermanの戦略は、空間を2つの部分に分割することです。一部には、偶数の因子を持つ多項式に対応するすべての点があります。もう1つは、すべての点が奇数の因子を持つ多項式に対応しています。



これはすでにタスクを単純化しています。有限体に対する単純な双子多項式の仮説は、1つの因子を持つ多項式にのみ適用されます（素数が1つの因子を持っているように、それ自体がそれです）。また、数値1は奇数であるため、因数が偶数の多項式を含む空間の一部はすぐに破棄できます。

トリックは除算を使用することです。たとえば球の表面などの2次元オブジェクトの場合、赤道が地球の表面をカットするのと同じように、1次元曲線によって半分にカットされます。次元数が多いオブジェクトは、次元数が1つ少ないオブジェクトで常にカットできます。

しかし、多項式の空間を分割する、次元の少ないこれらの形式は、赤道ほどエレガントではありません。メビウス関数と呼ばれる数式を使用して描画されます。。入力では多項式を取り、出力では多項式が偶数の素因数を持つ場合は1、奇数の場合は-1、因数分解される場合は0を返します（16は2×に分解できるため） 2×2×2）。

メビウス関数によって描かれた曲線は、狂ったように曲がったりねじれたりし、多くの場所でそれ自体と交差しています。特異点と呼ばれるこれらの場所は、分析が特に困難です（これらは、いくつかの同一の素因数に分解できる多項式に対応しています）。

SavinとShustermanの根本的な革新は、より少ない測定でループをより短いセグメントにカットする正確な方法を見つけたことです。これらのセグメントは、ループ全体よりも学習が容易でした。

奇数の素因数をもつ多項式のカタログを作成したが、これは最も困難でした。SavinとShustermanは、どれが単純で、どちらが双子かを判別するという課題に直面しました。これを行うために、彼らは数学者が通常の素数の素数を研究するために使用するいくつかの公式を適用しました。

SavinとShustermanは、これらの手法を使用して、特定の有限体における単純な多項式に関する2つの重要な点を証明しました。

まず、有限体における双子素数の仮説は真です。任意の値が異なる双対多項式のペアが無限に存在します。

第2に、そしてより重要なことに、この作業は、特定の次数の多項式の中で見つけることができる単純な双子多項式の数の正確な計算を提供します。これは、数直線上の十分長い間隔内にツインプライムがいくつあるかを知ることに似ています。これは数学者の夢にすぎません。

「これは整数に当てはまることの量的な類推を与える最初の作品であり、これは本当に素晴らしい結果です」とテルアビブ大学のZiv Rudnikは言いました。 「これまでのところ、このようなことは何も起こっていません。」

SavinとSchustermanの証明は、Andre Weilが有限体上の曲線のリーマン仮説を証明してからほぼ80年後、数学者がいかに頑固にこの方向に進んでいるかを示しています。現在、双子の素数仮説を扱う数学者は、SavinとShustermanの研究に目を向け、おそらくそれが彼らに深いインスピレーションの源を与えるでしょう。
