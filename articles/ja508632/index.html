CPU ARM Red Hat Enterprise Linux







OpenNETは次のように書いています。「世界の500台の最高性能コンピューターのランキングの第55版が公開されました。 6月のランキングは、ARMプロセッサーの使用で有名な日本クラスターの富岳という新しいリーダーによって導かれました。富岳クラスターは理化学研究所の理化学研究所でホストされており、415.5ペタフロップスのパフォーマンスを発揮します。これは、過去のランキングのリーダーが2位に追い抜かれた2.8を超えています。クラスターには、富士通A64FX SoCに基づく158976ノードが含まれ、クロック周波数2.2 GHzの48コアArmv8.2-A SVE CPU（512ビットSIMD）が搭載されています。合計すると、クラスターには700万を超えるプロセッサコア（以前の評価のリーダーの3倍）、FS Lusterに基づく5 PBのRAM、および150 PBの共有ストレージがあります。オペレーティングシステムはRed Hat Enterprise Linuxです。」

ヘビーデューティLinux Ubuntuラップトップ販売開始







CNewsは次のように書いています。購入すると、そのコンポーネントのほとんどすべてを構成でき、Linux Ubuntuとその修正バージョンのPop！_OSから選択することもできます。...基本構成では、Oryx Proの価格は1,623ドル（2020年6月26日の時点で中央銀行のレートで112.5千ルーブル）です。一方、最も高価なバージョンは4959ドル（34万ルーブル）です。



Oryx Proの場合、出版物によれば、15.6インチと17.3インチの対角線オプションがあります。Intel Core i7-10875Hプロセッサが使用されます。8つのコアがあり、16の同時データストリームを処理でき、2.3〜5.1 GHzの周波数で動作します。8 GBから64 GBまでの利用可能なメモリ構成オプション。デフォルトでは、ラップトップにはNvidia GeForce RTX 2060グラフィックチップと6 GBの独自のGDDR6メモリが付属しています。8 GB GDDR6を搭載したRTX 2070またはRTX 2080 Superと交換できます。

Ubuntu 20.04がプリロードされたDell XPS 13 Developer Editionを導入
