Model description


A hand-held mobile scanner with a weight of 1.13 kg is ready to work anywhere. In order to scan an object, it is enough to simply establish a connection between the device and a laptop using a connecting cable and get to work.

The data processing speed of the 3D scanner EinScan Pro 2X Plus is 1.5 million points at 30 frames / sec, which is 1.5 times higher than the speed of previous devices in this line. Information is transmitted to the laptop using a USB cable.

To solve various problems, Shining 3D engineers created four operating modes: automatic with and without a turntable; Manual with high precision, fast manual. The user has several alignment options to choose from: by markers using an automatic turntable, automatic, manual, by stickers on the object.
