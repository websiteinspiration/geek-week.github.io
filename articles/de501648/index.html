10 Erweiterungen für VS Code, ohne die ich nicht programmieren kann

VS Code ist mein Lieblingscode-Editor. Dies ist der beliebteste der vorhandenen Editoren, dessen Möglichkeiten nahezu unbegrenzt erweitert werden können. Und überraschenderweise wurde es von Microsoft entwickelt. Ich glaube, dass keiner der anderen Editoren und keine der anderen IDEs dem Entwickler mindestens die Hälfte dessen geben kann, was VS Code ihm geben kann. Eine der Stärken von VS Code ist das Erweiterungssystem. Sie können damit buchstäblich Erweiterungen für alle Gelegenheiten erstellen. Ich möchte Sie über meine Top 10-Erweiterungen für VS Code informieren.



1. Verschönern



Um die Erweiterung zu installieren, verwenden Sie die Tastenkombination in VS Code CTRL+P, geben Sie den folgenden Befehl in das sich öffnende Feld ein und klicken Sie auf Enter.

ext install HookyQR.beautify

Mit der Beautify-Erweiterung können Sie HTML-, JS-, CSS-, JSON- und Sass-Code formatieren. Formatierungsoptionen können gesteuert werden. Diese Erweiterung ist ein Add-On zum Standard-js-Verschönerungssystem und ermöglicht es dem Entwickler, den Code genau nach Bedarf zu formatieren.

2. Bessere Kommentare



ext install aaron-bond.better-comments

Mit der Erweiterung " Bessere Kommentare" können Sie die Möglichkeiten zum Arbeiten mit Kommentaren erweitern. Insbesondere können Sie Kommentare in Kategorien unterteilen. Dies können Benachrichtigungen, Anfragen, Aufgabenlisten und Notizen sein.


Bessere Kommentare verwenden

3. Lesezeichen



ext install alefragnani.Bookmarks

Die Lesezeichen- Erweiterung ist eine echte Lebensader für Programmierer. Besonders für diejenigen, die unter Tausenden von Codezeilen verloren gehen. Mit dieser Erweiterung können Sie Lesezeichen an Codezeilen anhängen. Wenn die Codezeile ein Lesezeichen enthält, können Sie mithilfe der Lesezeichenliste problemlos zu dieser Zeile wechseln.


Lesezeichen im Code

4. Bracket Pair Colorizer 2



ext install CoenraadS.bracket-pair-colorizer-2

Mit der Erweiterung Bracket Pair Colorizer 2 können Sie die Brackets einfärben , sodass Sie leichter Paare von öffnenden und schließenden Brackets finden können. Dies ist sehr nützlich in Fällen, in denen Sie mit verschachtelten Softwarekonstrukten arbeiten müssen.


Klammerpaare sind hervorgehoben.

5. Formatieren in Kontextmenüs



ext install lacroixdavid1.vscode-format-context-menu

Mit der Erweiterung Format in Kontextmenüs können Sie Dateien formatieren, indem Sie sie in der Seitenleiste markieren und den Kontextmenübefehl aufrufen. Dies ist besonders praktisch, wenn viele Dateien formatiert werden müssen, gleichzeitig jedoch in der vorhandenen Umgebung keine Unterstützung für Mittel zum Formatieren und Flusen des Codes vorhanden ist.


Arbeiten mit Format in Kontextmenüs

6. Git Graph



ext install mhutchie.git-graph

Mit der Git Graph- Erweiterung können Sie die Struktur des Repositorys anzeigen und mithilfe dieser Struktur verschiedene Vorgänge ausführen. Diese Erweiterung ist hochgradig anpassbar und verfügt über viele Funktionen. Um es richtig zu beschreiben, benötigen Sie wahrscheinlich einen separaten Artikel.


Arbeite mit Git Graph

7. GitLens



ext install eamodio.gitlens

Die GitLens-Erweiterung hilft bei der Identifizierung der Code-Urheberschaft mithilfe von Anmerkungen und Objektiven für Git-Schuld. Mit dieser Erweiterung können Sie unter anderem den Inhalt von Repositorys sehr bequem anzeigen und nützliche Informationen zum Code erhalten.


Arbeite mit GitLens

8. Regenbogen einrücken



ext install oderwat.indent-rainbow

Mit der Erweiterung "Regenbogen einrücken" können Sie Einrückungen einfärben und Einrückungen verschiedener Ebenen mit unterschiedlichen Farben hervorheben. Dies ist besonders nützlich, wenn Sie mit tief verschachtelten Konstrukten in Sprachen wie Python arbeiten.

9. Pfad Intellisense



ext install christian-kohler.path-intellisense

Die Erweiterung Path Intellisense bietet dem Editor die Möglichkeit, Dateipfade automatisch zu vervollständigen. Obwohl VS Code die Pfadvervollständigung ohne Erweiterungen unterstützt, ist diese Funktion nur auf HTML-, CSS- und JavaScript-Dateien beschränkt. Dank dieser Erweiterung kann die Pfadvervollständigung beim Arbeiten mit beliebigen Dateien und Sprachen verwendet werden.


Arbeiten Sie mit Path Intellisense

10. Gesamtanzahl der Zeilen



ext install praveencrony.total-lines

Die Erweiterung Total Lines ist ein kleines praktisches Tool, das Informationen zur Anzahl der Zeilen in einer geöffneten Datei in der Statusleiste anzeigt. Dies ist eine schöne Ergänzung zu meiner Sammlung von Erweiterungen.
