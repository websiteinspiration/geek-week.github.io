レベル8


サービスのソースコードが提供されます。エンコードされた文字列とエンコード方法が格納されます。



逆の順序でデコードする必要があります：

  1. 16進表示からバイナリ表現に変換します。
  2. 行を反転します。
  3. base64をデコードします。

<?php
$secret = "3d3d516343746d4d6d6c315669563362";
echo base64_decode(strrev(hex2bin($secret)))."\n";
?>



シークレットを送信してパスワードを取得します。

データはフィルタリングされないため、パイプラインを収集して他のOSコマンドを実行できます。たとえば、リクエストに文字列を渡した場合： "|| ls＃ "、その場合、完全な要求は次のようになります" grep -i || ls＃dictionary.txt。 " ||後のすべて -エラーで実行され、＃の後にコメント化されます。つまり、lsコマンドの出力のみが取得されます。



したがって、次のファイルを読み取ります： "|| cat / etc / natas_webpass / natas10＃ "。


脆弱性はOSコマンドインジェクションとして分類されます。
