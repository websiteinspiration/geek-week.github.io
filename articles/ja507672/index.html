PowerShellを使用してAzure VMを変更および削除する

エンジニアとIT管理者はPowerShellを使用して、オンプレミスだけでなく、特にAzureのクラウドインフラストラクチャを操作するときに、さまざまなタスクを正常に自動化します。場合によっては、PowerShellを使用する方が、Azureポータルを使用するよりもはるかに便利で高速です。そのクロスプラットフォーム機能のおかげで、PowerShellはあらゆるOSで使用できます。



Ubuntu、Red Hat、Windowsのどれを使用しても問題はありません。PowerShellはクラウドリソースの制御に役立ちます。たとえば、Azure PowerShellモジュールを使用すると、仮想マシンの任意のプロパティを設定できます。



この記事では、PowerShellを使用してAzureクラウド内のVMのサイズを変更し、VMとその関連オブジェクトを削除する方法について説明します。

AzureでのVMの削除についてはどうですか？



削除すると、見た目ほど単純で簡単なものばかりではありません。結局のところ、以下を含む、このマシンに関連付けられている多くのリソースを削除する必要があります。



  • 診断用ストレージコンテナーの保管
  • ネットワークインターフェース
  • パブリックIPアドレス
  • ステータスが保存されるシステムディスクとblob
  • データディスク


したがって、関数を作成して名前を付けRemove-AzrVirtualMachine

ネットワークインターフェイスとパブリックIPアドレスを削除する



私たちのVMには1つ（または複数）のネットワークインターフェイス（NIC）があります。それらを不要に削除するにはNetworkInterfaces



、VMオブジェクトのプロパティを調べてNICコマンドを削除しRemove-AzNetworkInterface



ます。複数のネットワークインターフェイスがある場合は、ループを使用します。同時に、各NIC IpConfiguration



について、プロパティをチェックして、インターフェイスにパブリックIPアドレスがあるかどうかを確認します。見つかった場合は、コマンドで削除してくださいRemove-AzPublicIpAddress



。



これは、すべてのNICをループして削除し、パブリックIPがあるかどうかを確認する、このようなコードの例です。ある場合は、プロパティを解析しPublicIpAddress



、対応するリソースの名前をIDで取得して削除します。 




foreach($nicUri in $vm.NetworkProfile.NetworkInterfaces.Id) {
    $nic = Get-AzNetworkInterface -ResourceGroupName $vm.ResourceGroupName -Name $nicUri.Split('/')[-1]
    Remove-AzNetworkInterface -Name $nic.Name -ResourceGroupName $vm.ResourceGroupName -Force

    foreach($ipConfig in $nic.IpConfigurations) {
        if($ipConfig.PublicIpAddress -ne $null) {
            Remove-AzPublicIpAddress -ResourceGroupName $vm.ResourceGroupName -Name $ipConfig.PublicIpAddress.Id.Split('/')[-1] -Force
        }
    }
}


システムドライブを削除する



OSディスクはblobであり、削除するためにコマンドRemove-AzStorageBlob

$osDiskUri = $vm.StorageProfile.OSDisk.Vhd.Uri
$osDiskContainerName = $osDiskUri.Split('/')[-2]
$osDiskStorageAcct = Get-AzStorageAccount | where { $_.StorageAccountName -eq $osDiskUri.Split('/')[2].Split('.')[0] }
$osDiskStorageAcct | Remove-AzStorageBlob -Container $osDiskContainerName -Blob $osDiskUri.Split('/')[-1]
