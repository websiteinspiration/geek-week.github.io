1. Azureでのコンテナー管理


Azureコンテナーインスタンスは、Azureでコンテナーを実行する最も速くて簡単な方法です。このトレーニングスキームは、コンテナを作成および管理する方法と、ACIを使用してKubernetesの柔軟なスケーリングを実現する方法を学習するのに役立ちます。

コースモジュール：

  • Dockerを使用してコンテナWebアプリケーションを作成します。
  • Azure Container Registryサービスを使用してコンテナーイメージを作成および保存します。
  • Azure Container Instancesサービスを使用してDockerコンテナーを起動します。
  • Azure Application Serviceを使用してコンテナーWebアプリケーションをデプロイして起動します。
  • Azure Kubernetesサービスの概要。

詳細を確認し、リンクをクリックしてトレーニングを開始してください

2. Azure Databricksを使用したデータエンジニアリング


Azure Databricksを適切に使用してソリューションのセットアップを高速化する方法を学びます。組み込みのコネクタサービスを使用して、Azure SQL Data Warehouseでデータを操作します。Azureで利用可能なデータサービスの概要。合理化されたワークフローを作成し、Apache Sparkに基づくインタラクティブな分析ワークスペースを操作します。ちなみに、コースは約8時間かかります。

モジュール：

  • Azure Databricksの概要。
  • Azure Databricksを使用してSQL Data Warehouseインスタンスにアクセスします。
  • Azure Databricksでデータを取得する。
  • Azure Databricksを使用したデータの読み取りと書き込み。
  • Azure Databricksの基本的なデータ変換。
  • Azure Databricksで高度なデータ変換を実行します。
  • Databricks Deltaを使用してデータパイプラインを作成する。
  • Azure Databricksでストリーミングデータを操作します。
  • Azure DatabricksとPower BIを使用してデータの視覚化を作成します。
