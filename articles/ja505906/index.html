overthewire.orgのウォークスルー盗賊

こんにちは、盗賊。氏の最初のシーズンを見たロボット、それは主人公と同じぐらい悪くないと決めて、自分のためにKali Linuxをインストールし、脆弱性と兄の電話で裸の女の子の写真を探してインターネットをサーフィンする準備ができました。 nmapを3回起動し、驚くことではないが、その人生は映画ではないことを発見したので、より構造化された方法で検索を行い、情報セキュリティと脆弱性検索の基本から始めることにしました。

私の次のステップは、reddit.comの r / hackingにアクセスし、overthewire.orgに言及している固定された投稿を読むことでした。初心者に適した「ハッカー」の1つとして、Banditゲーム。読者は（私が公開しているハブを前提として）キャプチャーザフラッグゲームのアイデアを持っていると思いますが、念のため説明します。リモートマシンに接続し、隔離されたスペースで、サイトの推奨事項を使用して次のマシンに接続するためのキーを探しています。さらに、私とあなたのすべてにとって、私はあなたに最初の10レベルのパッセージを与えます。行こう！

PSすべてのパスワードはネタバレの背後に隠されており、読んで心配する必要はありません。

レベル0


bandit.labs.overthewire.org、ポート2220でホストに接続し、bandit0としてパスワードbandit0でログインする必要があります。否や言うほどない。標準のコマンド構文を使用しますssh。

:~$ ssh bandit0@bandit.labs.overthewire.org -p 2220
This is a OverTheWire game server. More information on http://www.overthewire.org/wargames

bandit0@bandit.labs.overthewire.org's password: <b>bandit0</b>
...
bandit0@bandit:~$
