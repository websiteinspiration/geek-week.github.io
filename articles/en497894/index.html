Microsoft Azure Virtual Training Day: Modern Data Warehouse

Webinar, April 29, 2020, 10.00-13.00

We invite you to get acquainted with the modern data warehouse. Over the course of several sessions, you will become familiar with the components of Microsoft Azure for building an end-to-end analytical solution. The sessions will focus on processes: from collecting and transforming data from a variety of sources using the Azure Data Factory, storing data based on Azure Synapse, presenting it to users based on Power BI.
