Ein allgemeiner Überblick über künstliche Intelligenz (KI) und maschinelles Lernen. Die Module richten sich an Anwender mit geringen Kenntnissen auf dem Gebiet der Informatik und Statistik. Sie lernen die grundlegenden Konzepte der KI kennen und lernen, wie Sie benutzerdefinierte Lösungen für maschinelles Lernen mit kostenlosen und einfachen Tools verwenden. Sie lernen einfache, aber effektive Methoden kennen, mit denen KI-Datenexperten Vorhersagen über Objekte, Personen und die Zukunft treffen. Anschließend werden interessante und komplexe Themen untersucht, von denen Sie vielleicht schon gehört haben, wie z. B. neuronale Netze, Computer Vision, Deep Learning und unkontrolliertes Lernen.

Insgesamt benötigt der Kurs ungefähr 4 Stunden Ihrer Zeit, um 5 Module zu studieren.

2. Erstellen von Smart Bots in Azure Bot


Die Benutzerinteraktion mit Anwendungen per Chat mithilfe von Text, Bildern oder Sprache kann mithilfe von Bots implementiert werden. Dies kann ein einfacher „Frage-Antwort“ -Dialog oder ein komplexer Bot sein, der es Menschen ermöglicht, auf intelligente Weise mit Diensten zu interagieren, wobei Mustervergleichs-, Statusverfolgungs- und künstliche Intelligenzmethoden verwendet werden, die gut in vorhandene Dienste integriert sind. Erfahren Sie, wie Sie einen intelligenten Chatbot erstellen, indem Sie QnA Maker und LUIS integrieren.

Der Kurs dauert ca. 3 Stunden.

3. Textanalyse mit Azure Cognitive Language Services


Erfahren Sie, wie Sie mit Cognitive Language Services Text analysieren, Absichten ermitteln, Themen für Erwachsene entdecken und Abfragen in natürlicher Sprache verarbeiten können.

Kursmodule:

  • Klassifizierung und Moderation von Text mithilfe von Azure Content Moderator;
  • Hinzufügen von Dialoglogik zu Anwendungen mithilfe des Intelligent Speech Recognition Service (LUIS);
  • Verfolgen der Tonalität des Dialogs im Text mithilfe der Textanalyse-API.

4. AI-basierter Kurs für Grenzgeräteingenieure


Mit KI-Technologien, Cloud-Technologien und Grenzgeräten, die im CICD-Modus als Container bereitgestellt werden, können Sie ganze Branchen transformieren, indem Sie Ökosysteme mit Selbstlernfunktionen für eine bestimmte Branche über die gesamte Wertschöpfungskette hinweg erstellen. Unsere Aufgabe ist es, einen solchen Satz von Vorlagen und Methoden für die Bereitstellung von KI auf Edge-Geräten im Kontext der Cloud zu erstellen. In diesem Schulungsprogramm erhalten Sie die folgenden Informationen:

  • Informationen zum Erstellen von Lösungen über das Internet der Dinge und die Cloud;
  • Übersicht über die Bereitstellung von IoT-Lösungen auf Edge-Geräten
  • Informationen zur Implementierung von Modellen für Grenzgeräte mit Containern;
  • Erfahren Sie, wie Sie DevOps für Edge-Geräte verwenden.

Der Kurs wurde in Zusammenarbeit mit der University of Oxford entwickelt.

Fange an zu trainieren

5. Bildverarbeitung und Klassifizierung mit Azure Cognitive Vision Services


Microsoft Cognitive Services bietet integrierte Funktionen für die Verwendung von Computer Vision in Anwendungen. Erfahren Sie, wie Sie mit Cognitive Vision Services Gesichter erkennen, Bilder markieren und klassifizieren sowie Objekte identifizieren.

Kursmodule:

  • Identifizieren von Gesichtern und Emotionen mithilfe der Computer Vision-API in Azure Cognitive Services.
  • Bildverarbeitung mit dem Computer Vision Service;
  • Klassifizierung von Bildern mithilfe des visuellen Erkennungsdienstes des Benutzers;
  • Bewertung der Anforderungen für die Implementierung der User Visual Recognition API.
