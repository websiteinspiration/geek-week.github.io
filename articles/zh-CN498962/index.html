关于多任务：受控制的窗口

俄罗斯的Microsoft专家Marat Khairulin继续探索处理多个任务的细微差别，并讨论了组合窗口和分割屏幕，有关用于站点和文档的个人时间机器以及虚拟表的真正好处。

在上一篇  文章中，  我们写到了现代科学如何与多任务处理相关联，并分享了一些小技巧，以帮助您专注于一项任务。现在，我们想提醒（可能会介绍一些人）有用的功能，这些功能将在解决您的问题需要同时使用多个应用程序时提供帮助。



以新旧方式切换


应用程序之间的切换可能是我们大多数人在“机器上”所做的事情，当然，没有人会对Alt + Tab键组合感到惊讶。但是，如果您同时按Ctrl键（即Ctrl + Alt + Tab），则此组合将修复菜单并在屏幕上打开所有窗口，并允许您通过单击鼠标或触摸手指来选择所需的应用程序（也可以使用键盘上的箭头选择窗口，并按Enter打开）。当您打开许多窗口时，它可能会很有用。

鲜为人知的经典Windows + Tab键盘快捷键提供的功能比乍看之下要多。

通过在当前版本的Windows 10中按这些键，我们可以在“任务视图”部分中找到自己。在这里，您不仅可以在应用程序之间进行切换，还可以使用“时间轴”和“虚拟桌面”。顺便说一句，您可以单击“演示任务”按钮（通常位于“开始”按钮旁边）来代替热键，或者从左边缘向触摸屏中心滑动手指。顺便说一句，如果您有一台现代笔记本电脑，请尝试触摸板手势：用三个手指向上滑动。


任务介绍模式

“时间线”


两年前 ，“ 时间表 ”出现在Windows 10中。它可以帮助您恢复以前在计算机上处​​理过的任务。如有必要，您也可以使用您的帐户*与其他设备同步。

对我来说，“时间轴”已经成为一种时间机器。许多项目的工作持续数天。例如，如果上星期五我处理了某些站点和文档，并在星期三返回此项目，则可以轻松地还原图片。我只是将秤重绕到所需的日期-同一天的星期五，看到并可以打开当时我所浸入的站点和文档。


时间线搜索

在“时间轴”上的搜索也为我提供了不止一次的帮助。与通常的文件搜索不同，我无法在设备上的所有文档中搜索（并且可能有很多文档），也就是我最近使用的文档。您可能熟悉Ctrl + F的组合，它可以在Explorer和许多应用程序中启动搜索。此组合将在“任务演示”屏幕上起作用：也就是说，您可以先按Windows + Tab，然后按Ctrl + F并输入搜索词以在“时间轴”中进行搜索。

Windows 10虚拟桌面


虚拟桌面的概念并不是什么新鲜事物。如果我们谈论Windows，那么使用它们的选项之一就是Desktops实用程序  ，它曾经是Mark Russinovich开发的（2012年发布的最新版本）。在Windows 10中，  虚拟桌面  内置于系统中，可帮助共享任务流并在它们之间进行切换。

如果您以前没有使用过虚拟表，请想象一下类似的例子来理解它们的逻辑：您可以使用多个监视器，在每个监视器上，您都可以通过按工作流划分它们来打开必要的程序，例如：在一个监视器上-使用邮件和日历，在另一个监视器上-使用多个Word文档，第三个-使用浏览器和OneNote。在每个时刻，您仅查看一台具有其自己的应用程序的显示器（虚拟桌面）。在虚拟表之间切换时，您似乎正在从一台监视器切换到另一台监视器。


拖动窗口以将其转移到新的虚拟桌面

您可以在“任务视图”屏幕上创建一个新的虚拟桌面：按下Windows + Tab并将所需的打开的应用程序窗口拖到标有“ +创建桌面”的字段中，它们将被移至另一个虚拟桌面。您还可以创建一个新的空虚拟表（Windows + Ctrl + D），然后在其上打开必要的程序。

您可以通过在“任务视图”屏幕上选择所需的表格来“转换眼睛”（即在自定义桌面之间切换），但是使用热键进行切换更加方便：Windows + Ctrl +左/右箭头，以及 现代触摸板上的  4根手指左还是右。

有用的多应用解决方案


现在大约还有另一种日常必需品-同时处理多个应用程序。

分屏


我想提醒您很多年了，它的原始形式（称为Aero Snap）出现在Windows 7中。在Windows 10中，它的功能得到了扩展，称为Snap Assist。这是关于将屏幕拆分为两个（在Windows 10中最多为四个）应用程序。


Snap Assist建议在右侧选择第二个用于停靠的窗口

为此，您需要将应用程序放在最顶部的条带上，将其置于屏幕的右边界或左边界，直到其“阴影”出现在屏幕上并释放它（从而固定第一个应用程序），然后在旁边紧挨着的其他应用程序的缩略图中选择第二个固定在附近。该场景很简单，它适用于鼠标和手指。使用Windows键盘快捷键+左/右箭头键，可以使此操作更加简单。这种组合已经有10多年的历史了，但是第一次使用它的人现在有时会体验到“数字魔术”的感觉。

对于好奇的用户，我也提醒您，在Windows 10中，您可以通过将应用程序移动到某个角落（或使用其他Windows键+向上/向下箭头）将其发送到屏幕的“四分之一”。修复两个应用程序时，可以移动它们之间的边界，从而为某些应用程序留出更多空间。要选择要在屏幕上固定的应用程序，可以在“任务演示”屏幕上右键单击其缩略图。

顶部窗口


在很多情况下，有必要将一个应用程序的窗口固定在另一个应用程序的窗口上（似乎在电视上称为“画中画”模式），但是如果您有此需要，我将提醒您有关两种小可能性的注意事项。

内置视频播放器的迷你模式（“电影和电视”应用程序，默认情况下在Windows 10中播放视频）。启动视频，然后单击右下角的小按钮（以迷你模式播放），带有视频的窗口将放置在所有窗口的顶部。


顶部视频窗口

只有将浏览器窗口固定在所有应用程序的顶部，才能使用单独的实用程序获得类似的机会。一旦我需要处理文档，就需要不断地与一个在线服务的网站进行核对，而 Microsoft Store上提供的Always on Top应用程序帮助了我。它嵌入在Edge的“共享”菜单中，使您可以将任何站点发送到位于所有应用程序顶部的窗口。我可以开玩笑说，该选项非常适合在处理Excel中的数据透视表时“用一只眼睛”观看YouTube频道。但是，正如我们在第一篇文章中讨论的那样，这种多任务处理很可能会损害浏览和工作。
