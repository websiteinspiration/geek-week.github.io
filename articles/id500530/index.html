Lebar dan tinggi kanvas lebih baik ditentukan dengan menggunakan atribut:

<canvas width="300" height="300"></canvas>

Jika kami ingin seluruh area tampilan menjadi kanvas, maka kami melakukan hal berikut:

const width = canvas.width = innerWidth
const height = canvas.height = innerHeight

Saya biasanya mendefinisikan kanvas dan konteks gambar dua dimensi sebagai berikut:

const canvas = document.querySelector('canvas')
