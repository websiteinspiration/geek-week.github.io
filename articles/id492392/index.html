Meskipun dukungan tambahan untuk EditorConfig sangat kuat, itu bukan pengganti lengkap untuk pengaturan sinkronisasi IDE lama yang baik, yang juga dapat menyinkronkan plugin IDE Anda.


Kesimpulan


EditorConfig di IDEA menjadi jauh lebih kuat. Ini tidak hanya mendukung parameter EditorConfig asli, tetapi juga banyak parameter khusus IDEA. Dengan cara ini, Anda dapat dengan mudah berbagi konfigurasi IDE terpadu di antara anggota tim Anda dengan konfigurasi nol. Tidak ada lagi pengaturan impor / ekspor atau pengaturan yang membosankan untuk menyinkronkan pengaturan IDE. Selain itu, sekarang Anda dapat memiliki parameter konfigurasi yang berbeda untuk setiap direktori secara terpisah.
