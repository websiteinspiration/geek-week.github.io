PythonのENM EricssonでRESTを使用する

こんにちは。しばらく前に、エリクソンは新しいエリクソンネットワークマネージャー（ENM）管理システムをリリースしました。これは、一部のモバイルオペレーターにすでに登場しています。それを操作する上でいくつかの質問を分析することは興味深いでしょう、そしてこの記事では、これまで（OSS-RCでは）見られなかったNorthbound Interface、つまりRESTful APIを操作する問題に触れます。Pythonとリクエストライブラリを使用します。

ユーザー管理、CM一括インポート/エクスポート、仮想ネットワーク機能、コレクション管理、セル管理などの機能は、RESTインターフェースを介して利用できます。 ALEXのドキュメントには、プログラミング言語に縛られることなく、このAPIの機能のかなり詳細な説明が含まれています。例として、Pythonのリクエストライブラリを使用してNBIセル管理に接続してみましょう。インターフェイスの説明は、ALEXライブラリの「構成タスク-CMセル管理RESTノースバウンドインターフェイス」で入手できます。

指定された機能により、1つのENM内のセルLTE、WCDMA、およびGSMのセル、セル間の隣接、周波数隣接の構成を制御できます。隣接するENMとOSS-RCの両方へのハンドオーバーを管理することもできます。
RESTfulインターフェースは、次のアドレスで入手できます。

https：// <customer-domain> / configuration-tasks / v1 / tasks

JSONリクエストの構造は次のとおりです。
  • リクエストURL：「configuration-tasks / v1 / tasks」
  • リクエストタイプ：POST
  • コンテンツタイプ：アプリケーション/ json
  • 本文：選択したチームのドキュメントによると。

Pythonでは、リクエストライブラリのSessionオブジェクトを使用します。

import requests
import json
from requests.packages.urllib3.exceptions import InsecureRequestWarning
from requests import Session
from requests.exceptions import HTTPError

class enmRestSession(Session):

必要なENM認証といくつかの「デフォルト設定」で重み付けします。

    def __init__(self, enm, login, password):
        super().__init__()
       #  /       
        self.enm = enm if enm[-1] == "/" else f"{enm}/"
        #  
        self.headers.update({"Content-Type": "application/json"})
        #   https 
        self.verify = False
        #   
        requests.packages.urllib3.disable_warnings(InsecureRequestWarning)
        #    ENM
        login_str = f"{enm}login?IDToken1={login}&IDToken2={password}"
        rest_response = self.post(login_str)
        #     
        if rest_response.status_code != requests.codes.ok:
            raise HTTPError()

RESTリクエストを送信するメソッドを作ってみましょう。

    #        request_body
    def send_configuration_task(self, request_body):
        url = f"{self.enm}configuration-tasks/v1/tasks"
        #    POST    URL 
        resp = self.post(url, data=json.dumps(request_body))
        return resp

コンテキストマネージャを使用するときに、ENMに自動セッションクロージャを追加します。

    def __exit__(self, exc_type, exc_val, exc_tb):
        try:
            #      logout
            self.get(f"{self.enm}logout")
        finally:
            super().__exit__(self, exc_type, exc_val, exc_tb)

結果の小さなアドオンは、必要に応じてスクリプトで使用できます。たとえば、RNCノードのすべてのセルを受信します。

def main():
    param = {"name": "readCells", "fdn": "NetworkElement=RNC01"}
    with enmRestSession(
                        "https://iegtbl8030-7.gtoss.eng.ericsson.se/",
                        "login",
                        "pass"
                       ) as s:
        print(s.send_configuration_task(param).json())
