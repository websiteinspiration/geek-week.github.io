CO2ダイエット中の大腸菌：地球温暖化との戦いにおける新しいツール？



どのようなシステムでも、それがどれほど複雑で多層化されていても、独自の基盤があります。それがなければ、システムは機能するように機能しません。私たちの惑星の生物圏には、すべてが置かれている基本的なレンガもあります。それらは独立栄養生物であり、無機化合物を有機化合物に変換できる生物です。今日は、イスラエルの科学者が実験室で二酸化炭素を摂食する新しいタイプのバクテリアを作成した研究にあなたと会います。開発プロセスではどのような方法が使用されましたか、細菌はどのように機能しましたか？この作業は人類にとって何を意味しますか？これについては研究グループの報告書から学ぶ。行け。

研究の基礎


独立栄養生物は、地球上で最も古い生物の1つと呼ばれます。最初の独立栄養生物は20億年前に出現し、進化を通じて従属栄養（無機から有機物を合成することができない）細菌が光合成能力を獲得したと考えられています。この用語自体は、1892年にドイツの科学者アルベルトベルンハルトフランクによって提案されました。


アルバートベルナールフランク

いくつかの生物は、有機化合物から炭素を得るが、無機化合物からエネルギーを得るので、独立栄養および従属栄養合成の岐路に立っています。この論理に従って、独立栄養生物はいくつかの主要なものに分けることができます：光合成生物、化学栄養生物、放射線栄養生物、石栄養生物、および混合栄養生物。遷移グループもあり、その代表は、合成スペクトルの1つまたは別のエッジに起因することが非常に困難ですが、それらの分類はまだ完了していません。

Phototrophsは、その名前が示すように、光子をエネルギー源として、または太陽エネルギーとして使用します。光合成などの栄養素が発生するのはこれらの生物からです。

ケモトロファム化学は物理学よりも近い。そのような生物は、エネルギー源としてさまざまなレドックス反応を使用します。化学合成。

最も不十分な研究であり、同時に最も好奇心が強いタイプは放射性栄養物です-「キノコの無線刺激」と呼ばれる現象の結果。これは、電離放射線による微細な真菌代謝を刺激するプロセスです。これらの生物は初めて1991年に黒カビの一部として発見され、そのサンプルはチェルノブイリ原子力発電所から取得されました。少し後（2006年）に、ニューヨークの科学者は菌類の無線刺激の仮説をテストし、色素メラニンを含むCladosporium sphaerospermum種の3つの菌類、Wangiella dermatitidisおよびCryptococcus neoformansがバイオマスと蓄積された酢酸（酢酸、CH 3 OH）を増加させることを発見しました）放射線レベルが標準を500倍超えた環境。

リソトローフ無機化合物は、好気性または嫌気性呼吸によって必要なエネルギーと炭素に変換されます。古細菌（核と膜小器官のない単細胞）と細菌の代表者だけが化学合成栄養を誇ることができます。

Mixotrophは、複数の異なる種類の食品を同時に（または状況に応じて代わりに）使用できるため、普遍的な兵士です。例えば、光合成菌と化学合成菌の両方です。

私たちが今日検討している研究の著者は、独立栄養生物の生命活動のプロセスをより詳細に理解することで、獲得した知識を大規模に適用できると信じています。そして、彼らの意見では、独立栄養を研究する最良の方法は、合成独立栄養生物を作成することです。理論的には、二酸化炭素を食べるバクテリアを作ることができます。しかし、このプロセスを呼び出すことは簡単な言葉ではありません。研究者自身が、研究を実現するために完了する必要がある3つの主要な段階を特定します。

第一に、独立栄養栄養への完全な移行のために、体は入ってくる炭素が専らCO 2からなる経路でCO 2固定のメカニズムを使わなければならないそして、出力は中央の炭素代謝に入り、12の主要なバイオマス前駆体すべてを供給する有機分子です。

第二に、身体は酵素のメカニズムを使用して、非化学エネルギー（光、電気など）を収集するか、炭素源ではないレドックスプロセスによって再生力を得なければなりません。

第三に、唯一の炭素源がCO 2である場合、体が一緒に持続的な成長をサポートできるように、CO 2のエネルギー収集と固定の方法を規制および調整する必要があります。 以前は、CO 2を食べる生物を作ろうとする研究が行われていました

しかし、それらの作品には大きな欠点が1つありました。それは、体内にマルチカーボン有機化合物が存在することであり、それが「予備の」栄養源として機能しました。言い換えれば、CO 2だけから炭素を取り込む従属栄養生物を作ることはまだ不可能です。

彼らの研究の方法論的基礎として、科学者はカルビンサイクル（Calvin-Benson-Bassamサイクル）を使用しました-植物、シアノバクテリアなどの光合成中の一連の生化学反応。このサイクルは、最も一般的な二酸化炭素固定メカニズムです。

そして、主なテストは「E. coli」という名前で私たちによく知られている細菌Escherichia coliでした。

研究成果


第一に、独立栄養への移行を実現するためには、代謝の再編成と身体の実験室での進化を行う必要がありました。いくつかの候補化合物は、細菌が完全な独立栄養に移行することを可能にするCO 2の固定のための電子供与体として役立つと考えられた。

電子源としてギ酸*が選択されました。この1炭素有機化合物は、プロセスの還元部分の供給源として機能しますが、大腸菌の成長を自然にサポートせず、バイオマスに吸収されないためです。
ギ酸塩* -ギ酸の塩およびエステル。
ギ酸塩の回復電位（E 0 = 420 mV）は、細胞内の主要な電子キャリアであるNAD +を減らすのに十分なほど低い（E. coliではE 0 = 280 mV ）。別の利点は、バイオマスがカーボンネガティブになる一方で、再生可能な供給源から電気化学的に取得できることです。 ギ酸塩から電子を収集し、それらをNADH（ニコチンアミドアデニンジヌクレオチド）の主要な細胞エネルギー回収リザーバー、メチロトローフ細菌Pseudomonas spからのNAD +にリンクしたFDH（ギ酸デヒドロゲナーゼ）に導くため。



大腸菌の 代謝ネットワークの化学量論分析（化学物質の質量比）は、FDH、ルビスコ（リブロースビスホスフェートカルボキシラーゼ）およびPrk（ホスホリボキナーゼ）を大腸菌の代謝ネットワークに追加することで、独立栄養増殖に十分であることを示しました（下の画像）。


画像No. 1：実験室で改変された化学栄養細菌E. coliのスキーム。

残念ながら、大腸菌の主要菌株における3つの組換え酵素の同時発現（BW25113）は独立栄養条件下での成長をもたらさなかった。化学量論的分析は、酵素の動力学、発現および調節のレベルを考慮に入れていないので、独立栄養成長を達成するための代謝最適化のためのツールとして適応実験室進化を使用することが決定されました。

この方法は、外来酵素機構の異種発現が細胞の可能な代謝反応の空間を拡大し、独立栄養増殖の可能性を提供するという事実によるものです。問題は、必要なストリームが最近拡張された一連の反応を通過するという保証がないことです。

したがって、大腸菌の中枢代謝以来従属栄養成長に適応した場合、従属栄養成長をサポートするフロー分布が使用される可能性があります。そのため、実験室での進化を利用して、目的の代謝経路に沿って流れを変えることができました。

実験室進化の最も重要なプロセスの1つは、Rubiscoカルボキシル化ストリームへの依存を確立するための中央代謝の再配置と、元の従属栄養経路を通る流れを抑制するための成長培地の適応です（2A）。つまり、従属栄養の代謝メカニズムを利用して細菌を停止させ、独立栄養に切り替える必要がありました。


画像No. 2：従属栄養細菌E. coliを化学栄養性細菌に変換するために開発された進化戦略のスキーム。

まず第一に、人工進化の間に、中央の炭素交換の2つの酵素をコードする3つの遺伝子が除外されました：解糖系のホスホフルクトキナーゼ（Pfk）と酸化的ペントースリン酸経路のグルコース-6-リン酸デヒドロゲナーゼ（Zwf）。最初のものは、2つの遺伝子（pfkAとpfkB）によってエンコードされた2つのアイソザイムを持っています。キシロース*で細胞を成長させる場合、この再配置により、細胞の成長が化学合成への移行に必要なルビスコのカルボキシル化に依存することが保証されます。
キシロース*は、ペントース単糖（C 5 H 10 O 5）です。
次に、Rubisco、Prk、炭酸脱水酵素（CA）の異種発現を行い、CO 2と重炭酸塩とFDH を変換しました。これに続くのは、一定の炭素欠乏状態にある細胞をサポートする、キシロース制限ケモスタット*で細胞を成長させるプロセスです。
ケモスタット*は、微生物が成長する栄養培地で基質の最適なバランスと濃度が維持されている場合に微生物を培養する方法です。
このような培養液は、細胞の増殖を可能にしますが（細胞分裂による組織の成長）、従属栄養性の異化経路を遅くします。
異化作用* -複雑な物質のより単純な物質への代謝分解または物質の酸化（エネルギー代謝）。
細胞を増殖させたケモスタットも、ギ過剰を含有し、絶えずCOでパージした2富化空気（CO 2含有量の 10％）。

したがって、この成長培地は、従属栄養を遅くし、細胞が独立栄養になる傾向があります。細胞は文字通り、有機糖の外部炭素への依存を減らすことを余儀なくされています。

栽培方法は準備ができていて、それを確認する必要がありました。週に1回、サンプルをケモスタットから取り出し、独立栄養条件下での成長をテストしました。特に、これらは大腸菌の化学有機栄養条件であり、高濃度のCO 2を含む雰囲気で30 mM（ミリモル）ギ酸ナトリウムを添加したM9タイプの培地で構成されています。（10％）、他の炭素源はありません。

約150世代に相当するケモスタットで約200日間増殖した後、キシロースを欠く培地で増殖が検出されました（つまり、独立栄養条件下）。この表現型は、その日のすべてのサンプルに存在していました。 350日目に、キシロースは培地から完全に排除された（2B）。持続的な成長と濁度は、キシロース非依存性細胞のみがケモスタットに存在することを意味していました。サンプルには、その成長のために高濃度のCO 2を含む培地が必要であることがわかりました。これは、炭素固定メカニズムを示唆しています。

次に、科学者たちは、より深い分析のために、最も堅牢なクローンの 1つを選択しました。見つかった倍加時間* 18±4時間（2C）。
クローン* -この場合、遺伝的に同一の細胞のグループを意味します。
倍増時間* -サイズが2倍になるまでにかかる時間。
成長した細胞が本当に独立栄養であることを確認する必要があり、それらの成長の間、「隠れた」炭素源や従属栄養のギ酸塩の活性化はありませんでした。これを行うために、同位体の標識について実験が行われた。

まず、進化したクローンを13 Cでラベルしたギ酸塩と13 CO 2を含む培地で成長させました（安定した同位体状態が得られるまで10世代）。次に、さまざまな代謝産物の13 Cラベルの分析を、液体クロマトグラフィーとタンデム質量分析によって実施しました。
13 C * -炭素-13、安定した炭素同位体。
代謝物* -任意の化合物の代謝産物。

分析により、中央代謝バイオマスのビルディングブロック内の炭素原子の約98％が正常にラベル付けされたことが示されました。これらのデータは、栄養培地に溶解した約99％の13 Cと1％の非標識重炭酸塩を含む、標識されたギ酸塩とCO 2に対応しています。 この観察結果は、細胞のバイオマス中の炭素が正確にCO 2とギ酸塩に由来するという反駁できない証拠です。



さらに、ギ酸塩がバイオマスに濃縮されているかどうかを確認した。このために、炭素-13で標識されたギ酸塩を使用して、細胞をM9培地（大気中のCO 2の濃度は以前の実験と同様に10％であった）で成長させた。所定の環境での成長後のバイオマスのビルディングブロック

の13 C マーキングスキームは、1〜2％（3B）以内の13 C マークを示しました。これは、自然な13 C 含有量と少量の標識されたギ酸塩に基づく期待値です。言い換えれば、結果は細胞がギ酸塩を同化しないことを示した。 上記の実験の結果の合計は、栽培作物の炭素源がもっぱらCOであることを自信を持って示唆しています

2とギ酸塩。そして、これは今度は、実験室で進化を遂げた大腸菌細胞の100％の独立栄養を示しています。

科学者はこのステートメントを検証するために別の実験を行い、標識された13 CO 2および標識されていないギ酸塩が使用されました。13 CO 2のコストが高いため、実験が行われた船は閉鎖された。この小さなニュアンスは非常に重要です。密閉された環境（以前の実験では、コンテナは換気されていた）により、ラベルのないCO 2が蓄積されたためです。ギ酸塩の酸化により形成された。そして、これは観測結果を歪めます。しかしながら、「汚染」プロセスは、標識されたグルタミン酸を分析することによって追跡され、目でそれを修正することさえできます。

この経験は、中央代謝バイオマスのビルディングブロック内の炭素原子の約85〜90％が正常にラベル付けされていることを示しました。画像3Aと3Bに見られるように、13 Cでラベルされたコンポーネントの調整を適用すると、バイオマス内の原子のラベルはほぼ100％になり、大腸菌バクテリアの発生の独立栄養性を示します。

細菌が独立栄養菌になったことは疑いの余地がありません。研究室の進化の過程でどのような遺伝的変化、すなわち突然変異が発生したのかはまだ解明されていません。

これを明確にするために、科学者たちはギ酸塩で独立栄養増殖できる6つのクローンを単離し、ゲノムとプラスミドの配列を決定しました*。
プラスミド*は、染色体から物理的に分離され、自律複製できるDNA分子です（親DNA分子に基づいて2つの娘DNA分子を作成するプロセス）。
2つのクローン（クローン1および2）は、キシロースが培地にまだ存在しているときに分離され（進化の250日目）、キシロースがケモスタットの培地から除外された後の3つのクローン（クローン3、4および5）（進化400日目）。最後のクローン（クローン6）は、以前に分離されたクローンの1つ（クローン1）をいくつかの連続希釈サイクルで増殖させた後に分離されます。


画像4：独立栄養への移行の遺伝的基礎。

驚いたことに、突然変異の数は非常に少なかった。研究者たちはそれらを3つの主要なカテゴリーに分けました。

最初のカテゴリーは、カルビン回路の機能と直接代謝的に関連する酵素をコードする遺伝子で構成されています。リボースリン酸ジホスホキナーゼをコードするこの遺伝子は、リボースリン酸をバイオマスに誘導します。

突然変異した遺伝子の2番目のカテゴリーは、適応実験室進化に関する以前の実験で突然変異した遺伝子から成ります：pcnB（R161P）、rpoB（D866E）、rpoD（F563S）、malT（E359K）およびaraJ（W156）。科学者はこれらの突然変異を実験室の進化のまさにプロセスに関連付けます、すなわち、それらは必ずしも細菌の独立栄養への移行のプロセスに関連付けられていません。同様に、D-キシロース糖（E337K）の異化の原因となるオペロンの調節タンパク質をコードするxylR遺伝子に変異が見つかりました。これは、培養中のケモスタットにおけるキシロースの長期的な飢餓に関連付けられていますが、独立栄養には決して関連付けられていません。

突然変異の3番目のカテゴリーには、特徴的な役割を持たない突然変異が含まれ、「遺伝的ヒッチハイク」などの現象の結果である可能性があります。異なる分離株では、2〜27個の追加の突然変異遺伝子があり、その一部は独立栄養表現型の突然変異である可能性がありますが、厳密に必要なわけではありません。

将来的には、科学者は遺伝子変異の追加研究を実施して、バクテリアが独立栄養に移行するために基本的かつ必要なものを決定する予定です。

エピローグ


この研究では、科学者は優れた結果を達成することができました。まず第一に、それは実験室の進化に注目する価値があります-科学者によって制御されたプロセスであり、科学者が新しい「デザイン」に従って体を変形させることができます。

大腸菌は進化的操作によって文字通り独立栄養生物になり、CO 2を吸収して炭素を供給しました。このような変異菌は、二酸化炭素が原因の一つである地球温暖化に苦しむ社会にとって非常に有用である可能性があります。しかし、研究者自身が認めているように、現時点では、彼らの細菌はギ酸の吸収中により多くのCO 2炭素固定中に吸収するよりも。この「厄介な」状況にもかかわらず、科学者のこの研究を役に立たないと考えることは犯罪です。まず、これらは従属栄養生物の独立栄養生物への変換を理解するための最初のステップにすぎません。第二に、さらなる研究により、ギ酸よりもバクテリアに対してより効率的なエネルギー源を選択できるようになり、CO 2の放出が大幅に削減されます。主なことは、基礎がすでに敷かれているということです、レンガをレンガごと挿入するために徐々にそして非常に注意深く残っています。

地球温暖化から世界を救うバクテリア、それは信じられないほどですが、少しSFに聞こえます。しかし、そのようなバクテリアが作成されたとしても、これは地球上の環境状況の責任者を救うものではありません。それは純粋に彼らが掃除する場所ではなく、彼らがゴミを捨てない場所だからです。
