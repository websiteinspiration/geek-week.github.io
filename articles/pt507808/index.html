Como ignorar o controle deslizante captcha usando JS e Puppeteer

Amigos, trago à atenção de vocês uma tradução do artigo “Como ignorar o“ controle deslizante CAPTCHA ”com JS e Puppeteer de Filip Vitas.



Spam para o proprietário do site é um grande problema. Por outro lado, o captcha me deixa louco e piora a experiência do usuário.



Captcha é uma merda. Vamos apenas dizer. Há muitas maneiras de passar na verificação e cada uma delas é ruim.



Nos últimos anos, os bots se tornaram mais inteligentes a cada dia, e proteger o site deles está se tornando difícil. Com tempo e recursos suficientes, você pode contornar quase qualquer captcha. O Puppeteer possui plugins para passar no Recaptcha. Algumas empresas fornecem serviços especificamente para isso. Por exemplo, 2Capcha. Veja como usar o Puppeteer e o 2Captcha .



Existem sites onde, como alternativa, é utilizado o captcha com um controle deslizante ("deslize o controle deslizante para verificação"). Mas por que alguém usaria um captcha tão simples com um controle deslizante?



As razões são as seguintes:



  • A maioria dos bots não é executada em JS, portanto você pode detê-los.
  • Slider é mais amigável
  • Para usuários de dispositivos móveis, "deslize" o controle deslizante - a ação usual


Portanto, para as pessoas, os controles deslizantes são claros e simples.



Vamos tentar contornar alguns captcha com um controle deslizante.



Mova o controle deslizante para enviar



O jQuery é usado em formulários da web onde existe um elemento "deslize o controle deslizante para enviar". Essa é uma alternativa ao captcha que evita spam ao preencher formulários da web.







Primeiro preenchemos os campos de entrada. Para mover esse controle deslizante, precisamos:



  • passe o mouse sobre o centro do controle deslizante
  • clique no botão do mouse
  • mover o mouse
  • solte o botão do mouse






Feito. Foi fácil.



Controle deslizante Dipbit



Dipbit - um site para troca de moedas digitais. Ambas as páginas - login e registro - contêm o elemento "deslize o controle deslizante para verificação".







Dipbit está tentando ser um pouco inteligente, então precisamos adicionar código para ocultar a operação do Puppeteer.
