1C-Bitrix mencegah berhenti berlangganan buletin dengan persyaratan untuk mengirimkan data pribadi mereka

Banyak hal yang dituangkan ke kotak surat saya, termasuk dari Bitrix (sebenarnya perlu dicatat bahwa saya pernah mendaftar di situs web mereka).

Tapi sekarang saya bosan dengan surat-surat ini dan saya memutuskan untuk berhenti berlangganan. Ini sangat mudah dilakukan, bukan? Tapi ternyata, tidak dalam kasus 1C-Bitrix. Apa yang tidak saya harapkan dengan cara apa pun :) Yaitu, tanda centang wajib “Saya telah membaca dan menerima ketentuan kerahasiaan informasi pribadi, termasuk mengenai pemrosesan dan penggunaan data pribadi saya”.

Saya suka ini: mengapa? Artinya, saya ingin mereka tidak menulis kepada saya lagi, tetapi mereka ingin saya menjual jiwa mereka pada mereka pada akhirnya? Pertama kali saya bertemu dengan kurang ajar seperti itu.
Secara umum, saya tidak berhenti berlangganan, tetapi hanya melemparkan mereka ke dalam spam dan Google mengatakan bahwa saya tidak akan bosan dengan ini lagi.

Saya ingin mendengar pendapat pengacara yang berpengetahuan luas, seberapa sahkah kelancangan seperti itu? Sepertinya hukum tidak memiliki hak untuk mengganggu tanggapan.
