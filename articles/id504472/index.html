Alat Pengembang Gratis Teratas Saya

Ini adalah terjemahan dari artikel dari dev.to. Pengarangnya Fayaz Ahmed membagikan daftar alat pengembang favoritnya.



Saya mengembangkan proyek-proyek seperti Tvflix , Gson , Appy Dev , serta banyak microprojects yang berbeda, jadi saya tertarik dalam menemukan alat bantu gratis untuk pekerjaan saya.

Ini adalah daftar pribadi saya, tetapi Anda dapat menambahkannya dengan alat sukses lainnya. Inilah beberapa favorit saya.

Layanan dan platform untuk menyebarkan frontend / backend Anda


1. Netlify
Gunakan Netlify untuk menyebarkan situs web statis modern. Anda mendapatkan infrastruktur CDN, penyebaran berkelanjutan, HTTPS satu-klik dan layanan apa pun yang Anda butuhkan. Mulai secara gratis.

2. Vercel (sebelumnya Zeit)
Vercel adalah cara termudah untuk menyebarkan situs web. Tuan rumah proyek web tanpa konfigurasi Anda dengan infrastruktur SSL otomatis dan CDN global.

3. Surge
Forwarding proyek web harus cepat, mudah, dan aman. Surge adalah fitur penerbitan web statis untuk pengembang front-end langsung dari CLI.

4. Firebase
Alternatif untuk Netlify / Zeit dari Google. Firebase adalah platform seluler yang membantu Anda dengan cepat mengembangkan aplikasi berkualitas tinggi dan mengembangkan bisnis Anda.

5. Aplikasi Web Statis Azure
Mulai hari ini, Anda memiliki kesempatan untuk menggunakan Aplikasi Web Statis untuk meng-host aplikasi satu halaman di Azure.

6. Situs GitHub Pages
situs web untuk Anda dan proyek Anda yang diterbitkan langsung dari repositori GitHub Anda. Cukup untuk mengedit dan menjalankan sehingga perubahan menuju ke versi live.

7. GitLab Pages
Seperti GitHub, layanan GitLab Pages memungkinkan Anda untuk mempublikasikan situs web statis langsung dari repositori ke GitLab.

8. Aerobatik
Hosting statis tugas berat. Bekerja sangat baik dengan generator situs statis seperti Hugo, Jekyll, Gatsby, Gridsome.

9. Glitch
Mungkin ini adalah salah satu alat yang paling sering saya gunakan saat ini. Layanan gratis yang sederhana dan kuat untuk membuat dan meng-hosting aplikasi berdasarkan Node.js.

10. Heroku
Heroku, versi Glitch yang lebih canggih, adalah "platform as a service" (PaaS) yang memungkinkan pengembang untuk membuat aplikasi, menjalankannya dan mengelolanya sepenuhnya di cloud.

Alat Otomasi


1. Cron-job.org
Cron-job adalah layanan open source gratis dan memungkinkan Anda menjalankan tugas dalam jumlah tidak terbatas dalam format cron. Yang Anda butuhkan adalah menggunakan API atau alamat email webhook Anda, yang akan digunakan untuk terhubung ke layanan dengan frekuensi yang ditentukan.

2. n8n
Alat sumber terbuka ini merupakan alternatif untuk Zapier. Layanan ini menyediakan kemungkinan tak terbatas, terus memperbarui daftar alat dan aplikasi yang tersedia untuk sinkronisasi. Anda juga dapat meng-host n8n di server Anda sendiri.

3. Integrasi
Integromat adalah alat yang mudah digunakan namun efektif untuk mengotomatisasi proses manual dengan karakteristik unik. Sinkronkan aplikasi, layanan, dan perangkat favorit Anda satu sama lain, walaupun Anda tidak memiliki keterampilan pemrograman.

4. IFTTT
Coba IFTTT - cara termudah dan paling bebas untuk berbuat lebih banyak dengan aplikasi dan perangkat favorit Anda. Pekerjaan rumah akan membutuhkan lebih sedikit perhatian, pekerjaan akan menjadi lebih produktif, Anda akan menjaga privasi dan data Anda akan aman. Saya yakin semuanya bekerja lebih baik dalam kondisi sinkronisasi.

5. Pipedream
Pipedream adalah platform tempat pengembang dapat membuat dan menerbitkan proyek yang terkait dengan pemrosesan data. Buka proyek dan garpu pada platform yang berisi ribuan contoh kode sumber dan proyek yang diselesaikan.

Alat tanpa kode


1. NoCodeAPI
Cara termudah bagi pengembang front-end untuk menghubungkan aplikasinya ke layanan pihak ketiga tanpa menulis kode tambahan dan server khusus. NoCodeAPI dapat bekerja dengan Google Sheet, Airtable, Google Analytics, Twitter, Instagram, XML, RSS, JSON, Slack, sistem pembayaran, dan lebih dari lima puluh layanan.

2. GlideApps
Glide mengubah tabel dinamis menjadi aplikasi yang mudah digunakan, tanpa memerlukan kode. Pilih tabel atau mulai dengan templat, konfigurasikan pengaturan untuk aplikasi Anda, lalu bagikan dengan yang lain hanya dalam beberapa saat. Cobalah gratis hari ini!

3. Airtable
Airtable berfungsi seperti tabel dinamis, tetapi pada saat yang sama menyediakan kemampuan database untuk mengatur konten apa pun.

4. Bubble
Bubble memperkenalkan Anda dengan cara baru untuk membuat aplikasi web. Alat pemrograman ini tidak memerlukan pekerjaan dengan kode dan beroperasi dalam format point-and-click. Bubble menampung semua aplikasi di platform cloud-nya.

5. Material Icon Generator
Hasilkan SVG-ikon dalam gaya desain Material untuk aplikasi Anda. Layanan sumber terbuka.

Alat untuk SEO / Pemasaran Internet / Promosi / Dukungan Hukum


1. Tag Meta
Menghasilkan dan melihat dulu semua tag meta dan tag untuk jejaring sosial di halaman web.

2. Alat Kode Web
Buat skema, tag, dan banyak potongan kode berulang.

3. Karbon
Karbon adalah cara termudah untuk membuat dan menerbitkan gambar visual dari kode sumber Anda.

4. Codeimg
Buat dan publikasikan gambar visual dari kode sumber Anda.

5. GetTerms Hasilkan
teks dasar dari Perjanjian Pengguna dan Kebijakan Privasi untuk situs web Anda.

6. BEE Gratis
BEE adalah editor email yang memungkinkan Anda membuat email balasan hebat dengan cepat dan tanpa masalah. Penulis berusaha mengembangkan editor email yang sempurna. Dimungkinkan untuk menggunakan BEE online atau menyematkan dalam aplikasi Anda.

Anda dapat menemukan lebih banyak alat serupa di sini .
