Pure Function - A function that is deterministic and has no side effects.



That is, for a function to be pure, it must

be deterministic - that is, each time with the same set of arguments produce the same result.

Determined Function Example

  return a+b

print(sum(3,6))

The second important quality of a pure function is the lack of side effects

will show clearly 

my_list = [32,3,50,2,29,43]

def sort_by_sort(li):
  li.sort()
  print(li)

print(my_list)

>>>[2, 3, 29, 32, 43, 50]
>>>[2, 3, 29, 32, 43, 50]


The sort_by_sort function has side effects because it changes the original list of elements and displays something in the console. 



my_list = [32,3,50,2,29,43]

def sort_by_sorted(li):
  return sorted(li)

print(sort_by_sorted(my_list))
print(my_list)

>>>[2, 3, 29, 32, 43, 50]
>>>[32, 3, 50, 2, 29, 43]


Unlike the previous example, the sort_by_sorted function does not change the original array and returns the result without outputting it to the console on its own.



Pure functions are good because

- They are easier to read

- They are easier to maintain

- They are easier to test

- They do not depend on the order in which they are called



Functional programming can be characterized by the fact that it uses only pure functions



Higher-order functions - in programming, a function that takes other functions as arguments or returns another function as the result.





FVP example 



def func(num):
  return num**2

def higher_order_func(fun, num):
  return fun(num)+fun(num)

print(func(4))
print(higher_order_func(func,4))

>>>32

Lambda function



Lambda is a tool in python and other programming languages ‚Äã‚Äãfor calling anonymous functions. Most likely this will not say anything and will not clarify in any way how it works, so I will tell you just the mechanism of lambda expressions.



Everything is very simple.



Consider an example. For example, we need to write a function that would consider the area of ‚Äã‚Äãa circle with a known radius.



The circle area formula is



S = pi*(r**2)







where

S is the area of ‚Äã‚Äãthe circle

pi - the mathematical constant equal to 3.14 which we will get from the standard library Math

r - the radius of the circle is the only variable that we will pass to our function







Now let's format it all in python 



import math #  math

pi_const = round(math.pi, 2) # pi           3.141592653589793     

#              
def area_of_circle_simple(radius):
  return pi_const*(radius**2)

print(area_of_circle_simple(5))
print(area_of_circle_simple(12))
print(area_of_circle_simple(26))
>>>78.5
>>>452.16
>>>2122.64




It seems to be nice, but it can all look a lot cooler if you record it through lambda 



import math #  math

pi_const = round(math.pi, 2) # pi    
#       
#  3.141592653589793     

print((lambda radius: pi_const*(radius**2))(5))
print((lambda radius: pi_const*(radius**2))(12))
print((lambda radius: pi_const*(radius**2))(26))

>>>78.5
>>>452.16
>>>2122.64


To make it clearer, an anonymous function call implies that you use it without declaring it anywhere, as in the example above.



Lambda function works according to the following principle 




print((lambda     :     )( ))

>>>




Consider an example with two input arguments. For example, we need to calculate the volume of a cone using the following formula: 



V = (height*pi_const*(radius**2))/3






Let's write it all in python: 



import math #  math

pi_const = round(math.pi, 2) # pi           3.141592653589793     

#      
def cone_volume(height, radius):
  volume = (height*pi_const*(radius**2))/3
  return volume

print(cone_volume(3, 10))

>>>314.0


And now how it will look in lambda form: 



import math #  math

pi_const = round(math.pi, 2) # pi           3.141592653589793     

print((lambda height, radius : (height*pi_const*(radius**2))/3)(3, 10))

>>>314.0




The number of variables here is unlimited. For example, we calculate the volume of a truncated cone, where we take into account 3 different variables.



The volume of the truncated cone is calculated by the formula: 



V = (pi_const*height*(r1**2 + r1*r2 + r2**2))/3






And here is how it will look in python classically 



import math #  math

pi_const = round(math.pi, 2) # pi           3.141592653589793     

#      
def cone_volume(h,r1,r2):
  return (pi_const * h * (r1 ** 2 + r1 * r2 + r2 ** 2))/3

print(cone_volume(12, 8, 5))
print(cone_volume(15, 10, 6))
print(cone_volume(20, 12, 9))

>>>1620.24
>>>3077.20
>>>6970.8




Now let's show how it will look with lambda: 



import math #  math

pi_const = round(math.pi, 2) # pi           3.141592653589793     

print((lambda height, radius1, radius2 : (height*pi_const*(radius1**2 + radius1*radius2 + radius2**2))/3)(12, 8, 5))
print((lambda height, radius1, radius2 : (height*pi_const*(radius1**2 + radius1*radius2 + radius2**2))/3)(15, 10, 6))
print((lambda height, radius1, radius2 : (height*pi_const*(radius1**2 + radius1*radius2 + radius2**2))/3)(20, 12, 9))

>>>1620.24
>>>3077.20
>>>6970.8
