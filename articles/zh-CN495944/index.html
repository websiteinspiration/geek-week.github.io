现在详细介绍一些细节。自然，那些首次进入TOP50的项目不能不引起人们的兴趣-让我们看看它们的特性和功能。

如前所述，Christofari超级计算机大受好评：它在Linpack测试中的性能显示为6.7 PFlop / s，峰值性能为8.8 PFlop / s。如果考虑到T-Platforms公司的下一个Lomonosov-2系统分别具有2.5 PFlop / s和4.9 PFlop / s的数字，我们可以假设Kristofari稳固地处于领导地位。该项目的技术基础由配备Tesla V100图形加速器的NVIDIA DGX-2计算节点组成。超级计算机旨在与来自各个问题领域（医学诊断，面部识别，语音生物测定法）的人工智能算法一起使用。在世界500强排名中，克里斯托法里（Cristofari）排名第29位。



第一个“新手”是Christofari超级计算机

对于Hewlett Packard Enterprise的计算集群UITiS NOVATEK STC，给出了以下数据：在Linpack测试中，峰值性能为496.87 Tlop / s时为273.28 TFlop / s。通过使用第一代Xeon可扩展处理器系列来实现高功率。该系统的目的使其在其他参与者中脱颖而出：超级计算机是专门为解决地球物理学领域的研究和应用问题而开发的（提取知识，处理大量文本信息，建模过程和油气行业的现象以及更多）。



第二个“新”-UITiS集群NOVATEK STC

由Computer Ecosystems and TechnoCity制造的超级计算机“ PetaNode 1.0 Cluster”在Linpack测试中的性能为79.1 Tflop / s，峰值性能为141.47 Tflop / s。该开发的一个显着特征是使用AMD的加速器，每个节点使用10个GPU。这款超级计算机的主要目的是气候建模-这是TOP50参与者精通的一个小众市场。



最新的新颖之处是PetaNode 1.0集群超级计算机，

总体而言，由于首次亮相和更新，排名有所变化。由于该列表的新领导者和另外两个新来者，以前的TOP50列表中其余超级计算机的排名在评分中下降了1到5分。

关于项目总量的技术设备，变化不是根本的，而是有形的。英特尔处理器仍然是最新一代超级计算机必不可少的属性-目前，该排名中没有其他解决方案。 InfiniBand网络（33例对35例）和更多的频率-10千兆以太网（8例对6例）变得不那么普遍地用于节点之间的通信。

应当单独指出的是，该等级的三个新加入者中的两个（Kristofari和PetaNode 1.0集群）为使用图形加速器从构建异构集群和混合集群向同类系统的过渡（即使没有回报）树立了明确的趋势。这种趋势似乎受能源效率和密度标准的驱动。

在大多数情况下，超级计算机的应用程序保持不变。科学和教育领域仍然是最相关的-它由23个系统提供服务，而在应用研究中使用了10台超级计算机（比上一版本的超级计算机少三台）。该行业仍然无法获得最新的技术进步-每50台机器中只有3台可以满足其需求。此外，该评级还包括专为诸如地震预报，IT服务，云存储，为商业和金融部门提供服务等任务而设计的项目。

最后，惠普企业，RSK公司集团和T平台仍然是最高标准的超级计算机最活跃的供应商。
他们对TOP50的贡献总计为35个项目（分别为13、11和11）。其余职位也主要在知名玩家之间分配。

下一次评级将在今年9月21日之前进行审查和更新-我们正在等待下注。
