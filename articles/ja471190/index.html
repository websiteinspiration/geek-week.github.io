Laravelを学習するためのリソース


私が個人的にLaravelでクールだと思うのは、その中でどのように作業するかを簡単に学ぶことができるということです。Laravelエコシステムは、単なる開発ツールではありません。このフレームワークには、作業の基本的な機微を学ぶのに役立つプラットフォームとコースの優れたデータベースがあります。それらの一部を次に示します。

ララキャスト


Laracastsは、Web開発者向けの学習リソースです。このリソースには、Laravel、Vue、PHPの一般的な概念、データベースなどに関する優れたスクリーンキャストが含まれています。あなたの将来のキャリアにとってはNetflixのようなものだと言えるでしょう。Laravelでゼロからトレーニングを開始し、Laravel の新機能で常に最新の状態に保つことができます。

テスト駆動ララベル


開発テクニックについて話している場合、私はテスト（TDD）による開発を使用することを好みます。Laravelを使用すると、このテクニックは非常に使いやすくなります。これは、テスト駆動Laravelコースで学習できます。ビデオコースを受講すると、より信頼性の高いアプリケーションを構築できます。

自信のあるララベル


テストでコードをカバーする必要があることは誰もが知っていますが、テストを含むのは一部のアプリケーションだけです。自信を持ってLaravelビデオコースは、テストを書く際の障壁を取り除くのに役立ちます。経験豊富で意欲的な開発者はすべてテストを作成します。そしてもちろん、誰もがこのプロセスにかかる時間を短縮したいと考えています。自信を持ってLaravelは、Laravelアプリケーションのテストを作成し、完全にそれらに自信を持つ方法を教えます。

ララコンオンライン


Laracon Onlineは毎年開催されるオンライン会議です。これは、各開発者がLaracon会議の参加者のように感じるための最も便利で手頃な方法です。

効果的なPHP


Nuno Maduroの効果的なPHPビデオコースを視聴することを強くお勧めします。これは、すべてのPHP開発者の教育に大きく貢献します。このコースでは、コードを簡潔かつ明確に記述する方法を説明し、バグを回避するための基本原則などについても説明します。

Laravelコアアドベンチャー


Laravel Core Adventuresは、Laravelの基本的な概念に関する優れたビデオコースで、知識を得るだけでなく、楽しい時間を過ごすのにも役立ちます。

チャットボットコースを構築する


チャットボット の構築コースでは、PHPを使用してチャットボットと音声ボットを開発、拡張、テストする方法を学びます。

Laravel認定


そしてもちろん、Laravelで開発に必要なすべての知識とスキルをすでに取得している場合は、Laravel認定資格プログラムを使用してそれらを確認できます。

Laravelコミュニティ


Laravel自体はすでに優れたフレームワークです。しかし、彼の絶え間ない発展は彼のコミュニティと非常に密接に関係しています。開発者は常に自分の経験とベストプラクティスを共有し、ヒントやアドバイスを提供し、改善が必要なものに注意します。Laravel開発者に必要なリソースのいくつかを指摘したいと思います。

ララキャストフォーラム


Laracastsフォーラムは、開発者が経験とチュートリアルを共有するプラットフォームです。ここでは、いつでもアドバイスを求めたり、質問について話し合ったりできます。

LaravelIO


LaravelIOは、開発の問題についても説明する優れたリソースです。そこでは、さまざまな質問に対する答えを見つけて、あなたの経験を共有することができます。

Laravel Newsは、Laravelの公式ニュースリソースであり、Laravelコミュニティの最新トレンドに関する週刊ニュースレターでもあります。

Laravelポッドキャスト


Laravelポッドキャストは、Laravelコミュニティの主要なトレンドについて話し合うのに最適な場所です。

ララジョブズ


LaraJobsは、求人検索や、Laravel、PHP開発、その他の専門技術に関連するポジションの求人情報を掲載するためのリソースです。

Laravelブログ


Laravelブログは、Laravelの公式ブログであり、Laravelの最新リリースやさまざまなイベントのほか、Taylor Otvel（Laravelの作成者）のポッドキャストを見ることができます。

Laravelエコシステム


Laravelエコシステムは間違いなく開発者に提供するものを持っています。私はいつも自分が使っているものを褒めないようにしていますが、もっと批判する傾向があります。しかし、Laravelエコシステムに関して言えば、私はそれを助けることができません。ここでは、プロジェクトの開発に役立つ、いくつかのオリジナルで非常に便利なツールについて説明します。

開発環境


Laravel valet


Laravel Valetは、Laravelアプリケーション用の最小限の開発環境を簡単にインストールします。MacOSユーザーにとって本当に素晴らしいツールです。

主な特徴：

  • 高速（約7 MBのRAMを使用）
  • インストールが簡単
  • 設定は必要ありません（ルートディレクトリにフォルダーを作成するだけです）
  • 使いやすい（シンプルなコマンドラインツールを提供）
  • VagrantやDockerは不要
  • ngrokトンネルをサポート
  • Homebrewを使用して追加のサービス（RedisやMySQLなど）を手動でインストールできます

Laravel Valetはオープンソースソフトウェアです。これに関するすべてのドキュメントは、Laravel の公式Webサイトにあります。

ララヴェルホームステッド


Laravel Homesteadは、優れた開発環境を提供するビルド済みのVagrantビルドです。PHP、Webサーバー、またはその他のサーバーソフトウェアをローカルコンピューターにインストールする必要はありません。

主な特徴：

  • すべてのオペレーティングシステムで動作します：Mac、Windows、Linux
  • 浮浪者ボックスは簡単に取り外して再作成できます。
  • 必要なサービスはすべて「そのまま」使用できます（PHP 7.3、Nginx、MySQL、Redisなど）。
  • 仮想マシンとプライマリの間で共有フォルダーを使用する機能を提供します

Laravel Homesteadもオープンソースソフトウェアです。インストールに関するすべてのドキュメントは、Laravel の公式Webサイトにあります。

Laravel拡張とパッケージ


Laravelパスポート


Laravel Passportは最も簡単な認証APIツールです。アプリケーションに完全なOAuth2サーバー実装を提供します。

主な特徴：

  • コマンドラインツールにより、OAuth2サーバーのインストールプロセスが簡素化され、使いやすくなります
  • 必要なデータベースの移行、コントローラー、ルートがすべて付属しています
  • Vue.JS組み込みコンポーネントが含まれています

Laravel Passportはパブリックドメインです。ここで、インストールに必要なすべてのドキュメントを見つけることができます。

ララヴェルスカウト


Laravel Scoutを使用してEloquentモデルに全文検索を追加します。これは、検索インデックスをEloquentレコードと同期させるのに便利なソフトウェアです。

主な特徴：

  • Algoliaドライバーが付属
  • Algoliaの代わりにドライバーを記述して、Scoutを独自の検索実装（Elasticsearch、MySQL全文検索、TNTSearchなど）で補完できます。

Laravel Scoutはオープンソースソフトウェアです。上の公式ウェブサイトあなたは彼についての詳細を学ぶことができます。

ララヴェルスパーク


Laravel Sparkは、SaaSアプリケーションの開発を加速するための便利なツールです。

主な特徴：

  • あなたの製品を開発するための素晴らしいツール。
  • ユーザーのアプリケーションに付加価値を与えることに集中できます。
  • ユーザー認証、予定されたイベント、支払いなどを処理します
  • 高度にカスタマイズ可能
  • Vue.JSでアプリケーションにより多くの力を与えます
  • Stripe.js v3を搭載し、支払いおよびサブスクリプションプロセスを最大限に保護
  • Bootstrap 4.0を使用
  • Laravelの最新バージョンをサポート
  • 完全にローカライズされています
  • 便利で直感的な設定パネルを装備
  • ビジネスロジックに集中して、最終製品がどのように見えるかを確認できます

Laravel Sparkは、サイトごとに99ドルまたは299ドルの価格で購入できる有料パッケージで、サイトの数に制限はありません。詳細については、公式ウェブサイトをご覧ください。

ララヴェル・ノヴァ


Laravel Novaは、開発プロセスをスピードアップする優れたUIとUXデザインを備えたLaravelの管理パネルです。

  • アプリケーションのさまざまなメトリックを表示します（クエリを構築するためのヘルパーを含みます）
  • Laravel承認ポリシーと統合（リンク、レンズ、フィールド、ツールについても）

Laravel Dusk


アプリケーションをテストしてユーザーの目を通して見たい場合は、Laravel Duskを試してください。このツールを使用すると、開発者向けのAPIを使用して、ブラウザーからアプリケーションを自動的にテストできます。Laravel DuskにはデフォルトでChromedriverが付属しています。

主な特徴：

  • JDKまたはSeleniumのインストールは必要ありません（ただし、必要に応じて任意のSeleniumドライバーを使用できます）
  • JavaScriptを使用するWebアプリケーションに役立つツールです
  • アプリケーションのさまざまなクリック可能な要素のテストプロセスを簡素化
  • 失敗したテストのスクリーンショットとブラウザコンソール出力を保存するため、問題の原因をすぐに理解できます

このツールはパブリックドメインにあります。Laravel Duskの詳細については、公式ウェブサイトをご覧ください。

Laravel Socialiteは、Facebook、Twitter、Google、LinkedIn、GitHubなどのOAuthプロバイダーを使用した認証のための柔軟なインターフェースを提供するパッケージです。

主な特徴：

  • 使いやすい
  • ソーシャルサービスを使用して認証を実装するために必要なロジックのほとんどすべてが含まれています。ネットワーク

Laravelエコー


Laravel Echoは、フィードを購読し、WebSocketを介してブロードキャストイベントをリッスンできるJavaScriptライブラリです。

主な特徴：

  • アプリケーションにリアルタイムの更新を実装できます
  • あらゆるタイプのチャネル（パブリック、プライベート、プレゼンス）を提供します
  • ささやきの方法を使用してP2Pイベントをブロードキャストする機能を提供します
  • PusherサービスやSocket.IOライブラリなど、人気のあるすぐに使えるソリューションで動作します

Laravel Echoは無料でインストールできます。必要なドキュメントはすべて公式Webサイトにあります。

Laravelメディアライブラリー


Laravel Medialibraryパッケージは、すべてのファイルタイプをEloquentモデルに関連付けます。このパッケージにより、メディアオブジェクトの操作が驚くほど簡単になります。

主な特徴：

  • urlを使用してインターネットからメディアファイルを直接転送できます
  • 独自のディレクトリ構造を使用できます。
  • ファイルの変換を決定できます。画像サイズ、調整、効果など
  • 自動画像最適化を提供します
  • 1つのEloquentモデルに対して複数のメディアコレクションを作成できます

Laravel Medialibraryは、非常に多くの可能性を切り開きます。すべてのインストールドキュメントはここにあります。

Laravelミックス


Laravel Mix（旧称Laravel Elixir）は、ほぼ完全に制御されたフロントエンドビルドプロセスを提供するツールです。Laravelアプリケーション用のWebpackを構築するための基本的な手順を定義するための直感的で便利なAPIを提供します。

主な特徴：

  • ほぼすべてのニーズを満たす幅広いAPIを提供します。
  • Webpackのラッパーのように機能し、それを拡張できます
  • Webpackのインストールと操作に関連するすべての問題を解消します
  • 最新のjavascriptツールとフレームワークで動作します：Vue.JS、React.JS、Preact、TypeScript、Babel、CoffeScript
  • Less、Sass、StylusをCSSファイルに転置してリンクします
  • BrowserSync、ホットリロード、アセットのバージョニング、ソースマッピングを「そのまま」サポート

Laravel Mixは無料でインストールできます。すべてのドキュメントは公式ウェブサイトで見ることができます。

Laravelキャッシャー


Laravel Cashierは、請求サブスクリプションプロセスを大幅に簡略化するパッケージです。Stripe PHPライブラリ自体は優れていると思いますが、直接使用するのははるかに困難です。キャッシャーは潜在的な問題を回避し、サブスクリプション管理を簡素化します。

主な特徴：

  • Stripeサブスクリプション支払いサービスの使用を簡素化
  • 使いやすく、コードベースを理解する
  • Stripe PHP Libraryに明確で直感的なインターフェースを提供
  • クーポン、一時的なサブスクリプション、1回限りの支払いで機能し、PDF請求書などを生成します
  • ヨーロッパにSCAコンプライアンスを提供します

パッケージは完全に無料です。Laravel Cashierと必要なすべてのドキュメントをインストールします。

ララヴェル特使


Laravel Envoyは、簡潔で最小限の構文を備えた非常に便利なタスク起動ツールです。

主な特徴：

  • 導入を簡素化
  • ブレードスタイルの構文を使用
  • Laravelフレームワーク（およびPHP）以外でも使用できます。
  • 設定が非常に簡単
  • 複数のタスクを1つの便利な名前でグループ化する「ストーリー」の機能があります。このようにして、小さな特定のタスクを大きなタスクにグループ化できます。各「ストーリー」は通常のタスクとして実行できます。
  • 複数のサーバーでタスクを実行できます
  • 並列実行が可能
  • SlackとDiscordの通知をサポート（各タスクの完了後に通知を受け取ることができます）

Laravel Envoyは、ここからアクセスできる無料のツールです。

BotMan Studio


BotMan Studioは、チャットボットをより効率的に開発するためのLaravelフレームワーク上に構築されたアプリケーションです。

Laravelテナンシー


Laravel Tenancyは、マルチユーザーLaravelプラットフォームを開発するためのツールです。

主な特徴：

  • 柔軟性を失うことなく、Laravelの迅速なソリューションを提供します
  • プロジェクトの複雑さに関係なく、マルチユーザーSaaSプラットフォームにscaffoldメソッドを使用できます
  • 資産とデータベースを明確に分離します
  • Webサーバーへの閉じたオプションの統合が付属しています
  • 他のプロジェクトで機能を再利用することを好むマーケティング会社に最適です。
  • 構成、コード、ルートなどを追加できます。特定のテナント向け
  • Laravel PermissionsやLaravel Medialibraryなどの一般的なソリューションとの統合チュートリアルを提供します

Laravel Tenancyパッケージは、あらゆるタイプのプロジェクトで無料です。必要な情報はすべてここにあります。

ルーメン

  • マイクロサービスやさまざまなAPIに非常に役立ちます。
  • 完全なフレームワークを使用せずにEloquent ORMおよびその他のLaravelコンポーネントを操作できます
  • ルーティング、キャッシングなどを簡素化します。

Lumen-オープンソースソフトウェア。それに関する必要な情報はすべてここにあります。

ララヴェル望遠鏡


Laravel Telescopeは、Laravelアプリケーションに最適なデバッグアシスタントです。私の意味を理解するために、今まで使用した中で最高のデバッガーが、最も便利なツールを備えた別個のUIになったと想像してください。これがLaravel望遠鏡です。

主な特徴：

  • 開発プロセスを簡素化
  • アプリケーションの多くの側面を追跡およびデバッグするための便利なインターフェースを提供します
  • 開発プロセスの視野を広げ、幅広い情報に直接アクセスできます
  • バグの数を減らし、アプリケーションを改善する方法を理解するのに役立ちます
  • アプリケーションに送信されるリクエストを理解します。すべての例外、データベースクエリ、メール、ログ、キャッシュ操作、通知などを明確に理解できます
  • 必要なすべてのコマンドとキューを実行するプロセスにかかる時間に関する情報を収集します

Laravel Telescopeを無料でインストールします。すべてのドキュメントは公式ウェブサイトで入手できます。

Laravel WebSocketsパッケージ


WebSockets for Laravelは、開発者が待ち望んでいたパッケージです。LaravelでのWebSocketサーバーの実装を可能な限り簡素化します。

主な特徴：

  • WebSocketのサーバー側を完全に想定
  • プッシャーとLaravelエコーサーバーを置き換える
  • ラチェットに基づいていますが、ラチェットを自分で構成する必要はありません
  • リアルタイムデバッグパネルを装備
  • 主要なWebSocketsメトリック（ピーク接続、API経由で送信されたメッセージおよび受信されたメッセージの数）を追跡できるリアルタイムのグラフを提供します
  • マルチユーザーアプリケーションで使用できます。
  • プッシャープロトコルメッセージが付属しています（すでに使用しているすべてのパッケージはLaravel WebSocketsでも動作します）
  • Laravel Echoと完全互換
  • プッシャーのすべての主要機能（プライベートチャネルとプレゼンスチャネル、プッシャーHTTP API）を保持します

Laravel WebSocketsのインストールに関するドキュメントは、こちらにあります。

フレア


Flareは、Laravel開発者が待ち望んでいたバグ追跡ツールです。

主な特徴：

  • さまざまな問題のインスタントソリューションを提供し、これに必要なドキュメントを含みます
  • さまざまな問題を解決するための明確で焦点の合ったインターフェースを提供します
  • ローカルおよび本番環境のエラーを収集します
  • 点火老人エラーがワンクリックでエラーを自動的に修正できるようにします
  • 「例外の共有」機能を使用して共同でバグを効果的に修正できます。
  • バグを修正する時間を短縮できます
  • 例外と通知の追跡を提供します

Flareは7日間の無料トライアルを提供します。次に、あなたにとって便利なプランを選択できます。月額$ 29から$ 279の価格の3つのプランと、年間$ 319から$ 3,069の3つのプランがあります。詳細はこちらからご覧いただけます。
