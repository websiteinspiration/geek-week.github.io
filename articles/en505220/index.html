Tips for choosing a 2G / 3G / 4G / 5G cellular signal booster

An article about options for amplifying a cellular signal at various sites: a summer residence, office, warehouse ..., as well as a brief overview of popular offers on the market.


There are two main options for enhancing cellular communications:


  1. Using an antenna, modem and router.
  2. Using an active amplifier (repeater).

The first option to strengthen the Internet 3G / 4G / 5G, does not enhance voice communication in the usual format for us (but you can use calls through instant messengers: WhatsApp, Viber, Skype, Telegram).


The second option is universal - it strengthens the Internet and voice communication, but has a number of restrictions (about the restrictions below).



Amplification of 3G / 4G Internet using an antenna, modem and Wi-Fi router


The system consists of two nodes: an antenna on the street and a modem + Wi-Fi router indoors.



External antenna for 3G / 4G modem / router



4G modem connected by adapters to a street antenna and inserted into a Wi-Fi router


How it works? The antenna is directed towards the proposed base station of the cellular operator. Then the signal goes to a 4G modem with a router that distributes the Internet by distributing a Wi-Fi signal. The option is popular among private masters to install a suburban Internet. Cheap and works!
