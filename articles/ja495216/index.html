リモートでの学習、交渉、コンサルティング-パート2：Vinteo Server-3か月の無料ビデオ会議

現在世界中で行われているイベントは、必然的にリモート通信業界に影響を与え続けています。何年も前に、ビデオ会議（VKS）は会社の富の表れでした。近年、ビデオコミュニケーション用のソフトウェアソリューションの開発に伴い、高級品のVKSは効果的なビジネスツールのカテゴリに移行しました。しかし現在、検疫の状況により、ビデオ会議システムはリモートでの作業やトレーニングを提供するという強制的な決定になっています。
一方では、この方向性がここ数週間でますます人気が高まっていることを嬉しく思いますが、一方で、これがそのような悲しい状況下で起こったことを嬉しく思いません。

数か月前、私はビデオ会議 ハブの最初で唯一のサブスクライバーでした-そして今では約300人の私たちがいます。

今後数週間のうちに、このハブでシリーズ「トレーニング、交渉、コンサルティング」の記事を公開し、ビデオ会議システムの専門的なソリューションを取り上げ、会社の既存のインフラストラクチャで実際の状況で無料で使用できるように検疫する時間を提供する予定です。前回の記事では、Mind Serverとその無料版について説明しました。この記事は、クラシックなSIP / H.323アーキテクチャに基づいて構築された大規模ビデオ会議ソリューションの唯一のロシアの開発者であるVinteoによるアクションに特化しています。

アクションの本質


使い方は 簡単です。Vinteoサーバーの完全なローカルバージョンを3か月間無料で使用できます。接続数は、サーバーハードウェアのコンピューティングリソースによってのみ制限されます。
従業員のリモート作業を整理する必要がある場合、またはパートナーや顧客と高品質のビデオコミュニケーションを提供する必要がある場合は、プロモーションを使用して、Vinteoプロフェッショナルビデオ会議システムについて理解してください。

パイロットテストのリクエストは、vinteo @ prof-av.ruで行うことができます。

Vinteo Server


Vinteo Serverビデオ会議サーバーは
、ロシア通信省の第165号（2016年4月18日）の命令により、電子コンピューターおよびデータベース用のロシアプログラムの統合登録に含まれています。第142番。

サブスクライバーを接続する可能性


  • ゲートウェイ（トランク）経由。従来のPBXとVKS（Polycom、Ciscoなど）の両方を接続します。単一の番号計画を作成し、会議で固定電話や携帯電話にVKS端末とソフトウェアクライアントに短い番号を呼び出す機能。LDAP経由でサブスクライバーをダウンロードできます
  • SIP / H.323経由。Vinteoサーバーに登録する機能を備えたPCおよびモバイルデバイス上のビデオ会議端末とソフトウェアクライアントの接続。
  • WebRTC経由。オーディオ、ビデオ、コンテンツを送受信する機能を備えたブラウザを介した接続。

サーバーへの接続については、エンコードプロトコル、ビットレート、解像度を個別に設定できます。



レイアウトを操作する


サーバーには、会議用に35の標準レイアウトがあります。接続された参加者の数に適応する自動モザイクを使用することが可能です。


会議全体または任意の参加者に対して、特定のレイアウトを割り当てることができます。講師の位置を指定したり、参加者をレイアウト内の特定の場所に割り当てたり、参加者が画面の後ろにいる場合は、指定した頻度で参加者を画面に表示するようにスクロールパネルを設定したりできます。

SIP / H.323 VKSソリューションのパーソナルレイアウトは、このレベルの多くのソリューションにはないユニークで便利な機能です。


独自のレイアウト設計を開発するために、オプションのコンストラクターが提供されています。3つのオーバーレイレイヤーを使用して、多くのレイアウトオプションを作成できます。保存後に保存されたレイアウトはすべて、会議メニューで使用できるようになります。


会議の録音


Recordは、追加のライセンスを必要としない標準のVinteoサーバー機能です。記録の場合、一般的な会議で使用されるレイアウトとは異なる、参加者のウィンドウの別のレイアウトを選択できます。


さまざまな形式の記録ファイルのエクスポートをサポートします。


会議ブロードキャスト


Webキャストモードは、パーソナライズされたレイアウトの使用とユーザーチャットをサポートします。


ユーザーモードは、ビデオ会議ストリームをサードパーティの放送、録画、またはビデオ監視サーバーに送信するように設計されています。
