高速PACコントローラーWISE-5580


アーキテクチャの開放性に関するPC互換性のあるコンピュータの生産の急速な成長に加えて、産業用コントローラの最新の要件に関連して、後者は産業オートメーションタスクのコントローラとしてますます使用されています。このようなPC互換コントローラはSoftPLCと呼ばれます。この名前は、ハードウェアレベルで解決された従来のPLCのほとんどの機能がソフトウェアで解決できることを示しています。

PC互換コントローラの主な利点は次のとおりです。それらの開放性、拡張性、およびプログラミングツールを選択する機能。

欠点の1つは、コントローラーサイクルの継続時間が長く、その結果、パフォーマンスインジケーターが低いことです。

産業用コンピューターを30年間製造してきたアドバンテックは、2019年に高性能で高速な新しい産業用PC互換コントローラーWISE-5580のリリースを発表しました。

WISE-5580は、既成のハードウェアおよびソフトウェアプラットフォーム（Application Ready Platform）であり、高速データ入出力、このデータの高速処理、およびIoTプロトコルを介した上位レベルとのシステム全体の運用通信のタスクを解決するように設計されています。



高性能の新しいコントローラーには、IntelのCore i7 / i5 / Celeronプロセッサーと、リアルタイム機能とミニPCI Expressを備えた高速内部EtherCATネットワークバスが搭載されています。

ITとACSの架け橋


WISE-5580は、最新の管理プラットフォームの要件を満たすために開発され、オフィス/ ITスペシャリストの両方がトップレベルの問題を解決し、ACSスペシャリストがCoDeSysエグゼクティブシステムを使用して自動化の問題を解決できるようにします。



コントローラーは、一方ではI / Oモジュールによって、最高速かつ最も確定的なリアルタイムバスのEtherCATを介して拡張でき、他方では、ミニPCI Expressバスを介した通信モジュールによって拡張できます。

一般仕様：


  • Intel Core i7 / i5 / Celeronプロセッサー（最大2.6 GHz）
  • OS Windows 7/10
  • アラート機能付き電源バックアップ
  • 安全値I / O
  • 絶縁付きのRS-232 / 422/485シリアルポート2個
  • POEをサポートする2つのギガビットイーサネットポート
  • 4 USB 3.0ポート
  • 3G/LTE/Wi-fi/GPS
  • VGA HDMI
  • Codesys RTE
  • 61131-3
  • Modbus, EtherNet/IP, PROFINET, OPC/UA MQTT
  • : 139x100x80




I / Oモジュールを使用すると、さまざまなアナログおよびディスクリートセンサー、サーミスタ、熱電対、および制御アクチュエータからコントローラー信号を開始できます。

デジタルビデオカメラをUSB 3.0を備えた通信拡張モジュールとPoEを備えたギガビットイーサネットポートに接続し、受信した画像を分析した後、EtherCATモジュールを介してサーボに制御信号を送信できます。WISE-5580を使用するこのようなシナリオでは、ビジョンシステムを実装することができます。



1- ビジョンシステムの統合用WISE-5580のヘッドパーツ
2- WISE-5424V、4つのUSB 3.0ポートを備えた拡張モジュール
3- WISE-5001、4つのデジタル入力を備えたパワーモジュール
4- WISE-5079、拡張モジュールEtherCAT
5- WISE-5074、EtherCATバスカプラー
6-I / Oモジュール

オプションのEtherCATバスカプラー（WISE-5074）を使用すると、システム全体を2つの異なるDINに分割できます。レール、またはモジュールの一部を別のキャビネットに取り付けます。

無線インターフェース


Wi-Fi、3G、LTE、GPSなどのワイヤレス通信チャネルを介してコントローラーと通信するために、WISE-5400E通信モジュールは、ミニPCIeカードを取り付けることができるPCI Express バスの側面から接続され、SIMカード用のスロットがあります。



モジュールの上部カバーには、SMAアンテナコネクタ用の開口部があります。

ソフトウェア




WISE-5580の 既製のソフトウェアソリューションとして、CODESYS計測ソフトウェアパッケージが使用されます。これにより、コントローラーをIEC 61131-3に従ってプログラムすることができます。EtherCATバスサポートのおかげで、CODESYSはマイクロ秒プロセスを制御する機能を提供し、その結果、WISE-5580は最適化されたパフォーマンスのPAC（プログラム可能なオートメーションコントローラー）になることができます。

CODESYSには、さまざまなアプリケーションシナリオに対応する3レベルの実行システム（制御ランタイムシステム）と、データ視覚化機能が組み込まれています。
LabVIEWソフトウェアプラットフォームを介してシステム全体のステータスを監視することも可能です。DAQNaviソフトウェアパッケージの既製のドライバーとライブラリはソフトウェア開発を簡素化します。

組み込みのAdvantech Lmsensorドライバーは、コントローラー自体のパフォーマンスを最大化する機能を提供し、さまざまなWindowsシステムプラットフォームでMicrosoft Visual C ++、Embedded Visual C ++などのプログラミング言語などのツールを使用して開発されたプログラムでユニバーサルLmsensor操作を簡単に実行できるようにします。
