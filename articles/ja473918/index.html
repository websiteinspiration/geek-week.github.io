ロシアを旅するインターネット：スピードと機会

2019年春、据え置きで使えるLTEルーターのテストを行いました。これは、民家の居住者、またはワイヤレスよりも有線ネットワークアクセスのコストが高い人にとっては、かなり興味深いテストであることがわかりました。
今回は、移動中の携帯電話会社の機能を確認することにしました。別の地域に移動して、特定の電気通信会社のアクセス速度をテストしました。伝統的に、ビデオはビデオ愛好家のために作られました、そして残りのために写真付きのテキストがあります。


今回はモバイルインターネットを提供するWifireオペレーターのSIMカードを使うことにしました。さらに、1つの有名なオペレーターとは異なり、Wifire料金は4G（LTE）ネットワークだけでなく3Gでも有効です。これは、3G信号のカバレッジがより多く、接続が優れているため、信号レベルがそれほど高くないネットワークにとって重要です。そしてもちろん、ネットワークへのアクセス速度とローミングの欠如をテストすることは興味深いことだったので、少し休憩を取り、国のいくつかのエリアをドライブしました。



使いやすさとネットワークアクセスのスピードを存分に楽しむために、スマートフォンHuawei P30 Proを使用しました。これはHuaweiのスマートフォン掘り出しラインのトップエンドのスマートフォンであり、本当に優れた機能を備えています。このスマートフォンがLTEチャネルの集約をサポートし、そのようなオペレーターが提供できる場合、理論的には最大400 Mbpsの着信データ転送速度を提供できることが私にとって特に重要でした。チャネルの集約は、加入者へのデータ送信のために基地局で一度にいくつかのチャネルを結合したものです。通常、スマートフォンでは、このモードの操作は4G +として表示されます。

テスト方法


新しい場所ごとに、ほぼ同時に、午後にテストが実施されました。基地局に朝と夕方のピーク負荷がなく、データ伝送速度で実際のデータを受信することができたこと。送信速度は大量のデータを送信する場合にのみ重要であり、ビデオ会議またはストリーミング中の送信データストリームは3〜5 Mbpsに過ぎないため、私は受信速度にもっと興味を持っていました。テストは、人気のサービスSpeedtest.netを使用して実施されました。測定は数回行われ、中央値が取られました。過大評価または過小評価されたピークがある場合、それらは考慮されませんでした。スマートフォンは自動モードで動作しましたが、ほとんどすべての測定でLTE接続があり、1か所にのみ3G接続しかありませんでしたが、都市からは遠く離れていました。

なぜこれが必要なのですか？


ワイヤレス通信のニュアンスはテスト結果が私だけに適しているようなものなので、プロバイダーの比較テストを行うのはばかげています。場合によっては、100メートルを任意の方向に動かすだけで十分であり、さまざまなオペレーターの測定結果が劇的に変化します-ランドスケープ、基地局の負荷、およびクライアント機器の機能の両方がここに影響します。特定のプロバイダーから、仕事やレジャーの一環として頻繁に移動し、モバイルインターネットを使用するユーザーにモバイルインターネットを使用するのは、どれほど便利で快適なのかと思いました。道に行こう！



旅の最初のポイントは美しいニジニ・ノヴゴロドで、低温と澄んだ空に出会いました。街は当然のことながら人気のある観光ルートの1つに含まれています。風がほとんど打ち倒されたが、私はまだケーブルカーに乗った-それは面白かった。それから私は観光客や地元の人々に人気の場所-チカロフのはしごに来ました。正午頃、人は少なかったが、歩く人は十分だった。つまり、基地局の負荷は標準だった。速度の測定値も上方に変動しましたが、この結果を主な結果として数えました-51.9 / 26.0 Mbit / s（DL / UL）。人気のある訪問した場所のまともなインジケーター。



次のポイントは非常に興味深いスヴェトロイアル湖でした。ニジニノヴゴロド地域にあり、伝説によると、キテズの街がここに隠れています。地図を見ると、この湖はほぼ完全に丸い形をしているため、湖の起源のいくつかのバージョンがあります。氷河カルスト形成、構造断層、および隕石の落下によるものです。



巡礼者にも観光客にも人気の場所。場所はかなり訪問されているという事実にもかかわらず、観光インフラでは非常によくありません。実際、これらの言葉はロシアの多くの人気のあるルートについて言えるでしょう。ただし、ネットワークへのアクセスは、滞在場所に関する歴史的参照のためだけである必要があります。湖は高い森に囲まれていて、海岸では3G信号しか捉えられていません。そのような瞬間、地形の影響は特によく感じられます。ただし、この場所の速度は12.8 / 2.19 Mbit / sでした。これは、メッセンジャーでのビデオ通話や静かなサーフィンにも十分です。 YouTubeの動画もフリーズしています。



さらに、私の道はヴォルガ川の上流からトヴェリ市まで続いていました。多くの人がこの都市を故人のシャンソン歌手と関連付けていますが、私にとって、この都市は、まず第一に、インドに到達したアタナシウスニキティンの発祥の地です。彼の日記「三海を歩く」は、ロシアとインドの間の商品経済的および政治的相互作用の基礎となった。



2つの首都の間の有利な位置と首都へのより緊密な領土関係にもかかわらず、トヴェリは、特にニジニノヴゴロドの後で、私にむしろ憂鬱な印象を与えませんでした。皇后キャサリンの旅行の宮殿を復元する傾向は非常に満足していますが。もちろん、旅行中は写真をシェアしたり、話したり、時には仕事をしなくてはならないので、コミュニケーションはとても大切です。ボルガ川の堤防で速度測定をすることにしました。それは平日、労働時間だったので、基地局の負荷は典型的でした。負荷のピークは朝と夕方に観測されます。全員がモバイルデバイスからオンラインにアクセスしてニュースを読んだり、ソーシャルネットワークを閲覧したり、その日のイベントを共有したりします。接続は4Gで行われていて、人気の場所だったので、近くに基地局があり、信号レベルが優れていたからです。測定結果：60.6 / 40.8 Mbps。仕事にも娯楽にも非常に価値のある結果です。原則として、このような速度で、オンラインで4Kでビデオを視聴したり、FullHDで高ビットレートでブロードキャストしたりできます。



それから首都に向かい、途中で渋滞を集めました。直接通信速度を計測することにしました。そして、ステージの1つで、時速100 km強のスピードで、Speedtestユーティリティを起動しました。結果は非常に興味深いです：12.3 / 4.9 Mbps。車の中でこのような速度で、安全にオンライン映画を見たり、ビデオを介して通信したりできます。スマートフォンは新しいタワーに固執することが多いため、原則として移動中に速度が変動します。そのため、信号レベルとチャネル負荷の状況はまったく異なる場合があります。途中、ダウンロードで100Mbit / s以上の結果が出ましたが、信号機のある交差点で止まっていました。



私のモバイルインターネットの旅とテストは終わりに近づいており、Zaryadye Parkが最後のポイントになります。美しい景色、優れたインフラストラクチャ、いつでも多数の観光客。何百、何千もの写真がこの場所から世界中のすべての国に常に送信されるため、後者はテストをかなり複雑にするはずです。人々は、他の都市だけでなく、国からの対談者ともコミュニケーションをとります。ここでの測定結果はどうなるのでしょうか？そして、結果は予測可能ですが、いくらか驚くべきものです：58.4 / 3.1 Mbit / s。予想通り、ユーザーはほとんどここから写真を送信し、それによって発信データチャネルをダウンロードします。しかし、この速度でもビデオ通話には十分です。まあ、ダウンロード速度は、オンライン映画館を見ることさえも十分です。



結論


異なる地域でのデータの送受信に関するすべてのデータを表にまとめます。着信チャネルの速度が10メガビット/秒を下回らない場所はないことに注意してください。つまり、途中でデータを受信したり、ドキュメントを操作したりするのは非常に快適です。通信の問題を解決するために、テクニカルサポートに問い合わせる必要がありませんでした。 SIMカードのアクティベーションの数分後にインターネットアクセスが出現し、地域の変更はネットワークへのアクセスの可能性に影響を与えませんでした。 Wifireプロバイダーは、音声とSMSを提供せずに、2G / 3G / 4Gチャネルを介したセルラーネットワーク経由のアクセスを提供します。つまり、ネットワークへのアクセスは、信号が存在する場合、どのタイプの接続でも常に行われます。



関税、つまり1か月分のパッケージについては、毎月新しいトラフィックパッケージが更新されます。トラフィックを購入し、時間制限なしでそれを過ごすことは可能です。これは、ネットワークに頻繁にアクセスする必要のないあらゆる種類のスマートデバイスに適しています。たとえば、タブレット、モデム、またはあらゆる種類のスマートデバイスで、利用可能なWi-Fiネットワークの外部でガジェットをほとんど使用せず、インターネットアクセスが緊急に必要な場合。
