情報処理


テキストから改行文字を削除します。

def delete_new_line_symbols(text):
    text = text.replace('\n', ' ')
    return text

df['comment'] = df['comment'].apply(delete_new_line_symbols)
df.head()

コメントには実際のデータ型があり、それらを整数に変換する必要があります。次に、別の変数に保存します。

target = np.array(df['toxic'].astype('uint8'))
target[:5]

次に、Tokenizerクラスを使用してテキストを少し処理します。それのコピーを書きましょう。

tokenizer = Tokenizer(num_words=30000, filters='!"#$%&()*+,-./:;<=>?@[\\]^_`{|}~\t\n', 
                      lower=True, 
                      split=' ', 
                      char_level=False)

パラメータについてすばやく

  • num_words-記録された単語の数（最も一般的）

  • フィルター-削除する文字のシーケンス

  • lower-テキストを小文字にするかどうかを制御するブールパラメータ

  • split-文を分割するための主な記号

  • char_level-単一の文字が単語と見なされるかどうかを示します

次に、クラスを使用してテキストを処理します。

tokenizer.fit_on_texts(df['comment'])
matrix = tokenizer.texts_to_matrix(df['comment'], mode='count')
matrix.shape

サンプル行は14k、フィーチャ列は30kです。



私は2つのレイヤーからモデルを構築しています：密とドロップアウト。

def get_model():
    
    model = Sequential()
    
    model.add(Dense(32, activation='relu'))
    model.add(Dropout(0.3))
    model.add(Dense(16, activation='relu'))
    model.add(Dropout(0.3))
    model.add(Dense(16, activation='relu'))
    model.add(Dense(1, activation='sigmoid'))
    
    model.compile(optimizer=RMSprop(lr=0.0001), 
                  loss='binary_crossentropy',
                  metrics=['accuracy'])
    
    return model

マトリックスを正規化し、合意されたとおりにデータを2つの部分に分割します（トレーニングとテスト）。

X = normalize(matrix)
