12 Mata Kuliah Rekayasa Data Online


Menurut Statista, pada tahun 2025, pasar data besar akan tumbuh menjadi 175 zetabytes dibandingkan dengan 41 pada 2019 ( grafik ). Untuk mendapatkan pekerjaan di area ini, Anda perlu memahami cara bekerja dengan data besar yang disimpan di cloud. Cloud4Y telah menyusun daftar 12 program rekayasa data berbayar dan gratis yang akan memperluas pengetahuan Anda di bidang ini dan dapat menjadi titik awal yang baik untuk jalur sertifikasi cloud.

Kata Pengantar

Apa itu insinyur data? Ini adalah orang yang bertanggung jawab untuk membuat dan memelihara arsitektur data dalam proyek Ilmu Data. Tanggung jawabnya mungkin termasuk memastikan aliran data yang tidak terputus antara server dan aplikasi, mengintegrasikan perangkat lunak baru untuk manajemen data, meningkatkan proses data yang mendasarinya dan membuat jalur pipa data.

Ada sejumlah besar teknologi dan alat yang harus dimiliki oleh seorang insinyur data untuk bekerja dengan komputasi awan, pergudangan data, ETL (ekstraksi, transformasi, pemuatan), dll. Selain itu, jumlah keterampilan yang dibutuhkan terus bertambah, sehingga seorang insinyur data perlu secara teratur mengisi kembali kopernya. pengetahuan. Daftar kami memiliki kursus untuk pemula dan profesional berpengalaman. Pilih apa yang tepat untuk Anda.

1. Rekayasa Data Sertifikasi Nanodegree ( Udacity )

Anda akan belajar bagaimana merancang model data, membuat gudang data dan danau data, mengotomatisasi jalur pipa data dan bekerja dengan array dataset. Di akhir program, Anda menguji keterampilan baru Anda dengan menyelesaikan proyek Capstone.

Durasi : 5 bulan, 5 jam seminggu
Bahasa : Inggris
Harga : $ 1695
Level : Pemula

2. Menjadi Sertifikasi Insinyur Data ( Coursera )

Belajar dari dasar-dasarnya. Anda dapat berkembang secara bertahap menggunakan ceramah dan proyek praktis untuk mengerjakan keterampilan Anda. Pada akhir pelatihan, Anda akan siap bekerja dengan ML dan data besar. Disarankan untuk mengetahui Python setidaknya pada level minimum.

Durasi : 8 bulan, 10 jam seminggu
Bahasa : Inggris
Harga :?
Level : Pemula

3. Menjadi Insinyur Data: Menguasai Konsep ( Pembelajaran LinkedIn )

Anda akan mengembangkan keterampilan desain data dan keterampilan DevOps, mempelajari cara membuat aplikasi Big Data, membuat jalur pipa data, memproses aplikasi secara real time menggunakan database Hazelcast dan Hadoop .

Durasi: tergantung pada Anda
Bahasa : Inggris
Harga : bulan pertama - gratis
Level : Pemula

4. Kursus Rekayasa Data ( edX )

Berikut adalah serangkaian program yang memperkenalkan Anda pada rekayasa data dan mengajarkan Anda cara mengembangkan solusi analitis. Kursus dikategorikan berdasarkan tingkat kesulitan, sehingga Anda dapat memilih satu sesuai dengan tingkat pengalaman Anda. Selama proses pelatihan, Anda akan belajar cara menggunakan Spark, Hadoop, Azure, dan mengelola data perusahaan.

Durasi : tergantung pada Anda
Bahasa : Inggris
Harga : tergantung pada program studi yang dipilih
Level : dasar, menengah, lanjutan

5. Data Engineer ( DataQuest )

Kursus ini layak dipilih jika Anda memiliki pengalaman dengan Python dan ingin memperdalam pengetahuan Anda dan membangun karier sebagai spesialis pemrosesan data. Anda akan belajar bagaimana membangun jalur pipa data menggunakan Python dan panda, memuat kumpulan data besar ke dalam basis data Postgres setelah membersihkan, mengubah, dan memvalidasi.

Durasi : tergantung pada Anda
Bahasa : Bahasa Inggris
Harga : tergantung pada bentuk langganan
Level : pemula, menengah

6. Rekayasa Data dengan Google Cloud ( Coursera )

Kursus ini akan membantu Anda memperoleh keterampilan yang Anda butuhkan untuk membangun karier di Big Data. Misalnya, bekerja dengan BigQuery, Spark. Anda akan mendapatkan pengetahuan yang Anda butuhkan untuk mempersiapkan sertifikasi Google Cloud Professional Data Engineer yang diakui industri.

Durasi : 4 bulan
Bahasa : Inggris
Harga : sementara gratis
Level : pemula, menengah

7. Rekayasa Data, Big Data di Google Cloud Platform ( Coursera )

Kursus yang menarik yang memberikan pengetahuan praktis tentang sistem pemrosesan data dalam GCP. Selama pelajaran, Anda akan belajar bagaimana merancang sistem sebelum Anda memulai proses pengembangan. Selain itu, Anda juga akan menganalisis data terstruktur dan tidak terstruktur, menerapkan penskalaan otomatis, dan menerapkan metode ML untuk mengekstraksi informasi.

Durasi : 3 bulan
Bahasa : Inggris
Harga : sementara gratis
Level : Beginner, Intermediate

8. UC San Diego: Big Data Spesialisasi ( Coursera )

Kursus ini didasarkan pada penggunaan infrastruktur Hadoop dan Spark dan penerapan metode pemrosesan data besar ini dalam proses ML. Anda akan mempelajari dasar-dasar penggunaan Hadoop dengan MapReduce, Spark, Pig, dan Hive. Pelajari cara membuat model prediksi dan menggunakan analisis grafik untuk memodelkan masalah. Perhatikan bahwa kursus ini tidak memerlukan pengalaman pemrograman.

Durasi : 8 bulan, 10 jam seminggu
Bahasa : Inggris
Harga : sementara gratis
Level : awal

9. Menjinakkan Big Data Dengan Apache Spark dan Python ( Udemy )

Anda akan belajar cara menggunakan struktur aliran dan bingkai data di Spark3, dapatkan gagasan tentang cara menggunakan layanan Elastic MapReduce Amazon untuk bekerja dengan kluster Anda di Hadoop. Pelajari cara mengidentifikasi masalah saat menganalisis data besar dan memahami cara kerja pustaka GraphX ​​dengan analisis jaringan dan bagaimana Anda dapat menggunakan MLlib.

Durasi : tergantung pada Anda
Bahasa : Inggris
Harga : dari 800 rubel ke $ 149,99 (betapa beruntungnya)
Level : Pemula, menengah

10. Program PG di Big Data Engineering ( upGrad )

Kursus ini akan memberi Anda gambaran tentang bagaimana Aadhaar bekerja, bagaimana Facebook mempersonalisasi umpan, dan bagaimana Rekayasa Data dapat digunakan. Topik utama adalah pemrosesan data (termasuk pemrosesan waktu nyata), MapReduce, analitik data besar.

Durasi : 11 bulan
Bahasa : Inggris
Harga : sekitar $ 3000
Level : SD

11. Data Scientist ( Skillbox ) profesi

Anda akan belajar cara memprogram dalam Python, mempelajari kerangka kerja untuk melatih jaringan saraf Tensorflow dan Keras. Kuasai database MongoDB, PostgreSQL, SQLite3, pelajari cara bekerja dengan pustaka Pandas, NumPy, dan Matpotlib.

Durasi : 300 jam pelatihan
Bahasa : Rusia
Harga : setengah tahun pertama gratis, lalu 3900 rubel per bulan
Level : Pemula

12. Insinyur Data 7.0 ( Lab Profesi Baru )

Anda akan menemukan studi mendalam tentang Kafka, HDFS, ClickHouse, Spark, Aliran Udara, lambda-arsitektur dan kappa-architecture. Anda akan belajar cara menghubungkan alat satu sama lain, membentuk jaringan pipa, mendapatkan solusi dasar. Belajar membutuhkan pengetahuan minimum tentang Python 3.

Durasi : 21 pelajaran, 7 minggu
Bahasa : Rusia
Harga : 60.000 rubel
Level : menengah
