9 dicas de terminal do Windows por Scott Hanselman

Olá Habr! Você deve ter ouvido falar que o novo Terminal do Windows está chegando em breve. Já escrevemos sobre isso aqui . O nosso colega Scott Hanselman preparou algumas dicas sobre como trabalhar com o novo terminal. Entrar!



Então, você baixou o Windows Terminal e ... e agora?


No começo você pode não estar emocionado. Este ainda é o Terminal e não o levará, segurando a mão.

1) Leia a documentação do usuário para o Windows Terminal

2) As configurações são expressas no formato JSON . Você fará melhor se o seu editor de arquivos JSON for algo como o Visual Studio Code e suportar JSON e também intellisense.

  • Verifique suas configurações padrão! Para maior clareza, apresento meu profile.json (que não é o ideal). Instalei o requestTheme, alwaysShowTabs e defaultProfile.

3) Decida sobre os atalhos do teclado. O Terminal do Windows possui amplas opções de personalização .

  • Qualquer tecla que você pressionar pode ser reatribuída.

4) O design combina com seus desejos?

  • Há vários terminais temas sobre terminalsplash.com.
  • E aqui estão dezenas de esquemas de cores do iTerm2 , seus esquemas de cores podem ser vistos claramente aqui .

5) Deseja ir para o próximo nível? Explore imagens de fundo.

  • Você pode definir imagens de fundo ou até GIFs. Mais detalhes aqui .

6) Especifique seu diretório de inicialização.
  • Se você usa o WSL, mais cedo ou mais tarde, o diretório inicial deve estar localizado no sistema de arquivos Linux .

7) Você ainda pode usar Far, GitBash, Cygwin ou cmder, se desejar. Detalhes na documentação .

8) Examine os argumentos da linha de comando do Terminal do Windows.

  • Você pode saber que pode iniciar o Terminal do Windows usando o "wt.exe", mas agora pode usar argumentos de linha de comando! aqui estão alguns exemplos:

    wt ; split-pane -p "Windows PowerShell" ; split-pane -H wsl.exe
wt -d .
wt -d c:\github

    Neste ponto, você pode ir o mais longe que desejar. Faça outros ícones, fixe-os na barra de tarefas e desça ao máximo. Além disso, confira subcomandos como nova guia, painel dividido e guia de foco.

9) Gravei um vídeo demonstrando para uma pessoa que está acostumada com Mac e Linux como configurar o Terminal do Windows em conjunto com o WSL (Windows Subsystem para Linux), talvez seja interessante para você.
