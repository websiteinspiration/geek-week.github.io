Nvidia Jetson組み込み機械学習システム



機械学習ハードウェアプラットフォームは急速に成長し、価格も安くなっています。Nvidia Jetsonモジュールを使用すると、効率的で手頃な価格のエッジコンピューティングソリューションを作成できます。今日では、256個のNvidia Cudaグラフィックコアを備えた高性能システムを手のひらサイズのコンピューターに組み込むことが可能になりました。

この記事では、エッジコンピューティングとは何かを分析し、Nvidia Jetsonモジュールについて話し、それらに基づいて開発できたソリューションを示します。

エッジコンピューティング



エッジコンピューティングの概念には、コンピューティングパワーを可能な限り最終機器に近づけることでリソースを節約することが含まれます。

-------------------------------------------------- -------------------------------------------------- ---------------------

実際のプロジェクトの1つの例を考えてみましょう。顧客は、離れた場所にあるゲートに近づいている車の数を認識する必要があります。高解像度デジタルカメラがゲートの上に設置されています。オブジェクトとの唯一の接続は、高価なモバイルインターネットチャネルです。以前は、顧客はビデオ上の動きを処理し、認識のためにカメラからサーバーにフレームを送信する必要がありました。送信されたフレームのほとんどには、人、動物、気象現象などの誤検知が含まれていました。これにより、モバイルトラフィックの消費が増加しました。手頃なソリューションの登場により、エッジコンピューティングは、機械学習を使用して動いている車を他の現象と区別し、さらにその数を認識することで、施設でビデオストリームをローカルに処理できるようになりました。その結果、高価なインターネットチャネルを介してサーバーにデータを送信することは、車の番号を含むテキスト文字列の送信に限定されていました。これにより、モバイルトラフィックのコストを削減できる場合があります。

Nvidia Jetsonとは何ですか？




Nvidia Jetsonは、SoM（System On Module）フォームファクターの組み込みコンピューティングモジュールのファミリーであり、コンパクトでエネルギー効率の高い機械学習システムの作成に重点を置いています。Nvidia Jetsonモジュールは、プロセッサー、ビデオコア、RAM、USBコントローラーなど、本格的なコンピューターのすべてのコンポーネントを搭載したコンパクトなボードです。これらは、特定のタスク用に設計された他のボード（キャリアボード）に埋め込まれるように設計されています。

特定のソリューションのメーカーは、周辺機器用のキャリアボードを開発し、既製のコンピューティングモジュールを挿入するだけでよいため、SoMを使用すると、組み込みシステムの開発が大幅に簡素化されます。これにより、複雑なマザーボードの開発コストを削減し、ビルド品質と追加オプションに集中できます。また、最終的なデバイスの準備が整っていない状態でも、評価キットの形式で同じSoMモジュールを使用できるため、開発者にとっても簡単です。その結果、ソフトウェア開発者は予測可能なハードウェア環境を取得し、プログラムを最終的なデバイスに転送したときに、まったく同じパフォーマンスが得られると確信できます。これは、結果がアイロンの特性に大きく依存する機械学習システムを開発する場合に特に重要です。

Jetson nano




Nvidia Jetson Nanoは、カメラ、DVR、ロボット、インタラクティブ端末、家電などのエンドデバイスに組み込まれるように設計された、Jetsonシリーズの最年少モジュールです。ラップトップ用のRAMカードとしてSO-DIMMフォームファクターで設計されています。

特徴


プロセッサ：クアッドコアARM Cortex-A57 MPCore @ 1.4 GHz
GPU：128 CUDAコアを備えたMaxwell
ハードウェアビデオコーデック： 4K @ 30fpsエンコーディング（H.264 / H.265コーデック）および4K @ 60fpsデコード
ビデオ出力： HDMI 2.0またはDP1.2 | eDP 1.4 | DSI（1×2）、2つの同時
RAM： 4 GB LPDDR4 64ビット。 25.6 GB / s
フラッシュメモリ： 16 GB eMMC（オペレーティングシステム用）
インターフェイス： 1×1 /2/ 4 PCIE、1×USB 3.0、3×USB 2.0、ギガビットイーサネット
I / O： 1×SDIO / 2 ×SPI / 6×I2C / 2×I2S / GPIOピン
オペレーティングシステム： Linux
寸法： 69.5×45 mm
接続：260ピンSO-DIMM DDR4コネクタ

Jetson Nanoモジュールは低コストであるため、多くの開発者がなんとかそれを実際に試してみました。ハブでも、マシンビジョンとパターン認識にそれを使用することについてのいくつかの記事があります。JetsonNanoデモ-猫の認識。これは、機械学習システムをデプロイするための最も手頃な価格の一般的なモジュールです。

Nvidia Jetson Nano DVR MIC-710IVA



オールインワンのインテリジェントビデオ監視ソリューションであるNvidia Jetson Nano をベースにしたMIC-710IVAデジタル8チャネルDVR を紹介します。複数のデバイスの複雑な複合体を構築することなく、マシンビジョンシステムを統合できます。

MIC-710IVAの組み合わせ：

  • ストレージシステム付きデジタルビデオレコーダー
  • リアルタイムビデオ処理用の128 CUDAコアのコンピューティングモジュール
  • カメラを直接接続するための8ポートPoEスイッチ
  • リレー、リードスイッチ、アラームなどを制御するためのデジタル入力/出力ブロック。


機械学習は、すぐにビデオ監視のトレンドになっています。たとえば、人、車、自然現象の異常な行動の検出、事故の検出、オブジェクトのワークロードの分析など、人が以前にやらなければならなかったことを自動化することができます。あなたはもはやディスパッチャーの注意力に依存する必要はありません。このようなシステムは、小売、セキュリティ複合施設、輸送ハブで需要があります。

技術仕様MIC-710IVA


  • NVIDIA Jetson Nano上に構築
  • ビデオカメラ用の8つのPoEポート
  • 2つの3.5インチHDDをインストールする機能
  • Linux OS
  • 低消費電力
  • H.264 / H.265ビデオコーデックのサポート
  • RS-485インターフェースと8ビットデジタル入力/出力



MIC-710IVAビデオレコーダーのリアパネルMIC-710IVAビデオ

レコーダーを使用すると、ストリーム全体をリモートデータセンターにブロードキャストする必要なく、ビデオデータの初期処理をリアルタイムで実行できます。これにより、ネットワークの負荷が軽減され、機密データを第三者に転送する必要がなくなります。

MIC-720AIは、完全にパッシブな冷却機能を備えたJetson TX2プラットフォームに基づいており、マシンビジョンの産業システム、製造、および移動物体への設置用に設計されています。ファンレス設計は、完全なノイズレス動作を提供し、メンテナンスを必要とせずにほこりっぽい部屋でコンピューターを使用できるようにします。
