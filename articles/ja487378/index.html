2020年のベストスモールビジネスサーバー



クラウドサーバーが広く採用されているにもかかわらず、多くの人がより多くの制御を提供するオンプレミスサーバーを選択しています。TechRadarポータルは、小規模ビジネス向けに設計されたものから大企業が必要とする可能性のあるものまで、上位10台のサーバーのリストをまとめました。

1. Dell PowerEdge T30



多くの接続

プロセッサを搭載したXeonの夢：Intel Xeon E3-1225 v5 |グラフィック：Intel HD Graphics P530 | RAM：64GB |ストレージ：最大6つのSATA HDD |接続インターフェイス：10 x USBポート、HDMI、DisplayPort、VGA、シリアルポート、PS2、ギガビットイーサネット|寸法（幅x長さx高さ）：17.5cm x 43.5cm x 36cm

+高速クアッドコアXeonプロセッサー
+ 6台のハードドライブを搭載 -1

ギガビットネットワークポート
-ホットスワップドライブなし

周りには非常に多くのさまざまなサーバーがあり、自分に合った最適な小型サーバーを見つけるために、価格表全体を調査する必要がある場合があります。Dell PowerEdge T30をご覧ください。自己組み立ておよび構成用のバージョン、または完全に完成した形で購入できます。エントリーレベルの小規模オフィス市場をターゲットとするT30は、小さなタワー型の建物に拡張できる大きな可能性を秘めています。また、デフォルトで多くのサーバー機能を備えているため、オフィスワークステーションの理想的な代替品となります。

2. Dell PowerEdge T20



アセンブリソリューションがどれほど安価であるかを示します

プロセッサー：Intel Pentium G3220 |グラフィック：Intel HD Graphics | RAM：4GB |ストレージ：含まれていません|接続インターフェイス：最大USBポート（4 x USB 3.0）、2 x DisplayPort、VGA、シリアル、2 x PS2、ギガビットイーサネット|寸法（幅x長さx高さ）：17.5cm x 43.5cm x 36cm

+非常に安価
+コンパクトなミニタワーで内部部品に簡単にアクセス可能

-G3220はサーバープロセッサではなくデスクトップに近い
-ドライブとOSなし

サーバーは高価で大規模である必要はありません。 PowerEdge T20をご覧ください。これは、20年間のサーバー開発に関するデルの知識の最高のものです。ビルドバージョンにはハードドライブがありませんが、確かに安価です-それほど昔ではありませんが、コストは$ 130未満です（キャッシュバックを考慮すると、これらのマシンに常に注意する必要があります）。

このサーバーには、3 GHzに高速化され、最大32 GBのDDR3 ECC RAMをサポートするHaswellマイクロアーキテクチャーに基づくプロセッサーがあります（このモデルには4 GBが付属していることに注意してください）。拡張オプションには、4つのSATAポート（8 TBのハードドライブを取り付けた場合は32テラバイト）、4つのI / Oスロット、および10のUSBポートが含まれます。驚いたことに、この価格のコンピュータには2つのDisplayPortソケット、1つのVGA、2つのPS2、1つのシリアルポートがあります。ギガビットイーサネットポートに加えて、290ワットの電源とIntelのRAIDコントローラーも重要です。

3. Lenovo ThinkServer TS150



高性能タワーサーバー

プロセッサ：Intel Xeon E3-1200 v6 | RAM：最大64GB |ストレージ：最大40TBのHDD |接続インターフェイス：8 x USB 3.0、シリアル、ビデオ、2 x DisplayPort、オーディオ、ギガビットイーサネット|寸法（幅x長さx高さ）：17.5 cm x 37.5 cm x 43 cm

+ ThinkServerシリーズで最も手頃なモデル
+完全に静か

-不満はありません

Lenovoは2014年にIBMからx86サーバー市場を獲得し、最高のラインナップの1つであるThinkServerを作成しました。 TS150は、この範囲で最も手頃なモデルの1つです。これは、Dell T20に匹敵するエンタープライズクラスの4Uサーバーです。 RAID 0、1、10、および5をサポートしています（マザーボード上のコントローラーを介して）。競合他社と同様に、このサーバーは最大4台の3.5インチハードドライブに対応できます。つまり、対応するハードドライブを取り付けると、最大40 TBのストレージを持つことができます。比較的新しいIntel XeonプロセッサE3-1200 v6は、中小企業にとって十分な生産性を備えているはずです。

レノボはまた、TS150の作業音はライブラリのノイズレベルよりも静かで、約26 dBであると主張しています。他のサーバーと同様に、ポートと接続インターフェイスの印象的な配列が得られます。8つのUSBポート、4つのPCI / PCI-Eスロット、3つのビデオ出力（2つのDisplayPortスロットを含む）、シリアルポート、ギガビットイーサネット、および3つのコネクターオーディオ。

強力で用途の広い

プロセッサー：Intel Xeon Scalable 4210 | グラフィック：Intel HD Graphics P530 | RAM：16GB | ストレージ：ドライブは含まれません| 寸法（幅x長さx高さ）：18 cm x 65 cm x 47 cm

+スケーラブル
+まともなデザイン

-巨大なマシン
-ハードドライブなし

信頼性が高く用途の広いサーバーが必要な場合は、HPE ProLiant ML350 Gen 10が最適です。このモデルは、インテルXeonスケーラブルプロセッサーがインストールされているため、高いパフォーマンスを提供します。自分でストレージをインストールする必要がありますが、このサーバーは、グラフィックスやコンピューティングタスクの操作を含む幅広い高速操作をサポートしています。HPE ProLiant ML350 Gen 10はタワーサーバーとして提供されますが、ビジネスの拡大に伴い、ラックサーバーに変換することができます。これにより、追加コストなしで完全に新しいサーバーを購入する必要なくビジネスを拡大できます。

5. Fujitsu Primergy TX1310 M1



富士通の信頼性が保証された

比類のないプロセッサ：Intel Xeon E3-1226 v3 |グラフィック：Intel HD Graphics P4600 | RAM：16GB |ストレージ：2 x 1TB HDD |接続インターフェイス：5 x USB 3.0、2 xギガビットイーサネットポート|寸法（幅x長さx高さ）：17.5 cm x 42 cm x 39.5 cm

+ 最高の保証
+光学ドライブ

あり-RAID 5なし
-富士通-最も明白なブランドではありません

富士通は、サーバーに関して最初に思い浮かぶブランドではありません。ただし、富士通は、業務用サーバーからスーパーコンピューターまで、すべてを手がける数少ないメーカーのひとつです。 TX1310はエントリーレベルのビジネス指向サーバーであり、動作することが証明されており、市場で最高の信頼性が保証されています。購入後1年以内にサーバーに障害が発生した場合、富士通はサーバーを修理または交換するだけでなく、このサーバーに対して支払った金額も返金します。

この市場セグメントのほとんどのモデルと同様に、サーバーは24時間365日稼働するように設計されており、RAID 0/1/10をサポートしています（RAID 5はサポートされていません）。3番目のバージョンのIntel Xeon E3-1216、1 TBおよび16 GBのRAMを搭載した2つのハードドライブがインストールされています。また、オプティカルドライブと予備の2つのギガビットイーサネットポートがあることもプラスです。4つのDIMMスロットと4つのストレージベイを備えたこのサーバーは、最大32 TBのデータを保存でき、最大32 GBのRAMをサポートします。

最も急速に成長している市場セグメントの1つで、支配しているのは1社だけです。 HP Proliant Microserver Gen8は、魅力的な機能の組み合わせと多数の割引との競争に成功し、最終的には市場を席巻しました。これらの小型サーバーは、プロの消費者の間で人気があり、NAS NASに勝るアドバンテージが宣伝されているため、ニッチ以外の需要も獲得しています。

コンパクト（容積0.013m3未満）、軽量（7kg未満）でありながら、圧倒的な性能を誇ります。インテルXeon E3プロセッサーファミリーのサポート、最大16 GBのRAMをインストールできる機能、システム管理モード、2つのギガビットイーサネットポート、1つのPCIeスロット、RAID 0/1/10のサポート、DVDライター、最大4つのハードドライブ、内部microSDカードスロット、統合Matrox G200グラフィックチップ、7つのUSBポート。一方、このサーバーにはVGAコネクタが1つとRAM用のコネクタが2つしかありません。

最大8つのディスクを収容できる強力なサーバー

プロセッサー：Xeon E3-1200V6 |グラフィック：Intel HD Graphics P630 | RAM：最大64GB |ストレージ：2x内部固定5.25インチベイ、最大80TB |接続インターフェイス：6 x USB 3.0、シリアルポート、ビデオコネクタ、2 xギガビットイーサネット|寸法（幅x長さx高さ）：44.4 cm x 58.3 cm x 17.3 cm

+優れたパフォーマンス
+オンサイトサービス付きの3年間保証 -高

コスト
-大型

上記のサーバーよりも強力なものが必要な場合は、TS460をチェックしてください。これははるかに高価なソリューションですが、異なるレベルのサーバーを取得します。まず、以前のモデルよりもはるかに大きいです。容積は0.05 m3、重さは25 kgです。この5Uフォームファクターサーバーは、ターボブーストテクノロジーを搭載したIntel Xeon E3プロセッサーファミリーで動作し、オンサイトサービスが保証されています。

最大64 GBのRAMをサポートし、内蔵コントローラーは4つの主要なタイプのRAIDをサポートします。DVDライター、300ワットの電源装置、2つのギガビットイーサネットポートも付属します。最大8台のハードドライブと最大8つのUSBコネクタを取り付けることができます。ケースには、ロック可能なドア、ECCサーバーメモリのサポート、シリアルおよびVGAコネクタがあります。

8. HP ProLiant ML350 G9 5U



非常に充実したハイパワーサーバー

プロセッサー：Intel Xeon E5-2603 v3 |グラフィック：Matrox G200 | RAM：8GB |ストレージ：ドライブは含まれません|接続インターフェイス：4 xギガビットイーサネット|寸法（幅x長さx高さ）：不明

+ 6コアXeonプロセッサー
+ 3年間のオンサイトサービスにより、処理後の1日が保証されます

-ウォレットは必ず破産します
-これ以上不満はありません

このモデルは、TS440およびML350と同じカテゴリに表示されます。統合ビデオカード（Matrox G200）に加えて、3年間の翌日職場サービス保証、4つのギガビットイーサネットポート、12 Gb /秒SASサポート（これには2.5インチドライブのみが必要です）があります。
さらに、このサーバーは3番目のバージョンのIntel Xeon E5-2603プロセッサーでは動作せず（通常のE3プロセッサーでは動作しません）、2つのプロセッサーをサポートしています。E5は6コアのプロセッサーであり、より複雑なタスクの解決に特に適しています。RAMに利用可能なスロットの数（24スロット）にも感心します。これにより、合計3 TBを収集できます（128 GBのLRDIMMストリップをインストールする場合）。また、このサーバーは、ロック可能なフロントドアとストレージコントローラーに加えて、ホットスワップ可能な機能を備えた2つの500ワット冗長電源を搭載することで追加ポイントを獲得します。

2つのXeon

プロセッサーで実行されるモンスタープロセッサー：デュアルIntel Xeon E5-2603 v4 |グラフィック：Nvidia GT 610 | RAM：64GB |ストレージ：1TB HDD |接続インターフェイス：2 xギガビットイーサネット|寸法（幅x長さx高さ）：23.2 cm x 56 cm x 55.9 cm。

-安くはありません（ただし、電力を考慮すると、価格は良好です）
-その他の苦情は厳選されます

強力なサーバーは高価である必要はありません-これはスキャンが潜在的な顧客に伝えたいものです。 SMB市場向けに特別に設計された3XSは、可能な限りコンパクトで静かになるように設計されています。英国のサプライヤーは、すべての組み立てプロセスのリアルタイム監視を提供しています（サーバーは注文に応じて組み立てられ、生産には24時間のストレステストと88ポイントの品質テストが含まれます）。各デバイスには3年間のオンサイトサービス保証が付属しており、診断リカバリユーティリティ付きのUSBフラッシュドライブも提供されます。

これで十分でない場合、このアセンブリのコンポーネントは、レビューで最高のものの1つです。Broadwellマイクロアーキテクチャ上の4番目のバージョンの2つのIntel Xeon E5-2603プロセッサは、合計で12コアと30 MBのキャッシュを備えています。さらに、64 GBのRAM、Samsung DDR4 ECC RAM、エンタープライズクラスのWDハードドライブ、2つのIntelギガビットイーサネットポート、80 PLUS Gold認定の1000ワット電源、8つのハードドライブがあります。コルセアの筐体にはドアと吸音機能付きのパネルがあります。

10. Asus TS500



柔軟性に余裕のあるメインストリームタワーサーバー

プロセッサー：Intel Xeon E5-2600 v3 |グラフィック：不明| RAM：注文による|ストレージ：ドライブは含まれません|接続インターフェイス：8 x USBポート、VGA、PS2、3 xギガビットイーサネット|寸法（幅x長さx高さ）：22 cm x 54.5 cm x 45.5 cm

+サーバーとワークステーションに理想的なソリューション
+電力と柔軟性が不足することはあり

ません-誰もがこれらの機能を必要としているわけではありません
-Asusのロゴはいくつかを驚かせるかもしれません

Supermicroと同様に、Asusはそのサーバーであまり知られていません。コンピューターコンポーネントの最大のサプライヤーの1つである台湾の会社は、マザーボードを含む幅広い消費者向け製品でよく知られています。 TS500-E8-PS4は、サーバーまたはワークステーションとしての使用に最適な5Uメインストリームタワーサーバーです。

3番目のバージョンの最新のIntel Xeon E5-2600プロセッサー、8つのDDR4 DIMM RAMスロット（最大0.5 TBのRAMをサポート）、6つの拡張スロット、3つの5.25インチストレージベイ、および80 Plus Bronze証明書を備えた500ワットの電源装置を搭載。 3.5インチのホットスワップ可能なSATA / SASハードドライブベイも4つあり、柔軟なストレージが必要な場合は、8つのHDDベイと交換できます。

さらに、インテルのインテリジェントプラットフォームマネジメントインターフェイス（IPMI）と互換性のあるASMB8-iKVMモジュールを使用すると、サーバーをリモートで監視、制御、管理できます。その他の主な機能には、10個のSATAポート、DVDライター、8個のUSBポート、PS2ポート、1個のVGAポート、3個のギガビットイーサネットポートがあります。これはアセンブリソリューションですが、このマシンの在庫に大きな柔軟性があることは明らかです。
