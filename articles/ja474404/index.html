A.パラレルユニバースからの非同期API。 （15点）
状態。

パラレルユニバースの開発者から、新しい宇宙船制御ライブラリが送られてきました。なぜなら宇宙船は複雑なものであり、ライブラリのAPIはかなり「重い」ため、メソッドの正確な数は不明です。もちろん、ドキュメントはありません。しかし、パラレルユニバースでは、人々は天井を歩き、昼間は睡眠をとり、夜は仕事をし、非同期関数のみを使用し、常に最初の引数としてコールバックを渡します。奇妙な男！ここ地球上で長い間、誰もが約束について書いています。ただし、ライブラリはプロジェクトに統合する必要があります。そのため、同じAPIを提供するラッパーを作成するという約束を果たしました。

入力形式

ソースAPIの例：

const api = {  
  a: {  
    b: {  
      c: callback => setTimeout(() => callback(null, ’hello’), 100)  
    }  
  },  
  aa: {  
    bb: (callback, x, y) => setTimeout(() => callback(null, x + y), 200)  
  }  
};

出力形式

ソリューションを次のように送信します。

/**  
 * @param {Object} api -  API  
 * @returns {Object}  
 */  
module.exports = function promisify(api) {  
  // ...  
  return promisedApi;  
};

使用例：

const promisedApi = promisify(api);  
promisedApi.a.b.c()  
  .then(res => console.log(res)); // => ’hello’



元のAPIを呼び出すときにエラーが発生した場合 、ラッパーは拒否されたpromiseを返す必要があることに注意してください。コールバックは常に最初の引数でエラーを受け入れます。

callback(error, data)

元のAPIでは、定数（数値、文字列、ブール値）が存在する可能性があり、そのまま返す必要があります。

api.foo.myConst = 1;  
promisedApi.foo.myConst === 1;

ラッパーの初期化は「遅延」である必要があります。元のAPIには多数のネームスペースが存在する可能性があり、使用するときにそれらにアクセスする必要があります。

解決策

コンテストでは、この問題を完全に解決することはできず、解決策の15点のうち12.86点を得ました。私は再帰を使用してすべてのプロパティをバイパスし、関数の場合はpromiscurateしました。
そして、テストの終了後、Proxyオブジェクトを使用してソリューションを完成させました。このオプションのみがすべてのテストに合格しました。以下は、改善を考慮したソリューションです。

function promisify (obj) {
  const cache = {}

  return new Proxy(obj, {
    get (target, prop) {
      const value = target[prop]
      const type = Object.prototype.toString.call(value)

      if (type === '[object Object]') {
        cache[prop] = promisify(value) //  ,   - 
        return cache[prop]
      } else if (type === '[object Function]') {
        cache[prop] = function (...args) { //   
          return new Promise((resolve, reject) => {
            const callback = (err, result) => {
              if (err) {
                reject(err)
              } else {
                resolve(result)
              }
            }
            value.call(this, callback, ...args)
          })
        }

        return cache[prop]
      }
      return value
    }
  })
}

ノードjsにはすでにそのようなユーティリティnodejs.org/dist/latest-v8.x/docs/api/util.html#util_util_promisify_originalがあり、コールバックが最後の引数であり最初の引数ではない関数のみを受け入れることに注意してください。

ソリューションソリューション

の本質は、画像に合わせてキャンバスを作成し、そこに画像を描画し、getImageDataを使用して各ピクセルの色に関する情報を取得し、次に白いピクセルが見つかるまで左上端からピクセルを繰り返し処理することです。次に、水平軸と垂直軸に沿ってピクセルをソートし続け、再び白いピクセルに出会うまで、長方形の幅と高さを取得します。

残念ながら、私のコードはすべてのテストに合格せず、40ポイントのうち30ポイントを獲得しました。そして、私はまだエラーを理解できませんでした、あなたが欠点を指摘してくれれば私は感謝します：

function traceImage (imgSrc) {
  function loadImg (src) {
    return new Promise((resolve, reject) => {
      if (typeof src !== 'string') reject('wrong data')
      const img = new Image()
      img.addEventListener('load', () => resolve(img))
      img.addEventListener('error', err => reject(err))
      img.src = src
    })
  }

  function calcElementFromImage (img) {
    const TRANSPARENT_COLOR = [255, 255, 255, 1]
    const colorFromData = (data, i) => {
      return [data[i], data[i + 1], data[i + 2], data[i + 3] / 255]
    }
    const w = img.naturalWidth
    const h = img.naturalHeight
    const canvas = document.createElement('canvas')
    canvas.width = w
    canvas.height = h
    const ctx = canvas.getContext('2d')
    ctx.drawImage(img, 0, 0, w, h)
    const data = ctx.getImageData(0, 0, w, h).data

    let top, left, bgColor
    let lastIndex = 0
    for (let i = lastIndex; i < data.length; i += 4) {
      const color = colorFromData(data, i)
      if (!color.every((c, i) => c === TRANSPARENT_COLOR[i])) {
        const px = i / 4
        left = px % w
        top = px / w | 0
        bgColor = color
        lastIndex = i
        break
      }
    }

    let width
    const maxLeftIndex = (w - left) * 4 + lastIndex

    for (let i = lastIndex; i < maxLeftIndex; i += 4) {
      const color = colorFromData(data, i)
      if (color.every((c, i) => c === TRANSPARENT_COLOR[i])) {
        const x = i / 4 % w
        width = x - left
        break
      }
    }

    let height
    const maxTopIndex = (h - top - 1) * w * 4 + lastIndex
    const hStep = w * 4
    for (let i = lastIndex; i < maxTopIndex; i += hStep) {
      const color = colorFromData(data, i)
      if (color.every((c, i) => c === TRANSPARENT_COLOR[i])) {
        const y = i / 4 / w | 0
        height = y - top
        break
      }
    }
    bgColor = Object.values(bgColor).join(',')
    return {
      parentWidth: w,
      parentHeight: h,
      top,
      left,
      width,
      height,
      bgColor
    }
  }
  return loadImg(imgSrc).then(img => {
    const data = calcElementFromImage(img)
    const { parentWidth, parentHeight, top, left, width, height, bgColor } = data
    const div = `<div style="
  position: relative;
  width: ${parentWidth}px;
  height: ${parentHeight}px;">
    <div style=" 
    position: absolute;
    top: ${top}px;
    left: ${left}px;
    width: ${width}px;
    height: ${height}px;
    background-color: rgba(${bgColor})"></div>  
    </div>`
    return Promise.resolve(div)
  })
}
