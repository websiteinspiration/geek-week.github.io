シアンをこする


2020年3月中旬、モスクワの1部屋のアパートを水色でレンタルするための約9000件の提案を集めることができました。このサイトには54ページが表示されています。 jupyter-notebook 6.0.1、python 3.7を使用します。サイトからデータをアップロードし、リクエストライブラリを使用してファイルに保存します。

サイトで禁止されないようにするため、リクエストに遅延を追加し、ヘッダーを設定して、ブラウザーを介してリクエストを送信する非常に賢い人のように見えるように、人を装います。毎回サイトからの応答を確認することを忘れないでください、さもなければ、我々は突然発見されて、すでに禁止されています。ウェブサイトスクレイピングの詳細については、たとえばこちら：pythonを使用したウェブスクレイピングをご覧ください。

また、関数とロギングの速度を評価するためにデコレータを追加すると便利です。level = logging.INFOを設定すると、ログに表示されるメッセージのタイプを指定できます。ログをテキストファイルに出力するようにモジュールを構成することもできます。これは不要です。

def timer(f):
    def wrap_timer(*args, **kwargs):
        start = time.time()
        result = f(*args, **kwargs)
        delta = time.time() - start
        print (f'   {f.__name__}  {delta} ')
        return result
    return wrap_timer

def log(f):
    def wrap_log(*args, **kwargs):
        logging.info(f"  {f.__doc__}")
        result = f(*args, **kwargs)
        logging.info(f": {result}")
        return result
    return wrap_log
logging.basicConfig(level=logging.INFO)

def parse_tag(tag, tag_value, item):
    key = tag
    value = "None"
    if item.find('div', {'class': tag_value}):
        if key == 'link':
            value = item.find('div', {'class': tag_value}).find('a').get('href')
        elif (key == 'price' or key == 'price_meter'):
            value = parse_digits(item.find('div', {'class': tag_value}).text, key)
        elif key == 'pub_datetime':
            value = parse_date(item.find('div', {'class': tag_value}).text)
        else:
            value = item.find('div', {'class': tag_value}).text
    return key, value


def parse_digits(string, type_digit):
    digit = 0
    try:
        if type_digit == 'flats_counts':
            digit = int(re.sub(r" ", "", string[:string.find("")]))
        elif type_digit == 'price':
            digit = re.sub(r" ", "", re.sub(r"₽", "", string))
        elif type_digit == 'price_meter':
            digit = re.sub(r" ", "", re.sub(r"₽/²", "", string))
    except:
        return -1
    return digit

def parse_date(string):
    now = datetime.strptime("15.03.20 00:00", "%d.%m.%y %H:%M")
    s = string
    if string.find('') >= 0:
        s = "{} {}".format(now.day, now.strftime("%b"))
        s = string.replace('', s)
    elif string.find('') >= 0:
        s = "{} {}".format(now.day - 1, now.strftime("%b"))
        s = string.replace('',s) 
    if (s.find('') > 0):
        s = s.replace('','mar')
    if (s.find('') > 0):
        s = s.replace('','feb')
    if (s.find('') > 0):
        s = s.replace('','jan')
    return parse(s).strftime('%Y-%m-%d %H:%M:%S')
