Lenovo Thinkserver SE350: pahlawan dari pinggiran

Hari ini kami sedang mempertimbangkan kelas baru perangkat, dan saya sangat senang bahwa selama beberapa dekade pengembangan industri server, untuk pertama kalinya saya memegang sesuatu yang baru di tangan saya. Ini bukan "lama dalam kemasan baru", perangkat ini, dibuat dari awal, hampir tidak ada hubungannya dengan pendahulunya, dan ini adalah server Edge dari Lenovo.

Kami hanya bisa berbagi dengan Habr gambaran luar biasa dari server kami, yang dipublikasikan di situs HWP.RU.



Jika Anda masih tersesat dalam terminologi, dan tidak tahu apa itu "Edge", maka evolusi secara singkat berevolusi dengan cara ini:

  • Pada awalnya, perusahaan besar memutuskan bahwa Big Data akan membantu mereka meningkatkan efisiensi kerja, dari penambangan bijih hingga penjualan permen karet di kasir supermarket.

Secara alami, industri merespons tantangan hari ini dengan memperkenalkan kelas baru perangkat: server untuk segala jalan, kompak, kuat, anti-perusak, aman ... Anda merasa ... tidak ada properti di atas yang berlaku untuk server lama! Temui Lenovo ThinkServer SE350 .

Faktor bentuk


Edge server tidak memiliki format case tunggal: di bengkel pertukangan Anda menggantungnya di dinding, dan di tenda tiup - hanya di atas meja, sehingga Lenovo membuat ThinkServer SE350 sekompak mungkin. Ini memiliki ketinggian 1U (40mm), lebar tubuh - 0,5U (215mm), panjangnya - 376 mm.



Kita dapat mengatakan bahwa dalam ukuran server ini memiliki format 1 / 2U, dan dapat dengan mudah dibawa dengan Anda seperti laptop, tetapi tidak demikian halnya. Kami lupa tentang catu daya, yang ada dua, dan keduanya eksternal, cukup besar, dengan kapasitas masing-masing 240 W. Dengan barang bawaan seperti itu, seluruh kekompakan mesin dapat dengan aman dibagi menjadi dua, karena membutuhkan dua tempat di dinding, Anda meletakkan server di atas meja - catu daya di bawah meja, yah, hal semacam itu. Tentu saja, diizinkan untuk menginstal dua mesin dalam 1 unit atau dalam 2 unit kabinet server, tetapi opsi ini dianggap sebagai pengecualian, dan rel untuk instalasi semacam itu dibeli secara terpisah.



Secara default, pabrikan menyarankan menggunakan server dan catu daya untuk dipasang di dinding, di mana alat berat ini dilengkapi dengan braket baja yang kuat. Mempertimbangkan bahwa "di pinggiran" objek bahkan mungkin tidak memiliki saklar jaringan yang sederhana, dan Thinkserver SE350 sendiri melakukan peran titik akses Wi-Fi, menggantungnya lebih tinggi adalah ide yang baik. Yah, jangan lupa bahwa dua catu daya akan menggantung pada braket di sebelahnya. Omong-omong, pabrikan mereka adalah FSP, dan dengan segala hormat kepada perusahaan ini, pemasok ini bukan pilihan terbaik untuk perangkat Enterprise, saya lebih suka melihat pasokan listrik Delta atau Seasonic.





Menurut hasil pengujian, catu daya FSP menunjukkan kinerja yang baik untuk adaptor eksternal. Saya sarankan menghubungkan salah satu catu daya ke stopkontak UPS yang tidak berlebihan, atau ke yang mati oleh timer untuk memperpanjang masa pakai baterai server. UPS 2 KVA pada umumnya menyediakan masa pakai baterai sekitar 3 jam untuk server. Jadi, bahkan jika generator kehabisan diesel, Anda akan punya waktu untuk pergi ke pompa bensin.

Perlindungan Pencurian


Untuk memastikan bahwa orang yang lalu lalang atau pekerja sendiri tidak melepaskan atau menyeret server ke rumah, braket memiliki kunci Kensington dengan anti-pencurian yang kuat yang mengunci mesin di braket. Melepaskan ThinkServer SE350 dari dinding tanpa kunci hanya dimungkinkan dengan linggis, merobek braket dengan pasak.



Tetapi jika Anda takut telah mencuri server Anda, penyerang akan menjualnya pada Avito, maka jangan khawatir: Lenovo ThinkServer SE350 juga memiliki perlindungan anti-pencurian elektronik, seperti smartphone dan laptop modern. Saat membeli, Anda mengaktifkan server di portal Lenovo menggunakan kode QR pada kasing, model, dan nomor seri. Dengan menautkan mesin ke akun Anda, Anda cukup menyalakan perlindungan di BIOS dan mengaktifkan sensor bawaan. Ada dua di antaranya: yang pertama adalah sensor bukaan tutup yang kita tahu, yang, omong-omong, tidak dapat dibuka tanpa melepas braket, dan yang kedua adalah sensor posisi, yang mendeteksi, misalnya, bahwa mesin digantung pada posisi vertikal dan sekarang dalam posisi horizontal.



Segera setelah server menyadari bahwa mereka mencoba mencurinya, ia tidak hanya akan memblokir, tetapi juga "mematikan" antarmuka jaringannya, termasuk Wi-Fi, LTE dan kabel, dan Anda dapat mengembalikannya agar berfungsi hanya melalui prosedur re-aktivasi pada layanan cloud Lenovo menggunakan menghubungkan smartphone baik dari jarak jauh. Untuk memastikan bahwa sistem tidak memberikan hasil positif palsu di area seismik aktif, sensitivitas dan posisi dalam ruang dikonfigurasikan dari BIOS. Jadi, pencurian untuk dijual kembali tidak masuk akal, dan serangan MITM, menempatkan server di lingkungan jaringan mereka sendiri, para penyerang tidak. Saat menggunakan SED-drive, penghapusan kunci enkripsi juga dikonfigurasi, tetapi karena di banyak negara perangkat ini dilarang impor, kami mem-bypass fungsi ini, dan kami mengingatkan pembaca bahwa teknologi ini hanya melindungi platform itu sendiri,dan menolak untuk menggunakan Bitlocker, Truecrypt atau alat enkripsi lainnya tidak sepadan.

Tentu saja, di tempat pabrikan, saya akan meletakkan stiker besar di paket dengan server dengan tulisan seperti "komputer dikodekan" atau "anti-pencurian dipasang", karena penyerang tidak tahu bahwa ThinkServer SE350 tidak berguna untuk mencuri: Anda tidak akan mendapatkan masalah selain masalah.

Perlindungan debu


Ada debu halus bahkan di pusat data paling mahal, dan bahkan dalam produksi atau dalam bidang barang ini dalam jumlah besar, tetapi Lenovo memiliki perlindungan dalam bentuk filter busa yang dipasang pada bingkai di bawah bezel depan. Filter pertama menutup asupan udara utama di mana prosesor ditiup, dan yang kedua - papan ekspansi.



Selain itu, setiap port dan setiap soket eksternal di server memiliki steker karet tebal. Saya belum pernah melihat tutup pada Mini-USB, atau RJ45, tetapi di sini bahkan jack antena dan bahkan lubang untuk jack antena memiliki steker sendiri. Secara total, Anda tidak perlu khawatir server menelan pasir atau debu dan akan mulai membeku. Keren, Lenovo, bagus sekali!

Komunikasi nirkabel


Ada tiga versi Thinkserver SE350 dengan konfigurasi port jaringan berbeda yang ditunjukkan pada diagram di bawah ini. Yang kami miliki adalah yang top-end memiliki:

  • 2 slot SFP + pada 10 Gbps,
  • 2 slot SFP pada 1 Gbps,
  • 1 RJ45 untuk BMC
  • 2 RJ45 pada 1 Gb / s

Pengontrol jaringan utama di sini adalah Intel x722, yang menyediakan 4 saluran 10 Gb / s, dua di antaranya merupakan output ke slot SFP +, satu tidak digunakan, dan sisanya terhubung ke Edgeboard, yang pada dasarnya adalah router nirkabel yang bekerja hampir secara independen dari server .

Secara fisik, Edgeboard adalah modul yang menggabungkan dua kartu Mini-PCIe dan sakelar NXP LS1046A. Untuk memasang kartu Nano-SIM, Anda harus menghapus modul ini dari server, lepaskan semua antena dari itu. Anda hanya memiliki slot untuk satu kartu SIM, yang dalam dunia modern ponsel dual SIM dan router WLAN terlihat agak aneh. Modul nirkabel mendukung standar 802.11ac dengan dua aliran spasial, yang memberikan kecepatan maksimum 433 Mbit / s.



Saklar dikendalikan oleh perangkat keras, melalui pengontrol BMC, Kontrol XClarity. Di sini Anda dapat memilih peran port bernomor 7, 5 dan 6, serta 8 dan 9.



Artinya, secara sederhana, dalam sistem operasi Anda selalu memiliki port 10-gigabit 1 dan 2 dan Wi-Fi, LTE tidak pernah tersedia , port 5, 6 dan 7. Alih-alih, Anda akan memiliki adaptor virtual, dengan alamat IP-nya berupa 192.168.73.xx, yang melaluinya server menerima dan mendistribusikan Internet. Misalnya, bagian atas dapat 10G dari penyedia + LTE, dan bagian bawah melalui gigabit RJ45 - seluruh infrastruktur jaringan perusahaan + Wi-Fi dari titik akses server. Menariknya, dengan konfigurasi seperti itu, server bahkan tidak memperhatikan hilangnya jaringan di salah satu port. Dan ya, tentu saja Anda dapat menghubungkan ponsel cerdas Anda sebagai modem USB, jika tidak ada yang tersisa.

Mengapa Lenovo begitu pintar, dan bukankah lebih mudah untuk menempatkan modem sederhana dan Wi-Fi dengan kontrol dari OS? Mungkin yang utama adalah operasi independen modul Wi-Fi / LTE dari server: bisa kelebihan beban, bisa hang, instal pembaruan, dan perangkat yang terhubung ke port EdgeBoard harus selalu terhubung ke Internet. Plus, Anda memiliki isolasi WAN tambahan dari LAN di tingkat perangkat keras dan titik akses bawaan dengan saluran 4G yang berlebihan. Tapi, tentu saja, modul nirkabel memiliki pengaturan minimum, dan bagi pengguna operasinya terlihat buram, jadi dari sudut pandang saya, implementasi cadangan seperti itu sangat kontroversial.

Sistem penyimpanan


Kami dengan senang hati mengucapkan selamat tinggal pada hard drive, mengirim bahkan dilupakan faktor bentuk 2,5 inci. Lenovo ThinkServer SE350 hanya mendukung drive format M.2, tanpa hot swap. Secara logis, sistem penyimpanan di server dibagi menjadi drive boot (papan M.2 ada di belakang riser, kami tidak memilikinya dalam konfigurasi kami) dan drive data (NVME atau SATA600). Secara default, konfigurasi kami memiliki 4 M.2 slot NVME / SATA yang ditempati oleh dua drive SATA dengan kapasitas 480 GB.

Pengujian menunjukkan bahwa ini adalah disk server yang sangat cepat yang tidak melorot dalam kecepatan selama perekaman intensif dan memiliki waktu respons yang dapat diprediksi tanpa penyumbatan.




Selain itu, Anda dapat memasang riser yang sama di papan 4 M.2 alih-alih kartu ekspansi. Saat menggunakan SED-disk, Anda dapat membeli opsi untuk menghapus kunci enkripsi secara otomatis ketika sensor kerusakan dan sistem anti-pencurian dipicu. Fungsi RAID perangkat keras tersedia untuk adaptor boot, yang menginstal drive yang sama dalam mode mirror, dan untuk drive penyimpanan standar, Anda dapat menggunakan RAID perangkat lunak Intel standar. Secara total, anak ini dapat berisi 10 drive dalam format M.2, yang akan Anda setujui sangat keren untuk setengah dari case unit tunggal!

Opsi ekspansi


Tidak ada kolega saya yang tahu kunci pas di dalam server, tetapi semuanya sederhana: server memberikan kemampuan untuk mentransfer antena Wi-Fi / LTE dari belakang ke panel depan. Untuk inilah kuncinya. Secara default, server dilengkapi dengan 1 16 GB DDR4 2666MHz ECC Terdaftar modul dan slot low-profile kosong untuk kartu PCI Express. Modul memori yang didukung masing-masing hingga 64 GB dengan frekuensi 2133/2400/2666 MHz dengan total kapasitas hingga 256 GB.



Saat memesan, Anda dapat memilih modifikasi prosesor dari seri Xeon-D 2100. Ini adalah CPU yang sangat baik, yang telah kami uji sebelumnya, itu dibuat khusus untuk sistem tertanam, seperti gateway keamanan, NAS-s dan node komputasi periferal. Dalam sistem pengujian kami, Xeon-D 2123 dipasang, 4-core dengan frekuensi dasar 2,2 GHz, TurboBoost - 3,0 GHz, dukungan untuk HyperThreading, AVX-2 dan AVX-512.

Di server BIOS, dimungkinkan untuk mengatur mode konsumsi daya jenis OS Control mode untuk mematuhi rekomendasi VMware untuk menggunakan Turbo Boost di dalam VM, serta membatasi frekuensi maksimum Turbo Boost. Dalam pengujian kami, frekuensi turbo maksimum prosesor tidak melebihi 2,68 GHz, dan tidak peduli sekeras apa pun saya mencoba, saya tidak dapat mengeluarkan frekuensinya bahkan sedikit lebih tinggi bahkan dalam mode 1-stream. Dari pengalaman saya sendiri, saya dapat mengatakan bahwa untuk Xeon-D jumlah saluran memori yang terlibat tidak sepenting frekuensi modul yang dipasang. Plus, itu adalah prosesor yang sangat panas untuk 60 W, jadi jangan panik jika memanaskan di atas 60 derajat dengan hampir tanpa beban: sistem pendingin server tidak ada hubungannya dengan itu.



... selain itu, ini diwakili oleh tiga penggemar 40 mm yang diproduksi oleh Delta Electronics dengan dudukan anti-getaran dan kemungkinan penggantian yang mudah (Cold Swap). Kontrol kecepatan otomatis memungkinkan mereka untuk beroperasi dalam rentang yang sangat luas dari 3.000 RPM hingga 24.000 RPM. Dalam mode siaga pada suhu kamar, tingkat kebisingan sekitar 38 dB. Saya ingin sekali lagi mencatat bahwa Lenovo ThinkServer SE350 memiliki kisaran suhu operasi udara ambien yang diperluas: hingga +45 derajat Celcius dalam konfigurasi apa pun dan hingga +55 derajat di sebagian.



Server memiliki 1 slot ekspansi profil rendah PCI Express 3.0 x16 tanpa daya tambahan (batas - 75 watt). Jika Anda perlu menggunakan fungsi AI dan pembelajaran mesin, Anda dapat menginstal adaptor Nvidia Tesla T4 16 Gb, serta papan FPGA dan ASIC lainnya.

Manajemen Jarak Jauh dan UEFI


Lenovo ThinkServer SE350 menggunakan sistem kendali jarak jauh XC Clarity pada Pilot4 XE401. Omong-omong, pengontrol kontrol ini digunakan di semua server Lenovo modern. Fungsi dasar seperti membuka kunci atau mengaktifkan mesin dapat dilakukan saat menghubungkan smartphone melalui port microUSB di panel depan. Ngomong-ngomong, di sebelah kiri port ini terdapat tombol tersembunyi untuk mengatur ulang pengontrol nirkabel (lihat di atas) dan untuk mengirim sinyal NMI ke pengaturan ulang perangkat keras.



Tentu saja, antarmuka sangat indah: di sini Anda memiliki grafik, bagan, dan kompatibilitas yang sangat baik dengan browser seluler. Dan UEFI terlihat dalam gaya yang sama dengan kontrol mouse yang sama seperti BMC. Hingga 6 pengguna dapat bekerja dengan satu jendela konsol jarak jauh secara bersamaan, dimungkinkan untuk memasang disk virtual melalui protokol NFS, mengatur pintasan makro, dll. Tidak ada gunanya mendaftar semua item, lihat tangkapan layar.

Untuk penyebaran yang disederhanakan di BIOS, ada opsi untuk mempercepat instalasi Windows Server 2016 atau VMware ESXi 6.5 / 6.7: cukup masukkan kata sandi administrator Anda dan minum teh, sistem akan mempartisi disk dan menginstal OS.

Lenovo mengumumkan serangkaian standar sistem operasi perusahaan yang kompatibel:

  • Microsoft Windows Server 2019
  • Red Hat Enterprise Linux 7.6
  • SUSE Linux Enterprise Server 15
  • SUSE Linux Enterprise Server 15 Xen
  • VMware ESXi 6.5 U2
  • VMware ESXi 6.7 U2

Menjamin


Server Lenovo ThinkServer SE350 dicakup oleh garansi 3 tahun dengan opsi perpanjangan 5 tahun. Jaminan untuk SSD diberikan jika sumber daya siklus penulisan ulang yang ditentukan dalam buku petunjuk belum berakhir. Layanan di tempat tersedia dalam mode 9x5 dengan check-out pada hari berikutnya, serta paket peningkatan layanan.

Memesan rekomendasi


Seperti yang Anda lihat, Lenovo ThinkServer SE350 terlalu tidak biasa untuk dibandingkan dan dibandingkan dengan analog dari perusahaan lain: ia masih memiliki Intel Xeon, dengan memori ECC buffered, jaringan cepat bahkan dengan standar modern, kemampuan penyimpanan data yang sangat baik dan perlindungan dari pencurian. Ini tidak terjadi ketika mereka hanya mengambil nettop yang kuat pada Core i7 dan menggantungnya dengan antena dan menyebutnya server untuk Edge.

Pada awal artikel, saya mengatakan bahwa kata "Edge" biasanya menyiratkan konsep "di pinggiran", tetapi istilah ini juga memiliki konsep yang berbeda. Namun demikian, di dunia TI, istilah "Edge" biasanya disebut teknologi paling canggih, dan dalam kasus ThinkServer SE350, terjemahan ini harus dianggap benar. Beberapa solusi yang digunakan di sini untuk kita adalah rasa ingin tahu, dengan sesuatu, seperti menerapkan koneksi Wi-Fi dan LTE, kita harus menghabiskan berjam-jam memikirkan kembali, tetapi pada akhirnya itu adalah mesin yang tidak memiliki jenis, dan memesannya melalui tender, Anda dapat yakin bahwa Anda tidak akan mendapatkan analog.
