VPN de acesso remoto Check Point - curso gratuito de configuração de acesso remoto



Saudações amigos! Bem-vindo ao nosso próximo novo curso! Como prometi, o curso de Introdução não foi o último. Desta vez, discutiremos um tópico igualmente importante - VPN de acesso remoto (ou seja, acesso remoto). Com este curso, você pode se familiarizar rapidamente com as tecnologias Check Point em termos de organização do acesso remoto seguro aos funcionários. Como parte do curso, como sempre, combinaremos a parte teórica com a parte prática na forma de trabalho de laboratório. Além de demonstrar a configuração do Check Point, veremos várias maneiras de conectar usuários remotos.

Alguns comentários


  • Este curso não é um guia completo para a tecnologia VPN. Este é apenas um aperto muito breve das informações mais solicitadas. Essa. você não deve esperar que, após a visualização, você se torne o guru das VPNs. No entanto, para entender os princípios e começar rapidamente a trabalhar com a Check Point VPN, este curso será mais do que suficiente.
  • Eu recomendo iniciar este curso somente após concluir o anterior - Introdução ao Check Point. Não explicarei aqui algumas coisas básicas, como arquitetura, como configurar um endereço IP ou obter um gateway para o servidor de gerenciamento. Partirei da suposição de que você já sabe disso e nos concentraremos apenas na VPN.
