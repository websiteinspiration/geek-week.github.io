Arsitektur otak. Konsep saluran







Pada artikel ini saya akan berbicara tentang salah satu hipotesis saya mengenai arsitektur otak.







Penalaran didasarkan pada konsep saluran yang sangat sederhana. Mungkin pembaca akan menganggapnya naif. Namun demikian, saya harus berusaha keras untuk menghasilkan ini. Ini bukan artikel ilmiah. Jika seseorang menyebutnya spekulasi lain pada subjek otak, maka secara formal itu akan benar. Kritik dipersilahkan.







pengantar



Sejarah hipotesis ini dimulai pada 2015. Sekali lagi memikirkan struktur otak, saya sampai pada satu ide. Esensinya adalah bahwa otak entah bagaimana harus membangun model lingkungan. Diperlukan model untuk memutuskan hubungan langsung antara reseptor dan efektor. Makhluk tanpa model seperti itu hampir seluruhnya bergantung pada sinyal eksternal. Penggerak dimungkinkan sebagai taksi atau berjalan acak. Dalam arti tertentu, makhluk seperti itu seperti benda mati. Misalnya, bola hanya bergerak ketika menerima "sinyal eksternal" dari kaki seorang pemain bola. Ini tidak jauh berbeda dari uniseluler konvensional, yang bergerak sepanjang gradien kimia menuju nutrisi. Perilakunya juga sepenuhnya ditentukan oleh sinyal eksternal. Anda dapat mengubah pola ini dengan menempatkan memori antara reseptor dan efektor,yang menyimpan model lingkungan yang kasar.

class dotdict(dict):
    __getattr__ = dict.get
    def __setattr__(self, name, value):
        self[name] = value

class Machine:
    """
         tick.
            tock.
     tick  tock        .
    """

    def __init__(self, cells):
        self.cells = cells
        self.tick = dotdict()
        self.tock = dotdict()
        for key in cells:
            self.tick[key] = 0b0000
            self.tock[key] = 0b0000

    def step(self):
        """    """
        for name, func in self.cells.items():
            if func is None:
                #  ,     
                self.tock[name] = self.tick[name]
            else:
                #       
                self.tock[name] = func(self.tick)
        #   
        self.tick, self.tock = self.tock, self.tick
        self.show()

    def show(self, comment = None):
        """    """
        if comment is not None:
            print(f': {comment}')
        for key in cells:
            print(key, f'{self.tick[key]:04b}')
        print('------------------------------------------------')
