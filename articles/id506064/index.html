Kembangkan aplikasi web interaktif dalam C # dengan nol pengetahuan tentang JavaScript

Saya adalah pengembang .Net di Turkmen-Tranzit.

PM saya membutuhkan seseorang yang ingin merasakan Blazor. Menurut pendapat saya, saya telah melakukan pekerjaan dengan baik di bidang ini. Selama penelitian, saya menemukan masalah, jawaban yang tidak begitu mudah ditemukan di Internet berbahasa Rusia.



Mengapa Blazor, dan bukan JavaScript standar?

Dengan kata sederhana, teknologi Blazor memungkinkan Anda membuat aplikasi web yang sisi kliennya terdiri atas markup HTML dan kode C # alih-alih JavaScript.

Blazor dapat mengakses DOM untuk menggambar ulang halaman tanpa menyegarkan halaman
Aplikasi web pada Blazor terdiri dari komponen. Komponen dirancang untuk menampilkan elemen visual seperti indikator unduhan atau formulir unggah file. Markup elemen disimpan dalam file dengan ekstensi .razor, dan semua logika di kelas C # biasa dengan nama {ComponentName} .razor.cs.

Komponen-komponen dalam Blazor terbagi menjadi pintar dan bodoh.

Komponen bodoh tidak tergantung pada keseluruhan aplikasi, mereka dapat dicolokkan ke komponen lain, dan mereka bahkan dapat menggunakan komponen bodoh lainnya.

Komponen pintar dapat dibuang, mereka harus mempertahankan statusnya dan sangat jarang digunakan dalam komponen lain.

Paling sering Anda harus menulis komponen cerdas.

Di Blazor, Anda dapat memanggil fungsi JavaScript menggunakan kode C #, dan sebaliknya, Anda dapat memanggil metode .Net menggunakan JavaScript.

Ya, jika Anda memiliki pengetahuan yang mendalam tentang JavaScript, dipandu dalam kerangka kerja JavaScript dan Anda baru saja berkenalan dengan dunia .Net, maka lebih baik untuk tidak masuk ke Blazor, teknologinya relatif baru dan ketika Anda pertama kali mengetahuinya, Anda bisa merasa takut. Tujuan artikel ini adalah untuk orang-orang yang tahu cara menggunakan C #, tetapi untuk beberapa alasan yang tidak memiliki JavaScript, bicarakan tentang mitra yang menjanjikan dari Microsoft, di mana Anda dapat menggunakan semua fitur C #.

Misalnya, kami akan mengembangkan CRM, dengan daftar perusahaan yang berpotensi menjadi mitra kami, dengan pencarian otomatis untuk karyawan perusahaan, distribusi pesan secara otomatis ke semua pelanggan potensial, mencatat tindakan pengguna, dan banyak lagi.
