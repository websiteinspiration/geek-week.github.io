神の怒り


多くの古風な社会では、人々は霊と神がすべての善の怒りに値する人々に病気と破壊を送ると信じていました。この非科学的な認識はしばしば破滅的な結果を伴い、数千、さらには数百万の死に至りました。

ユスティニアヌスの伝染病の場合、ビザンチンの歴史家カエサレアのプロコピウスは、伝染病（細菌Yersinia pestis）の起源を中国とインド北東部までエジプトから陸と海への貿易ルートをたどって追跡し、そこから地中海の港を通じてビザンチン帝国に入りました。

病気の蔓延における地理学と貿易の役割についての彼の明白な理解にもかかわらず、プロコピウスは大衆疫病をユスティニアヌス帝に非難し、彼を悪魔と宣言し、彼の邪悪な行為に対する神の罰を科した。歴史家達は、そのような発展がローマ帝国の西部と東部の残党を再統合するための皇帝ユスティニアヌスの努力に終止符を打つ可能性が高いと結論づけました。それは暗黒時代の始まりを示しました。

幸い、病気の原因に対する人類の理解は徐々に高まり、パンデミックへの対応が根本的に改善されました。この反応はしばしば遅すぎて不完全でしたが。

輸入疾病


検疫の実施は、疫病の流行から沿岸都市を保護する手段として14世紀にさかのぼります。慎重な当局は、感染した港からヴェネツィアに到着する船が係留する前に40日間待つことを要求しました。イタリア語起源の「検疫」という言葉そのものは「quaranta giorni」に由来します。 40日。

流行に対抗するために地理学と統計分析が使用された最初のケ​​ースの1つは、コレラの発生中に19世紀半ばにロンドンで発生しました。 1854年に、ジョンスノー博士はコレラが汚染された水を介して広がるという結論に達し、死亡率データを地図上に直接表示することを決定しました。この方法により、人々が水を飲んだ特定の取水柱の周りの症例のグループが明らかになりました。

貿易と都市生活によって生み出された相互作用が重要な役割を果たしますが、特定の疾患の病原性の性質もパンデミックの経路を示しています。

感染性の追跡


科学者は、ベースライン測定値を使用して、R 0またはR-zeroとしても知られる、生殖の基本数と呼ばれる疾患の感染性を追跡します。この数は、平均して、各患者に感染する可能性のある感受性の高い人の数を示します。

はしかはR 0 = 12-18の範囲で最も伝染性であり、リストをリードしています。これは、1人がワクチン接種されていない集団で平均12〜18人に感染する可能性があることを意味します。

はしかは非常に危険である可能性がありますが、ワクチン接種の努力と集団免疫はその広がりを抑えています。病気に免疫のある人が多いほど、それが広がる可能性が低くなり、既知の治療可能な病気の再発を防ぐためにワクチン接種が重要になります。

パンデミックが本格化しており、研究者はまだこの新しい形態のコロナウイルスを研究しているため、COVID-19の真の影響を計算して予測することは現在問題となっています。

都市化と病気の蔓延


世界的なつながりの成長がパンデミックの原動力となっています。少なくとも狩猟採集民の小さな部族では、少なくともメガロポリスでは、人々のお互いへの依存が病気の蔓延の機会を開きます。

発展途上国の都市化は、人口の増加が環境により多くの圧力をかける一方で、ますます多くの農村部の人々がより密集した地域に陥るという事実につながります。同時に、旅客数は過去10年間でほぼ倍増しています。これらのマクロトレンドは、感染症の拡大に大きな影響を及ぼします。

組織と政府は現在、感染率を下げるために自発的に自己隔離に入るよう市民に要求しているので、デジタルの世界は人々がこれまでにないような社会的つながりと商取引を維持することを可能にします。
