1. 2つのオブジェクトの同一性をチェックする方法は？

混同しないでください。演算子はオブジェクトの等価性のみをチェックします。is====

more_animals      = animals
print(animals == more_animals) #=> True
print(animals is more_animals) #=> True
even_more_animals = ['python','gopher']
print(animals == even_more_animals) #=> True
print(animals is even_more_animals) #=> False

animals両者はeven_more_animals同じですが、同一ではない ことに注意してください。

37. Pythonの行は不変です。どういう意味ですか？


文字列は不変であるという事実は、文字列オブジェクトが作成された後は変更できないことを示唆しています。文字列を「変更」しても、元の文字列は変更されません。代わりに、完全に新しいオブジェクトがメモリに作成されます。関数を使用してこれを証明できますid()。

proverb = 'Rise each day before the sun'
print( id(proverb) )
#=> 4441962336
proverb_two = 'Rise each day before the sun' + ' if its a weekday'
print( id(proverb_two) )
#=> 4442287440

連結'Rise each day before the sun'し' if its a weekday'てメモリに格納すると、新しいオブジェクトが新しい識別子で作成されます。元のオブジェクトが変更された場合、オブジェクトは同じ識別子を持ちます。

40.文字列から母音を削除する方法は？


この質問に対する1つの答えは、文字列内の文字がリスト内包メカニズムを使用してソートされることです。シンボルは、母音を含むタプルと比較することによってチェックされます。文字がタプルに含まれていない場合は、新しい行に追加されます。

#=> 'Hll 1 Wrld 2'
