環境設定


現時点でGraphQLを操作するための最も人気のあるライブラリであるApolloを介してサーバーにリクエストを送信します。

仕事に取り掛かりましょう。まず、ネットワークを操作するために必要な権限をマニフェストに追加しましょう。

<uses-permission android:name="android.permission.INTERNET" />
<uses-permission android:name="android.permission.ACCESS_NETWORK_STATE" />

次に、アポロを接続する必要があります。メインのbuild.gradleに移動し、依存関係セクションで次の行を追加します。

classpath 'com.apollographql.apollo:apollo-gradle-plugin:2.0.0'

次に、appモジュールのbuild.gradleファイルに追加の依存関係を追加する必要があります。

implementation("com.apollographql.apollo:apollo-runtime:2.0.0")
implementation "com.apollographql.apollo:apollo-android-support:2.0.0"

さらに、ファイルの最上部にプラグイン接続を追加します。 

apply plugin: 'com.apollographql.apollo'

開発環境を最小化してターミナルを開きましょう。プロジェクトがあるフォルダーに移動します。

cd /Users/user/Desktop/ProjectName

クエリ実行


ApolloClientを提供するシングルトンNetworkServiceクラスを作成します。その中に2つのメソッドを作ります。トークンや追加のパラメーターを必要としないリクエストを実行するgetApolloClient（）、およびリクエストに対してトークンをスローするgetApolloClientWithTokenInterceptor（）。

import com.apollographql.apollo.ApolloClient
import okhttp3.Interceptor
import okhttp3.OkHttpClient
import okhttp3.Request

class NetworkService {

  fun getApolloClient(): ApolloClient {
    val okHttp = OkHttpClient
      .Builder()
      .build()

    return ApolloClient.builder()
      .serverUrl(BASE_URL)
      .okHttpClient(okHttp)
      .build()
  }

  fun getApolloClientWithTokenInterceptor(token: String): ApolloClient {

    val httpClient = OkHttpClient.Builder()
      .addInterceptor(Interceptor { chain: Interceptor.Chain ->
        val original: Request = chain.request()
        
        val builder: Request.Builder = original
          .newBuilder()
          .method(original.method, original.body)
        
        builder.header("Authorization", "Bearer $token")
        return@Interceptor chain.proceed(builder.build())
      })
      .build()

    return ApolloClient.builder()
      .serverUrl(BASE_URL)
      .okHttpClient(httpClient)
      .build()
  }

  companion object {
    private var mInstance: NetworkService? = null

    fun getInstance(): NetworkService? {
      if (mInstance == null) {
        mInstance = NetworkService()
      }
      return mInstance
    }
  }
}
