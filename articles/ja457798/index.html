OpenStack Foundationの概要


OpenStack Foundation（OSF）は、OpenStackオープンクラウドプラットフォームの開発をサポートするために2012年に設立されました。そして、組織はそのコミュニティで急速に「大きくなりすぎ」ました。現在、OpenStack Foundationのメンバーは500人を超えています。その中には、テレコム、クラウドプロバイダー、ハードウェアメーカー、さらにはドメイン名登録機関さえ含まれます。

OpenStack Foundationは長い間、その名を冠したプロジェクトの開発に従事してきました。しかし、今年の初めに、基金はベクトルを変えました。組織は、機械学習、CI / CD、周辺機器コンピューティング、コンテナ化に関連するプロジェクトのサポートを開始しました。

これに関して、いくつかの新しいプロジェクトが基金に加わりました。

どのようなプロジェクト。月にオープンインフラサミットの会議では、OSFの代表は話した最初の「新参者」について-これらがあっ カタコンテナとZuul。

最初のプロジェクトは、KubernetesおよびDockerコンテナーのパフォーマンスに匹敵する安全な仮想マシンの開発です。 VMは100ミリ秒を超えない速度でロードされるため、コンピューティングリソースをオンザフライで展開するためのアプリケーションがクラウドで見つかります。ちなみに、いくつかの大規模なIaaSプロバイダーがカタの開発にすでに参加しています。

2番目のプロジェクトZuulは、CI / CDシステムです。コード変更の並列テストを実施し、潜在的なクラッシュを防ぎます。

基金の見通し。OpenStack Foundationは、開発の方向性を変えることにより、有能な開発者とのコミュニティを強化できると語っています。しかし、誰もがそう思っているわけではありません。5月の会議で、Canonicalの創設者であるMark Shuttleworth は、ファンドのポートフォリオの拡大を「間違い」と呼びました。彼の意見では、OpenStack Foundationはリソースを非効率的に使用しており、最終的には主要製品であるOpenStackクラウドプラットフォームの品質に影響を及ぼします。これがそうであるかどうかは、今後見られる必要があります。

Linux Foundationの機能


財団は、Linuxの宣伝と標準化、およびオープンソースソフトウェアエコシステム全体の開発に取り組んでいます。ファンドのポートフォリオは定期的に新しいプロジェクトで更新されます-それらのいくつかは文字通り今週登場しました。

どのようなプロジェクト。 6月25日、FATEフレームワークはLinux Foundationの一部となりました。それはオープンソースの中国のWeBankとTencentに転送されました。新しいソリューションの目標は、にあるヘルプ満たすGDPR要件という安全な人工知能システムの開発企業。ディープラーニング手法、ロジスティック回帰、および「転移学習」を実装するためのツールが含まれています「（この場合、すでにトレーニングされたモデルが使用され、他の問題を解決するように調整されています）。プロジェクトのソースコードはGitHubにあります。

また、今年の初めに、Linux Foundation はCommunityBridgeプラットフォームを発表しました。オープンプロジェクトをスポンサーする準備ができている開発者と投資家の間の一種の架け橋として機能します。このプラットフォームは、新しい開発者をオープンソースに引き付けるのに役立つはずです。

それにもかかわらず、彼女はすでに批判されています。業界の専門家は言うLinux Foundationが提供する金融サービスは最小限であり、契約やライセンス供与などの問題は取り残されています。おそらく将来的には、CommunityBridgeの機能が拡張されるでしょう。

基金の見通し。昨年末、Linux FoundationはGraphQLとCephの 2つの新しい基盤を確立しました。組織は、オープンソースソフトウェアエコシステムの開発を継続する予定です。

たとえば、Linux FoundationとFacebook は計画中ですosqueryプロジェクト専用の新しいファンドを開きます。 Osqueryは、ソーシャルネットワークデベロッパーだけでなく、Airbnb、Netflix、Uberで使用されるオペレーティングシステムを監視するためのフレームワークです。このツールを使用すると、実行中のプロセス、ロードされたカーネルモジュール、およびネットワーク接続に関するデータを取得するプロセスを最適化できます。

Linux Foundationは、近い将来にポートフォリオを再び拡大することが期待できます。おそらく、彼らは、KubernetesとCoreDNSが出てきたCloud Native Computing Foundationの運命を待っています。あるいは、彼らはTizenファンドの足跡をたどるだろう。同名のオペレーティングシステムの不人気のため、その見通しは不透明なままである。
