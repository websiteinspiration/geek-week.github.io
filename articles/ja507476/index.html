初心者のWeb開発者のためのヒント

私は1995年以来、何らかの形でウェブサイトを作成しています。25年の仕事のあと、私は経験を積み重ねて、今では十分な知識を持っていると思います。以下は、初心者の開発者が反省したいいくつかのポイントをランダムに概説しています。



  • 時にはサイトは単なるサイトです。
  • ブラウザはすでにクライアントです。HTMLは、それが話す言語です。
  • Webはサーバー側の視覚化を中心に構築されています。
  • いくつかの方法でデータを提供できます。HTMLはデータを表示する1つの方法と考えてください。
  • サーバー側でのスケーリングはすべての人に役立ちますが、クライアント側でのスケーリングは強力なハードウェアと高速接続を備えたもののみを助けます。
  • 誰もがマウスを持っている（または使用できる）わけではありません。
  • 誰もがキーボードを持っている（または使用できる）わけではありません。

何人かの人々は、「ページの読み込みの間、クライアントの状態を維持する」と言って私が何を意味するかに興味を持ちました。これは教科書ではありませんが、次のようなことができます。 



<html><head>
<title>Div replacement test</title>
<script>
window.addEventListener('DOMContentLoaded', () => {
    [].slice.call(document.getElementsByClassName("replace")).forEach(elem => {
        elem.addEventListener('click', event => {
            event.preventDefault();
            fetch(elem.href)
                .then(response => response.text())
                .then(text => {
                    let parser = new DOMParser();
                    var other = parser.parseFromString(text, 'text/html');
                    document.title = other.querySelector('title').innerHTML;
                    var content = document.getElementById('content')
                    content.innerHTML = other.querySelector('#content').innerHTML
                })

        })
    })
})
</script>
</head><body>
<section id="content">
<p>Here is the content container. <a class="replace" href="replaced.html">The other document</a></p>
</section>
