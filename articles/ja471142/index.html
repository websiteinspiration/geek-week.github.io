12の簡単な例で12のES10機能

これは、Medium.com のCarlos Caballeroのブログの記事の翻訳です。著者は、2019 ES10バージョンに登場した機能について教えてくれます。



ES10は、2019用のECMAScriptの最新バージョンです。2015年にリリースされたES6バージョンほど多くの革新は含まれていませんが、いくつかの便利な機能が含まれています。

この記事では、ES10で導入された機能を簡単なコード例として説明します。詳細な説明がなくてもすぐに理解できます。もちろん、これには基本的なJavaScriptの知識が必要です。

ES2019の新しいJavaScript機能は次のとおりです。

  • Array#{flat,flatMap}
  • Object.fromEntries
  • String#{trimStart,trimEnd}
  • Symbol#description
  • オプションのバインディング try { } catch {} //
  • JSON⊂ECMAScript
  • 整形式 JSON.stringify
  • 安定した Array#sort
  • 更新された関数＃toString
  • BigInt ・シンプルタイプ（3段）
  • 動的インポート（ステージ3）
  • 標準化された施設global This（3段階）。

Array.flat（）およびArray.flatMap（）


2つの新しいメソッドがありますArray。

  • このメソッドArray.flat()は、ネストされたサブ配列のすべての要素が再帰的に指定された深さレベルに引き上げられた新しい配列を返します。
  • このメソッドArray.flatMap()は、最初に各要素に関数を適用し、次に結果をフラット構造に変換して、新しい配列に配置します。これは、depthパラメータが1 map()の関数flat()を後で使用する関数と同じですが、flatMap()両方のアプローチを1つの方法に組み合わせているため、より効果的であることがよくあります。



Object.fromEntries（）


コードのリストと値のペアをオブジェクトに変換します。



String.prototype.matchAll


文字列を正規表現と照合する場合、メソッドmatchAll()はキャプチャグループを含むすべての結果のイテレータを返します。



String.trimStart（）およびString.trimEnd（）


String文字列からスペースを削除するに は、2つの新しい方法があります。

  • このメソッドtrimStart()は、行の先頭のスペースを削除します。
  • このメソッドtrimEnd()は行末のスペースを削除します。



Symbol.Description


説明を取得するための新しいオプションを追加しましたSymbol。これで、を作成するときにSymbol、説明として行を追加できます。ES10では、このインジケーターにアクセスできます。



オプションのバインディング try/catch


以前は、catch構文からの式にtry/catchは変数が必要でした。今日、開発者はtry/catch不要なバインディングを作成せずに使用できます。

  • U + 2029-行区切り。



整形式のJSON.stringify（）


JSON.stringify()U+D800との間の文字を、U+DFFF対応するUTF-8文字がない値として返すことができます。ただし、JSON形式にはUTF-8エンコードが必要です。ペアになっていないサロゲートコードポイントを単一のUTF-16コードユニットのままにするのではなく、JSONシールドシーケンスとして提示するソリューションが提案されています。



安定したArray.prototype.sort（）


V8エンジンの以前の実装では、不安定な高速ソートアルゴリズムが10個を超える要素を含む配列に使用されていました。

安定した並べ替えアルゴリズムは、同じキーを持つ2つのオブジェクトが並べ替え前と同じ順序のままになる並べ替えアルゴリズムです。



更新されたFunction.toString（）


このメソッドtoString()は、関数のソースコードの文字列表現を返します。ES6ではtoString()、関数に対して呼び出されると、ECMAScriptエンジンの特定の実装に応じて異なる結果を返す可能性があります。可能な場合はソースコードが返され、それ以外の場合は標準化されたスタブが返されました。



BigInt-任意の長さの整数


BigInt7番目のプリミティブ型であり、任意の長さの整数です。このタイプの変数は、253個の数字で構成できます9007199254740992。数値に限定されません。

結論


JavaScriptは生きた言語であり、Webプログラミングに非常に有利です。2015年のES6の登場以来、そのダイナミックな発展を目の当たりにしています。この記事では、2019 ES10バージョンに登場した機能を取り上げました。ES11（2020）で安定性を獲得するいくつかの機能も導入されました。これらの機能は実装の3段階にあり、最終的には次のリリースで標準化される可能性があるためです。
