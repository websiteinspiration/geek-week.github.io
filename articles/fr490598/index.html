Annonces Microsoft sur les examens et les certificats

Au cours des 3 derniers mois, Microsoft a fait 3 annonces majeures:

  1. MCSA, MCSD, MCSE depuis 2021, ainsi que tous les examens précédents, à l'exception du nouveau basé sur les rôles, cessent d'exister .
  2. Presque tous les examens basés sur les rôles Azure précédents sont obsolètes. Cela peut être lu dans un article séparé avec analyse
  3. Il y aura plusieurs nouveaux examens et, en conséquence, des certificats pour eux:
    • Examen DP-300: Administration de bases de données relationnelles sur Microsoft Azure (bêta) / Microsoft Certified: Azure Database Administrator Associate
    • Examen DA-100: Analyse des données avec Microsoft Power BI (bêta) / Certifié Microsoft: Data Analyst Associate
    • Examen AZ-120: Planification et administration de Microsoft Azure pour SAP Workloads / Microsoft Certified: Azure pour SAP Workloads Specialty
    • Examen AZ-220: Développeur Microsoft Azure IoT / Certifié Microsoft: Spécialité développeur Azure IoT

Tout d'abord.

MCSA / MCSA / MCSE - tout. C'est tout.


Je suis sûr que l'obsolescence de MCSA / MCSA / MCSE a provoqué une réaction beaucoup plus grande que toutes les autres annonces. S'il n'y avait pas que de nouveaux examens dans SQL 2019 ou Windows Server 2019, ce n'est pas si effrayant. Mais en fait, Microsoft tous les certificats et examens qui ne sont pas dans Azure / Dynamics365 / Office365 sont multipliés par zéro. Bien sûr, nous avons adouci la pilule, en nous assurant que dans le relevé de notes de vos examens, le fait de réussir ne disparaîtra pas, mais cela ne change pas l'essence.

Azure SAP


Tout est compliqué ici, car la technologie Microsoft n'est pas combinée avec Azure, c'est-à-dire examen simultané sur l'infrastructure Azure / composants et services SAP (pour la surveillance, pour l'optimisation) et l'architecture.

Azure IoT


L'examen combine les compétences d'un développeur IoT en C #, administrateur / intégrateur d'une solution IoT (Device Provisioning est très demandé), un utilisateur Azure SaaS pour IoT et un architecte. Il est clair que l'IoT s'approfondit de plus en plus dans nos vies, et l'examen est nécessaire pour ceux qui veulent vérifier leurs compétences. Personnellement, je ne suis pas sûr que le nombre de développeurs IoT sera important, mais un examen était clairement requis pour fermer le créneau correspondant.

Power BI


C'est simple: comment travailler avec Power BI. Personnellement, je n'aime pas le nom du certificat donné pour cet examen, «Microsoft Certified: Data Analyst Associate». Il existe de nombreux autres excellents outils pour l'analyse des données, PowerBI est trop étroit pour un tel nom.

Administrateur Azure SQL


Ici, tous les mots sont importants - pas un seul ne peut être jeté. Ce n'est pas du développement, mais du déploiement, de l'optimisation, des performances, de la tolérance aux pannes, etc. Le serveur SQL local n'est considéré que comme faisant partie de l'hybride (il n'y a pas un seul mot sur privé dans les mesures de compétences sur privé).

Parce que Code d'examen DA-300, puis suivant la convention de dénomination, je ne serai pas surpris de voir un autre DP-301 sur le développement. Mais ce ne sont que des spéculations.

Pour résumer, je peux dire que Microsoft fait des pas de géant, et si vous êtes en route, le monde devient plus intéressant. Si cet itinéraire ne vous convient pas, ne vous inquiétez pas, il existe de nombreuses autres façons avec d'autres fournisseurs et fabricants de logiciels et d'équipements.
