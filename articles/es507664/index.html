Compras en la aplicación iOS: configuración y adición al proyecto

Hola a todos, mi nombre es Vitaliy, soy el fundador de Adapty. Las suscripciones son una forma de monetizar la aplicación. Con su ayuda, puede darle al usuario la oportunidad de obtener acceso constante al contenido actualizado en la aplicación o al servicio proporcionado. A diferencia de las compras regulares, donde Apple tiene una comisión del 30%, en las suscripciones esta comisión se reduce al 15% si el usuario está registrado por 1 año o más. Un punto importante: si el usuario cancela la suscripción, este contador se restablecerá después de 60 días.







En esta parte aprenderemos:







  • Crear compras en App Store Connect

private func requestProducts() {
        productRequest?.cancel()

        let productRequest = SKProductsRequest(productIdentifiers: productIdentifiers)
        productRequest.delegate = self
        productRequest.start()

        self.productRequest = productRequest
}

extension Purchases: SKProductsRequestDelegate {
    func productsRequest(_ request: SKProductsRequest, didReceive response: SKProductsResponse) {
        guard !response.products.isEmpty else {
            print("Found 0 products")
            return
        }

        for product in response.products {
            print("Found product: \(product.productIdentifier)")
        }
    }

    func request(_ request: SKRequest, didFailWithError error: Error) {
        print("Failed to load products with error:\n \(error)")
    }
}
