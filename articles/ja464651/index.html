パート5.マルチスレッドアプリケーションの設計


記事の前の部分では、ライブラリーを使用したプログラミングの基本について詳しく説明しました。前のパートでは、割り込みの実装と、割り込みを使用するときに発生する可能性のある制限について説明しました。投稿のこの部分では、Parallelクラスを使用して並列プロセスをプログラミングするための可能なオプションの1つについて詳しく説明します。このクラスを使用すると、複数の独立したプログラムフローでデータを処理する必要があるアプリケーションの作成を簡略化できます。


シングルコアシステムのすべてのマルチタスクシステムは互いに類似しています。それらのマルチスレッド化は、各スレッドにタイムスロットを割り当てるディスパッチャーの作業により実装され、終了すると、制御を取り、次のスレッドに制御を渡します。さまざまな実装の違いは詳細にのみあるため、この実装の特定の機能を中心に詳しく説明します。


スレッドでのプロセス実行の単位はタスクです。システムには無制限の数のタスクを存在させることができますが、ディスパッチャ内のワークフローの数によって制限されている特定の数のタスクのみをアクティブ化できます。この実装では、ワークフローの数はディスパッチャーコンストラクターで設定され、後で変更することはできません。その過程で、スレッドはタスクを実行したり、空き状態を維持したりできます。他のソリューションとは異なり、Parallel Managerタスクを切り替えません。タスクがディスパッチャに制御を戻すには、適切なコマンドをコードに挿入する必要があります。したがって、タスクのタイムスロットの期間の責任はプログラマにあります。プログラマは、タスクに時間がかかりすぎる場合にコードの特定の場所に割り込みコマンドを挿入し、タスクの完了時のスレッドの動作を決定する必要があります。このアプローチの利点は、プログラマーがタスク間の切り替えポイントを制御することです。これにより、タスクの切り替え時に保存/復元コードを大幅に最適化し、スレッドセーフなデータアクセスに関連するほとんどの問題を取り除くことができます。

            var m = new Mega328();
            m.FCLK = 16000000;
            m.CKDIV8 = false;
            var cData = m.DREG();
            var outDigit = m.ARRAY(4);
            var chex = Const.String("0123456789ABCDEF");
            m.ADC.Clock = eADCPrescaler.S64;
            m.ADC.ADCReserved = 0x01;
            m.ADC.Source = eASource.ADC0;
            m.Usart.Baudrate = 9600;
            m.Usart.FrameFormat = eUartFrame.U8N1;

            var os = new Parallel(m, 4);
            os.Heap = new StaticHeap(os, 8);
            var ADS = os.AddSignal(m.ADC.Handler, () => m.ADC.Data(cData));
            var trm = os.AddSignal(m.Usart.TXC_Handler);
            var starts = os.AddLocker();

            os.PrepareSignals();
            var t0 = os.CreateTask((tsk) =>
            {
                m.LOOP(m.TempL, (r, l) => m.GO(l), (r, l) =>
                   {
                       m.ADC.ConvertAsync();
                       tsk.Delay(500);
                   });

            }, "activate");
            var t1 = os.CreateTask((tsk) =>
            {
                var loop = AVRASM.NewLabel();
                var mref = m.ROMPTR();
                mref.Load(chex);
                m.TempL.Load(cData.High);
                m.TempL >>= 4;
                mref += m.TempL;
                mref.MLoad(m.TempL);
                m.TempL.MStore(outDigit[0]);
                mref.Load(chex);
                m.TempL.Load(cData.High);
                m.TempL &= 0x0F;
                mref += m.TempL;
                mref.MLoad(m.TempL);
                m.TempL.MStore(outDigit[1]);
                mref.Load(chex);
                m.TempL.Load(cData.Low);
                m.TempL >>= 4;
                mref += m.TempL;
                mref.MLoad(m.TempL);
                m.TempL.MStore(outDigit[2]);
                mref.Load(chex);
                m.TempL.Load(cData.Low);
                m.TempL &= 0x0F;
                mref += m.TempL;
                mref.MLoad(m.TempL);
                m.TempL.MStore(outDigit[3]);
                starts.Set();
                tsk.TaskContinue(loop);
            });
            var t2 = os.CreateTask((tsk) =>
            {
                var loop = AVRASM.NewLabel();
                trm.Clear();
                m.TempL.Load('0');
                m.Usart.Transmit(m.TempL);
                tsk.AWAIT(trm);
                m.TempL.Load('x');
                m.Usart.Transmit(m.TempL);
                tsk.AWAIT();
                m.TempL.MLoad(outDigit[0]);
                m.Usart.Transmit(m.TempL);
                tsk.AWAIT();
                m.TempL.MLoad(outDigit[1]);
                m.Usart.Transmit(m.TempL);
                tsk.AWAIT();
                m.TempL.MLoad(outDigit[2]);
                m.Usart.Transmit(m.TempL);
                tsk.AWAIT();
                m.TempL.MLoad(outDigit[3]);
                m.Usart.Transmit(m.TempL);
                tsk.AWAIT();
                m.TempL.Load(13);
                m.Usart.Transmit(m.TempL);
                tsk.AWAIT();
                m.TempL.Load(10);
                m.Usart.Transmit(m.TempL);
                tsk.TaskContinue(loop, starts);
            });
            var p = os.ContinuousActivate(os.AlwaysOn, t0);
            os.ActivateNext(p, ADS, t1);
            os.ActivateNext(p, starts, t2);
            m.ADC.Activate();
            m.Usart.Activate();
            m.EnableInterrupt();
            os.Loop();
