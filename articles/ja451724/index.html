最も興味深い特徴は、これらの極性スキルミオンの泡が磁気スキルミオンの電気的類似物であって、磁気スキルミオン自体ではないという事実です。極スキルミオンをよく研究すればするほど、それらを操作するためのツールが増えるため、キラリティーや負の静電容量を制御するためのツールも増えます。

スピントロニクスとスキルミオニクスの分野の研究は、正確に磁気スキルミオンを積極的に検討しています。しかし、この栄誉の極スキルミオンは授与されませんでした。

私たちが今日検討している発見に、科学者はどのようにして来ましたか？実際のところ、複雑なトポロジー構造は、そこに発生する現象やエキゾチックなフェーズを研究および検索するのに最適な場所です。科学者たちは、エピタキシャル制限を変更することにより、チタン酸鉛（PbTiO 3）の層で、チタン酸ストロンチウム（SrTiO 3）の層で両面がコーティングされた、つまりヘテロ構造（PbTiO 3）n /（SrTiO 3）nで、室温で極性スキルミオンの気泡を見つけました。

この研究は、強誘電体でナノドメインの気泡とスキルミオンのようなトポロジー構造を得ることが非常に可能であるという理論に基づいています* 弾性、静電および勾配エネルギーの相互作用による。
強誘電性（または強誘電性）*は、外部電場がなくても、特定の温度で結晶内に自発分極が発生することです。

そして、強弾性は単結晶物質と呼ばれ、その結晶格子は温度と相転移の低下に伴って自然に変形する可能性があります。
研究用サンプル-[（PbTiO 3）n /（SrTiO 3）n ] m（n = 12–20、m = 1–8）-チタン酸ストロンチウム（SrTiO 3）からパルスレーザースパッタリングを組み合わせて単結晶基板上に作製高速電子の回折。



RSMマッピングによるトポロジー（上記）の分析により、強度分布を伴うリング形成（画像bおよびd）が明らかになりました。相互空間におけるそのようなトポロジー現象の兆候は、強誘電性ナノドメインによるものである可能性があります。

また、RSM分析により、3層構造とSrTiO 3基板のトポロジーが一致することが示されました、リングのサイズと（g、h）平面でのリングの幅はほぼ同じです。この場合、以前にDyScO 3（e、f）で観察された周期的な渦のナノドメインからの強い違いが観察されます。

3層のSEM画像（透過型走査電子顕微鏡）により、円形（直径約8 nm）と細長い要素（1c）の混合物の存在を確認することができました。しかし、超格子では、円形要素（1d）の配列のみが優勢でした。

科学者は、磁壁が多くの平面方向に沿ってより近い順序の多くの領域に分割されているため、このようなトポロジは非常に珍しいと信じており、これにより、明確な円形の細長い要素が生まれます。以前は、これは観察されませんでした。

インセット上（1と1d）高速フーリエ変換後のX線画像が表示されます。これは、回転対称性を持つ、つまり偏波の非Yelike回転を持つ4つのローブを明確に示しています。

次に、科学者は一連の数学的計算を行って、観測されたトポロジーが磁気スキルミオンに非常に類似しており、正確で定義可能なスキルミオン数があることを確認しました。計算により、局所的な電気双極子は、PbTiO 3とSrTiO 3の境界（2aおよび2b）で平面内外を常に回転していることがわかりました。

PbTiO 3層の上部と下部双極子は、上部領域を発散分極で接続し、下部極性領域を収束で接続する平面分極を持っています（2c、2e）。ツイスト分極の圧縮は、PbTiO 3層の中央のx-y平面でも観察されます（2b、2d）。

PbTiO 3 / SrTiO 3（2c、2e）の上部と下部の界面における偏光テクスチャの分析により、ハリネズミ型のスキルミオンを明らかにすることが可能になりましたが、らせん型のスキルミオン（2d）がPbTiO 3層の内部で見つかりました。

科学者達はまた、局所秩序パラメーターがゼロ回転であるが、平面（2c）で正の発散（ベクトルは外側に向いている）である偏光テクスチャーを決定しました。下の平面では、反対の状況が発生します（2e）。負の発散があります（ベクトルは内向きです）。中間層では、平面偏光成分は、通常の螺旋スキルミオンのように、磁壁に平行な成分によってのみ現れます（2d）。

これらの観察から、科学者たちは、極性スカーミオンの三次元の気泡がフィルムの法線に沿って二次元スカーミオンの進化の一種であると結論：上からのPbTiOの底部に3層、それぞれスキルミオン「ハリネズミ」かららせん状に、そして再びハリネズミに。これらのトポロジー構造は、その進化にもかかわらず、継続的な変形により互いに変換できるため、同等のままです。

異常な特性を持つ異常なサンプルの研究の次のステップは、スキルミオン泡の原子構造の研究でした。このため、偏光マッピングが実行されました（画像No. 3）。

先に取得した画像でのチタンの変位ベクトルのマッピングは、1つのスキルミオンの泡の周囲に微視的な変位場が存在することを示していました。ベクトルマップ（3a）これらのデータに対応して、チタンの逆バイアスがエッジから中心に移動する領域を示します。これは、「ヘッジホッグ」タイプのスキルミオンの構造に似ています。

断面図（3b）では、逆平行（上下）の分極を持つ円柱状の極性領域がはっきりと見えます。偏光ベクトルは、PbTiO 3 / SrTiO 3界面付近の境界で回転します。これは、画像3aで観察された偏光の発散と一致しています。両方のタイプの画像（2次元および断面を含む）を組み合わせることにより、科学者たちはPbTiO 3層の上部に同様の構造を明らかにしました。

超格子の4D-PEMR分析による[（PbTiO 3）16 /（SrTiO 3）16 ] 8確率ストリーム（3e）を使用して、画像（3d）と極座標マップを取得しました。観察結果を確認するために、科学者たちは画像2に示すモデルの構造を通過するビームの伝播のシミュレーションを行いました。シミュレーション結果は3fと3gに示されています。

これらの観測の全体は、極性スキルミオン構造が形成されていることを示しており、その成分はハリネズミ構造に類似しており、偏光の方向はスキルミオンの中心から端に向かって上から下に回転します。極構造の上部と下部の分極の発散および収束回転は、強磁性体のネールスキルミオンのスピン配置に似ています。科学者によれば、そのような構造は、キラリティーやゼロスキルミオン数などの非標準的な特性を持つ必要があります。

超格子[（PbTiO 3）16 /（SrTiO 3）16 ] 8からのX線放射の共鳴回折が行われた理由を検証する必要がありました。

シミュレーションされた構造（画像No. 2）によると、各PbTiO 3層の中心面の任意の方向に沿った仮想線はブロッホ磁壁のみと交差し、その結果、局所分極はこの線に沿ってらせん状の回転を示します。同様のスパイラル偏光テクスチャは、極渦配列で発生する構成とほとんど同じです。したがって、これはキラル電子構造から同様の二色性信号を導くはずです。

RSMの調査では、2つの順序セットの存在が示されました（4a）。まず、平面の外側の方向に沿って、超格子の平面の外側の周期性（約12 nm）に関連する規則的なピークがあります。 2番目-衛星ピークがあります（そのうちの1つは4bにあります））垂直または平面方向で、極性構造の平面配列（約8 nm）に対応します。 X線エネルギーはチタンL 3およびL 2の吸収エッジを通じて調整されるため、衛星回折ピークの強度は、特に異方性チタン軌道のカイラル特性に対する周期的な歪みに敏感です。

右と左の円偏光（4c）でスペクトルを測定し、バックグラウンド蛍光（4d）を分離し、得られた2つのスペクトルの違い（4e）を考慮して、X線円偏光二色性を測定しました。 L 3の端に顕著な二色性の存在キラル構造の存在を示します。この場合、超格子と三層の両方で円二色性が観測された。

科学者は、観測された円偏光二色性の非磁性を、電荷四重極モーメントのカイラル構成によって説明します。

上記の測定に加えて、科学者はまた、側方散乱ベクトルの関数として共鳴散乱強度と円二色性をチェックしました（4f）。理論的には、平面に沿った反転対称性に違反しているため、ブロッホ構造コンポーネントの存在はフリーデルの法則に違反するはずです。鏡面回折ピークの強度で観測された非対称性がスキルミオン泡のブロッホ偏光成分の証拠であるのはこのためです。

すべての観測、計算、および測定を要約すると、極性のスキルミオンの気泡は、調査されたヘテロ構造に存在すると自信を持って言えます。

科学者たちは、このような3次元スキルミオン構造は、よく知られている2次元構造とは非常に異なることに注意しています。スキルミオンの泡は電界を使用して移動できます。これにより、以前は不可能であったより優れた制御と同様の構造を使用できるようになります。
