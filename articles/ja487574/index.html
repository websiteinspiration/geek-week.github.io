オーディオの操作：進行状況とデータの視覚化



こんにちは！

経験をオーディオと共有したい。「オーディオ」とは、HTMLAudioElementおよびWeb Audio APIを意味します。

私たちは何をしますか？

1つのトラックのプレーヤーのようなものを作成します（本格的なプレーヤーについて-次のいずれかの記事で）。

条件：

  • ボタンをクリックするか、ドラッグアンドドロップすることにより、ハードドライブの任意の場所からファイルをダウンロードする機能。
  • 円形のグラフィカルおよびテキストの進行状況インジケータ。
  • 音量のテキストインジケータ。
  • オーディオデータの視覚化。
  • キーボードを使用してプレーヤーを制御します。

ネットワークはHTMLAudioElementとWAAPIの両方の資料でいっぱいなので、実用的なコンポーネントに焦点を当てます。オーディオに加えて、ドラッグドロップとキャンバスも使用します。

プレーヤーを制御する機能を追加します（ボタンがないため、キーボードを使用するだけでプレーヤーを制御できます）。

document.addEventListener("keydown", e => {
    //   try/catch    Chrome,     
    //   try/catch       (console.log(audio.volume))   
    //    0  1 (       0  1)   ,     if(audio.volume>0 && audio.volume<1)
    //    ""   0.1  0.9   ,       
    //     ,  
    try {
        //    
        e.preventDefault()

        // 
        if (e.keyCode == 32) {
            // /
            audio.paused ? audio.play() : audio.pause();
        // enter
        } else if (e.keyCode == 13) {
            // 
            audio.load();
        //  
        } else if (e.keyCode == 39) {
            //   + 10 
            audio.currentTime += 10;
        //  
        } else if (e.keyCode == 37) {
            //   - 10 
            audio.currentTime -= 10;
        //  
        } else if (e.keyCode == 40) {
            //   - 10%
            audio.volume -= 0.1;
        //  
        } else if (e.keyCode == 38) {
            //   + 10%
            audio.volume += 0.1;
        }
    //  
    } catch {
        return;
    }
});

関数自体は次のとおりです。

function startAnimation() {
    //  
    C.style.display = "block";

    //    (   /  )
    piece = audio.currentTime / audio.duration;

    //   
    //     :   ,    
    radius = 105;
    
    //  
    $.clearRect(0, 0, W, H);
    
    //   
    $.beginPath();
    $.arc(centerX, centerY, radius, 0, Math.PI * (2 * piece));
    $.lineWidth = 30;
    $.stroke();
    
    //   
    volume.innerText = Math.trunc(audio.volume * 100) + "%";
    
    //   
    progress.innerText = Math.trunc(piece * 100) + "%";

    //      frequencyArray
    analyser.getByteFrequencyData(frequencyArray);
    //     
    for (let i = 0; i < bars; i++) {
        //  
        radius = 120;

        //     
        rads = Math.PI * 2 / bars;

        //   
        barHeight = frequencyArray[i] * 0.6;
        
        //   0   
        x = centerX + Math.cos(rads * i) * radius;
        y = centerY + Math.sin(rads * i) * radius;
        xEnd = centerX + Math.cos(rads * i) * (radius + barHeight);
        yEnd = centerY + Math.sin(rads * i) * (radius + barHeight);
        
        //  
        drawBar(x, y, xEnd, yEnd, barWidth, frequencyArray[i]);
    }

    //  
    requestAnimationFrame(startAnimation);
}
