Plugin pour la surveillance des applications K8s DevOpsProdigy KubeGraf v1.3.0: nouvelle version et nouvelles fonctionnalités

Cette semaine, la sortie d'une nouvelle version de notre plugin pour Grafana, conçue pour surveiller les applications kubernetes DevOpsProdigy KubeGraf v1.3.0, a eu lieu .

Petit avertissement: ce plugin est une version révisée du plugin officiel de Grafana (qui n'est pas pris en charge depuis environ deux ans maintenant). Nous le développons depuis plus de six mois. Les principales fonctionnalités du plugin sont:

  • intégration avec k8s-api pour construire une carte de vos applications, regroupées par espace de noms / nœud de cluster + liaison à des pods / services spécifiques;
  • Une page récapitulative avec toutes les erreurs / avertissements concernant le fonctionnement des nœuds et des applications de cluster;
  • la possibilité d'installer le plugin avec les fournisseurs de cloud k8s via une autorisation utilisant bearer-tokena.

Dans le processus de préparation de cette version, nous avons été confrontés à une situation où vous devez mettre à jour un plugin vers une version spécifique, mais cette version du plugin n'est pas encore dans le magasin Grafana officiel (par exemple, vous voulez construire un plugin à partir d'une balise spécifique dans un référentiel git). Pour résoudre ce problème, nous avons également développé un graphique de barre , avec lequel il est possible d'installer / mettre à jour n'importe quel plugin Grafana directement à partir du référentiel git avec la balise dont vous avez besoin.
