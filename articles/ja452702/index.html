コードのテストとテストのコード

PythonやJavaScriptなどの動的言語では、操作中にモジュールのメソッドとクラスを直接置き換えることができます。これはテストにとって非常に便利です。このテストのコンテキストで、重いまたは不要なロジックを除外する「パッチ」を置くだけです。


しかし、C ++で何をすべきか？行く？Java？これらの言語では、その場でテストするためにコードを変更することはできません。パッチを作成するには、別のツールが必要です。


このような場合は、テストするようにコードを具体的に記述する必要があります。これは、プロジェクトを100％カバーしたいというだけではありません。これは、サポートされた高品質のコードを書くためのステップです。


この記事では、テスト可能なコードの作成の背後にある主なアイデアについて話し、それらを簡単なgoプログラムの例でどのように使用できるかを示します。


複雑でないプログラム


VK APIにリクエストを送信する簡単なプログラムを作成します。これは、要求を生成し、それを作成し、応答を読み取り、JSONからの応答を構造にデコードしてユーザーに結果を表示する、かなり単純なプログラムです。


package main

import (
  "encoding/json"
  "fmt"
  "io/ioutil"
  "net/http"
  "net/url"

const token = "token here"

func main() {
  //    
  var requestURL = fmt.Sprintf(
    "https://api.vk.com/method/%s?&access_token=%s&v=5.95",
    "users.get",
    token,
  )

  //  
  resp, err := http.PostForm(requestURL, nil)

  //  
  if err != nil {
    fmt.Println(err)
    return
  }

  //      
  defer resp.Body.Close()

  //    
  body, err := ioutil.ReadAll(resp.Body)

  //  
  if err != nil {
    fmt.Println(err)
    return
  }

  //     
  var result struct {
    Response []struct {
      ID        int    `json:"id"`
      FirstName string `json:"first_name"`
      LastName  string `json:"last_name"`
    } `json:"response"`
  }

  //       
  err = json.Unmarshal(body, &result)

  //  
  if err != nil {
    fmt.Println(err)
    return
  }

  // ,   
  if len(result.Response) < 1 {
    fmt.Println("No values in response array")
    return
  }

  //   
  fmt.Printf(
    "Your id: %d\nYour full name: %s %s\n",
    result.Response[0].ID,
    result.Response[0].FirstName,
    result.Response[0].LastName,
  )
}

type fakeNetworker struct {
  ErrorOnPostForm  bool
  ErrorOnBodyRead  bool
  ErrorOnUnmarchal bool
  RawBody          string
}

func (fn *fakeNetworker) PostForm(string, url.Values) (*http.Response, error) {
  if fn.ErrorOnPostForm {
    return nil, fmt.Errorf("Error on PostForm")
  }

  if fn.ErrorOnBodyRead {
    return &http.Response{Body: ioutil.NopCloser(fakeReader{})}, nil
  }

  if fn.ErrorOnUnmarchal {
    fakeBody := ioutil.NopCloser(bytes.NewBufferString(""))

    return &http.Response{Body: fakeBody}, nil
  }

  fakeBody := ioutil.NopCloser(bytes.NewBufferString(fn.RawBody))

  return &http.Response{Body: fakeBody}, nil
}
