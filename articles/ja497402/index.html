私たちは何をしていますか：

  1. osおよびsysライブラリをインポートする
  2. osを使用して、コンパイラーの起動元を決定します（将来的には、その理由をお伝えします）。
  3. ファイルへのパスを作成+引数からファイル自体を取得-sys.argvから取得
  4. 次に、パスをファイルとともに表示します（将来的には削除する必要があります）
  5. すべての予約語を作成します-今のところは必要ありませんが、将来的にはエラー出力用に用意しました
  6. レクサーと変数のリストを作成する
  7. レクサー自体を作成します。これを行うには、ファイルを読み取りモードで開きます。
    エンコーディング= "UTF-8"
    これを行わないと、ロシア語の文字の代わりに象形文字が表示されます。
  8. 読み取り（）でのexの使用。すべてのテキストを単語に分割し、各単語を別々にexに配置します。
  9. 次に、変数リストに各単語を書き込みます
  10. そしてリスト自体を表示します-将来的には出力を削除する必要があります：/

2.コンパイラーの作成


コンパイラを作成しましょう！
これを行うには、次のようになるようにファイルにコードを追加します。

import sys #  sys
import os #  os

dir = os.path.abspath(os.curdir) #   
p = str(dir) + '\\' + str(sys.argv[1]) #sys.argv[1] -  1 
print(p) #   +  
modules = ['if', 'else', 'while', 'for', '=', '==', '{', '}', '[', ']', '(', ')', '//'] #    
var = [] #   
vars_ = {} #   
try:
    with open(p, 'r', encoding="UTF-8") as f: #   
        for ex in f.read().split(): #  
            var.append(ex) #     var
    print(var) # $   $
    a = -1 #     var
    for i in var: #  
        a = a + 1 #    
        if i == '=': #    "="
            vars_[var[a-1]] = var[a+1] #  vars_      "="
        if i == 'rprint':
            let = var[a+1]
            for key, value in vars_.items():
                if key == let:
                    print(value)
except FileNotFoundError:
    print('Error!   !')

何してるの？

varのすべての値を反復処理し、1つまたは別の予約語が見つかった場合、特定のアクションを実行します

変数の割り当て


「=」と一致する場合は、vars_リストに「=」の前、「=」の後に値を追加して、変数を形成します

可変出力


iが「rprint」に一致する場合、vars_リストでこの変数との一致を探し、これを友人のvarリストから取得します。
検索後、変数の値を表示します

4.インストーラーの作成

import os

with open('path\\xnn.exe', 'r', encoding='UTF-8') as f:
	lean = f.read()

	directory_folder = r"C:\\Windows\\System32\\xnn.exe"
	folder_path = os.path.dirname(directory_folder) #

	    os.makedirs(folder_path)
