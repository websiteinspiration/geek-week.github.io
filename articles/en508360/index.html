Python and beer - shake but don't mix





Hi Habr! I decided to describe a small automation system to create a wonderful drink. In order for a beer to be tasty, time and temperature, temperature and time are important. Moreover, they are just as necessary as the ingredients that make up the wort. To get to the finished product, it takes from 14 to 20 steps with different temperatures and brewing times. For anyone interested in this mysterious process, I ask under cat.



We will need two vats of different volumes, for three types:



  • wort ale
  • Wheat Wort
  • Pilsner


Each type goes through correspondingly different steps in time and temperature. Of course, one could use the instruction and the stopwatch, but it is terribly inconvenient. Therefore, we will compile a csv table in which we will store: step number, volume, target temperature, time.



Next, take rasbpberry pi zero and attach two DS18B20 temperature sensors to it .

Each of the sensors will allow you to track the temperature in the corresponding tank. In order to automate the process of counting time, install adafruit 128x64 oled on the raspberry , on the display of which all the necessary information will be displayed.

Buttons, we can control the passage through the steps of the table, as well as start the timer.



Timer code 
def timer_code(seconds, t_counter = 2, restart = False):
    time.sleep(0.023)
    seconds -= time.time() - t_counter
    t_counter = time.time()
    minutes = trunc(seconds / 60.0)
    return seconds, minutes, t_counter
