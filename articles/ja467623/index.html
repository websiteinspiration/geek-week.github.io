私たちは何を達成しようとしていますか？


私たちが扱っているものには2つのオプションがあります。
1つ目は、入力フィールドにフォーカスがない場合です。



碑文が入力フィールド内に表示され、そのサイズはテキストフィールドのサイズと同じです。色がくすんでいる。これはプロパティに非常に似ていますplaceholder。

2番目のオプションは、入力フィールドにフォーカスがある場合です。



碑文は入力フィールドの上に表示され、そのサイズは小さく、色は入力したテキストの色とは異なります。

最も単純な実装


最後に仕事に取り掛かることができます。これまでのところ、アニメーションはありません。
結局のところ、2つのUI状態があります。

  1. フィールドとフィールド内の碑文に焦点はありません。
  2. 入力フィールドの上の碑文であるフィールドに焦点があります。



実際、フィールドにフォーカスがあるかどうかの状態を保存できます。次に、この状態に応じて、碑文を配置する場所とそれに適用するスタイルを選択できます。

碑文は別の場所に配置する必要があり、コンポーネントの配置に影響を与えたくないので、絶対に配置します。十分なスペースがあることを確認するには、上から折り返しビューにインデントを追加する必要があります。

class FloatingLabelInput extends Component {
  state = {
    isFocused: false,
  };

  handleFocus = () => this.setState({ isFocused: true });
  handleBlur = () => this.setState({ isFocused: false });

  render() {
    const { label, ...props } = this.props;
    const { isFocused } = this.state;
    const labelStyle = {
      position: 'absolute',
      left: 0,
      top: !isFocused ? 18 : 0,
      fontSize: !isFocused ? 20 : 14,
      color: !isFocused ? '#aaa' : '#000',
    };
    return (
      <View style={{ paddingTop: 18 }}>
        <Text style={labelStyle}>
          {label}
        </Text>
        <TextInput
          {...props}
          style={{ height: 26, fontSize: 20, color: '#000', borderBottomWidth: 1, borderBottomColor: '#555' }}
          onFocus={this.handleFocus}
          onBlur={this.handleBlur}
        />
      </View>
    );
  }
}

なぜ使用しないのplaceholderですか？


もちろん、TextInputplaceholder プロパティを使用するのは魅力的です。ただし、碑文を表示する方法、タイミング、場所を制御する必要があるため、これは機能しません。

代わりに、フォーカスが置かれているときにラベルをテキストボックス内に配置します。そして、フォーカスが入力フィールドに表示されたときに上に移動したいと考えています。これは、同じ要素を操作する場合にのみ達成できます。

アニメーションはどうですか？


実は一番簡単な部分が残っていました。

碑文があるかもしれない2つの状態があり、フォーカスに応じて1つを選択するため、状態間のこの遷移のアニメーションは非常に簡単です。

このガイドから、以下を強調できます。

  • Animated.Value フィールドにフォーカスがあるか（1）ないか（0）を意味します。
  • フォーカスする場合は（1）に、それ以外の場合は（0）に番号を徐々に変更します。
  • この番号の形式で、碑文のスタイルを反映します。（0）と（1）でスタイルを定義し、React Nativeが自動的に中間スタイルを計算して適用します。そして色も。

これらすべてを実装することは難しくありません。

Animated.Valueに初期化する必要がありますcomponentWillMount。

componentWillMount() {
  this._animatedIsFocused = new Animated.Value(0);
}

次に、この数値の値は、入力フィールドにフォーカスがあるかどうかに基づく必要があり、ステータス情報のこのビットがすでにあるため、componentDidUpdateこの数値を次のように変更する関数を追加できますthis.state。

componentDidUpdate() {
  Animated.timing(this._animatedIsFocused, {
    toValue: this.state.isFocused ? 1 : 0,
    duration: 200,
  }).start();
}

：さて、このような用語で碑文のスタイルを反映するために、我々は2つだけ変更を加える必要がある

の変更をAnimated.Text。

条件を使用してスタイルを定義する代わりに、次のように定義します。

const labelStyle = {
  position: 'absolute',
  left: 0,
  top: this._animatedIsFocused.interpolate({
    inputRange: [0, 1],
    outputRange: [18, 0],
  }),
  fontSize: this._animatedIsFocused.interpolate({
    inputRange: [0, 1],
    outputRange: [20, 14],
  }),
  color: this._animatedIsFocused.interpolate({
    inputRange: [0, 1],
    outputRange: ['#aaa', '#000'],
  }),
};
