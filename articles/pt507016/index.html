A maneira mais fácil de atualizar do macOS para o Linux

O Linux permite que você faça quase a mesma coisa que o macOS. E mais: tornou-se possível graças à comunidade de código aberto desenvolvida.

Uma das histórias de transição do macOS para o Linux nesta tradução.


Quase dois anos se passaram desde que mudei do macOS para o Linux. Antes disso, eu usava o sistema operacional da Apple há 15 anos. Instalei minha primeira distribuição no verão de 2018. Então eu ainda era novo no Linux.

Agora eu uso exclusivamente Linux. Lá, eu posso fazer o que eu quiser: navegar e navegar na Internet regularmente na Netflix, escrever e editar conteúdo para o meu blog e até gerenciar uma startup.

É importante notar que eu não sou um desenvolvedor, não um engenheiro! Longe vão os dias em que se acreditava que o Linux não era adequado para usuários comuns, pois não possui uma interface amigável o suficiente.

Ultimamente, tem havido muita discussão sobre criticar o sistema operacional macOS; portanto, mais e mais pessoas estão pensando em mudar para o Linux. Compartilharei algumas dicas para migrar do macOS para o Linux para ajudar outras pessoas a fazer isso rapidamente e sem muita dor de cabeça.

Você precisa disso?


Antes de mudar do macOS para o Linux, seria bom pensar se o Linux é ideal para você. Se você deseja continuar sincronizando com o Apple Watch, fazer chamadas no FaceTime ou trabalhar no iMovie, não descarte o macOS. Estes são produtos proprietários que vivem no ecossistema fechado da Apple. Se você ama esse ecossistema, o Linux provavelmente não é adequado para você.

Eu não estava fortemente ligado ao ecossistema da Apple. Eu não tinha um iPhone, não usei o iCloud, FaceTime ou Siri. Eu tinha interesse em código aberto, restava apenas decidir e dar o primeiro passo.

Há alguma versão Linux do seu software favorito?


Comecei a estudar software de código aberto enquanto estava sentado no macOS e descobri que a maioria dos aplicativos que eu utilizo funcionará nas duas plataformas.

Aqui está o navegador Firefox, por exemplo, que funciona tanto no macOS quanto no Linux. Você já usou o VLC para reprodução de multimídia? No Linux, também funcionará. Utilizou o Audacity para gravar e editar áudio? Depois de mudar para o Linux, você pode levá-lo com você. Você transmitiu ao vivo no OBS Studio? Existe uma versão para Linux. Você usa o Telegram messenger? Você pode instalar o Telegram for Linux.

Isso se aplica não apenas ao software de código aberto. Os desenvolvedores da maioria (possivelmente até de todos) dos seus aplicativos proprietários não-Apple favoritos criaram versões do Linux: Spotify, Slack, Zoom, Steam, Discord, Skype, Chrome e muitos outros. Além disso, quase tudo o que você pode executar no seu navegador macOS pode funcionar no seu navegador Linux.

No entanto, é melhor verificar se existem versões Linux dos seus aplicativos favoritos. Ou talvez haja alternativas adequadas ou até mais interessantes para eles. Faça sua pesquisa: pesquise “seu aplicativo favorito + Linux” ou “seu aplicativo favorito + alternativas Linux” no mecanismo de pesquisa ou consulte Flathub aplicativos proprietários que você pode instalar no Linux usando o Flatpak.

Não se apresse em fazer o macOS "copiar" do Linux


Para se sentir confortável ao mudar para o Linux, você precisa mostrar flexibilidade suficiente e estar preparado para aprender as nuances do uso do novo sistema operacional. Para fazer isso, você precisa se dedicar um pouco de tempo.

Se você deseja que o Linux pareça e funcione exatamente como o macOS, é quase impossível. Em princípio, você pode criar um desktop Linux semelhante ao macOS, no entanto, na minha opinião, a melhor maneira de mudar para o Linux é começar com uma GUI mais padrão para Linux.

Dê a ela uma chance e use o Linux da maneira que foi originalmente planejado. Não tente transformar o Linux no que não é. E talvez você, como eu, goste de trabalhar no Linux muito mais do que no macOS.

Lembre-se de quando se sentou pela primeira vez em um Mac: você precisava de algum tempo para se acostumar. Portanto, no caso do Linux, também não se deve esperar um milagre.

Escolha a distribuição Linux correta


Ao contrário do Windows e do macOS, os sistemas baseados em Linux produzem sistemas operacionais muito diferentes. Eu usei e testei várias distribuições Linux. Eu também tentei vários desktops (ou GUIs de usuário). Eles são muito diferentes entre si em termos de estética, usabilidade, fluxo de trabalho e aplicativos incorporados.

Embora o ElementaryOS e o Pop! _OS geralmente atuem como alternativas para o macOS, recomendo iniciar o Fedora Workstation pelos seguintes motivos:

  • É fácil de instalar em uma unidade USB usando o Fedora Media Writer .
  • Ela pode reconhecer da caixa e trabalhar adequadamente com todo o seu hardware.
  • Ele suporta o mais recente software Linux.
  • Ele inicia o ambiente de área de trabalho GNOME sem nenhuma configuração adicional.
  • Ela tem uma grande comunidade e uma grande equipe de desenvolvimento.

Na minha opinião, o GNOME é o melhor ambiente de trabalho para Linux em termos de usabilidade, consistência, flexibilidade e usabilidade para quem migrou para o Linux com macOS.

O Fedora pode ser um ótimo começo para você e, quando se sentir confortável, tente outras distribuições, bem como ambientes de desktop e gerenciadores de janelas.

Conheça melhor o GNOME


O GNOME é instalado por padrão no Fedora e em muitas outras distribuições Linux. Sua recente atualização para o GNOME 3.36 trouxe uma estética moderna que os usuários de Mac apreciarão.

Esteja preparado para o fato de que o Linux, e até a Estação de Trabalho Fedora em combinação com o GNOME, ainda serão significativamente diferentes do macOS. O GNOME é muito limpo, minimalista, moderno. Não há distrações aqui. Não há ícones na área de trabalho e não há dock visível. Suas janelas nem têm os botões minimizar e maximizar. Mas não entre em pânico. Se você der uma chance, pode ser o melhor e mais produtivo sistema operacional que você já usou.

Quando você inicia o GNOME, você vê apenas o painel superior e a imagem de plano de fundo. O painel superior consiste no botão Atividades à esquerda, hora e data no centro, bem como ícones de bandeja para a rede, Bluetooth, VPN, som, brilho, carga da bateria (e assim por diante) no lado direito.

Como o GNOME é como o macOS


Você notará algumas semelhanças com o macOS, como encadernação de janelas e visualização de documentos pressionando a barra de espaço (funciona como o Quick Look).

Se você clicar em Atividades no painel superior ou pressionar a tecla Super (semelhante à tecla Apple) no teclado, verá algo semelhante ao MacOS Mission Control e Spotlight Search em uma garrafa. Assim, você pode visualizar informações sobre todos os aplicativos e janelas abertos. No lado esquerdo, você verá uma doca na qual todos os seus aplicativos favoritos (selecionados) são apresentados.

Na parte superior da tela, há uma caixa de pesquisa. Assim que você começar a digitar, seu foco estará nele. Assim, você pode iniciar uma pesquisa por aplicativos instalados e conteúdo de arquivo, encontrar aplicativos no Application Center, verificar a hora e o clima etc. Isso funciona da mesma forma que o Spotlight. Comece a digitar o que deseja encontrar e pressione Enter para abrir o aplicativo ou arquivo.

Você também pode ver uma lista de todos os aplicativos instalados (o mesmo que o Launchpad no Mac). Clique no ícone Mostrar aplicativos no dock ou no atalho de teclado Super + A.
O Linux geralmente funciona rápido o suficiente, mesmo em hardware mais antigo, e ocupa muito pouco espaço em disco comparado ao macOS. E, diferentemente do macOS, você pode remover todos os aplicativos pré-instalados que não são necessários.

Personalizar o GNOME


Revise suas configurações do GNOME para fazer alterações que possam torná-lo mais conveniente para você. Aqui estão algumas coisas que eu faço assim que instalo o GNOME:

  • No Mouse e no Touchpad, desativo a rolagem natural e ativo um clique no botão.
  • Em Monitores, acendo a luz noturna, o que torna a tela mais quente à noite para evitar fadiga ocular.
  • Também instalo o GNOME Tweaks para acessar configurações avançadas.
  • Nos ajustes, ative o ganho excessivo para que o som aumente o volume acima de 100%.
  • Nos ajustes, também incluo o tema Adwaita Dark, que prefiro o tema padrão da luz.

Lidar com teclas de atalho


O GNOME é orientado ao teclado, então tente usá-lo mais. Na seção Atalho de teclado das configurações do GNOME, você pode encontrar uma lista de diferentes atalhos de teclado.

Você também pode adicionar seus próprios atalhos de teclado. Configurei meus aplicativos usados ​​com mais freqüência para abrir com a tecla Super. Por exemplo, Super + B para o meu navegador, Super + F para arquivos, Super + T para o terminal e assim por diante. Também selecionei Ctrl + Q para fechar a janela atual.

Alterno entre aplicativos abertos usando o Super + Tab. E eu uso o Super + H para esconder a janela. Pressiono F11 para abrir o aplicativo em tela cheia. A seta Super + esquerda permite encaixar o aplicativo atual no lado esquerdo da tela. A seta Super + direita permite encaixá-lo no lado direito da tela. Etc.

Execute o Linux no modo de teste


Você pode tentar trabalhar com o Fedora no seu Mac antes de instalá-lo completamente. Basta baixar o arquivo de imagem ISO no site do Fedora . Monte o arquivo de imagem ISO em uma unidade USB usando o Etcher e inicialize a partir dessa unidade pressionando a tecla Opção na inicialização do computador para poder experimentar o sistema operacional ao vivo.

Agora você pode explorar com segurança a Estação de Trabalho Fedora sem instalar nada extra no seu Mac. Verifique como este sistema operacional funciona com seu hardware e rede: você pode se conectar ao WiFi? O touchpad funciona? E o áudio? Etc.

Passe algum tempo aprendendo o GNOME. Confira os vários recursos que descrevi acima. Abra alguns dos aplicativos instalados. Se tudo parecer bom, se você gosta da aparência da Estação de Trabalho do Fedora e do GNOME, pode concluir a instalação no seu Mac.

Bem-vindo ao mundo do Linux!
