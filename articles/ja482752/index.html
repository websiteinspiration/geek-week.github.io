「新しい叙事詩」。象を分けて食べる


この記事では、ゲーム「Epics」を開発するための作業環境をセットアップし、ゲームをOpenFaaSでの使用に適した部分に分割します。Linuxですべての操作を行います。KubernetesはVirtualBoxを使用してminikubeにデプロイされます。私の作業用マシンには2つのプロセッサコアと12GBのRAMがあり、システムディスクとしてSSDを使用しています。私は開発のメインシステムとしてdebian 8を使用します。emacs、sudo、git、virtualboxパッケージがインストールされています。残りはGitHubやその他のソースからダウンロードしてインストールします。特に指定のない限り、これらのアプリケーションを/ usr / local / binにインストールします。始めましょう！


作業環境の整備


インストールに行く


公式ウェブサイトの指示に従ってください：


$ curl -L0 https://dl.google.com/go/go1.13.5.linux-amd64.tar.gz -o go.tar.gz
$ sudo tar -C /usr/local -xzf go.tar.gz
$ echo 'export PATH=$PATH:/usr/local/go/bin' >> ~/.profile

パフォーマンスを確認します。


$ mkdir -p ~/go/src/hello && cd ~/go/src/hello
$ echo 'package main

import "fmt"

func main() {
fmt.Printf("hello, world\n")
}' > hello.go
$ go build
$ ./hello
hello, world
