Pythonでのファジー比較にFuzzyWuzzyライブラリを使用する。レーベンシュタイン距離（編集距離）

彼は彼が正しく聞いたことを確信していませんでした。それにとても依存していました！しかし、もう一度尋ねませんか？（c）ボリス・アクニン。全世界が劇場です。

前回の記事で触れた音声アシスタントの作業中に、最も美しいFuzzyWuzzyライブラリを皆さんと共有せざるを得ないことに気付きました。

一言で言えば、彼女のおかげで、何の問題もなくファジー文字列比較を行うことができます。

最初のステップ


開始するには、次の2つの手順を実行する必要があります：

/重要！Pythonバージョン2.7以降/

ステップ1.インストール。

コマンドラインを開き、次のように入力します。

pip install fuzzywuzzy

Enterキーを押します。
次に、同じ方法でpython-Levenshteinをインストールし、文字列マッチングを3〜10倍高速化します。

pip install python-Levenshtein

インストールが完了すると、ライブラリをインポートする準備が整います。

ステップ2.プロジェクトへのインポート。

from fuzzywuzzy import fuzz
from fuzzywuzzy import process

機能的


1.最も一般的な比較：

a = fuzz.ratio(' ', ' ')
print(a)
#  : 100

いくつかの文字を変更すると、出力の数値が異なります。

a = fuzz.ratio(' ', ' ')
print(a)
#  : 84

2.部分比較-2

行目全体でこのタイプの比較を行うと、最初の比較との一致が検索されます。次に例を示します。

a = fuzz.partial_ratio(' ', ' !')
print(a)
#  : 100

または

a = fuzz.partial_ratio(' ', ' ,  ')
print(a)
#  : 100

ただし、レジスタについて覚えておく必要があります。

a = fuzz.partial_ratio(' ', ' ,  ')
print(a) 

#  : 90


3.

トークンの比較1）トークンのソート比率の

単語は、大文字小文字の区別や順序に関係なく、互いに比較されます。

a = fuzz.token_sort_ratio('  ', '  ')
print(a)
#  : 100

a = fuzz.token_sort_ratio('  ', '   ')
print(a)
#  : 78

a = fuzz.token_sort_ratio('1 2   ', '1   2 ')
print(a)
#  : 100

2）トークンセットの比率
この比較は、過去とは異なり、違いが単語の繰り返しである場合、文字列と同じです。

a = fuzz.token_set_ratio('  ', '     ')
print(a)
#  : 100

4.高度な定期比較

多くの場合、大文字と小文字が区別され、句読点（文字列を分割しない）であるため、正確にWRatioを使用することをお勧めします。

a = fuzz.WRatio('  ', '!  !')
print(a)
#  : 100

a = fuzz.WRatio('  ', '!,  !')
print(a)
#  : 97

5.リスト

の操作行をリストの行と比較するには、プロセスモジュールを使用します。

city = ["", "-", "", "", "", "", "", "", "", "", ""]
a = process.extract("", city, limit=2)
#  limit     5
print(a)
#  : [('', 100), ('', 62)]

リストの最初の1つだけが必要な場合は、extractOneを使用することをお勧めします

city = ["", "-", "", "", "", "", "", "", "", "", ""]
a = process.extractOne("", city)
print(a)
#  : ('', 90)
