データベース充填


SQLiteは仮想テーブルを使用して全文検索を実装します。外見上、これらは通常のSQLiteテーブルのように見えますが、それらへのアクセスはすべてバックステージで行われます。

仮想テーブルを使用すると、検索を高速化できます。ただし、利点に加えて、欠点もあります。

  • 仮想テーブルにトリガーを作成することはできません。
  • 仮想テーブルに対してALTER TABLEおよびADD COLUMNコマンドを実行することはできません。
  • 仮想テーブルの各列にはインデックスが付けられます。つまり、検索に関与してはならないインデックス付けの列でリソースが浪費される可能性があります。

後者の問題を解決するには、情報の一部を含む追加のテーブルを使用して、仮想テーブル内の通常のテーブルの要素へのリンクを保存できます。

テーブルの作成は標準のテーブルとは少し異なります。キーワードVIRTUALとfts4：

 CREATE VIRTUAL TABLE movies USING fts4(id, title, overview);

fun firstSearch(searchString: String): List<Movie> {
        val query = "SELECT * FROM movies WHERE movies MATCH '$searchString'"
        val cursor = readableDatabase.rawQuery(query, null)
        val result = mutableListOf<Movie>()

        cursor?.use {
            if (!cursor.moveToFirst()) return result

            while (!cursor.isAfterLast) {
                val id = cursor.getInt("id")
                val title = cursor.getString("title")
                val overview = cursor.getString("overview")

                result.add(Movie(id, title, overview))

                cursor.moveToNext()
            }
        }

        return result
    }

アクセントを追加


検索をわかりやすくするために、補助関数を使用しますSNIPPET。これは、一致が見つかったフォーマット済みテキストを表示するために使用されます。

snippet(movies, '<b>', '</b>', '...', 1, 15)

  • 映画-テーブル名;
  • <b＆gtおよび</ b>-これらの引数は、検索されたテキストの一部を強調表示するために使用されます。
  • ...-結果が不完全な値であった場合のテキストのデザイン。
  • 1-テキストの断片が割り当てられるテーブルの列番号。
  • 15は、返されるテキスト値に含まれる単語のおおよその数です。

仕上げの改善


SQLiteには、字句解析を実行して元の検索クエリを変換できる組み込みのトークンがあります。テーブルの作成時に特定のトークナイザーを指定しなかった場合、「シンプル」が選択されます。実際、データを小文字に変換し、読み取り不可能な文字を破棄するだけです。それは私たちにはまったく合いません。

検索を質的に向上させるには、ステミング（特定のソース単語の単語の基礎を見つけるプロセス）を使用する必要があります。

SQLiteには、Porter Stemmer Stemmingアルゴリズムを使用する追加のトークナイザーが組み込まれています。このアルゴリズムは、いくつかの特定の規則を順番に適用し、語尾と接尾辞を切り取って単語の重要な部分を強調表示します。たとえば、「keys」を検索すると、「key」、「keys」、「key」という単語が含まれる検索結果が得られます。最後に、アルゴリズムの詳細な説明へのリンクを残します。

残念ながら、SQLiteに組み込まれたトークナイザーは英語でのみ機能するため、ロシア語の場合、独自の実装を作成するか、既成の開発を使用する必要があります。完成した実装は、algorithmist.ruサイトから取得します。

検索クエリを必要な形式に変換します。

  1. 余分な文字を削除してください。
  2. フレーズを単語に分割します。
  3. ステマーをスキップします。
  4. 検索クエリで収集します。

ポーターアルゴリズム

 object Porter {

    private val PERFECTIVEGROUND = Pattern.compile("((|||||)|((<=[])(||)))$")

    private val REFLEXIVE = Pattern.compile("([])$")

    private val ADJECTIVE = Pattern.compile("(|||||||||||||||||||||||||)$")

    private val PARTICIPLE = Pattern.compile("((||)|((?<=[])(||||)))$")

    private val VERB = Pattern.compile("((||||||||||||||||||||||||||||)|((?<=[])(||||||||||||||||)))$")

    private val NOUN = Pattern.compile("(|||||||||||||||||||||||||||||||||||)$")

    private val RVRE = Pattern.compile("^(.*?[])(.*)$")

    private val DERIVATIONAL = Pattern.compile(".*[^]+[].*?$")

    private val DER = Pattern.compile("?$")

    private val SUPERLATIVE = Pattern.compile("(|)$")

    private val I = Pattern.compile("$")
    private val P = Pattern.compile("$")
    private val NN = Pattern.compile("$")

    fun stem(words: String): String {
        var word = words
        word = word.toLowerCase()
        word = word.replace('', '')
        val m = RVRE.matcher(word)

        if (m.matches()) {
            val pre = m.group(1)
            var rv = m.group(2)
            var temp = PERFECTIVEGROUND.matcher(rv).replaceFirst("")
            if (temp == rv) {
                rv = REFLEXIVE.matcher(rv).replaceFirst("")
                temp = ADJECTIVE.matcher(rv).replaceFirst("")
                if (temp != rv) {
                    rv = temp
                    rv = PARTICIPLE.matcher(rv).replaceFirst("")
                } else {
                    temp = VERB.matcher(rv).replaceFirst("")
                    if (temp == rv) {
                        rv = NOUN.matcher(rv).replaceFirst("")
                    } else {
                        rv = temp
                    }
                }
            } else {
                rv = temp
            }

            rv = I.matcher(rv).replaceFirst("")

            if (DERIVATIONAL.matcher(rv).matches()) {
                rv = DER.matcher(rv).replaceFirst("")
            }

            temp = P.matcher(rv).replaceFirst("")
            if (temp == rv) {
                rv = SUPERLATIVE.matcher(rv).replaceFirst("")
                rv = NN.matcher(rv).replaceFirst("")
            } else {
                rv = temp
            }
            word = pre + rv
        }

        return word
    }
}

フレーズを単語に分解するアルゴリズム

val words = searchString
                .replace("\"(\\[\"]|.*)?\"".toRegex(), " ")
                .split("[^\\p{Alpha}]+".toRegex())
                .filter { it.isNotBlank() }
                .map(Porter::stem)
                .filter { it.length > 2 }
                .joinToString(separator = " OR ", transform = { "$it*" })
