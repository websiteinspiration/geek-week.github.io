この出版物は時系列のウェーブレット分析について説明します。ウェーブレット変換の主なアイデアは、主な特性-調和成分の平均値、分散、周期、振幅、および位相の時間的変化を示す多くの時系列の詳細に対応しています。知識のさまざまな分野で研究されているプロセスの大部分は、上記の特徴を持っています。

本書の目的は ... PyWaveletsライブラリを使用して、時系列の連続ウェーブレット変換の方法を記述するためにある

歴史のビットを

XX世紀の70年代後半の地球物理学者D.モルレー。高周波成分（地震活動）が短時間で、低周波成分（地殻の穏やかな状態）が長期間含まれる地震センサーの信号を解析する際に問題が発生しました。ウィンドウフーリエ変換を使用すると、高周波成分または低周波成分のいずれかを分析できますが、両方の成分を同時に分析することはできません。

したがって、窓関数の幅が低周波数では増加し、高周波数では減少する分析方法が提案されました。新しいウィンドウ変換は、1つの生成（いわゆるスケーリング関数-スケーリング関数、scalet）関数の拡張（圧縮）と時間オフセットの結果として得られました。この生成関数は、D。Morlaixによってウェーブレットと呼ばれていました。

連続ウェーブレット変換（CWT）


単一レベルのウェーブレット変換：

coefs、frequency = pywt.cwt（data、scales、wavelet）

ここで、

data：（array_like） -入力信号;

スケール （array_like）：-使用するウェーブレットのスケール。比率f = scale2frequency（scale、wavelet）/ Sampling_periodを使用して、物理的な周波数fを決定できます。ここでは、sampling_periodが秒のときのヘルツのf。

ウェーブレット（ウェーブレットオブジェクトまたは名前）：-使用するウェーブレット。

サンプリング周期（float）：-出力周波数のサンプリング周期（オプションのパラメーター）。 coefsに対して計算された値は、sampling_periodから独立しています（つまり、スケールは期間サンプリングによってスケーリングされません）。

Coefs（array_like）：-連続ウェーブレット-指定されたスケールとウェーブレットの入力信号の変換。

周波数 （array_like）：-サンプリング周期の単位が秒で設定されている場合、周波数はヘルツで表示されます。それ以外の場合、サンプリング周期は1と見なされます

注：係数配列のサイズは、入力配列の長さと指定したスケールの長さに依存します。

import numpy as np
import pywt
import matplotlib.pyplot as plt
vav='cmor1.5-1.0'
wav = pywt.ContinuousWavelet(vav)
#  ,     
print("       [{}, {}]".format(   wav.lower_bound, wav.upper_bound))
width = wav.upper_bound - wav.lower_bound
scales = [1, 2, 3, 4, 10, 15]
max_len = int(np.max(scales)*width + 1)
t = np.arange(max_len)
fig, axes = plt.subplots(len(scales), 2, figsize=(12, 6))
for n, scale in enumerate(scales):
    #       cwt
    int_psi, x = pywt.integrate_wavelet(wav, precision=10)
    step = x[1] - x[0]
    j = np.floor(
        np.arange(scale * width + 1) / (scale * step))
    if np.max(j) >= np.size(int_psi):
        j = np.delete(j, np.where((j >= np.size(int_psi)))[0])
    j = j.astype(np.int)
    # normalize int_psi    
    int_psi /= np.abs(int_psi).max()
  #    
    filt = int_psi[j][::-1]
# CWT         
#          .
    nt = len(filt)
    t = np.linspace(-nt//2, nt//2, nt)
    axes[n, 0].plot(t, filt.real, t, filt.imag)
    axes[n, 0].set_xlim([-max_len//2, max_len//2])
    axes[n, 0].set_ylim([-1, 1])
    axes[n, 0].text(50, 0.35, 'scale = {}'.format(scale))
    f = np.linspace(-np.pi, np.pi, max_len)
    filt_fft = np.fft.fftshift(np.fft.fft(filt, n=max_len))
    filt_fft /= np.abs(filt_fft).max()
    axes[n, 1].plot(f, np.abs(filt_fft)**2)
    axes[n, 1].set_xlim([-np.pi, np.pi])
    axes[n, 1].set_ylim([0, 1])
    axes[n, 1].set_xticks([-np.pi, 0, np.pi])
    axes[n, 1].set_xticklabels([r'$-\pi$', '0', r'$\pi$'])
    axes[n, 1].grid(True, axis='x')
    axes[n, 1].text(np.pi/2, 0.5, 'scale = {}'.format(scale))
axes[n, 0].set_xlabel('time (samples)')
axes[n, 1].set_xlabel('frequency (radians)')
axes[0, 0].legend(['real', 'imaginary'], loc='upper left')
axes[0, 1].legend(['Power'], loc='upper left')
axes[0, 0].set_title('filter: wavelet - %s'%vav)
axes[0, 1].set_title(r'|FFT(filter)|$^2$')
plt.show()

次に、上記のプログラムを使用して、ウェーブレット関数とそのプロパティを検討します。

メキシカンハットウェーブレット "mexh"：

φ(t)=23π4⋅exp−t22⋅(1−t2)



«morl»:

φ(t)=exp−t22⋅cos(5t)

«cmorB-C»

ここで、Bは、スループットです。Cは中心周波数です。 以下（特別な指示なしで）、B、Cの値は浮動小数点で設定されます。ガウスウェーブレット「gausP」φ(t)=1πBexp−t2B⋅expj2πCt









ここで、P - 1〜8の整数ウェーブレットの名前に挿入され、例えば、 "gaus8"。C組み込みの正規化定数。統合ガウスウェーブレット「cgauP」φ(t)=C⋅exp−t2







ここで、P - 1〜8の整数が、ウェーブレットの名前に挿入され、例えば、ウェーブレット関数の導関数の次数を「gaus8」と対応します。C組み込みの正規化定数。シャノンウェーブレット「shanB-C」φ(t)=C⋅exp−t2⋅exp−jt







ここで：B -帯域幅。 Cは中心周波数です。 PyWaveletのCWTは、離散データ-ウェーブレット積分のサンプルとの畳み込みに適用されます。スケールが小さすぎる場合、ウェーブレット 'cmor1.5-1.0'のグラフに示すように、サンプリングが不十分な平滑化につながる離散フィルターが得られます。φ(t)=B⋅sin(πBt)πBT⋅expj2piCt







グラフの左側の列は、さまざまなスケールのたたみ込みで使用される離散フィルターを示しています。右の列は、各フィルターの対応するフーリエパワースペクトルです。グラフ 'cmor1.5-1.0'のスケール1および2を使用すると、ナイキスト制約の違反により平滑化が発生することがわかります。示されている現象は、CとBを選択したときに他のウェーブレットでも発生する可能性があるため、ウェーブレットを使用する場合は、プログラムを使用する必要があります-リスト1.

特定のスケールを特定の信号周波数に関連付けるには、信号のサンプリング周期を知る必要があります。pywt.scale2frequency（）関数の使用スケールのリストを対応する周波数に変換できます。スケールの正しい選択は、選択したウェーブレットによって異なるため、信号の対応する周波数範囲を把握するには、pywt.scale2frequency（）を使用する必要があります。

>>> import pywt
>>> dt = 0.01  # 100 Hz sampling
>>> frequencies = pywt.scale2frequency('cmor1.5-1.0', [1, 2, 3, 4]) / dt
>>> frequencies

配列（[100.、50.、33.33333333、25.]）

def get_fft_values(y_values, T, N, f_s):
    f_values = linspace(0.0, 1.0/(2.0*T), N//2)
    fft_values_ = fft(y_values)
    fft_values = 2.0/N * abs(fft_values_[0:N//2])
    return f_values, fft_values 
def plot_fft_plus_power(time, signal):
    dt = time[1] - time[0]
    N = len(signal)
    fs = 1/dt    
    fig, ax = subplots(figsize=(15, 3))
    variance = std(signal)**2
    f_values, fft_values = get_fft_values(signal, dt, N, fs)
    fft_power = variance * abs(fft_values) ** 2     # FFT power spectrum
    ax.plot(f_values, fft_values, 'r-', label='FFT ')
    ax.plot(f_values, fft_power, 'k--', linewidth=1, label=' ')
    ax.set_xlabel('[ / ]', fontsize=18)
    ax.set_ylabel('', fontsize=18)
    ax.legend()
    show()

from numpy import *
import pandas as pd
from pylab import *
import pywt
def plot_wavelet(time, signal, scales, 
                 waveletname = 'cmor1.0-0.4', 
                 cmap = plt.cm.seismic, 
                 title = '-( ) ', 
                 ylabel = ' ()', 
                 xlabel = ''):
    
    dt = time[1] - time[0]
    [coefficients, frequencies] = pywt.cwt(signal, scales, waveletname, dt)
    power = (abs(coefficients)) ** 2
    period = 1. / frequencies
    levels = [2**-4 ,  2**-3 ,  2**-2 ,  2**-1 ,  2**0 ,  2**1 ,  2**2 ,  2**3]
    contourlevels = log2(levels)
  
    
    fig, ax = subplots(figsize=(15, 10))
    im = ax.contourf(time, log2(period), log2(power), contourlevels, extend='both',cmap=cmap)
    
    ax.set_title(title, fontsize=20)
    ax.set_ylabel(ylabel, fontsize=18)
    ax.set_xlabel(xlabel, fontsize=18)
    
    yticks = 2**arange(np.ceil(log2(period.min())), ceil(log2(period.max())))
    ax.set_yticks(log2(yticks))
    ax.set_yticklabels(yticks)
    ax.invert_yaxis()
    ylim = ax.get_ylim()
    ax.set_ylim(ylim[0], -1)
    
    cbar_ax = fig.add_axes([0.95, 0.5, 0.03, 0.25])
    fig.colorbar(im, cax=cbar_ax, orientation="vertical")
    show()

スケールグラムモジュールの使用


スケールグラムは、PyWaveletsライブラリに基づく連続ウェーブレット変換で1Dデータを分析するための便利なツールです。このモジュールは、迅速なデータ分析のための信頼できるツールを提供するように設計されています。

スケールグラムには次の機能があります。

  • 初心者向けの簡単な通話署名
  • 読み取り可能な軸と純粋なmatplotlib統合
  • ズーム、スペクトルフィルター、カラーバーの実装などの多くのオプション...
  • マーキングに応じた周波数および周波数単位のサポート
  • 速度、変換にはN * log（N）アルゴリズムを使用
  • 移植性：python2.7およびpython3.7でテスト済み
  • 詳細なエラーメッセージと例を含むドキュメント
