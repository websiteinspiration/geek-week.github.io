SDL 2レッスン：レッスン6-プリミティブ

みなさん、こんにちは。これはSDL 2の6番目のレッスンで、少々出てきましたが、行ってみる価値があります。すべてのレッスンはここにあります。

このレッスンでは、プリミティブの描画を描画します。始めましょう。レッスンにようこそ

プリミティブ


まず、プリミティブとは何かを見てみましょう。グラフィックプリミティブは、表示画面に表示される最も単純な幾何学的オブジェクトです：ポイント、ラインセグメント、長方形、円弧、円など。SDL2では、ポイント、長方形、および線のみを描画できます。

コードに取り掛かりましょう

#include <SDL2/SDL.h>
#include <iostream>

using namespace std;

int SCREEN_WIDTH = 640;
int SCREEN_HEIGHT = 480;

SDL_Window *win = NULL;
SDL_Renderer *ren = NULL;

変数の宣言については説明しません。次に進みましょう。

bool init() {
    bool ok = true;

    if (SDL_Init(SDL_INIT_VIDEO) != 0) {
        cout << "Can't init SDL: " << SDL_GetError() << endl;
    }

    win = SDL_CreateWindow("", SDL_WINDOWPOS_UNDEFINED, SDL_WINDOWPOS_UNDEFINED, SCREEN_WIDTH, SCREEN_HEIGHT, SDL_WINDOW_SHOWN);
    if (win == NULL) {
        cout << "Can't create window: " << SDL_GetError() << endl;
        ok = false;
    }

    ren = SDL_CreateRenderer(win, -1, SDL_RENDERER_ACCELERATED);
    if (ren == NULL) {
        cout << "Can't create renderer: " << SDL_GetError() << endl;
        ok = false;
    }
    return ok;
}

で初期化、 SDL 2初期化し、ウィンドウを作成し、ここでは、レンダリング、あまりにも、リンガません。何もロードする必要がないため、Load

関数は記述していません。Quit 関数では、ウィンドウを削除し、SDL 2をレンダリングして初期化を解除するだけです。



void quit() {
    SDL_DestroyWindow(win);
    win = NULL;

    SDL_DestroyRenderer(ren);
    ren = NULL;

    SDL_Quit;
}

Init 関数の呼び出しをスキップして、最も興味深い部分に進みます。長方形を描画するには、SDL_Rect型のオブジェクトを作成する必要があります。描画には座標と寸法が必要です。SDL_Rect

について少し説明します。このデータ型は、x、y、w、hの4つの数値の配列です。つまり、このコード：SDL_Rect rect = {1、1、1、1}; -完全に正しい。

これでSDL_Rectに関する理論が完成しました。コードを書き始めましょう。

    SDL_SetRenderDrawColor(ren, 0x00, 0x00, 0x00, 0x00);
    SDL_RenderClear(ren);
    SDL_SetRenderDrawColor(ren, 0xFF, 0xFF, 0xFF, 0xFF);

    SDL_Rect rect1 = {10, 10, 50, 50};
    SDL_RenderFillRect(ren, &rect1);

ここでは、レンダーに黒でペイントしてウィンドウ全体で塗りつぶすように指示されました。その後、彼らは白く塗ると言いました。次に、座標（10; 10）と寸法50x50の長方形を作成しました。SDL_RenderFillRect関数は長方形を描画します。

四角形の輪郭のみを描画しても、それほど大きな違いはありません。

    SDL_Rect rect2 = {70, 10, 50, 50};
    SDL_RenderDrawRect(ren, &rect2);

長方形を作成し、SDL_RenderDrawRect関数を呼び出しました。画面に長方形の輪郭を描くだけです。

次のステップは、線を引くことです。

    SDL_RenderDrawLine(ren, 10, 70, 640 - 10, 70);

線を描くのに長方形は必要ありません。SDL_RenderDrawLine関数は、レンダリング値と4つの座標を受け入れます。これらは、始点と終点の座標です。横から10ピクセルずつインデントした横線を引くことにしました。

点の描画は、線の描画とほとんど同じです。SDL_RenderDrawPoint関数を呼び出し、レンダリングとポイントの座標を渡します。しかし、単に、ポイントは興味深いものではありませんLETの書き込みを描画するために、より良いれる3つの画素を、我々はポイントを描画します。

    for (int i = 10; i <= 640-10; i +=4 ) {
        SDL_RenderDrawPoint(ren, i, 90);
    }

水平方向の点線が表示されました。

この図面上で停止しました。画面を更新し、一時停止時間を設定し、終了して0を返すだけです。

    SDL_RenderPresent(ren);

    SDL_Delay(5000);

    quit();
    return 0;
}

これでレッスンは終わりです。完全なコードは次のとおりです。

#include <SDL2/SDL.h>
#include <iostream>

using namespace std;

int SCREEN_WIDTH = 640;
int SCREEN_HEIGHT = 480;

SDL_Window *win = NULL;
SDL_Renderer *ren = NULL;

bool init() {
    bool ok = true;

    if (SDL_Init(SDL_INIT_VIDEO) != 0) {
        cout << "Can't init SDL: " << SDL_GetError() << endl;
    }

    win = SDL_CreateWindow("", SDL_WINDOWPOS_UNDEFINED, SDL_WINDOWPOS_UNDEFINED, SCREEN_WIDTH, SCREEN_HEIGHT, SDL_WINDOW_SHOWN);
    if (win == NULL) {
        cout << "Can't create window: " << SDL_GetError() << endl;
        ok = false;
    }

    ren = SDL_CreateRenderer(win, -1, SDL_RENDERER_ACCELERATED);
    if (ren == NULL) {
        cout << "Can't create renderer: " << SDL_GetError() << endl;
        ok = false;
    }
    return ok;
}

void quit() {
    SDL_DestroyWindow(win);
    win = NULL;

    SDL_DestroyRenderer(ren);
    ren = NULL;

    SDL_Quit;
}

int main (int arhc, char ** argv) {
    if (!init()) {
        quit();
        system("pause");
        return 1;
    }

    SDL_SetRenderDrawColor(ren, 0x00, 0x00, 0x00, 0x00);
    SDL_RenderClear(ren);
    SDL_SetRenderDrawColor(ren, 0xFF, 0xFF, 0xFF, 0xFF);

    SDL_Rect rect1 = {10, 10, 50, 50};
    SDL_RenderFillRect(ren, &rect1);

    SDL_Rect rect2 = {70, 10, 50, 50};
    SDL_RenderDrawRect(ren, &rect2);

    SDL_RenderDrawLine(ren, 10, 70, 640 - 10, 70);

    for (int i = 10; i <= 640-10; i +=4 ) {
        SDL_RenderDrawPoint(ren, i, 90);
    }

    SDL_RenderPresent(ren);

    SDL_Delay(5000);

    quit();
    return 0;
}
