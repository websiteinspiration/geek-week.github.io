Teste beta do analisador PVS-Studio no Linux e macOS


Mais e mais usuários do analisador C # do PVS-Studio estão interessados ​​na possibilidade de usá-lo para testar o código C # no Linux e no macOS. E hoje temos boas notícias.

Desde o final do ano passado, trabalhamos ativamente em portar nosso analisador C # para a plataforma .NET Core, além de garantir seu desempenho em sistemas operacionais semelhantes ao Unix. Planejamos lançar o PVS-Studio C # para .NET Core nas plataformas Linux e macOS no final de abril - primeira metade de maio.

No Windows, o analisador continuará trabalhando no .NET Framework, pois decidimos nos concentrar no lançamento do analisador para novos sistemas operacionais. Além disso, você pode verificar projetos .NET Core de plataforma cruzada no Windows agora.

Planejamos iniciar o teste beta no início e meados de abril. Se você estiver interessado em participar, preencha este formulário . Quando o teste beta for iniciado, enviaremos instruções para instalar o analisador e uma licença de avaliação.

PS Também estamos desenvolvendo um plug-in para o ambiente de desenvolvimento JetBrains Rider para todas as plataformas suportadas pelo analisador. Como parte do teste beta, como um "bônus", você também terá a oportunidade de experimentar este plug-in, inclusive com o analisador "clássico" PVS-Studio C # no Windows.
