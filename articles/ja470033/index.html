F＃2：FSI環境

多くの人が知っているように、私はC＃の人です。そのため、何かを試したいだけの場合は、通常LINQPadを開いてそこで何かを試します。状況が拡大していて、実験をより詳細に制御する必要がある場合は、LINQPadをあきらめて、Visual StudioにConsoleApplicationアプリケーションを展開します。これらのアプローチはどちらも正常に機能しますが、新しいプロジェクトを作成することなく、Visual Studio自体の内部で何かを試すことができるような環境があったらいいでしょう。

幸い、F＃に1つあります。

たとえば、「F＃インタラクティブウィンドウ」（FSI）と呼ばれる完全に優れたツールがあります。これは、Visual Studio（VS 2012を使用）を使用して見つけることができます。たとえば



、F＃インタラクティブウィンドウを開くと、次のように表示されます



。それほど壮大に見えないかもしれませんが、このウィンドウを使用すると、多くのことが非常に簡単に行えます。必要に応じて、ラピッドプロトタイピングとテストに使用できます。

したがって、あなたは、プロンプトを持っていることがわかります>あなたの入力を待っています。何か紹介しましょうか？

入力する


入力を取得するためにF＃インタラクティブウィンドウで行う必要があるのは、2つのセミコロン「;;」で行を終了することだけです。

正しい入力のいくつかの有効な例を以下に示します：

値を取り、それを正方形にする関数の宣言（タイプxを指定していないことに注意してください）：

let squarer x = x * x;;

次に、squarer関数の呼び出しの戻り値を含む変数を宣言します。これは次のように簡単です。

let squared = squarer 10;;

データ表示


データを見たい場合は、同様の方法で評価することもできます。したがって、上記の値を入力したと仮定すると、次のようになります。

  1. 二乗関数
  2. 関数によって返されるint型

したがって、それらの1つ、たとえば「二乗」を評価するには、FSI入力行に次のように入力するだけです。

squared;;

この行をFSIで実行すると、次のような値がFSIウィンドウに表示されます。

val it : int = 100

値が実際には100であることがわかります。これは、FSIウィンドウで作成された二乗関数への以前の呼び出しの結果です。

終了する前に別のクールなこと


この記事を終える前に、FSIスクリプトファイルと一緒にFSIウィンドウを使用して、高速に何かを試すことができることを述べておきます。

次のコードを含むF＃（FSX）スクリプトファイルがあるとします。

open System
 
let squareRootMe x = System.Math.Sqrt(x)

そして、私はこれをFSIで試したかったのですが、コードの行/セクションを選択し、FSIウィンドウでその実行を選択することができます。



現在はFSIウィンドウのスコープにあります。つまり、次のように関数を使用できます。

>  let x =    squareRootMe 144.0;;
 
val x : float = 12.0
