金士顿SA2000M8 / 1000G和R500G：硬件，规格和成本

ATTO磁盘基准

许多制造商经常使用ATTO Disk Benchmark实用程序为驱动器分配顺序读取和写入速度的规范。该基准测试也很有趣，因为它可以立即使我们直观地了解被测驱动器如何处理不同大小的数据块。为了使您可以将结果与竞争解决方案进行比较，我们保留了默认的基准设置，并多次运行了Kingston A2000 1TB。结果我们得到了什么？ ATTO Disk Benchmark记录的最大读写速度分别为2020 MB / s和1950 MB / s。





AS SSD基准1.9.5

现在，我们继续进行另一个同样受欢迎的测试-AS SSD Benchmark，通过该测试，我们还可以使用重1 GB的模拟文件进行多次运行。这里的结果类似于其他实用程序显示的结果：平均而言，数据读取速度为2057 MB / s，写入速度为1996 MB / s。





AIDA64磁盘基准

流行的一组用于监视PC及其组件状态的测试和实用程序AIDA64，还包括流行的AIDA64 Disk Benchmark实用程序，该实用程序提供了一组测试脚本，用于测量读取速度和对驱动器的访问时间。向她显示的结果接近于ATTO Disk Benchmark和CrystalDiskMark：平均读取速度为1794 MB / s。请注意，我们所说的是平均速度，而不是最大速度。在CrystalDiskMark上，此参数处于1700-1843 MB / s的级别。
