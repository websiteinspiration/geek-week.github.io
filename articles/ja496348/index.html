短いレビュー


Storageオブジェクトは、クライアント側にデータを格納するために使用され、この意味で、Cookieの代替として機能します。ストレージの利点は、ストレージサイズ（5 MBから、それはブラウザーに依存し、制限を超えると、エラー「QUOTA_EXCEEDED_ERR」がスローされます）であり、サーバーにアクセスする必要がないことです。重要な欠点はセキュリティです。ページにアクセスするには悪意のあるスクリプトが必要で、書き込みが失われます。したがって、機密情報をストレージに保存することはお勧めしません。

公平に言えば、今日クライアント側にデータを保存するためのより高度なソリューションがあることは注目に値します-これはIndexedDB、Service Workers + Cache APIなど

です。ServiceWorker についてはこちらを参照してください。

Web Storage APIには、localStorageとsessionStorageが含まれています。それらの違いは、データの保存時間です。localStorageでは、データは「明示的に」削除されるまで永続的に保存されます（ページを再読み込みしたり閉じたりしてもデータは削除されません）。名前が示すように、sessionStorageのデータストレージ時間は、ブラウザセッションによって制限されます。sessionStorageは実際にはほとんど使用されないため、localStorageのみを検討します。

localStorageについて知っておくべきことは何ですか？


  • localStorageを使用すると、Storageオブジェクトの表現が作成されます。
  • localStorageのデータは、キーと値のペアとして保存されます。
  • データは文字列として保存されます。
  • データはソートされていないため、それらが混合することがあります（タスクリストの例で確認します）。
  • ブラウザでシークレットモードまたはプライベートモードを有効にすると、localStorageを使用できなくなる場合があります（ブラウザによって異なります）。

localStorageには以下のメソッドがあります。

  • Storage.key（n）-インデックスnのキー名
  • Storage.getItem（）-アイテムを取得します
  • Storage.setItem（）-アイテムを書き込む
  • Storage.removeItem（）-アイテムを削除します
  • Storage.clear（）-ストレージをクリアする
  • Storage.length-ストレージ長（要素数-キー/値のペア）

仕様では、次のようになります。

interface Storage {
    readonly attribute unsigned long length;
    DOMString? key(unsigned long index);
    getter DOMString? getItem(DOMString key);
    setter void setItem(DOMString key, DOMString value);
    deleter void removeItem(DOMString key);
    void clear();
};

データは、次のいずれかの方法でストレージに書き込まれます。

localStorage.color = 'deepskyblue'
localStorage[color] = 'deepskyblue'
localStorage.setItem('color', 'deepskyblue') //

次のようなデータを取得できます。

localStorage.getItem('color')
localStorage['color']

ストレージキーを反復処理して値を取得する方法は？


//  1
for (let i = 0; i < localStorage.length; i++) {
    let key = localStorage.key(i)
    console.log(`${key}: ${localStorage.getItem(key)}`)
}

//  2
let keys = Object.keys(localStorage)
for (let key of keys) {
    console.log(`${key}: ${localStorage.getItem(key)}`)
}

前述のとおり、ストレージ内のデータは文字列形式であるため、オブジェクトの書き込みにいくつかの問題があり、JSON.stringify（）-JSON.parse（）タンデムを使用して簡単に解決できます。

localStorage.user = {
    name: 'Harry'
}
console.dir(localStorage.user) // [object Object]

localStorage.user = JSON.stringify({
    name: 'Harry'
})
let user = JSON.parse(localStorage.user)
console.dir(user.name) // Harry

localStorageと対話するために、特別なイベント-storage（onstorage）があります。これは、データの書き込み/削除時に発生します。次のプロパティがあります。

  • キー-キー
  • oldValue-古い値
  • newValue-新しい値
  • url-ストレージアドレス
  • storageArea-変更が発生したオブジェクト

仕様では、次のようになります。

[Constructor(DOMString type, optional StorageEventInit eventInitDict)]
    interface StorageEvent : Event {
    readonly attribute DOMString? key;
    readonly attribute DOMString? oldValue;
    readonly attribute DOMString? newValue;
    readonly attribute DOMString url;
    readonly attribute Storage? storageArea;
};

dictionary StorageEventInit : EventInit {
    DOMString? key;
    DOMString? oldValue;
    DOMString? newValue;
    DOMString url;
    Storage? storageArea;
};

localStorageはプロトタイピングを許可しますか？


Storage.prototype.removeItems = function() {
    for (item in arguments) {
        this.removeItem(arguments[item])
    }
}

localStorage.setItem('first', 'some value')
localStorage.setItem('second', 'some value')

localStorage.removeItems('first', 'second')

console.log(localStorage.length) // 0

localStorageのデータを確認する方法は？


//  1
localStorage.setItem('name', 'Harry')

function isExist(name) {
    return (!!localStorage[name])
}

isExist('name') // true

//  2
function isItemExist(name) {
    return (name in localStorage)
}

isItemExist('name') // true

ログインフォームで作業します


マークアップは次のようになります。

<form>
    Login: <input type="text">
    Password: <input type="text">
    <input type="submit">
</form>

フォームと3つの「入力」があります。パスワードには<input type = "text">を使用します。正しいタイプ（パスワード）を使用すると、Chromeは入力されたデータを保存しようとするため、独自の機能を実装できなくなります。

JavaScript：

//         
let form = document.querySelector('form')
let login = document.querySelector('input')
let password = document.querySelector('input + input')

//  localStorage  
//     
//    
if (localStorage.length != 0) {
    login.value = localStorage.login
    password.value = localStorage.password
}

//      "submit"
form.addEventListener('submit', () => {
    //      localStorage
    localStorage.login = login.value
    localStorage.password = password.value
    
    //    hello  world     , 
    //       "welcome"  
    if (login.value == 'hello' && password.value == 'world') {
        document.write('welcome')
    }
})

フォームの「検証」は行いませんのでご注意ください。これにより、特に、空の行をユーザー名とパスワードとして記録できます。

タスクリストのロジックを書く


マークアップは次のようになります。

<input type="text"><button class="add">add task</button><button class="clear">clear storage</button>

<ul></ul>

タスクを入力するための「入力」、リストにタスクを追加するためのボタン、リストとストレージをクリーニングするためのボタン、リストのコンテナがあります。

JavaScript：

//      
let input = document.querySelector('input')
input.focus()
//       
let addButton = document.querySelector('.add')
//    
let list = document.querySelector('ul')

//   localStorage  
if (localStorage.length != 0) {
    //     /
    for (let i = 0; i < localStorage.length; i++) {
        let key = localStorage.key(i)
        //   -  
        let template = `${localStorage.getItem(key)}`
        //    
        list.insertAdjacentHTML('afterbegin', template)
    }
    
    //      "close" -    
    document.querySelectorAll('.close').forEach(b => {
        //   
        b.addEventListener('click', e => {
            //    "li"
            let item = e.target.parentElement
            //    
            list.removeChild(item)
            //    localStorage
            localStorage.removeItem(`${item.dataset.id}`)
        })
    })
}

//       "Enter"
window.addEventListener('keydown', e => {
    if (e.keyCode == 13) addButton.click()
})

//           ""
addButton.addEventListener('click', () => {
    //   - 
    let text = input.value
    //  ,          "data-id"
    let template = `<li data-id="${++localStorage.length}"><button class="close">V</button><time>${new Date().toLocaleDateString()}</time> <p>${text}</p></li>`
    //   -   
    list.insertAdjacentHTML('afterbegin', template)
    //    localStorage
    localStorage.setItem(`${++localStorage.length}`, template)
    //   
    input.value = ''

    //           ""
    document.querySelector('.close').addEventListener('click', e => {
        //    -   
        let item = e.target.parentElement
        //    
        list.removeChild(item)
        //    localStorage
        localStorage.removeItem(`${item.dataset.id}`)
    })
})

//        ""
document.querySelector('.clear').onclick = () => {
    //  
    localStorage.clear()
    //    
    document.querySelectorAll('li').forEach(item => list.removeChild(item))
    //   
    input.focus()
}

結果は次のようになります。



リストに追加されたタスクは、既製のマークアップとしてlocalStorageに保存されます。ページがリロードされると、リストはストレージのデータから形成されます（前述の混合があります）。



緑のチェックマークをクリックしてリストからタスクを削除すると、対応するキーと値のペアがリポジトリから削除されます。

チャットの実装


マークアップは次のようになります。

<input type="text">
<button class="send">send message</button>
<button class="save">save chat</button>
<button class="clear">clear chat</button>

<div></div>

メッセージを入力するための入力、3つのボタンがあります。メッセージの送信、通信の保存、チャットとストレージのクリーンアップ、メッセージのコンテナです。

JavaScript：

//      
let input = document.querySelector('input')
input.focus()

//  
let sendButton = document.querySelector('.send')
let saveButton = document.querySelector('.save')
let clearButton = document.querySelector('.clear')

//  
let box = document.querySelector('div')

//      "messages"
if (localStorage.messages) {
    //  
    localStorage.messages
        .split('</p>,')
        .map(p => box.insertAdjacentHTML('beforeend', p))
}

//   
sendButton.addEventListener('click', () => {
    //   
    let text = document.querySelector('input').value
    //  
    let template = `<p><time>${new Date().toLocaleTimeString()}</time> ${text}</p>`
    //    
    box.insertAdjacentHTML('afterbegin', template)
    //   
    input.value = ''
    //    
    localStorage.message = template
})

//       "Enter"
window.addEventListener('keydown', e => {
    if (e.keyCode == 13) sendButton.click()
})

//   "storage"
window.addEventListener('storage', event => {
    //     "messages"
    //  
    if (event.key == 'messages') return
    //      null
    //     
    //     
    event.newValue == null ? clearButton.click() : box.insertAdjacentHTML('afterbegin', event.newValue)
})

//  
saveButton.addEventListener('click', () => {
    //  
    let messages = []
    //  
    document.querySelectorAll('p').forEach(p => messages.push(p.outerHTML))
    //    
    localStorage.messages = messages
})

//    
clearButton.addEventListener('click', () => {
    localStorage.clear()
    document.querySelectorAll('p').forEach(p => box.removeChild(p))
    input.focus()
})

なります。送信されるメッセージはlocalStorage.messageに書き込まれます。「ストレージ」イベントを使用すると、ブラウザのタブ間のメッセージの交換を整理できます。



チャットが保存されると、すべてのメッセージがlocalStorage.messagesに書き込まれます。ページがリロードされると、記録されたメッセージから通信が形成されます。

1つの製品のレイアウトは次のようになります。

<div class="item">
    <h3 class="title">Item1</h3>
    <img src="http://placeimg.com/150/200/tech" alt="#">
    <p>Price: <span class="price">1000</span></p>
    <button class="add" data-id="1">Buy</button>
</div>

商品のコンテナ、商品の名前、画像、価格、バスケットに商品を追加するためのボタンがあります。

また、バスケットの内容を表示し、バスケットとストレージを空にするボタン用のコンテナと、バスケット用のコンテナもあります。

<div class="buttons">
    <button id="open">Cart</button>
    <button id="clear">Clear</button>
</div>
<div id="content"></div>

JavaScript：

//    
let itemBox = document.querySelectorAll('.item'),
    cart = document.getElementById('content');

//    localStorage
function getCartData() {
    return JSON.parse(localStorage.getItem('cart'));
}

//    
function setCartData(o) {
    localStorage.setItem('cart', JSON.stringify(o));
}

//    
function addToCart() {
    //       
    this.disabled = true;
    //        ,   
    let cartData = getCartData() || {},
        //    "Buy"
        parentBox = this.parentNode,
        // id 
        itemId = this.getAttribute('data-id'),
        //  
        itemTitle = parentBox.querySelector('.title').innerHTML,
        //  
        itemPrice = parentBox.querySelector('.price').innerHTML;
    // +1  
    if (cartData.hasOwnProperty(itemId)) {
        cartData[itemId][2] += 1;
    } else {
        // + 
        cartData[itemId] = [itemTitle, itemPrice, 1];
    }
    //    localStorage
    if (!setCartData(cartData)) {
        //   
        this.disabled = false;
    }
}

//    ""    "Buy"
for (let i = 0; i < itemBox.length; i++) {
    itemBox[i].querySelector('.add').addEventListener('click', addToCart)
}

//  
function openCart() {
    //    
    let cartData = getCartData(),
        totalItems = '',
        totalGoods = 0,
        totalPrice = 0;
    //    
    if (cartData !== null) {
        totalItems = '<table><tr><th>Name</th><th>Price</th><th>Amount</th></tr>';
        for (let items in cartData) {
            totalItems += '<tr>';
            for (let i = 0; i < cartData[items].length; i++) {
                totalItems += '<td>' + cartData[items][i] + '</td>';
            }
            totalItems += '</tr>';
            totalGoods += cartData[items][2];
            totalPrice += cartData[items][1] * cartData[items][2];
        }
        totalItems += '</table>';
        cart.innerHTML = totalItems;
        cart.append(document.createElement('p').innerHTML = 'Goods: ' + totalGoods + '. Price: ' + totalPrice);
    } else {
        //    
        cart.innerHTML = 'empty';
    }
}
//  
document.getElementById('open').addEventListener('click', openCart);

//  
document.getElementById('clear').addEventListener('click', () => {
    localStorage.removeItem('cart');
    cart.innerHTML = 'leared';
});
