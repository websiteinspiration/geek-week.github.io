Azure IoT Edge Tools（Preview）拡張機能が発表されました

Visual Studio 2019- Azure IoT Edge Toolsの新しい拡張機能のプレビューバージョンが利用可能になったことをお知らせします。拡張機能は、Visual Studio 2019を使用したIoT Edgeソリューションの開発をサポートするための豊富な機能セットを提供します。



  • Azure IoT Edge (Linux amd64, Linux arm32v7, Windows amd64)
  • IoT Edge (C#/C)
  • , IoT Edge Visual Studio
  • Docker- IoT Edge
  • IoT Edge
  • IoT- IoT Edge ( Cloud Explorer)




  • Visual Studio 2019: “.NET desktop development” “Azure development workload”; C-, “Windows desktop development with C++”
  • Dockerデスクトップ。LinuxコンテナーモードまたはWindowsコンテナーモードで実行されているDocker CEを適切に構成する必要があります。
  • IoT Edgeソリューションをデバッグ、実行、テストするためのローカル開発環境をセットアップするには、Azure IoT EdgeHub Dev Toolが必要です。これを行うには、最初にPython（2.7 / 3.6）をインストールしてから、ターミナルで以下のコマンドを実行してiotedgehubdevをインストールします。Azure IoT EdgeHub開発ツールのバージョンが0.8.0以降であることを確認してください。

pip install --upgrade iotedgehubdev

取り付け


新しい拡張機能をインストールするには、2つのオプションがあります。

  1. Visual Studio Marketplaceから新しい拡張機能をダウンロードしてインストールします。
  2. Visual Studio 2019, Extensions -> Manage Extensions. Manage Extensions Online edge , “Azure IoT Edge Tools for VS 2019 [Preview]”.
