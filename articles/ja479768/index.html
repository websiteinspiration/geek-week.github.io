AlienVault SIEM（OSSIM）のインストールと構成

この記事の翻訳は、特にLinuxセキュリティコースの学生を対象に作成されました。




OSSIM（オープンソースセキュリティ情報管理）は、SIEM（セキュリティ情報とイベント管理）機能を提供するAlienvaultのオープンソースプロジェクトです。セキュリティの専門家が必要とする次のSIEM機能を提供します。

  • イベントコレクション
  • 正規化
  • 相関

OSSIMは、基本的なセキュリティ機能を提供する統合プラットフォームです。OSSIMプラットフォームには、多くの認められたオープンソースソフトウェアが組み込まれています。これは、統一されたセキュリティの可視性に向けて最初のステップを実行する最も速い方法であり続けます。

OSSIMプラットフォームは、次のオープンソースプログラム/プラグインをサポートしています。

  • Apache
  • IIS
  • Syslog
  • オセック
  • スネア
  • Snort
  • Openvas
  • ネッスス
  • ナギオス
  • Ntop
  • Nmap

OSSIMをインストールする


AlienVault からISOイメージをダウンロードし、仮想マシンにインストールします。このガイドでは、物理サーバーの代わりに、OSSIMを次の仕様の仮想マシンにインストール

します。サーバーの管理用とログの収集用、ネットワークデバイスの監視用の2つのインターフェイスがあります。仮想マシンの詳細を以下に示します。

プロセッサ：2 VCPU、RAM：2 GB、ハードディスクサイズ：8 GB、管理IPアドレス：192.168.1.150/24およびデバイスネットワーク：192.168.0.0/24

OSSIM仮想マシンがISOイメージから起動すると、インストールウィザードが表示されます次の2つのパラメーター。

OSSIMサーバーを構成するには、上の図の[スタート]ボタンをクリックします。

最初のオプションをクリックした後、別のウィンドウがネットワーク構成を要求します。これは下図に示されています。ログコレクターとOSSIMサーバー監視インターフェイス用にeth1を構成しました。



2番目のステップで、OSSIMはネットワークデバイスを自動的に検出します。[Device Discovery]オプション（2）を選択すると、次のウィンドウで構成を求められます。自動および手動のデバイス検出をサポートします。

OSSIMサーバー上のホストのタイプ：

  • ウィンドウズ
  • Linux
  • ネットワークデバイス



ネットワークをセットアップしてデバイスを検出したら、次のステップはWindows / LinuxデバイスにHIDSを導入して、ファイルの整合性、監視、ルートキットの検出、およびイベントログの収集を確認することです。HIDS展開デバイスのユーザー名/パスワードを入力します。



リストから目的のホストを選択し、[展開]ボタンをクリックしてHIDSを展開します。次に、[続行]ボタンをクリックして、図に示す展開プロセスを開始します。このプロセスは、選択したホストにHIDSを導入するのに数分かかります。





ログ管理


次の図は、さまざまなログを管理するために検出されたホストの構成を示しています。



構成ウィザードの最後のオプションは、OTX（AlienVault脅威共有プログラム）に参加することです。このオプションはサブスクライブしません。[完了]ボタンをクリックして、セットアップ手順を完了します。

ウェブインターフェース


OSSIMサーバーのWebインターフェイスは、メインのグラフィカルインターフェイスの次のオプションで構成されています。

  • ダッシュボード
  • 分析
  • 水曜日
  • 報告書
  • 構成

ダッシュボード


脅威の重大度、ネットワークノードの脆弱性、展開ステータス、リスクマップ、OTX統計など、すべてのOSSIMサーバーコンポーネントの完全なビューが表示されます。ダッシュボードのサブメニューを次の図に示します。



分析


分析は、SIEMデバイスの非常に重要なコンポーネントです。OSSIMサーバーは、ログに基づいてホストを分析します。このメニューには、アラーム、SIEM（セキュリティイベント）、チケット、未処理のログが表示されます。分析メニューはさらに次のサブメニューに分かれています。



水曜日


OSSIMサーバーのこのメニューでは、設定は組織のデバイスに関連しています。デバイス、グループとネットワーク、脆弱性、ネットワークフロー、検出設定が表示されます。これらすべての設定のサブメニューを下の図に示します。



報告書


レポートは、登録サーバーの重要なコンポーネントです。OSSIMサーバーは、特定のホストの詳細な調査に非常に役立つレポートも生成します。



構成


AlienVault SIEM（OSSIM）をインストールして構成するためのmeHow構成では、ユーザーはOSSIMサーバーの設定を変更できます。たとえば、管理インターフェイスのIPアドレスを変更したり、監視とログ用のホストを追加したり、さまざまなセンサーやプラグインを追加/削除したりできます。すべてのサービスのサブメニューを以下に示します。
