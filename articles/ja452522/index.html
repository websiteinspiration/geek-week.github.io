8つのあまり知られていないBashオプション

一部のBashオプションはよく知られており、よく使用されます。たとえば、多くはスクリプトの最初に書きます

set -o xtrace

デバッグ用、

set -o errexit

誤って終了するか

set -o errunset

呼び出された変数が設定されていない場合に終了します。

しかし、他の多くのオプションがあります。時々、それらは人間で説明されすぎて混乱しているので、私はここで説明とともに最も有用なもののいくつかを集めました。

注：Macには古いバージョンのbash（4.xではなく3.x）が搭載されている場合があり、これらのオプションのすべてが使用できるわけではありません。この場合は、こちらまたはこちらをご覧ください。

setまたはshopt？


bashパラメータを設定するには、スクリプトからまたはコマンドラインからの2つの方法があります。組み込みコマンドsetとを使用できますshopt。どちらもシェルの動作を変更し、ほとんど同じことを（異なる引数で）行い、その起源が異なります。パラメーターsetは他のシェルのパラメーターから継承または借用されますが、パラメーターはshoptbashで作成されます。

現在のオプションを確認したい場合は、以下を実行します。

$ set -o
$ shopt

オプションcをアクティブにsetするには、長い構文または短い構文を使用します。

$ set -o errunset
$ set -e

効果は同じです。

オプションを無効にするには、マイナスの代わりにプラスを置く必要があります：

$ set +e

長い間、ロジックが間違っているように見えるため、この構文を覚えることができませんでした（マイナスはオプションをオンにし、プラスはオプションをオフにします）。有効にすると無効のオプションは、（複数の論理）フラグ（セット）とは、（未設定）が使用されます。

shopt-s-u

$ shopt -s cdspell # <= on
$ shopt -u cdspell # <= off

ディレクトリの変更


ディレクトリの操作に役立つオプションがいくつかあります。

1. cdspell


この設定により、bashはタイプミスを理解し始め、入力した名前がエラーのあるフォルダーに移動します。

$ shopt -s cdspell
$ mkdir abcdefg
$ cd abcdeg
abcdefg
$ cd ..

私は長年このオプションを使用しており、非常にまれに（おそらく1年に1回）、非常に奇妙な決定を下します。しかし、それ以外の日にはcdspell、文字通り毎日、時間を節約できます。

2. autocd


繰り返し入力の非効率性に我慢できない場合はcd、このオプションを設定して、Xコマンドが存在しない場合にXフォルダーに移動できます。

$ shopt -s autocd
$ abcdefg
$ cd ..

オートコンプリートと組み合わせることで、フォルダーにすばやくジャンプできます。

$ ./abc[TAB][RETURN]
cd -- ./abcdefg

フォルダに名前を付けないでくださいrm -rf *（はい、ちなみに、これは可能です）。

3. direxpand


これは、Tabキーを押して環境変数を拡張する優れたオプションです。

$ shopt -s direxpand
$ ./[TAB]     #  ...
$ /full/path/to/current_working_folder
$ ~/[TAB]     #  ...
$ /full/path/to/home/folder
$ $HOME/[TAB] #   ...
$ /full/path/to/home/folder

きれいな出口


4. checkjobs


このオプションは、ジョブがまだバックグラウンドで実行されている場合に、セッションの終了を停止します。

終了する代わりに、不完全なタスクのリストが表示されます。それでも終了する場合は、もう一度入力してくださいexit。

$ shopt -s checkjobs
$ echo $$
68125             # <= ID   
$ sleep 999 &
$ exit
There are running jobs.
[1]+  Running                 sleep 999 &
$ echo $$
68125             # <= ID     
$ exit
There are running jobs.
[1]+  Running                 sleep 999 &
$ exit
$ echo $$
$ 59316           # <=    ID

代替能力


5. globstar


このオプションは、あなたに代用の超能力を与えます！入力した場合：

$ shopt -s globstar
$ ls **

その後、シェルはすべてのディレクトリとサブディレクトリを再帰的に表示します。

と組み合わせると、direxpand階層内の以下のすべてをすばやく確認できます。

$ shopt -s direxpand
$ ls **[TAB][TAB]
Display all 2033 possibilities? (y or n)

6.extglob


このオプションは、正規表現に関連付けられることが多い関数をアクティブにします。これは時々非常に役立ちます：

$ shopt -s extglob
$ touch afile bfile cfile
$ ls
afile bfile cfile
$ ls ?(a*|b*)
afile bfile
$ ls !(a*|b*)
cfile

ここでは、テンプレートは括弧内に配置され、縦棒で区切られています。利用可能な演算子は次のとおりです。

？=指定されたパターンの0回または1回の出現に一致
！=指定されたパターンに一致しないすべてを表示する
* = 0回以上の発生
+ = 1つ以上の発生
@ =正確に1回

事故保護


7. histverify


最初は、省略形!!との履歴からクイックスタートコマンドを使用するのは少し怖いかもしれません!$。

このオプションをhistverify使用すると、実際に開始する前に、Bashがコマンドをどのように解釈するかを最初に確認できます。

$ shopt -s histverify
$ echo !$          # <=   Enter   
$ echo histverify  # <=     ,
histverify         # <=

8.ノクロバー


繰り返しになりますが、事故から保護するため、つまりリダイレクト演算子（>）を使用して既存のファイルを上書きしないようにするためです。バックアップがない場合、これは惨事になる可能性があります。

オプションset -はそのような書き換えを禁止します。必要に応じて、演算子を使用して保護をバイパスできます>|。

$ touch afile
$ set -C
$ echo something > afile
-bash: afile: cannot overwrite existing file
$ echo something >| afile
$
