リーフレット。キャンバスで友達の画像



こんにちは、親愛なるカブラハブリテス！

リーフレットは、インタラクティブなマップをWebサイトに追加して簡単にカスタマイズできるライブラリです。今日は、マップのキャンバスキャンバスレイヤーに画像を配置する方法と、基本的なマーカーについて見ていきます。

仕事


さまざまなステータスのマークが付いたトラックを作成します。ステータスはマーカーでマークされます。各ステータスには独自の優先順位があります。

  • マップを最適化するには、オブジェクトのレンダリングをCanvasを使用して行う必要があります。
  • マーカーには、ポイントとイメージの2つのタイプがあります。
  • マーカーが重なる場合、優先度の高いステータスのマーカーが上に表示されます。
  • マウスをマーカーの上に置くと、各マーカーがアクティブになります（たとえば、追加情報を表示するため）。

トレーニング


Leaflet.js ライブラリを接続し、ベースマップを追加します。

const map = L.map('map', {
    preferCanvas: true,
}).setView([51.505, -0.09], 13);
L.tileLayer('https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png').addTo(map);

わかりやすくするために、優先度の高い順に3つの状態を使用します。基本（緑のマーカー）、メッセージ（画像）、エラー（赤いマーカー）です。

したがって、赤いマーカーは画像と重なり、画像は緑のマーカーと重なります。

/*   */
L.circleMarker(L.latLng(51.52, -0.109), {
    radius: 10,
    fillColor: '#27ae60',
    fillOpacity: 1,
    color: '#fff',
    weight: 3,
}).addTo(map);

/*   */
L.marker(L.latLng(51.52, -0.109), {
    icon: L.icon({
        iconUrl: 'icon.png',   // url 
        iconSize: [40, 40],   //  
        iconAnchor: [20, 20],   //   
    }),
}).addTo(map);

/*   */
L.circleMarker(L.latLng(51.52, -0.109), {
    radius: 8,
    fillColor: '#f44334',
    fillOpacity: 1,
    color: '#fff',
    weight: 3,
}).addTo(map);

問題


リーフレットは一度に1つずつマーカーを追加するため、後続の各マーカーは前のマーカーと重なる必要があります。しかし、実際にはそうではありません。L.markerは、Canvasレイヤーとは別に、通常のIMGとして画像を追加します。

キャンバスの前または下に配置できます。その結果、2つのL.circleMarkerの間にL.markerを配置することはできません。

したがって、標準のマーカーが追加された同じキャンバスに画像を配置する方法が必要です。

注意：ネットワーク上にCanvasに画像を追加できるプラグインがいくつかあります。しかし、それらは別個のCanvasまたはレイヤーのグループさえ作成します！その結果、単に優先マーカーを配置することは非常に困難になります。また、キャンバスレイヤー同士が重なり、下のレイヤーのマーカーをクリックできなくなります！

決定


ステップ1. L.CircleMarkerから子クラスを作成します。これは、「img」オブジェクトを受け取り、イメージをロードして、L.Canvasに追加します。

const CanvasMarker = L.CircleMarker.extend({
    _updatePath() {
        if (!this.options.img.el) { //  IMG
            const img = document.createElement('img');
            img.src = this.options.img.url;
            this.options.img.el = img;
            img.onload = () => {
                this.redraw();  //   
            };
        } else {
            this._renderer._updateImg(this);    // _updateImg
        }
    },
});

L.canvasMarker = function (...options) {
    return new CanvasMarker(...options);
};

ステップ2. L.Canvasの_updateImgメソッドについて説明します。彼はステップ1で渡した画像を含むオブジェクトを受け取り、それをキャンバスに描画します。

L.Canvas.include({
    _updateImg(layer) { //  img  Canvas-
        const { img } = layer.options;
        const p = layer._point.round();
        this._ctx.drawImage(img.el, p.x - img.size[0] / 2, p.y - img.size[1] / 2, img.size[0], img.size[1]);
    },
});

ステップ3. L.markerの代わりに、L.canvasMarkerを使用できます。'anchor'パラメータは使用されないことに注意してください。画像は自動的に調整されます！

/*   */
    L.circleMarker(L.latLng(51.52, -0.109), {
        radius: 10,
        fillColor: '#27ae60',
        fillOpacity: 1,
        color: '#fff',
        weight: 3,
    }).addTo(map);

    /*   */
    L.canvasMarker(L.latLng(51.52, -0.109), {
        img: {
            url: 'icon.png',
            size: [40, 40],
        },
    }).addTo(map);

    /*   */
    L.circleMarker(L.latLng(51.52, -0.109), {
        radius: 8,
        fillColor: '#f44334',
        fillOpacity: 1,
        color: '#fff',
        weight: 3,
    }).addTo(map);

結果として：

  • すべてのマーカーは、単一のキャンバスレイヤーに配置されます。
  • マーカーは、マップに追加された順序で互いに重なります。
  • マーカーをマウスでホバーすると、アクティブのままになります。

問題は解決された！


さらに


L.canvasMarkerメソッドを「ポンプ」して、マップ上の移動方向に画像を自動的に展開する機能を追加しましょう！

前のポイントの座標に基づきます。これを行うには、最初に_updateImgメソッドを変更します。

L.Canvas.include({
    _updateImg(layer) {
        const { img } = layer.options;
        const p = layer._point.round();
        if (img.rotate) {
            this._ctx.save();
            this._ctx.translate(p.x, p.y);
            this._ctx.rotate(img.rotate * Math.PI / 180);
            this._ctx.drawImage(img.el, -img.size[0] / 2, -img.size[1] / 2, img.size[0], img.size[1]);
            this._ctx.restore();
        } else {
            this._ctx.drawImage(img.el, p.x - img.size[0] / 2, p.y - img.size[1] / 2, img.size[0], img.size[1]);
        }
    },
});

例からわかるように、「img」には回転するための「rotate」プロパティが必要です。マーカーを追加するときに、手動で設定できます。

L.canvasMarker(L.latLng(51.52, -0.109), {
    img: {
        url: 'icon.png',
        size: [40, 40],
        rotate: 15, //  
    },
}).addTo(map);

ただし、前のポイントに基づいて回転角度を自動的に計算する必要があります。したがって、2つの座標（angleCrds）に基づく角度の計算を追加します。


const angleCrds = (map, prevLatlng, latlng) => {
    if (!latlng || !prevLatlng) return 0;
    const pxStart = map.project(prevLatlng);
    const pxEnd = map.project(latlng);
    return Math.atan2(pxStart.y - pxEnd.y, pxStart.x - pxEnd.x) / Math.PI * 180 - 90;
};

const CanvasMarker = L.CircleMarker.extend({
    _updatePath() {
        if (!this.options.img.el) {
            /*   */
            if (!this.options.img.rotate) this.options.img.rotate = 0;
            this.options.img.rotate += angleCrds(this._map, this.options.prevLatlng, this._latlng);

            const img = document.createElement('img');
            img.src = this.options.img.url;
            this.options.img.el = img;
            img.onload = () => {
                this.redraw();
            };
        } else {
            this._renderer._updateImg(this);
        }
    },
});

L.canvasMarker(L.latLng(51.52, -0.109), {
    prevLatlng: L.latLng(51.528, -0.1), //   
    img: {
        url: 'icon.png',
        size: [40, 40],
    },
}).addTo(map);
