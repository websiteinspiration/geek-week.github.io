Meine besten kostenlosen Entwicklertools

Dies ist eine Übersetzung des Artikels von dev.to. Sein Autor Fayaz Ahmed teilt eine Liste seiner bevorzugten Entwicklertools.



Ich entwickle Projekte wie Tvflix , GSON , Appy Dev sowie viele verschiedene Mikroprojekte und bin daher daran interessiert, kostenlose Tools für meine Arbeit zu finden.

Dies ist meine persönliche Liste, aber Sie können sie mit anderen erfolgreichen Tools ergänzen. Hier sind einige meiner Favoriten.

Dienste und Plattformen für die Bereitstellung Ihres Frontends / Backends


1. Netlify
Verwenden Sie Netlify, um moderne statische Websites bereitzustellen. Sie erhalten eine CDN-Infrastruktur, eine kontinuierliche Bereitstellung, HTTPS mit einem Klick und alle Dienste, die Sie benötigen. Fangen Sie kostenlos an.

2. Vercel (ehemals Zeit)
Vercel ist der einfachste Weg, Websites bereitzustellen. Hosten Sie Ihre Nullkonfigurations-Webprojekte mit automatischem SSL und globaler CDN-Infrastruktur.

3. Surge
Forwarding-Webprojekte sollten schnell, einfach und sicher sein. Surge ist eine statische Web-Publishing-Funktion für Front-End-Entwickler direkt über die CLI.

4. Firebase
Eine Alternative zu Netlify / Zeit von Google. Firebase ist eine mobile Plattform, mit der Sie schnell hochwertige Anwendungen entwickeln und Ihr Geschäft ausbauen können.

5. Statische Azure-Web-Apps
Ab heute haben Sie die Möglichkeit, statische Web-Apps zum Hosten von einseitigen Apps in Azure zu verwenden.

6. GitHub Pages-
Websites für Sie und Ihre Projekte, die direkt aus Ihrem GitHub-Repository veröffentlicht werden. Es reicht aus, zu bearbeiten und auszuführen, damit die Änderungen in die Live-Version übernommen werden.

7. GitLab-Seiten
Wie bei GitHub können Sie mit dem GitLab-Seiten-Dienst statische Websites direkt aus dem Repository in GitLab veröffentlichen.

8. Kunstflug
Hochleistungs-statisches Hosting. Funktioniert hervorragend mit statischen Site-Generatoren wie Hugo, Jekyll, Gatsby, Gridsome.

9. Panne
Vielleicht ist dies eines der Werkzeuge, die ich heute am häufigsten benutze. Ein einfacher und leistungsstarker kostenloser Dienst zum Erstellen und Hosten von Anwendungen auf der Basis von Node.js.

10. Heroku
Heroku, eine erweiterte Version von Glitch, ist eine "Platform as a Service" (PaaS), mit der Entwickler Anwendungen erstellen, ausführen und vollständig in der Cloud verwalten können.

Automatisierungstools


1. Cron-job.org
Cron-job ist ein kostenloser Open Source-Dienst, mit dem Sie eine unbegrenzte Anzahl von Aufgaben im Cron-Format ausführen können. Sie müssen lediglich Ihre API oder die E-Mail-Adresse Ihres Webhooks verwenden, über die mit der angegebenen Häufigkeit eine Verbindung zum Dienst hergestellt wird.

2. n8n
Dieses Open Source-Tool ist eine Alternative zu Zapier. Der Dienst bietet unbegrenzte Möglichkeiten und aktualisiert ständig die Liste der für die Synchronisierung verfügbaren Tools und Anwendungen. Sie können n8n auch auf Ihrem eigenen Server hosten.

3. Integromat
Integromat ist ein benutzerfreundliches und dennoch effektives Werkzeug zur Automatisierung manueller Prozesse mit einzigartigen Eigenschaften. Synchronisieren Sie Ihre bevorzugten Anwendungen, Dienste und Geräte miteinander, auch wenn Sie keine Programmierkenntnisse haben.

4. IFTTT
Probieren Sie IFTTT aus - die einfachste und kostenlosste Möglichkeit, mehr mit Ihren bevorzugten Anwendungen und Geräten zu tun. Hausarbeiten erfordern weniger Aufmerksamkeit, die Arbeit wird noch produktiver, Sie bewahren die Privatsphäre und Ihre Daten sind sicher. Ich bin sicher, dass unter Synchronisationsbedingungen alles besser funktioniert.

5. Pipedream
Pipedream ist eine Plattform, auf der Entwickler Projekte im Zusammenhang mit der Datenverarbeitung erstellen und veröffentlichen können. Öffnen Sie Projekte und Verzweigungen auf einer Plattform, die Tausende von Quellcodebeispielen und abgeschlossenen Projekten enthält.

Codeless Tools


1. NoCodeAPI
Der einfachste Weg für einen Front-End-Entwickler, seine Anwendung mit Diensten von Drittanbietern zu verbinden, ohne zusätzlichen Code und einen dedizierten Server zu schreiben. NoCodeAPI kann mit Google Sheet, Airtable, Google Analytics, Twitter, Instagram, XML, RSS, JSON, Slack, Zahlungssystemen und mehr als fünfzig Diensten arbeiten.

2. GlideApps
Glide verwandelt dynamische Tabellen in benutzerfreundliche Anwendungen, ohne dass Code erforderlich ist. Wählen Sie eine Tabelle aus oder beginnen Sie mit einer Vorlage, konfigurieren Sie die Einstellungen für Ihre Anwendung und geben Sie sie in einem Moment für andere frei. Probieren Sie es noch heute kostenlos aus!

3. Airtable
Airtable funktioniert wie eine dynamische Tabelle, bietet jedoch gleichzeitig Datenbankfunktionen zum Organisieren von Inhalten.

4. Bubble
Bubble führt Sie in eine neue Methode zum Erstellen von Webanwendungen ein. Dieses Programmiertool erfordert keine Arbeit mit Code und arbeitet im Point-and-Click-Format. Bubble hostet alle Anwendungen auf seiner Cloud-Plattform.

5. Material Icon Generator
Generieren Sie SVG-Icons im Stil des Materialdesigns für Ihre Anwendungen. Ein Open Source Service.

Tools für SEO / Internet Marketing / Promotion / Rechtsberatung


1. Meta-Tags
Generieren und zeigen Sie zuvor alle Meta-Tags und Tags für soziale Netzwerke auf einer Webseite an.

2. Webcode-Tools
Erstellen Sie Schemata, Tags und viele sich wiederholende Codefragmente.

3. Carbon
Carbon ist der einfachste Weg, visuelle Bilder Ihres Quellcodes zu erstellen und zu veröffentlichen.

4. Codeimg
Erstellen und veröffentlichen Sie visuelle Bilder Ihres Quellcodes.

5. GetTerms Generieren Sie den Basistext
der Benutzervereinbarung und der Datenschutzrichtlinie für Ihre Website.

6. BEE Free
BEE ist ein E-Mail-Editor, mit dem Sie schnell und problemlos großartige Antwort-E-Mails erstellen können. Die Autoren wollten den perfekten E-Mail-Editor entwickeln. Es ist möglich, BEE online zu verwenden oder in Ihre Anwendung einzubetten.
