Linux Foundation 2020: Linux, Cloud Engineer Bootcamp




如今，与云技术一起使用时，对GNU / Linux的知识比以往更加需要，即使在Microsoft Azure中，GNU / Linux也比Windows更为流行。特别重要的是人们如何以及在何处学习如何使用此免费系统。Linux Foundation当然也来了。ZDNet写道，Linux基金会是IT认证的先驱，早在2014年就提供了其首个远程认证计划。在此之前，几乎不可能在培训中心之外获得IT证书。Linux Foundation已经建立了可靠且经过验证的远程学习程序。这大大简化了培训，在大流行时期，对于想获得认证而无所事事的专业人员而言，这尤其重要。

humanID:





Linux.com讨论了一个旨在提高Internet工作安全性和舒适性的新项目。每天，数十亿人使用诸如Facebook登录之类的社交帐户来通过Internet访问应用程序。该出版物写道，该系统的主要缺点是无法区分机器人的真实用户。非营利组织humanID是哈佛大学社会影响基金的获得者，提出了一个创新构想：一键开发匿名登录名，作为社交输入的替代方法。 ”在humanID的帮助下，每个人都可以使用服务而无需放弃隐私并且不出售其数据。僵尸网络会被自动排除，而应用程序可以轻松阻止入侵者和巨魔，从而创建更多的公民数字社区，” humanID的共同创始人巴斯蒂安·珀勒（Bastian Purrer）说。

PineTab平板电脑可与Ubuntu Touch一起订购




OpenNET报告：“ Pine64社区已经开始接受UBports项目的Ubuntu Touch环境随附的10.1英寸PineTab平板电脑的订单。作为一种选择，提供了PostmarketOS和Arch Linux ARM程序集。这款平板电脑的售价为100美元，而其售价为120美元，带有可拆卸的键盘，可让您像普通笔记本电脑一样使用该设备。预计将于7月份开始交付。”

根据出版物的主要特征：

  1. 10.1英寸高清IPS屏幕，分辨率为1280×800;
  2. CPU Allwinner A64（64位4核ARM Cortex A-53 1.2 GHz），GPU MALI-400 MP2;
  3. 内存：2GB LPDDR3 SDRAM RAM，内置64GB eMMC闪存，SD卡插槽；
  4. 两个摄像头：后置5MP，1/4英寸（LED闪光灯）和前置2MP（f / 2.8、1 / 5英寸）；
  5. Wi-Fi 802.11 b / g / n，单频，热点，蓝牙4.0，A2DP;
  6. 1个完整的USB 2.0 Type A，1个微型USB OTG（可用于充电），用于扩展坞的USB 2.0端口，高清视频输出；
  7. 用于连接M.2扩展模块的插槽，该模块可选提供带有SATA SSD，LTE调制解调器，LoRa和RTL-SDR的模块；
  8. 锂电池6000 mAh;
  9. 尺寸258mm x 170mm x 11.2mm，可选键盘262mm x 180mm x 21.1mm。重量575克（带键盘的950克）。
