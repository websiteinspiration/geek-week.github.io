Desenvolva um aplicativo da web interativo em C # com zero conhecimento de JavaScript

Sou desenvolvedor .Net da Turkmen-Tranzit.

Meu PM precisava de alguém que quisesse sentir Blazor. Na minha opinião, fiz um bom trabalho nessa área. Durante o estudo, me deparei com problemas, cujas respostas não são tão fáceis de encontrar na Internet em russo.



Por que Blazor, e não o JavaScript padrão?

Em palavras simples, a tecnologia Blazor permite criar aplicativos da Web cujo lado do cliente consiste em marcação HTML e código C # em vez de JavaScript.

O Blazor pode acessar o DOM para redesenhar a página sem atualizar a página O
aplicativo Web no Blazor consiste em componentes. Os componentes foram projetados para exibir elementos visuais, como indicadores de download ou formulários de upload de arquivos. A marcação dos elementos é armazenada em um arquivo com a extensão .razor e toda a lógica nas classes C # comuns com o nome {ComponentName} .razor.cs.

Os componentes do Blazor são divididos em inteligentes e estúpidos.

Componentes estúpidos são independentes de todo o aplicativo, podem ser conectados a outros componentes e podem até usar outros componentes estúpidos.

Componentes inteligentes são descartáveis, eles precisam manter seu estado e muito raramente são usados ​​em outros componentes.

Na maioria das vezes, você precisará escrever componentes inteligentes.

No Blazor, você pode chamar funções JavaScript usando o código C # e vice-versa, você pode chamar métodos .Net usando JavaScript.

Sim, se você possui um profundo conhecimento de JavaScript, é guiado em estruturas de JavaScript e se familiarizou com o mundo do .Net, é melhor não entrar no Blazor, a tecnologia é relativamente nova e, quando você o conhece, pode se assustar. O objetivo do artigo é para pessoas que sabem usar o C #, mas, por algum motivo que não tem JavaScript, fale sobre sua contrapartida promissora da Microsoft, na qual você pode usar todos os recursos do C #.

Por exemplo, desenvolveremos o CRM, com uma lista de empresas que podem ser nossos parceiros, com pesquisa automatizada de funcionários da empresa, distribuição automatizada de mensagens para todos os clientes em potencial, registro de ações do usuário e muito mais.
