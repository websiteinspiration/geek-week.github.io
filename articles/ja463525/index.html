セレン、セレノイド、セレニド、セレンドロイド...これはどういう意味ですか？

こんにちは、ハブル！

オートメーションの世界では、初心者がナビゲートするのはかなり困難です。多くの概念を学び、既存のツールの機能を理解する必要があります。たとえば、ここでは：セレン、セレン化物、セレノイド、セレンドリオド-それは何ですか、どのように違うのですか？はい、それらを比較できますか？

これを整理するのに役立つ記事を書きました。誰も気にしない、猫へようこそ！



セレン


セレンは自動化されたブラウザー管理のためのツールです。

Seleniumプロジェクトの一環として、一連のオープンソースソフトウェア製品が開発されています。

  • Selenium Server
  • セレングリッド、
  • Selenium IDEなど

Selenium Serverは、特別なドライバーを使用してブラウザーを管理できるようにするプログラムです。それは私たちが動作しなければならないブラウザと同じコンピュータ上に配置する必要があります。

Selenium Serverをインストールしたら、別のコンピューターからSelenium Serverにアクセスして、HTTP経由で作成された特別なプロトコルを使用してブラウザーをリモート制御できます。

概要 -Selenium Serverは、特定のマシン上のブラウザーの管理に役立ちます。

詳細については、https：//www.seleniumhq.org/を参照してください

。SeleniumGridは 1台のコンピューターにインストールされ、Selenium Serverがインストールされている他のいくつかのコンピューターとリモートで連携できます。

これは、Selenium Gridがインストールされている1台のマシンのみを参照して、多数のマシンでテストを実行するために作成されました。

セッションを作成する（つまり、ブラウザーを起動する）ために、私たちは常にSelenium Gridが立っているコンピューターにアクセスします。彼と彼自身は、このために負荷が少ないマシンを決定し、チームをリダイレクトします。

Selenium Gridが動作するすべてのマシンは、さまざまなオペレーティングシステムを実行でき、さまざまなブラウザをインストールできます。

簡単に言うと、Selenium Serverがインストールされている複数のマシンで作業を整理するには、Selenium Gridが必要です。

詳細については、こちらをご覧ください：https : //www.seleniumhq.org/docs/07_selenium_grid.jsp

Selenium IDE-これは、Firefoxブラウザ用のプラグインであり、ユーザーアクションを記録および再生できます。

Selenium ServerやSelenium Gridと特別な関係はなく、ローカルブラウザーでのみ操作できます。

詳細については、こちらをご覧ください：https : //www.seleniumhq.org/selenium-ide/

セレニド


Selenideは、Webアプリケーションの自動テストのためのフレームワークの1つです。これを使用すると、HTTPコマンドを生成してSelenium ServerまたはSelenium Gridに送信するコードをすばやく比較的簡単に作成できます。

彼は、必要な要素の検索、イベントの確認、UIの操作など、Webアプリケーションの動作をテストするスクリプトを書いたことで投獄されました。

プログラムのセレンラインと名前が似ているにもかかわらず、他の人がセレン化物自体を開発しています。

つまり、Selenideを使用すると、Webアプリケーションのテスト用に特別に調整されたコードを簡単に記述できます。

詳細については、https：//ru.selenide.org/をご覧ください。

セレノイド


Selenoidは、特別なドライバーを使用してブラウザーとAndroidエミュレーターを制御できるプログラムです。それぞれをDockerコンテナーで分離して実行できます。

SelenoidはSelenium Serverの代替ソリューションですが、本質は同じですが、ドライバーの編成です。

セレノイドの主なアイデアは、セッションごとに新しいコンテナーを起動し（新しいブラウザーまたはエミュレーターを起動）、セッションを閉じた直後にそれを停止することです。

Selenoidを使用すると、追加のリソースコストをかけずに高い負荷を維持できます。

詳細については、こちらをご覧ください：https : //aerokube.com/selenoid/latest/

セレンドロイド


Selendroidは、Androidベースのモバイルアプリケーションの自動テストのためのフレームワークです。

Androidの初期バージョンで使用されます-最大17レベルのAPI（Android 4.2）。しかし、それ以上ではありません。

簡単に言うと、これはAndroidアプリケーションでの作業にあまり関係のないフレームワークではありません。

詳細については、こちらをご覧ください：http : //selendroid.io
