一般的なEMアルゴリズム


コースでは、記事の作成に基づいて、EMアルゴリズムがクラスタリングの方法の1つとして示されています。つまり、本当の答えが事前にわからない場合は、教師なしの機械学習方式です。この記事では、正規分布を持つ確率変数のクラスタリング手法の1つとして、アルゴリズムについても考察します。ただし、アルゴリズムの適用範囲が広いことに注意してください。 EMアルゴリズム（英語の期待値最大化）は、隠れた変数を持つモデルの尤度関数の推定値を見つけるための一般的な方法です。分布の混合から、複雑な確率分布を（近似）構築できます。

：問題をクラスタリングにおけるEMアルゴリズムは、各反復において2つのステップを実行する反復アルゴリズムとして使用される

E-ステップ。最初のEステップでは、たとえば、ランダムに、非表示の変数を選択します。この場合、数学的期待値になります-μ と標準偏差- σ。選択した変数を使用して、各オブジェクトを特定のクラスターに割り当てる確率を計算します。後続のEステップは、Mステップで定義された隠し変数を使用します。

Mステップ。Mステップでは、Eステップで取得した特定のクラスターに各オブジェクトを割り当てる確率に従って、隠れた変数を再計算します。μ そして σ

収束が発生するまで反復が繰り返されます。たとえば、特定の定数内で、非表示変数の値が大幅に変化しない場合に収束が発生したと想定できます。記事で検討されている最後の例では、定数を設定しないため、各反復で隠し変数の値の変化を特定しません。もっと簡単にしましょう-アルゴリズムを一定のステップ数に制限します。

Mステップは理解するのに十分単純であり、最後の例でこれを確認します。この記事では、Eステップに焦点を当てます。

Eステップは、ベイズの定理に基づいており、それに基づいて、各オブジェクトが1つまたは別のクラスターに割り当てられる確率を決定します。この定理が何であるか覚えておきましょう。

ベイズの公式を思い出してください


Pxi(Bj)=P(Bj)⋅PBj(xi)P(xi)



どこ Pxi(Bj) -イベントでの確率 xi 仮説がありました Bj

P(Bj) -仮説の確率 Bj

PBj(xi) -イベントの発生確率 xi 仮説に従う Bj

P(xi) -イベントの発生確率 xi各仮説（イベントの総確率の計算式で計算）が

満たされることを条件とするxi すでに起こった、仮説を達成する確率 Bj上記の式を使用して再評価されます。と言えるP(Bj) これは、仮説（テスト前）の事前確率です。 Pxi(Bj) イベント後に評価された同じ仮説の事後確率は xiつまり、イベントが xi 確実に起こりました。

ベイズの定理による例1


2つの異なるマシンで製造された部品を受け取ったと想像してください。倉庫で受け取った製品の次の特性がわかっています

。全体で、部品-10,000個（N=10000）、最初の機械で製造された部品-6000個 （N1=6000）、2番目-4000個 （N2=4000）

1台目の機械で製造された標準（つまり、良品）製品の割合は0.9で、2台目の機械で製造された標準製品の割合は0.8です。

受け取ったパーツの中からランダムに1つを抜き出したのが標準でした。

次の確率を決定する必要があります

。1）パーツが1番目のマシンで生産されている。

2）アイテムは2番目のマシンで生産されました。

決定

1）イベントxこの例では、標準製品を抽出しています。

2）では、仮説を決めましょうBj。2つの部分がたくさんあります。つまり、2つの仮説があります。

仮説B1：誤って取り外された製品が1号機で生産されました。

仮説B2：誤って取り外された製品が2号機で生産されました。

したがって、最初のマシンで製造された製品を抽出する確率はP(B1) 構成する N1/N=6000/10000=0.6。

2番目のマシンで製造された製品を抽出する確率はP(B2) 構成する N2/N=0.4。また、標準品・不良品は問わず、どのロットからの出荷かが重要です。

3） 1号機で製造された製品から標準製品を抽出する確率（つまり、最初の仮説の下）は、1号機で製造された標準製品のシェアに相当し、0.9です。PB1(xi)=0.9。

製品を抽出する確率は、仮説2の対象となる標準製品です。PB2(xi)=0.8

4）製品のセット全体から標準製品を抽出する確率を決定します-P(xi)。イベントの総確率の式に従ってP(xi)=P(B1)⋅PB1(xi)+P(B2)⋅PB2(xi)=0.6⋅0.9+0.4⋅0.8=0.54+0.32=0.86。ここで私たちはそれを理解しています0.14 -これは、倉庫に到着したすべての部品から不良部品を抽出する確率であり、合計すると1になります。 (0.86+0.14=1)。

5）これで、問題を解決するためのすべてのデータが揃いました。

5.1）ランダムに抽出された標準部品がマシンで生成される確率を決定する№1：

Pxi(B1)=P(B1)⋅PB1(xi)P(xi)=0.6⋅0.90.86≈0.63



5.2）ランダムに抽出された標準部品がマシン上で生成される確率を決定します№2：

Pxi(B2)=P(B2)⋅PB2(xi)P(xi)=0.4⋅0.80.86≈0.37



したがって、私たちは仮説を再評価しました B1 そして B2。再評価後、仮説はイベントの完全なグループも形成します。0.63+0.37=1。

確率変数の正規分布のパラメーターを使用したベイズの定理の2番目の例：数学的期待値 μ と標準偏差 σ



前の例の条件を少し変更してみましょう。 1号機と2号機で製造された標準製品の比率はわからないと仮定しますが、代わりに、製品の直径の平均寸法と各機械の製品の直径の標準偏差がわかっているとします。

マシンNo. 2は、直径52 mm、標準偏差2 mmの部品を製造します。

重要な条件は、製品のセット全体が正規分布またはガウス分布で記述されていることです。

それ以外の場合、条件は同じです。次のように記述します。

N1=6000,μ1=64,σ1=4

N2=4000,μ2=52,σ2=2

製品の受け入れの過程で小さな出来事が発生したため、すべての製品が混合されたと想定しています。

私たちの仕事は、詳細を分類し、それぞれが1号機または2号機で生産された確率を決定することです。また、パーツが機械で製造されたものである可能性が高いと仮定します。

ソリューション

まず、ソリューションアルゴリズムを分析します。各仮説の確率は簡単に計算できますが、過去の例からそれらの値はすでにわかっています。P(B1)=0.6、 P(B2)=0.4。しかし、利用可能な仮説のそれぞれについて、標準製品を個別に押収する確率をどのように計算するのでしょうか。すべてが簡単です！確率は考慮せず、代わりに、各仮説の直径値を使用して、削除するアイテムの確率密度を個別に決定します。

これを行うには、よく知られている式を使用します。

ρBj(xi)=1σj⋅2π⋅exp⁡(−(xi−μj)22σj2)



どこ xi -ランダムな値（この場合、製品の直径の実際のサイズ）、

μj -確率変数の数学的期待値 j仮説（この場合、製造された製品の直径の平均サイズ） j工作機械）

σj -確率変数の標準偏差 j仮説（この場合、製造された製品の直径の平均サイズからの偏差） j工作機械）。

したがって、ベイズの式の分子で確率密度を確率で巧妙に置き換え、分母で同様の置換を実行します。

公式をケースに合わせて調整します。

直径寸法のパーツがxi マシンNo. 1で生成される式は、次の式で決定されます。

Pxi(B1)=w1⋅ρB1(xi)w1⋅ρB1(xi)+w2⋅ρB1(xi)



直径寸法のパーツが xi マシンNo.2で生産された

Pxi(B2)=w2⋅ρB2(xi)w1⋅ρB1(xi)+w2⋅ρB1(xi)



仮説の確率の表記を、 P(Bj) に wj。これは、以前に指定されたコースの対応する指定によるものです。さらに、このような表記法のみを使用します。

def gaus_func_01(mu,sigma,x):
    return math.e**(-(x-mu)**2/(2*sigma**2)) / (sigma*(2*math.pi)**0.5)

#        
def proba_x(X, w1, w2, mu_1, mu_2, sigma_1, sigma_2):
    for i in X:
        P1_x = gaus_func_01(mu_1,sigma_1,i)
        P2_x = gaus_func_01(mu_2,sigma_2,i)
        P_x = w1*P1_x + w2*P2_x
        P_x_1 = (w1*P1_x)/P_x
        P_x_2 = (w2*P2_x)/P_x
        proba_temp = []
        proba_temp.append(P_x_1)
        proba_temp.append(P_x_2)
        proba_X.append(proba_temp)
    return proba_X

def gaus_func_02(k, m, x, mu, sigma):
    pj_xi = []
    for j in range(k):
        det_sigma_j = np.linalg.det(sigma[j])

#   E-
def e_step(x, k, m, n, w, mu, sigma):
    #      i-     j-  
    pj_xi = []
    for j in range(k):
        det_sigma_j = np.linalg.det(sigma[j])
        factor_1 = 1 / (((2 * math.pi)**(k/2)) * ((det_sigma_j)**0.5))
        factor_2 = []
        for i in x:
            factor_2.append(math.e**float(
                -0.5 * np.matrix(i - mu[j]) * np.matrix(np.linalg.inv(sigma[j])) * np.matrix(i - mu[j]).T))
        pj_xi.append(factor_1 * np.array(factor_2))
    pj_xi = np.array(pj_xi)
    
    #     ,  i-    j- 
    pj_xi_w = []
    for j in range(k):
        pj_xi_w.append(pj_xi[j] * w[j])
    pj_xi_w = np.array(pj_xi_w)
    
    #     i-      
    sum_pj_xi_w = np.sum(pj_xi_w, axis = 0)
    
    #         
    proba_xi = []
    for j in range(k):
        proba_xi.append(pj_xi_w[j] / sum_pj_xi_w)
    
    return np.array(proba_xi)

def m_step(x, proba_xi,N):
    w_new = np.sum(proba_xi, axis = 1) / N
    
    #   
    mu_new = (np.array((np.matrix(proba_xi) * np.matrix(X))).T / np.sum(proba_xi, axis = 1)).T
    
    #  
    cov_new = []
    for mu in range(mu_new.shape[0]):
        X_cd = []
        X_cd_proba = []
        count = 0
        for x_i in x:
            cd = np.array(x_i - mu_new[mu])
            X_cd.append(cd)
            X_cd_proba.append(cd * proba_xi[mu][count])
            count += 1
        X_cd = np.array(X_cd)
        X_cd = X_cd.reshape(N, m)
        X_cd_proba = np.array(X_cd_proba)
        X_cd_proba = X_cd_proba.reshape(N, m)

        cov_new.append(np.matrix(X_cd.T) * np.matrix(X_cd_proba))
    cov_new = np.array((np.array(cov_new) / (np.sum(proba_xi, axis = 1)-1)))
    #             ,    
    #       ,        
    if cov_new[0][0][1] < 0:
        cov_new[0][0][1] = 0
    if cov_new[0][1][0] < 0:
        cov_new[0][1][0] = 0
    
    if cov_new[1][0][1] < 0:
        cov_new[1][0][1] = 0
    if cov_new[1][1][0] < 0:
        cov_new[1][1][0] = 0
    
    #   
    sigma_new = cov_new**0.5
    return w_new, mu_new, sigma_new
