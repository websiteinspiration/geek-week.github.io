Salesforce认证指南


本文对于Salesforce平台上的新手开发人员将非常有用。在本文中，我将尝试总结自己在通过Salesforce认证方面的经验，并讲述如何在时间和财务上花费最少的方式进行认证。

在过去的6年中，Salesforce在2019年被公认为CRM系统全球市场的领导者。目前，全球的劳动力市场对Salesforce专业化的需求很高。因此，如果您面临任务：

  • 证明您在Salesforce中的知识，
  • 找工作时在候选人中脱颖而出，
  • 加深您在Salesforce中的理论知识，
  • 来增加薪水。

在这种情况下，您肯定需要通过Salesforce认证。

Salesforce认证是考试形式的考试，通过后您将获得证书。然后，将您的数据输入到认证专家的数据库中，现在每个人（包括潜在的雇主）都可以检查您的站点上是否有证书。

我将有条件地将该过程分为六个主要阶段，我可以凭经验将它们选出来：

  • 步骤1.选择证书
  • 阶段2。在Trailhead平台上进行准备
  • 阶段3.第三方准备资源
  • 第4阶段。节省$ 70
  • 步骤5.注册认证
  • 阶段6.通过在线考试

让我们按顺序开始。

步骤1.选择证书


根据Salesforce的说法，可以获取30多个证书，层次结构表示形式可以完美地理解方向的依赖性，以及值得接收证书的顺序。


如果您在选择方向时遇到任何困难，则可以转到“ 认证指南”寻求帮助，该指南隐藏在众多页面和官方资源链接的开头。

我们打开指南，阅读每份证书的描述，找到最合适的证书并熟悉要求。


我们关闭了指南，准备离开6个月。

建议-“有6个月的Salesforce经验”？如果您明智地进行准备，那么所需的时间就会大大减少。

选择证书时，请记住以下几点：

  1. 如果证书是第一份，那么标记建筑师是没有意义的，因为 对于此步骤，需要通过前面的步骤（请参见金字塔）。
  2. 我建议开发人员看看Administrator \ App Builder \ Developer的指示
  3. 经理可以查看顾问\ Marketing Cloud \ CPQ

确定证书？精细！

阶段2。在Trailhead平台上进行准备


如果您已经熟悉Trailhead平台，那么我想指出的第一件事是，您不应该使用蛮力方法处理模块的传递，不要关注徽章的数量。为了优化花费的时间，徽章的数量应保持在最低要求。

如果您已经拥有Ranger等级，并且拥有100多个徽章，则不要认为这足以满足认证要求，这将不是在实践中需要做任何事情的能力，而是需要系统的深厚理论知识。

因此，在“认证”部分中，我们着眼于这些快乐的，充满知识的面孔，我们希望在准备过程中，我们能感受到同样的情感...


...，然后根据所选方向打开建议的培训模块集。

例如，考虑一下App Builder的工具包：

  1. 66个小时的学习模块可以轻松地转换为2个月的夜间练习，最终不会成功，因为 不会为考试提供足够的理论知识。
  2. 我们专注于链接和PDF，它们包含真正有用的信息。
  3. 如果时间允许，那么可以通过主要模块进行审查，但是为证书设置一个目标以使它们全部通过是不值得的。
  4. 最好不要在Superbadge上浪费时间，因为 仅了解任务可能需要几天，因此，整个过程将减少为在Internet上找到解决方案

奖励：对于那些对徽章感兴趣的人，我建议您通过两个俄语超级徽章：

  • Security Specialist -介绍安全设置，几乎在各个方向都可以找到的问题
  • App Customization Specialist -介绍Salesforce中声明式开发的基本方法

处理Trailhead？继续！

阶段3.第三方准备资源


第三阶段是准备认证的关键，因为 从现在开始，将有必要集中精力通过接近真实测试的测试。
这是为了达到要求而反复进行的测试，以便“飞出牙齿”问题的答案是成功通过认证的关键。

目前，有一些第三方资源可用于准备：

focusonforce.com


-体面的付费资源，完全致力于为Salesforce认证做准备。

该资源分为学习指南和实践考试两部分，每部分花费19美元（在本文发表时）。

学习指南是从官方文档中榨取的理论资料，并添加了图表和各种图形元素，以便于吸收。


我想立即指出，我不建议购买《学习指南》，因为其中的信息被极大地压缩了，考试中根本没有任何材料出现。

您应该注意练习考试，它以接近真实考试的一组测试形式提供。使用实践考试，您可以通过及时回答问题来模拟真实的测试条件。该数据库总共为每个证书包含约400个问题，这些问题会定期更新（根据资源的作者）。通过测试后，您可以按部分进行％解析来获得结果。


这样的报告可以直观地表示其中的部分存在知识问题，然后可以对该部分进行多次测试以得出该部分。

此外，针对每个问题都提供了答案，并提供了注释，说明为什么此答案正确无误，请参考官方文档。


该资源有一个在线论坛，您可以在其中讨论所有无法理解的时刻。主管部门每天（根据经验）回答所有问题。

论坛

如果在准备之后，您定期获得所有测试的90％，则可以认为证书在您的口袋里！
缺点：已支付资源。

t.me/SalesforceA


-电报频道，讨论了认证准备工作，并发布了测试。它可以作为免费资源的基础。

听众会说英语，而且非常活跃，您可以放心提出问题，并希望给出的答案是正确的。

缺点：在测试中发现错误的答案，受众是准备认证的人员，这些人并不总是对Salersforce有知识。

proprofs.com和quizlet.com


-各种测试的集合，其中包含有关Salesforce的大量测试。
缺点：很多测试答案不正确。许多错误组成的问题，一些测试已过时或格式错误。

乌迪米

-资源已支付，但是在资源上布置的课程有很多不足之处，其中大多数已过时，因为 认证问题每个季节都会更新。

第4阶段。节省$ 70


认证费用为200美元，这不能算是“令人愉悦的”价格，但是有机会节省70美元。

Salesforce定期举办免费的为期一天的在线认证培训课程，之后，您可以获得价值200美元的任何证书的70美元折扣券。就是说，它并没有在任何地方做广告，我偶然发现了这些信息，并在开拓者社区中成群结队地走着。

我们转到“ 认证天数”组中的Salesforce Trailblazer社区的官方网站，在其中可以找到课程信息或提出问题。


通过单击该链接，可以找出要针对哪些证书进行培训，在哪几天进行培训，还可以注册所选课程。


完成课程后，实际上包括在特定时间连接到在线平台并听两个小时的讲座，然后，几天后，承诺的优惠券将通过邮件到达。


步骤5.注册认证


官方认证平台是www.webassessor.com，注册后，将有机会选择考试。

Salesforce提供2种考试选项-在线和现场。

“现场”是指前往专业中心。可以在链接上查看其位置。

在“在线”选项中，您可以选择一周中的任何一天（包括周六和周日）以及最适合您通过认证的时间。对我来说是星期天13:00。

如果这是第一次考试，则除了Director Architect以外，该领域的管理员，App Builder，Developer，Acreditation等费用为200美元。如果您收到了优惠券，可以在注册时使用，因此，考试费用将减少至130美元。

如果在第一次尝试通过测试后失败，则可以重新参加考试。每次重考的费用为100美元。如果有免费入场券，您可以在同一天重新参加考试。对于第三次重考，您必须等待14天。
