ANSI C


由緒あるCから始めましょう。ここで紹介するプログラムは、cat「Cプログラミング言語」という本の修正版です。

/* mcat.c */
#include <stdio.h>

int main(int argc, char *argv[])
{
  FILE *fp;
  int c;

  if ((fp = fopen(*++argv, "r")) == NULL) {
    printf("mcat: can't open %s\n", *argv);
    return 1;
  }

  while ((c = getc(fp)) != EOF)
    putc(c, stdout);

  fclose(fp);

  return 0;
}

コンパイル：

$ gcc -o mcat mcat.c

起動：

$ ./mcat helloworld.txt
Hello world!

上記のコードに関する説明は次のとおりです。

  • プログラムは、コマンドライン引数として渡されたファイルを開きます。
  • ループwhileは、ファイルから標準出力ストリームにデータをコピーします。データはバイト単位でコピーされます。これはファイルの終わりに達するまで行われます。
  • プログラムがに到達するEOFと、ファイルを閉じて終了します。

Python 3


PythonではEOF、ANSI Cで使用できるものと同様に、を明示的にチェックするメカニズムはありません。ただし、ファイルを文字単位で読み取るEOF場合、次の読み取り文字を格納する変数が空の場合に状態を明らかにできます。

with open(sys.argv[1]) as fin:

プログラムを実行し、返された結果を確認します。

行く


Goでは、Read（）によって返されたエラーを明示的にチェックして、ファイルの最後に到達したことを示しているかどうかを確認できます。

// mcat.go
package main

import (
    "fmt"
    "os"
    "io"
)

func main() {
    file, err := os.Open(os.Args[1])
    if err != nil {
        fmt.Fprintf(os.Stderr, "mcat: %v\n", err)
        os.Exit(1)
    }

    buffer := make([]byte, 1)  // 1-byte buffer
    for {
        bytesread, err := file.Read(buffer)
        if err == io.EOF {
            break
        }
        fmt.Print(string(buffer[:bytesread]))
    }
    file.Close()
}

プログラムを実行します。

$ go run mcat.go helloworld.txt
Hello world!

JavaScript（Node.js）


Node.jsには、を明示的にチェックするメカニズムはありませんEOF。ただし、ファイルの終わりに達したときに何か他のものを読み取ろうとすると、ストリーム終了イベントが発生します。

/* mcat.js */
const fs = require('fs');
const process = require('process');

const fileName = process.argv[2];

var readable = fs.createReadStream(fileName, {
  encoding: 'utf8',
  fd: null,
});

readable.on('readable', function() {
  var chunk;
  while ((chunk = readable.read(1)) !== null) {
    process.stdout.write(chunk); /* chunk is one byte */
  }
});

readable.on('end', () => {
  console.log('\nEOF: There will be no more data.');
});

プログラムを実行します。

$ node mcat.js helloworld.txt
Hello world!

EOF: There will be no more data.

/* syscat.c */
#include <sys/types.h>
#include <sys/stat.h>
#include <fcntl.h>
#include <unistd.h>

int main(int argc, char *argv[])
{
  int fd;
  char c;

  fd = open(argv[1], O_RDONLY, 0);

  while (read(fd, &c, 1) != 0)
    write(STDOUT_FILENO, &c, 1);

  return 0;
}

それを実行します：

  • EOF -これはシンボルではありません。
  • ファイルの最後に特殊文字はありません。
