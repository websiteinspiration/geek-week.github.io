最後に、解決策が見つかりました。

秘密の回復は、ラグランジュ補間多項式を使用して実行できます。これを行うには、多項式を（0）で推定し、次に他のn個の点（nは多項式の次数）で推定します。多項式は明示的な形式で記述できます。ゼロでは、多項式は1です。他の点では0です。多項式の推定は線形関数であるため、同じ点で推定を推定して補間できます。

from functools import reduce
from operator import mul

def retrieve_original(secrets):
    x_s = [s[0] for s in secrets]
    acc = Mod(0, P)
    for i in range(len(secrets)):
        others = list(x_s)
        cur = others.pop(i)
        factor = Mod(1, P)
        for el in others:
            factor *= el * (el - cur).inverse()
        acc += factor * secrets[i][1]
    return acc
