「1か月で、私はフルスタックの開発者になりました。」学生がABBYYでインターンシップについて語る

すでにITを始めていますか？それとも、まだその仕事を探しているスマートフォンに行き詰まっていますか？インターンシップは、キャリアの第一歩を踏み出し、やりたいことを理解するのに役立ちます。

夏には、モスクワ物理技術研究所、HSE、およびその他の大学の学生である26人の研修生がチームに参加しました。彼らは2ヶ月（7月から8月）の有給インターンシップのために来ました。秋には、多くがパートタイムのインターンシップとしてABBYYとの共同作業を続け、何人かが常勤の職に転じました。インターンは研究開発部門の業務に携わっています。私たちはすでに、Instagram でStoriesのメンバーとのミニインタビューを行っていました。少し前に、インターンZhenya からのHabré への投稿がありました-ABBYYでの彼の練習について。

そして今度は、ABBYYでのインターンシップの感想を3人の生徒に話してもらいました。彼らは会社でどのような経験と知識をすでに得ていますか？勉強と仕事を組み合わせて燃え尽きない方法は？さて、ズーマー、今からすべてをお伝えします。



ABBYY：なぜあなたはこの夏ABBYYを選んだのですか？

エゴール：私たちはインターンシップについて話すために学部に来ました、ABBYYからの代表者がいました。私もキャリアフェアに参加し、この会社にも招待されました。C＃の開発者が必要でした。今私はこれをやっています。

Anya：FKKで夏のインターンシップに関するプレゼンテーションが行われたとき、最も印象に残ったABBYYのプレゼンテーションは、それが私の魂に直接沈んだということでした。



ITの歩みについて


ABBYY：今では誰もがITに乗りたいと思っているようです。なぜこの分野で最初に勉強したのですか？

Egor：面白かったよ。私はフィステに入らなかった。私はモスクワ物理技術研究所のライセウムで学び、物理学と物理学のクラスで、オリンピックのすべてのパズルをクリックしました。そして、卒業した年にすべてのオリンピアードが劇的に変化し、私はFizTechオリンピアードの勝者にはならず、受賞者になりました。したがって、私は試験なしでFizTechに入ることはできませんでした。しかし、私は彼らが私をHSEに連れて行っていることを誤って知りました。最高のコンピューター部門へ！つまり、Fiztehに、FRTK（無線工学とサイバネティックスの学部）に行きたかったのですが、彼らは私に「とにかくプログラミングに行く」と言いました。私は幸せでした。

ABBYY：Lesha、MIPTの画像認識とテキスト処理部門で勉強していますか？どう思いますか？

アレックス： 上手。好き。



ABBYY：勉強と仕事を組み合わせるのに役立ちますか？

アレックス：もちろん、ABBYYのオフィスではカップルがここで開催され、この時間は労働時間としてカウントされます。

Egor：今はうらやましいです。しかし、それほどではありません。Fiztekhでは、このシステムは私には学問的すぎます。私にとって、それは難しすぎるでしょう。私は、ソプロマットなど、あらゆる種類の必須科目について話しています。たとえば、HSEのFCNには物理学はありません。

仕事、勉強、時間管理について


ABBYY：仕事と勉強をどうやって組み合わせるのですか？

Egor：落ち着いて組み合わせる。私は自分のために雇用を選び、週に3日働きます。別のudalenkaが助かります：講義で働くことができます。

Anya：私は週20時間働いています。彼らはそれが1〜2ヶ月かかるだろうと言った、そして私はどれだけ働きたいかを決める。

アレックス：私は週32時間働いています。私自身が時間数を選択し、必要に応じて変更することができます。

ABBYY：事務所に来る予定はありますか？

アレックス：ノボダクナヤから9:21に電車があります。私はそこに住んでいるので、電車に慣れています [ LeshaはDolgoprudnyに住んで勉強しています ]。

エゴール：後で来る、9：20から10:20までの列車が私に近づく。どれに目覚めるか。夏はひどかった。彼は1日8時間働き、10：30から11:00に到着して19:00まで働こうとしました。そして今、毎週異なります。

Anya：私は地下鉄に乗る。しかし、私のスケジュールはペアにも依存します。

ABBYY：LeshaとYegor、あなたはすでにインターンからパーマネントポジションに切り替えています。いかがですか？

アレックス：これまでのところ、大丈夫です。夏のインターンシップで楽になったとは言えません。勉強が始まるとすぐに感じました。

エゴール：それどころか、私にとっては簡単になりました。夏には完全雇用があ​​り、それから勉強やその他すべてのための自由な時間がありました。私はすべての講義に行くわけではありません。15分のセミナーでは、彼らは絞りを教えて、そのトピックに関する問題を解決することができます。

アビイ：勉強と仕事を組み合わせたいが、方法がわからない学生に何をアドバイスできますか？

Egor：優先順位を付けます。

アレックス：主なことはリラックスできることです。

Egor：処理する-決して：燃え尽きることは不可能です。時間を管理することが重要です。時間管理が決定します。

アレックス：「気にしないでください」これは私たちが呼ぶものです。



Anya：事前に計画を立てる必要があります。通常、あなたはあなたの勉強の締め切りが何であるか、そしてあなたが一週間の仕事で何をする必要があるかを理解しています。

新しい知識とスキルについて


ABBYY：夏のインターンシップで成長したと思いますか？

Egor：もちろんです。活動の方向性を変えたわけではありませんが、ここを歩いていると、サイトやWebアプリケーションではなく、バックエンドを扱うつもりだと思いました。 ABBYYで1か月間、私はフルスタックの開発者になりました-私の上司は冗談でそう言った。私はJavaScriptを学び、JSでアプリケーションを作成し、テストして、よりユーザーフレンドリーにしました。このテストからASP.NETのサーバー部分も学びました。現在、サーバーとクライアントの両方の部分を行っており、私はフルスタックの開発者です。自分の見方を変えて、興味があったことを実感しました。



Anya：私は独学で、自分が取り組んでいる分野で体系的な知識を持ったことはありません。私は1つのプロジェクトを書き、Androidを知っていると思いました。しかし、彼女はABBYYに来て、アプリケーションアーキテクチャ、プロダクション、およびGITに関して多くの知識を得ました。私はこれを理解したと感じています。

ABBYY：この分野でさらに発展させたいですか？

Anya：どこかでやってみたい。これが私の最初のインターンシップで、次は何なのかわかりません。これが私のものかどうかを理解するには、もっと時間がかかります。

アレックス：ABBYYで、興味があることに気付きました。開発できる範囲は広いです。それ以前は、機械学習の経験がありましたが、バックエンドとクラウドを試したかったのです。私はインターンシップで、長い間これを行う準備ができていると判断しました。

Egor：私も同じ状況です。今後2年間は、おそらくテストを行う予定です。

ABBYY：Lesha、ABBYY部門で得た知識は役に立ちますか？

Alex：はい、もちろん。学科のプログラムは常に変化しています。より多くの実践があります。これは便利だと思います。

ABBYY：チームで仕事をしたり、自分で仕事をしたりしますか？もっと何が好きですか？



Anya：ABBYY Mobileでインターンシップに行ったとき、自分がチームとして成長することを実感し、それが欲しかった。３ヶ月経ちましたが、座って飛ばしたいだけです。それどころか、心理的には、チームで作業する方が簡単です。私はこれとそれを行うことができますが、時々一人で働きたいです。

Egor：私たちはたった2人のチームで、全員が研修生です。それぞれにタスクの独自のコンベヤがあります。つまり、それぞれが独自の部分に従事しています。私たちは積極的に誰かと対話することはありませんが、別のチームリードを割り当てました。

アレックス：私のインターンタスクはプロセス全体から少し離れていました。私は一人で勉強し、座って理解しました。このモードの方が好きです。複数の人が同じ仕事をしていて、誰もが同じことをするなら、それは私をやる気にさせます。これで8人のチームができました。スタンドアップがあります。

複雑で興味深いタスクについて


ABBYY：あなたの仕事の結果はすでにABBYY製品またはソリューションで使用されていますか？

Egor：はい、一番好きです。インターンシップ中に作成した私のアプリケーションは、積極的に使用されています。それはテストについて報告し、他の部門はすでにそれに興味を持っています。彼らはそれが主なものになると決めました、私たちはこれのために部門を割り当てました-FlexiCapture Automation。私の同僚と私は自動テストに従事しており、チームには他の開発者がいますが、彼らは他のタスクに取り組んでいます。より多くのテストにより、さまざまな国からの請求書をシステムを通じて実行するときに、会社の国際性を感じることができます。



アレックス：私も似たような状況です。仕事が怠惰ではなかったことを理解するのはうれしいことです。ABBYY FineReaderでログを保存および処理するためのアプリケーションを作成しました。マイクロサービスに関するタスクもありました。これらのサービスはすべてクラウドに収集され、同じシステムに格納されて相互に作用するように計画されています。このシステムでリクエストをトレースすることがいかに便利かについて実験を行い、ABBYY内部ナレッジベースの記事を書き、私が何をしたか、どのような問題が発生したかを伝えました。今後、この記事は他の従業員にも役立つでしょう。

Anya：まだ準備ができていません。あるリリースでは、私がやっていることをプロダクションに入れ、人々が感じて試してくれることを願っています。

ABBYYチームの機能


ABBYY：部署でのインターンシップは誰にすすめますか？

アーニャ：チームで働く方法を知っていて、学校を適切に認識している人。

アレックス：それを哲学的に考えてください。

Egor：ええ、そうですね。これはすべてのITに起因していると思います。コミュニケーションを取り、自分のやっていることを説明できなければなりません。

Alex：聞いて。

ABBYY：あなたの意見では、ABBYYの企業文化は誰に適していますか？

Egor：学生向け-完璧です。

レッシュ：特に学生FIVTa [ FIVT -技術革新と高い技術MIPTの学科 ]。

エゴール：私たちの部署の責任者が会社に留まりたい条件を尋ねると、以前は別の場所でどのように働いていたのかを話し、1人の学生が仕事のためにアカデミーに行きました。学校を中退しないようにと彼はアドバイスしてくれたので、柔軟なスケジュールで取り組んでいます。



エゴール：ここで、生徒は最大限に会います。どこが同じかわからない。そして、ABBYYでの仕事は、意図的で落ち着きがあり、コミュニケーション、耳を傾け、理解し、説明することができます。

自由時間について


ABBYY：もちろん、インターンシップや勉強からの自由な時間に何をしていますか。

Anya：最近、スポーツを始め、ジムに行きました。大学でさえ、彼女は4つの異なる科目の助教になりました。

アレックス：走っています。週末にモスクワに遊びに行き、修理をします。

エゴール：私は散歩しています。基本的にはもちろん、女の子と過ごしたり、バーに行ったりしています。

ABBYY：ITのメディアやインフルエンサーをフォローしますか？

Alex：「典型的なプログラマー」

エゴール：私はJavaScriptとYevgeny Kovalchukが率いるフロントエンドを介してYouTubeチャンネルを視聴しました。

ITの未来について


ABBYY：10年後のテクノロジーの未来をどう見ていますか、そして私たちの生活はどのように変化しますか？

エゴール：すべてが非現実的な速度で飛ぶので、予測することは不可能です。しかし、私はまだ量子コンピューターが出てくるのを待っています。彼らのリリースで多くが変わるでしょう、しかしどのように正確に-誰も知りません。

Alex：量子コンピューターについても考えた。数十億回ではないにしても、通常の数百万回よりも高速になります。

Egor：理論的には、量子コンピューターの大量出力により、すべての暗号化とハッシュは、それらを拾うことができるので飛ぶでしょう。

Alex：すべてやり直さなければならない。量子コンピューターがそれらを解読することを学べば、量子コンピューター上で新しいハッシュを思いつくことは可能だと聞いた。

アーニャ：そして、私たちの生活のほとんどすべてがモバイルデバイスに行くと思います。間もなくプラスチックカードはなくなり、クレジットも他のカードもなくなるようです。



Egor：ソフトウェア開発に関しては、すべてがゆっくりと完全にインターネットに流れ込んでいます。インターネットの速度が上がるにつれて、すべてがクラウドに変わるように思えます。

Alex：要するに、クラウドは通常のトピックです。


ABBYYでキャリアを始めたいですか？私たちのページにアクセスし、アンケートに記入して、インターンシップの選考への招待状を最初に受け取り、教育プロジェクト、講義、マスタークラスについて学びましょう。私たちも定期的に開いた位置先輩学生や最近の卒業生のために。
