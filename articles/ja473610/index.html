新しいPlesk Obsidian Webコンソールの解析

最近、ホスティングおよびサイトコントロールパネルであるPleskのレビューをすでに公開しています。リンクを使用すると、コンソールと開発者に関する基本情報を確認し、さまざまなユーザーグループの機能とサイト管理者のインターフェイスについて知ることができます。この記事では、先日リリースされた新しいバージョンのパネルについて説明します。PleskObsidianは、VPSを注文すると無料でライセンスを取得できます。


Pleskは、サーバー、サイト、アプリケーション、ホスティング、クラウドビジネスでテストされた、基本的な機能を備えたウェブコンソールから強力な管理プラットフォームへと進化を続けています。 Plesk Obsidianを使用すると、Webプロフェッショナル、リセラー、サービスプロバイダーは、あらゆる規模のサーバー、アプリケーション、Webサイト、ホスティング企業をインテリジェントに管理、保護、実行できます。 

Pleskは、業界は急速に変化していると考えています。

Plesk Obsidianの新しい主な機能 


quick迅速なアプリケーションとサイト開発のための最新のWebスタック


Pleskを備えたObsidianは、最適化されたすぐに使用可能なWebスタックであり、開発者向けのすべての導入オプションと便利なツール（Git、Redis、Memcached、Node.jsなど）を備えた革新的なプラットフォームです。


PHP Composer-PHPの依存関係マネージャー

Web開発者が直面する多くの問題の1つは、依存関係に関連しています。新しいパッケージをプロジェクトに統合すると、多くの場合、メリットよりも面倒になります。これは特にPHP開発者に当てはまります。多くの場合、プログラマーは最初からモジュールを構築し、Webページ間でデータの一貫性を実現するのは面倒です。その結果、優れた開発者は冗長なタスクに多くの時間とリソースを費やすと同時に、生産性を高め、新しいコードを迅速にリリースしたいと考えています。そのため、Plesk Obsidianには、PHPプロジェクトの依存関係を簡単に管理できるようにする、PHP用のエレガントでシンプルな依存関係マネージャであるComposerがあります（拡張機能は手動でインストールされます）。

Docker NextGen-Dockerの便利なサービス提供機能

仮想マシンの代わりにコンテナでアプリケーションを実行することは、ITの世界で勢いを増しています。テクノロジーは、ソフトウェア業界の最近の歴史の中で最も急速に成長しているものの1つと考えられています。これは、ユーザーがコンテナー内のアプリケーションを簡単にパッケージ化、配布、および管理できるプラットフォームであるDockerに基づいています。 Docker NextGen関数を使用すると、すぐに使えるDockerベースのソリューション（Redis、Memcached、MongoDB、Varnishなど）を使いやすくなり、便利なDockerテクノロジー自体を明らかにする必要がなくなりました。ウェブサイトのサポートサービスはワンクリックで展開されます。 Pleskはサービスを構成してから、簡単に自動的にWebサイトに統合します。 （近日公開）。

Plesk Security Core


Webサイトへの最も一般的な攻撃や悪意のあるユーザーに対するファーストクラスのサーバー間保護がデフォルトで有効になっています。


デフォルトのホスティング

  • Mod_security（WAF）とfail2banはそのままでアクティブです。
  • デフォルトで、systemdは5秒後に失敗したPleskサービスを自動的に再起動するようになりました。
  • 新しく作成されたWebサイトでは、SEO用に最適化されたHTTP> HTTPSがデフォルトで有効になっています。
  • システムベースのLinux（CentOS 7、RHEL 7、Ubuntu 16.04 / 18.04、およびDebian 8/9）では、Plesk緊急サービスが自動的に再起動するようになりました。
  • PHP-FPMの制限は、しばしばmax_childrenと呼ばれ、サーバー上で実行される並列PHP-FPMプロセスの最大数を構成することです（以前は5つありました）。
  • SPF、DKIM、DMARCが、受信メールと送信メールでデフォルトで有効になりました。

ユーザーエクスペリエンス、UX



サーバーとウェブサイトの管理の

簡素化Plesk Obsidianは、サーバー管理をさらに簡単にする、完全に新しく合理化されたユーザーインターフェースを備えています。これで、顧客はすべてのWebサイトを1つの画面で快適に操作できます。詳細を参照し、一括管理を選択するか、選択したCMSの適切な機能とコントロールを使用して、リストまたはグループの形式で一度に1つずつ操作します。

インターフェースは、より便利で簡単になり、目に優しくなりました。色とフォントサイズが最適化され、すべての要素がグリッド上に配置されます。効率を上げるために、左側のメニューを減らすことができます。グローバル検索がより見やすくなりました。

サブスクリプション間でドメインを移動する

以前は、これは困難な手動タスクであり、高度なサーバー管理者スキルが必要でした。 Plesk Obsidianを使用すると、コンテンツ、設定ファイル、ログファイル、PHP設定、APSアプリケーション、およびサブドメインとドメインエイリアス（存在する場合）を使用して、ドメインを別のサブスクリプションに簡単に移動できます。コマンドラインからこれを行うことができます。
