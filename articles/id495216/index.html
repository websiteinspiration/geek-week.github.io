Belajar, bernegosiasi, konsultasi jarak jauh - Bagian 2: Vinteo Server - 3 bulan konferensi video gratis

Peristiwa yang terjadi sekarang di seluruh Dunia tak terhindarkan terus mempengaruhi industri komunikasi jarak jauh. Bertahun-tahun yang lalu, konferensi video (VKS) adalah tanda kekayaan perusahaan. Dalam beberapa tahun terakhir, setelah pengembangan solusi perangkat lunak untuk komunikasi video, VKS dari kategori mewah pindah ke kategori alat bisnis yang efektif. Tapi sekarang, situasi karantina membuat sistem konferensi video menjadi keputusan terpaksa untuk menyediakan pekerjaan atau pelatihan jarak jauh.
Di satu sisi, saya senang bahwa tren ini telah menjadi lebih dan lebih populer selama beberapa minggu terakhir, tetapi di sisi lain, saya tidak senang bahwa ini terjadi dalam keadaan yang menyedihkan.

Beberapa bulan yang lalu, saya adalah pelanggan pertama dan satu-satunya ke pusat konferensi video - dan sekarang ada hampir tiga ratus dari kita.

Dalam beberapa minggu ke depan, saya berencana untuk menerbitkan sejumlah artikel dari seri "Pelatihan, Negosiasi, Konsultasi Jarak Jauh" di hub ini , yang membahas solusi profesional dari sistem konferensi video, menyediakan waktu karantina untuk digunakan secara gratis dalam kondisi nyata di infrastruktur perusahaan yang ada. Dalam artikel sebelumnya, saya menggambarkan Mind Server dan versi gratisnya . Artikel ini didedikasikan untuk aksi dari Vinteo, satu-satunya pengembang Rusia dari solusi konferensi video massal yang dibangun di atas arsitektur SIP / H.323 klasik.

Inti dari tindakan


Sederhana - Anda mendapat kesempatan untuk menggunakan versi lokal lengkap dari server Vinteo selama 3 bulan secara gratis . Jumlah koneksi hanya dibatasi oleh sumber daya komputasi perangkat keras server Anda.
Jika Anda perlu mengatur pekerjaan jarak jauh karyawan, atau menyediakan komunikasi video berkualitas tinggi dengan mitra dan pelanggan, gunakan promosi untuk berkenalan dengan sistem konferensi video profesional Vinteo.

Vinteo Server


Server konferensi video Server Vinteo termasuk dalam daftar terpadu program Rusia untuk
komputer elektronik dan basis data atas perintah Kementerian Komunikasi Rusia No. 165 dari 04/18/2016. untuk No. 142.

Kemungkinan untuk menghubungkan pelanggan


  • Melalui gateway (trunk) . Menghubungkan PBX dan VKS tradisional (Polycom, Cisco, dll.). Membuat rencana penomoran terpadu dan kemampuan untuk memanggil nomor pendek ke sambungan telepon rumah dan ponsel, dalam konferensi, ke terminal VKS dan klien perangkat lunak. Dimungkinkan untuk mengunduh Pelanggan melalui LDAP
  • Melalui SIP / H.323 . Koneksi terminal konferensi video dan klien perangkat lunak pada PC dan perangkat seluler dengan kemampuan untuk mendaftar di server Vinteo.
  • Melalui WebRTC . Koneksi melalui browser, dengan kemampuan untuk menerima / mengirimkan audio, video, dan konten.

Untuk koneksi apa pun ke server, Anda dapat mengatur pengaturan individual untuk protokol penyandian, bit rate dan resolusi.



Bekerja dengan tata letak


Server memiliki 35 tata letak standar untuk konferensi. Dimungkinkan untuk menggunakan mosaik otomatis yang akan beradaptasi dengan jumlah peserta yang terhubung.


Untuk seluruh konferensi atau peserta mana pun, Anda dapat menetapkan tata letak tertentu, sementara Anda dapat menentukan posisi dosen, menetapkan peserta ke lokasi tertentu di tata letak, mengatur panel gulir untuk menampilkan peserta di layar dengan frekuensi yang ditentukan, jika mereka berada di belakang layar.

Tata letak pribadi untuk solusi SIP / H.323 VKS adalah fungsi unik dan nyaman yang tidak dimiliki oleh banyak solusi pada level ini.


Untuk mengembangkan desain tata letak Anda sendiri, konstruktor opsional disediakan. Ini memungkinkan Anda untuk membuat banyak opsi tata letak menggunakan 3 lapisan overlay. Semua tata letak yang disimpan setelah disimpan menjadi tersedia di menu konferensi.


Rekaman Konferensi


Record adalah fungsionalitas server Vinteo standar yang tidak memerlukan lisensi tambahan. Untuk merekam, Anda dapat memilih tata letak terpisah dari jendela peserta, berbeda dari tata letak yang digunakan untuk konferensi umum.


Mendukung ekspor file rekaman dalam berbagai format.


Siaran Konferensi


Mode webcast mendukung penggunaan tata letak yang dipersonalisasi, serta obrolan pengguna.


Mode pengguna dirancang untuk mentransmisikan aliran konferensi video ke server pihak ketiga yang disiarkan, direkam, atau video surveillance.
