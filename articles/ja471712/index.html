React Nativeでのモバイルチャットアプリケーションの作成

これは、apollo-server-koaとreact-nativeを使用してチャットを作成することを目的とした記事の2番目の部分（最初はここ）です。モバイルチャットアプリケーションの作成について説明します。前のパートでは、このチャットのバックエンドがすでに作成されています。このプロセスの詳細については、リンクを参照してください。

まず、expo-cliを使用して、新しいexpoプロジェクトを作成します。これを行うには、ターミナルでコマンドを実行します。

expo init

そして、ウィザードのすべての手順に従って、空のアプリケーションを作成します。

私たちが作成したアプリケーションを受け取った後、将来的に使用される、依存関係を追加する必要がある：

完了しました。次のステップは、バックエンドと連携するようにapolloクライアントを接続して構成することです。これを行うには、以下のコードをApp.jsに追加します。ここでは、httpおよびwebsocketプロトコルを介してバックエンドに接続するようにapolloを構成し、apolloのメモリ内キャッシュを作成し、2つのルートを作成することに注意してください。1つはログインフォーム用、もう1つはチャットページ用です。

get link() {
  const BACKEND_URL = Platform.OS === 'ios'
    ? 'localhost:3000'
    : '10.0.2.2:3000';

  const httpLink = new HttpLink({
    uri: `http://${BACKEND_URL}/graphql`
  });

  const wsLink = new WebSocketLink({
    uri: `ws://${BACKEND_URL}/graphql`,
    options: {
      reconnect: true
    }
  });

  return split(
    // split based on operation type
    ({ query }) => {
      const definition = getMainDefinition(query);
      return (
        definition.kind === 'OperationDefinition' &&
        definition.operation === 'subscription'
      );
    },
    wsLink,
    httpLink,
  );
 }

サブスクリプション機能を使用して、新しいチャットメッセージに関する通知を受け取り、メッセージが現在のユーザーによって作成されていない場合にそのスレッドのみを処理します。

useSubscription(MESSAGE_CREATED, {
 onSubscriptionData: ({ client, subscriptionData: { data: { messageCreated } } }) => {
   const { getLast100Messages } = client.readQuery({ query: GET_LAST_100_MESSAGES });

   if (messageCreated.user.id !== id) {
     client.writeQuery({
       query: GET_LAST_100_MESSAGES,
       data: {
         getLast100Messages: [
           ...getLast100Messages,
           messageCreated
         ]
       }
     });
   }
 }
});

もう1つの興味深い点は、既存のメッセージが含まれるチャットをダウンロードすると、最も古いメッセージが画面の上部に表示されます。これはユーザーにとっては珍しいことです。アプリケーションの動作をより自然にするため、次のコード行を追加します。

const scrollEl = useRef(null);
…
<Content
  ref={scrollEl}
  onContentSizeChange={() => {
    scrollEl.current.wrappedInstance.scrollToEnd({ animated: true });
  }}

これで、初期データを受信したとき、または新しいメッセージを追加したときに、チャットは最後のメッセージまでスムーズにスクロールします。
