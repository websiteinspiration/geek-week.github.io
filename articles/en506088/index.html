This is some kind of toy programming. Well, abandoned to hell.

I agreed, studied it, and now it is one of my favorite PLs. But. Typescript - this is not a sysharp at all. This is a language with a fundamentally different type system. Complex, powerful, but different.


There are several typification parameters themselves.


There is static / dynamic typing. Static is when data types are known at the compilation stage. The compiler knows the types of variables, and on their basis checks the correctness of the program. Dynamic is when types are known only at run-time, and the compiler does not check anything during assembly.
