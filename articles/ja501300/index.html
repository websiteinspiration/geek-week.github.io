AtlassianサーバーとデータセンタープラグインのReactとAtlaskit

みなさん、こんにちは！

この記事では、サーバー環境とデータセンター環境のAtlassianプラグインでReactとAtlaskitを使用する方法について説明します。

前書き


現在、サーバーおよびデータセンター用のアトラシアン製品のプラグインを開発している場合は、ボックスからユーザーインターフェイスを開発するために、vm、soy、requirejs、jquery、backboneを使用できます。ここでは、すぐに使用できるライブラリーの使用方法に関する私の記事を読むことができます。

このテクノロジースタックは古く、新しいものを使用したいと考えています。私が選んだtypescriptですと反応するスタックとして。また、サーバープラグインをクラウドに簡単に転送できるようにするために、ユーザーインターフェイス要素のライブラリatlaskitを選択しました。

この記事ではAtlassian Jiraについて説明しますが、他のAtlassianサーバーおよびデータセンター製品にも同じアプローチを使用できます。

この記事の例を再現するには、gitとAtlassian SDKをインストールする必要があります。

それでは始めましょう！

Mavenアーキタイプをインストールし、新しいプロジェクトを作成します。


Mavenアーキタイプを作成して、ReactとAtlaskitでプラグインを作成するために必要なすべての設定が既に含まれている新しいプロジェクトを簡単に作成できるようにしました。

アーキタイプを使用したくない場合は、このアーキタイプから既に作成されているプラグインを取得して、「プロジェクトをアセンブルして起動する」の部分に移動できます。

私のBitbucketリポジトリからアーキタイプのクローンを作成します。

git clone https://alex1mmm@bitbucket.org/alex1mmm/jira-react-atlaskit-archetype.git --branch v1 --single-branch

jira-react-atlaskit-archetypeフォルダーフォルダーに移動し、このアーキタイプをローカルのMavenリポジトリにインストールします。

cd jira-react-atlaskit-archetype
atlas-mvn install

その後、1レベル上のフォルダーに移動し、このアーキタイプに基づいて新しいプロジェクトを作成します。

cd ..
atlas-mvn archetype:generate -DarchetypeCatalog=local

希望のアーキタイプを選択するように質問されます。これがローカルリポジトリの私の原型です：

1: local -> com.cprime.jira.sil.extension:sil-extension-archetype (This is the com.cprime.jira.sil.extension:sil-extension plugin for Atlassian JIRA.)
2: local -> ru.matveev.alexey.sil.extension:sil-extension-archetype (This is the ru.matveev.alexey.sil.extension:sil-extension plugin for Atlassian JIRA.)
3: local -> ru.matveev.alexey.atlas.jira:jira-react-atlaskit-archetype-archetype (jira-react-atlaskit-archetype-archetype)

ru.matveev.alexey.atlas.jira：jira-react-atlaskit-archetype-archetypeを選択する必要があるため、答えとして3を指定しました。

次に、新しいプロジェクトのgroupidとArtifactidを指定する必要があります。

プロジェクトを組み立ててインストールします


私の場合、新しいプロジェクトはmy-tutorialフォルダにあります。このフォルダーに移動して、プロジェクトをビルドします。

FormServlet.java：

public class FormServlet extends HttpServlet{
    private static final Logger log = LoggerFactory.getLogger(FormServlet.class);
    private final ResourceService resourceService;
    private final SoyTemplateRenderer soyTemplateRenderer;

    public FormServlet(@ComponentImport  SoyTemplateRenderer soyTemplateRenderer, ResourceService resourceService) {
        this.resourceService = resourceService;
        this.soyTemplateRenderer = soyTemplateRenderer;
    }

    @Override
    protected void doGet(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException
    {
        String pluginKey = this.resourceService.getProperty("atlassian.plugin.key");
        Map<String, Object> map = new HashMap<>();
        map.put("contextPath", req.getContextPath());

        String html = soyTemplateRenderer.render(pluginKey + ":jira-react-atlaskit-resources", "servlet.ui.form", map);

        resp.setContentType("text/html");
        resp.getWriter().write(html);
        resp.getWriter().close();    }

    @Override
    protected void doPost(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException {
        StringBuffer jb = new StringBuffer();
        String line = null;
        try {
            BufferedReader reader = req.getReader();
            while ((line = reader.readLine()) != null)
                jb.append(line);
        } catch (Exception e) { /*report an error*/ }
        log.info(String.format("Post Data: %s", jb.toString()));

        String pluginKey = this.resourceService.getProperty("atlassian.plugin.key");
        Map<String, Object> map = new HashMap<>();
        map.put("contextPath", req.getContextPath());

        String html = soyTemplateRenderer.render(pluginKey + ":jira-react-atlaskit-resources", "servlet.ui.form", map);

        resp.setContentType("text/html");
        resp.getWriter().write(html);
        resp.getWriter().close();
    }}

2つの変数resourceServiceとsoyTemplateRendererを定義し、これらの変数をクラスコンストラクターで初期化します。resourceService-maven.propertiesファイルからプロパティを読み取るBean。soyTemplateRenderer-大豆テンプレートを呼び出すことができるJira Bean。
doGetメソッドで、atlassian.plugin.keyプロパティの値とコンテキストパスを取得します。次に、コンテキストパスをパラメーターとしてsoyテンプレートに渡し、soyテンプレートをservlet.ui.formという名前で呼び出します。
