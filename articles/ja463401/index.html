ナポリからローマへ：新しいAMD EPYC CPU



8月7日、第2世代AMD EPYC™ラインが世界中で発売されました。新しいプロセッサはZen 2マイクロアーキテクチャに基づいており、7 nmプロセスで構築されています。

特徴



「彼らは衣服で迎えられ、心で護衛されます」とフォークの知恵は言います。それでは、新しい世代の「服」から始めましょう。プロセッサーのマーキングが若干変更されました。世代を表す4桁目が1から2に変更されました。前と同じように、最初の数字はシリーズを示し、2番目と3番目のモデルはモデルを示します。 AMDは、マルチソケットシステムでの作業をサポートしないPインデックスのプロセッサを放棄していません。

第2世代は変更なしで第1世代からSP3ソケットを継承しました。これにより、マザーボードを更新せずに新しいプロセッサを使用できますが、これはそれらの潜在能力を完全に明らかにすることはできません。 DDR4メモリ用に3200 MHzをサポートする新しいマザーボードを使用すると、完全なパフォーマンスを実現できます。

プロセッサの「充填」は認識を超えて変化しました。技術プロセスが変化し、新しいZen 2マイクロアーキテクチャが適用され、新しい高速メモリコントローラが登場しました。

パフォーマンス


7 nmプロセス への移行により、結晶が圧縮され、コアの数が64に増えました。これは、第1世代の2倍です。第2世代プロセッサの基本周波数は、2.00〜2.90 GHzの範囲です。比較のために、第1世代のベースは2.30 GHzを超えませんでした。コアの数とプロセッサの基本周波数の増加により、予算バージョンでは最大120 ワット、上位バージョンでは最大225ワットの放熱が増加しました。

プロセスの改善だけがイノベーションではありません。Zen 2と呼ばれる新しいマイクロアーキテクチャプロセッサーの技術的特性の改善に貢献しました。改善はキャッシュとの相互作用を目的としています。L1とのデータ交換速度が2倍になり、キャッシュ間のデータ転送速度が向上し、L3キャッシュのサイズが増加しました。

ローマは、外部デバイスとの連携の改善を誇っています。初めて、外部x2APIC割り込みコントローラーと連携できるようになり、DDR4-3200および128 PCIeレーンもサポートするようになりました。新世代のプロセッサは、仮想マシンが物理的な周辺機器に直接アクセスできる仮想化IOMMUテクノロジーのサポートを含みます。

容量を増やし、仮想化の条件を改善すると、同時に実行されるアプリケーションの数が増加します。アプリケーションは、強化された保護を必要とする機密データを扱います。

安全性


更新されたプロセッサのラインでは、セキュリティの問題は最後の場所ではありませんでした。システムの内部では、ARM®Cortex®A5に基づくAMDセキュアプロセッサと呼ばれるチップにプロセッサがインストールされており、AES-128アルゴリズムを使用してキーを保存し、RAMの内容を暗号化します。AMD Secure Processorは2種類のメモリ暗号化を提供します（これらの方法にはOSサポートが必要です）：



  • SME（Secure Memory Encryption）;

    SMEは単一のキーでメモリを暗号化し、コールドブート攻撃などの物理的な攻撃から保護します。このタイプの暗号化を使用しても、ユーザーアプリケーションを変更する必要はありません。OSは、暗号化が必要なメモリページにマークを付けます。
  • SEV（Secure Encrypted Virtualization）。

    SEVは、仮想マシン（VM）を操作するときにセキュリティを提供するように設計されています。ハイパーバイザーと各VMが使用するメモリは、独自のキーで暗号化されます。このアプローチは、ハイパーバイザーとVMを互いに暗号的に分離します。

テスト


特にインテル®プロセッサーとAMDの比較について話しているので、テストはできるだけ客観的でなければなりません。したがって、最適化コンパイラを使用して、ソースコードとして提供されるテストをビルドしません。

GeekBench 4


GeekBenchは、独自のオンライン結果データベースを備えた人気のあるクロスプラットフォームプロセッサパフォーマンステストです。テストは既製の実行可能ファイルの形式で行われるため、特定のプロセッサー向けの最適化は提供されていません。

私たちにとって、GeekBenchグループの一般的な指標は重要です。

  • 暗号スコア;
  • 整数スコア;
  • 浮動小数点スコア。
  • 記憶スコア。

示されたテストのグループは、2つのモードで起動されます：シングルスレッドで、およびすべてのコアで同時に実行された場合結果に基づいて、GeekBenchは最終的な評価を付けます：シングルコアスコアとマルチコアスコア。


AMD EPYC™7452とインテル®Xeon®ゴールド5218の間の主な戦いは、すべてのテストで第1世代のEPYC™が第2世代よりも劣っています。

シングルスレッドテストを検討してください。ローマは、暗号化タスクとメモリを操作するときに優れた結果を示しますが、整数計算を実行すると失われます。その結果、第2世代のEPYC™は4893ポイントを獲得し、シングルコア候補の勝者となります。 2位と3位は、それぞれ第1世代のXeonとEPYCが4695ポイントと3981ポイントで占めています。


マルチスレッドテストでは、パワーのバランスが大幅に変化します。EPYC™7452は優れた計算能力を発揮しますが、暗号化タスクとメモリの操作に負けてしまい、マルチコア候補で96009ポイントのリーダーになることを妨げません。

スペックCPU 2017


SPEC CPU 2017は、プロセッサメーカーによって認められた一連のパフォーマンステストです。このセットのテストはソースコードの形式で配布されます。これにより、特定のオペレーティングシステムの特定の機器用にテストを最適化できます。

SPEC CPUは4つのテストスイートで構成されています。

  • int_rate;
  • int_speed;
  • fp_rate;
  • fp_speed。

テスト名の最初の部分は、プロセッサーでの計算のタイプを決定します。整数（int）または浮動小数点数（fp）以上です。2番目の部分は、テストのタイプ（シングルコア（レート）またはマルチコア（速度））を決定します。

4つすべてのテストスイートを実施しました。テストは、GNU 4.8.5コンパイラスイートを使用して、第3レベルの最適化でコンパイルされます。マルチコアテストは64スレッドで実行され、シングルコアテストは32コピーで実行されました。


最適化でコンパイルされたテストは、GeekBenchと一貫性のない結果を示します。第2世代AMD EPYC™は、intspeedを除くすべてのテストでIntel®プロセッサーを搭載したデュアルソケットシステムより優れていますが、GeekBenchよりもマージンが大幅に小さくなっています。

  • 暗号化タスクソリューション（Botan、OpenSSL、John the Ripper）;
  • レイトレーシングを使用したイメージレンダリング（C-Ray、POV-Ray、Smallpt）;

テストで初めて、AMD EPYC™7551が最初に登場します。AMDプロセッサは、世代に関係なく、メモリ、レンダリング、およびビデオトランスコーディングを使用したシングルスレッド処理に優れています。Intel®プロセッサは、GeekBenchテストで以前に明らかにされたように、暗号化タスクとメモリを使用したマルチスレッド処理により優れています。

調査結果


さまざまなテストにもかかわらず、第1世代と第2世代のAMD EPYC™の選択は明白です。大多数のテストで、ローマは前任者より優れています。ただし、第1世代は、メモリとレンダリングの操作で地位を失うことはありません。

Intel®とAMDプロセッサーの比較は、詳細な検討を必要とする巨人たちの真の戦いです。インストールされたソフトウェアソリューションは、第2世代AMD EPYC™を使用すると、平均してパフォーマンスが向上します。特に、ローマはマルチスレッドコンピューティングとシングルスレッドの暗号化タスクで優れた結果を示しています。ソースコードからコンパイルされたソフトウェアを使用する場合、新世代のEPYC™プロセッサが優先されます。これは、浮動小数点テストで先行しており、マルチスレッドの整数計算でわずかに遅れています。

インテル®Xeon®ゴールドは、マルチスレッド暗号化、オーディオトランスコーディング、メモリ処理において優れた結果を示しています。インテル®Xeon®プロセッサーを搭載したデュアルソケットシステムは、Webサーバーリクエストの処理時に優れたパフォーマンスを発揮しました。

要約すると、実行されたテストは総合的なものであり、実際のタスクの結果は異なる場合があることに注意してください。特定のタスクで正確な結果を取得するには、追加のテストが必要です。ここで、コストについて簡単に説明します。AMDEPYC®7452の推奨価格は2025ドル、インテル®Xeon®ゴールド5218の場合-1250ドル、つまりデュアルソケットシステムを構成する場合は2500ドルです。
新しいAMD EPYC™7452は、まもなくSelectelラボで入手可能になります。
