Ahora sigues con emociones

¡Hola, soy Lena de JetBrains! En la situación actual, parece especialmente importante para usted y el equipo de YouTrack cumplir nuestras promesas con respecto a lo que se planea lanzar para 2020. Por lo tanto, me complace presentar YouTrack 2020.1, en el que dicha alegría y otras emociones ahora se pueden expresar mediante reacciones emoji. Ahora pueden responder a los comentarios sobre las tareas, y el centro de notificaciones lo mantendrá informado sobre las reacciones que reciba.



Para todos los que quieran contribuir a la traducción del producto a otros idiomas, el modo de traducción ahora está disponible directamente desde el contexto de la interfaz de YouTrack. Para los administradores, YouTrack 2020.1 introdujo una nueva configuración de campos de tareas para admitir casos complejos de acceso a campos individuales sin escribir flujos de trabajo. Para los desarrolladores, hemos mejorado la integración con los sistemas de control de versiones.

Más sobre innovaciones: más adelante en la publicación.

Espero que las nuevas funciones también lo deleiten, ya que nos regocijamos en cada stand-up, discutiendo las "Reacciones" de swimmein en el tablero.

Respuesta a comentarios para todos


Las reacciones son una manera fácil de responder rápidamente a un comentario en YouTrack. Los usuarios adoran los emojis porque pueden reemplazar una palabra o incluso una oración completa, con reacciones que no pueden escribir comentarios innecesarios cuando se discute una tarea.
Para no perder accidentalmente una reacción importante, el centro de notificaciones tiene una lista de reacciones recientes a sus comentarios, desde allí puede abrir inmediatamente la tarea deseada.



YouTrack sigue siendo un administrador de tareas, no un mensajero, por lo que decidimos comenzar con un pequeño conjunto de emojis. Desarrollaremos la biblioteca de acuerdo con las opiniones de los usuarios, así que asegúrese de compartir sus impresiones con nosotros.

Modo de traducción de interfaz del contexto para usuarios extranjeros


YouTrack está traducido al inglés, español, alemán, ruso, francés y japonés. Además, puede cambiar a uno de los idiomas traducidos por nuestros usuarios. Es húngaro, hebreo, italiano, chino simplificado, coreano, polaco, portugués y checo.



Intentamos simplificar el proceso de localización de la interfaz para aquellos que desean contribuir. En el modo de traducción del contexto, YouTrack resalta todos los elementos de la interfaz que aún no se han traducido, y puede ofrecer sus opciones directamente en la aplicación. Después de que JetBrains apruebe su traducción, será visible para todos los que usen YouTrack en este idioma.

El proceso de traducción es visible en tiempo real. Incluso puede traducir la parte dinámica de la aplicación. Una vez habilitada, esta función de traducción estará disponible para todos los usuarios de su servidor YouTrack. Si no desea distraer a sus compañeros de equipo con esta oportunidad, simplemente use el plan de tarifas gratuito y cree un servidor de YouTrack separado específicamente para la traducción.



Configuración delgada de acceso a campos en tareas para administradores


La nueva configuración para acceder a los campos en las tareas le permite seleccionar grupos de usuarios individuales o equipos que pueden ver o actualizar los valores de los campos.

Anteriormente, podía usar campos privados para restringir el acceso. Pero esta era una solución de todo o nada: si tenía acceso para ver y actualizar un campo privado en un proyecto, nada le impedía ver y actualizar los valores de cualquier otro campo privado en el mismo proyecto.

En YouTrack 2020.1, un administrador puede abrir el acceso para ver y actualizar cada campo individualmente. Además, ahora los administradores de proyectos pueden crear un grupo dedicado de usuarios que pueden ver ciertos campos, y otro: aquellos que pueden actualizarlos. En lugar de escribir flujos de trabajo complejos, simplemente use la nueva configuración de campo para casos inusuales de distribución de roles entre equipos.



Integración mejorada de VCS y TeamCity para desarrolladores


Integración mejorada de YouTrack con TeamCity, Upsource y sistemas de control de versiones como GitHub.

Configuración general de integración de VCS para múltiples proyectos


Gracias a la integración, puede adjuntar cambios de VCS a la tarea, en los comentarios a los que se menciona el identificador de esta tarea. Ahora puede configurar primero la integración para un proyecto y luego distribuir los parámetros seleccionados a otros proyectos, simplemente seleccionándolos en la configuración. Por lo tanto, otros proyectos también mostrarán los cambios asociados a las tareas. Para compartir la configuración, simplemente seleccione los proyectos que necesita.



Procesando tareas relacionadas


La integración entre YouTrack y TeamCity ahora le permite elegir con mayor precisión qué tareas corresponden a una compilación específica. Como parte de la integración con TeamCity, las tareas que se mencionan en los cambios de VCS incluidos en la compilación se consideran "relacionadas" con esta compilación. Cuando se procesa la compilación, las tareas relacionadas en el estado "Completado" se actualizan con el número de compilación correspondiente. Ahora puede especificar una condición que determinará qué tareas relacionadas deben actualizarse.



Swagger Support


Otra mejora relacionada con el desarrollo: el archivo incorporado Swagger Open API 3.0 apareció en YouTrack 2020.1. Con él, puede usar la API REST para integrar su aplicación con YouTrack. En esta página de GitHub, puede encontrar herramientas de generación de código compatibles para su tiempo de ejecución.

Permiso "Aplicar el comando sin notificaciones"


El nuevo permiso le permite aplicar cambios masivos a las tareas sin enviar notificaciones a quienes las están mirando. De forma predeterminada, el permiso está habilitado para administradores de proyectos y administradores de sistemas que pueden otorgarlo a otros usuarios.
