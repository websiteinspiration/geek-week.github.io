¿Qué es el HTML?


HTML significa lenguaje de marcado de hipertexto. Este es el lenguaje de marcado para documentos en la World Wide Web (WWW). HTML es un lenguaje estandarizado para texto formateado. El navegador lo interpreta y muestra elementos de páginas web en la pantalla.

¿En qué consiste el texto con formato HTML?


En primer lugar, HTML es contenido y etiquetas. Las etiquetas le permiten especificar cómo se muestra el contenido en las páginas web.

¿Qué es una etiqueta HTML?


Una etiqueta es una palabra de servicio especial entre corchetes angulares. También se llama el "elemento HTML". Hay muchas etiquetas en el lenguaje HTML y cada una hace algo con el contenido, que generalmente se encuentra dentro de los corchetes o entre las etiquetas.
Si la etiqueta está emparejada, entonces la etiqueta <TEG> corresponde a </ TEG>.

¿Están todas las etiquetas emparejadas?


No, no todo. Hay etiquetas individuales, por ejemplo, <img> - para imágenes. En este caso, el contenido junto con otras palabras de servicio (por ejemplo, para <img> puede ser src = "url de imagen") se coloca entre los corchetes, pero después de la palabra img.

¿Cuáles son los principales tipos de listas utilizadas para crear una página web?


Al crear una página web, puede usar:

  1. Lista de menú (lista de menú - <menú>).
  2. Lista de directorios (lista de directorios - <dir>).
  3. Lista ordenada (lista numerada - <ol>).
  4. Lista desordenada (lista con viñetas - <ul>).
  5. Lista de definiciones (<dl>).


¿Por qué usar listas en HTML?


Las listas hacen que la información sea más visual y fácil de leer. Además, dentro de las listas (más precisamente, para la visualización arbitraria de elementos de la lista), también puede usar varias etiquetas.

¿Podemos escribir comentarios en HTML? ¿Cómo?


Si podemos. Para hacer esto, el texto del comentario debe colocarse dentro de <! - y ->.

¿Por qué algunos caracteres en una página web a veces se muestran incorrectamente?


Muy a menudo, los caracteres se muestran incorrectamente debido a problemas de codificación. Esa es la razón: en la configuración del navegador o servidor web.

¿Qué es el mapa de imagen?


En HTML, Image Map es una tecnología que le permite vincular enlaces a fragmentos de imágenes. Al hacer clic en partes individuales de la imagen, el usuario puede seguir este o aquel enlace a diferentes páginas web.

¿Qué es el espacio en blanco?


El espacio en blanco es una propiedad que le permite cambiar la cantidad de espacios entre las palabras al mostrar una página web. Por defecto, el navegador interpreta cualquier número de espacios en el código HTML como un espacio.

¿Cuál es la principal ventaja del espacio en blanco?


Al escribir código HTML, el desarrollador no necesita preocuparse por los espacios adicionales que podría ingresar: el navegador eliminará todos los espacios adicionales cuando se muestre la página web.
El desarrollador puede agregar espacios al código, no solo por accidente, sino también a propósito: para que el código sea más legible.

¿Puedo asignar valores a no todos los atributos de etiqueta?


Sí, en este caso el navegador usará los valores predeterminados. Si no se siente cómodo con la forma en que mostró la página web, entonces quizás debería especificar explícitamente los valores de algunos atributos.

¿Cómo colocar la marca de copyright en una página web porque no está en el teclado?


El símbolo de copyright se refiere a caracteres especiales. Para colocarlo en una página web, debe escribir la secuencia y copiar o & # 169 en el teclado en el fragmento correspondiente del código HTML.

¿Cómo crear enlaces a diferentes fragmentos (secciones) de la misma página web?


Para hacer esto, encuentre el comienzo del fragmento que necesitamos en el código HTML. En este lugar, cree el llamado "ancla" con el atributo de nombre de la etiqueta <a> y asigne el enlace al nombre del fragmento con el atributo href y el signo #:
href = "#link_name_to_fragment"

¿Podemos alinear elementos de la lista en un archivo HTML?


Sí, podemos almacenar elementos de la lista directamente en el archivo HTML usando sangrías. La sangría también se puede usar para cualquier lista anidada dentro del padre.

¿Qué dirección de página web generalmente se considera la dirección del sitio?


Esta suele ser una de las páginas de nivel superior, la página principal. Desde allí, el usuario accede a páginas ubicadas en niveles inferiores. Y su dirección se forma con mayor frecuencia sobre la base de la dirección de la página principal.

¿Por qué utilizamos texto alternativo (atributo alt de la etiqueta <img>) para las imágenes?


Se muestra texto alternativo cuando los gráficos están deshabilitados en el navegador del usuario o si hay un problema al cargar una imagen específica. Este texto deja en claro lo que debería haberse representado en esta imagen.

¿Pueden los archivos HTML funcionar bien en algún navegador raro o de vanguardia?


Por supuesto, los archivos HTML pueden y deben funcionar bien en cualquier navegador, porque el navegador debe cumplir con los estándares HTML. Si los desarrolladores decidieron no admitir algunas de las características del estándar, esto queda en su conciencia.

¿Crees que el hipervínculo solo puede ser texto?


No, un hipervínculo puede ser no solo texto, sino también, por ejemplo, gráfico. Esto significa que podemos convertir la imagen en un enlace que dirige al usuario a otra página. Un hipervínculo no depende del tipo de contenido que se encuentra "debajo de él".

¿Cuáles son los atributos de la etiqueta <li>, un elemento de la lista?


Hay dos de ellos:

  1. tipo establece la apariencia de una viñeta o un marcador de lista de viñetas;
  2. valor: el número desde el que comenzará la lista numerada.


¿Para qué sirven las hojas de estilo (CSS)?


Las hojas de estilo le permiten establecer reglas claras que determinan la apariencia del contenido, varios elementos HTML y toda la página en su conjunto. Además, las mismas reglas se pueden aplicar a varias páginas web a la vez, lo que simplifica la preservación y, si es necesario, cambia el estilo uniforme del sitio.

¿Qué tipos de numeración en las listas conoces?


Los siguientes valores pueden servir como elementos de numeración:

  • Números arábigos (1, 2, 3, ...);
  • letras mayúsculas latinas (A, B, C, ...);
  • letras latinas minúsculas (a, b, c, ...);
  • números romanos en mayúscula (I, II, III, ...);
  • números romanos en minúscula (i, ii, iii, ...).


¿Cómo establecer diferentes colores para fragmentos de texto de una página web?


Para establecer diferentes colores para fragmentos de texto, debe colocar cada uno de los fragmentos dentro de la etiqueta de fuente emparejada:

<font color=“color”>-    “color”</font>


¿Dónde se almacenan los códigos de caracteres numéricos en HTML?


En la tabla ASCII. Cuando se muestra una página web, el navegador toma los códigos de caracteres numéricos del documento HTML y los reemplaza con el carácter correspondiente de la tabla.

¿Cuáles son los beneficios de agrupar múltiples casillas de verificación (elementos de casillas de verificación)?


Existen varias ventajas para agrupar banderas:

  • ayuda a organizarlos y estructurarlos mejor;
  • esto le permite acceder a banderas específicas del grupo usando id;
  • Esto le permite crear otro grupo de banderas en la misma página web sin temor a confundir los nombres o identificadores de las banderas.


¿Cómo afecta la superposición de etiquetas a la visualización del contenido?


Las etiquetas superpuestas en HTML solo reconocen la primera etiqueta. Pero tales problemas solo ocurren si el navegador intenta reconocer etiquetas sin texto.

Si no hay texto entre las etiquetas, ¿cuál será el resultado? ¿Dar un ejemplo?


Si no hay texto entre las etiquetas, el contenido será imposible de formatear. Por ejemplo, las etiquetas individuales, como <img>, no requieren ingreso de texto y, por lo tanto, no se requiere formato en este caso.

¿Cómo especificar colores para los bordes de la mesa?


Esto se puede hacer en la hoja de estilo, y si falta, se usará el color del texto.

¿Podemos crear un enlace que conduzca a otra página web?


Sí, debe usar el atributo href para esto.

¿Pueden las hojas de estilo ayudar a alinear la imagen y determinar cómo se coloca el texto en relación con ella?


Sí, una hoja de estilo puede contener reglas para posicionar texto e imágenes.

¿Puede un solo hipervínculo conducir a páginas diferentes?


No, un hipervínculo solo puede conducir a una página web, la dirección que usted proporciona.

¿Hay alguna diferencia entre una lista con viñetas y una lista de directorios y menús?


Sí, la diferencia es que una lista desordenada tiene atributos para cambiar el estilo del marcador.

¿Cómo cambiar el color del marcador?


El color del marcador generalmente coincide con el color del primer carácter de la lista. Si cambia el color del primer carácter con la etiqueta <font>, el color del marcador también cambiará.
Si queremos establecer el color del marcador en la lista sin cambiar el color del texto, entonces debemos colocar la etiqueta <span> dentro de <li> y colocar el texto dentro de ella. Y luego el color del marcador estará determinado por el color de la propiedad de estilo del selector LI, y el color del texto por la propiedad de color del selector SPAN.

¿Es posible introducir algunas restricciones en el tamaño de los campos de texto en HTML?


El tamaño estándar es de 20-25 caracteres. Pero utilizando los atributos size o maxsize, puede establecer otras restricciones. Por ejemplo:

<input type="text" size="5" />


¿Cuál es el tamaño máximo de un campo de texto?


El ancho de la ventana del navegador.

¿Qué sucede si establece el tamaño del campo de texto en cero?


El navegador usará el valor predeterminado.

¿Cuáles son las similitudes entre los atributos de borde y reglas?


Si el atributo de borde se establece en un valor distinto de cero, el borde predeterminado de una celda con un grosor de un píxel se agregará automáticamente. Si el atributo de reglas se agrega a la etiqueta <table>, el atributo de borde no se incluirá.
Pero al mismo tiempo alrededor de cada celda (lo mismo que el borde), aparecerá un marco con un grosor de un píxel por defecto, formando así una cuadrícula.

¿Qué es la carpa? ¿Cómo podemos aplicar esto?


Marquee ayuda a personalizar el desplazamiento del texto en una página web. Para activar el desplazamiento, debe usar la etiqueta <marquee>.

¿Qué hace la etiqueta <br>?


La función de la etiqueta <br> es separar fragmentos de texto entre sí.

¿Hay alguna otra forma de dividir texto sin usar <br>?


Sí, hay otras formas de dividir texto. Puede usar la etiqueta <p> o la etiqueta <blockquote>.

¿Se puede mostrar el texto fuera de la ventana del navegador?


De manera predeterminada, el texto no se puede mostrar fuera del navegador, pero si el texto es parte de una celda de tabla con un ancho predeterminado, puede ir más allá de la ventana del navegador.

¿Cuál es la diferencia entre enlaces activos e inactivos?


El enlace activo tiene foco, es decir, el mouse está sobre él. En otros casos, el foco está en otro lugar.

¿La hoja de estilo tiene restricciones en el número de reglas?


No, las hojas de estilo no limitan el número de reglas para un selector. Sin embargo, cada nueva regla debe estar separada por punto y coma.

¿Las hojas de estilo tienen alguna jerarquía de reglas?


Sí, hay una jerarquía que incluye cuatro niveles (desde la prioridad más alta hasta la menos prioridad):

  1. Estilo en línea: se adjunta directamente al elemento para el que se crea el estilo. Ejemplo:
    <h1 style="color: #ffffff;">
  2. Un identificador es un identificador único para elementos de página, como #navbar.
  3. Clases, atributos y pseudo-clases: estas categorías incluyen .class [attribute] y pseudo-clases como: hover ,: focus, etc.
  4. Elementos y pseudo-elementos: esta categoría incluye elementos y pseudo-elementos, como h1, div ,: before y: after.


¿Podemos agrupar diferentes selectores con diferentes nombres de clase?


Sí, para esto solo necesita escribir los nombres de las clases con una coma.

¿Puedo adjuntar un archivo CSS externo en un navegador?


No, se puede incluir un archivo CSS externo utilizando la etiqueta <link /> en el documento HTML.

¿Puede una propiedad de tipo lista de estilo afectar a un párrafo?


No, la propiedad de tipo de estilo de lista no puede afectar a un párrafo, lo más probable es que lo ignore.

¿Qué es un elemento de lienzo?


El elemento de lienzo HTML5 se puede usar para insertar imágenes, degradados y animaciones complejas. También puede ayudar a crear imágenes 2D directamente dentro del código fuente de un documento HTML.

Conclusión


Las respuestas a las preguntas anteriores pueden darle una idea de lo que le pueden preguntar. Puedes superar el miedo y ganar confianza, lo que te ayudará a conseguir el trabajo que deseas. Las respuestas descritas anteriormente son en su mayoría bastante claras y cortas. Pero esto es suficiente para que el entrevistador entienda que el candidato está en el tema.

Conclusión


Además de preparar las respuestas descritas anteriormente, también recomendamos conversar con un amigo o familiar que haya sido entrevistado recientemente. Esto lo ayudará a obtener información sobre las tendencias de la industria y las expectativas de los empleadores. Por lo tanto, puede mejorar sus habilidades y capacidades en consecuencia, y luego usar esto para ofrecer sus servicios al empleador y presentarse adecuadamente de acuerdo con sus expectativas.

La comunicación con las personas que han sido entrevistadas recientemente, así como las respuestas a las preguntas descritas anteriormente, lo ayudarán a posicionarse como un candidato más digno y aumentar sus posibilidades de una oferta.
