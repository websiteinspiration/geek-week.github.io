PyCaretの紹介：オープンなローコードPython機械学習ライブラリ

みなさん、こんにちは。Pythonニューラルネットワークコースの開始に先立ち、別の興味深い資料の翻訳を用意しました。



低コード環境で教師の有無にかかわらずモデルを学習およびデプロイするためのオープンソースのPython機械学習ライブラリであるPyCaretをご 紹介します。 PyCaretを使用すると、選択したノートブック環境で数秒でデータの準備からモデルの展開に移行できます。

他のオープンな機械学習ライブラリと比較すると、PyCaretは数百行のコードをほんの数語で置き換えることができる低コードの代替手段です。より効率的な実験の速度は指数関数的に増加します。 PyCaretは基本的に、scikit-learn、XGBoost、Microsoft LightGBM、spaCyなどのいくつかの機械学習ライブラリ上のPythonシェルです。その他多数。

PyCaretはシンプルで使いやすいです。PyCaretによって実行されるすべての操作は、展開の準備が完全に整ったパイプラインに順次格納されます。欠損値の追加、カテゴリカルデータの変換、エンジニアリング機能、ハイパーパラメータの最適化など、PyCaretはこれらすべてを自動化できます。PyCaretの詳細については、こちらの短い動画をご覧ください。

PyCaretを使ってみる


PyCaretバージョン1.0.0の最初の安定版リリースは、pipを使用してインストールできます。コマンドラインインターフェースまたはノートブック環境を使用して、以下のコマンドを実行し、PyCaretをインストールします。

pip install pycaret

Azure NotebooksまたはGoogle Colab を使用している場合は、次のコマンドを実行します。

!pip install pycaret

3.モデルの比較

これは、教師トレーニング（分類または回帰）で作業するときに実行することが推奨される最初のステップです。この関数は、モデルライブラリ内のすべてのモデルをトレーニングし、Kブロック（既定では10ブロック）の交差検証を使用して推定指標を互いに比較します。推定指標は次のように使用されます。

  • 分類：精度、AUC、リコール、精度、F1、カッパ
  • 回帰の場合：MAE、MSE、RMSE、R2、RMSLE、MAPE



デフォルトでは、メトリックは10ブロックを超える相互検証を使用して評価されます。パラメータの値を変更することにより、ブロック数を変更できますfold。

デフォルトのテーブルは、最高値から最低値へと「精度」でソートされます。オプションを使用して、ソート順を変更することもできますsort。
