CorelとParallelsが米国からKKR投資グループに売却

2019年7月3日、世界有数の投資会社の1つであるKKRは、Corel Corporationの買収プロセスが完了したことを発表しました。それとともに、Corelが昨年購入したすべてのParallelsソフトウェア製品と資産が購入者に渡されました。

KKRがCorelの買収を計画しているという事実は、2019年5月に知られるようになりました。取引総額は非公開。


トランザクションが終了すると、KRRはCorelが以前に取得したすべての資産を自由に利用できるようになります。 KKRソフトウェアポートフォリオには、Parallels Desktop for Mac、Parallels Toolbox for Windows and Mac、Parallels Access、Parallels Mac Management for Microsoft SCCM、Parallels Remote Application Server（RAS）など、Parallels製品の全ラインが含まれています。
取引の財務面は明らかにされていない。

Parallelsは1999年に設立され、ワシントン州ベルビューに本社を置いています。 Parallelsは、クロスプラットフォームソリューションのグローバルリーダーです。

1980年代にカナダのオタワで設立されたCorelは、成長し続けるいくつかの大規模市場の交差点というユニークな位置にあり、主要な業種で合計250億ドル近くあり、さらに多くの機会を提供するソフトウェアソリューションの幅広いポートフォリオを提供しています。世界中で9000万人以上のナレッジワーカー。

Corelには買収と買収の長い歴史があり、その最新のものにはParallels、ClearSlide、MindManager製品の購入が含まれます。 Corelアセットリストには、独自のデザインのソフトウェア製品が少なくとも15含まれています。それらのほとんどは、何らかの形でグラフィックに関連しています。これらには、CorelDraw、ベクターグラフィックエディター、Corel Painter、デジタルペイントおよび描画プログラム、Corel Photo-Paint、ラスターグラフィックエディター、さらには独自のLinuxディストリビューション、Corel Linux OSも含まれます。 Corelが直接開発した製品に加えて、同社は、異なる年に買収したサードパーティのソフトウェア開発者も所有しています。これには、WordPerfectテキストエディター、WinDVDメディアプレーヤー、WinZipアーカイバー、Pinnacle Studioビデオ編集ソフトウェアが含まれます。 Corelが所有するサードパーティプログラムの数は15を超えています。

Corelは、印象的なITソリューションのポートフォリオを常に拡大することにより、独自の市場での地位を獲得しています。 KKRは、コーレルのリーダーシップと協力してさらなるビジネスの成長を刺激する一方で、M＆Aチームの豊富な経験を活用して、革新と成長の新しい章をグローバルに開始することを楽しみにしています」とKKR理事会のジョンパークは述べています。

KKRはまず第一に、特にクライアントとの取り組み、技術革新、および成功裏に実施された他社の買収および買収戦略に関して、従業員の価値とその印象的な業績を認めます。 KKRのサポートと共通のビジョンのおかげで、エキサイティングな新しい機会が私たちの会社、製品、ユーザーに開かれています。CorelのCEO、Patrick Nichols。

「Corelは何年にもわたってベクターキャピタルファミリーの重要な一部であり、KKRの売却により投資家にとって素晴らしい結果を達成できたことを嬉しく思います」と、ベクターキャピタルの創設者兼最高投資責任者であるアレックススルスキはコメントしました。この間、Corelはいくつかの革新的な買収を完了し、収益を増やし、収益性を大幅に改善しました。 CorelがKKRに直面して価値のあるパートナーを見つけたと確信しており、彼らが一緒に成功し続けることを願っています。」
