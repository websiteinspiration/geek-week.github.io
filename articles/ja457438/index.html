そして、再生可能な.m3uファイルを取得します。

だが！問題はそれです：

  • このファイルは、オーディオギャップへのリンクが有効な間のみ再生できます。
  • リンクが有効である限り、ファイルを変換することもできます
  • ファイルを変換できるのは、それが受信されたコンピュータ上だけです。リンクはPCのIPアドレスに関連付けられています。

import requests,re,os
def convert_mp3(f):
    z=open(f).read()
    con=list(map(lambda e: e.rstrip('\n').rstrip('#EXT-X-ENDLIST').rstrip("\n") if '#EXTM3U' not in e else '' ,re.split('#EXTINF:\d+.\d+,\n',z)))
    z = b''
    for r in con:
        if(r==''):continue;
        z+=requests.get(r).content
    open(f.strip(".m3u8")+".ts",'bw').write(z)
z=set()
for file in os.listdir():
    if file.endswith(".m3u8"):
        z.add(file)
        convert_mp3(file)
z=',\n'.join(z)
