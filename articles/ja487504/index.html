ロシア科学の日までに、2019年の国内科学者の最高の業績が挙げられます

Mail.ru GroupとNUST“ MISiS”は、ロシアの科学者が2019年に行った最も重要な発見の評価をまとめました。ロシア科学の日に、Search Mail.ru、Hi-Tech Mail.ru、およびNUST“ MISiS”は、ロシアの科学のどの成果がインターネットユーザーの間で最大の関心を呼び起こしたかを発見しました。

Mail.ru GroupとNUST MISiSの調査によると、インターネットユーザーの86％が科学ニュースに興味を持っています。ほぼ80％がロシアの科学者の業績に従っていると述べています。それにもかかわらず、ロシア科学の日などの休日の存在を知っているのはわずか29％です。

Search Mail.ruは、過去1年間のロシアの科学者の業績に対する要求の数を分析しました。含まれる検索クエリの数の点で上位5つ：

1）宇宙の地図を構築するための天文台、Spectra-RGの起動と作業の開始;


写真：CC Yuzhny / TsENKI

2）量子コンピューターの最初のプロトタイプの作成。


写真：Sergey Gnuskov / NUST“ MISiS”

3）天文学者Gennady Borisovによる最初の星間彗星の発見。


写真：左-写真の中央にある彗星C / 2019 Q4（ボリソフ）。彼女の小さな尾であるジェンナディボリソフ

4）人工肉の生産の始まりが見えます。


写真：Vyacheslav Prokofiev / TASS

5）スモレンスク近くのグデン将軍、ナポレオンの主な同盟者の遺跡の発見。


写真：Denis Maximov / AFP / Scanpix / LETA

Hi-tech Mail.ruはユーザーの間で調査を行い、ロシア人が人類にとって最も重要で有用であると考えるこれらの出来事を発見しました。

2019年のロシア科学の主要な成果であるユーザーは、最初の量子コンピューターのプロトタイプの作成を求めました。回答者の45％が彼に投票しました。 2位は、Spectrum RG観測所の立ち上げ（29％）です。上位3組のファイナリストには人工肉の生産も含まれ、調査参加者の14％がそれに賛成票を投じました。星間彗星の発見を重要であると認識した回答者はわずか9％で、ユニークな歴史的発見-ギュデン将軍の遺跡のダンスフロアの下にあるスモレンスクでの発見に投票したのはわずか3％でした。

量子コンピュータに取り組むユーザーの注意は、科学界に曖昧な反応を引き起こしました：

「科学に関係のない人々が量子コンピューターにそれほど大きな関心を示している理由を言うのは難しい。さらに、現代のスーパーコンピューターでは実行できない特別な問題を解決するように設計されているため、これまで使用していたデバイスに取って代わることはほとんどありません。新しい材料と化合物の合成の計算、量子化学と情報セキュリティ、複雑なシステムのスケジュールと操作の最適化鉄道や航空輸送、エネルギーネットワーク、パイプラインを介した原材料の輸送、金融セクターなど」とNITU MISiSの超伝導メタマテリアル研究所の責任者であるAlexei Ustinovはコメントしています。


アレクセイ・ウスティノフ

それとは対照的に、RSF専門委員会のアレクサンダークリメンコ議長である科学コミュニティの別の代表は、現代社会における量子テクノロジーへの関心を非常に論理的に考えています。アマチュアでさえ、これが人に与えたと想像して、そして新しい革命の期待は彼女への興味を育みます。さらに、最近、「ビッグ5」ITの巨人からの会社が「最も強力な量子コンピューター」を作成または作成する計画を立てているというニュースが定期的に発行されています。もちろん、これは一般の注目を集めています。」


アレクサンドル・クリメンコ

また、クリメンコによれば、宇宙圏での発見に対するユーザーの関心の高まりは非常に正当化されています。宇宙に近い「プロット-"最初の時間"、 "スターウォーズ"、および他の惑星での生命の存在の可能性に対する大きな関心。」

科学界はまた、これらのトピックに加えて、気候問題や技術研究も2019年に社会に大きな影響を与える可能性があると考えています。「リストに含まれるべきトピックの中で、地球規模の気候変動とそれに伴う自然災害の増加、新しい兵器の作成、人工知能、および人間との関係」とクリメンコは述べています。
