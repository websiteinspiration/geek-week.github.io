Installation von OpenCV-Python in einer virtuellen Umgebung für Superkessel

Hier finden Sie ein schrittweises Beispiel für die Installation der OpenCV-Bibliothek in Python.


  • Python-Installation
  • Einrichtung der virtuellen Umgebung
  • Installieren Sie OpenCV + jupiterlab, numpy, matplotlib
  • Testen

Ich habe alles auf einem Microsoft Surface-Tablet, Windows 10 Pro, mit einem 64-Bit-Betriebssystem getestet.


Angenommen, auf Ihrem Gerät ist im Voraus nichts installiert.


  1. Installieren Sie zuerst Python .
    Laden Sie die gewünschte Version herunter und führen Sie die EXE-Datei aus. Vergessen Sie nicht, das Kontrollkästchen Pfad hinzufügen zu aktivieren. Ich habe Python 3.7.3 vom 25. März 2019 installiert, da die neue Version von Python 3.7.4 vom 8. Juli 2019 nicht richtig funktionierte, dh einige Befehle hingen im Terminal. Öffnen Sie die Befehlszeile.
  2. Installieren Sie virtualenv.
    Wir brauchen eine virtuelle Umgebung, damit jedes einzelne Projekt seinen eigenen „Raum“ mit eigenen Versionen der installierten Bibliotheken hat, die nicht von anderen Projekten abhängen und miteinander verwechselt werden.
    Wir werden Pakete mit pip installieren . Es wurde kürzlich sofort mit Python ausgeführt, muss jedoch normalerweise mit dem folgenden Befehl
    python -m pip install --upgrade pip
    aktualisiert werden : Pip aktualisiert, jetzt die virtuelle Umgebung installieren : Gehen Sie mit dem
    pip install virtualenv
    Befehl cdzu dem Ordner, in dem Sie die Umgebung erstellen möchten, und geben Sie den folgenden Befehl ein:
    mkdir opencvtutorial_env- So haben wir eine Umgebung namens opencvtutorial_env erstellt.
    Geben Sie als Nächstes den Befehl ein virtualenv opencvtutorial_envund gehen Sie zum Aktivieren zum Umgebungsordner und aktivieren Sie ihn mit Tab.
    .\opencvtutorial_env\Scripts\activate
  3. OpenCV-Python, Numpy Matplotlib, opencv.
    . :
    pip install opencv-python
