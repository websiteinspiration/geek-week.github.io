HYPHAE（GIFA）[GITHUB]



我开始尝试一个系统，在该系统中，我生长出彼此无法重叠的圆圈。它的工作原理是这样的：将初始（圆）放置在某处，并为其设置半径和运动方向。然后尝试在移动方向上沿第一个节点的周长添加一个新节点。重要的是要有一个运动的方向，以便有一点“摆动”，并且每次角度都会略微偏离。另外，每次添加一个新节点时，请确保新节点的半径略小于前一个节点的半径。







要获得分支，可以随机选择节点，并尝试或多或少地垂直于分支运动的方向生长一个新节点。要么节点冲突，要么获得新的分支。为了获得更有趣的结果，可以使偏离角度与分支的宽度（半径）成比例。较粗的分支往往比细的分支更直。您显然可以做更多的事情。例如，如果调整随着分支变细而增长的“灵活”分支的数量，您将获得截然不同的结果。







还有一个小注释：当一个分支从一个现有分支中分离出来时，有必要使其半径比以前的分支要细得多。您可以在现实生活中的树木中看到它。底线是在穿过分支之前和之后的质量应该大致相同。（我在某处阅读过此书，但找不到。）



树木[GITHUB]



当我们谈论分支时，我意识到我们可以提到一个关于树木的小实验，因为它们本质上是相似的。







我想尝试以与Gif中相同的方式来增长分支，但是只是为了使新分支仅出现在其他分支的“末端”。另外，在此模拟中，我不想关注碰撞或类似情况。结果不是真正的树，但是如果您不努力观察的话，结果看起来很现实。



可能最有趣的是，与这些“树”不同，没有从固定宽度的树干上生长出一棵真正的树。随着时间的推移，树木的高度和宽度都会增加，这是一个非常动态的过程，其中包含许多因素。事实证明，该算法与真实树没有什么共同点，但是该算法的一些简单规则仍然允许您通过模拟真实树来获得不错的结果。







为了给树木提供深度效果，我在树枝生长时为其添加了阴影。这种错觉进一步发展，在一侧增加了越来越多的阴影。



重复行[ GITHUB ]



与生物学启发我做的Gifa和树木算法不同，以下示例基于John Frensen的画作。总的来说，Frensen绘制线，其中每条线都遵循先前绘制的线的轮廓。随后的每一行都会稍有变化，并且有趣的模式将很快出现。



在计算机上绘制形状非常容易。一条线由一系列顶点（点）组成，在它们之间有一条边（段）。要创建三角形，您需要获取三个边并将它们与三个顶点连接。要绘制一个圆，您需要做同样的事情，只需要很多顶点，这样就可以“看到”这实际上不是一个圆，这是“不可能的”。如果您对此方法有疑问，我会立即说，有更复杂，更准确的方式来绘制表格，但是这种方法非常简单且准确。



这意味着您可以通过绘制具有多个顶点的圆形形状，以与Frensen相同的方式绘制这些线。跟踪表单有些棘手，但是您可以尝试几种方法。我希望这种行为能够模仿上一幅图的实际绘制过程，同时继续前几行。我提出的算法无法正常运行。也就是说，他没有避免碰撞，但是结果仍然看起来非常好。







要跟踪轮廓，可以选择上一行中的一些相邻边，计算这些边的平均方向，然后沿该方向向当前行添加一个顶点。然后添加一些随机运动来模拟徒手画。一段时间以来，这种方法效果很好，但是结果中会出现一些“惯性”，形式适应得太慢。

添加到每个顶点的噪声数量至关重要。这种噪声是使整个系统创建有趣的形式的原因，因为对形式的校正会在过程中强制尝试重复总体运动和某些随机变形。

差分是我一直想做的算法。在某个时候，解决方案刚刚浮现在脑海，我设法使算法起作用。我想创建一个系统，其行为方式与自然界中彼此展开的表面相同。例如，像核桃或白菜层（我什至不告诉我花多少时间用谷歌搜索红白菜的图像，将其切成两半），肠子以及一些叶子和花瓣。可能还有许多其他事情。



在我开始寻找方法之前，我只看到了神经系统创建的Floraform，但是您还应该看到Andy Lomas的Cellular Forms。我后来发现的另一个重要文件是在这里。



最初，我尝试使用基于Koch分形曲线的方法。但是具有随机增长，并且不像分形那样对称且规则。我从未完成，并很快放弃了。但是，经过一段时间（前后约六个月），我意识到我可以在Koch曲线“增长”的同时引入新的节点，但是我需要使这些节点彼此相对移动。因此，他们可以像绳索上的珍珠一样动态地调整位置。因此，它们可以与其他节点（珍珠）保持正确的距离。



算法如下。首先，我们创建一组以某种形式（圆形或直线形）连接的节点。然后，我们在成对的现有节点之间随机引入新节点。在每次迭代中，节点将尝试优化其位置。他们会“想”靠近，但不要太靠近两个邻居。同时，他们将希望与所有其他节点尽可能远。请注意，它们不可能发生碰撞。







有许多因素可以影响此系统的行为。其中之一是我们引入新节点的频率。另一个是节点之间彼此避开的最大距离。最有趣的是（我认为），我们如何选择在何处插入新节点。我们可以像上面的GIF中那样均匀地执行此操作，也可以像下面的GIF中那样优先确定曲线较锐利的位置。
