Hari Pelatihan Virtual Microsoft Azure: Gudang Data Modern

Webinar, 29 April 2020, 10.00-13.00

Kami mengundang Anda untuk berkenalan dengan gudang data modern. Selama beberapa sesi, Anda akan menjadi terbiasa dengan komponen-komponen Microsoft Azure untuk membangun solusi analitik ujung ke ujung. Sesi akan fokus pada proses: dari mengumpulkan dan mengubah data dari berbagai sumber menggunakan Azure Data Factory, menyimpan data berdasarkan Azure Synapse, dan menyajikannya kepada pengguna berdasarkan Power BI.
