ラテン方格  -各行と各列で2回繰り返されることがないように文字で満たされたプレート。

A B C
B C A
C A B


このテーブルは、乗算テーブルとも呼ばれます。したがって、順序付きセットでの乗算の演算を定義しました {A,B,C}。例えば、A×B=B または B×C=A。そのようなテーブルを使用して指定された乗算は通常、非可換です（A×B≠B×A）続いて、要素を定義しますA×B 表に従って、用語と要素のシリアル番号の共通部分を調べます A 要素のシリアル番号を含む列 B。

ラテン語の正方形は、操作が次の場合にグループと呼ばれます× 連想、すなわち A×(B×C)=(A×B)×C要素の可能なすべてのトリプルについて。上の表はグループ1です（信じてください、信じてください）。そして、これがラテン方格です

A B C
C A B
B C A


連想性の性質を持たない（非連想ラテン方格はquasigroupと呼ばれる）本当に、C×(B×A)=A、および (C×B)×A=B。次は、男性が疑似グループをどのように探していたかという話です。

そして、ほとんど練習していません。
