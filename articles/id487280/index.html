8 aplikasi web penting untuk pengembang

Kami mengundang Anda untuk membiasakan diri dengan terjemahan artikel Jamie Bullock yang diterbitkan di medium.com. Penulis berbicara tentang aplikasi web mana yang biasanya ia gunakan dalam karyanya.


Foto oleh: Héctor Martínez . Diambil dari Unsplash

1. GDB online




GDB Online adalah kompiler peramban, juru bahasa, dan debugger. Ini mendukung 20 bahasa pemrograman, termasuk C, C ++, Python , Ruby , C #, Swift, dan JavaScript.

Saya menemukan aplikasi web ini sangat berharga untuk memeriksa cuplikan tanpa harus menjalankan sesi IDE atau terminal.

Selain itu, sangat cocok untuk meningkatkan keterampilan pemrograman atau belajar bahasa baru.

2. Validator Skema JSON




Ada banyak alat validasi JSON online , tetapi saya pikir validator Skema JSON adalah yang terbaik. Itu dapat memvalidasi JSON, skema JSON, dan juga memvalidasi JSON terhadap skema yang ditetapkan pengguna.

Ini juga berisi sejumlah besar skema yang telah ditetapkan, dibandingkan dengan yang validasi JSON dapat dilakukan. Saya tidak menggunakan validator Skema JSON setiap hari, tetapi jika saya mengalami kesulitan dalam mengurai file JSON, itu sangat diperlukan untuk memperbaiki masalah tersebut.

3. Regex 101




Regex 101 adalah alat yang hebat untuk belajar ekspresi reguler . Ini memungkinkan Anda untuk memasukkan teks apa pun dan menemukan kecocokan di dalamnya berdasarkan pada ekspresi reguler yang ditentukan pengguna.

Aplikasi web ini tidak hanya secara instan menemukan kecocokan, tetapi juga secara dinamis menampilkan hasil dari tindakan ekspresi reguler, termasuk mendeteksi kesalahan jika Anda mengetik sesuatu yang salah.

4. Font Keren




Font Awesome adalah direktori online yang berisi lebih dari 1.500 ikon gratis, termasuk ikon aplikasi standar, ikon perusahaan, dan simbol berguna lainnya.

Anda dapat mengunduh ikon sebagai satu set font OpenType atau dalam format SVG. Dalam aplikasi web, mereka dapat digunakan langsung melalui JavaScript menggunakan tag ikon.

Menurut pendapat saya, Font Awesome sangat berguna saat membuat prototipe: dengan itu, Anda dapat dengan cepat memasukkan ikon ke dalam aplikasi yang dikembangkan bahkan sebelum persetujuan desain resmi.

5. Pertukaran Stack




Jika saya harus memilih hanya satu aplikasi web penting, saya akan memilih Stack Exchange .

Stack Exchange adalah jaringan situs dengan pertanyaan dan jawaban tentang berbagai topik. Sumber daya ini digunakan oleh lebih dari 11 juta orang, yang menunjukkan soliditasnya. Di situs pemrograman Stack Overflow, pertanyaan sering dijawab dalam beberapa menit setelah diposting.

Saya belajar banyak di situs web Stack Overflow , ini membantu saya memecahkan banyak masalah praktis.

Jika Anda seorang programmer pemula atau tingkat pengetahuan Anda di bidang ini rata-rata, cobalah untuk mengajukan pertanyaan di sana saat lain kali Anda menghadapi kesulitan. Jika Anda seorang spesialis yang cukup maju, Anda bisaberikan jawaban atas beberapa pertanyaan sendiri .

6. GitHub




Sepertinya saya bahwa daftar ini tidak akan lengkap tanpa GitHub ! GitHub adalah layanan web untuk mengakses dan mengelola repositori Git . Ini juga termasuk fitur seperti pelacakan bug, wiki, rilis proyek dan papan kanban proyek.

Halaman GitHub menyediakan hosting untuk situs web atau blog dasar. Bagi saya, GitHub adalah sumber daya universal yang berisi semua yang Anda butuhkan untuk mengelola proyek perangkat lunak, mendokumentasikannya, dan melepaskannya.

7. Tabel Unicode




Unicode Table adalah basis data karakter Unicode yang sangat besar . Ini termasuk catatan, panah, emoji, tanda mata uang, simbol matematika, karakter Cina dan semua yang dapat Anda bayangkan.

Untuk setiap karakter, bersama dengan kode Unicode, HTML dan CSS, versi insert disediakan, serta data enkode UTF lengkap . Harap perhatikan bahwa font yang kompatibel dengan Unicode diperlukan untuk menampilkan karakter ini dalam aplikasi.

8. Musik untuk pemrograman




Akhirnya, temui salah satu aplikasi web favorit saya - musicForProgramming ! Ini berisi lebih dari 50 daftar putar dengan musik yang dirancang khusus untuk meningkatkan konsentrasi.

Ada argumen untuk fakta bahwa musik membantu untuk fokus, dan saya percaya bahwa beberapa gaya musik lebih baik untuk ini daripada yang lain. Saya tidak menggunakan situs ini setiap hari, tetapi bagi saya sepertinya luar biasa! Terkadang sumber daya ini persis seperti yang saya butuhkan untuk meningkatkan konsentrasi.
