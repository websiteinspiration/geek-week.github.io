Raspberry Piを有益に使用する5つの方法 パート3

こんにちは、ハブル。第二の部分、ラズベリーパイを使用する5つの方法を検討しました。そして、結局のところ、このマイクロコンピュータはさらに多くのことが可能です。次に、別の5つのRaspberry Piの興味深い機能を検討します。 前のパートと同様に、プログラミングを必要としないメソッドを検討します。 カットの下に続きました。








まず、前の部分で尋ねた質問に答えます。RaspberryPiのどのモデルを選択するのが良いですか。実際、バージョンによってパフォーマンスが異なり、テスト結果はGoogleで簡単に見つけることができます。



ご覧のとおり、最新モデルの中で最も高速なのはRaspberry Pi 3 B +とRaspberry Pi 4の最新バージョンです。Pi4の唯一の欠点は、Micro HDMIコネクタが珍しいことです。この点で3B +モデルの方が普遍的です。超コンパクトなモデルが必要な場合は、Raspberry Pi Zero Wに目を向けることができますが、一般的にはお勧めしません。 1つ目は、テストからわかるように、速度が著しく遅いことです。2つ目は、ゼロWには（もう一度！）別のビデオコネクタ（ミニHDMI）があり、イーサネットがなく、キーボードまたはマウスを接続するためにUSBアダプタが必要です。ゼロWは、なぜ超小型モデルが必要なのか正確にわかっている場合にのみ使用してください。通常バージョン（Pi 3B +またはPi 4）の方が高速で、より普遍的です。

ちなみに、超小型のマイクロPCが必要な場合は、次のような非常に興味深いモデルの形で提供されるNano Piに目を向けることができます



。したがって、Raspberry Piを使用する5つの方法があります。

1.情報キオスク



newscrewdriver.com/2019/08/10/raspberry-pi-web-kiosk-boots-faster-on-raspbian-than-ubuntu-coreの写真

最新バージョンのRaspbian OSには、複雑なWebでも正しく表示できる完全に最新のChromiumブラウザーが搭載されています-ページ。したがって、Jiraからのチケット、会社の株式の見積もり、列車のスケジュール、または少なくとも天気予報を大画面で表示する必要がある場合、Raspberry Piはこのタスクに完全に対応します。

これは非常にシンプルで、プログラミングは必要ありません。Chromiumをキオスクモードで実行し、マウスカーソルを非表示にして、スクリーンセーバーをオフにするだけです。

興味のある方のために、いくつかのチュートリアル：

  • pimylifeup.com/raspberry-pi-kiosk
  • blockdev.io/raspberry-pi-2-and-3-chromium-in-kiosk-mode

2. LoRaWANゲートウェイ




低電力のLoRa無線モデムを介してデータを受信するためにLoRaWANワイヤレスネットワークが必要な場合は、Raspberry Piをベースにすることができます。さまざまな価格カテゴリのオプションがあり、シングルチャネルゲートウェイの方が安く、8チャネルの方が高価です。問題のおおよその価格は、ゲートウェイの「トリック」に応じて、50ドルから500ドルです。

LoRaの詳細な調査は、このテキストの範囲を超えています。より詳細な調査の資料：

  • learn.adafruit.com/lora-and-lorawan-radio-for-raspberry-pi
  • www.cooking-hacks.com/documentation/tutorials/lorawan-for-arduino-raspberry-pi-waspmote-868-900-915-433-mhz
  • www.thethingsnetwork.org/docs/gateways/start/build.html

3.インターネットラジオとインターネットテレビ




Raspberry Piでのオーディオ再生のトピックは非常に広範囲です。3.5ドルの中国製USBサウンドカード、35ドルの金メッキコネクタ付きのHiFiシールド、および上の写真に示すようにスピーカー付きのボードをデバイスに接続できます。また、光出力を備えたボードを見つけるのも簡単です。これにより、古いオーディオシステムに最新の「ネットワーク」機能も追加されます。

ソフトウェアに関しては、多くのオプションがあります。インターネットラジオ局を1つだけハードコーディングして再生でき、最愛の祖母に「ラジオポイント」のようなものを作ったり、LCDディスプレイやボタン​​などを追加したりできます。定期購入するとSpotifyから再生できます。

興味のある方は、次のチュートリアルをご覧ください。

  • www.jan-holst.dk/pi-radio/pi-radio.html
  • www.hackster.io/Granpino/raspberry-pi-internet-radio-and-mp3-player-with-bluetooth-1aa591
  • learn.pimoroni.com/tutorial/sandyj/streaming-spotify-to-your-pi
  • pimylifeup.com/raspberry-pi-spotify

IP-TVについては、メディアセンターとしての使用に特化したソフトウェアであるKodiを使用することをお勧めします。私は個人的にIP-TVの設定を試したことがなく、有料サブスクリプションも持っていませんが、たとえば海外に住んでいてロシア語のテレビを視聴したい人には、これが関係するかもしれません。

興味のある方はチュートリアルを参照できます：

  • pimylifeup.com/raspberry-pi-kodi
  • www.raspberrypi.org/documentation/usage/kodi

IRレシーバー をRaspberry Pi に接続して、リモコンを使用することもできます。

4. FMトランスミッター



circuitdigest.com/microcontroller-projects/raspberry-pi-fm-transmitter Webサイトの写真

多くの人は知らないが、プログラムでPLL周波数を変更することにより、追加のハードウェアなしで500 MHzまでの範囲の無線信号を送信できます。ピンに接続するだけです。写真に示されているように、ワイヤーの一部、およびプログラムを実行します。もちろん、ここではHiFi品質はなく、出力電力は約10mWですが、無線デバイスをテストしたり、友人や隣人を演奏したりするだけの場合は、これが便利です。10mWの電力で、室内のFMラジオで音楽を再生できます。

より詳しく私は数年前にオタク時代の記事でこのテクノロジーについて説明しましたその記事はすでに古くなっていますが、より良い放送番組はここにあります。

5.ドローン用フライトコントローラー




Raspberry Piの次のプロジェクトも同様に興味深いものです。MultiWii、Arducopter、DJIなどのマルチコプターのコントローラーは、おそらく多くの人が知っています。Raspberry Piに基づいて同様のコントローラーを作成できます。NAVIO2プロジェクトは、モーターコントローラーを制御できる特別なシールドであり、必要なセンサー（加速度計、気圧計、GPSなど）を搭載しています。

そのような束が「通常の」Arducopter / Pixhawkよりどれほど優れているかを言うのは難しいですが、多くのタスクにとって、このアイデアは非常に有望です。まず、このようなヘリコプターから3G / 4Gモデムを介してテレメトリ（または画像）を制御または送信できるため、リンクの範囲が大幅に広がります。第二に、Raspberry Piは、カルマンフィルターやその他の数学などの非常に複雑なアルゴリズムを実行できます。私は個人的にはこのコントローラーを試したことがないので、詳しくは言えません。誰でもYoutubeでフライトのビデオを検索できます。
