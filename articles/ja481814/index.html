FunCode Java / Kotlinチャレンジバックエンドコンテスト

FunCorpは、550,000ルーブルの賞金プールで、Java / Kotlin言語のバックエンド開発者を対象としたコンテストを開催しています。

どなたでも参加できます。最高の作品の作者には、FunCorpから賞金やギフトが贈られるほか、チームに参加する機会が与えられます。



何をすべきか？


JavaやKotlinでアプリケーションを作成します。これにより、最も人気のあるエンターテインメントコンテンツ（写真と短いビデオ）が決定され、メタデータとともに、最新のものから古いものへと公開時刻順にソートされたテープとして返​​されます。

何を評価しますか？


  • 建築ソリューション
  • コード品質
  • パフォーマンス
  • リソース消費
  • オリジナルかつ大胆な技術ソリューション

受け付けられたタスクは、1月19日23:59まで受け付けられます。

結果と受賞者は2月17日までに発表されます。

私たちに関しては


FunCorpは、面白いUGCサービスとゲームを開発しています。

主な製品は、米国で2番目に人気のあるエンターテインメントアプリケーションであるiFunnyです（インストール数5,000万以上）。

何を使用していますか？


  • マイクロサービスのアーキテクチャ
  • Java / Kotlin
  • Dropwizard、Spring、Ratpack、Ktor
  • データストレージ用のMongoDBとRedis
  • デプロイ用のDocker
