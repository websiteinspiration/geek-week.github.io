アマス


Amassは、DNSサブドメインを検索および列挙し、外部ネットワークをマッピングするためのGoツールです。Amassは、インターネット上の組織が外部の観測者のように見えることを示すために設計されたOWASPプロジェクトです。Amassはさまざまな方法でサブドメイン名を受け取ります。このツールは、サブドメインの再帰的な列挙とオープンソース検索の両方を使用します。

相互接続されたネットワークセグメントと自律システム番号を検出するために、Amassは運用中に取得したIPアドレスを使用します。検出されたすべての情報は、ネットワークマップを構築するために使用されます。

Altdns


Altdnsは、DNSサブドメインを列挙するための辞書をコンパイルするためのPythonツールです。変異と順列を使用して、サブドメインの多くのバリアントを生成できます。このため、サブドメイン（たとえば、test、dev、stagingなど）でよく見られる単語が使用され、すべてのミューテーションと順列は、Altdnsに入力できる既知のサブドメインに適用されます。出力は存在する可能性のあるサブドメインのバリエーションのリストであり、このリストは後でDNSブルートフォースで使用できます。

長所：


  • 大きなデータセットでうまく機能します。

Massdns


MassDNSは、DNSサブドメインを見つけるための別のツールです。主な違いは、DNSクエリをさまざまなDNSリゾルバーに直接送信し、かなりの速度で実行することです。

長所：


  • 高速-1秒あたり35万を超える名前を解決できます。

マイナス：


  • MassDNSは、使用されているDNSリゾルバーに大きな負荷をかける可能性があり、これらのサーバーの禁止やプロバイダーへの苦情につながる可能性があります。また、会社のDNSサーバーがあり、解決しようとしているドメインを担当している場合、会社のDNSサーバーに大きな負荷がかかります。
  • 現在、リゾルバーのリストは古くなっていますが、壊れたDNSリゾルバーを選択して新しいリゾルバーを追加すると、すべて正常に動作します。



nsec3map


nsec3mapは、DNSSECによって保護されているドメインの完全なリストを取得するためのPythonツールです。

長所：


  • ゾーンでDNSSECサポートが有効になっている場合、最小限のクエリでDNSゾーンのホストをすばやく検出します。
  • John the Ripperのプラグインが含まれており、受信したNSEC3ハッシュをクラックするために使用できます。

マイナス：


  • 多くのDNSエラーは正しく処理されません。
  • NSECレコードの処理の自動並列化はありません-名前空間を手動で分割する必要があります。
  • 高メモリ消費。

Acunetix


Acunetixは、Webアプリケーションのセキュリティをチェックするプロセスを自動化するWeb脆弱性スキャナーです。SQLインジェクション、XSS、XXE、SSRF、およびその他の多くのWeb脆弱性についてアプリケーションをテストします。ただし、多くのWeb脆弱性を持つ他のスキャナーと同様に、複雑な一連の脆弱性やロジックの脆弱性を見つけることができないため、ペンテスターに​​代わるものではありません。しかし、ペンテスターが忘れる可能性のあるさまざまなCVEを含む多くの異なる脆弱性をカバーしているため、定期的なチェックの免除に非常に便利です。

長所：


  • 低い誤検知
  • 結果はレポートとしてエクスポートできます。
  • さまざまな脆弱性について多数のチェックを実行します。
  • 複数のホストの並行スキャン。

マイナス：


  • 重複排除アルゴリズムはありません（Acunetixは、URLが異なるため、ページは機能的に同じであると見なします）が、開発者はそれに取り組んでいます。
  • 別のWebサーバーにインストールする必要があるため、VPN接続を使用してクライアントシステムをテストし、ローカルクライアントネットワークの分離されたセグメントでスキャナーを使用することが複雑になります。
  • 調査中のサービスは、「ノイズを発生させる」可能性があります。たとえば、サイト上の通信の形式に非常に多くの攻撃ベクトルを送信し、ビジネスプロセスを非常に複雑にします。
  • これは独自仕様であり、したがって、フリーではないソリューションです。

wfuzz


wfuzz -Webアプリケーション用のPythonファザー。おそらく最も有名なWebフェイザーの1つです。原理は単純です。wfuzzを使用すると、HTTPリクエストの任意の場所をフェーズできます。これにより、GET / POSTパラメータ、Cookieやその他の認証ヘッダーを含むHTTPヘッダーをフェーズできます。同時に、優れた辞書を必要とする単純なブルートフォースのディレクトリとファイルにも便利です。また、Webサイトからの応答をさまざまなパラメーターでフィルタリングできる柔軟なフィルターシステムも備えており、効果的な結果を得ることができます。

長所：


  • 多機能-モジュール構造、組み立てには数分かかります。
  • 便利なフィルタリングとファジングメカニズム。
  • HTTPリクエストの任意の場所だけでなく、任意のHTTPメソッドをフェーズできます。

マイナス：


  • 開発中で。



ふふ


ffuf -wfuzzの「イメージと類似性」で作成されたGoのWebファザーを使用すると、ファイル、ディレクトリ、URLパス、GET / POSTパラメーターの名前と値、HTTPヘッダー（仮想ホストの総当たりのホストヘッダーを含む）をブルート化できます。Wfuzzは、高速であり、Dirsearch形式の辞書などのいくつかの新機能がサポートされている点で、対応するものとは異なります。

長所：


  • フィルターはwfuzzフィルターに似ており、ブルートフォースを柔軟に構成できます。
  • HTTPヘッダー、POSTリクエストデータ、およびGETパラメーターの名前と値を含むURLのさまざまな部分の値をファジングできます。
  • 任意のHTTPメソッドを指定できます。

マイナス：


  • 開発中で。



ゴバスター


gobuster -Goの偵察用ツールで、2つの操作モードがあります。1つ目はWebサイトのファイルとディレクトリをブルートフォースで攻撃するために使用され、2つ目はDNSサブドメインを反復処理することです。このツールは最初、ファイルとディレクトリの再帰的な列挙をサポートしていないため、もちろん時間を節約できますが、一方で、Webサイト上の新しい各エンドポイントのブルートフォースは個別に実行する必要があります。

長所：


  • ブルートフォースDNSサブドメインとブルートフォースファイルとディレクトリの両方で高速。

マイナス：


  • 現在のバージョンはHTTPヘッダーのインストールをサポートしていません。
  • デフォルトでは、HTTPステータスコードの一部のみが有効と見なされます（200、204、301、302、307）。



アルジュン


Arjunは、JSONだけでなく、GET / POSTパラメータの隠しHTTPパラメータの総当たりのツールです。組み込みの辞書には、Ajrunが約30秒でチェックする25,980語があります。コツは、Ajrunが各パラメーターを個別にチェックするのではなく、一度に最大1000個のパラメーターを即座にチェックし、答えが変更されたかどうかを確認することです。回答が変更された場合、この1000個のパラメーターを2つの部分に分割し、これらの部分のどれが回答に影響するかを確認します。したがって、単純なバイナリ検索を使用して、回答に影響を与えたために存在する可能性のあるパラメーターまたはいくつかの非表示パラメーターを見つけることができます。

長所：


  • 二分探索による高速;
  • GET / POSTパラメータ、およびJSON形式のパラメータのサポート。

Jsparser


JSParserは、TornadoとJSBeautifierを使用してJavaScriptファイルからの相対URLを解析するPythonスクリプトです。AJAX要求を検出し、アプリケーションが対話するAPIメソッドのリストを作成するのに非常に役立ちます。LinkFinderと効果的にペアリング。

長所：


  • JavaScriptファイルの高速解析。



sqlmap


sqlmapは、おそらくWebアプリケーションを分析するための最も有名なツールの1つです。Sqlmapは、SQLインジェクションの検索と操作を自動化し、SQLのいくつかの方言で動作し、「額の中に」ある引用符から時間ベースのSQLインジェクションの複雑なベクトルで終了するまで、膨大な数のさまざまな手法を備えています。さらに、さまざまなDBMSのための多くのメンテナンステクニックがあるため、SQLインジェクションのスキャナーとしてだけでなく、すでに見つかったSQLインジェクションを活用するための強力なツールとしても役立ちます。

長所：


  • 多数の異なる手法とベクトル。
  • 誤検知の数が少ない。
  • WAFをバイパスするための微調整、さまざまな手法、ターゲットデータベース、改ざんスクリプトの多くの可能性。
  • 出力データをダンプする機能。
  • たとえば、いくつかのデータベースには、さまざまな操作オプションがあります-自動ファイルアップロード/ダウンロード、コマンドを実行する機能（RCE）など。
  • 攻撃中に取得したデータを使用したデータベースへの直接接続のサポート。
  • Burp結果を含むテキストファイルを入力に送信できます。すべてのコマンドライン属性を手動で作成する必要はありません。

マイナス：


  • このためのドキュメントが不足しているため、たとえば、いくつかのチェックを書くようにカスタマイズすることは困難です。
  • 適切な設定がないと、不完全な一連のチェックが実行され、誤解を招く可能性があります。



NoSQLMap


NoSQLMapは、NoSQLインジェクションの検索と操作を自動化するためのPythonツールです。NoSQLデータベースだけでなく、NoSQLを使用してWebアプリケーションを監査するときにも直接使用すると便利です。

長所：


  • sqlmapと同様に、潜在的な脆弱性を見つけるだけでなく、MongoDBとCouchDBが悪用される可能性をチェックします。

マイナス：


  • Redis、CassandraのNoSQLはサポートされていません。この方向で開発が進行中です。


oxml_xxe


oxml_xxeは、何らかの形式でXML形式を使用するさまざまなタイプのファイルにXXE XMLエクスプロイトを埋め込むためのツールです。

長所：


  • DOCX、ODT、SVG、XMLなど、多くの一般的な形式をサポートしています。

マイナス：


  • PDF、JPEG、GIFのサポートは完全には実装されていません。
  • ファイルを1つだけ作成します。この問題を解決するには、さまざまな場所に多数のペイロードファイルを作成できるdocemツールを使用できます。

前述のユーティリティは、XMLを含むドキュメントをロードするときにXXEをテストする優れた機能を果たします。ただし、XMLフォーマットハンドラーは他の多くの場合にも見られることを忘れないでください。たとえば、XMLをJSONの代わりにデータフォーマットとして使用できます。

したがって、多数の異なるペイロードが含まれている次のリポジトリに注意することをお勧めします：PayloadsAllTheThings。

tplmap


tplmapは、サーバー側のテンプレートインジェクションの脆弱性を自動的に検出して悪用するためのPythonツールで、sqlmapと同様の設定とフラグがあります。ブラインドインジェクションを含むいくつかの異なる手法とベクトルを使用し、コードを実行して任意のファイルをロード/アンロードする手法も備えています。さらに、テンプレートの12種類の異なるエンジンの武器とeval（）を見つけるためのいくつかの手法（Python、Ruby、PHP、JavaScriptでの同様のコードインジェクション）があります。操作が成功した場合は、インタラクティブコンソールを開きます。

長所：


  • 多数の異なる手法とベクトル。
  • テンプレートをレンダリングするための多くのエンジンをサポートします。
  • 多くの操作技術。

セウル


CeWLは、指定されたWebサイトから一意の単語を抽出するように設計されたRuby辞書ジェネレーターで、サイト上のリンクを指定された深さまでたどります。コンパイルされた一意の単語の辞書は、後でサービスのブルートフォースパスワードや同じWebサイトのブルートフォースのファイルやディレクトリに使用したり、ハッシュキャットやJohn the Ripperを使用して受信したハッシュを攻撃したりできます。潜在的なパスワードの「ターゲット」リストをコンパイルするときに役立ちます。

長所：


  • 使いやすい。

マイナス：


  • 余分なドメインをキャプチャしないように、検索の深さに注意する必要があります。

弱いパス


Weakpassは、固有のパスワードを持つ多くの辞書を含むサービスです。これは、ターゲットサービスの単純なオンライン総当たりアカウントから、ハッシュキャットまたはJohn The Ripperを使用して受け取ったオフライン総当たりハッシュで終わる、パスワードのハッキングに関連するさまざまなタスクに非常に役立ちます。これは、4〜25文字の長さの約80億のパスワードで構成されます。

長所：


  • これには、特定の辞書と最も一般的なパスワードを持つ辞書の両方が含まれています。自分のニーズに合わせて特定の辞書を選択できます。
  • 辞書は更新され、新しいパスワードで更新されます。
  • 辞書はパフォーマンスでソートされます。オンラインでブルートフォースをすばやく実行するためのオプションと、最新のリークを含む膨大な辞書からパスワードを完全に選択するためのオプションを選択できます。
  • あなたの機器のパスワードの総当たり時間を表示する計算機があります。
