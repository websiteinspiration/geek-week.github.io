关于起飞概率的思考：配备Momentum True Wireless 2的Apple AirPods

长期以来，全无线耳机市场一直在等待能够为苹果产品提供适当替代产品的东西。经常有人试图将苹果公司从孵化场的至少一部分转移出去，但到目前为止，还没有人能够做到。索尼尝试使用他们的WF-1000XM3，但是“杀手级”的第一版笨重，“感谢”它在操作上存在人体工程学缺陷。



Sennheiser将进行另一种尝试。祖父弗里茨（Fritz）的创意也不是第一次尝试。与索尼不同，德国人的试球-Sennheiser Momentum True Wireless实现了中间目标，尽管价格高昂，但它却成为了真正抢手的小工具。我们计算出德国“狙击手” Sennheiser Momentum True Wireless 2是如何第二次瞄准美国苹果公司。

关于第一次尝试的一点点


Momentum True Wireless的第一版并未定位为Apple AirPods的竞争对手。首先，由于它被创建为昂贵的高级耳机，音乐和其他所有东西都需要它。传统上，Sennheiser一直非常关注驱动程序的质量，低失真，缺乏共鸣，简而言之，就是该公司产品受到音乐爱好者和其他哲学家的赞赏。

但是，概念主义者和营销人员森海塞尔的计算并不完全正确。大众市场关于“应该如何”的想法与音乐爱好者的需求相去甚远。市场需要，即使不是根本不同的耳机，也需要从“小插头发出的惊人声音”转向功能性和舒适性的设备。这是可以容忍的（在最坏的情况下），并且在音乐和发烧友的标准下，听起来很不错。



因此，Momentum True Wireless仅适用于对金耳有一定要求，对谐波失真，互调有要求的人，以及同样想要频率响应的人。传统上，它们以专有的“ Sencheiser声音”为特色，强调低，中高音和高音，这再次使以前欣赏Sennheiser的人感到高兴。

错误处理


在分析了市场的真正需求之后，这家欧洲旗舰耳机制造商决定在理论上做一些能够与日本和韩国公司的新发展相抗衡的事情，并在理想情况下从苹果手中抢占了一块市场。因此，第二版的Momentum True Wireless（第二代）获得了主动降噪，重新设计的设计以及更长的电池寿命。当前，一次充电后，设备最多可以自主运行7个小时，并且在使用充电盒（+3次自主充电）的情况下，自主使用的可能性增加到28个小时。重要的是要注意，我指出了最长的时间，如果使用非常频繁，则会减少3-4小时，但是即使在这种情况下，它也是一天的电池寿命（即在最坏的情况下，不低于AirPods Pro的上限）。



此外，耳机真的很大声且“低音”，灵敏度高达107 dB的强大放大器和7毫米驱动器可打造出真正深沉的底部。在第一个模型中，扩散器的直径更大，同时，其减小对低频性能没有太大影响。



作为“智能”创新，耳机从设备上移开后会暂停音乐播放，并在插回耳机时开始播放音乐。另外，如有必要，可以关闭降噪功能并使用“透明听力”模式，该模式在必要时会跳过外部噪声和大声讲话。该小工具可与Sennheiser Smart Control应用程序配合使用（适用于Android / iOS）。借助它，您可以配置每个耳机上的触摸面板以控制播放，音量，还可以拨打Google Assistant或Siri。

尺寸引起争议的时刻


与苹果相比，它们的体积要大得多。一方面，这对声音有好处，因为扬声器系统的体积越大，播放的保真度就越高，另一方面，它在人体工程学上的利润较低，并且可能使习惯于小巧的AirPods的用户感到恐惧。还应该注意的是，尺寸并不像索尼WF-1000XM3那样大，而回放的保真度与日文相当，甚至可能超过日文。
