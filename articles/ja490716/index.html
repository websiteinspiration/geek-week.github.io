小規模企業のローカルネットワークの高品質な運用を保証する方法

小規模な企業にはローカルネットワークが必要ですか？コンピュータ機器の購入、保守要員の給与、ライセンスされたソフトウェアの支払いに少なからず資金を費やす必要があるのでしょうか。

著者は、小企業（主にLLC）の所有者および管理者のさまざまなカテゴリ（主に若い）とコミュニケーションをとる必要がありました。さらに、ローカルエリアネットワークはビジネス開発の万能薬であり、それなしではすべてが失われ、運が悪いという意見から、ローカルエリアネットワークはビジネスリーダーにとって大きな負担であり、「頭痛の種」であるという意見まで、まったく反対の意見でした。

この記事では、著者はローカルエリアネットワークを使用することの長所と短所（もちろんすべてではありませんが、最も明白なもの）を理解しようとします。彼は自分自身を理解し、物語の主な目的-小規模企業が常にローカルネットワークを必要とするかどうかを読者に伝えようとします。

この記事を読んだ後（最後まで読んだ場合）、この出版物の著者の能力について意見を述べる前に、著者は彼が科学者ではなく、会社を管理しておらず、LLCの創設者ではないことを尋ねます。著者は、サンクトペテルブルグ州立技術研究所の通信講座の3年生で、研究対象の1つに割り当てられた内容に従って記事を執筆しようとしています。

小規模企業独自のローカルネットワークを持つかどうかの質問に答えるために、著者は少なくとも10人の従業員を持つ企業を検討します。

1人の従業員がCEOであるLLCを検討しても意味がありません。なぜ彼はローカルエリアネットワークを必要とするのですか？結局のところ、そのような会社の会計報告でさえ、自分のコンピュータとソフトウェアを備えた雇われた会計士によって保持されています。そのようなCEOは、コンピューターさえ持っていないかもしれませんが、特別なソフトウェアでさえ、なおさらです。

この記事を書くとき、著者は主にサービス部門で働く会社を考慮します。これらには、保険会社、不動産代理店、会計サービス会社が含まれます。

著者によると、主なタスクは、特定の企業向けのローカルエリアネットワークを開発して構築することではなく、ネットワークが必要かどうかを把握することです。ネットワークとその近代化を構築する上での障害は何ですか。

この場合、ローカルネットワークがネットワークの機器であるだけでなく、このネットワークを運用している会社のソフトウェアや従業員でもあることを即座に判断する必要があります。
多くの対談者（企業の通常の従業員と経営陣）によると、ローカルネットワークが必要です。これにより、作業が容易になり、専用のソフトウェアにアクセスできるようになり、会社のドキュメントを操作できるようになります。

多くの人によると、唯一の主な問題は、ローカルネットワークの機器とソフトウェアのコストが高いことです。

著者によれば、ネットワークのハードウェアについては、高度な技術を追いかけたり、エリート製造会社から常に高度な機器を購入したりする必要はありません。設立時の各企業は、必要な仕事の数をほぼ表します。したがって、ローカルネットワークを作成するとき、ケーブルを敷設するとき、ソケットをインストールするとき、および機器を購入するとき、著者によれば、25％のパワーリザーブを作成する必要があります。これにより、会社は数年間問題なく働くことができます。最大限に機器を絞り出してから、余裕を持って新しい強力な機器を購入する必要があります。

すぐに「必死」の速度でインターネットを購入する必要はありません。プロバイダーへの支払いを増やすことで、いつでもインターネットを増やすことができます。ただし、従業員がネットワークで何をしているかを監視し、必要に応じてインターネットアクセスを制限する必要があります。一部の従業員が大量のトラフィックを消費する「高度な」ゲームをプレイしているとは想定できません。また、ネットワークの速度が遅いため、働く専門家は不便を経験します。これらのプレーヤーがインターネット上でウイルスを「キャッチ」し、会社のソフトウェアに問題を引き起こすと、さらに悪化します。
会社の業務が順調に進んで利益が増加し、従業員数を増やす必要がある場合は、ネットワークのアップグレード、またはより強力な新しいネットワークの作成を検討できます。著者によると、すべてを最も高度なものにするために努力するのではなく、非常に古くて貧弱な機器で作業するのではなく、中間点を見つけることが必要です。

ソフトウェアを使用して、次の手順を実行します。著者は、WindowsやMac OSよりもオープンソースのオペレーティングシステムを使用する方が良いと考えています。これらの特許取得済みオペレーティングシステムのメーカーがユーザーを監視する詳細については触れません。商取引のみを扱います。 Linuxオペレーティングシステムは、サーバーやパーソナルコンピューターにインストールでき、コンピューターリソースの消費がはるかに少なく、さらに大手企業のソフトウェアもLinux用に作成されています。 Windows XPとWindows 7のように、企業が製品のサポートを停止するのを常に待つ必要はありません。同時に、ライセンスソフトウェアを使用するために少額を支払う必要があります。

保存してはいけないのは、ウイルス対策と会社の主要なアプリケーション（1C：経理など）だけです。これらのプログラムはコンピュータを保護し、会社に仕事を提供します。

偽造ソフトウェアをインストールしないでください。これは、ウイルスの感染、ハッキング、またはすべてのソフトウェアの完全な破壊に対する脅威をもたらすだけでなく、法律に問題を引き起こす可能性があります（確かにそれを引き起こすでしょう）。同じ理由で、従業員がローカルネットワークに接続していない場合でも、従業員のパーソナルコンピュータの使用を禁止する必要があります。

ソフトウェアの使用を制御する分野の政府機関が、ライセンスされていないソフトウェアを含む個人用ラップトップを使用して、会社の従業員を職場外に拘束した場合-これは違反ですが、会社はそれに関与していません。彼は責任を負います（行政または民事）が、罰金および請求の額は、多額ではありますが、それほど大きくはありません。そして、彼は独立して責任を負います。

しかし、その検証により、仕事やパソコンでのライセンスされていないソフトウェアの使用が明らかになり、会社の従業員の職場である場合、実際の問題が発生します。罰金や訴訟は非常に大きくなります。さらに、刑事責任が発生する可能性があります。

著者によると、ソフトウェアを使用するときは、会社の2つの基本原則を遵守する必要があります。

高品質のローカルネットワークの構成における3番目のコンポーネントは、有能でよく訓練されたスタッフです。システム管理者だけが会社のネットワークの編成と運用の原則に精通している必要はありません。ネットワークの基本的な理解は、すべての従業員がコンピュータで作業する必要があります。

企業がオープンソースコンピュータを使用する場合、従業員はそれを使用できるはずです。 Windowsを使用することは、習慣の力であり、ファッションへのオマージュであり、確立されたステレオタイプです。 Windows OSからLinux OSへの再トレーニングは、各企業で（著者が望んでいる）上級ユーザーにとって難しいことではありません。そうでない場合は、そのような従業員を再訓練するか、解雇するか、ライセンス取得済みのWindows OSを購入する必要があります。いずれにせよ、選択は常に会社の所有者と経営者に残ります。ただし、企業の企業プロファイルを教えることを熱心に学ぶコンピュータスペシャリストの方が、コンピュータでうまく機能する方法を学びたくない企業のスペシャリストよりもはるかに簡単であるという事実を常に考慮する必要があります。これは著者の個人的な意見ですが、彼は誰にも課さないようにしています。

中小企業向けのローカルネットワークを作成する必要性と、その高品質な運用を保証する能力を理解しようと試みて、著者は特定の結論に達しました。

まず、小規模な企業ではローカルネットワークが必要です。それは従業員の仕事を容易にし、加速し、経営陣が部下の仕事を管理し、会社の成功と問題に遅れないようにするのを助けます。

第二に、企業のローカルネットワークの作業を整理し、3つの主要な問題の包括的な解決策のみで作業状態を維持することが可能です。実行可能な機器、高品質のソフトウェア、および訓練を受けた要員が必要です。何かを改善することは不可能ですが、何かを悪化させることは、良いことにはつながりません。改善する必要があるだけで、コンプレックスで改善します。
