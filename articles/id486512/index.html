Secara singkat tentang cara mengajar Brother MFP untuk memindai di Windows Server



Ini bahkan bukan artikel, tetapi catatan tentang cara menggunakan fitur jaringan Brother MFP Scanner di lingkungan Microsoft Windows Server. Ini akan mengenai pemasangan penuh driver perangkat. Contoh dengan Brother MFC-L2700DRN akan dipertimbangkan . Secara resmi, fungsi pemindaian hanya didukung pada OS desktop, OS server hanya dapat mengirim untuk mencetak . Selama mencari solusi, saya sering menemukan forum dengan masalah yang sama dengan pabrikan ini. Semuanya bermuara pada penggunaan FTP, SMB atau Email.

Oleh karena itu, solusi universal akan disajikan di bawah ini.

Bagian 1. Modifikasi installer


Paket driver lengkap diperlukan untuk menginstal pemindai. Karena OS target dalam kasus saya adalah Windows Server 2016, penginstal akan untuk Windows 10 . Unduh, buka paket.
Selanjutnya, Anda perlu mencari tahu driver mana yang bertanggung jawab untuk pemindai. The install / MFC-L2700DN_series folder berisi Brinst_Common.ini berkas . Di dalamnya kita menemukan baris EnableServer = 0 di bagian [Spesifikasi] dan mengubah nilainya menjadi 1. Parameter ini menonaktifkan pemeriksaan untuk sistem server. Simpan file. Selanjutnya, cari baris ScannerInfName = di bagian [Driver] . Driver ini akan dimodifikasi.

File driver ada di folder instal \ driver \ gdi \ 32_64. Hal ini diperlukan untuk melakukan hal berikut di dalamnya: cari bagian [Pabrikan] dan tambahkan NTamd64.XY ke akhir daftar setelah titik desimal, di mana X dan Y sesuai dengan versi OS Anda. Anda bisa memeriksanya di sini . Untuk Server 2016 terkait dengan Windows 10 dan sama dengan NTamd64.10.0

Bagian 2. Server dan instalasi


Fitur kecil dari instalasi adalah kebutuhan untuk dua reboot. Ini karena itu, bahwa server perlu dimasukkan ke "mode uji" untuk menonaktifkan verifikasi tanda tangan digital driver. Setelah instalasi, hidupkan kembali semuanya dan itu akan berfungsi.

Buka baris perintah atau PS (sesuka Anda) dan masukkan bcdedit -set TESTSIGNING ON , setelah itu kita reboot sistem.

Dengan cara standar, jalankan installer Brother di install \ MFC-L2700DN_series \ setup.exe TETAPI! jangan menginstal driver printer jaringan, jika tidak installer akan mengeluarkan kesalahan dan Anda harus menghapus perangkat yang tidak dikenal secara manual (diinstal dengan menambahkan printer). Barang-barang yang tersisa secukupnya.



Setelah instalasi selesai, buka baris perintah atau PS dan masukkanbcdedit - atur TESTSIGNING OFF , lalu reboot sistem. Memeriksa pemindai.
