無料サイトの時間です

Hi％username％！




今日、多くの初心者Web開発者が大きな間違いを犯していますが、それは間違いではありません。彼らは何かを作り、それからホスティングを購入します。次に、ドメインを購入します。SSL証明書を登録して接続します。マイナスのカルマから逃げるとき、私はテストプロジェクトにお金を費やさない方法を教えます。

ちなみに、ここには広告が表示されていません。必要なリソースの説明があり、可能な限り明確になっている、別のチュートリアルです。

このようなプロジェクトごとに、新しいメールを作成し、個人のメールではなく、どこにでも登録することをお勧めします。

ホスティング


最初に 「無料ホスティング」グーグルを最初にリクエストすると、当然広告の後に000webhost.comが提供されます。これは非常に興味深いホスティングです。既に2年間使用していて、無料サイトの許容数やその他の条件が絶えず変化していることに気づきましたが、主なものは、それが最も便利であることです。

だから、今日のために彼らは提供します： 

  • 1つの無料サイト
  • 1つのMySQLデータベース
  • PHPの複数バージョン
  • ドメイン接続
  • SSDに300 MBの空き容量（ギガバイトありました）
  • FTP

以前はそれより優れていましたが、これは私たちのテストプロジェクトに適していると思います。さらに、これらの欠点は、競合他社の多くが自慢できない便利なコントロールパネルによって補われます。

何をすべきか？


  1. 登録は簡単！
  2. [サイトを作成]をクリックして、ユーザーの質問に答えます。

それで全部です。000webhostに戻ります。その間 ...

ドメイン名


作業プロジェクトに理想的な選択肢はありません。ただし、ミニプロジェクトに参加するので、必要なのはセカンドレベルのドメインだけです。Freenomは私たちの助けであり、検索結果の最初でもあります。類似物はありません。彼らはすべて購入し、一部の国からドメインの販売に関する独占を受けました。

-私たちは、近い問題になったように、任意の遠いアフリカ諸国のドメインのみが利用できる上www.freenom.com「：彼らは自由のための彼らのドメインを分散することにより、インターネットを促進することを決定したところ、.tk」、「.mlの」、「.gq」、「の.cf」 、「. Ga」。当然、彼らは000webhostのようなお金の愛好家であり、わずか12か月間無料でドメインを提供します。最大ですが、その後、再登録できます。

だから、選択してください。

手順＃1


  1. 登録は簡単！
  2. 上部の[サービス]タブに移動し、[新しいドメインを登録]をクリックします。
  3. 結局のところ、サービス自体が教えてくれます。
  4. ドメイン登録が完了したら、もう一度[サービス]をクリックし、[マイドメイン]をクリックします。このタブは閉じないでください。

無料のホスティングに戻ります...

手順2


  1. 再び000webhostにアクセスし、醜い第3レベルドメイン名（name.000webhost.com）を使用してサイトを表示します。修理する。
  2. 綺麗な絵にカーソルを合わせる-登場。碑文「サイトを管理する」をクリックします。
  3. 左側のサイドバーに「ツール」が表示され、リンクをクリックします。
  4. 「Webアドレスを指定」を直感的に選択
  5. ここにボタン-「+ Add Domain」、クリック！
  6. 美しいモーダルウィンドウが表示され、最初の項目を選択します。ドメインを「パーク」します。

クラウドフレア


「Cancer of the Internet」は、このような素晴らしい無料サービスの素晴らしい代替名です。似合うと思います。さらに、CloudFlareはDDOS攻撃から私たちを保護し、サイトをキャッシュして高速化し、無料の証明書を提供します。とても快適です。

かんたん


  1. CloudFlareに登録され、無料プランを選択しています。
  2. 私たちは私たちのサイトを追加します：あなたは再び行き、Freenomのネームサーバーを変更する必要があります-古いものを削除して、サービスが提供するものをインストールしてください。
  3. すぐにSSLを構成するように求められます。「柔軟」なアイテムをお勧めします。
  4. 設定には面白いことがたくさんあります。
