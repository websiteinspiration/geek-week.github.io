CSSのautoキーワードに関するすべて

CSSには、auto要素のさまざまなプロパティを操作するときに使用できるキーワードがあります。これらは、要素の位置、高さ、および幅に影響を与えるプロパティです。これらは、要素のインデントやその他の特性を調整するために設計されたプロパティです。自分の知っていることすべてをどこかに記録したいと思っていましたauto。たとえば、このすべてを、このキーワードの使用の複雑さに興味がある人のためのリファレンスになる可能性のある資料の形式で整理するために。

このキーワードautoは、さまざまなCSSプロパティで使用されると特別な意味を持ちます。autoこの値をさまざまなプロパティに適用することに関する特徴を分析します。 ここでは、まず、作業の技術的な詳細に関心があります。

Widthプロパティ：auto


<div>または などのブロック要素の最初の幅<p>はの値autoです。これは、そのような要素がそれらを含むブロックの水平空間全体を占めるという事実につながります。

「要素を含むブロックの幅」は、CSS仕様に従い、次の式で計算される値です。

‘margin-left’ + ‘border-left-width’ + ‘padding-left’ + ‘width’ + ‘padding-right’ + ‘border-right-width’ + ‘margin-right’

要素の幅が値を指定するとauto、それはこのような特性を構成することができmargin、padding、borderそしてそれは、従って、その親要素よりも大きくないであろう。ブロック要素のコンテンツを制限するブロックの幅は、コンテンツのサイズから、プロパティmargin（フィールド、外側のインデント）、padding（内側のインデント）、およびborder（境界）で定義されたパーツの幅を引いたものになります。

.item__action {
