ML.NETおよびモデルビルダーの更新がリリースされました：新機能

Model Builder  と  ML.NETのアップデートがリリースされました  。ML.NETは、.NET開発者向けのオープンソースのクロスプラットフォーム機械学習環境（Windows、Linux、macOS）です。

ML.NET  は、  AutoMLを使用してカスタムMLモデルを簡単に作成できるように設計されたモデルビルダー  （シンプルなユーザーインターフェースツール）とコマンドラインインターフェースを提供します。

開発者はML.NETを使用して、既存のツールとスキルセットを活用してAIをアプリケーションに開発および実装し、テキスト感情分析、推奨事項、画像分類などの一般的なシナリオ用のカスタム機械学習モデルを作成できます。



さらに主要なニュース：

モデルビルダーの更新


このモデルビルダーのリリースでは、新しいシナリオのサポートが追加され、ユーザーから報告された問題の多くが修正されました。



機会の開発：以前のバージョンのモデルビルダーでは、ファイルまたはSQL Serverからデータセットを選択した後、予測する列（ラベル）のみを選択する機会がありました。データセット内の他の列は自動的に予測に使用されました（機能）。含めたくない列は、モデルビルダーの外部のデータセット自体で変更する必要があり、その後、変更されたデータセットが読み込まれました。



モデルビルダーの以前のバージョンでは、アプリケーションでトレーニング済みモデルを使用するために、コードとモデルビルダーモデルを生成した後、作成したライブラリプロジェクトへのリンクの追加、[出力にコピー]プロパティを[コピーする場合]に設定するなど、多くの手順を実行する必要がありました。新しい」とMicrosoft.ML NuGetパッケージをアプリケーションに追加します。

これはすべて単純化され自動化されているため、次のステップからモデルビルダーにコードをコピーして貼り付けるだけで、アプリケーションを実行できます。

ML.NETの更新


これは、過去数か月間にML.NETに追加された機能と拡張機能の簡単な説明です。

  • .NET Core 3のサポート
  • 販売予測、異常検出などの新しいシナリオのサポート
  • プレビュー：リレーショナルデータベースで直接トレーニングできる独自のデータベースローダー
  • プレビュー：TensorFlowを使用して画像を分類するためのカスタムの深層学習モデルを作成します。



ドキュメントの更新


モデルビルダー、CLI、ML.NET Frameworkのドキュメントやチュートリアル、チュートリアルなどを追加するために一生懸命に取り組みました。また、ML.NETドキュメントの目次を簡略化して、必要なデータを簡単に見つけられるようにしました。
