ジュニアからミドル：パーサー

こんにちは、カブロフチャン！長い間、記事を書きたいという願望がありましたが、勇気はありませんでした。
私の一連の投稿は、ジュニアからミドル、そしておそらくシニアへの私のパスを説明します。 Pythonでプログラミングします。

あまり知られていないライブラリのPSドキュメントが最後に添付されます。

私自身について簡単に言うと、Pythonは2年前に研究を始めましたが、特別な進歩はありませんでした。
1年半教えたので、ファンダメンタルズ自体ががっかりしました。今、私は言語を深く研究しています。その構造とすべてがどのように起こるかを理解しています。次の記事では、コードをアップロードし、すべての新しいライブラリと新しいライブラリを使用します。ご存知のとおり、Pythonには多くの機能があります:)

どこから始めますか？数か月前、初めてパーサーを作成しました。パーサーを書くのは非常に簡単で、お金を稼ぐことさえできることがわかりました。これまでのところ、bs4 +リクエストからのスタックを使用した例をいくつか示します。お気に入りのHabrを解析します。

#     
from bs4 import BeautifulSoup as bs
import requests
from fake_useragent import UserAgent

誰もがfake_useragentライブラリに精通しているかどうかはわかりません。これは解析に非常に便利で、ユーザーエージェントの偽物を作成します。

ua = UserAgent()
headers = {'accept': '*/*', 'user-agent': ua.firefox}

これは、コードの次の小さなブロックです。最初の行では、UserAgentクラスのメソッドを使用するua変数を作成しました。2行目では、将来の解析に役立つ辞書を作成しました。

次に、解析関数自体を作成します。

def without_post(url, headers):
	response = requests.get(url, headers=headers)
	if response.status_code == 200:
		soup = bs(response.text, 'html.parser')
		links = {}
		for i in soup.find_all('a', {'class': 'post__title_link'}):
			links.update({i.text: i.get('href')})
		return links
	else:
		print("Connection Error")

この関数はリンクを解析し、「post__title_link」というクラスのタグの存在を示します。

データを保存する最も簡単な方法は、データをファイルに保存することです。じゃあやってみましょう。

url = "https://habr.com/ru/all/"
links = without_post(url, headers)
with open('parsed.txt', 'w') as f_obj:
	for name, href in links.items():
		f_obj.write(name + ':\n' + href + '\n\n')

最後に、最初のページから収集した投稿の名前とリンクを含むファイルを取得します。

ソース（完全）コード、コメントなしのみ：

from bs4 import BeautifulSoup as bs
from fake_useragent import UserAgent
import requests

ua = UserAgent()
headers = {'accept': '*/*', 'user-agent': ua.firefox}

def without_post(url, headers):
	response = requests.get(url, headers=headers)
	if response.status_code == 200:
		soup = bs(response.text, 'html.parser')
		links = {}
		for i in soup.find_all('a', {'class': 'post__title_link'}):
			links.update({i.text: i.get('href')})
		return links
	else:
		print("Connection Error")

url = "https://habr.com/ru/all/"
links = without_post(url, headers)
with open('parsed.txt', 'w') as f_obj:
	for name, href in links.items():
		f_obj.write(name + ':\n' + href + '\n\n')

良い結果です！行のスキップとコメントを除いて、正確に20行に達しました。手始めに、かなり良い:)
