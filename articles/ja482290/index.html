SQLiteを使用した経済性のあるDiscordボット

こんにちは、ハブル！


私は、sqliteやその他の小さなことを使用して、discordサーバー用の経済学を備えたボットを作成したときの経験を共有したいと思いました。

準備段階


ボットの作成


したがって、ボットを作成する前に、ボットを作成してトークンを取得する必要があります。

  1. デベロッパーサイトに移動
  2. 「新しいアプリケーション」ボタンをクリックして、ボットに名前を付けます
  3. ボットにログインし、設定リストでボットタブを見つけてボットトークンを取得します

必要なモジュール


結局のところ、ASMではなくpythonで記述しているため、いくつかのモジュールが必要になります。

主なパートの1つを作成しました。タイプのアカウント、ストアなどのチームを作成することは残っています。さらに直感的に思います。

@bot.command()
async def account(ctx): # _account ( "_",     )
    table=[["nickname","money","lvl","xp"]]
    for row in cursor.execute(f"SELECT nickname,money,lvl,xp FROM users where id={ctx.author.id}"):
        table.append([row[0],row[1],row[2],row[3]])
        await ctx.send(f">\n{tabulate(table)}")

@bot.command()
async def inventory(ctx):# _inventory ( "_",     )

    counter=0
    for row in cursor.execute(f"SELECT inventory FROM users where id={ctx.author.id}"):
        data=json.loads(row[0])
        table=[["id","type","name"]]
        for row in data:
            prt=row
            for row in cursor.execute(f"SELECT id,type,name FROM shop where id={prt}"):
                counter+=1
                table.append([row[0],row[1],row[2]])
                
                if counter==len(data):
                    await ctx.send(f'>\n{tabulate(table)}')


@bot.command()
async def shop(ctx):# _shop ( "_",     )
    counter=0
    table=[["id","type","name","cost"]]
    for row in cursor.execute(f"SELECT id,type,name,cost FROM shop"):
        counter+=1
        table.append([row[0],row[1],row[2],row[3]])
        if counter==4:
            await ctx.send(f'>\n{tabulate(table)}')
