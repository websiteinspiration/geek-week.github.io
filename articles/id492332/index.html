Buat aplikasi secara gratis dengan Azure Cosmos DB Free Tier

Ingin membuat aplikasi baru, mengembangkan dan menguji, atau menjalankan beban kerja produksi kecil dengan  Azure Cosmos DB ? Tingkat baru penggunaan gratis kami memungkinkan Anda untuk mulai bekerja tanpa biaya dan menghemat uang saat membuat dan mengembangkan aplikasi baru.



Dengan tier gratis Azure Cosmos DB yang disertakan, Anda mendapatkan 400 RU / s bandwidth pertama dan 5 GB penyimpanan di akun Anda secara gratis setiap bulan selama seluruh durasi akun. Ini berarti Anda dapat memulai dari yang kecil dan tumbuh dengan percaya diri, mengetahui bahwa aplikasi Anda akan berjalan dalam layanan database berkinerja tinggi. Anda hanya akan membayar jika akun Anda melebihi 400 RU / dt dan 5 GB. Selain itu, jika aplikasi Anda memiliki banyak kontainer, Anda dapat membuat hingga 25 kontainer dalam database dengan bandwidth total, dan semuanya akan menggunakan 400 RU / s gratis. Anda dapat memiliki hingga satu akun Azure Cosmos DB gratis untuk setiap langganan Azure.


Apa yang dapat Anda buat secara gratis?


Kami meminta anggota komunitas pengembangan kami untuk memberikan contoh aplikasi dengan 400 RU / s dan 5 GB ruang disk dan menerima banyak contoh bagus dari aplikasi yang mereka buat: aplikasi Alexa, yang membantu penumpang untuk menggunakan transportasi umum, serta aplikasi web untuk  mengelola dan berbagi daftar online dengan mudah sumber daya . Kami akan berbicara tentang beberapa pengembang ini dan aplikasi mereka dalam beberapa minggu dan bulan mendatang.

Apakah Anda memiliki aplikasi dengan bandwidth kurang dari 400 RU / s dan penyimpanan 5 GB? Ceritakan tentang dia  - kami akan dengan senang hati membagikannya dengan komunitas.

Awal pekerjaan


Buat akun Azure Cosmos DB baru dan aktifkan  tingkat gratis  selama pembuatan. Ada batasan satu akun berlangganan Azure gratis. Lihat panduan dan panduan memulai cepat kami   atau gunakan  contoh untuk memahami apa yang dapat Anda lakukan dengan Azure Cosmos DB.

Pertanyaan yang Sering Diajukan (FAQ)


T: Apakah Anda belum menawarkan opsi gratis sebelumnya? Apa yang berubah?

A: Kami selalu menawarkan cara untuk mencoba Azure Cosmos DB secara gratis - sebagai bagian dari  Akun Gratis Azure , dengan  emulator lokal , dan sebagai  uji coba gratis 30 hari  - tetapi opsi ini terbatas dalam ketersediaan dan volume. Tier gratis tersedia untuk waktu yang tidak terbatas.

T: Bagaimana diskon tingkat gratis ditampilkan di akun saya?

A: Di akun Free Tier, Anda akan mendapatkan 400 RU / s dan 5 GB penyimpanan pertama di akun Anda secara gratis. Setiap RU / s dan penyimpanan melebihi 400 RU / s dan 5 GB akan ditagih dengan tarif harga reguler per halaman harga. Pada tagihan, Anda tidak akan melihat tagihan atau item baris gratis 400 Ru / detik dan 5 GB, hanya RU / detik dan penyimpanan di luar apa yang dicakup oleh tingkat gratis. Di akun tingkat bebas, Anda akan menerima 400 RU / dtk dan 5 GB ruang disk pertama di akun Anda secara gratis. Setiap RU / s dan penyimpanan yang melebihi 400 RU / s dan 5 GB akan dibayar dengan tarif reguler. Pada akun Anda tidak akan melihat biaya atau posisi gratis 400 Ru / detik dan 5 GB, hanya RU / detik dan penyimpanan di luar apa yang dicakup oleh tingkat gratis.

T: Dapatkah Free Tier berlaku untuk semua API Azure Cosmos DB?

Ya, Free Tier berfungsi untuk semua API DB Azure Cosmos, termasuk Core (SQL), API untuk MongoDB, Cassandra, dan GREMLIN.

T: Dapatkah saya menerapkan diskon tingkat gratis untuk  basis data dan wadah autopilot ?

A: Ya. Dengan layanan autopilot, Anda akan dikenai tagihan untuk RU / d tertinggi, di mana basis data atau wadah diskalakan per jam. Ketika diskon tingkat gratis diterapkan, 400 RU / d dikurangkan dari nilai ini. Lihat dokumentasi kami  untuk detailnya.

T: Bagaimana cara diskon Free Tier berfungsi jika saya memiliki akun di beberapa daerah?

A: Dalam akun multi-regional RU / s, database atau wadah direplikasi di semua wilayah. Misalnya, jika Anda memiliki wadah dengan 400 RU / dtk, dan akun tersebut berada di 3 wilayah, jumlah total akun RU / dtk adalah 1200 RU / dtk. Ketika diskon diterapkan, Anda akan ditagih pada 1200 RU / s - 400 RU / s = 800 RU / s per jam. Lihat dokumentasi kami  untuk detailnya.

T: Saya sudah memiliki akun dan saya ingin mentransfer data saya ke akun tingkat gratis. Bagaimana saya harus melakukannya?

A: Setelah akun baru Anda dibuat dan diaktifkan dengan Free Tier, memindahkan data yang ada ke akun baru akan mudah.  Alat Migrasi Data dapat digunakan untuk mentransfer dataset kecil jika Anda menggunakan API Core (SQL) kami. Anda juga bisa memigrasi data dari Azure Data Factory  untuk API Azure Cosmos DB Core (SQL)  dan API Azure Cosmos DB  untuk MongoDB .

T: Opsi untuk mengaktifkan tier gratis saat membuat akun baru di portal Azure tidak ditampilkan. Dimana dia?

A: Langganan Azure dapat memiliki hingga satu akun tingkat gratis. Jika opsi ini tidak ditampilkan, itu berarti bahwa akun lain dalam langganan sudah diaktifkan di tingkat gratis.
