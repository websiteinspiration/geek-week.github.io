CSS


この方法は冒頭で触れましたが、とても良い方法です。JavaScriptライブラリをインポートして使用する代わりに、アセンブリが小さい場合、ブラウザーは多くのリソースを必要としません。もちろん、これはアプリケーションのパフォーマンスに影響します。アニメーションを比較的単純にする必要がある場合は、この方法に注意してください。

例はアニメーション化されたメニューです



。CSSプロパティとHTMLタグのclassName = "is-nav-open"のようなトリガーを備えた、比較的シンプルなメニューです。

この方法を使用するにはいくつかの方法があります。たとえば、ナビゲーションのラッパーを作成してから、フィールドの変更を呼び出します。ナビゲーションは250pxの一定の幅を持っているので、margin-leftまたはtranslateXプロパティを持つラッパーの幅は同じ幅でなければなりません。ナビゲーションを表示する必要がある場合は、ラッパーにclassName = "is-nav-open"を追加し、ラッパーをmargin-left / translateX：0;に移動します。

最終的に、アニメーションソースは次のようになります。

export default class ExampleCss extends Component {
    handleClick() {

    }
}

ReactTransitionGroup


ReactTransitionGroupコンポーネントは、ReactJSコミュニティチームによって開発されました。これを使用すると、基本的なCSSアニメーションと遷移を簡単に実装できます。

ReactTransitionGroupは、コンポーネントのライフサイクルを変更するときにクラスを変更するように設計されています。サイズは小さく、Reactアプリケーション用のパッケージにインストールする必要があります。これにより、全体のアセンブリサイズがわずかに増加します。さらに、CDNを使用できます。

ReactTransitionGroupには、Transition、CSSTransition、およびTransitionGroupの3つの要素があります。アニメーションを開始するには、コンポーネントをそれらにラップする必要があります。スタイルは、CSSクラスで記述する必要があります。

これがアニメーション、そしてそれを実装する方法です。



最初のステップは、CSSTransitionGroupをreact-transition-groupからインポートすることです。その後、リストをラップし、transitionNameプロパティを設定する必要があります。CSSTransitionGroupで子を追加または削除するたびに、アニメーションスタイルが取得されます。

<CSSTransitionGroup
    transitionName="example">
    {items}
</CSSTransitionGroup>

transitionName = "example"プロパティを設定する場合、スタイルシートのクラスは、例の名前で始まる必要があります。

.example-enter {
    opacity: 0.01;
}
 
.example-enter.example-enter-active {
    opacity: 1;
    transition: opacity 300ms ease-in;
}
 
.example-leave {
    opacity: 1;
}
 
.example-leave.example-leave-active {
    opacity: 0.01;
    transition: opacity 300ms ease-in;

上記はReactTransitionGroupを使用した例です。

連絡先のリストを追加する例を実装するためのロジックと2つのメソッドも必要です。

最初のメソッドはhandleAddです。新しい連絡先を追加し、ランダムな名前を取得してから、state.items配列に入れます。

state.items配列のインデックスで連絡先を削除するには、handleRemoveを使用します。

import React, { Component, Fragment } from 'react';
import { CSSTransitionGroup } from 'react-transition-group'
import random from 'random-name'
import Button from './button'
import Item from './item'
import './style.css';
export default class ReactTransitionGroup extends Component {
    
    constructor(props) {
        super(props);
        this.state = { items: ['Natividad Steen']};
        this.handleAdd = this.handleAdd.bind(this);
    }
 
    handleAdd() {
        let newItems = this.state.items;
        newItems.push(random());
        this.setState({ items: newItems });
    }
 
    render () {
        const items = this.state.items.map((item, i) => (
            <Item
            item={item}
            key={i}
            keyDelete={i}
            handleRemove={(i) => this.handleRemove(i)}
            />
        ));
 
    return (
        <Fragment>
            <Button onClick={this.handleAdd}/>
                <div className="project">
                    <CSSTransitionGroup
                    transitionName="example"
                    transitionEnterTimeout={500}
                    transitionLeaveTimeout={300}
                    >
                        {items}
                    </CSSTransitionGroup>
                </div>
        </Fragment>
    );
}
};

反応アニメーション


React-animationsは、animate.css上に構築されたライブラリです。扱いが簡単で、アニメーションのさまざまなコレクションを持っています。このライブラリは、ラジウム、アフロディーテ、スタイル付きコンポーネントなど、アニメーションのメインフレームを定義するためのオブジェクトの使用をサポートする任意のインラインスタイルライブラリと互換性があります。



私はあなたの考えを知っています：



では、跳ねるアニメーションの例でそれがどのように機能するかを確認しましょう。



まず、react-animationsからアニメーションをインポートします。

const Bounce = styled.div`animation：2s $ {keyframes` $ {bounce} `} infinite`;

次に、コンポーネントを作成した後、HTMLコードまたはコンポーネントをアニメーション用にラップします。

<bounce><h1>Hello Animation Bounce</h1></bounce>

例：

import React, { Component } from 'react';
import styled, { keyframes } from 'styled-components';
import { bounce } from 'react-animations';
import './style.css';
 
const Bounce = styled.div`animation: 2s ${keyframes`${bounce}`} infinite`;
 
export default class ReactAnimations extends Component {
    render() {
        return (
            <Bounce><h1>Hello Animation Bounce</h1></bounce>
        );
    }
}

すべてが機能し、アニメーションは非常にシンプルです。さらに、スクロール時にバウンスアニメーションを使用するための優れたソリューション-react-animate-on-scrollがあります。

反応を明らかにする


React Revealフレームワークには、フェード、反射、スケーリング、回転などの基本的なアニメーションがあります。小道具を使用してすべてのアニメーションを操作することができます。したがって、位置、遅延、距離、カスケードなどを含む追加の設定を指定できます。サーバー側のレンダリングや高次コンポーネントなど、他のCSS効果を使用できます。一般に、スクロールアニメーションが必要な場合は、このフレームワークを使用する必要があります。

import Fade from 'react-reveal/Fade';
 
<Fade top>
    <h1>Title</h1>
</Fade>



合計5つのブロックがあり、それぞれにフルスクリーンのページとタイトルがあります。

import React, { Component, Fragment } from 'react';

const animateList = [1, 2, 3, 4, 5];
 
export default class ReactReveal extends Component {
    render() {
        return (

                    <div style={styles.block} key={key}>
                        <Fade top>
                            <h1 style={styles.title}>{`block ${item}`}</h1>                       
                        </Fade>
                    </div>
                ))}
            </Fragment>
        );
    }
}
 
const styles = {
    block: {
        display: 'flex',
        alignItems: 'center',
        justifyContent: 'center',
        width: '100%',
        height: '100%',
        background: '#000',

次に、定数animateListを紹介します。配列には5つの要素が含まれます。マップ配列メソッドを使用した後、要素をヘッダーに挿入することで、Fadeコンポーネントの任意の要素をレンダリングできます。styles定数で定義されたスタイルは、ブロックとヘッダーの両方に短いCSSスタイルを取得します。上記は、フェードアニメーション付きの5つのブロックです。

Ant DesignでのTweenOneとアニメーション


Ant Designは、便利で使いやすいコンポーネントを多数含むReact UIライブラリです。エレガントなユーザーインターフェイスを作成する必要がある場合に適しています。多くのプロジェクトでライブラリを使用するAlibabaによって開発されました。



この例には、かなりの数のアニメーションコンポーネントがあります。それらのほとんどは同様のアニメーションを持っているので、実装例は上記よりも単純になります。これには、ボール、緑色のボール、および追加要素（たとえば、赤い正方形）のみが含まれます。



アニメーションはTweenOneコンポーネントを使用します。このコンポーネントには、軌道を正しく設定するためにPathPluginが必要です。これは
、PathPlugin をTweenOne.pluginsに配置した場合にのみ機能します。

TweenOne.plugins.push(PathPlugin);

主なアニメーションオプションは次のとおりです。

  • 継続時間-アニメーション時間（ミリ秒）。
  • 容易さ-滑らかなアニメーション;
  • ヨーヨー-繰り返すたびに前後に動きが変わる;
  • repeat-アニメーションを繰り返します。エンドレスアニメーションには-1を使用する必要があります。
  • p-アニメーションのパスの座標。
  • easePath-アニメーションのスムーズパスの座標。

最後の2つのパラメーターは非常に具体的ですが、心配する必要はありません。すべてが正常に機能します。

const ease = 'easeInOutSine';
const p =

const easePath =
  'M0,100 C7.33333333,89 14.3333333,81.6666667 21,78 C25.3601456,75.6019199 29.8706084,72.9026327 33,70 C37.0478723,66.2454406 39.3980801,62.0758689 42.5,57 C48,46.5 61.5,32.5 70,28 C77.5,23.5 81.5,20 86.5,16 C89.8333333,13.3333333 94.3333333,8 100,0';
const loop = {
  yoyo: true,
  repeat: -1,
  duration,
  ease,
};

これで、アニメーションオブジェクトの作成を開始できます。

  • redSquareには、ループパラメーターに加えて、Y座標、期間、遅延が含まれています。
  • greenBallには、オブジェクトのパラメーターx、y-pの値を持つパスが含まれています。さらに、継続時間、繰り返し、滑らかさは、2つのパラメーターを持つTweenOne.easing.pathの関数です。
  • パス-easePath。
  • lengthPixelは、400のセクションのみに分割された曲線です。
  • track-軸のある楕円形で、サイクルスタイルと回転パラメータがあります。

const animate = {
  redSquare: {
    ...loop,
    y: 15,
    duration: 3000,
    delay: 200,
  },
  greenBall: {
    path: { x: p, y: p },
    duration: 5000,
    repeat: -1,
    ease: TweenOne.easing.path(easePath, { lengthPixel: 400 }),
  },
  track: {
    ...loop,
    rotate: 15,
  },
};
