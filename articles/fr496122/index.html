Prendre soin de vos devoirs: sauvegardez votre ordinateur portable ou de bureau à l'aide des agents Veeam gratuits

Dans la vie ordinaire, comme beaucoup d'entre vous, chers lecteurs, je travaille la plupart du temps au bureau et cela me convient parfaitement. Mais aujourd'hui, la situation est telle que moi et mes collègues dans différentes parties du monde, et, je pense, la majorité des gens qui lisent cet article, nous travaillons tous à domicile.

Et ici, il sera tout à fait hors de propos de rappeler la nécessité de sauvegarder un ordinateur domestique ou un ordinateur portable. Lorsque l'environnement de travail change, la manière habituelle de travailler avec les données change également. Tout est évident avec le travail au bureau: accès direct aux ressources nécessaires, sauvegardes centralisées, le cas échéant, vous pouvez toujours contacter le service d'assistance. Avec le travail à domicile, beaucoup de choses que vous devez prévoir vous-même, y compris la sauvegarde.

Je pense que beaucoup d'entre vous utilisent déjà Veeam Agent pour Microsoft Windows ou Veeam Agent pour Linux pour cela. Si ce n'est pas encore fait, vous serez peut-être intéressé par la brève revue d'aujourd'hui. Alors bienvenue au chat.



En 2014, lorsque la première version de notre produit, désormais connue sous le nom de Veeam Agent pour Microsoft Windows , a été lancée , elle s'appelait Veeam Endpoint Backup Free et possédait la seule édition non commerciale.

Bien sûr, la solution a beaucoup changé depuis lors, et une version commerciale est sortie, dans laquelle il existe de nombreuses fonctionnalités avancées - gestion centralisée, pilote CBT, prise en charge de cluster, traitement de disque parallèle, et bien plus encore. Veeam Agent pour Linux se développe également, ce qui vous permet de sauvegarder des machines physiques et de restaurer des données à partir de sauvegardes. Mais maintenant, dans la cour de 2020, les nouvelles versions de V eeam Agent pour Microsoft Windows et Veeam Agent pour Linuxil existe également une édition gratuite conçue pour les ordinateurs individuels.

Donc, si vous devez sauvegarder votre ordinateur de bureau, votre ordinateur portable ou même une tablette fonctionnant sous Windows, utilisez votre santé.

Comment ça fonctionne


Le principe de fonctionnement de cette solution est le suivant: sur la base de l'image, une copie de sauvegarde est créée, et si nécessaire, vous pouvez effectuer une récupération à nu ou une autre option de récupération à partir de celle-ci.   3. Les données du stockage de destination sont enregistrées dans un fichier de sauvegarde.

Pour Veeam Agent pour Linux, l' histoire est similaire:

  1. Un instantané des volumes requis est créé - cela se fait à l'aide du module spécial Veeamsnap. (Un instantané est créé sur un volume avec suffisamment d'espace libre.)

    Lorsqu'un volume est sauvegardé, Veeam Agent copie la table de partition et le chargeur de démarrage à l'emplacement cible.
  2. Veeam Agent pour Linux lit les données du volume, effectue la compression et la copie vers l'emplacement cible.
  3. Une fois toutes les données nécessaires copiées, l'instantané est supprimé. Les données sont enregistrées dans un fichier de sauvegarde.

Travail de sauvegarde dans Veeam Agent pour Microsoft Windows


Pour la sauvegarde, configurez la tâche de sauvegarde en démarrant l'assistant Nouveau travail de sauvegarde . Dans celui-ci, sélectionnez le type de sauvegarde:

  • Ordinateur entier - la machine entière.
  • Sauvegarde au niveau du volume - sauvegarde des volumes sélectionnés.
  • Sauvegarde au niveau des fichiers - sauvegarde de fichiers spécifiques.

Veeam Agent pour Microsoft Windows peut utiliser différents types de systèmes de stockage comme emplacement de stockage de sauvegarde:

  • Stockage local - lecteur externe (USB, Firewire ou eSata). L'essentiel ici, comme vous le savez, n'est pas de vous sauvegarder sur le disque intégré de la machine.
  • Dossier partagé - Partage SMB sur un serveur de fichiers ou NAS.
  • Référentiel de sauvegarde Veeam - Référentiel Veeam (sur site).
  • Le référentiel Veeam Cloud Connect est un référentiel cloud fourni par le fournisseur de services Veeam.
  • Microsoft OneDrive

Dans mon exemple, la sauvegarde ira sur le lecteur USB - pour cela, sélectionnez le stockage local à l'étape de destination : Ensuite, nous définissons les paramètres spécifiques au type de stockage sélectionné. Important! Lorsque vous travaillez avec des lecteurs USB amovibles, vous devez garder à l'esprit que si un tel lecteur est formaté pour FAT32, il ne prend pas en charge le stockage de fichiers supérieurs à 4 Go. Il n'est donc pas recommandé d'utiliser de tels disques comme stockage de sauvegarde cible. Ensuite, configurez la planification dans l'étape Planification . Ici, vous pouvez spécifier si vous souhaitez sauvegarder régulièrement à certains moments ou le faire lorsque les événements sélectionnés se produisent:









  • Verrouiller - lorsque vous avez verrouillé la voiture
  • Déconnexion - lorsque vous êtes déconnecté du compte courant
  • Lorsque la cible de sauvegarde est connectée - lorsque le stockage de sauvegarde cible est connecté

Si la troisième option est spécifiée, vous pouvez également demander à Veeam Agent de terminer la déconnexion (éjecter) du stockage cible à la fin de la sauvegarde afin que la sauvegarde soit hors ligne. Cela réduira le risque d'exposition à une copie de sauvegarde du virus de chiffrement, s'il en détecte une.

Pour ce faire, sélectionnez l'option Lorsque la cible de sauvegarde est connectée → Éjecter le stockage amovible une fois la sauvegarde terminée (protection CryptoLocker) :



Agent Veeam pour Linux


Veeam Agent pour Linux est conçu pour protéger les données des machines fonctionnant sous Linux, et cette solution a également une édition gratuite. Le noyau Linux 2.6.32 et supérieur est pris en charge.

Les versions 64 et 32 ​​bits (le cas échéant) des distributions sont prises en charge:

  • Debian 8.0 - 10.3
  • Ubuntu 14.04, 16.04, 18.04, 19.10
  • RHEL 6.0 - 8.1
  • CentOS 6.0 - 8.1
  • Oracle Linux 6 - 8.1 (RHCK)
  • Oracle Linux 6 (à partir de UEK R1) - Oracle Linux 7 (jusqu'à UEK R5 U2)
  • SLES 11 SP4, 12 SP1 - 15 SP1
  • SLES pour SAP 11 SP4, 12 SP1 - 15 SP1
  • Fedora 30 - 31
  • openSUSE Leap 15.1
  • openSUSE Tumbleweed

Remarques:

  • Fedora est pris en charge jusqu'à la version 5.4.14 du noyau incluse.
  • openSUSE Tumbleweed est pris en charge jusqu'à la version 5.1.7 du noyau incluse.
  • Le noyau Linux 2.6.32-754.6.3 n'est pas pris en charge sur CentOS / RHEL et Oracle Linux (RHCK).

Pour plus d'informations, consultez la documentation (en anglais).

Les mêmes types de sauvegardes sont pris en charge que dans Veeam Agent pour Microsoft Windows, y compris les sauvegardes au niveau des fichiers. Les paramètres de sauvegarde peuvent être définis à l'aide des étapes de l'assistant ou à partir de la ligne de commande.

Vous trouverez plus d'informations sur chacune des solutions dans le guide de l'utilisateur pour Windows (en anglais) ou pour Linux (également en anglais).

À propos des limites de l'édition gratuite


Pour Veeam Agent pour Linux (par rapport à l'édition commerciale de Server):

  1. Un seul travail de sauvegarde peut être créé.
  2. Le référentiel Veeam Cloud Connect n'est pas pris en charge en tant que référentiel de sauvegarde cible.
  3. Les scripts de pré-gel et de post-dégel ne sont pas pris en charge pour le travail de sauvegarde.
  4. La sauvegarde prenant en compte le travail du SGBD Oracle, MySQL, PostgreSQL n'est pas prise en charge.
