ZyxelゲートウェイでのVPN接続の作成



VPN、ATP、およびUSGシリーズのZyxel Zywall VPNゲートウェイには、両方のエンドユーザーを接続し（ホストからネットワーク）、ゲートウェイ間の接続を作成する（ネットワークからネットワーク）ための安全な仮想ネットワーク（VPN）を作成する広範な機能があります。この記事では、リモートの従業員のためのVPN接続の作成と構成のすべてのポイントについて説明します。

適切なVPNの選択


ゲートウェイを使用すると、3つの接続シナリオを作成できます。

  • L2TP over IPSec VPN
  • SSL VPN
  • IPSec VPN

エンドデバイスのセキュリティ要件とプロトコルサポートに応じて、必要なものが選択されます。

それぞれの構成を検討してください。

プリセット


ゲートウェイのそのような豊富な機能があっても、十分に訓練されていない人にとって接続を行うことは難しくありません、心に留めておくべき主な事柄はいくつかのニュアンスです：

L2TP VPNがWindowsの組み込みクライアントを使用して正しく機能するためには、IKEEXTサービスがアクティブ化されているかどうかを確認する必要があります（キー交換用のIPsecキーモジュール）インターネットと認証付きIPプロトコル）



レジストリの編集に関するインターネットからの残りのヒントは、非常に害を及ぼすだけなので注意してください。

暗号化および暗号化機能を実装するロシア連邦へのネットワーク機器の輸入に関する税関の制限により、Zyxelゲートウェイでは「そのまま」使用できるのはDES暗号化のみです。

現在のファームウェアバージョン4.35では、Webインターフェイスから直接CLIで作業することが可能で、コンソールケーブル経由でゲートウェイに個別に接続する必要がないため、コマンドラインでの作業が大幅に簡略化されます。

残りのモード（3DES）をアクティブにするには、コンソールから次のコマンドを入力する必要があります。

Router> configure terminal
Router(config)# crypto algorithm-hide disable
Router(config)# write
Router(config)# reboot



rebootコマンドを入力すると、ゲートウェイが再起動し、必要な暗号化モードを選択できるようになります。



ニュアンスを理解したので、接続の設定に直接進みましょう。

すべての例は、VPN50ゲートウェイ、VPN2S、Windows PC、Androidスマートフォン、場合によってはiPhoneスマートフォンを使用して説明されます。

VPN50は、オフィス/サーバーにある「ヘッドゲートウェイ」として機能します。

L2TP接続の構成


VPN50でL2TPを構成する


VPNを設定する最も簡単な方法は、ウィザードを使用することです。彼はすぐにすべてのルールと接続を作成します。
ログイン/パスワードを入力すると、イージーモードが開きます（簡易管理モード）。



イージーウィザードで作成されたVPNモードはエキスパートモードで調整できないため、情報モードでのみ使用することを強くお勧めします。
右上のボタン「エキスパートモード」を押して、本格的なウェブ管理インターフェースに入ります。



左上から「魔法の杖」アイコン-クイックセットアップ：



インターネット設定がDHCPと異なる場合は、最初にWANインターフェースを設定する必要があります。 VPNを構成するには、右のアイコン



を選択し、L2TPを構成するには、最後の項目「L2TPのVPN設定」を選択します。



ここでは、接続の名前、トラフィックが通過するインターフェース（インターネット接続がPPPOE接続を介している場合は、それに応じてwan_pppを選択する必要があります）、および暫定キーを入力するように求められます。



次のページで、クライアントに割り当てるIPアドレスの範囲を入力します。
範囲はゲートウェイ上の他のインターフェースと重複してはいけません！

以下のチェックマークで選択されているVPNを介してすべてのトラフィック（クライアントインターネット）を開始する必要がない場合は、DNSサーバーを登録できません。



このページには、作成された接続に関する情報が表示され、すべてのデータが正しいかどうかが確認されて保存されます。

すべて！接続が作成されました。ゲートウェイに接続するユーザーを作成する必要があります。



[構成]-> [オブジェクト]-> [ユーザー/グループ]タブで、ユーザーが作成されます（名前とパスワードを入力する必要があります）。



次のタブでは、便宜上、グループを作成してユーザーのリストを追加できます。複数のリモートユーザーがいる場合は、グループを作成する必要があります。



[VPN-L2TP VPN]タブの[許可されたユーザー]で、以前に作成したグループまたはL2TP経由で接続できる単一のユーザーを選択します。

この場合、ゲートウェイでのL2TP接続の作成は構成済みと見なすことができます。
クライアントの設定に移りましょう。

L2TPクライアントの構成

WindowsクライアントでのL2TPの構成


Windows 10では、L2TP構成は標準ツールを使用して行われます。

 ->  ->    -> VPN ->  VPN-



  • サービスプロバイダー-Windows（組み込み）
  • 接続名-オプション
  • サーバーの名前またはアドレス-VPNゲートウェイのIPアドレス
  • 事前共有キー付きのVPNタイプL2TP
  • 共通キーは、ウィザードの最初のポイントで作成された現在のキーです
  • 許可されたグループのユーザーのユーザー名とパスワード

接続するにはこれで十分です。ただし、接続を少し調整する必要がある場合や、接続が以前のバージョンのWindowsで作成されている場合があります。

AndroidクライアントでL2TPを構成する


モバイル従業員を企業ネットワークに接続する必要がある場合があります。

Androidスマートフォンでは、これはメニューで行われます。

  ->   () -> VPN ->    (           -)



ここで、接続の名前を入力し、VPNタイプL2TP / IPSec PSKを選択し、共有キー、ユーザー名、パスワードを入力します。

接続後、「キー」アイコンが表示され、VPN接続について通知されます。

VPNを作成して接続すると、接続が確立したことを示す「VPN」アイコンが右上隅に表示されます。

VPN2SハードウェアクライアントでL2TPを構成する


次に、リモートクライアントを低コストのZyxel VPN2Sゲートウェイで企業ネットワークに接続する方法を示します。

VPN2SをL2TPクライアントに接続するには、セットアップも難しくありません。「ウィザード」も必要ありません。すべてのルールは最初に作成されました。



まず、構成-> VPN-> IPsec VPNで、標準構成を有効にする必要があります。



ゲートウェイ構成で、ピアゲートウェイアドレスとキーを入力します。



[L2TP VPN]タブで、[有効]チェックボックスをオンにし、IPsec（デフォルトの構成名）を適用します。ユーザー名とパスワードを入力し、必要に応じてNATを有効にします。



その後、[IPsec VPN]タブに戻り、接続トンネルで「緑色の地球」が点灯していることを確認して、接続が確立していることを知らせます。

SSL接続の構成


VPN50ゲートウェイでのSSLセットアップ


SSL VPNを構成するには（ちなみに、ライセンスがない場合はL2TPよりも小さいが、より安全な接続が必要です）、



[VPN]-> [SSL VPN]タブで、名前、ユーザーのリスト\グループ、発行されたIPアドレスのプールとリストを示す新しいルールが作成されます。クライアントがアクセスできるネットワーク。

重要なお知らせ！SSL VPNを機能させるには、HTTPSプロトコルをプロトコル構成に追加する必要があります。



これは、パスオブジェクト->サービス->サービスグル​​ープに沿って行われます。

「Default_Allow_Wan_To_ZyXall」グループにHTTPSを追加します。また、外部からゲートウェイのWebインターフェースにアクセスする必要があります。

SSLクライアントの構成


WindowsクライアントでのSSLの構成


Windowsクライアントに接続するには、ZyWALL SecuExtender 4.0.3.0プログラムを使用します。



設定は、ゲートウェイのIPアドレスとユーザーのユーザー名/パスワードの入力のみで構成されます。
毎回入力しないように、[ユーザーを記憶する]チェックボックスをオンにすることもできます。



接続中に、ポップアップ警告を受け入れます。



接続後、[ステータス]タブに接続情報が表示されます。
