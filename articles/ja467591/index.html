新しい証明は、piなどの数値の近似を解決します

古代ギリシャ人は、不合理な数を分数で大まかに表現できるかどうかに興味を持っていました。長年のDuffin-Schaffer仮説を証明し、2人の数学者が徹底的な答えを出しました。



バイナリ表記πは無限です。しかし、無限の数の分数がこの数に近づくと、ますます精度が高まります。

ナンバーラインの深いディップは、見かけほど難攻不落ではありません。これは、複雑な数値が単純な近似にどのように役立つかを示す新しい意味のある証拠の結果の1つです。

証明は、約80年前の問題を解決し、Duffin-Shaffer仮説として知られています。したがって、それは古代から数学者を占めてきた最終的な答えを与えます：どのような条件下で無限に続く無理数を表すことができます-piなどの22/7タイプの単純な分数で？証明は、このかなり一般的な質問への答えが単一の計算の結果として見つかることを証明します。

モントリオール大学のDimitris Cuculopoulosとの共同執筆者であるオックスフォード大学のJames Maynard氏は、「ほぼすべての数を概算できるか、実質的に数を概算できないかについては簡単な基準があります」と語った。

数学者は数十年前から疑っていましたが、この単純な基準は、適切な近似が得られる時期を理解するための鍵となりますが、それを証明することはできませんでした。 KukulopoulosとMaynardがこれを行うことができたのは、グラフの点と線の間の接続の観点から、この数値の問題を再定式化した後でのみでした。

「私は彼らが選択された道を進むのに十分に自信があった（そしてそれは明らかに正当化された）と思います」とDuffin-Schaffer仮説の初期の結果に関与したテキサス大学オースティンのJeffrey Waalerは言いました。- よくやった"。

算術エーテル


有理数、すべてが簡単です。それらには、オブジェクトを数えるための数と、分数で書くことができる他のすべての数が含まれます。

この筆記能力により、有理数は最もよく知られています。しかし、すべての実数の中で、有理数は実際には非常に少数です。ほとんどの数値は不合理で、無限小数表記であり、分数で書くことはできません。それらのいくつかは、記号表記に値するのに十分重要であることが判明しました-パイ、e、√2。残りは呼び出すことさえできません。それらは至る所にありますが、到達不可能です-算術エーテルのようです。

したがって、おそらく、それは考えるのは自然なことです。もし私たちが無理数を正確に表現できない場合、どれだけ近づけることができるでしょうか？これは合理的な近似の領域です。古代の数学者は、円周と直径のとらえどころのない比率は、分数22/7を使用してかなりよく近似できることに気づきました。その後、数学者たちは、piへのさらに正確でほぼ同じ圧縮された近似値355/113を発見しました。

「パイを書くのはとても難しい」とオックスフォードのベン・グリーンは言った。 「人々はpiの最も正確な近似を見つけようとしましたが、これを行う一般的な方法の1つは有理数を使用することでした。」



1837年、数学者ピーターグスタフレジューヌディリクレは、無理数が有理数を使用してどの程度正確に近似できるかを教えてくれるルールを発見しました。エラーの正確な値を設定しない場合、近似は簡単に見つけることができます。しかし、ディリクレは、分数、無理数、およびそれらを分離するエラーの間に明確な関係の存在を証明しました。

彼はどんな不合理な数でも無限に多くの分数が彼に近づいて近づいていることを証明しました。より正確には、各分数の誤差は、分母の二乗で除算された1以下になります。たとえば、分数22/7はおよそ1/7 2または1/49の精度でpiを表します。分数355/113はpiから1/113 2以下です、または1/12769。分母が増加するにつれて、Dirichletは無限に多くの分数があり、piにますます近づくことを証明しました。モントリオール大学のアンドリューグランビル氏は

、「驚くべきことは、実数を分数で大まかに表現できることです。誤差は1分母の2乗で除算されたものを超えません。」1913年の原稿では、数学者のSrinivasa Ramanujan Iyengorが分数355/113をpiの有理近似として使用しました。




ディリクレの発見は、合理的な近似に関する限定された声明でした。それは分母として任意の整数を使用でき、その逆二乗のサイズのエラーに満足している場合、任意の無理数について、それに近づく無限に多くの分数を見つけることができると言います。しかし、分母が整数の一部の（無限）サブセット、たとえば素数のセットや完全な二乗のセットに属する必要がある場合はどうでしょうか。近似誤差を0.00001にしたい場合、または他の値にしたい場合はどうなりますか？そのような状況で正確に無限に多くの近似分数を見つけることができるでしょうか？

Duffin-Shaffer仮説は、有理近似を処理するための最も一般化されたプラットフォームを作成する試みです。 1941年、数学者R.D.ダフィンとA.S. Schaferは次のシナリオを提示しました。まず、分母の無限のリストを選択します。奇数、10で割り切れる数、素数です。

次に、リスト内の各数値について、無理数を概算する必要がある精度を選択します。直感は、十分に大きな誤差を選択すると、近似の機会が増えることを示しています。小さなエラーサイズを選択すると、より困難になります。 「十分なスペースが残っていれば、どのシーケンスでも十分です」とKukulopoulos氏は述べています。

ここで、選択したパラメーター-一連の数値と特定のエラー-が発生すると、すべての無理数に近似する無限に多くの分数を見つけることが可能ですか？

仮説は、この質問を評価するための数学関数を提供します。パラメータは入力として機能します。結果は2つのオプションのいずれかになります。 DuffinとSchafferは、これらの2つのオプションは、シーケンスがほぼすべての無理数を必要な精度で概算できるか、または実際にはどれにも当てはまらないかどうかに正確に対応することを示唆しました（分母のセットには常に小さい数があるため十分に概算できる、または概算できない孤立した非合理的な数）。

「ほとんどすべて、またはほとんど何も得られません。中間的な選択肢はありません」とメイナードは言いました。

これは非常に一般的な声明であり、それに沿って、また全体にわたって有理数による近似を特徴付けることを試みました。DuffinとSchafferによって提案された基準は、数学者には正しいように思われました。ただし、関数のバイナリ出力に、近似が機能するかどうかを理解するために必要なすべてのものが含まれていることを証明するには、はるかに困難でした。

ダブルカウント


Duffin-Schaffer仮説の証明は、利用可能な各分母から得られる利点を理解することです。これを感じるには、このタスクのより小さなバージョンを検討することが有用です。

0から1までのセグメント上のすべての無理数を概算したいとします。1から10までのすべての自然数が分母として利用できると想像してください。可能な分数のリストは非常に大きいです。最初の1/1、次に1/2と2/2、次に1 / 3、2 / 3と3/3など、最大9/10と10/10まで。ただし、それらの利点はありません。

たとえば、2/10は1/5と同じで、5/10は1 / 2、2 / 4、3 / 6、4 / 8と同じです。 Duffin-Schaffer仮説が出現する前に、ソビエトの数学者Alexander Y. Khinchinは、幅が似ている有理近似の仮説を立てました。しかし、彼の定理は同等の分数は一度だけ数えられるべきであるという事実を考慮していませんでした。


グランビル氏は、ディミトリスククロプロス氏（左）とジェームズメイナード氏が、イタリアで開催された会議での証明の発表で

述べた。 「しかし、この場合、驚くべきことに、彼女は影響を与えました。」

したがって、Duffin-Shaffer仮説には、各分母の一意の分数（または簡約された分数）の数を数える項があります。この項はオイラーφ関数と呼ばれます。その発明者である18世紀の数学者レオンハルトオイラーに敬意を表して。 φ（10）は4です。これは、0と1の間に、分母が10である4つの分数のみがあるためです。1/ 10、3 / 10、7 / 10、9 / 10。

次のステップは、与えられた各分数を使用して近似できる無理数の数を計算することです。許容できるサイズエラーによって異なります。 Duffin-Shaffer仮説では、分母ごとにエラーを選択できます。たとえば、分母が7の分数の場合、許容誤差は0.02になります。分母10の場合、エラー0.01をとることができます。

エラーの分母と用語を特定したら、今度はネットワークを無理数に置く時が来ました。 0と1の間の数直線上に分数を作成し、両側の分数から伸びるネットワークの形でエラーを描画します。ネットワークに含まれるすべての無理数は、与えられた項について「十分に近似されている」と言えます。問題は次のとおりです。あなたは何個の不合理な数字を見つけましたか？



数行の任意の間隔で無数の無数の数が含まれているため、キャッチされた無数の正確な数を書き込むことはできません。代わりに、数学者は、各分数によって捕らえられた不合理な数の総数の割合について話します。彼らは、数のサブセットの「測度」などの概念を使用してこれらの比率を評価します。これは、量ではなく重量で捕獲された魚の数を推定するようなものです。

Duffin-Schaffer仮説は、各近似フラクションがキャッチした無理数のサブセットのすべての測定値を合計することを示唆しています。彼女はこの数を大きな算術合計として表します。次に、彼女は彼女の主な予測を行います。この合計が無限大になる場合、ほとんどすべての無理数を概算しました。合計したメジャーの数に関係なく、それが有限の値しか与えない場合、ほとんどすべての無理数を概算することができませんでした。

数学の多くの分野で、合計が無限に「発散」するのか、有限値に「収束」するのかという同様の問題が生じます。Duffin-Shaffer仮説の主要なステートメントは、分母とエラーの特定のセットを使用してほぼすべての無理数を近似できるかどうかを理解したい場合、知っておく必要があるのは、測定値の無限大が無限大に発散するか、収束するかです。最終的な値に。

「結局のところ、各分母の近似をどのように評価するかを決定しても、成功は完全に1つだけに依存します。無限シーケンスが発散するかどうかに関係します」とワーラー氏は述べています。

意思決定


疑問に思われるかもしれませんが、1つの分数で近似された数値が別の分数で近似された数値と交差する場合はどうでしょうか。メジャーを計算するときにそれらを2回考慮することはできますか？

一部の近似シーケンスでは、二重カウントは重要ではありません。数十年前の数学者は、この仮説が素数で構成される近似列に当てはまることを証明しました。しかし、他の多くの近似シーケンスでは、二重カウントが問題になります。したがって、数学者はこの仮説に80年間対処できませんでした。

異なる分母が交差する無理数のセットをキャッチする度合いは、すべての分母に共通の単純な除数の数に反映されます。 12と35の数値を考えてみます。12の素数は2と3です。35の素数は5と7です。つまり、12と35の一般的な素数はそうではありません。その結果、分母12と35は特に交差していません。

12と20の分母はどうですか？ 20では、素数は2と5であり、約数12と交差します。同様に、分母が20の分数で近似できる無理数は、分母が12の分数で近似できる無理数と交差します。Duffin-Shaffer仮説は証明が最も難しいこのような状況-近似シーケンスの数値に多くの共通の小さな素数除数があり、各分母が近似する数値の多くのサブセットの共通部分が発生する場合。

「あなたが選択する分母の多くに小さな単純な除数がたくさんあるとき、それらは互いに干渉し始めます」とオックスフォードのサム・チャウは言いました。

仮説を解く鍵は、共通の素数を持つ分母によって近似される無理数のサブセットの重複を正確に計算する方法を見つけることでした。 80年間、誰もこれを行うことができませんでした。 CuculopoulosとMaynardは成功し、この課題に関するまったく新しい視点を見つけました。


相互オーバーレイ数

新しい証明では、分母からグラフを作成します。それらはグラフの頂点としてそれらを構築し、それらに共通の素数が多い場合は頂点をエッジに接続します。グラフの構造は、各分母が近似する無理数のサブセットの重ね合わせを表します。そしてこのオーバーレイを直接調査することは困難ですが、KukulopoulosとMaynardはグラフ理論のツールを使用してグラフの構造を分析する方法を見つけました-そして彼らが必要とする情報はこのようにして見つけられました。

「グラフはタスクを視覚的に理解するのに役立ちます。それは問題を反映することができる美しい言語です」とKukulopulosは言いました。

CuculopoulosとMaynardは、Duffin-Shaffer仮説が実際に真実であることを証明しました：有効なエラーのある分母のリストが与えられた場合、ほぼすべての無理数を概算することが可能であるか、または対応するメジャーの合計が発散するかどうかを確認するだけでは実行できないかどうかを判断できます無限に、または有限値に収束します。

これはエレガントなテストであり、有理数による近似の性質について広範な質問を取り、それを単一の計算値に削減します。テストの普遍性を証明したククロプロスとメイナードは、数学にとって最大のことの1つを行いました。彼らは、それぞれの分野の根本的な質問に対する最終的な答えを出しました。

「彼らの証明は必要かつ十分な結果となっています」とグリーンは言いました。「彼は数学の次の章の終わりを示していると思います。」
