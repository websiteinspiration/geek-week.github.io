Comment simplifier la gestion des appareils domestiques et les sécuriser (partager des idées sur Kauri Safe Smart Home)

Nous sommes spécialisés dans l'utilisation des données - nous développons et mettons en œuvre des solutions Internet des objets (IoT) qui fonctionnent pour tous les secteurs d'activité. Mais récemment, nous avons tourné notre attention vers un nouveau produit destiné principalement à une maison ou un bureau «intelligent».

Maintenant, le résident moyen d'une métropole dans l'appartement dispose d'un routeur Wi-Fi, d'un décodeur d'un fournisseur Internet ou d'un lecteur multimédia, un hub pour les appareils IoT.

Nous avons pensé que tous ces appareils peuvent non seulement être combinés en un seul appareil, mais également sécuriser complètement le réseau domestique. Autrement dit, il s'agit d'un appareil qui combine un routeur, un pare-feu intelligent avec un antivirus, un routeur Zigbee (en option - traitement local des données et prise de décision, y compris l'exécution de script). Et, bien sûr, il fonctionne avec une application mobile de gestion et de surveillance. Une configuration de maison intelligente est possible par les spécialistes du fournisseur. L'appareil fonctionnera avec Alice, donc personne n'a annulé les discothèques et les jeux à domicile dans les villes.



Ainsi, selon la modification, l'appareil peut être:

a) Anti-Virus;
b) Point d'accès Wifi avec antivirus;
c) Point d'accès Wifi / Zigbee avec antivirus,
gérant éventuellement UD;
d) Routeur Wifi / Zigbee / Ethernet avec antivirus,
gérant éventuellement UD.

Malheureusement, il n'existe aucun système IoT complètement sécurisé. D'une manière ou d'une autre, ils sont tous vulnérables. Selon Kaspersky, au cours du premier semestre de 2019, les pirates informatiques ont attaqué les appareils Internet des objets plus de 100 millions de fois - le plus souvent en utilisant les réseaux de zombies Mirai et Nyadrop. Nous comprenons que la sécurité est un casse-tête pour l'utilisateur, c'est pourquoi notre Kauri Hub agit comme un antivirus. Il analyse tout le trafic sur le réseau pour détecter toute activité malveillante. Dès que l'appareil détecte une anomalie, il bloque toutes les tentatives d'accès aux gadgets sur le réseau depuis l'extérieur. Dans le même temps, le fonctionnement de l'antivirus n'affecte pas la vitesse d'Internet, mais absolument tous les appareils connectés seront protégés.

Anticiper quelques objections:

- Je peux moi-même l'assembler sur un routeur avec Zigbee USB et OpenWrt.

Oui, geek. Et si vous voulez bricoler - pourquoi pas? Et
vous écrirez certainement des applications pour le smartphone aussi. Mais y en a-t-il beaucoup comme toi?

- Les moissonneuses-batteuses ne font bien aucune fonction.

Pas certainement de cette façon. Le traitement des protocoles réseau est facilement combiné en un seul appareil. Les routeurs domestiques modernes combinent déjà de nombreuses fonctionnalités, nous en ajoutons juste quelques autres.

- Zigbee n'est pas sûr.

Oui, si vous utilisez les capteurs les moins chers avec une clé cousue par défaut. Nous vous suggérons d'utiliser la norme Zigbee 3.0 plus sécurisée. Mais les capteurs seront plus chers.

La rétroaction est très importante pour nous! Le projet Kauri Safe Smart Home est actuellement en cours de développement. Nous espérons qu'il sera utile non seulement pour les tâches domestiques, mais aussi pour le bureau. À cet égard, nous avons plusieurs questions pour les lecteurs:

  1. Êtes-vous intéressé par un tel appareil?
  2. Pour quel montant minimum seriez-vous intéressé à l'acquérir?
