クラスを備えた機能的なPowershell-oxymoronではなく、保証します

こんにちは、ハブル！クリストファークエックによる 「クラスを使用した関数型PowerShell。
それが矛盾することではないことを約束します」という記事の翻訳を紹介します。


オブジェクト指向と関数型プログラミングのパラダイムは互いに矛盾しているように見えるかもしれませんが、Powershellではどちらも同等にサポートされています。関数型であるかどうかに関係なく、ほとんどすべてのプログラミング言語には、名前と値の拡張バインディングの手段があります。構造体やレコードなどのクラスは、1つのアプローチにすぎません。クラスの使用を名前と値のバインディングに制限し、継承、ポリモーフィズム、または可変性などの「重い」オブジェクト指向プログラミングの概念を回避すると、コードを複雑にすることなくそれらを利用できます。さらに、型変換の不変のメソッドを追加して、機能コードをクラスで充実させることができます。


カーストマジック


カーストは、Powershellの最も強力な機能の1つです。値をキャストするときは、環境の機能に依存して、暗黙的な初期化と検証をアプリケーションに追加します。たとえば、[xml]内の文字列の単純なキャストは、パーサーコードを介して実行し、完全なxmlツリーを生成します。同じ目的でコード内でクラスを使用できます。


カスタムハッシュテーブル


コンストラクタがない場合は、ハッシュテーブルをクラスの型にキャストすることで、コンストラクタなしで続行できます。このパターンを最大限に活用するには、必ず検証属性を使用してください。同時に、クラスの型付きプロパティを使用して、より深い初期化および検証ロジックをトリガーできます。

class Node {
    [ValidateLength(3, 7)]
    [string] $Name
    [ValidateSet("INT", "PPE", "PROD")]
    [string] $FlightingRing
    [ValidateSet("EastUS", "WestUS", "NorthEurope", "WestEurope")]
    [string] $Region
    Node([string] $Name) {
        $Name -match "([a-z]+)(INT|PPE|PROD)([a-z]+)"
        $_, $this.Service, $this.FlightingRing, $this.Region = $Matches
        $this.Name = $Name
    }
}

class Datum {
    [string] $Name
    [int] $Value
    [Node] $Computer
    [int] Severity() {
        $this.Name -match "[0-9]+$"
        return $Matches[0]
    }
}

class Cluster {
    [ValidatePattern("^[A-z]+$")]
    [string] $Service
    [ValidateSet("TEST", "STAGE", "CANARY", "PROD")]
    [string] $FlightingRing
    [ValidateSet("EastUS", "WestUS", "NorthEurope")]
    [string] $Region
    [ValidateRange(0, 255)]
    [int] $Index
}

function New-Cluster {
    [OutputType([Cluster])]
    Param(
        [Parameter(Mandatory, ParameterSetName = "Id", Position = 0)]
        [ValidateNotNullOrEmpty()]
        [string] $Id,
        [Parameter(Mandatory, ParameterSetName = "Components")]
        [string] $Service,
        [Parameter(Mandatory, ParameterSetName = "Components")]
        [string] $FlightingRing,
        [Parameter(Mandatory, ParameterSetName = "Components")]
        [string] $Region,
        [Parameter(Mandatory, ParameterSetName = "Components")]
        [int] $Index
    )

    if ($Id) {
        $Service, $FlightingRing, $Region, $Index = $Id -split "-"
    }

    [Cluster]@{
        Service       = $Service
        FlightingRing = $FlightingRing
        Region        = $Region
        Index         = $Index
    }
}

Export-ModuleMember New-Cluster
