自分で行うか、Snom電話をカスタマイズする方法。パート2アイコンと画像

記事の最初の部分で約束したように、この続きはSnom電話のアイコンを個別に変更するためのものです。



それでは始めましょう。ステップ1、ファームウェアをtar.gz形式で取得する必要があります。こちらのリソースからダウンロードできます。すべてのsnomアイコンが利用可能で、ファームウェアの各バージョンに含まれています。



注：各ファームウェアバージョンには、このバージョンと電話モデルに固有の設定ファイルが含まれていることに注意してください。ファームウェアまたは電話と一致しない設定ファイルを使用すると、問題が発生します。


customizing.tar.gzファイルをダウンロードして開くと、次のようになります。ファイルの実際の内容は、電話のバージョンとファームウェアによって異なります。







ステップ2、電話のアイコンを準備します。ご存知のように、Snom電話にはそれぞれカラー画面とモノクロ画面が付属しており、アイコンは異なります。



I.カラーディスプレイ電話のアイコンを変更する



カラーディスプレイを備えた電話のアイコンと画像は、PNG形式で保存されます。これにより、ほとんどすべての最新の画像エディターで簡単に編集できます。ただし、編集後は、optipng、pngquant、pngcrushなどのツールを使用してpngファイルを最適化し、冗長な情報を削除してファイルサイズを最適化することをお勧めします。



アイコン画像サイズ：



  • 状況依存の鍵アイコン24x24px
  • SmartLabel 24x24pxおよび18x18px
  • タイトルバーのアイコン（タイトルバーアイコン）18x18px
  • メニューアイコン18x18px
  • 通話中（通話画面アイコン）18x18px-48x48px
  • ファイル形式：PNG


目的のアイコンを作成したら、それらを携帯電話にダウンロードします。ダウンロードする方法は2つあります。



  1. 手動モードのWebインターフェース経由
  2. 自動プロモーションの使用


最初のオプションから始めましょう -Webインターフェイスからダウンロードします。ダウンロードするには、Preference / Appearanceタブの電話のWebインターフェイスに移動し、Custom Imagesを選択します。







次に、変更するアイコンを見つけて、独自のバージョンをロードし







ます。独自のバージョンや「カーブ」が気に入らない場合は、「Reset」ボタンをクリックすることでいつでもロールバックできます」







注。「ソフトウェアの更新」と「工場出荷時の設定に戻す」では、ダウンロードした画像は削除されません。


手動モードで確認できるように、すべては非常に単純ですが、複数の電話を変更する必要がある場合、このプロセスには多くの時間がかかります。したがって、2番目のオプションに進みます。



オプション 2-自動プロビジョニングを介してダウンロードします。



まず、以前にダウンロードしたcustomizing.tar.gzアーカイブからtar形式のアーカイブを作成する必要があります。アーカイブを作成するときは、変更する必要のないすべてのファイルとディレクトリを削除しますが、ディレクトリ構造は維持してください。

注。元々アーカイブにあったすべてのファイルをアーカイブする必要はありません。変更したファイルのみをアーカイブすることで十分です。アーカイブに入れるファイルが多いほど、電話機がそのファイルを構成するのに時間がかかります。


次に、いくつかの手順を実行します



。1）たとえば、branding.xmlなどのXMLファイルを作成し、それをWebサーバー（HTTP）にコピーします。http：//yourwebserver/branding.xml： 



<?xml version="1.0" encoding="utf-8" ?>
<settings>
 <uploads>
  <file url="http://yourwebserver/branding/branding.tar" type="gui" />
</uploads>
</settings>


2）[詳細設定]-> [更新]-> [URLの設定]セクションで電話のWebインターフェースに移動し、ファイルyourwebserver / branding.xmlへのリンクを提供します















3）電話を再起動して結果をお楽しみください。

II.



モノクロデバイスのアイコンは、.pngや.jpgなどの通常の画像ファイルには保存されませんが、ユーザーインターフェイスで使用されるすべてのアイコンを含むビットマップフォントです。U + EB00で始まるUnicodeテーブルの私的使用の領域では、snomアイコンが定義されており、Font Forgeなどのツールを使用して直接変更できます。



Font Forgeでビットマップフォントファイルを開くと、使用されているアイコンのリストが表示されます。ファイルの実際の内容は、電話のバージョンとファームウェアによって異なり







ます。モノクロ表示の電話のアイコン仕様。



モデルD305、D315、D345、D385、D745、D785、D3、D7の場合：



  • 状況依存のキーアイコン17x17-ベースラインx→0 / y→-2
  • タイトルバーアイコン17x17-ベースラインx→0 / y→-2
  • ラベルパネルアイコン17x17-ベースラインx→0 / y→-2
  • アイコンの最大サイズ32x32


モデルD120、D710、D712、D715、D725の場合：



  • 状況依存キーアイコン7x7-ベースラインx→0 / y→0
  • タイトルバーアイコン7x7-ベースラインx→0 / y→0
  • SmartLabelアイコン7x7-ベースラインx→0 / y→0
  • アイコンの最大サイズ32x32


必要な「イメージ」を作成し、それをFont Forgeからエクスポートした







後、次の設定を使用する必要があります。



tar -cvf fonts.tar fontfile.bdf



実際には画像ではなくフォントを変更しているため、xml-file設定で指定して、自動プロモーションを通じてフォントとしてロードします。 



<?xml version="1.0" encoding="utf-8" ?>
<settings>
 <uploads>
  <file url="http://192.168.23.54:8080/fonts.tar" type="font" />
 </uploads>
</settings>
