LEGO MINDSTORMS Education EV3 + MicroPython：子どものコンストラクタを成人向け言語でプログラミングします

こんにちは、ハブル！我々はすでに話をした LEGO MINDSTORMS EV3教育プラットフォームについて。このプラットフォームの主な目的は、実践的な例に関するトレーニング、STEAMスキルの開発、エンジニアリング思考の形成です。それは力学と力学を研究するために実験室の仕事を行うことができます。LEGOブロックとデータを記録および処理するためのユーティリティで構成された実験室のスタンドは、実験をさらに面白く視覚的にし、子供たちが物理学をよりよく理解するのに役立ちます。たとえば、学童は融解温度に関するデータを収集し、アプリケーションを使用してそれらを整理し、グラフに表示できます。しかし、これはほんの始まりにすぎません。今日は、このセットをMicroPythonプログラミング環境で補完し、それを使用してロボット工学を教える方法を説明します。



EV3でプログラミングを学ぶ


現代の学生はカラフルな結果を見たいと思っています。はい、プログラムがコンソールに数字を表示する場合は退屈しており、カラーチャートや図を見て、移動してコマンドを実行する実際のロボットを作成したいと考えています。通常のコードは子供にとっても複雑すぎるようですので、何かを簡単に学習することをお勧めします。

基本的なプログラミング環境EV3はグラフィック言語LabVIEWに基づいており、ロボットのアルゴリズムを視覚的に設定できます。コマンドは、ドラッグして結合できるブロックの形式で表示されます。



この方法は、アルゴリズムの構築方法を示す必要がある場合に適切に機能しますが、ブロック数が多いプログラムには適していません。スクリプトを複雑にする場合、コードを使用したプログラミングに切り替える必要がありますが、子供がこの手順を実行することは困難です。 

ここにはいくつかのトリックがあり、そのうちの1つは、コードがブロックと同じタスクを実行することを示すことです。EV3環境では、これはMicroPythonとの統合を通じて行うことができます。子は、Microsoft Visual Studio CodeのブロックとPythonを使用して、基本プログラミング環境で同じプログラムを作成します。どちらの方法も同じように機能することがわかりますが、コードで複雑な問題を解決する方が便利です。

MicroPythonに行く


EV3環境はARM9プロセッサに基づいており、開発者は意図的にアーキテクチャを開いたままにしました。このソリューションにより、代替ファームウェアをロールアップすることができました。その1つは、MicroPythonを操作するためのイメージでした。Pythonを使用してEV3をプログラミングできるため、タイピングを実際のタスクにさらに近づけることができます。 

開始するには、EV3 MicroPythonイメージを任意のmicroSDカードにダウンロードし、EV3マイクロコンピューターにインストールしてオンにする必要があります。次に、Visual Studioの無料の拡張機能をインストールする必要があります。そして、あなたは仕事に取り掛かることができます。 

MycroPythonで最初のロボットをプログラムします





私たちのサイトには、ロボット工学の基本的な概念を習得するためのいくつかのレッスンがあります。EV3のモデルは、自動運転車、工場で組み立てられたロボット、CNCマシンで使用される基本を子供たちに紹介します。 

たとえば、パターンや幾何学的形状を描くことを教えることができるドローイングマシンを取り上げます。このケースは、大人の溶接ロボットまたはフライス盤の簡略版であり、EV3をMicroPythonと組み合わせて使用​​して学生に教える方法を示しています。また、描画機はプリント基板のパパ用の穴に印を付けることができますが、これは数学的な計算を必要とする別のレベルです。

まず、EV3モジュールライブラリを初期化します。

#!/usr/bin/env pybricks-micropython
from pybricks import ev3brick as brick
from pybricks.ev3devices import (Motor, TouchSensor, ColorSensor, GyroSensor)
from pybricks.parameters import Port, Stop, Direction, Color, ImageFile
from pybricks.tools import wait

モーターのようにハンドルを回転させるプラットフォームをポートBに設定しました。2段変速機のギア比をそれぞれ歯数20〜12〜28に設定しました。

turntable_motor = Motor(Port.B, Direction.CLOCKWISE, [20, 12, 28])

ハンドルのリフト機構をポートCのモーターとして構成します。

seesaw_motor = Motor(Port.C)

ポート2に、ハンドルの角度を測定するジャイロスコープをセットアップします。

gyro_sensor = GyroSensor(Port.S2)

ポート3でカラーセンサーを構成します。センサーは、ドローイングマシンの下で白い紙を検出するために使用されます。

color_sensor = ColorSensor(Port.S3)

ポート4でタッチセンサーを構成します。センサーを押すと、ロボットが描画を開始します。

touch_sensor = TouchSensor(Port.S4)
