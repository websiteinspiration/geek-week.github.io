マークアップ。

最初にすることは、文字「＆」、「<」、「>」をエスケープすることです

s = s.replace('&', '&amp;').replace('<', '&lt;').replace('>', '&gt;')

「＆」はエスケープされ、文字の組み合わせ「＆lt;」、「＆gt;」が表示されます その他の「＆...;」、

「<」および「>」の文字は、タグと競合しないようにエスケープされます。

多くのことをシールドできますが、私の意見では、utf-8ではこれで十分です。

マークアップアルゴリズム：

  • 再テンプレートを取り、それを満たすすべてのテキストフラグメントを探します。
  • 各フラグメントを切り取り、マークアップを追加して、マークされたフラグメントを配列に保存します（同時に、元のテキストを保存します。これは便利です）。
  • その場所に、番号が付いたスタブを挿入します。
  • そして、すべての色について。
  • すべてがペイントされたら、スタブを配列の色付きのフラグメントで置き換えます。

スタブは一意である必要がありますが、このテキストには単一の文字「<」および単一の「>」はありません。

スタブを作る：

f'<{i}>'

ここで、iはスタブ番号です。テキストにはそのようなことは絶対にありません。

フラグメント内にすでにスタブがある場合の対処方法。

たとえば、ソーステキストには次の行が含まれています。

`  :   /*     */ `
/* , `     ` */

オプション2：

  1. 無視します。この場合、文字列は2色になります。
  2. ネストされたスタブを見つけ、それらを元の（マークされていない）テキストに置き換えます。

これらすべてを実装する関数を作成します（10行のコード、10行のコメント）。

def oneRe(reStr, s, attr, ls, multiColor=False):
    '''
        s  ,          ls,
           (<0>, <1> ...<1528>...).
          .
    
    reStr - re
    s - 
    attr -  style/class/etc
    ls -   
    multiColor=False,     
    '''
    for block in set(re.findall(reStr, s)):
        toArr = block
        if not multiColor: #  multiColor==False,   
            for prev in set(re.findall(r'<[\d]+>', block)): #  : <0> ... <21>
                iPrev = int(prev[1:-1], 10)                #     
                toArr = toArr.replace(prev, ls[iPrev][1])  #   '<0> qwe <21>'  (<0>,<21>)  . 
        ls.append([f'<span {attr}>{toArr}</span>', toArr]) #     ls 2 :     
        s = s.replace(block, f'<{len(ls)-1}>')
    return s

まず、複数

文字列を見つけます。re.findall（r '' [\ s \ S] +？ ''、s）は、バッククォート文字間のテキストブロックのリストを返します。

各ブロックにはスペースまたは非スペース（[\ s \ S]、つまり何でも）が含まれています。

ブロック長1以上（+）。

貪欲なし（「？」は、ブロック内に記号「 `」がないことを意味します）。

見つかったブロック（コピーMSTRの変数をする）newFstrの変数。
$ {...}という式を持つブロックサブブロックを見つけます。

re.findall（r "\ $ \ {[\ s \ S] *？\}"、mStr）は、そのようなサブブロックのリストを返します。マークアップをls配列に保存し、サブブロックをnewFstr変数のスタブで置き換えます。
サブブロックがなくなると、元の文字列sに置き換えます元のブロック値を新しいものに。

セットは不必要ではありません。findallがいくつかの同一のブロックを返す場合、ソースコードの最初のブロックを処理すると、すべての同一のブロックが一度にスタブに置き換えられます。2番目の同じブロックを処理するとき、それはもはやソーステキストにはありません。セットは重複を削除します。

def oneRe(reStr, s, attr, ls, multiColor=False):
    '''
        s  ,          ls,
           (<0>, <1> ...<1528>...).
          .
    
    reStr - re
    s - 
    attr -  style/class/etc
    ls -   
    multiColor=False,     
    '''
    i = len(ls) #   
    for block in set(re.findall(reStr, s)):
        toArr = block
        if not multiColor: #  multiColor==False,   
            for prev in set(re.findall(r'<[\d]+>', block)): #  : <0> ... <21>
                iPrev = int(prev[1:-1], 10)                 #     
                toArr = toArr.replace(prev, ls[iPrev][1])   #   '<0> qwe <21>'  (<0>,<21>)  . 
        ls.append([f'<span {attr}>{toArr}</span>', toArr])  #     ls 2 :     
        s = s.replace(block, f'<{i}>')              #       
        i += 1
    return s

def operColor(s, ls, color):
    '''

      ,       ,  
    '''
    i = len(ls)
    for c in ['&lt;=', '&gt;=', '=&lt;', '=&gt;', '&lt;', '&gt;', '&amp;&amp;', '&amp;',
              '===', '!==', '==', '!=', '+=', '-=', '++', '--', '||']:
        ls.append([f'<span {color}>{c}</span>',0])
        s = s.replace(c, f'<{i}>')
        i += 1

        ls.append([f'<span {color}>{c}</span>',0])
        s = s.replace(c, f'<{i}>')
        i += 1

インポート：

from xml.dom.minidom import parseString

htmlテキストをxhtmlテキストに変換します。ほぼ同じですが、すべてのタグが閉じています。

doc = html.fromstring(htmlText)
ht = etree.tostring(doc, encoding='utf-8').decode()

ミニハウスを使用してツリーハウスに結果として得られるxhtmlパーシム。

dom = parseString(ht)

ノードを再帰的にジャンプして、Reactソースコードの形式で結果を生成する関数を作成します。

parseStringを呼び出した後、domツリーは、子を持つ子ノードを持つ父親ノードなどになります。

各ノードはその説明を含む辞書です：

  • nodeName-ノード名、文字列
  • childNodes-子ノード、リスト
  • 属性-属性、辞書
  • #textというノードにはnodeValue（文字列）があります

例：

<div class="A" style="color: red;">Red of course,<br> </div>

変換後、次のようになります。
{ 'nodeName':'div',
  'attributes': {'style': 'color: red;', 'class': 'A'},
  'childNodes': [
    {'nodeName':'#text', 'nodeValue': 'Red of course,'},
    {'nodeName':'br'},
    {'nodeName':'#text', 'nodeValue': ''},
  ],
}

domを文字列に変換するのは簡単です（pprintがあります）。Reactコードを生成するときに、クラスをclassNameに置き換えて、スタイル属性を再指定しました。

テキストノードでは、「{」、「}」、「<」、「>」がエスケープされます。

HtmlToReact.pyファイル 
# -*- coding: utf-8 -*- 
# -*- coding: utf-8 -*- 

from xml.dom.minidom import parseString
from lxml import html, etree

# *** *** ***

_react = ''

def htmlToReact(buf):
    '''
    buf - html-
     ReactJS- 
    '''
    global _react
    _react = ''

    try:
        r = re.search('<[\\s\\S]+>', buf)
        if r:
            doc = html.fromstring(r.group(0))
            ht = etree.tostring(doc, encoding='utf-8').decode()
            xHtmlToReact(parseString(ht).childNodes[0], '')
            return _react
        else:
            return '<empty/>'
    except Exception as ex:
        s = f'htmlToReact: \n{ex}'
        print(s)
        return s

# *** *** ***

def sU(a, c):
    '''
    xlink:show   ->  xlinkShow
    font-weight  ->  fontWeight
    '''
    l, _, r = a.partition(c)
    return ( (l + r[0].upper() + r[1:]) if r else a).strip()

# *** *** ***

def xHtmlToReact(n, shift='\n    '):
    '''
       -
      ,        upperCase
       global  _react
    '''
    global _react

    if n.nodeName.lower() in ['head', 'script']:
        return
    
    _react += shift + '<' + n.nodeName.lower()
    if n.attributes:
        for k, v in n.attributes.items():
            if k == 'style':
                style = ''
                for s in v.split(';'):
                    if s.strip():
                        l, _, r = s.partition(':')
                        style += f'''{sU(l, '-')}: "{r.strip()}", '''
                if style:
                    _react += ' style={{' + style + '}}'
            elif k == 'class':
                _react += f' className="{v}"'
            else:
                kk = k.replace('xlink:href', 'href') # deprcated
                _react += f''' {sU( sU(kk, ':'), '-' )}="{v}"'''
        
    _react += '>'
    if n.childNodes:
        for child in n.childNodes:
            if  child.nodeName == '#text':
                tx = child.nodeValue
                for x in ['{', '}', '<', '>']:
                    tx = tx.replace(x, '{"' + x + '"_{_')
                tx = tx.replace('_{_', '}')
                if tx[-1] == ' ':
                    tx = tx[:-1] + '\xa0'
                _react += tx.replace('\n', '<br/>')
            else:
                xHtmlToReact(child)
                
    _react += f'{shift}</{n.nodeName.lower()}>'

# *** *** ***
