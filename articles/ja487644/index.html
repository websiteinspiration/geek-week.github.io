Pythonを使用した株価情報の取得

こんにちは、ハブル！Ishan Shahによる記事「Pythonの過去の株価データ」の翻訳を紹介します。

yfinanceを使用して毎日の過去の株価データを取得する方法と、alpha vantageを使用して分データを取得する方法に関する記事。


ご存知のように、株式は非常に不安定な商品であり、取引を決定する前に価格の動向を注意深く分析することが非常に重要です。さて、最初にデータを取得する必要があり、Pythonがこれを支援できます。

Exchangeデータは、さまざまなパッケージを使用してダウンロードできます。この記事では、yahooファイナンスとアルファバンテージについて説明します。

Yahooファイナンス


まず、yfiananceパッケージを試してください。これは、pip install yfinanceコマンドを使用してインストールできます。以下のコードは、2016年から2019年までのAAPLのデータを取得し、調整済み終値（配当と分割の調整済み終値）をチャートに作成する方法を示しています。

# Import the yfinance. If you get module not found error the run !pip install yfianance from your Jupyter notebook
import yfinance as yf

# Get the data for the stock AAPL
data = yf.download('AAPL','2016-01-01','2019-08-01')

# Import the plotting library
import matplotlib.pyplot as plt
%matplotlib inline

# Plot the close price of the AAPL
data['Adj Close'].plot()
plt.show()



まあ、いくつかの共有を取得する必要がある場合は、コードに少し追加する必要があります。DataFrameは値を格納するために使用されます。matplotlibパッケージと受信したデータを使用して、毎日の収益性チャートを作成できます。

# Define the ticker list
import pandas as pd
tickers_list = ['AAPL', 'WMT', 'IBM', 'MU', 'BA', 'AXP']

# Import pandas
data = pd.DataFrame(columns=tickers_list)

# Fetch the data

for ticker in tickers_list:
    data[ticker] = yf.download(ticker,'2016-01-01','2019-08-01')['Adj Close']

# Print first 5 rows of the data
data.head()




# Plot all the close prices
((data.pct_change()+1).cumprod()).plot(figsize=(10, 7))

# Show the legend
plt.legend()

# Define the label for the title of the figure
plt.title("Adjusted Close Price", fontsize=16)

# Define the labels for x-axis and y-axis
plt.ylabel('Price', fontsize=14)
plt.xlabel('Year', fontsize=14)

# Plot the grid lines
plt.grid(which="major", color='k', linestyle='-.', linewidth=0.5)
plt.show()



ロシア株の価値については、わずかな微妙さがあります。銘柄名にはドットと大文字のMEが追加されます。smartlabの専門家に感謝します。



Alpha vantageで微細なデータを取得


残念ながら、無料バージョンのYahoo Financeでは、毎日よりも少ない頻度でデータを受信できません。これには、1分、5分、15分、30分、60分などの間隔を取得できるAlpha vantageパッケージを使用できます。



将来的には、これらのデータを分析し、取引戦略を作成し、pyfolioパッケージを使用して有効性を評価できます。その中で、シャープレシオ、ソルティーノ係数、最大ドローダウン、その他多くの必要な指標を評価できます。
