SOLIDWORKS Simulation 2020の新機能



SOLIDWORKS Simulation 2020のモットーは、さらに生産性が高く、さらに簡単です！

SOLIDWORKS Simulation 2020は誰にとってもFEMです：複雑な計算タスクは計算機だけのものではありません


SOLIDWORKS Simulation 2020は、シミュレーションと有限要素解析の広範な採用に対する障壁を取り除きます。これらの複雑なタスクは、通常は不足している狭い専門家だけでなく、幅広い設計エンジニアが利用できるようになります。作成する必要のある物理プロトタイプが少ないほど、節約が大きくなり、開発サイクルが短くなります。より信頼性が高く、品質が高く、より安全な製品。
SOLIDWORKS Simulation 2020の新機能は、工業企業が製品をより早く市場に投入するのを支援することを目的としているため、設計者の生産性を向上させます。

タイプConnectedの連絡先に対して、オプション「Incompatible grid」がデフォルトで提供されます


利点は何ですか：作業が簡単です。SOLIDWORKS Simulation 2020の

開発者は、コンポーネント間の接触設定ウィンドウに小さな、しかし基本的な変更を加えました：接触タイプとして「接続」を選択すると、「非互換グリッド」オプションが「パラメーター」グループに自動的に提供されます（以前のバージョンでは、デフォルト値は「互換性のあるネット」この革新により、面に触れることなくアセンブリの計算グリッドの構築が大幅に簡略化されます（ほとんどのアセンブリ、実際にはそうです）ユーザーはすぐに利点を理解できます。計算の事前調整モデルははるかに速く実行されます。 もちろん、オプション「互換性のあるグリッド」はどこにも存在せず、単にデフォルトで提供されなくなっただけです。





多くの人は、デフォルトで「互換グリッド」オプションを使用すると、隣接する面がないジオメトリ上に計算グリッドを構築することが非常に困難になることを認識しています。 「非互換グリッド」オプションが選択されている場合、接触している各オブジェクト上に構築されたグリッドには、共通のノードがありません。各オブジェクトには独自の独立したグリッドがあり、接触は接触要素を使用してモデル化されます。SOLIDWORKSシミュレーション



開発者「互換性のない」グリッドを構築する精度を高めるために、可能なことはすべて実行しました。新しいアルゴリズムは、エンジニアリング分析を専門とする別のダッソーシステムズ製品であるSIMULIAの開発チームと共同で作成されました。その結果、接触するオブジェクトを高速かつ正確にモデリングするためのクラス最高のテクノロジーを手に入れました。

新機能：1つの計算モデルで近似グリッドと正確グリッドを組み合わせる


利点は何ですか：デザイナーの生産性が向上します。

各設計者は、できる限り速く、同時により速く仕事をしたいと考えています。SOLIDWORKS 2020で計算グリッドを構築するための改良されたアルゴリズムにより、1つの計算モデルで線形要素（計算問題の解決を高速化）と2次要素（高い計算精度を提供）を組み合わせることができます。これは、計算を高速化するための素晴らしい方法です。つまり、製品をより早く市場に投入することができます。



ユーザー自身が、このメッシュまたはそのソリッドボディをモデル化するメッシュ（正確または近似）を選択します。近似グリッドは、迅速な実行を保証し、2次要素のグリッド-結果の精度を向上させます（精度が本当に必要な場所で）。利点は、プロジェクトをより迅速にチェックしながら、重要なコンポーネントの正確な結果を得ることができることです。

新機能：シミュレーションエバリュエーター


利点は何ですか：計算パラメーターが最適に調整されているという確信。

計算パラメータをどのように正しく設定したか考えたことはありますか？パラメータの最適な組み合わせを探すことで、多くの貴重な時間を失う可能性があります。新しいSimulation EvaluatorモジュールがSOLIDWORKS 2020に登場
しました。これにより、すべてが正しく構成されていることを確信できます。設定した条件とパラメーターが正常な計算の実行を妨げる場合、シミュレーションエバリュエーターウィンドウで修正アクションが提供されます。これは、計算をさらに高速かつ効率的に実行したいためです。


パラメータ設定が正しくないために計算の実行に失敗した場合は、シミュレーションエバリュエーターを試してください。彼は、実行の成功が依存する多くの条件をチェックします。結果のフォルダー、指定したドライブの十分な空き領域の可用性、指定した材料、計算グリッドのサイズです。

新機能：分散タイプの接続（ピンとボルト）


利点は何ですか：より正確な複合計算結果。SOLIDWORKS Simulationはクラス最高の複合計算モジュール

であると一般に認識されています。SOLIDWORKS 2020 の分散接続タイプが再設計および改善されました。このオプションを選択すると、結合されたサーフェスが変形する可能性があります。このようにして、接続とその動作がより現実的にモデル化されます。その結果、計算モデルの精度が向上します。SOLIDWORKS Simulation の開発サイクルへの関与が早いほど、有限要素法（FEM）によるモデルの計算結果がより正確になります。
