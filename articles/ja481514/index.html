JSONオブジェクト（JObjectクラス）


JSONオブジェクトは、キーと値のペアの順不同のセットです。オブジェクトは '{'（開始中括弧）で始まり、 '}'（終了中括弧）で終わります。各名前の後には：（コロン）が続き、キーと値のペアはコンマで区切られます。



JSON C＃クラスジェネレーターによって以前に提案された構造から、通貨ペアごとに6つの定数フィールドがあることがわかります。それらを別のクラスにラップし、Orderと呼びます。

    class Order
    {
        public int trade_id { get; set; }
        public string type { get; set; }
        public double quantity { get; set; }
        public double price { get; set; }
        public double amount { get; set; }
        public int date { get; set; }
    }

交換の動的に変化するデータ構造からこれらのフィールドのセットを「取得」できるようにするには、newtosoft.jsonライブラリで利用可能なデータ型をより詳細に理解する必要があります。より具体的には、newtonsoft.json.linq名前空間。

ライブラリを使用するには、ライブラリをインストールする必要があります。

ライブラリをインストールすると、newtosoft.json.linq名前空間がプロジェクトに接続できるようになります。

using Newtonsoft.Json.Linq;

Newtosoft.json.linqは、抽象JTokenクラスを使用してデータを表し、そこからJValue（単純な値を表す）およびJContainer（構造を表す）クラスが継承されます。構造体には、JArray（配列）、JConstructor（コンストラクタ）、JObject（オブジェクト）、またはJProperty（プロパティ）を使用できます。

JArrayクラスとJObjectクラスには、JSONデータを通常の文字列から対応するオブジェクトに変換できるParseメソッドがあります。したがって、Exmoトレード関数のデータ構造にJObject.Parse（String）メソッドを使用すると、次のようになります。

HttpClient httpClient = new HttpClient();
string request = "https://api.exmo.com/v1/trades/?pair=BTC_USD,ETH_USD";
HttpResponseMessage response = 
    (await httpClient.GetAsync(request)).EnsureSuccessStatusCode();
string responseBody = await response.Content.ReadAsStringAsync();

JObject jObject = JObject.Parse(responseBody);

それぞれがキーと値のペアを表す2つの要素の構造が表示されます。この場合のキーは、通貨ペアの名前（文字列タイプ）であり、値はトランザクションのリスト（JTokenタイプ）です。



これで、トランザクションのリストは、キー「BTC_USD」および「USD_ETH」によって使用できるようになります。たとえば、通貨ペア「BTC_USD」でのトランザクション：



次に、受信したデータを作業に便利なタイプに変換するだけで済みます。次に例を示します。リスト<注文>ToObject <>（）メソッドを使用：

HttpClient httpClient = new HttpClient();
string request = "https://api.exmo.com/v1/trades/?pair=BTC_USD,ETH_USD";
HttpResponseMessage response = 
    (await httpClient.GetAsync(request)).EnsureSuccessStatusCode();
string responseBody = await response.Content.ReadAsStringAsync();
JObject jObject = JObject.Parse(responseBody);

JToken list = jObject["BTC_USD"];
List<Order> trades = list.ToObject<List<Order>>();

構造全体をタイプに変換するか 辞書<string、リスト<Order >>：

HttpClient httpClient = new HttpClient();
string request = "https://api.exmo.com/v1/trades/?pair=BTC_USD,ETH_USD";
HttpResponseMessage response = 
    (await httpClient.GetAsync(request)).EnsureSuccessStatusCode();

foreach (KeyValuePair<string, JToken> obj in jObject)
  trades.Add(obj.Key, obj.Value.ToObject<List<Order>>());

JSON配列（JArrayクラス）


JSON配列は、順序付けられた値のコレクションです。配列は '['（開始角かっこ）で始まり、 ']'（終了角かっこ）で終わります。値はカンマで区切られます。

JSON配列がどのように見えるかは、リンクhttps://api.exmo.com/v1/currencyをたどることで確認できます。前のリクエストで行ったように、JObjectに変換しようとすると、次のようになります。

HttpClient httpClient = new HttpClient();
string request = "https://api.exmo.com/v1/currency";
HttpResponseMessage response = 
    (await httpClient.GetAsync(request)).EnsureSuccessStatusCode();
string responseBody = await response.Content.ReadAsStringAsync();

JObject jObject = JObject.Parse(responseBody);

Newtonsoft.Json.JsonReaderException例外が発生し、変換されるデータがオブジェクトではないことが通知されます。



配列は、JObjectのようにJContainerから継承する特別なJArrayタイプに変換する必要があります。

HttpClient httpClient = new HttpClient();
string request = "https://api.exmo.com/v1/currency";
HttpResponseMessage response = 
    (await httpClient.GetAsync(request)).EnsureSuccessStatusCode();
string responseBody = await response.Content.ReadAsStringAsync();
JObject jObject = JObject.Parse(responseBody);
JArray jArray = JArray.Parse(responseBody);

解析時に初期構造が事前にわからない場合、または両方のタイプのデータを処理する機能が必要な場合-変換されたオブジェクトのタイプとしてJContainerまたはJTokenを指定すると、解析中にデータが暗黙的に目的のタイプに変換されます。

HttpClient httpClient = new HttpClient();

string request = "https://api.exmo.com/v1/currency";
HttpResponseMessage response = 
    (await httpClient.GetAsync(request)).EnsureSuccessStatusCode();
string responseBody = await response.Content.ReadAsStringAsync();
JToken jArray = JToken.Parse(responseBody );
//     JArray
List<string> list = jArray.ToObject<List<string>>();

request = "https://api.exmo.com/v1/trades/?pair=BTC_USD,ETH_USD";
HttpResponseMessage response = 
    (await httpClient.GetAsync(request)).EnsureSuccessStatusCode();
string responseBody = await response.Content.ReadAsStringAsync();
JToken jObject = JToken.Parse(responseBody );
//     JObject
Dictionary<string, List<Order>> dict = 
    jObject.ToObject<Dictionary<string, List<Order>>>();

JSON値（JValueクラス）


JSON形式の説明によれば、値は二重引用符で囲まれた文字列、数値、true、false、null、オブジェクト、または配列になります。これらの構造は入れ子にすることができます。newotnsoft.jsonの実装では、単純型のオブジェクトのみが値になることができます。JArrayをJValueに変換しようとすると、「System.InvalidCastException」例外が発生します。

newtonsoft.jsonライブラリで特定の値（JValueクラス）を操作するための入門資料として、ドキュメントの1回と2回の例が適しています。

株価データの使用例：

            
HttpClient httpClient = new HttpClient();

string request = "https://api.exmo.com/v1/currency";
HttpResponseMessage response = 
    (await httpClient.GetAsync(request)).EnsureSuccessStatusCode();
string responseBody = await response.Content.ReadAsStringAsync();
JArray jArray = JArray.Parse(responseBody );
foreach (JValue value in jArray)
{
    string s = (string)value;
}

using System.Threading.Tasks;
using System.Collections.Generic;
using System.Net.Http;
using Newtonsoft.Json.Linq;

namespace JSONObjects
{
    class Order
    {
        public int trade_id { get; set; }
        public string type { get; set; }
        public double quantity { get; set; }
        public double price { get; set; }
        public double amount { get; set; }
        public int date { get; set; }
    }
    class Program
    {
        public static async Task Main(string[] args)
        {
            HttpClient httpClient = new HttpClient();
            string request = "https://api.exmo.com/v1/currency";
            HttpResponseMessage response = 
                (await httpClient.GetAsync(request)).EnsureSuccessStatusCode();
            string responseBody = await response.Content.ReadAsStringAsync();

            JToken jArray = JToken.Parse(responseBody);
            //     JArray
            List<string> list = jArray.ToObject<List<string>>();

            request = "https://api.exmo.com/v1/trades/?pair=BTC_USD,ETH_USD";
            response = (await httpClient.GetAsync(request)).EnsureSuccessStatusCode();
            responseBody = await response.Content.ReadAsStringAsync();
            JToken jObject = JToken.Parse(responseBody);
            //     JObject
            Dictionary<string, List<Order>> dict =
                jObject.ToObject<Dictionary<string, List<Order>>>();
        }
    }
}
