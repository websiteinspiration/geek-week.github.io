Menginstal dan Mengkonfigurasi RTKRCV (RTKLIB) pada Windows 10 Menggunakan WSL

Halo, artikel ini membahas proses menginstal dan menjalankan aplikasi RTKRCV dari perpustakaan RTKLIB terbuka, yang dikembangkan oleh tim Tomoji Takasu (Jepang).

RTKLIB adalah paket perangkat lunak sumber terbuka untuk penentuan posisi standar dan akurat menggunakan teknologi GNSS (sistem navigasi satelit global).

Untuk siapa artikel ini?


Materi ini sangat penting secara praktis bagi semua orang yang ingin mengintegrasikan penentuan posisi yang akurat ke dalam perangkat mereka, dan juga dapat berfungsi sebagai referensi untuk mengoptimalkan pekerjaan dengan produk perangkat lunak yang disediakan oleh perpustakaan RTKLIB. Informasi dalam materi ini dirancang untuk mengoptimalkan interaksi spesialis dengan CUI pada sistem Windows 10.

Pertama, mari kita cari tahu apa itu RTKRCV. RTKRCV adalah analog CUI dari produk perangkat lunak RTKNAVI yang dirancang untuk bekerja dengan data navigasi. Secara real time, AP menerima pengamatan mentah dari penerima GPS / GNSS dan melakukan pemrosesan navigasi real-time.

Saat memecahkan masalah teknik di bidang geodesi, dimungkinkan untuk menggunakan konsol analog RTKRCV. Untuk CUI ini, ada informasi yang cukup lengkap tentang penggunaan produk ini, yang ada di instruksi untuk perpustakaan.

Untuk mengembangkan skenario pasca-pemrosesan Anda sendiri atau untuk menyelesaikan tugas apa pun, beralih antara Linux dan Windows menyebabkan ketidaknyamanan yang serius dalam pekerjaan. Minimal, Anda harus memiliki sistem terpisah atau menggunakan mesin virtual, yang juga bisa merepotkan.

Keselamatan adalah kehadiran di Windows 10 dukungan untuk subsistem pada kernel Linux.

Algoritma untuk mengkonfigurasi subsistem dan memulai RTKRCV:

  1. Menginstal WSL (Ubuntu, Debian, dll.);
  2. Menginstal utilitas yang diperlukan di WSL;
  3. Mengkonfigurasi ssh-key (PC <-> GitHub);
  4. Mendapatkan repositori dengan RTKLIB;
  5. Instalasi RTKRCV;
  6. Mengkonfigurasi file konfigurasi;
  7. Luncurkan RTKRCV.

1. Menginstal WSL (ubuntu)


Sebelum memulai instalasi, Anda harus menjalankan PowerShell dengan hak administrator dan menjalankan perintah berikut:

Enable-WindowsOptionalFeature -Online -FeatureName Microsoft-Windows-Subsystem-Linux

Saat diminta, mulai ulang komputer Anda.

Untuk mengunduh dan menginstal distribusi pilihan Anda, Anda memiliki tiga opsi:

  • Unduh dan instal dari Microsoft Store;
  • Unduh dan instal menggunakan baris perintah atau skrip;
  • Unduh dan kemudian unzip dan instal secara manual.

Kami akan menginstal distribusi kami dari toko aplikasi Microsoft.


Contoh ini akan diperiksa menggunakan distribusi Ubuntu 18.04 LTS.

Setelah menginstal konfigurasi di PowerShell dan mengunduh paket distribusi, jalankan paket distribusi Anda dan tunggu sampai sepenuhnya terinstal.



Setelah instalasi selesai, Anda akan diminta untuk membuat akun pengguna (dan kata sandinya).

2. WSL


Kita perlu menginstal make untuk membangun aplikasi kita dan satu set kompiler gcc, karena pustaka RTKLIB ditulis dalam C / C ++.

sudo apt install make
sudo apt install gcc

3. Mengkonfigurasi ssh-key (PC <-> GitHub)


Buat koneksi SSH antara GitHub dan WSL kami

  1. ssh-keygen -t rsa -b 4096 -C “ E-mail”

    Pesan: Masukkan file untuk menyimpan kunci (* direktori standar *) (Tekan Enter jika Anda setuju dengan lokasi penyimpanan kunci ssh)
    Pesan: Masukkan frasa sandi: (Tetapkan kata sandi untuk koneksi)
  2. eval $(ssh-agent -s)

    * Menerima pesan seperti: Agen pid 1234 
  3. ssh-add ~/.ssh/id_rsa

    * Setelah memasukkan kata sandi, dapatkan: Identitas ditambahkan: * direktori *
  4. cat < ~/.ssh/id_rsa.pub

    * Salin pesan yang diterima *
  5. Kami masuk ke pengaturan akun GitHub kami dan menemukan tab kunci SSH dan GPG. Atur nama dan tempel pesan yang disalin ke kolom Kunci.


4. Mendapatkan repositori dengan RTKLIB


Setelah mengatur

kunci ss h, kita dapat menggunakan git dan menambahkan repositori dengan RTKLIB (https://github.com/tomojitakasu/RTKLIB/tree/rtklib_2.4.3)

  1. mkdir rtk

    Buat direktori rtk untuk repositori. Tindakan opsional
  2. cd rtk

    Pergi ke direktori rtk.
  3. git clone git@github.com:tomojitakasu/RTKLIB.git -b rtklib_2.4.3

    Sebelum mengunduh, periksa nama cabang.

5. Instal RTKRCV


Pergi ke direktori utama dengan RTKRCV:

cd ~/rtk/RTKLIB/app/rtkrcv/gcc

Kemudian kami membangun aplikasi kami:

make CC=gcc

Untuk memulai rtkrcv gunakan perintah ./rtkrcv
