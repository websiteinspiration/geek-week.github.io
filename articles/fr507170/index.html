Est-ce que quatorze personnes consultent ce produit avec certitude?



Les magasins en ligne feront tout leur possible pour vous pousser à acheter leurs produits. Cependant, généralement ces évasions ne sont toujours pas aussi transparentes que des mensonges purs et simples, dont je parlerai dans cet article.

Lorsque vous parcourez les pages d'une boutique en ligne, vous tombez parfois sur un nombre qui indique le nombre d'autres clients qui sont censés voir le même produit en ce moment. Cette tactique est particulièrement populaire auprès des services de réservation de billets d'avion et de chambres d'hôtel. Il est susceptible d'être mis en œuvre dans l'espoir que vous essayerez de faire un achat le plus rapidement possible avant que quelqu'un d'autre intercepte ce que vous voulez.

Parfois, c'est un vrai chiffre, et parfois aléatoire. Autrement dit, c'est juste complètement aléatoire.

Regardons une boutique en ligne appelée Dafni . Si vous ne savez pas de quel type de magasin il s'agit, ne vous inquiétez pas, je ne sais pas non plus. Je sais seulement que là-bas, premièrement, ils vendent quelque chose et, deuxièmement, ils trompent les gens sur le nombre de visiteurs qui consultent tel ou tel produit en ce moment. Veuillez garder à l'esprit que cet article n'était pas destiné à être une exposition ciblée d'une entité de commerce électronique particulière. Ce dont je vais parler est un objectif assez commun, mais dans d'autres cas, il ne frappe généralement pas autant. Sur l'exemple de ce site, vous pouvez clairement montrer comment les boutiques en ligne trompent leurs clients en général.

Je suis allé sur le site Web et j'ai sélectionné le premier produit apparu sur la page principale. Dans mon cas, le produit s'est avéré être comme ceci:



La capture d'écran a été prise le 11 juin 2020, le cadre orange a été ajouté par l'auteur de l'article.

Oui, je comprends que vous êtes déjà distrait par le fait qu'il dit en caractères gras et en majuscules: «L'innovation à son meilleur», bien que selon toutes les règles de la grammaire, elle devrait être «sa». Mais veuillez attirer votre attention sur l'inscription: «14 personnes consultent actuellement ce produit» («Ce produit est actuellement consulté par 14 personnes»).



Vous ne pouvez pas transmettre cela dans une capture d'écran statique, mais au départ, le nombre a attiré mon attention car il changeait très fortement toutes les quelques secondes. J'ai décidé de jeter un œil au code source pour comprendre quelle méthode est utilisée lors de la mise à jour de ces données.

Pour ceux qui ne sont pas au courant: lorsqu'un développeur doit mettre à jour certains éléments d'un site Web après qu'il soit déjà chargé, généralement les langues des clients sont utilisées. Ces langues effectuent des opérations dans votre navigateur Web et, en conséquence, vous pouvez observer de vos propres yeux ce qui se passe sous le capot. Pour ce faire, cliquez simplement sur le bouton droit de la souris à n'importe quel point de l'écran et sélectionnez "Source de la page" dans la liste déroulante ou quelque chose comme ça. Ensuite, vous pouvez lire le code responsable de la mise à jour de divers types d'éléments, y compris le nombre actuel de visiteurs sur la page.

Sur ce site, j'ai trouvé ce qui suit:



J'ai copié le fragment souhaité sans modifications. Oui, il a été écrit dans le code source, «prodiutc». Mais, plus intéressant, à partir de ce code, il devient clair que le nombre, qui devrait en théorie refléter le nombre de personnes qui consultent le produit, est déterminé de manière aléatoire.

Je vais traduire ce passage en langage humain:



Toutes les cinq secondes: sélectionnez un conteneur qui affiche un numéro, générez un nombre aléatoire dans la plage de 3 à 14, mettez ce numéro dans le conteneur sélectionné.

Ainsi, le nombre d'acheteurs intéressés par le produit n'est plus qu'une fiction. Le code affiche simplement un nombre arbitraire sur la page toutes les cinq secondes. Du coup, il semble que les visiteurs circulent constamment sur le site, que la vie y bat son plein, ce qui fait qu'il est populaire. Il faut supposer qu'un grand nombre d'acheteurs hypothétiques rendent également le magasin digne de confiance aux yeux de l'utilisateur. De plus, cela encourage les gens à effectuer un achat plus rapidement - il peut sembler que les biens qui les intéressent sont très demandés.

Dans le cas où quelqu'un tombe sur cet article dans un avenir lointain, lorsque le code a déjà été modifié, j'ai une bonne nouvelle: le 16 mai 2020, le site a été indexé par des robots de recherche d'archives Web - en particulier, ils ont également enregistré le fragment ci-dessus . Vous pouvez consulter la version indexée du site ici .

Si vous rencontrez quelque chose de similaire en vous promenant sur le World Wide Web sur un autre site, n'hésitez pas à entrer dans le code source - vous ne savez jamais quels monstres s'y cachent, dans les entrailles derrière l'interface. Les propriétaires de magasins mentent parfois, mais le code ne ment pas.
