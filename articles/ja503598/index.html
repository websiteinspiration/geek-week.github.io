GNOMEの特許手続きの中止




GNOME Foundationのは、特許侵害のプロジェクトを非難、大規模な特許トロールロスチャイルド特許イメージングLLCによって訴訟の成功した解像度を発表し、OpenNET報告：「当事者は原告がGNOMEに対する料金をすべてドロップする合意に達し、に関連した多くの訴訟を持っていない合意彼が保有する特許に違反して。さらに、ロスチャイルド特許イメージングは​​、OSIによって承認されたオープンライセンスに基づいてコードが提供されているオープンソースプロジェクトを提訴しないことを約束しました。コミットメントは、ロスチャイルド特許イメージングLLCが所有する特許ポートフォリオ全体をカバーしています」特に最初のクレームの苦痛と、手続きの結果に続く特許トロールに課された制限を比較すると、印象的な勝利です。OpenNETによると、この主張の本質は、Shotwellフォトマネージャーが画像受信デバイスと画像キャプチャデバイスのワイヤレス接続と、日付、場所、その他のパラメーターでフィルター処理されたその後の画像の選択的転送を説明する2008年の特許に違反したことです。

多くのオープンソースコミュニティはパンデミックにより深刻な課題を抱えています




コロナウイルスのために、多くの技術的なイベントがキャンセルされました。これは、これらのイベントに依存する企業やグループの財務状況に悪影響を及ぼします。 The Linux Foundationなどの一部のオープンソース組織は、これを乗り切ることができます。他はそれほど幸運ではありません。 Drupal Foundation、Open Source Initiative（OSI）、Open Source Matters（Joomla）、Ajv JSON Schema Validatorなど、一部は実際に問題を抱えているとZDNetは報告しています。それらのほとんどは資金調達と金融パートナーシップに依存しています。これらのイベントはもはや存在しません。オープンソースプログラムが数十億の企業を牽引していることを考えると、これは特に不愉快です。多くのオープンソースプロジェクトは、非常に限られた予算で機能します。しかしサステインのような組織がありますオープンコレクティブとコアインフラストラクチャイニシアチブは、影響を受ける人々と彼らが作成するコードが適切に機能するように支援しようとしています。

非推奨のオープンソースライブラリにより、アプリケーションの71％が脆弱になります
