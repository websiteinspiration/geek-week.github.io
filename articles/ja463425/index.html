ハイブリッド電話システム



ハイブリッドPBXは、従来の通信の利点とIPテレフォニーの機能を組み合わせ、幅広い機能を提供し、さまざまなビジネスニーズを柔軟に解決できるようにします。ハイブリッド電話システムは、IPシステムへの簡単で快適な移行を提供できます。これは、既存の電話ネットワークを維持しながら、統合ソリューションに切り替える顧客にとって有益です。

ハイブリッド電話システムは、デスクトップコンピュータおよびネットワークのSIP接続をサポートし、IP電話の使用を可能にし、標準のデジタルおよびアナログデバイスをサポートします。通常、ハイブリッド電話システムは、完全なIPソリューション、完全なデジタル、またはその両方の組み合わせとして設計されています。

つまり、ハイブリッドシステムの利点は次のとおりです。

  • 従来のテレフォニーからIPテレフォニーへの無痛の移行。
  • VoIPにより、高価なIPインフラストラクチャに投資する必要がなく、コストを削減できます。
  • 従来のテレフォニーインフラストラクチャへの投資を保護します。
  • 適切な場所にのみVoIPを導入します。

そして今、ハイブリッドシステムを使用する3つの実際の例があります。

バーミンガム女性病院


バーミンガム女性病院信託基金は、ウェストミッドランド中の女性とその家族に医療サービスを提供しています。年間1,500人の従業員を擁するこの慈善団体は、年間9000万ポンドの収入を得て、約50,000人の患者にサービスを提供し、約3,000件の手術を行っています。2013年の終わりに、病院は電話システムを更新しました。VaughanData SystemsはVoIPソリューションを導入しました。

問題


病院の電話システムは、シーメンスのPBXサービスに基づく外部のエンジニアチームによって作成されました。病院の経営陣は、音声送信とデータ処理の機能を組み合わせた新しいシステムを作成したいと考えていました。これにより、病院は独立してサービスを提供できるようになります。長期的な修理と復興プロジェクトの一環として、財務部門とIT部門はそれぞれのオフィスに移動しました。病院は、1,000台を超える電話を新しい、より費用対効果の高いIPソリューションに置き換えることを意図していました。新しい建物に光ファイバーが導入されましたが、内部配線がなかったため、論理的な手順は、音声とデータの伝送用にカテゴリ5とカテゴリ6のケーブルを分離することでした。競合分析により、カテゴリ5およびカテゴリ6のケーブルネットワークへの展開が容易である一方で、統合IPソリューションは古いPBXを拡張するよりも費用がかからないことが示されました。

ITマネージャーのDarren Adamsは、Vaughan Data Systems LtdおよびWorcestershireのBromsgroveにあるIT Solutions＆Support Companyに連絡して、3CXシステムをオンラインでさらに調査しました。高品質のVoIPシステムを提供し、既存の古いシステムとのスムーズな運用を保証することは困難な作業であったため、これはVDSにとって特に興味深いプロジェクトでした。

決定


最初に、VDS、パットンISDNゲートウェイ、3CXサービスによって提供される4台のSnom IP電話の試用システムを作成しました。インストールと構成に2日かかり、さらに1か月後に病院のスタッフがシステムをテストしました。

3CXは、ISDNカードを介してPBXにデータをルーティングするPatton ISDN30ゲートウェイを介して古いPBXに接続されていました。最初は複雑に見えましたが、3CXとEFL、パットンとスノムの機器の販売代理店の技術的な経験により、私たちはこのタスクに対処でき、請負業者はリモートでパットンゲートウェイを事前構成し、30日間の無料テストを実施するのに役立ちました。セットアップの主な遅れは、IT環境が高度に保護され、新しいデバイスへのアクセスがブロックされているため、病院システムへの安全なアクセスを提供するために時間を費やす必要があったという事実によるものです。

利点


テストは非常に成功し、問題はなく、さらに構成を変更する必要はありませんでした。プロジェクト管理およびITスタッフのセットアップと日常のシステム管理のトレーニングについて、VDSと契約が締結されました。すべての構成が試験運用段階でテストされたため、エンドユーザーはトレーニングを受ける必要がなく、展開は非常に高速でした。

パラディーズホテル、スイス


Paradies Hotelは、アーティストのハンス・ウォルター・バイエルの避難所として1910年に建てられました。ホテルには23室の客室があり、スイスアルプスで最も美しい高原の1つ、グラウビュンデン州の山中にあります。高級家具、絶妙なミシュランスターL'Autezzaレストランのシェフ-これらすべてが、この5つ星ホテルを特別な場所にしています。近年のすっきりとした近代化により、パラディーズホテルの特徴を維持し、その名前を正当化することができました。

問題


ハンブルクからの新しい所有者が1995年にホテルを購入したとき、独自の電話システムがすでにここに設置されていました。しかし、彼女は現代の要件を満たしていませんでした。 「私たちの電話システムは月に3回クラッシュし、スペア部品を見つけるために多くの労力とお金を費やす必要がありました。したがって、私たちが緊急に行動する必要があることは明らかでした」とホテルのレストランのシェフであるリサラッシュは思い起こします。

明らかに、すべての訪問者が携帯電話を持っている場合、ホテルはより費用対効果の高いソリューションを必要としています。ゲストがホテルの電話を使用して利益を上げていた時代はもう過ぎ去りました。高い維持費とレンタルコストが原因で、近代的なシステムを迅速に確立し、サードパーティへの依存を減らすという決定に至りました。主な要件は、テレフォニーをスタッフが無料で使用できないようにすることです。電話のデザインも重要な役割を果たし、ホテルのレベルに対応する必要がありました。

決定


システムインテグレーターのFüssenerCinehausは、ホテルのニーズを考慮したソリューションをリーズナブルな価格で適切な電話で提供しています。同社は、オープンSIPプロトコルに基づくシステムへの切り替えを推奨しました。そして2013年の春、VoIPテレフォニーが導入されました。これは、Xorcom TwinStar Asterisk IP PBXとSnom IP Phoneを組み合わせたソリューションでした。

Xorcomコンシェルジュソフトウェアを使用すると、PBXをMicros-Fidelioホテル管理システムに接続できます。Fidelioは非常に閉鎖的な製品であるため、これはユニークな機能です。レストランのキャッシュデスクを管理システムに接続することにより、経営者はホテルのすべてのニーズを考慮した完全な統合を活用できました。客室にはデンマーク企業のデザイナーの電話が備わっています。ホテルの管理者は電話にもっと多くの機能を持たせる必要があるため、エレガントで多機能なSnom 821モデルを選びました。受付には、タッチスクリーンとSnom Vision拡張モジュールが統合された便利で直感的なSnom 870が装備されていました。

利点


設置予定日は2日でしたが、Cinehausシステムインテグレーターはすぐに作業を開始しました。作業開始の翌日の正午には、ホテルには新しい電話システムが設置されていました。ホテルはまた、24時間体制のテクニカルサポートに関する契約を締結しました。無料のアスタリスクソリューションのおかげで、ホテルは1つのサプライヤーへの依存を解消しました。月々の費用は半分になり、新しいソリューションははるかに使いやすくなりました。

予定


ホテルは、光ファイバーケーブルでロウアーエンガディンのSkuol地区とFtanaサンテラスに連絡するのを楽しみにしていますが、今ではホテル名「paradise」と一致する状態に達しています。

ハンス・キュンツGmbH


HansKünzGmbHは、オーストリアのフォラルベルクで最も古く、最も成功したエンジニアリング会社です。それは1932年に設立され、すぐに主要な産業施設の開発者および建設業者になりました。Künzは建設用クレーンの製造を専門に始めましたが、後にコンテナクレーンと油圧装置に注力しました。ボーデン湖からさほど遠くないハードの本社に加えて、同社はシュタイアーマルクのグロスセントフロリアンに工場を所有しており、スロバキアと米国にも支社があります。同社は約300人を雇用しています。

問題


Künzは、古い固定電話システムの保守と管理のコストを削減するために、VoIPに切り替えることを決定しました。2010年2月、既存のネットワークに統合できる最新の電話システムを選択し始めました。Künzは8つのメーカーのVoIPシステムを比較しました。一部のソリューションは要件を満たしていないか、高すぎるか、必要な機能を持っていませんでした。リーダーは3つのIPシステムです。

決定


2011年の初めに、HansKünzGmbHは以前の電話システムをMicrosoft Communications Server 2007 R2とSnom 821ビジネス電話の組み合わせに置き換えました。OCS 2007 R2は、既存のMicrosoft環境およびMS Office製品ラインにシームレスに統合されています。しかし、スケールをMicrosoft OCS 2007 R2に有利に導いた決定的な要因は、既存の通信インフラストラクチャに直接統合できることでした。中間サーバーを使用すると、VoIPスイッチを直接固定回線に接続できます。同社は固定システムを介して引き続き外部に電話をかけることができ、すべての内部通信はVoIPを介して機能しました。

利点


大型の高解像度カラーTFTディスプレイとHDオーディオコーデックにより、Snom 821はMicrosoft OCS 2007 R2環境での作業に最適なデバイスです。 KünzはIP電話を管理し、外部のパフォーマーのサービスなしで新しいラインを拡張することができました。 KünzはSnom IP Phoneのインストールに確信を持っていました。広範囲の機能を備えているわけではありませんが、優れたコストパフォーマンスを備えています。デスクフォンとMicrosoft OCSの統合により、会社全体でユニファイドコミュニケーションの概念を実装できるようになりました。 Snom 821には「プライマリプレゼンス」機能があり、ユーザーが電話で話しているかどうかが自動的に報告されます。加入者ビジーに関する情報はOCSに送信されます。また、職場での従業員の存在に関する情報により、無駄な電話の回数が減りました。さらに、従業員はOutlookまたはERPシステムから直接電話をかけることができます。部屋探しに費やす時間を節約できます。自動応答機能は、発信者に営業時間と週末を自動的に通知し、メールボックスにリダイレクトします。 VoIPを使用する主な利点は、従業員が場所に関係なく、WLANを介してスマートフォンから電話をかけることができることです。 VoIPは、国際電話を含むコストも節約します。

予定


Snom 821での最初の良い経験により、Künzは支店にSnom DECT電話を装備することを検討しました。
