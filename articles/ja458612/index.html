宇宙。7年間

こんにちは、ハブル！

2012年に、趣味- コスモス：多細胞生物の進化のモデリングに関する投稿を書きました


それから約7年が経過し、その間私はこのプロジェクトの開発に取り組みました。今日は、私が学んだことと、プロジェクトをさらに発展させる計画について少しお話ししたいと思います。

動機「大きな進化」


Cosmos 2012は、「GPUでの生命のシミュレーション」を作成するためのアイデアの開発の出発点であった、表示からの「没入効果」の感覚の良い例です。

ただし、この進化はそこでは発生しませんでした。プロセスは継続的に開発され、起動後数分で最終的な状態にならないはずです。

最も困難なことは、「開発の限界に収束しない」種の可能な限り長い進化を達成し、実際にさまざまな形態を生成することだけであることが判明しました。その後、これらのフォームを探索することが可能になり、シミュレーションを数日、数週間、数か月実行することに真の関心が集まり、ますます複雑で開発されたフォームが出現するのを待ちます。

物理


任意の物理学について考えることができます。GPUの助けを借りて、リアルタイムの進化プロセスのフローに十分な要素数を持つ非常に幅広いモデルを構築できます。特定の「要素」、「原子」、「セル」の相互作用の原則の任意の組み合わせ-いくつかの複雑に編成されたシミュレーションが文字通り何もないことから自己組織化できるフォームの進化するコンストラクターを生み出します。

初めは、宇宙空間が「飛ぶことなく」直接触れ合い、ボールがモザイク状につながった構造で、実在する生物の細胞組織のように、多細胞物質を作りたかったのです。

生物が本物のようになることが必要であり、身体のさまざまな構造に物理的な意味が現れました。皮膚、手足、筋肉、足、内臓などの体をこのような細胞で「描く」ことができます。そのような体はシミュレーション内に文字通り存在し、発達します。

CUDA SDKの1つの例は、空間ラティスに基づいて丸い粒子の衝突を実現し、粒子が近くのものとのみ相互作用できるようにすることでした。

3Dの例がどのように機能するかを理解し、2Dのシミュレーションを作成しました。
当時のGTX 580では、これによりリアルタイムで256千個の粒子を計算できました。


CUDA + JavaScript


もう1つの問題は、JavaScriptで宇宙の概念を思い付くのに便利だったのに、本当に大規模なシミュレーションはGPUでしかできなくなったことです。

JSでは、約100倍便利で高速です。
また、GPUでは約1000倍高速です。

解決策は、CUDAで粒子相互作用の物理学を記述し、JavaScriptから粒子を作成してそのパラメーターを制御する機会を与え、シミュレーションにスクリプト言語として登場したGoogle V8エンジンを統合することでした。

最初のスペースで接続したのと同じ「ラバーバンド」でパーティクルを接続する機能を追加しました。その結果、JS制御スクリプトでコードを書くことができます：

var ball1 = new Ball();
var ball2 = new Ball();

ball1.xy = [0,0];
ball1.rgb = [1,1,1];

ball2.xy = [1,0];
ball2.rgb = [1,1,1];

var link = new Link(ball1, ball2);

画面に2つの白いボールが現れ、「弾性バンド」で相互に接続されました。

エンジンのこのレベルの開発では、仮想物理学の実験に興味深い機会がすでに現れています。


構造


結合の強さ、質量、サイズ、弾性などのいくつかのプロパティをボールに追加することで、JavaScriptで制御されるコンストラクターを取得し、さまざまな物理的プロパティのセルで構成される何千もの生き物の構造を描画できました。これはすべて、GPUによってリアルタイムで計算されました。

価格と品質のバランスとして、柔らかな流線型のタコを選びました。

体の各要素（中央、四肢）は、必要最小限の数のボールを使用して実現されますが、体積と質量は自然に分布します（四肢が中央に向かって厚くなります）。


運動性


次のステップは、宇宙体の運動性を生み出すことでした。筋肉は簡単に作ることができました-足に沿ってボール間の結びを締めるのに十分でした、そしてこれは筋肉収縮の影響を引き起こしました。

活動ニューロンはこれらの筋肉に接続されており、信号がある場合は筋肉が収縮します。四肢あたり2つの筋肉-完全に制御可能な2次元のタコ。

（10〜100ニューロンのオーダーの）小さなランダムに変化するリカレントニューラルネットワークと、たとえば「さらに移動できる人」などの基準に従って最良の個人を簡単に選択することにより、個々の体型に必要な運動のキネマティクスを呼び出すことができます。


創世記


別の積極的に探究されている体の形は「ヘキソイド」です。このような生き物の体は、六角形に近い非厳密な格子に基づいて同じサイズのボールから任意の形状を形成する細胞で構成されています。

このフォームは、体型と運動の同時進化を「組み合わせて」研究するのに最適であり、強度と機能が異なるさまざまなタイプの細胞を使った実験の可能性を最大限に引き出します。たとえば、宇宙のボディは、生き物のフレームを形成する固体細胞と、構造を動かす筋肉との組み合わせで構成できます。

将来的には、これらの原則に基づいて複雑な生息地に没頭すると、明らかに異なる形の生き物を開発することができます-いくつかはより「装甲」され、他のものは速い動きのために設計され、他のものは効果的な攻撃などのために設計されます。


狩猟と繁殖


宇宙が互いに狩りをするように教えられている場合、シミュレーションの物質の量は一定に保たれ、世代の生き物は互いに成功します。

次に、安定した食物連鎖が形成され、さまざまな生物種の共進化の大きなプロセスが始まります。

最初の実験では、4種類の細胞を使用しました。

  • 消化細胞（色付き） -別の宇宙の身体に触れると、破片を噛みます
  • 鎧（ライト） -かみ落とすことができないより強力な機械的ケージ
  • 結合組織（灰色） -柔らかくて伸縮性のある生地
  • モーターセル（暗い） -スペースがより速く移動できるようにします

コスモスが十分な数の細胞を食べるとすぐに「分裂」し、わずかに変異したコピーの近くに形成されます。

このようなプロセスは、すでに構造と動作が異なるフォームを生成しています。



生息地


空の冷たい空間は住むのに最適な場所ではありません。

仮想バイオームを作成するには、基本的な建築材料である、土、水、光の粒子、および成長可能な植物細胞を使用できます。

土の粒子が互いにくっつき、強固なプラットフォームを形成します。水滴がかかると土壌が濡れる可能性があります。

湿った土壌では、良い光のもとで植物が育ちます。植物の固まりは動物の形態のための有機性食糧の主要な源です。

動物の体は腐敗し、無機物質をバイオーム生態系に戻します。
