Cara melihat Oracle WebLogic Server menggunakan monitor besar dan konsol utilitas WLSDM

Di ruang terbuka utilitas ekstensi konsol Oracle WebLogic Server, ada satu yang sangat berguna - WLSDM , karena penulis sendiri memposisikannya - utilitas pemantauan Server WebLogic dengan sejumlah besar fitur. Jika Anda pergi ke situs pengembang, Anda akan melihat bahwa ada alat kuat lain di dekatnya, tetapi untuk biaya.

Setelah mengunduh arsip, hanya dalam dua klik kami mendapatkan alat yang sudah sesuai untuk bekerja, dengan seperangkat nilai ambang, kemampuan untuk mengirim pesan dan spedometer grafik besar. Khusus untuk monitor besar, atau untuk membantu insinyur DevOps berpasangan yang perlu mengikuti aplikasi pemintalan.

Beberapa kata tentang metrik server secara umum. Tentu saja, Anda dapat menggunakan tampilan metrik bawaan. Anda bahkan dapat mempelajari cara menyesuaikan tabel metrik di server itu sendiri.
Tetapi kami melanjutkan dari kenyamanan penggunaan dan kemiripan alat ini.
Plus, ambang batas yang ditetapkan segera ditampilkan pada grafik dan peringatan dikirim melalui surat. Yang sebenarnya diperlukan untuk DevOps.


  1. Instalasi agen.










    Anda dapat menginstal keduanya di server Admin, dan di setiap server yang Anda rencanakan untuk dipantau. Untuk kesederhanaan, kami disebarkan ke Admin.

    Perhatian! Jangan lupa untuk reboot untuk memulai konfigurasi.

    Setelah reboot, sebuah jendela muncul di konsol dengan utilitas pemantauan (di sebelah kiri). Menu utamanya (kanan).
