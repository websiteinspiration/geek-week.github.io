Lanzamiento de SQL: evento de Microsoft SQL Server 2019



Los mejores expertos de Microsoft hablarán sobre las principales innovaciones en SQL Server 2019: tecnologías de Big Data Clusters de SQL Server para trabajar con big data y aprendizaje automático, tecnologías de Polybase para acceder a datos en fuentes externas sin copiarlas, soporte de contenedores, trabajar en OS Linux y muchos otros ¡Nuevo en MS SQL Server 2019!

Se dedicará un informe separado a la plataforma de inteligencia empresarial Microsoft Power BI!

¡Lo invitamos a visitar la presentación de la nueva versión de la plataforma de administración de datos híbrida SQL Server 2019! Regístrese , pero los detalles están debajo del corte.

¿Sabía que hemos lanzado una nueva versión de SQL Server: SQL Server 2019? Gracias a muchas nuevas funciones útiles, esta versión ha generado un interés sin precedentes, y la velocidad de traducción de los sistemas de información por parte de las organizaciones supera el rendimiento de todas las versiones anteriores. Con la transición a SQL Server 2019, las empresas están abandonando gradualmente sus costosas instalaciones de Hadoop, dominando los clústeres de big data. Las formas de acceder a los datos también están cambiando: las capacidades modernas de visualización de datos le permiten diseñar nuevos proyectos sin requerir un costoso movimiento de datos y una copia preliminar de los datos. Las cargas de trabajo existentes son aceleradas por una gran cantidad de nuevas funciones inteligentes de base de datos.

Puede que no le resulte fácil hacer un seguimiento de todas las tendencias emergentes. Es por eso que nos complace invitarlo a visitar nuestro evento el 3 de marzo a las 09:30 en Spartakovsky Lane 21, entrada 7, para comprender las razones del interés en tantas empresas en SQL Server 2019. Los oradores de Microsoft de diferentes países y nuestros socios compartirán sus impresiones. Estamos seguros de que determinará usted mismo todas las ventajas potenciales de esta nueva plataforma. Esperamos reunirnos con usted y discutir las posibilidades de reducir costos, acelerar el trabajo en proyectos y una interacción cómoda con los usuarios.
