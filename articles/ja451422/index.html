世界最高のプログラミング言語

すべてのプログラマーには夢があります。自分のプログラミング言語を作成することです。もちろん、最良で最も便利なもの-既存のものとは異なります。私個人的には、修正のアイデアは、可能な限り短いが非常に雄弁な定型文が絶対にない言語を作成することでした。 2年間、私は望ましい結果を達成するために努力してきましたが、どこから始めても、結局、余分なものをすべて切り捨てた後、私は常にLispを手に入れました。そして、Lispを採用し、私のアイデアに従ってそれを改善するという独創的な考えが私に起こりました。最初に目にしたのを記念して、このプロジェクトはSovaと名付けられました。


すぐに使える言語をサーバー、ウェブ、モバイルプラットフォームで動作させるために、私はそれをJavascriptでコンパイルすることに決めました。これは、私の意見では、私たちの時代の最高の設計言語の1つであり、さらに重要なことに、強力なnpmエコシステムがあります。 Node.js、ReactおよびReact Native。


そして、私たちの旅を始める前に、記事をさらに読むことに興味をそそるように、Sova上の1つのExpressアプリケーションファイルの例を次に示します。


=-> './database' database
=-> 'express' express

= application express ()

application.get '/'
  -> (request response) (response.sendStatus 200)

application.get '/user/:id'
  -> (request response) (response.send (database.getUserById request.params.id))

application.listen 8080
  -> () (console.log 'Application is listening on port 8080')
