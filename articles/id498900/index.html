Rosbank Java-school - hasil pembelajaran

Halo semuanya!

Kami terus berbagi dengan Anda kisah-kisah anak muda yang kami bantu temui di bidang IT perbankan. Pada akhir tahun lalu, kami menulis bahwa kami membuka sekolah untuk pengembang Java di Nizhny Novgorod. Di antara aplikasi, kami memilih 25 orang, dan menurut hasil pelatihan lima siswa terbaik yang kami bawa ke tim TI kami. Bagaimana hari kerja mereka sekarang - baca di bawah potongan.



Eugene, pengembang backend




Sekarang saya sedang belajar di "Keamanan Informasi Jaringan Telekomunikasi" khusus di NNSU dinamai Lobachevsky, tetapi saya menjadi tertarik pada pemrograman di sekolah menengah.

Di sekolah menengah, saya mulai mencari pekerjaan dengan kesempatan untuk terlibat dalam proyek yang sangat menarik di bidang pengembangan perangkat lunak. Itu perlu untuk menentukan bahasa. Saya memilih Java karena bahasa ini digunakan di perusahaan besar yang memberi perhatian khusus pada keandalan dan keamanan - Saya selalu ingin bergabung dalam pengembangan sistem yang besar dan kompleks, menerapkan pengetahuan teoretis yang diperoleh di universitas. Pada jam, saya belajar tentang sekolah Jawa Rosbank. Saya tertarik dengan program pelatihan dan prospek pekerjaan.

Selama pelatihan, kami memiliki mentor latihan yang keren. Dia dengan jelas mengatakan teorinya, dan kami juga memiliki banyak latihan dan mengerjakan tugas nyata. Lebih dari 90 jam pelatihan, kami berkenalan dengan "dapur" pembangunan: JavaCore, Spring Framework, Hibenate, PostgreSQL, dll. Tapi, yang paling keren, setelah membela proyek akhir, saya disewa oleh Rosbank.

Sekarang saya bekerja di Pusat Pengembangan Kompetensi untuk EDMS dan solusi portal. Berkat sekolah, saya merasa percaya diri dan sedang mengembangkan backend untuk proyek internal bank. Karya ini menarik dan sangat serbaguna, yang hanya memicu minat dan keinginan untuk berkembang. Saya juga ingin memperhatikan kondisi kerja: kantor yang nyaman, pemimpin tim yang hebat dan tim.



Cyril, pengembang Java




Saya mulai belajar pemrograman pada usia 11 tahun. Setelah lulus dari bacaan fisik dan matematika, saya belajar di St. Petersburg sebagai spesialis IT.

Pada musim gugur 2019, di bursa kerja, ia belajar tentang sekolah Jawa di Rosbank. Ketertarikan pada Jawa disebabkan oleh fakta bahwa itu adalah bahasa pemrograman yang dikembangkan oleh infrastruktur.

Pada tahap seleksi, saya tidak berpikir akan lulus - pertanyaannya sulit. Akibatnya, saya termasuk di antara 25 siswa terbaik yang dibawa ke sekolah. Dalam pelajaran pertama, guru memberi kami nasihat yang baik: bulan depan untuk melupakan segalanya kecuali Jawa, dan berjanji bahwa kami akan mengingat belajar untuk waktu yang lama.

Secara terpisah, saya ingin mencatat program pelatihan: dalam waktu singkat saya berhasil membenamkan diri sepenuhnya di Jawa. Otaknya mendidih. Di malam hari, kuliah, dan hari berikutnya kami sudah mulai mempraktikkan materi yang sama. Kami juga harus belajar cara mempresentasikan hasil pekerjaan kami kepada perwakilan bisnis. Ini adalah keterampilan yang sangat penting.

Belajar itu sulit, kadang-kadang saya bahkan berpikir untuk menyerah. Namun, saya mencapai final, menciptakan proyek kelulusan dan mendapat pekerjaan. Sekarang saya terlibat dalam pengembangan layanan microser untuk penggunaan internal di Rosbank. Sangat suka.

Yana, Pengembang Java




Ketertarikan saya pada pemrograman muncul di sekolah. Ketika tiba saatnya untuk mendaftar ke universitas, saya tahu pasti bahwa saya ingin menjadi pengembang. Setelah belajar di universitas, saya menyadari bahwa saya telah memperoleh banyak pengetahuan mendasar, tetapi tidak ada cukup latihan. Saya ingin mendapatkan lebih banyak pengalaman pengembangan pada kasus nyata. Sebelum itu, saya sudah belajar sendiri bahasa pemrograman berikutnya dari awal, tetapi sekarang menarik untuk melakukan proyek baru, mendapat umpan balik dari guru. Pada saat itu saya mengetahui tentang sekolah Jawa Rosbank dan berpikir: "Ya, itulah yang saya butuhkan!"

Sekolah sekuat mungkin: banyak informasi dan pekerjaan rumah baru. Terkadang itu sangat sulit. Apakah itu layak? Pastinya. Untuk periode pelatihan singkat, kami menerima sejumlah besar pengetahuan profesional yang diajukan oleh guru. Semua tugas untuk eksekusi independen sedekat mungkin dengan yang nyata. Setelah lulus, saya merasa siap untuk mulai bekerja sebagai pengembang Java.

Saya suka bekerja di Rosbank. Kantor yang sangat nyaman, tidak ada kode berpakaian. Saya bekerja di departemen yang mengembangkan aplikasi untuk penggunaan internal oleh karyawan bank. Ini sangat keren - saya tahu saya membuat program yang diperlukan dan nyaman untuk orang yang bekerja di sebelah saya. Sekarang saya sedang mengembangkan layanan web menggunakan Spring. Saya bekerja di tim yang hebat - kapan saja, kolega senior dapat membantu dengan pertanyaan sulit.

Alexander, pengembang Java




Saya lulus dari Fakultas Teknik dan Fisika, kemudian bekerja untuk spesialisasi selama 2,5 tahun dan menyadari bahwa saya telah mencapai puncaknya. Dia mulai berpikir tentang pengembangan profesional lebih lanjut dan ingat bahwa sejak kecil dia telah menatap pemrograman. Atas saran teman-teman lamanya, ia memilih Jawa.

Dia belajar tentang sekolah Rosbank Java di portal jam dan memutuskan untuk mencoba tangannya.
Pelatihan ini ternyata cukup padat: 6 minggu teori dan praktik kualitatif, sangat penting, pekerjaan rumah, dan proyek kelulusan terkonsentrasi. Secara terpisah, perlu dicatat guru kursus dan tentu saja seorang profesional di bidangnya - Ilnaz Gilyazov.

Saya suka pekerjaan saya di Rosbank. Sekarang saya terlibat dalam pengembangan layanan untuk memproses aplikasi internal. Proyek ini menarik dan kompleks secara arsitektur, tetapi untungnya kolega dan kawan senior dengan senang hati membantu dalam situasi yang kontroversial.

Roman, pengembang backend




Saya lulus dari universitas sarjana. Lobachevsky ke arah "Matematika Terapan dan Ilmu Komputer". Setelah menyelesaikan tahun ke-4, saya mulai mencari pekerjaan, dan saya melihat iklan untuk sekolah Jawa. Saya pikir itu akan menjadi pengalaman yang baik, karena mereka menjanjikan banyak latihan (itu).

Pelatihan berlangsung selama satu setengah bulan, dan selama ini saya menyadari bahwa saya sangat menyukainya. Bagi saya sendiri, saya memutuskan bahwa saya pasti ingin pergi ke Rosbank, dan berusaha melakukan yang terbaik untuk mencapai ini.

Sebagai hasilnya, saya berhasil menyelesaikan pelatihan, membela proyek dan masuk ke tim pengembangan Rosbank. Saya mendapat pemimpin tim yang luar biasa yang akan menjelaskan dan membantu dalam situasi apa pun dan rekan-rekan saya sangat dekat dengan saya. Proyek pertama saya adalah membuat shell AOP untuk menyinkronkan dua aplikasi satu sama lain. Proyek ini memungkinkan saya untuk lebih memahami pengembangan backend.
