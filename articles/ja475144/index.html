こんにちは、ハブル。最近、私は興味深いプロジェクトを手に入れました。その単純さにもかかわらず、フレームワークを使用しないことを要求しました。パッケージの話はなかったので、通常のLaravelコンポーネントを使用することにしました。キュー、Eloquentまたはコンテナーを使用する必要がある場合-猫へようこそ。


フレームワークをコンポーネントに分割する単純さについて


Laravel 4以降、すべてのコンポーネントは個別のパッケージであり、理論的には任意のPHPプロジェクトで機能するように作成できます。ただし、一部のコンポーネントには追加のカスタマイズが必要です。これはメインフレームワークでは内部的に隠されています。


部品


容器


検討中のすべてのコンポーネントのうち、illuminate/containerインストールと使用が最も簡単です。


<?php

use Illuminate\Container\Container;

$container = Container::getInstance();

return $container->make(Foo::class);

ただし、Containerコンテナーを使用するたびにクラスで静的メソッドを呼び出すことはお勧めできません。フレームワークではavailableがあり app()、グローバルコンテナのインスタンスが返されますが、手動で作成する必要があります。


helpers.phpそのようなヘルパー用のファイルを作成し、スタートアップに追加します。


"autoload": {
    "files": [
        "src/helpers.php"
    ],
    "psr-4": {
        "YourApp\\": "src/"
    }
}

app()ファイルにヘルパーを追加します。


<?php

use Illuminate\Container\Container;

if (! function_exists('app')) {
    /**
     * Get the available container instance.
     *
     * @param  string|null  $abstract
     * @param  array  $parameters
     * @return mixed|\Illuminate\Container\Container
     */
    function app($abstract = null, array $parameters = [])
    {
        if (is_null($abstract)) {
            return Container::getInstance();
        }
        return Container::getInstance()->make($abstract, $parameters);
    }
}

キューの構成が完了しました。これで書き始めることができますqueue:work。


<?php

require_once __DIR__.'/bootstrap/bootstrap.php';

use Illuminate\Queue\Worker;
use Illuminate\Queue\Capsule\Manager;
use Illuminate\Queue\WorkerOptions;

$queueManager = Manager::getQueueManager();

$worker = new Worker($queueManager, app('events'), app('exception.handler'));

$worker->daemon('default', 'default', new WorkerOptions);

ここで、キューワーカーを開始するには、とだけ記述しphp worker.phpます。


使用する必要がある場合は、queue:listen2つの個別のファイルを作成する必要があります。1つは、キューをリッスンし、新しいジョブごとに2番目のファイルを実行するデーモンです。次に、2番目のファイルは、タスクの実行者として機能します。


worker.php


<?php

require_once __DIR__.'/bootstrap/bootstrap.php';

use Illuminate\Queue\Listener;
use Illuminate\Queue\ListenerOptions;

//  ,   " ",  Laravel  artisan,        work.php.
// https://github.com/laravel/framework/blob/6.x/src/Illuminate/Queue/Listener.php#L72
define('ARTISAN_BINARY', 'work.php');

$worker = app(Listener::class, ['commandPath' => __DIR__]);

$worker->setOutputHandler(function ($type, $line) {
    echo $line;
});

$worker->listen('default', 'default', new ListenerOptions);

require_once 'bootstrap/bootstrap.php';

$worker->runNextJob('default', 'default', new WorkerOptions);
