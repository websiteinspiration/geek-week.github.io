Crawc.net - Erstellen und Gestalten Ihrer Medienchats

Ich möchte über mein Projekt sprechen. Dies ist ein Chat oder vielmehr eine Medien- Chat- Plattform. Dies bedeutet, dass Sie hier Ihren eigenen Einraum-Chat mit Unterstützung für Medieninhalte in Nachrichten erstellen und erstellen und über einen Iframe in Ihre Website einfügen können. Das Projekt hat erst kürzlich die Beta-Phase verlassen.



Mir wurde oft gesagt, dass ich ein Fahrrad schreibe, aber das ist im RuNet noch nie passiert. Es gibt eine Reihe von Chatrooms, die jedoch nicht einmal die Hälfte der Funktionalität von Kravcha bieten , aber unglaublich schlecht aussehen. Ich habe viel Zeit mit Design und Benutzerfreundlichkeit verbracht, damit die Kommunikation so angenehm wie möglich war.











Lassen Sie uns ein wenig über die Funktionalität sprechen



Dies ist ein halb anonymer Chat, für den keine Überprüfung der Identität per Telefon oder E-Mail sowie andere persönliche Informationen erforderlich sind. Für die Registrierung reicht es aus, den gewünschten Benutzernamen und das gewünschte Passwort anzugeben. Sie können jedoch auch Ihr Alter und Ihren Wohnort angeben und kurz über sich selbst schreiben - diese Informationen zusammen mit Ihrem Fotoalbum wird in Ihrem Profil auf der Website angezeigt. Die Idee von Kravcha ist es, den Kanal des Raums und seinen Chat in einem Sichtbarkeitsbereich zu kombinieren. In demselben Bereich können Sie dieselben Profile anzeigen, sodass das Anzeigen von Profilen oder das Lesen des Kanals Sie nicht vom Gespräch ablenkt. Auf dem Kanal können Sie thematische Nachrichten, Chat-Regeln, Medieninhalte und alles, was für die Fantasie ausreicht, veröffentlichen, da die Beiträge des Chatrooms HTML unterstützen (ein kleiner visueller Editor wurde für Benutzer hinzugefügt, die weit von der Webentwicklung entfernt sind).Kanalnachrichten können farbig oder sogar zweifach hervorgehoben werden - Textfarbe und Hintergrundfarbe. Sie können auch detailliertere Einstellungen vornehmen, indem Sie auf Optionen klicken und die Stile Ihres Raums manuell bearbeiten. Dort können Sie den Namen des Raums ändern und ein Passwort festlegen, um ihn zu schützen.



Nun zum Chat selbst



Textnachrichten können in Nachrichten formatiert werden: Auf diese Weise können Sie Text mit Farbe hervorheben, die Schriftgröße angeben, ein Zitat, einen Code usw. einfügen (eine ausführlichere Dokumentation zur Nachrichtenformatierung finden Sie hier oder auf dem Kanal im Hauptchatraum ). Darüber hinaus können Sie in den oben genannten Nachrichten Medieninhalte veröffentlichen: Sie können sie entweder als Referenz oder durch Herunterladen von der Festplatte aufrufen. Sprachnachrichten werden unterstützt. Es gibt zwei Arten von Emoticons: eine kleine Sammlung von Fontello-Schriftarten und eine riesige Bibliothek japanischer Emoticons von jemoticons.com







Medikamente und Tabs



Sowie im sozialen. Netzwerke gibt es private Nachrichten. In einem Fenster können Sie mehrere Registerkarten mit Chats und mehreren persönlichen Konversationen öffnen (außerdem ist ihre Anzahl theoretisch nicht begrenzt). Außerdem wird auf der Registerkarte die Anzahl der neuen ungelesenen Nachrichten angezeigt. Wenn Sie zwischen Registerkarten wechseln, ändert sich Ihr URL-Hash: Wenn Sie diesen Hash in die Adressleiste schreiben, können Sie in einen bestimmten Chat oder eine bestimmte persönliche Korrespondenz gelangen (z. B. https://crawc.net/ru/#anime , https: // crawc). net / ru / # emo , https://crawc.net/ru/#2X2FAN- Verwenden derselben Links, um die Möglichkeiten benutzerdefinierter Stile zu demonstrieren. In der persönlichen Korrespondenz wie in einem Chatroom können Sie das Profil der Person anzeigen, mit der Sie sprechen - sie befindet sich rechts im Dialog. GIFs werden in Benutzeravataren, ihren Fotoalben und in den Nachrichten selbst unterstützt (die Korrespondenz mit Gifs sieht ziemlich cool aus).







zusätzlich



Es gibt eine interaktive Suche nach Chatrooms. Es werden mehrere Sprachen unterstützt (derzeit nur Russisch und Englisch). Zusätzlich zum Avatar kann der Benutzer ein persönliches Symbol aus dem Unicode-Set haben, das in der Chat-Benutzerliste angezeigt wird. Dies geschieht, damit Benutzer Gruppen beitreten können. Außerdem können Sie auf der Website Quests in Form von Text-Minispielen erstellen und ASCII- und Pixel-Art zeichnen.

Eine weitere Funktion: Sie können mehrere Bilder gleichzeitig auf einer Skala anzeigen, und auf jeder Registerkarte können sie unterschiedlich sein, was sehr praktisch ist.

Fazit



Das Überprüfen des Codes wäre sehr langweilig und ich weiß nicht, worauf es sich zu konzentrieren lohnt. Damit der Chat funktioniert, ist der gesamte Code wichtig. Sie können keine einzelnen Zeilen auswählen und andere unbeaufsichtigt lassen. Sie können sich mit dem Code des Kunden vertraut machen, indem Sie darauf klicken , während er noch in offener Form ist. Ich veröffentliche den Webanwendungsservercode aus offensichtlichen Gründen nicht. Obwohl es in Zukunft möglich ist, werde ich diesen Schritt in Richtung Open Source machen, oder ich werde Client-Skripte durch den Obfuscator führen.
