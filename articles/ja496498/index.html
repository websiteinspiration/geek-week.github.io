これまでにMinecraftをプレイしたことがありません。さらに言えば、彼は私にとって特に面白かったわけではありません。おそらく、私は彼の考えを理解していなかったためです。はい、私はその中にあなたが無限の数の構造を構築し、さまざまな種類の植物や動物を育て、メカニズムを構築できることを見ました。しかし、私は子供たちがプリミティブなグラフィックでこのゲームを終日丸一日過ごす方法を理解できませんでした。

しかしある晴れた日、8歳の息子は友達がほとんどなく、Minecraftをプレイする相手がほとんどいないと不満を漏らしました。私は彼がより多くの友達を見つけ、ネットワーク上のゲームの問題を解決するのを助ける方法を考え始めました。 RealmsというMinecraftネットワークゲームの代替バージョンがあることを知っていました。これは、独自のサーバーを作成できる有料サブスクリプションであり、Minecraftはホスティングサービスとして機能します。このオプションの欠点は、そのようなサーバーで最大10人がプレイできることです。 MODをインストールする方法もありません。元のMinecraftコンテンツを変更または補完するサードパーティの開発者によるアドオンです。 modのないそのようなサーバーはバニラとも呼ばれます。

したがって、少しトピックを勉強したので、子供に興味深い地図が付いたサーバーを取り上げ、そのアドレスをインターネットに配置して、他のプレイヤーがアクセスできるようにすることを提案しました。それが私の息子のために一緒に遊ぶ友達を見つける計画でした、そして私自身、これは私が長い間考えていた追加の収入源として、新しいプロジェクトにとって素晴らしい選択肢であると決めました。

Minecraftには2つのバージョンがあります。 1つはモバイルデバイスとWindows 10用に設計されたPocket Edition、もう1つはPC用のJava Editionです。結局のところ、それらは互いに互換性がないため、ゲームの世界にはより多くのオプションがあり、それに加えてプラグインの幅広い選択肢があるため、私はJava Editionを選択しました。まあ、私もJava開発者だからです。

ホスティングには、DigitalOceanを選択しました。私はすでにそれを使用した経験があり、液滴を作成するのに数分かかります。あなたにとって便利なホスティングを使用できます。

はじめに、2 CPUと2 GB RAMのための十分な構成があります。 1つのCPUを試しましたが、サーバーが1GBのRAMで非常に長い時間起動しました。大きなカードをインポートすると、サーバーがクラッシュすることがあります。



液滴を作成するときに、私は認証方法を選択しました-SSHキー



作成手順に従いました



これで、仮想サーバーの準備が整いました。



私たちはsshでそれに行きます：

ssh -i ~/minecraft root@IP_address

ここで〜/ minecraftは作成された秘密のSSHキーへのパス、IP_addressはホスティングから提供されたIPアドレスです。

rootユーザーを使用してサーバーを起動することはお勧めしません。これにより、脆弱なプラグインを介して攻撃者がサーバーに損害を与える可能性があります。新しいユーザーを追加する方法については、リンクをご覧ください。

つまり、ユーザーを作成します（dinikinを自分のユーザーに置き換えます）。

adduser dinikin


そして彼に管理者権限を与えます：

usermod -aG sudo dinikin


およびSSH経由のリモートアクセス権：

rsync --archive --chown=dinikin:dinikin ~/.ssh /home/dinikin


新しいセッションを開始し、新しいユーザー名でSSHを使用します。

ssh -i ~/minecraft dinikin@your_server_ip

最初のMinecraftサーバー用のフォルダーを作成し、そこに移動します。

mkdir PirateBay
cd PirateBay

次に、サーバーの実装を決定する必要があります。いくつかありますが、最も人気のあるのはスピゴット、ペーパー、フォージです。これらのサーバーの違いについて詳しくは、こちらをご覧ください。

ネイティブのバニラサーバーとの主な違いは、元のサーバーへの変更により、とりわけサードパーティのプラグインの起動が可能になり、ゲームの世界の機能が大幅に拡張されることです。このようなプラグインを使用すると、ミニゲームの作成、プレーヤーの制限の入力、サーバーのネットワークへの統合などが可能になります。

最初にSpigotをダウンロードしてインストールしました。これは、Minecraftで最もよく使用され最適化されたサーバーです。多くのプラグインが開発されており、開発者とユーザーの両方からなる巨大なコミュニティがあります。

ただし、マップを編集するためのプラグインの1つを使用すると、プラグインが正しく機能せず、ログのSpigot に基づいてPaperをより最適化されたフォークとして使用することを推奨するという事実に直面しました。

結局、私はPaperを選択しましたが、一部のサーバーインスタンスではSpigotを離れました。次に、選択したサーバーをインストールして起動する必要があります。最新のサーバーバージョンをダウンロードします。

curl -kLJ https://papermc.io/api/v1/paper/1.15.2/165/download -o Paper-1.15.2.jar

そして実行してみてください

java -Xms1G -Xmx1G -XX:+UseConcMarkSweepGC -jar Paper-1.15.2.jar

Javaがインストールされていないため、サーバーが起動しませんでした：
