Guide de test de sécurité Web mis à jour par le consortium OWASP



Le projet Web Security Testing Guide (WSTG) est la principale méthodologie de test de sécurité pour les développeurs d'applications Web et les professionnels de la sécurité de l'information. Il est développé par le consortium international OWASP.

Le projet de sécurité des applications Web ouvertes (OWASP) est un projet de sécurité des applications Web open source. La communauté OWASP comprend des entreprises, des organisations éducatives et des particuliers du monde entier. La communauté travaille à la création d'articles, de manuels, de documentation, d'outils et de technologies librement accessibles.

WSTG est un guide complet pour tester la sécurité des applications Web et des services Web. Le WSTG, un effort collaboratif de professionnels de la cybersécurité et de bénévoles, fournit un ensemble de meilleures pratiques utilisées par les pentesters et les organisations du monde entier.

La version actuelle - [Version 4.1] - 2020-04-2, la version précédente - [Version 4.0] - 2014-09-17 - environ 6 ans se sont écoulés entre les mises à jour, qui devraient couvrir une large couche de technologies modernes, cependant, des changements fondamentaux ne seront attendus que dans 5 versions de WSTG.
