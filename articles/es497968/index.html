Recursos para desarrolladores web: API y elementos de la interfaz de usuario

En nuestro blog, preparamos varios resúmenes con referencias de código abierto de servidores Linux para administradores de sistemas ( uno , dos ). Hoy hablaremos sobre herramientas abiertas para desarrolladores web.

Debajo del corte hay recursos con iconos vectoriales y SVG y API.


Fotos - Hal Gatewood - Unsplash


API públicas


Este es un agregador de API abiertas de servicios desarrollados por grandes compañías de TI como Google, TripAdvisor y Mailchimp, así como entusiastas de GitHub. El autor de la plataforma es el desarrollador front-end Mohd Danish, quien también presentó Tweet Jobs, un servicio de búsqueda de empleo .

Lo que es interesante : la biblioteca tiene más de 680 API. Todos ellos se dividen en 50 categorías, que incluyen: medicina , clima , noticias , seguridad de la información , datos abiertos , almacenamiento en la nube , aprendizaje automático . La lista de interfaces de software tiene herramientas para procesar el habla natural ,Raspado , automatización GitLab y otros. La búsqueda te ayuda a navegar. La API se puede buscar por palabras clave y por la empresa de desarrollo.

Atención : la base se repone bastante lentamente, pero puede sugerir independientemente las API faltantes. No hay una especificación de interfaces de software en el sitio, y para obtener más información sobre la herramienta, debe abrir la documentación.

Bonificación : Danish también lanzó NoCodeAPI , listo para usar sin modificaciones en el código de back-end, por ejemplo, lo que le permite convertir XML o RSS a JSON o configurar ganchos en Slack.


Abra la interfaz de usuario


Una lista con bibliotecas de componentes (botones, casillas de verificación, botones de opción) para páginas web. El recurso es supervisado por ingenieros de WICG (Web Platform Incubator Community Group), que pertenece al consorcio W3C y diseña estándares para la web. Los autores están convencidos de que con la interfaz de usuario abierta, los desarrolladores no reinventarán la rueda y perderán el tiempo recreando componentes.


Captura de pantalla: Abrir la página de UI / Botón

Lo interesante : Se presentan los principales sistemas de diseño: Ant Design, Atlaskit, Carbon Design System, FastDNA, Lightning Design System y muchos otros.

Atención: Para buscar información adicional sobre cada sistema de diseño, debe ponerse en contacto con la fuente. Los enlaces correspondientes se colocan en la parte superior de la página, por lo que debe rebobinarlo regularmente. El recurso en sí no está completo: los autores complementan la información sobre los sistemas de diseño para botones de selección , controles deslizantes y tablas .

Qué más : los que lo deseen pueden contribuir y documentar cualquier colección de activos para diseñar interfaces de usuario multiplataforma. Las instrucciones y requisitos se dan en el sitio .


Flaticon


Biblioteca de iconos vectoriales de los desarrolladores del archivo fotográfico Freepik. Se complementa con 370 autores independientes . El sitio ofrece más de 500 mil imágenes en formato PNG, SVG, EPS y PSD.


Captura de pantalla: Flaticon / Página principal

Ventajas : Los iconos se dividen en 10 mil conjuntos, lo que simplifica la búsqueda. Entre las categorías populares se pueden distinguir: construcción , vestimenta , mascotas , incluso hay un coronavirus , cuarentena y situaciones de emergencia . Antes de descargar imágenes, puede cambiar su color en un editor especial , para toda la colección de una vez o individualmente. También ofrece Flaticongenerador de adornos : imágenes repetidas que se pueden usar como telón de fondo en el sitio.

Desventajas : para acceder a todas las funciones del sitio, es necesario registrarse. No todos los íconos son gratuitos, pero para el contenido premium debes pagar 10 euros al mes. Las imágenes gratuitas solo se pueden usar con atribución.

Bonificación : Otro recurso con imágenes SVG gratuitas para desarrolladores web es Sparkk . Para pegar los íconos en el sitio, simplemente copie el código HTML apropiado.

En 1cloud.ru ofrecemos el servicio Cloud Storage . Es conveniente guardar copias de seguridad, archivos y documentos corporativos.
Los sistemas de almacenamiento se basan en tres tipos de unidades: HDD SATA, HDD SAS y SSD SAS. Su volumen total es de varios miles de terabytes.
