Bagaimana saya tidak menjadi manajer produk setelah pelatihan di Product Univercity Moreynis dan Chernyak

Halo semuanya.

Saya memutuskan untuk menulis artikel pertama saya di VC, tetapi pemimpin redaksi melarang saya karena saya meminta untuk menambah teman, dan sebelum itu, tiga kali kerja murah menghilangkan publikasi saya untuk beberapa alasan yang tidak diketahui, dan menggambarkan pengalaman saya dalam mencari pekerjaan sebagai manajer produk dan pelatihan di universitas produk.

Artikel itu ditulis pada musim panas 2019, pelatihan pada musim dingin 2018, tetapi hanya hari ini saya menerbitkannya (ada alasannya).

Secara singkat dengan anamnesis apa saya datang untuk belajar


Secara umum, berdasarkan profesi saya adalah analis bisnis / sistem. Tetapi pada tahun 2016, saya digigit oleh startup startup, dan saya memutuskan untuk menjadi startup dengan smoothie dan rekan kerja. Untuk melakukan ini, saya belajar bahasa dan kerangka kerja yang paling baru - ruby ​​on rails, muncul dengan ide dan mulai menerapkannya. Jadi ada sebuah proyek - pasar c2s dari koki rumah atau uber untuk koki seperti yang saya sebut saat itu. Saya melakukannya dalam kacamata merah muda, itu adalah waktu terbaik dalam hidup saya. Bekerja di pagi hari dan minggu, berkode, bersukacita dan semua itu. Pada musim panas di tahun yang sama saya pergi untuk kelaparan, membuat presentasi dengan gaya AirBnB awal, tentu saja, keluar dari tempat saya dan semua itu.

Secara umum, pada pitch pertama, "investor" menggulung saya dan setelah setengah jam menawari saya 2 juta sebagai imbalan untuk 20% dari perusahaan saya yang tidak ada. Secara umum, dia masih berbicara banyak tentang sesuatu, tetapi pada akhirnya tidak memberi uang, tetapi terus-menerus menuntut daya tarik. Akibatnya, saya membawa produk untuk dirilis, menerima pelanggan pertama, pergi ke spbstartupday, memenangkan nominasi "start-up terbaik untuk penduduk kota", saya tidak mendapatkan uang untuk iklan, saya tidak menerima investasi, tetapi saya diundang untuk melakukan proyek serupa di bidang teknologi makanan. Kali ini dengan investasi dan bahkan gaji. Saya juga diundang untuk belajar dalam korespondensi IIDF dan mendengarkan hal-hal yang sangat berguna selama dua hari, tetapi tidak pergi ke korespondensi itu sendiri, tetapi dalam dua hari kacamata startup saya jatuh, dan kebenaran yang mendalam tentang kehidupan dan kekecewaan telah dimulai.

Secara umum, saya meluncurkan startup kedua, melakukan penjualan pertama, menghitung unit ekonomi dan ... Saya menyadari bahwa tidak ada prospek. Kiri dan kembali untuk menyewa. Hipotek budak tidak memungkinkan saya untuk mencari pekerjaan sebagai produk saat itu, meskipun saya menginginkannya dan tidak ada banyak sensasi seperti sekarang. Sekitar waktu yang sama, saya menjalani NetX UX dan pelatihan Manajemen Produk. Kemudian saya lepas, proyek-proyek blockchain gila dengan dana bagus, dan di sini pada tahun 2019 setelah 2 tahun, saya memutuskan bahwa saya masih perlu mendapatkan produk, karena seperti T-Shaped, cowok juga diperlukan di sana-sini.

Saya baru saja menemukan tawaran yang menggoda untuk belajar dengan pembayaran tentang fakta pekerjaan dari universitas produk. Fantasi!!!

Tentang belajar


Di pintu masuk pelatihan dengan mengorbankan sumber daya media, sangat, sangat banyak orang. Dalam aliran saya ada sekitar 700. Pada awalnya ada webinar pengantar di mana semuanya diceritakan dan kemudian penelitian itu sendiri dimulai.

Bagi saya, pelatihan tampak seperti konveyor liar. Banyak materi, tenggat waktu yang sulit - Anda harus melaporkan setiap hari, tetapi yang paling penting, tenggat waktu hanya berdasarkan tanggal. Tidak ada verifikasi konten, mis. Anda dapat menerima omong kosong terus terang dan semuanya akan baik-baik saja. Sepertinya norma, saya memutuskan untuk memperbarui pengetahuan lama saya dan akhirnya mendapatkan pekerjaan. Ngomong-ngomong, saya berada dalam kondisi yang sangat tidak menguntungkan: di kota provinsi kecil, di mana perusahaan makanan dapat dihitung dengan jari di satu sisi, dan tidak dipekerjakan secara resmi (ada pendapatan pihak ketiga).

Secara umum, ketika mereka menunjukkan kontrak kepada kami, kami harus bergabung, tetapi untuk beberapa alasan saya tidak mendengarkan diri saya sendiri. Akibatnya, pada satu akhir pekan saya keluar retret selama tiga hari, dan saya tidak punya waktu untuk memenuhi tenggat waktu, dan mereka mengusir saya. Namun kewajiban berdasarkan kontrak tetap ada, tetapi tidak ada materi tentang materi yang diteruskan ...

Bagaimana saya mendapatkan produk


Secara umum, di kota kecil saya, kebutuhan saya untuk menghabiskan uang menurun tajam, dan saya perlahan-lahan menghabiskan parasut saya (keuangan yang ditangguhkan), dan memantau lowongan. Untungnya, satu lowongan masih muncul. Satu, Karl! Saya berhasil menanggapinya, tetapi tidak menerima tanggapan.

Itu pada bulan November ... Saya memanggil perekrut, menerima jawaban bahwa resume saya sedang dipertimbangkan oleh kepala, dan mereka akan menjawab saya besok. Jawabannya menarik - lowongan menghilang dengan hh. Saya mencetak gol sampai musim semi, ketika tiba-tiba dia muncul lagi. Kali ini saya memutuskan untuk mencari pembuat keputusan di dalam perusahaan, dan menjalaninya. Anehnya, semuanya ternyata, melewati 3 tahap wawancara, tes sukses dan semua itu. Saya menerima tawaran dengan gaji sepertiga kurang dari minimum yang saya minta ... di sini mereka adalah kenyataan pahit di provinsi ini. Tentu saja dia menolak. Kebanggaan saya tidak memungkinkan saya untuk mendapatkan sebanyak yang saya dapatkan dalam satu minggu paruh waktu per bulan.

Secara umum, bukan takdir untuk menjadi produk dalam perekrutan, dan karena sudah waktunya untuk bekerja, hampir secara kebetulan ia direngut oleh seorang analis. II, di bawah ketentuan kontrak, saya berutang setengah dari gaji saya, meskipun PU tidak ada hubungannya dengan mencari pekerjaan sebagai seorang analis. Ini adalah pengalaman masa lalu saya, dan pengetahuan produk tidak terlalu dibutuhkan di sana.

Secara umum, saya harus memikirkan apakah akan memenuhi kewajiban saya ... Dan Anda tahu, saya membayar. Bukan karena saya takut akan pembalasan, tetapi agar saya mendapatkan pengalaman ini, sehingga lain kali saya tidak akan menandatangani kontrak yang jelas tidak menguntungkan bagi saya. Saya juga dapat menghapus beberapa kisah karma, tetapi saya tidak akan melakukannya.

Secara umum, saya tidak menjadi produk dalam mempekerjakan, saya membayar uang, saya tidak menerima bahan apa pun.
