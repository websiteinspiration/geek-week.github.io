注意を引く可能性が最も高い見出しまたはHabraHabr

こんにちは、また何か役に立つものを共有したいという願望がありました！主婦の利益に関する記事はHabrには届かなかったので、habrhabr.ruがデジタルエージェンシーを宣伝するために特別に作成した記事を4年前に探し出しました。今回はもっと多くのアドバンテージがあることを願っています。

考えたことはありますか？間違いなく、インターネットで情報を検索するときは、見出しの一部をすぐにスキップします。通常、最後まで読みません。

単に面白くないからです。したがって、記事の作成者、ブロガー、コピーライターは、最初の言葉（言葉ではない場合でも！）これは明らかに常にそうであるとは限らないことがわかり、見出しを書くための黄金律はありません。しかし、いくつかの一般的な形式や種類の見出し構造が最も一般的です。

サイトgeektimes.ruとhabrahabr.ruのトップニュース（2008〜2016年）に100の見出しを選択し、同様のチェーンと論理的な接続を強調するようにしました。下記の8つのカテゴリが判明しました。

I）何かのX。ただし、 Xは、数個などの量的特性を持つ単語、または比較特性、特に最も多い単語に置き換えることができます。

  1. あなたがやめる必要がある8種類のサイト。
  2. いくつかの便利なサービス。
  3. 世界最小のゲーム（58バイト）。

II）過去の時制での私/彼/誰かが何かをしたなど、それを行う方法。さらに、個人的な/エイリアンの経験と「どのように/なぜ/何を知っている/必要なのか/なぜ私/彼/彼ら...動詞」などの見出し。ほとんどの場合、疑問符はありません。記事自体が質問に答えます。このフォーマットへのコンテキストはオプションです。また、レッスン/経験/ヒント（...）または履歴[方法について...]の形式にすることもできます。

  1. 3Dプリンターの購入方法
  2. Googleを離れた理由
  3. Mail.Ruがスパイウェアを開発しているのはなぜですか？
  4. Microsoft Internet Explorer 11が私たちに起こりうる最悪の事態であるのはなぜですか。
  5. 今日ICQに何が起こったかについて（パート1）
  6. リンゴの木とリンゴの木は何を理解しなかったのですか？

!!! このタイプに関連する意味：

  1. プログラムを購入しましたか？さあ... [プログラムの購入方法...];
  2. インターネットリソースのセキュリティ-サイトをハッキングするrospil.info [ハッキングしたとおり...];
  3. 古いブラウザの拒否[私が拒否したように...]。

III）反対のタイプの構造、否定 

  1. 動物を正しく書いていない=動物を正しく書く方法。
  2. スマートフォンの充電方法＝スマートフォンの充電方法。

ただし、「レッスン」という言葉で始まるフレーズには注意を払うことが重要です。それは、それ自体が推奨事項を意味するのではなく、何かの結果として得られたイベントや経験の結果について伝えるためです。この例は、「宇宙災害からの教訓」という見出しです。

IV）質問はステートメントに置き換えられます -「方法」の代わりに、タイプは、Do / Createの形式の動詞、または名詞Creation / Developmentなどで使用されます。これにはDIYタイプも含まれます-「自分で/自分の手で/ゼロから/即興の手段から。ただし、これらの単語は省略できますが、タイトルには欠落している箇所が明確に示されています。

  1. 古い液晶モニターからプライベートモニターを作ります。
  2. 18速ガウスガン;
  3. ある夜の強力な日曜大工のレーザー。
  4. ぼやけたぼやけた画像を復元します。品質を向上させます。
  5. 原子炉はゼロから家です。

V）...についての事実/機能/ ...の紹介/ ...に関するメモ

  1. プログラミングに関するインフルエンザA（H1N1）について
  2. SIMカードの簡単な紹介。
  3. サイト設計者へのメモ。
  4. ロシア開発の特徴。

ただし、見出し「半導体エレクトロニクス」および「宇宙および軍事用マイクロエレクトロニクス」、または「死んだ宇宙ステーションを救うためのソビエト作戦」の場合のように、導入フレーズは意味を失うことなく省略できます。

VI）純粋にニュース/サブサイト。通常、これらはおめでとうございます、サイトの変更に関するメッセージ、コンテンツの表示、警告などです。

  1. サービスXXXXXX.ruは一時的に閉鎖されています。
  2. 公式Habrahabraモバイルアプリ。
  3. 注意！XXX.RUハッキング！

VII）レビュー、評価、試乗、レビュー、伝記。
ヘッダー形式：多くの場合、オブジェクト自体。重要：レビューの完全性/深さ、本格的な結論または評価の存在、イベントへのコンテキストバインディング。

  1. Apple iPadの概要。
  2. CPU;
  3. 内観：CDおよびHDD;
  4. 私が知っていたスティーブジョブズ。

VIII）提案/行動を促す積極的な行動。これには、「感情について」、「怒っている」と書かれた記事も含まれます

  1. ねじるのをやめなさい！;

  3. プログラマをいじめないでください。
  4. QIP-憎悪の分（サーバー上の履歴）。

  1. Voronezhetsは銀行と契約を結び、彼自身の変更を行い、2400万ルーブルを提訴する予定です。
