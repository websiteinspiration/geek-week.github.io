PATH実行可能ファイルが検索されるパスを格納する環境変数です。シェルに実行を依頼するとls、最初にでls指定されたすべてのパスで実行可能ファイルが検索されPATHます。

$ echo $PATH
/usr/local/bin:/usr/bin:/usr/local/sbin:/usr/sbin:/home/v.pushtaev/.local/bin:/home/v.pushtaev/bin
$ which ls
/usr/bin/ls

この例では、パスはPATHを使用して区切られてい:ます。混乱は発生しません。パスにが含まれている場合、:では使用できませんPATH。

ただし、これはすべてのオペレーティングシステムに当てはまるわけではありません。Pythonでは、次のコマンドを使用して、OSの正しい区切り文字を見つけることができますos.pathsep。

Python 3.5.0 [...] on win32
Type "help", "copyright", "credits" or "license" for more information.
>>> import os
>>> os.pathsep
';'

混同しないでくださいos.pathsepとos.path.sepファイルパスの区切りであります：

>>> os.path.sep
'/'



正規表現を読みやすくするために、フラグを使用できますre.VERBOSE。これにより、好きな場所でスペースを追加したり、文字の後にコメントを追加したりできます#。

import re

URL_RE = re.compile(r'''
    ^
    (https?)://
    (www[.])?
    (
    (?: [^.]+[.] )+
    (   [^/]+    )  # TLD
    )
    (/.*)
    $
''', re.VERBOSE)

m = URL_RE.match('https://www.pythonetc.com/about/')
schema, www, domain, tld, path = m.groups()

has_www: bool = bool(www)

print(f'schema={schema}, has_www={has_www}')
print(f'domain={domain}, tld={tld}')
print(f'path={path}')

re.Xのエイリアスですre.VERBOSE。

表記a : b : cは、slice(a, b, c)ブラケットのみを使用して定義するために使用できます。

>>> [1, 2, 3, 4, 5][0:4:2]
[1, 3]

関数の引数としてスライスオブジェクトを渡す場合は、明示的に定義する必要があります。

def multislice(slc, *iterables):
    return [i[slc] for i in iterables]


print(multislice(
    slice(2, 6, 2),
    [1, 2, 3, 4, 5, 6, 7],
    [2, 4, 2, 4, 2, 4, 2],
))

これは、同様の関数ををサポートするオブジェクトに変換する方法です[a : b : c]。

from functools import partial


class SliceArgDecorator:
    def __init__(self, f):
        self._f = f

    def __getitem__(self, slc):
        return partial(self._f, slc)

slice_arg = SliceArgDecorator


@slice_arg
def multislice(slc, *iterables):
    return [i[slc] for i in iterables]


print(multislice[2:6:2](
    [1, 2, 3, 4, 5, 6, 7],
    [2, 4, 2, 4, 2, 4, 2],
))



__getattribute__-これは、適切なときに委任パターンを簡単に使用できる強力なツールです。したがって、比類のないオブジェクトと比較する機能を追加できます。

class CustomEq:
    def __init__(self, orig, *, key):
        self._orig = orig
        self._key = key

    def __lt__(self, other):
        return self._key(self) < self._key(other)

    def __getattribute__(self, name):
        if name in {'_key', '_orig', '__lt__'}:
            return super().__getattribute__(name)

        return getattr(self._orig, name)

class User:
    def __init__(self, user_id):
        self._user_id = user_id

    def get_user_id(self):
        return self._user_id


def comparable(obj, *, key):
    return CustomEq(obj, key=key)


user1 = comparable(User(1), key=lambda u: u.get_user_id())
user2 = comparable(User(2), key=lambda u: u.get_user_id())

print(user2 > user1)  # True
print(user2 < user1)  # False
print(user2.get_user_id())  # 2
