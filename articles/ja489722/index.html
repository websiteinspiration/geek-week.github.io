LCDディスプレイに値を表示するArduinoコントローラーの温度および湿度メーター

最近、私はArduinoマイクロコントローラーのプログラミングを研究し始めました。基本的な演習（LEDの点滅、ディスクリートおよびアナログ入力の操作、LCDディスプレイへの情報の表示など）を完了した後、私はある種の家庭用の便利なデバイスを作りたかったのです。

LCDにパラメータを表示しながら、室内に温湿度計を作ることにしました。また、室内の照明レベルに応じて明るさが変化するアダプティブディスプレイバックライトも作成しました。これにより、デバイスの消費電力を削減し、バッテリーの寿命を延ばすことができます。

このプロジェクトでは、次のコンポーネントを使用しました

。1. Arduino Unoボード



2. I2Cプロトコルサポートモジュールを備えたLCD1602ディスプレイ。ワイヤの数を減らしてディスプレイの接続を簡略化するために、I2Cプロトコルを介して接続することにしました。これは、特別なコンバーターモジュールをLCDディスプレイにはんだ付けしたためです。

"

3.温度および湿度センサーDHT11。20〜80％の湿度と0〜50 0Cの温度を測定できます。センサーには4つの出力がありますが、3つしか使用されません。電源出力とデータ出力の間に10kオームの抵抗を取り付ける必要があります。I私はあなたが安全にArduinoのボードに接続することができますので、既製のセンサーは、接続された抵抗を持つ基板上に実装されました。

」

4.フォトレジスター。 10kΩの抵抗を介して、GNDへのプルアップでArduinoボードに接続されています。



接続図は次のとおりです。



フォトレジスタは、Arduinoボードのアナログ入力A0に接続されています。 DHT11センサーデータラインはディスクリートピン2に接続され、電源ピンはそれぞれ+ 5V Arduinoに接続され、GNDピンはArduinoボードのグランドで接続されます。 LCDディスプレイはボード電源に接続され、SDAピンはA4に接続され、SCLピンはA5に接続されます。バックライト輝度制御ラインは、PWM信号が生成されるピン9に接続されています。

以下はソースコードです：

#include "DHT.h"                  //      DHT11
#include "Wire.h"               
#include "LiquidCrystal_I2C.h"    //     LCD 

int LD;                           //  

#define DHTPIN 2                  // PIN    DHT11
#define LED 9                     // PIN   -
#define FOTO 0                    // PIN   

DHT dht(DHTPIN, DHT11);           //  DHT11
LiquidCrystal_I2C lcd(0x27,16,2); // 


void setup() {
 pinMode(LED, OUTPUT);            // PIN LED  
 
 dht.begin();                     //    
 
 lcd.init();                      // LCD 
 lcd.backlight();
}


int light(int svet)               //  
{                                 //    
  if (svet<25)                    //  -
  {
    return 255;
  }

  if(svet>1020)
  {
    return 2;
  }
  else
  {
    int L=(int)(-0.2*svet+261.262);
    return L;
  }
}


void loop() {
   int f = analogRead(FOTO);              //    
   LD=light(f);
   delay(2000);
   analogWrite(LED, LD);                  //   
   
   float h=dht.readHumidity();            //  
   float t=dht.readTemperature();         //  

    lcd.setCursor(0,0);                   //      
    lcd.print("Temp ");                   //  
    lcd.print(t);
    
    lcd.setCursor(0,1);                   //      
    lcd.print("Hum ");                    //  
    lcd.print(h);

}

このコードをコンパイルすると、このような機能するデバイスが得られます。
