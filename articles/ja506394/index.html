最初の会議



300x300 mmの印刷テーブルは印象的です。大量の印刷を行わない場合でも、このようなテーブルで複数の小さなモデルを同時に印刷して、時間を大幅に節約できます。

型押しされた鉄骨は厚く、重く、実用的です-欲望の限界。

電源、マザーボード、ほとんどの内部コンポーネントは、プリンタの底部にある頑丈な金属製のハウジングに収納されています。これにより、プリンターの占有面積を大幅に削減することができました。

しっかりとしたベースのおかげで、プリンターを別の場所に移動した場合にプリンターを調整する必要はほとんどありません。

嬉しい驚き-Mega Xにはフィラメントエンドセンサーがあり、これは通常、3Dプリンターのより高価なモデルでのみ見られます。このセンサーは、糸が切れたり切れたりすると、印刷を一時停止します。この機能は、大規模なプロジェクトに非常に役立ちます。

そして、テーブルを水平にする素晴らしいペン！印刷プラットフォームを手動で調整した人は誰でも、小さなレベルのノブを回すのがいかに不便かを知っています。一方、プラットフォームを下から手で引っ掛けることができ、すべての労力が無駄になっています。 Anycubicがこの問題を修正しました-Mega Xのハンドルが大きく、レベリングスクリューのねじピッチが小さい。ハンドルは非常に回転しやすく、ほぼ1本の指で操作できます。

テーブルの表面はコーポレートガラスでできており、上部にメッシュの薄い層が付いているため、接着性に問題はありません。印刷物は印刷面にしっかりと保持されます。

サウンドトラックは少し迷惑です。画面に触れると、大きなガラガラという音が聞こえ、プリンタが起動すると、標準のサウンドクリップが再生されます。幸いなことに、これらのサウンドは簡単に一度だけオフにすることができます。

アセンブリ



組み立ては非常に簡単です。いくつかのネジを締めてケーブルを接続します。

Mega Xの組み立てにかかった時間はわずか20分です。現在市場に出回っているほとんどの3Dプリンターと同様に、Mega Xには、相互に接続する必要のある別個のパーツはほとんどありません。部品から3Dプリンターを完全に組み立てる必要があった時代は終わりました。

一般的に、組み立ては非常に簡単です。水平バーを固定しているボルトの1つを取り外し（ワイヤーが上にある）、誰かにプリンターの前面を持ち上げて、プリンターの中心に向かってフレームをスライドさせます。次に、両側の4つのボルトを締めます。

フィラメントセンサーホルダーと糸巻きホルダーを取り付けます。

最後に、4つのケーブルを接続します。すべてのケーブルにコネクタが付いているので、接続すると非常に便利です。

作業室のサイズ



すでに述べたように、Anycubic Mega Xの印刷ボリュームは300 x 300 x 305 mmです。これは、大きな印刷領域を持つ他の低予算の3Dプリンターと比較しても、印象的なスペースです。たとえば、同じラインの3Dプリンター（Anycubic Mega S）と比較すると、容量は3倍になり、9リットルから27リットルになりました。

このビルドボリュームの利点は、大きなオブジェクトを印刷できることだけではありません。非常に大きなモデルの複数のパーツまたは複数の異なるモデルを同時に印刷できます。旅行の前に、記事の著者はいくつかのホルダー、ハンガー、マウント、冷蔵庫用のスペアパーツを印刷する必要がありました。これらはすべて、1つの印刷サイクルで印刷し、夜間にジョブを完了することができました。

ベースの寸法と面積



プリンタが置かれているテーブルが十分に深く、プリンタが壁から十分に離れていることを確認します。

他のAnycubicプリンターと同様に、Mega Xコンポーネントは主にデバイスの下部にあります。したがって、プリンターはテーブル上の多くのスペースを占有しません。ただし、コイルホルダーは側面に突き出ているため、これを考慮する必要があります。

プリンターは非常にコンパクトであることがわかりました。幅と奥行きは50 cmですが、Y軸に沿った全ストロークを考慮すると、奥行きの最大距離は約65センチです。プリンタスタンドを選択するときは、この情報を使用してください。

整列



ジャイアントハンドルは、レベリングプロセスを大幅に促進します。

ほとんどの低予算3Dプリンターと同様に、Anycubic Mega Xは印刷プラットフォームの手動調整を提供します。幸いにも、そのような調整ノブを使用すると、これを行うのは非常に簡単です。プリンタをゼロに設定し、四隅すべてにXの形で用紙を挿入し、必要に応じて各ペンを調整します。

ハンドルのデザインとレベリングスクリューの小さなねじピッチにより、手全体を使用せずにプラットフォームに触れることなく、指で簡単に微調整できます。

品質を構築




Mega X 3Dプリンターは、耐久性のある、打ち抜き加工されたスチールパーツで主に構成されており、互いにうまくフィットします。車はとても頑丈に見えます。

モーションシステム設計



2つの円筒形ガイド、Z軸に沿った2つのステップスクリュー、およびY軸に沿った広いプロファイルにより、安定性が向上します。

多くの場合、低予算の3DプリンターはZ軸に1つのガイドしかありませんが、Mega Xには2つ（および2つのモーター）があります。これにより、振動がなくなり、印刷品質が向上します。さらに、幅広で安定したダブルアルミニウムY軸キャリッジが安定性を高めます。

Mega X電源と電子機器を含む大規模な筐体は、プリンタにさらなる安定性を与えます。

電源



Mega X 3DプリンターはMeanWell電源を使用します。その利点：

  • MeanWell電源は、電源の安定性を確保する高品質のコンポーネントから組み立てられています。
  • 電源ファンは、印刷中のみオンになるため、通常、プリンタは通常より静かになります。
  • 電源にはヒューズが付いています。ヒューズは、3Dプリンターの安全を確保するために必要です。

印刷されたテーブル素材



印刷された表の資料は非常に興味深いものです。AnycubicはこれをUltrabaseプラットフォームと呼んでいます。それは、上部に薄いメッシュのあるホウケイ酸ガラスの層（耐熱性と機械的損傷に対する耐性が強化されています）が存在する加熱アルミニウム層で構成されています。

モデルはそのような表面でしっかりと保持するので、ヘアスプレーや接着剤で固定する必要はありません。

押出機



現在 多くのFDM 3Dプリンターと同様に、Mega XにはBowdenタイプの押出機が装備されています。つまり、このモデルでは、フィードメカニズムを備えたハウジングがホットエンドから分離されています。ホットエンドまでのフィラメントは長いチューブを通して供給されます。重いステッピングモーターとケーシングが3Dプリンターのフレームに取り付けられているため、プリントヘッド付きのキャリッジが軽く、印刷速度が向上します。

著者は、押出機のチューブがZ軸のボールスクリューに触れて潤滑剤を消すことができることを好みませんでした。多分あなたはある種のホルダーを考え出すべきです。

インターフェース



少なくともOctoPrintを設定するまでは、Mega Xタッチスクリーンを使用してプリンターを制御し、印刷を開始する必要があります。

まあ、「しなければならない」のように、原理的には便利です。 3.5インチのカラーディスプレイは、かなり広い視野角を提供します。タッチセンサーは敏感で、大きなメニュー項目は押すのに便利です。

各メニュー項目は、アイコン付きのボタンで表されます。メニューを理解する必要があります。実際には、いくつかの項目をクリックすると追加のメニューがポップアップし、他の項目をクリックするとすぐにアクションが実行されます。

たとえば、温度アイコンに触れると温度制御メニューが表示され、冷却アイコンに触れるとすべてのヒーターがオフになります。順方向と逆方向のナビゲーションも少し混乱しています。

ただし、メニューは正しく機能し、ファームウェアの更新時にインターフェイスが改善される可能性が高くなります。いずれにしても、あまり成功していない3Dプリンターユーザーインターフェイスが一般的です。そのため、OctoPrintなどのプリンター管理ソフトウェアがあります。

印刷プロセス



3DモデルのスライスにはCuraスライサーが使用されますが、Anycubic Mega Xは最近リリースされたプリンターであるため、手動で構成する必要があります。

.gcodeファイルをSDドライブの任意のディレクトリに書き込み、コンピューターに挿入します。SD-USBアダプターはプリンターに付属しています。次に、メニューから印刷プロセスを開始します。

プリントのダウンロードに関する小さな問題は、SDカード上の.gcodeファイルがゴミ箱（.Trashesディレクトリ内）で削除されている場合でも、プリンターが再帰的に検索することです。つまり、ゴミ箱にあるファイルを削除しても、SDカードに残っていると、ファイルは印刷画面に表示されたままになります。このため、.Trashesディレクトリを定期的に削除するか、ファイルを保存するたびに新しい名前を選択することをお勧めします。
