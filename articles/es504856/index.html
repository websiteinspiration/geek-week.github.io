1. Oracle proporciona una

nube de segunda generación La infraestructura de nube de Oracle ofrece herramientas y arquitectura que ayudan a las empresas a migrar sin problemas de las instalaciones a la nube, utilizando automatización avanzada y seguridad incorporada para mitigar las amenazas, lo que en última instancia respalda la migración superior y la rentabilidad.

Infraestructura Oracle Cloud diseñada para la innovación. Esto incluye escalabilidad y disponibilidad líderes en la industria, gestión y control integrados y confiabilidad respaldada por SLA de extremo a extremo. La misión en la nube de Oracle se extiende para admitir nuevas tecnologías como inteligencia artificial, aprendizaje automático, Internet de las cosas, blockchain e interfaces orientadas a los humanos.

Los clientes corporativos de Oracle han pasado de experimentar con estas tecnologías a implementarlas para aplicaciones de misión crítica, crear nuevos modelos comerciales y crear nuevos valores comerciales.

Oracle Cloud Infrastructure aborda los desafíos clave de las soluciones en la nube de primera generación que no fueron diseñadas para trabajar con grandes sistemas financieros que son impulsados ​​por cargas de trabajo o aplicaciones que requieren grandes cantidades de datos. Las soluciones en la nube de primera generación se construyeron sobre tecnologías de hace una década, en las que la configuración de rendimiento, seguridad y migración siempre llegaba tarde. Oracle Cloud Infrastructure de próxima generación se ha diseñado específicamente para satisfacer las necesidades de la empresa actual, ofreciendo un rendimiento más rápido y más predecible, mejor precio y seguridad, y una mayor compatibilidad con las cargas de trabajo corporativas.

Oracle es el único proveedor de servicios IaaS, PaaS y SaaS como parte de su oferta en la nube de segunda generación. Y los servicios de base de datos Oracle independientes usan la misma red de alta velocidad que otros servicios, lo que le permite implementar rápidamente aplicaciones de misión crítica para respaldar la innovación continua.

2. Oracle Cloud respalda sus planes a largo plazo

Oracle Cloud Infrastructure es una arquitectura de nube abierta fundamentalmente nueva de segunda generación que sirve como base para Oracle Cloud. Esta infraestructura está diseñada específicamente para proporcionar un rendimiento predecible, seguridad desde el núcleo del sistema hasta su vanguardia y los controles necesarios para soportar cargas de trabajo críticas e intensivas.

Oracle Cloud proporciona capacidades y tecnologías para la informática, el almacenamiento de datos, las bases de datos, los servicios de red y plataforma necesarios para proporcionar resultados comerciales confiables, lo que le permite repensar las necesidades del centro de datos.

Oracle Cloud Infrastructure supera a AWS en rendimiento de carga de trabajo de extremo a extremo ( Andrew Reichman, "Oracle prueba mejor en rendimiento que Amazon Web Services", Oracle, 18 de septiembre de 2018 ).

3. Oportunidades para optimizar el rendimiento.

Oracle IaaS ofrece una amplia gama de capacidades que no tienen paralelo en la industria, desde una plataforma de segunda generación y un conjunto de servicios para plataformas sin un sistema operativo (bare metal) hasta acceso remoto directo a memoria (RDMA) para clústeres informáticos técnicos. Esta diferenciación garantiza que la infraestructura de la nube de Oracle muestre un rendimiento predecible y un buen aislamiento de los clientes de la nube.
La infraestructura en la nube de Oracle también abre la puerta a innovaciones de vanguardia como Oracle Autonomous Data Warehouse y Oracle Autonomous Transaction Processing.

Las cargas de trabajo independientes funcionan mejor en la infraestructura diseñada para proporcionar baja latencia, alta disponibilidad, tolerancia a fallas y rendimiento constante. Esto significa que sus datos se almacenan en el almacenamiento de objetos de Oracle y que varias copias se replican automáticamente, lo que garantiza una alta disponibilidad y tolerancia a fallas. Las capacidades de recuperación automática de Oracle aseguran que sus datos permanezcan "saludables" y que los usuarios siempre puedan acceder a la última réplica conocida.

4. Protección profunda

La seguridad es un principio de diseño clave dentro de la infraestructura de la nube de Oracle. Oracle ofrece protección desde el núcleo del sistema hasta su vanguardia, como el aislamiento del cliente, la seguridad de los datos, la detección de amenazas internas y su eliminación altamente automatizada. Oracle Infrastructure aísla los recursos informáticos y de red para proteger sus datos personales y el tráfico de otros usuarios. También separa su código, datos y recursos de las máquinas de control, lo que ayuda a evitar que los atacantes roben o manipulen datos en la nube.

Oracle Cloud presenta capacidades de ataque muy limitadas a través del aislamiento granular del cliente. Los niveles de seguridad con firewalls integrados, detectores de ataques DDoS y cifrado detectan y detienen las amenazas de manera proactiva. Los clientes pueden establecer una identidad en el nuevo perímetro y utilizar la autenticación adaptativa para agregar automáticamente una verificación adicional cuando la actividad del usuario indique un mayor nivel de riesgo. Finalmente, Oracle tiene una de las carteras más amplias de la industria de soluciones en la nube y locales.

5. Optimización para cargas de trabajo de Oracle

Oracle Cloud Infrastructure ofrece varias características y herramientas únicas que respaldan la migración eficiente y el lanzamiento de la cartera de aplicaciones empresariales y de bases de datos de Oracle. Mover las aplicaciones de Oracle requiere cambios mínimos, lo que reduce el costo y la duración de la migración a la nube. La prioridad de Oracle es proporcionar los últimos equipos y tecnología para aumentar continuamente la productividad del cliente.

Las bases de datos Oracle se amplían muchas veces en términos de almacenamiento y rendimiento de la competencia, alcanzando hasta 40 terabytes de capacidad y millones de operaciones de E / S por segundo (IOPS) por instancia. Cuando se requiere el número máximo de operaciones de E / S para realizar las cargas de trabajo de bases de datos autónomas más complejas, Oracle ofrece la mayor cantidad de memoria local no volátil (NVMe) no volátil (hasta 51.2 TB por instancia). El lanzamiento de los clústeres de Oracle Real Applications Clusters (Oracle RAC) en Oracle Cloud Infrastructure ofrece bases de datos de alta disponibilidad (HA) con conmutación por error en segundos, escala el rendimiento a cientos de miles de E / S y operaciones fluidas basadas en parches y actualizaciones continuas.Esta tecnología introduce un nuevo estándar en la nube para aplicaciones de bases de datos de producción.

Oracle Exadata Cloud permite a los clientes ejecutar máquinas con factor de forma Exadata en las mismas regiones extendidas y redes de nube virtual que la computación básica y otros servicios de infraestructura de nube de Oracle administrados en el mismo sistema que el sistema completo o en el rack completo. Kit de herramientas de administración y accesible a través de la misma consola / API.

La infraestructura en la nube de Oracle ofrece un conjunto de herramientas de migración, aprovisionamiento y administración para aplicaciones clave de Oracle, incluidas Oracle E-Business Suite y Oracle PeopleSoft, Siebel y JD Edwards, así como personalización y bases de datos relacionadas, lo que ayuda a los clientes a acelerar su transición a la nube. En particular, para una base de datos independiente, la infraestructura de nube de Oracle optimiza el análisis de transacciones y las cargas de trabajo de procesamiento de transacciones. Está específicamente diseñado para ejecutar cargas de trabajo de ML para proporcionar información basada en inteligencia artificial, lo que permite al cliente conectar múltiples fuentes de datos y combinar datos clave para respaldar decisiones comerciales acertadas. La infraestructura en la nube de Oracle también es la base de nuevos servicios avanzados, que incluyen integración, contenedores, microservicios y blockchain,y proporciona una escalabilidad mejorada para admitir el despliegue de Internet de las cosas.

6. Utilice el soporte completo para el desarrollo de aplicaciones

Oracle está innovando rápidamente y ofrece una plataforma de desarrollo de aplicaciones abierta, basada en estándares y totalmente integrada que le permite crear, implementar y administrar aplicaciones modernas basadas en la nube basadas en API y dispositivos móviles a un bajo costo. Además, Oracle ofrece soporte para el desarrollo de contenedores y la nube con una pequeña cantidad de código modificable.

La plataforma Oracle AppDev proporciona un entorno DevOps completo para la integración continua (CI) / entrega continua (CD), diagnósticos detallados de aplicaciones Java e integración simplificada con SaaS y software local.

7. Mejor relación precio / rendimiento

Oracle Cloud Infrastructure proporciona el mejor precio / rendimiento disponible en el mercado hoy. Las cargas de trabajo implementadas en la infraestructura de la nube de Oracle a menudo requieren menos servidores de cómputo y bloquean los volúmenes de almacenamiento, lo que reduce el costo de optimizar el rendimiento de la carga de trabajo.

La arquitectura de infraestructura de nube sin bloqueo de Oracle no solo ayuda a minimizar la latencia, sino que también reduce el riesgo de que otros usuarios de la red afecten los entornos de los clientes, y también cuenta con el respaldo del SLA de extremo a extremo más completo, financieramente seguro de la industria.

8. Oracle Cloud supera a AWS

Oracle Cloud ofrece un rendimiento óptimo al menor costo, lo que la convierte en la mejor opción para lanzar el estándar de la industria de Oracle Database para aplicaciones empresariales de misión crítica. AWS no ofrece Oracle RAC, Oracle Autonomous Data Warehouse u Oracle Autonomous Transaction Processing, y Amazon Aurora cierra al usuario en la nube de AWS, mientras que la base de datos Oracle proporciona y combina flexibilidad de diferentes fabricantes.

Los servidores informáticos de Oracle, como Oracle Virtual Machines y bare metal, ayudan a las empresas a lograr ahorros significativos de costos (25-65%) en comparación con las máquinas virtuales de la competencia como AWS.

Las bases de datos Oracle son más rápidas y cuestan hasta un 66% en comparación con AWS, lo que le permite obtener resultados basados ​​en datos comerciales de manera más eficiente y asequible ( Accenture, “Enterprise Workloads Meet the Cloud: Understanding Oracle Databases in the Cloud”). , ", Revisión de almacenamiento," Revisión de Oracle Cloud Infrastructure Compute Bare Metal Instances ).
