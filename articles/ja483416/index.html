Powershellを使用してプロセスを自動化しようとしています


この記事では、Powershellが大好きな理由をいくつかの実例を挙げて説明します。これらはすべて、インベントリ、特定のコンピュータやその他の事項の検索に役立ちます。Powershellには、便利でキャッチーなコマンドレット、すべてのオートコンプリート、直感的な名前が備わっているため、おそらく習得が最も簡単な言語です。

このガイドでは、著者の意見に従っていくつかのクールなチームを調べ、これがなぜクールなのかを説明します。スニペットから始めましょう。

便利なスニペット：


WinRm

経由でサーバーに接続するときに、コンピューターをTrustedHosts Usefulにします。コマンドは以前の値を上書きします。コンマで区切られたIPまたはホスト名を追加してください。すべてのホストがADにある場合は、このファイルを変更する必要はありません。

Set-Item WSMan:\localhost\Client\TrustedHosts -Value '192.168.0.1'

ワイルドカードは、毎回新しいホストをTrustedHostsに追加したくない場合にも機能します。

Set-Item WSMan:\localhost\Client\TrustedHosts -Value '*'

パスワードは暗号化された形式でファイルに保存します。これは

自動化に役立ちます。スケジューラーからスクリプトを実行するときの作業を大幅に簡素化しますが、パスワードは安全な形式で保存されます。 

ログインしたユーザーからスクリプトを実行すると、このユーザーのクレジットが使用されます。

Read-Host -AsSecureString | ConvertTo-SecureString -AsPlainText -Force | ConvertFrom-SecureString | Out-File -FilePath .\Password.txt

暗号化されたパスワードをファイルから取得します。

$Password = Get-Content C:\Password.txt | ConvertTo-SecureString

同じことがCredentialsでも実行できます。Read-HostをGet-Credentialに置き換えます。

Get-Credential | ConvertTo-SecureString -AsPlainText -Force | ConvertFrom-SecureString | Out-File -FilePath .\Credential.txt

OSの最後のブートの日付を取得します。

(gcim win32_operatingsystem).LastBootUpTime

同様に、稼働時間も得られます。

Get-CimInstance Win32_operatingsystem -ComputerName $computers |
Select-Object LastBootUpTime,
@{Name="Uptime";Expression = {(Get-Date) - $_.LastBootUptime}} 

インストールされているプログラムのリストを

取得します：つまり、コンポーネントではなくプログラム：


Get-ItemProperty HKLM:\Software\Microsoft\Windows\CurrentVersion\Uninstall\* |  Select-Object DisplayName, DisplayVersion, Publisher, InstallDate |
Format-Table –AutoSize


ネットワークドライブとドライブの空き容量を含むドライブのリストを取得します

Get-PSDrive -PSProvider filesystem | where-object {$_.used -gt 0} |
Select-Object -property Root,@{name="Size";expression={($_.used+$_.free)/1GB -as [int]}},
@{name="Used";expression={($_.used/1GB) -as [int]}},
@{name="Free";expression={($_.free/1GB) -as [int]}} 

その名前でプロセスを停止します。

この例では、Chromeを停止します。ワイルドカードは、すべてを完了したい場合にも適しています。

Get-Process -Name "chrome" | Stop-Process

フォルダーにアクセスするための設定を子フォルダーにコピーします。

ユーザーグループまたはユーザーからフォルダーとすべてのサブフォルダーを選択するか、使用を許可する場合に便利です。コマンドは、読み取り権限と書き込み権限を下から上に再帰的に実行します。

フォルダからACLをコピーします。

$Acl = Get-Acl -Path C:\folder\

ACLをすべてのファイルとサブフォルダーに挿入します。

Get-ChildItem -Path C:\Folder\ -Recurse | Set-Acl $Acl 

便利なスクリプト：


これが最悪の楽しみの始まりです。目録をとる必要がある場合は、調べてみると非常に便利です。コンピューターがADにない場合、スクリプトはもう少し複雑になり、事前にコンパイルされたファイルに対して反復する必要があり、パスワードはファイルから、またはGet-Credentialを介して取得する必要があります。すべての違い：

C AD：

$ADComputers = (Get-ADComputer -filter *).DNSHostName

ADなし：

$Credentials = Get-Credential
$Computers = Get-Content -Path C:\servers.txt

AD内の各コンピューターの最後の起動時間の稼働時間を取得します。


$ADComputers = (Get-ADComputer -filter *).DNSHostName

foreach ($i in $ADComputers) {
 
    Invoke-Command $i {
        Get-CimInstance Win32_operatingsystem |
        Select-Object LastBootUpTime,
        @{Name="Uptime";Expression = {(Get-Date) - $_.LastBootUptime}} 
    } 
 
}

すべてのADコンピューターにインストールされているすべてのプログラムを取得します。

$ADComputers = (Get-ADComputer -filter *).DNSHostName
foreach ($i in $ADComputers) {
 
    Invoke-Command -computername $i {
        gcim win32_product -computername $env:computername | Sort-Object -property Vendor,Name | Select-Object -property Vendor,Name, Caption 
    } 
 
}

コンポーネントについても、Invoke-Commandに置き換えて同じことができます。

Get-WindowsFeature | Where-Object -Property "Installed" -EQ "Installed"

必要なプロセスが実行されているコンピューターを

取得します。このコンピューターまたはそのプログラムが実行されているコンピューターを忘れた場合は、この方法で取得できます。ブラウザの例を考えてみましょう：

$ADComputers = (Get-ADComputer -filter *).DNSHostName
foreach ($i in $ADComputers) {
 
    Invoke-Command -computername $i {
        Get-Process -Name "Chrome"  -ErrorAction SilentlyContinue
    } 
 
}

ブラウザを強制終了するのは、プロセスを取得するのと同じくらい簡単です。

$ADComputers = (Get-ADComputer -filter *).DNSHostName
foreach ($i in $ADComputers) {
 
    Invoke-Command -computername $i {
        Get-Process -Name "Chrome" | Stop-Process -ErrorAction SilentlyContinue
    } 
 
}

特定のプログラムがインストールされているコンピューターを

入手します。この場合、Javaは秘密です。インベントリ中に役立ちます。それがインストールされているコンピューターの名前を確認できます。

$ADComputers = (Get-ADComputer -filter *).DNSHostName
foreach ($i in $ADComputers) {
 
    Invoke-Command -computername $i {
        gcim win32_product -computername $env:computername | Select-String -Pattern "Java" -AllMatches | Sort-Object -property Vendor,Name | Format-Table -ErrorAction SilentlyContinue
 
    } 
 
}
