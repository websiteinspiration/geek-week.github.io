モバイル開発におけるPython



Python の範囲が非常に広いことは明らかです。Webベースのテクノロジー、ゲーム業界、そしてNASAプロジェクトで終わります。 Pythonは、ハンドヘルドコンピュータやスマートフォンからネットワークサーバーまで、Windows、Linux / UNIX、macOS、Palm OS、Raspberry PIなど、ほとんどすべての有名なプラットフォームで動作します。おかげKivyの枠組みは、 2011年にPythonはまたのためのアプリケーション開発の面でモバイルプラットフォームをマスターのiOSとAndroidの、そして2015年に、使用してKivyMDの ライブラリーを、Pythonは使用することを学んだ材料設計を。

KivyMD ライブラリは、Kivyクロスプラットフォームフレームワークで使用するための、マテリアルデザインスタイルのウィジェットのコレクションです。1年前にリリースされた前回の記事「KivyMD-life gos」では、このライブラリのフォークについてすでに説明しましたが、PyPiからKivyMDをインストールするときに、フォークで宣言された機能が利用できないという通知を問題やメールでよく受け取りました。ので、それは、とても本当にだったは、PyPIは上のリポジトリから4年前にライブラリの古いバージョンが含まれていGitLabもはや作者によってサポートされていない、と私たちは、このようなKivyMD・フォークとして、私たちのフォークの新しい名前を使用してにパッケージをロードしたくなかったので、は、PyPI同じ名前で、それはへの書き込みに決定しましたKivyMDの 著者アンドレス・ロドリゲス（@mixedCase_）と彼のパッケージを削除するために彼に尋ねます。長い交渉の末、Andreはライブラリの共著者であり続ける限り、Pythonパッケージへの招待をくれました。これで公式のライブラリリポジトリがここにあり、最新のアップデートはPyPiで入手できます。

下部のアプリバー


マテリアルデザインの下部アプリバー

左側はFlutterデモアプリケーションからのBottom App Barの動作例で、右側はKivyMD Bottom App Barライブラリのデモです。

コード例
from kivy.factory import Factory

from kivymd.app import MDApp
from kivy.lang import Builder


Builder.load_string(
    """
<BottomAppBar@Screen>

    BoxLayout:
        spacing: dp(10)
        orientation: 'vertical'

        MDToolbar:
            title: "Title"
            md_bg_color: app.theme_cls.primary_color
            left_action_items: [['menu', lambda x: x]]

        Widget:

        MDBottomAppBar

            MDToolbar:
                title: "Title"
                icon: "git"
                type: "bottom"
                left_action_items: [["menu", lambda x: x]]
"""
)


class BottomAppBarTest(MDApp):

    def build(self):
        return Factory.BottomAppBar()


BottomAppBarTest().run()
