Sampel Median. Interval Keyakinan dan Perbandingan

Publikasi ini bersifat deskripsi jalur yang mengarah ke algoritmeisasi metode yang efisien untuk menghitung interval kepercayaan (Confidence Interval = CI) untuk


  • median distribusi;
  • perbedaan median dari dua distribusi.

Tugas ini murni praktis, Anda bisa menyelam ke kedalaman matematika, tetapi ini bukan tujuan itu sendiri, dan itu tidak selalu cukup untuk balon untuk sampai ke dasar.


Sampel berukuran besar, catatan 10 ^ 5 - 10 ^ 7, terasa asimetris, dengan ekor panjang, dapat memiliki beberapa mode. Dalam hal ini, median lebih tahan terhadap emisi.


Penerapan statistik klasik, misalnya, kriteria Wilcoxon-Mann-Whitney, untuk memperkirakan perbedaan median untuk volume seperti itu tidak lulus. Ya, dan banyak hal perlu dibaca di bawah tanda bintang agar dapat menerapkan kriteria ini dengan benar. Memang, kriteria ini sama sekali tidak memeriksa kesetaraan median, dan untuk median hanya bekerja dengan bentuk distribusi yang sama dari dua sampel. Dll dll.


Anda dapat mengambil palu bootstrap, tetapi Anda harus berpikir dengan itu + butuh waktu dan memori untuk mensimulasikan.


Di sisi lain, sangat sering matematikawan datang dengan berbagai penyederhanaan analitis di bawah asumsi tertentu, yang memungkinkan masalah kompleks diselesaikan dalam satu formula. Pencarian untuk pendekatan yang terakhir mengarah pada solusi berikut (dalam kaitannya dengan sampel yang dijelaskan di atas).

# test from table 3 of b&p 2002
x1 <- c(77, 87, 88, 114, 151, 210, 219, 246, 253, 262, 296, 299, 306,
        376, 428, 515, 666, 1310, 2611)
x2 <- c(59, 106, 174, 207, 219, 237, 313, 365, 458, 497, 515, 529,
        557, 615, 625, 645, 973, 1065, 3215)

# sort vectors
x1 <- sort(x1)
x2 <- sort(x2)

# get medians
x1_mdn <- median(x1)
x2_mdn <- median(x2)

# stuff to calculate variance of medians
x1_n <- length(x1)
x2_n <- length(x2)

x1_aj <- round((x1_n + 1) / 2 - x1_n ^ (1 / 2))
x2_aj <- round((x2_n + 1) / 2 - x2_n ^ (1 / 2))

z <- 1.855 # from table 1 of b&p 2002, see p. 376

# calculate variance
x1_var <- ((x1[x1_n - x1_aj + 1] - x1[x1_aj]) / (2 * z)) ^ 2
x2_var <- ((x2[x2_n - x2_aj + 1] - x2[x2_aj]) / (2 * z)) ^ 2

# contrast coefficients, such that its median(d) - median(dg)
x1_cj <- 1
x2_cj <- -1

# median difference
mdn_diff <- x1_mdn * x1_cj + x2_mdn * x2_cj

# standard error
mdn_diff_se <- (((x1_cj ^ 2) * x1_var) + ((x2_cj ^ 2) * x2_var)) ^ (1 / 2)

# 95% confidence interval
lb <- mdn_diff - 1.96 * mdn_diff_se
ub <- mdn_diff + 1.96 * mdn_diff_se

# within roundng error of p. 376 of b&p 2002
paste0(mdn_diff, " [", round(lb), ", ", round(ub), "]")
