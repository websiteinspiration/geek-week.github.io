「シーザー」25種類と英単語

暗号化の世界では、メッセージを暗号化する多くの簡単な方法があります。それらのそれぞれは独自の方法で良いです。それらの1つについて説明します。


イルチュ・シュクゴウ


または、「シーザーの暗号」からロシア語- シーザーの暗号に翻訳されました。



-その本質は何ですか？
-メッセージをエンコードし、各文字をNポイントずつシフトします。シーザーの古典的な暗号は、文字を3ステップ先に進めます。簡単に言えば、それは「abv」であり、「どこ」になりました。


「しかし、アルファベットの最後の文字はどうですか？」スペースはどうですか？
すべてが問題ありません。文字をシフトする場合、暗号はアルファベットの範囲を超えます-それはすべてをもう一度数え始めます。つまり、「えゆや」の文字が「abv」に変わります。そしてスペースはスペースのままです。


-Nは必ずしも3に等しい必要がありますか？
どういたしまして。Nはトリプルとは異なる場合があります。[1：M-1]の範囲の任意のNが許可されます。Mはアルファベットの文字数です。


このような暗号は、その存在を知っていれば解読が簡単です。しかし、私を引き付けたのは彼の「信頼性」ではなく、他の何かでした。

def cesar(word, abc, abc_len, step=3):
    word_list = list(word)
    result = []

    for w in word_list:
        i = abc.index(w)
        i = (i+step) % abc_len
        wn = abc[i]
        result.append(wn)

    result = ''.join(result)
    return result
