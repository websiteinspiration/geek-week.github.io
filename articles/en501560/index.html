"I am quieter than a whisper in an open field" or what is Ninjutsu-OS



Ninjutsu-OS is a specialized Windows 10-based OS (build 1909) that contains over 800 cybersecurity testing tools. A kind of Kali Linux, only from the world of Windows. The release is very fresh, took place on 04/18/2020.

Installing the OS itself is no different from the standard installation of Win 10, with the only exception that it is significantly longer.



After starting the OS itself, it will take about another 10 minutes to reinstall and configure everything. After all the manipulations, we are met by a slightly overloaded (in my opinion) and very colorful desktop.



















Now let's see what's inside.

In total, the assembly includes more than 800 utilities. The author had to adapt some of them so that they started to run under Windows.

In the upper part there is a separate bar, which contains all the utilities by category for quick access. Utilities are categorized with emphasis on the OSCP methodology with an eye for categorization in Kali Linux:
- Mr. Robot tools kit - the most commonly used utilities and other additional materials, according to the author of the assembly. Personal top, so to speak;
- Red Teaming - the most commonly used utilities for testing by category. In fact, these are all top utilities from all categories below under one tab;
- Information Gathering - collecting information;
- Exploitation Tools - exploitation of vulnerabilities;
- Malware analysis - malware analysis;
- Mobile Security Tools - testing mobile devices;
- Network Attack - network attacks;
- Password Attacks - cracking passwords;
- Proxy and Privacy Tools - proxying and anonymizing;
- Reverse Engineering - reverse engineering;
- Stress Testing - stress testing (DDoS);
- Vulnerability Analysis - search for vulnerabilities;
- Web Application Attack - web pentest;
- Wireless Attacks - attacks on wireless networks (Wi-Fi and Bluetooth),
- Fuzzing and payloads - fuzzing and workloads;
- Remote Control Tools - remote control;
- Utility Tools - other auxiliary utilities (Winrar, MySQL, etc.);

A complete list of available utilities can be found on the project website . The OS is perfect for both pure Pentest and people who are engaged in reverse engineering, OSINT or malware analysis. For several days of working on the OS, I did not catch any critical bugs or malfunctions in the software. Tested on machines with Hack the Box plus turned up an additional project for OSINT.

Separately, mention should be made of the Alias ​​added by the author for console commands. Personally from me - a huge human thanks for highlighting below 3. The



full list of added Alias ​​can also be viewed on the project website. In general, they greatly facilitate the work.



Also, having talked directly with the author, I clarified some questions about the future of the OS. At this stage, in fact, almost everything was added to the assembly. Further, depending on the feedback, the author of the assembly intends to add or remove utilities.


The dialogue is published with the consent of the interlocutor.

In this article I did not set myself the goal of making some kind of in-depth analysis of the OS or the success of its filling. I decided, so to speak, to share the joy with you that now we will not use Kali Linux alone.

If you are generally interested in having an analog of Kali Linux only in the Windows ecosystem appear in your arsenal, then do not pass by! Be sure to test this OS and write a review to the author about what you would bring or remove into it.
