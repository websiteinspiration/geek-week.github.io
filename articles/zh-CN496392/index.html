IT社区如何响应新IP计划

中国的许多电信公司充当了TCP / IP潜在替代方案的发起者。但是，包括国际电信联盟（ITU）在内的许多专家认为这不是一个好主意。我们讨论了实施协议的意见和可能的选择。


/不飞溅/ 罗宾·胡

为什么出现新IP


TCP / IP网络模型是在1972年开发的。在此期间，它揭示了许多漏洞，包括与身份验证相关的漏洞。因此，今天您可能会发现TCP / IP不适合在现代条件下工作的观点。这种观点在华为身上是存在的。该公司的代表指出，由于寻址机制的兼容性差，单个网络（例如，卫星网络或专用网络）之间的通信不稳定。

为了解决该问题，该公司在中国工业和信息化部以及中国电信的支持下，开始制定标准，该标准将成为新一代网络基础设施的基础。他被评为新IP。

到目前为止，对新IP的原理知之甚少。尽管中国公司披露的正式文件介绍了新IP数据包的标头结构。


它从所需的字段指示符（FI）条目开始。这是一个位图，指示哪些可选字段将封装在标头中。FI旁边是带有流量类别，流标签，有效负载长度和TTL的位链。

接下来是带有发件人和收件人地址的字段。与IPv4和IPv6中的类似字段不同，它们将具有可变的长度。对于数据编码，开发人员建议使用简洁的二进制对象表示（CBOR）格式。新的标头还包含可选的功能ID，元数据索引和元数据字段，用于组织用户定义的网络。

社区的想法


预计新IP 将由于更有效的寻址而提高数据传输的速度，并允许它们在网络上的设备之间直接传输。根据开发人员的说法，该协议将使早在2030年就可以有效地控制无人机和物联网的其他设备。

但是，并非所有人都如此乐观-该计划的批评者将新IP视为对全球互联网基础设施的威胁。大不列颠，瑞典和美国的代表已经表达了他们的关注。他们说新协议类似于金盾牌。担心与新IP开发人员提议引入killswitch这一事实有关，它允许阻止到达特定地址的所有数据包。结果，执法机构可以选择阻止个人访问网站。专家们还担心这种做法可能导致对网络的集中控制。

协议角度


关于新IP的介绍将在国际电信联盟的11月会议上进行演示，该会议定义了全球范围内网络之间的交互原理。中国公司希望他们的决定能够获得国际电联的批准，成为公认的标准。

该委员会将做出什么决定尚不清楚。华为可能不被美国代表的支持，因为很多美国IT企业已经打破与中国合作伙伴的关系。此外，国际电联瑞典代表团成员之一帕特里克·法斯特罗姆（PatrikFältström）在欧洲被称为“互联网之父”，反对新IP 。据他介绍，现代全球网络的主要优势是“开放性”，而新的穿孔可能会使此功能面临风险。


/ Unsplash / Dominik Vanyi

另一方面，国际电信联盟秘书长是周厚麟，他是在中国工业和信息化部的支持下提名的。这一事实可能会对新IP讨论的发展产生重大影响。无论如何，无论国际电联新互联网协议的命运如何发展，中国都计划在未经国际组织批准的情况下建设互联网基础设施。首次测试必须在2021年通过。
