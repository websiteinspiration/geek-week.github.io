レビュー：2019年に最も注目を集めたセキュリティインシデント



画像：Unsplash

毎年、サイバーセキュリティの問題はますます重要になっています。データ漏えいはますます小さくなっているわけではなく、サイバー犯罪者はより洗練されたハッキン​​グ方法と収益スキームを発明し続けており、企業のセキュリティはますます強力にテストされています。

過去数年と同様に、2019年は大規模なデータ漏えいからサイバースパイ活動、金融犯罪、ランサムウェア攻撃まで、非常に波乱に富んでいることが判明しました。SecurityLab.ruポータルによると、2019年の最も注目度の高いインシデントについて以下で説明します。

基本コレクション＃1-＃5


2019年1月、MEGAクラウドサービスで、さまざまなソースから収集された約7億7,300万の一意のメールアドレスと2,200万の一意のパスワードを含むアーカイブが発見されました。全体で、コレクション＃1と呼ばれるアレイには、12,000を超える個別のファイルと87 GBを超えるデータが含まれていました。データベース内の一部のパスワードはクリアテキストで保存されました。同じ月に、 22億の一意のユーザー名とパスワードのアーカイブがハッカーフォーラムに表示されました。コレクション＃2〜5としてバプテスマされた845 GBのデータアレイには、コレクション＃1の3倍の250億のレコードが含まれていました。

ドリームマーケット


1か月後、ハッキングされた16のウェブサイトのユーザーから盗まれた6億1700万のアカウントを含むデータベースが、地下のドリームマーケットで売りに出されました。データベースの売り手は、Gnosticplayersという偽名の誰かで、そのコストはビットコインで2万ドルでした。この金額では、誰もが1億6200万の侵害されたDubsmashアカウント、1億5100万のMyFitnessPal、9200万のMyHeritage、4100万のShareThis、2500万のAnimoto、2200万のAnimoto、2200万のEyeEm、2000万の8fit、1800万のホワイトページ、1600万のFotolog、15 5億px、1100万のアーマーゲーム、800万のBookMate、600万のCoffeeMeetsBagel、100万のArtsy、70万のDataCamp。その後、Gnosticplayers は、8つのサイトの1億2700万の盗まれたユーザーアカウントを含む2番目のアーカイブを売りに出し、4つのビットコインを要求しました。

Norsk Hydroへのハッカー攻撃


2019年には、ランサムウェアを使用した攻撃の数が大幅に増加し、多くの大企業に影響を及ぼしました。たとえば、3月に、世界最大のアルミニウム生産者であるNorsk Hydro は、LockerGogaランサムウェアによる攻撃により、生産設備を一時停止することを余儀なくされました。同社によると、事件の被害は約$ 35から41000000に達し、など特別な機器Aebi Schmidt氏、ドイツのラインメタル懸念、スイスのメーカーをした様々なランサムウェアプログラムの犠牲者の中にも。

ASUS Live Update Utilityのバックドア


今年の初めに、ASUSコンピュータのユーザーを狙った悪意のあるキャンペーンについて知られるようになりました。サイバー犯罪グループShadowHammerは、ASUS Live Updateユーティリティをハッキングして、BIOS、UEFI、およびソフトウェアの更新をASUSのラップトップおよびデスクトップコンピューターに配信し、バックドアを実装し、公式チャネルを通じて配布しました。専門家によると、感染の総数は100万人に達する可能性があります。

漏洩した検証IO LLC


3月に、セキュリティ研究者のBob DiachenkoとVinny Troia  は、合計808 539 939レコードの4つの個別のデータセットを含む150 GBのオープンソースデータベースを発見しました。データベースはマーケティング会社Verifications IO LLCに属し、リークを報告した後、データベースとサイトの両方を即座にオフにしました。

Facebookユーザーの公開データ


5億4,000万人を超えるFacebookユーザーの記録が、Amazon S3クラウドサーバーのパブリックドメインに保存されました。リークの原因は、テクノジャイアント自身ではなく、サードパーティのFacebookアプリケーション開発者であるメキシコのメディア会社Cultura ColectivaとAt the poolアプリでした。Cultura Colectivaが所有する146 GBのデータベースには、コメント、設定、ログイン、ユーザーIDなどのFacebookユーザーデータを含む5億4,000万件を超えるレコードが含まれていました。データベース「At the pool」には、名前、テキスト形式のパスワード、メールアドレスが保存されています22,000人のユーザー、および友達、いいね、グループなどに関する情報

WhatsApp Pegasusスパイウェアの脆弱性


5月の共鳴イベントの1つは、WhatsAppメッセンジャーの脆弱性CVE-2019-3568の発表でした。これは、イスラエルの企業NSOグループによって製造されたペガサススパイウェアをインストールするために使用されました。10月、WhatsAppはハッキングツールメーカーを訴え、NSOグループが政府の諜報機関が外交官、野党メンバー、ジャーナリスト、高官など、世界中の約1,400人のユーザーの電話を解読するのを手助けしたと非難しました。

デリーの50万人近くの住民のデータ漏洩


保護されていないMongoDBサーバーがネットワーク上で発見され、「GNCTD」と呼ばれる4.1 GBデータベースが含まれていました。アーカイブには、デリー（インド）の住民458,388人に関する機密情報が含まれていました。データベースには、Aadhaar番号、有権者カード番号、健康状態、教育、場所、家の有無、階数、インターネットの可用性などのデータを含む、人物の肖像画を作成できる詳細情報を含むいくつかのセクションが含まれていました。

暗号通貨交換のハッキング


暗号通貨が絶え間なく人気を博しているため、対応する取引所は引き続き攻撃者にとって一口です。2019年には、いくつかの主要な交換が犯罪者の手に一度に苦しみました。特に、4月に、暗号通貨で約2,000万ドルを失った韓国の暗号通貨取引所Bithumbがハッキングの被害者となり（3年間で3回目）、1か月後に、世界の5つの最大の暗号通貨取引所の1つであるBinanceも同様の状況に陥りました-攻撃者は高温の財布にいますサービスおよび7000以上のビットコイン（約4100万ドル）を引き出しました。さらに、彼らはまた、トレーダーの個人情報、秘密鍵、2要素認証パスワード、およびその他のデータを自由に利用できました。

グーグルは密かに蜂蜜を集めました。アメリカ居住者


グーグルは秘密のデータ収集を含むスキャンダルの中心にありました。結局のところ、テクノロジーの巨人とアセンションは、数百万人のアメリカ人の医療データを収集して分析するための共同秘密プロジェクトを実施しました。収集された情報には、検査結果、医師の診断と入院記録、完全な病歴-患者名と生年月日が含まれていました。このデータは、治療戦略の変更について特定の人々に推奨を提供する新しいAIベースのソフトウェアの開発に使用されました。

iPhone所有者に対する大規模な攻撃


専門家は、Apple iPhoneスマートフォンの所有者に対する歴史上最大のサイバー攻撃の1つを発見しました。攻撃者は、毎週数千人のユーザーがいる多数のサイトをハッキングし、その助けを借りて、オペレーティングシステムのゼロデイ脆弱性を通じてiOSデバイスをマルウェアに感染させました。このマルウェアは、被害者の機密情報を盗み、キーチェーンのパスワードや、Googleハングアウトなどの通信サービスの暗号化されていないメッセージのデータベース、さらには暗号化されたメッセージのWhatsApp、iMessage、Telegramにアクセスしました。

ソーシャルネットワークの10億人を超えるユーザーのデータ漏洩


4 TBを超える情報を含むデータベースがパブリックドメインであることが判明-ソーシャルネットワークFacebook、Twitter、LinkedIn、Githubの数億人のユーザーのプロファイルからのデータを含む、合計12億件のレコード（5000万の電話番号、6億2,200万の一意）メールアドレスと雇用履歴の記録。Google Cloud Servicesアーカイブには、パスワード、支払いカード番号、社会保障番号は含まれていませんでした。

スパイ活動のための通信会社の妥協


6月の終わりに、犯罪者が特定の個人に関する情報を傍受するために世界最大の通信会社のネットワークに侵入した大規模なサイバースパイ活動の詳細が発表されました。このキャンペーンは、中国に関連するAPT10グループによって組織されたとされています。攻撃者はなんとか約100 GBの情報を盗み、Call Detail Records（CDR）を使用して、それらに興味を持った人々の動きと行動を追跡しました。

リークキャピタルワン 


アメリカの銀行持株会社であるCapital One  は、 1億人以上の米国居住者と600万人のカナダ人居住者の大規模なデータ漏洩を報告しました。リークの原因は、影響を受ける企業のデータベースをホストしていたアマゾンウェブサービス（AWS）パブリッククラウドにアクセスしたアマゾンの元従業員でした。個人データに加えて、この漏洩は、140,000人のカード所有者の社会保障番号に影響を与え、銀行口座カードに関連する約80,000の番号にも影響を与えました。同社はハッキングによる被害を1億〜1億5000万ドルと推定している。

大きなアジアのリーク


昨年の12月は、いくつかの大きなデータリークがありました。特に、月の初めに、未知の人物が27億の電子メールアドレスと10億を超える暗号化されていないパスワードを含むデータベースをパブリックドメインに公開しました。データベース分析が示したように、ほとんどのデータは、2017年の初めにDoubleFlagという偽名でサイバー犯罪者が売りに出したリークでした。「Big Asian Diversion」と呼ばれるリークには、NetEase、Tencent、Sohu、Sinaなど、多くの中国のインターネット企業からのユーザーデータが含まれていました。

そして再びFacebook


12月のもう1つの重大なリークは、ソーシャルネットワークFacebookのユーザーに影響を与えました。 Unknowns は、Facebookユーザーアカウントの2億6700万を超える一意の識別子を含むデータベースをWeb上で公開しました。合計で、ソーシャルネットワークの267,140,​​436ユーザーのデータはパブリックドメインであることが判明しました。それらのほとんどは米国の居住者です。データベースには、Facebookアカウントの一意の識別子、電話番号、ユーザー名、姓、タイムスタンプが含まれていました。
