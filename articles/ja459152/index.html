Pythonを使用してMicrosoft Wordでドキュメントに入力します。パート2

文書を広める

。
前回の投稿では、Pythonを使用してクエリを単語で入力する方法について説明しました。今回は、複数のワードドキュメントにExcelテーブルのデータを一度に入力する方法を示します。

プログラムで作業する前に、ソースデータを含むExcelファイルを準備する必要があります。このようなデータを含むExcelファイルを作成するか、既成のファイルを使用して



みましょう。次に、プログラムを見てみましょう。

Excelファイルを読み取る必要があるため、対応するモジュールをインポートする必要があります。

import openpyxl

次に、テーブルを調べて、すべてのデータを1行ずつ読み取る必要があります。次のようになります。

test=[]
wb = openpyxl.load_workbook('zaprosi.xlsx')
sheet=wb.get_active_sheet()

for row in sheet['B3':'F7']:
    for cellObj in row:
        if cellObj.value==None or cellObj.value==" ":
            continue
        #print(cellObj.value)
        test.append(cellObj.value)

空のリストを作成し、将来テーブルから値を追加します。追加が発生するのは、セルにエントリ（空ではない）がある場合のみです。

操作が完了したら、収集したデータをWordテンプレートに追加します。

doc = DocxTemplate(".docx")
    context = { 'emitent' : test[0],'address1' : test[1],'' : test[2],'_' : test[3],'director' : test[4] }
    doc.render(context)
    doc.save('-final.docx')

その結果、テーブルのデータを含む完成した単語ファイルができます。ここだけで、テンプレートファイナルと呼ばれます。では、どのようにして文書を互いに区別するのでしょうか。

簡単なものから始めましょう。毎回異なる名前でファイルを保存するには、コードの最後で以下を変更する必要があります。

doc.save(test[x]+'.docx')

保存されたファイルには、リクエストを送信した会社の名前が含まれます。

プログラムの実行中にエラーが発生する可能性があります。



このエラーは、会社名の引用符が原因です。 Excelテーブルから引用符を削除すると、WordファイルはCompany-1.docxという名前で正常に保存されます。

テーブルの残りのデータを入力して、ドキュメントを「複製」します。

ここのすべても簡単です。収集したデータに従ってサイクルを作成します。

x=0
while x<len(test):
    doc = DocxTemplate(".docx")
    context = { 'emitent' : test[x],'address1' : test[x+1],'' : test[x+2],'_' : test[x+3],'director' : test[x+4] }
    doc.render(context)
    doc.save(test[x]+'.docx')    
    x+=5

サイクルの最後に、会社名を含む列の名前に「ジャンプ」して、単語ファイルが正しい名前で保存されるようにします。

プログラムの準備ができました！稼働中の企業を含むExcelテーブルを作成するだけです。
