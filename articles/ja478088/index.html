import os

def do_magic():
  now = datetime.datetime.now()
  return "Hello! {0}".format(now)

def application(env, start_response):
  start_response('200 OK', [('Content-Type','text/html')])
  return [do_magic().encode()]

    print("Content-type: text/html\n\n")

ここに私達のコンフィギュレーション・ファイルの内容は以下のとおりです。

server {
  listen 80;
  root /var/www/html;

  location / {
    include /etc/nginx/uwsgi_params;
    uwsgi_pass 127.0.0.1:9000;
    uwsgi_param Host $host;
    uwsgi_param X-Real-IP $remote_addr;
    uwsgi_param X-Forwarded-For $proxy_add_x_forwarded_for;
    uwsgi_param X-Forwarded-Proto $http_x_forwarded_proto;
  }
}

実際、設定は非常に簡単です。サーバーにポート80でリッスンし、それをルートディレクトリとして使用し、ルートディレクトリ/var/www/htmlにアクセスしたときにトラフィックをアドレスに転送するように要求します127.0.0.1:9000。転送するだけでなくuwsgi、デーモンがuwsgi彼を待ち、彼の作業の結果を提供する形式でそれを実行することで、ユーザーは私たちの素晴らしいアプリケーションを見ることができます。そしてそれがまさにその通りでnginxあり、uwsgi私たちはで作業しproductionます。
