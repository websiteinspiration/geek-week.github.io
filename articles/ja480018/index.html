次に、OrderProductLayoutウィジェットが何であるかを見てみましょう。


ご覧のとおり、これらはScrillViewに埋め込まれた4つのカードです。FloatLayoutから継承される親画面とは異なり、ここではすべてのウィジェットが上から下に読み取られます。


これは非常に便利です。ウィジェットの明確な階層、ツリー構造があり、どのウィジェット/コントロールがどのレイアウトに属しているかが一目でわかるからです。Kivyで最も一般的に使用されるレイアウトはBoxLayoutです。これは、ウィジェットを自分の中に垂直または水平に配置できるボックスです（デフォルトでは、最後のウィジェットです）。


これは、BoxLayoutの水平方向が使用されている次の図からより明確に見ることができます。


BoxLayoutがスペースの100％を使用することを禁止します-size_hint_y：なしと述べました-あなたの高さはあなたの中にネストされた最も高い要素の高さと正確に同じになるでしょう- 高さ：self.minimum_height。

リストの垂直スクロールを使用したい場合、GridLayoutを次のように変更する必要があります。

    ScrollView:

        GridLayout:
            size_hint_y: None
            height: self.minimum_height
            cols: 1

列（cols）の行 （rows）を置き換え、幅と高さの最小値を指定します。

from kivy.app import App
from kivy.lang import Builder
from kivy.metrics import dp
from kivy.uix.button import Button

KV = """
ScrollView:

    GridLayout:
        id: box
        size_hint_y: None
        height: self.minimum_height
        spacing: "5dp"
        cols: 1
"""


class MyApp(App):
    def build(self):
        return Builder.load_string(KV)

    def on_start(self):
        for i in range(20):
            self.root.ids.box.add_widget(
                Button(
                    text=f"Label {i}",
                    size_hint_y=None,
                    height=dp(40),
                )
            )


MyApp().run()
