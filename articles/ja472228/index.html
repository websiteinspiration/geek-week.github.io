Visual Studio Codeを操作するための9つのトリック

こんにちは、ハブル！今日、私たちはVisual Studio Codeを操作するための最も興味深いライフハックの9つを選択しました。その中には、便利なキーボードショートカットと機能的なウィジェットがあります。カットソー下ディテール！



前のパート：18トリック

1.インタラクティブな遊び場


ウェルカム ページの右下隅に、インタラクティブなプレイグラウンドへのリンクがあり、VS Codeの機能をインタラクティブに試すことができます。または、[ヘルプ]> [インタラクティブプレイグラウンド]からアクセスできます。



2.ファイル内の文字に移動します


キーの組み合わせ：Ctrl + Shift +



バージョン情報コロン@：を追加して、フォーム別にシンボルをグループ化できます。



3.ラインを上下に移動します


キーボードショートカット：Alt +上またはAlt +下



4.並べて編集する


キーボードショートカット：Ctrl + \

必要に応じて、新しい編集者グループを作成して、グループ間を自由に移動できます。



5.複数のエディターを切り替える


キーボードショートカット：Ctrl + 1、Ctrl + 2、Ctrl + 3



6.ファイルの最初/最後に移動します


キーボードショートカット：Ctrl + HomeおよびCtrl + End

7.サイドバーを切り替える


キーボードショートカット：Ctrl + B



8. IntelliSense


キーボードショートカット：Ctrl +スペース

「ヒント」ウィジェットを使用すると、使用可能なメソッド、パラメーターのヒント、簡単なドキュメントなどを表示できます。
