コマンドラインのクリスマスツリー

新年が近づいて、真面目な仕事を考えたくありません。

誰もが休日のために何かを飾ろうとしています：自宅、オフィス、職場...何かを飾りましょう！たとえば、コマンドプロンプト。ある程度、コマンドラインは作業場でもあります。

一部のディストリビューションでは既に「装飾」されています



。他のディストリビューションでは、灰色で目立たない



ようになっています。ただし、たとえば次のようにできます。



もちろん、すべてのフェルトペンは味と色が異なります。このような色付けが不器用で不適切であると思われる場合は、この視点に生命に対するあらゆる権利があることを確認してください。また、少し新年の気分を追加したい場合は、Cloud4Yからの小さな新年の記事を読んでください。

最初に、ターミナルの出力がどのように「色付け」されるかを説明します。これは、エスケープシーケンスを使用して行われます。具体的には、ANSI / VT100端末の制御コードシーケンス。これは自動的に端末エミュレータがこの標準をサポートする必要があることを意味します。そうでなければ、新年の奇跡は起こりません。そして、はい、あなたは$ SHELL bashを持っていると仮定されています。

これらのコマンドは、それぞれの先頭がASCIIエスケープ文字であるため、エスケープシーケンスと呼ばれます。多くの制御シーケンスがあり、たとえば、端末設定の表示、カーソルの表示と移動の制御、フォントの変更、テキストの削除と非表示などが可能です。さまざまな可能性の中から1つを選択します。テキストと背景の色を変更します。

ここでターミナルでコマンドをecho \033[0;31m \033[0m’

実行すると、...予測どおりにモノクロームabracadabraが出力として表示されます。



なぜですか？ echoコマンドの高度な機能を使用する必要があったからです。キーを1つ追加するだけです。echo -e ‘\033[0;31m \033[0m’

これで出力は正しく表示



されます。ターミナルの出力カラーリングを整理しました。次に、コマンドラインプロンプトに色を付ける方法を見てみましょう。

これは、変数PS1を変更することによって行われます。変数はコマンドラインプロンプトを担当します。エスケープシーケンスの使用を含め、外観も変更できます。ただし、わずかな違いがあります。シーケンスを文字「\ [」で開始し、文字「\]」で終了する必要があります。それ以外の場合は、端末に表示されます。

可能なすべてのコマンドシーケンスはbashのマニュアルに詳しく記載されているので、コマンドプロンプトで観察したいものを自分で選択することをお勧めします。例として、変数PS1に値を指定します。

\[\033[34;1m\]\t\[\033[0m\],\[\033[32m\]\u@\l@\h\[\033[0m\]:\[\033[33m\]\W\[\033[0m\],\[\033
[31m\]\!\[\033[0m\]\$\n

私はこの恐ろしい呪文を解読します：

\ [\ 033 [34; 1m \] -明るい青（2番目の属性）のフォント色をオンにします
\ t-現在の時刻をHH：MM：SS
\ [\ 033 [0m \]の形式で表示します-リセットフォントの色
、 -カンマのみ（予期せず、正しいですか？）
\ [\ 033 [32m \] -フォントの緑色をオンにします
\ u @ \ l @ \ h-ユーザー名、端末デバイス番号、短いホスト名を「@」で区切って表示します」
[\ 033 [0m \] -フォントの色設定をリセットします
： -コロンのみ（突然！）
\ [\ 033 [33m \] -黄色のフォントの色をオンにします
\ W-現在のディレクトリの名前を表示します
[\ 033 [0m \]-再びフォント色の設定をリセットし
、別のカンマ（、誰が考えを持っているでしょう！） -
\ [\ 033 [31メートル\]は -赤いフォントの色ターンオン
\！ -端末にコマンド番号を出力します
[\ 033 [0m \] -フォントの色設定をリセットすることを忘れないでください
\ $ -ルートには「＃」を、その他の全員には「$」を出力します
\ n-改行何のために？したがって、コマンドはウィンドウの左端から始まり、コマンドプロンプトの最後の位置からは始まりません。

変数をどこでオーバーライドするのですか？これを行う最も論理的な方法は〜/ .bashrcです。

招待状の外観はあなたの想像力によってのみ制限されます。原則として、上記のツールを使用してコマンドラインプロンプトをクリスマスツリーの形式で表示することを妨げるものはありません。確かに、そのような招待は非常に面倒であり、その仕事は確かにより便利ではありません。ログイン時に、クリスマスツリーをコマンドラインプロンプトの上に表示するだけです（〜/ .bashrcを編集する必要があります）。頑張れ！そして見てくれてありがとう。Cloud4Y
