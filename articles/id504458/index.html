Menginstal OpenCV-Python di lingkungan virtual untuk ketel-super

Di sini Anda akan menemukan contoh langkah-demi-langkah menginstal pustaka OpenCV dengan Python.


  • Instalasi python
  • Penyiapan lingkungan virtual
  • Instal OpenCV + jupiterlab, numpy, matplotlib
  • Pengujian

Saya menguji semuanya pada tablet Microsoft Surface, Windows 10 Pro, dengan sistem operasi 64-bit.


Misalkan perangkat Anda tidak diinstal terlebih dahulu.


  1. Pertama instal Python .
    Unduh versi yang diinginkan dan jalankan file .exe. Jangan lupa centang kotak add path. Saya menginstal Python 3.7.3 dari 25 Maret 2019, karena versi baru Python 3.7.4 dari 8 Juli 2019 tidak berfungsi dengan benar, yaitu, beberapa perintah tergantung di terminal. Buka baris perintah.
  2. Instal virtualenv.
    Kita membutuhkan lingkungan virtual sehingga setiap proyek individu memiliki "ruang" sendiri dengan versi sendiri dari perpustakaan yang diinstal, yang tidak akan bergantung pada proyek lain dan menjadi bingung satu sama lain.
    Kami akan menginstal paket menggunakan pip. Ini telah berjalan secara langsung dengan Python baru-baru ini, tetapi biasanya perlu diperbarui dengan perintah:
    python -m pip install --upgrade pip
    Diperbarui pip, sekarang instal lingkungan virtual : Dengan
    pip install virtualenv
    perintah, cdbuka folder di mana Anda ingin membuat lingkungan dan masukkan perintah:
    mkdir opencvtutorial_env- ini adalah bagaimana kami menciptakan lingkungan yang disebut opencvtutorial_env.
    Selanjutnya, masukkan perintah virtualenv opencvtutorial_envdan untuk mengaktifkan, pergi ke folder lingkungan dan kemudian gunakan Tab untuk mengaktifkan.
    .\opencvtutorial_env\Scripts\activate
  3. OpenCV-Python, Numpy Matplotlib, opencv.
    . :
    pip install opencv-python
    opencv-python numpy. matplotlib: pip install matplotlib.
