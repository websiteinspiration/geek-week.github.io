نشر وتكوين مصادقة عقدة حمراء على إنشاء عامل الميناء



نشر عقدة حمراء إلى إنشاء عامل الميناء مع تمكين التفويض واستخدام حجم عامل الميناء.



قم بإنشاء ملف docker-compose.yml: 



version: "3.7"

services:
  node-red:
    image: nodered/node-red
    environment:
      - TZ=Europe/Moscow
    ports:
      - "11880:1880" # 11880 -     , 1880 -     node-red  .
    volumes:
      - "node-red:/data" # node-red -    docker   , /data -   .
    restart: always
volumes:
  node-red: #   node-red  .


نطلق الحاوية في وضع التصحيح (يجب أن تكون البداية الأولى بدون مفتاح ` -d` ، لعرض الأخطاء عند ظهورها): 



$ docker-compose up node-red
Creating node-red_node-red_1_3e3e59f5e044 ... done
Attaching to node-red_node-red_1_bca4cb987984
node-red_1_bca4cb987984 |
node-red_1_bca4cb987984 | > node-red-docker@1.0.3 start /usr/src/node-red
node-red_1_bca4cb987984 | > node $NODE_OPTIONS node_modules/node-red/red.js $FLOWS "--userDir" "/data"
...


نوقف الحاوية ونشغل الأمر لعرض الحجم: 



$ docker volume ls
DRIVER              VOLUME NAME
local               node-red_node-red


عرض معلومات مفصلة عن الحجم: 



$ docker volume inspect node-red_node-red
[
    {
        "CreatedAt": "2020-05-02T18:37:33Z",
        "Driver": "local",
        "Labels": {
            "com.docker.compose.project": "node-red",
            "com.docker.compose.version": "1.23.0",
            "com.docker.compose.volume": "node-red"
        },
        "Mountpoint": "/var/lib/docker/volumes/node-red_node-red/_data", #

انتقل إلى دليل الصوت. يحتوي هذا الدليل بالفعل على الملفات التي أنشأها عامل الإرساء عند نشر الحاوية. 



$ sudo ls /var/lib/docker/volumes/node-red_node-red/_data
lib  package.json  settings.js


نحن مهتمون بملف settings.js . افتحه وابحث عن جزء من رمز adminAuth . أزعجها. 



 // Securing Node-RED
    // -----------------
    // To password protect the Node-RED editor and admin API, the following
    // property can be used. See http://nodered.org/docs/security.html for details.
    adminAuth: {
        type: "credentials",
        users: [{
            username: "admin",
            password: "$2a$08$zZWtXTja0fB1pzD4sHCMyOCMYz2Z6dNbM6tl8sJogENOMcxWV9DN.",
            permissions: "*"
        }]
    },


في مفتاح كلمة المرور: أدخل تجزئة عقدة كلمة المرور الحمراء.



الحصول على عقدة كلمة مرور التجزئة حمراء
قم بتثبيت حزمة node-red-admin على أي جهاز باستخدام node.js. 



npm i node-red-admin -g


: 



node-red-admin hash-pw




قم بتشغيل الحاوية والاتصال بالمنفذ 11880 . 



http://192.168.0.100:11880/


يجب أن تظهر نافذة تفويض.







أدخل كلمة مرور تسجيل الدخول.



إذا كان كل شيء يعمل ، فأعد تشغيل الحاوية باستخدام المفتاح -d . 



$ docker-compose up -d node-red


شيء من هذا القبيل.
