Windows Terminal Preview 0.7がリリースされました

新しいWindowsターミナルプレビューリリースのご紹介！これはリリースv0.7です。いつものように、ターミナルはMicrosoft Store  またはGitHubのリリースページから  ダウンロードできます  。そしてここにニュースがあります：

Windowsターミナルの更新


パネル


これで、ターミナルウィンドウをいくつかのパネルに分割できます。これにより、1つのタブで複数のコマンドラインを同時に開くことができます。

注：現在、デフォルトのプロファイルは新しいパネルでのみ開くことができます。選択したプロファイルを開くことは、将来のリリースに含める予定のオプションです。 カットの下の他のアップデートについての詳細を読んでください。





このリリースでは、パネルアクションを呼び出すために、次のキーボードショートカットがデフォルトで含まれています。

{ "command": "splitHorizontal", "keys": [ "alt+shift+-" ] },
{ "command": "splitVertical", "keys": [ "alt+shift+plus" ] },
{ "command": { "action": "moveFocus", "direction":"down" }, "keys": [ "alt+down" ] },
{ "command": { "action": "moveFocus", "direction":"left" }, "keys": [ "alt+left" ] },
{ "command": { "action": "moveFocus", "direction":"right" }, "keys": [ "alt+right" ] },
{ "command": { "action": "moveFocus", "direction":"up" }, "keys": [ "alt+up" ] },
{ "command": { "action": "resizePane", "direction":"down" }, "keys": [ "alt+shift+down" ] },
{ "command": { "action": "resizePane", "direction":"left" }, "keys": [ "alt+shift+left" ] },
{ "command": { "action": "resizePane", "direction":"right" }, "keys": [ "alt+shift+right" ] },
{ "command": { "action": "resizePane", "direction":"up" }, "keys": [ "alt+shift+up" ] },

タブの並べ替え


これでタブを並べ替えることができます！このMicrosoft Igniteのプール要求を実装してくれたJames Clark（@Clarkezone）に感謝します。

注意。現在、管理者としてターミナルを起動するときにタブを並べ替えることはできません（試行するとクラッシュします）。また、順序を変更すると、タブのユーザーインターフェイスが表示されなくなる場合があります。この問題はすでにGitHubで追跡されており、将来のアップデートで修正される予定です。



UIの改善


ターミナル周辺の太い境界線はなくなりました。境界線ははるかに細く、Windows設定の[色]ページでカスタマイズすると、アクセントカラーが表示されます。。

注意。このオプションが有効でない場合、フレームは白になります。





バグの修正


このリリースに含まれる主なバグ修正の一部を以下に示します。

  • 挿入時に行末が正しく動作する
  • Alt +矢印キーで余分な文字が印刷されなくなりました

カスカディアコードの更新


Cascadia Codeがメジャーアップデートを受け取りました！新機能は次のとおりです。

  • ギリシャ語、キリル文字、ベトナム語のアルファベットがサポートされるようになりました。
  • 現在、Powerlineバージョン（Cascadia Code PL）があります。
  • コード合字なしのバージョンもあります（Cascadia Mono）

Cascadia Codeを最大限に活用するには、GitHubリポジトリから好きなフォントファイルをダウンロードしてください  。

注：Windowsターミナルには、コード合字のみを含む基本バージョンが付属しています。
