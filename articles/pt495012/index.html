Você segue agora com emoções

Olá, eu sou a Lena do JetBrains! Na situação atual, parece especialmente importante para você e a equipe do YouTrack manter nossas promessas sobre o que está planejado para lançamento em 2020. Portanto, tenho o prazer de apresentar o YouTrack 2020.1, no qual essa alegria e outras emoções agora podem ser expressas usando reações emoji. Agora eles podem responder aos comentários sobre as tarefas, e o centro de notificações o manterá informado sobre as reações que você receber.



Para todos que desejam contribuir com a tradução do produto para outros idiomas, o modo de tradução agora está disponível diretamente no contexto da interface do YouTrack. Para os administradores, o YouTrack 2020.1 introduziu novas configurações de campos de tarefas para dar suporte a casos complexos de acesso a campos individuais sem gravar fluxos de trabalho. Para desenvolvedores, aprimoramos a integração com os sistemas de controle de versão.

Mais sobre inovações - mais adiante no post.

Espero que os novos recursos também o encantem, pois nos regozijamos em cada stand-up, discutindo a natação "Reações" no quadro.

Resposta aos comentários de todos


As reações são uma maneira fácil de responder rapidamente a um comentário no YouTrack. Os usuários adoram emojis porque podem substituir uma palavra ou até uma frase inteira - com reações que você não pode escrever comentários desnecessários ao discutir uma tarefa.
Para não perder acidentalmente uma reação importante, o centro de notificações possui uma lista de reações recentes aos seus comentários. A partir daí, você pode abrir imediatamente a tarefa desejada.



O YouTrack ainda é um gerenciador de tarefas, não um mensageiro, por isso decidimos começar com um pequeno conjunto de emojis. Vamos desenvolver a biblioteca de acordo com as opiniões dos usuários. Portanto, compartilhe suas impressões conosco.

Modo de tradução de interface do contexto para usuários estrangeiros


O YouTrack é traduzido para inglês, espanhol, alemão, russo, francês e japonês. Além disso, você pode mudar para um dos idiomas traduzidos por nossos usuários. É húngaro, hebraico, italiano, chinês simplificado, coreano, polonês, português e tcheco.



Tentamos simplificar o processo de localização da interface para quem deseja contribuir. No modo de tradução do contexto, o YouTrack destaca todos os elementos da interface que ainda não foram traduzidos e você pode oferecer suas opções diretamente no aplicativo. Depois que o JetBrains aprovar sua tradução, ela ficará visível para todos que usam o YouTrack neste idioma.

O processo de tradução é visível em tempo real. Você pode até traduzir a parte dinâmica do aplicativo. Uma vez ativado, esse recurso de tradução fica disponível para todos os usuários do seu servidor YouTrack. Se você não deseja distrair seus colegas de equipe com esta oportunidade, basta usar o plano tarifário gratuito e criar um servidor YouTrack separado especificamente para tradução.



Configurações finas de acesso a campos em tarefas para administradores


Novas configurações para acessar campos em tarefas permitem selecionar grupos de usuários ou equipes individuais que podem exibir ou atualizar valores de campo.

Anteriormente, você poderia usar campos particulares para restringir o acesso. Mas foi uma decisão da categoria "tudo ou nada": se você tivesse acesso para visualizar e atualizar um campo privado no projeto, nada o impedia de exibir e atualizar os valores de quaisquer outros campos privados no mesmo projeto.

No YouTrack 2020.1, um administrador pode abrir o acesso para visualizar e atualizar cada campo individualmente. Além disso, agora os administradores de projeto podem criar um grupo dedicado de usuários que podem ver determinados campos e outro - aqueles que podem atualizá-los. Em vez de escrever fluxos de trabalho complexos, basta usar as novas configurações de campo para casos incomuns de distribuição de funções entre equipes.



Integração aprimorada de VCS e TeamCity para desenvolvedores


Integração aprimorada do YouTrack com os sistemas TeamCity, Upsource e controle de versão, como o GitHub.

Configurações gerais de integração do VCS para vários projetos


Graças à integração, você pode anexar alterações do VCS à tarefa nos comentários aos quais o identificador desta tarefa é mencionado. Agora você pode primeiro configurar a integração para um projeto e depois distribuir os parâmetros selecionados para outros projetos, simplesmente selecionando-os nas configurações. Assim, outros projetos também exibirão alterações anexadas às tarefas. Para compartilhar a configuração, basta selecionar os projetos necessários.



Processando Tarefas Relacionadas


A integração entre o YouTrack e o TeamCity agora permite escolher com mais precisão quais tarefas correspondem a uma construção específica. Como parte da integração com o TeamCity, as tarefas mencionadas nas alterações do VCS incluídas na compilação são consideradas "relacionadas" a essa compilação. Quando a construção está sendo processada, as tarefas relacionadas no status “Concluído” são atualizadas com o número da construção correspondente. Agora você pode especificar uma condição que determinará quais tarefas relacionadas devem ser atualizadas.



Suporte do Swagger


Outra melhoria relacionada ao desenvolvimento: o arquivo Swagger Open API 3.0 embutido apareceu no YouTrack 2020.1. Com ele, você pode usar a API REST para integrar seu aplicativo ao YouTrack. Em desta página GitHub, você pode encontrar ferramentas compatíveis de geração de código para a sua execução.

Permissão "Aplicar o comando sem notificações"


A nova permissão permite aplicar alterações em massa a tarefas sem enviar notificações para quem as observa. Por padrão, a permissão é ativada para administradores de projeto e administradores de sistema que podem conceder a outros usuários.
