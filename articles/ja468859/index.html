「ポンピング用ルーター」：インターネットプロバイダー向けのTP-Link機器チューニング 

最新の統計によると、3300万人以上のロシア人がブロードバンドインターネットを使用しています。加入者ベースの成長は鈍化していますが、プロバイダーの収益は増え続けています。これには、既存のサービスの品質の向上や新しいサービスの出現などが含まれます。シームレスなWi-Fi、IP-TV、「スマートホーム」-これらの分野の開発のために、事業者はDSLからより高速なテクノロジーに切り替え、ネットワーク機器を更新する必要があります。この投稿では、インターネットプロバイダー向けのTP-Linkの機能とそれらをどのように支援できるかについて詳しく説明します。



インターネット開発統計


TMTコンサルティングの調査によると、2019年の第2四半期に、ロシアのブロードバンドインターネット市場は353億ルーブルに達し、前年比3.8％増加しました。普及率は60％に達しましたが、3,330万のプライベートブロードバンド加入者の70％は、ロシアの5大プロバイダーがサービスを提供しています。

  • 1190万（36％）-ロステレコム;
  • 380万（12％）-ERテレコムホールディング;
  • 335万（10％）-MTS;
  • 240万（7％）-ビーライン;
  • 180万（5％）-TransTeleCom（TTK）。

同時に、加入者ベースの成長率は低下しました：2019年の第2四半期の1.6％は、2018年の同時期の2.3％に比べました。研究者は、市場が飽和段階に入ったと指摘しています。それにもかかわらず、プロバイダーは利益を増やし続けています。ブロードバンド加入者あたりの平均月収は9ルーブル増加しました-2018年の347ルーブルから現在は356ルーブルです。関税の引き上げだけでなく所得も伸びている。 TMTコンサルティングによると、オペレーターはアクセス品質を改善し、新しいサービスとサービスを提供します。

プロバイダーは調査結果を確認します。 Rostelecomのプレスサービスレポート：高速光ファイバー技術は、レガシーDSLネットワークに徐々に取って代わっています。これにより、IPTVなどの追加サービスの技術的基盤が生まれます。 ERテレコムの代表者はまた、プロバイダーにとって比較的新しいニッチ、「スマートインターコム」、「デジタルテレビ」、および「スマートシティ」および「デジタルカントリー」プロジェクトの成長の可能性を認識しています。


インターネットへのアクセス、2018年のデータ

インターネットがより深く浸透し、カバレッジが拡大し、提供されるサービスの品質が向上しています。ちなみに、国家プログラム「ロシア連邦のデジタル経済」では、2024年までに、国内の世帯の97％が100 Mbit / sの速度でブロードバンドにアクセスできるようになると規定しています。このタスクを実行するには、プロバイダーから多大なコストがかかります。たとえば、Rostelecomは新しいネットワークに500〜700億ルーブルを投資する予定です。北西連邦管区でのみ、123億ルーブルの25.6千kmの光ファイバーケーブルが敷設されます。

大中規模のISP向け：工場でのカスタマイズとリモートデバイス管理


ロシアのプロバイダーは、数万キロのファイバーでは対応できません。最新のスイッチ、ルーター、コントローラー、Wi-Fiアクセスポイント、トランシーバー、メディアコンバーターなどが必要になります。当社の工場の1つは、ネットワークデバイスの製造とそのカスタマイズに専念しています。



お客様のご要望に応じて、ファームウェアの変更、機器の追加機能の搭載、逆にその機能の制限などを行います。

Wi-FiルーターはER-Telecom用に特別に開発されたもので、IPv6タイプの自動選択が追加されました。TR-069ベンダー固有のノードにより、オペレーターはプロアクティブなサービスのために機器のステータスとパフォーマンスを監視することができます。Wi-Fiドライバーを調整すると、2.4 GHzチップセットと5 GHzチップセットの間でバランスが取れるようになり、WLAN速度が2倍に向上しました。バンドステアリングも強化されています。



手順、デバイス自体の外観、およびパッケージングもカスタマイズされています。消費者はロシア語で必要な情報を学ぶことができます。以下は、ロシアのプロバイダー向けのパッケージとケースのカスタマイズの例です。







現在、インターネットプロバイダーにWi-Fiルーターの3つの主要モデルを提供しています。

  • TL-WR850N（イーサネット100 Mbps、2.4 GHz Wi-Fi 300 Mbps）。
  • Archer C20（イーサネット100 Mbps、2.4 GHz Wi-Fi 300 Mbps、5 GHz Wi-Fi 433 Mbps）。
  • Archer C5（イーサネット1 Gb /秒、2.4 GHz Wi-Fi 300 Mb /秒、5 GHz Wi-Fi 867 Mb /秒）。

すべてのルーターはIPv6、ペアレンタルコントロール、およびTR-069プロトコルをサポートしています。これにより、オペレーターはリモートでエンドユーザーデバイスを構成および管理できます。最新の標準に従って、この装置は、IPTVサービスおよびゲストネットワーク用のIGMPプロキシ、「ブリッジ」モード、802.1Q TAG VLANの動作を保証します-ゲストの個別のアクセス。イーサネットポートとWi-Fiの速度に加えて、Archer C5はUSB 2.0ポートの存在によって区別されます。これにより、3G / 4Gモデムを接続し、ネットワーク上でファイルまたはメディアを共有できます。

リモート管理用のTP-Link ACSサーバー


オペレーターは、ACSサーバーなどのツールを使用して、すべてのサブスクライバーのルーターをすぐにフラッシュしたり、特定の制限を設定したり、設定を変更したりすることができます。



Agile ACSホームページには、デバイスのステータスが図で表示されます。グラフのセクターまたは下線付きの番号をクリックすると、詳細を表示できます。



[デバイス]テーブルには、登録されたデバイスに関する基本情報（シリアル番号、モデル、ソフトウェア情報、IPアドレスなど）が表示されます。フィルターを使用して、指定された基準を満たす特定のデバイスを検索できます。



情報タブでは、現在の設定を確認して変更できます。









TR TREEはデバイスノード情報を表示します。検索ボックスで、特定のノードを見つけて構成できます。



ドロップダウンメニューを展開して、その他の設定にアクセスできます。



このツールを使用すると、1つのデバイスを監視して構成するだけでなく、アプリケーションでフィルタリングする機能を使用して、ファームウェアと構成ファイルを大幅に更新することもできます。現在、ACSは、Archer C5、Archer C20、TLWR840N、およびTL-WR850Nの4つのモデルをサポートしています。



ACSは、最大800 MBの最新のログを保存します。ログファイルのサイズが100 MBに達するとすぐに、システムはそれらをアーカイブします。デフォルトでは、デバイスID、時刻、ログの内容など、最新のログを最大200 MB表示できます。 



[システム設定]セクションで、ACS構成を表示および変更できます。管理者が設定できるホストIPアドレスに加えて、システムは永続的な管理IPアドレス169.254.0.199を提供します。



小規模プロバイダーの場合：独立した接続とカスタマイズ


少量の機器を購入する地元のインターネット企業にとって、工場のカスタマイズを注文するか、ACSのライセンスを取得することは不利です。そのために、TP-Linkルーターの基本設定をプロバイダーのネットワークの機能に適合させる代替ソリューションを提供しています。Agile Configユーティリティでカスタマイズされたデバイスは、完全なリセット後でも変更されたファームウェアを保存します。ユーザーは、ランダムリセットでネットワークを「破壊」することはできません。これにより、オペレーターの技術サポート部門の負担が大幅に軽減されます。

Agile Configを使用すると、SSID、WAN接続タイプ、パスワード、タイムゾーン、言語を変更できます。すべてのTP-Linkルーターに一般的な排他設定を設定するか、ルーターごとに個人設定を行うことができます。さらに、このユーティリティを使用すると、Webインターフェイスのブランドを設定できます-TP-Linkロゴをプロバイダー独自のロゴに変更します。 Agile Configでは、特定の機密設定（TR-069など）をロックおよび非表示にすることも計画されています。

ユーティリティを入手するには、サイトhttps://agile.tp-link.com/en/で簡単な登録手順を実行します。メールアドレスを確認したら、アカウントにログインして入力し、会社に関する情報を入力します。アプリケーションは24時間以内にレビューされ、Agile Configコンポーネント（Agile ServerおよびISP Generator）をダウンロードできるようになります。 

我々は持って準備された ビデオ命令、我々はそれをインストールして使用する方法について説明ユーティリティ、用を。



コンピューターにソフトウェアをインストールした後、ルーターの1つに接続して、すべてのデバイスにインストールするための一般的な設定を作成します。これを行うには、ルーターの設定ページにアクセスしてパスワードを設定します。管理者などの単純なものか、ユーザーが設定にアクセスできないようにするためのより複雑なもののいずれかです。必要な設定を行い、Wi-Fiネットワークの新しい名前とパスワードを設定します。設定は「バックアップ」セクションで保存されます。

各ルーターの個別の設定は、ISP Generatorユーティリティを介して設定されます。これを行うには、MAC.BIN.xlsファイルを作成し（プログラムがこれを自動的に行います）、Excelでファイルを開いて変更します。現在構成中のルーターのMACアドレス（データはデバイスの背面パネルに表示されます）、およびその他の個々の設定を示します。Webインターフェースにアクセスするためのログインとパスワード、PPPoE接続、Wi-Fiネットワークの場合。静的IPアドレスを使用する場合は、そのパラメーターをここで設定する必要があります。ファイルを保存するには、ISPジェネレータを再度使用します。



設定を適用するには、ルーターとコンピューターを任意のスイッチに接続します。コンピューターで、静的IPアドレス192.168.66.10を設定します。マスクがデフォルトです。その後、作成した両方の設定を含むファイルを同じフォルダーに移動します。ルーターをブランド化する場合は、ロゴとファビコンを配置します。サイズは6 kVを超えないようにしてください。



Agile Serverユーティリティを管理者として実行します。[ワークスペース]フィールドで、ファイルを含むフォルダへのパスを指定し、[OK]をクリックすると、サービスが自動的に開始されます。Agile ConfigはTL-WR850N、Archer C20およびArcher C5ルーターをサポートします。このユーティリティを使用すると、デバイスの大きなプールを同時にフラッシュできます。そのサイズは、スイッチポートの数によってのみ制限されます。

結論


1つの投稿でインターネットオペレータ用のすべてのTP-Link機器について詳しく説明する場合、最後まで読むのに忍耐力がないでしょう。ここでは、ロシアのプロバイダーの間で最も人気のあるTP-Link製品とサービスのみを紹介しました-実際には、もっとたくさんあります。提示されたルーターは、独自のソフトウェアを使用した工場でのカスタマイズとセルフチューニングの可能性を考慮して、ブロードバンドインターネットへの優れたアクセスと追加サービスのサポートを提供します。全体として、これはほとんどのロシアのISPと消費者のニーズを満たします。
