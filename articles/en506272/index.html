Vivaldi 3.1 - Notable joys



Hello, Habr!

When working on a computer on the network, we spend most of our time in the browser, and the main task that we solve at the same time is obtaining or creating information. Actually, everything is simple. But only in words. In fact, behind these simple descriptions is the very complex work of many programs and their components, which allows us today to easily and effortlessly access information of almost any volume and value. But this is only part of the work - the received information still needs to be processed and transformed for future use. And for these purposes, software solutions are again needed.

Someone prefers to have a separate program for each type of activity on the network, but it is much more convenient when everything you need is already at hand, in a browser. That is why we are trying to integrate components into the browser that allow us to cover the maximum range of tasks performed by the user. This release is not an exception, in which another unique tool of its kind has appeared. This is where we begin today with the new version of the Vivaldi 3.1 browser for desktops.

Today we are pleased to introduce you a tool similar to which is not yet available in other browsers - it is a full-featured note manager. Access to it is carried out in one click using the button in the start page menu:



You can also open this manager through the context menu of notes in the side panel. The notes manager interface consists of a note editing window and several columns with lists of notes themselves and their data:

All in all, a great tool for those who prefer to keep important texts handy. So, just in case.

Another new feature in today's browser version is the menu editor. This function is the first step in fulfilling a very popular and difficult to implement request from users who need maximum browser customization for their tasks and habits. So far, you can edit only the main menu of the browser, but in the future the other menus will also become customizable.



Actually, the menu editor is in the settings, in the Appearance section. Two fields, in the left - the current menu, in the right - the items available for adding. You can add and remove these items in the main menu, as well as add separators and move menu items, changing their places - the most frequently used ones are raised up, and the less used ones are dropped. It is worth noting that you can make individual settings for each menu option - both in the form of the V button, and in the classic, horizontal view.

Other changes are not so obvious, but quite important. First of all, we can note the optimization of launching the browser - it should now start faster.
