シェーダーを使用してゲーム内のパレットを置き換える

このブログでは、バガボンドゲームで積極的に使用している、パレットの交換という私のお気に入りのテクニックを紹介します。

パレットの交換は、テクスチャパレットの変更です。この記事では、シェーダーを使用して実装します。昔は、メモリを不必要に浪費することなくリソースに可変性を追加できる便利なテクニックでした。現在、手続き型の生成で新しいリソースを作成するために使用されています。



画像の準備


最初のステップは、パレットを置き換えるためのイメージを準備することです。では、ビットマップ、各ピクセルの色が含まれていますが、我々は代わりにパレットにその色のインデックスを含める必要があります。このため、画像の構造（同じ色の領域）を実際の色から分離します。

実際、一部の画像形式はこの保存方法をサポートしています。たとえば、PNG形式には、インデックス付きの色を保存する機能があります。残念ながら、画像がインデックス付きモードで保存されている場合でも、多くの画像読み込みライブラリは色の配列を作成します。これは私が使用するSFMLライブラリにも当てはまります。内部では、stb_imageを使用して、画像の「パレットを自動的に削除」します。インデックスを対応するパレットの色に置き換えます。

したがって、この問題を回避するには、イメージとパレットを別々に保存する必要があります。画像はグレーの濃淡で記録され、各ピクセルのグレーレベルはパレットのカラーインデックスに対応します。

受け取る予定の例を以下に示します。


これを実現するには、Pillowライブラリを使用する小さなPython関数を使用します。

import io
import numpy as np
from PIL import Image

def convert_to_indexed_image(image, palette_size):
    # Convert to an indexed image
    indexed_image = image.convert('RGBA').convert(mode='P', dither='NONE', colors=palette_size) # Be careful it can remove colors
    # Save and load the image to update the info (transparency field in particular)
    f = io.BytesIO()
    indexed_image.save(f, 'png')
    indexed_image = Image.open(f)
    # Reinterpret the indexed image as a grayscale image
    grayscale_image = Image.fromarray(np.asarray(indexed_image), 'L')
    # Create the palette
    palette = indexed_image.getpalette()
    transparency = list(indexed_image.info['transparency'])
    palette_colors = np.asarray([[palette[3*i:3*i+3] + [transparency[i]] \
        for i in range(palette_size)]]).astype('uint8')
    palette_image = Image.fromarray(palette_colors, mode='RGBA')
    return grayscale_image, palette_image

最初に、関数は画像をパレットモードに変換します。彼女はそれをグレースケール画像として再解釈します。次に、パレットを取得します。何も複雑ではありませんが、主な作業は枕ライブラリによって行われます。

シェーダー


画像を準備したら、パレットを置き換えるシェーダーを作成する準備ができました。パレットをシェーダーに転送するには2つの方法があります。テクスチャまたは同種配列を使用できます。ユニフォーム配列の方が使いやすいので、それを使いました。

これが私のシェーダーです。GLSLで作成しましたが、シェーダーを作成するために別の言語に簡単に転送できると思います。

#version 330 core
in vec2 TexCoords;

uniform sampler2D Texture;
uniform vec4 Palette[32];

out vec4 Color;

void main()
{
    Color = Palette[int(texture(Texture, TexCoords).r * 255)];
}

テクスチャを使用して、現在のピクセルの赤いチャネルを読み取ります。赤いチャネルは0〜1の範囲の浮動小数点値なので、255で乗算して変換intし、0〜255 の元のグレーレベルを取得します。これは画像に格納されます。次に、それを使用してパレットから色を取得します。

記事の冒頭のアニメーションは、ゲーム内のスクリーンショットから取られたもので、次のパレットを使用してキャラクターの体の色を変更しています。
