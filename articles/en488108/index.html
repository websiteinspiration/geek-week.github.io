SAP HANA Lab in Azure



As part of the SAP Digital Exchange event, we invite you to Microsoft labs to deploy SAP HANA in Azure, February 20, 2020.

The workshop consists of designing a migration process and hands-on work on deploying SAP HANA in Azure. Under the guidance of a Microsoft trainer, you will walk through the steps to install and configure system replication.





According to the results of laboratory work, you will be able to:

  • Prepare your Azure infrastructure components yourself
  • Set up a virtual machine
  • Deploy SUSE Linux Enterprise Clustering
  • Install SAP HANA
  • Set up system replication
  • Test high availability scenarios

To register, follow the link to the HandsOn tab or look for laboratory work in a search on the site for the SAP HANA tag in Azure.
