C ++ Build Insights SDKを使用してプログラムでビルドを分析する

CおよびC ++ APIを介してMSVCビルド時間情報にアクセスできるフレームワークであるC ++ Build Insights SDKのリリースをお知らせし  ます。このリリースに加えて  、GitHubでvcperfオープンソースを公開しています。vcperf自体はSDKを使用して構築されているため、独自のツールを開発する際の参照として使用できます。SDKを使用してどのアプリケーションを作成するかを喜んでお知らせします。フィードバックをお待ちしています！



バックグラウンド


昨年11月、MSVCユーザーがビルド時間を理解するのに役立つvcperfとそのWindowsパフォーマンスアナライザー（WPA）プラグインを導入しました。両方のコンポーネントは、C ++ Build Insightsの後援の下で発表されました。しかし、C ++ Build Insightsとは何ですか？

11月に、C ++ Build Insightsは Windowsオペレーティングシステムで使用できる便利なトレースメカニズムであるWindowsのイベントトレース（ETW）に基づいていると既に述べ  ました。しかし、ETWは、お客様が作成した非常に大規模なC ++ビルドにテクノロジーを拡張するのに十分ではありませんでした。イベントモデルと使用する分析アルゴリズムを微調整する必要がありました。この作業の結果、MSVCの新しいデータ分析プラットフォームが作成されました。このプラットフォームは、現在C ++ Build Insightsと呼ばれています。

今日、C ++ Build Insightsプラットフォームは、vcperfおよび一部の内部ツールがサポートするものです。それにもかかわらず、私たちはあなた方全員にこれから利益を得る機会を与えたかったのです。これを行うために、CおよびC ++インターフェースを使用してそれを組み立て、完全なソフトウェア開発キットを作成しました。

C ++ Build Insights SDKを使ってみる

#include <iostream>
#include <CppBuildInsights.hpp>
 
using namespace Microsoft::Cpp::BuildInsights;
using namespace Activities;
 
class LongCodeGenFinder : public IAnalyzer
{
public:
    //     ,    
    //    (trace). 
    AnalysisControl OnStopActivity(const EventStack& eventStack) override
    {
        // ,    
        //  TopFunctionsFinder::CheckForTopFunction's.
        //   ,     .
 
        MatchEventStackInMemberFunction(eventStack, this, 
            &LongCodeGenFinder::CheckForLongFunctionCodeGen);
 
        //       
 
        return AnalysisControl::CONTINUE;
    }
 
    //        , 
    //  CodeGeneration,     ,
    //      500 .
 
    void CheckForLongFunctionCodeGen(CodeGeneration cg, Function f)
    {
        using namespace std::chrono;
 
        if (f.Duration() < milliseconds(500)) {
            return;
        }
 
        std::cout << "Duration: " << duration_cast<milliseconds>(
            f.Duration()).count();
 
        std::cout << "\t Function Name: " << f.Name() << std::endl;
    }
};
 
int main(int argc, char *argv[])
{
    if (argc <= 1) return -1;
 
    LongCodeGenFinder lcgf;
 
    //    ,       (trace).     ; !
    auto group = MakeStaticAnalyzerGroup(&lcgf);
 
    // argv[1]      
    int numberOfPasses = 1;
    return Analyze(argv[1], numberOfPasses, group);
}

vcperfがオープンソースになりました


vcperf自体はC ++ Build Insights SDKを使用して作成され  、GitHubでオープンソースにしています。これを使用してSDKの詳細を学習し、必要に応じてvcperfをカスタマイズしてください。リポジトリには、エラーのために再起動されたリンカーを検出するためにvcperfを拡張するコミット例が含まれています  。この例では、WPAのC ++ Build Insightsのビルドエクスプローラービューでこれらの呼び出しを強調表示しています。このコミットの例を次の順序で読むことをお勧めします。

  1. RestartedLinkerDetector.h
  2. BuildExplorerView.cpp
  3. Commands.cpp
