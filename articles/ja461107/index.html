1つではありません。最も美しく、非常に高価な放射測定装置でさえ、「リアルタイム」で機能しません。彼は、単位時間あたりのイベント（量子のクリック）の数を測定し、これらの値を統合して、そこに定められたアルゴリズムに従って結果を生成する必要があります。最も新しいファッションの線量計でも使用されているソフトウェアや数学モデルに関係なく、主な主力製品は、すでに100年以上前のデバイスです。これはガイガーカウンターです。その中心には、1908年にドイツの物理学者ハンスガイガーによって提案されたこの装置は、ピエールキュリーが使用していた電離箱の続きであり、低圧のガスで満たされた電気コンデンサーでした。 1928年、ガイガーの指導の下、ウォルターミュラーはいくつかのタイプの放射線カウンターを作成しました。さまざまな電離粒子の登録を目的としています。デバイスのこの変更はガイガーミュラーカウンターです。


センサーはさまざまな種類の放射線に対して選択性がないため、電離放射線の強度の一般的な測定に完全に適しています。

ガイガーミュラーガス放電カウンターは、通常、十分に排気された密封ガラスまたは金属管の形で作られています。硬いベータ線とガンマ線に反応するカウンターのシリンダーは、通常、壁の厚さが0.05〜0.3 mmのステンレス鋼製のシリンダーの形状をしています。通常、カウンターは表面全体で放射を感知しますが、これのためにシリンダーに特別な「ウィンドウ」が設けられているものもあります。アルファ線とソフトベータ線に敏感なカウンターの入力ウィンドウは、3〜17ミクロンの厚さのマイカまたはマイラーでできています。X線カウンターの窓はベリリウムで作られ、紫外線カウンターは石英ガラスで作られています。

薄肉の金属管の場合は、剛性と外気圧に対する抵抗を与えるためにさらに波形になっているため、圧縮することはできません。チューブの端には、ガラスまたは熱硬化性プラスチックで作られたシール絶縁体があります。また、計器回路に接続するための端子キャップも含まれています。細いワイヤーをチューブの軸に沿って伸ばし、金属のシリンダーをチューブと同軸に配置します。下の画像-「全国」カウンターSBM-20の内部構造


チューブとワイヤーの両方が電極です：カソードチューブとアノードワイヤー。定電圧源からのマイナスは陰極に接続され、定電圧源からの大きな一定抵抗R-プラスは陽極に接続されます。


電気的には、分圧器が得られ、その中間点（抵抗とカウンターアノードの接合部）で、電圧はソースの電圧とほぼ等しくなります。メーターの電圧（通常300〜500 V）は、自己放電がなく、メーターに電流が流れないように選択されています。

メーターの操作は、衝撃イオン化に基づいています。放射性同位元素によって放出された量子は、カウンターの壁に落ちて、電子をノックアウトします。フラスコ内を移動し、ガス原子と衝突する電子は、それらから二次電子を打ち消し、正イオンと自由電子を生成します。陰極と陽極の間の電界は、衝撃電離が始まるエネルギーまで電子を加速します。

イオンなだれが発生します。電場の影響下で、電子は陽極の方向に加速され、正に帯電したガスイオンが管の陰極に向かって加速されます。したがって、メーターを流れる電流は瞬時に増加し、チューブの抵抗が減少します（これにより、分圧器の中点の電圧が低下します）。この場合、抵抗Ｒで電圧パルスが発生し、記録装置に供給される。カウンターがそれに入る次の粒子を登録できるように、なだれ電荷は消されなければなりません。これは自動的に行われます。電流パルスが抵抗Rに現れる瞬間、大きな電圧降下が発生するため、アノードとカソード間の電圧は急激に低下して放電を停止し、ミリ秒の数分後、メーターは再び動作の準備が整います。

ガスが充填されたカウンターでは、正イオンが陰極まで移動し、その近くで中和されて金属から電子を引き離します。これらの追加の電子は、それを防止して消火するための対策を講じない場合、次の放電の出現につながる可能性があります。ガイガー・ミュラーカウンターは非自己消火カウンターと呼ばれ、現在積極的に使用されている自己消火カウンターは、Trostによって提案された修正であり、充填ガス混合物の組成が異なることに注意してください。自己消火式メーターはより高速であり、それらを使用する場合、放電をクエンチする問題はありません。非自己消火カウンターで放電を停止するには、抵抗アノード回路に約10 GOhm（！）を含める必要があります。しかし、それでも、このようなスキームの大きな欠点は、時間分解能が低いことです。約10−3秒以上。

一般的なガス放電メーターでは、（電気的絶縁破壊を容易にするために）空気がバルブから排出され、不活性ガス（メーターに使用されるガス媒体が十分に小さい電子付着係数を持っている必要があるため不活性）が、代わりに、わずかな圧力下で、急冷添加剤の混合物とともに追加されます。アルコールは消光添加剤として使用され、後にハロゲン（塩素、臭素）が使用されました。アルコールからハロゲンへの移行は、そのようなメーターの製造ラインのコストの上昇にもかかわらず、検出器の寿命を向上させます。消光の過程で、多価アルコール分子はアセチレン、メタン、酸素などの分子に分解されます。平均的な幾何学的メーターは約10 20アルコール分子。各放電で、約10 10のアルコールイオンがそのようなカウンターで解離するため、10 10パルス後にすべての分子が崩壊します。これは、カウンターの劣化、その特性の変化（放電の点火電位の増加、プラトーの勾配の増加など）につながります。多くの場合、カウンタがあり、チェック中に常に「呼び出し」を開始し、指定されたモードでは動作できません。この現象は、ガス混合物中の急冷添加剤の臨界消費によって引き起こされます。これは、そのようなカウンターの深刻な動作時間の指標となる場合があります。理論的には、消火添加剤全体が「燃え尽きて」、メーターが非自己消火性になった場合、上記のように、10GΩの抵抗を使用して再度アニメーション化できます。

void setPwmFrequency(int pin, int divisor) {
  byte mode;
  if(pin == 5 || pin == 6 || pin == 9 || pin == 10) {
    switch(divisor) {
      case 1: mode = 0x01; break;
      case 8: mode = 0x02; break;
      case 64: mode = 0x03; break;
      case 256: mode = 0x04; break;
      case 1024: mode = 0x05; break;
      default: return;
    }
    if(pin == 5 || pin == 6) {
      TCCR0B = TCCR0B & 0b11111000 | mode;
    } else {
      TCCR1B = TCCR1B & 0b11111000 | mode;
    }
  } else if(pin == 3 || pin == 11) {

    }
    TCCR2B = TCCR2B & 0b11111000 | mode;
  }
}
