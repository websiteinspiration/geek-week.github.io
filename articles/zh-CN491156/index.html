可测量的共情：预测脑MRI的同情



机器人可以写交响乐吗？从技术角度来看，它可以。但是他能给她一种情感的色彩吗？这个问题仍在讨论中。然而，除了发达的大脑，直立的姿势等之外，人的主要区别特征之一是存在广泛的情感。该技能不是自然界中独有的，但在人类中已发展到足以将其与地球上其他生物区别开来。如果我们认为一个人是一个社会存在，那么许多情绪就以某种方式与一个人与另一个人或整个社会的互动联系在一起。这些情绪之一就是同情（或同理心）。

但是，由于这是一个抽象的概念，如何衡量共情？无论歌词指针在作品中如何指向心灵和灵魂作为所有情感的来源，它们都是由大脑产生的。并且可以分析大脑的工作。来自加利福尼亚大学（美国）的科学家进行了一项研究，其中确定了一个人的同理度不仅可以使用静止的脑部MRI来确定，而且可以预测调查的结果。移情的形成涉及大脑的哪些区域，它们是单独起作用还是发生共轭作用，这项研究如何帮助医学？我们从科学家的报告中了解到这一点。走。

学习基础


同情可以被称为一个人最无私的情感之一，因为它以对另一个人的情感状态的有意识同情的形式表现出来（理论上，如果动物甚至机器人表现出情感，则可以在这里添加它们）。

从进化的角度来看，有几种理论可以解释同理心。移情被认为是通过关怀他人促进亲社会和合作行为并预测周围个体的内部状态，行为和意图的必要条件。

换句话说，一个有同理心的人会通过各种刺激（视觉，声音等）接收有关另一个人的状态的信息，这些刺激会导致形成反应的大脑部分活动-同情心。

为了使移情作用发挥作用，我们的大脑必须以与对话者对我们自己的反应相同的方式对对话者的情绪做出反应。有时，当前额叶皮层受损时，可能会出现无意识的行为模仿行为（在这项研究中，科学家称这种情况为“共振”）。

从神经病学的角度来看，这种行为的最可能基础是所谓的“神经共振”-一种集体神经表示的现象，用于感知和厌恶，躯体感觉，情绪和运动行为。

自然地，影响共情和社会化程度的其他因素会添加到神经共鸣中：个人依恋，社会地位，在特定社会群体中的成员身份以及信任度。据认为，当上下文信息和意识评估与行为和决策中的情感，躯体感觉和运动过程相关联时，情感反应是自上而下的，这些行为和决策受前额叶和颞叶调节：颞顶叶（TPJ），背侧前额叶皮层（DMPFC）和背外侧前额叶皮层（DLPFC）。

这些监视神经元活动的系统在很大程度上与与意识评估过程以及移情或心理化的逻辑形式相关的系统重叠。控制机制很可能是抑制性的。一项研究证实了这一点，其中DMPFC和DLPFC的破坏性神经调节导致这些大脑区域对亲社会行为的抑制作用降低。

科学家认为，共振和控制过程是紧密共轭的，因此在认知功能内将它们分开非常困难。例如，据揭示，躯体运动和情感信息处理有助于评估他人的内部状态，信念和意图。

换句话说，每个负责其特定功能的大脑各个部分聚集在一起，以实现对对话者状态的情感反应，在这种情况下，形成同情。

因此，每个人的共情程度取决于神经系统共振和控制过程之间稳定，特征的相互作用。通过在活动状态下（执行某些任务的过程中）研究大脑的某些部位的实验证实了这一假设。

在同一项研究中，科学家建议可以通过评估静止大脑的功能来确定一个人的共情功能的个体差异。如果这一理论得到证实，则无需进行任何调查，问卷调查和测试来确定移情功能的程度（特别是考虑到这些方法不能给出准确的结果）。

此外，科学家们决定仔细检查不同年龄和性别代表之间在同理心方面大脑功能是否存在差异，因为早期的研究得出了相当矛盾的结果。

该研究涉及58位年龄在18至35岁之间的人（30位女性和28位男性）。受试者的主要选择标准是：惯用右手的人，没有任何先前记录的神经和/或精神疾病，没有发育障碍，没有药物或酒精滥用史。

在每个实验结束时，所有参与者都填写了人际反应指数（IRI）的问卷。这种问卷调查方法已经使用了很长时间，并且已经确立为评估受访者的认知和情绪状态的工具。问卷包括25份陈述，参与者对这些陈述的评价从5分（对我的描述不太好）到5分（对我的描述很好）的5分制。

陈述旨在测试共情的四个理论方面：

  • 幻想量表（FS）：感知虚构人物的倾向；
  • 移情态度（Empathic Concern-EU）：对他人痛苦的同情反应；
  • 观点采纳（PT）：倾向于接受他人观点的倾向；
  • 个人痛苦（PD）：对其他人的痛苦的不愉快反应。

在进行问卷调查之前，对参与者的大脑进行了MRI扫描，并保持镇静。特别注意了大脑的某些区域（表示为ROI-感兴趣区域，即感兴趣区域），其数目为185。


表1：在研究对象的大脑进行MRI扫描期间的感兴趣区域（ROI）。

研究成果


使用方差分析（ANOVA）对IRI得分进行分析。在前三个研究方面，男女之间没有特殊差异：FS（F = 2.68，p = 0.108），EU（F = 2.59，p = 0.114）和PT（F = 0.274，p = 0.603）。但是，女性的个人困扰程度明显高于男性（F = 9.79，p = 0.003）。


表2：男女IRI指标比较。

如上所述，在MRI期间，应特别注意大脑的某些区域：视觉，额顶叶，扣带腹，背侧，腹侧以及共振和控制区域。


图片编号1：指定共振和控制网络感兴趣区域的位置。

当研究从某个网络中某些大脑区域的活动来预测同情强度的能力时，发现躯体运动网络显着预测了测试者的同情反应程度（R = 0.374，p = 0.022）。

使用共振（R = 0.236，p = 0.037），对照（R = 0.22，p = 0.048）和单一手术网络（R = 0.242，p = 0.033）可以预测个人痛苦高于阈值。但是，在将数据调整为FDR（错误发现率）内的多个比较之后，这些结果仍然令人怀疑。


图片2：评估内网活动的共情预测。

移情的其他方面无法通过各种ROI的Intranet活动进行预测。如果对网络内活动的评估不能单独成功地预测对象的同情程度，则可能有必要考虑互连的选项。

不出所料，当考虑到谐振网络与控制网络之间的关系时，可以预测出同理心（R = 0.221，p = 0.0475，调整后的FDR数据）。这一事实完全证实了这项研究的基本理论，即人类大脑某些部位在静止状态下的共轭活性能够表现出共情的特征。因此，MRI可以成功地确定一个人的同理力，而无需其他调查表。


图片编号3：评估共情对相互联系的预测。

MRI研究表明，各个网络不断相互协作，表现出同理心的表现。这很重要，因为在扫描过程中未向测试对象分配任何特定任务。然而，他们的大脑处于静止状态，表现出明显的共情迹象，他们设法解决了。

要更详尽地了解这项研究的细微差别，建议您查看科学家的报告。

结语


通过这项研究，科学家能够成功地证明负责同情的神经连接的动态集成模型。这项研究进行的实验的主要区别特征是受试者的大脑处于静止状态。非常合乎逻辑的是，例如，小提琴家熟练掌握小提琴的能力在没有手时不会消失。但是，关于科学界的共情，围绕它的起源和形成机制存在许多争议。这项工作使我们有可能理解，即使在一个人的面前没有什么应该引起他的移情和同情的，他大脑的某些神经网络仍会继续形成移情。

除了关于移情的新数据外，科学家还获得了更多证据，证明不可能在活跃和平静的状态下划分大脑的工作，因为它们之间存在许多相关的过程，而与活动无关。

研究人员自己在工作中不仅看到理论上的，而且看到了实际的应用。例如，可以预测特定患者对特定干预的反应程度，从而允许创建最佳治疗选择。无论如何，我们对大脑的了解越多，我们对自己的了解就越好。
