バジェットポケットオシロスコープの選択

こんにちは！

仕事や趣味でエントリーレベルの家庭用コンパクトオシロスコープを選択するトピックに関する短い記事を追加します。

なぜそれがポケットとコンパクトになるのか-これらは最も予算の選択肢だからです。デスクトップオシロスコープは、よりかさばる機能的なデバイスであり、原則として、多くの機能を備えた4つのチャネルに対して非常に高価なモデル（200〜400ドル以上）です。
しかし、簡単な測定と波形の評価のための1チャネルのコンパクトモデルは、文字通り$ 20 ... $ 40で購入できます。



したがって、ハンドヘルドオシロスコープの主な技術的特性は、MHzで測定される動作帯域と、測定の品質に直接影響するサンプリング周波数です。

この記事では、私が個人的に持っていたオシロスコープについて説明し、これらのモデルの小さな利点と欠点を示します。

多くのハムが通過した最初のバージョンは、ATmegaマイクロコントローラーに基づいたオシロスコープであり、DSO138などの自己アセンブリを含むAliには多くのオプションがあります。 STM32マイクロコントローラーに基づいたその開発は、DSO150と呼ばれます。

DSO150オシロスコープ-これは、エントリーレベルのハムラジオに適したオシロスコープです。P6020プローブが含まれています。オシロスコープ自体の帯域幅は約200 kHzです。STM32に基づいて構築された、最大100万サンプルのADC。シンプルな電源（PWM）とオーディオパスをチェックするのに適したオプションです。初心者の方、たとえば音声信号の研究（アンプのチューニングなど）に適しています。マイナスのうち、波形のイメージを保存できないことと、帯域幅が狭いことに気付きました。



仕様：

  • リアルタイムサンプリングレート：1 MSa / s
  • アナログ帯域幅：0-200 kHz
  • 感度範囲：5-20 mV / div
  • 最大入力電圧：50 V max。（1xプローブ）
  • 掃引時間範囲：500 s / div– 10μs/ div

必要に応じて、より安価な非はんだ付けオプションを見つけることができます。 「意味のある」はんだ付けの訓練に適しています。

しかし、趣味はすぐに通過し、本格的なモデルに切り替えました。

2018年の初めに、最も人気のあるエントリーレベルのオシロスコープオプションの1つであるシンプルでありながら優れたオシロスコーププローブDSO188が登場しました。

DSO188は、1チャネル、メモリなしのシンプルな「ディスプレイ」ですが、カラーディスプレイ、300mAhバッテリーを搭載し、サイズは非常に小さいです。その利点はコンパクトさと携帯性にあり、周波数帯域はほとんどのアプリケーション（たとえば、オーディオ機器のチューニング）に十分です。

低コスト（30ドル）で、周波数1 MHz（5MSA /秒のサンプリング）の信号を表示します。作業にはMMCXプローブが使用されますが、キットにはMMCX-BNCアダプターが含まれています。別のADCが5MSPS、最大1 MHzの帯域、パネルで作られたプレハブケースにインストールされています。プロでは、DSO150（1MHz）と比較してコンパクトなサイズと適切な帯域、およびコンパクトな寸法に注目しています。従来のテスターと併用すると大変便利です。ポケットに簡単に収納できます。マイナス点は-ケースのデザインはオープンで、外部からの影響（変更が必要）から保護されておらず、保存された画像をコンピュータに転送する機能がないことです。MMCXコネクタの存在は便利ですが、完全な動作には、BNCまたは特別なプローブ用のアダプタが必要です。お金のために、これは非常に優れたエントリーレベルのオプションです。



仕様：

  • アナログ帯域幅：0-1 MHz

  • 最大入力電圧：40 V（1Xプローブ）、400 V（10Xプローブ）。組み込みの減衰器はありません。
  • 掃引時間範囲：100mS / div〜2uS / div
