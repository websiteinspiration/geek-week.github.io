マルコフ連鎖で歌詞を生成します

今日は、PythonとMarkovifyライブラリを使用して歌詞を生成する私の経験について話したいと思います


免責事項：著者は夕方に自分自身を楽しませたかったので、次のようなより良いものを思いつきませんでした：


チェーンの「トレーニング」のボディとして、キーズキーズのグループソングの歌詞を使用します。



上のpicchaは、マルコフ連鎖がどのように機能するかを示しています。しかし、良い記事。


開始


私は、読者がすでにPythonに精通しており、IDEおよびPythonバージョン> 3.5がプリインストールされていると想定しています。


必要なライブラリをインストールします。


pip install markovify

次に、自分用の歌詞を蓄積して少し処理します。これのために、単純なテキストプロセッサを作成します。コーラス、詩、数字などの「ごみ」の単語を削除します。このため、簡単なテキストプロセッサを作成します。


IND = ""
Find2 = ":"
INFILE = "corpus.txt"
OUTFILE = "output.txt"
ENC = "utf-8"
with open(INFILE, encoding="UTF-") as infile, open(OUTFILE, "w", encoding="UTF-8") as outfile:
    for line in infile:
        if FIND or Find2 not in line:
            outfile.write(line)

os.remove(INFILE)
os.rename(OUTFILE, INFILE)

すでに処理したテキスト


そして、この素晴らしいライブラリを使用して、最後にテキストを生成しましょう。


with open("C:/Users/alexd/PycharmProjects/untitled/corpus.txt",encoding='utf-8') as f:
    text = f.read()
text_model = markovify.Text(text)
for i in range(10):
    print(text_model.make_short_sentence(380))

     ,  ,    «».     :    ?

合計
