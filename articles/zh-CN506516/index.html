然后国王需要在随机点上评估多项式的​​值-也就是说，计算多项式[0] +多项式[1] * x +多项式[2] * x ** 2 ...

尽管存在用于评估多项式的​​第三方模块，但它们不适用于有限项按字段。因此，国王手动编写了评估代码：

def evaluate(coefficients, x):

最后，找到了解决方案。

可以使用Lagrange插值多项式执行秘密恢复。为此，我们估计多项式为（0），然后估计其他n个点（n为多项式的次数）。多项式公式可以以显式形式编写。在零处，多项式为1，在其他点处为0。由于多项式估计是线性函数，因此我们可以在相同点处估计并内插估计。

from functools import reduce
from operator import mul

def retrieve_original(secrets):
    x_s = [s[0] for s in secrets]
    acc = Mod(0, P)
    for i in range(len(secrets)):
        others = list(x_s)
        cur = others.pop(i)
        factor = Mod(1, P)
        for el in others:
            factor *= el * (el - cur).inverse()
        acc += factor * secrets[i][1]
    return acc
