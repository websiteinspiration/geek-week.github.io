とらえどころのない才能：ロシアは最高のIT専門家を失う



今日の優秀なITプロフェッショナルへの需要はかつてないほど高まっています。ビジネスの完全なデジタル化により、開発者は企業にとって最も貴重なリソースになりました。しかし、チームにふさわしい人材を見つけることは非常に難しく、有能な人材の不足が慢性的な問題になっています。

ITスタッフの不足


今日の市場の肖像画は次のとおりです。原則として、専門家はほとんどいない、彼らは実際には教えられていない、そして多くの人気のある分野で既製の専門家はいない。事実と数字を見てみましょう。

1.インターネットイニシアチブの開発財団が実施した調査によると、中等職業教育および高等教育では、年間6万人のIT専門家しか市場に参入していません。専門家によると、10年後、ロシア経済は技術の分野で欧米と競争するのに十分な約200万人の開発者を持たないかもしれません。

2.有資格者よりもすでに多くの空席があります。 HeadHunterによると、わずか2年間（2016年から2018年）で、ロシアの企業はITプロフェッショナル向けに30万件を超える求人を発表しています。同時に、広告の51％は1年から3年までの経験を持つ人に向けられ、36％-少なくとも4年の経験を持つ専門家に向けられ、初心者には9％だけが向けられています。

3. VTsIOMおよびAPKITが実施した調査によると、卒業生の13％のみが、自分の知識が実際のITプロジェクトで作業するのに十分であると考えています。大学や最先端の​​大学でさえ、教育プログラムを労働市場の要件に適応させる時間はありません。テクノロジー、ソリューション、製品の急速な変化に対応するのは困難です。

4. IDCによると、ITプロフェッショナルの3.5％のみが完全に最新です。多くのロシア企業は、ギャップを埋めて労働者に彼らのニーズに備えるために、独自のトレーニングセンターを開設しています。

たとえば、ParallelsにはMSTUに独自の研究所があります。バウマンとロシアの他の主要な技術大学との緊密な協力、およびティンコフ銀行はモスクワ州立大学の力学と数学の学部とフィンテック開発者のための無料の学校でコースを組織しました。

ロシアだけが有能な人員不足の問題に直面しただけではない。数は異なりますが、状況はアメリカ、イギリス、オーストラリア、カナダ、ドイツ、フランスでほぼ同じです...世界中で専門家の完全な不足があります。したがって、最善のための本当の闘争があります。そして、国籍、性別、年齢などのニュアンスは、雇用主が心配する最後の事柄です。

ロシアのIT専門家の海外移住


ロシアの開発者が国際的なプログラミングコンテストを支配していることは秘密ではありません。Google Code Jam、Microsoft Imagine Cup、CEPC、TopCoder-これは、私たちの専門家が最高の評価を受ける有名な選手権のほんの一部です。彼らは海外のロシアのプログラマーについて彼らが何と言っているか知っていますか？

-プログラミングが難しい場合は、アメリカ人に渡してください。非常に難しい場合-中国人に。それが不可能だと思ったら、ロシア人に渡してください！


グーグル、アップル、IBM、インテル、オラクル、アマゾン、マイクロソフト、フェイスブックなどの企業が私たちの開発者を魅了しているのは当然のことです。そして、これらの組織の採用担当者は実際に試す必要さえありません。ロシアのITスペシャリストの大多数自身がそのような雇用を夢見ており、最も重要なのは海外への移動です。どうして？これには少なくともいくつかの理由があります。

給料


はい、ロシアでは（特に開発者の間で）最低の給与ではありません。彼らはアジアやアフリカのいくつかの国よりも高いです。しかし、米国とEUでは、状況はより魅力的です...約3〜5倍。そして、お金が主なものではないと彼らがいかに言っても、現代社会における成功の尺度であるのは彼らである。彼らのために幸せを買うことはできませんが、まったく新しい機会と一定の自由があります。彼らはこれのために行っています。

州は給与の面で第1位になります。アマゾンのソフトウェア開発者は、年間平均121,931ドルを稼いでいます。より明確にするために、これは毎月約630,000ルーブルです。マイクロソフトとFacebookはさらに支払います-毎年、それぞれ140,000ドルと135,000ドル。ヨーロッパは物事の見通しにあまり動機がありません。たとえば、ドイツの年間給与は65,000ドル、スイスは53,000ドルですが、いずれにしても、ロシアの給与はまだヨーロッパの給与に達していません。

社会経済的要因


ロシアの通貨の弱さと不安定な経済状況、さらに海外で何が良いかについての理想主義的な考えも、才能のある開発者が故国を去ることを奨励しています。結局のところ、海外の抽象にはより多くの機会があり、気候はより良く、医学はより良く、食物はより美しく、そして一般的に生活はより簡単で快適です。

一般に、ITスペシャリストは、学習中に移動について考え始めます。国の主要な教育機関の廊下では、明るく魅力的なバナー「Work in the USA」が誇示され、採用担当者のオフィスが直接学部に配置されています。統計によると、6人のプログラマのうち4人が卒業後3年以内に仕事に出ます。この「頭脳流出」は、経済を維持するために必要な熟練労働者の国を奪います。

方法はありますか？


まず第一に、若者の政策は海外の人員の流出の削減に影響を与えるべきです。国家の経済発展にとって最も重要な条件は、次世代のコンピューターエンジニアのトレーニングだけでなく、資格のある人材を自宅に引き付けて維持する方法の模索でもあることを認識することは、国の利益になります。国の競争力はこれにかかっています。

豊かな人的資本を考えると、ロシアは世界の技術の中心地の1つであるべきです。しかし、これまでのところ、この可能性は実現されていません。現代の現実は、国家が「頭脳流出」にゆっくりと反応しているようなものです。このため、ロシアの企業は同じ才能を求めて世界中で競争しなければなりません。

貴重な開発者を維持するには？彼のトレーニングに投資することが重要です。 ITでは、スキルと知識を常に更新する必要があります。会社を犠牲にしてさらにトレーニングを行うことは、多くの人々が雇用主に望んで期待していることです。多くの場合、別の国に移動したいという欲求は、ロシアでは専門的に開発することも、新しい技術を学ぶこともできないという信念と関連しています。逆を証明する。

原則として、自己啓発の問題はさまざまな方法で解決できます。これらは有料コースや高価な国際会議である必要はありません。新しいテクノロジーやプログラミング言語を習得できる実用的なタスクを提供することをお勧めします。誰もが話していることが望ましい。開発者は課題を好みます。それらがなければ、彼らは退屈します。また、トレーニングを会社のプロジェクトに直接リンクすることは、従業員とビジネスの双方にとってメリットのあるオプションです。

***
才能のある開発者は、簡単で平凡な仕事を望んでいません。彼らは、問題の解決、元の解決策の発見、通常のモデルを超えることに関心があります。アメリカの巨大企業では、ITエキスパートが最初のポジションにいるわけではなく、複雑なことを委任されることはめったにありません。ロシアの組織の快適な環境でスキルを開発できるようにする興味深いタスクは、米国とヨーロッパの高給の魅力に対する優れた反作用です。
