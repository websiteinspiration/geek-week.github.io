私が書いたゲームのいくつかはここで見ることができます。

ESP8266用のVMと同時に、ブラウザー用のJavaScriptエミュレーターを作成しました。バイトコードを手動で編集しないようにするために、MOS6502のアセンブラでの私の経験に基づいて、簡単なアセンブラが追加されました。次に、より高いレベルの言語を追加することにしました。それにもかかわらず、私の主な仕事は簡単なゲームをすばやく書くことであり、アセンブリコードを長期間デバッグすることではありませんでした。 LLVMを追加するよりコンパイラを書く方が簡単だと私には思えました。そして私はそれを他の言語ほど悪くないことを知っているので、私はそれをJavaScriptで書いた。現在のところ、彼はC標準のサポートにはほど遠く、コンパイル時に、理解できない場所で理解できないエラーが発生する可能性があります。しかし、2000行もかからないため、彼は高速です。

そして、なぜ私がこれすべてをここに書いたのかという問題についてです。すでにゲームを書いて実行することができます。しかし、完成はまだ遠いです。誰かがESP LGEの完成を手助けしたり、自分のゲームを書いたりしたい場合は、私はとても幸せです。私のアイデアがおもしろいと思っていて、もっと知りたいのであれば、私はあなたの質問に答えさせていただきます。私はあなたに警告します、Arduinoのコードは読むのが非常に難しいです。自分で独学していることもあります。仕事の速度を上げるために関数呼び出しの数を減らしようとしたことも一因です。その結果、多くの関数にはコードの巨大な足跡が含まれています。妊娠中の女性や子供を画面に入れないでください。徐々に、私はこれを修正して読みやすさを改善するように努めます。

そして、ゲームの例を読んだ人のために。コンパイルされた形式では、100行未満で1KB未満です。

int stickCount;
char key,previouseKey,takenSticks;

void redraw(){
	int i;
	//  
	setcolor(2);
	//  
	for(i = 0; i < stickCount; i++)
		line(22 + i * 6, 74, 22 + i * 6, 84);
	//  
	setcolor(11);
	// 
	for(i = stickCount; i < 15; i++)
		line(22 + i * 6, 74, 22 + i * 6, 84);
	//    
	setcolor(1);
	//  
	delayredraw();
}

void playersMove(){
	//    ,     . 
	while(key == previouseKey){
		key = getkey();
	}
	while(key != KEY_LEFT && key != KEY_DOWN && key != KEY_RIGHT){
		key = getkey();
	}
	if(key & KEY_LEFT){
		takenSticks = 1;
	}else if(key & KEY_DOWN){
		takenSticks = 2;
	}else{
		takenSticks = 3;
	}
	printf("%d, ", takenSticks);
	stickCount -= takenSticks;
	previouseKey = key;
}

void computersMove(){
	if(stickCount % 4){
		//   ,   
		takenSticks = stickCount % 4;
	}else{
		//     
		takenSticks = 1 + random(1);
	}
	stickCount -= takenSticks;
	printf("%d, ", takenSticks);
}

void game(){
	//
	stickCount = 15;
	clearscreen();
	//      
	gotoxy(8,0);
	puts("");
	gotoxy(2,1);
	puts(" 1,2  3 .  ,       . :\n");
	// 27,25  26  
	printf(" %c 1    %c 2    %c 3", 27, 25, 26);
	gotoxy(0,12);
	redraw();
	while(1){
		playersMove();
		if(stickCount <= 0){
			gotoxy(3,8);
			puts(" ");
			return;
		}
		redraw();
		computersMove();
		redraw();
		if(stickCount <= 0){
			gotoxy(3,8);
			puts(" ");
			return;
		}
	}
}

void main(){
	while(1){
		game();
		// 
		settimer(1,1000);
		while(gettimer(1)){}
		while(getkey() == 0){}
		previouseKey = key;
	}
}
