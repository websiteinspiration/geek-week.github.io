Arduinoの自律デバイスは、温度の上昇（低下）を通知します

前書き


このデバイスは、離れた場所の温度を測定するように設計されています。必要なこのデバイスを組み立てるための既製の指示が見つからなかったため、2千ドルで購入したくなかったため、arduinoですべてを行うことにしました（高価ではなく、それほど難しくありません）。

組立説明書


必要なプロジェクトを構築するには：

  • DS18B20温度センサー1メートル。（封印）
  • 画面0.96 128x64 OLED
  • ブレッドボード
  • パパパパワイヤーセット
  • マイクロコントローラーArduino Uno
  • 抵抗器

プロジェクトを実装するには、arduino IDEといくつかのライブラリを一度にインストールする必要があります。

  • OneWire Arduinoライブラリ
  • ロシア語およびウクライナ語をサポートするOLED I2C_OLED_I2C。
  • ダラスの温度

ブレッドボードについて少し話しましょう。このブレッドボードは、はんだ付けや放射性元素、接続用のワイヤを使わずに回路をすばやく組み立てるために使用されます。開発ボードにはさまざまなサイズがありますが、ほとんどの場合、これらは同じブロックで構成されています。



ソケットの電気接続は右の図に示されています。各列の5つの穴（この場合は30）は電気的に相互接続されています。左側と右側に2本の電力線があります。ここでは、柱のすべての穴が相互に接続されています。真ん中のスロットは、ケース内のマイクロ回路の取り付けと取り出しに便利なように設計されています。回路を組み立てるには、無線コンポーネントとジャンパーを穴に挿入します。

ブレッドボードを見つけたら、モジュールの接続に移ります。はじめに、ディスプレイを次のように接続します。





この場合、5Vをブレッドボード（黄色のワイヤー）の「+」に接続し、GNDを「-」（白いワイヤー）に接続したため、後ですべての電源を「+」と「 -「ブレッドボード上。

次に、下の写真に示すように、抵抗を使用してDS18B20温度センサーを接続します。





この場合、便宜上、すべてのセンサーワイヤーをブレッドボードに接続し、「Papa-papa」オレンジワイヤーをピン10に接続しました。

作業の半分は完了しました。信号デバイス、つまりx2 LED（高温は赤、低温は青）と従来のピエゾエミッター（一般的な人々では「ツイーター」）を接続するだけです。LEDは非常に簡単に接続されます。「+」は長い棒、マイナスは短い棒です。それをボードに挿入し、抵抗を必ず接続し、「ダッドダッド」ワイヤを使用して青色LEDの短いロッドを5ピンに、赤色LEDの短いロッドを12ピンに接続します。

きしむ人は黒いワイヤーを「-」に、赤いワイヤーを13ピンに接続するだけです。

これはそれがうまくいくはずです：



最も重要なこと（スケッチ）


#include <Wire.h>
#include <Wire.h>
#include <OneWire.h>
#include <DallasTemperature.h> //   DS18B20
#define ONE_WIRE_BUS 10 // pin    DS18B20
OneWire oneWire(ONE_WIRE_BUS);
DallasTemperature sensors(&oneWire);
#include <OLED_I2C.h>
OLED  myOLED(8, 9, 8); //  , 8 pin-SDA, 9pin-SCL
extern uint8_t RusFont[]; //  
extern uint8_t BigNumbers[]; //  
extern uint8_t SmallFont[]; //  
void setup()
{
  pinMode(13, OUTPUT); //  pin 13  
  pinMode(12, OUTPUT);//  pin 12  
  pinMode(5, OUTPUT); //  pin 5  
  sensors.begin();//  
  myOLED.begin();//  
  myOLED.setFont(RusFont);
}
void loop()
{
  if (sensors.getTempCByIndex(0)>70){ //  (  >70     )
  digitalWrite(5,255);//  
  tone(13, 2500); //   2500 
  delay(400);//  0.4 
  digitalWrite(5,LOW);//  
  }
  else{
    digitalWrite(5,LOW);
  }
  if (sensors.getTempCByIndex(0)<25){  //  (  <45     )
    digitalWrite(12,255); //  
    tone(13, 2500); //   2500 
    delay(400); //  0.4 
    digitalWrite(12,LOW); //  
  }
  else{
    digitalWrite(12,LOW);
  }
  if ((sensors.getTempCByIndex(0)<70) and (sensors.getTempCByIndex(0)>45)){
  noTone(13); //  
}
sensors.requestTemperatures();
myOLED.clrScr(); //  
myOLED.setFont(RusFont);
myOLED.print("Ntvgthfnehf", CENTER, 0); //    
myOLED.setFont(SmallFont);
myOLED.print("vk.com/tupo_izi123", CENTER, 55); //   
myOLED.setFont(BigNumbers);
myOLED.print(String(sensors.getTempCByIndex(0) , 1), CENTER, 10);   //   ,    
myOLED.update();
delay(100);
}

このスケッチでは、温度を70度以上45度未満に設定しましたが、他の温度（-55から+125°C）を設定することもできます。
