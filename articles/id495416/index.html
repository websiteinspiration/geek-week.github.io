pipeplot - visualisasi interaktif aliran data dari pipa di konsol

Bagi mereka yang tidak ingin meninggalkan konsol, suka ascii-grafisnya, sebuah utilitas telah muncul untuk menggambar grafik dari pipa. Di bawah animasi dan contoh kucing.

Ping




ping ya.ru | grep --line-buffered time | sed -u -e 's#.*time=\([^ ]*\).*#\1#' | pipeplot --min 0

Dapat melacak coronavirus melalui covidAPI


while true; \
    do curl -s https://coronavirus-19-api.herokuapp.com/all \
    | jq '.deaths'; \
    sleep 60; \
done \
| pipeplot --color 1 --direction left

perlu menginstal jq ke parse json

Lima ratus melalui grafit


while true; \
do \
    curl -s 'http://graphite/render?target=my_app_rps_error&format=json&from=-5min&until=now' \
    | jq -c '.[0].datapoints[-1]'; \
    sleep 5; \
done \
| sed -u s/null/0/ \
| stdbuf -oL uniq \
| stdbuf -oL jq '.[0]' \
| pipeplot

Buffering


Banyak utilitas konsol buffer stdout. Karena itu, pipa serial pecah. Beberapa utilitas mendukung buffering: sed -u , grep --line-buffered . Jika tidak, Anda dapat menggunakan metode universal stdbuf -oL dan unbuffer .

Pilihan yang tersedia


WARNA --warna - warna kolom, jumlah palet menggunakan Xterm - simbol
SIMBOL - simbol untuk menampilkan kolom mendukung segel -
skala semua {,} jendela - pengaturan penskalaan otomatis minimum dan maksimum, semua - memperhitungkan semua data, jendela - hanya yang terlihat
--direction {kiri, kanan} - arah
--min MIN --max MAX - memperbaiki minimum dan maksimum, dikombinasikan dengan penskalaan otomatis

Instalasi


pip install pipeplot
