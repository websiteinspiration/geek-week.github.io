¿Cómo resolver? "ECONNREFUSED - Conexión rechazada por el servidor"

¿También ha encontrado el error ECONNREFUSED - conexión rechazada por el servidor en FileZilla ? Entonces es genial que hayas encontrado esta guía. Le mostraré tres métodos para solucionar este error de FTP.


El primer método. Cambiar el valor predeterminado de un puerto FileZilla


La causa del error puede ser el puerto incorrecto cuando se conecta a través de FileZilla. En esta situación, solo necesita cambiar el puerto FTP predeterminado al número de puerto SFTP predeterminado . Simplemente cambie 21 a 22 en el cuadro de entrada " Puerto ".




Segundo método Desactivar antivirus / cortafuegos


A veces, este error puede ocurrir cuando el software antivirus y / o el firewall rechazan el intento de FileZilla de establecer una conexión.


En caso de que un antivirus o firewall llame a ECONNREFUSED, solo necesita deshabilitar este software y luego volver a conectarlo. Primero te mostraré cómo hacerlo en macOS:
