PHP


プロジェクトを作成し、ターミナルを開いて、コマンドを実行します。

php composer.phar require google/apiclient:^2.0

composerとそのインストール方法がわからない場合は、よく知っておいてください。
インストール後に、index.phpとoauth2callback.phpの2つのファイルを作成します。ところで、
PHPサーバーはすでに起動できます。

php -S localhost:8080

また、ファイルsecret_json.json（前にダウンロードしたもの）をプロジェクトフォルダーに移動します。

Index.phpで直接、ユーザーに認証を送信します。Index.phpファイルの内容

<?php
require_once __DIR__ . '/vendor/autoload.php';

$client = new Google_Client();
//     
$client->setAuthConfig('client_secret.json');
//   
$client->setScopes([
    'https://www.googleapis.com/auth/youtube.readonly',
]);
//    refresh_token
$client->setAccessType("offline");

if (isset($_SESSION['access_token']) && $_SESSION['access_token']) {
    $client->setAccessToken($_SESSION['access_token']);
} else {
    $redirect_uri = 'http://' . $_SERVER['HTTP_HOST'] . '/oauth2callback.php';
    header('Location: ' . filter_var($redirect_uri, FILTER_SANITIZE_URL));
    exit();
}

$refreshToken = $_SESSION['access_token']['refresh_token'];
$accessToken = $_SESSION['access_token']['access_token'];
$tknType = $_SESSION['access_token']['token_type'];
$clientId = $client->getClientId();
$created = $_SESSION['access_token']['created'];
unset($_SESSION['access_token']);
unset($_SESSION['user']);
session_destroy();

$data = json_encode([
  'refreshToken' => $refreshToken,
    'accessToken' => $accessToken,
    'tknType' => $tknType,
    'clientId' => $clientId,
    'created' => $created
]);
echo $data;

実際、このコードはユーザーに必要なすべてのデータを提供し、それらをデータベース（またはメモ帳）に書き込むことができます。

ファイルoauth2callback.php：

<?php
require_once __DIR__.'/vendor/autoload.php';

session_start();

$client = new Google_Client();
$client->setAuthConfigFile('client_secret.json');
$client->setRedirectUri('http://' . $_SERVER['HTTP_HOST'] . '/oauth2callback.php');
$client->setScopes([
    'https://www.googleapis.com/auth/youtube.readonly',
]);
$client->setApprovalPrompt('force');
$client->setAccessType("offline");

if (!isset($_GET['code'])) {
    $auth_url = $client->createAuthUrl();
    header('Location: ' . filter_var($auth_url, FILTER_SANITIZE_URL));
} else {
    $client->authenticate($_GET['code']);
    $_SESSION['access_token'] = $client->getAccessToken();
    $redirect_uri = 'http://' . $_SERVER['HTTP_HOST'] . '/';
    header('Location: ' . filter_var($redirect_uri, FILTER_SANITIZE_URL));
}

このファイルは、ユーザーをGoogle認証ページにリダイレクトし、トークンデータをセッションに書き込みます。その後、index.phpが引き続き機能する場所にユーザーを戻します。
もちろん、すべてのデータをoauth2callbackに直接書き込み、ユーザーをインデックスにリダイレクトしないでください。または、都合のよいアドレスで次のポイントを指定できます。また、このコードでは承認エラーは考慮されていません。これを覚えておいてください。

これは許可を得て行われます。すでにaccess_tokenとrefresh_tokenがあります。YouTubeを取得しましょう。

refreshToken.php：

<?php
require_once __DIR__ . '/vendor/autoload.php';

/**
 * @param string $refreshToken
 * @param $db
 * @return array
 * @throws Google_Exception
 */
function refreshToken($refreshToken, $db)
{
    $client = new Google_Client();
    $client->setAuthConfig('client_secret.json');
    $client->setScopes([
        'https://www.googleapis.com/auth/youtube.readonly',
    ]);
    $client->setAccessType("offline");

    $client->refreshToken($refreshToken);
    $accessToken = $client->getAccessToken();

    $refreshToken = $accessToken['refresh_token'];
    $accessToken = $accessToken['access_token'];
    $clientId = $client->getClientId();
    $data = [
        'refreshToken' => $refreshToken,
        'accessToken' => $accessToken,
        'tknType' => 'Bearer',
        'clientId' => $clientId,
        'created' => time()
    ];

    //      , .

    return $data;
}

Googleが発行したトークンは1時間のみ有効です。日中はトークンの更新を25回しかリクエストできません。この関数はトークンを更新し、作業を続行できる新しい値を返します。 GetStreamsはその動作を示しています。
