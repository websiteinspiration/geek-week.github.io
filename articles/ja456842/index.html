PS2 / PSPエミュレーター+ゲームプレイストリーミング（YouTube、Facebook、Twitch）=新しいバージョンのOmega Red

読者のみなさん、こんにちは！

PlayStation 2とPlayStation Portable-Omega Redのソフトウェアエミュレーターのテーマを続けます。詳細に：

  • PS2 / PSPエミュレーター+ Googleドライブ+ YouTube =「狂気」は続く
  • PlayStation 2とPlayStation Portable（スポイラー-1つのエミュレーターに統合）を友達にした方法
  • Red Omega PS2エミュレータ

この出版物は、プロジェクトに追加した新機能（エミュレーターから有名なストリーミングインターネットサービスへのライブビデオおよびオーディオゲームプロセスのストリーミング放送）に関連して書いたものです。
キャットへようこそ。


まず、次の点に注意します。インターネットブロードキャストのサポートを統合するというアイデアは、他のプロジェクトであるCaptureManager SDKから生まれました。私はウェブカメラからの放送を実験し、実行可能なコードを得ました。そして、「アイデア」が登場しました-このコードをゲームコンソールエミュレータに埋め込むためです！

このタスクは、ビデオとオーディオのキャプチャという面で興味深いものです。カメラからのキャプチャは非常に簡単ですが、エミュレーターのレンダラーからのビデオのキャプチャはどうすればよいですか？デスクトップ全体をキャプチャするコードを見つけることができますが、とても簡単です。また、グラフィックレンダラから直接グラフィックをキャプチャするとどうなるでしょうか。オーディオキャプチャに関する同様のアイデア-通常、オーディオキャプチャは、スピーカーへのオーディオストリームの複製としてシステムミキサーから実行されます。システムミキサーの前にオーディオレンダラーから直接オーディオをキャプチャするとどうなるでしょうか。

面白い！元の！

実装方法：DirectX 11レンダラーとDirectSoundレンダラーにキャプチャコードが追加されます。グラフィックスのレンダリング結果は、DirectX 11ビデオテクスチャを介してH264ビデオエンコーダーに直接送信されます。これにより、グラフィックスアクセラレータでハードウェアアクセラレーションビデオエンコーディングを使用できます。オーディオストリームは、Windowsに組み込まれたAACオーディオエンコーダーにルーティングされます。

キャプチャコントロールは2つの部分で構成されています。

  • ライブビデオをブロードキャストするためのストリーミングインターネットサービスのエンコーディングと選択を設定します。
  • ミキサーのセットアップ-ライブビデオと追加のソース（ウェブカメラ、JPG / PNG / GIF（アニメーション）ファイル）をミックスします。

エンコーディング設定：


ストリーミングのセットアップは、キャプチャモードの選択「ストリーミング」から始まります。このモードでは、利用可能なインターネット接続速度に応じて、ビデオおよびオーディオストリームのデータレートを調整できます。これとは別に、ストリーミングサービスを選択するか、ストリーミングサーバーの完全なアドレスを取得できます。たとえば、MonaServerに基づく自分のサーバーなどです。開いている接続だけでなく、安全な接続（SSL）もサポートされていることに注意してください。ストリーミングサービスFacebook（はい、1つあります）が次の発表を公開しました。

「2019年11月1日から、RTMPサーバーURLのサポートを終了します。その後、保護されていないURLをビデオブロードキャストに使用することはできません。ビデオコーディングデバイスの設定を更新し、安全なRTMPS URLを使用する」-RTMPSサポートは有望であると判断し、「OpenSSLをRTMPライブラリにねじ込む」時間を費やしました。そしてコードは機能します!!! Facebookは安全な接続を介してストリームを取得します！

ストリームは、記録/ストリーミングボタンを押すことで開始します。


録音/ストリーミングの開始後にミキサー設定が利用可能になります：



メディアパネルの[メディアソース]タブが利用可能になります-このタブには、ミキシングに利用できるビデオおよびオーディオストリームソースが含まれます。Webカメラとマイクに加えて、JPG / PNG / GIF（アニメーション）ファイルを追加できます。

マイクの場合、ミキシングオーディオストリームの比率を制御できます。チェックボタンをクリックしてマイクを選択し、スライダーでミキシング制御を行います。


ビデオソースの場合、ビデオストリームのミキシングを制御できます。チェックボタンをクリックしてカメラを選択し、ドロップダウンメニューからカメラの画像サイズを選択します。



ミックスの透明度（または非透明度）は、別のスライダーで制御できます。


ミキシングビデオストリームの位置とサイズは、ストリーミング中にいつでも変更できます。
