Golangの映画館サイトの解析

みなさん、こんにちは。この記事では、今日人気の高いプログラミング言語Goのサイトからデータを収集するために書かれた小さなサンプルコードを見ていきます。

この記事は、囲碁について多くのことを聞いたが、まだ自分で試したことがない人には特に興味深いでしょう。

社内プロジェクトの1つでは、レンタルされている映画と上映のスケジュールに関するデータを収集する必要がありました。ここで、すべてが始まった最初の（最も単純な）パーサーのバージョンを検討します。

記事コードを読むのが面倒な人のために、私はすぐにリポジトリへのリンクを提供します。

最初のバージョンでは、パーサーは次の情報のみを収集できました。

  • 1つの映画館でのセッションについて、
  • 詳細な説明、
  • 題名、
  • スタジオ、

そしてそれをJSONで保存します。最初に、解析する適切なライブラリを選択する必要があります。

Googleは「golang web scraping」というクエリに対して多数のオプションを提供します。それらの多くはこのリストに反映されています。慣れることをお勧めします。JSレンダリングをサポートしているため、geziyorを選択しました（ちなみに、これでは使用しません）ただし、この機能はサイトを解析するときに非常に役立ちます）、使用方法は非常に簡単です。

したがって、ライブラリが選択されました。次のステップは、ライブラリをインストールして、コードで使用を開始することです。
ライブラリのインストールは非常に簡単です。

go get -u github.com/geziyor/geziyor

それでは、コードの作成に移りましょう。

main関数の本体内でパーサーを呼び出し、データの収集を開始するページのURLを渡し、結果をJSONファイルにエクスポートすることを示します。


func main() { 
    geziyor.NewGeziyor(&geziyor.Options{ 
        StartURLs: []string{"https://kinoteatr.ru/raspisanie-kinoteatrov/city/#"}, 
        ParseFunc: parseMovies, 
        Exporters: []export.Exporter{&export.JSON{}}, 
    }).Start() 
}

開始されましたが、データを収集するための十分なロジックがありません。これには、parseMovies関数を実装する必要があります。

収集ロジックは次のとおりです。

  • 映画情報を含むブロックを検索し、
  • このブロック内のすべてのセッションに関する情報の収集、
  • 映画と映画スタジオの名前のコレクション、
  • 映画の詳細情報が記載されたページへのリンクを収集し、
  • このページの説明のコレクション

この関数の実装に移りましょう。


ここでは、フィルムに関する情報を含むすべてのブロックが、さらに処理するために選択されます。

このようにして、セッションに関する情報が収集され、個人にとって便利な形式に変換されます（新しい行の余分なスペースとインデントを削除します）。


var sessions = strings.Split(s.Find(".shedule_session_time").Text(), " \n ") 
sessions = sessions[:len(sessions)-1] 

for i := 0; i < len(sessions); i++ { 
    sessions[i] = strings.Trim(sessions[i], "\n ") 
}

このコードブロックは、映画に関する詳細情報を含むページを取得し、その説明を取得する役割を果たします。


if href, ok := s.Find("a.gtm-ec-list-item-movie").Attr("href"); ok {
    g.Get(r.JoinURL(href), func(_g *geziyor.Geziyor, _r *client.Response) {
        description = _r.HTMLDoc.Find("span.announce p.movie_card_description_inform").Text() 
        description = strings.ReplaceAll(description, "\t", "") 
        description = strings.ReplaceAll(description, "\n", "") 
        description = strings.TrimSpace(description)

これによりAPIが呼び出され、結果がJSONファイルにエクスポートされます。
