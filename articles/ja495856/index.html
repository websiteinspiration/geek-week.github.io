Intel DevCloud for oneAPI-自宅に座る開発者向けのクラウドサービス


流行の間、多くの（すべてではないにしても）インターネットプラットフォームおよびサービスが家庭で特別なオファーを出します。私たちは、自己分離の状態であっても、勇気を持ってまたは穏やかに彼らと分離を共有する人々（シール）の写真を投稿する準備ができていない開発者向けのハードコアオプションを提供しますが、逆に、彼らの専門的スキルを駆使しようとします。Intel API for oneAPIはスタイリッシュでファッショナブルであり、7月1日まで無料です。

名前が示すように、oneAPI用のIntel DevCloudは、クラウドプラットフォーム経由でアクセスできる開発者向けのインテルのサーバー容量です。サーバーはIntel Xeon Scalableプロセッサーをベースにしており、Intel Arria 10 FPGAを搭載しており、ユーザーが利用できるプロセッサーノードとアクセラレーターの数は、検疫期間によって増加しています。専用の割り当て-220 GBのディスクストレージと192 GBのRAM。

ハードウェアに加えて、このプラットフォームはプリインストールされたソフトウェアコンポーネントも提供します。
Intel oneAPI Toolkitsは、新しい言語Data Parallel C ++用のコンパイラー、APIプログラミング用の一連のライブラリー、およびアプリケーションの分析とデバッグ用の一連のツールを含む一連の開発ツールです。より詳細には、コンポーネントのセットは次のとおりです。

  • コンパイラ

    • Intel OneAPI DPC ++コンパイラ
    • Intel C ++およびIntel Fortranコンパイラ
  • ツールキット

    • Intel oneAPI Base Toolkit
    • Intel oneAPI HPCツールキット
    • Intel oneAPI DLフレームワーク開発者ツールキット
    • インテルAIアナリティクスツールキット
    • OpenVINOツールキットのIntelディストリビューション
  • ユーティリティ、フレームワーク、ライブラリ
    • Intel OneAPI DPC ++ライブラリ
    • Intel OneAPI Math Kernel Library
    • Intel OneAPIディープニューラルネットワークライブラリ
    • Intel OneAPIデータ分析ライブラリ
    • Python用Intelディストリビューション
    • インテルVTuneプロファイラー
    • oneAPI Base Toolkit用のIntel FPGAアドオン

クラウドのすべてのソフトウェアコンポーネントが最新の最新バージョンに更新されます。対話型開発環境JupyterLabをインストールし、SSH経由でもアクセスできる。DevCloudサービスと直接onneAPIに関するオンラインテクニカルサポートとフォーラムがあります。

すべて準備完了-仕事に取り掛かります。
