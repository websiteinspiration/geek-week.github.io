Dicas para escolher um reforço de sinal de celular 2G / 3G / 4G / 5G

Um artigo sobre opções para amplificar um sinal de celular em vários locais: residência de verão, escritório, armazém ..., bem como uma breve visão geral das ofertas populares no mercado.


Existem duas opções principais para aprimorar as comunicações celulares:


  1. Usando uma antena, modem e roteador.
  2. Usando um amplificador ativo (repetidor).

A primeira opção para fortalecer a Internet 3G / 4G / 5G não aprimora a comunicação de voz no formato usual para nós (mas você pode usar chamadas por meio de mensagens instantâneas: WhatsApp, Viber, Skype, Telegram).


A segunda opção é universal - fortalece a Internet e a comunicação por voz, mas possui várias restrições (sobre as restrições abaixo).



Amplificação da Internet 3G / 4G usando antena, modem e roteador Wi-Fi


O sistema consiste em dois nós: uma antena na rua e um modem + roteador Wi-Fi dentro de casa.



Antena externa para modem / roteador 3G / 4G



Modem 4G conectado por adaptadores a uma antena de rua e inserido em um roteador Wi-Fi


Como funciona? A antena é direcionada para a estação base proposta pelo operador celular. Em seguida, o sinal vai para um modem 4G com um roteador que distribui a Internet distribuindo um sinal Wi-Fi. A opção é popular entre mestres particulares para instalar uma Internet suburbana. Barato e funciona!
