クリスマスツリーのトランススター



幼稚園の新年の前夜に、クリスマスツリーのおもちゃを作るように頼まれました。私は工芸品の製造の技術部分を手伝って引き受け、設計部分は子供に任せることにしました。なぜなら 簡単な方法を探しているのではなく、最も技術的なおもちゃを作ることにしました。それは、単なる火の玉のようなものではありません。方法はわかりませんが、頭の中ですぐに、ステッピングモーターとアドレス指定可能なLEDを使用して変身星を作ることを考えました。

使用される材料：

  1. エンド照明用プレキシガラス
  2. 28BYJ-48ステッピングモーター
  3. チップドライバーULN2003
  4. マイクロコントローラATTINY85
  5. LEDストリップWS2812

エンドライティングのためにプレキシガラスのエッジを照らすと、エッジだけでなく平面自体も光ります。そのようなプレキシガラスの内部構造の不均一性のために、内部の光は繰り返し反射され、したがって平面が強調表示されます。A4シートに1枚プリントし、星の部分を2つ切り取りました。





私がステッパーモーターで試したパーツの1つで、回転の中心を見つけ、丸い穴を切りました。そこに5つのLEDが付いたテープWS2812を接着しました。2番目の部分をモーターシャフトに取り付け、この全体をArduinoに接続しました。



ステッピングモーターを制御するための標準のステッパーライブラリにはバグがあり、反時計回りに回転できないため、TinyStepper_28BYJ_48ライブラリを使用しました。LEDのアニメーションには、ライブラリAdafruit_NeoPixelを使用しました。

変身星のスケッチ
// Documentation for this library can be found at:
// github.com/Stan-Reifel/TinyStepper_28BYJ_48
//
//
// This library requires that the stepper motor be connected to the Arduino
// using ULN2003 driver board.

// ***********************************************************************

#include <TinyStepper_28BYJ_48.h>
#include <Adafruit_NeoPixel.h>

// NEO Pixel configuration
#define PIN 0
#define LEDS_COUNT 5
#define RED strip.Color(255, 0, 0)
#define GREEN strip.Color(0, 255, 0)
#define BLUE strip.Color(0, 0, 255)
#define WHITE strip.Color(255, 255, 255)

// pin assignments, any digital pins can be used
#define MOTOR_IN1_PIN 1
#define MOTOR_IN2_PIN 2
#define MOTOR_IN3_PIN 3
#define MOTOR_IN4_PIN 4

// create the stepper motor object
TinyStepper_28BYJ_48 stepper;
Adafruit_NeoPixel strip = Adafruit_NeoPixel(LEDS_COUNT, PIN, NEO_GRB + NEO_KHZ800);

void setup() {
// connect and configure the stepper motor to its IO pins
stepper.connectToPins(MOTOR_IN1_PIN, MOTOR_IN2_PIN, MOTOR_IN3_PIN, MOTOR_IN4_PIN);
// set the speed and acceleration rates for the stepper motor
stepper.setSpeedInStepsPerSecond(256);
stepper.setAccelerationInStepsPerSecondPerSecond(512);

strip.begin();
strip.show(); // Initialize all pixels to 'off'
}

void loop() {
// Rotate the motor in the forward direction one revolution
stepper.moveRelativeInSteps(410);
for (byte i = 0; i < 20; i++){
fadeInOut(0xff, 0x00, 0x00); // red
}

// rotate backward 1 rotation, then wait 1 second
stepper.moveRelativeInSteps(-410);
for (byte i = 0; i < 20; i++){
fadeInOut(0x00, 0x00, 0xff); // blue
}
}

void fadeInOut(byte red, byte green, byte blue){
float r, g, b;

for(int k = 0; k < 256; k=k+1) {
r = (k/256.0)*red;
g = (k/256.0)*green;
b = (k/256.0)*blue;
setAll(r,g,b);
strip.show();
delay(5);
}

for(int k = 255; k >= 0; k=k-2) {
r = (k/256.0)*red;
g = (k/256.0)*green;
b = (k/256.0)*blue;
setAll(r,g,b);
strip.show();
delay(5);
}
}

void setAll(byte red, byte green, byte blue) {
for(int i = 0; i < LEDS_COUNT; i++ ) {
strip.setPixelColor(i, strip.Color(red, green, blue));
}
strip.show();
}


生きてる！！！

すべてをコンパクトに組み立て、Arduinoの代わりにATTINY85を使用する必要がありました。導かれますが、create.arduino.ccのウェブサイトからの指示 Arduinoのスケッチを簡単にかつ簡単にSEW ATTINY85ことができます。8脚のうち、ステッピングモーターに7、4、LEDに1、電源に2を使用し、RESETは空いたままにしました。



クリスマスツリーの王冠に固定するために、プランジャーからのハンドルは完全にフィットしました。



装飾的な仕事の後、星はかなりきれいに見え始めました。ハンドルは赤いサテンのリボンで包まれ、プレキシガラスはキラキラの透明なニスで覆われており、これによりグロー効果がはるかに強くなりました。



すべてが来たる2020年、新年の気分と創意工夫で！
