一つの層には、いくつかの半透明の層の削減

同様の実験は、私は不透明層（に相当し、不透明な背景画像のパラメータ計算の仕方を考えて作られたc0）と（それに重畳された半透明の層c1を持つ透明度に設定されているがa）。この場合、単一のレイヤーである最終的な画像は、元の画像をチャネルごとに処理することによって見つけることができますが、結果のチャネルのカラー値は次の式によって計算されます：

ここでch0-このチャネルが不透明下層（あるred、green又はblue-赤、緑、青のため）ch1-上部半透明層の適切なチャネル、およびインデックスでありa尺度は同じ半透明層の透明度を指定していること- 。

これらの引数をSCSSコードの形式で表すと、次のようになります。

/*       */
@function res-ch($ch0, $ch1, $a) {
  @return $ch0 + ($ch1 - $ch0)*$a
}

@function res-col($c0, $c1, $a) {
  $ch: 'red' 'green' 'blue'; /*   */
  $nc: length($ch); /*   */
  $ch-list: ();

  @for $i from 0 to $nc {
    $fn: nth($ch, $i + 1);
    $ch-list: $ch-list, 
      res-ch(call($fn, $c0), call($fn, $c1), $a);
  }

  @return RGB($ch-list)
}

これはインタラクティブな例です。試してみると、2つのレイヤーの外観と、1つのレイヤーとして表示された同等のレイヤーを比較できます。ここでは、2つのレイヤーのRGBカラー値と、2番目のレイヤーの透明度レベルを選択できます。
