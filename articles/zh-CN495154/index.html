永恒领域

每个域区域都有自己的组织，该组织为其设置规则。例如，.ru区域以前是由RosNIIROS（俄罗斯公共网络发展研究所）管理的，现在已经创建了一个单独的组织，即国家Internet域名协调中心。

该组织为域内的争议注册和解决制定了规则。

如何购买域名十年？


最大的域更新期限由域区域法规确定。 .RU区域为1年，.COM和.NET区域为10年。但是，并非所有注册服务商都提供10年的延期。重要的是要注意，某些注册服务商提供的域名续订服务的期限比域名区域规则所允许的期限更长。这是一种营销策略，实际上，他们只是向您收取几笔续费费用，然后秘密地续订该域名，恕不另行通知。域名更新的真实期限在有效期限行中的whois中指示。


实际域的到期日期仅在whois中指定

重要的是，能够将实际域名过期与注册服务商在其面板中显示的内容区分开，因为续订的价格和条款可能会发生变化，然后注册服务商可以查看续订义务。另外，将域名转让给其他注册服务商时，仅考虑实际的续订期限。

因此，我们决定在尽可能长的时间内更新该域，请记住.COM和.NET可以延长10年。为此，您需要选择一个可以在此期间真正续订域名的注册商。它们的数量并不多，大多数大型外国注册商都可以做到这一点，例如：uniregistry.com，domain.com等。Cloudflare服务它不允许您向其注册域，但可以让您将域名转让期限延长至10年。

自动安全更新


如果出现僵尸末日，您可能会太忙而无法阅读有关安全性和新漏洞的常规报告。在掩体中为生存而斗争时，您很容易不会注意到服务器上的软件已变得易受攻击。

如果您没有及时修补漏洞，您将很快被黑客入侵，并将在其上启动加密货币矿工或将其添加到DDoS僵尸网络中。在这种情况下，极有可能您因恶意活动或资源超载而被阻止。

为了应对这种情况，有自动安全更新。在Ubuntu中，这称为无人值守升级，但是其他发行版中也有类似的版本。

考虑在Ubuntu上进行设置：

#   
apt-get install unattended-upgrades

#     
dpkg-reconfigure  unattended-upgrades


激活自动安全更新

默认情况下，系统每天检查一次安全更新，如果有更新，则自动安装。为此，服务器可以自动重启，因此配置必要的服务以在启动时自动启动非常重要。应当记住，无人值守升级仅安装关键的安全更新，不会更改已安装程序和发行版的版本。
