Pengalaman dalam menggunakan teknologi Rutoken untuk registrasi dan otorisasi pengguna dalam sistem (bagian 2)

Selamat sore! Kami akan terus membahas topik ini ( bagian sebelumnya dapat ditemukan di sini ).

Hari ini kita beralih ke bagian praktis. Mari kita mulai dengan mengatur otoritas sertifikasi kami berdasarkan pustaka sumber terbuka openSSL kriptografi lengkap. Algoritma ini diuji menggunakan windows 7.

Dengan menginstal openSSL, kita dapat melakukan berbagai operasi kriptografi (misalnya, membuat kunci dan sertifikat) melalui baris perintah.

Algoritma tindakan adalah sebagai berikut:

  1. Unduh paket distribusi instalasi openssl-1.1.1g.
    OpenSSL memiliki berbagai versi. Dokumentasi untuk Rutoken mengatakan bahwa openSSL versi 1.1.0 atau yang lebih baru diperlukan. Saya menggunakan versi openssl-1.1.1g. Anda dapat mengunduh openSSL dari situs resmi, tetapi untuk instalasi yang lebih mudah, Anda perlu menemukan file instalasi untuk windows di jaringan. Saya melakukannya untuk Anda: slproweb.com/products/Win32OpenSSL.html Anda
    perlu menggulir ke bawah halaman dan mengunduh Win64 OpenSSL v1.1.1g Penginstal EXE 63MB.
  2. Instal openssl-1.1.1g di komputer.
    Instalasi harus dilakukan sesuai dengan jalur standar, yang secara otomatis ditentukan dalam folder C: \ Program Files. Program ini akan diinstal di folder OpenSSL-Win64.
  3. Untuk mengkonfigurasi openSSL yang Anda butuhkan, ada file openssl.cfg. File ini terletak di jalur C: \ Program Files \ OpenSSL-Win64 \ bin jika Anda menginstal openSSL seperti yang dijelaskan dalam paragraf sebelumnya. Buka folder tempat openssl.cfg disimpan dan buka file ini menggunakan, misalnya, Notepad ++.
  4. Anda mungkin menduga bahwa pusat sertifikasi akan diatur entah bagaimana dengan mengubah isi file openssl.cfg, dan Anda memang benar. Untuk melakukan ini, konfigurasikan perintah [ca]. Dalam file openssl.cfg, bagian awal teks tempat kami akan membuat perubahan dapat ditemukan sebagai: [ca].
  5. Sekarang saya akan memberikan contoh konfigurasi dengan deskripsinya:
