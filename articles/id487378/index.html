Server Usaha Kecil Terbaik di tahun 2020



Meskipun adopsi server cloud tersebar luas, banyak yang memilih server lokal yang memberikan lebih banyak kontrol. Portal TechRadar telah menyusun daftar 10 server teratas, dari yang dirancang untuk bisnis kecil hingga yang dibutuhkan perusahaan besar.

1. Dell PowerEdge T30



A Dream on Xeon dengan Banyak

Prosesor Konektivitas : Intel Xeon E3-1225 v5 | Grafik: Intel HD Graphics P530 | RAM: 64GB | Penyimpanan: Hingga enam HDD SATA | Antarmuka Koneksi: 10 x Port USB, HDMI, DisplayPort, VGA, Serial Port, PS2, Gigabit Ethernet | Dimensi (lebar x panjang x tinggi): 17.5cm x 43.5cm x 36cm

+ Prosesor quad-core Xeon cepat
+ Memegang enam hard drive

- Port jaringan satu gigabit
- Tidak ada hot-swap drive

Ada begitu banyak server berbeda di sekitar yang kadang-kadang Anda perlu mempelajari seluruh daftar harga untuk menemukan server kecil terbaik yang cocok untuk Anda. Lihatlah Dell PowerEdge T30 - Anda dapat membelinya dalam versi untuk perakitan dan konfigurasi sendiri, atau dalam bentuk yang sudah selesai. Menargetkan pasar entry-level dan kantor kecil, T30 memiliki potensi besar untuk ekspansi ke bangunan berbentuk menara kecil, dan juga dilengkapi dengan banyak fungsi server default, menjadikannya alternatif yang ideal untuk workstation kantor.

2. Dell PowerEdge T20



Memperlihatkan betapa murahnya solusi perakitan dapat menjadi

Prosesor: Intel Pentium G3220 | Grafik: Intel HD Graphics | RAM: 4GB | Penyimpanan: tidak termasuk | Antarmuka Koneksi: Hingga port USB (4 x USB 3.0), 2 x DisplayPort, VGA, serial, 2 x PS2, Gigabit Ethernet | Dimensi (lebar x panjang x tinggi): 17.5cm x 43.5cm x 36cm

+ Sangat murah
+ Menara mini

ringkas dengan akses mudah ke bagian internal - G3220 lebih merupakan desktop daripada prosesor server
- Tanpa drive dan OS

Server tidak harus mahal dan masif. Lihatlah PowerEdge T20: ia mendapatkan yang terbaik dari pengetahuan Dell tentang 20 tahun pembangunan server. Meskipun versi build tidak memiliki hard drive, itu pasti murah - belum lama ini harganya kurang dari $ 130 (dengan mempertimbangkan cashback - Anda harus selalu melacak mesin-mesin ini).

Server ini memiliki prosesor berbasis arsitektur mikro Haswell, yang berakselerasi hingga 3 GHz dan mendukung hingga 32 GB DDR3 ECC RAM (perhatikan bahwa model ini hadir dengan 4 GB). Opsi ekspansi mencakup empat port SATA (32 terabyte jika Anda menginstal hard drive 8 TB), empat slot I / O, dan 10 port USB. Anehnya, komputer seharga ini memiliki dua soket DisplayPort, satu VGA, dua PS2, dan satu port serial. Selain port gigabit Ethernet, catu daya 290-watt dan pengontrol RAID Intel juga menarik.

3. Lenovo ThinkServer TS150





Prosesor Tower Menara Kinerja Tinggi : Intel Xeon E3-1200 v6 | RAM: hingga 64GB | Penyimpanan: HDD hingga 40TB | Antarmuka Koneksi: 8 x USB 3.0, Serial, Video, 2 x DisplayPort, Audio, Gigabit Ethernet | Dimensi (lebar x panjang x tinggi): 17,5 cm x 37,5 cm x 43 cm

+ Model paling terjangkau di garis ThinkServer
+ Sepenuhnya tenang

- Tidak ada yang perlu dikeluhkan

Lenovo merebut pasar server x86 dari IBM pada tahun 2014 dan menciptakan salah satu jajaran terbaik, ThinkServer. TS150 adalah salah satu model paling terjangkau dalam jajaran ini, yang merupakan server 4U kelas perusahaan yang sebanding dengan Dell T20. Ini memiliki dukungan RAID 0,1,10 dan 5 (melalui controller pada motherboard). Seperti pesaing, server ini dapat menampung hingga empat hard drive 3,5 inci, yang berarti dapat memiliki penyimpanan hingga 40 TB saat memasang hard drive yang sesuai. Prosesor Intel Xeon E3-1200 v6 yang relatif segar harus cukup produktif untuk usaha kecil dan menengah.

Lenovo juga mengklaim bahwa suara kerja TS150 lebih tenang daripada tingkat kebisingan di perpustakaan, sekitar 26 dB. Seperti halnya server lain, Anda mendapatkan berbagai port dan antarmuka koneksi yang mengesankan: 8 port USB, 4 slot PCI / PCI-E, tiga output video (termasuk dua slot DisplayPort), port serial, gigabit Ethernet, dan 3 konektor untuk audio.

4. HPE ProLiant ML350 Gen 10





Prosesor yang Kuat dan Serba Guna: Intel Xeon Scalable 4210 | Grafik: Intel HD Graphics P530 | RAM: 16GB | Penyimpanan: drive tidak termasuk | Dimensi (lebar x panjang x tinggi): 18 cm x 65 cm x 47 cm

+ Scalable
+ Desain yang layak

- Mesin besar
- Dilengkapi tanpa hard drive

Jika Anda membutuhkan server yang andal dan serbaguna, maka HPE ProLiant ML350 Gen 10 adalah pilihan yang bagus. Model ini memberikan kinerja tinggi karena menginstal prosesor Intel Xeon Scalable. Anda sendiri perlu menginstal penyimpanan Anda sendiri, tetapi server ini mendukung berbagai operasi cepat, termasuk bekerja dengan tugas-tugas grafis dan komputasi. Meskipun HPE ProLiant ML350 Gen 10 hadir sebagai server menara, dengan perluasan bisnis Anda, itu dapat dikonversi ke server rak, yang memungkinkan Anda untuk memperluas bisnis Anda tanpa biaya tambahan dan kebutuhan untuk membeli server yang sama sekali baru.

5. Fujitsu Primergy TX1310 M1



Keandalan Fujitsu Dijamin

Prosesor Yang Tidak Tertandingi: Intel Xeon E3-1226 v3 | Grafik: Intel HD Graphics P4600 | RAM: 16GB | Penyimpanan: HDD 2 x 1TB | Antarmuka Koneksi: 5 x USB 3.0, 2 x Gigabit Ethernet Ports | Dimensi (lebar x panjang x tinggi): 17,5 cm x 42 cm x 39,5 cm

+ Jaminan luar biasa
+ Memiliki drive optik

- Tanpa RAID 5
- Fujitsu - bukan merek yang paling jelas

Fujitsu bukan merek pertama yang terlintas dalam pikiran ketika datang ke server. Namun, Fujitsu adalah salah satu dari sedikit pabrikan yang menangani semuanya, mulai dari server bisnis hingga superkomputer. TX1310 adalah server berorientasi bisnis entry-level yang telah membuktikan dirinya dalam operasi dan memiliki jaminan keandalan terbaik di pasar. Jika server gagal dalam waktu satu tahun setelah pembelian, Fujitsu tidak hanya akan memperbaiki atau menggantinya, tetapi juga akan mengembalikan jumlah yang Anda bayarkan untuk server ini.

Seperti kebanyakan model di segmen pasar ini, server dirancang untuk bekerja 24/7 dan mendukung RAID 0/1/10 (tanpa dukungan RAID 5). Intel Xeon E3-1216 dari versi ketiga dipasang di dalamnya, dua hard drive dengan 1 TB dan 16 GB RAM. Nilai plus adalah fakta memiliki drive optik dan dua port Ethernet gigabit untuk cadangan. Dengan empat slot DIMM dan empat ruang penyimpanan, server ini dapat menyimpan hingga 32 TB data dan mendukung hingga 32 GB RAM.

6. HP Proliant Microserver Gen8





Prosesor server yang ringkas namun sedikit bising : Intel Celeron G1610T | Grafik: Matrox G200 | RAM: 4GB | Penyimpanan: drive tidak termasuk | Antarmuka Koneksi: 4 x USB 2.0, 2 x USB 3.0, VGA, 2 x Gigabit Ethernet | Dimensi (lebar x panjang x tinggi): 23 cm x 24,5 cm x 23 cm

+ Profesional membangun kualitas dan kemudahan akses
+ Kekompakan

- Tidak ada hot-swap drive
- Bising

Di salah satu segmen pasar dengan pertumbuhan tercepat, hanya satu perusahaan yang mendominasi. HP Proliant Microserver Gen8 berhasil bersaing dengan kombinasi fitur yang menarik dan banyak diskon - dan akhirnya mengambil alih pasar. Server kecil ini juga mendapatkan permintaan di luar ceruk mereka karena popularitas di kalangan konsumen profesional dan iklan keunggulan dibandingkan NAS NAS.

Meskipun ringkas (volume kurang dari 0,013 m3) dan ringan (kurang dari 7 kg), mesin ini memiliki kemampuan yang mengesankan. Kita berbicara tentang dukungan untuk keluarga prosesor Intel Xeon E3, kemampuan untuk menginstal hingga 16 GB RAM, mode manajemen sistem, dua port Ethernet gigabit, satu slot PCIe, dukungan untuk RAID 0/1/10, penulis DVD, hingga empat hard drive , slot kartu microSD internal, chip grafis Matrox G200 terintegrasi, dan tujuh port USB. Di sisi lain, server ini hanya memiliki satu konektor VGA dan hanya dua konektor untuk RAM.

7. Lenovo ThinkServer TS460



Server tangguh yang dapat menampung hingga delapan disk.

Prosesor: Xeon E3-1200V6 | Grafik: Intel HD Graphics P630 | RAM: hingga 64GB | Penyimpanan: 2x teluk internal tetap 5,25 inci, maksimum 80TB | Antarmuka Koneksi: 6 x USB 3.0, port serial, konektor video, 2 x Gigabit Ethernet | Dimensi (lebar x panjang x tinggi): 44,4 cm x 58,3 cm x 17,3 cm

+ Kinerja mengesankan
+ Garansi tiga tahun dengan layanan di tempat -

Biaya
tinggi - Ukuran besar

Jika Anda menginginkan sesuatu yang lebih kuat dari server di atas, periksa TS460. Ini adalah solusi yang jauh lebih mahal, tetapi Anda akan mendapatkan server dari level yang berbeda. Pertama-tama, ini jauh lebih besar dari model sebelumnya. Ini memiliki volume 0,05 m3 dan berat 25 kg. Server faktor bentuk 5U ini berjalan pada keluarga prosesor Intel Xeon E3 dengan teknologi Turbo Boost dan dijamin layanan di tempat.

Ini mendukung hingga 64 GB RAM, dan pengontrol terintegrasi mendukung empat jenis utama RAID. Anda juga akan menerima penulis DVD, catu daya 300 watt, dan dua port Ethernet gigabit. Dimungkinkan untuk menginstal hingga delapan hard drive dan sebanyak delapan konektor USB. Kasing ini memiliki pintu yang dapat dikunci, mendukung memori server ECC, serta konektor serial dan VGA.

8. HP ProLiant ML350 G9 5U





Prosesor Server Daya Tinggi Sangat Dilengkapi: Intel Xeon E5-2603 v3 | Grafik: Matrox G200 | RAM: 8GB | Penyimpanan: drive tidak termasuk | Antarmuka Koneksi: 4 x Gigabit Ethernet | Dimensi (lebar x panjang x tinggi): tidak diketahui

+ Prosesor Xeon enam-inti
+ Layanan tiga tahun di tempat sehari setelah perawatan

- Dompet Anda pasti akan bangkrut
- Tidak ada lagi yang perlu dikeluhkan

Model ini disajikan dalam kategori yang sama dengan TS440 dan ML350. Selain memiliki kartu video terintegrasi (pada Matrox G200), ada jaminan layanan tempat kerja tiga tahun ke depan, empat port Ethernet gigabit dan dukungan SAS 12 Gb / s (perhatikan bahwa ini hanya membutuhkan drive 2,5 inci).
Selain itu, server ini tidak berfungsi dengan prosesor Intel Xeon E5-2603 dari versi ketiga (dan tidak dengan prosesor E3 biasa) dan memiliki dukungan untuk dua prosesor. E5 adalah prosesor enam inti, yang membuatnya sangat cocok untuk menyelesaikan tugas yang lebih kompleks. Kami juga terkesan dengan jumlah slot yang tersedia untuk RAM (24 slot), yang memungkinkan untuk mengumpulkan total 3 TB (saat memasang LRDIMM-strip dengan 128 GB). Selain itu, selain pintu depan yang dapat dikunci dan pengontrol penyimpanan, server ini mendapatkan poin tambahan karena memiliki dua catu daya redundan 500 watt dengan fitur hot-swappable.

9. Pindai 3XS SER-T25



Monster yang berjalan pada dua

prosesor Xeon Processor: Dual Intel Xeon E5-2603 v4 | Grafik: Nvidia GT 610 | RAM: 64GB | Penyimpanan: HDD 1TB | Antarmuka Koneksi: 2 x Gigabit Ethernet | Dimensi (lebar x panjang x tinggi): 23,2 cm x 56 cm x 55,9 cm.

+ Ringkas dan senyap
+ Sangat kuat

- Tidak murah (tapi mengingat daya, harganya bagus)
- Keluhan lain akan menjadi nit-picking

Server yang kuat tidak harus mahal - ini yang ingin disampaikan oleh Scan kepada pelanggan potensial. Didesain khusus untuk pasar SMB, 3XS dirancang untuk sekompak dan senyap mungkin. Pemasok Inggris menyediakan pemantauan real-time dari semua proses perakitan (server dirakit sesuai pesanan, produksi mencakup uji stres 24 jam dan uji kualitas 88 poin). Setiap perangkat dilengkapi dengan jaminan layanan di tempat selama tiga tahun, dan Anda juga akan menerima flash drive USB dengan utilitas pemulihan diagnostik.

Jika ini tidak cukup, komponen dari majelis ini adalah yang terbaik dalam tinjauan kami. Dua prosesor Intel Xeon E5-2603 dari versi keempat tentang arsitektur mikro Broadwell secara total memiliki 12 core dan 30 MB cache. Selain itu, ada 64 GB RAM, Samsung DDR4 ECC RAM, hard drive WD kelas perusahaan, dua port Intel gigabit Ethernet, catu daya 1000 watt dengan sertifikasi 80 PLUS Gold dan delapan hard drive. Kandang Corsair memiliki pintu dan panel dengan penyerapan suara.

10. Asus TS500



Mainstream tower server dengan margin fleksibilitas

Prosesor: Intel Xeon E5-2600 v3 | Grafik: tidak diketahui | RAM: tergantung pada urutan | Penyimpanan: drive tidak termasuk | Antarmuka Koneksi: 8 x port USB, VGA, PS2, 3 x Gigabit Ethernet | Dimensi (lebar x panjang x tinggi): 22 cm x 54,5 cm x 45,5 cm

+ Solusi ideal untuk server dan workstation
+ Tidak ada kekurangan daya dan fleksibilitas

- Tidak semua orang membutuhkan fitur ini
- Logo Asus mungkin mengejutkan beberapa

Seperti halnya Supermicro, Asus tidak terlalu terkenal dengan servernya. Perusahaan Taiwan, yang merupakan salah satu pemasok komponen komputer terbesar, lebih dikenal dengan berbagai produk konsumennya, termasuk motherboard. TS500-E8-PS4 adalah server menara utama 5U yang ideal untuk digunakan sebagai server atau workstation.

Ini dilengkapi dengan prosesor Intel Xeon E5-2600 terbaru dari versi ketiga, delapan slot RAM DDR4 DIMM (mendukung hingga setengah TB RAM), enam slot ekspansi, tiga ruang penyimpanan 5,25 inci dan catu daya 500 watt dengan catu daya 500 watt dengan sertifikat 80 Plus Bronze . Ada juga empat hard drive SATA / SAS hot-swappable 3,5 inci, yang dapat diganti dengan 8 bay HDD jika membutuhkan penyimpanan yang fleksibel.

Selain itu, menggunakan modul ASMB8-iKVM, kompatibel dengan Intel® Platform Management Interface (IPMI), Anda dapat memonitor, mengontrol, dan mengelola server dari jarak jauh. Fitur utama lainnya termasuk 10 port SATA, penulis DVD, 8 port USB, port PS2, satu port VGA, dan tiga port Ethernet gigabit. Harap dicatat bahwa ini adalah solusi perakitan, tetapi jelas bahwa ini memberi Anda fleksibilitas besar yang dimiliki mesin ini.
