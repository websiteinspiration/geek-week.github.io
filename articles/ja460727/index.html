単純なテキスト暗号化-複雑さを制御できる数値のタプル


Github


https://github.com/denisxab/SHISH

テキスト用の非常に単純な暗号化アプリケーションを作成したかったのですが、突然そのようなニーズが発生するでしょう。重要なのは

、暗号化は特定のビットからの乱数で行われ、2番目のrand値は各文字のキーの数を設定するため、繰り返されず、テキストの規則性を見つけることができません。

キー自体は.jsonに保存されるため、送信、検索、編集は簡単です。

使い方 ？


  1. キーを作成します。
  2. キーを共有します。
  3. 「エンコード」ウィンドウにテキストを入力し、ボタンをクリックします。
  4. デコードウィンドウからテキストを送信します。

tkinterの特殊性のため、リストではなく通常のstrで応答するため、デコードウィンドウの各インデントが重要です。

キーの作成


bit = input()
rand = input()
l = {}
a = list('','','','','')

for x in a:
	l[x]=[]
	for y in range(rand):
		l[x].append(random.getrandbits(bit))
print(l)
