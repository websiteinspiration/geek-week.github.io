Pythonプログラムを使用してダウンロードフォルダーをクリアする

前書き


私はPythonを学び始めたばかりの人として、直接的な実践を通じて新しい知識とスキルを習得しようとしています。私の意見では、このアプローチはプログラミングガイドやマルチボリュームを作成するよりも優れています。まず、これからお話しするようなプロジェクトで、私は自分のニーズを閉じようとしています。2番目の場所は、他の誰かが私が作成したものに役立つことができるという希望です（すべての既存のエラーが修正された後）

このプログラムのアイデアは、「ダウンロード」フォルダーに形成された「ゴミ」の中から以前にダウンロードしたファイルを見つけようとしたときに生まれました。音楽付きの画像、ドキュメント、アーカイブ、ビデオ：もちろん、この混乱を理解し、ファイルをフィルタリングすることはできますが、時間のコストが大きいため、気が進まないでしょう。そして、「ダウンロード」フォルダを解析し、「画像」、「ドキュメント」、「ビデオ」、「音楽」システムの標準フォルダにファイルを配布し、不要なファイルを削除するプログラムがあるといいと思いました。その結果、パーフェクショニストオーガナイザーが誕生しました。これは初心者から初心者までの記事であることをすぐに言っておく必要があります。経験豊富なPythonプログラマーはここでは何も役に立たないからです（コードを掘り下げて多くのエラーを指摘する機能を除く）。

一般的なアイデア


プログラムが最初に行うべきことは、プログラムが実行されているオペレーティングシステムを判別することです。LinuxとWindowsではDownloadsフォルダーの配置が異なるため、これが必要です（このOSを考慮に入れていないため、Macでプログラムをテストすることができません）。検出されたダウンロードフォルダーで、プログラムは利用可能なすべてのファイルを調べ、それらが属するタイプ（音楽、ビデオ、ドキュメントなど）を決定します。その後、検出されたファイルはデフォルトのフォルダに移動さ​​れ、プログラムはユーザーから移動に適さないファイルをどうするかを見つけます-そのままにするか削除します。

必要なライブラリをインポートし、システムのタイプを決定します


オペレーティングシステムとその中のファイルを操作するには、osライブラリが必要です。getpassライブラリはシステムのユーザー名を知らせ、プラットフォームはそれがWindowsかLinuxかを判断します。必要なすべてのライブラリを最初にインポートで接続し、OSのタイプとユーザー名を決定します。

import os
import getpass
import platform
type_os = platform.system()
usermane = getpass.getuser()

人気のあるファイルの拡張子とそれに関連付けられているデフォルトのフォルダーを含む辞書を作成する


標準のPythonツールを使用して、音楽、画像、動画、ドキュメントなど、最も一般的なファイルタイプの4つの辞書を作成します。

video_folder = {".3gp" : "/", ".avi" : "/", ".flv" : "/", ".m4v" : "/", ".mkv" : "/", ".mov" : "/", ".mp4" : "/", ".wmv" : "/", ".webm" : "/"}
music_folder = {".mp3" : "/", ".aac": "/", ".flac" : "/", ".mpc" : "/", ".wma" : "/", ".wav" : "/"}
pic_folder = {".raw" : "/", ".jpg" : "/", ".tiff" : "/", ".psd" : "/", ".bmp" : "/", ".gif" : "/", ".png" : "/", ".jp2" : "/", ".jpeg" : "/"}
doc_folder = {".doc" : "/", ".docx" : "/", ".txt" : "/", ".rtf" : "/", ".pdf" : "/", ".fb2" : "/", ".djvu" : "/", ".xls" : "/", ".xlsx" : "/", ".ppt" : "/", ".pptx" : "/", ".mdb" : "/", ".accdb" : "/", ".rar" : "/", ".zip" : "/", ".7z" : "/"}

最後に/があるので、フォルダ名の後にファイル名が残ることを忘れないでください。辞書を編集することで、新しいファイルタイプを簡単に追加できます。

ユーザーにダウンロードフォルダの名前を尋ねます


WindowsとLinuxでは、異なる名前のシステムにダウンロードフォルダーが表示されます。Windowsの場合、これはDownloadsフォルダーのローカライズに関係なく、Linuxディストリビューションではこのフォルダーは「Downloads」と呼ばれます。さらに、ユーザーがダウンロードしたファイルを含むフォルダーの名前を変更する場合があり、ディレクトリの現在の名前が標準の名前と異なるかどうかを確認する必要があります。

if type_os == "Linux":
    user_downloads_path = input("      ? (-: ) ") or ""
if type_os == "Windows":
user_downloads_path = input("      ? (-: ) ") or "Downloads"

or 構文を使用すると、inputでデフォルト値を使用できます。つまり、ユーザーがダウンロードしたフォルダーの名前を変更していない場合、何もする必要はありません。プログラム自体が、必要な値を以降のコードで置き換えます。

ダウンロードフォルダへのパスを設定します


オペレーティングシステムによっては、ダウンロードフォルダーが別の場所にある場合があります。Linuxの場合、このフォルダーはパス/ home / username / Downloads /にあり、Windowsの場合はパスC：/ Users / username / Downloads /です。これらのパスを後で手動で指定しないようにするには、これらのパスを指定する変数を作成する方が簡単です。

if type_os == "Linux":
    default_path_d = "/home/" + usermane + "/" + user_downloads_path + "/"
else:
    default_path_d_win = r"C:/Users/" + usermane + r"/" + user_downloads_path+ r"/"

特定のユーザーのダウンロードフォルダーへのパスを設定する


以前に接続されたモジュールの機能をさらに使用してオペレーティングシステムとファイルを操作するには、以前に作成して受け取った変数を使用して、ダウンロードフォルダーへのパスを指定する必要があります。

if type_os == "Linux":
    downloads_path = os.listdir("/home/" + usermane + "/" + user_downloads_path)
else:
    downloads_path_win = os.listdir(r"C:/Users/" + usermane + r"/" + user_downloads_path)

さて、そしてコードをさらに使用するために、フォーム/ホームフォルダー/ユーザー名のパスを設定します

if type_os == "Linux":
    default_path_u = "/home/" + usermane + "/"
else:
    default_path_u_win = r"C:/Users/" + usermane + r"/"

ダウンロードフォルダー内の特定のファイルを確認する


辞書を列挙することにより、そのキーをダウンロードフォルダーにあるファイル拡張子と比較します。拡張子がキーと一致する場合、必要なファイルを対応するデフォルトのフォルダーに移動する必要があります（名前はディクショナリの値です）。

#      .  ,     
for music_format in music_folder:
    if type_os == "Linux":
        for name_file in downloads_path:
            if name_file.endswith(music_format):
                result = name_file.split(str(music_format), 1)
                os.rename(default_path_d + result[0] + music_format, default_path_u + music_folder.get(music_format) + result[0] + music_format)
    if type_os == "Windows":
        for name_file in downloads_path_win:
            if name_file.endswith(music_format):
                result = name_file.split(str(music_format), 1)
                os.rename(default_path_d_win + result[0] + music_format, default_path_u_win + "Music" + r"/" + result[0] + music_format)
#      .  ,     
for pic_format in pic_folder:
    if type_os == "Linux":
        for name_file in downloads_path:
            if name_file.endswith(pic_format):
                result = name_file.split(str(pic_format), 1)
                os.rename(default_path_d + result[0] + pic_format, default_path_u + pic_folder.get(pic_format) + result[0] + pic_format)
    if type_os == "Windows":
        for name_file in downloads_path_win:
            if name_file.endswith(pic_format):
                result = name_file.split(str(pic_format), 1)
                os.rename(default_path_d_win + result[0] + pic_format, default_path_u_win + "Pictures" + r"/" + result[0] + pic_format)
#        .  ,     
for doc_format in doc_folder:
    if type_os == "Linux":
        for name_file in downloads_path:
            if name_file.endswith(doc_format):
                result = name_file.split(str(doc_format), 1)
                os.rename(default_path_d + result[0] + doc_format, default_path_u + doc_folder.get(doc_format) + result[0] + doc_format)
    if type_os == "Windows":
        for name_file in downloads_path_win:
            if name_file.endswith(doc_format):
                result = name_file.split(str(doc_format), 1)
                os.rename(default_path_d_win + result[0] + doc_format, default_path_u_win + "Documents" + r"/" + result[0] + doc_format)

当初、私は正規表現を使用してファイル拡張子を見つける問題を解決することを計画し、それらの定式化に従事しました。しかし、問題を解決したい場合は、正規表現を使用する必要がある最後のサイトだけを読んでみました。 Pythonには、他にも簡単で直感的な方法があり、必要なことを実行できます。したがって、これらの式の代わりに、ファイル拡張子は文字列メソッドendwith（）を使用して決定されます。辞書からキーを取得し、ファイルがそれで終わっているかどうかをチェックします。次に、split（）メソッドを使用して、os.rename（）を使用した移動のためにファイル名のみが取得されます。後者の引数として、以前に作成されたすべての変数が使用されます。

残りのファイルをどうするかをユーザーに尋ねます


必要なファイルがソートされた後、プログラムは終了するか、移動に該当しないファイル（exe-filesやdeb-packagesなど）を削除する必要があります。決定はユーザーが行い、デフォルトでは削除は行われません。

#       
delete_user_confirm = input('     ?     (-: ) ' or '')
if delete_user_confirm == '':
    if type_os == "Linux":
        files_to_remove = os.listdir(default_path_d)
        for remove_files in files_to_remove:
            os.remove(default_path_d + "/" + remove_files)
    if type_os == "Windows":
        files_to_remove = os.listdir(default_path_d_win)
        for remove_files in files_to_remove:
            os.remove(default_path_d_win + "/" + remove_files)
else:
    print('  .   .')
