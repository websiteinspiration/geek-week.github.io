Plesk、cPanel、ISPmanager：何を選択すればよいですか？

作業を始める前にプロバイダーから提供されたすべてのパネルをテストすることは難しいため、短いレビューで最も人気のある3つを収集しました。 クライアントがOS管理からホスティング関連のタスクに移行すると、問題が発生します。彼は、異なるCMSと多数のユーザーアカウントを持つ多くのサイトを管理する必要があります。人件費を削減するには、便利なWebインターフェースを介して対応するサービスを構成できるコントロールパネルをインストールする価値があります。また、サービスを顧客に販売するプロバイダーのパートナーにも必要です。今日、LinuxでVPSおよびVDSを注文するときに利用できる3つの人気のある製品を比較します。

▍Plesk


すべてのタイプのタスクに適した、最も用途の広いオプションの1つ。パネルは、人気のあるdebベースおよびrpmベースのLinuxディストリビューションだけでなく、Windowsでも機能します。 Windows VPS / VDSのお客様がサードパーティの管理ツールを必要とすることはほとんどありませんが、必要に応じてインストールできます。 Pleskはまた、以下を含む多数のサポートされているソフトウェアで競合他社とは異なります。従来のWebサーバー（Docker、NodeJS、Git、Rubyなど）ではほとんど使用されません。

開発者は、最小限の機能セットを備えた軽量バージョンを含む、製品のさまざまなエディションを提供します。Pleskを使用すると、サイトごとにPHPのバージョンを選択でき、PHP-fpmをサポートし、人気のCMSの組み込みインストーラーと、パネルの機能を補完する多数の拡張機能を備えています。エディションによっては、Pleskには請求パネルのほか、さまざまな料金プランや再販業者を作成する機能が含まれる場合があります。この製品は主にホスティング会社とWebスタジオを対象としており、一部のVPS / VDSでは機能が冗長になっています。この段階で判明したPleskの主な欠点は、ライセンスの価格が高く、拡張機能を購入する必要があることです。

▍cPanel＆WHM


このパネルは、RedHat Enterprise Linuxおよび一部の派生ディストリビューションで動作するように設計されています。使い方は簡単ですが、非常に機能的です。cPanelを使用すると、Webサーバーとデータベースの管理、ホスティングユーザーの制限の柔軟な設定、料金プランの設定、リセラーの作成、フィルターとニュースレターによるメールサービスの管理を行うことができます。Pleskと同様に、ここには多くの追加機能があり、cPanel機能は商用および無料のプラグインで拡張されます。さらに、このツールを使用すると、異なる動作モードと異なるバージョンのPHPを選択できます。深刻な欠陥のうち、かなり高いライセンスコストと、人気のあるdebベースのディストリビューションのサポートの欠如に注目できます。

▍ISPmanager


最後に確認したパネルは、低価格で他のパネルと異なります。さらに、CentOS（RHELクローン）だけでなく、Debian / Ubuntuでも機能します。パネルはホスティングタスク用に最適化されており、自動的に更新されます。ユーザーは、詳細なロシア語のドキュメントにアクセスしたり、各サイトのPHPバージョンを設定したり、Dockerコンテナー内に複数のバージョンのDBMSを同時にインストールしたりできます。PHP-fpmがサポートされています。一般的なスクリプトとCMSの組み込みインストーラー、および機能を拡張する多数の統合モジュールがあります。 

RuVDSの価格


上の表は、開発者サイトで購入した場合のPlesk、cPanel、ISPmanagerライセンスの価格帯を示しています。多くのホスティングプロバイダーは、すぐにサーバーにパネルを装備することを提案していますが、ライセンスのコストは低くなる可能性があります。新年キャンペーンの一環として、RuVDSはVPSを注文したお客様に、2019年12月31日までは無料でISPmanager Liteを、2020年1月31日まではPlesk Web管理版を使用する機会を提供します。プロモーション後、ライセンスの費用は1か月あたり200ルーブルと650ルーブルになります。cPanelの試用版は14日間無料で使用できますが、その後、開発者から直接ライセンスを購入する必要があります。

第一印象


パネルのインストールと起動にはクライアントが問題はありません。これはすでに対処済みです。これは、ホスティング業者を通じてライセンスを購入するもう1つの理由（価格を除く）です。サーバを注文するときは、3つの利用可能なオプションのいずれかを選択してください。ISPmanagerLite、Plesk Web管理エディション、または14日間の無料試用期間付きのcPanel＆WHM。PleskはWindows Server上で実行できますが、このオプションはそのままでは提供されないことに注意してください。MicrosoftのOS用のパネルが必要な場合は、自分でインストールする必要があります。これは一般的な方法です。WindowsのVPS / VDSにはサードパーティのソフトウェアは含まれていません。cPanelはCentOSマシンでのみ使用できますが、これもかなり自然なことです。 


サイトの初期設定と作成によって特定の問題が発生することはありませんが、特定の各パネルの機能はここではすでに重要です。それらの長所と短所を強調してみましょう。

▍Plesk


Pleskユーザインターフェースは、WordPress管理ダッシュボードに似ています。左側はメニュー（ナビゲーションバー）で、中央はワークスペースです。メニューは非常に論理的に構成されており、すべての設定が手元にあります。インターフェースはWordPress管理パネルに似ています。Pleskとこの人気のあるCMSの緊密な統合が非常に気に入っており、そのインストールはここで非常に自動化されています。他のサードパーティのスクリプトをインストールすることは非常に便利です-これは大きなプラスです。
 

ウィンドウの右側には、パネルの操作を簡素化する追加のインターフェース要素があります。これらにはさまざまな情報が含まれており、設定のさまざまなセクションにすばやく移動したり、追加のソフトウェアのインストールを提案したりできます。Pleskの主な利点は、膨大な数の拡張機能と、Webホスティング用のエキゾチックなソフトウェアとの互換性です。私たちは特に、すぐに使えるDockerサポートと豊富な既製イメージのセットが好きでした（独自のイメージをアップロードすることもできます）。

 
最後に、軟膏の小さなフライ：Plesk Web管理エディションでは基本機能のみが使用可能で、より高価なエディションではリストがはるかに広くなっています。ただし、これはエントリーレベルのバージョンの一般的なプロパティです。

▍cPanel＆WHM


ここでは、アカウントを2つのタイプ（ユーザーと管理者/リセラー）に分けることが好きでした。実際、この製品は、cPanel自体とWebHost Manager（WHM）の2つの異なるパネルで構成されています。 1つ目は、通常のホスティングユーザーを対象としており、非常に便利です。 


管理者とリセラー向けの料金プランを作成する機能を含む機能は、特別なWHMパネルから利用できます。このパネルのインターフェースは全体として論理的に構成されています。従来、検索バー付きの非表示の階層メニューは左側にあり、作業領域は右側にあります。多くの設定があり、これは一方では良いことです。一方、WHMメニューは便利とは言えません。Pleskで検索を使用する必要がほとんどなかった場合、ここの各セクションには非常に多くのオプションがあり、検索文字列がメインの管理者ツールになります。 


▍ISPmanager


このコントロールパネルと以前のコントロールパネルとの重要な違いは、最もシンプルで直感的なインターフェイスです。左側はナビゲーションメニュー、右側はワークスペースです。ワークスペースのタブでさまざまなメニューオプションを個別にまたは同時に開くことができます。これは、管理者が異なるパネル機能を並行して必要とすることが多いため、非常に便利です。ホスティングに直接関連する機能に加えて、管理者は、ウイルス対策スキャン、ファイルマネージャー、スケジューラー、ファイアウォールなどのシステム機能だけでなく、いくつかの追加機能にもアクセスできます。パッケージに含まれている追加のアプリケーションのリストには、Roundcube WebmailとphpMyAdminが含まれています。


初期設定の単純さとソフトウェアを自動的に更新する機能、およびパネルのすべてのロシア語ローカリゼーションとすべての関連ドキュメントが気に入りました。外国の開発ではこれが困難です。一方、簡素化されたインターフェースは、必要な設定の柔軟性を常に備えているわけではなく、ISPmanagerで利用可能な追加モジュールの数は、PleskとcPanelのコレクションと比較して、非常に少ないです。さらに、Liteの最も安いエディションでは、リセラーとクラスター構成を作成できません。


安全性


コントロールパネルは、サーバーにインストールされているオペレーティングシステムの幅広い権限を管理者に与えます。そのため、その中に脆弱性が存在する可能性は危険です。デフォルトでは、自己署名証明書を備えた暗号化が有効なHTTPSプロトコルを使用して、リストされているすべてのパネルの機能にアクセスします。この場合、購入した証明書のインストールをユーザーが禁止することはありません。さらに、cPanelおよびISPmanagerは、管理者/リセラーおよびクライアントの2要素ログイン認証を構成します。さらに、cPanelには管理ツールに対する追加の保護機能があります。たとえば、直接リンクを介したphpMyAdminアクセスはありません。また、3つのパネルはすべて定期的に更新され、Webサイト用のSSL証明書（自己署名証明書を含む）をインストールでき、さまざまなセキュリティ関連モジュールをパネルに追加できます。ウイルス対策ツールのように。

バックアップ


Pleskは、独自のストレージまたは外部リソースへの完全および増分バックアップをサポートしています。この場合、サーバー全体の完全なコピーと、個々のユーザーアカウントのデータのコピーの両方を作成できます。cPanelは、圧縮コピーと非圧縮コピー、および増分コピーを作成します-デフォルトではローカルに保存されます。スケジュールされたコピー手順を開始できること、およびデータ回復用の独自のインターフェースがないことは注目に値します。

私たちの考えでは、ISPmanagerのバックアップ設定は十分な柔軟性を備えていませんが、このパネルにはすべての主要機能があります。データはローカルディレクトリまたは外部リソースに保存され、パスワードで保護できます。デフォルトでは、すべてのユーザーのデータがコピーされますが、これは設定で変更できます。さらに、設定は、フルバックアップと日次バックアップの数を示します。

長所と短所 


3つのパネルすべてが、最も人気のあるリストのトップをレビューし、幅広い機能を特徴としています。 Pleskはさまざまなソフトウェアをサポートしており、幅広いタスクを解決できます。ユーザーは200,000を超える多様なDockerイメージを利用できます。また、膨大な数の拡張機能により、Pleskはホスティングだけでなく、ユニバーサルツールとしても最適です。 cPanelはホスティング関連の問題を解決するように設計されていますが、開発者はさまざまな機能へのアクセスを2つのレベルに分けました。通常のユーザーと管理者用に別々のパネルが作成されました。また、コンピューティングリソースに対する高い要求にも注目する必要があります。cPanelを低電力VPSにインストールしないでください。 ISPmanagerパネルもホスティング管理のみを目的としています。使いやすいですリソースを必要とせず、費用もかからない-おそらくこれは、エントリーレベルのVPSや初心者の管理者やホスティング業者にとって最良のオプションです。
