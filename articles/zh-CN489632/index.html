RedHat研究：开源将专有软件带出企业领域




RedHat团队在其研究报告中发布了这一结论，即开源软件正逐渐占据企业市场。在对全球950家IT公司高管进行调查之后，该公司得出了这样的结果。以下是摘要结果：

  1. 95％的人认为开源具有战略重要性；
  2. 多年来，专有软件在企业领域的份额一直在急剧下降，从2019年的55％下降到现在的42％，据预测，到2021年将下降到32％。
  3. 77％的人认为企业开源将继续增长；
  4. 开源的主要优势被称为“更高质量”，仅次于第二位的成本更低；
  5. 主要应用领域：安全性，云基础架构管理工具，数据库，大数据和分析；
  6. 使用障碍：代码安全性，支持级别，兼容性，缺乏自己的支持能力。

在向PaaS过渡的过程中，OpenShift在组织模型演变中的作用




RedHat在其Habré博客上发表了有关OpenShift容器化软件系列在迁移到Paas时如何改变IT组织的组织结构的说明。Quote-“只有在组织角色，职责范围（任务）和关系发生变化的情况下，PaaS的最大投资回报率才可能实现。” 导致这种变化的原则：

  1. 将工作分为几个小阶段，以便在早期阶段获得反馈，降低风险并避免“分析性瘫痪”；
  2. 充分自动化操作，以免在应用程序部署过程中造成障碍或瓶颈；
  3. 知识共享是建立信任的关键；
  4. 定期偿还技术债务，在每个工作周期中分配一定的时间进行系统改进。

在英国，建议父母如果孩子使用VirtualBox，Discord，Kali Linux和Tor，请与警方联系。




海报出现在英国西米德兰兹郡的学校中，呼吁家长和老师与学生联系，如果有学生使用Kali Linux OS，Discord Messenger，Tor洋葱路由工具，VirtualBox虚拟化工具和其他软件。据称这些海报是根据当地警察的命令发行的。此信息的发布立即引起了许多人的强烈反响，并立即赶紧将海报从NCA（英国国家反犯罪机构）的海报上撤下，NCA的徽标被放置在海报上。在批评中，反对这种张贴者的主要论点是作者的技术能力不足，而且所列软件根本没有清楚地表明孩子在做可疑行为。遭到批评后，当地警察也指责第三方。
