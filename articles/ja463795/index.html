英語の意味の濃淡：動詞の使い方アドバイス、推奨、提案



多くの場合、英語は同じドイツ語を話す他の言語ほど複雑ではないものとして認識されます。しかし、実際には、そこには十分な困難と微妙な点があります（「リバプールが書かれ、マンチェスターが読み取られます」）。

個人的に私に大きな問題を引き起こしているそれらの1つは、意味が近い言葉の正しい使用です。たとえば、助言、推奨、提案の動詞をいつ使用するかを理解することはそれほど簡単ではありませんでした。私はこの質問が解析された興味深い投稿を見つけ、それを適応させた翻訳を準備しました。

これらの動詞は何ですか？


一般に、これら3つの単語「推奨」、「提案」、「アドバイス」はすべて、ある人が他の人に何をすべきかを指示する状況で使用されます。はい、これらの単語は部分的に交換可能ですが、それらの意味には、置換を不適切にする可能性のある色合いがあります。

これらの同じ値について詳しく見てみましょう：

  • アドバイスは、より正式な動詞です。それは、1人がより高いレベルの誰か（職場の上司、当局の代表）または一部の専門家（医師、教師）から指示を受ける状況で使用されます。
  • 推奨はあまり正式ではない動詞です。この単語はより個人的なものです。個人的な経験に基づいたアドバイスをするときに使用します。
  • サジェストは、3つすべての中で最も正式な動詞です。それはいくつかのアイデア、意見などになると使用されます。

アドバイス、推奨、提案の使用方法


意味やパラメータ、単語の使用例を研究すればするほど、違いに気付き始めます。3つの動詞の特定のケースでは、ルールは次のとおりです。

何かを提案する


この形式の動詞は、アクションを提案する人が自身に関与している場合に使用されます。多くの場合、gerundと組み合わせて使用​​されます。
例：

彼女は昼食に行くことを提案しました。
この例では、夕食に出かけることを提案した人が明らかに出かけようとしています。

誰かが何かをする（すべき）ことを提案する（すべき）/誰かに何かをするように提案する


この場合、アクションを提供する人は、アクションへの参加を意味しません。
例：

彼は私が行ってこの仕事に応募すべきだと提案しました。
ただし、そのような関与が可能な状況があります。
彼は私たち全員が昼食に行くべきだと提案しました。
重要なポイント：「推奨」という動詞は「推奨」と同じ規則に従います。

何かを推奨する/誰かを誰かに推奨する/何かを推奨する/誰かを誰かに推奨する


そのような場合、「推奨/提案/アドバイス+名詞」（動詞+名詞）という特別なパターンがあります。提案の相手を文章で説明したい場合は、「名詞オブジェクト+から+人」というパターンを使用できます。
例：

彼女は歯科医を私に勧めました。
英語学習者は、動詞自体の前に「to + person」の部分を挿入することがよくあります。その理由は単純です-多くの言語では、そのようなフレーズはこのように構成されています。しかし、英語ではこれは間違いです。
False：彼女は私に彼女の歯科医を勧めました。

提案+その+主語+基本動詞/動詞+その+主語+基本動詞


おそらく、このトピック全体で最も難しい部分です。文の動詞の後に、いわゆる名詞句（1つの単語では不十分な場合は主語の説明）が来ると、仮定法の気分が得られる構造。

正確なパターンは、「提案の動詞+ that + subject + base verb」のようになります。
例：

私の同僚は、オフィスからタクシーで家に帰ることを勧めました。（同僚は彼女がオフィスから家までタクシーに乗ることを勧めました-そうするのがいいでしょう）。

営業担当者は、このオファーを保留にすることを提案しました。
