Microsoft Online认证-现场说明

微软俄罗斯已经在线翻译了一篇关于考试测试流程的文章。

在过去的一年中，数百名（即使不是数千名）正在准备和通过考试的人已经通过了我。他们中的许多人甚至在Covid-19之前以及在认证中心关闭之后都在线上交了文件，当时只有在线指导的考试是唯一的选择，而且流程只会增加。另外，我本人在过去一年中通过在线督导考试通过了8个证书，以便能够帮助我们的工程师进行认证准备。

结果，我有了一些想分享的经验。
本文的结构如下：副标题中引用了Microsoft文章，下面是我的评论和示例。

做系统测试


我想补充一点，这是必要的进行不仅是一个系统的试验，也可做它的地方，并在同一时间，你打算把它。

  • 否则，当一个房间有足够的wifi带宽，而坐下的房间没有足够的wifi带宽时，您可能会遇到问题。还有另一个例子，在中国，一个女孩在家中检查互联网，然后去办公室取了，因为 中国的办公室已经开张了，它在等待一个惊喜：没有足够的渠道（要么是中国大烟花汇演，管理员对网络有什么感觉，要么供应商在开玩笑）
  • 同时，最好在进行检查的同时进行检查，因为 碰巧在早上或晚上，一切都很好，而在午餐时间，当您要摄取时，频道已经下垂了
  • 我们有些情况下人们无法开始测试，因为 系统测试显示检查连接速度时出错。同时，考生一侧的网络没有问题：在独立测试中，双向都有50-60兆位。问题出在认证中心一边。结果，我们绕过了几天，通过VPN连接到另一个国家的办公室几天，然后Pearson Vue对其进行了修复。
  • 最好在注册考试之前参加考试，因为 您可以先花钱，然后再意识到自己有问题。一位工程师首先为我们签名，然后我意识到这些测试只能在Win / Mac下运行，但可以在Linux上运行，办公室关闭，BIOS受密码保护，您无法自己进行双重启动，也无法从USB闪存驱动器加载。通常，最好事先阅读要求。

“确保您的网络摄像头正常工作”和“尽可能使用有线连接。”


遵守第二段可能会导致第一段出现问题。但总的来说，我支持。这应该得到保护，这不仅是因为我们中的一些人密封了相机或禁止了相机驱动程序，还因为有一个有趣的案例。这个人决定不连接到wifi，而是通过电线连接，以获得更可预测的连接质量。但是，当被要求通过摄像机显示房间时，该人拿起笔记本电脑，电线消失了。我不得不重新连接，再次排队，为工作场所，我自己，我的文件等拍张照片。一般情况下，需要15分钟，并且会更加紧张。

如果可能，请考虑使用更大的屏幕。


我支持屏幕，但不理解有关实验室工作的评论。
由于Covid-19和Azure的能力不足，实际任务已从考试中删除，而没有特别声明。考试因此而蒙受了极大的损失，但是那里有些东西。

此外，考试变得越来越困难，因为研究最多的是使用免费试用版或msdn订阅，例如把钱花在“现收现付”或其他有偿资源上。

是的-这恰恰是缺乏能力。让我们称锹为锹：如果在msdn /免费试用版订阅中，不可能创建数据库或虚拟机（这可能是过去10年）-这是缺乏容量。由于AWS / GCP没有引入此类限制，因此官方立场（将能力重新分配给与covid-19挣扎的人们）。

警告您的家庭即将进行的考试


我想另外画最难的项目。这是必要的，不仅警告，但也投降之前提醒，因为他们可以从黑夜中忘记。就我个人而言，我在外门上悬挂了一张纸，上面刻着“考试正在进行中”的字样，并要求我的妻子坐在外面的椅子上后门（我可以出去，但即使是最健忘但仍然足够的人，这样的路障也显然会引发问题）。

不仅值得通知，而且还要求您的家庭不要发出噪音，因为对于过多的噪音，即使房间里没有人，您也很可能被取消考试（由于门外的噪音，我收到了2条警告，并说不会再进行第三次考试，但是考试将被取消。我必须在不偏离课程的前提下，通过封闭考试门大声地向家庭传达了问题的实质）。

要求不要打开洗衣机，不要洗碗，不要吸尘，不要与您最喜欢的婆婆在扬声器上谈论阴谋论，请孩子不要在走廊上追赶。检查您是否未下订单交货，也没有在炉子上留下任何东西。

听起来很荒谬，但这远非一个完整的清单，无论是我还是我所指导的人都遇到过。

删除所有不必要的


有很多有趣的事情要讲。理想情况下，与您保持一臂之力，除了电脑外什么都没有。

  • 我们的一位租房者决定预订一间会议室进行考试，笔记上留有一个永久性标记。别无选择-我不得不将其擦拭10分钟。
  • 另一位同事带了一间会议室，该会议室配备了可通过Internet进行大型会议的设备，大约15分钟后，他被迫断开所有电线，以参加考试。考试后，花了更多时间将所有这些重新连接起来，因为 当然，此人不记得压力条件中包含的内容。

保持手机方便


甚至没有必要在上面有SIM卡或钱。足够的wifi。手机用于在您要拍摄的地方拍摄自己的照片（拍摄自拍），拍摄文档的照片以及从不同角度拍摄的4张工作场所的照片。大约需要5-10分钟。

携带国家签发的身份证


这里打开了很多有趣的东西。

当所有考试中心都关闭时，事实证明，我们的许多“学生”根本没有文件，他们的名字和姓氏用拉丁字母（没有驾照或护照）书写。要让人们获得护照（这意味着要缴纳州税）或驾驶执照以通过“免费”考试，对我们来说太过分了。

有一个人似乎有证件，但该人通过了证件（在某些国家，用拉丁字母写在内部护照中），以便重新签发，现在，他仅拥有相关证书。同时，由于隔离，护照柜台几乎停止了工作，有些人至少无法得到带照片的某种文件。

官方要求说，注册期间帐户的名称必须与文档中的拼写一致。这似乎是一个平庸的要求，但是即使在这里，我们也学到了很多。

  • 在这里，我们了解到例如在俄罗斯，我们最近更改了Andrei的音译（从Andrey到Andrei，从VITALIY到VITALII）。该人已拥有新护照，并且在注册时已拥有旧拼写。
  • 我还了解到，实际上各种字母上的变音符号和其他圆点都可以忽略，如果使用特殊字符，则类似于仅在英语字母中拼写名称。
  • 有人更改了姓氏，而法定名称的更改也不那么容易。
  • 有人搬到了另一个国家/地区，拉丁字母中的名字和姓氏的拼写发生了变化，并且法律名称（如上文所述）发生了变化，您需要撰写一个完整的支持请求。

官方说明说要等待10-15分钟。实际上，我们的案件等待时间为2-3分钟，30分钟和1.5小时（我重复：90分钟或一个半小时）。但是，您甚至都不知道队列中的哪个人，以及他们何时会回答您。官方说明说，您需要关闭除测试系统以外的所有程序，但实际上等待监考员10-30分钟实在无聊，而且我们的许多“学生”都阅读了新闻，在考试前聊天了等等。没有任何后果。测试程序仍然关闭所有窗口，使考试期间的所有聊天和应用程序保持静音。

我们有一个人，经过一个小时的等待，他们受不了了，因此关闭了测试。显然，这些都是大样本中的孤立案例，但事实也是如此。这里的主要内容是创建支持请求让您有机会重新分配考试，而不仅仅是您的免费凭证被烧光了。

此外，如果您不是来自美国/英国或对AM / PM系统有信心，我不建议您指定12到13的考试。看起来那么复杂？！但是，没有，有几个人将考试安排在12:30 AM（午夜半点），并在下午12:30到达，他们对为什么无法参加考试感到非常惊讶。会很有趣，但是有几个这样的人。我不知道他们是怎么做到的，因为我选择广告位的时间以24小时格式显示，而不是AM / PM。

根据观察，现在的监理员比较软。我认为这是由于在线考试数量增加，结果，他们根本没有时间。通常我什至没有被要求出示房间，而是立即开始考试。最快的选择是他们甚至没有在聊天中给我写“嗨”，而是立即开始测试。我很惊讶，但最后我对这些手续也不感兴趣。但是，我们建议工程师在必须展示房间并准备满足监理员的其他要求时，为最严格的版本做准备。

不要碰你的脸


在此重要的是不要触摸脸部。重要的是您的眼睛和嘴唇要可见，以确保不会作弊。您可以支持下巴，并且不会提出任何问题。但是，如果您将手放在嘴唇上，使它们与相机合为一体，则会要求您遵守规则。

我们的一位租客喜欢坐在椅子下面，最后我收到了一条评论，因为 视频中只有他的额头和头发可见。第二个观察结果可能会导致失格。
