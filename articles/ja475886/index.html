Amazon AIは、ユーザーのわいせつなコンテンツへの取り組みを容易にします

数千社がAmazonのRekognitionマシンビジョンサービスを使用して、ユーザーがアップロードしたわいせつな画像や動画を検索しています




Amazonの物議を醸すRekognitionテクノロジーは、すでに食品のウェブサイトから陰茎の画像を削除するために使用されています。少なくともこれはその使用例の1つです。ある時点で、ロンドンを拠点とする食品配達サービスDeliverooがコンテンツ管理の問題に遭遇しました。食品に問題がある場合、Deliverooの顧客は、苦情とともに食品の写真を送信します。そして、彼らはしばしば彼らの性器で写真爆撃をします。または、食べ物のわいせつな写真を撮ります。はい、真剣に。

そのため、Deliverooの従業員は常にそのようなコンテンツを扱いたいとは限らないことがわかりました。したがって、同社はRekognitionを使用してわいせつな写真を認識し、人が見る前にぼかしたり削除したりします。

Problem Deliverooは、次第に複雑になる問題のやや奇妙な側面を示します。いずれにせよ、多くのオンライン企業はユーザーが作成したコンテンツに依存しています。近年、人間の本当のダークサイドのこのコンテンツへの浸透にますます直面しています。ユーザーが作成した偽のニュース、暴力、ジフフェイク、いじめ、攻撃的なレトリック、その他の有毒なコンテンツなどの不快なコンテンツにサイトが直面しているため、コンテンツの管理が優先事項となっています。あなたがFacebookの場合、この問題を解決するには、独自のAIを開発するか、モデレーターの軍隊を雇うか、その両方を行うことができます。ただし、リソースが少ない中小企業には、この機能がないことがよくあります。ここで、Amazonのコンテンツ管理サービスが役立ちます。

このサービスは、Amazon Web Servicesが提供するRekognitionコンピュータービジョンサービスパッケージの一部です。彼が同社が米国の移住サービスのために顔認識サービスを提供することに同意したという事実のためにマスコミで多くの批判されました。 RekognitionのWebサイトでは、追跡のためのサービスの使用の他の例を見つけることができます。たとえば、ビデオでさまざまな角度から撮影された車の番号を認識したり、カメラの記録を使用して人物の経路を追跡したりできます。

おそらく、コンピュータービジョンサービスのよりポジティブなイメージを求めて、Amazonはまず暴力やわいせつをなくすためにRekognitionを使用してユーザーコンテンツを監視することについて話しました。このサービスでは、サイトにアップロードされた画像や動画の安全でないまたは不快なコンテンツを認識することができます。

そして、このビジネスは成長しています。 「ユーザー生成コンテンツの役割は年々爆発的に高まっています。今日、ソーシャルネットワークで毎日2〜3枚の写真を友人や親戚と共有しています」とAmazonのアマゾンバイスプレジデントであるSwami Shivasubramanyanは語っています。 Shivasubramanyan氏は、Amazonが2017年に顧客の要求に応えてコンテンツ管理サービスの提供を開始したと述べています。

企業は、ダウンロード可能な画像を研究するために人々を雇う代わりに、Rekognitionの代金を支払うかもしれません。 AWSの他のサービスと同様に、従量課金モデルで機能し、そのコストはニューラルネットワークで処理される画像の数に依存します。

当然のことながら、デートサービスはコンテンツ管理の最初のユーザーの1人でした。ユーザープロファイルにアップロードされたセルフィーをすばやく処理する必要があります。 Amazonは、出会い系サイトCoffee Meets BagelとShaadiがこの目的のためだけにこのサービスを使用していると言います-人々が出会い系サイトを作成するのを助けるポルトガルのSoulサイトのように。

AIはヌードを探すだけではありません。ニューラルネットワークは、武器や暴力の画像、または一般的に不快な画像を含む疑わしいコンテンツを認識するように訓練されました。こちらが分類メニューです認識サイト：

露骨なヌード：

  • 裸の体;
  • 裸の男性の体のグラフィック画像。
  • 裸の女性の身体のグラフィック画像。
  • 性行為;
  • ヌードや性行為のデモンストレーション
  • 大人のためのおもちゃ。

不審なコンテンツ：

  • 女性の水着または下着。
  • 男性用水泳パンツまたは下着。
  • 部分的に裸の体;
  • 率直な服。

暴力を示すコンテンツ：

  • 暴力や血のグラフィック表現;
  • 身体的暴力;
  • 銃暴力;
  • 兵器;
  • 自分に傷害を与える。

不快な視覚的コンテンツ：

  • 衰弱した体;
  • 死体;
  • ぶら下げ。

使い方


AWSのすべてと同様に、Rekognitionはクラウドで実行されます。会社はサービスにどのような画像を見つける必要があるかを伝えることができます。次に、ユーザーから受け取った写真とビデオをフィードします-多くの場合、これらはとにかくAWSサーバーに保存できます。

ニューラルネットワークは画像を処理し、このコンテンツを検索して、不快な可能性のあるコンテンツを記録します。ニューラルネットワークは、発行されたラベルへの信頼のパーセンテージとともに、画像の内容を説明するメタデータを生成します。



このデータは次のようになります。このデータはすでにクライアント側プログラムによって処理されており、プログラムされたビジネスルールに基づいて、処理された画像をどう処理するかを決定します。自動的に削除したり、スキップしたり、一部をぼかしたり、レビューのためにモデレーターに送信したりできます。

深層画像処理ニューラルネットワークには多くの層があります。それぞれが画像のさまざまな側面を表すデータを評価し、計算を実行して、結果を次のレイヤーに送信します。まず、ネットワークは基本的なフォームや画像内の人物の存在などの低レベルの情報を処理します。

「それから、彼女は一貫してデータをますます洗練させ、次のレイヤーはより具体的になります。」とShivasubramanyanは説明します。徐々に、レイヤーごとに、ニューラルネットワークは、画像のコンテンツを確実に決定します。

AWSのAI副社長Matt Woodは、彼のチームは複数のセットからの何百万ものプライベートおよびパブリックにアクセス可能な画像のコンピュータービジョンモデルのトレーニングを行っていると述べています。アマゾンはユーザーから受け取った画像をこの目的で使用しないと彼は言います。

フレームごと


最大のRekognitionクライアントの一部は、このサービスを使用してユーザー生成コンテンツを管理していません。 Amazonによると、巨大なデジタルビデオライブラリを所有する大手メディア企業は、これらのビデオの各フレームのコンテンツを知りたいとのことです。 Rekognitionニューラルネットワークは、ビデオの1秒ごとを処理し、メタデータを使用してビデオを記述し、潜在的に危険な画像にフラグを立てることができます。

「機械学習がうまく機能するタスクの1つは、ビデオまたは画像にアクセスし、追加のコンテキストを提供することです」とWood氏は語ります。 「このビデオでは、女性が犬と一緒に湖の岸を歩いている」または「部分的に服を着た男性が描かれている」と言うことができます。このモードでは、ニューラルネットワークは危険な、有毒な、またはわいせつなコンテンツを画像内で高精度で認識することができると彼は言います。

それでも、コンピュータービジョンのこの領域はまだその成熟に達していません。科学者たちは、ニューラルネットワークアルゴリズムを最適化して画像をさらに正確かつ詳細に認識できるようにする新しい方法をまだ発見しています。 「まだ利益が減少している状態には達していません」とWood氏は言います。

Shivasubramanyanが私に言ったのは、コンピュータービジョンに取り組んでいるチームが先月だけ誤検出（画像が誤って危険だと見なされたとき）の数を68％、誤検出の数を36％削減したことです。 「これらのAPIの精度を向上させる機会があります」と彼は言います。

精度に加えて、顧客は画像のより詳細な分類を求めています。 AWSのウェブサイトによると、このサービスは安全でない画像のメインカテゴリと1つのサブカテゴリのみを提供します。したがって、たとえば、システムは画像にヌードがメインカテゴリとして含まれ、性的行為がサブカテゴリとして含まれることを示すことができます。 3番目のサブカテゴリには、性行為のタイプの分類が含まれる場合があります。

「これまでのところ、マシンは事実になりやすく、文字通り機能します-「これはそこに示されている」と教えてくれます」とPietro Peronaは言います、AWSのアドバイザーであるCaltechのコンピューティングおよびニューラルシステムの教授。 -しかし、科学者たちはこの枠組みを超えて、そこに描かれていることだけでなく、これらの人々が何が起こっていると考えているかについても報告したいと考えています。結果として、この領域はこの方向に発展したいと考えています-写真に示されているもののリストを提供するだけではありません。」

そして、そのような微妙な違いはコンテンツの管理にとって重要です。画像に不快感を与える可能性のあるコンテンツが含まれているかどうかは、そこに描かれている人々の意図に依存する場合があります。

「安全でない」および「不快な」画像のまさにその定義でさえ、かなりぼやけています。それらは時間の経過とともに変化する可能性があり、地域によって異なる可能性があります。そして文脈がすべてだとペロンは説明する。暴力の画像が良い例です。

「シリアでの本当の暴力など、暴力は1つの状況では受け入れられない場合があります」とペローナは言います。「サッカーの試合やタランティーノ映画のシーンなど、別の状況では受け入れられます。」

他のAWSサービスと同様に、Amazonはコンテンツ管理ツールを他の人に販売するだけではなく、それ自体が顧客です。同社は、このサービスを使用して、ユーザーが作成したコンテンツを、店舗のレビューに添付される画像とビデオに分類するとしている。
