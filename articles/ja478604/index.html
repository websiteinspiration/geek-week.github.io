労働者からPHPプログラマーまで。異常な開発経歴



本日、GeekBrainsの学生Leonid Khodyrev（レオニドジレフ）、彼は24歳です。彼のITへの道のりは、以前に公開されたストーリーとは異なります。レオニードは軍の直後にPHPをすぐに研究し始め、最終的に彼が良い仕事を見つけるのに役立ちました。

私のキャリアストーリーはおそらく他のすべてのものとは異なります。私はIT担当者のキャリアストーリーを読みます。ほとんどの場合、人は自信を持って前進し、目標を達成するためにすべてまたはほとんどすべてを行います。私はそうではありません-自分が誰になりたいのか全く知りませんでしたし、将来の計画を立てませんでした。多かれ少なかれ真剣に、私は軍隊から戻ったときにこれについて考え始めました。しかし、順番に見ていきましょう。



ウェイター、ローダー、弁護士補佐としてキャリアをスタート


私は早めに仕事を始め、最初の「専門」はチラシの配布でした。彼らは私に紙切れを与え、私はすべてを配りましたが、お金は受け取りませんでした。それにもかかわらず、この経験は役に立ちました。私は自分が直面する可能性があることを理解し始めました。

その後、ローダー、ウェイターとして働き、フィールドイベントでさまざまなタスクを実行し、これを研究と組み合わせました。彼は大学で学び、同時にサイト作成のトピックを習得しました。私は人気のあるCMSでシンプルなウェブサイトを作成しました。しかし、それでも私は流れに行きました、私は人生で何が必要かを本当に考えていませんでした。

それで、彼らは私を陸軍に連れて行ってくれました。おかげで私は国全体を見ました。すでに軍隊にいる私は、将来何をしたいのか考えました。私はサイトでの経験を思い出して、この分野で働くことが面白いと思いました。そして私は軍隊で遠隔教育の可能性を探し始めました。GeekBrainsのWeb開発コースが私の目を引きました。私が覚えている限り、私はキーとなる「プログラミング」または「プログラミングを教える」を検索に駆り立て、コースのウェブサイトを見て、リクエストを残しました。マネージャーが電話をかけてきて、私はすべてについて整然と尋ね始めました。

もちろん、陸軍で勉強することはできませんし、特別な手段もなかったので、将来のために勉強を延期しました。

ITの成果


退院後、お金は追加されませんでした。トレーニングを始めるために、ウェイターの前の仕事に戻らなければなりませんでした。給料をもらって、コースを買って始めました。残念ながら、フルタイムのウェイターとして働くことは多くの時間がかかることが明らかになりました、それは勉強するのに十分ではありませんでした。解決策がすぐに見つかりました-彼は弁護士の事務処理を手伝い始め、「ハイシーズン」にはウェイターとして働きに行きました。

残念ながら勉強は大変だったので3回やめました。しかし、私はこれが継続できないこと、ウェイターが良いことを認識しましたが、ITの方がはるかに重要です。そのため、仕事でタイムアウトになり、完全に勉強に専念しました。すぐに、私はそれが好きではなく、本当に好きだと気づきました。少し後、サイトを作成する最初の注文が表示され始めたので、喜びに加えて、このレッスンはより多くのお金をもたらし始めました。どういうわけか私は自分が好きなことをしていると自分自身に気づき、それに対して私も支払いを受けました！この瞬間、自分の将来を決めました。

ちなみに、実際のトレーニングでは、コンテンツ管理システムという非常に深刻なプロジェクトを自分で開発しました。私はそれを書いただけでなく、いくつかのサイトを接続することができました。プロジェクトの詳細については、こちらをご覧ください。



つまり、このプロジェクトはユーザーにとって便利なプラットフォームであり、ビジネスに必要なさまざまなサービスと統合することで簡単に拡張できます。対象読者-起業家とウェブマスター。彼らのために、私はあなたが商品のカテゴリー、商品自体、それらの特性、プロセス注文を管理することを可能にする「ショップ」拡張を書きました。

これは私の真剣なプロジェクトであり、それほど深刻ではない技術を使用して開発されました。もちろん、評価するときは、トレーニング中に開発したことを忘れないでください。

新しい事務


トレーニング中にウェブサイトの開発を注文したことはすでに述べました。そして、私は本当にそれが好きでした-実際、私は本当にオフィスで働きたくありませんでした。しかし、キャリアのある時点でほとんどの開発者が正式な仕事に就くため、チームとしての経験が必要であることに気付き始めました。それをすることにしました。

今覚えているように、月曜日の朝にhh.ruを開き、履歴書をアップロードし、証明書を追加して、アカウントを公開しました。それから彼は私の家（そして私はモスクワに住んでいます）に最も近い雇用主を探し、履歴書を送り始めました。

文字通り1時間後、私が興味を持っていた会社が答えました。同じ日に面接を受けるように勧められました。 「ストレステスト」など不思議なことはなかったと思いますが、まだ少し緊張していました。彼らは知識のレベル、仕事の経験、そして一般的にはすべてについて、私に好意的に質問し始めました。

思い通りではない質問に答えましたが、彼らは私を連れて行ってくれました。確かに、彼らは私たちを心配させました-最初に彼らは電話をかけ直すだろうと報告しました。実際、これは通常、候補者を採用したくない場合に回答されます。しかし、私は無駄に心配していました-大切な鐘が数時間後に鳴りました。翌日、すべての書類を集めて出勤しました。

代理店がホテルや送迎などを予約できるオンライン予約システムをサポートしたため、私はすぐに投獄されました。私はすべてが適切に機能することを確認し、機能を変更し、さまざまな機能を追加します（バグがない場合、これがない場合はどこにありますか）。

すでに行われたことの例：

  • 予約レポートモジュール。
  • 改善されたプラットフォームインターフェイス。
  • サービスプロバイダーとのデータベース同期。
  • ポイントシステム（プロモーションコード、ポイント）。
  • ワードプレスの統合。


ツールについては、主なものは次のとおりです。

  • レイアウト-html / css / js / jquery;
  • データベース-pgsql;
  • PHPフレームワークyii2で記述されたアプリケーション。
  • サードパーティのライブラリは、さまざまなライブラリを使用しています。


収入について言えば、以前よりもはるかに高いです。しかし、ここではすべてが相対的です。なぜなら、トレーニング中に月に約15,000ルーブルを獲得したからです。サイトを必要とする知人からの注文だけだったので、何もないこともありました。

労働条件をどちらかと比較することは何もありません。労働条件や給仕として私が持っていた条件よりもはるかに優れていることは明らかです。通勤には25分しかかかりません。これも楽しいことです。結局のところ、首都の多くの居住者はより多くの時間を費やしています。モスクワと言えば、両親と住んでいたゼレノグラードから首都に引っ越しました。彼は彼が勉強する間、注文するサイトの作成に従事していたときに首都に移動しました。ここではすべてが好きです。移動するつもりはありませんが、世界を見るつもりです。

次は何ですか？


私は仕事から喜びを得るように、私の開発パスを続けるつもりです-これは私が好きなものです。また、以前は難しかったと思っていた作業もまったく問題ありません。したがって、私はより大きなプロジェクトに取り組み、すべてが判明したときに喜びます。

私が勉強する必要があるトピックのいくつかは、自分で習得するのが難しい場合があるので、勉強を続けています。教師は、主要な学習コースが終了した後でも、物事を理解するのに役立ちます。

近い将来、新しいプログラミング言語を学び、英語を学びたいです。

はじめてのヒント


かつてIT関係者のキャリアについての記事を読んだり、「恐れる必要はない」などと多くの方に言われています。もちろん、これは正しいですが、恐れることはありません-戦いの半分。主なことはあなたが何が好きかを正確に知ることです。たとえば、ネットワークからのレッスンを使用して、言語の基本を学び、スクリプトまたは最も単純なアプリケーションを作成してください。あなたがそれを好きなら、それはそれから始める時が来ました。

そして別のヒント-横になった石にならないでください。その下では、ご存知のように、水は流れません。どうして？最近、仲間の生徒たちと一緒に物事がどうなっているかを学びました。結局のところ、誰もが仕事を得たわけではありません。私の会社には優れたスペシャリストが必要なため、私は数人の人を私の面接に招待しました。でも結局、面接に来る人は誰もいませんでしたが、その前にたくさんの質問を受けました。

これはやる価値がありません-就職活動の気分になったら、一貫してください。十分な経験がないように思われる場合でも、いくつかの面接を試してみてください。多くの企業は、家にスペシャリストを養成するために新規参入者を受け入れます。面接に失敗した場合、貴重な経験を積むことができ、雇用プロセスが内部からどのように見えるかを知ることができます。
