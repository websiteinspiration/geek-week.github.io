2. Azure Botでスマートボットを作成する


テキスト、写真、音声を使用したチャットを介したアプリケーションとのユーザー対話は、ボットを使用して実装できます。これは、単純な「質問と回答」ダイアログ、または既存のサービスと十分に統合されたパターンマッチング、状態追跡、人工知能の方法を使用して、人々が「スマート」な方法でサービスと対話できるようにする複雑なボットです。QnA MakerとLUISを統合してスマートチャットボットを作成する方法を学びます。

3. Azure Cognitive Language Servicesを使用したテキスト分析


Cognitive Language Servicesを使用してテキストを分析し、意図を決定し、成人向けのトピックを発見し、自然言語でクエリを処理する方法を学びます。

コースモジュール：

  • Azure Content Moderatorを使用したテキストの分類と管理。
  • Intelligent Speech Recognition Service（LUIS）を使用してアプリケーションに対話ロジックを追加します。
  • テキスト分析APIを使用して、テキストの対話の調性を追跡する。

4. AIベースのボーダーデバイスエンジニアコース

5. Azure Cognitive Vision Servicesによる画像処理と分類


Microsoft Cognitive Servicesは、アプリケーションでコンピュータービジョンを使用するための組み込み機能を提供します。Cognitive Vision Servicesを使用して、顔を検出し、画像にタグを付けて分類し、オブジェクトを識別する方法を学びます。

コースモジュール：

  • Azure Cognitive ServicesのコンピュータービジョンAPIを使用して顔と感情を識別します。
  • コンピュータビジョンサービスを使用した画像処理。
  • ユーザーの視覚認識サービスを使用した画像の分類。
