Verstecke einen Teil der Telefonnummer

Stellen Sie sich vor, Sie müssen einen Teil der Nummer unter den Sternchen verstecken. Das Ersetzen von +79999999999 durch +799 **** 9999 ist nicht schwierig, aber stellen Sie sich jetzt vor, dass es mehr als eine Zahlenmaske gibt, aber viel mehr, diese Zahlen sind russisch, kanadisch oder eine andere. In dieser Funktion habe ich versucht, so viele Zahlen wie möglich zu erfassen.

Ist die Zeichenfolge die Telefonnummer:

/ ^ (\ (? \ +? \ d {1,2} \) ?? \ (? \ d {1,3} \) ?? \ d + \ - ?? d + \ - ?? \ d +) $ /.

Ersatzfunktion:

function hideNumber(string, replaceTo = '*', elemsHide = 4, sliceFromback = 4) {
    result = string.match(/^(\(?\+?\d{1,2}\)? ?\(?\d{1,3}\)? ?\d+\-? ?\d+\-? ?\d+)$/);
    if (result !== null) {
          //    n    x
        const regex = new RegExp(`((\\(?\\ ?\\-?\\d\\ ?\\-?\\)?){${elemsHide}})((\\ ?\\-?\\d\\ ?\\-?){${sliceFromback}}$)`, 'gm');

        let m;
        while ((m = regex.exec(string)) !== null) {
            if (m.index === regex.lastIndex) {
                regex.lastIndex++;
            }

            const forRex = m[1];
            const str = m[1].replace(/(\d)/gm, replaceTo);
            const lasts = m[3];
            const full = string;
            const noBack = full.slice(0, -lasts.length).slice(0, -forRex.length);
            const out = noBack+''+str+''+lasts;
            return out;
        }
        return string;
    } else {
        return string;
    }
}

Im Kommentar mwizard Implementierung ohne Regelmäßigkeit hinzugefügt:

type Context = {
    readonly offset: number;
    readonly filtered: string;
}

function hidePhone(phone: string): string {
    return Array.from(phone).reduceRight((ctx: Context, char: string): Context => {
        const isDigit = char >= '0' && char <= '9';
        const offset = ctx.offset + (isDigit ? 1 : 0);
        const filteredChar = isDigit ? (offset >= 4 && offset < 8) ? '*' : char : char;
        const filtered = filteredChar + ctx.filtered;
        return { offset, filtered };
    }, { offset: -1, filtered: '' }).filtered;
}
