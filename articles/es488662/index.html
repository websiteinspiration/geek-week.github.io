Extensiones de navegador requeridas por cada desarrollador web

Los navegadores modernos no son solo excelentes aplicaciones de navegación web.

Los navegadores también son plataformas que ofrecen a los desarrolladores web una tonelada de herramientas útiles para ayudarlos a crear sitios excelentes. A disposición del desarrollador, además de las herramientas integradas, hay una gran cantidad de extensiones de navegador. El autor del artículo, cuya traducción publicamos hoy, ha seleccionado las mejores extensiones, en su opinión, que están diseñadas para ayudar a los programadores a resolver una variedad de tareas que enfrentan. Estas son principalmente extensiones para Google Chrome.



Herramientas de desarrollo para React / Redux y Vue.js


React y Vue.js son herramientas de desarrollo front-end bien conocidas. Para trabajar con ellos de la manera más eficiente posible, vale la pena instalar las extensiones de navegador apropiadas.

La extensión React Developer Tools mejora las capacidades de las herramientas de desarrollador. Utilizándolo, puede examinar Reaccionar componentes, propiedades pasadas a componentes y el estado de los componentes. Puede aportar beneficios reales a un desarrollador de React.


Reaccionar herramientas de desarrollador

Si encuentra algún problema mientras trabaja con el estado de la aplicación, la extensión Redux DevTools puede ayudarlo . Se puede usar para trabajar con estados que están organizados de diferentes maneras; funciona especialmente bien cuando se usa con Redux. Por cierto, aquí hay un consejo para ti. Instale esta extensión y visite un sitio como airbnb.com. Utiliza React / Redux, por lo que, con la ayuda de Redux DevTools, podrá descubrir cómo funciona la gestión del estado de las páginas de este sitio. Para mí, es muy informativo.

Vue.js devtools es lo mismo que React Developer Tools, pero para Vue.js. Esta extensión es muy conveniente para depurar componentes de Vue y es útil para todos los que usan este marco.


Vue.js devtools

Jsonview


¿Alguna vez te has dado cuenta de que en los navegadores de otros programadores los datos JSON se ven claros y precisos, pero no en los tuyos? Quizás el punto es que otros usan la extensión JSONView . Esta extensión formatea de forma rápida y precisa los datos JSON. JSONView es definitivamente una de mis extensiones de navegador favoritas.


Jsonview

Sniffer de biblioteca


Library Sniffer es una gran herramienta para cualquiera que quiera descubrir rápidamente qué herramientas, bibliotecas o marcos se usaron para crear las páginas que visualizan. Por ejemplo, con la ayuda de Library Sniffer, puede averiguar con qué se creó un sitio: WordPress, Drupal, React, Vue.js, AngularJS ...


Sniffer de biblioteca

CSS Scan 2.0


CSS Scan 2.0 es una extensión útil que permite a un desarrollador explorar y copiar CSS simplemente pasando el cursor sobre los elementos de la página. Es cierto que tiene un menos. El hecho es que para trabajar con él necesitas comprar una licencia.


CSS Scan 2.0

Esta extensión tiene una alternativa gratuita: CSS Peeper . Está diseñado principalmente para diseñadores, y no para aquellos que escriben código.

desarrollador web


La extensión Web Developer le permite equipar su navegador con un panel que contiene muchas herramientas diferentes. Con su ayuda, por ejemplo, puede desactivar JavaScript en las páginas, controlar la ubicación de la página y trabajar con imágenes. Esta extensión está disponible para Chrome y Firefox. 


desarrollador web

Whatfont


WhatFont es una extensión bastante pequeña que le permite identificar las fuentes utilizadas en las páginas web. Para saber para qué fuente está diseñado el fragmento de texto, es suficiente habilitar la extensión y mover el puntero del mouse sobre este fragmento. Trabajar con él, como puede ver, es extremadamente simple. Al igual que la extensión anterior, puede ser utilizado por aquellos que trabajan en Chrome y aquellos que trabajan en Firefox.


Whatfont

Colorzilla


La extensión ColorZilla se puede usar tanto en Chrome como en Firefox. Le da al desarrollador una herramienta para tomar pruebas de color, una herramienta para elegir colores, un generador de gradiente y otras herramientas útiles para trabajar con color. Por ejemplo, puede usarlo para averiguar qué color tiene un píxel en una página web. Almacena el historial de trabajo con color y contiene una herramienta para ver paletas.


Colorzilla

Espectro


La accesibilidad al contenido es un aspecto importante del desarrollo web moderno. El hecho es que muchas personas tienen una capacidad limitada para interactuar con páginas web. Una de esas limitaciones es la deficiencia de la visión del color (CVD). Afecta la capacidad de una persona para distinguir ciertos colores. La extensión Spectrum le permite investigar sitios para determinar su idoneidad para personas con diversos tipos de ECV.


Espectro

Paleta del sitio


La extensión de la paleta del sitio le permite extraer los colores utilizados en el sitio del sitio y organizarlos como una paleta de colores. Admite varias formas de generar paletas, le permite emitir paletas emitiendo un enlace especial, admite la capacidad de imprimir paletas y puede convertirlas en archivos PDF. Y esto no se limita a la paleta del sitio.


Paleta del sitio

Toby


La extensión Toby está diseñada para organizar las pestañas del navegador de tal manera que en el curso del trabajo no tenga que mantener abiertas una gran cantidad de pestañas. Este es un tipo de panel de control de pestañas que es fácil de usar con el mouse. Esta extensión puede afectar positivamente la productividad del desarrollador web.


Toby

Talend API Tester


Talend API Tester es una gran extensión que te permite trabajar desde un navegador con varias API. Estas pueden ser API basadas en tecnologías REST, SOAP y HTTP. Esto es similar, por ejemplo, a Postman. La versión gratuita de la extensión admite la ejecución de todo tipo de solicitudes HTTP, intercambio seguro de datos, autenticación.


Talend API Tester

Captura de pantalla de página completa


¿Alguna vez ha querido tomar una captura de pantalla completa de la página que está viendo sin movimientos innecesarios? Si es así, la extensión de captura de pantalla de página completa puede ser útil . Con él, puede obtener capturas de pantalla de todo el contenido de las páginas en formato PDF o como una imagen. Incluso puede funcionar con elementos que admiten desplazamiento y con elementos iframe.


Captura de pantalla de página completa
