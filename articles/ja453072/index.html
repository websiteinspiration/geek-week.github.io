過去数か月間、いくつかのコースを受講し、Google Cloudと並行してProfessional Data Engineer試験の準備をしました。それから私は試験に行って合格しました。数週間後、スウェットシャツが届きましたが、証明書の方が速くなりました。

翻訳

Google Cloud Professional Data Engineer証明書を取得する理由


データは私たちを取り囲み、どこにでもあります。したがって、今日の専門家は、データを処理して使用できるシステムを作成する方法を知っている必要があります。また、Google Cloudはこれらのシステムを構築するためのインフラストラクチャを提供します。

Google Cloudを使用するスキルをすでに持っている場合、将来の雇用主またはクライアントにどのように示すことができますか？これを行うには2つの方法があります。プロジェクトのポートフォリオを持っているか、認証に合格していることです。

証明書は、特定のスキルを持っていること、および正式な確認を得るために努力したことを潜在的な顧客および雇用者に通知します。

これは試験の公式説明に記載されています。

Google Cloudプラットフォームでデータ処理システムと機械学習モデルを設計および構築する能力を示します。

まだ適切なスキルがない場合は、認定のためのトレーニング資料を学習するときに、Google Cloudを使用して最高レベルのデータ処理システムを作成する方法について必要なすべてを学びます。

Google Cloud Professional Data Engineer証明書が必要なのは誰ですか？


あなたはその数を見ました-クラウドテクノロジーの領域は成長しています、彼らは長い間私たちと一緒にいます。統計に慣れていない場合は、「クラウド」が増加していると信じてください。

すでにデータ処理または分析のスペシャリスト、機械学習エンジニアとして働いている場合、またはデータ処理業界に移行したい場合は、Google Cloud Professional Data Engineer認定資格が必要です。

クラウドテクノロジーを使用する機能は、データを扱うすべての専門家にとって必須の要件になっています。

処理、データ分析、機械学習の専門家になるには、証明書が必要ですか？


番号。

Google Cloudを使用して、証明書なしでデータ処理ソリューションを操作できます。

証明書は、既存のスキルを確認するための1つの方法にすぎません。

それはどれくらいしますか？


試験に合格するための費用は200ドルです。彼に失敗した場合は、再度支払う必要があります。

さらに、準備コースとプラットフォーム自体の使用にもお金をかける必要があります。

プラットフォームでの作業のコストは、Google Cloudサービスを使用するための料金です。あなたがそのアクティブなユーザーであれば、あなたはこれをよく知っています。初心者で、この記事で説明するトレーニング資料の学習を始めたばかりの場合は、Google Cloudアカウントを作成して必要なすべてを行うことができます。登録時にGoogleがアカウントにクレジットする$ 300を実行します。

文字通りすぐにコースの費用に行きます。

証明書はどのくらいの期間有効ですか？


2年。この期間が過ぎると、再度試験を受ける必要があります。

また、Google Cloudは常に進化しているため、認定要件も変わる可能性があります（これは私が記事を書き始めたときに発生しました）。

試験の準備には何が必要ですか？


専門家として認定するには、GCPを使用したソリューションの開発と管理において、3年以上の業界経験と1年以上の経験があることをお勧めします。

私にはこれがありませんでした。

対応する経験は、いずれの場合も約6か月でした。

このギャップを埋めるために、私はいくつかのトレーニングオンラインリソースを使用しました。

どのコースを受講しましたか？


あなたのケースが私のものに似ていて、推奨される要件を満たしていない場合は、自分のレベルを向上させるために、以下からいくつかのコースを取ることができます。

認定の準備に使用しました。それらは、通過順にリストされています。

それぞれについて、認定試験に合格するためのコスト、タイミング、および有用性を示しました。


試験前のスキルを向上させるために使用したクールなオンライン学習リソースのいくつかは次のとおりです：A Cloud Guru、Linux Academy、Coursera。

試験後


試験に合格すると、「合格」または「不合格」という結果が得られます。トレーニング試験では最低70％を目指して頑張ることをお勧めしますので、90％を目指しました。

試験に合格すると、メールでアクティベーションコードと公式のGoogle Cloud Professional Data Engineer証明書が届きます。おめでとう！

アクティベーションコードは、Google Cloud Professional Data Engineer専用のストアで使用できます。ここでは、Tシャツ、バックパック、スウェットシャツを用意しています（納品時には、入手できない場合があります）。私はスウェットシャツを選びました。

証明書を受け取ったら、自分のスキルを（公式に）証明して、最善の作業であるシステムの構築に戻ることができます。

2年後にお会いしましょう-再認定について。
