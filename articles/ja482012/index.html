Raspberry Piおよびiperf-スマートホームおよびIoTデバイス用の帯域幅テスター



以前のプロジェクト「Raspberry Piからのテレビ電話」の一環として、Iperfユーティリティを使用して機器を構成し、問題を解決しました。このプログラムは、ローカルネットワークを扱うスペシャリストの間で長く知られています。

このユーティリティを使用すると、ネットワーク帯域幅を分析するためのトラフィックを生成できます。

その助けを借りて、2つのデバイス（サーバーとクライアント）間の最大ネットワーク帯域幅を測定し、通信チャネルの負荷テストを実施することは非常に簡単です。
ホームネットワーク機器の増加に伴い、仕事での不具合（不具合）の探索・分析や、WiFi機器の最適配置など、かなりの需要があると思います。

私の例では、プログラムは、MajorDoMoホームオートメーションシステム（基本的なイメージ）がインストールされたRaspberry Piと、Windows（ラップトップ）、Android（スマートフォン、TV、メディアプレーヤー）を実行しているデバイスにインストールされました。

インストールとテスト、およびMajorDoMoホームオートメーションシステムへの統合の簡単な例について少し説明します。

MajorDoMoホームオートメーションサーバーとして機能し、Debian 9で実行されるRaspberry Piに、iperfユーティリティをインストールします。 

sudo apt-get install iperf

デフォルトでは、バージョン2.0.9がインストールされています。iperfには2つのオプションがあります。iperf2とiperf 3です。これらは互いに互換性がなく、3番目のバージョンをインストールするには、コマンドを指定する必要があります

sudo apt-get install iperf3

2番目のデバイスで、WindowsでPCまたはラップトップを使用する場合は、Jperfをインストールします（2.0と2.2の両方をインストール）。 Jperfは、Iperf用のグラフィカルJavaアドオンです。アドオンはクロスプラットフォームで、Windowsとnixシステムの両方で動作します。 Jperfは、チャネル帯域幅をリアルタイムでスケジュールします。インストールは不要で、JavaマシンをOSにインストールする必要があります。

イーサネットをRaspberryネットワークケーブルでホームルーターに接続し、ラップトップを家庭用WiFiネットワークに接続します。

サーバーモードのRaspberry Piでユーティリティを実行すると、クライアントはトラフィックを生成し、サーバーはそれを期待します。

iperf -s -t 3600

どこ-sは、サーバーモードで起動-t 3600 3600秒または1時間。
Windowsコンピューターで* .batファイルjperf.batを実行し、サーバーアドレスを入力し、デフォルトのポートのままにして、テスト時間を設定し、開始アイコンをクリックします。

サーバーではテキスト形式で、クライアントではグラフィックで情報を取得します



。Androidオペレーティングシステム（スマートフォン、TV、TVセットトップボックスなど）の場合、プレイマーケットからプログラムをインストールします：Magic iPerf。Magic iPerf。

RPiのIPアドレスとキーを入力し、スライダーを停止位置から開始位置に移動してテストを開始します。



コマンドのヘルプ（ユーティリティ起動キー）は、ターミナルまたはモバイルアプリケーションに入力して表示できます-ヘルプ（-h）。



さらに、MajorDoMoホームオートメーションシステム、基本構成、Raspberri Piのイメージを使用する場合。

システムが起動すると、スピーカーはIPアドレスを話します。これはかなり便利です。
また、サーバーにはホスト名（ネットワークに接続されているコンピューターに割り当てられている名前で、ネットワークを一意に識別し、IPアドレスを使用せずにアクセスできるようにします）もあります。 IPアドレスがわからなくても、代わりにIP名Majordomopiを入力して、SSHターミナルまたはiperfクライアントに接続できます。

RPiの小さな利点は、比較的安価で寸法が大きいため、クローゼット、ナイトスタンド、19インチラックなど、ルーターにネットワークケーブルで接続してミニPCを配置できることです。

MajorDoMoの基本構成では、デフォルトでは何も使用されていない3つのコントロールボタンがシステムのメインページにあります。ライト1、ライト2、ライト3の



いずれかを割り当てて、ネットワーク帯域幅をテストしてみましょう。

MajorDoMoのスタートページを開きます。移動[コントロールパネル] -デバイス-シンプルな機器-編集スイッチ1 -アクション。

アクションでは、PHPでコード行を記述します。これにより、ユーザーモードのPiからサーバーモードでIperfユーティリティが600秒間実行されます。

exec ('sudo -u pi  iperf -s -t 600 >/dev/null &');




その後、ホームオートメーションシステムのメインページから「ライト1」ボタンを押すと、Iperfが10分間起動し、必要なデバイスを介してスループットがテストされます。
