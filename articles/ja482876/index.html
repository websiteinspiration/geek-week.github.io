ヨーロッパの中心にある世帯の自分のニーズに合わせた太陽光発電のコストの計算

ドイツの電気料金に関するコメントと「国民はこれにどれくらい耐えられるだろうか」という合理的な質問への回答として。この記事で計算を行うことにしました。



前書き


この計算は2回目です。最初の作業は数年前に行われ、次の作業は更新されたデータが表示されたらすぐに行います。彼は客観性に頼らず、「それは理にかなっていますか？」

今日の設備価格と現在の稼働期間を考慮し、さまざまな補助金、「グリーン料金」やその他のごみを考慮せずに、家庭で生成される太陽光発電のコストを計算するタスクは、遅かれ早かれキャンセルされ、太陽は絶えずキャンセルされることです。結果として得られる数値は、この地域の現在の電力価格と比較して、再生可能エネルギーへの投資が報われるかどうかを理解できます。
具体的には基本機材のみを考慮し、設置工事や配線等のコストは考慮していません。影響は少ないものの、計算が煩雑になるためです。

初期条件


計算には、このような初期条件を使用します。

  • ミュンヘンの近くなど、ヨーロッパの中心のどこかに家があるとしよう。これは日射量を決定するために必要であり、それに応じて太陽電池パネルの必要な面積を決定します。
  • バッテリーを設置するのに十分な広さがあります。
  • 私たちの家庭の年間消費量を4000 kWhとしましょう。数か月にわたって均等に分散されます。それら。毎月の消費量は4000/12 = 333 kW * hになります。

設備とそのコストの計算


最初の免責事項-すぐに言いますが、計算は、消費のピークが生産のピークと一致しない可能性がある「正直な」システムで行われるため、システムはソーラーパネル+バッテリー+インバーターで構成されます。私の意見では、これはせいぜい完全な自律性とネットワーク料金からの独立を可能にするシステムの唯一のバージョンです。最悪の場合、ネットワークから時々電力を引き込むでしょう。このソリューションの概略図を次の図に示します。



一般的には、次のように機能します。ソーラーパネルは、インバータを介して家庭のACネットワークに接続されます。バッテリーもインバーターを介して同じネットワークに接続されます。ホームネットワークも通常のネットワークに接続されています。スマート管理は、ソーラーパネルの潜在能力が常に最大限に活用されるように、インバーターの動作を制御します。それら。太陽のエネルギーがすべてのホームデバイスに電力を供給するのに十分である場合、余剰エネルギーはバッテリーによってホームネットワークから取得され、充電されます。太陽が消えると、ホームネットワークはバッテリーで電力を供給し始め、放電します。バッテリーが完全に放電し、太陽がない場合のみ、家はネットワークから電気を取り始めます。

2番目の免責事項-天候が一貫していないため、ここでは平均統計値について話します。実際には、1か月雨が降る可能性があり、すべての計算に意味がありません。

ソーラーパネル


それでは、太陽電池パネルから始めましょう。最悪の場合の電力需要を満たすために必要な数を知る必要があります。私たちは2つの数字を知っています-必要な電力量は4000 kWh /年で、場所はミュンヘンです。

日射量の計算


場所ごとに、平方メートルあたりの平均日射量を取得する必要があります。 kWh / m2 /日で考慮されます。つまり、1平方メートルの表面が1日に太陽から受け取るエネルギーの量です。計算には、この計算機を使用します。この計算機は、曇りの日や霧などを考慮して、月ごとの統計を提供します。

太陽が少し照る冬に電力が必要になるため、日射量が最も少ない月（12月または1月）に関心があります。これにより、最悪のシナリオが発生します。
あなたはそれを平らな表面のために取り、次にソーラーパネルの最適な角度を見つけることができますが、計算機がそれを私たちのために行いますので、すぐに冬に最適な勾配（27度）をクリックして大切な数値を取得します：



つまり、最低日射量は12月で、1.51 kWh / m2 /日になります。少ない？しかし、それが一日であることを忘れないでください。そして、1か月で1.51 * 30.5 = 46 kWh / m2と入力されます。

太陽電池の建物のキャッチは何ですか？すでにモデルの名前のもとにあるすべてのソーラーパネルのメーカーが大切な図を大切にしているという事実-1000 W / m2の公称日射量での生産。この場合、それは330Wに等しく、この数値だけで効率と面積の両方が決まります。
このソーラーパネルの面積は標準です-1.6m。したがって、その効率は330 /（1000 * 1.6）= 20.6％となり、データシートに対応します。そして、冗談は、330Wに1.51を掛けることで、12月の平均日射量は498W * hになるということです。これは、そのようなパネルが1日にミュンヘンで1日に生成する電力であり、冬のコーナーに合わせて調整されます。これは、今後の計算で重要な数値です。

b）に従って、必要なパネルの数は次のように定義されます。計算機は私たちに私たちに一日あたりの世代を与えたので、消費量は数日間再計算されなければなりません。それら。 4000 kWhを365で割ると、10.96 kWh /日が得られます。 1つのパネルで498 Whが得られることがわかっているので、10.96 / 0.498 = 22パネルが必要であると判断するのは簡単です。

それは多かれ少なかれです-誰もが自分で決める。そのようなニュアンスがあります：

  • これらのパネルは、27度の角度で真南に設置する必要があります。つまり、平らな屋根を使用すると、パネルの実際の占有面積が大きくなります。はるかに。
  • 屋根に傾斜があるが、真南を向いていない場合、バッテリーの性能は低下します。
  • 12月でも太陽からすべてのエネルギーを消費したい場合、22枚のパネルが必要になることを覚えておいてください。たとえば、11月、12月、1月にネットワークから吸引できると判断して、この条件を緩和すると、最低日射量はすでに2.59（10月）になり、必要なパネルの総数は10.96 /（2.59 * 0.330）= 13.つまり、ほぼ2分の1になります。

コストを考えると、パネルの数を選ぶ問題に戻ります。そうではありませんが、私は推測します。ここですぐに決めましょう。

それで、ソーラーパネルを販売しているサイトに移動し、パネルVBHN330SA16をグーグルで検索します。パネルあたりの価格は250ユーロから280ユーロです。つまり、22枚のパネルには22 * 270（平均）= 5 940ユーロかかります。

さて、注目！これは名詞ではないので、データシートを読むと、パナソニックが25年の保証をパネルに与えることがわかります。同時に、この間パネルが10％以上劣化しないことを保証します。この期間を寿命とし、25年後にこれらのパネルを廃棄することを考慮すると、年間4000 kWhのみを選択する場合、キロワット時あたりのコストを計算することは難しくありません。 25年間、100,000 kWh（100 MWh）を削減します。 100,000あたり5 940ユーロを分割すると、0.0594ユーロ/ kWh、またはkWhあたり約6ユーロセントになります。

これはソーラーパネルのコンポーネントにすぎないことを思い出します。そして、これは、発電した電力をどこかに蓄えておき、それを利用する場合（12月はもちろん）だけです。

kWhあたりのコスト


私たちが判明した太陽光発電の総コストは、

  • ソーラーパネル：0.06
  • インバーター：0.01

この量のうち、ライオンのシェアはバッテリーにかかっており、主に寿命が短い可能性があるため、わずか10年です。しかし、これがすぐに良くなることを願っています。ソーラーパネルがDC / DCコンバーターを介してPowerwallに直接接続されているため、節約が可能です。したがって、1つのインバーターを節約できます。しかし、これは結局、キロワット時のコストで数セントになります。
興味深いことに、総コストに占めるソーラーパネルのコストは、主に長い耐用年数のため、非常に低かったです。したがって、鉄を節約することは意味がありませんが、高所での電池の高額な交換を避けるために、信頼できる機器に投資することをお勧めします。まあ、最終的な電気代に大きな影響を与えることなく、パネルの数を変えることができます。
