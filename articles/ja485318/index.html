Jupyter Notebookに美しさとインタラクティブ機能を追加する

多くの人が仕事でJupyter Notebookを使用しています。しかし、プロジェクトの複雑さが増すにつれて、問題が発生します。セルはノートブックに赤いマークで表示されます。「開始する前に番号を示してください...」または「に基づいて反復回数を設定してください...」。コマンドラインへの何らかのロールバックが取得されます。

そして、画面へのデータの出力は、美しいテーブル、写真、その他の最新のインターフェース要素に慣れている部外者による説明なしでは、常に認識されているわけではありません。

たとえば、都市部と人口に関するデータがあります。それらを「従来の形式」で表示します。

tabledata = [["", 2561, 12615882],
         ["-", 1439, 5383890],
         ["", 205, 609828],
         ["", 383, 1321473]]
tabledata

私たちはノートを見る：

[['', 2561, 12615882],
 ['-', 1439, 5383890],
 ['', 205, 609828],
 ['', 383, 1321473]]

現代の甘やかされて育った人は、常にこの表示形式が好きではありません。データをより馴染みのある表形式にする必要があります。

広く普及しているパンダライブラリを使用できます

import pandas as pd
pd.DataFrame(tabledata, columns=[""," (. )", " ()"])



なんらかの理由でパンダの使用が適さない場合は、別のライブラリを使用するか、独自の関数を作成できます。

これらのライブラリの1つを検討してください-tabulate（https://pypi.org/project/tabulate/）
インストールするには、コマンドラインからpip install tabulateを実行します

from IPython.display import HTML, display
from tabulate import tabulate 
display(HTML(tabulate(tabledata, tablefmt='html')))



データは「仮の」形式で表示できます。

print(tabulate(tabledata))


ヘッダーを追加できます

print(tabulate(tabledata, headers=[""," (. )", " ()"]))


そして指数

display(HTML(tabulate(tabledata, headers=[""," (. )", " ()"], tablefmt='html', showindex="always")))


tabulateを使用すると、パンダと同じ結果を視覚的に得ることができます。

独自の関数を記述して、追加の機能を取得できます。

from IPython.display import HTML, display

def dataToTable(data, columns = None):
    if len(data) == 0 :  
        display(HTML('<b> </b>'))
        return
    
    hdr = ''
    if columns != None:
        for col in columns: #   
            hdr = hdr + '<th>' + col + '</th>'
            
        hdr = '<tr>' + hdr + '</tr>'

    dt = ''
    for row in data: #     
        dt = dt + '<tr>'
        for cell in row: #    
            dt = dt + '<td>' + str(cell) + '</td>'
        dt = dt + '</tr>'
            
    display(HTML('<table>' + hdr + dt + '</table>')) #    

dataToTable(tabledata, columns=[""," (. )", " ()"])

インタラクティブなユーザー対話


同じipywidgetsライブラリでは、表示だけでなく情報の入力も可能です。

ユーザーインタラクションの最も可能性が高い最も単純な例は、ボタンクリックに対する応答です。ipywidgetsライブラリを使用すると、指定したパラメーター（テキスト、スタイル、サイズ）でボタンを作成し、それを押すハンドラー関数を割り当てることができます。

from IPython.display import display
from ipywidgets import Button 

#

                        button_style='success' # 'success', 'info', 'warning', 'danger', ''
                        )

def on_button_clicked(b): #   
    print("")

button1.on_click(on_button_clicked) #      "on_click"

display(button1) #

ユーザーが数値と日付を簡単に入力できるように、FloatSliderおよびDatePickerコンポーネントがあります。

入力された値を取得するには、<component> .valueプロパティを使用します。

値が変化した瞬間をキャッチするには、observeイベントを使用する必要があります

from IPython.display import display
from ipywidgets import FloatSlider

fSlider = FloatSlider(
    value=7.5, #  
    min=0,     # 
    max=10.0,  # 
    step=0.1,  #  
    description=':',
    continuous_update=False,  # True -  observe       
    orientation='horizontal'  #    
)

def on_value_change(b):
    print(b['old'], '->', b['new'])

fSlider.observe(on_value_change, names='value')
display(fSlider)



現在の値へのアクセスを確認します。

複数のオプションから1つの値を選択するために、RadioButtonsリスト、Dropdownドロップダウンリスト、およびToggleButtonsボタンのグループがあります。valueとobserveは、これらのコンポーネントに対して同じ方法で使用されます。

値は、文字列値のリストまたはタプルのリストとして指定できます。

AI（インターフェースの喜び）


インターフェイス要素が多すぎる場合は、デスクトップアプリケーションのパスに沿って、個々の要素を個別のグループにまとめたいと思います。

これには、アコーディオンとタブが便利です。

from IPython.display import display
from ipywidgets import Accordion, IntSlider, Text

accordion = Accordion(children=[IntSlider(value=42), Text(value=' ')])
accordion.set_title(0, ' 1')
accordion.set_title(1, ' 2')
display(accordion)



from IPython.display import display
from ipywidgets import Tab, IntSlider, Text

tab = Tab()
tab.children = [IntSlider(value=42), Text(value=' ')]
tab.set_title(0, ' 1')
tab.set_title(1, ' 2')

display(tab)
