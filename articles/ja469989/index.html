正規表現


正規表現は、文字列がパターンに一致するかどうかを確認するために使用されます。正規表現（正規表現、正規表現、または正規表現）は、パターンを定義する文字のシーケンスです。パターンは、リテラル、数値、文字、演算子、または構成で構成できます。パターンは、文字列またはファイル内の一致を見つけるために使用されます。
正規表現は、入力のチェック、解析、文字列の検索によく使用されます。たとえば、有効な生年月日、社会保障番号、フルネーム（名前と姓がカンマで区切られている）の確認、部分文字列の出現数の検索、部分文字列の置き換え、日付形式、有効な電子メール形式、通貨形式など。

クラス正規表現


.NETでは、Regexクラスは正規表現エンジンを表します。大量のテキストをすばやく解析して特定の文字パターンを見つけ、テキストの部分文字列を抽出、編集、置換、または削除し、抽出した文字列をコレクションに追加してレポートを生成するために使用できます。

Regexクラスは、System.Text.RegularExpressions名前空間で定義されています。Regexクラスのコンストラクターは、テンプレート文字列を他のオプションのパラメーターと共にパラメーターとして受け取ります。

次のコードスニペットは、パターンから正規表現を作成します。ここでは、パターンは文字「M」で始まる単語に対応しています。

//    ,     "M"  
string pattern = @"\b[M]\w+";  
//   Regex   
Regex rg = new Regex(pattern);


次のコードフラグメントには、分析する必要がある著者の名前を含む長いテキストが含まれています。

//    
string authors = "Mahesh Chand, Raj Kumar, Mike Gold, Allen O'Neill, Marshal Troll";


Matchesメソッドは、正規表現内のすべての一致を検索するために使用され、MatchCollectionを返します。

//     
MatchCollection matchedAuthors = rg.Matches(authors);


次のコードは、一致のコレクションを通過します。

//      
for (int count = 0; count < matchedAuthors.Count; count++)  
Console.WriteLine(matchedAuthors[count].Value);


完全なコードは次のとおりです。

//    ,     "M"  
string pattern = @"\b[M]\w+";  
//   Regex  
Regex rg = new Regex(pattern);  
  
//    
string authors = "Mahesh Chand, Raj Kumar, Mike Gold, Allen O'Neill, Marshal Troll";  
//     
MatchCollection matchedAuthors = rg.Matches(authors);  
///      
for (int count = 0; count < matchedAuthors.Count; count++)  
Console.WriteLine(matchedAuthors[count].Value);


上記の例では、コードは文字「M」を探します。しかし、単語が「m」で始まる場合はどうでしょうか。次のコードスニペットではパラメータを使用しているRegexOptions.IgnoreCaseため、正規表現では大文字と小文字が区別されません。

//    ,     "M"  
string pattern = @"\b[m]\w+";  
//   Regex  
Regex rg = new Regex(pattern, RegexOptions.IgnoreCase);


複数のスペースを正規表現で置き換える


このメソッドはRegex.Replace()、一致した文字列を新しい文字列に置き換えるために使用されます。次の例では、文字列内の複数のスペースを1つのスペースで検索します。

//       
string badString = "Here is a strig with ton of white space." ;  
string CleanedString = Regex.Replace(badString, "\\s+", " ");  
Console.WriteLine($"Cleaned String: {CleanedString}");


次のコードスニペットは、スペースを「-」に置き換えます。

string CleanedString = Regex.Replace(badString, "\\s+", "-");


正規表現を使用して文字列を文字に分割する


次の例では、正規表現パターン[az] +とメソッドRegex.Split()を使用して、文字列を文字に分割します。大文字と小文字は区別されません。

//      
string azpattern = "[a-z]+";  
string str = "Asd2323b0900c1234Def5678Ghi9012Jklm";  
string[] result = Regex.Split(str, azpattern,  
RegexOptions.IgnoreCase, TimeSpan.FromMilliseconds(500));  
for (int i = 0; i < result.Length; i++)  
{  
    Console.Write("'{0}'", result[i]);  
    if (i < result.Length - 1)  
        Console.Write(", ");  
}


Cの正規表現＃


正規表現は、文字列を解析および変更するためのパターンマッチングの標準であり、ユーザーは、コンピュータープログラムがテキスト内の指定されたパターンを探す方法を表現し、指定されたパターンで各一致が見つかった場合の処理​​を表現することができます。時々それらは「正規表現」と省略されます。これらは、特定の形式の文字列を見つけて変更する強力な方法です。

以下は、正規表現の使用方法を示す簡単なC＃コードの例です。

using System;  
using System.Collections.Generic;  
using System.Linq;  
using System.Text;  
using System.Text.RegularExpressions;  
  
namespace RegularExpression1  
{  
    class Program  
    {  
        static void Main(string[] args)  
        {  
            Regex r = new Regex(@"^\+?\d{0,2}\-?\d{4,5}\-?\d{5,6}");  
            //  Regex    .  
  
            string[] str = { "+91-9678967101", "9678967101", "+91-9678-967101", "+91-96789-67101", "+919678967101" };  
            //         .  
            foreach (string s in str)  
            {  
                Console.WriteLine("{0} {1} a valid mobile number.", s,  
                r.IsMatch(s) ? "is" : "is not");  
                //  IsMatch    ,   ,     .  
            }  
        }  
    }  
}

メールをチェックするための正規表現


複数のメールアドレスをテストするには、次の正規表現を使用できます。区切り文字「;」を使用してアドレスを区切ります。

^((\w+([-+.]\w+)*@\w+([-.]\w+)*\.\w+([-.]\w+)*)\s*[;]{0,1}\s*)+$

区切り文字「、」を使用する場合は、以下

^((\w+([-+.]\w+)*@\w+([-.]\w+)*\.\w+([-.]\w+)*)\s*[,]{0,1}\s*)+$

を使用し、区切り文字「、」と「;」の両方を使用する場合。次に、これを使用します。

^((\w+([-+.]\w+)*@\w+([-.]\w+)*\.\w+([-.]\w+)*)\s*[;,.]{0,1}\s*)+$

したがって、上記の正規表現を使用して、一度に1つまたは複数のアドレスの両方をチェックできます。

CでRegex.split（regex）を使用して文字列を分割する


このパートでは、C＃でRegExを使用して文字列を分割する方法を学びます。正規表現は、パターンに基づいて文字列を分割します。テンプレートとして指定された区切り文字を処理します。これがRegexがstring.Splitよりも優れている理由です。C＃でRegexを使用して文字列を分割する方法の例をいくつか示します。コードを書いてみましょう。

Regexを使用して文字列を分割するには、次の名前空間を追加します。

using System;

using System.Collections.Generic;

string Text = "1 One, 2 Two, 3 Three is good.";

文字列の特殊文字を正規表現に置き換える


このパートでは、C＃で特殊文字を正規表現に置き換える方法について説明します。

特殊文字を含む文字列があり、それらを削除または置換する場合は、これに正規表現を使用できます。

次のコードを使用します。

Regex.Replace(your String, @"[^0-9a-zA-Z]+", "")


このコードはすべての特殊文字を削除しますが、コンマ「、」やコロン「：」などの一部の特殊文字を削除したくない場合は、次の変更を行います。

Regex.Replace(Your String, @"[^0-9a-zA-Z:,]+", "")


同様に、要件に応じて変更を加えることができます。

注：

正規表現は、すべての小さな文字列操作の万能薬ではないことに注意してください。Stringクラスまたは他のクラスによって提供される単純な解析が必要な場合は、それを試して使用してください。
