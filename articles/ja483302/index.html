import httplib2 
import apiclient.discovery
from oauth2client.service_account import ServiceAccountCredentials	

CREDENTIALS_FILE = 'seraphic-effect-248407-7ac2c44ec709.json'  #     ,    

#    
credentials = ServiceAccountCredentials.from_json_keyfile_name(CREDENTIALS_FILE, ['https://www.googleapis.com/auth/spreadsheets', 'https://www.googleapis.com/auth/drive'])

httpAuth = credentials.authorize(httplib2.Http()) #   
service = apiclient.discovery.build('sheets', 'v4', http = httpAuth) #      4  API 

spreadsheet = service.spreadsheets().create(body = {
    'properties': {'title': '  ', 'locale': 'ru_RU'},
    'sheets': [{'properties': {'sheetType': 'GRID',
                               'sheetId': 0,
                               'title': '  ',
                               'gridProperties': {'rowCount': 100, 'columnCount': 15}}}]
}).execute()
spreadsheetId = spreadsheet['spreadsheetId'] #   
print('https://docs.google.com/spreadsheets/d/' + spreadsheetId)

すべてがエラーなしで成功した場合、テーブルへのリンクが表示されます。

このリンクではファイル識別子が使用されています。これは変数スプレッドシートIDに保存し、将来使用する予定です。

それに従います。アクセス権がないことがGoogleから通知されます。



許可を求めないでください！ Google自身がシステムアカウントに割り当てたアドレスにリクエストレターを送信することができないという通知が届きます。ただし、このアドレスは変更できません。おそらく、これはフリーモードでのみ機能しません。

しかし、Googleドライブを介して自分自身にアクセス権を与えることができます。 my_test_address@gmail.comを自分のものに置き換える必要があります。

driveService = apiclient.discovery.build('drive', 'v3', http = httpAuth) #    Google Drive  3  API
access = driveService.permissions().create(
    fileId = spreadsheetId,
    body = {'type': 'user', 'role': 'writer', 'emailAddress': 'my_test_address@gmail.com'},  #    
    fields = 'id'
).execute()

別の質問：セル範囲を指定する方法。どうやら、Googleスプレッドシートは、さまざまなマネージャーの指導とさまざまなアーキテクトの助けの下で、さまざまなチームによって開発されました。セルの座標は2つの異なる方法で設定されるためです。

オプション1：テキスト形式「シート番号1！B2：D5」、つまり シートの名前、その後の感嘆符、-左上隅の形式の「文字（列）+数値（文字列）」+右下隅の同じ形式のセル。

{"range": "  !B2:D5"}

オプション2：json形式で、シートIDと左上と右下のセルの座標を数値形式で（行番号と列番号）

{"range":
    {
    "sheetId": sheetId, # ID 
    "startRowIndex": 1, #    startRowIndex 
    "endRowIndex": 5,#  endRowIndex - 1 (endRowIndex  !)
    "startColumnIndex": 0, #    startColumnIndex 
    "endColumnIndex": 1 #  endColumnIndex - 1
    }}

異なる関数は異なるフォーマットを使用します。

これで、セルにデータを入力し、フレームを描き、ヘッダーを強調表示するのに十分なことがわかりました。

テーブルからデータを読み取る


データ読み取りの機能を完全に明らかにするために、図に示すように、セルB4、C7、およびD5に手動で入力しました。



データを読み取るコード

ranges = ["  !A2:F8"] # 
          
results = service.spreadsheets().values().batchGet(spreadsheetId = spreadsheetId, 
                                     ranges = ranges, 
                                     valueRenderOption = 'FORMATTED_VALUE',  
                                     dateTimeRenderOption = 'FORMATTED_STRING').execute() 
sheet_values = results['valueRanges'][0]['values']
print(sheet_values)

結果
[[''、 'セルB2'、 'セルC2'、 'セルD2']、[''、 '25'、 '36'、 '0.9999996829']]

関数のいくつかのパラメーター：valueRenderOption-数値データを読み取るためのフォーマット。

  • FORMATTED_VALUE-表示形式に基づいた読み取り。それら。テーブルに表示されていたものが読み取られます。たとえば、セルD3では、数値は0.9999999ですが、形式は「小数点以下2桁」なので、「1.00」が表示され、この形式で読み取られます。
  • UNFORMATTED_VALUE-セルの内容が読み込まれ、フォーマットは考慮されません（つまり、0.9999999が読み込まれます）
  • FORMULA - 「= SIN（3,14 / 2）」は、式は、この場合（表示されている数がセルに入力されている場合、このモードでは、読み込まれます。

このコードはデータを読み取り、1行ずつ表示します。読み取り可能範囲A2：F8。
画面に見られるように：

  • 読み込んだ行のセルが入力されていない場合、その行のデータは表示されません。
  • 最後に入力されたセルの後のデータは表示されません。
