Comparação das plataformas de desenvolvimento 1C e Client Communicator

Neste artigo, considerarei o 1C e o KliK não como uma solução pronta para implementação, mas como uma plataforma para criar uma solução individual.
1C Comunicador do cliente
Versões gratuitas
Estão ausentes.
Um tutorial gratuito está disponível com muitas limitações diferentes.
 Qualquer configuração pode ser usada no modo de usuário único gratuitamente e sem restrições.
Instalação
Opção de arquivo (não recomendada para operação de rede):
1. Instale a plataforma;
2. Crie uma base de informações.
Versão do servidor :
1. Instalação de um DBMS de sua escolha (MS SQL, PostgreSQL, IBM DB2, PostgresPro, Oracle Database);
2. Instalando a plataforma no servidor;
3. Crie uma infobase no servidor;
4. Instale nos computadores clientes e configure o acesso à infobase.

A opção do servidor é instalada, por regra, por um especialista experiente da 1C.
 1. Instalando o MS SQL;
2. Restaure o banco de dados no servidor;
3. Instale nos computadores clientes e configure o acesso ao banco de dados.
A instalação nem é gerenciada por um especialista em TI.

Conclusões:
1. O preço de compra inicial na versão inicial é o mesmo.
2. O preço do serviço 1C é muito mais alto, porque muitas vezes é necessária a ajuda de um programador.
3. Parte da funcionalidade em 1C está disponível, mas por uma taxa adicional.
4. A interface do usuário no Client Communicator é muito mais conveniente.
5. As melhorias em 1C consomem mais tempo e exigem certa qualificação de um especialista.
6. Para 1C, a empresa usa o melhor servidor disponível, mesmo apenas para tarefas de contabilidade.
