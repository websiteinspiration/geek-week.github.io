Sumber Daya Pengembang Web: API dan Elemen UI

Di blog kami, kami menyiapkan beberapa intisari dengan tolok ukur open source dari server Linux untuk administrator sistem ( satu , dua ). Hari ini kita akan berbicara tentang alat terbuka untuk pengembang web.

Di bawah cut adalah sumber daya dengan ikon dan API vektor dan SVG.


Foto - Hal Gatewood - Unsplash


API Publik


Ini adalah agregator API terbuka dari layanan yang dikembangkan oleh perusahaan IT besar seperti Google, TripAdvisor dan Mailchimp, serta penggemar dengan GitHub. Penulis platform ini adalah pengembang front-end Mohd Danish, yang juga memperkenalkan Tweet Jobs, layanan pencarian pekerjaan .

Yang menarik : perpustakaan memiliki lebih dari 680 API. Semuanya dibagi menjadi 50 kategori, yang meliputi: obat-obatan , cuaca , berita , keamanan informasi , data terbuka , penyimpanan cloud , pembelajaran mesin . Daftar antarmuka perangkat lunak memiliki alat untuk memproses ucapan alami ,Menggores , otomasi GitLab dan lainnya. Pencarian membantu Anda menavigasi. API dapat dicari dengan kata kunci dan oleh perusahaan pengembang.

Perhatian : Basis mengisi ulang agak lambat, tetapi Anda dapat menyarankan secara independen API yang hilang. Tidak ada spesifikasi antarmuka perangkat lunak di situs, dan untuk mempelajari lebih lanjut tentang alat ini, Anda harus membuka dokumentasi.

Bonus : Denmark juga meluncurkan NoCodeAPI , siap digunakan tanpa modifikasi pada kode backend - misalnya, memungkinkan Anda untuk mengonversi XML atau RSS ke JSON atau mengkonfigurasi kait di Slack.


Buka ui


Daftar dengan pustaka komponen (tombol, kotak centang, tombol radio) untuk halaman web. Sumber daya ini diawasi oleh para insinyur dari WICG (Web Platform Incubator Community Group), yang dimiliki oleh konsorsium W3C dan merancang standar untuk web. Para penulis yakin bahwa dengan Open UI, pengembang tidak akan menemukan kembali roda dan membuang waktu membuat kembali komponen.


Tangkapan layar: Buka UI / Halaman tombol

Yang menarik : Sistem desain utama disajikan: Desain Semut, Atlaskit, Sistem Desain Karbon, FastDNA, Sistem Desain Petir dan banyak lainnya.

Perhatian: Untuk mencari informasi tambahan tentang setiap sistem desain, Anda perlu menghubungi sumbernya. Tautan yang sesuai ditempatkan di bagian atas halaman, jadi Anda harus memundurkannya secara teratur ke atas. Sumber daya itu sendiri tidak selesai - penulis menambah informasi tentang sistem desain untuk tombol pilihan , slider dan tabel .

Apa lagi : Mereka yang ingin dapat berkontribusi dan mendokumentasikan kumpulan aset untuk merancang UI multi-platform. Instruksi dan persyaratan diberikan di situs .


Flaticon


Perpustakaan ikon vektor dari pengembang stok foto Freepik. Ini dilengkapi oleh 370 penulis independen . Situs ini menawarkan lebih dari 500 ribu gambar dalam format PNG, SVG, EPS dan PSD.


Tangkapan layar: Flaticon / Halaman utama.

Keuntungan : Ikon dibagi menjadi 10 ribu set, yang menyederhanakan pencarian. Di antara kategori populer dapat dibedakan: konstruksi , pakaian , hewan peliharaan , bahkan ada coronavirus , karantina dan situasi darurat . Sebelum mengunduh gambar, Anda dapat mengubah warnanya dalam editor khusus - untuk seluruh koleksi sekaligus atau satu per satu. Juga menawarkan Flaticongenerator ornamen - gambar berulang yang dapat digunakan sebagai latar belakang di situs.

Kekurangan : Untuk mengakses semua fungsi situs, diperlukan pendaftaran. Tidak semua ikon gratis, tetapi untuk konten premium Anda harus membayar 10 euro per bulan. Gambar gratis hanya dapat digunakan dengan atribusi.

Bonus : Sumber daya lain dengan gambar SVG gratis untuk pengembang web adalah Sparkk . Untuk menempelkan ikon di situs, cukup salin kode HTML yang sesuai.
