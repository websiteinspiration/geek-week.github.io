番号区切り


プログラムで見つかった長い数字は読みにくいです。たとえば、1000000000-は10進数で10億です。しかし、一目で理解するのは難しいです。したがって、プログラムの読者が同様の何かに遭遇した場合-彼はそれを正しく認識するために、ゼロを慎重に検討する必要があります。

最近のJavaScriptでは、数字の桁の区切り文字-アンダースコア（_）を使用できます。これを使用すると、長い数字の読みやすさが向上します。デリミタを使用して記述された数値がコードでどのように見えるかを次に示します。

var billion = 1_000_000_000;
console.log( billion ); // 1000000000

セパレータは、数値をフラグメントに任意に分割するために使用できます。JavaScriptは、数値を処理する場合、セパレータを無視します。整数、浮動小数点、2進数、16進数、8進数など、任意の数値を書き込むときに使用できます。

console.log( 1_000_000_000.11 ); // 1000000000.11
console.log( 1_000_000_000.1_012 ); // 1000000000.1012
console.log( 0xFF_00_FF ); // 16711935
console.log( 0b1001_0011 ); // 147
console.log( 0o11_17 ); // 591

国際化API


国際化APIは、さまざまな地域の標準（ロケール）で慣習となっているように、文字列比較を整理し、数値、日付、および時間をフォーマットするように設計されています。このAPIへのアクセスは、オブジェクト を通じて編成されますIntl。このオブジェクトは、ソーターオブジェクトおよびデータをフォーマットするオブジェクトを作成するためのコンストラクターを提供します。オブジェクトIntlがサポートするロケールのリストは、ここにあります。

▍Intl.RelativeTimeFormat（）


多くのアプリケーションでは、相対形式で時間を出力する必要があることがよくあります。「5分前」、「昨日」、「1分前」などのようになります。ウェブサイトの資料が異なる言語に翻訳されている場合、サイトの組み立てに時間を表す相対的な構成の可能なすべての組み合わせを含める必要があります。

JSには、さまざまなロケールの日付と時刻のフォーマットシステムを作成できるコンストラクター Intl.RelativeTimeFormat(locale, config)があります。特に、さまざまな相対タイムスタンプを生成できるメソッド .format(value, unit)を持つオブジェクトについて説明しています。次のようになります。

// español ( )

  numeric: 'auto'

約束


JSには現在、静的メソッドPromise.all()とがありPromise.race()ます。メソッドPromise.all([...promises])は、引数としてメソッドに渡されたすべてのpromiseが解決された後に正常に解決されたpromiseを返します。転送された約束の少なくとも1つが拒否された場合、この約束は拒否されます。このメソッドPromise.race([...promises])は、転送されたいずれかのプロミスが解決された後に解決されるプロミスを返し、そのようなプロミスの少なくとも1つが拒否された場合は拒否されます。

JS開発者のコ​​ミュニティは静的メソッドを切望しており、渡されたすべてのpromiseが完了（許可または拒否）された後に解決されるpromiseが返されました。さらに、方法を連想させるrace()それはそれに与えられた約束のいずれかの解決までの約束を返します。
