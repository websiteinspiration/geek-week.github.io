パートII：CANNY 3タイニーの回路とプログラムの接続


そもそも、著作権を侵害しないように、フォーラムから借りましたが、あなたのタスク。

接続図は次のとおりです。



リミットスイッチとリードスイッチはコントローラNo. 6とNo. 5の端子に接続され、ヘッドフォンは端子No. 4（RF PWMを備えています）に接続され、UART RXは端子No. 1、UART TXは端子No. 2、端子No. 3 「+ 5V」を供給するために使用され、出力「-」は「グランド」との通信に使用されます。

アセンブリでは次のようになります。



CANNY 3のダイアグラム（プログラム）をCannyLabバージョン1.42で開発しました。おそらく他のバージョンの開発環境で、他のコントローラーを使用して、ダイアグラムに変更を加える必要があります。

これが起こったことです：



コントローラーの設定とUARTを介したメッセージの送信に関連するブロックは、以前の記事で逆アセンブルされました。

残りの2つについて詳しく見てみましょう。

「UARTメッセージの受信」ブロックは、サイレン（ヘッドフォン）をオンにします。原則として、UARTを介してメッセージを受信する例を解析する必要があります。

最初に、受信したデータがUARTにあるかどうかを確認します。ある場合は、Dトリガー入力「E」に1を送信します。この場合、トリガーは「D」入力から値をコピーし、UART経由で受信したメッセージの最初の2文字を書き込みます。すべてを複雑にしたくなかったので、簡単なスキームをさらに使用します。 UARTによって00から99までの任意の数が届くと想定し、この数をシンボリック形式から数値形式に変換します（コンバーターブロックのしくみを読むことをお勧めします。これには少し「プラグ」がありました）。さらに、リーディングエッジ検出器の入力に "> 0"を入力すると、単一の信号が発生し、出力5が5秒間オンになり、RF PWMモードで動作します。

RF PWMの充填期間を設定で再生できます。ヘッドフォンのサウンドはこれに依存します。

ブロックに移りましょう"メッセージの形成。"一見するとその実装は異常に見えるかもしれません。これは、シリアルBluetooth端末プログラムとXamarinの同じBluetoothプロトコルを使用する方法を実際には理解していなかったという事実によって説明されます。

少し先を走らせて、スマートフォンのコントローラーから送信されたメッセージを受信する方法をまだ学習していないと言います。前の記事の有線UARTですべてが明らかだった場合、実際には、Bluetoothでは送信されたメッセージの代わりに、その一部のみを読み取ることができ、送信されたコマンドの意味に違反しています。

私は、最も簡単な解決策は、単一の番号を送信することであると判断しました。

今回のケースでは、リードスイッチとリミットスイッチの離散状態を監視します。つまり、可能な組み合わせは4つだけです。リードスイッチとリミットスイッチがオフで、片方だけがオンで、両方がオンです。

リードスイッチとリミットスイッチはディスクリート信号（0/1）を提供するため、何らかの方法でそれらを区別する必要があります。これを行うには、リード信号の値に2を乗算します。これで、信号の合計によって0から3までの値が得られることがわかります。

次に、この値に50を追加した非自明なオプションを分析します。事実は、CannyLabがUARTに2文字、つまり3ではなく03を送信するということですが、前述したように、一部の情報が失われるリスクがあります。たとえば、値01から、スマートフォンのプログラムは最初の「0」のみを読み取ることができ、これはすでにエラーになります。

たとえば、レジ​​スタの「D1」文字を文字やスペースに置き換えて、混乱してデータを変換することは可能ですが、簡単にすることにしました。値を01から51に変換しました（02から52など）。 5つは信号を運びません、そして私はスマートフォンのプログラムレベルでそれを切り取りました。したがって、メッセージの有用な部分が残ることを常に保証します。

プログラムをコントローラーにロードし、「実行」をクリックします。すべてが予定どおりに機能する場合、HC-06は定期的に赤色のLEDを点滅させます。

次に、スマートフォンとアダプターをペアリングします。これで、アプリケーション「シリアルBluetooth端末」または同様の機能を持つその他のアプリケーションでパフォーマンスを確認できます。

Bluetoothアダプタのアドレスを書き留めてください。次の章で役立ちます。



ご覧のとおり、センサーの状態に応じてデータが送られてきます。「11」を送信すると、ヘッドフォンで不快なきしみ音が聞こえます。ここで止めることもできますが、プリミティブアプリケーションをスケッチしましょう。

namespace _6.Canny_Xanarin_Bluetooth_Android

    [Activity(Label = "Control Canny 3 tiny via bluetooth", Theme = "@style/AppTheme", MainLauncher = true)]
    public class MainActivity : AppCompatActivity
    {

        Button startSiren;
        TextView rSwitch;
        TextView EndSensor;
        Switch bltSwitch;
        TextView status;
        private Java.Lang.String dataToSend;
        private BluetoothAdapter mBluetoothAdapter = null;
        private BluetoothSocket btSocket = null;
        private Stream outStream = null;
        // don't forget change addres to your device:
        private static string address = "98:D3:91:F9:6C:F6";
        // MY_UUID can be saved as is
        private static UUID MY_UUID = UUID.FromString("00001101-0000-1000-8000-00805F9B34FB");

        protected override void OnCreate(Bundle savedInstanceState)
        {

            base.OnCreate(savedInstanceState);
            Xamarin.Essentials.Platform.Init(this, savedInstanceState);
            // Set our view from the "main" layout resource
            SetContentView(Resource.Layout.activity_main);

            startSiren = FindViewById<Button>(Resource.Id.startSiren);
            rSwitch = FindViewById<TextView>(Resource.Id.rSwitch);
            EndSensor = FindViewById<TextView>(Resource.Id.EndSensor);
            status = FindViewById<TextView>(Resource.Id.status);
            bltSwitch = FindViewById<Switch>(Resource.Id.bltSwitch);


            startSiren.Click += startSiren_ClickOnButtonClicked;
            bltSwitch.CheckedChange += bltSwitch_HandleCheckedChange;
            CheckBt();
        }

        private void CheckBt()
        {
            mBluetoothAdapter = BluetoothAdapter.DefaultAdapter;

            if (!mBluetoothAdapter.Enable())
            {
                Toast.MakeText(this, "Bluetooth Off",
                    ToastLength.Short).Show();
            }

            if (mBluetoothAdapter == null)
            {
                Toast.MakeText(this,
                    "Bluetooth does not exist or is busy", ToastLength.Short)
                    .Show();
            }
        }

        void startSiren_ClickOnButtonClicked(object sender, EventArgs e)
        {
            if (bltSwitch.Checked)
            {
                try
                {
                    dataToSend = new Java.Lang.String("11");
                    writeData(dataToSend);
                    System.Console.WriteLine("Send signal to siren");
                }
                catch (System.Exception execept)
                {
                    System.Console.WriteLine("Error when send data" + execept.Message);
                }

            }
            else status.Text = "bluetooth not connected";
        }
        void bltSwitch_HandleCheckedChange(object sender, CompoundButton.CheckedChangeEventArgs e)
        {
            if (e.IsChecked)
            {
                Connect();
            }
            else
            {
                status.Text = "bluetooth not connected";
                if (btSocket.IsConnected)
                {
                    try
                    {
                        btSocket.Close();
                        System.Console.WriteLine("Connection closed");
                    }
                    catch (System.Exception ex)
                    {
                        Console.WriteLine(ex.Message);
                    }
                }
            }
        }

        public void Connect()
        {
            BluetoothDevice device = mBluetoothAdapter.GetRemoteDevice(address);
            System.Console.WriteLine("Connection in progress" + device);
            mBluetoothAdapter.CancelDiscovery();
            try
            {
                btSocket = device.CreateRfcommSocketToServiceRecord(MY_UUID);
                btSocket.Connect();
                System.Console.WriteLine("Correct Connection");
                status.Text = "Correct Connection to bluetooth";
            }
            catch (System.Exception e)
            {
                Console.WriteLine(e.Message);
                try
                {
                    btSocket.Close();
                    System.Console.WriteLine("Connection closed");
                }
                catch (System.Exception)
                {
                    System.Console.WriteLine("Impossible to connect");
                    status.Text = "Impossible to connect";
                }
                System.Console.WriteLine("Socket Created");
  
            }
            beginListenForData();

        }

        public void beginListenForData()
        {
            try
            {
                inStream = btSocket.InputStream;
            }
            catch (System.IO.IOException ex)
            {
                Console.WriteLine(ex.Message);
            }
            Task.Factory.StartNew(() => {
                byte[] buffer = new byte[1024];
                int bytes;
                
                while (true)
                {

                    try
                    {
                        bytes = inStream.Read(buffer, 0, 1024);
                        System.Console.WriteLine("bytes " + bytes.ToString());
                        if (bytes > 0)
                        {
                            
                            RunOnUiThread(() => {
                                string valor = System.Text.Encoding.ASCII.GetString(buffer).Replace("5",String.Empty);
                                // transform string for deleate all symbols except 1-4(command from canny).
                                string command = new string(valor.Where(char.IsDigit).ToArray());

                                if (command.Length > 0)
                                {
                                     status.Text="data successfully readed";
                                    System.Console.WriteLine("command  " + command);
                                    switch (Int32.Parse(command))
                                    {
                                        case 0:
                                            rSwitch.Text = "reed switch - disconnected ";
                                            EndSensor.Text = "end sensor - not pressed ";
                                            break;
                                        case 1:
                                            rSwitch.Text = "reed switch - disconnected ";
                                            EndSensor.Text = "end sensor - pressed ";
                                            break;
                                        case 2:
                                            rSwitch.Text = "reed switch - connected ";
                                            EndSensor.Text = "end sensor - not pressed ";
                                            break;
                                        case 3:
                                            rSwitch.Text = "reed switch - connected ";
                                            EndSensor.Text = "end sensor - pressed ";
                                        break;
                                    }
                                }
                            });
                        }
                    }
                    catch (Java.IO.IOException)
                    {
                        RunOnUiThread(() => {
                            EndSensor.Text = "End sensor status - undefined";
                            rSwitch.Text = "Reed switch status - undefined ";
                        });
                        break;
                    }
                }
            });
        }

        private void writeData(Java.Lang.String data)
        {
            try
            {
                outStream = btSocket.OutputStream;
            }
            catch (System.Exception e)
            {
                System.Console.WriteLine("Error with OutputStream when write to Serial port" + e.Message);
            }

            Java.Lang.String message = data;

            byte[] msgBuffer = message.GetBytes();

            try
            {
                outStream.Write(msgBuffer, 0, msgBuffer.Length);
                System.Console.WriteLine("Message sent");
            }
            catch (System.Exception e)
            {
                System.Console.WriteLine("Error with  when write message to Serial port" + e.Message);
                status.Text = "Error with  when write message to Serial port";
            }
        }
