Consejos para elegir un amplificador de señal celular 2G / 3G / 4G / 5G

Un artículo sobre opciones para amplificar una señal celular en varios sitios: una residencia de verano, oficina, almacén ..., así como una breve descripción de las ofertas populares en el mercado.


Hay dos opciones principales para mejorar las comunicaciones celulares:


  1. Usando una antena, módem y enrutador.
  2. Usando un amplificador activo (repetidor).

La primera opción para fortalecer Internet 3G / 4G / 5G, no mejora la comunicación de voz en el formato habitual para nosotros (pero puede usar llamadas a través de mensajería instantánea: WhatsApp, Viber, Skype, Telegram).


La segunda opción es universal: fortalece Internet y la comunicación de voz, pero tiene una serie de restricciones (sobre las restricciones a continuación).



Amplificación de Internet 3G / 4G usando una antena, módem y enrutador Wi-Fi


El sistema consta de dos nodos: una antena en la calle y un módem + enrutador Wi-Fi en interiores.



Antena externa para módem / enrutador 3G / 4G



Módem 4G conectado por adaptadores a una antena de calle e insertado en un enrutador Wi-Fi


¿Cómo funciona? La antena se dirige hacia la estación base propuesta del operador celular. Luego, la señal va a un módem 4G con un enrutador que distribuye Internet mediante la distribución de una señal de Wi-Fi. La opción es popular entre los maestros privados para instalar un Internet suburbano. Barato y funciona!
