リソースの概要


Forex Service（FS）はサービスプロバイダーです。さまざまな通貨の為替レートを提供します。彼が外国為替取引所と通信し、さまざまな通貨の為替レートの現在の値を提供すると仮定します。

リクエストとレスポンスの例を以下に示します。

GET to http://localhost:8000/currency-exchange/from/EUR/to/INR

{
  id: 10002,
  from: "EUR",
  to: "INR",
  conversionMultiple: 75,
  port: 8000,
}

上記のリクエストは、ユーロの為替レートをINRに戻します。回答では、ConversionMultiple変換係数は75であることがわかります。

Exchange Value JPAリポジトリの作成


/spring-boot-microservice-forex-service/src/main/java/com/in28minutes/springboot/microservice/example/forex/ExchangeValueRepository.java

package com.in28minutes.springboot.microservice.example.forex;
import org.springframework.data.jpa.repository.JpaRepository;

public interface ExchangeValueRepository extends 
    JpaRepository<ExchangeValue, Long>{
  ExchangeValue findByFromAndTo(String from, String to);
}

注：

1. パブリックインターフェースExchangeValueRepositoryはJpaRepositoryを拡張します<ExchangeValue、Long>

は、 ExchangeValueとLongの2つのジェネリックを使用してJpaRepositoryを拡張します。ExchangeValueは管理対象オブジェクトであり、ExchangeValueの主キーはLongです。

2. ExchangeValue findByFromAndTo（文字列from、文字列to）

は、ある通貨から別の通貨への変換のコストを要求するためのメソッドを定義します。

ForexControllerリソースを作成する


/spring-boot-microservice-forex-service/src/main/java/com/in28minutes/springboot/microservice/example/forex/ForexController.java

@RestController
public class ForexController {
  
  @Autowired
  private Environment environment;
  
  @Autowired
  private ExchangeValueRepository repository;
  
  @GetMapping("/currency-exchange/from/{from}/to/{to}")
  public ExchangeValue retrieveExchangeValue
    (@PathVariable String from, @PathVariable String to){
    
    ExchangeValue exchangeValue = 
        repository.findByFromAndTo(from, to);
    
    exchangeValue.setPort(
        Integer.parseInt(environment.getProperty("local.server.port")));
    
    return exchangeValue;
  }
}

注：

1. @RestControllerパブリッククラスForexController {

＆nbsp＆nbsp＆nbspは、RESTサービスを提供するコントローラーを作成します

2。自動配線プライベート環境環境

＆nbsp＆nbsp＆nbspはサーバーポートを返します。これは、サービスのどのインスタンスが応答を返すかを決定するのに役立ちます。

3。自動配線プライベートExchangeValueRepositoryリポジトリ -自動リポジトリ接続。

4. ExchangeValue exchangeValue = repository.findByFromAndTo（from、to） -データベースから為替レートの値を取得します。

5. exchangeValue.setPort（Integer.parseInt（environment.getProperty（ "local.server.port"）））

＆nbsp＆nbsp＆nbspは、環境オブジェクトからポート属性を受け取り、それを応答Beanに設定します。
