Get best of the week

2回目のプログラミングチャンピオンシップ：MLトラックのタスクを分析します

10月、第2回プログラミング選手権が開催されました。 12,500件の応募があり、6,000人以上の人がコンテストに参加しました。今回、参加者は、バックエンド、フロントエンド、モバイル開発、機械学習のいずれかのトラックを選択できます。予選ステージとファイナルを通過するために必要な各ト​​ラック。

伝統的に、Habréでトラックの分析を公開します。機械学習の認定フェーズのタスクから始めましょう。チームはそのような5つのタスクを準備し、そのうち3つのタスクには2つのオプションがありました。最初のバージョンでは、タスクA1、B1、Cがあり、2番目のバージョンでは、A2、B2、Cでした。オプションは、参加者間でランダムに分配されました。タスクCの作成者は開発者のPavel Parkhomenkoで、残りのタスクは同僚のNikita Senderovichによって作成されました。

最初の単純なアルゴリズムタスク（A1 / A2）では、回答を正しく列挙することで、参加者は50ポイントを獲得できます。2番目のタスク（B1 / B2）には、ソリューションの有効性に応じて、10〜100ポイントを与えました。100ポイントを獲得するには、動的プログラミング手法を実装する必要がありました。3番目のタスクは、提供されたトレーニングデータに基づいてクリックモデルを構築することです。カテゴリ属性を操作するためのメソッドの適用と、非線形学習モデル（たとえば、勾配ブースティング）の使用が必要でした。このタスクでは、テストサンプルの損失関数の値に応じて、最大150ポイントを獲得できました。

def check_character(char, curchar):
    return curchar is None or char == "#" or curchar == char

def need_to_assign_curchar(char, curchar):
    return curchar is None and char != "#"


n = int(input())
s = input().strip()
for p in range(1, n + 1):
    is_ok = True
    for i in range(0, p):
        curchar = None
        for j in range(i, n, p):
            if not check_character(s[j], curchar):
                is_ok = False
                break
            if need_to_assign_curchar(s[j], curchar):
                curchar = s[j]
        if not is_ok:
            break
    if is_ok:
        print(p)
        break

n, k = map(int, input().split())
s = input().strip()

for p in range(1, n + 1):
    mistakes = 0
    for i in range(0, p):
        counts = [0] * 26
        for j in range(i, n, p):
            counts[ord(s[j]) - ord("a")] += 1
        mistakes += sum(counts) - max(counts)
    if mistakes <= k:
        print(p)
        break

決定は30ポイントです。mが3を超えない場合、可能なすべてのO（n3）3枚のカードの場合、制限を満たす総合的な関連性の観点から最適なオプションを選択します。

決定は70ポイントです（50ポイントは同じで、実装がより簡単です）。フラグが3つ以下の場合、すべてのパブリケーションを、それらが持つフラグのセットに従って8つの互いに素なグループに分割できます：000、001、010、011 100、101、110、111。各グループのパブリケーションは、関連性の降順でソートする必要があります。次に、O（m 4）について、グループ111、011、110、101からどれだけ多くの最良の出版物を取得するかを選別できます。それぞれから0からmまでの出版物を取得します。合計はm以下です。その後、要件を満たすためにグループ100、010、および001から収集する必要があるパブリケーションの数が明らかになります。残りのカードとの関連性が最も高いmに到達する必要があります。

[0] =0。他のすべてのパラメータセットでは、将来この値を最大化するために、値を-1に設定します。次に、一度に1枚ずつ「ゲームに参加」し、各カードでこの関数の値を改善します。ダイナミクスの各状態（i、a、b、...、z）で、フラグ付きのj番目のパブリケーション（a j、b ）を使用しますj、...、z j）、状態（i + 1、a + a j、b + b j、...、z + z）への遷移を試みることができますj）、この条件で結果が改善するかどうかを確認します。重要：遷移中は、i≥mである状態には関心がないため、そのようなダイナミクスの合計状態はm k + 1以下であり、結果の漸近動作はO（nm k + 1）です。ダイナミクスの状態が計算されるとき、答えは制約を満たし、最高の合計関連性スコアを与える状態です。

実装の観点から、動的プログラミングの状態と各パブリケーションのフラグを1つの整数でパッケージ化された形式で格納し、リストやタプルではなくプログラムの作業（コードを参照）を加速すると便利です。このソリューションではメモリ使用量が少なくなり、ダイナミクスの状態を効果的に更新できます。

完全なソリューションコード：

def pack_state(num_items, counts):
    result = 0
    for count in counts:
        result = (result << 8) + count
    return (result << 8) + num_items


def get_num_items(state):
    return state & 255


def get_flags_counts(state, num_flags):
    flags_counts = [0] * num_flags
    state >>= 8
    for i in range(num_flags):
        flags_counts[num_flags - i - 1] = state & 255
        state >>= 8
    return flags_counts


n, m, k = map(int, input().split())
scores, attributes = [], []
for i in range(n):
    score, flags = input().split()
    scores.append(int(score))
    attributes.append(list(map(int, flags)))
limits = list(map(int, input().split()))

dp = {0 : 0}
for i in range(n):
    score = scores[i]
    state_delta = pack_state(1, attributes[i])
    dp_temp = {}
    for state, value in dp.items():
        if get_num_items(state) >= m:
            continue
        new_state = state + state_delta
        if value + score > dp.get(new_state, -1):
            dp_temp[new_state] = value + score
    dp.update(dp_temp)

best_score = 0
for state, value in dp.items():
    if get_num_items(state) != m:
        continue

    flags_counts = get_flags_counts(state, k)
    satisfied_bounds = True
    for i in range(k):
        if flags_counts[i] < limits[i]:
            satisfied_bounds = False
            break

    if not satisfied_bounds:
        continue

    if value > best_score:
        best_score = value

print(best_score)

n, k = map(int, input().split())
f = list(map(float, input().split()))

prefix_sum = [0.0] * (n + 1)
prefix_sum_sqr = [0.0] * (n + 1)
for i in range(1, n + 1):
    prefix_sum[i] = prefix_sum[i - 1] + f[i - 1]
    prefix_sum_sqr[i] = prefix_sum_sqr[i - 1] + f[i - 1] ** 2

def get_best_sse(l, r):
    num = r - l + 1
    s_sqr = (prefix_sum[r] - prefix_sum[l - 1]) ** 2
    ss = prefix_sum_sqr[r] - prefix_sum_sqr[l - 1]
    return ss - s_sqr / num

dp_curr = [1e100] * (n + 1)
dp_prev = [1e100] * (n + 1)
dp_prev[0] = 0.0
for num_segments in range(1, k + 1):
    dp_curr[num_segments] = 0.0
    for num_covered in range(num_segments + 1, n + 1):
        dp_curr[num_covered] = 1e100
        for num_covered_previously in range(num_segments - 1, num_covered):
            dp_curr[num_covered] = min(dp_curr[num_covered], dp_prev[num_covered_previously] + get_best_sse(num_covered_previously + 1, num_covered))
    dp_curr, dp_prev = dp_prev, dp_curr

print(dp_prev[n])
