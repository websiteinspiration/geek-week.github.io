1時間でMySQLを使用するPython上のVK用ボット、パート1

Vkontakte用のボットを作成する方法については、インターネット上に多くのチュートリアルがありますが、初心者には重大な欠点があります。将来的に初心者が特定のものがどのように機能するかを理解することが困難になるため、これらは非常に最小限のベースを提供します。

このチュートリアルでは、かなり大きな基本機能を備えた最もシンプルなボットを作成する方法を紹介します。

それでは始めましょう。

私たちは何が必要なのか？


  • バージョン2.7以降のPython（すべてのコードはバージョン2.7.16および3.7.3でテストされています）
  • PyMySQLモジュール
  • Vk_apiモジュール
  • リクエストモジュール
  • Vkontakteグループ（注意！ボットはグループの側から正確に機能し、個人のアカウントの側からは機能しません）
  • データベース（ユーザーデータなどを含む）

接続


まず、すべてのライブラリを接続します。

import vk_api
from vk_api.utils import get_random_id
from vk_api.bot_longpoll import VkBotLongPoll, VkBotEventType
import pymysql.cursors
import requests

これで、ボットとデータベースの構成を開始できます。

私の場合、すべてのデータはconfig.pyファイルにあります。どこでも処方できます。機能には影響しません。

データベース接続


これを行うには、PyMySQLモジュールを使用して関数を作成します。


def get_connection():
    connection = pymysql.connect(host='you_host',
                                 user='you_user',
                                 password='you_password',
                                 db='you_db'
                                 charset='utf8mb4',
                                 cursorclass=mymysql.cursors.DictCursor)
    return connection

この関数は、データベースでさらに作業するためのデータを返します。したがって、「you_」が表示されている箇所をデータに置き換えます。

ボット接続を構成する


まず、グループの設定に移動する必要があります。



[APIの処理]タブを選択し、アクセスキーを作成します（将来、ボットに接続すると便利です。



[ロングポーリングAPI]タブに移動して有効にします。また、利用可能な最新バージョンを選択します



「イベントタイプ」に移動し、必要なものもすべて含めます。たとえば、私はすべてを含めまし



た最後に「メッセージ」>「ボットの設定」に移動して、ボットの機能を有効にします。これで、プログラム自体に移動できます。

ボットで作業する


メッセージでメッセージに応答するボット関数を追加しましょう。これを行うには、最後の行の後に次のように記述します。


    if event.type == VkBotEventType.MESSAGE_NEW: #  
        #      
        if event.obj.text != '': 
            #      
            if event.from_user:
                vk.messages.send(
                        user_id=event.obj.from_id,
                        random_id=get_random_id(),
                        message=event.obj.text)

次に、vk.messages.send（）メソッドについて説明します。これは、vk_apiモジュールの標準メソッドです。メソッドについては、公式のVkontakteドキュメントで読むことができます（リンクは下にあります）。

ユーザーにメッセージを送信できます。user_id

引数実際に送信するメッセージを示します。Event.obj.textは、メッセージで受け取ったテキストに関する情報を格納します。この引数も必要です。

発売


これでボットの準備が整い、ボットを開始できます。開始後、ボットに安全に書き込み、結果を確認できます。
