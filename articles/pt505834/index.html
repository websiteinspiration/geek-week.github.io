Amostras medianas. Intervalos de confiança e comparação

A publicação é da natureza da descrição do caminho que leva à algoritmo eficiente de métodos para calcular o intervalo de confiança (Intervalo de Confiança = IC) para


  • medianas de distribuição;
  • diferenças de medianas de duas distribuições.

A tarefa é puramente prática, você pode mergulhar nas profundezas da matemática, mas isso não é um fim em si e nem sempre é suficiente para um balão chegar ao fundo.


As amostras são grandes em tamanho, 10 ^ 5 - 10 ^ 7 registros, visivelmente assimétricos, com caudas longas, podem ter vários modos. Nesse caso, as medianas são mais resistentes às emissões.


A aplicação da estatística clássica, por exemplo, o critério de Wilcoxon-Mann-Whitney, para estimar a diferença de medianas para esses volumes, não passa. Sim, e muitas coisas precisam ser lidas sob asteriscos para aplicar corretamente esses critérios. De fato, esse critério de forma alguma verifica a igualdade de medianas e, para medianas, funciona apenas com as mesmas formas de distribuição de duas amostras. Etc. etc.


Você pode pegar o martelo de inicialização, mas precisa pensar com isso + leva tempo e memória para simular.


Por outro lado, frequentemente os matemáticos apresentam várias simplificações analíticas sob certas suposições, o que permite que problemas complexos sejam resolvidos em uma fórmula. A busca pela última abordagem levou às seguintes soluções (em relação às amostras descritas acima).

# test from table 3 of b&p 2002
x1 <- c(77, 87, 88, 114, 151, 210, 219, 246, 253, 262, 296, 299, 306,
        376, 428, 515, 666, 1310, 2611)
x2 <- c(59, 106, 174, 207, 219, 237, 313, 365, 458, 497, 515, 529,
        557, 615, 625, 645, 973, 1065, 3215)

# sort vectors
x1 <- sort(x1)
x2 <- sort(x2)

# get medians
x1_mdn <- median(x1)
x2_mdn <- median(x2)

# stuff to calculate variance of medians
x1_n <- length(x1)
x2_n <- length(x2)

x1_aj <- round((x1_n + 1) / 2 - x1_n ^ (1 / 2))
x2_aj <- round((x2_n + 1) / 2 - x2_n ^ (1 / 2))

z <- 1.855 # from table 1 of b&p 2002, see p. 376

# calculate variance
x1_var <- ((x1[x1_n - x1_aj + 1] - x1[x1_aj]) / (2 * z)) ^ 2
x2_var <- ((x2[x2_n - x2_aj + 1] - x2[x2_aj]) / (2 * z)) ^ 2

# contrast coefficients, such that its median(d) - median(dg)
x1_cj <- 1
x2_cj <- -1

# median difference
mdn_diff <- x1_mdn * x1_cj + x2_mdn * x2_cj

# standard error
mdn_diff_se <- (((x1_cj ^ 2) * x1_var) + ((x2_cj ^ 2) * x2_var)) ^ (1 / 2)

# 95% confidence interval
lb <- mdn_diff - 1.96 * mdn_diff_se
ub <- mdn_diff + 1.96 * mdn_diff_se

# within roundng error of p. 376 of b&p 2002
paste0(mdn_diff, " [", round(lb), ", ", round(ub), "]")
