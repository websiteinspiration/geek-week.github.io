Pythonを使用して棒グラフを作成する方法

過去1年間、私はヒストグラムと棒グラフを頻繁に描画する必要に直面してきました。加えて、私自身はそのような情報がほとんど欠けていました。この記事では、Pythonでこのようなグラフを作成する3つの方法の概要を説明します。

そもそも、私は自分の経験が浅いため、長い間知りませんでした。棒グラフとヒストグラムは2つの異なるものです。主な違いは、ヒストグラムが頻度分布を示すことです。Ox軸に値のセットを設定し、頻度は常にOyにプロットされます。棒グラフ（英語の文学ではbarplotと呼ぶのが適切です）では、横軸と縦軸の両方を指定します。

デモでは、打ちのめされたscikit learn Irisライブラリデータセットを使用します。インポートから始めましょう：

import pandas as pd
import numpy as np
import matplotlib
import matplotlib.pyplot as plt
from sklearn import datasets
iris = datasets.load_iris()

irisデータセットをデータフレームに変換します。将来的には、それを使用して作業する方が便利です。

data = pd.DataFrame(data= np.c_[iris['data'], iris['target']], columns= iris['feature_names'] + ['target'])

関心のあるパラメーターのうち、データにはがく片と花びらの長さとはがきと花びらの幅に関する情報が含まれています。

Matplotlibを使用します。
ヒストグラムを作成します。
花びらとがく片の長さの頻度分布を示す通常のヒストグラムを作成します。

fig, axs = plt.subplots(1, 2)
n_bins = len(data)
axs[0].hist(data['sepal length (cm)'], bins=n_bins)
axs[0].set_title('sepal length')
axs[1].hist(data['petal length (cm)'], bins=n_bins)
axs[1].set_title('petal length')



棒グラフの作成

葉とがく片の幅を比較するために、matplotlibメソッドを使用します。これは単一のチャートで行うのに最も便利なようです：

x = np.arange(len(data[:50]))  
width = 0.35

たとえば、図を簡略化するために、データフレームの最初の50行を取り上げます。

fig, ax = plt.subplots(figsize=(40,5))
rects1 = ax.bar(x - width/2, data['sepal width (cm)'][:50], width, label='sepal width')
rects2 = ax.bar(x + width/2, data['petal width (cm)'][:50], width, label='petal width')

ax.set_ylabel('cm')
ax.set_xticks(x)
ax.legend()



私はseabornメソッドを使用します。

私の考えでは、ヒストグラムを作成するための多くのタスクは、seabornメソッドを使用して実行する方が簡単で効率的です（さらに、私の意見では、seabornもそのグラフィカル機能で勝っています）。

1行のコードでseabornで解決されたタスクの例を示します。特にseabornは、ディストリビューションを構築する必要がある場合の勝者です。がく片の長さ分布を作成する必要があるとしましょう。この問題の解決策は次のとおりです。

sns_plot = sns.distplot(data['sepal width (cm)'])
fig = sns_plot.get_figure()



配布スケジュールのみが必要な場合は、次のように実行できます。

snsplot = sns.kdeplot(data['sepal width (cm)'], shade=True)
fig = snsplot.get_figure()

パンダヒストグラム

すべてがここでは簡単です。これは実際にはmatplotlib.pyplot.hist（）のシェルですが、pd.hist（）を介して関数を呼び出す方が、matplotlib-aの俊敏性の低い構成よりも便利な場合があります。詳しくは、pandasライブラリのドキュメントをご覧ください。

それはこのように動作します：

h = data['petal width (cm)'].hist()
fig = h.get_figure()
