So wechseln Sie in nur fünf Schritten von einem Host-Anbieter mit cPanel zu Plesk in Rusonix

Es kommt vor, dass Leute ihren Hosting-Anbieter wechseln. Dies geschieht aus verschiedenen Gründen: Einige sind einfach unzufrieden mit der Geschwindigkeit oder Qualität der Dienste, andere finden die gleiche Qualität, jedoch zu einem niedrigeren Preis. Wieder andere mögen keine Bedienfelder ...

Wenn Sie sich in dieser Situation befinden, ist es jetzt an der Zeit, Ihre Websites und Projekte an Rusonix zu übertragen .

Das Verschieben Ihrer Website kann jedoch eine entmutigende Aufgabe sein. In diesem Handbuch wird eine einfache Vorgehensweise mit schrittweisen Anweisungen erläutert, damit Sie alle Ihre Daten sicher und einfach auf Rusonix- Server übertragen können .

Das Übertragen Ihrer Site kann in nur fünf Schritten erfolgen:

Schritt 1. Hochladen von Site-Dateien von einem früheren Host-Anbieter

Kopieren Sie die Site-Dateien mit dem FTP-Manager oder Dateimanager Ihres aktuellen Host-Anbieters auf Ihren Computer. Stellen Sie sicher, dass Sie ein Backup auf einem unabhängigen Medium erstellen.

Schritt 2. Exportieren der Datenbank Ihrer Website

Da Sie cPanel verwenden, ist das Exportieren von Datenbanken keine schwierige Aufgabe. Verwenden Sie dazu phpMyAdmin, das ein wesentlicher Bestandteil jedes cPanels ist. Seien Sie nicht beunruhigt, wenn Sie seltsame Namen von Dateien und Datenbanken sehen: Auch wenn Sie nicht alles verstehen, können Sie dies dennoch selbst tun.

  • Suchen Sie die Registerkarte, auf der sich die Datenbanken verstecken, und öffnen Sie sie.
  • Klicken Sie auf das Symbol phpMyAdmin, um die Benutzeroberfläche zu öffnen.
  • Suchen Sie und klicken Sie auf die Schaltfläche "Exportieren".
  • Wählen Sie "Schnellmethode - nur die Mindestparameter anzeigen".

Alles verlief reibungslos? Ihre Datenbank kann jetzt auf Rusonix-Server übertragen werden. Fahren Sie mit dem nächsten Schritt fort.

Schritt 3. Site-Dateien herunterladen

Öffnen Sie das Plesk-Bedienfeld in Rusonix und gehen Sie zum Abschnitt "Websites und Domains". Klicken Sie auf die Schaltfläche "Herunterladen" und wählen Sie ein gespeichertes Archiv mit Ihrer Site aus. Bitte beachten Sie, dass das Plesk-Bedienfeld nur Archive im Zip-Format unterstützt.

Wählen Sie das auf dem Computer gespeicherte Archiv aus und klicken Sie auf die Schaltfläche "Dateien extrahieren".

Schritt 4. Importieren von Datenbanken

Wie beim Export verwenden Sie MySQL zum Importieren der Datenbanken. Es ist eigentlich nicht so schwierig. Sie müssen auch phpMyAdmin eingeben, die Schaltfläche "Importieren" suchen und darauf klicken. Wählen Sie dann einfach die gespeicherte SQL-Datei auf Ihrem Computer aus und importieren Sie sie.

Schritt 5. Ändern Sie DNS

Dies ist ein kleines Detail, das leicht zu übersehen ist. Nachdem Sie Ihre Site übertragen haben, müssen Sie den DNS-Eintrag der Domäne vom vorherigen Server auf den Rusonix-Server umstellen. Ist dies nicht der Fall, werden Benutzer mit einem Fehler auf die Seite weitergeleitet. Das neue DNS ist zur Unterstützung von Rusonix leicht zu erlernen.

Nachdem Sie eine Anforderung zum Verschieben von DNS-Einträgen gesendet haben, kann es mehrere Stunden und in einigen Fällen einen ganzen Tag dauern, diese zu wechseln.

Wenn Sie aus irgendeinem Grund keinen Erfolg haben, diese. Der Rusonix-Support überträgt alle Ihre Daten unabhängig voneinander auf den Server .

Sind Sie mit Ihrem aktuellen Hosting-Anbieter nicht zufrieden? Verschwenden Sie keine Zeit - wählen Sie Rusonix! Hier finden Sie nicht nur eine leistungsstarke und schnelle Infrastruktur, sondern Sie werden auch angenehm überrascht sein von der Professionalität und der herzlichen, freundlichen Atmosphäre des technischen Supports.
