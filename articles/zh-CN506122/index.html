486-构建，程序和基准

我认为，关于组装计算机，写很多东西是没有意义的，通常“组装计算机”一词就足够了，不必将一个小时的视频上传到YouTube。但是，使用retroPC时，装配本身通常会变成一种冒险，因此我绝对想在某些时候停下来。
因此，这里将涉及我的程序集的功能，一些软件安装和一些基准测试。



铁


住房


该案子选择了标准AT，这样设计就不会大喊它是一个过时的系统-这意味着它需要一个Turbo按钮，一个锁以禁用键盘，最优选的是一个频率指示器。而且它处于正常状态，因为我的表壳看上去非常漂亮，但是它的框架却起皱和生锈，所以我真的不想使用它。因此，我选择了这一点：


我决定安装ATX电源-我对旧的ATX设备不太信任。但是，ATX也选择了比较老的产品，但是更换它会容易得多。尽管情况正常，但电容器似乎都完好无损，而且，额外提供了-5伏特的电压，这不是特别必要，但可以使其更好。


通过带有速卖通的基本适配器 “重制” AT中的ATX 只需3美元。

信息载体


就像我在上一部分中所说的那样，在这个问题上，我更喜欢真实性的便利。因此，我拒绝使用硬盘驱动器来支持存储卡，而拒绝使用3.5英寸驱动器来支持USB仿真器。我安装了5.25英寸驱动器，但它更适合随行人员使用-尽管从原理上讲，它是可行的。光盘驱动器仍然是非常现代的，尽管它是老式的设计-也用于外观。我不会使用它-由于存在一个可以快速取出的存储卡，请从您的大哥那里复制必要的东西，然后放相反的东西。

用于CF卡的适配器很流行，它安装在系统单元的背面，而不是插头。


但是它们有两个缺点：
1. IDE电缆的长度可能不足以同时到达后壁和光盘驱动器，并且您通常只有一个IDE通道。
2.如果系统部件未直接放在桌子上，则可能难以接近后壁。这就是我的工作方式。

因此，我决定将存储卡放在前面板上。在3.5英寸的托架中有一个现成的安装解决方案，但是它不是很容易使用-它发生在几个地方，而且相对昂贵。因此，您必须自己做一些事情。

我立即拒绝了在外壳上开孔的选择，我决定在3.5英寸隔室中发明一些东西。最简单的方法是在插头上开孔，然后将适配器拧到背面。但是我真的不喜欢它的外观，我决定采用另一种方式-装上3.5英寸驱动器并将适配器隐藏在其中。然后在外部看起来就像是普通驱动器-尤其是在未插入存储卡的情况下。

用螺丝刀和钳子做一点工作，用一点热胶水和双面胶带-“驱动器”就准备好了。
磁盘活动期间，与机箱同步，LED点亮。



从我的解决方案的缺点：
1）连接了IDE尾部，因此无法将第二个设备挂在其上。您可以决定用IDE扩展程序mom-dad代替此尾巴，以便将爸爸带出来。阿里300卢布，也许我会点。
2）门挂了，它的弹簧不再按了。原则上，它不是很吓人-当未插入卡时，它通常会挂在其重量下。
3）软盘的弹出按钮已完全变成装饰性的，只需要关闭孔即可。内部被胶水覆盖，完全不动。没有治疗。轻触按钮即可弹出存储卡，这是一个坏主意，但是使用杠杆系统会使设计变得非常复杂。
4）由于某种原因，这里我的外壳欠缺12伏电压-可能是适配器有故障，也许在组装某些东西时，它在不成功的地方刮擦了。但是，由于这里不需要12伏特，因此我最终没有连接该触点。

外观很适合我，让所有驱动器都出彩色。但是，布莱克（Black）显得特别突出-我稍后会进行更改。

记忆


我想输入32 + 32 + 8，但模块本身的故障和主板笑话已介入。
32兆字节的模块之一具有486，其工作容量只有一半-在pentium上，这是全部容量的正常现象。
两个相邻插槽中的母板仅接受相同的模块。也就是说，如果将插槽32和8分别放在1和2中，则在启动时会看到64和内存错误。如果您输入8和32，则我将看到16。3和4插槽的工作原理类似。因此，实际上，我只能在1个和3个插槽中使用不同的模块，而现在我必须腾出2个和4个空闲的时间。现在，我只放置了一个32 MB的模块。因此对486很有好处。

鼠标和键盘


通常，只需单击一下，retroPC的键盘就可以寻找一些古老的机械键盘。但是不知何故我不怀念他们。我有一个IBM Model M，但是现在我已经习惯了用小笔触来剪键盘，这让我使用Model M变得不舒服。因此，该业务的粉丝将其出售了，并用486通过适配器简单地连接了一个简单的PS / 2。她是哪年-我什至都不知道，但是她已经在家里躺了很长时间了。我认为，从零开始。


用鼠标，这个问题要复杂一些。鼠标的PS / 2通常只出现在具有PCI的主板上。在此之前，他通常只在品牌整合的平台上见面。因此，习惯上为COM端口寻找鼠标。对于球形鼠标，我完全没有怀旧之情，因此对于COM，我拥有带协调地毯的光学鼠标。它是零的产生，但是操作的原理是九十年代的事。我有一个天才的样子。


但是我在笔记本电脑上使用的鼠标更多（也有PS / 2，但是将键盘固定在其中通常更方便-因为我必须将鼠标插入COM）。因此，对于固定式计算机，您需要自己的鼠标，以免再次抽动。当然，可以再获得一个相同的东西，但是将来我想通过KVM切换器连接retroPC，因此我需要一个选项来通过PS / 2连接到486鼠标。

并且有这样的选择。一个有用的论坛使COM-PS / 2转换板非常有用。如果您想要方便，而不是真实性。它被插入到ISA 8bit中，挂在一个COM端口的地址上（我将其挂在COM2上，保留为COM1本机），并且可以正常工作。看来您甚至可以通过适配器插入USB鼠标，但是我还没有认真尝试过。我的剃须刀Deathadder没有启动，但是手边没有什么比这更简单的了，只有任何无线设备。旧的Genius Netscroll EYE毫无疑问可以正常工作。

结论结论


根据系统测试-组装后的系统以足够的速度工作，与其他DX2-66相比，它在任何地方都没有显着差异。

在视频卡上-如果将我的测试结果放在上面的旁边，那么Trident 9440可能是测试速度最快的视频卡。仅在狼中，CL-5429 fps稍好一些。


我的测试与Vogons论坛上该主题的测试一致：DOS中最快的VLB视频卡，那里的9440也是VLB卡中的佼佼者之一。不是领导者-也有更快的领导者，但他表现得相当不错。
