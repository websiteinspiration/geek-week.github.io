地球と生物の電磁界を同期させる秘訣



人体は非常に複雑なメカニズムであり、その働きは部品の完全性だけでなく、外部要因の影響にも依存します。 「天気に敏感」「関節が痛い」「脾臓が雨」などの言葉をよく耳にします。一見、これはあまり科学的ではないように見えますが、何かを説明するための非標準的なオプションを拒否するべきではありません。システムの動作とそれを取り巻く環境の間には、常に何らかの関係があります。問題は、それがどのように現れ、それを証明するかです。本日は、テルアビブ大学の科学者が最初に、生体の電場の活動と環境の電場との関係の証拠を明らかにした研究についてお話しします。電界の関係はどのくらい正確ですかなぜそれが必要で、どこから来たのですか？これについては科学者の報告から学びます。行け。


この研究の著者は、生物学において最も難しい問題の1つが、極低周波の範囲における電場の振動の起源の決定であることを示しています。この謎の最も驚くべき特徴の1つは、脳のサイズ、脳の複雑さ、または皮質の存在に関係なく、多くの生物種（脊椎動物と無脊椎動物）が同じ低周波電気活動を示すという事実です。つまり、人、犬、猫、ワタリガラスなどは、ほぼ同じ電気的活動をしています。

より具体的には、海洋に生息する動物プランクトンは、7および14 Hzにピークを持つ電気的活動を示します（1a）。より大きな脊椎動物と無脊椎動物（アシカ、ヘビ、サメ、タコ; 1b）は、主に50 Hz未満で発生するスペクトルも示しています。人間では、インジケーターも50 Hz（1秒）を超えません。興味深いことに、ほとんどの種で、活動の支配的なピークは8 Hzです。もちろん、異なる種の電磁活動には違いがありますが、多くの類似点があります。

最も明白な違いの1つは、スペクトルの振幅です。これは、脊椎動物のクラスによって異なります。さらに、最も高い振幅は哺乳類で見られます。脊椎動物のほとんどの場合、最大値は5〜15 Hzです。これは、より高い周波数では、オクターブごとに約半分、100 Hzでは約1/10に低下します。
オクターブ* -この場合、1オクターブが周波数の2倍に相当する場合、周波数間の関係の対数単位です。たとえば、40 Hzから1オクターブより大きい周波数は80 Hzです。
人の電気的活動のほとんどは、50 Hz未満の周波数範囲で発生し、波の種類に応じて次のように分布します。

  • アルファ波（8〜13 Hz）は、深いリラクゼーション、瞑想、ストレスリリーフです。
  • ベータ波（14〜25 Hz）は、通常の不安な精神状態を表します。
  • 知覚と意識に関連するガンマ波（30-100 Hz）。デルタ波（0.5〜4 Hz）は深い睡眠を表します。
  • シータ波（4〜8 Hz）は、創造力と夢の状態を表しています。

脳の波動活動は被験者の活動に応じて特定のモードに制限されるため、各瞬間に特定のリズムのみが観察されます。ここでは、スペクトルの性質が身体的および精神的活動に応じて変化する可能性があるという事実を追加できます。人間の脳の研究は、人が深い休息の状態にあるとき、アルファ波が支配的であることを示しました。体内のハロタン含有量（麻酔薬）が上昇すると、脳活動は10 Hzの通常のアルファ信号から7〜8 Hzの主な信号にシフトします（下のグラフ）。

科学者たちは、私たちの惑星の原始生命体が「深い平和」の状態に近い状態を示していると示唆しています。つまり、10 Hzに近い通常のアルファスペクトルよりも高いスペクトルをグラフ上に示します。

人間の脳は約26 Hz（1c）の活動の増加を示すことが多く、これは4番目のシューマン共鳴モード*の周波数に近いことを忘れないでください。

シューマン共鳴*は、地球の表面と電離層の間に低周波数と超低周波数の定在電磁波が形成される現象です。


ウィンフリートオットーシューマン
1952年にドイツの物理学者Winfried Otto Schumann（1888-1974）は理論を述べました：地球と電離層の高い伝導率を考えると、地球電離圏空間は電磁波の一種の共鳴を持つべきです。

シューマンは、これらの調和定在波は非常に低い周波数の範囲にあるべきだと計算しました。共鳴が損失なく（電離層での吸収なしに）存在すると仮定すると、彼は、共鳴周波数の最初のモードが10 Hzで発生するはずであると予測しました。 BalzerとWagnerはすでに1960年に最初のスペクトル測定を行い、電離層の部分的な吸収により、共振周波数が約8、14、20、26、... Hzで発生することを示しました。

これらのシューマン共鳴波の発生源は、全球的な雷雨の活動であり、電磁波は、電荷移動のいくつかの垂直成分を持つ雷チャネルから放出されます。

これらの周波数では、大気中の減衰はほとんどありません（0.1 km / mmまたは10,000 kmあたり1 dB）。したがって、電離層と地球の表面によって形成された自然の導波路により、惑星上の任意の点での稲妻からの非常に低い周波数の波は、他の場所に伝播することができます。これらの電波が地球（40,075 km）の周りを移動する際の建設的な干渉は、シューマン共鳴として知られる定在波とその調和（λ〜nc / 40,000）の出現につながります。

惑星で毎秒50から100の雷が発生することを考えると、バックグラウンドのシューマン共鳴場は常に大気中に存在します（上のグラフ）。

シューマンの共鳴スペクトルは、雷雨地域と比較して、時刻、年間の時間、地球上の相対的な位置に応じて、振幅と周波数が異なります。現在のところ、雷雨活動のほとんどは熱帯の陸域（東南アジア、南東アフリカ、南アメリカ）で発生しており、全世界の雷雨活動の10％のみが海で発生していることが知られています。

雷雨から数千キロを超える距離では、電磁界は主に水平磁場と垂直電場から構成されます。定常的なシューマン共鳴波のモード構造と電場と磁場の直交性により、熱帯から10,000 kmの距離でのシューマン共鳴は、磁場では8 Hzで最大を示しますが、電場では少なくとも8 Hzで示します。逆の状況は、雷雨地域から20,000 kmの距離で観測されます。

さまざまなシューマン共振モードの振幅の比率は、音源から観測者までの距離が変化すると変化します。その結果、観測期間全体を通じて全球の雷雨活動が一定であっても、シューマン共鳴スペクトルはすべての場所で同じになるわけではありません。


2017年のコリマ火山（メキシコ）の噴火（写真：セルジオタピロ/セルジオタピロ）。

シューマンの共鳴は、前世紀の半ばに発見されましたが、大気と電離層の形成以来、惑星に存在しています。当初、大気は火山からのガスの発生によって作成されました。今日でも、火山の噴火に雷が伴う様子を観察できます。しかし、初期の地球での自然の大気対流は、雲の電化と落雷の形成にもつながります。電離層、つまり、シューマン共鳴を作成するために必要な導波路は、太陽の放射によって支えられます。太陽放射は、上層大気の原子や分子と衝突し、イオンと自由電子を生成します。これにより、極低周波数範囲で電磁波が反射します。

したがって、研究者たちは、シューマン共鳴は生命の最初から、または少なくとも200万から300万年以上の間、私たちの惑星に存在すると結論付けています。

そして、ここで最も興味深いのは、シューマン共鳴の観測された周波数と生物の電気的活動との間に驚くべき類似性があるためです。科学者たちは、これが単なる偶然なのか、それとも以前に気づかれていない何らかのつながりがまだあるのか疑問に思っています。以前、彼らは人、鳥、さらにはハエで実験を行うことによってこの質問に答えようとしました。しかし、現代の科学者たちは過去の経験を分析し、おそらくそれを自分たちの発見で補足することを決定したため、答えはあまり理解できませんでした。

したがって、私たちはすでに雷雨の活動、つまりシューマン共鳴が太古の昔から地球上に存在していたことを知っています。数十億年。これにより、非常に低い周波数の自然な背景フィールドが惑星全体に維持されました。この自然界には、約8 Hzの基本モードを持つ特定の最大周波数があります。

これを知って、生物種がこの自然界を使用して自分のシステムを訓練できるかどうかについて質問できますか？可能であるだけでなく、この質問をする必要があることもわかりました。

自然界の多くの非線形効果の中で、同期はおそらく多くの異なるシステムで最も頻繁に観察される現象です。同期は、時間的に変動する2つのオブジェクト間の関係です。同期は、2つのオブジェクト間に固定された位相関係がある場合に発生します。

17世紀には、Christian Huygens（1629–1695）が同期の効果を発見した最初の人物でした。彼は、共通のサポートにぶら下がっている振り子時計が時間の経過とともに位相同期の状態に移行する、つまり振り子の振動が一致し始めると指摘しました。


共通のサポートとクリスチャンホイヘンスの肖像画のペアの振り子時計。

オブジェクト間は、ある種の接続である必要があります。これにより、オブジェクトが同期されます。時計の場合、この接続は、ある時計から別の時計へと壁（共通のサポート）を介して伝わる弱い振動でした。


クリスチャンホイヘンスの観測を示す7つのメトロノームの同期。

同期効果は多くのシステムに存在します。たとえば、生物学的システムでは、細胞集団、単一ニューロン、大規模ニューラルネットワーク、人間の心肺発達のダイナミクス、さらには個々の生物の集団的行動においても、微視的レベルで同期が存在する可能性があります。

したがって、同期は複雑なシステムの自己組織化メカニズムであり、環境との相互作用やサブシステム間の相互作用によるシステムの自由度を大幅に削減します。

古典的な同期理論は、いわゆる自立周期発振器で動作します。対応する振幅と周波数の外部周期的な力が自律発電機に作用する場合、システムの振動は外部信号と同相で同期されます。したがって、同期はより具体的には位相と周波数の同期として定義できます。

この定義から、研究者の理論は成長します。科学者たちは、生物系の進化の過程で、シューマン共鳴によって決定される大気の背景電場と位相を同期させることができると考えています。進化の間、特に初期段階では、シューマン共鳴はそのような同期に利用できる唯一の一定の電磁界でした。

さらに、初期の生命体が海洋に起源を持っていることを考えると、惑星の波長を持つ非常に低い周波数の波が海の光ゾーン（太陽によって照らされる上部の水柱）に数百メートルも侵入する可能性があることに注意する必要があります。

電磁波の皮膚を貫通する深さは、次のように定義されます。
d〜503 * sqrt（1 / f *σ）
ここで、σは導電率です（S / m、つまりメートルあたりのジーメンス）; fはHz単位の周波数です。

海水（σ= 3.3 S / m）と血液（σ= 0.7 S / m）の場合、電磁波（8 Hz）の侵入深度はそれぞれ約100 mと210 mです。

これは、海水中の光領域（深さ100 mまで）の生物がシューマン共鳴波を感知し、生物の内部が大気中に見られるものと同様の場の振幅によって影響を受けることを意味します。したがって、海洋の生物は常にシューマン共鳴場に曝されています。

確率的同期のアイデアは魅力的ですが、大気中のシューマンの共鳴場は非常に小さいです。磁場の振幅はピコテスラで測定されます（1 pT = 10 -12テスラ）、地球の準静的な地磁気フィールドよりも1000万倍弱いですが、電界はmV / mで測定されます。確率的同期でさえ、そのような小さな大気フィールドはどのように生物システムに影響を与えることができますか？

確率論的共鳴は、非線形システムが通常は検出されない弱い周期信号にさらされたときに発生しますが、確率的ノイズと弱い決定論的周期信号間の共鳴現象により検出可能になります。

確率的共鳴の以前の研究では、バックグラウンドノイズレベルの増加が出力信号強度の増加につながることが多いことが示されました。

ノイズはランダムまたは系統的です。通常、ノイズは信号の送信と検出に関連する干渉として認識されます。ただし、確率共鳴はその逆を意味します。実際、適切な量のノイズを追加すると信号が増幅されるため、ノイズの多い環境での信号の検出に役立ちます。

外部ノイズの振幅をシステムの内部プロパティに調整することにより、周期的励起のメカニズムと外部ノイズが相互に作用し、エネルギーをノイズスペクトルから信号と一致する単一の周波数に転送できます。外部ノイズと信号の間のこの相互作用は、出力信号のパワースペクトルに明確な最大値をもたらし、信号対ノイズ比を増加させる可能性があります。ただし、ノイズの振幅も重要であり、ノイズが大きすぎると信号が乱されます。

研究の著者は、非常に低い周波数のフィールドと雷によって引き起こされるシューマン共鳴が、確率的共鳴の現象を通じて生物システムによって使用される「ノイズ」として作用する可能性があることを示唆しています。何百万年にもわたる進化の間のこの一定のノイズ源は、生物系の発達に影響を与え、生物の電気的活動を大きく左右する可能性があります。

上記の理論を確認することになっていた人々との実験がかつて行われたことを私たちは知っています。そのため、1973年に、概日リズム（24時間の人間の生体リズム）で実験が行われました。地下には2つの同じ部屋が建てられましたが、窓やドアはなく、視覚的に時刻を判断することは不可能でした。各部屋にボランティアが配置され、そのような状況で約1か月間生活しました。科学者たちは実験参加者の活動（睡眠と覚醒）と体温を追跡しました。

人が昼と夜の変化を見ることができるとき、これらの変数はかなり予測可能です。ただし、視覚信号がない状況では、被験者の体内時計が1日を25、26、および最大27時間まで「伸ばし」始めました（下のグラフ：X軸は時間、Y軸は月の日です）。

実験の最初の週に、被験者で観察された体内時計が1日あたり26.6時間に変化しました。次に、部屋の1つで、2週間目に、10 Hzの周波数の電界発生器を継続的にオンにしました。体内時計は明らかに安定しており、通常の毎日のリズムに戻ろうとしました（最大25.8時間の減少が観察されました）。 1週間後、フィールドはオフになり、生物時計は繰り返し、実際の毎日のリズムから1日あたり36.7時間にずれ始めました。

一方、外部電界に曝されていない2人目の被験者の体内時計は、3週間すべて安定しています。

この実験は繰り返されましたが、鳥が参加しました。結果は人間で観察されたものと同様でした-10 Hzの電気信号の影響による概日リズムの変化がありました。

8 Hzではなく正確に10 Hzを使用するのは、電離層に反射のエラーがないため、シューマン自身が非常に低い周波数の共振は正確に10 Hzであると当初信じていたためです。もちろん、これはそうではありません。最初のモードの真の周波数である8 Hzまたは7.8 Hzを使用する必要があったからです。

2016年には、さらに珍しい実験が行われ、脊髄損傷のラットが参加しました。実験用ラットは、2つの異なる周波数の磁場にさらされました：15.72（最初のシューマン共鳴モードの2倍）と26 Hz（4番目のシューマン共鳴モード）。

磁場は1日8分、週5日1ヶ月間適用されました。翌月、曝露時間は1日あたり20分、週5日に延長されました。

一般に、両方のグループのラットは、磁場が印加されていないコントロールグループのラットと比較して、大幅に速く回復しました。 15.72 Hzのフィールドの場合、60日の観察後に復元は限界に達しましたが、26 Hzの場合、復元は継続しました（下のグラフ）。

さらに、脳卒中のラットで同じ実験を行いました。この場合、復元の最良の結果は、0.5 x 7.8 Hzと2 x 7.8 Hzの周波数を示しました。

上記の実験は、この分野で蓄積されたすべての知識を考慮に入れた現代の実験をセットアップするための重要な歴史的経験です。

今日検討している研究の著者は、ラット心臓の筋細胞（筋肉細胞）に対する7.8 Hz磁場の影響を分析しました。磁場は3〜4日齢の細胞に影響を与えた。

観察はいくつかの段階で行われた。最初の段階では、科学者は光学顕微鏡を使用して、（磁場のある場合とない場合の）心臓細胞の自発的な機械的収縮を単純に観察しました。第二段階は、Ca +による自発的過渡現象の観察に専念しました。 3番目の段階は、低酸素またはH 2 O 2の追加によって引き起こされるストレスによる細胞損傷の研究です。

磁場の印加後30〜40分以内に、自然収縮が停止し、Ca +に沿った遷移プロセスが80％減少しました。最も興味深いのは、磁場が、ストレスによって引き起こされた損傷を、対照群と比較して約40％削減したことです。

これは、外部のシューマン共鳴場がストレスの状態で細胞の保護膜の役割を果たすことを示している可能性があります。

研究のニュアンスの詳細については、科学者のレポートをご覧になることをお勧めします。

エピローグ


著者は彼らの仕事が挑発的であると呼ばれることができるという事実を隠さない。一部の人にとっては奇妙で論理がないように見え、一部の人にとっては革命的です。そして、ここで1つの位置を選択することは困難です。これは、研究で考慮される科学の側面がその秘密を明らかにすることに非常に消極的であり、そこからそれらに基づく研究が客観的に判断することが非常に難しいためです。

それにもかかわらず、外部の電磁界と生物系の働き、つまり生物との関係の存在を否定することはできません。
