火から火へ：リチウムイオン電池用の耐火性固体電解質



現代の技術では、スマートフォンから電気自動車まで、リチウムイオン電池が使用されています。このタイプのエネルギー源にはいくつかの利点（大容量、低自己放電など）がありますが、欠点もあります。また、低温に長時間さらされている間に容量が失われることが厄介であるが、世界の終わりではない場合、可燃性は深刻な問題です。リチウムイオンバッテリーの発火の可能性は、損傷または欠陥がある場合に発火する可能性のある液体電解質がその中に存在することによって説明されます。この問題の解決策の1つは、電解液に特殊な難燃剤（燃焼プロセスを遅くすることで耐火性を提供する物質）を導入することでした。ただし、このホットな問題に対するより興味深い解決策は、液体電解質を固体（SSE-固体電解質）に置き換えることです。問題は、理論的にはより高い耐火性を提供するはずのSSEは、その脆弱性を低減するために使用される材料を考慮すると、それ自体が非常に可燃性であることです。問題はそこにありますが、アメリカ化学協会（ACS）の科学者たちは解決策を見つけました。新しいタイプのSSEの作成に使用された材料、新しいタイプのバッテリーの特性、およびそれに耐えられる温度は？これについては研究グループの報告書から学ぶ。行け。新しいタイプのSSEの作成に使用された材料、新しいタイプのバッテリーの特性、およびそれに耐えられる温度は？これについては研究グループの報告書から学ぶ。行け。新しいタイプのSSEの作成に使用された材料、新しいタイプのバッテリーの特性、およびそれに耐えられる温度は？これについては研究グループの報告書から学ぶ。行け。

研究の基礎


私たちが知っているように、リチウムイオン電池（以下、LIA）は文字通りどこにでも存在しています。LIAに対するこのような需要の増加により、メーカーや科学者は容量の点で品質を向上させる新しい方法を模索し始めました。誰もがバッテリーを長持ちさせ、よりゆっくりと放電し、より速く充電することを望んでいるからです。ただし、バッテリーの寿命と「電力」の追求により、安全性の問題、特に火災の問題は回避されています。研究の著者は、高電流密度でサイクルごとに徐々に増加するリチウムデンドライトが、電池の電極を分離するセパレーターを貫通し、短絡を引き起こす可能性があることを指摘しています。

液体電解質内の発火の問題には多くの解決策があります：セパレーターをセラミック粒子でコーティングする、電解質自体に難燃剤を導入する、内蔵の温度調節器、ポリマーによる耐火カプセル化など。

液体電解質から固体状態にすると、リチウムの不均一な堆積による樹枝状* Liの形成に問題が生じます。

デンドライト* -枝分かれした木のような複雑な結晶構造。
このようなリチウム「石筍」は、セパレーターやカソードにまで浸透する可能性があり、短絡、過熱、火災、さらには爆発に至る可能性があります。このようなプロセスの危険性に加えて、リチウムデンドライトが形成される電池の効率にも悪影響があります。

現在、既存の固体電解質は、無機（セラミック/ガラス）、ポリマー（SPE）、ハイブリッドの3つの主なカテゴリに分類できます。

無機固体電解質は、すべてのタイプのSSEの中でイオン伝導率が最も高いという点で興味深いものです。以前の研究では、Li 9.54 Si 1.74 P 1.44 Sの伝導率が25 mS cm cm -1の超イオンリチウム導体について説明しました液体電解質の性能を超える11.7 Cl 0.3。
シーメンス（cm）-電気伝導率の単位（1 cm = 1 /オーム）; mS（ミリ秒）= 10 -3 cm。
ただし、空気の不安定性、材料の脆弱性、大きな界面インピーダンス、およびいずれの場合でもLiが臨界電流密度に達した後に無機SSEに浸透するという事実は、リチウムイオンバッテリーでの無機SSEの完全な使用を妨げます。

ポリマー固体電解質（SPE）について話す場合、それらは主に固体ポリマーとリチウム塩で構成されており、固体不純物はリチウムイオン伝導体として機能します。現在最も研究されているのは、リチウム塩とポリエチレンオキシド（以下、PEO）の組み合わせです。この構造は、低コスト、高リチウムイオン伝導性（他のSPEと比較）、およびかなり軽量であり、ポータブルデバイスにとって重要です。ただし、このポリマーシステムの内部の柔らかさのため、リチウムデンドライトの広がりを抑えることができません。つまり、可能性はあるが、他のタイプの固体電解質に存在する必要な問題を解決しない。

彼らはこの問題を解決するために、ナノ粒子で補強し、架橋し、「柔軟な」電解質を硬い担体に結び付けました。これらの複雑な操作にもかかわらず、得られた複合ポリマーSSEは依然として可燃性のままです（1a）。

研究成果


複合SSEは、多孔性の二官能性ポリイミド（PI）とリチウムイオン伝導性SPEフィラーから作られました。 2機能性ベースは、厚さ10ミクロンの耐久性のある多孔性ポリイミド（PI）フィルムと軽量の難燃性デカブロモジフェニルエタン（DBDPE）で構成されています。後者は非常に耐久性があり、リチウムデンドライトの潜在的な浸透を確実に防止するだけでなく、耐火性もあります。

フィラーはPEO / LiTFSIで構成されており、高いイオン伝導率のSSEを提供します。

複合電解質のポリマー-ポリマーの性質は、完全に充電されたバッテリーに潜在的に高いエネルギー密度を提供します。つまり、このSSEは耐火性であるだけでなく、バ​​ッテリー容量も増加させます。

PI / DBDPE / PEO / LiTFSI固体電解質バッテリーで熱加速が発生すると、難燃性の不燃性PI DBDPEは、可燃性PEO / LiTFSIの燃焼を効果的に抑制します。

最初に、ポリアミック酸（PAA）とDBDPEの溶液が準備されました。次に、スキージを使用して溶液をガラス基板に塗布し、PAA / DBDPEフィルムを得た。 PAA / DBDPEの多孔度を取得するために、ジメチルアセトアミドとエタノールの溶液（DMAC / EtOH）を使用しました。乾燥が完了したら、PAA / DBDPEフィルムを300°Cでイミド化（アミド酸からポリイミドへの環化）して、最終的な多孔性PI / DBDPEフィルムを得ました。

乾燥後、PAA / DBDPE多孔質フィルムを300°Cでイミド化して、最終的なPI / DBDPE多孔質フィルム（写真フィルムで1秒間）を得ました。
