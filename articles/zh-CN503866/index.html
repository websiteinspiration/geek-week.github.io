发布了2020年路线图开发Zextras Suite

早些时候，我们在2020年初谈论了Zextras Suite Pro和Team Pro的创新。Zextras已发布了2020年Zextras Suite新功能开发的路线图  。


6月-小组：新功能（已改进）

  1. 回复邮件（回复）
  2. 清除聊天记录
  3. 编辑讯息
  4. 删除留言

八月-团队：视频服务器（NEW）

Zextras Team中内置视频服务器的外观将使您能够引入许多可以直接从管理员控制台控制的新功能，并提高了服务的生产率。音频和视频流将被压缩，优化和合并，以便为每个设备获得单独的流。此外，服务器将自动为每个设备选择适当的编解码器和比特率，以确保最高的通信质量。

8月-外部端点上的备份（NEW）

Zextras Backup将为您提供一个将系统备份到本地，混合或云S3兼容存储或BLOB元素外部存储（例如NFS）的机会。同时，本地元数据缓存将允许维持原始性能，尽管事实是在需要灾难恢复的情况下，它们本身也将存储在外部端点中。

9月-Zextras身份验证和登录（NEW）两因素

身份验证，联合SAML身份验证和授权系统以及多个移动密码将添加到Zimbra Collaboration Suite开源版中。更新的登录页面也会出现，并且可以进行自定义。

9月-小组隐私设置（新）

用户和管理员将能够选择是否显示有关其状态的信息。特别是，可以隐藏有关阅读消息，用户状态等的信息。9月-移动应用程序QR认证（新） QR登录系统使您可以使用移动应用程序，而不必通过使用交叉身份验证系统输入密码以及管理其注册和授权。9月-用于MS Outlook / MAC Mail / Thunderbird的Zextras通讯簿（新） 将Microsoft Outlook，Thunderbird和Mac Mail连接到集中式Zimbra GAL和您的通讯簿，以实现全面的用户便利。










通讯簿可以与IMAP / POP或EAS结合使用。 Microsoft Outlook可以使用EAS和通讯簿来提供与Outlook的完全交互，而无需在客户端中安装任何插件！

10月-Zextrasconfig基础结构管理器（NEW）

可以通过特殊的Zextrasconfig管理器管理基础结构中的所有Zextras组件。该管理器使Zimbra管理员可以管理各种基础结构节点上的Zextras组件，包括更新的集成Zextras Docs服务器和集成Zextras Team视频服务器。

十月-管理界面和用户首选项（重新整理）

新的Zextras组件管理界面是用React编写的。从Zextras的组件开始，这些组件将继续在熟悉的管理界面中显示，我们将所有选项转移到新技术中，而无需从熟悉的界面中删除它们。将来将进行全面过渡。

10月-使用本地Linux打包系统（NEW）的

新安装程序新的Zextras软件包将通过具有本地Linux支持的存储库分发。在安装和更新软件包时，可以使用“ apt-get install / update”之类的系统命令。

11月-文件服务器（新）

Zextras Docs服务器将成为基础架构的一部分，通过Zextrasconfig集中式基础架构管理管理器进行管理。不再需要在单个节点上安装和更新服务器。

11月-团队：会议的新功能（NEW）

由于有了新的视频服务器，Zextras团队中将出现许多新功能。其中：

  • 主持人模式
  • 能够使特定用户或所有用户立即静音
  • 启用网络研讨会模式和视频会议模式的能力
  • 事件日志
  • 聊天请求
  • 对讲机模式
  • 环聊记录可以上传到Zextras驱动器

11月-高可用性（NEW）

能够在不同基础结构节点之间同步用户，组，域甚至整个邮件服务器。能够配置与集群中的备用节点或另一个活动节点的同步。该系统旨在实现每月99.99％的可用性。

11月-驱动器个人外部帐户（IMPROVED）

能够连接Dropbox / Gdrive / nextCloud / OwnCloud帐户

12月-团队：即时会议（IMPROVED）

能够创建即时会议并使用永久和临时链接提供访问权限。

12月-电子邮件签名和加密（S / Mime-PGP）（新）

端到端加密将在Web客户端中实现。该系统将能够对发送的电子邮件进行加密和/或签名，以解密并验证接收到的电子邮件的签名。

到目前为止，这些功能中的许多功能都是Zimbra商业版所独有的。今年以来，Zimbra协作套件开源版的用户将能够在购买Zextras Suite时利用这些功能和其他功能。将来，我们将详细讨论Zextras Suite的新功能。
