今少し理論。

ブロードキャストを送信できない組み込みのバーコードスキャナーがあります。また、キーボードエミュレーションを備えたバーコードスキャナーもあります。この方法も彼らに適していると思います。どちらの場合も、たとえば「プレフィックス」を設定することをお勧めします"="(KEYCODE_EQUALS)。そして、サフィックスをに置き換え"CR"(KEYCODE_ENTER)ます。他のすべてのサフィックスとプレフィックスを削除します。標準設定でキーボードエミュレーションを備えたほとんどのスキャナーにはプレフィックスがありませんが、喜んで返します CRLF（2文字。キャリッジリターン、ラインフィード）。歴史的に起こったようです。

今度は実装自体です。MainActivityクラスで、2つの変数を作成します。1つはバーコード文字の連結を担当し、2つ目はプレフィックスとサフィックスの間の行でバーコードを収集するためのものです。

val barcodeStringBuilder = StringBuilder()
var isReadingBarcode = false

そしてクラスでは、関数を再定義する必要がありますonKeyUp。

override fun onKeyUp(keyCode: Int, event: KeyEvent?): Boolean {
// ,  ().   .         .
        if (event?.keyCode == KeyEvent.KEYCODE_EQUALS ){
            barcodeStringBuilder.clear()
            isReadingBarcode = true
            return true
        }
//     .  ,      .
        if (isReadingBarcode) {
            barcodeStringBuilder.append(event?.getUnicodeChar()?.toChar())
        }

//    ,      .       .
        if ((event?.keyCode == KeyEvent.KEYCODE_ENTER) && isReadingBarcode) {
            Log.d("BarcodeDebug", "$barcodeStringBuilder")
            setTextBarcode(barcodeStringBuilder.toString())
//  .    .
            isReadingBarcode = false
            return true
        }
//     .
        return super.onKeyUp(keyCode, event)
    }
