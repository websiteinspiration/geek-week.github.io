就職の面接とPythonタスクの説明

敬礼、ハブロバイト！コース「Web-developer in Python」の新しいスレッドの開始を見越して、新しい便利な翻訳を共有したいと思います。行こう！いくつかの面接に



再度行ってテストタスクに合格したので、面接担当者は次のようなタスクを好むことに気づきました。

def f(x, l=[]):
    for i in range(x):
        l.append(i * i)
    return l
>>> f(2)
>>> f(3, [0, 1, 2])
>>> f(3)


質問：このコードは何を出力しますか？

最初の2行の出力は明白ですが、3行目の結果はf(3)私にはそれほど簡単に思えませんでした。関数が初期化された後に何が起こるか見てみましょうf。このコードを実行するには、IPythonを使用します。

>>> f
<function __main__.f(x, l=[])>
>>> f.__defaults__
([],)


実行結果から表示される空のリストは、関数コードのf.__defaults__変数lです。

>>> f(2)
[0, 1]


特にない。

>>> f
<function __main__.f(x, l=[0, 1])>
>>> f.__defaults__
([0, 1],)


しかしながら！今、私たちは、変数がいることがわかりl価値あり[0, 1]でのおかげ参照してPythonのリストオブジェクトと伝達関数の引数の変動を。

>>> f(3, [0, 1, 2])
[0, 1, 2, 0, 1, 4]
>>> f
<function __main__.f(x, l=[0, 1])>


特別なこともありません。オブジェクトlistを変数として渡すだけですl。

>>> f(3)
[0, 1, 0, 1, 4]
>>> f
<function __main__.f(x, l=[0, 1, 0, 1, 4])>


そして今、最も興味深い。を起動するとf(3)、Pythonは関数コードで定義されている空のリストを使用せず、lfromの値を持つ変数を使用しますf.__defaults__ ([0, 1])。

PS

各呼び出しの後に空のリストを使用する関数が必要な場合は、このようなものを使用する必要があります（値‘l’をに設定‘None’）。

def f(x, l=None):
    if l is None:
        l = []
    for i in range(x):
        l.append(i * i)
    return l
>>> f(2)
[0, 1]
>>> f(3, [0, 1, 2])
[0, 1, 2, 0, 1, 4]
>>> f(3)
[0, 1, 4]
