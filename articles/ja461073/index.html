OverpassTurboとは何ですか？


そう。OpenStreetMapsなどの地図データベースがあります。海、大陸の輪郭、山、森、道路、建物、遊び場、さらにはスピードバンプまで、すべてが含まれています。各オブジェクトには、名前、座標、プロパティがあります。たとえば、道路-コーティング材、建物-階数など。


だからここに。今日インターネットで提供されているカードのほとんどは、この特定のデータベースに基づいて生成されています。しかし、これらすべての既製のカードが私たちに合わない場合はどうなりますか？自分で作れます！まあ、または少なくとも既存のものを補完するほうがはるかに簡単です。


これがOverpassTurbo.euが行っていることです。それはオンラインIDEです。これを使用して、OSMデータベースを照会できます。開始ボタンをクリックすると、リクエストがデータベースに送られ、しばらくするとデータが返されます。 OverpassTurboは、このデータを、背景レイヤー（OpenSteerMap.orgのマップ）の上にあるベクターマーカーとラインの形で視覚化します。

app.get( '/:x/:y/:z/:crossZoom', async ( req, res, next ) => {

    const x = req.params.x
    const y = req.params.y
    const z = req.params.z
    const crossZoom = req.params.crossZoom
    const scriptName = req.query.script

    //    
    if ( Number( z ) < Number( crossZoom ) ) {
        res.redirect( `http://tile.openstreetmap.org/${z}/${x}/${y}.png` )
    }

    //     
    const screenshot = await workersPool.exec( { x, y, z, scriptName } )

    const imageBuffer = Buffer.from( screenshot, 'base64' )

    res.writeHead( 200, {
        'Content-Type': 'image/png',
        'Content-Length': imageBuffer.length
    })

    res.end( imageBuffer )
})

})

async function doMyAsyncCode( params ) {
    await prepareEnviroment()

    //

    return screenshot
}

async function prepareEnviroment( ) {
    if ( browser === "empty" ) {
        const herokuDeploymentParams = {'args' : ['--no-sandbox', '--disable-setuid-sandbox']}
        browser = await puppeteer.launch( herokuDeploymentParams )
    }
