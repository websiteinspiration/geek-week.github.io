Minhas principais ferramentas gratuitas para desenvolvedores

Esta é uma tradução do artigo de dev.to. Seu autor Fayaz Ahmed compartilha uma lista de suas ferramentas de desenvolvedor favoritas.



Desenvolvo projetos como Tvflix , GSON , Appy Dev , além de muitos microprojetos diferentes, por isso estou interessado em encontrar ferramentas gratuitas para o meu trabalho.

Esta é a minha lista pessoal, mas você pode complementá-la com outras ferramentas bem-sucedidas. Aqui estão alguns dos meus favoritos.

Serviços e plataformas para implantar seu front-end / back-end


1. Netlify
Use o Netlify para implantar sites estáticos modernos. Você obtém infraestrutura CDN, implantação contínua, HTTPS com um clique e quaisquer serviços necessários. Comece gratuitamente.

2. Vercel (anteriormente Zeit)
Vercel é a maneira mais fácil de implantar sites. Hospede seus projetos da Web de configuração zero com SSL automático e infraestrutura global de CDN.

3. Os projetos da Web para encaminhamento de surtos
devem ser rápidos, fáceis e seguros. Surge é um recurso estático de publicação na web para desenvolvedores front-end diretamente da CLI.

4. Firebase
Uma alternativa ao Netlify / Zeit do Google. O Firebase é uma plataforma móvel que ajuda você a desenvolver rapidamente aplicativos de alta qualidade e expandir seus negócios.

5. Aplicativos estáticos da Web do Azure A
partir de hoje, você tem a oportunidade de usar os Aplicativos estáticos da Web para hospedar aplicativos de página única no Azure.

6.
Sites do GitHub Pages para você e seus projetos publicados diretamente do seu repositório GitHub. É suficiente editar e executar para que as alterações sejam transferidas para a versão ao vivo.

7. Páginas do GitLab
Como o GitHub, o serviço GitLab Pages permite publicar sites estáticos diretamente do repositório no GitLab.

8. Acrobático
Hospedagem estática para serviços pesados. Funciona muito bem com geradores de sites estáticos, como Hugo, Jekyll, Gatsby, Gridsome.

9. Falha
Talvez essa seja uma das ferramentas que mais uso hoje em dia. Um serviço gratuito simples e poderoso para criar e hospedar aplicativos baseados no Node.js.

10. Heroku
Heroku, uma versão mais avançada do Glitch, é uma "plataforma como serviço" (PaaS) que permite aos desenvolvedores criar aplicativos, executá-los e gerenciá-los completamente na nuvem.

Ferramentas de automação


1. Cron-job.org O
Cron-job é um serviço de código aberto gratuito e permite executar um número ilimitado de tarefas no formato cron. Tudo que você precisa é usar sua API ou o endereço de email do seu webhook, que será usado para conectar-se ao serviço com a frequência especificada.

2. n8n
Essa ferramenta de código aberto é uma alternativa ao Zapier. O serviço oferece possibilidades ilimitadas, atualizando constantemente a lista de ferramentas e aplicativos disponíveis para sincronização. Você também pode hospedar o n8n em seu próprio servidor.

3. Integromat
O Integromat é uma ferramenta fácil de usar e eficaz para automatizar processos manuais com características únicas. Sincronize seus aplicativos, serviços e dispositivos favoritos, mesmo que você não tenha habilidades de programação.

4. IFTTT
Experimente o IFTTT - a maneira mais fácil e gratuita de fazer mais com seus aplicativos e dispositivos favoritos. As tarefas domésticas exigirão menos atenção, o trabalho se tornará ainda mais produtivo, você manterá a privacidade e seus dados estarão seguros. Estou certo de que tudo funciona melhor em condições de sincronização.

5. Pipedream
O Pipedream é uma plataforma na qual os desenvolvedores podem criar e publicar projetos relacionados ao processamento de dados. Abra projetos e bifurque-se em uma plataforma que contém milhares de exemplos de código-fonte e projetos concluídos.

Ferramentas sem código


1. NoCodeAPI
A maneira mais fácil para um desenvolvedor front-end conectar seu aplicativo a serviços de terceiros sem escrever código adicional e um servidor dedicado. NoCodeAPI pode trabalhar com o Google Sheet, Airtable, Google Analytics, Twitter, Instagram, XML, RSS, JSON, Slack, sistemas de pagamento e mais de cinquenta serviços.

2. GlideApps
Glide transforma tabelas dinâmicas em aplicativos fáceis de usar, sem a necessidade de código. Selecione uma tabela ou comece com um modelo, defina as configurações do seu aplicativo e compartilhe-as com outras pessoas em apenas um momento. Experimente gratuitamente hoje!

3. Airtable
O Airtable funciona como uma tabela dinâmica, mas ao mesmo tempo fornece recursos de banco de dados para organizar qualquer conteúdo.

4. Bubble O
Bubble apresenta uma nova maneira de criar aplicativos da web. Esta ferramenta de programação não requer nenhum trabalho com código e opera no formato apontar e clicar. O Bubble hospeda todos os aplicativos em sua plataforma em nuvem.

5. Gerador de ícone de material
Gere ícones SVG no estilo de design de material para suas aplicações. Um serviço de código aberto.

Ferramentas para SEO / Marketing na Internet / Promoção / Suporte jurídico


1. Meta Tags
Gere e visualize previamente todas as meta tags e tags para redes sociais em uma página da web.

2. Ferramentas de código da Web
Crie esquemas, tags e muitos trechos de código repetidos.

3. Carbono O
carbono é a maneira mais fácil de criar e publicar imagens visuais do seu código-fonte.

4. Codeimg
Crie e publique imagens visuais do seu código fonte.

5. GetTerms Gere o
texto base do Contrato do Usuário e da Política de Privacidade do seu site.

6. SEM ABELHAS
O BEE é um editor de email que permite criar ótimos emails de resposta rapidamente e sem problemas. Seus autores procuraram desenvolver o editor de email perfeito. É possível usar o BEE online ou incorporar em seu aplicativo.
