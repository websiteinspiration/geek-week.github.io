Kampus Digital Banking - hasil program magang

Teman, salut!

Pada bulan Desember, kami menyelesaikan rekrutmen untuk program magang TI Kampus Digital Banking. 3 bulan berlalu, peserta pelatihan kami menjadi sedikit terbiasa, dan kami meminta mereka untuk berbagi cerita.



Jalur Yegor, Anna dan Alexei, trainee kami, ke bank spesialis IT berada di bawah jalan buntu.

Egor, pengembang tumpukan penuh




Saya memulai jalur saya di TI dengan keamanan informasi. Saya memasuki Fakultas "Keamanan Informasi Sistem Resmi" dan berharap bahwa akan ada banyak disiplin ilmu terkait dengan arah "Keamanan Komputer", tetapi dalam praktiknya ternyata dalam keamanan informasi banyak perhatian diberikan kepada manajemen keamanan informasi.

Saya menyadari bahwa saya tidak terlalu tertarik untuk mengelola, dan sekitar setahun yang lalu saya mulai belajar dalam bahasa pemrograman yang mendalam, serta beberapa bagian keamanan komputer. Sekarang saya mengerti bahwa pengembangan (apakah itu pengembangan web atau pengembangan sistem keamanan informasi apa pun) menarik saya lebih banyak.

Tahun lalu di bursa kerja, ia belajar tentang magang Kampus Digital Banking dan memutuskan untuk mencobanya. Saya tertarik pada kenyataan bahwa Anda dapat mencoba mengerjakan proyek nyata dalam tim pengembangan nyata.

Saya suka bekerja di Rosbank. Kami memiliki kantor yang luas dan indah dengan tempat kerja yang nyaman. Ada mesin VkusVill, kedai kopi, dan berbagai area relaksasi, bahkan kamar dengan PlayStation.



Pada awalnya, kolega saya dan saya bekerja pada layanan web uji kecil yang ditulis menggunakan bahasa pemrograman Python dan kerangka Botol. Kami sedang aktif mengembangkan layanan web. Saya merasakan kemajuan saya, dalam 3 bulan saya berhasil memompa pengetahuan teknis saya. Saya sangat suka pemimpin tim kami, orang yang bertanggung jawab dan tekad.

Anna, pengembang backend




Saya lulus dari sarjana ilmu politik Universitas Negeri Moskow, akan menjadi pengacara dan memasuki magistrasi Fakultas Hukum Internasional MGIMO, dengan jurusan “Hukum Keuangan”. Sebagai bagian dari kursus, bagian dari kuliah diberikan oleh para ahli dari Bank Sentral, dan dari mereka saya pertama kali belajar tentang fintech. Saya menjadi tertarik, dan saya memutuskan untuk mulai coding dalam javascript dan php. Awalnya, itu hanya hobi, karena Pada saat yang sama, saya bekerja di sebuah firma hukum Italia, tetapi pada akhir studi saya di MGIMO saya menyadari bahwa fintech jauh lebih menarik. Saya keluar dari firma hukum dan masuk Universitas Geek (Geek Brains) dari Mail.ru. Untuk studi, saya memilih tumpukan teknologi yang lebih kompleks - Java / Spring, yang aktif digunakan di fintech.

Pada musim gugur, di salah satu saluran di Telegram saya melihat posting tentang merekrut Kampus Digital Banking untuk program magang Rosbank. Magang ini melibatkan bekerja dengan microservices - sebuah pendekatan di mana aplikasi dipecah menjadi beberapa aplikasi kecil, yang menyederhanakan dukungan dan menambahkan fitur baru. Teknologi untuk pendekatan ini terus berkembang dan merupakan yang terbaru dan paling menjanjikan dalam pengembangan perusahaan. Saya tidak ingin ketinggalan pengalaman seperti itu, saya memutuskan untuk mencoba dan tidak kalah.

Sekarang saya telah berada di Rosbank selama beberapa bulan. Saya bekerja di kantor di Armagh. Ruang sejuk dan kreatif, dekat dengan Winery dan ArtPlay. Segera, ketika pembatasan diberlakukan karena virus berakhir, kami akan pindah ke kantor keren baru di Kota Moskow.



Tim saya dan saya sedang mengerjakan proyek untuk membuat aplikasi mobile bank baru untuk perorangan (Anda akan melihatnya segera), kami mentransfer monolith ke arsitektur layanan mikro. Saya terlibat dalam pengembangan backend. Dalam waktu singkat ini, kami berhasil mempelajari beberapa teknologi baru yang digunakan pada proyek (dari tumpukan Cloud Spring, kerangka kerja untuk mengembangkan layanan Microsoft), dan untuk mempelajari arsitektur solusi. Salah satu kesulitan utama pengembang backend java adalah Anda harus memiliki banyak teknologi, tahu banyak kerangka kerja. Pada proyek, ini terasa sangat tajam, jadi Anda harus belajar tanpa henti, bahkan dalam kondisi saat ini pekerjaan jarak jauh, yang tidak selalu sederhana, tetapi cukup nyata.

Untungnya, kolega senior saya selalu siap membantu, menjelaskan poin-poin sulit. Proyek ini memiliki pemimpin tim, mengelola tim scrum, arsitek utama. Mentor saya adalah pengembang senior yang juga mengarahkan pekerjaan programmer lain. Kami memiliki tim yang sangat keren - orang-orang keren, profesional di bidangnya, dengan siapa Anda dapat berkonsultasi dan hanya mengobrol. Semua ramah, tidak ada aturan berpakaian dan hierarki perusahaan.

Alexey, pengembang tumpukan penuh




Dan ayah saya adalah seorang programmer. Saya selalu menyukai apa yang dia lakukan, jadi saya memutuskan untuk mencobanya juga. Saya mulai pemrograman di sekolah, pergi ke berbagai kompetisi, dan kemudian saya memasuki Baumanka. Sekarang saya belajar di tahun ke 4 fakultas “Ilmu Komputer dan Teknik Komputer”.

Pada musim gugur di Instagram saya melihat pengumuman tentang Kampus Perbankan Digital Rosbank. Magang tertarik oleh fakta bahwa itu termasuk arah yang menarik bagi saya dan memungkinkan untuk berpartisipasi dalam pengembangan proyek nyata. Selain itu, teman saya dulu bekerja di Rosbank sebagai pengembang junior. Dia menyukainya, jadi ketika dia melihat bahwa magang berasal dari Rosbank, dia bahkan lebih terkesan dengan tempat kerja ini.



Semua harapan menjadi kenyataan. Aku juga menyukainya. Ruang sejuk, ruang makan sendiri, tidak ada kode berpakaian. Kantor ini berlokasi di antara Gerbang Merah dan Komsomolskaya.



Kami saat ini sedang mengembangkan aplikasi web untuk mendeteksi kesalahan dalam layanan tunai. Selama tiga bulan, ia belajar merancang aplikasi web, mengotomatisasi kerja dengan Excel, dan juga menulis sistem otorisasi pengguna.
