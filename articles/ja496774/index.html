ダウンロードしたBungeeCord.jarファイルをBungeeCordフォルダーにコピーし、paper-1.15.2.jarをMurderMysteryとSkyBlockにコピーします。また、すべての新しいフォルダーで、PirateBayからstart.shファイルをコピーします。

start.shファイルを編集して、各サーバーにscreenプログラムの独自のプロセス名を割り当てます。また、BungeeCordには、以前にダウンロードした別のjarファイルがあります。次のようにしました。

バンジーコード：

#!/bin/sh

screen -S BungeeCord java -Xms512M -Xmx512M -jar BungeeCord.jar

殺人ミステリー：

#!/bin/sh

screen -S MurderMystery java -Xms1G -Xmx1G -XX:+UseConcMarkSweepGC -jar Paper-1.15.2.jar

スカイブロック：

#!/bin/sh

screen -S SkyBlock java -Xms1G -Xmx1G -XX:+UseConcMarkSweepGC -jar Paper-1.15.2.jar

合計で4つのサーバーが3.5GbのRAMを消費するため、ホスティング上のドロップレットの構成を4Gbに増やす必要があったことに注意してください。

すでに完了したシナリオに従って、eula.txtファイルを編集することを忘れずに、3つの新しいサーバーを起動します。

BungeeCordサーバーの最初の起動後、PirateBay、MurderMystery、SkyBlockの3つのサーバーを構成に登録する必要があります。これを行うには、以下のサーバーを〜/ BungeeCord / config.ymlファイルに追加し、デフォルトで作成されたサーバーを削除します。

servers:

    restricted: false

また、Minecraftのライセンスを取得していない場合は、ip_forward：trueを設定し、online_mode：trueをfalseに変更することを忘れないでください。
