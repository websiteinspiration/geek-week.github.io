Vivaldi pour PC et Vivaldi pour Android - Jeu d'équipe avec score 3.0



Bonjour à tous!

La version mitrailleuse des versions de test et des versions candidates est terminée - nous sommes prêts à vous présenter une nouvelle version du navigateur Vivaldi. Et pas un, mais deux à la fois. Comme son nom l'indique, nous présentons aujourd'hui non seulement la prochaine version littérale de Vivaldi pour les ordinateurs de bureau, mais nous sommes également prêts à annoncer la sortie de la première version stable du navigateur Vivaldi pour Android - le navigateur mobile très attendu que les utilisateurs ont demandé à partir du premier jour de vie de la version de bureau. Paradoxalement, c'est précisément le cas aujourd'hui quand deux sont trois! Plus précisément - 3.0, car les deux navigateurs, y compris la version mobile, sortent aujourd'hui avec ce merveilleux numéro. Eh bien, regardons ensemble ce que ces navigateurs ont et comment ils vont simplifier la vie de millions d'utilisateurs à travers le monde.

Parler de deux navigateurs différents dans un article est assez problématique, mais cela vaut la peine d'essayer - car ils ont beaucoup en commun. Et pas parce qu'ils portent le même nom. Le même cœur bat en eux - les versions de bureau et mobile de Vivaldi sont basées sur le noyau Chromium. Nous avons immédiatement refusé d'utiliser le code natif d'Android comme base pour la version mobile - dans ce cas, il serait difficile de supprimer tous les détails de cet outil logiciel complexe. De plus, les navigateurs de bureau et mobiles devraient avoir des fonctionnalités aussi proches que possible afin qu'il soit plus facile pour les utilisateurs de passer d'un appareil à un autre au cours du processus. Et cela ne peut être réalisé qu'en unifiant le plus possible le code. Ainsi, le chrome est notre choix naturel.

Nous avons encore le temps de parler des différences, mais d'abord nous verrons ce qu'elles ont en commun. Et le plus important, ce qui devrait être appelé et discuté en premier est le bloqueur de publicités intégré et le suivi (les soi-disant trackers). Le bloqueur utilise le radar de suivi de DuckDuckGo, qui offre une protection simple et efficace contre les tentatives de certains sites Web de suivre l'activité des utilisateurs. Par défaut, dans toutes les versions de Vivaldi, ce bloqueur est désactivé - les utilisateurs eux-mêmes peuvent configurer le niveau de blocage qu'ils souhaitent eux-mêmes. Dans les deux versions, le bloqueur est accessible via une icône spéciale sous la forme d'un bouclier placé dans la barre d'adresse:



ici, vous pouvez sélectionner le niveau de blocage pour la page actuelle, ainsi que pour accéder à des paramètres de blocage plus détaillés:



Dans cette section des paramètres, vous pouvez définir le niveau de blocage général pour toutes les pages consultées, ainsi que créer des listes d'exclusion - spécifiez les sites Web pour lesquels les paramètres du bloqueur doivent être différents. Ainsi, par exemple, même avec le blocage général désactivé, vous pouvez placer le site Web sur la liste d'exclusion et lui attribuer un niveau de blocage individuel:

Il faut dire tout de suite qu'à l'heure actuelle, ni les paramètres de blocage ni les listes de blocage ne sont encore synchronisés entre les versions de bureau et mobile. Dans la version mobile, l'accès au bloqueur et aux paramètres ressemble à ceci:



puisque nous nous sommes souvenus de la synchronisation - oui, c'est dans la version mobile. Les signets, notes, adresses saisies, mots de passe, données de formulaire sont synchronisés.



Une fonctionnalité pratique est également l'accès aux onglets ouverts sur un autre appareil. Si vous avez travaillé avec certaines pages sur le bureau puis récupéré un smartphone avec le navigateur Vivaldi installé, vous pouvez accéder aux mêmes pages en utilisant la fonction correspondante. L'option inverse est également possible - l'accès aux onglets ouverts dans le navigateur mobile est possible via le bouton sous la forme d'un nuage en haut à droite, à côté du panier d'onglets récemment fermés.

Compte tenu de la liste des données synchronisées, il est facile de remarquer que la version mobile dispose également d'un tel outil que les notes:



mais les fonctionnalités non évidentes de la version mobile incluent une capture d'écran - dans ce cas, comme dans la version de bureau, vous pouvez filmer uniquement la partie visible, ou toute la page.

Dans la version de bureau, tout le monde est déjà habitué à utiliser les moteurs de recherche, en les modifiant à la volée à l'aide d'identificateurs de lettres: par exemple, il suffit de saisir "d" dans la barre d'adresse et de saisir une requête de recherche après un espace - le navigateur ouvrira les résultats de la recherche dans DuckDuckGo. La même fonction est également implémentée dans la version mobile - les identificateurs de lettres peuvent être vus dans les paramètres des moteurs de recherche.

Séparément, je dois dire à propos des onglets privés. Oui, dans la version mobile, ce ne sont que des onglets privés, sur le bureau, je vous le rappelle, il est possible de travailler uniquement dans des fenêtres privées.

En général, répertorier les fonctions du navigateur est une tâche ingrate, en particulier dans le cas de Vivaldi. Il y a beaucoup de ces fonctions. Champ infini de recherche et d'expérimentation. Et pour le rendre plus facile, nous avons essayé de faire l'interface familière dans la version mobile, dans laquelle toutes les fonctions populaires sont aux endroits habituels:



Il existe également des onglets familiers (la barre d'onglets est désactivée dans les paramètres si vous le souhaitez) et le panneau Express avec la possibilité de créer le vôtre (indice: essayez d'appuyer longuement sur la croix pour créer un nouveau lien vers le panneau Express), et même le panneau latéral est présent sous l'icône familière. Sur ce panneau, vous pouvez travailler avec des signets, afficher l'historique des adresses saisies, modifier les notes et gérer les fichiers téléchargés. Dans le même temps, presque aucune nouvelle compétence n'est requise - tout est familier et familier de la version de bureau.

Comme vous pouvez le voir, il existe de nombreuses similitudes entre les deux navigateurs. Et les différences? Il y a aussi quelque chose à raconter. Par exemple, dans la version mobile, le changement du mode d'affichage des pages Web est disponible - vous pouvez choisir la version mobile ou complète des sites Web, cette option est dans les paramètres. De plus, tout naturellement, dans la version mobile du navigateur, vous pouvez utiliser beaucoup plus largement le glisser-déposer et d'autres manipulations sur l'écran du smartphone. Par exemple, vous pouvez créer des dossiers sur le panneau Express en déplaçant simplement une cellule sur une autre. Il est également très pratique de fermer les onglets avec un décalage sur le côté - cette option se trouve également dans les paramètres du navigateur. Oui, et la barre d'onglets elle-même peut être facilement déplacée sur les côtés avec un seul doigt.

La version de bureau a ses propres fonctions uniques. Par exemple, dans la version d'aujourd'hui, nous avons complètement réécrit le code de navigation spatiale - maintenant il est devenu encore plus rapide et plus pratique. De plus, ce n'est que dans la version de bureau qu'il y a récemment eu une fonction telle que la visualisation d'une vidéo non attachée. Dans la nouvelle version de Vivaldi, nous avons ajouté la possibilité de désactiver le bouton d'activation de cette fonction, ainsi que la possibilité de rembobiner la vidéo vers l'avant et vers l'arrière à l'aide des touches fléchées.

Mais la nouveauté la plus notable de la version de bureau est peut-être l'horloge intégrée. Ils sont situés dans le coin inférieur droit et, lorsque vous cliquez dessus, ils ouvrent une boîte de dialogue de contrôle:



Ici, vous pouvez activer la minuterie pour qu'elle vous signale après un certain temps, et vous pouvez également régler l'alarme pour l'heure exacte. En répondant à la question de savoir pourquoi l'horloge est dans le navigateur, nous pouvons dire que cette horloge n'est qu'une partie du composant plus complexe et chronophage du navigateur, qui est toujours en cours de développement.

C'est tout pour le moment. Vous pouvez télécharger de nouvelles versions de navigateurs depuis le site officiel de Vivaldi ou depuis le catalogue d'applications de la plateforme Android .

Nous attendons vos commentaires et suggestions. Une liste complète des changements dans la version de bureau peut être trouvée à cette adresse .

Merci à tous pour votre aide dans la préparation des versions d'aujourd'hui du navigateur Vivaldi! Nous avons fait du bon travail et il y a encore beaucoup de nouveautés à venir.
