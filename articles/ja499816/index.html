第10世代Intel Coreプロセッサの仕様と価格を公開



最後に、2020年4月までに新しいCPUの売上の最大90％を獲得した超人気のAMD Ryzenプロセッサに対するIntelの対応が明らかになりました。

Intelの答えは、Comet Lake-Sというコードネームの第10世代Intel Coreシリーズです。新しいプロセッサの公式発表は本日行われ、完全な情報とプレゼンテーションのスライドが数日で流出しました。

正式には、Intelは第10世代の32（！）個の新しいプロセッサを正式に発表しましたが、カットの下の表を注意深く調べると、それらのほとんどすべてがトップおよびプレトップの古いCPUの生まれ変わりであり、実際の新製品はIntel Core i9-10900とIntel Core i9-の2つだけです。 10900K。

CPUリストに移動


Intel Core i9-10900K：10コア、20スレッド、最大5.3 GHzの周波数（小文字で表記：1コアのみ、適切な冷却システムのみ）、488ドル

14 nmプロセステクノロジーを使用して、Comet Lake-Sプロセッサーの新しいファミリーが製造されています。各プロセッサの名前の文字「K」は、無料のオーバークロック用のロック解除されたマルチプライヤを意味し、文字「F」は統合グラフィックスがないことを意味します。文字「T」は、最大35 Wの消費電力が削減されたシリーズです（モバイルCPUの可能性があります）。

簡単な新機能：

  • ターボで最大5.3 GHzのクロック周波数（Thermal Velocity Boost）
  • 新しいIntel Turbo Boost Max 3.0オーバークロックテクノロジー
  • インテルハイパースレッディングテクノロジーマルチスレッディングがすべてのプロセッサーで有効
  • プロセッサーあたり最大10コア、20 MBキャッシュ（インテルスマートキャッシュ）
  • DDR4-2933 ( , Ryzen 3000 DDR4-3200 3400-3600 )
  • CPU
  • Intel 400 Series
  • Intel Ethernet I225 (Foxville).
  • AX201 Wi-Fi 6

すべてのComet Lake-Sプロセッサーは、Thunderbolt 3、Intel Optaneテクノロジー、および同じ40 PCIe 3.0レーンをサポートしています（AMDは、PCIe 4.0をサポートするプロセッサーをほぼ1年間リリースしています）。

コメットレイクSで、インテルは再びスカイレイクベースのプロセッサー用の3番目のソケットである新しいLGA1200ソケットを導入します。CPUにより多くの電力を供給し、冷却を劇的に向上させる必要があるため、新しいボードの必要性は理解できました。Z490チップセットを搭載した新しいマザーボードでは、プロセッサーの電源ゾーンの上にある大規模なヒートシンクが印象的です。LGA1200マウントはLGA115xと同じなので、古いクーラーとの互換性は維持されます。

オーバークロック


Intelは公開されたプレゼンテーションで、CPUを選択する際に高いクロック速度が依然として重要な要素である理由を説明しています。スライドの1つは「クロック速度が重要な理由」と呼ばれています。 Intelは、ゲームとほとんどのアプリケーションは依然として高周波に敏感であり、「ゲームの約60％が1つの高性能コアで動作するように最適化されている」と主張しています。

インテルは、Extreme Tuning UtilityとIntel Perfomance Maximizer をオーバークロックするためのブランドプログラムを更新しました。


プロセッサーをオーバークロックするためのユーティリティエクストリームチューニングユーティリティ（インテルXTU）

インテルXTUプログラムのハイパートレッドをコアでオン/オフできるようになり、電圧とクロック設定が改善され、PEG / DMI乗数が増加しました。



次のスライドに示すように、クロック速度を上げてCPUのオーバークロックを改善するために、STIM（はんだベースのサーマルインターフェースマテリアル）サーマルインターフェースに変更を加えました。特に、ヒートシンク（一体型ヒートスプレッダー、IHS）のカバーの厚さは、シリコン結晶自体の厚さの減少に伴って増加しています。



グラフィックアート


Fモデルを除くすべてのCoreプロセッサーには、Intel UHD Graphics 630グラフィックが統合されており、

CeleronプロセッサーもIntel UHD Graphics 610グラフィックを使用しています。

Ryzen?


一部のコメンテーターによると、そのようなIntelの行動は競合他社の活動によってのみ説明されています-Ryzenプロセッサーを搭載したAMDが市場を押し上げ、業界の科学的および技術的進歩を加速し、価格が崩壊しました。 Intelの新しいCore i3は、第7世代（7000）までのCore i7プロセッサよりもパフォーマンスが劣っていません。

表を見ると、Core i3-10100、4コア、8スレッド、最大4.3 GHzの周波数は、Core i7-7700と変わりません。

Core i5-10600K、6コア、12スレッド、周波数4.8 GHzまで、ロック解除された乗数は、Core i7-8700Kの生まれ変わりと非常に似ています。

同じことがテストで確認されています。以下の表は、新旧のIntelおよびAMDプロセッサーに対するCinebench R20テストの結果を示しています（CPU-Monkey Webサイトのベンチマーク）。


CM。Intel Core i3-10300およびi3-10100とRyzen 3 3300Xおよび3100

の比較このラインで本当に新しいプロセッサは、10コアCore i9-10900およびCore i9-10900Kだけです。残りはすべて、基本的に前世代のトップまたはプレトップソリューションの類似物であり、割引価格で販売されます。

一方、数十の新しいモデルが多数発表されているにもかかわらず、誰もがインテルの技術的な遅れを完全に理解しています。新しいプロセッサは、2015年と同じ古いSkylakeアーキテクチャで古い14 nmプロセステクノロジーを使用して製造されています。これは、当時としては素晴らしい成果でしたが、それ以来改善されていません。KabyLake（7th generation）およびCoffee Lake（8 2世代目と9世代目）はSkylakeとほとんど同じです。

同時に、AMDは1年間7nmプロセッサをリリースしており、2020年の秋には、Apple A14チップが5nmになるはずです。 AMDは新しいZen 2アーキテクチャと最大16コアのデスクトッププロセッサを発表したため、新しいIntelプロセッサは16コアのRyzen 9 3950Xとパフォーマンスを比較できません。 Intelはこのトッププロセッサの競合他社をリリースしていません。

公式仕様のIntelは125 Wの最大熱放散を示していますが、最初の独立したテストははるかに高いTDPを示しています。たとえば、スクリーンショットでわかるように、4.6 GHzの周波数のCore i9-10900は93°Cまで加熱し、220 Wを消費します



。比較のために、280 Wの熱放散には、約3 GHzの周波数で動作する64コアのRyzen Threadripper 3990Xがあります。

一部の専門家によると、過度の発熱が原因で、第10世代Intel Coreプロセッサーは、極低温の設置を除いて、多くのオーバークロックを行うことができない可能性があります。

新しいIntelプロセッサは、AMDの同等のパフォーマンスモデルよりもコストがかかりますが、違いはそれほど大きくありません。確かに多くの潜在的なバイヤーは少し待つことを好むでしょう：今年の終わりか来年の初めに、インテルは第11世代プロセッサー（ロケット湖）をリリースします。同じ14 nmプロセステクノロジーにもかかわらず、IPC、PCI Express 4.0サポート、新しい統合グラフィックス、DMI 3.0 x8チップセット接続（帯域幅2倍）、最大20 Gbpsの速度でのThunderbolt 4およびUSB 3.2のサポートを備えた真に新しいアーキテクチャに基づいています。 / c。

7ナノメートルのトポロジーを備えた最初のIntelデスクトッププロセッサ、DDR5メモリおよびPCI-Express 4.0インターフェイスのサポートは、2022年までにリリースされます。噂によると、コードネームMeteor Lakeのプロセッサーは、Golden Coveアーキテクチャーに基づくx86コアを受け取ります。その開発には、2018年からIntelで働いているAMD Zenアーキテクチャーの作成者の1人であるJim Keller（ジムケラー）が参加しました。

AMD自体は、2022年までに新しいZen 4アーキテクチャでプロセッサも5 nm標準でリリースし、これもDDR5をサポートします。AMDのサーバーセグメントでの5ナノメートルおよびDDR5への移行は、2021年に新しいGenoa Epycプロセッサのリリースとともにさらに早く行われます。
