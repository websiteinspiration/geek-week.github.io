3月和4月在线活动摘要



最近几周，日常生活在许多方面发生了巨大变化。因此，鉴于当前事件，一段时间以来，我们正在更改摘要的格式。幸运的是，IT社区在虚拟环境中有足够的自由，可以相对轻松地将其迁移到那里进行通讯，培训和联合活动的站点-这意味着生活在继续。在本期中，我们收集了在线mitap，会议和网络研讨会，这将有助于增加隔离检疫生活的多样性并为头脑而写作。

免费的IT网络研讨会

时间： 3月26日，4月2日
参与条款：免费，需要注册

ITHub College发起的一项举措可以帮助学生和刚刚开始的IT专业人员驾驭市场并预先计算选择权。发言者在报告中考虑了领域内的不同方向，概述了需求，准入门槛，必要的技能以及其他可能影响选择的因素。第一个网络研讨会专门介绍有前途的职业，第二个网络研讨会涉及gamedev行业。

Moscow Python Conf ++ 2020

时间： 3月27日，
参与条件： 7533卢布。

经典形式的莫斯科Python Conf ++将于秋季举行，对于Python社区来说，同一事件也可以视为一种奖励。会议将持续一整天，并将为与会者提供不同类型的内容：来自俄罗斯和国外演讲者的报告（Django，Cerberus，JupyterHub，测试，数据验证），播客，在线咨询，讨论，网络，甚至是体育锻炼的人士。传统的聚会也将举行-这次是在Zoom。脱机版Moscow Python Conf ++的门票持有人可以免费使用虚拟平台。

Virtual PHP Mitap

时间： 3月28日
参与条款：免费

一个面向PHP爱好者的在线mitap也正在准备一个丰富的程序：来自实践开发人员的六个演示文稿（CQRS，针对微服务的整体体系结构，遗留代码，REST API的拒绝等等）以及面向观众的各种形式的参与。听众将能够现场提问（在家庭中以披萨形式的奖品会鼓励这种活动），与演讲者进行公开聊天就报告的主题发表讲话，在晚会上谈论任何话题，或者最糟糕的是与主持人交谈。会场还将举行抽奖活动，包括PHP俄罗斯会议在线和离线版本的门票以及PhpStorm的许可证（根据测验结果）。

#StayHomeClub聚会

时间： 3月30日，
参与条款：免费，需要注册

那些在环境压力下第一次搬到家里工作的人的支持小组。具有经验的工作人员将连续分析初学者面临的所有常见困难和不寻常的时刻：建立的内容，时间的流逝，组建分布式团队的特征，在线会议的外观以及以一百万个聊天通知发疯的现实性。

数据质量聚会

时间： 3月31日，
参与条款：免费，需要注册

对于认为自己是数据质量工程师或只是参与大数据质量控制的人员来说，这是一个小事件。从EPAM员工的两份报告中，第一手聆听者将了解担任该职位的具体细节：数据质量工程师要解决哪些任务，他的绩效测试与经典有何不同，必须开发哪些技能以及职业成长的里程碑？

春季-熟识和速潜

时间： 4月1日，
参与条款：免费，需要注册

在专家和支持者的陪伴下，Spring Java框架的深入开发。迷你课程的主要目的是使您熟悉Spring Core的全部功能以及最受欢迎的模块（Spring MVC，Spring Data，Spring REST）。该程序是为初学者Java开发人员设计的，它直接涵盖了Spring的基础武库的理论基础和应用。网络研讨会的详细计划已发布在活动网站上。

Redmadrobot上的Android mitap时间

： 4月2日
参加条款：免费，需要注册

由Redmadrobot组织的一次Android开发人员会议，旨在获得普遍利益和娱乐。有关详细信息，请参阅有关Habré的官方公告。

Hackathon EcoHack 2020

时间： 4月3-5日
参加条件：免费，需要注册

面对一个全球性问题，别忘了其他问题，而黑客马拉松创建解决环境问题的软件是保持平衡的好方法。将有可能将四个任务之一纳入工作，每个任务都与一个单独的主题相关：废物分类，生态旅游，环境监测等。前两天将进行远程设计和开发；与专家和行政部门的沟通将通过Telegram进行。第二天晚上，将在第三天举行代码审查会议-实况项目演示。为了提高透明度，所有项目的代码都将公开提供。此外，组织者还提供团队参加HERE Technologies举办的大师班，将于4月3日播出。

移动开发实用Flutter在线课程

时间：从4月9日开始，
参与条件： 36 000卢布。

组织者为移动开发人员提供了在入门级别学习如何使用Flutter和Dart的机会。该课程包括11个大块，重点放在实际任务上。除了开发本身之外，该程序还涉及服务器文档，设计应用程序体系结构，代码审查，评估和工具选择等工作领域。培训在导师的指导下进行-经验丰富的开发人员和社区的活跃成员，并在Telegram中进行专业聊天。

INFOSTART MEETUP圣彼得堡

时间： 4月10日
参加条款：3000卢布

Infostart社区的IT专家在线会议，讨论与管理和1C技术相关的紧迫问题。演讲的主题包括高负载优化，集成和数据交换，工具包形成，开发方法以及IT部门管理。已经发布了十多个报告，这些报告将通过多个渠道进行现场交流。

Mitap“团队管理”

时间： 4月11日，
参加条件：免费，需要注册

五名领导和部门负责人将聚集在监视器上，与战友分享他们的案子，并讨论与团队密切相关的问题。有了这样的介绍性议程，议程肯定会有所变化：这里有一个空间，可以讲述初创企业中角色的分布情况，以及与初级人员建立生产性工作的技巧，以及组织交叉审查和计划冲刺的一系列动作...报告将以非正式的形式编入问题，讨论和新闻共享。

在线监控日，

时间： 4月15日，
参与条款：免费，需要注册

来自Monhouse.tech社区的IT监视和基础架构领域的专家虚拟聚集。该活动包含多种形式：报告，与合作伙伴公司的代表（Grafana，PRTG，Selectel）举行的专家会议以及在Zoom中演讲者与听众之间的最紧密互动。

IT中的女孩

时间： 4月15日
参加条款：免费，需要注册

在线会议专门讨论编程中的女性问题：为什么俄罗斯这个领域的专家很少，是什么促使女孩们进入该行业以及如何激发她们对技术的兴趣。该活动主要对那些试图决定职业道路的人来说非常有趣。三名参与者将现场表演，之后将开始对所有与会人员进行一般性讨论。

面对COVID-19的投注

时间： 4月16日，
参与条件： 5000 卢布。

一些会议是与冠状病毒相反的会议，而有些会议是由它们产生的。博彩业原来是市场的第一位利基市场，因此决定了解其当前状况和新兴前景。博彩和赌博领域的专家将分享他们对隔离措施将对行业造成多大影响，是否有可能以牺牲电子竞技为代价来纠正这种情况以及IT解决方案提供商应重点关注的问题的看法。
