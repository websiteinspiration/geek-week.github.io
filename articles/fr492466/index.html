Comment passer de n'importe quel fournisseur d'hébergement avec cPanel à Plesk dans Rusonix en seulement cinq étapes

Il arrive que les gens changent de fournisseur d'hébergement. Cela se fait pour diverses raisons: certains sont tout simplement mécontents de la rapidité ou de la qualité des services, d'autres trouvent la même qualité, mais à un prix inférieur. D'autres n'aiment pas les panneaux de contrôle ...

Si vous êtes dans cette situation, il est temps de transférer vos sites et projets vers Rusonix .

Cependant, déplacer votre site peut sembler une tâche ardue. Dans ce guide, nous vous expliquerons une procédure simple avec des instructions étape par étape afin que vous puissiez transférer en toute confiance et facilement toutes vos données vers les serveurs Rusonix .

Le transfert de votre site peut se faire en seulement cinq étapes:

Étape 1. Téléchargement des fichiers de site à partir d'un ancien fournisseur d'

hébergement À l'aide du gestionnaire FTP ou du gestionnaire de fichiers de votre fournisseur d'hébergement actuel, copiez les fichiers de site sur votre ordinateur. Assurez-vous de faire une sauvegarde sur un support indépendant.

Étape 2. Exportation de la base de données de votre site Web

Puisque vous utilisez cPanel, l'exportation des bases de données ne sera pas une tâche difficile. Pour ce faire, utilisez phpMyAdmin, qui fait partie intégrante de chaque cPanel. Ne vous inquiétez pas si vous voyez des noms de fichiers et de bases de données étranges: même si vous ne comprenez pas tout, vous pouvez toujours le faire vous-même.

  • Recherchez l'onglet où se cachent les bases de données et ouvrez-le.
  • Cliquez sur l'icône phpMyAdmin pour ouvrir l'interface.
  • Recherchez et cliquez sur le bouton "Exporter".
  • Sélectionnez «méthode rapide - afficher uniquement les paramètres minimum».
  • «SQL» .
  • «», .
  • . , — .

Tout s'est bien passé? Votre base de données est maintenant prête pour le transfert vers les serveurs Rusonix. Passez à l'étape suivante.

Étape 3. Téléchargement des fichiers de site

Ouvrez le panneau Plesk dans Rusonix et accédez à la section "Sites Web et domaines". Cliquez sur le bouton "Télécharger" et sélectionnez une archive enregistrée avec votre site. Veuillez noter que le panneau Plesk ne prend en charge que les archives au format zip.

Sélectionnez l'archive enregistrée sur l'ordinateur et cliquez sur le bouton «Extraire les fichiers».

Étape 4. Importation de bases de données

Comme pour l'exportation, vous utiliserez MySQL pour importer les bases de données. Ce n'est en fait pas si difficile. Vous devez donc également saisir phpMyAdmin, trouver le bouton "Importer" et cliquer dessus. Sélectionnez ensuite le fichier sql enregistré sur votre ordinateur et importez-le.

Étape 5. Changer le DNS

C'est un petit détail qui est facile à manquer. Après avoir transféré votre site, vous devez basculer l'enregistrement DNS du domaine du serveur précédent vers le serveur Rusonix. Si cela n'est pas fait, les utilisateurs seront redirigés vers la page avec une erreur. Le nouveau DNS est facile à apprendre avec le support Rusonix.

Une fois que vous avez envoyé une demande de déplacement des enregistrements DNS, le basculement peut prendre plusieurs heures et, dans certains cas, une journée entière.

Si, pour une raison quelconque, vous ne réussirez pas, ceux-là. Le support Rusonix transférera indépendamment toutes vos données sur le serveur .

Vous n'êtes pas satisfait de votre hébergeur actuel? Ne perdez pas votre temps - choisissez Rusonix! Ici, vous trouverez non seulement une infrastructure puissante et rapide, mais vous serez également agréablement surpris par le professionnalisme et l'atmosphère chaleureuse et conviviale du support technique.
