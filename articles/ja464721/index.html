Oracle Database SQL認定を準備して合格した方法（1Z0-071）

なぜ記事が書かれているのですか？



Java 8でOCAとOCPを準備しているときに、Habrに関する記事をたくさん見つけました。そのおかげで、最適なパスを選択し、多くの時間を節約できました。


ただし、OCA Oracle Database SQL（1Z0-071）に備えて、Habréには資料がなく、インターネットにもほとんどありません。そのため、時間をかけずにかなり大規模な試験に合格することに関心のある人に役立つ詳細な記事を書くことにしました。


なぜ引き渡されたのですか？


主な理由は、長年の研究と仕事で得た知識を合理化し、すべての白い点を埋めたいという願望でした。また、データベースベンダー自体（この場合はOracle）が開発者にどの程度の理解を期待しているかを知りたいと思っていました。


私はSQLの本を読むだけでは不十分だと気づきました。練習をしないと、読んだすべてを学んだような気分になるからです。認定の準備をしているときは、読んだほとんどすべての行のテストを解く必要があります。これは、いくつかのポイントを完全には理解しておらず、再度読んで戻ってくることを示しています。
