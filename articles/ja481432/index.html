Pythonプログラミング用の6つのコアライブラリ

こんにちは、ハブル！あなたに私の存在「の記事の翻訳Pythonプログラミングのため6つのエッセンシャルのPythonライブラリ」により、Vijaydeveloper。

Python（python）は、プログラマーのコミュニティーをリードする最も人気のあるものの1つとなった、高水準の汎用プログラミング言語です。その機能に応じて、単純化されたアプリケーションの開発から、同じレベルの複雑さを持つ複雑な数学的計算まで分類されます。

主要なプログラミング言語の1つであるため、使用できる多くのフレームワーク（アプリケーションを構築するためのプラットフォーム）とライブラリがあります。プログラミング言語ライブラリは、このプログラミング言語を使用していくつかの特定の操作を容易にするモジュールと関数の集まりです。

そこで、すべてのPython開発者が知っておくべき6つの主要なPythonプログラミングライブラリを次に示します。

•ケラス


タイプ-ニューラルネットワークライブラリ。

初期バージョンは2015年3月です

。Kerasは、Pythonで記述されたオープンニューラルネットワークライブラリです。コンパクト、モジュール式、拡張可能に設計されている一方で、ディープラーニングネットワークでの運用作業を目的としています。

Kerasは、ニューラルネットワークを表現するためのより単純なメカニズムを提供するだけでなく、モデルのコンパイル、データセットの処理、およびグラフの視覚化に最適な機能もいくつか提供しています。バックエンド（サーバー）では、KerasはTheanoまたはTensorFlowを使用します。

Kerasはサーバーインフラストラクチャを使用して計算グラフを作成し、それを使用して操作を実行するため、他の機械学習ライブラリよりも動作が遅くなります。ただし、Kerasのすべてのモデルは移植可能です。

特徴：

  • 完全にPythonで記述されているため、デバッグと調査が簡単です。
  • 活性化関数、レベル、目標、オプティマイザなど、広く使用されているニューラルネットワークのビルディングブロックの多数の実装が含まれています。
  • 驚異的な表現力と柔軟性により、革新的な研究に最適です。
  • これは、Inception、MNIST、ResNet、SqueezeNet、VGGなど、いくつかの前処理済みデータセットと事前トレーニング済みモデルを提供します。
  • 畳み込み、埋め込み、完全接続、プーリング、再帰を含む、ニューラルネットワークのほぼすべてのモデルをサポートします。さらに、これらのモデルを組み合わせて、さらに複雑なモデルを開発できます。
  • CPU（中央処理装置）とGPU（プロセッサコア）の両方で動作します

応用分野：

  • Netflix、Square、Uber、Yelpはすでに使用されています。
  • 深層学習の研究用。CERNおよびNASAの研究者によって採用されました。
  • ディープラーニングスタートアップに人気。

•NumPy


タイプ-テクニカルコンピューティングライブラリ。

初期バージョンは1995（数値として）です。

2006（NumPyと同様）。

NumPyは、競合するNumarrayライブラリーの機能をNumericライブラリーに組み込み、大幅な変更を加えることにより、2005年にTravis Oliphantによって作成されました。無料のオープンソースライブラリには、世界中からいくつかの寄稿者がいます。

Python、TensorFlow、およびその他のいくつかのライブラリで最も人気のある機械学習ライブラリの1つは、NumPy Pythonライブラリを内部で使用して、いくつかのテンソル演算を実行します。

特徴：

  • アクティブなコミュニティサポート
  • 完全に無料でオープンソース
  • 行列乗算などの複雑な行列演算
  • インタラクティブで超使いやすい。
  • 複雑な数学的実装を容易にする
  • 読みやすいコンセプトでコーディングが簡単

応用分野：

  • 複雑な数学的計算を実行するには
  • N次元の実数の配列の形式で画像、音波、およびその他の形式のバイナリ生ストリームを表現する
  • 機械学習プロジェクトの場合

•枕


タイプ-画像処理ライブラリの

初期バージョン-1995（Python Imaging LibraryまたはPILのような）

2011（枕のような）

Pillowは、開発されたプログラミング言語とほぼ同じくらい古いPythonライブラリです。実際、PillowはPIL（Python Imaging Library）のフォークです。さまざまな画像ファイルを開いたり、操作したり、保存したりするには、自由に使用できるPythonライブラリが必要です。

Pillowは、いくつかのLinuxディストリビューション、特にDebianとUbuntuで、元のPILの代替として採用されています。ただし、MacOSおよびWindowsでも使用できます。

特徴：

  • 画像にテキストを追加します
  • ぼかし、明るさ調整、輪郭、シャープネスを含む画像の強化とフィルタリング
  • 迷彩と透明性
  • ピクセル操作
  • BMP、GIF、JPEG、PNG、PPM、TIFFなど、多くの画像ファイル形式をサポートしています。利用可能なファイル形式のライブラリを拡張する新しいファイルデコーダーの作成をサポートします。

応用分野：

  • 画像処理用

•PYGLET


タイプ-ゲーム開発ライブラリ

初回リリース-2015年4月

Python用のマルチプラットフォームクロッピングおよびマルチメディアライブラリであるPYGLETは、Pythonを使用してゲームを開発するための一般的な名前です。ゲームに加えて、ライブラリは視覚的に豊かなアプリケーションを作成するように設計されています。

フレーミングのサポートに加えて、PYGLETは、イメージとビデオのロード、サウンドと音楽の再生、OpenGLグラフィックス、およびユーザーインターフェイスイベントの処理をサポートします。

特徴：

  • 複数のモニターで複数のウィンドウとデスクトップを使用する
  • ほとんどすべての形式で画像、音声、動画をダウンロード
  • 外部の依存関係やインストール要件はありません
  • オープンソースのBSDライセンスの下で提供されるため、個人的な目的と商用目的の両方で自由に使用できます。
  • Python 2とPython 3の両方をサポートします

応用分野：

  • 視覚的に豊かなアプリケーションを開発するため
  • ゲーム開発用

• リクエスト


タイプ-HTTPライブラリ

初回リリース-2011年2月

リクエスト-Python HTTPライブラリ。HTTPリクエストをより簡単で便利にするために設計されています。Kenneth Reitzと他のいくつかの貢献者によって開発されたRequestsを使用すると、人間の介入なしにHTTP / 1.1要求を送信できます。

NikeやSpotifyからAmazonやMicrosoftに至るまで、何十もの大規模な組織が内部でクエリを使用してHTTPをより適切に処理しています。完全にPythonで記述されたRequestsは、Apache2ライセンスの下で無料のオープンソースライブラリとして利用できます。

特徴：

  • コンテンツの自動デコード
  • 基本/ダイジェスト認証
  • ブラウザスタイルのSSL検証
  • 部分的なリクエストと接続タイムアウト
  • .netrcおよびHTTP（S）プロキシのサポートを提供します
  • Cookieセッション
  • Unicodeレスポンスボディ

応用分野：

  • Pythonを使用してHTTP / 1.1リクエストを送信し、ヘッダー、フォームデータ、マルチパートファイルなどのコンテンツを追加できます
  • クエリ文字列をURLに自動的に追加するには
  • POSTデータを自動的にエンコードするには

•TensorFlow


タイプ-機械学習ライブラリ

初回リリース-2015年11月

TensorFlowは、データフローと差別化プログラミングに関連する多数のタスクを解決するために設計された無料のオープンソースPythonライブラリです。ただし、TensorFlowシンボリック数学ライブラリは、Pythonで最も広く使用されている機械学習ライブラリの1つです。

内部使用のためにGoogle Brainによって開発されたライブラリは、商業目的および研究目的で使用されます。

テンソルは、データを表すN次元の行列です。 TensorFlowライブラリを使用すると、多数のテンソル演算を含む新しいアルゴリズムを作成できます。

ニューラルネットワークは計算グラフとして表現できるため、テンソルに対する一連の操作としてTensorFlowライブラリを使用して簡単に実装できます。

特徴：

  • グラフの各部分を視覚化できます。
  • 完全に無料でオープンソース
  • 分散コンピューティング用のCPU（中央処理装置）とGPU（プロセッサコア）で学習しやすい
  • 優れたコミュニティサポート
  • パフォーマンスに柔軟性を提供します。最も必要な部品はスタンドアロンにすることができます
  • 大規模システムで効率的なモデルを作成するために、複数のニューラルネットワークと複数のGPUのトレーニングをサポート
  • XLAなどの方法を使用して線形代数演算を高速化

応用分野：

  • 機械学習プロジェクトの場合
  • ニューラルネットワークプロジェクトの場合
  • DeepDreamなどの自動字幕起こしソフトウェア
  • GoogleフォトやGoogle音声検索などのGoogle製品での機械学習
