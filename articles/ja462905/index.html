テレボットライブラリパート2を使用してPythonで電報ボットを作成する

みなさん、こんにちは！これは、電報リクエストとサーバーに時間を割く記事の第2部です。他の誰かが最初の部分を見たことがなければ、私はそれを読むことをお勧めします。3番目の部分では、データベースの操作方法について説明します。さて、もう一度質問がある場合は、電報で私に書いてください：ディマゴロフツォフ

メッセージとは？


おそらく、私の以前の記事でボットを作成した多くの人がメッセージとは何か、たとえば、メッセージを送信するために、send_message関数パラメーターでmessage.chat.idを指定する必要があるのか​​と尋ねました。確認するには、コンソールにメッセージを表示します。

@bot.message_handler(commands=['start'])
def start(message):
    print(message)

たとえば、これらすべての情報から、メッセージの送信元のチャットのIDを取得します。これを行うには、チャットキーに切り替えます。

問い合わせ：

print(message.chat)

回答：

{'type': 'private', 'last_name': 'Gorovtsov', 'first_name': 'Dmitry', 'username': 'dimagorovtsov', 'id': 687595402, 'title': None, 'all_members_are_administrators': None, 'photo': None, 'description': None, 'invite_link': None, 'pinned_message': None, 'sticker_set_name': None, 'can_set_sticker_set': None}

チャットキーには、first_name、last_name、usernameなどのキーがいくつかあり、それぞれに独自の値があります。次に、idキーを見てみましょう。

print(message.chat.id)

ご覧のとおり、目的の値を取得するには、キーの名前をドットで記述する必要があります。次に、サーバーからの応答を確認します。

687595402

すべてが正常に進んでいます！希望どおりにチャットIDを取得しました。送信者の名前を取得します。ここで、気づいたように、from_userキーを使用する必要があります。

問い合わせ：

print(message.from_user)

回答：

{'id': 687595402, 'is_bot': False, 'first_name': 'Dmitry', 'username': 'dimagorovtsov', 'last_name': 'Gorovtsov', 'language_code': 'ru'}

ここで、first_nameキーの値を取得します。

print(message.from_user.first_name)

以上です！数秒で、チャットIDと私の名前を電報で取得することができました。繰り返しますが、理解できない人のために：



first_nameキーの値を取得するには、最初にチャットキーに切り替え、次にfirst_nameに切り替える必要があります。

ここで、メッセージをチャットに送信するには、send_message関数でいくつかのパラメーターを指定する必要があります。最初のパラメーターはchat_id、つまりチャットID自体です。2番目は、メッセージのテキストであるテキストです。そして、ご想像どおり、message.chat.idを書き込む代わりに、データを書き込むことができます。これは、パラメーターにIDを指定した場合にメッセージを自分に送信する方法です。

bot.send_message(< id>, ' ')

まあ、message.chat.idを記述するときは、ボットが呼び出し元のチャットにメッセージを送信することを意味します。
