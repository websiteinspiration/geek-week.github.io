レジスターからの抽出で、信頼性に関する情報を確認してください。PythonでのPDFの接着

現時点では、税務当局がUSRLEから会社を除外する可能性は、会社に関するいわゆる不正確な情報を単に「明らかにする」だけで、依然として非常に関連のあるトピックです。統計が示すように、2018年9月から、連邦税務局は9万の組織を法人の統一州登録簿から除外し、法人の頭、創設者、または住所に関する情報の不正確さの記録を残しました。会社に関する不正確な情報があるという事実を発見することは、登録簿から抽出することによってのみ見ることができます。

この問題は、税務当局の行動の結果として、不正確さに関するデータが関係者の要求と「自分自身」の両方で表示される可能性があるという事実によって悪化します。レジスターからの突然の逸脱から身を守るために、定期的に抽出物を受け取る必要があります。持ち株会社が多数ある場合に、これをすばやく簡単に行う方法については、前回の投稿で検討しました。

今回は、法人の登記簿の抜粋で、信頼性に関する情報を探す方法を分析します。

FTS Webサイトからダウンロードしたステートメントのn番目の数があると仮定します。ステートメントには拡張子.pdfといくつかの名前が付いています。

私たちに必要なのは、各PDFファイルで「欠けている」という単語を検索することだけです。

ステートメントでPDFを開いて検索することは、私たちの方法ではありません。これには非常に長い時間がかかる場合があります。Abbyy Finereaderですべてのファイルを接着できますが、十分な時間もかかります。

すべてのPDFファイルを1つに接着するプログラムを作成します。Pythonでは、これを数秒で実行できます。

将来的には、作成されたファイルを開いて、レジスターからのすべての抽出物ですぐに目的の単語の検索を実行できるようになります。

始めましょう。

* USRLEからのステートメントは、C：\ 1ディレクトリにあります。
新しいpythonファイルでは、pdfとシステム全体を操作するためのモジュールをインポートします。

import PyPDF2, os

次に、空のリストを作成し、すべてのステートメントが配置されるディレクトリC：\ 1に移動します。

このディレクトリは空である必要はありません。プログラムでは、pdf拡張子を持つファイルのみの処理を提供しました。

pdfFiles = []
os.chdir('C:\\1')
for filename in os.listdir('.'):
    if filename.endswith('.pdf'):
        pdfFiles.append(filename)
pdfFiles.sort()

次のブロックはステートメントを互いに接着し、後続の各ステートメントを最後に追加します。

pdfWriter = PyPDF2.PdfFileWriter()
# Loop through all the PDF files.
for filename in pdfFiles:
    pdfFileObj = open(filename, 'rb')
    pdfReader = PyPDF2.PdfFileReader(pdfFileObj)

    # Loop through all the pages and add them.
    for pageNum in range(0, pdfReader.numPages):
        pageObj = pdfReader.getPage(pageNum)
        pdfWriter.addPage(pageObj)

結果を保存するだけです。

pdfOutput = open('all.pdf', 'wb')
pdfWriter.write(pdfOutput)

そのため、プログラムの作業終了後、all.pdfファイルを受け取りました。これにより、虚偽の情報に関する必要な情報を検索することがすでに可能になっています。

レジスターのステートメントを切り取り、各ステートメントの最初の4ページを残しました。
陪審員の信頼性に関する情報。人はレジスタのさまざまな部分に分類されます。
抽出の最後には、税によって取り消された信頼性のない記録が含まれています。
したがって、合法的な州法人登録簿からの抜粋全体に対してプログラムを実行することはほとんどお勧めできません。プログラムはこれらの古いレコードも検出します。
したがって、pyhtonを使用してUSRLEからダウンロードしたステートメントを切り取り、それぞれから最初の4ページを保存します。原則として、これらのページは住所または唯一の執行機関に不正確な兆候を見つけるのに十分です。
以前にダウンロードしたすべてのステートメント（pdfファイル）をパス「C：\ 1 \ 2」に沿って条件付きフォルダーに移動し、Pythonコードを実行します。
#! python3
import PyPDF2, os
from datetime import datetime

start = datetime.now()
os.chdir('C:\\1\\2')
pdfFiles = []

for filename in os.listdir('.'):
    if filename.endswith('.pdf'):
        pdfFiles.append(filename)
pdfFiles.sort()

pdfWriter = PyPDF2.PdfFileWriter()

# Loop through all the PDF files.
for filename in pdfFiles:
    pdfFileObj = open(filename, 'rb')
    pdfReader = PyPDF2.PdfFileReader(pdfFileObj)

    # Loop through all the pages and add them.
    for pageNum in range(0, 4):
        pageObj = pdfReader.getPage(pageNum)
        pdfWriter.addPage(pageObj)

# Save the resulting PDF to a file.
pdfOutput = open('all-small.pdf', 'wb')
pdfWriter.write(pdfOutput)
pdfOutput.close()
print(datetime.now()- start)

出口では、1つのpdfファイル「all-small.pdf」に接着された登録ステートメントを受け取ります。また、各ステートメントからは最初の4ページのみになります。

次に、「欠陥」という語句を検索して「all-small.pdf」を実行します。
#!/usr/bin/python

import fitz,os

filename = "all-small.pdf"  
search_term = ""  
pdf_document = fitz.open(filename)

for current_page in range(len(pdf_document)):  
   page = pdf_document.loadPage(current_page)
   if page.searchFor(search_term):
       print("%s    %i" % (search_term, current_page+1))
