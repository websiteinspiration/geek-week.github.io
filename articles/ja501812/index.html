Python Telegramページネーションボット

最近、電報ボットの要素のリストにページ番号を付けるというタスクに直面しました。そして、彼女はそのような一見典型的なタスクのために、PyPiに単一のライブラリーがないことを発見して驚いた。したがって、この迷惑な省略を修正し、その実装を公開することが決定されました。


コンソールで実行することでPyPiからパッケージをインストールできます：


pip install python-telegram-bot-pagination

telegram_bot_paginationパッケージのInlineKeyboardPaginatorクラスを使用すると、リストページ内を移動するためのインラインボタンのセットを簡単に取得できます。要素のリストが分割されるページ数、および現在のページとボットボタンを押して返されるデータを生成するためのパターンをデザイナーに転送する必要があります。その後、ページ編集者自身が必要なキーボードを形成します。これは、markupプロパティを通じて使用できます。


from telegram_bot_pagination import InlineKeyboardPaginator

paginator = InlineKeyboardPaginator(
        10,
        current_page=1,
        data_pattern='elements#{page}'
    )

これで、paginator.markupがsendMessage Telegram Bot APIメソッドのreply_markupフィールドのjsonオブジェクトを返します。この場合、5つのボタンが形成されます。ページ数が5以下の場合は、ボタンも対応します。ページが1つしかない場合、マークアップは空白になります。5ページを超える場合、キーボードには次の5つのボタンが含まれます。


  • 一ページ目;
  • 前のページ;
  • 現在のページ;
  • 次のページ;
  • 最後のページ。

たとえば、文字のリストを表示するボットを作成するためのハリーポッターの文字の説明を含むcharacter_pagesの配列がある場合、現在のリストページを送信する関数が必要です。


def send_character_page(message, page=1):
    paginator = InlineKeyboardPaginator(
        len(character_pages),
        current_page=page,
        data_pattern='character#{page}'
    )

    bot.send_message(
        message.chat.id,
        character_pages[page-1],
        reply_markup=paginator.markup,
        parse_mode='Markdown'
    )

そしてボットのボタンハンドラー：



@bot.callback_query_handler(func=lambda call: call.data.split('#')[0]=='character')
def characters_page_callback(call):
    page = int(call.data.split('#')[1])
    bot.delete_message(
        call.message.chat.id,
        call.message.message_id
    )
    send_character_page(call.message, page)

* この例では、pyTelegramBotAPIを使用してボットを操作します。
