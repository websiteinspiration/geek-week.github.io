选择闪存卡：保护数字品种的详细指南



选择硬盘驱动器时，“高级”买家通常不会遇到任何特殊问题。以相同的6 TB WD6003FZBX为例：主轴速度为7200 rpm，具有256 MB的超大容量缓存，并具有6 Gb / s的高速SATA III接口，这雄辩地表明我们拥有一种高性能的解决方案，可以同时满足两种需求游戏玩家和发烧友都一样的专业人士。这里唯一需要考虑的细微差别是6 Gb / s指示符是总线本身的带宽：尽管从高速缓存到系统的信息传输速度为550 MB / s，但系统与所讨论模型的驱动器之间的实际数据交换速率达到227 MB / s，对于传统硬盘而言，这仍然是一个出色的结果。但是，几乎所有精通技术的PC用户都知道这一点。

对于闪存卡，即使是经验丰富的管理员也可能陷入困境。您无需走太远，例如：只需打开SanDisk Extreme PRO UHS-I存储卡的页面。除了通常的行“读取速度：最高170 MB / s”和“写入速度：最高90 MB / s”外，规格中还标有“视频速度：C10，U3，A2，V30”这一神秘字样。但是，让我知道我们的产品速度特性。为什么我们还需要其他一些晦涩的速度类，甚至四个量级？这些指标之间的根本区别是什么？正是在这些问题中，我们必须在今天的材料中弄清楚。

安全数字存储卡的第一版于1999年由SanDisk（目前是Western Digital的子公司），Panasonic和Toshiba参与开发。非营利组织SD卡协会早在2000年就已宣布为国际消费电子展（CES）的一部分，其创始人是上述公司。迄今为止，已有1000多家公司参与其工作，其工程师帮助设计和开发了一系列规格，这些规格几乎描述了可移动存储卡的所有参数-从外形尺寸到与客户端设备的数据交换接口的功能。该文档称为“ SD规范”，其内容可以在官方网站https://www.sdcard.org上找到。。让我们仔细检查资源，看看各种缩写和标记的含义。

安全数字存储卡尺寸


让我们从最简单的形式开始。SD规范提供了三种尺寸的存储卡：

  • SD是经典的闪存卡，外形尺寸为24×32×2.1 mm。在有趣的功能中，值得强调的是机械覆盖保护开关的存在。当设置为“锁定”位置时，它将阻止将新文件写入卡，删除现有文件或对其进行格式化的功能。



  • miniSD是闪存卡的一种更紧凑的版本，尺寸为20×21.5×1.4 mm，主要用于便携式设备。目前，几乎没有发行这种格式的卡。



  • microSD — SD-, 11×15×1 . miniSD. , , MP3-, , . , microSD , — TransFlash (T-Flash TF).



除了上述之外，还有另一种独特的外形尺寸-nanoSD，由华为于2018年10月推出。存储卡的数据大小与nanoSIM完全相同，尺寸为12.3×8.8×0.67 mm，即比传统的microSD几乎缩小了45％。它们与智能手机Huawei Mate 20和Mate 20 Pro同时发布，仅支持使用这种格式的闪存卡，以及专有的双端口（USB + Type-C）读卡器，可以与其他型号的移动设备，台式计算机和笔记本电脑一起使用。



不幸的是，微型尺寸竟然是新卡中唯一的王牌，而就价格和技术特性而言，它们明显不如经典的microSD。例如，容量为128 GB的华为Nano SD和90 MB / s的连续读取速度，在官方商店中的售价为近6000卢布。相比之下，具有类似容量的Sandisk Extreme具有更令人印象深刻的特性（读/写速度分别高达160/90 MB / s，并具有防水，防高温和低温，防冲击和X射线辐射的保护功能）的价格为2,200–2,500卢布根据Yandex.Market（所提供的信息在撰写本文时是相关的）。

由于新格式尚未与SD卡协会达成协议，也尚未获得正式地位，因此谈论进一步发行还为时过早。鉴于美国对华为的制裁以及与中国的贸易战，nanoSD的前景变得更加模糊。

安全数字卡标准


如果采用外形尺寸，一切都非常明显，那么采用闪存卡的标准，情况就更加复杂了。迄今为止，SD规范描述了5代安全数字卡，它们在容量和其他技术特征方面有所不同。让我们分别考虑它们。

标清


该标准有两个版本：1.0和1.1。SD 1.0存储卡的范围从8 MB到2 GB，而符合1.1规范的设备则达到4 GB。这种类型的SD卡使用字节寻址和32位地址（这解释了4 GB的最大容量限制），支持FAT16和FAT32文件系统。

数据中心


符合SD规范2.0版的存储卡。它们与以前的主要区别在于对逐扇区寻址的支持，这使闪存卡的最大数量增加到32 GB。这种优势的不利方面是与面向普通SD卡的设备缺乏向后兼容性。使用的文件系统是FAT32。

数码相机


SD协会在2009年作为国际消费电子展（CES）的一部分推出的标准。该缩写代表安全数字扩展容量（“ SD卡高容量”）。SDXC存储卡的容量可以达到2 TB，并可以使用exFAT文件系统，并且还接收了UHS总线（我们将在下面讨论），该总线可以在四位模式下工作，并提供高达312 MB / s的数据传输速度。新标准的另一个功能是与以前的标准具有直接和向后兼容性：启用SDXC的设备可以与SD和SDHC存储卡一起使用，并且只要它们已预先格式化为FAT32，则SDXC存储卡可以在支持SDHC的设备中使用。

SDUC


该标准在SD规范版本7.0中。与SDXC一样，存储卡数据也使用exFAT文件系统，但是其最大容量可以达到128 TB。

SD Express


SD协会于2018年6月27日向公众提出了该标准。它立即包括三种最大容量互不相同的闪存卡：SDHC Express（最大32 GB），SDXC Express（最大2 TB）和SDUC Express最高128 TB。新一代存储卡与前代产品根本不同，因为它使用PCI Express 3.0接口和NVMe 1.3协议（在第二行触点上），这使它们能够开发高达0.9 GB / s的数据传输速度。同时，这种类型的闪存卡向后兼容支持UHS总线操作的设备。

对于SDXC和SDUC，有必要了解SD卡协会在制定标准方面处于领先地位，因为创建适当的技术和所采用规范的实际实施需要很长时间。迄今为止，容量最大，速度最快的存储卡是1 TB SanDisk Extreme：除了记录量外，该卡还显示出令人印象深刻的连续读取速度（最高160 MB / s），而如上所述，SDXC标准本身提供了最大存储卡容量高达2 TB，数据传输速度高达312 MB / s，也就是说，潜在的SDXC闪存卡容量可以提高2倍，速度可以提高两倍。



该卡的创建是通过使用最新的64层3D NAND BiCS 3D闪存实现的，该闪存是使用28纳米工艺技术制造的，我们在先前有关固态驱动器的材料中对此进行了详细介绍。与上一代三维存储器相比，单元包装密度提高了近1.4倍，这帮助我们创建了具有现代市场标准的创纪录容量的闪存卡。有前途的技术的进一步发展将使将来将三维晶体中的层数增加到128个，并创造出更符合SDUC规范的高级产品。

SD卡的速度特性


我们已经弄清楚了标准，现在该学习速度等级了。但是，为了避免进一步的混乱，有必要处理诸如UHS之类的概念。该缩写代表超高速-“超快速数据传输”。该术语具有两个含义。首先，这是总线的名称，该总线的规范首先出现在安全数字标准的第三版中。 UHS与之前版本之间的主要区别是对4位数据传输模式的支持，这使我们能够将闪存卡的性能提升到一个全新的水平。因此，其规格在安全数字3.01标准中定义的UHS-I支持50或104 MB / s的数据交换速率，而UHS-II（包括在安全数字4.0标准的更新版本中）-已经为156 MB / s或312 MB / with，而对于具有高速接口的SD卡，梦境的限制为25 MB / s。

顺便说一句，写入“ 50或104 MB / s”不是错误。这不是速度范围，而是两种可能的操作模式。 UHS-I卡能够在SDR50或SDR104模式下运行。在SDR（单数据速率）模式下，每个周期发送一个数据字，并接收一个控制命令。因此，在100 MHz的频率下，总线能够每秒传输50兆字节，而在208 MHz的频率下-已经为104 MB / s。

使用UHS-II界面，事情变得有些复杂。这种存储卡具有两行触点。



顶部的一个提供了与高速和UHS-I接口的向后兼容性，而底部的一个则允许卡以两种附加模式运行：FD156和HF312。在双工模式下使用一对低压（0.4 V）频段（而FD只不过是全双工），您可以在52 MHz的频率下实现诚实的156 MB / s，并且在半双工模式（HD，即半双工）下-相同频率下为312 MB / s，但是在任何给定时间只能在一个方向上传输数据。

缩写UHS也用于表示配备此接口的闪存卡的速度等级（全名-UHS Speed Class）。但是，在进行讨论之前，先确定使用的术语是有意义的。为此，我们转向SanDisk Extreme PLUS SDHC / SDXC UHS-I存储卡的技术规格。



在这里，数字90和60 MB / s是在某些理想情况下（客户端设备完全显示特定卡的潜力）在某些情况下（分别为顺序读取和写入）可达到的标称速度。实际上，这种情况极为罕见：大多数情况下，卡以混合模式工作，并且设备本身可能是“瓶颈”。在这方面，出现了“速度等级”的概念-在兼容设备的最坏测试条件下，最小稳定性能指标。今天有4种速度等级：

  • 速度级
  • UHS速度等级，
  • 视讯速度类
  • 应用程序性能等级。

前三个类别确定了闪存卡是否适合录制一定分辨率的视频，而应用性能类则声明了与应用一起工作时卡的高速性能，除带宽外，还考虑了每秒最少稳定的I / O操作数。

闪存卡本身的不断发展和新视频格式的传播都说明了同时存在三种分类（它们决定了卡是否适合流视频）的存在。第一类是速度类，其中包括4个性能类（从2到10 MB / s）。 UHS Speed Class是专门针对具有UHS总线的设备引入的，仅包括两个类别：1（10 MB / s）和3（30 MB / s）。 SD卡协会于2016年在横滨举行的年度CP +展会期间首次引入了Video Speed类。该类已包含在SD 5.0规范中。此分类考虑了对录制超高分辨率流视频（4K，8K和3D）的支持，并且覆盖的速度范围为6到90 MB / s。

应用程序性能类包含在SD规范5.1的增强版本中。它的开发需求很大程度上是由于出现了将闪存卡用作基于Android的移动设备中的内部存储的可能性（自从第六版的棉花糖操作系统出现了将智能手机自身的内存与闪存卡空间完全结合的可能性）。尽管存储卡在录制流视频时显示出良好的效果，但是它们的性能通常不足以与应用程序一起使用，并且出现了旨在解决此缺陷的新标准。

列出的所有分类均使用相同的缩写格式：XY，其中X是表示所用分类类型的字母，Y是表示类本身的数字。以下拉丁字母对应于闪存卡速度等级：

  • 速度级-C
  • UHS速度等级-U，
  • 视频速度类-V，
  • 应用程序性能等级-A。

工作坊：解码神秘字母


现在，有了所学知识，您就可以轻松地解密任何闪存卡的标记。让我们回到SanDisk Extreme PRO UHS-I并再次仔细阅读其规格。



让我们按顺序开始：

  • microSD-存储卡的整体尺寸为11×15×1 mm（宽×高×厚）。
  • SDXC-摆在我们面前的是配备有高速UHS总线的安全数字扩展容量卡。
  • C10-该卡在向后兼容模式下（高速总线）显示最低稳定写入速度为10 MB / s。
  • U3-该卡通过UHS总线显示的最低稳定写入速度为30 MB / s。
  • V30-该卡可用于以每秒60/120帧的频率记录分辨率高达4K的视频流。
  • A2-该卡在10 MB / s的数据传输速率下在随机读取操作中表现出最低的稳定性能，在随机写入操作中表现出4000 IOPS的水平，并且在基于Android操作系统的智能手机和平板电脑上扩展了内部存储器的性能。

如您所见，值得对该问题进行一点了解，并且卡片的标记为我们提供了有关其性能，用途和兼容设备的全面信息。现在，您已经足够了解各种规格，并选择了完全满足您需求的闪存卡。
