Tinjauan umum tentang keyboard mekanis Vortex Core RGB

Saya tidak mengerti apa pun di keyboard, dalam hidup saya hanya ada Genius, Logitech dan keyboard poppy. Saya ingin mencintai keyboard, tetapi mereka membuat saya kesal. Saya tidak bisa belajar mencetak secara membabi buta, saya tidak bisa membuka tombol, saya tidak menjangkau beberapa, saya perlu mengingat tata letak yang berbeda di mana tombol yang sama bisa berada di tempat yang berbeda. Selain itu, ini membuat marah bahwa tata letak standar tidak nyaman untuk pemrograman: semua tanda kurung dan tanda dekat jari kelingking tangan kanan, kelebihan beban, tidak ada kesenangan. Panah dibuat untuk tangan kanan, dan ada juga mouse di dekatnya, Anda sering harus menggerakkan tangan Anda.

Ketidakpuasan saya akhirnya menghabisi posting Nikita tentang desain keyboard. Saya menyadari bahwa ini bukan satu-satunya dan Anda dapat mencari beberapa opsi. Saya pergi ke ruang pamer dan memilih yang paling tidak biasa: keyboard kompak di 40%, dua spasi, panah di tangan, kurung di tengah. Saya tidak bisa menolak, dan setelah beberapa bulan digunakan, saya siap untuk memeriksanya.



Berapa? Mahal


Keyboard adalah perangkat elektronik yang cukup sederhana: puluhan tombol, papan, pengontrol untuk pengenalan sinyal dan penutup.

Pada keyboard biasa, alih-alih tombol, ada membran, tombol memantul karena sifat kenyal karet. Ini menurunkan harga keyboard, tetapi tidak dapat diperbaiki dan sulit untuk memberikan pengalaman pencetakan khusus.

Jika tidak: letakkan mekanisme pegas terpisah di setiap tombol. Kemudian ternyata untuk mengontrol sifat menekan tombol, waktu respons dan suara tombol. Mekanisme untuk kunci semacam itu telah ditemukan sejak lama, Anda dapat membacanya di Wikipedia .

Keyboard membran harganya sangat sedikit, harga mulai dari 100 rubel. Keyboard mekanik lebih mahal karena membuat lusinan mekanisme lebih sulit. Karena itu, harga cepat naik menjadi 10.000 rubel, mudah ditemukan dan lebih mahal. Seiring dengan harga, kualitas keyboard dan perasaan menggunakannya semakin meningkat.

Dari semua keyboard di ruang pamer, saya menyukai Vortex Core RGB, itu yang paling tidak biasa: ringkas, panah langsung dalam huruf. Harganya 10.000 rubel, tetapi saya meyakinkan diri saya untuk membelinya, karena ini adalah alat utama untuk bekerja. Omong-omong, Apple Magic Keyboard harganya sama.

Penting bagi saya bahwa saya bisa mengetik di kamar tempat bayi itu tidur, jadi saya membeli keyboard saya dengan sakelar linear tenang. Ternyata benar-benar tenang, tetapi tombol linier tidak cocok untuk saya: kadang-kadang saya menyentuh tombol tetangga dan saya tidak mengerti apakah tombol itu ditekan atau tidak. Tetapi mengklik itu sangat bagus.

Lima lapisan dan memuat redistribusi


Bahkan keyboard paling biasa memiliki setidaknya 2 lapisan: yang normal dan setelah menekan Shift. Juga sering dimungkinkan untuk menahan Alt dan mendapatkan akses ke karakter langka.
Core Vortex memiliki tombol yang jauh lebih sedikit, tetapi fungsinya tidak menghilang di mana pun, tetapi telah pindah ke lapisan tambahan (pada keyboard mereka ditandatangani di sisi berwarna merah, hijau dan biru). Mereka diaktifkan dengan menjepit tombol fungsi:

  • Fn akan memberikan akses ke panah dan multimedia.
  • Fn1 - ke angka dan tombol-F.
  • Fn1 + Shift - ke tanda-tanda khusus.



Anda bisa terbiasa dengan tata letak ini dalam sebulan. Belajar jauh lebih mudah daripada beralih ke DVORAK atau COLEMAK, di mana surat-surat mengubah tempat mereka. Di sini, pengaturan baru hanya untuk kunci tambahan dan beberapa huruf Rusia.

Membeli keyboard ini, saya harapkan:

  • Panah di lokasi yang nyaman.
  • Kurung di tengah keyboard ditekan dengan jari yang berbeda, bukan hanya jari kelingking.
  • Angka dan tanda yang tidak perlu Anda jangkau.

Sebenarnya, sesuatu menjadi lebih nyaman, tetapi minus yang tak terduga juga muncul, jadi saya akan memberi tahu Anda detailnya.

Panah


Pada keyboard biasa, panah berada di sebelah kanan, yang jauh dari posisi dasar lengan. Karena itu, tangan harus dipindahkan tidak hanya ke mouse, tetapi juga ke panah. Namun semakin sedikit gerakan, semakin mudah bekerja. Anda dapat mengatasi masalah ini dengan menggerakkan panah pada tombol IJKL. Di keyboard ini, panah berdiri sebagaimana mestinya:



Di sebelah kanan ruang adalah tombol Fn , jika Anda menahannya, semua tanda tangan biru pada keyboard akan tersedia . Saya hanya membutuhkan panah dan tombol volume. Mudahnya, Fn terletak tepat di bawah ibu jari tangan kanan dan ditekan dalam satu gerakan. Saya terbiasa dalam setengah jam.

Angka dan F1-F12




Ada tombol pengubah lain - Fn1 . Ini menggantikan tombol dengan teks merah : angka muncul di garis dasar, dan tombol F di tempat qwerty. Saya juga tidak harus terbiasa dengan ini untuk waktu yang lama. Rasanya aneh bagi saya bahwa angka-angka digeser ke kiri dengan satu tombol - 1 di tempat Tab.

Tanda kurung dan tanda


Dengan tombol biru dan merah disortir, tetap hijau . Di sini kekecewaan utama menyusul saya - saya berharap mereka akan dihidupkan dengan tombol Pn, tetapi ternyata mereka akan dihidupkan dengan kombinasi tombol Fn1 + Shift .

Dan ini adalah kesalahan keyboard paling penting! Untuk menekan kedua tombol, Anda harus menggeser tangan Anda ke kanan, tetapi kemudian sulit untuk mencapai tombol tengah. Ketegangan seluruh tangan terlihat pada ibu jari, meskipun tidak terlibat dalam proses:



Akibatnya, seluruh tangan bergeser, dan kenyamanan keyboard kecil hilang. Baik sekali Anda dapat merombak keyboard. Hal pertama yang saya lakukan adalah memindahkan Fn1 ke ruang yang tepat, dan masalahnya hilang.

Semua kurung berada di bawah jari telunjuk dan tengah, dan ini sangat nyaman. Saya harus membiasakan diri selama sekitar satu minggu.

Tombol Pn


Jika Pn tidak menyertakan tombol hijau, lalu apa fungsinya? Ternyata Pn hanya mengontrol lampu latar. Anda dapat memilih warna, mode berkedip, setelah menekan tombol, lampu mungkin berbeda dalam gelombang, tetapi semuanya tampak berlebihan. Tombolnya terlalu penting dan terlalu jarang digunakan. Tetapi di Tahun Baru +1 karangan bunga.

Huruf Rusia


Dan kemudian rasa sakit mulai. Namun untuk memahaminya, Anda membutuhkan latar belakang kruk di dunia keyboard.

Klava terdiri dari bagian fisik dan perangkat lunak. Perangkat lunak telah tenang, kami menyambungkan keyboard apa pun ke komputer mana pun, dan biasanya itu mulai berfungsi segera. Kode kunci tetap ada sejak zaman kuno, dan bersama mereka modifikator mereka. Oleh karena itu, Russian X akan berada pada kunci bahasa Inggris {, sehingga dimasukkan ke dalam sistem. Anda dapat mengubah tata letak di komputer, tetapi kemudian Anda tidak bisa hanya menyambungkan keyboard dan mulai bekerja, Anda harus memasang tata letak. Karena kerumitan ini, beberapa orang repot.

Ada lebih banyak huruf Rusia daripada bahasa Inggris, jadi mereka ditempatkan pada segala macam tanda-tanda khusus. Jika kurung dipindahkan lebih dekat ke tengah (nyaman), maka tombol Rusia akan pergi bersama mereka (tidak nyaman). Jadi huruf E, E, X, Kommersant berada di tengah-tengah lapisan merah. Lokasi baru harus belajar paling lama, karena tombol-tombol ini tidak masuk dan harus ditemukan oleh pencarian lengkap. Kunci ekstra


Pada keyboard kecil, setiap tombol harus bermanfaat. Tetapi saya segera menemukan beberapa kunci tambahan. Ada cukup omong kosong:

  1. Ruang kedua. Secara default, tidak ada manfaatnya.
  2. Tombol Pn mengontrol cahaya latar, tetapi akan lebih baik jika menyertakan lapisan hijau.
  3. Tombol Fn1 tidak nyaman. Anda dapat memindahkannya ke ruang yang tepat.
  4. Tombol Del, meskipun Backspace berada di sebelahnya. Jika Anda benar-benar ingin, Anda bisa melakukannya melalui Fn + Backspace.
  5. Alt Kanan dan Menang. Apakah ada yang menggunakannya?

Pintasan menjadi berbeda


Saya menggunakan banyak cara pintas saat memprogram. Beberapa dari mereka terhubung dengan tanda kurung. Karena sekarang kurung berada di tempat baru, maka gerakan pintas menjadi berbeda. Untuk mengubah kebiasaan ini adalah yang paling sulit, untuk setiap pintasan Anda harus melalui 3 tahap:

  1. ingat tombol mana yang saya tekan sebelumnya
  2. mengerti di mana harus mengklik sekarang,
  3. untuk menghafal gerakan ini.

Akibatnya, saya bahkan lupa beberapa jalan pintas, karena malas mengajar begitu banyak. Dan beberapa dari mereka menjadi sangat panjang sehingga tidak mungkin untuk menekan mereka bahkan dengan dua tangan, dan saya berhenti menggunakannya.

Layers dan Macro


Kami sudah berurusan dengan empat lapisan: biasa, merah, biru dan hijau. Tapi bukan itu saja! Selain itu, Anda dapat merekam 4 lapisan lagi dan beralih ke mereka jika perlu menggunakan Fn. Tapi bukan itu saja: keyboard dapat menyimpan makro, dan Anda bahkan tidak perlu program untuk merekamnya, semuanya dilakukan dengan benar di keyboard.

Mungkin ini sangat penting bagi seseorang, tetapi saya belum menemukan aplikasi. Saat mengganti layer, LED menyala dan menyala sepanjang waktu, dan ini mengganggu.

Ergonomi


Keyboard-nya kecil. Ini lebih kecil dari Apple Magic Keyboard yang ringkas, dan dibandingkan dengan yang lainnya, sepertinya kecil sekali. Sebagai perbandingan, foto menunjukkan Optimus Maximus 120%.



Saya bekerja pada laptop, dengan keyboard tambahan itu menjadi tidak nyaman pada awalnya: kedalaman meja tidak cukup untuk keyboard dan laptop. Dudukan tidak hanya menyelesaikan masalah ini, tetapi juga memperbaiki posisi kepala, Anda tidak perlu lagi melihat ke bawah.



Keyboardnya tinggi. Saya biasa mencetak di laptop, kuncinya ada di bidang meja, dan yang ini tiga sentimeter lebih tinggi. Tempat sikat membuat pencetakan lebih nyaman.
Keyboard berbentuk persegi panjang seperti batu bata: baik kasing atau tombol dimiringkan. Pada awalnya, tidak nyaman bagi saya untuk mencapai baris atas, dan kemudian saya menyadari bahwa secara umum saya memegang tangan saya secara tidak benar. Sikat saya hampir lurus, tetapi saya harus meremas jari-jari saya sehingga ditempatkan secara vertikal di baris utama.

Micro-USB terletak di atas. Jika Anda menggunakan kabel standar, itu akan menjulur dari keyboard dan mengganggu meletakkannya dekat dengan monitor. Kawat berbentuk A dari Auchan sepenuhnya mengatasi masalah ini.



Bergerak, tetapi rentan.Karena ukuran keyboard, nyaman untuk dibawa, tetapi tombol sudut terlalu dekat ke tepi, sehingga mudah terbang. Setiap kali saya mengeluarkannya dari tas kerja, saya harus mencari salah satu tombol dan meletakkannya di tempatnya. Kemungkinan besar, gantungan kunci usang, dan seiring waktu masalah akan menjadi lebih akut. Ini diselesaikan dengan tas kain, tetapi tidak termasuk.

Keyboardnya berat, meskipun ukurannya. Pada dasarnya adalah logam. Portofolio saya hanya miliknya, tetapi jika Anda mengenakan sesuatu yang lain, maka bobot ekstra 400 gram dapat mengganggu. Sebagai perbandingan: Vortex Core RGB dua kali lebih berat dari keyboard Magic, tetapi sedikit lebih ringan dari iPad.

Berkedip


Tata letak dapat (dan harus) diubah. Keyboard mendukung lapisan utama dan hingga empat yang tambahan, Anda dapat beralih di antara mereka dengan pintasan. Anda dapat merekam makro langsung dari keyboard (saya tidak menggunakan ini), atau Anda dapat menulis ulang tata letak dari komputer (saya sering menggunakannya).

Anda dapat mengunduh firmware dari pabrikan, yang di situs yang sama memungkinkan Anda mengonfigurasi keyboard . Menginstal firmware hanya mungkin dengan Windows, jadi saya harus berkunjung. Firmware baru membuat keyboard menjijikkan: tombol Rusia berangkat ke tempat baru, dan dari Caps Lock LED mulai menyala tepat di mata. Anda masih tidak dapat mengubah lapisan standar.

Untung Freddie Wang telah mengalami masalah ini . Dia membuat situs web dan firmware, yang memungkinkan Anda untuk mengubah lapisan utama.

Modifikasi tata letak saya


Saya mengatur Fn1 ke ruang yang tepat untuk menyalakan tombol merah. Menjadi lebih mudah untuk menekannya bersama dengan Shift untuk satu set karakter, tetapi ternyata saya banyak menekan spasi dengan tangan kanan saya, saya harus melepaskannya. Di tempat pabrikan, saya akan melangkah lebih jauh dan memecah ruang menjadi tombol yang lebih kecil.

Memindahkan tombol dan / dari lapisan tambahan ke yang utama, mereka terlalu diperlukan untuk pemrograman dan pintasan Xcode. Mereka berada di tengah, mendorong mereka ke posisi yang lebih akrab. Huruf Rusia juga bergerak bersama mereka: E ternyata berada di kanan atas, dan E di kanan bergeser pada layer Fn1. X dan Kommersant tetap berada di lapisan merah.




Saya menyukai format keyboard empat puluh persen. Pelatihan ulang untuk tata letak tombol yang berbeda tidaklah sulit, tetapi mungkin ada masalah dengan bahasa Rusia dan pintasan. Saya menjadi terbiasa dalam waktu sekitar satu bulan, tetapi keyboard kehilangan aksesibilitas untuk pengguna komputer saya yang lain.

Keyboard Vortex Core RGB meninggalkan kesan beragam. Di satu sisi, dia membuat pekerjaan saya lebih nyaman: set itu sendiri menjadi lebih menyenangkan, tombol untuk pemrograman ada di tempat yang tepat. Di sisi lain, saya tidak bisa merekomendasikan hal ini kepada semua orang: Saya harus menyelesaikan bahasa Rusia saya, tetapi saya harus terbiasa dengan tata letak tombol.

Sayangnya, saya tidak tenang. Setelah berlatih pada simulator keyboard, saya menyadari bahwa pergeseran tombol secara horizontal mengganggu saya. Fingers tidak bergerak seperti itu, tetapi saya tidak terbiasa menggeser tombol. Simulator memperbaiki posisi tangan saya ke yang lebih benar, tetapi sendi pergelangan tangan menjadi bosan dengan ini.


Pada akhirnya, saya mencari keyboard split ortho-linear. Tentang apa itu dan apakah itu membantu saya - lain kali. Sementara itu, foto-foto keyboard baru dapat ditemukan di saluran DodoMobile .
