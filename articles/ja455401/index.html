簡単な例におけるCisco IOSのTcl

ゼロから同じタイプの数十のシスコアクセススイッチを展開する必要があると想像してください。一般的な構成には、ホスト名とドメイン、デフォルトゲートウェイ、パスワード、ユーザーのリスト、SVIのIPアドレス、VLAN番号、アップリンクトランクの設定などが含まれます。毎回手で入力するのは非常に長く、非生産的です。もちろん、一般的な構成を作成して（T）FTP経由でアップロードできますが、最初に、コンソールから少なくとも最小限の構成が必要です。次に、可変構成パラメーターを変更する必要があります。これら（および他の多くの）タスクを解決するために、Cisco IOSには強力な自動化ツールである組み込みTclインタープリター（Cisco IOSスクリプトw / Tcl）が含まれています。

Tclとは


Tcl（読み取り可能な「くすぐる」、時々「流れる」）は、コンソールアプリケーションに埋め込むために80年代後半に開発されたインタプリタ型プログラミング言語です。最新のTclの機能の範囲は非常に広く、OOPや高度な正規表現ツール、動的配列などがサポートされています。

この言語のサポートは、Cisco IOS 12.2（3）Tプラットフォームで最初に登場し（一部のソースでは12.3（2）で示されていますが、これの確認は見つかりませんでした）、現在いくつかのオプションがあります。

  • コマンドラインインターフェイスを備えたTclインタープリター。これは、IOS XEおよびXRを含むCisco IOSプラットフォームのさまざまなバージョンに統合されており、幅広いデバイスで利用できます。Tclコマンドを実行したり、既成のスクリプトをファイルとして実行したりできます。オペレーティングシステムとしてiOSではなく、Cat OSやASA（同じ名前のファイアウォール内）を使用するデバイスには、インタープリターコマンドラインが含まれていません。
  • T.N. 「組み込みイベントマネージャー」またはEEM-リアルタイムで自動的に応答できるイベントトラッキングシステム。たとえば、電子メール通知でリモートホストを監視します。EEMスクリプト（アプレット）はTclで記述されていますが、EEM自体は個別のTclコマンドラインを提供していません。使用例はこちらをご覧ください。EEMは、バージョン9.2（1）以降のNexus（NX OS）およびASAプラットフォームで使用できます。
  • 音声メニューシステムIVR（インタラクティブ音声応答）。

それでは、実際的な問題の解決に移りましょう。私たちの前には、48ポートのCat2950Sがあります。以下のスクリプト

  • コンソールからの要求sw_numのシリアル番号
  • switch_ <sw_num>の形式でホスト名を設定します
  • 特権EXECコンソールのパスワードを要求および設定します
  • 入力されたスイッチ番号に従って、Fa0制御インターフェース（192.168.0.x）とVlan1インターフェース（10.0.x.254）のアドレスを構成します
  • ポートベースのDHCP予約と48個のアドレスのプールを作成します。各クライアントに1つのIPアドレスが予約され、その下位オクテットは、このクライアントが接続されているポートのシリアル番号と同じです。

puts "Enter Switch number:"
set sw_num [gets stdin]
ios_config "hostname switch_$sw_num"

puts "Enter password (secret):"
set pass [gets stdin]
ios_config "enable secret 0 $pass"

ios_config "line 0 16" "password 0 $pass" "login"

ios_config "int fa0" "ip address 192.168.0.$sw_num 255.255.255.0" "no shut"
ios_config "int vlan1" "ip address 10.0.$sw_num.254 255.0.0.0" "no shut"

ios_config "ip dhcp use subscriber-id client-id"
ios_config "ip dhcp subscriber-id interface-name"

#    48       subscriber-id
for {set i 1} {$i <= 48} {incr i} {ios_config "int Gi1/0/$i" "ip dhcp server use subscriber-id client-id"}

ios_config "ip dhcp pool POOL1" "network 10.0.0.0 255.0.0.0" "reserved-only" "default-router 10.10.0.254"
#    48  IP-,   . 
for {set i 1} {$i <= 48} {incr i} {ios_config "ip dhcp pool POOL1" "address 10.0.$sw_num.$i client-id Gi1/0/$i ascii"}
#
