RおよびPowerShellを使用した仮想マシンステータスの日次レポート


前書き


こんにちは。半年の間、私たちは仮想マシンの状態に関する（そしてそれだけではない）レポートを生成するスクリプト（より正確には、スクリプトのセット）を実行してきました。作成体験とコード自体を共有することにしました。私はこの資料が誰かに役立つかもしれないという批判と事実に依存しています。


形成が必要


多数の仮想マシンがあります（3番目のvCenterに分散された約1,500のVM）。新しいものが作成され、古いものはかなり頻繁に削除されます。順序を維持するために、vCenterにいくつかのカスタムフィールドが追加され、VMをサブシステムに分割し、それらがテストされているかどうか、および誰がいつ行ったかを示します。人的要因により、車の半分以上に空白のフィールドが残され、作業が複雑になったという事実につながりました。 6か月に1度、誰かがびっくりしてこれらのデータの更新に取り掛かり始めましたが、結果は1週間半ほど関連性がなくなりました。
機械を作成するためのアプリケーションや、それらを作成するためのプロセスなどがあることを誰もが理解していることをすぐに明確にします。等そして、このすべてのプロセスが厳格に順守されていますが。残念ながら、これは私たちには当てはまりませんが、これは記事の主題ではありません:)


一般に、フィールドへの入力の正しさの検証を自動化することが決定されました。
私たちは、責任のあるすべてのエンジニアとその上司のために、誤って入力されたマシンのリストが記載された毎日の手紙が良いスタートになると判断しました。


この時点で、同僚の1人が既にPowerShellスクリプトを実装しており、毎日、スケジュールに従ってすべてのvCenter-sのすべてのマシンの情報を収集し、共有ディスクに配置された3つのcsvドキュメント（それぞれが独自のvCenter）を形成していました。このスクリプトをベースとして、私がある程度経験したR言語を使用したチェックで補足することにしました。


改良の過程で、ソリューションはメールで通知し、メインテーブルと履歴テーブルを備えたデータベース（これについては後で詳しく説明します）に加えて、vSphereログを分析してvmの実際の作成者とそれらが作成された時刻を見つけることで大きくなりました。

    #  ,       ,     $Using

    #

        Write-Host ""

        Write-Host ""

        #    ,        
        $con = Connect-VIServer -Server $Using:vCenterNames -User $Using:vCenterUsername -Password $Using:vCenterPassword

        #   vm
        $vm = Get-VM -Name $Using:row

      #  2  .     ,  - .   ,
      $Event = $vm | Get-VIEvent -Start $Using:start -Finish $Using:finish -Types Info | Where { $_.Gettype().Name -eq "VmBeingDeployedEvent" -or $_.Gettype().Name -eq "VmCreatedEvent" -or $_.Gettype().Name -eq "VmRegisteredEvent" -or $_.Gettype().Name -eq "VmClonedEvent"}

      #      ,      -
      If (($Event | Measure-Object).Count -eq 0){
         $User = $Using:UnknownUser
         $Created = $Using:UnknownCreatedTime
         $CreatedFormat = $Using:UnknownCreatedTime
      } Else {
         If ($Event.Username -eq "" -or $Event.Username -eq $null) {
            $User = $Using:UnknownUser
         } Else {
         $User = $Event.Username
         } # Else
            $CreatedFormat = $Event.CreatedTime
            #     ,      ,   .      .
            $Created = $Event.CreatedTime.ToString('yyyy-MM-dd')
         } # Else

      Write-Host "Creator for $vm is $User. Creating object."

      #  .  .
      $Vmresult = New-Object PSObject
      $Vmresult | add-member -MemberType NoteProperty -Name "VM Name" -Value $vm.Name  
      $Vmresult | add-member -MemberType NoteProperty -Name "CreatedBy" -Value $User
      $Vmresult | add-member -MemberType NoteProperty -Name "CreatedOn" -Value $CreatedFormat
      $Vmresult | add-member -MemberType NoteProperty -Name "CreatedOnFormat" -Value $Created           
      #

    } # Inline

} # ForEach
