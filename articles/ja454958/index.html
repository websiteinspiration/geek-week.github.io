事実＃3：スイスの政治システムはユニークです


事実2から、スイスは直接民主主義が可能で効果的な世界で非常に数少ない国の1つであることがわかります。はい、スイスはあらゆる理由で意志を表明するのが非常に好きです-砲撃を使用して雪崩を発進させてコンクリートまたはより生態学的な木から家を建てるかどうかから（スイスにはたくさんの原材料がありますが、この桟橋は自然の美しさを殺し、実際：醜いが、「美しい」木で、緊張した）。

ここでの主なこと-普遍的および普遍的な投票を擁護する熱心な中で-800万人強がスイスに住んでいることを思い出し、あらゆる問題に対する投票を組織することは比較的面倒な仕事です。統計情報の収集も簡単です。ログインパスワードを記載したメールを送信して入力するだけです。


これは、統計収集システムがどのように見えるかです。これまでのところ、投票するには自分で投票する必要がありますが、投票権を持っているのは市民だけです。

ちなみに、それは非常に便利で、毎年便利な統計を生成することができます。たとえば、スイスの過去150年間の人口統計データを1つのファイルに格納します。

事実＃4：スイスの強制兵役


ただし、サービス自体はストラップを引っ張るのではなく、フェンスから日没まで祖国への借金を継続的に返済するのではなく、45歳までの男性のための必須の健康キャンプです。確かに、子供の頃の最初の40年間は、男性の人生の中で最も困難です。雇用主にも、従業員が料金を請求された場合に拒否する権利はなく、費やされた時間（通常1〜2週間）は全額支給されます。

事実5：スイス-チーズ、チョコレート、ナイフ、時計だけではありません


スイスという言葉で多くの人々は、チーズ（グリュイエール、エメンタラー、またはティルシター）、チョコレート（通常、すべての免税で販売されているため、トブラロン）、アーミーナイフ、そして非常に高価な時計を思い出します。
スウォッチグループウォッチ（ティソ、バルメイン、ハミルトンなどのブランドも含まれます）の購入を検討している場合、最大1,000フランまで、ほとんどすべての時計が同じ製造元で製造され、すべての時計の充填量はほぼ同じです。アッパーレンジ（ラドー、ロンジン）からのみ、少なくともいくつかの「チップ」が表示されます。

実際、スイスはこのような世界的な秩序を採用しており、その技術は国内で作成および開発され、その後国内から輸出されています。最も有名な例は、第二次世界大戦中にドイツ国防軍とクリーグマリンによって使用されたネスレミルクパウダーとエリコンライフルバレルです。同時に、国は独自のマイクロエレクトロニクス（ABB-電力、EMマイクロエレクトロニクス-RFID、スマートカード、スマートウォッチなどの商品の命名法リストに従って）、独自の複雑なコンポーネントとアセンブリの製造、独自の列車のアセンブリ（2階建てのボンバルディア）を製造しています。、たとえば、Villeneuveの下で収集され、さらにリストに含まれます。そして、農場の大部分がスイスにあるという事実について-私は巧みに黙っています（バーゼルと地区のシエール、ロシュとノバルティスの新しいクラスターのロンザ、ローザンヌ（ローザンヌ）のデバイオファームとマーティンと（マルティニー）と多くの新興企業や中小企業）。

ファクトNo. 7：「礼儀正しさ」


正直に言って、最後にサービススタッフに「こんにちは」と「ありがとう」と言ったのはいつですか。そしてスイスでは、小さな村で息を吸ったり吐いたりするのと同じ習慣が強まっています。たとえば、ここではほとんどすべての人が会話の冒頭でbonjour / guten Tag / buongiorno（こんにちは）、あるサービスの後でmerci / Danke / gracie（ありがとう）とbonne journee /Tschüss/ ciao（こんにちは）はさようならと言う必要があります。そしてハイキングでは、出会った誰もがあなたを迎えてくれます-すごい！

そして、これはアメリカ人の「ハワイ」ではありません。胸のどこかにいる人が、背を向けた途端にチョップするために斧を持っているときです。スイスでは国が小さく、最近までかなりの「村」の人口がいるため、自動化ではありますが、米国よりも誠実に皆が挨拶します。

しかし、スイス人のおもてなしと優しさにだまされないでください。帰化に関する最も厳しい法律のいくつかが国で適用されていることを思い出させてください。これには、実務生活、言語の知識、および試験が含まれます。外側は親切で、内側は少しナショナリストです。

事実＃8：スイスの村はすべての生きているよりも生きています


驚くべきことに、それは事実です。スイスでは、村は絶滅するだけでなく、かなり発展し拡大しています。それは、山羊や牛が飛び跳ねている生態学や緑の芝生の問題ではなく、純粋に経済的です。スイスは連邦であるため、税金（特に個人所得税）はここで3つのレベルで支払われます：地方（村/市）、州（「地域」）、連邦。連邦はすべて1つですが、家族が「村」に住んでいる場合、他の2つの「操作」は言葉の意味で-大幅に減税できます。

次のパートで税金について詳しく説明しますが、ここでは、ローザンヌの場合、つまり都市に住んでいる場合、条件付きの税負担は1人あたり約25％であり、同じヴォー州にある見捨てられた村の場合は、たとえば、 Mollie-Margotさん、15〜17％になります。あなたが自分で家を維持し、芝生を刈り取り、車と街で働く方法にお金を払わなければならないので、この違いのすべてがポケットに入るわけではないことは明らかですが、住宅価格は低く、食べ物は農場であり、子供たちは広大な広がりの中を走り回っています。

そして、はい、結婚に対して非常に奇妙な態度があります。子どものいない家族に対する課税は1人の個人の課税を大幅に超える場合があるため、スイスは地元の登記所に逃げることをそれほど急いでいません。経済は経済的でなければなりません。この問題については国民投票さえ行われた。しかし、次の部分の税金についてです。
