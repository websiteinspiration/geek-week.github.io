import streamlit as st
import pandas as pd

@st.cache
def load_metadata():
    DATA_URL = "https://streamlit-self-driving.s3-us-west-2.amazonaws.com/labels.csv.gz"
    return pd.read_csv(DATA_URL, nrows=1000)

@st.cache
def create_summary(metadata, summary_type):
    one_hot_encoded = pd.get_dummies(metadata[["frame", "label"]], columns=["label"])
    return getattr(one_hot_encoded.groupby(["frame"]), summary_type)()

# Piping one st.cache function into another forms a computation DAG.
summary_type = st.selectbox("Type of summary:", ["sum", "any"])
metadata = load_metadata()
summary = create_summary(metadata, summary_type)
st.write('## Metadata', metadata, '## Summary', summary)

StreamlitはGPUで動作するように設計されています。Streamlitを使用すると、TensorFlow、PyTorch、その他の同様のライブラリを直接操作できます。たとえば、このデモでは、StreamlitのキャッシュにNVIDIAの有名人のGANが保存されています [2]。これにより、スライダーの値を変更するときにほぼ瞬時の応答を実現できます。


Streamlitアプリは、Shaobo GuanのTL-GAN  [3] を使用して、NVIDIA [2]の有名人のGANを示します。

Streamlitはオープンソースプロジェクトです。許可を求めることなく、Streamlitでアプリケーションを自由に配布できます。インターネットに接続せずにStreamlitでローカルにアプリケーションを実行することもできます！また、既存のプロジェクトはStreamlitを徐々に実装できます。
