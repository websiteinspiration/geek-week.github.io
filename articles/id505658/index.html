Baru-baru ini, di bawah ilham dari saluran The Coding Train , saya memutuskan untuk berpartisipasi dalam salah satu tantangan 10 menit di mana saya harus menciptakan ilusi terbang di luar angkasa dengan kecepatan tinggi.

Untuk mengimplementasikan proyek, saya memilih p5.js yang terkenal - pustaka untuk JavaScript yang dirancang untuk membuat karya seni dengan cara algoritmik. Mengapa tidak mungkin menggunakan paket grafik standar dari Adobe?

Mari kita membuat fungsi baru yang disebut Star (). Seharusnya memiliki empat parameter utama: x, posisi y dan variabel z, di mana kita akan mengubah segalanya. Untuk membuat kemunculan acak dari titik awal, kita dapat menggunakan fungsi acak (), dan sebagai parameter mengatur batas layar kita dalam lebar dan tinggi.

Untuk mendapatkan pergerakan bintang yang indah dari tengah ke tepi, mari kita lihat bagaimana posisi harus berubah, yaitu variabel x dan y. Setiap bintang akan muncul pada sinar imajiner dan mengikutinya ke tepi layar.

Itu saja, kami terbang, tetapi ada dua masalah. Yang pertama - karena titik referensi koordinat secara default tidak di tengah, tetapi di sudut kiri atas, kita hanya melihat bagian bintang-bintang yang tidak memiliki nilai koordinat awalnya negatif. Sisanya tidak terlihat.



Untuk menyelesaikan masalah ini cukup mudah - Anda hanya perlu memindahkan pusat melalui perintah translate (lebar / 2, tinggi / 2).



Masalah kedua adalah bahwa bintang-bintang tidak berubah ukurannya ketika mendekati pengamat. Pada tingkat bawah sadar, kita merasa ada sesuatu yang tidak beres.

Untuk mengatasi masalah ini, mari kembali ke elips kita, buat variabel lain r, yang juga akan bergantung pada z dan akan menentukan parameter elips ketiga dan keempat, yang bertanggung jawab atas diameter horizontal dan vertikal. Kode terakhir adalah sebagai berikut:

let stars = [];
let speed = 25;

function setup() {
  createCanvas(600, 600);
  for (let i = 0; i < 800; i++) {
    stars[i] = new Star();
  }
}

function draw() {
  background("black");
  translate(width / 2, height / 2);
  for (let i = 0; i < stars.length; i++) {
    stars[i].update();
    stars[i].show();
  }
}

function Star() {
  this.x = random(-width, width);
  this.y = random(-height, height);
  this.z = random(width);
 
  this.update = function() {
    this.z = this.z - speed;
    if (this.z < 1) {
      this.x = random(-width, width);
      this.y = random(-height, height);
      this.z = width;
    }
  };

  this.show = function() {
    fill("white");
    var sx = map(this.x / this.z, 0, 1, 0, width);
    var sy = map(this.y / this.z, 0, 1, 0, height);
    var r = map(this.z, 0, width, 10, 0);
    ellipse(sx, sy, r, r);
  };
}
