SILとSalesforce

この記事では、SILからSalesforceを操作する方法について説明します。

SILは、Atlassian JiraとConfluenceでアクションを自動化するためのプログラミング言語です。SILについて詳しくは、  こちらをご覧ください。

SILからSalesforceを操作するための3つのオプションがあります。

  1. Power Salesforceコネクタプラグインを使用します  。
  2. SILで関数を記述します。
  3. あなたの書くSILの拡張子を。

この記事では、最初の2つのオプションについて説明します。SIL用の独自の拡張機能を作成する方法については、こちらをご覧ください。

ただし、コードに進む前に、Salesforceを構成する必要があります。

Salesforceの設定


Salesforce Rest APIを 使用してSalesforceを操作するため、最初に接続アプリケーションを構成する必要があります。
プラグインアプリケーションを構成するには、ギア->セットアップ：



アプリを選択->アプリマネージャー：



[新しい接続済みアプリ]ボタンをクリック：



OAuthを有効にするオプションを選択します（OAuth設定を有効にする）。フルアクセスをインストールし  、[ 追加 ]ボタンをクリックしてから、[  保存 ]ボタンをクリックします  。

次のページのOAuth設定の下に、コンシューマキーとコンシューマシークレットを開くオプションが表示されます。これら2つのパラメーターを覚えておいてください。 Salesforceとの接続を確立するために必要になります。

さらに、ユーザーシークレットトークンを生成する必要があります。これを行うには、「ユーザー設定」→「セキュリティトークンをリセット」に移動します。新しいシークレットユーザーキーが電子メールに送信されます。



これで、SILからSalesforceを操作するために必要なものがすべて揃いました。

パワーセールスフォースコネクター


Power Salesforceコネクタを使用すると、Salesforce内の取引先や商談などのオブジェクトを操作できます。さらに、SOQLでクエリ結果を取得できます。SILプラグインで利用可能なすべての関数は、ここで見つけることができます。

プラグインを使用する利点は、Salesforceで動作するようにSIL関数を記述して、Salesforce Rest APIの複雑さを理解する必要がないことです。これにより、技術的なタスクではなく、ビジネスタスクに集中できます。

そこで、Power Salesforceコネクタを使用することにしたとしましょう。

最初にこのプラグインをAtlassian Marketplaceからインストールし、次にAtlassian JiraからSalesforceへの接続を作成する必要があります（ギア->アプリの管理-> SFDC接続構成）：



[接続の追加]ボタンをクリックします：



必要なフィールドに入力して[保存]ボタンをクリックします。パスワードを入力するときは、ユーザーのパスワードを入力してから、ユーザーの秘密鍵を入力する必要があります。

これで、SILでコードを記述できます。

まず、Salesforceから商談を選択します。

SIL Managerに移動して、新しいファイルを作成し、次のコードを記述します。

SFDCConnection sfdcConnection = connectToSalesforce("My SFDC Connection");
SFDCOpportunity opp = sfdcGetOpportunity(sfdcConnection, "0064F00000NKA7CQAX");
runnerLog(opp);

ご覧のとおり、Opportunityデータを書き込むために、コードを2行だけ記述しました。最初の行で、Salesforceに接続します（私の接続をMy SFDC Connectionと呼びます）。2行目では、商談IDを使用してSalesforceからデータを取得します。

Opportunityの一部のフィールドを変更するために、次のコードを記述します。

SFDCConnection sfdcConnection = connectToSalesforce("My SFDC Connection");
SFDCOpportunity opp;
opp.Description = "My new description";
sfdcUpdateOpportunity(sfdcConnection, "0064F00000NKA7CQAX", opp);

ご覧のとおり、コードは非常にシンプルです。さらに、プラグインのドキュメントには、プラグインに実装されている各関数の例が含まれています。例は  ここにあります。

しかし、Power Salesforceコネクタがあなたに合わない場合はどうなりますか？この場合、独自のSIL関数を作成できます。

独自のSIL関数を作成する


まず、Salesforce接続関数を作成します。

struct SFDCConnection {
    string access_token;
    string instance_url;
    string id;
    string token_type;
    string issued_at;
    string signature;
}

function connectToSalesforce(string consumer_key,
                      string consumer_secret,
                      string user_name,
                      string user_password) {
    HttpRequest request;
    HttpHeader header = httpCreateHeader("Content-Type", "application/json");
    request.headers += header;
    string dummyPayload;
    string url = "https://login.salesforce.com/services/oauth2/token";
    string url_string = "?grant_type=password&client_id=" + consumer_key+ 
                    "&client_secret=" + consumer_secret + 
                    "&username=" + user_name + 
                    "&password=" + user_password; 
    SFDCConnection connection = httpPost(url + url_string, request, dummyPayload);
    runnerLog(httpGetErrorMessage());
    return connection;
}

最初に、接続時にSalesforceから戻るJSONベースのフレームワークを作成しました。そして、https：//login.salesforce.com/services/oauth2/tokenメソッド   を使用してSalesforceに接続しました。

次に、商談データを取得する関数を作成しましょう。これを行うには、XX.X / sobjects / SObjectNam / id / methodを使用します。

struct Opportunity {
    string Id;
    string AccountId;
    string Name;
    string Description;
    string StageName;
    string OwnerId;
}

function getOppFromOppId(string access_token,
                string url,
                string oppId) {
    Opportunity result;
    HttpRequest request;
    HttpHeader header = httpCreateHeader("Content-Type", "application/json");
    request.headers += header;
    header = httpCreateHeader("Authorization", "Bearer " + access_token);
    request.headers += header;
    result = httpGet(url + "/services/data/v39.0/sobjects/Opportunity/" + oppId, request);
    return result;
}

ここでも、商談データの構造を定義してから、Salesforce Rest APIメソッドを呼び出しました。

次に、関数を使用するコードを記述します。

SFDCConnection connection = connectToSalesforce(consumer_key, consumer_secret, user_name, user_password);
Opportunity opp = getOppFromOppId(connection.access_token, connection.instance_url, "0064F00000NKA7CQAX");
runnerLog(opp);
