Debug kontainer PHP menggunakan Xdebug dan PhpStorm

Terjemahan artikel disiapkan sebelum dimulainya kursus "Pengembang PHP Backend" .









Instruksi Docker # 9: xdebug



Saya akan membuat halaman php yang sangat sederhana dan menjalankannya dengan xdebug dan PhpStorm.



File sumber dapat ditemukan di sini:

github.com/ikknd/docker-study di folder resep-09



1. Buat "Dockerfile" di folder "docker": 



FROM php:7.2-fpm

#Install xdebug
RUN pecl install xdebug-2.6.1 && docker-php-ext-enable xdebug

CMD ["php-fpm"]


Jalankan perintah ini dari folder buruh pelabuhan untuk membuat gambar: 



docker build -t php-xdebug-custom -f Dockerfile .


2. Buat file docker-compose.yml di folder docker: 



version: "3.7"

services:

  web:
    image: nginx:1.17
    ports:
      - 80:80
    volumes:
      - /var/www/docker-study.loc/recipe-09/php:/var/www/myapp
      - /var/www/docker-study.loc/recipe-09/docker/site.conf:/etc/nginx/conf.d/site.conf
    depends_on:
      - php

  php:
    image: php-xdebug-custom
    volumes:
      - /var/www/docker-study.loc/recipe-09/php:/var/www/myapp
      - /var/www/docker-study.loc/recipe-09/docker/php.ini:/usr/local/etc/php/php.ini


Di sini saya menggunakan gambar php-xdebug-custom alih-alih php: 7.2-fpm ‚Äû



3. Buat pengaturan berikut dalam file php.ini: 



[xdebug]
zend_extension=xdebug.so
xdebug.profiler_enable=1
xdebug.remote_enable=1
xdebug.remote_handler=dbgp
xdebug.remote_mode=req
xdebug.remote_host=host.docker.internal
xdebug.remote_port=9000
xdebug.remote_autostart=1
xdebug.remote_connect_back=1
xdebug.idekey=PHPSTORM


4. Konfigurasikan server di PhpStorm:



File -> Pengaturan -> Bahasa dan Kerangka Kerja -> PHP -> Server

Tambahkan server baru menggunakan ikon + dan konfigurasikan seperti yang ditunjukkan pada tangkapan layar berikut:







Pastikan Anda memilih "Gunakan pemetaan jalur" ( "Gunakan pemetaan jalan") dan memetakan folder php ke /var/www/myapp



.



5. Konfigurasikan debugger jarak jauh PHP di PhpStorm:



Jalankan -> Edit konfigurasi -> PHP Remote Debug



Tambahkan konfigurasi baru dan tetapkan nilai-nilai padanya, seperti pada tangkapan layar berikut:







6. Pilih konfigurasi debug di panel debug PhpStorm







7. Pergi ke /var/www/docker-study.loc/recipe-09/docker/



dan jalankan: 



docker-compose up -d


Jika saya mengetikkan myapp.loc/



browser sekarang, saya akan melihat hasil dari file index.php



.



Saya dapat mengatur breakpoint, mulai mendengarkan koneksi di panel debug PhpStorm dan memuat ulang halaman.



Selamat debug!
