1.感情分析




ソースコードを使用して、データサイエンスプロジェクトの完全な実装をご覧ください-Rのプロジェクト別のセンチメント分析。

感情分析は、ポジティブまたはネガティブになり得る気分や意見を特定するための単語分析です。これは分類のタイプであり、クラスはバイナリ（ポジティブとネガティブ）または複数（ハッピー、イービル、悲しい、厄介な...）になります。このデータサイエンスプロジェクトをR言語で実装し、janeaustenRパッケージのデータセットを使用します。 AFINN、bing、loughranなどの汎用ディクショナリを使用して内部結合を実行し、最後に結果を表示するワードクラウドを作成します。

2. Fake News Detection ( )


初心者向けのデータサイエンスプロジェクトに取り組み、Pythonを使用して偽のニュースを発見し、スキルを次のレベルに引き上げましょう。



偽のニュースとは、政治的目標を達成するためにソーシャルネットワークやその他のオンラインメディアを通じて広められた虚偽の情報です。このデータサイエンスプロジェクトのアイデアでは、Pythonを使用して、ニュースが本物か偽物かを正確に判断できるモデルを構築します。TfidfVectorizerを作成し、PassiveAggressiveClassifierを使用してニュースを「本物」と「偽物」に分類します。7796×4のフォームデータセットを使用して、Jupyterラボですべてを実行します。

言語： Python データセット

/パッケージ： news.csv

データサイエンスプロジェクトのアイデア-XGBoostによるパーキンソン病の特定に進みましょう。



データサイエンスを使用してヘルスケアとサービスを改善し始めました。病気を早期に予測できれば、多くのメリットがあります。したがって、このデータサイエンスプロジェクトのアイデアでは、Pythonを使用してパーキンソン病を検出する方法を学びます。これは中枢神経系の神経変性進行性疾患で、動きに影響を与え、震えやこわばりを引き起こします。脳のドーパミン産生ニューロンに影響を及ぼし、毎年インドでは100万人以上が影響を受けています。

言語： Python データセット

では、さまざまなライブラリの使用方法を学びましょう。このデータサイエンスプロジェクトでは、音声認識にlibrosaを使用しています。SERは、音声から人間の感情と感情状態を決定するプロセスです。ピッチとピッチを使用して感情を声で表現するため、SERが適切です。しかし、感情は主観的であるため、音に注釈を付けることは困難な作業です。mfcc、chroma、mel関数を使用し、RAVDESSデータセットを使用して感情を認識します。このモデルのMLPC分類子を作成します。

7.ドライバーの眠気検知


トップデータサイエンスプロジェクト（OpenCVとKerasを備えた睡眠検出システム）で作業しながらスキルを高めましょう。



眠い運転は非常に危険であり、運転中に運転者が眠りに落ちることにより、毎年約1000件の事故が発生しています。このPythonプロジェクトでは、眠いドライバーを検出し、サウンドシグナルでドライバーに通知できるシステムを作成します。

このプロジェクトは、KerasとOpenCVを使用して実装されます。OpenCVを使用して顔と目を検出します。Kerasでは、ディープニューラルネットワーク技術を使用して、目の状態（開いているか閉じているか）を分類します。

ソースコード付きの完全なプロジェクト実装-CNNおよびLSTM付きの画像キャプションジェネレーターを確認してください。



画像の内容を説明するのは人にとって簡単な作業ですが、コンピュータの場合、画像は各ピクセルのカラー値を表す一連の数値にすぎません。これはコンピュータにとって難しい作業です。画像の内容を理解し、自然言語（たとえば、英語）で説明を作成することは、もう1つの難しい作業です。このプロジェクトでは、詳細な探査手法を使用して、革命ニューラルネットワーク（CNN）とリカレントニューラルネットワーク（LSTM）を実装し、画像記述ジェネレーターを作成します。データセット

顧客セグメンテーションは、人気のある教師なし学習アプリケーションです。企業はクラスタリングを使用して、潜在的なユーザーベースと連携する顧客セグメントを定義します。彼らは、性別、年齢、興味、支出習慣などの一般的な特性に従ってクライアントをグループに分け、各グループに効果的に製品を販売できるようにします。K-meansクラスタリングを使用し、性別と年齢による分布を視覚化します。次に、彼らの年収と支出のレベルを分析します。

13.乳がんの分類


Pythonのデータサイエンスプロジェクトの完全な実装- ディープラーニングによる乳がんの分類をご覧ください。



データサイエンスの医学的貢献に戻り、Pythonを使用して乳がんを検出する方法を学びましょう。IDC_regularデータセットを使用して、乳癌の最も一般的な形態である浸潤性管癌を検出します。それは乳管で発生し、乳管の外の乳腺の線維性または脂肪組織に浸透します。この科学的データ収集プロジェクトのアイデアでは、分類にディープラーニングとKerasライブラリを使用します。

14.交通標識の認識


CNNオープンソースを使用 したデータサイエンスの標識認識プロジェクトで自動運転技術の精度を達成する。



交通標識や交通ルールは、すべてのドライバーが事故を回避するために非常に重要です。ルールに従うには、最初に交通標識の外観を理解する必要があります。人は、車両を運転する権利を与えられる前に、すべての道路標識を学ばなければなりません。しかし、今では自動運転車の数が増えており、近い将来、人々は独立して機械を制御することができなくなります。 「交通標識認識」プロジェクトでは、プログラムが画像を入力信号として受け入れることにより、交通標識のタイプを認識する方法を学びます。ドイツの道路標識認識チェックリスト（GTSRB）は、道路標識が属するクラスを認識するためのディープニューラルネットワークを構築するために使用されます。また、アプリケーションと対話するためのシンプルなグラフィカルインターフェイスも作成します。

言語： Python データセット

： GTSRB（ドイツの交通標識認識ベンチマーク）
