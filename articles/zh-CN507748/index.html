Node-RED仪表板模板示例（AngularJS） 文章显示了如何在Angular Scope上下文中Template节点与其他交换味精消息的节点进行交互的示例。

假设我们将标记名称作为msg.topic对象的属性传递，并将msg.payload作为值传递。对于每个味精对象更改事件，我们使用storeValue和updateValues方法更新SCADA对象 



 (function(scope) {
        scope.$watch('msg', function(msg) {
            if (msg) 
                office.storeValue(msg.topic, msg.payload);
                office.updateValues();
            }
        });
        })(scope);


同样使用资源https://scadavis.io/learn.html中的示例以及Template节点的功能，您可以处理SCADA对象的各种事件并将其传递给其他节点，例如： 
(function(scope) {
substationsynoptic.on("click", function (event, tag) {
    var v = substationsynoptic.getValue(tag);
    if (event.currentTarget.id === "TAPUP")
        substationsynoptic.setValue(tag, v + 1, false, false);
    else
        if (event.currentTarget.id === "TAPDOWN")
            substationsynoptic.setValue(tag, v - 1, false, false);
    if (event.currentTarget.id === "XCBROPEN")
        substationsynoptic.setValue(tag, false, false, false);
    else
        if (event.currentTarget.id === "XCBRCLOSE")
            substationsynoptic.setValue(tag, true, false, false);
    if (v === true)
        substationsynoptic.setValue(tag, false, false, false);
    else
        if (v === false)
            substationsynoptic.setValue(tag, true, false, false);
    scope.send( {'payload':{ 'tag': tag, 'value': v }} );
});
})(scope);




图形编辑器



关于用于为SCADA系统构建可视页面的图形编辑器的几句话。如上所述，建议在场外购买SCADAvis.io摘要编辑器。再一次，这不是唯一的选择。从描述本身可以知道，该编辑器基于Inkscape SVG编辑器。实际上，这是带插件的Inkscape，它允许您将标签（变量）和其他属性分配给svg文件的各种图形组件。那些。为了在scadavis的上下文中轻松显示页面，拥有一个svg文件就足够了，可以在任何方便的编辑器中对其进行编辑。但这还不是全部。 ECAVA INTEGRAXOR的编辑器之间有很多相似之处，网址为：www.integraxor.com。Inkscape 0.91 SAGE 4.16免费分发（您只需要在网站上注册）和Synoptic Editor。事实证明，实际上可以使用Inkscape 0.91 SAGE 4.16为Scadavis创建和编辑svg文件。设计脚本动画和SAGE编辑器的“对象”属性中缺少的其他一些特定内容时，会观察到这种差异。







根据SCADAvis.io摘要编辑器的描述，为了根据包含条件分配对象的颜色，必须在“限制”列中指定数字2：



  • 0-未初始化状态
  • 1-假（关闭）状态
  • 2-真（开启）状态
  • 129-错误（关闭）状态加上失败值
  • 130-真（开启）状态加上失败值
