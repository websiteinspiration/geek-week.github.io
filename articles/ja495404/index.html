株価情報を取得する


まず、過去2年間の株価を取得する必要があります。私はそれらをFinamのWebサイトからダウンロードし、csvをインポートしました。

ファイル内の見積もりの​​収集を始めましょう


パンダをインポート：

import pandas as pd

ウィンドウの表示に関する制限を解除しましょう（PyCharmでこれを行う必要があります）：


pd.set_option('display.max_columns', None)
pd.set_option('display.expand_frame_repr', False)
pd.set_option('max_colwidth', 80)
pd.set_option('max_rows', 60000)

同様の方法で各株をインポートする必要があります。


print('GAZP')
GAZP = pd.read_csv("GAZP_180101_191231.csv",sep=';', header=0, index_col='<DATE>', parse_dates=True)
GAZP = GAZP.sort_values(by='<DATE>')
#      
GAZP['30_days_growth']=((GAZP['<CLOSE>']/GAZP['<CLOSE>'].shift(30))-1)*100
GAZP['after_30_days_growth']=((GAZP['<CLOSE>'].shift(-30)/GAZP['<CLOSE>'])-1)*100
GAZP['ticket']='GAZP'

  1. 銘柄名を表示
  2. ファイルから読み取ります
  3. 日付けで並び替え
  4. 毎日30％の株の増加を示す列を追加します。
  5. 今後30日間の変化率の列を追加します。
  6. チケットを含む列を追加します。これにより、列データグラムを日付と組み合わせることができます。

ドローダウンが15％を超えるすべての株式の利益の計算


次に、ドローダウンが15％を超えるすべての株式について、30日間の平均利回りを計算します。

すべてを1つのデータフレームに結合する必要があります。

これを行うには、すべての共有を含むシートを作成します。


all_tickets = [GAZP,LKOH,SBER,SBERP,NVTK,MOEX,MGNT,MFON,MTSS,MAGN,GMKN,AFLT,POLY,ROSN,HYDR,RTKM,RUAL,YNDX,ALRS,VTBR,TATN,TATNP,SNGS,SNGSP,PLZL,NLMK,CHMF,IRAO,TRNFP,PHOR]

組み合わせる：


all_stocks = pd.concat(all_tickets)

ドローダウンが15％を超えるラインのみが含まれるデータフレームを作成します。


tickets15 = all_stocks[all_stocks['30_days_growth']<-15]
print(tickets15)

平均利回りを計算します。


print(tickets15['after_30_days_growth'].mean())



このレベルの多様化を考慮すると、 7.78570670526497％とかなり良い結果が得られます。

同様に、10％ドローダウンの利回りを計算します



tickets10 = all_stocks[all_stocks['30_days_growth']<-10]
print(tickets10)
print(tickets10['after_30_days_growth'].mean())

3.1％

30日間で15％以上成長したシェアを下げることでプレーする価値はありますか？



tickets15plus = all_stocks[all_stocks['30_days_growth']>15]
print(tickets15plus)
print(tickets15plus['after_30_days_growth'].mean())

成長率は2％です。この方法では利益を得ることができないことがわかりますが、この場合、この場合の可能性が低いことは明らかです。しかし、これは驚くべきことではありません。

もちろん数値は興味深いですが、必要な範囲内にある間、すべてのドローダウンを毎日購入する場合にこの収益性が得られるので、これがベンチマークであることを考慮する価値があります。

同じドローダウンの重複がないように、近い日付のラインを削除する必要がありますが、これは別の話です。

すべての逸脱の理由も考慮する必要があります。明らかに、ガスプロムが新しい部門方針を導入するならば、近い将来の秋を待つ価値はありません。
