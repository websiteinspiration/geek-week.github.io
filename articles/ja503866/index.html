2020年のロードマップ開発Zextras Suiteを公開

以前、2020年の初めにZextras Suite ProとTeam Proの革新について話しました。Zextrasは、2020年の新しいZextras Suite機能の開発のためのロードマップを公開しました  。

8月-チーム：ビデオサーバー（新機能）

Zextrasチームの組み込みのビデオサーバーの外観により、管理者コンソールから直接制御できる多数の新機能を導入でき、サービスの生産性が向上します。オーディオストリームとビデオストリームは、デバイスごとに個別のストリームを取得するために、圧縮、最適化、および結合されます。さらに、最高の通信品質を確保するために、サーバーは各デバイスに適切なコーデックとビットレートを自動的に選択します。

8月-外部エンドポイントでのバックアップ（新規）

Zextras Backupは、システムをローカル、ハイブリッド、クラウドS3互換ストレージ、またはNFSなどのBLOB要素の外部ストレージにバックアップする機会を提供します。同時に、ローカルメタデータキャッシュを使用すると、災害復旧が必要になった場合にそれら自体も外部エンドポイントに格納されるという事実にもかかわらず、元のパフォーマンスを維持できます。

9月-Zextras認証およびログイン（

新規） Zimbra Collaboration Suiteオープンソースエディションには、2要素認証、統合SAML認証および承認システム、および複数のモバイルパスワードが追加されます。カスタマイズ可能な更新されたログインページも表示されます。

9月-チームのプライバシー設定（新規）

ユーザーと管理者は、自分の存在に関する情報を表示するかどうかを選択できます。特に、メッセージの読み取り、ユーザーのステータスなどの情報を非表示にすることができます。9月-モバイルアプリのQR認証（新機能） QRログインシステムでは、相互認証システムを使用してパスワードを入力しなくてもモバイルアプリケーションを使用でき、登録と承認を管理できます。9月-MS Outlook / MAC Mail / Thunderbird用のZextrasアドレス帳（新規） Microsoft Outlook、Thunderbird、Mac Mailを集中管理されたZimbra GALとアドレス帳に接続して、ユーザーの利便性を高めます。










アドレス帳は、IMAP / POPまたはEASと連携して機能します。 Microsoft Outlookは、EASとアドレス帳を使用して、クライアントにプラグインなしでOutlookと完全に対話できます。

10月-Zextrasconfigインフラストラクチャマネージャー（新機能）

インフラストラクチャ内のすべてのZextrasコンポーネントは、特別なZextrasconfigマネージャーを介して管理できます。このマネージャーを使用すると、Zimbra管理者は、更新された統合Zextras Docsサーバーや統合Zextras Teamビデオサーバーなど、さまざまなインフラストラクチャノード上のZextrasコンポーネントを管理できます。

10月-管理インターフェースとユーザー設定（REFACTORING）

新しいZextrasコンポーネント管理インターフェースはReactで作成されています。使い慣れた管理インターフェースで引き続き表示されるZextrasコンポーネントから始めて、使い慣れたインターフェースからそれらを削除することなく、すべてのオプションを新しいテクノロジーに転送します。将来的には完全な移行が行われます。

10月-ネイティブLinuxパッケージングシステムを使用した

新しいインストーラー（新規）新しいZextrasパッケージは、ネイティブLinuxをサポートするリポジトリを通じて配布されます。パッケージをインストールおよび更新するとき、「apt-get install / update」のようなシステムコマンドを使用することが可能になります。

11月-ドキュメントサーバー（新規）

Zextras Docsサーバーは、Zextrasconfig Centralized Infrastructure Management Managerによって管理されるインフラストラクチャの一部になります。個々のノードにサーバーをインストールして更新する必要はなくなります。

11月-チーム：会議の新機能（新機能）

新しいビデオサーバーのおかげで、Zextrasチームには多くの新機能が登場します。その中で：

  • モデレーターモード
  • 特定のユーザーまたはすべてのユーザーに対してサウンドをミュートする機能
  • ウェビナーモードとビデオ会議モードを有効にする機能
  • イベントログ
  • チャットリクエスト
  • トランシーバーモード
  • ハングアウトの録画をZextras Driveにアップロードできます

11月-高可用性（新機能）

インフラストラクチャの異なるノード間でユーザー、グループ、ドメイン、さらにはメールサーバー全体を同期する機能。クラスター内のスペアノードまたは別のアクティブノードとの同期を構成する機能。システムは、1か月あたり99.99％の可用性を実現するように設計されています。

12月-メールの署名と暗号化（S / Mime-PGP）（新規）

エンドツーエンドの暗号化がWebクライアントに実装されます。システムは、送信された電子メールを暗号化および/または署名して、受信した電子メールの署名を復号化および検証できます。

これらの機能の多くは、これまでのところZimbraの商用バージョンに限定されています。今年すでに、Zimbra Collaboration Suite Open-Source Editionのユーザーは、Zextras Suiteを購入すると、これらの機能やその他の機能を利用できるようになります。将来的には、Zextras Suiteの新機能について詳しく説明します。
