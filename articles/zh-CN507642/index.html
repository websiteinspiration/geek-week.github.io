纯功能 -确定性且无副作用的功能。



也就是说，对于纯函数而言，它必须

是确定性的-也就是说，每次带有相同参数集的每次生成相同结果。

确定的功能示例 

def sum(a,b):
  return a+b

print(sum(3,6))
print(sum(3,6))
print(sum(3,6))

>>>9
>>>9
>>>9


这个例子不是确定性的 



from datetime import date

todays_weekday = date.today().weekday()
def sum(a,b):
  if todays_weekday == 1:
    result = a+b
    print(f'sum is {result} and result class is {type(result)}')
  else:
    result = str(a+b)
    print(f'sum is {result} and result class is {type(result)}')

sum(4,5)
todays_weekday = 4
sum(4,5)

>>>sum is 9 and result class is <class 'int'>
>>>sum is 9 and result class is <class 'str'>


每次更改星期几（不是函数的参数）时，函数都会产生不同的结果。



不确定性函数的最明显示例是随机的 



import random
print(random.random())
print(random.random())
print(random.random())

>>>0.5002395668892657
>>>0.8837128676460416
>>>0.5308851462814731




纯功能的第二个重要品质是缺乏副作用

将清楚显示 

my_list = [32,3,50,2,29,43]

def sort_by_sort(li):
  li.sort()
  print(li)

sort_by_sort(my_list)
print(my_list)

>>>[2, 3, 29, 32, 43, 50]
>>>[2, 3, 29, 32, 43, 50]


sort_by_sort函数具有副作用，因为它会更改元素的原始列表并在控制台中显示某些内容。 



my_list = [32,3,50,2,29,43]

def sort_by_sorted(li):
  return sorted(li)

print(sort_by_sorted(my_list))
print(my_list)

>>>[2, 3, 29, 32, 43, 50]
>>>[32, 3, 50, 2, 29, 43]


与前面的示例不同，sort_by_sorted函数不会更改原始数组并返回结果而不将其单独输出到控制台。

高阶函数 -在编程中，该函数将其他函数作为参数或返回另一个函数作为结果。





FVP示例 



def func(num):
  return num**2

def higher_order_func(fun, num):
  return fun(num)+fun(num)

print(func(4))
print(higher_order_func(func,4))

>>>16
>>>32

首先，您需要了解以下内容-什么是函数式编程。我个人知道日常编程中最常提到的两个范例-OOP和FP。



如果将其完全简化并用手指进行解释，则可以按以下方式描述这两种范例：



  • OOP是面向对象的编程-一种编程方法，当使用哪种对象可以作为参数传递并用作值时。
  • 使用此逻辑，可以确定FP是一种编程方法，当使用FP可以将哪些函数作为参数传递给其他函数并用作其他函数返回的值时...答案隐藏在名称本身中。




就像我最喜欢的老师说的 ZverokVictor Shepelev：“编程中的所有工作都是处理数据。“他拿了一些数据，与他们一起玩然后将其返回。”



这也适用于FP-获取一些数据，执行一些功能，与它们一起玩并给出一些内容。



我不会画所有东西，否则会很长。本文的目的是帮助您理解而不是解释其工作方式和作用，因此在这里我们来看一下FP的主要功能。



在大多数情况下，FP（据我所知）只是简化的代码编写。以功能风格编写的任何代码都可以很容易地以常规风格重写，而不会降低质量，但是更原始。FP的目标是编写更简单，更易理解且易于维护的代码，并且占用更少的内存，而如果没有它，那么编程的主要永恒道德就是DRY（不要重复自己-不要重复） 。



现在，我们将分析FP中使用的主要功能之一-Lambda函数。



在以下文章中，我们将分析诸如Map，Zip，Filter和Reduce的功能。



Lambda函数



Lambda是python和其他编程语言中用于调用匿名函数的工具。这很可能不会说什么，也不会以任何方式阐明它的工作方式，因此，我仅向您介绍lambda表达式的机制。



一切都非常简单。



考虑一个例子。例如，我们需要编写一个函数来考虑半径已知的圆的面积。



圆面积公式



S = pi*(r**2)







，其中

S是圆的面积

PI -数学常数等于3.14，我们从标准库数学得到

的R -圆的半径-唯一的变量，我们将通过我们的功能







现在让我们所有格式蟒蛇 



import math #  math

pi_const = round(math.pi, 2) # pi           3.141592653589793     

#              
def area_of_circle_simple(radius):
  return pi_const*(radius**2)

print(area_of_circle_simple(5))
print(area_of_circle_simple(12))
print(area_of_circle_simple(26))
>>>78.5
>>>452.16
>>>2122.64




看起来不错，但是如果您通过lambda录制的话，看起来会更酷 



import math #  math

pi_const = round(math.pi, 2) # pi    
#       
#  3.141592653589793     

print((lambda radius: pi_const*(radius**2))(5))
print((lambda radius: pi_const*(radius**2))(12))
print((lambda radius: pi_const*(radius**2))(26))

>>>78.5
>>>452.16
>>>2122.64


为了更加清楚，匿名函数调用意味着您可以在不声明任何地方的情况下使用它，如上例所示。



Lambda函数根据以下原理工作

考虑一个带有两个输入参数的示例。例如，我们需要使用以下公式来计算圆锥体的体积： 



V = (height*pi_const*(radius**2))/3






让我们全部用python编写： 



import math #  math

pi_const = round(math.pi, 2) # pi           3.141592653589793     

#      
def cone_volume(height, radius):
  volume = (height*pi_const*(radius**2))/3
  return volume

print(cone_volume(3, 10))

>>>314.0


现在，它以lambda形式显示： 



import math #  math

pi_const = round(math.pi, 2) # pi           3.141592653589793     

print((lambda height, radius : (height*pi_const*(radius**2))/3)(3, 10))

>>>314.0




这里的变量数量是无限的。例如，我们计算一个截锥的体积，其中考虑了3个不同的变量。



截锥的体积由以下公式计算： 



V = (pi_const*height*(r1**2 + r1*r2 + r2**2))/3






这是它在python中的经典外观 



import math #  math

pi_const = round(math.pi, 2) # pi           3.141592653589793     

#      
def cone_volume(h,r1,r2):
  return (pi_const * h * (r1 ** 2 + r1 * r2 + r2 ** 2))/3

print(cone_volume(12, 8, 5))
print(cone_volume(15, 10, 6))
print(cone_volume(20, 12, 9))

>>>1620.24
>>>3077.20
>>>6970.8




现在让我们展示一下lambda的外观： 



import math #  math

pi_const = round(math.pi, 2) # pi           3.141592653589793     

print((lambda height, radius1, radius2 : (height*pi_const*(radius1**2 + radius1*radius2 + radius2**2))/3)(12, 8, 5))
print((lambda height, radius1, radius2 : (height*pi_const*(radius1**2 + radius1*radius2 + radius2**2))/3)(15, 10, 6))
print((lambda height, radius1, radius2 : (height*pi_const*(radius1**2 + radius1*radius2 + radius2**2))/3)(20, 12, 9))

>>>1620.24
>>>3077.20
>>>6970.8
