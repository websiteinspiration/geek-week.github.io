最新のJavaScriptルーターの作成

こんにちは！

React、Vue、または純粋なJavaScriptに基づくシンプルな1ページのアプリが、私たちのいたるところを囲んでいます。適切な「1ページ」は、適切なルーティングメカニズムを前提としています。

「navigo」や「react-router」などのライブラリは非常に便利です。しかし、それらはどのように機能しますか？ライブラリ全体をインポートする必要がありますか？それとも、たとえば10％で十分でしょうか？実際、高速で便利なルーターを自分で簡単に作成でき、少し時間がかかり、プログラムは100行未満のコードで構成されます。

必要条件


私たちのルーターは：

  • ES6 +で記述
  • 履歴とハッシュに対応
  • 再利用可能なライブラリ

通常、ウェブアプリケーションはルーターの1つのインスタンスを使用しますが、多くの場合、複数のインスタンスが必要であるため、シングルトンをテンプレートとして使用することはできません。ルーターが機能するには、次のプロパティが必要です。

  • routers：登録済みルーターのリスト
  • モード：ハッシュまたは履歴
  • ルート要素：アプリケーションのルート要素（履歴使用モードの場合）
  • コンストラクタ：新しいルーターインスタンスを作成するためのメイン関数

class Router {
    routes = []
    mode = null
    root = '/'

    constructor(options) {
        this.mode = window.history.pushState ? 'history' : 'hash'
        if (options.mode) this.mode = options.mode
        if (options.root) this.root = options.root
    }
}

export default Router

ルーターの追加と削除


ルーターを追加および削除するには、配列要素を追加および削除します。

class Router {
    routes = []
    mode = null
    root = '/'

    constructor(options) {
        this.mode = window.history.pushState ? 'history' : 'hash'
        if (options.mode) this.mode = options.mode
        if (options.root) this.root = options.root
    }

    add = (path, cb) => {
        this.routes.push({
            path,
            cb
        })
        return this
    }

    remove = path => {
        for (let i = 0; i < this.routes.length; i += 1) {
            if (this.routes[i].path === path) {
                this.routes.slice(i, 1)
                return this
            }
        }
        return this
    }

    flush = () => {
        this.routes = []
        return this
    }
}

export default Router

現在のパスを取得する


特定の時点でアプリケーションのどこにいるかを知る必要があります。

このため、両方のモード（履歴とハッシュ）を処理する必要があります。最初のケースでは、2番目の「＃」で、window.locationからルート要素へのパスを削除する必要があります。すべてのルーターを削除する（clearSlash）関数も必要です（最初から最後までの行）。

    clearSlashes = path =>
        path
        .toString()
        .replace(/\/$/, '')
        .replace(/^\//, '')

    getFragment = () => {
        let fragment = ''

        if (this.mode === 'history') {
            fragment = this.clearSlashes(decodeURI(window.location.pathname + window.location.search))
            fragment = fragment.replace(/\?(.*)$/, '')
            fragment = this.root !== '/' ? fragment.replace(this.root, '') : fragment
        } else {
            const match = window.location.href.match(/#(.*)$/)
            fragment = match ? match[1] : ''
        }
        return this.clearSlashes(fragment)
    }
}

export default Router

ナビゲーション


OK、URLを追加および削除するためのAPIがあります。現在の住所を取得する機会もあります。次のステップは、ルーターのナビゲートです。「mode」プロパティを使用します。

[...]

    getFragment = () => {
        let fragment = ''

        if (this.mode === 'history') {
            fragment = this.clearSlashes(decodeURI(window.location.pathname + window.location.search))
            fragment = fragment.replace(/\?(.*)$/, '')
            fragment = this.root !== '/' ? fragment.replace(this.root, '') : fragment
        } else {
            const match = window.location.href.match(/#(.*)$/)
            fragment = match ? match[1] : ''
        }
        return this.clearSlashes(fragment)
    }

    navigate = (path = '') => {
        if (this.mode === 'history') {
            window.history.pushState(null, null, this.root + this.clearSlashes(path))
        } else {
            window.location.href = `${window.location.href.replace(/#(.*)$/, '')}#${path}`
        }
        return this
    }
}

export default Router

変化を見守る


次に、リンクを使用するか、作成した「ナビゲート」メソッドを使用して、住所の変更を追跡するロジックが必要です。また、最初のアクセス時に正しいページが表示されるようにする必要もあります。アプリケーションの状態を使用して変更を登録することもできますが、調査するために、これをsetIntervalで行います。

class Router {
    routes = [];
    mode = null;
    root = "/";

    constructor(options) {
        this.mode = window.history.pushState ? "history" : "hash";
        if (options.mode) this.mode = options.mode;
        if (options.root) this.root = options.root;

        this.listen();
    }

    [...]

    listen = () => {
        clearInterval(this.interval)
        this.interval = setInterval(this.interval, 50)
    }

    interval = () => {
        if (this.current === this.getFragment()) return
        this.current = this.getFragment()

        this.routes.some(route => {
            const match = this.current.match(route.path)

            if (match) {
                match.shift()
                route.cb.apply({}, match)
                return match
            }
            return false
        })
    }
}

export default Router
