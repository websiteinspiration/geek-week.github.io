SD Express: padrão SD 8.0 e PCIe 4.0. O que é isso e por quê?

Em 19 de maio de 2020, a SD Association anunciou o padrão SD 8.0 para cartões SD Express. Como sempre, tudo o que há de novo é melhor e mais rápido com o PCIe 4.0 e suporta compatibilidade com versões anteriores. Vamos ver o que nos espera nesta forma bastante popular de cartões de memória no próximo (ou, como de costume, não tão cedo).


Pode haver uma descrição longa de onde os cartões SD são encontrados, mas você provavelmente já sabe disso.

01. Breve informação


Então, o que você precisa saber sobre o SD Express no momento:


  • Todo o SD Express será compatível com versões anteriores;
  • O SD Express teoricamente permite uma largura de banda de 3.938 MB / s (2 pistas PCIe 4.0, atualmente apenas SD em tamanho real com 27 pinos);
  • O microSD com PCIe 4.0 ainda não é suportado;
  • O SD Express conectado via PCIe será reconhecido pelo host como dispositivos NVMe padrão.
