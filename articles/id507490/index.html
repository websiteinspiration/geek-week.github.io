Bekerja dengan kode: tiga alat terbuka

Mari kita bicara tentang beberapa editor teks dan utilitas untuk mendesain kode dalam dokumentasi.



Alat Pengembang Lainnya: API dan Elemen UI





Foto - ThisEngineering RAEng - Unsplash





Mikro



Ini adalah editor teks terminal, dipertajam oleh sistem modern. Kode sumber ada di GitHub dan didistribusikan di bawah lisensi MIT . Para pengembang mengatakan bahwa produk tersebut adalah penerus ideologis dari editor Nano untuk sistem operasi mirip Unix.



Di antara fungsi mikro yang menarik, kami dapat memilih dukungan untuk mouse (seret, klik dua kali, klik tiga kali) dan beberapa kursor, penautan otomatis kode (pengecekan kesalahan) dan penyorotan sintaks untuk lebih dari 90 bahasa . Dalam utas tematik tentang Hacker News, penduduk mencatat bahwa utilitas ini mirip dengan editor Teks Sublime yang eksklusifdan memiliki banyak fungsi: pengeditan multi-baris, manajemen plugin - bahkan hotkey serupa.



Alat ini muncul sekitar tiga tahun lalu, sehingga bug masih ditemui dalam pekerjaannya . Meskipun salah satu warga HN mencatat bahwa stabilitas mikro baru-baru ini telah tumbuh. Bagaimanapun, alat ini memiliki komunitas (lebih dari 14 ribu bintang di GitHub), dan ada komunikasi aktif dalam obrolan Gitter .





Atom



Ini adalah editor teks lintas-platform dan alat pengembangan yang disediakan oleh GitHub. Itu dibangun di atas HTML, JavaScript, CSS, Node.js dan menggunakan kerangka kerja Electron .



Ini menawarkan pengaturan luas untuk antarmuka dan teks - Anda dapat mengubah gaya, ikatan kunci, bahkan tampilan editor, jika Anda tahu JavaScript. Hampir semua pengaturan dalam file CSON (format JSON, tetapi untuk objek CoffeeScript). Karena GitHub mengembangkan Atom, ia memiliki ekstensi yang memungkinkan Anda untuk mengakses sistem kontrol versi langsung dari editor teks - kelola cabang, jalankan komit , tekan dan tarik perintah . Ada juga paket Teletype- Menggunakannya, kode dapat ditulis oleh beberapa pengembang secara bersamaan.





Tangkapan layar: atom.io / Bekerja dengan Git di Atom



Atom adalah alat yang relatif muda, sehingga ada masalah kinerja, namun, penulis terlibat dalam optimasi .





Karbon



Alat ini memungkinkan Anda untuk membuat tangkapan layar kode yang indah untuk presentasi, artikel atau dokumentasi. Cukup dengan menyalin sebagian program di lapangan di situs dan menyimpan gambar (jika mau, Anda dapat memposting gambar langsung ke Twitter). Insinyur Brian Dennis dari FOSSA , sistem manajemen aplikasi sumber terbuka untuk seluruh perusahaan, terlibat dalam pengembangan Carbon .


Dengan menggunakan Carbon, Anda dapat memilih warna latar belakang dan menyesuaikan penyorotan sintaksis - didukung lebih dari 30 bahasa dan kerangka kerja: dari C dan Python ke Vue dan R. Berikut adalah contoh gambar yang dapat dihasilkan di situs (kode tersebut adalah "rintisan" yang ditawarkan oleh Carbon):







Penulis proyek berencana untuk menambah jumlah bahasa yang tersedia, merilis versi untuk perangkat seluler dan menambahkan lebih banyak pengaturan. Kode sumber untuk Carbon dapat ditemukan di repositori di GitHub .



Sebuah komunitas yang cukup besar telah terbentuk di situs di sekitar instrumen - lebih dari 24 ribu bintang . Anggotanya mengembangkan ekstensi khusus. Misalnya, ada plugin untuk mengintegrasikan Karbon dengan editor teks Emacs , Vim dan Atom yang telah disebutkan.
