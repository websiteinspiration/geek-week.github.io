Cara pindah dari penyedia host mana pun dengan cPanel ke Plesk di Rusonix hanya dalam lima langkah

Kebetulan orang mengubah penyedia hosting mereka. Ini dilakukan karena berbagai alasan: ada yang tidak senang dengan kecepatan atau kualitas layanan, yang lain menemukan kualitas yang sama, tetapi dengan harga yang lebih rendah. Yang lain lagi tidak suka panel kontrol ...

Jika Anda berada dalam situasi ini, sekarang saatnya untuk mentransfer situs dan proyek Anda ke Rusonix .

Namun, memindahkan situs Anda bisa tampak seperti tugas yang menakutkan. Dalam panduan ini, kami akan menjelaskan prosedur sederhana dengan petunjuk langkah demi langkah sehingga Anda dapat dengan percaya diri dan mudah mentransfer semua data Anda ke server Rusonix .

Mentransfer situs Anda dapat dilakukan hanya dalam lima langkah:

Langkah 1. Mengunggah file situs dari mantan penyedia host

Menggunakan manajer FTP atau manajer file penyedia hosting Anda saat ini, salin file situs ke komputer Anda. Pastikan untuk membuat cadangan pada media independen.

Langkah 2. Mengekspor basis data situs web

Anda Karena Anda menggunakan cPanel, mengekspor basis data tidak akan menjadi tugas yang sulit. Untuk melakukan ini, gunakan phpMyAdmin, yang merupakan bagian integral dari setiap cPanel. Jangan khawatir jika Anda melihat nama file dan database yang aneh: bahkan jika Anda tidak mengerti segalanya, Anda masih bisa melakukannya sendiri.

  • Temukan tab tempat database bersembunyi dan buka.
  • Klik ikon phpMyAdmin untuk membuka antarmuka.
  • Temukan dan klik tombol "Ekspor".
  • Pilih "metode cepat - hanya tampilkan parameter minimum."

Semuanya berjalan lancar? Basis data Anda sekarang siap untuk ditransfer ke server Rusonix. Lanjutkan ke langkah berikutnya.

Langkah 3. Unduh file situs

Buka panel Plesk di Rusonix dan buka bagian "Situs Web dan Domain". Klik tombol "Unduh" dan pilih arsip yang disimpan dengan situs Anda. Harap dicatat bahwa panel Plesk hanya mendukung arsip dalam format zip.

Pilih arsip yang disimpan di komputer dan klik tombol "Ekstrak file".

Langkah 4. Mengimpor basis data

Seperti halnya ekspor, Anda akan menggunakan MySQL untuk mengimpor basis data. Sebenarnya tidak terlalu sulit. Anda juga perlu memasukkan phpMyAdmin, cari tombol "Impor" dan klik. Kemudian cukup pilih file sql yang disimpan di komputer Anda dan impor.

Langkah 5. Ubah DNS

Ini adalah detail kecil yang mudah untuk dilewatkan. Setelah Anda mentransfer situs Anda, Anda harus mengalihkan data DNS domain dari server sebelumnya ke server Rusonix. Jika ini tidak dilakukan, pengguna akan diarahkan ke halaman dengan kesalahan. DNS baru mudah dipelajari untuk mendukung Rusonix.

Setelah Anda mengirim permintaan untuk memindahkan catatan DNS, mungkin diperlukan beberapa jam, dan dalam beberapa kasus, sepanjang hari, untuk mengubahnya.

Jika, karena alasan tertentu, Anda tidak akan berhasil, itu. Dukungan Rusonix secara mandiri akan mentransfer semua data Anda ke server .

Apakah Anda tidak puas dengan penyedia hosting Anda saat ini? Jangan buang waktu Anda - pilih Rusonix! Di sini Anda tidak hanya akan menemukan infrastruktur yang kuat dan cepat, tetapi Anda juga akan terkejut dengan profesionalisme dan suasana dukungan teknis yang ramah dan hangat.
