プログラミングやはんだ付けなしで、新年のカラフルなアドレスLED

そろそろ正月が近づいてきますが、いろんな理由で元気がお正月ではないこともあり、お正月の灯りが役に立ちます。アドレス指定可能なLEDガーランドとリボンでの私の経験についてお話ししたいと思います。私はアメリカを発見することはほとんどありませんが、美しく安価になりました。



私は2つのオプションを検討します-個別に既存のホームオートメーションなしで、「スマートホーム」と連携して動作します。


アドレス指定可能なLEDテープにはかなり多くの構造化情報が表示されているため、情報に基づいた選択を簡単に行うことができます。たとえば、購入する前に、DrZzs（英語）チャンネルとThe Hook Up（英語）チャンネルから対応するビデオを注意深く見て、必要なテープのコストについて結論を出しました。


彼は中国からの2つの安価なオプションを選びました：


  1. WS2812Bテープ-300個のLED -5メートル。
  2. WS2811ガーランド-50個のLED x 3個-約10メートル（3 x 3.3 m）

      seconds = seconds % (24 * 3600);
      int hours = seconds / 3600;
      seconds = seconds % 3600;
      int minutes = seconds /  60;
      seconds = seconds % 60;
      return { (String(days) +"d " + String(hours) +"h " + String(minutes) +"m "+ String(seconds) +"s").c_str() };

    update_interval: 113s

      if (id(wifis_signal).state > -50 ) {

        return {"Weak"};
      } else {
        return {"None"};
      }

    name: ${upper_devicename} MAC Address
    lambda: 'return {WiFi.macAddress().c_str()};'
    icon: mdi:fingerprint
    update_interval: 1d

switch:
  - platform: restart
    name: ${upper_devicename} Restart
