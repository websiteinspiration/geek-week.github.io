Seleniumを使用してcian.ruから写真を解析する

こんにちは親愛なるKhabrovitesです。この小さな例では、JavaScriptウィジェットを使用してデータがロードされるページを解析する方法を紹介します。さらに、この例のページは簡単に保存できますが、これらのウィジェットのため、ページから必要なすべての写真を解析することはできません。この場合、サイトcian.ruを例として使用します。これには独自のapiがありますが、これは使用しませんが、代わりにSeleniumを使用します。私はcian.ruで働いていません。このサイトを例として使用しています。パーサーのコードはシンプルで、初心者向けに設計されています。


簡単な紹介-暇なときにcian.ruでの修理の例を見て、好きな写真を保存しておくといいと思いましたが、手動で保存するのは時間がかかるので、これは私たちの方法ではないので、これを書くことにしましたパーサー。


パーサーは、私がこれらのリンクとは別にインストールしたAnaconda、Selenium、およびchromedriverバイナリのディストリビューションからpython3で書かれています。（そしてもちろん、Google Chromeブラウザがシステムにインストールされている必要があります）


以下は完全なパーサーコードです。次に、要点を個別に分析します。


from selenium import webdriver
from selenium.webdriver.chrome.options import Options
from selenium.common.exceptions import NoSuchElementException
import chromedriver_binary 
import urllib
import time

print('start...')

site = "https://www.cian.ru/sale/flat/222059642/"

chrome_options = Options()
chrome_options.add_argument("--headless")

driver = webdriver.Chrome(options=chrome_options)
#driver = webdriver.Chrome()
driver.get(site)

i = 0

while True:   
    try:
        url = driver.find_element_by_xpath("//div[contains(@class, 'fotorama__active')]/img").get_attribute('src')
    except NoSuchElementException:
        break        

    i += 1
    print(i, url)

    driver.find_element_by_xpath("//div[@class='fotorama__arr fotorama__arr--next']").click()

    name = url.split('/')[-1]
    urllib.request.urlretrieve(url, name)

    time.sleep(2)

print('done.')
