Projets JavaScript pour l'apprentissage et le développement



Bonjour mes amis!

Je vous présente une sélection de 22 projets en JavaScript pur ou, comme on dit, vanille (personnellement, je préfère juste JavaScript).

Pour moi, c'est une sorte de sous-total dans l'apprentissage de JS.

Le code est écrit à différents moments, il a donc un aspect différent et un niveau de complexité différent.

La plupart des projets ne peuvent pas être appelés des applications à part entière, mais chacun d'eux réussit à résoudre la tâche.

Le code du projet est ici .

Le format sera le suivant: nom du projet, sa brève description, bac à sable.

Il existe des articles séparés pour certains projets. Selon d'autres, ils pourraient un jour apparaître.

Alors allons-y.

1. Progression audio et visualisation


Une application pour lire et visualiser des données audio à l'aide de l'API Web Audio et de l'API Canvas.


2. Évasion


Le jeu "Breakout".


3. Calculatrice


Calculatrice. Bibliothèque utilisée Math.js. V. 2.0.


4. Convertisseur de devises


Convertisseur de devises. Une API spéciale est utilisée.


5. Validateur de formulaire


Validation du formulaire.


6. Pendu


Le jeu "Pendu".


7. Galerie d'images et Slider Maker


Générateur de galerie d'images flexible avec curseur intégré.


8. Recherche de paroles


Recherche de paroles (paroles). Une API spéciale est utilisée.


9. Recherche de repas


Recherchez des plats avec une méthode de cuisson et des ingrédients. Une API spéciale est utilisée.


10. App de méditation


La demande de méditation. Travaillez avec l'audio et la vidéo.


11. Cartes mémoire


Cartes pour se souvenir. Curseur interactif.


12. Notes App


Demande de notes (tâches) avec la possibilité de définir la date du rappel. Pour enregistrer les données côté client, IndexedDB est utilisé.


13. Number Guess


Devinez le jeu de nombres. Utilisé par l'API Web Speech.


14. Ping Pong


Le jeu "Ping Pong".


15. Ciseaux de papier de roche


Le jeu "Pierre, ciseaux, papier."


16. Typage rapide


Simulateur de saisie de vitesse.


17. Tetris


Le jeu "Tetris".


18. Lecteur de texte


Demande de lecture de texte. Utilisé par l'API Web Speech. Comparez les voix de Microsoft et de Google.


19. Tic Tac Toe


Le jeu "Tic Tac Toe." Un algorithme spécial est utilisé. Essayez de battre la voiture.


20. Carrousel en forme d'amadou

21. Clavier virtuel


Clavier virtuel.


22. Weather App


Application pour afficher la météo. Une API et un proxy spéciaux sont utilisés.
