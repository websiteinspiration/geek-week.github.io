＃4。絆


債券は、特定の利息収入が所定の期間に所定の金額で支払われる金融商品です。

債券の支払い構造がどのようになるかを考えてみましょう。 3年間の債券があるとします。債券を購入する場合、買い手は債券の額面金額を支払う必要があります。この例では、ボンドNの額面を100ルーブルとします。債券のクーポンの支払いは定期的に行われます。クーポンの支払いはパーセンテージで測定されます。 5％に等しいと仮定します（C = 5％）。債券の有効期間が終了すると（満期時）、所有者は額面と最後のクーポンの支払いを受け取ります。


図に示すように、クーポンの支払いは毎年行われます。原則として、債券発行者は、四半期ごとまたは半年ごとなど、任意の頻度の支払いを割り当てることができます。

ゼロクーポン債


クーポンがまったく支払われないタイプのボンドもあります - ゼロ - クーポンボンド（ゼロクーポンボンド）。このような債券は、額面から割引価格で（割引価格で）購入されます。この債券の額面金額は満期時に支払われ、最初に購入者は額面金額よりも少ない特定の金額を支払います。そのような債券の利益は、債券の購入価格とこの債券の額面価格の差になります。その額面が100ルーブルN = 100であり、この債券の割引が5％d = 5％で、1年で返済するとします。割引式によれば、債券の現在価値は、この5％の率で割引された額面価格と等しくなります。

PV = N /（1 + d）= 100 / 1.05 = 95.24ルーブル。

これは、最も単純な債券の公正価格を計算する例です。なお、有効期限が1年のゼロクーポン債が本日発行されなかった場合、たとえば6か月前に発行したいと考えて購入した場合、債券の価格が高くなります。すでに6か月が経過しているため、低金利で割引するため、債券の費用が高くなります。

債券の有効期間は満期と呼ばれます。満期時（額面金額の支払い）に、債券の額はその額面、この例では100ルーブルと等しくなります。

PV = C /（1 + r 1）+ C /（1 + r 2）2 +（C + F）/（1 + r 3）3

満期までの利回り


ツールとしての債券の重要な指標の1つは、YTM（満期利回り）という概念です。これは、債権者が満期日まで債券を保有する場合に受け取る債券利回りの単一の金利です。つまり、すべてのクーポンと額面金額が最後に支払われるのを待っています（途中で売却するのではありません）。この収益性を判断する方法は？

ゼロクーポン債の場合、すべてが非常に簡単です。債券を購入する際の割引率は、満期までの利回りになります。この例では、5％の割引で購入すると、債券の利回りは5％になります。クーポン債についてはどうですか？実際のところ、各クーポンは独自のレートで割引されます。今のところ、これらのレートの計算方法を下げます（これらは、商品の現在の価格に基づいて計算されます）。満期までの債券の利回りを知るには、割引する金利ではなく、債券の現在の価格を知る必要があり、単一のレートに置き換えます。 n次の方程式を解くと、この収益性の大きさを決定できます。

また、金利は債券の価格に影響を与える重要な要素であることも注目に値します。満期日まで債券を保有している場合、金利の変動は影響を与えません。債券の現在の価格がどのように変化するかは問題ではないため、クーポン収入のためにそれを保持します。有効期間中に債券を売却したい場合、金利の変動は債券の価格の変動につながることを理解することが重要です。

したがって、今日、100ルーブルの債券を購入した場合、金利が上がると債券の価格が変わり、債券の費用が安くなります。金利と債券価格の変化は逆転します-金利が上がると債券の価格が下がり、その逆になります（金利が下がると債券の価格が上がります）。

＃5。市場協定


市場には、すべての参加者が使用する一定の合意があります。トランザクションを締結する場合、これらの合意は個別に指定されることすらありません。これは、ある意味で、共通の理解です。また、債券価格にも関係しています。

債券の価格は、本質的に債券の額面に対する割合です。債券の額面が1000または1万ルーブルであるとしましょう。債券の売り手に価格の名前を付けるように依頼すると、彼は98、100、105などの番号を付けます。これらの番号は、債券の額面のパーセンテージによって決定されます。つまり、債券の価格は額面のパーセンテージです。したがって、価格が100未満であると言われた場合、これは債券が割引価格で（割引価格で）取引されていることを意味します。債券が100枚売られた場合、それは額面価格で販売されることを意味します。まあ、債券の価格が100を超える場合、プレミアムで取引されています。

債券を割引またはプレミアムで販売できるのはなぜですか？クーポンの金利が既存の市場金利よりも高い場合、そのような債券はプレミアムで売却される可能性が高くなります。これは、債券の買い手が市場に存在する他の商品よりも大きなクーポン収入を得る機会だからです。したがって、この機会は債券の価格に含まれています。購入時に、購入者は額面以上の金額を支払います。

同様に割引あり。クーポンの金利が市場の既存の金利よりも低いことが判明した場合、買い手はこの債券を購入するために、債券の額面に対して割引を提供されます。

日カウント契約


債券市場におけるもう1つの重要な市場合意は、日数の計算に関する合意です（日数計算規則）。債券の価格を計算するための式を思い出して、日数計算の慣習とは何かを検討してみましょう。



債券の価格は、割引されたクーポンの支払いと、債券の割引額面価格の合計に等しくなります。特定のケースでは、クーポン収入が年に1回より多く発生する場合、レートr nは年次で表されるため、係数が表示されます。彼はこのレートを債券の発行者によって提供される発生頻度にもたらすでしょう。

発生が四半期ベース、つまり年に4回発生するとします。次に、各ベットrnに4分の1を掛ける必要があります。すべてが単純に見えるかもしれませんが、そこにはありませんでした。市場と発行者によっては、日数計算の概念があります。クーポンの支払いの間に経過した現在の日数（四半期や月など）を360または365で割った値を度数の乗数として使用する必要があると記載されています。

次の式は文献に記載されています：

ACT / 360
ACT / 365

さらに、ACT / ACTエントリがあります。したがって、クーポン収入の計算が毎月発生し、当月が31日の場合、31/365（うるう年の場合は31/366）として数える必要があります。また、月や年に関係なく、割引クーポンの支払いを計算するための係数が30/360である場合にも、この契約を使用すると便利です。

債券価格には何が影響しますか？前述のように、債券価格と市場金利の関係は反比例します。満期までの残り時間も債券価格に影響します。これを模式的に描きましょう。

3つの結合があったとしましょう。 1つは額面で取引され、もう1つはプレミアムで取引され、3番目の債券は割引価格で取引されました。 3つの債券はすべて同じ満期であると仮定します-5年。これらの債券の価格は、満期までの時間によって異なります。


Pは価格、Tは満期までの残り時間です。プレミアム付きで販売された債券は、償還の時期に近づくと、価格が名目値まで下がります。割引価格で販売される債券は、返済の時期に近づくと、価格が名目価格まで上昇します。額面価格で売却された債券の価値は、満期までの残り時間には依存しません。

これは、満期までの時間に対する価格の依存にのみ適用されることに注意してください。他の多くの要因が債券の価値に影響を与える可能性があります。たとえば、市場での金利の変化は、それ自体に影響を及ぼします（金利の上昇に伴い債券の価格は低下し、金利の低下に伴い債券の価格は上昇します）。

ダーティでクリーンな価格と日数契約の使用例


私たちは3年の絆を持っています。このボンドで支払われるクーポンは5％です。この債券の利回りは4.5％です。英語での収益性は「成熟度への利回り」のように聞こえます。さらに、便宜上、英語の頭字語YMTを使用します。

3Y、C = 5％
YMT = 4.5％

債券は年に1回支払われます。それはこのように見えます：年の間に、クーポン収入は年の終わりに蓄積されて支払われます。 2年目も同じことが起こります。 3年目には、クーポン収入が支払われ、債券の額面が所有者に返されます。

私たちのボンドは2019年5月15日に発行され、2019年6月1日にそれを誰かに売却することを決定しました。価格はいくらですか？債券の価格は、割引クーポンの支払い額に債券の割引額面額を加えたものに等しいことを思い出してください。



ここから次の情報が得られます。最初のクーポンが支払われるまで349日が経過します。 2番目のクーポンが支払われるまで-714日間。そして、ボンドの額面と一緒に最後のクーポンは1079日で支払われます。


日数のカウントに関する合意を使用してクーポンの支払いを正確に計算するには、これらの日が必要です。

次の形式で日数のカウントに関する合意を使用します：ACT / 365。次に、債券の価格は、クーポン（5％）に債券の額面額を掛けたものに等しく、100を1で割ったものに利回りの利率を加えたもので、349/365（これは、購入の瞬間から最初のクーポンが支払われるまでの日数です） ）

同様に、次のクーポン：5を1で割って利回りを714/365の程度まで（これは、2番目のクーポンが支払われる前にどれだけ残っているかです）。そして、クーポンに3年目の終わりに支払われる債券の額面を1プラス利回りで割った値が1079/365になります。これはすべて101.57になります。


いくらですか？これは汚い債券価格です。それは初年度に支払うときにすべての累積クーポン収入が含まれているためです。ただし、債券を購入し、最初の17日間の累積クーポン収入を売り手に支払います。正味価格を計算するには、累積クーポン収入を計算する必要があります。これは、クーポン（5ルーブル）に17/365（これは債券保有者が受け取った収入です）を掛けたものに等しくなり、最終的には0.23ルーブルになります。
