Googleハングアウト



www：hangouts.google.com

Googleハングアウトは、2013年に導入された同社のいくつかのサービスの相続人です。

利点この

ツールは、カレンダーやGmailに関連付けられているという点で便利です。つまり、Googleカレンダーで予約すると、Googleハングアウトルームへのリンクが自動的に生成されます。

Googleハングアウトプラグインは、人気のSlackからのグループビデオ通話に使用できます（メッセンジャーは、チーム内のコミュニケーションに重点を置いているため、コレクションには含まれていませんでした）。

Googleハングアウトでは追加の登録やアクティベーションは必要ありません。Googleアカウントをお持ちの場合は、ハングアウトにも登録があることを考慮してください。プラットフォーム間の切り替えや、チャットとビデオ通話の履歴の同期に問題はありません。

欠点

このサービスは、不安定な環境でのビデオ会議ではうまく機能しません。無料版は10人以下の参加者をサポートします-ただし、ローカル会議は自動的にYouTubeにブロードキャストできます。

2017年、このツールに基づいて、企業ユーザー向けの2つのサービス、Hangouts MeetとHangouts Chatがすぐに登場しました。 2018年の終わりに、ユーザーの新しいサービスへの移行に伴い、Googleハングアウトの閉鎖が発表されました。確かに、特定の日付は呼び出されませんでした。世界的な状況を考えると、Googleは今後数か月でそうすることを敢えてする可能性は低いです。逆にコロナウイルスのため、同社は3月の初めから、ビデオ会議への参加者の許容人数を250人に増やしました。この制限は、2020年7月1日まで継続します。

Googleハングアウトには、Pexipゲートウェイを介して、Skype for Businessなどの他のビデオ会議ツールとやり取りする機能があることは注目に値します。これは、各開発者が独自のプロトコルとコーデックを使用し、競合するサービスと対話できないビデオ市場における重要なステップです。ただし、この相互作用のセットアップは別の問題です。

ズーム



www：zoom.usほんの

数日で、ズームは小さなウェビナーやディスカッションの事実上の標準になりました。ニュースでは、雪崩のようなユーザーの増加が指摘されています。これには、教育セグメントが含まれるため、ほぼ完全になくなりました。

同じツールを大規模なイベントで使用して、スピーチの後またはその間にリスナーとスピーカーとのやり取りを整理します（たとえば、最近翻訳されたMoscow Python Conf ++ 2020オンラインフォーマットのフレームワークでZoomがこのように使用されました）。

メリット

ズームは、比較的大規模な会議に焦点を当てています。競合他社とは異なり、「手を上げる」機能と、ビデオ通信を制御できるさまざまなオーガナイザーツールがあります。イベントがローカルシステムとクラウドの両方に記録できることも重要です。競合他社に加えて、ズームは比較的少ないトラフィックを消費します。

有料の関税に関して、Zoomには追加機能を実装するためのAPIがあります。一般的に、ツールには多くのオプションがあり、使用した機能に対してのみ支払うことができます。リストには、レポートやOutlookまたはLyncとのやり取りも含まれています。特定の対象者向けのウェビナーの主催者向けの個別のパッケージがあります。主催者はこの料金のみを支払います-参加者は無料で接続されます。

欠点

彼らは条件付きです。たとえば、会議は24時間を超えることはできません。その他の制限は、選択した料金プランによって異なります。

Zoomでは無料で、最大100人までのビデオハングアウトを40分以内で開催できます。1対1の時間制限は通信には適用されません。

シスコのビデオ会議ソリューションの「モンスター」によるエンタープライズレベルのツール。企業にとってのソリューションのプラスは、ハードウェアビデオ会議ソリューションおよび以前に統合できた特殊なデバイスとインターフェースすることです。最近リモートサイトにアクセスしたばかりのエンドユーザーにとって、この機能は特に興味深いものではありませんが、解決策については言及できませんでした。

正式には、このツールはパートナーを通じてのみロシアで利用可能です-VPNなしでクライアントを登録してダウンロードすることはできません。関税政策もパートナーに依存します。

Getcourse



www：getcourse.ru

このプラットフォームは、この市場でのオンライン教育と収益のために設計されています。ライブで開催される、または特定のスケジュールに従って自動的にレコーディングでサブスクライバーに提供されるレッスンとコースを作成できます（事前に記録されたコースを送信する頻度を設定できます）。

サービスの料金は、システムで許可されている接続ユーザー（コースリスナー）の数によって異なります。ファイルストレージは別途お支払いいただきます。

利点

ビデオコース自体に加えて、このプラットフォームは、タスクを完了するためのツール、監視、マルチレベルのユーザー管理、資金調達、モバイルプラットフォーム用の講師とのコミュニケーション-Chatiumを提供します。

残念ながら、このツールには無料の関税はありませんが、1000ユーザーまでの制限付きの試用版が提供されています。有償で、講義は幅広い視聴者（レビューの用語-放送）とWebセミナーで利用できます。参加者数（500人まで）には制限がありますが、誰でも発言できます。

不和



www：discordapp.com

Discord messengerはもともとコンピューターゲームのファンのために作成されましたが、udalenkaへの大規模な移行により、オンラインイベントや学校でのオンライン学習（生徒の個人的な主導）にも使用されるようになりました。大規模なオンライン会議では、メインの講義と並行して、参加者とスピーカー間のコミュニケーションのマルチチャネル手段として使用されます。

サービスの主な機能は無料で提供されます。月額5ドル（または4.99ドル）で、プロファイルをパーソナライズし、大きなファイルと1080pビデオストリームを60フレーム/秒で転送できます（画面のデモ-最大1080p @ 30fpsまたは720p @ 60fps）。ブーストサーバーを含む、より高価な料金があります。

パンデミックにより、会議の参加者が一時的に10人から50人に増えたことは注目に値します。

一般的に、一般ユーザーはDiscordに非常に好意的です。

YouTubeブロードキャスト




記述されている他のサイトを背景にして、YouTubeは、スピーカーと対話せずにビデオストリームをブロードキャストすることに重点を置いています。質問をする唯一の方法は、コメントにそれを書いて、スピーカーがそこを見ることを望むことです。

ブロードキャストはどのアカウントでも利用できません-電話番号を確認した人のみ。

すべての通信制限により、YouTube放送はそのシンプルさに感動します。ビデオは、コンピューターのWebカメラから、またはアプリケーションのモバイルデバイスからブロードキャストできます。YouTubeライブはHDビデオをサポートしていますが、パンデミック中のトラフィックの増加により、サービスはSDに強制的な制限を課しました。
