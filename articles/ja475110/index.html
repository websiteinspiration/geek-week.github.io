機能一覧


画面出力：ノートブックに出力を表示する方法はいくつかあります。以下の画像に示す方法のいずれかを使用できます。



オブジェクトフォーマッタ：デフォルトでは、.NETメモ帳では、ユーザーはオブジェクトに関する有用な情報を表形式で表示できます。



HTML出力：デフォルトでは、.NETメモ帳には、HTMLを記述するためのいくつかのヘルパーメソッドが付属しています。HTMLとして文字列を記述したり、JavaScriptを出力したりできる基本的なヘルパーから、PocketViewを使用したより複雑なHTMLまで。



パッケージのインポート：次の構文を使用して、NuGetパッケージをダウンロードできます。

#r "nuget:<package name>,<package version>"

例えば

# r "nuget:Octokit, 0.32.0"
# r "nuget:NodaTime, 2.4.6"
using Octokit; using NodaTime; 
using NodaTime.Extensions; 
using XPlot.Plotly;

Apache Sparkの.NET


ビッグデータを

使用する場合に対処するには、.NET Notebookのビッグデータのサポートが不可欠です。ノートブックを使用すると、データサイエンティスト、機械学習エンジニア、アナリスト、およびビッグデータに関心のあるすべての人がモデルをすばやく作成し、クエリを実行および分析できます。

現在、.NET開発者は、ノートブックで.NET for Apache Sparkクエリを実行するために、Azure Synapse Analytics  Notebooksと  Azure HDInsight Spark + Jupyter Notebooksの 2つのオプションを利用できます  。どちらのオプションでも、データの読み取り、変換、視覚化などの完全で複雑なビッグデータスクリプトの開発に加えて、高速のアドホッククエリを記述して実行できます。

オプション1：  Azure Synapse Analytics Apache Spark（C＃）の既製の.NETサポートが付属しています。



オプション2：  GitHub  でApache Sparkの.NET チュートリアルを  確認し、 HDInsight + Jupyterメモ帳で.NET for Apache Sparkを使い始める方法を学びます。
