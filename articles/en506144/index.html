3. Create an infobase on the server;
4. Install on client computers and configure access to the infobase.

The server option is installed, as a rule, by an experienced 1C specialist.

Conclusions:
1. The initial purchase price in the initial version is the same.
2. The price of 1C service is much higher, because very often the help of a programmer is required.

4. The user interface in the Client Communicator is much more convenient.
5. Improvements in 1C are more time-consuming and require a certain qualification of a specialist.
6. For 1C, the enterprise uses the best server available, even only for accounting tasks.
