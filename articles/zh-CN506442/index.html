透明时钟




找到同志页面后，TopText程序提出了编写手表的想法。所做的最大更改是拖动桌面上的铭文并选择文本选项-字体的颜色和类型。这些选项位于系统托盘图标中，SetWindowCallback（@WndProc（））用于处理消息。

这个想法的实质是以文本形式形成窗口区域（以win32api Region的术语）。创建的区域叠加在窗口上，因此变成半透明的。
在Internet上描述了拖动窗口的方法（但没有描述），因此，使用消息处理代码WM_LBUTTONDOWN。

鼠标时钟




类似的程序出现在DonationCoder.com上，但我不喜欢显示时间的方式-没有出现秒数。另外，如何编写类似的程序变得很有趣？MSDN文章对了解其工作原理有所帮助。

首先，创建“工具提示”窗口：

   hwndTT.l = CreateWindowEx_(#WS_EX_TOPMOST, #TOOLTIPS_CLASS, 0,#WS_POPUP | #TTS_NOPREFIX | #TTS_ALWAYSTIP,#CW_USEDEFAULT, #CW_USEDEFAULT, #CW_USEDEFAULT, #CW_USEDEFAULT,WindowID(0), 0, GetModuleHandle_(0),0)
   SetWindowPos_(hwndTT, #HWND_TOPMOST, 0, 0, 0, 0, #SWP_NOMOVE | #SWP_NOSIZE | #SWP_NOACTIVATE);

要更改文本，将结构添加到创建的窗口中。

    ti.TOOLINFO
     ti\cbSize   = SizeOf(TOOLINFO);
     ti\uFlags   = #TTF_SUBCLASS;
     ti\hwnd     = hwnd
     ti\hinst    = hinst
     ;ti\lpszText = @"This is your tooltip string."
     GetClientRect_(WindowID(0), @ti\rect);
     SendMessage_(hwndTT, #TTM_ADDTOOL, 0, @ti);

完成操作后，AddWindowTimer计时器（0，123，25）启动。

   Repeat
     Event = WaitWindowEvent()

     Select event
     Case #PB_Event_Timer:

       SendMessage_(hwndTT, #TTM_TRACKACTIVATE, #True, @ti)

         GetLocalTime_(st)

        SendMessage_(hwndTT, #TTM_UPDATETIPTEXT, 0, ti);

TTM_TRACKACTIVATE 消息告诉窗口激活工具提示。消息TTM_UPDATETIPTEXT更改使用API​​函数形成的文本-时间。最后一个操作-TTM_TRACKPOSITION消息更改了附加到光标的工具提示的位置。
