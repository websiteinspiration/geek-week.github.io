パートI.プロジェクトのユーザーインターフェイスを改善するための5つのiOSライブラリ（オリジナル）


I.1。グメニュー


Swift 5.1で作成され、SwiftUIを使用して実装された強力なライブラリであるGmenuから始めましょう。これは、Gmailアプリケーション用に作成された再設計されたメニューですが、プロジェクトに簡単に統合できます。 このライブラリは、上部の検索バーのボタンから始まり、大きなアイコンのメニューを表示し、行を（表形式で）下にスクロールします。

I.2。ImageViewer.swift




ImageViewer.swiftは、画像ギャラリーに使用される使いやすいライブラリです。

「夜間」モード、横向き、ドラッグアンドドロップをサポートします。

I.3。サイドメニュー


SideMenuは、カスタマイズ可能なコントロールを備えたサイドメニューです。ストーリーボードまたはプログラムで実装でき、8つの異なるスタイルのアニメーションが利用できます。

設定可能な「スワイプ」ジェスチャーがサポートされ、サイドメニューを両方向に切り替えることができます。

I.4。SPPermissions



SPPermissions-ユーザーに許可を求めるAPI。3つの異なるモード：リスト、ダイアログ、ネイティブ。

アップルスタイルのインターフェースは、iPhone、iPad、ナイトモード、tvOSをサポートしています。

サポートされている許可要求：•カメラ •フォトライブラリ •通知 •場所 •マイク •カレンダー •連絡先 •動き •メディア •ボイスメッセージ •Bluetooth

I.5。SPStorkController



最後に、SPStorkControllerはSwiftで作成されたコントローラーであり、Apple Music、Mail、Podcastアプリに似ています。

ナビゲーションバーの左側に閉じるボタンと中央に矢印を追加するだけです。

パートII 次のアプリケーション用の5つのiOSライブラリ（オリジナル）


II.1。PhoneNumberKit



以下のためのスターター、PhoneNumberKit、軽量iOSのライブラリーは完全に作成されたスウィフト5.0書式設定、および国際電話番号を確認し、分析のために、。

import PhoneNumberKit
let phoneNumberKit = PhoneNumberKit()

do {
    let phoneNumber = try phoneNumberKit.parse("+39 3 12 542644")
    let phoneNumberCustomDefaultRegion = try phoneNumberKit.parse("+39 23 2245 23442", withRegion: "IT", ignoreType: true)

catch {
    print("Generic parser error")

  • withExamplePlaceholder属性attributePlaceを使用して、の例の番号を表示しcurrentRegionます。


II.2。DateScrollPicker



DateScrollPickerは、Swift 5で記述された完全にカスタマイズ可能な日付ピッカービューです。

このカレンダーには、無限の日付セルのコレクションと、独自のフォントと色を選択してカスタマイズできるアニメーションの選択が含まれています。

たとえば、表示される日数の調整、日付形式のマーク、フォント/色の変更、その他の多くのパラメーターの設定は簡単です。

II.3。PostalCodeValidator



PostalCodeValidatorは、200以上の地域をサポートする郵便番号のバリデーターです。

機能する数行のコード：
import Foundation
import PostalCodeValidator

if let validator = PostalCodeValidator(locale: .current) { // United States (IT)
    validator.validate(postalCode: "43100")) // true

    validator.validate(postalCode: "123456789")) // false
}

II.4。液体スワイプ



リキッドスワイプは、swipe-containerアプリケーションで使用できる使いやすいコントロールです。たとえば、チュートリアルの最初の実行や適応のために使用できます。

これを使用するにLiquidSwipeContainerControllerは、ストーリーボードまたはコードからクラスのインスタンスを作成し、クラスに従ってデータソースを設定する必要がありますLiquidSwipeContainerDataSource。

II.5。フローティングパネル



最後に、このセクションでは、FloatingPanelは、Apple MapsなどのiOSアプリケーション用のクリーンで軽量なフローティングパネルのユーザーインターフェイスです。

ライブラリには、キャプチャマーカー、キャプチャ記述子のレイアウト、コンテンツの塗りつぶし、ジェスチャのフィールドサイズやイベントなど、多くの設計設定が用意されています。パーツの追加のフローティングパネルを作成することもできます。

詳細については、公式のREADMEがあります。
