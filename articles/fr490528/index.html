1. SQL Server sur Linux


SQL Server fonctionne désormais sous Linux. Ce cours vous aide à acquérir une compréhension de base de SQL Server sur Linux. Ensuite, vous apprendrez à exécuter des conteneurs SQL Server sur Linux et à déployer SQL Server sur Linux. Après cela, vous apprendrez comment configurer automatiquement le déploiement de SQL Server sur Linux.

Vous pouvez commencer la formation ici

2. Créez des solutions d'analyse open source avec Azure HDInsight


Dans ce cours, vous découvrirez HDInsight et comment appliquer cette technologie pour résoudre une variété de tâches du monde réel.

Parmi les modules de cours:

  • Introduction à l'offre d'analyse open source
  • Choisissez la configuration HDInsight appropriée pour créer des solutions d'analyse open source
  • Création et configuration d'un cluster HDInsight
  • Effectuer des analyses Zero ETL avec HDInsight Interactive Query

Commence l'entraînement

3. Implémentation d'une solution de flux de données à l'aide d'Azure Streaming Analytics


Découvrez les principes du traitement des événements et de la diffusion en continu et découvrez comment cela s'applique à Azure Stream Analytics. Ensuite, configurez votre travail Stream Analytics pour diffuser des données et découvrez comment surveiller et suivre l'exécution des travaux.

Commence l'entraînement

4. SQL Server Learning New


La dernière version de SQL Server offre de nombreux avantages pour des performances et des fonctionnalités accrues, ainsi qu'une prise en charge améliorée de la plate-forme de base de données. Dans ce didacticiel, vous apprendrez à inventorier les déploiements SQL Server à l'aide de Microsoft Assessment and Planning Toolkit. En outre, vous apprendrez à mettre à niveau les bases de données SQL Server à l'aide de l'assistant de migration de données et à optimiser votre système mis à niveau à l'aide de l'assistant d'expérimentation de base de données et de l'assistant de réglage de requête.

Commence l'entraînement

5. Utilisation des données NoSQL dans Azure Cosmos DB


Les données NoSQL sont un moyen efficace de stocker des données qui ne répondent pas aux exigences d'une base de données SQL relationnelle. Découvrez comment utiliser le portail Azure, l'extension Azure Cosmos DB pour Visual Studio Code et le SDK .NET Core pour Azure Cosmos DB pour travailler avec les données NoSQL dans n'importe quel emplacement, ainsi que pour fournir aux utilisateurs un haut niveau d'accessibilité, où qu'ils se trouvent.

Commence l'entraînement

6. Azure pour les ingénieurs de données


Découvrez comment le monde des données évolue et comment l'avènement de la technologie cloud offre de nouvelles opportunités commerciales. Vous découvrirez les différentes technologies de plate-forme de données disponibles et comment un ingénieur de données peut tirer parti de ces technologies au profit de l'organisation.

Parmi les modules:

  • Le monde évolutif des données
  • Présentation des services Azure Data Platform
  • Définition des tâches d'un spécialiste de l'ingénierie des données dans une architecture hébergée dans le cloud

Commence l'entraînement

7. Créez et utilisez des rapports analytiques dans Power BI


Ce cours vous présentera Power BI et vous apprendra à utiliser et à créer des rapports de Business Intelligence.

Parmi les modules:

  • Premiers pas avec Power BI
  • Récupération de données à l'aide de Power BI Desktop
  • Modélisation des données dans Power BI
  • Utilisation d'éléments visuels dans Power BI
  • Explorer les données dans Power BI
  • Publier et partager dans Power BI

Commence l'entraînement
