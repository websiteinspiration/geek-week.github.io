意外的HTTP标头

几天前，我在Creditkarma博客上采摘并注意到此HTTP标头：

X-hacker: If you're reading this, you should visit wpvip.com/careers and apply to join the fun, mention this header.

（X-hacker：如果您正在阅读此书，则应转到wpvip.com/careers并申请加入该活动，请提及此标头）。

我的第一个念头是：“哇，一旦程序员介绍了一个千年错误以节省一些日期，现在该公司发布了有关使用HTTP标头的全部报价！”

我变得非常有兴趣，并且做了一些研究。

如果您在WordPress VIP（由Automattic托管的WordPress托管公司）上托管网站，则该特定标题似乎是“标准”的。在许多知名网站上都可以找到相同的标题，例如：

  • nypost.com
  • techcrunch.com
  • www.nbcolympics.com
  • www.thesun.co.uk
  • 成千上万

网站开发人员和网站所有者可以将其关闭，但是说实话，我怀疑他们什至知道此标头包含在每个HTTP数据包中。当然，第二个想法马上就来了-检查其他公司是否有任何创意头条。

结果是惊人的！

您可以在HTTP标头中找到各种工作机会。


是! 世界上最酷的公司似乎在“标准” HTTP标头中发布工作机会x-recruiting。

以下是一些示例：

PayPal.me

x-recruiting: If you are reading this, maybe you should be working at PayPal instead! Check out www.paypal.com/us/webapps/mpp/paypal-jobs

Booking.com

x-recruiting: Like HTTP headers? Come write ours: careers.booking.com

Etsy.com

x-recruiting: Is code your craft? www.etsy.com/careers

Otto.de

x-recruiting: Seems you like http headers. To write ours, apply at job.otto.de and mention this header.

想查看完整列表吗？为此，我在GitHub上打开了存储库。

除了提供工作以外，在研究过程中还发现了更多有创造力的东西。他们真是使我成为神秘废话的忠实粉丝。

神秘的HTTP标头


9kw.eu网站似乎是一个验证码提供程序，它显示秘密消息“ 42”：

X-Secret-Message: 42

不再支持istreetview.com，但是可以在标题中找到秘密Web表单的地址。

X-Secret-URL: https://appio.link/secret

我给他们写了我的答案...

在

x-secret-sauce: Confidential

thetradersdomain.com 标题上有一个“隐藏的酱汁”，但这是机密的：在images-dnxlive.com上的HTTP标头之一中还有一些“秘密”链接：

X-Secret-Message: camscv.dnxnetwork.lu

如果您喜欢豪华车，那么这里是美洲虎。 ro带有用于检测漫游器的标头：

X-Bot: false

但是，如果替换用户代理（Jaguar的对不起人员），此方法将无法很好地工作并且会失败。

但是...您见过带有昵称的服务器吗？这里有一对：

X-men.com

X-ServerNickName: clint

Howgoodisyourseo.com

X-ServerNickName: The Internet

最后，也是一个有趣的例子。来自m.bidorbuy.co.ke的朋友在HTTP标头中表达了他们的热情：

x-powered-by: Passion and tiny cute kittens
x-servernickname: The Beast
x-hacker: If you are reading this, maybe you should be working at bidorbuy instead

奖金


许多新潮的IT公司似乎还有其他 HTTP标头，其中大多数包含工作机会。

因此，我认为向该站点添加一个附加标题将是很棒的！
