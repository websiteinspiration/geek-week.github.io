Webページのいくつかの要素が互いに近くにある場合、ユーザーはこれらの要素に共通点があると感じます。要素のグループ化は、ユーザーがそれらの間の距離を評価することにより、それらの関係を理解するのに役立ちます。すべての要素が互いに同じ距離に配置されている場合、ユーザーがページを参照して、どの要素が互いに関連していて、どれが関連していないかを見つけるのは困難です。 この記事は、要素間の距離の設定と要素の内部スペースの設定について知っておく必要があるすべてのものを扱っています。特に、内部パディングを使用する価値がある状況と、外部（マージン）について説明します。




CSS距離タイプ


CSSでカスタマイズ可能な要素とそのパーツ間の距離には、2つのタイプがあります。それらは、要素に関連して、内部と外部に分けられます。要素があると想像してください。その内部にある一部のパーツ間に一定の距離がある場合、これは内部のインデントです。個々の要素間の距離について話している場合-これは外側のインデントです。


内部空間と外部空間

CSSでは、要素とそのコンポーネント間の距離は次のように調整できます。

.element {
    padding: 1rem;
    margin-bottom: 1rem;
}

プロパティはpaddingここでインデントを調整するために使用され、プロパティmarginはインデントを構成するために使用されます。すべてが非常に簡単です。真実？ただし、多くの小さなパーツと子を持つコンポーネントを操作する場合、CSSで距離を調整することは非常に複雑になる可能性があります。

▍マージンプロパティ-インデント


このプロパティはmargin、個々の要素間の距離を設定するために使用されます。たとえば、前の例ではCSSプロパティmargin-bottom: 1remを使用して、上下に配置された2つの要素間の垂直距離を追加しました。

外部インデントは、要素の4つの側面（上、右、下、左-上、右、下、左）に設定できます。したがって、距離を設定するさまざまな方法の例と議論に進む前に、いくつかの基本的な概念に光を当てることが重要です。

テーブルの行と列の間の距離を調整する必要があります。ここにマークアップがあります：

私は通常、コンポーネントをカプセル化された状態に保ち、外側のインデントを設定しないようにします。このため、grid_itemcard要素を配置する要素があります。

.grid--4 {
    display: flex;
    flex-wrap: wrap;
}

.grid__item {
    flex-basis: 25%;
    margin-bottom: 16px;
}

このCSSコードのおかげで、各行に4枚のカードがあります。それらの間の距離を調整する1つの可能な方法を次に示します。

.grid__item {
    flex-basis: calc(25% - 10px);
    margin-left: 10px;
    margin-bottom: 16px;
}

CSS関数を使用すると、calc()インデントがから差し引かれflex-basisます。ご覧のとおり、これはそれほど単純なソリューションではありません。私は実際には以下を好みます：

  • グリッドアイテムのプロパティを設定しますpadding-left。
  • 親要素をmargin-left、yと同じ値の負の外側マージンを持つように設定しpadding-leftます。

私は何年も前にこの方法について学びました。ここで名前を忘れてしまったいくつかの記事を読んだ後（どのような種類の記事を知っているなら、教えてください）。

.grid--4 {
    display: flex;

    margin-left: -10px;
}

.grid__item {
    flex-basis: 25%;
    padding-left: 10px;
    margin-bottom: 16px;
}

ここで負の値を使用した理由margin-leftは、最初のカードにはpadding-left実際には必要ないプロパティがあるためです。その結果、ラッパー要素を左に移動して、不要なスペースを取り除きます。
