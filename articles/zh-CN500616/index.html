立即注册。

数据中心基础设施的创建和开发，创新的引入，对所用技术要求的不断增加需要大量时间，并且要具备广泛的能力，大量的资源和资金。
传统的方法是将各种组件组合起来以解决类似的问题，这是一个复杂且耗时的过程，通常会给企业带来额外的风险。

集成的Fujitsu PRIMEFLEX系统为市场提供了一种不同的方法-一种现成的高效合理的解决方案！

富士通PRIMEFLEX是功能强大，经过预先配置和测试的系统的代表，并且仍在不断增长。每个系统都作为一个单独的组件交付，包括服务器和预安装的软件，这大大降低了后续部署和操作的复杂性。

适用于Microsoft Azure Stack HCI的PRIMEFLEX是一个集成系统，其中包括部署基于Microsoft技术的超融合IT基础结构所需的所有必要硬件和软件。此解决方案使用Windows Server 2019中集成的高性能节能x86富士通PRIMERGY服务器，软件定义的服务器和存储技术（Hyper-V和Storage Spaces Direct）以及Windows Admin Center中集成的Azure混合服务，可轻松连接本地企业服务到云。

在网络研讨会上，您将学到：

  • 什么是富士通PRIMEFLEX集成系统
  • Azure Stack HCI的用途是什么？
  • 适用于Azure Stack HCI的Fujitsu PRIMEFLEX应用方案
