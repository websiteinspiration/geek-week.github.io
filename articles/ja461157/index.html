フィードバックアシスタントが提供するもの-バグレポーターに代わる開発プラットフォーム

今日、私たちはITコミュニティに影響を与えるAppleポリシーの変更やその他の変容について話すことを決定しました。何が起こっているのか、コミュニティがそれにどう反応するのかについて話し合います。


写真-Maurits Bausenhart -Unsplash

バグレポーターは過去のものです


Appleは6月の会議中に、Bug Reporterのサポートを停止すると発表しました。これは、アプリケーションでエラーメッセージを送信するためのツールです。現在、開発者はフィードバックアシスタントを介してすべてのバグレポートを送信する必要があり、bugreporter.apple.comは既に訪問者をfeedbackassistant.apple.comにリダイレクトしています。

このアプリケーションは、WebおよびiOS 12.4ベータ、iOS 13ベータ、およびmacOS Catalinaベータシステムで利用できます。Macでは、フィードバックアシスタントは/ System / Library / CoreServices / Applicationsディレクトリにあります。applefeedback：// URLスキームを使用することもできます。

新機能


Appleによれば、フィードバックアシスタントを使用すると、開発者はバグレポートをより速く送信できるようになります。次のサービス機能がこれに貢献します。

バグレポーターからのレポートは、フィードバックアシスタントで利用できます。以前の通話のID、それらに添付されたファイル、テクニカルサポートの連絡文書がAppleに移行されました。


写真-Nigmatic -Unsplash

しかし、それでも多くのバグレポートが削除されました。これは、GDPRの要件を満たすために行われました。彼の5番目の記事によると、企業は処理目的が必要とする期間よりも長く個人データを保存することはできません。したがって、Appleは5年以上前に議論が終了したレポートを取り除きました。また、少なくとも60日間更新されなかった「バグレポート」が送信されました。

コミュニティが考えること


一般的に、コミュニティは新しい変化を積極的に歓迎しています。開発者は、ネイティブアプリケーションによってバグレポートの送信が本当に簡単になると言います。さらに、バグを分類するためのサブカテゴリがさらにあります。エキスパートは、新しいAPIやその他のアップルソリューションがより速くアップグレードすることを期待しています。しかし、ある楽器から別の楽器への移行は、希望どおりにスムーズに進みませんでした。彼のブログの開発者の1人は、彼の古いレポートにはすべて1つの日付が含まれていたため、検索がより困難になったと述べています。また、誰かのレポート識別子が変更されました。

同時に、フィードバックアシスタントはまだ不安定です。一部のユーザーはできませんバグレポートを送信します。スクリーンショットをダウンロードしようとすると、アプリケーションがクラッシュします。この問題は回避できますが、写真なしでレポートをダウンロードし、編集する必要があります。アップルのエンジニアが近い将来にすべての欠点を解消することが期待されています。

その他の変更について


6月初旬、Apple はアプリケーション開発者向けガイドラインの変更も発表しました。それらの最大のものは、子供向けのアプリケーションでの「サードパーティの分析サービスまたは広告」の使用の禁止と呼ぶことができます。その目的は、アメリカとヨーロッパの議員の要件、つまりインターネットプライバシー保護法とGDPRに準拠することです。

一部の企業は、新しい要件に関する否定的な意見をすでに表明しています。たとえば、PBSチャネルの代表者は指摘し、今では彼らがアプリケーションの性能を評価するために困難になり、結果として、その機能を向上させること。

VPNサービスの新しい禁止を強調することも価値があります。現在、ネットワーク上で匿名化ツールを開発している企業は、ユーザーの個人データを第三者に処理、販売、転送することを禁じられています。この決定は、最近のTop10VPN調査による可能性があります。組織によると、30の最も人気のあるVPNアプリケーションの半分は中国の企業が所有しています。この事実はアメリカ社会に懸念を引き起こした。

さらに、AppleはApp Storeから、許可なしにサードパーティのソースからユーザーデータを収集するすべてのアプリケーションを除外する予定です。近い将来、新会社方針の最初の結果を見ることができます。開発者は、2019年9月3日までに、新しい要件に従ってすべてのアプリケーションをストアに持ち込む必要があります。
