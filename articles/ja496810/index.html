まず、少し理論：

線形回帰について


線形回帰は、従属変数（予測しようとしている変数）と独立変数（予測に使用される変数または変数、あるいはその両方）の間で線形関係が想定される場合に、予測モデルとして使用されます。

最も単純なケースでは、検討するときに、ある変数を使用して、別の変数を予測しようとしています。この場合の式は次のとおりです

。Y= C + M * X

  • Y =従属変数（結果/予測/推定）
  • C =一定（Y切片）
  • M =回帰直線の傾き（推定直線の傾きまたは勾配。これは、Xを1単位増やすと、Yが平均して増える量です）
  • X =独立変数（予測Yで使用される予測子）

実際、従属変数といくつかの独立変数の間にも関係がある場合があります。これらのタイプのモデル（線形性を想定）では、次の形式の多重線形回帰を使用できます：

Y = C + M1X1 + M2X2 + ...

つまり、株式が14％成長し、市場が10％しか成長しなかった場合、株式のベータ係数は1.4になります。通常、ベータが高い市場ほど、報酬（およびリスク）の条件が改善されます。



練習


次のPythonコードには、線形回帰の例が含まれています。ここで、入力変数はモスクワ取引所指数の利回りであり、推定変数はアエロフロート株の利回りです。

データをダウンロードして計算に必要な形式にする方法を覚える必要をなくすために、コードは、データがダウンロードされた瞬間から結果が取得されるまで与えられます。以下は、statsmodelsを使用してPythonで線形回帰を行うための完全な構文です。

# 
import pandas as pd
import yfinance as yf
import numpy as np
import matplotlib.pyplot as plt
import statsmodels.api as sm

# 
ticker = ['AFLT.ME','IMOEX.ME']
stock = yf.download(ticker)

#    
all_adj_close = stock[['Adj Close']]

#   
all_returns = np.log(all_adj_close / all_adj_close.shift(1))

#      
aflt_returns = all_returns['Adj Close'][['AFLT.ME']].fillna(0)

#      
moex_returns = all_returns['Adj Close'][['IMOEX.ME']].fillna(0)


#   DataFrame

return_data = pd.concat([aflt_returns, moex_returns], axis=1)[1:]
return_data.columns = ['AFLT.ME', 'IMOEX.ME']


#      X  y
X = sm.add_constant(return_data['IMOEX.ME'])
y = return_data['AFLT.ME']


#  
model_moex = sm.OLS(y,X).fit()

#  
print(model_moex.summary())

FB株とSP500指数についても同じ分析。ここでは、元の計算と同様に、月次利回りで計算が行われます。

sp_500 = yf.download('^GSPC')
fb = yf.download('FB')

#    
fb = fb.resample('BM').apply(lambda x: x[-1])
sp_500 = sp_500.resample('BM').apply(lambda x: x[-1])

monthly_prices = pd.concat([fb['Close'], sp_500['Close']], axis=1)
monthly_prices.columns = ['FB', '^GSPC']

monthly_returns = monthly_prices.pct_change(1)
clean_monthly_returns = monthly_returns.dropna(axis=0)  

X = clean_monthly_returns['^GSPC']
y = clean_monthly_returns['FB']

X1 = sm.add_constant(X)

model_fb_sp_500 = sm.OLS(y, X1)

results_fb_sp_500 = model_fb_sp_500.fit()
print(results_fb_sp_500.summary())

  • 決定係数であり、0から1までの値を取るR-2乗。係数の値が1に近いほど、依存性が強くなります。
  • 調整。R-squared-観測数と自由度の数に基づいて調整されたR-2乗。
  • std err-係数推定の標準誤差。
  • P> | t | -p値0.05未満の値は統計的に有意であると見なされます。
  • 0.025と0.975は、信頼区間の下限値と上限値です。
