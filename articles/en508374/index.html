So, let's begin. Step one, you need to get the firmware in tar.gz format. You can download from our resource here . All snom icons are available and included in each version of the firmware.



Note : please note that each firmware version contains configuration files specific to this version and phone model . Using settings files that do not match the firmware or phone will cause problems.

Step two, preparing icons for the phones. As you know, Snom phones come with a color and monochrome screen, respectively, and the icons will differ.



I. Change Icons for Color Display Phones



Icons and images on phones with color displays are stored in PNG format. This allows them to be easily edited in almost all modern image editors. However, after editing, it is recommended that you optimize png files using tools such as optipng, pngquant or pngcrush to remove any redundant information and optimize file size.



Icon Image Sizes:



  • Context-Sensitive Key Icons 24x24px
  • SmartLabel 24x24px & 18x18px
  • Icons in the title bar (Titlebar Icons) 18x18px
  • Menu Icons 18x18px
  • During a call (Call Screen Icons) 18x18px - 48x48px
  • File Format: PNG


After creating the desired icons, download them to your phone. There are two ways to download:



  1. Via the web interface in manual mode
  2. Using auto-promotion

As you can see in manual mode, everything is quite simple, but if you need to change several phones, this process will take a lot of time. Therefore, we move on to the second option.



Option two - download via auto-provisioning.



First of all, you need to create an archive in tar format from the previously downloaded customizing.tar.gz archive . When creating an archive, delete all files and directories that you do not need to change, but make sure that you keep the directory structure .

Note . You do not need to archive all files that were originally in the archive. It is sufficient and recommended that you only archive files that you have modified. The more files you put in the archive, the more time it will take for the phone to configure it.


Next, we take several steps:



1) create an XML file, for example, branding.xml and copy it to your web server (HTTP), i.e. http: //yourwebserver/branding.xml: 



<?xml version="1.0" encoding="utf-8" ?>
<settings>
 <uploads>
  <file url="http://yourwebserver/branding/branding.tar" type="gui" />
</uploads>
</settings>


2) We go to the phone‚Äôs web interface in the section Advanced -> Update -> Setting URL and provide a link to our file yourwebserver / branding.xml















3) We reboot the phone and enjoy the result.
