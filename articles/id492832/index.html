IoT-gateway untuk protokol industri berdasarkan i.MX6 dan Linux



WISE-710 adalah perangkat universal yang secara bersamaan dapat menjadi gateway untuk antarmuka industri, perangkat untuk mengumpulkan dan memproses data, hub untuk perangkat IoT dan router. Dibangun pada SoC i.MX6 dan berjalan di Ubuntu atau Yocto Linux.

Bahkan, ini adalah komputer kompak dalam desain industri dengan antarmuka I / O bawaan untuk menghubungkan otomatisasi, sensor, dan periferal industri lainnya. Bersama dengan perangkat, pengembang memiliki akses untuk membuka SDK untuk membangun kernel dan program Linux, serta opsi dengan perangkat lunak EdgeLink pra-instal yang mendukung lebih dari 200 protokol industri berpemilik, serta protokol terbuka seperti MQTT, Modbus, dll.

Dalam artikel tersebut, kami menganalisis fungsi perangkat dan menggunakan case.

Spesifikasi


Perangkat ini dilengkapi dengan prosesor yang kuat untuk menjalankan program produktif: penyimpanan lokal dan pemrosesan data, konversi protokol, dan komputasi. Modul komunikasi nirkabel memungkinkan Anda untuk menggunakan perangkat sebagai gateway IoT mandiri dan perangkat komunikasi dengan objek.

  • Prosesor: ARM Cortex-A9 i.MX6 dua core 1GHz
  • Drive: 8GB eMMC
  • RAM: 1GB DDR3
  • Slot Micro-SD: 1
  • Antarmuka: 8 port input-output digital, RS-485 / RS-232
  • Ethernet: port 2x1 Gb / s
  • Modul Ekspansi: LTE, WiFi, Port I / O
  • USB: port 1x dalam mode Host



Perangkat ini dapat dipasang di rel DIN dan di permukaan apa pun menggunakan lubang pemasangan di bagian bawah kasing. Untuk pemasangan kontak antarmuka yang andal, digunakan terminal penjepit tanpa sekrup.



Modern SoC (System on Chip) i.MX6 memiliki konsumsi daya yang rendah dan tidak memerlukan pendinginan aktif atau bahkan memasang heat sink pada prosesor. Foto di bawah ini menunjukkan papan WISE-710 . Selama operasi, perangkat hampir tidak memanas. Untuk bekerja dengan data sensitif, perangkat ini memiliki modul kriptografi built-in ATECC508A, yang memungkinkan Anda untuk melakukan operasi dengan kunci pribadi tanpa kemampuan untuk mengekstrak kunci dalam bentuk yang jelas. Ini melindungi terhadap kompromi data dan kebocoran kunci jika perangkat dicuri.

Modul Ekspansi Antarmuka



WISE-710 memiliki port mPCIe onboard untuk menghubungkan modul ekspansi. Ini dapat digunakan untuk menghubungkan modul Advantech bermerek dan papan pihak ketiga, seperti modem, pengontrol, dll. Penutup belakang perangkat terbuka tanpa sekrup, yang memungkinkan Anda menghubungkan modul dengan cepat. Saat ini, kami sedang mengembangkan modul ekspansi yang akan kami perkenalkan dalam waktu dekat.

Buka SDK



Bersamaan dengan perangkat, toolchain gratis disediakan untuk perakitan mandiri Yocto Linux untuk kebutuhan pelanggan. Jika perlu, sistem operasi dapat sepenuhnya dikonfigurasi ulang, termasuk kernel OS. Misalnya, untuk menginstal patch Linux-RT yang memungkinkan Anda untuk melakukan operasi waktu nyata.

Kesimpulan


Perangkat WISE-710 segera menggantikan seluruh perangkat: router jaringan, dan sakelar, perangkat akuisisi dan transmisi data, pengontrol logika yang dapat diprogram, dan lainnya. Platform yang sepenuhnya terbuka memberikan peluang tak terbatas untuk mengembangkan solusi Anda sendiri untuk otomatisasi, pengumpulan data, komputasi lokal, dan banyak lagi.
