GoからRust関数を呼び出す

かつて、GoからRustコードを呼び出す方法に関するHabrに関する記事がありました。記事は悪くないが、理解するのはかなり難しく、実際に初心者は両方の言語に目を向けたいという欲求から反発します。この投稿の目的は、言語をまたがる呼び出しの本質を理解することではなく、それがいかに簡単に行えるかを示すことです。



私たちは、遠くに行くと反映されません例を錆言語を学ぶ上での本から。

この例で行うことは、10個のスレッドを開始することだけです。その中で、変数「x」を500万にインクリメントし、ストリームの終了に関するメッセージを表示します。

use std::thread;

#[no_mangle]
pub extern "C" fn process() {
    let handles:Vec<_> = (0..10).map(|_|{
        thread::spawn(||{
            let mut x = 0;
            for _ in 0..5_000_000 {
                x += 1
            }
            x
        })
    }).collect();

    for h in handles {
        println!("Thread finished with count={}",
                 h.join().map_err(|_| "Could not join thread!").unwrap());
    }
    println!("Done!");
}


また、貨物ファイルに行を追加して編集する必要があります。
crate-type = ["cdylib"]
その結果、Foreign Function Interface（FFI）を介して関数を呼び出すことができるライブラリが作成されます。

libembed.dylibはMac OSのライブラリであり、Linuxの場合はlibembed.so、Windowsの場合はlibembed.dllになることに注意して

ください。ありがとう：bingo347


Cargo.toml
[package]
name = "embed"
version = "0.1.0"

[lib]
crate-type = ["cdylib"]


一般的に、Rustライブラリでこれを行う必要があるのはこれだけです。本の例では、これについてより詳細に説明しているため、これについては詳しく説明しません。

次のコマンドでライブラリをコンパイルします。
カーゴビルド-リリース
ここでの目標は、Goアプリケーションからこのコードを呼び出すことです。シンプルなアプリケーションを作成し、プロジェクト内に/target/release/libembed.dylibファイルをコピーするlibフォルダーを追加します。内部では、関数の名前でファイルを作成し、その呼び出し署名を記述します。

lib / process.h
void process();


Goファイル内にそのようなディレクティブを追加すると、main.goは次のようになります

package main

/*
#cgo LDFLAGS: -L./lib -lembed
#include "./lib/process.h"
*/
import "C"

func main() {
	C.process()
}


プロジェクトをまとめる
go -ldflags = "-r / lib" main.go
ldflagsパラメーターに注意してください。この場合は、ELF動的リンカーへのパスを設定するだけです。

すべて。プログラムを実行して出力を取得します。



教えをGoプログラムからRustライブラリに転送できることも個別に言及する価値があります。これを行うには、Rustライブラリの関数を変換して、文字列値を取るようにします。

extern crate libc;
use std::thread;
use std::ffi::CStr;

#[no_mangle]
pub extern "C" fn process(name: *const libc::c_char) {

    let buf_name = unsafe { CStr::from_ptr(name).to_bytes() };
    let str_name = String::from_utf8(buf_name.to_vec()).unwrap();

    let handles:Vec<_> = (0..10).map(|_|{
        thread::spawn(||{
            let mut x = 0;
            for _ in 0..5_000_000 {
                x += 1
            }
            x
        })
    }).collect();

    for h in handles {
        println!("{}:Thread finished with count={}\n",
                 str_name,
                 h.join().map_err(|_| "Could not join thread!\n").unwrap());
    }
    println!("Done!");
}


プロジェクトをアセンブルし、ライブラリをlibフォルダーに再度コピーし、このようにprocess.hファイルを変更します
voidプロセス（char * name）;

Goアプリケーションからの行を渡します（この場合、「Hello from Golang」）。

package main

/*
#cgo LDFLAGS: -L./lib -lembed
#include "./lib/process.h"
*/
import "C"

func main() {
	C.process(C.CString("Hello from Golang !!"))
}


すべて。プログラムを実行して出力を取得します。
