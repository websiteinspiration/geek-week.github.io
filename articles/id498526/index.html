Apa yang Baru di Ubuntu 20.04


23 April mengadakan rilis Ubuntu versi 20.04, dengan nama kode Focal Fossa, yang merupakan rilis Ubuntu berikutnya untuk dukungan jangka panjang (LTS) dan merupakan kelanjutan dari Ubuntu 18,04 LTS, yang dirilis pada 2018.

Sedikit tentang nama kodenya. Kata "Focal" berarti "titik pusat" atau "bagian paling penting", yaitu, dikaitkan dengan konsep fokus, pusat segala sifat, fenomena, peristiwa, dan "Fossa" memiliki akar "FOSS" (Perangkat Lunak Bebas dan Sumber Terbuka - perangkat lunak bebas dan sumber terbuka), dan dalam tradisi penamaan versi Ubuntu dengan nama binatang, Fossa adalah mamalia predator terbesar dari keluarga wyverder dari pulau Madagaskar.
Pengembang memposisikan Ubuntu 20.04 sebagai pembaruan penting dan berhasil dengan dukungan selama 5 tahun ke depan untuk komputer desktop dan server.
Ubuntu 20.04 adalah kelanjutan logis dari Ubuntu 19.04 "Disco Dingo" dan Ubuntu 19.10 "Eoan Ermine". Dalam versi desktop, mengikuti tren terbaru, tema gelap telah muncul. Jadi di Ubuntu 20.04 ada tiga variasi dari tema Yaru standar:

  • Cahaya
  • Gelap
  • Standar.

Aplikasi Amazon juga telah dihapus. Ubuntu 20.04 menggunakan GNOME 3.36 terbaru sebagai shell grafis default .


Perubahan kunci


Ubuntu 20.04 didasarkan pada kernel 5.4, yang dirilis pada 24 November 2019. Dalam versi ini, beberapa inovasi penting diterapkan, yang akan kita diskusikan di bawah ini.

lz4


Insinyur Canonical menguji berbagai algoritma kompresi untuk kernel dan initramfs boot image awal, mencoba mencari kompromi antara kompresi terbaik (ukuran file lebih kecil) dan waktu dekompresi. Algoritma kompresi lossless lz4 menunjukkan hasil yang paling nyata dan ditambahkan ke Ubuntu 19.10, yang memungkinkannya untuk mengurangi waktu boot dibandingkan dengan rilis sebelumnya (Ubuntu 18.04 dan 19.04). Algoritma yang sama akan tetap ada di Ubuntu 20.04.

Linux Lockdown Kernel


Fungsi Lockdown meningkatkan keamanan kernel Linux dengan membatasi akses ke fungsi yang dapat memungkinkan kode arbitrer untuk dijalankan melalui kode yang disediakan oleh proses pengguna. Sederhananya, bahkan akun root superuser tidak dapat mengubah kode kernel. Ini memungkinkan Anda untuk mengurangi kerusakan dari serangan potensial, bahkan ketika akun root dikompromikan. Dengan demikian, keamanan keseluruhan sistem operasi juga meningkat.

exFAT


Sistem file Microsoft FAT tidak memungkinkan mentransfer file yang lebih besar dari 4 GB. Untuk mengatasi batasan ini, Microsoft menciptakan sistem file exFAT (dari bahasa Inggris. Extended FAT - "extended FAT"). Sekarang Anda dapat memformat, misalnya, drive USB ke exFAT menggunakan dukungan sistem file exFAT bawaan .

Wireguard


Meskipun Ubuntu 20.04 tidak akan menggunakan kernel 5.6, setidaknya saat ini juga, ia sudah menggunakan backport WireGuard di kernel 5.4. WireGuard adalah kata baru dalam industri VPN , sehingga dimasukkannya WireGuard di kernel sekarang memberi Ubuntu 20,04 keuntungan di cloud.

Memperbaiki bug dengan kuota CFS dan sekarang aplikasi multithread dapat berjalan lebih cepat. Seorang pengemudi telah ditambahkan yang memungkinkan bekerja dengan sensor suhu dan tegangan dari jalur prosesor Ryzen.

Semua ini jauh dari semua inovasi yang muncul di kernel 5.4. Ulasan terperinci dapat ditemukan di sumber kernelnewbies.org (dalam bahasa Inggris) dan di forum OpenNET (dalam bahasa Rusia).

Dukungan ZFS yang ditingkatkan


Meskipun Linus Torvalds mungkin tidak menyukai ZFS , itu masih merupakan sistem file yang populer dan dukungan eksperimental dengan Ubuntu 19.10 telah ditambahkan.
Hal ini cukup nyaman dan stabil untuk penyimpanan data, arsip rumah yang sama atau penyimpanan server di tempat kerja (di luar kotak dapat melakukan lebih dari LVM yang sama). ZFS mendukung ukuran partisi hingga 256 kuadriliun Zettabytes (karenanya huruf "Z" dalam nama) dan dapat memproses file hingga 16 Exabytes.

ZFS melakukan pemantauan integritas data berdasarkan bagaimana itu ditempatkan pada disk. Fungsi salin selama perekaman memastikan bahwa data yang digunakan tidak ditimpa. Sebaliknya, informasi baru ditulis ke blok baru, dan metadata sistem file diperbarui untuk menunjukkannya. ZFS memungkinkan Anda untuk membuat snapshot (snapshot dari sistem file) yang melacak perubahan yang dibuat pada sistem file dan bertukar data dengannya untuk menghemat ruang disk.

ZFS menetapkan checksum untuk setiap file pada disk dan terus-menerus memeriksa statusnya terhadapnya. Jika dia menemukan bahwa file tersebut rusak, dia akan mencoba mengembalikannya secara otomatis. Item terpisah telah muncul di penginstal Ubuntu yang memungkinkan Anda untuk menggunakan ZFS. Anda dapat membaca lebih lanjut tentang riwayat ZFS dan fitur-fiturnya di blog It's FOSS .

Selamat tinggal Python 2.X


Versi ketiga Python diperkenalkan kembali pada 2008, tetapi bahkan 12 tahun tidak cukup untuk mengadaptasi proyek Python 2 ke dalamnya.
Kembali di Ubuntu 15.10, upaya dilakukan untuk meninggalkan Python 2, tetapi dukungannya terus berlanjut. Dan sekarang, pada 20 April 2020, Python 2.7.18 dirilis , yang merupakan rilis terakhir dari cabang Python 2. Tidak akan ada lagi pembaruan untuk itu.

Ubuntu 20.04 tidak lagi mendukung Python 2 dan menggunakan Python 3.8 sebagai versi default Python. Sayangnya, ada banyak proyek yang menjalankan Python 2 di dunia, dan bagi mereka, transisi ke Ubuntu 20.04 bisa sangat menyakitkan.

Anda dapat menempatkan versi terbaru Python 2 dalam satu perintah:

# apt install python2.7

Selain Python 3.8, pengembang dapat menghargai toolkit yang diperbarui, yang meliputi:

  • MySQL 8,
  • glibc 2.31,
  • OpenJDK 11,
  • PHP 7.4,
  • Perl 5.30,
  • Golang 1.14.

Selamat tinggal 32 bit


Selama beberapa tahun, Ubuntu belum menyediakan gambar ISO untuk komputer 32-bit. Sekarang pengguna Ubuntu 32-bit yang sudah ada dapat melakukan upgrade ke Ubuntu 18.04, tetapi mereka tidak akan dapat melakukan upgrade ke Ubuntu 20.04. Artinya, jika Anda saat ini menggunakan Ubuntu 18.04 32-bit, Anda dapat tetap menggunakannya hingga April 2023.

Cara meningkatkan


Memutakhirkan ke Ubuntu 20.04 dari versi sebelumnya semudah menjalankan perintah berikut:

# sudo apt update && sudo apt upgrade
# sudo do-release-upgrade
