骨董品：ThinkPad 380E、Economy 90s、Windows 95

古い機器を集めることは、あなたが学ぼうとしている時間の現実に少し不誠実かもしれません。販売開始時のほとんどのビンテージノートブックは、高価（ThinkPad 600、ThinkPad T22）または非常に高価（ThinkPad X301）でした。当時、私が不用意に貧しくなければ、おそらくそのようなデバイスは購入しません。私たちは常に価格と品質の妥協点を見つけようとします。試すのが楽しいテクノロジーにお金を費やすことはありませんが、それらはまったく必要ありません。


今日お話ししたいノートパソコンは、1997年のエコノミークラスです。適度にパワフルで、適度に機動性がありますが、旗艦に比べて著しく劣っています。昨年、IBM ThinkPad 380Eをハードドライブとバッテリーを搭載した状態で購入しました。それは開かれたことがなく、すべてのユーティリティを備えた元のWindows 95がまだそこにインストールされているようです。ラップトップとこの最初の真にモダンなWindowsの両方を見てみましょう。そして、その信頼性がそれについて伝えられたすべてのジョークに対応するかどうかを決定します。

私は古い鉄の収集家の日記をリアルタイムで電報に記録します。今後数週間で、Windows CEの最初のPDAの1つであるPhilips Velo 1

の印象を共有します。IBMThinkPad 380シリーズのラップトップは1997年に発売されました。 1998年3月、ラインナップを更新しました。少なくともプロセッサに関しては、これは大きな進歩の時代でした。私自身の座標系では、1996年にリリースされたPentium MMXも信じられないほど強力であるように思われ、1997年5月にPentium IIが導入されました。ただし、1998年のIBMノートブックラインへの更新では、200または233メガヘルツの周波数のPentium MMXモバイルプロセッサがまだ使用されています。しかし、今日私たちは嘲笑の投稿がありますしたがって、節約は150メガヘルツで十分です。1997年のラップトップの価格はいくらですか？


ラップトップは高価でした。これは1997年11月のPC Magazineからの広告です。何年も前に私のヴィンテージのノートブック趣味を始めたThinkPad 380EDラップトップに最も注意が向けられています。脚注では、この記事の主人公の仕様と価格を見ることができます。


私のラップトップの仕様は広告と完全に一致しています：

プロセッサー：Intel Pentium MMX 150 MHz
メモリー：16メガバイトEDO SDRAM（後でボリュームを32メガバイトに増やしました、最大64）
ハードディスク：2.1ギガバイト
ディスプレイ：12インチ、800x600ピクセル、テクノロジーFRSTN、65k色
ビデオカード：Neomagic Magicgraph 128ZV、1,125メガバイトの内部メモリ
コネクタおよび通信：IrDA 1.1、PCMCIA 16ビット、シリアルおよびパラレルポート、VGA、ポートレプリケーター
バッテリー：NiMH、最大2時間のバッテリー寿命
寸法および重量： 300x233x60 mm 3キログラム


このような米国のラップトップの価格は2,099ドル、つまりインフレ調整済み​​の3,342ドルです。 CD-ROMドライブとより強力なプロセッサをラップトップに追加する小さなアップグレードには、500ドルかかります。しかし、シリーズの最上位デバイス（TFTマトリックスと5ギガバイトのハードドライブを備えた380ED）は、4,000ドル（インフレに合わせて調整された6,367）のコストがかかります。 2019年には、非常に優れたラップトップを2〜3千ドルで購入できますが、当時は、そのようなお金には予算オプションしか利用できませんでした。 ThinkPad 3xxシリーズは一般的にエントリーレベルのポータブルコンピューターで構成されていたことを忘れないでください。より薄い第5シリーズのラップトップと7xxデスクトップの代用品の価格はさらに高かった。同じ広告には、エントリーレベルの構成で4,700ドル（最新の7,500）のThinkPad 760XDが記載されている。


彼らが予算、通常のデバイスとプレミアムデバイスを分離するためにこの3-5-7デジタルシーケンスを最初に思いついたのはどこですか？BMWでは？2,000ドルを下回ると、パフォーマンスがさらに悪化した場合、またはあまり知られていないブランドに目を向けた場合にのみ、下落する可能性がありました。1998年に、IBMはThinkPad 310Eラップトップを1600の開始価格でリリースします。それは投稿の主人公とそれほど変わらないでしょう、それははるかに悪いように思われますが、画面だけがさらに悪いでしょう？

画面がひどいとき


何と言っても、動画を表示する方が簡単です。


2008年のフラグシップであるThinkPad X301のレビューで、私はすでに不満を述べました。これは、優れたラップトップであり、現代のタスクには比較的適しているようですが、ディスプレイです。今日、私たちは完全に大きな視野角を備えたIPSスクリーンに甘やかされており、普通のTFTに疑問を抱いています。スクリーンを少し変えると価値があり、色がすぐに歪むようになります。 1997年に選ばれたのは、スーパーツイステッドネマティックテクノロジーを採用した、新しくて高価なTFTと安価なディスプレイのどちらかでした。 STNテクノロジーによると、最初のラップトップの白黒画面が作成され、その後カラー画面が登場しました。通常のTFTスクリーン（当時は「アクティブマトリックス」と呼ばれ、各ピクセルのオンとオフを個別に制御する機能を意味していました）から、非常に高い応答時間、低いコントラスト、小さな動作視野角が特徴でした。


ThinkPad 380Eのラップトップディスプレイは、Fast Response STNテクノロジーを使用して作られていますが、このFastは150ミリ秒のパスポート応答時間（最初のTFT画面では75ms）に相当します。コントラスト比は35対1です（これまでのTFTディスプレイでは100対1、最新のTFTディスプレイでは1000対1）。通常、CSTNディスプレイを備えたヴィンテージのラップトップを識別するには、2つのコントロールが存在します。ディスプレイの通常の明るさに加えて、コントラストを調整できます。 380Eはこのルールに準拠していません。明るさはまったく調整できず、1つのスライダーでコントラストを制御します。これは重要なレギュレーターであり、「何も表示されていない」状態から「何かが読み取れるが困難な」状態にディスプレイを調整できます。正直なところ、ラップトップで作業する場合、色の再現も応答時間も問題になりません。しかし、視野角はひどいです。しかし、価格では、CSTNとTFTのラップトップは、通常のIPS画面を備えたラップトップとOLEDディスプレイを備えた高度なモデルが異なる点で異なりました。


視点で予算のラップトップ


90年代後半にラップトップが必要になったのはなぜですか？1998年1月の PC Magazineエディトリアルでは、雑誌の編集者であるMichael Millerが明確に対象読者を定義しています。インターネットへのアクセス、大量のデータへのアクセス、およびいつでも彼らと一緒に仕事をする能力を必要とする出張の多い専門家です。ついでに言えば、職場との間だけでラップトップを持ち歩く人のカテゴリがあります。モニターと配線を備えたデスクトップを設置するよりも、新入社員にノートパソコンを渡す方が簡単な状況はまだ形成されていません。コンピュータが当時のように、一般に警備員と秘書までのすべての従業員に配置されたとは思いません。予算モデルでさえ誰もが利用できるわけではありませんでした。1998年に、私は店の棚と映画以外でラップトップを見ました。


380Eには、出張中の6か月の従業員の主要な要素である内蔵モデムがありません。一般的に、これは私のコレクションで特定の問題が発生した唯一のデバイスです。光学ドライブもUSBもありません（1年後に表示されます）。大量のデータをどのようにアップロードできますか？フロッピーディスクに？ただし、PCMCIA拡張カードをインストールすることで、ラップトップをローカルネットワークに接続したり、USBフラッシュドライブを追加したりするのと同じ方法で、モデムを問題なく追加できます。このラップトップは32ビットPCMCIAカードを認識しませんが、適切な古いアダプターをいくつか在庫しています。しかし、一般的に、そのような意図的に制限されたデバイスがあるので、シリアルポートを介したモデム接続をシミュレートすることは興味深いでしょう。


Pentium II以降のデバイスは、どういうわけか現代性を引き付けるためにまだ試すことができますが、ここでは状況が異なります-なぜ、ポイントは何ですか？私はこの博物館の展示を理解することさえせず、インストールされたソフトウェアを研究し始めました。


97-99年のラップトップのBIOSは非常に制限されており、ここで設定できるのは起動シーケンスのみです。他のすべての設定（組み込みデバイスのオン/オフ、割り込みの配布）は、WindowsとDOSの両方のバージョンで利用可能なIBM独自のユーティリティで行われます。ウィンドウズは確かにもっと便利です。

Windows 95モダンアセスメント


90年代、ロシアのユーザーは（そしてそれだけでなく）Windowsに関して愛と憎しみのこのような複雑な組み合わせを形成しました。ほとんど誰もがWindowsを使用しており、OS / 2またはUnix / Linuxの形式の代替手段は非常にまれでした。同時に、Windowsは不安定性とグリッチの問題に関するジョークの日常的なトピックでした。 Windows 95とWindows 98の印象が記憶に残り、問題の多い場所やメリットの多い場所を特定するのが困難になりました。


Windows 95が画期的なオペレーティングシステムであったことは確かです。以前のバージョンのWindows 3.1は、経験のない現代のユーザーにとっては複雑になります。どこに何があり、どのように始まるのかは完全には明確ではありません。 Windows 95は、これまでに使用されたユーザーインターフェイスの概念を示しています。同じ名前のメニューのあるスタートボタン、コントロールパネル、時計のあるタスクバーと常駐プログラムのアイコンです。バギーで信頼できませんでしたか？ある意味、そうです。それは部分的には、テクノロジーの根本的な変化、仮想デバイスドライバーの登場、および長い名前のサポートによって引き起こされました。すべてのソフトウェアとデバイスドライバーの開発者がこれらのイノベーションに対応できるとは限りませんでした。新しい環境では正しく動作しない可能性のある古いプログラムが多数ありました。

おそらく、ロシア市場の詳細はWindowsを支持しませんでした：起源が不明なコンピューターにインストールされた海賊版のコピーは、何からでも組み立てられます。精神的には、DOSキーボードのテキストインターフェースからマウスシャープのWindowsに再構築することはすぐには不可能でした。一部の場所ではまったく機能しませんでしたが、多くはまだパネルファイルマネージャーを使用しています。私はいつもWindows 95をWindows 98の一種のベータ版と見なしていました。95回目のリリースから3年後にバグを修正し、不足している機能を追加して、ようやく正常になりました。これはそうですが、ここに別の視点があります。


元のバージョンのWindows 95は、そのままではTCP / IPネットワークをサポートしていませんでした。これは、Microsoftのリーダーシップが（現在の形で）インターネットの見通しの評価を真剣に誤計算した瞬間でした。私は自分のスタイルで、独自のマイクロソフト独自のインターネットをMicrosoft Networkの形で提出しようとしました。


Windows 98のリリースまでに、エラーは修正されましたが、方法：Webをプッシュする必要がないように見える場所にWebを配置しました。最も有名な例は、常に低下し続けるアクティブなデスクトップで、実際にはデスクトップの壁紙ではなくブラウザです。最終的に、このWindows 98は、95ではなくライブプレゼンテーションにクラッシュしました。


この観点では、Windows 95はベータ98ではなく、Microsoftの最新の純粋なオフラインオペレーティングシステムです。これは、すべての人に適したWindows 7と比較できます。また、タッチスクリーンでの最大のモバイル性とシャープネスに不当な賭けが行われたWindows 98と比較してみてください。もちろん、時間に合わせて調整されています。Windows95にはおなじみの機能が多くないため、定期的にフラストレーションが発生します。しかし、Windows 95専用に作成されたかなり強力なコンピュータであるThinkPad 380Eでは、このシステムが非常に安定して動作することを確認できます。

ミラクルチェスト


ThinkPad 380Eは、より高価な1998年モデルであるThinkPad 600と比較しても巨大に見えます。現代のラップトップは言うまでもありません（上の写真にはThinkpad X1 Carbonがあり、このコンテキストではラップトップのようにも見えません）。


ただし、サイズ（6センチの厚さ！）と重量は、完全に予算の画面に対して重大な欠点のようには見えません。ほぼ10年前に、Linuxをインストールして急流をダウンロードするように、もう少し高度なThinkPad 380EDを改造しました。予算のラップトップでは何もしません。オリジナルのソフトウェア、デスクトップの壁紙、独自のユーティリティを備え、歴史的に信頼できる「タイムカプセル」のままにしておきます。時間内に分解する必要がない限り、ハードドライブを取り外してどこかに画像を保存してください。ハードドライブを除いて、このラップトップの他のすべては非常に信頼性が高く、何年も動作するようです。


1997年、ThinkPad 380Eはクールなラップトップでしたが、より強力で、より薄く、よりファッショナブルなモデルを求めています。現在、高価なIBMモデルと比較すると、1つの重要な利点があります。経済的理由により、ブランド名のラバーコーティングは380シリーズのケースには適用されませんでしたが、最終的には完全に使用できなくなります。この安価なモデルの最初の所有者がアップグレードを行い、ラップトップをそれほど積極的に使用しなかった可能性があります。それは私のコレクションの中で最も優れたラップトップではないかもしれませんが、間違いなく最高の保存状態です。
