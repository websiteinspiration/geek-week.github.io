12 Data Engineering Online-Kurse


Laut Statista wird der Big-Data-Markt bis 2025 auf 175 Zetabyte wachsen, verglichen mit 41 im Jahr 2019 ( Grafik ). Um in diesem Bereich arbeiten zu können, müssen Sie wissen, wie Sie mit in der Cloud gespeicherten Big Data arbeiten. Cloud4Y hat eine Liste von 12 kostenpflichtigen und kostenlosen Data Engineering-Kursen zusammengestellt, die Ihr Wissen in diesem Bereich erweitern und ein guter Ausgangspunkt auf dem Weg zu Cloud-Zertifizierungen sein können.

Vorwort

Was ist ein Dateningenieur? Dies ist die Person, die für die Erstellung und Pflege der Datenarchitektur im Data Science-Projekt verantwortlich ist. Zu seinen Aufgaben kann gehören, einen unterbrechungsfreien Datenfluss zwischen dem Server und der Anwendung sicherzustellen, neue Software für das Datenmanagement zu integrieren, die zugrunde liegenden Datenprozesse zu verbessern und Datenpipelines zu erstellen.

Es gibt eine Vielzahl von Technologien und Tools, über die ein Dateningenieur verfügen muss, um mit Cloud Computing, Data Warehousing, ETL (Extraktion, Transformation, Laden) usw. arbeiten zu können. Darüber hinaus wächst die Anzahl der erforderlichen Fähigkeiten ständig, sodass ein Dateningenieur sein Gepäck regelmäßig auffüllen muss Wissen. Unsere Liste enthält Kurse für Anfänger und erfahrene Profis. Wählen Sie, was für Sie richtig ist.

1. Data Engineering Nanodegree-Zertifizierung ( Udacity )

Sie lernen, wie Sie Datenmodelle entwerfen, Data Warehouses und Data Lakes erstellen, Datenpipelines automatisieren und mit Dataset-Arrays arbeiten. Am Ende des Programms testen Sie Ihre neuen Fähigkeiten, indem Sie das Capstone-Projekt abschließen.

Dauer : 5 Monate, 5 Stunden pro Woche
Sprache : Englisch
Preis : $ 1695
Level : Anfänger

2. Werden Sie Data Engineer Certification ( Coursera )

Lernen Sie aus den Grundlagen. Sie können sich schrittweise anhand von Vorlesungen und praktischen Projekten entwickeln, um an Ihren Fähigkeiten zu arbeiten. Am Ende des Trainings sind Sie bereit, mit ML und Big Data zu arbeiten. Es wird empfohlen, dass Sie Python mindestens auf dem Mindestniveau kennen.

Dauer : 8 Monate, 10 Stunden pro Woche
Sprache : Englisch
Preis :?
Stufe : Anfänger

3. Werden Sie Dateningenieur: Beherrschen der Konzepte ( LinkedIn Learning )

Sie entwickeln Datenentwurfs- und DevOps-Kenntnisse, lernen, wie Sie Big Data-Anwendungen erstellen, Datenpipelines erstellen und Anwendungen in Echtzeit mithilfe der Hazelcast- und Hadoop- Datenbank verarbeiten .

Dauer: hängt von Ihnen ab
Sprache : Englisch
Preis : erster Monat - kostenlos
Stufe : Anfänger

4. Data Engineering-Kurse ( edX )

Hier finden Sie eine Reihe von Programmen, die Sie in das Data Engineering einführen und Ihnen die Entwicklung analytischer Lösungen beibringen. Die Kurse sind nach Schwierigkeitsgrad geordnet, sodass Sie einen nach Ihrem Erfahrungsgrad auswählen können. Während des Schulungsprozesses lernen Sie, wie Sie Spark, Hadoop, Azure verwenden und Unternehmensdaten verwalten.

Dauer : hängt von Ihnen ab
Sprache : Englisch
Preis : hängt vom gewählten Kurs ab
Stufe : Grundstufe, Mittelstufe, Fortgeschrittene

5. Data Engineer ( DataQuest )

Dieser Kurs ist eine Wahl wert, wenn Sie Erfahrung mit Python haben und Ihr Wissen vertiefen und eine Karriere als Datenverarbeitungsspezialist aufbauen möchten. Sie lernen, wie Sie Datenpipelines mit Python und Pandas erstellen und große Datenmengen nach dem Bereinigen, Konvertieren und Überprüfen in eine Postgres-Datenbank laden.

Dauer : hängt von Ihnen ab
Sprache : Englisch
Preis : hängt von der Form des Abonnements ab
Stufe : Anfänger, Mittelstufe

6. Data Engineering mit Google Cloud ( Coursera )

Dieser Kurs hilft Ihnen dabei, die Fähigkeiten zu erwerben, die Sie für eine Karriere in Big Data benötigen. Beispiel: Arbeiten mit BigQuery, Spark. Sie erhalten das Wissen, das Sie benötigen, um sich auf die branchenweit anerkannte Zertifizierung als Google Cloud Professional Data Engineer vorzubereiten.

Dauer : 4 Monate
Sprache : Englisch
Preis : kostenlos
Stufe : Anfänger, Mittelstufe

7. Data Engineering, Big Data auf der Google Cloud Platform ( Coursera )

Ein interessanter Kurs, der praktische Kenntnisse über Datenverarbeitungssysteme in GCP vermittelt. Während des Unterrichts lernen Sie, wie Sie Systeme entwerfen, bevor Sie mit dem Entwicklungsprozess beginnen. Darüber hinaus analysieren Sie sowohl strukturierte als auch unstrukturierte Daten, wenden eine automatische Skalierung an und wenden ML-Methoden an, um Informationen zu extrahieren.

Dauer : 3 Monate
Sprache : Englisch
Preis : solange kostenlos
Level : Anfänger, Mittelstufe

8. UC San Diego: Big Data-Spezialisierung ( Coursera )

Der Kurs basiert auf der Verwendung der Hadoop- und Spark-Infrastruktur und der Anwendung dieser Big-Data-Verarbeitungsmethoden im ML-Prozess. Sie lernen die Grundlagen der Verwendung von Hadoop mit MapReduce, Spark, Pig und Hive. Erfahren Sie, wie Sie Vorhersagemodelle erstellen und mithilfe von Diagrammanalysen Probleme modellieren. Beachten Sie, dass für diesen Kurs keine Programmiererfahrung erforderlich ist.

Dauer : 8 Monate, 10 Stunden pro Woche
Sprache : Englisch
Preis : kostenlos
Stufe : anfänglich

9. Big Data mit Apache Spark und Python zähmen ( Udemy )

Sie erfahren, wie Sie die Stream-Struktur und die Datenrahmen in Spark3 verwenden und wie Sie den Elastic MapReduce-Dienst von Amazon für die Arbeit mit Ihrem Cluster in Hadoop verwenden. Erfahren Sie, wie Sie Probleme bei der Analyse von Big Data identifizieren und wie GraphX-Bibliotheken mit der Netzwerkanalyse funktionieren und wie Sie MLlib verwenden können.

Dauer : hängt von Ihnen ab
Sprache : Englisch
Preis : von 800 Rubel bis 149,99 USD (wie viel Glück)
Stufe : Anfänger, Mittelstufe

10. PG-Programm in Big Data Engineering ( upGrad )

Dieser Kurs gibt Ihnen eine Vorstellung davon, wie Aadhaar funktioniert, wie Facebook den Feed personalisiert und wie Data Engineering verwendet werden kann. Hauptthemen sind Datenverarbeitung (einschließlich Echtzeitverarbeitung), MapReduce und Big Data-Analyse.

Dauer : 11 Monate
Sprache : Englisch
Preis : ca. 3000 USD
Stufe : Grundstufe

11. Beruf als Data Scientist ( Skillbox )

Sie lernen, wie man in Python programmiert und die Rahmenbedingungen für das Training neuronaler Netze Tensorflow und Keras erlernt. Beherrschen Sie die Datenbanken MongoDB, PostgreSQL und SQLite3 und lernen Sie, wie Sie mit den Bibliotheken Pandas, NumPy und Matpotlib arbeiten.

Dauer : 300 Stunden Training
Sprache : Russisch
Preis : Das erste halbe Jahr ist kostenlos, dann 3900 Rubel pro Monat.
Stufe : Anfänger

12. Data Engineer 7.0 ( New Professions Lab )

Hier finden Sie eingehende Studien zu Kafka, HDFS, ClickHouse, Spark, Airflow, Lambda-Architektur und Kappa-Architektur. Sie lernen, wie Sie Werkzeuge miteinander verbinden, Pipelines bilden und eine Basislösung erhalten. Das Lernen erfordert Mindestkenntnisse in Python 3.

Dauer : 21 Lektionen, 7 Wochen
Sprache : Russisch
Preis : 60.000 Rubel
Stufe : Mittelstufe
