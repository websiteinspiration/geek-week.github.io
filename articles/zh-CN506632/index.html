用GDAL Python语言制作的火星精美且详细的地质图



本周是根据USCS制作的地图，创造性地重新思考了火星的地质（或地质学？）地图。使用了与原始数据相同的地质数据，但增加了更多的地名和签名，重新设计了视觉样式，并简化了地图图例中的术语，以便于理解。

该项目最困难的部分之一是将符号翻译成非专业人员的语言。原始地图是为地质学家制作的，因此大多数词汇表都必须进行复习。例如，破火山口边缘值的新解释是“火山喷发后剩余的空岩浆腔室的边界”。在原始版本中，有一个“卵形结节，概述了一个或几个部分或完全闭合的空洞合并；与喷发和可能的爆发性喷发有关的火山倒塌。”

在许多情况下，翻译后的名称比原始名称更不准确或信息量更少，因此，它们还包括每种地质元素类型的原始缩写。如果您突然想以科学的术语了解每种类型的地质构造的更多信息，这些标签可能包含指向源数据的链接。



火星上一些值得注意的地质特征。 1：奥林匹斯山，太阳系中最大的火山。 2：水手谷（Mariner Valley），是一个深峡谷系统，全长超过4000公里。 3：海拉斯平原，太阳系中最大的可见撞击坑。 4：火星在地质上分为北部低地（淡绿色）和南部高地（棕色）。由坠落的小行星和彗星（霓虹黄）形成的撞击坑遍布整个星球。



以下单独的图层用于地图。 1-3：USGS数据集中的地质元素，地质接触和地质特征。 4-5：倾斜度，来自两个USGS来源。 6：IAU（国际天文学联盟）的命名法。 7-8：在Photoshop中开发的网格线和3D效果。

为该项目选择一个投影花费了很多时间。为了精确显示行星的哪个部分由每个地质构造组成，最终使用了Eckert IV的投影。。这种类型的地图投影会扭曲对象的轮廓，但会保留行星整个表面上人物的相对面积。这样的投影不适用于极地区域的可视化，因此，我们还在拐角处添加了四个地图插入项，以显示火星的每个半球（北部，南部，东部和西部）。

要比较地图的这些不同投影，可以使用天梭（Tissot）指标 -一组大小相同的圆，它们在地球上的不同位置绘制。所有地图投影都会扭曲空间，但是您会看到效果因投影而异。



1：等距。2：埃克特四世。3-5：以不同经度和纬度为中心的正射投影

(DEM)


地形模型是包含有关行星上各个点的高度信息的数据。该项目使用了从美国地质调查局获得的数据。GeoTIFF文件中的每个像素都是一个16位数字，用于描述该点的高度。

注意：许多程序无法正常读取此文件，因此在预览格式或在其他非专业程序中看起来很奇怪。在该项目的框架内，与他们的合作经历了GDAL（英语地理空间数据抽象库-地理空间数据抽象库）。

创建地图投影


如上所述，Eckert IV投影用于主地图，以保持面积不变，而正射投影则用于嵌入式极点地图。

更改DEM文件中的投影：源文件使用相等的投影，因此您需要在命令行中输入以下代码进行转换。该代码使用原始的intif文件并以eck4（Eckert IV）投影格式创建一个新的intif。

gdalwarp -t_srs "+proj=eck4" ./path_to_intif.tif ./path_to_outtif.tif

然后，通过使用平均值方法将每个像素的大小简单地减小到1500x1500米，可以降低DEM分辨率。这将减少处理时间，并且此时降低分辨率比以后更容易。

gdalwarp -tr 1500 1500 -r average ./path_to_intif.tif ./path_to_outtif.tif

水洗和坡度图


接下来，我们使用生成的文件为该行星的每个半球创建一个带有冲洗和坡度贴图的贴图。

洗涤后，地图显示来自假想光源的阴影；光线从上方落在地图上。这是虚构的，因为在现实世界中这不会发生-单个光源会在球的不同部位以不同角度产生阴影。GDAL的内置山体阴影功能将整个地图的光入射角设置为相同。对于此项目，将垂直放大倍数z设置为20。这会将每个高度值增加20倍，以使浮雕更具对比度并提供阴影。

gdaldem hillshade -z 20 ./path_to_intif.tif ./path_to_hillshade.tif

坡度图突出显示了地图中最陡峭的部分，并向地形图添加了更多信息（强调绝对高度而不是陡度）。

gdaldem slope ./path_to_intif.tif ./path_to_slope.tif



正射投影


除地图外，埃克特四世的投影中还绘制了四张火星半球的地图。Eckert IV投影不太适合显示北极和南极，因此这两个插图对于理解它们非常有用。要获得清洗和坡度图，请稍作更改以重复代码-用正交代替eck4，并为每张图指定经度和纬度的中心（北：+ lat_0 = 90 + lon_0 = 90，南：+ lat_0 = -90 + lon_0 = 90，东： + lat_0 = 0 + lon_0 = 90，西：+ lat_0 = 0 + lon_0 = 270）。

gdalwarp -t_srs "+proj=ortho +lat_0=90 +lon_0=0" ./path_to_intif.tif ./path_to_outtif.tif
gdalwarp -tr 1500 1500 -r average ./path_to_intif.tif ./path_to_outtif.tif
gdaldem hillshade -z 20 ./path_to_intif.tif ./path_to_hillshade.tif
gdaldem slope ./path_to_intif.tif ./path_to_slope.tif



图例卡


国际天文学联合会负责命名外星物体。您只需简单地以CSV格式下载文件，即可直接从其站点下载每个行星的所有对象。为此，请使用“ 高级搜索”功能下载您星球的所有特征类型-目标（火星），但只有IAU批准状态为MAC批准。在要包括的列部分中，选择特征ID，特征名称，清洁特征名称，目标，直径，中心纬度/经度，特征类型和特征类型代码。如果要向项目添加有关对象的其他信息（例如，以纪念该对象的名称），还可以启用Origin（来源）。

地质构造与元素


地质图显示了各种类型的岩石和其他特征，例如断层线和河床。USGS拥有一幅精美的火星地质图，我们将其用作资料来源。要使用此数据，请从USGS地质图下载数据库档案（790 MB）。关于Python地图设计的下一部分将说明如何访问和显示此数据库中的每种数据类型。

Python地图设计


我们创建了六个具有地质元素，结构，特征，两种类型的文本标签和网格的图形（图）。通常需要分离数据进行处理，以便在必要时轻松地从Photoshop或Illustrator应用效果。 Matplotlib使用gridspec来自定义所有元素，以便每个子图在装饰框架中占据特定位置。



地质元素-这些是构成地球表面的不同类型的浮雕。通常用不同的颜色表示不同的岩石元素。 USGS数据集使用每个项目类型的2-3个字母代码标记每个品种。通过在Mars_geologic_units.csv中编译一个代码-颜色对表，为每个字母代码分配颜色。我们在构造cartopy和matplotlib中的每个元素时都引用此文件。将图形参数保存在单独的文件中有助于使用各种配色方案，并使设计与代码分离。



地质接触-这些是地质构造之间的界限。一些地质边界是近似的或隐藏在尘埃中。对于地质构造，我们使用Mars_geologic_boundaries.csv配置文件以不同的颜色显示每种类型的地质接触。在最终地图上，我们以虚线形式配置一些地质接触的显示，只需在Illustrator中打开PDF并选择相同颜色的所有对象即可（选择->相同->描边颜色）。



地质特征-这些是行星表面上的其他可见线，例如运河，火山口边缘或山脊。数据集中有13种不同的类型。为了以特殊的样式显示它们，每种颜色都以不同的颜色显示，然后在Illustrator中设置线条的样式。这允许使用特殊效果（例如渐变尾部）来显示流出通道和熔岩流的运动方向。



传说


标签使用了两个数据集。首先是国际天文学联合会的官员。标签尺寸是根据对象的尺寸创建的，尽管您仍然需要在Illustarator中最终确定尺寸。



第二组标签为每个主要地质元素添加了缩写。由于某些颜色会因阴影而变化，因此这使使用地图图例更加容易。必须创造性地处理代码，即在地质形式的几何中心绘制每个缩写，这并不是在所有情况下都理想。但是对于这个项目，这是正常的，因为按照计划，然后按照相同的方式，这些签名将手动移动以避免将来的签名重叠。



降落地点：为了显示火星探测器和其他航天器的着陆点，使用了来自NASA的增强型数字图像作为背景。为了显示该星球的所有4个半球，我必须更改投影-landing_sites.ipynb。

从Matplotlib保存结果


通常，结果以pdf格式保存，以在Illustrator中对文本和表单进行最终编辑。以下是几个便于后续编辑的标准命令：

import matplotlib
import matplotlib.pyplot as plt
import matplotlib.backends.backend_pdf as pdf

#     ,     :
matplotlib.rcParams['pdf.fonttype'] = 42

#     
matplotlib.rcParams['image.composite_image'] = False

#      subplot
ax.axis('off')

#  padding  margins 
plt.margins(0,0)
plt.subplots_adjust(top=1, bottom=0, right=1, left=0, hspace=0, wspace=0)
plt.gca().xaxis.set_major_locator(plt.NullLocator())
plt.gca().yaxis.set_major_locator(plt.NullLocator())

#  pdf
pp = pdf.PdfPages('./savename.pdf', keep_empty=False)
pp.savefig(fig)
pp.close()

#

保存PDF之后，您需要对其进行编辑，以便可以单独编辑每个对象。在Illustrator中，选择文件中的所有内容，然后转到“对象”->“剪切蒙版”->“释放”。在这一点上，如果您之前已经做过，还值得删除背景和边框。
