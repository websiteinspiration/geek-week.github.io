インフィニオンは、暗号チップにクロス署名証明書を実装しています


第4世代スマートプラント（Industry 4.0）は自律的に動作します。コンベアロボットは相互に通信し、不良品と廃棄物の割合を減らし

ます。世界最大のチップメーカーの1つであるインフィニオンは、グローバル暗号化機関であるGlobalSignと、産業用暗号チップ Optiga TPM SLM 9670に対して独自に発行する証明書に相互署名する契約を結びました。現在、これらの署名はGlobalSignルート証明機関まで検証され、管理が大幅に簡素化されています。

Optiga TPM SLM 9670 は、TPM 2.0標準を満たす最初の産業用暗号チップと見なされています。これはOptiga TPMファミリーの最新モデルです。


Optiga TPM SLM 9670モジュール

このチップは、信頼性、広い温度範囲での安定性、および長期保証が重要な産業機器への取り付けを主な目的として設計されています。これらは、ルーター、ファイアウォール、ワイヤレスアクセスポイント、スイッチなどの産業用コンピューター、サーバー、PLCコントローラー、産業用ネットワークデバイスおよび機器です。

TPM 2.0は、Industry 4.0仕様およびスマートファクトリの一部です。仕様には次の要件が含まれます。

  • 強力なデジタル識別子とデバイス認証
  • データのプライバシーと知的財産の保護のための安全な通信
  • ソフトウェアの更新を含む、デバイスとソフトウェアの完全性の保護


第4次産業革命（Industry 4.0）は、サイバーフィジカルシステムと自律型プラントの導入を規定してい

ます。SLM9670モジュールは、2019年後半に発売されました。それは現代の産業機器が必要とするすべての機能をサポートします：

  • 重要なデータと秘密鍵の安全な保管
  • 物理的および論理的攻撃に対する高度な防御メカニズム
  • 暗号化アルゴリズムのサポートRSA-1028、RSA-2048、ECC NIST P256、ECC BN256、SHA-1、SHA-256
  • 動作温度範囲--40°C〜105°C
  • 耐用年数20年
  • 産業適格性認定JEDEC JESD47
  • 独立したセキュリティ評価と認定




Optiga TPM SLM 9670機能図

GlobalSignとの合意により、ローカルのInfineon証明機関とグローバルに認められたGlobalSign証明機関による相互署名が提供され、Infineonが独自に発行し、各Optiga TPM SLM 9670チップで自動的にフラッシュする証明書の全体的な信頼性が向上します。これらの証明書は、ルートCA GlobalSign。

検証可能な証明書を使用すると、各Infineon TPMは、ライフサイクル全体を通じていつでも IoT Identity PlatformのGlobalSign IoT Edge Enroll Registration Centerに接続できます。



とりわけ、相互署名証明書は、Microsoft Azure IoT HubやIoT Hub Device Provisioning Serviceなどのクラウドベースのデバイス管理サービスへのデバイス登録を簡素化します。デバイス識別子の統合という複雑な問題を軽減し、文字通りチップからクラウドまでIoTデバイスを保護するための実証済みのパスを提供します。

「ユニークなデバイス識別子は、クラウドへの安全な接続のために不可欠であり、」言ったユルゲン反乱、インフィニオンテクノロジーズの組み込みシステムのセキュリティの副社長兼ゼネラルマネージャー。「当社の新しいOptiga TPM統合キットを使用すると、デバイスをMicrosoft Azure IoTに1時間未満で安全に接続できます。」
