Tambahkan logika untuk memproses foto, yang akan diimplementasikan sebagai Fungsi Azure . Fungsi Azure adalah model pemrograman. Memungkinkan Anda untuk menulis fungsi yang diaktifkan pada peristiwa di cloud.

Fungsi Anatomi Azure



Fungsi Azure memiliki dua bagian penting. Yang pertama adalah fungsi Python __init__.py



. Fungsi ini menerima argumen dan mengembalikan respons. 



def main(req:func.HttpRequest) -> func.HttpResponse:
    logging.info('Execution begins…')

    name = req.params.get('name')
    
    if name:
        return func.HttpResponse(f"Hello {name}!")
    else:
        return func.HttpResponse(
            "Need name query parameter",
            status_code=400)


Yang kedua adalah file function.json



yang menggambarkan data apa yang sedang diinput dan di-output. Ini disebut integrasi . 



{
    "scriptFile": "__init__.py",
    "bindings": [
        {
            "authLevel": "function",
            "type": "httpTrigger",
            "direction": "in",
            "name": "req",
            "methods": [
                "get",
                "post"
            ]},
        {
            "type": "http",
            "direction": "out",
            "name": "$return"

Permintaan diajukan ke input, di mana metode GET dan POST digunakan, dan fungsinya dipicu oleh pemicu HTTP. Input dan output mungkin berbeda. Misalnya, sebagai output, file dalam penyimpanan dapat digunakan sehingga fungsi tersebut bekerja tanpa terlihat di dalam cloud, melakukan tindakan tertentu. Menggunakan Fungsi Azure dimungkinkan untuk membangun seluruh arsitektur layanan microser dari aplikasi kita.
