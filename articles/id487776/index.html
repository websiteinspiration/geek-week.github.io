Hari Pelatihan Microsoft Azure: Migrasi Infrastruktur Server (pendaftaran selesai)



Pada 13 Februari, kami mengundang Anda untuk berpartisipasi dalam seminar teknis mendalam yang berfokus pada persiapan infrastruktur server lokal untuk implementasi solusi hybrid dengan transfer skenario spesifik ke cloud. Sebagai bagian dari acara tersebut, kami akan menggunakan contoh praktis untuk mempertimbangkan migrasi perusahaan besar tempat Windows Server 2008 R2 digunakan baik pada server fisik dan dalam lingkungan virtual dengan transfer domain dan infrastruktur jaringan, file server, serta SQL Server.

Kami mengundang para profesional TI, administrator sistem dan arsitek yang berencana untuk bermigrasi ke Windows Server 2019 ke seminar.

Dalam kerangka seminar ini kami akan mempertimbangkan topik-topik berikut: 

  1. Migrasi layanan infrastruktur dasar ke Windows Server 2019,
  2. Integrasi infrastruktur lokal dengan layanan cloud,
  3. Migrasi mesin virtual ke platform Azure.
