Hack the Crisisオンラインハッカソンは、30か国から100,000人以上の参加者を集め、世界中の人々の生活の社会的および経済的要素を改善するソリューションを作成しました。

ロシアでは、＃HacktheCrisisRussiaハッカソンが、同じ名前のプロフェッショナルコミュニティである Mail.ru Cloud Solutions（MCS）および Ontiko IT会議の主催者の支援を得て開催されました。

猫の下で、オンラインピザハッカソンの主な結果をまとめ、ロボット弁護士、ガベージコレクションアプリケーションなどがコロナウイルスとの戦いにどのように役立つかを説明します。

トレーニング


ハッカソンの主な目的は、さまざまなIT分野の熱心なファンを集めて共通の目標を達成することです。つまり、コロナウイルスのパンデミックによって引き起こされた危機に立ち向かい、開発者が社会的に重要なプロジェクトを実装できるようにすべての参加者にリソースを提供することです。

ハッカソンの前の週の間、チームはメンターの助けを借りました。彼らは講義や相談を行い、プロジェクトに関する専門家の意見を共有し、アイデアの詳細な調査を助けました。だから、例えば、アンナタラセンコ、の創設者兼最高経営責任者（CEO）7ビットは、ゼロからMVPを作る方法を共有しました。メンターと慎重にアイデアを練った後、5月16日、チームはプロジェクトの開発を開始しました。そして、デリバリークラブが本当に重要なタスクから開発者の邪魔にならないように 参加者の家へのハッカソン用の無料の伝統的なピザの配達を組織しました。

チームプロジェクト


#HacktheCrisisRussiaの結果に基づいて、ロシア人の生活の質と社会的サポートの向上に焦点を当てた最も有望な6つの開発が選択されました。

1. Rosvosrosチーム
キャプテン：Sergey Valigursky

Rosvosrosは、AIに基づいて作成された情報および分析システムであり、既成の電子ドキュメント（動的なフォームを考える）の形式で個人および法人に対象となる法的サービスを提供します。裁判官、州および地方自治体の従業員、および商業組織の格付けシステムを含む、結果のその後の評価を伴う、適切な裁判所、州または地方自治体の当局および商業組織へのインターネットを介した請求、異議申し立て、罰金など、およびそれらの指示。
ウェブサイト：http：//rosvosros.rf

2. P2P（人から人へ）
チームキャプテン：Alexander Berdennikov

P2Pは、意識的なゴミの収集と分別のためのアグリゲーターです。アプリケーションは、ゴミの場所からリサイクルまたは廃棄のポイントまでの輸送チェーンを構築します。プロセスのすべての参加者は、作業に対して金銭的報酬を受け取ります（kgあたりの生ゴミ処理プラントからの支払い）。ハッカソンでは、glideappsを使用して模擬アプリケーションが作成され、2つの材料（プラスチック、段ボール）の例を使用してモデルの経済的な計算が行われました。

3. CloseContactWorksチーム
キャプテン：Nikolay Ryabinin

アニーユアコンパニオンは、大企業の従業員とリモートで交流するためのチャットボットです。 Annieは、さまざまなレベルの実装で使用できます。分析データを収集するためのチャットボット。従業員の利益と新しい同僚の適応に取り組みます。従業員を興味ごとにマッチングするサービスであるAI教育モデルの開発。

4. CoNetResチーム
キャプテン：Arina Matveeva

CoNetResは、オープンコミュニティのデザインパターンであるワンストップツールキットであり、地域社会で利用可能なリソースとニーズを実証し、関連付けるのに役立ちます。その結果、危機的状況に迅速に適応し、ボトムアップソリューションを見つける能力。 CoNetResは、ソフトウェアに基づいて、ソーシャルサイバネティックスを使用してデータを収集および視覚化するためのWebアプリケーションを作成します。 CoNetResは、機密性、ローテクソリューション、運用の可用性に重点を置いています。

5. Dabba
チームキャプテン：Rudenko Sergey

Dabbaは、中小企業向けのデリバリーデザイナーです。Dabbaは、短時間でビジネスを支援し、最小限の投資で配信形式に移行します。これは、オンラインストアフロント、注文および支払い処理システム、マーケティングおよび宅配便サービスなどの組織にサービスデザイナーを提供することにより、パンデミックで非常に人気を博しています。

6. EMOScannerチーム
キャプテン：Alexander Berdennikov

EMOScannerは、ビデオ録画によって人の感情的な状態を判断するためのサービスです。EMOScannerは、候補者の感情的な要素を評価することにより、人事部が大量の欠員のロボットインタビューを実施するのに役立ちます。このサービスは、目、手、姿勢、表情などのパラメータの動きを分析します。

#HacktheCrisisRussiaの結果


#HacktheCrisisRussiaは、専門家、プログラマー、マネージャー、さまざまな専門分野の人々の無関心と協力したいという願望の中心となっています。新しい形式にもかかわらず、ハッカソンの基本的な価値を維持することができました：スピード、コミュニケーションの雰囲気、専門知識、そして最も重要なのは、プロジェクトの途中でのプロジェクトです。

最終段階のすべての参加者は、さまざまなIT分野の専門家によるメンタリングサポート、プロジェクトを実装するためのMail.ru Cloud Solutionsクラウドプラットフォームの能力、およびOntiko オンライン会議RIT ++のチケットを受け取りました。
