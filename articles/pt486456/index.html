Censura no código-fonte do .NET Framework

Na noite de sexta-feira, olhando a fonte da Fonte de Referência , acidentalmente me deparei com um comentário suspeito:



Algumas palavras foram substituídas por um traço 4. Bem, às vezes os desenvolvedores são realmente muito expressivos e não têm vergonha de expressões duras ... Mas com todo o desejo, não consegui me lembrar de uma única maldição que se encaixaria aqui em significado. De onde veio?

Felizmente, o Github permite pesquisar por código de repositório, e você pode direcionar esses quatro traços para a pesquisa:

github.com/microsoft/referencesource/search?q=----&unscoped_q=---- Ignoramos

correspondências aleatórias com formatação ASCII e tropeçamos para vários outros casos em que tudo fica claro a partir do contexto: A



palavra tabu é "raça". Todas as referências são feitas no contexto de uma " condição de raça ", isto é, uma condição de raça para recursos, não a raça humana. E, no entanto, eles são substituídos.

Além disso!





Alguém de todo o coração odeia exceções? Não parece. Acho que a palavra tabu aqui é armadilha. Existe uma expressão estável - "interceptar uma exceção", o que significa capturar uma exceção para que ela não apareça. Mas essa palavra tem outravalor de gíria.

De qualquer forma. Estes são todos os comentários, você nunca sabe o que está escrito neles. Isso não afetará a qualidade do código? ..



Aqui a censura recorta algumas palavras de constantes de texto que são usadas para identificar localidades. Suspeito que no primeiro caso tenha a palavra "Simples" e no segundo - "Cultura", mas o que é sedicioso neles?

A classe EnglishPluralizationService , usada no EF6 para converter automaticamente palavras do singular para o plural e vice-versa, foi especialmente afetada . Uma quantidade razoável de palavras do vocabulário tem sido corte lá:





A maioria deles eu não posso adivinhar, mas t----zium- isso é t estupro zium (trapézio), e ch---is- este é ch ass é (corpo).

Felizmente, o problema é observado apenas nas fontes postadas neste repositório. Se você descompilar esta assembléia, não haverá censura: a



discriminação é realmente um problema sério na sociedade moderna. Mas é muito triste quando, sob o pretexto de combatê-lo, eles divulgam isso ...
