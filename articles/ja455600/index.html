3DEXPERIENCEプラットフォームが未来の公共交通機関の構築を支援



工業生産、都市化、および地球上の困難な環境状況の絶えず増加する成長率は、貨物および旅客輸送を編成する際の新しいアプローチを必要とします。既存の輸送の問題は、従来の方法では解決できません。

ベラルーシのエンジニアであり発明者でもあるアナトリーユニツキーは、メガロポリスとその郊外に住む人々に、通常のライフスタイルを損なうことなく、環境への影響を最小限に抑えながら、モビリティを高めるまったく新しい選択肢を提供しました。

「私は若い頃、宇宙旅行のアイデアを賞賛しましたが、私たちは最初に地球上にインフラストラクチャを構築し、輸送問題を解決する必要があることにすぐに気付きました」とUnitskyは言います。 -世界が高速道路に囲まれることを夢見ていたヘンリーフォードとは異なり、私はすべての大陸に長距離、郊外、長距離輸送のための安全で経済的で環境に優しい高架道路の道路を設置してほしいと思います。そのため、2014年にSkyWay Technologies Co.を設立しました。」



SkyWay輸送システムの開発プロセスでは、さまざまなプロファイルの多くの専門家が関与し、同時に最終製品の作成に参加します。旧SkyWay Technologies Co.さまざまなシステムやコンポーネントの設計、構築、製造に、関連のないさまざまなソフトウェア製品を使用していました。データをすばやく交換することはほとんど不可能でした。

すべてのアクティビティを単一の環境で実行するために、スカイウェイはダッソーシステムズ3Dエクスペリエンスプラットフォームに切り替えました。これは、複雑な構造やデバイスの設計に関連するすべての困難に対処するのに役立ちます。

今日、200人を超える会社の従業員がこのソフトウェアプラットフォームに取り組んでおり、設計、油圧、空気圧、シャーシ、空調、支持構造、電気機器、電子機器などのさまざまなコンポーネントを担当しています。

多くの情報フローを1つのシステムに埋め込んで、通常は互いに通信しないさまざまな専門家を統合する必要があります。 SkyWay Technologies Co.でのコラボレーションを改善することにより十分な情報に基づいた意思決定を行うようになりました。

SkyWayは、CATIAシステムエンジニアリングを使用して、輸送システムのさまざまなコンポーネントの相互作用における競合とエラーを解決します。単一のプラットフォームへの移行により、エンジニアは相手のアクションを確認し、コンポーネントのドッキングを監視し、既存の制限を考慮して独自のタスクをすばやく実行できます。



「単一の環境での輸送システム、車両、高速道路インフラストラクチャ、制御システム全体のデジタルレイアウトは非常に大きなメリットです」とSkyWay Technologies Co.の組織開発担当ディレクターであるAlexey Voilenkoは述べています。

-技術設計と建設関連のパラメータの寸法許容差をプロジェクトに統合することはそれほど簡単ではありません。これが、統合輸送システムの生産と構築を本当に難しい課題にしているものです。3D EXPERIENCEを使用すると、すべての専門家がさまざまな技術標準と概念をシームレスに組み合わせることができます。」

制作への適切なアプローチを決定する上で大きな役割を果たすのは、デジタルモデリングです。3DプラットフォームのシミュレーションモードEXPERIENCEスペシャリストは真剣な計算を行い、DELMIAモジュールの助けを借りて生産オペレーションを管理します。物理的な実施前の技術プロセスは、事実上テストされています。これは、生産への最適なアプローチを事前に選択するのに役立ちます。プロジェクトに変更が加えられた場合、3D EXPERIENCEプラットフォームでは設計と生産データが密接に関連しているため、生産データは自動的に更新されます。その結果、生産効率が大幅に向上します。

「数十億トンの燃料が車の中で燃やされ、それが環境問題を引き起こします：オゾン層の破壊、温室効果」とUnitskyは言います。 -これは、人々の安全、人口増加、自家用車の数の増加などの要因と相まって、上記のすべての欠点がない、輸送の形で新​​しいパラダイムを開発するように促しました。世界の主要な首都では、私たちのストリングロードの建設により、人々が旅行に費やす時間を3分の1にすることができます。私たちは完璧な輸送手段を作成し、すでにマリナゴルカ（ミンスクの南東約63 km）にあるEcoTechnoParkでそれを示しているので、プロジェクトの成功を信じています。最も厳しい要件を満たす4つのテストサイトが構築されました。現在までに、4つの車両モデルがベラルーシで認定されています。
