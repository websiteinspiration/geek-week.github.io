Tinkov API、投資。最初のステップ

ほぼ最初の日から私はTinkov。Investmentsのクライアントになりました。

そしてその瞬間から、漠然とした疑問が私を苦しめる-個人のアカウントは客観的な現実を反映していますか？

事実、私はドル建ての証券を購入していますが、LCではすべての資産の価格はドルで表示され、ポートフォリオの合計値はルーブルで表示されます。

そして、私はドルが成長したのか、私がそのような成功した投資家であるのか分かりませんか？

しかし、手数料、税金、その他の配当についてはどうでしょうか。

株式会計と同様に、すべての取引をFIFOに書き込んで、受け取りたい配当金を上に置き、税金を差し引きます。

そして、私が理解している結果が表示されます。

TinkovにはAPIがあり、トレーディングロボット（私はこれにはまったく興味がありません）を作成したり、彼のポートフォリオやオペレーションに関するデータをアップロードしたりできることがわかりました。

このAPIには公式の説明がありますが、すべてが私には明らかではなかったので、理解する必要がありました。
これらの対決​​の結果はあなたの注意に現れます。

トークンを取得してライブラリをインストールする


始める前に、ライブラリをインストールしてトークンを取得する必要があります。

ライブラリのインストール：

pip install -i https://test.pypi.org/simple/ --extra-index-url=https://pypi.org/simple/ tinkoff-invest-openapi-client

私は公式のトークン受け取り指示を引用します：

  1. tinkoff.ruでアカウントにログインします。
  2. 投資セクションに移動します
  3. 設定に移動
  4. 「コードによるトランザクションの確認」機能を無効にする必要があります
  5. エクスチェンジとサンドボックスのOpenApiトークンを発行します。おそらく、システムは再度ログインするように要求します。心配しないでください。これは、ロボットを取引プラットフォームに接続するために必要です。
  6. トークンをコピーして保存します。トークンは一度だけ表示され、後で表示することはできませんが、発行できるトークンの数に制限はありません。

この記事の執筆時点では、トークンはページwww.tinkoff.ru/invest/settings、つまりページの下部にあるボタンで発行されていました。



サンドボックスのトークンでエラーが発生したため、戦闘バージョンの実験を開始しました。私があなたに望んでいること（注意：何かを購入したり、誤って何かを売ったりしないでください）。

from openapi_client import openapi

このデータから、人間が読み取れるアセットの名前を見つけることができます。

このリクエストでは、これは必要ありませんが（名前フィールドを参照）、他の場合には便利です。

操作のリストを取得する


しかし、最も興味深いのは、私の操作のリストを取得することです。次のアクションは操作に該当します（私の場合）：

  • PayIn-仲介口座の補充
  • PayOut-お金を引き出す
  • BuyCard-カードから購入
  • 販売-販売
  • BrokerCommission-ブローカー委員会
  • 配当-配当支払い
  • 税金-税金
  • TaxDividend-配当税
  • ServiceCommission-サービス料金

私達は分野に興味があります：

  • 日付-取引日
  • figi-資産コード
  • operation_type-操作のタイプ
  • 支払い-トランザクションの金額。税金または手数料については、示されるのは彼女です。なしの価格
  • 価格-1枚の紙の価格
  • 数量-証券の計画額
  • 取引-実際の取引

質問がすぐに浮上しました-価格と数量があるのになぜ私たちはいくつかの取引が必要なのですか？

すべてがそれほど単純ではありません（計画と事実）


私が理解しているように、数量は購入したい論文の数を示しています。そして、実際に購入されるのは、取引[i]量です。

それら。実際の取引に移りたい場合は、取引にあるものを整理する必要があります。

場合によっては、なしがあります-たとえば、税金や預金/資金の引き出し。

実数を取得するには、トランザクションを見て、トランザクションを交換する必要があります。

for op in ops.payload.operations: #  
    print(op.figi) # figi    
    print(op.operation_type)   #    
    if op.trades == None:      #    
        print('price:', op.price)       #     
        print('payment:', op.payment)   #  
        print('quantity:', op.quantity) #   
    else:     
        for t in op.trades:                   #     -   
            print('price:', t.price)          #     
            print('quantity:', t.quantity)
    print('--------------')
