すべての機会のための10の反応コンポーネント

Reactのさまざまな開発タスクを解決するには、コンポーネントライブラリが提供するユニバーサルソリューションではなく、高度に専門化された小さなツールを使用する方が良いと思います。私は自分でコンポーネントを選択するのが好きです。長いドキュメントを読むのは好きではありません。また、無意味なライブラリの更新を避けたいと思っています。その後、プロジェクトの何かが頻繁に壊れます。



特定のフロントエンドライブラリは、コンポーネントの統一された設計を開発者に提供することができます。これは非常に重要です。しかし、そのようなライブラリを使用する予定のプロジェクトには、独自の設計システムまたは独自のUIライブラリが既にあることが判明する場合があります。この状況では、開発者は新しいライブラリを探すのではなく、再利用と作業の高速化に適したカスタムコンポーネントのみを探します。このような場合、ライブラリ全体ではなく、個別のコンポーネントを検索する方が理にかなっています。同時に、開発者が本当に必要としているもの、絶対に必要ではない多くのものが含まれています。

本日翻訳する資料は、10個のReactコンポーネントのストーリーに特化しています。

1. React Awesome Slider-「カルーセル」



React Awesome Slider コンポーネントを使用すると、スムーズで美しく、高度にカスタマイズ可能なトランジションを作成できます。これは、すべての主要なブラウザがサポートする非常に小さな（7 Kb）コンポーネントです。こちらが React Awesome Sliderのデモページです。

2. React Data Grid-テーブル



React Data Gridは、Excelワークシートのようなものをページに表示できる、強力で高速で使いやすいコンポーネントです。彼はキーボードコマンドの処理方法、データのコピーと貼り付け方法を知っており、通常、スプレッドシートと同じように動作します。このコンポーネントは、開発者にシンプルなAPIを提供します。これは非常に迅速に処理できます。その使用例を参照してくださいここに。

3. CogoToast-通知



そのようなコンポーネントがいくつ存在するかを考えると、通知を実装するコンポーネントを見つけることは簡単な作業ではありません。この方向付けには多くの優れたコンポーネントがありますが、ここでは私のお気に入り、簡単でわかりやすいCogoToastについてお話ししたいと思います。コンポーネントパッケージのサイズは控えめ（4 Kb）で、シンプルなAPIがあり、そのすっきりした外観は多くのプロジェクトに適しています。これは、このコンポーネントの使用例が記載された GogoToastページです。

4. Google Maps React-マップ



Google Maps Reactは、Googleマップ上で他のReactコンポーネントの（座標を使用した）配置を高速化および促進するかなり人気のあるコンポーネントです。

5. Reactツールチップ-ツールチップ



React Tooltipは、多くの設定をサポートするツールチップ実装コンポーネントです。それは、あらゆるインターフェース設計に適合します。このコンポーネントを使用すると、ツールチップを使用して、必要なあらゆることを実行できます。たとえば、カスタムイベント、遅延、ツールチップコンテンツのインタラクティブな更新などをサポートしています。

6. React Block UI-画面の読み込みとブロック



React Block UIは、ユーザーが何かをロードしているときにページインターフェースを操作できないようにするための使いやすいコンポーネントです。

7. React Credit Cards-支払いデータを入力するためのフォーム



私はこのレビューに「支払いデータを入力するための最良のコンポーネント」のようなものを含めるつもりはありませんでしたが、React Credit Cardsは、私が話さざるを得ない魅力的なコンポーネントにすぎません。それはカスタマイズ可能で、それを使った作業はシンプルで快適です。特に、入力されたカード番号がどの会社に対応するかを自動的に判別し、対応するロゴが付いたカードの画像を表示します。

8. React Markdown-Markdownマークアップのレンダリング



React Markdownは、Markdownマークアップ（HTMLを含む）をレンダリングするための高速で便利なソリューションです。このレビューの他のコンポーネントと同様に、彼は1つの問題しか解決できませんが、彼の仕事はとても上手です。

9. Coloreact-色を選択するためのツール



多くのUIライブラリは、色を選択するための独自のツールを開発者に提供しますが、Coloreactは使いやすさとカスタマイズの完璧なバランスを兼ね備えています。開発者は、独自のメカニズムを研究するために多くの時間を必要とせずに、自分の外観と動作をカスタマイズできます。

10. React Big Calendar-カレンダー



「プロフェッショナル」カレンダーにあるすべての機能を備えたカレンダーコンポーネントが必要な場合-これは、React Big Calendarを確認する必要があることを意味します。開発者にスケジューラ機能、ローカリゼーション、日付と時刻のフォーマットなど、多くの機能と設定を提供します。
