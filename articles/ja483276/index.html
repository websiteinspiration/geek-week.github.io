Python 3.8の新機能とPython 2へのアップグレード

2020年が来ました。つまり、Python 2はサポートされなくなりました。正確には、主要な開発者はすでにブランチへの対応を停止しており、2020年4月にPyCon USに特化したリリース2.7.18のリリースにより、Python 2に関連するすべてのアクティビティが完全に停止します。


一方、Python 3.8の最近のリリースが行われ、言語に少し構文上の砂糖が追加されました。ただし、Python 3.9 はまだ間もなく予定されておらず、言語に興味深いものは何も追加されていないようです。


したがって、突然Python 2を放棄しなかった場合、それを引っ張るだけの意味はありません。2番目のバージョンのサポートはすでに停止しており、3.8への即時の移行により、この言語を最新の状態で長期間使用できます。


さて、決定をより簡単にするために、以下はPython 3.8の主な革新の概要であり、すべてのPythonistに役立ちます。


1.割り当て式


セイウチ演算子とも呼ばれます。別の式内の変数に値を割り当てることができる新しい構文です。これはおそらく最も有名で、バージョン3.8のイノベーションの中で議論されています。


a = 6

#    b  a ** 2
#  ,  b > 0
if (b := a ** 2) > 0: 
    print(f' {a}  {b}.') #  6  36.

演算子を導入する目的は、コードの可読性を高めることであるため、（他のすべての演算子と同様に）コードを使用する必要はなく、適切な場合にのみ必要です。

6. finally + continue


以前は、この機能の実装が難しいため、finally内でcontinueステートメントを使用できませんでした。今できます。 


for i in range(2):
    try:
        print(i)
    finally:
        print('.')
        continue
        print('    .')
