VS Code的10个扩展，没有它们，我无法编程

VS Code是我最喜欢的代码编辑器。这是现有编辑器中最受欢迎的编辑器，其可能性几乎可以无限扩展。而且，令人惊讶的是，它是由Microsoft开发的。我相信其他任何编辑器和其他IDE都不能给开发人员至少VS Code所能提供的一半。VS Code的优势之一是其扩展系统。它允许您在所有情况下按字面意义创建扩展。我想告诉你我对VS Code的十大扩展。



1.美化



要安装扩展，请使用VS Code中的键盘快捷键，在CTRL+P打开的字段中输入以下命令，然后单击Enter。

ext install HookyQR.beautify

在美化扩展可以让你以格式HTML，JS，CSS，JSON和青菜代码。格式选项可以控制。此扩展是标准js-beautify系统的附加组件，使开发人员可以完全按照自己的需要设置代码格式。

2.更好的评论



ext install aaron-bond.better-comments

在更好的评论扩展可以扩展与评论工作的准备。特别是，它允许您将注释分为几类。这些可以是通知，请求，任务列表，注释。


使用更好的评论

3.书签



ext install alefragnani.Bookmarks

书签 扩展是程序员真正的生命线。特别是对于那些在数千行代码中丢失的代码。此扩展允许您将书签附加到代码行。如果代码行中有书签，则可以使用书签列表轻松转到该行。


代码中的书签

4.支架对着色器2



ext install CoenraadS.bracket-pair-colorizer-2

Bracket Pair Colorizer 2 扩展名使您可以为支架着色，从而使查找成对的打开和闭合支架更加容易。在必须使用嵌套软件结构的情况下，这非常有用。


成对的括号突出显示。

5.在上下文菜单中设置格式



ext install lacroixdavid1.vscode-format-context-menu

通过在上下文菜单中设置格式扩展名，可以通过在侧栏中突出显示文件并调用上下文菜单命令来设置文件格式。这在需要格式化许多文件的情况下特别方便，但是在现有环境中同时不支持用于格式化和整理代码的方法。


在上下文菜单中使用格式

6. Git图



ext install mhutchie.git-graph

Git Graph 扩展允许您查看存储库的结构，并使用此结构执行各种操作。此扩展程序是高度可定制的，并具有许多功能。为了正确地描述它，您可能需要一整篇单独的文章。


使用Git Graph

7. GitLens



ext install eamodio.gitlens

该GitLens扩展有助于识别使用Git怪注释和镜头代码作者。除其他功能外，此扩展使您可以非常方便地查看存储库的内容并了解有关代码的有用信息。


使用GitLens

8.缩进彩虹



ext install oderwat.indent-rainbow

缩进彩虹 扩展允许您对缩进进行颜色显示，突出显示具有不同颜色的不同级别的缩进。当使用Python等语言的深层嵌套构造时，这特别有用。

9.路径智能感知



ext install christian-kohler.path-intellisense

Path Intellisense 扩展使编辑器能够自动完成文件路径。尽管VS Code支持不带扩展名的路径完成，但此功能仅限于HTML，CSS和JavaScript文件。由于有了此扩展，在处理任何文件和任何语言时都可以使用路径完成。


使用Path Intellisense

10.总行



ext install praveencrony.total-lines

“ 总行数”扩展名是一个便捷的小型工具，可在状态栏中显示有关打开文件中的行数的信息。这是对我的扩展集合的一个很好的补充。
