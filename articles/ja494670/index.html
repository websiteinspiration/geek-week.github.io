Intersection Observer APIのケーススタディ



こんにちは！

概観


Intersection Observer API（IOA）を使用すると、アプリケーションは、要素（ターゲット）とその親（ルート）またはビューポート（ビューポート）との交差を非同期で監視できます。つまり、このAPIは、ターゲット要素がルートまたはビューポートと交差するたびに、特定の関数への呼び出しを提供します。

使用例：

  • 画像の遅延または遅延読み込み
  • 無限のページスクロール
  • インプレッションのコストを計算する目的で、広告の可視性に関する情報を受け取る
  • ユーザーの視野でプロセスまたはアニメーションを開始する


IOAの使用を開始するには、コンストラクターを使用して、コールバック関数と設定の2つのパラメーターを持つオブザーバーオブジェクトを作成する必要があります。

// 
let options = {
    root: document.querySelector('.scroll-list'),
    rootMargin: '5px',
    threshold: 0.5
}

//   
let callback = function(entries, observer){
    ...
}

// 
let observer = new IntersectionObserver(callback, options)

設定：

  • root-ターゲットのビューポートとして機能する要素（ターゲット要素の祖先またはビューポートのnull）
  • rootMargin-ルートの周囲のマージン（CSSのマージン、デフォルトではすべてのマージンは0）
  • しきい値-ターゲットとルートの共通部分の許容可能な割合を示す数値または数値の配列

次に、オブザーバーが監視するターゲット要素が作成されます。

let target = document.querySelector('.list-item')
observer.observe(target)

コールバック呼び出しは、ターゲット要素で発生した変更のレコードを含むオブジェクトを返します。

let callback = (entries, observer) => {
    entries.forEach(entry => {
        // entry () - 
        //   entry.boundingClientRect
        //   entry.intersectionRatio
        //   entry.intersectionRect
        //   entry.isIntersecting
        //   entry.rootBounds
        //   entry.target
        //   entry.time
    })
}

このネットワークには理論に関する情報が満載ですが、IOAの使用に関するかなりの資料があります。このギャップを少し埋めることにしました。

例


遅延（遅延）画像の読み込み


タスク：ユーザーがページをスクロールするときに画像をアップロード（表示）します。

コード：

//    
window.onload = () => {
    //  
    const options = {
        //    -  
        root: null,
        //  
        rootMargin: '0px',
        //   -  
        threshold: 0.5
    }

    //  
    const observer = new IntersectionObserver((entries, observer) => {
        //   - 
        entries.forEach(entry => {
            //    
            if (entry.isIntersecting) {
                const lazyImg = entry.target
                //     -   
                console.log(lazyImg)
                //   
                lazyImg.style.background = 'deepskyblue'
                //  
                observer.unobserve(lazyImg)
            }
        })
    }, options)

    //       img  
    const arr = document.querySelectorAll('img')
    arr.forEach(i => {
        observer.observe(i)
    })
}

結果：


ビューポート外のコンテナーの背景は白になります。



表示領域を半分だけ横切ると、背景が水色に変わります。

画像置換


タスク：ユーザーがページをスクロールしたときに、プレースホルダー画像を元の画像に変更します。

コード：

window.onload = () => {
    const observer = new IntersectionObserver((entries, observer) => {
        entries.forEach(entry => {
            if (entry.isIntersecting) {
                console.log(entry)
                //        "data-src"
                entry.target.src = entry.target.dataset.src
                observer.unobserve(entry.target)
            }
        })
    }, { threshold: 0.5 })

    document.querySelectorAll('img').forEach(img => observer.observe(img))
}

結果：


表示領域にあるため、最初の画像がアップロードされます。2番目はプレースホルダーです。



さらにスクロールすると、プレースホルダーは元の画像に置き換わります。

コンテナの背景を変更する


タスク：ユーザーがページをそこにスクロールしたときにコンテナーの背景を変更します。

コード：

window.addEventListener('load', event => {
    let box = document.querySelector('div')
    // ratio -   
    let prevRatio = 0.0

    let observer = new IntersectionObserver((entries, observer) => {
        entries.forEach(entry => {
            let curRatio = entry.intersectionRatio
            
            //      -  -
            //           (  )
           //      
            curRatio > prevRatio ? entry.target.style.background = `rgba(40,40,190,${curRatio})` : entry.target.style.background = `rgba(190,40,40,${curRatio})`

            prevRatio = curRatio
        })
    }, {
        threshold: buildThresholdList()
    })

    observer.observe(box)
    
    //    
    //      20 ,   
    function buildThresholdList() {
        let thresholds = []
        let steps = 20

        for (let i = 1.0; i <= steps; i++) {
            let ratio = i / steps
            thresholds.push(ratio)
        }
        return thresholds
    }
})

結果：


コンテナの背景が水色...



青から...



赤に変わります。

ビデオで作業する


タスク：実行中のビデオを一時停止し、表示領域に入るビデオに応じて、ビデオを再開します。

コード：

window.onload = () => {
    let video = document.querySelector('video')

    let observer = new IntersectionObserver(() => {
        //   
        if (!video.paused) {
            //  
            video.pause()
        //      (   > 0)
        } else if(video.currentTime != 0) {
            //  
            video.play()
        }
    }, { threshold: 0.4 })

    observer.observe(video)
}

結果：



ビデオが表示領域にある間、再生されます。



ビデオが表示領域を40％以上超えるとすぐに、再生が一時停止します。表示領域>ビデオの40％に到達すると、再生が再開されます。

ページビューの進行状況


タスク：ユーザーがページをスクロールするときに、ページの表示の進行状況を表示します。

コード：

//          
let p = document.querySelector('p')
// n -   
let n = 0

let observer = new IntersectionObserver((entries, observer) => {
    entries.forEach(entry => {
        if(entry.isIntersecting){
            // observer   div
            //       
            //     
            p.textContent = `${n++} div viewed`
            observer.unobserve(entry.target)
        }
    })
}, {threshold: 0.9})

document.querySelectorAll('div').forEach(div => observer.observe(div))

結果：



ページが読み込まれたばかりなので、コンテナーはまだ確認していません。



ページの最後に到達すると、段落に4つのdivの表示に関する情報が表示されます。

無限のスクロール


タスク：無限のリストを実装します。

コード：

let ul = document.querySelector('ul')
let n = 1

//    
function createLi(){
    li = document.createElement('li')
    li.innerHTML = `${++n} item`
    ul.append(li)
}

//  ,        
//     
//         li
//       () 
let observer = new IntersectionObserver((entries, observer) => {
    entries.forEach(entry => {
        if (entry.isIntersecting) {
            createLi()
        }
        observer.unobserve(entry.target)
        observer.observe(document.querySelector('li:last-child'))
    })
}, {
    threshold: 1
})

observer.observe(document.querySelector('li'))

結果：



12のリストアイテムがあります。最後のアイテムはビューポートの外にあります。



最後の要素に到達しようとすると、ユーザーから隠された新しい（最後の）要素が作成されます。無限に続きます。

親のサイズを変更するときに子のサイズを変更する


タスク：ある要素のサイズの別の要素への依存を確立します。

コード：

//      -   
//      
let info = document.querySelector('.info')
let parent = document.querySelector('.parent')
let child = document.querySelector('.child')
//     50px  
child.style.width = parent.offsetWidth - 50 + 'px'
//     
info.textContent = `child width: ${child.offsetWidth}px`

let options = {
    //      
    root: parent,
    threshold: 1
}

let observer = new IntersectionObserver((entries, observer) => {
    entries.forEach(entry => {
        //         50px
        if ((entry.target.parentElement.offsetWidth - entry.target.offsetWidth) < 50) {
            //     50px
            entry.target.style.width = entry.target.offsetWidth - 50 + 'px'
        }
    })
}, options)

observer.observe(child)

//  ,  ,       IOA
//       resize
window.addEventListener('resize', () => {
    info.textContent = `child width: ${child.offsetWidth}px`
    if ((parent.offsetWidth - child.offsetWidth) > 51) {
        child.style.width = child.offsetWidth + 50 + 'px'
    }
})

結果：



初期状態。



親要素の幅を小さくすると、子要素の幅が小さくなります。同時に、それらの間の距離はほぼ常に50pxに等しくなります（「ほぼ」は、逆のメカニズムの実装によるものです）。

アニメーションを操作する


タスク：オブジェクトが表示されているときにアニメーション化します。

コード：

//        
//     
//  - 
let observer = new IntersectionObserver((entries, observer) => {
    entries.forEach(entry => {
        entry.isIntersecting ? entry.target.classList.replace('to-left', 'to-right') : entry.target.classList.replace('to-right', 'to-left')
    })
}, {
    threshold: .5
})

observer.observe(document.querySelector('img'))

結果：



バートの頭の一部が表示されます。表示領域の左側に押されたバート。



バートの50％以上が表示領域に入る場合、彼は中央に移動します。バートの50％以上が表示領域を離れると、最初の位置に戻ります。
