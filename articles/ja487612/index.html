CMakeプロジェクトのファイルと目標を簡単に追加、削除、名前変更

Visual Studio 2019 16.5 Preview 2でのCMakeプロジェクトの操作がこれまでになく簡単になりました。これで、CMakeスクリプトを手動で編集しなくても、IDEからCMakeプロジェクトのソースファイルとターゲットを追加、削除、名前変更できます。ソリューションエクスプローラーを使用してファイルを追加または削除すると、Visual StudioによってCMakeプロジェクトが自動的に編集されます。ソリューションエクスプローラーのCMakeターゲットビューを使用して、プロジェクト目標を追加、削除、名前変更することもできます。



カットの下の詳細。

CおよびC ++ソースファイル


Visual Studioは、CおよびC ++ソースファイルがソリューションエクスプローラーで追加、名前変更、または削除されるときに追跡し、基になるCMakeプロジェクトを自動的に変更します。この機能はVisual Studio 2019 16.5 Preview 2ではデフォルトで有効になっていますが、Visual Studioでプロジェクトが自動的に変更されないようにするには、[ツール]> [オプション]> [CMake]の[CMakeスクリプトの自動変更を有効にする...]でこの機能を無効にできます。



目的と推奨事項


CMake Targets Viewはさらに多くの機能を提供します。ここから、ファイルの追加と削除に加えて、ターゲットを追加、名前変更、削除できます。CMakeターゲットビューにアクセスするには、[ホーム]ボタンの右側にある[ソリューションエクスプローラー]ドロップダウンメニューをクリックします。
