ADAM-6700 — modul komunikasi objek Linux Universal



Modul seri ADAM adalah produk yang paling dikenal dari perusahaan kami. Diluncurkan kembali pada tahun 1992, mereka terus berkembang, mempertahankan desain klasik yang dapat dikenali. Hari ini adalah perangkat multifungsi yang kuat yang menggabungkan beberapa kelas perangkat untuk menyelesaikan berbagai masalah.

ADAM-6700 adalah serangkaian perangkat baru yang didasarkan pada prosesor Cortex-A8 yang menjalankan Linux RT (waktu nyata). Ini menggabungkan fungsi modul komunikasi dengan objek, pengontrol logika yang dapat diprogram (PLC), perangkat untuk mengumpulkan dan memproses data, serta gateway IP untuk antarmuka digital dan analog.

Ini memiliki server Node-RED bawaan dan kemampuan untuk mengkonfigurasi melalui antarmuka web. Berkat platform terbuka, pada modul Anda dapat menjalankan aplikasi klien dalam C dan menerapkan logika kerja apa pun secara terprogram.



Spesifikasi


Perangkat baru dari seri ADAM-6700 didasarkan pada arsitektur ARM dan merupakan sistem produktif yang memungkinkan transformasi, penyimpanan, dan pemrosesan data lokal. Mereka menjalankan versi OS Linux dengan dukungan untuk operasi waktu nyata, sehingga dapat digunakan untuk operasi yang penting untuk pengaturan waktu.

  • Sistem Operasi: Linux RT
  • CPU: Cortex-A8 32-Bit 1GHz
  • RAM: 512MB
  • Port LAN: 2 independen
  • USB Slave: 1 (port konsol untuk konfigurasi)
  • Host USB: 1 (untuk menghubungkan perangkat)
  • Port input / output
  • Input Digital: 5 (Model ADAM-6717 ) 12 (Model ADAM-6750 )
  • Output digital: 4 (model ADAM-6717 ) 12 (model ADAM-6750 )
  • Input Analog: 8 (Model ADAM-6717 )



Kustomisasi


Konfigurasi awal perangkat dapat dilakukan melalui ADAM / Apax.NET Utility , di mana Anda dapat mengatur mode operasi jaringan dan alamat IP, atau melalui antarmuka web. Di antarmuka web, tersedia dua opsi konfigurasi perangkat: shell eksklusif Advantech dan Node-RED. Di halaman awal, Anda harus memilih salah satu opsi.


Antarmuka web

Node-Red shell untuk memilih shell adalah lingkungan open source yang populer untuk pemrograman visual dan kontrol aliran data. Perangkat seri ADAM-6700 memiliki server Node-RED bawaan, dan pemrograman dapat dilakukan baik secara langsung melalui browser web atau dengan mengimpor konfigurasi.


Antarmuka Web Node-RED

Antarmuka I / O digital dan analog tersedia melalui API untuk bahasa C, dan dapat digunakan untuk mengimplementasikan protokol mereka sendiri, untuk integrasi dengan peralatan pelanggan.

Konfigurasi melalui Micro USB



Untuk kemudahan konfigurasi, ADAM-6700 dapat dihubungkan melalui Micro USB sebagai perangkat pendukung ke komputer. Dalam hal ini, modul akan diberdayakan melalui port USB, yang sangat menyederhanakan pengaturan, karena tidak memerlukan daya penghubung melalui terminal, yang berarti dapat dilakukan oleh seorang insinyur tanpa alat dan peralatan tambahan, seperti catu daya dan obeng. Dalam sistem host, perangkat didefinisikan sebagai port COM virtual.

Antarmuka Host USB


ADAM-670 0 memiliki port USB A penuh untuk menghubungkan perangkat dalam mode Host. Ini memungkinkan Anda untuk menghubungkan drive eksternal, USB-stick, serta modul ekspansi, seperti modul WiFi ADAM 600W. Jika perlu, kernel sistem dapat dikompilasi dengan dukungan untuk driver apa pun untuk perangkat USB.

Modul WiFi ADAM-600W


Modul ekspansi ADAM-600W menambahkan dukungan WiFi ke seri ADAM-6700 . Koneksi melalui USB dan tidak memerlukan instalasi perangkat lunak tambahan. Setelah menghubungkan modul ADAM-600W, menu pengaturan koneksi WiFi akan tersedia di antarmuka web.

ADAM-6700 dengan modul ekspansi WiFi dapat bertindak sebagai gateway untuk menghubungkan perangkat lain yang terhubung melalui Ethernet ke server.

Kesimpulan


Perangkat universal seri ADAM-6700 dapat digunakan untuk menyelesaikan berbagai tugas otomasi, pengumpulan data, dan manajemen periferal. Sistem operasi multitasking tidak membatasi pengembang dalam menulis aplikasi yang kompleks. Dukungan untuk platform Node-RED terbuka dan kemampuan untuk mengkonfigurasi melalui antarmuka web mengurangi ambang entri dan memungkinkan pengembang untuk bekerja di lingkungan yang akrab dan akrab.
