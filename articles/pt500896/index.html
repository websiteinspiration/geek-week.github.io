Python aplicado, Technopark, outono de 2018


O Technopark é um dos nossos projetos educacionais conjuntos, neste caso com o MSTU. Bauman.

O curso apresenta a linguagem Python - uma das linguagens mais populares e populares do mercado de TI atualmente. A demanda pela linguagem não nasce do zero: a simplicidade de entrada e sintaxe, a mais rica escolha de ferramentas para resolver todos os tipos de tarefas - isso e muito mais levou ao fato de que o Python é amplamente usado em todo o mundo. Graças a este curso, você pode ingressar no ecossistema do idioma.

  • Aula 1. Introdução. Ferramentas de desenvolvimento, conhecendo o Git. Uma breve introdução aos princípios, conceitos e recursos do Python. Sintaxe básica.
  • Aula 2. Estruturas de dados . Estruturas básicas de dados em Python. Coleção de módulos.
  • Aula 3. Módulos . Como o código é organizado em Python. O que são módulos, como eles são criados, como o Python trabalha com eles. OOP.
  • Aula 4. Entrada-saída . O gerenciador de contexto, trabalha com o FS, trabalha com a rede, o básico do multithreading.
  • Palestra 5. Metaprogramação . Métodos mágicos, decoradores, metaclasses, descritores.
  • Aula 6. Bancos de dados relacionais . O dispositivo, recursos, métodos de aplicação.
  • Aula 7. Programação assíncrona . rendimento de, asyncio, aiohttp, asyncio-redis.
  • Aula 8. Desenvolvimento Web no Django . Noções básicas de HTTP, estruturas da web, segurança da informação, implantação de aplicativos da web.
