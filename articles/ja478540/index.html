第10回フォーラム「Positive Hack Days 10：Getting Started」が始まります



パーソナルコンピュータ用の最初の既知のウイルスは1981年に登場し、 7年後、ネットワークウイルスは9,600万ドルの損失をもたらしました。今日、ボットネット、標的型攻撃、サイバー兵器の時代に生きています。コンピュータ時代はどういうわけか間違って始まった。私たちとの間違いを修正し、ハイテクの安全な世界を構築したいですか？サークルを真の開発に変えますか？2020年5月13〜14日、クラスナヤプレスニャのエキスポセンターで開催されるアニバーサリーフォーラムPHDaysでお待ちしています。私たちは防御と攻撃について話すだけでなく、物語を再開することも試みます。

実用的なセキュリティPositive Hack Daysに関する第10回国際フォーラムを開催し、未来の情報セキュリティのコンセプトの共同開発に参加することをお勧めします。私たちは、ハッカーや情報セキュリティの専門家だけでなく、有名な政治家である大規模なビジネスマンも含めて、10,000人以上のゲストと参加者を期待しています。

展示ホールでは、50以上のスタンドを設置し、テーマに沿った関心領域を備え、セキュリティとハッキングシステムのデモンストレーションに高品質の通信とコンピューティングパワーを提供します。フォーラムのプログラムには、講義、コンテスト、コンテスト、そしてもちろん、ライブコミュニケーションとリアルタイムトレーニングが含まれます。

PHDaysがThe Standoffを主催したのは今回が5回目です。これは、フォーラムのゲストによく知られている攻撃者、防御者、およびセキュリティのエキスパートセンター（SOC）のチーム間のサイバー戦いです。最大のロシアおよび国際企業のチームが競争に招待され、セキュリティセンターは最も身近な防御手段を使用し、攻撃者は最も関連性の高い攻撃手法と方法を使用します。競争は主催者の独立したSOCによって監視され、Positive Technologies製品（MaxPatrol SIEM、PT Network Attack Discovery、PT Application Firewall、PT MultiScanner、PT Industrial Security Incident Manager）に基づいて展開され、バトルマップ全体をオンラインで見ることができます。

The Standoffの不可欠な部分は、Positive Technologiesの専門家が近代的な都市のインフラストラクチャを再構築するサイバートレーニングの場です。これは、大企業の企業システム、産業企業、およびその他の重要なインフラストラクチャを含むデジタル環境です。埋立地では、産業用制御システムの実システム、国内および海外の主要メーカーのSCADAおよびPLCを使用しています。

コンテストが開催されるサイバー注文の形式により、情報セキュリティの専門家、公共部門や企業の代表、開発者、ジャーナリストは、今日の真の脅威を理解することができます。参加企業にとって、これはハッカーからの本当のプレッシャーという困難な状況でインフラストラクチャの保護を解決するユニークな機会です。

5月13日と14日に、Krasnaya PresnyaのExpocenterでPositive Hack Days 10：The Beginningで会います。
