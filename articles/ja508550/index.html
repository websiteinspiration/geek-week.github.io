脳のアーキテクチャ。チャンネルのコンセプト







この記事では、脳の構造に関する私の仮説の1つについてお話します。







推論は、チャネルの非常に単純な概念に基づいています。おそらく読者はそれが素朴だとさえ感じるでしょう。それにもかかわらず、私はこれを思いつくために多大な努力をしなければなりませんでした。これは科学的な記事ではありません。誰かがそれを脳の主題についての別の推測と呼ぶならば、正式にはそれは正しいでしょう。批評は大歓迎です。







前書き



この仮説の歴史は2015年に始まりました。もう一度脳の構造について考えたとき、私は一つのアイデアに思いつきました。その本質は、脳が何らかの形で環境のモデルを構築しなければならないということでした。受容体とエフェクターの間の直接的なつながりを壊すためのモデルが必要です。そのようなモデルのない生き物は、ほとんど完全に外部信号に依存しています。移動は、タクシーまたはランダムウォークとして可能です。ある意味では、そのような生き物は無生物のようなものです。たとえば、フットボール選手の足から「外部信号」を受け取った場合にのみ、ボールは動きます。これは、栄養素に向かって化学勾配に沿って移動する従来の単細胞と大差ありません。彼の行動も同様に完全に外部信号によって決定されます。このパターンを変更するには、受容体とエフェクターの間に記憶を配置します。環境の大まかなモデルを格納します。

class dotdict(dict):
    __getattr__ = dict.get
    def __setattr__(self, name, value):
        self[name] = value

class Machine:
    """
         tick.
            tock.
     tick  tock        .
    """

    def __init__(self, cells):
        self.cells = cells
        self.tick = dotdict()
        self.tock = dotdict()
        for key in cells:
            self.tick[key] = 0b0000
            self.tock[key] = 0b0000

    def step(self):
        """    """
        for name, func in self.cells.items():
            if func is None:
                #  ,     
                self.tock[name] = self.tick[name]
            else:
                #       
                self.tock[name] = func(self.tick)
        #   
        self.tick, self.tock = self.tock, self.tick
        self.show()

    def show(self, comment = None):
        """    """
        if comment is not None:
            print(f': {comment}')
        for key in cells:
            print(key, f'{self.tick[key]:04b}')
        print('------------------------------------------------')
