9 Tips Terminal Windows oleh Scott Hanselman

Halo, Habr! Anda mungkin pernah mendengar bahwa Terminal Windows baru akan segera hadir. Kami sudah menulis tentang ini di sini . Rekan kami, Scott Hanselman, menyiapkan beberapa tip tentang cara bekerja dengan terminal baru. Bergabung sekarang!



Jadi, Anda mengunduh Terminal Windows dan ... sekarang apa?


Pada awalnya Anda mungkin tidak senang. Ini masih Terminal, dan itu tidak akan menuntun Anda, memegang tangan.

1) Baca dokumentasi pengguna untuk Windows Terminal

2) Pengaturan dinyatakan dalam format JSON . Anda akan melakukan lebih baik jika editor file JSON Anda adalah sesuatu seperti Visual Studio Code dan mendukung JSON serta intellisense.

  • Periksa pengaturan default Anda! Untuk kejelasan, saya menyajikan profil saya. Json (yang sama sekali tidak ideal). Saya menginstal Tema yang diminta, alwaysShowTabs, dan defaultProfile.

3) Tentukan pintasan keyboard. Terminal Windows memiliki banyak opsi penyesuaian .

  • Tombol apa pun yang Anda tekan dapat dialihkan.

4) Apakah desain sesuai dengan keinginan Anda?

  • Ada beberapa terminal tema pada terminalsplash.com.
  • Dan di sini ada puluhan skema warna iTerm2, skema warnanya dapat dilihat dengan jelas di sini .

5) Ingin naik ke level selanjutnya? Jelajahi gambar latar belakang.

  • Anda dapat mengatur gambar latar belakang atau bahkan GIF. Lebih detail di sini .

6) Tentukan direktori starting Anda.
  • Jika Anda menggunakan WSL, Anda mungkin cepat atau lambat ingin direktori home Anda berada di sistem file Linux .

7) Anda masih bisa menggunakan Far, GitBash, Cygwin, atau cmder jika Anda mau. Detail dalam dokumentasi .

8) Periksa argumen baris perintah Terminal Windows.

  • Anda mungkin tahu bahwa Anda dapat memulai Terminal Windows menggunakan "wt.exe", tetapi sekarang Anda dapat menggunakan argumen baris perintah! Berikut ini beberapa contohnya:

    wt ; split-pane -p "Windows PowerShell" ; split-pane -H wsl.exe
wt -d .
wt -d c:\github

    Pada titik ini, Anda dapat melangkah sejauh yang Anda inginkan. Buat ikon lain, sematkan di bilah tugas, turunkan semaksimal mungkin. Juga, periksa sub-perintah seperti tab baru, panel terpisah, dan tab fokus.

9) Saya merekam video yang menunjukkan kepada seseorang yang terbiasa dengan Mac dan Linux cara mengkonfigurasi Terminal Windows bersama dengan WSL (Windows Subsystem untuk Linux), mungkin itu akan menarik bagi Anda.

Silakan bagikan di bawah kiat, profil, dan topik favorit Anda untuk terminal!
