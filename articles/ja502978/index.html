デザイナーの助けなしの配色


きっとあなたの多くは装飾用の色を素早く選ぶ必要がある状況にあり、デザイナーは気分が悪いときや休暇中にもっと重要な仕事で忙しいです。難しいことではありませんが、答えを数日待たなければならない場合があります。

私の名前はエミールフロロフです。私はDomKlikの内部サービスチームのテクライドです。この記事では、ライブラリのアイデアがどのようにして生まれたのかを説明します。これにより、色を選択するときに多くの時間を節約できます。ソリューションはシンプルですが、非常に便利です。

特に、ダークスキームの色を選択する必要がある場合。

仕事


DomKlikには企業ポータルがあり、各セクションには独自の配色があります。以前は、新しいセクションを作成するために、デザイナーを苦しめ、デザイナーに新しい色のセットを取得してインターフェースに転送するよう依頼する必要がありました。膨大な量の余分なコードが判明し、通信、待機、調整に多くの時間を費やしました。プロセス全体を単純化してスピードアップしたかったのです。

インターネットで既製のソリューションを探しましたが、適切なものが見つかりませんでした。次に、ライブラリを作成することにしました。デザイナーが指定した色（ブランド）を入力すると、ライブラリがさらに適切な色をいくつか選択します。また、ライブラリに暗い配色を生成することも必要でした。
これは幸福のレシピではなく、誰もがプロジェクトの枠組みの中で開発できるという考えです。ここで小さなデモを見ることができます。

そして、要素のテキストを表示する色は何ですか？

その仕事は思ったほど難しくはなかった。16進数の値から解決するには、要素の明るさを取得する必要があります。2つのヘルパー関数でこれを行いました。1つ目は、hexをRGBに変換します。

const hexToRgb = (hex) => {
  const shorthandRegex = /^#?([a-f\d])([a-f\d])([a-f\d])$/i;
  const newHex = hex.replace(shorthandRegex, (
    magenta,
    red,
    green,
    blue
  ) => red + red + green + green + blue + blue);

  const result = /^#?([a-f\d]{2})([a-f\d]{2})([a-f\d]{2})$/i.exec(newHex);

  return result ? {
    red: parseInt(result[1], 16),
    green: parseInt(result[2], 16),
    blue: parseInt(result[3], 16)
  } : null;
};

RGBの2番目の関数は、背景の明るさを受け取り、色が暗いか明るいかを決定します。

export const getBgTextColor = (bgColor) => {
  const rgb = hexToRgb(bgColor);
  const light = (rgb.red * 0.8 + rgb.green + rgb.blue * 0.2) / 510 * 100;

  return light > 70 ? '#000000' : '#ffffff';
};

2日間で、デザイナーの助けを借りて、ライブラリを作成しました。このライブラリは、出力時に暗いテーマと明るいテーマのカラーパレットを提供します。

次の質問：使い方は？

とても簡単です。生成された配色を実装するために、スタイルブロックを介したCSS変数の挿入を使用しました。これにより、他のCSSライブラリが使用するCSS変数との競合が回避されます。

  const colors = getColors(color, themeKey === 'dark');
  const colorsVars = Object.keys(colors).map((key) => `--${key}: ${customColors[key]}`).join(';');
  
  const link = document.createElement('style');
  const headTag = document.getElementsByTagName('head')[0];

  link.type = 'text/css';
  link.id = 'project-theme-scope';
  const stylesBody = `:root {${colorsVars}}`;

  link.innerText = stylesBody;

  headTag.append(link);

そして今、最もおいしい。注目注意！数行で、要素の半分にダークテーマのサポートを追加すると、次のようになります。

  body {
    background: var(--color-bg-ghost);
    color: var(--color-typo-primary);
  }

その結果、メインのブランドカラーを渡して一連のCSS変数を取得し、プロジェクトの色付けに使用するライブラリができます。
