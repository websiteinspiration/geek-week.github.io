自分で電子署名を作成する方法

私はすぐに予約をします-私は電子デジタル署名（EDS）に関連する問題のほとんど素人です。最近、自然の好奇心に駆られて、これを少し整理することにしました。さまざまな認証センターでデジタル署名証明書を取得すること、およびドキュメントに署名するためのさまざまな既製のアプリケーションを使用するための多数の指示について、インターネット上で100,500件の記事を見つけました。一部の場所では、「経験豊富なプログラマ」のサービスを使用すれば、未熟な署名を個別に作成できることが言及されていました。


また、私は少なくとも少しは「経験者」になり、このキッチンを内側から理解したいと思っていました。楽しみのために、PGPキーを生成する方法、修飾されていない署名でドキュメントに署名する方法、およびその信頼性を確認する方法を学びました。アメリカが開かれていないことを理解していても、EDSでの作業に関して私と同じであるアマチュア向けに、この短いチュートリアルを提供します。私は特に理論と詳細を掘り下げるのではなく、質問の簡潔で短い紹介だけを書きました。すでにEDSを使用している人にとっては、これは面白そうにありませんが、初心者にとって、最初の知り合いにとっては-ちょうどいいです。


電子署名とは


すべての用語と定義は法律に定められているため、私たちは、彼らが言うように、言葉遣いの絶対的な法的正確さを装うことなく、あなた自身の言葉ですべてを述べます。


電子デジタル署名（EDS）は、ドキュメントの作成者（またはアクションの実行者）が自分を作成者と呼ぶ人物であることを明確に確認できるようにするツールのコレクションです。この意味で、EDSは従来の署名と完全に類似しています。通常の「紙」の文書で作成者がイワノフであり、ペトロフの署名が下にある場合、イワノフの作成者を疑うことができます。


電子署名はシンプルで強化されています。単純な署名は、標準の暗号アルゴリズムの使用を意味するものではありません。EDSの時代の前に発明されたすべての認証方法（承認）-これは基本的に単純な電子署名です。たとえば、公共サービスのウェブサイトに登録し、多機能センターに出向いて身元を確認し、このウェブサイトを通じてさまざまな政府機関に申し立てを送信した場合、公共サービスのウェブサイトのユーザー名とパスワードは、この場合は単純な電子署名になります。

let cont   = {doc:'', sig:''},
    flag   = {doc:false, sig:false},
    pubkey = '',
    mess   = '';

//    ( ),
//    ( )
const readDoc = contKey => {
    let reader = new FileReader();
    reader.onload  = async e => {
        cont[contKey] = contKey == "sig" ?
                        e.target.result :
                        new Uint8Array(e.target.result);
        flag[contKey] = true;
        pubkey = await (await fetch("public.key")).text();   
        if (flag["doc"] && flag["sig"])
            document.querySelector("button").disabled = false;
    }
    reader.onerror = err => alert("  ");

    let fileObj = document.querySelector(`#${contKey}`).files[0];
    if (contKey == "sig") reader.readAsText(fileObj);
    else                  reader.readAsArrayBuffer(fileObj);
}

//  
const check = async () => {
    try {   
       const verified = await openpgp.verify({
           message:    openpgp.message.fromBinary(cont["doc"]),
           signature:  await openpgp.signature.readArmored(cont["sig"]),
           publicKeys: (await openpgp.key.readArmored(pubkey)).keys
       });
       const {valid} = verified.signatures[0];
       mess = "    !";
       if (valid) mess = "   .";
    } catch(e) {mess = "    .";}
    document.querySelector("output").innerHTML = mess;
}
