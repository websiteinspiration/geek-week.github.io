だから、私は正確に何をしましたか（以下、フラッシュドライブなしですべてができる方法を説明します、私は後でシステムをよりよく理解したときにすでに理解しました）：

1. 32GBフラッシュドライブを購入しました、おそらく16GB（それはインストーラーに必要です） ）

2.インターネット回復モードで起動します（コマンド（⌘）-オプション（⌥）-R）。

3.「ディスクユーティリティ」を実行し、ハードドライブ（Macintosh HDハードドライブの名前を持っています）とUSBフラッシュドライブをこのようなパラメーターでフォーマットします。



4.さらに、ターミナルからイメージをダウンロードすることは可能ですが、残念ながら、リカバリモードMacOS Lionはインターネットからファイルをダウンロードするための基本的な「curl」コマンドをサポートしていないため、別の方法を見つけました。

Safariを開き、上部メニューの「Safari→設定→ダウンロードしたフォルダに保存」に移動し、HDDを選択します。



5.閉じる設定やアドレスバーにアドレスを入力します。

http://swcdn.apple.com/content/downloads/29/03/091-94326/45lbgwa82gbgt7zbgeqlaurw2t9zxl8ku7/BaseSystem.dmg

を押して「入力」および負荷への画像のための待ち時間。



6.トップメニューの[Safari→End Safari]でSafariを閉じ、[ユーティリティ]→[ターミナル]を開きます

。7.次に、イメージ「OS X Base System」をマウントします。ターミナルに次のコマンドを

入力します：hdiutil mount / Volumes / Macintosh \ HD / BaseSystem.dmg

（トピックとは少し異なります。左から右へのスラッシュは名前にスペースを意味します。つまり、このコマンドは次のように入力することもできます：hdiutil mount“ / Volumes / Macintosh HD /BaseSystem.dmg”）
イメージがマウントされるまで待機します。

8.次に、上部メニューの「ターミナル→ターミナルの終了」で

9.「ディスクユーティリティ」を再度開き、スクリーンショットのようにブートローダーをフラッシュドライブに復元します（復元中に、パーティションではなくソースイメージ自体を選択し、宛先-フラッシュドライブパーティションを選択することに注意してください）

11.リカバリモードに入りますが、Mac OS High Sierraはすでにインストールされており、「macOSのインストール」を選択するだけです。

その後、すべてがすでにうまくいき、問題は発生しません。

フラッシュドライブを購入する機会がない人のためのオプション。

アクションは同じです。ディスクユーティリティでは、ハードディスクを2つのセクションに分割します。1つはインストーラー用に16 GBにします。そのような選択肢がある場合は、ハードディスクの最後に追加することをお勧めします。さらに、アクションは同じです。イメージをメインパーティションにダウンロードしてマウントし、USBフラッシュドライブに復元しませんが、作成した16GBパーティションを選択します。オプション（⌥）キーを押して再起動すると、リカバリパーティションがリストに表示され、そこから起動してメインパーティションにOSをインストールします。
