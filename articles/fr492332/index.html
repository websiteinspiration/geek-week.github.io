Créez des applications gratuitement avec Azure Cosmos DB Free Tier

Vous souhaitez créer une nouvelle application, développer et tester ou exécuter de petites charges de travail de production avec  Azure Cosmos DB ? Notre nouveau niveau d'utilisation gratuite vous permet de commencer à travailler sans frais et d'économiser de l'argent lors de la création et du développement de nouvelles applications.



Avec le niveau gratuit Azure Cosmos DB inclus, vous obtenez gratuitement les premiers 400 RU / s de bande passante et 5 Go de stockage dans votre compte chaque mois pendant toute la durée du compte. Cela signifie que vous pouvez commencer petit et évoluer en toute confiance, sachant que votre application s'exécutera dans un service de base de données hautes performances. Vous ne paierez que si votre compte dépasse 400 RU / s et 5 Go. De plus, si votre application possède de nombreux conteneurs, vous pouvez créer jusqu'à 25 conteneurs dans la base de données avec une bande passante totale, et tous utiliseront 400 RU / s gratuitement. Vous pouvez avoir jusqu'à un compte Azure Cosmos DB gratuit pour chaque abonnement Azure.


Que pouvez-vous créer gratuitement?


Nous avons demandé aux membres de notre communauté de développement de fournir des exemples d'applications avec 400 RU / s et 5 Go d'espace disque et avons reçu de nombreux excellents exemples des applications qu'ils ont créées: l'application Alexa, qui aide les passagers à utiliser les transports en commun, ainsi qu'une application Web pour gérer et partager facilement  des listes en ligne ressources . Nous parlerons de certains de ces développeurs et de leurs applications dans les semaines et mois à venir.

Avez-vous une application avec une bande passante inférieure à 400 RU / s et 5 Go de stockage? Parlez-nous de lui  - nous serons heureux de le partager avec la communauté.

Début des travaux


Créez un nouveau compte Azure Cosmos DB et activez le  niveau gratuit  lors de la création. Il existe une limite à un compte d'abonnement Azure gratuit. Consultez nos  guides et guides de démarrage rapide  ou utilisez  un exemple pour comprendre ce que vous pouvez faire avec Azure Cosmos DB.

Foire aux questions (FAQ)


Q: N'avez-vous pas offert une option gratuite auparavant? Qu'est ce qui a changé?

R: Nous avons toujours proposé des moyens d'essayer gratuitement Azure Cosmos DB - dans le cadre du  compte gratuit Azure , avec un  émulateur local et en tant  qu'essai gratuit de 30 jours  - mais ces options étaient limitées en termes de disponibilité et de volume. Le niveau gratuit est disponible pour une durée illimitée.

Q: Comment la remise de niveau gratuite s'affiche-t-elle sur mon compte?

R: Dans les comptes de niveau gratuit, vous obtiendrez gratuitement les 400 premiers RU / s et 5 Go de stockage dans votre compte. Tout RU / s et stockage au-delà de 400 RU / s et 5 Go seront facturés aux tarifs habituels sur la page de tarification. Sur la facture, vous ne verrez pas de frais ou d'élément de ligne pour les 400 Ru / s et 5 Go gratuits, uniquement les RU / s et le stockage au-delà de ce qui est couvert par le niveau gratuit. Dans les comptes de niveau gratuit, vous recevrez gratuitement les 400 premiers RU / s et 5 Go d'espace disque sur votre compte. Tout RU / s et stockage supérieur à 400 RU / s et 5 Go seront payés aux tarifs réguliers. Sur le compte, vous ne verrez pas de frais ou de position pour 400 Ru / s et 5 Go gratuits, uniquement des RU / s et du stockage au-delà de ce qui est couvert par le niveau gratuit.

Q: Le niveau gratuit peut-il s'appliquer à toutes les API Azure Cosmos DB?

Oui, Free Tier fonctionne pour toutes les API Azure Cosmos DB, y compris Core (SQL), les API pour MongoDB, Cassandra et Gremlin.

Q: Puis-je appliquer un rabais de niveau gratuit pour les  bases de données et les conteneurs du pilote automatique ?

R: Oui. Avec le service de pilote automatique, vous êtes facturé pour le RU / s le plus élevé, par lequel la base de données ou le conteneur est mis à l'échelle par heure. Lorsqu'un rabais de niveau gratuit est appliqué, 400 RU / s sont déduits de cette valeur. Consultez notre  documentation pour plus de détails.

Q: Comment fonctionne la remise du niveau gratuit si j'ai un compte dans plusieurs régions?

R: Dans les comptes RU / s multirégionaux, la base de données ou le conteneur est répliqué dans toutes les régions. Par exemple, si vous avez un conteneur avec 400 RU / s et que le compte est situé dans 3 régions, le nombre total de comptes RU / s est de 1200 RU / s. Lorsque la remise est appliquée, vous serez facturé à 1200 RU / s - 400 RU / s = 800 RU / s par heure. Consultez notre  documentation pour plus de détails.

Q: J'ai déjà un compte et je souhaite transférer mes données sur le compte du niveau d'utilisation gratuite. Comment dois-je procéder?

R: Une fois votre nouveau compte créé et activé avec Free Tier, il sera facile de déplacer les données existantes vers un nouveau compte.  Outil de migration des données peut être utilisé pour transférer de petits ensembles de données si vous utilisez notre API Core (SQL). Vous pouvez également migrer des données à partir d'Azure Data Factory  pour l'API Azure Cosmos DB Core (SQL)  et l' API Azure Cosmos DB  pour MongoDB .

Q: L'option d'activation du niveau gratuit lors de la création d'un nouveau compte sur le portail Azure n'est pas affichée. Où est-elle?

R: Un abonnement Azure peut avoir jusqu'à un compte de niveau gratuit. Si cette option n'est pas affichée, cela signifie qu'un autre compte de l'abonnement est déjà activé au niveau gratuit.
