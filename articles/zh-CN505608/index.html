Vuex-用新方法解决旧争议

Vuex有一种流行的做法-不使用mapState和mapMutations。通常。取而代之的是，在商店中创建新值时，我们会立即得到它的吸气剂，并对每个突变进行操作。


就像是：


export default new Vuex.Store({
  state: {
    // 1
    count: 0
  },
  getters: {
    // 1
    count: (state) => state.count
  },
  mutations: {
    // 2
    increment(state) {
      state.count++;
    }
  },
  actions: {
    // 2
    increment({ commit }) {
      commit('increment');
    }
  }
});
