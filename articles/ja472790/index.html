骨董品：ビジネスのコミュニケーター、i-Mate Jasjar

ゼロの後半では、HTCはスマートフォン市場のリーダーではないにしても、一時的にWindows Mobileプラットフォームをベースにした最先端のデバイスのメーカーになりました。 iPhoneがまだ奇妙な好奇心であった短い期間でした。Androidは市場をリードするために戦い始めたばかりで、幅広い機能を備えていますが、Nokiaは従来の電話のメーカーとして知られていました。大きなタッチスクリーンと通常よりも少し多くの機能が必要な場合は、Windows Mobileデバイスが最適でした。ソフトウェア、マルチタスク、および多くの機能の幅広い選択肢を備えた、かなり大人のプラットフォームです。



わずか3年前、HTCはより有名なブランドのODMベンダーとして専門家にのみ知られていました。たとえば、HP iPaq hx4700 Pocket PC はHTCによって開発されました。注文されていない自社デバイスは、もともとQtekブランドで販売されていました。今日は、移行中のコミュニケーターについてお話します。 HTC Universal、i-Mate Jasjar、Qtek 9000、Dopod 900、およびO2 XDA Execと呼ばれるデバイス。多面的なヤヌスは、一度にすべてを1つのデバイスにまとめようとする最初の試みの1つです。大画面とキーボード、2つのカメラ、そして第3世代の通信です。現代のデバイスの遠い祖先ですが、ウェアラブルコンピューターを構築するまったく別の学校を表しています。

私は古い鉄の収集家の日記を電報に保管しています。

ミレニアムの初めには、NokiaスマートフォンよりもWindows Mobileデバイスを好みました。後者は通常の携帯電話のように見え、タッチスクリーンがなく、キーパッドが装備されていました。代わりに、BluetoothとGPRSを備えたシンプルな携帯電話とハンドヘルドコンピューターのバンドルを使用しました。どちらも安くて便利です。電話は通話とSMSには十分であり、比較的短期間のバッテリー駆動のPDAは、Webサーフィン、メールの確認、写真とビデオの表示、本の閲覧を目的としています。これらの2つのデバイスを1つに結合するのは論理的です。はい、しかし、最初の試みはかなり奇妙に見え、うまくいきました。たとえば、2002年にリリースされたコミュニケーターO2 XDA（これもHTCによって製造されています）は次のとおりです。


これは現在、画面が6〜7インチのスマートフォンに混同されていません。 2002年に、単純な携帯電話が本当にミニチュアになったとき、そのようなシャベルは奇妙に見えました。そして、それは他の人が考えることだけではありませんでした：これらのデバイスを使用することは率直に言って不便でした。管理-スタイラスでのみ、いくつかの基本的なタスク（番号のダイヤル、メッセージへの回答）には両手が必要でした。必要なセンサーがない場合、会話中に所有者は画面上で耳を押し、電話は自分の生活を送っていました。 2004年に、サイズの問題は2.8インチの画面を備えたQtek S100デバイス（HTC Magician）のリリースで解決され、寸法は従来の携帯電話とほとんど変わりませんでした。これは私がマスコミで読んだだけでなく、友人や知人と定期的に見ていた最初のコミュニケーターでした。


2005年後半にリリースされたHTC Universal（別名Jasjar、Qtek 9000など）は、S100とは逆の動きでした。HTCは、最も機能的なデバイスを作ることを決定しました。これは、電話であるだけでなく、ラップトップを置き換えることさえできると主張しています。同時に、それは広々としたポケットにも収まります。


デバイスの仕様は次のとおりです。
プロセッサ：Intel PXA270 520 MHz
ディスプレイ：640x480 3.7インチ
内部メモリ：128 MB
RAM：64 MB
メモリカードスロット：SD / MMC、公式には最大4 GB、実際には最大32 GBのSDHC
通信： GSM / GPRS + WCDMA 2100 MHz、Bluetooth 1.2、WiFi 802.11b、IrDA
バッテリー：1620 mAh、取り外し可能
寸法：128x81x25 mm
重量：285グラム


悪くない！公式には、スマートフォンはオペレーティングシステムWindows Mobile 5.0でリリースされました。 Windows Mobile 6.1の非公式アセンブリをインストールしました-このバージョンのOSは2007年にリリースされました。彼らは2008年に、このプラットフォームの最後のメジャーリリースであるWindows Mobile 6.5をリリースしました。このバージョンとは異なり、6.1は伝統的に見えますが、すべての栄光は明るい緑で快適ですが、スタイラス駆動のインターフェースのみを示しています。


WM6.1はそのままの状態でSDHCカードをサポートしますが、5番目のバージョンでは非公式の「松葉杖」が必要でした。デバイスは当初WM6で動作するように設計されていましたが、画面の下に2つのハードウェアボタンがあり、画面の下部にショートカットキーを複製していました。カスタムファームウェアのもう1つの利点は、実際の「真の」VGAモードをサポートすることですが、デフォルトでは、Windows Mobileは320x240の解像度に基づいてインターフェイスをレンダリングしました。ちなみに、これは現代のスマートフォンの動作と非常によく似ています。紙の上では画面解像度が非常に高いのですが、実際には速度とバッテリー節約のために低い解像度で動作します。


その印象的な厚さ（2.5センチ！）、小さい画面、奇妙なフォームファクターにもかかわらず、HTC Universalには、最新のデバイスと同様のその他の機能があります。ステレオスピーカーを搭載：最近のスマートフォンにこんな贅沢が出てきました。 1本のケーブルでデータの充電と送信を行います。さて、コネクタ標準は時代遅れです-miniUSB-当時は、それは信じられないほどクールでした。以前の多くのPDAやスマートフォンには、同じiPaq hx4700のような独自のコネクタがありました。ケーブルまたは自分の電源を紛失または忘れた-書き込みがない-旅行中に充電するものは何もありません。ここでは、非常に低速ですが、コンピュータのUSBポートから充電できます。


完璧な状態で携帯電話を手に入れましたが、バッテリーが疲れていました。文字通り10分の作業に十分であり、その後デバイスがオフになった。入手困難な新しいバッテリーは、その自律性に非常に満足していませんでした。せいぜい、ビデオ表示モードで5時間です。当時のスマートフォンは、現在のスマートフォンほど活発に利用されていませんでした。ビデオストリーミングはできません-もちろん可能ですが、比較的微妙な入力では厳しいテストになります。インターネットでさえ完全に使用され、ICQ内の誰かと長時間通信する必要がある場合、これはネットワークからの電力で行われました。そして、もし本当にあなたができるだけ長く自律的に働く必要があるなら、あなたはより厚いバッテリー（2600、3800、またはさらに5200 mAhの容量）を購入することができます。このようなバッテリーは、デバイスの厚さと重量を2倍にしました。充電せずに12時間できないのは！現在でも存在する自律性の欠如は、外部の電力銀行によって解決するのがはるかに簡単です。

I-Mateを旅行に連れて行って、2005年にこのようなデバイスを使用するのがどのようなものかを実際に感じました。 14年前、私はそのようなスマートフォンの存在を知っていましたが、間違いなくそれを買う余裕はありませんでした（代わりにiPaqが購入されました）。現在の主力スマートフォンのように、1000ドル以上かかりますが、インフレを調整しません。しかし、今ではそれが（ここ数年で）当たり前になり、少しワイルドになりました。同じお金でノートパソコン全体を購入できます。 OSがゼロから再フラッシュされたため（リンク切れの海でライブアセンブリを探すのに苦労した）、去る前に標準のソフトウェアセットを配置する必要がありました。まず、クラック：パラゴンのレン、スマートフォンに付けられたシリアル番号で問題なく起動しました。


ハードウェアキーボードはタイピングを大幅に簡素化します。スタイラスを比較的小さな画面ボタンに向ける必要はもうありません。レイアウトはFn + Shiftの組み合わせを使用して切り替えられますが、別のユーティリティを使用すると、1つのボタンで切り替えを構成できます。ロシア語の文字の一部が物理的なキーボードに収まらず、下の行に残されました。クラックに加えて、標準セットのソフトウェアをインストールしました。HaaliReaderリーダー、TCPMPビデオプレーヤー、スクリーンショットを撮るユーティリティです。


古いデバイスで最新のページをどうにかして開く方法はありますが、Webには触れないことにしました。組み込みのInternet Explorerが「互換性のある」レトロサイトで動作することを確信しており、それで十分です。当然、Windows Mobileの最新のメッセンジャーは機能しません。それらのほとんどは、プラットフォームが廃止された後に表示されました。


3Gインターネットのサポートは比較的新しい技術でした。ベースステーションがそれをサポートしている場合、384キロビット/秒の速度でデータを転送できます。 2007年にリリースされた最初の第3世代Apple iPhoneは通信をサポートしていませんでした。 2019年には、これがi-Mate Jasjarをオンラインにする最も簡単な方法です。組み込みWiFiは802.11b標準のみをサポートし、WPA2暗号化をサポートしません。つまり、アクセスポイントの標準セキュリティ機能をオフにする必要があります（強く推奨）、または最新のスマートフォンから「オープン」WiFiを配布する必要があります。 SIMカードを挿入する方が簡単です。


すべてうまくいきます...あまり安定していないとしましょう。再起動後、電話はかなり長い時間ネットワークにログインし、数回通話に応答するとソフトリセットが発生します。ここで、電話の異常なファームウェアのすべての罪を非難することができますが、実際には、Windows Mobileの電話部分は私にとって非常に印象的でした。しかし、一般的には、個々の不具合を除いて、すべてが機能します。


シリーズのいくつかのシリーズをMpeg4形式で、i-MateのVGA解像度で投げました-このデバイスは、古いデバイスとは異なり、消化できます。そして、それは消化しますが、完全ではありません。 TCPMPプレーヤーには、ビデオのすべてのフレームを再生しようとする組み込みのベンチマークがあり、速度は比較的正常であると見なされます。結果は93％でした。つまり、プレーヤーはこのビデオに特に困難に対処しています。実際には、これはそれほど目立ちません。一部のフレームは単純に破棄されます。このようなプロセッサーを搭載したデバイスはVGAビデオを使用できますが、ビットレートの選択を減らし、品質を低下させる必要があります。特別なユーティリティを使用して、プロセッサをわずかに624メガヘルツにオーバークロックできます。原則として、これは機能しますが、ビデオの再生は特に改善されません。

Windows Mobileの時代は、ファームウェア、プロセッサの周波数、モデムのバージョンさえも選択して、すべてが多かれ少なかれ動作し、速度が低下しないように、そのような無限のカスタマイズと微調整の時代です。これで、デバイス（ブックとビデオ）を最小限にして、デバイスですべてを実行したいと思いました。ビデオの品質に戻る：最後に、800 MHzと1 GHzのプロセッサーを搭載した10年間の終わりのモデルでは、DVD品質でのビデオ再生の制限が解除されました。しかし、その後すぐに高解像度ビデオが現れ、すべてが再び始まりました。


さて、私の意見では、最も重要な欠点は、デバイス本体へのスクリーンマウントの軸にある2つのインジケーターです。 1つは、充電とセルラーネットワーク信号の存在を示します。もう1つは、無線モジュールと通知の操作に関するものです。一般に、これらは2つのLEDであり、デバイスをポートレートモードで操作すると（上記の写真のように）、常に目に光ります。それらはオフにならないようです、そして私のiPaqのようにそのような問題は電気テープだけで解決されます。


それにもかかわらず、2005年の「多機能」デバイスの概念は、現在のものとは異なりました。また、2019年のi-Mateの要件を満たしていない場合は、すべて問題ありません。データ転送、通話、SMSが機能します。あなたは本を読むことができます、この投稿の約3分の1が入力された非常に便利なキーボードがあります。 i-Mateを数時間見て音楽を聴きましたが、今ほど便利ではありませんでしたが、うまくいきました。当時の従来のPDAと比較すると、折りたたみ式（および回転式）画面を備えたデバイス形式は非常に便利であることがわかりました。コミュニケーターをテーブルに置いて、即興アイテムの複雑なデザインなしで映画を見ることができます。本格的なヘッドホンジャックさえあります。このことは、15年前の多くのデバイスだけでなく、最近のデバイスにもありません。


しかし、このデバイスのカメラは、品質をあまり気にせずに「そのまま」挿入されました。ビルトインの写真およびビデオソフトウェアは、インストールされたSPB Pocket Plus拡張機能での起動を拒否しました。写真アプリが利用可能なメモリをすべて使い果たしてしまったように感じます。 SDカードに画像を保存することはできません。限られた内部ストレージでのみ可能です。写真の解像度-1.3メガピクセル、焦点合わせだけでなく、露出設定も修正されています。秋の晴れた晴れた日には、通常、写真を露出することは不可能であり、露出オーバーは避けられません。上の写真は、i-Mateで描写できた最高のものです。 「もはやフラッグシップではない」Samsung S9 Plusの品質と比較してみましょう。


まったく別の問題！具体的には、アプリケーションに、撮影日時をフレームに直接印刷するオプションを含めました。以前はひどい時代錯誤のように思えましたが、今ではこれらの石鹸のような写真にヴィンテージが追加されています。0.5 m未満の距離でシャープネスを実現するのは困難です。おそらく、フレーム内のすべてがぼやけます。下の写真は、私たちがなんとかして最大のシャープネスを達成した写真です。

また、このデバイスに搭載されている0.3メガピクセルの前面カメラの撮影方法は示していません。そして、私はしません、それは悲痛な光景です。それにもかかわらず、すでに2005年には、i-Mateの所有者はビデオ会議に完全に参加することができました。メインカメラの評決は次のとおりです。1998年のフロッピーディスクのカメラのほうがよく撮影されます。


ミレニアムの初めには、「ビジネスのためのデバイス」のような論理的な構造がありました。ビジネスパーソン向けのNokia 6310i電話の箱には、フォーマルな服を着た幸せな女性と紳士が描かれ、最新のモバイル通信技術を使用して重要な話し合いが行われました。 i-Mate Jasjar（HTC Universal、Qtekなど）-ビジネスコミュニケーターでした。自分のためではなくデバイスを購入した人だけがそれを買う余裕があったからです。実際、ビジネスはそれとは何の関係もありません。 Qtek 9000（I-Mate、HTC ...）-厚みと低い自律性に耐えることができる愛好家向けの電話であり、さらに（比較的単純な携帯電話）でさえソフトウェアの不安定性があり、これらすべてが進歩のためにありました。これは、組み込みのGPSが欠落していない限り、一度にすべてが存在する最初の正常なモバイルデバイスの1つでした。

そのような装置は長い間関係がありませんでした。たとえば2008年には、HTC Touch Diamondモデルがリリースされました。これは、ほぼ最新のスマートフォンですが、独自のインターフェイス「拡張機能」を備えたWindows Mobileを使用しています。HTCがリーダーシップを維持できず、アップル、サムスン、低コストデバイスの中国メーカーに負けた別の時代が始まりました。しかし、それはすべてとてもうまく始まった。I-Mate Jasjarは、通信の有無にかかわらず「スマートな」ハンドヘルドコンピュータが夢だった時代から、ユニバーサルウェアラブルデバイスの古い学校の代表です。厳しい現実ではありません。
