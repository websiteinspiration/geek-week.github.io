過去50年間で最も影響力のある10のプログラミング言語とその作成者

ITの世界には多くのプログラミング言語があります。毎年ますます多くあります。たとえば、Scala、Kotlin、Go、Closureという言語が最近登場しました。しかし、歴史は、それらのうちのほんの少しだけが生き残るであろうと私たちに伝えます。

ここで議論されている言語は、ソフトウェア開発の世界に多大な貢献をしています。だからこそ、彼らは過去50年間で最も影響力のある10の言語の1つに数えられています。 言語はプログラミング業界の重要な部分です。それらはしばしば議論され、批判され、時間とともに改善されます。彼らは誰にでもよく知られていますが、誰もがその歴史、少なくとも作成者の名前を知っているわけではありません。これは一般的な言語ではあまり一般的ではありません。たとえば、誰もがジェームズゴズリング



Javaの父です。しかし、すべてのプログラマーが誰がPerl、Pascal、Lisp、またはErlangを作成したかを知っているわけではありません。

本日の翻訳を公開している記事の著者は、上記の状況が彼にそれを書くように促したと述べています。ここでは、最も人気があり成功しているプログラミング言語とその作成者について説明します。

トップ10のプログラミング言語とその作成者


10のプログラミング言語とその作成者のストーリーを紹介します。ここでの言語はランダムな順序ですが、リストの最初のJava言語は例外です。私はJava開発者なので、そうすることにしました。

私は多くのCプログラマーが私に同意しないことを知っており、リストの最初の言語はCである必要があると考えています。Cはまだ広く使用されている言語の中で最も古いため、理解できます。しかし、この記事は言語の特定の評価ではありません。これは、プログラミング言語とソフトウェア開発の世界を変えたコンピューターアートの達人をすべての読者に思い出させるために設計された資料です。

1. Java-James Gosling



James Gosling

Javaは、最も人気があり成功しているプログラミング言語の1つです。JavaはJames Arthur Gosling博士によって作成されました。彼は「Javaの父」として知られています。

当初、Sun MicrosystemsはJavaの開発とサポートに関与していました。2010年1月以降、SunがOracleを買収した後、Oracleはこれを行ってきました。

JavaはWORAの原則に基づいています（「Write Once Run Anywhere」、「Written once、workwhere。」）。Javaのプラットフォーム非依存性は、この言語が企業環境で成功するための基本的な理由の1つになっています。

Javaは現在最も人気のある言語の1つです。Javaについて学びたい場合は、このチュートリアルをご覧ください。

2. C-デニス・リッチー



Dennis Ritchie

アメリカのコンピューター専門家であるDennis Macalaird Ritchieは、1967〜1973年にAT＆T Bell LabsでCプログラミング言語の作成に関与しました。

C言語はまだ非常に人気があります。システムプログラミングで広く使用されています。これはJavaより古いですが、根拠はありません。

ちなみに、デニス・リッチーは、世界的に有名なUnixオペレーティングシステムも作成しました。彼は長年の同僚ケン・トンプソンとこれをしました。

デニス・リッチーの人気とビル・ゲイツやスティーブ・ジョブズの人気を比較すると、ここで比較するものはほとんどないことがわかります。しかし、彼が情報技術の世界にどのような貢献をしたかを彼と比較することはできません。すべてのプログラマはこれについて知っておくべきです。Cを学びたいなら、これを見てください Courseraを専門としています。

3. ++ —



ビョルンStraustrup

ビョルンStraustrupはオーフス、デンマークの都市で1950年12月30日に生まれました。広く使用されているC ++プログラミング言語の作成と開発で知られています。 

C ++は、言語の名前からもわかるように、C言語に続く新世代の言語であり、オブジェクト指向プログラミング機能のサポートによって区別されます。 C ++と比較すると、非常に興味深いものに見えました。

C ++は依然として最も人気のある言語の1つです。たとえば、C ++で記述されたコードはコンピューターのシステム機能にかなり近いため、この言語は現在人気のあるオブジェクト指向機能をサポートしているため、高頻度取引システムの開発で広く使用されています。 。

C ++をゼロから学びたい場合は、このチュートリアルをご覧になることをお勧めします。それは理論と実践をうまく組み合わせています。

4. Python-グイドファンロッサム



グイドファンロッサム

Python言語は、数学およびコンピューターサイエンスセンター（CWI）のグイドファンロッサムによって開発されました。Pythonは、汎用の高級言語です。設計する際には、コードの読みやすさに特に注意が払われました。その構文は、純粋で表現力豊かと見なされます。

米国では、Pythonはほとんど科学コミュニティのJavaに取って代わりました。特に、これは以前のように、現代の学生がプログラミングをCやJavaではなくPythonの学習から学び始めるという事実に変換されます。

Pythonは、さまざまな分野、特にWeb開発や情報セキュリティの分野で広く使用されています。この言語は、Google、Yahoo、Spotifyなどの企業で使用されています。ここに良いPythonチュートリアルがあります。

5. JavaScript-ブレンダンアイヒ



ブレンダン・アイヒ

過去5〜10年間で最も人気のあるプログラミング言語を教えていただければ、これはJavaScriptだと言えます。特に既存のJavaScriptライブラリーが原因で、クライアント側のWeb開発の世界を支配しています。さらに、Node.jsプラットフォームのおかげで、サーバー環境で非常に人気があります。

JavaScriptは、プロトタイプ継承メカニズムを使用するスクリプト言語です。これは動的で弱く型付けされた言語であり、高階関数の操作をサポートしています。 JavaScriptは、Netscape Communications Corporationでこの言語の開発中に働いたBrendan Eichによって作成されました。

JavaScriptコードはブラウザーで実行でき、最新のWebインターフェースを動かすすべてのものはJavaScriptで書かれています。JavaScriptは、ほとんどすべての深刻なサイトで使用されています。

JavaScriptを学びたい場合-あなたのサービスで膨大な量の資料。

6. PHP-ラスムスレルドルフ



Rasmus Lerdorf

PHP言語（元々Personal Home Page Toolsと呼ばれ、「個人のWebページを作成するためのツール」という意味）は、1995年にRasmus Lerdorfによって作成されました。この言語をいかに嫌いかに関係なく、最新のWebプロジェクトのサーバー部分の開発で非常に人気があるという事実から逃れることはできません。

PHPリファレンス実装は現在、PHPグループによって開発されています。PHPは、Microsoft Active Server Pages（ASP）およびJava Server Pages（JSP）テクノロジと競合し、その結果、それらよりもはるかに要求が厳しくなりました。Facebook、Wikipedia、Wordpress、Joomlaなどのインターネット大手が使用するオープンソース言語です。

PHPについて学びたい場合は、こちらをご覧ください。

7. Perl-ラリーウォール



Larry Wall

Perlは、高水準で動的なインタープリター型の汎用プログラミング言語です。1980年代半ばにラリーウォールによって作成されました。Perlの人気は、その驚くべきワープロ機能にあります。

Perlは、UNIXシステムでスクリプトを開発するために使用される主な言語です。Perlは、コンピュータグラフィックスの分野、データベースアプリケーション、ネットワークプログラミングの分野で使用されています。この言語を使用する企業には、IMDB、Amazon、Pricelineがあります。これは優れたPerlチュートリアルです。

8. Ruby-松本幸宏



松本幸宏

Ruby言語は、1990年代半ばに日本の松本幸宏によって作成されました。Rubyプログラミングは楽しいです。Ruby on Rails Webフレームワークを試してみたら、私を理解できるでしょう。

Perl、Ada、Lisp、Smalltalkなどの言語がRubyに影響を与えています。この言語は、その上でプログラムを書くことが迅速で快適であると期待して作成されました。

Rubyは主にWebアプリケーション開発で使用されます。特に、この言語はTwitter、Hulu、Grouponなどの企業で使用されています。

これが Rubyのチュートリアルです。

9. Lisp-ジョンマッカーシー



John McCarthy

Lisp言語（LISt処理言語）はJohn McCarthyによって開発されました。今日でも使用されている最も古い高級言語の1つです。

私はLispを研究したことがありませんが、この言語はHaskell、Erlang、Scalaなどの関数型プログラミング言語の祖先であると言われています。この言語はさまざまな分野で使用されています。特に、軍隊では。Lispのための多くのトレーニングコースはありません。ここにその一つがあります。

10.パスカル-ニクラウスワース



Nicklaus Wirth

Pascalは、1960年代後半にNicklaus Wirthによって開発された命令型手続き言語です。この言語は、IT業界に大きな影響を与えました。これは、開発への構造的アプローチを使用して優れたプログラミングスタイルを維持することを目的とした、コンパクトで効率的な言語として作成されました。Lispと同様に、Pascalには多くのトレーニングコースはありません。しかし、言語はかなり古いです。ただし、学習したい場合は、こちらをご覧ください。
