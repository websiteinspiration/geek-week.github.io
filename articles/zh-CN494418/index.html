英特尔重置处理器型号。但这还不确定-将进行投票

英特尔在2008年发布了首批标记为Core i3 / i5 / i7的处理器和一个数字索引。众所周知，现在有第十代Intel Core和型号，其开头是10。尽管这种编号具有逻辑性质，但它有一个隐藏的陷阱：随着索引的增长，它越来越多地与其他表示某些内容的数字混合在一起，并且不再被视为处理器的属性。比方说，Core 9400型号对我们来说很熟悉，Core 12800已经类似于电池，而21280通常是VAZ经典产品中的东西。为了使该指数在心理上让消费者感到满意，英特尔正在考虑选择每10代重置一次数字。

但是，这种想法有明显的缺点。首先，计算机技术已经有12年了-虽然时间很长，但是还没有超过。在运行中和二级市场上，有许多第一代Intel Core处理器，带有可爱的老式短语：Intel Core i7-860，Intel Core i5-750等。英特尔的新举措会导致混乱和混乱吗？



为了不打断肩膀，在采取这种严厉措施之前，该公司决定咨询其客户。最终决定之前，将在英特尔网站上进行全球调查，任何有Internet访问权限的人都可以参加。该调查原定于4月20日开始，但是在目前的情况下还不能确定。关注英特尔新闻！
