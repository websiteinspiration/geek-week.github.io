React Russia 2019カンファレンスはすでに6月1日です

友達、春がまたやってきた！美しい夏の天候が近づいています。生産的な日常生活は新しい結果をもたらしました。つまり、再び会う時間です！

今年は特別なものを用意しました。今回は平日の夕方ミタップ（ミニ会議）だけでなく、終日と土曜日に会います。はい、そうです。昨年お約束したことから始めます。会いましょう！

React Russia 2019は、ロシアでReact.jsとReact Nativeを使用するプロの開発者による会議です。React MoscowとReact SPBのすべての会議の論理的な続きとして2016年から開催され、市場の最大の企業のサポートを得て、アイデアを共有し、新しいソリューションとテクノロジーについて発表し、話し合い、経験を共有し、共同コミュニケーションによって、仕事の生産性、楽しさ、効率をさらに高めています。
