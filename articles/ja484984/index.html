Unityゲーム開発者。OTUSからの新しいコース

注意！この記事はエンジニアリングではなく、Unityでのゲーム開発の分野の教育に関心のある読者を対象としています。ほとんどの場合、学習に関心がない場合、この資料はあなたにとって興味がありません。



子供の頃、母親が朝までコンピュータゲームをしたことで私たちを叱ったことを覚えていますか。そして、私たちの多くは単に遊ぶだけでなく、世界で最もクールなゲームを開発し、それでお金を稼ぐことを夢見ていました。今、子供の頃の夢が叶いました -OtusはUnityでのゲーム開発に関するトレーニングコースを開始しました。

このコースでは、アイデア自体からすぐに実装されるまで、ゲーム開発のすべての段階を学びます。ここでは、ゲームのデザインとローカリゼーション、オンライン無料プレイゲームの収益化について学び、プレーヤーを引き付けて維持する方法を学びます。

コースの一環として、最大3つのゲームを作成できます。それぞれ個別に検討してみましょう

。1. RAID Shadow Legends（モバイルゲーム）のような段階的な3Dバトル。

  • 簡単なアニメーションを作成する
  • Unity UI
  • パーティクルシステム
  • 音響再生など

2. 2Dプラットフォーマーランナー。（モバイルゲーム）

  • サイレントマップ
  • 2D物理学
  • 画面上のジョイスティックとジョイスティック
  • Cinemachineらによるカメラ制御

3. ネットワークゲームと「バトルロワイヤル」（PC用）のメカニズムを備えた3Dアリーナブレイバー。

  • マルチプレイヤーゲームの作成
  • マスター3D物理学と地形
  • ナビゲーションメッシュ
  • Photon Unity Networking et al。

Unityでのゲーム開発と は何かを知るために、講師Nikolai Zapolnovが実施したオープンウェビナー「ゾンビシューターインアアワー」を視聴できます。


このウェビナーでは実際の実際のケースについて説明し、教師は生徒からの質問に答え、Otusでのトレーニングの形式を示します。

Nikolai Zapolnovは、ゲーム開発の分野で素晴らしい経験と深い知識を持つ教師です。彼はCut the Rope 2、The Last Camp、Phoenix Rangers：Puzzle RPG、Smash Leagueに貢献し、Rovio（Angry Birdsの作成者）のシニアゲーム開発者です。

1月27日20:00に、ニコライは「Unity Game Developer」というトピックに関するオープンデーを開催します。その間、彼はコースプログラム、オンライントレーニング形式の機能、コース参加者が開発中に習得できるスキルと能力について詳しく話します。学習。

Unity Games Developerコースで学ぶにはどのような知識が必要ですか？

これで十分です：

  • プログラミングの基礎
  • OOPの原則
  • コンピュータ代数と線形代数の基礎知識

また、次の知識がある場合にも役立ちます。

  • Cでの経験＃
  • Unityでの経験

あなたのスキルがトレーニングに十分かどうかを理解するには、入学試験を受けることができます。

このコースに興味のある方は、 1月27日に開催される公開日に、より詳しい情報が得られることを思い出します。またね！
