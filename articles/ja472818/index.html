研究の基礎


マルクスによると、アリは常にワーカホリズム、君主制、社会の階級分割に関連付けられています。アリの家族は南極大陸を除いて地球の隅々に住んでいる約14,000種を持っています。地球上のすべてのアリを数えると、陸生動物のバイオマスの約10〜25％を占めます。アリの成功は、変化する環境条件と信じられないほどの組織に適応する能力に基づいています。


帝国のアリのドキュメンタリー（BBC、デビッドアッテンボロー、2018年）。

アリが信じられないほどのコロニーを構築し、組織化された列に移動し、止まらないように見える方法を見ると、アリのそれぞれが単一の生物の粒子であるという印象を受けます-集合的な心。それが科学者によるこれらの昆虫への関心が衰えない理由です。

葉っぱカッターは、噛んだ葉の基質にキノコを栽培する農場を構築します。

アリはある時点（蟻塚）に住んでいるため、多くの旅行をしなければなりません。アリが大量の食料源を見つけると、それは家から食料までの化学経路を切り開き、その後、その親族がたどることができます。磁束密度は蟻塚の個体数に強く依存し、1分あたり数百の蟻に達する可能性があります。この場合、交通渋滞、事故、電話保険会社はありません。流れは絶え間なく移動し、アリは効率的に職務を遂行し続けます（労働者は食料を手に入れ、兵士は彼らを守ります）。

研究者は、道路建設において、人々の密度（k）と流れ（q = vk、つまり速度×密度）は、基本図（1A）を使用して記述されることがよくあります。

検討中のシステムによって、速度密度図と流れ密度図の間に違いがありますが、基本的にそれらには共通の特徴があります。

最初に、フローqは密度kとともにゼロから最大値まで増加し、その後、いわゆる最大マッシュ密度kjでゼロに戻るまで減衰します。磁束密度曲線は、原則として、最大の磁束または存在量が達成される方法で、kの最適値を持つ凹型の形状を持っています。

第二に、個人の速度は、一人で移動する場合（自由流速vf）が最大であり、密度kの増加に伴って減少します。マッシュ密度がv（kj）= 0に達すると、速度はゼロに低下します。運動のすべての参加者が停止します。

同様の計算方法がアリに繰り返し適用されています。たとえば、葉を切るアリや火のアリでは、密度が増すにつれて移動速度が低下しますが、森林や遊牧のアリでは、密度が増すにつれて速度は一定のままです。

最も高い密度、ならびに葉を刈るアリ、木および遊牧のアリによって記録された推定雇用（アリによって覆われた領域の割合）は比較的低かった：0.8 / cm2（雇用0.20）、0.6 / cm 2（0.13）および0.3 / cm 2（0.10）。このような指標はコルクを形成するのに十分な高さではありません。蟻は移動経路の境界容量を決して超えなかったため、経路の幅に対応する最大許容フロー値に付着しました。

今日検討している研究では、科学者たちはアリが異なる磁束密度でプラグの形成を回避できるかどうかを確認することを決定しました。主な登場人物は種のアリ（Linepithema humile-アルゼンチンのアリ）でした。この種は、家族の中で最も数が多く普及している種の1つです。

橋を介して食品の場所に接続されているアリのコロニー（1B）、その幅が変更され（5、10、20 mm）、磁束密度の操作が可能になりました。異なる数のいくつかのコロニーが実験に参加しました：400から25,600アリ。合計170の実験観測が行われ、その間、1秒あたりの流れと密度が固定されました。橋の幅をコロニーから食物に変更できることで、フラックス密度（0〜18アリ/ cm 2）と雇用（0〜0.8）のさまざまな指標を得ることができました。

実験結果


170の実験すべてのデータを分析する前に、科学者はこれらのデータが歪んでいないことを確認しました。第一に、食物ゾーンに入るアリの数は摂食行動に影響を及ぼさないことがわかった。ほとんどのアリは一度食べると、負のフィードバックの存在がなくなります。これは、摂食場所での個体の集中が原因である可能性があります。次に、科学者たちは橋の幅がアリの速度に影響を及ぼさないように制御しました。相互作用がない場合、およびアリが一人で移動する場合、それらの速度は橋の幅に関係なく同じでした。

最初のステップは、巨視的なレベルでアリの動きを研究することでした。 Antフローq両方向への移動は、密度の関数として図2Aに示されています。磁束qは、密度kとともに特定のポイントまで増加し、その後一定のままでした。

グラフ2Bは、所定のルートに沿ったアリの動きの3つの異なるマクロ関数を使用して実行された、kとqの関係の分析を示しています。すべての関数パラメーターは、最小二乗法*を使用して選択されました。
最小二乗法*は、目的の変数からの一部の関数の偏差の二乗の合計を最小化することに基づく数学的な方法です。
実験データに基づいて、2相フロー関数が作成され、q - k比を、線形に増加するフローと、マッシュ密度に達した瞬間のフローの定数値を持つ区分的線形関数*として記述しました。
区分的線形関数*は、一連の実数で定義される関数であり、定義のドメインを構成する各区間で線形です。
二相流関数は次のとおりです。
q（k）= kV（k≤kjの場合）
そして
q（k）= kjv（k﹥ kjの場合）
次に、統計モデルが選択され、すべての統計モデルに条件付き確率を割り当てることが可能になりました。大量のデータにより、明確な結果（2フェーズ統計モデル（2C））を得ることが可能になりました。

したがって、なぜ蟻が交通渋滞に巻き込まれないのかという質問への答えは、高密度での時空間フロー構成である可能性があります。

アリの両方のレーンが完全に交差せず、空間で分離されている場合、ストリームは空間的に編成されたと呼ばれます。一時的な組織化は、振動の変化が移動方向に発生すると発生し、そこから流れは周期的に一方向になります。動きの方向が変わります。

運動の組織化のどちらの場合でも、蟻間の衝突の制限（衝突）があるため、蟻は中断のない流れを維持できます。

しかし、科学者が行った実験では、このような組織は見られませんでした。アリの密度が臨界しきい値に達すると、入力フローと出力フローが時間と空間の両方で混合されます（3A）。

アリは5 mmの厚さの橋に沿って移動します。このビデオは、生息地ゾーンと食物ゾーンの間の接続を確立してから10分後のアリの動きをキャプチャします。

全体としての流れの組織化の側面のあいまいさを考慮して、研究者たちは、Hercule Poirotの足跡をたどる、つまり詳細、つまり流れの中の個々のアリの行動を慎重に検討することにしました。

個人の行動の観点から見ると、移動するストリームのほとんどの機能は、個人間の「摩擦」により、個人の速度が密度とともに非線形的に低下することを示唆しています。

ただし、2相フローモーション関数は、密度が1 cm 2あたり8アリ未満の場合、アリ間のこのような摩擦は検出されなかったことを示唆しています。、つまり、流れは直線的に増加しました。密度が8を超えると、摩擦が発生しましたが、密度とともに直線的に増加しました。流れは広範囲の密度にわたって一定のままでした。したがって、発生する摩擦を何らかの方法で測定して分析する必要があります。

単一のアリの速度に影響を与える最も重要な要素は、その親族との接触（衝突）の数です。これにより、アリが停止し、全体的な速度が低下します。

接点の数が密度と速度をつなぐ隠された変数の役割を果たすかどうかを確認するために、科学者は、接点の数C、密度k、および通過時間Tを測定する実験を行いました長さ2 cmの橋7900個体のコロニーからのすべての個々のアリのデータが考慮されました。

密度kが増加するにつれて、接点の数Cは直線的に増加しました（C = 0.61 k、4A）。密度が高いほど、より多くの接触が観察されました。移動時間Tに対する接触数Cの線形効果も見つかりました。各接触は実際にアリを遅くしました（N = T 0 + C・∆ T、ここでT 0 = 0.95 sおよび∆ T = 0.24 s、4 V）。変数t0は、接点のないブリッジを通過するのにかかる時間を表示します。∆ Tは、接点が原因で失われた時間です。

中間的な結論は、密度が流量にマイナスの影響を与えるということでした。密度は個人間の接触数を増加させ、 ΔTを増加させ、結果として橋を通過するのに必要な時間を増加させました。これは非常に論理的ですが、フェーズ1では、状況はもう少し興味深いものでした。

2相運動の図では、密度 kは速度 vに影響を与えませんでした。したがって、このフェーズでは、速度 vに対する密度 kの正の影響があるはずです。したがって、 T間の接続、kとCははるかに非標準です。

複数の効果を組み合わせるために、密度kと接点の数Cに応じて、予想される移動時間Tを推定しました。科学者によって確立された接触の数Cについて、平均通過時間T（5A）が異なる密度値kに対して計算されました。画像№5 隣接する曲線間の垂直距離は、値ΔTを決定しました。予想どおり、通過時間Tはコンタクト数Cの増加に伴って増加しました




しかし、好奇心の観察は、実際の走行時間の減少につながったその当初密度KたT（TO kは ≈5）。

この正の密度効果をさらに確認するために、自由流速度vfを推定しました。接点なし（5V）：vf = L /（T - C・∆ T）、ここでL = 2 cm（観測が行われた橋の部分）vf

インデックスは最初、1 cm 2あたり最大5アリの密度で増加します。、元の値に戻ります。そのような変動の説明は数学ではなく、生物学にあります。アルゼンチンのアリはフェロモンを使用します。フェロモンは、道に迷わないようにマークし、親類が食べ物や家に行く道を示します。

すべての効果を組み合わせて、科学者はアリの速度の独自の式を提案しました：
v（k）= [ L / T 0 + ∆ T・C（k）]・（⍺+β・k・e - γ・k）
C（k）は0.61 kに等しいコンタクトの平均数です。
⍺、β、γ-フェロモン効果をシミュレートします：⍺-フェロモンでマークされていない橋の内部の魅力に対応します。 β-プラスの効果kを表します。 γはフェロモン効果が発生する範囲です。これらの3つのパラメーターは、非線形回帰アルゴリズムを使用して推定されました：⍺= 0.812±0.009、β= 0.160±0.010、γ= 0.156±0.007。

増加に伴い、速度v（6A）の減少が観察されました。この速度ダイナミクスでは、流れ全体に次の式が割り当てられました。
q(k) = k · v(k)
グラフ6Bでは、予測されたフラックスqが作成され、その推定値として、実験観測のデータが使用されました（N = 7900観測）。モデルと実験データの間に明確な一致が見つかりました。関数定数で達成されたQ ≈センチメートル当たり10匹のアリ2秒。

密度kが増加するにつれて、移動時間Tを増加させる接触が多くなり、フラックスqに悪影響を及ぼしました。密度が<5アリ/ cm 2の密度で個体が速く移動し、フラックスに正の影響を与えました。

これらの2つの効果はバランスが取れており、密度kでの磁束qの線形増加につながります（フェーズ1）。密度が8アリ/ cm 2を超えると、ルート（橋）のオーバーフローにもかかわらず、アリは一定の流れqを維持しました。接触の増加により速度v（k）は減少し続けましたが、磁束q（k）へのこの悪影響はkの増加によって補償されました。 つまり、グラフ6Bのフローは、高密度kで増加します。

アリのいない橋の面積が密度の増加に伴って減少したことを考えると、接触の数が密度に比例して直線的に増加したことは驚くべきことです。

エピローグ
