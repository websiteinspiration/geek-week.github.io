目的：ランダム性の改ざんの可能性とランダム性の改ざんを克服する現実を証明すること。

積分対数スペクトルに分割された自然なランダム性は、短い繰り返し距離と長い繰り返し距離が大量に存在しないことを予測可能に示しますが、シーケンスをソートすると、分離スペクトルが理論的な形になります。

ロシアとソ連とCISの啓蒙のギャップ：低学年の積分と対数は研究されておらず、その後、最も単純でおそらく複雑であると見なされます。

datasov.basおよびdatasov.csプログラムは、反転配列をソートすることにより、元の配列の要素を再配置します。

Qbasic順列プログラム

' datasov.bas
DIM a(55000), d(55000)
OPEN "aa.txt" FOR INPUT AS #1
OPEN "dd.txt" FOR OUTPUT AS #2

FOR i = 1 TO 55000
    INPUT #1, a(i): d(55000 - i + 1) = a(i):NEXT

FOR i = 1 TO 54999: FOR j = i TO 55000
        IF d(i) > d(j) THEN SWAP d(i), d(j): SWAP a(i), a(j)
NEXT: NEXT

FOR i = 1 TO 55000: PRINT #2, a(i): NEXT: CLOSE
