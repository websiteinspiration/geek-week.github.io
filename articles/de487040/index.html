Firefox Preview Nightly für Android unterstützt uBlock Origin

Wie versprochen hat Mozilla die erste Erweiterung, uBlock Origin, vom empfohlenen Erweiterungsprogramm auf die nächtliche Version von Firefox Preview (Codename „Fenix“) verschoben . Sie können es bereits installieren und das Blockieren von Anzeigen in der nächsten Version von Firefox für Android aktivieren.

Mozilla hat kürzlich Firefox Preview v3.0 mit neuen Funktionen und Verbesserungen veröffentlicht. Nach Benutzeranfragen Erweiterung Unterstützung hinzufügen, kündigte das Unternehmen auf seinem Blog , dass Erweiterung Unterstützung in Firefox Vorschau im Januar 2020 erscheinen wird, und dass es funktioniert erste Erweiterungen auf ihre Liste der empfohlenen Aufbauprogramme hinzuzufügen.

„Es wird nicht über Nacht geschehen, GeckoView und Firefox Preview mit den APIs in Einklang zu bringen, die zuvor in Firefox für Android unterstützt wurden. Für den Rest des Jahres 2019 und bis 2020 konzentrieren wir uns auf die Unterstützung benutzerdefinierter Erweiterungen aus unserem empfohlenen Erweiterungsprogramm, die stabil auf Mobilgeräten funktionieren und viele Funktionen unterstützen “, sagte Phillip Quish.

Firefox Preview Nightly, basierend auf GeckoView, unterstützt jetzt WebExtensions, und uBlock Origin ist die erste Erweiterung, die nach dem Update hinzugefügt wurde.

So installieren Sie uBlock Origin in Firefox Preview Nightly:

  1. Laden Sie Firefox Preview Nightly aus dem Play Store herunter und installieren Sie es .
  2. Starten Sie Firefox Preview Nightly, klicken Sie auf das Dreipunktmenü und gehen Sie zu Einstellungen.
  3. Geben Sie im Abschnitt "Erweitert" die "Add-Ons" ein, die über dem Abschnitt "Sprachen" angezeigt werden.
  4. Sie sehen uBlock Origin als empfohlenes Add-On. Klicken Sie zum Installieren auf das Symbol „+“.



Sie können uBO steuern, indem Sie zu den Einstellungen gehen. Nach der Installation von uBO wird die Schaltfläche Add-ons Manager im Menü Firefox Preview Nightly angezeigt. In naher Zukunft können Sie weitere Erweiterungen für die Installation im Abschnitt "Empfohlen" in Firefox Preview Nightly erwarten.

Seit Sommer 2019 versucht Mozilla, die Funktionalität von Firefox der nächsten Generation für Android mit der vorhandenen Version zu vergleichen. Denn eine der wichtigsten Funktionen, die Firefox Preview fehlte, war die Unterstützung von Erweiterungen.

Während Mozilla an einem neuen Browser arbeitet, hat sich die stabile Version von Firefox für Android für immer auf Version 68 festgelegt und erhält nur Spot-Updates. Normale Benutzer können die stabile Version von Firefox bis zur Veröffentlichung von Firefox Preview weiter verwenden, das irgendwann in der ersten Hälfte dieses Jahres zum Hauptbrowser wird.

Mit dem Firefox Preview Nightly auf dem Play Store zweimal täglich aktualisiert Master - Zweig - Code .
