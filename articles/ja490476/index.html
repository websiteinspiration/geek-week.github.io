高度な機能を備えた開閉用ワイヤレスセンサー

私は、Habrのすべての読者、特に「DIYまたはDo it own」セクションの読者を歓迎します。そして、私がそのようなものを思い付くことができないなら、私はアルドゥイーノです、私はできます...エレベーターの運転台の管理の主要なトピックに触れません:)ちょっと考えて、なぜか開閉センサーを作りたかったんです。このセンサーは、私が最近行っている他の工芸品と同様に、Nordic Semiconductorのチップに基づいています。センサーは2つのバージョンで行うことを決定しました。1つはnRF52840チップ、もう1つはnRF52811チップです。



バージョンにはnRF52840チップモジュールE73_2G4M08S1C社EBYTEが使用され、verciyaチップモジュールnRF52811 MC50SFA社MiNEWが使用されました。率直に言って、手頃なnRF52811チップの検索はまだ冒険でした。しかし、このデバイスの冒険の結果として、MINEWのnRF52811チップ上のモジュールと、これらのモジュールにはんだ付けされたチップのいくつかのバージョンの形のバンは、nRF52810とnRF52832です。



デバイスの主な機能は、リードスイッチに基づく開閉の検出です。リードスイッチの回路はアンチバウンスで再位相化されています。

この開閉のセンサーの主な機能を希釈することが適切であることを考えて、私はこれについて市場に何があるのか​​を見ることにしました。それはほとんど何も判明しなかったので、開閉センサーはアフリカでも開閉センサーです。最も「高度な」ソリューションはREDMONDで見つかりました。彼らのBLEセンサー（ちなみに、ノルディックのチップにも搭載）には、リードスイッチに加えて、温度センサーと静電容量ボタンがTTP223チップに実装されています。しかし、何らかの理由で、これはあまり良い解決策ではないように思われました。ドアまたは窓の近くの温度測定値（およびチップによる測定を妨げたもの）の有用性と、窓またはドアにぶら下がっているセンサーのボタンを使用することが適切な状況（まあ、入力以外はそうです:)）。その結果、センサーのセキュリティ機能を拡張することにしました。



このセンサーにCR2032バッテリーを使用することが決定されたため、主な選択基準は追加センサーの消費でした。候補の中の勝者は、LIS2DW12加速度計とDRV5032FB磁場センサーの2つのセンサーでした。

LIS2DW12は現在、おそらく最も経済的な加速度計です。低消費モードでは、この加速度計は1μAを消費します（データシート）。 DRV5032FB磁場センサーも優れた消費特性を示しました。その消費量は約500nAです（データシート）。

加速度センサーは、衝撃センサーモードで使用することを決定し、磁界センサーはその目的に使用します。衝撃センサーの機能について落ち着いていたのであれば、磁場センサーを使用することは依然として非常に実験的な解決策ですが、温度センサーよりも優れています。



プロジェクトのソフトウェア部分は、Maysensorsネットワークでセンサーを操作するために作成されました。少なくとも今のところは。ノルディックチップ（nRF24（+ atmega 328、stm32f1）、nRF51およびnRF52）を低レベルで処理するバリアントのマイセンサーは、Nordic独自のプロトコル-Enhanced ShockBurst（ESB）を使用して、nRF24およびnRF51-52のデバイスの互換性を確保します。 MaysensorsはオープンなArduinoプロジェクトであり、その周りで、世界の多くの国でかなり大きなコミュニティがすでに形成されています。しかし、nRF52チップの優れたソリューションは、メイセンサー（ESB）を使用する必要がないことです。チップはマルチプロトコルであるため、ZigbeeまたはBLEプロトコルに基づくソフトウェアを単に置き換えるだけで十分です。BLEについて少し説明しますが、E73_2G4M08S1Cモジュールから素晴らしいArduino NANO 33 Bleを作成できることを確認します。私のNANO 33のコストは4ドルです。



センサーのスケッチはArduinoで作成されました。追加ライブラリのIDEがLIS2DW12加速度計のライブラリを使用しました。私はデフォルトのレジスタ設定に関して少し変更しました。私のバージョンでは、最小の消費電力オプションの設定ですぐに機能します（私のgitで利用可能）。

プログラムのロジックについて説明します。メイン操作モードでは、センサーは、4つの割り込みのみが設定された外部割り込みが設定された夢の中にあります。割り込みには2つの構成があります。configsは、リードスイッチの状態に応じて、プログラム操作中に割り込みを再構成します。ドアが開いている場合、衝撃センサーと磁場センサーの割り込みは無効になります。ドアが閉じるとすぐに、これらのセンサーの2つの割り込みがアクティブになります。また、開扉時に、リードスイッチの前にショックセンサーが作動し、開錠時の振動が原因であるという事実にも遭遇しました。この問題は、加速度計の構成された高感度でのみ記録されました。

この問題を解消するために、加速度計がトリガーされたときに2秒の待機が導入され、その間にリードスイッチのピンが監視されました。待機中にリードスイッチのピンでレベル変更が発生した場合、加速度計からの割り込みによるイベントの以降の処理が停止し、リードスイッチからのイベントの処理が開始されます。

センサーには設定モードがあります。サービスボタンが押されると、センサーは中断時に起動し、無線モジュールはリスニングモードに入り、UDコントローラーからのコマンドの受信を待ちます。コマンドが受信されると、センサーは新しい値をメモリに書き込み、すぐにスリープ状態になる動作モードに切り替わります。次のコマンドを送信するには、構成モードのアクティブ化を繰り返す必要があります。設定モードでセンサーが30秒以内に何も受信しない場合は、この時間後に動作モードに切り替わり、スリープ状態になります。設定モードに加えて、サービスボタンからセンサーセンサーの提示と工場出荷時の状態へのリセットを開始できます（センサーはセンサーがセンサーを追加したネットワークを忘れ、リセット後のセンサーの登録を再度行う必要があります）。

void loop() {
  if (flag_update_transport_param == 1) {
    update_Happy_transport();
  }
  if (flag_sendRoute_parent == 1) {
    present_only_parent();
  }
  if (isTransportReady() == true) {
    if (flag_nogateway_mode == 0) {
      if (flag_find_parent_process == 1) {
        find_parent_process();
      }
      if (configMode == 0) {
        if ((axelIntStatus == AXEL_INT) || (buttIntStatus == PIN_BUTTON) || (gerkIntStatus == GERKON_INT) || (magIntStatus == MAGNET_INT)) {
          nosleep = 1;
          newmillis = millis();
          interrupt_time = newmillis - oldmillis;
          BATT_TIME = BATT_TIME - interrupt_time;
          if (BATT_TIME < 60000) {
            BATT_TIME = SLEEP_TIME;
            batteryVoltage = hwCPUVoltage();
            batt_flag = 1;
          }

          if (gerkIntStatus == GERKON_INT) {
            send_Gerkon();
            axel_time = millis();
            nosleep = 0;
          }

          if (magIntStatus == MAGNET_INT) {
            send_Magnet();
            nosleep = 0;
          }

          if (axelIntStatus == AXEL_INT) {
            if (millis() - axel_time0 >= 2000) {
              send_Axel();
              nosleep = 0;
            } else {
              if (digitalRead(GERKON_INT) == LOW) {
                send_Gerkon();
                axel_time = millis();
                nosleep = 0;
              }
            }
          }

          if (buttIntStatus == PIN_BUTTON) {
            if (digitalRead(PIN_BUTTON) == 0 && button_flag == 0) {
              button_flag = 1;
              previousMillis = millis();
              ledsOff();
            }
            if (digitalRead(PIN_BUTTON) == 0 && button_flag == 1) {
              if ((millis() - previousMillis > 0) && (millis() - previousMillis <= 1750)) {
                if (millis() - lightMillisR > 70) {
                  lightMillisR = millis();
                  onoff = !onoff;
                  digitalWrite(BLUE_LED, onoff);
                }
              }
              if ((millis() - previousMillis > 1750) && (millis() - previousMillis <= 2000)) {
                ledsOff();
              }
              if ((millis() - previousMillis > 2000) && (millis() - previousMillis <= 3750)) {
                if (millis() - lightMillisR > 50) {
                  lightMillisR = millis();
                  onoff = !onoff;
                  digitalWrite(GREEN_LED, onoff);
                }
              }
              if ((millis() - previousMillis > 3750) && (millis() - previousMillis <= 4000)) {
                ledsOff();
              }
              if ((millis() - previousMillis > 4000) && (millis() - previousMillis <= 5750)) {
                if (millis() - lightMillisR > 30) {
                  lightMillisR = millis();
                  onoff = !onoff;
                  digitalWrite(RED_LED, onoff);
                }
              }
              if (millis() - previousMillis > 5750) {
                ledsOff();
              }
            }

            if (digitalRead(PIN_BUTTON) == 1 && button_flag == 1) {
              if ((millis() - previousMillis <= 1750) && (button_flag == 1))
              {
                ledsOff();
                blinky(2, 2, BLUE_LED);
                button_flag = 0;
                buttIntStatus = 0;
                presentation();
                nosleep = 0;
              }
              if ((millis() - previousMillis > 2000) && (millis() - previousMillis <= 3750) && (button_flag == 1))
              {
                ledsOff();
                blinky(2, 2, GREEN_LED);
                configMode = 1;
                button_flag = 0;
                configMillis = millis();
                interrupt_Init(1);
                NRF_POWER->DCDCEN = 0;
                buttIntStatus = 0;
                NRF5_ESB_startListening();
                wait(50);
              }

              if ((millis() - previousMillis > 4000) && (millis() - previousMillis <= 5750) && (button_flag == 1))
              {
                ledsOff();
                blinky(3, 3, RED_LED);
                //new_device();
              }

              if ((((millis() - previousMillis > 1750) && (millis() - previousMillis <= 2000)) || ((millis() - previousMillis > 3750) && (millis() - previousMillis <= 4000)) || ((millis() - previousMillis > 5750))) && (button_flag == 1))
              {
                ledsOff();
                nosleep = 0;
                button_flag = 0;
                buttIntStatus = 0;
              }
            }
          }
        } else {
          batteryVoltage = hwCPUVoltage();
          BATT_TIME = SLEEP_TIME;
          sendBatteryStatus(1);
          nosleep = 0;
        }
      } else {
        if (millis() - configMillis > 30000) {
          blinky(3, 3, GREEN_LED);
          configMode = 0;
          nosleep = 0;
          interrupt_Init(0);
          NRF_POWER->DCDCEN = 1;
          wait(50);
        }
      }
    } else {
      if (buttIntStatus == PIN_BUTTON) {
        if (digitalRead(PIN_BUTTON) == 0 && button_flag == 0) {
          button_flag = 1;
          nosleep = 1;
          previousMillis = millis();
          ledsOff();
        }
        if (digitalRead(PIN_BUTTON) == 0 && button_flag == 1) {
          if ((millis() - previousMillis > 0) && (millis() - previousMillis <= 1750)) {
            if (millis() - lightMillisR > 25) {
              lightMillisR = millis();
              onoff = !onoff;
              digitalWrite(GREEN_LED, onoff);
            }
          }
          if ((millis() - previousMillis > 1750) && (millis() - previousMillis <= 2000)) {
            ledsOff();
          }
          if ((millis() - previousMillis > 2000) && (millis() - previousMillis <= 4000)) {
            if (millis() - lightMillisR > 25) {
              lightMillisR = millis();
              onoff = !onoff;
              digitalWrite(RED_LED, onoff);
            }
          }
          if (millis() - previousMillis > 4000) {
            ledsOff();
          }
        }

        if (digitalRead(PIN_BUTTON) == 1 && button_flag == 1) {
          if ((millis() - previousMillis <= 1750) && (button_flag == 1))
          {
            ledsOff();
            blinky(2, 2, BLUE_LED);
            button_flag = 0;
            buttIntStatus = 0;
            check_parent();
            nosleep = 0;
          }
          if ((millis() - previousMillis > 2000) && (millis() - previousMillis <= 4000) && (button_flag == 1))
          {
            ledsOff();
            blinky(3, 3, RED_LED);
            //new_device();
          }

          if ((((millis() - previousMillis > 1750) && (millis() - previousMillis <= 2000)) || ((millis() - previousMillis > 4000))) && (button_flag == 1))
          {
            ledsOff();
            nosleep = 0;
            button_flag = 0;
            buttIntStatus = 0;
          }
        }
      } else {
        check_parent();
      }
    }
  }

  if (_transportSM.failureCounter > 0)
  {
    _transportConfig.parentNodeId = loadState(101);
    _transportConfig.nodeId = myid;
    _transportConfig.distanceGW = loadState(103);
    mypar = _transportConfig.parentNodeId;
    nosleep = 0;
    flag_fcount = 1;
    err_delivery_beat = 6;
    happy_node_mode();
    gateway_fail();
  }

  if (nosleep == 0) {
    oldmillis = millis();
    axelIntStatus = 0;
    buttIntStatus = 0;
    gerkIntStatus = 0;
    magIntStatus = 0;
    sleep(SLEEP_TIME_W, false);
    nosleep = 1;
  }
}


void blinky(uint8_t pulses, uint8_t repit, uint8_t ledColor) {
  for (int x = 0; x < repit; x++) {
    if (x > 0) {
      wait(150);
    }
    for (int i = 0; i < pulses; i++) {
      if (i > 0) {
        wait(40);
      }
      digitalWrite(ledColor, LOW);
      wait(10);
      digitalWrite(ledColor, HIGH);
    }
  }
}

}

}

    hwWriteConfig(EEPROM_NODE_ID_ADDRESS, 255);

  if (loadState(100) == 0) {

void gpiote_event_handler(uint32_t event_pins_low_to_high, uint32_t event_pins_high_to_low)
{
  MY_HW_RTC->CC[0] = (MY_HW_RTC->COUNTER + 2); // Taken from d0016 example code, ends the sleep delay

  if (PIN_BUTTON_MASK & event_pins_high_to_low) {
    if ((buttIntStatus == 0) && (axelIntStatus == 0) && (gerkIntStatus == 0) && (magIntStatus == 0)) {
      buttIntStatus = PIN_BUTTON;
    }
  }
  if (flag_nogateway_mode == 0) {
    if (AXEL_INT_MASK & event_pins_low_to_high) {
      if ((axelIntStatus == 0) && (buttIntStatus == 0) && (gerkIntStatus == 0) && (magIntStatus == 0) && (door_status == 1)) {
        axelIntStatus = AXEL_INT;
        axel_time0 = millis();
      }
    }
    if ((GERKON_INT_MASK & event_pins_low_to_high) || (GERKON_INT_MASK & event_pins_high_to_low)) {
      if ((axelIntStatus == 0) && (buttIntStatus == 0) && (gerkIntStatus == 0) && (magIntStatus == 0)) {
        gerkIntStatus = GERKON_INT;
      }
    }
    if (MAGNET_INT_MASK & event_pins_high_to_low) {
      if ((axelIntStatus == 0) && (buttIntStatus == 0) && (gerkIntStatus == 0) && (magIntStatus == 0) && (door_status == 1)) {
        magIntStatus = MAGNET_INT;
      }
    }
  }
}

  • すべてのmesensorsデータ型、

もちろん、システムの自然な発展の過程で、以前に追加されたシリアルゲートウェイのサポートに欠陥があり、Majordomoは長い寿命を命じ、現在サポートされていません。マイセンサーについて知る前にこの機能が利用できなくなったため、Majordomoでこのタイプのゲートをテストする機会すらありませんでした。モジュール開発者は2019年9月までにこの機能を再び追加することを約束しましたが、19日の秋が過ぎており、一連のゲートウェイのサポートはまだありません:(。

私のセンサーでは、衝撃センサーと磁界センサーはトリガーされたときにのみユニットを送信しますが、これは小さな問題であることがわかりました。 Simple Devicesモジュールはこれらのタイプのセンサーをサポートしていません。もちろん一般的なセンサーはありますが、設定のカスタマイズは非常に制限されています。センサーを追加するとき、不快な問題は次のユニットがセンサーから来たときに、タイマーで指定された時間間隔の後にゼロがオブジェクトプロパティに書き込まれるように、リバースタイマーを開始しなければならないことでした。しかし、すべてが「ステータス更新」メソッドを介して機能するため、ゼロを記録すると、新しい状態を受信するmeysensorsモジュールがこのデータを含むメッセージをデバイスに送信し、ポイントはゼロになります。最も簡単な解決策は、property1からproperty2への状態が転送され、タイマーが開始されてproperty2にゼロを書き込む新しいメソッドを追加することです。シンプルなデバイスで作成されたオブジェクトは、property2で動作し、Maysensorsモジュールではproperty1で動作します。
