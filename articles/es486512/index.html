Brevemente sobre cómo enseñarle a las impresoras multifunción Brother a escanear en Windows Server



Esto ni siquiera es un artículo, sino una nota sobre cómo usar las funciones de red del escáner Brother MFP en un entorno de Microsoft Windows Server. Se tratará de la instalación completa del controlador del dispositivo. Se considerará un ejemplo con el Brother MFC-L2700DRN . Oficialmente, la función de escaneo solo es compatible con el sistema operativo de escritorio, el sistema operativo del servidor solo puede enviar para imprimir . Durante la búsqueda de una solución, a menudo me encontré con foros con problemas similares de este fabricante en particular. Todo se reduce a usar FTP, SMB o correo electrónico.

Por lo tanto, a continuación se presentará una solución universal.

Parte 1. Modificación del instalador.


Se requiere un paquete de controladores completo para instalar el escáner. Dado que el sistema operativo de destino en mi caso es Windows Server 2016, el instalador será para Windows 10 . Descargar, descomprimir.
A continuación, debe averiguar qué controlador es responsable del escáner. La carpeta install / MFC-L2700DN_series contiene el archivo Brinst_Common.ini . En él encontramos la línea EnableServer = 0 en la sección [Especificación] y cambiamos el valor a 1. Este parámetro deshabilita la verificación de los sistemas del servidor. Guarda el archivo. A continuación, busque la línea ScannerInfName = en la sección [Controlador] . Este controlador será modificado.

El archivo del controlador está en la carpeta install \ driver \ gdi \ 32_64. Es necesario hacer lo siguiente en él: busque la sección [Fabricante] y agregue NTamd64.XY al final de la lista después del punto decimal, donde X e Y corresponden a la versión de su sistema operativo. Puedes consultar aquí . Para Server 2016 corresponde a Windows 10 y es igual a NTamd64.10.0

Parte 2. Servidor e instalación


Una pequeña característica de la instalación es la necesidad de dos reinicios. Esto se debe a que el servidor debe ponerse en "modo de prueba" para deshabilitar la verificación de firma digital de los controladores. Después de la instalación, vuelva a encender todo y funcionará.

Abra la línea de comando o PS (como desee) e ingrese bcdedit -set TESTSIGNING ON , después de lo cual reiniciaremos el sistema.

De la manera estándar, ejecute el instalador de Brother en install \ MFC-L2700DN_series \ setup.exe PERO! no instale el controlador de impresora de red, de lo contrario, el instalador arrojará un error y tendrá que eliminar manualmente el dispositivo desconocido (se instala agregando una impresora). Los artículos restantes a gusto.



Una vez completada la instalación, abra la línea de comando o PS e ingresebcdedit -set TESTSIGNING OFF , luego reinicia el sistema. Comprobando el escáner.
