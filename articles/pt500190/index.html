O que é HTML?


HTML significa HyperText Markup Language. Essa é a linguagem de marcação para documentos na World Wide Web (WWW). HTML é uma linguagem padronizada para texto formatado. O navegador interpreta e exibe elementos de páginas da web na tela.

Em que consiste o texto formatado em HTML?


Primeiro de tudo, HTML é conteúdo e tags. As tags permitem especificar como o conteúdo é exibido nas páginas da web.

O que é uma tag HTML?


Uma tag é uma palavra de serviço especial entre colchetes angulares. Também é chamado de "elemento HTML". Existem muitas tags na linguagem HTML e cada uma faz algo com o conteúdo, que geralmente está localizado entre colchetes ou entre as tags.
Se a tag estiver emparelhada, a tag <TEG> corresponderá a </ TEG>.

Todas as tags estão emparelhadas?


Não, não todos. Existem tags únicas, por exemplo, <img> - para imagens. Nesse caso, o conteúdo junto com outras palavras de serviço (por exemplo, para <img> pode ser src = "url da imagem") é colocado entre colchetes, mas após a palavra img.

Quais são os principais tipos de lista usados ​​para criar uma página da web?


Ao criar uma página da web, você pode usar:

  1. Lista de menus (lista de menus - <menu>).
  2. Lista de diretórios (lista de diretórios - <dir>).
  3. Lista ordenada (lista numerada - <ol>).
  4. Lista não ordenada (lista com marcadores - <ul>).
  5. Lista de definições (<dl>).


Por que usar listas em HTML?


As listas tornam as informações mais visuais e fáceis de ler. Além disso, dentro das listas (mais precisamente, para exibição arbitrária de itens da lista), você também pode usar várias tags.

Podemos escrever comentários em HTML? Quão?


Sim, nós podemos. Para fazer isso, o texto do comentário deve ser colocado dentro de <! - e ->.

Por que alguns caracteres em uma página da Web às vezes são exibidos incorretamente?


Na maioria das vezes, os caracteres são exibidos incorretamente devido a problemas de codificação. Essa é a razão - nas configurações do navegador ou servidor da web.

O que é o Image Map?


Em HTML, o Image Map é uma tecnologia que permite vincular links a fragmentos de imagem. Ao clicar em partes individuais da imagem, o usuário pode seguir este ou aquele link para diferentes páginas da web.

O que é espaço em branco?


Espaço em branco é uma propriedade que permite alterar o número de espaços entre as palavras ao exibir uma página da web. Por padrão, o navegador interpreta qualquer número de espaços no código HTML como um espaço.

Qual é a principal vantagem do espaço em branco?


Ao escrever o código HTML, o desenvolvedor não precisa se preocupar com os espaços extras que ele poderia inserir: o navegador removerá todos os espaços extras quando a página da web for exibida.
O desenvolvedor pode adicionar espaços ao código, não apenas por acidente, mas também de propósito: tornar o código mais legível.

Posso atribuir valores a nem todos os atributos de tag?


Sim, nesse caso, o navegador usará os valores padrão. Se você não se sentir confortável com a maneira como ele exibiu a página da web, talvez seja necessário especificar explicitamente os valores de alguns atributos.

Como colocar a marca de direitos autorais em uma página da Web, porque não está no teclado?


O símbolo de direitos autorais refere-se a caracteres especiais. Para colocá-lo em uma página da web, você precisa digitar a sequência e copiar ou & # 169 no teclado no fragmento correspondente do código HTML.

Como criar links para diferentes fragmentos (seções) da mesma página da web?


Para fazer isso, encontre o início do fragmento que precisamos no código HTML. Nesse local, crie a chamada "âncora" usando o atributo name da tag <a> e atribua o link ao nome do fragmento usando o atributo href e o sinal #:
href = "#link_name_to_fragment"

Podemos alinhar itens da lista em um arquivo HTML?


Sim, podemos armazenar itens da lista diretamente no arquivo HTML usando recuos. O recuo também pode ser usado para qualquer lista aninhada dentro do pai.

Qual endereço da página da web é geralmente considerado o endereço do site?


Geralmente, essa é uma das páginas de nível superior, a página principal. A partir dele, o usuário acessa as páginas localizadas em níveis inferiores. E o endereço deles geralmente é formado com base no endereço da página principal.

Por que usamos texto alternativo (atributo alt da tag <img>) para imagens?


O texto alternativo é exibido quando os gráficos são desativados no navegador do usuário ou se houver um problema ao carregar uma imagem específica. Este texto deixa claro o que deveria ter sido representado nesta figura.

Os arquivos HTML podem funcionar bem em algum navegador raro ou de ponta?


Obviamente, os arquivos HTML podem e devem funcionar bem em qualquer navegador, porque o navegador deve estar em conformidade com os padrões HTML. Se os desenvolvedores decidiram não suportar alguns dos recursos do padrão, isso permanece em sua consciência.

Você acha que o hiperlink pode ser apenas texto?


Não, um hiperlink pode ser não apenas texto, mas também, por exemplo, gráfico. Isso significa que podemos converter a imagem em um link que direciona o usuário para outra página. Um hiperlink não depende do tipo de conteúdo "abaixo dele".

Quais são os atributos da tag <li> - um item da lista?


Existem dois deles:

  1. tipo define a aparência de um marcador ou lista de marcadores;
  2. valor - o número a partir do qual a lista numerada começará.


Para que servem as folhas de estilo (CSS)?


As folhas de estilo permitem definir regras claras que determinam a aparência do conteúdo, vários elementos HTML e a página inteira como um todo. Além disso, as mesmas regras podem ser aplicadas a várias páginas da web ao mesmo tempo, o que simplifica a preservação e, se necessário, a alteração do estilo uniforme do site.

Que tipos de numeração nas listas você conhece?


Os seguintes valores podem servir como elementos de numeração:

  • Números arábicos (1, 2, 3, ...);
  • letras maiúsculas latinas (A, B, C, ...);
  • letras latinas minúsculas (a, b, c, ...);
  • números romanos maiúsculos (I, II, III, ...);
  • números romanos minúsculos (i, ii, iii, ...).


Como definir cores diferentes para fragmentos de texto de uma página da web?


Para definir cores diferentes para fragmentos de texto, você precisa colocar cada um dos fragmentos dentro da tag da fonte emparelhada:

<font color=“color”>-    “color”</font>


Onde os códigos de caracteres numéricos são armazenados em HTML?


Na tabela ASCII. Quando uma página da web é exibida, o navegador pega os códigos numéricos do documento HTML e os substitui pelo caractere correspondente da tabela.

Quais são os benefícios de agrupar várias caixas de seleção (elementos da caixa de seleção)?


Existem várias vantagens no agrupamento de sinalizadores:

  • ajuda a melhor organizá-los e estruturá-los;
  • isso permite acessar sinalizadores específicos do grupo usando id;
  • isso permite que você crie outro grupo de sinalizadores na mesma página da web sem medo de confundir os nomes ou identificadores dos sinalizadores.


Como a sobreposição de tags afeta a exibição do conteúdo?


Tags sobrepostas em HTML reconhecem apenas a primeira tag. Mas esses problemas só ocorrem se o navegador tentar reconhecer tags sem texto.

Se não houver texto entre as tags, qual será o resultado? Dê um exemplo?


Se não houver texto entre as tags, será impossível formatar o conteúdo. Por exemplo, tags únicas, como <img>, não requerem entrada de texto e, portanto, a formatação não é necessária neste caso.

Como especificar cores para as bordas da tabela?


Isso pode ser feito na folha de estilo e, se estiver faltando, a cor do texto será usada.

Podemos criar um link que leva a outra página da web?


Sim, você precisa usar o atributo href para isso.

As folhas de estilo podem ajudar a alinhar a imagem e determinar como o texto é posicionado em relação a ela?


Sim, uma folha de estilo pode conter regras para o posicionamento de texto e imagens.

Um único hiperlink pode levar a páginas diferentes?


Não, um hiperlink pode levar apenas a uma página da Web, cujo endereço você fornece.

Existe uma diferença entre uma lista com marcadores e uma lista de diretórios e menus?


Sim, a diferença é que uma lista não ordenada possui atributos para alterar o estilo do marcador.

Como mudar a cor do marcador?


A cor do marcador geralmente corresponde à cor do primeiro caractere da lista. Se você alterar a cor do primeiro caractere usando a tag <font>, a cor do marcador também será alterada.
Se quisermos definir a cor do marcador na lista sem alterar a cor do texto, precisamos colocar a tag <span> dentro do <li> e colocar o texto dentro dele. E a cor do marcador será determinada pela cor da propriedade style do seletor LI e a cor do texto pela propriedade color do seletor SPAN.

É possível introduzir algumas restrições no tamanho dos campos de texto em HTML?


O tamanho padrão é de 20 a 25 caracteres. Mas, usando os atributos size ou maxsize, você pode definir outras restrições. Por exemplo:

<input type="text" size="5" />


Qual é o tamanho máximo de um campo de texto?


A largura da janela do navegador.

O que acontece se você definir o tamanho do campo de texto como zero?


O navegador usará o valor padrão.

Quais são as semelhanças entre os atributos border e rules?


Se o atributo de borda estiver definido como um valor diferente de zero, a borda padrão de uma célula com espessura de um pixel será adicionada automaticamente. Se o atributo de regras for adicionado à tag <table>, o atributo de borda não será incluído.
Mas, ao mesmo tempo em torno de cada célula (o mesmo que borda), um quadro com espessura de um pixel por padrão aparecerá, formando uma grade.

O que é marca de seleção? Como podemos aplicar isso?


O letreiro ajuda a personalizar a rolagem do texto em uma página da web. Para ativar a rolagem, você precisa usar a tag <marquee>.

O que a tag <br> faz?


A função da tag <br> é separar partes do texto uma da outra.

Existe alguma outra maneira de dividir o texto sem usar o <br>?


Sim, existem outras maneiras de dividir o texto. Você pode usar a tag <p> ou a <blockquote>.

O texto pode ser exibido fora da janela do navegador?


Por padrão, o texto não pode ser exibido fora do navegador, mas se o texto fizer parte de uma célula da tabela com uma largura predeterminada, poderá ir além da janela do navegador.

Qual é a diferença entre links ativos e inativos?


O link ativo tem foco, ou seja, o mouse está sobre ele. Em outros casos, o foco está em outro lugar.

A folha de estilo tem restrições sobre o número de regras?


Não, as folhas de estilo não limitam o número de regras para um seletor. No entanto, cada nova regra deve ser separada por ponto e vírgula.

As folhas de estilo têm alguma hierarquia de regras?


Sim, existe uma hierarquia que inclui quatro níveis (da prioridade mais alta à menos prioridade):

  1. Estilo embutido: anexa diretamente ao elemento para o qual o estilo foi criado. Exemplo:
    <h1 style="color: #ffffff;">
  2. Um identificador é um identificador exclusivo para elementos da página, como #navbar.
  3. Classes, atributos e pseudo-classes: essas categorias incluem .class [attribute] e pseudo-classes como: hover ,: focus, etc.
  4. Elementos e pseudoelementos: esta categoria inclui elementos e pseudoelementos, como h1, div ,: before e: after.


Podemos agrupar seletores diferentes com nomes de classes diferentes?


Sim, para isso, basta escrever os nomes das turmas com uma vírgula.

Posso anexar um arquivo CSS externo em um navegador?


Não, um arquivo CSS externo pode ser incluído usando a tag <link /> no próprio documento HTML.

Uma propriedade do tipo estilo de lista pode afetar um parágrafo?


Não, a propriedade do tipo de estilo de lista não pode afetar um parágrafo, provavelmente o ignorará.

O que é um elemento de tela?


O elemento de tela HTML5 pode ser usado para inserir imagens, gradientes e animações complexas. Também pode ajudar a criar imagens 2D diretamente dentro do código-fonte de um documento HTML.
