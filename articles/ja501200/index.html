ディスカッション：172 Tbit / s-マルチコアオプティクスのデータ転送速度の新記録

国立情報通信技術研究所（NICT）の日本人エンジニアによってインストールされました。彼らがそのような結果をどのように達成したかを話し、他の解決策について議論します。


/ Unsplash / Luboミナール

実験の技術的要素


NICTの専門家がテストデータ送信システムを構築しました。これは、光学的に結合されたファイバー（結合コアマルチコアファイバー）を備えた3コア光ファイバーケーブルに基づいていました。その中で、ライトガイドワイヤは互いに非常に近くに配置され（40ミクロン未満の距離）、密度が増加し、構造がより柔軟になります。しかし、このアプローチは追加のクロストークを作成します。光学核の相互影響を制御することにより、システムのスペクトル効率を向上させることができます。この「バインディング」のために、MIMO DSPシグナルハンドラーが受信側にインストールされます。

実験中、NICTからの専門家が生成さ異なる波長の数百のレーザービームと直交振幅変調（16-QAM）を実行しました。次に、60キロメートルのケーブルで数回放送され、2040 kmの距離での情報の伝送をシミュレートしました。信号が最終的なMIMO処理を受けた後。

個々のチャネルのデータ転送速度は400〜550ギガビット/秒でした。同時に、総スループットは172 Tbit / sを超えました。

この数値は、今日展開されている最も「強力な」ケーブルシステムであるパシフィックライトケーブルネットワークの能力を超えています。これは、接続している米国、フィリピン、台湾、香港をし、そのスループットは144テラビット/秒。将来、NICTソリューションはこの指標を超えるでしょう。しかし、日本のエンジニアはこの技術は海底ケーブルには不向きだと言っています。状況は近い将来変わるかもしれませんが。

その他の成果


今年3月には、Nokiaのベル研究所から専門家が設定され、別のレコードを。商用データセンターのネットワークで400 Gb /秒の帯域幅を実現しました。将来、エンジニアは800 Gb /秒の速度を実現したいと考えています。そのため、既存のインフラストラクチャの効率を高め、新しいインフラストラクチャを展開するコストを削減できます。


/ Unsplash / John Adams

昨年、ドイツのエンジニアが500 Gbit /秒の速度でのデータ転送。しかし、彼らはこれを実験室条件ではなく、「戦闘」条件で行いました。ミュンヘン工科大学（TUM）のスペシャリストが、信号コンステレーションの確率的形成のための確率的コンステレーションシェーピング（PCS）アルゴリズムを開発しました。これは、直交振幅変調（QAM）を補完する方法です。アルゴリズムは自動的に最適なポイントグループ（信号振幅値）を選択して、データの歪みやエネルギー消費の可能性を減らします。このテクノロジーはすでにネットワークプロバイダーのInfineraで使用されています。同社の代表は、PCSはモバイルオペレーターがITインフラストラクチャの開発コストを削減するのに役立つと述べています。

オーストラリアからのエンジニアの育成も注目に値します。彼らは提出したファイバーのスループットを100倍にするソリューション。それらのシステムにより、光をらせん状に「ねじり」、光子のスピンを使用して情報をエンコードできます。現在、この技術はRMIT研究所でテストされています。しかし、開発者は今後数年でそれを市場に出すことを計画しています。
