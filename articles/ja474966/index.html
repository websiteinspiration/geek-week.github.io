OntologyネットワークでスマートPythonコントラクトを作成する方法。パート1：ブロックチェーンとブロックAPI



これは、SmartXスマートコントラクト開発ツールを使用して、Ontologyブロックチェーンネットワーク上のPythonでスマートコントラクトを作成する一連のチュートリアルの最初の部分です。

この記事では、Ontology Smart Contract APIの紹介を始めます。オントロジースマートコントラクトAPIは7つのモジュールに分かれています。

  1. ブロックチェーンとブロックAPI、
  2. API、ストレージ、
  3. APIランタイム、
  4. ネイティブAPI
  5. アップグレードAPI
  6. 実行エンジンAPIおよび
  7. 静的および動的呼び出しAPI。

Blockchain＆Block APIは、オントロジーのスマートコントラクトシステムの重要な部分です。Blockchain APIは、現在のブロックの高さの取得などの基本的なブロックチェーン要求操作をサポートし、Block APIは、特定のブロックのトランザクション数の要求などの基本的なブロック要求操作をサポートします。

始めましょう！

開始するには、SmartXで新しい契約を作成してから、以下の手順に従います。

1.ブロックチェーンAPIの使用方法


スマートコントラクト機能へのリンクは、Pythonリンクと同じです。必要に応じて適切な機能を入力できます。たとえば、次のステートメントでは、GetHeight（現在のブロックの高さを取得する関数）とGetHeader（ブロックのヘッダーを取得する関数）を紹介しています。

from ontology.interop.System.Blockchain import GetHeight, GetHeader

ゲータイト


GetHeightは、以下の例に示すように、ブロックチェーンの最後のブロックシーケンス番号を取得するために使用されます。最後の例では、便宜上、Main関数をスキップしますが、必要に応じて追加できます。

from ontology.interop.System.Runtime import Notify
from ontology.interop.System.Blockchain import GetHeight
def Main(operation):
    if operation == 'demo':
        return demo()
    return False

def demo():
    height=GetHeight()
    Notify(height) # print height
    return height #return height after running the function

Getheader


ブロックのタイトルを取得するために使用されるGetHeader、パラメーターはブロックチェーン内のブロックのシリアル番号です。例：

from ontology.interop.System.Runtime import Notify
from ontology.interop.System.Blockchain import GetHeader
def demo():
    block_height=10
    header=GetHeader(block_height) 
    Notify(header)
return header

GetTransactionByHash


GetTransactionByHashは、トランザクションのハッシュを通じてトランザクションを取得するために使用されます。トランザクションハッシュは、バイト配列形式のパラメーターとしてGetTransactionByHashに送信されます。この関数の鍵は、16進形式のトランザクションのハッシュをバイト配列形式のトランザクションのハッシュに変換することです。これは重要なステップです。そうしないと、そのようなブロックハッシュを持つブロックがないことを示すエラーが発生します。16進数形式のトランザクションハッシュを例として、それをバイト配列形式に変換してみましょう。次に例を示します。

9f270aa3a4c13c46891ff0e1a2bdb3ea0525669d414994aadf2606734d0c89c1

まずトランザクションハッシュを元に戻します。

c1890c4d730626dfaa9449419d662505eab3bda2e1f01f89463cc1a4a30a279

開発者は、SmartXが提供する16進数（リトルエンディアン）番号変換ツールを使用してこの手順を実行できます。

次に、結果をバイト配列形式に変換します。

{0xc1,0x89,0x0c,0x4d,0x73,0x06,0x26,0xdf,0xaa,0x94,0x49,0x41,0x9d,0x66,0x25,0x05,0xea,0xb3,0xbd,0xa2,0xe1,0xf0,0x1f,0x89,0x46,0x3c,0xc1,0xa4,0xa3,0x0a,0x27,0x9f}

これは、SmartXが提供する文字列バイト配列変換ツールを使用して行うことができます。最後に、結果のバイト配列を同様の文字列に変換します。

\xc1\x89\x0c\x4d\x73\x06\x26\xdf\xaa\x94\x49\x41\x9d\x66\x25\x05\xea\xb3\xbd\xa2\xe1\xf0\x1f\x89\x46\x3c\xc1\xa4\xa3\x0a\x27\x9f

以下は、トランザクションハッシュを使用してトランザクションを取得するGetTransactionByHash関数の例です。

from ontology.interop.System.Blockchain import GetTransactionByHash
def demo():
    # tx_hash="9f270aa3a4c13c46891ff0e1a2bdb3ea0525669d414994aadf2606734d0c89c1"    
    tx_hash=bytearray(b"\xc1\x89\x0c\x4d\x73\x06\x26\xdf\xaa\x94\x49\x41\x9d\x66\x25\x05\xea\xb3\xbd\xa2\xe1\xf0\x1f\x89\x46\x3c\xc1\xa4\xa3\x0a\x27\x9f")
    tx=GetTransactionByHash(tx_hash)
    return tx


GetTransactionHeight


GetTransactionHeightは、トランザクションハッシュを通じてトランザクションの高さを取得するために使用されます。上記の例からハッシュを取りましょう：

from ontology.interop.System.Blockchain import  GetTransactionHeight
def demo():
    #   tx_hash="9f270aa3a4c13c46891ff0e1a2bdb3ea0525669d414994aadf2606734d0c89c1"    
    tx_hash=bytearray(b"\xc1\x89\x0c\x4d\x73\x06\x26\xdf\xaa\x94\x49\x41\x9d\x66\x25\x05\xea\xb3\xbd\xa2\xe1\xf0\x1f\x89\x46\x3c\xc1\xa4\xa3\x0a\x27\x9f")
    height=GetTransactionHeight(tx_hash)
    return height

Getcontract


開発者はGetContract関数を使用して、契約ハッシュを介して契約を取得できます。コントラクトのハッシュを変換するプロセスは、前述のトランザクションのハッシュを変換するための対応するプロセスです。

from ontology.interop.System.Blockchain import GetContract
def demo():
    # contract_hash="d81a75a5ff9b95effa91239ff0bb3232219698fa"    
    contract_hash=bytearray(b"\xfa\x98\x96\x21\x32\x32\xbb\xf0\x9f\x23\x91\xfa\xef\x95\x9b\xff\xa5\x75\x1a\xd8")
    contract=GetContract(contract_hash)
    return contract

Getblock


GetBlockは、ブロックを取得するために使用されます。特定のブロックを取得するには2つの方法があります。

1.ブロックの高さごとにブロックを取得します。

from ontology.interop.System.Blockchain import GetBlock
def demo():
    block=GetBlock(1408)
    return block

2.ブロックハッシュごとにブロックを取得します。

from ontology.interop.System.Blockchain import GetBlock
def demo():    
    block_hash=bytearray(b'\x16\xe0\xc5\x40\x82\x79\x77\x30\x44\xea\x66\xc8\xc4\x5d\x17\xf7\x17\x73\x92\x33\x6d\x54\xe3\x48\x46\x0b\xc3\x2f\xe2\x15\x03\xe4')
    block=GetBlock(block_hash)

2.ブロックAPIの使用方法


：ブロックAPIで利用可能な3つの機能がありGetTransactions、GetTransactionCount、およびGetTransactionByIndexは。一つずつ並べていきます。

Gettransactioncount


GetTransactionCountは、特定のブロックのトランザクション数を取得するために使用されます。

from ontology.interop.System.Blockchain import GetBlock
from ontology.interop.System.Block import GetTransactionCount
def demo():
    block=GetBlock(1408)
    count=GetTransactionCount(block)
    return count

Gettransactions


開発者は、GetTransactions関数を使用して、特定のブロック内のすべてのトランザクションを取得できます。

from ontology.interop.System.Blockchain import GetBlock
from ontology.interop.System.Block import GetTransactions 
def demo():
    block=GetBlock(1408)
    txs=GetTransactions(block)
    return txs

GetTransactionByIndex


GetTransactionByIndexは、このブロックで特定のトランザクションを取得するために使用されます。

from ontology.interop.System.Blockchain import GetBlock
from ontology.interop.System.Block import GetTransactionByIndex
def demo():
    block=GetBlock(1408)
    tx=GetTransactionByIndex(block,0) # index starts from 0.
    return tx
