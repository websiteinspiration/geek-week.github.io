Amazon DeepLensディープラーニングカメラ。プロジェクトの開梱、接続、デプロイ



年の初めに、同僚と私は人工知能と機械学習に関連するさまざまな技術に興味を持ち始めました。まず、Amazon製品であるAWS DeepLensカメラをマスターすることにしました。私たちは公式ウェブサイトでデバイスを注文し、使用経験を共有したいと考えています。カメラのセットアップと接続には多くのニュアンスがあり、現時点ではこの問題に関する構造化されたドキュメントはほとんどないため、インストールと構成の際に多くの労力が必要になります。このチュートリアルが時間の無駄を省き、すぐにデバイスの使用を開始できることを願っています。

AWS DeepLensとは何ですか？


AWS DeepLensは、Amazon Deep Learningカメラです。Intel AtomプロセッサーとUbuntu 16.04 LTSシステムを搭載した小型システムユニットに取り付けられたカメラです。数年前の彼女について、ハブレに関するニュースがありました。

デバイスの特性は、ディープラーニングを使用するための初期設定にあります。AWS DeepLensには、ディープラーニングでの作業用に準備されたいくつかのプロジェクトをデプロイできる特別なソフトウェアがプリインストールされています。

以下のプロジェクトが展開用に準備されています。

  • オブジェクトの正確な検出と認識;
  • 「ホットドッグの有無」の基準による食品の分類。
  • DeepLensを使用した犬と猫の認識。
  • 約200種の鳥の認識;
  • 歯を磨く、口紅をつける、ギターを弾くなど、30種類以上のアクションを認識します。
  • 人間の顔の検出;
  • 9つの異なる角度での頭の位置の決定。

各プロジェクトはトレーニング済みモデルであり、すぐに使用できます。AWS DeepLensでは、モデルのトレーニング自体はできません。プロジェクトはインターネットからダウンロードされます。しかし、カメラを読み込んだ後、インターネットなしでローカルで論理的な結論を引き出すことができます。

モデルを準備してトレーニングするには、別のサービス-AWS SageMakerを使用する必要があります。モデルが準備およびトレーニングされたら、AWS DeepLensに転送し、ここでテストおよび変更できます。AWS DeepLensは、外部トレーニング済みモデルをインポートすることもできます。

開梱


AWS DeepLensのパッケージは、公式ウェブサイトで注文してから2週間後に到着しました。指定された住所のオフィスに直接配達されます。このカメラは、Amazonの「プライム」高速配信サービスの専用パッケージで提供されていました。



カメラ自体は会社の箱に梱包されています。



同梱されているのは、製品登録ページへのリンクを含むクイックスタートガイドです。



キットに含まれるもの：

  • SanDisk 32 GBのMicroSDメモリカード。
  • 2つのパーツのネットワークに接続するためのアダプター。
  • コンピュータに接続するためのUSBケーブル。



これは、カメラ自体が前後に見える方法です。





装置自体はコンパクト（94x47x168 mm）で、テーブルのスペースを取らないので非常に便利です。



接続と設定


接続および構成プロセス全体は、いくつかのステップで構成されています。

  • ワイヤーを介したカメラとコンピューターの初期接続およびローカルでの製品登録
  • AWSマネジメントコンソールでのWi-fiインターネット接続と登録。
  • 新しいプロジェクトを作成し、準備されたモデルを選択する。
  • モデルを含むプロジェクトをAWS DeepLensデバイスにダウンロードします（デバイスには16 GBの内部メモリ+ 32 GBカードが含まれています）;
  • デバイスへのプロジェクトの展開。

すべての手順が完了したら、カムコーダーの入力データを操作して、デバイスから必要な論理応答を受信できます。

1.カメラをインターネットに接続します


AWS DeepLensをインターネットに接続するには、コンピューターからカメラの管理パネルに移動し、この接続のWi-Fi設定をセットアップする必要があります。コンピューターとAWS DeepLensをwi-fiで接続します。

まず、つまようじを 持って「リセット」ボタンを押します。wi



-fiインジケーター



が青色に変わるまで待ちます。コンピューターからwi-fi経由でAWS DeepLensに接続します。 AWS DeepLensネットワークは、「AMDC-Q4US」などと呼ばれる場合があります。各デバイスは独自のネットワーク名を持つことができ、その名前はパッケージに示されています。



次に、ブラウザーを開き、リンクを介してデバイスに移動します：deeplens.config



次に、管理パネルに入った後、カメラの下部に表示されているデバイス番号とパスワードを使用してWi-fiに接続します。



その後、デバイスはインターネットに接続され、コンソールのデバイスのステータスが緑色で表示されます。



2.セキュリティ証明書をインストールする


カメラとAmazonサーバー間の安全で暗号化されたデータ転送には、安全な証明書が必要です。これは、deeplens.configのインストール手順で指定されたリンクからダウンロードできます。



証明書をダウンロードしたら、デバイスへのアクセスコード（ssh-password）を設定して覚えておく必要があります。
重要：パスワードを忘れた場合は、「工場出荷時の状態にリセット」して、AWS DeepLensにオペレーティングシステムを完全に再インストールする必要があります。

デバイスとPCをワイヤーで接続します


Amazonの組み込みチュートリアルでは、方法と実行方法について説明しています。



証明書をインストールしたら、AmazonのWebサイトに後で登録するために、USBケーブルを使用してカメラとコンピューターを接続する必要があります。



すべてが正常に完了すると、青いインジケーターが点灯します。



ワイヤーが接続された後-デバイスのシリアル番号を入力します。



カメラがUSB経由でコンピュータとネットワークに接続されていることを確認します。



4. Amazon Webサイトでデバイスを登録します


AWSアカウントがない場合は、最初に作成する必要があります。portal.aws.amazon.com / billing / signup



アカウントが作成されたら、製品ページに移動して[DeepLensの登録]をクリックします。 deeplens



次に、「ルートユーザー」を選択し、AWSアカウントのユーザー名とパスワードを入力します。



デバイスの名前を入力し、「デバイスの登録」をクリックします。



デバイスが登録されるのを待っています。



やったー、登録しました！



これで、USBを取り外し、Wi-Fi経由でカメラを制御できます。
個別のWi-fiネットワークごとに、新しいデバイスを登録する必要があります。つまり、自宅のコンピュータとオフィスのコンピュータから異なるデバイスが表示されます。

5.プロジェクトを作成する


コントロールパネルに入ったら、「AWSサービス」の行にデバイスの名前を入力し、「AWS DeepLens」を選択します。



次に、「新しいプロジェクトを作成」を選択します。



「プロジェクトテンプレートを使用する」を選択します。以下に、完成したプロジェクトテンプレートを示します。 「次へ」をクリックします。 「ゼロから」プロジェクトを選択できますが、これは今後の検討のための別のトピックです。



次に、「特別プロジェクトの詳細」画面に移動します。ここでは、テンプレートのパターンを確認できます。何も変更して「作成」をクリックすることはできません。







次に、デプロイ可能なプロジェクトを再度選択します。



プロジェクトを展開するデバイスを選択します。



以前に入力した情報をすべて確認し、[展開]をクリックします。



次に、プロジェクトがデバイスにデプロイされるのを待ちます。



プロジェクトがデバイスにデプロイされたら、[入力の表示]を選択します。



ビデオストリームが配信されるブラウザを選択します。



「ストリームの表示」をクリックします。



以前にダウンロードしたセキュリティ証明書を選択します



。最後に、カメラからブラウザーへのビデオストリームの出力を取得します。



これで、カメラからのすべての情報とオブジェクトに関する論理的な結論がコンソールに収集されます。



おめでとうございます。AWSDeepLensをインストールして構成し、データを操作する準備ができました！
