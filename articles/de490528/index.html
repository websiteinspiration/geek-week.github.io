1. SQL Server unter Linux


SQL Server läuft jetzt unter Linux. Dieser Kurs vermittelt Ihnen ein grundlegendes Verständnis von SQL Server unter Linux. Anschließend erfahren Sie, wie Sie SQL Server-Container unter Linux ausführen und SQL Server unter Linux bereitstellen. Danach erfahren Sie, wie Sie die SQL Server-Bereitstellung unter Linux automatisch konfigurieren.

Hier können Sie mit dem Training beginnen

2. Erstellen Sie Open Source-Analyselösungen mit Azure HDInsight


In diesem Kurs lernen Sie HDInsight kennen und erfahren, wie Sie diese Technologie anwenden, um eine Vielzahl von realen Aufgaben zu lösen.

Unter den Kursmodulen:

  • Einführung in das Open Source Analytics-Angebot
  • Wählen Sie die richtige HDInsight-Konfiguration aus, um Open-Source-Analyselösungen zu erstellen
  • Erstellen und Konfigurieren eines HDInsight-Clusters
  • Führen Sie Zero HDL-Analysen mit HDInsight Interactive Query durch

Fange an zu trainieren

3. Implementieren einer Datenflusslösung mit Azure Streaming Analytics


Lernen Sie die Prinzipien der Ereignisverarbeitung und des Streaming kennen und erfahren Sie, wie dies für Azure Stream Analytics gilt. Richten Sie dann Ihren Stream Analytics-Job für das Streaming von Daten ein und erfahren Sie, wie Sie die Jobausführung überwachen und verfolgen.

Fange an zu trainieren

4. SQL Server Learning New


Die neueste Version von SQL Server bietet viele Vorteile für mehr Leistung und Funktionalität sowie verbesserte Unterstützung für die Datenbankplattform. In diesem Lernprogramm erfahren Sie, wie Sie SQL Server-Bereitstellungen mithilfe des Microsoft Assessment and Planning Toolkit inventarisieren. Darüber hinaus erfahren Sie, wie Sie SQL Server-Datenbanken mithilfe des Datenmigrationsassistenten aktualisieren und Ihr aktualisiertes System mithilfe des Datenbankexperimentierungsassistenten und des Abfrageoptimierungsberaters optimieren.

Fange an zu trainieren

5. Arbeiten mit NoSQL-Daten in Azure Cosmos DB


NoSQL-Daten sind eine effiziente Methode zum Speichern von Daten, die nicht den Anforderungen einer relationalen SQL-Datenbank entsprechen. Erfahren Sie, wie Sie das Azure-Portal, die Azure Cosmos DB-Erweiterung für Visual Studio Code und das .NET Core SDK für Azure Cosmos DB verwenden, um an jedem Ort mit NoSQL-Daten zu arbeiten und Benutzern unabhängig von ihrem Standort ein hohes Maß an Zugänglichkeit zu bieten.

Fange an zu trainieren

6. Azure für Dateningenieure


Erfahren Sie, wie sich die Datenwelt entwickelt und wie das Aufkommen der Cloud-Technologie neue Geschäftsmöglichkeiten bietet. Sie lernen die verschiedenen verfügbaren Datenplattformtechnologien kennen und erfahren, wie ein Dateningenieur diese Technologien zum Nutzen des Unternehmens nutzen kann.

Unter den Modulen:

  • Die sich entwickelnde Welt der Daten
  • Übersicht über die Azure Data Platform Services
  • Definieren der Aufgaben eines Data Engineering-Spezialisten in einer in der Cloud gehosteten Architektur

Fange an zu trainieren

7. Erstellen und verwenden Sie Analyseberichte in Power BI


In diesem Kurs lernen Sie Power BI kennen und lernen, wie Sie Business Intelligence-Berichte verwenden und erstellen.

Unter den Modulen:

  • Erste Schritte mit Power BI
  • Abrufen von Daten mit Power BI Desktop
  • Datenmodellierung in Power BI
  • Verwenden visueller Elemente in Power BI
  • Daten in Power BI untersuchen
  • Veröffentlichen und freigeben Sie in Power BI
