Testing StoreKit on Xcode 12 and iOS 14

Hello, Habr!







My name is Sam, I'm a Qonversion developer. This article describes the new features for testing subscriptions introduced by Apple during the Introducing StoreKit Testing in Xcode session at WWDC 2020.







The mobile app subscription market is growing rapidly, but the test suite before iOS 14 was scarce. In addition, everything was complicated by the instability of the Apple sandbox.

Innovations in Xcode 12 and iOS 14 make a big difference.













What's new



  1. StoreKit Configuration File - allows you to configure products for testing directly in Xcode without the need to create them in App Store Connect;
  2. StoreKit Transaction Manager - transaction manager in Xcode, allows you to see all local purchases and change the status of them;
  3. StoreKitTest Framework - a new framework for tests in Xcode, gives full control over the testing process;
  4. Manage your Sandbox device subscriptions.
