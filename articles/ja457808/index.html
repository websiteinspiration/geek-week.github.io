サーマルイメージャーSeek Thermal Compact

私の小さなアシスタント-Seek Thermal Compactモバイルサーマルイメージャーセットトップボックスに関するレビューを追加します。スマートフォンやタブレット用のサーマルイメージャーは、熱や冷気の漏れの検出と排除、電気配線の問題の発見、機器の局所的な加熱または過熱の場所の確認、狩りの獲物の発見などに役立ちます。Seek Thermalは、スマートフォンやタブレットに接続し、「大人の」プロフェッショナルモデルの全機能を備えた、安価で手頃な価格のコンパクトなデバイスを作成することに成功しました。



Seek Thermal Compactモバイルサーマルイメージャーは、スマートフォン用の小さな外部カメラセットトップボックスで、特別なLWIRレンズを備えたマイクロボロメトリックマトリックスを備えています。サーマルイメージャーは米国で開発され、中国の偽物とは異なり、優れた特性を備えています。



主な特徴：
ブランド：Seek Thermal
モデル：Compact
熱センサーの解像度：206 x 156ピクセル
FOV視野角：36度
検出範囲：最大300メートル
スキャン周波数：9 Hz
オートフォーカス：はい



サーマルイメージャーは、LWIR範囲（赤外線）で動作します。従来のレンズはLWIR放射を透過しないため、サーマルイメージングデバイス用のゲルマニウム（Ge）から作られています。マイクロボロメータセンサーは、小型の薄膜サーミスタのマトリックスです。サーマルイメージャーレンズによってセンサーに集められて集束された赤外線は、観測されたオブジェクトの温度分布に従ってマトリックスの要素を加熱します。電源を入れるたびに、連続動作中に、特徴的なクリックで定期的な再校正が行われます。



キットには、デバイスを持ち運び、保管するための特別なケースがあります。ケースは衝撃や衝撃から保護するだけでなく、湿気からも保護します。



内部には、サーマルイメージャーを衝撃やその他の影響から保護するソフトインサートがあります。



ケースはコンパクトで、キーホルダーに掛けるためのリングが付いています。ポケットに静かに収まります。



レンズには画像を調整するためのリングがあります。



スマートフォンまたはタブレットに接続するには、MicroUSBコネクタ（またはAppleデバイスのLightning）を使用します。



Type-C搭載のスマートフォンには、従来のアダプターが適しています。安価なモデルでは接点がバウンスするため、サーマルイメージャーは定期的にオフになるため、お金を節約せずにアダプターをより高価にすることをお勧めします。やり過ぎた方がいいです。



デバイスを操作するには、App StoreまたはGoogle Playから特別な「シークサーマル」アプリケーションをダウンロードしてインストールする必要があります。



次に、シークサーマルをスマートフォンまたはタブレットに直接接続するか、アダプターを介して接続します。



アプリケーションを起動し、カメラへのアクセスを許可し、モードを選択して構成します。



このアプリケーションには、画像のコントラストを上げるために選択できる設定、マーカー、パレット、およびビデオを撮影する機能がいくつかあります。



さまざまなSeek Thermalモデルの比較モデル

ごとに異なるFOV角度（20°〜36°）に注意してください。
デバイスのプロバージョンには、320 x 240ピクセルの解像度を持つマトリックスがあり、残りの206 x 156ピクセルです。 XRモデルにはフォーカス可能なレンズシステムが搭載されていますが、若いコンパクトには固定フォーカスが搭載されています。



古いSeek Thermal Compact XRが横になっているので、これらのモデルを「ライブ」で比較するための写真が数枚あります（左の写真-Compact XR、右の写真-更新されたCompact）。



コストに加えて、主な違いは可変焦点距離の存在です。レンズの大きな回転リングを調整します。写真では、2つのモデルのレンズの違いが顕著です（左側の写真-更新されたコンパクト、右側のコンパクトXR）。

更新：古いコンパクトモデルに焦点が当てられていませんでした。 （記事内の）新しいSeek Thermal Compactモデルでは、設計が更新されましたが、この機能が導入されました。少なくともCompact XRモデルとの違い。



しかし、比較として、Flir One Pro（第3世代）は、スマートフォンやタブレット用の最も高価なセットトップボックスの1つです。このFlir Oneモデルのボロメトリックマトリックスには、合計160 x 120センサー（Seekの場合は206 x 156ポイント）、および同様のFOV視野角：55 x 43°（Seek 36°の場合）があります。そして、これはSeekの3倍のコストです。



これは、バッテリーが内蔵されたかなりかさばるモデルで、可視範囲の追加のカメラです。アダプター付きのFlir One Proモデルも使用しています。

Seek Thermal Compact Thermal Imagerのアプリケーションシナリオ

私たちの祖先が洞窟に定住した瞬間から、住宅の断熱の問題は人類を占領しました。自分の手で、そして目で、寒さの影響から部屋を守ることができます。しかし、どのようにうまく行ったかを確認するにはどうすればよいですか？キャンドルを持って歩いたり、ドラフトの方向を調べたり、壁に霜を付けたりできます。また、調査中の空間内の温度差の完全な画像を受け取ったSeek Thermalモバイルサーマルイメージャーを使用して、部屋の熱画像検査を行うことができます。しかし、これはミニサーマルイメージャーの唯一の使用法にはほど遠いものです。モバイルサーマルイメージャーは、暑い場所や寒い場所を確認する必要がある場合に役立ちます。主な用途は、電気配線、配電盤機器の問題の検索と診断、床または壁に壁に囲まれた通信の検索、冷媒の漏れの検出、仕上げ作業の欠陥、ドラフトの原因の検索、熱漏れなど。

最近、サーマルインスペクション-カントリーハウス、コテージ、シティアパートの建物の欠陥の診断の種類、壁の断熱材のチェック、プラスチック窓の設置、「暖かい床」の診断、サーマルイメージャーを使用した暖房と配管-が非常に重要になっています。

写真は一部が外部から断熱されたパネルハウスです。同様の方法が断熱材として使用され、アパートの建物の継ぎ目やパネルに大きな欠陥があります。さらに、サーマルイメージャーを使用すると、欠陥を予測し、追加の断熱材を適用する場所を特定したり、断熱材の結果を診断したりできます。家を借りるとき、断熱のとき、屋外の仕事のときなど、仕事を受け入れるときは、熱画像装置（または熱検査サービス）を明確に見つけようとします。



先日、建物内のエンジニアリングネットワークを検査するためにサーマルイメージャーを撮影しました。視覚的な制御に加えて、怠惰ではなく、サーマルイメージャーに目を通してください。電気と照明（不必要な過熱または加熱してはならない場所）、温水と冷水の供給の通信をチェックできるだけでなく、空調および換気システムを評価することもできます。写真には明るいスポットがあります-省エネランプ、そして通信の熱トレースも見えます。



そして、ここで断熱材は明らかに損傷しています。次回の定期メンテナンスに向けて書き込みます。
