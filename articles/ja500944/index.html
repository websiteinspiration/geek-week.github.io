WexLang Reader-喜んで外国語で本を読む方法（104言語）

こんにちは！

私は自分のプロジェクトについてすでにここに書いた-Wexlang。プロジェクトが更新され、現在は無料のデスクトップアプリケーションです。

アプリケーションには104言語があり、使用可能な言語間の翻訳指示があります。



古いプロジェクトについて少し：機能性、複雑さ、およびマーケティングの高コストなどの問題のため、Wexlang

Webアプリケーションが中止されました。 しかし、プロジェクトを破棄したくなかったため、機能を完成させ、エラーを修正し、プロジェクトをデスクトップアプリケーションにパックすることにしました。



WexLangリーダー


私は主に読書と通信に英語を使用しているので、プロジェクトはこれらの特定のスキルの開発を目指しています。

アプリケーションはphp + laravel + sqlite + jQueryで記述され、PHPデスクトッププロジェクトのおかげでデスクトップアプリケーションにパッケージ化されています。

能力


  • 104言語と翻訳の方向
  • テキストおよびテキストの各ページに関する詳細な統計、各単語の使用頻度に関する統計
  • 任意のテキストをダウンロードできます。著作権の制限はありません。
  • ページ上のすべての単語をよく知っている/学習済みとしてマークできます
  • ワードプロセッサは句動詞と任意のフレーズで動作します
  • 新しい単語と学習した単語の強調表示を有効/無効にすることができます
  • デスクトップアプリケーションなので、何かが動作しなくなることを心配する必要はありません。単語ベースをバックアップできます。
  • 単語のインポートとエクスポート

アプリケーションの操作方法


テキストの各単語には、次の3つのステータスのいずれかが割り当てられます。

  • 新規-テキスト内で青色で強調表示されます
  • 調査済み-テキスト内でオレンジ色で強調表示されています
  • よく知っている-テキストで目立たない

1.まず、外国語でテキストをダウンロードします



。2.各テキストには詳細な統計があります。新しい、よく知られた、学習された単語の数、およびすべてのユニークな単語のリスト。

各単語は使用頻度を示します-単語がテキストに出現する回数と単語の総数の何パーセントか。

テキストの統計では、単語を調べて、単語に望ましいステータスを設定できます。単語のリストをエクスポートすることもできます。



3.テキストをダウンロードしたら、読み始めることができます。



次のオプションは、テキスト読み取りページで使用できます。

  • 見慣れない単語を強調表示/強調表示しない
  • 学習する単語を強調表示する/強調表示しない
  • ステータスページのすべての単語を勉強するように設定する
  • 見慣れたステータスページのすべての単語を設定する
  • 単語をクリックすると、Googleに翻訳されます。単語が以前に翻訳されている場合、翻訳はデータベースから取得されます
  • ワードステータス変更

また、各ページには単語の統計があります。一意の単語のリストと、各単語がテキストに出現する回数。ここでは、ステータスという単語を設定することもできます。



4. [ 私の単語]ページでは、単語の追加、削除、翻訳の編集を行うことができます。



5.アプリケーション設定で、インターフェース言語、言語ペアを選択し、Google翻訳用のAPIキーを設定できます。



転送


単語はグーグル翻訳によって翻訳されます。翻訳中、単語と翻訳はデータベースに保存されるため、単語はGoogle で1回だけ翻訳されます。

2つのオプションがあります。

  1. 無料のグーグル翻訳を介して翻訳を使用してください-この場合、制限があり、Googleはしばらくの間あなたのIPをブロックするかもしれません。特に、「すべての単語を次のようにマークする」機能を頻繁に使用する場合。
  2. 公式のGoogle翻訳APIを使用してください。1か月あたり最大10ドルのAPIは無料です。単語がデータベースに保存されていることを考えると、APIの料金を支払う必要がないか、金額が非常に少なくなります。

現在の問題


  • グーグルからの翻訳はあまり良くありません。
  • 象形文字の言語がどのように配置されているのか手掛かりがないため、修正方法を知るまで、機能はアジアの言語ではうまく機能しません。

予定


翻訳と辞書の機能を少しやり直し、単語ごとに異なる品詞と翻訳を単語に追加する機能を追加し、説明辞書から（外国語で）単語の値を読み込む計画があります。
