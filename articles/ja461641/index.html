問題解決コイン1


署名coin1のアイコンをクリックすると、接続用のアドレスとポートが提供されます。



接続後、ゲームをプレイし、ゲームのルールを提供することが提案されます。また、完了までに60秒かかるため、すべてを自動化する必要があります。



ゲームのルールによれば、彼らは私たちにN個のコインを与えます。1個を除いてそれぞれ10個の重さです-その重さは9です。1つのゲームのチャンス（ラウンド）Cの数が与えられます。各ラウンドで、コインインデックスを送信し、総重量も与えられます。したがって、バイナリ検索を使用して、目的のコインを見つけます。

コードを書いてみましょう。開始するには、サーバーへの接続を確立し、番号NとCを受け入れて解析します。

from pwn import *

r = remote('pwnable.kr', 9007)
r.recv()

s = r.recv()
print(s)
n = int(s.split(' ')[0][2:])
c = int(s.split('=')[2].split('\n')[0])
print(n, c)



いいね。次に、1つのレベルを渡すための部分を記述します。これを行うには、配列の半分が送信される各反復で、1からN + 1までの値の配列とCステップのサイクルが必要です。回答で返された重みを余りなく10で割った場合、私たちのコインは配列の別の部分にあります。したがって、残りの半分を再度分割して、同じように処理します。コインが発見されるまで。

mas = range(1,n+1)
for i in range(c):
	s = ""
	if len(mas)==1:
		mas.append(mas[0])
	for j in mas[:len(mas)/2]:
