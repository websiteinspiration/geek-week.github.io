Anúncios da Microsoft sobre exames e certificados

Nos últimos 3 meses, a Microsoft fez três anúncios importantes:

  1. MCSA, MCSD, MCSE desde 2021, juntamente com todos os exames anteriores, exceto o novo baseado em funções, deixam de existir .
  2. Quase todos os exames anteriores baseados em Funções do Azure estão desatualizados. Isso pode ser lido em um artigo separado com análise
  3. Haverá vários novos exames e, consequentemente, certificados para eles:
    • Exame DP-300: Administrando bancos de dados relacionais no Microsoft Azure (beta) / Microsoft Certified: Azure Database Administrator Associate
    • Exame DA-100: Analisando dados com o Microsoft Power BI (beta) / Certificado pela Microsoft: Data Analyst Associate
    • Exame AZ-120: Planejando e administrando o Microsoft Azure para cargas de trabalho SAP / Microsoft Certified: Azure for SAP Workloads Specialty
    • Exame AZ-220: Desenvolvedor de IoT do Microsoft Azure / Certificado pela Microsoft: Especialidade de desenvolvedor de IoT do Azure

Primeiras coisas primeiro.

MCSA / MCSA / MCSE - tudo. Isso é tudo.


Estou certo de que a obsolescência do MCSA / MCSA / MCSE causou uma reação muito maior do que todos os outros anúncios. Se não houve novos exames no SQL 2019 ou no Windows Server 2019, não é tão assustador. Mas, na verdade, todos os certificados e exames da Microsoft que não estão no Azure / Dynamics365 / Office365 são multiplicados por zero. Obviamente, adoçamos a pílula, garantindo que, na transcrição de seus exames, o fato de passar não desaparecerá, mas isso não muda a essência.



A lista de exames a serem marcados como obsoletos é muito longa.

SAP do Azure


Tudo é complicado aqui, porque a tecnologia da Microsoft não é combinada com o Azure, ou seja, exame simultaneamente na infraestrutura do Azure / componentes e serviços SAP (para monitoramento, otimização) e arquitetura.

IoT do Azure


O exame combina as habilidades de um desenvolvedor de IoT em C #, administrador / integrador de uma solução de IoT (o provisionamento de dispositivos é muito solicitado), um usuário do Azure SaaS para IoT e um arquiteto. Está claro que a IoT está se aprofundando cada vez mais em nossas vidas, e o exame é necessário para quem deseja verificar suas habilidades. Pessoalmente, não tenho certeza de que o número de desenvolvedores de IoT seja grande, mas foi claramente necessário um exame para fechar o nicho correspondente.

Power BI


É simples: como trabalhar com o Power BI. Pessoalmente, não gosto do nome do certificado fornecido para este exame, "Microsoft Certified: Data Analyst Associate". Existem muitas outras ótimas ferramentas para análise de dados, o PowerBI é muito estreito para esse nome.

Administrador SQL do Azure


Aqui todas as palavras são importantes - nenhuma delas pode ser descartada. Isso não é desenvolvimento, mas implantação, otimização, desempenho, tolerância a falhas etc. O SQL Server local é considerado apenas como parte do híbrido (não há uma única palavra sobre privado nas medições de habilidades sobre privado).

Porque Código de exame DA-300, depois da convenção de nomenclatura, não ficarei surpreso ao ver outro DP-301 sobre desenvolvimento. Mas isso é tudo especulação.
