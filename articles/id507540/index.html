Bekerja dengan acara khusus di htmlix.js

Artikel ini akan memeriksa prinsip-prinsip dasar bekerja dengan peristiwa pengguna di Htmlix dan membuat aplikasi kecil dari empat komponen: formulir untuk memasukkan nama pengguna, salam, tombol keluar untuk aplikasi, dan array dengan pengguna aplikasi.



Contoh yang sudah jadi bisa diklik di sini .



Anda dapat membaca htmlix api di sini.



Setelah memasukkan nama pengguna dalam formulir, aplikasi akan menampilkan namanya di sapaan, tombol keluar dan membuat wadah baru dalam array dengan pengguna, serta menyimpan data di localStorage. Setelah memuat ulang halaman, itu akan mengambil nama dari localStorage. Mengklik tombol "profil keluar" akan menghapus data dari semua komponen dan variabel dari Penyimpanan lokal.



Mari kita membuat empat komponen: formulir login - formulir, salam - salam, tombol logout - logout dan array dengan semua pengunjung - users_array, dan lihat bagaimana kita akan membuat komunikasi di antara mereka jika kita tidak menggunakan acara pengguna.

Sekarang kita perlu menampilkannya di properti komponen sambutan dan logout, dan juga membuat wadah baru di array users_array. Untuk memahami mengapa acara khusus diperlukan, pertama mari kita coba lakukan ini dengan berbagai cara tanpa menggunakannya.



1. cara - dalam metode klik formulir itu sendiri, langsung ke setiap komponen dan atur properti teks:

Sekarang komponen dienkapsulasi, namun, kami mengakses komponen dengan namanya. Bagaimana jika nanti kita ingin mengubah nama komponen atau metode, atau menghapus komponen sama sekali, atau kita akan memiliki banyak komponen seperti itu yang mendengarkan properti yang sama. Atau kita akan memiliki banyak properti seperti itu, jadi kita harus membuat banyak metode seperti itu.



Untuk memfasilitasi tugas ini, Anda dapat menggunakan acara khusus. Komponen berlangganan acara, dan ketika itu terjadi, melakukan sesuatu dengan propertinya.



Mari mengedit kode, sekarang menggunakan acara khusus: 



var StateMap = {

   eventEmiters: {	
             //   -  	
             ["emiter-entry-user"]: {prop: ""}, 		
   },
   form: {
	container: "form",
	props: ["input", "click"],
	methods: {
			
		click: function(){
		    event.preventDefault();
		    var text = this.parent.props.input.getProp();

		     //        
		    this.rootLink.eventProps["emiter-entry-user"].setEventProp(text); 	
		    window.localStorage.setItem('user_name', text);							
		}			
	},		
},
