Seabornライブラリを使用してDataFrameから列を視覚化する

DataFrameに格納されている広告キャンペーンのデータを視覚化してみましょう。

与えられた：


DataFrame。次の指標の広告キャンペーンに関する統計を保存します。

  • キャンペーン名
  • 日付
  • 感想
  • クリック
  • 中央
  • 費用
  • Avgcpc
  • 直帰率
  • AvgPageviews
  • 変換バージョン
  • CostPerConversion
  • 変換



必要なものはすべてインポートします。


import seaborn as sns
from pandas import Series,DataFrame

csvからDataFrameを読み取る


f=DataFrame.from_csv("cashe.csv",header=0,sep='',index_col=0,parse_dates=True)


 AvgCpc列データを視覚化する


sns.distplot(f['AvgCpc'],bins=25)
plt.show()

次のグラフが表示されます。



このグラフは、クリックのコストの分布を示しています。グラフは、ほとんどの場合、クリックに約3.5ルーブルかかることを示しています。

グラフをより正確にするには、「ビン」の値を増やします。このパラメーターは、グラフが分割される部分の数を反映します。

sns.distplot(f['AvgCpc'],bins=50)
plt.show()

以下を取得します。



ヒストグラムをラグプロット（ラグ）で置き換えることもできます。

sns.distplot(f['AvgCpc'],bins=25,rug=True,hist=False)
plt.show()



ヒストグラムに戻りましょう。

名前と色を設定する


行を青に、列を青に着色します。

sns.distplot(f['AvgCpc'],bins=25,
kde_kws={'color':'indianred','label':''},
hist_kws={'color':'blue','label':''})
plt.show()
