ニューラルネットワーク環境の設定R-CNNマスク

こんにちは、ニューラルネットワークの研究の一環として、多くの人が環境の設定に困難に直面しています。このため、のどが渇いた初心者を助けるために記事を書くことにしました。

タスクの一環として、彼はMask R-CNNのアーキテクチャを使用しました。

すべてのディストリビューションへのリンクは、投稿の最後に提供されます。

それでは、始めましょう。ニューラルネットワークの機能を使用するには、他のソフトウェアと同様に、インストールして構成する必要があります。このペーパーでは、次の一連のアクションについて説明します。

  1. すべてのNVIDIAドライバー/ソフトウェアを削除し、プログラムファイル（すべてx86）からすべてのNVIDIAファイルを削除します。古いドライバが新しくインストールされたコンポーネントの動作に影響を与えないようにする必要があります。
  2. Visual Studio [1]をインストールします。インストールプロセス中に、IDEとともに、Visual C ++などの追加作業に必要なライブラリを提供します。プログラムの最小バージョンは2015年です。
  3. CUDA [2]をインストールします。これは、Nvidia GPUを使用してコンピューティングパフォーマンスを大幅に向上させることができる並列コンピューティング用のソフトウェアおよびハードウェアアーキテクチャです。最小バージョンは9.0です。
  4. ダウンロードcuDNN [3]は、グラフィックアクセラレータをサポートするニューラルネットワークの展開を操作するためのライブラリです。バージョンはCUDAバージョンと一致する必要があります。
  5. ダウンロードして解凍したcuDNNフォルダーの内容を、C：\ Program Files \ NVIDIA GPU Computing Toolkit \ CUDA \ Vx.x \の適切なフォルダーにコピーします。
  6. システムの環境変数に移動してパスを編集し、次の2つのディレクトリをパスに追加します。

    • C：\ Program Files \ NVIDIA GPU Computing Toolkit \ CUDA \ v9.0 \ bin
    • C：\ Program Files \ NVIDIA GPU Computing Toolkit \ CUDA \ v9.0 \ libnvvp
  7. Python [4]をダウンロードしてインストールします-開発者の生産性とコードの読みやすさの向上を目的とした高水準の汎用プログラミング言語。最小バージョンは3.5です。
  8. Anaconda [5] – Python R, , . Anaconda to PATH , , IDE Windows CMD. Python.
