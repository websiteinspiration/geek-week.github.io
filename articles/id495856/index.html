Intel DevCloud for oneAPI - layanan cloud untuk pengembang yang duduk di rumah


Selama epidemi, banyak (jika tidak semua) platform dan layanan Internet membuat penawaran khusus untuk publik di rumah. Kami menawarkan opsi hardcore untuk pengembang yang, bahkan dalam kondisi isolasi diri, tidak siap memposting foto mereka yang dengan berani atau lembut berbagi isolasi dengan mereka (anjing laut), tetapi, sebaliknya, mencoba untuk memompa keterampilan profesional mereka. Intel DevCloud untuk oneAPI penuh gaya, modis dan gratis untuk semua orang hingga 1 Juli.

Sesuai namanya, Intel DevCloud untuk oneAPI adalah kapasitas server Intel untuk pengembang dengan akses melalui platform cloud. Server didasarkan pada prosesor Intel Xeon Scalable dan dilengkapi dengan Intel Arria 10. FPGAs. Jumlah node prosesor dan akselerator yang tersedia untuk pengguna telah meningkat pada periode karantina. Kuota khusus - 220 GB penyimpanan disk dan RAM 192 GB.

Selain perangkat keras, platform ini juga menyediakan komponen perangkat lunak yang sudah diinstal sebelumnya.
Intel oneAPI Toolkits adalah seperangkat alat pengembangan yang mencakup kompiler untuk bahasa baru Data Parallel C ++, satu set perpustakaan untuk pemrograman API dan satu set alat untuk menganalisis dan men-debug aplikasi. Secara lebih rinci, himpunan komponen adalah sebagai berikut.

  • Kompiler

    • Intel OneAPI DPC ++ Compiler
    • Intel C ++ & Intel Fortran Compiler
  • Toolkit

    • Intel oneAPI Base Toolkit
    • Intel oneAPI HPC Toolkit
    • Intel oneAPI DL Framework Developer Toolkit
    • Intel AI Analytics Toolkit
    • Distribusi Intel OpenVINO Toolkit
  • Utilitas, kerangka kerja dan perpustakaan
    • Perpustakaan Intel OneAPI DPC ++
    • Perpustakaan Intel Kernel Matematika OneAPI
    • Intel OneAPI Deep Neural Network Library
    • Perpustakaan Analisis Data Intel OneAPI
    • Intel Distribtion untuk Python
    • Intel VTune Profiler
    • Add-on Intel FPGA untuk oneAPI Base Toolkit

Semua komponen perangkat lunak cloud diperbarui ke versi terbaru saat ini. Menginstal lingkungan pengembangan interaktif JupyterLab , juga memiliki akses melalui SSH. Ada dukungan teknis dan forum online pada layanan DevCloud dan langsung onneAPI .

Semuanya sudah siap untuk kita - mulai bekerja.
