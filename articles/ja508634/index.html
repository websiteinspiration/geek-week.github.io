単純なJavascriptのビットは多くのことができます

私はプロのフロントエンド開発者として働いたことはありません。小規模なサイドプロジェクト用にHTML / CSS / JSを15年間書いていますが、すべてのプロジェクトは非常に小規模でした。これらのプロジェクトの合間に何年もJavaScriptで書かないことがあり、すべてが正しく行われていることを確信できないことがよくあります。



これが、ライブラリを頻繁に使用した理由の1つです。10年前はjQueryを使用しており、2017年以降は小規模なプロジェクトでvue.jsを頻繁に使用しています（Vueの紹介として作成した「sapper」のような小さなゲームを見ることができます）。



でも先週、久しぶりにライブラリなしの簡単なJavascriptを書いたので楽しかったので、少しお話したかったです！

Vueでこれがどのように可能になるかは実際にはわかりませんでした（VueはすべてのHTMLを作成したいため）。これは小さなプロジェクトだったので、ライブラリなしのプレーンなJavaScriptで作成することにしました。HTML/ CSSを作成して追加するだけ<script src="js/script.js"> </script>



です。



私はしばらくこれをしていませんでしたが、その過程で、最初に思ったよりもプロセスを簡単にするいくつかのことを学びました。



CSSクラスの追加と削除をほぼすべて行います



トランジションをアニメーション化する場合は、CSSクラスを追加および削除し、CSSトランジションを使用するだけで、ほぼすべてのユーザーインターフェイスを実装することにしました。質問の



ボタンnext



をクリックしてクラスdone



を親divに追加する小さな例を次に示します。 



div.querySelector('.next-question').onclick = function () {
    show_next_row();
    this.parentElement.parentElement.classList.add('done');
}


それはかなりうまくいきました。私のCSSはいつものように少し混乱していますが、扱いやすいようです。



.classListを使用したCSSクラスの追加/削除



私は次のようにクラスの編集から始めましたx.className = 'new list of classes'



。これは少し汚いですが、もっと良い方法があるかどうか疑問に思いました。そして彼はそうでした！



CSSクラスは次のように追加できます。 



let x = document.querySelector('div');
x.classList.add('hi');
x.classList.remove('hi');


element.classList.remove('hi')

document.querySelectorAllを使用してアイテムを検索する



私がjQueryの学習を始めたとき、DOMで何かを簡単に見つける必要がある場合は、jQuery（たとえば$('.class')



）を使用する必要があると思ったのを覚えています。今週私はあなたが代わりdocument.querySelectorAll('.some-class')



に書くことができることを知りました、そしてあなたはどのライブラリにも依存する必要はありません！



紹介されたとき、気になったquerySelectorAll



。私は少しグーグルしました、そしてセレクターAPIが2008年と2013年の間にどこかで作成されたようです- 2008年に提案された実装について議論しているjQueryの作者からのメッセージと2011年のブログ投稿を見つけましたそれは当時のすべての主要なブラウザにあったので、jQueryを使い始めたときには存在していなかったかもしれませんが、かなり長い間存在していました:)

.scrollIntoViewを使用してページをスクロールする



私が見つけた最後の面白いことはこれ.scrollIntoView



です。誰かが「次の質問」ボタンをクリックしたときに、次の質問に自動的にスクロールしたかったのです。これはたった1行のコードです。
