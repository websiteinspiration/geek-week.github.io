WSLを使用してWindows 10にRTKRCV（RTKLIB）をインストールして構成する

こんにちは、この記事では、高須知二（日本）のチームが開発したRTKLIBオープンライブラリからRTKRCVアプリケーションをインストールして実行するプロセスについて説明します。

RTKLIBは、GNSSテクノロジー（グローバルナビゲーション衛星システム）を使用した標準的で正確な測位用のオープンソースソフトウェアパッケージです。

この記事は誰のためのものですか？


この資料は、正確な位置決めをデバイスに統合したいすべての人にとって実用的に重要であり、RTKLIBライブラリによって提供されるソフトウェア製品での作業を最適化するためのリファレンスとしても役立ちます。この資料の情報は、Windows 10システムでの専門家とCUIの相互作用を最適化するように設計されています。

まず、RTKRCVが何であるかを理解しましょう。 RTKRCVは、ナビゲーションデータを操作するために設計されたRTKNAVIソフトウェア製品のCUIアナログです。 APはリアルタイムでGPS / GNSSレシーバーから生の観測を受信し、リアルタイムのナビゲーション処理を実行します。

測地学の分野でエンジニアリング問題を解決する場合、RTKRCVのコンソールアナログを使用することが可能です。このCUIについては、この製品の使用に関する完全な情報がライブラリの説明書にあります。

独自の後処理シナリオを開発したり、タスクを解決したりするために、LinuxとWindowsを切り替えると、深刻な不便が生じます。少なくとも、別のシステムを使用するか、仮想マシンを使用する必要がありますが、これも不便な場合があります。

救いは、Windows 10でのLinuxカーネルのサブシステムのサポートの存在であることがわかりました。

サブシステムを構成してRTKRCVを開始するためのアルゴリズム：

  1. WSL（Ubuntu、Debianなど）のインストール。
  2. WSLに必要なユーティリティをインストールします。
  3. ssh-keyの構成（PC <-> GitHub）;
  4. RTKLIBでリポジトリを取得する。
  5. RTKRCVのインストール。
  6. 構成ファイルの構成。
  7. RTKRCVを起動します。

1. WSL（ubuntu）のインストール


インストールを開始する前に、管理者権限でPowerShellを実行し、次のコマンドを実行する必要があります。

Enable-WindowsOptionalFeature -Online -FeatureName Microsoft-Windows-Subsystem-Linux

プロンプトが表示されたら、コンピューターを再起動します。

好みのディストリビューションをダウンロードしてインストールするには、3つのオプションがあります。

  • Microsoft Storeからダウンロードしてインストールします。
  • コマンドラインまたはスクリプトを使用してダウンロードしてインストールします。
  • ダウンロードしてから、手動で解凍してインストールします。

2. WSL


RTKLIBライブラリはC / C ++で記述されているため、アプリケーションと一連のgccコンパイラをビルドするにはmakeをインストールする必要があります。

sudo apt install make
sudo apt install gcc

3. ssh-keyの設定（PC <-> GitHub）


GitHubとWSLの間にSSH接続を確立する

  1. ssh-keygen -t rsa -b 4096 -C “ E-mail”

    メッセージ：キーを保存するファイルを入力してください（*標準ディレクトリ*）（sshキーの保存場所に同意する場合はEnterキーを押してください）
    メッセージ：パスフレーズを入力してください：（接続用のパスワードを設定してください）
  2. eval $(ssh-agent -s)

    *次のようなメッセージを受信します：エージェントpid 1234 
  3. ssh-add ~/.ssh/id_rsa

    *パスワードを入力した後、IDが追加されました：*ディレクトリ*
  4. cat < ~/.ssh/id_rsa.pub

    *受信したメッセージをコピーします*
  5. GitHubアカウントの設定に移動して、SSHおよびGPGキーのタブを見つけます。名前を設定し、コピーしたメッセージをキー列に貼り付けます。


4. RTKLIBでリポジトリを取得する

hキーを設定した後、gitを使用してRTKLIBでリポジトリを追加できます（https://github.com/tomojitakasu/RTKLIB/tree/rtklib_2.4.3）

  1. mkdir rtk

    リポジトリ用のrtkディレクトリを作成します。オプションのアクション
  2. cd rtk

    rtkディレクトリに移動します。
  3. git clone git@github.com:tomojitakasu/RTKLIB.git -b rtklib_2.4.3

    ダウンロードする前に、ブランチの名前を確認してください。

cd ~/rtk/RTKLIB/app/rtkrcv/gcc
