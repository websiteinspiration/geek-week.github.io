AnonymousIdentityを使用したIdentityServer4での匿名jwtトークンのサポート


最近、ASP.NET CoreプラットフォームにOpenId ConnectとOAuth 2.0に基づく匿名ユーザー認証のサポートを実装する必要がありました。これらのプロトコルの仕様についてはここでは説明しません。これについては、Habrに関する完全な記事があります。要点に行きましょう。


なぜ匿名トークンが必要なのですか？さらに、特にマイクロサービスのアーキテクチャで、APIリソースで匿名ユーザーを承認するために、彼はアプリケーションの状態を変更できます。たとえば、Vasyaは猫のいるTシャツが好きで、オンラインストアのバスケットにそれを追加し、おそらくゲストとして注文します。それがVasilyであることを理解するために、匿名トークンには匿名ユーザーの識別子とセッション識別子が含まれています。Vasilyがログインすると、これらのパラメーターは認証済みトークンに含まれます。


さらに、匿名アクセストークンには、追加のステートメント（またはクレーム）を含めることができます。


ツール


  • OpenId ConnectとOAuth 2.0をサポートするIdentityServer4
  • IdentityServer4で匿名トークンをサポートするAnonymousIdentity

実装


IdentityServer4の GitHubには多くの例があります。


匿名トークンのサポートの追加は非常に簡単です。


  • IdneitityServer4を使用するプロジェクトにAnonymousIdentityをインストールする
    Install-Package AnonymousIdentity
  • Startup.csにAnonymousIdentityを登録する
    services.AddIdentityServer()
//  
.AddAnonymousAuthentication();

    response_mode=json
