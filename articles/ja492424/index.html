Angular：退会する別の方法

このタスクをAsyncPipeにシフトすることにより、コンポーネントコードのサブスクリプションを回避する必要がありますが、これが常に可能であるとは限りません。サブスクリプションを終了するにはさまざまな方法がありますが、すべて手動でサブスクリプションを解除するか、takeUntilを使用することになります。


時間の経過とともに、購読解除のためにデコレータを使用するようになりました。それがどのように配置され、適用されるかを考えてみましょう。この方法が好きかもしれません。


基本的な考え方は、サブスクリプションはメソッドから返される必要があるということです。それら。すべてのサブスクリプションは別の装飾されたメソッドで発生し、次の署名があります。


(...args: any[]) => Subscription;

Typescriptを使用すると、メソッドとその結果を変更できるメソッドにデコレータを掛けることができます。


3つの操作が必要になります。


  1. ngOnInitを呼び出すとき、ある種のサブスクリプションリポジトリを作成する必要があります。
  2. サブスクリプションを返す装飾されたメソッドを呼び出すとき、このサブスクリプションはリポジトリに保存する必要があります。
  3. ngOnDestroyが呼び出されると、リポジトリからのすべてのサブスクリプションが完了する（サブスクライブ解除）必要があります。

クラスメソッドデコレータがどのように作成されるかを思い出させてください。公式ドキュメントはこちら


デコレータの署名は次のとおりです。


<T>(target: Object, propertyKey: string | symbol, descriptor: TypedPropertyDescriptor<T>) => TypedPropertyDescriptor<T> | void;

デコレーターは、クラスコンストラクター、プロパティの名前、および記述子を入力として受け取ります。私は記述子を無視します。それはこのタスクには何の役割も果たしません。メソッド記述子を変更する人はほとんどいません。

function wrapHooks(target: ClassWithSubscription) {
  if (!target.hasOwnProperty(subSymbol)) {
    target[subSymbol] = null;
    wrapOneHook(target, 'OnInit', t => t[subSymbol] = new Subscription());
    wrapOneHook(target, 'OnDestroy', t => t[subSymbol].unsubscribe());

function wrapOneHook__ViewEngine(target: any, hookName: string, wrappingFn: (target: ClassWithSubscription) => void): void {
  const veHookName = 'ng' + hookName;
  if (!target[veHookName]) {
    throw new Error(`You have to implements ${veHookName} in component ${target.constructor.name}`);
  }
  const originalHook: () => void = target[veHookName];
  target[veHookName] = function (): void {
    wrappingFn(target);
    originalHook.call(this);
  };
}

export class ChildComponent implements OnInit {
  id: string;

  ngOnInit() {
    this.id = Math.random().toString().slice(-3);
    this.sub1();
    this.sub2();
  }

  @UntilOnDestroy()
  sub1(): Subscription {
    console.log(this.id, 'sub1 subscribe');
    return NEVER.pipe(
      finalize(() => console.log(this.id, 'sub1 unsubscribe'))
    )
      .subscribe();
  }

  sub2(): Subscription {
    console.log(this.id, 'sub2 subscribe');
    return NEVER.pipe(
      finalize(() => console.log(this.id, 'sub2 unsubscribe'))
    )
      .subscribe();
