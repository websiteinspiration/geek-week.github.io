マイクロサービスブログの作成-パート3「ユーザー」

一連の記事「マイクロサービスについてブログを書く」の第2部では、API Gatewayについて説明しました。

ここでは、ユーザーマイクロサービスの実装について説明します。

私たちのマイクロサービスは次のことができるはずです：

  • ログサービスの呼び出しとTraceIdを示す中間状態（api-gwによって発行されたものと同じ、パート2「APIゲートウェイ」を参照）
  • ログイン（SignIN）および登録（SignUp）関数を実装する
  • CRUD関数（データベース内のレコードの作成、読み取り、編集、削除）を実装します。データベースとしてMongoDBを使用します。

まず、protoファイル（./services/user/protobuf/user.proto）にサービスを記述します。
使用する構文を指定します-proto3。protobufパッケージの名前を示します。このパッケージでは、サーバーとクライアントのパーツの自動生成コードが実装されます。

アノテーションライブラリgoogle / api / annotations.protoをインポートします。RESTAPIを生成するためのディレクティブを記述するために必要になります。

syntax = "proto3";
package protobuf;
import "google/api/annotations.proto";

Userサービスの説明。直接持つ必要があるインターフェース（メソッド）。たとえば、SignUpインターフェース（登録）：ユーザー名、パスワード、FirstName、LastName属性を含むSignUpRequestメッセージを受け取り、Slug（ユーザーID）、ユーザー名、ロール属性を含むSignUpResponseメッセージで応答します。また、インターフェースの説明のオプションセクションで、postディレクティブを指定します。 / サインアップ。

これで既製のプロトファイルができました。プロジェクトのルートディレクトリに移動し、sh ./bin/protogen.shコマンドを実行します。このスクリプトはメインコードを生成します。
次に、。/ services / userディレクトリに移動し、functions.goファイルに宣言されたインターフェースの実装を記述します。

まず、ミドルウェアを実装します。サービスへのリクエストごとに、リクエストコンテキストからTraceId、UserId、UserRoleパラメータを引き出し、ログファイルに書き込みます。ここでは、リクエスト認証を実装できます。

//--------------------------------------------------
// Midelware
//--------------------------------------------------
func AccessLogInterceptor(ctx context.Context,req interface{},info *grpc.UnaryServerInfo,handler grpc.UnaryHandler,) (interface{}, error) {

    start:=time.Now()
    md,_:=metadata.FromIncomingContext(ctx)

    // Calls the handler
    reply, err := handler(ctx, req)

    var traceId,userId,userRole string
    if len(md["trace-id"])>0{
        traceId=md["trace-id"][0]
    }
    if len(md["user-id"])>0{
        userId=md["user-id"][0]
    }
    if len(md["user-role"])>0{
        userRole=md["user-role"][0]
    }
    
    msg:=fmt.Sprintf("Call:%v, traceId: %v, userId: %v, userRole: %v, time: %v", info.FullMethod,traceId,userId,userRole,time.Since(start))
    app.AccesLog(msg)

    return reply, err
}

//--------------------------------------------------

func (o *Server) DbConnect() error {
    var client *mongo.Client
    
    // Create client
    strURI:=fmt.Sprintf("mongodb://%s:%s@%s:%s",os.Getenv("MONGO_USER"),os.Getenv("MONGO_PASS"),os.Getenv("MONGO_HOST"),os.Getenv("MONGO_PORT"))
    client, err:= mongo.NewClient(options.Client().ApplyURI(strURI))
    if err != nil {
        return err
    }

    // Create connect
    err = client.Connect(context.TODO())
    if err != nil {
        return err
    }
    o.DbClient=client
    return nil
}
