1. SQL Server di Linux


SQL Server sekarang berjalan di Linux. Kursus ini akan membantu Anda mendapatkan pemahaman dasar tentang SQL Server di Linux. Kemudian Anda akan belajar cara menjalankan wadah SQL Server di Linux dan menggunakan SQL Server di Linux. Setelah itu, Anda akan belajar cara mengkonfigurasi penyebaran SQL Server di Linux.

Anda dapat memulai pelatihan di sini

2. Buat solusi analitik open source dengan Azure HDInsight


Dalam kursus ini, Anda akan belajar tentang HDInsight dan bagaimana menerapkan teknologi ini untuk menyelesaikan berbagai tugas di dunia nyata.

Di antara modul kursus:

  • Pengantar Penawaran Analytics Sumber Terbuka
  • Pilih Konfigurasi HDInsight yang benar untuk membangun solusi analitik sumber terbuka
  • Membuat dan mengonfigurasi cluster HDInsight
  • Lakukan analisis Zero ETL dengan Permintaan Interaktif HDInsight

Mulai pelatihan

3. Menerapkan solusi aliran data menggunakan Azure Streaming Analytics


Pelajari prinsip-prinsip pemrosesan acara dan streaming dan lihat bagaimana ini berlaku untuk Azure Stream Analytics. Kemudian siapkan pekerjaan Stream Analytics Anda untuk mengalirkan data dan pelajari cara memantau dan melacak pelaksanaan pekerjaan.

Mulai pelatihan

4. SQL Server Belajar Baru


Versi terbaru dari SQL Server memberikan banyak manfaat untuk peningkatan kinerja dan peningkatan fungsionalitas, serta dukungan yang ditingkatkan untuk platform database. Tutorial ini menunjukkan kepada Anda cara inventaris penyebaran SQL Server menggunakan Toolkit Penilaian dan Perencanaan Microsoft. Selain itu, Anda akan belajar cara memutakhirkan database SQL Server menggunakan Data Migration Assistant dan mengoptimalkan sistem Anda yang ditingkatkan menggunakan Database Experimentation Assistant dan Query Tuning Assistant.

Mulai pelatihan

5. Bekerja dengan data NoSQL di Azure Cosmos DB


Data NoSQL adalah cara efisien untuk menyimpan data yang tidak memenuhi persyaratan database SQL relasional. Cari tahu cara menggunakan portal Azure, ekstensi Azure Cosmos DB untuk Visual Studio Code, dan .NET Core SDK untuk Azure Cosmos DB untuk bekerja dengan data NoSQL di lokasi mana pun, serta memberi pengguna aksesibilitas tingkat tinggi di mana pun mereka berada.

Mulai pelatihan

6. Azure untuk insinyur data


Pelajari bagaimana dunia data berkembang dan bagaimana kemajuan teknologi cloud memberikan peluang bisnis baru. Anda akan belajar tentang berbagai teknologi platform data yang tersedia dan bagaimana seorang insinyur data dapat memanfaatkan teknologi ini untuk kepentingan organisasi.

Di antara modul:

  • Dunia data yang berkembang
  • Ikhtisar Layanan Platform Data Azure
  • Mendefinisikan tugas-tugas spesialis rekayasa data dalam arsitektur yang dihosting di cloud

Mulai pelatihan

7. Membuat dan menggunakan laporan analitik di Power BI


Kursus ini akan memperkenalkan Anda pada Power BI dan mengajarkan Anda cara menggunakan dan membuat laporan intelijen bisnis.

Di antara modul:

  • Memulai dengan Power BI
  • Mengambil data menggunakan Power BI Desktop
  • Pemodelan Data dalam Power BI
  • Menggunakan elemen visual di Power BI
  • Menjelajahi Data dalam Daya BI
  • Publikasikan dan bagikan dalam Power BI

Mulai pelatihan
