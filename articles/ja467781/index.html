CSセンターのC ++コース、前編

コンピューターサイエンスセンターは、POMI RASのコンピューターサイエンスクラブ、JetBrains、Yandex Data Analysis Schoolの共同イニシアチブです。

このセンターは、有能な学生や卒業生がコンピュータサイエンス、データサイエンス、ソフトウェアエンジニアリングなどの興味のある分野で成長できるようにするために存在しています。

中央の2部構成のC ++コースは、Valery Lesinが教えています。valery-l、CSMセンターの講師、ITMOとJetBrainsのジョイントマスタープログラム「ソフトウェアエンジニアリング/ソフトウェアエンジニアリング」、CTO Simlabs。

コースの最初の部分は C ++ベースを構築するために必要です。学期の終わりまでに、学生は問題を解決するためにこの言語で十分なプログラミングスキルを習得する可能性が高くなります。これまでのところ、移動のセマンティクスなしでは、sfinae、しかしこれを開始するには、おそらく、十分です。

C ++の使用経験のある学生にとって、このパートはギャップを埋めるのに役立ちます。たとえば、リンク、メモリの操作、ラムダ関数などのトピックです。講義では、確立された実践の回顧と、最新の言語標準が提供する可能性の概要の両方を提示します。コースは、さまざまなレベルの学生が自分のために新しい教材を見つけることができるように設計されています。
