1. GameMaker Studio 2.


Deskripsi:

  • Lisensi dan harga: $ 99 untuk lisensi abadi untuk mengembangkan game desktop, masa percobaan tersedia.
  • Ideal untuk: platformer 2D pendek dan RPG, game lintas platform.
  • Game yang dibuat: Nidhogg, Undertale, Risiko Hujan, Hyper Light Drifter.

GameMaker Studio 2 adalah hal pertama yang harus Anda perhatikan jika Anda ingin mulai mengembangkan game. Mesin memungkinkan pengembang untuk bekerja menggunakan bahasa scripting mereka sendiri.

Pro

Mark Essen, pencipta Nidhogg 1 dan 2, mengatakan bahwa GameMaker sangat bagus untuk pemula karena memiliki skrip terbuka dan sejumlah besar dokumentasi, yang memungkinkan pemula untuk merasa nyaman dengan cepat. Alex Preston, pencipta Hyper Light Drifter, mengatakan komunitas GameMaker yang besar adalah keuntungan besar. Pengembang muda harus mencari tempat terbaik untuk mendapatkan bantuan komunitas dan menggunakan teknik terbaik untuk mencapai tujuan mereka.

Minus

GameMaker sangat sederhana dalam kodenya, sehingga Anda dapat dengan cepat menjadi bingung dalam suatu proyek. Pada tahap awal, Anda dapat beralih dengan sangat cepat dan fokus pada desain game, tetapi jika Anda tidak mematuhi standar organisasi pribadi apa pun, Anda akan melihat sisi buruk dari membuat game di GameMaker. Duncan Drummond, pencipta proyek Risk of Rain yang benar-benar luar biasa, mencatat bahwa kesederhanaan platform bisa berubah menjadi masalah bagi pengembang. Anda dapat dengan mudah mengembangkan game, tetapi jika Anda melakukannya dengan salah, Anda akan kehilangan produktivitas dengan sangat cepat.
GameMaker bukan pilihan terbaik jika Anda ingin beralih ke mesin lain di masa depan, seperti Unity.

Tips Pemula

Cara terbaik untuk belajar adalah menghapus pekerjaan Anda dan mulai dari awal lagi. Jadi Anda dapat memompa keterampilan desain Anda. Untuk mempelajari cara mengembangkan game, Anda hanya perlu memulai. Bekerjalah, meskipun hasilnya buruk, cobalah untuk mendapatkan bantuan sebanyak mungkin. Ingat, semakin banyak kesalahan yang Anda buat, semakin banyak Anda belajar.

2. Persatuan


Deskripsi:

  • Lisensi dan biaya: versi awal gratis, Unity Plus - $ 35 per bulan, Unity Pro - $ 125 per bulan.
  • Ideal untuk: hampir semua game indie.
  • Game Dibuat: Ori and the Blind Forest, Cuphead, West of Loathing, Galak-Z, Pokemon Go.

Unity adalah salah satu platform pengembangan game indie utama. Meskipun memiliki kemampuan 3D yang luar biasa, game 2D juga dibuat di mesin. Unity lebih sulit dikuasai daripada mesin lain di pos ini, tetapi komunitas besar dan dokumentasi akan membantu Anda merasa nyaman dengan cepat.

Toko Unity memiliki banyak add-on yang akan membuat pengembangan game 2D di mesin ini semakin nyaman. Anda dapat menggunakan alat seperti Corgi Engine dan Rex Engine, yang menawarkan fisika dan kontrol yang tidak berlaku.

pro

Bahkan jika Anda terbiasa membuat Ira dengan cara kuno, Unity akan dapat menarik minat Anda. Dengan itu, Anda dapat dengan cepat membuat konsep dan prototipe permainan. Industri pengembangan game menggunakan banyak mesin sederhana dan kompleks. Unity, tentu saja, adalah salah satu mesin terbaik dan memungkinkan untuk mencapai produktivitas pengembangan tinggi. Ini dapat digunakan baik untuk membuat game kecil, dan dalam proyek AAA.

Kontra

Bahkan mesin ini memiliki kekurangannya. Jika Anda menemukan beberapa jenis bug, maka itu sepenuhnya tergantung pada pengembang mesin. Lagi pula, Anda harus menunggu sampai pengembang menyelesaikan masalah, karena proyek telah menutup kode sumber. Editor dan debugger jauh dari selalu stabil, meskipun pengembang telah secara terbuka menyatakan bahwa memperbaiki kesalahan tersebut adalah baris pertama dalam prioritas mereka.

Tips untuk pemula,

tidak selalu semuanya tergantung pada kemampuan untuk melakukan sesuatu. Penting untuk menetapkan tujuan yang tepat dan menemukan cara untuk mencapainya. Bahkan jika Anda tidak memiliki pengetahuan yang cukup untuk membuat game di Unity sekarang, cukup tetapkan tujuan.

3. Ren'Py


Deskripsi:

  • Lisensi dan biaya: gratis.
  • Ideal untuk: novel visual 2D, simulasi.
  • Kompatibel dengan: Python.
  • Game Dibuat: Analog: Kisah Benci, Hidup Sang Ratu.

Ren'Py adalah mesin open source yang mudah dipelajari. Meskipun membutuhkan pengetahuan pemrograman, pada kenyataannya Anda perlu tahu cara menggunakan editor teks dan program pemrosesan gambar. Bekerja dengan mesin ini akan sangat berguna bagi pemula bahasa Python.

pro

Ren'Py adalah mesin lintas platform yang membuka peluang besar bagi pengembang. Ini nyaman untuk pemula, tetapi membutuhkan beberapa keterampilan mengedit untuk file skrip. Ya, Anda tidak bisa hanya menarik dan melepas barang yang Anda butuhkan, tetapi menulis kode tidak menakutkan. Mesin ini memungkinkan Anda untuk memecahkan masalah utama pengembang pemula - ketakutan untuk memulai. Pengembangan cepat dan mudah akan membantu Anda membuat game pertama Anda. Bahkan jika di masa depan Anda ingin menulis sesuatu yang berbeda dari novel, Ren'Py akan memberi Anda pengalaman yang berharga. Ketika orang lain memainkan permainan yang Anda buat, sikap terhadap diri Anda dan kemampuan Anda berubah.

Minus

Mesin agak terbatas dalam mendukung fungsi grafis dan mekanik. Sangat bermasalah untuk membuat game dengan 3D atau Live2D, Anda akan menghadapi banyak masalah, jadi jika ini adalah tujuan Anda, pilih platform yang berbeda.

4. Tinta


Deskripsi:

  • Lisensi dan biaya: gratis.
  • Ideal untuk: game petualangan teks.
  • Kompatibel dengan: Unity, C #, HTML.
  • Game Dibuat: Sihir, 80 hari.

Tinta adalah add-on gratis untuk Unity. Sangat cocok untuk membuat dialog bercabang, cerita. Tinta mudah dipelajari dan diintegrasikan dengan Unity.
Tinta dibuat sebagai perangkat lunak opsional. Artinya, Anda membuat game di Unity, dan kemudian menghubungkan skrip dan dialog yang dibuat menggunakan Tinta.

Plisov Sangat

sulit untuk menulis skrip terperinci dan sistem dialog bercabang. Tinta membantu untuk tidak bingung dan secara serius menyederhanakan kehidupan pengembang. Tinta akan menarik bagi mereka yang lebih tertarik untuk menulis cerita untuk game daripada pemrograman murni.

Kontra

Menggunakan tinta sebagai mesin mandiri penuh sangat sulit, sehingga sebagian besar pengembang menganggapnya sebagai tambahan untuk proyek mereka.
