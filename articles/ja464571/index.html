彼は彼自身のものです-私たちの30分のVPS


安価なVPSが不要な理由について3〜4年前に書いた競合他社やユーザーからの記事を読みました。ええ、そうですね、「1ペニー」のVPSは純粋なマーケティングであり、通常の作業機会を提供できませんでした。しかし、時代は変わり、仮想リソースのコストは低くなり、毎月30ルーブルでこれを提供する準備が整いました。

  • プロセッサー：Intel Xeon 2 GHz（1コア）
  • Linuxシステム（Debian、Ubuntu、CentOSから選択）
  • 1つの専用IPv4アドレス
  • エンタープライズクラスの高速SSDでのデータストレージ用に10 GB
  • RAM：512 MB
  • 秒単位の請求
  • 無制限のトラフィック

関税は、追加の技術的制限、当社のクールなオファーのページの詳細-30ルーブルのVPSの対象となります。

- VPS 30 —


30ルーブルは非常に小さいので、カードに支払いをしてテストするのをためらうことさえあります。私たちは時々とても怠惰ですが、今回はあなたのためにすべてをしました。戦闘でサーバーを起動する前に、すべての詳細をチェックし、この料金表でサーバーが何ができるかを示すテストを実施しました。より興味深いものにするために、エクストリームスポーツを追加し、密度と負荷が私たちが設定した値を超えた場合のこの構成の動作を確認しました。 

ホストは、プロセッサでさまざまなタスクを実行し、ディスクサブシステムをアクティブに使用する多数の仮想マシンの負荷がかかっていました。目標は、高い配置密度と、戦闘と同等かそれ以上の荷重をシミュレートすることです。

一定の負荷に加えて、sysbenchを使用して合成メトリックを収集する3つの仮想マシンをインストールしました。その平均結果は以下のとおりです。追加の負荷を作成した50の仮想マシン。すべてのテスト仮想マシンは同じ構成（1コア、512 GB RAM、10 GB SSD）であり、RUVDSのユーザーに提供される標準のdebian 9.6イメージがオペレーティングシステムとして選択されました。

負荷は、戦闘に匹敵する性質とサイズでシミュレーションされました。

  • 仮想マシンの一部が低負荷で起動されました。
  • 一部のマシンは、プロセッサの負荷をシミュレートするテストシナリオを（ねじれた）（ストレスユーティリティを使用して）
  • 残りの仮想マシンでは、ddを使用してスクリプトを実行し、pvを使用して制限を設定して、事前に準備されたデータからディスクにデータをコピーしました（例はこちらとこちらにあります）。

また、覚えているように、合成メトリックを収集する3台の車がありました。

スクリプトは、15分ごとに周期的に各マシンで実行され、プロセッサ、メモリ、およびディスクの標準sysbenchテストを実行します。

スクリプトsysbench.sh

プリミティブテストサービスはpythonでひざまずいて書かれており、次の4つの操作を実行します。

  1. getState：ステータスを返します
  2. getEmployee：従業員データベースからの戻り値（+給与、+役職）
  3. patchEmployee：従業員フィールドを変更します
  4. insertSalary：給与の挿入を実行します

サービスソース（dbtest.py）
#!/usr/bin/python
import mysql.connector as mariadb
from flask import Flask, json, request, abort
from mysql.connector.constants import ClientFlag

app = Flask(__name__)

def getFields(cursor):
    results = {}
    column = 0
    for d in cursor.description:
        results[d[0]] = column
        column = column + 1
    return results

PAGE_SIZE = 30

@app.route("/")
def main():
    return "Hello!"

@app.route("/employees/<page>", methods=['GET'])
def getEmployees(page):
    offset = (int(page) - 1) * PAGE_SIZE
    connection = mariadb.connect(user='admin', password='q5XpRomdSr', database='employees')
    cursor = connection.cursor()
    cursor.execute("SELECT * FROM employees LIMIT {} OFFSET {}".format(PAGE_SIZE, offset))
    return {'employees': [i[0] for i in cursor.fetchall()]}

@app.route("/employee/<id>", methods=['GET'])
def getEmployee(id):
    id = int(id)
    connection = mariadb.connect(user='admin', password='q5XpRomdSr', database='employees')
    cursor = connection.cursor()
    cursor.execute("SELECT * FROM employees WHERE emp_no = {}".format(id))
    fields = getFields(cursor)
    employee = {}
    found = False
    for row in cursor.fetchall():
        found = True
        employee = {
            "birth_date": row[fields["birth_date"]],
            "first_name": row[fields["first_name"]],
            "last_name": row[fields["last_name"]],
            "gender": row[fields["gender"]],
            "hire_date": row[fields["hire_date"]]
        }
    if not found:
        abort(404)
    cursor.execute("SELECT * FROM salaries WHERE emp_no = {}".format(id))
    fields = getFields(cursor)
    salaries = []
    for row in cursor.fetchall():
        salary = {
            "salary": row[fields["salary"]],
            "from_date": row[fields["from_date"]],
            "to_date": row[fields["to_date"]]
        }
        salaries.append(salary)
    employee["salaries"] = salaries
    cursor.execute("SELECT * FROM titles WHERE emp_no = {}".format(id))
    fields = getFields(cursor)
    titles = []
    for row in cursor.fetchall():
        title = {
            "title": row[fields["title"]],
            "from_date": row[fields["from_date"]],
            "to_date": row[fields["to_date"]]
        }
        titles.append(title)
    employee["titles"] = titles
    return json.dumps({
        "status": "success",
        "employee": employee
    })

def isFieldValid(t, v):
    if t == "employee":
        return v in ["birdth_date", "first_name", "last_name", "hire_date"]
    else:
        return false

@app.route("/employee/<id>", methods=['PATCH'])
def setEmployee(id):
    id = int(id)
    content = request.json
    print(content)
    setList = ""
    data = []
    for k, v in content.iteritems():
        if not isFieldValid("employee", k):
            continue
        if setList != "":
            setList = setList + ", "
        setList = setList + k + "=%s"
        data.append(v)
    data.append(id)
    print(setList)
    print(data)
    connection = mariadb.connect(user='admin', password='q5XpRomdSr', database='employees', client_flags=[ClientFlag.FOUND_ROWS])
    cursor = connection.cursor()
    cursor.execute("UPDATE employees SET {} WHERE emp_no = %s".format(setList), data)
    connection.commit()
    if cursor.rowcount < 1:
        abort(404)
    return json.dumps({
        "status": "success"
    })

@app.route("/salary", methods=['PUT'])
def putSalary():
    content = request.json
    print(content)
    connection = mariadb.connect(user='admin', password='q5XpRomdSr', database='employees', client_flags=[ClientFlag.FOUND_ROWS])
    cursor = connection.cursor()
    data = [content["emp_no"], content["salary"], content["from_date"], content["to_date"]]
    cursor.execute("INSERT INTO salaries (emp_no, salary, from_date, to_date) VALUES (%s, %s, %s, %s)", data)
    connection.commit()
    return json.dumps({
        "status": "success"
    })


@app.route("/state", methods=['GET'])
def getState():
    return json.dumps({
        "status": "success",
        "state": "working"
    })

if __name__ == '__main__':
    app.run(host='0.0.0.0',port='5002')
