均一で加速された動き


動きから始めましょう。

均一な動きの場合、次のコードを使用できます。

function move(dt) {
    x += vx * dt
    y += vy * dt
}

ここで、xとyはオブジェクトの座標、vxとvyはそれぞれ水平軸と垂直軸に沿ったオブジェクトの速度です。dt（時間デルタ-時間デルタ）は2つのタイマーマーク間の時間で、JSではrequestAnimationFrameへの2回の呼び出しに相当します。

たとえば、座標150、50のポイントにあるオブジェクトを南西に移動する場合は、次の操作を実行できます（1つのタイマーマークまたは1つのステップ）。

x = 150 += -1 * 0.1 - > 149.9
y = 50 += 1 * 0.1 - > 50.1

動きさえ退屈なので、オブジェクトに加速を与えましょう：

function move(dt) {
    vx += ax * dt
    vy += ay * dt
    x += vx * dt
    y += vy * dt
}

ここで、axおよびayは、それぞれx軸およびy軸に沿った加速度です。加速を使用して速度を変更します（vx / vy）。ここで、前の例を使用して、x軸に沿って（西へ）加速度を追加すると、次のようになります。

vx = -1 += -1 * 0.1 - > -1.1 // vx += ax * dt
vy = 1 += 0 * 0.1 - > 1 // vy += ay * dt
x = 150 += -1.1 * 0.1 - > 149.89 // x += vx * dt;     -0.01
y = 50 += 1 * 0.1 - > 50.1 // y += vy * dt

衝突


動いている体が衝突することがあります。衝突により、オブジェクトが他のオブジェクトによって押し出されるか、一部のオブジェクトが他のオブジェクトから跳ね返ります。まず



、プッシュについて説明しましょう。まず、衝突があったことを確認する必要があります。

class Collision {
    constructor(o1, o2, dx, dy, d) {
        this.o1 = o1
        this.o2 = o2

        this.dx = dx
        this.dy = dy
        this.d = d
    }
}

function checkCollision(o1, o2) {
    let dx = o2.x - o1.x
    let dy = o2.y - o1.y
    let d = Math.sqrt(Math.pow(dx, 2) + Math.pow(dy, 2))
    if(d < o1.r + o2.r){
        return {
            collisionInfo: new Collision(o1, o2, dx, dy, d),
            collided: true
        }
    }
    return {
        collisionInfo: null,
        collided: false
    }
}

2つの衝突するオブジェクトを表すCollisionクラスを宣言します。checkCollision関数では、最初にオブジェクトのx座標とy座標の差を計算し、次にそれらの実際の距離dを計算します。オブジェクトの半径の合計がオブジェクト間の距離よりも小さい場合、これらのオブジェクトの衝突があった-Collisionオブジェクトを返します。
