Aktualisiertes Web Security-Testhandbuch des OWASP-Konsortiums



Das WSTG-Projekt (Web Security Testing Guide) ist die primäre Sicherheitstestmethode für Entwickler von Webanwendungen und Fachleute für Informationssicherheit und wird vom internationalen OWASP-Konsortium entwickelt.

Das Open Web Application Security Project (OWASP) ist ein Open Source-Sicherheitsprojekt für Webanwendungen. Zur OWASP-Community gehören Unternehmen, Bildungseinrichtungen und Einzelpersonen aus aller Welt. Die Community arbeitet an der Erstellung frei zugänglicher Artikel, Handbücher, Dokumentationen, Tools und Technologien.

WSTG ist ein umfassender Leitfaden zum Testen der Sicherheit von Webanwendungen und Webdiensten. Die WSTG, eine Zusammenarbeit von Cybersicherheitsfachleuten und Freiwilligen, bietet eine Reihe von Best Practices, die von Pentestern und Organisationen auf der ganzen Welt angewendet werden.

Die aktuelle Version - [Version 4.1] - 2020-04-2, die vorherige Version - [Version 4.0] - 2014-09-17 - zwischen den Aktualisierungen sind ungefähr 6 Jahre vergangen, die eine große Schicht moderner Technologien abdecken sollten. Grundlegende Änderungen werden jedoch nur in erwartet 5 Versionen von WSTG.
