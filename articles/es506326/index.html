Nuevos seminarios web y consultas gratuitas de expertos de Lenovo Data Center Group

Hace algún tiempo, ya hablamos sobre una serie de reuniones en línea organizadas por especialistas del Grupo de centros de datos de Lenovo. El objetivo principal de estos eventos es contar en un lenguaje simple y accesible sobre tecnologías y soluciones para centros de datos de cualquier escala: definir tareas, diferencias de enfoques, elegir, configurar y administrar ofertas de Lenovo y mucho más. No solo teoría, sino también una práctica tangible + bonificación: los participantes en las reuniones tienen la oportunidad de recibir consejos individuales de expertos de Lenovo, tanto en el seminario web como después por correo electrónico / teléfono.

El seminario web más cercano es el 16 de junio a las 15:00. La presentación discutirá la pila de software de control de supercomputadora del Orquestador de Computación Inteligente de Lenovo ( LiCO).) Además, Andrey Sysoyev, gerente de producto de HPC & AI, hablará sobre el sistema de almacenamiento de datos con el sistema de archivos paralelos DSS-G y la tecnología de refrigeración líquida.
