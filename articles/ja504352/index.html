Figmaが新しい設計結果の達成などに貢献した10社



みなさん、こんにちは！次の記事では、Figmaの使用が新たな高みへの道の出発点となった10社のリストを紹介します。

これらは、現代の設計ツールが決定的な役割を果たす、問題と解決策の物語です。

おそらく、あなたはあなた自身の有望なプロジェクトを開発することを計画していて、どのツールを使用するかを考えています。または、設計やプロトタイピングに他のツールを使用している会社で働いています。いずれにせよ、この記事を読んだ後で、Figmaを使い始める10の理由があります。

フィグマのウェブサイト、ユーチューブ、以下の企業の公式ウェブサイト、その他のオープンソースのデータを使用して、情報の調査と分析に何時間も費やしました。

評価は小さな会社から始め、Figmaを使用する大手企業で終わります。

じゃ、行こう！

10. 概念




従業員数：30人以上。

業界：個人の生産性と同僚とのコラボレーション。

何が問題でしたか：現在、Notionは個人の生産性とコラボレーションのための主力アプリケーションの1つです。しかし2015年には、すべてがかなり暗くなり、スタートアップとしてのNotionはほとんど死にました。そのバージョンのアプリケーションの技術スタックの可能性は不十分であり、それが常に問題を引き起こしていました。失敗は、「人々の日常の問題を解決するアプリケーション」と彼らが主張したように、作成を夢見ていたNotionの作成者であるIvan ZhaoとSimon Lastの野心にぶつかりました。

Figmaが問題の解決にどのように貢献したか：2018年にNotion 1.0がリリースされる前に、緊急にUXを確認し、アプリケーションのUIを調整し、多くの眠らない夜を過ごして、図でデザインを作成し、コード行に転送する必要がありました。Notionのユーザーエクスペリエンスは非常に優れているため、UXの歴史の中で最高の1つのタイトルを獲得しています。Notionの作成者は、Figでのリアルタイムのコラボレーションが重要な役割を果たしたと言います。同社は現在100万人以上のユーザーを抱えています。

9. ドリブル




従業員数：40人以上

支店：デザイナー向けオンラインプラットフォーム

問題： Dribbbleはリモート作業の促進者です。この会社のすべての従業員がリモートで作業しています。従業員は都合がよく、柔軟なスケジュールで、世界中のどこからでも作業できます。しかし、このモデルでも、異なる場所にいる異なるタイムゾーンの従業員が同じ部屋にいるかのようにうまく調整されたコラボレーションを作成する必要がある場合、ドリブル管理は問題に直面しました。

figmaがいかに役立ったか：彼女はこの問題を解決できる唯一のツールになりました。Dribbbleのすべての従業員は、同じファイルをリアルタイムで作業できることを喜んでいました。ワークフローが加速され、チームは設計に直接焦点を当てることができ、ファイル管理の時間の無限の無駄に集中することはできませんでした。さらに、Figmaはデザイナーと開発者のチームだけでなく、すべての利害関係者が共同で問題を解決することを可能にしました。

その結果、DribbbleはFigでデザインを作成するだけでなく、そのAPIを統合に使用するようになりました。これにより、数回クリックするだけで、作品をFigからDribbbleのアカウントに直接送信できます。

8. 電球




従業員数：500人以上

業界：再生可能エネルギーの供給

問題：電球は再生可能エネルギーのサプライヤーであり、ニッチ市場で大きな成功を収めています。しかし、経済的な成功とブランドの認知にもかかわらず、電球のワークフローには多くの矛盾がありました。会社のすべての製品は異なる機関によって作成され、もちろん、各製品はデザインとコードの両方の点で異なるテンプレートを持っていました。電球は違いを生む必要がありました。

Figmaによる支援：同社は独自の統一設計システムを作成することを決定しました。これは、チームのすべての設計者と開発者のためのガイドとして機能します。その結果、スタイリッシュで非常に使いやすいソーラー設計システムが実現しました。。Bulbにとって、デザイナー、プログラマー、カスタムリサーチスペシャリストなど、会社の全従業員が設計システムの作成プロセスに関与し、すべてがどのように機能するかを理解することが非常に重要でした。

その結果、図1のおかげで、会社は作業プロセスの設計スプリントを数倍に増やすことができました。

7. ガスト




従業員数：500人以上

業種：中小企業向けオンラインプラットフォーム

問題の内容： Gustoは、給与計算、健康保険、チーム管理ツールなどの中小企業を支援しているため、有能で高品質なUXが必要でした。また、ユーザーはプラットフォームの機能を気に入っていましたが、プラットフォームの使用経験は会社のユーザーサービスの哲学を完全には反映していないことを理解していました。

figmaがいかに役立ったか：ブランド変更は簡単な作業ではありません。あらゆる段階ですべてをさまざまな利害関係者と調整する必要があり、会社は問題の解決策としてFigmaを使用することにしました。さらに、コラボレーション機能のおかげで、Gustoチームは、ワークフローがどの段階にあるのか、なぜ特定の決定が行われるのかを全員に知らせました。ブランド変更の結果、同社はより認知され、現在、10万人以上の顧客に満足しています。

6. GitHub




従業員数：1,000人以上

業種：ITプロジェクトとその共同開発のホスティング

問題： GitHubホスティング自体は、何百万人もの開発者、デザイナー、製品マネージャーが共同で問題を解決し、協力するという点で優れたプラットフォームです。それにもかかわらず、同社は他の多くの企業と同じ問題に直面しました-GitHubの従業員はリモートベースで作業しており、人々が同じ物理的な空間にいるかのように、人々が互いに協力する条件を作成する必要がありました。さらに、ドキュメントがなく、テンプレートが古く、設計者が単一のシステムを持っていなかったため、作業プロセスを最適化する必要がありました。設計者は同じコンポーネントを何度も繰り返し作成しました。

figmaがいかに役立ったか：同社は統一された設計システムを作成し、作業プロセスの文書化を開始しました。また、FigmaとそのAPIの組み合わせのおかげで、GitHubユーザーがプロジェクトに変更を加える必要があったときに、反復と更新の問題が解決されました。これからは、ファイルを更新する他のソフトウェアを見つけてインストールする必要がなくなります。そしてもちろん、単一のリンクの助けを借りてすべての従業員がリアルタイムで特定のファイルを表示または編集できるため、フィードバックプロセスが加速しました。

現在、GitHubの全従業員は既にFigmaを自社の文化の一部であると考えており、このツールなしでは実現できません。

5. ズーム




従業員数：2,000人以上

業界：ビデオコミュニケーション、オンライン会議によるコラボレーション

問題：会社の開発が成功したにもかかわらず、Zoomの従業員はさまざまなツールを使用していたため、作業プロセスは理想的とは言えませんでした。素材を開発者に転送するためのプロトタイプとZeplin。あるツールから別のツールに切り替えると、ワークフローが大幅に遅くなり、本当に重要なことに集中することができなくなりました。

figmaがいかに役立ったか：Zoomは上記のすべてのツールを1つのパッケージで受け取り、チームは別のプログラムを使用するために引き裂かれる必要がなくなり、作業が大幅に加速し、プロジェクトで大幅な進歩を遂げました。さらに、Figmaの使いやすさのおかげで、同社の設計者はスキルを向上させ、より正確なコンポーネントを作成する方法を学びました。

4. Deliveroo




従業員数：5,000人以上

業界：食品配達

問題： Deliverooは、世界中の500を超える都市で事業を展開している最も急速に成長している企業の1つと考えられています。アプリケーションのUIとUXはとても快適で使いやすいです。ただし、DeliverooチームはZoomチームと同じアプローチを使用しました。Zoomチームは多くの異なるツールを使用したため、効果的に連携することができず、アプリケーションの異なるバージョンでのファイル同期で問題が発生しました。

figmaがいかに役立ったか：Deliverooの従業員はすぐにこのツールの力と効果を感じました。製品の設計システムを作成したDeliverooチームは、大幅に改善されたコラボレーション、およびFigmaが提供する機能（スマートセレクションなどの些細なことからファイルを同期する機能まで）に満足し、アイデアから完成への移行プロセスを実現しました製品ははるかに速くて簡単です。

テレグラムチャンネル「デザイナーに役立つ」を購読すると、以下の投稿やその他の多くの有用なデザインリソースが発表されます。私は常に遅れないようにして、デザイナー、開発者、チームマネージャーの加速を支援するアナウンス、ツール、アプリケーションのリリースに従います。

figmaがいかに役立ったか：AirBnBの従業員がFigmaを使い始めたとき、彼らは彼らがいつも探していたまさにそのツールを見つけたことに気づきました。ブラウザでの作業、より高速なフィードバック、「マルチプレイヤー」モード、同僚がいつでも誰とでもファイルを共有できるとき、およびその他のFigmaの優れた機能により、全員の生産性が向上し、会社の一体感が生まれました。FigmaはAirBnBの開発に大きな影響を与え、オープンデザインの文化を加えました。
