Visual Studio for Mac：IDEをキーで制御する

コンピューティングにおける大きな議論には、1つの共通点があります。タブとスペース、またはViとEmacsのどちらに直面しても、この議論を結びつけるスレッドはキーボードの効率です。真実は、プログラムでの作業に何時間も費やしている場合、ホットキーの使用は、優れたピアニストやアスリートが持っているのと同じ筋肉の記憶である、自動的なものになるということです。

突然、巨匠ピアニストにピアノの鍵盤の幅が2倍になり、鋭い鍵盤や平らな鍵盤が自然な鍵盤より低く、高くない場合、新しいアレンジを勉強している間、最も単純なメロディーでさえ演奏する必要があります。同様に、お気に入りのIDEのキーボードショートカットについても、変更を加えるとすぐに混乱してしまいます。さいわい、Visual Studio for Macでは、キーボードショートカットを好みに合わせてカスタマイズできるように、キーバインディングを幅広くカスタマイズできます。



開始


Visual Studio for Macを初めて使用するユーザーは、IDEが多くの異なるキーボードショートカットをサポートしていることにすぐに気づきます。これは、Visual Studio for Macを初めて起動したときに、最も便利なオプションを選択するように求められるためです。

コードの最初の行から最も生産性の高い作業を行うためのショートカットには、4つのオプションがあります。うーん、それはあなたにとって十分ではありませんか？まあ、Visual Studio for Macもその準備ができています！



その他の設定


デフォルトのレイアウトオプションを選択することは確かに便利ですが、それでも、すべての場合の唯一の解決策ではありません。他のIDEで使用したユーザーの組み合わせ、または事前構成されたパラメーターの配列の外にある特定のコマンドがある場合があります。 [キーバインディング]ウィンドウを使用すると、IDEで使用可能なすべてのコマンドを特定の組み合わせにマップできます。キー割り当てオプションを表示するには、[ Visual Studio]> [設定]> [環境]> [キー割り当て]を選択します。



このウィンドウでは、いくつかの機能に注目し、それぞれについて順番に説明します。最初に気づいたのは、新しいユーザーがIDEを最初にインストールしたときに表示されるパラメータにマップされているさまざまな「スキーム」のドロップダウンリストがあることです。この機能を使用すると、「Visual Studio」、「VS Code」、「Xcode」など、キーバインディングの多くのオプションの1つを選択できます。



しかし、これで十分でない場合はどうでしょうか。Control-Option-Dで「Find Derived Symbols」を本当に開きたい場合？問題じゃない。 Visual Studio for Macでは、キーボードショートカットのカスタマイズは非常に簡単です。開始するには、使用可能なコマンドのリストをスクロールするか、検索フィールドで目的のコマンドを検索します。使用可能なコマンドのリストはカテゴリに分類されており、ナビゲーションを簡略化するために折りたたむことができます。目的のコマンドを見つけたら、それを選択し、「バインディングの編集」に目的のキーバインディングを入力して、「適用」をクリックします。下のGIFでは、「新しいブレークポイント」のバインディングをControl-Shift-Bに設定しています。



同様に、既存のブレークポイントを編集することもできます。下のGIFで、「新しいファイル」コマンドを編集して、標準ではなくControl-Shift-Nの組み合わせで実行する方法を確認できます。Command-N。私がしなければならなかったことは、私が好むコマンドを入力して「適用」をクリックするだけであることに気付くでしょう。複数の異なる組み合わせを追加する場合は、「適用」ではなく「追加」をクリックします。



最後に、覚えるコマンドが非常に多いため、重複を避けるのが難しい場合があります。各組み合わせが一意であることを確認するために、Visual Studio for Macは構成されたすべてのコマンドをチェックし、重複が検出された場合は警告を出します。 IDEは重複するコマンドもチェックするため、元のバインディングまたは新しく作成されたバインディングを置き換えることができます。以下のGIFは、「新しいブレークポイント」コマンドが「コピー」と競合する「Command-C」キーボードバインディングにマップされたときに何が起こるかを示しています。



キーボードショートカットを便利な方法で編集する方法を理解したので、IDEを完全にカスタマイズして、最も便利なコードの記述を開始できます。キーバインディングの詳細については、チャンネル9のこのトピックに関するビデオをご覧ください。
