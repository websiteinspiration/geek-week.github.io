鉱山掘削機の自動制御システム



前書き


ショベルは市内のどの建設現場でも見ることができます。通常の掘削機は一人の運転手が担当します。それを管理するには、複雑な自動化システムは必要ありません。

しかし、掘削機が何倍も馴染みがあり、5階建ての建物の高さに達している場合、ランドクルーザーをバケットに配置できます。「充填」は、電気モーター、ケーブル、および車のサイズのギアで構成されますか。そして、彼は石炭と鉱業の採石場で24時間年中無休で働いていますか？

そのような掘削機は産業システムであり、その維持費は非常に高価です。

技術プロセスの自動化は、産業システムの運用コストを削減します。自動プロセス制御システムはACS TPと呼ばれています。説明したものと同様の掘削機も例外ではありません。

これはどのような掘削機ですか？どのプロセス制御システムが使用されていますか？

どのような掘削機について話しているのですか？


私たちは、掘削機の採掘について話しています。これらの機械は、採鉱および採石場を開発しています。

寸法：マイニングショベルは、5階建ての建物の高さに達します。

移動：掘削機は、追跡された着陸装置の助けを借りて移動します。カートは以下で構成されています：

  • 追跡フレーム;
  • 毛虫
  • 旅行ドライブ;
  • 車台の潤滑回路。

掘削：マイニングショベルは、直接シャベルメカニズムを使用して掘削します。メカニズムはバケツ、棒およびブームから成っています。バケットはハンドルに取り付けられています。ハンドルはバケットに並進運動を与えるように設計されています。ブームを横切る位置にあります。ブームには圧力機構が搭載されており、バケットでハンドルの圧力と戻り動作を行います。ロープの複雑なシステムがこのメカニズムを動かします。



デバイス（構成）：掘削機は3つの拡大ノードで構成されています。

  • 作業機器;
  • 機構付き回転プラットフォーム;
  • トロリーを実行しています。

作業機材については上記で説明しています。これがまさに「ダイレクトショベル」のメカニズムです。

鉱山掘削機は、掘削、機械本体の回転、移動など、多くの操作を実行します。オペレーションごとに個別のエンジンが設計されています。これらすべての操作を実行するには、膨大な数のシステムが必要です。予想通り、すべてのシステムとメカニズムは「コンピュータルーム」にあります。

掘削機の機械室はターンテーブルです。これには、バケットリフト機構、回転機構、制御および監視システムを備えた掘削機の電気機器、補助機構、空気圧システム、集中自動潤滑システムが含まれています。

労働条件と耐用年数：鉱山掘削機は24時間年中無休で稼働し、耐用年数は実際には30〜40年です。

食糧/燃料：鉱山のシャベルは電気で動く。セクションの各山岳セクションは、変電所から35/6 kVの電圧で電気を受け取ります。

掘削機にはどの

ような自動化機能が搭載されていますか？ホイール式掘削機は産業用システムです。掘削機を操作するタスクは、産業施設を操作するタスクと同様です。

  • 運動システムのパラメータの制御;
  • 機器摩耗監視;
  • 外部および内部の脅威からの機器の保護：過負荷、短絡など。
  • エネルギー計測;
  • 掘削機の位置制御;
  • 運転中の機器の検査;
  • 死角制御;
  • 掘削機のパフォーマンス監視;
  • イベントログ。
  • 集中会計のためのデータ転送。

これらすべてのタスクは、1人のオペレーターによって処理されます。これは自動化によって可能です。

掘削機の「オンボード」自動制御システムには、次のシステムが含まれます。コントローラーは

、運動のパラメーターを監視するためにインストールされます。オペレーターは、駆動制御システムの操作、システムコンポーネントの加熱温度、空気圧システムの圧力、およびグリースのパラメーターを監視します。

消費されて与えられた有効電力量と無効電力量を計算するために、電気メーターが設置されています。オペレーターの画面には

、ブラインドゾーン、機械設備の操作、作業面が表示されます。このため、ビデオカメラがインストールされています。掘削機の性能

を計算して説明するにはコントローラからのデータが使用されます。指標は特定の時間間隔で計算されます：シフトごと、月ごと、チームごと。

すべてのイベントはイベントログに保存され、必要な時間間隔が保存されます。

データ転送はどのように構成されていますか？


上記のように、掘削機はトロリーとターンテーブルで構成されています。

ターンテーブルは、車台に対して360度自由に回転できます。これら2つのパーツ間でデータを転送するためにワイヤーを使用することは非常に問題があります。彼らは非常に速く粉砕します。

掘削機のパーツ間のデータはWi-Fi経由で送信されます。フェニックスコンタクトの Wi-Fi WLAN 5100機能モジュールは、特別なRAD-CAB-EF393-10MケーブルとRAD-ISM-2459-ANT-FOOD-6-0-N全方向性アンテナとともに、ベースとターンテーブルに取り付けられています。掘削機には合計3本のアンテナが設置されており、安定した通信が可能です。4G TC ROUTER 3002T-4Gルーター

も掘削機に設置されていますTC ANT MOBILE WALL 5M広指向性アンテナおよびサージ保護デバイスCSMA-LAMBDA / 4-2.0-BS-SETを使用。
