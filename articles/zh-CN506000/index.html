DIY心电图背包


现代大学的校园有一个共同点：学生密集！

如今，由于社会和科学方面的压力，当今的学生经常一次在多个领域之间挣扎。因此，许多学生感到自己无法应付，最终在科学，体育和交流方面均表现不佳。

这个学期，我们的班级研究了各种生物测量仪器，这些仪器可以测量患者身体发出的各种生物信号，例如肌肉收缩，脑波和心脏波。我们想开发一种医疗工具，该工具可用于根据脉搏跟踪学生的压力水平。我们的“ ECG背包”配备了ECG传感器，可以监测患者的脉搏，并通过LED和通过蓝牙与之通信的电话应用程序提供即时反馈。基于静息脉搏，当心电图背包的压力水平比正常水平高得多时，ECG背包将通过LED来警告用户。来自移动应用程序的数据可以导出到Excel，并用于进一步分析。

步骤2：准备



一个心动周期

要了解ECG背包的所有可能性，您首先必须对它实际记录ECG的内容有基本的了解。心电图记录来自心脏的电信号。在每个心脏周期中，心脏都会经历几个不同的心脏收缩（收缩）和心脏舒张（松弛），并且每个心脏都对应一个特定的波动。在研究心电图时，我们看到三个主要部分：P波，QRS复合波和T波。它们如图所示。更高。两个QRS复合体之间的时间使我们能够确定心率。所有这些都将在代码中加以考虑。

除了对ECG方法有基本了解之外，您还必须能够对Arduino进行编程。您将需要添加Flora板支持。。最后，您需要从GitHub下载用于Adafruit蓝牙的库。

步骤3：安全性




使用电子电路时，必须注意一些预防措施：

  • 通过更换电路，确保电源已关闭。
  • 请勿在电路附近放置液体或食物。
  • 运动时请勿携带ECG背包，以免电线从板上脱落或汗水渗入电子设备。
  • 使用绝缘电线和工具。

注意！这不是医疗设备-只是跟踪您的心率的工具。请咨询您的医生以获取更准确的测量结果和提示。

步骤9：焊接Bluefruit蓝牙模块上的按钮

步骤10：缝制按钮并将Bluefruit蓝牙连接到Flora




首先，布置按钮以使其与微处理器和蓝牙模块上的按钮位置一致是非常重要的。可以使用粉笔来完成-用粉笔擦每个按钮，然后将木板按到正确的位置。

之后，从背包的背面开始，用导电线缝制按钮，然后将其穿过四个孔中的每个。

要将蓝牙模块和Flora板彼此连接，必须使用导电线连接相应的按钮。

在此过程的最后，在背包的背面用打结的线扎住。

要连接两个板，需要在以下触点之间运行线程：

  • 植物区GND-蓝牙GND
  • Flora TX＃1-蓝牙RX
  • Flora RX＃0-蓝牙TX
  • Flora 3.3V-蓝牙3.3V

在上面的照片中，蓝牙模块在左侧，而Flora板在右侧。

步骤11：给电路加电




在植物群下面开一个小孔，以拉伸电池中的电线。将电线穿过孔，然后将其连接到板上。当Flora板打开时，绿色的LED应该亮起，而在蓝牙上，红色的LED将闪烁。

步骤12：缝制按钮以为电位计和LED供电




现在我们需要缝制Flora面板顶部的按钮。其中：

  • GND，我们将其连接到LED和电位器。
  • 3.3 V，我们将其连接到电位计。
  • 引脚10，我们将其连接到电位器。
  • 9号针脚，我们将其连接到LED。

缝制钮扣并将线拉到背包后部后，我建议保留线迹，以便以后可以继续缝制。另一方面，其中一些组件将连接到按钮。

步骤13：缝制将要连接多个组件的内部按钮




由于需要将多个组件接地并连接到3.3 V电压，因此必须在背包内添加一个按钮，以便可以将多个“鳄鱼”连接到背包。

在上一步中，我们留下了线程的供应。用它缝制另一个按钮，只需确保电线不交叉即可。

步骤14：组装ECG电缆




电缆由ECG芯片，带三个触点的电极电缆和UC-E6电缆组成。要将电缆连接到电路，我将鳄鱼连接到各种ECG组件：

  • 红色ECG线连接到红色鳄鱼。
  • 心电图的黑线是绿色的鳄鱼。
  • 紫色ECG线与蓝色鳄鱼线相同。

步骤15：继续组装ECG电缆




我们继续通过添加短线和鳄鱼皮来补充ECG电缆。我如何连接电线：

  • 红色ECG电缆-红色鳄鱼-橙色电线。
  • 黑色电缆ECG-绿色鳄鱼-黄色电线。
  • 紫色ECG电缆-蓝色鳄鱼-绿色电线。

为了提高可靠性，我用电工胶带包裹了连接。需要使用这些短接线将电路连接到Flora板上。

步骤16：切一个孔以将电线从ECG布线到背包顶部




在所有电线都位于的前袋的远角，切一个孔，将袋连接到背包的主隔间。这样您就可以在主隔间中布置来自ECG电极和Bitalino的导线。上图显示了导线如何穿过孔。

步骤17：将ECG的电线穿过背包，以使电线穿过皮带




如上所示，将电线穿过孔（用红色正方形圈起来）后，必须将其穿过主隔间，以便它们从上方悬挂。

步骤18：将电线穿过背包的皮带




在衬里所在的每个皮带顶部切一个孔。将电线沿皮带向下布线到指示的位置。

黑色电极应位于用户右侧，并且应位于红色电极上方。红色电极应位于左侧和下方。它们之间的线应形成一条贯穿心脏的对角线。白色电极必须放在骨骼上。可以将其放置在肩带的顶部，以适合肩骨。

步骤19：将ECG连接到Flora




如上所述，将ECG连接到鳄鱼，然后将电线连接到鳄鱼上：

  • 红色ECG电缆-红色鳄鱼-橙色电线。
  • 黑色电缆ECG-绿色鳄鱼-黄色电线。
  • 紫色ECG电缆-蓝色鳄鱼-绿色电线。

之后，您需要将电线连接到6号引脚（模拟7），接地（GND）和3.3V

。

将电线连接到引脚的方法如下：

  • 绿线（紫色ECG）-针脚编号6。
  • 橙色（红色ECG）-3.3 V.
  • 黄色（黑色ECG）-接地（GND）。

连接到地线和3.3 V电线，我将相应的按钮包裹起来。绿色导线缠绕在6号针脚上。

步骤20：将电位计连接到Flora




电位计必须连接到10号引脚，接地和3.3V。这些位置在照片中用圆圈表示。引脚10只是连接到此按钮和Flora板的螺纹。下载电位计连接图，以方便参考。

步骤21：打一个小孔，让用户看到LED




如上图所示，在LED灯可以穿透的地方打一个小孔。左边是我打完孔后的照片。在右侧-已经带有插入的LED。为了使它发光，我连接了它并下载了代码。

步骤22：将LED连接到Flora




要连接LED：

  • 使LED穿过孔，并压下其腿，使它们与背包平行。
  • 将短脚（阴极）连接到鳄鱼，然后连接到220欧姆电阻。使用第二条鳄鱼，将电阻器接地。
  • 将LED的阳极（长脚）连接到9号插针。

步骤23：将代码上传到Flora！



为了使用Flora，您需要将代码加载到其中。可以从GitHub下载。该视频提供了有关下载和使用代码的详细说明。

在下载之前，您可能需要安装必要的软件。有关说明，请访问Adafruit网站。

背包快要准备好了！

步骤24：更正代码，以便当心率超过正常值时LED亮起




平板显示正常的心率值，具体取决于性别和年龄。找到您的值并将其粘贴到代码中。

脉冲的上限在代码中设置，等于90。如果要离开该边界，则不要进行任何更改。否则，请在以下代码部分中更改数字：

if (HR > 90) { 
  digitalWrite(LEDpin, HIGH);
} // If the resting HR is above 90, or another preset HR, turn the LED on. HR number can be changed based off of chart in Instructable

步骤25：将粘性电极放在身体上




将电极放置在人体上图片中指示的位置。带有绿点的电极位于肩膀的骨头上。红点表示心电图的黑线，黄点表示心电图的红线。

步骤26：将电缆从背包连接到身体上的电极




电极连接在带子外的地方。调整绑带，使其适合用户，并允许您连接位于身体上的电极。

步骤27：调整电位器




将Flora连接到计算机，然后单击左上角的箭头下载代码。加载后，选择“串行绘图仪”（serial plotter）。旋转电位计时，您应该在图形上看到一条线。确保QRS复合波的峰值通过电位计线，如上图所示，否则心率计算将无法正确进行。

步骤28：下载Bluefruit LE Con​​nect




下载 Bluefruit LE connect 应用程序。有了它，您可以将数据下载到手机上。

步骤29：将手机连接到蓝牙




启动应用程序，并使用上面显示的设置连接到Adafruit Bluefruit LE。按Uart，然后将显示数据。心率将是该行的第一个数字。

步骤30：放在背包上




首先打开Flora。然后将其放在背包上并随身携带，尝试避免跳跃和奔跑-这样的运动可能会导致伪影。

步骤31：导出数据




收集数据后，单击应用程序右上角的齿轮，然后单击“导出”。您可以通过电子邮件将文本文件发送给自己或想要对其进行分析的人。

步骤32：在Excel中检查数据




将数据复制到Excel。选择所有数据并绘制折线图。黄线表示电位器的阈值，灰线表示心跳。脉搏定义为每分钟收缩的次数，即每分钟灰线越过黄线的次数。

步骤33：有什么可以改善的地方




该方案可以改进：

  • 通过将电位计放置在更方便的调谐位置。现在每次为此我都需要打开电子设备的口袋。
  • 添加压力表以跟踪呼吸次数。这将使您能够跟踪增加的呼吸频率并得出有关患者压力和放松的结论。背包通常有将胸带绑在胸前的紧固件，在此放置传感器很方便。

步骤34：最终产品






第一张照片显示了应用程序通过蓝牙接收的数据。此时，脉搏为每分钟96次。我将数据与Apple Watch进行了比较，它们非常出色-在同一时间段内，时钟显示为95至98次/分钟。照片中的LED点亮，因为脉冲超过90 bpm。在我的情况下，90是一个阈值，它的过量表示电压和激励水平很高。
