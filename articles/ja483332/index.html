CES 2020、IBM、および新しい量子コンピューターの最初の真の商用利用

伝統的に、1月の第1週にラスベガスは最大の国際展示会CESを開催します。そこでは、すべての世界の主要なテクノロジー企業が最先端の開発の大部分を発表します。例外なく、IBM はその最新の量子コンピューターの最初の商用利用を予想外に発表しました。パートナーはダイムラーであることが判明した。電気自動車の開発における彼の活動には、分子（硫黄リチウム）の特性をシミュレートするための高精度の計算が必要でした。これは、新世代の自動車用固体電池の基礎となる予定です。この市場のリーダーは現在、電気輸送業界における次の大きな飛躍を固体リチウム電池の技術と結びつけています。量子コンピューター計算に使用スーパーコンピュータ上の従来の高性能コンピューティングよりも魅力的であることが判明しました。そして、それが美しい理由です。

そして、ここで予期しないことは何ですか？私たちは何を期待していましたか？


過去のCESでのQシステム発表


ちょうど1年前のCES 2019で、IBM Q System Oneが発表されました。これは20キュビットの量子コンピューターで、超低温への独自の冷却システムであり、あらゆる意味で機能的に自給自足しています。このイベントの詳細については、私のメモの1つを参照してください。

早すぎると思われるステートメント


同時に、この量子コンピューターのラインは純粋に研究目標を超えるものであることが発表されました。これらは自由に利用できる技術のエンジニアリングサンプルではありません。現在、これは量子コンピューティングの商業化を目標とする製品です。このコンテキストでは、特別に作成された組織に参加した企業のみが新製品にアクセスできます。また、Q System One自体は、クラウドを介してアクセスし、リソースを共有する機能を備えた独立したコンピューティングユニットとして設計されています。

しかし、ほとんどの観察者と専門家にとって、そのような動きは時期尚早のように見えました。「量子優位性」（従来のコンピュータでは物理的に実行できないような複雑な計算を実行する量子コンピュータの能力）の証拠はありませんでしたが、通常、マイルストーンと見なされるのは量子の優位性であり、その克服は技術の商業化に必要です。

一年で何が起こるか知っているようだった


多くの人にとって、来年にIBMに何が期待されるかはすでに「明確」になっています。以前の発表のスタイル、いくつかのリーク、新鮮な情報に基づいて、次の予測は明らかでした：IBMはネットワーク内のできるだけ多くのパートナーを引き付けようとし、CES 2020では50キュービットの次のモデルを表示します（テクノロジージャイアントで利用可能な通常のモデルで引き続きシミュレーションできます）スーパーコンピュータ）、これについてはCES 2021まではこのトピックを忘れることができます。

実際、IBMの意図の妥当性と現実性を信じる人はほとんどいません。

IBMは「数字だけ」を自慢しませんでした


それにもかかわらず、最も注意深いオブザーバーは何かにしがみついていました。数か月前、小さいが話している信号灯が点滅した。ノイズと野心のないIBMは、53キュービットを備えた次のモデルを発表しました。彼らはより多くのキュービットを作成し、以前に行い、それの多くを自慢しませんでした。彼らがCESに注意を払うより重要なニュースを持っていることは明らかなシグナルでした。

IBMの量子コンピューター技術は計画よりも速く成長する


ムーアの成長の約束


量子コンピューティングに投資した真面目なプレーヤーの数、および以前の研究のダイナミクスと最初のコンピューター革命の経験に基づいて、IBM は、毎年少なくとも2倍のキュビット数を確保するために、他の誰も侵入しなかった非常に野心的なタスクを設定しました。

約束よりも速い成長


さらに、現在のダイナミクスを見ると、結果はさらに良くなります！2017年に5キュビット、2018年に16キュビット、2019年の初めに20キュビット、2019年の終わりに53キュビット。2年間で約11倍の成長！

実際、IBMは他の指標のみを見ており、計画通りに成長しています。


ただし、これは特異点の発生を意味するものではありません（申し訳ありませんがレイ）。実際、成長は厳密に計画に沿ったものであり、IBMが別の指標を基礎として使用しているだけです。

量子ボリューム（量子ボリューム）-特別に開発および公開された、量子コンピューターの実際の（現在の技術水準では達成できない）方法のパフォーマンスを測定および評価します。この評価を使用すると、厳密な進展が見られ、最も近い計画について予測を立てることができます。

実際、内部評価の方法論によれば、年に1回程度、容量が2倍になります。普遍的に受け入れられているメトリックの欠如を考えると-それは完全に正当化されます。さらに、次の発表のパラメーターを想定できる、一連の小さなデータがあります。2020年の終わりに、IBMは80から128の物理キュービットがある新しいモデルを導入します。この場合の量子体積は64になり、おそらく、少なくとも特定の範囲の問題で量子優位性が達成されます。

計画によると、パートナーを引き付ける


IBMのビジネス戦略の一部は、量子コンピューティングサービスの提供です。現在の技術時代に基づいてIBMが学んだ教訓は、鉄の平凡な販売ではなくサービスです。そのために、Qネットワークと呼ばれる独自のエコシステムを作成および開発しています。競争上の優位性を獲得したい組織はこのネットワークに参加し、最新の鉄の開発、コンサルタント、研究者にアクセスできます。 IBMは、最初のエコシステムがパートナーの最先端かつ重要な研究および投資プロセスに十分に深く入り込むことができると期待しているため、将来、競合他社が市場に登場したときに、サービスの変更に投資することは不利益になるでしょう。アカウントはすでに100パートナーを超えており、最近Qネットワークに参加しましたデルタ航空、ゴールドマンサックス、ロスアラモス国立研究所。

今すぐ新しい産業の出現


現在開発中（NISQ）


現在テクノロジージャイアントによって開発されているこれらの量子コンピューターは、多くの点で初期のマイクロチップとそのテクノロジーに似ています。技術的な制限により、さまざまなアルゴリズムおよび構成のトリックを発明して、有用な計算結果（たとえば、少数のピクセルカラーの迅速な変更を使用して、ビデオゲームで知覚可能なカラーパレットを拡張する方法）を取得する必要があります。

現在の量子コンピューターは、ノイズの多い中規模量子（NISQ）として説明できます。これは、キュービットの操作結果にはノイズが多く、量子情報を確実に保存するためのテクノロジーが設計されているだけなので、多くのトリックや松葉杖を使用する必要があるためです。



現在、現代の知識の各ブランチに対するNISQの利点は明らかではありませんが、化学、生物学、経済学の成果に関連する研究と革新は、量子コンピューターの利点を今すぐ見つけることができます。これの証拠は、投稿タイトルからのイベントです。少なくとも1つの真面目なテクノロジーグループが量子計算を選択して、その財務上の将来の主要テクノロジーの研究を提供しています。量子コンピュータの商品化には、量子優位性が必須条件ではないことがわかった。

予想される成長の速さ


もちろん、技術開発の最も明白な基準はキャッシュフローです。そして、私の意見では、最新の予測により、何が起こっているのかという深刻さを理解することができます。したがって、2020年代には、量子コンピューティングへの世界的な支出は年間数億ドルに達するでしょう。そしてすでに2030年代には、年間支出は数百億に達するでしょう。



そしてこれはすべて、技術開発（NISQ）の最初の段階に過ぎません。完全に新しいアーキテクチャおよびテクノロジーソリューション（2045年頃）への移行により、別の注文によって市場が拡大する可能性があります。

誰がどのようにテクノロジーを使用するか


最初に、量子コンピューターの使用の可能性がある大企業は、サムスン、ダイムラー、ホンダ、JPモーガンチェイス、バークレイズです。

サムスンは、マイクロエレクトロニクスの新素材分野の開発とその工場生産の革新に関心を持っています。現世代の量子コンピューターはこれに役立つかもしれません。発表をお待ちしております。

ダイムラーは、電池の化学分野の開発（これまでに学んだように、そうしています）のほか、ナビゲーション分野のソリューション（巡回セールスマンタスクのようなタスクエリアで、従来の量子コンピューターとは異なり、量子コンピューターでの計算の最適化が可能です）に興味を持っています。

JPモルガンチェースは、財務分析、リスク分析、取引などの新しいアプローチに関心を持っています。発表は、私たちが待つことはまずありません。

ハイテクおよびIT産業への影響


到達不可能であることは、より良くなる唯一の方法ではありません。これが、量子コンピューティングの懐疑論者が受け入れない主な考えだと思います。財の最大の敵であり、ビジネスでは、いくつかの点で、1つでは達成できないほど優れているというよりは、ほんの少しだけ優れていることがよくあります。おそらく、最も高価で人気が高い小型の地球スーパーコンピュータは、今後数年間で量子の優位性を回避できるでしょう。しかし、これは、アクセスキューなしで同じ計算を実行する量子コンピューターの開発と使用に反対するものではありません。

確かに、NISQアプリケーションの卓越した技術に基づいてビジネスを行っている企業は、その使用において競争上の優位性を持つと言えます。そして、量子コンピューティングの目標速度を考えると、この利点は劇的に増加します。すでに次の5年間で、リーダーシップと競争のチャンスを当てにしているすべての企業は、量子コンピューターを使用する必要があります。それで、今、私たちは新しい専門家、量子コンピュータープログラマーの誕生を見ました。
