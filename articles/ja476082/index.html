JavaScriptでのArray.spliceおよびArray.slice

こんにちは、ハブル！Kunal Tandonによる記事「Array.spliceおよびArray.slice in JavaScript」の翻訳を紹介します。

JavaScript Arrayプロトタイプで利用可能な上記の関数を使用するだけです。しかし、それらは非常に似ているため、それらの間で混乱することがあります。

主な違い


Array.spliceは元の配列を変更し、削除された要素を含む配列を返します。

Array.sliceは元の配列を変更しません。元の配列のサブセットである要素の新しい配列を返すだけです。

Array.splice


スプライスは、配列の内容を変更するために使用されます。これには、要素の削除、既存の要素の置き換え、またはこの配列への新しい要素の追加さえ含まれます。

スプライス機能を使用すると、元の配列が更新されます。

次の配列について考えてみます。

const arr = [0, 1, 2, 3, 4, 5, 6, 7, 8];

構文を含むArray.splice


arr.splice（fromIndex、itemsToDelete、item1ToAdd、item2ToAdd、...）;

アイテムを削除する


配列から要素を削除するには、次のように記述します。

var deleItems = arr.splice (3, 2);

これにより、インデックス3で始まる2つの要素が削除され、リモート配列が返されます。その結果、以下が得られます。

deleItems // [3, 4]
arr // [0, 1, 2, 5, 6, 7, 8]

新しいアイテムを追加する


配列に新しい要素を追加するには、次のように記述します。

const arr = [0, 1, 2, 3, 4, 5, 6, 7, 8];
var arr2 = arr.splice (2, 0, 100, 101);

配列の2番目の要素から始めて、100と101の数が追加されます。最終的な値は次のようになります。

arr2 // [],        
arr // [0, 1, 100, 101, 2, 3, 4, 5, 6, 7, 8]

既存のアイテムを変更する


spliceを使用して配列内の既存の要素を巧妙に変更し、インデックス番号で要素を削除して、その場所に新しい要素を挿入できます。

const arr = [0, 1, 2, 3, 4, 5, 6, 7, 8];

3を100に置き換えるには、次のように記述します。

var arr2 = arr.splice (3, 1, 100); 
//   -   3  1    100

上記のコードスニペットを実行すると、arrとarr2に次の値が取得されます。

arr2 // [3]      3   arr
arr // [0, 1, 2, 100, 4, 5, 6, 7, 8]
// 3   100   arr

Array.slice


spliceは配列要素を挿入および更新することもできますが、slice関数は配列から要素を削除するためにのみ使用されます。
配列から要素を削除するには、スライス関数

Array.slice構文

arr.slice（startIndex、endIndex）; を使用する必要があります。

startIndex-新しい配列の要素の開始インデックス。新しい配列で
取得する必要があります。startIndex.endIndex（オプション）-この要素を含まない、スライスが実行される終了インデックス。

次の配列について考えてみます。

const arr = [0, 1, 2, 3, 4, 5, 6, 7, 8];

値[2、3、4、5]から配列の一部を取得するには、次のように記述します。

var slicedArr = arr.slice (2, 6);

ここでは5ではなく2番目の引数6を指定したことに注意してください。上記のコードを実行すると、次の配列が取得されます。

arr // [0, 1, 2, 3, 4, 5, 6, 7, 8]
slicedArr // [2, 3, 4, 5]

spliceステートメントが元の配列を更新している間、arr変数はスライスステートメントが実行された後も変更されません。

したがって、元の配列を更新する場合はsplice関数を使用しますが、配列の一部のみが必要な場合はスライスを使用します。
