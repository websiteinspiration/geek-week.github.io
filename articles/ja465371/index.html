帰無仮説（H0）を公式化します


最初の段階で、特定の実務経験が資格のレベルを非常に正確に意味するものではないことがわかりました。次に、帰無仮説（反駁する必要がある仮説）を作成し

ます（たとえば）には、多くのオプションがあります。

  • 14年目の給与の年功への依存度は、19日と同じです。
  • 後輩の給与は14年以来変わっていません。

ただし、示されているエクスペリエンスは悪い指標であり、特定のカテゴリの計算は混乱を招く可能性があるため、単純でより実質的なオプションを採用します。14でのsnの平均レベルは、19と同じで、帰無仮説H0（2）です。

つまり、5年間の給与は変更されていないと仮定します。

分析の指標


メトリックとして平均値を選択することは論理的です。他のオプションも可能です。たとえば、中央値は、かなりの数の外れ値の場合によく行われます。ただし、推定値としての平均は理解しやすく、優れたアイデアにもなります。

ブートストラップ関数を記述します。

#   bootstraping
def bootstrap(data, func):
    boots = np.random.choice(data, len(data))
    return func(boots)

def bootstrapping(data, func=np.mean, size=1):
    reps = np.empty(size)
    for i in range(size):
        reps[i] = bootstrap(data, func)
    return reps

統計を計算します。 

#  14  19   -      
data = np.concatenate((data_14_1['Salary'].values, data_19_1['Salary'].values))

#     2  
data_mean = np.mean(data)

#      14  19 ,           

data_14_shifted = data_14_1['Salary'].values - np.mean(data_14_1['Salary'].values) + data_mean
data_19_shifted = data_19_1['Salary'].values - np.mean(data_19_1['Salary'].values) + data_mean

#  10000     ,  

data_14_bootsted = bootstrapping(data_14_shifted, np.mean, size=10000)
data_19_bootsted = bootstrapping(data_19_shifted, np.mean, size=10000)

#       .      .

mean_diff = data_19_bootsted - data_14_bootsted

#  P value               . 

p_value = sum(mean_diff >= diff_mean_salary) / len(mean_diff)

print('p-value = {}'.format(p_value))

14年と19年の給与レベルの調査の結果、次の結論に達しました。

  1. 調査データに基づくと、指定された勤続年数は、給与と資格のレベルを決定するための完全に適切な基準ではありません。
  2. スキルレベルへの分割は、おそらく給与のレベルに基づいています。
  3. プログラマーの給与は14から19に（平均8.5％）増加しましたが、これは偶然の結果ではありません。
