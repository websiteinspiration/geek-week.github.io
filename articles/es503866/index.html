Desarrollo de la hoja de ruta publicada Zextras Suite para 2020

Anteriormente, hablamos sobre las innovaciones de Zextras Suite Pro y Team Pro a principios de 2020. Zextras ha publicado la Hoja de ruta para el desarrollo de nuevas características  de Zextras Suite para 2020 .


Junio ​​- Equipo: nuevas funciones (MEJORADO)

  1. Responder a un mensaje (responder)
  2. Borrar historial de chat
  3. Editar mensaje
  4. Borrar mensaje

Agosto - Equipo: Servidor de video (NUEVO)

La aparición del servidor de video incorporado en Zextras Team le permitirá introducir una serie de nuevas funciones que se pueden controlar directamente desde la consola de administración, así como aumentar la productividad del servicio. Las transmisiones de audio y video se comprimirán, optimizarán y combinarán para obtener una transmisión por separado para cada dispositivo. Además, el servidor seleccionará automáticamente el códec y la velocidad de bits adecuados para cada dispositivo a fin de garantizar la máxima calidad de comunicación.

Agosto: copia de seguridad en punto final externo (NUEVO)

Zextras Backup brindará la oportunidad de realizar una copia de seguridad de su sistema en un almacenamiento local, híbrido o compatible con S3 en la nube, o en un almacenamiento externo de elementos BLOB, como NFS. Al mismo tiempo, un caché de metadatos local le permitirá mantener el rendimiento original, a pesar de que ellos mismos también se almacenarán en un punto final externo en caso de necesidad de recuperación ante desastres.

Septiembre: Autenticación e inicio de sesión de Zextras (NUEVO)

La autenticación de dos factores, un sistema federado de autenticación y autorización SAML, así como múltiples contraseñas móviles se agregarán a la edición de código abierto de Zimbra Collaboration Suite. También aparecerá una página de inicio de sesión actualizada con la posibilidad de personalización.

Septiembre: configuración de privacidad del equipo (NUEVO)

Los usuarios y administradores podrán elegir si se mostrará la información sobre su presencia. En particular, será posible ocultar información sobre la lectura de mensajes, el estado del usuario, etc. Septiembre - Autenticación QR de aplicaciones móviles (NUEVO) El sistema QR-Login le permite usar aplicaciones móviles sin tener que ingresar una contraseña mediante el uso de un sistema de autenticación cruzada, así como administrar su registro y autorización. Septiembre - Libreta de direcciones de Zextras para MS Outlook / MAC Mail / Thunderbird (NUEVO) Conecte Microsoft Outlook, Thunderbird y Mac Mail a un Zimbra GAL centralizado y sus libretas de direcciones para la conveniencia completa del usuario.










La libreta de direcciones puede funcionar junto con IMAP / POP o EAS. ¡Microsoft Outlook puede usar EAS y la libreta de direcciones para proporcionar una interacción completa con Outlook sin ningún complemento en el cliente!

Octubre - Zextrasconfig Infrastructure Manager (NUEVO)

Todos los componentes de Zextras en la infraestructura se pueden administrar a través de un administrador especial de Zextrasconfig. Este administrador permite a los administradores de Zimbra administrar los componentes de Zextras en varios nodos de infraestructura, incluido el servidor Zextras Docs integrado actualizado y el servidor de video Zextras Team integrado.

Octubre - Interfaz de administrador y preferencias de usuario (REFACTORING)

La nueva interfaz de administración del componente Zextras está escrita en React. Comenzando con los componentes de Zextras, que se seguirán presentando en la interfaz de administración familiar, transferiremos todas las opciones a la nueva tecnología sin eliminarlas de la interfaz familiar. Se realizará una transición completa en el futuro.

Octubre: nuevo instalador utilizando el sistema de empaquetado nativo de Linux (NUEVO) El

nuevo paquete Zextras se distribuirá a través de repositorios con soporte nativo de Linux. Al instalar y actualizar el paquete, será posible utilizar comandos del sistema como "apt-get install / update".

Noviembre: servidor de documentos (NUEVO)

El servidor Zextras Docs se convertirá en parte de la infraestructura, administrada a través del Gerente de Administración de Infraestructura Centralizada de Zextrasconfig. Instalar y actualizar el servidor en nodos individuales ya no será necesario.

Noviembre - Equipo: nuevas funcionalidades de la reunión (NUEVO)

Gracias al nuevo servidor de video, aparecerán varias características nuevas en el equipo de Zextras. Entre ellos:

  • Modo moderador
  • La capacidad de silenciar el sonido para un usuario específico o inmediatamente para todos
  • Posibilidad de habilitar el modo de seminario web y el modo de videoconferencia
  • El registro de eventos
  • Solicitud de chat
  • Modo Walkie Talkie
  • Las grabaciones de Hangouts se pueden cargar en Zextras Drive

Noviembre: alta disponibilidad (NUEVO) La

capacidad de sincronizar usuarios, grupos, dominios e incluso servidores de correo completos entre diferentes nodos de infraestructura. Capacidad para configurar la sincronización con un nodo de reserva u otro nodo activo en el clúster. El sistema está diseñado para lograr un 99.99% de disponibilidad por mes.

Noviembre - Impulsar cuenta externa personal (MEJORADA)

Capacidad para conectar cuentas de Dropbox / Gdrive / nextCloud / OwnCloud

Diciembre - Equipo: Reunión instantánea (MEJORADA)

Capacidad para crear una reunión instantánea y proporcionar acceso mediante enlaces permanentes y temporales.

Diciembre: firma y cifrado de correo electrónico (S / Mime-PGP) (NUEVO)

La criptografía de extremo a extremo se implementará en el cliente web. El sistema podrá cifrar y / o firmar los correos electrónicos enviados para descifrar y verificar la firma de los correos electrónicos recibidos.

Muchas de estas características hasta ahora han sido exclusivas de la versión comercial de Zimbra. Ya este año, los usuarios de Zimbra Collaboration Suite Open-Source Edition podrán aprovechar estas y otras características al comprar Zextras Suite. En el futuro hablaremos en detalle sobre las nuevas características de Zextras Suite.
