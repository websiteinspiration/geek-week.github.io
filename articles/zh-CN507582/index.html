理念



熟悉进化论之后，我永不停止欣赏如此简单的想法如何使我们描述难以置信的复杂生物系统出现的过程。



当学习任何东西时，独立地尝试通过教育实例来尝试实现或测试所提出的模型总是有用的。更有趣的是自己发明它们。因此，我有了进化论，在熟悉了基本概念之后，我想在模型系统上进行尝试。并查看如果您构建的仿真模型足够简单，可以观察到，并且足够复杂，可以在其中实现进化选择，那么会发生什么有趣的事情。了解不断发展的代理商的结构和行为如何变化，这种不可简化的复杂性如何产生，以及物种形成。



朝这个方向发展了许多不同的模型。甚至存在用于解决应用问题的进化建模方法。在这些视频中，对进化的主要方面进行了流行的阐述：



  • 不断变化的颜色矩形 ;

  • 神经蚂蚁 Alexander Panchin
  • 由M. Burtsev和V. Redko撰写的科学出版物，有关他的作品的一个受欢迎的故事在Gordon广播。


我想创建一个模拟模型，该模型具有稍微复杂的机制，因此具有更广泛的开发和复杂化可能性。开发模型的主要特征：



  1. 模拟代理种群的进化-D shell，代表人工神经网络，其结构和参数由代理的染色体确定；
  2. 代理可以在环形二维空间中移动和交互（正方形的相对侧彼此封闭）。同时，该空间被划分为多个单元，最多可同时放置两个代理。
  3. D shell 世界中的时间是离散的；
  4. 有限的资源蛋糕，是代理人的能源，在空间中分布不均。
  5. 特工可以执行动作（运动，营养，繁殖，攻击），同时消耗先前存储的能量；

代理从他们周围的一个小社区接收信息，以及有关其内部状态（能量储备）的信息。通过单个神经网络的此信息会影响代理的行为。



我在进化建模方面的经验将由几个部分组成。



在第一部分中，我将描述D- shell 宇宙的结构和力学，并描述潜在的代理所具有的。

第三部分将致力于对宇宙D的简化，随机模型的分析贝类。在此Universe的实现中，从某种意义上说，代理的个体行为是完全随机的，因为它不依赖于代理外部或内部环境的状态。在这种情况下，代理人行为的概率分布由一条染色体确定。这种模型的结果将使将来有可能注意到“意识”的表现，我希望代理人的行为复杂。同时，不要将其与自组织效应的表现相混淆。

宇宙D贝类



特工生活在一个二维空间中，该空间被折叠成一个圆环，因此没有边缘，特工始终可以移动而不会撞到障碍物（图1）。





图1.雷神为了获得扫掠效果，将圆环沿着选定的圆切开并在矩形平面上旋转。



可用于代理的所有空间均被划分为六边形单元（图2）。他们的世界看起来像一个蜂窝。在模拟中，我使用了100×100个单元的世界大小（总面积为10,000个单元）。一次在一个单元中最多只能放置两种代理。如果代理在移动过程中越过矩形的边界之一，则它会从相反的一侧返回到矩形，就像它沿着甜甜圈爬行一样。



除了细胞中的药剂外，还有营养资源- 蛋糕，药剂可以吃。食物是根据特殊算法在细胞中产生的。



D类贝类的食物



每个单元可以包含任意数量的整个蛋糕（图2）。在一个时间步骤中，代理商可以吃一个蛋糕，这将通过50 J补充其能量储备-食物的能量。





图2.一份食物在太空中的位置示例。灰色细胞不包含食物，深蓝色细胞包含1个蛋糕，黄色细胞包含15个蛋糕。



太空中的食物数量在不断变化。存在太空中食物降解和扩散的机制，这对于维持太空中固定数量的食物是必需的。这些机制是限制资源可用性的因素，同时又确保环境的可变性。食物的数量因以下过程而异：



  • 被代理商食用时减少；
  • 由于蛋糕部分的降解而减少;
  • 由于“增长”而增加；
  • 代理商死亡时增加；
  • 随着某些特工成功攻击其他特工而增加，结果蛋糕掉入了太空。


空间中食物的平均总量取决于媒介的人口密度。当用试剂填充空间的密度小于每个细胞一种试剂（1 Dr /细胞）时，空间中食物的固定密度为20 块蛋糕/细胞（或能量单位为1000 J /细胞）。但是，如果每个单元中有一种以上的药剂，则可以减少太空中可维持的固定食物量。并且当媒介完全充满空间（2 Dr / cell）时，食物的静止量变为零（图3）。





图3.食物固定量对空间中媒介密度的依赖性



如果食物量小于给定的固定值，那么在当前时间结束时，空间中将产生当前食物不足的50％。所需的食物量是在随机的位置，随机的体积（但不小于250个蛋糕）中分批产生的。生成这些部分，直到在空间中放置所需量的食物为止。



如果食物量超过预定的固定值，则在当前时间步结束时，将减少随机选择的相邻细胞组中的食物量。该组由19个单元组成（图4）。选择此类组的数量，以使食物过量减少10％。这可以解释为无法在太空中维持过多的食物，从而导致食物降解。随着时间的流逝，太空中的食物数量将趋于给定的固定值。



这种用于产生食物的算法可以防止人口过多。



设备D外壳



所有代理具有相同的结构。区别仅在于其神经网络的参数值。在此模型中，每个代理由以下特征集表示：



  • 代理的名称，在整个模拟过程中是唯一的，是自然数；
  • 代理的能源供应，用整数J表示；
  • 代理所在的小区号；
  • 代理的年龄，即在每个时间步结束时增加一个的自然数；
  • 代理生成，自然数。在代理出生时，此参数的设置要比其中一个父代的生成多（将具有更多特征的父代从两者中选出）。


每个代理都有三个染色体，其中两个决定了其神经网络的参数：



  1. 染色体WEIGHTS -隐藏层的神经元的权重矩阵;
  2. 染色体BIAS-隐藏层神经元的位移向量；
  3. 染色体颜色 -定义了试剂颜色的三个组成部分-红色，绿色，蓝色。这三个染色体在繁殖过程中遗传。


代理具有四组受体，它们将信息从环境和环境传输到代理的神经网络的输入层。



  • 邻近细胞（19个受体）中是否存在食物-允许该代理接收有关其附近食物量的信息（图4）；
  • 相邻细胞中存在试剂（19个受体）-允许该试剂接收有关其附近其他试剂数量的信息（图4）；
  • 相邻试剂的颜色（3个受体）和本征颜色（3个受体）-允许该试剂接收有关细胞中第二种试剂的颜色的信息（如果有）及其颜色；
  • 药剂（1个受体）的当前能量储备；




图4.代理可见的单元格布局以中间的紫色圆圈显示。代理可以对这些单元中的每个单元采取步骤。从1到6，步长等于一个单元。从7日到18日-步长为两个单元格。



神经网络的输出连接到一组效应器，效应器的状态确定代理在当前时间步将执行的动作。特工可以做以下六件事之一：传球，第一步，第二步，进食，攻击，做爱。

食物进入从系统的外部的空间与在食物流。内部还有两种补充食物的内部流动：伤害，死亡。第一种是由被邻近特工攻击的特工的能量损失引起的。他们损失的能量以与受伤的药剂同等数量的蛋糕形式掉落。第二种流是食物在药剂死亡后出现在细胞中，其数量等于药剂在死亡之前剩余的能量。

自然选择



在D宇宙中，Rakosh进行自然选择。



为了生存，代理商需要消耗食物以补充其能量消耗，而该能量供应一直在消耗。进食效率更高的病原体可以持续更长的时间，从而留下更多的后代。繁殖的有效性取决于性行为的频率和父母传递给后代的能量。除了由代理接收的信息及其神经网络的参数之外，还确定了代理的行为。它们在繁殖过程中是遗传的，但它们会随机变化-突变。



这些因素的作用导致了这样一个事实，即不同的因素有不同的生存和离开后代的可能性。
