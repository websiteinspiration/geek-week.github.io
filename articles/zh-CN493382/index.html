.NET Core Windows窗体设计器更新

我们发布了Visual Studio 16.6的初步版本-Visual Studio 2019版本16.6 Preview 1并附带了Windows Forms的.NET Core构造函数的新版本。



此版本包括


  • 支持以下控件：

    揭露
    • FlowLayoutPanel，
    • 组框
    • 图片清单
    • MenuStrip ( PropertyBrowser ),
    • Panel,
    • SplitContainer,
    • Splitter,
    • TabControl,
    • TableLayoutPanel,
    • ToolStrip ( PropertyBrowser, designer actions).

  • 本地资源和本地化形式包含在构造函数中。
  • 通过工具->选项支持LayoutMode和ShowGrid / SnapToGrid设置。
  • 可靠性和性能改善。
  • 其他较小的修复和调整。

进一步


在将来的版本中，我们将致力于对用户控件和第三方控件的支持，与流行控件提供者的集成，对数据控件和相关方案的支持，性能改进以及其他功能。

如何使用构造函数


  • 您需要使用Visual Studio预览供稿
  • 您需要在Visual Studio中启用设计器。转到工具→选项→环境→预览功能，然后选择选项“将Windows Forms Designer用于.NET Core应用程序”。
