言語への関心を引き付けるRubyミームジェネレータ


私の意見では、Ruby言語は2012年から2014年のように人気がなくなったので、RubyはWeb開発にとって最も便利で実用的な言語であるため、言語が完全に忘れられるまで、再び大衆に届けたいと思っていました。
1週間前、私は電報コミュニティを作り、初心者プログラマーを見つけ、Ruby開発を明るい面に持って行きたくなりました。

計画はコミュニティを作成することでしたが、誰もがプロジェクトの問題やバグについて書く（コミュニティは十分にあります）のではなく、Rubyの世界からの有用な情報が投稿されるコミュニティです。

まず、ミームの形で、教育的な内容で投稿したかったのです。

そのようです


ミームジェネレーターのサイトで10枚の写真を作成したところ、すべてが非常に遅く問題があることに気づきました。それに加えて、10のみのルビーメソッドを作成しました。独自のミームジェネレーターを作成し、インターフェイスをこれらすべてに関連付け、Ruby言語を使用してすべてを実行することが決定されました。

RMagicによる画像生成


レンダリングをあまり気にしないために、既存の宝石が基礎として使用されました`rmagick`。この宝石のおかげで、写真を簡単に変更できます。

以下のコードでは、新しいオブジェクト（テキスト）を写真の形式で作成し、それらを基本として写真に配置しました。

メインロジックコード
img = Magick::ImageList.new("1.jpg")

  title = Magick::Draw.new
  title.pointsize = 126
  title.font_weight = Magick::BoldWeight
  title.fill = 'white'
  title.gravity = Magick::CenterGravity
  title.annotate(img, 0, 0, 0, -400, first_text)

  descritpion = Magick::Draw.new
  descritpion.pointsize = 72
  descritpion.font_weight = Magick::BoldWeight
  descritpion.fill = 'white'
  descritpion.gravity = Magick::CenterGravity
  descritpion.annotate(img, 0, 0, 0, 300, second_text)

  img.write("actual_mem/#{first_text}.jpg")


フォントサイズ、太さ、色、テキストの重力などの基本的な属性を入力した後。次に、actual_memフォルダーに画像を保存します。このフォルダが後で必要になるのはなぜですか。

シナトラのウェブインターフェース


今度はインターフェースを実装する必要がありました。小さなアプリケーションのために大きなRailsプロジェクトをデプロイしないようにするために、Sinatraと呼ばれる別のRubyフレームワークを使用することにしました。

その結果、アプリケーションを実行するためのコードを含む小さなファイルを取得しました

Sinatraアプリケーション起動ファイル
require 'sinatra'
get('/') do
  haml :index
end

post('/') do
  if params[:title] && params[:description]
    require 'rmagick'

    b = params[:description].scan(/[-a-zA-Z-0-9_.,()—-]+/)
    second_text = ""
    b.each_with_index do |text, index|
      if index.modulo(4) == 3
        second_text += text + "\n"
      else
        second_text += text + " "
      end
    end

    img = Magick::ImageList.new("../1.jpg")

    title = Magick::Draw.new
    title.font = ''
    title.pointsize = 126
    title.font_weight = Magick::BoldWeight
    title.fill = 'white'
    title.gravity = Magick::CenterGravity
    title.annotate(img, 0, 0, 0, -400, params[:title])

    descritpion = Magick::Draw.new
    descritpion.font = ''
    if b.size > 20
      descritpion.pointsize = 48
    else
      descritpion.pointsize = 72
    end
    descritpion.font_weight = Magick::BoldWeight
    descritpion.fill = 'white'
    descritpion.gravity = Magick::CenterGravity
    descritpion.annotate(img, 0, 0, 0, 300, second_text)

    img.write("../actual_mem/#{params[:title]}.jpg")
  end

  haml :index
end

今後の計画


将来的には、フォームの完成を検討しています-上部と下部のテキストのフォントサイズを設定するスライダー、テキストを配置するスライダーを追加し、プレビューブロックを作成することもできます。
