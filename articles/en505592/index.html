Programming for BK0010 using Android



The story happened in the BK00010 Telegram channel, I was a participant.

The question arose: how to write programs? The emulator does not support burning discs, so using the assembler in the emulator is unlikely to succeed. To top it all off, the picture on smartphones is very small.

A solution was proposed (smaslovski), which I will describe. You need to download Termux, the version for Android 5.0 is available here , the termux-app-git-debug.apk file. The version of the program is 0.83, it supports customizing the keybar.

After installing and starting Termux, I reconfigured the keys, the procedure is as follows:

mkdir $HOME/.termux

echo "extra-keys = [['ESC','/','-','HOME','UP','END','PGUP'],['TAB','CTRL','ALT','LEFT','DOWN','RIGHT','PGDN']]" > $HOME/.termux/termux.properties

termux-reload-settings

It has become more convenient



Now you need to install the necessary Finish, MACRO-11 will be used as the assembler for creating .BIN:
pkg install build-essential
pkg install git




mkdir src
cd src
git clone https://gitlab.com/Rhialto/macro11.git

Then I installed the shell:

pkg install mc

and text editor

pkg install nano

in the src \ macro11 folder you need to fix the Makefile , change the parameter

OBJFORMAT = -DDEFAULT_OBJECTFORMAT_RT11=1

type make and wait for the result,

copy the received files

cp macro11 $PREFIX/bin

cp dumpobj $PREFIX/bin

To compile, you need a script to build

cd ~/src 
git clone https://github.com/smaslovski/pdp11dasm.git
cd pdp11dasm/utils
chmod 755 obj2bkb
cp obj2bkb $PREFIX/bin

compilation script, I called compile :

#!/data/data/com.termux/files/usr/bin/sh
n=$1
macro11 $n.mac -o $n.obj -l $n.lst
obj2bkb --binary --base=01000 -o $n.bin $n.obj
du -b $n.bin
cp $n.bin /sdcard

(you can create nano compile )

The script needs to be copied

chmod 755 compile
cp compile $PREFIX/bin

For testing, I wrote a simple gg.mac program :



.title proga
.psect code
.=1000

 MOV #40000,R0
M: mov r0,(r0)+
 tst r0
 bpl m
 halt
.end

launch: compile gg
the gg.bin program can be opened in the emulator.
