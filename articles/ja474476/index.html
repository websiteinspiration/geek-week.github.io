2K19でDVDに写真を保存（2190で？2238で？）


私の最初のデジタルカメラは14年前に登場しました。それから写真を保存する問題がありました。幸いにも、当時、彼女は迅速かつ明確に-ディスクに書き込む期間を決定しました。外付けHDDと内蔵HDDも高価でした。私の意見では、SSDドライブはまったく存在しませんでした。存在する場合は、おそらく桁違いに高価です。映像をメモリカードに正しく保存し、毎回新しいカードを購入するのは無駄な狂気です。一般的に、家庭では、DVDに代わる合理的な手段はありませんでした。さらに、ディスクには多くのゲームがあり、コンピューターにはDVDドライブが必須でした。私はすぐに書き込みドライブをしました。

ある時点で、これは2〜3年前のことですが、DVD-RWディスクに記録されたカーシミュレーターの友達に彼のところに行き、彼の新しいコンピューターにはディスクリーダーがまったくないことに気付きました。少しショックを受けました。友人が笑った：ドライブをすることは、カートで馬のカーレースに行くことと同じです。そして、CD-DVDの時代が去っていくことに気づきました。次に、データ転送用に大容量のフラッシュドライブを購入する必要がありました。



しかし、これまでのところ、写真を保存するためにDVDを放棄することはできません。価値のある代替案が見当たらないだけです。ディスクは3〜5年で「崩れる」ため、ディスクは非常に信頼性の低いストレージであると主張する人もいます。これがそうかどうか見てみましょう。最初の写真から14年が経ちました。それらの古いディスクを机の引き出しから取り出します。


実際、これが私のアーカイブ全体です。これは82 DVD + Rディスクで、ボリュームは約360 GBです。ボックスに入ったいくつかのCD-R-カウントされません。見た目がいいので、近くに置いただけです-それらは古いレコードの形式で、一度購入したものですが、何も書きませんでした。

なぜアーカイブはほとんど必要ないのですか？まあ、私が最も簡単なカメラを手に入れる前は、写真の重さはほとんどありませんでした。また、私はプロの写真家ではないため、RAWを保存することはありません。アーカイブ全体は、ある種の家族の写真、さまざまな旅行、イベント、ごちそうの瞬間です。これらはすべて、ある種のハイアートのふりをするものではなく、単なる「ホームフォトアルバム」であり、思い出と同じくらい高価なものです。ええと、写真を記録する前に厳密な検閲（冗談）が行われます。多かれ少なかれ面白く、重複のないことが判明した写真だけがディスクに入れられます。



古いディスクを読み取ってみましょう。私はいくつかのDVDバーナーを交換しましたが、それらはすべてNEC、Sony、Sony-NECなどです。いずれにせよ、私はこれら二つのブランドだけを覚えています。そして、写真用にディスクを無駄にせず、常に高価なものを購入しました。通常は逐語的で、時にはTDKです。

ディスク1


そう。アーカイブの最初のディスクは自転車に乗っています。


ディスクは問題なく読み取られました。その写真は2007年から2010年にさかのぼり、2011年4月に記録されました。そうではなく、彼は古代であることが判明しました。


それらの写真をフォルダに分散させ、すぐにディスクに書き込んでHDDから削除したことさえ覚えています。私の意見では、その前はいくつかのCDに保管されていましたが、残念ながら私はそれらを捨てました。





ディスク2


もっと古いものを探しましょう。次のディスクは、「Trips 2005-2007」という一連の旅行の最初のものです。


彼の記録は2007年以前に作成されたものではないと仮定するのは当然です。ディスクを開きます。2008年12月以降のフォルダ。はい、私はそれらをどのように作成し、ブランクに丸めたかを覚えています。


ええと、11歳もいいです。明らかに、以前はこれらの写真はCD-ROMにも保存されていました。このDVDの場合：11年-通常の飛行。





ディスク3


そして、興味深いインスタンスがあります。これらは友達が録画した動画です。私はそれらの旅行にいなかった、私はただディスクに書き直すように頼んだ。


このディスクはちょうど2005年でした。当時は、そのようなボリュームをハードドライブに保存する余裕がなかったからです。ディスクは長時間（約20分）コピーされましたが、コピーされました！


それはおそらくバックアップする時です。しかし、それでも、録音直後は、どういうわけかすぐには読まなかったのを覚えています。明らかに、なんらかの欠陥がありました。でも、いつでも友達からこれらのビデオを撮ることができたので、別のディスクを台無しにすることはありませんでした。

さらに10枚のディスク


残念ながら、もっと古いディスクは見つかりませんでした。アーカイブからさらに10個のランダムディスクをテストし、2007-2010に対応する最も古いディスクを取得しようとしました。例外なく、それらはすべてためらうことなくエラーなしで読まれました。

私は常に6倍速でディスクを記録したと言っておく必要があります。高速でディスクを書き込むと、よりずさんな書き込みができるということです。

HDD /SSD /


HDD / SSDにすべての写真を保存してみませんか？はい、私は今日、さらに大きなSSDがすべてのDVDを組み合わせたものよりも安いことを理解しています。しかし、最初に、私のディスクは必要に応じて徐々に購入されました。次に、写真専用のHDDまたはSSD全体が常に高価な1回限りの購入でした。そして、そのようなお金がいつか現れたとしても、それは常に新しいビデオカードや追加のRAMなどの何かに費やされていました。私のような他の人がいるのだろうか？さらに、高速のために、HDDからSSDに切り替えて、HDDをまったく認識しないことがありました。さて、私のHDDはもう古いので、あえてアーカイブにも使用しませんでした。そして、新しい種類のドライブ、つまりSSDは、何度も高額な購入のように思えました。HDDのように。彼らはまた、純粋にアーカイブの下で購入することが非現実的になりました：以前はHDDに置いていた必要なものすべてに合うように...

しかし、財務上の問題を破棄し、技術的な観点からストレージを検討すると、DVDと比較した場合のHDD / SSD上のストレージの欠点は次のとおりです。

  1. 誤ってファイルを削除する機能。
  2. オン/オフ/電力サージをオンにすると、デバイスの電気系統に損傷が発生する可能性があります。
  3. ウイルスへの暴露。
  4. 記録は磁気原理に基づいているため、ストレージの耐久性が低い。

クラウド写真ストレージ



これらは私の偏見かもしれませんが、長期的なアーカイブに関してクラウドサービスを信頼していません。DVDと比較した短所：

  1. クラウドサービスのアカウントがブロックされているか、サービスが完全に閉鎖されている可能性があります。
  2. 一部のサービスでは、写真を学習アルゴリズムやその他の目的で使用するなど、写真に何らかの権利が必要になる場合があります。
  3. 年間の支払いは4000ルーブル以上になることがあります。
  4. サーバーのクラッシュは小さなマイナスで書くこともできますが、すべての情報が何度も複製されることを願っています。しかし、これを100％確信することはできませんが、それでも...
  5. あまりにも個人的な写真がある場合、一般的にインターネット上のどこかに漏れるのはあまり楽しいことではありません。

自分のRAID /自分のサーバーに写真を保存する


プロセスを自分で制御することにより、情報の安全性をより確実にすることができます。しかし-失敗したメディアを交換するコスト、ソフトウェア構成の時間の浪費。また、キャリアのタイプは基本的にクラウドと同じです。

紙に写真を保存する


実際、ここにはDVDに勝る特別な利点はありません。紙も時間とともに劣化します。ただし、紙の場合は、写真を印刷するときに追加のデータ変換が行われます。デジタル画像はアナログに転送されます。もちろん、パンチカードにそれらを保存しない限り...

結論


一般的に、まともなDVDは見つかりませんでした。要点は、情報の長期保存という観点からは磁気メディアが好きではないということではありませんが、それらを扱うには、ブランクを記録するよりも多くの労力が必要です。複数のコピーを含むいくつかの追加のデバイス、それらを接続して定期的にこれらの同じコピーを作成...

その間、DVDドライブはディスク自体と同様に販売から姿を消しています。岩絵よりも信頼できる恐竜が歴史に残るのは残念だ。結局のところ、基本的にはそれは良い考えでした。しかし、ソニーは2010年までカセットオーディオプレーヤーをリリースしました。そのため、DVDを使用したこのストーリー全体が長い間忘れられないことが望まれます。そしてそこに-あなたは、組み込みの電気機器なしで、磁気原理に結び付けられていないいくつかの新しいリムーバブルメディアを見て、発明します。おそらく、いくつかの場所に写真を保存して、一度にすべての最新の手段を使用することは、まだ価値があるのでしょうか？
