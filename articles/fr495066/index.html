Check Point Remote Access VPN - Cours gratuit de configuration de l'accès à distance



Salutations amis! Bienvenue dans notre prochain nouveau cours! Comme je l'ai promis, le cours de démarrage n'était pas le dernier. Cette fois, nous aborderons un sujet tout aussi important - VPN d'accès à distance (c'est-à-dire l'accès à distance). Avec ce cours, vous pouvez rapidement vous familiariser avec les technologies Check Point en termes d'organisation de l'accès à distance sécurisé pour les employés. Dans le cadre du cours, nous allons, comme d'habitude, combiner la partie théorique avec la partie pratique sous forme de travaux de laboratoire. En plus de démontrer la configuration de Check Point, nous examinerons différentes façons de connecter des utilisateurs distants.

Certains commentaires


  • Ce cours n'est pas un guide complet de la technologie VPN. Ceci n'est qu'un bref aperçu des informations les plus demandées. Ceux. vous ne devriez pas vous attendre à ce que, après avoir visionné, vous deviendrez le gourou des VPN. Cependant, pour comprendre les principes et commencer rapidement à travailler avec Check Point VPN, ce cours sera plus que suffisant.
  • Je recommande fortement de commencer ce cours uniquement après avoir terminé le précédent - Check Point Getting Started. Je n'expliquerai pas ici certaines choses de base, telles que l'architecture, comment configurer une adresse IP ou obtenir une passerelle vers le serveur de gestion. Je partirai de l'hypothèse que vous le savez déjà et nous nous concentrerons uniquement sur le VPN.

Un autre point. Pourquoi faisons-nous ce cours? Tout d'abord, pour aider les deux premières catégories. Au cours des trois dernières semaines, nous avons simplement un tas de demandes d'aide pour la configuration d'un VPN. Nous aidons absolument tous nos clients et essayons de réussir à aider tout le monde gratuitement. Une période si difficile à aider. Malheureusement, le temps n'est pas toujours suffisant, mais les gens demandent. Je pense que ce cours sera juste à temps et vous permettra de toucher un peu plus de monde.

Le contenu des cours


  1. Introduction;
  2. Check Point RA VPN - un bref aperçu des technologies;
  3. Stand de laboratoire (agencement);
  4. Installation et configuration de base de la passerelle Check Point;
  5. VPN IPSec
  6. VPN SSL (portail SSL);
  7. VPN pour Android / iOS;
  8. Authentification à deux facteurs
  9. Secure Remote, L2TP;
  10. Surveillance des utilisateurs distants;
  11. Licence.
