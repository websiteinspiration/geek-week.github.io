PVS-Studio Analyzer Beta-Test # unter Linux und MacOS


Immer mehr Benutzer des C # -Analysators von PVS-Studio interessieren sich für die Möglichkeit, damit C # -Code unter Linux und macOS zu testen. Und heute haben wir gute Nachrichten.

Seit Ende letzten Jahres arbeiten wir aktiv daran, unseren C # -Analysator auf die .NET Core-Plattform zu portieren und seine Leistung unter Unix-ähnlichen Betriebssystemen sicherzustellen. Wir planen, PVS-Studio C # für .NET Core Ende April - erste Maihälfte auf Linux- und MacOS-Plattformen zu veröffentlichen.

Unter Windows funktioniert der Analysator weiterhin unter .NET Framework, da wir uns auf die Veröffentlichung des Analysators für neue Betriebssysteme konzentriert haben. Darüber hinaus können Sie jetzt plattformübergreifende .NET Core-Projekte unter Windows überprüfen.

Wir planen, den Beta-Test Anfang bis Mitte April zu starten. Wenn Sie daran interessiert sind, füllen Sie dieses Formular aus . Wenn der Betatest beginnt, senden wir Ihnen Anweisungen zur Installation des Analysators und eine Testlizenz.

PS Wir entwickeln außerdem ein Plug-In für die JetBrains Rider-Entwicklungsumgebung für alle vom Analysator unterstützten Plattformen. Im Rahmen des Betatests haben Sie als „Bonus“ auch die Möglichkeit, dieses Plugin auszuprobieren, auch mit dem „klassischen“ PVS-Studio C # -Analysator unter Windows.
