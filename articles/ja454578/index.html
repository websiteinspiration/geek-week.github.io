Android WebViewでPhotoSwipeギャラリーを接続する方法

プログラマーになるために勉強しましたが、私はプロの開発者ではありません。現在、私はシステム管理者として働いており、開発者に移動する予定です。私は自分のために、人気のあるIT関連のWebサイトの1つを解析し、ネイティブのAndroidアプリケーションで記事を表示するアプリケーションを作成しています。habrモバイルアプリケーションの場合と同様に、記事内のすべての写真をギャラリーで開くようにしました。

ほとんどの場合、マニュアルはシンプルであることが判明しますが、フィードバックを得たいと思います。おそらく、このマニュアルは誰かに役立つでしょう。

いくつかのプラグインを試しました。起動しなかったものもあれば、jQueryが必要なものもあり、アプリケーションでのページの読み込みが遅くなります。最後に、PhotoSwipeプラグインを選択しました。プラグインはjQueryを必要とせず、迅速に動作し、ほとんど重量がありません。

まず、アプリケーションのどこにプラグインのソースファイルを配置するかを検討する必要があります。まず、未経験でファイルをrawフォルダーに配置しましたが、assetsフォルダーに配置することをお勧めします。アプリを右クリック→新規→フォルダ→アセットフォルダ

ドキュメントには画像をクリックしてプラグインを有効にする方法の例が記載されていますが、PhotoSwipeはプラグインが正確に開始するかどうかを気にしません。最後に、私は自分のスクリプトを書きました。アプリケーションに他の誰かのサイトの記事を表示しているので、既製のHTMLを取得します。レイアウトを動的に操作しないようにするために、スクリプトでは、すべての画像のクラスを変更します（記事には不要な画像はないため）、イベントハンドラーを割り当てます（クリックすると、URLが渡されます-ギャラリーの識別子として機能し、プラグインが起動します）。

プラグインは画像のサイズを知っている必要があります。ページ全体が読み込まれるとギャラリーが接続されるため、画像のnaturalWidthプロパティとnaturalHeightプロパティを通じて画像の最終的なサイズを簡単に見つけることができます。

スクリプトコード。まず、ページが読み込まれると、init（）関数が開始します。次に、画像openGallery（）をクリックすると、

コード
//        
function init() {

	var images = document.images;

	for (var i = 0; i < images.length; i++) {
       
		images[i].className = "photoSwipe";
		images[i].onclick = function () { openGalery(this.attributes["src"].value); }
	}
}

//   
function openGalery(url) {

	var images =  document.getElementsByClassName("photoSwipe");
	var items = new Array();

	for (var i = 0; i < images.length; i++) {
        items.push(getAttributes(images[i]));
    }

    startPhotoSwipe(items, getIndex(images, url));
}

//   
function getAttributes(image) {

    return { 

        src: image.attributes["src"].value,
        w: image.naturalWidth,
        h: image.naturalHeight
    }
}
 
//   ,   
function getIndex(images, url) {

	for(var i = 0; i < images.length; i++) {

		if (url == images[i].attributes["src"].value)
		    return i;
	}

	return false;
}

//  
function startPhotoSwipe(items, index) {

	var pswpElement = document.querySelectorAll('.pswp')[0];

    // define options (if needed)
    var options = {
        // optionName: 'option value'
        // for example:
        index: index, // start at index slide
        bgOpacity: 0.9,
        pinchToClose: false,
        fullscreenEl: false,
        closeEl:false,
        zoomEl: false,
        shareEl: false,
        indexIndicatorSep: '  ',
        tapToClose: true
    };

    callAndroid.setGallery(true); //     Java  ,   

    // Initializes and opens PhotoSwipe
    var gallery = new PhotoSwipe( pswpElement, PhotoSwipeUI_Default, items, options);
    gallery.init();
}

この段階でページが読み込まれ、画像をクリックするとプラグインが開始しますが、ギャラリーを開いた状態で戻るボタンを押すと、プラグインではなくアプリケーション全体が閉じます。

これを修正するために、Javasciptブリッジを作成しました。

まず、ギャラリーの状態をシングルトンにする必要があります。

コード
package me.denko.photoswipe;

class GalleryState {
    private static final GalleryState ourInstance = new GalleryState();

    static GalleryState getInstance() {
        return ourInstance;
    }

    private GalleryState() {
    }

    private boolean isGallery = false;

    public boolean isGallery() {
        return isGallery;
    }

    public void setGallery(boolean gallery) {
        isGallery = gallery;
    }
}


WebViewから呼び出すための追加インターフェース

コード
package me.denko.photoswipe;

import android.webkit.JavascriptInterface;

public class JavascriptFromWebView {

    @JavascriptInterface
    public static void setGallery(boolean bool) {

        GalleryState.getInstance().setGallery(bool);
    }
}


このインターフェイスをWebViewに接続する必要があります

webView.addJavascriptInterface(new JavascriptFromWebView(), "callAndroid");

追加する必要があるギャラリーを開くためのスクリプト
callAndroid.setGallery(true);


photoswipe.jsファイル内のギャラリー終了スクリプト

コード
close: function() {
		if(!_isOpen) {
			return;
		}

		callAndroid.setGallery(false); //   

		_isOpen = false;
		_isDestroying = true;
		_shout('close');
		_unbindEvents();

		_showOrHide(self.currItem, null, true, self.destroy);
	},


これで、ギャラリーを開くと状態がtrueに設定され、閉じると状態がfalseに設定されます。あとは、WebViewの戻るボタンハンドラーを追加するだけです。

コード
webView.setOnKeyListener(new View.OnKeyListener() {
            @Override
            public boolean onKey(View v, int keyCode, KeyEvent event) {

                if (event.getAction() != KeyEvent.ACTION_DOWN && keyCode == KeyEvent.KEYCODE_BACK) {
                    if (GalleryState.getInstance().isGallery()) {
                        webView.reload();
                        GalleryState.getInstance().setGallery(false);
                    } else
                        onBackPressed();
                }

                return true;
            }
        });


これでギャラリーを開いたり、戻るボタンで閉じることができます。
