Overlay tekstur 2d pada objek 3d menggunakan p5.js (bagian 1 - membuat pola)


Melapisi gambar datar atau pola pada objek tiga dimensi selalu merupakan tugas yang tidak sepele, dan jika pada saat yang sama Anda ingin melakukannya dengan beberapa jenis animasi, kerumitannya meningkat pada waktu-waktu tertentu.

Pada saat yang sama, ada p5.js perpustakaan yang cukup sederhana, dengan mana Anda dapat membuat pola-pola spektakuler yang kompleks hanya dalam beberapa lusin baris kode. Saya baru-baru ini membuat proyek seperti itu dan sekarang saya ingin memberikan contoh.

Cara termudah untuk memulai adalah membuat proyek baru di editor.p5js.org online

Secara default, proyek baru adalah sebagai berikut:

function setup() {
  createCanvas(400, 400);
}

function draw() {
  background(220);
}

function setup () - mengatur fungsi yang dieksekusi hanya sekali
function draw () - mengulangi dalam loop tak terbatas.

Kita mulai dengan membuat pola radial 2D, yang kemudian kita terapkan pada kubus kita. Saat membuat itu, pertama-tama kita akan menggunakan koordinat Polar , dan hanya kemudian kita akan menerjemahkannya ke dalam koordinat Cartesian dari bentuk x, y yang akrab bagi kebanyakan orang.

Yang penting untuk dipahami: koordinat kutub dari suatu titik ditentukan oleh dua angka:

r adalah jarak dari titik sumber (misalnya, pusat kanvas) ke titik target.
a adalah sudut garis imajiner relatif terhadap titik awal di mana titik target

berada.Jadi, parameter pertama dapat bervariasi dari 0 hingga tak terbatas, dan yang kedua - dari 0 hingga 360.

Keindahan dari pendekatan ini adalah memungkinkan Anda untuk menggambar pola yang cukup kompleks hanya dengan satu formula.



Mawar polar diberikan oleh persamaan r (Ï†) = 2 \ sin Ï†

Untuk membuat variabel, mari kita tambahkan dua variabel di bagian paling awal proyek: yang pertama bertanggung jawab untuk jarak dari pusat dan yang kedua untuk sudut: let r = 10 let a = 0

Kemudian, dalam fungsi draw (), yang berjalan dalam loop tak terbatas, kita tambahkan perubahan variabel: a = a + 0,4

Sebagai hasilnya, Anda mendapatkan kode berikut:

let r =10
let a =0
function setup() {
  createCanvas(400, 400);

}

function draw() {
  background(220);
    a = a + 0.4
}

Sekarang kita mulai menerjemahkan koordinat ke dalam koordinat Cartesian x y yang kita kenal. Jika Anda ingin memahami apa yang sebenarnya terjadi dalam kode, lebih baik membaca tutorial trigonometri, tetapi karena ini tidak begitu penting untuk hasilnya, Anda cukup menempelkan kode ke fungsi draw () yang sama:

let x = r*cos(a)
let y = r*sin(a)

Untuk akhirnya melihat hasil perhitungan matematis kita, kita perlu melakukan dua hal:

  • melukis objek dengan warna (Anda dapat menggunakan kamus warna internal atau menentukan kode format RGB yang biasa - tiga angka)
  • mengatur bentuk distribusi titik. Misalnya, elips.

function draw() {
  background(220);
  
  
  let x = r*cos(a);
  let y = r*sin(a);
  fill(255,0,0);
  ellipse (x,y,10,10);
  r = r + 0.4
    
}



Parameter elips (posisi sepanjang X, posisi sepanjang Y, diameter sepanjang X, diameter sepanjang Y) ditetapkan menggunakan variabel sepanjang x dan y. Karena variabel x tergantung pada variabel r, yang dalam siklus kami terus meningkat sebesar 0,4, lingkaran merah yang kita lihat akan terus bergerak di sudut kiri atas.

Untuk memindahkan titik referensi ke pusat, tambahkan fungsi terjemahan (200.200) sebelum variabel x. Sekarang lingkaran akan muncul di tengah kanvas kita.

Dan mari kita selesaikan masalah lain - sekarang semua lingkaran yang muncul terhapus dengan setiap siklus baru. Untuk keluar dari lingkaran setan ini, kita perlu memindahkan pembuatan kanvas () di luar fungsi draw () dan mentransfernya ke fungsi setup (), yang bertanggung jawab untuk menciptakan lingkungan.

Setelah perubahan ini, kita akan melihat sesuatu yang mirip dengan lingkaran pemuatan. Masalah utama adalah bahwa sekarang mereka berada sangat dekat satu sama lain. Mari kita tambahkan beberapa ruang dengan menetapkan variabel orde tinggi C dengan nilai 20 dan menambahkannya untuk mengubah setiap siklus dengan 0,6 (di sebelah a). Nah, kami akan memperkenalkannya ke dalam rumus trigonometri kami: x = r + c * cos (a) dan y = r + c * sin (a)
