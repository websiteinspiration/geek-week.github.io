SQL Server now runs on Linux. This course will help you get a basic understanding of SQL Server on Linux. Then you will learn how to run SQL Server containers on Linux and deploy SQL Server on Linux. After that, you will learn how to automatically configure SQL Server deployment on Linux.

  • Choose the correct HDInsight Configuration to build open-source analytics solutions

  • Perform Zero ETL analytics with HDInsight Interactive Query

3. Implementing a data flow solution using Azure Streaming Analytics


Learn the principles of event processing and streaming and see how this applies to Azure Stream Analytics. Then set up your Stream Analytics job for streaming data and learn how to monitor and track job execution.

4. SQL Server Learning New


The latest version of SQL Server provides many benefits for increased performance and improved functionality, as well as enhanced support for the database platform. In this tutorial, you will learn how to inventory SQL Server deployments using the Microsoft Assessment and Planning Toolkit. In addition, you will learn how to upgrade SQL Server databases using the Data Migration Assistant and optimize your upgraded system using the Database Experimentation Assistant and the Query Tuning Advisor.

5. Working with NoSQL data in Azure Cosmos DB


NoSQL data is an efficient way to store data that does not meet the requirements of a relational SQL database. Find out how to use the Azure portal, the Azure Cosmos DB extension for Visual Studio Code, and the .NET Core SDK for Azure Cosmos DB to work with NoSQL data in any location, as well as provide users with a high level of accessibility no matter where they are.
