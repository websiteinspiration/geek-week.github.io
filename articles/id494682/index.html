Open Source vs. COVID-19: Bagaimana Pengembang Dapat Membantu Memerangi Virus




Programmer berada dalam posisi yang unik, mereka tidak hanya dapat bekerja di rumah selama pandemi COVID-19, tetapi mereka dapat membantu, menulis Jaxenter dan memberikan contoh:

  1. 2019 Novel Coronavirus COVID-19 (2019-nCoV) Data Repository oleh Johns Hopkins CSSE - visualisasi dari penyebaran virus di dunia.
  2. Proyek Pelacakan COVID - data "mentah" tentang penyebaran virus di Amerika Serikat.
  3. COVID-19 Italia - Monitoraggio situazione - data virus dari Italia;
  4. CoronaTracker adalah aplikasi untuk iOS dan macOS dengan peta dan grafik penyebaran virus.

Selain itu, ada proyek Folding @ Home, yang memungkinkan Anda untuk menyewakan kekuatan komputer Anda untuk penelitian ilmiah tentang melawan virus.

SUSE Menawarkan Bantuan Terhadap COVID-19




SUSE, salah satu perusahaan perangkat lunak open-source terbesar, memberikan bantuan kepada organisasi yang memproduksi perangkat medis untuk memerangi COVID-19, Tfir melaporkan. " Pandemi global saat ini membutuhkan lebih dari sekadar upaya untuk bertahan hidup ," kata CEO SUSE Melissa Di Donato. " Kami telah mengembangkan teknologi open source dan pengetahuan yang dapat membantu orang lain dalam perjuangan untuk menyelamatkan nyawa, dan kami akan membagikannya segera dan gratis ." SUSE telah membuat dukungan dan pemeliharaan sistem operasinya gratis untuk organisasi medis, dan versi OS ini juga tersedia untuk server dan komputer desktop dan untuk peralatan tertanam.

China bersiap untuk mengganti Windows dengan GNU / Linux




Rencana China untuk mengganti Windows dan beralih ke sistem operasi buatan sendiri bukanlah hal baru, tetapi tahun ini semuanya akhirnya dapat terjadi, Softpedia melaporkan. Inilah yang dijanjikan perusahaan China Union Tech, karena sistem operasi berbasis Linux UOS yang terpadu telah mencapai kesuksesan luar biasa selama beberapa bulan terakhir. Secara khusus, perusahaan China bekerja dengan perusahaan lokal lainnya untuk meluncurkan sistem operasi Linux pada chip yang dikembangkan di dalam negeri.

Linux Strace: Sejarah, Desain, dan Penggunaan




Dalam sistem operasi mirip Unix, program berkomunikasi dengan dunia luar dan sistem operasi melalui serangkaian fungsi kecil - panggilan sistem. Jadi, untuk keperluan debugging, berguna untuk memata-matai panggilan sistem yang dilakukan oleh proses. Shortwave - Pemutar Radio Internet Open Source Modern untuk GNU / Linux




Shortwave adalah pemutar radio open source yang menarik yang menawarkan antarmuka pengguna yang bagus dan kemampuan mendengarkan yang hebat ke stasiun Internet. Ini menggunakan basis data radio online yang dihasilkan komunitas. Shortwave sebenarnya merupakan penerus dari aplikasi radio GNU / Linux Gradio yang populer. Pengembangnya Felix bergabung dengan GNOME dan menghentikan Gradio untuk membuat gelombang pendek dari awal dalam bahasa pemrograman Rust. Jika Anda menggunakan Gradio sebagai pemutar radio internet pilihan Anda, Anda juga dapat mengimpor perpustakaan ke Shortwave. Shortwave baru-baru ini merilis rilis stabil pertama dan tampaknya terus merilis pembaruan setelah itu.

System76 Meluncurkan Laptop Lemur Pro Baru




System76 telah memproduksi PC berbasis GNU / Linux selama lebih dari satu dekade, dan perusahaan terus merilis sistem baru untuk mereka yang lebih memilih sistem operasi hegemoni Windows "alternatif". Dengan laptop Lemur Pro yang baru, perusahaan ini melengkapi jajaran laptop, desktop, dan server yang sudah kuat, tulis ZDNet. Lemur Pro menjadi laptop System76 ketiga yang menggunakan prosesor 14nm generasi ke-10 (atau Comet Lake), yang menggabungkan model Galago Pro dan Darter Pro, menawarkan baik Core i5-10210U atau Core i7-10510U. Ini mendukung hingga 40 GB RAM dan, dengan sepasang slot M.2 SSD, hingga 4 TB memori. Ini juga menampilkan layar Full HD 1920x1.080 14.1-inch, serta sepasang port USB 3.0 dan port USB 3.1 Type-C dengan opsi pengisian daya.

PostgreSQL Open Source




Linux.com menerbitkan artikel dengan tinjauan singkat tentang prinsip-prinsip yang mendasari PostgreSQL. Prevalensi dan demokrasi adalah dua kata yang paling menggambarkan sistem manajemen basis data open source PostgreSQL. Dan, seperti Linux, ini adalah contoh bagus "open source yang terbaik." Dalam hal prevalensinya, Postgres telah ada selama lebih dari 30 tahun, dan hari ini tampaknya berfungsi secara lokal atau sebagai layanan hosting dari puluhan pemasok di seluruh dunia. Dan, dari sudut pandang demokratis, proyek perangkat lunak ini dikelola oleh The PostgreSQL Global Development Group, yang merupakan pengembang dan sukarelawan dari seluruh dunia, dengan lebih dari 500 peserta. Tidak ada perusahaan yang bertanggung jawab, mendominasi atau mengontrol desain dan pengembangan perangkat lunak,oleh karena itu tidak ada bahaya ketergantungan pada orang atau organisasi tertentu. Tidak ada yang bisa "menangkap" Postgres. Bersama-sama, prevalensi dan demokrasi menciptakan kebebasan.

Tentang perdebatan tentang mengaktifkan dukungan exFAT di Linux




Ars Technica memahami perselisihan berikutnya antara pendukung dan penentang Open Source dan menulis bahwa ketika pada bulan Agustus raksasa perangkat lunak Microsoft mengumumkan dukungannya untuk dimasukkannya sistem file exFAT langsung di kernel Linux, ini tidak menerima banyak perhatian di media. Tetapi baru-baru ini, vendor sistem file Paragon Software memperhatikan bahwa bulan ini, versi exFAT yang dibuat oleh Microsoft dan sebagian besar dibuat oleh Samsung bergabung ke dalam repositori VFS untuk rilis kernel Linux berikutnya. Dan ini tidak menyenangkan Paragon, yang mengarah ke penerbitan artikel yang mengkritik solusi Open Source untuk masalah dengan exFAT.

Cloudflare telah menyiapkan tambalan yang secara dramatis mempercepat enkripsi disk di Linux




Pengembang Cloudflare berbicara tentang bekerja untuk mengoptimalkan kinerja enkripsi disk di kernel Linux, menurut OpenNET. Sebagai hasilnya, tambalan disiapkan untuk subsistem dm-crypt dan API Crypto, yang memungkinkan pengujian sintetik lebih dari dua kali lipat bandwidth untuk membaca dan menulis, serta membagi dua latensi. Saat menguji pada peralatan nyata, overhead dari enkripsi berkurang hingga hampir tingkat yang diamati ketika bekerja dengan disk tanpa menggunakan enkripsi data.

Rilis MythTV Media Center 31.0




Peluncuran platform untuk membuat pusat media rumah MythTV 31.0, yang memungkinkan Anda untuk mengubah PC desktop menjadi TV, VCR, pusat musik, album dengan foto, stasiun untuk merekam dan menonton DVD, telah dirilis. Pada saat yang sama, rilis antarmuka web MythWeb yang dikembangkan secara terpisah untuk mengelola pusat media melalui browser web terbentuk. Versi baru mendukung Python 3. Kemampuan yang terkait dengan decoding dan playback video telah meningkat secara signifikan: OpenGL sekarang digunakan untuk pemutaran video, dukungan untuk mempercepat decoding video menggunakan VAAPI VDPAU, NVDEC, VideoToolBox, Video4Linux2, MMAL dan MediaCodec disediakan. Dukungan untuk layanan DataDirect telah dihentikan, dan XMLTV harus digunakan sebagai gantinya. Alat pemindaian saluran yang diperluas secara signifikan.

Apache 21




Apache Software Foundation nirlaba sedang merayakan hari jadinya yang ke-21. Awalnya, organisasi ini diciptakan untuk memberikan dukungan hukum dan keuangan bagi pengembang server http Apache, tetapi kemudian diubah menjadi platform produsen individu yang netral dan independen untuk mengembangkan berbagai proyek terbuka yang menggunakan lisensi Apache, aturan pengembangan umum, prinsip meritokrasi, dan budaya komunikasi terpadu. . Pada saat yang sama, server HTTP httpd Apache, 25 tahun Apache OpenOffice office suite dan Apache Jakarta 20 tahun, Subversion dan Tomcat merayakan 25 tahun. Jumlah proyek yang dikembangkan oleh Apache telah melampaui 350 (di antaranya 45 di dalam inkubator), mencakup bidang-bidang seperti pembelajaran mesin, pemrosesan data yang besar, manajemen pembangunan, sistem cloud, manajemen konten, DevOps, IoT,pengembangan aplikasi mobile, sistem server dan kerangka kerja web. Pengembangan ini diawasi oleh lebih dari 7600 pengendara. Jumlah peserta yang mendukung dana selama 21 tahun terakhir telah meningkat dari 21 menjadi 765. Total biaya pengembangan dari awal 300 proyek Apache dengan lebih dari 200 juta baris kode diperkirakan $ 20 miliar ketika dihitung menggunakan model estimasi biaya COCOMO 2.

Sistem GNU / Linux modern harus mendukung menjalankan game lama - opini komunitas




LibrePlanet 2020, yang diselenggarakan oleh FSF, diadakan pada tingkat tinggi pada 15 Maret 2020. Ada banyak hal yang dibahas pada konferensi online. Salah satu topik diskusi adalah permainan di sistem GNU / Linux. Pengembang Dennis Payne mencoba untuk melihat ke belakang dan menunjukkan bahwa permainan gratis lama tidak lagi berjalan pada distribusi modern, tulis FossBytes. Menurut Payne, FSF perlu menggunakan "kebebasan untuk belajar dan memodifikasi" game gratis lama mereka untuk versi distribusi modern. Ini adalah satu-satunya cara untuk membawa mereka ke generasi berikutnya, dan kehilangan mereka seperti kehilangan bagian dari sejarah kita.
