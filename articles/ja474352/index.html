「マリオ」ユニティ2Dのようなジャンプによる敵の破壊

こんにちは！この記事では、マリオのように、飛躍による敵の破壊を実現する方法を共有したいと思います。私はまだ自分で勉強しているので、役に立つヒントやヒントがあれば、コメントに書いてください。まあ、チュートリアルと記事を読んだ後、興味深いものは2つしか見つかりませんでした。おそらくオプションを実装する最も簡単な方法は次のとおりです。

  • 最初はコライダーポイントの接触です。
  • 2つ目はコライダーの接触です。1つのオブジェクトに2つのコライダーがぶら下がっている場合、1つはプレイヤーの命を奪い、2つ目はアタッチされているオブジェクトを破壊します。

まず、プレーヤーの寿命を延ばす関数を作成します。これは、両方のオプションに必要です。

public int health = 3; 
void Hurt()
    {
        health--;
        if (health <= 0)
            SceneManager.LoadScene(SceneManager.GetActiveScene().buildIndex);
    }

1つのオプション


接触ポイントが高さ0.6以上の場合、最初のオプションを分析します。プレーヤーが敵の上からジャンプして破壊されます。それ以外の場合、プレーヤーはダメージを受けます。

void OnCollisionEnter2D(Collision2D collision)
    {
        Enemy enemy = collision.collider.GetComponent<Enemy>();
        if (enemy != null)
        {
            foreach (ContactPoint2D point in collision.contacts)
            {
                if (point.normal.y >= 0.6f)
                {
                    enemy.EnemyHurt();
                }
                else
                {
                    Hurt();
                }
            }
        }

（プレーヤーにハングアップしたコードのスニペット）

public void EnemyHurt()
    {
        Destroy(this.gameObject);
    }

（敵に掛かるコードの断片）

オプション2


コライダーを追加し、メインコライダーより少し高くし、オブジェクトに「is Trigger」というチェックマークを付け、スクリプトをハングさせて、このコードフラグメントを追加します。 

private void OnTriggerEnter2D(Collider2D collision)
    {
        if (collision.gameObject.name == "Player")
            Destroy(this.gameObject);
    }

（敵にぶら下がっているコードのスニペット）

コライダーが「プレイヤー」と呼ばれるオブジェクトに触れると、オブジェクトは破棄されます（コンタクトマークとして、タグを使用できます）。

さらに同じオブジェクトで、もう1つのgameObjectを作成し、DeathZoneと呼びます。コライダーを追加し、少し大きくして、「is Trigger」ボックスをチェックします。



  private void OnTriggerEnter2D(Collider2D collision)
    {
        if (collision.gameObject.name == "DeathZone")
        {
            Hurt();
        }
    }

（プレーヤーでハングするコードの断片）

ご覧のとおり、最初のオプションとは異なり、2番目のバージョンでは、コライダーの使用がわずかに大きく、トリガーを使用する必要がありませんでした。


（2番目のオプション）

敵の破壊後にジャンプするようにプレーヤーに追加することもできます。さて、すべてがそうであるように、あなたの注意をありがとう！
