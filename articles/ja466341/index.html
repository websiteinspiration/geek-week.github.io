Cisco 200-125 CCNA v3.0のトレーニング。33日目。ICND1試験の準備

CCNA 100-105コースICND1試験に合格するために必要なトピックの学習が完了したので、本日、ピアソンVUE Webサイトでこの試験に登録し、テストに合格して証明書を取得する方法について説明します。また、これらの一連のビデオチュートリアルを無料で保存する方法と、NetworKingコンテンツを使用するためのベストプラクティスを紹介します。

これで、ICND1試験のすべてのトピックを学習し、登録、つまりテストにサインアップできるようになりました。まず、ブラウザを起動してcisco.comにアクセスします。



翻訳者のメモ：2017年7月14日にビデオチュートリアルを更新するために、2019年6月現在のシスコのWebサイトのスクリーンショットを以下に示し、レッスンのテキストをそれに応じて修正しました。

次に、ページの左上にある[メニュー]タブをクリックし、サイトのセクションのドロップダウンリストに移動して、[トレーニングとイベント-認定-CCENT]セクションを選択します。



CCENTリンクをクリックすると、認証ページに移動します。



ここでは、シスコ認定が必要な理由の詳細情報を見つけることができます。ページを下にスクロールすると、興味のある試験100-105 ICND1へのリンクが表示されます。



このリンクをクリックすると、この試験の詳細な説明が記載されたページに移動します。



試験の指定の下に、正常に完了した後に取得できる証明書が表示され、試験は90分間続き、質問の数は45〜55であり、利用可能な試験言語は英語と日本語です。中東地域にいる場合は、アラビア語もオプションとして利用できます。

翻訳者注：ロシアにいて英語を選択した場合、外国語に対応するために試験に合格するためにさらに20分（90分ではなく110分）を与えることができます。シスコの地域の証明機関でロシア語の試験に合格するには、同じ90分かかります。

試験のトピックのリンクをクリックすると、この試験がカバーするすべてのトピックを表示できます。私はこれに時間を費やしませんが、最も重要なこと、つまりテストへの登録方法について話します。

登録するには、「ピアソンVUEで登録」リンクを使用します。それをクリックすると、世界中のシスコの証明書を取得するための試験を管理する組織であるピアソンVUEのページが表示されます。同社は多くの組織に対してテストを実施する権利を提供しており、受験者向け、つまり「試験に合格した人向け」のリンクをクリックすると、受験する権利を持つすべての人と知り合うことができます。ただし、関心があるのはPearson VUE with Cisco試験のみであり、対応するページはhome.pearsonvue.com/ciscoにあります。



アカウントを作成する必要があります。無料です。アカウントの作成ボタンをクリックするだけです。すでにアカウントを持っているので、[サインイン]ボタンをクリックして[ホーム]タブに移動します。ここでは、監督付き試験ボタン、つまり認定シスコ担当者の監督下で行われるフルタイム試験に関心があります。

翻訳者のメモ：登録時に、ユーザーはユーザー名、パスワードを考え、電話番号、電子メール、住所を示し、秘密の質問を2つ選択し、それらに回答する必要があります。ユーザー名とその識別子による登録の確認は、数分以内にメールアドレスに届きます。



監督付き試験ボタンをクリックすると、興味のある試験を選択するためのページが表示されます。



名前を手動で入力しないようにするには、ドロップダウンメニューの[監督付き試験]をクリックする必要があります。その後、すべてのフルタイム試験のリストがページに表示されます。



ICND1試験を受ける場合は、100-105行目をクリックしてください。ICND2コースの2番目の部分が200-105行目である場合、CCNA総合試験に合格する場合は、200-125を選択してください。したがって、100-105をクリックすると、試験の言語（英語または日本語）を選択するためのページが表示されます。



私は英語を選択し、試験の費用が記載された次のページに進みます。 [テストポリシーの表示]リンクをクリックすると、試験に合格するためのすべてのルールを確認できます。テストの費用は165ドルです。



「この試験のスケジュールを設定する」ボタンをクリックすると、シスコ試験の契約条件への同意を確認するページにリダイレクトされます。



[はい、同意します]チェックボックスをオンにする前に、上記のリンクをクリックして、.pdf形式の追加情報を確認できます。

次に、近くのテストセンターを選択する必要があります。登録時に自宅の住所を指定した場合、システムはそれを「最寄りのテストセンターを検索」の行に自動的に配置し、住所を提供します。ページの右側には、最も近いセンターの場所が記載されたマップがあります（翻訳者注：スクリーンショットは、モスクワ南西部行政区の認証センターを示しています）。



登録時に住所を入力しなかった場合は、ロンドンなどの都市を入力すると、この都市にあるすべてのシスコテストセンターが表示されます。ご覧のように、そもそも最寄りの中心部は市の中心部から1.9マイルの距離にあり、残りはロンドン中心部から順に削除されています。

名前の左側にあるチェックボックスで鳥でマークすることにより、任意の中心を選択できます。センターを選択すると、システムは最も近い利用可能な日付を選択するためのページにリダイレクトします。この場合、空きスペースを探してカレンダーを移動するか、より適切な日付の別のセンターを選択する必要があります。

翻訳者注：2019年6月17日、モスクワのulにある教育センターで試験に合格する最も近い日付。あっピリジナ、9月4〜3日。



日付を決めるとすぐに、システムは試験の開始時間を選択するように提案します。時間を選択すると、生成された注文のページが表示され、



日付、時間、試験に合格した場所、試験の費用がここに表示されます。このページでは、[予定の変更]リンクをクリックして日時を変更したり、[テストセンターの変更]リンクをクリックしてテストセンターを変更したりできます。また、試験価格の近くにある[削除]ボタンをクリックして、注文自体を削除することもできます。ページの下部に、選択した追加のテスト（Cisco Approved Test 200-105など）を考慮して、試験の最終的な費用が表示されます。



合計金額の下に、[チェックアウトに進む]-[チェックアウトに移動]ボタンがあります。このボタンをクリックすると、個人情報（名前と電話番号）の確認ページに移動します。このページでは、試験の言語を変更できます。次に、2番目のステップ、シスコポリシーの理解と受け入れ、3番目のステップ-クレジットカードによる試験の支払いを行います。行われた支払い済みの注文に関する情報がメールアドレスに送信され、予定されている試験の記録がピアソンVUEのプロフィールページに表示されます。

試験の指定時間の15〜20分前に出頭する必要があります。たとえば、パスポートと運転免許証、またはパスポートと軍のIDなど、2種類の身分証明書が必要です。試験の前に写真を撮り、タブレットにサインするように求めて電子署名を行います。その後、テストが行​​われるコンピューターへのアクセス権が与えられます。試験を開始する前に、15分以内にシステムに慣れてください。次に、回答オプション付きの1つの質問が画面に表示されます。回答を選択してクリックし、次の質問に進みます。いくつかの質問には、より多くの回答があります。同じ日に同じセンターで友達と一緒に受験した場合、チャンスはありませんあなたが同じ質問に出くわすこと。

試験に合格するために必要なポイント数は事前に不明であり、試験の終了まで、必要なポイント数を獲得したかどうかは質問の数と複雑さによって異なるため、わかりません。試験が完了すると、システムは試験に合格するために必要なポイント数、獲得したポイントを表示し、試験に合格したかどうかを示します。

このテストがどのようなものかを事前に知りたい場合は、選択したシスコ試験サイトwww.cisco.com/c/en/us/training-events/training-certifications/exams/current-list/100-105-icnd1のページにアクセスしてください。 html [試験問題のサンプル]ボタンをクリックします。



その後、learningnetwork.cisco.com / docs / DOC-34312ページが表示されます。Flash Playerを表示する必要があるトレーニングビデオがあるので、長いページの読み込みに驚かないでください。ここでは、コンピューター画面でテストがどのように行われるかを確認します。



これらのビデオは、試験の様子を事前に理解するのに役立ちます。
そこで、ピアソンVUE Webサイトでの登録方法、適切な試験の選択方法、テストセンター、および完了日について説明しました。問題なくICND1に合格していただければ幸いです。

次に、ビデオチュートリアルを無料で入手する方法を説明します。 3年前、講義をYouTubeにアップロードし始めたとき、自分が何を望んでいるのかまったくわかりませんでした。適当な無料のトレーニング資料が見つからず、このトピックに関する無料のYouTube動画の質がひどいので、なんとかして状況を修正する必要があると思いました。 3年間、私は約35本のビデオを録画しましたが、これは私の主な仕事ではないため、すべてのレッスンであなたのコメントに応答する時間がないために罪悪感を感じています。時間があれば、次のトレーニングシリーズを録画してアップロードします。

私は自分の主な仕事にフルタイムで取り組み、いくつかのプロジェクトを実行し、家業を行っており、これらすべてを同時に行っています。しかし、動画の下のコメントに返信しないと、視聴料金を支払って対応したサービスを受けられなかったかのように気分を害する人もいます。しかし、私はそれを無料で行います、私は人々に私の助けをもらいたいです。もっと時間をかけたいのですが、余裕がありません。これらのビデオチュートリアルの下に数百、数千のコメントが表示され、一部の人からこのコースを有料にするように依頼されます。これらのビデオチュートリアルを速く行うことはできませんでしたが、今はペースを加速する必要があると感じています。 ICND2コースのトピックをカバーするために、約35のエピソードが残っていますか？そして、もし2か月以内にそれらを作ることができるかどうか尋ねれば、私は答えることができません。知りません、これに十分な時間がありますか？私は他のプロジェクトを犠牲にしてこれにもっと多くの時間を割くことができたかもしれませんが、私は自分の財政状況を悪化させて、有給労働を犠牲にして自由労働を引き受ける余裕がないので、すべて経済的利益に依存します。

人々は私の仕事への自発的な寄付を受け入れない理由を尋ねられます。彼らは私の仕事を財政的に支援したいと思っているからです。やりたくありませんでしたが、多くの人がこのプロジェクトに貢献する準備ができているので、そういう機会を与えることにしました。寄付をしたい場合は、当社のウェブサイトnwking.orgにアクセスし、PayPalを使用してSupport usリンクを使用してください。この動画の右上にあるリンクをクリックすると、寄付ページにアクセスできるようになります。

ビデオチュートリアルでの「いいね」は、コースの人気に貢献するため、より重要です。そしてもちろん、「共有」ボタンを使用することを忘れないでください、それは私が新しいビデオを投稿したことをあなたの友達に表示します。



ICND2コースの有料版を作成することを決定した場合、寄付を行う人々は有利になります。現在、最低寄付額は10ドルですが、有料のビデオレッスンはこのお金を稼ぐ人には完全に無料です。そのため、わずか10ドルを支払うことで、有料版でより多くを節約する機会を得られます。サイトによってはサービスに1〜2ドルかかるため、コースのバージョンを取得するためにそれらを支払う必要がある場合がありますが、いずれの場合も実際のコストよりもはるかに安くなります。寄付をしたすべての人が無料でビデオチュートリアルを受け取ることを約束します。
もう1つの重要なことは、電子メールで問題が発生していることです。人々からたくさんの手紙が送られてくるので、全員に答える方法はまったくありません。したがって、私はそのようなポリシーを使用することを決定しました-私は自発的な寄付をした人にのみメールに返信します。これを行うには、特別なメールフィルターを使用して、これらの人々の手紙がすべての着信メッセージの先頭に配置されるようにし、それらに応答します。私は決して寄付を強制するものではありません。無料のビデオレッスンがネットワークに投稿されている場合は、それらを使用してください。ただし、有料のビデオレッスンが無料で利用できるとは限りません。すぐに私は有料ビデオチュートリアルの出現に関する問題を解決すると思います。
次に、ビデオチュートリアルを最大限に活用する方法について説明します。まずはレッスンをじっくり見てください！私の最初のビデオを見始めた一部のユーザーは、ネットワーキングの完璧な「初心者」でした。しかし、現在、約35のビデオチュートリアルを見て、彼らはもっと多くのことを知っています。

これまで、いくつかのトピックは理解できないように思われたかもしれません。以前に理解できなかったことを理解できるようになったため、知識を習得したので、レッスンに戻ってもう一度確認することをお勧めします。一部の人々は概念を暗記しようとしますが、これは学ぶための最良の方法ではありません。あなたは学び、何が危機に瀕しているのかを理解しなければなりません。概念を理解すると、それを覚える必要はすぐになくなります。問題の本質を理解すれば、他のすべてはすぐに簡単になります。

だから、ビデオを確認してください。初めてサブネットなどのトピックがわからない場合は、戻ってもう一度ビデオチュートリアルをご覧ください。 ASLの内容がわからない場合は、このビデオをもう一度ご覧ください。ビデオを見るたびに、初めて気づかなかった新しいことを学びます。一度見ると何もわからないかもしれませんが、もう一度見ると何かを学びます。これが脳の働きです。何か新しいことを学ぶとき、私たちは何かを理解し始めます。

次に重要なことは、レッスンを見ながらノートに自分でメモをとることです。ビデオを見た後、ラップトップ、すべての電子機器を脇に置き、ペンと紙のノートを取り、すべての主要なポイントを書き留め、レッスンのコンセプトを母国語で述べます。将来、メモを読み直すと、忘れたものを思い出すことができるようになります。

学生時代、メモをとって、緑のペンでコードを書き、重要なトピックを赤で強調し、普通のメモを青で書きました。古いメモを見つけたら、サンプルをTwitterに投稿して、見られるようにします。さて、何かを忘れてしまったら、以前のメモに戻ります。これにより、すべてのトピックを平等に思い出すことができます。誰が教えても、自分のノートが一番の先生です。
