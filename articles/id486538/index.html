Bagaimana saya mengunggah soft skill online dan apa yang diberikannya kepada saya

Latar Belakang


Halo semuanya! Nama saya Alexey. Selama umur pendek saya (30 tahun), saya telah mengubah banyak profesi. Fitur ini - saya tidak bisa melakukan satu hal untuk waktu yang lama. Bunga hilang, bahkan sampai ke masalah kesehatan. Baru-baru ini, saya memutuskan untuk masuk ke perusahaan itu. Lulus kursus, baca buku untuk siap.

Sampai akhir tahun lalu, saya tidak tahu tentang hal seperti "soft skill". Dan saya belajar tentang dia dari bibir jam. Semua perjalanan wawancara terakhir berakhir dengan kegagalan setelah saya ditanya tentang keterampilan yang sangat lembut ini.

Sebagai contoh, saya ditanyai pertanyaan seperti itu:

  • "Kapan terakhir kali kamu tersinggung oleh seseorang?"
  • "Apa pemimpin terbaikmu?"
  • "Bagaimana kamu sampai ke tugas yang tidak dikenal?"
  • "Bagaimana perasaanmu tentang kritik?"
  • "Apakah kamu tidak terlalu banyak bicara?"

Pertanyaan terakhir sama sekali membuat saya gelisah dan membuat saya gugup.

Saya berulang kali membuat pengamatan bahwa keterampilan komunikasi saya tidak cukup berkembang. Secara alami, saya marah dan tidak mengerti apa yang mereka inginkan dari saya. Di sini saya duduk di depan Anda, berbicara. Apalagi saya menyelesaikan tugas tes. Dan secara umum, saya selalu percaya bahwa saya dapat berkomunikasi dengan orang-orang.

Cari


Saya tidak akan menyerah. Setelah kemarahan hilang, saya mulai mencari informasi tentang soft skill. Secara umum, ini adalah topik yang menarik, tetapi tidak segera jelas. Ini adalah beberapa hal mental yang, seperti, dapat dipelajari.

Cari memberi saya pilihan buku. Tetapi semua buku pengembangan diri terakhir yang dibaca hanyalah diisi dengan air dan prangko. Saya segera menolak opsi ini. Tetapi jika Anda mempercayai buku, maka berikut adalah daftar singkat dari apa yang saya temukan:

Saya juga skeptis tentang pelatihan. Bagaimana seseorang dapat mempelajari keterampilan dalam beberapa jam?

Saya mulai membaca artikel tentang topik itu, berusaha mencari informasi yang berharga.

Setelah beberapa saat saya menemukan disini materi menarik tentang perkelahian manajerial. Topiknya sangat menarik, karena secara langsung memengaruhi komunikasi dan ketahanan terhadap stres, tetapi, seperti yang beruntung, tidak ada klub semacam itu di kota saya.

"Bagaimana cara mengembangkan keterampilan sialan ini jika mereka tidak punya tempat untuk belajar?" - Saya pikir, dan pada saat itu algoritma mesin pencari menunjukkan kepada saya sebuah iklan dengan permintaan untuk memompa soft skill dalam 3 minggu di sebuah turnamen online. Saya tidak akan rugi - saya mengikuti tautan.

Singkatnya, saya ditawari simulator online untuk mengembangkan soft skill. Penyelenggara mengatakan bahwa proses ini dibangun dalam format permainan. Karena video game adalah hasrat saya, itu menyuap saya, dan saya mendaftar.

Partisipasi


Segera ada hambatan - tidak ada versi iOS lengkap. Saya harus mengambil smartphone Android istri saya dari istri saya setiap malam. Tetapi kenyataan bahwa semuanya terjadi secara online dan tidak ada keterikatan pada tempat tersebut adalah nilai tambah.

Jadi, saya mendapat akses ke platform, di mana beberapa kasus diberikan, di mana situasi yang berbeda dimainkan antara bawahan dan bos, siswa dan guru, dua rekan kerja, dll. Itu perlu untuk memilih dan berduel dengan lawan. Awalnya semuanya sangat tidak bisa dimengerti, saya bingung, tidak tahu harus berkata apa. Peserta lain menunjukkan bahwa mereka memiliki cerita yang sama. Pikiran itu muncul: “Yah, saya katakan ke kamera, jadi apa? Bagaimana ini mengajari saya? Ada skype. "

Tetapi di webinar mereka menjelaskan kepada kami bahwa ini normal dan perlu dilanjutkan. Nah, begitu saya mulai, saya memutuskan untuk tidak mundur.

Minggu pertama adalah pelatihan. Saya pikir ini benar, karena itu perlu untuk memahami segalanya. Selama waktu ini, penghalang menghilang - tidak nyaman bagi saya untuk berkomunikasi dengan orang asing melalui kamera. Tinggal tidak ada masalah seperti itu. Berdasarkan pengalaman bermain game saya, saya mulai memperkenalkan lawan sebagai NPC dari game bermain peran yang diprogram untuk replika tertentu.

Setelah perkelahian, semua orang mengevaluasi dirinya dan lawannya. Tampaknya Anda hanya mengambil dan memberikan suara untuk diri sendiri. Tetapi setiap peserta memiliki parameter "Kecukupan harga diri", yang jatuh ketika Anda menilai secara tidak adil. Jika ada hasil seri, pertandingan akan dinilai oleh orang lain dari para peserta.

Hal yang paling menarik dimulai pada minggu kedua, ketika semua orang dibagi oleh tim. Di sini turnamen sesungguhnya telah dimulai. Setiap pertandingan membawa dan mengambil poin tidak hanya dari saya, tetapi juga dari seluruh tim. Ada rasa tanggung jawab. Dalam obrolan tim, kami berbicara satu sama lain dan menyelesaikan perkelahian.

Bukan tanpa hal-hal yang tidak menyenangkan. Karena perbedaan zona waktu, tidak selalu mungkin untuk menemukan siapa yang bermain, karena seseorang berasal dari Vladivostok, dan seseorang dari Tambov.

Terkadang platform berhenti bekerja dan tidak bisa bermain. Setelah sorti ini, saya harus login lagi. Dukungan teknis menjawab bahwa masalah muncul karena platform ini baru dan mereka meluncurkan turnamen semacam itu untuk pertama kalinya. Untungnya, mereka menjawab dengan cepat.

Akibatnya, tim kami masuk lima besar, tetapi secara pribadi saya tidak memiliki cukup poin untuk mencapai tahap akhir, di mana semua orang sudah siap untuk dirinya sendiri.

hasil


Pada awalnya, tidak sepenuhnya jelas bagi saya bagaimana dan apa yang diajarkan percakapan tentang komunikasi video kepada saya. Tetapi ternyata hasilnya diperbaiki dengan pengulangan yang berulang. Karena pekerjaan, saya jatuh selama dua hari dan merasakan kemunduran. Perlahan-lahan, saya mulai menyadari betapa cepatnya saya berusaha menentukan inti dari konflik, tidak hanya dalam permainan, tetapi juga dalam percakapan dalam kehidupan. Setelah belajar tentang bidang-bidang konflik, saya dengan sadar mencoba mempertimbangkan setiap situasi dari sudut pandang yang berbeda.

Ngomong-ngomong, sangat nyaman bahwa Anda bisa menonton perkelahian Anda di catatan. Ternyata saya memiliki ekspresi wajah yang aneh. Saya memperhatikan diri saya sendiri dan mengubah perilaku saya.

Apa yang saya sebut soft skill tanpa disadari, memang, terpompa. Saya tidak bisa mengatakan berapa persen atau berapa kali, karena, saya ulangi, sulit untuk diukur. Tetapi hasilnya adalah sebagai berikut:

  • Dia mendapat pekerjaan di perusahaan IT sebagai analis.
  • Saya secara sadar mengikuti bahasa tubuh saya. Belajar untuk mengenali bagaimana postur dan ekspresi wajah terkait dengan suasana hati.
  • Ini menjadi kurang digunakan dalam pidato kata-kata parasit dan mooing. Format pertarungan tidak memungkinkan pemborosan waktu.
  • Saya tidak belajar untuk menghindari konflik, tetapi menjadi lebih mudah untuk menyelesaikannya.
  • Menjadi lebih mudah untuk menyampaikan pemikiran Anda kepada orang lain
  • Mengetahui apa yang akan menjadi pembicaraan, kebiasaan itu muncul untuk memikirkan skenarionya di muka.
  • Dia mulai membedakan antara konsep "mendengarkan" dan "mendengar."

Panitia berjanji akan meluncurkan klub online setelah turnamen.

Jika kota Anda memiliki klub perjuangan manajemen, saya sangat menyarankan Anda untuk mendaftar di sana. Ini akan bermanfaat bagi perwakilan dari profesi apa pun. Jika tidak, maka versi online sangat baik.
