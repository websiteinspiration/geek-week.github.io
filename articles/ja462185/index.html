革命は終わりました。リチウムイオン電池の代替品はありますか？



最近、私たちはリチウムイオン電池の発明の歴史について話しました。それは携帯用電子機器の開発に強力な原動力を与えました。毎年、技術メディアは次のエネルギー革命について私たちに知らせています-ほんの少し、もう1年か2年、そして世界は素晴らしい特性を備えたバッテリーを目にするでしょう。時が経ち、革命が見えなくなったとき、私たちの電話、ラップトップ、クワッドコプター、電気自動車、およびスマートウォッチには、リチウムイオン電池のさまざまな変更がまだインストールされています。それで、すべての革新的なバッテリーはどこに行きましたか、そしてリチウムイオンの代替品はまったくありますか？

バッテリーの革命をいつ待つべきか？


あなたを怒らせるのは残念ですが、それはすでに過ぎ去っています。それは数十年にわたって伸びただけなので、ほとんど気付かれませんでした。事実、リチウムイオン電池の発明は、化学電池の進化の頂点でした。

化学電流源は、元素間のレドックス反応に基づいています。周期表には、そのような反応に参加できる天然元素は90しかありません。したがって、リチウムは極度の特性を備えた金属であることがわかりました。つまり、最低の質量、最低の電極電位（–3.05 V）、最高の電流負荷（3.83 A・h / g）です。

リチウムは地球上で最も優れたカソード活性物質です。他の要素を使用すると、1つの特性が向上し、必然的に別の特性が悪化する可能性があります。そのため、リチウム電池の実験は30年前から続けられています-リチウムが常に存在する材料を組み合わせることにより、研究者は、非常に狭い用途を見出す必要な特性を持つタイプの電池を作成します。前世紀の80年代から登場した、コバルト酸リチウムカソードを備えた古き良きバッテリーは、電圧、電流負荷、エネルギー密度の優れた組み合わせにより、依然として最も一般的で普遍的なものと見なすことができます。

したがって、メディアの口から次のスタートアップが世界に日々のエネルギー革命を大声で約束するとき、科学者たちは新しいバッテリーにまだ解決されていないいくつかの問題と制限があるという事実について控えめに沈黙しています。通常は解決できません。

「革命的な」バッテリーの主な問題


今日、リチウムを使用しないなど、化学組成が異なる多くの種類の電池があります。その特性を持つ各タイプは、特定のタイプのテクノロジーに適用されています。軽量、薄型、高電圧のリチウムコバルト電池は、コンパクトなスマートフォンに長い間処方されてきました。丈夫で強力ですが、非常に大型のチタン酸リチウム電池が公共交通機関に適合します。また、低容量の耐火性リン酸リチウム電池は、発電所の大型アレイとして使用されています。

それでも、最も需要が高いのは、民生用モバイル機器用のリチウムコバルト電池です。彼らが満たす主な基準は、単位体積あたりの高いエネルギー強度を維持しながら、3.6 Vの高電圧です。残念ながら、多くの代替タイプのリチウム電池には、3.0 V未満、さらには2.0 V未満というはるかに低い電圧があり、それから現代のスマートフォンに電力を供給することは不可能です。

バッテリーを組み合わせてセルにすることで、特性の沈下を補正できますが、その後寸法が大きくなります。したがって、奇跡的な特性を備えた次の有望なバッテリーがモバイルテクノロジーや電気自動車での使用に適さない場合、その将来はほぼ決まっています。10万回の寿命と急速充電が可能なバッテリーが必要なのはなぜですか？

失敗した実験


以下で説明するすべてのバッテリーが失敗したと見なすことができるわけではありません-非常に長い開発が必要なものもあれば、スマートフォンではなく特殊な機器にそれらのアプリケーションが見つかる場合もあります。それにもかかわらず、これらすべての開発は、スマートフォンのリチウムイオン電池の代替品として位置付けられました。

2007年、アメリカのスタートアップLeyden Energyは、いくつかのベンチャーキャピタルファンドから450万ドルの投資を受け、彼らが主張するように、新世代のリチウムイオンバッテリーを生み出しました。同社は新しい電解質（ソルベントインソルト）とシリコンカソードを使用し、エネルギー強度と300°Cまでの高温への耐性を大幅に向上させました。開発に基づいてラップトップバッテリーを作る試みは失敗したため、ライデンエナジーは電気自動車市場に移行しました。

数千万ドルの絶え間ない注入にもかかわらず、同社は安定した特性を持つバッテリーの生産を確立することができませんでした-インジケーターはコピーからコピーへと変動しました。会社にもっと時間と資金がある場合、おそらく2012年には、機器や特許を販売したり、別のエネルギー会社であるA123 Systemsの傘下に入る必要はないでしょう。

リチウム金属電池-ニュースではありません：非充電式リチウム電池はその1つです。 SolidEnergyは充電式リチウム金属電池の開発に着手しました。新製品はリチウムコバルト電池と比較して2倍のエネルギー強度を持っていました。つまり、前の巻では2倍のエネルギーを収めることができました。カソードに従来のグラファイトの代わりに、リチウム金属箔を使用しました。最近まで、リチウム金属電池はデンドライトの成長（木質金属層のアノードとカソードで成長）のために非常に爆発的であり、短絡を引き起こしましたが、電解質に硫黄とリンを追加することでデンドライトを取り除くことができました（SolidEnergyにはまだテクノロジーがありませんが） ）非常に高い価格に加えて、SolidEnergyバッテリーの既知の問題には、長時間の充電（1時間あたり容量の20％）があります。

マグネシウム硫黄元素
に関する積極的な作業は、トヨタがこの分野の研究を発表した2010年代に始まりました。このようなバッテリーのアノードはマグネシウム（良いが、同等のリチウムアナログではありません）、カソードは硫黄とグラファイトで構成され、電解質はNaClの通常の塩溶液です。電解液の問題は、硫黄を破壊してバッテリーを作動不能にすることです。そのため、使用する直前に電解液を充填する必要がありました。

トヨタのエンジニアは、非求核性粒子から、硫黄に対して非攻撃的な電解質を作成しました。結局のところ、安定化されたバッテリーは、50サイクル後に容量が半分に低下するため、長期間使用することは依然として不可能です。2015年、リチウムイオン添加剤がバッテリーに統合され、さらに2年後に電解質が更新され、バッテリーの寿命は110サイクルになりました。このような不機嫌そうなバッテリーで作業が続行される唯一の理由は、高い理論エネルギー強度（1722 Wh / kg）です。しかし、成功したプロトタイプが登場する頃には、マグネシウム硫黄元素はもはや必要ないことが判明するかもしれません。

エネルギー貯蔵の代わりに発電


何人かの研究者は反対から行くことを提案します：貯蔵しないでください、しかしデバイスで直接エネルギーを生成します。スマートフォンを小さな発電所にすることは可能ですか？過去10年間、主電源から充電する必要をガジェットから取り除くための試みがいくつかあります。現在のスマートフォンの充電方法から判断すると、その試みは失敗に終わりました。最も「成功した」発明を思い出してみましょう。

メタノール直接分解燃料電池（DFMC）。メタノール燃料電池をモバイル技術に導入する試みは、2000年代半ばに始まりました。現時点では、長寿命の押しボタン電話から大画面の要求の厳しいスマートフォンへの移行がありました。リチウムイオンバッテリーは最大2日間持続したため、瞬時に充電するというアイデアは非常に魅力的であるように思われました。

燃料電池では、電解質として機能する高分子膜上のメタノールが二酸化炭素に酸化されます。水素陽子はカソードに移動し、酸素と結合して水を形成します。ニュアンス：効果的な反応には約120°Cの温度が必要ですが、元素のコストに自然に影響するプラチナ触媒に置き換えることができます。

燃料電池を電話ケースに収めることは不可能であることが判明しました。燃料室の寸法が大きすぎることが判明しました。したがって、2000年代の終わりまでに、DFMCのアイデアはポータブルバッテリー（パワーバンク）の形で具体化しました。 2009年、東芝はDynarioと呼ばれるシリアルメタノールパワーバンクを立ち上げました。重さは280 gで、30,000 mAhの最新のポータブルバッテリーに似ています。つまり、手のひらのサイズでした。日本のDynarioの価格は、それぞれ50 mlのメタノールの5つのバイアルのセットに対して、328ドルと36ドルという印象的でした。 1回の「燃料補給」には14 mlが必要です。その容量は、500 mAの電流でUSBを介してプッシュボタン式電話を2回充電するのに十分でした。

Toshiba Dynarioの給油と操作を説明するビデオ

3,000部の実験的なバッチのリリースを超えたわけではありませんでした。これは、燃料電池があまりにも物議を醸していることが判明したためです。さらに、メタノールは有毒であり、一部の国では、メタノールを販売し、さらには購入するためのライセンスが必要です。

透明な太陽電池パネル。ソーラーパネルは、太陽の無限のエネルギー（今世紀）を抽出するための優れたソリューションです。このようなパネルは、高効率で低コストで低電力ですが、発電する最も簡単な方法です。しかし、人類の本当の夢は透明なソーラーパネルであり、ガラスの代わりに家、車、温室の窓に取り付けることができます。言うまでもなく、ビジネスと喜びを組み合わせる-発電と宇宙の自然照明。良いニュースは、透明な太陽電池パネルが存在することです。悪い点は、それらが実質的に役に立たないことです。


開発者とミシガン大学は、フレームのない透明なパネルを紹介しています。出典：YouTube /ミシガン州立大学

光子を「キャッチ」して電気に変換するために、ソーラーパネルは原則として透明にすることはできませんが、新しい透明な材料はUVおよびIR放射を吸収し、すべてを赤外線範囲に変換してパネルの端まで取り除きます。透明パネルの端には、通常のシリコン太陽電池パネルがフレームとして設置されており、赤外線領域で抽出された光を受けて発電します。このシステムは、1〜3％の効率でのみ機能します...最新のソーラーパネルの平均効率は20％です。

ソリューションの疑わしい有効性にもかかわらず、有名な時計メーカーであるTAG Heuerは2014年にプレミアムタグ電話Tag Heuer Meridiist Infiniteを発表し、Wysis製の透明なソーラーパネルが画面の上に設置されました。スマートフォン向けのソリューションが発表されている間でも、Wysisは約5 mWの太陽電池の電力を1 cm2の画面と非常に小さいと約束しました。たとえば、iPhone Xの画面ではこれはわずか0.4 Wですが、Appleスマートフォンの完全なアダプターが5 Wの低電力のために叱られていることを考えると、0.4 Wでは充電できないことは明らかです。

ちなみに、メタノールでは機能しませんでしたが、水素燃料電池は生きるきっかけとなり、トヨタの未来電気自動車や東芝のモバイル発電所の基礎となりました。

そして何が起こったのか：Li-Ionで成功した実験


世界をひっくり返したくないが、電池の個々の特性を改善することに単に努力した人々によって成功は達成されました。正極材料を変更すると、電圧、エネルギー強度、およびバッテリー寿命に大きく影響します。次に、称賛されている開発について話します。これは、リチウムイオン技術の普遍性をもう一度確認します。「革新的な」開発ごとに、より効率的で安価な既存のアナログがあります。

リチウムコバルト（LiCoO2、またはLCO）。動作電圧：3.6 V、最大200 W・h / kg、最大1000サイクルの寿命。グラファイトアノード、リチウムコバルト酸化物カソード、上記のクラシックバッテリー。この組み合わせは、単位体積あたりの高いエネルギー消費が必要なモバイル機器のバッテリーで最もよく使用されます。

リチウムマンガン（LiMn2O4、またはLMO）。動作電圧：3.7 V、最大150 W・h / kg、最大700サイクルの寿命。最初の効果的な代替組成物は、リチウムイオン電池自体の販売が始まる前から開発されていました。正極にはリチウムマンガンスピネルを使用し、内部抵抗を低減し、出力電流を大幅に増加させました。リチウムマンガン電池は、電動工具などの要求の厳しい現在の機器で使用されています。

リチウムニッケルマンガンコバルト（LiNiMnCoO2、またはNMC）。動作電圧：3.7 V、最大220 W・h / kgのエネルギー消費、最大2000サイクルの寿命。ニッケル、マンガン、コバルトの組み合わせは非常に成功し、電池は所定の電流のエネルギー強度と電力の両方を増加させました。同じ「バンク」では、18650の容量は2800 mAhに増加し、最大出力電流は最大20 Aでした。NMCバッテリーはほとんどの電気自動車に搭載されており、寿命が長いため、リチウムマンガンセルで希釈される場合があります。


メーカーの計算によると、新しいNMCバッテリー電気自動車Nissan Leafの寿命は22年です。以前のLMOバッテリーは容量が少なく、消耗がはるかに速くなりました。出典：日産

リチウム鉄リン酸塩（LiFePO4、またはLFP）。動作電圧：3.3 V、最大120 W・h / kgのエネルギー消費、最大2000サイクルの寿命。 1996年に発見されたこの組成物は、アンペア数を増やし、リチウムイオン電池のライフサイクルを最大2000充電まで延ばすのに役立ちました。リン酸リチウム電池は、以前のものよりも安全であり、過充電に耐えます。それはそれらのエネルギー強度がモバイル機器に適していないだけです-電圧が3.2 Vに上げられると、エネルギー強度はリチウムコバルト組成物に比べて少なくとも半分減少します。しかし、その一方で、LFPは自己放電が少なく、低温に対する特別な耐性があります。


総容量が145.6 kWhのリン酸リチウム電池のアレイ。このようなアレイは、太陽電池パネルからのエネルギーを安全に貯蔵するために使用されます。出典：Yo-Co-Man / Wikimedia

Lithium-Nickel-Cobalt-Aluminum-Oxide（LiNiCoAlO2、またはNCA）。動作電圧：3.6 V、最大消費電力260 W・h / kg、最大500サイクルの寿命。NMCバッテリーと非常によく似ており、公称電圧が3.6 Vのほとんどの機器に適した優れた電力消費量を備えていますが、高コストで適度な寿命（約500充電サイクル）では、NCAバッテリーは競合他社を打ち負かすことができません。これまでのところ、それらは一部の電気自動車でのみ使用されています。

。動作電圧：2.4 V、最大80 W・h / kg、最大7000サイクルの寿命（SCiB：最大15000サイクル）。アノードがチタン酸リチウムナノ結晶で構成される、最も興味深いタイプのリチウムイオン電池の1つ。結晶は、アノードの表面積をグラファイトの3 m2 / gから100 m2 / gに、つまり30倍以上に増やすのに役立ちました！チタン酸リチウム電池は、全容量まで5倍速く充電され、他の電池よりも10倍高い電流を供給します。ただし、チタン酸リチウム電池には、電池の範囲を制限する独自のニュアンスがあります。つまり、低電圧（2.4 V）とエネルギー消費は、他のリチウムイオンバッテリーの2〜3分の1です。つまり、同様の容量を実現するには、チタン酸リチウム電池を数倍に増やす必要があるため、同じスマートフォンに挿入することはできません。

しかし、チタン酸リチウム電池はすぐに輸送に入り、急速充電、加速時の大電流、耐寒性が重要です。たとえば、電気自動車のホンダフィットEV、三菱i-MiEV、モスクワの電気バスなどです。プロジェクトの開始時、モスクワのバスは異なるタイプのバッテリーを使用していたため、ルートの最初の通路の途中でも問題が発生しましたが、東芝のチタン酸リチウムバッテリーを取り付けた後は、電気バスの放電に関するメッセージはありませんでした。東芝のSCiBバッテリーは、アノードにチタンニオブを使用しているため、わずか5分間で最大容量の90％まで回復します。これは、バスが充電ステーションのある最終駅で停車するための許容時間です。 SCiBバッテリーが耐えることができる充電サイクルの数は15,000を超えます。

半世紀以上にわたり、人類は原子力エネルギーをバッテリーに入れることを夢見てきました。バッテリーは何年もの間電力を供給することになります。実際、1953年にベタボルタイックエレメントが発明され、放射性同位元素のベータ崩壊の結果、電子が半導体原子をイオンに変換し、電流を生成しました。そのような電池は、例えばペースメーカーで使用されている。

安定したリチウムイオン電池が発明されてから量産が開始されるまで、10年以上が経過しました。おそらく画期的な電源に関する最新のニュースの1つが予言的になるでしょう、そして2030年代までに私たちはリチウムと電話の毎日の充電の必要性に別れを告げるでしょう。しかし今のところ、ウェアラブル電子機器と電気自動車の分野の進歩を決定づけているのはリチウムイオン電池です。
