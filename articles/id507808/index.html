Cara memintas slider captcha menggunakan JS dan Puppeteer

Teman-teman, saya membawa perhatian Anda pada terjemahan dari artikel “Cara memintas“ slider CAPTCHA ”dengan JS dan Dalang oleh Filip Vitas.



Spamming untuk pemilik situs adalah masalah besar. Di sisi lain, captcha membuatku gila dan memperburuk pengalaman pengguna.



Captcha menyebalkan. Katakan saja. Ada banyak cara untuk lolos verifikasi, dan masing-masing buruk.



Dalam beberapa tahun terakhir, bot menjadi lebih pintar setiap hari, dan melindungi situs dari mereka menjadi sulit. Dengan waktu dan sumber daya yang cukup, Anda dapat berkeliling hampir semua captcha. Dalang memiliki plugin untuk melewati Recaptcha. Beberapa perusahaan menyediakan layanan khusus untuk ini. Misalnya, 2Capcha. Inilah cara menggunakan Puppeteer dan 2Captcha .



Ada situs di mana, sebagai alternatif, captcha dengan slider digunakan ("geser slider untuk verifikasi"). Tetapi mengapa ada orang yang bahkan menggunakan captcha sederhana dengan slider?



Alasannya adalah sebagai berikut:



  • Kebanyakan bot tidak berjalan di JS, jadi Anda bisa menghentikannya.
  • Slider lebih ramah pengguna
  • Untuk pengguna perangkat seluler, “gesek” bilah geser - tindakan biasa


Jadi bagi orang-orang, bilah geser jelas dan sederhana.



Mari kita coba menyiasati beberapa captcha dengan slider.



Gerakkan slider untuk mengirim



jQuery digunakan dalam formulir web di mana ada elemen "pindahkan penggeser untuk mengirim". Ini adalah alternatif untuk captcha yang mencegah spam saat mengisi formulir web.







Pertama kita mengisi kolom input. Untuk memindahkan slider ini, kita perlu:



  • arahkan kursor ke tengah slider
  • klik tombol mouse
  • gerakkan mouse
  • lepaskan tombol mouse






Selesai Itu mudah.



Slider Registrasi Dipbit



Dipbit - situs pertukaran mata uang digital. Kedua halaman - baik login dan registrasi - berisi elemen "geser slider untuk verifikasi".







Dipbit berusaha menjadi sedikit pintar, jadi kita perlu menambahkan kode untuk menyembunyikan operasi Puppeteer.
