Cloudflare PHP APIバインディング

この記事の翻訳は、バックエンドPHP開発者コースの開始前に準備されました。



PHP 7.0以降を使用する開発者向けに、CloudflareはPHP APIバインダーを提供しています。Cloudflare APIの最新の4番目のバージョンをサポートしています。このSDKは、次のようなさまざまな目的に使用できます。

  • CloudflareのDNSエントリの変更を管理および自動化します。
  • プログラムでアカウントにゾーンを追加します。
  • ページルールの変更のバージョン管理と管理
  • ゾーンのロックダウンとUserAgentルールを使用して、IPアドレスとユーザーエージェントを自動的にブロックします。
  • Cloudflare IP範囲を取得してファイアウォールでホワイトリストを自動化する

サポートされている機能


この記事では、Cloudflare PHPバインディングAPIの一般的な使用例をいくつか紹介します。現在、次のエンドポイントと認証方法をサポートしています：

V4エンドポイント

  • DNS
  • IP
  • ページのルール
  • ユーザーエージェントルール
  • ユーザー管理（部分）
  • ゾーンのロックダウン
  • ゾーン

認証

  • ユーザーサービスキー
  • APIキー

取り付け


Cloudflare PHP APIバインディングは、Packagistのcloudflare / sdkで利用できます。これは、composer require cloudflare / sdkを呼び出すことでComposerを使用してインストールできます。



また、ソースコードを直接読み取るか、プロジェクトに貢献したい場合は、ソースコードをcloudflare / cloudflare-phpリポジトリのGitHub 。

仕事の始まり


これは、APIを操作する小さな例です

<?php

require_once('vendor/autoload.php');

$key = new \Cloudflare\API\Auth\APIKey('mjsa@junade.com', 'API KEY GOES HERE');
$adapter = new Cloudflare\API\Adapter\Guzzle($key);
$user = new \Cloudflare\API\Endpoints\User($adapter);

echo 'Your user ID is: ' . $user->getUserID() . PHP_EOL;


この簡単なスクリプトをコマンドラインで実行すると、次の出力が得られます。



ゾーンの一覧表示


次のコードを使用して、アカウントのすべてのゾーンを一覧表示する方法は次のとおりです。

<?php

require_once('vendor/autoload.php');

$key = new \Cloudflare\API\Auth\APIKey('mjsa@junade.com', 'API KEY GOES HERE');
$adapter = new Cloudflare\API\Adapter\Guzzle($key);

$zones = new \Cloudflare\API\Endpoints\Zones($adapter);
foreach ($zones->listZones()->result as $zone) {
   echo $zone->name.' ('.$zone->plan->name.')'.PHP_EOL;
}

このスクリプトをコマンドラインで実行すると、次の出力が得られます。



すべてのWebサイトのキャッシュをクリアする


次に、Purge Cacheエンドポイントを使用して、アカウントの各Webサイトのキャッシュを完全にクリアする別の例を示します（cachePurgeEverythingの代わりにcachePurgeメソッドを使用すると、キャッシュを使用する個々のファイルをクリアできます）。

<?php

require_once('vendor/autoload.php');

$key = new \Cloudflare\API\Auth\APIKey('mjsa@junade.com', 'API KEY GOES HERE');
$adapter = new Cloudflare\API\Adapter\Guzzle($key);

$zones = new \Cloudflare\API\Endpoints\Zones($adapter);
foreach ($zones->listZones()->result as $zone) {
   echo "Cache purge for " . $zone->name . ": ";
   echo $zones->cachePurgeEverything($zone->id) == true ? "successful" : "failed";
   echo PHP_EOL;

}

このスクリプトをコマンドラインで実行すると、次の出力が表示されます。



ページルールの作成


SDKを使用して、プログラムでページルールをCloudflareゾーンに追加することもできます。次に、キャッシュバイパスルールを追加する簡単な例を示します。

<?php

require_once('vendor/autoload.php');

$key = new \Cloudflare\API\Auth\APIKey('mjsa@junade.com', 'API KEY GOES HERE');
$adapter = new Cloudflare\API\Adapter\Guzzle($key);
$zones = new \Cloudflare\API\Endpoints\Zones($adapter);

$zoneID = $zones->getZoneID("junade.com");

$pageRulesTarget = new \Cloudflare\API\Configurations\PageRulesTargets('https://junade.com/noCache/*');

$pageRulesConfig = new \Cloudflare\API\Configurations\PageRulesActions();
$pageRulesConfig->setCacheLevel('bypass');

$pageRules = new \Cloudflare\API\Endpoints\PageRules($adapter);
$pageRules->createPageRule($zoneID, $pageRulesTarget, $pageRulesConfig, true, 6);

ZonesエンドポイントクラスのgetZoneIDメソッドを使用して、目的のゾーンのIDを簡単に取得できます。このヘルパーメソッドは、ゾーンの名前でIDを返します。

SDKは依存関係注入を使用して、ページルールの目的と構成を指定することに注意してください。これが、PageRulesTargetsクラスとPageRuleActionsクラスのインスタンスをcreatePageRuleメソッドに渡す必要がある理由です。

DNS


SDKを使用して、プログラムでレコードを追加することもできます。DNSレコードを追加する例を次に示します。

<?php

require_once('vendor/autoload.php');

$key = new \Cloudflare\API\Auth\APIKey('mjsa@junade.com', 'API KEY GOES HERE');
$adapter = new Cloudflare\API\Adapter\Guzzle($key);
$zones = new \Cloudflare\API\Endpoints\Zones($adapter);

$zoneID = $zones->getZoneID("junade.com");

$dns = new \Cloudflare\API\Endpoints\DNS($adapter);
if ($dns->addRecord($zoneID, "A", 'example', '8.8.8.8', 0, true) === true) {
   echo "DNS record created.". PHP_EOL;
}

さらに、このSDKを使用してDNSレコードの詳細を削除、一覧表示、表示することもできます。たとえば、ゾーン内の各DNSレコードのタイプと名前をリストする簡単なスクリプトを作成してみましょう。

<?php

require_once('vendor/autoload.php');

$key = new \Cloudflare\API\Auth\APIKey('mjsa@junade.com', 'API KEY GOES HERE');
$adapter = new Cloudflare\API\Adapter\Guzzle($key);
$zones = new \Cloudflare\API\Endpoints\Zones($adapter);

$zoneID = $zones->getZoneID("icyapril.com");

$dns = new \Cloudflare\API\Endpoints\DNS($adapter);
foreach ($dns->listRecords($zoneID)->result as $record) {
   echo $record->type." ".$record->name.PHP_EOL;
}

これは、自分のゾーンの1つに対してこのスクリプトを実行したときの出力です。
