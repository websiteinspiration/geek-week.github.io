Microsoft tidak akan lagi mengeluarkan sertifikat MCSE / MCSD / MCSA

Dalam lima tahun, di bawah kepemimpinan Satya Nadella yang bijaksana, Microsoft telah kehilangan kepemimpinan teknologi global. Terlepas dari kenyataan bahwa semua upaya dilemparkan ke arah cloud, Azure tidak dapat mengejar ketinggalan dan bahkan lebih memotong popularitas platform dari Amazon, pelopor dan pemimpin permanen pasar ini. Tidak ingin menarik pembiayaan luar untuk pengembangan Azure, perusahaan memompa uang dari arah lain, sebagai akibatnya Microsoft kehilangan tablet Apple dan desktop profesional, teknologi seluler Google, game dan AR / VR untuk semua orang secara berturut-turut, sistem korporat Salesforce tidak dapat membawa iklan mereka jaringan Bing dan mesin pencari, tetapi orang-orang baik dengan wanita bahkan tidak ingat tentang jejaring sosial Yammer. Menjadi tidak terorganisir, dimakan secara internal oleh kompetisi internal dan tidak mampu melakukan inovasi,baru-baru ini, dia hanya membeli perkembangan orang lain: GitHub, LinkedIn, Xamarin dan proyek lainnya yang sudah selesai dan matang.

Intinya, Microsoft bergerak menuju menjadi pembuat editor teks utama dan Linux hoster. Tentu saja, perusahaan seperti itu tidak memerlukan mitra dan pengembang, oleh karena itu, pada bulan Februari tahun ini, diumumkan bahwa mereka menutup program sertifikasi untuk pengembang dan administrator insinyur , yaitu:

  • MCSA: Pelaporan BI
  • MCSA: Dynamics 365 untuk Operasi
  • MCSA: Pengembangan BI SQL 2016
  • MCSA: Admin Database SQL 2016
  • MCSA: SQL 2016 Database Dev
  • MCSA: SQL Server 2012/2014
  • MCSA: Platform Windows Universal
  • MCSA: Aplikasi Web
  • MCSA: Windows Server 2012
  • MCSA: Windows Server 2016
  • MCSD: Pembuat Aplikasi
  • MCSE: Aplikasi Bisnis
  • MCSE: Infrastruktur Inti
  • MCSE: Manajemen Data & Analisis
  • MCSE: Produktivitas

Seperti yang dapat Anda lihat dari daftar, Microsoft percaya bahwa di dunia modern mereka tidak diperlukan: pengembang web, pengembang dan administrator database, pengembang aplikasi desktop dan tablet, administrator server di tempat. Seharusnya menghentikan sertifikasi untuk program-program ini sampai 30 Juni.

Rangkaian ujian baru terutama melibatkan spesialis pelatihan dalam menyiapkan kartu kredit perusahaan untuk membayar layanan Azure, mengklik mouse untuk mendaftar di Azure Active Directory dan mempromosikan editor teks yang sama sebagai bagian dari Office 365:



Tidak akan ada kebahagiaan, tetapi kemalangan membantu: sehubungan dengan pandemi COVID-19, Microsoft Learning Memperpanjang Batas Waktu untuk Menerima Aplikasi Sertifikasi Terbaruprogram untuk spesialis nyata hingga 31 Januari 2021, dan juga berjanji untuk meningkatkan jumlah ujian yang diambil dari jarak jauh.

Jika Anda masih belum memutakhirkan MCSA Anda, sekarang Anda tahu apa yang harus dilakukan selama karantina.
