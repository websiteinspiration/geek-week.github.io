実験室実験のシミュレーションモデルの実装例


中学生の生徒と上級生徒のための熱力学のコースからの実験室実験「リアルガスと相転移点」を検討してください。この実験の目的は、実際のガス（六フッ化硫黄SF6）を定量化し、その相転移点を決定することです。実験室での経験は、3B Scientificによって製造された実験室のセットアップ（図1）で実行されます[2]。


図1-実際のガスの相転移点を決定するための実験室のセットアップの自然な類似体の外観

提示された実験施設には、ガスタンク、タンク内のガス圧力を測定するためのゲージ、ねじ機構とスケールを備えた可動ピストン、バルブシステム、およびガス温度を測定するための電子体温計が装備されています。フライホイールを回転させると、ピストンが動き、ガスリザーバーの自由体積が変わります。ガスタンクの作動部分は半透明の素材でできており、実験中の被験物質の状態を目視で確認することができます。

実際のガスの相転移点は、臨界温度、臨界圧力、臨界密度によって特徴付けられます。臨界温度以下では、物質は大容量では気体状態になり、小容量では液体状態になります。中間の体積では、それは液体と気体の混合物として存在する可能性があり、等温条件下で体積が変化すると、凝集の状態が変化します。気体の割合は体積の増加に伴って増加し、混合物の圧力は一定のままです。液体と蒸気は密度が異なるため、重力場で分離されます。温度が上昇すると、液体密度は減少し、ガス密度は、これらの2つの密度が同じ臨界密度値と等しくなるまで増加します。臨界温度を超えると、ガスは液体状態に変換できません。ただし、等温条件下では、ガスの温度が臨界温度を大幅に超えるまで、ガスは理想ガスの法則に従いません。

ガスの状態は、圧力（P）、体積（V）、温度（T）のパラメーターに応じて、Clapeyron状態図を使用して表されます。実験の数学的な基礎として、ファンデルワールス気体モデルが使用されます。

Pr（Vr）=8Tr/（３Vr-1）-３/Vr2


ここで、Prは、実際の圧力と臨界点の圧力の比（Pr = P / Pc）に等しい減少したガス圧力です。 Vrは、臨界点での体積に対する実際の体積の比率に等しい減少したガス体積です（Vr = V / Vc）。 Trは、実際の温度と臨界点の温度の比（Tr = T / Tc）に等しい還元ガス温度です。

ファンデルワールスの式を使用すると、所定の形式のテストガスの等温線（等温線）を計算できます（図2）。

図2-気液平衡を除く六フッ化硫黄SF6のクラペイロン相図

図2の図からわかるように、臨界温度（Tc）未満のガス温度では、等温線のdPr / dVr> 0の部分のファンデルワールス方程式が振動特性を持っています。これらの領域では、物質は気体と液体の混合物の形をしており、実際には等圧経路に沿って気体の凝縮が発生します（一定の圧力で体積が変化します）。したがって、いわゆる双曲線（図の黒い曲線）内のファンデルワールス図[3]は、実際のガスの挙動を不適切に説明しています。

この問題を解決するために、等しい面積のマクスウェル法が使用されました[4]の本質は、等温線の振動部分を水平線（等圧線）で置き換えることであり、等温線で示される領域を2つの等しい大きさの領域に分割します（図3）。

図3 -Maxwellの等しい面積の方法へ

以下は、記述された問題を解決するためのプログラムコード（Matlab）の一部です[4]。

function maxwell_equal_area_2
clc; clear all; close all
%  Tr  Tc:  Tr = T/Tc
Tr = 0.9; 
%    Pr (  --)
Prfh = @(Vr) 8/3*Tr./(Vr - 1/3) - 3./(Vr.^2);
%   
Vr = linspace(0.5,4,100);   
%   
Pr = Prfh(Vr);

関数の最終的な形：
function Z = equalArea(y)
Tr = 0.9;
vdWp = [1 -1/3*(1+8*Tr/y) 3/y -1/y];
