.netオンラインストアのラベルと値札の印刷

なぜオンラインストアの場合、あなたは尋ねますか？もちろん、オンラインストアについては、私だけではありません。通常、大規模な商社の取引はソフトウェアを自社で作成するのではなく、ソフトウェアを側で注文します。そして、それがどんなに奇妙に見えても、手が値札に届かないことがよくあります、そしてこの文書は貿易において重要な役割を果たします。値札の完成が150ユーロだった時代を覚えていますが、この作業は簡単なことではなかったようです。記事の最後に、作業草案 c＃、＃core、＃.net #MVCが添付されています。トレーディングシステムを試してみてください。前向きなフィードバックをありがとうございます。

そこで、オンラインストア（通常のストア）のオーナーは、退屈な白黒の値札を変更することにしました。そして、私たちの品揃えには何があるのですか？



私の以前の記事、クレームの海はxmlデータソースを上げました。熟練したプログラマーが考えられる例を他のソースに簡単に適応できることは私には明らかであるように見えましたが、コメントが示すように、これは間違っていることが判明しました。したがって、データセットに少し注意を払います。ユニバーサルコンポーネントのDataSetを使用します。読者が同様の方法でデータを入力したり、別のソースに置き換えたりできることを願っています

コントローラを検討する

public class HomeController : MyController 
    { 
        private IHostingEnvironment _env; 
  
        public HomeController(IHostingEnvironment env) 
        { 
            _env = env; 
        } 
  
        public IActionResult Index() 
        { 
            ViewBag.Items = InitData().Tables["Items"].Rows; 
            return View(); 
        } 
  
        public IActionResult PrintTags() 
        { 
            WebReport webReport = new WebReport(); 
            webReport.Width = "100%"; 
  
            DataSet d = InitData(); 
  
            webReport.Report.Load(System.IO.Path.Combine(_env.WebRootPath + "/reports", "tags.frx")); 
            webReport.Report.RegisterData(d, "ItemsDataSet"); 
            webReport.Report.GetDataSource("Items").Enabled = true; 
            (webReport.Report.FindObject("Data1") as DataBand).DataSource = webReport.Report.GetDataSource("Items"); 
            webReport.Report.Prepare(); 
  
            ViewBag.WebReport = webReport; 
  
            return View(); 
        } 
    }

これは、データセットが初期化されるMyControllerクラスの子孫です。これは、メインコントローラのコードに過負荷をかけないようにするためだけに行われます。索引ページにストアフロントが表示されます。 PrintTagsページには、価格タグが表示されます。

したがって、値札への出力用のデータが準備された後、FastReportデモバージョンをロードして、値札の



デザインを開発します。デザインは完全に自発的であり、バーコードであれ写真であれ、タスクのコンポーネントを入力できます。テンプレートファイルは、プロジェクトのwwwroot / reports / tags.frxにあります。修正するか、印刷が表示される場所に応じて独自の値札またはラベルを作成できます。おそらくこれは粘着テープであり、熱ラベルに印刷されます。

プロジェクトを作成するときは、必ず





Setup.cs addにパッケージリンクを追加してください。

app.UseFastReport();

コントローラ内 

using FastReport; 
using FastReport.Web;

これはすべてプロジェクト で確認できます。

値札の印刷は、[タグの印刷]ボタンから開始します。
