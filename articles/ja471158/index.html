ID @ Xbox in Russia：ゲームの視野を広げる



最近では、独立系ゲームデベロッパーID @ Xboxのプログラムの一環としてモスクワでイベントが開催されました。参加者は、Felix the Reaper、King's Bounty 2など14の新しいインディーゲームを紹介し、ID @ XboxのヨーロッパのリーダーであるAgostino Simonettaとチャットしました。プログラムの存在の5年間で、そのようなイベントは最初にロシアで開催されました。

平均的なゲーマーは最も有名なゲームをプレイする可能性が高く、すぐに何か珍しいものを呼ぶ可能性は低いです。一方、ゲーム業界では、AAAシリーズのゲームをリリースする大規模なスタジオだけではありません。興味深い運命と常に珍しい製品を持つ独立した開発者は非常に多くいます。誰かがドリフトに恋をしているが、彼が自分のレースを書いている車にお金がなくて、誰かはすでにデザイナーとして働いているが、彼の余暇に彼自身の何かを作成している。そして、ゲームが作成された後も、それを宣伝して幅広い聴衆に届ける必要があります。しかし、どうやって？

イベントの様子



Agostino Simonetta

ID @ Xboxプログラムでは、すべての視野を広げることができます。新しい開発者を業界に引き付け、ゲームクリエイターがファンを見つけられるように支援し、ゲーマーが新しいジャンルを試すのを支援します。たとえば、ゲームがゲームパスに該当する場合、より多くのゲーマーに連絡する機会があります。会議では、Agostino Simonettaが珍しいパズルの歴史の例を紹介しました。Human：Fall Flat：そのアクティブなファンの40％は、これまでにこのようなものをプレイしたことがありません。そして、聴衆が広いほど、開発者の財務見通しは良くなります。 XboxのプログラムIDに基づくゲームのデジタルバージョンからの総収入は、14億ドルを超えました。



今日、ID @ Xboxは、成功を最大化するために独立したスタジオが必要とするツールとサポートを提供しています。このプログラムへの参加は、あらゆる規模のゲーム開発者に彼らの創造的な可能性を明らかにする機会を提供します。 ID @ Xboxを使用すると、Xbox Liveを使用してXbox OneおよびWindows 10でデジタルゲームを個別に公開したり、iOSまたはAndroidのゲームにXbox Liveを追加したりできます。



ID @ Xboxプログラムは、最初の5年間で67か国の3,400を超えるスタジオをカバーしており、そのうち950はロシアが属するEMEA地域（ヨーロッパ、中東、アフリカ）にあります。このプログラムは、ゲームパスの125を含む1300ゲームをリリースしました。別の1800ゲームが活発に開発中です。



最初のイベント後も、マイクロソフトはロシアでID @ Xboxプログラムを積極的に実装し、独立系開発者の国内コミュニティの発展に貢献していきます。



ID @ Xboxプログラムの詳細


ID @ Xbox プログラムを使用すると、有資格の開発者は、Xbox LiveでXbox OneおよびWindows 10のデジタルゲームを個別に公開し、iOSまたはAndroidのゲームにXbox Liveサポートを追加し、スタジオにツールを提供して最大化を支援することで、クリエイティブな可能性を解き放つことができます。彼らの成功。

Xbox LiveがiOSおよびAndroid開発者向けに利用可能に




新しいXbox Live SDKは、iOSおよびAndroid開発者がXbox Liveの機能にアクセスできるようにし、楽しく魅力的なゲームの作成に集中できるようにします。開発者は、Xbox Liveの実績、ゲームのスコア、および基本的な統計にアクセスできます。これにより、クリエイティブな機能が拡張され、ゲーム体験が向上します。柔軟なカスタマイズツールにより、開発者は要件に最適なサービスのみを使用できます。
