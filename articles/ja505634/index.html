WSL（Ubuntu）のプロキシ設定

この記事では、プロキシの下からWSLで正しく機能するための設定について説明します。


  • apt-get
  • カール
  • wget
  • ギット
  • npm

apt-get


注：以下では、次のプロキシデータが使用されます。


  • ホスト： http://proxy.example.com
  • ポート： 7777
  • ログインする： user@example.com
  • パスワード： somePassword

正しく機能させるには、ファイルに行を追加するapt-get必要があり/etc/apt/apt.conf.d/proxy.confます。Acquire::http::Proxy "http://user@example.com:somePassword@proxy.example.com:7777";
これを行うには、次のコマンドを実行します。


sudo cat <<<EOF >>/etc/apt/apt.conf.d/proxy.conf
Acquire::http::Proxy "http://user@example.com:somePassword@proxy.example.com:7777";`
EOF

設定が正しいことを確認します。


cat /etc/apt/apt.conf.d/proxy.conf

結果には次のものが含まれているはずです。


Acquire::http::Proxy "http://user@example.com:somePassword@proxy.example.com:7777";
