適応フィルター

画像変換の実装には、ニューラルネットワークを使用できます。ニューラルネットワークの主な利点は、自動学習機能です。この場合、ニューラルネットワークに基づく適応フィルター[Shovin 2018]をトレーニングして、元の画像がサンプルで、2番目の画像が既存のフィルターまたは変換によってサンプルから変更された画像のペアを使用することを提案します。適応フィルターは、この変換を復元する必要があります。これには、dx x dyのサイズのウィンドウと3つのカラーチャネルが使用されます。その結果、ニューラルネットワークはdx • dyを使用します。•入力ニューロンの3つの入力信号。ネットワークは、フィルター変換のカラーファクターに関連付けられたニューロン数の非表示レイヤーによって補足できます。ニューロンの出力で3つのニューロンを使用することを提案します。ニューロンの出力信号は、出力画像からのウィンドウの中央ピクセルで3つの色（rgb-赤、緑、青）にマップされます。ピクセルのカラーチャネルの信号は線形に値の範囲に変換されます[ -0.5 ; 0.5 ]。一方、値の間隔をもつ反対称シグモイド関数[ -1 ; 1]。画像の境界ピクセルの場合、ウィンドウが画像の境界を超えると、そのようなピクセルに対応するネットワークニューロンの入力値は0に設定されます。ニューラルネットワークは、誤差の逆伝播の方法により、出力画像のすべてのピクセルについてウィンドウでトレーニングされます。
