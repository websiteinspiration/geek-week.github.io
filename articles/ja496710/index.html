コンピュータ情報：シンプルで高速



多くの場合、追加のソフトウェアや「カバーを開ける」ことなく、据え置き型のコンピューターまたはラップトップに関する情報を情報に基づいてすばやく簡単に取得する必要があります。

これは、たとえば、WindowsまたはPowerShellのコマンドラインツールを使用して実装できます。
CMD-Windowsのすべてのバージョンにある、実績のある機能。
さらに、単純な管理タスクの場合、cmdがより一般的で、どこかでより便利です。

どちらが良いですか-CMDまたはPowerShell？私はこの質問に明確に答える準備ができていません。
ただし、両方を使用することを妨げるものは何もありません。すべてタスクに依存します。

PCに関するすべての情報を収集するわけではありません。このため、多くの専用ソフトウェアがあります。


CMDを使用した実装。


Windows環境の変数とWMIスクリプトの実行を使用して情報を収集します。

Windowsのすべての環境変数とその値を表示するには、setコマンドを使用します。

WMICコマンドを使用して、ハードウェアとシステムに関する情報を取得し、プロセスとそのコンポーネントを管理し、Windows Management Instrumentation（Windows Management InstrumentationまたはWMI）の機能を使用して設定を変更します。

次のコマンドで、wmic.exeユーティリティの操作に関するヒントを得ることができます。
  • wmic /？-一般的なヘルプを表示します。
  • wmic /？：Brief-簡単なヘルプを表示します。
  • wmic /？：FULL-完全なヘルプを表示します。


我々は使用するだろう：
  • BASEBOARD（マザーボード管理）;
  • COMPUTERSYSTEM（コンピューター制御）;
  • CPU（CPU制御）;
  • DISKDRIVE（物理ディスク管理）;
  • MEMORYCHIP（メモリチップ情報）。


スクリプトには、FORを使用した多くのループが含まれています。
FOR / Fの特徴的な機能は、トークンを処理する機能と、追加のキーワードのサポートです。
  • スキップ（ファイルの先頭から特定の数の処理された行をスキップします）;
  • delims（別の区切り文字（s）、デフォルト、スペース、タブを設定）
  • トークン（ループ本体で受信されたトークン（サブストリング）の数、および区切り文字による分割の制限）。また、最初のループ変数に該当する特定のトークン番号を設定することもできます。
  • usebackq（IN（...）内での引用符の使用に関するルールの変更）。


以下はスクリプトです。
@echo off
:    
set fname=pcinfo.txt
: 
Echo pcname: %computername% >>%fname%
:IP-    
FOR /F "usebackq tokens=2 delims=[]" %%i IN (`ping %Computername% -n 1 -4`) DO if not "%%i"=="" Set ip=%%i
Echo IP %computername%: %ip% >>%fname%
:  
Echo username: %username%  >>%fname%
: 
set cmd=wmic computersystem get model
for /f "skip=1 delims=" %%Z in ('%cmd%') do (
    set _pn=%%Z
	GOTO BREAK1
)
:BREAK1
echo CS Model: %_pn% >>%fname%
:
SETLOCAL ENABLEDELAYEDEXPANSION
set mmr=0
for /f "skip=1 delims=" %%i in ('wmic cpu get name') do (
for /f "tokens=1-2 delims=" %%A in ("%%i") do (
set CPULbl=%%A
set /a mmr=!mmr!+1
echo CPU !mmr!: !CPULbl! >>%fname%
))
: 
set cmd=wmic baseboard get product
for /f "skip=1 delims=" %%Z in ('%cmd%') do (
    set _mb=%%Z
    GOTO BREAK2
)
:BREAK2
echo MB: %_mb% >>%fname%
: 
SETLOCAL ENABLEDELAYEDEXPANSION
set mmr=0
for /f "skip=1 delims=" %%i in ('WMIC MemoryChip get BankLabel^,DeviceLocator^,PartNumber^,Speed^,Capacity') do (
for /f "tokens=1-5 delims=" %%A in ("%%i") do (
set BnkLbl=%%A
set /a mmr=!mmr!+1
echo Memory !mmr!: !BnkLbl! >>%fname%
wmic MEMORYCHIP get banklabel, partnumber, capacity, speed,  manufacturer
))
:
SETLOCAL ENABLEDELAYEDEXPANSION
set mmr=0
for /f "skip=1 delims=" %%i in ('wmic diskdrive get model^,size') do (
for /f "tokens=1-2 delims=" %%A in ("%%i") do (
set HDDLbl=%%A
set /a mmr=!mmr!+1
echo DISK !mmr!: !HDDLbl! >>%fname%
))


PowerShellによる実装。


PowerShellでは、スクリプトを実行する前に、現在のシェルセッションで署名されていないスクリプトを実行できるコマンドを実行する必要があります。
Set-ExecutionPolicy RemoteSigned -Scope Process

情報は主にGet-WmiObject -Class win32を使用して収集されます。ループを操作することで、すべてが簡単です。

我々は使用するだろう：
  • Get-WmiObject -Class win32_processor;
  • Get-WmiObject -Class win32_baseboard;
  • Get-WmiObject Win32_PhysicalMemory;
  • Get-PhysicalDisk;
  • Get-WmiObject -Class Win32_ComputerSystem;
  • Get-WmiObject Win32_NetworkAdapter;
  • Win32_NetworkAdapterConfiguration。


アクティブなネットワークカードのIPアドレスを決定する際に、多少の困難があります。これを行うには、Win32_NetworkAdapterクラスからMACアドレス情報を取得し、Win32_NetworkAdapterConfigurationクラスからIPアドレスを取得して、それを1つの要求に結合した後、NetConnectionStatusフィルター-2を使用します。
PS C:\Users\admin> Get-WmiObject Win32_NetworkAdapter -Filter 'NetConnectionStatus=2'
ServiceName      : Qcamain10x64
MACAddress       : 58:00:E3:7D:87:3F
AdapterType      : Ethernet 802.3
DeviceID         : 1
Name             : Qualcomm Atheros QCA61x4A Wireless Network Adapter
NetworkAddresses :
Speed            : 144400000

アクティブなネットワークカードのMACAddressによってネットワークパラメータを取得するには、Win32_NetworkAdapterConfigurationを追加で読み取ります。

以下はスクリプトです。
#    
$fname = "pcinfo.txt"
$CPU = Get-WmiObject  -Class win32_processor
$MB = Get-WmiObject  -Class win32_baseboard
$MEM = Get-WmiObject Win32_PhysicalMemory
$DD = Get-PhysicalDisk
$pcn = Get-WmiObject -Class Win32_ComputerSystem

# 
"pcname:	"+$pcn.Name | Out-File -FilePath $fname -Append -Encoding Default
#IP-    
Get-WmiObject Win32_NetworkAdapter -Filter 'NetConnectionStatus=2' |
ForEach-Object {
$pcip = 1 | Select-Object IP
$config = $_.GetRelated('Win32_NetworkAdapterConfiguration') 
$pcip.IP = $config | Select-Object -expand IPAddress
$pcip
}
foreach($aip in $pcip) {
"IP:	"+$aip.IP | Out-File -FilePath $fname -Append -Encoding Default
}
#  
"username:	"+$pcn.PrimaryOwnerName | Out-File -FilePath $fname -Append -Encoding Default
# 
"CS Model:	"+$pcn.Model | Out-File -FilePath $fname -Append -Encoding Default
#
$num = 0
foreach($processor in $CPU) {
$num = $num+1
"CPU "+$num+":	"+$processor.Name | Out-File -FilePath $fname -Append -Encoding Default
}
# 
"MB:	"+$MB.Product | Out-File -FilePath $fname -Append -Encoding Default
# 
$num = 0
foreach($memory in $MEM) {
$num = $num+1
"MEMORY "+$num+":	"+$memory.PartNumber+"	"+$memory.Capacity+"	"+$memory.Speed | Out-File -FilePath $fname -Append -Encoding Default
}
#
$num = 0
foreach($disk in $DD) {
$num = $num+1
"DISK "+$num+":	"+$disk.FriendlyName+"	"+$disk.Size+"	"+$disk.MediaType | Out-File -FilePath $fname -Append -Encoding Default
}
