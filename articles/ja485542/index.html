Notionにグラフィックを追加する

多くの人がNotionのグラフィックを欠いています。だから私はそれらを生成するために自動のものを提出することにしました。

パート1.問題の説明


問題は、実際、Notionにはグラフがなく、タブレットからの情報を視覚化することはできないということです（ただし、望んでいます）。したがって、次のようなものを作成する必要があります。

  1. グラフが潜在的に存在する可能性があるページのリストを取得します
  2. これらのページからスケジュールの説明を収集します。+ページリーダーは理解しやすいため、説明はページに表示する必要があります。コードを頻繁に修正して修正しないようにしてください。
  3. 説明の直後にグラフを追加し、以前のバージョンのグラフを削除します。
  4. これを自動的に（1時間に1回）、できれば無料で行います。

パート4.コードの記述


Notionに移動し、Ctrl + Shift + Jを押して、Application-> Cookiesに移動し、token_v2をコピーしてTOKENと呼びます。これは、ライブラリが何らかの形でNotion APIと対話できるようにするために必要です。

グラフの説明が見つかる可能性のあるページのリストを何らかの形で保存する必要があります。このビジネスをシンプルに保ちます。

PAGES = [
    "https://www.notion.so/mixedself/Dashboard-40a3156030fd4d9cb1935993e1f2c7eb"
]

説明自体を何らかの方法で解析するには、次のキーワードが必要です。

  1. データがX軸上にあるフィールド
  2. データがY軸上にあるフィールド
  3. プレート上のURL

次のようになります。

BASE_KEY = "Base:"
X_AXIS_KEY = "X axis:"
Y_AXIS_KEY = "Y axis:"

つまり、グラフの空の説明は次のようになります。

def get_empty_object():
    return {
        "database": "",
        "x": "",
        "y": ""
    }

また、説明が空かどうかを確認する必要もあります。このための特別な機能を提出します。すべてのフィールドが空ではない場合、オブジェクトは完成しており、グラフの描画を開始できます。

def is_not_empty(thing):
    return thing != ""


def check_for_completeness(object):
    return is_not_empty(object["database"]) and is_not_empty(object["x"]) and is_not_empty(object["y"])

説明を生成するには、データ（実際は単なるテキストです）を何とかしてクリアする必要があります。そのための関数をいくつか作成しましょう。簡単な説明：Notionは太字（カットの上の図のように）をここに格納します

def br_text(text):
    return "__" + text + "__"


def clear_text(text):
    return text.replace(br_text(BASE_KEY), "").replace(BASE_KEY, "") \
        .replace(br_text(X_AXIS_KEY), "").replace(X_AXIS_KEY, "") \
        .replace(br_text(Y_AXIS_KEY), "").replace(Y_AXIS_KEY, "").strip()

さて、おそらく、私たちの小さなことの主な機能を書いてみましょう。コードの下には、ここで何が行われているのかの説明があります。

def plot():
    client = NotionClient(token_v2=TOKEN)

    for page in PAGES:
        blocks = client.get_block(page)
        thing = get_empty_object()

        for i in range(len(blocks.children)):
            block = blocks.children[i]
            print(block.type)

            if block.type != "image":
                title = block.title

                if BASE_KEY in title:
                    thing["database"] = clear_text(title).split("](")[0].replace("[", "")

                elif X_AXIS_KEY in title:
                    thing["x"] = clear_text(title)

                elif Y_AXIS_KEY in title:
                    thing["y"] = clear_text(title)

                    if check_for_completeness(thing):
                        # not last block
                        if i != len(blocks.children) - 1:
                            next_block = blocks.children[i + 1]

                            # if next block is picture, then it is previous
                            # version of the plot, then we should remove it
                            if blocks.children[i + 1].type == "image":
                                next_block.remove()

                        draw_plot(client, thing, block, blocks)
                        thing = get_empty_object()

次に、プレートからデータを収集する関数を記述します。

def get_lines_array(thing, client):
    database = client.get_collection_view(thing["database"])
    rows = database.default_query().execute()
    lines_array = []

    for i in range(1, len(rows)):
        previous_row = rows[i - 1]
        current_row = rows[i]
        line = [(get_point_from_row(thing, previous_row)), (get_point_from_row(thing, current_row))]
        lines_array.append(line)

    return lines_array

ここでベースを取得し、そのすべてのラインを取得して、すべてのラインを通過し、ポイントからポイントへのラインのセットを形成します。

get_point_from_rowとは何ですか？実際、オブジェクトが日付の場合（多くの場合、X軸に日付を表​​示する必要があるだけです）、表示することはできず、さらに処理する必要があります。

def get_point_from_row(thing, row):
    x_property = row.get_property(thing["x"])
    y_property = row.get_property(thing["y"])

    if thing["x"] == "date":
        x_property = x_property.start

    if thing["y"] == "date":
        y_property = y_property.start

    return x_property, y_property

次のようなランダムなファイル名を取得できます。

def random_string(string_length=10):
    letters = string.ascii_lowercase
    return ''.join(random.choice(letters) for i in range(string_length))

また、matplotlibは現在実装されている方法とは異なるデータ表現を受け取るため、ポイントを再解析する必要があります。

def reparse_points(points):
    return [
        [points[0][0], points[1][0]],
        [points[0][1], points[1][1]],
    ]

よく見ると、X軸に沿ったデータが日付であるかどうかがメソッドによってチェックされています。日付である場合は、正しく表示する必要があります。

def x_axis_dates(ax=None, fig=None):
    if ax is None:
        ax = plt.gca()

    if fig is None:
        fig = plt.gcf()

    loc = mdates.AutoDateLocator()
    fmt = mdates.AutoDateFormatter(loc)

    ax.xaxis.set_major_locator(loc)
    ax.xaxis.set_major_formatter(fmt)

    fig.autofmt_xdate()
