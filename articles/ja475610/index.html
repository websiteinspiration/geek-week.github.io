視覚を保持する安全なLEDテーブルランプ



現代の技術の発展により、スペクトルの異なる新しいカラーソースが登場しています。以前は、白熱フィラメントの光源が使用されていたため、すでに光のスペクトルが不均一な蛍光灯が普及していました。それぞれの種には長所と短所がありました。高品質の照明を提供し、欠陥のないLEDテーブルランプをテストします。

宣言された仕様




フローラとオリオンのサンプルは、ドイツの製造業者レミリヒトGmBHの代表であるLLC RomatiのRemez商標で表されます。彼らは韓国の太陽光に似たLEDを使用しています。これは、スペクトルの自然な太陽光にできるだけ近く、視界に対して安全で、ホルモンの背景に違反せず、高い演色評価数を持つ照明を提供します。

力 10W 8W

外形寸法 441 x 486 x 200 mm。 465 x 480 x 130 mm。

電力は外部電源から供給されるため、LED卓上ランプのサイズを最小限に抑え、高電圧部分をテーブルから離すことができます。子供が塗料で塗装するときに水をこぼした場合、濡れた表面に危険な高電圧はありません。

オリオンランプデザイン




Orion LEDランプには4つの可動要素があるため、多くの位置設定があります。LEDが配置されている要素の傾斜角度も変化します。これにより、どの位置でも必要な角度で十分な照明が得られます。後壁には、5ボルト1アンペアの充電器用のUSBソケットがあります。









フローラデザイン




フローラテーブルランプは、文房具を置くことができる植木鉢の形をしたベースを持っています。LEDマウントの大きな緑色のステムは、完全に柔軟で移動可能です。小さな半透明のシートがナイトランプとして機能します。シートの底面には、シートを内側から照らすLEDがあります。









イルミネーション




パラメータが安定するまでサンプルを1時間加熱します。デスクトップの照度は、光度計を使用して最小および最大の明るさで測定します。
基準によれば、職場の照明レベルは少なくとも300ルクスである必要があります。この指標は、マイナーな作品、読書、描画、その他のタイプのパフォーマンスに関連しています。
レベル オリオン フローラ
ミニマムスイート 610lx 520lx
最大、ルクス 1320lx 970lx

測定結果は、それぞれ最大1320ルクスと970ルクスの最大照度を示しました。これは、正確な作業には十分です。

従来の照明器具とは異なり




これらのLEDテーブルランプのモデルは、従来のランプにはないさまざまな光源の利点を組み合わせています。

  1. 白熱灯のように、高い演色評価数CRI> 95。
  2. 蛍光灯のように8〜10 Wの低消費電力。
  3. 白熱から発光まで、色温度の滑らかな変化。
  4. LED照明光源のようなコンパクトな寸法。
  5. デザインは、デザインの多様性と組み合わされています。

スペクトラム




従来のLEDと蛍光灯のスペクトルの研究は、人間の視覚と状態への影響を明らかにしました。通常のLEDでは、450 nmの波長に青色が多く含まれているため、瞳孔を正常に狭めることができません。瞳孔は多くの青色光を通過させ、視力の低下を加速させます。大量の青色光は、免疫システムを弱め、他のプロセスに影響を与える可能性がある睡眠ホルモンメラトニンの生成を減らします。



OrionとFloraのテーブルランプは、新しい韓国のSunlike LEDを使用しています。これらのLEDは、スペクトルが均一で、青色光の大きなピークがありません。光は、スペクトルの自然太陽光に可能な限り近く得られます。



多くのスマートフォンにはすでにナイトモードと読書モードがあり、外部照明がないときに使用され、スマートフォンの画面が唯一の光源になります。これらのモードでは、青色スペクトルの放射が減少し、画像の色相がスペクトルの暖かい部分にシフトします。

リップルファクター


従来の照明器具では、明るさを変えると100Hzの周波数でリップルが現れます。
それらは目に見えず、非常に有害であり、健康、作業能力、頭痛、疲労が悪化します。安い照明のあるオフィスで働いていたときに、これらすべての症状を感じました。


測定により、明るさが変化してもリップルがないことがわかります。これは良い結果であり、フィクスチャーにはほとんど見られません。

色温度




OrionおよびFlora LEDテーブルランプの利点の1つは、色温度を連続的に調整できることです。オンにすると、4300Kの白色光でニュートラルに光り、2つのタッチボタンでスムーズに色相が変化します。測定された調整範囲は2700Kから6700Kで、暖かい白色光と冷たい白色光に対応しています。これはあなたの好みに応じて快適な職場の照明を作成します。



演色評価数




高い演色評価数は正確な色再現とコントラストを提供します。これは、ドキュメントを読んで作業するときに特に必要です。測定結果によると、演色性はCRI 93-97の範囲内であり、優れた結果です。
従来のLED光源の場合、演色評価数はCRI 70-80で、標準では住宅の敷地内で最低限のCRI80が必要です。正確な作業を読むには、CRI90以上が必要です。ほとんどの場合、CRI80は仕様に示されていますが、製造業者はCRI70で最も安価なLEDを保存してインストールします。
