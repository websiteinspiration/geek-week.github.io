実用的な例でCSS変数を学ぶ



こんにちは！

かつてウェブサーフィンがこの素晴らしいコードに導いてくれました。

その「素晴らしさ」（つまり、コード）は、CSSでの変数の実用的な使用であり、スタイルの記述を非常に容易にします（変数の使用は、異常なCSS構造も引き起こします）。そしてもちろん、トピックは非常に印象的です-太陽系の惑星。

元のプロジェクトはPugとSassで記述されていますが、HTML / CSSで分析します。

じゃ、行こう。

次のようになります。

  • 地球のラッパーとその情報-.card
  • ラッパーを特定の惑星にバインドする-.mercury
  • 惑星のコンテナ-.planet
  • 内部は大気のコンテナです-.atmosphere
  • その内部は惑星の表面です-.surface
  • 惑星情報コンテナ-.info
  • 対応する惑星の名前は.titleです。
  • 追加情報コンテナ-.form
  • 情報ユニットコンテナ-.item
  • 合計すると、3つの情報単位があります。
    1. 1つ目は、惑星の軸の傾きに関する情報と矢印の画像です。
    2. 秒単位-時間単位での日の長さに関する情報
    3. 3番目-年間の日数

CSSに移りましょう。

CSSで変数を宣言するには、次の構文を使用します。

:root {
  ---: ;
}

ルートの疑似クラスの詳細については、こちらをご覧ください。

惑星の変数は次のように宣言されています。

:root {
    --mercury-image: url(https://thebestcode.ru/media/solarSystem/mercury.jpg);
    --mercury-tilt: rotate(0.034deg);
    --mercury-day: 1407.6;
    --mercury-color: #999;
}

ご覧のとおり、画像、傾斜角、1日の長さ、惑星の色に対して変数がそれぞれ宣言されています。

惑星を回転させるには、次のアニメーションが使用されます。

@keyframes planetRotate {
    0% {
        background-position: 0% center;
    }

    100% {
        background-position: -200% center;
    }
}

CSSで使用される変数は次のとおりです。

var(---);

次に、変数を使用して惑星のスタイルを作成する方法を示します。

.mercury .atmosphere {
    box-shadow: inset 10px 0px 12px -2px rgba(255, 255, 255, 0.2), inset -70px 0px 50px 0px black, -5px 0px 10px -4px var(--mercury-color);
}

.mercury .surface {
    background-image: var(--mercury-image);
    transform: var(--mercury-tilt) scale(1.2);
    animation: planetRotate calc(var(--mercury-day)*.1s) linear infinite;
}

.mercury .planet::before {
    transform: var(--mercury-tilt) scale(1.2);
    border-color: var(--mercury-color);
    color: var(--mercury-color);
}

.mercury .icon {
    transform: var(--mercury-tilt);
    color: var(--mercury-color);
}

日の長さの変数（--mercure-day）を使用してアニメーションの期間（惑星の回転速度）を決定する方法に注意してください。
