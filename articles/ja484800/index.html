Microsoft Application Inspectorソースコードアナライザー



マイクロソフトは、Microsoft Application Inspectorをリリースしました。これは、ソースコードを分析するためのクロスプラットフォームオープンソース（！）ツールです。


Application Inspectorは、「良い」または「悪い」パターンを特定しようとしない点で、従来の静的分析ツールとは異なります。彼は、暗号化の使用など、セキュリティに影響を与える機能を含む、機能を検出するための500以上のルールテンプレートの初期セットに基づいて、検出することを報告します。


Application Inspectorの主な目的は、他の一般的なスタティックアナライザーにはないソースコード関数の体系的でスケーラブルな識別です。これにより、開発者やセキュリティ専門家は、コンポーネントの意図された目標を確認できます。たとえば、使用されているライブラリは、その中で宣言されているものだけを実行します。
