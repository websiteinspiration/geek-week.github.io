それは起こった！CRI 90+を備えた安価なLEDランプ

光の演色評価数は、オブジェクトの見え方に影響します。このインデックスが高いほど、色がより自然になり、色合いがより見分けやすくなります。太陽と白熱灯の演色評価数は100で、安価なLEDランプの場合は70、標準的なものの場合は80です。今日まで、演色評価数が90を超えるLEDランプは、IKEAストア（Ledareシリーズ）でしか購入できませんでした。

最後に、最初のロシアのメーカーは、演色評価数の高い一連の電球をリリースしました。先駆者はNanosvetでした。

1週間前にロシアに届いた新しいランプの26のモデルを製造元から受け取り、それらをテストしました。


新しいシリーズには、「梨」、「G45ボール」、「キャンドル」、スポットGU10、​​GU5.3、GX53があります。すべてのランプは、2700Kウォームホワイトライトと4000Kニュートラルホワイトライトの2つの色温度で利用できます。


私のテストの結果：


すべてのランプの演色評価数CRI（Ra）は、実際には90を超えています。暖かい光のあるランプの場合、測定された演色評価数は90〜92で、中性光のあるランプの場合は92〜94でした。

すべてのランプの測定された光束は宣言された（平均105％）の94-113％に達しました。
すべてのランプの測定電力は、宣言されたものと1 W以上違いはありません。
すべてのランプの光リップル係数は1％未満です。

すべてのランプはICドライバー上に構築されているため、電源電圧が広範囲に変化しても輝度は変化しません。すべてのランプは動作し、パッケージに記載されている供給電圧範囲（170〜250 V）で輝度を変更しませんが、多くのランプは動作し、低電圧でも輝度を低下させません（表の「最小電圧」の列は、ランプの輝度が5％低下します）。

すべてのランプは、インジケーター付きのスイッチで正常に動作します（スイッチがオフのときは点滅せず、弱く点灯しません）。

ほとんどのロシアブランドのランプテスト結果と比較すると、これらの結果は驚くべきものに見えます。Nanosvetは、パッケージのパラメーターを過大評価せず、消費者をだまさないほとんど唯一のロシアブランドのままであるようです。

これらのランプは、Lamptest.ruサイトの最終的な評価を計算するための自動システムの5ポイントの最大評価を受けるロシアブランドの最初のランプでした。

光束、色温度、演色評価数は、2メートルの積分球とInstrument Systems CAS 140 CT分光計、消費電力と力率をRobiton PM-2で、脈動をUprtek MK350Dで測定しました。光束が公称値の5％以下しか低下しない最小動作電圧は、Lamptest-1デバイス、Stabil Instab 500スタビライザー、およびLantra Suntek TDGC2-0.5を使用して測定されました。測定前に、ランプのパラメータを安定させるために、ランプを30分加熱しました。

暖かく中性的な光のランプのスペクトルを示します。
最初の例は「ナシ」600 lm 2700K L160 LE-GLS-8 / E27 / 927、2番目の例は「ボール」G45 E14 680 lm 4000K L205 LE-P45-8 / E14 / 940です。


ちなみに、この梨は、60ワットの白熱電球に代わるもので、わずか149ルーブルです。CRI 90のランプの場合、これは非常にクールです。電球の価格は219ルーブルですが、75ワットの白熱灯のように輝くので、これも悪くありません。

表の「タイプ」フィールドに60°と示されているスポットには、中心に光を集中させてグレア効果を軽減できるレンズが装備されています（habr.com/ru/company/lamptest/blog/391349）。


強力なGU10スポットは細長いボディを持っています。


製造元は、すべてのランプに対して2年間の保証を提供しています。

小さなスプーン一杯のタール。 NanosvetはすべてのランプをCRI 90+に移行する予定なので、新しいランプのバーコードはCRI 80のランプのバーコードと一致し、それらの名前は非常に似ています。たとえば、新しいシリーズでは600 lmの「ナシ」はL160 LE-GLS-8 / E27 / 927と呼ばれ、古いL160 LE-GLS-8 / E27 / 827では呼ばれます。 1桁違います。このため、店舗は新旧のランプを混同し、それらの違いに気付かない場合があります。

ようやく、演色評価数の高い安価なLEDランプが登場してよかったです。私の知る限り、CRI 80と90が同じ電力の同一のランプの工場出荷時の価格は5％だけ異なるが、「人々がこれを必要としないので、主なことはそれをより安くすることである」 」少なくとも小さなNanosvetがそのようなランプを注文してロシアに持ってきたのは良いことです。今の主なことは、これらのランプの信頼性があれば、すべてが他のパラメータと同じくらい良いことです。

NanosvetのPS試験結果は、CRI 90+有するランプLamptest.ruで。
