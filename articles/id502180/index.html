Microsoft: Zero Trust. Amankan Tempat Kerja Jarak Jauh


12 Mei 2020 / 10:00 - 11:30 (GMT +3)

Mari kita bicara tentang konsep Zero Trust dan penerapannya dalam skenario kerja jarak jauh:

  • Mengapa paradigma perimeter jaringan tradisional tidak efektif akhir-akhir ini?
  • Apa itu deperimetrization dan perimeter yang ditentukan perangkat lunak.
  • Prinsip, arsitektur, dan komponen model
    Zero Trust.

Implementasi praktis dari model Zero Trust berdasarkan pada solusi Microsoft: Azure AD, Azure Identity Protection, Azure MFA, Akses Bersyarat, Proxy Aplikasi Azure, Keamanan Aplikasi Cloud, Azure Sentinel.

Pembicara:  Artyom Sinitsyn  (CISSP, MCSE, Manajer Program Keamanan Informasi MCITP di Eropa Tengah dan Timur, Microsoft)

Softline: Microsoft 365. Keamanan lebih terjangkau daripada yang terdengar


18 Mei 2020 / 10:00 - 11:30 (GMT +3)

Di webinar kami akan mendemonstrasikan solusi Akses Bersyarat + MFA, Office 365 ATP, Intune, Microsoft Defender ATP. Dan kami juga akan memberi tahu bagaimana teknologi Microsoft dapat mengurangi risiko saat bekerja di luar jaringan perusahaan.

  • Mendapatkan akses penuh menggunakan "kondisional".
  • Tautan dan lampiran aman dalam surat.
  • Pekerjaan jarak jauh dari perangkat seluler.
  • Perlindungan Endpoint.

Pembicara:  Denis Teplyakov  (Manajer Pengembangan, Microsoft Cybersecurity, Softline) &  Nikolay Ivanov  (Arsitek, Microsoft Cybersecurity, Softline)

Keamanan Informasi: SOC karantina
