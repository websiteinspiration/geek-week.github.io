Eternal virtual servers

My name is Leonid, I am the developer of the site Search VPS. As the name implies, the project specializes in assisting in finding virtual servers, so I try to keep track of all the innovations in this area.



One of the latest products that some hosters have presented is an eternal virtual server, that is, VPS / VDS with a one-time payment: the user pays a fixed amount and can use such a server for as long as necessary.







I‚Äôve gotten used to marketing tricks like ‚Äúunlimited traffic‚Äù or ‚Äúan unlimited number of sites‚Äù for a long time, and I feel normal about such things, although I understand that there is nothing unlimited in fact, and the traffic will be limited at best by the maximum channel speed, and infinite the number of sites can run into a limit on the number of files or resources consumed. There are also sometimes offers with unlimited disk space (this is mainly true for shared hosting), but most often there are a lot of caveats that such a tariff cannot be used for posting pictures, file storages and other projects that take up a lot of space.



All of the above marketing techniques have one difference from the eternal servers: the user pays constantly.



In the case of an eternal server, the situation is exactly the opposite: the hoster receives the amount (albeit many times the cost of rent for a month) once and further gives the user the right to use the server without time limits. I saw a lot of criticism of such tariffs, basically it all came down to unprofitability or to the fact that the hoster could go broke or at some point say that the tariff ceases to be eternal.
