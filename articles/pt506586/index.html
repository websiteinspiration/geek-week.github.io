Um prólogo é uma curiosa oportunidade de transferir condições de tarefas para "símbolos"; é um sistema baseado em lógica de predicados e procedimentos internos para provar automaticamente teoremas. A formulação do problema, se expressa com sucesso, permite que você encontre uma solução para o problema em si, além de ter imediatamente uma solução para o problema "inverso".
