Canonical telah menerbitkan bagian pertama panduan transisi dari Windows 7 ke Ubuntu




Pada bagian sebelumnya dari tinjauan ( vk.com/@permlug-foss-news-0 ) kami menulis tentang aktivasi komunitas FOSS karena penghentian dukungan untuk Windows 7. Pertama dengan menerbitkan daftar alasan untuk transisi dari Windows 7 ke Ubuntu, Canonical melanjutkan topik ini dan membuka seri. artikel dengan panduan transisi. Pada bagian pertama, pengguna diperkenalkan dengan terminologi sistem operasi dan aplikasi yang tersedia untuk pengguna di Ubuntu, menunjukkan bagaimana mempersiapkan transisi ke OS baru dan cara membuat cadangan data. Di bagian selanjutnya dari manual ini, Canonical berjanji untuk merinci proses instalasi untuk Ubuntu.

Kali Linux 2020.1 Rilis Distribusi Penelitian Keamanan




Rilis distribusi Kali Linux 2020.1 telah dirilis, dirancang untuk memeriksa sistem untuk kerentanan, melakukan audit, menganalisis informasi residual dan mengidentifikasi efek dari serangan berbahaya. Semua perkembangan asli yang dibuat sebagai bagian dari distribusi didistribusikan di bawah GPL dan tersedia melalui repositori Git publik. Beberapa opsi untuk iso-gambar disiapkan untuk diunduh, dengan ukuran 285 MB (gambar minimum untuk instalasi melalui jaringan), 2 GB (perakitan Langsung) dan 2,7 GB (instalasi penuh).

Build tersedia untuk arsitektur x86, x86_64, ARM (armhf dan armel, Raspberry Pi, Banana Pi, ARM Chromebook, Odroid). Desktop Xfce default ditawarkan, dan KDE, GNOME, MATE, LXDE, dan Enlightenment e17 juga didukung.

CERN Facebook Workplace Mattermost Discourse




Pusat Penelitian Nuklir Eropa (CERN) telah mengumumkan penghentian penggunaan produk perusahaan untuk komunikasi internal karyawan Facebook Workplace. Alih-alih platform ini, CERN akan menggunakan solusi open source, Mattermost untuk olahpesan cepat dan mengobrol, dan Wacana untuk diskusi panjang.

Penolakan terhadap Facebook Workplace adalah karena kekhawatiran tentang privasi, kurangnya kontrol atas data mereka dan keinginan untuk tidak bergantung pada kebijakan perusahaan pihak ketiga. Selain itu, tarif platform telah diubah.

Pada tanggal 31 Januari 2020, migrasi ke perangkat lunak sumber terbuka selesai.

Perubahan Lisensi Qt Framework




Berita ini terutama menyangkut pengembang dan perusahaan yang menggunakan produk berbasis Qt.

Qt Company, yang mendukung dan menyediakan layanan konsultasi untuk kerangka kerja populer C ++ lintas platform Qt, mengumumkan perubahan dalam hal akses ke produk-produknya.

Ada tiga perubahan utama:

  1. Untuk menginstal binari Qt, Anda memerlukan akun Qt.
  2. Edisi Dukungan Jangka Panjang (LTS) dan penginstal offline hanya akan tersedia untuk pemegang lisensi komersial.
  3. Penawaran Qt baru untuk bisnis kecil akan muncul.

Item pertama hanya memberikan sedikit ketidaknyamanan, Anda harus mendaftar di situs web perusahaan. Namun, mengingat tren yang semakin meningkat dalam pengumpulan data pribadi oleh semua yang bisa dan sering skandal dengan kebocoran, ini tidak mungkin untuk menyenangkan siapa pun.

Poin kedua secara signifikan lebih tidak menyenangkan - sekarang komunitas proyek yang bergantung pada Qt harus lebih berupaya mendukung kode. Sebagai contoh, versi distribusi LTS akan perlu secara mandiri memelihara cabang QT LTS untuk membuat pembaruan keamanan dan yang penting lainnya di sana, atau meningkatkan ke versi terbaru, yang dapat menyebabkan masalah dengan program-program dalam kerangka kerja ini, yang semuanya tidak mungkin dapat dengan cepat mem-porting kode mereka.

Ketiga, mereka mengembalikan lisensi untuk perusahaan baru dan usaha kecil sebesar $ 499 per tahun, yang mencakup semua fitur biasa kecuali untuk lisensi distribusi dan dengan dukungan penuh (hanya dukungan instalasi yang disediakan). Lisensi semacam itu akan tersedia untuk perusahaan dengan pendapatan atau pembiayaan kurang dari $ 100.000 per tahun dan kurang dari lima karyawan.

XCP-ng, versi gratis Citrix XenServer, adalah bagian dari proyek Xen




Para pengembang XCP-ng, pengganti gratis dan gratis untuk platform manajemen infrastruktur cloud milik XenServer (Citrix Hypervisor), telah mengumumkan aksesi mereka ke Proyek Xen, yang sedang dikembangkan sebagai bagian dari Linux Foundation. Transisi ke Proyek Xen akan memungkinkan kita untuk mempertimbangkan XCP-ng sebagai kit distribusi standar untuk menggunakan infrastruktur mesin virtual berdasarkan pada Xen cross-platform hypervisor yang didistribusikan berdasarkan ketentuan GNU GPL v2 dan XAPI. XCP-ng serta Citrix Hypervisor (XenServer) memiliki antarmuka yang sederhana dan intuitif untuk instalasi dan administrasi dan memungkinkan Anda untuk dengan cepat menggunakan infrastruktur virtualisasi server dan workstation dan termasuk alat untuk mengelola, mengelompokkan, berbagi sumber daya, migrasi dan bekerja dengan sistem penyimpanan.

Detail

Distribusi Linux Mint Debian 4 sedang bersiap-siap




Selain Linux Mint 20, yang akan muncul tahun ini dan akan didasarkan pada Ubuntu 20,04 LTS, tim Linux Mint sedang mempersiapkan Linux Mint Debian 4 (LMDE) berdasarkan distribusi Debian 10. Janji baru adalah untuk mendukung matriks HiDPI dan meningkatkan subproyek Mint X-Apps , Desktop Cinnamon, enkripsi, dukungan untuk kartu NVIDIA dan banyak lagi.
