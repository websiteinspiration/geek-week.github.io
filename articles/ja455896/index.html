アバロニアの機能とチップについて話しましょう


明らかに、比較がある場合は、wpfと比較されます。

msのフレームワークの後に最初に目を引くのは、コマンドをメソッドにバインドする機能です。はい、機能的にはこれにより機会が少なくなりますが、それに応じてビューモデルを開発するアプローチが簡素化されます。

<Button Command="{Binding RunTheThing}" CommandParameter="Hello World">
Do the thing!
</Button>

public void RunTheThing(string parameter)
{
            // Code for executing the command here.
}

Avaloniaは、リアクティブ拡張機能を使用して構築されているため、LINQクエリの宣言構文を使用して、最初のクラスのオブジェクトとしてフレームワークのイベントを操作でき、簡潔で読みやすいコードを記述できます。

しかし、もう少し深くすると、すべてがそれほどバラバラで明白ではなくなり、ドキュメントはまだ思い浮かびません（しかし、あなたは助けることができます）。

画像へのバインド


プロジェクトを更新するときにほぼ即座に遭遇した問題は、ウィンドウに画像を表示する方法でした。

まず、イメージを含むディレクトリをAvaloniaリソースとして登録する必要があります。これは、すべてのユーザーがアイコンを表示し、すべてのコードと便利にそれらをパックすることが重要であるためです。

 <ItemGroup>
        ...
        <AvaloniaResource Include="Assets\*" />
</ItemGroup>

これでイメージタグで、選択したイメージを簡単に設定できます。

<Image  Source="/Assets/error.ico" />

ただし、vmソースを使用する場合は、画像を文字列に添付するだけでなく、ビットマップを使用する必要があります。

<Image   Source="{Binding ImagePath}" />

 public Bitmap ImagePath { get; private set; }

したがって、今度はリソースから画像を抽出する必要があります。Avaloniaロケーターを使用する理由

ImagePath = new Bitmap(AvaloniaLocator.Current.GetService<IAssetLoader>()
                    .Open(new Uri($" avares://ASSEMBLYNAME/relative/project/path/{ImageName}.ico")));

uriをコンパイルするときの埋め込みリソースの場合、接頭辞resm：//が使用され、avalonia- avares：//の場合は注目に値します。

スタイル


スタイル自体については十分にドキュメントに記載されています。これらは少しCSSでおなじみのスタイルであることは注目に値します。別のファイルからスタイルを適用する方法を示したかったのです。まず、すべてのxamlファイルはavaloniaリソースでなければなりません。

<ItemGroup>
        <AvaloniaResource Include="**\*.xaml">
            <SubType>Designer</SubType>
        </AvaloniaResource>
      .....
    </ItemGroup>

2番目に、スタイルは、uriおよびロケーターを使用して画像リソースと同じ方法で適用されます。

YourControl.Styles.Add(new StyleInclude(new Uri("avares://ASSEMBLYNAME/relative/project/path.xaml")){Source = new Uri("avares://ASSEMBLYNAME/relative/project/path.xaml")});

愉快なささいなこと


TextBoxにはWatermarkプロパティがあります。これにより、サードパーティのパッケージを検索したり、TextBoxの上でTextBlockをブロックしたりできなくなります。これは、wpfの多くのチュートリアルで読むことができます。

<TextBox Watermark="Street address" />

また、すべてのバインダーはデフォルトでPropertyChangedを使用して、関連するプロパティの更新をトリガーします。
