ジェネレーターのABPを自分で行うには





数年前、私は発電機から国内で作業するための自動トランスファースイッチ（自動予備入力）を作りました。現在、多くのIT労働者が離れた場所に移動しており、コテージから作業を行っています。この場所では、電源の品質を最大限に活用できます。そのため、ATmega8Aマイクロコントローラーでの自家製ABPに関する私の経験について書くことにしました。トピックが興味深い場合は、katへようこそ。たくさんの文字とコードがあります。



接地について



電気で何かをする前に、あなたはあなたの家で良い接地を持っていることに気をつける必要があります。通常の家庭用ガス/ディーゼル/ガス発生器を家庭の電源にそのまま接続することはできません。注意事項を遵守する必要があります。 1つ目は、発電機が十分に接地されている必要があることです。そうすれば、お気に入りのセーターの静電気が発電機の巻線の絶縁体を突破しても、感電しない可能性が高くなります。一般に、実行中のジェネレーターに不必要に手を触れないでください。



ネットワークが常に220Vであるとは限らないことを覚えておく価値があります。ラインの切り替え、遠方での雷放電、静電気放電は、ネットワーク内で数キロボルトの短いパルスが頻繁に発生するようなピックアップを提供します。彼らは家の入り口に避雷器とSPDを設置することによってこれに苦労していますが、これはロシア連邦では非常にまれな習慣です。ですから、火花はあなたを通してではなく、地球の中に行きましょう-家中の良い接地をしてください。これがないと、これ以上何もできません！



発電機について



ところで、多くの家庭用ガソリン発電機では、巻線は決して地面に接続されていません。そして、発電機から1つの電動工具を使用している場合、これは非常に正常です。ただし、発電機を家に接続する必要がある場合は、中性線（N）と相線（L）を作成する必要があります。これを行うには、発電機端子の1つを接地します。この接地点から、2本のワイヤーを独立して家に引き込む必要があります。1つはニュートラルN、もう1つは保護アース（PE）です。発電機を選ぶとき、あなたはその出力を接地することが可能であるかどうかに注意を払う必要があります、時々それは発電機の指示で禁止されています、そしてそのような発電機はあなたに合いません。



多くの場合、ネットワーク上では、N線とPE線を接地したり分離したりせずに、発電機の接続図を見ることができます。これを行わないでください。長生きします。このようなスキームは、最初の不幸な一連の状況までうまく機能します。最新の電子機器の一般的な電源には、L、Nラインからグランドへのコンデンサが含まれます。Nが発電機で接地されていない場合、Nラインのこれらのコンデンサにより、運が良ければ、接地に対して110ボルトになります。ちなみに、このモードの多くのガスボイラーは一般的に停止します。接地のない静力学の影響については、すでに上で書いています。



ABPスキームについて



ATSを実装するためのいくつかの異なるスキームがあります。さらに、私の観点から単相ATSの最も安全な回路について書きます。 1つの接触器でABPを慎重に行うことや、1つの相線のみを切り替えることはお勧めしません。ニュートラルのみ。







上の図では、ネットワークと発電機からの電力は、入力1と2を介して供給されます。これらは、ペアのマシンによって保護されています。追加の機械を介して、スイッチングおよび表示回路に電力が供給されます。リレーコイルが相互に電気的に連動していることがわかります。特定の入力を含めるために、簡単にするために、図に示されていない、スイッチングポイントTK1またはTK2で回路を閉じるマイクロコントローラーが責任を負います。



基本的なポイントは、ATS 2x遮断方式の存在です-コンタクタのスイッチング入力の相互機械的遮断とコンタクタの相互電気遮断。経済のために自作したのですが、彼らのデザインではこれらのロックを無視していますが、無駄です。ブロッキングのない回路はしばらく動作する可能性がありますが、ある時点で接点が燃えて、リターンスプリングが弱まり、入力間で短絡が発生します。第一に、両方のラインが通電されている場合、それは大きなブロードを脅かしますが、これは最大の問題ではありません。予期せぬことに配線を修理する電気技師にとって、発電機が共通ネットワークに電圧を供給することができることははるかに重要です-状況の好ましくない組み合わせでは、ラインを修理する電気技師は死ぬ可能性があります。これはすでにあなたのための犯罪記事です。



接触器について



したがって、従来のリレーの使用はなくなり、専門の接触器のみが適しています。大容量の場合でも、電動ドライブのオプションはありますが、一般的な家庭での使用には高価で冗長です。



機械的インターロックを作成するには、ペアで動作できる接触器を選択する必要があります。通常、相互遮断は、同じ接触器を隣り合わせに設置し、追加のオプション（機械的インターロック）を設置することで実現されます。コンタクタとは別に販売されており、1ペニーの費用がかかります。



接触器に開路時に作動する追加の信号接点がある場合、相互の電気的インターロックが可能です。それらはすぐに分離膜コンタクターに組み込まれる場合もあれば、購入してオプションとしてインストールできる場合もあります。



接触器の主要メーカーは、そのような機器を自社ラインに持っています。したがって、キットを見つけて購入することは難しくありません。確かに、ブランドの接触器の価格は私たち/中国よりはるかに高いです。スイッチングサイクルの数は多くないと予想されるため、中国のコンタクタの選択は正当化されます。不利な点は、動作中に接触器のコイルが非常に多くの話題を呼んでいるという事実にのみ起因する可能性があります。



スイッチング電源の詳細。接触器の接点は、家で消費することが許可されている最大電力に耐える必要があります。私はそれを10 kW持っているので、約50アンペアで1つの接点を流れる許容電流のコンタクタを選択しました。なんらかの理由で、一般的な三相接触器のスイッチ電力がすべての三相のパスポート合計に示されているため、1つの接点を流れる許容電流を注意深く確認する必要があることに注意してください。

void counter2_init( void )
{
   ASSR = 0;  /* AS0 = 0 */  /* disable asynchronous mode */
   while (ASSR); /*EMPTY*/

   OCR2 = 223;             /* 70    */

   TCCR2 |= (1 << CS22) | (1 << CS21) | (1 << CS20);  /* prescale /1024 */
   
   TCCR2 |= (1 << WGM21);                /* mode CTC */
   TCCR2 &= ~(1 << WGM20);
   
   TCCR2 &= ~(1 << COM21);              /*    OC2 */
   TCCR2 &= ~(1 << COM20);
   
   TIMSK |= (1 << OCIE2);          /* enable compare interrupt */
}

void ADC_init() // ADC1,0
{
   //   2,5, 10 bit 
   ADMUX = (0 << REFS0) | (0 << REFS1) | (0 << ADLAR) |
   (0 << MUX3) | (0 << MUX2) | (0 << MUX1) | (0 << MUX0); // ADC0
   // , free running,  
   ADCSRA = (1 << ADEN) | (1 << ADFR) | (1 << ADIE) |
   (1 << ADPS2) | (1 << ADPS1) | (1 << ADPS0); //  128
   
   ADCSRA |= (1 << ADSC);             // Start ADC Conversion
}

ISR(ADC_vect) //     2500    
{
   if ((ADMUX & (1 << MUX0))) { //    ADC1
      if (sampleCount++) //   
         tempVoltage1 += ADC;
      if (sampleCount >= SAMPLES) {
         sampleCount = 0;
         sumVoltage1 = tempVoltage1;
         tempVoltage2 = 0;
         tempVoltage1 = 0;
         ADMUX &= ~(1 << MUX0); //   ADC0
      }        
   } else { //    ADC0
      if (sampleCount++) //   
         tempVoltage2 += ADC;
      if (sampleCount >= SAMPLES) {
         sampleCount = 0;
         sumVoltage2 = tempVoltage2;
         tempVoltage2 = 0;
         tempVoltage1 = 0;
         ADMUX |= (1 << MUX0); //   ADC1
      }  
   }
   ADCSRA |=(1 << ADIF);              // Acknowledge the ADC Interrupt Flag
}

   }

   }

プログラムは無料のAVR Studioを使用して開発され、標準のAVRライブラリを使用します。



メインループでは、プログラムは入力の入力の電圧をチェックし、接触器のスイッチオンの状態を評価し、プログラムタイマーの動作を考慮に入れ、リレーと接触器をオンまたはオフにして必要な調整を行った後、休止状態に入ります。デバッグのために、マイクロコントローラのシリアルポートへの印刷をデバッグするという結論が下されます。



フリーズを制御するために、ウォッチドッグタイマーが用意されています。



すべての測定サイクルは、割り込みおよびハードウェアタイマーを使用して行われます。2番目のカウンターはタイマー1に設定されます。タイマー1が中断すると、ソフトウェアタイマーが更新され、接触器と発電機のリレーのオンとオフの遅延を担当します。

2番目のタイマーは、ステータスLEDの点滅効果を作成するために使用されます。 3つの点滅パターンがあります。タイマー2の割り込みで点滅パターンの値をとります。点滅することでコントローラーの状態を判断できます。



2つのADCは、タイマーと平均2500サンプルの電圧測定でも機能します。測定値を実際のボルトに変換するために、校正定数が用意されています。それらの値は、ATSの構成中に修正する必要があります。



さらに、試運転中に決定する必要のある定数がいくつかあります。 



abp_timers[TMR_220N_ON].set_secs = 10; //     
abp_timers[TMR_220G_ON].set_secs = 60; //        
abp_timers[TMR_220N_OFF].set_secs = 5; //     
abp_timers[TMR_220G_OFF].set_secs = 5; //     
abp_timers[TMR_GEN_OFF].set_secs = 60; //


発電機からの運転中は発電機停止リレーがオンとなり、発電機停止回路への電圧供給が遮断されます。発電機のアイドルタイマーが完了すると、リレーがオフになり、電流がこのリレーを介して発電機停止回路に流れます。発電機自体には特別なユニットがあり、遅れて電圧が発生した後、点火回路を接地に閉じ、発電機を停止させます。同じブロックには発電機充電回路が含まれています。詳細に興味がある人がいたら、コメントに書き込んでください。このブロックについて別の投稿をします。その中にコードはありません、すべてがハードウェアです。



誰かがABPを繰り返すことにした場合、設定を調整する価値があります。プログラムはATSの構成中に修正する必要があるため、完成したファームウェアは公開しません。
