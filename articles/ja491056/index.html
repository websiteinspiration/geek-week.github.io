Ampere Altra-世界初の80コアARMプロセッサ



カリフォルニアの会社、アンペレは、64ビットアーキテクチャのアンペレアルトラに業界初の80コアサーバーARMベースのプロセッサを導入しました。

数年前から、専門家は、ARMプラットフォームがデータセンターでx86と競合すると予測していますが、これは起こりません。 2019年の終わりに、Intelが95.5％、AMD-4.5％のシェアで圧倒します。

ただし、SPECrate 2017整数ベンチマークの新しいARMプロセッサは、最速の64コアAMD EPYCまたはカスケードレイクファミリーのトップエンド28コアXeonよりも高いパフォーマンスを示します。これは深刻なアプリケーションです（ただし、ベンチマーク結果は少し「ねじれ」ていますが、以下を参照してください）。

ARMの主な利点はエネルギー効率です。これは、アーキテクチャ上、x86プロセッサーは本質的には一致しません。 80コアのアンペアアルトラは、45〜210ワットのTDP、3 GHzのクロック周波数を備えています。

Ampereは、2つのコアではなく1つのコアにつき1つのスレッドが、より高いセキュリティに寄与すると信じています。





プロセッサは、データ分析、人工知能、データベース、ストレージ、テレコミュニケーションスタック、ボーダーコンピューティング、Webホスティング、クラウドアプリケーションなどのサーバーアプリケーション用に設計されています。特にハードウェアレベルの機械学習アプリケーションでは、データ形式FP16（半精度数値）とINT8（整数の1バイト表現）のサポートが実装されています。 AESおよびSHA-256のハードウェアアクセラレーションハッシュもあります。



マイクロチップはTSMC工場で7 nmプロセステクノロジーを使用して製造されています。最初のCPUサンプルはすでに潜在顧客に送信されており、大量生産は2020年半ばに開始される予定です。

Ampere CEOと元インテル社長のRenéeJamesは、ARMサーバープロセッサも設計した破産したApplied Micro Circuits Corporation（1979–2017）を基にして、2017年10月にAmpere Computingを設立しました。特に、2011年に彼女はARMv8-Aに基づく64ビットX-Geneプラットフォームを発表しました。

ジェームズは現在、アンペレコンピューティングのエグゼクティブディレクターおよび取締役会の会長と、米国大統領に助言を行う国家安全保障通信諮問委員会の副会長の地位を兼務しています。

ARMプロセッサをサーバー市場に投入するための新しい試みはどれほど成功するのでしょうか。

「市場で最も多くのコアを持つプロセッサをリリースしました」と James 氏は言います。 「今、それを[テスト用に]業界最大のクラウドサービスプロバイダーの一部に送りました...人々は驚かれると思います。」 [以前のテクノロジーの置き換え]は常に何か新しいものをもたらします。そして、既存の会社からではなく、新しい会社からです。業界の発展における次の段階であると私が考えるものに取り組むことは非常に興味深いです。

AMDと前述のApplied Microが同様のプロセッサを製造しようとしたときに、過去数年間に64ビットARMサーバーチップについて多くの話がありました。しかし、これらの企業は失敗しました。 AMDはARMプロジェクトを閉鎖し、Applied Microの資産が売却されましたマコム。 2017年、カーライルグループは彼女からARMプロセッサ部門を買収しました。契約は2019年末に終了し、ジェームズは新会社のCEOを引き継ぎ、カーライルグループの最高執行責任者を辞任しました。

Jeff Wittichは、テストでは、AmpereプロセッサはAMDの最速EPYCプロセッサより4％優れた結果を示し、消費電力は14％少ないと述べています。私たちは
、225 WのTDPと6950ドルのコストの64コアEPYC 7742 プロセッサについて話しています。これは、Zen 2マイクロアーキテクチャ上のEPYC 2プロセッサフ​​ァミリで最も強力です。このファミリは2019年8月に導入されました。





Wittichはまた、Cascade Lakeファミリーの28コアXeonプロセッサーを挙げました。 Ampere Altraプロセッサは、「パフォーマンスが2.23倍、エネルギー効率が2.11倍」を上回っています。ここでは、指標が28コアのXeon Platinum 8280（205 W）と比較され、1つのコアに基づいてエネルギー効率が計算されました。

伝えられるところによると、整数のベンチマークSPECrate 2017では、Ampere Altraプロセッサーは259を超える結果を示しています。これにより、ASUS RS720A-E9（KNPP-D32）サーバーシステム（2.20 GHz、AMD EPYC 7601）およびASUS RS500A-E10（KRPA-U16）サーバーシステム2.25 GHz、AMD EPYC 7742のピークパフォーマンスの下の表が表示されます。

ただし、アンペアが適用されましたAMD C / C ++コンパイラーはARMのGCCよりも最適化されたコードを生成するため、AMDの結果では、AMD64コンパイラーパッケージを使用してベンチマークコードをコンパイルするため、係数は0.85です。

ベンチマークのそのような「ひねり」にもかかわらず、アンペレアルトラは、パフォーマンスとエネルギー効率の点で非常に印象的です。約3500のプロセッサコアは、12.5 kWの電源を備えた標準の42Uサーバーラックに「パッケージ化」できるため、コアあたりのワット数の計算から節約できます。





そして、これはほんの始まりにすぎません。 Jeff Wittich氏は、1年後にはコード名がMystiqueと呼ばれる別の製品が市場に登場し、その中でAmpereがコアの数をさらに増やすと語った。

Mystiqueは同じコネクタをサポートするため、マザーボードの交換は必要ありません。次世代のSiryn SoCは2022年に計画されています。



近年、さまざまな企業からサーバー側ARMプロセッサをリリースしようとする試みがいくつか観察されています。Broadcom/ Cavium / Marvell、Calxeda、Huawei、Fujitsu、Phytium、Annapurna / AmazonおよびAppliedMicro / Ampereです。これらの試みのほとんどは失敗しました。しかし、状況が変化している兆候があります。 2019年12月、Amazon は64コアのGraviton2 ARMプロセッサーを搭載した本番サーバーを公開しました。ARM Neoverse N1コアと同じコア上のシステムオンチップです。一部のテストでは、ARMインスタンス（M6gおよびM6gd）はx86よりも優れており、時にははるかに優れていました。

2019年11月、米国の新興企業Nuvia がベンチャーキャピタルの資金調達で5,300万ドルを調達したと報告されましたこの新興企業は、AppleとGoogleでプロセッサの作成に関与した3人の主要なエンジニアによって設立されました。また、IntelやAMDと競合するサーバープロセッサの開発も約束しています。入手可能な情報によると、NuviaはARMアーキテクチャの「トップ」に構築できるプロセッサコアをゼロから設計しましたが、ARMライセンスを取得していません。

これはすべて、RISCプロセッサがモバイルデバイスだけでなく、サーバーやデスクトップコンピュータやラップトップでもアプリケーションを見つけられることを示しています。ところで、将来のApple MacBookラップトップもARMプロセッサで登場するという噂があります。



実際、A12X ARMプロセッサを搭載した最新のiPad Proモデルは、Core i7およびCore i9プロセッサを搭載した15インチMacBook Proとほぼ同じなので、このようなアップグレードは非常に論理的になるでしょう。
