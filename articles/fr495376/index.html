Ennui de quarantaine. Comment dépenser le plus d'argent dans Azure en un clic

Il y a quelques jours, Corey Quinn a tweeté :

«Quelqu'un a demandé - je réponds. Une instance entièrement réservée de db.r5.24xlarge Enterprise Multi-AZ pour SQL Server à Bahreïn coûte 3 118 367 $. Un concours pour l'appel d'API le plus cher dans le cloud AWS est annoncé.

Une discussion très productive a commencé dans le fil de discussion, où de nombreuses personnes sympathiques ont essayé de trouver un meilleur moyen de dépenser de l'argent sur AWS. En tant qu'utilisateur Azure, je me sentais triste et seul, j'ai donc décidé de corriger cette injustice. Si vous êtes en quarantaine et que vous voulez dépenser une somme d'argent folle quelque part, je peux vous le dire. Vous n'avez besoin que d'un abonnement Pay-As-You-Go dans Azure - et je vous garantis que tout votre argent durement gagné disparaîtra en un instant!

Réchauffer


La première chose que toute personne sensée qui veut dépenser le maximum d'argent de n'importe quel fournisseur de cloud va essayer de faire est d'acheter la machine virtuelle la plus grande et la plus chère. Malheureusement, ce n'est pas très efficace:



une instance Standard_M416ms_v2 avec 416 vCPU et 11 400 Go de RAM, payée trois ans à l'avance, ne coûtera que 802 milliers d'euros, et ce n'est même pas un million! Pire encore, Azure offre une remise de 71%, et ces remises sont complètement contraires à notre objectif. Donnons du crédit à l'intelligence artificielle: Azure dit que le nombre recommandé de ces instances est zéro pour moi. Devinez assez précis.

Nous franchissons le million (Azure Blob Storage)


Qu'en est-il du stockage d'objets blob? C'est généralement très bon marché, mais que se passe-t-il si, pour une raison quelconque, vous devez stocker un pétaoctet dans le sud du Brésil pendant les trois prochaines années?



Malheureusement, ils donnent un entrepôt avec une simple réservation géographique (géo-redondant). Il ne reste plus qu'à supposer combien coûterait le stockage avec la réservation de zones (géo-zone redondante) s'il était disponible.



Nous franchissons le million (Azure Databricks)


Notre prochain candidat est Azure Databricks. Je n'ai aucune idée de ce que c'est, et je m'en fiche! Je sais seulement que c'est un service assez cher, et c'est exactement ce dont nous avons besoin:



pour trois millions d'euros, vous pouvez acheter six millions de ces «blocs de données». Au prix de 0,53 euros la brique, pour être honnête, je n'ai aucune idée si une telle offre est avantageuse ou non.
