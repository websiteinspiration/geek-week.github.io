Azure Cosmos DBの無料利用枠でアプリを無料で構築する

Azure Cosmos DBを使用して、新しいアプリケーションの作成、開発とテスト、または小規模な本番ワークロードの実行をし  たいですか？新しいレベルの無料使用により、新しいアプリケーションを作成および開発する際に、コストなしで作業を開始し、コストを節約できます。



含まれているAzure Cosmos DBの無料枠を使用すると、アカウントの最初の400 RU /秒と5 GBのストレージをアカウントで毎月無料で利用できます。これは、アプリケーションが高性能のデータベースサービスで実行されることを知っていれば、小規模から始めて自信を持って成長できることを意味します。アカウントが400 RU /秒および5 GBを超える場合にのみお支払いいただきます。さらに、アプリケーションに多数のコンテナーがある場合、合計帯域幅を使用してデータベース内に最大25のコンテナーを作成でき、それらすべてが無料の400 RU /秒を使用します。Azureサブスクリプションごとに、最大1つの無料のAzure Cosmos DBアカウントを持つことができます。

よくある質問（FAQ）


Q：これまで無料のオプションを提供したことがありませんか？何が変わったの？

A：Azure Cosmos DBを無料で試す方法を常に提供していました-Azure無料アカウントの一部として  、  ローカルエミュレーターを使用し、  30日間の無料トライアルとして  -これらのオプションは、可用性とボリュームに制限がありました。無料枠は無制限に利用できます。

Q：アカウントに無料利用枠の割引はどのように表示されますか？

A：無料利用枠のアカウントでは、アカウントに最初の400 RU /秒と5 GBのストレージが無料で提供されます。 RU /秒および400 RU /秒および5 GBを超えるストレージは、料金ページごとに通常の料金レートで請求されます。請求書には、無料の400 Ru /秒と5 GBの料金やラインアイテムは表示されず、無料枠の対象を超えるRU /秒とストレージのみが表示されます。無料レベルのアカウントでは、アカウントの最初の400 RU /秒と5 GBのディスク容量を無料で受け取ります。 RU /秒および400 RU /秒および5 GBを超えるストレージは、通常の料金で支払われます。アカウントでは、無料の400 Ru /秒と5 GBの料金や位置は表示されません。無料階層の対象を超えるRU /秒とストレージのみが表示されます。

Q：無料枠はすべてのAzure Cosmos DB APIに適用できますか？

はい。無料利用枠は、Core（SQL）、MongoDB、Cassandra、GremlinのAPIを含むすべてのAzure Cosmos DB APIで機能します。

Q：自動操縦データベースとコンテナーに無料枠割引を適用できます  か？

A：はい。オートパイロットサービスでは、データベースまたはコンテナーが1時間ごとにスケーリングされる最大のRU /秒に対して請求されます。無料枠割引が適用されると、400 RU /秒がこの値から差し引かれます。詳細については、  ドキュメントをご覧ください。

Q：複数の地域でアカウントを持っている場合、無料枠割引はどのように機能しますか？

A：マルチリージョンのRU /秒アカウントでは、データベースまたはコンテナーはすべてのリージョンで複製されます。たとえば、400 RU /秒のコンテナがあり、アカウントが3つのリージョンにある場合、RU /秒アカウントの総数は1200 RU /秒です。割引が適用されると、1時間あたり1200 RU /秒-400 RU /秒= 800 RU /秒で請求されます。詳細については、  ドキュメントをご覧ください。

Q：すでにアカウントを持っていますが、無料で使用できるレベルのアカウントにデータを転送したいのですが。どうすればよいですか？

A：新しいアカウントを作成して無料利用枠で有効化すると、既存のデータを新しいアカウントに簡単に移動できます。  データ移行ツール Core（SQL）APIを使用している場合は、小さなデータセットの転送に使用できます。Azure Data FactoryからAzure Cosmos DBコア（SQL）API  およびAzure Cosmos DB  API for MongoDBのデータを移行することもでき  ます。

Q：Azureポータルで新しいアカウントを作成するときに無料枠を有効にするオプションが表示されません。彼女はどこ？

A：Azureサブスクリプションには、最大1つの無料枠アカウントを含めることができます。このオプションが表示されない場合は、サブスクリプションの別のアカウントが無料枠ですでに有効になっていることを意味します。
