私のゲーム用にPythonで1000の顔を生成した方法

ゲーム開発のある段階で、アバターの顔を使った対話システムが必要だと気づきました。

そこで、有名なゲームペーパーズをベースにフェイスジェネレーターを作ることにしました。


Dukope Face Generator（ペーパーから、お願いします）

データ収集


pixabay.comから約50枚のポートレートをダウンロードしました（すべての画像はCCライセンスであり、帰属は必要ありません）。

次に、Photoshopで各画像の背景を切り取りました（ポリゴン選択ツールですべてを選択し、[削除]をクリックしました）。将来的には、カットアウト効果を使用してすべての欠点を隠す予定でした。


背景画像

顔のセグメンテーション


顔のセグメンテーションにはレイヤーを使用します。「鼻と口」をブラシでぼかし、次に「目と眉」をぼかして、PSでそれらの新しいレイヤーを作成します。これを行うには、表示されているピクセルを選択し（Ctrl +レイヤーのサムネイルをクリック）、選択したレイヤーから新しいレイヤーを作成します。顔を覆うすべての髪、メガネ、その他のオブジェクトは、「hair_overlap」と呼ばれるレイヤーに送られます。50枚すべての画像のセグメントにタグを付けます。


レイヤー

を操作するプロセスでは、新しく作成した要素の面を準備する必要もあります。クローンスタンプツールを使用して、スキンのクローンを作成し、すべての顔の特徴をワイプします（新しいレイヤーはスマッジツールでワイプします）。

顔の特徴を抽出する


私がダウンロードしたいくつかのポートレート写真は、全体としての使用に適していないため、ジェネレータで使用できる顔の特徴（たとえば、鼻と唇のみ）のみを抽出します。

Pythonライブラリのpsd_toolsとPillowを使用して機能を抽出します（pip install psd-tools Pillowコマンドを使用するか、IDEを使用してインストールできます）。


このコードスニペットは、すべてのレイヤーとそれらの境界の長方形をリストする必要があります。

カットアウトの動作の影響

残念ながら、Pythonでは、Sobelフィルター、中央値フィルター、およびディザリングを使用しないと、この効果を作成することはできません。同じことを行うGIMP FU_artist_cutout.scmのプラグインを見つけましたが、正しく使用されません。オンラインUIもあるG'MICライブラリを使用します。


G'MICオンラインユーティリティ

結局のところ、gmicはスクリプトのみを使用でき、カットアウト効果のコマンドラインパラメーターはありません。カットアウト効果の定義をgmic_stdlib.gmicファイルで見つけました：

fx_cutout :
-repeat $! -l[$>] -split_opacity -l[0]
-median {10-$3}
-quantize $1
--area. 0,1 med=${-med} -rm.
-inpaint_holes {$med*$2%},0,1
-if $4 -n 0,255 -endif-endl -a c -endl -done

これで、g'mic用の独自のカットアウトフィルターを作成できます。

# File `myCutout.gmic`
cutout :
-echo "Arguments : $1,$2"
-input $1
-repeat $! -l[$>] -split_opacity -l[0]
-median 6
-quantize 20
--area. 0,1 med=${-med} -rm.
-inpaint_holes {$med*0%},0,1
-endl -a c -endl -done -output $2

それとコマンドラインツールがあれば、次の作業の準備ができています。


G'MICの実際


ランダムな顔。髪のオーバーレイは素晴らしい作品です。
