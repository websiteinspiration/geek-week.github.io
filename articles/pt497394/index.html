Dez recursos com cursos de ciência da computação disponíveis gratuitamente em abril



O auto-isolamento continua e, com ele, uma nova tendência de presentes, que as plataformas de conteúdo digital estão tentando apoiar e divertir os usuários trancados em quatro paredes. Em março, muitas empresas anunciaram que colocariam parte de seus serviços e materiais em domínio público, para que as pessoas pudessem passar algum tempo estimulando suas atividades cerebrais. Ao longo do mês, seu número só aumentou. Nesta revisão, reunimos sites com cursos para programadores que ingressaram no movimento geral e declararam abril como um mês de treinamento gratuito.

Oracle

Os criadores do Oracle Cloud e do Oracle Autonomous Database anunciaram planos em larga escala para apoiar desenvolvedores e arquitetos de sistemas que desejam melhorar seu nível. Os novos alunos poderão não apenas usar a biblioteca com materiais sobre tópicos relacionados (ciência de dados, aprendizado de máquina, integração com outros serviços em nuvem) de graça, mas também passar em um exame de certificado. Cada participante terá que passar por um programa com duração de aproximadamente cinquenta horas, que incluirá vídeos, conselhos de especialistas e demonstrações visuais dos procedimentos. Para passar no exame, você deve escolher imediatamente uma especialidade e nível de preparação e seguir o programa apropriado.

Udacity

A Udacity, um dos maiores centros de auto-educação para representantes de profissões técnicas, é claro, não permaneceu distante de iniciativas humanitárias. Aqueles que estão cumprindo sua quarentena na Europa ou na América têm mais sorte - eles são convidados a dominar um dos quarenta Nanodegrees pagos anteriormente de sua escolha. Todos os outros podem se registrar, inserir informações em áreas de interesse para seu perfil pessoal e receber um código promocional para um desconto pessoal (como mostra a experiência pessoal dos pesquisadores, cerca de 40%) para o curso correspondente. O site apresenta programas nas seguintes áreas: inteligência artificial, sistemas autônomos, ciência de dados, serviços em nuvem, programação e desenvolvimento.

Executar programa

O público-alvo do Execute Program são desenvolvedores que já passaram do nível básico e agora estão tentando aprimorar suas habilidades de programação. O principal método de treinamento é uma análise interativa de pequenos pedaços de código que o sistema seleciona com base em uma análise dos pontos fortes e fracos do usuário. Atualmente, o site contém materiais em três idiomas (JavaScript, SQL e TypeScript). Até o final de abril, os novatos são convidados a participar gratuitamente dos seguintes cursos: Matrizes JavaScript, simultaneidade JavaScript, JavaScript moderno, expressões regulares, SQL, TypeScript.

Oficinas de Packt

Um pequeno, mas diversificado conjunto de oficinas interativas da Packt pode ser realizado gratuitamente por qualquer pessoa que se inscreva no site antes do dia 31 de maio. Três workshops são para desenvolvedores da Web (HTML e CSS, Javascript, PHP), outros três são para especialistas em dados (curso geral em Ciência de Dados, Visualização de Dados e SQL), os seis restantes são voltados para aqueles que desejam aprender algo novo por si mesmos linguagem de programação (Python, Java, C ++, Ruby, Go ou Clojure). Um bom bônus: o acesso aos cursos "adquiridos" é fornecido aos usuários por um ano - você pode dominar o material armazenado sem pressa.

LearnSQL

O recurso LearnSQL, que, como você pode imaginar, é especializado em materiais de treinamento SQL (cobrindo MS SQL e PostgreSQL), abriu o acesso ao conjunto completo de cursos até 20 de abril, inclusive. Para abrir a coleção inteira, você só precisa criar uma conta gratuita no período especificado. No total, o site apresenta cerca de dez a quinze cursos para iniciantes e contínuos. Quem começa a entender o assunto do zero pode usar o plano de aula pronto, que a administração oferece em um post por referência - os cursos são organizados na seqüência correta e complementados por materiais de vídeo.

Visão Plural

O Pluralsight apresenta uma biblioteca abrangente de cursos técnicos on-line sobre diversos tópicos. A coleção é dividida em vários blocos temáticos: desenvolvimento de software (Python, Java, Javascript, C #, desenvolvimento web, desenvolvimento móvel), Ops (gerenciamento de banco de dados, gerenciamento de servidor, virtualização), processamento de dados (big data, SQL, Tableau , Oracle), aprendizado de máquina (TensorFlow, R, Microsoft Cognitive Services), segurança cibernética, arquitetura, design, serviços em nuvem. Dentro de cada bloco, o usuário pode escolher entre várias dezenas de tutoriais em vídeo que variam muito em nível de dificuldade e duração (de uma a seis horas). A biblioteca estará aberta ao público até o final de abril.

Progate

Progate cria programas educacionais com alcance médio - as principais direções de suas atividades estão relacionadas ao ensino de linguagens de programação e ao trabalho com estruturas. Agora, em solidariedade com o auto-isolamento, a administração do site abriu para todos cinco cursos populares de vários assuntos e volumes de seu arsenal: Python (cinco lições e dois artigos adicionais), SQL (cinco lições), Go (duas lições), Linha de Comando (uma lição) ) e Git (uma lição e dois artigos). Após a inscrição, o participante é automaticamente incluído no grupo de estudo, onde pode competir com outros alunos. A oferta será válida ao longo de abril.

Vue master y

Um recurso especializado para desenvolvedores no Vue.js anuncia uma semana de generosidade sem precedentes: cursos pagos estarão disponíveis para todos os usuários registrados para visualização e estudo. O prazo dessa ação é bastante restrito - exatamente uma semana, de 13 a 19 de abril. Durante esse período, você precisa ter tempo para dominar uma quantidade considerável de materiais: mais de uma dúzia de cursos são apresentados no site, abrindo caminho do iniciante ao avançado, e cada um deles consiste em cinco a dez lições. Para quem gosta, é fornecida uma oferta especial: a oportunidade de emitir uma assinatura anual com um desconto de 30% no código promocional FREEWEEK2020 para a mesma semana.

Unidade

A comunidade Unity oferece oportunidades avançadas para desenvolvedores e designers envolvidos no gamedev - até 20 de junho, ingressar no programa Unity Learn Premium será em termos preferenciais. O programa envolve o acesso a um bom suprimento de palestras, conversas abertas com especialistas em tempo real, jam sessions com outros usuários, em uma palavra, o máximo de diversidade na autoeducação. A gama de tópicos nos materiais de treinamento também é muito ampla: arte conceitual, prototipagem, vários aspectos do desenvolvimento, AR e VR, uso de inteligência artificial, promoção de produtos no mercado.

365 ciência de dados

O 365 Data Science fornece uma imersão realmente profunda no trabalho com dados: os especialistas dessa equipe prepararam mais de vinte cursos em quatro blocos temáticos (aulas introdutórias, desenvolvimento de SQL, Tableau, Python, R, aprendizado profundo, especialização restrita em tarefas). Se cada um desses blocos foi aprovado com êxito pelo participante, no final do programa, ele recebe quatro certificados. Todo o material do site ainda está disponível gratuitamente, mas restam apenas algumas horas até a promoção expirar.
