Petergofsky 19の私たちのオフィスで開催される次のオープンなDINS Java EVENINGで、Java開発に無関心なすべての人に会えて嬉しいです。今夜、DINSのAlexander GruzdevがAWS Lambdaに基づくサーバーレスアプリケーションの開発を実演します。YandexのIgor ShalaruがAkkaフレームワークとそれを使用する利点を紹介します。

カットの下-レポートとスピーカーの詳細、会議に参加するために登録するリンク、前回の会議の資料。



報告書


AWS Lambda in Action（Alexander Gruzdev、DINS）

Alexanderは、何らかの理由で新しいマイクロサービスを作成するのに飽きた人や、EC2のダウンタイムにお金を払いたくない人にとって興味のある開発へのアプローチについて話します。具体例を使用して、ラムダの作成からテスト、デプロイメント、ローカルデバッグまで、プロセス全体を分析します。このレポートは、AWS Lambdaまたはサーバーレスアプローチ全般についてすでに聞いたことがある読者を対象としています。

Alexanderは5年以上にわたり、ライフサイエンスから電気通信まで、さまざまなアプリケーション分野でエンタープライズアプリケーションを大規模に開発しています。彼は、開発者が常に既製のツールについて市場を監視する必要があるという考えに固執し、ほとんどの場合、彼は自分の「バイク」を書かないようにします。

「高負荷システムのコアとしてのAkka」（Igor Shalaru、Yandex）

Akkaは、Java開発者の武器庫に長く登場しています。これは、アプリケーションを開発するための強力で便利なツールです。レポートのフレームワークでは、アクターモデルとは何か、Akkaの既製のモジュールは何かを分析します。例を使用して、Akkaで開発を開始する方法を見て、これが将来どのような利点をもたらすかを見てみましょう。このレポートは、すでにAkkaに精通しているか、単に知りたいと思っているあらゆるレベルのJava開発者を対象としています。

過去7年間、イゴールはJavaでプログラミングしてきました。この間、彼は単純な開発者からTechに移りました。リダ。彼は、負荷とフォールトトレランスの厳しい要件を持つプロジェクトで幅広い経験を持っています。彼はギガバイト/テラバイトのデータを迅速に処理するアプリケーションを書くのが好きです。
