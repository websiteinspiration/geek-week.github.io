    /**
     *       ,       
     * @param folder
     * @param arrayOfStructures
     * @returns {[Object]}
     */

    readFolder(folder: string, arrayOfStructures: object[]): object[] {
        const projectStructure = {};
        let currentDirectory = fs.readdirSync(folder, 'utf8');
        currentDirectory.forEach(file => {
            let pathOfCurrentItem = path.join(folder, file);

            if (fs.statSync(pathOfCurrentItem).isFile()) {
                arrayOfStructures.push({
                    name: file,
                    path: folder.concat('/' + file),
                    extension: path.extname(file),
                    type: 'file'
                });
            }
            else {
                projectStructure[file] = {name: file, path: folder, type: 'directory', arrayOfStructures: []};
                arrayOfStructures.push(projectStructure[file]);
                this.readFolder(pathOfCurrentItem, projectStructure[file].arrayOfStructures);
            }
        });
        return arrayOfStructures;
    }

Script ini menerima direktori dari mana perlu untuk mulai mengumpulkan informasi tentang elemen-elemen yang bersarang, dan sebuah array kosong di mana kami akan menambahkan data yang diterima. Untuk mempermudah, kami membagi item yang dihasilkan menjadi dua jenis: file dan direktori.
fs.readdirSync (folder, 'utf8') mengembalikan hasil membaca direktori kepada kami, setelah itu kita hanya perlu melihatnya dan melihat apa yang sebenarnya ada di dalamnya. Jika ini adalah file, maka cukup tambahkan ke arrayOfStructures array , tetapi jika jenis yang diterima adalah direktori maka kita memanggil fungsi readFolder kami secara rekursif dan mulai dari awal, hanya dalam kasus ini, kami sudah akan mentransfer nama direktori baru. Pada output, kita mendapatkan array objek yang sudah bisa dikirim ke klien.

    /**

    sendDirectoryEngine(request: Context<object>): void {
        const folder = '/GameEngine/Client';
        const structure = this.readFolder(folder, [{}]);

        request.options.parentCtx.params.res.writeHead(200, {'Content-Type': 'text/plain'});
        request.options.parentCtx.params.res.end(JSON.stringify({
            data: [{
                name: 'root',
                type: 'directory',
                arrayOfStructures: structure
            }]
        }));
    }

                result.data[0].arrayOfStructures.sort((a, b) => a.type === 'directory' ? -1 : 1);

The getDirectory metode panggilan metode mengakses logika bisnis dan kemudian bekerja dengan janji diterima, kita menulis data yang diterima ke negara di mana:

  • directoryProject mengimplementasikan objek yang dengannya kita akan membangun komponen visualisasi struktur pohon proyek.
  • selectedDirectory mengimplementasikan larik dengan data untuk membangun komponen visualisasi dari direktori yang dipilih (secara default, ini adalah akar dari proyek).

Kami akan membuat dua komponen utama dari data proyek Direktori dan detail dari direktori yang dipilih.
