GoogleアシスタントをPorfiryevichに置き換えます


アプリケーションを開いてマイクボタンをタップするだけで、音声ポルフィリエビッチを起動できます。ただし、多くのスマートフォンにはアシスタントサポートが組み込まれているため、通常のGoogleアシスタントの代わりに実行する方がはるかに快適です（たとえば、私のOneplus 7では、どのボタンからでも電源ボタンを押してアシスタントをアクティブにできます）。

これを行うには、AndroidManifestに別のインテントフィルターandroid.intent.action.ASSISTを登録し、フラグandroid：launchMode =«singleTop»とMainActivityを実装するだけで、アプリケーションがアシスタントモードで呼び出されたときに音声認識を実行するonNewIntentメソッドを実装できます。

    override fun onNewIntent(intent: Intent?) {
        super.onNewIntent(intent)
        if (intent?.action == Intent.ACTION_ASSIST) {
            val aimybox = (application as PorfirApplication).aimybox
            val channel = aimybox.stateChannel.openSubscription()
            launch {
                channel.consume {
                    aimybox.startRecognition()
                }
            }.invokeOnCompletion { channel.cancel() }
        }
    }

スマートフォンの設定で、デフォルトのアシスタントとしてPorfirevichを選択する必要があります。設定->アプリケーション->デフォルトのアプリケーション
