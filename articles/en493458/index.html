Further ... Unlocking is possible only when the smartphone is connected to the Internet. There is some kind of nuance with factory unlocking and first you need to press certain buttons on the smartphone, and only then use fastboot.

Further ... I quote ...
After the process is completed, we return to the previous menu. Click on the “Advanced Wipe” button and select “System” and “Cache”.

So, the menu item “Cache” does not exist there - instead of it there is “Dalvik / apk cache”. It seems like this is the only menu item with the word cache. But I, for example, as a person who is not far - from IT is not too far, in the sense - I know that caches are different. And this cache is the same cache that needs to be cleared? Or the wrong one?

I write these pebbles solely to show - if we are already writing instructions, it would be nice to give the menu items and steps completely as they really are. In the instructions there are no things that are taken for granted and obvious.

Email


Of course, I use the K-9 Mail recommended in the original article. And it just works, there's nothing to say.

A wonderful feature is a bonus - all letters, regardless of their layout and embellishment, are displayed in readable font. It is necessary to clarify here ...

Today, all letters are sent in HTML and in the mobile client are often displayed in completely unreadable font in millimeter size. I had it in a regular client on Nokia, it was in GMail, it was in the Yandex.Mail application. K-9 resolves this issue.

The regular email client LineageOS was, to put it mildly, dumb. I displayed the first 25 letters in the folder (the most recent), reported that there was more, offered to display them, but did not display them.

This app was discovered by F-Droid in search of an app for a shopping list. It turned out that this is a classic (a rarity these days) application that operates on files. Files are arranged in folders. The files themselves are either plain text or markdown text.
