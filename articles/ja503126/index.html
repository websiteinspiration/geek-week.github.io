-Windows Serverのシステム要件は何ですか？

最小構成は2 GBのRAM、ハードドライブに20 GBです。 Win Server 2003の場合は1つのコアで十分ですが、Win Server 2016または2019の場合はすでに2つ必要です。ハードドライブにもう少しスペースを追加することをお勧めします。両方のオペレーティングシステムは20 GBでタイトになります。たとえば、すでにS2012などはアップグレードできません。 500 MBのRAMを搭載したVPSにWinをインストールするユーザーもいます。 1か月あたり579ルーブルの関税をお勧めします（これにはHabrの割引は含まれていません。以下のプロモーションコードから15％、年払いの場合はさらに20％ダイヤルできます）。1つのコア、2 GBのRAMと20 GBのHDDがあります。安い料金では、Windows Server Coreまたは他のOSを使用することをお勧めします。

-反対側のWindowsが応答を停止した場合はどうなりますか？

オペレーティングシステムまたはターミナルアクセスサービスに何かが発生すると、RDP経由で接続できなくなります。これには、動作中のOS、ネットワークドライバー、動作中のネットワーク接続、およびRDPが有効になっている必要があるためです。箱から出して、すべてが揃っています。しかし、Winが「故障」した場合は、もう1つの特別なモードを使用できます。ハイパーバイザーレベルから直接ビデオをブロードキャストし、ハイパーバイザーを介してキーストロークとクリックを送信する機能があります。このモードはリソースをかなり要求するため、使用には制限があります。ただし、問題が発生した場合は、セーフモードで再起動できます。問題がより深刻な場合は、サポートに連絡するか、数回クリックするだけでWindowsを再インストールできます。

緊急モードはアクションに対してかなり長い応答を持つことができますが、視覚的にはかなり普通に見えます。日常的に使用しているユーザーがいるほど一般的です。しかし、Windowsにはより正確で快適な方法があるため、緊急事態以外の日常生活で使用する必要はありません。もちろん、これはRDPです。

-OSライセンスについてはどうですか？

通常、VPSホスティングは仮想サーバーを提供し、仮想サーバーで何をしているのかを見ません（リソースとトラフィックの性質によるダウンロードのみ）。つまり、理論的には、海賊版ソフトウェアを含むあらゆるソフトウェアをそこに置くことができます。しかし実際には、ホスティングテンプレートの1つを使用してOSを備えたマシンを作成する方が便利です。私たちの場合、すべてのホストがHyper-Vでホストされており、完全にライセンスされているため、Windows Server（またはグラフィカルインターフェイスのないWindows Server Core）のライセンスはすでに料金プランに含まれています。現在、他の一部のホスティング会社は、ライセンスのレンタルの月額料金で新しい「スター」をオープンしています。

-自分のOSイメージを持ち込むことはできますか？

はい、できます。2つのオプションがあります。 Hyper-Vの標準VHDXコンテナーを作成して、サポートに転送できます。この場合、ネットワーク接続をセットアップする段階で問題が発生する可能性があります。おそらく完成したイメージには、適切なネットワークカード用のドライバーがありません。接続してインストールする必要があります。それでも、これは、たとえば古いラップトップからオンラインにすべてを転送するための良い方法です。または、仮想サーバーにマウントされているISOイメージにサポートを転送し、そこから個人アカウントを使用して緊急モードを使用して、インストールを自分で完了することができます。

これらのサービスは、月額1,000ルーブルの料金プランでのみ提供されます。

-他のソフトウェアのライセンスについてはどうですか？

ホスティングでは通常、何がインストールされているかを確認できません（たとえば、特別なサポートを求めない限り、私たちは確認できません）。これはあなたの責任範囲であるため、そこに何があり、どのように機能するかは関係ありません。海賊版ソフトウェアのサポートが提供されていないこと、および何かが機能しない場合に注意してください。

-どのストレージが使用されていますか？

すべてのドライブはサーバーではSSDシェルフですが、HDDのある関税では、スピンドルハードドライブの速度に制限されています。複製-RAID5。いくつかのRAID 6が残っていますが、すでにかなりの数あります。ドライブの1つに障害が発生すると、レイドの再構築が始まります。この時点で停電が発生すると、データが失われます。この場合、各物理サーバーへの個別のケーブルルートを持つ2つの個別の電源と、マシン全体の技術的なバックアップ（個別のストレージへの選択的なアクセスなし）があります。ハッカー攻撃による損害はAIGで保証されています。
