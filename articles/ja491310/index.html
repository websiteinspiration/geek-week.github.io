自分でパスワードアーカイブをクラックする方法


アーカイブがあり、内容を抽出する必要がありますが、パスワードを忘れてしまいました。グーグルして、さまざまなプログラム（1つ、2つ、3つ）を信頼できますが、

RARアーカイブをクラックする唯一の方法はブルートフォースです。

初期データ


アーカイブ、パスワード辞書（％username％をアーカイブに入れる場合は、頻繁に使用するものをローカルにのみ追加することを忘れないでください！）

したがって、最初のバージョンのアルゴリズムは単純です。パスワードを使用してファイルを開き、壊れるまでアーカイブのパスワードを繰り返し処理します。


Goを使用することを忘れないでください。
パスワードファイル


dictFile, err := os.Open(dictionary)
if err != nil {
    log.Fatalln(err)
    }
defer dictFile.Close()

アーカイブ自体


zipr, err := zip.OpenReader(zipfile)
if err != nil {
    log.Fatal(err)
}

そして私たちの 素晴らしい アルゴリズム


scanner := bufio.NewScanner(dictFile)
for scanner.Scan() {
    pass := scanner.Text()
    for _, z := range zipr.File {
        z.SetPassword(pass)
        _, err := z.Open()
        //   
        if err == nil {
            println("[+] Found password")
            println("[+] Password = " + pass)
            os.Exit(0)
        }
    }

そして、パスワードがアーカイブにある場合-出来上がり！
