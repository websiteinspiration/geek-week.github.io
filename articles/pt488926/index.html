2. Criando bots inteligentes no Azure Bot


A interação do usuário com os aplicativos via chat usando texto, imagens ou fala pode ser implementada usando bots. Pode ser um diálogo simples de "pergunta-resposta" ou um bot complexo que permite que as pessoas interajam com os serviços de maneira "inteligente", usando correspondência de padrões, rastreamento de estado e métodos de inteligência artificial que estão bem integrados aos serviços existentes. Aprenda a criar um chatbot inteligente, integrando o QnA Maker e o LUIS.

O curso levará cerca de 3 horas.

3. Análise de texto com os Serviços de Linguagem Cognitiva do Azure


Aprenda a usar os Serviços de Linguagem Cognitiva para analisar texto, determinar intenções, descobrir tópicos adultos e processar consultas em linguagem natural.

Módulos do Curso:

  • Classificação e moderação de texto usando o Moderador de Conteúdo do Azure;
  • Adicionando lógica de diálogo a aplicativos usando o LUIS (Intelligent Speech Recognition Service);
  • Rastreando a tonalidade do diálogo no texto usando a API de análise de texto.

4. Curso de engenheiro de dispositivos de fronteira baseado em IA


As tecnologias de IA, tecnologias de nuvem e dispositivos de fronteira implantados como contêineres no modo CICD permitem transformar indústrias inteiras, criando ecossistemas com recursos de autoaprendizagem para uma indústria específica que abrange toda a cadeia de valor. Nossa tarefa é criar um conjunto de modelos e metodologias para implantar a IA em dispositivos de borda no contexto da nuvem. Neste esquema de treinamento, você receberá as seguintes informações:

  • Informações sobre a criação de soluções usando a Internet das coisas e a nuvem;
  • Visão geral da implantação de soluções de IoT em dispositivos de borda
  • Informações sobre a implementação de modelos para dispositivos de fronteira usando contêineres;
  • Aprenda a usar o DevOps para dispositivos de borda.

O curso foi desenvolvido em conjunto com a Universidade de Oxford.

Comece a treinar

5. Processamento e classificação de imagens com os Serviços Cognitive Vision do Azure


O Microsoft Cognitive Services fornece recursos internos para o uso da visão computacional nos aplicativos. Aprenda a usar o Cognitive Vision Services para detectar rostos, marcar e classificar imagens e identificar objetos.

Módulos do Curso:

  • Identifique rostos e emoções usando a API de visão computacional nos Serviços Cognitivos do Azure;
  • Processamento de imagens usando o serviço de visão computacional;
  • Classificação de imagens utilizando o serviço de reconhecimento visual do usuário;
  • Avaliação de requisitos para a implementação da API de reconhecimento visual do usuário.
