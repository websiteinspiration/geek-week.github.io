9 anuncios importantes de la conferencia Microsoft Build 2020

Nuestra conferencia de dos días Microsoft Build 2020, la transmisión de la pista principal que realizamos en Habré , ha llegado a su fin. En este artículo hablaremos sobre los anuncios más ambiciosos e interesantes del evento pasado. ¡Mira debajo del gato!

¿Y qué novedad te gustó más? Comparte en los comentarios.



Lanzamiento de Windows Terminal 1.0


¡Nos complace anunciar el lanzamiento de Windows Terminal 1.0! Windows Terminal ha recorrido un largo camino desde su anuncio en Microsoft Build 2019 . Como siempre, puede descargar Windows Terminal desde Microsoft Store o desde la página de lanzamiento en GitHub . Windows Terminal tendrá actualizaciones mensuales a partir de julio de 2020.

Ayer hablamos con más detalle sobre la nueva terminal en este artículo sobre Habr.



Creó una de las 5 supercomputadoras más poderosas del mundo.


La nueva supercomputadora de Microsoft se  desarrolló  sobre la base de la nube de Azure en  colaboración  y exclusivamente para  OpenAI , una organización de investigación fundada por Elon Musk y Sam Altman. Su capacidad es de más de 285,000 núcleos de procesador (núcleo de CPU), 10,000 procesadores gráficos (GPU), y la velocidad de conexión de red es de 400 gigabits por segundo para cada servidor. Está diseñado específicamente para entrenar a la próxima generación de modelos de IA distribuidos masivos que pueden realizar una gama más amplia de funciones y, en consecuencia, requieren significativamente más recursos informáticos.

Lea los detalles en nuestro sitio de noticias (en ruso).



Vista previa del Administrador de paquetes para Windows


El administrador de paquetes está diseñado para ayudarlo a ahorrar tiempo. De hecho, es un conjunto de herramientas de software que ayudan a automatizar el proceso de descarga de software a su computadora. Usted indica qué aplicaciones desea instalar, busca la última versión (o la que especificó) y la instala en la computadora.

Casi todos los desarrolladores querían tener un administrador de paquetes nativo en Windows. Este día finalmente ha llegado. Una de las buenas noticias es que es de código abierto. Ahora es muy simple, por ejemplo, winget install terminal, luego winget install powershell, y luego winget install powertoys.

Lea más sobre Windows Package Manager en nuestro blog .



Actualizaciones de marco fluido


La plataforma de colaboración web interactiva Fluid Framework,  presentada hace un  año en Build 2019, se está  convirtiendo en  un software de código abierto. Y también, pronto la funcionalidad de Fluid Framework estará parcialmente disponible no solo para los desarrolladores, sino también para los usuarios finales: sus componentes y entornos de trabajo aparecerán en Office.com y en la versión web de Outlook.

La primera integración de Fluid Framework en Microsoft 365 permitirá que varios usuarios realicen cambios simultáneos en los documentos y creen componentes conectados que se puedan compartir de forma instantánea y muy fácil en las aplicaciones. Por ejemplo, las tablas, gráficos y listas de tareas se pueden pegar en la versión web de Outlook para que sus datos de ventas, tareas de proyectos e informes de investigación se actualicen constantemente para reflejar los últimos cambios. En Office.com, puede crear y administrar espacios de trabajo de Fluid Framework, incluso en el flujo de actividad de su documento, en la lista de recomendaciones, en @ menciones o buscarlos en Office.com. Y dado que los componentes de Fluid Framework son de bajo peso, las ediciones se aplican al instante.

Un entorno de red fluido puede transformar instantáneamente su aplicación en una herramienta de colaboración. Incluye estructuras de datos de sincronización de baja latencia y un servicio de conmutación para conectar puntos finales. Si reemplaza estructuras de datos estáticos con estructuras de datos Fluid, su aplicación comenzará a admitir instantáneamente la colaboración en tiempo real.

Es posible liberar todo el potencial de Fluid Framework al crear una comunidad de desarrolladores multicultural, abierta y activa. Por esta razón, Microsoft convertirá el Fluid Framework en un entorno de código abierto, lo que permitirá a los desarrolladores aprovechar el Fluid Framework principal en sus propias aplicaciones. Lanzamos documentación y herramientas adicionales, y también invitamos a los desarrolladores a trabajar con Microsoft en la creación y lanzamiento de Fluid Framework.

Y todo esto mientras se mantiene la productividad.

Microsoft Edge: mejoras para desarrolladores


Microsoft Edge presenta una serie de nuevas características para mejorar los flujos de trabajo para consumidores, desarrolladores y clientes comerciales.

En primer lugar, esta es una forma más rápida y contextual de buscar usando la barra lateral. Simplemente seleccione una palabra o frase, haga clic derecho y seleccione "buscar en la barra lateral" en el menú contextual. Los resultados se mostrarán en el panel en el lado derecho de la ventana del navegador, para que el usuario no pierda el contexto del contenido original.

Para los desarrolladores, hemos contribuido al proyecto Chromium, incluidas las mejoras de accesibilidad, escritura a mano, desplazamiento y localización. Las versiones de vista previa de Webview 2.0 también están disponibles para el desarrollo con .NET y UWP. Y gracias a las nuevas mejoras en la Progressive Web App (PWA) (en vista previa), estas aplicaciones se sienten más nativas, con una interfaz de usuario más natural para compartir, notificaciones y más.



Iniciativas educativas para desarrolladores


Nos introdujimos  nuevos módulos de formación gratuita para el Microsoft Aprender plataforma, lo que permitirá a los desarrolladores reciben capacitación en el trabajo con la computación cuántica utilizando el lenguaje de programación #Q y el Kit de Desarrollo de Quantum. También  lanzaremos  el programa diario Learn TV para desarrolladores, que incluirá programación en tiempo real y varias discusiones con expertos.



Lanzar Project Reunion: una nueva forma de desarrollar aplicaciones de Windows


Los usuarios pasan un total de más de 4 billones de minutos al mes en Windows 10, es decir, un 75% más que el año pasado. Presentamos Project Reunion para ayudar a los desarrolladores a crear aún más innovación para mil millones de dispositivos activos con Windows 10 por mes. La iniciativa tiene como objetivo proporcionar una fácil integración entre las interfaces de programación de Win32 y la Plataforma universal de Windows, proporcionando una única plataforma de desarrollo con soporte de compatibilidad con versiones anteriores.

Lea más en esta publicación en nuestro blog (en inglés).



Herramientas para el uso responsable del aprendizaje automático


Ayudarán a los desarrolladores a comprender mejor el comportamiento de los modelos de aprendizaje automático, controlarlos y garantizar la confidencialidad de los datos durante todo el ciclo de trabajo con ellos, lo que le permitirá corregir problemas como, por ejemplo, conjuntos de datos no representativos, inseguridad de los datos utilizados y mucho más. Esto ayudará a proporcionar un enfoque más responsable y ético para el desarrollo de algoritmos. La nueva funcionalidad estará disponible en Azure Machine Learning y como código abierto en GitHub.
