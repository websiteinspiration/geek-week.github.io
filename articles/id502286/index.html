Robot api

Saat menyebutkan "robot api", kebanyakan dari kita membayangkan robot yang akan dibawa oleh petugas pemadam kebakaran dan mengirimkannya ke neraka, di mana tidak satu pun pemadam kebakaran bisa masuk. Yaitu, robot itu begitu dibalut dengan baju besi, di trek yang tidak takut api dan nyala ... Singkatnya, "superman" di antara robot, pahlawan untuk pencuci piring dan pembuat kopi pintar.

Tetapi ada robot api lain, atau lebih tepatnya sistem api robot, yang dipasang di tempat untuk memadamkan api pada tahap awal. Itu tentang mereka dan akan dibahas.

Kompleks semacam itu dipasang setelah konstruksi atau perbaikan bangunan - mereka adalah sistem pemadam api standar utama. "Kecerdasan buatan" diperlukan agar tidak menghitung seberapa banyak yang sia-sia dan memadamkan api, dan tidak membanjiri seluruh ruangan. Kompleks seperti itu menguntungkan untuk dipasang di kamar besar - dengan luas lebih dari 5-10 ribu km. m dan dengan langit-langit di atas 8 m (meskipun ada pengecualian).

Anda tentu saja dapat menginstal sistem lain. Tetapi mereka memiliki kelemahan yang signifikan, dan karenanya rumit dan mahal (di kamar-kamar ini):

  • Sistem otomatis air dan busa. Jika terjadi kebakaran, ruangan dibanjiri dengan air atau busa dari langit-langit (alat penyiram dipasang di sana). Kami mewakili mereka dengan baik dalam film - pahlawan muncul lebih ringan dan seluruh ruangan mulai diairi, dan mereka yang mandi di dalamnya. Kekurangan di wajah - semuanya banjir. Tidak hanya kebakaran, tetapi juga peralatan, barang, furnitur, kabel listrik. Semua ini paling baik mengering, paling buruk ke tempat pembuangan sampah (kadang-kadang, seperti film di mana sistem ditembak).
    Kedua, air (termasuk untuk menghasilkan busa) harus diambil dari suatu tempat. Dari pasokan air kamu tidak bisa kemana-mana. Maka Anda harus memasang dekorasi yang indah dari wilayah - kapasitas seratus meter kubik.
    Ketiga, sistem seperti itu menciptakan beban besar di atap karena banyak pipa. Dan di langit-langit Anda masih harus menggantung pencahayaan, ventilasi, jangan lupakan salju ... Yaitu, selamat tinggal pada atap yang mudah dan murah ...



  • Arti dari pemadaman api gas adalah mengisi ruangan dengan gas dan memindahkan oksigen darinya. Untuk ruangan besar, ini hampir mustahil dilakukan, dan orang harus dievakuasi.

Sistem robot api, atau jika lebih sederhana, ini adalah kereta api stasioner (dengan kata lain, perangkat untuk memasok jet air atau busa), yang dapat berputar di sekitar porosnya dan memasok air naik turun 180 derajat. Sensor IR dipasang pada robot untuk mengarahkan jet ke api. Ditambah sistem kontrol dalam bentuk pengontrol. Robot semacam itu dapat dipasang baik di lantai, dan pada kolom pendukung, langit-langit atau menara khusus - untuk memadamkan, yang nyaman.



Kerugian utama dari robot adalah stasioner - mereka tidak bisa bergerak di ruang angkasa. Beberapa robot semacam itu harus ditempatkan di tempat itu. Menurut pengalaman kami, dibutuhkan 6 unit per bengkel 33 x 105 m, yang mahal.

Kami, perusahaan "Sistem Penyelamatan Otomatis", muncul dengan gagasan, "menawan dengan hal-hal baru" - untuk memasang robot api semacam itu pada kereta. Otomatis tentu saja.

Dalam kasus kami, "perlindungan ganda" diperoleh - pertama sensor stasioner harus "menangkap" api, kemudian sensor IR robot harus melokalkannya. Jika Anda memasang sensor sensitif di dalam ruangan, maka tidak akan ada pemadam yang salah. Tetapi ini memungkinkan Anda untuk mencegah kebakaran pada tahap pembusukan.

Misalnya, di bengkel atau gudang tertentu, kabel mulai membara di malam hari. Sensor gas (yang mencatat kenaikan level CO) telah mendeteksi ini. Robot api kami (2 pcs). Tiba di sinyal, tetapi tidak mendeteksi api. Jadi, tidak perlu mengisi dan ruangan itu aman untuk orang. Dalam mode siaga, robot berjalan di sepanjang rel, ruang pemindaian (untuk berjaga-jaga). Dan petugas mengunjungi ruangan - mereka memeriksanya, menangani penyebab alarm dan, sebagai akibatnya, menghilangkan energi kabel yang membara, yang diganti pada hari kerja berikutnya oleh giliran tugas tukang listrik. Praktis tidak ada kerusakan. Kalau tidak, setelah beberapa jam membara, api akan muncul, itu akan menyebar ke ruangan. Dan hanya dengan begitu sistem api akan bekerja. Dan akan membanjiri segalanya ...

"Sistem penyelamatan otomatis" mengembangkan sistem pemadaman api ini pada robot seluler dari awal. Saat ini, sampel kerja telah dikumpulkan untuk pengujian.
