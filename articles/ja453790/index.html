私の名前はAndrey Churanovです。私はUniSenderカスタマーエクスペリエンスマネジメントチームのリーダーです。

UniSenderは、シンプルなメールおよびSMSメーリングサービスです。私たちは、マーケティング担当者が美しい手紙を作成し、それらを直列に組み合わせ、テストを実施し、マーケティングの効果を監視するのを支援します。

毎日、クライアントと会社の間のすべての接点でカスタマーエクスペリエンスを調査します。私たちが使用するメトリックは、多くのSaaSに近く、理解可能です。

SaaS製品の顧客チャーンを測定する機能


流出について話すときは、まず最初に、2つの質問に答える必要があります。

  1. 私たちの顧客基盤はどれくらい安定していますか？理想的には、月初と月末の顧客数は一定です。
  2. 退職した顧客の割合は？理想的にはゼロ。

これらの質問への回答には、2つの重要な指標が役立ちます。

顧客維持率（CRR）。次の式で

計算できます：チャーン率（CR）。私たちはこれを信じています。

これらのメトリックを使用して、会社のサービスを使用しなくなった顧客の割合を計算できます。計算式は新しい顧客を考慮に入れます-このようにして、ベースの急速な成長の影響を任意の期間で相殺できます。合計で、CCRとCRは100％になります。

例を使用してこれらのメトリックを読み取る方法：

与えられた：

  • 期間の開始時の顧客= 100。
  • 期末の顧客= 105。
  • 新規顧客= 10。

顧客維持率を検討します。

  • CCR =（105-10）/ 100 = 95％。この例では、指標が100％未満であるため、顧客数が減少していることがわかります。

与えられた：

  • 期間の開始時の顧客= 100。
  • 出発されたお客様の数= 5。

解約率を検討します。

  • CR = 5/100 = 5％。これは流出の指標です。それが特定の会社や業界に大きく依存するのか、それとも少し依存するのか。

顧客維持率と解約率の何が問題になっていますか


CCRとCRの計算に関する問題は、流出の修正です。SaaS製品の使用を停止するには、ほとんどの場合、残高を補充する必要はありません。そして、クライアントが一時的にそれを行った場合、数か月後に戻りますか？この場合、CCRとCRの計算は不正確になる可能性があります。

これが特徴です！

クライアントが永遠に去ったのか、それとも一時的な非活動なのかを示す行動の仮説を選択する必要があります。

顧客の流出をより正​​確に計算するために私たちが使用するメトリックを教えてください。

プロセスパフォーマンスメトリック


これらの測定基準は、流出管理作業がどれだけ成果を上げているかを理解するのに役立ちます。

アクティブな顧客シェア


各企業には「アクティブな顧客」という独自のコンセプトがあることにすぐに気づきます。たとえば、UniSenderは、請求期間中にニュースレターを作成し、サービスでアカウントを補充したアクティブなユーザーを検討します。クライアントがアクティブになるために必要なアクションを理解する必要があります。その後、会社のすべての顧客からアクティブな顧客のシェアを計算できます。

なぜか。「顧客の何パーセントがサービスを利用するのか」という質問に答える方が簡単です。このシェアが上昇または下降し始めた場合、対応する必要があります。変化している顧客のセグメントを見つけ、それをどうするかを決定します。

このような期間のアクティブな顧客の割合を考慮することをお勧めします。

  • 1か月のアクティブな顧客（月間アクティブユーザー、MAU）。
  • 2か月間アクティブユーザー（2か月アクティブユーザー、2MAU）。
  • 四半期中にアクティブなクライアント（Quarter Active Users、QAU）。

アクティブな顧客はお金をもたらします。非アクティブ-これは流出です。

アクティビティが異なる4つのクライアントの例でMAU、2MAU、QAUを計算する方法（ "1"は、クライアントが今月アクティブだったことを意味します）：



  • MAU = 3/4 = 75％。
  • 2MAU = 2/4 = 50％。
  • QAU = 1/4 = 25％。

何を探すか。増加する顧客活動に集中する必要があるのは当然です。まず最初に、非活動の原因が調査され、各期間でそれらは異なる場合があります。たとえば、UniSenderでは、顧客は次の理由で製品の使用を停止します。

  • メールマーケティングが効果的でないことを考慮してください。
  • 新しいメーリングリストのニュースレターはありません。
  • 郵送に従事する時間/専門家はありません。

お客様がメールマーケティングを拒否する主な理由については、こちらの記事をご覧ください。

また、活動につながるドライバーを研究する必要があります。たとえば、UniSenderでは、アクティビティドライバーはサービスでのクライアントのアクション（オフィスへのログイン、レターテンプレートの作成、ニュースレターの作成、ニュースレターの送信）です。これらのアクションの後に、多くの場合、口座残高が補充されます。つまり、クライアントに、サービスをすぐに販売するのではなく、できるだけ頻繁に「触れる」ように勧める必要があります。

返品された顧客収入


ユーザーがサービスを辞めると判断し、なんらかのチャネルでユーザーに連絡しました。クライアントは協力を続けることに同意し、支払いました。

返品された顧客からの収入は、いずれかのチャネルでのコミュニケーション後に顧客が支払った金額です。

なぜか。この指標は、平均請求額を計算するために使用されます。平均チェックは、クライアントごとに個別に、一般にすべての流出通信の後で検討する必要があります。

何を探すか。この指標の分析の別のカットは、通信チャネル（電話、電子メール、個人的な会議）とクライアントセグメントです。複数のインジケーターでセグメント化できます。

  • 以前にどんな収入がありましたか？
  • 最後の入金はいつですか？
  • 以前に使用したサービスは何ですか？

たとえば、サービスを終了して電話をかけた顧客を強調表示しました。当然、届かなかったお客様もいらっしゃると思います。ダイヤルではなく、顧客にダイヤルした後の収益額を比較します。アウトフローセグメントの選択が正しく行われた場合、ダイヤル後の収入額は、顧客にダイヤルした後の額を大幅に超えるはずです。

連絡後30日、60日、90日後の支払回数


支払回数は、プロセスの安定性と流出管理戦略の正しい選択を特徴付けます。

顧客をサービスに戻したとします。長い休憩の後、彼は彼のバランスを補充しました。それは良いです！次は何？あなたは未来を支配していますか？これは必要だと思います。

なぜか。通常、連絡した返品された顧客からの定期的な支払いのシェアは、連絡がなかった顧客からの定期的な支払いの割合を大幅に超える必要があります。この傾向は、適切な戦略を選択したことを裏付けています。

この指標を確認するには、最初の入金後にクライアントが再度支払いを行ったかどうかを確認するだけです。

UniSenderの生活の例。

1.四半期のサンプルを作成します。ダイヤル後とそれなしの支払い回数を調べます。ダイヤル後の支払いのシェアが高くなっています。私たちの戦略は正しいです-サービスに戻るというクライアントの決定に影響を与えます。


ダイヤル後の支払いなし

2.次に、顧客の何パーセントが複数の支払いを行ったかを確認します。


ダイヤル後、顧客は複数の支払いを行うことがよくあり

ます。3.財務指標を修正します。ダイヤル後とそれなしの平均的な顧客チェックを比較します。支払い回数の違いは明らかです。また、返品後すでに4回の支払いを行っているお客様は、平均小切手が高いことがわかります。私たちの場合、そのようなクライアントのシェアは小さい（約3％）ので、傾向がある可能性があります。

ます。クライアントが戻った場合、彼は自分が残した問題が解決されたと信じていました。

指標が成長するには、顧客の流出の理由を分析し、流出の理由に基づいて保持ツールを開発する必要があります。

たとえば、メール配信が不十分だったために、クライアントがメールニュースレターのサービスを終了したとします。私たちはお客様に連絡し、配達を改善する方法についての推奨事項を提供します。彼の帰国後、私たちは彼が私たちのアドバイスを使用したかどうかを確認する必要があります。指標を上げることができない場合は、デリバリーを増やす方法にもう一度取り組んでいます。これが行われない場合、クライアントは再びネガティブな体験を得てサービスを放棄するリスクを負います。

プロセス品質指標


品質指標はバランスの取れたプロセスを示します。たとえば、返品された顧客から10,000ドルを受け取った場合、これは良いことですか、悪いことですか？そして、もしある顧客がこのお金を持ってきたら？そして10,000人の顧客なら？以下のインジケーターを使用すると、プロセスに歪みがあるかどうかを確認できます。

平均チェック


返品された顧客の平均請求書を計算するには、連絡があった顧客からの収入を、連絡後に支払いを行った顧客の数で割る必要があります。

平均値の計算にはいくつかの欠点があります。たとえば、値に大きなばらつきがある配列の場合、平均は歪んでいます。数値100、200、および–300の場合、算術平均は0になるため、常に解釈できるとは限りません。したがって、標準偏差と中央値を測定することをお勧めします。

標準偏差は、各指標がサンプルの平均値から平均して何単位ずれているかを示します。計算には、MS Excelの式を使用します。



中央値は、サンプルを2つの等しい部分に分割します。観察の半分は中央値よりも下にあり、残りの半分は高くなっています。 MS Excelでは式を使用します。



何を探すか。平均請求額を増やすための良い方法（関税の変更はカウントされません）は、源泉徴収ツールを接続することです。顧客への返品に関する会社の申し出は、役に立ち、理解できるものでなければなりません。そして最も重要なこと-クライアントが遭遇した問題の解決策を含みます。

たとえば、当社のオファーの1つには、個人的な専門家のアドバイスが含まれています。このツールを使用すると、クライアントの声を聞いて聞くことができます。彼は私たちです。私たちは追加の価値を提供し、その後、クライアントは最小ではなく、以前の支払いを支払う準備ができています。

変換


コンバージョン-コミュニケーションがあったすべての顧客のうち、サービスに戻った顧客の割合。

何を探すか。つまり、「流出していない」顧客を処理しないでください。これを行うには、流出に関する顧客の行動を常に分析する必要があります。

たとえば、UniSenderは、サービス内でアカウントとメールの補充があった期間を調べます。私たちにとって、これら2つの基準を組み合わせることで、発信する顧客を決定できます。

返品について話し合った顧客のコンバージョン率を、コミュニケーションがない場合のコンバージョンと比較することをお勧めします。 2番目のケースの変換の方が高い場合、アクションは間違った顧客を対象としています。お客様は私たちの努力なしで支払います。したがって、これらは発信顧客ではありません。

このようなアクションは、会社のリソースを節約し、サービスを辞めた顧客に集中するのに役立ちます。

お客様の声


お客様の声は、お客様が戻った後にサービスに満足しているかどうかを理解するのに役立ちます。当然、この見積もりを上げることに焦点を当てるべきです。

私たちはどう思いますか：

  • CSAT（顧客満足度指数）。これは、会社との連絡先（電話、電子メール、または販売時点）で顧客がサービスにどのように満足しているかを示しています。たとえば、UniSenderでは、顧客は通信の直後にテクニカルサポートの品質を評価します。CSAT UniSender 95が100％になりました。
  • CES（Customer Efforts Score）。クライアントがサービスを使用することがいかに便利かを示します。私たちの質問は、「サービスへの登録はどれほど簡単でしたか？」です。
  • NPS（ネットプロモータースコア）。顧客ロイヤルティインデックス。顧客間のブランドプロモーターの割合を示します。たとえば、「会社での経験に基づいて、友人や知人に勧める準備はできていますか？」NPSの範囲は-100〜100です。UniSenderには+41があります。

私たちは、教えてあなたはどのように顧客からのフィードバックを収集するために、どのようにそれがUniSenderで動作します。

何を探すか。低いスコアは、不満のより詳細な研究につながるはずです。分析と変更は、特定の従業員、従業員グループ、保持ツール、または会社の製品に関連している必要があります。

フィードバックを収集するプロセスは一定である必要があり、その分析は周期的です。プロセスの品質と顧客満足度を向上させるための主なツールは次のとおりです。

  • フィードバックの永続的なコレクション。
  • 測定の周期性。
  • 実装された変更の有効性を改善および監視するための提案のコレクション。

流出を測定するセクション


固定間隔


UniSenderは、日ごと、週ごと、四半期ごと、年ごとなどの間隔で流出を測定しています。

できるだけ頻繁に指標を測定することをお勧めします。サービスでは新しいアクションが毎日発生します。したがって、それらを制御する必要があります。

顧客セグメント


セグメントによってインジケータが異なるのは正常です。コンバージョン数と平均請求額では、目標を設定しています。

セグメンテーションは、流出管理コストも最適化できます。たとえば、収益性の高い顧客には電話を使用し、収益性の低いクライアントには自動メッセージチェーンを設定します。

通信チャネル


電話、Viberでのメッセージ、メールなど、どのチャネルが顧客をより適切に保持しているかを判断します。チャネルごとに、異なる流出メトリックを取得します。
