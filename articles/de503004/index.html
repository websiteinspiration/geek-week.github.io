Windows Terminal 1.0 Release


Wir freuen uns, die Veröffentlichung von Windows Terminal 1.0 bekannt zu geben! Windows Terminal hat seit seiner Ankündigung bei Microsoft Build 2019 einen langen Weg zurückgelegt . Wie immer können Sie Windows Terminal aus dem Microsoft Store oder von der Release-Seite auf GitHub herunterladen . Windows Terminal wird ab Juli 2020 monatlich aktualisiert.

Wir haben gestern in diesem Artikel über Habr ausführlicher über das neue Terminal berichtet .



Erstellt einen der 5 leistungsstärksten Supercomputer der Welt


Der neue Microsoft - Supercomputer wurde  entwickelt  auf der Basis des Azure Cloud in  Zusammenarbeit  und ausschließlich für  OpenAI , eine Organisation Forschung gegründet von Elon Musk und Sam Altman. Die Kapazität beträgt mehr als 285.000 Prozessorkerne (CPU-Kern), 10.000 Grafikprozessoren (GPUs) und die Netzwerkverbindungsgeschwindigkeit beträgt 400 Gigabit pro Sekunde für jeden Server. Es wurde speziell für das Training der nächsten Generation massiver verteilter KI-Modelle entwickelt, die einen größeren Funktionsumfang erfüllen können und dementsprechend erheblich mehr Rechenressourcen erfordern.

Lesen Sie die Details auf unserer Nachrichtenseite (in russischer Sprache).



Vorschau des Paketmanagers für Windows


Der Paketmanager soll Ihnen helfen, Zeit zu sparen. Tatsächlich handelt es sich um eine Reihe von Softwaretools, mit denen das Herunterladen von Software auf Ihren Computer automatisiert werden kann. Sie geben an, welche Anwendungen Sie installieren möchten, suchen nach der neuesten Version (oder der von Ihnen angegebenen) und installieren sie auf dem Computer.

Fast jeder Entwickler wollte einen nativen Paketmanager unter Windows. Dieser Tag ist endlich da. Eine der coolen Neuigkeiten ist, dass es Open Source ist. Jetzt ist es sehr einfach, zum Beispiel Winget-Installationsterminal, dann Winget-Installations-Powershell und nach Winget-Installation Powertoys.

Weitere Informationen zum Windows Package Manager finden Sie in unserem Blog .



Fluid Framework-Updates


Die vor einem  Jahr auf der Build 2019 eingeführte interaktive Web Collaboration-Plattform Fluid Framework  wird  zu  Open Source-Software. In Kürze wird die Funktionalität des Fluid Framework nicht nur Entwicklern, sondern auch Endbenutzern teilweise zur Verfügung stehen: Die Komponenten und Arbeitsumgebungen werden auf Office.com und in der Webversion von Outlook angezeigt.

Durch die erste Integration des Fluid Framework in Microsoft 365 können mehrere Benutzer gleichzeitig Änderungen an Dokumenten vornehmen und verbundene Komponenten erstellen, die sofort und sehr einfach in Anwendungen freigegeben werden können. Beispielsweise können Tabellen, Diagramme und Aufgabenlisten in die Webversion von Outlook eingefügt werden, sodass Ihre Verkaufsdaten, Projektaufgaben und Forschungsberichte ständig aktualisiert werden, um die neuesten Änderungen widerzuspiegeln. Auf Office.com können Sie Fluid Framework-Arbeitsbereiche erstellen und verwalten, einschließlich im Aktivitätsdatenstrom Ihres Dokuments, in der Liste der Empfehlungen, in @ Erwähnungen oder auf Office.com danach suchen. Und da die Komponenten des Fluid Frameworks ein geringes Gewicht haben, werden Änderungen sofort angewendet.

Eine flüssige Netzwerkumgebung kann Ihre Anwendung sofort in ein Tool für die Zusammenarbeit verwandeln. Es enthält Synchronisationsdatenstrukturen mit geringer Latenz und einen Vermittlungsdienst zum Verbinden von Endpunkten. Wenn Sie statische Datenstrukturen durch Fluid-Datenstrukturen ersetzen, unterstützt Ihre Anwendung sofort die Zusammenarbeit in Echtzeit.

Das volle Potenzial des Fluid Frameworks freizusetzen ist möglich, wenn eine multikulturelle, offene und aktive Community von Entwicklern geschaffen wird. Aus diesem Grund wird Microsoft das Fluid Framework in eine Open Source-Umgebung verwandeln, sodass Entwickler das zentrale Fluid Framework in ihren eigenen Anwendungen nutzen können. Wir veröffentlichen zusätzliche Dokumentationen und Tools und laden Entwickler ein, bei der Erstellung und Veröffentlichung des Fluid Framework mit Microsoft zusammenzuarbeiten.

Microsoft Edge: Verbesserungen für Entwickler


Microsoft Edge führt eine Reihe neuer Funktionen ein, um die Arbeitsabläufe für Verbraucher, Entwickler und gewerbliche Kunden zu verbessern.

Erstens ist dies eine schnellere und kontextbezogenere Möglichkeit, über die Seitenleiste zu suchen. Wählen Sie einfach ein Wort oder eine Phrase aus, klicken Sie mit der rechten Maustaste und wählen Sie im Kontextmenü "In der Seitenleiste suchen". Die Ergebnisse werden im Bereich auf der rechten Seite des Browserfensters angezeigt, damit der Benutzer den Kontext des ursprünglichen Inhalts nicht verliert.

Für Entwickler haben wir zum Chromium-Projekt beigetragen, einschließlich Verbesserungen bei Barrierefreiheit, Handschrift, Bildlauf und Lokalisierung. Vorschauversionen von Webview 2.0 sind auch für die Entwicklung mit .NET und UWP verfügbar. Dank neuer Verbesserungen der Progressive Web App (PWA) (in der Vorschau) fühlen sich diese Anwendungen nativer an und bieten eine natürlichere Benutzeroberfläche für die Freigabe, Benachrichtigungen und mehr.



Bildungsinitiativen für Entwickler


Wir eingeführt  neue kostenlose Trainingsmodule für die Microsoft - Plattform erfahren, die es Entwicklern ermöglicht Ausbildung zu bekommen auf die Arbeit mit Quanten Berechnung der # Q - Programmiersprache und die Quantum Development Kit verwenden. Wir werden auch   das tägliche Programm Learn TV für Entwickler starten , das Echtzeitprogramme und verschiedene Diskussionen mit Experten umfasst.



Starten Sie Project Reunion - Eine neue Möglichkeit, Windows-Anwendungen zu entwickeln


Benutzer verbringen insgesamt mehr als 4 Billionen Minuten pro Monat in Windows 10 - das sind 75% mehr als im Vorjahr. Einführung von Project Reunion, um Entwicklern dabei zu helfen, noch mehr Innovationen für 1 Milliarde aktive Windows 10-Geräte pro Monat zu entwickeln. Die Initiative zielt auf eine einfache Integration zwischen Win32- und Universal Windows Platform-Programmierschnittstellen ab und bietet eine einzige Entwicklungsplattform mit Unterstützung für Abwärtskompatibilität.

Lesen Sie mehr in diesem Beitrag in unserem Blog (auf Englisch).



Werkzeuge für den verantwortungsvollen Umgang mit maschinellem Lernen


Sie helfen Entwicklern dabei, das Verhalten von Modellen für maschinelles Lernen besser zu verstehen, sie zu steuern und die Vertraulichkeit der Daten während des gesamten Arbeitszyklus sicherzustellen. So können Sie Probleme wie beispielsweise nicht repräsentative Datensätze, Unsicherheit der verwendeten Daten und vieles mehr beheben. Dies wird dazu beitragen, einen verantwortungsbewussteren und ethischeren Ansatz für die Entwicklung von Algorithmen bereitzustellen. Neue Funktionen werden in Azure Machine Learning und als Open Source auf GitHub verfügbar sein.
