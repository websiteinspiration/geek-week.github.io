キャンバス2Dの事前レンダリング

キャンバスで作成された2Dアニメーションを最適化しようとして、興味深いオプションが1つ見つかりました。



予備的な視覚化-事前レンダリング。


「すべてのフレームを事前に記録し、アニメーションの終了後にそれらを表示するとどうなるでしょうか？」-私たちは友人と考えました、そしてそれは翌日起こったことです。


ライブデモ


amedomary.ru/prerender-canvas


コードv1.0


ページ自体はGithubにあります。


ロジックは非常にシンプルで、2つのステップで実装されます。


ステップ1-アニメーションの各フレームを画像の配列に保存します


ステップ1-JavaScriptコード
  function renderLoop() {
    draw();

    // create prerender images
    canvasPrerender.toBlob(blob => {
      const reader = new FileReader();
      reader.readAsDataURL(blob);

      reader.onload = function() {
        const image = document.createElement("img");
        image.src = reader.result;
        prerenderArr.push(image);
      };
    });

    if (prerenderIsCompleted) {
      requestAnimationFrame(renderLoop);
    } else {
      requestAnimationFrame(drawPreImg);
    }
  }

    ctx.clearRect(0, 0, canvas.width, canvas.height);
