前書き

enumタイプエイリアス（type）を介した列挙オプション（）へのリンク


Rust 1.37.0以降、列挙型オプション（enum）への参照は、型エイリアスを通じて可能になりました。


type ByteOption = Option<u8>;

fn increment_or_zero(x: ByteOption) -> u8 {
    match x {
        ByteOption::Some(y) => y + 1,
        ByteOption::None => 0,
    }
}

これはimplブロックのSelfタイプエイリアスとして機能するため、Rust 1.37.0では、構文構文を使用して列挙型オプションを参照できるようになりましたSelf::Variant。


impl Coin {
    fn value_in_cents(&self) -> u8 {
        match self {
            Self::Penny => 1,
            Self::Nickel => 5,
            Self::Dime => 10,
            Self::Quarter => 25,
        }
    }
}

か、錆があなたを介して転送のバリエーションを参照することができ、今ある「タイプ相対の解像度です」、<MyType<..>>::Variant。詳細については、安定化レポートをご覧ください。
