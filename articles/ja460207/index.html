Linux管理者がUbuntu 16.04で音楽認識システムを使用して児童合唱団を管理した方法。マイクとコード

夏は何を楽しむ必要がありますか？音楽が必要です！しかし、音楽が紙のメモで録音されており、あなたが普通のシステム管理者であり、子供たちの合唱団を作成することを任されている場合、この問題の解決策の1つとして、いくつかの便利なプログラムを紹介します。これらはJavaで記述されており、Ubuntu 16.04リポジトリとGithubの両方で利用でき、ノートのマシン認識の問題を解決し、音楽パーツを作成し、結果の音楽製品をダウンロードして一般的な形式に編集し、個々のアーティストにファイル（およびスラップ）を配布します。このトピックは、私たちのブログの読者にとって興味深いものになると思います。





児童キャンプの実際の状況を想像してみてください。合唱団をすばやく組み立てて子供たちに新しい歌を教える必要がありましたが、人員配置表で規定された音楽監督の役職を希望する人はいませんでした。ローカルシステム管理者に尋ねた。そしてそれは行きました！

Ubuntu 16.04がインストールされたラップトップが1台ある場合、管理者は次のように問題を解決しました。

3.しかし、私たちは21世紀に住んでいます。新しい何かを歌ってみることにしました。しかし、プログラムで歌って楽しんでいると、誰もノートを知らないことがわかりました。では、Linux管理者は何をするのでしょうか？

A. OCR TESSERACT認識システムを配置します。たとえば、ここから：https : //github.com/tesseract-ocr/tesseract/wiki。注意してください、4.0ではなく、以前のバージョン-3.5をインストールする必要があります。
音楽作品と音符で正確に使用されている言語（英語、イタリア語、ドイツ語、フランス語、ロシア語）のインストールを確認してください。

B.素晴らしいノート認識システムをインストールします！唯一の制限は、300DPIスキャン、3.5 MB以下、PDFで1ページです。
あなたはここからそれを置く必要があります：

$> git clone https://github.com/Audiveris/audiveris.git
cd audiveris
gradlew.bat clean build
./gradlew clean build
./gradlew run
./gradlew run -PcmdLineArgs="your,arguments,here"
