CSSに乱数はありますか？



CSSを使用すると、インターネット上で動的なレイアウトとインターフェースを作成できますが、マークアップ言語としては静的なので、値を設定した後で変更することはできません。チャンスのアイデアは議論されていません。実行時の乱数の生成は、CSSではなくJavaScriptの領域で​​す。

か否か？少しのユーザー操作を許可すると、CSSである程度のランダム性を実際に生成できます。見てみよう！

ランダムネスin other languages


CSSのトリック に関する記事で Robin Rendleが説明しているように、CSS変数を使用して「動的ランダム化」を行う方法はいくつかあります。ただし、これらのソリューションは、CSS変数を新しいランダムな値で更新するためにJavaScriptを必要とするため、100％CSSではありません。

SassやLessなどのプリプロセッサを使用してランダムな値を生成できますが、CSSコードがコンパイルされてエクスポートされるとすぐに、値が固定されてランダム性が失われます。

CSSのランダムな値が気になるのはなぜですか？


過去には、私のようなシンプルなCSS専用アプリケーション開発クイズ、CSサイモンのゲーム、そして手品を、私は、より複雑な何かをやってみたかったです。

そこで、ゲーム「リーラ」（「蛇とはしご」でもあります）の開発を始めました。

これはシンプルなゲームです。要するに、ゲームの目標は、ヘビを避け、階段を登ろうとすることで、チップをボードの最初から最後まで進めることです。

プロジェクトは実現可能のように見えましたが、何かを見逃しました。
サイコロを投げる（他のゲームでは、コインを投げる）は、ランダムな値を取得するための一般的に受け入れられている方法です。サイコロを振るか、コインを投げて、そのたびに未知の価値を手に入れます。すべてがシンプルに見えます。

しかし、私は特定の制限に出くわしました。乱数を取得しましたが、「ダイス」をクリックしても、フィールドから値が返されない場合がありました。

アニメーションの時間を増やしてみたところ、少しは助かったようですが、やはり思いがけない値が残っていました。

その後、ほとんどの開発者が解決できない障害を見つけたときに私がしたことを行いました-StackOverflowのヘルプを求めてポップアップしました。お勧めします。

幸い、私にはいつでも手伝ってくれる人がいます。私の場合、彼の説明をしたのはテマニ・アフィフでした。

単純化しようとすると、問題は、要素がアクティブな状態（マウスがクリックされたとき）にある場合にのみブラウザーがクリックイベントを発生させることです。つまり、マウスボタンが押されたときにアクティブになるのと同じ要素です。

アニメーションが変化しているため、実際には、マウスが押されたときの最上位レイヤー（ラベル）は、マウスが押されたときの最上位レイヤー（ラベル）ではありませんでした。サイクルのすべての値を通過することができました。これが、アニメーション時間の増加がこの問題を隠している理由です。

解決策は、「静的な」位置を適用してスタックコンテキストを壊し、::前や::後などの疑似要素を使用して、より高いz-indexを代わりに配置することでした。したがって、マウスを上に置くと、アクティブなラベルが常に一番上になります。

/*   label      */ 
label:active {
  margin-left: 200%;
  position: static;
}
/*         z-index */
label:active::before {
  content: "";
  position: absolute;
  top: 0;
  right: 0;
  left: 0;
  bottom: 0;
  z-index: 10;
}

以下は、アニメーション時間が短縮されたソリューションのコードです。


この変更を行った後も、マウスでクリックできる疑似立方体を描画するための小さなインターフェイスが作成され、CSS ゲームのSnakes and Laddersが完成しました。

この手法には明らかな欠点がいくつかあります。


  1. ユーザーとの対話が必要です-「乱数の生成」を引き起こすためにラベルをクリックする必要があります。
  2. このメソッドは適切にスケーリングされません-値のセットが小さい場合はうまく機能しますが、範囲が大きい場合は面倒です。
  3. これは本当のランダムではありませんが、疑似ランダムであり、コンピュータはどの瞬間にどの値が生成されるかを簡単に決定できます。

しかし、一方で、CSSは100％（プリプロセッサや他の外部ヘルパーは必要ありません）であり、人にとっては100％ランダムに見えることがあります。

そして、はい、この方法は、乱数だけでなく、すべての乱数にも使用できます。この場合、それを使用して、ゲーム「Rock-Paper-Scissors」でコンピューターを「ランダムに」選択しました。
