Zabbix 5.0 released

The Zabbix team is pleased to announce the release of the new version of Zabbix 5.0 LTS, which focuses on security and scalability issues.



The new version has become even more convenient, safer and closer. The main strategy that the Zabbix team follows is to make Zabbix as accessible as possible. This is a free open source solution, and now Zabbix can be deployed both locally and in the cloud, it is also available on the latest versions of Linux platforms, containers and distributions from RedHat / IBM, SuSE, Ubuntu.

Zabbix installation is now available in one click on Azure, AWS, Google Cloud, IBM / RedHat Cloud, Oracle and Digital Ocean, and technical support services on Red Hat Marketplace and Azure Marketplace.

Moreover, the Zabbix monitoring system provides a number of completely ready-made integrations for working with instant messengers, ticket and notification systems, and also extends the list of supported services and applications that can be tracked without much effort.

What's New in Zabbix 5.0:

  • Automation and Discovery : Added automatic discovery of hardware components, resources running Windows systems, and advanced detection of Java metrics.
  • Scalability : Zabbix frontend is now optimized for monitoring millions of devices.
