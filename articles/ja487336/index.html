すべての始まり


みなさん、こんにちは。UnityエンジンでのAndroid向けゲームの開発のある段階で、ビルドに1つの問題が発生しました。AppodealやGoogle Playゲームなどのプラグインをプロジェクトに追加した後、使用されたメソッドの数が65Kを超え、ビルド中に次のエラーが表示されました。

Error:Execution failed for task ':app:transformClassesWithDexForDebug'.
> com.android.build.api.transform.TransformException: com.android.ide.common.process.ProcessException: java.util.concurrent.ExecutionException: com.android.dex.DexIndexOverflowException: method ID not in [0, 0xffff]: 65536

Appodealのドキュメントを読んだ後、Multidexを有効にする必要があることに気付きました。それをすることにしました。このサイトには、Android Studioの説明へのリンクがありました。そして、Unityで、私たちは何をすべきでしょうか？

そして、すべてが非常に簡単です。3つのアクションを実行する必要があります。

Gradleビルドシステムを有効にする


AndroidプロジェクトのUnityでGradleを有効にするには

  1. Unityでビルド設定を開きます（ファイル >> ビルド設定）。
  2. プラットフォームを選択していない場合は、今すぐ選択してください（プラットフォーム >> Android）。
  3. [ビルドシステム]リストで、[Gradle（ビルドシステム >> Gradle）]を選択します。
