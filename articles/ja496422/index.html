国際的なゲーム開発者で作業する方法：スタジオの共同所有者となったUlyanovskの3Dモデラーからのチュートリアル


COVID-19は、ヘルスケアの分野だけでなく、身近な世界に深刻な変化をもたらします。多くの人が新しい仕事を探す必要に直面するでしょう。

世界中の人々がパンデミック後の生活に恐怖を感じる一方で、ゲーム開発者は「この特定の活動分野を選んだのは良いことだ」と考えています。

ゲーム開発は過去30年間成長している業界であり、大変動にもかかわらず、止まることはありません。ハリウッド、観光、空の旅、レストランのビジネス、そして実際の生産は、流行のために麻痺しており、長い間彼らの理にかなったことでしょう。 Gamedevは1日も止まりませんでした。誰もが孤立していて、リモートで作業を続けています。

当社では5518・スタジオ本格的な作業：お客様は新しいヒーロー、アートワーク、3Dモデル、レベル、アニメーション、その他のコンテンツを待っています。私たちのプログラマーやアーティストは、仕事なしで放置されるのではなく、そのボリュームにどう対処するかについてもっと心配しています。

したがって、ユーザー名であるあなたが、ゲームの趣味を興味深く満足できる職業に変えるかどうかを考えているだけなら、私はこう言います。ゲーム開発の競争はすでにかなり激しく、パンデミックにより多くの失業者が出たときに、疑いを抱くほど、関連する地域や遠くの地域から業界に参入したい人が増えます。

この記事では、あなたの疑問や懸念を払拭するようにします。何百万人ものゲーマーがプレイする国際的な規模のゲームの開発に取り掛かるために検討する必要がある複雑さについて、専門分野を決定する手助けをします。

1.専門教育がない場合はどうなりますか？


独学する。ゲーム業界は急速に発展しており、その専門職は高等教育のシステムよりも速く見えます。だから人々は文字通りどこからでもgamedevにやって来ます！プログラマーは、原則として、ITの他の分野での経験を既に持っています。しかし、どんな背景を持つ人でも、QAテスターやアートマネージャーになることができます。主なことは、彼が誠実に仕事をし、新しいことをすばやく学ぶ方法を知っていること、そして個人やチームの一員として雇用主に適していることです。

たとえば、5518 Studiosで新しい人材を採用する場合、「クラスト」は考慮しません。私たちはテスト課題を与えます、そして候補者がそれをうまく完了したなら-私たちは面接を求めます。個人的な会話でのテスト結果と適切な行動-卒業証書ではなく、人についてもっと話してください。



2.どこから始めますか？


2.1。情報に基づく選択から

フォーカス：人生の次の10年間（少なくとも）を、エンターテイメントではなく仕事という形でゲームに費やしたいですか？ゲーム開発はマニアや夢想家にとって天国ではありませんが、非常に深刻な職業です。

第一に、ゲーム開発者は最もグローバル化され、競争の激しい業界です。つまり、掘り下げる必要があります。背中の後ろで、他の5〜10人のロシア人、ベトナム人、ブラジル人、インド人が常にあなたの給与を受け取りたいと思っているからです。したがって、9時から17時までの所定の時間に座る必要はありません。その半分はチャットやミームの友達との交換に費やすことができます。私たちは一握りのお尻を入れて生産的に一生懸命働かなければなりません-国際市場では彼らは数時間ではなく結果に対して支払うからです。

第二に、映画の特殊効果からクールなゲームまで、現在のエンターテインメント部門のすべての驚異は、骨の折れる作業、そして大部分は日常業務によって生み出されています。想像力があなたを素晴らしい作品に引き寄せるなら、あなたが初日から世界的に有名なキャラクター、素晴らしい武器と革命的なゲームプレイを作成する-それは少し冷やす価値があります。

最初は、もっと普通のことをする必要があります。したがって、十分な責任があり、ゲームに対する真の愛情があり、彼らの進化に貢献したいという大きな願望がある場合にのみ、gamedevに参加する価値があります。同じ作業を行うのに数年かかる場合でも、非常に厳しい期限を守り、多くのことを学びます。

初心者が面接に来たとき、最初の3つのメモから、私たちの人かどうかということです。私たちの停止信号として機能する行動の主なパターンは、焦点をぼかすことです。人がゲームを作り、同時に自分のコーヒーショップを開き、10億ドルを稼ぐための新しい方法を発明したいとき、彼はこれのいずれにも成功しません。彼がモスクワ州立大学の卒業証書を持っていたとしても。

しかし、経済学者のカティアを見て、カティアが言うなら、「おい、私は次の30年間ゲームをプレイしたい。グラフィックタブレット、ライセンスされたPhotoshop、CGキャラクターの作成に関するチュートリアルを購入しました。私は1日12時間描画する準備ができています」と彼女の目は燃えています-これのほぼ100％が良いでしょう。



2.2。トレーニングから実践へ

今日のゲーム開発への参加はこれまで以上に簡単です。あなたの街には大学がなくても、おそらくインターネットを利用しています。そして、そこには教科書やオンラインコースがあり、それらの多くは完全に無料で利用できます！検疫を離れることなく、職業の基本を習得できます。

さらに、明らかな優れた才能の欠如は重大ではありません。新しいテクノロジー、責任、自己組織化、自己規律、耕す意欲をすばやく学習する能力は、天才やナゲットの微妙な精神的組織よりも高く評価されることがよくあります。

熟練には経験が必要です。3〜5年のささやかなミドル農民でも、クールなプロになることができます。もちろん、彼がたくさん、本当にたくさんの仕事をしない限り、彼は日常的で燃えている締め切りと無限の編集を恐れません。はい、編集されます。編集を「悪い評価」としてではなく、通常のフィードバックとして認識して、それらを受け入れ、理解し、やり直す必要があります。

ゲームを作成してスキルを磨くプロセスを詳しく調べるために、アマチュアのインディーズゲームの開発に参加することから始めます。これは、実務経験、業界でのつながり、ポートフォリオの最初の仕事を獲得する絶好の機会です。

3.いつお金が支払われ始めますか？


基本的なスキルがあれば、月額500〜1000ドルから始めることができます。 6か月の仕事の後、すでに良い会社でインターンシップに応募できます。さらに3年後-良心に安定して取り組み、期限に間に合わない場合、$ 2〜3千のレベルに達します。さらに2年後、給与は5,000ドルから始まります。ロシアについてです。海外での生活や仕事に招待された場合、これらの数も増える可能性があります。

しかし、最近は海外に行きたいIT専門家が少なくなってきています。ほとんどがロシアに滞在し、遠隔地の企業で働くことを好みます。常に注文と安定した給与がある場合、これは高い生活水準です。

ロシアで、特に地域で、給与をドルで稼ぐことで、あなたはとてもよく生きることができます。モスクワ、さらにはヴォロネジまたはカザンでの生活水準は、給与水準よりも米国とは大きく異なります。つまり、条件付きのカザンで生活し、仕事をしているCGアーティストは、150〜20万ルーブルを受け取り、カリフォルニアで働いて7,000ドルを受け取るアーティストよりもはるかに多くのお金を稼ぐことができます。すべての価格差は巨大。

ロサンゼルスでは、この給与を30％の税金、アパートの場合は2,500〜3,000ドル、多かれ少なかれ高品質の食品の場合は1,500ドル、レンタカーの場合は1,000ドルで支払います。そして、あなたはまた、蜂蜜のために毎年支払う必要があります。保険。毎週末、カフェで食事をしたり、映画を見たりすることはできません。休暇中は、保存して保存する必要があります。

ロシアでは、子供たちは主に首都ではありません。彼らの住居は寝室ではなく中央にあります。彼らはまともなフィットネスクラブに行き、歯をいい医者で治療し、年に数回、トルコだけでなく世界中を旅し、美しいロシアの女の子と結婚し、子供、コーギー、猫を飼い、人生を楽しんでいます。



4.ゲーム業界にはどのような専門家が必要ですか。また、好きな仕事を選ぶにはどうすればよいですか？


ゲームの世界とキャラクターが複雑になるほど、面白いストーリーが得られます-ゲーム業界の各専門分野は狭くなります。そして今日、gamedevには最大100の職業があります！

私は、需要とかなり低い入学基準を組み合わせたものについてのみ話します（卒業証書や、ジュニアポジションであっても長年の経験は必要ありません）。それらのすべてをリモートで習得することができます。

4.1。プログラマー

彼がしていること：アーティスト、アニメーター、サウンドエンジニア、ゲームデザイナーのすべての業績を結集します。彼はすべてのアイデアを最適な方法でコードに変換する方法を考え出します。最終的には、ゲームが悲観的に遅くなるか、それとも弱いコンピューターでも正常にプレイできるかどうかに依存するのは、彼の能力です。



なにができる：2つの「エンジン」のうち少なくとも1つ（UnrealまたはUnity）に精通している。 C ++、C＃、および最も重要なことでプログラムできる-ゲーム開発の詳細を理解している（TKは100％確実ではない-この機能またはその機能の実装方法を決定する;決定は迅速に行う必要がある;何百人ものプログラマーが理解できるようにコードを書く必要があるこのプロジェクトに取り組んでいる世界中から）。

だれに適しているか： 8〜12時間コンピュータの前に座る準備ができている人は、ほとんど話さず、一生懸命考えて、楽しんでください。工学的な考え方を持つ人々、正確な科学、アルゴリズム、最適化されたプロセスを好む人々。

いくら稼いでる？ロシアとCISでは、1,000ドルから-初心者開発者のジュニア、2,000ドルから-ミドル、数年の経験。優れた学問的知識を持つシニアプログラマー-1か月あたり3〜5千ドル。エンジン、フロントエンド、バックエンド、およびゲームデザイナーのスキルを備えたクールなスペシャリストは、10,000ドルを獲得できます。

ただし、仕事のためにお金を稼ぐためだけにプログラマーに行った人は、仕事に興味がなく、すぐに燃え尽きてしまいます。

開始するには？ C＃やC / C ++、Java、Javascript、SQLなどのさまざまなプログラミング言語を学びます。将来のキャリアをロボット工学だけでなくロボット工学にも結びつけることを計画している場合、Pythonプログラミング言語の知識は害になりません。

主なことは、常に練習することです。人気の「エンジン」で独自のゲームを作成し、Githubでコードを確認し、メンターを探し、教授に質問してください。フォーラム。

4.2。ゲームデザイナー

彼がしていること：ゲームのルールとゲームのメカニズムを開発し、「機能」、ゲームプレイ要素を思いつきます。彼は1日に少なくとも8時間ゲームをプレイします-数十回連続して個々のミッションをテストし、パラメーターを理想に調整します。デザイナーとプログラマーのためにTKを書きます。別のレベルと場所を作成するゲームデザイナーについて話している場合、これは彼がレベルデザインも行うことを意味します。

できること：新鮮なアイデアを生み出し、聴衆のニーズを理解する。一部のゲームが人気または失敗した100の理由を分析テーブルに記述します。ゲームの複雑さとプレイアビリティのバランスをとるため：キャラクターが脚のショットで死ぬことはなく、核爆撃後に無傷のままになることもありません。

それが誰に合うか：モバイルゲームに真剣に情熱を傾けている人は、すでに何千時間ものゲームプレイを逃しており、さらにそれに没頭する準備ができています。



いくら稼いでる？平均して、2020年-1〜2千ドル。非常に経験豊富なゲームデザイナー-最大3,000ドル。さらに、多くの場合、ゲームデザイナーはゲームの売り上げの一部を受け取ります。

開始するには？ゲームのテストを開始するには、これらのゲームを実行できるコンピューター、電話、またはその他のデバイス、Excelのプレート、ジョイスティックが必要です。基本的な知識のうち、数学、物理学、幾何学の学校のコースで十分です。ここでは、細心の注意、徹底、分析的な考え方が必要です。

繰り返しになりますが、経験とポートフォリオのために、いくつかの非営利プロジェクトでゲームデザイナーの役割に参加すると便利です。 Unity、Unreal Engine、Lumberyard、CryEngineエンジンを扱います。

ところで、優れたゲームデザイナーはQAテスターから成長することがよくあります。

4.3。 2Dアーティストと3Dアーティスト

彼の活動： 2D-コンセプトアート、ポスター、マップ、ゲームの背景、キャラクターを描きます。 3D-コーナーのクローゼットからメインキャラクター、武器、乗り物まで、ゲームの世界の「詰め物」全体を作成します。

彼ができること：気まぐれな創造性を押しのけ、技術要件に従って作業し、締め切りに間に合います。今日、キャラクターを一から発明し、明日は落ち着いて現在と変わらない百番目の樽型を作る。

2D-才能、芸術教育、豊かな想像力を持つ人々のために。 3D-芸術的才能はないが、グラフィックスを扱う上での優れた技術的基盤を備えている人々にも。自分の仕事の結果をすぐに見たい人。

いくら稼いでる？リアルな設定のジュニア3Dアーティストは、$ 500から$ 1,000、環境スペシャリスト-$ 1-2千を受け取ります。キャラクター、クールな装備、武器を作る人-$ 3から$ 4,000です。

$ 500から$ 3,000の平均パスには3かかります年、しかし時にはそれ以下。私たちの会社には、2年間で人が工場労働者から一流の3Dアーティストに進化した例があります。専門教育なし。しかし、彼は毎日最善を尽くしました。

1時間あたり50ドルを稼ぐスペシャリストもいます。たとえば、トップレベルのヘアテクスチャ3Dアーティストは、ゲームの3Dヘアアーティストです。これは、エンジンでリアルに見える3Dゲームのヘアスタイルのヒーローを演じることができますが、同時に、非常に多くのポリゴンとディテールのために、ゲームを「遅くする」ことはありません。

これは非常に難しい技術的な作業です。すでにCGでしっかりした経験があり、3人のキャラクターのヘアスタイルを開発しました-論文よりはるかに難しかったです。見事にそれを行う方法を知っている人々-全世界には約20人がいて、彼らのための4,000ドルは最低限のバーです。

開始するには？ソフトウェアの学習：Maya、3Ds Max、Zbrush、Blender、Cinema 4D、Modo、Photoshopなど。プロセスを最適化し、あらゆる種類のツールを使用して効率と速度を向上させる方法を学びます（Substanceペインターなど）。

ゲームカンファレンスに参加し、最新のすべてを再生し、クールなCGで映画を鑑賞し、アートとアニメーションのトレンドを追跡し、Artstation、Behance、Deviantart、Sketchfabなどを使用した新技術の出現を常に確認してください。



4.4。アニメーター

何ができるか：モーションを作成するときに物理的に正しい結果を得るには-キーフレーム、モーションキャプチャテクノロジー、およびソフトウェアを使用します。

いくら稼いでる？平均すると、ロシアのアニメーターは1,000ドルから受け取ります。国際企業では、5,000ドルから受け取り

ます。もちろん、ここでは教育を受ける方がよいでしょう。専門分野は「アニメーションとコンピュータグラフィックのアーティスト」、「アニメーションとイラスト」などです。

ただし、少なくとも-ソフトウェアを理解するために：Autodesk Maya、3Ds Max、Cinema 4D、Adobe After Effectsなど。ビデオを作成するプロセスを理解し、理想的には、脚本家とカメラマンの作品についてのアイデアを得ます。アニメーターのサバイバルキットを読む-アニメーションに関する本の中でも定番

4.5。 QA、beta-tester。

機能：ゲームのバグをテストします。彼は終日プレイし、ゲームがプロセスでクラッシュするかどうか、すべてが正しく書き込まれるかどうか、ヒーローがテクスチャに落ちるかどうか、すべてのオブジェクトとキャラクターが期待どおりに動作するかどうかを確認します。毎日検出されるすべてのエラーについて、プログラマーにレポート（レポート）を書き込み、プログラマーがこれらの誤解を修正するようにします。

なにができる：ささいなことに気づき、一般的なユーザーの行動モデルを再現し、可能な限りすべての方法で同じゲームアクションを実行します。一般的なエラーを自動的に検出する小さなプログラムを個別に作成します。



誰のためですか：完璧主義者。お金のために一日中おもちゃを切りたい人。しかし、同時に、ユーザーの行動について考えられるすべてのシナリオを完全にシミュレートし、小さな誤動作に気づき、それをレポートに記録し、チェックの非常に厳しい締め切りを妨害しないようにすることは慎重です。

どのくらい稼いでいますか？ 300ドルから1,500ドル。上級QAテスターの場合、s / nは2,000ドルと3,000ドルの両方

になる可能性があります。最小化を手動で行う必要があるように、自動化をテストするためのさまざまなツールを学びます。バグトラッカーの使い方を簡単に学びますが、自分の考えを明確に表現してください。非標準的な考え方を開発します。他の人のコードを読んだり、他の人の3Dモデルのファイルや構造を理解したりすることを学ぶ。
