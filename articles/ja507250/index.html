2つのポーカーハンドを比較し、勝者を決定することです。

ポーカーハンドは、スペースで区切られた2文字のサブストリング（個々のカードを表す）を含む行で表されます。2Hは2つのワーム、TCはクロステンなどです。

最初の文字はランキングを意味します。
可能なすべてのオプション、年功序列：2 3 4 5 6 7 8 9 TJQKA。
数字による指定では、文字と同様にすべてが明確です。Tは10（T en）、Jはジャック（J ack）、Qはクイーン（Q ueen）、Kはキング（K ing）、A-エース（A ce）。

2番目の文字はスーツを定義します。
可能なオプション：H-ワーム（H earts）、C-クロス（C lubs）、S-ピーク（S pades）、D-タンバリン（D iamonds）。

たとえば、「2C 5C 3C 4C 6C」という手の完全なライン-この場合、これは2から6のクロスストレートフラッシュです。

ハンドランキングは、テキサスホールデムの場合と同じです。

あなたは隠された困難があることを理解する必要があります-ドロー状況を解決するための追加のカードの使用、ペアの額面の使用など。

決定


では、この問題を解決するにはどうすればよいですか？最初に、ランキングハンドの問題を解決する比較方法を決定する必要があります。次に、ランキングが一致する場合は、可能であれば追加のカードを比較して勝者を決定します。

問題の説明では、スートが同等であると想定しているため、単純なオブジェクトを使用して、ハンドのランキングを表します。

{ 
   rank: 1,       //     1  9,   ,    
   value: 'ABCDE' //  ,        ,    ASCII-    
}

これで、この構造の形で表された2つの手を比較するための簡単な関数を書くことができます。

function compareHands(h1, h2) {
    let d1 = getHandDetails(h1)
    let d2 = getHandDetails(h2)
    if (d1.rank === d2.rank) {
        if (d1.value < d2.value) {
            return ""
        } else if (d1.value > d2.value) {
            return ""
        } else {
            return ""
        }
    }
    return d1.rank < d2.rank ? "" : ""
}

だから今私たちがしなければならないすべては私たちの手からオブジェクトを作成することです-そしてここから楽しみが始まります！

重複を定義する


このステップはもう少し複雑です。手元にある各額面の金額を計算する必要があります。そして、このようにあなたの手のランキングを単純化するために、ペアやトリプルなどを決定するいくつかの方法も必要です：

  • 集計を検索可能な提出に変換する。

手にある「4」（正方形）または3つの存在、ペアの数（1つ、1つ、または2つではない）などを特定できることを確認する必要があります。

function count(c, a) {
    c[a] = (c[a] || 0) + 1
    return c

したがって、ストレートフラッシュは、ランク1（複雑化しないために、誰かがロイヤルフラッシュを持っている場合は引き分けが可能であると考えられます）、次にフォーオブカインド、フルハウスなどの最も強力な組み合わせです。

この場合、&&演算子は素晴らしく使用され、前の括弧内の値が正しい場合、最後の値までスキャンします。したがって、(flush && direct && 1) flushおよびdirectがtrueの場合は1を返し、それ以外の場合はfalseを返します。

描かれた状況


2つのハンドが同じランクの場合、可能であれば、カードの額面によって勝者が決定されます。これにはいくつかのルールが関連付けられています。

  • カップル対カップル。額面の高いペアが勝ちます。値が一致した場合、次の最高のカードが勝ちます。（2つのペアの場合も同様に解決されます）。

    たとえば、「2H 2D AH KC 3D」のハンドと「4H 4C JC TC 3H」のハンドを比較すると、最初のハンドで最高のカードが最高の額面-エースですが、2番目のハンドから4が勝ちます。
  • フルハウス対フルハウス。額面の高いトリオが勝利します。両手のトリプルの金種が等しい場合、2の金種を調べます。
  • 同じランクの手札を比較するときも同様に行動します-最高のカードの組み合わせの最高額を決定します。


したがって、最初に数量でソートし、次に出力ラインの額面でソートする必要があります。5文字の文字列全体が、一致するハンドランキングのケースを解決するために使用されることに注意してください。

全シバン


const order = "23456789TJQKA"
function getHandDetails(hand) {
    const cards = hand.split(" ")
    const faces = cards.map(a => String.fromCharCode([77 - order.indexOf(a[0])])).sort()
    const suits = cards.map(a => a[1]).sort()
    const counts = faces.reduce(count, {})
    const duplicates = Object.values(counts).reduce(count, {})
    const flush = suits[0] === suits[4]
    const first = faces[0].charCodeAt(0)
    const straight = faces.every((f, index) => f.charCodeAt(0) - first === index)
    let rank =
        (flush && straight && 1) || /* -,     - -        */
        (duplicates[4] && 2) || /*  -     */
        (duplicates[3] && duplicates[2] && 3) || /* - -  +  */
        (flush && 4) || /*  -      */
        (straight && 5) || /*  -       */
        (duplicates[3] && 6) || /*  -     */
        (duplicates[2] > 1 && 7) || /*   */
        (duplicates[2] && 8) || /* -     */
        9 /*     ,      */

    return { rank, value: faces.sort(byCountFirst).join("") }

    function byCountFirst(a, b) {
        //    -  ,  
        const countDiff = counts[b] - counts[a]

        if(countDiff) return countDiff //   ,  
        return b > a ? -1 : b === a ? 0 : 1
    }

    function count(c, a) {
        c[a] = (c[a] || 0) + 1
        return c
    }
}

function compareHands(h1, h2) {
    let d1 = getHandDetails(h1)
    let d2 = getHandDetails(h2)
    if (d1.rank === d2.rank) {
        if (d1.value < d2.value) {
            return ""
        } else if (d1.value > d2.value) {
            return ""
        } else {
            return ""
        }
    }
    return d1.rank < d2.rank ? "" : ""
}
