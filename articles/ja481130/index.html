TOP12学際的科学発見2019

ネタバレ！記事では、おそらく2019年の科学の世界で最も興味深いものです。しかし、そうではありません。

私たちについて簡単に言えば、私たちはベラルーシの科学的スカウトであり、その若さは現実の花崗岩に分裂しているところです。私たちは、私たちが必要と考えている科学的イベントを実行します。自分をSCITEENと呼びます。
また、新しいTOP12プロジェクトを実装しました。ここでは、過去1年間で学際的な科学的発見の中で最も興味深い12のプロジェクトを紹介しています。

当初は研究について公開講演を行う予定でしたが、作者の中にはテレビ会議で聴衆と話すことを事前に承諾してくださった方もいました。したがって、私たちは講義室を起源の人々の物語で補足します:)アイデアがどのようにして生まれたのか、または将来何が計画されているのかについて質問できます。VKと投稿の下で、グループ内の興味のあるトピックを選択するために、投票を組織しました..

そして、実際には、トップ自体です。教材がマニアにもマストドンにもアピールできることを願っています！

1.赤外線ビジョンを取得するには？

中国科学技術大学のTian Xueとマサチューセッツ大学医学部のGang Khanが率いる研究チームは、通常の光を見る能力を維持しながら、近赤外光（NIR）を見ることができるようにマウスのビジョンを変えました。これは彼らの目に特別なナノ粒子を注入することによって達成されました。効果は深刻な副作用なしで約10週間続きました。

DOI：10.1016 / j.cell.2019.01.038

2.太陽電池とナノテクノロジー

現在、太陽電池とシリコンベースのモジュールの効率の最高値は25％です。ナノチューブの配向により、米大学（米国）の科学者は熱を光に変換することができ、効率を最大80％向上させることができました。王立工科大学KTH（スウェーデン）の研究者も、赤外線をエネルギーに変換するナノ材料を使用する方法を発見しました。

DOI：10.1039 / C9NR03105G
DOI：10.1021 / acsphotonics.9b00452

3.反応を確認

化学反応は非常に短いプロセスです。ハーバード大学（米国）の研究者は、絶対零度に近い条件を作成し、2つの金属の原子を衝突させることができました。その結果、科学者はプロセスのすべての段階を観察することができました。少し前に、東京大学（日本）の科学者は、研究した分子をナノチューブの端に固定して化学反応の経過を研究する方法を提案しました。これにより、化学反応の進行を追跡することもできました。

DOI：10.1126 / science.aay9531
DOI：10.1038 / s41467-019-11564-4

4.培養液からのキメラと水たまりでの生活

2019年は生命の起源の分野の研究にとって豊かな年になりました。東京工業大学（日本）とマレーシア大学の科学者たちは、生命が地球に現れる前に、多数の多様な分子が存在すると結論付けました。一次ブロスからのキメラ合成スキームが提案され、リボースは小惑星で発見され、生命は以前考えられていたように、海ではなく小さな池と貯水池で発生したことが示唆されました。

DOI：10.1073 / pnas.1902336116
DOI：10.1038 / s41557-019-0322-x
DOI：10.1029 / 2018GC008082
DOI：10.1073 / pnas.1907169116

5.反応が3〜4桁加速

ミネソタ大学（米国）とマサチューセッツ大学（米国）の科学者チームは、化学反応を最大で10,000倍高速化できる可能性のある新技術を発見しました。これは、波動周波数と自然反応周波数が一致するときの共振により振動触媒の過程で得られる共振により可能になった。この場合、化学合成の速度が上がると、プラスチック、肥料、燃料などの製造に使用される化学プロセスのコストを下げることができます。

DOI：10.1021 / acscatal.9b01606

6.ペットボトルで車に燃料を補給する

国連によると、人類は年間3億トン以上のプラスチック廃棄物を生み出しています。多くの異なる研究所が、この問題を解決するための合理的なアプローチを求めています。しかし、パーデュー大学の研究者は、プラスチック加工に対して非常に珍しいアプローチを採用しており、それを燃料に変えています。これを達成するために、彼らはポリオレフィン廃棄物の90％以上を高品質のガソリンまたはディーゼルに変換できる新しい化学変換プロセスを導入しました。

DOI：10.1021 / acssuschemeng.8b03841

7.気候が逆転

マサチューセッツ工科大学（米国）のエンジニアは、400 ppm以上の含有量でCO2を吸収できる装置を開発しました。次に、オーストラリア王立工科大学の科学者は、室温で二酸化炭素を固体炭素粒子に戻すことを学びました。また、ライス大学（米国）の研究者は、ビスマスベースの触媒を使用して二酸化炭素からギ酸を取得しました。

DOI：10.1038 / s41467-019-08824-8
DOI：10.1039 / C9EE02412C
DOI：10.1038 / s41560-019-0451-x

8.炭素の新しい同素体修飾

オックスフォード大学（英国）の研究チームとチューリッヒ（スイス）のIBM Research Labは、熱力学的に安定した18員の炭素環を生成することができました。新しい化合物の特性の研究は、驚くべき発見につながる可能性があります-以前に得られたフラーレンとグラフェンは、それぞれ1996年と2010年にノーベル賞を受賞しました。

10.超伝導体は

、マックスプランクインスティテュート（ドイツ）、高磁場研究所（米国）、シカゴ大学（米国）、物理化学研究所、PAN（ポーランド）の科学者からの科学グループに近づいています。、固体物理研究所は、ロシア科学アカデミー（ロシア）が-23の過去最高の超伝導体を受け取った上で Cの温度と圧力は170万の雰囲気でした。実用的な室内超伝導について話すのはおそらく時期尚早ですが、このデータは、高温超伝導と化合物の結晶構造との関係についての理論を裏付けています。

DOI：10.1038 / s41586-019-1201-8

11.最も薄い金

リーズ大学（英国）の研究者が初めて、厚さ2原子、0.47ナノメートルの金の層を受け取りました。ナノ粒子の代わりとして、類似の金を触媒として使用して、電子工学などで人工酵素を作成することができます。この研究は、層状構造を持たない極薄材料への道を開きます。

DOI：10.1002 / advs.201900911

12.歯のエナメル質の修復

浙江大学（中国）の科学者は、歯の表面に結晶が成長するため、歯のエナメル質を修復する方法を開発しました。新しくなったエナメルの厚さはわずか0.0027mmであり、実用上十分とは言えませんが、この場合、エナメルの機械的強度が回復し、摩擦係数が低下します。

DOI：10.1126 / sciadv.aaw9569
