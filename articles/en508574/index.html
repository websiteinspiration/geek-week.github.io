Deploy and configure node-red authentication on docker-compose



Deploy node-red to docker-compose with authorization enabled and using docker volume.



Create the docker-compose.yml file: 



version: "3.7"

services:
  node-red:
    image: nodered/node-red
    environment:
      - TZ=Europe/Moscow
    ports:
      - "11880:1880" # 11880 -     , 1880 -     node-red  .
    volumes:
      - "node-red:/data" # node-red -    docker   , /data -   .
    restart: always
volumes:
  node-red: #   node-red  .


We launch the container in debug mode (the first launch should be without the ` -d` key , to view errors when such appear): 



$ docker-compose up node-red
Creating node-red_node-red_1_3e3e59f5e044 ... done
Attaching to node-red_node-red_1_bca4cb987984
node-red_1_bca4cb987984 |
node-red_1_bca4cb987984 | > node-red-docker@1.0.3 start /usr/src/node-red
node-red_1_bca4cb987984 | > node $NODE_OPTIONS node_modules/node-red/red.js $FLOWS "--userDir" "/data"
...


We stop the container and run the command to view volume: 



$ docker volume ls
DRIVER              VOLUME NAME
local               node-red_node-red


View detailed information on volume: 



$ docker volume inspect node-red_node-red

Go to the volume directory. This directory already contains the files that docker created when deploying the container. 



$ sudo ls /var/lib/docker/volumes/node-red_node-red/_data
lib  package.json  settings.js


We are interested in the settings.js file . Open it and look for a piece of adminAuth code . Uncomment it. 



 // Securing Node-RED
    // -----------------
    // To password protect the Node-RED editor and admin API, the following
    // property can be used. See http://nodered.org/docs/security.html for details.
    adminAuth: {
        type: "credentials",
        users: [{
            username: "admin",
            password: "$2a$08$zZWtXTja0fB1pzD4sHCMyOCMYz2Z6dNbM6tl8sJogENOMcxWV9DN.",
            permissions: "*"
        }]
    },


In the password key : insert the hash of the password node-red.



Getting hash password node-red
Install the node-red-admin package on any machine with node.js. 



npm i node-red-admin -g


: 



node-red-admin hash-pw




Launch the container and connect to port 11880 . 



http://192.168.0.100:11880/


An authorization window should appear.







Enter the login password.



If everything works, restart the container with the -d switch . 



$ docker-compose up -d node-red
