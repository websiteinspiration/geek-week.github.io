さらに簡単なのは、R0の定義を取得して読み取ることです。このインジケーターが実際の流行の速度とは関係がないことを確認してください。ただし、これは特定の数学モデルの人工的なパラメーターにすぎません。それにもかかわらず、感染の危険性について話をしたいときはいつでも、基本的な生殖数は減少します-伝染性の指標（これは存在しますが、完全に異なるものを意味します）、感染伝染のレベルなどです。しかし、これはそうではありません。R0は非常に明確な意味を持っています。そのためには、赤い言葉ではなく、R0を適用する必要があります。

正確なこと-発生率モデルがR0 <1に対応している場合、流行は始まりません。流行している場合は、平均して各患者が1人未満の新しい患者を生成する場合、遅かれ早かれ「最後の患者」は誰にも疾患を伝染しないため、それは減少します。したがって、基本的な定義は次のとおりであるため、R0は削減可能であり、削減する必要があります

。R0 =βγ、ここで

βは特定の時間内に新しい患者につながる連絡先の数です。
γは、患者が健康な人と接触している期間です。

たとえば、患者が他の人との接触から1日で平均1人の新しい人に感染し、患者を特定して隔離するまで平均5日間感染すると、特定の環境におけるこの特定の疾患のR0は5になります。 5つの新しいケースを生成します。

もちろん、数学的モデルは実際のプロセスの近似にすぎないため、R0の値はなく、正確な値であるとは限りませんが、この値により、特定の条件下で特定の地域で病気が流行の形で広がるかどうかを理解できます。

繰り返しますが、流行の速度を評価するのではなく、流行が可能かどうかを評価します。コロナウイルスのR0は、いくつかの情報源によって正確に1よりも大きいと推定されています。したがって、それを減らすための対策を講じなければ、流行が発生する可能性があります。

ここで最も重要な結果は、R0を削減するために、1つまたは複数の条件を満たす必要があるということ

です。ポイント1.患者と他の人との接触の数を減らす-病気の人を隔離し、不審な人を隔離します。
ポイント2.患者と他の人との接触期間を短縮する-患者をすばやく特定し、パラグラフ1に従って続行します。

。βは、伝染性（攻撃率）の指標における接触数の積-特定の感染または病原体の実際の指標-患者の割合この疾患に特徴的な、感染のキャリアを持つ患者と接触した人の数（通常は診療から決定されます）。

つまり、βが大きくなるほど、疾患の伝染性が大きくなり、患者はワクチン接種を受けていない健康な多数の人々とコミュニケーションをとることになります。

たとえば、はしかの伝染性は50〜90％と推定されています。つまり、患者と接触している10人のうち、半分以上が病気になります。たとえば、平均して5人が毎日患者とコミュニケーションをとっている場合、毎日3〜4人の患者が平均して現れます（β= 5 x 0.5–0.9 = 2.5–4.5）。麻疹患者は3日後に隔離され（γ= 3）、合計で約8〜14人の新しい患者が現れます（R0 =βγ= 2.5–4.5 x 3）。したがって、R0はしかと8-14になります。

さらに、検疫を宣言し、それによって患者と通信する人々の数を毎日2人に減らすと、同じ3日間の「通信」で3〜5人の新しい患者だけが表示されます（2 x 3 x 0.5–0.9）、つまりこの分布モデルのR0は5より低くなりますが、それでも1より高くなります
。R0を1未満の値に減らすには、患者との接触数を3日間で最大1に制限する必要があります。これにより、患者は隔離されます-（1 x 0.9 = 0.5〜0.9）または検出から患者の隔離までの時間を最大で数時間以下に短縮し、他の人との接触の可能性と数を減らします。

死亡率と死亡率


たまたま医療統計では、「死亡率」と「死亡率」という用語の意味が異なります。

死亡率（致死率）は、総症例数に対する死亡数の比率です。たとえば、1000人の患者のうち20人が死亡した場合、死亡率は20/1000 = 0.02または2％になります。

死亡率- 人口全体に対する単位時間あたりの死亡数の比率（通常は1年あたり）。たとえば、同じ20人が亡くなり、合計で10万人が国に住んでいる場合、死亡率は20 / 100,000 = 20人になります。10万人あたり-この形式の記録は、通常、さまざまな国または地域の指標を比較するのに便利です。

たとえば、中国のコロナウイルス：

  • 3,267人が亡くなり、
  • 81500人が病気になり、
  • 全人口は14億人です。


死亡率： 3,267 / 81,500 = 0.04、または症例数の4％。
死亡率：3,267 / 1,200,000,000 = 0.00000233、または中国の総人口の0.000233％。

中国の人口は症例数（81.5千人に対して14億人）よりはるかに多いので、相対死亡率は死亡率（4％に対して0.000233％）よりはるかに低くなっています。症例の割合が高いほど、死亡率が高く、死亡率に近くなります。

10万人あたりの中国のコロナウイルスによる死亡率：0.00000233 x 100,000 = 0.233人。10万人あたり。

これは非常に低い指標です。たとえば、中国の交通事故による死亡率は18人です。 10万人、つまり77倍も高く、「交通事故で死ぬ可能性はコロナウイルスよりもさらに高い」という声明が真実であると思われるかもしれません。年間25万人以上が中国での事故で亡くなっているからです。 COVID-2019による死者3000人と比較。

しかし、中国の全人口がコロナウイルスに罹患しており、死亡率は4％であるとすると、5,600万人（14億x 0.04）が死亡、つまり事故の224倍以上になります。

もちろん、これは実際には起こりません。はしかなどの最も感染性の高い疾患でさえ、万人から遠く、コロナウイルスによる実際の死亡率は10万人もはるかに低いためです。しかし、国によっては、たとえばイタリアでの事故による死亡率を24.03だけ上回っています。 2020-ほぼ2倍（コロナウイルスの人口10万人あたり10症例対交通事故での10万人あたり5.6死）、したがって、コロナウイルスは間違いなく交通事故よりも危険ではないと言うことは、すべて特定の国の患者数に依存します。

今、死亡率と死亡率の概念の混乱について-おそらく多くの人がメディアでの死亡率はしばしば死亡率と呼ばれていることに気づきました、そしてこれは英語の情報源では珍しくありません：

一般的に、ブルームバーグは1つの記事で両方の用語を使用していますが、死亡率の数値は4％ですが、死亡率について話しているのは明らかです。

もちろん、ジャーナリストにはもっと正確にしたいのですが、現実をそのまま受け入れる場合は、まず次元に注意を払う必要があります。

保護とマスク


マスクやマスクのブームは、インフルエンザや類似のウイルスの流行に伴って発生します。これは、私の内部の懐疑論者を混乱させます。なぜなら、通常のマスクは実際には感染を防ぐことができず、混雑した場所で病気になる可能性をわずかに減らすだけだからです。

一方、くしゃみや咳をするときは、感染の拡大を抑えるために、病気の人がマスクを着用する必要があることを知っています。コロナウイルスの場合、人は自分が病気であることを知らないかもしれませんが、すでにウイルスを広めているため、他人に感染しないように誰もがマスクを着用し、無症候性であることがわかります。

正しく使用した場合のマスクのもう1つの利点は、手で口や鼻に触れることが少なくなることです。つまり、接触感染の可能性が低くなります。

問題は1つだけです-マスクに似たオブジェクトの非常に高い価格-10回、または50回です。まあ、あなたは売り手に利益のために貪欲に何もせずに他の方法を使わなければなりません。ちなみに、あなたが健康であれば、路上でマスクを着用することはまったく意味がありません。主なことは、インフルエンザやコロナウイルスは言うまでもなく、ペストから守るのに十分な 1〜2メートル未満の人に近づかないことです。

くしゃみや咳や不健康な人から遠ざけることは、証拠の船長からのヒントですが、それを使用することができ、使用する必要があります。この点で、スーパーマーケットのキューも望ましくない現象です。ラッシュアワーではなく、ショッピングの時間を選択することをお勧めします。

信頼できる消毒の処理時間は少なくとも30秒であることを覚えておくことも重要です。
