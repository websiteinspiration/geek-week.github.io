Rilis distribusi Manjaro Linux 19.0




Menurut OpenNET, rilis GNU / Linux dari distribusi Manjaro Linux 19.0, dibangun berdasarkan Arch Linux, tetapi ditujukan untuk pemula, telah dirilis. Manjaro memiliki installer grafis yang lebih sederhana, dukungan untuk deteksi perangkat keras otomatis dan instalasi driver. Distribusi datang dalam bentuk rakitan langsung dengan lingkungan grafis KDE, GNOME dan Xfce. Manjaro menggunakan toolkit BoxIt-nya sendiri, yang dirancang sesuai gambar Git, untuk mengelola repositori. Selain repositori sendiri, ada dukungan untuk menggunakan repositori AUR (Arch User Repository). Pada versi 19.0, kernel Linux 5.4 diperkenalkan, versi terbaru Xfce 4.14 (dengan tema baru untuk Matcha), GNOME 3.34, KDE Plasma 5.17, Aplikasi KDE 19.12.2. GNOME menawarkan sakelar mode desktop dengan berbagai tema. Hingga versi 9.3 manajer paket Pamac telah diperbarui dan dengan dukungan default untuk paket mandiri dalam format snap dan flatpak disertakan, yang dapat diinstal melalui antarmuka manajemen aplikasi Bauh yang baru.

Rilis pertama Monado, sebuah platform untuk perangkat realitas virtual




OpenNET mengumumkan rilis pertama proyek Monado, yang bertujuan untuk menciptakan implementasi terbuka dari standar OpenXR. OpenXR adalah standar terbuka, bebas lisensi untuk akses ke platform dan perangkat untuk realitas virtual dan augmented reality. Kode proyek ditulis dalam C dan didistribusikan di bawah Lisensi Boost Software 1.0 gratis, kompatibel dengan GPL. Monado menyediakan lingkungan runtime yang sepenuhnya memenuhi persyaratan OpenXR, yang dapat digunakan untuk bekerja dengan virtual dan augmented reality di smartphone, tablet, PC, dan perangkat lainnya. Dalam kerangka Monado, beberapa subsistem dasar dikembangkan:

  1. mesin penglihatan spasial;
  2. mesin pelacakan karakter;
  3. server komposit;
  4. mesin interaksi;
  5. alat.

Perubahan Pemimpin Proyek Arch Linux di Arch Linux




Menurut OpenNET, Aaron Griffin mengundurkan diri sebagai manajer proyek Arch Linux. Griffin telah menjadi pemimpin sejak 2007, tetapi baru-baru ini dia tidak begitu aktif dan memutuskan untuk memberikan tempatnya kepada orang baru. Selama pemungutan suara pengembang, Levente Poliak terpilih sebagai pemimpin proyek yang baru. Ia lahir pada tahun 1986, merupakan bagian dari Tim Keamanan Arch dan menyertai 125 paket. Untuk referensi: Arch Linux, menurut Wikipedia, adalah tujuan umum independen distribusi GNU / Linux yang dioptimalkan untuk arsitektur x86-64, yang bertujuan untuk menyediakan versi stabil terbaru dari program-program mengikuti model peluncuran rilis.

Open Cybersecurity Alliance




ZDNet mengumumkan kedatangan OpenDXL Ontology, sebuah kerangka kerja yang dirancang untuk bertukar data dan perintah antara program terkait keamanan siber. Kerangka kerja baru, yang dirancang untuk menjembatani fragmentasi antara alat cybersecurity, diperkenalkan ke komunitas Open Source. OpenDXL Ontology dikembangkan oleh Open Cybersecurity Alliance (OCA), konsorsium penyedia cybersecurity termasuk IBM, Crowdstrike, dan McAfee. OCA mengatakan OpenDXL Ontology adalah "bahasa open source pertama untuk menghubungkan alat cybersecurity melalui sistem pesan umum." Ontologi OpenDXL bertujuan menciptakan bahasa bersama antara alat dan sistem keamanan siber, menghilangkan kebutuhan untuk integrasi kustom antara produk yang paling efektif ketika berinteraksi satu sama lain,antara sistem akhir, firewall dan banyak lagi, tetapi menderita fragmentasi dan arsitektur khusus vendor.



Brave archive.org




Menurut OpenNET, proyek Archive.org (Internet Archive Wayback Machine), yang telah menyimpan arsip banyak situs sejak 1996, berbicara tentang inisiatif bersama dengan pengembang browser web Brave, yang dapat meningkatkan akses Internet jika ada masalah dengan ketersediaan situs. Jika Anda mencoba membuka halaman yang tidak ada atau tidak dapat diakses di Brave, browser akan memeriksa keberadaan halaman di archive.org dan, jika ditemukan, tampilkan proposal untuk membuka salinan arsip. Fitur ini diimplementasikan dalam rilis Brave Browser 1.4.95. Ada add-on dengan fungsionalitas serupa untuk Safari, Chrome, dan Firefox. Pengembangan browser Brave dipimpin oleh Brendan Ike, pencipta bahasa JavaScript dan mantan kepala Mozilla. Browser didasarkan pada mesin Chromium,Ini berfokus pada memastikan privasi dan keamanan pengguna dan didistribusikan di bawah lisensi MPLv2 gratis.
