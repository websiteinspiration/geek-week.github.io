Emacs-6つの生産性トリック

以前、私はEmacs をC ++ IDEおよびその他の技術機器として使用することについて書きました。しかし、私はEmacsを使用していることや、他の多くのことを扱うことにあまり注意を払っていませんでした。正直なところ、ここで共有する機能がなければ、このエディターを完全に使用することはできませんでした。また、これらすべてをそのまま実行するために環境で使用した設定（文字通り、.emacsのコピー）についても説明します。

はじめに


tl; dr：特にイライラしている人は、このセクションをスキップして、Helmの設定に直接進むことができます。
Emacs-26.1をインストールし、ソースからコンパイルしました。あなたはそれを必要としません。すべてのパッケージは、Emacsパッケージマネージャーからインストールされます。実行：

M-x list-packages

MELPA で利用可能なパッケージのリストが表示されます。これについて聞いたことがない場合も心配しないでください。これは、Debian / UbuntuディストリビューションのDebianリポジトリのように、すべてのアドオンパッケージのリポジトリのようなものです。したがって、GIFのように、利用可能なパッケージの長いリストがあります。パッケージ



を選択すると、新しい画面が短い説明とともに表示されます。通常、クイックスタートの手順が含まれています。パッケージをインストールするには、iを押してからxを押すだけです。以下のパッケージでも同じことができます。

それが何であるかまだわからない場合は、すべてを削除して注意してください。ほとんどの場合、それは価値があります。説明は必要ないと思います。公式ウェブサイトの説明で十分です：
「HelmはEmacsフレームワークであり、ファイル名、バッファ名、および可能なオプションのリストから項目を選択する必要があるその他のアクションのインクリメンタル検索とオートコンプリートです」

デモ




(require 'helm)

(setq-default helm-M-x-fuzzy-match t)
(global-set-key "\C-x\C-m" 'helm-M-x)
(global-set-key "\C-c\C-m" 'helm-M-x)
(define-key evil-ex-map "x" 'helm-M-x)

(define-key evil-ex-map "b " 'helm-mini)
(define-key evil-ex-map "e" 'helm-find-files)

悪モード


EVILはEmacsの拡張可能なVIレイヤーの略です。これは明らかに論争の的になる大きなトピックであり、Emacsを使用するための純粋なユーザーシナリオから離れています。正直なところ、そのようなシナリオはありません。私の意見では、Emacsの力は主にそれを何かに変えることができることから来ています。私はviしかなくて育ったので、かなり使用されていましたが、うまくいきました。

(require 'evil)

;;;; define shortcuts for powerful commands

運命のテーマ


ここでは美学に焦点を当てますが、これは主観的なものです。とにかくすべてに満足している場合は、下にスクロールしますが、上の画像に感銘を受けた場合は、この情報が役立ちます。
Doom Themesは、エディターをよりモダンに見せるために役立ちました。時々、色が私を悲しくした（精神分析の瞬間）ので、Emacsの「同じテーマ」を探し始めました。zenburnを長い間使用していましたが、コントラストフォントが本当に好きであることに気付きましたが、派手でシャープさが少し低下しました。Doom Themes、特に最近のAtom IDEによく似ているdoom-molokaiをチェックしてください。必要な最小構成を以下に示します。私は盗んだ修正版を使います インターネットから。

(require 'doom-themes)

(require 'indent-guide)
(indent-guide-global-mode)
(set-face-background 'indent-guide-face "dimgray")

;; Global settings (defaults)
(setq doom-themes-enable-bold t    ; if nil, bold is universally disabled
      doom-themes-enable-italic t) ; if nil, italics is universally disabled

;; Load the theme (doom-one, doom-molokai, etc); keep in mind that each
;; theme may have their own settings.
(load-theme 'doom-molokai t)

;; Enable flashing mode-line on errors
(doom-themes-visual-bell-config)

;; Enable custom neotree theme
(doom-themes-neotree-config)  ; all-the-icons fonts must be installed!

(require 'doom-modeline)
(doom-modeline-mode 1)
