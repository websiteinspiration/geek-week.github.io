新着情報


  • 2019入学キャンペーン結果

この夏、ハブレに関するブログで、ITMO大学のカリキュラムについて話し、卒業生のキャリア成長の経験を共有しました。これらの資料は、今年の入学キャンペーンの過程に貢献しました-4,000人以上が郡政に申請し、12,000人以上が集まり、学部および専門コースで研究を始めました。

今年は、ITMO大学だけでなく他の大学からもロシアのさまざまな都市の卒業生が増加し、教育プロセスの最初の段階（学部生）で、平均試験スコアが92.7に上昇したことに気付きました。これは昨年より2.5ポイント多い。しかし多くの人にとって-オリンピックとコンテストの勝者（ITMO.STARSなど））-入学試験に合格する必要性を回避できました。また、トレーニングの初日から700人以上の奨学金が支給されます。

  • Google Code JamでのGennady Korotkevichの6回目の勝利

輝かしい大学院生の1人であり、ICPCを 2度受賞したGennady は、Google Code Jamの数万人の参加者の中で最高になりました。彼と一緒に、 ITMO大学のBoris MinaevとEvgeny Kapの卒業生が決勝に達しました。

  • 上海の大学の格付けのトップ100に入っています

この評価に基づいて作成された調査には、世界のさまざまな場所にある4000を超える大学が関与しています。この方法では、Web of Scienceによって索引付けされたジャーナルの科学出版物を考慮に入れました。大学をラン​​ク付けするために、科学論文は54の主題領域に結び付けられました。「電子工学および電気工学」から「自動化および制御」までです。提示された知識のいくつかの領域でポジションを改善し、他の多くの分野で高い結果を維持することができました。米国と中国は、大学がほとんどの主題分野でトップを占める国々のリーダーであり続けています。

3. «INTRODUCTION DAYS»


日時 ： 9月2〜3日日時と場所：学士号、修士号

特別オリエンテーションの目的は、ITMO大学で提供される機会について最大限の情報を提供することです。そんなイベントを面白くするために、大学の「駅」「ITMO.LOVE」「チームビルディング」「サイエンス」「グローバル」などを通じて、インタラクティブな「クエスト」という形で開催しています。

初年度の学部生のために、教育プロセスの複雑さとそれを最も効果的にする方法について話します。さらに、科学者向けの実際の売り込みセッションを準備しました。これは、キャリア開発に必要な関心領域と、国際的な研究チームに参加してインターンシップに参加する方法を理解するのに役立ちます。
