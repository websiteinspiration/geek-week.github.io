Faire les fentes de jeu dans le titre du site

Salutations, dans cet article, nous allons créer le jeu "Slots" à l'intérieur du nom du site (par exemple, TitleRun ). La mécanique du jeu sera très simple. appuyez sur le bouton - vous obtenez des emplacements aléatoires, si tous les emplacements correspondent - vous gagnez.







Pourquoi même faire des jeux au nom du site?



Oui, cela ressemble peut-être à quelque chose de très étrange, mais, néanmoins, c'est assez intéressant. En tout cas, plus intéressant que de faire le jeu de la manière habituelle.

Nous le rendons plus pratique



La première chose que je voudrais faire est une fonction qui remplace la logique winAnimation. Il affichera un message puis exécutera une autre fonction. Comme argument, il prendra un objet avec des textes, le nombre d'itérations, la vitesse et le rappel. 



const message = async ({text = [], count = 'infinity', speed = 1000, callback = null}) =>  {
    if(!text[0]) throw new Error('need array for text');
    let i = 0;
    let textIndex = 0;
    while(true) {
        if(count <= i && count !== 'infinity') {
            //    ,   
            if(callback) callback();
            break;
        }
        if(textIndex >= text.length) textIndex = 0;
        //      1    
        document.title = text[textIndex];
        i++;
        textIndex++;
        await sleep(speed)
    }
};
// 
message({text: ['️WINNER', 'WINNER️'], count: 3, speed: 800, callback: mainAnimation})




Je voudrais également créer un gestionnaire d'animation avec lequel vous pouvez les exécuter et les contrôler. Ce sera une classe avec un référentiel d'animation et plusieurs méthodes. 

class Animations {
    animations = [];
    setAnimation(animation, args = {}) {
        //   
        for(const item of this.animations) {
            item.play = false
        }
        //  ,    
        const Animation = this.animations.find(item => item.name === animation);
        if(!Animation) return;
        Animation.play = true;
        Animation.callback(args)
    };
    isPlay(animation) {
        return this.animations.find(item => item.name === animation).play
    }
    createAnimation(name, callback) {
        this.animations.push({
            play: false,
            name,
            callback,
        })
    }
}


Description des méthodes:

setAnimation - appelle l'animation.

isPlay est une fonction de condition pour vérifier si une animation est en cours de lecture.

createAnimation - Ajoute une animation au référentiel.

Possibilité d'ajouter un emplacement



Pour changer, ajoutons la possibilité d'ajouter un emplacement aléatoire.

(Ici, nous prenons simplement des emoji de l'API, les ajoutons au tableau et activons l'animation principale) 



const AddSlot = async () => {
    //       API
    animations.setAnimation('message', {text: ['Getting slot','Getting slot'], speed: 500});
    try {
        const data = await fetch(`https://emoji-api.com/emojis?access_key=`);
        const body = await data.json();
        const emoji = await body[Math.floor(Math.random() * body.length -1)];
        slots.push(emoji.character);
    } catch (e) {
        //   
        console.log(e);
        animations.setAnimation('none');
        document.title = 'Error, try again';
        await sleep(800);
    } finally {
        animations.setAnimation('main');
    }
};
