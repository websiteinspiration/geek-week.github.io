2019年のAppleは2000年のLinuxです

注：この投稿は、歴史の循環的な性質に関する皮肉な観察です。この観察自体には実用的な利点はありませんが、本質的には非常に正確であるため、聴衆と共有する必要があると判断しました。そしてもちろん、コメントでお会いしましょう。


先週、私がMacOS開発に使用しているラップトップは、Xcodeのアップデートが利用可能であると発表しました。インストールしようとしましたが、インストールプログラムを実行するのに十分な空きディスク領域がないとシステムから報告されました。 OK、たくさんのファイルを削除して再試行しました。すべて同じエラー。私は先に進んで、たくさんのファイルを削除し、付属物で仮想マシンの未使用のイメージをいくつか削除しました。これらの操作によって数十ギガバイトのディスク領域が解放されたため、すべてが機能するはずでした。私はバスケットを空にして、通常そうであるように何もそこに「ぶら下がらない」ようにしました。

しかし、これでも助けにはなりませんでした：同じエラーが発生しました。

ターミナルを立ち上げる時だと気づきました。確かに、からの情報によるとdf、40ギガバイトを超えるファイルを削除しただけですが、ディスクには8ギガバイトしかありませんでした（私は、グラフィカルインターフェイスでrmはなく、を使用してこれを実行したため、「存続」する機会はありませんでした。 WHO）。長い検索の結果、削除されたすべてのファイルがファイルシステムの「予約済みスペース」に移動したことがわかりました。そして、それらにアクセスして削除する方法はありませんでした。ドキュメントを読んだ後、OS自体がこれらのファイルを「より多くのスペースが必要なときにオンデマンドで」削除することがわかりました。これはあまり喜ばしいことではありませんでした。なぜなら、あなたは通常Appleのソフトウェアがそのようなことをエラーなしで行うと思っているのですが、システムがそれがすべきことを確実に実行しないからです。

何が問題なのかを理解しようと何度か試みた後、私はRedditの奥に隠されたスレッドに出くわしました。そこに、誰かが予約スペースをクリーンアップできる魔法の通路をリストしました。実際には、これらのパッセージには、ローンチなどが含まれていましたtmutil。さらに、立ち上げは、一見したところ、何をしたいのか意味も関係もない一連の議論によって行われます。しかし、驚くべきことに、このシャーマニズムは機能し、結局Xcodeを更新することができました。

血圧が正常に戻ったとき、私は圧倒的なデジャヴの感覚を感じました。この全体的な状況は、2000年代初頭のLinuxでの私の経験を痛感しました。何かが完全にランダムであり、適切で理解できる理由がないので、「すべてを元に戻す」唯一の方法は、いくつかのテーマ別フォーラムでコンソールの頑固なコマンドを掘り下げて、最善を尽くすことです。そして、この事実を認めた瞬間、目に見えました。

結局のところ、ファイルシステム領域の話は孤立したケースではありません。Parallelsはどこにでもあります。例えば：

外部モニター


Linux 2000：セカンドモニターを接続すると、ほとんどの場合失敗します。ファンは、モデルに関する完全な情報を提供していないのはメーカーのせいだと言っています。

Apple 2019：プロジェクター接続が失敗する可能性があります。彼らのハードウェアがアップルのテクノロジーのすべてのモデルで動作することを保証しないので、ファンはすべてのメーカーが責任があると言います。

ソフトウェアのインストール


Linux 2000：人種的に正しいソフトウェアのインストール方法は、1つだけです：yuzayパッケージマネージャー。他のことをするなら、あなたはナンセンスで、苦しむ必要があります。

Apple 2019：人種的に正しいソフトウェアのインストール方法は1つだけです。Appleストアを使用してください。他のことをすると、あなたはナンセンスで苦しむことになります。

ハードウェアの互換性


Linux 2000：箱から出して、ハードウェアの非常に限られたリストは、3Dビデオカードなどの人気のあるデバイスであっても機能します。機器がまったく機能しないか、機能が切り捨てられているか、機能しているように見えますが、明らかな理由もなく時々クラッシュします。

Apple 2019：Androidフォンなどの人気のあるデバイスについて話していても、非常に限られた範囲の機器がそのまま使用できます。機器がまったく機能しないか、機能が切り捨てられているか、機能しているように見えますが、明らかな理由もなく時々クラッシュします。

技術サポート


Linux 2000：問題の答えが検索結果の最初のページに表示されない場合は、それだけです。友達に連絡して助けを求めると、友達があなたの問題を検索エンジンに入力し、問題の最初のリンクから情報を読み取るだけになります。

Apple 2019：問題の答えが検索結果の最初のページに表示されない場合、それだけです。テクニカルサポートサービスに問い合わせても、問題が検索エンジンに入力され、最初の問題のリンクから情報が読み取られるだけです。

ノートブックの機能


Linux 2000：USBポートが3つ以上あるラップトップを見つけるのは非常に困難です。

Apple 2019：3つ以上のUSBポートを備えたラップトップを見つけることは非常に困難です。

死ぬまで愛する


Linux 2000：ペンギンファンは、彼らのシステムが最高であり、遅かれ早かれすべてのPCに搭載されることをはっきりと告げています。言及されているファンは、傲慢なコンピュータオタクです。

Apple 2019：Appleファンは彼らのシステムが最高であり、遅かれ早かれすべてのPCに搭載されることをはっきりと告げています。言及されているファンは、手にラテを持っている傲慢なヒップスターのデザイナーです。
