如何制作一个带有圆形工作面的交互式桌子？

出于对解决非标准问题的热爱，我想与读者分享解决方案的发展-媒体博物馆中的交互式圆桌会议。也许你们中的某些人将需要此选项。



背景-客户和设计师共同制定了一种解决方案，房间内的家具呈圆形，圆柱形，圆形（在极端情况下为椭圆形）。因此，多媒体设备在可能的情况下也应与该概念一致并且具有圆形形状。在房间的其中一个区域中，提供了交互式圆桌会议。但是将它们吸引到设计项目中是一回事，而要实现则是另一回事。

使它美观，实用且便宜


当您寻找非标准问题的解决方案时，价格问题并不总是首当其冲，但仍然不会失去其重要性。因此，您必须根据多种标准寻找客户最终会批准的各种选项：样式，功能和成本。

桌子设计


一张表-尽管看起来很简单，但实际上是一组以下设备：



基于多媒体设备中的现有技术，我们有以下选择。在表格设计的主要细节中，区别在于显示介质和传感器膜（在某些情况下为IR框）。

选项1-“ 16：9显示屏+矩形胶片”



一种简单的方法是将传统的16：9格式的显示器与触摸膜或交互式显示器一起使用。
优点 缺点
  • 简单的“组装”；
  • 应用现成的交互式模型的能力；
  • 对角线上的模型种类繁多；
  • 互动技术的选择（电容式，红外边框等）；
  • “开放式框架”结构（无外壳，“插入物下方”带有紧固元件）；
  • 成本。
  • 桌子是圆形的，但是图像和交互部分仅在显示器的表面上。
  • 如果不使用保护性玻璃，则嵌入式面板和桌子表面之间将存在间隙（灰尘），这在美观上是不令人满意的。

选项2-“ 1：1展示+矩形胶片”



一种简单但有趣的方法是使用带触摸膜的1：1（正方形）显示器。
据我所知，22平方英寸的矩阵是由三星UD22B生产的，仍由Leyard Planar AD22生产，其屏幕尺寸为387.4 x 387.4 mm，物理尺寸为395 x 395 mm。

优点 缺点
  • 简单的“组装”；
  • 异常的显示格式会引起注意（主观意见）；
  • 互动技术的选择（电容式，红外边框等）；
  • 成本是有争议的（因为只有一个制造商，所以成本可能很高）。
  • 型号范围始于22英寸；
  • 桌子是圆形的，但是图像和交互部分仅在显示器的表面上。
  • 如果不使用保护性玻璃，则嵌入式面板和桌子表面之间将存在间隙（灰尘），这在美观上是不令人满意的。

选项3-“投影仪+矩形胶片”



一种困难的方法，但是图像将是圆形的-使用短焦投影机，投影和触摸胶片。

目的是在圆的整个区域上显示图像（内容），并将胶片尽可能多地放置在圆的区域中。
优点 缺点
  • 图像在圆的整个区域上的投影;
  • 异常的显示格式会引起注意（主观意见）；
  • 成本。
  • 难以选择和放置短焦投影机；
  • 由于投影仪和镜头的选择和放置的特殊性，工作台的设计尺寸有限；
  • 结果，亮度和色彩饱和度不如使用LCD面板时的“强”，因此-您必须衰减工作台所在区域的光线，以实现所传输图像的“鲜度”；
  • 交互部分仅在传感器膜的表面上-您需要“正确”的内容。
    您可以在桌子的底部使用红外摄像机，但是，获得经验之后，就不再需要重复以前的“成功”了。

选项4-“投影机+圆形胶片”



可以选择使用短距投影仪，投影仪和圆形互动电影。
据我所知，只有葡萄牙公司Displax Round Skin生产这种胶片；我找不到其他制造商。



目标是制作一个交互式圆桌会议：在圆的整个区域上显示图像（内容），并以圆的形式施加电容式触摸膜。
优点 缺点
  • 图像在圆的整个区域上的投影;
  • 互动功能适用于整个圈子;
  • 异常的显示格式会引起注意（主观意见）；
  • 成本-与正方形显示器一样，​​这是一个有争议的观点（由于缺乏类似物，因此难以评估）。
  • 难以选择和放置短焦投影机；
  • 由于投影仪和镜头的选择和放置的特殊性，工作台的设计尺寸有限；
  • 结果，亮度和色彩饱和度不如使用LCD面板时的“强”，因此-您必须衰减工作台所在区域的光线，以实现所传输图像的“鲜度”；
  • 桌子的互动表面将需要在圆周上形成一个小的边框（约60毫米）。
