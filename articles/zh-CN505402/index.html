使用p5.js在3d对象上叠加2d纹理（第2部分-将图案应用于多维数据集）



这是项目的第二部分，其中我们在3d对象上施加了动态2d模式。您可以从第一部分（第1部分）开始，或者简单地将生成的代码复制到下面的几段中。

前一段时间，我创建了一个模式，该模式是使用p5.js库中的几行代码生成的。
p5.j​​s是2014年创建的Javascript库。它的主要目标是成为设计师，艺术家，老师和其他创意专业代表的程序设计世界的窗口。它同时支持动画和图表，可轻松转换为Web格式。

现在，我将告诉您如何制作最终零件-将图案应用于3d对象（在本例中为立方体）。

要从头开始工作，您需要在在线编辑器editor.p5js.org中创建一个项目，我们将继续更改以下代码：

let r=10

为了使我们的画布丰富，我们需要在创建画布的函数中添加WEBGL参数：createCanvas（400,400，WEBGL）。由于此类对象的参考点已在中心，因此我们可以（并且需要！）移除平移（200,200），该平移在平面图像中执行了校正。

为了使配色方案多样化并制作出更加精美的图像，让我们用变量r，a，c替换fill（）参数-现在随着无限draw（）循环的每次新迭代，椭圆的颜色都会略有变化。另外，让我们通过更改变量c和a的变化率来增加圆和倾斜角度之间的距离。

   =  + 0.06
  a = a + 0.3

作为最后的改进，让我们在模式代码之前添加push（），在其之后添加pop（）。通过这种方式进行框架处理可以使代码隔离-不会受到我们将创建的其他对象的影响。

let r=10
let a=0
let c=20

function setup() {
  background(220);
  createCanvas(400, 400, WEBGL);
  
}

function draw() {
  push()
  let x = r+c*cos(a);
  let y = r+c*sin(a);
  fill(r,a,c);
  ellipse (x,y,10,10);
  c = c + 0.09
  a = a + 0.8
  pop()
}



最后，我们得到了这样的东西。是时候创建一个多维数据集了。

在同一个draw（）函数中，在pop（）之后，添加框代码：

  push()
  box(200)
  pop()

现在，大小为200的盒子仅在一侧转向我们，看起来就像一个正方形。要估算数量，我们需要部署它。为此，将第四个变量top添加到已经存在的r，a，c中，其角度默认值为0。要使此参数影响立方体，请在第二次push（）之后但在创建框之前。添加三个函数：rotateX（），rotateY（），rotateZ（），每个函数都带有角度参数。

逻辑上，旋转角度为0度的对象不会旋转，因此，我们将可变角度的值添加为0.003（对于启动器来说就足够了）。结果，我们得到以下代码：

let r=10
let a=0
let c=20
let angle = 0

function setup() {
  background(220);
  createCanvas(400, 400, WEBGL);
  
}

function draw() {
  push()
  let x = r+c*cos(a);
  let y = r+c*sin(a);
  fill(r,a,c);
  ellipse (x,y,10,10);
  c = c + 0.09
  a = a + 0.8
  pop()
  
  push()
  rotateX(angle), 
  rotateY(angle), 
  rotateZ(angle)
  box(200)
  angle = angle + 0.003
  pop()
}

由于我们的背景仅创建一次（在设置中），因此-我们捕捉到第一个错误，我们的多维数据集在旋转时会擦除背景。



要解决这个问题，您可以在draw（）函数中简单地传递背景（200），但是，现在我们遇到了另一个问题-我们的圆像开始时一样消失了，因为现在每个新的循环都覆盖了它们，并且仅从虚无。

要解决此问题，我们需要在其他顶级变量的最开始处创建一个没有默认值的新变量（例如art）。

在setup（）函数内部，我们将其等同于createGraphics（400,400）函数，然后在其他地方使用它。
确保设置与创建Canvas时相同的参数以保持尺寸。

现在，在模式代码中，我们通过圆点将art变量添加到椭圆及其属性（例如，如果创建了它们，则为fill（）等）。 

art.fill(r,a,c);
  art.ellipse (x,y,10,10);

之后，仍然需要将图案作为纹理施加在立方体上。为此，请在多维数据集框代码内添加texture（艺术）属性。逻辑如下：我们创建了图形隔离对象艺术的本质，它调用了用于创建具有指定属性的模式的函数，然后，我们为art提供了作为属性参数的立方体纹理。

let r=10
let a=0
let c=20
let angle = 0
let art

function setup() {
  createCanvas(400, 400, WEBGL);
  art = createGraphics(400,400);
}

function draw() {
  background(220);
  push()
  let x = r+c*cos(a);
  let y = r+c*sin(a);
  art.fill(r,a,c);
  art.ellipse (x,y,10,10);
  c = c + 0.09
  a = a + 0.8
  pop()
  
  push()
  texture(art)
  rotateX(angle), 
  rotateY(angle), 
  rotateZ(angle)
  box(200)
  angle = angle + 0.003
  pop()
}



解决最后一个问题还有待解决：我们的模式从立方体的角开始出现。要从中心开始，我们可以简单地将立方体的宽度和长度的一半加到x和y参数上，以移动绘制图案起点的中心。

art.ellipse (x+200,y+200,10,10);
