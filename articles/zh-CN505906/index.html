来自overthewire.org的演练强盗

大家好，土匪看了先生的第一个季节。机器人，我认为它不比主角差，并为自己安装了Kali Linux，可以上网浏览互联网，以查找我兄弟手机上的裸女照片和照片。在启动了nmap三次之后，毫无疑问，生活不是电影，我决定对搜索进行更结构化的研究，并从信息安全和漏洞搜索的基础开始。

我的下一步是访问reddit.com上的r / hacking 并阅读提及overthewire.org的固定帖子和Bandit游戏，作为适合初学者的“黑客”之一。我确信读者对Capture The Flag游戏有一个想法（考虑到我发布的中心），但是以防万一，我会解释一下。您连接到远程计算机，并且在隔离的空间中使用站点上的建议来寻找连接到下一台计算机的密钥。此外，对于我自己和你们所有人，我将通过前十个级别，因为只记录下来以免忘记和与任何人都不分享是一种罪过。走！

PS所有密码都隐藏在扰流板后面，请阅读，不用担心。

0级


您需要通过bandit.labs.overthewire.org端口2220连接到主机，并使用bandit0密码以bandit0登录。说到做到。我们使用标准命令语法ssh。

:~$ ssh bandit0@bandit.labs.overthewire.org -p 2220
This is a OverTheWire game server. More information on http://www.overthewire.org/wargames

bandit0@bandit.labs.overthewire.org's password: <b>bandit0</b>
...
bandit0@bandit:~$

2级


下一级别的密码与名称一起存储在文件中，-这是唯一不同于上一级别的密码。我们使用相同的命令cat，但稍作更改：

4级


下一级的密码位于目录中的隐藏文件中inhere，为了在控制台中显示该密码，我们使用ls带有选项的命令-a。让我们转到目录。

5级


密码位于一个文件中，您可以从目录中（字面上）阅读该文件inhere。要获取有关文件的信息，我们将使用命令file。首先，让我们进入目录并查看其中的内容：

bandit4@bandit:~/inhere$ ls

所有文件均以开头-，因此我们使用第二级的语法并进行补充*以一次性获取有关所有文件的信息。
