2020年に知っておく必要がある配列を操作する15のJavaScriptメソッド

これは、ウェブサイトdev.toに公開された記事の翻訳です。投稿者：Ibrahima Ndaw。
オリジナルは著者のブログで公開されています。

JavaScriptの配列は、さまざまな要素を格納するために使用される特別なデータ構造です。組み込みのプロパティとメソッドを使用して、必要に応じてデータを追加、削除、反復、または処理できます。JavaScriptで配列を操作する方法を理解すると、専門的な開発スキルが次のレベルに上がります。

この記事では、JavaScriptの配列をより効率的に操作するのに役立つ15のメソッドについて説明します。



基本的に、パラメーターとして渡される関数を単純化することに注意してください。

// Instead of using this way
myAwesomeArray.some(test => {
  if (test === "d") {
    return test
  }
})
// We'll use the shorter one
myAwesomeArray.some(test => test === "d")

1.いくつか（）


このメソッドは、渡された関数で指定された条件を満たす配列の要素があるかどうかをチェックします。true少なくとも1つの要素がテスト対象の関数と一致する場合は値を返し、そうでない場合は値を返しますfalse。

const myAwesomeArray = ["a", "b", "c", "d", "e"]

myAwesomeArray.some(test => test === "d")
//-------> Output : true

2.減らす（）


このメソッドは、アキュムレータと値を引数として取る関数を取ります。関数をアキュムレータと配列の各値に適用するため、1つの値のみが返されます。

const myAwesomeArray = [1, 2, 3, 4, 5]

myAwesomeArray.reduce((total, value) => total * value)
// 1 * 2 * 3 * 4 * 5
//-------> Output = 120

3.すべて（）


このメソッドは、配列のすべての要素が、渡される関数で指定された条件を満たすかどうかをチェックします。true各要素がテストされる関数と一致する場合は値を返し、そうでない場合は値を返しますfalse。

const myAwesomeArray = ["a", "b", "c", "d", "e"]

myAwesomeArray.every(test => test === "d")
//-------> Output : false

const myAwesomeArray2 = ["a", "a", "a", "a", "a"]

myAwesomeArray2.every(test => test === "a")
//-------> Output : true

4.マップ（）


このメソッドは、関数をパラメーターとして取り、配列の各要素に対して指定された関数を呼び出した結果を含む新しい配列を作成します。常に同じ数のアイテムを返します。

const myAwesomeArray = [5, 4, 3, 2, 1]
myAwesomeArray.map(x => x * x)

//-------> Output : 25
//                  16
//                  9
//                  4
//                  1

5.フラット（）


このメソッドは、配列の配列を引数として取り、ネストされた配列を平滑化して最上位の配列にします。この方法は1つのレベルでのみ機能することに注意してください。

const myAwesomeArray = [[1, 2], [3, 4], 5]

myAwesomeArray.flat()
//-------> Output : [1, 2, 3, 4, 5]

6.フィルター（）


このメソッドは、関数をパラメーターとして取り、フィルター関数が引数として渡された配列のすべての要素を含む新しい配列を返し、値とともに返しますtrue。

const myAwesomeArray = [
  { id: 1, name: "john" },
  { id: 2, name: "Ali" },
  { id: 3, name: "Mass" },
  { id: 4, name: "Mass" },
]

myAwesomeArray.filter(element => element.name === "Mass")
//-------> Output : 0:{id: 3, name: "Mass"},
//                  1:{id: 4, name: "Mass"}

7. forEach（）


このメソッドは、配列の各要素に関数を適用します。

const myAwesomeArray = [
  { id: 1, name: "john" },
  { id: 2, name: "Ali" },
  { id: 3, name: "Mass" },
]

myAwesomeArray.forEach(element => console.log(element.name))
//-------> Output : john
//                  Ali
//                  Mass

8. findIndex（）


このメソッドは、関数をパラメーターとして取り、それを配列に適用します。要素が引数として渡されたチェック関数の条件を満たす場合、検索された要素のインデックスを返します。満足しない場合はを返します–1。

const myAwesomeArray = [
  { id: 1, name: "john" },
  { id: 2, name: "Ali" },
  { id: 3, name: "Mass" },
]

myAwesomeArray.findIndex(element => element.id === 3)
//-------> Output : 2

myAwesomeArray.findIndex(element => element.id === 7)
//-------> Output : -1

9.（）を見つける


このメソッドは、関数を引数として取り、それを配列に適用します。要素がチェック関数の条件を満たす場合、配列で見つかった要素の値を返します。それ以外の場合は、値を返しますundefined。

const myAwesomeArray = [
  { id: 1, name: "john" },
  { id: 2, name: "Ali" },
  { id: 3, name: "Mass" },
]

myAwesomeArray.find(element => element.id === 3)
//-------> Output : {id: 3, name: "Mass"}

myAwesomeArray.find(element => element.id === 7)
//-------> Output : undefined

10.ソート（）


このメソッドは、関数をパラメーターとして受け取ります。配列の要素を並べ替えて返します。

const myAwesomeArray = [5, 4, 3, 2, 1]

// Sort from smallest to largest
myAwesomeArray.sort((a, b) => a - b)
//-------> Output : [1, 2, 3, 4, 5]

// Sort from largest to smallest
myAwesomeArray.sort((a, b) => b - a)
//-------> Output : [5, 4, 3, 2, 1]

11.連結（）


このメソッドは、2つ以上の配列/値を組み合わせて、新しい配列を返します。

const myAwesomeArray = [1, 2, 3, 4, 5]
const myAwesomeArray2 = [10, 20, 30, 40, 50]
myAwesomeArray.concat(myAwesomeArray2)
//-------> Output : [1, 2, 3, 4, 5, 10, 20, 30, 40, 50]

12.塗りつぶし（）


このメソッドは、開始インデックス（デフォルト0）から終了インデックス（デフォルトarray.length）まで、配列のすべての要素に同じ値を設定します。

const myAwesomeArray = [1, 2, 3, 4, 5]

// The first argument (0) is the value
// The second argument (1) is the starting index
// The third argument (3) is the ending index
myAwesomeArray.fill(0, 1, 3)
//-------> Output : [1, 0, 0, 4, 5]

13.含む（）


このメソッドはtrue、配列に特定の要素が含まれている場合は値を返し、含まれていない場合は値を返しますfalse。

const myAwesomeArray = [1, 2, 3, 4, 5]

myAwesomeArray.includes(3)
//-------> Output : true

myAwesomeArray.includes(8)
//-------> Output : false

14.逆（）


このメソッドは、配列内の要素の順序を逆にします。最初の要素が最後になり、最後が最初になります。

const myAwesomeArray = ["e", "d", "c", "b", "a"]

myAwesomeArray.reverse()
//-------> Output : ['a', 'b', 'c', 'd', 'e']

15. flatMap（）


このメソッドは、配列の各要素に関数を適用し、結果を新しい配列に平滑化します。メソッドflat()とメソッドmap()を1つの関数に結合します。

const myAwesomeArray = [[1], [2], [3], [4], [5]]

myAwesomeArray.flatMap(arr => arr * 10)
//-------> Output : [10, 20, 30, 40, 50]

// With .flat() and .map()
myAwesomeArray.flat().map(arr => arr * 10)
//-------> Output : [10, 20, 30, 40, 50]
