ADAM-6700 - Universelles Linux-Objektkommunikationsmodul



Die Module der ADAM-Serie sind das bekannteste Produkt unseres Unternehmens. Sie wurden 1992 eingeführt und entwickeln sich ständig weiter, wobei ein klassisch erkennbares Design beibehalten wird. Heute ist es ein leistungsstarkes multifunktionales Gerät, das mehrere Geräteklassen zur Lösung einer Vielzahl von Problemen kombiniert.

ADAM-6700 ist eine neue Serie von Geräten, die auf dem Cortex-A8-Prozessor mit Linux RT (Echtzeit) basieren. Es kombiniert die Funktionen eines Kommunikationsmoduls mit einem Objekt, einer speicherprogrammierbaren Steuerung (SPS), einem Gerät zum Sammeln und Verarbeiten von Daten sowie einem IP-Gateway für digitale und analoge Schnittstellen.

Es verfügt über einen integrierten Node-RED-Server und kann über eine Webschnittstelle konfiguriert werden. Dank der offenen Plattform können Sie auf dem Modul Clientanwendungen in C ausführen und jede Arbeitslogik programmgesteuert implementieren.



Technische Eigenschaften


Die neuen Geräte der ADAM-6700- Serie basieren auf der ARM-Architektur und sind ein produktives System, das lokale Transformationen, Speicherung und Datenverarbeitung ermöglicht. Sie führen eine Version des Linux-Betriebssystems mit Unterstützung für Echtzeitvorgänge aus, sodass sie für zeitkritische Vorgänge verwendet werden können.

  • Betriebssystem: Linux RT
  • CPU: Cortex-A8 32-Bit 1 GHz
  • RAM: 512 MB
  • LAN-Ports: 2 unabhängig
  • USB-Slave: 1 (Konsolenanschluss zur Konfiguration)
  • USB-Host: 1 (zum Anschließen von Geräten)
  • Eingangs- / Ausgangsanschlüsse
  • Digitale Eingänge: 5 (Modell ADAM-6717 ) 12 (Modell ADAM-6750 )
  • Digitale Ausgänge: 4 (Modell ADAM-6717 ) 12 (Modell ADAM-6750 )
  • Analoge Eingänge: 8 (Modell ADAM-6717 )



Anpassung


Die Erstkonfiguration des Geräts kann über das ADAM / Apax.NET-Dienstprogramm erfolgen , in dem Sie die Netzwerkbetriebsarten und die IP-Adresse festlegen können, oder über die Weboberfläche. In der Weboberfläche stehen zwei Gerätekonfigurationsoptionen zur Verfügung: die proprietäre Advantech-Shell und Node-RED. Auf der Startseite müssen Sie eine der Optionen auswählen.


Die

Node-Red- Shell-Weboberfläche zur Auswahl einer Shell ist eine beliebte Open-Source-Umgebung für die visuelle Programmierung und Datenflusssteuerung. Die Geräte der ADAM-6700- Serie verfügen über einen integrierten Node-RED-Server. Die Programmierung kann entweder direkt über einen Webbrowser oder durch Importieren einer Konfiguration erfolgen.


Knoten-ROTES Webinterface

Digitale und analoge E / A-Schnittstellen sind über die API für die Sprache C verfügbar und können zur Implementierung eigener Protokolle für die Integration in Kundengeräte verwendet werden.

Konfiguration über Micro USB



Zur Vereinfachung der Konfiguration kann der ADAM-6700 über Micro-USB als Slave-Gerät an einen Computer angeschlossen werden. In diesem Fall wird das Modul über den USB-Anschluss mit Strom versorgt, was die Einrichtung erheblich vereinfacht, da kein Anschluss über die Klemmen erforderlich ist. Dies bedeutet, dass es von einem Techniker ohne Werkzeuge und zusätzliche Geräte wie Netzteile und Schraubendreher ausgeführt werden kann. Im Hostsystem ist das Gerät als virtueller COM-Port definiert.

USB-Host-Schnittstelle


Der ADAM-670 0 verfügt über einen vollständigen USB-A-Anschluss zum Anschließen von Geräten im Host-Modus. Hier können Sie externe Laufwerke, USB-Sticks sowie Erweiterungsmodule wie das ADAM 600W WiFi-Modul anschließen. Bei Bedarf kann der Kernel des Systems mit Unterstützung für alle Treiber für USB-Geräte kompiliert werden.

WiFi-Modul ADAM-600W


Das ADAM-600W-Erweiterungsmodul erweitert die ADAM-6700- Serie um WiFi-Unterstützung . Die Verbindung erfolgt über USB und erfordert keine Installation zusätzlicher Software. Nach dem Anschließen des ADAM-600W-Moduls wird das Menü zum Einrichten der WLAN-Verbindung in der Weboberfläche verfügbar.

Der ADAM-6700 mit WiFi-Erweiterungsmodul kann als Gateway für die Verbindung anderer über Ethernet mit dem Server verbundener Geräte dienen.

Fazit


Universelle Geräte der ADAM-6700- Serie können zur Lösung einer Vielzahl von Automatisierungsaufgaben, Datenerfassung und Peripherieverwaltung verwendet werden. Ein Multitasking-Betriebssystem schränkt den Entwickler beim Schreiben komplexer Anwendungen nicht ein. Die Unterstützung der offenen Node-RED-Plattform und die Möglichkeit zur Konfiguration über die Weboberfläche verringern die Eintrittsschwelle und ermöglichen Entwicklern das Arbeiten in einer vertrauten, vertrauten Umgebung.
