Server virtual abadi

Nama saya Leonid, saya adalah pengembang situs Cari VPS. Sesuai namanya, proyek ini berspesialisasi dalam membantu menemukan server virtual, jadi saya mencoba melacak semua inovasi di bidang ini.



Salah satu produk terbaru yang disajikan oleh beberapa host adalah server virtual abadi, yaitu, VPS / VDS dengan pembayaran satu kali: pengguna membayar jumlah yang tetap dan dapat menggunakan server seperti itu selama diperlukan.







Saya sudah terbiasa dengan trik pemasaran seperti "lalu lintas tidak terbatas" atau "situs dengan jumlah tidak terbatas" untuk waktu yang lama, dan saya merasa normal tentang hal-hal seperti itu, meskipun saya mengerti bahwa sebenarnya tidak ada yang tidak terbatas, dan lalu lintas akan dibatasi dengan kecepatan saluran maksimum maksimum, dan tidak terbatas jumlah situs dapat mencapai batas pada jumlah file atau sumber daya yang dikonsumsi. Kadang-kadang ada juga penawaran dengan ruang disk yang tidak terbatas (ini terutama berlaku untuk shared hosting), tetapi paling sering ada banyak peringatan bahwa tarif semacam itu tidak dapat digunakan untuk memposting gambar, penyimpanan file, dan proyek lain yang menghabiskan banyak ruang.



Semua teknik pemasaran di atas memiliki satu perbedaan dari server abadi: pengguna membayar terus-menerus.



Dalam kasus server yang kekal, situasinya justru sebaliknya: hoster menerima jumlah (meskipun berkali-kali lebih tinggi daripada sewa selama sebulan) sekali dan selanjutnya memberi pengguna hak untuk menggunakan server tanpa batas waktu. Saya melihat banyak kritik terhadap tarif seperti itu, pada dasarnya semua itu tidak menguntungkan atau fakta bahwa hoster bisa bangkrut atau pada beberapa titik mengatakan bahwa tarif berhenti menjadi abadi.



Mari kita coba mencari tahu bagaimana model bisnis ini memiliki hak untuk hidup.



Saat ini saya tahu tiga hosters yang menyediakan server abadi seperti itu, tetapi saya yakin bahwa di masa depan jumlah mereka hanya akan bertambah:

Vdsina Eternalhost Hosting cPanel
Masing-masing hosters memiliki beberapa tarif abadi untuk dipilih dengan jumlah sumber daya yang berbeda dan untuk uang yang berbeda. Tarif abadi termurah ditawarkan oleh EternalHost untuk 5.990 rubel, itu termasuk 1 inti prosesor, 1 GB RAM dan 16 GB pada drive NVMe SSD. Host lain menawarkan karakteristik serupa dengan pengecualian ruang disk: VDSina memiliki 30 GB disk untuk 8640 rubel, dan Hosting cPanel menawarkan 10 GB disk untuk 8800 rubel.



Cara termudah untuk memahami seberapa menguntungkan penjualan seperti itu bagi sang hoster (dan karenanya berapa banyak risiko yang dimiliki klien di masa depan yang tidak mendapatkan apa - apa dari donat alih-alih server) adalah menghitung berapa bulan server semacam itu akan membayar jika klien membayar setiap bulan.

Dari sisi tuan rumah, semuanya ternyata cukup baik: klien membayar segera selama 3 tahun di muka, yang dapat disebut semacam investasi. Selama 3 tahun ini, peralatan tempat klien berada kemungkinan akan membayar, sehingga hoster perlu membayar hanya untuk server jika tidak ada DC, atau hanya biaya sewa dan staf jika ada DC. Plus, ada kemungkinan bahwa setelah beberapa waktu pengguna akan berhenti menggunakan server, dan tidak ada yang membatalkan overselling, sehingga biaya penempatan pengguna seperti itu akan relatif rendah.



Untuk pengguna, kondisinya terlihat cukup menggoda: dibayar dan gunakan sebanyak yang Anda inginkan. Tawaran ini bermanfaat, tetapi menurut saya ada beberapa risiko:



  1. Pada titik tertentu, sumber daya dari tarif yang dibeli mungkin tidak cukup, dan hoster mungkin tidak memberikan biaya tambahan dan beralih ke tarif lain.
  2. Saya hampir yakin bahwa hosters tidak akan mengizinkan Anda untuk mentransfer server seperti itu dari akun ke akun, jadi kemungkinan besar Anda tidak akan bisa "menjual" server abadi.
  3. Sesuatu dapat terjadi pada sang hoster, dan kemudian Anda hampir tidak dapat mengharapkan pengembalian uang yang diinvestasikan. Ada sangat, sangat banyak kasus ketika hoster kecil dan tidak terlalu meninggalkan bisnis, terutama sejumlah besar dari mereka pada tahun 2019.
