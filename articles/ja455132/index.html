骨董品：Philips Velo 1、テクノロジーマニアの悪夢

英語には「アーリーアダプター」というものがあります。ロシア語に適切に翻訳できるかどうかはわかりませんが、頭に浮かぶのは「急いで-人を笑わせる」ということだけです。アーリーアダプターは、技術の熱狂者であり、生産開始時に避けられない多くの問題や問題の解決を待たずに、できるだけ早く自分でテストしたいと考えています。 2007年、米国から持ち込まれたそのような愛好家は、オペレーターとロシア化されていない最初のiPhoneに閉じ込められました。現在、彼らは3DプリンティングとVRテクノロジーをマスターし、柔軟な画面を備えたスマートフォンを購入しています。


一般的に、彼らは実際に既製のテスト済みデバイスとして販売されている製品のベータテストに参加します。しかし、違います。まず、高い故障率に遭遇し、運が悪ければ保証付きの購入を3回変更します。次に、致命的な欠陥を回避しようとします。バッテリーが1時間以内に消費されてから、アプリケーションにバグがあり、それから何か他のものがあります。 1年後、第2世代のデバイスが登場し、夜通しの変更により、コストのかかる進歩のピークが、時代遅れで不適切でバグの多い過去の共有に変わります。しかし、あなたは文句を言っていません。つまり、あなたは文句を言いますが、それでも満足しています。結局のところ、そのようなデバイスを持っている人はほとんどいませんし、あなたも持っています。

1996年、マイクロソフトはウェアラブルデバイス市場に大規模な攻撃を開始しました。1997年に、第1世代のデバイス（キーボードPDA）が登場しました。その中でも、パフォーマンスと機能、および可能なアップグレードの点で優れています。Pocket PC Philips Velo 1は、開拓者の呪いから逃れられず、すぐに時代遅れになり、多くの部分が壊れ、当然のことながら忘れられました。多くの個人的な理由で、私はこのPDAを購入し、22年前のモバイルWindowsがどのようなものかを自分で体験することにしました。

私は古い鉄の収集家の日記をリアルタイムで電報に記録します。

タイムマシンをオンにする


ハンドヘルドコンピューターの開発の頂点は、現代のスマートフォンと見なすことができます。21世紀の20世紀になって初めて、数億枚のコピーが販売されるようになりました。 90年代には、携帯電話とPDAはまだ1つのデバイスに統合されておらず、せいぜい相互に接続することが可能でした。 PDAは年間数百万、せいぜい数十万では販売されませんでした。


1997年には、2つのデバイスが非常に優れていました。これは、便利なキーボード、EPOC32高速専用オペレーティングシステム、およびスマートビジネスツール（連絡先、カレンダー、タスク管理）を備えたPDAであるPsionシリーズ5です。そして、Palmハンドヘルドコンピュータ（当時はPalmPilotとして知られていました）は、スタイラスを使用したテキスト入力を備えた、キーボードなしのデバイスの最初の成功した実装です。明らかに制限された機能、最小限のメモリ（2メガバイト）を備えたコンピュータのプレフィックス、安価で限られた一連のタスクを実行するデバイス。写真-私のコレクションで最も古いPalm PDA、1999年の予算モデルIIIe。1997年のモデルと大差ありません。


マイクロソフトによると、当時、デスクトップOS市場で支配的な地位を占めていたため、ハンドヘルドコンピューターは大型コンピューターとほぼ同じになるはずです。したがって、デバイスメーカーの要件：480x240の解像度の大画面、シリアルポートと赤外線ポート、PCMCIAスロット（！）、フルキーボード。 Windows 95のユーザーエクスペリエンスはほぼ完全に再現されています。同じデスクトップにアイコン、タスクバー、スタートボタン、エクスプローラー、時計が隅にあります。


目撃者によると、Palm（90年代後半-誰もが認める市場リーダー）とPsionの両方が、Microsoftの市場参入を恐れていました。マーケティング予算と制作パートナーを見つける能力により、比較的小規模な競合他社を簡単に倒産させることができました。両社はこの脅威に対応し、自社のOSを他の企業にライセンス供与し始めました。どちらの場合も、ライセンスプロセスにより、企業内のソフトウェアとハ​​ードウェアの開発者間の円滑なやり取りが妨げられましたが、これは別の記事のトピックです。マイクロソフトは1996年の秋にWindows CE 1.0を発表しました。これは、携帯爆撃の感覚と信じられないほどの多様性のファンを約束します。実際、それはあまりうまくいきませんでした。WindowsCEプラットフォーム上のデバイスのすべてのメーカーは、1998年末までにハンドヘルドコンピューターの市場の25％しか占有できませんでした。

しかし、それは微視的市場での競争でした。1997年にデータを処理するための新しいデバイスを選択した場合、Microsoftプラットフォームに基づくデバイスであるPalmからPsionを購入できます。そして、安価な電子手帳やペニーペーパーノートブックを購入することもできます。最後のオプションは、誰もが通常のコンピュータを購入できるわけではない状況で、ほとんどの人が選択しました。CCP-見通しが不明瞭な場合、進歩に多額の費用を払うことをいとわない愛好家の選択でした。しかし、推測ではなく、デバイスを見てみましょう。どうですか？


いいね！最初、アクセサリなしでデバイスを購入しました。ドッキングステーションのみが含まれていたため、PDAを専用ポートと同期させることはできませんでした。しかし、このインスタンスはほとんど新しいもののようでした。PDAの仕様は次のとおりです。

マイクロソフトは、メーカーに仕様に厳密に従うことを強制することはできませんでした。たとえば、HPは画面が大きいため（横480ピクセルではなく600ピクセル）、PDAを割り当てることにしました。フィリップスはPCMCIAスロットをVelo 1に統合することを拒否し、代わりに下部のプラグインモジュールに移動しました。プロセッサがより強力になり、メモリがより高速になる可能性があるため、Velo 1は1997年の初めにPC Magazineテストで優勝しました。確かに、すべての第一波デバイスは調査に参加しませんでした。


Philips PDAのもう1つのユニークな機能は、内蔵モデムです。以前のバージョンでは、画面の近くにトリッキーなコネクタがあり、電話ケーブルが直接挿入されていました。電話があれば、オンラインにしたり、追加のデバイスなしでBBSに接続したりできます。他のメーカーは、ユーザーがPCMCIAスロットにモデムをインストールすることを期待していたか、携帯電話を接続するためのアダプターを販売していました。


第5シリーズのサイオンとは異なり、Velo 1は主に「電卓」キーボードが異なります。これらのラバーボタンを入力してもすぐには機能しませんが、レイアウトはラップトップのレイアウトと完全に一致しています。このため、たとえばキリル文字を入力するときに、文字の一部が奇妙で予期しない場所に配置される問題はありません。


ビルトインPCMCIAを放棄したフィリップスは、Velo 1に拡張カード用の2つのスロットを提供しました。1つはRAMの容量を増やすように設計され、もう1つは不揮発性フラッシュドライブを挿入できます。両方のタイプのカードの標準は同じでした。コンパクトフラッシュおよびSD / MMCカードとの競合で失敗した、今ではほとんど知られていないミニチュアカードです。


私が最初に会ったとき、率直に言って、デバイスの何が問題だったのかは明らかではありません。なぜそれらは比較的売れ行きが悪かったのですか？デスクトップのMS WordとExcelの優れたコピー、電子メールクライアント、および組み込みブラウザー（PsionにもPalmにもなかったため、個別に購入する必要がありました）。いよいよソリティア！ 90年代後半、PalmはMicrosoftのマーケティング活動に対抗しようとしました。パートナーや業界の専門家と会って、実際のタスクについてデバイスを比較します。たとえば、電話帳で番号をすばやく見つけるにはどうすればよいですか。 Palmがスタイラスで2、3回ストロークする必要がある場合、Microsoftはインターフェイスが重いため、まったく特別な操作が必要です。


Philips Velo 1は、その汎用性だけで興味深いものです。ラップトップのように、よりコンパクトになっています。直接インターネット接続、メールおよびWebを操作する機能。ただし、サブブラインドのモノクロ画面、4メガバイトのメモリのみが対象であり、プログラムの実行とデータの格納の両方に使用されることを忘れないでください。理論的には、このようなデバイスはほぼメインコンピュータになる可能性があります。実際には-まあ、そうではありません。


ビンテージPDAを購入してから数週間後、私はコレクターの運に恵まれました。付属品一式が揃った安価な2台目のデバイスを箱に入れて注文しましたが、バッテリーの液漏れで破損しました。分解して酸化物を取り除く必要があります。そうしないと、一度電源が入ります。これらの初期のPDAのハードウェアの信頼性について話す時が来ました。


私は見てお勧めします、このいずれかをPhilips Velo 1の所有者のサイトのアーカイブコピーです。彼は、デバイスに対する彼の苦痛をかなり詳細に説明していますが、新機能と便利さを賞賛していることを忘れていません。「ぶら下がっている」画面のため、保証期間中にPDAを何度か交換する必要がありました。ディスプレイの金属マウントが信頼できなくなり、ケースのプラスチック部分から飛び出しました。後でこの問題は解決されましたが、別の問題が発生しました-時間の経過とともに、メカニズム全体が機能しなくなります。そのため、新しいVelo 1は開いた状態ではまったく固定されておらず、完全に折りたたまれています。YouTubeのレビューから判断すると、これはよくある問題です。しかし、これだけではありません。タッチスクリーンのフリーズとデータ損失の両方がありました。「標準」機能は、モデムまたはPCMCIAモジュールの動作モードでした-規定の10〜12時間ではなく、数分でバッテリーを消耗しました。

Windows CE 2.0


2番目のVelo 1の最も重要な成果物は、Windows CE 2.0へのアップグレードキットでした。フィリップスベロ1は、1997年9月に比較的遅れて幅広い販売に参加しました。同年11月にすでにマイクロソフトはアップデートを発表した。 OSの新しいバージョン（詳細はこちら）では、カラースクリーン、TrueTypeフォントのサポートが導入され、Microsoft Powerpointのポータブルバージョンが追加され、PCMCIAカードのサポートが拡張されました。キーボードレスPDAのサポートが初めて登場しました。それらはPalm PCと呼ばれ、激しい抗議の後、PalmはPalm-sizeに名前が変更されました。第一世代のデバイスは文字通り半年で時代遅れになり、すべてがはるかに新しいバージョンに更新される可能性があります。新機能により、システム要件は大幅に増加しました。2メガバイトのRAMでは不十分でした。


いくらですか？最初はフィリップスはこの質問に対する答えを知りませんでした。 Velo 1の所有者は、4メガバイトのRAM拡張カードを購入するようにアドバイスされました。多くの人がこれを行った後、8メガバイトのカードが必要であることが判明しました（ボードにはんだ付けされた場合、12メガバイト）。その結果、Windows CE 2.0へのアップグレードキットは99ドルで販売され、2枚の拡張カードが含まれていました。8メガバイトのRAMと、OS自体のROMです。かなり重要なシャーマニズム（私にはない外部電源が必要であり、外部ROMカードからのロードを使用するためにバッテリーを巧妙な方法で挿入および削除する必要がありました）の後、更新されたオペレーティングシステムを受け取りました。しかし、カラースクリーンではありません。


Windows CE 2.0にアップグレードすると、PDAに慣れるための最終段階であるクラックのインストールに進むことができました。 WinCE 1.0のデバイスは市場に出回っているため、2019年に互換性のあるソフトウェアを見つけるのは非常に困難です。 Windows CE 2.0以降で実行されているPDAの場合、McCenter Webサイトにはまだ無料の亀裂があります（アーカイブを保存してくれたことに再度感謝します）。


コンピューターと同期してソフトウェアをインストールするには、H / PC Explorerが必要です。 1999年には、Microsoft Activesyncという通称が付けられました。このプログラムの3番目のバージョンであるWindows 98をサポートする最新バージョンは、iPaq hx4700での実験用にすでにインストールされています。古いVelo 1では問題なく動作しましたが、その前にPDA自体の設定で説明を読んで「接続を作成」する必要がありました（そのため、彼らはそうしました！）。接続はシリアルポートを経由するため、同期は非常にゆっくりと行われます。しかし、Palm OSベースのデバイスの場合のように、キリル文字でのドキュメントのエンコードにそのような問題はありません。

最初のバージョンと2番目のバージョンの両方のWindows CEは、Velo 1での実行がかなり低速です。開始プログラムが標準である場合の数秒の遅延。それと比較すると、それらの年のPalm OSは信じられないほど高速であり、EPOC32で十分に最適化されたPsionは、ユーザーコマンドへの迅速な応答という素晴らしい効果をもたらしました。 Windows CEは、最新世代のキーボードPDA（200メガヘルツ未満のプロセッサと64メガバイトのRAM）で最適に動作します。 Velo 1は、ユニバーサルモバイルOSの基盤を構築する典型的なパイオニアです。理論的には、彼は多くの能力がありますが、当時存在していた競合他社よりも少し悪いビジネスに必要な特定の機能に対処できます。しかし、1桁も悪くはありません。今では、90年代のすべてのハンドヘルドコンピュータがモバイル中世として認識されています。


Velo 1の可能性をさらに探求する予定です。モデムを介して、場合によってはイーサネットを介してインターネットに接続し、適切なソフトウェアを選択します。 20年前の基準ではありますが、何か役に立つものにふさわしい博物館のコピーを作成したいと思います。それ以外の場合、可能性のすべての種類から、唯一の不変のソリティア「コシンカ」が残っています。それまでの間、1997年から15人のウィッシュリストのリストの最後の項目の前にチェックマークを付けました。まだ何も買う余裕がなかったので、そのときは何でも夢見ることができました。しかし、何らかの理由で、私は何よりもこれらの科学技術の成果を望んでいました：ラップトップ、多機能ハンドヘルドコンピューター、ミニディスクプレーヤー、デジタルカメラ。写真の遺物の総費用は約3,700ドルで、現代のお金では約6,000ドルです。夢が叶う、ちょっと待ってください！
