Como prometemos na primeira parte do artigo , essa continuação é dedicada à mudança independente dos ícones nos telefones Snom.



Então, vamos começar. Etapa 1, você precisa obter o firmware no formato tar.gz. Você pode fazer o download em nosso recurso aqui . Todos os ícones snom estão disponíveis e incluídos em cada versão do firmware.



Nota : observe que cada versão do firmware contém arquivos de configurações específicos para esta versão e modelo de telefone. O uso de arquivos de configuração que não correspondem ao firmware ou telefone causará problemas.

Etapa 2, preparando ícones para os telefones. Como você sabe, os telefones Snom vêm com uma tela colorida e monocromática, respectivamente, e os ícones diferem.



I. Alterar ícones para telefones com visor colorido



Ícones e imagens em telefones com telas coloridas são armazenados no formato PNG. Isso permite que eles sejam editados facilmente em quase todos os editores de imagens modernos. No entanto, após a edição, é recomendável otimizar os arquivos png usando ferramentas como optipng, pngquant ou pngcrush para remover qualquer informação redundante e otimizar o tamanho do arquivo.



Tamanhos da imagem do ícone:



  • Ícones principais sensíveis ao contexto 24x24px
  • SmartLabel 24x24px e 18x18px
  • Ícones na barra de título (Ícones da barra de título) 18x18px
  • Ícones do menu 18x18px
  • Durante uma chamada (Ícones da tela de chamada) 18x18px - 48x48px
  • Formato de arquivo: PNG


Depois de criar os ícones desejados, faça o download para o seu telefone. Existem duas maneiras de fazer o download:



  1. Via interface web no modo manual
  2. Usando a promoção automática

Como você pode ver no modo manual, tudo é bem simples, mas se você precisar trocar vários telefones, esse processo levará muito tempo. Portanto, passamos para a segunda opção.



Opção dois - download via provisionamento automático.



Primeiro, você precisa criar um arquivo no formato tar a partir do arquivo customizing.tar.gz baixado anteriormente . Ao criar um arquivo morto, exclua todos os arquivos e diretórios que você não precisa alterar, mas mantenha a estrutura de diretórios .

Nota . Você não precisa arquivar todos os arquivos que estavam originalmente no arquivo morto. É suficiente e recomendado que você arquive apenas os arquivos que modificou. Quanto mais arquivos você colocar no arquivo, mais tempo levará para o telefone configurá-lo.


Em seguida, executamos várias etapas:



1) crie um arquivo XML, por exemplo, branding.xml e copie-o no seu servidor Web (HTTP), ou seja, http: //yourwebserver/branding.xml: 



<?xml version="1.0" encoding="utf-8" ?>
<settings>
 <uploads>
  <file url="http://yourwebserver/branding/branding.tar" type="gui" />
</uploads>
</settings>


2) Iremos para a interface da web do telefone na seção Avançado -> Atualizar -> Configuração de URL e forneceremos um link para nosso arquivo yourwebserver / branding.xml















3) Reiniciaremos o telefone e apreciaremos o resultado.

II.



Os ícones em dispositivos monocromáticos não são armazenados em arquivos de imagem comuns, como .png ou .jpg, mas são fontes de bitmap que incluem todos os ícones usados ​​na interface do usuário. Na área de uso privado da tabela unicode, iniciada com U + EB00, os ícones snom são definidos e podem ser modificados diretamente usando ferramentas como o Font Forge .



A abertura de um arquivo de fonte de bitmap com o Font Forge deve mostrar uma lista de ícones usados. O conteúdo real dos arquivos depende da versão do telefone e do firmware:







Especificação do ícone para telefones com tela monocromática.



Para os modelos D305, D315, D345, D385, D745, D785, D3, D7:



  • Ícones principais sensíveis ao contexto 17x17 - Linha de base x → 0 / y → -2
  • Ícones da barra de título 17x17 - Linha de base x → 0 / a → -2
  • Ícones do painel de etiquetas 17x17 - Linha de base x → 0 / a → -2
  • Tamanho máximo de ícones 32x32


Para os modelos D120, D710, D712, D715, D725:



  • Ícones principais sensíveis ao contexto 7x7 - Linha de base x → 0 / y → 0
  • Ícones da barra de título 7x7 - Linha de base x → 0 / y → 0
  • Ícones SmartLabel 7x7 - Linha de base x → 0 / y → 0
  • Tamanho máximo de ícones 32x32


Depois de criar a "imagem" necessária e exportá-la do Font Forge, você precisa usar as seguintes configurações:







Após a exportação, crie um arquivo tar contendo o arquivo recém-criado com o nome do arquivo a ser substituído.



tar -cvf fonts.tar fontfile.bdf



Como na verdade estamos alterando não as imagens, mas a fonte, carregue-a através da promoção automática como fonte, especificando nas configurações do arquivo xml: 



<?xml version="1.0" encoding="utf-8" ?>
<settings>
 <uploads>
  <file url="http://192.168.23.54:8080/fonts.tar" type="font" />
 </uploads>
</settings>
