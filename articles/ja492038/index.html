3月にオンラインまたは移行するPHPコミュニティのMitap



秋には、国のさまざまな地域でのPHP に関する会議のリストがあり、Pronskiyプロンスキーは彼の素晴らしいダイジェストでそれらの最も近いものを短い行で公開します。今回はもう少し進んで、各コミュニティについてもう少し詳しく説明します。

サンクトペテルブルクBeerPHPチャットの2回目の会議：3月13日、金曜日、夜


チャットはBeerPHPインフォーマルムーブメント（厳密な議題なしに都市のバーで専門的なトピックについて話し合う）の一部として始まりましたが、約12のバーのミーティングとグループの100人以上の参加者の後、彼らは完全なレポートで最初の仮想化に成熟しました。


1月の集会では、近い席がありました、彼らは新しいサイトを見つけました。

今回は、モノリス、DDDをカットし、すべてのバックエンドスクリプトをHaskellで置き換えることに関するレポートが計画されています。

Mitapは合格しました。会議のビデオはYoutubeチャンネルで見つけることができます。

Voronezh PHPチャットの2番目のmitap：移動


現在、市はITイベントのブームを経験しており、1か月あたりmitapを発行している間、彼らはそれほど遅れていません。チャットは文字通り新年の前に登場し、すでに2月中旬にその活動家が最初の会議を組織しました-サーシャマカロフはYiiの新しいアイテムについて秘密の報告をしました。


会議には写真はありませんが、最初の地域会議の30人は素晴らしいです！

2番目の会議では、ReactPHP + Ratchet + RabbitMQの束にバックエンドを使用してゲームプロジェクトを作成した経験に関するレポートが計画されています。

イベントのステータスを監視する場所：会議は3月16日の夕方に計画されましたが、月曜日の朝に男たちは無期限に延期を報告しました。

会議のビデオを見る場所：ミタパスはまだ放送されていませんが、彼らはビデオを録画する計画を持っています。コミュニティチャットに追加します。資料が表示された場合は、まずそこに追加します。

エカテリンブルクPHPコミュニティの第3回会議：延期


PUM（PHP Ural Meetup）は昨年の夏にリリースされ、2019年にはオープンマイクを使用してレポートを交互に表示する形式で実験を行いました。この会議は古典的なmitapになります。ちなみに、空きスロットが1つある間、最大3つの30分レポートが計画されます。彼らは、3人目のスピーカーが準備するのを手伝う準備ができています-彼らは、さまざまなエカテリンブルクとモスクワの会議のスピーカーと一緒に働いた経験があります。


過去には、チャットは混雑し、みんなが新しいサイトを見つけました。

プログラムは、Web開発者の目を通してコードレビューとセキュリティに関するレポートを発表しました。

イベントのステータスを監視する場所：主催者が無期限の延期を報告した場合、詳細は登録ページとコミュニティの電報チャットに表示されます。

会議のビデオを視聴する場所：会議のビデオはYoutubeチャネルに投稿されます（最初の会議のビデオもそこで利用できます）。

2番目の主要なKazan PHP-mitap：オンラインで開催されます


タタールスタンのPHPチャットは、2019年12月の最初のカザン会議で生まれました。イベントが週末に行われるという事実により、Ulyanovsk、Yoshkar-Ola、およびその他の近隣の人たちは、エリアがそれらに到達することができなかったため、チャットは「マルチリージョン」であることが判明しました。


最初のmitapは90人以上の人々と6つのレポートを集めたため、大規模と呼ばれました。

繰り返しになりますが、土曜日の要素のため、他の都市や地域の講演者が喜んで会議に来ています。前回はヴォロネジだけの代表者がいましたが、今回はエカテリンブルクからスピーカーが予定されています（そしてそれだけではありません）。プログラムは豊富であることが判明しました。CQRSに関するあまり知られていないユーティリティや紛争のレビューから、レガシーとの戦いの永遠の苦痛とリモートサイトでの作業の側面まで、幅広いトピックをカバーする6つのレポートです。

会議はオンラインで開催されます：同時に、スピーカー、ピザ、その他のインタラクティブアクティビティとのライブコミュニケーションを企画しています。登録ページにサインアップして（オンラインの場合は十分なスペースがありますが、リマインダーを受け取ることができます）、コミュニティの電報チャットに参加します。

会議のビデオをどこで視聴するか：このYoutubeチャンネルで放送が計画されています（現時点では、最初の会議の録画を視聴できます）。

私の町のどこでPHPコミュニティチャットを見つけることができますか？


多くの都市では、独自のチャット、非公式会議、会議があります。上記に加えて、ウリヤノフスク、ヨシュカルオラ、モスクワ、ニジニ、ロストフオンドン、ヤロスラヴリ、ノボシビルスク、そしてそれらだけではありません。彼らのリストはここにあります。
