F＃7：レコード

したがって、私たちはより多くのF＃タイプへの旅を続けます。今回はレコードのタイプを見ていきます。

投稿を作成する方法


新しいF＃エントリを作成すると、C＃の匿名オブジェクトを思い出させる場合があります。これはあなたがそれらを作成する方法です。C＃の匿名オブジェクトと非常に似ていると思うので、C＃を使用した場合、F＃エントリは複雑にならないはずです。

すべては、録音用の新しいタイプを作成することから始まります。タイプ定義には、以下に示すように、プロパティの名前とプロパティのタイプが一覧表示されます。

レコードタイプの定義がある場合は、Letを使用して新しいインスタンスを値に関連付けることができます。ここでも、この例を以下に示します。Letバインディングを使用して新しいレコードをバインドし、レコードインスタンスのプロパティをコンソール出力に出力します。

type Person = { Age : int; Sex: string; Name:string; }  
....
....
....
....
let sam = { Age = 12; Sex="Male"; Name ="Sam" }
printfn "let sam = { Age = 12; Sex=\"Male\"; Name=\"Sam\" }"
printfn "Person with Age is %i and Sex is %s and Name is %s" sam.Age sam.Sex sam.Name



レコードを変更/複製するにはどうすればよいですか


エントリを複製して変更できます。これは通常、次のように行われます。

let sam = { Age = 12; Sex="Male"; Name ="Sam" }
let tom = { sam with Name="Tom" }
printfn "let tom = { sam with Name=\"Tom\" }"
printfn "Person with Age is %i and Sex is %s and Name is %s" tom.Age tom.Sex tom.Name



Tom Personの新しいインスタンスを作成するときに、withキーワードをどのように使用したかに注目してください。この形式のレコード式は、「コピーおよび更新レコード式」と呼ばれます。使用できる別のオプション（これについても、次の投稿で詳しく説明します）は、投稿タイプで可変プロパティを使用することです。エントリはデフォルトで不変です。ただし、変更可能なフィールドを明示的に指定することもできます。

以下に例を示します。MutableNameという可変プロパティを持つ新しい型を作成したことに注目してください。可変フィールドを定義することで、レコードのMutableNameプロパティの値を更新できます。これは、「<-」演算子を使用して実行できます。これにより、新しい値を割り当てることができます。

type MutableNamePerson = { Age : int; Sex: string; mutable MutableName:string; }   
....
....
....
....
//create
let sam = { Age = 12; Sex="Male"; MutableName ="Sam" }
printfn "let sam = { Age = 12; Sex=\"Male\"; Name=\"Sam\" }"
printfn "Person with Age is %i and Sex is %s and Name is %s" sam.Age sam.Sex sam.MutableName
//update
sam.MutableName <- "Name changed"
printfn "sam.MutableName <- \"Name changed\""
printfn "Person with Age is %i and Sex is %s and Name is %s" sam.Age sam.Sex sam.MutableName



レコードの比較


レコードタイプが等しいのは、すべてのプロパティが同じと見なされる場合のみです。次に例を示します。

type Person = { Age : int; Sex: string; Name:string; }   
....
....
....
....
let someFunction p1 p2 =
    printfn "p1=%A, and p2=%A, are they equal %b" p1 p2 (p1=p2)
 
 
let sam =  { Age = 12; Sex = "Male"; Name = "Sam" }
let john = { Age = 12; Sex = "Male"; Name = "John" }
let april = { Age = 35; Sex = "Female"; Name = "April" }
let sam2 = { Age = 12; Sex = "Male"; Name = "Sam" }
do someFunction sam john

メソッドとプロパティ


エントリにメンバーを追加することもできます。ここでは、「詳細」プロパティを追加する例（私たちは、toString（）メソッドをオーバーライドすることで達成できるように、それ以降のOOPの方法について、より詳細に限り）私たちは一つの特性を使用してレコードに関する完全な情報を取得できるようにすることです

注場合は、そのスクリーンショットに示すように、レコードタイプ定義に要素を追加しようとすると、奇妙なエラーが発生します。



これは、レコードプロパティ定義を新しい行に配置し、要素も新しい行で始まり、このインデント（スペース）を確認することで簡単に解決できます）、これはすべてF＃で重要であるため：
