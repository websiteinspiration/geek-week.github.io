カントリールーター用の2つのSIM-それはたくさんですか、それとも少しですか？

昨年、カントリーハウス向けに3G / 4Gルーターのテストを実施しましたが、このトピックは多くの人にとって興味深いものでした。テストの後、外出先で安定したインターネットアクセスを確保するために車にルーターを1つ組み込みましたが、それでも最も重要なことは、自宅のネットワークへのアクセスの安定性でした。昨年末、私はロシアのデュアルSIM 3G / 4Gルーターの開発が市場に出ていることに気づきました。そして、これらはUSBモデムに基づく工芸品ではなく、Linuxマシンに基づく本格的なデバイスです。私はそれを手に入れ、1か月間テストし、今日は私の観察結果を共有することにしました。伝統的に、彼は読みたくない人のためにビデオを作りました。


なぜ2SIMなのか？


この質問は、リンゴの木の所有者がAndroidの支持者にSIMカードのペアを持っていることを尋ねた質問に似ています。答えは明白で単純です。2つのSIMカードを使用すると、2つの異なるオペレーターを使用して、最適な料金/速度を選択できます。さらに、これにより信頼性が大幅に向上します。1人のオペレーターのBSが落ちるか、残高が不足すると、ルーターはバックアップオペレーターに切り替わり、ネットワークへのアクセスは中断されません。最後に、2つのSIMカードの存在により、マストで外部アンテナが使用されている場合、地上での通信事業者の選択が容易になり、加速します。ルーターのアンテナを毎回取り外したり、ルーターを取り外してカードを交換したりする代わりに、プログラムでオペレーターのカードを変更し、アンテナの調整を続けることができます。

ルータ自体は、すべての要素が搭載されたボードであり、外部アンテナのピグテールと独自のWi-Fiアンテナが接続されています。インターフェースには、PoE対応の100 Mbit / s用のイーサネットコネクタと、外部電源アダプタを接続するためのコネクタがあります。ボードは、9〜36 Vの範囲の電圧で電力を供給できます。つまり、このルーターは、自動車、少なくとも24 V電源が供給されているトラック、さらには12〜24 Vの冗長電源を備えた自家製のシステムでも簡単に使用できます。最も便利なのは、そのようなルーターが、外部ネットワークなしで数日間動作を保証するのに十分な容量を持つ従来の自動車またはゲルバッテリーで動作できることです。つまり、外部ネットワークがオフになっていて、インバーターなしの無停電電源装置がある場合でも、ルーターをバッテリーに直接接続でき、インターネットが失われることはありません。



次に、ルーターの技術的特徴について触れましょう。写真は、Quectel EC25-Eチップがオペレーターとの通信を担当していることを明確に示しています。メーカーのウェブサイトにあるモジュールの技術的特徴から、モジュールが2G / 3G / 4G（LTE）テクノロジーをサポートし、LTEがCat.4カテゴリでサポートされていることがわかります。これにより、最大150 Mbpsおよび最大50 Mbpsのアップロードのダウンロード速度を実現できます。 。同時に、速度が100 Mbpsのイーサネットインターフェイスがボードにはんだ付けされています。つまり、いずれの場合も、有線接続のパフォーマンスについて詳しく説明しますが、これについては後で説明します。このモジュールは非常に感度が高く、3Gネットワ​​ークで-110 dBmまでの弱い信号でも機能することが重要です。簡単に言うと、このルーターを使用すると、BSから遠く離れた場所にある非常に弱い信号でもネットワークに入ることができます。今年から、メーカーはさらに優れた感度インジケーターを備えたQuectel EC25-EUチップに切り替えました。弱い信号を改善するには、アンテナテストを使用します。



このデバイスは、屋内と屋外の両方で気密ボックスに設置するように設計されています。激しい霜でもルータを簡単に起動できるように、暖房システムが装備されています。4つの抵抗器（赤で囲まれている）がボードにはんだ付けされており、起動する前にボードとその周囲の要素を暖めます。メーカーは、-40度でもルーターの動作を保証しています。ボードの温度が-18度未満の場合、ルータの起動時に暖房自体がオンになり、-15に達すると暖房がオフになります。さらに、チップの動作による自己発熱により、ボードは正常に機能します。

ルーターには、良好な信号レベルと広いカバレッジを備えた2.4 GHz Wi-Fiモジュールが装備されていますが、高速性は期待できません。最大150 Mbpsのチャネル速度でラップトップを接続した場合でも、ダウンロード用に55 Mbpsを超えるデータを受信しませんでした。ただし、この速度は、ネットサーフィンやビデオの遅延なく快適に視聴するには十分です。全速力を使いたい場合は、ケーブル接続の方が確実に優れています。
