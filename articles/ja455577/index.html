SDL 2レッスン：レッスン3-イベント

みなさん、こんにちは！これはSDL 2の3番目のレッスンです。情報はこのソースから取得されます。Lasy Fooのイベント処理チュートリアルは好きではなかったので、自分で作成しますが、資料はこことここにあります。

そして始めましょう、レッスンへようこそ

イベント


今日は、ジョンの助けを借りてイベントに応答するようにプログラムを教えます。

それでは始めましょう。SDL 2を接続し、3つのグローバル変数と2つの定数を作成します。

#include <SDL2/SDL.h>

const int SCREEN_WIDTH = 640;
const int SCREEN_HEIGHT = 480;

SDL_Window *win = NULL;
SDL_Surface *scr = NULL;
SDL_Surface *john = NULL;

次に、前回のレッスンですでにわかっている3つの関数を記述します。

初期化
int init() {
    if (SDL_Init(SDL_INIT_VIDEO) != 0) {
        std::cout << "Can't init: " << SDL_GetError() << std::endl;
        system("pause");
        return 1;
    }

    win = SDL_CreateWindow("", SDL_WINDOWPOS_UNDEFINED, SDL_WINDOWPOS_UNDEFINED, SCREEN_WIDTH, SCREEN_HEIGHT, SDL_WINDOW_SHOWN);
    if (win == NULL) {
        std::cout << "Can't create window: " << SDL_GetError() << std::endl;
        system("pause");
        return 1;
    }

    scr = SDL_GetWindowSurface(win);

    return 0;
}

負荷
int load() {
    john = SDL_LoadBMP("john.bmp");

    if (john == NULL) {
        std::cout << "Can't load image: " << SDL_GetError() << std::endl;
        system("pause");
        return 1;
    }

    return 0;
}

終了する
int quit() {
    SDL_FreeSurface(john);
    
    SDL_DestroyWindow(win);

    SDL_Quit();

    return 0;
}


注意（！）、前のレッスンのコードをコピーする場合は、笑顔をjohnに変更することを忘れないでください。

言うべきことはすべて：初期化とイメージの読み込みの段階でエラーを表示することにしました。

続けましょう。mainを開き、Init関数とLoad関数を呼び出します。

int main (int argc, char ** args) {
    if (init() == 1) {
        return 1;
        
    }

    if (load() == 1) {
        return 1;
    }
}

次に、アプリケーションがどのように機能するかを見てみましょう。実行するすべてのプログラム（Google Chrome、Paint、Minecraft）には、何回も実行されるメインループがあります。これはダイナミズムに必要です。以前は、2秒間笑顔を見せたプログラムを作成しましたが、プログラムがユーザーの操作に応答する方法（または少なくとも画像を無限に表示する方法）を学習するには、Whileループが必要です。

それは常に繰り返され、その中ですでに、たとえばイベントを処理するなど、私たちが望むことを行うことができます。しかし、グランサイクルを開始する前に、いくつかの変数を作成しましょう。

    bool run = true;
    SDL_Event e;
    SDL_Rect r;

    int x = 0;
    int y = 0;

    r.x = x;
    r.y = y;

今からすべてを分析します。まず、boolean型の実行変数を作成します。この変数は、ループの実行に責任がある：プログラムの実行中に、それが本当の私たちが実行を停止したい場合は、その後にこの変数を設定し、偽。次に、クラスSDL_Eventのeオブジェクトを作成します。このオブジェクトのプロパティには、受信したイベントのデータが含まれ、さらに詳細になります。その後、SDL_Rectクラスのオブジェクトrを作成します。これは、必要な座標を持つオブジェクトを描画するために必要な長方形です。次に、int型の変数xとyを作成します。それらの値をすぐにゼロに割り当てます。これらはキャラクターの座標になります。次の項目では、変数rxとry にそれぞれxとyの値を割り当てます。rxとryは、長方形rの座標です。

メインサイクルの開始の時間です。

    while (run) {
        while(SDL_PollEvent(&e) != 0) {
            if (e.type == SDL_QUIT) {
                run = false;
            }

            if (e.type == SDL_KEYDOWN) {
                if (e.key.keysym.sym == SDLK_UP) {
                    y -= 1;
                }
                if (e.key.keysym.sym == SDLK_DOWN) {
                    y += 1;
                }
                if (e.key.keysym.sym == SDLK_RIGHT) {
                    x += 1;
                }
                if (e.key.keysym.sym == SDLK_LEFT) {
                    x -= 1;
                }
            }
        }
        r.x = x;
        r.y = y;

        SDL_FillRect(scr, NULL, SDL_MapRGB(scr -> format, 255, 255, 255));

        SDL_BlitSurface(john, NULL, scr, &r);

        SDL_UpdateWindowSurface(win);
    }

このコードを解析してみましょう。 2つのサイクルがあります。最初のメインはメインのもので、前述のように変数runを実行することで機能します。 2番目のループは、イベントの処理を担当します。SDL_PollEvent（）

関数は、メインループの1回の実行で受信したイベントのキューを開き、最後のイベントを削除し、この関数に渡したオブジェクトの変数にこのイベントに関するデータを書き込みます。これはeです。この関数は、キューからすべてのイベントが削除された場合、0、NULL、またはfalseを返すため、while条件（SDL_PollEvent（＆e）！= NULL）を書き込みます。

さらに調べます。最初のチェックは、ウィンドウフレームの十字をクリックすることです。これをクリックすると、eオブジェクトのtype変数に値SDL_QUITが割り当てられます。これが発生した場合は、実行変数にfalseを割り当ててループを終了します。 次に、キーボードのキーストロークテストを作成しました。ボタンが押された場合、e.type変数はSDL_KEYDOWNと等しくなります。次に、キー値を確認します。変数e.key.keysym.symにはボタンの値が格納されます。この値を望ましい値と比較し、特定のアクションを実行します。ジョンのコーディネートを変更します。上矢印を押すと、y

減少すると1、減少すると増加します。

イベントを処理した後、rxとryを再割り当てして、Manの座標を更新します。次に、すでにおなじみの関数SDL_FilRectを記述します。これは非常に大きな役割を果たします。記述しなかった場合、ジョンは移動時に醜い痕跡を残していたことになります。そのため、フレームごとに白い背景を描画し、その上にスプライトを描画します。その後、SDL_BlitSurface関数を使用してヒーローを描画します。ここでは、長方形rまたは変数xとyを使用して、左上だけでなく必要な座標も描画します。次に、ウィンドウを更新します。

メインループの後、return quit（）;を書き込みます。プログラムを完了します。

以上で、このレッスンは終了です。これがそのようなコードです：

コード
#include <SDL2/SDL.h>
#include <iostream>

const int SCREEN_WIDTH = 640;
const int SCREEN_HEIGHT = 480;

SDL_Window *win = NULL;
SDL_Surface *scr = NULL;
SDL_Surface *john = NULL;

int init() {
    if (SDL_Init(SDL_INIT_VIDEO) != 0) {
        std::cout << "Can't init: " << SDL_GetError() << std::endl;
        system("pause");
        return 1;
    }

    win = SDL_CreateWindow("", SDL_WINDOWPOS_UNDEFINED, SDL_WINDOWPOS_UNDEFINED, SCREEN_WIDTH, SCREEN_HEIGHT, SDL_WINDOW_SHOWN);
    if (win == NULL) {
        std::cout << "Can't create window: " << SDL_GetError() << std::endl;
        system("pause");
        return 1;
    }

    scr = SDL_GetWindowSurface(win);

    return 0;
}

int load() {
    john = SDL_LoadBMP("john.bmp");

    if (john == NULL) {
        std::cout << "Can't load image: " << SDL_GetError() << std::endl;
        system("pause");
        return 1;
    }

    return 0;
}

int quit() {
    SDL_FreeSurface(john);
    
    SDL_DestroyWindow(win);

    SDL_Quit();

    return 0;
}
int main (int argc, char ** args) {
    if (init() == 1) {
        return 1;
        
    }

    if (load() == 1) {
        return 1;
    }

    bool run = true;
    SDL_Event e;
    SDL_Rect r;

    int x = 0;
    int y = 0;

    r.x = x;
    r.y = y;

    while (run) {
        while(SDL_PollEvent(&e) != NULL) {
            if (e.type == SDL_QUIT) {
                run = false;
            }

            if (e.type == SDL_KEYDOWN) {
                if (e.key.keysym.sym == SDLK_UP) {
                    y -= 1;
                }
                if (e.key.keysym.sym == SDLK_DOWN) {
                    y += 1;
                }
                if (e.key.keysym.sym == SDLK_RIGHT) {
                    x += 1;
                }
                if (e.key.keysym.sym == SDLK_LEFT) {
                    x -= 1;
                }
            }
        }
        r.x = x;
        r.y = y;

        SDL_FillRect(scr, NULL, SDL_MapRGB(scr -> format, 255, 255, 255));

        SDL_BlitSurface(john, NULL, scr, &r);

        SDL_UpdateWindowSurface(win);
    }

    return quit();
}
