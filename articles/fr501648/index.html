10 extensions pour VS Code, sans lesquelles je ne peux pas programmer

VS Code est mon éditeur de code préféré. C'est le plus populaire des éditeurs existants, dont les possibilités peuvent être étendues presque indéfiniment. Et, étonnamment, il a été développé par Microsoft. Je crois qu'aucun des autres éditeurs et aucun des autres IDE ne peuvent donner au développeur au moins la moitié de ce que VS Code est capable de lui donner. L'une des forces de VS Code est son système d'extension. Il vous permet de créer des extensions littéralement pour toutes les occasions. Je veux vous parler de mes 10 meilleures extensions pour VS Code.



1. Embellissez



Pour installer l'extension, utilisez le raccourci clavier dans VS Code CTRL+P, entrez la commande ci-dessous dans le champ qui s'ouvre et cliquez sur Enter.

ext install HookyQR.beautify

L' extension Beautify vous permet de formater le code html, js, css, JSON et sass. Les options de formatage peuvent être contrôlées. Cette extension est un complément au système js-beautify standard et permet au développeur de formater le code exactement comme il a besoin.

2. De meilleurs commentaires



ext install aaron-bond.better-comments

L' extension Better Comments vous permet d'étendre les possibilités d'utilisation des commentaires. En particulier, il vous permet de diviser les commentaires en catégories. Il peut s'agir de notifications, de demandes, de listes de tâches, de notes.


Utiliser de meilleurs commentaires

3. Signets



ext install alefragnani.Bookmarks

L'extension Signets est une véritable bouée de sauvetage pour les programmeurs. Surtout pour ceux qui sont perdus parmi des milliers de lignes de code. Cette extension vous permet d'attacher des signets à des lignes de code. Si la ligne de code a un signet, vous pouvez facilement accéder à cette ligne en utilisant la liste des signets.


Signets dans le code

4. Bracket Pair Colorizer 2



ext install CoenraadS.bracket-pair-colorizer-2

L'extension Bracket Pair Colorizer 2 vous permet de coloriser les supports, ce qui facilite la recherche de paires de supports d'ouverture et de fermeture. Ceci est très utile dans les cas où vous devez travailler avec des constructions logicielles imbriquées.


Les paires de crochets sont mises en évidence.

5. Formater dans les menus contextuels



ext install lacroixdavid1.vscode-format-context-menu

L'extension Format dans les menus contextuels vous permet de formater des fichiers en les mettant en surbrillance dans la barre latérale et en appelant la commande du menu contextuel. Cela est particulièrement pratique dans les cas où de nombreux fichiers doivent être formatés, mais en même temps dans l'environnement existant, il n'y a pas de support pour les moyens de formatage et de peluchage du code.


Utilisation du format dans les menus contextuels

6. Git Graph



ext install mhutchie.git-graph

L'extension Git Graph vous permet de visualiser la structure du référentiel et, à l'aide de cette structure, d'effectuer diverses opérations. Cette extension est hautement personnalisable et possède de nombreuses fonctionnalités. Probablement, afin de le décrire correctement, vous aurez besoin d'un article entier séparé.


Travailler avec Git Graph

7. GitLens



ext install eamodio.gitlens

L' extension GitLens permet d' identifier la paternité du code à l'aide d'annotations et d'objectifs git blame. Cette extension, entre autres, vous permet de visualiser très facilement le contenu des référentiels et d'apprendre des informations utiles sur le code.


Travailler avec GitLens

8. Rainbow en retrait



ext install oderwat.indent-rainbow

L'extension Indent Rainbow vous permet de colorer l'indentation, en mettant en évidence l'indentation de différents niveaux avec différentes couleurs. Cela est particulièrement utile lorsque vous travaillez avec des constructions profondément imbriquées dans des langages comme Python.

9. Path Intellisense



ext install christian-kohler.path-intellisense

L'extension Path Intellisense offre à l' éditeur la possibilité de compléter automatiquement les chemins de fichiers. Bien que VS Code prenne en charge la complétion de chemin sans extensions, cette fonctionnalité est limitée uniquement aux fichiers HTML, CSS et JavaScript. Grâce à cette extension, l'achèvement du chemin peut être utilisé lorsque vous travaillez avec n'importe quel fichier et n'importe quelle langue.


Travailler avec Path Intellisense

10. Total des lignes



ext install praveencrony.total-lines

L'extension Total Lines est un petit outil pratique qui affiche des informations sur le nombre de lignes dans un fichier ouvert dans la barre d'état. Ceci est un bel ajout à ma collection d'extensions.
