機能インターフェイス... VBA

「アマチュアをばかに非難することをアマチュアで見つめることに嫌がらない人は、JavaとVisual Basicが出生時に双子であり、C ++が遠い親類でさえないことを証明する方法を観察させてください。」

ブルース・マッキーニー「Die Hard Basic Basic」

前書き


関数型プログラミングアプローチへの継続的な関心は、現在、従来のプログラミング言語が関数型の手段を積極的に獲得しているという事実につながります。純粋な関数型言語はまだそれほど人気が​​ありませんが、C ++、Java、JavaScript、Pythonなどの言語で機能がしっかりと確立されています。VBA言語は、Microsoft Officeユーザーのかなり多くの読者の間で長年よく知られていますが、この言語には実質的に機能的なツールは含まれていません。

このギャップを埋めようとしましょう。VBAによって実装される機能インターフェイスの完全な（おそらく完璧ではない）実装を提案します。実装は、その後の改良と改善の基礎として役立ちます。

関数の引数の問題


このパスで最初に遭遇する問題は、関数の引数を関数またはメソッドに渡す問題です。 VBA言語には適切なツールが含まれていません（AddressOf演算子はWindows API関数にアドレスを転送するためだけに使用され、完全に安全に使用できるわけではありません）。ポインタによる関数の呼び出しのよく知られた方法についても同じことが言えます（実際のサンクトペテルブルクのG. Magdanurov Visual Basic：「BHV Petersburg」、2008年）。リスクを取らないようにしましょう。実装には標準言語機能と標準ライブラリのみを使用します。

残念ながら、ここではPLOはあまり役に立ちません。機能オブジェクトをプロシージャまたは関数に転送するために、VBAは標準的な機会を提供します-必要な機能をオブジェクトシェルでラップします（オブジェクトを作成します。そのメソッドの1つは必要な機能になります）。オブジェクトはパラメーターとして渡すことができます。このアプローチは機能しますが、非常に重いです-必要な機能ごとに、独自のクラスとこのクラスのオブジェクトを作成する必要があります。

より簡単で、機能ごとに個別のクラスを作成する必要がない別の方法もあります。

引数を1つインクリメントする特定のprocプロシージャに無名関数を渡したいとします。この関数は次のように書くことができます：

x -> x+1

匿名関数を割り当てるというこの表記法は、ほぼ「事実上の標準」になりました。このような関数をパラメーターに渡す唯一の方法は、文字列表現を使用することです。

r=proc(a,b,”x->x+1”)

ここで、aとbは通常のパラメーターであり、3番目のパラメーターは名前のない関数です。これは非常に明確であり、一般的なプログラミング言語のエントリとほとんど変わりません。

この方法で定義された無名関数を使用するには、まずそれをVBA関数の標準形式にする必要があります。これにより、次のユーティリティ手順が実行されます。


Private Function prepCode(Code As String) As String
         k% = InStr(Code, "->")
         parms$ = Trim$(Left$(Code, k% - 1))
         body$ = Mid$(Code, k% + 2)
         If Left$(parms$, 1) <> "(" Then parms$ = "(" + parms$ + ")"
         If InStr(body$, "self") = 0 Then body$ = ";self=" & body$ & ";"
         body$ = Replace(body$, ";", vbCrLf)
         prepCode = "function self" & parms & vbCrLf & body & _ 
                             vbCrLf & "end function"
End Function

この関数は、パラメーターのリストと計算本体を選択してから、selfという関数を作成します。この場合、self関数は次の形式になります。


function self(x)
     self=x+1
End function

明らかに、VBA構文に従って、この関数は無名関数が実行する必要があることを正確に実行します。引数の値を1増やします。真、この関数はまだVBA関数ではなく、指定されたコードを含む行のみです。文字列を関数にするには、標準のMicrosoftライブラリ「Msscript.ocx」を使用できます。このCOMライブラリを使用すると、文字列形式で指定された任意のVBAコードを実行できます。これを行うには、以下を実行します。

-ScriptControlオブジェクトを作成します。-
言語インストールメソッド（VBScript）を呼び出します。
-関数の読み込みメソッドを呼び出します。
-evalメソッドを呼び出して呼び出します。

それはすべてこのように見えます：


Set locEv=new ScriptControl
locEv.Language = "VBScript"
locEv.AddCode prepCode(“x->x+1”)
r=locEv.eval(“self(5)”)

このコードを実行すると、変数rの値は6になります。

ここでは、次の3つの点に注意してください。

  • 無名関数の本体には複数の行を含めることができます。この場合の個々のステートメントはセミコロンで終わります。最終コードから、文字「;」除外されます。複数行の本文を使用すると、匿名関数に高度な機能を実装できます。
  • 「実際には」匿名関数が「自己」という名前を持っているという事実は、予期しないボーナスを与えます-匿名関数は再帰的です。
  • ScriptControlオブジェクトは2つの言語（VBScriptとJscript）をサポートしているため、匿名関数は（理論的には）Jscriptで記述できます（希望するユーザーはそれを試すことができます）。

次に、オブジェクト実装モデルについて説明する。

例


フィボナッチ連続番号の印刷：

Sub Test_1() 
Dim fibGen As aIter
    Set fibGen = New Generator
    fibGen.Init Array(1, 0), "(c,p)->c+p"
    For i% = 1 To 50
        Debug.Print fibGen.getNext()
    Next i%
End Sub

ここでは、初期値0と1、およびフィボナッチ数列に対応する生成関数を使用してジェネレーターが作成されます。次に、最初の50個の数字がループで出力されます。
マップとフィルター：


Sub Test_2() 
Dim co As aIter
Dim Z As aIter
Dim w As aIter
    Set co = New Container
    co.Init frange(1, 100)
    Set Z = co.map("x -> 1.0/x"). _
                 take(20).filter(" x -> (x>0.3) or (x<=0.1)")
    iii% = 1
    Do While Z.hasNext()
       Debug.Print iii%; " "; Z.getNext()
       iii% = iii% + 1
    Loop
End Sub

コンテナーが作成され、1から100の範囲の数値シーケンスによって初期化されます。次に、mapを使用した数値が逆に置き換えられます。これらのうち、21番目が取られます。次に、この母集団がフィルタリングされ、0.3より大きいまたは0.1より小さい数値がそこから選択されます。結果はコンテナに返され、その構成が印刷されます。
畳み込みの使用：


Sub Test_4() 
Dim co As aIter
    Set co = New Container
    co.Init frange(1, 100)
    v = co.reduce(0, "(acc,x)->acc+x")
    Debug.Print v
    v = co.reduce(1, "(acc,x)->acc*x")
    Debug.Print v
End Sub

ここでは、畳み込みを使用して、1から100までの数の合計と積が考慮されます。


Sub Test_5() 
Dim co1 As aIter
Dim co2 As aIter
Dim co3 As aIter
    Set co1 = New Generator
    co1.Init Array(123456789), "x -> INT(x/10)"
    Set co2 = co1.takeWhile(100, "x -> x > 0")
    Set co3 = co2.map("x -> x mod 10")
    Debug.Print co3.maximun
    Debug.Print co3.minimum
    Debug.Print co3.summa
    Debug.Print co3.production
End Sub

この例では、元の数を次数10で順次除算してジェネレーターco1を作成します。次に、ゼロが表示されるまで商を選択します。その後、結果の商のリストは、10で割った余りをとる関数によって表示されます。結果は、数値の桁のリストです。リストが要約され、彼は最大値、最小値、積を計算しました。
