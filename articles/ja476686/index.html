Raspberry?


Raspberry Piは、あらゆるタスクに柔軟に設定できる、小型で安価なユニバーサルマイクロコンピューターです。 2014年以降、独立したコンピュートモジュールコンピューティングモジュールとしてリリースされました。つまり、さまざまなインターフェイスとコネクタを備えた使い慣れたボードから、最も重要なもの、つまりプロセッサ、ROM、RAMだけが残っています。この設計により、このデバイスをあらゆる考えられるタスクと考えられないタスクに使用できます。すべて、モジュールの周りにハーネスを作成する想像力の可能性に依存しています。また、デバイスの最初のバージョンのリリース以降、モジュールの3つのバージョンがすでにリリースされており、Raspberry Pi 4のリリース後、近い将来に4番目のバージョンが期待できることにも注意してください。これらすべては、開発者が積極的に製品を開発し、その能力と速度を向上させていることを示唆しています。そして彼らのデバイスは人々の間で人気があること。この人気は根拠がないわけではありません。これまでずっと、あらゆる状況でさまざまなレベルのタスクを解決できる信頼できるデバイスとしての地位を確立しています宇宙で。Raspberry Piでのプログラミングも条件付きで簡単で、世界中に多数のインテグレーターがいます。

オープンなLinuxオペレーティングシステムでは、必要なタスクに応じて、デバイスにすべてのソフトウェアをインストールできます。たとえば、スマートホームの分野のソリューションでは、OpenHab、Home Assistant、iRidiumMobile、NodeRedなどとのドッキングが可能です（スマートホーム用のデバイスの使用例は、こちらにあります）。業界では、CODESYS、Rapid SCADA、OpenSCADAなどのSCADAシステムをインストールして、MQTT、http、REST API、またはCoAPプロトコルを使用してデータを上位レベルに転送するゲートウェイとしてデバイスを使用することができます。さまざまなクラウドサービスとの統合も可能です。

インターフェイスとは何ですか？



図2-ボードの上面図と底面

図最小バージョンのデバイスは、次のインターフェイスをサポートしています。

  • RS485 x 2;
  • RS232 x 1;
  • CanBus x 1;
  • 1-Wire x 1;
  • USB x 1;
  • イーサネットx 1;
  • SMA x 2;
  • SIM x 1;
  • miniPCIe x 2;
  • HDMI 4k x 1;
  • MicroUSB x 1;
  • MicroSD x 1;
  • GPIO x 1。
  • LED x 1（プログラム可能）;

上記のインターフェイスを使用すると、任意のプロジェクトでデバイスを実装できます。また、サードパーティメーカーのMini PCI-eスロットにインストールするための追加のハードウェアモジュールは、デバイスでの通信とインターネットの可用性の問題を解決します。USBアダプター（いわゆる「ホイッスル」）の存在は非常に信頼性が高く、高品質なオプションではないため、偶然ではなく通信モジュールのインストールでこのパスを選択しました。RobustelR2000-3Pなどの産業用ルーターのインストールは高価です（約12 tr） 。）。したがって、Mini PCI-eモジュール用に2つのスロットを取り付けることにしました。これは自由に使用できます。

  • 3G、LTE、GPRSモジュール（HUAWEI MU709s-2、価格：2.5 tr）;
  • 指向性アンテナを接続できるWi-Fiモジュール。
  • NB-IoTモジュール;
  • モノのインターネットネットワークを構築するためのLoraWanモジュール。

したがって、特定のタスクに適した通信インターフェースを選択する際には、一定のばらつきと柔軟性があります。
