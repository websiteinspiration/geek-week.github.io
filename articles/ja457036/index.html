定期的なパスワード変更は時代遅れの慣行であり、それを放棄する時がきた

多くのITシステムには、パスワードを定期的に変更するための必須ルールがあります。これはおそらく、最も嫌われ、最も役に立たないセキュリティ要件です。一部のユーザーは、ライフハックとして、最後に数字を変更するだけです。

この習慣は多くの不便を引き起こしました。しかし、これは安全のためです。今、このアドバイスは完全に無関係です。 2019年5月で、でもMicrosoftが最終的に、定期的にWindowsの10の個人とサーバのバージョンのセキュリティ要件の基本的なレベルからパスワードを変更する必要性を取り除く：ここにある公式ブログ声明のWindows 10 V 1903への変更をリストには、（フレーズに注意したパスワードを削除します-定期的なパスワード変更を必要とする有効期限ポリシー）Windows 10 Version 1903およびWindows Server 2019 Security Baselineのルールとシステムポリシー自体は、Microsoft Security Compliance Toolkit 1.0に含まれています。

これらの文書を上司に見せて、「時代は変わった」と言うことができます。必須のパスワード変更-古きよき時代、ほぼ公式に。セキュリティ監査でさえ、この要件を検証しなくなります（Windowsコンピューターの基本的な保護に関する公式ルールに焦点を当てている場合）。


Windows 10 v1809の基本的なセキュリティポリシーとそれに対応するパスワード有効期限ポリシーが適用されなくなった1903年の変更を含むリストのスニペット。ちなみに、新しいバージョンでは、管理者とゲストのアカウントもデフォルトでキャンセルされます

。Microsoftは、ブログでパスワードの必須変更のルールを拒否した理由をよく説明しています。その有効期間と無許可の人によって使用されます。パスワードが盗まれない場合、パスワードを変更しても意味がありません。また、パスワードが盗まれた証拠がある場合は、問題を修正するために有効期限を待たずに、すぐに対処したいことは明らかです。」

マイクロソフトはさらに、現代の状況では、この方法を使用してパスワードの盗難から身を守るのは間違っていると説明しています。デフォルト値は42日です。それは途方もなく長い時間のように思えませんか？実際、これは非常に長い時間ですが、頻繁な期限切れを強制すると固有の問題が発生するため、現在のベースラインは60日に設定されています。そして、パスワードが盗まれる必要がない場合は、これらの問題が発生しても何のメリットもありません。さらに、ユーザーがパスワードをキャンディーと交換する用意がある場合、パスワードの有効期限ポリシーは役に立ちません。」

オルタナティブ


Microsoftは、その基本的なセキュリティポリシーは、適切に管理され、セキュリティを意識した企業が使用するように設計されていると書いています。また、監査人にガイダンスを提供することも求められます。そのような組織が禁止パスワードのリスト、多要素認証、ブルートフォースパスワードによる攻撃の検出、およびシステムへの異常な試みの検出を実装している場合、パスワードの定期的な有効期限が必要ですか？また、最新のセキュリティ機能を導入していない場合、パスワードの有効期限は役立ちますか？

マイクロソフトのロジックは驚くほど説得力があります。2つのオプションがあります。

  1. 同社は近代的なセキュリティ対策を導入しています。
  2. 同社は現代のセキュリティ対策を実施していません。

最初のケースでは、定期的なパスワード変更は追加の利点を提供しません。

2番目のケースでは、定期的なパスワード変更は役に立ちません。

したがって、パスワードの有効期限の代わりに、最初に多要素認証を使用する必要があります。上記の追加のセキュリティ対策は、禁止されたパスワードリスト、ブルートフォースの検出、その他の異常なログイン試行です。

「定期的なパスワードの有効期限は古くて時代遅れのセキュリティ対策です-Microsoftを要約します。また、基本的な保護レベルに特定の値を使用する必要があるとは考えていません。それを私たちの基本レベルから削除することにより、組織は、私たちの推奨事項に矛盾することなく、意図したニーズに最も適したものを選択できます。

結論


今日、企業がユーザーに定期的にパスワードを変更することを強制している場合、外部の観察者は何を考えているでしょうか？

  1. 前提：会社は古風な防御メカニズムを使用しています。
  2. 前提：会社は現代の防御メカニズムを実装していません。
  3. 結論：これらのパスワードは取得および使用が簡単です。

定期的にパスワードを変更することで、企業は攻撃に対してより魅力的なターゲットになることがわかりました。
