セロリ+非同期

こんにちは、ハブル！Celeryでの非同期タスクの効率的な競合実行の問題をどのように解決したかを教えてください。



前書き


Celeryは、複雑な歴史と下位互換性の大きな負担を持つ大きなプロジェクトです。主要なアーキテクチャ上の決定は、Pythonにasyncioが登場するずっと前に行われました。したがって、何とかして非同期でasyncioタスクをセロリで実行できることはさらに興味深いことです。

ボアモードがオン
#839. . mode off.

バニラセロリで非同期タスクを実行する


ボックスからasyncioタスクを開始できます。

import asyncio

from .celeryapp import celeryapp


async def async_task():
    await asyncio.sleep(42)


@celeryapp.task
def regular_task():
    coro = async_task()
    asyncio.run(coro)

明らかな利点：

  • できます！
  • 単に
  • 追加の外部依存関係なし

実際に何が問題なのですか？

  • イベントループは各ワーカー内に作成されます。
  • コルーチンの実行間でのコンテキスト切り替えはありません
  • ある時点で、ワーカーはコルーチンを1つだけ実行します
  • 共有リソースはタスク間で共有されません
  • ボイラープレート

つまり、この場合のasyncioはasyncioに使用されますが、利点はありません。

トリッキーにしてみませんか？


私はパフォーマンスのために戦った：

import asyncio
import threading

from .celeryapp import celeryapp

celeryapp.loop = asyncio.get_event_loop()
celeryapp.loop_runner = threading.Thread(
    target=celeryapp.loop.run_forever,
    daemon=True,
)
celeryapp.loop_runner.start()

async def async_task():
    await asyncio.sleep(42)


@celeryapp.task
def regular_task():
    coro = async_task()
    asyncio.run_coroutine_threadsafe(
        coro=coro,
        loop=celeryapp.loop,
    )

ビンゴ！長所：

  • まだ働いている
  • 多かれ少なかれ効果的
  • リソースでさえ、ワーカー内のコルーチン間でシャッフルされます。
  • 追加の外部依存関係はまだありません

ビンゴ？問題は長くは続かなかった：

  • セロリはコルチンの実行について何も知りません
  • タスクの実行を制御できなくなります
  • 例外の制御を失う
  • ボイラープレート

製品でこのような素晴らしいエンジニアリングソリューションを使用するために、私の手はどういうわけか上昇しませんでした

問題の定式化


  • うまくいくはず
  • コルーチンは競争力がなければなりません
  • リソースは多くの実行可能ファイルの間で共有されるべきです
  • 定型文なし
  • シンプルな予測可能なAPI

つまり、私の期待：

import asyncio

from .celeryapp import celeryapp


@celeryapp.task
async def async_task():
    await asyncio.sleep(42)

概要


私は自分のアイデアをcelery-pool-asyncioライブラリに実装しました。このライブラリは現在のプロジェクトでチームによって使用されており、すでに製品版に展開されています。

簡単な機能


asyncioタスクを直接実行するほかに、celery-pool-asyncioは問題も解決します。

  • 非同期タスクシェディング
  • セロリ信号でのコルチンのサポート

celery-pool-asyncioを機能させるために、モンキーパッチを使用しました。ランタイムで使用されるパッチごとに、それを無効にすることができます。

詳細については、ドキュメントをご覧ください。

予定


良い方法では、プールをセロリに統合する必要があります。一方、セロリの開発者はセロリ5.0のプロトタイプを作成していますが、これは非同期になります。ゲームはキャンドルの価値がありますか？
