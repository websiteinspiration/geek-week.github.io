Passo a passo Bandit de overthewire.org

Bom dia, bandidos. Tendo assistido a primeira temporada do Sr. Robô, decidi que não era pior que o personagem principal e instalei o Kali Linux para mim, pronto para navegar na Internet em busca de vulnerabilidades e fotos de garotas nuas no telefone do meu irmão . Tendo lançado o nmap três vezes e descobrindo, sem muita surpresa, que a vida não é um filme, decidi adotar uma abordagem mais estruturada para minhas pesquisas e começar com o básico da segurança da informação e pesquisa de vulnerabilidades.

Meu próximo passo foi visitar r / hacking no reddit.com e ler um post fixado mencionando overthewire.orge o jogo Bandit, como um dos "hackers" adequados para iniciantes. Tenho certeza de que o leitor tem uma idéia dos jogos Capture The Flag (dado o hub em que publico), mas, apenas por precaução, explicarei. Você se conecta a uma máquina remota e, em um espaço isolado, procura uma chave para se conectar à próxima máquina usando as recomendações no site. Além disso, para mim e para todos vocês, darei a você a passagem dos 10 primeiros níveis, porque gravar apenas para não esquecer e não compartilhar com ninguém é pecado. Vai!

PS Todas as senhas estão escondidas atrás de spoilers, leia e não se preocupe.

Nível 0


Você precisa se conectar ao host em bandit.labs.overthewire.org, porta 2220 e efetuar login como bandit0 com a senha bandit0. Não antes de dizer que acabou. Usamos a sintaxe de comando padrão ssh.

:~$ ssh bandit0@bandit.labs.overthewire.org -p 2220
This is a OverTheWire game server. More information on http://www.overthewire.org/wargames

bandit0@bandit.labs.overthewire.org's password: <b>bandit0</b>
...
bandit0@bandit:~$

Nível 1


A senha para ir para o próximo nível está no arquivo leia - me no diretório inicial. Para abri-lo, precisamos de uma equipe cat.

bandit0@bandit:~$ cat readme

Senha
boJ9jbbUNNfktd78OOpsqOltutMc3MY1

No final de cada nível, nos conectamos usando ssho seguinte usuário para localhost:

Passamos o aviso pelos nossos ouvidos, conectamos ao bandit2 e passamos para o próximo nível.

Nível 4


A senha para o próximo nível está localizado em um arquivo oculto no diretório inhere, para mostrá-lo no console usamos o comando lscom a opção -a. Vamos para o diretório

bandit3@bandit:~/inhere$ ssh bandit4@localhost

Nível 5


A senha está em um único arquivo que você pode ler (literalmente) em um diretório inhere. Para obter informações sobre o arquivo, vamos usar o comando file. Primeiro, vamos ao diretório e ver o que há dentro:

bandit4@bandit:~$ cd inhere
bandit4@bandit:~/inhere$ ls
-file00  -file01  -file02  -file03  -file04  -file05  -file06  -file07  -file08  -file09

Todos os arquivos começam com -, então usamos a sintaxe do segundo nível e complementamos *para obter informações sobre todos os arquivos de uma só vez

bandit4@bandit:~/inhere$ file ./-*
./-file00: data
./-file01: data
./-file02: data
./-file03: data
./-file04: data
./-file05: data
./-file06: data
./-file07: ASCII text
./-file08: data
./-file09: data

Como podemos ver, o único arquivo que podemos ler é esse -file07. O que vamos fazer usando nossa equipe familiar cat.

bandit4@bandit:~/inhere$ cat ./-file07

Senha
koReBOKuIDDepwhWk7jZC0RTdopnAYKh

bandit4@bandit:~/inhere$ ssh bandit5@localhost
