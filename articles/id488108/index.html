SAP HANA Lab di Azure



Sebagai bagian dari acara SAP Digital Exchange, kami mengundang Anda ke lab Microsoft untuk menggunakan SAP HANA di Azure, 20 Februari 2020.

Lokakarya ini terdiri dari merancang proses migrasi dan pekerjaan langsung tentang penerapan SAP HANA di Azure. Di bawah bimbingan seorang pelatih Microsoft, Anda akan berjalan melalui langkah-langkah untuk menginstal dan mengkonfigurasi replikasi sistem.





Menurut hasil pekerjaan laboratorium Anda akan dapat:

  • Persiapkan sendiri komponen infrastruktur Azure Anda
  • Siapkan mesin virtual
  • Menyebarkan SUSE Linux Enterprise Clustering
  • Instal SAP HANA
  • Siapkan replikasi sistem
  • Uji skenario ketersediaan tinggi

Untuk mendaftar, ikuti tautan ke tab HandsOn atau cari pekerjaan laboratorium dalam pencarian di situs untuk tag SAP HANA di Azure. Sesi
