C ++での固定小数点演算

今日は、固定小数点とは何か、それが必要な理由とその使用方法について説明します。

浮動小数点計算の特殊性のためにアプリケーションのパフォーマンスが著しく低下する可能性がある場合、このような問題があります。原則として、整数演算用にCPUがシャープになり、その中のFPU（浮動小数点ユニット）コプロセッサーは1桁遅い速度で動作します。 FPUがまったくなく、数値を使用して操作をエミュレートするのに時間がかかるようなプラットフォームがあります。たとえば、FPUが存在する場合、浮動小数点数の乗算は1つのfmulコマンドだけで実行され、FPUがない場合、乗算は__mulsf3エミュレーション関数によって実行されます。 fmulコマンドと比較して、__ mulsf3関数は浮動小数点数の演算をエミュレートしますが、計算は整数形式で実行されます。これにより、fmulコマンドがこの演算を迅速に実行する一方で、マシンコードと実行時間が増加します。ハードウェアを使用します。

この問題には解決策があり、整数型の固定小数点を使用して計算を実行できます。

このタイプの原理は、Nビットによる数値の固定シフトであり、その結果、小数は整数として表すことができ、ポイントの後に2 ^ Nの精度があります。浮動小数点数を8ビットのオーダーの固定小数点数に変換する例（2 ^ 8 = 1024）。

以下は、浮動小数点数を固定小数点数に変換する例です。

Fixed(12345,6789) = 1024 * 12345,6789 = 12641975,<s>1936</s>

この数値は、小数点の後、小数点の後に2 ^ 8の精度があります。

固定小数点数から浮動小数点数への逆変換の例。

Float(12641975) = 12641975 / 1024 = 12345,678<s>7109375</s>

この場合、逆変換後の数値は12345.6787109375の形式になり、ピリオドの後は3桁正確で、最大精度は実際には2 ^ 8 = 1024です。

固定小数点型の計算はどのように行われますか？


合計および差分演算は、通常の整数演算と同等です。

Fixed(x) + Fixed(y) Fixed(x) - Fixed(y)、任意の順序で。
(1024 * x) + (1024 * y) (1024 * x) - (1024 * y)

このような数の乗算は、この形式で行われます。
(Fixed(x) * Fixed(y)) / p、これは同等で、8ビットの
((1024 * x) * (1024 * y)) / 1024

除算の桁数です。
(Fixed(x) * p) / Fixed(y)、これも8ビットのオーダーで、これは
(1024 * 1024 * x)*(1024 * y)

オーバーフロー


乗算と除算の演算を実行すると、オーバーフローが発生する可能性があり、誤った結果につながります。これは、たとえば、32ビット整数型が使用され、計算中にこの型のオーバーフローが発生し、このオーバーフローの結果、数値が最上位ビットを失う場合に発生します。オーバーフローを解消するには2つの方法があります。

  • 64ビット整数型で計算を実行します。
  • たとえば、（xi + xf）*（yi + yf）= xi * yi + xf * yf + xi * yf + yi * xfのように乗算する場合、プリフィックスiとfは部分全体とその後の部分を意味するポイント。

C ++で固定小数点を操作するためのクラス


#define DIGITS 1024 // 
#define EPS 20 //      

using namespace std;
typedef signed int __int32_t;
      
class Fixed {
      signed int x;

      Fixed(signed int a){
            x = a;
      }
      public:
            Fixed(){
                  x = 0;
            }
            static Fixed fromInt(signed int val){
                  return Fixed(val*DIGITS);
            }
            static Fixed fromFloat(float val){
                  return Fixed((signed int)(val*DIGITS));
            }
            float fixed2float(){
                  return ((float)x)/DIGITS;
            }
            Fixed sum(Fixed a,Fixed b){
                  return Fixed(a.x+b.x);
            }
            Fixed diff(Fixed a,Fixed b){
                  return Fixed(a.x-b.x);
            }
            Fixed mul(Fixed a,Fixed b){
                  signed int c=a.x*b.x;
                  if(c/b.x != a.x){
                        // Overflow!
                        signed int i1 = a.x/DIGITS;
                        signed int i2 = b.x/DIGITS;
                        signed int f1 = (a.x&(DIGITS-1));
                        signed int f2 = (b.x&(DIGITS-1));
                        return Fixed((i1*i2)*DIGITS+(f1*f2)/DIGITS+i1*f2+i2*f1);
                  }else{
                        return Fixed(c/DIGITS);
                  }
            }
            Fixed div(Fixed a,Fixed b){
                  if(a.x>(1<<21)){
                        // Overflow!
                        signed int i = a.x/DIGITS;
                        signed int f = (a.x&(DIGITS-1));
                        return Fixed(((i*DIGITS)/b.x)*DIGITS+(f*DIGITS)/b.x);
                  }else{
                        return Fixed((a.x*DIGITS)/b.x);
                  }
            }
            Fixed sqrt(Fixed k){
                  Fixed tmp(0);
                  tmp.x = k.x/2;
                  signed int min = 0;
                  signed int max = k.x;
                  Fixed quick(0);
                  do{
                        tmp.x = (min+max)/2;
                        quick = Fixed::mul(tmp,tmp);
                        if(abs(quick.x-k.x)<EPS) return Fixed(tmp);
                        if(quick.x>k.x){
                              max = tmp.x;
                        }else{
                              min = tmp.x;
                        }
                  }while(true);
            }
};
