Development of an aircraft electrical network using model-oriented design

This publication presents the transcription of the webinar "Development of an electrical network of an aircraft using model-oriented design . " The webinar was conducted by Mikhail Peselnik, an engineer at CITIT Exhibitor .)


Today we learn that you can customize models for the optimal balance between the reliability and accuracy of the simulation results and the speed of the simulation process. This is the key to leveraging the simulation and making sure that the level of detail of your model matches the task you are about to complete.




We will also find out:


  • How you can speed up simulations by using optimization algorithms and parallel computing;
  • How to distribute simulations on several computer cores, accelerating tasks such as parameter estimation and parameter selection;
  • How to speed up development by automating simulation and analysis tasks using MATLAB;
  • How to use MATLAB scripts for harmonic analysis and document the results of any types of tests using automatic report generation.
