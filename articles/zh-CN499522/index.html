Windows 10 2004（20H1）的所有创新

Windows 10 2004（20H1）的最终ISO映像已于今晚发布。他们成为内部版本号19041.208.vb_release_svc_im.200416-2050。只要您自己知道位置，就可以在MSDN上获得开发人员的图像。



与ISO映像一起，发布了更新KB4556803，安装后，其操作系统版本为19041.264。

最终映像发布后，您可以在Windows 10的5月更新中讨论对用户有哪些新功能。

更好地控制可选更新



Windows Update自动安装许多更新，但其中一些是可选的。现在出现了一个按钮，在一个位置显示所有可选更新。

这里将提供驱动程序更新，5月这样的主要更新（本应于5月28日开始通过Windows Update传播），每月更新和与安全无关的更新。

要查找可选更新列表，请转至设置→更新和安全性→Windows更新→查看可选更新。

从云端下载以重新安装Windows 10



Windows 10获得了一项新功能“从云下载”，以使计算机恢复到原始状态。现在，如果转到设置>更新和安全>恢复，而不是创建可启动媒体来重新安装Windows，则可以下载并安装Windows 10的最新版本。

Windows更新的带宽限制



现在，参数将为下载Windows更新提供更多带宽控制。在当前版本的Windows中，您可以将最大限制设置为可用带宽的百分比。Windows 10版本2004将使您能够设置绝对带宽限制，以更精确地控制下载的更新量。以前，此选项仅通过组策略可用。
您可以转到设置→更新和安全性→交付优化→高级选项，找到新功能。

具有Linux内核的WSL 2



新的Windows Linux子系统（WSL）可能是Windows 10 20H1中最重要的更改。这是WSL版本2，它比第一个版本强大得多。 WSL 2使用真正的Linux内核为Windows 10提供完整的Linux环境

。Microsoft已经为WSL 2准备了自己的Linux内核，它将通过Windows Update进行更新。用户将能够组装自己的Linux内核并在WSL 2

上使用它。在WSL 2上，他们承诺“将大大提高I / O性能”和“完全的系统调用兼容性”。这种兼容性意味着需要支持诸如Docker之类的技术，这与WSL 1不同，在WSL 1中该功能不可用。

此外，Microsoft添加了对ARM64设备的支持。换句话说，WSL现在可以在具有ARM体系结构的PC上运行。

加急搜索



微软表示，现在，通过检测峰值负载，操作系统将能够优化索引编制。

默认情况下，在Windows中搜索也将更快地进行，因为默认情况下，索引将从搜索中排除.git，.hg，.svn，.Nuget等“开发人员文件夹”。这将影响代码的编译和同步速度。

任务管理器中的磁盘类型



现在，Windows 10中的任务管理器将显示磁盘类型：SSD或HDD。您可以在“性能”选项卡中找到此信息。

任务管理器中的GPU温度



如果您的视频卡具有最低版本的WDDM 2.4驱动程序，则还可以在“性能”部分中找到GPU数据。此功能仅适用于外部图形卡。

FPS Xbox游戏酒吧柜台



在游戏过程中，按Windows + G打开游戏栏，您将看到帧计数器。

Windows登录无密码


要使用Windows Hello并不使用密码登录，请转到设置→帐户→登录选项。该设置允许您使用面部识别（Windows Hello）通过个人识别码或通过指纹输入。


重命名虚拟桌面



现在，您可以重命名它们而不是标准名称“ Desktop 1”，“ Desktop 2”等。为此，只需单击“任务视图”窗格顶部的桌面名称。

改进的网络状态信息页面



重新设计了“网络状态”页面：设置→网络和Internet→状态。现在，在页面顶部显示所有可用的网络接口。微软表示，新的界面设计``提供了有关连接设备的更多信息，通过合并多个页面可以使您清楚地了解如何连接到Internet。'' Windows还会显示每个接口的消耗流量。

内置对IP摄像机的支持



Windows 10内置了对本地网络上IP摄像机的支持。以前，您需要第三方软件才能连接到此类软件。

使用此更新，您可以通过转到设置→设备→蓝牙或其他设备→添加蓝牙或其他设备来添加网络摄像机。如果Windows 10在本地网络上检测到受支持的摄像机，则可以一键将其添加到系统中。

要访问IP摄像机，内置的Camera应用程序将为您提供帮助。目前宣布支持摄像机标准ONVIF ProfileS。

登录时控制重新启动应用程序



重新启动PC后，Windows 10会自动重新启动许多应用程序。现在出现了一个新选项，可以轻松禁用它。

转到设置→帐户→登录选项。切换到“关” 按钮“注销时自动保存我的可重新启动应用程序，注销后重新启动它们”。

以前，此选项是隐藏的，并与“使用我的登录信息来自动完成设备设置并在重新启动或更新后打开我的应用程序”结合使用。现在，这是两个单独的参数。

磁盘清理将不再清除“下载”文件夹



Microsoft已从传统磁盘清理中排除了Downloads文件夹。本段的反对者说，清理此文件夹中的所有文件的可能性很高，尤其是如果您是高级用户并且不知道此项目已添加到磁盘清理中的话。
此项已移至“设备内存”，位于设置→系统→设备内存→配置内存控制或启动它。

Paint和写字板现在是可选组件



Microsoft已将Paint和WordPad移至“其他组件”类别。默认情况下，它们仍然是预安装的，但是您可以删除它们以释放一些空间。转到“设置”→“应用程序”→“应用程序和功能”→“其他组件”，您将在其中找到“ Paint”和“写字板”以及其他其他组件，例如Windows Media Player。

设置应用程序中的横幅



微软很早以前就在Parameters中试验了横幅，最后在20H1版本中将其返回。“选项”主页顶部显示一个新的横幅，其中显示配置文件图标，帐户名以及用于编辑Microsoft帐户信息的链接。您还可以看到指向OneDrive的快速链接，更新控制中心的设置以及它们的状态信息。

改善蓝牙设备的配对



如果在附近以配对模式找到设备，您将看到一条提示您配对的通知。整个配对过程直接通过弹出通知进行，而无需打开“设置”。

到目前为止，快速配对仅适用于Microsoft Surface的键盘和鼠标，但是将来它也将能够与其他设备一起使用，从而使蓝牙设备的配对过程更快。

文字光标指示器



现在，您可以在Windows 10中调整文本光标指示器的大小和颜色。
要熟悉设置，请转至设置→辅助功能→文本光标。打开“文本框指示器”，然后选择光标的大小和颜色。

通过眼球运动进行拖放



Windows 10具有眼睛控制功能和专用的眼睛跟踪设备。残疾人的此功能使您可以通过眼球运动来控制PC。在2020年5月更新中，此功能已得到改进：现在，您可以使用鼠标通过眼睛移动来拖动对象。

更新了语言设置页面



现在，“语言设置”页面显示默认的Windows界面语言，应用程序，站点，键盘布局，语音识别和区域设置。

重做的通知设置




现在，当您收到通知时，出现了一个选项来关闭此应用程序的通知或立即转到该应用程序的通知设置。

多显示器增强



Windows 10版本2004包括Windows显示驱动程序模型（WDDM）2.7，当结合使用多台显示器时，可提高整体游戏性能，视频输出和刷新率。

该更新增加了对可变加速器视频刷新率的支持，这将使视频卡可以管理其视频内存。根据Microsoft的说法，此设置应减少延迟并提高游戏的平均帧速率。

您可以通过转到设置→系统→显示来启用可变刷新率。Microsoft通知需要Intel 27.20.100.7859驱动程序和NVIDIA驱动程序450.12或更高版本。

由于AMD PC不支持Microsoft的硬件加速功能，因此AMD尚未发布支持WDDM 2.7的Windows 10兼容驱动程序。

使用多个监视器时，刷新率得到改善，差距得到解决



如果将程序窗口从一个监视器移动到另一个监视器，并且后者的更新频率不同，则更新可减少中断。

对于使用一台显示器的用户，已修复了一个错误，该错误会导致在移动第二个窗口时可能会在以窗口模式运行的游戏中引起抽动。

计算器




添加了在其他窗口上方显示计算器的功能。

日历



添加了直接从任务栏快速添加事件或提醒的功能

减少安装功能更新的时间


从版本1703开始，安装功能更新的时间减少了：在版本1703上，平均更新时间超过80分钟（2004年）不到20分钟，仅重新启动设备一次。

改进的预留存储控制


添加了用于管理保留存储的新命令

Get-WindowsReservedStorageState
Set-WindowsReservedStorageState -State Disable

改进了更新待处理通知



有关即将进行的更新的通知现在将出现在通知中心中，提供以下选项：“立即重启”，“夜间重启”和“选择时间”。该通知还将提醒您将设备连接到电源，以使更新不会中断。
