Publier la bibliothèque de conception de composants VKUI dans Figma

Les bibliothèques dans les outils de conception sont une partie importante d'un système de conception. Nous pensons que les concepteurs devraient travailler avec des éléments aussi proches que possible de l'implémentation dans le code. Par conséquent, dans les bibliothèques, nous avons rassemblé pour eux les composants les plus pertinents qui reflètent toutes les mêmes conditions qui sont disponibles pendant le développement.


Le code source de React a déjà été publié sur Github - à l'instar de nos développeurs, nous partageons maintenant les bibliothèques de composants dans Figma.



Quatre bibliothèques sont disponibles sur la page VK de Figma Community :


  • Bibliothèque de base VKUI - une bibliothèque de base contenant plus de 650 icônes et tous les jetons de couleur pour un thème clair et sombre (sous forme de styles);
  • VKUI iOS Library et VKUI Android Library - bibliothèques de composants pour iOS et Android, collectées sur des jetons de la bibliothèque de base, ainsi que des modèles d'écran prêts à l'emploi;
  • Palette de couleurs VKUI - une palette de couleurs des applications VK qui collectent des jetons de couleur.
