変数とデータ型


数学を理解したので、言語でどのような変数が意味するかを決定しましょう。


C ++では、プログラマーは選択肢を持っています。スタックに配置された自動変数を使用するか、値をプログラムデータメモリに保持して、これらの値へのポインターのみをスタックに配置します。Pythonでこれらのオプションの1つだけを選択した場合はどうなりますか？


もちろん、大きなデータ構造はスタックに収まらない、またはスタック上でのそれらの一定の動きがパフォーマンスの問題を引き起こすため、変数の値のみを常に使用できるわけではありません。したがって、Pythonではポインタのみを使用します。これは概念的に言語を単純化します。


だから式


a = 3

プログラムデータメモリ（いわゆる「ヒープ」）にオブジェクト「3」を作成し、「a」という名前をそのオブジェクトへの参照にしたことを意味します。そして表現


b = a

この場合、変数「b」がメモリ内の「a」が参照するのと同じオブジェクトを参照するように強制したこと、つまり、ポインタをコピーしたことを意味します。
