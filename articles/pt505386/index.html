Instalando o Debian no Netgear Stora

Outro dia, tive esse milagre em minhas mãos: netgear ms 2000 - stora . Decidi abandonar imediatamente o uso do sistema operacional incorporado e instalar o debian no disco rígido.

As informações na rede estão um pouco dispersas, há muito tempo que os links morriam, então decidi atualizar o processo de instalação do debian no stora. Quem se importa, bem-vindo ao gato.

Este artigo foi a fonte principal .

Primeiro, precisamos de imagens para instalar o sistema: peguei aqui . Baixe os dois arquivos. Esses arquivos são gravados na raiz em uma unidade flash formatada em fat32.
Também é necessário um conversor de USB para UART PL2303TA.

Eu tive isso


Você também precisará de um software para conectar-se a um pedaço de ferro, por exemplo, hiperterminal ou massa de vidraceiro (eu não peguei com massa de vidraceiro: todo o tempo fiquei louco, então usei hiperterminal.

Para conectar a peça de ferro com um cabo, preciso desmontá-lo primeiro. O processo é simples, por isso não vou descrevê-lo Bem, não se esqueça de inserir o disco rígido no primeiro slot do lado em que a instalação real ocorrerá.

Após a análise do hardware, conectamos o adaptador Atenção, não conecte o fio vermelho, ou seja, você só precisa conectar 3 fios (da bateria: preto , verde branco).
Então, o fio está conectado, os drivers estão conectados. No driver da porta com, definimos os parâmetros: velocidade 115200, número de bits 8, bits de parada 1, sem paridade. Depois disso, ligue o pedaço de ferro e conecte-o no terminal. Quando vir Pressione qualquer tecla ... pressione qualquer tecla para entrar no carregador de inicialização em u.

Uma pequena digressão.

A lista de comandos com os quais operaremos e quais serão úteis:
usb reset, ide reset — usb, ide
fatls, ext2ls — fat ext2 .
setenv —
saveenv —
reset —
printenv —
printenv NAME — NAME
help —

Depois de entrar no carregador de inicialização, defina os parâmetros de rede, inicialize o dispositivo usb, verifique se há arquivos necessários na unidade flash, salve esses parâmetros na memória do dispositivo e reinicie:

Equipas 
usb reset
fatls usb 0
setenv mainlineLinux yes
setenv arcNumber 2743
setenv ipaddr your_IP
setenv gatewayip your_GW_IP
setenv dnsip your_DNS_IP
saveenv
reset

Após a reinicialização, digite os comandos para iniciar a instalação do debian:

usb reset
fatload usb 0 0x200000 uImage
fatload usb 0 0x800000 uInitrd
setenv bootargs console=ttyS0,115200n8 base-installer/initramfs-tools/driver-policy=most
bootm 0x200000 0x800000

Depois disso, a instalação padrão do debian no modo de texto será executada. Instalamos o sistema, reinicializamos após a instalação, entram no uboot e nos comandos para inicializar o dispositivo a partir do disco rígido:

setenv bootcmd_ide 'ide reset; ext2load ide 0 0x200000 /uImage; ext2load ide 0 0x800000 /uInitrd'
setenv bootcmd 'setenv bootargs $(console) root=/dev/sda2; run bootcmd_ide; bootm 0x200000 0x800000'
saveenv
reset

Após a reinicialização, o debian é carregado a partir do disco rígido, que foi originalmente procurado.

PS Restaurando o carregador de inicialização original:
