現代物理学におけるニュートンの第一法則の異常な解釈について

20世紀初頭、賢いオタクが物理学に現れ、科学的創造性の方法論そのものを認識せず、ニュートン力学の適用範囲を狭めるいくつかの「制限」を発表しました（I.ニュートン）。

ニュートンの法則が現代の物理学の教科書でどのように解釈されるかから始めましょう。たとえば、A.V。ペリシキナとE.M. 42ページのGutnik [1]には次のように書かれています。
<...>現代の概念の観点から、ニュートンの第1法則は次のように定式化されます。

他の物体がそれらに作用しない場合、または他の物体の動作が補償される場合、物体が速度を変更せずに維持する基準システムがあります。
ちなみに、この本のスクワロとニュートン自身の公式[2]を、同じ本の前のページにある[2]と比較してください。
適用された力によって力が加えられてこの状態が変化するまで、すべての身体は静止状態または均一な直線運動の状態を保ち続けます。
ご覧のとおり、参照フレームについては問題ありません。だから何？ニュートンは彼の法律を「改善した」現代の「賢明な人」より「愚か」でしたか？番号！ニュートンは...の質問を詳細に調べます：「明白な」および「本当の」動き！これは、「定義」セクションの「教え」で彼が書いたものです（[2]を参照）。
真の動きと見かけの動きを区別する起源の原因は、これらの動きを生み出すために体に加えなければならない力です。移動体自体に直接加えられる力の作用を除いて、真の絶対運動は発生も変更もできませんが、物体の相対運動は、この物体に力を加えなくても生成および変更できます。この動きが決定される関係で力が体に加えられることで十分です。
つまり、ニュートンはリファレンスシステムの問題を非常に注意深く検討し、すべての動きの根本的な原因と動きの変化、つまり力の作用の理解を妨げる明白な（相対的な）動きから抽象化することに成功しました。

ニュートンと彼の前任者であるコペルニクスの天才は、相対的な見かけの動きの記述にのみ基づいており、したがって、原因となる理由を決定するのに無力である、プトレマイオスの惑星運動の理論（Πτολεμαΐος）を拒否しましたそのような動き。 「修正する」ニュートンがプトレマイオスとアリストテレスの時代に私たちを元に戻そうとしていることがわかります（Αριστοτέλης）...

ちなみに、アインシュタインの相対性理論全体は、相対的で見かけの動きを正確に考慮しており、この理論の結果が解決できないパラドックスであることは驚くに値しません。

アインシュタインの理論では、休息の状態は絶対化されていることにも注意してください。これも、アリストテレスとプトレマイオスの時代への回帰です。そしてニュートンは彼の第一法則により、v = 0 = constはv = const > 0と比較して利点がないことを特に強調しています。
