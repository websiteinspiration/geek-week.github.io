Pemain yang berbicara di raspberry

Selamat datang semuanya. Saya memiliki masalah penglihatan, hobi saya pemrograman.

Entah bagaimana di masa lalu, saya belajar tentang komputer single-board raspberry, deskripsi saya sangat mengesankan dan saya mendapatkannya.

Jadi, sebagai distribusi resmi - raspbian, berdasarkan debian, saya pikir saya akan memasukkan fitur aksesibilitas, pembaca layar orca dan saya akan memiliki komputer mini yang berfungsi penuh.
Pada kenyataannya, semuanya ternyata tidak begitu baik. Saya mencoba berbagai OS, tetapi operasi normal pembaca layar dengan dukungan untuk bahasa Rusia ternyata tidak.

Ada pembaca layar fenrir normal, tetapi tanpa dukungan bahasa Rusia.

Sebelumnya, saya menulis sketsa untuk Arduino dengan antarmuka audio, saya memutuskan untuk mencoba menulis sesuatu yang mirip dengan raspberry, pada saat yang sama akan ada insentif untuk belajar python, karena saya tidak tahu itu.

Akibatnya, ia menulis naskah yang memutar file audio lokal, stasiun radio Internet, synthesizer mengatakan waktu dan tanggal.

Script memiliki menu pembicaraan, navigasi yang dilakukan menggunakan tombol panah, hingga | bawah memilih item, kanan mengaktifkan suatu item, kiri melompat kembali.
Anda dapat mengaktifkan item menu menggunakan bilah spasi atau masuk.

Jika pemain aktif | tombol radio naik | beralih trek ke bawah, jika tidak Anda dapat memilih dari daftar.

Saya juga menggantung tindakan pada tombol exc untuk mematikan komputer.

Semua tindakan disuarakan dalam bahasa Rusia oleh synthesizer RHVoice.

Kelebihan dari solusi ini adalah bahwa seseorang dengan pengetahuan pemrograman minimal dapat membuat pemain minimal sangat nyaman dengan kontrol sederhana.
Semua pengaturan ditulis dalam skrip dan dapat diubah sesuai kebijaksanaan Anda.
Letakkan skrip dalam autorun, dan ternyata menjadi tifloplayer yang berfungsi penuh.
Rencana termasuk multibahasa, dukungan untuk synthesizer lain, menghubungkan perpustakaan online, dll. pada hal-hal kecil.

Mungkin saya melakukan sesuatu yang salah, saya akan berterima kasih atas informasi untuk pengembangan ke arah ini.
