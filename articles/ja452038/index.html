危険な取引

新しいサービスは新しい機会です。すべての最新のビジネスを非常に迅速に追跡する攻撃者を含みます。

たとえば、5月6日、SberbankはSafe Transactionサービスを開始しました。これは、参加者によるトランザクションの支払いを保証し、参加者の権利を保護することを目的としています。そのようなサービスの提供を専門とする会社であるSafeCrowは、銀行の技術パートナーとして行動しました。

銀行のウェブサイトに対応するページが作成され、サービスの個人アカウントは別のドメイン（sb-sdelka.ru）にあります。

ちょうど1週間後の5月13日、リソースsberbank-service.onlineがネットワークに現れ、前述のサービスを模倣しました。それらを比較してみましょう。

これは、Sberbank Webサイトのサービスページの外観です。



そう-攻撃者のサイトで。



両方のページの重要な要素は、[Create Trade]ボタンです。ただし、銀行のウェブサイトでこのボタンをクリックすると、個人アカウントが表示されます。このアカウントでは、SMSの電話番号とコードを使用してログインすることができます。



その悪意のあるリソースは、何の説明もなく、単にパスワードの入力を求めます。



詐欺サイトについて詳しく知る。

元のドメインの場合Sberbankサービスによって使用されたものは、RU-CENTERを通じてSafeCrowによって登録され、その後、アメリカの会社Network SolutionsがSBERBANK-SERVICE.ONLINEドメイン名のレジストラとして機能しました。同時に、Whoisの国別識別子はロシアを指し、RU-NVSは地域フィールドに表示されます。これは明らかにノボシビルスクを意味します。ドメインへのバインディングに、メールアドレスsb.service.help@gmail.comが表示されます。
このサイトはWix.comプラットフォームでホストされています。ホストされているだけでなく、Wixは主にオンラインサイトビルダーであるためです。ページコードを見るとすぐに、meta name = "generator" content = "Wix.com Website Builder"がわかります。攻撃者は気にせず、オンラインビルダーに直接フィッシングサイトをまとめたようです。

ちなみに、これは他の同様のリソースとは区別されます。過去数か月間、Sberbankの顧客をだますために設計されたサイトのほとんどは、JavaScriptのスプラッシュを含む純粋なHTMLで作成されているか、何らかの独自のエンジンを使用しています。そして、ここでも写真はstatic.wixstatic.com/mediaでホストされています。

サイトには有効なSSL証明書があるため、Google Chromeは、リソースを最も親密な方法で信頼できることを慎重に通知します。



ページコードの分析は特別な結果をもたらしません。 Wixから受け継がれた、ジャンクとJavaScriptの堅固さ。このサイトにはgoogle-site-verificationタグとGoogleアナリティクススクリプトがありますが、誰もが対象者を調査したいと思っているため、フィッシングリソースであっても珍しくありません。

サイトの上部領域とフッターは銀行のサイトからほぼ正確にコピーされていますが、フィッシングリソースは完全に拡大縮小する機能を失い、元のフォントを失っています。トップメニューが変更されました。その中のいくつかのリンクはSberbank Webサイトに入りますが、ボタンの数と名前はオリジナルとは異なり、「ホーム」、「ライセンス」、「取引」ボタンはフィッシングリソースの要素を指します。「ライセンス」セクションには、Sberbankの詳細を含む表と、docs.wixstatic.comにある中央銀行の一般ライセンスのスキャンを含むpdfファイルへのリンクが含まれています。メインページの画像は、Istockの写真ストックから取得されました。

要約する


現在の形態では、サイトは犯罪計画の要素の1つとして使用できます。パスワード入力フォーム、ログインと登録の欠如は、サイトに侵入した被害者が攻撃者によって送信された準備ができたパスワードをすでに持っていることを示唆しています。

現時点では詐欺的なスキームの詳細をすべて調査することは不可能であるという事実にもかかわらず、このサイトは現在、銀行の顧客を誤解させることを意図しているため、脅威になる可能性があります。

識別された脅威についてSberbank PJSCおよびSafeCrowに通知し、最初の被害者が現れる前にフィッシングリソースが存在しなくなることを願っています。
