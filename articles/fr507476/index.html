Conseils pour les développeurs Web novices

Je crée des sites Web sous une forme ou une autre depuis 1995. Après 25 ans de travail, je pense avoir accumulé de l'expérience et maintenant j'ai suffisamment de connaissances. Ci-dessous sont présentés au hasard quelques points sur lesquels j'aimerais que les développeurs débutants réfléchissent.



  • Parfois, un site n'est qu'un site.
  • Le navigateur est déjà un client; HTML est la langue qu'il parle.
  • Le Web est construit autour de la visualisation côté serveur.
  • Vous pouvez fournir des données de plusieurs manières; considérer le HTML comme une façon de présenter les données.
  • La mise à l'échelle sur le côté de votre serveur aidera tout le monde, tandis que la mise à l'échelle sur le côté client n'aidera que ceux qui disposent d'un matériel puissant et de connexions rapides.
  • Tout le monde n'a pas (ou ne peut pas utiliser) de souris.
  • Tout le monde n'a pas (ou ne peut pas utiliser) de clavier.

Plusieurs personnes se sont intéressées à ce que je voulais dire en disant «garder l'état du client entre les chargements de pages». Ce n'est pas un manuel, mais je veux dire que vous pouvez faire quelque chose comme ça: 



<html><head>
<title>Div replacement test</title>
<script>
window.addEventListener('DOMContentLoaded', () => {
    [].slice.call(document.getElementsByClassName("replace")).forEach(elem => {
        elem.addEventListener('click', event => {
            event.preventDefault();
            fetch(elem.href)
                .then(response => response.text())
                .then(text => {
                    let parser = new DOMParser();
                    var other = parser.parseFromString(text, 'text/html');
                    document.title = other.querySelector('title').innerHTML;
                    var content = document.getElementById('content')
                    content.innerHTML = other.querySelector('#content').innerHTML
                })

        })
    })
})
</script>
</head><body>
<section id="content">
<p>Here is the content container. <a class="replace" href="replaced.html">The other document</a></p>
</section>
