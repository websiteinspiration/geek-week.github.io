海を征服する：フローティングデータウェアハウス

データセンターの波打つ海のテーマを継続します。マイクロソフトは2015年の夏の終わりに、新しい水中データセンターの実験用プロトタイプ、つまりネイティックプロジェクトによるとコード名がLeona Philpotのプロトタイプを浸しました。 2018年、ナティックプロジェクトの第2ステージが行われ、12ラックと864台のサーバーを備えた40フィートのISOコンテナが、オークニー諸島（スコットランド）近くの北海の海に「送られました」。



Keppel Data CentersがNautilus Data Technologiesの立ち上げに1000万ドルを投資してから2年が経過しました。 Keppel Data Centersは、シンガポールで最大の3つのデータセンタープロバイダーの1つです。シンガポールに加えて、ケッペルはマレーシア、香港、オーストラリア、ドイツ、オランダ、イギリス、アイルランドにデータセンターを持っています。

NDTはカリフォルニアの新興企業であり、フローティングデータセンターです。マール島（マレ島）の近くにデータバージを配置する予定です。これはEli Mと呼ばれる最初のNautilus商用データセンターです。現在、水域は建設中で、2019年の秋に就役しますが、最初の航海は2017年に計画されました。


2017年に建設中です。

このようなフローティングサーバーファームの利点の1つは、地上ベースのデータセンターの建設が、建設用スペースの不足、土地の高価格など、さまざまな理由で不可能である地域に配置できることです。主な要件は、十分な水の可用性です。 Keppel Data Centersは、このようなフローティングデータセンターの新しい場所の検索を開始しました。

同社は、シンガポール沖にデータセンターをホストする計画を発表しました。これは、土地が不足しているにもかかわらず、データセンターを展開するのに役立ちます。小さな島には十分なスペースがなく、当局は土地利用を厳しく管理しています。政府はデジタルインフラストラクチャの成長を促進することに関心を持っていますが、データセンターにサイトを割り当てる主な利点は、GoogleやAmazonなどの巨人にあります。 Keppel Data Centersは、データセンターのホスティングは陸上よりも大幅にコストが安くなると主張しています。したがって、同社は新しいハイパースケールデータセンターの建設と立ち上げのために沿岸ポイントを積極的に探しています。

CEO Keppel（Wong Wai Meng）によれば、

現時点では、建設にかかる費用とデータセンター自体の正確な数値はわかりません。しかし、20〜50％の価格優位性を予測できます。これが私たちが直面する目標です。

時間の経過とともに浮かぶデータセンターのアイデアは、市場で勢いを増しています。
Keppelのビジョンは、「発電モジュール」を共有するモジュール式のフローティングデータセンターで満たされた、はしけ全体を作成することです。

現在、シンガポール島周辺にデータセンターを建設するために調査および検討しているサイトがいくつかあります。ほとんどのサイトはオフショアにあります。

CNAのYouTubeビデオは、そのような浮動データセンターの艦隊をコンピューターで視覚化したものです。通常のヨット桟橋で見られるのと同様の経路に沿って、20隻以上のバージが水上に配置されています。 4隻のはしけに動力装置がある。



CEOケッペルとのインタビューでは、そのような1つのはしけのデータセンターの推定容量は約10 MWになります。 2018年にサーバーファームの総容量が約360 MW（Structure Researchによると）になったことを考えると、フローティングデータセンターはシンガポール市場での供給を2倍にするのに役立ちます。これにより、新しいサーバーファームの建設のためにそのような要求された領域が不足するという問題が解決されます。



はしけは、Nautilus Data Technologies（カリフォルニア）が開発したものと似ています。 Nautilusは、少なくとも3つのデータセンター建設プロジェクトを実装しています。すべて異なる段階で：Vallejo（カリフォルニア）、Limerick（アイルランド）、Alamida（カリフォルニア）です。

これらのNautilusデータセンターの主要な設計機能は、冷却システムです。冷却媒体として海水を使用しているため、データセンターのエネルギー消費量は従来のデータセンターよりも低くなっています。

同社の計画は完全には明らかにされなかったが、ケッペルはモジュールが陸上で製造され、浮体構造に接続されると付け加えた。

設備は、液化天然ガス（LNG）で駆動され、フローティングモジュールに供給されます。シンガポールの電力の95％は天然ガスから発電されているため、これは最も手頃なエネルギー源です。報告によると、水はデータセンターの冷却に使用されます。
