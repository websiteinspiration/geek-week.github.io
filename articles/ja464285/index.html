Vkontakteプロフィール写真を自動更新

VK APIを使用してVkontakteのプロフィール写真を更新するためのコード。captchaでの作業を検討し、コードをサーバーにアップロードして、写真を自動的に更新します。



必要なライブラリ


Python用VK APIと連携するvk_apiライブラリをインストールします。pipinstall vk_api

Vk_api作業例

プロフィール写真の更新


photos.saveOwnerPhoto（）はユーザーの写真を保存します。

写真を更新すると、この写真の投稿が壁に追加されます。これらの投稿を削除しないと、友達フィードにアバターの更新に関する投稿が詰まります。

posts = vk.wall.get()
post_id = posts["items"][0]["id"]
vk.wall.delete(post_id = post_id)

コードを実行すると、プロフィールの写真が更新されます。
自動化するには、無限ループを追加し、写真を更新するたびに1分間遅延させます。

キャプチャを使用する


vk_api.VkApi（）メソッドはすでにキャプチャ機能を実装しています。ログインとパスワードに加えて、キャプチャ処理関数captcha_handlerを転送する必要があります。

vk_sessionを変更する

vk_session = vk_api.VkApi('+7999132****', '**********', captcha_handler=captcha_handler)

captcha_handler（captcha）関数を追加します。この関数は、キャプチャアドレスを取得し、ユーザーのメッセージにキャプチャ画像を送信し、ユーザーからのキャプチャメッセージを待ちます。

def captcha_handler(captcha):
    
    captcha_url = captcha.get_url()

    key = send_captcha(captcha_url)

    return captcha.try_again(key)

def send_captcha(captcha_url):
    token = ""
    
    vk_session = vk_api.VkApi(token = token)
    vk = vk_session.get_api()
    
    url = vk.photos.getMessagesUploadServer()['upload_url']
    
    request = requests.post(url, files={'photo': open("captcha.jpg", 'rb')}).json()
    
    photo = vk.photos.saveMessagesPhoto(server=request['server'],
                                        photo = request['photo'],
                                        hash = request['hash'])
    
    attachment = 'photo{}_{}'.format(photo[0]['owner_id'], photo[0]['id'])
    
    vk.messages.send(
        user_id=_ID,
        attachment = attachment,
        random_id=get_random_id())
    
    os.remove("captcha.jpg")
    
    key = ''
    while (key == ''):
        messages = vk.messages.getDialogs()['items'][0]
        if 'attachments' not in messages['message'].keys():
            key = messages['message']['body']
    return key
