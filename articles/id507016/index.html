Cara termudah untuk memutakhirkan dari macOS ke Linux

Linux memungkinkan Anda untuk melakukan hal yang hampir sama dengan macOS. Dan terlebih lagi: itu menjadi mungkin berkat komunitas open source yang dikembangkan.

Salah satu cerita transisi dari macOS ke Linux dalam terjemahan ini.


Hampir dua tahun telah berlalu sejak saya beralih dari macOS ke Linux. Sebelum itu, saya telah menggunakan sistem operasi Apple selama 15 tahun. Saya memasang distribusi pertama saya di musim panas 2018. Kemudian saya masih baru di Linux.

Sekarang saya menggunakan Linux secara eksklusif. Di sana saya dapat melakukan apa pun yang saya inginkan: menjelajahi Internet secara teratur dan menjelajahi Netflix, menulis dan mengedit konten untuk blog saya, dan bahkan mengelola startup.

Penting untuk dicatat bahwa saya bukan pengembang, bukan insinyur! Lama berlalu adalah hari-hari ketika diyakini bahwa Linux tidak cocok untuk pengguna biasa, karena tidak memiliki antarmuka yang cukup ramah.

Ada banyak diskusi akhir-akhir ini tentang mengkritik sistem operasi macOS, sehingga semakin banyak orang mempertimbangkan untuk beralih ke Linux. Saya akan membagikan beberapa tips untuk bermigrasi dari macOS ke Linux untuk membantu orang lain melakukan ini dengan cepat dan tanpa terlalu banyak sakit kepala.

Apakah kamu membutuhkannya?


Sebelum beralih dari macOS ke Linux, alangkah baiknya untuk memikirkan apakah Linux tepat untuk Anda. Jika Anda ingin tetap menyinkronkan dengan Apple Watch, melakukan panggilan di FaceTime, atau bekerja di iMovie, jangan jatuhkan macOS. Ini adalah produk eksklusif yang hidup di ekosistem tertutup Apple. Jika Anda menyukai ekosistem ini, kemungkinan besar Linux tidak cocok untuk Anda.

Saya tidak terlalu terikat dengan ekosistem Apple. Saya tidak memiliki iPhone, saya tidak menggunakan iCloud, FaceTime atau Siri. Saya tertarik pada open source, hanya tinggal memutuskan dan mengambil langkah pertama.

Apakah ada versi Linux dari perangkat lunak favorit Anda?


Saya mulai mempelajari perangkat lunak open source ketika saya sedang duduk di macOS, dan menemukan bahwa sebagian besar aplikasi yang saya gunakan akan bekerja pada kedua platform.

Berikut ini adalah peramban Firefox, yang berfungsi pada MacOS dan Linux. Sudahkah Anda menggunakan VLC untuk pemutaran multimedia? Di Linux, itu juga akan berfungsi. Audacity digunakan untuk merekam dan mengedit audio? Setelah beralih ke Linux, Anda dapat membawanya. Apakah Anda menyiarkan langsung di OBS Studio? Ada versi untuk Linux. Apakah Anda menggunakan messenger Telegram? Anda dapat menginstal Telegram untuk Linux.

Ini tidak hanya berlaku untuk perangkat lunak sumber terbuka. Pengembang sebagian besar (mungkin bahkan semua) aplikasi eksklusif non-Apple favorit Anda telah membuat versi Linux: Spotify, Slack, Zoom, Steam, Discord, Skype, Chrome dan banyak lainnya. Selain itu, hampir semua yang dapat Anda jalankan di browser macOS Anda dapat bekerja di browser Linux Anda.

Namun, lebih baik untuk memeriksa apakah ada versi Linux dari aplikasi favorit Anda. Atau mungkin ada alternatif yang memadai atau bahkan lebih menarik bagi mereka. Lakukan riset: cari "aplikasi favorit Anda + Linux" atau "aplikasi favorit Anda + Linux alternatif" di mesin pencari, atau lihat Flathub aplikasi berpemilik yang dapat Anda instal di Linux menggunakan Flatpak.

Jangan terburu-buru membuat "copy" macOS dari Linux


Agar merasa nyaman saat beralih ke Linux, Anda perlu menunjukkan fleksibilitas yang cukup dan bersiaplah untuk mempelajari nuansa menggunakan sistem operasi baru. Untuk melakukan ini, Anda perlu memberi diri Anda sedikit waktu.

Jika Anda ingin Linux terlihat dan berfungsi seperti macOS, itu hampir mustahil. Pada prinsipnya, Anda dapat membuat desktop Linux yang mirip dengan macOS, namun, menurut saya, cara terbaik untuk beralih ke Linux adalah memulai dengan GUI yang lebih standar untuk Linux.

Beri dia kesempatan dan gunakan Linux dengan cara yang semula dimaksudkan. Jangan mencoba mengubah Linux menjadi apa yang bukan. Dan mungkin Anda, seperti saya, akan menikmati bekerja di Linux lebih dari pada macOS.

Ingat ketika Anda pertama kali duduk di Mac: Anda perlu waktu untuk terbiasa dengannya. Jadi, dalam kasus Linux, orang seharusnya tidak berharap untuk keajaiban juga.

Pilih distribusi Linux yang tepat


Tidak seperti Windows dan macOS, sistem berbasis Linux membuat sistem operasi yang sangat berbeda. Saya telah menggunakan dan menguji beberapa distribusi Linux. Saya juga mencoba beberapa desktop (atau GUI pengguna). Mereka sangat berbeda satu sama lain dalam hal estetika, kegunaan, alur kerja dan aplikasi yang tertanam.

Meskipun ElementaryOS dan Pop! _OS sering bertindak sebagai alternatif untuk macOS, saya sarankan memulai dengan Fedora Workstation karena alasan berikut:

  • Mudah untuk menginstal pada drive USB menggunakan Fedora Media Writer .
  • Dia dapat mengenali dari kotak dan bekerja secara memadai dengan semua perangkat keras Anda.
  • Ini mendukung perangkat lunak Linux terbaru.
  • Ini meluncurkan lingkungan desktop GNOME tanpa pengaturan tambahan.
  • Dia memiliki komunitas besar dan tim pengembangan besar.

Menurut pendapat saya, GNOME adalah lingkungan kerja terbaik untuk Linux dalam hal kegunaan, konsistensi, fleksibilitas, dan kegunaan bagi mereka yang telah bermigrasi ke Linux dengan macOS.

Fedora dapat menjadi awal yang baik untuk Anda, dan ketika Anda merasa nyaman, Anda dapat mencoba distribusi lainnya, serta lingkungan desktop dan manajer jendela.

Mengenal GNOME Lebih Baik


GNOME diinstal secara default untuk Fedora dan banyak distribusi Linux lainnya. Upgrade terbarunya ke GNOME 3.36 telah menghadirkan estetika modern yang akan dihargai oleh pengguna Mac.

Bersiaplah untuk kenyataan bahwa Linux, dan bahkan Fedora Workstation dalam kombinasi dengan GNOME, masih akan sangat berbeda dari macOS. GNOME sangat bersih, minimalis, modern. Tidak ada gangguan di sini. Tidak ada ikon di desktop, dan tidak ada dok yang terlihat. Jendela Anda bahkan tidak memiliki tombol perkecil dan maksimalkan. Tapi jangan panik. Jika Anda memberinya kesempatan, itu mungkin bagi Anda sistem operasi terbaik dan paling produktif yang pernah Anda gunakan.

Ketika Anda memulai GNOME, Anda hanya melihat panel atas dan gambar latar belakang. Panel atas terdiri dari tombol Aktivitas di sebelah kiri, waktu dan tanggal di tengah, serta ikon baki untuk jaringan, Bluetooth, VPN, suara, kecerahan, pengisian daya baterai (dan sebagainya) di sisi kanan.

Bagaimana GNOME seperti macOS


Anda akan melihat beberapa kesamaan dengan macOS, seperti mengikat jendela dan mempratinjau dokumen dengan menekan tombol spasi (berfungsi seperti Quick Look).

Jika Anda mengklik Aktivitas di panel atas atau menekan tombol Super (mirip dengan tombol Apple) pada keyboard, Anda akan melihat sesuatu yang mirip dengan Kontrol Misi MacOS dan Pencarian Spotlight dalam satu botol. Dengan demikian, Anda dapat melihat informasi tentang semua aplikasi dan jendela yang terbuka. Di sisi kiri Anda akan melihat dok di mana semua aplikasi (pilihan) favorit Anda disajikan.

Di bagian atas layar ada kotak pencarian. Segera setelah Anda mulai mengetik, fokus Anda akan ada di sana. Jadi, Anda dapat mulai mencari aplikasi yang terinstal dan konten file, menemukan aplikasi di Pusat Aplikasi, memeriksa waktu dan cuaca, dan sebagainya. Ini berfungsi sama dengan Spotlight. Mulailah mengetik apa yang ingin Anda temukan, dan tekan Enter untuk membuka aplikasi atau file.

Anda juga dapat melihat daftar semua aplikasi yang diinstal (sama seperti Launchpad di Mac). Klik ikon Tampilkan Aplikasi di dok atau pintasan keyboard Super + A.
Linux biasanya bekerja cukup cepat bahkan pada perangkat keras yang lebih lama dan membutuhkan ruang disk yang sangat sedikit dibandingkan dengan macOS. Dan tidak seperti macOS, Anda dapat menghapus aplikasi yang sudah diinstal sebelumnya yang tidak Anda butuhkan.

Kustomisasi GNOME


Tinjau pengaturan GNOME Anda untuk membuat perubahan yang mungkin membuatnya lebih nyaman bagi Anda. Berikut adalah beberapa hal yang saya lakukan segera setelah saya menginstal GNOME:

  • Di Mouse & Touchpad, saya mematikan pengguliran alami dan mengaktifkan klik tombol.
  • Dalam Layar , saya menyalakan lampu malam, yang membuat layar lebih hangat di malam hari untuk mencegah kelelahan mata.
  • Saya juga menginstal Tweak GNOME untuk mengakses pengaturan lanjutan.
  • Dalam tweak, saya mengaktifkan gain yang berlebihan untuk suara untuk meningkatkan volume di atas 100%.
  • Dalam tweak, saya juga menyertakan tema Adwaita Dark, yang saya lebih suka tema cahaya default.

Menangani hotkey


GNOME berorientasi pada keyboard, jadi cobalah untuk menggunakannya lebih banyak. Di bagian Pintasan Keyboard dari pengaturan GNOME, Anda dapat menemukan daftar pintasan keyboard yang berbeda.

Anda juga dapat menambahkan pintasan keyboard Anda sendiri. Saya mengkonfigurasi aplikasi saya yang paling sering digunakan untuk membuka dengan tombol Super. Misalnya, Super + B untuk browser saya, Super + F untuk file, Super + T untuk terminal, dan sebagainya. Saya juga memilih Ctrl + Q untuk menutup jendela saat ini.

Saya beralih di antara aplikasi yang terbuka menggunakan Super + Tab. Dan saya menggunakan Super + H untuk menyembunyikan jendela. Saya menekan F11 untuk membuka aplikasi dalam layar penuh. Super + panah kiri memungkinkan Anda untuk mengambil aplikasi saat ini ke sisi kiri layar. Panah super + kanan memungkinkan Anda untuk menggesernya ke sisi kanan layar. Dll

Jalankan Linux dalam mode uji


Anda dapat mencoba bekerja dengan Fedora di Mac Anda sebelum menginstalnya sepenuhnya. Cukup unduh file gambar ISO dari situs web Fedora . Pasang file gambar ISO ke drive USB menggunakan Etcher dan boot dari drive itu dengan menekan tombol Option saat startup komputer sehingga Anda dapat mencoba OS live.

Sekarang Anda dapat menjelajahi Fedora Workstation dengan aman tanpa menginstal apa pun tambahan di Mac Anda. Periksa bagaimana OS ini bekerja dengan perangkat keras dan jaringan Anda: dapatkah Anda terhubung ke WiFi? Apakah touchpad berfungsi? Bagaimana dengan audio? Dll

Luangkan waktu untuk belajar GNOME. Lihat berbagai fitur yang saya jelaskan di atas. Buka beberapa aplikasi yang diinstal. Jika semuanya terlihat baik, jika Anda menyukai tampilan Fedora Workstation dan GNOME, maka Anda dapat menyelesaikan instalasi di Mac Anda.

Selamat datang di dunia Linux!
