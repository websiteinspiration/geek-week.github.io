金士顿KC600 512GB：固体火箭



最近，许多制造商越来越关注M.2 NVMe驱动器的设计和生产，而许多PC用户仍然继续使用2.5英寸SSD。很高兴金士顿不会忘记这一点，继续发布2.5英寸解决方案。今天我们来进行回顾-512 GB 金士顿KC600，它支持通过SATA III总线进行连接（也有出售的版本，容量为256 GB和1 TB）。

根据零售商的统计，这是买家中运行量最大的。好吧...这很合逻辑。不管怎么说，SSD仍然比传统HDD贵，因此容量为1 TB的固态解决方案很容易越过10,000卢布的心理障碍。同时，如果用户玩游戏并使用“繁重的”程序（例如，来自Adobe的图形设计软件包），则256 GB几乎没有什么用。

金士顿KC600是金士顿UV500驱动器固有的传统的延续。的确，与UV系列相比，金士顿KC驱动器便宜得多。而且，容量越高，成本差异越大。为了不打折扣，我们以Yandex.Market的价格标签为例，其中金士顿UV500 480GB（SATA III）的平均价格为7,000卢布，金士顿KC600 512GB（SATA III）的价格为6,300卢布。

金士顿KC600：规格


金士顿KC600采用吸塑包装，可立即告知我们该驱动器具有5年保修。我们将打开包装，并没有任何乐趣-驱动器盒（只有7毫米厚）不是由某种塑料制成，而是由铝制成，铝不仅起到保护组件底座的作用，而且还可以起到散热作用。



机箱内部有一块紧凑的电路板：在其一侧有两个96层微米3D TLC NAND闪存模块（每个128 GB）和一个Kingston 512 MB LPDDR4 RAM缓冲内存模块（每1 GB驱动器内存1 MB DRAM内存） ，第二个-另外两个闪存模块（每个模块也为128 GB）和一个4通道Silicon Motion SM2259控制器。

通常，要么将一小部分SSD分配给高速缓存（从2到16 GB的静态SLC高速缓存），要么将部分单元动态切换到SLC模式（在这种情况下，最多可以将10％的容量分配给高速缓存），或者其中两个可以同时工作方法（静态缓存由动态补充）。该驱动器的主要功能之一是其全部容量都可以用作快速的SLC缓存：也就是说，取决于“磁盘”的大小，内存的类型会动态变化（SLC中的TLC）。这样，您就可以在记录整个磁盘容量时，平整较慢的TLC内存的工作，并消除了在静态SLC模式下速度的急剧下降。



如果我们再次提到5年保修，则值得一提的是该驱动器的MTBF。原则上，在遗忘之前，可以将多少数据写入驱动器？根据金士顿KC600的技术规范，容量为512 GB的驱动器的TBW（记录的字节总数）将为150 TB。据统计，在典型的家用PC中，一年使用主动使用的SSD会覆盖10到30 TB的数据。因此，您可以确定金士顿KC600可以在五年内正常工作并且不会超过保修期，然后才有理由成为不可靠的存储设备。此外，制造商保证了100万小时的MTBF。



除了高数据传输速率（> 500 MB / s）之外，金士顿KC600还支持SMART属性，TRIM，NCQ，TCG Opal 2.0规范，AES 256位硬件加密和eDrive。我们还建议您从制造商的官方网站下载Kingston SSD Manager程序，该程序可让您管理安全功能，更新固件，执行格式化并仅监视SSD的状态。

一段时间以来，能够对整个驱动器进行加密一直是高性能SSD的功能，但金士顿（Kingston）在这里提供了此功能，为KC600配备了与三星860系列产品相抗衡的全部功能。在任何台式机和移动计算机上，但是从性能方面将向我们展示什么？

金士顿KC600 512GB：性能测试


评估SATA SSD仅有三个重要因素：价格，性能和耐用性。如果您不考虑价格问题，那么目前任何SATA驱动器的性能都主要受SATA接口的限制，因此带宽上限为6 Gb / s（768 MB / s）。这些只是理论指标。实际上，在读取和写入数据时，没有一个固态驱动器会达到这样的速度。



格式化后，金士顿KC600 512GB的实际容量为488.3 GB。存储器的其余部分用于控制闪存。我们在具有64位Windows 10版本18.363的游戏计算机上执行的所有测试。至于我们在其上“驱动”驱动器的测试台，其配置如下表所示。



如今，测试人员拥有许多不同的程序，它们通过综合模拟负载来衡量SSD解决方案的性能。但是，它们都无法尽可能准确地测量工作速度。因此，对于测试，我们使用各种软件，然后依靠平均结果。

CrystalDiskMark 5.2.1


在CrystalDiskMark测试中，速度指示器的读取速度为564 MB / s，写入速度为516 MB / s，这对于SATA III驱动器而言是一项了不起的成就。在某些情况下，这些结果看起来似乎很熟悉，这并不奇怪：尽管三星860 EVO驱动器具有不同的内存和控制器，但它们可以观察到相同的指示器。







ATTO磁盘基准


ATTO Disk Benchmark显示的结果总是很有趣，因为该程序显示了传输数据块的大小与读取/写入速度之间的关系。查看图表，我们看到金斯敦KC600的潜力在256 KB块大小的处理过程中得到了体现。底线：写入的最大速度值为494 MB / s，数据读取的最大速度值为538 MB / s。





AS SSD基准1.9.5


AS SSD Benchmark综合测试套件是另一个基准测试工具，可以模拟在各种工作负载下处理大多数不可压缩的数据。结果稍微适中，但CrystalDiskMark之间的差距不是很大：读取速度为527 MB / s，写入数据速度为485 MB / s。







HD Tune Pro 4.60


测试方案HD Tune Pro被视为参考。该程序一次测量三个参数：读写时的最大，平均和最小速度。但是，如果将其结果与AS SSD Benchmark和CrystalDiskMark进行比较，他们总是会更加怀疑。在这种情况下，该实用程序会显示最大400 MB / s的写入速度和446 MB / s的读取速度。

在测试过程中，HD Tune Pro模拟了写入大小为8 GB的文件驱动器的过程（直到磁盘完全装满），然后模拟了从40 GB文件中读取信息的过程。在第一种情况下，数据记录速度平均在325 MB / s至275 MB / s之间变化。在第二项测试中，数据读取速度介于446 MB / s至334 MB / s之间。同时，图中的速度没有强烈的下沉。





AnvilPro 1.1.0


AnvilPro是一种老式的但仍可靠的工具，用于测量数据存储设备的性能，该性能可记录读取/写入速度，输入输出操作（IOPS）的数量以及负载下的耐久性系数。对于金士顿KC600 512GB，测量结果如下：读取时为512 MB / s，写入时为465 MB / s。每秒平均I / O操作数为：读取85,731 IOPS，写入87,709 IOPS。
