Distribuições de jogos Linux

O Linux oferece distribuições universais para trabalho e jogos e distribuições especializadas para jogos. Neste post sobre quais distribuições de jogos Linux são relevantes hoje (puramente IMHO, é claro), e você pode participar da pesquisa e compartilhar sua opinião sobre esse assunto.


Os jogos Linux começaram a ganhar popularidade em 2013, quando a Valve anunciou a criação do SteamOS baseado em Linux. Além disso, em janeiro de 2013, a Valve transportou seu jogo Half-Life para o Linux. Os usuários do Linux tiveram a oportunidade de jogar jogos de alta qualidade, que no passado precisavam do sistema operacional Windows ou mesmo de um console de jogos. Tudo começou de maneira bastante modesta: apenas algumas empresas (CD Projekt Red, Deep Silver, Valve e outras) adicionaram o ícone do pingüim Linux à sua lista de compatibilidade. Mas, no final, até a Gearbox e a Square Enix lançaram versões Linux dos seus melhores jogos. Hoje, graças ao projeto Valve Proton, os jogos que não possuem um lançamento oficial para Linux ainda podem funcionar no SteamOS e em outras distribuições Linux.

No entanto, não apenas a Valve portou jogos para o Linux. A id Software é uma das primeiras empresas no mercado de jogos Linux. As versões dos jogos criados pela empresa para Linux eram muito populares. Muitos deles receberam os prêmios “Readers 'Choice” e “Editors' Choice” do Linux Journal . O primeiro jogo de software id portado para Linux foi o Doom , e muitos outros jogos da empresa também foram portados . Independentemente de você gostar de jogos modernos ou retrô, entre as distribuições de jogos de código aberto do Linux, certamente haverá opções.

Aqui está uma rápida visão geral de algumas distribuições Linux que os jogadores vão adorar.

SteamOS



SteamOSde propriedade da Valve. Ele permite que você "transforme" seu PC em um console de jogos. Os computadores para jogos da Valve (Steam Machine) são dispositivos que podem se tornar um substituto para os consoles. E então a escolha do SteamOS é ainda mais óbvia. Não há nada de especial no SteamOS: na verdade, é apenas o Debian GNU / Linux com o Steam instalado como o aplicativo padrão. Quando você inicializa o computador, o Steam inicia automaticamente e você pode interagir com ele usando apenas o Steam Controller ou qualquer gamepad no estilo Xbox. Você pode criar a mesma configuração instalando o Steam em qualquer distribuição e ativando o modo Big Picture. Após a instalação do SteamOS, você pode escolher entre duas sessões: SteamOS Desktop, onde encontrará o ambiente familiar baseado no GNOME e a Steam-session, onde você poderá fazer login na sua conta Steam.

O SteamOS ainda possui recursos personalizados para jogos. É improvável que você o use para trabalhar ou estudar. O SteamOS também é um firmware que se concentra em máquinas com UEFI e BIOS, uma ampla variedade de placas gráficas, incluindo Nvidia e ATI, além de componentes de hardware modernos.

Lakka



Lakka emula a interface do Playstation 3 para jogos retrô. Instalei o Lakka em um Raspberry Pi Rev 1 usando o Etcher . Lakka carrega uma interface que é muito familiar para os jogadores do PS3 e, assim como no Playstation, você pode controlar tudo usando apenas o controle do jogo.

Lakka é especialista em jogos retrô e fornece emuladores de jogos para sistemas e mecanismos mais antigos. Se você tiver as imagens de ROM apropriadas, poderá usar emuladores para jogar jogos da Nintendo, Sega Genesis, Dreamcast, N64 ou jogos caseiros POWDER, Warcraft Tower Defense e assim por diante.

Não há jogos pré-instalados no Lakka OS, mas facilita adicioná-los via SSH ou Samba.

Pop_OS!



Nem todo mundo quer criar um console de videogame a partir do computador. Às vezes, tudo o que você realmente precisa é de um bom computador com a capacidade de executar jogos com desempenho máximo. System76 desenvolveu a área de trabalho Pop_OS! com base na área de trabalho padrão do GNOME com alguns complementos. Pop_OS! Não está tentando ser o mais moderno, seus designers simplesmente implementaram configurações padrão convenientes. Para jogadores - é fácil acesso ao Steam, Proton, WINE, emuladores de jogos, PlayOnLinux, reconhecimento e configuração automáticos do controlador de jogos e muito mais. Pop_OS! não muito longe do Ubuntu, foi modificado conforme necessário para poder adicionar as funções necessárias.

Entre outras coisas, Pop_OS! - Este é um desktop de alto desempenho. Ele implementa todas as conveniências internas do GNOME para maximizar a eficiência e faz as modificações necessárias para atender aos padrões universais formados por décadas no KDE Plasma, Finder e Explorer. Pop_OS! - Este é um ambiente intuitivo que ajuda o usuário a se concentrar diretamente na jogabilidade, e não na manutenção.

Drauger os



Drauger- Distribuição Linux baseada no Ubuntu (filial LTS). Ainda é um projeto jovem, mas incorpora uma filosofia interessante de processos de computador e jogo e foi projetado para economizar recursos para a tarefa. Para isso, o Drauger OS abandonou o conceito de área de trabalho tradicional e, em vez disso, fornece um painel de controle simplificado que permite iniciar um cliente de jogo (por exemplo, Steam, PlayOnLinux, Lutris e assim por diante) e configurar serviços (por exemplo, sua rede) ou iniciar um aplicativo. No início, é um pouco desorientador, especialmente quando o painel de controle praticamente se dissolve em segundo plano, mas após várias horas de trabalho você entende que geralmente não é necessário um desktop completo. Afinal, é improvável que você concorde que o principal em um computador é sua área de trabalho.De fato, é importante que você entre no aplicativo de maneira mais rápida e fácil e que esse aplicativo funcione bem.

É impossível não dizer sobre desempenho. O Drauger OS tenta maximizar o desempenho dos jogos usando um kernel de baixa latência. O kernel faz parte do sistema operacional que interage com dispositivos externos, como controladores de jogo, mouses e teclados, e até discos rígidos, memória e placas de vídeo. O kernel universal fornecido com a maioria das distribuições Linux presta mais ou menos a mesma atenção a todos os processos. Um kernel de baixa latência pode dar preferência a determinados processos, incluindo vídeo e gráficos, para garantir que os cálculos para tarefas importantes sejam executados rapidamente, enquanto a prioridade das tarefas comuns do sistema será reduzida. O núcleo do Drauger é orientado para o desempenho, portanto seus jogos terão precedência sobre todos os outros processos.

se você observar os recursos mais básicos das "distribuições de jogos" do Linux, elas são muito semelhantes. Surpreendentemente, até jogo no meu laptop com RHEL (graças ao Flatpak da Red Hat que permite a instalação do Steam). Portanto, se você quiser jogar no Linux nesta década, a questão não é como fazê-lo, mas qual sistema usar.

A resposta mais simples para a pergunta de qual distribuição Linux usar é a seguinte: escolha a que funciona melhor em sua máquina. Você precisa encontrar uma distribuição Linux que possa reconhecer facilmente seus controladores de hardware e jogos. Encontre e você será feliz: tudo o que resta é instalar os jogos e começar a jogar.
