Cloud Security Alarm Security Hub

El sistema de alarma ruso securityHub funciona con sensores cableados e inalámbricos, puede controlar de forma remota varios dispositivos, está integrado con videovigilancia iVideon y utiliza una conexión Ethernet por cable e Internet móvil para la comunicación.

Llevo dos semanas estudiando este sistema.



Vi el stand de la empresa Kazan Teko en la exposición Securika hace un año. Entre otras cosas, mostraron la alarma de la casa de Security Hub. Me interesaban dos cosas: no solo era compatible con sensores inalámbricos sino también con cables y la capacidad de conectarse a Internet a través de un cable Ethernet normal.

Ahora, muchos fabricantes producen alarmas domésticas que solo admiten sensores inalámbricos. No solo es costoso, también es menos confiable. Creo que al menos dos sensores (abrir la puerta de entrada y movimiento en el pasillo) deben estar conectados. La conexión por cable al enrutador también parece más confiable que el Wi-Fi.

Comenzaré con las características de Security Hub.

  • soporte para seguridad cableada y sensores de incendio para cortocircuito, abierto o de dos niveles;
  • hasta 4 entradas para sensores con cable (se puede conectar cualquier cantidad de sensores a cada entrada);
  • soporte para hasta 30 sensores inalámbricos con su propio protocolo de intercambio seguro;
  • soporte para hasta seis salidas cableadas (balizas, sirenas, actuadores);
  • Entrada de memoria táctil (TM) para administrar teclas y / o teclado iButton;
  • gestión y transmisión de información sobre alarmas y estado a través de Internet por cable (conector RJ45) e Internet móvil (dos tarjetas SIM);
  • gestión y configuración a través de una aplicación móvil, programa informático o sitio web;
  • integración con el servicio iVideon (grabación de clips de 10 segundos durante una alarma);
  • alimentado por tres fuentes independientes, batería incorporada.

Hay tres configuraciones de Security Hub:

  • solo el controlador para 4768 rublos;
  • "Kit de inicio" para 7691 rublos (controlador, sensores inalámbricos para abrir puertas / ventanas y movimiento);
  • "Kit con cámara de video" por 13,653 rublos ("kit de inicio" + cámara Ivideon Wi-Fi).


Para crear un sistema de alarma para el hogar económico, puede comprar un controlador y usarlo con sensores con cable baratos.


Por supuesto, para cualquier configuración que pueda comprar sensores inalámbricos, hay muchos de ellos.

Security Hub tiene muchos modos de funcionamiento y configuraciones sutiles, pero si percibe la alarma como un "kit listo para usar", todo es bastante simple. La alimentación se suministra desde una fuente de alimentación USB convencional a través del cable USB AB. La aplicación Security HUB se instala en el teléfono inteligente, se crea una cuenta.


Puede armar y desarmar la alarma usando la aplicación móvil. En caso de una alarma, aparecerá una notificación de inserción, una notificación de sonido (existe la oportunidad de configurar sus sonidos para cualquier evento) y una notificación de voz.

Las tarjetas SIM en Security Hub se usan solo para la transferencia de datos. El sistema de alarma no sabe cómo llamar y enviar SMS. Para aumentar la confiabilidad de la entrega de mensajes de alarma, puede configurar una de las salidas y conectar un marcador externo con su propia tarjeta SIM, por ejemplo, un rastreador GPS de automóvil barato.

Si la alarma debe activarse y eliminarse con frecuencia, los botones de la aplicación en el teléfono inteligente no son la opción más conveniente. Puede conectar el lector TM al Security Hub (150-200 rublos) y usar las tabletas-tecla "intercomunicador" "i-button" (30 rublos cada una) para armar y retirar de él.


También se ofrecen otros dispositivos adicionales para configurar / eliminar: un teclado con cable para ingresar un código (1411 rublos), un teclado inalámbrico (2065 rublos), un llavero (1062 rublos).


Es posible configurar la demora de los sensores (esto es necesario cuando el sensor de apertura está en la puerta delantera, y el armado / desarmado se lleva a cabo utilizando la tecla TM o el teclado interior). De manera predeterminada, el retraso es de 45 segundos para salir y de 60 segundos para ingresar, utilizando el programa de configuración opcional LocalConfig , puede cambiar el tiempo de retraso (20-30-45-60-90-120 segundos).

El "kit con una cámara de video" incluye un controlador Security HUb con una batería de 3.7V 600 mAh instalada, una fuente de alimentación USB 5V 1A, un cable de alimentación USB AM-BM de dos metros, cable de conexión UTP de 1 metro, 4 resistencias de 3.9 kΩ, sensor inalámbrico de puerta / ventana abierta Astra -3321 con batería LS14500, sensor de movimiento inalámbrico con termómetro Astra-5121 con batería CR123A, cámara Ivideon Cute Wi-Fi (Nobelic NBQ-1110F), fuente de alimentación USB 5V 1A, cable de alimentación USB-MicroUSB de 3 metros, guías de configuración de alarma rápida y cámaras, tacos y tornillos para montar sensores, instrucciones de inicio rápido, pasaporte.


La placa del controlador tiene una ranura para dos tarjetas SIM, un interruptor de sabotaje, un bloque de dos pines para conectar un lector TM, contactos para conectar dos fuentes de alimentación de 12 V, contactos para cuatro entradas / salidas programables y dos salidas.


La instalación de tarjetas SIM es opcional, pero si Ethernet no funciona, los datos se transmitirán a través de la primera tarjeta SIM, si no funciona y a través de la segunda. También es posible trabajar solo a través de Internet móvil sin una conexión por cable.

El sensor de movimiento Astra-5121 detecta el movimiento de objetos de más de 20 kg (insensibles a los animales) a una distancia de 2 a 10 metros en la zona de 90 ° en el plano horizontal. Funciona con baterías de hasta 4 años, se puede utilizar en el rango de temperatura de -10 a + 50 ° C. El sensor tiene un termómetro incorporado.



El sensor de apertura de puerta / ventana Astra-3321 funciona con baterías por hasta 7 años, se puede usar a temperaturas de -20 a + 50 ° C, hay contactos para conectar sensores cableados adicionales.


La cámara Wi-Fi Nobelic NBQ-1110F tiene una resolución de 1.3MP (1280x720 píxeles), admite el protocolo de compresión H.264, detección de movimiento, tiene iluminación IR de hasta 10 m para visión nocturna, está equipada con un micrófono incorporado, altavoz y ranura para tarjeta microSD. La cámara funciona con el servicio iVideon.


Para evaluar la calidad de la imagen, coloqué la cámara en el centro del extremo de la habitación a una altura de 2 metros. La distancia al armario es de 5 metros.

Disparos a la luz artificial. Haga clic en la foto para abrirla a tamaño completo.


Disparo en completa oscuridad con una cámara con iluminación IR.


A un ritmo gratuito, la cámara graba clips de diez segundos utilizando su detector de movimiento (puede verlos en la aplicación iVideon). Cuando se activa una alarma, la cámara también graba clips de diez segundos (se pueden ver en la aplicación Security Hub). Los rodillos se almacenan día.


La aplicación muestra información completa sobre sensores inalámbricos e historial de eventos.


Desafortunadamente, no todas las configuraciones de Security Hub son intuitivas. Entonces, para agregar la clave TM, debe hacer clic en el botón "Administración", allí "+", "identificador real", "habilitar el modo interactivo".


Luego, toque la tecla del lector, asigne un nombre a la tecla e indique las secciones que se colocarán / eliminarán con esta tecla.


No hay problemas al agregar sensores: simplemente vaya a "sensores", presione "+", seleccione el método de conexión del sensor (cableado o inalámbrico), especifique el número de entrada para el sensor cableado, seleccione el tipo de sensor y la sección en la que se agregará.


Con las salidas, las cosas son un poco más complicadas. Los primeros cuatro contactos pueden ser entradas o salidas. Quinto y sexto: solo salidas.

Lo más simple es agregar un relé controlado manualmente. Vamos al "relé / salidas", presionamos "+", "agregar relé", seleccionamos el método de conexión, damos un nombre, ingresamos el número de terminal del 1 al 6 para el relé cableado, seleccionamos "control manual".



En la sección "Relés / salidas", el relé creado aparecerá con los botones "encender" y "apagar". Tal relé se puede usar para controlar la luz, la calefacción o para encender manualmente una sirena potente.


Hay dos modos de salida especiales: "baliza" y "sirena". "Faro" es un indicador luminoso que muestra el estado de la alarma con luz y parpadeo (se enciende cuando la alarma está armada, parpadea lentamente durante el retraso de extracción / armado y parpadea rápidamente durante una alarma). La "Sirena" se enciende continuamente durante una alarma normal e intermitentemente durante un incendio.

El control de script le permite implementar varias lógicas de las salidas. En total, Security Hub tiene 23 scripts en cinco categorías. Enumere brevemente algunas de las características:

  • Encender el relé durante un intervalo determinado (en lugar de encenderlo / apagarlo, el relé tendrá un pulso de botón, que lo encenderá, por ejemplo, durante 5 segundos).
  • Encender el relé mediante cualquier sensor y apagarlo después del tiempo especificado (puede usarse para controlar la luz mediante sensores de movimiento);
  • La inclusión de plantas de riego en un sensor de humedad;
  • Control de puerta (aplicando un pulso al relé cuando se presiona el botón);
  • Gestión de grifos desde el sensor de fugas;
  • Gestión de salida programada (una vez al día o tres veces al día);
  • Control de relé según temperatura.

La configuración y administración de Security Hub es posible no solo a través de la aplicación móvil. Hay un sitio de administración cloud.security-hub.ru.


También hay un programa para la estación de monitoreo de PC Astra.


Grabé una breve reseña de video.


Security Hub tiene muchas características y tiene muchas ventajas:

  • soporte para sensores cableados e inalámbricos que funcionan a través de un protocolo seguro;
  • Soporte TM;
  • la disponibilidad de salidas con grandes capacidades de control;
  • Conexión a Internet a través de Internet por cable y móvil;
  • gestión y configuración a través de una aplicación móvil, programa informático o sitio web;
  • integración con iVideon.

El principal inconveniente de Security Hub es la falta de alertas a través de llamadas y SMS. Todo funciona solo a través de la nube. Está claro que el sistema se desarrolló, en primer lugar, para el control remoto, y allí no es necesario, pero dado que Security Hub se vende como un sistema de "autoprotección", se podrían agregar llamadas de alarma. Por supuesto, puede conectar un marcador externo con una tarjeta SIM separada a una de las salidas del controlador, pero estas son "muletas".

Un inconveniente menos significativo es la falta de la posibilidad de armarse usando el botón (y sería conveniente salir de la casa simplemente presionando el botón) y la lógica de operación TM: todas las teclas están armadas y desarmadas, de modo que una tecla está armada, y otra oportunidad, no puedes.

Y el Security Hub no tiene instrucciones detalladas. Tener un WIKIcon una descripción de las funciones y configuraciones, pero, por ejemplo, la lista de scripts con una explicación de lo que todos hacen no está allí (solo están en la aplicación móvil y debes hacer clic en todos los elementos seguidos para comprender lo que están haciendo).

Sin embargo, Security Hub impresiona con la cantidad de configuraciones y características. Me imagino cuánto tiempo y esfuerzo se ha dedicado a desarrollar un sistema de este tipo. Realmente espero que los desarrolladores agreguen alertas de llamadas y SMS en el nuevo firmware, así como también agreguen la capacidad de armar el sistema bloqueando la entrada TM y escribir instrucciones detalladas.
