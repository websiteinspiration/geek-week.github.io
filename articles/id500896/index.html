Kami melanjutkan pemilihan materi menarik kami ( bagian pertama , bagian kedua ). Kali ini, satu kursus pendidikan Technosphere, sebuah pembicaraan tentang pengujian unit dan satu acara talk show untuk Oh, kode saya.

Python Terapan, Technopark, musim gugur 2018


Technopark adalah salah satu proyek pendidikan bersama kami, dalam hal ini dengan MSTU. Bauman.

Kursus ini memperkenalkan bahasa Python - salah satu bahasa yang paling populer dan populer di pasar TI saat ini. Permintaan bahasa tidak lahir dari awal: kesederhanaan input dan sintaksis, pilihan alat terkaya untuk menyelesaikan semua jenis tugas - ini dan banyak lagi telah mengarah pada fakta bahwa Python banyak digunakan di seluruh dunia. Berkat kursus ini, Anda dapat bergabung dengan ekosistem bahasa.

  • Kuliah 1. Pendahuluan. Alat pengembangan, untuk mengenal Git. Pengantar singkat tentang prinsip, konsep, dan fitur Python. Sintaks dasar.
  • Kuliah 2. Struktur data . Struktur data dasar dalam Python. Koleksi modul.
  • Kuliah 3. Modul . Bagaimana kode diatur dalam Python. Apa itu modul, bagaimana mereka dibuat, bagaimana Python bekerja dengannya. OOP.
  • Kuliah 4. Input-output . Manajer konteks, bekerja dengan FS, bekerja dengan jaringan, dasar-dasar multithreading.
  • Kuliah 5. Metaprogramming . Metode ajaib, dekorator, metakacamata, deskriptor.
  • Kuliah 6. Basis data relasional . Perangkat, fitur, metode aplikasi.
  • Kuliah 7. Pemrograman asinkron . hasil dari, asyncio, aiohttp, asyncio-redis.
  • Kuliah 8. Pengembangan web tentang Django . Dasar-dasar HTTP, kerangka kerja web, keamanan informasi, penyebaran aplikasi web.
