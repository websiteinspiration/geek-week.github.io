Was ist HTML?


HTML steht für HyperText Markup Language. Dies ist die Auszeichnungssprache für Dokumente im World Wide Web (WWW). HTML ist eine standardisierte Sprache für formatierten Text. Der Browser interpretiert es und zeigt Elemente von Webseiten auf dem Bildschirm an.

Woraus besteht HTML-formatierter Text?


Zunächst einmal ist HTML Inhalt und Tags. Mit Tags können Sie festlegen, wie Inhalte auf Webseiten angezeigt werden.

Was ist ein HTML-Tag?


Ein Tag ist ein spezielles Servicewort in spitzen Klammern. Es wird auch als "HTML-Element" bezeichnet. Es gibt viele Tags in der HTML-Sprache und jeder macht etwas mit dem Inhalt, der sich normalerweise in den Klammern oder zwischen den Tags befindet.
Wenn das Tag gepaart ist, entspricht das <TEG> -Tag </ TEG>.

Sind alle Tags gepaart?


Überhaupt nicht. Es gibt einzelne Tags, z. B. <img> - für Bilder. In diesem Fall wird der Inhalt zusammen mit anderen Dienstwörtern (zum Beispiel für <img> kann es src = "Bild-URL" sein) zwischen die Klammern gesetzt, jedoch nach dem Wort img.

Was sind die Haupttypen von Listen, die zum Erstellen einer Webseite verwendet werden?


Beim Erstellen einer Webseite können Sie Folgendes verwenden:

  1. Menüliste (Menüliste - <Menü>).
  2. Verzeichnisliste (Verzeichnisliste - <Verzeichnis>).
  3. Geordnete Liste (nummerierte Liste - <ol>).
  4. Ungeordnete Liste (Liste mit Aufzählungszeichen - <ul>).
  5. Definitionsliste (<dl>).


Warum Listen in HTML verwenden?


Listen machen Informationen visueller und leichter lesbar. Darüber hinaus können Sie innerhalb der Listen (genauer gesagt für die willkürliche Anzeige von Listenelementen) auch verschiedene Tags verwenden.

Können wir Kommentare in HTML schreiben? Wie?


Ja wir können. Dazu muss der Kommentartext in <! - und -> platziert werden.

Warum werden einige Zeichen auf einer Webseite manchmal falsch angezeigt?


In den meisten Fällen werden Zeichen aufgrund von Codierungsproblemen falsch angezeigt. Das ist der Grund - in den Einstellungen des Browsers oder Webservers.

Was ist Image Map?


In HTML ist Image Map eine Technologie, mit der Sie Links zu Bildfragmenten verknüpfen können. Durch Klicken auf einzelne Teile des Bildes kann der Benutzer diesem oder jenem Link zu verschiedenen Webseiten folgen.

Was ist Leerraum?


Leerzeichen ist eine Eigenschaft, mit der Sie die Anzahl der Leerzeichen zwischen Wörtern beim Anzeigen einer Webseite ändern können. Standardmäßig interpretiert der Browser eine beliebige Anzahl von Leerzeichen im HTML-Code als ein Leerzeichen.

Was ist der Hauptvorteil von Leerraum?


Beim Schreiben von HTML-Code muss sich der Entwickler nicht um die zusätzlichen Leerzeichen kümmern, die er eingeben kann: Der Browser entfernt alle zusätzlichen Leerzeichen, wenn die Webseite angezeigt wird.
Der Entwickler kann dem Code nicht nur versehentlich, sondern auch absichtlich Leerzeichen hinzufügen, um den Code besser lesbar zu machen.

Kann ich nicht allen Tag-Attributen Werte zuweisen?


Ja, in diesem Fall verwendet der Browser die Standardwerte. Wenn Sie mit der Art und Weise, wie er die Webseite anzeigt, nicht vertraut sind, sollten Sie möglicherweise die Werte einiger Attribute explizit angeben.

Wie platziere ich das Copyright-Zeichen auf einer Webseite, weil es sich nicht auf der Tastatur befindet?


Das Copyright-Symbol bezieht sich auf Sonderzeichen. Um es auf einer Webseite zu platzieren, müssen Sie die Sequenz & copy oder & # 169 auf der Tastatur in das entsprechende Fragment des HTML-Codes eingeben.

Wie erstelle ich Links zu verschiedenen Fragmenten (Abschnitten) derselben Webseite?


Suchen Sie dazu den Anfang des benötigten Fragments im HTML-Code. Erstellen Sie an dieser Stelle den sogenannten "Anker" mit dem Namensattribut des <a> -Tags und geben Sie den Link zum Fragmentnamen mit dem href-Attribut und dem # -Zeichen an:
href = "#link_name_to_fragment"

Können wir Listenelemente in einer HTML-Datei ausrichten?


Ja, wir können Listenelemente mithilfe von Einrückungen direkt in der HTML-Datei speichern. Einrückung kann auch für jede verschachtelte Liste innerhalb des übergeordneten Elements verwendet werden.

Welche Webseitenadresse wird normalerweise als Site-Adresse betrachtet?


Dies ist normalerweise eine der Seiten der obersten Ebene, die Hauptseite. Von dort aus geht der Benutzer zu Seiten, die sich auf niedrigeren Ebenen befinden. Und ihre Adresse wird meistens anhand der Adresse der Hauptseite gebildet.

Warum verwenden wir alternativen Text (alt-Attribut des <img> -Tags) für Bilder?


Alternativer Text wird angezeigt, wenn Grafiken im Browser des Benutzers deaktiviert sind oder wenn beim Laden eines bestimmten Bilds ein Problem auftritt. Dieser Text macht deutlich, was auf diesem Bild hätte dargestellt werden sollen.

Können HTML-Dateien in einem seltenen oder hochmodernen Browser gut funktionieren?


Natürlich können und sollten HTML-Dateien in jedem Browser gut funktionieren, da der Browser den HTML-Standards entsprechen muss. Wenn die Entwickler beschlossen, einige der Funktionen des Standards nicht zu unterstützen, bleibt dies auf ihrem Gewissen.

Denken Sie, dass der Hyperlink nur Text sein kann?


Nein, ein Hyperlink kann nicht nur Text sein, sondern beispielsweise auch Grafik. Dies bedeutet, dass wir das Bild in einen Link konvertieren können, der den Benutzer auf eine andere Seite verweist. Ein Hyperlink hängt nicht von der Art des Inhalts ab, der „darunter“ liegt.

Was sind die Attribute des <li> -Tags - eines Listenelements?


Es gibt zwei davon:

  1. Typ legt das Erscheinungsbild eines Aufzählungszeichens oder einer Aufzählungslistenmarkierung fest.
  2. value - Die Nummer, ab der die nummerierte Liste beginnt.


Wofür sind Stylesheets (CSS)?


Mit Stylesheets können Sie klare Regeln festlegen, die das Erscheinungsbild von Inhalten, verschiedenen HTML-Elementen und der gesamten Seite bestimmen. Darüber hinaus können dieselben Regeln auf mehrere Webseiten gleichzeitig angewendet werden, was die Aufbewahrung vereinfacht und gegebenenfalls den einheitlichen Stil der Website ändert.

Welche Arten der Nummerierung in Listen kennen Sie?


Die folgenden Werte können als Nummerierungselemente dienen:

  • Arabische Zahlen (1, 2, 3, ...);
  • lateinische Großbuchstaben (A, B, C, ...);
  • lateinische Kleinbuchstaben (a, b, c, ...);
  • Römische Großbuchstaben (I, II, III, ...);
  • römische Kleinbuchstaben (i, ii, iii, ...).


Wie stelle ich verschiedene Farben für Textfragmente einer Webseite ein?


Um unterschiedliche Farben für Textfragmente festzulegen, müssen Sie jedes der Fragmente in das gekoppelte Schriftart-Tag einfügen:

<font color=“color”>-    “color”</font>


Wo sind numerische Zeichencodes in HTML gespeichert?


In der ASCII-Tabelle. Wenn eine Webseite angezeigt wird, nimmt der Browser die numerischen Zeichencodes aus dem HTML-Dokument und ersetzt sie durch das entsprechende Zeichen aus der Tabelle.

Was sind die Vorteile der Gruppierung mehrerer Kontrollkästchen (Kontrollkästchenelemente)?


Das Gruppieren von Flags bietet mehrere Vorteile:

  • es hilft, sie besser zu organisieren und zu strukturieren;
  • Auf diese Weise können Sie mit id auf bestimmte Flags aus der Gruppe zugreifen.
  • Auf diese Weise können Sie eine weitere Gruppe von Flags auf derselben Webseite erstellen, ohne befürchten zu müssen, die Namen oder Kennungen der Flags zu verwechseln.


Wie wirkt sich die Überlappung von Tags auf die Anzeige von Inhalten aus?


Überlappende Tags in HTML erkennen nur das erste Tag. Solche Probleme treten jedoch nur auf, wenn der Browser versucht, Tags ohne Text zu erkennen.

Was ist das Ergebnis, wenn zwischen den Tags kein Text steht? Gib ein Beispiel?


Wenn sich zwischen den Tags kein Text befindet, kann der Inhalt nicht formatiert werden. Beispielsweise erfordern einzelne Tags wie <img> keine Texteingabe. Daher ist in diesem Fall keine Formatierung erforderlich.

Wie spezifiziere ich Farben für Tabellenränder?


Dies kann im Stylesheet erfolgen. Wenn es fehlt, wird die Textfarbe verwendet.

Können wir einen Link erstellen, der zu einer anderen Webseite führt?


Ja, dafür müssen Sie das href-Attribut verwenden.

Können Stylesheets dabei helfen, das Bild auszurichten und zu bestimmen, wie Text relativ dazu positioniert ist?


Ja, ein Stylesheet kann Regeln zum Positionieren von Text und Bildern enthalten.

Kann ein einzelner Hyperlink zu unterschiedlichen Seiten führen?


Nein, ein Hyperlink kann nur zu einer Webseite führen, deren Adresse Sie angeben.

Gibt es einen Unterschied zwischen einer Liste mit Aufzählungszeichen und einer Liste von Verzeichnissen und Menüs?


Ja, der Unterschied besteht darin, dass eine ungeordnete Liste Attribute zum Ändern des Markierungsstils enthält.

Wie ändere ich die Markerfarbe?


Die Farbe des Markers entspricht normalerweise der Farbe des ersten Zeichens in der Liste. Wenn Sie die Farbe des ersten Zeichens mit dem <font> -Tag ändern, ändert sich auch die Farbe des Markers.
Wenn Sie die Farbe des Markers in der Liste festlegen möchten, ohne die Farbe des Texts zu ändern, müssen Sie das <span> -Tag in das <li> und den Text darin einfügen. Und dann wird die Farbe des Markers durch die Farbe der Stileigenschaft des LI-Selektors und die Farbe des Textes durch die Farbeigenschaft des SPAN-Selektors bestimmt.

Ist es möglich, einige Einschränkungen für die Größe von Textfeldern in HTML einzuführen?


Die Standardgröße beträgt 20-25 Zeichen. Mit den Attributen size oder maxsize können Sie jedoch andere Einschränkungen festlegen. Zum Beispiel:

<input type="text" size="5" />


Was ist die maximale Größe eines Textfelds?


Die Breite des Browserfensters.

Was passiert, wenn Sie die Größe des Textfelds auf Null setzen?


Der Browser verwendet den Standardwert.

Was sind die Ähnlichkeiten zwischen Grenz- und Regelattributen?


Wenn das Rahmenattribut auf einen Wert ungleich Null festgelegt ist, wird der Standardrand einer Zelle mit einer Dicke von einem Pixel automatisch hinzugefügt. Wenn das Attribut rules zum Tag <table> hinzugefügt wird, wird das Attribut border nicht berücksichtigt.
Gleichzeitig wird um jede Zelle (wie der Rand) standardmäßig ein Rahmen mit einer Dicke von einem Pixel angezeigt, wodurch ein Raster gebildet wird.

Was ist ein Festzelt? Wie können wir das anwenden?


Das Auswahlrechteck hilft beim Anpassen des Bildlaufs von Text auf einer Webseite. Um das Scrollen zu aktivieren, müssen Sie das <marquee> -Tag verwenden.

Was macht das <br> -Tag?


Die Funktion des <br> -Tags besteht darin, Textteile voneinander zu trennen.

Gibt es eine andere Möglichkeit, Text ohne Verwendung von <br> zu teilen?


Ja, es gibt andere Möglichkeiten, Text zu teilen. Sie können das <p> -Tag oder das <blockquote> -Tag verwenden.

Kann Text außerhalb des Browserfensters angezeigt werden?


Standardmäßig kann Text nicht außerhalb des Browsers angezeigt werden. Wenn der Text jedoch Teil einer Tabellenzelle mit einer vorgegebenen Breite ist, kann er über das Browserfenster hinausgehen.

Was ist der Unterschied zwischen aktiven und inaktiven Links?


Der aktive Link hat den Fokus, dh die Maus ist darüber. In anderen Fällen liegt der Fokus woanders.

Hat das Stylesheet Einschränkungen hinsichtlich der Anzahl der Regeln?


Nein, Stylesheets beschränken die Anzahl der Regeln für einen Selektor nicht. Jede neue Regel muss jedoch durch Semikolons getrennt werden.

Haben Stylesheets eine Hierarchie von Regeln?


Ja, es gibt eine Hierarchie mit vier Ebenen (von der höchsten zur niedrigsten Priorität):

  1. Inline-Stil: Wird direkt an das Element angehängt, für das der Stil erstellt wurde. Beispiel:
    <h1 style="color: #ffffff;">
  2. Ein Bezeichner ist ein eindeutiger Bezeichner für Seitenelemente, z. B. #navbar.
  3. Klassen, Attribute und Pseudoklassen: Diese Kategorien umfassen .class [Attribut] und Pseudoklassen wie: Hover ,: Focus usw.
  4. Elemente und Pseudoelemente: Diese Kategorie umfasst Elemente und Pseudoelemente wie h1, div,: before und: after.


Können wir verschiedene Selektoren mit unterschiedlichen Klassennamen gruppieren?


Ja, dafür müssen Sie nur die Klassennamen mit einem Komma schreiben.

Kann ich eine externe CSS-Datei in einem Browser anhängen?


Nein, eine externe CSS-Datei kann mithilfe des <link /> -Tags im HTML-Dokument selbst eingefügt werden.

Kann eine Eigenschaft vom Typ Listenstil einen Absatz beeinflussen?


Nein, die Eigenschaft vom Typ "Listenstil" kann einen Absatz nicht beeinflussen. Sie wird ihn höchstwahrscheinlich ignorieren.

Was ist ein Canvas-Element?


Mit dem HTML5-Canvas-Element können Bilder, Verläufe und komplexe Animationen eingefügt werden. Es kann auch helfen, 2D-Bilder direkt im Quellcode eines HTML-Dokuments zu erstellen.
