.NET Core Windows Bentuk Pembaruan Desainer

Kami merilis versi awal Visual Studio 16.6 - Visual Studio 2019 versi 16.6 Pratinjau 1 dan dengan itu versi baru konstruktor .NET Core Windows Forms.



Rilis ini termasuk


  • Dukungan untuk kontrol berikut:

    Untuk mengungkap
    • FlowLayoutPanel,
    • GroupBox
    • ImageList
    • MenuStrip ( PropertyBrowser ),
    • Panel,
    • SplitContainer,
    • Splitter,
    • TabControl,
    • TableLayoutPanel,
    • ToolStrip ( PropertyBrowser, designer actions).

  • Sumber daya lokal dan formulir terlokalisasi dimasukkan dalam konstruktor.
  • Dukungan untuk pengaturan LayoutMode dan ShowGrid / SnapToGrid melalui Alat -> Opsi.
  • Peningkatan keandalan dan kinerja.
  • Perbaikan kecil dan perubahan kecil lainnya.

Lebih lanjut


Dalam rilis mendatang, kami akan bekerja untuk mendukung Kontrol Pengguna dan kontrol pihak ketiga, berintegrasi dengan penyedia kontrol populer, mendukung Kontrol Data dan skenario terkait, peningkatan kinerja, dan fitur lainnya.

Cara menggunakan konstruktor


  • Anda perlu menggunakan umpan Pratinjau Visual Studio
  • Anda harus mengaktifkan perancang di Visual Studio. Pergi ke Alat → Pilihan → Lingkungan → Fungsi Pratinjau dan pilih opsi “Gunakan Desainer Formulir Windows untuk .NET Core Applications”.



Cara melaporkan masalah


Tanggapan Anda sangat penting. Silakan laporkan masalah dan kirimkan permintaan melalui saluran umpan balik Visual Studio. Gunakan ikon Kirim Umpan Balik di sudut kanan atas Visual Studio, seperti yang ditunjukkan di bawah ini, dan tunjukkan bahwa itu terkait dengan area WinForms .NET Core.
