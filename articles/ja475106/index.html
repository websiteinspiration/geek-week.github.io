Microsoft Ignite 2019カンファレンスの主な発表



 マイクロソフト は最大の年次会議  Ignite 2019で、ビジネス開発および組織と従業員のより生産的な仕事のための幅広い新しいツールとサービスを紹介しました。この投稿では、主な発表を集めました。猫の下を見てください。

ちなみに、会議のすべてのレポートを含む興味深いテーブルがあります。こちらで学んでください。

紺碧の弧




統合ハイブリッド環境を作成するためのソリューションの暫定バージョンの立ち上げ。Azure Arcは、AmazonやGoogleなどのマルチベンダークラウド、オンプレミスサーバー、エッジコンピューティングデバイスなどのインフラストラクチャ全体でAzureの統合サービス、管理ツール、セキュリティツールを活用する機能を提供します。

新しいMicrosoft Edge




WindowsおよびmacOS向けのChromiumエンジン上のMicrosoft Edgeブラウザーは、2020年1月15日に90を超える言語で公開される予定です。現在  、試用版のダウンロードが可能です。詳細はこちらをご覧ください。

Officeユニバーサルモバイルアプリケーション




用途の広いOfficeモバイルアプリケーションは、Word、Excel、およびPowerPointを組み合わせたもので、個別のアプリケーションをインストールする必要がありません。これを使用すると、写真をテキストまたは表に変換する機能を使用するなど、ドキュメントを作成および編集できます。このアプリケーションは、現在Android  および  iOSユーザー向けの公開評価版として入手できます  。

プロジェクトシリカ




マイクロソフトとワーナーブラザーズ。1978年のスーパーマン映画を石英ガラスに記録した。両社は、Microsoft Research Project Silicaプロジェクトのコンセプトの最初の確認を発表しました  。レーザー光学と人工知能の最新の進歩を利用してデータを保存します。

Azureブロックチェーントークン




このソリューションでは、物理資産またはデジタル資産をトークン化し、それらを管理して、さまざまなタイプのブロックチェーンプラットフォームで使用できます。Azure Blockchain Tokensを使用すると、ブロックチェーン開発者は、Token Taxonomy Initiative（TTI）標準にアクセスする   か、テンプレートを使用して最も人気のある4つのタイプのトークンを作成できます。このサービスは、トークンと対話するための一連のオープンAPIへのアクセスを提供します。 ビジネスでこのようなトークンを使用するシナリオに  は、サプライチェーンのデジタル化、ロイヤルティプログラム、ワークフローなどがあります。現時点では、このソリューションは暫定バージョンで利用できます。

プロジェクト皮質


Microsoft Teamsの発売以来、最初の新しいMicrosoft 365サービス。このAIツールは、企業のコンテンツ分類とデジタルアシスタントのクロスツールです。インテリジェントサービスは、トピックに関する情報（Office、Outlook、Microsoft Teamsアプリケーションなど）を自動的に見つけて配布し、さまざまなトピックの専門家を特定して、組織の「知識ネットワーク」を作成できます。このソリューションは、2020年前半、クローズドテスト中の時点で公開されます。

Microsoft 365のインサイダーリスク管理




組織向けのインテリジェントなセキュリティソリューション。機械学習アルゴリズムを使用して、非定型的で疑わしいユーザーの行動（ファイルに対するアクションなど）のケースを匿名で分析および検出できます。これにより、標準のツールや方法では検出できない多くのリスクを防ぐことができます。ソリューションはクローズドテストモードで利用できます。

UIフロー




Power Automate（旧称Microsoft Flow） のプロセス自動化サービスのパブリックプレビュー。新しいソリューションは、APIを介した統合をサポートしていないものも含め、あらゆるアプリケーションでの自動化のためのインターフェースロボットを作成します。したがって、UIフローは、それを構成するためのプログラミング知識を必要としないため、組織内のプロセスの自動化を大幅に簡略化できます。ユーザーは、自動化したいアクションの実行を示すだけで済みます。

ストリーム音声録音の改善


Microsoft Stream ビデオストリーミングサービスは、機械学習アルゴリズムを使用して、マウスをクリックするだけで、ビデオ内の不要なバックグラウンドノイズを検出して排除します。

Outlookのチームチャットボタン




Outlookでの会話とチームでのチャットを切り替えることで、便利に対応できます。

Azure Synapse Analytics




Azure SQL Data Warehouseの機能を拡張する新しいサービス。Azure Synapse Analyticsは、すべてのデータソース、ストレージ、およびビッグデータ分析システムからの情報を集約します。これにより、企業はデータをより速く、より安全に使用できるようになります。
