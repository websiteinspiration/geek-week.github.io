The best Microsoft experts will talk about the main innovations in SQL Server 2019: SQL Server Big Data Clusters technologies for working with big data and machine learning, Polybase technologies for accessing data in external sources without copying them, container support, working on OS Linux and many others New to MS SQL Server 2019!

A separate report will be devoted to the platform for business intelligence Microsoft Power BI!

We invite you to visit the presentation of the new version of the hybrid data management platform SQL Server 2019! Register , but the details are under the cut.

Did you know that we have released a new version of SQL Server - SQL Server 2019? Thanks to many new useful features, this release aroused unprecedented interest, and the speed of translation of information systems by organizations exceeds the performance of all previous versions. With the transition to SQL Server 2019, companies are gradually abandoning their costly installations of Hadoop, mastering big data clusters. Ways to access data are also changing: modern data visualization capabilities allow you to design new projects without requiring expensive data movement and preliminary copying of data. Existing workloads are accelerated by a host of new intelligent database functions.

It may not be easy for you to keep track of all emerging trends. That is why we are pleased to invite you to visit our event on March 3 at 09:30 at Spartakovsky Lane 21, entrance 7, to understand the reasons for the interest in so many companies in SQL Server 2019. Microsoft speakers from different countries and our partners will share their impressions. We are sure that you will determine for yourself all the potential advantages of this new platform. We hope to meet with you and discuss the possibilities of reducing costs, accelerating work on projects and comfortable interaction with users.
