Habrastatistics：サイトの最も訪問されたセクションと最も訪問されていないセクションの探索

こんにちは、ハブル。

で前の部分の記事、自分の意見や評価の数- 、Habrの出席が主なパラメータで分析しました。しかし、サイトのセクションの人気の問題は考慮されていません。これをより詳細に検討し、最も人気があり、最も人気のないハブを見つけることが面白くなりました。最後に、「geektimes効果」を詳細に検討します。最終的に、読者は新しい評価に関する最高の記事の新しいセレクションを受け取ります。



何が起こったかを気にする人は、カットの下で続けました。

統計と評価は公式ではないことを改めてお伝えします。内部情報はありません。また、私がどこかに間違えられたり、見逃したりしていないことも保証されていません。しかし、それでも興味深いことがわかりました。最初にコードから始めますが、これは関係ありません。最初のセクションはスキップできます。

データ収集


パーサーの最初のバージョンでは、ビューの数、コメント、および記事の評価のみが考慮されました。これはすでに適切ですが、より複雑なクエリを作成することはできません。それでは、サイトのテーマ別セクションを分析します。これにより、たとえば「C ++」セクションの人気が数年にわたってどのように変化したかを確認するなど、非常に興味深い研究を行うことができます。

サイトの主なテーマ別ハブのリストを取得します。

def get_as_str(link: str) -> Str:
    try:
        r = requests.get(link)
        return Str(r.text)
    except Exception as e:
        return Str("")

def get_hubs():
    hubs = []
    for p in range(1, 12):
        page_html = get_as_str("https://habr.com/ru/hubs/page%d/" % p)
        # page_html = get_as_str("https://habr.com/ru/hubs/geektimes/page%d/" % p)  # Geektimes
        # page_html = get_as_str("https://habr.com/ru/hubs/develop/page%d/" % p)  # Develop
        # page_html = get_as_str("https://habr.com/ru/hubs/admin/page%d" % p)  # Admin
        for hub in page_html.split("media-obj media-obj_hub"):
            info = Str(hub).find_between('"https://habr.com/ru/hub', 'list-snippet__tags') 
            if "*</span>" in info:
                hub_name = info.find_between('/', '/"')
                if len(hub_name) > 0 and len(hub_name) < 32:
                    hubs.append(hub_name)
    print(hubs)

find_between関数とStrクラスは、2つのタグの間の行を強調表示します。以前に使用しました。テーマ別ハブには「*」のマークが付いているため、簡単に強調表示できます。対応する行のコメントを外して、他のカテゴリのセクションを取得することもできます。

処理


分析を開始する時が来ました。データセットを読み込み、ハブのデータを処理します。

def to_list(s: str) -> List[str]:
    # "user:popular_science+astronomy" => [popular_science, astronomy]
    return s.split(':')[1].split('+')

def to_date(dt: datetime) -> datetime.date:
    return dt.date()

df = pd.read_csv("habr_2019.csv", sep=',', encoding='utf-8', error_bad_lines=True, quotechar='"', comment='#')
dates = pd.to_datetime(df['datetime'], format='%Y-%m-%dT%H:%MZ')
dates += datetime.timedelta(hours=3)
df['date'] = dates.map(to_date, na_action=None)
hubs = df["hubs"].map(to_list, na_action=None)
df['hubs'] = hubs
df['is_profile'] = hubs.map(is_profile, na_action=None)
df['is_geektimes'] = hubs.map(is_geektimes, na_action=None)
df['is_geektimes_only'] = hubs.map(is_geektimes_only, na_action=None)
df['is_admin'] = hubs.map(is_admin, na_action=None)
df['is_develop'] = hubs.map(is_develop, na_action=None)
