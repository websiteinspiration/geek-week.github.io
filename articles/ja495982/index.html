信号レベルメーターを作成する


前回の記事では、メディアストリーマーを使用したプログラムの正しい終了について説明しました。


この記事では、信号レベルメーターの回路を組み立て、フィルターから測定結果を読み取る方法を学びます。測定の精度を推定します。


メディアストリーマーによって提供される一連のフィルターには、MS_VOLUMEというフィルターがあります。MS_VOLUMEは、フィルターを通過する信号のrmsレベルを測定し、信号を減衰させ、多くの有用で予期しない機能を実行します。このフィルターについては、後で記事全体を取り上げます。しかし、今度はそれをメーターとして使用します。


信号源としてトーンジェネレータを使用します。この信号は、MS_VOLUMEフィルターに送信され、その出力にはサウンドカードが接続されています。


この例では、少し異なるモードでジェネレーターフィルターを使用します。これにより、シングルトーン信号が生成されます。1つの正弦波振動のみを含む信号。

#include <mediastreamer2/msfilter.h>
#include <mediastreamer2/msticker.h>
#include <mediastreamer2/dtmfgen.h>
#include <mediastreamer2/mssndcard.h>
#include <mediastreamer2/msvolume.h>

int main()
{
    ms_init();
    /*   . */
    MSFilter  *voidsource=ms_filter_new(MS_VOID_SOURCE_ID);
    MSFilter  *dtmfgen=ms_filter_new(MS_DTMF_GEN_ID);
    MSFilter  *volume=ms_filter_new(MS_VOLUME_ID);
    MSSndCard *card_playback=ms_snd_card_manager_get_default_card(ms_snd_card_manager_get());
    MSFilter  *snd_card_write=ms_snd_card_create_writer(card_playback);

    /*  . */
    MSTicker *ticker=ms_ticker_new();

    /*    . */
    ms_filter_link(voidsource, 0, dtmfgen, 0);
    ms_filter_link(dtmfgen, 0, volume, 0);
    ms_filter_link(volume, 0, snd_card_write, 0);

    /*   . */
    ms_ticker_attach(ticker,voidsource);

    MSDtmfGenCustomTone dtmf_cfg;

   /*    ,   ,     
       ,    . */
