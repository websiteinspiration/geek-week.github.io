グランドストリームSIP電話の選び方-一般的に、特に？



私はGrandstreamディストリビューターであるFactor Groupでエンジニアとして働いており、さまざまなプロジェクトの販売前のコンサルティングに参加しています。クライアントは先進企業だけでなく初心者でもあります。したがって、Grandstreamデバイスのラインについてだけでなく、SIPテレフォニーのいくつかの基本原則についてもよく話します。そして、私の相談とサポートの結果がこの記事でした。一般的な情報と興味深い詳細の両方を伝えようとしました。情報がおもしろいことを願っています-これは私の記事を書いた最初の経験です。

したがって、SIPテレフォニーに切り替えるか、障害が発生したデバイスや古いデバイスを新しいデバイスに交換する場合は、IP電話が必ず必要になります。この記事は、Grandstream Networksの電話回線の全範囲を理解し、自分や従業員に最適なデバイスを選択するのに役立つように書かれています。

Grandstreamには、ベーシック、ミッドレンジ、ハイエンド、プレミアム、DECT、Wi-Fiの6種類のIP電話があります。この記事では、ベーシック、ミッドレンジ、ハイエンドの3つのIP電話のカテゴリを分析します。価格帯、外観、そしてもちろん機能性が異なります。たとえば、ハイエンドカテゴリの電話は、最高の機能を備え、見栄えが良いだけでなく、対応する価格帯もあります。ただし、順番に各カテゴリを個別に検討します。

基本的なIP電話


図1.基本的なIP電話

GXP1600シリーズのIP電話は、ホームユーザーだけでなく、中小企業向けに特別に設計されています。これらのシンプルなデバイスは、使いやすいインターフェイス、IPテレフォニーのすべての基本機能のサポート、および最大限の使いやすさを備えています。

このカテゴリの電話は、GXP1610、GXP1615、GXP1620、GXP1625、GXP1628、GXP1630の6つのモデルで構成されています。
図2. GXP1610 / GXP1615

GXP1610とGXP1615の電話機は、電源に加えてネットワークケーブルからデバイスに電力を供給できるPoE（Power-over-Ethernet）テクノロジをサポートしている点でのみ異なります。私たちが検討しているカテゴリの他のデバイスと比較すると、これらの電話だけがバックライトを備えておらず、HDオーディオをサポートしていません。

これらのモデルは、1つのSIPアカウントと2つの回線をサポートしているため、1つの内部番号を割り当てて2つの通話を同時に処理できます。また、3方向の音声会議をサポートし、デュアル10 / 100Mbpsネットワークインターフェイス（LANおよびPC）を備えています。 PCコネクタを使用すると、従業員の職場でローカルネットワークに接続されているコンピュータが1台だけの場合に非常に便利です。コンピュータをネットワークケーブルでPC電話ポートに接続すると、電話はスイッチとして機能し、メインスイッチの追加ポートは必要ありません。ベーシック、ミッドレンジ、ハイエンド、プレミアムのカテゴリのすべての電話がこのモードをサポートしています。

すべてのGrandstream卓上電話で使用できるもう1つの機能は、RJ9ヘッドセットポートです。ただし、少額の予約が必要です。製造元は、さまざまな種類のヘッドセットをテストしています。彼らのサポートは保証されており、16番目のシリーズの電話についてはこのリストを、ここでは他のすべての電話についてこのリストを調べることができます。
図3. GXP1620 / GXP1625

GXP1620とGXP1625の電話機も、GXP1625がPoEテクノロジーをサポートしている点のみが異なります。そして、私はすべてのラインの電話のうち、GXP1610とGXP1620の2つのモデルのみがこのテクノロジーをサポートしていないことに注意してください。

GXP1620およびGXP1625モデルは、2つのSIPアカウントと2つの回線をサポートし、バックライト、HDオーディオ、3方向オーディオ会議、およびデュアル10 / 100Mbpsネットワークインターフェイスをサポートしています。
図4. GXP1628 GXP1628

モデルは、同僚の忙しい電話のステータスを監視できるBLF機能を備えた8つのMPKキー（多目的キー）を除いて、GXP1625モデルに似ています。マルチタスクMPKキーを使用すると、最も一般的なワンタッチアクションを実行できます。原則として、これらのキーには、最も頻繁にダイヤルされる加入者へのクイックコールまたはコール転送が割り当てられます。これにより、電話番号をメモリに保存する必要がなくなり、電話帳でキーを検索する時間が無駄になりません。ショートカットキーの存在は、特に多くの電話をかける従業員にとって、IP電話での日常の作業を非常に容易にします。また、このモデルは、ギガビットネットワークポートの存在によってGXP1625と異なります。
図5. GXP1630

GXP1630はGXP1628に似ていますが、独自の違いもあります。GXP1630は3つのSIPアカウント、3つの回線をサポートし、4方向の音声会議を開催できます。

ミッドレンジIP電話


図6.ミッドレンジIP電話

GXP1700シリーズIP電話は、中小企業向けに設計されています。これらのスタイリッシュなデバイスは、成長する組織で必要とされるすべての必要な機能を備えています。同時に、GXP1700シリーズデバイスはリーズナブルな価格で、企業の予算への負担を軽減します。機能性と価格に加えて、電話は優れたデザインと人間工学に基づいた配慮が施されているため、快適に作業できます。

このカテゴリの電話は、GXP1760、GXP1760w、GXP1780、およびGXP1782の4つの電話で構成されています。 GXP1760w電話機にはデュアルバンドWi-Fiモジュールが組み込まれているため、Wi-Fi電話機に関連することをすぐに説明します。詳細については、別の記事で説明します。
図7. GXP1760 GXP1760から始めましょう

。このモデルには、バックライト付きの2色のLCD画面、3つのSIPアカウント、6つの回線のサポート、および5方向の音声会議を開催する機能があります。ディスプレイの側面にある6つのキー、およびホーム画面の4つのページをナビゲーションキー「右」と「左」で切り替える機能により、合計24の仮想BLFキーが提供されます。
図8. BLFキー

パーソナライズオプションの場合、GXP1760はさまざまな着信音だけでなく、天気予報などのWebアプリケーションもサポートします。GXP1760には、ケンジントンセキュリティスロットも装備されており、通路、ホテルのホール、受付などの公共の場所から盗まれるのを防ぎます。2つの10/100 Mbit / sポートを使用してネットワークに接続します。デバイスは、Webインターフェースを介して、およびさまざまな自動チューニングテクノロジーを使用して構成できます。

  • GrandstreamからIP-PBXを使用する場合の自動プロビジョニング。
  • DHCPオプション66を介した.xml構成ファイルの自動読み込み。
  • 自動チューニングサービス3CXおよびBroadsoftのサポート。
  • Grandstream Automated Provisioning System（GAPS）は、IT管理者と通信事業者がデバイスのプロビジョニングを直接かつ安全に制御し、中央制御パネルからGrandstream製品の大規模な展開を最適化できる自動システムです。

図9. GXP1780 / 1782

GXP1780は、4つのSIPアカウントと8つの回線をサポートするという点でGXP1760とは異なります。このモデルには8つの物理BLFキーがあり、ホーム画面を裏返す機能により、32の仮想BLFキーが提供されます。GXP1782は、ギガビットネットワークインターフェースのみがGXP1780と異なります。

ヘッドセットを接続するためのRJ9ポートに加えて、1700シリーズの電話には、外部EHSアダプター用の別のEHSポートがあります。

ハイエンドIP電話


図11.ハイエンドIP電話

高品質のGXP2100シリーズIP電話は、高度な電話機能を必要とするユーザー向けに設計されています。これらのデバイスには豊富な機能があります。高品質HDスピーカー、クリアカラーディスプレイ、デスクトップの壁紙をカスタマイズする機能、さまざまなBLFキーの組み合わせ、追加の拡張パネルのサポート、すべてのモデルの組み込みBluetooth、PoE、ギガビットネットワークポート-これらはすべて、シリーズの高度なIP電話の際立った特徴ですGXP2100。

このカテゴリには、GXP2130、GXP2135、GXP2140、GXP2160、GXP2170の5つの電話モデルが含まれます。
図12. GXP2130

このラインの最初のモデルはGXP2130です。この電話には、320 x 240ピクセルの解像度の2.8インチカラーLCDが搭載されています。 3つの回線と3つのSIPアカウントをサポートし、4方向の音声会議を手配する機能を提供します。この電話の特徴は、BLF機能付きの8つのMPKキーの存在です。ただし、ミッドレンジ電話と同様に、ハイエンドデバイスにはBLF仮想キーを設定する機能もあります。ディスプレイの左側にある3つの物理キーとスクロール用の4ページの存在により、8つのMPKキーに加えて、合計12の仮想BLFキーが提供されます。

図13. BLFキー

このシリーズの電話でBluetoothが必要な理由を個別に説明します。 Bluetoothテクノロジーのサポートは、21XXシリーズのすべてのハイエンド電話に存在し、電話の機能を大幅に拡張します。まず、外部のBluetoothヘッドセットを接続すると、有線のヘッドセットを使用する場合のように、電話に「バインド」する必要がなくなります。また、携帯電話をIP電話に接続することも可能です。

図14.携帯電話アカウント

このスキームでは、IP電話自体が携帯電話のヘッドセットとして機能します。接続すると携帯電話の「アカウント」が自動的にIP電話に表示され、携帯電話を使わなくてもIP電話から電話をかけることができる、いわゆるハンズフリーモードです。携帯電話への着信もIP電話に表示されます。また、携帯電話に既に存在する連絡先に電話をかける必要がある場合は、Bluetoothを介してIP電話と携帯電話の間で連絡先を同期する機能を利用できます。
図15. GXP2135

次のモデルはGXP2135です。これは、回線数、SIPアカウント、多数のBLFキーの存在、およびMPKキーの欠如が以前のものと異なります。この電話は、8つの回線と4つのSIPアカウントをサポートしています。この電話機には8つのBLFキーがあり、ホーム画面をスクロールできるため、これらのキーの数が32に増えています。
図16. GXP2140

GXP2140、GXP2160、およびGXP2170電話の 3つの最新モデルには、拡大画面があります。その寸法：4.3インチ、480x272ピクセルの解像度。GXP2140は4つの回線、4つのSIPアカウントをサポートし、5方向の音声会議を開催できます。BLF機能を備えた画面の左側にある4つのキーは、他のモデルと同様に、16個の仮想BLFキーを作成することを可能にします。

このモデルの特徴は、GXP2200EXT拡張パネルに接続できることです。

図17. GXP2200EXT
図18. GXP2140およびGXP2200EXT

GXP2200EXTは、Grandstream GXP2140、GXP2170、およびGXV3240 IP電話に追加の機能、柔軟性、およびスケーラビリティを提供するグラフィカルLCDディスプレイを備えたオプションのキーパッドです。このモジュールには、128 x 384の大きなグラフィックLCDディスプレイと20個のプログラム可能なキー（各キーは2色で光ることができます）があり、画面を切り替える機能により、1つのパネルに40個のキーがあります。合計で、最大4つのGXP2200EXTパネルをGrandstream IP電話に接続できます。したがって、4つの拡張モジュールを使用する場合、最大160のショートカットキーがサポートされます。拡張モジュールが接続され、電力が供給され、電話によって制御されるため、モジュールを接続するとすぐに追加機能が提供されます。
図19. GXP2160

次の電話はGXP2160です。 6つの回線、6つのSIPアカウント、および5方向オーディオ会議の組織をサポートします。画面の左側にある6つのBLFキーとホーム画面をスクロールする機能により、最大24の仮想BLFキーが追加されます。この電話は、オプションの拡張パネルの接続をサポートしていませんが、同様の統合パネルを備えています。 BLF機能付きの24個のMPKキーがあります。
図20. GXP2170

トップエンドのGrandstream GXP2170 IP電話は、12回線、6つの独立したSIPアカウントをサポートし、5方向の音声会議を開催できます。画面の側面にBLF機能を備えた12個のキーとスクロール機能により、最大48個の仮想BLFキーを構成できます。この電話は、追加の拡張パネルGXP2200EXTの接続もサポートしているため、仕事で電話通信を積極的に使用する従業員に最適です。
