.NET Interactiveが登場！| .NETノートブックプレビュー2



2019年11月に、C＃とF＃を利用できるJupyterノートブックの.NETコンテンツを発表しました。本日、Preview 2 .NET Notebookのリリースをお知らせいたします。この記事では、このリリースの主な新機能をすべて紹介します。今すぐ参加！

新着情報


新しい名前-.NETインタラクティブ


本日、.NET Interactiveへの正式な名称変更を発表します。

.NETインタラクティブは、ユーザーがWeb、マークダウン、ノートブックでインタラクティブ機能を作成できるようにするCLIツールとAPIのセットです。

.NET Interactiveを変更する

  • dotnet interactive

     グローバルツール  ：.NETノートブック用（Jupyterおよび操作）
  • dotnet try

     グローバルツール  ：セミナーやドキュメント用。プロジェクトをサポートするインタラクティブなマークダウン。
  • trydotnet.js

    API（まだ公開されていません）：オンラインドキュメント。たとえば  、ドキュメントのページ  と  .NETページ。現在お取り扱いできません。

新しいリポジトリ  -dotnet /インタラクティブ


ツールdotnet try を  dotnet interactive 1つのリポジトリにまとめることにしました  。

  • .NET Notebooksで問題が発生した場合は、.NET Interactiveリポジトリにアクセスしてください。
  • インタラクティブマークダウンとtrydotnet.jsで問題が発生した場合は、Try .NETリポジトリにアクセスし  てください。

新しいグローバルツール-dotnet interactive


.NET Interactiveのインストール方法


まず、次のものがインストールされていることを確認してください：

  • .NET 3.0 SDK。
  • ジュピター。JupyterはAnacondaを使用してインストールできます  。
  • Anacondaプロンプト（Windows）またはターミナル（macOS）を開き、Jupyterがインストールされていることを確認します。

> jupyter kernelspec list
  python3        ~\jupyter\kernels\python3

  • 次に、通常のコンソールでdotnet interactive グローバルツールをインストールし  ます。

> dotnet tool install --global Microsoft.dotnet-interactive

  • Anacondaプロンプトで次のコマンドを実行して、.NETカーネルをインストールします。

> dotnet interactive jupyter install
[InstallKernelSpec] Installed kernelspec .net-csharp in ~\jupyter\kernels\.net-csharp
.NET kernel installation succeeded

[InstallKernelSpec] Installed kernelspec .net-fsharp in ~\jupyter\kernels\.net-fsharp
.NET kernel installation succeeded

[InstallKernelSpec] Installed kernelspec .net-powershell in ~\jupyter\kernels\.net-powershell
.NET kernel installation succeeded

  • Anacondaプロンプトで次のコマンドを実行して、インストールを確認できます。

> jupyter kernelspec list
  .net-csharp    ~\jupyter\kernels\.net-csharp
  .net-fsharp    ~\jupyter\kernels\.net-fsharp
  .net-powershell ~\jupyter\kernels\.net-powershell
  python3        ~\jupyter\kernels\python3

注：詳細についてdotnet try は、dotnet / tryにアクセスして  ください。

新しい言語サポート-PowerShell


PowerShellノートブック




PowerShellノートブックは、PowerShell管理機能とノートブックの豊富なビジュアルエクスペリエンスを組み合わせたものです。PowerShell実行可能エクスペリエンスとワイドテキストおよび視覚化を統合することで、PowerShellユーザーがトレーニングとドキュメントサポートを統合および改善するためのシナリオが開かれます。例として、この新しいPowerShell機能のデモは、共同のオンラインチュートリアルに簡単に変換できます。



インタラクティブな.NETコアによって提供されるマルチコアインタラクションのおかげで、PowerShellをサポートする1​​つのノートブックで、コントロールプレーンとデータプレーンの両方を効率的にターゲットにできます。

データベース管理者、システム管理者、およびサポートの専門家は、リソースの管理に役立つPowerShellノートブックを見つけました。たとえば、このノートブックは、PowerShellでAzure仮想マシンを作成する方法をユーザーに教えます。



nteract.ioでの.NETコードの実行



 Jupyter Notebook で.NETコードを使用することに加えて  、ユーザーはインタラクティブにコードを記述できるようになりました。Interactは、SDK、アプリケーション、およびライブラリーを作成するオープンソースツールであり、インタラクティブなノートブックとREPLを最大限に活用するのに役立ちます。.NETユーザーがインタラクティブデスクトップアプリケーションを含むインタラクティブREPLを最大限に活用できることを嬉しく思います。



対話形式で.NET Interactiveを開始するには、Interactデスクトップアプリケーション  をダウンロードし、  .NETカーネルをインストールし  ます。
