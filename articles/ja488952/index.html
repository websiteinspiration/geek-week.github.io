ニューラルネットワークを使用したフレーズ感情分析

みなさん、こんにちは！

それでも、追放されずに高等教育を受けたすべての人々は、卒業証書を書く段階に達します。私も例外ではありませんでした。興味深いものを実装して、これまで未踏だったものを習得したかったので、ニューラルネットワークと人工知能一般のトピックに注目を集めました。そして私がそれを使って解決したのは、テキストの調性の分析であり、これはすでにさまざまな監視システムで広く使用されています。この記事では、そのソリューションのプロセスについて説明します。

つまり、目的は、フレーズに肯定的な意味合いがあるか、否定的な意味合いがあるかを理解することです。この問題は、ニューラルネットワークだけでなく、いくつかの方法で解決できることをすぐに言いたいと思います。単語の位置が記された辞書などが作れます。（すべての方法がハブで豊富にあります）、しかし、それぞれの方法は記事に従ってさらに進むかもしれませんので、後で検討するために残します。

データ


私の道の最初のタスクは、トレーニング用のデータを収集して前処理することでした。このような場合に適したデータセットは、以前にTwitterで収集された否定的な文章と肯定的な文章に分けられた Y. Rubtsova による短いテキストのコーパスです。特に便利なのは、これらすべてがCSV形式で存在することです。

トレーニングの準備


多くの絵文字、リンク、不要な文字、ヒットなど、データが表示される形式に注意してください。これらすべては重要な情報ではなく、学習を妨げるだけであり、さらに、すべてをラテン語で削除する必要があります。したがって、テキストは前処理に適しています。

def preprocessText(text):
    text = text.lower().replace("", "")
    text = re.sub('((www\.[^\s]+)|(https?://[^\s]+))', ' ', text)
    text = re.sub('@[^\s]+', ' ', text)
    text = re.sub('[^a-zA-Z--1-9]+', ' ', text)
    text = re.sub(' +', ' ', text)
    return text.strip()

ファイルからすべての文を削除したら、それらを小文字にし、「」を「e」に置き換え、参照、言及、意味のない英語の単語を単に削除します。つまり、それらを同じタイプにして、トレーニングに不要な「ゴミ」を一掃します。

from tensorflow import keras
from tensorflow.keras import layers

モデル自体について少し。入力、非表示、および出力のレイヤーがあります。

各層には独自のアクティベーション機能があります。

簡単な説明：人工ニューラルネットワークでは、ニューロン活性化関数が出力信号を決定します。出力信号は、入力信号または入力信号のセットによって決定されます。さまざまなタスクに多数存在するので、ここで詳細を読むことができますが、ここでは2つだけを使用します。

アクティベーション関数ReLuを最初の2つのレイヤーに割り当てます。休日はSoftmaxです。

レイヤーを追加することに加えて、「ドロップアウト」という言葉に気づくことができます。それは何ですか？奇妙なことに、ニューラルネットワークの学習不足の問題に加えて、その予測が真でない場合は、過剰トレーニングの問題があります。モデルは、トレーニングに含まれていない例を分類することを学習するのではなく、トレーニングサンプルから適応したトレーニングサンプルの例のみをよく説明します。これは、以前に見事に仕事をした美しいデータである新しいデータの正体であり、単に「飛んで」不快にあなたを驚かせ始めます。したがって、Dropoutは、特定の確率でグリッドからニューロンを「オフにする」という事実に取り組んでいるため、ニューロンは学習プロセスに参加しなくなります。次に、いくつかのネットワークからの結果が平均化されます（ニューロンがネットワークから除外されると、新しいニューロンが取得されます）。

もちろん、ネットワークは複数回必要になるため、後世のためにネットワークを保存する方法を知っておくとよいでしょう。そうすれば、ネットワークを再トレーニングする必要がなくなります。
