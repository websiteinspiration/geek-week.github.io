すべてのF＃ステートメントと記号の完全なリストは、このMSDNページにあります。

このリンクをたどると、最初に理解するのは、F＃には多くのステートメントと記号があり、1つのメッセージに含めるには多すぎるということです。ブログに。そのことを念頭に置いて、私はオペレーターについてまともなブログ投稿を作成するつもりで、スコープを少し切り取って、最も一般的なオペレーターの議論のみを含めることにしました。これは、1/4だけと言えるでしょう。または、まともな投稿の1/8です。

算術演算子


+
制御されていない合計。考えられるオーバーフロー条件は、数値が合計され、その合計が型でサポートされている最大絶対値を超えた場合です。

-減算は
制御されていません。符号なしの型が減算される場合、または浮動小数点値が小さすぎて型で表すことができない場合に発生する可能性があるアンダーフィル状態。

*
制御されていない乗算。数値が乗算され、積が型でサポートされている最大絶対値を超えた場合に発生する可能性のあるオーバーフロー条件。

/部門
ゼロによる除算は、整数型に対してDivideByZeroExceptionを発生させます。浮動小数点型の場合、ゼロで除算すると、特別な浮動小数点値+ Infinityまたは-Infinityが得られます。浮動小数点数が小さすぎて型で表現できない場合にも、アンダーフィル条件が発生する可能性があります。

％
除算演算モジュールの残りを返します。結果の符号は、最初のオペランドの符号と同じです。

**露出
結果がタイプの最大絶対値を超えた場合に起こりうるオーバーフロー状態。指数演算子は、浮動小数点型でのみ機能します。

ビット演算子


次の表は、F＃でアンパックされた整数型に対してサポートされるビット演算子を示しています。

&&&
ビットごとの演算子「AND」。結果としてビットがあれば1の値を有し、両方のソースオペランドの対応するビットが1である場合にのみ

|||
ビットごとのOR演算子。ソースオペランドの対応するビットのいずれかが1の場合、結果のビットの値は1になります。

^^^
ビット単位のOR演算子。結果としてのビットの値は、元のオペランドのビットの値が等しくない場合にのみ1になります。

~~~
ビットごとの否定演算子。これは、元のオペランドのすべての0ビットが1ビットに変換され、すべての1ビットが0ビットに変換される結果を生成する単項演算子です。

<<<
ビット単位の左シフト演算子。結果は、第2オペランドのビット数だけビットが左にシフトされた第1オペランドです。最上位の位置からシフトされたビットは、最下位の位置にはなりません。最下位ビットにはゼロが埋め込まれます。2番目の引数の型はint32です。

>>>
ビット単位の右シフト演算子。その結果、第1オペランドは、第2オペランドのビット数だけ右にシフトされたビットで取得されます。最下位の位置からシフトされたビットは、最上位の位置にはなりません。符号なしの型の場合、最上位ビットにはゼロが埋め込まれます。符号付きの型の場合、最上位ビットには1が埋め込まれます。2番目の引数の型はint32です。

オペレーターのオーバーロード


すでに述べたように、独自の演算子ロジックを実装する必要がある場合があります。他の.NET言語では、独自のオペレーターオーバーロードを提供することでこれを実現できます。F＃でも同じようにする必要があるのも不思議ではありません。私はまだクラスとオブジェクト指向に進みたくありませんでしたが、それは現在の議論と一致しているようですので、F＃で独自の演算子を実装する方法を見てみましょう。

MSDNからもう一度盗みました。次のコードは、演算子が2つしかないベクトルクラスを示しています。1つは単項マイナス用、もう1つはスカラーによる乗算用です。この例では、オペレーターはベクトルとスカラーの出現順序に関係なく機能する必要があるため、スカラー乗算には2つのオーバーロードが必要です。

type Vector(x: float, y : float) =
   member this.x = x
   member this.y = y
   static member (~-) (v : Vector) =
     Vector(-1.0 * v.x, -1.0 * v.y)
   static member (*) (v : Vector, a) =
     Vector(a * v.x, a * v.y)
   static member (*) (a, v: Vector) =
     Vector(a * v.x, a * v.y)
   override this.ToString() =
     this.x.ToString() + " " + this.y.ToString()
