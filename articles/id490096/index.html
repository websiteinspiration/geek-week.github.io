Cara mudah untuk membuat hub USB kecil untuk Raspberry Pi



Terjemahan sebuah artikel dari situs web buatan Retrocution. Pernahkah

Anda kekurangan port USB saat membuat proyek di Raspberry Pi Zero? Selain itu, hub USB eksternal atau HATS tidak sesuai karena keterbatasan ukuran? Jika ya, atau jika Anda hanya mencari proyek yang menarik dengan bagian-bagian penyolderan, yang akan menghasilkan sesuatu yang bermanfaat, maka proyek ini adalah untuk Anda!



Saya terus-menerus mencoba untuk mendorong Raspberry Pi ke dalam item apa pun untuk proyek saya, dan beberapa di antaranya menggunakan Pi Zero, karena sama sekali tidak memiliki ruang kosong untuk USB Hub / HAT. Saya mencari dan menemukan beberapa situs yang menjual papan kecil dengan hub USB, tetapi biayanya cukup tinggi. Oleh karena itu, saya memutuskan untuk membuat syal saya sendiri, yang cukup sederhana untuk dirakit dari hanya 6 komponen murah. Ia memiliki versi berukuran paku untuk 2 port dan versi yang sedikit lebih besar untuk 4 port. Saya memberikan akses umum ke desain papan di OSH Park dan Anda dapat memesannya dari sana:

2 Port - ORDER AT OSH Park

4 Port - ORDER AT OSH Park

Tentu saja, dewan dapat terus meningkat, terutama berdasarkan umpan balik pembaca. Sejauh ini, mereka bekerja dengan baik, tetapi jangan segera memesannya untuk 100 buah.

OSH Park menjual papan 3 bagian dengan pengiriman gratis [bahkan internasional]. 3 papan untuk 2 port akan dikenakan biaya $ 2,95, dan 3 papan untuk 4 port - $ 3,55. Cukup murah! Setelah memesan papan, toko akan menawarkan Anda untuk memesan stensil (lebih lanjut tentang itu nanti).


Sisi kebalikan dari papan adalah 4 port

Bahan untuk pembuatan


FE1.1s SSOP-28 IC, 1 pc (mereka harus dipesan dari China agar lebih murah - dengan eBay atau AliExpress; Saya membeli 50 seharga sekitar $ 20) /

Kapasitor keramik 0603 10 uF, 3 pcs (hal yang sama - eBay atau AliExpress)

Resistor 0603 2,7 kOhm (serupa)

Oscillator pada 12 MHz, 1 pc (dengan kaki; di Cina Anda dapat membeli 100 pcs seharga $ 5)

Tempat terbaik untuk pembelian tersebut adalah China. Anda harus menunggu 2-3 minggu, tetapi pada akhirnya akan lebih murah daripada penjual di AS [hal yang sama berlaku untuk Rusia / kira-kira. diterjemahkan.]. Saya selalu membeli komponen dalam jumlah besar untuk bertahan lama.

Majelis dewan


Foto-foto menunjukkan versi lama papan, tetapi proses perakitan yang baru tidak berbeda.

Pertama, Anda perlu solder FE1.1. Ini bisa sulit karena nada (jarak antara kaki) hanya 0,64 mm. Saya sudah menyoldernya secara manual, namun, hanya menghubungkan kontak saja sudah cukup. Kadang-kadang Anda dapat menggunakan besi solder untuk memisahkan kontak (jangan lupa untuk menambahkan fluks!), Tetapi jika ini tidak berhasil, coba gunakan kuncir untuk menghapus solder.

Saya melakukan pekerjaan penyolderan yang bagus dengan blow dryer, dan sekarang saya selalu menggunakan metode ini. Untuk metode ini, Anda dapat memesan stensil dari OSH Stensil setelah memesan papan. Stensil akan membantu menerapkan pasta solder ke area kecil papan.





Menggunakan stensil, saya mendapatkan chip yang sangat bersih. Stensil saya hanya memiliki potongan-potongan untuk chip, karena saya memesannya sebelum perubahan akhir papan, tetapi jika Anda memesan stensil bersama-sama dengan papan, mereka akan memiliki potongan untuk kapasitor dan resistor.



Saya menggunakan pengering rambut seperti solder. Ini sangat murah ( Amazon , Ali ), ia memiliki banyak versi, tetapi semuanya tampaknya dibuat di pabrik yang berbeda sesuai dengan satu pola. Jika Anda tidak terlalu sering memasang komponen yang dipasang di permukaan, maka pengering rambut ini akan melayani Anda. Untuk penyolderan seperti itu, saya sangat merekomendasikan membeli tikar silikon agar tidak membakar meja.

Lalu saya biasanya meletakkan tiga kapasitor pada 10 uF. Karena template saya tidak memiliki guntingan untuk mereka, saya hanya menyoldernya secara manual. Saya menambahkan fluks, dan saya memegang kapasitor dengan pinset, menyolder setiap sisi secara bergantian. Pastikan untuk memegangnya dengan pinset, jika tidak akan menempel pada besi solder.



Menyolder kapasitor, saya menggunakan metode yang sama untuk menyolder resistor 2,7 kÎ©.



Setelah menginstal komponen planar, saya mengatur osilator ke 12 MHz.



Ketika semua komponen sudah terpasang dan fluks dibersihkan, Anda mendapatkan hub USB paling kecil dan keren! Omong-omong, warna papan di foto disebut After Dark - ini adalah salah satu opsi yang ditawarkan oleh OSH Park. Standar mereka berwarna ungu, tapi saya sangat suka tampilan papan hitam-tembaga ini, dan selain itu, mereka tidak mengambil uang tambahan untuk warna!



Jika Anda memilih papan 4-port, rakitannya akan persis sama.
