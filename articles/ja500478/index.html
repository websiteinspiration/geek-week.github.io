スリーブ上のテトリス：ウェアラブル電子機器用のガス透過性素材



ウェアラブルエレクトロニクスは、多くの現代人の生活に不可欠な要素となっています。フィットネスブレスレットやスマートウォッチから、拡張現実メガネやスマートシャツまで、既存のデバイスの範囲は、明らかに便利なものから未来的なものまでさまざまです。それでも、機能性に加えて「ウェアラブル」になると、快適性について考える必要があります。ノースカロライナ大学（米国）の科学者は、ウェアラブルエレクトロニクス用の新しいガス透過性材料を開発しました。呼吸することができます。新しい材料を作成するために使用された技術、得られたプロトタイプの特性、および電子機器自体をどれだけ快適に着用できるか これについては科学者の報告から学びます。行け。

研究の基礎


私たちが身に着けているものは、多かれ少なかれガス透過性です。これは、私たちの生理学に準拠する必要があるためです。人間の皮膚は、体の排泄システムの重要な要素であり、汗を通じて代謝産物を排出します。したがって、完全に「密封された」材料を使用してこの機能の実行をブロックすることは、日常生活ではお勧めできません（特殊な衣服や装備は考慮されません）。

ウェアラブル電子機器については、その開発と通常のブレスレットからほぼ本格的なワードローブ要素への変換により、科学者は使用する材料の物理的特性だけでなく、デバイスの機能を直接実行するために重要であるユーザーの快適性に貢献する特性についても考え始めました。

研究者自身が指摘しているように、最新のウェアラブルガジェットは、ポリジメチルシロキサン（PDMS）、ポリエチレンテレフタレート（PET）、ポリイミド（PI）などの固体ポリマー基板をベースに作られています。この研究では、科学者たちは、前述の前駆物質のように優れた導電性と柔軟性だけでなく、優れたガス透過性も備えた新しい材料について説明しています。

似たようなものを作成する試みはすでに行われていますが、それらはすべて、製造中に特定の困難に直面したり、使用上の制限に直面したりしました。

たとえば、ポリビニルアルコール（PVA）をベースにした極薄材料が比較的最近開発されました。この材料のガス透過性は優れていましたが、製造は極めて困難でした。言い換えれば、ゲームはキャンドルの価値はありません。

銀ナノワイヤー（AgNW）の開発もあります。このオプションは高い電気的安定性を提供しましたが、裸のナノワイヤは長期安定性の期間を制限しました。

別のユニークな材料は、砂糖マトリックスをベースにしたPDMS（ポリジメチルシロキサン）スポンジでした。問題は、糖の粒子サイズが限られていることであり、これにより微孔性構造を得ることが困難になりました。さらに、この方法は超薄膜の製造には使用できません。

上記のプロトタイプを思い出して、科学者は、必要なすべての特性を組み合わせながら、製造が容易である本当に良い材料を作成することは非常に難しいと述べたいと思います。しかし、彼らによると、彼らはそれをしました。

彼らは、車輪を再発明するのではなく、既存の開発を組み合わせて、同時に欠点を取り除くことにしました。結果として、蒸発によって作製された多孔質熱可塑性ポリウレタン（ＴＰＵまたはＴＰＵ）フィルムの表面のすぐ下にＡｇＮＷを組み込むことによって、引張（すなわち、可撓性）導電性フィルムが得られた。

蒸着法は、フォトリソグラフィ、真空蒸着、エッチングなどの複雑なステップを必要とせずに、多孔質ポリマーフィルムを製造するためのシンプルで効率的でスケーラブルな自己組織化プロセスです。

研究成果




画像1

画像1aは、プロトタイプの製造プロセスの図を示しています。多孔質の熱可塑性ポリウレタン（TPU）フィルムを蒸着法で作製した後、熱プレスによりAgNW（銀ナノワイヤー）を表面に導入しました。

蒸発の過程で、溶媒の役割はテトラヒドロフラン（THF）によって演じられました。さらに、少量のポリエチレングリコール（PEG）（TPU：PEG = 10：1重量で）を溶液に追加して、水滴の整然とした集合を促進しました。

有機溶媒の蒸発は、基板を冷却した。次に、水分は基板上で凝縮し、液滴除去装置に自然に集まります。

科学者たちは、通常の状況下では、液滴の結合は避けられるべきであり、無秩序な構造の形成につながる可能性があることに注意しています。ただし、この作業では、液滴の和解によって、多孔質構造の形成が促進されました。1bに

示すように、細孔径は、溶液の濃度を変更することによって制御することができます。より高い濃度（2 wt。％TPU + 0.2 wt。％PEG）は、より小さな孔径とより規則的な構造をもたらしましたが、詰まった孔（つまり、その場所のために役割を果たさない孔）の割合は高くなりました。一方、濃度が低すぎる場合（1 wt。％TPU + 0.1 wt。％PEG）、結果として生じる構造は、100μmを超える細孔径でより不規則になる傾向がありました。そのような大きな孔は肉眼で見ることができ、電極の解像度を制限します。

数回の試行の後、溶液の最適濃度は1.5 wt。％TPUと0.15 wt。％PEGであることがわかりました。その結果、均一な多孔質構造が得られた（1b、1e））孔の形状は直径が約40μmの円形に近く、表面被覆係数は約39％（1eおよび1f）でした。

AgNWは、AgNWと水の溶液に浸漬することにより、多孔質TPUフィルムに埋め込まれました。重要なことに、細孔径はAgNWの長さ（〜20μm）よりもはるかに大きかった。多孔質AgNW / TPUフィルムの顕微鏡（1c）は、AgNWが細孔を塞ぐことなくTPUの表面に均一に堆積していることを示しました。

TPUフィルム表面のAgNWは、TPUフィルムから非常に簡単に分離されるため、この問題を解決するために熱処理を行う必要がありました。 TPUの融点は約130°Cであるため、熱プレスには150°Cの温度を使用することが決定されました。

写真1d2でヒートプレス後、ほとんどのAgNWがTPUの内部に直接埋め込まれ、表面に露出したのはごく一部であることがわかります。この処理により、膜厚も6.8μmから4.6μmに減少しました。


画像番号2

画像2aは、多孔質HP-AgNW / TPUフィルム（HP-熱処理後）の光学画像を示しています。グラフ2bは、ディッピングサイクルの数（つまり、AgNW取り込みサイクルの数）の関数としてのフィルム抵抗を示しています。抵抗は最初の4サイクル後にのみ減少し、その後安定し、約14.5オーム/平方（オーム/平方）に達しました。したがって、フィルムの製造プロセスでは、正確に4回の塗布サイクルが使用されました。熱プレス処理によりさらに抵抗が減少しました。これは、圧力と熱アニーリングによって引き起こされたAgNW化合物の接触の改善によって説明できます。たとえば、熱処理後、膜抵抗は7.3オーム/平方に減少しました。

フィルムの多孔性構造は、固体フィルムと比較して光透過性の増加につながります。多孔質TPUフィルムの光透過率は、550 nmで72％でした（2c）そしてAgNWでコーティングした後63％に減少しました。 TPUフィルムの幅がわずかに増加したため、熱プレス後の透過率はさらに61％に減少しました。

次に、ASTM E96に基づいて水蒸気透過率を推定しました。予想通り、多孔性TPUフィルムは、多孔性構造のないフィルムと比較して、大幅に改善された蒸気透過性を示します（2d）。水蒸気透過率は次のとおりです。固体TPUフィルムの場合、2 mg / cm 2 h -1。多孔質TPUフィルムの場合、38 mg / cm 2 h -1 ; 多孔質AgNW / TPUの場合は36 mg / cm 2 h -1および23 mg / cm 2 h -1多孔質HP-AgNW / TPU用。

研究者たちは、透湿性の増加が材料の耐摩耗性も向上させることを示唆しています。この仮説をテストするために、フィルムが皮膚に着用されたときに長期着用テストが行​​われました。人の肌に7日間装着した後、アレルギー反応や汗の蓄積はありませんでした。フィルムで覆われた皮膚の領域と接触領域の周りの領域との間に違いは観察されませんでした。

貫通多孔構造により、汗や湿気がフィルムに浸透し、肌への刺激の可能性が減少し、快適な着用感と耐摩耗性が向上することは明らかです。

さらに、フィルムは生理食塩水に浸され、汗との接触における長期安定性を実証しました（2e） 100時間後、多孔性AgNW / TPUおよびHP-AgNW / TPUフィルムの抵抗は、それぞれ60％および15％増加しました。

フィルムと粘着テープ（2f）間およびフィルムと皮膚間で剥離試験を行った。図2fは、AgNW / TPUフィルムがテープを使用して簡単に剥がせることも示しています（右側の画像は、AgNWが粘着テープに転写されていることを示しています）一方で、HP-AgNW / TPUフィルムははるかに安定しています。

さらに、AgNW / TPUフィルムは剥離テスト後に導電性を失い、HP-AgNW / TPUフィルムは導電性を保持しました。

AgNW / TPUフィルムを皮膚から取り除いた後、一部のAgNWがまだ皮膚に残っていました。しかし、HP-Ag NW / TPUフィルムを使用した同様のテストでは、皮膚にAgNW粒子がないことが示されました。

したがって、熱処理により、長期間の使用におけるフィルムの導電性と安定性を効果的に改善できます。

TPUフィルムの表面の下にAgNWを組み込むことにより、結果のHP-AgNW / TPU多孔質フィルムは、AgNWとTPUの間の接着性を大幅に改善し、安定性を示しました。

HP-AgNW / TPUフィルムは、表面だけでなく厚さ方向にも導電性があることに注意してください。フィルムの上側と下側は導電性ですが、厚さ全体の細孔の端で銀ナノワイヤーによって接続されています。したがって、フィルムはバルク導電性材料として機能しますが、機械的特性の劣化を引き起こす可能性のある多数の導電性フィラーを必要としません。


LEDでテストします。

この特性を実証するために、フィルムはLED回路に接続され、両面導体として使用されました。 2滴の液体金属をフィルムの両面に塗布して、LEDに接続しました。点灯しているLEDは、フィルムの両面が導電性で接続されていることを示します。


Image No. 3

HP-AgNW / TPUフィルムは、その物理的性質により、レーザー切断によりさまざまな形態をとることができます。写真3aは、線幅0.5 mmのフィラメント状蛇行構造に構造化されたフィルム電極を示しています。この場合、フィルムは極薄のままであり、皮膚との密接な接触を保証します。


HP-AgNW / TPUフィルムを皮膚に塗布する手順。

圧縮、ねじれ、皮膚に付着した状態で発生する可能性のあるその他の変形の後で、フィルムは完全に復元されます。必要に応じて、テープを使用してHP-AgNW / TPUフィルムを皮膚から剥がして再利用できます。


シンプルな粘着テープを使用したHP-AgNW / TPUの取り外し手順。

グラフ3bは、フィルムの伸張に応じた抵抗のダイナミクスを示しています。フィルムに5％のひずみがかかると、抵抗は2倍になります。電圧（変形）を取り除くと、抵抗は10％低下しました。その後のサイクルでは、フィルムの伸長とその通常の状態が交互になり、抵抗はほとんど常に一定で可逆的でした。

変形が10％と15％の場合、抵抗は初期値と比較してそれぞれ約4倍と7倍増加しました。このような大きな変動にもかかわらず、興味深い傾向が見られました。各変形レベルで、フィルムは最初のストレッチ中に「プログラム」でき、その後、最初のストレッチによって決定された範囲内で抵抗が可逆的に変化します。言い換えると、最初のひずみサイクルが最も重要な役割を果たし、それが後続のサイクルの抵抗変化の「リズム」を設定します。

その結果、1000サイクルの変形（10％）の後、抵抗は7％未満増加しました。このテストでは、フィルムが非常に柔軟であることもわかりました。したがって、フィルムが0.55 mm-1の曲率に曲げられた場合、抵抗は0.8％だけ増加しました（3秒）そして、10,000回の曲げサイクルの後、抵抗は0.7％増加しました（3d）。 HP-AgNW / TPUフィルムは、導電率を最大45％まで維持します。そして、フィルムの破壊は350％の変形でのみ起こります。

科学者たちは、彼らの開発は電気生理学的信号の継続的なモニタリングに優れていると指摘しています。心電図は一般的に心臓のリズム障害を診断するために使用されますが、EMGは刺激のレベル、筋ニューロパシー、運動行動の分析に使用できます。

ECGとEMG（心電図検査と筋電図検査）の測定では、コンフォーマルな接触と低い皮膚電極インピーダンスが、高い信号対雑音比を得るために重要です。最も正確な情報を取得します。

多孔質HP-AgNW / TPU電極と皮膚の接触を評価するために、Exoflexから作られた人工皮革を使用しました。これは、人間の皮膚とほとんど同じで、ヤング率も同じです。


画像番号4

画像4aは、人工皮膚に転写した後の電極を示しています。顕微鏡検査は、極薄電極が皮膚と共形の（密接な）接触を形成したことを明確に示しています。

初期および拡張HP-AgNW / TPU多孔性電極の複素抵抗は、市販のAg / AgClゲル電極（4d）、固体のAgNW / PDMSフィルム（厚さ0.2 mm）よりも低い。これは、皮膚とのフィルムの接触の質によるものです。 HP-AgNW / TPUフィルムは厚みが薄く、柔軟性が高いため、曲げ剛性が低下し、AgNW / PDMSソリッドフィルムよりもコンフォーマルな接触が得られます。

さらに、HP-AgNW / TPU多孔性電極を使用して取得したECGおよびEMG信号を、市販のAg / AgClゲル電極（4eおよび4f）を使用して取得した信号と比較しました。 ECGおよびEMGテストの電極の位置をそれぞれ4bおよび4cに示します。

ECGの場合、HP-AgNW / TPU多孔性電極は、ゲル電極と同等の品質の信号を提供しました。HP-AgNW / TPU多孔性電極によるECGの測定SNR（信号対雑音比）は7.0 dBで、これはゲル電極（7.1 dB）に匹敵します。

一定の動きの場合、信号品質が低下し、SNR値は多孔質で6.3 dB、ゲル電極で6.2 dBに低下しました。

EMGにより、さまざまなグリップ力に対する筋収縮に対応する信号を明確に区別できます。多孔質HP-AgNW / TPU電極の信号はゲル電極の信号よりも弱いことは注目に値しますが、これは2つのタイプの電極の配置が異なるためでした（両方のタイプの電極が同時に使用されたため）。多孔性電極を備えたEMGのSNR値は24.9 dBで、これはゲル電極のSNR（25.9 dB）に匹敵します。

市販のゲル電極とは異なり、多孔質電極は導電性ゲルを必要としないことは注目に値します。データ収集中のゲルの欠如は、ゲルの劣化などの要因がないため、品質を向上させます。開発されたフィルムのガス透過性を考慮に入れて、これらの実験はさらに、人間の状態の長期連続モニタリングに多孔性電極HP-AgNW / TPUを使用する可能性を示しています。

開発された電極を配置できる場所は人間の皮膚だけではありません。 2番目のオプションはテキスタイルです。


画像5

画像5aは、静電容量式タッチセンサーの図を示しています。上の図5bセンサーにタッチして押した容量値を示しています。

さらに、タッチセンサーシステムの感度は、タッチが発生したときの読み取り値の変化率として定義されます。このシステムでは、感度は86％でした。次に、安定性はタッチセンサーの読み取り値の分散として定義され、約1.65です。 S / N比は35：1で、応答時間は0.1秒未満でした。

ワイヤレスタッチセンサーのシステム（5c）を組み立てるために、HP-AgNW / TPU 50x100 mmフィルムがファブリックスリーブに組み込まれ、レーザー切断を使用して4つのタッチボタンとして表示されました。各ボタンには、左、下、回転、右という独自の機能がありました。

エピローグ


現代のテクノロジーは、機能性だけでなく快適な使用にも長い間関わってきました。ウェアラブル電子機器も例外ではありません。ウェアラブルガジェットの製造に使用される最新の材料のほとんどは、基本的な機能を完全に発揮しますが、小さいながらも重要な詳細はありません。そのような詳細の1つはガス透過性であり、皮膚にデバイスを長期間装着した場合に自由な発汗を提供します。

開発されたHP-AgNW / TPUフィルムには、多数の規則的な細孔があります。このような設計は、主なタスクを完全に実行する能力を維持しながら、フィルムの物理的特性に大きな影響を与えませんでした。

調査中に、いくつかのプロトタイプが作成され、HP-AgNW / TPUのアプリケーションの範囲が示されました。最初のプロトタイプは、ユーザーの健康状態に関する重要な情報を収集することを目的としていました。2つ目は、HP-AgNW / TPUフィルムをほとんどユーモラスに使用して、ワイヤレステトリスゲームパッドを作成することです。どちらの場合も、プロトタイプは優れた結果を示し、多孔質フィルムの特性と性能は現在使用されている商用オプションに匹敵しました。

研究者たちは、ウェアラブルガジェットの使用は快適であると信じているため、将来、ガス透過性材料に関する研究を継続する予定です。まあ、あなたはそれについて議論することはできません。
