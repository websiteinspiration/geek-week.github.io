Distribusi Gaming Linux

Linux menawarkan distribusi universal untuk bekerja dan bermain game, dan distribusi khusus untuk bermain game. Dalam posting ini tentang distribusi game Linux mana yang relevan hari ini (murni IMHO, tentu saja), dan Anda dapat berpartisipasi dalam survei dan membagikan pendapat Anda tentang hal ini.


Game Linux mulai mendapatkan popularitas pada tahun 2013 ketika Valve mengumumkan pembuatan SteamOS berbasis Linux. Selain itu, pada Januari 2013, Valve mengirim game Half-Life ke Linux. Pengguna Linux mendapat kesempatan untuk memainkan game berkualitas tinggi, yang di masa lalu membutuhkan sistem operasi Windows atau bahkan konsol game. Semuanya dimulai dengan cukup sederhana: hanya beberapa perusahaan (CD Projekt Red, Deep Silver, Valve dan beberapa lainnya) menambahkan ikon penguin Linux ke daftar kompatibilitas mereka. Tetapi pada akhirnya, bahkan Gearbox dan Square Enix merilis versi Linux dari permainan terbaik mereka. Hari ini, berkat proyek Valve Proton, game yang tidak memiliki rilis resmi untuk Linux masih dapat bekerja di SteamOS dan distribusi Linux lainnya.

Namun, tidak hanya Valve porting game ke Linux. id Software adalah salah satu perusahaan pertama di pasar permainan Linux. Versi gim yang dibuat oleh perusahaan untuk Linux sangat populer. Banyak dari mereka dianugerahi penghargaan “Readers 'Choice” dan “Editors' Choice” dari Linux Journal . Game Software id pertama yang di-porting ke Linux adalah Doom , dan banyak game perusahaan lainnya juga di- porting . Terlepas dari apakah Anda suka game modern atau retro, di antara distribusi game Linux open source, pasti ada opsi.

Berikut ini gambaran umum singkat dari beberapa distribusi Linux yang akan disukai para gamer.

SteamOS



SteamOSdimiliki oleh Valve. Ini memungkinkan Anda untuk "mengubah" PC Anda menjadi konsol game. Komputer game dari Valve (Steam Machine) adalah perangkat yang dapat menjadi pengganti konsol. Dan kemudian pilihan SteamOS semakin jelas. Tidak ada yang istimewa tentang SteamOS: pada kenyataannya, itu hanya Debian GNU / Linux dengan Steam diinstal sebagai aplikasi default. Saat Anda mem-boot komputer Anda, Steam mulai secara otomatis, dan Anda dapat berinteraksi dengannya hanya menggunakan Steam Controller atau gamepad gaya-Xbox. Anda dapat membuat konfigurasi yang sama dengan menginstal Steam pada distribusi apa pun dan mengaktifkan mode Big Picture. Setelah menginstal SteamOS, Anda dapat memilih antara dua sesi: SteamOS Desktop, di mana Anda akan menemukan lingkungan berbasis Steam yang akrab dan sesi Steam, di mana Anda dapat masuk ke akun Steam Anda.

SteamOS masih memiliki fitur yang dirancang khusus untuk bermain game. Kecil kemungkinan Anda akan menggunakannya untuk bekerja atau belajar. SteamOS juga merupakan firmware yang berfokus pada mesin dengan UEFI dan BIOS, berbagai kartu grafis termasuk Nvidia dan ATI, serta komponen perangkat keras modern.

Lakka



Lakka mengemulasi antarmuka Playstation 3 untuk game retro. Saya menginstal Lakka pada Raspberry Pi Rev 1 menggunakan Etcher . Lakka memuat antarmuka yang sangat akrab bagi pemain PS3, dan seperti halnya di Playstation, Anda dapat mengontrol semuanya hanya menggunakan pengontrol permainan.

Lakka berspesialisasi dalam game retro dan menyediakan emulator game untuk sistem dan mesin yang lebih lama. Jika Anda memiliki gambar ROM yang sesuai, Anda dapat menggunakan emulator untuk memainkan game dari Nintendo, Sega Genesis, Dreamcast, N64 atau game homebrew POWDER, Warcraft Tower Defense dan sebagainya.

Tidak ada game yang diinstal sebelumnya di Lakka OS, tetapi membuatnya mudah untuk menambahkannya melalui SSH atau Samba.

Pop_OS!



Tidak semua orang ingin membuat konsol game dari komputer mereka. Terkadang yang Anda butuhkan hanyalah komputer yang bagus dengan kemampuan menjalankan game dengan kinerja maksimal. System76 telah mengembangkan desktop Pop_OS! berdasarkan desktop GNOME standar dengan beberapa tambahan. Pop_OS! Ini tidak berusaha menjadi yang paling modern, para desainernya hanya menerapkan pengaturan default yang nyaman. Untuk gamer - ini adalah akses mudah ke Steam, Proton, WINE, emulator game, PlayOnLinux, pengakuan otomatis dan konfigurasi pengendali game dan banyak lagi. Pop_OS! tidak jauh dari Ubuntu, itu dimodifikasi sesuai kebutuhan untuk dapat menambahkan fungsi yang diperlukan.

Antara lain, Pop_OS! - Ini adalah desktop berkinerja tinggi. Ini mengimplementasikan semua kenyamanan bawaan GNOME untuk memaksimalkan efisiensi dan membuat modifikasi yang diperlukan untuk memenuhi standar universal yang telah terbentuk selama beberapa dekade di KDE Plasma, Finder dan Explorer. Pop_OS! - Ini adalah lingkungan intuitif yang membantu pengguna fokus langsung pada gameplay, dan bukan pada pemeliharaannya.

Drauger os



Penipu- Distribusi Linux berdasarkan Ubuntu (cabang LTS). Ini masih merupakan proyek muda, tetapi mewujudkan filosofi yang menarik dari proses komputer dan game dan dirancang untuk menghemat sumber daya untuk tugas tersebut. Untuk tujuan ini, Drauger OS meninggalkan konsep desktop tradisional dan sebagai gantinya menyediakan panel kontrol yang disederhanakan yang memungkinkan Anda untuk meluncurkan klien game (misalnya, Steam, PlayOnLinux, Lutris dan sebagainya) dan mengonfigurasi layanan (misalnya, jaringan Anda) atau meluncurkan aplikasi. Pada awalnya itu sedikit membingungkan, terutama ketika panel kontrol praktis larut di latar belakang, tetapi setelah beberapa jam bekerja Anda mengerti bahwa desktop yang lengkap umumnya tidak diperlukan. Lagi pula, Anda tidak mungkin setuju bahwa hal utama dalam komputer adalah desktop-nya.Faktanya, penting bagi Anda untuk masuk ke aplikasi lebih cepat dan lebih mudah dan aplikasi ini berfungsi dengan baik.

Tidak mungkin untuk tidak mengatakan tentang kinerja. OS Drauger berupaya memaksimalkan kinerja game menggunakan kernel latensi rendah. Kernel adalah bagian dari sistem operasi Anda yang berinteraksi dengan perangkat eksternal seperti pengontrol permainan, mouse dan keyboard, dan bahkan hard drive, memori, dan kartu video. Kernel universal yang dikirimkan dengan sebagian besar distribusi Linux membayar kurang lebih perhatian yang sama untuk semua proses. Kernel latensi rendah dapat memberikan preferensi untuk proses-proses tertentu, termasuk video dan grafik, untuk memastikan bahwa perhitungan untuk tugas-tugas penting akan dilakukan dengan cepat, sementara prioritas tugas sistem biasa akan berkurang. Inti dari Drauger adalah berorientasi pada kinerja, sehingga game Anda akan didahulukan dari semua proses lainnya.

Linux impian Anda



Elementary OS - salah satu distribusi Linux yang paling indah.
Jika Anda melihat fitur paling dasar dari "distribusi game" Linux, ternyata sangat mirip. Yang mengejutkan, saya bahkan bermain game di laptop saya dengan RHEL (terima kasih kepada Redp's Flatpak yang memungkinkan Anda menginstal Steam). Jadi jika Anda ingin bermain di Linux dekade ini, pertanyaannya bukan bagaimana melakukannya, tetapi sistem apa yang digunakan.

Jawaban paling sederhana untuk pertanyaan tentang distribusi Linux mana yang akan digunakan terlihat seperti ini: pilih salah satu yang bekerja paling baik di mesin Anda. Anda perlu menemukan distribusi Linux yang dapat dengan mudah mengenali perangkat keras dan pengontrol permainan Anda. Temukan dan Anda akan senang: yang tersisa hanyalah menginstal game dan mulai bermain.

Tentu saja, ada distribusi Linux lain untuk permainan: Fedora Games Spin , RetroPie , Clear Linux , Manjaro . Daftar ini dapat dilanjutkan untuk waktu yang lama. Bisakah kamu? -)
