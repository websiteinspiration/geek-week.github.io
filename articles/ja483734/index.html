JS / C＃/ Rustでasync / awaitを使用するために支払う価格

こんにちは、ハブル！


Javascript / Typescriptを使用しているときに、非同期APIが同様の同期APIよりも遅いことに長い間気づきましたが、そうであるはずです。しかし、前のプロジェクトでは、ファイルシステムとの非同期作業がボトルネックになり、測定を行いました。


awaitは非同期関数またはブロック内でのみ使用できることが知られています。つまり、最低のAPIレベルの非同期がある場合、明らかに不要な場所であっても、ほぼすべての場所でasync / awaitを使用する必要があります。


たとえば、ストアからキーでオブジェクトを取得するサービス関数を記述します。ストレージとして、ファイル、データベース、マイクロサービス、つまり非同期インターフェースを備えた低速ソースを使用できます。パフォーマンスを向上させるために-関数内で、以前に抽出したオブジェクトをキャッシュします（Mapに入れます）。ストアへの実際のアクセスのプログラムが小さくなると、オブジェクトは高速キャッシュから返されますが、関数インターフェイスは非同期のままです！


非同期呼び出しごとにいくら支払う必要がありますか？
テスト結果は憂鬱です...


単純な関数を取り、asyncとマークします。ループで呼び出し、合計時間を測定し、同様の同期コードと比較します。構文を比較するために、全文を3つの言語で引用します。


Typescript（Deno）


同期コード
const b = Date.now()
let j = 0.0
for (let i = 0; i < 1_000_000_000; i++) {
    j += f(i)
}
console.log(j + ', ' + (Date.now() - b)/1000 + 's')

function f(i: number): number {
    return i / 3.1415926
}

非同期コード：


(async () => {
    const b = Date.now()
    let j = 0.0
    for (let i = 0; i < 1_000_000_000; i++) {
        j += await f(i)
    }
    console.log(j + ', ' + (Date.now() - b)/1000 + 's')
})()

async function f(i: number): Promise<number> {
    return i / 3.1415926
}

C＃（.NET Core）


同期コード
using System;

class App {
    static void Main(string[] args) {
        var b = DateTime.Now;
        var j = 0.0;
        for (var i = 0L; i < 1_000_000_000L; i++) {
            j += f(i);
        }
        Console.WriteLine(j + ", " + (DateTime.Now - b).TotalMilliseconds / 1000 + "s");
    }

    static double f(long i) {
        return i / 3.1415926;
    }
}

非同期コード：


using System;
using System.Threading.Tasks;

class App {
    static async Task Main(string[] args) {
        var b = DateTime.Now;
        var j = 0.0;
        for (var i = 0L; i < 1_000_000_000L; i++) {
            j += await f(i);
        }
        Console.WriteLine(j + ", " + (DateTime.Now - b).TotalMilliseconds / 1000 + "s");
    }

    static async Task<double> f(long i) {
        return i / 3.1415926;
    }
}

さび


同期コード
fn main() {
    let tbegin = std::time::SystemTime::now();
    let mut j = 0.0;
    for i in 0..1_000_000_000i64 {
        j += f(i);
    }
    println!("{:?}, {:?}", j, tbegin.elapsed().unwrap());
}

fn f(i: i64) -> f64 {
    return i as f64 / 3.1415926
}

非同期コード：


//[dependencies]
//futures = "0.3"
use futures::executor::block_on;

fn main() {
     block_on(async {
        let tbegin = std::time::SystemTime::now();
        let mut j = 0.0;
        for i in 0..1_000_000_000i64 {
            j += f(i).await;
        }
        println!("{:?}, {:?}", j, tbegin.elapsed().unwrap());
    });
}

async fn f(i: i64) -> f64 {
    return i as f64 / 3.1415926
}
