从立陶宛到纽约：克格勃致力于情报

最近，克格勃间谍技术只能在有关詹姆斯·邦德的书籍中阅读或在电影中窥探。尽管一系列技术已转让用于军事工业，但由于没有苏联的普通公民，签署了一系列特殊法案，所有规则都严格淘汰了陈旧的设备。



立陶宛考纳斯2014年。核掩体已成为一个独特的间谍博物馆的天堂。在地下6米深处，是私人收藏的不寻常展品。有趣的是博览会“克格勃间谍博物馆”。这是NKVD，KGB和其他特殊服务所使用的秘密监视设备和仪器的集合。“ KGB掩体”中装有用于秘密黑客入侵和搜索的设备：电话，便携式秘密无线电通信，无线电发射器和接收器，通信阻止程序，测向仪，错误和各种间谍软件。博物馆的策展人是Urbaitis家族：父女。一家之主将他毕生的事业献给了30多年，这归功于他非同寻常的爱好，他有机会看到并触摸了克格勃的秘密特殊装备。

掩蔽装置


隐藏的摄像机和窃听设备（错误）被用来记录秘密行动中的对话。然后可以将以此方式制作的人物，地点和文件的照片用于勒索。收集有害材料的操作不仅需要训练有素的专家（间谍），而且还需要使用昂贵的设备。听音设备可以放置在任何地方：房屋，汽车，公园，浴室，甚至是您喜欢的宠物的耳朵上的芯片。所有这些使克格勃可以记录和跟踪可用于任何有利可图的特殊服务任命的信息。









摄影机


对于任何代理人来说，必不可少的设备都是相机。在克格勃博物馆中，可以亲眼看到间谍相机的发展历史，包括超小型，隐藏式和复印机。



此处还介绍了超小型TOCHKA-58相机。它是在1958年基于美国MINOX相机开发的。一个独特的原型，使用了相同的原理-胶片供应方案，暗盒设计和设备的其他细节。机械超小型POINTS（尺寸-83x30x15 mm）的目的不过是纯粹的间谍活动。这种相机的技术特征：负片大小-8.5 x 11毫米，用过的胶卷-在Minox暗盒中为9.5毫米，快门速度-1 / 10、1 / 50、1 / 150和1/400秒，胶卷倒片-依次使用弹簧马达和翘曲百叶窗27帧。

录音机


博物馆展示了间谍活动中必不可少的设备-录音设备，他们的克格勃特工用来收集信息，秘密记录对话，听电话，窃听房间里的虫子。



博物馆的展品中包括冷战Mi-51的最微型记录设备。尺寸仅为17 x 11 x 3.5厘米的技术奇迹，可轻松放入口袋或钱包。这个电池供电的小工具可以正常工作，并且可以录制两个小时。美国情报机构还使用了Mi-51相机。


配备Mi-51的腕表麦克风。麦克风通过电线连接到录音机。



博物馆展出的另一种独特设备是间谍语音记录器Yachta / Yacht / Yavir-1。1987年（乌克兰基辅）开发了一种两通道录音机，是传奇的瑞士录音机Nagra SNST的副本。从外观上讲，它与瑞士原装表款的区别仅在于没有功率/信号电平的百分表。

记录仪Yacht-1M / Yavir-1的规格：

  • 尺寸-156 x 103 x 27毫米
  • 重量-600克（不含电池和线圈）
  • 稳定的皮带速度-2.38厘米/秒
  • 胶带宽度-3.81毫米
  • 电源-2-3 V，两节AA电池，根据护照上的规定，使用A-316 Prima或TsNK-0.45电池
  • 频带-170-6000 Hz
  • 爆震系数-不大于0.35％
  • 谐波失真系数-不超过3％



密码机


特别值得一提的是冷战时期的密码学机器展览。最复杂的展览是传说中的Enigma的苏联孙女-密码机Violet M-125。机电式10旋转密码机于1956年开发，很快成为华沙公约国家中最受欢迎的机器。在1989年柏林墙倒塌和苏联解体之后，幸存的紫罗兰被拆除或摧毁。尽管如此，博物馆还是设法得到了一份并保存了下来。参观者可以欣赏到苏联启发的杰作。



间谍电台


探究间谍无线电台的发展历史，追踪特种设备无线电设备的发展演变，也同样有趣。除了可以长距离接收和发送消息外，间谍无线电台还必须具有足够的紧凑性，以进行难以察觉的运输。

RION，SEGA A610，R-350 Orel-展览的荣誉展品。

短波发射接收便携式电报无线电台``RION''旨在进行单工和半双工双向通信，用于秘密操作，工作频率范围为2.5至10 MHz（发射机）和2至12 MHz（接收机PR-56） ）一个重37公斤，包含所有组件的站点被放置在两个手提箱中，该设备可以用容量至少为45安时的12伏电池在野外供电。


P-350鹰

P-350-第一个苏联间谍电台开始量产。特种部队，GRU，MGB降落伞突击的机动侦察团使用了极为易于维护的可靠站。该设备由发射器，接收器和电源组成。R-350进行了改进，并开始作为R-350M生产。发射器的频率范围为1800至1200 kHz（167-25 m），输出功率不少于6瓦。接收器的频率范围是1800至7000 kHz（167-42.9 m）。

特殊电话


风车是封闭的专用通信系统的电话，用于苏联政党和国家权力机关。另外，转盘不适合秘密谈判，只有与其他安全的政府通讯系统配合使用才能达到共同的目的。 ATS-1（依靠最高级别的权力），ATS-2（依靠副部长的级别）-带有标志的象牙电话盘。



有趣的是，甚至在第二次世界大战之前，仍使用所谓的“高频”电话服务来维护机密电话连接。他们被认为是苏联最安全的电话线。政府的HF通信旨在“通过电话向苏联政府和NKVD领导提供清晰，及时且高质量的服务。”最初，仅允许个人安装了HF通信设备的人或其直属代表使用HF通信设备，并且为了保护国家机密，进行秘密协商或通过HF通信以明文形式发送电报，仅允许使用该设备。



从苏联时代起对间谍设备的兴趣就很高。事实的证明是，去年在美国另一个独特的博物馆Urbaitis家族的监督下开幕了（业主保留在阴凉处保留的权利）。曼哈顿西14街上的3500个展品是苏联情报事务的无声见证。



但是据Urbaitis女士说，这个博物馆是非政治的。它具有历史价值，致力于技术进步。她的父亲，现年55岁的朱利叶斯·乌尔巴蒂斯（Julius Urbaitis），花了3年的时间收集藏品。最初，他对第二次世界大战中的文物感兴趣，后来又购买了阿道夫·希特勒（Adolf Hitler）拥有的窃听设备，因此开始了他的间谍之旅。核掩体的一些展品迁移到了新博物馆。







无线电发射机，密码机，微型相机...这些只是展览中展示的一些对象。除设备外，还有克格勃特工的工作场所的重建，沉重的钢手铐，带皮带的审问椅和毒针伞，这在1978年杀死了保加利亚持不同政见者格奥尔基·马尔科夫。











乍看之下，博物馆里有许多平凡的东西-盘子，靴子，配件。但是，他们从自己的眼睛中隐藏了多少种设备，他们只需要更“深入”即可。参观了这种文化机构之后，任何物体，即使是最无害的物体，无论是口红还是雨伞，都将成为间谍工具。

为什么选择立陶宛和美国？“例如，在俄罗斯储存此类特殊设备属于刑事犯罪，可处以最高6年的监禁。俄罗斯的这项技术仍未解密，因此它还不存在，并且在不久的将来将没有这种博物馆。”
