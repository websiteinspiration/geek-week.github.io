Kali Linuxは、Windowsおよびバージョン2019.4の他の機能で視覚的模倣モードを受け取りました

最新のアップデートでは、セキュリティとハッカーの間で人気のあるKali Linuxディストリビューションが、奇妙な「アンダーカバー」モード-「アンダーカバー」を受け取りました。


こんにちは、Kali Linuxです。

開発者自身は、新しいKali機能の必要性を非常に簡単に説明しました。公共の場所で作業するには、他人を困惑させたり、疑惑を引き起こしたりしないように、覆面が必要です。結局のところ、Windowsを実行しているラップトップを持ち歩いている人よりも何が無害なのでしょうか。



Kali Linuxは、ハッカーと組織のセキュリティの独立した監査を行う専門家の間で人気があるため、この新機能は少々ばかげていますが、かなり実行可能です。さらに、Kali 2019.4の更新により、システムにいくつかの変更が加えられました。

ここに更新の短いリストがあります：

  • 新しいデフォルトのデスクトップ環境（Xfce）。
  • 新しいGTK3テーマ（GnomeおよびXfce用）。
  • 覆面モード。
  • Kali Documentationが移動し、Gitで動作するようになりました。
  • Public Packagingは、Kaliに新しいツールを追加したいすべての人のためのマニュアルです。
  • Kali NetHunter KeX-AndroidからのKaliの本格的な起動。
  • セットアップ中のBTRFS。
  • PowerShellを追加しました。
  • カーネルをバージョン5.3.9に更新します。

「アンダーカバー」モードの後のジュースは、Kali NetHunter KeXです。

ハッキングや監査に関しては、自分のラップトップを持参することが常に可能であるとは限りません。スマートフォンではるかに簡単：それは誰にも不安を引き起こしません。

Kali Linux開発者は、以前にWindows Phoneを実行しているデバイスに実装されていたシステムに触発され、Androidを実行しているデバイスでKali Linuxの完全版をリリースしました。


これは、Microsoft Nokia 950および950XL用のMicrosoft Display Dock USBハブの外観で、スマートフォンを周辺機器を備えた実質的に本格的なWindowsベースのワークステーションに変えました。また、同じドッキングステーションが、Microsoft Surface Go、Microsoft Surface Book 2、および標準をサポートする同社の他のデバイスに付属しています。MyDP

システムを最大限に活用するには、上記のタイプのUSBハブ、キーボードおよびマウス（Bluetoothを使用できます）が必要です。必要に応じて、HDMIモニターを入手したり、会議室のテレビに接続したりすることもできます。小さな画面を見れば、望みはありません。



つまり、スマートフォンの側面から快適に操作するための新しいモードの機能は、特定の瞬間における周囲の一連の周辺機器とUSBハブのポート数によってのみ制限されます。

PowerShellと新しいカーネルバージョンの紹介


開発者はついに、PowerShellスクリプトの起動をすぐに実装しました。以前に使用する必要があった松葉杖や追加のツールはありませんでした。



今カーネルのために。現在のリリース2019.4はKali Linuxの最新バージョンであり、そのままで8GB SDカードをサポートします。以降のバージョンのディストリビューションから、サポートされる最小ボリュームは約16 GBに設定されます。これはおそらく、強力なワークステーションから「ラズベリー」、珍しいPC、および一般にARMプロセッサを搭載したデバイスに至るまで、デバイスの動物園全体をサポートすることが困難なためです。

テスト中、インターフェースに問題があったため、開発者は手動で更新することをお勧めします：

apt update && apt dist-upgrade

次に、コンソールから次のように入力します。

rm -rf .cache/ .config/ .local/ && sync && reboot

バージョン2019.4にアップグレードするには、次の操作を行います。

root@kali:~# cat </etc/apt/sources.list
deb http://http.kali.org/kali kali-rolling main non-free contrib
EOF
root@kali:~#
root@kali:~# apt update && apt -y full-upgrade
root@kali:~#
root@kali:~# [ -f /var/run/reboot-required ] && reboot -f

新しいXfceに切り替えたい場合：

root@kali:~# apt -y install kali-desktop-xfce

次に、2019.4をビルドするために実際にアップグレードしたかどうかを簡単に確認します。

root@kali:~# grep VERSION /etc/os-release
VERSION="2019.4"
VERSION_ID="2019.4"
VERSION_CODENAME="kali-rolling"
root@kali:~#
root@kali:~# uname -v
#1 SMP Debian 5.3.9-3kali1 (2019-11-20)
root@kali:~#
root@kali:~# uname -r
5.3.0-kali2-amd64
root@kali:~#

注：「uname -r」の出力は、プラットフォームアーキテクチャによって異なる場合があります。

ソーシャルエンジニアの貯金箱では、Androidデバイス用のUSBハブのアイデアを組み込むことができます。ここでは、Kali Linuxの開発者は何も新しいことを思いつきませんでしたが、多くの専門家がこの機能を気に入ってくれると確信しています。
