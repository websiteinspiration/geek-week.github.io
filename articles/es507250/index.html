Definir una mano de póquer ganadora usando map / reduce en JavaScript

Consiste en comparar dos manos de póker y determinar la ganadora.

Las manos de póker están representadas por líneas que contienen subcadenas (que denotan cartas individuales) de 2 caracteres, separadas por espacios. 2H es un par de gusanos, TC es un cruce de diez, etc.

El primer personaje significa ranking.
Todas las opciones posibles, ordenadas por antigüedad: 2 3 4 5 6 7 8 9 TJQKA .
Con las designaciones numéricas, todo está claro con respecto a las letras: T es 10 ( T en), J es el gato ( J ack), Q es la reina ( Q ueen), K es el rey ( K ing),A - As ( A ce).

El segundo personaje define el traje.
Posibles opciones: H - gusanos ( H earts), C - cruces ( C lubs), S - picos ( S pades), D - panderetas ( D iamonds).

Por ejemplo, una línea tan completa para la mano "2C 5C 3C 4C 6C" - en este caso, esta es una escalera cruzada de dos a seis.

La clasificación de manos se toma como en el Texas Hold'em .

Debe comprender que existen dificultades ocultas: el uso de tarjetas adicionales para resolver situaciones de robo, el uso del valor nominal de pares, etc.

Decisión


Entonces, ¿cómo resolver este problema? Primero debe determinar un método de comparación que resuelva el problema de clasificar las manos, y luego, si la clasificación coincide, el ganador se determina, si es posible, comparando cartas adicionales.

Como el enunciado del problema supone que los trajes son equivalentes, usamos un objeto simple para representar la clasificación de manos:

{ 
   rank: 1,       //     1  9,   ,    
   value: 'ABCDE' //  ,        ,    ASCII-    
}

Ahora podemos escribir una función simple para comparar dos manos representadas en la forma de esta estructura:

function compareHands(h1, h2) {
    let d1 = getHandDetails(h1)
    let d2 = getHandDetails(h2)
    if (d1.rank === d2.rank) {
        if (d1.value < d2.value) {
            return ""
        } else if (d1.value > d2.value) {
            return ""
        } else {
            return ""
        }
    }
    return d1.rank < d2.rank ? "" : ""
}

Así que ahora todo lo que tenemos que hacer es crear un objeto de nuestras manos, ¡y aquí comienza la diversión!

Comprobar flash


Esto es bastante simple, ya que analizamos la línea entrante y la clasificamos por trajes. Si el palo de la última carta coincide con el palo de la primera, tenemos un flash.

const flush = suits[0] === suits[4]

Comprobar calle


La calle no es mucho más complicada: si todas las cartas (independientemente de su palo) tienen una denominación consistente, entonces esta es.

Comenzamos con la primera carta en la mano y para cada siguiente chequeo si su valor nominal es consistente con la carta anterior. Usando el índice pasado a través de la devolución de llamada, de la siguiente manera:

    const first = faces[0].charCodeAt(0)
    const straight = faces.every((f, index) => f.charCodeAt(0) - first === index)

Definir duplicados


Este paso es un poco más complicado. Debemos calcular la cantidad para cada valor nominal en nuestra mano. Y también necesita alguna forma de determinar pares, triples, etc., para simplificar la clasificación de sus manos de esta manera:

  • Cantidad de conteo para cada valor nominal
  • Convertir una cuenta en una presentación en la que pueda buscar.

Es necesario asegurarse de que sea posible determinar la presencia de los "cuatro" (cuadrados) o tres en la mano, el número de pares (no uno, uno o dos), etc.

Entonces, primero calculamos cuántas veces se encuentran las tarjetas de cada valor:

    const counts = faces.reduce(count, {})

function count(c, a) {
    c[a] = (c[a] || 0) + 1
    return c
}

Y luego buscamos estos cálculos, en el sentido literal de "contar los cálculos".

    const duplicates = Object.values(counts).reduce(count, {})

Clasificación de manos


Ahora tenemos toda la información necesaria para clasificar las manos, hasta ahora sin tener en cuenta las posibles situaciones de sorteo.

    let rank =
        (flush && straight && 1) || /* -,     - -        */
        (duplicates[4] && 2) || /*  -     */
        (duplicates[3] && duplicates[2] && 3) || /* - -  +  */
        (flush && 4) || /*  -      */
        (straight && 5) || /*  -       */
        (duplicates[3] && 6) || /*  -     */
        (duplicates[2] > 1 && 7) || /*   */
        (duplicates[2] && 8) || /* -     */
        9 /*     ,      */

Por lo tanto, una escalera de color es la combinación más fuerte con un rango de 1 (para no complicarlo, consideramos que es posible un empate si alguien tiene una escalera real), luego cuatro de un tipo, casa llena, etc.

En este caso, el operador && se usa fantásticamente , lo que escanea hasta el último valor si los valores entre paréntesis anteriores son correctos. Por lo tanto, (flush && direct && 1) devuelve 1 si flush y direct son verdaderos; de lo contrario, devuelve false.

Situación dibujada


Si dos manos tienen el mismo rango, el ganador se determina adicionalmente por el valor nominal de las cartas, si es posible. Algunas reglas están asociadas con esto.

  • Pareja vs pareja. Un par con un valor nominal más alto gana. Si los valores coinciden, gana la siguiente carta más alta. (Para dos pares se resuelve de manera similar).

    Por ejemplo, si comparamos la mano “2H 2D AH KC 3D” con la mano “4H 4C JC TC 3H”, entonces los cuatro de la mano ganadora, aunque en la primera mano la carta más alta tiene el valor nominal más alto: un as.
  • Casa llena versus casa llena . El trío con el valor nominal más alto gana. Si las denominaciones de triples en ambas manos son iguales, entonces miramos las denominaciones de dos.
  • Actuamos de manera similar al comparar cualquier mano con el mismo rango: decide el valor nominal de la carta más alta en combinación.


Por lo tanto, primero debemos ordenar por cantidad y luego por valor nominal en nuestra línea saliente. Recuerde, la cadena completa de cinco caracteres se usa para resolver casos con clasificaciones de manos coincidentes.

    let value = faces.sort(byCountFirst).join("")

function byCountFirst(a, b) {
    //    -  ,  
    const countDiff = counts[b] - counts[a]

    if(countDiff) return countDiff //   ,  
    return b > a ? -1 : b === a ? 0 : 1
}

const order = "23456789TJQKA"
function getHandDetails(hand) {
    const cards = hand.split(" ")
    const faces = cards.map(a => String.fromCharCode([77 - order.indexOf(a[0])])).sort()
    const suits = cards.map(a => a[1]).sort()
    const counts = faces.reduce(count, {})
    const duplicates = Object.values(counts).reduce(count, {})
    const flush = suits[0] === suits[4]
    const first = faces[0].charCodeAt(0)
    const straight = faces.every((f, index) => f.charCodeAt(0) - first === index)
    let rank =
        (flush && straight && 1) || /* -,     - -        */
        (duplicates[4] && 2) || /*  -     */
        (duplicates[3] && duplicates[2] && 3) || /* - -  +  */
        (flush && 4) || /*  -      */
        (straight && 5) || /*  -       */
        (duplicates[3] && 6) || /*  -     */
        (duplicates[2] > 1 && 7) || /*   */
        (duplicates[2] && 8) || /* -     */
        9 /*     ,      */

    return { rank, value: faces.sort(byCountFirst).join("") }

    function byCountFirst(a, b) {
        //    -  ,  
        const countDiff = counts[b] - counts[a]

        if(countDiff) return countDiff //   ,  
        return b > a ? -1 : b === a ? 0 : 1
    }

    function count(c, a) {
        c[a] = (c[a] || 0) + 1
        return c
    }
}

}
