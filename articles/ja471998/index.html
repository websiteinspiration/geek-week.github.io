F＃10：リスト

あらゆる言語での深刻なプログラミングには、常にリストが含まれます。したがって、あなたはF＃がそのListモジュールでリストを非常によくサポートしていることを知って嬉しく思います。F＃のリストは、同じタイプの順序付けられた、変化しない一連の要素です。

リストを作成する


F＃には、リストを作成する方法がいくつかあります。

  • 空のリストを作成する
  • 1から10までのアイテムを含む単純なリストを作成する
  • 1から10の範囲の奇数のリストを作成します
  • forループを使用してリストを作成します。

let prettyPrint desc list =
    printfn desc
    printfn "%A" list
         
 
//empty list
let listEmpty = []
//simple list
let list1 = [1 .. 10 ]
//simple list, with step value
let list2 = [1 ..2..10 ]
//using for loops to create lists
let list3 = [for i in 1 .. 10 -> i*2 ]
 
prettyPrint "let listEmpty = []" listEmpty
prettyPrint "let list1 = [ 1 .. 10 ]" list1
prettyPrint "let list2 = [ 1 .. 2..10 ]" list2
prettyPrint "[ for i in 1 .. 10 -> i*2 ]" list3



上記の最後の例は、forループを使用してリストを作成する方法を示しています。これは非常に優れていますが、F＃ツールキットにはさらに優れた強力な機能、つまりリスト内包があります。

リスト内包表記は、関数/ループ/条件などを含むほぼ標準のF＃を使用してリストを作成できる強力な方法です。

リストを作成する方法の例を見ていきましょう。

let is2 x = match x with
    | 2 -> "YES"
    | _ -> "NO"
 
 
 
//yield directly

    yield 1;
    yield 2;
    yield 3;

//yield numbers between 1 and 20 where we use the
//Math.Pow function to return a new number

                yield Math.Pow(i,2.0)
            ]
     
//yield only numbers between 1 and 20 that 
//can be divided by 5
let list3 = [
                for i in 1 .. 20 do
                    if i % 5 = 0 then
                        yield i
            ]
 
 
//yields YES/NO strings depending on
//whether source int = 2 or not
let list4 = [for i in 1 .. 5 ->
                is2 i
            ]



いくつかの便利なリスト演算子


短所演算子


cons演算子「::」を使用して既存のリストに値を追加できるため、次のリストがあるとします。

let list1 = [1;2;3;4]
let list2 = 42 :: list1



連結演算子


別の非常に便利な演算子は「@」演算子です。これを使用すると、同じタイプのリストを組み合わせることができます。したがって、たとえば、次のような場合：

let list1 = [1;2;3;4]
let list2 = [5;6;7;8]
let list3 = list1 @ list2

すべて（forall）


コレクション内のすべての要素が特定の述語を満たすかどうかを確認します。この例では、真の戻り値を取得するには、リスト全体に0が含まれている必要があります。

let isAllZeroes list = List.forall (fun elem -> elem = 0.0) list
printfn "%b" (isAllZeroes [0.0; 0.0])
printfn "%b" (isAllZeroes [0.0; 1.0])

リストの再帰についての簡単な考察


再帰について語らない限り、リストの説明は完全ではありません。再帰は投稿全体を対象とするものですが、ここでは、F＃のリストで機能する再帰関数を記述するために何が必要かを見てみましょう。

let printIt desc x = 
    printfn "%A %A" desc x
 
let rec printList list =
    match list with
    | h :: t -> 
        printIt "head=" h
        printIt "tail=" t
        printList t
    | [] -> ()
 
printList [1;2;3;4;5]

この関数が再帰的であることをF＃コンパイラに伝えるrec キーワードの使用に注意してください。このキーワードがなければ、再帰的な関数呼び出し-非常に明確なエラーテキストではありませんが、コンパイルエラーメッセージが表示されます。
