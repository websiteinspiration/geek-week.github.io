“Encontrado, visto, recebido”: convites incomuns para entrevistas - do cabeçalho HTTP à mensagem no mecanismo de pesquisa

Algumas empresas ocultam ofertas para desenvolvedores em código HTML ou cabeçalhos HTTP, enquanto outras oferecem primeiro para resolver problemas algorítmicos no site. Vamos falar sobre vários desses casos.





— Nikita Kachanovsky — Unsplash



«»



Muitas empresas ocultam ofertas de emprego em código HTML. Por exemplo, a Microsoft inseriu anúncios de RH em uma página do Azure. Uma abordagem semelhante é seguida pelo The New York Times e Imgur . A Mozilla também envia convites para trabalhar no console do navegador. Mas algumas organizações são muito inventivas e ocultam ofertas em cabeçalhos HTTP .



Em 2015, Troy Hunt, especialista em segurança da informação, realizou um workshop em uma empresa australiana. Durante o exercício, um dos alunos notou uma resposta HTTP interessante recebida pelas APIs móveis do servidor Airbnb. Continha um convite para trabalhar e o endereço de e-mail do chefe do departamento de desenvolvimento da plataforma on-line para encontrar moradia: 



X-Hi-Human: The Production Infrastructure team added this header.
Come work with us! Email kevin.rice+hiring@airbnb.com


Há um mês , o material apareceu no Hacker News , cujo autor descobriu outros sites com ofertas de emprego em cabeçalhos HTTP. Eles foram encontrados no Paypal.me, Booking.com, Etsy.com e Otto.de e muitos outros. O autor publicou a lista completa em seu repositório no GitHub .



Às vezes, para encontrar um convite oculto, você precisa se aprofundar um pouco mais. Por exemplo, em 2017, uma página especial com a oferta foi publicada na Apple. Para abri-lo, foi necessário analisar o tráfego de aplicativos no iPhone. Alguns deles encaminharam pacotes para servidores de blobstorecomumente usado para hospedar dados do iCloud. A oferta foi localizada em um dos endereços, mas a Apple precisou removê-la do site. O fato é que ele foi descoberto pelo editor da revista ZDNet e publicado no Twitter. Depois disso, a corporação começou a receber muitas solicitações.



As tarefas



Curiosamente, de maneira semelhante, eles até selecionam o departamento de estado britânico de contra-inteligência. Os participantes são convidados a analisar a imagem (no final da página pelo link ) e decodificar a mensagem contida nela - após limpar a imagem do “ruído” no editor de gráficos e usar codificações na RFC 4648 , o autor da nota encontrou este texto: 



$ python solution.py | tr -d '-' | xxd -p

resolver vários problemas. O engenheiro Max Rosett estava procurando informações sobre as funções lambda do Python quando uma mensagem apareceu no mecanismo de busca: “Você está falando nossa língua. Preparado para um desafio? ” ("Você fala a nossa língua. Pronto para passar no teste?"). Max concordou e uma aba do navegador foi aberta com um console semelhante ao Unix e seis tarefas algorítmicas. Quando Max os decidiu (levou 48 horas), ele foi convidado a enviar um currículo - e, como resultado, os engenheiros foram contratados pela corporação.
