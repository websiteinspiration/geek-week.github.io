Raspberry PiおよびSIM7600E 4G HATモデム

全てにおいて良い日。この記事では、Raspberry Piに接続できるSIM7600X 3Gモデムを使用した経験についてお話します。この記事は、WIFIネットワークがない場合にRaspberryを自律的に使用する必要があるユーザーにとって役立つ場合があります。

システムパラメーター


Raspberry Pi 3B +
OS：Raspbian GNU / Linux 10（バスター）
モデムSIM7600E 4G HAT、ファームウェアバージョンLE11B08SIM7600M22



ラズベリーにモデムを接続する


SIM7600E 4G HATモデムの価格は現在約80ドルです。こちらから注文できます。

モデムには、ラズベリーに接続するための40ピンスロットがあります。モデムボードの背面にSIMカードスロットがあります。メガホンSIMカードを使用しました。



モデムの電源を入れると、PWRとNETの2つのLEDインジケーターが点灯することに注意してください。



ドキュメントには、モデムを制御するATコマンドはUSBポートを介して送信する必要があると記載されています。したがって、このポートをRaspberry USBポートに接続するには、別のワイヤーが必要です。



次に、minicomを使用してモデムに接続してみましょう。この場合、デバイスファイル/ dev / ttyUSB2または/ dev / ttyS3を使用できます。

sudo apt-get install minicom
sudo minicom -D /dev/ttyUSB2

モデムがATコマンドに適切に応答することを確認してください。たとえば、AT + CPIN？モデムがSIMカードを認識しているかどうかを確認できます。

AT
OK
AT+CPIN?
+CPIN: READY

GPS座標を取得しています。


SIM7600モデムにはGPSレシーバーが装備されています。使用するには、まず外部GPSアンテナをGNSSジャックに接続します。その後、アンテナを信頼性の高いGPS信号受信のゾーンに持ち込みます（つまり、GPSはコンクリートの建物に引っ掛からないため、少なくとも窓からアンテナを投げます）。



GPSレシーバーをオンにするには、次のコマンドを実行します。

AT+CGPS=1
OK

レシーバーが起動したことを確認します。

AT+CGPS?
+CGPS: 1,1

GPS座標を取得してみましょう。

AT+CGPSINFO                                                                     
+CGPSINFO: ,,,,,,,,

受信機が「衛星を捕まえる」まで、最後のコマンドを数回繰り返す必要がある場合があります。

AT+CGPSINFO                                                                     
+CGPSINFO: ,,,,,,,,
AT+CGPSINFO                                                                     
+CGPSINFO: 
2232.43534,N,11404.325345,E,300618, 0855320.0,96.0,0.0,0.0

モデムをGSMネットワークおよび3Gネットワ​​ークに接続します。


ネットワークにアクセスするには、最近リリースされたqmcliユーティリティを使用すると便利です。この方法の詳細については、ニックネームmkrzysztofowiczの人に感謝します。（その前に、私はsakis3gおよびumtskeeperユーティリティを使用する必要がありましたが、これは信頼性が低く、あまり便利ではありませんでした。）最初に、必要なパッケージをインストールします。

sudo apt-get update 
sudo apt-get install libqmi-utils udhcpc

SIM7600モデム無線モジュールはデフォルトでオフになっています。それをオン：

sudo qmicli -d /dev/cdc-wdm0 --dms-set-operating-mode='online'

モデムのNETインジケーターが点滅し始めます。これは、モデムが検索に切り替えてGSMネットワークに接続したことを意味します。ネットワークが見つかり、信号レベルが低すぎないことを確認します。

sudo qmicli -d /dev/cdc-wdm0 --dms-get-operating-mode
sudo qmicli -d /dev/cdc-wdm0 --nas-get-signal-strength
sudo qmicli -d /dev/cdc-wdm0 --nas-get-home-network

qmi-wwanドライバーは、ラズベリーに接続しているSIM7600モデムを見て、wwan0ネットワークインターフェイスを作成します。デフォルトでは、このインターフェースのプロトコルとして802-3がインストールされていますが、必要なプロトコルは明らかにraw-ipです。この変更を加えましょう：

sudo qmicli -d /dev/cdc-wdm0 -w	#    ,   wwan0
sudo ip link set wwan0 down		
echo 'Y' | sudo tee /sys/class/net/wwan0/qmi/raw_ip 
sudo ip link set wwan0 up

次に3Gネットワ​​ークに接続します。APNのパラメーターは、GSMプロバイダーのページにあります。

sudo qmicli -p -d /dev/cdc-wdm0 --device-open-net='net-raw-ip|net-no-qos-header' --wds-start-network="apn='<YOUR_APN>',username='<YOUR_USERNAME>',password='<YOUR_PASSWORD>',ip-type=4" --client-no-release-cid

たとえば、メガホンの場合：

apn='internet',username='gdata',password='gdata'

最後に、IPアドレスを取得します。

sudo udhcpc -i wwan0

ifconfigコマンドを使用すると、wwan0インターフェースに特定のIPアドレスが割り当てられていることがわかります。もしそうなら、おめでとうございます、すべてがうまくいきました！

ifconfig

wwan0: flags=4305<UP,POINTOPOINT,RUNNING,NOARP,MULTICAST>  mtu 1500
        inet 10.245.223.144 netmask 255.255.255.224  destination 10.245.223.144

それで全部です！私たちの困難な事業の成功をお祈りします。
