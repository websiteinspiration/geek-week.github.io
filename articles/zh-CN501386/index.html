MNT改革-完全开放式笔记本，消除偏执



MNT改革是有史以来最开放的偏执狂笔记本。所有电路板，引导加载程序和驱动程序都是打开的，没有网络摄像头，并且18650罐可以更换为电池，这真是令人兴奋的庆祝。

i.MX8 ARM处理器以现成的SoM（模块上系统）的形式使用，具有完全被动的冷却和仅4GB的RAM。

技术指标




处理器：恩智浦/飞思卡尔i.MX8MQ，具有4个ARM Cortex-A53内核（1.5 GHz）和1个Cortex-M4F内核。处理器和内存设计为可替换的SO-DIMM。
内存： 4GB LPDDR4
GPU：Vivante GC7000Lite主线Linux内核的GPU驱动程序，并支持OpenGL 2.1，ES 2.0
显示全高清（1920x1080像素）12.5英寸 IPS eDP， 通过MIPI-DSI接口连接很少有128x32px OLED屏幕可显示状态消息。 。
该USB： 3倍USB 3.0（类型A）
网络接口：将以太网端口千兆MINIPCIE Wi-Fi接入（仅适用于装配选项）
盘：内部M.2 M-键全尺寸床NVMe SD卡插槽的SSD插槽..
PCIe：1个miniPCIe插槽（PCIe 2.0 1x），1个M.2 / NGFF插槽M键（PCIe 2.0 1x）
键盘：带Kailh Choc棕色开关的机械式USB键盘，可调光的背光灯，开放式固件
Trackball（选件）：光学式带5个机械开关的USB轨迹球（Kailh Choc Brown），开放式固件
Trackpad（选件）：改革电容性USB触控板，开放式固件
外壳：模块化外壳，由CNC铣削，喷砂处理，阳极氧化的6061铝制成。底盖由半透明的丙烯酸磨制而成。
声音： Wolfson WM8960 ADC / DAC，立体声扬声器，3.5英寸耳机/麦克风插孔（无内部麦克风）
摄像头：内部MIPI-CSI连接器
电池：LiFePO4电池技术-与LiPo电池相比，它具有更高的防火安全性和更多的充电周期。8个用户可维修的18650电池，总计12 Ah / 3.2V 。大约5小时的电池寿命
系统控制器：恩智浦LPC11U24 ARM Cortex-M0芯片，具有开放式固件和可入侵的扩展端口
操作系统：预装了Debian GNU / Linux 11，Linux 5.x主线内核
尺寸： 29 x 20.5 x 4厘米
重量：〜1.9公斤

母板




具有处理器，视频核心和RAM的SoM作为SO-DIM模块插入主板中。在照片中，它被散热器关闭。值得注意的是，HDMI / DisplayPort未用于连接显示器，因为这需要专有固件。显示器通过MIPI DSI-> eDP转换器连接。

系统模组（SoM）




作为大脑，使用了来自Boundary Devices的现成的SoM。该开发板具有带视频核心和4GB RAM的处理器。这是USB和PCIe控制器。该模块只需通过SO-DIMM连接器（如笔记本电脑中的RAM）插入主板，因此任何人都可以设计自己的SoM并完全替换平台。

电池




伙计们决定不打扰，放普通的18650罐，这些元素对于处理电子烟的每个人都是熟悉的。这个想法是，任何人都可以从本地商店的易于使用的电池中更换电池。

键盘




带Kailh Choc开关的背光机械键盘。布局很奇怪，由于某种原因，间隙很小，中间有两个中提琴。

状态OLED屏幕




键盘上方是一个128 x 32 px的小OLED屏幕，您可以根据需要使用它。当前的键盘布局和状态消息可以显示在上面。

显示




12.5“ Innolux的IPS eDP（嵌入式DisplayPort）面板，24位彩色，FullHD分辨率（1920 x 1080）。

轨迹球或触摸板






您可以选择轨迹球或触摸板。两者的固件和驱动程序自然是开放的。
