注意深いリスナーへのギフト：オーディオCDの「プレギャップ」に隠されていたオーディオフック

すでにレコードを保存する驚きについて話しました。それは1901年のヴァイナル、ピンク・フロイドとB-52の作曲、小さなプログラム、さらには光学実験でさえありました。コメントでの

あなたの返答が気に入ったので、トピックを拡大することにしました。レコードと他のフォーマットの両方を見てみましょう-さまざまなアルバムに隠されている新しいイースターエッグについて話しましょう。


写真提供：Cristina Gottardi

レコードは「機械的」フォーマットであるため、トラックを完全に非表示にすることはできません。注意深く見ると追加のトラックが簡単に見つかり、好奇心旺盛なリスナーはすぐにその内容を再現しようとします。私たちがCDについて話す場合、彼らの助けを借りて、ファンとのより微妙な「ゲーム」を行うことができます。これらの手法の1つは「プレギャップ」と呼ばれていました。

これは、「Red Book」と呼ばれる、CD上のデジタルサウンドの標準的な「バーンアウト」に従って動作します。ちなみに、CDの一般的な仕様セットをさらに興味深い「レインボーブック」という名前で入力すると、「レッドブック」になりました（このトピックは別のハブポストに値するように思えますが、どう思いますか？）さらに、Red BookはしばしばCD-ROMと混同されますが、念のため、これがまだCDDA（コンパクトディスクデジタルオーディオ）であることを明確にすることは価値があります。


写真提供：Evan / CC BY-ND

したがって、Red Bookでは、ディスク上の各トラックの前に少なくとも150の空のブロックが必要です。このポーズは、約2秒続く仕様に従って、目次の目次に記載されています。このトラックのゼロ（「インデックス00」）インデックス（「インデックス01」）。これらのブロックで「燃え尽きる」ためにアルバムをマスタリングおよび準備する場合、「ミュージカルイースターエッグ」を録音することが可能です。

非表示のトラックを表示できるCUEシートの例：

PERFORMER "Bloc Party"
TITLE "Silent Alarm"
FILE "Bloc Party - Silent Alarm.flac" WAVE
 TRACK 01 AUDIO
    TITLE "Like Eating Glass"
    PERFORMER "Bloc Party"
    > INDEX 00 00:00:00
    INDEX 01 03:22:70
 TRACK 02 AUDIO
    TITLE "Helicopter"
    PERFORMER "Bloc Party"
    INDEX 00 07:42:69
    INDEX 01 07:44:69

一方、非表示のトラックを聴くのは簡単ではありません。標準のプレーヤーでは、異常な音が見られなかったり、エラーが発生したりすることはありませんが、通常のトラックを再生して最初に巻き戻す（つまり「シークする」）と、非表示のレコードになります。聞くことができます。簡略図（下の画像）では、「0」のカットオフとして表示されます。


Image Gerard Fuguet / CC BY

この手法はさまざまな目的で使用されました。たとえば、彼の作品の追加の「イントロ」として。 1999年のラムシュタインライブアルバムの一部のエディションには、バンドのコンサートの1つで歓喜する聴衆とのそのようなプレギャップが含まれています。もちろん、他の例もあります。

そのため、悲観的なムードで知られる伝説のエモアルバム「悪魔と神は怒り狂う私」で、ブランニューは重ねられた電話の会話からギャップ前の構成に置かれました。また、Unkleによるイギリスのアルバム「Psyence Fiction」の前には、アルバムを録音するようにインスピレーションを与える隠された曲がミックスされています（以下のビデオ）。

注意：ビデオの説明には、この作曲で使用されたすべてのサンプルの完全な筆記録が、作者とともに記載されています。元のトラックの名前とタイムコードも含まれています。

「プレギャップ」は、通常の非表示のトラック（リミックス、アウトテイク、コンポジションなど）を配置するためにも使用できます。これらのトラックは、何らかの理由でアルバムの公式プレイリストに配置したくないものです。

これは、ほとんどのグループやアーティストが行うことです。たとえば、REMバンドのMurmurアルバムとReckoningアルバムのアニバーサリー再リリースでは、ラジオで元のレコードを宣伝するために80年代に録音されたオーディオクリップが非表示になっています。ちなみに、83年と84年のアルバムには隠された「無題」のトラックが含まれています。最初は「シェイクスルー」と「ウィーウォーク」の間の小さなスニペットの形式であり、2番目はミニトラックです-「Camera」と「（Do n't Go Back To）Rockville」の間ですが、すでにアルバム「Reckoning」に含まれています。

90年代後半には、エンハンストCDのソフトウェア部分にプリギャップトラックが配置され始めましたが、これは別の話です。これは、ハブレに関する次の資料の1つで取り上げます。
