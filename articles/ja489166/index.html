ポート、バス、プロトコルの違いは何ですか？バスは、ポートと同様に、一連の接点（配線）とその使用方法に関する合意です。ポートのみが外部デバイスを接続するための接続を備えており、バスはマザーボード上のコンポーネントとの通信に使用されます。ポートは、本質的には中央にコネクタがあるバスです。プロトコルは、連絡先を介した対話です。 PS / 2の例では、これはClockおよびDataコンタクトを介したデータ転送の順序でした。

void SendBit(BYTE bit) {
	float power = (bit != 0) ? 3.3f : 0.0f;
	SetPower(DATA, power);
	SetPower(CLOCK, 3.3f);
	Sleep(50);
	SetPower(CLOCK, 0.0f);
	Sleep(50);
}

void SendData(BYTE data) {
	SendBit(0);
	
	for (BYTE i = 0; i < sizeof(BYTE); i++) {
		BYTE bit = (data >> i) & 1;
		SendBit(bit);
	}

	SendBit(0);
	SendBit(1);
}
