マルチラインテキスト


上記の選択調整手法は、残念ながら、複数行のテキストには適していません。それでも、そのようなテキストに同様の何かを実装するには、JavaScriptの機能を利用して、各単語を埋め込み（小文字）要素（たとえば）に配置する必要があります<span>。各単語が独自のelementに出現し<span>たら、これらの各要素に疑似要素を追加する必要があります。その後、上記の効果を複数行のテキストに適用できます。

次に、各単語を<span>コンテナに入れるスクリプトを示します。

let paragraph = document.querySelector(".text");
const words = paragraph.textContent.split(" ");

paragraph.innerHTML = "";

words.forEach(function (word) {
  let wordItem = document.createElement("span");
  wordItem.setAttribute("data-word", word);
  wordItem.innerHTML = word + " ";
  paragraph.appendChild(wordItem);
});

その後、要素を<span>スタイル設定する必要があります。次に、それぞれに疑似要素を追加する必要があります。

span {
    position: relative;
    font-size: 1.25rem;
    line-height: 1.4;
  }

  span::after {
    content: attr(data-word);
    position: absolute;
    left: 0;
    right: 0;
    top: -0.28em;
    height: 75%;
    padding-top: 0.14em;
    background: #fff;
    pointer-events: none;
  }

  span::selection {
    background: rgba(#4C7DE1, 0.18);
  }

ユーザー選択プロパティの調査


このプロパティをuser-select使用すると、ユーザーが特定のテキストを選択する機能を設定できます。このプロパティは、テキストを選択する機能を無効にする場合に役立ちます。これは、ユーザーが互いに隣接して配置されているマテリアルを選択する機能を制限する場合に役立ちます。これはを説明する標準ですuser-select。

グローバル選択無効を使用しない


選択をグローバルに無効にすることはお勧めしません。選択を無効にする場合は、意味のない要素に対してのみ無効にするようにしてください。これを行うには、ヘルパークラスを作成します。例えば-これ：

.disable-selection {
    -webkit-user-select:text;
    -moz-user-select:text;
    -ms-user-select:text;
    user-select:text;
}
