14 perpustakaan Node.js paling populer di tahun 2020



Selamat siang teman!

Saya mempersembahkan kepada Anda terjemahan artikel "14 Perpustakaan NodeJS yang Paling Berguna di 2020" .

Dalam artikel ini, penulis membagikan temuannya di antara banyak paket Nodejs.

Mengekspresikan


Kerangka pengembangan web yang cepat, fleksibel, minimalis, mudah dipelajari.

Bagus untuk

  • mengembangkan aplikasi menggunakan tumpukan MEAN (MongoDB, Express, Angular, Node) atau MERN (React bukannya Angular)
  • penanganan mudah berbagai jenis permintaan, seperti GET, PUT, POST dan DELETE
  • penyebaran cepat aplikasi satu halaman, multi-halaman dan hybrid

Jumlah unduhan per minggu

11 juta

Alternatif

Koa, Hapi, Meteor

Cheerio


Dirancang untuk memproses ("parsing") markup, misalnya, HTML, menyediakan antarmuka untuk memanipulasi ("melintasi") struktur data akhir.

	const cheerio = require('cheerio')
	const $ = cheerio.load('<ul id="fruits">...</ul>')

Bagus untuk

  • membuat pemindai web ("pencakar")
  • sintaks dan toolkit mudah, intuitif

Unduhan mingguan

4,2 juta

Alternatif

dalang jsdom

Nodemailer


Dirancang untuk mengirim email.

	const nodemailer = require('nodemailer')

	const testAccount = await nodemailer.createTestAccount()

	const transporter = nodemailer.createTransport({
		host: 'smtp.ethereal.email',
		port: 587,
		secure: false,
		auth: {
			user: testAccount.user,
			pass: testAccount.pass
		}
	})

	const info = await transporter.sendMail({
		from: '"John Smith" <smith@example.com>',
		to: 'foo@example.com, bar@example.com',
		subject: 'hello',
		text: 'hello crazy world',
		html: '<strong>hello crazy world</strong>'
	})

Bagus untuk

  • mengirim pesan melalui SMTP

Unduhan mingguan

980 ribu

Alternatif

sendmail, emailjs

Socket.io


Dirancang untuk mengatur koneksi berbasis acara dua arah secara real time.

	const server = require('http').createServer()
	const io = require('socket.io')(server)
	io.on('connection', client => {
		client.on('event', data => { ... })
		client.on('disconnect', () => { ... })
	})
	server.listen(3000)

Bagus untuk

  • menerapkan analisis langsung, aliran data biner, pengiriman pesan, kolaborasi dengan dokumen

Unduhan mingguan

3 juta

Alternatif

pendorong, soket web

Pemalsu


Dirancang untuk menerima informasi palsu di browser dan "node".

	const faker = require('faker')

	const randomName = faker.name.findName()
	const randomEmail = faker.internet.email()
	const randomCard = faker.helpers.createCard() //

Bagus untuk

  • menguji frontend dalam hal bekerja dengan data sampai pengembangan backend
  • Sejumlah besar metode, termasuk mendapatkan alamat sewenang-wenang, nama perusahaan, basis data, gambar, nama depan, nama belakang, dll.

Unduhan mingguan

1,4 juta

Alternatif

santai

Morgan


Ini adalah middleware untuk mendaftarkan permintaan.

	morgan
	(':method:url :status :res[content-length] - :response-time ms')

	---

	const express = require('express')

	const morgan = require('morgan')

	const app = express()

	app.use(morgan('combined'))

	app.get('/', (req, res) => res.send('hello crazy world'))

Bagus untuk

  • tulis permintaan ke konsol, file, basis data
  • debugging dan "logging"

Unduhan mingguan

2 juta

Http-error


Dirancang untuk membuat kesalahan spesifik untuk menguji aplikasi pada Express, Koa, Connect, dll.

	app.use((req, res, next) => {
		if(!req.user) return next(createError('401', ',    .'))
		next()
	})

Bagus untuk

  • kesalahan emulasi
  • sejumlah besar properti objek Kesalahan tersedia

	expose
	headers
	message
	status
	statusCode

Unduhan mingguan

27 juta

Pengurai tubuh


Ini adalah middleware untuk parsing data.

Mem-parsing permintaan masuk (req.body) sebelum memprosesnya.

	const express = require('express')
	const bodyParser = require('body-parser')

	const app = express()

	//  application/x-www-form-urlencoded
	app.use(bodyParser.urlencoded({ extended: false }))

	//  application/json
	app.use(bodyParser.json())

Bagus untuk

  • interpretasi tubuh permintaan
  • mengandung banyak fungsi yang berguna (mengembang, mengetik, memverifikasi, dll.)

Unduhan mingguan

13 juta

Sekuel


Ini adalah teknologi berbasis janji untuk pemetaan objek-relasional Postgres, MySQL, MariaDB, SQLite, dan Microsoft SQL Server.

Ini mendukung transaksi yang aman (andal), pertukaran data, pemuatan energik dan tertunda, replikasi data, dll.

	const sequelize = new Sequelize(
		'database', 'username', 'password', {
			host: 'localhost',
			dialect: /*   'mysql' | 'mariadb' | 'postgres' | 'mssql' */
		}
	)

Unduhan mingguan

720 ribu

Paspor


Ini adalah middleware yang kompatibel dengan Express untuk otentikasi.

Tugas utama adalah untuk mengotentikasi permintaan menggunakan satu set plugin - strategi.

	passport.use(new LocalStrategy(
		(username, password, done) => {
			User.findOne({ username: username }, (err, user) => {
				if(err) { return done(err) }

				if(!user) { return done(null, false) }

				if(!user.verifyPassword(password)) { return done(null, false) }

				return done(null, user)
			})
		}
	))

Bagus untuk

  • autentikasi
  • integrasi dengan OAuth dan OpenID (masuk dengan akun Facebook, Twitter, dll.)

Unduhan mingguan

810 ribu

Dotenv


Ini adalah modul ketergantungan nol (titik) untuk memuat variabel dari file .env ke process.env.

Membuat variabel terpisah dari kode.

	require('dotenv').config()

	const db = require('db')

	db.connect({
		host: process.env.DB_HOST,
		username: process.env.DB_USER,
		password: process.env.DB_PASS
	})

	//  .env
	DB_HOST = localhost
	DB_USER = root
	DB_PASS=12345

Bagus untuk

  • memuat variabel lingkungan seperti AWS, nama pengguna sql, kata sandi yang diperlukan untuk menyebarkan aplikasi, menghubungkan ke alat lain
  • menyimpan pengaturan lingkungan secara terpisah dari kode

Unduhan mingguan

10 juta

Multer


Middleware untuk memproses multipart / formulir-data - memuat gambar.

	const express = require('express')

	const multer = require('multer')

	const upload = multer({ desc: 'uploads/' })

	const app = express()

	app.post('/profile', upload.single('avatar'), (req, res, next) => {
		// req.file -   'avatar'
		// req.body     
	})

	app.post('/photos/upload', upload.array('photos', 12), (req, res, next) => {
		// req.files -    'photos'
		// req.body      
	})

Bagus untuk

  • pengunduhan cepat multipart / formulir-data file

Unduhan mingguan

92 ribu

CORS


Connect / Express middleware dirancang untuk mendefinisikan aturan kebijakan asal-asal.

	const express = require('express')

	const cors = require('cors')

	const app = express()

	app.use(cors())

	app.get('/products/:id', (req, res, next) => {
		res.json({ msg: '    !' })
	})

	app.listen(80, () => {
		console.log('ok')
	})

Bagus untuk

  • Tugas terkait CORS

Unduhan mingguan

3,7 juta
