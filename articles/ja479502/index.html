地球の歴史の中で最も厳しい氷河期を生き残るには？



漫画「氷河期」では、世界の氷河期の雑多な動物のグループの楽しい冒険を観察します。実際には、条件ははるかに厳しく、生物は存在を拡大するために最善を尽くしました。私たちが地球の歴史の中で最も厳しい氷河期である極低温を思い出すと、明白な疑問が生じます-生物はどのようにして生き残ったのでしょうか？マギル大学（カナダ、モントリオール）の科学者グループは、低温学に関するすべての入手可能な情報をまとめることを決定し、真核生物が「酸素オアシス」を通じて生き残っていることを発見しました。地球上の生命の継続に貢献した物理化学的プロセスは何ですか、そしてこれで溶けた氷河水はどのような役割を果たしましたか？これについては、研究グループの報告から学ぶだけではありません。行け。

研究の基礎


時間として極低温を決定するには、地質年代学的に入れ子になった人形を分解する必要があります：先カンブリア時代-原生代-新原生代-極低温。それは約7億2000万年前に始まり、6億3500万年前に終わりました。 8500万年続いた。

氷河期は総称です。これは、惑星の歴史にはさまざまな時代に発生したものがいくつかあったためです。新生代（2000〜3000万年前）。古生代（3億8億-2億4000万年前）;新原生代（9億-5億9千万年前）;古原生代（25〜22億年前）。

新原生代の第2期である極低温（トリウムが存在する前、その後はエディアカリア）は、最長または最も寒い氷河期とは言えません。そのようなものがヒューロン氷河であり、古原生代の間に約3億年続いた。しかし、極低温はしばしば最も厳しい氷河作用と呼ばれます。これは、この時期、地球のほぼ全面が氷と雪で覆われていたことから、この現象を説明する「雪だるま地球」という理論名が由来したためです。

低温学は比較的最近に公式に認識され、1990年に、層序学*、地質学および地球年代学の問題を地球規模で扱う国際層序に関する委員会によって承認されました。
層序学*は、層状の堆積岩と火山岩の地質年代を決定することを目的とした地質学の一分野です。
この時期に発達したと考えられている化石アメーバ（Arcellinida）は、極低温から現在まで生き残っています。また、最古の海綿化石は極低温期にまでさかのぼります。当時の気候の厳しさにもかかわらず、紅藻、緑藻、脚足類、繊毛虫、渦鞭毛藻、貝のアメーバが発生しました。そして、極低温の終わりに、単細胞藻類と原核生物を食べて従属栄養プランクトンが形成されました。

実際の着氷が極低温の期間中の生活に悪影響を及ぼしたほどの温度ではないことは注目に値します。氷で覆われた海は、酸素が不足しているために、生命にとって不適当になりました。しかし、私たちがここにいるので、それは真核生物が氷河期のすべての困難を克服し、それらの発展を続け、惑星の周りに広がるのを助ける何かがあったことを意味します。私たちが今日調査している研究では、科学者たちは、真核生物は氷河の溶けた水を通じて海で形成された「酸素オアシス」によって助けられたと示唆しました。彼らは地質データの分析を行い、この理論は絶対に真実であるという結論に達しました。

科学者たちは、地球全体で見られる極低温期の多くの連続*で、鉄鉱床が発見されたことに注意しています（IF鉄の形成）。
継承* -土壌または堆積岩の年代順の層（層または層）のシーケンス。動物相の継承の法則によれば、ネアンデルタール人の遺跡はメガロサウルスの遺跡と同じ土壌層には見られないという。これらの2種は数百万年離れた異なる地質時代に住んでいたためです。
これらの調査結果は、氷河作用の期間中に海で発生するプロセスに関する重要な地球化学的データです。極度の氷河作用の際の海洋におけるレドックスプロセスの全体像を把握するために、科学者は、ナミビア、オーストラリア、米国などの近代的な場所に対応する3つの古大陸（古代大陸）から、IFを含む9つの連続からデータを収集しました。

画像2：鉄鉱床（IF）のレドックスプロット。

氷河作用期のレドックスプロセスの全体像を作成するために、鉄鉱床の元素および同位体地球化学データが岩石学スキャンによって収集されました。

鉄鉱床（IF）は、珪質セメントおよび砕屑物の形で不純物を含む細粒ヘマタイト（Fe2O3）です。これらの堆積物の沈殿は、海水中の溶存鉄の酸化が原因で発生します。

IFの地球化学データは明確なシーケンスを示しており、これは氷と岩の接触線への相対的な近接性によって説明でき、これはレドックス反応（OVR）の特定の勾配の存在を示している可能性があります。

したがって、氷と接触した場合、鉄が最も豊富になります（2A）堆積物に鉄が存在するという事実は、氷と岩の接触領域での破片の堆積率が高いにもかかわらず、これらの条件下での鉄の酸化の急速なプロセスを示しています。

鉄と同様に、マンガン（Mn）はOVRに敏感な金属であり、無酸素条件下で溶解し、O 2の存在下で酸化物を形成します。マンガン酸化物は、海水中の第一鉄の存在下で急速に還元溶解します。したがって、酸化マンガンによる富化は、酸素化水の流れと関連している可能性があります。

海水のOVR変動は、セリウム（Ce）などのIFに存在する希土類元素の地球化学によっても確認されます。セリウムは、酸素精製によりOVRの影響を受けない希土類元素と比較して、酸素条件下で海水中で枯渇し、酸素海水中でセリウム異常*（Ce n / Ce n * <1）を引き起こします。
セリウム異常*は、セリウム（Ce）の濃度が他の希土類元素と比較して岩石で減少または増加する地球化学の現象です。
この観察はまた、着氷帯の近くに酸素が豊富な海水の存在を確認します。

研究された鉄鉱床からのFeの同位体組成は、極低温期の海洋環境におけるSIRのダイナミクスの追加のアイデアを提供します。 IFSは、同位体の非常に広い範囲（有する2Dを極めて低いδで）56個のFe 値（最大-1.8‰）と異常に高い値（2.7‰まで）。 δの印象的な負の値56のFe（δの平均値56 Feが= -0.57‰であり; n = 14） -岩接触ゾーン氷からのサンプルにおいてのみ見出されました。他のすべての試料は、δが正の値有する56 δの平均値：鉄56Fe = 1.1‰、n = 21（氷河近位ゾーン）; δ56 Feの平均値= 1.5; n = 46（氷河遠位ゾーン）。これは、氷と岩の接触帯から離れると値が大きくなることを示唆しています。

上記の結果（金属、Ce異常、Fe同位体のレドックス濃縮）を要約すると、氷床のすぐ近くで海水の酸素化（酸素濃縮）が増加する傾向が見られます。したがって、O 2の発生源は氷床自体であり、これは雪の圧縮の結果として形成されたものです。このプロセスの結果として、氷の中に気泡が「捕獲」されました。

溶けた氷河水は、氷河の上層から、氷河の底部から、そして氷棚の底部から、地熱フロー、圧力および摩擦加熱を介して形成できます。その結果、氷河下の環境を提供する酸素に富んだ融水が得られます。


画像3：氷河下の無酸素環境における酸素に富む基礎水の洗い出し。

酸素が豊富な溶融水と基礎水を混合すると、鉄鉱床（IF）に関連する地球化学プロセスとその場所が完全に説明されます。その結果、鉄鉱床は氷河下の「酸素オアシス」の理論の直接の証拠です。

実際、非常に厳しい気候条件にもかかわらず、地球上の生命は存在しなくなっただけでなく、発展し続けました。極低温からの化石の分析は、この期間の間に、古細菌、オピストコンツおよびアメーバ動物を含む多くの真核生物が形成されたことを示した。

これらの真核生物の多くは好気性菌です。なぜなら、融氷水による氷河下環境の酸素化は生命を維持するための非常に重要な基盤だからです。酸素のオアシスは、南極の凍った郊外の現代の観察によって証明されるように、何キロにも及ぶかもしれません。そのため、そのような生息地は大きいだけでなく、他の酸素化の理論で言及されている氷河割れ目よりも安定している可能性があります。

O 2の利用が複雑な多細胞生命の発達に不可欠であることを考慮すると、酸素オアシスは初期の動物にとって重要な避難所*であったと考えることができます。
Refugium *は、生物種または種のグループが生き残った、またはそれらにとって不都合な期間の地質学的時間を経験している生息地エリアです。避難所では、種は生き残ることができ、そこから広がることさえできると考えられています。
分子時計の現代の研究*は、動物の多細胞性が極低温の前でさえ発達したことを示しています。
分子時計* -生体分子のモノマーの進化的に重要な変化がほぼ一定の速度で発生するという仮説に基づいて、系統発生（種間の進化的関係）イベントを年代測定する方法。
スポンジは動物の基本的な宝物と見なされていることを考えると、最も初期の動物はおそらく底生動物（下）でした。これに基づいて、科学者は氷河下の海洋環境の酸素化が底生のマクロ動物相の発達と拡大における重要なプロセスであったことを示唆しています。さらに、溶けた真水の供給は、塩分の程度を減らすのにも役立ちます。これは、初期のスポンジにとって重要です。現代のスポンジの生息地は非常に広く多様ですが、彼らの祖先の生物学的特性はまだ謎です。したがって、海面の氷結によって形成された超塩水環境で生活できるかどうかは難しい。

エピローグ


低温学は、私たちの惑星の生物種にとって最も簡単な時期からほど遠いものでした。それにもかかわらず、生命は状況からの脱出方法を見つけました-新鮮な水を溶かし、酸素で飽和させ、海と混ぜ、それが生存と開発さえも許容できる生息地を生み出しました。

最も厳しい氷河期における真核生物の生存の秘密は、遠い過去の写真のように、数百万年前に行われたプロセスに関する貴重な地球化学的情報を含んだ鉄鉱床によって助けられました。

研究者たち自身によると、彼らの研究により、極低温での生物の生存の謎を解明するだけでなく、それまで約10億年にわたって失われていた地質試料中の鉄鉱床の再発についても説明することができました。

もちろん、酸素は生存にとって非常に重要ですが、これに加えて、食物も必要です。科学者たちは、真核生物が極低温中に何を食べたかを知るために研究を続けるつもりです。
