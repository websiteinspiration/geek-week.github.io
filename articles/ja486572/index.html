組み込みのスライダーで画像ギャラリージェネレーターを作成する



こんにちは、友達！

導入の代わりに（問題ステートメント）


これは、他のすべての人のスライダー（のようなネットワーク上の既製ソリューションの研究を始めbxslider、owlcarouselとスリック）。いつか私はこれらのツールを使った詳細なチュートリアルを書きます（甘い夢）。スライダーを書きたいという要望がありました。しかし、すぐに（Habréに関するいくつかの記事を読んだ後も含めて）、スライダーは弱虫のためのものであることに気付きました。もっと根本的なものが必要です。

その結果、次のタスクを思いつきました：組み込みのスライダーを備えたアダプティブギャラリージェネレーターを作成すること。

条件：

  • （ハードドライブのどこからでも）任意の数の画像をアップロードする機能。
  • ギャラリーはアップロードされた画像で構成され、マークアップはHTML5セマンティクスに従って「オンザフライ」で形成されます。
  • ギャラリーは、解像度の異なる画面でも同じように表示されます。
  • 画像をクリックすると、スライダーが生成されます。
  • スライダーを生成すると、背景が暗くなります。
  • クリックした画像が最初のスライドです。
  • スライド切り替えは、DOMを介して実装されます。
  • スライドはスムーズに切り替わります。
  • ボタンとキーボードを使用してスライド切り替えを制御する機能。
  • 現在のスライドとボタンをクリックするだけでなく、キーボードを使用してギャラリーに戻る機能。
  • 純粋なJavaScript（JSによるすべてのマークアップ）。
  • 最小コード。

さあ、行きましょう（ガガーリンが言ったように、宇宙へ行きます）。

マークアップは次のようになります。
<div class="wrap">
    <input type="file" multiple accept="image/*">
    <button>generate gallery</button>
</div>

興味深いのは、おそらく、入力タグの複数属性と受け入れ属性です。最初の属性では複数のファイルをアップロードできます。2番目の属性では、ダウンロードできるファイルのタイプにフィルターを設定します。この場合、acceptの値は「image / *」です。つまり、アップロードできるのは画像（任意）だけです。

ボタン作成コード：
function generateButtons() {
    if (document.querySelector(".buttons") == null) {
        //    
        let buttons = document.createElement("div");
        buttons.className = "buttons";
        slider.appendChild(buttons);

        //   
        let leftButton = document.createElement("button");
        leftButton.className = "button left";
        let leftImg = document.createElement("img");
        leftImg.src = "https://thebestcode.ru/media/sliderGenerator/buttons/left.png";
        leftButton.appendChild(leftImg);
        buttons.appendChild(leftButton);
        leftButton.addEventListener("click", () => changeSlide("-"));

        //   
        let rightButton = document.createElement("button");
        rightButton.className = "button right";
        let rightImg = document.createElement("img");
        rightImg.src = "https://thebestcode.ru/media/sliderGenerator/buttons/right.png";
        rightButton.appendChild(rightImg);
        buttons.appendChild(rightButton);
        rightButton.addEventListener("click", () => changeSlide("+"));

        //    
        let closeButton = document.createElement("button");
        closeButton.className = "button close";
        let closeImg = document.createElement("img");
        closeImg.src = "https://thebestcode.ru/media/sliderGenerator/buttons/close.png";
        closeButton.appendChild(closeImg);
        buttons.appendChild(closeButton);
        closeButton.addEventListener("click", () => generateSlider(figure));
    } else {
        //   ,  
        let buttons = document.querySelector(".buttons");
        slider.removeChild(buttons);
    }
}


スライドの切り替えは、「+」または「-」というパラメーターとしてそれぞれ渡される関数changeSlideを使用して実装されます。
function changeSlide(e) {
    //    
    for (i = 0; i < figures.length; i++) {
        figures[i].style.opacity = 0;
    }
    if (e == "-") {
        //      ,    
        if (figure == figures[0]) {
            figure = figures[figures.length - 1];
        } else {
            figure = figure.previousElementSibling;
        }
    } else if (e == "+") {
        //      ,    
        if (figure == figures[figures.length - 1]) {
            figure = figures[0];
        } else {
            figure = figure.nextElementSibling;
        }
    }
    //    
    figure.style.opacity = 1;
}

キーボードを使用してスライドを切り替えてギャラリーを閉じる機能を追加します。
document.addEventListener("keydown", e => {
    //  
    if (e.keyCode == 37 || e.keyCode == 189) {
        changeSlide("-");
    //  
    } else if (e.keyCode == 39 || e.keyCode == 187) {
        changeSlide("+");
    // esc
    } else if(e.keyCode == 27) {
        generateSlider(figure);
    }
});
