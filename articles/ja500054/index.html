別の自転車：HTTPリクエストを処理するためのシンプルなライブラリ

みなさん、こんにちは。

APIの操作は、HTTPリクエストを操作しないと完了しません。誰かが気にせず、グローバル配列$ _GET、$ _POST、および$ _REQUESTを使用します。率直に言って、私は自分でそれを行いましたが、少し前までは、使いやすさのために何らかのラッパーが必要だという考えに戸惑いました。そのようなライブラリはすでにあるかもしれませんが、Bitrix APIを除いて（おそらく見栄えが悪いので）、まだ見つけていません。そのため、自分で作成することにしました。さらに、コードを操作するときにライブラリを使用する方がはるかに快適であることに同意する必要があります。

これまでのところ、ライブラリはGETメソッドとPOSTメソッド、およびphp：//入力から取得したjson構造で機能します。彼はまた、httpsをチェックしてヘッダーを取得する方法も知っています。

接続と通話


ライブラリを接続するには、Composerを使用します。

composer require ramapriya/http-request

require __DIR__ . '/vendor/autoload.php';

use Ramapriya\Request\Request;

メソッド検証


これでライブラリを使用できます。たとえば、リクエストのタイプを確認する場合は、GetRequestMethod（）メソッドを呼び出します。

$method = Request::GetRequestMethod();

switch($method) {
    case 'GET':
        //  
        break;
    case 'POST':
        //  
        break;
}

ただし、通常、メソッドのタイプは事前にわかっているため、追加の変数を作成しないようにするには、getおよびpostのチェックにメソッドを使用するだけで十分です-isGet（）およびisPost （）、それぞれ：

if(Request::isPost() !== false) {
   //  
} else if(Request::isGet() !== false) {
   //  
}

キーを受け取る


クエリパラメータ（値ではなくキー自体）のリストを取得する必要がある場合があります。これには、GetとPostの2つのメソッドもあります。

$GetParams = Request::GetParams();

if(in_array($needle, $GetParams)) {
   //  
}

$postParams = Request::PostParams();

if(in_array($needle, $postParams)) {
   //  
}

値の取得


そしてもちろん、パラメータ値を取得するためのいくつかのメソッドがありました-Get（）とPost（）。最も興味深いのは、個々のパラメーターと配列全体（ちなみに、オブジェクトに変換されます-理由は尋ねないで、オブジェクトを操作するのが好きなだけです）の両方を取得できることです。

if(!empty(Request::Get('user'))) {
   $user = Request::Get('user');
}

$request = Request::Post();
if(Request::isPost() && !empty($request)) {
   //  
}

生のリクエスト（php：//入力）


また、php：//入力の操作方法についても説明します。このisRaw（） -未加工のリクエスト、オブジェクトに変換されたjson文字列を返すRaw（）、およびリクエストキーを返すRawParams（）をチェックします。Sendpulse APIを使用していたときに、次のように記述しなければならなかったことを覚えています。

$rawRequest = file_get_contents('php://input');
$request = json_decode($rawRequest);

if(!empty($request)) {
   //  
}

もちろん、1つの変数を省略できます。
$request = json_decode(file_get_contents('php://input'));

しかし、あなたは認めなければなりません、それはかなり混乱しています。Raw（）およびisRaw（）

メソッドを使用すると、コードはすでにより美的な喜びをもたらします。

if(Request::isRaw() === true) {
$request = Request::Raw();
}

ヘッダーを操作する


ライブラリには、ヘッダーを操作するためのいくつかのメソッドもあります

。GetAllHeaders（） -すべてのヘッダーを取得します。
$headers = Request::GetAllHeaders();

GetHostName（） -ホスト名を取得する
$domain = Request::GetHostName();

isHttps（） -httpsを確認する

if(Request::isHttps() !== true) {
   die("Application works only with HTTPS!");
}

GetUserAgent（） -ユーザーエージェントを取得しています。これは誰かにとって重要な場合があります。

$userAgent = Request::GetUserAgent();
