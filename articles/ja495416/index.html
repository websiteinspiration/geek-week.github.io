pipeplot-コンソールでのパイプからのデータストリームのインタラクティブな視覚化

コンソールを離れたくない人のために、彼らはそのアスキーグラフィックを愛し、パイプからグラフを描くためのユーティリティが登場しました。猫のアニメーションと例の下。

ping




ping ya.ru | grep --line-buffered time | sed -u -e 's#.*time=\([^ ]*\).*#\1#' | pipeplot --min 0

covidAPIを介してコロナウイルスを追跡できます


while true; \
    do curl -s https://coronavirus-19-api.herokuapp.com/all \
    | jq '.deaths'; \
    sleep 60; \
done \
| pipeplot --color 1 --direction left

jsonを解析するためにjqをインストールする必要があります

グラファイトを通して500


while true; \
do \
    curl -s 'http://graphite/render?target=my_app_rps_error&format=json&from=-5min&until=now' \
    | jq -c '.[0].datapoints[-1]'; \
    sleep 5; \
done \
| sed -u s/null/0/ \
| stdbuf -oL uniq \
| stdbuf -oL jq '.[0]' \
| pipeplot

バッファリング


多くのコンソールユーティリティはstdoutをバッファします。このため、シリアルパイプが壊れます。一部のユーティリティはバッファリングをサポートしています：sed -u、grep --line-buffered。それ以外の場合は、ユニバーサルメソッドstdbuf -oLおよびunbufferを使用できます。

利用可能なオプション


COLOR --color-列の色、パレットの数はXtermを使用します
--symbol SYMBOL-シンボルを表示する列はシールをサポートします
--scale all {、} window-自動スケーリングの最小値と最大値を設定します-all-すべてのデータを考慮しますwindow-可視のみ
--direction {左、右} -方向
--min MIN --max MAX-自動スケーリングと組み合わせて、最小値と最大値を修正します

取り付け


pip install pipeplot
