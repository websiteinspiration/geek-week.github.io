E com a expansão da análise, o valor dos dados coletados e acumulados pelas empresas está aumentando constantemente. Além disso, eles costumam falar sobre crescimento explosivo e exponencial no volume de dados corporativos gerados. Note-se que 90% de todos os dados foram criados nos últimos dois anos. 


O crescimento nos volumes de dados é acompanhado por um aumento em seu valor.


Os dados são criados e usados ​​por sistemas de análise de big data, Internet das Coisas (IoT), inteligência artificial etc. Os dados coletados são a base para melhorar a qualidade do serviço ao cliente, tomada de decisão, apoiar as atividades operacionais das empresas, para diversas pesquisas e desenvolvimento.

Plataformas de gerenciamento de dados: por que elas são necessárias e qual o seu papel?


Não é de surpreender que os problemas de gerenciamento de dados estejam se tornando uma prioridade para as empresas, afetando diretamente seu trabalho. Para resolvê-los, às vezes é preciso superar dificuldades como heterogeneidade de sistemas, formatos de dados, métodos de armazenamento e uso, abordagens de gerenciamento no “zoológico” de soluções que foram introduzidas em momentos diferentes. 


O resultado dessa abordagem unificada é a fragmentação de matrizes de dados armazenadas e processadas em diferentes sistemas, procedimentos diferentes para garantir a qualidade dos dados. Esses problemas típicos aumentam os custos trabalhistas e financeiros ao trabalhar com dados, por exemplo, ao receber estatísticas e relatórios ou ao tomar decisões de gerenciamento. 

O modelo de negócios de gerenciamento de dados deve ser personalizado, adaptado às necessidades, tarefas e objetivos da empresa. Não existe um sistema automatizado único, plataforma de gerenciamento de dados, que cubra todas as tarefas. No entanto, sistemas de gerenciamento de dados modernos, abrangentes, flexíveis e escalonáveis ​​geralmente fornecem software versátil de gerenciamento e armazenamento de dados. Eles incluem as ferramentas e serviços necessários para o gerenciamento eficaz de dados. 

Os desenvolvimentos mais recentes permitem que as empresas repensem o gerenciamento de dados em toda a organização, tenham uma idéia clara de quais dados estão disponíveis, quais políticas estão associadas a eles, onde e quanto tempo os dados são armazenados e, finalmente, possibilitam o fornecimento oportuno das informações certas às pessoas certas. Estas são soluções que expandem os recursos das empresas e permitem: 

  • Gerencie arquivos, objetos, dados de aplicativos, bancos de dados, dados de ambientes virtuais e em nuvem, acesse vários tipos de dados.
  • Usando as ferramentas de orquestração e automação, mova os dados para o local em que são armazenados com mais eficiência - na infraestrutura de armazenamento primário e secundário, no datacenter do fornecedor ou na nuvem.
  • Use recursos abrangentes de proteção de dados.
  • Forneça integração de dados.
  • Obtenha análises operacionais dos dados. 

A plataforma de gerenciamento de dados pode ser construída com base em vários produtos de software ou representa um único sistema unificado. Uma plataforma abrangente fornece gerenciamento unificado de dados em toda a infraestrutura de TI, incluindo backup, restauração, arquivamento, gerenciamento de snapshots e relatórios de hardware.

Essa plataforma permite implementar uma estratégia de várias nuvens, expandir o data center para a nuvem, migrar rapidamente para a nuvem, usando a possibilidade de substituir o equipamento e introduzir as opções mais econômicas de armazenamento de dados.

Algumas soluções podem arquivar dados automaticamente. E com a ajuda da inteligência artificial, eles podem detectar que “algo deu errado” e tomar medidas corretivas automaticamente ou notificar o administrador, além de identificar e suprimir ataques de vários tipos. A automação de serviços ajuda a otimizar as operações de TI, permite liberar a equipe de TI, minimizar erros devido a fatores humanos e minimizar o tempo de inatividade. 

Quais qualidades devem ter uma moderna plataforma de gerenciamento de dados e onde essas soluções são aplicadas na prática?

A abordagem “uma solução para todos” não funciona com plataformas de gerenciamento de dados. Cada empresa possui seus próprios requisitos de dados, depende do tipo de negócio, experiência de trabalho etc. Uma plataforma universal deve, por um lado, fornecer personalização para trabalhar com dados em uma determinada empresa e, por outro, ser independente das especificidades do setor aplicado, escopo um produto construído com base em seu ambiente de informações.

As empresas que trabalham efetivamente com dados, em geral, obtêm maior sucesso em comparação aos concorrentes, comercializam rapidamente produtos e serviços, entendem melhor as necessidades do público-alvo e podem responder rapidamente às mudanças na demanda. As plataformas de gerenciamento de dados fornecem a capacidade de "limpar" dados, obter informações relevantes e de alta qualidade, transformar dados e avaliar estrategicamente os dados da empresa. 

Um exemplo de plataforma universal para a construção de sistemas corporativos de gerenciamento de dados é o Russian Unidata, criado com base em software de código aberto. Ele oferece ferramentas para criar um modelo de dados e meios de expandir a funcionalidade para integração em vários ambientes de TI e sistemas de informações de terceiros: da manutenção de recursos materiais e técnicos ao processamento seguro de grandes quantidades de dados pessoais.

Ele fornece automação de tarefas como coleta, acumulação, limpeza, comparação, consolidação, controle de qualidade, disseminação de dados e ferramentas para automatizar os sistemas de tomada de decisão. 

Plataformas de gerenciamento de dados (DPM) em publicidade e marketing 


Em publicidade e marketing, o conceito de uma plataforma de gerenciamento de dados (DMP) tem um significado mais restrito. Essa é uma plataforma de software que, com base nos dados coletados, permite que as empresas determinem segmentos de público-alvo para direcionar anúncios a usuários específicos e o contexto de campanhas de publicidade on-line. Esse software é capaz de coletar, processar e armazenar qualquer tipo de dados em sala de aula e também pode usá-los por meio de canais de mídia familiares.


De acordo com a previsão do Market Research Future (MRFR), o mercado global de plataformas de gerenciamento de dados (DMP) pode chegar a US $ 3 bilhões até o final de 2023, com um crescimento médio anual de 15%, e em 2025 seu volume será superior a US $ 3,5 bilhões.

A solução permite criar uma descrição estendida do perfil dos segmentos de público-alvo em uma plataforma integrada às ferramentas de marketing. O DMP une soluções e serviços do Grupo Mail.ru no campo do marketing omnichannel e trabalha com o público. Os clientes poderão armazenar, processar e estruturar seus próprios dados anônimos, além de ativá-los em comunicações publicitárias, aumentando a eficiência comercial e de marketing. 

Gerenciamento de dados na nuvem


Outra categoria de soluções de gerenciamento de dados são as plataformas em nuvem. Em particular, o uso de uma solução moderna de proteção de dados como parte do gerenciamento de dados na nuvem evita possíveis problemas - de ameaças à segurança a problemas com migração de dados e degradação do desempenho, além de solucionar os desafios de transformação digital da empresa. Obviamente, as funções desses sistemas não se limitam à proteção de dados.


Os recursos da plataforma de gerenciamento de dados baseados na nuvem do Gartner: alocação, automação e orquestração de recursos; gerenciamento de solicitações de serviço; gerenciamento de alto nível e aplicação de políticas; monitoramento e medição de parâmetros; suporte para ambientes com várias nuvens; otimização e transparência de custos; otimização de capacidades e recursos; Migração na nuvem e tolerância a desastres (DR) Gerenciamento de nível de serviço; segurança e identificação; Atualizações de configuração de automação.

O gerenciamento de dados na nuvem deve fornecer um alto nível de disponibilidade, controle, automação do gerenciamento de dados nos datacenters, na rede e na nuvem. 


O Cloud Data Management (CDM) é uma plataforma usada para gerenciar dados corporativos em vários ambientes de nuvem, levando em consideração as abordagens privadas, públicas, híbridas e com várias nuvens.

Um exemplo dessa solução é a Veeam Cloud Data Management Platform. De acordo com os desenvolvedores do sistema, ele ajuda as organizações a mudar sua abordagem de gerenciamento de dados, fornece gerenciamento inteligente automatizado de dados e sua disponibilidade em qualquer aplicativo ou infraestrutura de nuvem.


O gerenciamento de dados em nuvem na Veeam é considerado parte integrante do gerenciamento inteligente de dados, tornando-o acessível às empresas de qualquer lugar. 

A Veeam Cloud Data Management Platform permite atualizar backups e abandonar sistemas legados, acelerar a implantação de nuvem híbrida e a migração de dados, automatizar a segurança dos dados e a conformidade regulamentar.

Como você pode ver, as plataformas modernas de gerenciamento de dados representam uma classe de soluções bastante extensa e diversificada. Talvez eles estejam unidos por uma coisa: foco no trabalho eficaz com dados corporativos e na transformação de uma empresa ou organização em uma empresa moderna orientada a dados.

As plataformas de gerenciamento de dados são uma evolução necessária do gerenciamento tradicional de dados. Cada vez mais organizações estão transferindo dados para a nuvem. Em um número crescente de configurações locais e na nuvem, surgem novos problemas que precisam ser resolvidos precisamente do ponto de vista do gerenciamento de dados. Gerenciamento de dados em nuvem - uma abordagem atualizada, um novo paradigma que estende o gerenciamento de dados para suportar novas plataformas, aplicativos e cenários de uso.

Além disso, de acordo com um estudo do Veeam Cloud Data Management Report de 2019, as empresas planejam integrar a nuvem, a nuvem híbrida, a análise de big data, a inteligência artificial e a Internet das Coisas mais profundamente. Espera-se que essas iniciativas digitais tragam benefícios significativos para as empresas.

As empresas estão adotando rapidamente as tecnologias de plataforma de dados e estão prontas para usar a nuvem para realizar cargas de trabalho analíticas, mas muitas enfrentam problemas ao tentar usar todos os seus dados para obter melhores resultados de negócios, dizem analistas da 451 Research. As plataformas de gerenciamento de dados mais recentes ajudarão as empresas a navegar nos processos complexos de trabalhar com dados em várias nuvens, usar ferramentas de gerenciamento de dados e realizar suas análises, não importa onde esses dados estejam localizados.
