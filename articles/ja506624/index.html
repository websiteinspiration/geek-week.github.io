Linux Foundation 2020: Linux, Cloud Engineer Bootcamp




クラウドテクノロジーを使用する場合、GNU / Linuxの知識はこれまで以上に求められています。MicrosoftAzureでも、GNU / LinuxはWindowsよりも人気があります。特に重要なのは、人々がこの無料のシステムで作業する方法と場所を学ぶことです。そして当然のことながら、Linux Foundationが登場します。ZDNetによると、The Linux FoundationはIT認定のパイオニアであり、2014年に最初のリモート認定プログラムを提供しています。これ以前は、トレーニングセンターの外でIT証明書を取得することはほとんど不可能でした。Linux Foundationは、信頼性が高く実証済みのリモート学習手順を確立しています。これにより、トレーニングが大幅に簡略化され、パンデミックの日に、どこにも行かずに認定を取得したいプロフェッショナルにとって、特に重要です。

humanID:





Linux.comは、インターネットでの作業のセキュリティと快適さを向上させるために設計された新しいプロジェクトについて語っています。毎日、何十億もの人々がFacebookログインなどのソーシャルアカウントを使用して、インターネット経由でアプリケーションにアクセスしています。このシステムの主な欠点は、実際のユーザーとボットを区別できないことです。ハーバード大学ソーシャルインパクトファンドの受益者である非営利団体のhumanIDは、革新的なアイデアを思い付きました。ワンクリックで匿名ログインを開発することです。これはソーシャル入力の代わりとして機能します。 」humanIDの助けを借りて、誰もがプライバシーを放棄したり、データを販売したりせずにサービスを使用できます。ボットネットは自動的に除外されますが、アプリケーションは侵入者やトロールを簡単にブロックして、より市民のデジタルコミュニティを作成できます」と、humanIDの共同創設者であるBastian Purrerは言います。

Ubuntu Touchで利用可能なPineTabタブレットが注文可能




OpenNETレポート：「Pine64コミュニティは、UBportsプロジェクトからUbuntu Touch環境に同梱されている10.1インチPineTabタブレットの注文を受け付け始めました。オプションとして、PostmarketOSおよびArch Linux ARMアセンブリが提供されます。タブレットの価格は100ドルで、120ドルで、通常のラップトップのようにデバイスを使用できる取り外し可能なキーボードが付いています。配達は7月に始まる予定です。」

出版物によると、主な特徴：

  1. 解像度が1280×800の10.1インチHD IPS画面。
  2. CPU Allwinner A64（64ビット4コアARM Cortex A-53 1.2 GHz）、GPU MALI-400 MP2;
  3. メモリ：2 GB LPDDR3 SDRAM RAM、内蔵64 GB eMMCフラッシュ、SDカードスロット。
  4. 2つのカメラ：背面5MP、1/4インチ（LEDフラッシュ）および前面2MP（f / 2.8、1 / 5インチ）;
  5. Wi-Fi 802.11 b / g / n、シングルバンド、ホットスポット、Bluetooth 4.0、A2DP;
  6. 1つのフルUSB 2.0タイプA、1つのマイクロUSB OTG（充電に使用可能）、ドッキングステーション用のUSB 2.0ポート、HDビデオ出力。
  7. M.2拡張機能を接続するためのスロット。SATASSD、LTEモデム、LoRaおよびRTL-SDRを備えたモジュールがオプションで提供されます。
  8. バッテリーLi-Po 6000 mAh;
  9. サイズ258mm x 170mm x 11.2mm、キーボード付きオプション262mm x 180mm x 21.1mm。重量は575グラム（キーボードは950グラム）。

シロビキがあなたのホスティング業者に来た場合の対処




HabréのRUVDSホスティング業者のブログで、かなり標準的ではない脅威からデータを保護することについて、小さいながらも興味深い記事が公開されましたが、残念ながらそれほど信じられないほどではありません。著者は序文に次のように書いています。「サーバーをレンタルした場合、サーバーを完全に制御することはできません。これは、特別に訓練された人々がいつでもホスティング業者に来てあなたのデータを要求できることを意味します。また、要件が法律で正式に定められている場合、ホスティング業者はそれらを返還します。あなたは本当にあなたのウェブサーバーのログやユーザーデータが他の誰かに到達することを望んでいません。理想的な防御を構築することはできません。ハイパーバイザーを所有し、仮想マシンを提供するホスティング業者を防御することはほとんど不可能です。しかし、おそらくリスクをわずかに減らすことが可能になるでしょう。」
