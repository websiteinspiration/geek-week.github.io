私のトップ無料開発者ツール

これは、dev.toからの記事の翻訳です。著者のFayaz Ahmedが、お気に入りの開発者ツールのリストを公開しています。



私はTvflix、GSON、Appy Devなどのプロジェクト、および多くの異なるマイクロプロジェクトを開発しているので、自分の作業用の無料ツールを見つけることに興味があります。

これは私の個人的なリストですが、他の成功したツールで補うことができます。ここに私のお気に入りのいくつかがあります。

フロントエンド/バックエンドをデプロイするためのサービスとプラットフォーム


1. Netlify Netlifyを
使用して、最新の静的Webサイトを展開します。 CDNインフラストラクチャ、継続的な展開、ワンクリックHTTPS、および必要なサービスを利用できます。無料で始めましょう。

2. Vercel（旧Zeit）
Vercelは、Webサイトを展開する最も簡単な方法です。自動SSLおよびグローバルCDNインフラストラクチャを使用して、構成不要のWebプロジェクトをホストします。

3. サージ
転送Webプロジェクトは、迅速、簡単、かつ安全でなければなりません。 Surgeは、CLIから直接フロントエンド開発者向けの静的Webパブリッシング機能です。

4. Firebase
GoogleのNetlify / Zeitの代替。 Firebaseは、高品質のアプリケーションをすばやく開発してビジネスを成長させるのに役立つモバイルプラットフォームです。

5. Azure静的Webアプリ
今日から、静的Webアプリを使用して、Azureで単一ページアプリをホストする機会があります。

6. GitHub
リポジトリから直接公開されたGitHub Pages Webサイト。変更がライブバージョンに反映されるように編集して実行するだけで十分です。

7. GitLab Pages
GitHubと同様に、GitLab Pagesサービスを使用すると、静的なWebサイトをリポジトリからGitLabに直接公開できます。

8. 曲技飛行
頑丈な静的ホスティング。Hugo、Jekyll、Gatsby、Gridsomeなどの静的サイトジェネレーターと連携して機能します。

9. グリッチ
おそらくこれは、私が今日最も頻繁に使用するツールの1つです。Node.jsに基づいてアプリケーションを作成およびホストするためのシンプルで強力な無料サービス。

10. Heroku
Herokuは、Glitchのより高度なバージョンであり、開発者がアプリケーションを作成して実行し、クラウドで完全に管理できる「サービスとしてのプラットフォーム」（PaaS）です。

自動化ツール


1. Cron-job.org
Cron-jobは無料のオープンソースサービスであり、cron形式で無制限の数のタスクを実行できます。必要なのは、指定した頻度でサービスに接続するために使用されるAPIまたはWebhookのメールアドレスを使用することだけです。

2. n8n
このオープンソースツールはZapierの代替ツールです。このサービスは無限の可能性を提供し、同期に使用できるツールとアプリケーションのリストを常に更新します。独自のサーバーでn8nをホストすることもできます。

3. インテグロマット
Integromatは、独自の特性を持つ手動プロセスを自動化するための、使いやすく効果的なツールです。プログラミングのスキルがなくても、お気に入りのアプリケーション、サービス、デバイスを相互に同期できます。

4. IFTTTを
試すIFTTT-お気に入りのアプリケーションやデバイスをさらに活用するための最も簡単で無料の方法です。家事はそれほど注意を必要とせず、仕事はさらに生産的になり、あなたはプライバシーを守り、あなたのデータは安全になります。同期条件ですべてがうまく機能すると確信しています。

5. Pipedream
Pipedreamは、開発者がデータ処理に関連するプロジェクトを作成および公開できるプラットフォームです。何千ものソースコード例と完成したプロジェクトを含むプラットフォームでプロジェクトを開き、フォークします。

コードレスツール


1. NoCodeAPI
フロントエンド開発者がコードや専用サーバーを追加せずにアプリケーションをサードパーティのサービスに接続する最も簡単な方法。 NoCodeAPIは、Googleシート、Airtable、Googleアナリティクス、Twitter、Instagram、XML、RSS、JSON、Slack、支払いシステム、および50以上のサービスで動作します。

2. GlideApps
Glideは、コードを必要とせずに動的テーブルをユーザーフレンドリーなアプリケーションに変えます。テーブルを選択するか、テンプレートから始めて、アプリケーションの設定を構成し、それを他のユーザーとすぐに共有します。今日は無料でお試しください！

3. エアテーブル
Airtableは動的テーブルのように機能しますが、同時にコンテンツを整理するためのデータベース機能を提供します。

4. Bubble
Bubbleは、Webアプリケーションを作成する新しい方法を紹介します。このプログラミングツールは、コードを操作する必要がなく、ポイントアンドクリック形式で動作します。Bubbleは、クラウドプラットフォームですべてのアプリケーションをホストします。

5. マテリアルアイコンジェネレーター
アプリケーションのマテリアルデザインのスタイルでSVGアイコンを生成します。オープンソースのサービス。

SEO /インターネットマーケティング/プロモーション/法的サポートのためのツール


1. メタタグWebページ上のソーシャルネットワークのすべてのメタタグとタグを
生成してプレビューします。

2. Webコードツール
スキーム、タグ、および多数の繰り返しコードスニペットを作成します。

3. カーボン
カーボンは、ソースコードのビジュアルイメージを作成および公開する最も簡単な方法です。

4. Codeimg
ソースコードのビジュアルイメージを作成して公開します。

5. GetTerms
Webサイトのユーザー同意書の基本テキストとプライバシーポリシーを生成します。
