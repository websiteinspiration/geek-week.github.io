TypeScriptのよくある10の質問（短い回答付き）


1.除外タイプは何ですか？


回答：これは、元のタイプから除外されるプロパティを指定できる新しいタイプです。


例：


type Person = { name: string; age: number; location: string; };
type QuantumPerson = Omit<Person, 'location'>; //   
QuantumPerson = { name: string; age: number; };

2.キーワード「宣言」はいつ使用する必要がありますか？


回答：TypeScriptプロジェクトで宣言されていないJavaScriptライブラリを使用する場合。


例：


declare const libraryName;

9.「不明」というキーワードはいつ使用されますか？


回答：anyキーワードを使用したくない場合、または事前に正確なタイプがわからないが、後で割り当てたい場合。


例：


let person: unknown = 'John';
if (typeof person === string) {
  let name: string = person;
}

10.「.map」ファイルとは何ですか。どのように、そしてなぜそれを使用するのですか？


回答：マップファイルは、デバッグ時に使用できるソースマップファイルです。tsconfig.jsonファイルでsourceMapコンパイラオプションをtrueに設定することで生成できます。


例：


{
"compilerOptions": {
    ...
    "sourceMap": true,
  }
}
