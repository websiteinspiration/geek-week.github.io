Chrome 81リリース




GoogleがChrome 81 Webブラウザのリリースを発表したと同時に、Chromeのベースとして機能する無料のChromiumプロジェクトの安定版リリースが利用可能であるとOpenNETのレポートが報告しています。したがって、この出版物は、ChromeブラウザーがGoogleロゴの使用、クラッシュ時の通知システムの存在、オンデマンドでのFlashモジュールのロード機能、保護されたビデオコンテンツを再生するモジュール（DRM）、アップデートを自動的にインストールしてRLZパラメーターを検索するときに送信するシステムによって区別されることを想起します。当初、Chrome 81は3月17日にリリースされる予定でしたが、SARS-CoV-2コロナウイルスの流行と開発者の在宅勤務への移行により、リリースは延期されました。Chrome 82の次のリリースはスキップされます。Chrome83のリリースは5月19日に予定されています。いくつかの革新：

  1. FTPサポートは無効です。
  2. すべてのユーザーに対して、タブをグループ化する機能が有効になっています。これにより、目的の似たいくつかのタブを視覚的に分離されたグループに組み合わせることができます。
  3. Googleの利用規約が変更され、Google ChromeとChrome OSの個別のセクションが表示されました。
  4. 安定化され、BadgingプログラムのインターフェースがOrigin Trialsの外部に配布されるようになりました。これにより、Webアプリケーションはパネルまたはホーム画面に表示されるインジケーターを作成できます。
  5. Web開発者向けツールの改善。
  6. TLS 1.0およびTLS 1.1サポートの無効化は、Chrome 84のリリースまで延期されました。

Chrome OSのアップデートもリリースされました。シンプルなナビゲーションジェスチャーと新しいクイックシェルフドック、CNetレポートが装備されています。
