在超级水壶的虚拟环境中安装OpenCV-Python

在这里，您将找到在Python中安装OpenCV库的分步示例。


  • Python安装
  • 虚拟环境设置
  • 安装OpenCV + jupiterlab，numpy，matplotlib
  • 测试中

我在带有64位操作系统的Microsoft Surface平板电脑Windows 10 Pro上测试了所有内容。


假设您的设备尚未预先安装。


  1. 首先安装Python。
    下载所需的版本并运行.exe文件。不要忘记选中添加路径框。我从2019年3月25日安装了Python 3.7.3，因为自2019年7月8日起的新版本的Python 3.7.4无法正常工作，即某些命令挂在了终端上。打开命令行。
  2. 安装virtualenv。
    我们需要一个虚拟环境，以便每个单独的项目都有自己的“房间”，并带有自己版本的已安装库，而这些版本不会依赖于其他项目并且会彼此混淆。
    我们将使用pip安装软件包。它最近已经直接在Python上运行，但是通常需要使用以下命令进行更新：
    python -m pip install --upgrade pip
    更新了pip，现在安装了虚拟环境：使用该
    pip install virtualenv
    命令，cd转到要在其中创建环境的文件夹并输入命令：
    mkdir opencvtutorial_env-这就是我们创建名为opencvtutorial_env的环境的方式。
    接下来，输入命令virtualenv opencvtutorial_env并激活，进入环境文件夹，然后使用Tab键激活。
    .\opencvtutorial_env\Scripts\activate
  3. OpenCV-Python, Numpy Matplotlib, opencv.
    . :
    pip install opencv-python
    opencv-python numpy. matplotlib: pip install matplotlib.
  4. pip install jupyterlab jupyter notebook.
