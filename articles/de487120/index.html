Verteilung von Kätzchen durch DHCP


  1. Binden Sie eine Leine an ein Kätzchen
  2. Führen Sie das Kätzchen in die Menge
  3. Wenn der Besitzer gefunden wird, wird er das Kätzchen von der Leine lösen.
