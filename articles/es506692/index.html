Las 14 bibliotecas más populares de Node.js en 2020



¡Buen dia amigos!

Les presento la traducción del artículo "14 Bibliotecas NodeJS más útiles en 2020" .

En este artículo, el autor comparte sus hallazgos entre los muchos paquetes de Nodejs.

Rápido


Marco de desarrollo web rápido, flexible, minimalista y fácil de aprender.

Grandioso para

  • desarrollo de aplicaciones utilizando la pila MEAN (MongoDB, Express, Angular, Node) o MERN (React en lugar de Angular)
  • manejo sencillo de varios tipos de solicitudes, como GET, PUT, POST y DELETE
  • despliegue rápido de aplicaciones de una página, múltiples páginas e híbridas

Número de descargas por semana.

11 millones

Alternativas

Koa, Hapi, Meteorito

Cheerio


Diseñado para el marcado de procesamiento ("análisis"), por ejemplo, HTML, proporciona una interfaz para manipular ("atravesar") la estructura de datos final.

	const cheerio = require('cheerio')
	const $ = cheerio.load('<ul id="fruits">...</ul>')

Grandioso para

  • crear escáneres web ("raspadores")
  • sintaxis y kit de herramientas fáciles e intuitivos

Descargas semanales

4,2 millones

Alternativas

jsdom titiritero

Nodemailer


Diseñado para enviar correos electrónicos.

	const nodemailer = require('nodemailer')

	const testAccount = await nodemailer.createTestAccount()

	const transporter = nodemailer.createTransport({
		host: 'smtp.ethereal.email',
		port: 587,
		secure: false,
		auth: {
			user: testAccount.user,
			pass: testAccount.pass
		}
	})

	const info = await transporter.sendMail({
		from: '"John Smith" <smith@example.com>',
		to: 'foo@example.com, bar@example.com',
		subject: 'hello',
		text: 'hello crazy world',
		html: '<strong>hello crazy world</strong>'
	})

Grandioso para

  • enviar mensajes a través de SMTP

Descargas semanales

980 mil

Alternativas

sendmail, emailjs

Socket.io


Diseñado para configurar una conexión bidireccional basada en eventos en tiempo real.

	const server = require('http').createServer()
	const io = require('socket.io')(server)
	io.on('connection', client => {
		client.on('event', data => { ... })
		client.on('disconnect', () => { ... })
	})
	server.listen(3000)

Grandioso para

  • Implementación de análisis en vivo, flujo de datos binarios, mensajería, colaboración con documentos

Descargas semanales

3 millones

Alternativas

empujador, websockets

Falsificador


Diseñado para recibir información falsa en el navegador y el "nodo".

	const faker = require('faker')

	const randomName = faker.name.findName()
	const randomEmail = faker.internet.email()
	const randomCard = faker.helpers.createCard() //

Grandioso para

  • probar el frontend en términos de trabajo con datos hasta el desarrollo del backend
  • Una gran cantidad de métodos, incluida la obtención de una dirección arbitraria, nombre de la empresa, base de datos, imagen, nombre, apellido, etc.

Descargas semanales

1,4 millones

Alternativas

casual

Morgan


Es un middleware para registrar solicitudes.

	morgan
	(':method:url :status :res[content-length] - :response-time ms')

	---

	const express = require('express')

	const morgan = require('morgan')

	const app = express()

	app.use(morgan('combined'))

	app.get('/', (req, res) => res.send('hello crazy world'))

Grandioso para

  • escribir solicitudes a la consola, archivo, base de datos
  • depuración y "registro"

Descargas semanales

2 millones

Http-errores


Diseñado para crear errores específicos para probar aplicaciones en Express, Koa, Connect, etc.

	app.use((req, res, next) => {
		if(!req.user) return next(createError('401', ',    .'))
		next()
	})

Grandioso para

  • errores de emulación
  • una gran cantidad de propiedades del objeto Error están disponibles

	expose
	headers
	message
	status
	statusCode

Descargas semanales

27 millones

Analizador de cuerpo


Es un middleware para analizar datos.

Analiza las solicitudes entrantes (req.body) antes de procesarlas.

	const express = require('express')
	const bodyParser = require('body-parser')

	const app = express()

	//  application/x-www-form-urlencoded
	app.use(bodyParser.urlencoded({ extended: false }))

	//  application/json
	app.use(bodyParser.json())

Grandioso para

  • interpretaciones del cuerpo de consulta
  • contiene muchas funciones útiles (inflar, escribir, verificar, etc.)

Descargas semanales

13 millones

Secuela


Es una tecnología basada en Promise para el mapeo relacional de objetos de Postgres, MySQL, MariaDB, SQLite y Microsoft SQL Server.

Admite transacciones seguras (confiables), intercambio de datos, carga energética y demorada, replicación de datos, etc.

	const sequelize = new Sequelize(
		'database', 'username', 'password', {
			host: 'localhost',
			dialect: /*   'mysql' | 'mariadb' | 'postgres' | 'mssql' */
		}
	)

Descargas semanales

720 mil

Pasaporte


Es un middleware Express compatible para autenticación.

La tarea principal es autenticar las solicitudes mediante un conjunto de complementos: estrategias.

	passport.use(new LocalStrategy(
		(username, password, done) => {
			User.findOne({ username: username }, (err, user) => {
				if(err) { return done(err) }

				if(!user) { return done(null, false) }

				if(!user.verifyPassword(password)) { return done(null, false) }

				return done(null, user)
			})
		}
	))

Grandioso para

  • autenticación
  • integración con OAuth y OpenID (inicie sesión con su cuenta de Facebook, Twitter, etc.)

Descargas semanales

810 mil

Dotenv


Es un módulo de dependencia cero (punto) para cargar variables del archivo .env en process.env.

Mantiene las variables separadas del código.

	require('dotenv').config()

	const db = require('db')

	db.connect({
		host: process.env.DB_HOST,
		username: process.env.DB_USER,
		password: process.env.DB_PASS
	})

	//  .env
	DB_HOST = localhost
	DB_USER = root
	DB_PASS=12345

Grandioso para

  • cargar variables de entorno como AWS, nombres de usuario sql, contraseñas necesarias para implementar aplicaciones, conectarse a otras herramientas
  • almacenar la configuración del entorno por separado del código

Descargas semanales

10 millones

Multer


Middleware para procesar datos multiparte / formulario - carga de imágenes.

	const express = require('express')

	const multer = require('multer')

	const upload = multer({ desc: 'uploads/' })

	const app = express()

	app.post('/profile', upload.single('avatar'), (req, res, next) => {
		// req.file -   'avatar'
		// req.body     
	})

	app.post('/photos/upload', upload.array('photos', 12), (req, res, next) => {
		// req.files -    'photos'
		// req.body      
	})

Grandioso para

  • descarga rápida de archivos multiparte / datos de formulario

Descargas semanales

92 mil

Axios


Cliente HTTP basado en promesas para navegador y nodo.

	const axios = require('axios')

	//     ID
	axios.get('/user?ID=12345')
		.then(response => {
			// 
			console.log(response)
		}).catch(error => {
			//  
			console.log(error)
		}).finally(() => {
			//  
		})

Grandioso para

  • enviando solicitudes desde Node
  • enviando solicitudes desde el navegador
  • Promesa de apoyo

Descargas semanales

9.6 millones

CORS


Connect / Express middleware diseñado para definir reglas de políticas de origen común.

	const express = require('express')

	const cors = require('cors')

	const app = express()

	app.use(cors())

	app.get('/products/:id', (req, res, next) => {
		res.json({ msg: '    !' })
	})

	app.listen(80, () => {
		console.log('ok')
	})

Grandioso para

  • Tareas relacionadas con CORS

Descargas semanales

3.7 millones
