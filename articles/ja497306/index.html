DLLスプーフィング（DLLハイジャック）

みなさん、こんにちは。現在、OTUSは4月の更新されたリバースエンジニアリングコースの開始に向けてセットをオープンしました。コースの開始を見越して、私たちは伝統的に興味深い資料の翻訳を準備しました。



Windowsオペレーティングシステムでは、起動時にアプリケーションとサービスが適切に機能するために必要なDLLを探します。これらのDLLが見つからない場合、またはそれらのロードが危険な方法で実装されている場合（絶対パスを使用せずにDLLが呼び出される）、アプリケーションに悪意のあるDLLファイルをダウンロードして実行するように強制することにより、特権を増やすことができます。

アプリケーションがDLLをダウンロードする必要がある場合、その検索は次の順序で実行されることに注意してください。

  • アプリケーションのダウンロード元のディレクトリ
  • C：\ Windows \ System32
  • C：\ Windows \ System
  • C：\ Windows
  • 現在の作業ディレクトリ
  • ユーザーPATH環境変数のディレクトリ
  • PATHシステム環境変数のディレクトリ
