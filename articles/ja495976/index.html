例としてWeather APIからのデータを使用してAndroidでVolleyライブラリを使用する

すべてのAndroid開発者は遅かれ早かれネットワークに取り組む必要があります。Androidには、Retrofit、OkHttp、またはVolleyなどの多くのオープンソースライブラリがあり、本日詳しく説明します。

このライブラリはどのようなものですか？

Volleyは、Androidアプリケーションのネットワークを簡素化および高速化するHTTPライブラリです。
GitHubライブラリコード。

したがって、Volleyを開始するには、それをbuild.gradle（モジュール：アプリ）に追加する必要があります。

dependencies {
    ...
    implementation 'com.android.volley:volley:1.1.1'
}

アプリケーションのマニフェストでインターネットを使用するための許可を追加することも必要です。

<uses-permission android:name="android.permission.INTERNET" />

次に、APIが必要です。このチュートリアルでは、openweathermap.org / apiの Weather APIを使用します。

このAPIの例

まず、API（アプリケーションプログラミングインターフェイス）から取得したデータを表示するための簡単なマークアップを作成します。

<?xml version="1.0" encoding="utf-8"?>
<RelativeLayout xmlns:android="http://schemas.android.com/apk/res/android"
    xmlns:app="http://schemas.android.com/apk/res-auto"
    xmlns:tools="http://schemas.android.com/tools"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    tools:context=".MainActivity">

    <TextView
        android:id="@+id/tempTextView"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text=": "
        android:textSize="20sp"
        android:layout_alignParentLeft="true"
        android:layout_marginLeft="50dp"
        android:layout_alignParentTop="true"
        android:layout_marginTop="50dp"
        />

    <TextView
        android:id="@+id/windTextView"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:layout_alignParentLeft="true"
        android:layout_marginLeft="50dp"
        android:textSize="20sp"
        android:layout_marginTop="10dp"
        android:text=" :"
        />

</RelativeLayout>

次に、MainActivityに移動して、必要なフィールドを作成します。

private static final String testUrl = "https://samples.openweathermap.org/data/2.5/weather?id=2172797&appid=b6907d289e10d714a6e88b30761fae22"; //url,      JSON- 
RequestQueue mRequestQueue; //  
TextView tempTextView,windTextView; //     
double temp = 0,windSpeed = 0; //      JSON

作成したフィールドfieldsをonCreateで初期化します。

tempTextView = findViewById(R.id.tempTextView);
windTextView = findViewById(R.id.windTextView);

mRequestQueue = Volley.newRequestQueue(this);

次に、このチュートリアルのメイントピックに

進みます。Volley ライブラリを使用してAPIからデータを取得します。1）MainActivityでGetWeatherメソッドを作成します。

private void getWeather(String url) {
        final JsonObjectRequest request = new JsonObjectRequest(Request.Method.GET, //GET - API-   
                url, null, new Response.Listener<JSONObject>() {
            @Override
            public void onResponse(JSONObject response) {
                try {
                    JSONObject weather = response.getJSONObject("main"),wind = response.getJSONObject("wind"); // JSON- main  wind (   - ,   -  (JSONArray).
                    temp = weather.getDouble("temp"); 
                    windSpeed = wind.getDouble("speed");
                    //      API
                    setValues(); //   setValues    
                } catch (JSONException e) {
                    e.printStackTrace();
                }
            }
        }, new Response.ErrorListener() { //    
            @Override
            public void onErrorResponse(VolleyError error) {
                error.printStackTrace();
            }
        });

        mRequestQueue.add(request); //    
}


jsonformatterは、jsonを通常の形式にするために使用されました 。
オブジェクトの名前は、APIの場合とまったく同じ方法で記述する必要があります。それ以外の場合は、取得できません。

2）setValuesメソッドを直接作成します：

private void setValues() {
        tempTextView.setText(": " + temp);
        windTextView.setText(" : " + windSpeed);
}

3）onCreate（）でgetWeather（）およびsetValues（）メソッドを呼び出します。

@Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);

        getWeather(testUrl);
        setValues();
}

4）アプリケーションを実行すると...完了です！
