Como resolver? "ECONNREFUSED - Conexão recusada pelo servidor"

Você também encontrou o erro ECONNREFUSED - conexão recusada pelo servidor no FileZilla ? Então é ótimo que você tenha encontrado este guia. Vou mostrar três métodos para corrigir esse erro de FTP.


O primeiro método. Alterando o valor padrão de uma porta FileZilla


A causa do erro pode ser a porta incorreta ao se conectar via FileZilla. Nessa situação, você só precisa alterar a porta FTP padrão para o número da porta SFTP padrão . Apenas altere 21 para 22 na caixa de entrada " Port ".




Segundo método. Desativando o antivírus / firewall


Às vezes, esse erro pode ocorrer quando o software antivírus e / ou o firewall recusa a tentativa do FileZilla de estabelecer uma conexão.


Caso um antivírus ou firewall chame ECONNREFUSED, você só precisa desativar este software e reconectar. Primeiro, mostrarei como fazer isso no macOS:
