Chrome電報アラートプラグインの作成



トラブル、同志！

子供（〜10歳）は遠隔学習に切り替え、コンピューターのある部屋に座って、レッスンに集中できませんでした！

それらは、グーグルファミリー、マイクロソフトファミリーに追加され、コンピューターでの時間が制限され、アダルトサイトはopendnsを通じて削除されます。しかし、それだけ-そして、レッスンのズーム会議の最中にさえ、子供はYouTubeを見始めます。そして、先生は宿題を出します-ビデオを見て、エッセイを書くこと。そして、ビデオもyoutubeにあります。手動によるモデレートは不可欠です。

猫の下で、電報で開かれた各Webサイトを隣の部屋で働く邪悪な親に送信する魔法のペンダル、つまりクロムプラグインを作成します。

ボット


最初に、電報ボットとリンクが来るチャネルを作成します。

  1. BotFatherを開き、/ newbotを記述し、botId（50文字以下の組み合わせ）を取得します。
  2. 新しいチャネルを作成し、そこに管理者としてボットを追加します。
  3. チャネルに何かを書き込んで、リンクhttps://api.telegram.org/bot botId / getUpdatesを開き、そこにresult-> channel_post-> chat-> idを見つけます。

その結果、botIdとchatIdがあります。

プラグイン


  1. フォルダーを作成すると、2つのファイル（manifest.jsonとindex.js）が含まれます。
  2. これを最小限に抑えるために、Chromeプラグインマニフェストを作成します。

    {
  "manifest_version": 2,
  "name": "Url Telegram Logger",
  "version": "1.0.0",
  "content_scripts": [
    {
      "matches": [
        "<all_urls>"
      ],
      "js": ["index.js"]
    }
  ]
}

    ここでは、index.jsスクリプトが任意のページに埋め込まれることになります（一致：すべてのURL）。
  3. . SPA, document.location.href. MutationObserver. . , ( , ). ( , ). navigator.sendBeacon:

    const settings = {
  botId: '...',
  chatId: '...',
  userName: 'Matvey',
  pollingInterval: 10000
}

const mutedUrls = [
  'https://gmail.com',
  'https://www.eduka.lt',
  // .. ,   ..
]

const sendMessage = (type, href) => {
  if (mutedUrls.find(url => href.startsWith(url))) { return false }
  const data = new FormData()
  data.append('chat_id', settings.chatId)
  data.append('text', `${settings.userName} ${type} ${document.title} ${href}`)
  navigator.sendBeacon(
    `https://api.telegram.org/bot${settings.botId}/sendMessage`, 
    data
  )
}

let latestHref = null
let timeout = 0

const run = () => {
  if (window.location.href !== latestHref) {
    latestHref = window.location.href
    sendMessage('opened', latestHref)
  }
  timeout && clearTimeout(timeout)
  timeout = setTimeout(run, settings.pollingInterval)
}

window.addEventListener('load', run)
window.addEventListener('unload', () => sendMessage('closed', latestHref))
