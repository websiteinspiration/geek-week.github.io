So zeigen Sie Oracle WebLogic Server mit einem großen WLSDM-Dienstprogramm für Monitor und Konsole an

Auf den Freiflächen der Konsolenerweiterungsdienstprogramme Oracle WebLogic Server gab es eine sehr nützliche - WLSDM , wie die Autoren es selbst positionieren - ein WebLogic Server-Überwachungsdienstprogramm mit einer Vielzahl von Funktionen. Wenn Sie auf die Website des Entwicklers gehen, werden Sie feststellen, dass sich in der Nähe ein weiteres leistungsstarkes Tool befindet, das jedoch kostenpflichtig ist.

Nach dem Herunterladen des Archivs erhalten wir mit nur zwei Klicks ein bereits für die Arbeit geeignetes Tool mit einer Reihe von Schwellenwerten, der Möglichkeit zum Senden von Nachrichten und großen Geschwindigkeitsmessgeräten. Besonders für einen großen Monitor oder um einem gepaarten DevOps-Ingenieur zu helfen, der sich über rotierende Anwendungen auf dem Laufenden halten muss.

Ein paar Worte zu Servermetriken im Allgemeinen. Natürlich können Sie die integrierte Metrikansicht verwenden. Sie können sogar lernen, wie Sie Metriktabellen im Server selbst anpassen.
Wir gehen jedoch von der Benutzerfreundlichkeit und der vergleichenden Ähnlichkeit dieses Tools aus.
Außerdem werden die eingestellten Schwellenwerte sofort in den Diagrammen angezeigt und Warnungen per Brief gesendet. Was in der Tat für DevOps notwendig ist.


  1. Agenteninstallation.










    Sie können es sowohl auf dem Admin-Server als auch auf jedem Server installieren, den Sie überwachen möchten. Der Einfachheit halber haben wir für Admin bereitgestellt.
  2. Installieren Sie die Konsolenerweiterung auf WebLogic Server.



    Beachtung! Vergessen Sie nicht, einen Neustart durchzuführen, um die Konfiguration zu starten.

    Nach dem Neustart wird in der Konsole ein Fenster mit dem Überwachungsdienstprogramm (links) angezeigt. Sein Hauptmenü (rechts).
