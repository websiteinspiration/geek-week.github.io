「5つの違いを見つける。」スケーラブルで世代の違い-新しいテスト



発表から2年足らずで、インテルは新しいカスケードレイクアーキテクチャに第2世代のインテルXeonスケーラブルプロセッサーを導入しました。正式に-4月2日。同社自身は、これを史上最大のローンチと呼んでおり、戦略的に非常に重要です。さて、これらの新しいスケーラブルの何が特別なのかを考えてみましょう。

何が残ったの？



プロセッサカスケードレイク、またはカスケードレイクSPは、前身のスカイレイクと同様に、プラットフォームPurleyに属し、現在は第2世代のPurley Refreshに属しています。コネクタレベルでのSkylake、チップセット、および第1世代から継承されたマザーボードと完全に互換性があります。しかし、ニュアンスがある-例えば、新しい経歴。
プロセス技術は変わっていません。ただし、最適化された同じ14 nm。

プラチナ、ゴールド、シルバー、ブロンズシリーズの一般的な命名と命名方式は同じままです。確かに、「サフィックス」は他にもあります。新しいY、N、V、およびSが既存のL、M、およびTに追加されました。番号付けでは、2番目の位置（数百）の値が変更されました。1の代わりに2になりました。つまり、ゴールド6240がゴールド6140などの後継となります。

それ以外の場合、基本的な特性と機能のセットは変更されていません。コアの数とキャッシュサイズは位置を保持します：コアあたり最大28および1 MB L2 +最大38.5 MBの合計L3。PCI-Eラインの数とタイプは、バージョン3.0の48ラインと同じです。スケーラビリティは同じです。システム内の10.4 GT / sあたり最大3 UPIライン、および最大8（シームレス）ソケット。

何を追加しましたか？


一般に、さまざまなマイクロアップデートがありますが、多かれ少なかれ重要なものからこれらを選びます。

まず、カスケードレイクは昨年、センセーショナルな脆弱性に対するハードウェアパッチを導入しました。Intelは、オプション2（Spectre）、3、3aおよび4（Specter NG）、L1TF（Foreshadow）に対するソフトウェアおよびハードウェアソリューションを導入しました。Spectre Variant 1では、ソフトウェアパッチのみが引き続き提供されます。つまり、すでにIntel Core i9ラインにあるすべてのものです。そして、それはプレスリリースに見えます：

  • オプション1. OSとVMM（仮想マシンモニター）を使用して保護を実行する
  • オプション2.ハードウェアブランチ予測の強化（この方法による将来の攻撃の防止）+ OSおよびVMM
  • オプション3.ハードウェアの強化
  • オプション3a。ハードウェア
  • オプション4.ハードウェア+ OS / VMM
  • L1TF。ハードウェア強化オプション3によりすでに終了しています

次に、DDR4-2933メモリのサポートが登場しました。ただし、予約がある場合：ゴールドおよびプラチナライン（ブロンズおよびシルバーは引き続きDDR4-2400で動作します）のみで、チャネルごとにDIMMが1つだけあります-チャネルごとにDIMMが2つある構成では、周波数は2666 MT / sに低下します。

第三に、インテルOptane DCパーシステントメモリ（DCPM）が初公開されました。それが何であるかについての最も明確な表現は、ティスコマによって得られたので、私は引用します：
「インテルOptane DCパーシステントメモリ（DCPM）は、データセンターで使用するための「メモリとストレージ」と呼ばれる概念を組み合わせた新しいクラスのテクノロジーです。」

Intelが以前にXeon Skylake用のIntelメモリードライブテクノロジーを導入したことを思い出してください。ハイパーバイザー（Xen）+ Optane NVMeモジュール。私たちはこの問題についてもテストを行いましたが、結果は刺激的ではなく、より印象的な解決策を待つことにしました。待っていたようです=）

インテルの新しいソリューションは、DIMMに視覚的に類似し、電気的および機械的に互換性のあるDCPMMに基づいています。彼らは2666 MT / sの速度で動作し、128/256/512 GBの容量を持っています。論理的なレベルでは、それらはDDR4-T（トランザクション）プロトコルを使用します。これはIntelによるとJEDECによって承認されていますが、実際にはCascade Lakeメモリコントローラーでのみサポートされています。つまり、3D XPointテクノロジーを使用して作られたエネルギーに依存しないメモリがDDR4 DDR4コネクタに搭載されました。これは、速度や耐用年数など、Intelによると、広く使用されているNANDフラッシュよりも3桁（1000倍）優れています。

このソリューションは非常に興味深く、あいまいであることがわかりました。もちろん、操作機能（それなしではありません）、価格、アプリケーションの領域があります。ただし、このプロセッサラインのキラー機能には焦点を当てません。この詳細については、今日の記事の範囲をはるかに超えています。このテクノロジーのすべての可能な動作モードでのテストの準備ができたら、すぐにLongRidをロールアウトします:-)

4番目に、Intel Resource Director Technology（RDT）、Speed Select（SST）、およびIntel DL Boostテクノロジーがスキルを駆使します。

RDTから始めます。これは、アプリケーションの実行とリソースの使用に対するかなり細かい監視と制御のメカニズムを表しています。事は新しいものではありませんが、このラインでは彼らはそれに優れた手腕を持ち、詳細に働きました。要するに、時間的に優先度の高いアプリケーションが必要なものすべてを手に入れるということです。当然、他のアプリケーションの「権利の侵害」によるものです。

今SST。ここでも同じですが、核のレベルで、他の核よりも優先度が高くなる核のグループをしっかりと区別できます。今回の登場はデビューではありませんが、見事です。

そしてデザートには、Intel DL Boost。イノベーションは、以前はベクターニューラルネットワーク命令（VNNI）と呼ばれていた新しい命令セットに関係しています。 AI向けギズモ、またはディープラーニングネットワークのより柔軟なトレーニング用。実際、AVX-512とは別のアドオンです。

そして最後に、第五。古い伝統によると、より多くの周波数があり、Intelの更新のためのコアがより多くあります:-)基本周波数とブーストの周波数の両方が200〜300 MHz増加しました。一部の例外を除いて、プロセッサごとに2つのコアが追加されました。サポートされるRAMの量が増加しました。

これとは別に、キャッシュとRAMの使用を最適化するIntelの取り組みは注目に値します。おそらく、SpectreおよびMeltdownファミリの脆弱性に対するパッチの悪影響を最小限に抑えるためです。

カスケードレイクのアーキテクチャの詳細については、wikichipをご覧ください。読むことをお勧めします。そして今-従来のテスト。

新しいGold 62xx Intel Xeonスケーラブルプロセッサは、新しいタイプのDDR4-2933 RAMをサポートしています。私たちは、論理的には、自分自身に問いかけました。RAMの周波数がシステム全体のパフォーマンスにどの程度影響するか。一般に、プラスからプラスは常に何かがプラスになるとの仮定に基づいて、新しいメモリと組み合わせた新しいプロセッサが優れていることが証明されると信じられていました。しかし、それは仮定することと、実験的に検証することです。

人間にとっては小さな一歩、人類にとっては大きな一歩


カムノイェドフ同志（私はStrugatskyが大好きです）が言うように、「ほぼそのような受け入れで」IntelはXeonスケーラブルプロセッサの新しいラインを位置づけています。記事の冒頭で、新しいScalable for Intel自体のリリースは重要な戦略的ステップであると述べました。今説明します。

一方、新しいスケーラブルは、データセンタープラットフォームのグローバルなアップグレードを先導しました。そして、今年の後半には、いくつかの興味深い発表が待っています。一方、すべてのイノベーションは偶然ではありません。これは、現在の業界のニーズに対する答えです。そして、かなりまともな答え。メモリーが充分ではありません？これがOptane DC Persistent Memoryです。プロセスとコアのハードウェア優先順位付けが必要ですか？ SSTとRDTをポンプで送ってください。ネットワークの専門的なトレーニングを夢見ましたか？ :-)ここで、AIの新しい命令セットに署名します。インテルの場合、喜ぶことができます。

個人的には、このリリースにはウィッシュリストが含まれているようですが、インテルは前回実装できませんでした。そしてもちろん、ハードウェアの穴で何かをしなければなりませんでした。さまざまな専門家のための検索はすでに一種の娯楽になっています。 IntelがSpectrum-Meltownaの穴のあるユーザーから奪ったすべてのものを今や返し、価格を節約しました。

さらに、AMDはすべての方向から来ており、その決定はSpectrum-Meltdownsによる影響がはるかに少なく、最近は特にデスクトップのようにIntelの「不安定な」インテルになりました（私はそのような立派な時代にそのような若さを持ちたいと思います）。ややサーバーセグメントで。ちなみに、後者に関しては、新しい世代のAMD Epyc Romeがどのように表示されるかを見るのは非常に興味深いです。現在の世代のEpycは個人的に私を無関心にしていなかったからです。

しかし、スケーラブルに戻ります。

AIや訓練されたネットワークに負担をかけられていないユーザーにとって、最終的には何が必要ですか？より多くのコア、より高いベース周波数、およびターボブーストの周波数による生産性の明白な増加。異なる世代のゴールドプロセッサの場合、この増加は最大23％に達します-それらの両方が良い場合、シルバーの一部のテストでは40％に達します。ほとんど変わらない値を考えると、違いはかなり楽しいですが、私は常にもっと欲しいです=）

これがほんの始まりにすぎないというIntelの声明に頼るなら、私のような懐疑者でさえ、将来何が提供されるかを知りたがります。
