Proyek JavaScript untuk pembelajaran dan pengembangan



Selamat siang teman!

Saya mempersembahkan kepada Anda sejumlah 22 proyek murni atau, seperti kata mereka, vanilla JavaScript (secara pribadi, saya lebih suka hanya JavaScript).

Bagi saya, ini adalah semacam subtotal dalam belajar tentang JS.

Kode ditulis pada waktu yang berbeda, sehingga terlihat berbeda dan memiliki tingkat kerumitan yang berbeda.

Sebagian besar proyek tidak dapat disebut aplikasi lengkap, tetapi masing-masing dari mereka berhasil menyelesaikan tugas.

Kode proyek ada di sini .

Formatnya adalah sebagai berikut: nama proyek, deskripsi singkatnya, kotak pasir.

Ada beberapa artikel terpisah untuk beberapa proyek. Menurut yang lain, suatu hari mereka mungkin muncul.

Jadi ayo pergi.
