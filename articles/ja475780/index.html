quoted-printableのキリル文字デコーダ

こんにちは、友達！私は近い将来、面白い仕事を見つけたので、Android用の「ダイアラー」を書くことにしました。アプリケーションはシステムの連絡先と同期し、特定のアクションを実行します。quoted-printableはそれが何であり、なぜ私がそれを必要としたのかと関係があります-私は記事で伝えます。


したがってquoted-printableは、バイナリテキストを印刷可能なASCII文字を使用してテキストにエンコードするシステムであり、Wikipediaの英語版のページで判断すると、電子メールメッセージのデータのエンコード/デコードに使用されます。


実際、これは真実ではありません。そのようなファイル形式があります-vCard。そして、連絡先が任意のAndroidスマートフォンからインポート/エクスポートされるのはこのフォーマットです。したがって、バージョン2.1のこの形式（拡張子.vcf）では、quoted-printableエンコーディングも使用されます。このエンコーディングのキリル文字の形式（例）は次のとおりです"=D0=9F=D1=80=D0=B8=D0=B2=D0=B5=D1=82"。最初に、各キリル文字がUTF-8で2バイトのシーケンスにエンコードされ、次に各バイトが "="に等しい符号で16進表記で書き込まれます。


この形式では、キリル文字のすべての連絡先がインポートされます。ファイルの読み取りと編集の問題がないことは明らかです。それがまさに私が必要としていることです。テキストエディタでプラグインを試してみました...この問題は解決できますが、操作が多すぎます。要するに、私はデコーダーを書くために座らなければなりませんでした。


その過程で、私は別の障害に遭遇しました。実際のところ、quoted-printableエンコード標準では、最大長が75文字の行が提供され、ハイフンが作成され、文字「=」が複製されます。ラップされた行を組み合わせるには、追加の関数が必要でした。


スクリプトはquopriモジュールを使用します（インストールなしですぐにインポートされました）。


import quopri
import os

List_contact = []
File = ".vcf"
with open (File) as file: #    
    for i in file:
        List_contact.append (i)

#     
def Func (List_for_change):
    List_contact_1 = []
    for i in List_contact:
        if i[-2] == '=':
            List_contact_1.append (i[:-2])
        else:
            List_contact_1.append (i)
    with open ('File.txt', 'w') as file:
        for i in List_contact_1:
            file.write (i)
    List_contact_1 = []
    with open ('File.txt') as file:
        for i in file:
            List_contact_1.append (i)
    os.unlink ('File.txt') #  temp 
    return (List_contact_1)

List_contact = Func (List_contact)

#    
with open ('Contacts_Decode.txt', 'w') as file:
    for i in List_contact:
        Str_1 = bytes (i, 'UTF-8') #  quopri     
        Str_2 = quopri.decodestring (Str_1)
        file.write (Str_2.decode ('UTF-8'))
