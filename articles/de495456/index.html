14. Sitemap




Wenn Sie eine Karte Ihrer Site erstellen müssen, hilft Ihnen dieses Plugin bei dieser Aufgabe.
Die Funktionalität des Sitemap-Plugins ist recht einfach: Geben Sie die URL Ihrer Site ein oder fügen Sie sie ein und klicken Sie auf "Karte erstellen". Das Plugin zeigt alle Links an, die es aus der HTML-Quelle findet.

13. Halbtöne




Mit Hilfe von Halbtönen können Sie coole Punkt- und Halbtonfilter auf Ihre Bilder anwenden.

Wählen Sie einfach ein oder mehrere Bilder aus und wählen Sie den Filtertyp aus, den Sie anwenden möchten (Punkte oder Halbton-Effekt). Bestimmen Sie dann mit den Schiebereglern den Winkel und die Größe des Bildes, klicken Sie auf "Übernehmen" und voila - Ihr Bild ist trendiger geworden.

12. Flipbook




Verwässern Sie unsere Bewertung mit einer kleinen Portion Spaß durch das Flipbook-Plugin! Dies ist ein Plugin zum Animieren Ihres Designs in Figma. Es verfügt über viele Funktionen, einschließlich Echtzeitvorschau, Geschwindigkeitsänderung, Anpassung der Animationsgeschwindigkeit mithilfe des Schiebereglers sowie umgekehrte Animation und Export nach GIF.

Es gibt auch eine Pro-Version dieses Plugins. Es hat kein Wasserzeichen und auch eine Zwiebelschalenfunktion (Zwiebelschale, d. H. Die Funktion zum Anzeigen der Rahmenabdeckung zur Überprüfung Ihrer Animation).

11. Zauberinspektor




Haben Sie jemals ein Tool benötigt, das Rechtschreibfehler in der Abbildung für Sie findet? Nun, das Spell Inspector Plugin wird Ihnen dabei helfen.

Der Zauberinspektor verwendet die spezielle SpellCheck-API. Derzeit besteht das Wörterbuch aus 40.000 Wörtern. Der Rechtschreibinspektor überprüft die Seite, auf der Sie Rechtschreibfehler finden möchten, und gibt eine Liste der relevanten Sätze an. Wir hoffen, dass die Entwickler dieses Plugins in Zukunft weitere Sprachen hinzufügen und das Wörterbuch erweitern werden.

10. CSS fokussieren




Dieses Plugin verbessert die Arbeit mit CSS in Figma und erkennt inkonsistente Stile.

Plugin-Funktionen:

  1. CSS-Generierung für alle Elemente im gesamten Figma-Dokument;
  2. Die Möglichkeit, die CSS-Eigenschaften auszublenden, die Sie nicht in den Code kopieren möchten.
  3. Es gibt auch eine Filterfunktion, mit der Sie nur die Arten von CSS-Ebenen anzeigen können, die Sie benötigen.

9. Verfolgung




Mit diesem Plugin können Sie Tracking-Anmerkungen direkt aus der Figma-Datei erstellen, aktualisieren oder anhängen.

Über Tracking können Sie Entwicklern erklären, was genau Sie verfolgen möchten, und sicherstellen, dass Sie die benötigten Elemente aus der Vogelperspektive verfolgen. Darüber hinaus verfügt das Plugin über eine Statusfunktion, sodass Sie wissen, ob das Tracking durchgeführt wurde oder nicht.

Um besser zu verstehen, wie das Plugin verwendet wird, finden Sie das Handbuch hier .

8. Filter




Trotz der Tatsache, dass dieses Plugin letzten Monat veröffentlicht wurde, wird es ständig aktualisiert und der Autor fügt dort immer mehr interessante Filter hinzu. Aus diesem Grund haben wir beschlossen, dieses wunderbare Plugin in unsere Liste aufzunehmen.

Mit Filter können Sie Ihre Kreativität steigern und Ihren Workflow beschleunigen. Wenn Sie dieses Plugin in Ihrem Figma-Arsenal haben, müssen Sie Figma nicht mehr beenden, um Photoshop verwenden zu können, da Sie mit Filter die Einstellungen von Bildern oder Knoten hervorragend steuern können.

Im Moment stehen Ihnen mehr als 30 Filter zur Verfügung, darunter: Bulge Pinch, Farbhalbton, Cartoon, Dreiecksunschärfe, Rauschunterdrückung, Linsenunschärfe, Nachtsicht, Schraffur und viele andere Filter.

7. SwiftUI Inspector




Arbeiten Sie mit Swift zusammen und suchen Sie nach einem Tool, mit dem Sie Elementstile von Figma auf Xcode übertragen können? Sie können Ihre Suche beenden, da SwiftUI Inspector dieses Problem löst. Mit diesem coolen Plugin können Sie die Stile Ihrer Designelemente in SwiftUI-Code exportieren.

Unterstützte Funktionen sind: Farben, Textbeschriftungen, Grundformen. Der Autor des Plugins verspricht, in naher Zukunft Unterstützung für Elemente wie Komponenten, Vektoren und Schatten hinzuzufügen.

6. Geometrisch




Fügen Sie Ihren Projekten mit diesem großartigen Plugin etwas mathematische Schönheit hinzu. Dank Geometric ist das Erstellen komplexer Formen jetzt einfacher als je zuvor. Zu Ihrer Verfügung stehen Figuren wie Rose, Polygone, Spiralen verschiedener Typen, Superellips, Astroid, Trigonometrische Funktionen. Sie können sie mit verschiedenen Parametern ändern. Und laut dem Ersteller des Plugins wird es in naher Zukunft noch interessantere mathematische Zahlen geben.

5. Abstandsmanager




Viele Designer sind auf die Praxis gestoßen, unterschiedliche Einrückungen zwischen Elementen anzuwenden. Einige Elemente wurden 10px eingerückt, andere - 14px usw. Wenn Sie später Einrückungsänderungen vornehmen mussten, mussten sie jedes Element manuell ändern. AutoLayout hat dieses Problem behoben. Mit dem Spacing Manager-Plugin können Sie jedoch Ihren Einrückungsprozess weiter verbessern.

4. Farben suchen und ersetzen




Hatten Sie jemals Situationen, in denen Sie ein unorganisiertes Design aus Sketch importierten? Oder sendet Ihnen Ihr Kollege eine Figma-Vorlage ohne Farbstile? Mit diesem Plugin können Sie leicht alle Farben in der Datei finden und durch die benötigten ersetzen.

Das Plugin ist für die Arbeit mit allen Dateitypen verfügbar, sei es Komponenten, Instanzen, Rechtecke, Rahmen, Vektorobjekte oder Texte.

Das Plugin ist einfach zu bedienen: Installieren Sie es und führen Sie es aus. Klicken Sie dann auf das Eingabefeld neben der entsprechenden Farbe und geben Sie einen neuen Farbwert ein. Danach können Sie mit dem Ersetzen aller Übereinstimmungen beginnen, indem Sie auf die Schaltfläche „Farben ersetzen“ klicken.

3. Batch Styler




Batch Styler ist eine großartige Lösung für Situationen, in denen es viele Textstile gibt und Sie die Schriftarten dieser Textstile schnell und ohne unnötige Aktionen ändern müssen.

Unabhängig davon, wie viele Textstile oder Schriftarten Sie ändern müssen, kümmert sich Batch Styler um alles und speichert alle Arten von Schriftstärken und anderen Stilen.

2. Style Organizer




Dieses Plugin hilft beim Anzeigen und Organisieren von Farbstilen auf der Seite. Mit organisierten Stilen generiert dieses Tool automatisch eine Farbpalette.
Was in diesem Plugin ignoriert wird: Füllen / Strichen von Bildern, unsichtbare Einrückung von Ebenen / unsichtbaren Knoten.

Beachten Sie auch, dass Style Organizer die automatische Aktualisierungsfunktion bei Designänderungen nicht unterstützt. Sie müssen auf die Schaltfläche Aktualisieren klicken, nachdem Sie die Änderungen vorgenommen haben.

1. AutoGrid




Vielleicht sind Sie mit einer Situation konfrontiert, in der Sie mehrere AutoLayout-Ebenen erstellen müssen, um sicherzustellen, dass die Karten in Ihrem Layout in einem ordentlichen Zustand sind?

Dieses Plugin ist eine großartige Lösung. AutoGrid kümmert sich um alles, was mit dem Raster zu tun hat. Sie müssen lediglich die Spalten, Zeilen und Einzüge definieren, den Rest erledigt AutoGrid.

Das Plugin ist ziemlich einfach zu bedienen:

  1. Wählen Sie das Stammelement für Ihr Raster aus (z. B. eine Karte).
  2. Definieren Sie die Rasterstruktur: wie viele Zeilen, Spalten und Einzüge.
  3. Klicken Sie auf Neues Raster erstellen.

AutoGrid erstellt einen Klon Ihres Stammelements, sodass es das neu erstellte Raster ausfüllt.
Das Plugin hat auch eine Auto-Update-Funktion. Mit dieser Funktion müssen Sie Ihr Raster nicht jedes Mal manuell aktualisieren, wenn sich Ihre Struktur ändert.
