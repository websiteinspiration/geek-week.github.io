So erstellen Sie den Moiré-Mustereffekt in der p5.js-Bibliothek für Anfänger (Kurzanleitung)



Das Moiré-Muster ist eine optische Täuschung, die auftritt, wenn zwei periodische Maschenmuster angewendet werden. Die sich wiederholenden Elemente der beiden Muster überlappen sich entweder oder bilden Lücken.

Im normalen Leben begegnen wir dem oft, wenn wir zum Beispiel die Tüllvorhänge betrachten. Es überrascht nicht, dass der französische Name Moire selbst ursprünglich eine ähnliche Art von Seidenstoff bedeutete.



Dieser Effekt kann häufig in verschiedenen Bereichen beobachtet werden - von der Fotografie über die Radiologie bis hin zur Navigation.

Um diesen Effekt zu erzielen und gleichzeitig in p5.js zu üben, habe ich beschlossen, einfachen Code zu schreiben, der ihn neu erstellt.

p5.js ist eine 2014 erstellte Javascript-Bibliothek. Ihr Hauptziel ist es, Designern, Künstlern, Lehrern und anderen Vertretern kreativer Berufe ein Fenster in die Welt des Programmierens zu bieten. Es unterstützt sowohl Animationen als auch Diagramme und lässt sich problemlos in das Webformat übersetzen.

Um zu beginnen, müssen Sie einen neuen Projekt-Online-Editor erstellen. Standardmäßig lautet das neue Projekt wie folgt:

function setup() {
  createCanvas(400, 400);
}

function draw() {
  background(220);
}

function setup () - legt eine Funktion fest, die nur einmal ausgeführt wird.
function draw () - wiederholt sich in einer Endlosschleife.

Wir beginnen mit dem Einrichten unserer „Zeichenfläche“. Für die Funktion createCanvas () setzen wir die Parameter 720 und 500 (Breite bzw. Höhe).

Um eine Ellipse im Hintergrund erscheinen zu lassen, fügen wir die folgenden Parameter in die Funktion draw () ein (die, wie wir uns erinnern, für die Wiederholung verantwortlich ist):

{
  noFill()
  stroke ('purple') 
  strokeWeight(3)
  ellipse(360,250,100,100) }

Was geht hier vor sich?

noFill () - füllt das Innere der Ellipse nicht mit einer Farbe.
Strich ('lila') - Erstellt eine Linie mit lila Farbe.
StrokeWeight (3) - Legt die Dicke dieser Linie fest.
Ellipse (360, 250, 100, 100) - besagt, dass die Linie mit den angegebenen Parametern (Position entlang X, Position entlang Y, Durchmesser entlang X, Durchmesser entlang Y) zu einer Ellipse verdreht werden soll. Der erste und der zweite Parameter entsprechen der Hälfte der Parameter in der Funktion createCanvas (400, 400) - genau in der Mitte. Die gleichen 3 und 4 Parameter erzeugen die Form eines Kreises, nicht eines Ovals.

Als Ergebnis erhalten Sie so etwas.



Noch nicht beeindruckend, oder?

Jetzt müssen wir diese Ellipsen mehr machen, jede etwas mehr als die vorherige. Dazu müssen wir eine Schleife erstellen, unsere Ellipse darin entfernen und die Variablen für den dritten und vierten Parameter festlegen.

 background(220);
  for(let i=0;i<1000; i+=10) {
  noFill()
  stroke ('purple') 
  strokeWeight(3)
  ellipse(360,250,i,i)

Wenn Sie sich jemals mindestens einmal in Ihrem Leben mit Zyklen befasst haben, werden Sie verstehen, was passiert. Wir setzen den Anfangswert i = 0, also zeichnen wir zu Beginn eine Ellipse in einem Radius von Null. Dann setzen wir die Grenze auf den Wert i, mit dem wir den Zyklus wiederholen (bis zu 1000). Danach bestimmen wir den Änderungsschritt i + = 10 - jedes Mal, wenn ich zum vorherigen Ergebnis i hinzugefügt werde.

Wir setzen den Parameter i als dritten und vierten Parameter in die Ellipsenzeichnungsfunktion ein verantwortlich für den Durchmesser in Breite und Höhe.

Wenn wir nun den Code ausführen, erhalten wir Folgendes.



Dies ist kein Moiré, aber es ist bereits ähnlich. Was hat der Code getan? Er zeichnete einfach konsequent 100 Kreise (bis der Durchmesser des letzteren 1000 wurde), die sich im Durchmesser um 10 unterscheiden, wobei jedes Mal das Ergebnis gespeichert wurde.
Vergessen Sie nicht, noFill () einzugeben, da Sie sonst nur das Ergebnis der letzten Iteration speichern und anzeigen.

Um die zweite Ellipse zu erhalten, geben wir sie in dieselbe Schleife ein, jedoch mit leicht unterschiedlichen Parametern:

function draw() {
  background(220);
  for(let i=0;i<1000; i+=10) {
  noFill()
  stroke ('purple') 
  strokeWeight(3)
  ellipse(360,250,i,i)
    
  stroke("red")
  strokeWeight(3)
  ellipse(360, 250,i-500,i-500)
  }

Als Ergebnis erhalten Sie eine Folge von Kreisen, von denen der größte einen Durchmesser von 1000 minus 500 hat. Die Höhe unserer Leinwand beträgt nur 500, so dass der interne Perfektionist zufrieden ist. Nun, es wird rot sein.

Es ist Zeit für die letzte Aktion - diese Kreise aneinander vorbeiziehen zu lassen.

Fügen Sie dazu außerhalb der Schleife die Variable x mit dem Wert 0 ganz am Anfang des gesamten Codes und der if-Schleife NACH der Funktion draw () hinzu

let x = 0
function setup() {
  createCanvas(720, 500);
}

function draw() {
  background(220);
  for(let i=0;i<1000; i+=10) {
  noFill()
  stroke ('purple') 
  strokeWeight(3)
  ellipse(360,250,i,i)
    
  stroke("red")
  strokeWeight(3)
  ellipse(x,250,i-500,i-500)
  }
if (x>width) {
  x= 0
} else {x= x+3}
}

Es gibt eine Bedingung. Wenn x kleiner als die Breite der Zeichenfläche ist, setzen wir x auf den Standardwert Null zurück. Bis dies passiert, addieren wir 3 zu unserem x-Wert. Es versteht sich, dass wir durch X die Koordinate entlang der X-Achse für den zweiten Kreis festlegen möchten. Da es sich jedes Mal geringfügig ändert, bewegt sich unser Kreis horizontal nach rechts, bis seine Mitte den Bildschirmrand erreicht.


Vergessen Sie nicht, den ersten Parameter der zweiten Ellipse von 360 in die Variable x zu ändern.

Erledigt. Als Ergebnis erhalten Sie einen sich wiederholenden, endlos moiré Effekt auf Ihrem Computerbildschirm, der sich leicht in Ihre Webprojekte integrieren lässt.
