云安全警报安全中心

俄罗斯的SecurityHub火警系统可与有线和无线传感器配合使用，可以远程控制各种设备，与iVideon视频监控集成，并使用有线以太网连接和移动Internet进行通信。

我已经研究这个系统两个星期了。



一年前，我在Securika展览上看到了喀山公司Teko的展位。除其他外，他们展示了Security Hub家庭警报。我对两件事感兴趣：不仅支持无线，还支持有线传感器，以及通过常规以太网电缆连接到Internet的能力。

现在，许多制造商生产仅支持无线传感器的家庭警报器。它不仅价格昂贵，而且可靠性较低。我认为至少必须连接两个传感器（打开前门和走廊中的移动）。与路由器的有线连接似乎比Wi-Fi更可靠。

我将从安全中心功能开始。

  • 支持有线安全和火警传感器，以实现短路，开路或两级短路；
  • 多达4个用于有线传感器的输入（每个输入可以连接任意数量的传感器）；
  • 支持多达30个带有自己的安全交换协议的无线传感器；
  • 支持多达六个有线输出（信标，警报器，执行器）；
  • 触摸内存（TM）输入，用于管理iButton键和/或键盘；
  • 通过有线Internet（RJ45连接器）和移动Internet（两个SIM卡）管理和传输有关警报和状态的信息；
  • 通过移动应用程序，计算机程序或网站进行管理和配置；
  • 与iVideon服务集成（在警报期间记录10秒剪辑）；
  • 由三个独立的电源供电，内置电池。

安全中心有三种配置：

  • 仅控制器为4768卢布；
  • “入门套件”，价格为7691卢布（控制器，用于打开门/窗和移动的无线传感器）；
  • 13,653卢布的“带摄像机的套件”（“入门套件” + Ivideon Wi-Fi摄像机）。


要创建廉价的家庭报警系统，您可以购买一个控制器并与廉价的有线传感器配合使用。


当然，对于任何配置，您都可以购买无线传感器，其中有很多。

Security Hub具有许多操作模式和细微的设置，但是如果您将警报视为“开箱即用的工具包”，则非常简单。电源是通过USB AB电缆从常规USB电源提供的。Security HUB应用程序已安装在智能手机上，并已创建一个帐户。


您可以使用移动应用程序布防和撤防警报。发生警报时，将发出推送通知，将发出声音通知（有机会为任何事件设置声音）和语音通知。

Security Hub中的SIM卡仅用于数据传输。警报系统不知道如何呼叫和发送SMS。为了提高警报消息传递的可靠性，您可以配置输出之一，并将外部拨号器与您自己的SIM卡相连，例如，廉价的车载GPS跟踪器。

如果必须经常对警报进行布防和清除，则智能手机上应用程序中的按钮不是最方便的选择。您可以将TM读卡器连接到Security Hub（150-200卢布），然后使用“对讲机”按键片“ i-button”（每个30卢布）进行装卸。


还提供其他用于设置/删除的其他设备：用于输入代码的有线键盘（1411卢布），无线键盘（2065卢布），钥匙串（1062卢布）。


可以设置传感器的延迟时间（当打开的传感器在前门上并且使用TM键或室内键盘进行布/撤防时，这是必需的）。默认情况下，延迟是退出的45秒，进入的延迟是60秒，使用可选的LocalConfig配置程序，您可以更改延迟时间（20-30-45-60-90-120秒）。

“带摄像机的套件”包括一个安全HUb控制器，该控制器安装了3.7V 600 mAh电池，USB 5V 1A电源，两米USB AM-BM电源线，1米UTP跳线，4个3.9kΩ电阻器，Astra无线门/窗打开传感器-3321，带LS14500电池，带Astra-5121温度计的无线运动传感器，带CR123A电池，Ivideon Cute Wi-Fi摄像机（Nobelic NBQ-1110F），USB电源5V 1A，3米长的USB-MicroUSB电源线，快速警报设置指南以及照相机，用于安装传感器的销钉和螺钉，快速入门说明，护照。


控制器板上有一个用于两个SIM卡的插槽，一个防拆开关，一个用于连接TM读卡器的两脚模块，用于连接两个12 V电源的触点，用于四个可编程输入/输出和两个输出的触点。


安装sim卡是可选的，但是如果以太网不起作用，则数据将通过第一个SIM卡传输（如果它不起作用，则通过第二个SIM卡传输）。也可以仅通过移动Internet进行工作而无需有线连接。

Astra-5121运动传感器可在水平面90°区域内2到10米的距离内检测20公斤以上的物体（对动物不敏感）的运动。它可使用长达4年的电池，可在-10至+ 50°C的温度范围内使用。该传感器具有内置温度计。



门/窗开启传感器Astra-3321可以在电池上使用长达7年，可以在-20至+ 50°C的温度下使用，有用于连接其他有线传感器的触点。


Nobelic NBQ-1110F Wi-Fi摄像机的分辨率为1.3MP（1280x720像素），支持H.264压缩协议，运动检测，红外照明长达10 m的夜视能力，并配有内置麦克风，扬声器和microSD卡插槽。相机可以使用iVideon服务。


为了评估图片的质量，我将相机放在房间末端中央的2米高处。到壁橱的距离是5米。

在人造光下拍摄。单击照片以完整尺寸将其打开。


用红外照明的照相机在完全黑暗的环境中拍摄。


摄像机免费使用其运动检测器录制十秒钟的剪辑（您可以在iVideon应用程序中观看它们）。触发警报时，摄像机还会录制十秒钟的剪辑（可以在Security Hub应用程序中查看它们）。滚筒存放一天。


该应用程序显示有关无线传感器和事件历史记录的完整信息。


不幸的是，并非所有的安全中心设置都是直观的。因此，要添加TM密钥，需要单击“管理”按钮，其中有“ +”，“真实标识符”，“启用交互模式”。


接下来，触摸阅读器按键，为按键命名，并指出将被该按键放置/移除的部分。


添加传感器没有问题：只需转到“传感器”，按“ +”，选择连接传感器（有线或无线）的方法，指定有线传感器的输入编号，选择传感器的类型以及将添加传感器的部分。


有了出口，事情就变得有些复杂了。前四个触点可以是输入或输出。第五和第六-仅退出。

最简单的事情是添加一个手动控制的继电器。我们转到“继电器/输出”，按“ +”，“添加继电器”，选择连接方法，想出一个名称，为有线继电器输入从1到6的端子号，选择“手动控制”。



在“继电器/输出”部分，创建的继电器将带有“打开”和“关闭”按钮。这种继电器可用于控制照明，加热或手动打开强大的警笛。


有两种特殊的退出模式-“信标”和“警笛”。“灯塔”是一种指示灯，以灯光和闪烁的方式显示警报的状态（在布防警报时亮起，在拆除/布防延迟期间缓慢闪烁，在警报期间快速闪烁）。在正常警报期间，“警笛”会持续亮起，在火灾期间会间歇性地亮起。

脚本控制使您可以实现输出的各种逻辑。总体而言，安全中心具有五个类别的23个脚本。简要列出一些功能：

  • 在给定的时间间隔内打开继电器（而不是打开/关闭，继电器将具有一个按钮脉冲，例如将其打开5秒钟）。
  • 通过任何传感器打开继电器，并在指定时间后将其关闭（可用于通过运动传感器控制光）；
  • 在湿度传感器上包括浇水植物；
  • 门控（按下按钮时向继电器施加脉冲）；
  • 管理泄漏传感器的水龙头；
  • 预定的出口管理（一天一次或一天三次）；
  • 继电器控制取决于温度。

不仅可以通过移动应用程序配置和管理安全中心。有一个管理站点cloud.security-hub.ru。


PC监控站Astra也有一个程序。


我拍摄了一段简短的视频评论。


安全中心具有许多功能，并且具有许多优点：

  • 支持通过安全协议运行的有线和无线传感器；
  • TM支持；
  • 具有强大控制能力的输出的可用性；
  • 通过有线和移动互联网的互联网连接；
  • 通过移动应用程序，计算机程序或网站进行管理和配置；
  • 与iVideon集成。

安全中心的主要缺点是缺少通过电话和SMS发出的警报。一切都只能通过云进行。显然，该系统首先是为远程控制而开发的，并且没有必要，但是由于Security Hub是作为“自我保护”系统出售的，因此可以添加警报呼叫。当然，您可以将带有单独SIM卡的外部拨号器连接到控制器输出之一，但这是“拐杖”。

另一个不太明显的缺点是缺少使用按钮进行布防的可能性（只需按下按钮即可离开房屋）和TM操作逻辑：所有键都已布防和撤防，因此一个键已布防，再拍一次，你不能。

而且Security Hub没有详细的说明。有维基带有功能和设置的描述，但是，例如，没有脚本列表来解释每个人的操作（它们仅在移动应用程序中，您需要连续单击所有项目以了解它们在做什么）。

但是，安全中心对设置和功能的数量印象深刻。我可以想象在开发这样的系统上花了多少时间和精力。我真的希望开发人员可以在新固件中添加呼叫和SMS警报，以及通过关闭TM输入并编写详细说明来添加武装系统的功能。
