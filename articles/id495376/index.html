Kebosanan karantina. Cara menghabiskan uang paling banyak di Azure dengan satu klik

Beberapa hari yang lalu, Corey Quinn tweeted :

"Seseorang bertanya - saya jawab. Contoh db.r5.24xlarge Enterprise Multi-AZ yang sepenuhnya dicadangkan untuk SQL Server di Bahrain berharga $ 3,118,367. Kompetisi untuk panggilan API paling mahal di cloud AWS diumumkan. "

Diskusi yang sangat produktif dimulai di utas, di mana banyak orang baik berusaha mencari cara yang lebih baik untuk menghabiskan uang pada AWS. Sebagai pengguna Azure, saya merasa sedih dan kesepian, jadi saya memutuskan untuk memperbaiki ketidakadilan ini. Jika Anda berada di karantina dan ingin menghabiskan jumlah uang yang gila di suatu tempat, saya dapat memberitahu Anda. Anda hanya perlu berlangganan Pay-As-You-Go di Azure - dan saya jamin bahwa semua uang hasil jerih payah Anda akan hilang dalam sekejap!

Pemanasan


Hal pertama yang setiap orang waras yang ingin menghabiskan jumlah uang maksimum dari penyedia cloud mana pun akan coba lakukan adalah membeli mesin virtual terbesar dan termahal. Sayangnya, ini tidak terlalu efektif:



Satu contoh Standard_M416ms_v2 dengan 416 vCPU dan 11.400 GB RAM, dibayar untuk tiga tahun sebelumnya, hanya berharga 802 ribu euro, dan ini bahkan tidak jutaan! Lebih buruk lagi, Azure menawarkan diskon 71%, dan diskon seperti itu sepenuhnya bertentangan dengan tujuan kami. Mari kita beri penghargaan untuk kecerdasan buatan: Azure mengatakan jumlah yang direkomendasikan dari contoh ini adalah nol untuk saya. Tebakan yang cukup akurat.
