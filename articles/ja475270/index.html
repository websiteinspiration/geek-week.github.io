SwiftUI：リストビューで展開可能/折りたたみ可能なセクションを作成する



iOS開発の一般的なタスクは、UITableViewのセクションの展開/折りたたみです。今日、私たちはSwiftUIを使用してこのタスクを実現しています。小さなひねりとして、アニメーション化された三角形をセクションヘッダーに追加し、セルも展開します。

Xcode 11.2 for macOS Catalina 10.15.1で開発が行われました

プロジェクトを開始する


Xcode、File-New Project-Single View Appを起動します。ダイアログボックスで、Swift開発言語を指定します。SwiftUIを使用してUIを形成します。



データ


デモンストレーションデータとして、ラテン語の面白い翼のある表現とロシア語への翻訳を使用します。

新しいSwiftファイルをプロジェクトに追加し、Data.swiftという名前を付けて、そこに次のように書き込みます。

struct QuoteDataModel : Identifiable {
    
    var id: String {
        return latin
    }

    var latin : String
    var russian : String
    var expanded = false
}

struct SectionDataModel : Identifiable {
    
    var id: Character {
        return letter
    }
    
    var letter : Character
    var quotes : [QuoteDataModel]
    var expanded = false
}

QuoteDataModelは単一の式のモデルであり、将来的には個々のセルのコンテンツになります。その中に、式の元のテキスト、その翻訳、および「展開された」セルの記号（デフォルトでは「折りたたまれた」）を

格納します。SectionDataModelは、個々のセクションのモデルです。ここでは、セクションの「文字」、この文字で始まる引用符の配列、および記号を格納します。セクション「展開された」（デフォルトでは「折りたたまれた」）

将来的には、これをすべてリストビューに表示します。これには、そのデータが識別可能なプロトコルに準拠している必要があります。これを行うには、idプロパティを定義します。これは、リストの各要素に対して一意である必要があります。

インターフェースを扱いましょう


次に、セクションの見出しと各セルの外観を決定します。

メニューから[ファイル]-[新規]-[ファイル]-[SwiftUIビュー]を選択します。ファイルにHeaderView.swiftという名前を付け、その内容を次のように置き換えます。

import SwiftUI

struct HeaderView : View {
    
    var section : SectionDataModel
    
    var body: some View {
        HStack() {
            Spacer()
            Text(String(section.letter))
                .font(.largeTitle)
                .foregroundColor(Color.black)
            Spacer()
        }
        .background(Color.yellow)
    }
}

struct HeaderView_Previews: PreviewProvider {
    static var previews: some View {
        HeaderView(section: latinities[0])
    }
}



再びファイル-新規-ファイル-SwiftUIビュー。ファイルにQuoteView.swiftという名前を付け、その内容を次のように置き換えます。

import SwiftUI

struct QuoteView: View {
    
    var quote : QuoteDataModel
    
    var body: some View {
        VStack(alignment: .leading, spacing: 5) {
            Text(quote.latin)
                .font(.title)
            if quote.expanded {
                Group() {
                    Divider()
                    Text(quote.russian).font(.body)
                }.transition(.move(edge: .top)).animation(.default)
            }
        }
    }
}

struct QuoteView_Previews: PreviewProvider {

    }

次に、ファイルContentView.swiftを開き、次のようにContentViewの構造を変更します。

struct ContentView: View {
    var body: some View {
        List {
            ForEach(latinities) { section in
                Section(header: HeaderView(section: section), footer: EmptyView()) {
                    if section.expanded {
                        ForEach(section.quotes) { quote in
                            QuoteView(quote: quote)
                        }
                    }
                }
            }
        }
        .listStyle(GroupedListStyle())
    }
}

セクションヘッダーを少し変更


HeaderView.swiftファイルに戻ります。HeaderView構造内で、本文の直後に、次を追加します。

struct Triangle : Shape {
        func path(in rect: CGRect) -> Path {
            var path = Path()
            path.move(to: CGPoint(x: 0, y: 0))
            path.addLine(to: CGPoint(x: 0, y: rect.height - 1))
            path.addLine(to: CGPoint(x: sqrt(3)*(rect.height)/2, y: rect.height/2))
            path.closeSubpath()
            return path
        }
    }

この構造は正三角形を返します。次に、三角形をヘッダーに追加します。HStack内で、最初のSpacerの前にこれを追加します。

風景を復活させる


ファイルContentView.swiftに戻ります。ContenView構造内で、userData定義の直後に、2つの関数を追加します。

func sectionIndex(section : SectionDataModel) -> Int {
        userData.latinities.firstIndex(where: {$0.letter == section.letter})!
}
    
func quoteIndex(section : Int, quote : QuoteDataModel) -> Int {
        return userData.latinities[section].quotes.firstIndex(where: {$0.latin == quote.latin})!
}

作成するセクションヘッダーとセル にonTapGesture修飾子を追加します。ボディコンテンツの最終ビュー：

var body: some View {
        List {
            ForEach(userData.latinities) { section in
                Section(header: HeaderView(section: section)
                    .onTapGesture {
                        self.userData.latinities[self.sectionIndex(section: section)].expanded.toggle()
                }, footer: EmptyView()) {
                    if section.expanded {
                        ForEach(section.quotes) { quote in
                            QuoteView(quote: quote)
                                .onTapGesture {
                                    let sectionIndex = self.sectionIndex(section: section)
                                    let quoteIndex = self.quoteIndex(section: sectionIndex, quote: quote)
                                    self.userData.latinities[sectionIndex].quotes[quoteIndex].expanded.toggle()
                            }
                        }
                    }
                }
            }
        }
        .listStyle(GroupedListStyle())
    }

sectionIndexとquoteUndex機能は、渡されたセクションや表現のインデックスを返します。これらのインデックスを受け取ったら、緯度の配列の展開されたプロパティの値を変更します。これにより、セクションまたは式の折りたたみ/展開が行われます。
