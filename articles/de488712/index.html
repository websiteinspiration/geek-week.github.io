Check Point WatchTower - Verwalten Sie NGFW von einem Smartphone aus



Guten Tag, heute möchte ich auf das Thema Ausrüstung für kleine Unternehmen und Büros für bis zu 150-300 Benutzer eingehen. Moderne Sicherheitsbedrohungen erfordern einen Schutz des Netzwerkumfangs, unabhängig von der Größe des Unternehmens. CheckPoint bietet eine Reihe von Produkten in der Serie speziell für diese Aufgaben: 1400 Serie , 1500 - Serie . Equipment (SMB) wird mit einer speziell entwickelten Version von Gaia Embedded (für ARM-Architektur) geliefert, die ihre eigenen Merkmale beim Einrichten und bei der Interaktion mit dem Administrator aufweist. Die Verwaltung kann entweder lokal, dh direkt über das Security Gateway selbst (über die Webschnittstelle) oder zentral über einen separaten Verwaltungsserver (über SmartConsole) erfolgen.

Vielleicht ist die Neuigkeit für Sie das Erscheinen der dritten Option zur Verwaltung Ihres Gateways mit einem Smartphone . Mit CheckPoint WatchTower können Sie mithilfe einer speziellen mobilen Anwendung eine Verbindung zu Ihrem Security Gateway herstellen.

Nach der Synchronisierung erhalten Sie sofort Benachrichtigungen über Sicherheitsereignisse. Sie können infizierte Endgeräte blockieren, Statistiken anzeigen und Sicherheitsrichtlinien konfigurieren.

Eine wichtige Voraussetzung für das Funktionieren der Anwendung ist die installierte Version von Gaia R77.20.86 und höher sowie die lokale Verwaltung von Gateway-Richtlinien!
