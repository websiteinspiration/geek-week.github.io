Installer Debian sur Netgear Stora

L'autre jour, j'ai eu ce miracle entre les mains: netgear ms 2000 - stora . J'ai décidé d'abandonner immédiatement l'utilisation du système d'exploitation intégré et d'installer Debian sur le disque dur.

Les informations sur le réseau sont un peu dispersées, les liens sont morts depuis longtemps, j'ai donc décidé de mettre à jour le processus d'installation de Debian sur stora. Peu importe, bienvenue au chat.

Cet article était la principale source .

Tout d'abord, nous avons besoin d'images pour installer le système: je l'ai pris ici . Téléchargez les deux fichiers. Ces fichiers sont écrits à la racine sur un lecteur flash formaté en fat32.
Un convertisseur USB vers UART PL2303TA est également requis.

J'ai eu ça


Vous aurez également besoin d'un logiciel pour vous connecter à un morceau de fer, par exemple hyperterminal ou mastic (je ne l'ai pas obtenu avec du mastic: tout le temps que je suis entré dans le terminal, j'ai utilisé l'hyperterminal.

Pour connecter le morceau de fer avec un câble, je dois d'abord le démonter. Le processus est simple, je ne le décrirai donc pas . Eh bien, vous ne devez pas oublier d'insérer un disque dur dans le premier emplacement du côté, sur lequel l'installation réelle aura lieu.

Après avoir analysé le matériel, nous connectons l'adaptateur. Attention, ne connectez pas le fil rouge, c'est-à-dire que vous devez connecter uniquement 3 fils (à partir de la batterie: noir , vert blanc).
Ainsi, le fil est connecté, les pilotes sont accrochés. Dans le pilote du port com, nous définissons les paramètres: vitesse 115200, nombre de bits 8, bits d'arrêt 1, sans parité. Après cela, allumez le morceau de fer et connectez-le dans le terminal. Lorsque vous voyez Appuyez sur n'importe quelle touche ... appuyez sur n'importe quelle touche pour accéder au chargeur u-boot.

Une petite digression.

La liste des commandes avec lesquelles nous allons opérer et qui seront utiles:
usb reset, ide reset — usb, ide
fatls, ext2ls — fat ext2 .
setenv —
saveenv —
reset —
printenv —
printenv NAME — NAME
help —

Après avoir entré le chargeur de démarrage, définissez les paramètres réseau, initialisez le périphérique USB, vérifiez qu'il existe des fichiers nécessaires sur le lecteur flash, enregistrez ces paramètres dans la mémoire du périphérique et redémarrez:

Équipes 
usb reset
fatls usb 0
setenv mainlineLinux yes
setenv arcNumber 2743
setenv ipaddr your_IP
setenv gatewayip your_GW_IP
setenv dnsip your_DNS_IP
saveenv
reset

Après le redémarrage, entrez les commandes pour démarrer l'installation de Debian:

usb reset
fatload usb 0 0x200000 uImage
fatload usb 0 0x800000 uInitrd
setenv bootargs console=ttyS0,115200n8 base-installer/initramfs-tools/driver-policy=most
bootm 0x200000 0x800000

Après cela, l'installation standard de Debian en mode texte ira. Nous installons le système, redémarrons après l'installation, saisissons uboot et saisissons les commandes pour démarrer l'appareil à partir du disque dur:

setenv bootcmd_ide 'ide reset; ext2load ide 0 0x200000 /uImage; ext2load ide 0 0x800000 /uInitrd'
setenv bootcmd 'setenv bootargs $(console) root=/dev/sda2; run bootcmd_ide; bootm 0x200000 0x800000'
saveenv
reset

Après le redémarrage, le debian est chargé à partir du disque dur, ce qui était initialement recherché.

PS Restauration du chargeur de démarrage d'origine: 
setenv mainlineLinux=no
setenv arcNumber
setenv bootcmd_ide
setenv bootcmd 'nand read.e 0x800000 0x100000 0x300000; setenv bootargs $(console) $(bootargs_root); bootm 0x800000'
saveenv
