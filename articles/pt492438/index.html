Os jogadores de PC do Reddit combinam seu poder de computação em torno do projeto Folding @ Home para combater o COVID-19

Os moderadores do subreddit da PC Master pediram aos proprietários de poderosas placas de vídeo para jogos que se juntassem à computação em nuvem científica do projeto Stanford Folding @ Home , que agora está envolvido no cálculo de modelos em oito direções ao mesmo tempo para combater o COVID-19. A iniciativa da comunidade Reddit já era apoiada pela Nvidia, que instou o Twitter a fornecer aos cientistas o poder computacional de suas placas de vídeo:


É importante notar que o subreddit da raça PC Master não está tentando exagerar no tópico do coronavírus : o suporte ao Folding @ Home e o primeiro mega-thread sistemático dedicado ao projeto foram publicados há 4 meses, quando eles nem ouviram falar sobre o coronavírus. Além disso, os usuários mencionaram o F @ H no subreddit nos últimos cinco anos.

Como participar de cálculos coletivos, mesmo se você não se importa com o COVID-19 e deseja encontrar rapidamente uma cura para o câncer - sob o gato.

Para participar dos cálculos e apoiar os cientistas com suas próprias capacidades, você tem três opções.

Primeiro: entre no grupo reddit para computadores da corrida subreddit PC Master.

Segundo: insira o código específico do estudo de interesse e conte-o você mesmo, na direção escolhida.

Uma opção para quem não se importa com o COVID-19: escolha entre as quatro áreas propostas (câncer, alzheimer, etc.) e conte lá.

Para se tornar parte do grupo de computação, você precisará instalar um software F @ H especial no seu PC. Uma lista completa dos instaladores pode ser encontrada aqui . Agora pronto para o uso, Windows 10, MacOS, Ubuntu, CentOS, Fedora, RedHat, Debian e Mint são suportados. Também vale a pena ler este artigo.e o tópico mencionado anteriormente no próprio Reddit . Para os interessados, um repositório no GitHub .

Para não perder tempo: para se conectar aos cálculos, você deve especificar o código do projeto (grupo de liquidação) nas configurações de software F @ H. Os desenvolvedores sugerem o uso de uma interface da web, mas pode ser chamada de estável com grande dificuldade. Na configuração do cliente, no campo "Número da equipe", insira o código do grupo Reddit, que considera tudo em uma linha ou um cálculo específico usando o COVID.

Para estações de trabalho poderosas com placas gráficas de última geração ou com equipamentos completos na GPU, é recomendável obter uma chave pessoal entrando em contato com a Stanford. Isso otimizará a carga e os cálculos.

Importante! Inicialmente, pode parecer que o FAHControl não mostra sinais de vida e que tudo quebrou nas configurações automáticas. Nesse caso, você provavelmente enfrentou os mesmos problemas do escritório editorial do blog: muito tempo para conseguir o emprego. Após várias sessões e conexões, o software deve finalmente receber os dados para o cálculo e sua placa de vídeo fará o trabalho.
