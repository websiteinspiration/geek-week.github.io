gogrep：Go代码的结构化搜索和替换

gogrep是我最喜欢使用Go的实用程序之一。它允许您按语法模式查找代码，按表达式类型过滤结果，以及执行替换（也按模式）。


在本文中，我将告诉您如何使用它gogrep以及VS Code扩展，以便gogrep直接从编辑器中进行操作。



为什么需要gogrep


如果是摘要，则对以下gogrep情况可能有用：


  • 重构
  • 代码基础研究
  • 搜索可疑代码（例如：ruleguard）

考虑一个示例，该示例演示了结构化搜索的优雅性和有效性。


功能a()和b()执行相同的操作：


func a(xs []int) []int {
  xs = append(xs, 1)
  xs = append(xs, 2)
  return xs
}

func b(xs []int) []int {
  xs = append(xs, 1, 2)
  return xs
}

假设我们要重写所有append可以折叠呼叫的位置。


让我们尝试gogrep：


  • 使用-x模板查找所有匹配对。$x=append($x,$a); $x=append($x,$b)
  • 通过-s模板，$x=append($x,$a,$b)我们可以获得所需的替换
  • 通过传递参数，-w将更新所有受影响的文件。

gogrep -w -x '$x=append($x,$a);$x=append($x,$b)' -s '$x=append($x,$a,$b)' ./...

如果您安装VS Code扩展，它将变得更加容易。
